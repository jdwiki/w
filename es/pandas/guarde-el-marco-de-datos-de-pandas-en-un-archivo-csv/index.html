Parámetros #

Parámetro Descripción
ruta_o_buf cadena o identificador de archivo, valor predeterminado Ninguno Ruta de archivo u objeto, si se proporciona Ninguno, el resultado se devuelve como una cadena.
septiembre carácter, por defecto ‘,’ Delimitador de campo para el archivo de salida.
na_rep cadena, por defecto ’’ Representación de datos faltantes
formato_flotante cadena, predeterminado Ninguno Cadena de formato para números de punto flotante
columnas secuencia, Columnas opcionales para escribir
encabezado booleano o lista de cadenas, predeterminado Verdadero Escriba los nombres de las columnas. Si se proporciona una lista de cadenas, se supone que son alias para los nombres de columna
índice booleano, predeterminado Verdadero Escribir nombres de fila (índice)
etiqueta_índice cadena o secuencia, o Falso, predeterminado Ninguno Etiqueta de columna para la(s) columna(s) de índice si lo desea. Si se proporciona Ninguno y el encabezado y el índice son verdaderos, se utilizan los nombres de índice. Se debe dar una secuencia si el DataFrame usa MultiIndex. Si es falso, no imprima campos para nombres de índice. Utilice index_label=False para facilitar la importación en R
nanRep Ninguno en desuso, utilice na_rep
modo str Modo de escritura Python, por defecto ‘w’
codificación cadena, opcional Una cadena que representa la codificación a usar en el archivo de salida, por defecto es ‘ascii’ en Python 2 y ‘utf-8’ en Python 3.
compresión cadena, opcional una cadena que representa la compresión a usar en el archivo de salida, los valores permitidos son ‘gzip’, ‘bz2’, ‘xz’, solo se usan cuando el primer argumento es un nombre de archivo
terminador_de_línea cadena, por defecto ’n’ El carácter de nueva línea o la secuencia de caracteres que se utilizará en el archivo de salida
citando la constante opcional del módulo csv tiene como valor predeterminado csv.QUOTE_MINIMAL
cita cadena (longitud 1), carácter predeterminado ‘"’ utilizado para citar campos
comillas dobles booleano, cotización predeterminada de True Control de quotechar dentro de un campo
escapechar cadena (longitud 1), carácter predeterminado Ninguno utilizado para escapar de sep y quotechar cuando corresponda
tamaño trozo int o Ninguna filas para escribir a la vez
tupleize_cols booleano, predeterminado Falso escribir columnas multi_index como una lista de tuplas (si es Verdadero) o nuevo (formato expandido) si es Falso)
formato_fecha cadena, predeterminado Ninguno Cadena de formato para objetos de fecha y hora
decimales cadena, por defecto ‘.’ Carácter reconocido como separador decimal. P.ej. use ‘,’ para datos europeos

Crea un marco de datos aleatorio y escribe en .csv #

Cree un marco de datos simple.

import numpy as np
import pandas as pd

# Set the seed so that the numbers can be reproduced.
np.random.seed(0)  

df = pd.DataFrame(np.random.randn(5, 3), columns=list('ABC'))

# Another way to set column names is "columns=['column_1_name','column_2_name','column_3_name']"

df

      A         B         C
0  1.764052  0.400157  0.978738
1  2.240893  1.867558 -0.977278
2  0.950088 -0.151357 -0.103219
3  0.410599  0.144044  1.454274
4  0.761038  0.121675  0.443863

Ahora, escribe en un archivo CSV:

df.to_csv('example.csv', index=False)

Contenido de ejemplo.csv:

A,B,C
1.76405234597,0.400157208367,0.978737984106
2.2408931992,1.86755799015,-0.977277879876
0.950088417526,-0.151357208298,-0.103218851794
0.410598501938,0.144043571161,1.45427350696
0.761037725147,0.121675016493,0.443863232745

Tenga en cuenta que especificamos index=False para que los índices generados automáticamente (números de fila 0,1,2,3,4) no se incluyan en el archivo CSV. Inclúyalo si necesita la columna de índice, así:

df.to_csv('example.csv', index=True)  # Or just leave off the index param; default is True

Contenido de ejemplo.csv:

,A,B,C
0,1.76405234597,0.400157208367,0.978737984106
1,2.2408931992,1.86755799015,-0.977277879876
2,0.950088417526,-0.151357208298,-0.103218851794
3,0.410598501938,0.144043571161,1.45427350696
4,0.761037725147,0.121675016493,0.443863232745

También tenga en cuenta que puede eliminar el encabezado si no es necesario con header=False. Esta es la salida más simple:

df.to_csv('example.csv', index=False, header=False)

Contenido de ejemplo.csv:

1.76405234597,0.400157208367,0.978737984106
2.2408931992,1.86755799015,-0.977277879876
0.950088417526,-0.151357208298,-0.103218851794
0.410598501938,0.144043571161,1.45427350696
0.761037725147,0.121675016493,0.443863232745

El delimitador se puede establecer mediante el argumento sep=, aunque el separador estándar para archivos csv es ','.

df.to_csv('ejemplo.csv', índice=Falso, encabezado=Falso, sep='\t')

1.76405234597    0.400157208367    0.978737984106
2.2408931992    1.86755799015    -0.977277879876
0.950088417526    -0.151357208298    -0.103218851794
0.410598501938    0.144043571161    1.45427350696
0.761037725147    0.121675016493    0.443863232745

Guarde Pandas DataFrame de la lista a dictados a csv sin índice y con codificación de datos #

import pandas as pd
data = [
    {'name': 'Daniel', 'country': 'Uganda'},
    {'name': 'Yao', 'country': 'China'},
    {'name': 'James', 'country': 'Colombia'},
]
df = pd.DataFrame(data)
filename = 'people.csv'
df.to_csv(filename, index=False, encoding='utf-8')
