Creando tu primera página de hola mundo #

Cree una nueva aplicación de rieles hello-world desde el comando en Windows y Terminal en Linux.

rails new hello-world

Ahora muévase al nuevo directorio de aplicaciones

cd hello-world

Ahora genera un controlador

rails generate controller hello_world index

Aquí index es el nombre del método en el controlador hello_world. Puede comprobarlo abriendo el archivo app/controllers/hello_world_controller.rb en el directorio de su aplicación. El código se ve así:

class HelloWorldController < ApplicationController
  def index
  end
end

Una ruta se agrega automáticamente en su archivo config/routes.rb que apunta a su método. Vea el código en su archivo routes.rb.

    Rails.application.routes.draw do
      get 'hello_world/index'

      # For details on the DSL available within this file, see http://guides.rubyonrails.org/routing.html
   end

Ahora abre el archivo app/views/hello_world/index.html.rb Borrar todo el contenido y escribir

Hola, esta es mi primera página sobre raíles.

Inicie el servidor de rieles:

rails server

Abre esta url en tu navegador:

http://localhost:3000/hello_world/

Debería ver:

Hello, this is my first rails page

Haz de tu nueva página, tu página de inicio. En el archivo route.rb en la carpeta de configuración, elimine la línea get 'hello_world/index' y agregue:

root 'hello_world#index'

Ahora abre: http://localhost:3000/ Verás: Hola, este es mi primer riel Estás listo.

Configurar Ruby On Rails en Ubuntu 17.04 Zesty Zapus #

Esto tomará alrededor de 30 minutos. Estaremos configurando Ruby on Rails Development Environment en Ubuntu 16.10 Yakkety Yak. Querrá descargar la última versión de escritorio aquí: http://releases.ubuntu.com/17.04/

Abre tu terminal usando Ctrl + Alt + T.

Instalando Rubí #

El primer paso es instalar Dependencias para Ruby.

sudo apt-get update
sudo apt-get install git-core curl zlib1g-dev build-essential libssl-dev libreadline-dev libyaml-dev libsqlite3-dev sqlite3 libxml2-dev libxslt1-dev libcurl4-openssl-dev python-software-properties libffi-dev nodejs

Usaremos Ruby versión 2.4.0 y se recomienda. Instale Ruby usando uno de los tres métodos. Cada uno tiene sus propios beneficios, la mayoría de las personas prefieren rbenv, pero si está familiarizado con rvm, hágalo.

Nota: Solo los usuarios de rbenv necesitan ejecutar rbenv rehash después de instalar bundler.

Instalación con rbenv: #

cd
git clone https://github.com/rbenv/rbenv.git ~/.rbenv
echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc
echo 'eval "$(rbenv init -)"' >> ~/.bashrc
exec $SHELL

git clone https://github.com/rbenv/ruby-build.git ~/.rbenv/plugins/ruby-build
echo 'export PATH="$HOME/.rbenv/plugins/ruby-build/bin:$PATH"' >> ~/.bashrc
exec $SHELL

rbenv install 2.4.0
rbenv global 2.4.0
ruby -v

Luego instale Bundler:

gem install bundler

Use rbenv rehash después de instalar el paquete.

Instalación con rvm: #

sudo apt-get install libgdbm-dev libncurses5-dev automake libtool bison libffi-dev
gpg --keyserver hkp://keys.gnupg.net --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3
curl -sSL https://get.rvm.io | bash -s stable
source ~/.rvm/scripts/rvm
rvm install 2.4.0
rvm use 2.4.0 --default
ruby -v

Luego instale Bundler:

gem install bundler

Instalación con fuente: #

cd
wget http://ftp.ruby-lang.org/pub/ruby/2.4/ruby-2.4.0.tar.gz
tar -xzvf ruby-2.4.0.tar.gz
cd ruby-2.4.0/
./configure
make
sudo make install
ruby -v

Luego, el último paso es instalar Bundler:

gem install bundler

Instalación de rieles #

Rails se envía con tantas dependencias en estos días que necesitaremos instalar un tiempo de ejecución de JavaScript como NodeJS. Esto le permite usar Coffee-script y Asset Pipeline en Rails, que combina y minimiza su JavaScript para proporcionar un entorno de producción más rápido.

Instale NodeJS usando el repositorio oficial:

curl -sL https://deb.nodesource.com/setup_4.x | sudo -E bash -
sudo apt-get install -y nodejs

Ahora instala Rails 5.0.1:

gem install rails -v 5.0.1

Si está utilizando rbenv, ejecute el siguiente comando para que Rails esté disponible:

 rbenv rehash

Ahora que Rails está instalado, ejecute rails -v para asegurarse de que Rails esté instalado correctamente:

rails -v
# Rails 5.0.1

Si obtiene un resultado diferente por algún motivo, significa que es posible que su entorno no esté configurado correctamente.

Configuración de PostgreSQL #

Rails se envía con sqlite3 como base de datos predeterminada. Lo más probable es que no quiera usarlo porque está almacenado como un archivo simple en el disco. Probablemente querrá algo más robusto como MySQL o PostgreSQL.

Hay mucha documentación sobre ambos, por lo que puede elegir uno con el que se sienta más cómodo.

sudo sh -c "echo 'deb http://apt.postgresql.org/pub/repos/apt/ xenial-pgdg main' > /etc/apt/sources.list.d/pgdg.list"
wget --quiet -O - http://apt.postgresql.org/pub/repos/apt/ACCC4CF8.asc | sudo apt-key add -
sudo apt-get update
sudo apt-get install postgresql-common
sudo apt-get install postgresql-9.5 libpq-dev

La instalación de PostgreSQL no configura un usuario por usted, por lo que deberá seguir estos pasos para crear un usuario con permiso para crear bases de datos. Siéntase libre de reemplazar Hizqeel con su nombre de usuario.

sudo -u postgres createuser hizqeel -s

# If you would like to set a password for the user, you can do the following
sudo -u postgres psql
postgres=# \password hizqeel

Instalación o configuración #

En la plataforma de Windows

Paso 1: Instalación de Ruby

Si ya ha instalado Ruby en su PC, puede omitir este paso.

  • Vaya a la página de descargas de rubyinstaller.org y descargue una de las versiones estables de ruby ​​correspondientes a la arquitectura de su plataforma Windows.
  • De nuevo, descargue el kit de desarrollo para su versión de Ruby [formulario aquí] 1.
  • Ahora instala ruby.
  • Este es el momento de mejorar su instalación de Ruby con el kit de desarrollo de Ruby. Esto es requerido por algunas de las gemas para compilar en tu pc.
  • Extraiga el archivo del kit de desarrollo de Ruby en una carpeta cerca de la ruta de instalación de Ruby C:\Ruby-**. Por ejemplo C:\rubyDevkit.
  • Ahora abre cmd.exe y muévete al directorio donde extrajiste tu devkit.
  • Luego ejecute este comando ruby dk.rb init. Esto inicializará su instalación. Luego ejecute ruby dk.rb install.

Paso 2: Instalar rieles

  • Después de la instalación exitosa de Ruby. El siguiente paso es instalar rieles. Antes de instalar Rails, instale la gema del paquete ejecutando gem install bundler desde la ventana del símbolo del sistema.

  • Después de instalar el paquete ahora ejecute gem install rails -v version_of_rails por ejemplo -v 5.0.0.1.

  • Si todo el proceso se completó sin errores, instaló Rails con éxito.
