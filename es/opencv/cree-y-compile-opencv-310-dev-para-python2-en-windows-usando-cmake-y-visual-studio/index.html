Cree y compile opencv 3.1.0-dev para Python2 en Windows usando CMake y Visual Studio

  • Lectura de imágenes y conversión a escala de grises

En esta página

  • Lectura de imágenes y conversión a escala de grises

Construir y compilar opencv 3.1.0-dev para obtener acceso a módulos no gratuitos puede ser un dolor de cabeza para algunas personas, especialmente en máquinas con Windows. A diferencia de Ubuntu, la configuración de opencv para Windows lleva algo de tiempo y requiere que se instalen un par de dependencias pf antes de construir y compilar.

Los programas que debe descargar e instalar antes de continuar con cualquier paso son:

  1. Python 2.7.x o Python 3.x.x
  2. CMake

Si va a descargar Python para Win32, también debe descargar CMake para Win32 incluso si está utilizando una máquina de 64 bits.

Se recomienda descargar los programas de 32 bits porque algunas bibliotecas de Python solo son compatibles con máquinas de 32 bits, así que para evitar problemas, simplemente instale todo en la versión de 32 bits.

  1. Comunidad de Visual Studio 2013

  2. Numpy para Python2.7 Win32

Después de instalar todas las dependencias anteriores, reinicie su PC y luego estará listo para continuar con el siguiente paso.

Paso 2:

Si no eres el tipo de persona que prefiere leer, puedes ver este tutorial. El tutorial lo lleva desde aquí hasta el final de esta documentación.

Deberá obtener opencv y opencv_contrib de github. Puedes encontrar ambos en:

  1. cv abierto
  2. opencv_contrib

Cree un directorio llamado opencv-3.1.0 donde en este director creará otros dos directorios, uno para la compilación y otro para las fuentes. Colocará los dos archivos zip descargados en el archivo de fuentes después de la extracción.

Por ejemplo, su directorio opencv-3.1.0 se encuentra en la unidad C, por lo que tendrá tres rutas:

  1. C:\opencv-3.1.0
  2. C:\opencv-3.1.0\construir
  3. C:\opencv-3.1.0\fuentes

El tercer directorio incluirá dos rutas:

  1. C:\opencv-3.1.0\fuentes\opencv
  2. C:\opencv-3.1.0\fuentes\opencv_contrib

** Ahora está hecho con preparación. Hagamos algunas cosas útiles. **

Paso 3:

Ejecute CMake como administrador. Aparecerá una ventana como esta y tendrás que proporcionar dos directorios uno para las fuentes y el otro para donde se compilará el opencv. La imagen de abajo puede ayudarte más que las palabras.

A continuación, haga clic en configurar y será promovido para proporcionar los generadores; es decir, compiladores; para opencv. Debe proporcionar el cl.exe ubicado en Microsoft Visual Studio 2013. Haga clic en especificar generadores nativos y aparecerá una ventana emergente como la siguiente:

Las rutas serán algo así: C:/Program Files (x86)/Microsoft Visual Studio 12.0/VC/bin/cl.exe. Proporcione su ruta para los campos C y C++. Haga clic en finalizar y espere hasta que finalice la configuración. Debería obtener cero errores si siguió todos los pasos anteriores correctamente.

Después de que CMake termine de configurarse, verá nuevos elementos que aparecen en la ventana de CMake que están resaltados en rojo. Será algo como:

Marque las compilaciones que necesita haciendo clic en el cuadro cuadrado pequeño. Busque la línea OPENCV_EXTRA_MODULES_PATH y proporcione el directorio de módulos dentro de opencv_contrib dentro del directorio de fuentes.

Una vez que haya terminado todo lo que necesita y haya proporcionado la ruta para los módulos adicionales, presione configurar nuevamente para actualizar. Las líneas resaltadas previamente ya no estarán resaltadas y los nuevos campos se resaltarán en rojo en su lugar.

También marque las casillas para lo que necesite construir.

Asegúrate de que BUILD_opencv_contrib_world y BUILD_opencv_world estén ambos desmarcados. Probablemente hay un error en el que se produce un error cuando se comprueba alguno de estos últimos.

Al final de este paso, haga clic en Generar y habrá terminado con CMake y podrá cerrarlo. Si no hay errores, recibirá un mensaje al final del panel inferior que dice Generando hecho.

Paso 4:

Abra el directorio de compilación ubicado en opencv-3.1.0 y encontrará un montón de carpetas y archivos nuevos en su interior. Era una carpeta vacía al comienzo de este proceso.

Solo tratará con el archivo OpenCV.sln y no hará nada con los demás archivos. Abra este archivo con la versión que utilizó durante la compilación en CMake en el paso anterior. Tiene que ser Visual Microsoft 2013.

Cuando abra el archivo .sln, tenga paciencia, ya que lleva algo de tiempo preparar todo para la construcción. Cuando Listo está fijo (sin cambios), puede comenzar a construir sus objetivos. Comience a construir como se muestra en la imagen a continuación. También asegúrese de que la “Configuración de la solución” sea “Liberar” y no “Depurar”.

Paso 5:

Cuando finalice la construcción, deberá copiar y pegar un par de archivos del directorio de construcción en el directorio Python27.

Busque el archivo cv2.pyd y cópielo en el directorio site-packages en Python27. El cv2.pyd debería estar presente en C:\opencv-3.1.0\build\lib\Release. Después de eso, copie solo los archivos .dll dentro de C:\opencv-3.1.0\build\bin\Release en el directorio principal de Python27 en esta ubicación C:\Python27 .

Al final de este paso, reinicie su PC.

Verificación:

Abra IDLE y dentro del shell de Python escriba:

>>> import cv2
>>> print cv2.__version__
3.1.0-dev

Lectura de imágenes y conversión a escala de grises #

import cv2
import numpy as np

img = cv2.imread('<your_image>')
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

cv2.imshow('image', img)
cv2.imshow('gray', gray)

cv2.waitKey(0)
cv2.destroyAllWindows()
