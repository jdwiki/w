Modificación del color de la imagen en OpenCV - kmeans(). Para escanear todos los píxeles de una imagen y reemplazar los valores de píxeles con colores genéricos. #

<código> #incluye opencv2/opencv.hpp> #incluir <código>vector> <código>utilizando espacio de nombres estándar; utilizando el espacio de nombres cv; int principal() { Mat3b img = imread(“test.jpg”);

imshow("Original", img);

// Cluster

int K = 8;
int n = img.rows * img.cols;
Mat data = img.reshape(1, n);
data.convertTo(data, CV_32F);

Mat labels;
Mat1f colors;
kmeans(data, K, labels, cv::TermCriteria(), 1, cv::KMEANS_PP_CENTERS, colors);

for (int i = 0; i < n; ++i)
{
    data.at<float>(i, 0) = colors(labels.at<int>(i), 0);
    data.at<float>(i, 1) = colors(labels.at<int>(i), 1);
    data.at<float>(i, 2) = colors(labels.at<int>(i), 2);
}

Mat reduced = data.reshape(3, img.rows);
reduced.convertTo(reduced, CV_8U);



imshow("Reduced", reduced);
waitKey(0);

return 0;

}

Modificación de imágenes píxel a píxel #

En OpenCV, las imágenes pueden ser RGB/BGR, HSV, en escala de grises, en blanco y negro, etc. Es crucial conocer el tipo de datos antes de trabajar con imágenes.

Los tipos de datos de imagen son principalmente CV_8UC3 (Matriz de uchar con 3 canales) y CV_8U (Matriz de uchar con 1 canal), sin embargo, también es posible la conversión a otros tipos como CV_32FC3, CV_64F. (ver [tipos de datos][1])

Considere que la imagen es una imagen RGB que se lee mediante la función imread.

Mat rgb = imread('path/to/rgb/image', CV_LOAD_IMAGE_COLOR);
//to set RED pixel value of (i,j)th to X,
rgb.at<Vec3b>(i,j)[0] = X;

Del mismo modo, si la imagen está en escala de grises,

gray.at<uchar>(i,j) = X;

Tenga en cuenta que, en OpenCV, las imágenes en blanco y negro se almacenan como tipo CV_8U con los valores 0 y 255. Por lo tanto, cambiar las imágenes BW es lo mismo que las imágenes grises.

[1]: https://www.wikiod.com/es/opencv/estructuras-basicas#Tipo de datos

Establecer toda la imagen en un color sólido #

Dado un cv::Mat img no vacío de algún tamaño, podemos llenarlo con un color sólido de varias maneras:

img = cv::Scalar(blueVal,greenVal,redVal);

o, de forma más general, soporte de máscaras, cv::Mat::setTo():

img.setTo(cv::Scalar(blueVal,greenVal,redVal));

Si está utilizando la antigua API C de OpenCV con IplImage* img:

Usar:

cvSet(img, CV_RGB(redVal,greenVal,blueVal));
