Conceptos básicos #

  • Debe tener una línea de comando de Linux o conectarse usando SSH a su servidor para poder usar scripts de shell.
  • Ve a tu MAGENTO_ROOT/shell
  • El script se puede ejecutar escribiendo, es decir,
php -f indexer.php help

Métodos Core Shell por archivos #

  1. resumen.php

  2. indexador.php

  3. compilador.php

  4. registro.php

Scripts de shell php personalizados #

A veces, necesitamos acceder a Magento fuera de un navegador web para omitir los tiempos de ejecución o establecer diferentes cosas que no afectarán la interfaz.

Hay 2 formas de arrancar Magento, pero solo una es la forma de Magento. Lea más arriba en la sección de ejemplos.

Usar shell sin extender Mage_Shell_Abstract #

Bootstrapping Magento llamando: #

require_once 'app/Mage.php';
Mage::app();
// Your code

Esta es la forma más simple pero no realmente la forma de Magento porque no estamos usando una clase que extiende Mage_Shell_Abstract - la clase que cuando se extiende nos proporciona herramientas para analizar los argumentos de la línea de comandos, llama a __applyPhpVariables() en su constructor (función analiza los archivos .htaccess y aplica la configuración de php al script de shell).

Usar shell a la manera de Magento - extender Mage_Shell_Abstract #

Modo Magento #

El archivo reside en shell/custom.php

<?php
require_once' abstract.php';

class Stackoverflow_Shell_Custom extends Mage_Shell_Abstract
{

protected $_argname = array();
 
    public function __construct() {
        parent::__construct();
 
        // Time limit to infinity
        set_time_limit(0);     
 
        // Get command line argument named "argname"
        // Accepts multiple values (comma separated)
        if($this->getArg('argname')) {
            $this->_argname = array_merge(
                $this->_argname,
                array_map(
                    'trim',
                    explode(',', $this->getArg('argname'))
                )
            );
        }
    }

 // Shell script point of entry
    public function run() {
 
    }
 
    // Usage help
    public function usageHelp()
    {
        return <<<USAGE
Usage:  php -f scriptname.php -- [options]
 
  --argname <argvalue>       Argument description
 
  help                   This help
 
USAGE;
    }
}
// Instantiate
$shell = new Stackoverflow_Shell_Custom();
 
// Initiate script
$shell->run();

}

Realización de reindexación desde CLI #

Ver el estado de:

php indexer.php status

Reindexar todo

php indexer.php reindexall

Reindexar índice específico

php indexer.php --reindex CODE (see list below)
