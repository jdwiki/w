Enlace a casillas de verificación #

Casillas de verificación básicas

export class MyViewModel {
    favoriteColors = [];
    colors = ['Red', 'Yellow', 'Pink', 'Green', 'Purple', 'Orange', 'Blue'];
}

<template>
  <label repeat.for="color of colors">
    <input type="checkbox" value.bind="color" checked.bind="favoriteColors" />
    ${color}
  </label>
  
  <p>Favourite colors:</p>
  <ul if.bind="favoriteColors">
    <li repeat.for="color of favoriteColors">${color}</li>
  </ul>
</template>

Casillas de verificación con matrices de objetos

export class MyViewModel {
    people = [];
    selectedPeople = [];
    
    constructor() {
        this.people = [
            {name: 'John Michaels'},
            {name: 'Gary Stevens'},
            {name: 'Carrie Smitch'},
            {name: 'Jesus Wohau'}
        ];
    }
}

<template>
  <label repeat.for="person of people">
    <input type="checkbox" model.bind="person" checked.bind="selectedPeople" />
    ${person.name}
  </label>
  
  <p>Selected people:</p>
  <ul if.bind="selectedPeople">
    <li repeat.for="person of selectedPeople">${person.name}</li>
  </ul>
</template>

Casilla de verificación con un valor booleano

export class MyViewModel {
    agreeToTerms = false;
}

<template>
  <label><input type="radio" name="terms" model.bind="true" checked.bind="agreeToTerms" />Yes</label>
  <label><input type="radio" name="terms" model.bind="false" checked.bind="agreeToTerms" />No</label>
  <br><br>
  <strong>${agreeToTerms ? 'I agree' : 'I disagree'}</strong>
</template>

Enlace a elementos seleccionados #

Matriz de cadenas

Al seleccionar un valor en el menú desplegable de selección y proporcionar una matriz de cadenas, el valor seleccionado se vinculará a la propiedad de valor del elemento seleccionado como una cadena que podemos mostrar mediante la interpolación de cadenas.

export class MyViewModel {
    animals = [];
    favouriteAnimal = null;

    constructor() {
        this.animals = [
            'Cat',
            'Dog',
            'Fish',
            'Rabbit',
            'Tiger',
            'Bat'
        ];
    }
}

<template>
    ${favouriteAnimal}
    <select name="animals" value.bind="favouriteAnimal">
        <option repeat.for="animal of animals">${animal}</option>
    </select>
</template>

Matriz de objetos

A diferencia del ejemplo anterior, cuando se proporciona una matriz de objetos, cuando se selecciona un valor en un menú desplegable, el modelo vinculado a esa opción en particular es el objeto proporcionado.

export class MyViewModel {
    animals = [];
    favouriteAnimal = null;

    constructor() {
        this.animals = [
            {label: 'Cat', value: 99},
            {label: 'Dog', value: 493},
            {label: 'Fish', value: 934839200},
            {label: 'Rabbit', value: 8311},
            {label: 'Tiger', value: 22},
            {label: 'Bat', value: 3711}
        ];
    }
}

<template>
    <p>Favourite animal ID: ${favouriteAnimal.value}</p>
    <p>Favourite animal name: ${favouriteAnimal.label}</p>
    <select name="animals" value.bind="favouriteAnimal">
        <option repeat.for="animal of animals" model.bind="animal">${animal.label}</option>
    </select>
</template>

Estilos de encuadernación #

Enlace al atributo estilo nativo del navegador usando Aurelia. Si usa la interpolación de cadenas, debe usar el alias css para que el estilo funcione en Internet Explorer.

Cadena de estilo

export class MyViewModel {
  constructor() {
    this.styleString = 'color: #F2D3D6; background-color: #333';
  }
}

<template>
  <div style.bind="styleString"></div>
</template>

Objeto de estilo

export class MyViewModel {
  constructor() {
    this.styles = {color: '#F2D3D6', 'background-color': '#333'};
  }
}

<template>
  <div style.bind="styles"></div>
</template>

Interpolación de cadenas

Muy similar a la vinculación de cadenas anterior, esto le permite usar la interpolación de cadenas para vincular estilos en su lugar. Si alguno de los valores cambia, se actualizarán en la vista en consecuencia.

Nota: para compatibilidad con Internet Explorer, usamos el alias css para enlazar estilos. Esto garantiza que la interpolación de cadenas funcione en Internet Explorer.

export class MyViewModel {
    color = 'red';
    height = 350;
    width = 350;
}

<template>
    <div css="width: ${width}px; height: ${height}px; color:${color}">My Text</div>
</template>

Mostrar y ocultar condicionalmente un elemento HTML #

Cuando se usa show.bind, el elemento permanece en la página y está oculto o visible mediante el uso de display:none o display:block detrás de escena.

export class MyViewModel {
    isVisible = false;
}

<template>
    <div show.bind="isVisible"><strong>I can be both hidden or visible, but not at the same time</strong></div>
</template>

Agrega o elimina condicionalmente un elemento HTML #

A diferencia de show.bind cuando se usa if.bind, el elemento se eliminará de la página si el valor de enlace proporcionado es falso o se agregará a la página si el valor es verdadero.

export class MyViewModel {
    isVisible = false;
}

<template>
    <div if.bind="isVisible"><strong>If you can read this, I am still here.</strong></div>
</template>

Elementos agrupados

A veces puede haber una situación en la que desee agregar o eliminar un grupo completo de elementos a la vez. Para esto, podemos usar un elemento <plantilla> para mostrar u ocultar elementos adicionales sin la necesidad de un elemento de marcador de posición como un DIV.

export class MyViewModel {
    hasErrors = false;
    errorMessage = '';
}

<template if.bind="hasErrors">
    <i class="icon error"></i>
    ${errorMessage}
</template>
