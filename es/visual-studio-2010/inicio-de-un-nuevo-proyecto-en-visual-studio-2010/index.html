Inicio de un nuevo proyecto en Visual Studio 2010 #

Para iniciar un nuevo proyecto en Visual Studio 2010:

  • Haga clic en el menú Archivo
  • Seleccionar proyecto

A continuación, tenemos una lista de tipos de proyectos e idiomas disponibles:

  • Seleccione Idiomas disponibles desde el panel más a la izquierda [Abajo tenemos idiomas como Visual C#, VB, F#, etc.]
  • Una vez que seleccionamos los idiomas de las plantillas instaladas, tenemos una lista de tipos de proyectos/aplicaciones.
  • Seleccione los tipos de proyectos, asigne un nombre a su proyecto y haga clic en Aceptar.

Algunos ejemplos comunes de tipos de proyectos:

Aplicaciones de Windows Forms:

Proyecto de creación de aplicación con interfaz de usuario de Windows Forms.

Aplicaciones WPF:

Aplicación cliente de Windows Presentation Foundation.

Aplicaciones de consola:

Aplicaciones de línea de comandos. Sin interfaz gráfica de usuario (GUI).

ASP.NET:

Aplicaciones de interfaz de usuario web, es decir. aplicaciones web

Aplicaciones Silverlight:

Aplicaciones ricas de Internet usando Silverlight

Abrir un proyecto existente #

  • Abra Visual Studio y seleccione Menú Archivo -> Abrir -> Proyecto

  • A continuación, seleccione Archivo de solución de la carpeta del proyecto

  • Los archivos del proyecto se mostrarán en Solution Explorer.

Asociaciones de teclas predeterminadas de Microsoft Visual Studion 2010 Microsoft® Visual C#® #

Comandos de archivo

  • Archivo.NuevoProyecto :

CTRL + MAYÚS + N

Muestra el cuadro de diálogo Nuevo proyecto.

  • Archivo.AbrirProyecto:

CTRL + MAYÚS + O

Muestra el cuadro de diálogo Abrir proyecto, donde los proyectos existentes se pueden añadido a la solución.

  • Proyecto.AddClass:

MAYÚS + ALT + C

Muestra el cuadro de diálogo Agregar nuevo elemento y selecciona Plantilla de clase como predeterminado.

  • Proyecto.AddExistingItem:

MAYÚS + ALT + A

Muestra el cuadro de diálogo Agregar elemento existente, donde los archivos existentes pueden ser añadido al proyecto actual.

  • Proyecto.AddNewItem:

CTRL + MAYÚS + A

Muestra el cuadro de diálogo Agregar nuevo elemento, donde se puede agregar un nuevo archivo el proyecto actual.

Edición

  • Editar.OcultarSelección:

CTRL + M, CTRL + H

Contrae la selección actual en una región delimitada.

  • Edit.ToggleAllOutlining:

CTRL + M , L

Alterna todas las regiones de contorno colapsadas previamente entre colapsadas y estados expandidos.

  • Edit.ToggleOutliningExpansion:

CTRL + M , M

Alterna la región contraída actualmente seleccionada entre la región contraída y estado expandido.

  • Edit.StopOutlining:

CTRL + M , P

Elimina toda la información de esquema de todo el documento.

  • Editar.Selección de comentario:

CTRL + K , C o CTRL + E , C

Inserta // al principio de la línea actual o cada línea de la selección actual.

Editar.DescommentarSelección CTRL + K, U o CTRL + E, U Elimina // al principio de la línea actual o cada línea de la selección actual.

  • Editar.FormatearDocumento:

CTRL + K, D o CTRL + E, D

Da formato al documento actual de acuerdo con la sangría y el código configuración de formato especificada en el panel Formato en: Herramientas | Opciones | Editor de texto | C#.

  • Editar.Selección de Formato:

CTRL + K, F o CTRL + E, F

Da formato a la selección actual de acuerdo con la sangría y el código configuración de formato especificada en el panel Formato en Herramientas | Opciones | Editor de texto | C#.

  • Editar.Insertar Fragmento:

CTRL + K, X

Muestra el selector de fragmentos de código. El fragmento de código seleccionado se insertado en la posición del símbolo de intercalación.

Editar.EnvolventeCon:

CTRL + K, S

Muestra el selector de fragmentos de código. El fragmento de código seleccionado se estar envuelto alrededor del texto seleccionado.

  • Editar.InvokeSnippetFromShortcut:

TABULACIÓN

Inserta el fragmento de código expandido del nombre del acceso directo.

  • Editar.CicloClipboardRing:

CTRL +MAYÚS +V

Pega texto del anillo del Portapapeles en la ubicación del símbolo de intercalación en el archivo. El uso posterior de la tecla de método abreviado itera a través de los elementos en el Anillo portapapeles.

  • Editar.Reemplazar:

CTRL + H

Muestra las opciones de reemplazo en la pestaña Reemplazo rápido de la función Buscar y Reemplazar cuadro de diálogo.

  • Editar.ReemplazarEnArchivos:

CTRL + MAYÚS + H

Muestra las opciones de reemplazo en la pestaña Reemplazar en archivos de Buscar y Reemplazar cuadro de diálogo.

Comandos de ventana:

  • Ver.ClassView:

CTRL + W, C

Muestra la ventana Vista de clase.

  • Ver.Ventana de definición de código:

CTRL + W, D

Muestra la ventana de definición de código.

  • Ver.Lista de errores:

CTRL + W, E

Muestra la ventana Lista de errores.

  • Ver.ObjectBrowser:

CTRL + W, J

Muestra el Explorador de objetos.

  • Ver.Salida:

CTRL + W, O

Muestra la ventana Salida, donde los mensajes de estado se pueden ver en tiempo de ejecución.

  • Ver.Ventana de Propiedades:

CTRL + W, P

Muestra la ventana Propiedades, que enumera las propiedades en tiempo de diseño y eventos para el elemento seleccionado actualmente.

  • Ver.Explorador de soluciones:

CTRL + W, S

Muestra el Explorador de soluciones, que enumera los proyectos y archivos en el solución actual.

  • Ver.Lista de tareas:

CTRL + W, T

Muestra la ventana Lista de tareas, que muestra tareas personalizadas, comentarios, accesos directos, advertencias y mensajes de error.

  • Ver.Caja de herramientas:

CTRL + W, X

Muestra la Caja de herramientas, que contiene controles que se pueden incluir o utilizado con su código.

  • Ver.ServerExplorer:

CTRL + W, L

Muestra Server Explorer, que le permite ver y manipular la base de datos servidores, registros de eventos, colas de mensajes, servicios web y otros servicios del sistema.

  • Ventana.CerrarVentanaHerramienta:

MAYÚS + ESC

Cierra la ventana de herramientas actual.

  • Datos.MostrarFuentesDeDatos:

MAYÚS + ALT + D

Muestra la ventana Orígenes de datos.

  • Ventana.CerrarVentanaDocumento:

CTRL + F4

Cierra la pestaña actual

Comando de depuración:

  • Depuración.Autos

CTRL+D, A

Muestra la ventana Automático, que muestra las variables utilizadas en el la línea de código actual y la línea de código anterior.

  • Depuración.Pila de llamadas:

CTRL+D, C

Muestra la ventana Pila de llamadas, que muestra una lista de todas las métodos o marcos de pila para el hilo de ejecución actual.

  • Depuración.Inmediata:

CTRL+D, I

Muestra la ventana Inmediato, donde se pueden evaluar las expresiones.

  • Depuración.Locales:

CTRL+D, L

Muestra la ventana Locales, que muestra las variables locales y sus valores para cada método en el marco de pila actual.

  • Depuración.Reloj rápido:

CTRL+D, Q

Muestra el cuadro de diálogo QuickWatch que tiene el valor actual del expresión seleccionada.

  • Depuración.Inicio:

F5

Inicia la aplicación bajo el depurador según la configuración desde el proyecto de inicio. Cuando está en modo Pausa, al invocar este comando ejecutará la aplicación hasta el siguiente punto de interrupción.

  • Depurar.Comenzar sin depurar:

CTRL+F5

Inicia la aplicación sin invocar al depurador.

  • Depuración.Paso a paso:

F11

Ejecuta el código una declaración a la vez, siguiendo la ejecución en llamadas a métodos.

  • Depuración.Paso a lo específico:

MAYÚS+ALT+F11

Solicita seleccionar en qué subexpresión entrar, independientemente de la configuración de opciones de herramientas para omitir propiedades y operadores.

  • Depuración.Paso a paso:

MAYÚS+F11

Ejecuta las líneas restantes de un método en el que el actual se encuentra el punto de ejecución.

  • Depuración.Paso a paso:

F10

Ejecuta la siguiente línea de código, pero no sigue la ejecución hasta el final. cualquier método llama.

  • Depurar.Detener la depuración:

MAYÚS+F5

Detiene la ejecución de la aplicación actual bajo el depurador.

  • Depuración.ToggleBreakpoint:

F9

Establece o elimina un punto de interrupción en la línea actual.

  • Depuración.Ver:

CTRL+D, W

Muestra la ventana Watch, que muestra los valores de los variables o ver expresiones.

  • Depuración.EnableBreakpoint:

CTRL+F9

Alterna el punto de interrupción entre deshabilitado y habilitado.

  • Prueba.EjecutarTodasLasPruebasEnSolución:

CTRL+R, A

Ejecuta todas las pruebas unitarias en la solución.

Comandos de construcción:

  • Construir.ConstruirSolución:

F6 o CTRL+MAYÚS+B

Construye todos los proyectos en la solución.

  • Build.BuildSelection:

MAYÚS+F6

Construye el proyecto seleccionado y sus dependencias.
