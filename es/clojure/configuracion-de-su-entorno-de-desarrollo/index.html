##Emacs Para configurar Emacs para trabajar con Clojure, instale el paquete clojure-mode y cider desde melpa:

M-x package-install [RET] clojure-mode [RET]
M-x package-install [RET] cider [RET]

Ahora, cuando abra un archivo .clj, ejecute M-x cider-jack-in para conectarse a un REPL. Alternativamente, puede usar C-u M-x (cider-jack-in) para especificar el nombre de un proyecto lein o boot, sin tener que visitar ningún archivo en él. Ahora debería poder evaluar expresiones en su archivo usando C-x C-e.

Editar código en lenguajes similares a ceceo es mucho más cómodo con un complemento de edición con reconocimiento de paren. Emacs tiene varias buenas opciones.

  • paredit Un modo de edición Lisp clásico que tiene una curva de aprendizaje más pronunciada, pero proporciona mucha potencia una vez que se domina.

    M-x package-install [RET] paredit [RET]

  • smartparens Un proyecto más nuevo con objetivos y usos similares a paredit, pero que también proporciona capacidades reducidas con lenguajes que no son Lisp.

    M-x package-install [RET] smartparens [RET]

  • parinfer Un modo de edición de Lisp mucho más simple que funciona principalmente a través de la inferencia del anidamiento de paréntesis adecuado a partir de la sangría.

    Installation is more involved, see the Github page for parinfer-mode for setup instructions.

Para habilitar paredit en clojure-mode:

(add-hook 'clojure-mode-hook #'paredit-mode)

Para habilitar smartparens en clojure-mode:

(add-hook 'clojure-mode-hook #'smartparens-strict-mode)

IntelliJ IDEA + cursiva #

Descargar e instalar la última versión de IDEA.

Descargar e instalar la última versión del complemento Cursive.

Después de reiniciar IDEA, Cursive debería estar funcionando de inmediato. Siga la guía del usuario para ajustar la apariencia, las combinaciones de teclas, el estilo del código, etc.

Nota: Al igual que IntelliJ, Cursive es un producto comercial, con un período de evaluación de 30 días. A diferencia de IntelliJ, no tiene una edición comunitaria. Las licencias no comerciales gratuitas están disponibles para personas para uso no comercial, incluida la piratería personal, el código abierto y el trabajo de los estudiantes. La licencia es válida por 6 meses y se puede renovar.

Átomo #

Instale Atom para su distribución aquí.

Después de eso, ejecute los siguientes comandos desde una terminal:

apm install parinfer
apm install language-clojure
apm install proto-repl

Spacemacs + SIDRA #

Spacemacs es una distribución de emacs que viene con muchos paquetes preconfigurados y fáciles de instalar. Además, es muy amigable para aquellos que están familiarizados con el estilo de edición vim. Spacemacs proporciona una [capa Clojure basada en CIDER][CIDER].

Para instalarlo y configurarlo para usarlo con Clojure, primero instale emacs. Luego haga una copia de seguridad de sus configuraciones anteriores:

$ mv ~/.emacs.d ~/.emacs.d.backup

Luego clona el repositorio de spacemacs:

$ git clone https://github.com/syl20bnr/spacemacs ~/.emacs.d

Ahora, abra emacs. Le hará algunas preguntas sobre sus preferencias. Luego descarga más paquetes y configura sus emacs. Después de que Spacemacs esté instalado, estará listo para agregar soporte Clojure. Presione SPC f e d para abrir su archivo .spacemacs para la configuración. Busque dotspacemacs-configuration-layers en el archivo, debajo hay un paréntesis abierto. En cualquier lugar entre los paréntesis en una nueva línea, escriba clojure.

(defun dotspacemacs/layers ()
  (setq-default
   ;; ...
   dotspacemacs-configuration-layers
   '(clojure
     ;; ...
    )
   ;; ...
))

Presione SPC f e R para guardar e instalar la capa clojure. Ahora, en cualquier archivo .clj, si presiona , s i, spacemacs intentará generar una nueva conexión clojure REPL a su proyecto, y si tiene éxito, se mostrará en la barra de estado, que luego puede presionar , s s para abrir un nuevo búfer REPL para evaluar sus códigos.

Para obtener más información sobre spacemacs y sidra, comuníquese con su documentación. Documentos de Spacemacs, Documentos de Cider

Vim #

Instale los siguientes complementos usando su administrador de complementos favorito:

  1. fireplace.vim: soporte Clojure REPL
  2. vim-sexp: Para domar esos abrazos alrededor de tus llamadas de función
  3. vim-sexp-mappings-for-regular-people: Asignaciones de sexo modificadas que son un poco más fáciles de soportar
  4. vim-surround: elimine, cambie y agregue fácilmente “alrededores” en pares
  5. salve.vim: compatibilidad con Static Vim para Leiningen y Boot.
  6. rainbow_parentheses.vim: Mejores paréntesis de arcoíris

y también se relacionan con el resaltado de sintaxis, la terminación omni, el resaltado avanzado, etc.:

  1. vim-clojure-static (si tiene un vim anterior a 7.3.803, las versiones más nuevas se envían con esto)
  2. [punto culminante de vim-clojure] (https://github.com/guns/vim-clojure-highlight)

Otras opciones en lugar de vim-sexp incluyen paredit.vim y [vim-parinfer](https://github.com/bhurlow/vim- parinferir).

Mesa ligera #

Light Table es un buen editor para aprender, experimentar y ejecutar proyectos de Clojure.
También puede ejecutar proyectos lein/boot abriendo el archivo project.clj. Cargará todas las dependencias del proyecto.

Admite evaluación en línea, complementos y mucho más, por lo que no es necesario agregar declaraciones de impresión y verificar la salida en la consola. Puede ejecutar líneas individuales o codificar bloques presionando ctrl + enter. Para ejecutar un código parcial, seleccione el código y presione ctrl + enter. consulte la siguiente captura de pantalla para saber cómo puede usar Light Table para aprender y experimentar con el código Clojure.

Los binarios preconstruidos de Light Table se pueden encontrar aquí. No se requiere ninguna configuración adicional.

Light Table puede ubicar automáticamente su proyecto de Leiningen y evaluar su código. Si no tiene instalado Leiningen, instálelo siguiendo las instrucciones aquí.

Documentación: [docs.lighttable.com][4]
