Comentar cambios #

Durante una revisión, el usuario tiene la posibilidad de agregar comentarios para explicar su punto de vista. Hay varios tipos de comentarios.

  1. Comentar un archivo: el símbolo amarillo permite al usuario agregar comentarios en todo el archivo

  1. Comentario de línea: haga clic en el número de línea para comentarlo

  2. Selección de palabras: la última opción es comentar palabras seleccionándolas y presionando el teclado ‘c’.

Atajo

Presione ‘Esc’ para cancelar la edición de comentarios

Presiona ‘Ctrl + s’ para guardar el borrador del comentario
