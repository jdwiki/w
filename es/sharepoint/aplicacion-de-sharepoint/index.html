SharePoint 2013: acceda a los datos del servicio de perfil de usuario mediante JSOM en SharePoint 2013 ​

En este artículo, aprenderemos a administrar o acceder a la aplicación del servicio de perfil de usuario (UPS) usando JSOM (modelo de objetos de JavaScript) y crear una aplicación básica. Antes de comenzar, veamos primero la terminología básica de UPS.

Perfil de Usuario – Tiene toda la información de las personas de una organización de manera organizada. Muestra todas las propiedades como AccountName, FirstName, LastName, WorkEmail, etc. relacionadas con un usuario.

Aplicación de servicio de perfil de usuario: se considera una ubicación centralizada para almacenar todos los perfiles de usuario y también permite a los administradores configurar o administrar perfiles, sincronización de perfiles, Mi sitio, etiquetas sociales, etc. También puede extraer información de servicios de directorio como Active Directory.

Mi sitio: un sitio personalizado para que los usuarios individuales administren su información y almacenen documentos, enlaces, etc. Proporciona ricas funciones sociales y de redes al permitir que los usuarios compartan información sobre ellos mismos o sus actividades. Se puede acceder a Mi sitio haciendo clic en Nombre de usuario en la esquina superior derecha de la página de SharePoint.

Administrar y acceder a los datos del perfil de usuario

Como vamos a trabajar con JSOM, solo podemos realizar operaciones de ‘Lectura’ con la excepción de que la imagen de perfil se puede cambiar con JSOM (o CSOM o REST)

*El código del lado del servidor permite leer/escribir ambas operaciones.

Recuperar las propiedades del perfil de usuario mediante JSOM

Vamos a crear una aplicación alojada de SharePoint y recuperar la información del usuario en esa aplicación:

Inicie Visual Studio 2013 y seleccione “Aplicación para SharePoint 2013” en Nuevo proyecto. Después de seleccionar el tipo de proyecto anterior, se le presenta una ventana para conectarse al sitio de SharePoint y seleccionar el tipo de aplicación que se implementará (vea la captura de pantalla a continuación) Aquí proporcioné mu URL del sitio del desarrollador de SharePoint Online y seleccioné la aplicación alojada de SharePoint. Haga clic en Finalizar.

3.) Después de crear el proyecto, verá un conjunto de carpetas/archivadores agregados en el Explorador de soluciones agregado de manera predeterminada al proyecto.

4.) Si abre la página “Default.aspx”, encontrará algunas bibliotecas de JavaScript ya agregadas a la página.
