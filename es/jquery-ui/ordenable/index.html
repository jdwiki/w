Sintaxis #

  • $("#sortable").sortable({ /*Opciones aquí*/ }); //Inicializar ordenable

  • $("#sortable").sortable(“option”, “option_name”, option_value); //Establecer opción fuera del inicializador

  • var value = $("#sortable").sortable(“option”, “option_name”); //Obtiene el valor de una opción

Parámetros #

Parámetro Descripción
Opciones  
agregar a (jQuery, Elemento, Selector, Cadena) [Predeterminado: “principal”] El elemento al que se agrega el ayudante
eje (Cadena) [Predeterminado: falso] Las direcciones en las que se puede arrastrar el elemento (x o y)
cancelar (Selector) [Predeterminado: “entrada, área de texto, botón, selección, opción”] No comienza a ordenar si comienza en un elemento que coincide con el selector
clases (Objeto) [Predeterminado: {}] Especifique clases adicionales para agregar a los elementos clasificables cuando se agreguen las clases estructurales. ({ui-sortable-helper: clase_personalizada})
conectar con (Selector) [Predeterminado: falso] Permite arrastrar elementos de un ordenable a otro
contención (Elemento, Selector, Cadena) [Predeterminado: falso] El elemento al que están restringidos los elementos
cursor (String) [Predeterminado: “auto”] Define el tipo de cursor que se mostrará al ordenar
cursorEn (Objeto) [Predeterminado: falso] Define la posición desde la que parece que se está moviendo el ayudante
discapacitado (Booleano) [Predeterminado: falso] Deshabilita la clasificación si es verdadero
soltar en vacío (Booleano) [Predeterminado: verdadero] Si los elementos falsos de este ordenable no se pueden colocar en ordenables vacíos
forceHelperSize (Booleano) [Predeterminado: falso] Obliga al asistente a tener un tamaño
forcePlaceholderSize (Booleano) [Predeterminado: falso] Obliga al marcador de posición a tener un tamaño
rejilla (Array) [Predeterminado: falso] Define una cuadrícula para ajustar el ayudante a ([ x,y ])
mango (Selector, Elemento) [Predeterminado: falso] Define los elementos en los que puede comenzar la clasificación. Opuesto a cancelar
ayudante (Cadena, Función) [Predeterminado: “original”] Cadena “original” o “clon”, o función que devuelve el elemento que se utilizará como auxiliar.
articulos (Selector) [Predeterminado: “> *”] Define los elementos que deben ordenarse
opacidad (Número 0.01 a 1) [Predeterminado: falso] Define la opacidad para el ayudante
marcador de posición (Cadena) [Predeterminado: falso] Define una clase o clases que se aplicarán al marcador de posición
revertir (Booleano, Número) [Predeterminado: falso] El tiempo que tarda el ayudante en deslizarse a su nueva posición
pergamino (Booleano) [Predeterminado: verdadero] Si se desplazará cuando esté en los bordes de la página
desplazamientoSensibilidad (Número) [Predeterminado: 20] Define qué tan cerca del borde de la página debe estar el cursor para comenzar a desplazarse
velocidad de desplazamiento (Número) [Predeterminado: 20] La velocidad a la que se desplaza
tolerancia (Cadena) [Predeterminado: “intersección”] Define qué modo usar al calcular cuándo un elemento está sobre otro (“intersección” o “puntero”)
índicez (Entero) [Predeterminado: 1000] Define el índice z del ayudante al ordenar
Métodos  
cancelar() Cancela la ordenación actual y devuelve los elementos a su posición antes de que comenzara la ordenación
destruir() Elimina la funcionalidad clasificable y devuelve el elemento a su estado previo a la inicialización
deshabilitar() Deshabilita el ordenable
habilitar () Habilita el ordenable
instancia() Devuelve el objeto de instancia ordenable
opción() Obtiene pares de valores clave de todas las opciones para el sortable
opción(Cadena) Obtiene el valor de una opción
opción(Cadena, Cualquiera) Establece el valor de la opción especificada por String
opción(Objeto) Establece una o más opciones con el objeto como pares de opciones de valor clave
refrescar() Actualiza las opciones ordenables recargando todos los elementos ordenables. Esto hace que se reconozcan nuevos artículos
actualizarPosiciones() Refresca las posiciones almacenadas en caché de los elementos ordenables
serializar(Objeto) Serializa los identificadores de elementos (de forma predeterminada) en una cadena que se puede enviar o agregar a una URL. Opciones de objeto: {clave: establece la clave en la cadena serializada, atributo: [“ID” predeterminado] establece el atributo para mirar, expresión :[Predeterminado: “/(.+)-=_/”] expresión regular para dividir el atributo en pares de valores clave}
aArray(Objeto) Serializa la identificación de los elementos ordenables en una matriz. El objeto puede contener un atributo de parámetro que tiene el atributo para poner en la matriz por defecto es id
artilugio() Devuelve un objeto jQuery del elemento ordenable
Eventos  
activar (evento, ui) Activado cuando la lista está conectada, cada lista conectada al inicio de arrastre la recibe
beforeStop(evento, ui) Se activa antes de que la clasificación se detenga cuando el ayudante se ha desplazado a la misma posición que el marcador de posición
cambio (evento, ui) Se activa cuando los elementos cambian de posición, es decir. cuando se mueve el marcador de posición
crear (evento, interfaz de usuario) Se activa cuando se crea ordenable
desactivar (evento, ui) Se activa cuando se detiene la clasificación. Esto también va a todas las listas conectadas
fuera (evento, ui) Se activa cuando el elemento se saca de la lista ordenable
terminado (evento, ui) Se activa cuando el elemento se mueve a una lista ordenable
recibir (evento, ui) Se activa cuando un elemento de una lista conectada se coloca en otra. Target es la lista de recepción
remove(evento, ui) Se activa cuando un elemento de una lista conectada se coloca en otra. El objetivo es la lista de donaciones
sort(evento, ui) Activado durante la clasificación
inicio (evento, ui) Activado cuando comienza la clasificación
parada (evento, ui) Activado cuando se detiene la clasificación
update(evento, ui) Se activa cuando se detiene la clasificación y se actualiza la posición del DOM

Cuadrícula clasificable con diseño flexible #

Esto usó el diseño flexible con ordenable para crear una cuadrícula de cuadros receptivos que se pueden mover arrastrando y soltando.

HTML

<div id="sortable">
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
  <div>5</div>
</div>

JS

$(function(){
    $('#sortable').sortable({
        //pass all options in here
    });
});

CSS

#sortable{
    width: 500px;
    display: flex;
    flex-wrap: wrap;
}
#sortable div {
    margin: 10px;
    background-color: #f00;
    flex-basis: 100px;
    height: 100px;
}

Elementos estacionarios al arrastrar #

Este ejemplo usa una clase en el marcador de posición para convertirlo en una línea y hacer que no ocupe espacio.

HTML

<div id="sortable">
    <div>1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
</div>

JS

$("#sortable").sortable({
    placeholder: 'placeholder',
    helper: 'clone',
    start: function(event, ui){
        ui.item.show();
    }
});

CSS

#sortable div{
  background-color: #f00;
  width: 50px;
  height: 50px;
  margin: 10px;
  padding: 0px;
}
#sortable div.placeholder{
  height: 4px;
  margin: -7px 10px;
}

Sortable - Animar la reversión del elemento no aceptado #

Ejemplo de trabajo: https://jsfiddle.net/Twisty/4f5yh3pa/7/

La cancelación y reversión de una clasificación no está bien documentada. Las ayudas muestran cómo se puede cancelar condicionalmente el movimiento de un elemento de una lista a otra lista conectada. por defecto, esto no está animado por ordenable, este ejemplo incluye una animación.

Resultado: la lista n.º 2 solo acepta elementos que tengan una clase de aceptable. Ambas listas se pueden ordenar naturalmente de otra manera.

HTML

<div class="ui-widget">
  <ul id="sortable1" class="connectedSortable">
    <li class="ui-state-default acceptable">Item 1</li>
    <li class="ui-state-default">Item 2</li>
    <li class="ui-state-default">Item 3</li>
    <li class="ui-state-default">Item 4</li>
    <li class="ui-state-default">Item 5</li>
  </ul>
  <ul id="sortable2" class="connectedSortable">
    <li class="ui-state-default">Item 6</li>
    <li class="ui-state-default acceptable">Item 7</li>
  </ul>
</div>

CSS

.ui-widget {
  position: relative;
}

.connectedSortable {
  border: 1px solid #eee;
  width: 142px;
  min-height: 20px;
  list-style-type: none;
  margin: 0;
  padding: 5px 0 0 0;
  float: left;
  margin-right: 10px;
}

#sortable1 {
  background: #fff;
}

#sortable2 {
  background: #999;
}

.connectedSortable li {
  margin: 0 5px 5px 5px;
  padding: 5px;
  font-size: 1.2em;
  width: 120px;
}

JavaScript

$(function() {
  $(".connectedSortable").sortable({
    connectWith: ".connectedSortable",
    receive: function(e, ui) {
      var $self = $(this);
      var $item = ui.item;
      var $sender = ui.sender;
      // Restrict condition to only one specific list if desired
      if ($(e.target).attr("id") == "sortable2") {
        if ($item.hasClass("acceptable")) {
          // Item Accepted
          console.log($self.attr("id") + " accepted item from: #" + $sender.attr("id") + " > " + $item.text());
        } else {
          // Item Rejected
          console.log($self.attr("id") + " rejected item from: #" + $sender.attr("id") + " > " + $item.text());
          // Animate the return of the items position
          $item.css("position", "absolute").animate(ui.originalPosition, "slow", function() {
            // Return the items position control to it's parent
            $item.css("position", "inherit");
            // Cancel the sortable action to return it to it's origin
            $sender.sortable("cancel");
          });
        }
      }
    }
  }).disableSelection();
});

Ejemplo sencillo #

Tome cualquier lista y agregue un identificador al envoltorio externo (ul, div)

<ul id="sortable">
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
<li>Item 4</li>
</ul>

En tu jquery:

$(function(){
    $('#sortable').sortable({
        //pass all options in here
    });
});

Esto permitirá que todos los li en el envoltorio #sortable se arrastren y suelten en la lista
