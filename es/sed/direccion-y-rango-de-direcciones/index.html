Dirección y rango de direcciones

  • Líneas que coinciden con el patrón de expresión regular

Los comandos Sed se pueden especificar para que actúen solo en ciertas líneas usando direcciones o intervalos de direcciones.

Líneas que coinciden con el patrón de expresión regular #

$ cat ip.txt 
address
range
substitution
pattern
sample
Add Sub Mul Div

  • Líneas que coinciden con un patrón

    $ sed '/add/d' ip.txt 
range
substitution
pattern
sample
Add Sub Mul Div

$ sed -n '/t/p' ip.txt 
substitution
pattern

$ sed -n '/[A-Z]/ s| |/|gp' ip.txt 
Add/Sub/Mul/Div

  • Gama de patrones

    $ sed -n '/add/,/sub/p' ip.txt 
address
range
substitution

$ sed -n '/a/,/e/p' ip.txt 
address
range
pattern
sample

Nota

  • En el segundo ejemplo, coincidió con dos rangos: líneas 1,2 y líneas 4,5
  • Consulte [Uso de diferentes delimitadores](https://www.wikiod.com/es/sed/expresiones-regulares#Usando diferentes delimitadores different-delimiters#t=201611031024036829908) sobre cómo usar otros caracteres en lugar de / para especificar el patrón

  • Coincidencia que no distingue entre mayúsculas y minúsculas

    $ sed -n '/add/Ip' ip.txt 
address
Add Sub Mul Div

$ sed -n '/add/I,/sub/p' ip.txt 
address
range
substitution
Add Sub Mul Div

Línea específica #

$ cat ip.txt
address
range
substitution
pattern
sample

  • Línea N

    $ sed -n '2p' ip.txt 
range

$ sed '3d' ip.txt 
address
range
pattern
sample

  • Última línea

    $ sed -n '$p' ip.txt 
sample

Rango específico de líneas #

$ cat ip.txt 
address
range
substitution
pattern
sample

  • El rango especificado incluye esos números de línea

    $ sed -n '2,4p' ip.txt 
range
substitution
pattern

  • $ se puede usar para especificar la última línea. Se puede usar espacio entre la dirección y el comando para mayor claridad.

    $ sed -n '3,$ s/[aeiou]//gp' ip.txt 
sbstttn
pttrn
smpl

  • iésima línea a i+jésima línea

    $ sed '2,+2d' ip.txt 
address
sample

  • iésima línea y i+j, i+2j, i+3j, etc.

    $ sed -n '1~2p' ip.txt 
address
substitution
sample

Especificar el rango usando tanto el número como el patrón #

$ cat ip.txt 
address
range
substitution
pattern
sample
Add Sub Mul Div

  • Patrón de coincidencia de número de línea a línea

    $ sed -n '2,/pat/p' ip.txt 
range
substitution
pattern

  • Patrón de coincidencia de línea con número de línea

    $ sed '/pat/,$d' ip.txt 
address
range
substitution

  • Patrón de coincidencia de líneas más el número de líneas que lo siguen

    $ sed -n '/add/I,+1p' ip.txt 
address
range
Add Sub Mul Div

  • 0 se puede usar como número de línea inicial para indicar el final del rango cuando el patrón coincide con la primera línea de entrada

    $ sed -n '0,/r/p' ip.txt 
address

$ sed -n '1,/r/p' ip.txt 
address
range

$ sed -n '0,/u/p' ip.txt 
address
range
substitution

Intervalo de direcciones negadas #

$ cat ip.txt 
address
range
substitution
1234
search pattern
sample
Add Sub Mul Div

  • Eliminar líneas que no sean la dirección especificada

    $ sed '/[0-9]/!d' ip.txt 
1234

$ sed -n '/[0-9]/p' ip.txt 
1234

$ sed '$!d' ip.txt 
Add Sub Mul Div

$ sed -n '$p' ip.txt 
Add Sub Mul Div

  • Buscar y reemplazar en líneas que no coincidan con un patrón

    $ sed '/ /! s/^/#/' ip.txt 
#address
#range
#substitution
#1234
search pattern
#sample
Add Sub Mul Div

$ sed '/add/,/sub/! s/[aeiou]//gi' ip.txt 
address
range
substitution
1234
srch pttrn
smpl
