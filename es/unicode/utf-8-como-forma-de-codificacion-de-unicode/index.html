¿Qué es UTF-8?

UTF-8 es una codificación, que es de longitud variable y utiliza unidades de código de 8 bits, por eso UTF-8. En Internet, UTF-8 es la codificación dominante (antes de 2008, ASCII, que también puede manejar cualquier punto de código Unicode).

¿UTF-8 es lo mismo que Unicode?

“Unicode” no es una codificación, es un conjunto de caracteres codificados, es decir, un conjunto de caracteres y una asignación entre los caracteres y los puntos de código entero que los representan. Pero mucha documentación lo usa para referirse a codificaciones. En Windows, por ejemplo, el término Unicode se usa para referirse a UTF-16.

UTF-8 es solo una de las formas de codificar Unicode y, como codificación, convierte las secuencias de bytes en secuencias de caracteres y viceversa. UTF-16 y -32 son otros formatos de transformación Unicode.

BOM de UTF-8

Los tres pueden tener marcas de orden de bytes específicas, que al ser un número mágico señala varias cosas importantes para un programa (por ejemplo, Notepad ++), por ejemplo, el hecho de que el flujo de texto importado es Unicode; también ayuda a detectar el arte de Unicode utilizado para esta transmisión. Sin embargo, el consorcio Unicode recomienda almacenar UTF-8 sin ninguna firma. Algunos programas, por ejemplo, el compilador gcc, se quejan si un archivo contiene la firma UTF-8. Muchos programas de Windows, por otro lado, usan la firma. Y tratar de detectar la codificación de un flujo de bytes no siempre funciona.

Cómo verificar si su proyecto tiene codificación UTF-8 o no

UTF-8 aún no es universal, y los ingenieros de software y los científicos de datos a menudo enfrentan problemas de codificación de flujos de texto. A veces, se supone que se debe usar UTF-8 en el proyecto, sin embargo, se usa otro ecndoing. Hay varias herramientas para detectar la codificación del archivo:

  • Algunas herramientas CMD, como la herramienta de línea de comandos de Linux ‘archivo’ o powershell;
  • Paquete Python “chardet”
  • Notepad ++ como quizás la herramienta más popular para la verificación manual.

Cómo cambiar la codificación predeterminada del servidor a UTF-8 #

A veces, los usuarios de otras regiones que no son de habla inglesa tienen problemas con la codificación mientras, por ejemplo, programan un proyecto php. Puede ser que el servidor tenga otra codificación que UTF-8, y si alguien quiere crear un proyecto php en UTF-8 en este servidor, su texto podría mostrarse incorrecto.

Ejemplo: puede ser que la codificación predeterminada de su servidor sea Windows-1251; entonces, debe eliminar AddDefaultCharset windows-1251 del archivo del servidor .htaccess y escribir AddDefaultCharset utf-8.

Para verificar qué codificación tiene su servidor, no configure la etiqueta <META charset> y active "detección automática de codificación" en su navegador.

Guardar un archivo de Excel en UTF-8 #

Excel -> Guardar como -> Guardar como tipo -> “Valor separado por comas (*.csv)” Y Herramientas (a la izquierda del botón Guardar) -> Opciones web -> Codificación -> Guardar este documento como -> Unicode (UTF-8 )

Cómo convertir una matriz de bytes de datos UTF-8 en una cadena Unicode en Python #

def make_unicode(data):
    if type(data) != unicode:
        data =  data.decode('utf-8')
        return data
    else:
        return data
