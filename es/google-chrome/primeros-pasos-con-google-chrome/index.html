Instalación o configuración #

Instala Chrome considerando primero qué versión usar, relevante si vas a hacer desarrollo web:

Google Chrome lanza versiones principales aproximadamente cada 6 semanas y brinda acceso a múltiples versiones del producto a medida que se estabilizan. Este proceso se realiza en una serie de canales.

  • Canal estable es la versión lanzada para todos los usuarios, es la experiencia de instalación predeterminada. Recibe actualizaciones de seguridad según sea necesario.
  • Beta Channel es donde se estabiliza la próxima versión principal durante aproximadamente 1 mes antes de pasar a Stable. Se actualiza aproximadamente semanalmente.
  • Dev Channel está muy cerca del código más reciente, con pruebas mínimas. Se actualiza 1 o 2 veces por semana.

Una instalación de escritorio de Chrome se puede mover de Estable a Beta o Dev en las preferencias de Chrome.

La compilación Canary se somete a pruebas mínimas y se publica a diario. Brinda acceso al código más reciente y se puede instalar al mismo tiempo que las versiones estable, beta y de desarrollo de Chrome.

Instálalo siguiendo los enlaces a tu elección de Estable (y luego modificándolo a Beta o Dev) o Canario. En Android e iOS, utilice la respectiva Play/App Store.

Pasos para instalar:

  1. Descarga de Chrome para PC/Mac/Linux

  2. Haga clic en “Descargar Chrome”. Esto abrirá la ventana Términos de servicio.

  3. Determina si quieres que Chrome sea tu navegador predeterminado.

    • If you set it as the default browser, it will open whenever a link for a webpage is clicked in another program, such as email.

    • You can opt to send usage data back to Google by checking the box labeled “Help make Google Chrome better…” This will send back crash reports, preferences and button clicks. It does not send any personal information or track websites.

  4. Haga clic en “Aceptar e instalar” después de leer los Términos de servicio. El instalador se iniciará y tendrás Google Chrome instalado cuando haya terminado.

Extensiones #

Google Chrome admite extensiones que mejoran la forma en que funciona el navegador. Pueden agregar funcionalidad a las páginas web o a la interfaz de usuario del navegador. Cualquier desarrollador puede crear una extensión y publicarla en Chrome Web Store.
