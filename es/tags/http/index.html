HTTP

Cómo manejar el código de respuesta HTTP AJAX

Además de .done, .fail y .always prometen devoluciones de llamada, que se activan en función de si la solicitud fue exitosa o no, existe la opción de activar una función cuando el servidor devuelve un código de estado HTTP específico. Esto se puede hacer usando el parámetro statusCode.
