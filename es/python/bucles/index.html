Como una de las funciones más básicas en la programación, los bucles son una pieza importante para casi todos los lenguajes de programación. Los bucles permiten a los desarrolladores configurar ciertas partes de su código para que se repitan a través de una serie de bucles que se conocen como iteraciones. Este tema cubre el uso de múltiples tipos de bucles y aplicaciones de bucles en Python.

Sintaxis #

  • while <expresión booleana>:
  • para <variable> en <iterable>:
  • para <variable> en rango(<número>):
  • para <variable> en el rango (<número_inicial>, <número_final>):
  • para <variable> en rango(<número_inicial>, <número_final>, <tamaño_paso>):
  • para i, <variable> en enumerate(<iterable>): # con índice i
  • para <variable1>, <variable2> en zip(<iterable1>, <iterable2>):

Parámetros #

Parámetro Detalles
expresión booleana expresión que se puede evaluar en un contexto booleano, p. x < 10
variable nombre de variable para el elemento actual de iterable
iterable cualquier cosa que implemente iteraciones

Romper y continuar en bucles #

instrucción romper #

Cuando una instrucción break se ejecuta dentro de un bucle, el flujo de control “se sale” del bucle inmediatamente:

i = 0
while i < 7:
    print(i)
    if i == 4:
        print("Breaking from loop")
        break
    i += 1

El bucle condicional no se evaluará después de que se ejecute la instrucción break. Tenga en cuenta que las declaraciones break solo se permiten dentro de bucles, sintácticamente. Una declaración break dentro de una función no se puede usar para terminar bucles que llamaron a esa función.

Al ejecutar lo siguiente, se imprimen todos los dígitos hasta el número ‘4’ cuando se cumple la instrucción ‘romper’ y el ciclo se detiene:

0
1
2
3
4
Breaking from loop

Las declaraciones break también se pueden usar dentro de bucles for, la otra construcción de bucle proporcionada por Python:

for i in (0, 1, 2, 3, 4):
    print(i)
    if i == 2:
        break

Ejecutar este ciclo ahora imprime:

0
1
2

Tenga en cuenta que 3 y 4 no se imprimen porque el bucle ha terminado.

Si un bucle tiene una cláusula else, no se ejecuta cuando el bucle termina mediante una sentencia break.

instrucción continuar #

Una declaración continuar saltará a la siguiente iteración del ciclo sin pasar por el resto del bloque actual pero continuando el ciclo. Al igual que con break, continue solo puede aparecer dentro de bucles:

for i in (0, 1, 2, 3, 4, 5):
    if i == 2 or i == 4:
        continue
    print(i)

0
1
3
5

Tenga en cuenta que 2 y 4 no se imprimen, esto se debe a que continue pasa a la siguiente iteración en lugar de continuar con print(i) cuando i == 2 o i == 4 .

Bucles anidados #

break y continue solo funcionan en un único nivel de bucle. El siguiente ejemplo solo saldrá del bucle interno for, no del bucle externo while:

while True:
    for i in range(1,5):
        if i == 2:
            break    # Will only break out of the inner loop!

Python no tiene la capacidad de salir de múltiples niveles de bucle a la vez; si se desea este comportamiento, refactorizar uno o más bucles en una función y reemplazar break con return puede ser el camino a seguir.

Usar return desde dentro de una función como break #

La declaración [return](https://www.wikiod.com/es/python/funciones#Devolver valores de funciones) sale de una función, sin ejecutar el código que viene después.

Si tiene un bucle dentro de una función, usar return desde dentro de ese bucle es equivalente a tener un break ya que el resto del código del bucle no se ejecuta (tenga en cuenta que cualquier código después del bucle tampoco se ejecuta) :

def break_loop():
    for i in range(1, 5):
        if (i == 2):
            return(i)
        print(i)
    return(5)

Si tiene bucles anidados, la instrucción return interrumpirá todos los bucles:

def break_all():
    for j in range(1, 5):
        for i in range(1,4):
            if i*j == 6:
                return(i)
            print(i*j)

generará:

1 # 1*1
2 # 1*2
3 # 1*3
4 # 1*4
2 # 2*1
4 # 2*2
# return because 2*3 = 6, the remaining iterations of both loops are not executed

1: https://www.wikiod.com/es/python/bucles#Bucles con una cláusula “else”

Para bucles #

Los bucles for iteran sobre una colección de elementos, como list o dict, y ejecutan un bloque de código con cada elemento de la colección.

for i in [0, 1, 2, 3, 4]:
    print(i)

El bucle for anterior itera sobre una lista de números.

Cada iteración establece el valor de i para el siguiente elemento de la lista. Así que primero será 0, luego 1, luego 2, etc. La salida será la siguiente:

0  
1
2
3
4

rango es una función que devuelve una serie de números en forma iterable, por lo que puede usarse en bucles for:

for i in range(5):
    print(i)

da exactamente el mismo resultado que el primer bucle for. Tenga en cuenta que 5 no se imprime ya que el rango aquí son los primeros cinco números contados desde 0.

Objetos iterables e iteradores #

El bucle for puede iterar sobre cualquier objeto iterable que sea un objeto que defina una función __getitem__ o __iter__. La función __iter__ devuelve un iterador, que es un objeto con una función siguiente que se usa para acceder al siguiente elemento del iterable.

Iterando sobre listas #

Para iterar a través de una lista, puede usar for:

for x in ['one', 'two', 'three', 'four']:
    print(x)

Esto imprimirá los elementos de la lista:

one
two
three
four

La función rango genera números que también se usan a menudo en un bucle for.

for x in range(1, 6):
    print(x)

El resultado será un tipo de secuencia de rango especial en python >=3 y una lista en python <=2. Ambos se pueden recorrer mediante el bucle for.

1
2
3
4
5

Si desea recorrer los elementos de una lista y también tienen un índice para los elementos, puede usar la función enumerar de Python:

for index, item in enumerate(['one', 'two', 'three', 'four']):
    print(index, '::', item)

enumerate generará tuplas, que se descomprimen en index (un número entero) y item (el valor real de la lista). El bucle anterior se imprimirá

(0, '::', 'one')
(1, '::', 'two')
(2, '::', 'three')
(3, '::', 'four')

Iterar sobre una lista con manipulación de valores usando map y lambda, es decir, aplicar la función lambda en cada elemento de la lista:

x = map(lambda e :  e.upper(), ['one', 'two', 'three', 'four'])
print(x)

Producción:

['ONE', 'TWO', 'THREE', 'FOUR'] # Python 2.x

NB: en Python 3.x, map devuelve un iterador en lugar de una lista, por lo que, en caso de que necesite una lista, debe emitir el resultado print (list (x)) (ver http://stackoverflow.com/ documentación/python/809/incompatibilities- between-python-2-and-python-3/8186/map) en https://www.wikiod.com/es/python/incompatibilidades-al-pasar-de-python-2-a-python-3 between-python-2-and-python -3 ).

Bucles con una cláusula “else” #

Las declaraciones compuestas for y while (bucles) pueden tener opcionalmente una cláusula else (en la práctica, este uso es bastante raro).

La cláusula else solo se ejecuta después de que un bucle for termina iterando hasta completarse, o después de que un bucle while termina cuando su expresión condicional se vuelve falsa.

for i in range(3):
    print(i)
else:
    print('done')

i = 0
while i < 3:
    print(i)
    i += 1
else:
    print('done')

producción:

 0
 1
 2
 done

La cláusula else no se ejecuta si el ciclo termina de alguna otra manera (a través de una declaración break o lanzando una excepción):

for i in range(2):
    print(i)
    if i == 1:
        break
else:
    print('done')

producción:

0
1

La mayoría de los otros lenguajes de programación carecen de esta cláusula opcional else de bucles. El uso de la palabra clave else en particular a menudo se considera confuso.

El concepto original de una cláusula de este tipo se remonta a Donald Knuth y el significado de la palabra clave else queda claro si reescribimos un ciclo en términos de declaraciones if y goto declaraciones de días anteriores a la programación estructurada o de un lenguaje ensamblador de nivel inferior.

Por ejemplo:

while loop_condition():
    ...
    if break_condition():
        break
    ...

es equivalente a:

# pseudocode

<<start>>:
if loop_condition():
    ...
    if break_condition():
        goto <<end>>
    ...
    goto <<start>>

<<end>>:

Estos siguen siendo equivalentes si adjuntamos una cláusula else a cada uno de ellos.

Por ejemplo:

while loop_condition():
    ...
    if break_condition():
        break
    ...
else:
    print('done')

es equivalente a:

# pseudocode

<<start>>:
if loop_condition():
    ...
    if break_condition():
        goto <<end>>
    ...
    goto <<start>>
else:
    print('done')

<<end>>:

Un bucle for con una cláusula else se puede entender de la misma manera. Conceptualmente, hay una condición de bucle que sigue siendo verdadera siempre que el objeto iterable o la secuencia todavía tenga algunos elementos restantes.

¿Por qué uno usaría esta extraña construcción? #

El principal caso de uso de la construcción for...else es una implementación concisa de la búsqueda como, por ejemplo:

a = [1, 2, 3, 4]
for i in a:
    if type(i) is not int:
        print(i)
        break
else:
    print("no exception")

Para que else en esta construcción sea menos confusa, se puede considerar como “si no se rompe” o “si no se encuentra”.

Algunas discusiones sobre esto se pueden encontrar en [Python-ideas] Summary of for…else threads, ¿Por qué Python usa ’else’ después de los bucles for y while? y [Cláusulas Else en declaraciones de bucle](http://python- notes.curiousefficiency.org/en/latest/python_concepts/break_else.html)

Iterando sobre diccionarios #

Teniendo en cuenta el siguiente diccionario:

 d = {"a": 1, "b": 2, "c": 3}

Para iterar a través de sus claves, puede usar:

for key in d:
    print(key)

Producción:

"a"
"b"
"c"

Esto es equivalente a:

for key in d.keys():
    print(key)

o en Python 2:

for key in d.iterkeys():
    print(key)

Para iterar a través de sus valores, use:

for value in d.values():
    print(value)

Producción:

1
2
3

Para iterar a través de sus claves y valores, use:

for key, value in d.items():
    print(key, "::", value)

Producción:

a :: 1
b :: 2
c :: 3

Tenga en cuenta que en Python 2, .keys(), .values() y .items() devuelven un objeto list. Si simplemente necesita iterar a través del resultado, puede usar el equivalente .iterkeys(), .itervalues() y .iteritems().

La diferencia entre .keys() y .iterkeys(), .values() y .itervalues(), .items() y .iteritems() es que iter* Los métodos son generadores. Por lo tanto, los elementos dentro del diccionario se obtienen uno por uno a medida que se evalúan. Cuando se devuelve un objeto de lista, todos los elementos se empaquetan en una lista y luego se devuelven para una evaluación adicional.

Tenga en cuenta también que en Python 3, el orden de los elementos impresos de la manera anterior no sigue ningún orden.

La declaración de aprobación #

pass es una declaración nula para cuando la sintaxis de Python requiere una declaración (como dentro del cuerpo de un bucle for o while), pero el programador no requiere ni desea ninguna acción. Esto puede ser útil como marcador de posición para el código que aún no se ha escrito.

for x in range(10):
    pass #we don't want to do anything, or are not ready to do anything here, so we'll pass

En este ejemplo, no pasará nada. El bucle for se completará sin errores, pero no se ejecutarán comandos ni códigos. pass nos permite ejecutar nuestro código con éxito sin tener todos los comandos y acciones completamente implementados.

De manera similar, pass se puede usar en bucles while, así como en selecciones y definiciones de funciones, etc.

while x == y:
    pass

Mientras Bucle #

Un ciclo while hará que las instrucciones del ciclo se ejecuten hasta que la condición del ciclo sea falso. El siguiente código ejecutará las declaraciones de bucle un total de 4 veces.

i = 0 
while i < 4:
    #loop statements
    i = i + 1

Mientras que el bucle anterior se puede traducir fácilmente a un bucle for más elegante, los bucles while son útiles para comprobar si se ha cumplido alguna condición. El siguiente ciclo continuará ejecutándose hasta que myObject esté listo.

myObject = anObject()
while myObject.isNotReady():
    myObject.tryToGetReady()

Los bucles while también pueden ejecutarse sin una condición mediante el uso de números (complejos o reales) o True:

import cmath

complex_num = cmath.sqrt(-1)
while complex_num:      # You can also replace complex_num with any number, True or a value of any type
    print(complex_num)   # Prints 1j forever

Si la condición es siempre verdadera, el bucle while se ejecutará para siempre (bucle infinito) si no se termina con una sentencia break o return o una excepción.

while True:
    print "Infinite loop"
# Infinite loop
# Infinite loop
# Infinite loop
# ...

Iterando diferentes partes de una lista con diferentes tamaños de paso #

Suponga que tiene una larga lista de elementos y solo está interesado en todos los demás elementos de la lista. Tal vez solo desee examinar el primer o el último elemento, o un rango específico de entradas en su lista. Python tiene fuertes capacidades integradas de indexación. Aquí hay algunos ejemplos de cómo lograr estos escenarios.

Aquí hay una lista simple que se usará a lo largo de los ejemplos:

lst = ['alpha', 'bravo', 'charlie', 'delta', 'echo']

Iteración sobre toda la lista #

Para iterar sobre cada elemento de la lista, se puede usar un bucle for como el siguiente:

for s in lst:
    print s[:1] # print the first letter

El bucle for asigna s para cada elemento de lst. Esto imprimirá:

a
b
c
d
e

A menudo, necesita tanto el elemento como el índice de ese elemento. La palabra clave enumerar realiza esa tarea.

for idx, s in enumerate(lst):
    print("%s has an index of %d" % (s, idx))

El índice idx comenzará con cero y se incrementará con cada iteración, mientras que s contendrá el elemento que se está procesando. El fragmento anterior generará:

alpha has an index of 0
bravo has an index of 1
charlie has an index of 2
delta has an index of 3
echo has an index of 4

Iterar sobre la sublista #

Si queremos iterar sobre un rango (recordando que Python usa indexación basada en cero), use la palabra clave rango.

for i in range(2,4):
    print("lst at %d contains %s" % (i, lst[i]))

Esto daría como resultado:

lst at 2 contains charlie
lst at 3 contains delta

La lista también se puede dividir. La siguiente notación de segmento va desde el elemento en el índice 1 hasta el final con un paso de 2. Los dos bucles for dan el mismo resultado.

for s in lst[1::2]:
    print(s)

for i in range(1, len(lst), 2):
    print(lst[i])

Los resultados del fragmento anterior:

bravo
delta

La indexación y el corte es un tema propio.

El “medio bucle” do-while #

A diferencia de otros lenguajes, Python no tiene una construcción do-hasta o do-while (esto permitirá que el código se ejecute una vez antes de que se pruebe la condición). Sin embargo, puede combinar while True con [break](https://www.wikiod.com/es/python/bucles#Romper y continuar en bucles) para lograr lo mismo objetivo.

a = 10
while True:
    a = a-1
    print(a)
    if a<7:
        break
print('Done.')

Esto imprimirá:

9
8
7
6
Done.

Bucle y desempaquetado #

Si desea recorrer una lista de tuplas, por ejemplo:

collection = [('a', 'b', 'c'), ('x', 'y', 'z'), ('1', '2', '3')]

en lugar de hacer algo como esto:

for item in collection:
    i1 = item[0]
    i2 = item[1]
    i3 = item[2]
    # logic

o algo como esto:

for item in collection:
    i1, i2, i3 = item
    # logic

Simplemente puede hacer esto:

for i1, i2, i3 in collection:
    # logic

Esto también funcionará para la mayoría de los tipos de iterables, no solo para las tuplas.
