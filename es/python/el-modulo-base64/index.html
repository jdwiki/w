La codificación Base 64 representa un esquema común para codificar binario en formato de cadena ASCII usando radix 64. El módulo base64 es parte de la biblioteca estándar, lo que significa que se instala junto con Python. La comprensión de los bytes y las cadenas es fundamental para este tema y se puede revisar [aquí] (https://goo.gl/N4qIYd). Este tema explica cómo usar las diversas funciones y bases numéricas del módulo base64.

Sintaxis #

  • base64.b64encode(s, altchars=Ninguno)
  • base64.b64decode(s, altchars=Ninguno, validar=Falso)
  • base64.standard_b64encode(s)
  • base64.standard_b64decode(s)
  • base64.urlsafe_b64encode(s)
  • base64.urlsafe_b64decode(s)
  • base64.b32encode(s)
  • base64.b32decode(s)
  • base64.b16encode(s)
  • base64.b16decode(s)
  • base64.a85encode(b, foldspaces=False, wrapcol=0, pad=False, adobe=False)
  • base64.a85decode(b, foldpaces=False, adobe=False, ignorechars=b’\t\n\r\v')
  • base64.b85encode(b, pad=Falso)
  • base64.b85decode(b)

Parámetros #

Parámetro Descripción
base64.b64encode(s, altchars=Ninguno)  
s Un objeto similar a bytes
altares Un objeto similar a bytes de longitud 2+ de caracteres para reemplazar los caracteres ‘+’ y ‘=’ al crear el alfabeto Base64. Los caracteres adicionales se ignoran.
base64.b64decode(s, altchars=Ninguno, validar=Falso)  
s Un objeto similar a bytes
altares Un objeto similar a bytes de longitud 2+ de caracteres para reemplazar los caracteres ‘+’ y ‘=’ al crear el alfabeto Base64. Los caracteres adicionales se ignoran.
validar Si valide es True, los caracteres que no están en el alfabeto Base64 normal o en el alfabeto alternativo no se descartan antes de la comprobación de relleno
base64.standard_b64encode(s)  
s Un objeto similar a bytes
base64.standard_b64decode(s)  
s Un objeto similar a bytes
base64.urlsafe_b64encode(s)  
s Un objeto similar a bytes
base64.urlsafe_b64decode(s)  
s Un objeto similar a bytes
b32encode(s)  
s Un objeto similar a bytes
b32decode(s)  
s Un objeto similar a bytes
base64.b16encode(s)  
s Un objeto similar a bytes

Hasta que apareció Python 3.4, las funciones de codificación y decodificación base64 solo funcionaban con los tipos bytes o bytearray. Ahora estas funciones aceptan cualquier [objeto similar a bytes] (https://docs.python.org/3/glossary.html#term-bytes-like-object).

Codificación y decodificación Base64 #

Para incluir el módulo base64 en su script, primero debe importarlo:

import base64

Las funciones de codificación y decodificación base64 requieren un [objeto similar a bytes] (https://docs.python.org/3/glossary.html#term-bytes-like-object). Para convertir nuestra cadena en bytes, debemos codificarla utilizando la función de codificación integrada de Python. Por lo general, se usa la codificación UTF-8; sin embargo, se puede encontrar una lista completa de estas codificaciones estándar (incluidos los idiomas con diferentes caracteres) [aquí] (https://docs.python.org/3/library/codecs .html#standard-encodings) en la documentación oficial de Python. A continuación se muestra un ejemplo de codificación de una cadena en bytes:

s = "Hello World!"
b = s.encode("UTF-8")

La salida de la última línea sería:

b'¡Hola Mundo!'

El prefijo b se usa para indicar que el valor es un objeto de bytes.

Para codificar en Base64 estos bytes, usamos la función base64.b64encode():

import base64
s = "Hello World!"
b = s.encode("UTF-8")
e = base64.b64encode(b)
print(e)

Ese código generaría lo siguiente:

b'SGVsbG8gV29ybGQh'

que todavía está en el objeto bytes. Para obtener una cadena de estos bytes, podemos usar el método decode() de Python con la codificación UTF-8:

import base64
s = "Hello World!"
b = s.encode("UTF-8")
e = base64.b64encode(b)
s1 = e.decode("UTF-8")
print(s1)

La salida sería entonces:

SGVsbG8gV29ybGQh

Si quisiéramos codificar la cadena y luego decodificarla, podríamos usar el método base64.b64decode():

import base64
# Creating a string
s = "Hello World!"
# Encoding the string into bytes
b = s.encode("UTF-8")
# Base64 Encode the bytes
e = base64.b64encode(b)
# Decoding the Base64 bytes to string
s1 = e.decode("UTF-8")
# Printing Base64 encoded string
print("Base64 Encoded:", s1)
# Encoding the Base64 encoded string into bytes
b1 = s1.encode("UTF-8")
# Decoding the Base64 bytes
d = base64.b64decode(b1)
# Decoding the bytes to string
s2 = d.decode("UTF-8")
print(s2)

Como habrás esperado, la salida sería la cadena original:

Base64 Encoded: SGVsbG8gV29ybGQh
Hello World!
