Obtener ayuda general #

Si se llama a la función ayuda en la consola sin ningún argumento, Python presenta una consola de ayuda interactiva, donde puede obtener información sobre los módulos, símbolos, palabras clave y más de Python.

>>> help()

Welcome to Python 3.4's help utility!

If this is your first time using Python, you should definitely check out
the tutorial on the Internet at http://docs.python.org/3.4/tutorial/.

Enter the name of any module, keyword, or topic to get help on writing
Python programs and using Python modules.  To quit this help utility and
return to the interpreter, just type "quit".

To get a list of available modules, keywords, symbols, or topics, type
"modules", "keywords", "symbols", or "topics".  Each module also comes
with a one-line summary of what it does; to list the modules whose name
or summary contain a given string such as "spam", type "modules spam".

Refiriéndose a la última expresión #

Para obtener el valor del último resultado de su última expresión en la consola, use un guión bajo _.

>>> 2 + 2
4
>>> _
4
>>> _ + 6
10

Este valor de guión bajo mágico solo se actualiza cuando se usa una expresión de Python que da como resultado un valor. La definición de funciones o bucles for no cambia el valor. Si la expresión genera una excepción, no habrá cambios en _.

>>> "Hello, {0}".format("World")
'Hello, World'
>>> _
'Hello, World'
>>> def wontchangethings():
...     pass
>>> _
'Hello, World'
>>> 27 / 0
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ZeroDivisionError: division by zero
>>> _
'Hello, World'

Recuerde, esta variable mágica solo está disponible en el intérprete interactivo de Python. Ejecutar scripts no hará esto.

Abriendo la consola de Python #

La consola de la versión principal de Python generalmente se puede abrir escribiendo py en la consola de Windows o python en otras plataformas.

$ py
Python 3.4.3 (v3.4.3:9b73f1c3e601, Feb 24 2015, 22:44:40) [MSC v.1600 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>>

Si tiene varias versiones, sus ejecutables se asignarán de forma predeterminada a python2 o python3 respectivamente.

Esto, por supuesto, depende de que los ejecutables de Python estén en su RUTA.

La variable PYTHONSTARTUP #

Puede configurar una variable de entorno llamada PYTHONSTARTUP para la consola de Python. Cada vez que ingrese a la consola de Python, este archivo se ejecutará, lo que le permitirá agregar funciones adicionales a la consola, como la importación automática de módulos de uso común.

Si la variable PYTHONSTARTUP se configuró en la ubicación de un archivo que contiene esto:

print("Welcome!")

Luego, abrir la consola de Python daría como resultado este resultado adicional:

$ py
Python 3.4.3 (v3.4.3:9b73f1c3e601, Feb 24 2015, 22:44:40) [MSC v.1600 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.
Welcome!
>>>

Obtener ayuda sobre un objeto #

La consola de Python agrega una nueva función, ayuda, que se puede usar para obtener información sobre una función u objeto.

Para una función, help imprime su firma (argumentos) y su cadena de documentación, si la función tiene una.

>>> help(print)
Help on built-in function print in module builtins:

print(...)
    print(value, ..., sep=' ', end='\n', file=sys.stdout, flush=False)

    Prints the values to a stream, or to sys.stdout by default.
    Optional keyword arguments:
    file:  a file-like object (stream); defaults to the current sys.stdout.
    sep:   string inserted between values, default a space.
    end:   string appended after the last value, default a newline.
    flush: whether to forcibly flush the stream.

Para un objeto, help enumera la cadena de documentación del objeto y las diferentes funciones miembro que tiene el objeto.

>>> x = 2
>>> help(x)
