Regresión lineal #

Dado que Aprendizaje supervisado consiste en una variable objetivo o de resultado (o variable dependiente) que se va a predecir a partir de un conjunto dado de predictores (variables independientes). Usando este conjunto de variables, generamos una función que asigna las entradas a las salidas deseadas. El proceso de entrenamiento continúa hasta que el modelo alcanza el nivel deseado de precisión en los datos de entrenamiento.

Por lo tanto, hay muchos ejemplos de algoritmos de aprendizaje supervisado, por lo que en este caso en el que me gustaría centrarme en Regresión lineal

Regresión lineal Se utiliza para estimar valores reales (coste de las viviendas, número de llamadas, ventas totales, etc.) en función de variable(s) continua(s). Aquí, establecemos la relación entre las variables independientes y dependientes ajustando una mejor línea. Esta recta de mejor ajuste se conoce como recta de regresión y está representada por una ecuación lineal Y= a *X + b.

La mejor manera de entender la regresión lineal es revivir esta experiencia de la infancia. Digamos que le pides a un niño de quinto grado que organice a las personas en su clase en orden creciente de peso, ¡sin preguntarles sus pesos! ¿Qué crees que hará el niño? Él / ella probablemente miraría (analizaría visualmente) la altura y la constitución de las personas y las organizaría usando una combinación de estos parámetros visibles.

¡Esto es regresión lineal en la vida real! El niño realmente ha descubierto que la altura y la complexión estarían correlacionadas con el peso mediante una relación, que se parece a la ecuación anterior.

En esta ecuación:

Y – Dependent Variable
a – Slope
X – Independent variable
b – Intercept

Estos coeficientes a y b se derivan de la minimización de la suma de la diferencia al cuadrado de la distancia entre los puntos de datos y la línea de regresión.

Mira el siguiente ejemplo. Aquí hemos identificado la línea de mejor ajuste que tiene la ecuación lineal y=0.2811x+13.9. Ahora usando esta ecuación, podemos encontrar el peso, conociendo la altura de una persona.

La regresión lineal es principalmente de dos tipos: regresión lineal simple y regresión lineal múltiple. La regresión lineal simple se caracteriza por una variable independiente. Y, la regresión lineal múltiple (como sugiere el nombre) se caracteriza por múltiples (más de 1) variables independientes. Mientras encuentra la mejor línea de ajuste, puede ajustar una regresión polinomial o curvilínea. Y estos se conocen como regresión polinomial o curvilínea.

Solo una pista sobre cómo implementar la regresión lineal en Python

#Load Train and Test datasets
#Identify feature and response variable(s) and values must be numeric and numpy arrays

Clasificación #

Imagina que un sistema quiere detectar manzanas y naranjas en una canasta de frutas. El sistema puede recoger una fruta, extraer alguna propiedad de ella (por ejemplo, el peso de esa fruta).

¡Supongamos que el sistema tiene un maestro! que le enseña al sistema qué objetos son manzanas y cuáles son naranjas. Este es un ejemplo de un problema supervisado de clasificación. Está supervisado porque hemos etiquetado ejemplos. Es una clasificación porque la salida es una predicción de a qué clase pertenece nuestro objeto también.

En este ejemplo consideramos 3 características (propiedades/variables explicativas):

  1. el peso de la fruta seleccionada es superior a 0,5 gramos
  2. tiene un tamaño superior a 10 cm
  3. es el color es rojo

(0 significa No y 1 significa Sí)

Entonces, para representar una manzana/naranja, tenemos una serie (llamada vector) de 3 propiedades (a menudo llamada vector de características)

(por ejemplo, [0,0,1] significa que el peso de esta fruta no supera los 0,5 gramos, su tamaño no supera los 10 cm y su color es rojo)

Entonces, elegimos 10 frutas al azar y medimos sus propiedades. El maestro (humano) luego etiqueta cada fruta manualmente como manzana => 1 o naranja => [2].

ej.) El profesor selecciona una fruta que sea manzana. La representación de esta manzana para el sistema podría ser algo como esto: [1, 1, 1] => 1, Esto significa que esta fruta tiene 1.peso mayor a .5 gramos, 2.tamaño mayor a 10cm y 3. el color de esta fruta es rojo y finalmente es una manzana(=> 1)

Entonces, para las 10 frutas, el maestro etiquete cada fruta como manzana [=> 1] o naranja [=> 2] y el sistema encontró sus propiedades. como adivinas, tenemos una serie de vectores (que se llama matriz) para representar 10 frutas enteras.

Introducción al aprendizaje supervisado #

Hay bastantes situaciones en las que uno tiene grandes cantidades de datos y tiene que clasificar un objeto en una de varias clases conocidas. Considere las siguientes situaciones:

Banca: Cuando un banco recibe una solicitud de un cliente para una tarjeta bancaria, el banco tiene que decidir si emite o no la tarjeta bancaria, en función de las características de sus clientes que ya disfrutan de las tarjetas para las cuales se conoce el historial crediticio. .

Médico: Uno puede estar interesado en desarrollar un sistema médico que diagnostique a un paciente si tiene o no una enfermedad en particular, con base en los síntomas observados y las pruebas médicas realizadas en ese paciente.

Finanzas: A una empresa de consultoría financiera le gustaría predecir la tendencia del precio de una acción que puede clasificarse en alcista, bajista o sin tendencia en función de varias características técnicas que rigen el movimiento del precio.

Expresión génica: A un científico que analiza los datos de expresión génica le gustaría identificar los genes y los factores de riesgo más relevantes involucrados en el cáncer de mama, a fin de separar a las pacientes sanas de las pacientes con cáncer de mama.

En todos los ejemplos anteriores, un objeto se clasifica en una de varias clases conocidas, según las mediciones realizadas en una serie de características, que puede pensar que discriminan los objetos de diferentes clases. Estas variables se denominan variables predictoras y la etiqueta de clase se denomina variable dependiente. Tenga en cuenta que, en todos los ejemplos anteriores, la variable dependiente es categórica.

Para desarrollar un modelo para el problema de clasificación, necesitamos, para cada objeto, datos sobre un conjunto de características prescritas junto con las etiquetas de clase a las que pertenecen los objetos. El conjunto de datos se divide en dos conjuntos en una proporción prescrita. El mayor de estos conjuntos de datos se denomina conjunto de datos de entrenamiento y el otro, conjunto de datos de prueba. El conjunto de datos de entrenamiento se utiliza en el desarrollo del modelo. Como el modelo se desarrolla utilizando observaciones cuyas etiquetas de clase se conocen, estos modelos se conocen como modelos de aprendizaje supervisado.

Después de desarrollar el modelo, se evaluará el rendimiento del modelo utilizando el conjunto de datos de prueba. El objetivo de un modelo de clasificación es tener un mínimo probabilidad de clasificación errónea en las observaciones no vistas. Las observaciones que no se utilizan en el desarrollo del modelo se conocen como observaciones no vistas.

La inducción del árbol de decisiones es una de las técnicas de construcción de modelos de clasificación. El modelo de árbol de decisión creado para la variable dependiente categórica se denomina Árbol de clasificación. La variable dependiente podría ser numérica en ciertos problemas. El modelo de árbol de decisiones desarrollado para variables dependientes numéricas se denomina Árbol de regresión.
