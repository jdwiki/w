Matemáticas usando dc #

dc es uno de los lenguajes más antiguos de Unix.

Está usando la notación polaca inversa, lo que significa que primero está apilando números, luego operaciones. Por ejemplo, 1+1 se escribe como 1 1+.

Para imprimir un elemento desde la parte superior de la pila, use el comando p

echo '2 3 + p' | dc
5

or

dc <<< '2 3 + p'
5

Puede imprimir el elemento superior muchas veces

dc <<< '1 1 + p 2 + p'
2
4

Para números negativos use el prefijo _

dc <<< '_1 p'
-1

También puede usar letras mayúsculas de A a F para números entre 10 y 15 y . como punto decimal

dc <<< 'A.4 p'
10.4

dc usa precisión abitraria, lo que significa que la precisión está limitada solo por la memoria disponible. Por defecto, la precisión se establece en 0 decimales

dc <<< '4 3 / p'
1

Podemos aumentar la precisión usando el comando k. 2k usará

dc <<< '2k 4 3 / p'
1.33

dc <<< '4k 4 3 / p'
1.3333

También puedes usarlo en varias líneas.

dc << EOF
1 1 +
3 *
p
EOF
6

bc es un preprocesador para dc.

Matemáticas usando bc #

bc es un lenguaje de calculadora de precisión arbitraria. Podría usarse de forma interactiva o ejecutarse desde la línea de comandos.

Por ejemplo, puede imprimir el resultado de una expresión:

echo '2 + 3' | bc
5

echo '12 / 5' | bc
2

Para la aritmética de post flotante, puede importar la biblioteca estándar bc -l:

echo '12 / 5' | bc -l
2.40000000000000000000

Se puede utilizar para comparar expresiones:

echo '8 > 5' | bc
1

echo '10 == 11' | bc
0

echo '10 == 10 && 8 > 3' | bc
1

Matemáticas usando capacidades de bash #

El cálculo aritmético también se puede hacer sin involucrar ningún otro programa como este:

Multiplicación:

echo $((5 * 2))
10

División:

echo $((5 / 2))
2

Módulo:

echo $((5 % 2))
1

Exponenciación:

echo $((5 ** 2))
25

Matemáticas usando expr #

expr o Evaluar expresiones evalúa una expresión y escribe el resultado en la salida estándar

Aritmética básica

expr 2 + 3
5

Al multiplicar, debe escapar del signo *

expr 2 \* 3
6

También puedes usar variables

a=2
expr $a + 3
5

Tenga en cuenta que solo admite números enteros, por lo que una expresión como esta

expr 3.0 / 2

arrojará un error expr: no es un número decimal: '3.0'.

Admite expresiones regulares para hacer coincidir patrones.

expr 'Hello World' : 'Hell\(.*\)rld'
o Wo

O encuentre el índice del primer carácter en la cadena de búsqueda

Esto lanzará expr: error de sintaxis en Mac OS X, porque usa BSD expr que no tiene el comando index, mientras que expr en Linux es generalmente GNU expr

expr index hello l
3

expr index 'hello' 'lo'
3
