Por identificación #

document.getElementById('uniqueID')

recuperará

<div id="uniqueID"></div>

Siempre que exista un elemento con el ID dado, document.getElementById devolverá solo ese elemento. De lo contrario, devolverá null.

Nota: Los ID deben ser únicos. Varios elementos no pueden tener el mismo ID.

Por nombre de etiqueta #

document.getElementsByTagName('b')

recuperará

<b>All</b>
<b>of</b>
<b>the b elements.</b>

Si no existen elementos con el nombre de etiqueta dado, se devolverá una colección vacía.

Por selector de CSS #

Considere seguir el código html

<ul>
  <li id=“one” class=“main”>Item 1</li>
  <li id=“two” class=“main”>Item 2</li>
  <li id=“three” class=“main”>Item 3</li>
  <li id=“four”>Item 4</li>
</ul>

El siguiente árbol dom se construirá en función del código html anterior

                      ul
    
                      |
    
     |          |         |         |
    
    li         li        li        li
     |          |         |         |
Item 1        Item 2    Item 3    Item 4

Podemos seleccionar elementos del árbol DOM con la ayuda de selectores CSS. Esto es posible por medio de dos métodos javascript, a saber, querySelector() y querySelectorAll().

El método querySelector() devuelve el primer elemento que coincide con el selector css dado del DOM.

document.querySelector('li.main')

devuelve el primer elemento li cuya clase es main

document.querySelector('#two')

devuelve el elemento con id dos

NOTA: Si no se encuentra ningún elemento, se devuelve null. Si la cadena del selector contiene un pseudoelemento CSS, el retorno será nulo.

El método querySelectorAll() devuelve todos los elementos que coinciden con el selector css dado del DOM.

document.querySelectorAll('li.main')

devuelve una lista de nodos que contiene todos los elementos li cuya clase es main.

NOTA: Si no se encuentra ningún elemento, se devuelve una lista de nodos vacía. Si la cadena de selectores contiene un pseudoelemento CSS, la lista de elementos devuelta estará vacía

Selectores de consulta #

En los navegadores modernos [1], es posible usar un selector similar a CSS para consultar elementos en un documento, de la misma manera que sizzle.js (usado por jQuery ).

querySelector #

Devuelve el primer [Element][elemento-api] del documento que coincide con la consulta. Si no hay ninguna coincidencia, devuelve null.

// gets the element whose id="some-id"
var el1 = document.querySelector('#some-id');
 
// gets the first element in the document containing "class-name" in attribute class
var el2 = document.querySelector('.class-name');
 
// gets the first anchor element in the document
var el2 = document.querySelector('a');
 
// gets the first anchor element inside a section element in the document
var el2 = document.querySelector('section a');

consultaSelectorAll #

Devuelve una [NodeList][api-nodelist] que contiene todos los elementos del documento que coinciden con la consulta. Si ninguno coincide, devuelve una NodeList vacía.

// gets all elements in the document containing "class-name" in attribute class
var el2 = document.querySelectorAll('.class-name');
 
// gets all anchor elements in the document
var el2 = document.querySelectorAll('a');

// gets all anchor elements inside any section element in the document
var el2 = document.querySelectorAll('section a');

Por nombre de clase #

document.getElementsByClassName('class-name')

recuperará

<a class="class-name">Any</a>
<b class="class-name">tag</b>
<div class="class-name an-extra-class">with that class.</div>

Si ningún elemento existente contiene la clase dada, se devolverá una colección vacía.

Ejemplo:

<p class="my-class">I will be matched</p>
<p class="my-class another-class">So will I</p>
<p class="something-else">I won't</p>

var myClassElements = document.getElementByClassName('my-class');
console.log(myClassElements.length); // 2
var nonExistentClassElements = document.getElementByClassName('nope');
console.log(nonExistentClassElements.length); // 0
