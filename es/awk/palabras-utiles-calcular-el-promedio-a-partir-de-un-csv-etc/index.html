Calcule el promedio de valores en una columna a partir de datos tabulares #

Dado un archivo usando ; como delimitador de columna. Calculamos la media de los valores en la segunda columna con el siguiente programa, la entrada provista es la lista de calificaciones de un grupo de estudiantes:

awk -F';' '{ sum += $2 } END { print(sum / NR) }' <<EOF
Alice;2
Victor;1
Barbara;1
Casper;4
Deborah;0
Ernest;1
Fabiola;4
Giuseppe;4
EOF

La salida de este programa es 2.125.

Recuerde que ‘NR’ contiene el número de la línea que se está procesando, en el bloque ‘END’, por lo tanto, contiene el número total de líneas en el archivo.

Recuerda que en muchas aplicaciones (seguimiento, estadísticas), la mediana es una información mucho más útil. Ver el ejemplo correspondiente.

Procesamiento robusto de datos tabulares (CSV et al.) #

Procesar datos tabulares con awk es muy fácil, siempre que la entrada tenga el formato correcto. La mayoría del software que produce datos tabulares utiliza características específicas de esta familia de formatos, y los programas awk que procesan datos tabulares a menudo son específicos de los datos producidos por un software específico. Si se requiere una solución más genérica o robusta, los lenguajes más populares brindan bibliotecas que se adaptan a muchas características que se encuentran en los datos tabulares:

  • nombres de columna opcionales en la primera línea
  • mezcla de valores de columna entre comillas y sin comillas
  • varios delimitadores
  • formatos localizados para números flotantes

Si bien definitivamente es posible manejar todas estas funciones de manera limpia y genérica con awk, probablemente no valga la pena el esfuerzo.

Intercambio de dos columnas en datos tabulares #

Dado un archivo usando ; como delimitador de columna. La permutación de la primera y la segunda columna se logra mediante

awk -F';' -v 'OFS=;' '{ swap = $2; $2 = $1; $1 = swap; print }'

Selección de columnas específicas en datos tabulares #

Asumimos un archivo usando ; como delimitador de columna. Seleccionar un conjunto específico de columnas solo requiere una instrucción print. Por ejemplo, el siguiente programa selecciona las columnas 3, 4 y 7 de su entrada:

awk -F';' -v 'OFS=;' '{ print $3, $4, $7 }'

Como de costumbre, es posible elegir con más cuidado las líneas para imprimir. El siguiente programa selecciona las columnas 3, 4 y 7 de su entrada cuando el primer campo es Alice o Bob:

awk -F';' -v 'OFS=;' '($1 == "Alice") || ($1 == "Bob") { print $3, $4, $7 }'

Calcule la mediana de los valores en una columna a partir de datos tabulares #

Dado un archivo usando ; como delimitador de columna. Calculamos la mediana de los valores en la segunda columna con el siguiente programa, escrito para GNU awk. La entrada proporcionada es la lista de calificaciones de un grupo de estudiantes:

gawk -F';' '{ sample[NR] = $2 }
 END {
   asort(sample);
   if(NR % 2 == 1) {
     print(sample[int(NR/2) + 1])
   } else {
     print(sample[NR/2])
   }
}' <<EOF
Alice;2
Victor;1
Barbara;1
Casper;4
Deborah;0
Ernest;1
Fabiola;4
Giuseppe;4
EOF

La salida de este programa es 1.

Recuerde que ‘NR’ contiene el número de la línea que se está procesando, en el bloque ‘END’, por lo tanto, contiene el número total de líneas en el archivo.

Muchas implementaciones de awk no tienen una función para ordenar matrices, por lo que deben definirse antes de que se pueda usar el código anterior.

Seleccionar un conjunto de líneas entre dos patrones #

La coincidencia de patrones se puede usar de manera efectiva con awk, ya que controla las acciones que le siguen, es decir, { patrón } { acción }. Un buen uso de la coincidencia de patrones es seleccionar múltiples entre dos patrones en un archivo, digamos patrón A y patrón B

$ awk '/patternA/,/patternB/' file

Supongamos que el contenido de mi archivo es el siguiente y quiero extraer las líneas solo entre el patrón anterior:

$ cat file
This is line - 1
This is line - 2
patternA
This is line - 3
This is line - 4
This is line - 5
patternB
This is line - 6

$ awk '/patternA/,/patternB/' file
patternA
This is line - 3
This is line - 4
This is line - 5
patternB

El comando anterior no realiza ninguna { acción } específica aparte de imprimir las líneas que coinciden, pero cualquier acción específica dentro del subconjunto de líneas se puede aplicar con un bloque de acción ({}).
