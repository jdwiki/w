Sesiones de depuración simultáneas con PhpStorm #

Uso #

A veces, podría tener que depurar el código en otro proyecto PhpStorm, debe actualizar la configuración.

Configuración PHP #

En php.ini, edite el archivo y ponga xdebug.remote_autostart = 1

Configuración de PhpStorm #

También tienes que configurar tu IDE:

En la configuración phpStorm, Máx. las conexiones simultáneas deben establecerse en un valor superior a uno.

Documentación oficial #

Más información en la documentación oficial

Configuración #

Eche un vistazo a su archivo de configuración php.ini y habilite Xdebug, agregue la siguiente declaración:

[Xdebug]
zend_extension=<full_path_to_xdebug_extension>
xdebug.remote_enable=1
xdebug.remote_host=<the host where PhpStorm is running (e.g. localhost)>
xdebug.remote_port=<the port to which Xdebug tries to connect on the host where PhpStorm is running (default 9000)>

por ejemplo con Wamp configurado tengo:

; XDEBUG Extension

zend_extension = "d:/wamp/bin/php/php5.5.12/zend_ext/php_xdebug-2.2.5-5.5-vc11.dll"
;
[xdebug]
xdebug.remote_enable = 1
xdebug.profiler_enable = off
xdebug.profiler_enable_trigger = off
xdebug.profiler_output_name = cachegrind.out.%t.%p
xdebug.profiler_output_dir = "d:/wamp/tmp"
xdebug.show_local_vars=0
xdebug.remote_host=localhost
xdebug.remote_port=9000

Reiniciar Apache

Luego, en un shell, ejecute este comando para verificar si Xdebug se está ejecutando correctamente:

php --versión

Una vez que se crea su proyecto PHP, si desea depurarlo, debe configurar su intérprete de php y Xdebug en configuración:

Ajuste de configuración si se usa php-fpm. #

Si es así, el puerto Xdebug 9000 entra en conflicto con el puerto php-fpm predeterminado.

Debe elegir un puerto diferente, ya sea en php-fpm o Xdebug.

Además, cuando se usa php-fpm, TAMBIÉN debe reiniciar php-fpm para que cualquier cambio en los archivos php-ini surta efecto.

Uso de un archivo Xdebug.log #

En algunos casos, deberá mantener registros de lo que sucede en su servidor. Un archivo de registro podría ayudarlo a resolver sus problemas.

La documentación oficial te ayudará a realizar esta funcionalidad

Puede ayudar mucho al tratar de averiguar qué está roto en la instalación.
