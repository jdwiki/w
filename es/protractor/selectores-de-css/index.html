Sintaxis #

  • by.css(‘css-selector’)
  • por.id(‘id’)
  • por.modelo(‘modelo’)
  • by.binding(‘binding’)

Parámetros #

Parámetro Detalles
selector css Un selector css como '.class-name' para seleccionar el elemento en la base del nombre de la clase
id Id del elemento dom
modelo Modelo utilizado para el elemento dom
binding Nombre del enlace que se utiliza para enlazar a cierto elemento

¿Cómo escribir selectores css?

Los atributos más importantes para escribir selectores css son la clase y la identificación de dom. Por ejemplo, si un dom html se ve como el siguiente ejemplo:

  <form class="form-signin">
      <input type="text" id="email" class="form-control" placeholder="Email">
      <input type="password" id="password" class="form-control" placeholder="Password">
      <button class="btn btn-block" id="signin-button" type="submit">Sign in</button>
  </form>

Luego, para seleccionar el campo de entrada de correo electrónico, puede escribir el selector css de la siguiente manera:

  1. Usando el nombre de la clase: El nombre de la clase en el selector css comienza con el carácter especial .(punto). El selector css para eso será como este .form-control.

    by.css('.form-control')

Dado que la clase form-control es compartida por ambos elementos de entrada, surge la preocupación de la duplicidad en los localizadores. Entonces, en tal situación, si la identificación está disponible, siempre debe preferir usar la identificación en lugar del nombre de la clase.

  1. Usando ID: La identificación en el selector css comienza con el carácter especial # (hash). Entonces, el selector css que usa id para el elemento de entrada de correo electrónico se escribirá como se muestra a continuación:

    by.css('#email')

  2. Uso de varios nombres de clase: si el elemento dom tiene varias clases, puede hacerlo con una combinación de clases como selector css. Por ejemplo, si el elemento dom es así:

    // css selector using multiple classes by.css('.username-class.form-control')

  3. Usando el nombre de la etiqueta con otros atributos: La expresión general para escribir el selector css usando el nombre de la etiqueta y otros atributos es tagname[attribute-type='attribute-vallue']. Entonces, siguiendo la expresión, el localizador css para el botón de inicio de sesión se puede formar así:

    by.css(“button[type=‘submit’]”) //or by.css(“button[id=‘signin-button’]”)

Accesos directos del localizador del selector CSS $ y $$ #

La API del transportador permite que los localizadores de elementos CSS utilicen la notación abreviada $() similar a jQuery.

Localizador de elementos CSS normal:

element(by.css('h1.documentation-text[ng-bind="title"]'));
element(by.css('[ng-click="submit"]));

Acceso directo $() Localizador de elementos CSS:

$('h1.documentation-text[ng-bind="title"]');
$('[ng-click="submit"]');

Para encontrar múltiples elementos bajo un localizador, use la notación abreviada $$().

Localizador de elementos CSS normal:

element.all(by.css('h1.documentation-text[ng-bind="title"]'));
element.all(by.css('[ng-click="submit"]));

Acceso directo $$() Localizador de elementos CSS:

$$('h1.documentation-text[ng-bind="title"]');
$$('[ng-click="submit"]');

Seleccionar elemento por un valor de atributo HTML exacto #

Para seleccionar un elemento por un atributo HTML exacto, use el patrón localizador css [atributo=valor]

//selects the first element with href value '/contact'
element(by.css('[href="/contact"]')); 

//selects the first element with tag option and value 'foo'
element(by.css('option[value="foo"]')); 

//selects all input elements nested under the form tag with name attribute 'email'
element.all(by.css('form input[name="email"]'));

Seleccionar elemento por un atributo HTML que contiene un valor especificado #

Para seleccionar un elemento por un atributo HTML que contiene un valor específico, use el patrón localizador css [atributo*=valor]

//selects the first element with href value that contains'cont'
element(by.css('[href*="cont"]')); 

//selects the first element with tag h1 and class attribute that contains 'fo'
element(by.css('h1[class*="fo"]')); 

//selects all li elements with a title attribute that contains 'users'
element.all(by.css('li[title*='users']'));

Introducción a los localizadores #

Un localizador en Protractor se usa para realizar acciones en elementos HTML dom. Los mejores y más comunes localizadores utilizados en Protractor son css, id, model y binding. Por ejemplo, los localizadores de uso común son:

by.css('css-selector')
by.id('id')
