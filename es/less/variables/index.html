Introducción #

En Less, a diferencia de Sass o Shell, las variables se declaran con nombres que comienzan con un símbolo @. Por ejemplo:

@sky-blue: #87ceeb;

body {
  background-color: @sky-blue;
}

El ejemplo anterior te da:

body {
  background-color: #87ceeb;
}

Aquí explica cómo declarar una variable y hacer uso de ellas.

Operaciones en color #

Considere el siguiente ejemplo:

@sky-blue: #87ceeb;
@dark-sky-blue: @sky-blue + #333;

body {
  background-color: @dark-sky-blue;
}

El ejemplo anterior te da:

body {
  background-color: #baffff;
}

Aquí explica cómo declarar una variable y también cómo realizar operaciones en una variable en particular.

Concatenar valor de dos o más variables #

Para concatenar el valor de dos o más variables en una sola cadena e imprimirla como salida, necesitamos hacer uso de la interpolación.

El siguiente código Menos,

#demo:after {
  @var1: Hello;
  @var2: World!!!;
  content: "@{var1} @{var2}";
}

cuando se compila establecería “Hello World!!!” como valor para la propiedad content. A continuación se muestra el CSS compilado:

#demo:after {
  content: "Hello World!!!";
}

Si el valor de dos o más variables solo necesita colocarse uno al lado del otro en forma separada por espacios, entonces no se requiere la interpolación.

#demo {
  @top: 4px;
  @right: 2px;
  @bottom: 6px;
  @left: 4px;
  padding: @top @right @bottom @left;
}

Cuando se compila el código anterior, produciría el siguiente CSS.

#demo {
  padding: 4px 2px 6px 4px;
}

Este enfoque no funcionará cuando no deba haber espacio entre los valores de las variables (o) cuando la cadena resultante deba estar entre comillas. Para esos casos, el uso de la interpolación sería obligatorio.

Hacer referencia a una variable dentro de una función CSS #

Por defecto, LESS usará su propio calc() a menos que se indique lo contrario. Asi que:

@column-count: 2;

.class-example {
    width: calc(100% / @column-count);
}

Se compilaría a esto:

.class-example {
    width: 50%;
}

Si bien es nuestro ancho deseado, LESS ha usado su propia función calc() para calcular el ancho. La función calc() nunca llega a nuestro CSS. Si desea que LESS no use su función calc(), debe escapar sus valores de esta manera:

width: calc(~"100% - @{column-count}");

Aquí hemos antepuesto nuestros valores con un ~ y los hemos envuelto entre comillas. También se puede hacer referencia a las variables, pero debe envolver el nombre de la variable entre corchetes { }. Esto le permite usar la función CSS calc() para cálculos más complejos como este:

@column-count: 2;
@column-margin: 24px;

.class-example {
    width: calc("~(100% / @{column-count}) - @{column-margin}");
}

Esto compila a:

.class-example {
    width: calc((100/2) - 24px);
}

Las variables pueden facilitar su trabajo recursivo #

Declaración de variables

@buttonColor: #FF0000;

/* Ahora puede usar la variable @buttonColor con funciones css. */

.product.into.detailed {
    additional-attributes{
        .lib-table-button(
         background-color: @buttonColor;
         );
    }
}
