Sintaxis #

  • para i en iter; …; final
  • mientras que cond; …; final
  • descanso
  • Seguir
  • @parallel (op) para i en iter; …; final
  • @parallel para i en iter; …; final
  • etiqueta @goto
  • etiqueta @label

Siempre que haga que el código sea más corto y más fácil de leer, considere usar funciones de orden superior, como mapa o filtro, en lugar de bucles.

Encuentra el factor primo más pequeño #

En algunas situaciones, es posible que desee regresar de una función antes de terminar un ciclo completo. La instrucción return se puede utilizar para esto.

function primefactor(n)
    for i in 2:n
        if n % i == 0
            return i
        end
    end
    @assert false  # unreachable
end

Uso:

julia> primefactor(100)
2

julia> primefactor(97)
97

Los bucles también se pueden terminar antes con la instrucción break, que termina solo el bucle que lo contiene en lugar de toda la función.

Iteración multidimensional #

En Julia, un bucle for puede contener una coma (,) para especificar la iteración en varias dimensiones. Esto actúa de manera similar a anidar un bucle dentro de otro, pero puede ser más compacto. Por ejemplo, la siguiente función genera elementos del producto cartesiano de dos iterables:

function cartesian(xs, ys)
    for x in xs, y in ys
        produce(x, y)
    end
end

Uso:

julia> collect(@task cartesian(1:2, 1:4))
8-element Array{Tuple{Int64,Int64},1}:
 (1,1)
 (1,2)
 (1,3)
 (1,4)
 (2,1)
 (2,2)
 (2,3)
 (2,4)

Sin embargo, la indexación sobre arreglos de cualquier dimensión debe hacerse con eachindex, no con un bucle multidimensional (si es posible):

s = zero(eltype(A))
for ind in eachindex(A)
    s += A[ind]
end

Bucles de reducción y paralelos #

Julia proporciona macros para simplificar la distribución de la computación entre varias máquinas o trabajadores. Por ejemplo, lo siguiente calcula la suma de una cierta cantidad de cuadrados, posiblemente en paralelo.

function sumofsquares(A)
    @parallel (+) for i in A
        i ^ 2
    end
end

Uso:

julia> sumofsquares(1:10)
385

Para obtener más información sobre este tema, consulte el ejemplo en @parallel dentro de Parallel Processesing [ tema] (https://www.wikiod.com/es/julia-lang/procesamiento-en-paralelo).

Zumbido de burbujas #

Un caso de uso común para un bucle for es iterar sobre un rango o colección predefinidos y hacer la misma tarea para todos sus elementos. Por ejemplo, aquí combinamos un bucle for con una sentencia condicional [if-elseif-else](https://www.wikiod.com/es/julia-lang/condicionales#declaración if con múltiples ramas -ramas#t=201608180532598795485):

for i in 1:100
    if i % 15 == 0
        println("FizzBuzz")
    elseif i % 3 == 0
        println("Fizz")
    elseif i % 5 == 0
        println("Buzz")
    else
        println(i)
    end
end

Esta es la clásica pregunta de entrevista de Fizz Buzz. La salida (truncada) es:

1
2
Fizz
4
Buzz
Fizz
7
8
