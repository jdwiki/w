Parámetros #

|Parámetro|Detalles | |———|————————————— ————————–| |x |variable del eje x. Puede proporcionar data$variablex o data[,x]| |y |variable del eje y. Puede proporcionar data$variabley o data[,y]| |main |Título principal de la trama | |sub |Subtítulo opcional de la trama | |xlab |Etiqueta para el eje x | |ylab |Etiqueta para el eje y | |pch |Entero o carácter que indica el símbolo de trazado | |col |Entero o cadena que indica el color | |type |Tipo de parcela. "p" para puntos, "l" para líneas, "b" para ambos, "c" para las líneas que forman parte sola de "b", "o" para ambas ‘sobregraficadas’ ‘, "h" para líneas verticales tipo ‘histograma’ (o de ‘alta densidad’), "s" para escalones, "S" para otros escalones, "n" para no trazar |

Los elementos enumerados en la sección “Parámetros” son una pequeña fracción de los posibles parámetros que pueden modificarse o establecerse mediante la función par. Consulte par para obtener una lista más completa. Además, todos los dispositivos gráficos, incluidos los dispositivos gráficos interactivos específicos del sistema, tendrán un conjunto de parámetros que pueden personalizar la salida.

Combinación de parcelas #

Suele ser útil combinar varios tipos de diagramas en un gráfico (por ejemplo, un diagrama de barras junto a un diagrama de dispersión). R facilita esto con la ayuda de las funciones par() y layout().

par() #

par usa los argumentos mfrow o mfcol para crear una matriz de nrows y ncols c(nrows, ncols) que servirá como cuadrícula para sus gráficos. El siguiente ejemplo muestra cómo combinar cuatro parcelas en un gráfico:

par(mfrow=c(2,2))
plot(cars, main="Speed vs. Distance")
hist(cars$speed, main="Histogram of Speed")
boxplot(cars$dist, main="Boxplot of Distance")
boxplot(cars$speed, main="Boxplot of Speed")

diseño() #

El layout() es más flexible y le permite especificar la ubicación y la extensión de cada gráfico dentro del gráfico combinado final. Esta función espera un objeto de matriz como entrada:

layout(matrix(c(1,1,2,3), 2,2, byrow=T))
hist(cars$speed, main="Histogram of Speed")
boxplot(cars$dist, main="Boxplot of Distance")
boxplot(cars$speed, main="Boxplot of Speed")

Diagrama de densidad #

Un seguimiento muy útil y lógico de los histogramas sería trazar la función de densidad suavizada de una variable aleatoria. Una trama básica producida por el comando.

plot(density(rnorm(100)),main="Normal density",xlab="x")

se vería como

Puede superponer un histograma y una curva de densidad con

x=rnorm(100)
hist(x,prob=TRUE,main="Normal density + histogram")
lines(density(x),lty="dotted",col="red")

lo que da

Trama basica #

Un gráfico básico se crea llamando a plot(). Aquí usamos el marco de datos incorporado cars que contiene la velocidad de los autos y las distancias tomadas para detenerse en la década de 1920. (Para obtener más información sobre el conjunto de datos, utilice help(cars)).

plot(x = cars$speed, y = cars$dist, pch = 1, col = 1, 
     main = "Distance vs Speed of Cars", 
     xlab = "Speed", ylab = "Distance")

Podemos usar muchas otras variaciones en el código para obtener el mismo resultado. También podemos cambiar los parámetros para obtener resultados diferentes.

with(cars, plot(dist~speed, pch = 2, col = 3, 
     main = "Distance to stop vs Speed of Cars", 
     xlab = "Speed", ylab = "Distance"))

Se pueden agregar funciones adicionales a este diagrama llamando a points(), text(), mtext(), lines(), grid(), etc.

plot(dist~speed, pch = "*", col = "magenta", data=cars,
     main = "Distance to stop vs Speed of Cars", 
     xlab = "Speed", ylab = "Distance")
mtext("In the 1920s.")
grid(,col="lightblue")

Primeros pasos con R_Plots #

  • Gráfico de dispersión

Tienes dos vectores y quieres graficarlos.

x_values <- rnorm(n = 20 , mean = 5 , sd = 8) #20 values generated from Normal(5,8)
y_values <- rbeta(n = 20 , shape1 = 500 , shape2 = 10) #20 values generated from Beta(500,10)

Si desea hacer un gráfico que tenga y_values en el eje vertical y x_values en el eje horizontal, puede usar los siguientes comandos:

plot(x = x_values, y = y_values, type = "p") #standard scatter-plot
plot(x = x_values, y = y_values, type = "l") # plot with lines
plot(x = x_values, y = y_values, type = "n") # empty plot

Puede escribir ?plot() en la consola para leer más opciones.

  • Diagrama de caja

Tienes algunas variables y quieres examinar sus Distribuciones

#boxplot is an easy way to see if we have some outliers in the data.
   
z<- rbeta(20 , 500 , 10) #generating values from beta distribution
z[c(19 , 20)] <- c(0.97 , 1.05) # replace the two last values with outliers      
boxplot(z) # the two points are the outliers of variable z.
  • Histogramas

Manera fácil de dibujar histogramas

hist(x = x_values) # Histogram for x vector
hist(x = x_values, breaks = 3) #use breaks to set the numbers of bars you want
  • Gráficos circulares

Si quieres visualizar las frecuencias de una variable solo dibuja un pastel

Primero tenemos que generar datos con frecuencias, por ejemplo:

P <- c(rep('A' , 3) , rep('B' , 10) , rep('C' , 7) )
t <- table(P) # this is a frequency matrix of variable P
pie(t) # And this is a visual version of the matrix above

Histogramas #

Los histogramas permiten un pseudográfico de la distribución subyacente de los datos.

hist(ldeaths)


hist(ldeaths, breaks = 20, freq = F, col = 3)

Diagrama de matriz #

matplot es útil para trazar rápidamente múltiples conjuntos de observaciones del mismo objeto, particularmente de una matriz, en el mismo gráfico.

Aquí hay un ejemplo de una matriz que contiene cuatro conjuntos de sorteos aleatorios, cada uno con una media diferente.

xmat <- cbind(rnorm(100, -3), rnorm(100, -1), rnorm(100, 1), rnorm(100, 3))
head(xmat)
#          [,1]        [,2]       [,3]     [,4]
# [1,] -3.072793 -2.53111494  0.6168063 3.780465
# [2,] -3.702545 -1.42789347 -0.2197196 2.478416
# [3,] -2.890698 -1.88476126  1.9586467 5.268474
# [4,] -3.431133 -2.02626870  1.1153643 3.170689
# [5,] -4.532925  0.02164187  0.9783948 3.162121
# [6,] -2.169391 -1.42699116  0.3214854 4.480305

Una forma de trazar todas estas observaciones en el mismo gráfico es hacer una llamada a “trazar” seguida de tres llamadas más a “puntos” o “líneas”.

plot(xmat[,1], type = 'l')
lines(xmat[,2], col = 'red')
lines(xmat[,3], col = 'green')
lines(xmat[,4], col = 'blue')

Sin embargo, esto es tedioso y causa problemas porque, entre otras cosas, por defecto, los límites de los ejes están fijados por plot para ajustarse solo a la primera columna.

Mucho más conveniente en esta situación es usar la función matplot, que solo requiere una llamada y automáticamente se ocupa de los límites de los ejes y cambia la estética de cada columna para que sean distinguibles.

matplot(xmat, type = 'l')

Tenga en cuenta que, de forma predeterminada, matplot varía tanto el color (col) como el tipo de línea (lty) porque esto aumenta el número de combinaciones posibles antes de que se repitan. Sin embargo, cualquiera (o ambas) de estas estéticas pueden fijarse en un único valor…

matplot(xmat, type = 'l', col = 'black')

…o un vector personalizado (que se reciclará según el número de columnas, siguiendo las reglas estándar de reciclaje de vectores R).

matplot(xmat, type = 'l', col = c('red', 'green', 'blue', 'orange'))

Los parámetros gráficos estándar, incluidos main, xlab, xmin, funcionan exactamente de la misma manera que plot. Para obtener más información sobre ellos, consulte ?par.

Al igual que plot, si se le da solo un objeto, matplot asume que es la variable y y usa los índices para x. Sin embargo, x e y pueden especificarse explícitamente.

matplot(x = seq(0, 10, length.out = 100), y = xmat, type='l')

De hecho, tanto x como y pueden ser matrices.

xes <- cbind(seq(0, 10, length.out = 100),
             seq(2.5, 12.5, length.out = 100),
             seq(5, 15, length.out = 100),
             seq(7.5, 17.5, length.out = 100))

plot(ecdf(rnorm(100)),main="Cumulative distribution",xlab="x")
