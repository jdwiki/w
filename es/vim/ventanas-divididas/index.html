Sintaxis #

  • : dividir <archivo>
  • :vsplit <archivo>
  • :sp <- abreviatura de división
  • :vsp <- abreviatura de vsplit

Cuando se llama desde la línea de comandos, se pueden proporcionar varios archivos en el argumento y vim creará una división para cada archivo. Cuando se llama desde el modo ex, solo se puede abrir un archivo por invocación del comando.

Abriendo múltiples archivos en divisiones desde la línea de comando #

Horizontalmente #

vim -o file1.txt file2.txt

Verticalmente #

vim -O file1.txt file2.txt

Opcionalmente, puede especificar el número de divisiones para abrir. El siguiente ejemplo abre dos divisiones horizontales y carga file3.txt en un búfer:

vim -o2 file1.txt file2.txt file3.txt

Abriendo una nueva ventana dividida #

Puede abrir una nueva división dentro de Vim con los siguientes comandos, en modo normal:

Horizontalmente:

:split <file name>
:new

Verticalmente:

:vsplit <file name> 
:vnew

split abrirá el archivo en una nueva división en la parte superior o izquierda de su pantalla (o división actual). :sp y :vs son atajos convenientes.

nuevo abrirá una división vacía

Cambiando el tamaño de un split o vsplit #

En ocasiones, es posible que desee cambiar el tamaño de un split o vsplit.

Para cambiar el tamaño de la división actualmente activa, use :resize <nuevo tamaño>. :resize 30, por ejemplo, haría que la división tuviera 30 líneas de alto.

Para cambiar el tamaño del vsplit actualmente activo, use :redimensionamiento vertical <nuevo tamaño>. :redimensionamiento vertical 80, por ejemplo, haría que vsplit tuviera 80 caracteres de ancho.

Atajos #

  • Ctrl + w y + aumentan el tamaño de la ventana dividida
  • Ctrl + w y - reducen el tamaño de la ventana dividida
  • Ctrl + w y = establecen el mismo tamaño para las ventanas divididas

Cerrar todas las divisiones excepto la actual #

Modo normal Ctrl-wo

modo Ex

:only

o corto

:on

Moverse entre divisiones #

Para pasar a dividir a la izquierda, use <C-w><C-h> Para pasar a dividir a continuación, use <C-w><C-j> Para pasar a dividir a la derecha, use <C-w><C-k> Para pasar a dividir arriba, use <C-w><C-l>

Apertura dividida sana #

Es una mejor experiencia abrir la división de abajo y a la derecha.

configúralo usando

set splitbelow
set splitright

Gestión de ventanas divididas abiertas (métodos abreviados de teclado) #

Después de haber abierto una ventana dividida en vim (como lo demuestran muchos ejemplos bajo esta etiqueta), es probable que desee controlar las ventanas rápidamente. Aquí se explica cómo controlar las ventanas divididas mediante atajos de teclado.

Mover para dividir arriba/abajo:

  • Ctrl + w y k
  • Ctrl + w y j

Mover para dividir izquierda/derecha:

  • Ctrl + w y h
  • Ctrl + w y l

Mover para dividir arriba/abajo (envolver):

  • Ctrl + w y w

Crear nueva ventana vacía:

  • Ctrl + w y n -o- :nuevo

Crear nueva división horizontal/vertical:

  • Ctrl+W, S (mayúsculas)
  • Ctrl+W, v (minúsculas)

Haga que la división actualmente activa sea la que está en pantalla:

  • Ctrl + w y o -o- :on
