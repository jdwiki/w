Manejo de duplicados #

# example data
DT = data.table(id = c(1,2,2,3,3,3))[, v := LETTERS[.I]][]

Para lidiar con los “duplicados”, combine [contar filas en un grupo][2] y subdividir filas por grupo.

Mantenga una fila por grupo #

También conocido como “soltar duplicados”, también conocido como “desduplicar”, también conocido como “uniquificar”.

unique(DT, by="id")
# or
DT[, .SD[1L], by=id]
#    id v
# 1:  1 A
# 2:  2 B
# 3:  3 D

Esto mantiene la primera fila. Para seleccionar una fila diferente, uno puede jugar con la parte 1L o usar order en i.

Mantener solo filas únicas #

DT[, if (.N == 1L) .SD, by=id]
#    id v
# 1:  1 A

Mantener solo filas no únicas #

DT[, if (.N > 1L) .SD, by=id]
#    id v
# 1:  2 B
# 2:  2 C
# 3:  3 D
# 4:  3 E
# 5:  3 F

[2]: https://www.wikiod.com/es/data-table/calculo-de-estadisticas-de-resumen#Contando filas por grupo
