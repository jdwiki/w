Una de las mayores ventajas de LaTeX es su habilidad para componer ecuaciones. Aquí se describen los fundamentos de la composición tipográfica de ecuaciones, algunos de los diversos paquetes que se pueden utilizar, así como los símbolos comunes.

Sintaxis #

  • \begin{ecuación} … \end{ecuación}
  • texto $ … $ texto
  • \usepackage{amsmath} … \begin{ecuación*} … \end{ecuación*}

Aquí hay algunas ideas básicas para asegurarse de que su código no se rompa y que sus ecuaciones se vean mejor:

  1. Asegúrese de que todos los corchetes, llaves, signos de dólar y comandos \begin{} \end{} coincidan. Esto es algo en lo que un pequeño error puede estropear todo el código a lo grande.
  2. Si obtiene errores, asegúrese de tener cargado el paquete adecuado (por ejemplo, no use el comando \begin{equation*} sin el paquete amsmath).
  3. Nunca, nunca, nunca use signos de dólar dobles ($$una ecuación aquí$$) en lugar de \begin{ecuación}.
  4. Nunca use el modo matemático como una forma de poner su texto en cursiva.
  5. ¿Completamente atascado? Pruebe TeX.SX, un sitio para responder preguntas sobre TeX, LaTeX y lenguajes relacionados.

¡Buena suerte!

Ecuaciones sencillas en línea

Puedes hacer una ecuación en línea simple usando $una ecuación aquí$.

Ecuaciones numeradas y centradas

Al escribir artículos u otros documentos, a veces es preferible tener las ecuaciones centradas y numeradas, en lugar de en línea. Luego, usa los comandos \begin{equation} y \end{equation}.

Por ejemplo, si usamos el código

\begin{equation}
\lim\limits_{n \to \infty} \frac{1}{2^n} i\bar z
\end{equation}

Puedes eliminar la numeración de la ecuación usando \begin{equation*} y \end{equation*}.

Por ejemplo, si usamos el código

\begin{equation*}
\lim\limits_{n \to \infty} \frac{1}{2^n} i\bar z
\end{equation*}

Encontrar símbolos #

A veces, puede ser difícil encontrar el símbolo matemático que necesita. Hay varias opciones aquí. El primero (y el más rápido) es usar Detexify, donde dibujas el símbolo que te gustaría e intenta encontrar lo que buscas, como se muestra a continuación:

Otra opción es utilizar la lista completa de símbolos de LaTeX, que se puede encontrar aquí. Si está utilizando el paquete unicode-math esta lista de todos los símbolos admitidos puede ser útil. Otra opción es este sitio web, que tiene símbolos matemáticos comunes.

Paquetes disponibles para su uso #

Si bien LaTeX estándar es todo lo que se necesita para la mayoría de las fórmulas y ecuaciones matemáticas simples, a veces se necesitan más símbolos y herramientas. Hay varios paquetes disponibles que mejorarán sus ecuaciones y le brindarán más con qué trabajar. Tres de los paquetes principales se describen a continuación. Recuerde, para cargar un paquete, escriba \usepackage{paquete} en el preámbulo de su documento.

ammath

El paquete amsmath es un paquete increíblemente útil. Se usa para permitir que sus ecuaciones estén centradas pero no numeradas, como en \begin{equation*}, se usa para crear matrices (como se describe a continuación) e introduce muchos otros comandos útiles, como \overset y \underset, que se describen a continuación. La documentación del paquete amsmath se puede encontrar aquí.

herramientas matemáticas

El paquete mathtools se basa en el paquete amsmath y agrega más símbolos y herramientas útiles. Carga automáticamente el paquete amsmath, por lo que no necesita cargar ambos en el preámbulo de su documento. La documentación de mathtools se puede encontrar aquí.

amssymb

El paquete amssymb proporciona muchos símbolos adicionales que pueden ser muy útiles para ecuaciones más complejas. La documentación amssymb se puede encontrar aquí.

Paquetes de fuentes

También hay varias fuentes que puede usar para sus ecuaciones, como se describe en esta pregunta en el intercambio de pila de TeX, para TeX, LaTeX y amigos.

Este documento es una explicación concisa de las diferentes funciones proporcionadas por algunos paquetes, así como por LaTeX estándar; es de mucha ayuda.

Buenos comandos para saber #

Algunos de los comandos más comunes incluyen:

  • Fracciones y raíces cuadradas: Para fracciones, use \frac {numerador}{denominador}. Para raíces cuadradas, use \sqrt[root]{number}.
  • Letras griegas: usa los comandos dados en la siguiente tabla:

  • Operadores: \leq da el símbolo menor o igual, \geq da el símbolo mayor o igual, \neq da el símbolo diferente, \sum da la sumatoria \partial da el símbolo de derivada parcial, \nabla da el operador laplaciano, \times da el producto vectorial o símbolo de multiplicación, \cdot da el producto escalar o símbolo de multiplicación, y \int da el símbolo integral.
  • Flechas: \rightarrow y \leftarrow dan flechas derecha e izquierda, respectivamente.
  • Porcentajes: Si escribe % en LaTeX, es importante incluir una barra invertida, \%, ya que el símbolo de porcentaje se usa normalmente para los comentarios.
  • Superíndices y subíndices: Para hacer un superíndice, puede escribir x^2 o, para superíndices más largos, x^{2x}. Para hacer un subíndice, puede escribir x_a o, para subíndices más largos, x_{ab}.
  • Negrita: Usa \boldmath{...} para hacer que tus símbolos matemáticos estén en negrita. Se dan otras opciones en esta pregunta de TeX.SX. Los símbolos matemáticos se ponen en cursiva automáticamente; si no desea que esto sea cierto, haga el texto de su ecuación como se describe a continuación.
  • Infinito: Para escribir infinito, usa el comando \infty.
  • Mover elementos sobre o debajo de otro: Primero, solo para operadores matemáticos, hay un método alternativo. Puede escribir el operador matemático, decir \int y luego usar el comando \limits. Un ejemplo es \int\limits_{\infty} o \int\limits^{\infty}. Entonces, para casos normales, puedes hacer \overset{top}{normal} o \underset{bottom}{normal}. Esto puede ser muy útil para hacer vectores. Por ejemplo, puede hacer \overset{\rightarrow}{x} El paquete amsmath es necesario para overset y underset.
  • ** Llaves: ** Debido a que las llaves se usan en los comandos, es necesario escribir \{ o \} para obtener llaves.
  • Texto: Para incluir texto en las ecuaciones, escriba \usepackage{amsmath} en el preámbulo y luego escriba \text{...}.
  • Espacio: Para agregar espacio en sus ecuaciones, escriba \quad entre los dos elementos que desea separar (por ejemplo, podría tener $2x \quad cos).

Creación de nuevos símbolos #

Digamos que no puede encontrar el símbolo que necesita en ninguna parte. Puede crear un símbolo personalizado. Por ejemplo, el código

\documentclass{article}
\usepackage{graphicx,amsmath,amssymb}
\DeclareRobustCommand{\diamondtimes}{%
  \mathbin{\text{\rotatebox[origin=c]{45}{$\boxplus$}}}%
}

\begin{document}
$a\diamondtimes b$
\end{document}

crea y llama a un símbolo, dando

Este es un ejemplo más simple; simplemente tiene que rotar un símbolo ya existente. Sin embargo, puede crear símbolos más complejos.

Matrices #

Matrices

Siempre debe usar el paquete amsmath si va a usar los siguientes comandos. Hay cuatro tipos principales de matriz, como se muestra en el siguiente código:

\begin{matrix} 
    a & b \\
    c & d 
\end{matrix}
\quad
\begin{pmatrix} 
   a & b \\
   c & d 
\end{pmatrix}
\quad
\begin{bmatrix} 
    a & b \\
    c & d 
\end{bmatrix}
\quad
\begin{vmatrix} 
    a & b \\
    c & d 
\end{vmatrix}
\quad
\begin{Vmatrix} 
    a & b \\
    c & d 
\end{Vmatrix}

Este código produce

Hay un par de cosas importantes a tener en cuenta sobre esto:

  1. Es importante que coloque su matriz dentro del entorno ecuación, ecuación* o $...$; el comando bmatrix no es un entorno matemático por sí solo.
  2. La construcción de la matriz es en realidad bastante simple. Para cada fila, crea cada elemento (digamos x_{11}), luego coloca un & y luego escribe el siguiente elemento. Para filas múltiples, al final de cada fila, coloque \\ (no tiene que hacer esto para la última fila). Es bastante similar a una tabla en esto.
