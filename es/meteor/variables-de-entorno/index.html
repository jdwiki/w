Uso de variables de entorno con Meteor #

Las variables de entorno se pueden definir antes de la llamada del meteorito, así:

PORT=4000 meteor
NODE_ENV="staging" meteor

Configuración del servidor SMTP Meteor #

Ejemplo de Gmail

MAIL_URL=smtp://username%40gmail.com:password@smtp.gmail.com:465/

Nota: Esta configuración solo permite enviar 2000 correos electrónicos por día. Consulte https://support.google.com/a/answer/176600?hl=en para obtener configuraciones alternativas.
