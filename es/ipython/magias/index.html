IPython amplía Python con un conjunto de comandos llamados magics. Estas son funciones especiales cuyos nombres comienzan con un % que son reconocidas por el shell de IPython. Las magias cuyo nombre comienza con un solo % toman como argumento el resto de la línea y se denominan magias de línea. Las magias que comienzan con un signo de doble porcentaje ‘%%’ toman un argumento de varias líneas y se denominan magias de celdas.

Las magias %timeit y %time #

La magia %timeit ejecuta el código dado muchas veces, luego devuelve la velocidad del resultado más rápido.

In [1]: %timeit sum(range(100000))
100 loops, best of 3: 2.91 ms per loop

La magia de la celda %%timeit se puede usar para cronometrar bloques de código.

In [2]: %%timeit
   ...: a = 0
   ...: for i in range(100000):
   ...:     a += i
   ...:
100 loops, best of 3: 9.67 ms per loop

La magia %time cronometra una sola ejecución de una función, similar al comando time de Unix. A diferencia de %timeit, %time también muestra el resultado.

In [3]: %time sum(range(100000))
CPU times: user 2.68 ms, sys: 3 µs, total: 2.68 ms
Wall time: 2.69 ms
Out[3]: 4999950000

Magia integrada de líneas y celdas #

Las magias cuyo nombre comienza con un solo % toman como argumento el resto de la línea y se denominan magias de línea. Las magias que comienzan con un signo de doble porcentaje ‘%%’ toman un argumento de varias líneas y se denominan magias de celdas.

Una magia comúnmente utilizada es [%timeit][1], un envoltorio alrededor de la función timeit.timeit de Python, para medir el tiempo de ejecución de una pieza de código.

In [35]: ra = [random.randint(0,1000) for r in range(1000)]
In [35]: %timeit sorted(ra)
1000 loops, best of 3: 507 µs per loop

Un ejemplo de magia de celda es %%bash (equivalente a %%script bash) para ejecutar la entrada como código bash

In [49]: %%bash
    ...: i=3
    ...: while [ $i -ge 0 ]
    ...: do
    ...: echo $i
    ...: i=$(($i-1))
    ...: done
    ...:
3
2
1
0

Tenga en cuenta que el final de una celda está marcado por una línea vacía.

Automagic is ON, % prefix IS NOT needed for line magics.

Tenga en cuenta que, de forma predeterminada, ‘automagic’ está activado y, por lo tanto, las líneas mágicas se pueden llamar sin el prefijo ‘%’ (pero las funciones de celda aún necesitan un prefijo ‘%%’).
