NppExec #

NppExec [sourceforge] le permite ejecutar comandos y scripts desde una ventana de consola en Notepad++. Se puede encontrar en la barra de menú en Complementos -> NppExec o simplemente presionando la tecla F6 (el atajo Ctrl+F6 ejecutará el último comando).

Ejemplo: el siguiente testamento

  • Configure la consola en output_var: on, lo que significa que podemos usar la salida de la consola

  • ejecutar una consulta SQL desde el archivo C:\scripts\query.sql

  • tome la salida de la consola y colóquela en el archivo activo en el cursor

  • cerrar la consola

  • borrar la consola

    NPE_CONSOLE v+ sqlcmd -S 111.111.1.1 -U UserName -P “password” -i C:\scripts\query.sql sel_settext $(OUTPUT) NPP_CONSOLE 0 cls

NppExec también le permite guardar sus scripts. Después de guardarlos, puede ir a Complementos -> NppExec -> Opciones avanzadas y ejecutarlo en cualquier momento en que Notepad ++ se inicie, cierre o incluso agregue ese script a los comandos del complemento usando macros. Por ejemplo, al guardar el ejemplo anterior como “Ejecutar consulta”, podría usar los campos de la parte inferior izquierda en las Opciones avanzadas para agregarlo al menú.

La secuencia de comandos estará disponible como una macro después de reiniciar Notepad++, siempre que la casilla “Colocar en el submenú Macros” esté marcada.

Finalmente, se puede asignar un acceso directo a la macro/comando usando Configuración -> Asignador de accesos directos -> Comandos de complemento.

Visor JSON #

JSON Viewer SourceForge es un complemento para visualización y formato JSON. Es útil para sangrar/formatear documentos JSON y se puede usar para buscar archivos JSON complejos usando una herramienta de vista de árbol.

A partir de un fragmento JSON sin formato (Ejemplo de http://www.json.org):

{“glosario”: {“título”: “glosario de ejemplo”,“GlossDiv”: {“título”: “S”,“GlossList”: {“GlossEntry”: {“ID”: “SGML”,“SortAs”: “SGML”,“GlossTerm”: “Lenguaje de marcado generalizado estándar”,“Acrónimo”: “SGML”,“Abbrev”: “ISO 8879:1986”,“GlossDef”: {“for”: “Un metamarcado lenguaje, utilizado para crear lenguajes de marcado como DocBook.”,“GlossSeeAlso”: [“GML”, “XML”]},“GlossSee”: “marcado”}}}}}

Para formatear y sangrar el código:

  • seleccione todo el fragmento json
  • haga clic en “Complementos”/“JSON Viewer”/“Formatear JSON” o use el atajo Ctrl + Alt + Shift + M

SourceCookifier #

SourceCookifier analiza el código fuente actual para componentes como clase, función y nombres de variables y los muestra en una vista de árbol en un panel lateral. La navegación entre estos miembros es posible haciendo doble clic en el nombre del componente. El complemento es compatible con una serie de idiomas y las personalizaciones son posibles para idiomas indeterminados o definidos por el usuario. Esto es útil cuando se trabaja con códigos grandes.

De forma predeterminada, el modo de sesión de SourceCookifier está configurado en Modo de archivo único. Otros modos disponibles son el modo de sesión N++ (todos los archivos se abren en el Bloc de notas++) o el modo de sesión de cookies.

Para usar el modo de sesión de cookies:

  1. Arrastre y suelte sus carpetas con el código fuente en la ventana Source Cookifier

  2. Seleccione el tipo de archivos que desea analizar

Notas:

  • Puede guardar y cargar las sesiones de cookies; además, Source Cookifier muestra el historial de las últimas sesiones
  • Al abrir el archivo de una sesión guardada en Notepad ++, SourceCookifier cambiará automáticamente al modo de sesión de cookies y cargará esta sesión.
  • Mantener un modificador de teclado (CTRL, SHIFT o ALT - son todos similares) mientras que arrastrar y soltar la carpeta acelerará el análisis agregando solo INCLUYE y no etiquetas

Otra característica útil es la capacidad de saltar a la definición de un símbolo (por ejemplo, función)

  1. Seleccione el “modo de sesión” correcto: utilice modo cookie si la definición del símbolo está en otro archivo que no está abierto en Notepadd++, o modo de archivo único si la definición está en el mismo archivo
  2. En Notepad++, coloque el cursor en la función/tipo de la que desea obtener la definición y presione CTRL+SHIFT+ENTER para saltar a la definición. Si hay varias definiciones, puede elegir el archivo que desea abrir

  1. Puede presionar ALT+IZQUIERDA para volver a la ubicación anterior (y ALT+DERECHA para ir volver a la definición)

Efectos de texto #

TextFX [SourceForge] es un complemento para conversiones de caracteres avanzadas (caracteres de escape, etc.) y formato de código (código HTML o C++).
