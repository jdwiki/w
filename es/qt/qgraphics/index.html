Panorámica, zoom y rotación con QGraphicsView #

QGraphics se puede usar para organizar escenas complicadas de objetos visuales en un marco que los hace más fáciles de manejar.

Hay tres tipos principales de objetos utilizados en este marco QGraphicsView, QGraphicsScene y QGraphicsItems. QGraphicsItems son los elementos visuales básicos que existen en la escena.

Hay muchos tipos que están prediseñados y se pueden usar, como Elipses, Líneas, Trayectos, Pixmaps, Polígonos, Rectángulos y Texto.

También puede crear sus propios elementos heredando QGraphicsItem. Luego, estos elementos se colocan en un QGraphicsScene, que es básicamente el mundo que planea mirar. Los elementos se pueden mover dentro de la escena, lo que es como si se movieran en el mundo que estás mirando. El posicionamiento y la orientación de los elementos se manejan mediante matrices de transformación llamadas QTransforms. Qt tiene buenas funciones incorporadas, por lo que generalmente no necesita trabajar con QTransforms directamente, sino que llama a funciones como rotar o escalar que crean las transformaciones adecuadas para usted. Luego, la escena se ve desde la perspectiva definida en QGraphicsView (nuevamente con QTransforms), que es la pieza que colocaría en un widget en su interfaz de usuario.

En el siguiente ejemplo, hay una escena muy simple con solo un elemento (un mapa de píxeles), que se coloca en una escena y se muestra en una vista. Al activar la bandera Modo de arrastre, la escena se puede desplazar con el mouse y al usar las funciones de escalar y rotar, se puede escalar hacia adentro y hacia afuera con el desplazamiento del mouse y rotar con las teclas de flecha.

Si desea ejecutar este ejemplo, cree una instancia de View que se mostrará y cree un archivo resource con el prefijo /images que contenga una imagen my_image.png.

#include <QGraphicsView>
#include <QGraphicsScene>
#include <QGraphicsPixmapItem>
#include <QWheelEvent>
#include <QKeyEvent>

class View : public QGraphicsView
{
  Q_OBJECT
public:
  explicit View(QWidget *parent = 0) :
    QGraphicsView(parent)
  {
    setDragMode(QGraphicsView::ScrollHandDrag);

    QGraphicsPixmapItem *pixmapItem = new QGraphicsPixmapItem(QPixmap(":/images/my_image.png"));
    pixmapItem->setTransformationMode(Qt::SmoothTransformation);

    QGraphicsScene *scene = new QGraphicsScene();
    scene->addItem(pixmapItem);
    setScene(scene);
  }

protected Q_SLOTS:
  void wheelEvent(QWheelEvent *event)
  {
    if(event->delta() > 0)
      scale(1.25, 1.25);
    else
      scale(0.8, 0.8);
  }

  void keyPressEvent(QKeyEvent *event)
  {
    if(event->key() == Qt::Key_Left)
      rotate(1);
    else if(event->key() == Qt::Key_Right)
      rotate(-1);
  }
};
