ArrayField un campo específico de PostgreSQL

Sintaxis #

  • desde django.contrib.postgres.fields import ArrayField
  • class ArrayField(base_field, size=Ninguno, **opciones)
  • FooModel.objects.filter(array_field_name__contains=[objetos, a, comprobar])
  • FooModel.objects.filter(array_field_name__contained_by=[objetos, a, comprobar])

Tenga en cuenta que aunque el parámetro size se pasa a PostgreSQL, PostgreSQL no lo aplicará.

Al usar ArrayFields, se debe tener en cuenta esta palabra de advertencia de la documentación de matrices de Postgresql.

Consejo: las matrices no son conjuntos; la búsqueda de elementos de matriz específicos puede ser un signo de mal diseño de la base de datos. Considere usar una mesa separada con un fila para cada elemento que sería un elemento de matriz. esto sera mas facil para buscar, y es probable que se escale mejor para una gran cantidad de elementos.

Consultando la membresía de ArrayField con contiene #

Esta consulta devuelve todos los conos con una bola de chocolate y una bola de vainilla.

VANILLA, CHOCOLATE, MINT, STRAWBERRY = 1, 2, 3, 4  # constants for flavors
choco_vanilla_cones = IceCream.objects.filter(scoops__contains=[CHOCOLATE, VANILLA])

No olvides importar el modelo IceCream desde tu archivo models.py.

También tenga en cuenta que django no creará un índice para ArrayFields. Si va a buscarlos, necesitará un índice y deberá crearlo manualmente con una llamada a RunSQL en su archivo de migraciones.

Un ArrayField básico #

Para crear un ArrayField de PostgreSQL, debemos darle a ArrayField el tipo de datos que queremos que almacene como un campo como su primer argumento. Dado que almacenaremos calificaciones de libros, usaremos FloatField.

 from django.db import models, FloatField
 from django.contrib.postgres.fields import ArrayField
 
 class Book(models.Model):
     ratings = ArrayField(FloatField())

Especificando el tamaño máximo de un ArrayField #

 from django.db import models, IntegerField
 from django.contrib.postgres.fields import ArrayField
 
 class IceCream(models.Model):
     scoops = ArrayField(IntegerField()  # we'll use numbers to ID the scoops
                   , size=6)  # our parlor only lets you have 6 scoops

Cuando usa el parámetro de tamaño, se pasa a postgresql, que lo acepta y luego lo ignora. Por lo tanto, es bastante posible agregar 7 enteros al campo scoops anterior usando la consola postgresql.

Campos de matriz anidados #

Puedes anidar ArrayFields pasando otro ArrayField como base_field.

from django.db import models, IntegerField
from django.contrib.postgres.fields import ArrayField

class SudokuBoard(models.Model):
    numbers = ArrayField(
        ArrayField(
            models.IntegerField(),
            size=9,
        ),
        size=9,
    )

Consultando todos los modelos que contienen cualquier elemento en una lista con contains_by #

Esta consulta devuelve todos los conos con una cucharada de menta o una cucharada de vainilla.

minty_vanilla_cones = IceCream.objects.filter(scoops__contained_by=[MINT, VANILLA])
