Sintaxis #

  • disposición de la tabla: automático | fijado;
  • colapso de borde: separado | colapsar;
  • espacio entre bordes: <longitud> | <longitud> <longitud>;
  • celdas vacías: mostrar | ocultar;
  • lado del título: arriba | abajo;

Estas propiedades se aplican tanto a los elementos <table> (*) como a los elementos HTML que se muestran como display: table o display: inline-table

(*) Los elementos <table> obviamente están diseñados de forma nativa por UA/navegadores como display: table

Las tablas HTML son semánticamente válidas para datos tabulares. No se recomienda utilizar tablas para el diseño. En su lugar, utilice CSS.

disposición de la tabla #

La propiedad table-layout cambia el algoritmo que se utiliza para el diseño de una tabla.

A continuación se muestra un ejemplo de dos tablas configuradas en width: 150px:

La tabla de la izquierda tiene table-layout: auto mientras que la de la derecha tiene table-layout: fixed. El primero es más ancho que el ancho especificado (210 px en lugar de 150 px) pero el contenido encaja. Este último toma el ancho definido de 150px, sin importar si el contenido se desborda o no.

Valor Descripción
automático Este es el valor predeterminado. Define el diseño de la tabla que será determinado por el contenido de sus celdas.
fijo Este valor establece que el diseño de la tabla esté determinado por la propiedad de ancho proporcionada a la tabla. Si el contenido de una celda excede este ancho, la celda no cambiará de tamaño, sino que dejará que el contenido se desborde.

celdas vacías #

La propiedad empty-cells determina si las celdas sin contenido deben mostrarse o no. Esto no tiene efecto a menos que border-collapse se establezca en separate.

A continuación, un ejemplo con dos tablas con diferentes valores establecidos en la propiedad empty-cells:

La tabla de la izquierda tiene “celdas vacías: mostrar” mientras que la de la derecha tiene “celdas vacías: ocultar”. El primero muestra las celdas vacías mientras que el segundo no.

Valor Descripción
mostrar Este es el valor predeterminado. Muestra celdas incluso si están vacías.
ocultar Este valor oculta una celda por completo si no hay contenido en la celda.

Más información:

  • https://www.w3.org/TR/CSS21/tables.html#empty-cells
  • https://developer.mozilla.org/en-US/docs/Web/CSS/empty-cells
  • http://codepen.io/SitePoint/pen/yfhtq
  • https://css-tricks.com/almanac/properties/e/empty-cells/

colapso de borde #

La propiedad border-collapse determina si los bordes de una tabla deben separarse o fusionarse.

A continuación, un ejemplo de dos tablas con diferentes valores para la propiedad border-collapse:

La tabla de la izquierda tiene border-colapso: separar mientras que la de la derecha tiene border-colapso: colapsar.

Valor Descripción
separar Este es el valor predeterminado. Hace que los bordes de la mesa se separen entre sí.
colapso Este valor establece que los bordes de la tabla se fusionen, en lugar de ser distintos.

espacio entre bordes #

La propiedad border-spacing determina el espacio entre celdas. Esto no tiene efecto a menos que border-collapse se establezca en separate.

A continuación, un ejemplo de dos tablas con diferentes valores para la propiedad border-spacing:

La tabla de la izquierda tiene border-spacing: 2px (predeterminado) mientras que la de la derecha tiene border-spacing: 8px.

Valor Descripción
<longitud> Este es el comportamiento predeterminado, aunque el valor exacto puede variar entre navegadores.
<longitud> <longitud> Esta sintaxis permite especificar valores horizontales y verticales separados respectivamente.

lado del título #

La propiedad caption-side determina la posición vertical del elemento <caption> dentro de una tabla. Esto no tiene efecto si dicho elemento no existe.

A continuación, un ejemplo con dos tablas con diferentes valores establecidos en la propiedad caption-side:

La tabla de la izquierda tiene caption-side: top mientras que la de la derecha tiene caption-side: bottom.

Valor Descripción
arriba Este es el valor predeterminado. Coloca el título encima de la tabla.
fondo Este valor coloca el título debajo de la tabla.
