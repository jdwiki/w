Sintaxis #

  • Recorte
  • ruta del clip: <fuente del clip> | [ <forma-básica> || <clip-geometry-box> ] | ninguna
  • Enmascaramiento
  • imagen-máscara: [ninguna | <referencia-máscara> ]#
  • modo máscara: [ <modo máscara> ]#
  • repetición de máscara: [ <estilo de repetición]#
  • posición de la máscara: [ <posición> ]#
  • máscara-clip: [ <geometry-box> | no hay video ]#
  • máscara-origen: [ <geometry-box> ]#
  • tamaño de máscara: [ <bg-size> ]#
  • máscara-compuesto: [ <compositing-operator> ]#
  • máscara: [ <mask-reference> <masking-mode>? || <posición> [ / <tamaño de fondo> ]? || <repetir-estilo> || <cuadro-de-geometría> || [ <cuadro-de-geometría> | sin clip ] || <compositing-operator> ]#

Parámetros #

Parámetro Detalles
clip-fuente Una URL que puede apuntar a un elemento SVG en línea (o) un elemento SVG en un archivo externo que contiene la definición de la ruta del clip.
forma básica Se refiere a uno entre inset(), circle(), elipse() o polygon(). Usando una de estas funciones se define la ruta de recorte. Estas funciones de forma funcionan exactamente igual que en Formas para flotadores
clip-geometría-caja Esto puede tener uno entre content-box, padding-box, border-box, margin-box, fill-box, stroke-box, view-box como valores. Cuando se proporciona sin ningún valor para <basic-shape>, los bordes del cuadro correspondiente se utilizan como ruta de recorte. Cuando se usa con una <basic-shape>, actúa como el cuadro de referencia para la forma.
máscara-referencia Esto puede ser ninguno o una imagen o una URL de referencia a una fuente de imagen de máscara.
estilo de repetición Esto especifica cómo se debe repetir o colocar en mosaico la máscara en los ejes X e Y. Los valores admitidos son repeat-x, repeat-y, repeat, space, round, no-repeat.
modo máscara Puede ser “alfa”, “luminancia” o “automático” e indica si la máscara debe tratarse como una máscara alfa o una máscara de luminancia. Si no se proporciona ningún valor y la referencia de máscara es una imagen directa, se considerará una máscara alfa (o) si la referencia de máscara es una URL, se considerará una máscara de luminancia.
posición Esto especifica la posición de cada capa de máscara y tiene un comportamiento similar a la propiedad background-position. El valor se puede proporcionar en una sintaxis de 1 valor (como top, 10%) o en una sintaxis de 2 valores (como top right, 50% 50%).
caja de geometría Esto especifica el cuadro al que se debe recortar la máscara (área de pintura de máscara) o el cuadro que se debe usar como referencia para el origen de la máscara (área de posicionamiento de máscara) según la propiedad. La lista de valores posibles son content-box, padding-box, border-box, margin-box, fill-box, stroke-box, view-box. La explicación detallada de cómo funciona cada uno de esos valores está disponible en [W3C Spec] (https://www.w3.org/TR/css-masking/#the-mask-clip).
tamaño bg Esto representa el tamaño de cada capa de imagen de máscara y tiene la misma sintaxis que tamaño de fondo. El valor puede ser longitud o porcentaje o automático o cubrir o contener. La longitud, el porcentaje y el auto se pueden proporcionar como un solo valor o como uno para cada eje.
operador de composición Puede ser cualquiera entre añadir, restar, excluir, multiplicar por capa y define el tipo de operación de composición que debe usarse para esta capa con las que están debajo. La explicación detallada de cada valor está disponible en W3C Specs.

CSS Clipping and Masking son conceptos muy nuevos y, por lo tanto, el soporte del navegador para estas propiedades es bastante bajo.

Máscaras: #

En el momento de escribir este artículo (julio de 2016), Chrome, Safari y Opera admiten estas propiedades con el prefijo -webkit-.

Firefox no requiere prefijos, pero solo admite máscaras cuando se usa con elementos máscara SVG. Para los elementos mask de SVG en línea, la sintaxis es mask: url(#msk) mientras que para usar elementos mask en un archivo SVG externo, la sintaxis es mask: url('yourfilepath/yourfilename.svg#msk') . #msk en ambos casos se refiere al id del elemento mask al que se hace referencia. Como se indica en esta respuesta, actualmente Firefox no admite ningún parámetro que no sea mask-reference en la propiedad máscara.

Internet Explorer (y Edge) aún no ofrece soporte para esta propiedad.

La propiedad mask-mode actualmente no es compatible con ningún navegador con o sin prefijos.

Clip-ruta: #

Al momento de escribir (julio de 2016), Chrome, Safari y Opera admiten clip-path cuando la ruta se crea utilizando formas básicas (como círculo, polígono) o la sintaxis url(#clipper) con SVG en línea. No admiten el recorte basado en formas que forman parte de archivos SVG externos. Además, requieren que esté presente el prefijo -webkit.

Firefox solo admite la sintaxis url() para clip-path, mientras que Internet Explorer (y Edge) no ofrecen soporte.

Máscara simple que desvanece una imagen de sólida a transparente #

CSS #

div {
  height: 200px;
  width: 200px;
  background: url(http://lorempixel.com/200/200/nature/1);
  mask-image: linear-gradient(to right, white, transparent); 
}

HTML #

<div></div>

En el ejemplo anterior hay un elemento con una imagen como fondo. La máscara que se aplica a la imagen (usando CSS) hace que parezca que se desvanece de izquierda a derecha.

El enmascaramiento se logra mediante el uso de un “degradado lineal” que va de blanco (a la izquierda) a transparente (a la derecha) como máscara. Como es una máscara alfa, la imagen se vuelve transparente donde la máscara es transparente.

Recorte (círculo) #

CSS: #

div{
  width: 200px;
  height: 200px;
  background: teal;
  clip-path: circle(30% at 50% 50%); /* refer remarks before usage */
}

HTML #

<div></div>

Este ejemplo muestra cómo recortar un div a un círculo. El elemento se recorta en un círculo cuyo radio es del 30 % en función de las dimensiones del cuadro de referencia con su punto central en el centro del cuadro de referencia. Aquí, dado que no se proporciona un <clip-geometry-box> (en otras palabras, un cuadro de referencia), el border-box del elemento se utilizará como cuadro de referencia.

La forma del círculo debe tener un radio y un centro con coordenadas (x,y):

circle(radius at x y)

Recorte (Polígono) #

CSS: #

div{
  width:200px;
  height:200px;
  background:teal;
  clip-path: polygon(0 0, 0 100%, 100% 50%); /* refer remarks before usage */
}

HTML: #

<div></div>

En el ejemplo anterior, se utiliza un trazado de recorte poligonal para recortar el elemento cuadrado (200 x 200) en forma de triángulo. La forma de salida es un triángulo porque la ruta comienza en (es decir, las primeras coordenadas están en) 0 0, que es la esquina superior izquierda del cuadro, luego va a 0 100%, que está en la parte inferior izquierda esquina de la caja y finalmente a 100% 50% que no es más que el punto medio derecho de la caja. Estos caminos se cierran solos (es decir, el punto de inicio será el punto final) y, por lo tanto, la forma final es la de un triángulo.

Esto también se puede usar en un elemento con una imagen o un degradado como fondo.

Uso de máscaras para cortar un agujero en medio de una imagen #

CSS #

div {
  width: 200px;
  height: 200px;
  background: url(http://lorempixel.com/200/200/abstract/6);
  mask-image: radial-gradient(circle farthest-side at center, transparent 49%, white 50%); /* check remarks before using */
}

HTML #

En el ejemplo anterior, se crea un círculo transparente en el centro usando “gradiente radial” y luego se usa como una máscara para producir el efecto de un círculo cortado desde el centro de una imagen.

Usar máscaras para crear imágenes con formas irregulares #

CSS #

div { /* check remarks before usage */
  height: 200px;
  width: 400px;
  background-image: url(http://lorempixel.com/400/200/nature/4);
  mask-image: linear-gradient(to top right, transparent 49.5%, white 50.5%), linear-gradient(to top left, transparent 49.5%, white 50.5%), linear-gradient(white, white);
  mask-size: 75% 25%, 25% 25%, 100% 75%;
  mask-position: bottom left, bottom right, top left;
  mask-repeat: no-repeat;
}

HTML #

<div></div>

En el ejemplo anterior, tres imágenes de “degradado lineal” (que cuando se colocan en sus posiciones adecuadas cubrirían el 100 % x 100 % del tamaño del contenedor) se utilizan como máscaras para producir un corte triangular transparente en la parte inferior de la imagen.
