Sintaxis #

  • opacidad: número (* estrictamente entre 0 y 1) | heredar | inicial | desarmado;

Si no desea aplicar opacidad, puede usar esto en su lugar:

[fondo: rgba(255, 255, 255, 0.6);][1]

[1]: https://www.wikiod.com/es/css/antecedentes#Color de fondo con opacidad

Recursos:

  • MDN: https://developer.mozilla.org/en/docs/Web/CSS/opacity;
  • Transparencia W3C: la propiedad de ‘opacidad’: https://www.w3.org/TR/css3-color/#transparency
  • Compatibilidad con navegador: http://caniuse.com/#feat=css-opacity

Propiedad de opacidad #

La opacidad de un elemento se puede establecer usando la propiedad opacity. Los valores pueden ir desde 0.0 (transparente) hasta 1.0 (opaco).

Uso de ejemplo

<div style="opacity:0.8;">
    This is a partially transparent element
</div>
Valor de la propiedad Transparencia
opacidad: 1.0; opaco
opacidad: 0,75; 25% transparente (75% opaco)
opacidad: 0.5; 50% transparente (50% opaco)
opacidad: 0,25; 75% transparente (25% opaco)
opacidad: 0.0; Transparente

Compatibilidad con IE para opacidad #

Para usar opacity en todas las versiones de IE, el orden es:

.transparent-element {
  /* for IE 8 & 9 */
  -ms-filter:"progid:DXImageTransform.Microsoft.Alpha(Opacity=60)"; // IE8
  /* works in IE 8 & 9 too, but also 5, 6, 7 */
  filter: alpha(opacity=60); // IE 5-7
  /* Modern Browsers */
  opacity: 0.6;
}
