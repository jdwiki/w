Sintaxis #

  • contorno: contorno-color contorno-estilo contorno-ancho | inicial | heredar;
  • contorno-ancho: medio | delgado | grueso | longitud | inicial | heredar;
  • estilo de esquema: ninguno | escondido | punteado | punteado | sólido | doble | surco | cresta | recuadro | comienzo | inicial | heredar;

Parámetros #

Parámetro Detalles
punteado contorno punteado
punteado contorno discontinuo
sólido contorno sólido
doble doble contorno
ranura Contorno acanalado 3D, depende del valor del color del contorno
cresta Contorno estriado 3D, depende del valor del color del contorno
recuadro Contorno de inserción 3D, depende del valor del color del contorno
outset contorno inicial 3D, depende del valor del color del contorno
ninguno sin contorno
oculto esquema oculto

outline ahora se describe en [IU básica] (https://www.w3.org/TR/css-ui-3/#outline-props), un módulo CSS de nivel 3 (ya se describió en REC CSS2. 1)

La propiedad de contorno se define de forma predeterminada en los navegadores para elementos enfocables en el estado :focus. No debe eliminarse, consulte http://outlinenone.com que establece:

¿Qué hace la propiedad de contorno?

Proporciona retroalimentación visual para enlaces que tienen “foco” al navegar por un documento web usando la tecla TAB (o equivalente). Esto es especialmente útil para las personas que no pueden usar un mouse o tienen una discapacidad visual. Si elimina el esquema, está haciendo que su sitio sea inaccesible para estas personas. (…)

Interesantes ejemplos relacionados con Stack Overflow:

  • http://stackoverflow.com/questions/1457849/how-to-remove-the-border-highlight-on-an-input-text-element?rq=1
  • http://stackoverflow.com/questions/71074/how-to-remove-firefoxs-dotted-outline-on-buttons-as-well-as-links?rq=1

Visión general #

El contorno es una línea que rodea el elemento, fuera del borde. A diferencia de borde, los contornos no ocupan ningún espacio en el modelo de caja. Por lo tanto, agregar un contorno a un elemento no afecta la posición del elemento ni de otros elementos.

Además, los contornos pueden no ser rectangulares en algunos navegadores. Esto puede suceder si se aplica outline en un elemento span que tiene texto con diferentes propiedades de font-size dentro. A diferencia de los bordes, los contornos no pueden tener esquinas redondeadas.

Las partes esenciales de outline son outline-color, outline-style y outline-width.

La definición de contorno es equivalente a la definición de borde:

Un contorno es una línea alrededor de un elemento. Se muestra alrededor del margen del elemento. Sin embargo, es diferente de la propiedad de borde.

outline: 1px solid black;

estilo de contorno #

La propiedad outline-style se utiliza para establecer el estilo del contorno de un elemento.

p {
  border: 1px solid black;
  outline-color:blue;
  line-height:30px;
}
.p1{
  outline-style: dotted;
}
.p2{
  outline-style: dashed;
}
.p3{
  outline-style: solid;
}
.p4{
  outline-style: double;
}
.p5{
  outline-style: groove;
}
.p6{
  outline-style: ridge;
}
.p7{
  outline-style: inset;
}
.p8{
  outline-style: outset;
}

HTML

<p class="p1">A dotted outline</p>
<p class="p2">A dashed outline</p>
<p class="p3">A solid outline</p>
<p class="p4">A double outline</p>
<p class="p5">A groove outline</p>
<p class="p6">A ridge outline</p>
<p class="p7">An inset outline</p>
<p class="p8">An outset outline</p>
