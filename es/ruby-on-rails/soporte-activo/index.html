ActiveSupport es una joya de utilidad de herramientas de uso general utilizadas por el resto del marco de Rails.

Una de las principales formas en que proporciona estas herramientas es mediante el parcheado de los tipos nativos de Ruby. Estas se denominan Extensiones principales.

Extensiones principales: acceso a cadenas #

Cadena#en #

Devuelve una subcadena de un objeto de cadena. Misma interfaz que String#[].

str = "hello"
str.at(0)      # => "h"
str.at(1..3)   # => "ell"
str.at(-2)     # => "l"
str.at(-2..-1) # => "lo"
str.at(5)      # => nil
str.at(5..-1)  # => ""

[Cadena#desde][desde] #

Devuelve una subcadena desde la posición dada hasta el final de la cadena.

str = "hello"
str.from(0)  # => "hello"
str.from(3)  # => "lo"
str.from(-2) # => "lo"

Cadena#a #

Devuelve una subcadena desde el principio de la cadena hasta la posición dada. Si la posición es negativa, se cuenta desde el final de la cadena.

str = "hello"
str.to(0)  # => "h"
str.to(3)  # => "hell"
str.to(-2) # => "hell"

from y to se pueden usar en tándem.

str = "hello"
str.from(0).to(-1) # => "hello"
str.from(1).to(-2) # => "ell"

Cadena#primero #

Devuelve el primer carácter o un número determinado de caracteres hasta la longitud de la cadena.

str = "hello"
str.first    # => "h"
str.first(1) # => "h"
str.first(2) # => "he"
str.first(0) # => ""
str.first(6) # => "hello"

Cadena#último #

Devuelve el último carácter, o un número determinado de caracteres desde el final de la cadena contando hacia atrás.

str = "hello"
str.last    # => "o"
str.last(1) # => "o"
str.last(2) # => "lo"
str.last(0) # => ""
str.last(6) # => "hello"

Extensiones principales: conversión de cadena a fecha/hora #

[String#to_time][to_time] #

Convierte una cadena en un valor de tiempo. El parámetro form puede ser :utc o :local, por defecto es :local.

"13-12-2012".to_time               # => 2012-12-13 00:00:00 +0100
"06:12".to_time                    # => 2012-12-13 06:12:00 +0100
"2012-12-13 06:12".to_time         # => 2012-12-13 06:12:00 +0100
"2012-12-13T06:12".to_time         # => 2012-12-13 06:12:00 +0100
"2012-12-13T06:12".to_time(:utc)   # => 2012-12-13 06:12:00 UTC
"12/13/2012".to_time               # => ArgumentError: argument out of range

[Cadena#hasta_la_fecha][hasta_la_fecha] #

Convierte una cadena en un valor de fecha.

"1-1-2012".to_date   # => Sun, 01 Jan 2012
"01/01/2012".to_date # => Sun, 01 Jan 2012
"2012-12-13".to_date # => Thu, 13 Dec 2012
"12/13/2012".to_date # => ArgumentError: invalid date

String#to_datetime #

Convierte una cadena en un valor DateTime.

"1-1-2012".to_datetime            # => Sun, 01 Jan 2012 00:00:00 +0000
"01/01/2012 23:59:59".to_datetime # => Sun, 01 Jan 2012 23:59:59 +0000
"2012-12-13 12:50".to_datetime    # => Thu, 13 Dec 2012 12:50:00 +0000
"12/13/2012".to_datetime          # => ArgumentError: invalid date

Extensiones principales: Exclusión de cadenas #

Cadena#excluir? #

Lo contrario de String#include?

"hello".exclude? "lo" # => false
"hello".exclude? "ol" # => true
"hello".exclude? ?h   # => false

Extensiones principales: filtros de cadena #

[String#squish][squish] #

Devuelve una versión de la cadena dada sin espacios en blanco iniciales o finales, y combina todos los espacios en blanco consecutivos en el interior en espacios individuales. La versión destructiva squish! opera directamente en la instancia de la cadena.

Maneja espacios en blanco ASCII y Unicode.

%{ Multi-line
   string }.squish                   # => "Multi-line string"
" foo   bar    \n   \t   boo".squish # => "foo bar boo"

Cadena#eliminar #

Devuelve una nueva cadena con todas las ocurrencias de los patrones eliminados. La versión destructiva remove! opera directamente en la cadena dada.

str = "foo bar test"
str.remove(" test")                 # => "foo bar"
str.remove(" test", /bar/)          # => "foo "

Cadena#truncar #

Devuelve una copia de una cadena dada truncada en una longitud determinada si la cadena es más larga que la longitud.

'Once upon a time in a world far far away'.truncate(27)
# => "Once upon a time in a wo..."

Pase una cadena o expresión regular :separator para truncar en un corte natural

'Once upon a time in a world far far away'.truncate(27, separator: ' ')
# => "Once upon a time in a..."

'Once upon a time in a world far far away'.truncate(27, separator: /\s/)
# => "Once upon a time in a..."

String#truncate_words #

Devuelve una cadena truncada después de un número determinado de palabras.

'Once upon a time in a world far far away'.truncate_words(4)
# => "Once upon a time..."

Pase una cadena o expresión regular para especificar un separador de palabras diferente

'Once<br>upon<br>a<br>time<br>in<br>a<br>world'.truncate_words(5, separator: '<br>')
# => "Once<br>upon<br>a<br>time<br>in..."

Los últimos caracteres serán reemplazados con la cadena :omisión (predeterminado en “…”)

'And they found that many people were sleeping better.'.truncate_words(5, omission: '... (continued)')
# => "And they found that many... (continued)"

[String#strip_heredoc][heredoc] #

Tiras de sangría en heredocs. Busca la línea no vacía menos sangrada y elimina esa cantidad de espacios en blanco iniciales.

if options[:usage]
  puts <<-USAGE.strip_heredoc
    This command does such and such.

    Supported options are:
      -h         This message
      ...
  USAGE
end

el usuario vería

This command does such and such.

Supported options are:
-h         This message
...

Extensiones principales: Inflexión de cuerdas #

String#pluralizar #

Devuelve la forma plural de la cadena. Opcionalmente, toma un parámetro count y devuelve la forma singular si count == 1. También acepta un parámetro locale para la pluralización específica del idioma.

'post'.pluralize             # => "posts"
'octopus'.pluralize          # => "octopi"
'sheep'.pluralize            # => "sheep"
'words'.pluralize            # => "words"
'the blue mailman'.pluralize # => "the blue mailmen"
'CamelOctopus'.pluralize     # => "CamelOctopi"
'apple'.pluralize(1)         # => "apple"
'apple'.pluralize(2)         # => "apples"
'ley'.pluralize(:es)         # => "leyes"
'ley'.pluralize(1, :es)      # => "ley"

[String#singularize][singularize] #

Devuelve la forma singular de la cadena. Acepta un parámetro locale opcional.

'posts'.singularize            # => "post"
'octopi'.singularize           # => "octopus"
'sheep'.singularize            # => "sheep"
'word'.singularize             # => "word"
'the blue mailmen'.singularize # => "the blue mailman"
'CamelOctopi'.singularize      # => "CamelOctopus"
'leyes'.singularize(:es)       # => "ley"

[String#constantizar][constantizar] #

Intenta encontrar una constante declarada con el nombre especificado en la cadena. Genera un NameError cuando el nombre no está en CamelCase o no está inicializado.

'Module'.constantize  # => Module
'Class'.constantize   # => Class
'blargle'.constantize # => NameError: wrong constant name blargle

String#safe_constantize #

Realiza una constancia pero devuelve nil en lugar de generar NameError.

'Module'.safe_constantize  # => Module
'Class'.safe_constantize   # => Class
'blargle'.safe_constantize # => nil

String#camelizar #

Convierte cadenas a UpperCamelCase de forma predeterminada, si :lower se da como parámetro se convierte en lowerCamelCase en su lugar.

alias: camelcase

Nota: también convertirá / a ::, lo cual es útil para convertir rutas en espacios de nombres.

'active_record'.camelize                # => "ActiveRecord"
'active_record'.camelize(:lower)        # => "activeRecord"
'active_record/errors'.camelize         # => "ActiveRecord::Errors"
'active_record/errors'.camelize(:lower) # => "activeRecord::Errors"

[String#título][título] #

Pone en mayúsculas todas las palabras y reemplaza algunos caracteres en la cadena para crear un título más bonito.

alias: título

'man from the boondocks'.titleize # => "Man From The Boondocks"
'x-men: the last stand'.titleize  # => "X Men: The Last Stand"

Cadena#guion bajo #

Crea una forma subrayada en minúsculas a partir de la expresión en la cadena. El reverso de camelize.

Nota: guión bajo también cambiará :: a / para convertir espacios de nombres en rutas.

'ActiveModel'.underscore         # => "active_model"
'ActiveModel::Errors'.underscore # => "active_model/errors"

[String#desordenar][desgastar] #

Reemplaza guiones bajos con guiones en la cadena.

'puni_puni'.dasherize # => "puni-puni"

[String#desmodular][demod] #

Elimina la parte del módulo de la expresión constante en la cadena.

'ActiveRecord::CoreExtensions::String::Inflections'.demodulize # => "Inflections"
'Inflections'.demodulize                                       # => "Inflections"
'::Inflections'.demodulize                                     # => "Inflections"
''.demodulize                                                  # => ''

String#desconsantizar #

Elimina el segmento más a la derecha de la expresión constante en la cadena.

'Net::HTTP'.deconstantize   # => "Net"
'::Net::HTTP'.deconstantize # => "::Net"
'String'.deconstantize      # => ""
'::String'.deconstantize    # => ""
''.deconstantize            # => ""

String#parametrizar #

Reemplaza caracteres especiales en una cadena para que pueda usarse como parte de una URL ‘bonita’.

"Donald E. Knuth".parameterize # => "donald-e-knuth"

Preserve el caso de los caracteres en una cadena con el argumento :preserve_case.

"Donald E. Knuth".parameterize(preserve_case: true) # => "Donald-E-Knuth"

Un caso de uso muy común para parametrizar es anular el método to_param de un modelo ActiveRecord para admitir slugs de URL más descriptivos.

class Person < ActiveRecord::Base
  def to_param
    "#{id}-#{name.parameterize}"
  end
end

Person.find(1).to_param # => "1-donald-e-knuth"

[String#tableize][tableize] #

Crea el nombre de una tabla como lo hace Rails para modelos a nombres de tablas. Pluraliza la última palabra de la cadena.

'RawScaledScorer'.tableize # => "raw_scaled_scorers"
'ham_and_egg'.tableize     # => "ham_and_eggs"
'fancyCategory'.tableize   # => "fancy_categories"

Cadena#clasificar #

Devuelve una cadena de nombre de clase de un nombre de tabla plural como lo hace Rails para nombres de tabla a modelos.

'ham_and_eggs'.classify # => "HamAndEgg"
'posts'.classify        # => "Post"

String#humanizar #

Pone en mayúscula la primera palabra, convierte los guiones bajos en espacios y quita un ‘_id’ final si está presente.

'employee_salary'.humanize              # => "Employee salary"
'author_id'.humanize                    # => "Author"
'author_id'.humanize(capitalize: false) # => "author"
'_id'.humanize                          # => "Id"

String#upcase_first #

Convierte solo el primer carácter a mayúsculas.

'what a Lovely Day'.upcase_first # => "What a Lovely Day"
'w'.upcase_first                 # => "W"
''.upcase_first                  # => ""

Cadena#clave_extranjera #

Crea un nombre de clave externa a partir de un nombre de clase. Pase el parámetro falso para deshabilitar la adición de _ entre el nombre y el id.

'Message'.foreign_key        # => "message_id"
'Message'.foreign_key(false) # => "messageid"
'Admin::Post'.foreign_key    # => "post_id"
