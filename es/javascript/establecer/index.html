El objeto Set le permite almacenar valores únicos de cualquier tipo, ya sean valores primitivos o referencias a objetos.

Los objetos establecidos son colecciones de valores. Puede iterar a través de los elementos de un conjunto en orden de inserción. Un valor en el Conjunto solo puede ocurrir UNA VEZ; es único en la colección del Set. Los valores distintos se discriminan mediante el algoritmo de comparación SameValueZero.

Especificación estándar sobre el conjunto

Sintaxis #

  • nuevo conjunto ([iterable])
  • mySet.add(valor)
  • miConjunto.clear()
  • mySet.delete(valor)
  • miConjunto.entradas()
  • mySet.forEach(devolución de llamada[, thisArg])
  • mySet.has(valor)
  • miConjunto.valores()

Parámetros #

Parámetro Detalles
iterable Si se pasa un objeto iterable, todos sus elementos se agregarán al nuevo Conjunto. nulo se trata como indefinido.
valor El valor del elemento que se agregará al objeto Set.
devolución de llamada Función a ejecutar para cada elemento.
esteArg Opcional. Valor para usar como este al ejecutar la devolución de llamada.

Debido a que cada valor en el Conjunto tiene que ser único, la igualdad de valores se verificará y no se basará en el mismo algoritmo que el utilizado en el operador ===. Específicamente, para Conjuntos, +0 (que es estrictamente igual a -0) y -0 son valores diferentes. Sin embargo, esto ha cambiado en la última especificación ECMAScript 6. A partir de Gecko 29.0 (Firefox 29/Thunderbird 29/SeaMonkey 2.26) (error 952870) y un Chrome nocturno reciente, +0 y -0 se tratan como el mismo valor en los objetos Set. Además, NaN e indefinido también se pueden almacenar en un Conjunto. NaN se considera lo mismo que NaN (aunque NaN !== NaN).

Crear un conjunto #

El objeto Set le permite almacenar valores únicos de cualquier tipo, ya sean valores primitivos o referencias a objetos.

Puede insertar elementos en un conjunto e iterarlos de manera similar a una matriz de JavaScript simple, pero a diferencia de la matriz, no puede agregar un valor a un Conjunto si el valor ya existe en él.

Para crear un nuevo conjunto:

const mySet = new Set();

O puede crear un conjunto a partir de cualquier objeto iterable para darle valores iniciales:

const arr = [1,2,3,4,4,5];
const mySet = new Set(arr);

En el ejemplo anterior, el contenido del conjunto sería {1, 2, 3, 4, 5}. Tenga en cuenta que el valor 4 aparece solo una vez, a diferencia de la matriz original utilizada para crearlo.

Agregando un valor a un Conjunto #

Para agregar un valor a un Conjunto, use el método .add():

mySet.add(5);

Si el valor ya existe en el conjunto, no se volverá a agregar, ya que los conjuntos contienen valores únicos.

Tenga en cuenta que el método .add () devuelve el conjunto en sí mismo, por lo que puede encadenar llamadas de agregar juntas:

mySet.add(1).add(2).add(3);

Eliminar valor de un conjunto #

Para eliminar un valor de un conjunto, utilice el método .delete():

mySet.delete(some_val);

Esta función devolverá “verdadero” si el valor existía en el conjunto y se eliminó, o “falso” en caso contrario.

Comprobando si existe un valor en un conjunto #

Para verificar si un valor dado existe en un conjunto, use el método .has():

mySet.has(someVal);

Devolverá true si someVal aparece en el conjunto, false de lo contrario.

Borrar un conjunto #

Puedes eliminar todos los elementos de un conjunto usando el método .clear():

mySet.clear();

Obtener la longitud establecida #

Puede obtener la cantidad de elementos dentro del conjunto usando la propiedad .size

const mySet = new Set([1, 2, 2, 3]);
mySet.add(4);
mySet.size; // 4

Esta propiedad, a diferencia de Array.prototype.length, es de solo lectura, lo que significa que no puede cambiarla asignándole algo:

mySet.size = 5;
mySet.size; // 4

En modo estricto, incluso arroja un error:

TypeError: Cannot set property size of #<Set> which has only a getter

Convertir conjuntos en arreglos #

A veces, es posible que necesite convertir un conjunto en una matriz, por ejemplo, para poder usar métodos Array.prototype como .filter(). Para hacerlo, use [Array.from()](https://www.wikiod.com/es/javascript/arreglos#Conversión de objetos tipo matriz en matrices -array#t=201608050855343146834) o desestructuración-asignación:

var mySet = new Set([1, 2, 3, 4]);
//use Array.from
const myArray = Array.from(mySet);
//use destructuring-assignment
const myArray = [...mySet];

Ahora puede filtrar la matriz para que contenga solo números pares y volver a convertirla en Set usando el constructor Set:

mySet = new Set(myArray.filter(x => x % 2 === 0));

mySet ahora contiene solo números pares:

console.log(mySet); // Set {2, 4}

Intersección y diferencia en Conjuntos #

No hay métodos incorporados para la intersección y la diferencia en Conjuntos, pero aún puede lograrlo convirtiéndolos en matrices, filtrando y volviendo a convertir a Conjuntos:

var set1 = new Set([1, 2, 3, 4]),
    set2 = new Set([3, 4, 5, 6]);

const intersection = new Set(Array.from(set1).filter(x => set2.has(x)));//Set {3, 4}
const difference = new Set(Array.from(set1).filter(x => !set2.has(x))); //Set {1, 2}

Conjuntos de iteración #

Puede usar un bucle for-of simple para iterar un conjunto:

const mySet = new Set([1, 2, 3]);

for (const value of mySet) {
  console.log(value); // logs 1, 2 and 3
}

Al iterar sobre un conjunto, siempre devolverá los valores en el orden en que se agregaron por primera vez al conjunto. Por ejemplo:

const set = new Set([4, 5, 6])
set.add(10)
set.add(5) //5 already exists in the set
Array.from(set) //[4, 5, 6, 10]

También hay un método .forEach(), similar a Array.prototype.forEach(). Tiene dos parámetros, callback, que se ejecutará para cada elemento, y thisArg opcional, que se usará como this al ejecutar callback.

callback tiene tres argumentos. Los primeros dos argumentos son el elemento actual de Set (por consistencia con Array.prototype.forEach() y Map.prototype.forEach()) y el tercer argumento es el propio Set.

mySet.forEach((value, value2, set) => console.log(value)); // logs 1, 2 and 3
