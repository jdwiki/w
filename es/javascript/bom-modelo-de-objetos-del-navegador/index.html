El método window.stop() no es compatible con Internet Explorer.

Introducción #

El BOM (Browser Object Model) contiene objetos que representan la ventana y los componentes del navegador actual; objetos que modelan cosas como historia, pantalla del dispositivo, etc.

El objeto superior en BOM es el objeto “ventana”, que representa la ventana o pestaña actual del navegador.

  • Documento: representa la página web actual.
  • Historial: representa páginas en el historial del navegador.
  • Ubicación: representa la URL de la página actual.
  • Navegador: representa información sobre el navegador.
  • Pantalla: representa la información de visualización del dispositivo.

Métodos de objetos de ventana #

El objeto más importante en el Modelo de Objetos del Navegador es el objeto ventana. Ayuda a acceder a información sobre el navegador y sus componentes. Para acceder a estas funciones, tiene varios métodos y propiedades.

Método Descripción
window.alert() Crea un cuadro de diálogo con un mensaje y un botón Aceptar
window.blur() Quitar el foco de la ventana
ventana.cerrar() Cierra una ventana del navegador
window.confirm() Crea un cuadro de diálogo con un mensaje, un botón Aceptar y un botón Cancelar
ventana.getComputedStyle() Obtener estilos CSS aplicados a un elemento
ventana.moverA(x,y) Mover el borde izquierdo y superior de una ventana a las coordenadas proporcionadas
ventana.abrir() Abre una nueva ventana del navegador con la URL especificada como parámetro
ventana.imprimir() Le dice al navegador que el usuario quiere imprimir el contenido de la página actual
ventana.prompt() Crea un cuadro de diálogo para recuperar la entrada del usuario
ventana.scrollBy() Desplaza el documento por el número especificado de píxeles
ventana.scrollTo() Desplaza el documento a las coordenadas especificadas
ventana.establecerIntervalo() Hacer algo repetidamente a intervalos específicos
ventana.setTimeout() Hacer algo después de una cantidad de tiempo específica
ventana.stop() Evitar que la ventana se cargue

Propiedades de objetos de ventana #

El objeto de ventana contiene las siguientes propiedades.

Propiedad Descripción
ventana.cerrada Si la ventana se ha cerrado
ventana.longitud Número de elementos <iframe> en la ventana
ventana.nombre Obtiene o establece el nombre de la ventana
ventana.innerHeight Altura de la ventana
ventana.innerWidth Ancho de la ventana
window.screenX Coordenada X del puntero, relativa a la esquina superior izquierda de la pantalla
window.screenY Coordenada Y del puntero, relativa a la esquina superior izquierda de la pantalla
ventana.ubicación URL actual del objeto de ventana (o ruta del archivo local)
ventana.historial Referencia al objeto de historial para la ventana o pestaña del navegador.
ventana.pantalla Referencia al objeto de pantalla
window.pageXOffset El documento de distancia se ha desplazado horizontalmente
window.pageYOffset El documento de distancia se ha desplazado verticalmente
