Linux + CMake (generador de “Unix Makefiles”) + Qt (opcional) #

Debería tener un proyecto de CMake simple myproject, y vamos a crear un espacio de trabajo de Eclipse fuera de él: myproject/ .git/ CMakeLists.txt src/ main.cpp workspace/ myproject/ Release/ Debug/

Qt (opcional) #

  • Obtenga el CDT de Eclipse más reciente y luego instale el paquete Qt a través de “Ayuda -> Instalar nuevo software”.

espacio de trabajo #

  • Cree un directorio de “área de trabajo” vacío junto con su directorio de origen del proyecto CMake.
  • Inicie Eclipse y cambie a ese directorio de “área de trabajo”.
  • Cree un proyecto C++ (para Qt con Eclipse anterior a Neon: cree un “Proyecto Qt Makefile” y luego elimine el archivo *.pro, makefile y main.cpp de él)

Adjuntar fuentes al proyecto #

  • Vaya a Propiedades del proyecto -> Rutas y símbolos -> Ubicación de origen -> Carpeta de enlaces.
  • Marque “Avanzado” y vincule la carpeta de origen del proyecto CMake así: ../../myproject/src/. Funciona porque el espacio de trabajo está justo fuera del directorio del proyecto CMake.

Generador CMake #

  • Crear carpeta Release en el proyecto.
  • Vaya a la vista “Crear objetivo” (Ctrl+3 y luego escriba “Crear objetivo” si es difícil de encontrar). La vista “Crear objetivo” se parece a la vista del proyecto.
  • Haga clic con el botón derecho en la carpeta “Liberar” y luego haga clic en “Nuevo…”.
  • Desmarque “Igual que el nombre de destino”.
  • Desmarque “Usar la configuración del constructor”.
  • Escriba “Release” en el campo “Target name”.
  • Deje “Hacer objetivo” vacío.
  • Establezca “Comando de compilación” en algo como cmake ../../../myproject/.
  • Haga clic en Aceptar.
  • Haga doble clic en este destino de marca “Release” que se acaba de crear en la carpeta Release. Eso ejecutará la generación de cmake.

Construir #

  • Vaya a Propiedades del proyecto y cree una configuración de “Lanzamiento”.
  • Activar la configuración “Release”.
  • Para la configuración “Lanzamiento”, desmarque “Generar Makefiles automáticamente”.
  • Establezca el directorio de compilación en “Lanzamiento”.
  • Habilitar compilación paralela.

Ahora, puede compilar el proyecto desde Eclipse con Ctrl+b “Construir”.

Volver a ejecutar CMake (para volver a generar los archivos MAKE) #

  • Eliminar todo del directorio “Release”.
  • Vaya a la vista “Hacer objetivo”.
  • Haga doble clic en el objetivo “Liberar”.
