Instalación y configuración #

Para instalar Eclipse, vaya a la página web Descargas de Eclipse donde suele haber un enlace directo para descargar la última versión de Eclipse. Al iniciar Eclipse Mars (versión 4.5), se puede descargar un instalador que lo guía a través del procedimiento de instalación, en lugar de descargar toda la instalación como un archivo comprimido (sin embargo, esta opción aún está disponible). También hay enlaces para descargar paquetes antiguos de Eclipse.

Eclipse viene en varios paquetes diferentes que se dirigen a diferentes usuarios, como se muestra en la siguiente captura de pantalla del instalador. Por ejemplo, Eclipse IDE para desarrolladores de Java contiene herramientas básicas que admiten el desarrollo, la depuración y la creación de aplicaciones Java, así como soporte básico para el control de versiones, como un complemento que permite crear versiones de proyectos usando Git, mientras que Eclipse para desarrolladores de Android proporciona un entorno para crear aplicaciones de Android.

Una vez que se selecciona un paquete, la siguiente página le permite al usuario seleccionar el directorio de instalación, junto con otras opciones. La siguiente captura de pantalla ilustra el procedimiento en una máquina con Windows.

Al hacer clic en el botón Instalar, se iniciará la instalación del paquete de Eclipse en ese directorio.

Si la máquina ya tiene Java correctamente instalado, Eclipse debería iniciarse correctamente y, por lo general, no se requiere configuración ni instalación. Sin embargo, es una buena práctica cambiar algunas opciones de configuración para Eclipse, por ejemplo, para especificar en qué JVM debe ejecutarse Eclipse y configurar la memoria mínima y máxima para esa JVM. Para ello, existe un archivo llamado eclipse.ini en el directorio de instalación, donde se encuentra esta configuración de inicio. Esta página contiene detalles sobre cómo configurar Eclipse usando ese archivo.

Importación de proyectos existentes #

En el menú Archivo, elija la opción ‘Importar…’.

Esto abre el cuadro de diálogo Importar, que solicita el tipo de proyecto/archivo que desea importar. Para un proyecto Java básico, elija ‘Proyectos existentes en el espacio de trabajo’ de la carpeta ‘General’.

A continuación, seleccione el directorio donde se encuentran los proyectos utilizando el botón ‘Examinar’. Todos los proyectos que se pueden importar a Eclipse se mostrarán en la sección ‘Proyectos:’. Si el proyecto ya se ha importado, aún se mostrará pero la casilla de verificación estará deshabilitada.

También puede importar proyectos directamente desde un archivo comprimido eligiendo ‘Seleccionar archivo’ y luego haciendo clic en el botón ‘Examinar’.

Una vez que haga clic en ‘Finalizar’, el proyecto ahora estará visible en su Explorador de proyectos y estará listo para usar.

Instalar Marketplace en Eclipse #

Algunas de las versiones clásicas de eclipse no vienen preinstaladas con Marketplace, esto puede instalarse siguiendo los siguientes pasos:

  1. Ir a Ayuda → Instalar nuevo software
  2. Agregue un nuevo repositorio (sitio especificado a continuación)
  3. Herramientas de propósito general → Cliente de Marketplace
  4. Haga clic en Finalizar y listo.

Sitios de actualización del mercado:

Oxygen - http://download.eclipse.org/releases/oxygen/
Neon - http://download.eclipse.org/releases/neon/
Mars - http://download.eclipse.org/releases/mars/
Luna - http://download.eclipse.org/mpc/luna
Helios - http://download.eclipse.org/releases/helios
Juno - http://download.eclipse.org/releases/juno/

Atajos de teclado útiles #

Administrar archivos y proyectos #

  • Ctrl+Shift+R: Abrir recurso (archivo, carpeta o proyecto)
  • Ctrl+Shift+S : Guardar todos los archivos
  • Ctrl+W : Cierra el archivo actual
  • Ctrl+Shift+W : Cierra todos los archivos

Ventana del editor #

  • F12: Saltar a la ventana del editor
  • Ctrl+E: Muestra la lista de editores abiertos. Use las teclas de flecha e ingrese para cambiar
  • Ctrl+Página Abajo/Arriba : Cambiar al siguiente editor / cambiar al editor anterior
  • Ctrl+M: Maximiza o minimiza la ventana del editor actual

Navegar en el Editor #

  • Ctrl+L : Ir a la línea
  • Ctrl+Q : Saltar a la última ubicación editada
  • Ctrl+Shift+P : Con un corchete seleccionado: salta al corchete de cierre o apertura correspondiente
  • Ctrl+Flecha Abajo/Arriba: Editor de desplazamiento sin cambiar la posición del cursor

Editar texto #

  • Ctrl+D : Borrar línea
  • Alt+Shift+Y : Ajustar el texto seleccionado (ajustar el ancho del texto a la pantalla)
  • Alt+Shift+S : Opciones del menú Open Source
  • Alt+Shift+R : Refactorizar la palabra resaltada en todos los archivos
  • Ctrl+Alt+Arriba/Ctrl+Alt+Abajo : Copia las líneas seleccionadas arriba/abajo.

Buscar y reemplazar #

  • Ctrl+J: escriba un término para buscar y luego use Ctrl+J / Ctrl+shift+J para subir/retroceder
  • Ctrl+K/Ctrl+Shift+K : Ctrl+H luego cierra la ventana de búsqueda. A continuación, busque la aparición anterior/siguiente del término de búsqueda.

Mover un bloque de código #

  • Shift+tab : Mover a la izquierda
  • Alt+Arriba/Alt+Abajo : Mover hacia arriba/abajo.

Crear y ejecutar un programa Java HelloWorld #

Desde la barra de herramientas, abra la Perspectiva de Java.

Crear un nuevo proyecto Java #

Haga clic derecho en el Explorador de paquetes, y desde el menú seleccione Nuevo -> Proyecto Java

En el próximo cuadro de diálogo, ingrese un nombre de proyecto, luego haga clic en Finalizar.

Ahora tienes el nuevo proyecto en tu espacio de trabajo.

Crear una nueva clase Java #

Haga clic con el botón derecho en su proyecto y, en el menú, seleccione Nuevo -> Clase.

En el cuadro de diálogo, escriba el nombre de la clase (debe comenzar con una letra mayúscula), también seleccione la casilla de verificación public static void main(String[] args), luego haga clic en Finalizar.

Ahora tiene el primer archivo Java en su proyecto. El editor abrirá automáticamente este nuevo archivo.

Dentro del método principal escriba algún código para imprimir ¡Hola mundo!.

Ejecuta tu clase Java #

Haga clic con el botón derecho en su clase Java y, en el menú, seleccione Ejecutar como -> Aplicación Java.

Voila, verá la salida de su programa Java en la Consola.
