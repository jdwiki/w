Aumento de la memoria de almacenamiento dinámico máxima para Eclipse #

Para aumentar la cantidad máxima de memoria de almacenamiento dinámico utilizada por Eclipse, edite el archivo eclipse.ini que se encuentra en el directorio de instalación de Eclipse.

Este archivo especifica las opciones para el inicio de Eclipse, como qué JVM usar y las opciones para la JVM. Específicamente, debe editar el valor de la opción JVM -Xmx (o crear una si no existe).

A continuación se muestra una configuración de ejemplo que establece una memoria de almacenamiento dinámico máxima de 1 GB (1024 m). La línea relevante es -Xmx1024m; esto reemplazaría la línea existente -Xmx* en su configuración:

-startup
plugins/org.eclipse.equinox.launcher_1.3.200.v20160318-1642.jar
--launcher.library
C:/Users/user1/.p2/pool/plugins/org.eclipse.equinox.launcher.win32.win32.x86_64_1.1.400.v20160518-1444
-product
org.eclipse.epp.package.java.product
--launcher.defaultAction
openFile
-showsplash
org.eclipse.platform
--launcher.appendVmargs
-vmargs
-Xms256m
-Xmx1024m

Especificando la JVM #

Un problema común que encuentran los usuarios de Eclipse está relacionado con la JVM predeterminada del sistema.

Una situación típica es un Windows de 64 bits que tiene instaladas versiones de Java de 32 y 64 bits, y un Eclipse de 32 bits. Si la versión de 64 bits de Java es la predeterminada del sistema, cuando se inicia Eclipse se muestra un cuadro de diálogo de error.

Especificar la JVM explícitamente en eclipse.ini resolverá esto. La entrada -vm debe agregarse directamente encima de la sección -vmargs.

El siguiente ejemplo muestra cómo utilizar una JVM de 32 bits en un Windows de 64 bits:

-startup
plugins/org.eclipse.equinox.launcher_1.3.200.v20160318-1642.jar
...
-vm
C:/Program Files (x86)/Java/jdk1.7.0_71/bin/javaw.exe
-vmargs
-Xms256m
-Xmx1024m

Cómo configurar el tamaño de fuente de las vistas en Eclipse en Linux #

Eclipse no le brinda la posibilidad de cambiar el tamaño de fuente de las vistas como ‘Explorador de proyectos’ o ‘Servidores’, lo que se ve feo en Linux ya que Eclipse usa el tamaño de fuente predeterminado (escritorio). Pero puede editar archivos de configuración específicos para obtener los tamaños de fuente adecuados.

Para corregir este molesto tamaño de fuente, vaya a /[SU_DIR_INSTALACIÓN]/eclipse/plugins/org.eclipse.ui.themes_[ÚLTIMA_INSTALACIÓN]/css y agrega este contenido…

.MPart Tree{
  font-family: Sans;
  font-size: 8px;
}

al final de los siguientes archivos:

e4_clásico_winxp.css e4_clásico_win7.css
