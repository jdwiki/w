Proceso paso a paso #

Pruebas paralelas con Appium usando GRID: Describiré la forma en que funcionó para mí. Crear Selenium Grid con Appium

  1. Configurar la red de selenio Descargue el archivo jar del servidor independiente de Selenium en el sistema de archivos local Abra su terminal y navegue hasta el directorio donde colocó el archivo jar y ejecute el siguiente comando:

    java -jar selenium-server-standalone-2.53.3.jar -role hub Open http://localhost:4444/grid/console and you should be able to see GRID console in your browser.

  2. Configurar los nodos de Appium Aquí tienes que crear los archivos json. Suponga que desea ejecutar en dos dispositivos y luego cree dos archivos json diferentes. Aquí hay un archivo json, lo tengo como: { “capacidades”: [ { “nombre de la aplicación”: “ONEPLUS A3003”, “nombre del navegador”: “ONEPLUS A3003”, “nombreplataforma”: “ANDROID”, “instancias máximas”: 1 } ], “configuración”: { “ciclo de limpieza”: 2000, “tiempo de espera”: 30000, “proxy”: “org.openqa.grid.selenium.proxy.DefaultRemoteProxy”, “anfitrión”: “127.0.0.1”, “puerto”: 4723, “sesión máxima”: 1, “registrar”: cierto, “ciclo de registro”: 5000, “Puerto concentrador”: 4444, “hubHost”: “su dirección IP” } } guarde el archivo anterior como jasonFile1.json Aquí el nombre de la aplicación será -> Tu móvil->configuraciones->acerca del teléfono->Número de modelo Aquí hubHost será su dirección IP Aquí tenga en cuenta que debe ir a la ubicación cmd predeterminada y luego ejecutar el siguiente comando

aplicación –nodeconfig C:/richa/jasonfile1.json -p 4723 -bp 4724 -U xxxx

i) Tenga en cuenta que debe proporcionar el pasado absoluto del archivo json ubicado ii) puerto como 4723 iii) puerto Bootstrap como 4724 iv) -U por ejemplo, he dado como xxxx

puede encontrar la identificación del dispositivo como -> Su móvil->configuración->estado->Número de serie También puede hacer “dispositivo adb” y verificar la identificación de este dispositivo.

Luego creará Selenium Grid con un dispositivo.

Ahora vuelva a ejecutar el segundo archivo json y comenzará appium Aquí está el segundo archivo json:

{ “capacidades”: [ { “nombre de la aplicación”: “Lenovo K50a40”, “nombre del navegador”: “Lenovo K50a40”, “nombreplataforma”: “ANDROID”, “instancias máximas”: 1 } ], “configuración”: { “ciclo de limpieza”: 2000, “tiempo de espera”: 30000, “proxy”: “org.openqa.grid.selenium.proxy.DefaultRemoteProxy”, “anfitrión”: “127.0.0.1”, “puerto”: 4730, “sesión máxima”: 1, “registrar”: cierto, “ciclo de registro”: 5000, “Puerto concentrador”: 4444, “hubHost”: “su dirección IP” } } guarde el archivo anterior como jasonFile2.json

Inicie el segundo nodo con Lenovo mobile. appium –nodeconfig C:/richa/jasonFile2.json -p 4730 -bp 4731 -U xxxx

Selenium Grid se verá así
