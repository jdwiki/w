Shopify desarrollo local con Theme Kit #

Los temas de Shopify también se pueden desarrollar localmente. Siga los pasos a continuación para configurar el entorno de desarrollo local:

  • Instalar el kit de temas de Shopify
  • Obtenga acceso a la API
  • Use un nuevo tema o configure un tema existente con el kit de temas

La explicación de los pasos anteriores es la siguiente:


Instalar el kit de temas de Shopify


Instalación automática

Si está en Mac o Linux, puede usar el siguiente script de instalación para descargar e instalar automáticamente Theme Kit por usted.

curl -s https://raw.githubusercontent.com/Shopify/themekit/master/scripts/install | sudo python

Cerveza casera

Si tiene instalado homebrew, puede instalar Theme Kit ejecutando los siguientes comandos.

brew tap shopify/shopify
brew install themekit

**Instalación de Windows**


Instalación automática de Powershell

Ejecute los siguientes comandos en Powershell como administrador.

(New-Object System.Net.WebClient).DownloadString("https://raw.githubusercontent.com/Shopify/themekit/master/scripts/install.ps1") | powershell -command -

Manual

  • Cree una carpeta dentro de C:\Program Files\ llamada Theme Kit
  • Descargue el kit de temas (a continuación) y copie el programa extraído en C:\Program Files\Theme Kit
  • Vaya a Panel de control > Sistema y seguridad > Sistema. Otra forma de llegar allí es hacer clic derecho en Mi PC y elegir el elemento de propiedades
  • Busque el botón o enlace llamado Variables de entorno
  • En el segundo panel, busque el elemento llamado Ruta y haga doble clic en él. Esto debería abrir una ventana con un campo de texto rebosante de contenido.
  • Mueva el cursor hasta el final y agregue lo siguiente: ;C:\Program Files\Theme Kit\
  • Haga clic en Aceptar hasta que desaparezcan todas las ventanas.
  • Para verificar que se ha instalado Theme Kit, abra cmd.exe y escriba el tema.

Instalación manual

Descargue el ejecutable de la última versión.

Ventanas 32 bits Descargar

**Obtenga acceso a la API**


Deberá configurar una clave API para agregar a nuestra configuración y crear una conexión entre su tienda y el Kit de temas. La clave API permite que Theme Kit hable y acceda a su tienda, así como a sus archivos de temas.

Para hacerlo, inicie sesión en la tienda Shopify y cree una aplicación privada. En Shopify Admin, ve a Aplicaciones y haz clic en Ver aplicaciones privadas. Desde allí, haga clic en Generar credenciales de API para crear su aplicación privada. Asegúrese de configurar los permisos de las plantillas de tema y los activos de tema para tener acceso de lectura y escritura para generar las credenciales de API adecuadas, luego haga clic en Guardar.

Complete la información en la parte superior y configure los permisos de las plantillas de tema y los activos de tema para acceso de lectura y escritura. Presione Guardar y se le presentará la siguiente pantalla. En él verás tus credenciales de acceso. Por favor tome nota de la contraseña. Lo necesitarás más tarde.


Usar un nuevo tema.

Si está comenzando desde cero y desea obtener un comienzo rápido, ejecute lo siguiente:

theme bootstrap --password=[your-password] --store=[your-store.myshopify.com]

Esto creará un nuevo tema para su tienda en línea a partir de la plantilla Timber. Luego, descargará todos esos activos de Shopify y creará automáticamente un archivo config.yml para usted.


Configurar un tema existente.

Para conectar un tema existente, necesita el número de identificación del tema. La forma más fácil de obtener el número de identificación de su tema es ir al Editor de temas, hacer clic en Editar HTML/CSS y copiar el número de identificación del tema de la URL; tendrá los últimos dígitos después de mystore.myshopify.com/admin/themes/.

Luego, una vez que haya anotado su ID de tema, ejecute los siguientes comandos:

# create configuration
theme configure --password=[your-password] --store=[you-store.myshopify.com] --themeid=[your-theme-id]
# download and setup project in the current directory
theme download

Instalación o Configuración #

Instrucciones detalladas sobre cómo configurar o instalar Shopify.

Configuración del tema local #

Hay algunas formas de configurar una copia local de tu tema de Shopify:

  • Node.js (golpe/gruñido)
  • Rubí
  • Aplicación de temas (solo Mac)

Puede haber algunas otras formas, pero encontré que la forma ruby ​​es la más fácil de trabajar.

Para trabajar localmente en los archivos de temas de Shopify, necesitará algunas cosas:

  • Instalador de Ruby ( http://rubyinstaller.org/downloads/ )
  • Kit de desarrollo ( http://rubyinstaller.org/downloads/ )
  • Certificado SSL - (https://curl.haxx.se/ca/cacert.pem)
  • Joya temática de Shopify - ( https://rubygems.org/gems/shopify_theme/versions/0.0.25 )

instalador rubí #

Lo primero es instalar el instalador de Ruby, usa la última versión.

El proceso de instalación es sencillo, pero le pedirá que agregue Ruby a la ruta global del entorno de Windows y debe hacer clic en la casilla de verificación y agregarlo allí, para que no tenga que hacerlo manualmente después.

Kit de desarrollo #

Debe elegir la versión correcta del DevKit que va con el instalador de Ruby que agregó.

El proceso de instalación también es fácil.

  1. Descomprimes el archivo en C:\DevKit por ejemplo
  2. Haces cmd en esa carpeta y escribes ruby dk.rb init que debería crear un config.yml en el filtro
  3. Si abre el archivo config.yml debería ver en la parte inferior - C:/Ruby23-x64 (dependiendo de dónde instaló su Ruby y qué versión es) si no está allí, debe agregarlo manualmente
  4. En la misma carpeta, luego ejecute el siguiente ruby dk.rb install
  5. Si todo está bien, debería poder ejecutar el siguiente comando: gem install json --platform=ruby que confirmará que su instalación de Ruby está funcionando

También se pueden ver instrucciones más detalladas aquí: https://github.com/oneclick/rubyinstaller/wiki/Development-Kit

Certificado SSL #

Deberá agregar el archivo cacert.pem a las variables globales de su ventana.

  1. Tienes que descargar el siguiente archivo: https://curl.haxx.se/ca/cacert.pem (guárdalo como .pem y no como .pem.txt)
  2. Debe ingresar a la pantalla Variables de entorno y, en Variables del sistema, hacer clic en Nuevo
  3. Para el nombre de la variable, ingrese SSL_CERT_FILE y para el valor de la variable, ingrese la ruta al archivo, por ejemplo, C:\RailsInstaller\cacert.pem.

Joya temática de Shopify #

Abra su cmd e ingrese el siguiente código: gem install shopify_theme

Esto debería instalar la gema del tema de Shopify que necesitaremos.

Si realizó todos los pasos anteriores una vez, ya no los volverá a realizar.

Ahora vienen los pasos que tendrás que hacer cada vez que inicies un nuevo Shopify Project:

  • Crear una aplicación privada (desde la tienda Shopify)
  • Cree un archivo de configuración utilizando la información de la aplicación privada

Crear una aplicación privada #

Debe iniciar sesión en el panel de administración de su tienda Shopify e ir a la pantalla de la aplicación -> Aplicación privada:

https://NOMBRE-DE-TU-TIENDA.myshopify.com/admin/apps/private

Y haga clic en el botón Crear aplicación privada.

Te pedirá un título. (ingrese lo que quiera)

La parte importante es cambiar todos los permisos a Lectura y escritura

Una vez que cambie los permisos y guarde la aplicación, lo redirigirá a una página desde la cual obtendrá la clave API y la contraseña para esta aplicación. (los necesita para el archivo de configuración que crearemos a continuación)

Crear un archivo de configuración #

Vaya a una carpeta local donde desee que estén los archivos del tema y cmd dentro.

Escribe el siguiente comando:

tema configurar API-CLAVE ID DE DOMINIO DE CONTRASEÑA

Dónde:

  • Clave API: se toma de ellos Aplicación privada que creamos
  • Contraseña: se toma de ellos Aplicación privada que creamos
  • Dominio: este es tu dominio NOMBRE-DE-TU-TIENDA.myshopify.com, sin https:// antes
  • ID: esta es la ID del tema, si va a trabajar en el tema en vivo, puede omitir esto, pero si está trabajando en un tema clonado, tendrá que ingresar la ID del tema aquí

Una vez que ejecute el código anterior, debería crear un archivo llamado config.yml donde estará presente toda la información proporcionada.

Ahora debes cmd (en caso de que hayas cerrado la consola) en la carpeta donde está config.yml y escribes el siguiente código: theme download, que debería comenzar a descargar los archivos uno por uno.

Una vez que complete el proceso de descarga, escriba reloj temático y cada vez que modifique los archivos descargados, también actualizará los archivos en la tienda Shopify después de 1-2 segundos. Esto es cierto si agrega archivos también, pero no cuando elimina archivos.

Puede encontrar información más detallada sobre la gema aquí: https://github.com/Shopify/shopify_theme
