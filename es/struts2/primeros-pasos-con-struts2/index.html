Descargar #

Se puede descargar una distribución desde el sitio web de Apache Struts. La distribución completa contiene:

  • el archivo struts2-core.jar
  • dependencias relacionadas
  • aplicaciones de ejemplo
  • una copia de la documentación en formato HTML
  • el código fuente completo.

Solicitud en blanco

La aplicación web en blanco en el directorio de aplicaciones de la distribución es una plantilla.

Incluso hay un archivo por lotes simple en el directorio del código fuente que podemos usar para volver a compilar la aplicación en su lugar.

Podemos hacer una copia de blank.war, implementarla en nuestro contenedor y usar la copia ampliada como base para nuestra aplicación.

Nueva aplicación de Maven Archetype #

Para aquellos de nosotros que usamos Maven como sistema de compilación, podemos usar Maven Archetype para crear una nueva aplicación.

Primero consulta el catálogo de arquetipos maven

mvn archetype:generate -DarchetypeCatalog=http://struts.apache.org/

puedes usar uno de los siguientes achetypes:

El Arquetipo de la Convención en Blanco (struts2-archetype-convention)

Tiene las siguientes características:

  • Validación basada en convenciones
  • Acciones de ejemplo
  • Paquete de recursos a nivel de paquete
  • Examen de la unidad
  • Consciente de Google AppEgine

para descargarlo a través de maven ejecute el comando

mvn archetype:generate -B -DgroupId=com.mycompany.mysystem \
                            -DartifactId=myWebApp \
                            -DarchetypeGroupId=org.apache.struts \
                            -DarchetypeArtifactId=struts2-archetype-convention \
                            -DarchetypeVersion=<CURRENT_STRUTS_VERSION> \
                            -DremoteRepositories=http://struts.apache.org

El arquetipo en blanco (struts2-archetype-blank)

Tiene las siguientes características:

  • Configuración basada en XML, demuestra que incluye un archivo de configuración adicional
  • Acciones de ejemplo
  • Paquete de recursos a nivel de paquete
  • Validación basada en XML
  • Examen de la unidad

para descargarlo a través de maven ejecute el comando

mvn archetype:generate -B -DgroupId=com.mycompany.mysystem \
                            -DartifactId=myWebApp \
                            -DarchetypeGroupId=org.apache.struts \
                            -DarchetypeArtifactId=struts2-archetype-blank \
                            -DarchetypeVersion=<CURRENT_STRUTS_VERSION> \
                            -DremoteRepositories=http://struts.apache.org

El arquetipo de arranque (struts2-archetype-starter)

Tiene las siguientes características:

  • Integración de malla de sitio
  • Ejemplo de acción (ejemplificado tanto a través de Spring como de Struts)
  • Integración de primavera
  • Ejemplo de validación (nivel de acción y alias de acción)
  • Ejemplo de conversión (nivel global y de acción)
  • Paquete de recursos (a nivel global, de acción y de paquete)

para descargarlo a través de maven ejecute el comando

mvn archetype:generate -B -DgroupId=com.mycompany.mysystem \
                            -DartifactId=myWebApp \
                            -DarchetypeGroupId=org.apache.struts \
                            -DarchetypeArtifactId=struts2-archetype-starter \
                            -DarchetypeVersion=<CURRENT_STRUTS_VERSION> \
                            -DremoteRepositories=http://struts.apache.org

El arquetipo AngularJS (struts2-archetype-angularjs)

Tiene las siguientes características:

  • Configuración de acciones basada en convenciones
  • Acciones de ejemplo con JSON Result
  • Ejemplo para la integración de AngularJS y Struts2

para descargarlo a través de maven ejecute el comando

mvn archetype:generate -B -DgroupId=com.mycompany.mysystem \
                            -DartifactId=myWebApp \
                            -DarchetypeGroupId=org.apache.struts \
                            -DarchetypeArtifactId=struts2-archetype-angularjs \
                            -DarchetypeVersion=<CURRENT_STRUTS_VERSION> \
                            -DremoteRepositories=http://struts.apache.org

El Portlet Blank Archetype (struts2-archetype-portlet)

Tiene las siguientes características:

  • Ejemplos de modo Ver, Editar y Ayuda
  • Formulario simple para preferencias en el modo de edición.
  • Se puede implementar como una aplicación de servlet o portlet
  • Puede usar el complemento Maven Jetty para implementar con el perfil incrustado de pluto (use ‘mvn jetty:run -Ppluto-embedded’, luego acceda http://localhost:8080/<Id. de artefacto>/pluto/index.jsp)

para descargarlo a través de maven ejecute el comando

mvn archetype:generate -B -DgroupId=com.mycompany.mysystem \
                            -DartifactId=myWebApp \
                            -DarchetypeGroupId=org.apache.struts \
                            -DarchetypeArtifactId=struts2-archetype-portlet \
                            -DarchetypeVersion=<CURRENT_STRUTS_VERSION> \
                            -DremoteRepositories=http://struts.apache.org

El arquetipo de la base de datos de portlet (struts2-archetype-dbportlet)

Tiene las siguientes características:

  • Utiliza Spring y Hsql para mostrar una consulta de base de datos real
  • Almacenamiento en caché incorporado de los resultados de la consulta
  • Ejemplos de modo Ver, Editar y Ayuda
  • Formulario simple para preferencias en el modo de edición.
  • Se puede implementar como una aplicación de servlet o portlet
  • Puede usar el complemento Maven Jetty para implementar como una aplicación web de servlet

para descargarlo a través de maven ejecute el comando

mvn archetype:generate -B -DgroupId=com.mycompany.mysystem \
                            -DartifactId=myWebApp \
                            -DarchetypeGroupId=org.apache.struts \
                            -DarchetypeArtifactId=struts2-archetype-dbportlet \
                            -DarchetypeVersion=<CURRENT_STRUTS_VERSION> \
                            -DremoteRepositories=http://struts.apache.org

El arquetipo del complemento (struts2-archetype-plugin)

Tiene las siguientes características:

  • Ejemplo nuevo tipo de resultado
  • Ejemplo de configuración basada en XML

para descargarlo a través de maven ejecute el comando

mvn archetype:generate -B -DgroupId=com.mycompany.mysystem \
                            -DartifactId=myPlugin \
                            -DarchetypeGroupId=org.apache.struts \
                            -DarchetypeArtifactId=struts2-archetype-plugin \
                            -DarchetypeVersion=<CURRENT_STRUTS_VERSION> \
                            -DremoteRepositories=http://struts.apache.org
