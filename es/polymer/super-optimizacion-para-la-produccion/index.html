Una vez que haya terminado su proyecto, se preguntará cómo cargará esas importaciones de HTML 100 en su servidor web e incluso si lo hace, cuántas horas tardará su sitio en cargarse para un solo cliente. En este tema, verá cómo convertir el desorden de desarrollo en archivos html y js únicos refinados.

Sintaxis #

  • npm install: guarda paquetes usando Node.js Package Manager
  • npm install -g: guarda paquetes de npm como globales (útil para paquetes de interfaz de línea de comandos)
  • cd: establece el foco de la línea de comandos en una biblioteca específica en la que se pueden realizar procesos
  • vulcanizar: reduce todas las importaciones de HTML a un solo archivo
  • Crisper: Convierte JS en línea en un archivo HTML a un archivo JS externo

Instalación de todas las herramientas necesarias #

Ejecute el siguiente comando uno por uno:

npm install -g vulcanize crisper

Usar #

  • Vulcanizar: tritura todos los archivos de importación HTML en un solo archivo
  • Crisper: extrae js en línea a su propio archivo

Nota: Es posible que los usuarios de Ubuntu necesiten prefijar el comando anterior con sudo.

Poniendolo todo junto #

Ponga todas las importaciones html en sus archivos en un solo archivo elements.html. No se preocupe si un archivo se importa más de una vez, se reducirá a una sola importación.

Correr vulcanizar y nítido #

en su archivo elements.html, ejecute los siguientes comandos:

cd PATH/TO/IMPORTFILE/
vulcanize elements.html -o elements.vulc.html --strip-comments --inline-css --inline-js
crisper --source elements.vulc.html --html build.html --js build.js

Vulcanice el código fuente recuperado de todas las importaciones, luego reemplazó las importaciones por su código fuente.
Crisper sacó todo el js del archivo elements.vulc.html, lo colocó en un solo archivo build.js, estableció una etiqueta script que hace referencia al archivo build.js en el archivo build.html

Minimizando los archivos #

  • Abra [minificador HTML] (https://kangax.github.io/html-minifier/)

  • Abra build.html

  • Copiar todo su código

  • En la primera área de texto de Minfier HTML, pegue el código que copió de build.html

  • Haz clic en el botón Minimizar

  • En la segunda área de texto, aparecerá el código minimizado. Entendido

  • Cree un archivo build.min.html y pegue todo su código copiado en él

  • Abra [JSCompress] (https://jscompress.com/)

  • Seleccione la segunda pestaña, es decir, Cargar archivos JavaScript

  • Haga clic en Elegir archivos

  • Seleccione el archivo build.js

  • Haz clic en el botón Comprimir

  • Descargue el archivo como build.js en el mismo directorio que build.min.html

importando build.min.html #

Elimine todas las importaciones anteriores de los archivos HTmL desde los que copió y pegó las importaciones. Reemplazar las importaciones con

<link rel="import" href="PATH/TO/IMPORTFILE/build.min.html">
