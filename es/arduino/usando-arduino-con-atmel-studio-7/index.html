Usando Arduino con Atmel Studio 7

  • Ejemplo de boceto importado de Atmel Studio 7

En esta página

  • Ejemplo de boceto importado de Atmel Studio 7

Configuración #

  • Descargue e instale Atmel Studio 7 desde aquí.
  • Compra un depurador. Puede arreglárselas con un programador de ISP, pero si desea capacidades de depuración, que es una de las grandes ventajas de usar Atmel Studio, querrá un depurador. Recomiendo Atmel ICE, ya que proporciona capacidades de depuración para arduinos basados ​​en AVR (como Uno, pro mini, etc.) y Arduinos basados ​​en ARM, como Zero y Due. Si tiene un presupuesto ajustado, puede obtenerlo sin la caja de plástico y tenga cuidado de no sorprenderlo.

Conexiones #

  • Para el Uno, utilice el cable ICSP de 6 pines. Enchufe un lado en el Uno como se muestra. Enchufe el otro lado en el puerto AVR del depurador.

Para el Arduino Pro Mini, use el cable mini squid como se muestra, conectando nuevamente el otro lado del puerto AVR del depurador.

Consideraciones de depuración #

Para la depuración con el Uno, deberá cortar el rastro de activación de reinicio (siempre puede volver a soldarlo para usarlo con el IDE de Arduino):

Usando el Pro Mini, si tiene la intención de conectar el puerto serie a su computadora usando una placa FTDI, no conecte la línea DTR, ya que interferirá con la interfaz de depuración de cable serie (SWD) de Atmel. Simplemente conecto alimentación, tierra, Tx y Rx como se muestra a continuación. Rx y Tx en Arduino van a Tx y Rx, respectivamente, en la placa FTDI. Algunas placas FTDI están etiquetadas de manera diferente, por lo que si el puerto serie no funciona, intercambie Rx y Tx.

Deberá proporcionar energía por separado al Arduino porque el depurador no lo alimentará. Esto se puede hacer en el Pro Mini a través de la placa FTDI como se muestra arriba, o con un cable USB o adaptador de CA en el Uno.

Configuración del programa #

Conecte Atmel ICE a su computadora, inicie Atmel Studio y ahora puede importar un proyecto Arduino existente.

En Atmel Studio, vaya a Archivo -> Nuevo -> Proyecto y seleccione “Crear proyecto a partir del boceto de Arduino”. Rellene las opciones, incluidos los menús desplegables de la placa y el dispositivo.

Vaya a Proyecto -> Propiedades de yourProjectName, haga clic en Herramienta, seleccione Atmel ICE en depurador/programador y debugWire en interfaz. Vaya a Depurar -> Iniciar depuración y romper. Debería ver una advertencia y se le preguntará si desea configurar el fusible DWEN. Elija Aceptar, desenchufe el Arduino de la alimentación y vuelva a enchufarlo. Puede detener la depuración haciendo clic en el botón cuadrado rojo y comenzar haciendo clic en el botón triangular verde. Para devolver el Arduino a un estado en el que se puede usar en el IDE de Arduino, mientras realiza la depuración, elija Depurar -> deshabilitar debugWIRE y cierre.

Tenga en cuenta que cualquier función que agregue debe incluir también un prototipo de función (el bucle y la configuración no los necesitan). Puede ver las que agregó Atmel Studio en la parte superior del boceto si había alguna función cuando importó su proyecto a Atmel Studio (consulte el código de muestra, por ejemplo).

La compatibilidad con C++ 11 está habilitada de forma predeterminada en Arduino 1.6.6 y superior. Esto proporciona más funciones del lenguaje C++ y habilitarlo puede aumentar la compatibilidad con el sistema Arduino. Para habilitar C++11 en Atmel Studio 7, haga clic con el botón derecho en el archivo de su proyecto, seleccione propiedades, haga clic en ToolChain a la izquierda, haga clic en Varios en AVR/GNU C++ Compiler y coloque ```-std=c++11` `` en el campo Otras banderas.

Para incluir bibliotecas en su boceto #

Copie el archivo de la biblioteca .cpp en C:\Users\YourUserName\Documents\Atmel Studio\7.0\YourSolutionName\YourProjectName\ArduinoCore\src\core, luego en Atmel Studio, abra la ventana Solution Explorer, haga clic con el botón derecho en la carpeta Arduino Core/src/core, elija agregar -> elemento existente y elija el archivo que agregó. Haga lo mismo con el archivo de biblioteca .h y la carpeta YourProjectName/Dependances.

Para agregar la ventana de terminal #

Siempre puede tener el IDE de Android abierto y usar esa ventana Serial (simplemente seleccione el puerto serial correcto); sin embargo, para agregar una ventana Serial integrada a Atmel Studio, vaya a Herramientas -> Extensiones y actualizaciones, haga clic en Descargas disponibles y busque Terminal Window o Terminal para Atmel Studio e instálelo. Una vez instalado, vaya a Ver -> Ventana de terminal.

Beneficios #

Programar Arduino con un IDE moderno como Atmel Studio 7 le brinda numerosas ventajas sobre el IDE de Arduino, que incluyen depuración, finalización automática, salto a definición y declaración, navegación hacia adelante/atrás, marcadores y opciones de refactorización, por nombrar algunas.

Puede configurar combinaciones de teclas yendo a Herramientas -> Opciones -> Entorno -> Teclado. Algunos que realmente aceleran el desarrollo son:

  • Edit.CommentSelection, Edit.UncommentSelection
  • Ver.Navegar hacia adelante, Ver.Navegar hacia atrás
  • Edit.MoveSelectedLinesUp, Edit.MoveSelectedLinesDown
  • Edit.GoToDefinition

Ejemplo de boceto importado de Atmel Studio 7 #

Este es un ejemplo de cómo se ve un simple boceto de Arduino después de importarlo a Atmel Studio. Atmel Studio agregó las secciones generadas automáticamente en la parte superior. El resto es idéntico al código Arduino original. Si expande el proyecto ArduinoCore que se creó y busca en la carpeta src -> core, encontrará main.cpp, el punto de entrada para el programa. Allí puede ver la llamada a la función de configuración de Arduino y un bucle interminable que llama a la función de bucle de Arduino una y otra vez.

/* Begining of Auto generated code by Atmel studio */
#include <Arduino.h>
/* End of auto generated code by Atmel studio */


// Beginning of Auto generated function prototypes by Atmel Studio
void printA();
// End of Auto generated function prototypes by Atmel Studio

void setup() {
  Serial.begin(9600);
}

void loop() {
    printA();
}

void printA() {
    Serial.println("A");
}
