Introducción #

Artisan es una utilidad que puede ayudarlo a realizar tareas repetitivas específicas con comandos bash. Cubre muchas tareas, que incluyen: trabajar con bases de datos migraciones y siembra, borrar caché, crear archivos necesarios para la configuración de Autenticación, haciendo nuevos controladores, modelos, clases de eventos y mucho más.

Artisan es el nombre de la interfaz de línea de comandos incluida con Laravel. Proporciona una serie de comandos útiles para su uso mientras desarrolla su aplicación.

Para ver una lista de todos los comandos de Artisan disponibles, puede usar el comando de lista:

php artisan list

Para obtener más información sobre cualquier comando disponible, solo anteponga su nombre con la palabra clave ayuda:

php artisan help [command-name]

Listar todas las rutas registradas filtradas por múltiples métodos #

php artisan route:list --method=GET --method=POST

Esto incluirá todas las rutas que acepten los métodos GET y POST simultáneamente.

Creando y registrando nuevo comando artesanal #

Puede crear nuevos comandos a través de

php artesanal hacer: comando [nombre del comando]

Entonces esto creará la clase de comando [commandName] dentro del directorio app/Console/Commands.

dentro de esta clase encontrará las variables protected $signature y protected $description, que representan el nombre y la descripción de su comando que se utilizará para describir su comando.

después de crear el comando, puede registrar su comando dentro de la clase app/Console/Kernel.php donde encontrará la propiedad commands.

para que pueda agregar su comando dentro de la matriz $command como:

protected $commands = [
    Commands\[commandName]::class
];

y luego puedo usar mi comando a través de la consola.

entonces, como ejemplo, he nombrado mi comando como

protected $signature = 'test:command';

Así que cada vez que corra

php artisan test:command

llamará al método handle dentro de la clase que tiene la firma test:command.

Ejecutar comandos de Laravel Artisan usando código PHP #

También puede usar los comandos de Laravel Artisan desde sus rutas o controladores.

Para ejecutar un comando usando código PHP:

Artisan::call('command-name');

Por ejemplo,

Artisan::call('db:seed');
