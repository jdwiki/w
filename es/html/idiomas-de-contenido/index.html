Sintaxis #

  • <element lang="code_language">   <!– El código de idioma debe tener el formato [ISO 639-1](https://en.wikipedia.org/wiki/ISO_639-1 ) –>

El valor del atributo lang debe ser una etiqueta de idioma BCP 47 válida o la cadena vacía (si se desconoce el idioma).

Las etiquetas de idioma BCP 47 se enumeran en el [Registro de subetiquetas de idioma de IANA](http://www.iana.org/assignments/language -subetiqueta-registro).

Accesibilidad #

Los criterios de éxito relevantes de las WCAG 2.0 son:

  • 3.1.1 Idioma de la página
  • 3.1.2 Idioma de las partes

Las Técnicas WCAG 2.0 relacionadas son:

  • H57: Uso de atributos de idioma en el elemento html
  • H58: Uso de atributos de lenguaje para identificar cambios en el lenguaje humano

Idioma del documento base #

Es una buena práctica declarar el idioma principal del documento en el elemento html:

<html lang="en">

Si no se especifica ningún otro atributo lang en el documento, significa que todo (es decir, el contenido del elemento y los valores de texto del atributo) está en ese idioma.

Si el documento contiene partes en otros idiomas, estas partes deben tener sus propios atributos lang para “sobrescribir” la declaración del idioma.

Idioma del elemento #

El atributo lang se usa para especificar el idioma del contenido del elemento y los valores del texto del atributo:

<p lang="en">The content of this element is in English.</p>

<p lang="en" title="The value of this attribute is also in English.">The content of this element is in English.</p>

La declaración de idioma se hereda:

<div lang="en">
  <p>This element contains English content.</p>
  <p title="This attribute, too.">Same with this element.</p>
</div>

Elementos con múltiples idiomas #

Puede “sobrescribir” una declaración de idioma:

<p lang="en">This English sentence contains the German word <span lang="de">Hallo</span>.</p>

Manejo de atributos con diferentes idiomas #

Puede “sobrescribir” la declaración de idioma de un elemento principal introduciendo cualquier elemento además de applet, base, basefont, br, frame, frameset, hr, iframe, meta , param, script (de HTML 4.0) con un atributo propio lang:

<p lang="en" title="An English paragraph">
    <span lang="de" title="A German sentence">Hallo Welt!</span>
</p>

URL regionales #

Es posible agregar el atributo hreflang a los elementos <a> y <area> que crean hipervínculos. Tal especifica el idioma del recurso enlazado. El idioma definido debe ser una etiqueta de idioma BCP 471 válida.

<p>
    <a href="example.org" hreflang="en">example.org</a> is one of IANA's example domains.
</p>
