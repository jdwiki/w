Sintaxis #

  • <script type="text/javascript"> //algún código</script>
  • <script type="text/javascript" src="URL"></script>
  • <script type="text/javascript" src="URL" asíncrono>//código asíncrono</script>

Parámetros #

Atributo Detalles
fuente Especifica la ruta a un archivo JavaScript. Ya sea una URL relativa o absoluta.
tipo Especifica el tipo MIME. Este atributo es obligatorio en HTML4, pero opcional en HTML5.
asíncrono Especifica que el script se ejecutará de forma asíncrona (solo para scripts externos). Este atributo no requiere ningún valor (excepto XHTML).
aplazar Especifica que la secuencia de comandos se ejecutará cuando la página haya terminado de analizarse (solo para secuencias de comandos externas). Este atributo no requiere ningún valor (excepto XHTML).
juego de caracteres Especifica la codificación de caracteres utilizada en un archivo de script externo, p. UTF-8
origen cruzado Cómo maneja el elemento las solicitudes de origen cruzado
nonce Nonce criptográfico utilizado en las comprobaciones de Content Security Policy CSP3

Si el código JavaScript incrustado (archivo) se usa para manipular https://www.wikiod.com/es/javascript Elementos, coloque sus etiquetas <script></script> a la derecha ** antes de la etiqueta de cierre </body>** o use métodos o bibliotecas de JavaScript (como jQuery para manejar una variedad de navegadores) que asegure que DOM está leído y listo para ser manipulado.

Manejo de Javascript deshabilitado #

Es posible que el navegador del cliente no sea compatible con Javascript o que tenga deshabilitada la ejecución de Javascript, quizás por razones de seguridad. Para poder decirles a los usuarios que se supone que se debe ejecutar un script en la página, se puede usar la etiqueta <noscript>. El contenido de <noscript> se muestra cada vez que Javascript está deshabilitado para la página actual.

<script>
  document.write("Hello, world!");
</script>
<noscript>This browser does not support Javascript.</noscript>

Vincular a un archivo JavaScript externo #

<script src="example.js"></script>

El atributo src funciona como el atributo href en las anclas: puede especificar una URL absoluta o relativa. El ejemplo anterior vincula a un archivo dentro del mismo directorio del documento HTML. Esto generalmente se agrega dentro de las etiquetas <head> en la parte superior del documento html

Incluyendo directamente el código JavaScript #

En lugar de vincular a un archivo externo, también puede incluir el código JS tal cual en su HTML:

<script>
// JavaScript code
</script>

Incluyendo un archivo JavaScript que se ejecuta de forma asíncrona #

<script type="text/javascript" src="URL" async></script>
