Tutorial de biztalk

Microsoft BizTalk Server es un producto de servidor para la gestión de procesos empresariales y la integración de aplicaciones empresariales, desarrollado por Microsoft.

BizTalk permite a las organizaciones integrar sus procesos comerciales y gestionarlos mediante el intercambio de documentos comerciales electrónicos (como órdenes de compra, facturas, notas de entrega, etc.) entre aplicaciones en XML, JSON y otros formatos. , así como fuera de los límites de la organización . BizTalk admite varios formatos EDI estándar, como EDIFACT, X12 y RosettaNet, y hay muchos adaptadores disponibles para facilitar las conexiones a varias plataformas (incluidas SAP y PeopleSoft) a través de varios protocolos, como HTTP, FTP y SMTP.

BizTalk Server utiliza la tecnología Microsoft .NET y es compatible con SOAP, RESTful y servicios web. El software para BizTalk Server se realiza a través de Visual Studio .NET.

Un producto maduro en su novena versión, BizTalk Server 2013 R2 brinda una solución que permite a las organizaciones conectar sistemas dispares más fácilmente. Incluyendo más de 25 adaptadores multiplataforma y una sólida infraestructura de mensajería, BizTalk Server proporciona conectividad entre los sistemas centrales tanto dentro como fuera de su organización. Además de la funcionalidad de integración, BizTalk también proporciona mensajería sólida y duradera, un motor de reglas comerciales (BRE), conectividad EDI, monitoreo de actividad comercial (BAM), capacidades RFID y conectividad IBM Host/Mainframe.

El sistema de mensajería de BizTalk se basa principalmente en XML, ya que todos los mensajes entrantes a este Middleware se convierten en un mensaje XML legible, a través de una canalización. BizTalk luego le permite desarrollar su propia lógica para procesar los mensajes a través de la aplicación. Los tres tipos principales de procesamiento se realizan a través de orquestaciones, mapas y código XSLT.

BizTalk Server 2013 R2 agregó más capacidades de integración en la nube y es compatible con REST

BizTalk Server 2016 Developer Edition está disponible para suscriptores de MSDN o a través de Visual Studio Dev Essentials En Azure, puede obtener un VHD precompilado con Biztalk 2016, SQL Server 2016 y Visual Studio 2015
