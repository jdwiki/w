Tutorial angular

Angular (comúnmente conocido como “Angular 2+” o “Angular 2”) es un marco web front-end de código abierto basado en TypeScript dirigido por el Equipo angular en Google y por una comunidad de individuos y corporaciones para abordar todas las partes del flujo de trabajo del desarrollador mientras crea aplicaciones web complejas. Angular es una reescritura completa del mismo equipo que creó AngularJS. ¹

El marco consta de varias bibliotecas, algunas de ellas básicas (@angular/core por ejemplo) y otras opcionales (@angular/animations).

Usted escribe aplicaciones Angular al componer HTML templates con marcado Angularizado, escribir component clases para administrar esas plantillas, agregar lógica de aplicación en services y empaquetar componentes y servicios en [modules] 11.

Luego inicia la aplicación mediante bootstrapping el módulo raíz. Angular se hace cargo, presenta el contenido de su aplicación en un navegador y responde a las interacciones del usuario de acuerdo con las instrucciones que ha proporcionado.

Podría decirse que la parte más fundamental del desarrollo de aplicaciones Angular son los componentes. Un componente es la combinación de una plantilla HTML y una clase de componente que controla una parte de la pantalla. Aquí hay un ejemplo de un componente que muestra una cadena simple:

src/app/app.component.ts

importar {Componente} desde ‘@angular/core’;

@Componente({ selector: ‘mi-aplicación’, plantilla: <h1>Hola {{nombre}}</h1> }) exportar clase AppComponent { nombre = ‘Angular’; }

Cada componente comienza con una función decoradora @Component que toma un objeto metadatos. El objeto de metadatos describe cómo funcionan juntas la plantilla HTML y la clase de componente.

La propiedad selector le dice a Angular que muestre el componente dentro de una etiqueta personalizada <my-app> en el archivo index.html.

index.html (dentro de la etiqueta cuerpo)

<mi-aplicación>Cargando contenido de AppComponent aquí…</mi-aplicación>

La propiedad de plantilla define un mensaje dentro de un encabezado <h1>. El mensaje comienza con “Hola” y termina con {{nombre}}, que es una expresión angular enlace de interpolación. En tiempo de ejecución, Angular reemplaza {{name}} con el valor de la propiedad name del componente. El enlace de interpolación es una de las muchas características de Angular que descubrirá en esta documentación. En el ejemplo, cambie la propiedad name de la clase de componente de 'Angular' a 'World' y vea qué sucede.

Este ejemplo está escrito en TypeScript, un superconjunto de JavaScript. Angular usa TypeScript porque sus tipos facilitan el apoyo a la productividad del desarrollador con herramientas. Además, casi todo el soporte es para TypeScript, por lo que usar JavaScript simple para escribir su aplicación será difícil. Sin embargo, es posible escribir código angular en JavaScript; esta guía explica cómo.
