Tutorial de WordPress

WordPress es un sistema de gestión de contenido (CMS) de código abierto que se utiliza para crear y administrar sitios web. WordPress es el CMS más popular en Internet por una milla de país, impulsando [alrededor de la mitad] 2 de todos los sitios web de CMS en el momento de escribir este artículo y aproximadamente [una cuarta parte de todos los sitios web] 11 en Internet.

WordPress comenzó su vida como una plataforma para blogs, pero ha evolucionado a lo largo de los años para adaptarse a la mayoría de los tipos de sitios web. La interfaz se puede usar sin conocimientos de codificación, lo que la hace popular entre principiantes y desarrolladores que desean capacitar a sus clientes para que administren su propio sitio web.

Otro factor importante en la popularidad de WordPress es su flexibilidad, principalmente debido a los sistemas de temas y complementos del núcleo. El sistema de complementos facilita la ampliación de la funcionalidad principal sin modificar el código principal. De manera similar, el sistema de temas facilita cambiar el diseño y la estética del sitio web. Ahora hay miles de complementos y temas de WordPress gratuitos y premium disponibles. Muchos de estos se encuentran en wordpress.org repositorio de complementos y repositorio de temas respectivamente.

WordPress es desarrollado por su propia comunidad, pero está fuertemente asociado con la empresa Automattic, que emplea a muchos de los principales desarrolladores de WordPress.

Código

WordPress se basa en el lenguaje de secuencias de comandos del servidor PHP y el lenguaje de consulta MySQL. WordPress usa MySQL como un almacén de datos para el contenido y la configuración del usuario. El PHP clasifica los datos de contenido en una página web HTML con todos los activos necesarios.

wordpress.com frente a wordpress.org

Puede usar WordPress registrándose en el servicio de Automattic wordpress.com y alojando su sitio web en sus servidores, o puede descargar el software de WordPress desde wordpress.org y alojar su sitio web en un servidor bajo su control. La primera opción es fácil pero no puede editar ningún código de sitio. Solo puede realizar cambios a través de la interfaz de WordPress. La segunda opción requiere más trabajo pero le brinda flexibilidad para hacer lo que quiera con el código de su sitio web. Si es un usuario de StackOverflow, probablemente opte por la segunda opción.

Fuente abierta

WordPress es un software de código abierto, lo que significa que es de uso gratuito y cualquiera puede ver el código fuente y contribuir con él. Los posibles contribuyentes pueden comenzar leyendo la página de contribuciones del códice de WordPress..

Los errores se pueden informar enviando un error en el rastreador de tickets de WordPress.

Documentación

WordPress está documentado oficialmente en el WordPress Codex en WordPress.org. Los desarrolladores que trabajan con WordPress estarán especialmente interesados ​​en la sección Developer Codex y la sección Developer Reference de wordpress.org.
