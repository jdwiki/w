Uso de la clase btn de twitter-bootstrap #

La clase btn de Twitter-bootstrap se puede usar con cualquiera de los siguientes elementos html.

  1. ancla
  2. botón
  3. ingrese con type="button" y type="submit"

A continuación se muestran ejemplos de todos los posibles casos de uso de la clase btn

<a class="btn" href="#" role="button">Link</a>
<button class="btn" type="submit">Button</button>
<input class="btn" type="button" value="Input">
<input class="btn" type="submit" value="Submit">

Aunque la clase btn se puede usar de cualquiera de las cuatro formas anteriores, se recomienda usar el elemento <button> siempre que sea posible

Agregar Glyphicon al botón #

Los glifos se pueden usar en texto, botones, barras de herramientas, navegación, formularios, etc. (Fuente: W3Schools). Los glifos son básicamente formas de íconos que se pueden usar para diseñar cualquiera de los mencionados anteriormente. Estos ejemplos describen el uso de glyphicons dentro de dos tipos de botones simplemente usando un lapso dentro de los botones que tienen una clase de tipo glyphicon:

Botón HTML

<button type="button" class="btn btn-info">
    <span class="glyphicon glyphicon-search"></span> Search
</button>

Botón ASP

<asp:LinkButton runat="server" CssClass="btn btn-info" >
    <span class="glyphicon glyphicon-envelope"></span> Email
</asp:LinkButton>
