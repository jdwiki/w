Los temas organizan los ejemplos en una página fácilmente digerible. Deben ser autónomos y escritos para el beneficio del lector apurado.

Anatomía de un tema #

Un tema se compone de varias secciones:

Título: Cuál es el tema del tema. Busque temas existentes primero para asegurarse de que no está repitiendo contenido que ya existe. Se pueden eliminar los temas ya cubiertos en otra parte de Stack Overflow.

Versiones: Las versiones donde se aplica el material cubierto por el tema. Puede elegir cualquiera de las versiones enumeradas en el tema de introducción o eliminarlas todas para indicar que el tema se aplica a todas las versiones. Consulte Versiones para obtener más información.

Introducción (opcional): Explica el principio organizativo de los ejemplos contenidos tanto para los lectores como para los colaboradores. Se encuentra arriba de los ejemplos y está limitado a 500 caracteres.

Ejemplos: Ejemplos de código para el tema. El código debe ser fácil de leer y bien comentado. La prosa también debe usarse para explicar los conceptos necesarios para comprender el código, pero trate de no ser demasiado detallado.

Se puede “fijar” un ejemplo por tema, lo que significa que flota sobre los otros ejemplos. Esto puede ser útil para una introducción extendida que es demasiado larga para la sección Introducción.

Sintaxis (opcional): Puede ser apropiado incluir una sintaxis para el tema en cuestión. Esta sección es siempre un bloque de código y debe ser una lista de rebajas:

- class.method(required_parameter [, optional])
- function f(int x) = { code defining what the function does }

Parámetros (opcional): al describir un tema con sintaxis, también puede ser apropiado incluir detalles de los parámetros. Los parámetros tienen el formato de una tabla de dos columnas:

| Parameter | Details |  
| --------- | ------- | 
|       yin |    dark | 
|      yang |   light |

Observaciones (opcional): Cualquier información adicional sobre el tema que pueda resultar valiosa para el lector final. Los comentarios son de forma libre.

Crear un tema #

Utilice los siguientes pasos para crear un tema:

  1. Haga clic en Crear nuevo tema en el tablero de la etiqueta.
  2. Nombre el tema.
  3. Proporcione ejemplos detallados en la sección “Ejemplos”.
  4. Si tiene más para compartir, use las secciones “Sintaxis”, “Parámetros” y “Comentarios”.

¡Has creado un tema con éxito! El tema estará disponible para todos, y todos pueden editarlo.

¡Recuerde que en la documentación de desbordamiento de pila, los ejemplos son la estrella del espectáculo! #

El tema introductorio #

Cuando se crea una nueva etiqueta de documentación, StackOverflow genera automáticamente un tema introductorio, que actúa como marcador de posición para el primer tema real.

El tema introductorio contiene textos de marcador de posición y punteros para señalar a los contribuyentes en las direcciones correctas para ayudar a expandir la nueva etiqueta de documentación.
