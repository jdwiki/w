Creando un nuevo proyecto de Symfony usando el Instalador de Symfony #

El Instalador de Symfony es una herramienta de línea de comandos que te ayuda a crear nuevas aplicaciones Symfony. Requiere PHP 5.4 o superior.

Descarga e instalación del instalador de Symfony en Linux/MacOS #

Abra una terminal y ejecute los siguientes comandos:

sudo mkdir -p /usr/local/bin
sudo curl -LsS https://symfony.com/installer -o /usr/local/bin/symfony
sudo chmod a+x /usr/local/bin/symfony

Esto crea un ejecutable Symfony global al que se puede llamar desde cualquier lugar. Tienes que hacer esto solo una vez: ahora puedes crear tantos proyectos Symfony como quieras.

Creando un nuevo proyecto con la última versión de Symfony #

Una vez que el instalador está instalado, puedes usarlo para crear un nuevo proyecto Symfony. Ejecute el siguiente comando:

symfony new my_project_name

Este comando creará un nuevo directorio (llamado my_project_name) que contiene la versión más reciente de Symfony Standard Edition. También instalará todas sus dependencias (incluidos los componentes reales de Symfony) usando Composer.

Creando un nuevo proyecto usando una versión específica de Symfony #

Si quieres seleccionar una versión específica de Symfony en lugar de la última, puedes usar el segundo argumento opcional del comando nuevo.

Para seleccionar una versión secundaria:

symfony new my_project_name 3.2

Para seleccionar una versión de parche:

symfony new my_project_name 3.2.9

Para seleccionar una versión beta o una versión candidata:

symfony new my_project 2.7.0-BETA1
symfony new my_project 2.7.0-RC1

Para seleccionar la versión más reciente de soporte a largo plazo (LTS):

symfony new my_project_name lts

Creando un nuevo proyecto Symfony usando Composer #

Si por alguna razón usar Symfony Installer no es una opción, también puedes crear un nuevo proyecto usando Composer. En primer lugar, asegúrese de tener Composer instalado.

A continuación, puede usar el comando create-project para crear un nuevo proyecto:

composer create-project symfony/framework-standard-edition my_project_name

Similar al Instalador de Symfony, este instalará la última versión de [Symfony Standard Edition][3] en un directorio llamado my_project_name y luego instalará sus dependencias (incluyendo los componentes de Symfony).

Instalar una versión específica de Symfony #

Al igual que con el instalador de Symfony, puedes seleccionar una versión específica de Symfony proporcionando un tercer argumento opcional:

composer create-project symfony/framework-standard-edition my_project_name "2.8.*"

Sin embargo, tenga en cuenta que no todos los alias de versión (como lts por ejemplo) están disponibles aquí.

Ejecutar la aplicación Symfony utilizando el servidor web integrado de PHP #

Después de crear una nueva aplicación Symfony, puedes usar el comando server:run para iniciar un servidor web PHP simple, para que puedas acceder a tu nueva aplicación desde tu navegador web:

cd my_project_name/
php bin/console server:run

Ahora puedes visitar http://localhost:8000/ para ver la página de bienvenida de Symfony.

Importante: si bien el uso del servidor web incorporado es excelente para el desarrollo, no debe usarlo en producción. En su lugar, utilice un servidor web con todas las funciones, como Apache o Nginx.
