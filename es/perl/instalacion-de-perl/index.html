Instalación de Perl

  • sistema operativo X
  • Ventanas

En esta página

  • sistema operativo X
  • Ventanas

Voy a comenzar esto con el proceso en Ubuntu, luego en OS X y finalmente en Windows. No lo he probado en todas las versiones de Perl, pero debería ser un proceso similar.

Use Perlbrew si desea cambiar fácilmente entre diferentes versiones de Perl.

Quiero decir que este tutorial es sobre Perl en su versión de código abierto. Hay otras versiones como activeperl con sus ventajas y desventajas, que no forman parte de este tutorial.

##Linux Hay más de una forma de hacerlo:

  • Usando el administrador de paquetes:

     sudo apt install perl

  • Instalación desde la fuente:

     wget http://www.cpan.org/src/5.0/perl-version.tar.gz
 tar -xzf perl-version.tar.gz
 cd perl-version
 ./Configure -de
 make
 make test
 make install

  • Instalación en su directorio $home (no se necesita sudo) con Perlbrew:

     wget -O - https://install.perlbrew.pl | bash

    See also Perlbrew

sistema operativo X #

Hay varias opciones:

  • Perlbrew:

    # You need to install Command Line Tools for Xcode  
curl -L https://install.perlbrew.pl | bash

  • Perlbrew con soporte para subprocesos:

    # You need to install Command Line Tools for Xcode  
curl -L https://install.perlbrew.pl | bash

    After the install of perlbrew, if you want to install Perl with thread support, just run:

    perlbrew install -v perl-5.26.0 -Dusethreads

  • De la fuente:

    tar -xzf perl-version.tar.gz
cd perl-version
./Configure -de
make
make test
make install

Ventanas #

  • Como dijimos antes, vamos con la versión de código abierto. Para Windows, puede elegir fresa o DWIM. Aquí cubrimos la versión fresa, ya que DWIM se basa en ella. La manera fácil aquí es instalar desde el ejecutable oficial.

Ver también berrybrew - el perlbrew para Windows Strawberry Perl
