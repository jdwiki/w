En esta página

  • Sintaxis
  • Parámetros
  • Buscar nodos con nombre como Luz, Dispositivo o Sensor
  • Buscar nodos cuyo nombre comience con Estrella
  • Buscar nodos con luz de nombre (sin distinción entre mayúsculas y minúsculas)
  • Buscar nodos que tengan un nombre que contenga Luz
  • Buscar nodos cuyo nombre termine en Ball

Sintaxis #

  1. Dentro de un nodo específico:

    {path-to-parent}/name()=‘search string’]

  2. En cualquier parte del documento:

    //*[name()=‘search string’]

Parámetros #

función valor devuelto
nombre-local() el nombre del nodo sin prefijo

El resultado del nombre local () no incluye el prefijo (búsqueda del nombre () Función XPATH para él)

Buscar nodos con nombre como Luz, Dispositivo o Sensor #

XML

<Galaxy>
    <Light>sun</Light>
    <Device>satellite</Device>
    <Sensor>human</Sensor>
    <Name>Milky Way</Name>
</Galaxy>

XPATH

/Galaxy/*[local-name()='Light' or local-name()='Device' or local-name()='Sensor']

o

//*[local-name()='Light' or local-name()='Device' or local-name()='Sensor']

PRODUCCIÓN

<Light>sun</Light>
<Device>satellite</Device>
<Sensor>human</Sensor>

Buscar nodos cuyo nombre comience con Estrella #

XML

<College>
    <FootBall>
        <Members>20</Members>
        <Coach>Archie Theron</Coach>
        <Name>Wild cats</Name>
        <StarFootballer>David Perry</StarFootballer>
    </FootBall>
    <Academics>
        <Members>100</Members>
        <Teacher>Tim Jose</Teacher>
        <Class>VII</Class>
        <StarPerformer>Lindsay Rowen</StarPerformer>
    </Academics>
</College>

XPATH

/College/*/*[starts-with(local-name(),"Star")]

o

//*[starts-with(local-name(),"Star")]

PRODUCCIÓN

<StarFootballer>David Perry</StarFootballer>
<StarPerformer>Lindsay Rowen</StarPerformer>

Buscar nodos con luz de nombre (sin distinción entre mayúsculas y minúsculas) #

XML

<Galaxy>
    <Light>sun</Light>
    <Device>satellite</Device>
    <Sensor>human</Sensor>
    <Name>Milky Way</Name>
</Galaxy>

XPATH

/Galaxy/*[lower-case(local-name())="light"]

o

//*[lower-case(local-name())="light"]

PRODUCCIÓN

<Light>sun</Light>

Buscar nodos que tengan un nombre que contenga Luz #

XML

<Data>
    <BioLight>
        <name>Firefly</name>
        <model>Insect</model>
    </BioLight>
    <ArtificialLight>
        <name>Fire</name>
        <model>Natural element</model>
        <source>flint</source>
    </ArtificialLight>
    <SolarLight>
        <name>Sun</name>
        <model>Star</model>
        <source>helium</source>
    </SolarLight>
</Data>

XPATH

/Data/*[contains(local-name(),"Light")]

o

//*[contains(local-name(),"Light")]

PRODUCCIÓN

<BioLight>
  <name>Firefly</name>
  <model>Insect</model>
</BioLight>
<ArtificialLight>
  <name>Fire</name>
  <model>Natural element</model>
  <source>flint</source>
</ArtificialLight>
<SolarLight>
  <name>Sun</name>
  <model>Star</model>
  <source>helium</source>
</SolarLight>

Buscar nodos cuyo nombre termine en Ball #

XML

<College>
    <FootBall>
        <Members>20</Members>
        <Coach>Archie Theron</Coach>
        <Name>Wild cats</Name>
        <StarPlayer>David Perry</StarPlayer>
    </FootBall>
    <VolleyBall>
        <Members>24</Members>
        <Coach>Tim Jose</Coach>
        <Name>Avengers</Name>
        <StarPlayer>Lindsay Rowen</StarPlayer>
    </VolleyBall>
    <FoosBall>
        <Members>22</Members>
        <Coach>Rahul Mehra</Coach>
        <Name>Playerz</Name>
        <StarPlayer>Amanda Ren</StarPlayer>
    </FoosBall>
</College>

XPATH

/College/*[ends-with(local-name(),"Ball")]

o

//*[ends-with(local-name(),"Ball")]

PRODUCCIÓN

<FootBall>
  <Members>20</Members>
  <Coach>Archie Theron</Coach>
  <Name>Wild cats</Name>
  <StarPlayer>David Perry</StarPlayer>
</FootBall>

  <Members>24</Members>
  <Coach>Tim Jose</Coach>

  <StarPlayer>Lindsay Rowen</StarPlayer>
</VolleyBall>
<FoosBall>
  <Members>22</Members>
