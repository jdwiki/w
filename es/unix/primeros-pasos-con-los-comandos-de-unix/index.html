Primeros pasos con los comandos de Unix

Este tema proporcionará una cobertura completa de los comandos básicos de Unix.

Una lista no exhaustiva de comandos de Unix #

$man <command>

Muestra las páginas del manual en línea para el comando

$clear

Borra la pantalla del terminal

$pwd

Devuelve el nombre del directorio de trabajo

$echo <string>

Escribe la cadena en la salida estándar

$printf <string>

Formatear e imprimir la cadena Ejemplo: imprimir $RUTA $printf “%s\n” $PATH

$uptime

Mostrar cuánto tiempo ha estado funcionando el sistema

$which <program>

Localice un archivo de programa en la ruta del usuario

$whereis <program>

Comprueba los directorios binarios estándar para los programas especificados, imprimiendo el

ARCHIVOS

$cd [directory]

Cambio de directorio Símbolos de directorio de uso común:

  • . : Directorio actual

  • .. : Directorio de padres

  • ~ : directorio de inicio

  • / : Directorio raíz

    $ls

  • $ls -a : Mostrar archivos ocultos

  • $ls -l : Mostrar lista larga

  • $ls -1: muestra solo el nombre de archivo por línea

  • $ls -h : formato legible por humanos

    $file Determinar el tipo de archivo (por ejemplo, gzip)

LEER ARCHIVOS

$more

Mostrar el contenido de un archivo una pantalla a la vez
barra espaciadora: Desplazarse a la pantalla siguiente; b=pantalla anterior
ingresar: desplazar una línea
h : Ayuda para más
q : Salir de la ayuda

$less <file>

Less es un programa similar a more, pero que permite el movimiento hacia atrás en el archivo así como hacia adelante

$cat <file>

Lee archivos secuencialmente, escribiéndolos en la salida estándar

$head [-number] <file>

Mostrar las primeras líneas de un archivo

$tail [-number] <file>

Muestra el contenido del archivo o, por defecto, su entrada estándar, a la salida estándar

  • $tail -f : Ver cambios en el archivo en tiempo real

    $touch Establece los tiempos de modificación y acceso de los archivos. Si algún archivo no existe, se crea con permisos por defecto

    $tee Copia la entrada estándar a la salida estándar. Presione ctrl-d para dejar de agregar contenido

  • $tee -a: agrega la salida a los archivos en lugar de sobrescribirlos

    $mkdir crear un directorio

    $wc Mostrar el número de líneas, palabras y bytes contenidos en cada archivo de entrada o entrada estándar

  • wc -l : Contar líneas

  • wc -w : Contar palabras

  • wc -m : Contar caracteres

    $diff Compara dos archivos línea por línea. Imprimirá solo las diferentes líneas.

    $locate Localizar archivos en disco

  • $localizar -q : suprimir errores

    $find Buscar archivos por nombre o contenido

  • $find -name : Buscar por nombre de archivo

  • $encontrar -tamaño <+/-n> Ejemplo: busque archivos en el directorio actual que tengan más de 10k $find . -size +10

    $rm Eliminar <archivo o directorio>

  • rm -f : Saltar confirmación

  • rm -i : Aprueba cada eliminación

  • rm -r : Recursivo

Ejemplo: eliminar el y su contenido $rm -r 

$mv <source_file> <target_file>

Cambia el nombre de un archivo

$mv <source_file> <target_directory>

Mueve un archivo

  • $mv -i : No anule los archivos existentes
  • $mv -r : Recursivo

Ejemplo: mover el directorio hacia arriba en la jerarquía $mv ..

$cp

Copie un archivo/directorio dentro de la misma máquina (Use el comando scp para copiar a una máquina remota)

  • $cp -i : No anule los archivos existentes
  • $cp -r :recursivo

Ejemplo: Copiar y renombrar un archivo

$cp <file_name> <new_file_name>

Ejemplo: Copiar al directorio

$cp <file_name> <directory_name>

Ejemplo: Copiar y renombrar un directorio

$cp -R <directory> <new_directory>

Ejemplo: copiar todos los archivos de un tipo específico a un directorio

$cp *.txt <directory>


$ln -s <file> <link name>

Crear un alias (enlace) a un archivo

  • $ln -s : Crear un enlace flexible (un enlace que funciona entre máquinas)

    $sort Ordenar el contenido de un archivo -r reverse sorts -n numeric sort

Ejemplo: ordenar el y escribir el resultado en sorted.txt $sort | uniq -u > sorted.txt

$uniq [-ucd] filename(s)

Busca líneas duplicadas. Los datos deben ser ordenados primero

  • $uniq -d: muestra solo una copia de las líneas duplicadas

  • $uniq -u: muestra solo las líneas que no están duplicadas

  • $uniq -c: muestra cada línea precedida por un recuento de ocurrencias Ejemplo: mostrar usuarios que están conectados más de una vez $who | cut -d’ ‘ -f1 | sort | uniq -d

    $grep <file_name> Imprime líneas que contienen una coincidencia para un patrón.

  • $grep -i : Realizar coincidencias sin distinción entre mayúsculas y minúsculas

  • $grep -v : Imprime todas las líneas que no contienen la expresión regular

  • $grep -r <patrón> : busca recursivamente los subdirectorios enumerados e imprime los nombres de archivo con la aparición del patrón

  • $grep -I : Excluir archivos binarios

    $tr “string1” [“string 2”] Herramienta de búsqueda y reemplazo. tr solo acepta su entrada de tuberías y redirecciones. no acepta archivos como entrada.

  • tr -d : Elimina todas las apariciones de todos los PERSONAJES en la cadena1

Ejemplo: Imprima un.txt en la pantalla después de eliminar todas las apariciones de “;” $cat a.txt | tr -d “;”

  • tr -s : Reemplaza las ocurrencias con un solo carácter

Ejemplo: $echo “SSSS SS” | tr -s “S” “S”

$tar

Crea y manipula archivos de archivo de transmisión. Esta implementación puede extraer imágenes tar, pax, cpio, zip, jar, ar e ISO y puede crear tar, archivos pax, cpio, ar y shar.


USO DEL DISCO

$du [file or directory]

Muestra el uso del bloque del sistema de archivos para cada archivo o directorio. Si no se especifica ningún archivo/directorio, se muestra el uso del bloque del directorio actual.

  • $du -a : Archivos y directorios (el valor predeterminado es solo directorios)

  • $du -h : formato legible por humanos Ejemplo: mostrar el uso del disco, ordenado, solo archivos MB $du -h | grep -i “m\t” | sort -n Ejemplo: Descubra los 10 archivos/directorios más grandes $du -a /var | sort -n -r | head -n 10

    $df Mostrar espacio libre en disco

  • $df -h : formato legible por humanos


REDIRECCIONES Y TUBERÍAS

> Redirigir la salida estándar. No sobrescriba el archivo si existe

>! Redirigir la salida estándar. Sobrescribir archivo si existe

>& Redirigir la salida estándar y el error estándar

Ejemplo: redirigir la salida del comando a un archivo

$ls > result.txt

Use el archivo > /dev/null para eliminar el mensaje de error

Ejemplo: busque un archivo llamado my_file_name e imprima el resultado en ~/find.txt; Ocultar errores (por ejemplo, “permisos denegados”)

$find / -name my_file_name.* > /dev/null > ~/find.txt

< Redirigir entrada estándar

>> Agregar salida estándar <comando> » : Agrega la salida del al final de un existente

<comando> < : Redirigir la entrada a un comando desde un archivo

| Redirigir la salida estándar a otro comando (tubería)

Ejemplo: Mostrar detalles paginados de procesos en ejecución

$ps -ex | more

| : La salida de la tubería del comando1 será la entrada del comando2 (si se desea un archivo de salida en medio de una tubería, use el comando tee)

Ejemplo: contar el número de usuarios conectados

$who | wc -l



PROCESOS

$ps

Mostrar procesos activos

  • $ps -e : Muestra información sobre el proceso
  • $ps -x: Mostrar procesos ocultos

Ejemplo: Buscar procesos por nombre $grep -l <process_name_regex>

$kill [-signal] pid

Matar un proceso.

Algunas de las señales más utilizadas:

  • 3 : SALIR (salir)

  • 9 : KILL (muerte no atrapable, no ignorable)

  • 15 : TERM (señal de terminación de software)

    $top Mostrar y actualizar información ordenada sobre procesos Consulte las páginas del manual para obtener una lista de posibles claves. Las claves comunes son: CPU, subprocesos, puertos.

  • $superior -o

    $htop Mostrar y actualizar información ordenada sobre procesos


    PERMISOS DE USUARIO

    $sudo Ejecutar el comando como superusuario

    $su (usuario sustituto) abre una sesión como administrador

    $exit Salir de raíz

    $whoami Mostrar identificación de usuario efectiva

    $who Imprimir todos los nombres de usuario conectados

    $passwd Cambia la contraseña

    $chmod Cambiar el acceso de propietario/grupo a un archivo o directorio

Quién: u usuario; grupo g; u otro; un todo arriba

Operación: + añadir; - retirar; = establecer (es decir, restablecer a nada y establecer solo lo que se especificó)

Permisos: r w x

Ejemplo: agrega permisos de lectura/ejecución al grupo

$chmod g +rx <file>

Ejemplo:

$chmod 743 <file>

Tenga en cuenta que para $cd en un directorio necesita los permisos x
