Uno de los errores más comunes que encontramos en el código Vue en StackOverflow es el mal uso de this. Los errores más comunes generalmente se encuentran en dos áreas, usar this en devoluciones de llamadas para promesas u otras funciones asincrónicas y usar funciones de flecha para definir métodos, propiedades calculadas, etc.

¡EQUIVOCADO! Usar una función de flecha para definir un método que se refiera a “esto” #

new Vue({
  el:"#app",
  data:{
    foo: "bar"
  },
  methods:{
    // This is wrong! Arrow functions capture "this" lexically
    // and "this" will refer to the window.
    doSomething: () => this.foo = "baz"
  }
})

¡CORRECTO! Definir métodos con la sintaxis de función típica #

new Vue({
  el:"#app",
  data:{
    foo: "bar"
  },
  methods:{
    doSomething: function(){
      this.foo = "baz"
    }
  }
})

Alternativamente, si está utilizando un compilador de javascript o un navegador compatible con Ecmascript 2015

new Vue({
  el:"#app",
  data:{
    foo: "bar"
  },
  methods:{
    doSomething(){
      this.foo = "baz"
    }
  }
})

¡EQUIVOCADO! Usando “esto” en una devolución de llamada dentro de un método Vue. #

new Vue({
  el:"#app",
  data:{
    foo: "bar"
  },
  methods:{
    doSomethingAsynchronous(){
      setTimeout(function(){
        // This is wrong! Inside this function,
        // "this" refers to the window object.
        this.foo = "baz";
      }, 1000);
    }
  }
})

¡CORRECTO! Use un cierre para capturar “esto” #

Puede capturar el ’esto’ correcto usando un cierre.

new Vue({
  el:"#star-wars-people",
  data:{
    people: null
  },
  mounted: function(){
    // Before executing the web service call, save this to a local variable
    var self = this;
    $.getJSON("http://swapi.co/api/people/", function(data){
      // Inside this call back, because of the closure, self will
      // be accessible and refers to the Vue object.
      self.people = data.results;
    })
  }
})

¡CORRECTO! Utilice una función de flecha. #

new Vue({
  el:"#star-wars-people",
  data:{
    people: null
  },
  mounted: function(){
    $.getJSON("http://swapi.co/api/people/", data => this.people = data.results);
  }
})

¡Precaución! Las funciones de flecha son una sintaxis introducida en Ecmascript 2015. Todavía no es compatible, pero todos los navegadores modernos, así que utilícelo solo si tiene como objetivo un navegador que sabe que lo admite, o si está compilando su javascript a la sintaxis ES5 usando algo como [babel] 1.

¡CORRECTO! Usa enlazar. #

Puede vincular la función de devolución de llamada.

new Vue({
  el:"#star-wars-people",
  data:{
    people: null
  },
  mounted:function(){
    $.getJSON("http://swapi.co/api/people/", function(data){
      this.people = data.results;
    }.bind(this));
  }
})

¡EQUIVOCADO! Usar “esto” dentro de una promesa. #

new Vue({
  el:"#star-wars-people",
  data:{
    people: null
  },
  mounted: function(){
    $.getJSON("http://swapi.co/api/people/", function(data){
      // Again, this is wrong! "this", here, refers to the window.
      this.people = data.results;
    })
  }
})
