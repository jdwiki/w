Animación de sprites 2D #

La animación Sprite consiste en mostrar una secuencia existente de imágenes o fotogramas.

Primero importe una secuencia de imágenes a la carpeta de activos. Cree algunas imágenes desde cero o descargue algunas de la Tienda de activos. (Este ejemplo usa este recurso gratuito.)

Arrastre cada imagen individual de una sola animación desde la carpeta de activos a la vista de escena. Unity mostrará un cuadro de diálogo para nombrar el nuevo clip de animación.

Este es un atajo útil para:

  • crear nuevos objetos de juego
  • asignar dos componentes (un Sprite Renderer y un Animator)
  • crear controladores de animación (y vincularles el nuevo componente Animator)
  • crear clips de animación con los fotogramas seleccionados

Obtenga una vista previa de la reproducción en la pestaña de animación haciendo clic en Reproducir:

Se puede usar el mismo método para crear nuevas animaciones para el mismo objeto del juego y luego eliminar el nuevo objeto del juego y el controlador de animación. Agregue el nuevo clip de animación al controlador de animación de ese objeto de la misma manera que con la animación 3D.

Animación básica para correr #

Este código muestra un ejemplo simple de animación en Unity.

Para este ejemplo, debe tener 2 clips de animación; Ejecutar e inactivo. Esas animaciones deben ser movimientos Stand-In-Place. Una vez seleccionados los clips de animación, cree un controlador de animación. Agregue este controlador al jugador o al objeto del juego que desea animar.

Abra la ventana de Animator desde la opción de Windows. Arrastre los 2 clips de animación a la ventana de Animator y se crearán 2 estados. Una vez creado, utilice el pestaña de parámetros izquierda para agregar 2 parámetros, ambos como bool. Nombre uno como “PerformRun” y otro como “PerformIdle”. Establezca “PerformIdle” en verdadero.

Realice transiciones del estado Inactivo a Ejecutar y Ejecutar a inactivo (Consulte la imagen). Haga clic en Inactivo-> Ejecutar transición y en la ventana Inspector, anule la selección de HasExit. Haz lo mismo con el otro transición. Para la transición Idle->Run, agregue una condición: PerformIdle. Para Run->Idle, agregue una condición: PerformRun. Agregue el script C# que se proporciona a continuación al objeto del juego. Debería ejecutarse con animación utilizando el botón Arriba y rotar con los botones Izquierda y Derecha.

using UnityEngine;
using System.Collections;

public class RootMotion : MonoBehaviour {

//Public Variables
[Header("Transform Variables")]
public float RunSpeed = 0.1f;
public float TurnSpeed = 6.0f;


Animator animator;

void Start()
{
    /**
    * Initialize the animator that is attached on the current game object i.e. on which you will attach this script.
    */
    animator = GetComponent<Animator>();
}

void Update()
{
    /**
    * The Update() function will get the bool parameters from the animator state machine and set the values provided by the user.
    * Here, I have only added animation for Run and Idle. When the Up key is pressed, Run animation is played. When we let go, Idle is played.
    */
    
    if (Input.GetKey (KeyCode.UpArrow)) {
            animator.SetBool ("PerformRun", true);
            animator.SetBool ("PerformIdle", false);
        } else {
            animator.SetBool ("PerformRun", false);
            animator.SetBool ("PerformIdle", true);
        }
}

void OnAnimatorMove()
    {
        /**
         * OnAnimatorMove() function will shadow the "Apply Root Motion" on the animator. Your game objects psoition will now be determined 
         * using this fucntion.
         */
        if (Input.GetKey (KeyCode.UpArrow)){
            transform.Translate (Vector3.forward * RunSpeed);
            if (Input.GetKey (KeyCode.RightArrow)) {
                transform.Rotate (Vector3.up * Time.deltaTime * TurnSpeed);
            }
            else if (Input.GetKey (KeyCode.LeftArrow)) {
                transform.Rotate (-Vector3.up * Time.deltaTime * TurnSpeed);
            }
        }

    }


   
}

Creación y uso de clips de animación #

Este ejemplo mostrará cómo crear y usar clips de animación para objetos o jugadores del juego.

Tenga en cuenta que los modelos utilizados en este ejemplo se descargan de Unity Asset Store. El reproductor se descargó desde el siguiente enlace: https://www.assetstore.unity3d.com/en/#!/content/21874.

Para crear animaciones, primero abra la ventana de animación. Puede abrirlo haciendo clic en Ventana y Seleccionar animación o presionando Ctrl+6. Seleccione el objeto del juego al que desea aplicar el clip de animación, desde la Ventana de jerarquía, y luego haga clic en el botón Crear en la Ventana de animación.

Asigne un nombre a su animación (como IdlePlayer, SprintPlayer, DyingPlayer, etc.) y guárdela. Ahora, desde la ventana de animación, haga clic en el botón Agregar propiedad. Esto te permitirá cambiar la propiedad del objeto del juego o jugador con respecto al tiempo. Esto puede incluir Transformaciones como rotación, posición y escala y cualquier otra propiedad que se adjunte al objeto del juego, p. Colisionador, renderizador de malla, etc.

Para crear una animación en ejecución para el objeto del juego, necesitará un modelo 3D humanoide. Puede descargar el modelo desde el enlace anterior. Siga los pasos anteriores para crear una nueva animación. Agregue una propiedad Transformar y seleccione Rotación para una de las piernas del personaje.

En este momento, su botón de Reproducir y los valores de Rotación en la propiedad del objeto del juego se habrían vuelto rojos. Haga clic en la flecha desplegable para ver los valores de rotación X, Y y Z. El tiempo de animación predeterminado se establece en 1 segundo. Las animaciones utilizan fotogramas clave para interpolar entre valores. Para animar, agregue teclas en diferentes puntos en el tiempo y cambie los valores de rotación desde la ventana Inspector. Por ej. el valor de rotación en el tiempo 0.0s puede ser 0.0. En el tiempo 0,5 s, el valor puede ser 20,0 para X. En el tiempo 1,0 s, el valor puede ser 0,0. Podemos terminar nuestra animación en 1.0s.

La duración de su animación depende de las últimas Claves que agregue a la Animación. Puede agregar más claves para que la interpolación sea más fluida.

Curvas de animación #

Las curvas de animación le permiten cambiar un parámetro flotante mientras se reproduce la animación. Por ejemplo, si hay una animación de duración de 60 segundos y desea un valor/parámetro flotante, llámelo X, para variar a través de la animación (como en el tiempo de animación = 0.0s; X = 0.0 , en tiempo de animación = 30.0s; X = 1,0, en tiempo de animación = 60,0 s; X = 0,0).

Una vez que tenga el valor flotante, puede usarlo para traducir, rotar, escalar o usarlo de cualquier otra forma.

Para mi ejemplo, mostraré un objeto de juego de jugador en ejecución. Cuando se reproduce la animación de correr, la velocidad de traducción del reproductor debe aumentar a medida que avanza la animación. Cuando la animación llega a su fin, la velocidad de traducción debería disminuir.

Tengo un clip de animación en ejecución creado. Seleccione el clip y luego en la ventana del inspector, haga clic en Editar.

Una vez allí, desplácese hacia abajo hasta Curvas. Haga clic en el signo + para agregar una curva. Asigne un nombre a la curva, p. ForwardRunCurve. Haga clic en la curva en miniatura en la derecha. Se abrirá una pequeña ventana con una curva predeterminada.

Queremos una curva con forma de parabólica donde sube y luego baja. Por defecto, hay 2 puntos en la línea. Puede agregar más puntos haciendo doble clic en la curva. Arrastre los puntos para crear una forma similar a la siguiente.

En la ventana de Animator, agregue el clip en ejecución. Además, agregue un parámetro flotante con el mismo nombre que la curva, es decir, ForwardRunCurve.

Cuando se reproduce la animación, el valor flotante cambiará de acuerdo con la curva. El siguiente código mostrará cómo usar el valor flotante:

using UnityEngine;
using System.Collections;

public class RunAnimation : MonoBehaviour {

Animator animator;
float curveValue;

void Start()
{
    animator = GetComponent<Animator>();
}

void Update()
{
    curveValue = animator.GetFloat("ForwardRunCurve");

    transform.Translate (Vector3.forward * curveValue);
}

}

La variable curveValue contiene el valor de la curva (ForwardRunCruve) en un momento dado. Estamos usando ese valor para cambiar la velocidad de la traducción. Puede adjuntar este script al objeto del juego del jugador.
