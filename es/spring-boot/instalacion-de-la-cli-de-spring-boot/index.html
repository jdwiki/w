La CLI de Spring Boot le permite crear y trabajar fácilmente con aplicaciones Spring Boot desde la línea de comandos.

Una vez instalado, Spring Boot CLI se puede ejecutar con el comando spring:

Para obtener ayuda de la línea de comandos:

$ spring help

Para crear y ejecutar su primer proyecto Spring Boot:

$ spring init my-app
$ cd my-app
$ spring run my-app

Abra su navegador en localhost:8080:

$ open http://localhost:8080

Obtendrá la página de error de etiqueta blanca porque aún no ha agregado ningún recurso a su aplicación, pero está listo para comenzar con solo los siguientes archivos:

my-app/
├── mvnw
├── mvnw.cmd
├── pom.xml
└── src/
    ├── main/
    │   ├── java/
    │   │   └── com/
    │   │       └── example/
    │   │           └── DemoApplication.java
    │   └── resources/
    │       └── application.properties
    └── test/
        └── java/
            └── com/
                └── example/
                    └── DemoApplicationTests.java
  • mvnw y mvnw.cmd - Scripts de envoltorio de Maven que descargarán e instalarán Maven (si es necesario) en el primer uso.
  • pom.xml - La definición del proyecto Maven
  • DemoApplication.java - la clase principal que inicia su aplicación Spring Boot.
  • application.properties: un archivo para las propiedades de configuración externalizadas. (También se le puede dar una extensión .yml).
  • DemoApplicationTests.java - Una prueba unitaria que valida la inicialización del contexto de la aplicación Spring Boot.

Instalar en Mac OSX con HomeBrew #

$ brew tap pivotal/tap
$ brew install springboot

Instalación manual #

Consulte la página de descarga para descargar y descomprimir manualmente la última versión, o siga los enlaces a continuación:

  • [spring-boot-cli-1.5.1.RELEASE-bin.zip](http://repo.spring.io/release/org/springframework/boot/spring-boot-cli/1.5.1.RELEASE/spring -boot-cli-1.5.1.RELEASE-bin.zip)

  • [spring-boot-cli-1.5.1.RELEASE-bin.tar.gz](http://repo.spring.io/release/org/springframework/boot/spring-boot-cli/1.5.1.RELEASE /spring-boot-cli-1.5.1.RELEASE-bin.tar.gz)

Instalar en Mac OSX con MacPorts #

$ sudo port install spring-boot-cli

¡Instálelo en cualquier sistema operativo con SDKMAN! #

SDKMAN! es el administrador del kit de desarrollo de software para Java. Se puede usar para instalar y administrar versiones de Spring Boot CLI, así como Java, Maven, Gradle y más.

$ sdk install springboot
