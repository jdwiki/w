Listas numeradas #

1. Lists
2. Can be
3. Numbered
  1. Listas
  2. Puede ser
  3. Numerados

Tenga en cuenta que los números en sí mismos se ignoran:

1. This is the first item
5. This is the fifth item
7. This is the seventh item
  1. Este es el primer elemento
  2. Este es el quinto ítem
  3. Este es el séptimo ítem

Sin embargo, el primer número se utiliza para iniciar la numeración:

3. This list starts at #3
2. However, this item is #4, despite being prefixed with `2.`
  1. Esta lista comienza en el #3
  2. Sin embargo, este artículo es el #4, a pesar de tener el prefijo 2.

Esto se puede usar para reanudar una lista después de que haya sido interrumpida por otro texto/una imagen/una tabla/etc.

My very favorite colors are:    
1. Blue
5. Red

(I like red because that's the best flavor of Skittle. But I digress.)

3. Orange  
9. [etc]

Mis colores favoritos son:

  1. Azul
  2. Rojo

(Me gusta el rojo porque ese es el mejor sabor de Skittle. Pero estoy divagando).

  1. Naranja
  2. [etc.]

También puede anotar un elemento de la lista de esta manera sin interrumpir la numeración:

My very favorite colors are:    
1. Blue
5. Red  
(I like red because that's the best flavor of Skittle. But I digress.)
9. Orange  
11. [etc]

Tenga en cuenta que no hay una línea en blanco entre 5. y la declaración entre paréntesis, y hay dos espacios al final de “Rojo”, por lo que obtenemos:

Mis colores favoritos son:

  1. Azul
  2. Rojo (Me gusta el rojo porque ese es el mejor sabor de Skittle. Pero estoy divagando).
  3. Naranja
  4. [etc.]

Sin los dos espacios, esa sección sería:

  1. Azul
  2. Rojo (Me gusta el rojo porque ese es el mejor sabor de Skittle. Pero estoy divagando).

… debido a cómo Markdown trata los saltos de línea.

Listas con viñetas #

Characters for bulleted lists:
* Asterisks
+ Plus signs
- Minus signs

Caracteres para listas con viñetas:

  • Asteriscos
  • Signos más
  • Signos menos

Tenga en cuenta: Para obtener los mejores resultados, debe usar el mismo carácter porque, como puede ver en el ejemplo a continuación, diferentes signos hacen que la lista se rompa.
