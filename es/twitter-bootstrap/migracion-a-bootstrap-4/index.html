Bootstrap 4 es una [reescritura importante] (http://v4-alpha.getbootstrap.com/migration/) y hay muchos cambios a tener en cuenta al actualizar desde Bootstrap 3. Estos son los cambios de nombre de clase, consejos y ejemplos de migrando su código Bootstrap 3.x a Bootstrap 4.x.

Este es solo un pequeño ejemplo, ejemplos más detallados a seguir.

Barra de navegación Bootstrap 4 #

El nuevo componente Bootstrap 4 Navbar se ha mejorado con respecto a su predecesor Bootstrap 3.x. En Bootstrap 4, la barra de navegación responsiva de forma predeterminada y utiliza flexbox para facilitar la alineación del contenido de la barra de navegación. También es una simple cuestión de usar las nuevas clases navbar-toggleable-* para cambiar el punto de interrupción de la barra de navegación. Ahora la barra de navegación tiene 6 tamaños de punto de interrupción o “estados” para que pueda tener fácilmente una de las siguientes opciones de la barra de navegación.

  • La barra de navegación nunca colapsa en la vista móvil vertical y siempre es horizontal.
  • La barra de navegación siempre está colapsada en la vista vertical y se alterna a través de la hamburguesa.
  • La barra de navegación colapsa en vista vertical en uno de los 4 puntos de interrupción de respuesta.

Barra de navegación básica de Bootstrap 4

<nav class="navbar navbar-toggleable-md">
  <button class="navbar-toggler navbar-toggler-right" type="button" data-toggle="collapse" data-target="#navbar1">
    <span class="navbar-toggler-icon"></span>
  </button>
  <a class="navbar-brand" href="#">Navbar</a>
  <div class="collapse navbar-collapse" id="navbar1">
    <ul class="navbar-nav mr-auto">
      <li class="nav-item active">
        <a class="nav-link" href="#">Home</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Link</a>
      </li>
    </ul>
  </div>
</nav>

Como puede ver en el código anterior, la clase navbar-header se eliminó de Bootstrap 4, y el container-fluid ya no es necesario para una barra de navegación de ancho completo.

Cambiar el punto de interrupción de la barra de navegación

La clase navbar-toggleable-md hace que la barra de navegación anterior se colapse verticalmente (y muestre el ícono de alternar) en el punto de interrupción medio (md) de 992px. Para cambiar esto a un punto de interrupción diferente, solo tendríamos que cambiar navbar-toggleable-md con uno de estos…

  • navbar-toggleable = colapsar en xs anchos <576px
  • navbar-toggleable-sm = colapsar en sm anchos <768px
  • navbar-toggleable-lg = colapsar en anchos lg <1200px

[Demostración de la barra de navegación de punto de interrupción de Bootstrap 4] (http://www.codeply.com/go/imngr08QRi)

Cambiar la alineación de la barra de navegación

Flexbox nos permite cambiar fácilmente la alineación de la barra de navegación y su contenido (marca, enlaces, formularios o texto). El contenido predeterminado de la barra de navegación está alineado a la izquierda. Por supuesto, hay muchos otros escenarios de alineación…

  • Marca izquierda (predeterminada), enlaces centro y derecha
  • Centro de marca, enlaces izquierdo y derecho
  • Marca a la izquierda y enlaces a la derecha
  • Entrada de formulario de marca, enlaces y ancho de relleno
  • Sin marca, enlaces centro y derecha
  • Marca a la izquierda, enlaces a la derecha dentro del contenedor
  • Enlaces justificados (ancho de relleno) centrados

Bootstrap 4 Centrado #

Cómo centrar un elemento, columna o contenido dentro de una columna funciona de manera diferente en Bootstrap 4.

Horizontal Centro #

  • text-center todavía se usa para elementos display:inline
  • mx-auto reemplaza center-block para centrar los elementos display:block
  • offset-* o mx-auto se puede usar para centrar las columnas de la cuadrícula

mx-auto (márgenes automáticos del eje x) centrará los elementos display:block o display:flex que tengan un ancho definido, (%, vw, px, etc. ). Flexbox se usa de forma predeterminada en las columnas de la cuadrícula, por lo que también hay varios métodos de centrado de Flexbox.

Centrar texto o elementos en línea: texto-centro

<div class="container">
    <h1 class="text-center">i'm centered</h1>
    <div class="row">
        <div class="col text-center">i'm centered!</div>
    </div>
</div>

Centrar display:block o display:flex: mx-auto

<div class="row">
    <div class="col-12">
        <img class="mx-auto d-block" src="//placehold.it/200x150?text=mx-auto">
    </div>
</div>

Columnas centrales usando compensaciones: offset-*

<div class="row">
    <div class="col-4 offset-4">
        <h6>I'm .col-4 centered (offset 4)
    </div>
</div>

Las columnas también se pueden centrar con: mx-auto

<div class="row">
    <div class="col-4 mx-auto">
        <h6>I'm .col-4 centered</h6>
    </div>
</div>

Bootstrap 3 a Bootstrap 4 CSS Cambios #

Dado que Bootstrap 4 es una reescritura importante, muchos de los nombres de clase de Bootstrap 3.x han cambiado o se han eliminado. La reestructuración de componentes como la barra de navegación y la introducción de nuevas clases CSS y la compatibilidad con Flexbox significan que actualizar a 4.x no es un proceso de conversión simple desde 3.x.

Sin embargo, hay algunas clases CSS de Bootstrap 3.x que tienen un reemplazo específico de Bootstrap 4.

Cambios de nombre/selector de clase CSS de Bootstrap 3.3.7 a 4 (alfa 6)

{t}: representa un nivel o un punto de interrupción (es decir, sm, md, lg, etc.).
El nivel xs es el predeterminado y no es necesario especificarlo: col- 3, col-6, etc..

{u}: representa un tamaño de unidad de columna (es decir, 1-12)

Cambios en el diseño de columnas del sistema de cuadrícula en Bootstrap 4 #

El primer bloque de código está escrito en Bootstrap 3. En Bootstrap 3 hay 4 tipos de especificaciones de columna, a saber, col-md-* col-lg-* col-sm-* col-xs-*. Un diseño totalmente receptivo se verá así en Bootstrap 3:

<div class="row">
    <div class="col-lg-4 col-md-8 col-sm-8 col-xs-8">
        contents
    </div>
    <div class="col-lg-4 col-md-4 col-sm-4 col-xs-4">
        contents
    </div>
</div>

En Bootstrap 4, agregaron un nuevo nivel de cuadrícula sm por debajo de 768 px para un control más granular. Entonces Bootstrap 4 tiene col-* (xs), col-sm-*, col-md-*, col-lg-* y col-xl-*. Entonces, lo que solía ser .col-md-6 en v3 ahora es .col-lg-6 en v4. Tenga en cuenta que el infijo -xs se ha eliminado, por lo que .col-6 representa 6 unidades de columna en el punto de interrupción extra pequeño (predeterminado).

Entonces, si ahora queremos escribir el mismo ejemplo anterior en Bootstrap 4, se vería así:

<div class="row">
    <div class="col-xl-8 col-lg-8 col-md-8 col-sm-8 col-8">
        contents
    </div>
    <div class="col-xl-8 col-lg-8 col-md-4 col-sm-4 col-4">
        contents
    </div>
</div>

Cambios en la compatibilidad del navegador #

En twitter-bootstrap 4, se eliminó la compatibilidad con IE8, IE9 e iOS 6. v4 ahora es solo IE10+ e iOS 7+. Para los sitios que necesitan cualquiera de esos, use v3.

En twitter-bootstrap 4 se ha agregado el soporte oficial para Android v5.0 Lollipop's Browser and WebView. Las versiones anteriores del navegador de Android y WebView siguen siendo solo ’no oficialmente compatibles'.

Eliminación de clases de afijos #

Affix se elimina de Bootstrap 4.

Se recomienda utilizar un polyfill posición: pegajoso en su lugar.

Si estaba usando Affix para aplicar estilos adicionales que no son de posición, es posible que los rellenos poliméricos no sean compatibles con su caso de uso. Una opción para tales usos es la biblioteca ScrollPos-Styler de terceros.

Según la documentación de Bootstrap

Se eliminó el complemento Affix jQuery. Recomendamos usar una posición: polyfill pegajoso en su lugar. Consulte la entrada de HTML5 para obtener detalles y recomendaciones específicas de polirrelleno.

Si estaba usando Affix para aplicar estilos adicionales sin posición, el polyfills podría no ser compatible con su caso de uso. Una opción para tales usos es la biblioteca ScrollPos-Styler de terceros.

JAVASCRIPT

$(document).ready(function() {

  var toggleAffix = function(affixElement, scrollElement, wrapper) {
  
    var height = affixElement.outerHeight(),
        top = wrapper.offset().top;
    
    if (scrollElement.scrollTop() >= top){
        wrapper.height(height);
        affixElement.addClass("affix");
    }
    else {
        affixElement.removeClass("affix");
        wrapper.height('auto');
    }
      
  };
  

  $('[data-toggle="affix"]').each(function() {
    var ele = $(this),
        wrapper = $('<div></div>');
    
    ele.before(wrapper);
    $(window).on('scroll resize', function() {
        toggleAffix(ele, $(this), wrapper);
    });
    
    // init
    toggleAffix(ele, $(window), wrapper);
  });
  
});

Bootstrap 4 alineación vertical #

Lograr que los elementos centro o abajo se alineen verticalmente siempre ha sido un desafío con CSS y Bootstrap. La alineación vertical deseada puede estar dentro de un contenedor principal o relativa a elementos adyacentes.

Ahora que Bootstrap 4 es flexbox por defecto, hay muchos enfoques diferentes para la alineación vertical usando: Auto-margins, Flexbox Utilities, o Display Utilities junto con con [Utilidades de alineación vertical] (http://v4-alpha.getbootstrap.com/utilities/vertical-align/).

Al principio, las [Utilidades de alineación vertical] (http://v4-alpha.getbootstrap.com/utilities/vertical-align/) parecerían una opción obvia, pero solo funcionan con elementos de visualización en línea y de tabla. Aquí hay algunas opciones y escenarios de alineación vertical de Bootstrap 4…

1 - Centro vertical con márgenes automáticos:

Una forma de centrar verticalmente es usar my-auto. Esto centrará el elemento dentro de su contenedor. Por ejemplo, h-100 hace que la fila tenga la altura completa y my-auto centrará verticalmente la columna col-sm-12.

<div class="row h-100">
    <div class="col-sm-12 my-auto">
        <div class="card card-block w-25">Card</div>
    </div>
</div>

Centro vertical con márgenes automáticos Demostración

my-auto representa márgenes en el eje y vertical y es equivalente a:

margin-top: auto;
margin-bottom: auto;

2 - Centro vertical con Flexbox:

Dado que Bootstrap 4 .row ahora es display:flex, simplemente puede usar align-self-center en cualquier columna para centrarla verticalmente…

       <div class="row">
           <div class="col-6 align-self-center">
                <div class="card card-block">
                 Center
                </div>
           </div>
           <div class="col-6">
                <div class="card card-inverse card-danger">
                    Taller
                </div>
          </div>
    </div>

o use align-items-center en toda la .row para alinear verticalmente al centro todas las col-* en la fila…

       <div class="row align-items-center">
           <div class="col-6">
                <div class="card card-block">
                 Center
                </div>
           </div>
           <div class="col-6">
                <div class="card card-inverse card-danger">
                    Taller
                </div>
          </div>
    </div>

Columnas de diferente altura del centro vertical Demostración

3 - Centro vertical usando utilidades de pantalla:

Bootstrap 4 tiene display utils que se pueden usar para display:table, display:table-cell, display:inline , etc. Estos se pueden usar con las [utilidades de alineación vertical] (https://v4-alpha.getbootstrap.com/utilities/vertical-align/) para alinear elementos en línea, bloques en línea o celdas de tablas.

<div class="row h-50">
    <div class="col-sm-12 h-100 d-table">
        <div class="card card-block d-table-cell align-middle">
            I am centered vertically
        </div>
    </div>
</div>

Bootstrap Orden de 4 columnas #

Cambiar el orden (o la posición) era posible en Bootstrap 3 usando las clases push pull. En Bootstrap 4, las clases push pull aún funcionan y, además, se puede usar flexbox order.

En Bootstrap 4, las clases push pull ahora son push-{viewport}-{units} y pull-{viewport}-{units} y el infijo xs- ha sido remoto. Considere este ejemplo que cambia el orden de las columnas al diseño 1-3-2 en xs y sm:

<div class="row">
    <div class="col-3 col-md-6">
        1
    </div>
    <div class="col-3 col-md-6 push-6 push-md-0">
        2
    </div>
    <div class="col-6 col-md-12 pull-3 pull-md-0">
        3
    </div>
</div>

Demostración de contrafase de Bootstrap 4

Dado que la nueva versión 4 es flexbox, otra opción es usar las clases de la utilidad de flexbox para cambiar el orden de las columnas. Ahora, las columnas col-*-12 de 12 unidades de ancho completo se pueden invertir utilizando el orden flexbox.

<div class="row">
    <div class="col-md-12">
        Col 1
    </div>
    <div class="col-md-12 flex-first flex-md-unordered">
        Col 2
    </div>
</div>
