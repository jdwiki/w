El sistema de cuadrícula de Bootstrap consta de 12 unidades conocidas como Columnas (clases CSS .col-*-*) que se utilizan para diseñar el contenido de izquierda a derecha en la ventana gráfica. Las columnas están contenidas dentro de Rows (clase CSS .row) para crear grupos horizontales de columnas. Las filas se colocan dentro de un Contenedor fijo o de ancho completo (.container o .container-fluid, respectivamente) para una alineación adecuada. Las columnas tienen relleno que crea un espacio (conocido como “medianil”) entre el contenido de las columnas.

Bootstrap incluye un primer sistema de cuadrícula fluida móvil con capacidad de respuesta que se escala adecuadamente hasta 12 columnas a medida que aumenta el tamaño del dispositivo o de la ventana gráfica. Incluye clases predefinidas para crear rápidamente diseños de página a través de una serie de filas y columnas que albergan su contenido.

Bootstrap filas y columnas #

El sistema de cuadrícula de Bootstrap tiene 12 unidades conocidas como Columnas que se pueden usar para diseñar contenido horizontalmente en la ventana gráfica.

La razón de una cuadrícula de 12 unidades (en lugar de 10, 16, etc.) es que 12 se divide uniformemente en 6 (mitades), 4 (cuartos) y 3 (tercios). Esto hace que la adaptación a una variedad de diseños sea mucho más fácil. Las columnas de la cuadrícula de Bootstrap se identifican mediante diferentes clases de CSS col-{breakpoint}-{units}. [Obtenga más información sobre el ancho de la ventana gráfica y los puntos de interrupción (A.K.A. Tiers)](https://www.wikiod.com/es/twitter-bootstrap/sistema-de-red#Niveles de cuadrícula Bootstrap (puntos de interrupción))

Entonces, por ejemplo, col-md-3 representa una columna que ocupa 3 de las 12 unidades (o el 25 %) en una ventana gráfica de ancho medio (md). Para usar un ancho de columna en su diseño, simplemente use la clase col-{breakpoint}-{units} apropiada en su marcado HTML.

<div class="col-{punto de interrupción}-{unidades}">

El ancho de columna es fluido (no ancho fijo), por lo que las columnas consumen un porcentaje de su contenedor.

Unidades de columna en Bootstrap 3

  • col-*-1: 1 de 12 (8.33333333% ancho)
  • col-*-2: 2 de 12 (16.66666667% ancho)
  • col-*-3: 3 de 12 (25% de ancho)
  • col-*-4: 4 de 12 (33.3333333% ancho)
  • col-*-5: 5 de 12 (41.66666667% ancho)
  • col-*-6: 6 de 12 (50% ancho)
  • col-*-7: 7 de 12 (58.33333333% ancho)
  • col-*-8: 8 de 12 (66.66666667% ancho)
  • col-*-9: 9 de 12 (75% ancho)
  • col-*-10: 10 de 12 (83.33333333% ancho)
  • col-*-11: 11 de 12 (91.66666667% ancho)
  • col-*-12: 12 de 12 (100% ancho)

La fila de Bootstrap

La clase Bootstrap .row se usa para contener las Columnas. Las columnas siempre deben colocarse en Filas, y las Filas siempre deben colocarse dentro de un Contenedor (contenedor o contenedor-fluido). La Fila usa márgenes negativos (-15 px) para garantizar el espacio adecuado entre el contenido de la columna y el borde del navegador. Las filas se utilizan para agrupar columnas horizontalmente.

<div class="container">
    <div class="row">
        <!-- one more columns -->
        <div class="col-{breakpoint}-{units}">..</div>
    </div>
</div>

Las columnas llenarán el .row horizontalmente de izquierda a derecha y envolverán en una nueva línea cada 12 unidades de columna. Por lo tanto, puede usar .rows para crear cortes horizontales, o puede agregar más de 12 unidades de columna en un solo elemento .row para tener columnas que se ajustan ( o apilar) verticalmente hacia abajo en la ventana gráfica.

Al usar el ajuste de columnas (más de 12 unidades en una .fila), deberá [usar reinicios receptivos (o arreglos claros)] (http://getbootstrap.com/css/#grid-responsive-resets) para asegúrese de envolver uniformemente el contenido de la columna desigual. Esto es fundamental cuando el contenido de las columnas varía en altura.

Niveles de cuadrícula Bootstrap (puntos de interrupción) #

Además del concepto de [unidades de columna] (https://www.wikiod.com/es/twitter-bootstrap/sistema-de-red#Bootstrap filas y columnas), Bootstrap tiene diferentes **puntos de interrupción ** o tamaños de cuadrícula conocidos como niveles. La cuadrícula de Bootstrap 3 tiene cuatro (4) niveles para adaptarse a diferentes anchos de pantalla (o ventana gráfica). Los niveles de Bootstrap 3 son xs, sm, md y lg. Las columnas de la cuadrícula de Bootstrap se identifican mediante diferentes clases de CSS col-{breakpoint}-{units}.

Cada nivel de cuadrícula abarca un rango que está diseñado para adaptarse mejor a los anchos de pantalla típicos de los dispositivos, como los de las computadoras de escritorio, las computadoras portátiles, las tabletas y los teléfonos inteligentes.

Bootstrap utiliza consultas de medios CSS para crear puntos de interrupción receptivos que establecen un límite para cada tamaño de cuadrícula. Estos tamaños de cuadrícula le permiten cambiar el diseño de las columnas para que coincidan mejor con diferentes anchos de pantalla y dispositivos, la esencia del diseño receptivo.

  • col-xs-* — para los anchos de pantalla más pequeños como teléfonos inteligentes < 768 px
  • col-sm-* — para anchos de pantalla pequeños como teléfonos inteligentes y tabletas >= 768 px
  • col-md-* — para anchos de pantalla medianos como tabletas y portátiles >= 992 px
  • col-lg-* — para anchos de pantalla grandes como escritorios >= 1200 px

Referencia: [Sistema de cuadrícula] (http://getbootstrap.com/css/#grid)

Mismo ancho de columna para cada dispositivo

Para crear una columna que siempre sea * 50% * del ancho de la ventana gráfica (en todos los dispositivos), puede configurar col-*-6 para cada nivel.

<div class="col-xs-6 col-sm-6 col-md-6 col-lg-6">..</div>

Sin embargo, este es un margen adicional innecesario, ya que col-xs-6 significa 6 unidades en xs y más. El nivel más pequeño que establezca (xs, sm o md) también define el tamaño para anchos de pantalla más grandes. Para la columna del mismo tamaño en todos los niveles, solo establezca el ancho para la ventana gráfica más pequeña.

Código más corto:

<div class="col-xs-6">..</div>

Diferente ancho de columna para cada dispositivo (diseño receptivo)

Las clases col-*-* se pueden combinar para controlar los anchos de columna en diferentes tamaños de cuadrícula.

Por ejemplo, cree una columna de ancho 50% en el nivel sm y una columna de ancho 25% en el nivel md…

<div class="col-md-3 col-sm-6">..</div>

Las cuadrículas sm, md y lg se “apilarán” verticalmente en anchos de ventana de visualización inferiores a 768 píxeles. Aquí es donde encaja la cuadrícula xs. Las columnas que usan las clases col-xs-* no se apilarán verticalmente y seguirán reduciéndose en las pantallas más pequeñas.

Manipulación del orden de las columnas usando empujar y tirar #

<div class="container content">
    <div class="row">
        <!--Main Content-->
        <div class="col-lg-9 col-lg-push-3">
            Main Content
        </div>
        
        <!--Sidebar-->
        <div class="col-lg-3 col-lg-pull-9">
            Sidebar
        </div>
    </div>
</div>

Esto cambia el orden de las columnas de cuadrícula integradas.

Sintaxis: .col-md-push-* y .col-md-pull-*.

Contenedores #

Bootstrap requiere un elemento contenedor para envolver el contenido del sitio y albergar nuestro sistema de cuadrícula. Puede elegir uno de los dos contenedores para usar en sus proyectos.

Use la clase .container para un contenedor de ancho fijo receptivo.

<div class="container">
  ...
</div>

Use la clase .container-fluid para un contenedor de ancho completo, que abarque todo el ancho de su ventana gráfica.

<div class="container-fluid">
  ...
</div>

Nota: Los contenedores no se pueden anidar (no se puede poner un contenedor dentro de otro contenedor), debido al “relleno” y más.

Preguntas de los medios #

Las consultas de medios en Bootstrap le permiten mover, mostrar y ocultar contenido según el tamaño de la ventana gráfica. Las siguientes consultas de medios se utilizan en archivos LESS para crear los puntos de interrupción clave en el sistema de cuadrícula Bootstrap:

/* Small devices (tablets, 768px and up) */
@media (min-width: @screen-sm-min) { ... }

/* Medium devices (desktops, 992px and up) */
@media (min-width: @screen-md-min) { ... }

/* Large devices (large desktops, 1200px and up) */
@media (min-width: @screen-lg-min) { ... }

Ocasionalmente, estos se amplían para incluir un ancho máximo para limitar el CSS a un conjunto más limitado de dispositivos:

@media (max-width: @screen-xs-max) { ... }
@media (min-width: @screen-sm-min) and (max-width: @screen-sm-max) { ... }
@media (min-width: @screen-md-min) and (max-width: @screen-md-max) { ... }
@media (min-width: @screen-lg-min) { ... }

Columnas de compensación #

Estas clases aumentan el margen izquierdo de una columna en * columnas. Por ejemplo, .col-md-offset-4 mueve .col-md-4 sobre cuatro columnas.

<div class="row">
  <div class="col-lg-4"></div>
  <div class="col-lg-4 col-lg-offset-4"></div>
</div>
<div class="row">
  <div class="col-lg-5 col-lg-offset-1"></div>
  <div class="col-lg-5 col-lg-offset-1"></div>
</div>
