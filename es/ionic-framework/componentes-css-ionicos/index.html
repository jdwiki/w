Ionic tiene una gran cantidad de excelentes componentes CSS declarados listos para facilitarle la vida mientras codifica su próxima aplicación móvil híbrida. Estos componentes varían desde un sistema de cuadrícula básico hasta diseñar sus formularios. Estos componentes están en su uso si elige instalar Ionic con las hojas de estilo CSS preestablecidas.

Una de las funciones básicas que trae Ionic CSS es que viene con un conjunto de colores para empezar, pero como regla general, los colores están destinados a ser anulados. Se agregan colores de utilidad para ayudar a establecer una convención de nomenclatura. Podrías llamarlo un tema básico de la aplicación. Para personalizar los colores, simplemente puede anular los que provienen del archivo CSS ionic.css. Además, dado que Ionic está construido con Sass, para obtener más potencia y flexibilidad, también puede modificar y ampliar las variables de color dentro del archivo _variables.scss.

Puede configurar un proyecto Ionic para usar SASS muy fácilmente ejecutando el comando ionic setup sass en su terminal.

Puede encontrar la documentación oficial sobre Ionic CSS aquí: http://ionicframework.com/docs/components/

Sintaxis básica del sistema de cuadrícula #

Cuadrícula básica #

En Ionic puedes declarar filas configurando la clase fila a un elemento. Las filas serán elementos alineados horizontalmente y cualquier cosa dentro de este elemento pertenecerá a la fila. Dentro de la fila puede declarar columnas de diferentes anchos. Puede elegir entre las siguientes declaraciones.

Clase Ancho
.col-10 10%
.col-20 20%
.col-25 25%
.col-33 33,3333%
.col-50 50%
.col-67 66,6666%
.col-75 75%
.col-80 80%
.col-90 90%

El valor máximo que las columnas pueden tener dentro de una fila es 100. Aquí hay algunos ejemplos de la cuadrícula básica.

<div class="row">
  <div class="col col-50">.col.col-50</div>
  <div class="col">.col</div>
  <div class="col">.col</div>
</div>

<div class="row">
  <div class="col col-75">.col.col-75</div>
  <div class="col">.col</div>
</div>

<div class="row">
  <div class="col">.col</div>
  <div class="col col-75">.col.col-75</div>
</div>

<div class="row">
  <div class="col">.col</div>
  <div class="col">.col</div>
</div>

Cuadrículas compensadas #

También puede establecer col-offset-<value> en las columnas. En el ejemplo a continuación, el tercio de una columna de ancho tiene un tercio del ancho de desplazamiento, lo que significa que tendrá un tercio de ancho de ancho y estará centrado en la página debido a su desplazamiento.

<div class="row">
  <div class="col col-33 col-offset-33">.col</div>
  <div class="col">.col</div>
</div>

Alinear columnas #

Es posible alinear columnas verticalmente configurando col-<align_value> en una columna por separado como esta.

<div class="row">
  <div class="col col-top">.col</div>
  <div class="col col-center">.col</div>
  <div class="col col-bottom">.col</div>
  <div class="col">1<br>2<br>3<br>4</div>
</div>

Lo anterior alineará cada columna por sí misma. Si desea alinear todas las columnas dentro de la fila, puede establecer el valor de alineación en la fila misma. En el siguiente ejemplo, todas las columnas dentro de esta fila se alinearán verticalmente en el centro de la fila.

<div class="row row-center">
  <div class="col">.col</div>
  <div class="col">.col</div>
  <div class="col">.col</div>
  <div class="col">1<br>2<br>3<br>4</div>
</div>

Cuadrícula receptiva #

También es posible que desee que las columnas respondan a partir de que se apilarán una encima de la otra en algún ancho de ventana gráfica. Tienes tres opciones.

Clase Punto de interrupción (aproximadamente)
.responsive-sm Más pequeño que el teléfono paisaje
.responsive-md Más pequeña que la tableta vertical
.responsive-lg Más pequeña que la tableta horizontal

En este ejemplo, estas columnas se apilarán bajo el ancho de aproximadamente un teléfono horizontal.

<div class="row responsive-sm">
  <div class="col">.col</div>
  <div class="col">.col</div>
  <div class="col">.col</div>
  <div class="col">.col</div>
</div>

Por supuesto, también puede realizar sus propias consultas de medios si estos puntos de interrupción no son adecuados para usted y/o si necesita puntos de interrupción más específicos.

Sintaxis básica de elementos de lista #

Casi todas las aplicaciones tienen algún tipo de lista. Ionic tiene sus propias declaraciones CSS de elementos de lista listas para usar para que sea más fácil hacer listas dentro de su aplicación. Puede usar elementos HTML y declarar una clase para el o usar la directiva ion-list para crearlos. El ejemplo de una directiva está en la parte inferior.

Sintaxis CSS básica del elemento de la lista:

<ul class="list">
  <li class="item"></li>
</ul>

Lista con divisores:

<div class="list">
  <a class="item" href="#">
    List item
  </a>
  <div class="item item-divider">
    Divider that looks a bit different from items
  </div>
  <a class="item" href="#">
    Another list item
  </a>
</div>

Lista de elementos con iconos:

<div class="list">
  <a class="item item-icon-left" href="#">
    <i class="icon ion-trash-b"></i>
    List item with a trashcan icon on the left
  </a>
</div>

También puede establecer iconos en ambos lados de los elementos con la siguiente sintaxis:

<div class="list">
  <a class="item item-icon-left item-icon-right" href="#">
    <i class="icon ion-trash-b"></i>
    List item with a trashcan icon on the left and a briefcase icon on the right
    <i class="icon ion-briefcase"></i>
  </a>
</div>

Un elemento de lista con botón o botones se puede crear así:

<div class="list">
  <div class="item item-button-right">
    Item with a button on the right that has a clock icon in it
    <button class="button button-positive">
      <i class="icon ion-clock"></i>
    </button>
  </div>
</div>

También es posible crear elementos de lista con avatares, miniaturas y recuadros que crearán relleno alrededor de los elementos de la lista. Ionic también maneja la configuración de iconos, etc. en los elementos de la lista configurando el relleno de acuerdo con los elementos de la lista.

Ionic también tiene sus propias directivas para casillas de verificación, botones de radio, etc. Aquí hay un ejemplo de una lista de casillas de verificación con Ionic.

<ion-list>
  <ion-checkbox ng-model="choice1">Choice 1</ion-checkbox>
  <ion-checkbox ng-model="choice2">Choice 2</ion-checkbox>
</ion-list>

Uso básico de colores utilitarios #

Preset Ionic CSS tendrá un tema y colores preestablecidos para él. Puede modificar o anular los valores básicos en ionic.css o en su archivo CSS personalizado. También puede definirlos con SASS y para usar SASS en Ionic solo necesita ejecutar el comando ionic setup sass en su terminal.

Uso básico de colores en un botón. La clase button-<frase> hará que el fondo del botón y los bordes tengan el color del tema establecido.

<button class="button button-positive">
  button-positive
</button>

<button class="button button-calm">
  button-calm
</button>

<button class="button button-balanced">
  button-balanced
</button>

Sus opciones de prefijo CSS son las siguientes:

  • <elemento>-luz

  • <elemento>-estable

  • <elemento>-positivo

  • <elemento>-calma

  • <elemento>-equilibrado

  • <elemento>-energizado

  • <elemento>-asertivo

  • <elemento>-real

  • <elemento>-oscuro

Estas clases también se pueden agregar, por ejemplo, en insignias, etc. Aquí hay un ejemplo de una insignia:

<span class="badge badge-positive">Positive badge</span>
