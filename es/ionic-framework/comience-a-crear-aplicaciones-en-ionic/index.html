Iniciar una aplicación Ionic #

Iniciando una aplicación Ionic #

$ ionic start myapp [template]

Las plantillas de inicio pueden provenir de una plantilla con nombre, un repositorio de Github, un Codepen o un directorio local. Una plantilla inicial es lo que se convierte en el directorio www dentro del proyecto Cordova.

Iniciadores de plantilla con nombre #

  • pestañas (Predeterminado)

  • menú lateral

  • [en blanco] (https://github.com/driftyco/ionic-starter-blank)

Iniciadores de Github Repo #

  • Cualquier URL de repositorio de Github, por ejemplo: https://github.com/driftyco/ionic-starter-tabs
  • Las plantillas con nombre son simplemente alias de los repositorios de inicio de Ionic

Arrancadores de URL Codepen #

  • Cualquier URL de Codepen, por ejemplo: http://codepen.io/ionic/pen/odqCz

  • Demostraciones de Codepen Ionic

Arrancadores de directorio local: #

  • Ruta relativa o absoluta a un directorio local

Indicadores/opciones de la línea de comandos: #

--appname, -a ....... Human readable name for the app (Use quotes around the name) 

--id, -i ............ Package name set in the <widget id> config ex: com.mycompany.myapp 

--no-cordova, -w .... Do not create an app targeted for Cordova
