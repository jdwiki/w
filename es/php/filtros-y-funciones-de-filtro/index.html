Esta extensión filtra datos al validarlos o desinfectarlos. Esto es especialmente útil cuando la fuente de datos contiene datos desconocidos (o extraños), como entradas proporcionadas por el usuario. Por ejemplo, estos datos pueden provenir de un formulario HTML.

Sintaxis #

  • mixed filter_var (mixed $variable [, int $filter = FILTER_DEFAULT [, mixed $options]])

Parámetros #

Parámetro Detalles
variables Valor a filtrar. Tenga en cuenta que los valores escalares se convierten en cadenas internamente antes de que se filtren.
—— ——
filtro El ID del filtro a aplicar. La página del manual Tipos de filtros enumera los filtros disponibles. Si se omite, se utilizará FILTER_DEFAULT, que es equivalente a FILTER_UNSAFE_RAW. Esto dará como resultado que no se realice ningún filtrado de forma predeterminada.
—— ——
opciones Matriz asociativa de opciones o disyunción bit a bit de banderas. Si el filtro acepta opciones, se pueden proporcionar indicadores en el campo “indicadores” de la matriz. Para el filtro de “devolución de llamada”, se debe pasar el tipo invocable. La devolución de llamada debe aceptar un argumento, el valor que se va a filtrar, y devolver el valor después de filtrarlo/desinfectarlo.

Validar correo electrónico #

Al filtrar una dirección de correo electrónico, filter_var() devolverá los datos filtrados, en este caso, la dirección de correo electrónico, o falso si no se puede encontrar una dirección de correo electrónico válida:

var_dump(filter_var('john@example.com', FILTER_VALIDATE_EMAIL));
var_dump(filter_var('notValidEmail', FILTER_VALIDATE_EMAIL));

Resultados:

string(16) "john@example.com"
bool(false)

Esta función no valida caracteres no latinos. El nombre de dominio internacionalizado se puede validar en su formato xn--.

Tenga en cuenta que no puede saber si la dirección de correo electrónico es correcta antes de enviarle un correo electrónico. Es posible que desee realizar algunas comprobaciones adicionales, como comprobar un registro MX, pero esto no es necesario. Si envía un correo electrónico de confirmación, no olvide eliminar las cuentas no utilizadas después de un breve período.

Validar que un valor sea un número entero #

Al filtrar un valor que debería ser un número entero, filter_var() devolverá los datos filtrados, en este caso el número entero, o falso si el valor no es un número entero. Los flotantes son no enteros:

var_dump(filter_var('10', FILTER_VALIDATE_INT));
var_dump(filter_var('a10', FILTER_VALIDATE_INT));
var_dump(filter_var('10a', FILTER_VALIDATE_INT));
var_dump(filter_var(' ', FILTER_VALIDATE_INT));
var_dump(filter_var('10.00', FILTER_VALIDATE_INT));
var_dump(filter_var('10,000', FILTER_VALIDATE_INT));
var_dump(filter_var('-5', FILTER_VALIDATE_INT));
var_dump(filter_var('+7', FILTER_VALIDATE_INT));

Resultados:

int(10)
bool(false)
bool(false)
bool(false)
bool(false)
bool(false)
int(-5)
int(7)

Si espera solo dígitos, puede usar una expresión regular:

if(is_string($_GET['entry']) && preg_match('#^[0-9]+$#', $_GET['entry']))
    // this is a digit (positive) integer
else
    // entry is incorrect

Si convierte este valor en un número entero, no tiene que hacer esta verificación y puede usar filter_var.

Validación de un número entero dentro de un rango #

Al validar que un número entero cae en un rango, la verificación incluye los límites mínimo y máximo:

$options = array(
    'options' => array(
        'min_range' => 5,
        'max_range' => 10,
    )
);
var_dump(filter_var('5', FILTER_VALIDATE_INT, $options));
var_dump(filter_var('10', FILTER_VALIDATE_INT, $options));
var_dump(filter_var('8', FILTER_VALIDATE_INT, $options));
var_dump(filter_var('4', FILTER_VALIDATE_INT, $options));
var_dump(filter_var('11', FILTER_VALIDATE_INT, $options));
var_dump(filter_var('-6', FILTER_VALIDATE_INT, $options));

Resultados:

int(5)
int(10)
int(8)
bool(false)
bool(false)
bool(false)

Al filtrar una URL, filter_var() devolverá los datos filtrados, en este caso la URL, o falso si no se encuentra una URL válida:

URL: ejemplo.com

var_dump(filter_var('example.com', FILTER_VALIDATE_URL, FILTER_FLAG_SCHEME_REQUIRED));
var_dump(filter_var('example.com', FILTER_VALIDATE_URL, FILTER_FLAG_HOST_REQUIRED));
var_dump(filter_var('example.com', FILTER_VALIDATE_URL, FILTER_FLAG_PATH_REQUIRED));

Desinfectar filtros #

podemos usar filtros para desinfectar nuestra variable según nuestra necesidad.

Ejemplo

$string = "<p>Example</p>";
$newstring = filter_var($string, FILTER_SANITIZE_STRING);
var_dump($newstring); // string(7) "Example"

anterior eliminará las etiquetas html de la variable $string.

Validación de valores booleanos #

var_dump(filter_var(true, FILTER_VALIDATE_BOOLEAN, FILTER_NULL_ON_FAILURE)); // true
var_dump(filter_var(false, FILTER_VALIDATE_BOOLEAN, FILTER_NULL_ON_FAILURE)); // false
var_dump(filter_var(1, FILTER_VALIDATE_BOOLEAN, FILTER_NULL_ON_FAILURE)); // true
var_dump(filter_var(0, FILTER_VALIDATE_BOOLEAN, FILTER_NULL_ON_FAILURE)); // false
var_dump(filter_var('1', FILTER_VALIDATE_BOOLEAN, FILTER_NULL_ON_FAILURE)); // true
var_dump(filter_var('0', FILTER_VALIDATE_BOOLEAN, FILTER_NULL_ON_FAILURE)); // false
var_dump(filter_var('', FILTER_VALIDATE_BOOLEAN, FILTER_NULL_ON_FAILURE)); // false
var_dump(filter_var(' ', FILTER_VALIDATE_BOOLEAN, FILTER_NULL_ON_FAILURE)); // false
var_dump(filter_var('true', FILTER_VALIDATE_BOOLEAN, FILTER_NULL_ON_FAILURE)); // true
var_dump(filter_var('false', FILTER_VALIDATE_BOOLEAN, FILTER_NULL_ON_FAILURE)); // false
var_dump(filter_var([], FILTER_VALIDATE_BOOLEAN, FILTER_NULL_ON_FAILURE)); // NULL
var_dump(filter_var(null, FILTER_VALIDATE_BOOLEAN, FILTER_NULL_ON_FAILURE)); // false

Validar un número es un flotante #

Valida el valor como flotante y lo convierte en flotante en caso de éxito.

var_dump(filter_var(1, FILTER_VALIDATE_FLOAT));
var_dump(filter_var(1.0, FILTER_VALIDATE_FLOAT));
var_dump(filter_var(1.0000, FILTER_VALIDATE_FLOAT));
var_dump(filter_var(1.00001, FILTER_VALIDATE_FLOAT));
var_dump(filter_var('1', FILTER_VALIDATE_FLOAT));
var_dump(filter_var('1.0', FILTER_VALIDATE_FLOAT));
var_dump(filter_var('1.0000', FILTER_VALIDATE_FLOAT));
var_dump(filter_var('1.00001', FILTER_VALIDATE_FLOAT));
var_dump(filter_var('1,000', FILTER_VALIDATE_FLOAT));
