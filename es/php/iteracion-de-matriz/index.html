Sintaxis #

  • for ($i = 0; $i < cuenta($matriz); $i++) { incremental_iteration(); }
  • for ($i = cuenta ($array) - 1; $i >= 0; $i–) { reverse_iteration(); }
  • foreach ($datos como $datos) { }
  • foreach ($datos como $clave => $datos) { }
  • foreach ($datos como &$datos) { }

Comparación de métodos para iterar una matriz #

Método Ventaja
foreach El método más simple para iterar una matriz.
foreach por referencia Método simple para iterar y cambiar elementos de una matriz.
for con índice incremental Permite iterar la matriz en una secuencia libre, p. saltar o invertir múltiples elementos
Punteros de matriz internos Ya no es necesario usar un bucle (para que pueda iterar una vez cada llamada de función, recepción de señal, etc.)

Iterando varias matrices juntas #

A veces, dos matrices de la misma longitud deben iterarse juntas, por ejemplo:

$people = ['Tim', 'Tony', 'Turanga'];
$foods = ['chicken', 'beef', 'slurm'];

array_map es la forma más sencilla de lograr esto:

array_map(function($person, $food) {
    return "$person likes $food\n";
}, $people, $foods);

que dará salida:

Tim likes chicken
Tony likes beef
Turanga likes slurm

Esto se puede hacer a través de un índice común:

assert(count($people) === count($foods));
for ($i = 0; $i < count($people); $i++) {
    echo "$people[$i] likes $foods[$i]\n";
}

Si las dos matrices no tienen claves incrementales, se puede usar array_values($array)[$i] para reemplazar $array[$i].

Si ambas matrices tienen el mismo orden de claves, también puede usar un bucle foreach-with-key en una de las matrices:

foreach ($people as $index => $person) {
    $food = $foods[$index];
    echo "$person likes $food\n";
}

Las matrices separadas solo se pueden recorrer si tienen la misma longitud y también tienen el mismo nombre de clave. Esto significa que si no proporciona una clave y están numeradas, estará bien, o si nombra las claves y las coloca en el mismo orden en cada matriz.

También puede usar array_combine.

$combinedArray = array_combine($people, $foods);
// $combinedArray = ['Tim' => 'chicken', 'Tony' => 'beef', 'Turanga' => 'slurm'];

Luego puede recorrer esto haciendo lo mismo que antes:

foreach ($combinedArray as $person => $meal) {
    echo "$person likes $meal\n";
}

Usando un índice incremental #

Este método funciona incrementando un número entero de 0 al mayor índice de la matriz.

$colors = ['red', 'yellow', 'blue', 'green'];
for ($i = 0; $i < count($colors); $i++) {
    echo 'I am the color ' . $colors[$i] . '<br>';
}

Esto también permite iterar una matriz en orden inverso sin usar array_reverse, lo que puede generar una sobrecarga si la matriz es grande.

$colors = ['red', 'yellow', 'blue', 'green'];
for ($i = count($colors) - 1; $i >= 0; $i--) {
    echo 'I am the color ' . $colors[$i] . '<br>';
}

Puede omitir o rebobinar el índice fácilmente usando este método.

$array = ["alpha", "beta", "gamma", "delta", "epsilon"];
for ($i = 0; $i < count($array); $i++) {
    echo $array[$i], PHP_EOL;
    if ($array[$i] === "gamma") {
        $array[$i] = "zeta";
        $i -= 2;
    } elseif ($array[$i] === "zeta") {
        $i++;
    }
}

Producción:

alpha
beta
gamma
beta
zeta
epsilon

Para arreglos que no tienen índices incrementales (incluidos los arreglos con índices en orden inverso, por ejemplo, [1 => "foo", 0 => "bar"], ["foo" => "f", "bar" => "b"]), esto no se puede hacer directamente. En su lugar, se pueden usar array_values o array_keys:

$array = ["a" => "alpha", "b" => "beta", "c" => "gamma", "d" => "delta"];
$keys = array_keys($array);
for ($i = 0; $i < count($array); $i++) {
    $key = $keys[$i];
    $value = $array[$key];
    echo "$value is $key\n";
}

Uso de punteros de matriz internos #

Cada instancia de matriz contiene un puntero interno. Al manipular este puntero, se pueden recuperar diferentes elementos de una matriz de la misma llamada en diferentes momentos.

Usando cada #

Cada llamada a each() devuelve la clave y el valor del elemento de matriz actual e incrementa el puntero de matriz interna.

$array = ["f" => "foo", "b" => "bar"];
while (list($key, $value) = each($array)) {
    echo "$value begins with $key";
}

Usando siguiente #

$array = ["Alpha", "Beta", "Gamma", "Delta"];
while (($value = next($array)) !== false) {
    echo "$value\n";
}

Tenga en cuenta que este ejemplo asume que ningún elemento en la matriz es idéntico al booleano false. Para evitar tal suposición, use key para verificar si el puntero interno ha llegado al final de la matriz:

$array = ["Alpha", "Beta", "Gamma", "Delta"];
while (key($array) !== null) {
    echo current($array) . PHP_EOL;
    next($array);
}

Esto también facilita la iteración de una matriz sin un bucle directo:

class ColorPicker {
    private $colors = ["#FF0064", "#0064FF", "#64FF00", "#FF6400", "#00FF64", "#6400FF"];
    public function nextColor() : string {
        $result = next($colors);
        // if end of array reached
        if (key($colors) === null) {
            reset($colors);
        }
        return $result;
    }
}

Uso de foreach #

Bucle directo #

foreach ($colors as $color) {
    echo "I am the color $color<br>";
}

Bucle con teclas #

$foods = ['healthy' => 'Apples', 'bad' => 'Ice Cream'];
foreach ($foods as $key => $food) {
    echo "Eating $food is $key";
}

Bucle por referencia #

En los bucles foreach de los ejemplos anteriores, modificar el valor ($color o $food) directamente no cambia su valor en la matriz. El operador & es necesario para que el valor sea un puntero de referencia al elemento de la matriz.

$years = [2001, 2002, 3, 4];
foreach ($years as &$year) {
    if ($year < 2000) $year += 2000;
}

Esto es similar a:

$years = [2001, 2002, 3, 4];
for($i = 0; $i < count($years); $i++) { // these two lines
    $year = &$years[$i];                // are changed to foreach by reference
    if($year < 2000) $year += 2000;
}

Concurrencia #

Las matrices de PHP se pueden modificar de cualquier forma durante la iteración sin problemas de concurrencia (a diferencia, por ejemplo, de las Listas de Java). Si la matriz se itera por referencia, las iteraciones posteriores se verán afectadas por los cambios en la matriz. De lo contrario, los cambios en la matriz no afectarán las iteraciones posteriores (como si estuviera iterando una copia de la matriz). Compara bucles por valor:

$array = [0 => 1, 2 => 3, 4 => 5, 6 => 7];
foreach ($array as $key => $value) {
    if ($key === 0) {
        $array[6] = 17;
        unset($array[4]);
    }
    echo "$key => $value\n";
}

Producción:

0 => 1
2 => 3
4 => 5
6 => 7

Pero si la matriz se itera con referencia,

$array = [0 => 1, 2 => 3, 4 => 5, 6 => 7];
foreach ($array as $key => &$value) {
    if ($key === 0) {
        $array[6] = 17;
        unset($array[4]);
    }
    echo "$key => $value\n";
}

Producción:

0 => 1
2 => 3
6 => 17

El conjunto clave-valor de 4 => 5 ya no se itera, y 6 => 7 se cambia a 6 => 17.

Uso del iterador ArrayObject #

Php arrayiterator le permite modificar y anular los valores mientras itera sobre matrices y objetos.

Ejemplo:

$array = ['1' => 'apple', '2' => 'banana', '3' => 'cherry'];

$arrayObject = new ArrayObject($array);

$iterator = $arrayObject->getIterator();

for($iterator; $iterator->valid(); $iterator->next()) {
    echo $iterator->key() . ' => ' . $iterator->current() . "</br>";
}

Producción:

1 => apple
2 => banana
3 => cherry
