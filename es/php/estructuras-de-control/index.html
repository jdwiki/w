si no #

La declaración if en el ejemplo anterior permite ejecutar un fragmento de código, cuando se cumple la condición. Cuando desee ejecutar un fragmento de código, cuando no se cumpla la condición, extienda el ‘if’ con un ’else’.

if ($a > $b) {
  echo "a is greater than b";
} else {
  echo "a is NOT greater than b";
}

Manual PHP - Estructuras de Control - Else

El operador ternario como sintaxis abreviada para if-else

El operador ternario evalúa algo en función de si una condición es verdadera o no. Es un operador de comparación y se usa a menudo para expresar una condición if-else simple en una forma más corta. Permite probar rápidamente una condición y, a menudo, reemplaza una declaración if de varias líneas, lo que hace que su código sea más compacto.

Este es el ejemplo de arriba usando una expresión ternaria y valores de variables: $a=1; $b=2;

echo ($a > $b) ? "a is greater than b" : "a is NOT greater than b";

Salidas: a NO es mayor que b.

Sintaxis alternativa para estructuras de control #

PHP proporciona una sintaxis alternativa para algunas estructuras de control: if, while, for, foreach y switch.

Cuando se compara con la sintaxis normal, la diferencia es que la llave de apertura se reemplaza por dos puntos (:) y la llave de cierre se reemplaza por endif;, endwhile;, endfor;, endforeach; , o endswitch;, respectivamente. Para ver ejemplos individuales, consulte el tema sobre sintaxis alternativa para estructuras de control.

if ($a == 42):
    echo "The answer to life, the universe and everything is 42.";
endif;

Múltiples sentencias elseif usando una sintaxis corta:

if ($a == 5):
    echo "a equals 5";
elseif ($a == 6):
    echo "a equals 6";
else:
    echo "a is neither 5 nor 6";
endif;

Manual de PHP - Estructuras de control - Sintaxis alternativa

tiempo #

El bucle while itera a través de un bloque de código siempre que una condición específica sea verdadera.

$i = 1;
while ($i < 10) {
    echo $i;
    $i++;
}

Salida: 123456789

Para obtener información detallada, consulte [el tema de bucles] (https://www.wikiod.com/es/php/bucles#tiempo).

hacer mientras #

El bucle do-while primero ejecuta un bloque de código una vez, en todos los casos, luego itera a través de ese bloque de código siempre que una condición específica sea verdadera.

$i = 0;
do {
    $i++;
    echo $i;
} while ($i < 10);

Output: `12345678910`

Para obtener información detallada, consulte [el tema de bucles] (https://www.wikiod.com/es/php/bucles#hacer… mientras).

ir #

El operador goto permite saltar a otra sección del programa. Está disponible desde PHP 5.3.

La instrucción goto es un goto seguido de la etiqueta de destino deseada: goto MyLabel;.

El destino del salto se especifica mediante una etiqueta seguida de dos puntos: MyLabel:.

Este ejemplo imprimirá Hello World!:

<?php
goto MyLabel;
echo 'This text will be skipped, because of the jump.';
 
MyLabel:
echo 'Hello World!';
?>

declarar #

declare se usa para establecer una directiva de ejecución para un bloque de código.

Se reconocen las siguientes directivas:

  • marcas
  • codificación
  • tipos_estrictos

Por ejemplo, establezca las marcas en 1:

declare(ticks=1);

Para habilitar el modo de tipo estricto, la declaración declare se usa con la declaración strict_types:

declare(strict_types=1);

incluir y requerir #

requerir #

require es similar a include, excepto que producirá un error fatal de nivel E_COMPILE_ERROR en caso de falla. Cuando el require falla, detendrá el script. Cuando incluir falla, no detendrá el script y solo emitirá E_WARNING.

require 'file.php';

Manual PHP - Estructuras de control - Requerir

incluir #

La instrucción include incluye y evalúa un archivo.

./variables.php

$a = 'Hello World!';

./principal.php`

include 'variables.php';
echo $a;
// Output: `Hello World!`

Tenga cuidado con este enfoque, ya que se considera un olor de código, porque el archivo incluido altera la cantidad y el contenido de las variables definidas en el ámbito dado.

También puede incluir el archivo, que devuelve un valor. Esto es extremadamente útil para manejar arreglos de configuración:

configuración.php

<?php 
return [
    'dbname' => 'my db',
    'user' => 'admin',
    'pass' => 'password',
];

principal.php

<?php
$config = include 'configuration.php';

Este enfoque evitará que el archivo incluido contamine su alcance actual con variables modificadas o agregadas.

Manual PHP - Estructuras de Control - Incluir

include & require también se puede usar para asignar valores a una variable cuando se devuelve algo por archivo.

Ejemplo :

archivo include1.php:

<?php
    $a = "This is to be returned";

    return $a;
?>

archivo index.php:

    $value = include 'include1.php';
   // Here, $value = "This is to be returned"

devolver #

La instrucción return devuelve el control del programa a la función de llamada.

Cuando se llama a return desde dentro de una función, la ejecución de la función actual finalizará.

function returnEndsFunctions()
{
   echo 'This is executed';
   return;
   echo 'This is not executed.';
}

Cuando ejecuta returnEndsFunctions(); obtendrá el resultado Esto se ejecuta;

Cuando se llama a return desde dentro de una función con un argumento, la ejecución de la función actual finalizará y el valor del argumento se devolverá a la función de llamada.

por #

Los bucles for se utilizan normalmente cuando tienes un fragmento de código que quieres repetir un número determinado de veces.

for ($i = 1; $i < 10; $i++) {
    echo $i;
}

Salidas: 123456789

Para obtener información detallada, consulte [el tema de bucles] (https://www.wikiod.com/es/php/bucles#por).

para cada #

foreach es una construcción que le permite iterar sobre arreglos y objetos fácilmente.

$array = [1, 2, 3];
foreach ($array as $value) {
    echo $value;
}

Salidas: 123.

Para usar el bucle foreach con un objeto, tiene que implementar la interfaz Iterator.

Cuando itera sobre matrices asociativas:

$array = ['color'=>'red']; 

foreach($array as $key => $value){
    echo $key . ': ' . $value; 
}

Salidas: color: rojo

Para obtener información detallada, consulte [el tema de bucles] (https://www.wikiod.com/es/php/bucles#para cada).

si más si más #

si no

elseif combina if y else. La sentencia if se amplía para ejecutar una sentencia diferente en caso de que no se cumpla la expresión if original. Pero, la expresión alternativa solo se ejecuta cuando se cumple la expresión condicional elseif.

El siguiente código muestra “a es mayor que b”, “a es igual a b” o “a es menor que b”:

if ($a > $b) {
    echo "a is bigger than b";
} elseif ($a == $b) {
    echo "a is equal to b";
} else {
    echo "a is smaller than b";
}

Varias declaraciones elseif

Puede usar varias sentencias elseif dentro de la misma sentencia if:

if ($a == 1) {
    echo "a is One";
} elseif ($a == 2) {
    echo "a is Two";
} elseif ($a == 3) {
    echo "a is Three";
} else {
    echo "a is not One, not Two nor Three";
}

si #

La construcción if permite la ejecución condicional de fragmentos de código.

if ($a > $b) {
  echo "a is bigger than b";
}

Manual PHP - Estructuras de Control - Si

cambiar #

La estructura switch realiza la misma función que una serie de sentencias if, pero puede hacer el trabajo en menos líneas de código. El valor que se va a probar, tal como se define en la sentencia switch, se compara para determinar si es igual a los valores de cada una de las sentencias case hasta que se encuentra una coincidencia y se ejecuta el código de ese bloque. Si no se encuentra ninguna sentencia case coincidente, se ejecuta el código del bloque default, si existe.

Cada bloque de código en una instrucción case o default debe terminar con la declaración break. Esto detiene la ejecución de la estructura switch y continúa la ejecución del código inmediatamente después. Si se omite la sentencia break, se ejecuta el siguiente código de sentencia case, incluso si no hay ninguna coincidencia. Esto puede causar una ejecución de código inesperada si se olvida la declaración break, pero también puede ser útil cuando varias declaraciones case deben compartir el mismo código.

switch ($colour) {
case "red":
    echo "the colour is red";
    break;
case "green":
case "blue":
    echo "the colour is green or blue";
    break;
case "yellow":
    echo "the colour is yellow";
    // note missing break, the next block will also be executed
case "black":
    echo "the colour is black";
    break;
default:
    echo "the colour is something else";
    break;
}

Además de probar valores fijos, la construcción también puede ser obligada a probar sentencias dinámicas proporcionando un valor booleano a la sentencia switch y cualquier expresión a la sentencia case. Tenga en cuenta que se usa el primer valor coincidente, por lo que el siguiente código generará “más de 100”:

$i = 1048;
switch (true) {
case ($i > 0):
    echo "more than 0";
    break;
case ($i > 100):
    echo "more than 100";
    break;
case ($i > 1000):
    echo "more than 1000";
    break;
}

Para posibles problemas con la escritura suelta al usar la construcción switch, consulte Sorpresas de Switch
