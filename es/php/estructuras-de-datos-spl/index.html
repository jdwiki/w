Matriz Fija Spl #

Diferencia de PHP Array #

El tipo de matriz predeterminado de PHP en realidad se implementa como mapas hash ordenados, lo que nos permite crear matrices que consisten en pares clave/valor donde los valores pueden ser de cualquier tipo y las claves pueden ser números o cadenas. Sin embargo, no es así como se crean tradicionalmente las matrices.

Entonces, como puede ver en esta ilustración, una matriz PHP normal se puede ver más como un conjunto ordenado de pares clave/valor, donde cada clave puede asignarse a cualquier valor. Observe que en esta matriz tenemos claves que son tanto números como cadenas, así como valores de diferentes tipos y la clave no tiene relación con el orden de los elementos.

$arr = [
    9     => "foo",
    1     => 4.2,
    "bar" => null,
];

foreach($arr as $key => $value) {
    echo "$key => $value\n";
}

Entonces, el código anterior nos daría exactamente lo que esperábamos.

9 => foo 1 => 4,2 barra =>

Las matrices PHP regulares también tienen un tamaño dinámico para nosotros. Crecen y se reducen a medida que empujamos y extraemos valores hacia y desde la matriz, automáticamente.

Sin embargo, en una matriz tradicional, el tamaño es fijo y consiste completamente en el mismo tipo de valor. Además, en lugar de claves, se accede a cada valor por su índice, que se puede deducir por su desplazamiento en la matriz.

Dado que sabríamos el tamaño de un tipo dado y el tamaño fijo de la matriz, un desplazamiento es entonces el tamaño de tipo * n donde n representa la posición del valor en la matriz. Entonces, en el ejemplo anterior, $arr[0] nos da 1, el primer elemento de la matriz y $arr[1] nos da 2, y así sucesivamente.

SplFixedArray, sin embargo, no restringe el tipo de valores. Solo restringe las claves a tipos de números. También es de un tamaño fijo.

Esto hace que SplFixedArrays sea más eficiente que las matrices PHP normales de una manera particular. Son más compactos por lo que requieren menos memoria.

Instanciando la matriz #

SplFixedArray se implementa como un objeto, pero se puede acceder a él con la misma sintaxis familiar con la que se accede a una matriz PHP normal, ya que implementan la interfaz ArrayAccess. También implementan las interfaces Countable e Iterator para que se comporten de la misma manera en que estarías acostumbrado a que las matrices se comporten en PHP (es decir, cosas como count($arr) y foreach($arr as $k => $ v) funciona de la misma manera para SplFixedArray que para matrices normales en PHP.

El constructor SplFixedArray toma un argumento, que es el tamaño de la matriz.

$arr = new SplFixedArray(4);

$arr[0] = "foo";
$arr[1] = "bar";
$arr[2] = "baz";

foreach($arr as $key => $value) {
    echo "$key => $value\n";
}

Esto le da lo que usted esperaría.

0 => foo 1 => barra 2 => baz 3 =>

Esto también funciona como se esperaba.

var_dump(count($arr));

Nos da…

int(4)

Observe que en SplFixedArray, a diferencia de una matriz PHP normal, la clave representa el orden del elemento en nuestra matriz, porque es un índice verdadero y no solo un mapa.

Cambiar el tamaño de la matriz #

Solo tenga en cuenta que debido a que la matriz tiene un tamaño fijo, count siempre devolverá el mismo valor. Entonces, mientras unset($arr[1]) resultará en $arr[1] === null, count($arr) sigue siendo 4.

Entonces, para cambiar el tamaño de la matriz, deberá llamar al método setSize.

$arr->setSize(3);

var_dump(count($arr));

foreach($arr as $key => $value) {
    echo "$key => $value\n";
}

Ahora tenemos…

int(3) 0 => foo 1 => 2 => baz

Importar a SplFixedArray y exportar desde SplFixedArray #

También puede importar/exportar un PHP Array normal dentro y fuera de un SplFixedArray con los métodos fromArray y toArray.

$array      = [1,2,3,4,5];
$fixedArray = SplFixedArray::fromArray($array);

foreach($fixedArray as $value) {
    echo $value, "\n";
}
1 2 3 4 5

Yendo hacia el otro lado.

$fixedArray = new SplFixedArray(5);

$fixedArray[0] = 1;
$fixedArray[1] = 2;
$fixedArray[2] = 3;
$fixedArray[3] = 4;
$fixedArray[4] = 5;

$array = $fixedArray->toArray();

foreach($array as $value) {
    echo $value, "\n";
}
1 2 3 4 5
