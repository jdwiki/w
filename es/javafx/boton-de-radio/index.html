Creación de botones de opción #

Los botones de radio le permiten al usuario elegir un elemento de los dados. Hay dos formas de declarar un RadioButton con un texto al lado. Ya sea usando el constructor predeterminado RadioButton() y configurando el texto con el método setText(String) o usando el otro constructor RadioButton(String).

RadioButton radioButton1 = new RadioButton();
radioButton1.setText("Select me!");
RadioButton radioButton2= new RadioButton("Or me!");

Como RadioButton es una extensión de Labeled, también puede haber una Imagen especificada para RadioButton. Después de crear el RadioButton con uno de los constructores, simplemente agregue la Imagen con el método setGraphic(ImageView) como aquí:

Image image = new Image("ok.jpg");
RadioButton radioButton = new RadioButton("Agree");
radioButton.setGraphic(new ImageView(image));

Usar grupos en botones de opción #

Se usa un ToggleGroup para administrar los RadioButtons de modo que solo se pueda seleccionar uno en cada grupo en cada momento.

Cree un ToggleGroup simple como el siguiente:

ToggleGroup group = new ToggleGroup();

Después de crear un Togglegroup, se puede asignar a RadioButtons usando setToggleGroup(ToggleGroup). Utilice setSelected(Boolean) para preseleccionar uno de los RadioButtons.

RadioButton radioButton1 = new RadioButton("stackoverlow is awesome! :)");
radioButton1.setToggleGroup(group);
radioButton1.setSelected(true);

RadioButton radioButton2 = new RadioButton("stackoverflow is ok :|");
radioButton2.setToggleGroup(group);

RadioButton radioButton3 = new RadioButton("stackoverflow is useless :(");
radioButton3.setToggleGroup(group);

Eventos para botones de radio #

Por lo general, cuando se selecciona uno de los RadioButtons en un ToggleGroup, la aplicación realiza una acción. A continuación se muestra un ejemplo que imprime los datos de usuario del ‘RadioButton’ seleccionado que se ha configurado con ‘setUserData(Object)’.

radioButton1.setUserData("awesome")
radioButton2.setUserData("ok");
radioButton3.setUserData("useless");

ToggleGroup group = new ToggleGroup();
group.selectedToggleProperty().addListener((obserableValue, old_toggle, new_toggle) -> {
    if (group.getSelectedToggle() != null) {
        System.out.println("You think that stackoverflow is " + group.getSelectedToggle().getUserData().toString());
    }
});

Solicitud de enfoque para botones de opción #

Digamos que el segundo RadioButton de tres está preseleccionado con setSelected(Boolean), el foco aún está en el primer RadioButton por defecto. Para cambiar esto usa el método requestFocus().

radioButton2.setSelected(true);
radioButton2.requestFocus();
