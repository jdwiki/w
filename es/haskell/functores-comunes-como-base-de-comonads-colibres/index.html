Cofree Vacío ~~ Vacío #

Dado

data Empty a

tenemos

data Cofree Empty a
   --  = a :< ...  not possible!

Cofree (Const c) ~~ Escritor c #

Dado

data Const c a = Const c

tenemos

data Cofree (Const c) a
     = a :< Const c

que es isomorfo a

data Writer c a = Writer c a

Identidad Cofree ~~ Corriente #

Dado

data Identity a = Identity a

tenemos

data Cofree Identity a
     = a :< Identity (Cofree Identity a)

que es isomorfo a

data Stream a = Stream a (Stream a)

Cofree Quizás ~~ NonEmpty #

Dado

data Maybe a = Just a
             | Nothing

tenemos

data Cofree Maybe a
     = a :< Just (Cofree Maybe a)
     | a :< Nothing

que es isomorfo a

data NonEmpty a
     = NECons a (NonEmpty a)
     | NESingle a

Cofree (Escritor w) ~~ WriterT w Stream #

Dado

data Writer w a = Writer w a

tenemos

data Cofree (Writer w) a
     = a :< (w, Cofree (Writer w) a)

que es equivalente a

data Stream (w,a)
     = Stream (w,a) (Stream (w,a))

que puede escribirse correctamente como WriterT w Stream con

data WriterT w m a = WriterT (m (w,a))

Cofree (Cualquier e) ~~ NonEmptyT (Escritor e) #

Dado

data Either e a = Left e
                | Right a

tenemos

data Cofree (Either e) a
     = a :< Left e
     | a :< Right (Cofree (Either e) a)

que es isomorfo a

data Hospitable e a
     = Sorry_AllIHaveIsThis_Here'sWhy a e
     | EatThis a (Hospitable e a)

o, si promete evaluar el registro solo después del resultado completo, NonEmptyT (Writer e) a con

data NonEmptyT (Writer e) a = NonEmptyT (e,a,[a])

Cofree (Lector x) ~~ Moore x #

Dado

data Reader x a = Reader (x -> a)

tenemos

data Cofree (Reader x) a
     = a :< (x -> Cofree (Reader x) a)

que es isomorfo a

data Plant x a
     = Plant a (x -> Plant x a)

también conocido como [máquina Moore] (http://hackage.haskell.org/package/machines-0.6.1/docs/Data-Machine-Moore.html#t:Moore).
