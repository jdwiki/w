Aquí vamos a discutir sobre la historia de Node.js, la información de la versión y su estado actual.

Eventos clave en cada año #

#2009 #

  • 3 de marzo: El proyecto se denominó “nodo”
  • 1 de octubre: Primera vista previa muy temprana de npm, el paquete Node administrador
  • 8 de noviembre: Original de Ryan Dahl (Creador de Node.js) Charla de Node.js en JSConf 2009

2010 #

  • Express: un marco de desarrollo web Node.js
  • Lanzamiento inicial de Socket.io
  • 28 de abril: [Soporte experimental de Node.js en Heroku] (https://blog.heroku.com/archives/2010/4/28/node_js_support_experimental)
  • 28 de julio: Charla técnica de Google de Ryan Dahl sobre Node.js
  • 20 de agosto: Lanzamiento de Node.js 0.2.0

#2011 #

  • 31 de marzo: Guía de Node.js
  • 1 de mayo: npm 1.0: Lanzamiento
  • 1 de mayo: AMA de Ryan Dahl en Reddit
  • 10 de julio: [Se completa el Libro para principiantes de Node, una introducción a Node.js] (http://nodebeginner.org/index.html).
    • A comprehensive Node.js tutorial for beginners.
  • 16 de agosto: LinkedIn usa Node.js
    • LinkedIn launched its completely overhauled mobile app with new features and new parts under the hood.
  • 5 de octubre: Ryan Dahl habla sobre la historia de Node.js y por qué lo creó
  • 5 de diciembre: Node.js en producción en Uber
    • Uber Engineering Manager Curtis Chambers explains why his company completely re-engineered their application using Node.js to increase eficiencia y mejorar la experiencia de socios y clientes.

#2012 #

  • 30 de enero: el creador de Node.js, Ryan Dahl, se aleja del día a día de Node
  • 25 de junio: Node.js v0.8.0 [stable] sale
  • 20 de diciembre: se lanza [Hapi, un marco Node.js] (http://hueniverse.com/2012/12/20/hapi-a-prologue/)

#2013 #

  • 30 de abril: The MEAN Stack: MongoDB, ExpressJS, AngularJS y Node.js
  • 17 de mayo: [Cómo creamos la primera aplicación Node.js de eBay] (http://www.ebaytechblog.com/2013/05/17/how-we-build-ebays-first-node-js-application/)
  • 15 de noviembre: [PayPal lanza Kraken, un marco Node.js] (https://github.com/krakenjs/kraken-js/releases/tag/v0.6.1)
  • 22 de noviembre: Fuga de memoria de Node.js en Walmart
    • Eran Hammer of Wal-Mart labs came to the Node.js core team complaining of a memory leak he had been tracking down for months.
  • 19 de diciembre: Koa - Framework web para Node.js

#2014 #

  • 15 de enero: TJ Fontaine se hace cargo del proyecto Node
  • 23 de octubre: Consejo asesor de Node.js
    • Joyent and several members of the Node.js community announced a proposal for a Node.js Advisory Board as a next step towards a fully open modelo de gobierno para el proyecto de código abierto Node.js.
  • 19 de noviembre: Node.js en Flame Graphs - Netflix
  • 28 de noviembre: IO.js – E/S con eventos para V8 Javascript

#2015 #

P1 #

  • 14 de enero: IO.js 1.0.0
  • 10 de febrero: [Joyent avanza para establecer la fundación Node.js] (http://www.joyent.com/about/press/joyent-moves-to-establecer-nodejs-foundation)
    • Joyent, IBM, Microsoft, PayPal, Fidelity, SAP and The Linux Foundation Join Forces to Support Node.js Community With Neutral and Open Governance
  • 27 de febrero: Propuesta de reconciliación de IO.js y Node.js

Q2 #

  • 14 de abril: [Módulos privados de npm] (https://www.npmjs.com/private-modules)
  • 28 de mayo: [El líder de nodo, TJ Fontaine, renuncia y deja Joyent] (http://venturebeat.com/2015/05/08/node-lead-tj-fontaine-is-stepping-down-and-leaving-joyent) -también/)
  • 13 de mayo: Node.js e io.js se fusionan bajo Node Foundation

Q3 #

  • 2 de agosto: [Trace - Supervisión y depuración del rendimiento de Node.js] (http://trace.risingstack.com/)
    • Trace is a visualized microservice monitoring tool that gets you all the metrics you need when operating microservices.
  • 13 de agosto: 4.0 es el nuevo 1.0

Q4 #

  • 12 de octubre: [Node v4.2.0, primera versión de soporte a largo plazo] (https://nodejs.org/en/blog/release/v4.2.0/)
  • 8 de diciembre: [Apigee, RisingStack y Yahoo se unen a la Fundación Node.js] (http://finance.yahoo.com/news/apigee-risingstack-yahoo-join-node-170000939.html)
  • 8 y 9 de diciembre: Node Interactive
    • The first annual Node.js conference by the Node.js Foundation

#2016 #

P1 #

  • 10 de febrero: Express se convierte en un proyecto incubado
  • 23 de marzo: El incidente del leftpad
  • 29 de marzo: Google Cloud Platform se une a la Fundación Node.js

Q2 #

  • 26 de abril: npm tiene 210.000 usuarios

Q3 #

  • 18 de julio: CJ Silverio se convierte en el CTO de npm
  • 1 de agosto: Trace, la solución de depuración de Node.js estará disponible de forma general
  • 15 de septiembre: El primer Node Interactive en Europa

Q4 #

  • 11 de octubre: [Se lanzó el administrador de paquetes de hilo] (https://code.facebook.com/posts/1840075619545360)
  • 18 de octubre: Node.js 6 se convierte en la versión LTS
