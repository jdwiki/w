Enrutamiento de servidor web exprés #

Creando Servidor Web Express

El servidor Express fue útil y se profundiza a través de muchos usuarios y comunidades. Se está volviendo popular.

Vamos a crear un servidor Express. Para la gestión de paquetes y la flexibilidad para la dependencia, utilizaremos NPM (Node Package Manager).

  1. Vaya al directorio Proyecto y cree el archivo package.json. package.json { “nombre”: “exprésRouter”, “versión”: “0.0.1”, “guiones”: { “start”: “node Server.js” }, “dependencias”: { “express”: “^4.12.3” } }

  2. Guarde el archivo e instale la dependencia express usando el siguiente comando npm install. Esto creará node_modules en el directorio de su proyecto junto con la dependencia requerida.

  3. Vamos a crear Express Web Server. Go to the Project directory and create server.js file. servidor.js

    var express = require(“express”); var app = express();

//Creando objeto enrutador()

var enrutador = express.Router();

// Proporcione todas las rutas aquí, esto es para la página de inicio.

router.get("/",function(req,res){
res.json({"message" : "Hello World"});

});

app.use("/api",enrutador);

// Escuche este puerto

aplicación.escucha(3000,función(){ console.log(“Live at Port 3000”); });

For more detail on setting node server you can see [here][1].

  1. Ejecute el servidor escribiendo el siguiente comando.

    node server.js

    If Server runs successfully, you will se something like this..

  2. Ahora ve al navegador o al cartero y realiza una solicitud

    http://localhost:3000/api/

    The output will be .

Eso es todo, lo básico del enrutamiento Express.

Ahora manejemos GET, POST, etc.

Cambie su archivo server.js como

var express = require("express");
var app = express();

//Creating Router() object

var router = express.Router();

// Router middleware, mentioned it before defining routes.

router.use(function(req,res,next) {
  console.log("/" + req.method);
  next();
});

// Provide all routes here, this is for Home page.

router.get("/",function(req,res){
  res.json({"message" : "Hello World"});
});


app.use("/api",router);


app.listen(3000,function(){
  console.log("Live at Port 3000");
});

Ahora, si reinicia el servidor y realiza la solicitud a

http://localhost:3000/api/

Acceso al parámetro en enrutamiento

También puede acceder al parámetro desde la URL, como http://example.com/api/:name/. Entonces se puede acceder al parámetro de nombre. Agregue el siguiente código en su server.js

router.get("/user/:id",function(req,res){
  res.json({"message" : "Hello "+req.params.id});
});

Ahora reinicie el servidor y vaya a http://localhost:3000/api/user/Adem, la salida será como .
