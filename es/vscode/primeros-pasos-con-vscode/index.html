Instalación o configuración #

en ventanas #

  • Descargue el instalador de Visual Studio Code para Windows.
  • Una vez descargado, ejecuta el instalador (VSCodeSetup-version.exe). Esto solo tomará un minuto.

De manera predeterminada, VS Code se instala en C:\Program Files (x86)\Microsoft VS Code para una máquina de 64 bits.

Nota: Se requiere .NET Framework 4.5.2 para VS Code. Si está utilizando Windows 7, asegúrese de que esté instalado .NET Framework 4.5.2.

Sugerencia: El programa de instalación opcionalmente agregará Visual Studio Code a su %PATH%, por lo que desde la consola puede escribir ‘code’. para abrir VS Code en esa carpeta. Deberá reiniciar su consola después de la instalación para que el cambio en la variable ambiental %PATH% surta efecto.

En Mac #

  • Descargar Visual Studio Code para Mac.
  • Haga doble clic en el archivo descargado para expandir el contenido.
  • Arrastre Visual Studio Code.app a la carpeta Aplicaciones para que esté disponible en el Launchpad.
  • Agregue VS Code a su Dock haciendo clic con el botón derecho en el ícono y eligiendo Opciones, Mantener en el Dock.

También puede ejecutar VS Code desde la terminal escribiendo ‘código’ después de agregarlo a la ruta:

  • Inicie el código VS.
  • Abra la paleta de comandos (Ctrl+Shift+P) y escriba ‘comando de shell’ para encontrar el comando de Shell: instale el comando ‘código’ en el comando PATH.

Reinicie el terminal para que se aplique el nuevo valor de $PATH. Podrá escribir ‘código’. en cualquier carpeta para comenzar a editar archivos en esa carpeta.

Nota: Si aún tiene el alias de código antiguo en su .bash_profile (o equivalente) de una versión anterior de VS Code, elimínelo y reemplácelo ejecutando el comando Shell: instale el comando ‘código’ en el comando PATH.

en linux #

Distribuciones basadas en Debian y Ubuntu #

La forma más sencilla de instalar distribuciones basadas en Debian/Ubuntu es descargar e instalar el paquete .deb (64 bits) a través del centro de software gráfico si está disponible o a través de la línea de comandos con:

sudo dpkg -i <file>.deb
sudo apt-get install -f # Install dependencies

La instalación del paquete .deb instalará automáticamente el repositorio apt y la clave de firma para habilitar la actualización automática utilizando el mecanismo normal del sistema. Tenga en cuenta que los binarios de 32 bits y .tar.gz también están disponibles en la página de descarga.

El repositorio y la clave también se pueden instalar manualmente con el siguiente script:

curl https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > microsoft.gpg
sudo mv microsoft.gpg /etc/apt/trusted.gpg.d/microsoft.gpg
sudo sh -c 'echo "deb [arch=amd64] https://packages.microsoft.com/repos/vscode stable main" > /etc/apt/sources.list.d/vscode.list'

Luego actualice el caché del paquete e instale el paquete usando:

sudo apt-get update
sudo apt-get install code # or code-insiders for insiders build

Distribuciones basadas en RHEL, Fedora y CentOS #

Actualmente enviamos el VS Code estable de 64 bits en un repositorio yum, el siguiente script instalará la clave y el repositorio:

sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
sudo sh -c 'echo -e "[code]\nname=Visual Studio Code\nbaseurl=https://packages.microsoft.com/yumrepos/vscode\nenabled=1\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" > /etc/yum.repos.d/vscode.repo'

Luego actualice el caché del paquete e instale el paquete usando dnf (Fedora 22 y superior):

dnf check-update
sudo dnf install code

O en versiones anteriores usando yum:

yum check-update
sudo yum install code

Distribuciones basadas en openSUSE y SLE #

El repositorio yum anterior también funciona para sistemas basados ​​en openSUSE y SLE, el siguiente script instalará la clave y el repositorio:

sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
sudo sh -c 'echo -e "[code]\nname=Visual Studio Code\nbaseurl=https://packages.microsoft.com/yumrepos/vscode\nenabled=1\ntype=rpm-md\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" > /etc/zypp/repos.d/vscode.repo'

Luego actualice el caché del paquete e instale el paquete usando:

sudo zypper refresh
sudo zypper install code

Paquete AUR para Arch Linux #

Existe un [paquete Arch User Repository (AUR)] 4 mantenido por la comunidad para VS Code.

Instalar el paquete .rpm manualmente El paquete .rpm (64 bits) también se puede descargar e instalar manualmente; sin embargo, la actualización automática no funcionará a menos que se instale el repositorio anterior. Una vez descargado, se puede instalar usando su administrador de paquetes, por ejemplo con dnf:

sudo dnf install <file>.rpm

Tenga en cuenta que los binarios de 32 bits y .tar.gz también están disponibles en la página de descarga.

Primeros pasos (C++): HolaMundo.cpp #

El primer programa que uno escribe normalmente en cualquier idioma es el script “hola mundo”. Este ejemplo demuestra cómo escribir este programa y depurarlo usando Visual Studio Code (a partir de ahora me referiré a Visual Studio Code como VS Code).

Crear el proyecto

El paso 1 será crear un nuevo proyecto. Esto se puede hacer de varias maneras. La primera forma es directamente desde la interfaz de usuario.

  1. Abra el programa VS Code. Será recibido con la pantalla de bienvenida estándar (tenga en cuenta que las imágenes se tomaron mientras trabajaba en una Mac, pero deberían ser similares a su instalación):

  1. En el menú Inicio, seleccione Nuevo archivo. Esto abrirá una nueva ventana de edición donde podemos comenzar a construir nuestro script. Continúe y guarde este archivo (puede usar el menú Archivo > Guardar para hacer esto). Para este ejemplo, llamaremos al archivo HelloWorld.cpp y lo colocaremos en un nuevo directorio al que llamaremos VSC_HelloWorld/.

  2. Escriba el programa. Esto debería ser bastante sencillo, pero no dude en copiar lo siguiente en el archivo:

    #include

    int main() { std::cout « “Hello world!” « std::endl; return 0; }

Ejecutar el código

A continuación, queremos ejecutar el script y verificar su salida. hay muchas maneras de hacer esto. La más sencilla es abrir una terminal y navegar hasta el directorio que creamos. Ahora puede compilar el script y ejecutarlo con gcc escribiendo:

$ g++ HelloWorld.cpp -o helloworld
$ ./helloworld
Hello World!

¡Sí, el programa funcionó! Pero esto no es realmente lo que queremos. Sería mucho mejor si pudiéramos ejecutar el programa desde dentro de VSCode. ¡Aunque estamos de suerte! VSCode tiene una terminal integrada a la que podemos acceder a través del menú “Ver” > “Terminal integrada”. Esto abrirá una terminal en la mitad inferior de la ventana desde la cual puede navegar al directorio VSC_HelloWorld y ejecutar los comandos anteriores.

Por lo general, hacemos esto ejecutando una Ejecutar tarea. En el menú, seleccione “Tareas” > “Ejecutar tarea…”. Notará que aparece una pequeña ventana emergente cerca de la parte superior de la ventana con un mensaje de error (algo parecido a

Primer programa (C++): Hola Mundo.cpp #

Este ejemplo le presenta la funcionalidad básica de VS Code al demostrar cómo escribir un programa “hola mundo” en C++. Antes de continuar, asegúrese de tener instalada la extensión “ms-vscode.cpptools”.

Inicializar el Proyecto

El primer paso es crear un nuevo proyecto. Para hacer esto, cargue el programa VS Code. Debería ser recibido con la típica pantalla de bienvenida:

Para crear el primer programa, seleccione “Inicio” > “Nuevo archivo” en la pantalla de bienvenida. Esto abrirá una nueva ventana de archivo. Continúe y guarde el archivo ("Archivo" > “Guardar”) en un nuevo directorio. Puede nombrar el directorio como desee, pero este ejemplo llamará al directorio “VSC_HelloWorld” y al archivo “HelloWorld.cpp”.

Ahora escriba el programa real (siéntase libre de copiar el texto a continuación):

#include <iostream>

int main()
{
    // Output the hello world text
    std::cout << "Hello world!" << std::endl;
    return 0;
}

¡Excelente! También notará que debido a que instaló la extensión “ms-vscode.cpptools”, también tiene un bonito resaltado de código. Ahora pasemos a ejecutar el código.

Ejecución del script (básico)

Podemos ejecutar “HelloWorld.cpp” desde el mismo VS Code. La forma más sencilla de ejecutar un programa de este tipo es abrir el terminal integrado ("Ver" > “Terminal integrado”). Esto abre una ventana de terminal en la parte inferior de la vista. Desde dentro de este terminal podemos navegar a nuestro directorio creado, compilar y ejecutar el script que hemos escrito. Aquí hemos usado los siguientes comandos para compilar y ejecutar el código:

$ g++ HelloWorld.cpp -o hellowold
$ ./hellowold

Observe que obtenemos la salida esperada Hello World!.

Ejecutando el Script (un poco más avanzado)

Genial, pero también podemos usar VS Code directamente para compilar y ejecutar el código. Para esto, primero debemos convertir el directorio “VSC_HelloWorld” en un espacio de trabajo. Esto se puede hacer por:

  1. Abriendo el menú Explorer (elemento superior en el menú vertical en el extremo izquierdo)
  2. Seleccione el botón Abrir carpeta
  3. Seleccione el directorio “VSC_HelloWorld” en el que hemos estado trabajando. Nota: si abre un directorio dentro de VS Code (usando “Archivo” > “Abrir…” por ejemplo) ya estará en un espacio de trabajo.

El menú Explorer ahora muestra el contenido del directorio.

A continuación, queremos definir las tareas reales que queremos que ejecute VS Code. Para ello, seleccione “Tareas” > “Configurar tarea de compilación predeterminada”. En el menú desplegable, seleccione “Otro”. Esto abre un nuevo archivo llamado “tasks.json” que contiene algunos valores predeterminados para una tarea. Tenemos que cambiar estos valores. Actualice este archivo para que contenga lo siguiente y guárdelo:

{
    "version": "2.0.0",
    "tasks": [
        {
            "taskName": "build",
            "type": "shell",
            "command": "g++ HelloWorld.cpp -o helloworld"
        },
        {
            "taskName": "run",
            "type": "shell",
            "command": "${workspaceRoot}/helloworld"
        }
    ]
}

Tenga en cuenta que lo anterior también crea un directorio oculto .vscode dentro de nuestro directorio de trabajo. Aquí es donde VS Code coloca los archivos de configuración, incluidos los archivos de configuración específicos del proyecto. Puede encontrar más información sobre Tasks aquí.

En el ejemplo anterior, ${workspaceRoot} hace referencia al directorio de nivel superior de nuestro espacio de trabajo, que es nuestro directorio “VSC_HelloWorld”. Ahora, para compilar el proyecto desde dentro del método, seleccione “Tareas” > “Ejecutar tarea de compilación…” y seleccione nuestra tarea “construir” creada y “Continuar sin escanear la tarea output” de los menús desplegables que aparecen. Luego podemos ejecutar el ejecutable usando “Tareas” > “Ejecutar tarea…” y seleccionando la tarea “ejecutar” que creamos. Si tiene abierto el terminal integrado, notará que el mensaje “¡Hola mundo!” el texto se imprimirá allí.

Es posible que la terminal se cierre antes de que pueda ver la salida. Si esto sucede, puede insertar una línea de código como esta int i; std::cin >> i; justo antes de la declaración de retorno al final de la función main(). Luego puede finalizar el script escribiendo cualquier número y presionando <Enter>.

¡Y eso es! Ahora puede comenzar a escribir y ejecutar sus scripts de C++ desde VS Code.
