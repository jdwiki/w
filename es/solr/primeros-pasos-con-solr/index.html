Instalación de SOLR #

Introducción #

El siguiente procedimiento se probó en una instancia de prueba en AWS, con Redhat y Solr 6.1.0. Es posible que deba ajustar el proceso a su sistema operativo y entorno en consecuencia.

Requisitos previos #

  1. Asegúrese de utilizar RedHat o un sistema operativo similar (basado en Fedora). gato /etc/redhat-release muestra la versión de su sistema operativo.

  2. Compruebe si está instalado java 1.6 o superior que java

  3. Instala Java si es necesario sudo yum lista java disponible* sudo yum instalar java-1.8.0-openjdk.x86_64

  4. Comprueba si está instalado correctamente what java muestra el inicio de Java java -version muestra la versión de Java

  5. Cree un usuario SOLR. ``sudo adduser solr''

  6. Agregue una contraseña para el usuario. sudo passwd solr

  7. Habilite sudo en el usuario SOLR, ejecute visudo sudo visudo

  8. Busque las siguientes líneas: ## Permite que las personas en la rueda de grupo ejecuten todos los comandos # %rueda TODO=(TODO) TODO

  9. Si %wheel está comentado, elimine el comentario de la segunda línea eliminando el carácter #. %rueda TODO=(TODO) TODO

  10. Si realizó un cambio, use :wq; de lo contrario, use :q para salir.

  11. Agregue el usuario solr al grupo de ruedas. sudo usermod -aG rueda solr

  12. Cambie al usuario solr y verifique si tiene privilegios de root: su solr - sudo whoami

Descargando SOLR #

  1. Encuentra tu réplica local en: http://www.apache.org/dyn/closer.lua/lucene/solr/

  2. Cambie la ubicación de su directorio para optar: cd /optar/

  3. Descargue una copia del paquete desde el espejo: sudo curl -O http://www.trieuvan.com/apache/lucene/solr/6.1.0/solr-6.1.0.tgz

  4. Desempaquete el paquete: sudo tar zxvf solr-6.1.0.tgz

  5. Copie el script del instalador en su carpeta: sudo cp /opt/solr-6.1.0/bin/install_solr_service.sh .

  6. Elimina los archivos innecesarios: sudo rm -rf solr-6.1.0

Instalar #

  1. Ejecute el script de instalación: sudo ./install_solr_service.sh solr-6.1.0.tgz
  2. Haga que el servicio SOLR se inicie automáticamente cuando se reinicie el servidor. sudo chkconfig --add solr chkconfig | grep solr
  3. Cambiar el propietario del servicio sudo chown -R solr:solr /var/solr/

Probando su instalación #

  1. Cree un núcleo desde la línea de comando: sudo su - solr -c "/opt/solr/bin/solr create -c NewCore1 -n data_driven_schema_configs"

  2. Abra Admin en un navegador: http://<servidor_solr>:8983/solr/#/

  3. Abra la lista de núcleos en el menú para ver el núcleo NewCore1.

  4. Pruebe si los núcleos están pegajosos: reinicio del servicio sudo solr

  5. Actualice el administrador en un navegador: http://<servidor_solr>:8983/solr/#/ Asegúrese de que la página de administración se vuelva a cargar y que el núcleo vuelva a aparecer después del reinicio.

  6. Ver el estado del servidor en la línea de comandos: Estado de solr del servicio sudo

  7. Prepare una nueva configuración de núcleo para la creación de núcleo en Web Admin, cambiando la ubicación de su directorio a datos: cd var/solr/datos/

  8. Aquí es donde se almacenan los nuevos núcleos: ll

  9. La carpeta conf del núcleo recién creada se puede usar como plantilla: ll Nuevo Núcleo1/

  10. Cree una carpeta para otro núcleo que creará en Web Admin: mkdir CoreFromWebAdmin

  11. Copie el directorio conf en la nueva ubicación: sudo cp -R NewCore1/conf/CoreFromWebAdmin

  12. Cambie a la interfaz de administración web en su navegador

  13. Haga clic en Agregar núcleo

  14. Agregue CoreFromWebAdmin como nombre y carpeta para el nuevo núcleo.

  15. Abra el nuevo núcleo.

  16. Haga clic en documentos para agregar documentos.

  17. Seleccione el formato XML y pegue el código a continuación:

Introducción #

Solr es un servidor de búsqueda empresarial independiente con una API similar a REST. Pones documentos en él (llamado “indexación”) a través de JSON, XML, CSV o binario a través de HTTP. Lo consulta a través de HTTP GET y recibe resultados JSON, XML, CSV o binarios. Solr utiliza la biblioteca de búsqueda de Lucene y la amplía.

Estas son algunas de las principales características que proporciona solr:

- Funciones avanzadas de búsqueda de texto completo

Con la tecnología de Lucene™, Solr habilita poderosas capacidades de coincidencia que incluyen frases, comodines, uniones, agrupaciones y mucho más en cualquier tipo de datos

  • Optimizado para tráfico de alto volumen

Solr está probado a escalas extremadamente grandes en todo el mundo

  • Interfaces abiertas basadas en estándares: XML, JSON y HTTP

Solr usa las herramientas que usted usa para hacer que la creación de aplicaciones sea muy sencilla

  • Interfaces de administración integrales

Solr se envía con una interfaz de usuario administrativa receptiva integrada para facilitar el control de sus instancias de Solr

  • Fácil Monitoreo

¿Necesita más información sobre sus instancias? Solr publica muchos datos métricos a través de JMX

  • Altamente escalable y tolerante a fallas

Basado en el Apache Zookeeper probado en batalla, Solr facilita la ampliación y reducción. Solr incluye replicación, distribución, reequilibrio y tolerancia a fallas desde el primer momento.

  • Flexible y Adaptable con fácil configuración

Solr’s está diseñado para adaptarse a sus necesidades al mismo tiempo que simplifica la configuración

  • Indización casi en tiempo real

¿Quieres ver tus actualizaciones ahora? Solr aprovecha las capacidades de indexación en tiempo casi real de Lucene para asegurarse de que vea su contenido cuando quiera verlo.

  • Arquitectura de complemento extensible

Solr publica muchos puntos de extensión bien definidos que facilitan la inserción de complementos tanto de índice como de tiempo de consulta. Por supuesto, dado que es un código abierto con licencia de Apache, ¡puede cambiar cualquier código que desee!

Algunas características geniales de solr:

  • Esquema cuando quieras, sin esquemas cuando no

Utilice el modo sin esquema basado en datos de Solr al comenzar y luego bloquéelo cuando llegue el momento de la producción.

  • Extensiones poderosas

Solr se envía con complementos opcionales para indexar contenido enriquecido (por ejemplo, PDF, Word), detección de idioma, agrupación de resultados de búsqueda y más

  • Búsqueda y filtrado por facetas

Corte y trocee sus datos como mejor le parezca utilizando una gran variedad de algoritmos de facetado

  • Búsqueda geoespacial

Habilitar la búsqueda basada en la ubicación es simple con el soporte incorporado de Solr para la búsqueda espacial

  • Sugerencias de consulta, ortografía y más

Solr se envía con capacidades avanzadas para autocompletar (búsqueda de escritura anticipada), revisión ortográfica y más

  • Análisis de documentos enriquecidos

Solr se envía con Apache Tika incorporado, lo que facilita la indexación de contenido enriquecido como Adobe PDF, Microsoft Word y más.
