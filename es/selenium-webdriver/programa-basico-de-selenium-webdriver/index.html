Este tema tiene como objetivo mostrar el programa básico del controlador web en lenguajes compatibles con Selenium como C#, Groovy, Java, Perl, PHP, Python y Ruby.

Journey incluye abrir el controlador del navegador –> Página de Google –> apagar el navegador

C# #

using OpenQA.Selenium;
using OpenQA.Selenium.Chrome;

namespace BasicWebdriver
{
    class WebDriverTest
    {
        static void Main()
        {
            using (var driver = new ChromeDriver())
            {
                driver.Navigate().GoToUrl("http://www.google.com");
            }
        }
    }
}

El ‘programa’ anterior navegará a la página de inicio de Google y luego cerrará el navegador después de cargar completamente la página.

using (var driver = new ChromeDriver())

Esto instancia un nuevo objeto WebDriver utilizando la interfaz IWebdriver y crea una nueva instancia de ventana del navegador. En este ejemplo estamos usando ChromeDriver (aunque esto podría ser reemplazado por el controlador apropiado para cualquier navegador que quisiéramos usar). Estamos envolviendo esto con una declaración using, porque IWebDriver implementa IDisposable, por lo que no es necesario escribir explícitamente driver.Quit();.

En caso de que no haya descargado su WebDriver usando [NuGet][1], deberá pasar un argumento en forma de ruta al directorio donde se encuentra el controlador “chromedriver.exe”.

navegando #

driver.Navigate().GoToUrl("http://www.google.com");

y

driver.Url = "http://www.google.com";

Ambas líneas hacen lo mismo. Le indican al controlador que navegue a una URL específica y que espere hasta que se cargue la página antes de pasar a la siguiente instrucción.

Hay otros métodos vinculados a la navegación como Atrás(), Adelante() o Refresh().

Después de eso, el bloque using se cierra de forma segura y desecha el objeto.

[1]: https://www.wikiod.com/es/selenium-webdriver/primeros-pasos-con-selenium-webdriver#Instalación o Configuración

##Java

El siguiente código se trata de 3 pasos.

  1. Abrir un navegador Chrome
  2. Abriendo la página de google
  3. Cierra el navegador
import org.openqa.selenium;
import org.openqa.selenium.chrome;

public class WebDriverTest {
    public static void main(String args[]) {
        System.setProperty("webdriver.chrome.driver", "C:\\path\\to\\chromedriver.exe");
        WebDriver driver = new ChromeDriver();

        driver.get("http://www.google.com");
        driver.quit();
    }
}

El ‘programa’ anterior navegará a la página de inicio de Google y luego cerrará el navegador antes de completar.

System.setProperty("webdriver.chrome.driver", "C:\\path\\to\\chromedriver.exe");
WebDriver driver = new ChromeDriver();

La primera línea le dice al sistema dónde encontrar el ejecutable ChromeDriver (chromedriver.exe). Luego creamos nuestro objeto de controlador llamando al constructor ChromeDriver(), nuevamente podríamos estar llamando a nuestro constructor aquí para cualquier navegador/plataforma.

driver.get("http://www.google.com");

Esto le dice a nuestro controlador que navegue a la URL especificada: http://www.google.com. La API de Java WebDriver proporciona el método get() directamente en la interfaz de WebDriver, aunque se pueden encontrar más métodos de navegación a través del método navigate(), p. controlador.navegar.atrás().

Una vez que la página ha terminado de cargarse llamamos inmediatamente:

driver.quit();

Esto le dice al controlador que cierre todas las ventanas abiertas del navegador y elimine el objeto del controlador, ya que no tenemos otro código después de esta llamada, esto termina efectivamente el programa.

driver.close();

Es una instrucción (que no se muestra aquí) para que el controlador cierre solo la ventana activa; en este caso, como solo tenemos una ventana, las instrucciones generarían resultados idénticos a los de llamar a quit().

pitón #

from selenium import webdriver
from selenium.webdriver.common.keys import Keys

def set_up_driver():
    path_to_chrome_driver = 'chromedriver'
    return webdriver.Chrome(executable_path=path_to_chrome_driver)

def get_google():
    driver = set_up_driver()
    driver.get('http://www.google.com')
    tear_down(driver)

def tear_down(driver):
    driver.quit()

if '__main__' == __name__:
    get_google()

El ‘programa’ anterior navegará a la página de inicio de Google y luego cerrará el navegador antes de completar.

if '__main__' == __name__:
    get_google()

Primero tenemos nuestra función principal, nuestro punto de entrada al programa, que llama get_google().

def get_google():
    driver = set_up_driver()

get_google() luego comienza creando nuestra instancia driver a través de set_up_driver():

def set_up_driver():
    path_to_chrome_driver = 'chromedriver'
    return webdriver.Chrome(executable_path=path_to_chrome_driver)

Por lo que indicamos dónde se encuentra chromedriver.exe e instanciamos nuestro objeto de controlador con esta ruta. El resto de get_google() navega a Google:

driver.get('http://www.google.com')

Y luego llama a tear_down() pasando el objeto del controlador:

tear_down(driver)

tear_down() simplemente contiene una línea para apagar nuestro objeto controlador:

driver.quit()

Esto le dice al controlador que cierre todas las ventanas abiertas del navegador y elimine el objeto del navegador, ya que no tenemos otro código después de esta llamada, esto termina efectivamente el programa.

Clases de página: página de inicio de sesión y página de inicio Clase de página de inicio de sesión:

import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.support.FindBy;

public class LoginPage {

    WebDriver driver;
    public LoginPage(WebDriver driver){
        this.driver = driver;
    }
    @FindBy(id="email")
    private WebElement loginTextBox;

    @FindBy(id="pass")
    private WebElement passwordTextBox;

    @FindBy(xpath = ".//input[@data-testid='royal_login_button']")
    private WebElement loginBtn;

    public void enterUserName(String userName){
        if(loginTextBox.isDisplayed()) {
            loginTextBox.clear();
            loginTextBox.sendKeys(userName);
        }
        else{
            System.out.println("Element is not loaded");
        }
    }
    public void enterPassword(String password){
        if(passwordTextBox.isDisplayed()) {
            passwordTextBox.clear();
            passwordTextBox.sendKeys(password);
        }
        else{
            System.out.println("Element is not loaded");
        }
    }
    public HomePage clickLogin(){
        if(loginBtn.isDisplayed()) {
            loginBtn.click();
        }
        return new HomePage(driver);
    }
}

Clase de página de inicio:

import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.support.FindBy;

public class HomePage {

    WebDriver driver;
    public HomePage(WebDriver driver){
        this.driver = driver;
    }

    @FindBy(xpath=".//a[@data-testid='blue_bar_profile_link']/span")
    private WebElement userName;

    public String getUserName(){
        if(userName.isDisplayed()) {
            return userName.getText();
        }
        else {
            return "Username is not present";
        }
    }

}
