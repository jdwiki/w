Parámetros #

parámetro detalles
nombreOrId Seleccione un marco por su nombre de id.
índice Seleccione un cuadro por su índice de base cero.
frameElement Seleccione un marco usando su WebElement previamente ubicado

Para cambiar a un marco usando Java #

Por ejemplo, si el código fuente html de una vista o elemento html está envuelto por un iframe como este:

<iframe src="../images/eightball.gif" name="imgboxName" id="imgboxId">
   <p>iframes example</p>
   <a href="../images/redball.gif" target="imgbox">Red Ball</a>
</iframe><br />

… luego, para realizar cualquier acción en los elementos web del iframe, primero debe cambiar el enfoque al iframe, utilizando cualquiera de los métodos a continuación:

Usando el Id. de marco (debe usarse solo si conoce el Id. del iframe).

driver.switchTo().frame("imgboxId"); //imgboxId - Id of the frame

Usando el nombre del marco (debe usarse solo si conoce el nombre del iframe).

driver.switchTo().frame("imgboxName"); //imgboxName - Name of the frame

Usando el índice de marco (debe usarse solo si no tiene la identificación o el nombre del iframe), donde el índice define la posición del iframe entre todos los marcos.

driver.switchTo().frame(0); //0 - Index of the frame

Nota: si tiene tres marcos en la página, el primer marco estará en el índice 0, el segundo en el índice 1 y el tercero en el índice 2.

Utilizando un elemento web previamente ubicado (debe usarse solo si ya ha ubicado el marco y lo ha devuelto como un WebElement).

driver.switchTo().frame(frameElement); //frameElement - webelement that is the frame

Entonces, para hacer clic en el ancla Red Ball:

driver.switchTo().frame("imgboxId");
driver.findElement(By.linkText("Red Ball")).Click();

Cambia a un marco usando C# #

1. Cambiar a un cuadro por Índice.

Aquí estamos cambiando al índice 1. El índice se refiere al orden de los marcos en la página. Esto debe usarse como último recurso, ya que la identificación o los nombres de los marcos son mucho más confiables.

driver.SwitchTo().Frame(1);

2. Cambiar a un cuadro por Nombre

driver.SwitchTo().Frame("Name_Of_Frame");

3. Cambie a un marco por Título, Id u otros pasando IWebElement

Si desea cambiar a un marco por id o título, debe pasar un elemento web como parámetro:

driver.SwitchTo().Frame(driver.FindElement(By.Id("ID_OF_FRAME")));
driver.SwitchTo().Frame(driver.FindElement(By.CssSelector("iframe[title='Title_of_Frame']")));

También tenga en cuenta que si su marco tarda unos segundos en aparecer, es posible que deba usar [esperar] 1:

new WebDriverWait(driver, TimeSpan.FromSeconds(10))
    .Until(ExpectedConditions.ElementIsVisible(By.Id("Id_Of_Frame")));

Salir de un marco:

driver.SwitchTo().DefaultContent()

Para salir de un marco usando Java #

Para cambiar el enfoque al documento principal o al primer marco de la página. Tienes que usar la siguiente sintaxis.

driver.switchTo().defaultContent();

Espera a que carguen tus marcos #

En bastantes casos, es posible que su marco no aparezca de inmediato y probablemente tenga que esperar hasta que se cargue para cambiar. De lo contrario, tendrá NoSuchFrameException.

Por lo tanto, siempre es una buena opción esperar antes de cambiar. La siguiente es una forma ideal de esperar hasta que se cargue un marco.

try{
        new WebDriverWait(driver, 300).ignoring(StaleElementReferenceException.class).
                        ignoring(WebDriverException.class).
                        until(ExpectedConditions.visibilityOf((driver.findElement(By.id("cpmInteractionDivFrame"))));}
    catch{

// arroja una excepción solo si su marco no está visible en su tiempo de espera de 300 segundos }

Para salir de un marco usando C# #

Para cambiar el enfoque al documento principal o al primer marco de la página. Tienes que usar la siguiente sintaxis.

webDriver.SwitchTo().DefaultContent();

Cambia entre marcos secundarios de un marco principal. #

Considere que tiene un Marco principal (Frame-Parent). y 2 marcos secundarios (Frame_Son, Frame_Daughter). Veamos varias condiciones y cómo manejar.

1.De padre a hijo o hija:

 driver.switchTo().frame("Frame_Son");
 driver.switchTo().frame("Frame_Daughter");

2.De hijo a padre: si el padre es el marco predeterminado, cambie al marco predeterminado; de lo contrario, cambie del marco predeterminado al marco principal. Pero no puede cambiar directamente de hijo a padre.

driver.switchTo().defaultContent(); 
driver.switchTo().frame("Frame_Parent");

  1. De hijo a hija: si su hermana comete algún error, no le grite, solo comuníquese con su padre. De manera similar, le da control al marco principal y luego al marco secundario.

    driver.switchTo().defaultContent(); driver.switchTo().frame(“Frame_Parent”); driver.switchTo().frame(“Frame_Daughter”);
