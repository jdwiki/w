  1. Lista de las principales características del IDE.
  2. Lista de los sistemas operativos soportados por el IDE.
  3. Proceso de instalación.
  4. Pasos de configuración adicionales (si los hay).

SW4STM32: Banco de trabajo del sistema para STM32 #

Introducción #

System Workbench para STM32 es un IDE gratuito en Windows, Linux y OS X. Descripción de ST Microelectronics:

La cadena de herramientas de System Workbench, llamada SW4STM32, es un multi-OS gratuito entorno de desarrollo de software basado en Eclipse, que soporta la gama completa de microcontroladores STM32 y placas asociadas.

La cadena de herramientas SW4STM32 se puede obtener del sitio web www.openstm32.org, que incluye foros, blogs y capacitaciones para soporte técnico. Una vez registrados en este sitio, los usuarios obtendrán instrucciones de instalación en la página Documentación > System Workbench para continuar con la descarga de la cadena de herramientas gratuita.

La cadena de herramientas System Workbench y su sitio web colaborativo han sido construido por AC6, una empresa de servicios que brinda capacitación y consultoría en sistemas embebidos.

Este producto es suministrado por un tercero no afiliado a ST. Para la información más reciente sobre la especificación, consulte la tercera sitio web del partido: www.ac6.fr.

Características clave

  • Soporte integral para microcontroladores STM32, placas Nucleo STM32, kits Discovery y placas de evaluación, así como STM32 firmware (biblioteca de periféricos estándar o STM32Cube HAL)
  • compilador GCC C/C++
  • Depurador basado en GDB
  • Eclipse IDE con gestión de trabajo en equipo
  • Compatible con complementos de Eclipse
  • Soporte ST-LINK
  • Sin límite de tamaño de código
  • Compatibilidad con varios sistemas operativos: Windows®, Linux y OS X®

Instalación #

  1. Vaya a: http://www.openstm32.org/HomePage.
  2. Regístrese e inicie sesión en el sitio.
  3. Vaya a: http://www.openstm32.org/Downloading+the+System+Workbench+for+STM32+installer.
  4. Descargue la última versión para su sistema operativo.
  5. Ejecute el instalador descargado.

IAR-EWARM #

Introducción #

IAR Entorno de desarrollo integrado y optimización del compilador C/C++ para ARM Cortex-M. Descripción de ST Microelectronics:

El IAR-EWARM es un paquete de desarrollo de software entregado con archivos de configuración de dispositivos listos para usar, cargadores flash y ejemplo 4300 proyectos incluidos. IAR Embedded Workbench es compatible con otros Compiladores compatibles con ARM®EABI y compatibles con los siguientes núcleos ARM®para STM32:

Características clave

  1. Componentes clave:
  • Entorno de desarrollo integrado con herramientas de gestión de proyectos y editor
  • Compilador C y C++ altamente optimizado para ARM®
  • Comprobación automática de las reglas MISRA C (MISRA C:2004)
  • Conformidad con ARM® EABI y CMSIS
  • Amplio soporte de sistema de destino HW
  • Sondas de depuración en circuito I-jet y JTAGjet™-Trace opcionales
  • Depuración de energía para visualizar el consumo de energía en correlación con el código fuente
  • Bibliotecas de tiempo de ejecución, incluido el código fuente
  • Reubicación del ensamblador ARM®
  • Herramientas de enlazador y bibliotecario
  • Depurador C-SPY® con simulador ARM®, soporte JTAG y soporte para depuración compatible con RTOS en hardware
  • Complementos RTOS disponibles de proveedores de IAR Systems y RTOS
  • Más de 3100 proyectos de muestra para placas de evaluación de muchos fabricantes diferentes
  • Guías de usuario y de referencia en formato PDF
  • Ayuda en línea sensible al contexto
  1. Soporte específico para chips:
  • 4300 proyectos de ejemplo que incluyen placas de evaluación de STMicroelectronics
  • Soporte para aplicaciones de 4 Gbyte en modo ARM® y Thumb®
  • Cada función se puede compilar en modo ARM® o Thumb®
  • Generación de código de coprocesador de punto flotante vectorial VFP
  • Compatibilidad con NEON™ intrínseco
  1. Soporte de depuración de hardware:
  • STMicroelectronics ST-LINK V2: admite dispositivos STM32
  • STMicroelectronics ST-LINK: admite dispositivos STM32
  1. Soporte RTOS: consulte el sitio web de IAR http://www.iar.com
  2. Dispositivos compatibles: consulte el sitio web de IAR http://www.iar.com

Atollic - TrueSTUDIO #

Introducción #

C/C++ IDE para desarrollo ARM.

Atollic TrueSTUDIO® está probado y verificado en los siguientes sistemas operativos:

  • Microsoft® Windows®Vista (versión de 32 bits)
  • Microsoft® Windows® Vista (versión de 64 bits)
  • Microsoft® Windows® 7 (versión de 32 bits)
  • Microsoft® Windows® 7 (versión de 64 bits)
  • Microsoft® Windows® 8 (versión de 64 bits)
  • Microsoft® Windows® 10 (versión de 64 bits)
  • El soporte de Linux se espera para fines del cuarto trimestre de 2016
  • Compatibilidad con Mac OS X prevista para el segundo trimestre de 2017

TrueSTUDIO solo está disponible como una aplicación de 32 bits.

Instalación #

El producto Atollic TrueSTUDIO se entrega como un ejecutable instalador. Asegúrese de que la cuenta de usuario desde la que se se inicia el instalador, tiene privilegios administrativos. No hay necesidad para registro o conexión a internet durante la instalación. Cuando TrueSTUDIO está instalado, se ejecutará en modo Lite si no hay licencias detectado.

  1. Vaya a: http://atollic.com/resources/downloads/ .
  2. Descargue la última versión estable o la última versión beta.
  3. Ejecute el instalador.

Código #

Introducción #

CooCox CoIDE, un entorno de desarrollo de software gratuito y altamente integrado para MCU ARM Cortex. Descripción de ST Microelectronics:

CoIDE es un entorno de desarrollo de software libre basado en Eclipse y Cadena de herramientas de GCC, que se ha personalizado y simplificado para ofrecer a los usuarios un fácil acceso a los microcontroladores ARM® Cortex®-M.

Este producto es suministrado por un tercero no afiliado a ST. Para información completa y más reciente sobre las especificaciones y los paquetes de las piezas compradas, consulte el sitio web del tercero www.coocox.org.

Características clave

  • Compatibilidad completa con microcontroladores STM32, placas STM32 Nucleo y bibliotecas de software STM32Cube.
  • compilador GCC C/C++.
  • Depurador basado en GDB.
  • IDE simplificado de Eclipse.
  • Compatibilidad con ST-Link.
  • Soporte multilingüe: inglés, chino.
