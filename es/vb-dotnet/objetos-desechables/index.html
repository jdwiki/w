Concepto básico de IDisposable #

Cada vez que crea una instancia de una clase que implementa IDisposable, debe llamar a .Dispose1 en esa clase cuando haya terminado de usarla. Esto permite que la clase limpie cualquier dependencia administrada o no administrada que pueda estar usando. No hacer esto podría causar una pérdida de memoria.

La palabra clave Using asegura que se llame a .Dispose, sin que tengas que llamarlo explícitamente.

Por ejemplo sin Usar:

Dim sr As New StreamReader("C:\foo.txt")
Dim line = sr.ReadLine
sr.Dispose()

Ahora con Usando:

Using sr As New StreamReader("C:\foo.txt")
    Dim line = sr.ReadLine
End Using '.Dispose is called here for you

Una de las principales ventajas que tiene Using es cuando se lanza una excepción, porque garantiza que se llame a .Dispose.

Considera lo siguiente. Si se lanza una excepción, debe recordar llamar a .Dispose, pero es posible que también deba verificar el estado del objeto para asegurarse de que no obtenga un error de referencia nula, etc.

    Dim sr As StreamReader = Nothing
    Try
        sr = New StreamReader("C:\foo.txt")
        Dim line = sr.ReadLine
    Catch ex As Exception
        'Handle the Exception
    Finally
        If sr IsNot Nothing Then sr.Dispose()
    End Try

Un bloque de uso significa que no tiene que recordar hacer esto y puede declarar su objeto dentro de try:

    Try
        Using sr As New StreamReader("C:\foo.txt")
            Dim line = sr.ReadLine
        End Using
    Catch ex As Exception
        'sr is disposed at this point
    End Try

Declarar más objetos en uno Usando #

A veces, tienes que crear dos objetos Desechables seguidos. Hay una manera fácil de evitar el anidamiento de bloques ‘Usando’.

este codigo

Using File As New FileStream("MyFile", FileMode.Append)
    Using Writer As New BinaryWriter(File)
        'You code here
        Writer.Writer("Hello")
    End Using
End Using

se puede acortar en este. La principal ventaja es que ganas un nivel de sangría:

Using File As New FileStream("MyFile", FileMode.Append), Writer As New BinaryWriter(File)
    'You code here
    Writer.Writer("Hello")
End Using
