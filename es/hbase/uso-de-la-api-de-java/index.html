Sintaxis #

  • HBaseConfiguration.create(); //Crear un archivo de configuración

  • Configuration.set (clave de cadena, valor de cadena); //Añadir una clave a la configuración

  • ConnectionFactory.createConnection(configuración HBaseConfiguration); //Se conecta a HBase

  • Conexión.getAdmin(); // Instanciar un nuevo administrador

  • nuevo HTableDescriptor(Table.valueOf(String tableName));; //Crear un descriptor de tabla

  • HTableDescriptor.addFamily(nuevo HColumnDescriptor(String familyName)); //Añadir una familia al descriptor de la tabla

  • Admin.createTable(descriptor HTableDescriptor); //Crear una tabla como se describe en el descriptor

  • Admin.deleteTable(TableName.valueOf(String tableName)); //Eliminar una tabla

  • Connection.getTable(TableName.valueOf(String tableName)); //Obtener un objeto de tabla

  • nuevo Get(Bytes.toBytes(String row_key)); //Crear un nuevo Obtener

  • table.get(Get get) //Devuelve un resultado

  • nuevo Put(String fila_clave); //Crear una nueva Venta

  • table.put(Poner poner); //Inserte la(s) fila(s)

  • nuevo scan(); //Crear nuevo escaneo

  • table.getScanner (escaneo de exploración); //Devolver un ResultScanner

  • nueva eliminación (Bytes.toBytes (String row_key)); //Crear una nueva Eliminación

  • table.delete(Borrar borrar); //Eliminar una fila de la tabla

Parámetros #

Parámetro Valores posibles
ComparaOp CompareOp.EQUAL , CompareOp.GREATER , CompareOp.GREATER_OR_EQUAL , CompareOp.LESS , CompareOp.LESS_OR_EQUAL , CompareOp.NOT_EQUAL , CompareOp.NO_OP (sin operación)

Este tema muestra varios ejemplos de cómo usar la API de Java para HBase. En este tema, aprenderá a crear y eliminar una tabla, insertar, consultar y eliminar filas de una tabla, pero también usará los filtros de Escaneo.

Notará que muchos métodos de esta API toman Bytes como parámetros, por ejemplo, el nombre de columnFamily, esto se debe a la implementación de HBase. Para fines de optimización, en lugar de almacenar los valores como Cadena, Entero o lo que sea, almacena una lista de Bytes, es por eso que necesita analizar todos esos valores como Bytes. Para ello, el método más sencillo es utilizar Bytes.toBytes(algo).

Por favor, siéntase libre de notarlo si ve algún error o malentendido.

Consultar HBase, Obtener, Poner, Eliminar y Escanear #

En HBase, puede usar 4 tipos de operaciones

  • Obtener: recupera una fila
  • Put : inserta una o más filas
  • Eliminar: eliminar una fila
  • Escanear: recupera varias filas

Si simplemente desea recuperar una fila, dada su row_key, puede usar el objeto Get:

Get get = new Get(Bytes.toBytes("my_row_key"));
Table table = this.connection.getTable(TableName.valueOf("myTable"));
Result r = table.get(get);
byte[] value = r.getValue(Bytes.toBytes(columnFamily), Bytes.toBytes("myColumn"));
String valueStr = Bytes.toString(value);
System.out.println("Get result :" + valueStr);

Aquí solo obtenemos el valor de la columna que queremos, si desea recuperar toda la columna, use el atributo rawCell del objeto Get:

Get get = new Get(Bytes.toBytes(rowKey));
Table table = this.connection.getTable(TableName.valueOf(tableName));
Result r = table.get(get);
System.out.println("GET result :");
    for (Cell c : r.rawCells()) {
        System.out.println("Family : " + new String(CellUtil.cloneFamily(c)));
        System.out.println("Column Qualifier : " + new String(CellUtil.cloneQualifier(c)));
        System.out.println("Value : " + new String(CellUtil.cloneValue(c)));
        System.out.println("----------");
    }

Bueno, ahora podemos recuperar datos de nuestra tabla, fila por fila, pero ¿cómo ponemos algunos? Usas el objeto Put:

Put put = new Put("my_row_key");
put.addColumn(Bytes.toBytes("myFamily"), Bytes.toBytes("myColumn"), Bytes.toBytes("awesomeValue");
//Add as many columns as you want

Table table = connection.getTable(TableName.valueOf("myTable");
table.put(put);

NB: Table.put también puede tomar como parámetro una lista de puts, lo que significa que, cuando desea agregar muchas filas, es mucho más eficiente que put by put.

Bien, ahora puedo poner algunas filas y recuperar algunas de mi HBase, pero ¿qué pasa si quiero obtener varias filas y no conozco mis row_keys?

¡Capitán aquí! Puede utilizar el objeto Scan:

Un escaneo básicamente busca todas las filas y las recupera, puede agregarle varios parámetros, como filtros y fila de inicio/fin, pero lo veremos en [otro ejemplo] (https://www.wikiod.com/es/hbase/uso-de-la-api-de-java /usando-la-api-java/15541/usando-los-filtros-de-escaneo#t=201607281342145028956).

Si desea escanear todos los valores de columna de su tabla, dada una columna, use las siguientes líneas:

Table table = this.connection.getTable(TableName.valueOf("myTable"));
Scan scan = new Scan();
scan.addColumn(Bytes.toBytes("myFamily"), Bytes.toBytes("myColumn"));
ResultScanner rs = table.getScanner(scan);
    try {
        for (Result r = rs.next(); r != null; r = rs.next()) {
            byte[] value = r.getValue(Bytes.toBytes("myFamily"), Bytes.toBytes("myCOlumn"));
            String valueStr = Bytes.toString(value);
            System.out.println("row key "+new String(r.getRow()));
            System.out.println("Scan result :" + valueStr);
        }
    } finally {
        rs.close(); // always close the ResultScanner!
   }

Realmente quiero insistir en el hecho de que siempre debes cerrar el ResultScanner (por cierto, lo mismo que cualquier ResultSet de una base de datos)

Casi terminado ! Ahora aprendamos cómo eliminar una fila. Tienes un objeto Delete para esto:

Table table = this.connection.getTable(TableName.valueOf("myTable"));
Delete d = new Delete(Bytes.toBytes("my_weird_key"));
table.delete(d);
System.out.prinln("Row " + row_key + " from table " + tableName + " deleted");

Una última cosa: antes de ejecutar cualquiera de las operaciones, compruebe siempre que la tabla existe, o obtendrá una excepción.

Eso es todo por ahora, puede administrar sus datos en HBase con este ejemplo.

Crear y eliminar tablas #

En HBase, los datos se almacenan en tablas con columnas. Las columnas se reagrupan en familias de columnas, que pueden ser, por ejemplo, “personales” o “profesionales”, cada una de las cuales contiene información específica.

Para crear una tabla, debe usar el objeto Admin, créelo usando:

Admin admin = connection.getAdmin();

Una vez que tenga este administrador, puede comenzar a crear tablas. En primer lugar, asegúrese de que esta tabla no exista ya con la línea

admin.tableExists(TableName.valueOf("myTable);

Este método devolverá verdadero si la tabla existe. Cuando haya marcado esto, puede crear su tabla usando las líneas

HTableDescriptor descriptor = new HTableDescriptor(TableName.valueOf("myTable"));
descriptor.addFamily(new HColumnDescriptor("myFamily"));
admin.createTable(descriptor);

Debe configurar al menos una familia para la tabla, y el libro de referencia de HBase recomienda no obtener más de 3 familias de columnas, de lo contrario perderá rendimiento.

Felicidades ! ¡Tu mesa ha sido creada!

Si necesita eliminar su tabla, puede usar

this.admin.disableTable(TableName.valueOf(tableName));
this.admin.deleteTable(TableName.valueOf(tableName));

¡Asegúrese de deshabilitar siempre la tabla primero!

Ahora sabe cómo administrar tablas en HBase.

Uso de los filtros de escaneo #

Básicamente, el objeto Scan recupera todas las filas de la tabla, pero ¿qué sucede si desea recuperar solo las filas donde el valor de una columna dada es igual a algo? Permítanme presentarles los Filtros, funcionan como WHERE en SQL.

Antes de comenzar a usar los filtros, si sabe cómo se almacenan sus filas_claves, puede establecer una fila inicial y una final para su Escaneo, lo que optimizará su consulta.

En HBase, row_keys se almacenan en el orden lexicográfico, pero aún puede usar salting para cambiar la forma en que se almacena, no explicaré salting en este tema, tomaría demasiado tiempo y ese no es el punto.

Volvamos a nuestros límites de fila, tiene dos métodos para usar para establecer la fila inicial y final

Scan scan = new Scan();
scan.setStartRow(Bytes.toBytes("row_10"));
scan.setStopRow(Bytes.toBytes("row_42"));

Esto cambiará el comportamiento de su escáner para obtener todas las filas entre “fila_10” y “fila_42”.

NB: como en la mayoría de los métodos “sub” (por ejemplo, substring), startRow es inclusivo y stopRow es exclusivo.

Ahora que podemos vincular nuestro escaneo, ahora debemos agregar algunos filtros a nuestros escaneos, hay muchos, pero veremos aquí los más importantes.

  • Si desea recuperar todas las filas que tienen una clave de fila que comienza con un patrón determinado

Utilice RowPrefixFilter:

Scan scan = new Scan();
scan.setRowPrefixFilter(Bytes.toBytes("hello"));

Con este código, su escaneo solo recuperará las filas que tengan una clave de fila que comience con “hola”.

  • Si desea recuperar todas las filas donde el valor de una columna dada es igual a algo

Utilice SingleColumnValueFilter:

Scan scan = new Scan();
SingleColumnValueFilter filter = new SingleColumnValueFilter(Bytes.toBytes("myFamily"),Bytes.toBytes("myColumn"), CompareOp.EQUAL, Bytes.toBytes("42"));
scan.setFilter(filter);

Con este código, obtendrá todas las filas donde el valor de la columna myColumn es igual a 42. Tiene diferentes valores para CompareOp que se explican en la sección Parámetros.

-Bien, pero que pasa si quiero usar expresiones regulares

Use el filtro RegexStringComparator:

Scan scan = new Scan();
RegexStringComparator comparator = new RegexStringComparator(".hello.");
SingleColumnValueFilter filter = new SingleColumnValueFilter(Bytes.toBytes("myFamily"),Bytes.toBytes("myColumn"), CompareOp.EQUAL, comparator);
scan.setFilter(filter);

Y obtendrá todas las filas donde la columna myColumn contiene hola.

Tenga en cuenta también que el método Scan.setFilter() también puede tomar una lista de Filter como parámetros

Conexión a HBase #

Si desea conectarse a un servidor HBase, primero debe asegurarse de que la IP del servidor esté en su archivo /etc/hosts, por ejemplo, agregue la línea

255.255.255.255    hbase

Luego puede usar la API de Java para conectarse a zookeeper, solo tiene que especificar el puerto del cliente y la dirección de zookeeper

Configuration config =  HBaseConfiguration.create();
config.set("hbase.zookeeper.quorum", "hbase");
config.set("hbase.zookeeper.property.clientPort","2181");

Después de configurar la conexión, puede probarla usando

HBaseAdmin.checkHBaseAvailable(config);

Si tiene un problema con la configuración de HBase, se generará una excepción.

Finalmente para conectarse al servidor, solo use

Connection connection = ConnectionFactory.createConnection(config);
