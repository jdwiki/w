Creando un nuevo proyecto (C# / XAML) #

Después de iniciar Visual Studio 2015, vaya a Archivo → Nuevo → Proyecto. En el cuadro de diálogo Nuevo proyecto, navegue en el árbol de plantillas hasta Visual C# → Windows → Universal y seleccione Aplicación en blanco (Universal Windows).

A continuación, debemos completar el formulario para describir la aplicación:

  1. Nombre: este es el nombre de la aplicación que se mostrará al usuario. Establézcalo en HelloWorld o use un título personalizado.
  2. Ubicación: indica dónde se almacenará el proyecto
  3. Nombre de la solución: este es un tipo de contenedor de proyectos que agrupa varios proyectos relacionados con la misma aplicación (por ejemplo, una solución podría estar compuesta por un proyecto de interfaz de usuario y un proyecto modelo). Puedes poner el mismo Nombre que tu proyecto inicial.

Contenido del proyecto por defecto #

Obtendrá un proyecto con los siguientes archivos:

  1. Package.appxmanifest: describe las propiedades de su aplicación. Contiene algunas configuraciones de interfaz de usuario, como su nombre de pantalla, su logotipo, las rotaciones admitidas. Y también contiene configuraciones técnicas como el punto de entrada de la aplicación (que es la clase App por defecto). Finalmente, también enumera las autorizaciones que requiere su aplicación en la pestaña Capacidades; por ejemplo, si desea utilizar la cámara web en su aplicación, deberá verificar las capacidades correspondientes.
  2. App.xaml / App.xaml.cs: la clase App es el punto de entrada predeterminado de su aplicación. Los archivos xaml pueden contener recursos compartidos en toda la aplicación, como la configuración de estilos o la instancia de una clase que desea compartir, como un localizador de ViewModel. Los archivos de código subyacente contienen todo el código de inicio de la aplicación. Por defecto, implementa el método OnLaunched que es invocado por el usuario final. Inicializa la ventana y navega a la primera página de la aplicación (por defecto, la clase MainPage).
  3. MainPage.xaml / MainPage.xaml.cs: esta es la página inicial de nuestra aplicación. Contiene solo una cuadrícula vacía que es un control de diseño.

Modificar la vista #

Abra MainPage.xaml y reemplace el control Grid con

<Grid Background="{ThemeResource ApplicationPageBackgroundThemeBrush}">
        <Grid.ColumnDefinitions>
            <ColumnDefinition Width="auto" />
            <ColumnDefinition Width="*" />
        </Grid.ColumnDefinitions>
        <Button Click="Button_Click">Say Hello !</Button>
        <TextBlock Grid.Column="1"
                   VerticalAlignment="Center"
                   x:Name="myText"
                   Text="Click the button." />
</Grid>

Esto creará una cuadrícula con dos columnas. La primera columna como un ancho establecido en auto, lo que significa que se establecerá automáticamente en función del tamaño de sus hijos. La segunda columna se estirará para llenar el espacio restante en la ventana.

Esta grilla contiene dos elementos:

  • un Botón que se encuentra en la primera columna. El evento de clic está vinculado al método Button_Click en el código subyacente y su título Texto es "¡Di hola!".
  • un TextBlock que se encuentra en la segunda columna. Su texto está establecido en “Haga clic en el botón”.. Y le hemos dado un nombre a este control con la ayuda del atributo x:Name. Esto es necesario para poder utilizar el control en el código subyacente.

En MainPage.xaml.cs, agregue el siguiente código:

private void Button_Click(object sender, RoutedEventArgs e)
{
    this.myText.Text = "Hello World!";
}

Este es el método que se llamará cuando el usuario haga clic (o toque) el botón. Y actualizará TextBlock y establecerá su texto en "¡Hola mundo!".

Ejecutando la aplicación #

Para ejecutar la aplicación, puede utilizar el menú Depurar → Iniciar depuración o el acceso directo F5. De forma predeterminada, ejecutará la aplicación en su máquina local.

Instalación o configuración #

Si no tiene Visual Studio 2015, vaya a https://www.visualstudio.com/en-us/downloads/download-visual-studio-vs.aspx y seleccione Visual Studio Community. Es una versión gratuita que tiene todas las capacidades necesarias para desarrollar aplicaciones de Windows 10. Puede instalar Enterprise Edition si tiene la licencia requerida. Luego inicie el instalador.

Si ya tiene instalado Visual Studio 2015, en el Panel de control de Windows, vaya a Programa y características y en la lista haga clic con el botón derecho en Microsoft Visual Studi0 2015 (este nombre puede cambiar en función de la versión instalada de Visual Studio). Studio 2015) y seleccione Cambiar.

En el cuadro de diálogo Instalación de Visual Studio, haga clic en “Modificar” y, en Desarrollo web y de Windows, instale todas las Herramientas universales de desarrollo de aplicaciones de Windows. Instalará el SDK de Windows 10, un emulador móvil de Windows 10 (con tecnología Hyper-V) y el Visual Studio necesario para crear aplicaciones de Windows 10.

Cuando finalice la instalación, puede crear un nuevo proyecto de Windows 10.

Requisitos del sistema: ventanas 10

Visual Studio 2015 (función Herramientas universales de desarrollo de aplicaciones de Windows)

Instrucciones para habilitar la función Herramientas universales de desarrollo de aplicaciones de Windows durante la instalación durante Visual Studio 2015.

i.Durante la instalación, es necesario seleccionar personalizar y verificar las herramientas universales de desarrollo de aplicaciones de Windows.

ii.Si ya tiene Visual Studio 2015, ahora puede agregar estas herramientas a su instalación de Visual Studio existente. Puede ejecutar el instalador o abrir Programas y características desde el Panel de control, seleccionar Visual Studio y hacer clic en Cambiar. Luego, en la configuración, haga clic en Modificar y seleccione Herramientas para aplicaciones universales de Windows.

iii.Después de instalar las herramientas, se puede crear un proyecto utilizando nuevas plantillas de proyecto (Windows->Universal) en Visual Studio 2015 en el lenguaje de su elección: C#, VB, C++ o JavaScript.

Para comenzar con el desarrollo de aplicaciones universales de Windows 10, debe tener Visual Studio 2015 en una computadora con Windows 10. Puedes comprar Windows 10 desde aquí. Puede descargar Visual Studio 2015 desde aquí. Puede usar las versiones Community, Professional o Enterprise de Visual Studio. Algunas funciones estarán limitadas en la versión Express.

Puede encontrar una guía detallada sobre cómo instalar Windows 10 aquí.

Al instalar Visual Studio, elija Personalizado y, en ‘Desarrollo web y de Windows’, seleccione ‘Herramientas universales de desarrollo de aplicaciones de Windows’ en la selección de funciones.

¡Cuando termine, estará listo para desarrollar aplicaciones universales de Windows!
