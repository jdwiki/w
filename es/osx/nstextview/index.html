NSTextView

  • Creando un NSTextView
  • C objetivo

En esta página

  • Creando un NSTextView
  • C objetivo

NSTextView es el controlador principal de Apple del sistema de texto de AppKit. Contiene todo lo que necesita para crear un espacio de visor/editor de texto en aplicaciones OS X (renombrado macOS).

Creando un NSTextView #

Gráficamente #

En XCode, se puede crear un NSTextView simple arrastrando y soltando uno desde la Biblioteca de objetos.

Este NSTextView se encuentra dentro de un NSScrollView que se configura automáticamente para expandirse verticalmente con la vista de texto. Asegúrese de que al arrastrar la opción (⌥) haga conexiones a la vista de texto y no a la vista de desplazamiento.

Programáticamente #

La creación de un NSTextView mediante programación permite un mayor control y personalización. Es un poco más difícil y requiere conocimiento del sistema de texto para utilizarlo en todo su potencial. Más información sobre el sistema de texto y más está disponible aquí. Los elementos básicos necesarios para crear una vista de texto mediante programación son los siguientes:

  • Se requieren otros tres objetos para que funcione un NSTextView completamente funcional:

    1. An NSLayoutManager - performs glyph/character layout.
    2. An NSTextContainer - controls graphical space that glyphs/characters can inhabit.
    3. An NSTextStorage - holds the actual string data that NSTextView displays.

  • Un NSTextStorage puede tener muchos NSLayoutManager, pero un NSLayoutManager solo puede tener un NSTextStorage. Esto es útil si desea mostrar los mismos datos de diferentes maneras al mismo tiempo.

  • NSLayoutManager puede tener muchos NSTextContainer. Útil para texto paginado.

  • NSTextView solo puede tener un NSTextContainer a la vez.

  • Ciertas cosas integradas en NSTextView están prohibidas en el momento de escribir este artículo. Por ejemplo, las funciones integradas de Buscar y reemplazar no se pueden personalizar, pero se pueden anular con funciones personalizadas.

Puede encontrar más información sobre las formas de utilizar el sistema de texto aquí.

Ahora para el código. Este código creará un NSTextView simple, sin siquiera desplazarse. Cosas como el desplazamiento y la paginación estarán en otro ejemplo.

C objetivo #

// This code resides in an NSDocument object's windowControllerDidLoadNib:(NSWindowController *)windowController method.
// This is done simply because it is easy and automatically gets called upon.

// This method is also where the following NSRect variable gets size information. We need this information for this example.
NSRect windowFrame = windowController.window.contentView.frame;
NSTextStorage *textStorage = [[NSTextStorage alloc] initWithString:@"Example text!"];
NSLayoutManager *manager = [[NSLayoutManager alloc] init];
NSTextContainer *container = [[NSTextContainer alloc] initWithContainerSize:NSMakeSize(windowFrame.size.width, windowFrame.size.height)];
NSTextView *textView = [[NSTextView alloc] initWithFrame:windowFrame textContainer:container];

[textStorage addLayoutManager:manager];
[manager addTextContainer:container];
[windowController.window setContentView:textView];

¡Felicidades! ¡Ha creado un NSTextView programáticamente!
