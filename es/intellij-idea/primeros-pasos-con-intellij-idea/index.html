¡Hola Mundo! #

Esto te enseñará cómo hacer tu primer proyecto usando IDEA.

Inicie IDEA y haga clic en Crear nuevo proyecto en la pantalla de inicio: Haga clic en Siguiente en la siguiente pantalla. Estamos creando un proyecto Java simple, por lo que no necesitamos complementos ni extras para este proyecto. Utilice la siguiente pantalla para crear el proyecto de plantilla Java Hello World: Finalmente, asigne un nombre a su proyecto y seleccione una ubicación en el disco, y haga clic en Finalizar: Debería terminar con una ventana que se parece a esto: En este punto, el proyecto está listo para funcionar, simplemente haga clic en el botón Ejecutar o vaya a Ejecutar -> Ejecutar 'Principal' ¡Y tu estas listo! ¡La consola aparecerá automáticamente, saludando al mundo!

Instalación o configuración #

Hay dos versiones principales de IntelliJ IDEA: la edición Community y la edición Ultimate. La edición Community es gratuita y no carece de características en términos de desarrollo de Java SE.

ventanas y linux #

Descargue IntelliJ IDEA desde el sitio web de JetBrains y siga los procedimientos de instalación. Si el kit de desarrollo de Java (JDK) no está instalado, descargue e instale el JDK. Tenga en cuenta que necesita el JDK, solo tener Java Runtime Enviroment (JRE) no es suficiente.

Una vez que se haya descargado IntelliJ IDEA:

  1. Ejecute el instalador
  2. Pulse siguiente
  3. Elija una carpeta para instalar IntelliJ IDEA (en la mayoría de los casos, déjela como predeterminada)
  4. Elija una carpeta del menú de inicio para crear accesos directos de IntelliJ IDEA (en la mayoría de los casos, déjelo como predeterminado)
  5. Elija si desea crear un acceso directo en el escritorio y elija si desea asociar varios archivos Java con IntelliJ IDEA
  6. Presione siguiente y espere a que se instale

OS X/mac OS #

Descargue IntelliJ IDEA desde el sitio web de JetBrains, abra el archivo de imagen de disco (*.dmg) descargado y arrastre y suelte la aplicación en el alias de su carpeta /Applications.

arco linux #

IntelliJ IDEA se puede instalar en Arch Linux usando su administrador de paquetes, pacman. Abra una terminal e ingrese el siguiente comando.

sudo pacman -S intellij-idea-community-edition

No es necesario usar sudo si está ejecutando como usuario root.

ubuntu #

(1) Instale el paquete ubuntu-make.

Para ubuntu 16.04 y posteriores,

sudo apt install ubuntu-make

Para versiones anteriores de ubuntu,

sudo add-apt-repository ppa:ubuntu-desktop/ubuntu-make  
sudo apt-get update
sudo apt-get install ubuntu-make

(2) Después de instalar Ubuntu Make, haga una

umake ide idea

Ruta de instalación predeterminada: /home/current-user/.local/share/umake/ide/idea

Siga el proyecto hello_world mencionado anteriormente.

Siga ubuntu-make page para cambiar la instalación predeterminada e instalar otros IDE.

Migración desde Eclipse #

Intellij IDEA intenta atraer a la amplia base de fans de Java que utiliza Eclipse para su desarrollo al permitir que los desarrolladores migren sus proyectos de Eclipse a una estructura IDEA con unos pocos clics.

Primero, inicie IDEA y haga clic en Importar proyecto desde la ventana de inicio: Luego, seleccione su proyecto Eclipse usando la ventana del explorador

Intellij le pedirá el modelo desde el que está importando, asegúrese de que Eclipse esté seleccionado antes de hacer clic en Siguiente

La siguiente pantalla mostrará una confirmación de la ruta que desea importar, simplemente haga clic en Siguiente:

A continuación, seleccione los módulos que desea crear. En el proyecto de ejemplo particular, solo los proyectos Alice y BuggyRos importaban cuando se trabajaba en Eclipse

Finalmente, asegúrese de seleccionar la versión correcta del JDK antes de Terminar

¡Y el proyecto Eclipse se migró por completo a Intellij! El proyecto aún se abrirá en ambos IDE y será completamente funcional en ambos.
