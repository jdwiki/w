Navegación básica #

Ir al editor (desde la ventana de herramientas) #

Esc

Cambiar el foco a la ventana de herramientas correspondiente #

Windows: Alt + <número de la ventana de herramientas> #

OS X / macOS: Cmd + <número de la ventana de herramientas> #

Por ejemplo, cambiar el enfoque a la ventana del proyecto #

Ventanas: Alt + 1 #

OS X/macOS: Cmd + 1 #

Ventana emergente de archivos recientes #

Ventanas: Ctrl + E #

OS X/macOS: Cmd + E #

Buscar acción #

Ventanas: Ctrl + Shift + A #

OS X/macOS: Cmd + Mayús + A #

Navegar a #

  • Expediente : #

    • Windows: Ctrl + Shift + N #

    • OS X / macOS: Cmd + Shift + N #

  • Clase : #

    • Windows: Ctrl + N #

    • OS X / macOS: Cmd + N #

  • Símbolo (nombre de clase/método/variable/constante) : #

    • Windows: Ctrl + Alt + Shift + N #

    • OS X / macOS: Cmd + Option + Shift + N #

    Note that you can use class name to narrow down the method/variable/constant search, for example to find symbol usersCollection in class UserDAO type:

      UserDAO.usersCollection

  • En todas partes : #

    • Windows: Shift + Shift #

    • OS X / macOS: Shift + Shift #

Para buscar algo que tiene varias palabras, por ejemplo, InetAddressCachePolicy puede simplemente escribir InAddCacPo o algo similar que contenga partes de palabras en el nombre completo.

OS X/macOS: Cmd + L #

OS X / macOS: Cmd + Mayús + Retroceso #

Selección #

Selección con alcance creciente

Esto es útil cuando desea seleccionar un bloque para extraer una variable/método, etc., no es necesario hacer una coincidencia precisa entre paréntesis, simplemente coloque el signo de intercalación en algún lugar de la declaración y siga haciendo esto.

Ventanas: Ctrl + W #

OS X/macOS: Cmd + W #

Selección con alcance decreciente

Ventanas: Ctrl + Shift + W #

OS X/macOS: Cmd + Mayús + W #

Esta característica también es muy útil al editar/jugar con documentos json en su IDE.

Selección vertical

presione y mantenga

Ventanas: Alt #

OS X / macOS: Optar #

y seleccione normalmente usando el mouse / trackpad (la forma en que selecciona una palabra en una fila, etc.)

Así es como debería verse

Múltiples símbolos de intercalación

presione y mantenga

Ventanas: Alt + Mayús #

OS X/macOS: Optar + Mayús #

y haz clic donde quieras poner un símbolo de intercalación. Puede optar por poner varios símbolos de intercalación en una sola línea o entre líneas en diferentes posiciones.

Ahora puede realizar todas las operaciones que habría podido realizar en una sola palabra seleccionada (mantenga presionada Ctrl (windows) u opción (mac OS) y use Izquierda o Derecha para saltar entre palabras) y todas esas afectarán todas las posiciones de intercalación.

Incluso puede cortar/pegar múltiples selecciones de un lugar a otro.

Tener varios símbolos de intercalación es muy útil cuando desea cambiar la estructura del texto en muchas líneas/muchas posiciones en la misma línea.

Seleccionando ocurrencias duplicadas

Seleccione algún texto y presione

Ventanas: Alt + J #

OS X/macOS: ctrl + G #

para seleccionar la próxima aparición del mismo texto.

Obtiene un signo de intercalación en cada ocurrencia seleccionada que podría usarse para cambiar cada ocurrencia simultáneamente.
