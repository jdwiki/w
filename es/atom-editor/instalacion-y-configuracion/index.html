Instalación de Atom en Windows #

Usando el instalador oficial #

Descargue el instalador desde [el sitio web oficial] (https://atom.io/). Automáticamente agregará atom y apm (Atom Package Manager) a su variable %PATH%.

Construyendo desde la fuente #

Requisitos:

  • Node.js 4.4.x o posterior -Python 2.7.x
  • 7zip
  • Visual Studio (una de las versiones a continuación)
    • Visual C++ Build Tools 2015
    • Visual Studio 2013 Update 5 (Express Edition or better)
    • Visual Studio 2015 (Community Edition or better)
  • Git

Ejecute los siguientes comandos en el símbolo del sistema:

cd C:/
git clone https://github.com/atom/atom.git
cd atom
script/build

Instalación de Atom en Mac #

Instalar desde un zip #

  1. Descargue el archivo zip atom-mac.zip del repositorio de Atom GitHub aquí
  2. Descomprima el archivo haciendo doble clic sobre él en Finder
  3. Arrastre la aplicación Atom a su carpeta “Aplicaciones”
  4. Ejecute la aplicación Atom.

Construcción desde la fuente #

Requisitos:

  • macOS 10.8 o superior
  • Node.js 4.4x o posterior
  • npm 3.10.x o posterior -Xcode

Instalación:

git clone https://github.com/atom/atom.git
cd atom
script/build

Después de compilar, instálelo con script/build --install

Instalación de Atom en Linux #

Instalar desde un paquete #

Debian, Ubuntu, etc #

$ sudo dpkg -i atom-amd64.deb
$ sudo apt-get -f install

RedHat Enterprise, CentOS, Oracle Linux, Scientific Linux, etc. #

$ sudo yum install -y atom.x86_64.rpm

Fedora (gestor de paquetes DNF) #

$ sudo dnf install -y atom.x86_64.rpm

SUSE (administrador de paquetes Zypp) #

$ sudo zypper in -y atom.x86_64.rpm

Construcción desde la fuente #

Requisitos:

  • SO con arquitectura de 64 o 32 bits
  • Cadena de herramientas C++ 11
  • Git
  • Node.js 4.4x o posterior
  • npm 3.10.x o posterior
  • Encabezados de desarrollo de llaveros de GNOME

Ejecute los siguientes comandos:

git clone https://github.com/atom/atom.git
cd atom
script/build

Para obtener instrucciones específicas relacionadas con una única distribución de Linux, lea estas instrucciones.
