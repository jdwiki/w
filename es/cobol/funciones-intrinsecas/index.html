Funciones intrínsecas

Las funciones intrínsecas se incluyen en el estándar COBOL como un conjunto de funciones que devuelven valores de un algoritmo específico, con cero o más argumentos. Estas funciones intrínsecas se proporcionan como una facilidad del compilador y del sistema de tiempo de ejecución. Los elementos devueltos son campos COBOL temporales y pueden ser datos de caracteres, campos de bits o valores numéricos.

Los ejemplos incluyen funciones trigonométricas, rutinas de fecha y hora, conversiones de tipos de datos, desviación estándar y otros algoritmos de soporte.

Se requiere la palabra clave FUNCTION a menos que la fuente (o la opción de tiempo de compilación) incluye

ENVIRONMENT DIVISION.
CONFIGURATION SECTION.
REPOSITORY.
    FUNCTION ALL INTRINSIC.

Donde TODO INTRÍNSICO puede ser una lista de funciones que se utilizarán sin el Prefijo FUNCTION en sentencias PROCEDURE DIVISION.

La función LENGTH tiene un historial ordenado. Algunos compiladores incluyen un LENGTH palabra reservada. Para GnuCOBOL, esta palabra reservada solo se reconoce cuando se usa en la frase ‘LENGTH OF’, se requiere el token ‘OF’ para eliminar la ambigüedad función de la antigua extensión de palabra reservada.

Ejemplo de RECORTE DE FUNCIÓN #

01 some-string PIC X(32).

...

MOVE "    a string literal" TO some-string

DISPLAY ":" some-string ":"
DISPLAY ":" FUNCTION TRIM(some-string) ":"
DISPLAY ":" FUNCTION TRIM(some-string LEADING) ":"
DISPLAY ":" FUNCTION TRIM(some-string TRAILING) ":"

MAYÚSCULAS #

MOVE FUNCTION UPPER-CASE("Hello World!") TO SOME-FIELD
DISPLAY SOME-FIELD

Producción

HELLO WORLD!

Función MINÚSCULAS #

MOVE FUNCTION LOWER-CASE("HELLO WORLD!") TO SOME-FIELD
DISPLAY SOME-FIELD

Producción

hello world!
