Uso del procesamiento con texto sublime #

Para agregar un idioma a Sublime Text, usa Control de paquete. Puede hacerlo pegando el código de Python correcto (disponible en el sitio de Control de paquetes vinculado anteriormente) o descargando el archivo .sublime-package (también disponible para descargar en el sitio). Una vez que configure Package Control, reinicie Sublime Text.

Para instalar el paquete de Procesamiento desde Package Control, siga los siguientes pasos:

  • Ejecutar Herramientas | Instale el procesamiento de Java para instalar la herramienta de procesamiento de Java. Esta herramienta es necesaria para crear bocetos de procesamiento en la línea de comandos y editores como Sublime Text y Atom.
  • Abra la paleta de comandos (Ctrl+Shift+P o Cmd+Shift+P)
  • Busque ‘Control de paquetes: Paquete de instalación’
  • Busque ‘Procesando’ e instale el paquete
  • Reiniciar texto sublime

Después de seguir estos pasos, debería poder seleccionar Procesamiento como idioma. Esto facilitará la codificación en Procesamiento en Sublime Text.

Uso de Processing con el editor Atom #

Hay varios paquetes que pueden ejecutar bocetos de procesamiento en el editor Atom. Estas instrucciones utilizan el paquete Script. También hay paquetes disponibles para resaltado de sintaxis y autocompletar, que son necesarios para Script para identificar los tipos de archivos de procesamiento.

Instale el complemento Script. Ya sea ejecutando apm install script desde la línea de comandos o buscando el paquete ‘script’ por rgbkrk en la pestaña Instalar en la configuración de Atom (acceso directo command +, o ctrl +,).

Una vez que Script esté instalado, deberá instalar processing-java. Esta herramienta viene con el software principal de Processing y es necesaria para crear bocetos de Processing en la línea de comandos y en los editores:

  • MacOS: Ejecute Herramientas > Instalar "processing-java".

  • Linux: Agregue el directorio Processing a su variable de entorno PATH (reemplace /path/to/processing con la ruta donde está instalado Processing):

    sudo ln -s /path/to/processing/processing-java /usr/local/bin/

  • Windows: Agregue el directorio de Procesamiento a su variable de entorno PATH:

  • Abra Configuración avanzada del sistema ejecutando sysdm.cpl o buscando en el Panel de control.

  • Haga clic en el botón Variable de entorno en la pestaña Avanzado.

  • Edite la variable RUTA para incluir el directorio de procesamiento en las variables de usuario (solo para su cuenta) o en las variables del sistema (para todos los usuarios).

Ahora, puede ejecutar bocetos de procesamiento ejecutando Paquetes> Script> Ejecutar script. El acceso directo predeterminado es comando + shift + b o ctrl + shift + b, pero para disminuir aún más el dolor de la transición, puede vincular el acceso directo del IDE de procesamiento para ejecutar el boceto. Para hacer eso:

  • Abra el archivo Atom Keymap ejecutando File > Keymap

  • Pegue las siguientes líneas al final del archivo (siéntase libre de cambiar el enlace a lo que quiera).

    'atom-text-editor':
  'ctrl-r': 'script:run'

Uso del procesamiento con Eclipse #

Para usar Processing en Eclipse, comience creando un nuevo proyecto Java. Luego, seleccione Archivo > Importar y luego elija General > Sistema de archivos para ubicar el archivo core.jar. Se puede encontrar en PATH_TO_PROCESSING/core/library/ para Windows o /Applications/Processing 3.app/Contents/Java/core/library/ para Mac. Una vez que esto se complete, haga clic con el botón derecho en core.jar y agréguelo a la ruta de compilación.

El modelo para usar Processing en Eclipse es el siguiente:

import processing.core.PApplet;

public class UsingProcessing extends PApplet {

    // The argument passed to main must match the class name
    public static void main(String[] args) {
        PApplet.main("UsingProcessing");
    }

    // method used only for setting the size of the window
    public void settings(){
        
    }

    // identical use to setup in Processing IDE except for size()
    public void setup(){
        
    }

    // identical use to draw in Prcessing IDE
    public void draw(){

    }
}

El método settings() se utiliza para establecer el tamaño de la ventana. Por ejemplo, para crear una ventana de 400x400, escriba lo siguiente:

 public void settings(){
    size(400,400);   
 }

Todo lo demás como se describe en la documentación de Hello World en términos del uso de setup() y draw() se aplica aquí.

Como ejemplo final, aquí está el código del [ejemplo de dibujo de una línea][2] si se escribiera en Eclipse:

import processing.core.PApplet;

public class UsingProcessing extends PApplet {

    // The argument passed to main must match the class name
    public static void main(String[] args) {
        PApplet.main("UsingProcessing");
    }

    // method for setting the size of the window
    public void settings(){
        size(500, 500);
    }

    // identical use to setup in Processing IDE except for size()
    public void setup(){
        background(0);
        stroke(255);
        strokeWeight(10);
    }

    // identical use to draw in Prcessing IDE
    public void draw(){
        line(0, 0, 500, 500);
    }
}
