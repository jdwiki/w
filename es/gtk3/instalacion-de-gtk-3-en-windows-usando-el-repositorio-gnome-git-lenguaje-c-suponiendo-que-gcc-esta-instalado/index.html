Descarga de GTK+3 (también apto para otras versiones) y configuración #

Descargando el Paquete:

La forma más fácil de descargar el paquete GTK requerido es buscarlo y descargarlo usando este enlace: https://git.gnome.org/browse/gtk+ (GNOME GIT Repository)

GNOME GIT Repository proporciona los paquetes para diferentes versiones y puede encontrar fácilmente la versión deseada desplazándose por la lista. Estos son cargados por otros usuarios o autores y, a menudo, encontrará nuevos paquetes.

Alternativas:

La otra opción o el método alternativo para adquirir el paquete es usar MSYS2. Este método es un poco complicado y no está bien explicado ni documentado. La mejor manera es descargar directamente el paquete desde un sitio web de alojamiento de archivos externo.

Método MSYS2-

  1. http://www.gtk.org/download/windows.php (recurso del sitio web oficial sobre la adquisición del paquete)

  2. https://msys2.github.io/ (descargar MSYS2)

Nota: el resto de esta documentación se basa en la suposición de que el paquete GTK se descarga del repositorio GNOME GIT. No probado con el método MSYS2. Dado que el paquete sigue siendo universal, el método utilizado no debería ser un problema. Basado en la facilidad, la forma altamente recomendada de obtener el paquete es usar el repositorio GNOME GIT.

Extracción y almacenamiento del paquete en una ubicación deseada (explicado usando la unidad C:).

  1. Cree una carpeta denominada gtk en C: Drive. Puede elegir otros nombres para la carpeta, pero este nombre nos ayuda a identificar el contenido con bastante facilidad y recordar el nombre se vuelve más fácil. Denominemos la ubicación de la carpeta Gtk como %GD% para simplificar y en este caso es C:\gtk (podría ser diferente en su caso).

  2. Configuración de la ruta para el binario Gtk en las variables de entorno de Windows. Esto es importante para acceder y obtener las bibliotecas Gtk requeridas mientras se usa Cmd para compilar y ejecutar los programas. Para configurar la variable de entorno de ruta:

  • Windows XP : haga clic derecho en "Mi PC" -> "Propiedades".
  • Windows Vista/7/8/10 : haga clic derecho en "Equipo" -> "Propiedades" -> "Configuración avanzada del sistema". Haga clic en "pestaña Avanzado" -> "Entorno variables".
  • Agregue una nueva Ruta del archivo bin Gtk (%GD%\bin) a la variable Ruta (en Variables del sistema o agregue una variable llamada Ruta a la Sección de variables del usuario y agregue el valor en ella) disponible en el cuadro de diálogo que se abre . También debe agregar una nueva variable llamada PKG_CONFIG_PATH y darle el siguiente valor: C:\gtk\lib\pkgconfig (ya que mi carpeta gtk se encuentra en la unidad C:).
Nombre de la variable Valor
Camino %GD%\bin
PKG_CONFIG_PATH %GD%\lib\pkgconfig

%GD% es la ubicación de la carpeta Gtk Ahora hemos terminado con la extracción y configuración de las variables de entorno. Encontrará que he agregado el valor a la variable Ruta en las secciones de variables del sistema, así como en la sección de variables de usuario. Añadir a cualquiera de estas secciones es más que suficiente.

3.Probar la configuración (debe tener GCC instalado en su sistema, preferiblemente MinGW (http://www.mingw.org/): Ejecute los siguientes comandos en el cmd:

pango-querymodules:%GTKDIR%\etc\pango\pango.modules

gdk-pixbuf-query-loaders:%GTKDIR%\lib\gdk-pixbuf-2.0\2.10.0\loaders.cache

gtk-query-immodules-3.0: %GTKDIR%\lib\gtk-3.0\3.0.0\immodules.cache

  • En la consola, verifique que “pkg-config” imprima algo razonable escribiendo: pkg-config --cflags --libs gtk+-3.0 Debería imprimir algo similar a lo que se muestra en la siguiente imagen.[! Resultado del comando]4

Eso es todo. Ha descargado y configurado GTK en su sistema Windows.
