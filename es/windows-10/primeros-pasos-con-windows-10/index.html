Instalación o configuración #

Instalarse en Windows 10 es tan fácil como siempre y, a pesar de los rumores (verdaderos) que circulan acerca de que Windows 10 es la última versión “normal” de Windows, ha sido y siempre será el mismo proceso.

Instalar una nueva versión

  1. Obtenga la imagen de Windows (vaya directamente al paso 4 si tiene un disco comprado en una tienda)
    2. Vaya al sitio de Windows technet y descargue la versión adecuada de Windows 10. Deberá ingresar la clave de producto que compró.
  2. Descargar un reproductor de imágenes USB
    3. Busque y descargue la herramienta de imágenes Rufus USB. Ejecuta esto.
  3. Cargue la imagen en su USB
    4. Haga clic en el menú desplegable en la parte superior y seleccione su USB. Luego haga clic en el segundo menú desplegable y seleccione la opción de imagen ISO. Finalmente, haga clic en la imagen al lado del menú desplegable y busque su archivo ISO. Selecciónelo y luego haga clic en 'Iniciar'. Cuando esto haya terminado, expulse su USB.
  4. Instalando
    5. Apague su computadora portátil y vuelva a encenderla. Esté atento a cualquier texto que aparezca diciendo 'Presione para ingresar al menú de arranque'. Cuando vea esto, tome nota mental del nombre del botón y búsquelo en su teclado. Luego inserte su medio de instalación y reinicie nuevamente, y cuando aparezca el texto, presione el botón. Si todo va bien, debería tener la opción de ejecutar los medios de instalación. Una vez que eso suceda, simplemente siga las instrucciones en pantalla para instalar Windows 10.

Actualización

  1. Descargando la iso
    2. Siga el paso 1 en la guía anterior
  2. Montar el archivo ISO
    3. Haga clic derecho en este archivo y haga clic en la opción 'Montar'
  3. Ejecute el instalador
    4. Vaya a 'Esta PC' y abra la imagen del disco (debe aparecer como un disco). Luego ejecute el archivo 'setup.exe' allí. Finalmente, simplemente siga las instrucciones en pantalla. Esto requiere derechos de administrador y lleva un tiempo, pero una vez que esto haya sucedido, todos sus archivos estarán donde estaban antes, ¡solo será Windows 10!

    OneGet y gestores de paquetes #

    OneGet fue originalmente un producto del Open Source Technology Center de Microsoft. No solo está inspirado en los administradores de paquetes de Linux de código abierto, sino que OneGet también es de código abierto. Ahora es parte de PowerShell

    A diferencia de los administradores de paquetes basados ​​en Unix (como apt-get, yum o dpkg), Windows permite el uso de Oneget a través de PowerShell. Para hacer esto, primero debe abrir un Powershell como administrador y ejecutar Set-ExecutionPolicy RemoteSigned. Esto es para garantizar que los scripts de Powershell disponibles en Oneget puedan ejecutarse. Luego (usando la misma instancia de PowerShell), ejecute Import-Module –Name OneGet. Esto importa el módulo PoerShell. Luego, reinicie su instancia de PowerShell y ejecute Get-Command –Module OneGet para ver todos los comandos disponibles. Para instalar un paquete, ejecute el comando Find-Package | <Nombre_paquete. Cuando se le solicite instalar Nuget, haga clic en Sí.

    Agregar Chocolatey como proveedor de OneGet #

    Desafortunadamente, OneGet no tiene el proveedor de paquetes chocolatey instalado de manera predeterminada, pero es posible agregarlo.

    Abra una ventana de PowerShell y ejecute get-packageprovider -name chocolatey. A continuación, debería ver el siguiente mensaje:

    El proveedor ‘chocolatey v2.8.5.130’ no está instalado. mayo achocolatado ser descargado manualmente desde https://oneget.org/ChocolateyPrototype-2.8.5.130.exe y instalado. ¿Le gustaría que PackageManagement descargue automáticamente e instalar ‘chocolatey’?

    [Y] Sí [N] No [S] Suspender [?] Ayuda (el valor predeterminado es “Y”):

    Escriba ‘Y’ y luego presione enter.

    Name                     Version          
----                     -------           
Chocolatey               2.8.5.130

    Encontrar paquetes #

    Para la mayoría de los programas conocidos, suele funcionar el nombre de la aplicación (por ejemplo, Chrome). pero, al igual que el chocolate puede ser difícil de navegar, también lo puede ser OneGet. Afortunadamente, OneGet tiene una implementación de choco search. Este comando ahora es find-package <Nombre del paquete>. Esto le dará una lista de paquetes como se ve a continuación:

    Name                           Version          Source                         Summary
----                           -------          ------                         -------
thunderbird                    45.2.0           chocolatey                     A free email client from Mozilla
Google Chrome                  47.5.8           OneGet                         A high end web browser

    Instalando paquetes #

    Para instalar un paquete, debe ejecutar PowerShell como administrador y ejecutar el comando install-package <Package Name>

    Los paquetes provienen de una fuente de paquetes que no está marcada como confiado. ¿Está seguro de que desea instalar el software de ‘chocolatey’? [Y] Sí [A] Sí a todo [N] No [L] No a todo [S] Suspender [?] Ayuda (el valor predeterminado es “N”):

    Para esto, solo puede presionar y y/o enter. Luego verá una pantalla similar a la siguiente

    Name                           Version          Source           Summary
----                           -------          ------           -------
<Package Name>                 <Most recent>    <Server name>    <Description>

    Desinstalando #

    Para desinstalar un paquete, simplemente use el siguiente comando

    uninstall-package vlc
