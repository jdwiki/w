Sintaxis #

  • String.characters // Devuelve una matriz de los caracteres en la cadena
  • String.characters.count // Devuelve el número de caracteres
  • String.utf8 // A String.UTF8View, devuelve los puntos de caracteres UTF-8 en la cadena
  • String.utf16 // A String.UTF16View, devuelve los puntos de caracteres UTF-16 en la cadena
  • String.unicodeScalars // A String.UnicodeScalarView, devuelve los puntos de caracteres UTF-32 en la cadena
  • String.isEmpty // Devuelve verdadero si la cadena no contiene ningún texto
  • String.hasPrefix(String) // Devuelve verdadero si la cadena tiene como prefijo el argumento
  • String.hasSuffix(String) // Devuelve verdadero si String tiene como sufijo el argumento
  • String.startIndex // Devuelve el índice que corresponde al primer carácter de la cadena
  • String.endIndex // Devuelve el índice que corresponde al lugar después del último carácter de la cadena
  • String.components(separatedBy: String) // Devuelve una matriz que contiene las subcadenas separadas por la cadena de separación dada
  • String.append(Character) // Agrega el carácter (dado como argumento) a la Cadena

Una String en Swift es una colección de caracteres y, por extensión, una colección de escalares Unicode. Debido a que Swift Strings se basa en Unicode, pueden ser cualquier valor escalar de Unicode, incluidos idiomas distintos del inglés y emojis.

Debido a que dos escalares pueden combinarse para formar un solo carácter, la cantidad de escalares en una Cadena no es necesariamente siempre la misma que la cantidad de caracteres.

Para obtener más información sobre cadenas, consulte El lenguaje de programación Swift y la [Referencia de estructura de cadenas](https: //developer.apple.com/library/tvos/documentation/Swift/Reference/Swift_String_Structure/index.html).

Para obtener detalles de implementación, consulte “Swift String Design”

Literales de cadenas y caracteres #

String Los literales en Swift están delimitados con comillas dobles ("):

let greeting = "Hello!"  // greeting's type is String

Los caracteres se pueden inicializar a partir de literales de cadena, siempre que el literal contenga solo un grupo de grafemas:

let chr: Character = "H" // valid
let chr2: Character = "😊" // valid
let chr3: Character = "abc" // invalid - multiple grapheme clusters

Interpolación de cadenas #

Interpolación de cadenas permite inyectando una expresión directamente en un literal de cadena. Esto se puede hacer con todo tipo de valores, incluidas cadenas, enteros, números de punto flotante y más.

La sintaxis es una barra invertida seguida de paréntesis que envuelven el valor: \(valor). Cualquier expresión válida puede aparecer entre paréntesis, incluidas las llamadas a funciones.

let number = 5
let interpolatedNumber = "\(number)"  // string is "5"
let fortyTwo = "\(6 * 7)"             // string is "42"

let example = "This post has \(number) view\(number == 1 ? "" : "s")"
// It will output "This post has 5 views" for the above example.
// If the variable number had the value 1, it would output "This post has 1 view" instead.

Para los tipos personalizados, el comportamiento predeterminado de la interpolación de cadenas es "\(myobj)" es equivalente a String(myobj), la misma representación utilizada por print(myobj). Puede personalizar este comportamiento implementando el protocolo CustomStringConvertible para su tipo.

Para Swift 3, de acuerdo con SE-0089, String.init <T>(_:) ha sido renombrado a String.init<T>(describiendo:).

La interpolación de cadenas "\(myobj)" preferirá el nuevo inicializador String.init<T: LosslessStringConvertible>(_:), pero recurrirá a init<T>(describiendo:) si el valor es no LosslessStringConvertible.

Caracteres especiales #

Ciertos caracteres requieren una secuencia de escape especial para usarlos en cadenas literales:

Personaje Sentido
\0 el caracter nulo
\\ una barra invertida simple, \
\t un carácter de tabulación
\v una pestaña vertical
\r un [retorno de carro] (https://en.wikipedia.org/wiki/Newline)
\n un avance de línea (“nueva línea”)
\" una comilla doble, "
\' una comilla simple, '
\u{n} el punto de código Unicode n (en hexadecimal)

Ejemplo:

let message = "Then he said, \"I \u{1F496} you!\""

print(message) // Then he said, "I 💖 you!"

Concatenar cadenas #

Concatene cadenas con el operador + para producir una nueva cadena:

let name = "John"
let surname = "Appleseed"
let fullName = name + " " + surname  // fullName is "John Appleseed"

Agregue a una cadena [mutable](http://stackoverflow.com/questions/24002092/what-is-the-difference- between-let-and-var-in-swift) usando += operador de asignación compuesto, o usando un método:

let str2 = "there"
var instruction = "look over"
instruction += " " + str2  // instruction is now "look over there"

var instruction = "look over"
instruction.append(" " + str2)  // instruction is now "look over there"

Agregue un solo carácter a una cadena mutable:

var greeting: String = "Hello"
let exclamationMark: Character = "!"
greeting.append(exclamationMark) 
// produces a modified String (greeting) = "Hello!"

Agregar varios caracteres a una cadena mutable

var alphabet:String = "my ABCs: "
alphabet.append(contentsOf: (0x61...0x7A).map(UnicodeScalar.init)
                                         .map(Character.init) )
// produces a modified string (alphabet) = "my ABCs: abcdefghijklmnopqrstuvwxyz"

Se ha cambiado el nombre de appendContentsOf(_:) a append(_:).

Une una secuencia de cadenas para formar una nueva cadena usando joinWithSeparator(_:):

let words = ["apple", "orange", "banana"]
let str = words.joinWithSeparator(" & ")

print(str)   // "apple & orange & banana"

Se ha cambiado el nombre de joinWithSeparator(_:) a joined(separator:).

El separador es la cadena vacía por defecto, así que ["a", "b", "c"].joined() == "abc".

Codificación y descomposición de cadenas #

Una [Cadena] Swift (https://developer.apple.com/library/ios/documentation/Swift/Conceptual/Swift_Programming_Language/StringsAndCharacters.html) está hecha de [Unicode] (http://www.unicode.org/standard /standard.html) puntos de código. Se puede descomponer y codificar de varias maneras diferentes.

let str = "ที่👌①!"

Cadenas en descomposición #

Los caracteres de una cadena son Unicode [grupos de grafemas extendidos] (https://developer.apple.com/library/mac/documentation/Cocoa/Conceptual/Strings/Articles/stringsClusters.html):

Array(str.characters)  // ["ที่", "👌", "①", "!"]

Los unicodeScalars son [puntos de código] Unicode (https://en.wikipedia.org/wiki/Code_point) que forman una cadena (observe que ที่ es un grupo de grafemas, pero 3 puntos de código: 3607, 3637 , 3656, por lo que la longitud de la matriz resultante no es la misma que con caracteres):

str.unicodeScalars.map{ $0.value }  // [3607, 3637, 3656, 128076, 9312, 33]

Puede codificar y descomponer cadenas como [UTF-8] (https://en.wikipedia.org/wiki/UTF-8) (una secuencia de UInt8s) o [UTF-16] (https://en .wikipedia.org/wiki/UTF-8) .wikipedia.org/wiki/UTF-16) (una secuencia de UInt16s):

Array(str.utf8)   // [224, 184, 151, 224, 184, 181, 224, 185, 136, 240, 159, 145, 140, 226, 145, 160, 33]
Array(str.utf16)  // [3607, 3637, 3656, 55357, 56396, 9312, 33]

Longitud de cadena e iteración #

Los caracteres, unicodeScalars, utf8 y utf16 de una cadena son todos Colección, por lo que puede obtener su conteo e iterar sobre ellos:

// NOTE: These operations are NOT necessarily fast/cheap! 

str.characters.count     // 4
str.unicodeScalars.count // 6
str.utf8.count           // 17
str.utf16.count          // 7

for c in str.characters { // ...
for u in str.unicodeScalars { // ...
for byte in str.utf8 { // ...
for byte in str.utf16 { // ...

Examinar y comparar cadenas #

Compruebe si una cadena está vacía:

if str.isEmpty {
    // do something if the string is empty
}

// If the string is empty, replace it with a fallback:
let result = str.isEmpty ? "fallback string" : str

Compruebe si dos cadenas son iguales (en el sentido de [equivalencia canónica Unicode] (http://www.unicode.org/glossary/#canonical_equivalent)):

"abc" == "def"          // false
"abc" == "ABC"          // false
"abc" == "abc"          // true

// "LATIN SMALL LETTER A WITH ACUTE" == "LATIN SMALL LETTER A" + "COMBINING ACUTE ACCENT"
"\u{e1}" == "a\u{301}"  // true

Compruebe si una cadena comienza/termina con otra cadena:

"fortitude".hasPrefix("fort")      // true
"Swift Language".hasSuffix("age")  // true

Invertir cuerdas #

let aString = "This is a test string."

// first, reverse the String's characters 
let reversedCharacters = aString.characters.reverse()

// then convert back to a String with the String() initializer
let reversedString = String(reversedCharacters)

print(reversedString) // ".gnirts tset a si sihT"

let reversedCharacters = aString.characters.reversed()
let reversedString = String(reversedCharacters)

Comprobar si la cadena contiene caracteres de un conjunto definido #

Letras

let letters = CharacterSet.letters

let phrase = "Test case"
let range = phrase.rangeOfCharacter(from: letters)

// range will be nil if no letters is found
if let test = range {
   print("letters found")
}
else {
   print("letters not found")
}

let letters = NSCharacterSet.letterCharacterSet()

let phrase = "Test case"
let range = phrase.rangeOfCharacterFromSet(letters)

// range will be nil if no letters is found
if let test = range {
   print("letters found")
}
else {
  print("letters not found")
}

La nueva estructura CharacterSet que también está unida a la clase NSCharacterSet de Objective-C define varios conjuntos predefinidos como:

  • decimalDigits
  • letras en mayúscula
  • alfanuméricos
  • caracteres de control
  • Caracteres ilegales
  • y más puede encontrar en la referencia NSCharacterSet.

También puede definir su propio conjunto de caracteres:

let phrase = "Test case"
let charset = CharacterSet(charactersIn: "t")

if let _ = phrase.rangeOfCharacter(from: charset, options: .caseInsensitive) {
   print("yes") 
}
else {
   print("no")
}

let charset = NSCharacterSet(charactersInString: "t")

if let _ = phrase.rangeOfCharacterFromSet(charset, options: .CaseInsensitiveSearch, range: nil) {
   print("yes")
}
else {
    print("no")
}

También puede incluir rango:

let phrase = "Test case"
let charset = CharacterSet(charactersIn: "t")

if let _ = phrase.rangeOfCharacter(from: charset, options: .caseInsensitive, range: phrase.startIndex..<phrase.endIndex)) {
   print("yes") 
}
else {
   print("no")
}

Iteración de cadenas #

let string = "My fantastic string"
var index = string.startIndex

while index != string.endIndex {
    print(string[index])
    index = index.successor()
}

Nota: endIndex está después del final de la cadena (es decir, string[string.endIndex] es un error, pero string[string.startIndex] está bien). Además, en una cadena vacía (""), string.startIndex == string.endIndex es true. Asegúrese de buscar cadenas vacías, ya que no puede llamar a startIndex.successor() en una cadena vacía.

En Swift 3, los índices de cadena ya no tienen successor(), predecessor(), advancedBy(_:), advancedBy(_:limit:) o distanceTo(_:).

En su lugar, esas operaciones se trasladan a la colección, que ahora es responsable de incrementar y disminuir sus índices.

Los métodos disponibles son .index(después:), .index(antes:) y .index(_:, offsetBy:).

let string = "My fantastic string"
var currentIndex = string.startIndex

while currentIndex != string.endIndex {
    print(string[currentIndex])
    currentIndex = string.index(after: currentIndex)
}

Nota: estamos usando currentIndex como nombre de variable para evitar confusiones con el método .index.

Y, por ejemplo, si quieres ir al otro lado:

var index:String.Index? = string.endIndex.predecessor()

while index != nil {
    print(string[index!])
    if index != string.startIndex {
        index = index.predecessor()
    }
    else {
        index = nil
    }
}

(O simplemente puede invertir la cadena primero, pero si no necesita recorrer toda la cadena, probablemente prefiera un método como este)

var currentIndex: String.Index? = string.index(before: string.endIndex)

while currentIndex != nil {
    print(string[currentIndex!])
    if currentIndex != string.startIndex {
        currentIndex = string.index(before: currentIndex!)
    }
    else {
        currentIndex = nil
    }
}

Tenga en cuenta que Index es un tipo de objeto y no Int. No puede acceder a un carácter de cadena de la siguiente manera:

let string = "My string"
string[2] // can't do this
string.characters[2] // and also can't do this

Pero puede obtener un índice específico de la siguiente manera:

index = string.startIndex.advanceBy(2)

currentIndex = string.index(string.startIndex, offsetBy: 2)

Y puede ir hacia atrás así:

index = string.endIndex.advancedBy(-2)

currentIndex = string.index(string.endIndex, offsetBy: -2)

Si puede exceder los límites de la cadena, o si desea especificar un límite, puede usar:

index = string.startIndex.advanceBy(20, limit: string.endIndex)

currentIndex = string.index(string.startIndex, offsetBy: 20, limitedBy: string.endIndex)

Alternativamente, uno puede simplemente iterar a través de los caracteres en una cadena, pero esto podría ser menos útil según el contexto:

for c in string.characters {
    print(c)
}

##Unicode

Configuración de valores #

Usando Unicode directamente

var str: String = "I want to visit 北京, Москва, मुंबई, القاهرة, and 서울시. 😊"
var character: Character = "🌍"

Usando valores hexadecimales

var str: String = "\u{61}\u{5927}\u{1F34E}\u{3C0}" // a大🍎π
var character: Character = "\u{65}\u{301}" // é = "e" + accent mark

Tenga en cuenta que el ‘Carácter’ de Swift puede estar compuesto por múltiples puntos de código Unicode, pero parece ser un solo carácter. Esto se llama un Clúster de Grafema Extendido.

Conversiones #

Cadena –> Hexadecimal

// Accesses views of different Unicode encodings of `str`
str.utf8
str.utf16
str.unicodeScalars // UTF-32

Hex –> Cadena

let value0: UInt8 = 0x61
let value1: UInt16 = 0x5927
let value2: UInt32 = 0x1F34E

let string0 = String(UnicodeScalar(value0)) // a
let string1 = String(UnicodeScalar(value1)) // 大
let string2 = String(UnicodeScalar(value2)) // 🍎

// convert hex array to String
let myHexArray = [0x43, 0x61, 0x74, 0x203C, 0x1F431] // an Int array
var myString = ""
for hexValue in myHexArray {
    myString.append(UnicodeScalar(hexValue))
}
print(myString) // Cat‼🐱

Tenga en cuenta que para UTF-8 y UTF-16 la conversión no siempre es tan fácil porque cosas como emoji no se pueden codificar con un solo valor UTF-16. Se necesita un par sustituto.

Dividir una cadena en una matriz #

En Swift, puede separar fácilmente una cadena en una matriz al cortarla en un carácter determinado:

let startDate = "23:51"

let startDateAsArray = startDate.components(separatedBy: ":") // ["23", "51"]`

let startDate = "23:51"

let startArray = startDate.componentsSeparatedByString(":") // ["23", "51"]`

O cuando el separador no está presente:

let myText = "MyText"

let myTextArray = myText.components(separatedBy: " ") // myTextArray is ["MyText"]

let myText = "MyText"

let myTextArray = myText.componentsSeparatedByString(" ") // myTextArray is ["MyText"]

Cadenas en mayúsculas y minúsculas #

Para convertir todos los caracteres de una cadena en mayúsculas o minúsculas:

let text = "AaBbCc"
let uppercase = text.uppercaseString // "AABBCC"
let lowercase = text.lowercaseString // "aabbcc"

let text = "AaBbCc"
let uppercase = text.uppercased() // "AABBCC"
let lowercase = text.lowercased() // "aabbcc"

Cadenas de formato #

Ceros a la izquierda #

let number: Int = 7
let str1 = String(format: "%03d", number) // 007
let str2 = String(format: "%05d", number) // 00007

Números después de decimales #

let number: Float = 3.14159
let str1 = String(format: "%.2f", number) // 3.14
let str2 = String(format: "%.4f", number) // 3.1416 (rounded)

De decimal a hexadecimal #

let number: Int = 13627
let str1 = String(format: "%2X", number) // 353B
let str2 = String(format: "%2x", number) // 353b (notice the lowercase b)

Alternativamente, uno podría usar un inicializador especializado que hace lo mismo:

let number: Int = 13627
let str1 = String(number, radix: 16, uppercase: true) //353B
let str2 = String(number, radix: 16) // 353b

Decimal a un número con raíz arbitraria #

let number: Int = 13627
let str1 = String(number, radix: 36) // aij

Radix es Int en [2, 36].

Convirtiendo una cadena Swift a un tipo de número #

Int("123") // Returns 123 of Int type
Int("abcd") // Returns nil
Int("10") // Returns 10 of Int type
Int("10", radix: 2) // Returns 2 of Int type
Double("1.5") // Returns 1.5 of Double type
Double("abcd") // Returns nil

Tenga en cuenta que hacer esto devuelve un valor Opcional, que debería estar [desenvuelto](https://www.wikiod.com/es/docs/ /swift/247/opcionales/913/unwrapping-an-opcional#t=201607230855233123287) antes de utilizarlo.

Convertir cadena hacia y desde datos / NSData #

Para convertir String hacia y desde Data / NSData, necesitamos codificar esta cadena con una codificación específica. El más famoso es UTF-8, que es una representación de 8 bits de caracteres Unicode, adecuada para la transmisión o el almacenamiento mediante sistemas basados ​​en ASCII. Aquí hay una lista de todas las [Codificaciones de cadenas] (https://developer.apple.com/library/mac/documentation/Cocoa/Reference/Foundation/Classes/NSString_Class/#//apple_ref/c/tdef/NSStringEncoding )

Cadena a Datos/NSData

let data = string.data(using: .utf8)

let data = string.dataUsingEncoding(NSUTF8StringEncoding)

Data/NSData a String

let string = String(data: data, encoding: .utf8)

let string = String(data: data, encoding: NSUTF8StringEncoding)

Cuenta las ocurrencias de un carácter en una cadena #

Dada una ‘Cadena’ y un ‘Carácter’

let text = "Hello World"
let char: Character = "o"

Podemos contar el número de veces que el ‘Carácter’ aparece en la ‘Cadena’ usando

let sensitiveCount = text.characters.filter { $0 == char }.count // case-sensitive
let insensitiveCount = text.lowercaseString.characters.filter { $0 == Character(String(char).lowercaseString) } // case-insensitive

Elimina los espacios en blanco iniciales y finales y la nueva línea #

let someString = "  Swift Language  \n"
let trimmedString = someString.stringByTrimmingCharactersInSet(NSCharacterSet.whitespaceAndNewlineCharacterSet())
// "Swift Language"

El método stringByTrimmingCharactersInSet devuelve una nueva cadena creada al eliminar de ambos extremos de los caracteres String contenidos en un conjunto de caracteres determinado.

También podemos eliminar solo espacios en blanco o nueva línea.

Eliminando solo espacios en blanco:

let trimmedWhiteSpace = someString.stringByTrimmingCharactersInSet(NSCharacterSet.whitespaceCharacterSet())
// "Swift Language  \n"

Eliminando solo nueva línea:

let trimmedNewLine = someString.stringByTrimmingCharactersInSet(NSCharacterSet.newlineCharacterSet())
// "  Swift Language  "

let someString = "  Swift Language  \n"

let trimmedString = someString.trimmingCharacters(in: .whitespacesAndNewlines)
// "Swift Language"

let trimmedWhiteSpace = someString.trimmingCharacters(in: .whitespaces)
// "Swift Language  \n"

let trimmedNewLine = someString.trimmingCharacters(in: .newlines)
// "  Swift Language  "

Nota: todos estos métodos pertenecen a Foundation. Use import Foundation si Foundation no se ha importado ya a través de otras bibliotecas como Cocoa o UIKit.
