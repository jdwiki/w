Array es un tipo de colección ordenado de acceso aleatorio.

Las matrices son uno de los tipos de datos más utilizados en una aplicación. Usamos el tipo Array para contener elementos de un solo tipo, el tipo Element de la matriz. Una matriz puede almacenar cualquier tipo de elementos, desde números enteros hasta cadenas y clases.

Sintaxis #

  • Matriz<Elemento> // El tipo de una matriz con elementos de tipo Elemento
  • [Elemento] // Azúcar sintáctico para el tipo de una matriz con elementos de tipo Elemento
  • [elemento0, elemento1, elemento2, … elementoN] // Un literal de matriz
  • [Element]() // Crea una nueva matriz vacía de tipo [Element]
  • Array(count:repeatedValue:) // Crea una matriz de elementos count, cada uno inicializado a Valor repetido
  • Array(_:) // Crea una matriz a partir de una secuencia arbitraria

Las matrices son una colección ordenada de valores. Los valores pueden repetirse pero deben ser del mismo tipo.

Conceptos básicos de matrices #

Array es un tipo de colección ordenada en la biblioteca estándar de Swift. Proporciona acceso aleatorio O(1) y reasignación dinámica. Array es un tipo genérico, por lo que el tipo de valores que contiene se conocen en tiempo de compilación.

Como Array es un tipo de valor, su mutabilidad se define por si se anota como var (mutable) o let (inmutable).

El tipo [Int] (es decir, una matriz que contiene Ints) es azúcar sintáctico para Array<T>.

Obtenga más información sobre matrices en [El lenguaje de programación Swift](https://developer.apple.com/library/ios/documentation/Swift/Conceptual/Swift_Programming_Language/CollectionTypes.html#//apple_ref/doc/uid/TP40014097-CH8- ID107).

matrices vacías #

Las siguientes tres declaraciones son equivalentes:

// A mutable array of Strings, initially empty.

var arrayOfStrings: [String] = []      // type annotation + array literal
var arrayOfStrings = [String]()        // invoking the [String] initializer
var arrayOfStrings = Array<String>()   // without syntactic sugar

Array de literales #

Un literal de matriz se escribe con corchetes que rodean los elementos separados por comas:

// Create an immutable array of type [Int] containing 2, 4, and 7
let arrayOfInts = [2, 4, 7]

El compilador generalmente puede inferir el tipo de una matriz en función de los elementos en el literal, pero las anotaciones de tipo explícitas pueden anular el valor predeterminado:

let arrayOfUInt8s: [UInt8] = [2, 4, 7]   // type annotation on the variable
let arrayOfUInt8s = [2, 4, 7] as [UInt8] // type annotation on the initializer expression
let arrayOfUInt8s = [2 as UInt8, 4, 7]   // explicit for one element, inferred for the others

Arreglos con valores repetidos #

// An immutable array of type [String], containing ["Example", "Example", "Example"]
let arrayOfStrings = Array(repeating: "Example",count: 3)

Creando arreglos a partir de otras secuencias #

let dictionary = ["foo" : 4, "bar" : 6]

// An immutable array of type [(String, Int)], containing [("bar", 6), ("foo", 4)]
let arrayOfKeyValuePairs = Array(dictionary)

Arreglos multidimensionales #

En Swift, una matriz multidimensional se crea anidando matrices: una matriz bidimensional de Int es [[Int]] (o Array<Array<Int>>).

let array2x3 = [
    [1, 2, 3],
    [4, 5, 6]
]
// array2x3[0][1] is 2, and array2x3[1][2] is 6.

Para crear una matriz multidimensional de valores repetidos, use llamadas anidadas del inicializador de matriz:

var array3x4x5 = Array(repeating: Array(repeating: Array(repeating: 0,count: 5),count: 4),count: 3)

3: https://www.wikiod.com/es/swift/estructuras#Las estructuras son tipos de valor

Extrayendo valores de un tipo dado de un Array con flatMap(_:) #

La matriz cosas contiene valores de tipo Cualquiera.

let things: [Any] = [1, "Hello", 2, true, false, "World", 3]

Podemos extraer valores de un tipo dado y crear una nueva matriz de ese tipo específico. Digamos que queremos extraer todos los Int(s) y ponerlos en una matriz Int de forma segura.

let numbers = things.flatMap { $0 as? Int }

Ahora numbers se define como [Int]. La función flatMap descarta todos los elementos nil y el resultado contiene solo los siguientes valores:

[1, 2, 3]

Aplanando el resultado de una transformación Array con flatMap(_:) #

Además de poder crear una matriz filtrando nil de los elementos transformados de una secuencia, también hay una versión de flatMap(_:) que espera la transformación cierre para devolver una secuencia S.

extension SequenceType {
    public func flatMap<S : SequenceType>(transform: (Self.Generator.Element) throws -> S) rethrows -> [S.Generator.Element]
}

Cada secuencia de la transformación se concatenará, dando como resultado una matriz que contiene los elementos combinados de cada secuencia: [S.Generator.Element].

Combinando los caracteres en una matriz de cadenas #

Por ejemplo, podemos usarlo para tomar una matriz de cadenas principales y combinar sus caracteres en una sola matriz:

let primes = ["2", "3", "5", "7", "11", "13", "17", "19"]
let allCharacters = primes.flatMap { $0.characters }
// => "["2", "3", "5", "7", "1", "1", "1", "3", "1", "7", "1", "9"]"

Desglosando el ejemplo anterior:

  1. primos es un [String] (Como una matriz es una secuencia, podemos llamar a flatMap(_:)).
  2. El cierre de la transformación toma uno de los elementos de primos, un String (Array<String>.Generator.Element).
  3. El cierre luego devuelve una secuencia de tipo String.CharacterView.
  4. El resultado es una matriz que contiene los elementos combinados de todas las secuencias de cada una de las llamadas de cierre de transformación: [String.CharacterView.Generator.Element].

Aplanando una matriz multidimensional #

Como flatMap(_:) concatenará las secuencias devueltas de las llamadas de cierre de transformación, se puede usar para aplanar una matriz multidimensional, como una matriz 2D en una matriz 1D, una matriz 3D en una matriz 2D, etc.

Esto se puede hacer simplemente devolviendo el elemento dado $0 (una matriz anidada) en el cierre:

// A 2D array of type [[Int]]
let array2D = [[1, 3], [4], [6, 8, 10], [11]]

// A 1D array of type [Int]
let flattenedArray = array2D.flatMap { $0 }

print(flattenedArray) // [1, 3, 4, 6, 8, 10, 11]

Combinando los elementos de un Array con reduce(_:combine:) #

reduce(_:combine:) se puede usar para combinar los elementos de una secuencia en un solo valor. Se necesita un valor inicial para el resultado, así como un cierre para aplicar a cada elemento, lo que devolverá el nuevo valor acumulado.

Por ejemplo, podemos usarlo para sumar una matriz de números:

let numbers = [2, 5, 7, 8, 10, 4]

let sum = numbers.reduce(0) {accumulator, element in
    return accumulator + element
}

print(sum) // 36

Estamos pasando 0 al valor inicial, ya que ese es el valor inicial lógico para una suma. Si pasáramos un valor de N, la suma resultante sería N + 36. El cierre pasado a reduce tiene dos argumentos. accumulator es el valor acumulado actual, al que se le asigna el valor que devuelve el cierre en cada iteración. element es el elemento actual en la iteración.

Como en este ejemplo, estamos pasando un cierre (Int, Int) -> Int a reduce, que simplemente genera la suma de las dos entradas; en realidad podemos pasar el operador + directamente, como los operadores son funciones en Swift:

let sum = numbers.reduce(0, combine: +)

Filtrando cero de una transformación de matriz con flatMap(_:) #

Puede usar flatMap(_:) de manera similar a map(_:) para crear una matriz aplicando una transformación a los elementos de una secuencia.

extension SequenceType {
    public func flatMap<T>(@noescape transform: (Self.Generator.Element) throws -> T?) rethrows -> [T]
}

La diferencia con esta versión de flatMap(_:) es que espera que la transformación cierre devuelva un valor Opcional T? para cada uno de los elementos. Luego desenvolverá de forma segura cada uno de estos valores opcionales, filtrando nil, lo que dará como resultado una matriz de [T].

Por ejemplo, puede hacer esto para transformar un [String] en un [Int] utilizando el inicializador fallable String de Int, filtrando cualquier elemento que no se pueda convertir:

let strings = ["1", "foo", "3", "4", "bar", "6"]

let numbersThatCanBeConverted = strings.flatMap { Int($0) }

print(numbersThatCanBeConverted) // [1, 3, 4, 6]

También puede usar la capacidad de flatMap(_:) para filtrar nil para simplemente convertir una matriz de opcionales en una matriz de no opcionales:

let optionalNumbers : [Int?] = [nil, 1, nil, 2, nil, 3]

let numbers = optionalNumbers.flatMap { $0 }

print(numbers) // [1, 2, 3]

2: https://www.wikiod.com/es/swift/arreglos#Transformando los elementos de un Array con map(:) 3: https://www.wikiod.com/es/swift/cierres 4: https://www.wikiod.com/es/swift/opcionales 5: http://swiftdoc.org/v2.2/type/Int/#init_radix

Aplanando perezosamente una matriz multidimensional con flatten() #

Podemos usar flatten() para perezosamente reducir el anidamiento de una secuencia multidimensional.

Por ejemplo, aplanar perezosamente una matriz 2D en una matriz 1D:

// A 2D array of type [[Int]]
let array2D = [[1, 3], [4], [6, 8, 10], [11]]

// A FlattenBidirectionalCollection<[[Int]]>
let lazilyFlattenedArray = array2D.flatten()

print(lazilyFlattenedArray.contains(4)) // true

En el ejemplo anterior, flatten() devolverá una FlattenBidireccionalCollection, que aplicará perezosamente el aplanamiento de la matriz. Por lo tanto, contains(_:) solo requerirá que se aplanen las dos primeras matrices anidadas de array2D, ya que provocará un cortocircuito al encontrar el elemento deseado.

Subíndice de una matriz con un rango #

Se puede extraer una serie de elementos consecutivos de un Array usando un Rango.

let words = ["Hey", "Hello", "Bonjour", "Welcome", "Hi", "Hola"]
let range = 2...4
let slice = words[range] // ["Bonjour", "Welcome", "Hi"]

Suscribir un Array con un Range devuelve un ArraySlice. Es una subsecuencia del Array.

En nuestro ejemplo, tenemos un Array of Strings, por lo que recuperamos ArraySlice<String>.

Aunque un ArraySlice se ajusta a CollectionType y se puede usar con sort, filter, etc., su propósito no es para el almacenamiento a largo plazo sino para cálculos transitorios: debe volver a convertirse en un Array tan pronto como He terminado de trabajar con él.

Para esto, usa el inicializador Array():

let result = Array(slice)

Resumiendo en un ejemplo sencillo sin pasos intermedios:

let words = ["Hey", "Hello", "Bonjour", "Welcome", "Hi", "Hola"]
let selectedWords = Array(words[2...4]) // ["Bonjour", "Welcome", "Hi"]

Eliminar elemento de una matriz sin saber su índice #

Generalmente, si queremos eliminar un elemento de una matriz, necesitamos saber su índice para poder eliminarlo fácilmente usando la función remove(at:).

¡Pero qué pasa si no conocemos el índice pero sabemos el valor del elemento que se eliminará!

Entonces, aquí está la extensión simple de una matriz que nos permitirá eliminar un elemento de la matriz fácilmente sin conocer su índice:

Swift3 #

extension Array where Element: Equatable {

    mutating func remove(_ element: Element) {
        _ = index(of: element).flatMap {
            self.remove(at: $0)
        }
    }
}

p.ej.

    var array = ["abc", "lmn", "pqr", "stu", "xyz"]
    array.remove("lmn")
    print("\(array)")    //["abc", "pqr", "stu", "xyz"]
    
    array.remove("nonexistent")
    print("\(array)")    //["abc", "pqr", "stu", "xyz"]
    //if provided element value is not present, then it will do nothing!

También si, por error, hicimos algo como esto: array.remove(25) es decir, proporcionamos valor con un tipo de datos diferente, el compilador arrojará un error mencionando- no se puede convertir el valor al tipo de argumento esperado

Filtrando una matriz #

Puede usar el método filter(_:) en SequenceType para crear una nueva matriz que contenga los elementos de la secuencia que satisfacen un predicado dado, que se puede proporcionar como un cierre.

Por ejemplo, filtrar números pares de un [Int]:

let numbers = [22, 41, 23, 30]

let evenNumbers = numbers.filter { $0 % 2 == 0 }

print(evenNumbers)  // [22, 30]

Filtrando una [Persona], donde su edad es menor de 30:

struct Person {
    var age : Int
}

let people = [Person(age: 22), Person(age: 41), Person(age: 23), Person(age: 30)]

let peopleYoungerThan30 = people.filter { $0.age < 30 }

print(peopleYoungerThan30) // [Person(age: 22), Person(age: 23)]

Ordenar una matriz de cadenas #

La forma más sencilla es usar sorted():

let words = ["Hello", "Bonjour", "Salute", "Ahola"]
let sortedWords = words.sorted()
print(sortedWords) // ["Ahola", "Bonjour", "Hello", "Salute"]

o clasificar()

var mutableWords = ["Hello", "Bonjour", "Salute", "Ahola"]
mutableWords.sort()
print(mutableWords) // ["Ahola", "Bonjour", "Hello", "Salute"]

Puede pasar un cierre como argumento para ordenar:

let words = ["Hello", "Bonjour", "Salute", "Ahola"]
let sortedWords = words.sorted(isOrderedBefore: { $0 > $1 })
print(sortedWords) // ["Salute", "Hello", "Bonjour", "Ahola"]

Sintaxis alternativa con cierre final:

let words = ["Hello", "Bonjour", "Salute", "Ahola"]
let sortedWords = words.sorted() { $0 > $1 }
print(sortedWords) // ["Salute", "Hello", "Bonjour", "Ahola"]

Pero habrá resultados inesperados si los elementos de la matriz no son consistentes:

let words = ["Hello", "bonjour", "Salute", "ahola"]
let unexpected = words.sorted()
print(unexpected) // ["Hello", "Salute", "ahola", "bonjour"]

Para abordar este problema, ordene una versión en minúsculas de los elementos:

let words = ["Hello", "bonjour", "Salute", "ahola"]
let sortedWords = words.sorted { $0.lowercased() < $1.lowercased() }
print(sortedWords) // ["ahola", "bonjour", "Hello", "Salute"]

O importar Foundation y usar los métodos de comparación de NSString como caseInsensitiveCompare:

let words = ["Hello", "bonjour", "Salute", "ahola"]
let sortedWords = words.sorted { $0.caseInsensitiveCompare($1) == .orderedAscending }
print(sortedWords) // ["ahola", "bonjour", "Hello", "Salute"]

Alternativamente, use localizedCaseInsensitiveCompare, que puede administrar signos diacríticos.

Para ordenar correctamente las cadenas por el valor numérico que contienen, use compare con la opción .numeric:

let files = ["File-42.txt", "File-01.txt", "File-5.txt", "File-007.txt", "File-10.txt"]
let sortedFiles = files.sorted() { $0.compare($1, options: .numeric) == .orderedAscending }
print(sortedFiles) // ["File-01.txt", "File-5.txt", "File-007.txt", "File-10.txt", "File-42.txt"]

Acceder a los índices de forma segura #

Al agregar la siguiente extensión a la matriz, se puede acceder a los índices sin saber si el índice está dentro de los límites.

extension Array {
    subscript (safe index: Int) -> Element? {
        return indices ~= index ? self[index] : nil
    }
}

ejemplo:

if let thirdValue = array[safe: 2] {
    print(thirdValue)
}

Métodos útiles #

Determinar si una matriz está vacía o devolver su tamaño

var exampleArray = [1,2,3,4,5]
exampleArray.isEmpty //false
exampleArray.count //5

Invertir una matriz Nota: el resultado no se realiza en la matriz a la que se llama el método y debe colocarse en su propia variable.

exampleArray = exampleArray.reverse()
//exampleArray = [9, 8, 7, 6, 5, 3, 2]

Ordenar una matriz #

var array = [3, 2, 1]

Creando una nueva matriz ordenada #

Como Array se ajusta a SequenceType, podemos generar una nueva matriz de los elementos ordenados utilizando un método de clasificación integrado.

En Swift 2, esto se hace con el método sort().

let sorted = array.sort()  // [1, 2, 3]

A partir de Swift 3, se le cambió el nombre a sorted().

let sorted = array.sorted()  // [1, 2, 3]

Ordenar una matriz existente en su lugar #

Como Array se ajusta a MutableCollectionType, podemos ordenar sus elementos en su lugar.

En Swift 2, esto se hace usando el método sortInPlace().

array.sortInPlace() // [1, 2, 3]

A partir de Swift 3, se le cambió el nombre a sort().

array.sort() // [1, 2, 3]

Nota: Para utilizar los métodos anteriores, los elementos deben cumplir con el protocolo Comparable.

Ordenar una matriz con un orden personalizado #

También puede ordenar una matriz usando un cierre para definir si un elemento debe ordenarse antes que otro, lo que no se limita a las matrices donde los elementos deben ser ‘Comparables’. Por ejemplo, no tiene sentido que un ‘Punto de referencia’ sea ‘Comparable’, pero aún puede ordenar una serie de puntos de referencia por altura o nombre.

struct Landmark {
    let name : String
    let metersTall : Int
}

var landmarks = [Landmark(name: "Empire State Building", metersTall: 443),
                 Landmark(name: "Eifell Tower", metersTall: 300),
                 Landmark(name: "The Shard", metersTall: 310)]

// sort landmarks by height (ascending)
landmarks.sortInPlace {$0.metersTall < $1.metersTall}

print(landmarks) // [Landmark(name: "Eifell Tower", metersTall: 300), Landmark(name: "The Shard", metersTall: 310), Landmark(name: "Empire State Building", metersTall: 443)]

// create new array of landmarks sorted by name
let alphabeticalLandmarks = landmarks.sort {$0.name < $1.name}

print(alphabeticalLandmarks) // [Landmark(name: "Eifell Tower", metersTall: 300), Landmark(name: "Empire State Building", metersTall: 443), Landmark(name: "The Shard", metersTall: 310)]

// sort landmarks by height (ascending)
landmarks.sort {$0.metersTall < $1.metersTall}

// create new array of landmarks sorted by name
let alphabeticalLandmarks = landmarks.sorted {$0.name < $1.name}

Nota: la comparación de cadenas puede producir resultados inesperados si las cadenas son inconsistentes, consulte [Ordenar una matriz de cadenas][10].

[10]: https://www.wikiod.com/es/swift/arreglos#Ordenar una matriz de cadenas

Transformando los elementos de un Array con map(_:) #

Como Array se ajusta a SequenceType, podemos usar map(_:) para transformar una matriz de A en una matriz de B usando un cierre de tipo (A) throws -> B.

Por ejemplo, podríamos usarlo para transformar un arreglo de Int en un arreglo de String así:

let numbers = [1, 2, 3, 4, 5]
let words = numbers.map { String($0) }
print(words) // ["1", "2", "3", "4", "5"]

map(_:) iterará a través de la matriz, aplicando el cierre dado a cada elemento. El resultado de ese cierre se usará para llenar una nueva matriz con los elementos transformados.

Dado que String tiene un inicializador que recibe un Int, también podemos usar esta sintaxis más clara:

let words = numbers.map(String.init)

Una transformación map(_:) no necesita cambiar el tipo de matriz; por ejemplo, también podría usarse para multiplicar una matriz de Ints por dos:

let numbers = [1, 2, 3, 4, 5]
let numbersTimes2 = numbers.map {$0 * 2}
print(numbersTimes2) // [2, 4, 6, 8, 10]

Encontrar el elemento mínimo o máximo de un Array #

Puede usar los métodos minElement() y maxElement() para encontrar el elemento mínimo o máximo en una secuencia determinada. Por ejemplo, con una matriz de números:

let numbers = [2, 6, 1, 25, 13, 7, 9]

let minimumNumber = numbers.minElement() // Optional(1)
let maximumNumber = numbers.maxElement() // Optional(25)

A partir de Swift 3, se cambió el nombre de los métodos a min() y max() respectivamente:

let minimumNumber = numbers.min() // Optional(1)
let maximumNumber = numbers.max() // Optional(25)

Los valores devueltos por estos métodos son Opcional para reflejar el hecho de que la matriz podría estar vacía; si lo está, se devolverá nil.

Nota: Los métodos anteriores requieren que los elementos se ajusten al protocolo Comparable.

Encontrar el elemento mínimo o máximo con un pedido personalizado #

También puede usar los métodos anteriores con un cierre personalizado, que define si un elemento debe ordenarse antes que otro, lo que le permite encontrar el elemento mínimo o máximo en una matriz donde los elementos no son necesariamente ‘Comparables’.

Por ejemplo, con una matriz de vectores:

struct Vector2 {
    let dx : Double
    let dy : Double
    
    var magnitude : Double {return sqrt(dx*dx+dy*dy)}
}

let vectors = [Vector2(dx: 3, dy: 2), Vector2(dx: 1, dy: 1), Vector2(dx: 2, dy: 2)]

// Vector2(dx: 1.0, dy: 1.0)
let lowestMagnitudeVec2 = vectors.minElement { $0.magnitude < $1.magnitude } 

// Vector2(dx: 3.0, dy: 2.0)
let highestMagnitudeVec2 = vectors.maxElement { $0.magnitude < $1.magnitude } 

let lowestMagnitudeVec2 = vectors.min { $0.magnitude < $1.magnitude }
let highestMagnitudeVec2 = vectors.max { $0.magnitude < $1.magnitude }

Semántica del valor #

Copiar una matriz copiará todos los elementos dentro de la matriz original.

Cambiar la nueva matriz no cambiará la matriz original.

var originalArray = ["Swift", "is", "great!"]
var newArray = originalArray
newArray[2] = "awesome!"
//originalArray = ["Swift", "is", "great!"]
//newArray = ["Swift", "is", "awesome!"]

Los arreglos copiados compartirán el mismo espacio en la memoria que el original hasta que se cambien. Como resultado de esto, hay un impacto en el rendimiento cuando la matriz copiada recibe su propio espacio en la memoria cuando se cambia por primera vez.

Acceso a los valores de matriz #

Los siguientes ejemplos usarán esta matriz para demostrar el acceso a valores

var exampleArray:[Int] = [1,2,3,4,5]
//exampleArray = [1, 2, 3, 4, 5]

Para acceder a un valor en un índice conocido, use la siguiente sintaxis:

let exampleOne = exampleArray[2]
//exampleOne = 3

Nota: El valor en índice dos es el tercer valor en el Array. Arrays usa un índice basado en cero, lo que significa que el primer elemento en Array está en el índice 0.

let value0 = exampleArray[0]
let value1 = exampleArray[1]
let value2 = exampleArray[2]
let value3 = exampleArray[3]
let value4 = exampleArray[4]
//value0 = 1
//value1 = 2 
//value2 = 3
//value3 = 4
//value4 = 5

Acceda a un subconjunto de un Array usando el filtro:

var filteredArray = exampleArray.filter({ $0 < 4 })
//filteredArray = [1, 2, 3]

Los filtros pueden tener condiciones complejas como filtrar solo números pares:

var evenArray = exampleArray.filter({ $0 % 2 == 0 })
//evenArray = [2, 4]

También es posible devolver el índice de un valor dado, devolviendo nil si no se encontró el valor.

exampleArray.indexOf(3) // Optional(2)

Hay métodos para el primer, último, máximo o mínimo valor en un Array. Estos métodos devolverán nil si Array está vacío.

exampleArray.first // Optional(1)
exampleArray.last // Optional(5)
exampleArray.maxElement() // Optional(5)
exampleArray.minElement() // Optional(1)

Modificando valores en una matriz #

Hay varias formas de agregar valores a una matriz

var exampleArray = [1,2,3,4,5]

var sixOnwards = [7,8,9,10]

y eliminar valores de una matriz

Agrupación de valores de matriz #

Si tenemos una estructura como esta

struct Box {
    let name: String
    let thingsInside: Int
}

y una matriz de Box(es)

let boxes = [
    Box(name: "Box 0", thingsInside: 1),
    Box(name: "Box 1", thingsInside: 2),
    Box(name: "Box 2", thingsInside: 3),
    Box(name: "Box 3", thingsInside: 1),
    Box(name: "Box 4", thingsInside: 2),
    Box(name: "Box 5", thingsInside: 3),
    Box(name: "Box 6", thingsInside: 1)
]

podemos agrupar las cajas por la propiedad thingsInside para obtener un Diccionario donde la clave es el número de cosas y el valor es una matriz de cajas.

let grouped = boxes.reduce([Int:[Box]]()) { (res, box) -> [Int:[Box]] in
    var res = res
    res[box.thingsInside] = (res[box.thingsInside] ?? []) + [box]
    return res
}

Ahora agrupado es un [Int:[Box]] y tiene el siguiente contenido

[
    2: [Box(name: "Box 1", thingsInside: 2), Box(name: "Box 4", thingsInside: 2)], 
    3: [Box(name: "Box 2", thingsInside: 3), Box(name: "Box 5", thingsInside: 3)],
    1: [Box(name: "Box 0", thingsInside: 1), Box(name: "Box 3", thingsInside: 1), Box(name: "Box 6", thingsInside: 1)]
]

Comparando 2 arreglos con zip #

La función zip acepta 2 parámetros de tipo SequenceType y devuelve un Zip2Sequence donde cada elemento contiene un valor de la primera secuencia y uno de la segunda secuencia.

Ejemplo

let nums = [1, 2, 3]
let animals = ["Dog", "Cat", "Tiger"]
let numsAndAnimals = zip(nums, animals)

nomsAndAnimals ahora contiene los siguientes valores

secuencia1 secuencia1
1 "Perro"
2 "Gato"
3 "Tigre"

Esto es útil cuando desea realizar algún tipo de comparación entre el n-ésimo elemento de cada Array.

Ejemplo

Dadas 2 matrices de Int(s)

let list0 = [0, 2, 4]
let list1 = [0, 4, 8]

desea verificar si cada valor en list1 es el doble del valor relacionado en list0.

let list1HasDoubleOfList0 = !zip(list0, list1).filter { $0 != (2 * $1)}.isEmpty
