No puede definir más de tres formatos condicionales para un rango. Utilice el método Modificar para modificar un formato condicional existente o utilice el método Eliminar para eliminar un formato existente antes de agregar uno nuevo.

FormatConditions.Add #

Sintaxis: #

FormatConditions.Add(Type, Operator, Formula1, Formula2)

Parámetros: #

Nombre Obligatorio / Opcional Tipo de datos
Tipo Requerido XlFormatConditionType
Operador Opcional Variante
Fórmula1 Opcional Variante
Fórmula2 Opcional Variante

Enumeración XlFormatConditionType: #

Nombre Descripción
xlAboveAverageCondition Condición superior a la media
xlBlanksCondición Condición de espacios en blanco
xlValorCelda Valor de celda
xlColorScale escala de colores
xlBarra de datos Barra de datos
xlErrorsCondición Condición de errores
xlExpresión Expresión
XlIconSet conjunto de iconos
xlNoBlanksCondición Sin condición de espacios en blanco
xlNoErrorsCondition Sin condición de errores
xlTextString cadena de texto
xlTimePeriod Periodo de tiempo
xlTop10 10 valores principales
xlValoresÚnicos Valores únicos

Formato por valor de celda: #

With Range("A1").FormatConditions.Add(xlCellValue, xlGreater, "=100")
    With .Font
        .Bold = True
        .ColorIndex = 3
     End With
End With

Operadores: #

Nombre
xlEntre
xlIgual
xlMayor
xlMayorIgual
xlMenos
xlMenosIgual
xlNoEntre
xlNoigual
Si Tipo es xlExpresión, el argumento Operador se ignora.

El formato por texto contiene: #

With Range("a1:a10").FormatConditions.Add(xlTextString, TextOperator:=xlContains, String:="egg")
    With .Font
        .Bold = True
        .ColorIndex = 3
    End With
End With

Operadores: #

Nombre Descripción
xlEmpiezaCon Comienza con un valor especificado.
xlContiene Contiene un valor especificado.
xlNoContiene No contiene el valor especificado.
xl termina con Termina con el valor especificado

Formateo por período de tiempo #

With Range("a1:a10").FormatConditions.Add(xlTimePeriod, DateOperator:=xlToday)

Eliminar formato condicional #

Eliminar todo el formato condicional en el rango: #

Range("A1:A10").FormatConditions.Delete

Eliminar todo el formato condicional en la hoja de trabajo: #

Cells.FormatConditions.Delete

FormatConditions.AddUniqueValues #

Resaltar valores duplicados #

With Range("E1:E100").FormatConditions.AddUniqueValues
   .DupeUnique = xlDuplicate
   With .Font
       .Bold = True
       .ColorIndex = 3
   End With
End With

Resaltar valores únicos #

With Range("E1:E100").FormatConditions.AddUniqueValues
   With .Font
       .Bold = True
       .ColorIndex = 3
   End With
End With

FormatConditions.AddTop10 #

Destacando los 5 valores principales #

With Range("E1:E100").FormatConditions.AddTop10
    .TopBottom = xlTop10Top
    .Rank = 5
    .Percent = False
    With .Font
        .Bold = True
        .ColorIndex = 3
    End With
End With

FormatConditions.AddAboveAverage #

With Range("E1:E100").FormatConditions.AddAboveAverage
    .AboveBelow = xlAboveAverage
    With .Font
        .Bold = True
        .ColorIndex = 3
    End With
End With

Operadores: #

Nombre Descripción
XlPor encima de la media Por encima del promedio
xlpor encima de la desviación estándar Por encima de la desviación estándar
XlPor debajo del promedio Por debajo del promedio
XlPor debajo de la desviación estándar Por debajo de la desviación estándar
XlEqualAboveAverage Igual por encima del promedio
XlEqualBelowPromedio Igual por debajo del promedio

FormatConditions.AddIconSetCondition #

Range("a1:a10").FormatConditions.AddIconSetCondition
With Selection.FormatConditions(1)
    .ReverseOrder = False
    .ShowIconOnly = False
    .IconSet = ActiveWorkbook.IconSets(xl3Arrows)
End With

With Selection.FormatConditions(1).IconCriteria(2)
    .Type = xlConditionValuePercent
    .Value = 33
    .Operator = 7
End With

With Selection.FormatConditions(1).IconCriteria(3)
    .Type = xlConditionValuePercent
    .Value = 67
    .Operator = 7
End With
