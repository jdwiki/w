Puntas:

  • Recuerda revisar siempre el archivo /usr/ports/UPDATING antes de actualizar. Puede haber algunos cambios significativos en los programas que usa o en su configuración que romperán su configuración actual.

Creación e instalación de software #

Si ha encontrado su software en el árbol de puertos, ahora es el momento de construirlo.

Construcción e instalación simples con configuración manual #

cd /usr/ports/www/apache24
make
make install

Construcción e instalación simples con configuración automática #

cd /usr/ports/www/apache24
make BATCH=yes
make install

Obtener el árbol de puertos #

Complemento de puerto #

portsnap fetch
portsnap extract

actualizando el árbol de puertos con portsnap #

portsnap update

programar trabajo cron para actualizaciones diarias #

0 3 * * * root /usr/sbin/portsnap cron

#SVN

cabeza #

cd /usr/ports
svnlite checkout https://svnweb.freebsd.org/ports/head .

trimestral #

El equipo de FreeBSD Ports congela el árbol de puertos cada 3 meses. Para obtener este árbol de puertos, puede usar ramas de puertos:

cd /usr/ports
svnlite checkout https://svnweb.freebsd.org/ports/branches/2016Q4 .

Tarball (http o ftp) #

cd /usr/ports
fetch http://ftp.freebsd.org/pub/FreeBSD/releases/amd64/11.0-RELEASE/ports.txz
tar xJvf ports.txz

Git #

git clone https://github.com/freebsd/freebsd-ports

Software de búsqueda #

búsqueda por palabra clave #

cd /usr/ports
make search key=apache

búsqueda de nombre #

cd /usr/ports
make search name=apache24

Uso de deportes al aire libre #

El sitio web oficial de puertos de FreeBSD (http://freshports.org/) le brinda una buena manera de encontrar puertos y toda la información relacionada.

Configuración de fuentes de software #

Si desea una configuración personalizada de los puertos, puede configurarla antes de compilarla make config. La configuración de todos los puertos se almacena en /var/db/ports/${CATEGORY_NAME}/options como archivo MAKE.

Configurando www/apache24 #

cd /usr/ports/www/apache24
make config
make
make install

Esta configuración se guardará en /var/db/ports/www_apache24/options.

Embalaje #

Empaquetado manual #

Puede hacer su propio paquete basado en puertos.

cd /usr/ports/www/apache24
make package BATCH=yes

Este comando almacenará su paquete en /usr/ports/packages/All.

Usando polvorín #

poudriere es actualmente el generador de paquetes oficial para FreeBSD.

Instalación del polvorín #

pkg install poudriere
# or
cd /usr/ports/ports-mgmt/poudriere
make
make install

Configuración del polvorín #

La configuración poudriere se almacena en /usr/local/etc/poudriere.conf y /usr/local/etc/poudriere.d

Desplegando cárcel de barril de pólvora #

poudriere jail -c -j myjail

Actualización de la cárcel poudriere #

poudriere jail -u -j myjail

Implementación del árbol de puertos de barril de pólvora #

poudriere ports -c -p myports

Actualización del árbol de puertos poudriere #

poudriere ports -u -p myports

Construcción masiva #

poudriere bulk -j myjail -p myports www/apache24
