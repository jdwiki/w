Programa básico de SFML #

Si todo se configuró correctamente, el siguiente fragmento mostrará una ventana titulada “¡SFML funciona!” con un círculo verde:

#include <SFML/Graphics.hpp>

int main()
{
    sf::RenderWindow window(sf::VideoMode(200, 200), "SFML works!");
    sf::CircleShape shape(100.f);
    shape.setFillColor(sf::Color::Green);

    while (window.isOpen())
    {
        sf::Event event;
        while (window.pollEvent(event))
        {
            if (event.type == sf::Event::Closed)
                window.close();
        }

        window.clear();
        window.draw(shape);
        window.display();
    }

    return 0;
}

Instalación - Linux #

Existen diferentes enfoques para la instalación de SFML en Linux:

  1. Instálelo directamente desde el repositorio de paquetes de su distribución
  2. Obtenga el código fuente, constrúyalo e instálelo
  3. Descargue el SDK precompilado y copie manualmente los archivos

La opción 1 es la preferida; si la versión de SFML que desea instalar está disponible en el repositorio oficial, instálela usando su administrador de paquetes. Por ejemplo, en Debian harías:

sudo apt-get install libsfml-dev

La opción 2 requiere más trabajo: debe asegurarse de que todas las dependencias de SFML, incluidos sus encabezados de desarrollo, estén disponibles, asegúrese de que CMake esté instalado y ejecute manualmente algunos comandos. Esto dará como resultado un paquete que se adapta a su sistema. Si quiere seguir este camino, hay un tutorial dedicado sobre cómo crear SFML usted mismo.

Finalmente, la opción 3 es una buena opción para una instalación rápida si SFML no está disponible como paquete oficial. Descargue el SDK desde la página de descarga, descomprímalo y copie los archivos en su ubicación preferida: ya sea una ruta separada en su carpeta personal (como /home/me/sfml) o una ruta estándar (como /usr/local).

Si ya tenía instalada una versión anterior de SFML, ¡asegúrese de que no entre en conflicto con la nueva versión!

Instalación - Windows #

La forma más común de instalar SFML en Windows es descargar el SDK oficial

A continuación, puede descomprimir el archivo y utilizarlo en el entorno de su elección.

vcpkg #

Aunque todavía está en desarrollo, si usa Visual Studio 2017 o posterior, también puede instalar SFML a través de vcpkg, que se integra con Visual Studio, lo que simplifica enormemente el proceso de instalación:

vcpkg instalar sfml

Instalación - macOS #

En primer lugar, debe descargar el SDK de SFML. Luego, para comenzar a desarrollar aplicaciones SFML, debe instalar los siguientes elementos:

Archivos de encabezado y bibliotecas #

SFML está disponible como dylibs o como marcos. Solo se requiere un tipo de binario, aunque ambos se pueden instalar simultáneamente en el mismo sistema. Recomendamos usar los marcos.

Marcos #

  • Copie el contenido de Frameworks en /Library/Frameworks.

dilib #

  • Copie el contenido de lib en /usr/local/lib y copie el contenido de include en /usr/local/include.

Dependencias de SFML #

SFML depende de algunas bibliotecas externas en Mac OS X. Copie el contenido de extlibs en /Library/Frameworks.

Plantillas Xcode #

Si usa Xcode, se recomienda encarecidamente instalar las plantillas. Copie el directorio SFML de las plantillas a /Library/Developer/Xcode/Templates (cree las carpetas si aún no existen).

Hola mundo en una ventana SFML #

Escribamos un pequeño programa que abrirá una ventana y escribiremos “Hello World” en la pantalla.

#include <SFML\Graphics.hpp>
#include <cassert>

int main() {
    sf::RenderWindow sfmlWin(sf::VideoMode(600, 360), "Hello World SFML Window");
    sf::Font font;
    //You need to pass the font file location
    if (!font.loadFromFile(/*
                           Put the filename that identify the font file you want to load*/"myfont.ttf")) {
        return -1;
    }
    sf::Text message("Hello, World !", font);

    while (sfmlWin.isOpen()) {

        sf::Event e;
        while (sfmlWin.pollEvent(e)) {

            switch (e.type) {
            case sf::Event::EventType::Closed:
                sfmlWin.close();
                break;
            }
        }

        sfmlWin.clear();
        sfmlWin.draw(message);
        sfmlWin.display();
    }
    return 0;
}

Vamos a explicar lo que hicimos allí.

Primero, creamos un objeto sf::Font. Necesitamos este objeto para almacenar los datos de la fuente que usaremos para mostrar el texto. Después de eso, llamamos al método loadFromFile, usado para cargar la fuente en la memoria. Debemos tener en cuenta que SFML no conoce las fuentes de su sistema, por lo que debe proporcionar un nombre de archivo, no un nombre de fuente.

Después de eso, creamos un objeto sf::Text. Llamamos a un constructor de 3 parámetros tomando:

  • La cadena que desea mostrar
  • La fuente que usará el objeto
  • El tamaño del carácter en píxeles, que no pasamos aquí, por lo que se establecerá en el valor predeterminado: 30

Dado que el objeto sf::Text está listo, solo necesitamos dibujarlo en el ciclo principal de sfml, llamando al método de dibujo en el objeto de ventana de representación sfmlWin que creamos antes

Instalación o configuración #

  • Primero, descargue una copia de SFML de la página oficial.
  • Guárdelo en cualquier lugar de su computadora donde se pueda acceder fácilmente.
  • Abrir bloques de código.
  • Vaya a Proyecto->Opciones de compilación->pestaña Configuración del enlazador.
  • Haga clic en el botón Agregar y vaya a la carpeta bin de SFML y seleccione todos los archivos presentes allí.
  • Ahora vaya a la pestaña Directorios de búsqueda y agregue la carpeta ‘incluir’ de SFML.
  • En la misma pestaña, haga clic en la subpestaña Configuración del vinculador y agregue la carpeta ‘bin’.
