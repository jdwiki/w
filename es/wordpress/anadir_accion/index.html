Sintaxis #

  • add_action ($ etiqueta, $ function_to_add)
  • add_action ($etiqueta, $función_a_agregar, $prioridad)
  • add_action ($etiqueta, $función_a_agregar, $prioridad, $args_aceptados)

Parámetros #

Parámetro Detalles
*$etiqueta (cadena) El nombre de la acción a la que se conectará el procedimiento $function_to_add.
$funcion_a_agregar (invocable) La función/procedimiento invocable que desea llamar.
$prioridad (int) El nivel de prioridad al que se ejecutará $function_to_add. Opcional. Predeterminado 10.
$args_aceptados (int) El número de argumentos que acepta la función invocable $function_to_add. Opcional. Predeterminado 1.

Devolución de llamada de función directa #

add_action( 'init', function() {
    // do something here
} );

Uso de un bloque de funciones para enlazar un conjunto de instrucciones. Con el gancho init, el conjunto de instrucciones se ejecutará justo después de que wordpress haya terminado de cargar los componentes necesarios.

Devolución de llamada de referencia de nombre de función #

function my_init_function() {
    // do something here
}

add_action( 'init', 'my_init_function' );

Usar el nombre de la función para enlazar un conjunto de instrucciones. Con el gancho init, el conjunto de instrucciones se ejecutará justo después de que wordpress haya terminado de cargar los componentes necesarios.

Devolución de llamada de método estático de clase #

class MyClass {
    static function my_init_method() {
        // do something here
    }
}

add_action( 'init', array( 'MyClass', 'my_init_method' ) );

Usar un método estático de una clase para enganchar un conjunto de instrucciones. Con el gancho init, el conjunto de instrucciones se ejecutará justo después de que wordpress haya terminado de cargar los componentes necesarios.

Devolución de llamada del método de objeto #

class MyClass {
    function my_init_method() {
        // do something here
    }
}

$obj = new MyClass();

add_action( 'init', array( $obj, 'my_init_method' ) );

Usar un método de un objeto para enganchar un conjunto de instrucciones. Con el gancho init, el conjunto de instrucciones se ejecutará justo después de que wordpress haya terminado de cargar los componentes necesarios.
