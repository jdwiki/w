Sintaxis #

  • wp_insert_post(arreglo $argumentos, bool $wp_error);

Parámetros #

Parámetro Descripción
$args (matriz requerida) Una matriz de valores clave de los siguientes elementos.
$wp_error (booleano opcional) Retorna un WP_Error en caso de falla.

Argumentos #

La siguiente tabla le muestra una lista de elementos que puede usar dentro del primer parámetro (Array).

Parámetro Descripción
identificación (Int) El ID de la publicación. Si es igual a algo distinto de 0, se actualizará la publicación con esa ID. Predeterminado 0.
post_autor (Int) El ID del usuario que agregó la publicación. El valor predeterminado es el ID de usuario actual.
post_fecha (String) La fecha de la publicación. El valor predeterminado es la hora actual.
post_date_gmt (Cadena) La fecha de la publicación en la zona horaria GMT. El valor predeterminado es el valor de $post_date.
post_contenido (Mixto) El contenido de la publicación. Vacío por defecto.
post_contenido_filtrado (String) El contenido de la publicación filtrada. Vacío por defecto.
post_título (Cadena) El título de la publicación. Vacío por defecto.
post_categoría (Matriz) Matriz de valores de categoría de publicación.
post_excerpt (String) El extracto de la publicación. Vacío por defecto.
post_status (Cadena) El estado de la publicación. Borrador predeterminado.
post_type (Cadena) El tipo de publicación. Publicación predeterminada.
comentario_estado (String) Si la publicación puede aceptar comentarios. Acepta abierto o cerrado. El valor predeterminado es el valor de la opción default_comment_status.
estado_ping (String) Si la publicación puede aceptar pings. Acepta abierto o cerrado. El valor predeterminado es el valor de la opción default_ping_status.
post_contraseña (String) La contraseña para acceder a la publicación. Vacío por defecto.
nombre_de_la_publicación (String) El nombre de la publicación o slug. El valor predeterminado es el título de publicación saneado al crear una nueva publicación.
to_ping (Cadena) Lista de direcciones URL separadas por espacios o retornos de carro para hacer ping. Vacío por defecto.
ping (Cadena) Lista de direcciones URL separadas por espacios o retornos de carro a las que se ha hecho ping. Vacío por defecto.
post_modificado (Cadena) La fecha en que se modificó por última vez la publicación. El valor predeterminado es la hora actual.
post_modificado_gmt (Cadena) La fecha en la que se modificó por última vez la publicación en la zona horaria GMT. El valor predeterminado es la hora actual.
post_padre (Int) Configure esto para la publicación a la que pertenece, si corresponde. Predeterminado 0.
menu_order (Int) El orden en que se debe mostrar la publicación. Predeterminado 0.
post_tipo_mimo (String) El tipo mimo de la publicación. Vacío por defecto.
guía (Cadena) Id. único global para hacer referencia a la publicación. Vacío por defecto.
entrada_de_impuestos (Matriz) Matriz de términos de taxonomía codificados por su nombre de taxonomía. Vacío por defecto.
meta_entrada (Array) Matriz de metavalores de publicación codificados por su clave de metadatos de publicación. Vacío por defecto.

Evite publicaciones duplicadas #

Cuando ejecuta esta función, probablemente podría obtener una publicación duplicada, al menos eso me sucedió a mí. (Puede consultarlo en la sección Publicar WordPress)

Encontré una solución:

if( !get_page_by_title( $title, 'OBJECT', 'post' ) ){
    $my_post = array('post_title' => $title,
        'post_content' => 'Content',
        'tags_input' => $tags,
        'post_category' => array(2),
        'post_status' => 'publish'
    );

    $result = wp_insert_post( $my_post );
}

Explicación #

Antes de guardar una nueva publicación, valide si la nueva publicación ya existe usando el título de la publicación como parámetro, si no hay un título de la publicación, puede guardar su nueva publicación.

Consulta la documentación de get_page_by_title aquí.

Crear una página básica #

$basic_page_args = array(
    'post_title' => 'My Basic Page',
    'post_content' => 'This is a basic content',
    'post_type' => 'page',
    'post_status' => 'publish',
    'post_author' => 1
);

wp_insert_post( $basic_page_args );

Crear una publicación básica #

$basic_post_args = array(
    'post_title' => 'My Basic Post',
    'post_content' => 'This is a basic content',
    'post_status' => 'publish',
    'post_author' => 1,
    'post_category' => array(8, 59)
);

wp_insert_post( $basic_post_args );

Introducción #

A veces tenemos otro editor en otro lugar en lugar de TinyMCE (Wordpress Default Editor). Eso sucede cuando estamos creando nuestro propio Tema, Complemento o algo específico; y necesitamos escribir y manipular un tipo de publicación y guardarla en nuestra base de datos de WP.

Entonces, si se encuentra en esa situación, puede usar una función de Wordpress llamada:

wp_insert_post( array $args, bool $wp_error );
