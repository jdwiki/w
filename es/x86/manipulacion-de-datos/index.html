Sintaxis #

  • .386: Le dice a MASM que compile para una versión mínima de chip x86 de 386.
  • .model: Establece el modelo de memoria a usar, consulte .MODEL.
  • .código: Segmento de código, utilizado para procesos como el proceso principal.
  • proc: Declara proceso.
  • ret: utilizado para salir de funciones con éxito, consulte Trabajar con valores devueltos.
  • endp: Termina la declaración del proceso.
  • público: hace que el proceso esté disponible para todos los segmentos del programa.
  • end: finaliza el programa o, si se usa con un proceso, como en “end main”, hace que el proceso sea el método principal.
  • call: llama al proceso y empuja su código de operación a la pila, consulte Flujo de control.
  • ecx: Registro contador, ver registros.
  • ecx: Registro contador.
  • mul: Multiplica valor por eax

mov se utiliza para transferir datos entre los registros.

Usando MOV para manipular valores #

Descripción:

mov copia valores de bits del argumento de origen al argumento de destino.

El origen/destino común son registros, generalmente la forma más rápida de manipular valores con [en] CPU.

Otro grupo importante de valores source_of/destination_for es la memoria de la computadora.

Finalmente, algunos valores inmediatos pueden ser parte de la codificación de la instrucción mov, ahorrando tiempo de acceso a la memoria por separado al leer el valor junto con la instrucción.

En la CPU x86 en modo de 32 y 64 bits, hay muchas posibilidades para combinarlos, especialmente varios modos de direccionamiento de memoria. En general, la copia de memoria a memoria está fuera del límite (excepto las instrucciones especializadas como MOVSB), y tal manipulación requiere primero el almacenamiento intermedio de valores en los registros.

Paso 1: Configure su proyecto para usar MASM, consulte Ejecutar ensamblado x86 en Visual Studio 2015 Paso 2: Escribe esto:

.386
.model small
.code

public main
main proc
    mov ecx, 16       ; Move immediate value 16 into ecx
    mov eax, ecx      ; Copy value of ecx into eax
    ret               ; return back to caller
        ; function return value is in eax (16)
main endp
end main

Paso 3: Compilar y depurar.

El programa debería devolver el valor 16.
