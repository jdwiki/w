Sintaxis #

  • func (t T) exampleOne(i int) (n int) { return i } // esta función recibirá una copia de la estructura
  • func (t *T) ejemploDos(i int) (n int) { return i } // este método recibirá un puntero a struct y podrá modificarlo

Métodos básicos #

Los métodos en Go son como funciones, excepto que tienen receptor.

Por lo general, el receptor es algún tipo de estructura o tipo.

package main

import (
    "fmt"
)

type Employee struct {
    Name string
    Age  int
    Rank int
}

func (empl *Employee) Promote() {
    empl.Rank++
}

func main() {

    Bob := new(Employee)

    Bob.Rank = 1
    fmt.Println("Bobs rank now is: ", Bob.Rank)
    fmt.Println("Lets promote Bob!")

    Bob.Promote()

    fmt.Println("Now Bobs rank is: ", Bob.Rank)

}

Producción:

Bobs rank now is:  1
Lets promote Bob!
Now Bobs rank is:  2

Métodos de encadenamiento #

Con los métodos en golang, puede “encadenar” el método pasando el puntero al método y devolviendo el puntero a la misma estructura de esta manera:

package main

import (
    "fmt"
)

type Employee struct {
    Name string
    Age  int
    Rank int
}

func (empl *Employee) Promote() *Employee {
    fmt.Printf("Promoting %s\n", empl.Name)
    empl.Rank++
    return empl
}

func (empl *Employee) SetName(name string) *Employee {
    fmt.Printf("Set name of new Employee to %s\n", name)
    empl.Name = name
    return empl
}

func main() {

    worker := new(Employee)

    worker.Rank = 1

    worker.SetName("Bob").Promote()

    fmt.Printf("Here we have %s with rank %d\n", worker.Name, worker.Rank)

}

Producción:

Set name of new Employee to Bob
Promoting Bob
Here we have Bob with rank 2

Operadores de incremento-decremento como argumentos en Métodos #

Aunque Go admite los operadores ++ y – y el comportamiento es casi similar al de c/c++, las variables con dichos operadores no se pueden pasar como argumento a la función.

    package main

    import (
        "fmt"
    )
    
    func abcd(a int, b int) {
     fmt.Println(a," ",b)
    }
    func main() {
        a:=5
        abcd(a++,++a)
    }

Salida: error de sintaxis: inesperado ++, esperando coma o)
