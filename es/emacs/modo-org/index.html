Org es un modo para tomar notas, mantener listas de TODO y planificar proyectos con un sistema de texto sin formato rápido y efectivo. También es un sistema de autoría con soporte único para programación alfabetizada e investigación reproducible.

sitio oficial del modo org

Sintaxis de marcado #

Org proporciona un lenguaje de marcado completo que ayuda a estructurar el documento y se refleja con la mayor precisión posible al exportar a otros formatos (como HTML o LaTeX).

Estructura #

Titulo del documento #

#+TITLE: This is the title of the document

Seccionamiento #

* First level
** Second level

Listas #

Ordered list (items can also be numbered like '1)', with a perenthesis):
1. foo
2. bar
3. baz

Unordered list (items can also start with '+' or '*'):
- foo
- bar
- baz

Description
- lorem ipsum :: this is example text
- foo bar :: these are placeholder words

Casillas de verificación #

Cada elemento de una lista simple se puede convertir en una casilla de verificación comenzando con la cadena ‘[ ]’.

* TODO [2/4] (or [50%])
  - [-] call people [1/3]
    - [ ] Peter
      - [X] Sarah
      - [ ] Sam
  - [X] order food
  - [ ] think about what music to play
  - [X] talk to the neighbors
  • C-c C-c org-toggle-checkbox
  • C-c C-x C-b org-toggle-checkbox
  • M-S- org-insert-todo-heading
  • C-c C-x o org-toggle-ordered-property
  • C-c # org-update-statistics-cookies

Énfasis y monoespaciado #

You can make words *bold*, /italic/, _underlined_, =verbatim=
and ~code~, and, if you must, ‘+strike-through+’.

El texto en el código y la cadena textual no se procesa para la sintaxis específica del modo Org, se exporta textualmente.

Enlaces y referencias #

Enlaces #

Org-mode reconocerá los formatos de URL y los activará como enlaces en los que se puede hacer clic. Sin embargo, los enlaces se pueden declarar explícitamente así:

You will find more information in the [[http://orgmode.org/org.html][Org Manual]].

o alternativamente :

The org manual is located here: [[http://orgmode.org/org.html]]

Notas al pie #

Las notas al pie se pueden nombrar:

See the org manual[fn:manual] to get more details.
...
[fn:manual] You will find it here: http://orgmode.org/org.html

o anónimo y en línea:

See the org manual[fn:: You will find it here: http://orgmode.org/org.html]
to get more details.

Combinaciones de teclas básicas #

Para alternar el nivel de contorno mostrado:

  • Tab Nivel de esquema de ciclo para un encabezado
  • Shift-Tab Ciclo de nivel de esquema para todo el documento

Para recorrer los estados TODO:

  • Mayúsculas-Flecha derecha
  • Mayúsculas-Flecha izquierda

Para aumentar o disminuir el nivel jerárquico de un encabezado

  • Meta-Flecha derecha Hacer un nivel más bajo (“aumentar sangría”)
  • Flecha meta-izquierda Hacer un nivel más alto (“reducir sangría”)

Para alternar la prioridad de un encabezado determinado:

  • Mayús-Flecha arriba
  • Mayús-Flecha Abajo

Para mover un encabezado hacia arriba o hacia abajo:

  • Flecha meta-arriba
  • Meta-Flecha Abajo

(Meta se refiere a diferentes teclas en diferentes teclados. La mayoría de las veces es Alt o ⌘).

Bloques de código #

Para agregar un bloque de código, rodéelo con #+BEGIN_SRC language y #+END_SRC. idioma debe corresponder al modo principal del idioma en cuestión, p. el modo principal para Emacs Lisp es emacs-lisp-mode, así que escribe #+BEGIN_SRC emacs-lisp.

#+BEGIN_SRC emacs-lisp
(defun hello-world ()
  (interactive)
  (message "hello world"))
#+END_SRC

#+BEGIN_SRC python
print "hello world"
#+END_SRC

Puede abrir el bloque de código en un búfer separado escribiendo C-c '' (para org-edit-special). Si no tiene el modo principal para el idioma especificado, aparecerá un mensaje de error como No existe tal modo de idioma: foo-mode`.

Si el contenido que quieres poner en el bloque no está en ningún lenguaje de programación, puedes usar #+BEGIN_EXAMPLE y #+END_EXAMPLE en su lugar.

#+BEGIN_EXAMPLE
output from a command I just ran
#+END_EXAMPLE

Hay [plantillas fáciles] (http://orgmode.org/manual/Easy-Templates.html) para ambos. Al comienzo de la línea, escriba <s o <e, y luego presione TAB. Se expandirá en un bloque con marcadores de inicio y fin para SRC o EXAMPLE, respectivamente.

Estos marcadores no distinguen entre mayúsculas y minúsculas, por lo que puede escribir #+begin_src, etc., si lo prefiere.

Mesas #

| Name  | Phone | Age |
|-------+-------+-----|
| Peter | 1234  | 17  |
| Anna  | 4321  | 25  |

Para agregar una tabla en modo org, simplemente rodee sus columnas con una barra (|)

| column1 | column2 | this column is wider |

Cuando presiona Return desde dentro de una columna, org-mode creará automáticamente una nueva fila con las barras.

  • Tab y Return se moverán respectivamente a la siguiente celda o fila (o crearán una nueva si no hay ninguna)
  • Puede intercambiar filas y columnas con M-ArrowKey
  • M-S-Down y M-S-Right crearán respectivamente una fila (sobre la actual) y una columna (a la izquierda de la actual)
  • M-S-Up y M-S-Left eliminarán respectivamente la fila actual y la columna actual
  • C-c i crear un separador
