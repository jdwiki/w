Parámetros #

Combinaciones de teclas Detalles
^ Tecla Ctrl
! Tecla Alt
+ Tecla de mayúsculas
# Tecla de Windows
{entrar} enviar tecla enter
{pestaña} enviar tecla de tabulador
* comodín, se puede presionar cualquier tecla
~ la función nativa de la tecla no se bloqueará
<símbolo especifica la tecla izquierda (<+ es desplazamiento a la izquierda)
>símbolo especifica la tecla derecha

Cuerda caliente #

Para crear un script que reemplace una frase, use la sintaxis ::abbreviation:: hotstring. Reemplazará por cierto con por cierto cada vez que ingrese por cierto y luego la tecla de espacio.

::btw::by the way

Si desea crear una secuencia de comandos de inicio de sesión para que el inicio de sesión sea más rápido, puede crear una secuencia de comandos como esta (el archivo no está encriptado, por lo que cualquier información en su secuencia de comandos será visible para cualquier persona con acceso al archivo).

::lmi::user{tab}password{enter}

Hotkeys y Hotstrings sensibles al contexto #

Para crear una tecla de acceso rápido o una cadena de acceso rápido que solo se active cuando ciertas ventanas estén activas o existan, puede colocar una o varias de las siguientes directrices antes de la definición de la tecla de acceso rápido:

#IfWinActive [, WinTitle, WinText]
#IfWinExist [, WinTitle, WinText]
#IfWinNotActive [, WinTitle, WinText]
#IfWinNotExist [, WinTitle, WinText]

Ejemplo: desea que se envíe stackoverflow.com cada vez que escriba so (y un espacio en blanco después de eso) en Google Chrome, pero ignore la cadena de acceso en cualquier otra ventana.

#IfWinActive, ahk_class Chrome_WidgetWin_1
::so::stackoverflow.com

Al usar #If [, Expression ], puede activar una tecla de acceso rápido solo cuando una expresión arbitraria es verdadera, por ejemplo:

#If A_Hour < 9
F1::
    MsgBox, It is too early to ask for help!
return

Reasignar teclas #

El siguiente ejemplo reasigna la clave Z a Y y viceversa, p. si desea trabajar con el diseño QWERTY en un teclado QWERTZ.

z::y
y::z

tecla de acceso directo #

Para crear una tecla de acceso rápido que envíe la secuencia de teclas ‘Hello World’ presionando Ctrl + J en la ventana activa (se puede demostrar en el bloc de notas, por ejemplo)

^j::
    Send, Hello World
Return

Varias pulsaciones de teclas #

Para ejecutar un script cuando se presionan varias teclas, utilice & entre las teclas.

Numpad0 & Numpad1::
    MsgBox You pressed 0 and 1
return

Teclas de acceso rápido alternables #

El siguiente script ingresa cadenas predefinidas al presionar teclas de acceso rápido si el bloqueo de desplazamiento está activo. Esto puede ser útil si pega con frecuencia varias cadenas repetidas. Tecla de acceso rápido incluida para actualizar el script (por ejemplo, si necesita editar cadenas que se puedan pegar).

; refresh script hotkey
Numpad9:: 
    GetKeyState, state, ScrollLock, T
    if ( state = "D" )
        Reload
Return

Numpad1:: 
    GetKeyState, state, ScrollLock, T
    if ( state = "D" )
        Send,         Hello 
Return

Numpad2:: 
    GetKeyState, state, ScrollLock, T
    if ( state = "D" )
        Send,         World
Return
;...
