aws komut satırı aracı ve git komutuyla yerel geliştirme makinenizi ayarlayarak hazırlanın.

git komut satırı için Codecommit Kurulumu #

AWS Codecommit, özel GIT havuzları için depolama olarak kullanılabilir. Kurulum, halihazırda geçerli bir AWS hesabınız olduğunu varsayarak birkaç adımı içerir.

  1. AWS Codecommit’e kaydolun. Şu anda yalnızca “us-east-1” bölgesi mevcuttur.

  2. Depolara erişimi olacak bir IAM kullanıcısı oluşturun, örneğin “codecommit-user”

  3. Bu kullanıcıya “AWSCodeCommitFullAccess” izin rolünü ekleyin

  4. Bu kullanıcı için yeni bir “Erişim Anahtarı” oluşturun ve “anahtar kimliği” ile “gizli kodu” not edin.

  5. Şimdi devam edin ve yerel makinenizde bir AWS Yapılandırma profili oluşturun

    $ aws configure –profile codecommit-user

Bir sonraki adımda, “aws” komutunu, aşağıdaki komutlarla kimlik bilgisi yardımcısı olarak “git” ile ilişkilendiririz:

$ git config --global credential.helper \
    '!aws --profile codecommit-user codecommit credential-helper $@'
$ git config --global credential.UseHttpPath true

Bu kurulumu daha sonra doğrulayabilir veya düzenleyebilirsiniz:

$ git config --global --edit

Bir bölüme dikkat etmelisiniz:

[credential]
    helper = !aws --profile codecommit-user codecommit credential-helper $@
    UseHttpPath = true

Artık git’i her zamanki gibi komut satırından kullanabilirsiniz.

SourceTree’yi AWS Codecommit ile kullanın #

Atlassian SourceTree, Mac ve Windows için kaynak kod havuzlarını yönetmeye yönelik görsel bir araçtır. Bu, Codecommit ile uzak bir depo olarak kullanılabilir, ancak codecommit ile bağlanabilmek için SourceTree’deki yerel depoya fazladan bir yapılandırma seçeneği eklemeniz gerekir.

İlk olarak, yerel git için Codecommit’i kurun.

“codecommit"e göndermek istediğiniz yerel bir “git” deponuz olduğunu varsayarsak, şu adımları izleyin:

  1. Web konsolunu kullanarak AWS Codecommit’te oturum açın.

  2. Yeni bir havuz oluşturun, örneğin “my-project”

  3. HTTPS URL’sini kopyalayın, “https://git-codecommit.us-east-1.amazonaws.com/v1/repos/my-project” gibi görünmelidir.

  4. Şimdi SourceTree’de Ayarlar / Uzaktan Kumandalar panelini açın

  5. Yeni uzaktan kumandayı şu adla ekleyin: origin ve Url / Path: daha önce kopyaladığınız bağlantı

  6. Son olarak, Yapılandırma Dosyasını Düzenle seçeneğini açın ve aşağıdaki parçacığı ekleyin:

    [credential] helper = /usr/local/bin/aws –profile codecommit-user codecommit credential-helper $@ UseHttpPath = true

Yapılandırma dosyasını kaydettikten sonra şöyle görünmelidir:

[core]
    repositoryformatversion = 0
    filemode = true
    bare = false
    logallrefupdates = true
    ignorecase = true
    precomposeunicode = true
[branch "master"]
    remote = origin
    merge = refs/heads/master
[remote "origin"]
    url = https://git-codecommit.us-east-1.amazonaws.com/v1/repos/digitaloffice.nu
    fetch = +refs/heads/*:refs/remotes/origin/*
[credential]
    helper = /usr/local/bin/aws --profile codecommit-user codecommit credential-helper $@
    UseHttpPath = true

Lütfen dikkat: Bu, OS-X kurulumuna dayanmaktadır. aws (bu durumda /usr/local/bin/aws olan) yoluna özellikle dikkat edin ve kesinlikle diğer Unix’ler veya Windows konfigürasyonları altında farklı olacaktır.
