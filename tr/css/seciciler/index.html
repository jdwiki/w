CSS seçicileri, belirli HTML öğelerini CSS stilleri için hedefler olarak tanımlar. Bu konu, CSS seçicilerinin HTML öğelerini nasıl hedeflediğini kapsar. Seçiciler, CSS dili tarafından sunulan, öğeler, sınıflar, kimlikler, sözde öğeler ve sözde sınıflar ve kalıplar dahil olmak üzere 50’den fazla seçim yöntemini kullanır.

Sözdizimi #

  • #kimlik

  • .sınıf adı

  • :sözde-sınıf adı

  • ::sözde öğe adı

  • [attr] /* attr özniteliğine sahiptir. */

  • [attr="value"] /* attr özniteliğine sahiptir ve değeri tam olarak “değer“dir. */

  • [attr~="value”] /* attr özniteliğine sahiptir ve değeri boşluk üzerinde bölündüğünde “değer” içerir. */

  • [attr|="value”] /* attr özelliğine sahip ve değeri tam olarak “value”, veya değeri “value-” ile başlıyor . */

  • [attr^="value"] /* attr özelliğine sahiptir ve değeri “value” ile başlar. */

  • [attr$="value"] /* attr özelliğine sahiptir ve değeri “value” ile biter. */

  • [attr*="değer"] /* attr özniteliğine sahiptir ve değeri “değer” içerir. */

  • eleman-adı

  • *​

  • Bazen yalnızca bir (:) yerine iki nokta üst üste (::) görürsünüz. Bu, sözde sınıfları sözde öğeler‘den ayırmanın bir yoludur.

  • Internet Explorer 8 gibi eski tarayıcılar yalnızca sözde öğeleri tanımlamak için tek bir iki nokta üst üste (:) destekler.

  • Sözde sınıflardan farklı olarak, seçici başına yalnızca bir sözde öğe kullanılabilir, eğer varsa, seçicinin öznelerini temsil eden basit seçiciler dizisinden sonra görünmelidir (W3C belirtiminin gelecekteki bir sürümü izin verebilir seçici başına birden çok sözde eleman).

Detaylar #

“[özellik]” #

Verilen özniteliğe sahip elemanları seçer.

div[data-color] {
  color: red;
}

<div data-color="red">This will be red</div>
<div data-color="green">This will be red</div>
<div data-background="red">This will NOT be red</div>

Verilen öznitelik ve değere sahip öğeleri seçer.

div[data-color="red"] {
  color: red;
}

<div data-color="red">This will be red</div>
<div data-color="green">This will NOT be red</div>
<div data-color="blue">This will NOT be red</div>

[attribute*="değer"] #

Verilen özniteliğe ve verilen özniteliğin verilen değeri herhangi bir yerde (alt dize olarak) içerdiği değere sahip öğeleri seçer.

[class*="foo"] {
  color: red;
}

<div class="foo-123">This will be red</div>
<div class="foo123">This will be red</div>
<div class="bar123foo">This will be red</div>
<div class="barfooo123">This will be red</div>
<div class="barfo0">This will NOT be red</div>

[özellik~="değer"] #

Verilen değerin boşlukla ayrılmış bir listede göründüğü, verilen özniteliğe ve değere sahip öğeleri seçer.

[class~="color-red"] {
  color: red;
}

<div class="color-red foo-bar the-div">This will be red</div>
<div class="color-blue foo-bar the-div">This will NOT be red</div>

JSBin’de Canlı Demo

[özellik^="değer"] #

Verilen özniteliğe ve verilen özniteliğin değerle başladığı değere sahip öğeleri seçer.

[class^="foo-"] {
  color: red;
}

<div class="foo-123">This will be red</div>
<div class="foo-234">This will be red</div>
<div class="bar-123">This will NOT be red</div>

JSBin’de Canlı Demo

[attribute$="değer"] #

Verilen özniteliğin verilen değerle bittiği yerde, verilen özniteliğe ve değere sahip öğeleri seçer.

[class$="file"] {
  color: red;
}

<div class="foobar-file">This will be red</div>
<div class="foobar-file">This will be red</div>
<div class="foobar-input">This will NOT be red</div>

JSBin’de Canlı Demo

[attribute|="değer"] #

Özniteliğin değerinin tam olarak verilen değer olduğu veya tam olarak verilen değerin ardından “-” (U+002D) olan belirli bir öznitelik ve değere sahip öğeleri seçer

[lang|="EN"] {
  color: red;
}

<div lang="EN-us">This will be red</div>
<div lang="EN-gb">This will be red</div>
<div lang="PT-pt">This will NOT be red</div>

JSBin’de Canlı Demo

[attribute="değer" i] #

Belirli bir özniteliğe ve özniteliğin değerinin “Değer”, “DEĞER”, “değer” veya büyük/küçük harfe duyarsız diğer herhangi bir olasılık olarak temsil edilebileceği değerlere sahip öğeleri seçer.

[lang="EN" i] {
  color: red;
}

<div lang="EN">This will be red</div>
<div lang="en">This will be red</div>
<div lang="PT">This will NOT be red</div>

Öznitelik seçicilerin özgüllüğü #

“0-1-0” #

Sınıf seçici ve sözde sınıf ile aynı.

*[type=checkbox] // 0-1-0

Bunun bir öznitelik seçicinin, bir öğeyi kimliğine göre seçmek için kullanılabileceği anlamına geldiğini unutmayın, ID seçici: [id="my-ID"] hedeflerine göre daha düşük bir özgüllük düzeyinde “#my-ID” ile aynı öğe ancak daha düşük özgüllük ile.

Daha fazla ayrıntı için Sözdizimi Bölümüne bakın.

2: https://www.wikiod.com/tr/css/seciciler#kimlik seçiciler

Temel seçiciler #

Seçici Açıklama
* Evrensel seçici (tüm öğeler)
‘div’ Etiket seçici (tüm <div> öğeleri)
.mavi Sınıf seçici (“mavi” sınıfına sahip tüm öğeler)
.mavi.kırmızı ‘blue’ ve ‘red’ sınıfına sahip tüm öğeler (bir tür Bileşik seçici)
“#başlık” Kimlik seçici (“id” özniteliği “başlık” olarak ayarlanmış öğe)
:sözde sınıf Sözde sınıflı tüm öğeler
::sözde eleman Sözde öğeyle eşleşen öğe
:lang(tr) :lang bildirimiyle eşleşen öğe, örneğin <lang span="en">
div > p çocuk seçici

Not: Bir kimliğin değeri, bir web sayfasında benzersiz olmalıdır. Bir kimliğin değerini aynı belge ağacında birden fazla kullanmak HTML standardının ihlalidir. .

Seçicilerin tam listesi CSS Selectors Level 3 spesifikasyonunda bulunabilir.

Birleştiriciler #

Genel Bakış #

Seçici Açıklama
div yayılma Descendant seçici (bir “
” öğesinin soyundan gelen tüm ““ler)
div > yayılma Alt seçici (bir “
” öğesinin doğrudan çocuğu olan tüm ““ler)
a ~ yayılma Genel Kardeş seçici (bir ““dan sonra kardeş olan tüm ““ler)
a + yayılma Bitişik Kardeş seçicisi (bir ““dan hemen sonra gelen tüm ““ler)

Not: Kardeş seçiciler, kaynak belgede kendilerinden sonra gelen öğeleri hedefler. CSS, doğası gereği (basamaklı) önceki veya ana öğeleri hedefleyemez. Ancak, flex ‘order’ özelliği kullanılarak önceki bir kardeş seçici görsel medyada simüle edilebilir.

Descendant Birleştirici: ‘seçici seçici’ #

En az bir boşluk karakteriyle ( ) temsil edilen alt bir birleştirici, tanımlanan öğenin alt öğesi olan öğeleri seçer. Bu birleştirici, öğenin tüm alt öğelerini seçer (alt öğelerden aşağıya doğru).

div p {
  color:red;
}

<div>
  <p>My text is red</p>
  <section>
    <p>My text is red</p>
  </section>
</div>

<p>My text is not red</p>

Çocuk Birleştirici: seçici > seçici #

Alt (>) birleştiricisi, belirtilen öğenin alt öğeleri veya doğrudan alt öğeleri olan öğeleri seçmek için kullanılır.

div > p {
  color:red;
}

<div>
  <p>My text is red</p>
  <section>
    <p>My text is not red</p>
  </section>
</div>

” öğesinden türetilen tek paragraf olduğu için yalnızca ilk “

” öğesini seçer.

İkinci “

” öğesi, “

” öğesinin doğrudan alt öğesi olmadığı için seçilmedi.

Bitişik Kardeş Birleştirici: seçici + seçici #

Bitişik kardeş (’+’) birleştiricisi, belirtilen bir öğeyi hemen takip eden bir kardeş öğe seçer.

p + p {
  color:red;
}

<p>My text is not red</p>
<p>My text is red</p>
<p>My text is red</p>
<hr>
<p>My text is not red</p>

” öğesinin doğrudan önüne gelen “

” öğelerini seçer.

Genel Kardeş Birleştirici: seçici ~ seçici #

Genel kardeş (~) birleştiricisi, belirtilen öğeyi izleyen tüm kardeşleri seçer.

p ~ p {
  color:red;
}

<p>My text is not red</p>
<p>My text is red</p>
<hr>
<h1>And now a title</h1>
<p>My text is red</p>

Sözde sınıflar #

Sözde sınıflar, belge ağacının dışında kalan veya diğer seçiciler veya birleştiriciler tarafından ifade edilemeyen bilgilere dayalı olarak seçime izin veren anahtar kelimelerdir. Bu bilgi belirli bir durumla (durum ve dinamik sözde sınıflar), konumlarla (yapısal ve hedef sözde sınıflar), öncekinin (olumsuz sözde sınıfın) veya dillerin (lang sözde sınıfın) olumsuzlamaları. Örnekler, bir bağlantının izlenip izlenmediğini (:ziyaret edildi), farenin bir öğenin üzerinde olup olmadığını (:hover), bir onay kutusunun işaretli olup olmadığını (:işaretli), vb.

Sınıf Adı Seçiciler #

Sınıf adı seçici, hedeflenen sınıf adına sahip tüm öğeleri seçer. Örneğin, “.warning” sınıf adı aşağıdaki “

” öğesini seçer:

<div class="warning">
    <p>This would be some warning copy.</p>
</div>

Öğeleri daha spesifik olarak hedeflemek için sınıf adlarını da birleştirebilirsiniz. Daha karmaşık bir sınıf seçimini sergilemek için yukarıdaki örneği geliştirelim.

CSS

.important {
    color: orange;
}
.warning {
    color: blue;
}
.warning.important {
    color: red;
}

HTML

<div class="warning">
    <p>This would be some warning copy.</p>
</div>

<div class="important warning">
    <p class="important">This is some really important warning copy.</p>
</div>

Bu örnekte, “.warning” sınıfına sahip tüm öğeler mavi metin rengine, “.important” sınıfına sahip öğelere sahip turuncu bir metin rengine ve “.important” ve “hem” hem öğelerine sahip tüm öğelere sahip olacaktır. .warning` sınıf adı kırmızı bir metin rengine sahip olacaktır.

CSS içinde, “.warning.important” bildiriminin iki sınıf adı arasında boşluk olmadığına dikkat edin. Bu, yalnızca “sınıf” özniteliklerinde “uyarı” ve “önemli” sınıf adlarını içeren öğeleri bulacağı anlamına gelir. Bu sınıf adları, öğe üzerinde herhangi bir sırada olabilir.

CSS bildiriminde iki sınıf arasına bir boşluk eklenmişse, yalnızca “.warning” sınıf adlarına sahip üst öğelere ve “.important” sınıf adlarına sahip alt öğelere sahip öğeleri seçer.

Kimlik seçicinin yüksek özgüllüğü olmadan kimliğini kullanarak öğe seçin #

Bu numara, ID seçicinin yüksek özgüllüğünden kaçınmak için bir öznitelik seçici için bir değer olarak ID’yi kullanarak bir öğe seçmenize yardımcı olur.

HTML:

<div id="element">...</div>

CSS

#element { ... } /* High specificity will override many selectors */

[id="element"] { ... } /* Low specificity, can be overridden easily */

Onay kutulu genel boole:işaretli ve ~ (genel kardeş birleştirici) #

~ seçiciyle, JavaScript kullanmadan global olarak erişilebilir bir boole kolayca uygulayabilirsiniz.

Onay kutusu olarak boole ekle #

Belgenizin en başına, benzersiz bir “id” ve “gizli” öznitelik kümesiyle istediğiniz kadar boole ekleyin:

<input type="checkbox" id="sidebarShown" hidden />
<input type="checkbox" id="darkThemeUsed" hidden />

<!-- here begins actual content, for example: -->
<div id="container">
    <div id="sidebar">
        <!-- Menu, Search, ... -->
    </div>

    <!-- Some more content ... -->
</div>

<div id="footer">
    <!-- ... -->
</div>

Boole değerini değiştirin #

“for” öznitelik kümesiyle bir “etiket” ekleyerek boole arasında geçiş yapabilirsiniz:

<label for="sidebarShown">Show/Hide the sidebar!</label>

CSS ile boole değerine erişme #

Normal seçici (".color-red” gibi) varsayılan özellikleri belirtir. “true” / “false” seçicileri izlenerek geçersiz kılınabilirler:

/* true: */
<checkbox>:checked ~ [sibling of checkbox & parent of target] <target>

/* false: */
<checkbox>:not(:checked) ~ [sibling of checkbox & parent of target] <target>

“”, “[sibling …]” ve “” öğelerinin uygun seçicilerle değiştirilmesi gerektiğini unutmayın. [kardeş ...] belirli bir seçici olabilir, (genellikle tembelseniz) sadece * veya hedef zaten onay kutusunun bir kardeşi ise hiçbir şey olabilir.

Yukarıdaki HTML yapısına örnekler:

#sidebarShown:checked ~ #container #sidebar {
    margin-left: 300px;
}

#darkThemeUsed:checked ~ #container,
#darkThemeUsed:checked ~ #footer {
    background: #333;
}
