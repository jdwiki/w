CSS ile bir öğenin arka planı olarak renkleri, degradeleri ve görüntüleri ayarlayabilirsiniz.

Çeşitli görüntü, renk ve degrade kombinasyonları belirlemek ve bunların boyutunu, konumunu ve tekrarını (diğerlerinin yanı sıra) ayarlamak mümkündür.

Sözdizimi #

  • arka plan rengi: renk | şeffaf | başlangıç ​​| miras;
  • arka plan resmi: url | hiçbiri | başlangıç ​​| miras;
  • arka plan konumu: değer;
  • arka plan boyutu: <bg-size> [<bg-size>]
  • <bg-size>: otomatik | uzunluk | kapak | içerir | başlangıç ​​| miras;
  • arka plan-tekrar: tekrar | tekrar-x | tekrar-y | tekrarsız | başlangıç ​​| miras;
  • arka plan-kökeni: padding-box | sınır kutusu | içerik kutusu | başlangıç ​​| miras;
  • arka plan klibi: kenarlık kutusu | dolgu kutusu | içerik kutusu | başlangıç ​​| miras;
  • arka plan eki: kaydırma | sabit | yerel | başlangıç ​​| miras;
  • arka plan: bg-color bg-image konumu / bg-size bg-repeat bg-origin bg-clip bg-ek başlangıç ​​| miras;
  • CSS3 gradyanları, Internet Explorer’ın 10’dan küçük sürümlerinde çalışmayacaktır.

Arka plan rengi #

“background-color” özelliği, bir renk değeri kullanarak veya “transparent”, “inherit” veya “initial” gibi anahtar sözcükler aracılığıyla bir öğenin arka plan rengini ayarlar.

  • şeffaf, arka plan renginin şeffaf olması gerektiğini belirtir. Bu varsayılandır.

  • devralır, bu özelliği üst öğesinden devralır.

  • ilk, bu özelliği varsayılan değerine ayarlar.

Bu, tüm öğelere ve ::first-letter/::first-line sözde öğeler öğelerine uygulanabilir.

CSS’deki renkler farklı yöntemlerle belirtilebilir.

Renk adları #

CSS

div {
  background-color: red;  /* red */
}

HTML

<div>This will have a red background</div>
  • Yukarıda kullanılan örnek, CSS’nin tek bir rengi temsil etmesi gereken birkaç yoldan biridir.

Hex renk kodları #

Hex kodu, bir rengin RGB bileşenlerini 16 tabanlı onaltılık gösterimde belirtmek için kullanılır. Örneğin #ff0000, rengin kırmızı bileşeninin 256 bit (ff) ve rengin karşılık gelen yeşil ve mavi kısımlarının 0 (00) olduğu parlak kırmızıdır.

Üç RGB eşleştirmesinin (R, G ve B) her birindeki her iki değer de aynıysa, renk kodu üç karaktere (her eşleştirmenin ilk basamağı) kısaltılabilir. #ff0000, #f00 olarak kısaltılabilir ve #ffffff, #ffff olarak kısaltılabilir.

Hex notasyonu büyük/küçük harfe duyarsızdır.

body {
  background-color: #de1205; /* red */
}

.main {
  background-color: #00f; /* blue */
}

RGB / RGBa #

Bir rengi bildirmenin başka bir yolu da RGB veya RGBa kullanmaktır.

RGB, Kırmızı, Yeşil ve Mavi anlamına gelir ve 0 ile 255 arasında, sırasıyla kırmızı, yeşil ve mavi için ondalık renk değerlerine karşılık gelen, parantezler arasına yerleştirilmiş üç ayrı değer gerektirir.

RGBa, opaklığı tanımlamak için 0.0 ile 1.0 arasında ek bir alfa parametresi eklemenize olanak tanır.

header {
  background-color: rgb(0, 0, 0); /* black */
}

footer {
  background-color: rgba(0, 0, 0, 0.5); /* black with 50% opacity */
}

HSL / HSL #

Bir rengi bildirmenin başka bir yolu da HSL veya HSLa kullanmaktır ve RGB ve RGBa’ya benzer.

HSL, renk tonu, doygunluk ve hafiflik anlamına gelir ve genellikle HLS olarak da adlandırılır:

  • Ton, renk çarkındaki bir derecedir (0’dan 360’a kadar).
  • Doygunluk %0 ile %100 arasında bir yüzdedir.
  • Hafiflik de %0 ile %100 arasında bir yüzdedir.

HSLa, opaklığı tanımlamak için 0.0 ile 1.0 arasında ek bir alfa parametresi eklemenize izin verir.

li a {
  background-color: hsl(120, 100%, 50%); /* green */
}

#p1 {
  background-color: hsla(120, 100%, 50%, .3); /* green with 30% opacity */
}

Arka plan görüntüsüyle etkileşim #

Aşağıdaki ifadelerin tümü eşdeğerdir:

body {
  background: red;
  background-image: url(partiallytransparentimage.png);
}

body {
  background-color: red;
  background-image: url(partiallytransparentimage.png);
}

body {
  background-image: url(partiallytransparentimage.png);
  background-color: red;
}

body {
  background: red url(partiallytransparentimage.png);
}

Hepsi, görüntünün bölümlerinin saydam olduğu veya görüntünün gösterilmediği (belki de ‘arka plan tekrarı’nın bir sonucu olarak) görüntünün altında kırmızı rengin gösterilmesine yol açacaktır.

Aşağıdakilerin eşdeğer olmadığını unutmayın:

body {
  background-image: url(partiallytransparentimage.png);
  background: red;
}

Burada, “arka plan” değeri, “arka plan resminizi” geçersiz kılar.

“background” özelliği hakkında daha fazla bilgi için bkz. https://www.wikiod.com/tr/css/arka-plan#Arka Plan Steno

Arka Plan Gradyanları #

Gradyanlar, CSS3’e eklenen yeni görüntü türleridir. Bir resim olarak, degradeler [background-image](https://www.wikiod.com/tr/css/arka-plan#Arka plan görüntüsü) özelliği veya “background” kısayoluyla ayarlanır.

Doğrusal ve radyal olmak üzere iki tür gradyan işlevi vardır. Her türün yinelenmeyen bir varyantı ve yinelenen bir varyantı vardır:

  • doğrusal-gradyan()
  • yinelenen-doğrusal-gradyan()
  • ‘radyal-gradyan()’
  • yinelenen-radyal-gradyan()

doğrusal gradyan() #

Bir “doğrusal gradyan” aşağıdaki sözdizimine sahiptir

background: linear-gradient( <direction>?, <color-stop-1>, <color-stop-2>, ...);
Değer Anlamı
<yön> ‘yukarıya’, ‘aşağıya’, ‘sağa’ veya ‘sola’ gibi bir argüman olabilir; veya “0deg”, “90deg” olarak bir açı… . Açı yukarıdan başlar ve saat yönünde döner. derece, grad olarak belirtilebilir ), rad veya turn . Atlanırsa, degrade yukarıdan aşağıya doğru akar
<renk-durdurma listesi> İsteğe bağlı olarak her birinin ardından yüzde veya uzunluk ile birlikte görüntülenecek renk listesi. Örneğin, sarı 10%, rgba(0,0,0,.5) 40px, #fff 100%…

Örneğin, bu, sağdan başlayan ve kırmızıdan maviye geçiş yapan doğrusal bir gradyan oluşturur.

.linear-gradient {
  background: linear-gradient(to left, red, blue); /* you can also use 270deg */
}

Hem yatay hem de dikey bir başlangıç ​​konumu bildirerek bir “diyagonal” gradyan oluşturabilirsiniz.

.diagonal-linear-gradient {
  background: linear-gradient(to left top, red, yellow 10%);
}

Bir degradede virgülle ayırarak istediğiniz sayıda renk durağı belirtmek mümkündür. Aşağıdaki örnekler, 8 renk duraklı bir degrade oluşturacaktır.

.linear-gradient-rainbow {
  background: linear-gradient(to left, red, orange, yellow, green, blue, indigo, violet)
}

radyal-gradyan() #

.radial-gradient-simple {
  background: radial-gradient(red, blue);
}

.radial-gradient {
  background: radial-gradient(circle farthest-corner at top left, red, blue);
}
Değer Anlamı
“daire” Gradyan şekli. Değerler “daire” veya “elips”, varsayılan “elips"tir.
’en uzak köşe’ Bitiş şeklinin ne kadar büyük olması gerektiğini açıklayan anahtar kelimeler. Değerler ’en yakın kenar’, ’en uzak kenar’, ’en yakın köşe’, ’en uzak köşe’
‘sol üst’ Degrade merkezinin konumunu, “arka plan konumu” ile aynı şekilde ayarlar.

Yinelenen gradyanlar #

Yinelenen degrade işlevleri, yukarıdaki örneklerle aynı argümanları alır, ancak degradeyi öğenin arka planına döşer.

.bullseye {
  background: repeating-radial-gradient(red, red 10%, white 10%, white 20%);
}
.warning {
  background: repeating-linear-gradient(-45deg, yellow, yellow 10%, black 10%, black 20% );
}
Değer Anlamı
-45deg Açı birimi. Açı yukarıdan başlar ve saat yönünde döner. derece, grad olarak belirtilebilir ), rad veya turn .
sola Gradyan yönü, varsayılan olarak “alt” şeklindedir. Sözdizimi: to [y ekseni(üst VEYA alt)] [x ekseni(sol VEYA sağ)] yani sağ üst
sarı 10% Renk, isteğe bağlı olarak, ardından görüntülenecek bir yüzde veya uzunluk. İki veya daha fazla kez tekrarlandı.

Renk adları yerine HEX, RGB, RGBa, HSL ve HSLa renk kodlarının kullanılabileceğini unutmayın. Renk adları örnek olsun diye kullanılmıştır. Ayrıca radyal-gradyan sözdiziminin lineer-gradyan sözdiziminden çok daha karmaşık olduğunu ve burada basitleştirilmiş bir versiyonun gösterildiğini unutmayın. Tam açıklama ve teknik özellikler için MDN Dokümanlarına bakın

Arka plan görüntüsü #

‘background-image’ özelliği, eşleşen tüm öğelere uygulanacak bir arka plan görüntüsünü belirtmek için kullanılır. Varsayılan olarak, bu görüntü kenar boşluğu hariç tüm öğeyi kaplayacak şekilde döşenir.

.myClass {
  background-image: url('/path/to/image.jpg');
}

Birden çok görüntüyü ‘arka plan görüntüsü’ olarak kullanmak için virgülle ayrılmış ‘url()’ tanımlayın

.myClass {
  background-image: url('/path/to/image.jpg'),
                    url('/path/to/image2.jpg');
}

Görüntüler, ilk beyan edilen görüntü diğerlerinin üzerinde olacak şekilde sıralarına göre yığılır ve bu böyle devam eder.

Değer | Sonuç —————————-|——————— ———– url('/path/to/image.jpg') | Arka plan görüntüsünün yollarını veya veri URI şemasıyla belirtilen bir görüntü kaynağını belirtin (kesme işaretleri atlanabilir), katları virgülle ayırın “hiç” | Arka plan resmi yok “ilk” | Varsayılan değer ‘miras’ | Ebeveynin değerini devral

Arka Plan Resmi için Daha Fazla CSS

Bu aşağıdaki nitelikler çok kullanışlıdır ve neredeyse gereklidir.

background-size:     xpx ypx | x% y%;
background-repeat:   no-repeat | repeat | repeat-x | repeat-y;
background-position: left offset (px/%) right offset (px/%) | center center | left top | right bottom;

Arka Plan Steno #

“background” özelliği, arka planla ilgili bir veya daha fazla özelliği ayarlamak için kullanılabilir:

Değer Açıklama CSS Ver.
‘arka plan resmi’ Kullanılacak arka plan resmi 1+
‘arka plan rengi’ Uygulanacak arka plan rengi 1+
‘arka plan konumu’ Arka plan resminin konumu 1+
‘arka plan boyutu’ Arka plan resminin boyutu 3+
‘arka plan-tekrar’ Arka plan resmi nasıl tekrarlanır 1+
‘arka plan kökenli’ Arka plan nasıl konumlandırılır (“arka plan eki” “sabit” olduğunda yoksayılır) 3+
‘arka plan klibi’ ‘içerik kutusu’, ‘border-box’ veya ‘dolgu kutusu’na göre arka plan nasıl boyanır 3+
‘arka plan eki’ Arka plan görüntüsünün nasıl davrandığı, içerdiği blokla birlikte kayan veya görünüm alanı içinde sabit bir konuma sahip olup olmadığı 1+
“ilk” Özelliği varsayılan değere ayarlar 3+
‘miras’ Üst öğeden özellik değerini devralır 2+

Değerlerin sırası önemli değildir ve her değer isteğe bağlıdır

Sözdizimi #

Arka plan stenografi bildiriminin sözdizimi şöyledir:

background: [<background-image>] [<background-color>]  [<background-position>]/[<background-size>] [<background-repeat>] [<background-origin>] [<background-clip>] [<background-attachment>] [<initial|inherit>];

Örnekler #

background: red;

Basitçe “red” değeriyle bir “arka plan rengi” ayarlayın.

background: border-box red;

Bir “arka plan klibi"ni kenarlık kutusuna ve bir “arka plan rengi"ni kırmızıya ayarlama.

background: no-repeat center url("somepng.jpg");

Bir “arka plan-tekrar"ı tekrarsız olarak, “arka plan-kökeni"ni merkeze ve bir “arka plan resmi"ni bir görüntüye ayarlar.

background: url('pattern.png') green;

Bu örnekte, öğenin ‘arka plan rengi’, varsa ‘pattern.png’ ile ‘yeşil’ olarak ayarlanır, varsa rengin üzerine bindirilir ve öğeyi doldurmak için gerektiği kadar tekrarlanır. “pattern.png” herhangi bir saydamlık içeriyorsa, arkasında “yeşil” renk görünür.

background: #000000 url("picture.png") top left / 600px auto no-repeat;

Bu örnekte, üstünde ‘picture.png’ resmi bulunan siyah bir arka planımız var, resim her iki eksende de tekrarlanmıyor ve sol üst köşede konumlanıyor. Konumdan sonraki /, bu durumda 600 piksel genişlik ve yükseklik için otomatik olarak ayarlanan arka plan görüntüsünün boyutunu içerebilecektir. Bu örnek, düz bir renge dönüşebilen bir özellik görüntüsü ile iyi çalışabilir.

NOT: Steno arka plan özelliğinin kullanılması, bir değer verilmemiş olsa bile önceden ayarlanmış tüm arka plan özellik değerlerini sıfırlar. Yalnızca önceden ayarlanmış bir arka plan özelliği değerini değiştirmek istiyorsanız, bunun yerine bir uzun el özelliği kullanın.

Arka Plan Boyutu #

Genel Bakış #

background-size özelliği, kişinin background-imagenin ölçeklemesini kontrol etmesini sağlar. Ortaya çıkan görüntünün ölçeğini/boyutunu dikey ve yatay yönde belirleyen en fazla iki değer alır. Özellik eksikse, hem “genişlik” hem de “yükseklik” olarak “otomatik” olarak kabul edilir.

“auto”, belirlenebilirse görüntünün en boy oranını koruyacaktır. Yükseklik isteğe bağlıdır ve “otomatik” olarak kabul edilebilir. Bu nedenle, 256 px times; 256 px bir resimde, aşağıdaki tüm “arka plan boyutu” ayarları, yüksekliği ve genişliği 50 px olan bir resim verir:

background-size: 50px;
background-size: 50px auto; /* same as above */
background-size: auto 50px;
background-size: 50px 50px;

Bu nedenle, aşağıdaki resimle başlarsak (belirtilen boyutu 256 px × 256 px),

kullanıcı ekranında, öğemizin arka planında yer alan bir 50 px time; 50 px ile sonuçlanacağız:

Görüntüyü öğeye göre ölçeklendirmek için yüzde değerleri de kullanılabilir. Aşağıdaki örnek, 200 px × 133 px çizilmiş bir resim verir:

#withbackground {
    background-image: url(to/some/background.png);

    background-size: 100% 66%;
    
    width: 200px;
    height: 200px;

    padding: 0;
    margin: 0;
}

Davranış, background-origin öğesine bağlıdır.

En boy oranını korumak #

Önceki bölümdeki son örnek orijinal en boy oranını kaybetti. Daire bir elipse, kare bir dikdörtgene, üçgen başka bir üçgene dönüştü.

Uzunluk veya yüzde yaklaşımı, en boy oranını her zaman tutacak kadar esnek değildir. Öğenizin hangi boyutunun daha büyük olacağını bilemeyebileceğiniz için “auto” yardımcı olmaz. Ancak, belirli alanları bir görüntüyle (ve doğru en boy oranıyla) tamamen kaplamak veya bir arka plan alanında tamamen doğru en boy oranına sahip bir görüntü içermek için, “içerir” ve “kapak” değerleri ek işlevsellik sağlar.

“içerir” ve “kapak” için yumurta açıklaması #

Kötü kelime oyunu için özür dileriz, ancak tanıtım için Biswarup Ganguly tarafından hazırlanan günün resmi kullanacağız. Diyelim ki bu sizin ekranınız ve gri alan görünür ekranınızın dışında. Gösterim için, 16 x; 9 bir oran kabul edeceğiz.

Günün yukarıda belirtilen resmini arka plan olarak kullanmak istiyoruz. Ancak, bir nedenden dolayı görüntüyü 4x3’e kırptık. ‘background-size’ özelliğini sabit bir uzunluğa ayarlayabiliriz, ancak ‘contain’ ve ‘cover’a odaklanacağız. Ayrıca, “gövde"nin genişliğini ve/veya yüksekliğini karıştırmadığımızı varsaydığımı da unutmayın.

“içerir” #

içermek Görüntüyü, kendi en boy oranını (varsa) koruyarak, hem genişliği hem de yüksekliği arka plan konumlandırma alanına sığacak şekilde en büyük boyuta ölçeklendirin.

Bu, arka plan görüntüsünün her zaman arka plan konumlandırma alanında tamamen yer almasını sağlar, ancak bu durumda ‘arka plan renginizle’ doldurulmuş bir miktar boş alan olabilir:

‘kapak’ #

kapak Görüntüyü, kendi en boy oranını (varsa) koruyarak, hem genişliği hem de yüksekliği arka plan konumlandırma alanını tamamen kapsayabilecek şekilde en küçük boyuta ölçeklendirin.

Bu, arka plan görüntüsünün her şeyi kapsamasını sağlar. Görünür bir “arka plan rengi” olmayacak, ancak ekranın oranına bağlı olarak görüntünüzün büyük bir kısmı kesilebilir:

Arka Plan Konumu #

background-position özelliği, bir arka plan görüntüsü veya degrade için başlangıç ​​konumunu belirtmek için kullanılır

.myClass {
  background-image: url('path/to/image.jpg');
  background-position: 50% 50%;
}

Konum, bir X ve Y koordinatı kullanılarak ayarlanır ve CSS içinde kullanılan birimlerden herhangi biri kullanılarak ayarlanabilir.

Birim Açıklama
değer% değer% Yatay öteleme için bir yüzde, (arka plan konumlandırma alanının genişliği - arka plan görüntüsünün genişliği) değerine göredir.
Dikey öteleme için bir yüzde, (arka plan konumlandırma alanının yüksekliği - arka plan görüntüsünün yüksekliği) değerine göredir.
Görüntünün boyutu, ‘background-size’ tarafından verilen boyuttur.
değerpx değerpx Arka plan konumlandırma alanının sol üst köşesine göre piksel cinsinden verilen bir uzunluğa göre arka plan görüntüsünü kaydırır

CSS’deki birimler farklı yöntemlerle belirlenebilir (bkz. burada).

Uzunlamasına Arka Plan Konumu Özellikleri #

Yukarıdaki stenografi özelliğine ek olarak, ‘background-position-x’ ve ‘background-position-y’ uzun el arka plan özellikleri de kullanılabilir. Bunlar, x veya y konumlarını ayrı ayrı kontrol etmenizi sağlar.

Arka Plan Eki #

background-attachment özelliği, bir arka plan görüntüsünün sabit mi yoksa sayfanın geri kalanıyla birlikte mi kaydırılacağını ayarlar.

body { 
  background-image: url('img.jpg');
  background-attachment: fixed;
}
Değer Tanım
kaydırma Arka plan, öğeyle birlikte kayar. Bu varsayılandır.
sabit Arka plan, görünüm alanına göre sabitlenir.
yerel Arka plan, öğenin içeriğiyle birlikte kayar.
başlangıç ​​ Bu özelliği varsayılan değerine ayarlar.
miras Bu özelliği üst öğesinden devralır.

Örnekler #

arka plan eki: kaydırma #

Gövde kaydırıldığında varsayılan davranış, arka plan onunla birlikte kayar:

body {
  background-image: url('image.jpg');
  background-attachment: scroll;
}

arka plan eki: sabit #

Arka plan resmi sabitlenecek ve gövde kaydırıldığında hareket etmeyecek:

body {
  background-image: url('image.jpg');
  background-attachment: fixed;
}

arka plan eki: yerel #

Div’in içeriği kaydırıldığında div’in arka plan görüntüsü kayar.

div {
  background-image: url('image.jpg');
  background-attachment: local;
}

Çoklu Arka Plan Resmi #

CSS3’te, aynı öğede birden fazla arka planı istifleyebiliriz.

#mydiv {
  background-image: url(img_1.png), /* top image */
                    url(img_2.png), /* middle image */
                    url(img_3.png); /* bottom image */
  background-position: right bottom,
                       left top,
                       right top;
  background-repeat: no-repeat,
                     repeat,
                     no-repeat;
}

Görüntüler, ilk arka plan üstte ve son arka plan arkada olacak şekilde üst üste yığılacaktır. “img_1” üstte olacak, “img_2” ve “img_3” altta olacak.

Bunun için arka plan stenografi özelliğini de kullanabiliriz:

#mydiv {
  background: url(img_1.png) right bottom no-repeat,
              url(img_2.png) left top repeat,
              url(img_3.png) right top no-repeat;
}

Ayrıca görüntüleri ve degradeleri de istifleyebiliriz:

#mydiv {
  background: url(image.png) right bottom no-repeat,
              linear-gradient(to bottom, #fff 0%,#000 100%);
}

Opaklık ile Arka Plan Rengi #

Bir öğeye ‘opaklık’ ayarlarsanız, tüm alt öğelerini etkiler. Yalnızca bir öğenin arka planında bir opaklık ayarlamak için RGBA renklerini kullanmanız gerekir. Aşağıdaki örnek, 0,6 opaklığa sahip siyah bir arka plana sahip olacaktır.

/* Fallback for web browsers that don't support RGBa */
background-color: rgb(0, 0, 0);

/* RGBa with 0.6 opacity */
background-color: rgba(0, 0, 0, 0.6);

/* For IE 5.5 - 7*/
filter: progid:DXImageTransform.Microsoft.gradient(startColorstr=#99000000, endColorstr=#99000000);

/* For IE 8*/
-ms-filter: "progid:DXImageTransform.Microsoft.gradient(startColorstr=#99000000, endColorstr=#99000000)";
