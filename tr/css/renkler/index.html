Sözdizimi #

  • renk: #rgb
  • renk: #rrggbb
  • renk: rgb[a](<kırmızı>, <yeşil>, <mavi>[, <alpha>])
  • renk: hsl[a](<hue>, <doygunluk%>, <açıklık%>[, <alpha>])
  • renk: [colorkeyword](https://www.wikiod.com/tr/css/renkler#Renk Anahtar Kelimeleri) /* yeşil, mavi, sarı, turuncu, kırmızı, ..etc */

akımRenk #

“currentColor”, geçerli öğenin hesaplanan renk değerini döndürür.

Aynı öğede kullanın #

Burada currentColor, “renk” özelliği “kırmızı” olarak ayarlandığından kırmızı olarak değerlendirilir:

div {
   color: red;     
   border: 5px solid currentColor;
   box-shadow: 0 0 5px currentColor;
}

Bu durumda, kenarlık için currentColor belirtilmesi büyük olasılıkla gereksizdir, çünkü bunu atlamak aynı sonuçları üretecektir. [daha spesifik](https://www.wikiod.com/tr/css/basamakl-ve-ozgulluk#Seçici Özgüllüğünün Hesaplanması) nedeniyle üzerine yazılacaksa, aynı öğe içinde border özelliğinin içinde yalnızca currentColor kullanın. -specity#t=201607221033325129018) seçici.

Hesaplanan renk olduğu için, birinci kuralı geçersiz kılan ikinci kural nedeniyle aşağıdaki örnekte kenarlık yeşil olacaktır:

div {
   color: blue;
   border: 3px solid currentColor; 
   color: green;
}

Üst öğeden devralındı #

Ebeveynin rengi miras alınır, burada currentColor ‘mavi’ olarak değerlendirilir ve alt öğenin kenarlık rengini mavi yapar.

.parent-class {
    color: blue;
}

.parent-class .child-class {
    border-color: currentColor;
}

currentColor, normalde color özelliğinden miras alınmayacak, örneğin background-color gibi diğer kurallar tarafından da kullanılabilir. Aşağıdaki örnek, arka plan olarak ebeveyndeki renk setini kullanan çocukları göstermektedir:

.parent-class {
    color: blue;
}

.parent-class .child-class {
    background-color: currentColor;
}

Renkli Anahtar Kelimeler #

Çoğu tarayıcı, bir renk belirtmek için renkli anahtar sözcüklerin kullanılmasını destekler. Örneğin, bir öğenin “rengi"ni maviye ayarlamak için “mavi” anahtar sözcüğünü kullanın:

.some-class {
    color: blue;
}

CSS anahtar sözcükleri büyük/küçük harfe duyarlı değildir; “mavi”, “Mavi” ve “MAVİ”, tümü “#0000FF” ile sonuçlanır.

Onaltılık Değer #

Arka fon #

CSS renkleri, üyelerin bir rengin kırmızı, yeşil ve mavi bileşenlerini temsil ettiği altıgen üçlü olarak da temsil edilebilir. Bu değerlerin her biri, ondalık gösterimde “00” ila “FF” veya “0” ila “255” aralığında bir sayıyı temsil eder. Büyük harf ve/veya küçük harf Onaltılık değerler kullanılabilir (yani “#3fc” = “#3FC” = “#33ffCC”). Tarayıcı “#369"u “#336699” olarak yorumlar. ‘#306090’ istediğiniz bu değilse, bunu açıkça belirtmeniz gerekir.

Altıgen gösterimle temsil edilebilecek toplam renk sayısı 256 ^ 3 veya 16.777.216’dır.

Sözdizimi #

color: #rrggbb;
color: #rgb
Değer Açıklama
rr 00 - FF kırmızı miktarı için
‘gg’ yeşil miktarı için 00 - FF
bb 00 - FF mavi miktarı için 
.some-class {
    /* This is equivalent to using the color keyword 'blue' */
    color: #0000FF;
}

.also-blue {
    /* If you want to specify each range value with a single number, you can!
       This is equivalent to '#0000FF' (and 'blue') */
    color: #00F;
}

Onaltılık gösterim, W3C’nin ‘Sayısal renk değerleri’ uyarınca RGB renk biçiminde renk değerlerini belirtmek için kullanılır. org/TR/css3-color/#numerical).

İnternette onaltılık (veya sadece onaltılık) renk değerlerini aramak için birçok araç bulunmaktadır.

Bir sürü seçenek bulmak için favori web tarayıcınızla “altıgen renk paleti” veya “altıgen renk seçici” arayın!

Onaltılık değerler her zaman bir sayı işaretiyle (#) başlar, en fazla altı “basamak” uzunluğundadır ve büyük/küçük harfe duyarlı değildir: yani, büyük harf kullanımı umurlarında değildir. #FFC125 ve #ffc125 aynı renktedir.

rgb() Gösterimi #

RGB, renkleri kırmızı, yeşil ve mavi ışığın karışımları olarak temsil eden ilave bir renk modelidir. Özünde, RGB gösterimi, Onaltılı Gösterimin ondalık eşdeğeridir. Onaltılık sistemde her sayı 00-FF arasında değişir, bu da ondalık olarak 0-255 ve yüzde olarak %0-100’e eşittir.

.some-class {
    /* Scalar RGB, equivalent to 'blue'*/
    color: rgb(0, 0, 255);
}

.also-blue {
    /* Percentile RGB values*/
    color: rgb(0%, 0%, 100%);
}

Sözdizimi #

rgb(<red>, <green>, <blue>)
Değer Açıklama
<kırmızı> 0 - 255 arasında bir tam sayı veya 0 - 100 arasında yüzde
<yeşil> 0 - 255 arasında bir tam sayı veya 0 - 100 arasında yüzde
<mavi> 0 - 255 arasında bir tam sayı veya 0 - 100 arasında yüzde

rgba() Gösterimi #

[rgb() notasyonuna](https://www.wikiod.com/tr/css/renkler#rgb() Gösterimi) benzer, ancak ek bir alfa (opaklık) değerine sahiptir.

.red {
    /* Opaque red */
    color: rgba(255, 0, 0, 1);
}

.red-50p {
    /* Half-translucent red. */
    color: rgba(255, 0, 0, .5);
}

Sözdizimi #

rgba(<red>, <green>, <blue>, <alpha>);
Değer Açıklama
<kırmızı> 0 - 255 arasında bir tam sayı veya 0 - 100 arasında yüzde
<yeşil> 0 - 255 arasında bir tam sayı veya 0 - 100 arasında yüzde
<mavi> 0 - 255 arasında bir tam sayı veya 0 - 100 arasında yüzde
<alfa> 0 - 1 arasında bir sayı, burada 0.0 tamamen şeffaf ve 1.0 tamamen opaktır

hsl() Gösterimi #

HSL, renk tonu (“hangi renk”), doygunluk (“ne kadar renk”) ve hafiflik (“ne kadar beyaz”) anlamına gelir.

Ton, 0° ila 360° (birimler olmadan) bir açı olarak temsil edilirken, doygunluk ve açıklık yüzde olarak temsil edilir.

p {
    color: hsl(240, 100%, 50%); /* Blue */
}

Sözdizimi #

color: hsl(<hue>, <saturation>%, <lightness>%);
Değer Açıklama
<renk> 0° kırmızı, 60° sarı, 120° yeşil, 180° camgöbeği, 240° mavi, 300° macenta ve 360° kırmızı olmak üzere renk çarkı çevresinde derece cinsinden belirtilir (birimler olmadan).
<doygunluk> %0’ın tamamen doymamış (gri tonlamalı) ve %100’ün tamamen doygun (canlı renkli) olduğu yüzde olarak belirtilir
<hafiflik> %0’ın tamamen siyah ve %100’ün tamamen beyaz olduğu yüzde olarak belirtilir

Notlar #

  • %0 doygunluk her zaman gri tonlamalı bir renk üretir; tonu değiştirmenin bir etkisi yoktur.
  • %0’lık bir açıklık her zaman siyahı, %100’ü ise her zaman beyazı üretir; tonu veya doygunluğu değiştirmenin hiçbir etkisi yoktur.

hsla() Gösterimi #

[hsl() gösterimine](https://www.wikiod.com/tr/css/renkler#hsl() Gösterimi) benzer, ancak alfa (opaklık) değeri eklenmiştir.

hsla(240, 100%, 50%, 0)     /* transparent */
hsla(240, 100%, 50%, 0.5)   /* half-translucent blue */
hsla(240, 100%, 50%, 1)     /* fully opaque blue */

Sözdizimi #

hsla(<hue>, <saturation>%, <lightness>%, <alpha>);
Değer Açıklama
<renk> 0° kırmızı, 60° sarı, 120° yeşil, 180° camgöbeği, 240° mavi, 300° macenta ve 360° kırmızı olmak üzere renk çarkı çevresinde derece cinsinden belirtilir (birimler olmadan).
<doygunluk> %0’ın tamamen doymamış (gri tonlamalı) ve %100’ün tamamen doygun (canlı renkli) olduğu yüzde
<hafiflik> %0’ın tamamen siyah ve %100’ün tamamen beyaz olduğu yüzde
<alfa> 0’ın tamamen şeffaf ve 1’in tamamen opak olduğu 0 - 1 arasında bir sayı
