Bir CSS mesafe ölçümü, hemen ardından bir uzunluk birimi (px, em, pc, in, …) gelen bir sayıdır.

CSS, bir dizi uzunluk ölçüm birimini destekler. Bunlar mutlak veya görecelidir.

Sözdizimi #

  • değerbirim
  • 1 saat

Parametreler #

Birim Açıklama
% Boyutları, özelliğe bağlı olarak üst nesneler veya geçerli nesne açısından tanımlayın
em Öğenin yazı tipi boyutuna göre (2em, geçerli yazı tipinin boyutunun 2 katı anlamına gelir)
geri Kök öğenin yazı tipi boyutuna göre
vw Görüntü alanının genişliğinin %1’ine göre*
vh Görünüm alanının yüksekliğinin %1’ine göre*
vmin Görünüm alanının* küçük boyutunun %1’ine göre
vmax Görünüm alanının* daha büyük boyutunun %1’ine göre
cm santimetre
mm milimetre
içinde inç (1 inç = 96 piksel = 2,54 cm)
piksel piksel (1 piksel = 1/96 / 1 inç)
puan puan (1 puan = 1/72 / 1in)
bilgisayar pika (1 adet = 12 pt)
s saniye (animasyonlar ve geçişler için kullanılır)
ms milisaniye (animasyonlar ve geçişler için kullanılır)
eski Geçerli yazı tipinin x yüksekliğine göre
ch Sıfır (0) karakterinin genişliğine göre
fr kesirli birim (CSS Izgara Düzeni için kullanılır)

  • Sayı ve birim arasında boşluk görünemez. Ancak değer 0 ise, birim atlanabilir.

  • Bazı CSS özellikleri için negatif uzunluklara izin verilir.

rems ve ems kullanarak ölçeklenebilir öğeler oluşturma #

Öğeleri biçimlendirmek için, “font-size” değerlerini “rem” değerine ayarlayarak “html” etiketinizin “font-size” tarafından tanımlanan “rem"i kullanabilir ve öğeleri oluşturmak için “em"i öğenin içinde kullanabilirsiniz. global font-size ile ölçeklendirin.

HTML:

<input type="button" value="Button">
<input type="range">
<input type="text" value="Text">

İlgili CSS:

html {
  font-size: 16px;
}

input[type="button"] {
  font-size: 1rem;
  padding: 0.5em 2em;
}

input[type="range"] {
  font-size: 1rem;
  width: 10em;
}

input[type=text] {
  font-size: 1rem;
  padding: 0.5em;
}

Olası Sonuç:

son sürüm

rem ile yazı tipi boyutu #

CSS3, “root em” anlamına gelen rem birimi dahil olmak üzere birkaç yeni birim sunar. Şimdi ‘rem’in nasıl çalıştığına bakalım.

İlk olarak, “em” ve “rem” arasındaki farklara bakalım.

  • em: Üst öğenin yazı tipi boyutuna göre. Bu bileşik soruna neden olur
  • rem: Kökün veya “” öğesinin yazı tipi boyutuna göre. Bu, html öğesi için tek bir yazı tipi boyutu bildirmenin ve tüm ‘rem’ birimlerini bunun bir yüzdesi olarak tanımlamanın mümkün olduğu anlamına gelir.

Yazı tipi boyutlandırma için ‘rem’ kullanmanın temel sorunu, değerlerin kullanımının biraz zor olmasıdır. Temel boyutun 16 piksel olduğu varsayılarak, “rem” birimlerinde ifade edilen bazı yaygın yazı tipi boyutlarına bir örnek:

  • 10 piksel = 0.625 rem
  • 12 piksel = 0.75 rem
  • 14 piksel = 0.875 rem
  • 16 piksel = 1rem (taban)
  • 18 piksel = 1.125 rem
  • 20 piksel = 1.25 rem
  • 24 piksel = 1.5 rem
  • 30 piksel = 1.875 rem
  • 32 piksel = 2rem

KOD:

html { 
  font-size: 16px; 
}

h1 { 
  font-size: 2rem;          /* 32px */
}

p { 
  font-size: 1rem;          /* 16px */ 
}

li { 
  font-size: 1.5em;         /* 24px */
}
son sürüm

vmin ve vmax #

  • vmin: Görüntü alanının daha küçük boyutunun yüzde 1’ine göre
  • vmax: Görüntü alanının daha büyük boyutunun yüzde 1’ine göre

Başka bir deyişle, 1 vmin, 1 vh ve 1 vw’nin küçüğüne eşittir.

1 “vmax”, 1 vh ve 1 vw değerlerinden büyük olanına eşittir

Not: “vmax” desteklenmez:

  • Internet Explorer’ın herhangi bir sürümü
  • 6.1 sürümünden önceki Safari

vh ve vw #

CSS3, boyutu temsil etmek için iki birim tanıttı.

  • “Görüntü alanı yüksekliği” anlamına gelen “vh”, görüntü alanı yüksekliğinin %1’ine göredir
  • “Görüntü alanı genişliği” anlamına gelen “vw”, görüntü alanı genişliğinin %1’ine göredir
div { 
  width: 20vw; 
  height: 20vh;
}
son sürüm

Yukarıda, div boyutu, görünümün genişliğinin ve yüksekliğinin %20’sini kaplar.

yüzde yüzdesi kullanılıyor #

Duyarlı bir uygulama oluştururken kullanışlı birimlerden biri.

Boyutu, ana kabına bağlıdır.

Denklem:

( Ana Kapsayıcının genişliği ) * ( Yüzde(%) ) = Çıktı

Örneğin:

Ebeveyn 100px genişliğe sahipken Çocuk %50 genişliğe sahiptir.

Çıktıda, Alt öğesinin genişliği Üstün genişliğinin yarısı (%50), yani 50 piksel olacaktır.

HTML

<div class="parent">
   PARENT
   <div class="child">
     CHILD
   </div>
</div>

CSS

<style>

*{
  color: #CCC;
}

.parent{
  background-color: blue;
  width: 100px;
}

.child{
  background-color: green;
  width: 50%;
}

</style>
