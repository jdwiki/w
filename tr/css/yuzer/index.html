Metnin İçinde Bir Resmi Kaydırın #

Bir şamandıranın en temel kullanımı, bir görüntünün etrafına metin sarmaktır. Aşağıdaki kod, ikinci paragraf görüntünün etrafında akan iki paragraf ve bir görüntü üretecektir. Yüzen öğenin etrafından akan kayan öğeden her zaman sonra içerik olduğuna dikkat edin.

HTML:

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio. Praesent libero. Sed cursus ante dapibus diam. Sed nisi. Nulla quis sem at nibh elementum imperdiet. Duis sagittis ipsum. Praesent mauris. Fusce nec tellus sed augue semper porta. Mauris massa. Vestibulum lacinia arcu eget nulla. </p>

<img src="http://lorempixel.com/200/100/" />

<p>Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos. Curabitur sodales ligula in libero. Sed dignissim lacinia nunc. Curabitur tortor. Pellentesque nibh. Aenean quam. In scelerisque sem at dolor. Maecenas mattis. Sed convallis tristique sem. Proin ut ligula vel nunc egestas porttitor. Morbi lectus risus, iaculis vel, suscipit quis, luctus non, massa. Fusce ac turpis quis ligula lacinia aliquet. </p>

img {
  float:left;

}

özelliği temizle #

clear özelliği doğrudan yüzenlerle ilgilidir. Mülk değerleri:

  • yok - Varsayılan. Her iki tarafta yüzen elemanlara izin verir

  • sol - Sol tarafta kayan öğelere izin verilmez

  • sağ - Sağ tarafta kayan öğelere izin verilmez

  • her ikisi - Sol veya sağ tarafta kayan öğelere izin verilmez

  • ilk - Bu özelliği varsayılan değerine ayarlar. Başlangıç ​​hakkında bilgi edinin

  • inherit - Bu özelliği üst öğesinden devralır. Miras hakkında bilgi edinin

Düzeltme #

Düzeltme hilesi, yüzer ifadeleri içermenin popüler bir yoludur (N. Gallagher, diğer adıyla @necolas)

[clear](https://www.wikiod.com/tr/css/yuzer#mülkü temizle) özelliğiyle karıştırılmaması için, clearfix bir kavramdır (aynı zamanda yüzenlerle ilgilidir, olası karışıklık). Float’ları _içermek için, kapsayıcıya (üst) ‘.cf’ veya ‘.clearfix’ sınıfını eklemeniz ve bu sınıfı aşağıda açıklanan birkaç kuralla stillendirmeniz gerekir.

Biraz farklı efektlere sahip 3 versiyon (kaynaklar :A new micro clearfix hack N. Gallagher ve [clearfix yeniden yüklendi](http://yuiblog.com/ blog/2010/09/27/clearfix-reloaded-overflowhidden-demystified/) T. J. Koblentz tarafından):

Clearfix (içerilen şamandıraların üst kenar boşluğu çökmesi hala meydana geliyor) #

.cf:after {
    content: "";
    display: table;
}

.cf:after {
    clear: both;
}

Clearfix ayrıca, içerilen şamandıraların üst kenar boşluğunun çökmesini de önler #

/**
 * For modern browsers
 * 1. The space content is one way to avoid an Opera bug when the
 *    contenteditable attribute is included anywhere else in the document.
 *    Otherwise it causes space to appear at the top and bottom of elements
 *    that are clearfixed.
 * 2. The use of `table` rather than `block` is only necessary if using
 *    `:before` to contain the top-margins of child elements.
 */
.cf:before,
.cf:after {
    content: " "; /* 1 */
    display: table; /* 2 */
}

.cf:after {
    clear: both;
}

Güncel olmayan IE6 ve IE7 tarayıcılarının desteğiyle Clearfix #

.cf:before,
.cf:after {
    content: " ";
    display: table;
}

.cf:after {
    clear: both;
}

/**
 * For IE 6/7 only
 * Include this rule to trigger hasLayout and contain floats.
 */
.cf {
    *zoom: 1;
}

Kodpen düzeltme efektini gösteriyor

Diğer kaynak: Clearfix hakkında bildiğiniz her şey yanlış (clearfix ve BFC - HasLayout eski tarayıcılar IE6 belki 7 ile ilgiliyken Biçimlendirme Bağlamı’nı Engelle)

Float kullanarak satır içi DIV #

‘div’ blok düzeyinde bir öğedir, yani tüm sayfa genişliğini kaplar ve kardeşler genişliklerinden bağımsız olarak alt alta yerleştirilir.

<div>
    <p>This is DIV 1</p>
</div>
<div>
    <p>This is DIV 2</p>
</div>

Aşağıdaki kodun çıktısı olacaktır

Bunları div’e bir ‘float’ css özelliği ekleyerek in-line yapabiliriz.

HTML:

<div class="outer-div">
    <div class="inner-div1">
        <p>This is DIV 1</p>
    </div>
    <div class="inner-div2">
        <p>This is DIV 2</p>
    </div>
</div>

CSS

.inner-div1 {
    width: 50%;
    margin-right:0px;
    float:left;
    background : #337ab7;
    padding:50px 0px;
}
 
.inner-div2 {
    width: 50%;
    margin-right:0px;
    float:left;
    background : #dd2c00;
    padding:50px 0px;
}
 
p {
    text-align:center;
}

Basit İki Sabit Genişlikli Sütun Düzeni #

Basit bir iki sütunlu düzen, iki sabit genişlikte, kayan öğeden oluşur. Bu örnekte kenar çubuğu ve içerik alanının aynı yükseklikte olmadığına dikkat edin. Bu, kayan noktaları kullanan çok sütunlu mizanpajlara sahip zor parçalardan biridir ve birden çok sütunun aynı yükseklikte görünmesini sağlamak için geçici çözümler gerektirir.

HTML:

<div class="wrapper">

<div class="sidebar">
  <h2>Sidebar</h2>

  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio.</p>
</div>

<div class="content">
  <h1>Content</h1>

  <p>Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos. Curabitur sodales ligula in libero. Sed dignissim lacinia nunc. Curabitur tortor. Pellentesque nibh. Aenean quam. In scelerisque sem at dolor. Maecenas mattis. Sed convallis tristique sem. Proin ut ligula vel nunc egestas porttitor. Morbi lectus risus, iaculis vel, suscipit quis, luctus non, massa. Fusce ac turpis quis ligula lacinia aliquet. </p>
</div>

</div>

CSS:

.wrapper {
  width:600px;
  padding:20px;
  background-color:pink;

  /* Floated elements don't use any height. Adding "overflow:hidden;" forces the
     parent element to expand to contain its floated children. */
  overflow:hidden;
}

.sidebar {
  width:150px;
  float:left;
  background-color:blue;
}

.content {
  width:450px;
  float:right;
  background-color:yellow;
}

<div class="wrapper">

  </div>
  <div class="content">
    <h1>Content</h1>

Aenean quam. In scelerisque sem at dolor. Maecenas mattis. Sed convallis tristique sem. Proin ut ligula vel nunc egestas porttitor. Morbi lectus risus, iaculis vel, suscipit quis, luctus non, massa. </p>
  </div>
  <div class="right-sidebar">
    <h1>Right Sidebar</h1>
    <p>Fusce ac turpis quis ligula lacinia aliquet.</p>
  </div>
</div>

CSS:

.wrapper {
  width:600px;
  background-color:pink;
  padding:20px;

  /* Floated elements don't use any height. Adding "overflow:hidden;" forces the
     parent element to expand to contain its floated children. */
  overflow:hidden;
}

.left-sidebar {
  width:150px;
  background-color:blue;
  float:left;
}

.content {
  width:300px;
  background-color:yellow;
  float:left;
}

.right-sidebar {
  width:150px;
  background-color:green;
  float:right;
}

İki Sütunlu Tembel/Açgözlü Düzen #

Bu düzen, tanımlanmış genişlikleri olmayan iki sütunlu bir düzen oluşturmak için bir kayan sütun kullanır. Bu örnekte, sol kenar çubuğu “tembeldir”, çünkü yalnızca ihtiyaç duyduğu kadar yer kaplar. Bunu söylemenin başka bir yolu, sol kenar çubuğunun “küçültülmüş” olmasıdır. Sağdaki içerik sütunu, kalan tüm alanı kaplaması bakımından “açgözlüdür”.

HTML:

<div class="sidebar">
<h1>Sidebar</h1>
<img src="http://lorempixel.com/150/200/" />
</div>

<div class="content">
<h1>Content</h1>
<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio. Praesent libero. Sed cursus ante dapibus diam. Sed nisi. Nulla quis sem at nibh elementum imperdiet. Duis sagittis ipsum. Praesent mauris. Fusce nec tellus sed augue semper porta. Mauris massa. Vestibulum lacinia arcu eget nulla. </p>
<p>Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos. Curabitur sodales ligula in libero. Sed dignissim lacinia nunc. Curabitur tortor. Pellentesque nibh. Aenean quam. In scelerisque sem at dolor. Maecenas mattis. Sed convallis tristique sem. Proin ut ligula vel nunc egestas porttitor. Morbi lectus risus, iaculis vel, suscipit quis, luctus non, massa. Fusce ac turpis quis ligula lacinia aliquet. Mauris ipsum. Nulla metus metus, ullamcorper vel, tincidunt sed, euismod in, nibh. </p>
</div>

CSS:

.sidebar {
  /* `display:table;` shrink-wraps the column */
  display:table;
  float:left;
  background-color:blue;
}

.content {
  /* `overflow:hidden;` prevents `.content` from flowing under `.sidebar` */
  overflow:hidden;
  background-color:yellow;
}

Floatları temizlemek için taşma özelliğinin kullanımı #

Bir öğeye “taşma” değerini “gizli”, “otomatik” veya “kaydır” olarak ayarlamak, o öğe içindeki tüm kayan noktaları temizleyecektir.

Not: overflow:scroll kullanımı her zaman kaydırma kutusunu gösterecektir
