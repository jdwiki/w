Sözdizimi #

  • sayaç seti: [ <counter-name> <integer>? ]+ | Yok

  • sayaç sıfırlama: [ <counter-name> <integer>? ]+ | Yok

  • karşı artış: [ <counter-name> <integer>? ]+ | Yok

  • sayaç(<counter-name> [, <counter-style> ]?)

  • sayaçlar(<counter-name>, <connector-string> [, <counter-style> ]?)

Parametreler #

parametre Ayrıntılar
karşı isim Oluşturulması veya arttırılması veya yazdırılması gereken sayacın adıdır. Geliştiricinin istediği gibi herhangi bir özel ad olabilir.
tamsayı Bu tam sayı, sayaç adının yanında sağlandığında, sayacın başlangıç ​​değerini (“sayaç-set”, “sayaç-sıfırlama” özelliklerinde) veya sayacın artırılması gereken değeri (’’ olarak) temsil edecek isteğe bağlı bir değerdir. karşı artış’).
hiçbiri Bu, tüm 3 counter-* özelliği için başlangıç ​​değeridir. Bu değer ‘sayaç artışı’ için kullanıldığında, sayaçların hiçbirinin değeri etkilenmez. Bu, diğer ikisi için kullanıldığında, sayaç oluşturulmaz.
karşı tarz Bu, sayaç değerinin görüntülenmesi gereken stili belirtir. ’list-style-type’ özelliği tarafından desteklenen tüm değerleri destekler. “none” kullanılırsa sayaç değeri hiç yazdırılmaz.
bağlayıcı-string Bu, iki farklı sayaç seviyesinin değerleri arasına yerleştirilmesi gereken dizeyi temsil eder (“2.1.1"deki “.” gibi).

Sayaçlar, CSS’de yeni bir konu değildir. CSS Seviye 2 Spesifikasyonlarının (kesin olmak gerekirse Revizyon 1) bir parçasıydı ve bu nedenle çok yüksek tarayıcı desteğine sahip.

IE6 ve IE7 dışındaki tüm tarayıcılar CSS Sayaçlarını destekler.

Sayaç çıktısına roma rakamları stili uygulama #

CSS #

body {
  counter-reset: item-counter;
}

.item {
  counter-increment: item-counter;
}

.item:before {
  content: counter(item-counter, upper-roman) ". "; /* by specifying the upper-roman as style the output would be in roman numbers */
}

HTML #

<div class='item'>Item No: 1</div>
<div class='item'>Item No: 2</div>
<div class='item'>Item No: 3</div>

Yukarıdaki örnekte, geliştirici açıkça sayacın stilini belirlediğinden, sayacın çıktısı olağan 1, 2, 3 yerine I, II, III (roma sayıları) olarak görüntülenecektir.

CSS Sayacı kullanarak her öğeyi numaralandırın #

CSS #

body {
  counter-reset: item-counter; /* create the counter */
}
.item {
  counter-increment: item-counter; /* increment the counter every time an element with class "item" is encountered */
}
.item-header:before {
  content: counter(item-counter) ". "; /* print the value of the counter before the header and append a "." to it */
}

/* just for demo */

.item {
  border: 1px solid;
  height: 100px;
  margin-bottom: 10px;
}
.item-header {
  border-bottom: 1px solid;
  height: 40px;
  line-height: 40px;
  padding: 5px;
}
.item-content {
  padding: 8px;
}

HTML #

<div class='item'>
  <div class='item-header'>Item 1 Header</div>
  <div class='item-content'>Lorem Ipsum Dolor Sit Amet....</div>
</div>
<div class='item'>
  <div class='item-header'>Item 2 Header</div>
  <div class='item-content'>Lorem Ipsum Dolor Sit Amet....</div>
</div>
<div class='item'>
  <div class='item-header'>Item 3 Header</div>
  <div class='item-content'>Lorem Ipsum Dolor Sit Amet....</div>
</div>

Yukarıdaki örnek, sayfadaki her “öğeyi” numaralar ve öğenin numarasını başlığından önce ekler (".item-header” öğesinin “:before” sözde öğesinin “content” özelliğini kullanarak). Bu kodun canlı bir demosunu burada bulabilirsiniz.

CSS sayaçlarını kullanarak çok seviyeli numaralandırmayı uygulama #

CSS #

ul {
  list-style: none;
  counter-reset: list-item-number; /* self nesting counter as name is same for all levels */
}
li {
  counter-increment: list-item-number;
}
li:before {
  content: counters(list-item-number, ".") " "; /* usage of counters() function means value of counters at all higher levels are combined before printing */
}

HTML #

<ul>
  <li>Level 1
    <ul>
      <li>Level 1.1
        <ul>
          <li>Level 1.1.1</li>
        </ul>
      </li>
    </ul>
  </li>
  <li>Level 2
    <ul>
      <li>Level 2.1
        <ul>
          <li>Level 2.1.1</li>
          <li>Level 2.1.2</li>
        </ul>
      </li>
    </ul>
  </li>
  <li>Level 3</li>
</ul>

Yukarıdaki, CSS sayaçlarını kullanan çok seviyeli numaralandırma örneğidir. Sayaçların kendi kendine yuvalama kavramını kullanır. Kendi kendine yuvalama, bir öğenin zaten belirtilen ada sahip bir sayacı varsa, ancak başka bir tane oluşturmak zorundaysa, onu mevcut sayacın çocuğu olarak oluşturduğu bir kavramdır. Burada, ikinci düzey “ul”, “list-item-number” sayacını üst öğesinden zaten devralır, ancak daha sonra kendi “list-item-number"ını (çocukları “li” için) oluşturmak zorundadır ve böylece “list-item-numarası"nı oluşturur. item-number[1](ikinci seviye için sayaç) ve onulist-item-number[0]` (birinci seviye için sayaç) altına yerleştirir. Böylece çok seviyeli numaralandırma elde edilir.

Çıktı, ‘counters()’ işlevi yerine ‘counters()’ işlevi kullanılarak yazdırılır, çünkü ‘counters()’ işlevi, çıktı yazdırılırken tüm daha yüksek seviyeli sayaçların (ana) değerinin önüne eklenecek şekilde tasarlanmıştır.
