CSS Değişkenleri, yazarların bir CSS belgesinde kullanılabilecek yeniden kullanılabilir değerler oluşturmasına olanak tanır.

Örneğin, bir belge boyunca tek bir rengin yeniden kullanılması CSS’de yaygındır. CSS Değişkenlerinden önce bu, bir belge boyunca aynı renk değerinin birçok kez yeniden kullanılması anlamına gelir. CSS Değişkenleri ile renk değeri bir değişkene atanabilir ve birden çok yerde referans alınabilir. Bu, değerleri değiştirmeyi kolaylaştırır ve geleneksel CSS değerlerini kullanmaktan daha anlamsaldır.

Sözdizimi #

  • :root {} /* değişkenlerin daha genel tanımını sağlayan sözde sınıf */
  • –değişken-adı: değer; /* değişken tanımla */
  • var(–değişken-adı, varsayılan-değer) /* tanımlı değişkeni varsayılan değerle kullan geri dönüş */

CSS Değişkenleri şu anda deneysel bir teknoloji olarak kabul edilmektedir.

TARAYICI DESTEĞİ / UYUMLULUK

Firefox: Sürüm 31+ (Varsayılan olarak etkindir)

Mozilla’dan daha fazla bilgi

Chrome: Sürüm 49+ (Varsayılan olarak etkindir).

“Bu özellik, ‘deneysel Web Platformu’ özelliği etkinleştirilerek test için Chrome Sürüm 48’de etkinleştirilebilir. Bu ayara erişmek için Chrome adres çubuğunuza ‘chrome://flags/’ yazın.”

IE: Desteklenmiyor.

Edge: Geliştirme Aşamasında

Safari: Sürüm 9.1+

Değişken Renk #

:root {
  --red: #b00;
  --blue: #4679bd;
  --grey: #ddd;
}
.Bx1 {
  color: var(--red);
  background: var(--grey);
  border: 1px solid var(--red);
}

Değişken Boyutlar #

:root {
  --W200: 200px;
  --W10: 10px;
}
.Bx2 {
  width: var(--W200);
  height: var(--W200);
  margin: var(--W10);
}

Değişken Basamaklı #

CSS değişkenleri, diğer özelliklerle aynı şekilde basamaklanır ve güvenli bir şekilde yeniden ifade edilebilir.

Değişkenleri birden çok kez tanımlayabilirsiniz ve seçilen öğeye yalnızca en yüksek özgüllüğe sahip tanım uygulanır.

Bu HTML’yi varsayarsak:

<a class="button">Button Green</a>
<a class="button button_red">Button Red</a>
<a class="button">Button Hovered On</a>

Bu CSS’yi yazabiliriz:

.button {
  --color: green;
  padding: .5rem;
  border: 1px solid var(--color);
  color: var(--color);
}

.button:hover {
  --color: blue;
}

.button_red {
  --color: red;
}

Geçerli/Geçersiz #

Biz CSS değişkenlerini adlandırırken, diğer CSS özellikleri gibi yalnızca harfler ve tireler içerir (örneğin: line-height, -moz-box-sizing) ancak çift tire (–) ile başlamalıdır.

//These are Invalids variable names
--123color: blue;
--#color: red;
--bg_color: yellow
--$width: 100px;

//Valid variable names
--color: red;
--bg-color: yellow
--width: 100px;

CSS Değişkenleri büyük/küçük harfe duyarlıdır.

/* The variable names below are all different variables */
--pcolor: ;
--Pcolor: ;
--pColor: ;

Boş Alana Karşı Boşluk

/* Invalid */
    --color:;

/* Valid */
  --color: ; /* space is assigned */

Birleştirmeler

    /* Invalid - CSS doesn't support concatenation*/
    .logo{
        --logo-url: 'logo';
        background: url('assets/img/' var(--logo-url) '.png');
    }

    /* Invalid - CSS bug */
    .logo{
        --logo-url: 'assets/img/logo.png';
        background: url(var(--logo-url));
    }

    /* Valid */
    .logo{
        --logo-url: url('assets/img/logo.png');
        background: var(--logo-url);
    }

Birimleri kullanırken dikkatli olun

    /* Invalid */
    --width: 10;
    width: var(--width)px;

    /* Valid */
    --width: 10px;
    width: var(--width);

    /* Valid */
    --width: 10;
    width: calc(1px * var(--width)); /* multiply by 1 unit to convert */
    width: calc(1em * var(--width));

Medya sorgularıyla #

Değişkenleri medya sorguları içinde yeniden ayarlayabilir ve bu yeni değerlerin kullanıldığı her yerde basamaklandırılmasını sağlayabilirsiniz; bu, ön işlemci değişkenleriyle mümkün olmayan bir şeydir.

Burada bir medya sorgusu, çok basit bir ızgara oluşturmak için kullanılan değişkenleri değiştirir:

HTML

<div></div>
<div></div>
<div></div>
<div></div>

CSS

:root{
    --width: 25%;
    --content: 'This is desktop';
}
@media only screen and (max-width: 767px){
    :root{
        --width:50%;
        --content: 'This is mobile';
    }
}
@media only screen and (max-width: 480px){
    :root{
        --width:100%;
    }
}

div{
    width: calc(var(--width) - 20px);
    height: 100px;
}
div:before{
    content: var(--content);
}

/* Other Styles */
body {
    padding: 10px;
}

div{
    display: flex;
    align-items: center;
    justify-content: center;
    font-weight:bold;
    float:left;
    margin: 10px;
    border: 4px solid black;
    background: red;
}
