        Mermi Konumu #

        Liste, içeren bir öğenin (’

          1. ’ öğelerinden oluşur. Hem liste öğeleri hem de kap, liste öğesi içeriğinin belgedeki tam konumunu etkileyen kenar boşluklarına ve dolgulara sahip olabilir. Kenar boşluğu ve dolgu için varsayılan değerler her tarayıcı için farklı olabilir. Aynı tarayıcı düzenini elde etmek için bunların özel olarak ayarlanması gerekir.

            Her liste öğesi, madde işareti işaretini içeren bir ‘işaret kutusu’ alır. Bu kutu, liste öğesi kutusunun içine veya dışına yerleştirilebilir.

            list-style-position: inside;

            list-style-position: outside;

            “İçeride” ve “dışta” konumlandırmanın sonucunu gösterme: jsfiddle

            Madde İşareti veya Numaralandırma Türü #

            Sırasız bir liste (<ul>) içindeki <li> etiketlerine özel:

            list-style: disc;                /* A filled circle (default) */
list-style: circle;              /* A hollow circle */
list-style: square;              /* A filled square */
list-style: '-';                 /* any string */

            Sıralı bir liste (<ol>) içindeki <li> etiketlerine özel:

            list-style: decimal;             /* Decimal numbers beginning with 1 (default) */
list-style: decimal-leading-zero;/* Decimal numbers padded by initial zeros (01, 02, 03, … 10) */
list-style: lower-roman;         /* Lowercase roman numerals (i., ii., iii., iv., ...) */
list-style: upper-roman;         /* Uppercase roman numerals (I., II., III., IV., ...) */
list-style-type: lower-greek;    /* Lowercase roman letters (α., β., γ., δ., ...) */
list-style-type: lower-alpha;    /* Lowercase letters (a., b., c., d., ...) */
list-style-type: lower-latin;    /* Lowercase letters (a., b., c., d., ...) */
list-style-type: upper-alpha;    /* Uppercase letters (A., B., C., D., ...) */
list-style-type: upper-latin;    /* Uppercase letters (A., B., C., D., ...) */

            Spesifik olmayan:

            list-style: none;                /* No visible list marker */
list-style: inherit;             /* Inherits from parent */

            Madde İşaretlerini / Sayıları Kaldırma #

            Bazen bir liste herhangi bir madde işareti veya sayı göstermemelidir. Bu durumda, kenar boşluğunu ve dolguyu belirtmeyi unutmayın.

            <ul>
    <li>first item</li>
    <li>second item</li>
</ul>

            CSS

            ul {
    list-style-type: none;
}
li {
    margin: 0;
    padding: 0;
}
