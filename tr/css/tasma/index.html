Sözdizimi #

  • taşma: görünür | gizli | kaydırma | oto | başlangıç ​​| miras;

Parametreler #

“Taşma” Değeri Ayrıntılar
“görünür” Öğenin dışındaki tüm taşan içeriği gösterir
kaydır Taşan içeriği gizler ve bir kaydırma çubuğu ekler
‘gizli’ Taşan içeriği gizler, her iki kaydırma çubuğu da kaybolur ve sayfa sabitlenir
‘otomatik’ İçerik taşarsa “kaydırma” ile aynıdır, ancak içerik sığıyorsa kaydırma çubuğu eklemez
‘miras’ Bu özellik için üst öğenin değerini devralır

overflow özelliği, içeriğin kırpılıp kırpılmayacağını, kaydırma çubukları kullanın veya içeriği görüntülemek için bir kapsayıcıyı uzatın. blok düzeyindeki kapsayıcısını aşıyor.

Bir öğe, içeriğini görüntüleyemeyecek kadar küçük olduğunda ne olur? Varsayılan olarak, içerik yalnızca taşacak ve öğenin dışında görüntülenecektir. Bu, işinizin kötü görünmesine neden olur. Çalışmanızın iyi görünmesini istiyorsunuz, bu nedenle taşan içeriği istenen şekilde işlemek için taşma özelliğini ayarladınız.

“Taşma” özelliğinin değerleri, her eksen boyunca uygulanmaları dışında “taşma-x” ve “taşma-y” özellikleriyle aynıdır.

‘overflow-wrap’ özelliği, ‘word-wrap’ özelliği olarak da bilinir.

Önemli not: Taşma özelliğinin şundan farklı bir değerle kullanılması görünür, yeni bir blok biçimlendirme bağlamı oluşturur.

taşma sarma #

“overflow-wrap”, tarayıcıya hedeflenen öğenin içindeki bir metin satırını, aksi takdirde kırılmaz bir yerde birden çok satıra bölebileceğini söyler. Konteynerinin taşması nedeniyle yerleşim sorunlarına neden olan uzun bir metin dizisinin önlenmesine yardımcı olur.

CSS

div {
    width:100px;
    outline: 1px dashed #bbb;
}

#div1 {
    overflow-wrap:normal;
}

#div2 {
    overflow-wrap:break-word;
}

HTML

<div id="div1">
      <strong>#div1</strong>: Small words are displayed normally, but a long word like <span style="red;">supercalifragilisticexpialidocious</span> is too long so it will overflow past the edge of the line-break
</div>

<div id="div2">
      <strong>#div2</strong>: Small words are displayed normally, but a long word like <span style="red;">supercalifragilisticexpialidocious</span> will be split at the line break and continue on the next line.
</div>

overflow-wrap – Değer Ayrıntılar
’normal’ Satırdan uzunsa bir sözcüğün taşmasına izin verir
‘break-word’ Gerekirse, bir kelimeyi birden çok satıra böler
‘miras’ Bu özellik için üst öğenin değerini devralır

taşma: kaydırma #

HTML

<div>
    This div is too small to display its contents to display the effects of the overflow property.
</div>

CSS

div {
    width:100px;
    height:100px;
    overflow:scroll;
}

Sonuç

Yukarıdaki içerik, taşan içeriği görüntülemek için kaydırma yapılabilen 100 piksele 100 piksellik bir kutuda kırpılmıştır.

Çoğu masaüstü tarayıcı, herhangi bir içerik kırpılmış olsun veya olmasın, hem yatay hem de dikey kaydırma çubuklarını görüntüler. Bu, dinamik bir ortamda görünen ve kaybolan kaydırma çubuklarıyla ilgili sorunları önleyebilir. Yazıcılar taşan içeriği yazdırabilir.

taşma: görünür #

HTML

<div>
     Even if this div is too small to display its contents, the content is not clipped.
</div>

CSS

div {
     width:50px;
     height:50px;
     overflow:visible;
}

Sonuç

İçerik kırpılmaz ve kapsayıcı boyutunu aşarsa içerik kutusunun dışında işlenir.

Taşma ile Oluşturulan Blok Biçimlendirme Bağlamı #

“Görünür” değerinden farklı bir değerle “overflow” özelliğinin kullanılması, yeni bir blok biçimlendirme bağlamı oluşturacaktır. Bu, bir blok elemanını kayan bir elemanın yanına hizalamak için kullanışlıdır.

CSS

img {
    float:left;
    margin-right: 10px;
}
 div {
     overflow:hidden; /* creates block formatting context */
}

HTML

<img src="http://placehold.it/100x100">
<div>
  <p>Lorem ipsum dolor sit amet, cum no paulo mollis pertinacia.</p>
  <p>Ad case omnis nam, mutat deseruisse persequeris eos ad, in tollit debitis sea.</p>
</div>

Sonuç

Bu örnek, “overflow” özellik kümesine sahip bir div içindeki paragrafların kayan bir görüntüyle nasıl etkileşime gireceğini gösterir.

taşma-x ve taşma-y #

Bu iki özellik, “taşma” özelliğiyle benzer şekilde çalışır ve aynı değerleri kabul eder. “overflow-x” parametresi yalnızca x veya soldan sağa ekseninde çalışır. “Taşma-y”, y veya yukarıdan aşağıya ekseninde çalışır.

HTML

<div id="div-x">
    If this div is too small to display its contents, 
    the content to the left and right will be clipped.
</div>

<div id="div-y">
    If this div is too small to display its contents, 
    the content to the top and bottom will be clipped.
</div>

CSS

div {
    width: 200px;
    height: 200px;
}

#div-x {
    overflow-x: hidden;
}

#div-y {
    overflow-y: hidden;
}
