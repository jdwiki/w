Flexbox’ı Kullanma #

HTML:

<div class="container">
  <img src="http://lorempixel.com/400/200" />
</div>

CSS:

html, body, .container {
  height: 100%;
}    
.container {
  display: flex;
  justify-content: center; /* horizontal center */
}
img {
  align-self: center; /* vertical center */
}

Sonucu Görüntüle

HTML:

<img src="http://lorempixel.com/400/200" />

CSS:

html, body {
  height: 100%;
}   
body {
  display: flex;
  justify-content: center; /* horizontal center */
  align-items: center;     /* vertical center */
}

Sonucu Görüntüle

Bkz. [Dinamik Dikey ve Yatay Merkezleme](https://www.wikiod.com/tr/css/esnek-kutu-duzeni-flexbox#Dinamik Dikey ve Yatay Merkezleme (öğeleri hizalama, yaslama içeriği) #t=201607231920579121055) flexbox ve stillerin ne anlama geldiği hakkında daha fazla ayrıntı için Flexbox belgelerine bakın.

Tarayıcı Desteği

Flexbox, tüm büyük tarayıcılar tarafından desteklenir, 10’dan önceki IE sürümleri hariç.

Safari 8 ve IE10 gibi bazı yeni tarayıcı sürümleri için satıcı önekleri gerekir.

Ön ek oluşturmanın hızlı bir yolu için üçüncü taraf bir araç olan Autoprefixer vardır.

Daha eski tarayıcılar için (IE 8 ve 9 gibi) a Polyfill kullanılabilir.

Flexbox tarayıcı desteğine daha ayrıntılı bir bakış için [bu cevaba] bakın( http://stackoverflow.com/a/35137869/3597276).

CSS dönüşümünü kullanma #

CSS dönüşümleri öğelerin boyutuna bağlıdır, bu nedenle öğenizin ne kadar uzun veya geniş olduğunu bilmiyorsanız, onu ilgili bir kabın üstünden ve solundan kesinlikle %50’ye kadar konumlandırabilir ve çevirebilirsiniz. Dikey ve yatay olarak ortalamak için %50 sola ve yukarı.

Bu teknikle, öğenin tamsayı olmayan bir piksel sınırında oluşturularak bulanık görünmesine neden olabileceğini unutmayın. Geçici bir çözüm için SO’daki bu yanıta bakın.

HTML

<div class="container">
  <div class="element"></div>
</div>

CSS

.container {
  position: relative;
}

.element {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

JSFiddle’daki örneği görüntüleyin

ÇAPRAZ TARAYICI UYUMLULUĞU #

transform özelliğinin eski tarayıcılar tarafından desteklenmesi için öneklere ihtiyacı vardır. Chrome<=35, Safari<=8, Opera<=22, Android Tarayıcı<=4.4.4 ve IE9 için önek gerekir. CSS dönüşümleri, IE8 ve daha eski sürümler tarafından desteklenmez.

Önceki örnek için ortak bir dönüşüm bildirimi:

-webkit-transform: translate(-50%, -50%); /* Chrome, Safari, Opera, Android */
    -ms-transform: translate(-50%, -50%); /* IE 9 */
        transform: translate(-50%, -50%);

Daha fazla bilgi için bkz. canIuse.

DAHA FAZLA BİLGİ #

  • Öğe, statik olmayan ilk üst öğeye göre konumlanıyor (“konum: göreli”, “mutlak” veya “sabit”). Bu keman ve bu belgeleme konusu daha fazlasını keşfedin.

  • Yalnızca yatay ortalama için, “sol: %50” ve “dönüştür: çevirX(-50%)” kullanın. Aynısı yalnızca dikey merkezleme için de geçerlidir: “üst: %50” ve “dönüştürme: çeviriY(-50%)” ile ortalayın.

  • Bu merkezleme yöntemiyle statik olmayan genişlik/yükseklik öğelerinin kullanılması, ortalanmış öğenin ezilmiş görünmesine neden olabilir. Bu çoğunlukla metin içeren öğelerde olur ve ‘sağ kenar boşluğu: -50%;’ ve ‘alt kenar boşluğu: -50%;’ eklenerek düzeltilebilir. Daha fazla bilgi için bu keman görüntüleyin.

5: https://www.wikiod.com/tr/css/konumlandrma#Mutlak Konum 6: https://jsfiddle.net/4xxmxca0/

Kenar boşluğunu kullanma: 0 otomatik; #

Nesneler, blok öğelerse ve tanımlanmış bir genişliğe sahiplerse margin: 0 auto; kullanılarak ortalanabilir.

HTML

<div class="containerDiv">
    <div id="centeredDiv"></div>
</div>

<div class="containerDiv">
    <p id="centeredParagraph">This is a centered paragraph.</p>
</div>

<div class="containerDiv">
    <img id="centeredImage" src="https://i.kinja-img.com/gawker-media/image/upload/s--c7Q9b4Eh--/c_scale,fl_progressive,q_80,w_800/qqyvc3bkpyl3mfhr8all.jpg" />
</div>

CSS

.containerDiv {
    width: 100%;
    height: 100px;
    padding-bottom: 40px;
}

#centeredDiv {
    margin: 0 auto;
    width: 200px;
    height: 100px;
    border: 1px solid #000;
}

#centeredParagraph {
    width: 200px;
    margin: 0 auto;
}

#centeredImage {
    display: block;
    width: 200px;
    margin: 0 auto;
}

JSFiddle örneği: Nesneleri kenar boşluğuyla ortalama: 0 auto;

Metin hizalamayı kullanma #

En yaygın ve en kolay merkezleme türü, bir öğedeki metin satırlarıdır. CSS, bu amaç için “metin hizalama: merkez” kuralına sahiptir:

HTML

<p>Lorem ipsum</p>

CSS

p {
    text-align: center;
}

Bu, tüm blok öğelerini ortalamak için çalışmaz. “metin hizalama”, yalnızca üst blok öğesindeki metin gibi satır içi içeriğin hizalamasını kontrol eder.

Tipografi bölümünde “metin hizalama” hakkında daha fazla bilgi edinin.

Konumu kullanma: mutlak #

Eski tarayıcılarda çalışma (IE >= 8)

“Sol” ve “sağ” veya “üst” ve “alt” ofsetler için sıfır değerleriyle eşleştirilen otomatik kenar boşlukları, üst öğesinde kesinlikle konumlandırılmış öğeleri ortalar.

Sonucu Görüntüle

HTML

<div class="parent">
  <img class="center" src="http://lorempixel.com/400/200/" />
</div>

CSS

.parent {
  position: relative;
  height: 500px;
}

.center {
  position: absolute;
  margin: auto;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
}

Görüntüler gibi kendi örtük genişliğine ve yüksekliğine sahip olmayan öğeler, tanımlanan bu değerlere ihtiyaç duyacaktır.

Diğer kaynaklar: CSS’de Mutlak Merkezleme

calc() kullanma #

calc() işlevi, piksel, yüzde vb. gibi çeşitli değerleri kullanarak öğenizin hangi boyutu/konumu kapladığını (matematiksel olarak) hesaplayabileceğiniz CSS3’teki yeni bir sözdiziminin parçasıdır. Not:- Bu işlevi her kullandığınızda , her zaman iki calc(100% - 80px) değeri arasındaki boşluğa dikkat edin.

CSS

.center {
    position: absolute;
    height: 50px;
    width: 50px;
    background: red;
    top: calc(50% - 50px / 2); /* height divided by 2*/
    left: calc(50% - 50px / 2); /* width divided by 2*/
}

HTML

<div class="center"></div>

Her şeyi 3 satır kodla dikey olarak hizalayın #

IE11+ tarafından desteklenir

Sonucu Görüntüle

Hemen hemen her şeyi dikey olarak hizalamak için bu 3 çizgiyi kullanın. Kodu uyguladığınız div/image’in yüksekliğe sahip bir ebeveyni olduğundan emin olun.

CSS

div.vertical {
  position: relative;
  top: 50%;
  transform: translateY(-50%);
}

HTML

<div class="vertical">Vertical aligned text!</div>

Satır yüksekliğini kullanma #

Bir kap içinde tek bir metin satırını dikey olarak ortalamak için “line-height"ı da kullanabilirsiniz:

CSS

div {
    height: 200px;
    line-height: 200px;
}

Bu oldukça çirkin, ancak bir “” öğesi içinde yararlı olabilir. ’line-height’ özelliği yalnızca ortalanacak metin tek bir satıra yayıldığında çalışır. Metin birden çok satıra sarılırsa, elde edilen çıktı ortalanmaz.

Hayalet eleman tekniği (Michał Czernow’un hilesi) #

Bu teknik, konteynerin boyutları bilinmediğinde bile çalışır.

Ortalanacak kabın içinde %100 yükseklikte bir “hayalet” öğe ayarlayın, ardından hem bunda hem de ortalanacak öğede “dikey hizalama: orta” kullanın.

CSS

/* This parent can be any width and height */
.block {
  text-align: center;

  /* May want to do this if there is risk the container may be narrower than the element inside */
  white-space: nowrap;
}
 
/* The ghost element */
.block:before {
  content: '';
  display: inline-block;
  height: 100%;
  vertical-align: middle;

  /* There is a gap between ghost element and .centered,
  caused by space character rendered. Could be eliminated by
  nudging .centered (nudge distance depends on font family),
  or by zeroing font-size in .parent and resetting it back
  (probably to 1rem) in .centered. */
  margin-right: -0.25em;
}

/* The element to be centered, can also be of any width and height */ 
.centered {
  display: inline-block;
  vertical-align: middle;
  width: 300px;
  white-space: normal; /* Resetting inherited nowrap behavior */
}

HTML

<div class="block">
  <div class="centered"></div>
</div>

Başka bir öğeye göre merkezleme #

Yakın bir öğenin yüksekliğine göre içeriğin nasıl ortalanacağını göreceğiz.

Uyumluluk: IE8+, diğer tüm modern tarayıcılar.

HTML

<div class="content">
  <div class="position-container">
    <div class="thumb">
      <img src="http://lorempixel.com/400/200/">
    </div>
    <div class="details">
      <p class="banner-title">text 1</p>
      <p class="banner-text">content content content content content content content content content content content content content content</p>
      <button class="btn">button</button>
    </div>
  </div>
</div>

CSS

.content * {
  box-sizing: border-box;
}
.content .position-container {
  display: table;
}
.content .details {
  display: table-cell;
  vertical-align: middle;
  width: 33.333333%;
  padding: 30px;
  font-size: 17px;
  text-align: center;
}
.content .thumb {
  width: 100%;
}
.content .thumb img {
  width: 100%;
}

JSFiddle bağlantısı

Ana noktalar 3 “.thumb”, “.details” ve “.position-container” kapsayıcılarıdır:

  • “.position-container”, “ekran: tablo” içermelidir.
  • “.details”, “width: ….” ve “display: table-cell”, “dikey hizalama: orta” olarak ayarlanmış gerçek genişliğe sahip olmalıdır.
  • “.thumb”, “genişlik: %100” olmalıdır, kalan tüm alanı kaplamasını ve “.details” genişliğinden etkilenmesini istiyorsanız.
  • “.thumb” içindeki resim (eğer bir resminiz varsa) “genişlik: %100” olmalıdır, ancak doğru orantılara sahipseniz gerekli değildir.

Bir görüntüyü div içinde dikey olarak hizalayın #

HTML

<div class="wrap">
    <img src="http://lorempixel.com/400/200/" />
</div>

CSS

.wrap {
    height: 50px;/* max image height */
    width: 100px;
    border: 1px solid blue;
    text-align: center;
}
.wrap:before {
  content:"";
  display: inline-block;
  height: 100%;
  vertical-align: middle;
  width: 1px;
}

img {
    vertical-align: middle;
}

Yükseklik veya genişlik konusunda endişelenmeden dikey ve yatay olarak merkezleme #

Aşağıdaki teknik, içeriğinizi bir HTML öğesine eklemenize ve hem yatay hem de dikey olarak yüksekliği veya genişliği hakkında endişelenmeden ortalamanıza olanak tanır.

Dış kap #

  • ekran: tablo; olmalıdır

İç kap #

  • ekran: tablo hücresi; olmalıdır
  • dikey hizalama: orta; olmalıdır
  • metin hizalama: merkez; olmalıdır

İçerik kutusu #

  • gösterge: satır içi blok; olmalıdır
  • yatay metin hizalamasını örn. Metnin ortalanmasını istemiyorsanız, text-align: left; veya text-align: right;

Demo #

HTML

<div class="outer-container">
   <div class="inner-container">
     <div class="centered-content">
        You can put anything here!
     </div>
   </div>
</div>

CSS

body {
    margin : 0;
}

.outer-container {
    position : absolute;
    display: table;
    width: 100%; /* This could be ANY width */
    height: 100%; /* This could be ANY height */
    background: #ccc;
}

.inner-container {
    display: table-cell;
    vertical-align: middle;
    text-align: center;
}

.centered-content {
    display: inline-block;
    text-align: left;
    background: #fff;
    padding: 20px;
    border: 1px solid #000;
}

Sabit boyutta merkezleme #

İçeriğinizin boyutu sabitse, içeriğinizin genişliğinin ve yüksekliğinin yarısını azaltan “margin” ile %50’ye mutlak konumlandırma kullanabilirsiniz:

HTML

<div class="center">
    Center vertically and horizontally
</div>

CSS

.center {
    position: absolute;
    background: #ccc;

    left: 50%;
    width: 150px;
    margin-left: -75px;  /* width * -0.5 */

    top: 50%;
    height: 200px;
    margin-top: -100px;  /* height * -0.5 */
}

Yalnızca sabit genişlikte yatay merkezleme #

İçeriğin yüksekliğini bilmeseniz bile öğeyi yatay olarak ortalayabilirsiniz:

HTML

<div class="center">
    Center only horizontally
</div>

CSS

.center {
    position: absolute;
    background: #ccc;

    left: 50%;
    width: 150px;
    margin-left: -75px;  /* width * -0.5 */
}

Sabit yükseklikte dikey merkezleme #

Elemanın yüksekliğini biliyorsanız, elemanı dikey olarak ortalayabilirsiniz:

HTML

<div class="center">
    Center only vertically
</div>

CSS

.center {
    position: absolute;
    background: #ccc;

    top: 50%;
    height: 200px;
    margin-top: -100px;  /* width * -0.5 */
}

Dinamik yükseklik öğelerini dikey olarak hizalayın #

CSS’yi sezgisel olarak uygulamak istenen sonuçları vermez çünkü

  • “dikey hizalama:orta” [uygun değil blok düzeyindeki öğelere 1
  • “margin-top:auto” ve “margin-bottom:auto” [kullanılan değerler sıfır2 olarak hesaplanır
  • “margin-top:-50%” yüzdeye dayalı marj değerleri içeren bloğun genişliğine göre hesaplanır

En geniş tarayıcı desteği için yardımcı öğeler içeren bir geçici çözüm:

HTML

<div class="vcenter--container">
  <div class="vcenter--helper">
    <div class="vcenter--content">
      <!--stuff-->
    </div>
  </div>
</div>

CSS

.vcenter--container {
  display: table;
  height: 100%;
  position: absolute;
  overflow: hidden;
  width: 100%;
}
.vcenter--helper {
  display: table-cell;
  vertical-align: middle;
}
.vcenter--content {
  margin: 0 auto;
  width: 200px;
}

Tablo düzenini kullanarak Yatay ve Dikey merkezleme #

Bir alt öğe, ’table’ görüntüleme özelliğini kullanarak kolayca ortalanabilir.

HTML

<div class="wrapper">
    <div class="parent">
        <div class="child"></div>
    </div>
</div>

CSS

.wrapper { 
    display: table; 
    vertical-align: center; 
    width: 200px; 
    height: 200px; 
    background-color: #9e9e9e; 
}
.parent { 
    display: table-cell; 
    vertical-align: middle; 
    text-align: center; 
}
.child { 
    display: inline-block; 
    vertical-align: middle; 
    text-align: center; 
    width: 100px; 
    height: 100px; 
    background-color: teal; 
}
