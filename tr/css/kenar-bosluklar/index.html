Sözdizimi #

  • kenar boşluğu: <üst & sağ & alt & sol>;
  • kenar boşluğu: <üst>, <sol ve sağ>, <alt>;
  • kenar boşluğu: <üst ve alt>, <sol ve sağ>;
  • kenar boşluğu: <üst>, <sağ>, <alt>, <sol>;
  • üst kenar boşluğu: <üst>;
  • sağ kenar boşluğu: <sağ>;
  • kenar boşluğu-alt: <alt>;
  • sol kenar boşluğu: <sol>;

parametre Ayrıntılar

“Daraltma Kenar Boşlukları” hakkında daha fazla bilgi: burada.

Marj Daralması #

İki kenar boşluğu dikey olarak birbirine değdiğinde daraltılır. İki kenar boşluğu yatay olarak birbirine dokunduğunda çökmezler.

Bitki dikey kenar boşlukları örneği:

Aşağıdaki stilleri ve işaretlemeyi göz önünde bulundurun:

div{
    margin: 10px;
}

<div>
    some content
</div>
<div>
    some more content
</div>

Dikey kenar boşlukları birbiri üzerine çöktüğü için aralarında 10 piksel olacak. (Aralık, iki kenar boşluğunun toplamı olmayacaktır.)

Bitki yatay kenar boşlukları örneği:

Aşağıdaki stilleri ve işaretlemeyi göz önünde bulundurun:

span{
    margin: 10px;
}

<span>some</span><span>content</span>

Yatay kenar boşlukları birbiri üzerine çökmediğinden aralarında 20 piksel olacaktır. (Aralık, iki kenar boşluğunun toplamı olacaktır.)

Farklı boyutlarda örtüşme

.top{
    margin: 10px;
}
.bottom{
    margin: 15px;
}

<div class="top">
    some content
</div>
<div class="bottom">
    some more content
</div>

Bu öğeler dikey olarak 15 piksel aralıklarla yerleştirilecektir. Kenar boşlukları olabildiğince örtüşür, ancak daha büyük kenar boşluğu, öğeler arasındaki boşluğu belirleyecektir.

Örtüşen kenar boşluğu yakalandı

.outer-top{
    margin: 10px;
}
.inner-top{
    margin: 15px;
}
.outer-bottom{
    margin: 20px;
}
.inner-bottom{
    margin: 25px;
}

<div class="outer-top">
    <div class="inner-top">
        some content
    </div>
</div>
<div class="outer-bottom">
    <div class="inner-bottom">
        some more content
    </div>
</div>

İki metin arasındaki boşluk ne olacak? (cevabı görmek için üzerine gelin)

!Aralık 25 piksel olacaktır. Dört kenar boşluğu da birbirine değdiği için, dördün en büyük kenar boşluğunu kullanarak çökeceklerdir.

Şimdi, yukarıdaki işaretlemeye bazı sınırlar eklersek ne olur?

div{
    border: 1px solid red;
}

İki metin arasındaki boşluk ne olacak? (cevabı görmek için üzerine gelin)

!Aralık 59 piksel olacaktır! Şimdi yalnızca .outer-top ve .outer-bottom öğelerinin kenar boşlukları birbirine değiyor ve yalnızca daraltılmış kenar boşlukları bunlar. Kalan kenar boşlukları kenarlıklarla ayrılır. Yani elimizde 1px + 10px + 1px + 15px + 20px + 1px + 25px + 1px var. (1px’ler sınırlardır…)

Üst ve Alt Öğeler Arasındaki Daralan Kenar Boşlukları:

HTML:

<h1>Title</h1>
<div>
  <p>Paragraph</p>
</div>

CSS

h1 {
  margin: 0;
  background: #cff;
}
div {
  margin: 50px 0 0 0;
  background: #cfc;
}
p {
  margin: 25px 0 0 0;
  background: #cf9;
}

Yukarıdaki örnekte, yalnızca en büyük kenar boşluğu geçerlidir. Paragrafın h1’den 60 piksel uzakta olmasını beklemiş olabilirsiniz (div öğesinin üst kenar boşluğu 40 piksel ve p üst kenar boşluğu 20 piksel olduğundan). Bu, kenar boşlukları bir kenar boşluğu oluşturmak üzere birlikte çöktüğü için gerçekleşmez.

Verilen Tarafa Marj Uygula #

Yöne Özgü Özellikler #

CSS, marjın uygulanacağı belirli bir tarafı belirtmenize olanak tanır. Bu amaçla sağlanan dört özellik şunlardır:

  • “sol kenar boşluğu”
  • “sağ kenar boşluğu”
  • “kenar boşluğu”
  • “marj-alt”

Aşağıdaki kod, seçilen div’in sol tarafına 30 piksellik bir kenar boşluğu uygular. Sonucu Görüntüle

HTML

<div id="myDiv"></div>

CSS

#myDiv {
    margin-left: 30px;
    height: 40px;
    width: 40px;
    background-color: red;
}
parametre Detaylar
kenar-sol Kenar boşluğunun uygulanması gereken yön.
30 piksel Kenar boşluğunun genişliği.

Steno Özelliğini Kullanarak Yön Belirleme #

Standart “margin” özelliği, seçilen öğelerin her iki yanında farklı genişlikler belirtmek için genişletilebilir. Bunu yapmak için sözdizimi aşağıdaki gibidir:

margin: <top> <right> <bottom> <left>;

Aşağıdaki örnek, div’in üstüne sıfır genişlikte bir kenar boşluğu, sağ tarafa 10 piksel kenar boşluğu, sol tarafa 50 piksel kenar boşluğu ve sol tarafa 100 piksel kenar boşluğu uygular. Sonucu Görüntüle

HTML

<div id="myDiv"></div>

CSS

#myDiv {
    margin: 0 10px 50px 100px;
    height: 40px;
    width: 40px;
    background-color: red;
}

Marj özelliği basitleştirme #

p {
    margin:1px;                /* 1px margin in all directions */
    
    /*equals to:*/
    
    margin:1px 1px;
    
    /*equals to:*/
    
    margin:1px 1px 1px;
    
    /*equals to:*/
    
    margin:1px 1px 1px 1px;
}

Başka bir örnek:

p{
    margin:10px 15px;        /* 10px margin-top & bottom And 15px margin-right & left*/
    
    /*equals to:*/
    
    margin:10px 15px 10px 15px;
    
    /*equals to:*/
    
    margin:10px 15px 10px;
    /* margin left will be calculated from the margin right value (=15px) */
}

Öğeleri kenar boşluğunu kullanarak bir sayfada yatay olarak ortalayın #

Öğe bir blok olduğu ve açıkça ayarlanmış bir genişlik değerine sahip olduğu sürece, blok öğelerini bir sayfada yatay olarak ortalamak için kenar boşlukları kullanılabilir.

Pencerenin genişliğinden daha düşük bir genişlik değeri ekleriz ve marjın auto özelliği, kalan alanı sola ve sağa dağıtır:

#myDiv {
 width:80%;
 margin:0 auto;
}

Yukarıdaki örnekte, önce üst ve alt kenar boşluğu değerlerine “0"ı ayarlamak için (bu herhangi bir değer olabilir) ve ardından tarayıcının alanı otomatik olarak sol ve sağ kenar boşluğu değerleri.

Yukarıdaki örnekte, #myDiv öğesi %80 genişliğe ayarlanmıştır, bu da artık %20’lik kullanım bırakır. Tarayıcı bu değeri kalan taraflara şu şekilde dağıtır:

(%100 - %80) / 2 = %10

Negatif marjlar #

Marj, negatif değerlere ayarlanabilen birkaç CSS özelliğinden biridir. Bu özellik, mutlak konumlandırma olmaksızın öğelerin üst üste bindirilmesi için kullanılabilir.

div{
  display: inline;
}

#over{
  margin-left: -20px;
}

<div>Base div</div>
<div id="over">Overlapping div</div>
