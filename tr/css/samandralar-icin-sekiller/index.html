Sözdizimi #

  • şekil-dış: yok | [ <temel-şekil> || <şekil-kutusu> ] | <resim>

  • şekil-marj: <uzunluk> | <yüzde>

  • şekil-görüntü-eşiği: <sayı>

Parametreler #

parametre Ayrıntılar
hiçbiri “none” değeri, kayan alanın (bir kayan öğenin etrafındaki içeriği sarmak için kullanılan alan) etkilenmediği anlamına gelir. Bu, varsayılan/ilk değerdir.
temel şekil ‘inset()’, ‘circle()’, ’elipse()’ veya ‘polygon()’ arasından birini ifade eder. Bu işlevlerden ve değerlerinden biri kullanılarak şekil tanımlanır.
şekil kutusu ‘margin-box’, ‘border-box’, ‘padding-box’, ‘content-box’ arasından birini ifade eder. Yalnızca <shape-box> sağlandığında (<basic-shape> olmadan) bu kutu *şekildir. <basic-shape> ile birlikte kullanıldığında, bu, referans kutusu görevi görür.
resim Bir görüntü değer olarak sağlandığında, şekil, belirtilen görüntünün alfa kanalına göre hesaplanır.

CSS Shapes modülü için tarayıcı desteği bu noktada çok sınırlıdır.

Tarayıcı/satıcı önekleri olmadan Chrome v37+ ve Opera 24+‘da desteklenir. Safari bunu v7.1+ sürümünden destekler, ancak -webkit- öneki ile.

Henüz IE, Edge ve Firefox’ta desteklenmemektedir.

Temel Şekil ile Dış Şekil – daire() #

shape-outside CSS özelliği ile, satır içi içeriğin şamandıranın kutusu yerine şekli sarması için kayan alan için şekil değerleri tanımlayabilirsiniz.

CSS

img:nth-of-type(1) {
  shape-outside: circle(80px at 50% 50%);
  float: left;
  width: 200px;
}
img:nth-of-type(2) {
  shape-outside: circle(80px at 50% 50%);
  float: right;
  width: 200px;  
}
p {
  text-align: center;
  line-height: 30px; /* purely for demo */
}

HTML

<img src="http://images.clipartpanda.com/circle-clip-art-circlergb.jpg">
<p>Some paragraph whose text content is required to be wrapped such that it follows the curve of the circle on either side. And then there is some filler text just to make the text long enough. Lorem Ipsum Dolor Sit Amet....</p>

Yukarıdaki örnekte, her iki görüntü de aslında kare görüntülerdir ve metin ‘shape-outside’ özelliği olmadan yerleştirildiğinde, her iki taraftaki dairenin etrafında akmayacaktır. Yalnızca görüntünün içeren kutusunun etrafında akacaktır. “şekil-dış” ile, kayan alan bir daire olarak yeniden tanımlanır ve içeriğin, “şekil-dış” kullanılarak oluşturulan bu hayali daire etrafında akması sağlanır.

Kayan alanı yeniden tanımlamak için kullanılan hayali daire, görüntünün referans kutusunun orta-orta noktasından çizilen 80 piksel yarıçaplı bir dairedir.

Aşağıda, “shape-outside” kullanıldığında ve kullanılmadığında içeriğin nasıl sarılacağını gösteren birkaç ekran görüntüsü verilmiştir.

Şekil marjı #

shape-margin CSS özelliği, shape-outside öğesine bir marj ekler.

CSS

img:nth-of-type(1) {
  shape-outside: circle(80px at 50% 50%);
  shape-margin: 10px;
  float: left;
  width: 200px;
}
img:nth-of-type(2) {
  shape-outside: circle(80px at 50% 50%);
  shape-margin: 10px;
  float: right;
  width: 200px;  
}
p {
  text-align: center;
  line-height: 30px; /* purely for demo */
}

HTML

<img src="http://images.clipartpanda.com/circle-clip-art-circlergb.jpg">
<img src="http://images.clipartpanda.com/circle-clip-art-circlergb.jpg">
<p>Some paragraph whose text content is required to be wrapped such that it follows the curve of the circle on either side. And then there is some filler text just to make the text long enough. Lorem Ipsum Dolor Sit Amet....</p>

Bu örnekte, “şekil-marj” kullanılarak şekil etrafına 10 piksellik bir kenar boşluğu eklenir. Bu, kayan alanı tanımlayan hayali daire ile etrafta akan gerçek içerik arasında biraz daha fazla boşluk yaratır.
