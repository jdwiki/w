Sözdizimi #

  • Döndürme Dönüştür
  • dönüştür: döndür(<açı>)
  • Dönüştürmeyi Çevir
  • transform: translate(<uzunluk-veya-yüzde> [, <uzunluk-veya-yüzde>]?)
  • transform: translateX(<uzunluk-veya-yüzde>)
  • transform: translateY(<uzunluk-veya-yüzde>)
  • Eğrilik Dönüşümü
  • dönüştür: çarpık(<açı> [, <açı>]?)
  • dönüştürme: skewX(<angle>)
  • dönüştürme: çarpık(<açı>)
  • Ölçek Dönüşümü
  • dönüştürme: ölçek(<ölçek-faktör> [, <ölçek-faktör>]?)
  • dönüştürme: scaleX(<scale-factor>)
  • dönüştürme: ölçekY(<ölçek-faktör>)
  • Matris Dönüşümü
  • dönüştürme: matris(<sayı> [, <sayı> ]{5,5})

Parametreler #

Fonksiyon/Parametre Ayrıntılar
döndür(x) Öğeyi Z ekseninde sabit bir nokta etrafında hareket ettiren bir dönüşüm tanımlar
çeviri(x,y) Elemanın konumunu X ve Y ekseninde hareket ettirir
translateX(x) Öğenin konumunu X ekseninde taşır
çeviriY(y) Elemanın konumunu Y ekseninde taşır
ölçek(x,y) X ve Y eksenindeki öğenin boyutunu değiştirir
ölçekX(x) X eksenindeki öğenin boyutunu değiştirir
ölçekY(y) Y eksenindeki öğenin boyutunu değiştirir
skew(x,y) Bir elemanın her noktasını her yönde belirli bir açıyla çarpıtan kesme eşlemesi veya transveksiyon
skewX(x) Bir elemanın her noktasını yatay yönde belirli bir açıyla bozan yatay kayma haritalama
skewY(y) Bir elemanın her noktasını düşey yönde belirli bir açıyla bozan düşey kesme haritalama
matris() Bir dönüşüm matrisi biçiminde bir 2B dönüşümü tanımlar.
açı Elemanın döndürülmesi veya eğrilmesi gereken açı (kullanıldığı işleve bağlı olarak). Açı, derece (‘derece’), gradyan (‘grad’), radyan (‘rad’) veya dönüş (‘dönüş’) olarak sağlanabilir. ‘skew()’ işlevinde ikinci açı isteğe bağlıdır. Sağlanmazsa, Y ekseninde (0) çarpıklık olmaz.
uzunluk veya yüzde Öğenin çevrilmesi gereken uzunluk veya yüzde olarak ifade edilen mesafe. ’translate()’ işlevinde ikinci uzunluk veya yüzde isteğe bağlıdır. Sağlanmazsa, Y ekseninde (0) öteleme olmaz.
ölçek faktörü Elemanın belirtilen eksende kaç kez ölçeklenmesi gerektiğini tanımlayan bir sayı. ‘scale()’ işlevinde, ikinci ölçek faktörü isteğe bağlıdır. Sağlanmazsa, ilk ölçek faktörü Y ekseni için de uygulanacaktır.

2D Koordiyen sistemi #

Dönüşümler 2D X/Y koordinat sistemine göre yapılır. Aşağıdaki resimde gösterildiği gibi X ekseni sağdan sola ve Y ekseni aşağı doğru gider:

Yani pozitif bir ’translateY()’ aşağı doğru gider ve pozitif bir ’translateX()’ doğru gider.

Tarayıcı desteği ve önekleri #

  • IE, -ms- öneki ile IE9’dan beri bu özelliği desteklemektedir. Eski sürümler ve Edge ön eke ihtiyaç duymaz
  • Firefox, 3.5 sürümünden beri desteklemektedir ve sürüm 15’e kadar -moz- ön ekine ihtiyaç duyar.
  • Sürüm 4’ten bu yana ve sürüm 34’e kadar olan Chrome, “-webkit-” ön ekine ihtiyaç duyar
  • Safari, sürüm 8’e kadar -webkit- ön ekine ihtiyaç duyuyor
  • Opera, sürüm 11.5 için “-o-” önekine ve sürüm 15’ten 22’ye kadar “-webkit-” önekine ihtiyaç duyar
  • Android’in 2.1 sürümünden 4.4.4 sürümüne kadar -webkit- ön ekine ihtiyacı var

Ön ekli dönüşüm örneği: #

-webkit-transform: rotate(45deg);
    -ms-transform: rotate(45deg);
        transform: rotate(45deg);

Döndür #

HTML

<div class="rotate"></div>

CSS

.rotate {
    width: 100px;
    height: 100px;
    background: teal;
    transform: rotate(45deg);
}

Bu örnek, div’i saat yönünde 45 derece döndürür. Dönme merkezi, div’in merkezinde, soldan “%50” ve üstten “%50"dir. ’transform-origin’ özelliğini ayarlayarak dönüş merkezini değiştirebilirsiniz.

transform-origin: 100% 50%;

Yukarıdaki örnek, dönme merkezini sağ taraftaki ucun ortasına ayarlayacaktır.

Ölçek #

HTML

<div class="scale"></div>

CSS

.scale {
    width: 100px;
    height: 100px;
    background: teal;
    transform: scale(0.5, 1.3);
}

Bu örnek, div’i X ekseninde “100 piksel * 0,5 = 50 piksel” ve Y ekseninde “100 piksel * 1,3 = 130 piksel” olarak ölçeklendirir.

Dönüşümün merkezi, div’in merkezinde, soldan “%50” ve üstten “%50"dir.

Eğme #

HTML

<div class="skew"></div>

CSS

.skew {
    width: 100px;
    height: 100px;
    background: teal;
    transform: skew(20deg, -30deg);
}

Bu örnek, div’i X ekseninde 20 derece ve Y ekseninde - 30 derece eğecektir. Dönüşümün merkezi, div’in merkezinde, soldan “%50” ve üstten “%50"dir.

Sonuca buradan bakın.

Çoklu dönüşümler #

Bir özellikteki bir öğeye aşağıdaki gibi birden çok dönüşüm uygulanabilir:

transform: rotate(15deg) translateX(200px);

Bu, öğeyi saat yönünde 15 derece döndürecek ve ardından 200 piksel sağa çevirecektir.

Zincirleme dönüşümlerde, koordinat sistemi elemanla birlikte hareket eder. Bu, çevirinin yatay olmayacağı, ancak aşağıdaki resimde gösterildiği gibi bir eksende saat yönünde 15 derece döndüğü anlamına gelir:

Dönüşümlerin sırasını değiştirmek çıktıyı değiştirecektir. İlk örnek farklı olacak

transform: translateX(200px) rotate(15deg);

<div class="transform"></div>

.transform {
  transform: rotate(15deg) translateX(200px);
}

Bu resimde gösterildiği gibi:

Çevirmek #

HTML

<div class="translate"></div>

CSS

.translate {
    width: 100px;
    height: 100px;
    background: teal;
    transform: translate(200px, 50%);
}

Bu örnek, div’i X ekseninde 200 piksel ve Y ekseninde “100 piksel * %50 = 50 piksel” hareket ettirecektir.

Ayrıca, çevirileri tek bir eksende belirleyebilirsiniz.

X ekseninde:

.translate {
    transform: translateX(200px);
}

Y ekseninde:

.translate {
    transform: translateY(50%);
}

Kökeni Dönüştür #

Dönüşümler, ’transform-origin’ özelliği ile tanımlanan bir noktaya göre yapılır.

Özellik 2 değer alır: transform-origin: X Y;

Aşağıdaki örnekte, ilk div (.tl) transform-origin: 0 0; ile sol üst köşede döndürülür ve ikinci (.tr) transform- ile sağ üst köşede dönüştürülür. köken: %100 0. Döndürme fareyle üzerine gelindiğinde uygulanır:

HTML:

<div class="transform originl"></div>
<div class="transform origin2"></div>

CSS:

.transform {
    display: inline-block;
    width: 200px;
    height: 100px;
    background: teal;
    transition: transform 1s;
}

.origin1 {
    transform-origin: 0 0;
}

.origin2 {
    transform-origin: 100% 0;
}

.transform:hover {
    transform: rotate(30deg);
}

transform-origin özelliği için varsayılan değer, öğenin merkezi olan “%50 %50"dir.
