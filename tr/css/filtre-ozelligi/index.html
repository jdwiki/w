Sözdizimi #

  • filtre: yok (varsayılan değer)
  • filtre: başlangıç ​​(varsayılanı yok);
  • filtre: devral (varsayılan değer üst değerdir);
  • filtre: bulanıklaştırma(px)
  • filtre: parlaklık(sayı | %)
  • filtre: kontrast(sayı | %)
  • filtre: gölge (yatay-gölge-px dikey-gölge-px gölge-bulanıklık-px gölge- - yayılmış renk)
  • filtre: gri tonlamalı(sayı | %)
  • filtre: ton döndürme (derece)
  • filtre: ters çevir(sayı | %)
  • filtre: opaklık(sayı | %)
  • filtre: doymuş(sayı | %)
  • filtre: sepya(sayı | %)

Parametreler #

Değer Açıklama
blur(x) Görüntüyü x piksel bulanıklaştırır.
parlaklık(x) Görüntüyü 1,0 veya %100’ün üzerindeki herhangi bir değerde parlaklaştırır. Bunun altında görüntü karartılır.
contrast(x) 1,0 veya %100’ün üzerindeki herhangi bir değerde görüntüye daha fazla kontrast sağlar. Bunun altında görüntü daha az doygun hale gelir.
drop-shadow(h, v, x, y, z) Görüntüye bir gölge verir. h ve v negatif değerlere sahip olabilir. x, y ve z isteğe bağlıdır.
gri tonlamalı(x) Görüntüyü maksimum 1,0 veya %100 değerinde gri tonlamalı olarak gösterir.
hue-rotate(x) Görüntüye bir ton dönüşü uygular.
invert(x) Görüntünün rengini maksimum 1,0 veya %100 değerinde tersine çevirir.
opacity(x) Görüntünün ne kadar opak/saydam olduğunu, maksimum değeri 1.0 veya %100 olarak ayarlar.
saturate(x) Görüntüyü 1,0 veya %100’ün üzerindeki herhangi bir değerde doyurur. Bunun altında, görüntü doygunluğunu kaybetmeye başlayacaktır.
sepya(x) Görüntüyü maksimum 1.0 veya %100 değerle sepyaya dönüştürür.

  1. Filtre deneysel bir özellik olduğundan, -webkit önekini kullanmalısınız. Sözdizimi ve davranışta değişebilir, ancak değişiklikler muhtemelen küçük olacaktır.

  2. Büyük tarayıcıların eski sürümlerinde desteklenmeyebilir. Mobil tarayıcılarda tamamen desteklenmeyebilir.

  3. Nispeten sınırlı desteği nedeniyle, ‘filter: drop-shadow()’ yerine ‘box-shadow’ kullanmayı deneyin. “filtre: opacity()” yerine “opaklık” kullanın.

  4. Javascript/jQuery ile canlandırılabilir. Javascript için object.style.WebkitFilter kullanın.

  5. Daha fazla bilgi için W3Schools veya MDN kontrol edin.

  6. W3Schools’da ayrıca tüm farklı filtre değerleri için bir demo sayfası vardır.

Bulanıklaştırma #

HTML

<img src='donald-duck.png' alt='Donald Duck' title='Donald Duck' />

CSS

img {
    -webkit-filter: blur(1px);
    filter: blur(1px);
}

Sonuç

Gözlüklerini ovmak istemeni sağlıyor.

Alt Gölge (mümkünse bunun yerine kutu gölgesi kullanın) #

HTML

<p>My shadow always follows me.</p>

CSS

p {
    -webkit-filter: drop-shadow(10px 10px 1px green);
    filter: drop-shadow(10px 10px 1px green);
}

Sonuç

Ton Döndürme #

HTML

<img src='donald-duck.png' alt='Donald Duck' title='Donald Duck' />

CSS

img {
    -webkit-filter: hue-rotate(120deg);
    filter: hue-rotate(120deg);
}

Çoklu Filtre Değerleri #

Birden çok filtre kullanmak için her değeri bir boşlukla ayırın.

HTML

<img src='donald-duck.png' alt='Donald Duck' title='Donald Duck' />

CSS

img {
    -webkit-filter: brightness(200%) grayscale(100%) sepia(100%) invert(100%);
    filter: brightness(200%) grayscale(100%) sepia(100%) invert(100%);
}

Sonuç

Rengi Ters Çevir #

HTML

<div></div>

CSS

div {
    width: 100px;
    height: 100px;
    background-color: white;
    -webkit-filter: invert(100%);
    filter: invert(100%);
}
