Esnek Kutu modülü veya kısaca ‘flexbox’, kullanıcı arayüzleri için tasarlanmış bir kutu modelidir ve kullanıcıların, sayfa düzeni farklı, bilinmeyenleri barındırması gerektiğinde öğelerin tahmin edilebilir şekilde davranacağı şekilde, kullanıcıların bir kap içindeki öğeler arasında alanı hizalamasına ve dağıtmasına olanak tanır. ekran boyutları. Esnek bir kap, öğeleri mevcut alanı dolduracak şekilde genişletir ve taşmayı önlemek için küçültür.

Sözdizimi #

  • ekran: esnek;
  • esnek yön: satır | sıra-ters | sütun | sütun-ters;
  • esnek sarma: nowrap | sarma | sarma-ters;
  • esnek akış: <‘esnek yön’> || <‘esnek sarma’>
  • haklı içerik: esnek başlangıç ​​| esnek uç | merkez | boşluk | uzay-etrafında;
  • hizalama öğeleri: esnek başlangıç ​​| esnek uç | merkez | temel | uzatmak;
  • hizalama içeriği: esnek başlangıç ​​| esnek uç | merkez | boşluk | uzay-etrafında | uzatmak;
  • sıra: <tamsayı>;
  • esnek büyüme: <sayı>; /* varsayılan 0 */
  • flex-shrink: <sayı>; /* varsayılan 1 */
  • esnek temel: <uzunluk> | Oto; /* varsayılan otomatik */
  • esnek: yok | [ <‘esnek büyüme’> <‘esnek küçültme’>? || <‘esnek temel’> ]
  • kendini hizala: otomatik | esnek başlangıç ​​| esnek uç | merkez | temel | uzatmak;

Satıcı Önekleri #

  • ekran: -webkit-kutusu; /* Krom <20 */
  • ekran: -webkit-flex; /* Krom 20+ */
  • görüntüleme: -moz kutusu; /* Firefox */
  • ekran: -ms-flexbox; /* IE */
  • ekran: esnek; /* Modern tarayıcılar */

Dinamik Dikey ve Yatay Merkezleme (öğeleri hizalama, yaslama içeriği) #

Basit Örnek (tek bir öğeyi merkezleme) #

HTML #

<div class="aligner">
  <div class="aligner-item">…</div>
</div>

CSS #

.aligner {
  display: flex; 
  align-items: center;
  justify-content: center;
}

.aligner-item {
  max-width: 50%; /*for demo. Use actual width instead.*/
}

Akıl yürütme #

Mülk Değer Açıklama
‘öğeleri hizala’ “merkez” Bu, “esnek-yön” ile belirtilenden farklı olan öğeleri eksen boyunca ortalar, yani yatay esnek kutu için dikey merkezleme ve dikey esnek kutu için yatay merkezleme.
‘haklı içerik’ “merkez” Bu, öğeleri “esnek yön” tarafından belirtilen eksen boyunca ortalar. Yani, yatay (esnek-yön: satır) bir esnek kutu için, bu yatay olarak ortalanır ve dikey bir esnek kutu (esnek-yön: sütun) bir esnek kutu için dikey olarak ortalanır)

Yapışkan Değişken Yükseklik Alt Bilgisi #

Bu kod, yapışkan bir altbilgi oluşturur. İçerik görünümün sonuna ulaşmadığında, alt bilgi görünümün altına yapışır. İçerik, görünümün alt kısmını aştığında, alt bilgi de görünüm alanının dışına itilir. Sonucu Görüntüle

HTML:

<div class="header">
  <h2>Header</h2>
</div>

<div class="content">
  <h1>Content</h1>
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio. Praesent libero. Sed cursus ante dapibus diam. Sed nisi. Nulla quis sem at nibh elementum imperdiet. Duis sagittis ipsum. Praesent mauris. Fusce nec tellus sed augue semper porta. Mauris massa. Vestibulum lacinia arcu eget nulla. Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos. Curabitur sodales ligula in libero. </p>
</div>

<div class="footer">
  <h4>Footer</h4>
</div>

CSS:

html, body {
  height: 100%;
}

body {
  display: flex;
  flex-direction: column;
}

.content {
  /* Include `0 auto` for best browser compatibility. */
  flex: 1 0 auto;
}

.header, .footer {
  background-color: grey;
  color: white;
  flex: none;
}

Öğeleri kapsayıcılarına en uygun şekilde sığdırın #

Flexbox’ın en güzel özelliklerinden biri, kapların ana öğelerine en uygun şekilde oturmasına izin vermesidir.

Canlı demo.

HTML:

<div class="flex-container">
  <div class="flex-item">1</div>
  <div class="flex-item">2</div>
  <div class="flex-item">3</div>
  <div class="flex-item">4</div>
  <div class="flex-item">5</div>
</div>

CSS:

.flex-container {
    background-color: #000;
    height: 100%;
    display:flex;
    flex-direction: row;
    flex-wrap: wrap;
    justify-content: flex-start;
    align-content: stretch;
    align-items: stretch;
}

.flex-item {
    background-color: #ccf;
    margin: 0.1em;
    flex-grow: 1;
    flex-shrink: 0;
    flex-basis: 200px; /* or % could be used to ensure a specific layout */
}

Sütunlar, ekran yeniden boyutlandırıldıkça uyarlanır.

Flexbox kullanarak Kutsal Kase Düzeni #

Kutsal Kase düzeni, sabit yükseklikte üstbilgi ve altbilgiye ve 3 sütunlu bir merkeze sahip bir düzendir. 3 sütun, sabit genişlikte bir yan navigasyon, bir değişken merkez ve reklamlar gibi diğer içerikler için bir sütun içerir (değişken merkez, işaretlemede önce görünür). CSS Flexbox bunu çok basit bir işaretleme ile başarmak için kullanılabilir:

HTML İşaretleme:

<div class="container">
  <header class="header">Header</header>
  <div class="content-body">
    <main class="content">Content</main>
    <nav class="sidenav">Nav</nav>
    <aside class="ads">Ads</aside>
  </div>
  <footer class="footer">Footer</footer>
</div>

CSS:

body {
  margin: 0;
  padding: 0;
}

.container {
  display: flex;
  flex-direction: column;
  height: 100vh;
}

.header {
  flex: 0 0 50px;
}

.content-body {
  flex: 1 1 auto;
  
  display: flex;
  flex-direction: row;
}

.content-body .content {
  flex: 1 1 auto;
  overflow: auto;
}

.content-body .sidenav {
  order: -1;
  flex: 0 0 100px;
  overflow: auto;
}

.content-body .ads {
  flex: 0 0 100px;
  overflow: auto;
}

.footer {
  flex: 0 0 50px;
}

Flexbox’lı kartların içinde mükemmel şekilde hizalanmış düğmeler #

Bu günlerde, içerdiği kartların içindeki eylem çağrısını dikey olarak hizalamak tasarımda normal bir kalıptır:

Bu, ‘flexbox’ ile özel bir numara kullanılarak başarılabilir.

HTML

<div class="cards">
  <div class="card">
    <p>Lorem ipsum Magna proident ex anim dolor ullamco pariatur reprehenderit culpa esse enim mollit labore dolore voluptate ullamco et ut sed qui minim.</p>
    <p><button>Action</button></p>
  </div>
  <div class="card">
    <p>Lorem ipsum Magna proident ex anim dolor ullamco pariatur reprehenderit culpa esse enim mollit labore dolore voluptate ullamco et ut sed qui minim.</p>
    <p>Lorem ipsum Magna proident ex anim dolor ullamco pariatur reprehenderit culpa esse enim mollit labore dolore voluptate ullamco et ut sed qui minim.</p>
    <p>Lorem ipsum Magna proident ex anim dolor ullamco pariatur reprehenderit culpa esse enim mollit labore dolore voluptate ullamco et ut sed qui minim.</p>
    <p>Lorem ipsum Magna proident ex anim dolor ullamco pariatur reprehenderit culpa esse enim mollit labore dolore voluptate ullamco et ut sed qui minim.</p>
    <p><button>Action</button></p>
  </div>
</div>

Her şeyden önce, kapsayıcıya display: flex; uygulamak için CSS kullanıyoruz. Bu, içinde doğal olarak akan içerikle eşit yükseklikte 2 sütun oluşturacaktır.

CSS

.cards {
    display: flex;
}
.card {
    border: 1px solid #ccc;
    margin: 10px 10px;
    padding: 0 20px;
}
button {
    height: 40px;
    background: #fff;
    padding: 0 40px;
    border: 1px solid #000;
}
p:last-child {
    text-align: center;
}

Düzen değişecek ve şöyle olacak:

Butonları bloğun en altına taşımak için, yön ‘sütun’ olarak ayarlıyken kartın kendisine ‘display: flex;’ uygulamamız gerekiyor. Bundan sonra, kartın içindeki son elemanı seçmeli ve ‘margin-top’u ‘auto’ olarak ayarlamalıyız. Bu, son paragrafı kartın altına itecek ve istenen sonucu elde edecektir.

Son CSS:

.cards {
    display: flex;
}
.card {
    border: 1px solid #ccc;
    margin: 10px 10px;
    padding: 0 20px;
    display: flex;
    flex-direction: column;
}
button {
    height: 40px;
    background: #fff;
    padding: 0 40px;
    border: 1px solid #000;
}
p:last-child {
    text-align: center;
    margin-top: auto;
}

İç içe kaplarda aynı yükseklik #

Bu kod, tüm iç içe kapların her zaman aynı yükseklikte olmasını sağlar. Bu, tüm iç içe öğelerin, içeren ana div ile aynı yükseklikte olması sağlanarak yapılır. Çalışma örneğine bakın: https://jsfiddle.net/3wwh7ewp/

Bu efekt, “öğeleri hizala” özelliğinin varsayılan olarak “uzat” olarak ayarlanması nedeniyle elde edilir.

HTML

<div class="container">
  <div style="background-color: red">
    Some <br />
    data <br />
    to make<br />
    a height <br />
 </div>
  <div style="background-color: blue">
    Fewer <br />
    lines <br />
 </div>
</div>

CSS

.container {
    display: flex;
    align-items: stretch; // Default value
}

Not: 10’un altındaki IE sürümlerinde çalışmaz
