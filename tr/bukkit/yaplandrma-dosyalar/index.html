Sözdizimi #

  • String s = config.getString("path.to.string");
  • int i = config.getInt("path.to.int");
  • double d = config.getDouble("path.to.double");
  • List<String> sl = config.getStringList("path.to.stringlist");
  • List<Double> dl = config.getDoubleList("path.to.doublelist");
  • List<Integer> il = config.getIntegerList("path.to.integerlist");

Bukkit yapılandırma dosyaları doğrudan Y.A.M.L (Yine Başka Bir İşaretleme Dili) dosyalarıdır ve bu şekilde uygulanır.

Eklenti Config.yml #

Doğrudan jar dosyanızdan yüklenen bir config.yml dosyanız olabilir. Plugin.yml dosyasıyla aynı şekilde projenizin klasörüne eklenmelidir.

Bu dosyada, yapılandırmanız için varsayılan değerlere sahipsiniz.

Örnek yapılandırma:

# This is an YML comment
adminName: "Kerooker"
moderators: ["Romario", "Pelé", "Cafú"]

Örnek yapılandırma dosyası proje klasörüne eklenmelidir.

Varsayılan yapılandırma dosyasını eklentinizin klasörüne yüklemek için onEnable() öğenize aşağıdaki kod eklenmelidir:

saveDefaultConfig();

Bu, projedeki config.yml dosyanızı eklentinizin yapılandırma dosyası yapacak ve onu eklentinizin klasörüne ekleyecektir.

Buradan, eklenti örneğinizi kullanarak yapılandırma dosyanıza her yerden erişebilirsiniz:

JavaPlugin plugin; // Your plugin instance
FileConfiguration config = plugin.getConfig();  //Accessing the config file

Oradan, eklentinin yapılandırmasında ayarlanan her şeye erişebiliriz.

Not: Kullanıcı, klasörde oluşturulan config.yml dosyasını düzenlemek isterse, varsayılan yapılandırma dosyasının değerleri değişebilir.

String adminName = config.getString("adminName");
List<String> moderators =  config.getStringList("moderators");

Çoklu Yol Bölümü #

Yapılandırma dosyanızda olabilecek şey, birden çok bölümden geçen bir değişkenin yolunun bulunmasıdır.

Örnek Yapılandırma

admins:
 first-tier: "Kerooker"
 second-tier: "Mordekaiser"
 third-tier: "Yesh4"

“Kerooker” adı, “yöneticiler” bölümündeki “birinci kademe” bölümünden gelmektedir. Dosyamızın iç yollarına erişmek için basit bir ‘.’ kullanıyoruz. sonraki bölümü istediğimizi söylemenin bir yolu olarak. Yani, “Kerooker"a erişmemiz için şunu yapıyoruz:

config.getString("admins.first-tier");
