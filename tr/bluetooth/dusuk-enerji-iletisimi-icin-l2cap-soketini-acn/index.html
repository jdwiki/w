Düşük Enerji iletişimi için L2CAP soketini açın

  • C’de, Bluez ile

Bu sayfada

  • C’de, Bluez ile

C’de, Bluez ile #

int get_l2cap_connection () {

Öncelikle, ihtiyacımız olan tüm değişkenler, açıklamaları uygun yerde gelecektir.

    int ssock = 0;
    int csock = 0;
    int reuse_addr = 1;
    struct sockaddr_l2 src_addr;
    struct bt_security bt_sec;
    int result = 0;

Öncelikle, bağlantı kabul edebileceğimiz bir soket oluşturmamız gerekiyor. Soket ailesi ‘PF_BLUETOOTH’, soket tipi ‘SOCK_SEQPACKET’ (raw değil, TCP benzeri bir sokete sahip olmak istiyoruz) ve protokol Bluetooth protokolü L2CAP (‘BTPROTO_L2CAP’).

    ssock = socket(PF_BLUETOOTH, SOCK_SEQPACKET, BTPROTO_L2CAP);

Başarılı olduğundan emin olmak istiyoruz:

    if (ssock < 0) {
        perror("Opening L2CAP socket failed");
        return -1;
    }

Artık kaynak adres yapısını bir joker adresle doldurmamız gerekiyor, böylece herhangi bir adrese sahip herhangi bir Bluetooth cihazı bize bağlanabilir. Joker karakter adresi, “bluetooth.h” içinde “BDADDR_ANY” olarak tanımlanır. Adres yapısına kopyalamak için bacpy fonksiyonunu kullanabiliriz. Ayrıca adres ailesini, adres türünü ve kanal kimliğini de ayarlamamız gerekiyor.

    memset(&src_addr, 0, sizeof(src_addr));
    bacpy(&src_addr.l2_bdaddr, BDADDR_ANY);
    src_addr.l2_family = AF_BLUETOOTH;
    src_addr.l2_bdaddr_type = BDADDR_LE_PUBLIC;
    src_addr.l2_cid = htobs(CID_ATT);

SO_REUSEADDR seçeneğinin ayarlanması, gerekirse bind’i tekrar hızlı bir şekilde çağırmamıza izin verecektir (bu atlanabilir):

    setsockopt(ssock, SOL_SOCKET, SO_REUSEADDR, &reuse_addr, sizeof(reuse_addr));

Daha sonra, az önce tanımladığımız kaynak adres yapısı ile soketi bağlamamız gerekiyor. Yine, çalıştığından emin olmak için dönüş değerini kontrol ediyoruz.

    result = bind(ssock, (struct sockaddr*) &src_addr, sizeof(src_addr));
    if (result < 0) {
        perror("Binding L2CAP socket failed");
        return -1;
    }

Sıradaki güvenlik seviyesini ayarlamak. Bu adımın isteğe bağlı olduğunu unutmayın, ancak güvenlik düzeyini ORTA olarak ayarlamak, cihazla otomatik eşleştirmeye izin verecektir (çekirdek gerçek eşleştirmeyi gerçekleştirir).

    memset(&bt_sec, 0, sizeof(bt_sec));
    bt_sec.level = BT_SECURITY_MEDIUM;
    result = setsockopt(ssock, SOL_BLUETOOTH, BT_SECURITY, &bt_sec, sizeof(bt_sec));
    if (result != 0) {
        perrorno("Setting L2CAP security level failed");
        return -1;
    }

Şimdi çekirdeğe ssock’ımızın bir bağlantıyı kabul edecek pasif bir soket olduğunu söyleyebiliriz. İkinci parametre biriktirme listesidir. Daha fazlasını öğrenmek istiyorsanız, listen’in kılavuz sayfası ihtiyacınız olan tüm bilgileri içerir.

    result = listen(ssock, 10);
    if (result < 0) {
        perror("Listening on L2CAP socket failed");
        return -1;
    }

Şimdi gelen bir bağlantı için bekleyebiliriz. Peer_addr yapısı, geri dönüşleri kabul ettikten sonra bağlı cihazın adresini içerecektir. csock, bağlı cihazla iletişim kurmak için okuyabileceğimiz/yazabileceğimiz soketin dosya tanımlayıcısı olacaktır.

    memset(peer_addr, 0, sizeof(*peer_addr));
    socklen_t addrlen = sizeof(*peer_addr);
    csock = accept(ssock, (struct sockaddr*)peer_addr, &addrlen);
    if (csock < 0) {
        perror("Accepting connection on L2CAP socket failed");
        return -1;
    }

Bağlı cihazın adresini yazdırabiliriz (isteğe bağlı, elbette). Bluetooth adresini bir dizgeye dönüştürmek için batostr işlevini kullanabiliriz.

    printf("Accepted connection from %s", batostr(&peer_addr->l2_bdaddr));

Başka bir cihazın bağlanmasını istemiyorsak sunucu soketini kapatmalıyız. Cihazla iletişiminiz bittikten sonra aynı işlemi csock ile yapın.

    close(ssock);
    return csock;
}
