Aralıklarla ve Sabit Adla Grafik Oluşturma #

Grafikler, doğrudan grafik verilerini tanımlayan ‘Seri’ nesnesi ile çalışılarak oluşturulabilir. Mevcut bir grafik olmadan ‘Seri’ye ulaşmak için, verilen bir ‘Çalışma Sayfası’nda bir ‘ChartObject’ yaratır ve ondan ‘Chart’ nesnesini alırsınız. ‘Seri’ nesnesiyle çalışmanın iyi tarafı, ‘Aralık’ nesnelerine başvurarak ‘Değerler’ ve ‘XDeğerler’i ayarlayabilmenizdir. Bu veri özellikleri, bu aralıklara atıfta bulunarak “Seri"yi doğru şekilde tanımlayacaktır. Bu yaklaşımın dezavantajı, ‘Ad’ belirlenirken aynı dönüştürmenin yapılmamasıdır; sabit bir değerdir. Orijinal “Aralık"taki temel verilerle uyum sağlamayacaktır. ‘SERİ’ formülü kontrol edildiğinde adın sabit olduğu açıktır. Bu, doğrudan “SERİ” formülü oluşturularak ele alınmalıdır.

Grafik oluşturmak için kullanılan kod

Bu kodun “Grafik” ve “Çalışma Sayfası” için fazladan değişken bildirimleri içerdiğine dikkat edin. Kullanılmazlarsa, bunlar atlanabilir. Bununla birlikte, stili veya diğer grafik özelliklerini değiştiriyorsanız faydalı olabilirler.

Sub CreateChartWithRangesAndFixedName()

    Dim xData As Range
    Dim yData As Range
    Dim serName As Range
    
    'set the ranges to get the data and y value label
    Set xData = Range("B3:B12")
    Set yData = Range("C3:C12")
    Set serName = Range("C2")
    
    'get reference to ActiveSheet
    Dim sht As Worksheet
    Set sht = ActiveSheet
    
    'create a new ChartObject at position (48, 195) with width 400 and height 300
    Dim chtObj As ChartObject
    Set chtObj = sht.ChartObjects.Add(48, 195, 400, 300)
    
    'get reference to chart object
    Dim cht As Chart
    Set cht = chtObj.Chart
    
    'create the new series
    Dim ser As Series
    Set ser = cht.SeriesCollection.NewSeries
    
    ser.Values = yData
    ser.XValues = xData
    ser.Name = serName
    
    ser.ChartType = xlXYScatterLines

End Sub

Orijinal veriler/aralıklar ve kod çalıştırıldıktan sonra ortaya çıkan “Grafik”

“SERİ” formülünün, seri adı için onu oluşturan “Aralık"a referans yerine “B” içerdiğini unutmayın.

Boş bir Grafik Oluşturma #

Grafik kodunun büyük çoğunluğunun başlangıç ​​noktası, boş bir “Grafik” oluşturmaktır. Bu “Grafik"in etkin olan ve aslında boş olmayabilecek varsayılan grafik şablonuna tabi olduğunu unutmayın (şablon değiştirilmişse).

“ChartObject"in anahtarı, konumunu belirlemektir. Çağrının sözdizimi ChartObjects.Add(Left, Top, Width, Height) şeklindedir. ‘ChartObject’ oluşturulduktan sonra, grafiği gerçekten değiştirmek için ‘Chart’ nesnesini kullanabilirsiniz. “ChartObject”, grafiği sayfa üzerinde konumlandırmak için daha çok bir “Şekil” gibi davranır.

Boş bir grafik oluşturmak için kod

Sub CreateEmptyChart()
    
    'get reference to ActiveSheet
    Dim sht As Worksheet
    Set sht = ActiveSheet
    
    'create a new ChartObject at position (0, 0) with width 400 and height 300
    Dim chtObj As ChartObject
    Set chtObj = sht.ChartObjects.Add(0, 0, 400, 300)
    
    'get refernce to chart object
    Dim cht As Chart
    Set cht = chtObj.Chart
    
    'additional code to modify the empty chart
    '...

End Sub

SERIES formülünü Değiştirerek Grafik Oluşturun #

Yeni bir “Grafik” ve “Seri” nesnesi üzerinde tam kontrol için (özellikle dinamik bir “Seri” adı için), “SERİ” formülünü doğrudan değiştirmeye başvurmalısınız. ‘Range’ nesnelerini kurma süreci basittir ve ana engel basitçe ‘SERIES’ formülü için dizi oluşturmaktır.

“SERIES” formülü aşağıdaki sözdizimini alır:

=SERIES(Name,XValues,Values,Order)

Bu içerikler, veri öğeleri için referanslar veya dizi değerleri olarak sağlanabilir. “Sipariş”, grafikteki seri konumunu temsil eder. Sayfa adıyla tam olarak nitelenmedikçe verilere yapılan başvuruların çalışmayacağını unutmayın. Çalışan bir formül örneği için, mevcut herhangi bir seriyi tıklayın ve formül çubuğunu kontrol edin.

“SERIES” formülünü kullanarak bir grafik oluşturma ve verileri ayarlama kodu

‘SERIES’ formülünü oluşturmak için dize oluşturmanın ‘.Address(,,,True)’ kullandığını unutmayın. Bu, harici Aralık referansının kullanılmasını sağlar, böylece sayfa adıyla tam nitelikli bir adres eklenir. Sayfa adı hariç tutulursa bir hata alırsınız.

Sub CreateChartUsingSeriesFormula()

    Dim xData As Range
    Dim yData As Range
    Dim serName As Range
    
    'set the ranges to get the data and y value label
    Set xData = Range("B3:B12")
    Set yData = Range("C3:C12")
    Set serName = Range("C2")
    
    'get reference to ActiveSheet
    Dim sht As Worksheet
    Set sht = ActiveSheet
    
    'create a new ChartObject at position (48, 195) with width 400 and height 300
    Dim chtObj As ChartObject
    Set chtObj = sht.ChartObjects.Add(48, 195, 400, 300)
    
    'get refernce to chart object
    Dim cht As Chart
    Set cht = chtObj.Chart
    
    'create the new series
    Dim ser As Series
    Set ser = cht.SeriesCollection.NewSeries
    
    'set the SERIES formula
    '=SERIES(name, xData, yData, plotOrder)
    
    Dim formulaValue As String
    formulaValue = "=SERIES(" & _
        serName.Address(, , , True) & "," & _
        xData.Address(, , , True) & "," & _
        yData.Address(, , , True) & ",1)"
    
    ser.Formula = formulaValue
    ser.ChartType = xlXYScatterLines

End Sub

Orijinal veriler ve sonuç tablosu

Bu grafik için, seri adının istenen hücreye bir aralıkla uygun şekilde ayarlandığını unutmayın. Bu, güncellemelerin “Grafik"e yayılacağı anlamına gelir.

Grafikleri Izgaraya Düzenleme #

Excel’deki çizelgelerle ortak bir görev, tek bir sayfada birden çok grafiğin boyutunu ve düzenini standart hale getirmektir. El ile yapılırsa, hücre sınırlarına “yapışmak” için grafiği yeniden boyutlandırırken veya taşırken ALT tuşunu basılı tutabilirsiniz. Bu, birkaç çizelge için işe yarar, ancak bir VBA yaklaşımı çok daha basittir.

Bir ızgara oluşturmak için kod

Bu kod, belirli bir (Üst, Sol) konumdan başlayarak, tanımlanmış sayıda sütun ve tanımlanmış bir ortak grafik boyutu ile bir grafik ızgarası oluşturacaktır. Grafikler oluşturuldukları sıraya göre yerleştirilecek ve yeni bir satır oluşturmak için kenarlara sarılacaktır.

Sub CreateGridOfCharts()

    Dim int_cols As Integer
    int_cols = 3
    
    Dim cht_width As Double
    cht_width = 250
        
    Dim cht_height As Double
    cht_height = 200
    
    Dim offset_vertical As Double
    offset_vertical = 195
    
    Dim offset_horz As Double
    offset_horz = 40

    Dim sht As Worksheet
    Set sht = ActiveSheet

    Dim count As Integer
    count = 0
    
    'iterate through ChartObjects on current sheet
    Dim cht_obj As ChartObject
    For Each cht_obj In sht.ChartObjects
        
        'use integer division and Mod to get position in grid
        cht_obj.Top = (count \ int_cols) * cht_height + offset_vertical
        cht_obj.Left = (count Mod int_cols) * cht_width + offset_horz
        cht_obj.Width = cht_width
        cht_obj.Height = cht_height

        count = count + 1

    Next cht_obj
End Sub

Birkaç grafikle sonuç

Bu resimler, grafiklerin orijinal rastgele düzenini ve yukarıdaki kodu çalıştırarak elde edilen ızgarayı gösterir.
