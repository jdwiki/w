Anında Pencere #

Tüm bir alt çalıştırmaya gerek kalmadan bir makro kodu satırını test etmek isterseniz, komutları doğrudan Anında Pencereye yazabilir ve satırı çalıştırmak için ‘ENTER’ tuşuna basabilirsiniz.

Bir satırın çıktısını test etmek için, doğrudan Anlık Pencereye yazdırmak için önüne bir soru işareti ? koyabilirsiniz. Alternatif olarak, çıktının yazdırılması için “yazdır” komutunu da kullanabilirsiniz.

Visual Basic Düzenleyicisi’ndeyken, Anlık Pencereyi açmak için CTRL + G tuşlarına basın. Seçili sayfanızı “Örnek Sayfa” olarak yeniden adlandırmak için, Anında Pencereye aşağıdakini yazın ve ‘ENTER’ tuşuna basın.

   ActiveSheet.Name = "ExampleSheet"

Şu anda seçili olan sayfanın adını doğrudan Anında Pencerede yazdırmak için

? ActiveSheet.Name
ExampleSheet

Bu yöntem, kodda uygulamadan önce yerleşik veya kullanıcı tanımlı işlevlerin işlevselliğini test etmek için çok yararlı olabilir. Aşağıdaki örnek, bir işlevin veya bir dizi işlevin bir bekleneni doğrulamak için çıktısını test etmek için Anında Pencerenin nasıl kullanılabileceğini gösterir.

'In this example, the Immediate Window was used to confirm that a series of Left and Right 
'string methods would return the desired string

'expected output: "value"
print Left(Right("1111value1111",9),5) ' <---- written code here, ENTER pressed
value                                  ' <---- output

Anında Pencere, Uygulama, Çalışma Kitabı veya diğer gerekli özellikleri ayarlamak veya sıfırlamak için de kullanılabilir. Bu, bir alt yordamda beklenmedik bir şekilde bir hata oluşturan ve değeri “True” değerine sıfırlamadan kapanmasına neden olan (sinir bozucu ve beklenmeyen işlevselliğe neden olabilir) “Application.EnableEvents = False” varsa yararlı olabilir. Bu durumda, komutlar Doğrudan Anlık Pencereye yazılabilir ve çalıştırılabilir:

? Application.EnableEvents       ' <---- Testing the current state of "EnableEvents"
False                            ' <---- Output
Application.EnableEvents = True  ' <---- Resetting the property value to True
? Application.EnableEvents       ' <---- Testing the current state of "EnableEvents"
True                             ' <---- Output

Daha gelişmiş hata ayıklama teknikleri için, satır ayırıcı olarak iki nokta üst üste : kullanılabilir. Bu, aşağıdaki örnekte döngü gibi çok satırlı ifadeler için kullanılabilir.

x = Split("a,b,c",","): For i = LBound(x,1) to UBound(x,1): Debug.Print x(i): Next i '<----Input this and press enter
a '<----Output
b '<----Output
c '<----Output

Hata Ayıklama.Yazdır #

Anında Pencereye Hata Kodu açıklamalarının bir listesini yazdırmak için, bunu ‘Debug.Print’ işlevine iletin:

Private Sub ListErrCodes()
    Debug.Print "List Error Code Descriptions"
    For i = 0 To 65535
        e = Error(i)
        If e <> "Application-defined or object-defined error" Then Debug.Print i & ": " & e
    Next i
End Sub

Anında Pencereyi şu şekilde gösterebilirsiniz:

  • **Görünüm | **Menü çubuğundan **acil Pencere
  • Klavye kısayolunu kullanma Ctrl-G

Performanstaki Darboğazları Bulmak için Zamanlayıcıyı Kullanın #

Hızı optimize etmenin ilk adımı, kodun en yavaş bölümlerini bulmaktır. ‘Zamanlayıcı’ VBA işlevi, Windows tabanlı bilgisayarlarda saniyenin 1/256’sı (3.90625 milisaniye) hassasiyetle gece yarısından bu yana geçen saniye sayısını döndürür. VBA işlevleri “Şimdi” ve “Zaman” yalnızca bir saniyeye kadar doğrudur.

Dim start As Double       ' Timer returns Single, but converting to Double to avoid 
start = Timer             ' scientific notation like 3.90625E-03 in the Immediate window
' ... part of the code
Debug.Print Timer - start; "seconds in part 1" 

start = Timer
' ... another part of the code
Debug.Print Timer - start; "seconds in part 2"

Hata Ayıklayıcı Yerelleri Penceresi #

Locals penceresi, çalıştırdığınız fonksiyon veya alt yordam kapsamındaki değişkenlerin ve nesnelerin mevcut değerine kolay erişim sağlar. Sorunları bulmak için kodunuzda hata ayıklamak ve değişiklikler arasında adım atmak için önemli bir araçtır. Ayrıca var olduğunu bilmediğiniz mülkleri keşfetmenize de olanak tanır.

Aşağıdaki örneği alın,

Option Explicit
Sub LocalsWindowExample()
    Dim findMeInLocals As Integer
    Dim findMEInLocals2 As Range
    
    findMeInLocals = 1
    Set findMEInLocals2 = ActiveWorkbook.Sheets(1).Range("A1")
End Sub

VBA Düzenleyicide, Görünüm –> Yereller Penceresi’ni tıklayın.

Daha sonra alt rutinin içine tıkladıktan sonra F8’i kullanarak kodu adımlayarak, findMeinLocals’ı atamadan önce durduk. Aşağıda değerin 0 olduğunu görebilirsiniz — ve ona hiç bir değer atamamış olsaydınız bu kullanılacaktı. Aralık nesnesi ‘Hiçbir şey’dir.

Altprogram bitmeden hemen önce durursak, değişkenlerin son değerlerini görebiliriz.

![Bir hata ayıklama noktası ayarlayın… isterseniz]]4

1 değerinde ve Tamsayı türünde findMeInLocals’ı ve Aralık/Aralık türünde FindMeInLocals2’yi görebiliriz. + işaretine tıklarsak nesneyi genişletebilir ve sayım veya sütun gibi özelliklerini görebiliriz.

Durmak #

Durdur komutu, çağrıldığında yürütmeyi duraklatacaktır. Oradan, süreç devam ettirilebilir veya adım adım yürütülebilir.

Sub Test()
    Dim TestVar as String
    TestVar = "Hello World"
    Stop                    'Sub will be executed to this point and then wait for the user
    MsgBox TestVar
End Sub

Kodunuza bir Kesme Noktası ekleme #

Yürütmenin durdurulmasını istediğiniz VBA kod satırının solundaki gri sütuna tıklayarak kodunuza kolayca bir kesme noktası ekleyebilirsiniz. Sütunda kırmızı bir nokta görünür ve kesme noktası kodu da kırmızıyla vurgulanır.

Kodunuz boyunca birden çok kesme noktası ekleyebilirsiniz ve menü çubuğunuzdaki “oynat” simgesine basarak yürütmeye devam edebilirsiniz. Tüm kodlar değişken tanımlama satırları olarak bir kesme noktası olamaz, bir prosedürün ilk veya son satırı ve açıklama satırları kesme noktası olarak seçilemez.
