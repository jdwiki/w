*) Satır numaraları tamsayıları temsil eder, yani -32.768 ila 32.767 aralığında işaretli 16 bitlik bir veri türüdür, aksi takdirde bir taşma oluşturursunuz. Genellikle satır numaraları, kodun bir kısmı veya bir bütün olarak bir modülün tüm prosedürleri üzerine 10’luk adımlarla eklenir.

Çalışma Sayfası Güncellemesini Devre Dışı Bırakma #

Çalışma sayfasının hesaplanmasını devre dışı bırakmak, makronun çalışma süresini önemli ölçüde azaltabilir. Ayrıca olayları, ekran güncellemelerini ve sayfa sonlarını devre dışı bırakmak faydalı olacaktır. Aşağıdaki Sub bu amaçla herhangi bir makroda kullanılabilir.

Sub OptimizeVBA(isOn As Boolean)
    Application.Calculation = IIf(isOn, xlCalculationManual, xlCalculationAutomatic)
    Application.EnableEvents = Not(isOn)
    Application.ScreenUpdating = Not(isOn)
    ActiveSheet.DisplayPageBreaks = Not(isOn)
End Sub

Optimizasyon için aşağıdaki sözde kodu izleyin:

Sub MyCode()
    
    OptimizeVBA True

    'Your code goes here

    OptimizeVBA False

End Sub

Genişletilmiş Hata Ayıklama ile Hata Aramayı Optimize Etme #

Satır Numaralarının Kullanılması … ve hata durumunda belgelenmesi (“Erl’i görmenin önemi”)

Hangi satırın bir hata oluşturduğunu saptamak, herhangi bir hata ayıklamanın önemli bir parçasıdır ve neden aramayı daraltır. Belirlenen hata satırlarını kısa bir açıklama ile belgelemek, en iyi ihtimalle modül ve prosedür adları ile birlikte başarılı bir hata takibini tamamlar. Aşağıdaki örnek, bu verileri bir günlük dosyasına kaydeder.

Arka fon

Hata nesnesi, hata numarasını (Err.Number) ve hata açıklamasını (Err.Description) döndürür, ancak hatanın nerede bulunacağı sorusuna açıkça yanıt vermez. Bununla birlikte, Erl işlevi yapar, ancak koda *satır numaraları ) eklemeniz koşuluyla (BTW, eski Temel zamanlara verilen diğer birkaç ayrıcalıktan biridir).

Hiç hata satırı yoksa, Erl işlevi 0 döndürür, numaralandırma eksikse prosedürün önceki son satır numarasını alırsınız.

Option Explicit


Public Sub MyProc1()
Dim i As Integer
Dim j As Integer
On Error GoTo LogErr
10     j = 1 / 0    ' raises an error
okay:
Debug.Print "i=" & i
Exit Sub

LogErr:
MsgBox LogErrors("MyModule", "MyProc1", Err), vbExclamation, "Error " & Err.Number
Stop
Resume Next
End Sub

Public Function LogErrors( _
           ByVal sModule As String, _
           ByVal sProc As String, _
           Err As ErrObject) As String
' Purpose: write error number, description and Erl to log file and return error text
  Dim sLogFile As String: sLogFile = ThisWorkbook.Path & Application.PathSeparator & "LogErrors.txt"
  Dim sLogTxt  As String
  Dim lFile    As Long

' Create error text
  sLogTxt = sModule & "|" & sProc & "|Erl " & Erl & "|Err " & Err.Number & "|" & Err.Description

  On Error Resume Next
  lFile = FreeFile

  Open sLogFile For Append As lFile
  Print #lFile, Format$(Now(), "yy.mm.dd hh:mm:ss "); sLogTxt
      Print #lFile,
  Close lFile
' Return error text
  LogErrors = sLogTxt
 End Function

Yürütme zamanını kontrol etme #

Farklı prosedürler aynı sonucu verebilir, ancak farklı işlem süreleri kullanırlar. Hangisinin daha hızlı olduğunu kontrol etmek için şöyle bir kod kullanılabilir:

time1 = Timer

For Each iCell In MyRange
   iCell = "text"
Next iCell

time2 = Timer

For i = 1 To 30
   MyRange.Cells(i) = "text"
Next i

time3 = Timer

debug.print "Proc1 time: " & cStr(time2-time1)
debug.print "Proc2 time: " & cStr(time3-time2)

MicroTimer:

Private Declare PtrSafe Function getFrequency Lib "Kernel32" Alias "QueryPerformanceFrequency" (cyFrequency As Currency) As Long
Private Declare PtrSafe Function getTickCount Lib "Kernel32" Alias "QueryPerformanceCounter" (cyTickCount As Currency) As Long

Function MicroTimer() As Double
    Dim cyTicks1 As Currency
    Static cyFrequency As Currency

    MicroTimer = 0
    If cyFrequency = 0 Then getFrequency cyFrequency        'Get frequency
    getTickCount cyTicks1                                   'Get ticks
    If cyFrequency Then MicroTimer = cyTicks1 / cyFrequency 'Returns Seconds
End Function

Bloklarla Kullanma #

Bloklarla birlikte kullanmak, bir makro çalıştırma sürecini hızlandırabilir. Bir aralık, grafik adı, çalışma sayfası vb. yazmak yerine aşağıdaki gibi bloklarla kullanabilirsiniz;

With ActiveChart
    .Parent.Width = 400
    .Parent.Height = 145
    .Parent.Top = 77.5 + 165 * step - replacer * 15
    .Parent.Left = 5
End With

Hangisi bundan daha hızlı:

ActiveChart.Parent.Width = 400
ActiveChart.Parent.Height = 145
ActiveChart.Parent.Top = 77.5 + 165 * step - replacer * 15
ActiveChart.Parent.Left = 5

Notlar:

  • With bloğu girildiğinde, nesne değiştirilemez. Sonuç olarak, bir dizi farklı nesneyi etkilemek için tek bir With ifadesini kullanamazsınız.

  • Bloklarla bloklara atlamayın. With bloğundaki ifadeler yürütülürse, ancak With veya End With ifadesi yürütülmezse, nesneye bir başvuru içeren geçici bir değişken, siz prosedürden çıkana kadar bellekte kalır

  • Özellikle önbelleğe alınmış nesne bir yineleyici olarak kullanılıyorsa, With deyimlerinin içinde Döngü Yapma

  • Bir With bloğunu diğerinin içine yerleştirerek With ifadelerini iç içe yerleştirebilirsiniz. Ancak, dış With bloklarının üyeleri iç With blokları içinde maskelendiğinden, bir dış With bloğundaki bir nesnenin herhangi bir üyesine bir iç With bloğunda tam nitelikli bir nesne referansı sağlamalısınız.

Yuvalama Örneği:

Bu örnek, tek bir nesne üzerinde bir dizi ifadeyi yürütmek için With ifadesini kullanır. Nesne ve özellikleri, yalnızca açıklama amacıyla kullanılan genel adlardır.

With MyObject 
    .Height = 100               'Same as MyObject.Height = 100. 
    .Caption = "Hello World"    'Same as MyObject.Caption = "Hello World". 
    With .Font 
        .Color = Red            'Same as MyObject.Font.Color = Red. 
        .Bold = True            'Same as MyObject.Font.Bold = True. 
        MyObject.Height = 200   'Inner-most With refers to MyObject.Font (must be qualified
    End With
End With

Satır Silme - Performans #

  • Satırları silmek, özellikle hücreler arasında dolaşırken ve satırları tek tek silerken yavaştır

  • Silinecek satırları gizlemek için Otomatik Filtre kullanmak farklı bir yaklaşımdır.

  • Görünür aralığı kopyalayın ve yeni bir Çalışma Sayfasına yapıştırın

  • İlk sayfayı tamamen çıkarın

  • Bu yöntemle ne kadar çok satır silinirse o kadar hızlı olur

Örnek:

Option Explicit

'Deleted rows: 775,153, Total Rows: 1,000,009, Duration: 1.87 sec

Public Sub DeleteRows()
    Dim oldWs As Worksheet, newWs As Worksheet, wsName As String, ur As Range

    Set oldWs = ThisWorkbook.ActiveSheet
    wsName = oldWs.Name
    Set ur = oldWs.Range("F2", oldWs.Cells(oldWs.Rows.Count, "F").End(xlUp))

    Application.ScreenUpdating = False
    Set newWs = Sheets.Add(After:=oldWs)    'Create a new WorkSheet

    With ur    'Copy visible range after Autofilter (modify Criteria1 and 2 accordingly)
        .AutoFilter Field:=1, Criteria1:="<>0", Operator:=xlAnd, Criteria2:="<>"
        oldWs.UsedRange.Copy
    End With
    'Paste all visible data into the new WorkSheet (values and formats)
    With newWs.Range(oldWs.UsedRange.Cells(1).Address)
        .PasteSpecial xlPasteColumnWidths
        .PasteSpecial xlPasteAll
        newWs.Cells(1, 1).Select: newWs.Cells(1, 1).Copy
    End With

    With Application
        .CutCopyMode = False
        .DisplayAlerts = False
            oldWs.Delete
        .DisplayAlerts = True
        .ScreenUpdating = True
    End With
    newWs.Name = wsName
End Sub

Büyük makroları çalıştırmadan önce Tüm Excel İşlevlerini Devre Dışı Bırakma #

Aşağıdaki prosedürler, Çalışma Kitabı ve Çalışma Sayfası düzeyinde tüm Excel özelliklerini geçici olarak devre dışı bırakacaktır.

  • FastWB(), Açık veya Kapalı bayraklarını kabul eden bir geçiştir.

  • FastWS(), İsteğe Bağlı Çalışma Sayfası nesnesini kabul eder veya hiçbirini kabul etmez

  • ws parametresi eksikse, koleksiyondaki tüm Çalışma Sayfaları için tüm özellikleri açar ve kapatır

  • Kapatmadan önce tüm ayarları yakalamak için özel bir tür kullanılabilir

  • İşlemin sonunda başlangıç ​​ayarları geri yüklenebilir

Public Sub FastWB(Optional ByVal opt As Boolean = True)
    With Application
        .Calculation = IIf(opt, xlCalculationManual, xlCalculationAutomatic)
        If .DisplayAlerts <> Not opt Then .DisplayAlerts = Not opt
        If .DisplayStatusBar <> Not opt Then .DisplayStatusBar = Not opt
        If .EnableAnimations <> Not opt Then .EnableAnimations = Not opt
        If .EnableEvents <> Not opt Then .EnableEvents = Not opt
        If .ScreenUpdating <> Not opt Then .ScreenUpdating = Not opt
    End With
    FastWS , opt
End Sub

Public Sub FastWS(Optional ByVal ws As Worksheet, Optional ByVal opt As Boolean = True)
    If ws Is Nothing Then
        For Each ws In Application.ThisWorkbook.Sheets
            OptimiseWS ws, opt
        Next
    Else
        OptimiseWS ws, opt
    End If
End Sub
Private Sub OptimiseWS(ByVal ws As Worksheet, ByVal opt As Boolean)
    With ws
        .DisplayPageBreaks = False
        .EnableCalculation = Not opt
        .EnableFormatConditionsCalculation = Not opt
        .EnablePivotTable = Not opt
    End With
End Sub

Tüm Excel ayarlarını varsayılana geri yükle

Public Sub XlResetSettings()    'default Excel settings
    With Application
        .Calculation = xlCalculationAutomatic
        .DisplayAlerts = True
        .DisplayStatusBar = True
        .EnableAnimations = False
        .EnableEvents = True
        .ScreenUpdating = True
        Dim sh As Worksheet
        For Each sh In Application.ThisWorkbook.Sheets
            With sh
                .DisplayPageBreaks = False
                .EnableCalculation = True
                .EnableFormatConditionsCalculation = True
                .EnablePivotTable = True
            End With
        Next
    End With
End Sub
