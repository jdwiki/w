Bir aralık için üçten fazla koşullu biçim tanımlayamazsınız. Mevcut bir koşullu biçimi değiştirmek için Değiştir yöntemini kullanın veya yeni bir biçim eklemeden önce var olan bir biçimi silmek için Sil yöntemini kullanın.

FormatKoşullar.Ekle #

Sözdizimi: #

FormatConditions.Add(Type, Operator, Formula1, Formula2)

parametreler: #

İsim Gerekli / İsteğe Bağlı Veri tipi
Tip Gerekli XlFormatConditionType
Operatör Opsiyonel varyant
Formül1 Opsiyonel varyant
formül2 Opsiyonel varyant

XlFormatConditionType numaralandırma: #

İsim Tanım
xlAboveAverageKoşul Ortalamanın üzerinde durum
xlBlanksKoşul Boşluk durumu
xlCellValue hücre değeri
xlColorScale Renk skalası
xlVeri çubuğu veri çubuğu
xlErrorsKoşul Hata durumu
xlİfade İfade
XIconSet Simge seti
xlNoBlanksKoşul Boşluk yok koşulu
xlNoErrorsKoşul Hata durumu yok
xlTextString Metin dizesi
xlTimePeriod Zaman dilimi
xlTop10 En iyi 10 değer
xlUniqueDeğerler Benzersiz değerler

Hücre değerine göre biçimlendirme: #

With Range("A1").FormatConditions.Add(xlCellValue, xlGreater, "=100")
    With .Font
        .Bold = True
        .ColorIndex = 3
     End With
End With

Operatörler: #

İsim
xlArasında
xlEşit
xlBüyük
xlGreaterEqual
xlDaha Az
xlLessEqual
xlArada Değil
xlEşit Değil
Tür xlExpression ise, Operatör bağımsız değişkeni yok sayılır.

Metne göre biçimlendirme şunları içerir: #

With Range("a1:a10").FormatConditions.Add(xlTextString, TextOperator:=xlContains, String:="egg")
    With .Font
        .Bold = True
        .ColorIndex = 3
    End With
End With

Operatörler: #

İsim Tanım
xlBirlikte Başlıyor Belirtilen bir değerle başlar.
xlİçerir Belirtilen bir değeri içerir.
xlDoesNotContain Belirtilen değeri içermiyor.
xlBitiyor Belirtilen değerle biter

Zaman aralığına göre biçimlendirme #

With Range("a1:a10").FormatConditions.Add(xlTimePeriod, DateOperator:=xlToday)
    With .Font
        .Bold = True
        .ColorIndex = 3
    End With
End With

Koşullu biçimi kaldır #

Aralıktaki tüm koşullu biçimi kaldırın: #

Range("A1:A10").FormatConditions.Delete

Çalışma sayfasındaki tüm koşullu biçimi kaldırın: #

Cells.FormatConditions.Delete

FormatConditions.AddUniqueValues #

Yinelenen Değerleri Vurgulama #

With Range("E1:E100").FormatConditions.AddUniqueValues
   .DupeUnique = xlDuplicate
   With .Font
       .Bold = True
       .ColorIndex = 3
   End With
End With

Benzersiz Değerleri Vurgulamak #

With Range("E1:E100").FormatConditions.AddUniqueValues
   With .Font
       .Bold = True
       .ColorIndex = 3
   End With
End With

FormatKoşullar.AddTop10 #

İlk 5 Değeri Vurgulamak #

With Range("E1:E100").FormatConditions.AddTop10
    .TopBottom = xlTop10Top
    .Rank = 5
    .Percent = False
    With .Font
        .Bold = True
        .ColorIndex = 3
    End With
End With

FormatKoşullar.AddAboveAverage #

With Range("E1:E100").FormatConditions.AddAboveAverage
    .AboveBelow = xlAboveAverage
    With .Font
        .Bold = True
        .ColorIndex = 3
    End With
End With

Operatörler: #

İsim Tanım
XlAboveAverage Ortalamanın üstü
XlAboveStdDev standart sapmanın üstünde
XlOrtalamanın Altında Ortalamanın altında
XlBelowStdDev Standart sapmanın altında
XlEqualAboveAverage Ortalamanın üzerinde eşit
XlEqualBelowAverage Ortalamanın altında eşit

FormatConditions.AddIconSetCondition #

Range("a1:a10").FormatConditions.AddIconSetCondition
With Selection.FormatConditions(1)
    .ReverseOrder = False
    .ShowIconOnly = False
    .IconSet = ActiveWorkbook.IconSets(xl3Arrows)
End With

With Selection.FormatConditions(1).IconCriteria(2)
    .Type = xlConditionValuePercent
    .Value = 33
    .Operator = 7
End With

With Selection.FormatConditions(1).IconCriteria(3)
    .Type = xlConditionValuePercent
    .Value = 67
    .Operator = 7
End With
