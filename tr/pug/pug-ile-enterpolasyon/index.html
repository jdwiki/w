Web sitenizde sunucu taraflı değişkenleri kullanabilmeniz önemlidir. Pug, sunucunuz tarafından oluşturulan verileri HTML, CSS ve hatta JavaScript kodunda enterpolasyon yapmanıza olanak tanır.

Sözdizimi #

  • res.render(yol, değişkenler) // “yol” yolunda oluşturulacak bir pug dosyası arar ve ona “değişkenleri” iletir
  • #{değişken} // “Değişkeni” değerlendirdikten sonra, “değişkeni” çevreleyen Jade koduyla aynı hizada enterpolasyon yapar
  • !{değişken} // “Değişkeni” çevreleyen Jade koduyla aynı hizada, “değişkeni” değerlendirmeden enterpolasyon yapar.
  • #[element] // Mevcut bir Pug HTML öğesinin içindeki “eleman"ı enterpolasyon yapar. Enterpolasyonlu HTML öğelerinin sözdizimi, normal HTML öğelerinin sözdizimi ile aynıdır.

Parametreler #

parametre Ayrıntılar
yol “res.render” içinde kullanılır. Bu, oluşturacağımız Pug dosyasının yolu. Yol, Express uygulamanızda ayarlanan klasörün kökünden alınır: app.set("views", "templates/views"). Örneğin, “res.render(“index”)”, “templates/views/index.pug” konumunda bir Pug dosyası arayacaktır. Alt dizinler de belirtilebilir; res.render("admin/index"), templates/views/admin/index.pug konumunda bir Pug dosyası arar.
değişkenler “res.render” içinde kullanılır. ‘path’ (yukarıda) ile tanımlanan Pug dosyasına erişilebilir hale getirilecek değişkenlerden oluşan bir JavaScript nesnesi. Pug dosyası içinde, yukarıdaki JavaScript nesnesinin anahtarları değişkenler olarak kullanılabilir hale gelir. Değişkenler = {başlık: “Merhaba”, renk: “kırmızı”}’ ise, “başlık” ve “renk” değişkenini kullanabiliriz. İç içe nesnelerin alt özellikleri de mevcuttur.
değişken Köşeli ayraç sözdiziminde #{} veya !{} kullanılır. “Değişken"in değeri, onu çevreleyen Pug kodu bağlamında çıkarılacaktır. Açılış küme parantezinin başına bir pound sembolü eklenirse, çıktı alınmadan önce “değişken” değerlendirilir. Açılış küme parantezinin başına bir ünlem işareti eklenirse, “değişken” değerlendirilmeyecektir.
eleman Köşeli parantez sözdiziminde “#[]” kullanılır. HTML öğesi (Pug sözdiziminde, normal HTML sözdiziminde değil) değerlendirilecek ve çevresindeki Pug koduyla satır içi olarak çıkarılacaktır.

PugJS enterpolasyonu hakkında daha fazla bilgi için [resmi PugJS enterpolasyonu belgelerine] bakın.(https://pugjs.org/language/interpolation.html)

Sunucu Tarafı Değişken İnterpolasyonu #

Dinamik içerik veya komut dosyası oluşturma için sunucunuzdaki değişkenleri Pug’a geçirmek mümkündür. Pug şablonları, Express’te “res.render” işlevine iletilen değişkenlere erişebilir (veya Express kullanmıyorsanız “pug.renderFile”, bağımsız değişkenler aynıdır).

index.js

let colors = ["Red", "Green", "Blue"];
let langs  = ["HTML", "CSS", "JS"];
let title  = "My Cool Website";

let locals = {
    siteColors: colors,
    siteLangs:  langs,
    title:      title
};
res.render("index", locals);

‘index.pug’ dosyanızın içinde, anahtarları aracılığıyla ’locals’ değişkenine erişebilirsiniz. Pug dosyanızdaki değişkenlerin adları “siteColors” ve “siteNames” olur.

Bir HTML öğesinin tamamını bir değişkene eşitlemek için, bunu yapmak için eşittir operatörünü = kullanın. Değişkeninizin satır içine gömülmesi gerekiyorsa, bunu yapmak için köşeli ayraç sözdizimini “#{}” kullanın.

index.pug

doctype html
html
    head
        title= title
    body
        p My favorite color is #{siteColors[0]}.
        p Here's a list of my favorite coding languages
        ul
            each language in siteLangs
                li= language

index.pug çıktısı

<!DOCTYPE html>
<html>
    <head>
        <title>My Cool Website</title>
    </head>
    <body>
        <p>My favorite color is Red.</p>
        <p>Here's a list of my favorite coding languages</p>
        <ul>
            <li>HTML</li>
            <li>CSS</li>
            <li>JS</li>
        </ul>
    </body>
</html>

HTML’de Ham Değişken İnterpolasyonu #

Köşeli parantez sözdizimi ile enterpolasyon yapılan içerik, çıktısı HTML çıktınıza dahil edilen kod için değerlendirilecektir.

başlık, bir şablonu yerel olarak değerlendirmek için temel kalıbı takip eder, ancak #{ ve } arasındaki kod değerlendirilir, kaçar ve sonuç, oluşturulan şablonun çıktısında arabelleğe alınır. [Kaynak]

Ham HTML sözdizimi eklemeniz gerekiyorsa, pound sembolü yerine bir ünlem işareti kullanın (#{} yerine !{}).

index.js

let tag = "<div>You can't escape me!</div>";
res.render("index", {
    myTag: tag
});

index.pug

doctype html
html
    head
    body
        !{myTag}

index.pug çıktısı

<!DOCTYPE html>
<html>
    <head></head>
    <body>
        <div>You can't escape me!</div>
    </body>
</html>

JavaScript Kodunda Değer İnterpolasyonu #

İstemci tarafı JavaScript’e (veya bunu gerektiren diğer dillere) bir sunucu tarafı değişkeni iletmeniz gerekiyorsa, değerleri enterpolasyon yapmak yararlıdır.

Değişkenler, sayılar, dizeler ve benzerleri söz konusu olduğunda, bu tür değişkenleri, köşeli ayraç sözdizimi artı bir açıklama noktası ile doğrudan JavaScript’inize iletebilirsiniz (böylece köşeli ayraçlar içindeki kod değerlendirilmez_.) Bu, parametrelendirme için kullanışlıdır. Sunucunuzdan bir şey gerektiren JavaScript kodu.

Aşağıdaki örnekte, JavaScript’in onu bir dizge olarak yorumlaması için “kullanıcı adını” tırnak içine almamız gerekiyor; Pug, değişkenin içeriğini olduğu gibi çıkaracaktır, bu nedenle uygun bir JavaScript dizesi olması için onu tırnak işaretleri içine almamız gerekir. Bu, JavaScript’in numaramızı istediğimiz gibi (sayı olarak) yorumlayacağı “sayı” için gerekli değildir.

index.js

let number   = 24;
let username = "John";
res.render("index", {
    number:   number,
    username: username
});

index.pug

html
    head
    script.
        // Sets the username of the current user to be displayed site-wide
        function setUsername(username) {
            // ...
        }
        var number   = #{number};
        var username = "#{username}";
        setUsername(username);
        
    body
        p Welcome to my site!

index.pug çıktısı

<html>
    <head>
        <script>
            // Sets the username of the current user to be displayed site-wide
            function setUsername(username) {
                // ...
            }
            var number   = 24;
            var username = "John";
            setUsername(username);
        </script>
    </head>
    <body>
        <p>Welcome to my site!</p>
    </body>
</html>

Bir JavaScript nesnesinin değerini (örneğin, bir kullanıcı hakkındaki tüm bilgiler) enterpolasyon yapmanız gerekiyorsa, bunun bir JavaScript nesnesi olarak ele alınması için çıktıyı Pug’da dizmeniz gerekir. Değişkenin değerlendirilmiş formu yerine ham içeriğinin çıktısını almak da gereklidir. Çıkışlı değişkenin (var user = #{JSON.stringify(user)}) çıktısını yapacak olsaydınız, nesnenin escaped sürümünü alırsınız (tırnak işaretleri ve kesme işaretleri &quot; biçimine dönüştürülür), JSON.stringifyın üzerinde çalışması için istediğimiz şey bu değil.

index.js

var myUser = {
    name:    "Leeroy Jenkins",
    id:      1234567890,
    address: "123 Wilson Way, New York NY, 10165"
};

res.render('index', {
    user: myUser
});

index.pug

doctype html
html
    head
        script.
            window.onload = function () {
                function setUsername(username) {
                    return username;
                }

                var user = !{JSON.stringify(user)};
                document.getElementById("welcome-user").innerHTML = setUsername(user.name);
            };

    body
        div(id="welcome-user")

index.pug çıktısı

<!DOCTYPE html>
<html>
    <head>
        <script>
            window.onload = function() {
                function setUsername(username) {
                    return username;
                }

                var user = {
                    "name": "Leeroy Jenkins",
                    "id": 1234567890,
                    "address": "123 Wilson Way, New York NY, 10165"
                };
                document.getElementById("welcome-user").innerHTML = setUsername(user.name);
        };
        </script>
    </head>
    <body>
        <div id="welcome-user"></div>
    </body>    
</html>

“#welcome-user"ın “innerHTML"si “Leeroy Jenkins"e eşit olur. “user” değişkeninin içeriği doğrudan HTML kaynağına yazdırılır

HTML Öğesi İnterpolasyonu #

HTML etiketlerini iç içe yerleştirmek gerekebilir. Öğe enterpolasyonu, değişken enterpolasyonuna benzer bir sözdiziminde yapılır; burada küme parantezleri yerine köşeli parantezler kullanılmıştır. Enterpolasyonlu HTML öğelerinin sözdizimi, normal HTML öğelerinin uygulanmasıyla aynıdır.

index.pug

doctype html
html
    head
        title My Awesome Website
    body
        p The server room went #[b boom]!
        p The fire alarm, however, #[u failed to go off...]
        p Not even #[a(href="https://stackoverflow.com/") Stack Overflow] could save them now.

index.pug çıktısı

<!DOCTYPE html>
<html>
    <head>
        <title>My Awesome Website</title>
    </head>
    <body>
        <p>The server room went <b>boom</b>!</p>
        <p>The fire alarm, however, <u>failed to go off...</u></p>
        <p>Not even <a href="https://stackoverflow.com/">Stack Overflow</a> could save them now.</p>
    </body>
</html>
