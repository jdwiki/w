Kurulum ve Kurulum #

Eclipse’i yüklemek için, genellikle Eclipse’in en son sürümünü indirmek için doğrudan bir bağlantının bulunduğu Eclipse İndirmeleri Web sayfasına gidin. Eclipse Mars’ı (sürüm 4.5) başlatırken, tüm kurulumu sıkıştırılmış bir dosya olarak indirmek yerine kurulum prosedürü boyunca size rehberlik eden bir yükleyici indirilebilir (ancak bu seçenek hala mevcuttur). Eski Eclipse paketlerini indirmek için bağlantılar da vardır.

Eclipse, yükleyiciden aşağıdaki ekran görüntüsünde gösterildiği gibi farklı kullanıcıları hedefleyen birkaç farklı pakette gelir. Örneğin, Java Geliştiricileri için Eclipse IDE, Java uygulamaları geliştirmeyi, hata ayıklamayı ve oluşturmayı destekleyen temel araçların yanı sıra Git kullanarak projelerin sürümlenmesine izin veren bir eklenti gibi sürüm kontrolü için temel desteği içerirken, Eclipse Android Geliştiricileri için, Android uygulamaları oluşturmak için bir ortam sağlar.

Bir paket seçildiğinde, sonraki sayfa kullanıcının diğer seçeneklerle birlikte kurulum dizinini seçmesine izin verir. Aşağıdaki ekran görüntüsü, bir Windows makinesindeki prosedürü göstermektedir.

Yükle düğmesine tıklamak, Eclipse paketinin bu dizine kurulumunu başlatacaktır.

Makinede Java düzgün bir şekilde kuruluysa, Eclipse düzgün şekilde başlatılır ve genellikle hiçbir yapılandırma veya kurulum gerekmez. Ancak, örneğin hangi JVM Eclipse’in çalışacağını belirtmek ve bu JVM için minimum ve maksimum belleği yapılandırmak gibi Eclipse için bazı yapılandırma seçeneklerini değiştirmek iyi bir uygulamadır. Bunu yapmak için, bu başlatma yapılandırmasının bulunduğu kurulum dizininde ’eclipse.ini’ adlı bir dosya bulunur. Bu sayfa, Eclipse’in bu dosyayı kullanarak nasıl yapılandırılacağıyla ilgili ayrıntıları içerir.

Mevcut Projeleri İçe Aktarma #

Dosya menüsünde, ‘İçe Aktar…’ seçeneğini seçin.

Bu, içe aktarmak istediğiniz proje/dosya türünü soran İçe Aktar iletişim kutusunu açar. Temel bir Java projesi için, ‘Genel’ klasöründen ‘Çalışma Alanına Mevcut Projeler’i seçin.

Ardından, ‘Gözat’ düğmesini kullanarak projelerin bulunduğu dizini seçin. Eclipse’e aktarılabilen tüm projeler ‘Projeler:’ bölümünde görünecektir. Proje zaten içe aktarılmışsa, görüntülenmeye devam edecek ancak onay kutusu devre dışı bırakılacaktır.

Ayrıca, ‘Arşiv dosyası seç’i seçip ardından ‘Gözat’ düğmesini tıklatarak projeleri doğrudan sıkıştırılmış bir dosyadan içe aktarabilirsiniz.

‘Bitir’i tıkladığınızda, proje artık Proje Gezgini’nde görünür ve kullanıma hazırdır.

Eclipse’de Marketplace’i kurun #

Eclipse classic sürümlerinden birkaçı, marketplace ile önceden yüklenmiş olarak gelmez, bu, aşağıdaki adımlar kullanılarak yüklenebilir:

  1. Yardıma Git → Yeni Yazılım Yükle
  2. Yeni Depo ekleyin (aşağıda belirtilen site)
  3. Genel Amaçlı Araçlar → Pazar Yeri Müşterisi
  4. Bitir’e tıklayın ve bitirdiniz.

Pazar yeri güncelleme siteleri:

Oxygen - http://download.eclipse.org/releases/oxygen/
Neon - http://download.eclipse.org/releases/neon/
Mars - http://download.eclipse.org/releases/mars/
Luna - http://download.eclipse.org/mpc/luna
Helios - http://download.eclipse.org/releases/helios
Juno - http://download.eclipse.org/releases/juno/

Kullanışlı Klavye Kısayolları #

Dosyaları ve Projeleri Yönetin #

  • Ctrl+Shift+R : Açık Kaynak (dosya, klasör veya proje)
  • Ctrl+Shift+S : Tüm dosyaları kaydet
  • Ctrl+W : Geçerli dosyayı kapat
  • Ctrl+Shift+W : Tüm dosyaları kapat

Düzenleyici Penceresi #

  • F12 : Düzenleyici Penceresine Atla
  • Ctrl+E : Açık Düzenleyicilerin listesini göster. Ok tuşlarını kullanın ve geçiş yapmak için girin
  • Ctrl+Page Down/Up : Sonraki düzenleyiciye geç / önceki düzenleyiciye geç
  • Ctrl+M : Mevcut Düzenleyici Penceresini büyüt veya küçült

Editör’de gezinin #

  • Ctrl+L : Satıra git
  • Ctrl+Q : Düzenlenen son konuma atla
  • Ctrl+Shift+P : Bir parantez seçiliyken: eşleşen kapanış veya açılış dirseğine atlayın
  • Ctrl+Aşağı/Yukarı Ok : İmleç konumunu değiştirmeden Düzenleyiciyi Kaydır

Metni düzenle #

  • Ctrl+D : Satırı Sil
  • Alt+Shift+Y : Seçili metni kaydır (metnin genişliğini ekrana sığdır)
  • Alt+Shift+S : Açık Kaynak menü seçenekleri
  • Alt+Shift+R : Vurgulanan kelimeyi tüm dosyalarda yeniden düzenleyin
  • Ctrl+Alt+Yukarı/Ctrl+Alt+Aşağı : Seçili satırları yukarı/aşağı kopyalayın.

Ara ve Değiştir #

  • Ctrl+J : Aramak için bir terim yazın, ardından yukarı/geri gitmek için Ctrl+J / Ctrl+shift+J tuşlarını kullanın
  • Ctrl+K/Ctrl+Shift+K : Ctrl+H ardından bulma penceresini kapatın. Ardından, arama teriminin önceki / sonraki oluşumunu bulun.

Bir kod bloğunu taşı #

  • ÜstKrktr+sekme : Sola git
  • Alt+Yukarı/Alt+Aşağı : Yukarı/aşağı hareket eder.

Java HelloWorld Programı Oluşturma ve Çalıştırma #

Araç çubuğundan Java Perspektifini açın.

Yeni bir Java projesi oluşturun #

Paket Gezgini’ne sağ tıklayın, ve menüden Yeni -> Java Projesini seçin

Yaklaşan iletişim kutusunda bir proje adı girin ve ardından ‘Bitir’i tıklayın.

Artık çalışma alanınızda yeni projeniz var.

Yeni bir Java sınıfı oluşturun #

Projenize sağ tıklayın ve menüden Yeni -> Sınıfı seçin.

İletişim kutusunda sınıfın adını yazın (büyük harfle başlamalıdır), ayrıca public static void main(String[] args) onay kutusunu seçin, ardından ‘Bitir’i tıklayın.

Artık projenizdeki ilk Java dosyasına sahipsiniz. Düzenleyici bu yeni dosyayı otomatik olarak açacaktır.

‘Ana’ yöntem içinde, ‘Merhaba dünya!’ yazdırmak için bazı kodlar yazın.

Java sınıfınızı çalıştırın #

Java sınıfınıza sağ tıklayın ve menüden Farklı çalıştır -> Java uygulaması seçeneğini seçin.

Voila, Java programınızın çıktısını Konsolda görüyorsunuz.
