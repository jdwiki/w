Sözdizimi #

  • &<variable> - referansa göre erişim (=işaretçiyi değişkenin verilerine götürür)
  • *<değişken> - sapma operatörü (=bir işaretçiden veri nesnesini alır)
  • <type>* - <type>‘a işaret eden veri türü (ör. `int*)

İşaretçiler #

D bir sistem programlama dilidir ve bu nedenle hafızanızı manuel olarak yönetmenize ve karıştırmanıza izin verir. Bununla birlikte, D, kullanılmayan belleği boşaltmak için varsayılan olarak bir çöp toplayıcı kullanır.

D, C’deki gibi T* işaretçi türlerini sağlar:

void main()
{
    int a;
    int* b = &a; // b contains address of a
    auto c = &a; // c is int* and contains address of a

    import std.stdio : writeln;
    writeln("a ", a);
    writeln("b ", b);
    writeln("c ", c);
}

Öbek üzerinde tahsis #

Yığındaki yeni bir bellek bloğu, yönetilen belleğe bir işaretçi döndüren “new” ifadesi kullanılarak ayrılır:

void main()
{
    int* a = new int;
    *a = 42; // dereferencing
    import std.stdio : writeln;
    writeln("a: ", *a);
}

@güvenli D #

a tarafından başvurulan belleğe artık programdaki herhangi bir değişken aracılığıyla başvurulmadığı anda, çöp toplayıcı belleğini boşaltacaktır.

D ayrıca, “@safe” olarak işaretlenmiş kod dışında işaretçi aritmetiğine de izin verir.

void safeFun() @safe
{
    writeln("Hello World");
    // allocating memory with the GC is safe too
    int* p = new int;
}

void unsafeFun()
{
    int* p = new int;
    int* fiddling = p + 5;
}

void main()
{
    safeFun();
    unsafeFun();
}

SafeD hakkında daha fazla bilgi için D tasarım ekibinin makalesine bakın.
