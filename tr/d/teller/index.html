Bir dizeyi tersine çevirme #

‘string’, ‘alias string = immutable(char)[];’ olarak tanımlanır: bu nedenle, tersine çevrilmeden önce değiştirilebilir bir karakter dizisi oluşturmak için ‘dup’ kullanmanız gerekir:

import std.stdio;
import std.string;

int main() {

    string x = "Hello world!";
    char[] x_rev = x.dup.reverse;
    
    writeln(x_rev); // !dlrow olleH
    
    return 0;

}

Boş veya boş bir dize testi yapın #

Boş dize #

Boş dize boş değil ama uzunluğu sıfır:

string emptyString = "";
// an empty string is not null...
assert(emptyString !is null);

// ... but it has zero lenght
assert(emptyString.length == 0);

Boş dize #

string nullString = null;

boş bir dize boştur (De Lapalisse)

assert(nullString is null);

ancak, C#’tan farklı olarak, boş bir dizenin uzunluğunu okuyun, hata oluşturmaz:

assert(nullString.length == 0);
assert(nullString.empty);

Dizeyi ubyte[]‘ye dönüştürün ve tam tersi #

Değişmez ‘ubyte[]’ için dize #

string s = "unogatto";
immutable(ubyte[]) ustr = cast(immutable(ubyte)[])s;

assert(typeof(ustr).stringof == "immutable(ubyte[])");
assert(ustr.length == 8);
assert(ustr[0] == 0x75); //u
assert(ustr[1] == 0x6e); //n
assert(ustr[2] == 0x6f); //o
assert(ustr[3] == 0x67); //g
assert(ustr[7] == 0x6f); //o

‘ubyte[]’ için dize #

string s = "unogatto";
ubyte[] mustr = cast(ubyte[])s;

assert(typeof(mustr).stringof == "ubyte[]");

assert(mustr.length == 8);
assert(mustr[0] == 0x75);
assert(mustr[1] == 0x6e);
assert(mustr[2] == 0x6f);
assert(mustr[3] == 0x67);
assert(mustr[7] == 0x6f);

‘ubyte[]’ dizgeye #

ubyte[] stream = [ 0x75, 0x6e, 0x6f, 0x67];
string us  = cast(string)stream;
assert(us == "unog");
