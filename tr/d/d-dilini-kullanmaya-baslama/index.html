Kurulum veya Kurulum #

D programlama dilinin standart derleyici DMD’si tüm büyük platformlarda çalışabilir. DMD’yi yüklemek için buraya bakın. Komut satırı ile kurmak için şu komutu çalıştırabilirsiniz (D web sitesinde bulunur):

curl -fsS https://dlang.org/install.sh | bash -s dmd

Paket Yöneticileri #

Arch Linux #

pacman -S dlang

çikolatalı #

choco install dmd

Gentoo #

layman -f -a dlang

OSX Homebrew #

brew install dmd

Debian/Ubuntu #

Debian/Ubuntu dağıtımlarına kurulum için APT deposunun kaynaklar listesine eklenmesi gerekir.

wget http://master.dl.sourceforge.net/project/d-apt/files/d-apt.list -O /etc/apt/sources.list.d/d-apt.list
wget -qO - https://dlang.org/d-keyring.gpg | sudo apt-key add -
apt-get update
apt-get install dmd-bin

Diğer derleyiciler #

LDC, resmi DMD derleyici ön ucunu ve arka ucu olarak LLVM’yi kullanan bir D derleyicisidir.

GDC kod oluşturmak için GCC arka ucunu kullanan bir D derleyicisidir.

IDE #

Hayatı kolaylaştırmak için bir IDE (Entegre Geliştirme Ortamı) da kurmak isteyebilirsiniz. D-Language Wiki’de tüm Platformlar için mevcut IDE’lerin ve Eklentilerin bir listesi vardır burada.

Selam Dünya #

import std.stdio;

// Let's get going!
void main()
{
    writeln("Hello World!");
}

Derlemek ve çalıştırmak için bu metni main.d adlı bir dosya olarak kaydedin. Programı derlemek için komut satırından dmd main.d komutunu çalıştırın. Son olarak, programı bir bash kabuğunda çalıştırmak için ./main komutunu çalıştırın veya Windows’ta yürütülebilir dosyaya tıklayabilirsiniz.

Bir dizeden değerleri oku #

import std.format;

void main() {
    string s = "Name Surname 18";
    string name, surname;
    int age;
    formattedRead(s, "%s %s %s", &name, &surname, &age);
    // %s selects a format based on the corresponding argument's type
}

Biçim dizeleri için resmi belgeler şu adreste bulunabilir: https://dlang.org/phobos/std_format.html#std.format

Selam Dünya! #

Klasik “Merhaba, dünya” yazdırma programını oluşturmak için, aşağıdaki kodu içeren bir metin düzenleyiciyle “merhaba.d” adlı bir dosya oluşturun:

import std.stdio;

void main() {
    writeln("Hello, World!");    //writeln() automatically adds a newline (\n) to the output
}

Açıklama : #

import std.stdio

Bu satır derleyiciye Standart Kitaplık modülü std.stdio içinde tanımlanan fonksiyonların kullanılacağını söyler. Derleyici onları nerede arayacağını bildiği sürece herhangi bir modül alınabilir. D’nin devasa Standart Kitaplığının bir parçası olarak birçok işlev sağlanır.

void main() {

Bu satır, ‘ana’ işlevini bildirir ve ‘void’ değerini döndürür. C ve C++‘dan farklı olarak D’nin main’in “void” türünde olmasına izin verdiğini unutmayın. ‘Main’ işlevi programın giriş noktası olduğu için özeldir, yani programın yürütülmesinin başladığı yer burasıdır. Genel olarak işlevler hakkında birkaç not:

  • Bir işlevin adı, bir harfle başlayan ve harflerden, rakamlardan ve alt çizgilerden oluşan herhangi bir şey olabilir.

  • Beklenen parametreler, değişken adlarının ve veri türlerinin virgülle ayrılmış bir listesi olacaktır.

  • İşlevin döndürmesi beklenen değer, mevcut herhangi bir veri türü olabilir ve işlev içindeki dönüş ifadesinde kullanılan ifade türüyle eşleşmelidir.

{ … } küme parantezleri, bir kod bloğunun nerede başlayıp nerede bittiğini belirtmek için çiftler halinde kullanılır. Birçok şekilde kullanılabilirler, ancak bu durumda işlevin nerede başlayıp nerede bittiğini gösterirler.

writeln("Hello, World!");

“writeln”, agrumentlerini “stdout"a yazan, “std.stdio” içinde bildirilen bir işlevdir. Bu durumda, argümanı konsola yazılacak olan “Merhaba, Dünya”‘dır. C’nin ‘printf’ tarafından kullanılanlara benzer çeşitli biçim karakterleri, ‘\n’, ‘\r’, vb. kullanılabilir.

Her ifadenin noktalı virgülle sonlandırılması gerekir.

Yorumlar, kodu okuyan kişiye bir şey belirtmek için kullanılır ve derleyici tarafından boş olarak değerlendirilir. Yukarıdaki kodda bu bir yorumdur:

//writeln() automatically adds a newline (\n) to the output

Bunlar derleyici tarafından yok sayılan kod parçalarıdır. D’de yorum yapmanın üç farklı yolu vardır:

  1. // - Tüm metni // işaretinden sonra aynı satırda yorumlayın
  2. /* yorum metni */ - Bunlar çok satırlı yorumlar için kullanışlıdır
  3. /+ yorum metni + - Bunlar aynı zamanda çok satırlı yorumlardır

Bir işlevin/kod parçasının ne yaptığını diğer bir geliştiriciye iletmek için çok kullanışlıdırlar.

Programın Derlenmesi ve Çalıştırılması #

Bu programı çalıştırmak için kodun bir yürütülebilir dosyaya derlenmesi gerekir. Bu derleyici yardımı ile yapılabilir.

Referans D derleyicisi olan DMD’yi kullanarak derlemek için bir terminal açın, oluşturduğunuz hello.d dosyasının konumuna gidin ve ardından çalıştırın:

dmd merhaba.d

Herhangi bir hata bulunmazsa, derleyici kaynak dosyanızdan sonra adlandırılmış bir yürütülebilir dosya çıkaracaktır. Bu şimdi yazarak çalıştırılabilir

./merhaba

Yürütüldüğünde, program Merhaba Dünya! yazdıracak ve ardından yeni bir satır yazacaktır.
