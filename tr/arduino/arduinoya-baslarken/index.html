Sadece Minimum #

İşte ‘çıplak minimum’ Arduino taslağı. Bu, File > Samples > 01. Basics > Bare Minimum seçilerek Arduino IDE’ye yüklenebilir.

void setup() {
  // put your setup code here, to run once
}

void loop() {
  // put your main code here, to run repeatedly
}

setup() işlevindeki kod, program başladığında bir kez çalıştırılacaktır. Bu, G/Ç pinlerini ayarlamak, değişkenleri başlatmak vb. için kullanışlıdır. loop() fonksiyonundaki kod, Arduino kapatılana veya yeni bir program yüklenene kadar tekrar tekrar çalışacaktır. Etkili bir şekilde, yukarıdaki kod Arduino çalışma zamanı kitaplığında şöyle görünür:

setup();
while(1) {
  loop();
}

Bilgisayarınızda çalışan programların aksine Arduino kodu asla çıkamaz. Bunun nedeni, mikrodenetleyicinin içinde yalnızca bir program yüklü olmasıdır. Bu program kapanırsa, mikrodenetleyiciye ne yapacağını söyleyecek hiçbir şey kalmaz.

İlk Kurulum #

Gerekli yazılım: Arduino IDE

Kurmak #

Arduino uyumlu kartların çoğunda bir USB bağlantı noktası bulunur ve bir USB kablosuyla birlikte gelir. USB kablosunu kullanarak Arduino’yu takın ve Arduino IDE’yi başlatın.

Arduino IDE, tipik olarak boş bir ‘setup()’ ve ’loop()’ işlevleriyle yeni bir çizim ile başlayacaktır. Bu, bir Arduino kartına yüklemek için yeterlidir, ancak hiçbir şey yapmaz. “Blink” örnek çizimi, bir Arduino kartını ilk kez kullanırken basit bir test olarak çalışır. Dosyaya Git → Örnekler → 01.Temel Bilgiler → Göz kırpmak. Bu, Blink taslağı ile yeni bir pencere açacaktır.

Tahtanızı seçin. Araçlar’a gidin → Yönetim Kurulu → [Arduino panonuzun adı].

Anakartınız için COM bağlantı noktasını seçin. Aurduino uyumlu kartların çoğu, seri iletişim (hata ayıklama) ve kartı programlamak için kullanılan sahte bir COM bağlantı noktası oluşturacaktır. COM 1 genellikle zaten mevcuttur ve panonuz yeni bir tane oluşturacaktır, ör. COM 4. Bunu Araçlar’dan seçin → Bağlantı noktası → COM 4 (veya diğer COM numarası).

Bazı panolarda Araçlar menüsünde saat hızı gibi ek ayarlar bulunur. Bunlar panodan panoya değişir, ancak genellikle kabul edilebilir bir varsayılanlar kümesi zaten seçilmiştir.

Yüklemek #

Artık Blink’i yüklemeye hazırsınız. Yükle düğmesini tıklayın veya Çizim → Yüklemek. Eskiz derlenecek ve ardından Arduino kartınıza yüklenecektir. Her şey yolunda giderse, yerleşik LED her saniye yanıp sönmeye başlayacaktır.

Seri monitör #

Arduino IDE ypu’da seri bir monitör var. Açmak için pencerenin sağ tarafındaki seri monitör düğmesini kullanın.

Monitörü açmadan önce kodun yüklendiğinden emin olun. Yükleme ve izleme aynı anda çalışmayacaktır!

Göz kırpmak #

İşte setup() ve loop() fonksiyonlarını gösteren kısa bir örnek. Bu, `Dosya > Örnekler > 01. Temel Bilgiler > Yanıp Sönme’ seçilerek Arduino IDE’ye yüklenebilir. (Not: Çoğu Arduino kartında zaten pin 13’e bağlı bir LED bulunur, ancak bu çizimin etkilerini görmek için harici bir LED eklemeniz gerekebilir.)

// the setup function runs once when you press reset or power the board
void setup() {
  // initialize digital pin 13 as an output.
  pinMode(13, OUTPUT);
}

// the loop function runs over and over again forever
void loop() {
  digitalWrite(13, HIGH);   // turn the LED on (HIGH is the voltage level)
  delay(1000);              // wait for a second
  digitalWrite(13, LOW);    // turn the LED off by making the voltage LOW
  delay(1000);              // wait for a second
}

Yukarıdaki snippet:

  1. setup() fonksiyonunu tanımlar. Her Arduino programında çalıştırıldığında ilk olarak setup() işlevi çağrılır.

    1. Sets pin 13 as an output.

      Without this, it might be set to an input, which would make the LED not work; however once it is set as an output it will stay that way so this only needs to be done once when the program starts.

  2. loop() fonksiyonunu tanımlar. Program çalıştığı sürece loop() fonksiyonu tekrar tekrar çağrılır.

    1. digitalWrite(13, HIGH); turns the LED on.
    2. delay(1000); waits one second (1000 milliseconds).
    3. digitalWrite(13, LOW); turns the LED off.
    4. delay(1000); waits one second (1000 milliseconds).

Program çalıştığı sürece loop() tekrar tekrar çalıştığından, LED 2 saniye (1 saniye açık, 1 saniye kapalı) aralıklarla yanıp sönecektir. Bu örnek Arduino Uno’ya ve Pin 13’e bağlı bir LED’i olan diğer herhangi bir karta dayanmaktadır. Kullanılan kartta bu pine bağlı yerleşik bir LED yoksa, harici olarak takılabilir.

Zamanlama hakkında daha fazla bilgi (örneğin gecikmeler ve ölçüm süresi): https://www.wikiod.com/tr/arduino/zaman-yonetimi

LED - Düğme kontrollü #

Bu kodu, bir çekme direncine sahip bir düğme anahtarlı bir LED’i ayarlamak için de kullanabilirsiniz, bu, tercihen ilk LED denetleyiciyi kurduktan sonraki adımda olabilir.

int buttonState = 0; // variable for reading the pushbutton status

void setup()
{
    // initialize the LED pin as an output:
    pinMode(13, OUTPUT); // You can set it just using its number
    // initialize the pushbutton pin as an input:
    pinMode(2, INPUT);
}

void loop()
{
    // read the state of the pushbutton value:
    buttonState = DigitalRead(2);

    // check if the pushbutton is pressed.
    // If it's not, the buttonState is HIGH : if (buttonState == HIGH)
    {
        // turn LED off:
        digitalWrite(13, LOW);
    }
    else
    {
        // turn LED off:
        digitalWrite(13, HIGH);
    }
}
