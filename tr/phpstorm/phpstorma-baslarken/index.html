Kurulum #

Giriiş #

JetBrains PhpStorm, PHP için ticari, platformlar arası bir IDE’dir. JetBrains’in Java ile yazılmış IntelliJ IDEA platformu üzerine inşa edilmiştir. Böylece Java’yı destekleyen tüm büyük işletim sistemlerinde çalışacaktır.

Kullanıcılar, IntelliJ Platformu için oluşturulan eklentileri yükleyerek IDE’yi genişletebilir.

Gereksinimler #

Donanım #

  • Minimum 1 GB RAM, 2 GB RAM önerilir
  • 1024x768 minimum ekran çözünürlüğü

Yazılım #

Birlikte verilen Oracle Java Runtime Environment’ı kullanmanız önerilir. OpenJDK beklenmeyen davranışlara yol açabilir ve resmi olarak desteklenmez.

PhpStorm’u Yükleme #

PhpStorm’u kullanmaya başlamak için, istediğiniz işletim sisteminin sürümünü PhpStorm indirme sayfasından indirin. Bir sonraki adım işletim sistemine bağlıdır.

İşletim Sistemine Özel talimatlar #

Linux #

  • İndirilen .tar.gz dosyasını paketinden çıkarın.
  • “bin” alt dizininden “PhpStorm.sh” dosyasını çalıştırın

OSX #

  • PhpStorm-2016.2.dmg Mac OS X Disk Image dosyasını indirin
  • Sisteminize başka bir disk olarak monte edin
  • PhpStorm’u Uygulamalar klasörünüze kopyalayın

Pencereler #

  • Yükleme Sihirbazını başlatan indirilen .exe dosyasını çalıştırın.
  • Sihirbaz tarafından önerilen tüm adımları izleyin.
  • İlgili kurulum seçeneklerine özellikle dikkat edin.

Lisans #

PhpStorm, 30 günlük ücretsiz deneme için paket değerlendirme lisans anahtarı içerir. Bu süreden sonra IDE’yi kullanabilmek için PhpStorm için bir lisans [Jetbrains’ten 2 edinilmiş] olması gerekir.

Çeşitli lisanslar ücretsiz veya indirimli olarak sunulmaktadır:

  • Eğitim ve öğretim / öğrenci ve öğretmen indirimi: ücretsiz
  • Açık Kaynak projeleri indirimi: ücretsiz
  • Öğrenci mezuniyet indirimi %25 indirim
  • Startup indirimi: %50 indirim
