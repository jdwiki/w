  • Yardım metni, fonksiyon satırı ile yardım metninin başlangıcı arasında kod olmadığı sürece ‘fonksiyon’ satırından önce veya sonra yer alabilir.
  • İşlev adının büyük harfle yazılması yalnızca adı kalınlaştırır ve gerekli değildir.
  • Bir satırın başına ‘Ayrıca bakınız’ eklenirse, satırdaki arama yolundaki bir sınıf veya işlevin adıyla eşleşen tüm adlar otomatik olarak o sınıfın/fonksiyonun belgelerine bağlanır.
  • Global işlevler, adlarının önüne bir `' koyarak burada atıfta bulunulabilir. Aksi takdirde, adlar önce üye işlevleri deneyecek ve çözecektir.
  • <a href="matlab:web('url')">Ad</a> biçimindeki köprülere izin verilir.

Basit İşlev Belgeleri #

function output = mymult(a, b)
% MYMULT Multiply two numbers.
%   output = MYMULT(a, b) multiplies a and b.
%
%   See also fft, foo, sin.
%
%   For more information, see <a href="matlab:web('https://google.com')">Google</a>.
    output = a * b;
end

help mymult daha sonra şunları sağlar:

mymult İki sayıyı çarpın.

çıktı = mymult(a, b) a ve b’yi çarpar.

Ayrıca bkz. fft, foo, sin.

Daha fazla bilgi için Google adresine bakın.

“fft” ve “sin” otomatik olarak ilgili yardım metinlerine bağlanır ve Google, google.com bağlantısıdır. “foo”, arama yolunda “foo” adında belgelenmiş bir işlev/sınıf olmadığı sürece, bu durumda herhangi bir belgeye bağlanmaz.

Bir fonksiyon imzası alma #

MATLAB’ın bir işlevin 1st satırını yazdırması genellikle yararlıdır, çünkü bu genellikle girişler ve çıkışlar dahil olmak üzere işlev imzasını içerir:

dbtype <functionName> 1

Örnek:

>> dbtype fit 1

1  function [fitobj,goodness,output,warnstr,errstr,convmsg] = fit(xdatain,ydatain,fittypeobj,varargin)

Yerel İşlev Belgeleri #

Bu örnekte, “baz” yerel işlevine ilişkin belgelere (“foo.m” ile tanımlanır) ya “help foo” içindeki sonuç bağlantısıyla ya da doğrudan “help foo>baz” aracılığıyla erişilebilir.

function bar = foo
%This is documentation for FOO.
% See also foo>baz

% This wont be printed, because there is a line without % on it.
end

function baz
% This is documentation for BAZ.
end

Bir Fonksiyonu Örnek Komut Dosyası ile Belgeleme #

Bir işlevi belgelemek için, işlevinizi kullanan örnek bir komut dosyasına sahip olmak genellikle yararlıdır. Matlab’daki yayınlama işlevi daha sonra gömülü resimler, kod, bağlantılar vb. içeren bir yardım dosyası oluşturmak için kullanılabilir. Kodunuzu belgelemek için sözdizimi burada bulunabilir.

İşlev Bu işlev Matlab’da “düzeltilmiş” bir FFT kullanır.

function out_sig = myfft(in_sig)

out_sig = fftshift(fft(ifftshift(in_sig)));

end

Örnek Komut Dosyası Bu, girdileri, çıktıları açıklayan ve düzeltmenin neden gerekli olduğunu açıklayan bir örnek veren ayrı bir komut dosyasıdır. Bu işlevin asıl yazarı Wu, Kan’a teşekkürler.

%% myfft
% This function uses the "proper" fft in matlab. Note that the fft needs to
% be multiplied by dt to have physical significance.
% For a full description of why the FFT should be taken like this, refer
% to: Why_use_fftshift(fft(fftshift(x)))__in_Matlab.pdf included in the
% help folder of this code. Additional information can be found:
% <https://www.mathworks.com/matlabcentral/fileexchange/25473-why-use-fftshift-fft-fftshift-x----in-matlab-instead-of-fft-x-->
%
%% Inputs
% *in_sig* - 1D signal
% 
%% Outputs
% *out_sig* - corrected FFT of *in_sig*
% 
%% Examples
% Generate a signal with an analytical solution. The analytical solution is
% then compared to the fft then to myfft. This example is a modified
% version given by Wu, Kan given in the link aboce.
%%
% Set parameters
fs = 500;           %sampling frequency
dt = 1/fs;          %time step
T=1;                %total time window
t = -T/2:dt:T/2-dt; %time grids
df = 1/T;           %freq step
Fmax = 1/2/dt;      %freq window
f=-Fmax:df:Fmax-df; %freq grids, not used in our examples, could be used by plot(f, X)
%%
% Generate Gaussian curve
Bx = 10; A = sqrt(log(2))/(2*pi*Bx);    %Characteristics of Gaussian curve
x = exp(-t.^2/2/A^2);                   %Create Gaussian Curve
%% 
% Generate Analytical solution
Xan = A*sqrt(2*pi)*exp(-2*pi^2*f.^2*A^2); %X(f), real part of the analytical Fourier transform of x(t)

%%
% Take FFT and corrected FFT then compare
Xfft = dt *fftshift(fft(x));    %FFT
Xfinal = dt * myfft(x);         %Corrected FFT
hold on
plot(f,Xan);
plot(f,real(Xfft));
plot(f,real(Xfinal),'ro');
title('Comparison of Corrected and Uncorrected FFT');
legend('Analytical Solution','Uncorrected FFT','Corrected FFT');
xlabel('Frequency'); ylabel('Amplitude');
DT = max(f) - min(f);
xlim([-DT/4,DT/4]);
