Facebook sosyal grafiğini okumak ve yazmak için FaceBook Erişim Simgesini alma #

Bu belge, facebook erişim belirteçlerini elde etme adımlarını ve belirteçleri FB beslemelerini getirmek için kullanma adımlarını detaylandırmaktadır.

Örnek: Canlı bir örnek mevcuttur

https://newtonjoshua.com

Graph API’ye Giriş: Graph API, Facebook platformuna veri almanın ve bu platformdan çıkmanın birincil yoludur. Verileri sorgulamak, yeni hikayeler göndermek, reklamları yönetmek, fotoğraf yüklemek ve bir uygulamanın yapması gerekebilecek çeşitli diğer görevleri yapmak için kullanabileceğiniz düşük seviyeli HTTP tabanlı bir API’dir.

FaceBook Uygulamaları:

https://developers.facebook.com

Bir Facebook uygulaması oluşturun. Bir App_Id ve App_Secret alacaksınız

Grafik API Gezgini:

https://developers.facebook.com/tools/explorer/{{App_Id}}/?method=GET&path=me%2Ffeed&version=v2.8

Kısa ömürlü bir ‘access_token’ alacaksınız. Yani bu bizim short_lived_access_tokenımız olacak.

not: erişim belirteci oluştururken, ihtiyacınız olan tüm fb alanlarını seçin. Bu, erişim belirtecinin bu alanları getirmesine izin verecektir.

Erişim Simgesi Uzantısı:

https://graph.facebook.com/oauth/access_token?grant_type=fb_exchange_token&client_id={{App_Id}}&client_secret={{App_Secret}}&fb_exchange_token={{short-lived-access_token }}

2 ay geçerliliği olan bir ‘access_token’ alacaksınız.

Erişim Simgesi Hata Ayıklayıcı:

https://developers.facebook.com/tools/debug/accesstoken?q={{access_token}}&version=v2.8

‘access_token’ ayrıntılarını kontrol edebilirsiniz.

JavaScript için Facebook SDK’sı: SDK’yı eşzamansız olarak sayfanıza yüklemek için aşağıdaki JavaScript’i HTML’nize ekleyin

<script>
        (function (d, s, id) {
            var js, fjs = d.getElementsByTagName(s)[0];
            if (d.getElementById(id)) {
                return;
            }
            js = d.createElement(s);
            js.id = id;
            js.src = "//connect.facebook.net/en_US/sdk.js";
            fjs.parentNode.insertBefore(js, fjs);
        }(document, 'script', 'facebook-jssdk'));
    </script>

Grafik API’si: FB kimliğimizi, profil resmimizi, kapak resmimizi ve yayınlarımızı almak için bir API araması yapalım.

<script>
window.fbAsyncInit = function () {
    FB.init({
        appId: '{{App_Id }}',
        xfbml: true,
        version: 'v2.7'
    });
    FB.api(
        '/me',
        'GET', {
            fields: 'id,picture{url},cover,feed',
            access_token: {{access_token}}
        },
        function (response) {
        if (response.error) {
                console.error(response.error.message);
            }
            if (response.picture.data.url) {
                profilePic = response.picture.data.url;
            }
            if (response.cover.source) {
                coverPic = response.cover.source;
            }
            if (response.feed.data) {
        feeds = response.feed.data;
                feeds.forEach(function (feed) {
            // view each feed content
                });
            }
            if (response.feed.paging.next) {
                nextFeedPage = response.feed.paging.next;
        // a request to nextFeedPage will give the next set of feeds
            }
            
        }
    );
};

</script>

‘alanlara’ girilmesi gereken sorgunuzu tasarlamak için Grafik API Gezgini’ni kullanın (ör. ‘id,resim{url},kapak,feed’)

Artık access_token’inizi kullanarak facebook verilerinizi Facebook Graph API’sinden alabilirsiniz. https://developers.facebook.com/docs/graph-api/overview/ adresine bakın

not: ‘access_token’ınızın süresi 2 ay içinde dolacak. Bundan sonra yeni bir access_token oluşturun.
