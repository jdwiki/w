DB Open'da Shift Tuşunu EtkinleştirDevre Dışı Bırak

  • Shift İşlev Kodunu Devre Dışı Bırak
  • Shift İşlev Kodunu Etkinleştir

Bu sayfada

  • Shift İşlev Kodunu Devre Dışı Bırak
  • Shift İşlev Kodunu Etkinleştir

Bu kod, kullanıcının bir veritabanını açarken varsayılan form açılışını atlamak için Shift tuşunu basılı tutma özelliğini kapatacak ve kullanıcının Gezinti Bölmesi ve VB Düzenleyicisine erişmesine izin verecektir. Kullanıcıların bunlardan herhangi birine erişmesini istemediğiniz DB’lerde (Mevcut Veritabanı Seçeneklerinde Özel Anahtarların kullanımını devre dışı bırakmakla birlikte), bu kod veritabanının kilitli kalmasına yardımcı olacaktır.

Genel olarak, aşağıdaki kod basEnableDisableShift modülü olarak adlandırılabilen kendi Modülüne yerleştirilir, ancak bu sadece bir öneridir ve zaten sahip olabileceğiniz herhangi bir modüle yerleştirebilirsiniz.

Shift tuşunu devre dışı bırakmak için, VB Editor ekranınızda, Anında pencerenize ‘DisableShift’ yazın ve Enter’a basın. Daha sonra Shift Tuşunun Devre Dışı Bırakıldığını bildiren bir mesaj alacaksınız.

Shift tuşunu yeniden etkinleştirmek için, bir kez daha VB Editör ekranına dönmeniz, Anında pencerenize ‘EnableShift’ yazmanız ve Enter’a basmanız gerekecektir. Hemen pencerenizde Shift Tuşunun Etkinleştirildiğini bildiren bir mesaj alacaksınız.

NOT: Bu, Shift Key’i etkinleştirmenin ve devre dışı bırakmanın kusursuz bir yolu değildir, ancak MS Access ve VBA konusunda yetkin olmayan kullanıcılara bir veritabanı dağıtıyorsanız, kullanıcıların veritabanınızdaki VB Düzenleyicisine ve/veya Gezinti Bölmesine erişim.

Shift İşlev Kodunu Devre Dışı Bırak #

Function DisableShift()
'This function disable the shift at startup. This action causes
'the Autoexec macro and Startup properties to always be executed.

On Error GoTo errDisableShift

    Dim db As DAO.Database
    Dim prop As DAO.Property
    Const conPropNotFound = 3270

    Set db = CurrentDb()

    'This next line disables the shift key on startup.
    db.Properties("AllowByPassKey") = False

    'The function is successful.
    Debug.Print "Disabled Shift Key - Successful"
    Exit Function

errDisableShift:
    'The first part of this error routine creates the "AllowByPassKey
    'property if it does not exist.
    If Err = conPropNotFound Then
        Set prop = db.CreateProperty("AllowByPassKey", _
        dbBoolean, False)
        db.Properties.Append prop
        Resume Next
        Else
            MsgBox "Function 'ap_DisableShift' did not complete successfully."
            GoTo ExitHere
    End If

ExitHere:
    Set prop = Nothing
    Set db = Nothing
    Exit Function

End Function

Shift İşlev Kodunu Etkinleştir #

Function EnableShift()
'This function enables the SHIFT key at startup. This action causes
'the Autoexec macro and the Startup properties to be bypassed
'if the user holds down the SHIFT key when the user opens the database.

On Error GoTo errEnableShift

    Dim db As DAO.Database
    Dim prop As DAO.Property
    Const conPropNotFound = 3270

    Set db = CurrentDb()

    'This next line of code disables the SHIFT key on startup.
    db.Properties("AllowByPassKey") = True

    'function successful
    Debug.Print "Enabled Shift Key - Successful"
    GoTo ExitHere

errEnableShift:
    'The first part of this error routine creates the "AllowByPassKey
    'property if it does not exist.
    If Err = conPropNotFound Then
        Set prop = db.CreateProperty("AllowByPassKey", _
        dbBoolean, True)
        db.Properties.Append prop
        Resume Next
        Else
            MsgBox "Function 'ap_DisableShift' did not complete successfully."
            GoTo ExitHere
    End If

ExitHere:
    Set prop = Nothing
    Set db = Nothing
    Exit Function

End Function
