Oluşturmada parametrelerle modal: #

Parametreleri bir kipine geçirmek, değerleri bir NavController’a nasıl ilettiğimize benzer. Bunu yapmak için, home.html’deki listemizi, bir liste öğesini tıklatırken bir modal açmak için değiştiriyoruz ve gerekli parametreleri create yöntemine ikinci bir argüman olarak iletiyoruz.

Ana Sayfa.html

 <ion-list>
    <ion-item *ngFor="let datum of data" (click)="openModalwithNavParams(datum)">
      <h1>{{datum.name}}</h1>
    </ion-item>
  </ion-list>

Home.ts

import {EditProfilePage} from '../edit-profile/edit-profile';

  openModalwithNavParams(data){
    let modalWithNavParams = this.modalCtrl.create(EditProfilePage,{Data: data});
    modalWithNavParams.present();
  }

Diğer görünümlere benzer şekilde, önceki görünümden gönderilen verileri almak için NavParams kullanıyoruz.

Profile.ts Düzenle

import { Component } from '@angular/core';
import { ViewController, NavParams } from 'ionic-angular';
@Component({
  templateUrl: 'build/pages/edit-profile/edit-profile.html',
})
export class EditProfilePage {
  viewCtrl;
  navParams;
  data;
  name;
  email;
  mobile;
  nickname;
  constructor(viewCtrl: ViewController, navParams: NavParams) {
    this.viewCtrl = viewCtrl;
    this.navParams = navParams;
    this.data = this.navParams.get('Data');
    this.name = this.data.name;
    this.email = this.data.email;
    this.mobile = this.data.mobile;
    this.nickname = this.data.nickname;
    
  }
  dismiss(){
    this.viewCtrl.dismiss();
  }
}

Basit Modal #

Modal, geçerli sayfanızın üstünde görüntülenen geçici bir kullanıcı arayüzüdür. Bu genellikle oturum açmak, kaydolmak, mevcut seçenekleri düzenlemek ve seçenekleri belirlemek için kullanılır.

Kullanılan modallerle basit bir örneğe bakalım. Başlangıç ​​olarak iyonik boş bir proje oluşturuyoruz. Bir mesaj görüntüleyen basit bir modal oluşturalım ve butona tıklandığında çıkalım. Bunu yapmak için önce modul için bir görünüm oluşturuyoruz.

Mesaj.html

<ion-header>
  <ion-toolbar>
    <ion-title>
      Modal
    </ion-title>
    <ion-buttons start>
      <button (click)="dismiss()">
        <span primary showWhen="ios">Cancel</span>
        <ion-icon name="md-close" showWhen="android,windows"></ion-icon>
      </button>
    </ion-buttons>
  </ion-toolbar>
</ion-header>
<ion-content padding>
  <h1>Modal Without Params is created successfully.</h1>
  <button full (click)="dismiss()"> Exit </button>
</ion-content>

Mesaj.ts

import { Component } from '@angular/core';
import { ViewController } from 'ionic-angular';
@Component({
  templateUrl: 'build/pages/message/message.html',
})
export class MessagePage {
  viewCtrl;
  constructor(viewCtrl: ViewController) {
    this.viewCtrl = viewCtrl;
  }
  dismiss(){
    this.viewCtrl.dismiss();
  }
}

Bu mod bir mesaj görüntüler. Modal, View controllers dismiss yöntemi kullanılarak kapatılabilir veya “kapatılabilir”.

Ana Sayfa.html

<ion-header>
  <ion-navbar>
    <ion-title>
      Modal Example
    </ion-title>
  </ion-navbar>
</ion-header>
<ion-content padding>
  <button full (click)="openModal()">ModalWithoutParams-Message</button>
</ion-content>

Home.ts

import { Component } from '@angular/core';
import { ModalController } from 'ionic-angular';
import {MessagePage} from '../message/message';
@Component({
  templateUrl: 'build/pages/home/home.html'
})
export class HomePage {
  modalCtrl;
  data;
  constructor(modalCtrl: ModalController) {
    this.modalCtrl = modalCtrl;
    this.data = [{name: "aaa", email: "aaa.a@som.com", mobile: "1234567890", nickname: "zzz"},
      {name: "bbb", email: "bbb.a@som.com", mobile: "1234567890", nickname: "yyy"},
      {name: "ccc", email: "ccc.a@som.com", mobile: "1234567890", nickname: "xxx"}]
  }
  openModal() {
    let myModal = this.modalCtrl.create(MessagePage);
    myModal.present();
  }
}

Şimdi ModalController ve veri modelimiz MessagePage’i içe aktararak ana sayfamızı oluşturuyoruz. ModalController’ın create yöntemi, myModal kontrol değişkenine kaydedilen MessagePage veri modelimiz için mod oluşturur. Mevcut yöntemi, geçerli sayfamızın en üstünde kipi açar.

Kapatılan Parametreler ile Modal: #

Artık bir modelin nasıl oluşturulacağını biliyoruz. Peki ya modalden ana sayfamıza bazı verileri aktarmak istersek. Bunu yapmak için, ana sayfaya parametreleri geçiren Register page olarak modal ile bir örnek inceleyelim.

Register.html

<ion-header>
  <ion-toolbar>
    <ion-title>
      Login

    <ion-buttons start>
      <button (click)="dismiss()">
        <span primary showWhen="ios">Cancel</span>
        <ion-icon name="md-close" showWhen="android,windows"></ion-icon>
      </button>
    </ion-buttons>
  </ion-toolbar>

<ion-content padding>

      <ion-input type="text" [(ngModel)]="email"></ion-input>

</ion-content>

Register.ts

import { Component } from '@angular/core';
import { ViewController } from 'ionic-angular';
@Component({
  templateUrl: 'build/pages/register/register.html',
})
export class ResisterPage {
  viewCtrl;
  name;
  email;
  mobile;
  nickname;
  constructor(viewCtrl: ViewController) {
    this.viewCtrl = viewCtrl;
    this.name = "";
    this.email = "";
    this.mobile = "";
    this.nickname = "";
  }
  dismiss(){
    this.viewCtrl.dismiss();
  }
  add(){
    let data = {"name": this.name, "email": this.email, "mobile": this.mobile, "nickname": this.nickname};
    this.viewCtrl.dismiss(data);
  }
}

Register modal, kullanıcı tarafından girilen değerlerle veri nesnesini alır ve parametreler, viewControllers disable yöntemi ile göz ardı edildiğinde mevcut sayfamıza iletilir. Şimdi parametreler gönderilir.

Peki ana sayfadaki parametreleri nasıl alacağız? Bunun için ana sayfada bir buton oluşturuyoruz ve tıklandığında Register modal diyoruz. Kullanıcıyı görüntülemek için bir liste görüntülüyoruz.

　 Ana Sayfa.html

<ion-list>
  <ion-item *ngFor="let datum of data">
    <h1>{{datum.name}}</h1>
  </ion-item>
</ion-list>
<button full secondary (click)="openModalParams()">ModalWithParams-Register</button>

Home.ts

import {ResisterPage} from '../register/register';

  openModalParams(){
    let modalWithParams = this.modalCtrl.create(ResisterPage);
    modalWithParams.present();

    modalWithParams.onDidDismiss((result) =>{
      if(result){
        this.data.unshift(result);
      }
    });
  }

Bir mod kapatıldığında ViewController onDidDismiss yöntemi yürütülür. Veriler modal’dan parametre olarak geçilmişse onDidDismiss metodunu kullanarak alabiliriz. Burada kullanıcı tarafından girilen veriler mevcut verilere eklenir. Parametre olarak hiçbir veri iletilmezse, döndürülen değer null olacaktır.
