Sözdizimi #

  • pygame.draw.rect(Yüzey, renk, Dikdörtgen, genişlik=0)
  • pygame.draw.polygon(Yüzey, renk, nokta listesi, genişlik=0)
  • pygame.draw.circle(Yüzey, renk, konum, yarıçap, genişlik=0)
  • pygame.draw.ellipse(Yüzey, renk, Dikdörtgen, genişlik=0)
  • pygame.draw.arc(Yüzey, renk, Dikdörtgen, start_angle, stop_angle, genişlik=1)
  • pygame.draw.line(Yüzey, renk, start_pos, end_pos, genişlik=1)
  • pygame.draw.lines(Yüzey, renk, kapalı, nokta listesi, genişlik=1)
  • pygame.draw.aaline(Yüzey, renk, başlangıç ​​noktası, bitiş noktası, karışım=1)
  • pygame.draw.aalines(Yüzey, renk, kapalı, nokta listesi, karışım=1)

Parametreler #

parametreler Ayrıntılar
Yüzey Şeklin çizileceği Yüzey.
renk Kırmızı, yeşil ve maviyi (ve alfayı) temsil eden, her biri 0-255 arasında değişen 3 veya 4 tamsayı dizisi.
Rekt Şeklin çizileceği dikdörtgen bir alan.
genişlik Çizgilerin genişliği. Genişlik=0 ise şekil doldurulacaktır.
puan listesi Piksel (x, y) cinsinden rastgele miktarda nokta/köşe listesi.
konum Dairenin merkezinin piksel (x, y) cinsinden konumu.
yarıçap Daire yarıçapı piksel cinsinden.
kapalı Doğruysa, şekli kapatarak son ve ilk nokta arasında bir çizgi çizilir.
karışım=1 Doğruysa, gölgeler, üzerine yazmak yerine mevcut piksel gölgeleriyle karıştırılacaktır.
başlangıç_açı Yayın başlangıç ​​açısı, radyan cinsinden.
stop_angle Yayın son açısı, radyan cinsinden.
start_pos Çizginin piksel cinsinden başlangıç ​​konumu.
end_pos Çizginin piksel cinsinden bitiş konumu

Draw modülü ile çizim #

Pygame, şekilleri doğrudan bir Yüzeye çizebilen işlevleri içeren ‘pygame.draw’ adlı bir modüle sahiptir.

fonksiyon Açıklama
pygame.draw.rect bir dikdörtgen şekli çiz
pygame.draw.poligon herhangi bir sayıda kenarı olan bir şekil çizin
pygame.draw.circle bir noktanın etrafına bir daire çiz
pygame.draw.ellipse bir dikdörtgenin içine yuvarlak bir şekil çizin
pygame.draw.arc elipsin kısmi bir bölümünü çiz
pygame.draw.line düz bir çizgi parçası çiz
pygame.draw.lines birden çok bitişik çizgi parçası çizme
pygame.draw.aaline ince antialiased çizgiler çizin
pygame.draw.aalines birbirine bağlı bir dizi antialiased çizgi çizin

Modül nasıl kullanılır #

Modülü kullanmak için önce pygame’i doğru bir şekilde içe aktarmanız ve başlatmanız ve ekran için bir mod ayarlamanız gerekir. Renk sabitlerini önceden tanımlamak, kodunuzu daha okunabilir ve daha güzel hale getirmek uygundur. Tüm işlevler üzerinde çizim yapmak için bir Yüzey, bir pygame Rect veya 2 elemanlı bir tamsayı/kayan nokta dizisi olan bir renk ve bir konum argümanı alır (“pygame.draw.circle” tanımsız nedenlerden dolayı yalnızca tam sayıları alacaktır).

Örnek #

Aşağıdaki kod, tüm farklı işlevleri, bunların nasıl kullanıldığını ve nasıl göründüklerini gösterecektir. Örneklerden önce pygame’i başlatacağız ve bazı sabitleri tanımlayacağız.

import pygame
from math import pi
pygame.init()

screen = pygame.display.set_mode((100, 100))
WHITE = pygame.Color(255, 255, 255)
RED = pygame.Color(255, 0, 0)

Siyah renk, Yüzeyin varsayılan rengidir ve Yüzeyin üzerine çizilmemiş kısmını temsil eder. Her işlevin parametreleri aşağıda Parametreler bölümünde açıklanmıştır.

Doğru #

size = (50, 50)

rect_border = pygame.Surface(size)  # Create a Surface to draw on.
pygame.draw.rect(rect_border, RED, rect_border.get_rect(), 10)  # Draw on it.

rect_filled = pygame.Surface(size)
pygame.draw.rect(rect_filled, RED, rect_filled.get_rect())

Çokgen #

size = (50, 50)
points = [(25, 0), (50, 25), (25, 50), (0, 25)]  # The corner points of the polygon.

polygon = pygame.Surface(size)
pygame.draw.polygon(polygon, RED, points, 10)

polygon_filled = pygame.Surface(size)
pygame.draw.polygon(polygon_filled, RED, points)

Daire #

size = (50, 50)
radius = 25

circle = pygame.Surface(size)
pygame.draw.circle(circle, RED, (radius, radius), radius, 10)  # Position is the center of the circle.

circle_filled = pygame.Surface(size)
pygame.draw.circle(circle_filled, RED, (radius, radius), radius)

Delikler, pygame’in çizim algoritmasının talihsiz bir sonucudur.

Elips #

size = (50, 25)  # Minimize it's height so it doesn't look like a circle.

ellipse = pygame.Surface(size)  
pygame.draw.ellipse(ellipse, RED, ellipse.get_rect(), 5)

ellipse_filled = pygame.Surface(size)
pygame.draw.ellipse(ellipse_filled, RED, ellipse.get_rect())

Delikler, pygame’in çizim algoritmasının talihsiz bir sonucudur.

Yay #

size = (50, 50)

arc = pygame.Surface(size)
pygame.draw.arc(arc, RED, arc.get_rect(), 0, pi)  # 0 to pi is 180° creating a half circle.

Astar #

size = (50, 50)

line = pygame.Surface(size)
pygame.draw.line(line, RED, (0, 0), (50, 50))  # Start at topleft and ends at bottomright.

Çizgiler #

size = (50, 50)
points = [(25, 0), (50, 25), (25, 50), (0, 25)]

lines = pygame.Surface(size)
pygame.draw.lines(lines, RED, False, points)

lines_closed = pygame.Surface(size)
pygame.draw.lines(lines_closed, RED, True, points)

Kenar yumuşatılmış satır #

size = (50, 50)

antialiased_line = pygame.Surface(size)
pygame.draw.aaline(antialiased_line, RED, (0, 0), (50, 50))

Kenar yumuşatılmış çizgiler #

size = (50, 50)
points = [(25, 0), (50, 25), (25, 50), (0, 25)]

antialiased_lines = pygame.Surface(size)
pygame.draw.aalines(antialiased_lines, RED, False, points)

antialiased_lines_closed = pygame.Surface(size)
pygame.draw.aalines(antialiased_lines_closed, RED, True, points)

Yüzeyler #

Pygame’de genellikle nesnelerin görünümünü temsil etmek için Yüzeyleri ve konumlarını temsil etmek için Dikdörtgenleri kullanırsınız. Yüzey, renkleri veya görüntüleri içeren boş bir kağıt sayfası gibidir. Yüzey oluşturmanın iki yolu vardır: sıfırdan boş veya bir görüntü yükleyerek.

Bir Yüzey Oluşturun #

Bir Yüzey oluşturmak için minimum boyutuna ihtiyacınız vardır; bu, boyutu piksel cinsinden temsil eden 2 elemanlı genişlik ve yükseklik tamsayı dizisidir.

Bit derinliğini, maskeleri ve ek özellikleri kontrol etmek için bir Yüzey oluştururken piksel başına alfa olarak ek argümanlar iletebilir ve/veya görüntü belleğinde görüntü oluşturabilirsiniz. Ancak bu, bu örneğin kapsamı dışındadır.

size = width, height = (32, 32)
empty_surface = pygame.Surface(size)

Yüzey üzerinde şekiller çizmek için pygame.draw modülünü kullanabilir veya Yüzey yöntemini fill(color) çağırarak bir renkle doldurabilirsiniz. color argümanı 3 veya 4 elemanlı bir tamsayı dizisi veya bir “pygame.Color” nesnesidir.

Bir resim yükleyin #

Çoğu zaman bir oyunda kendi resimlerinizi kullanmak istersiniz (sprite olarak adlandırılır). Resminiz açıkken bir Yüzey oluşturmak şu kadar kolaydır:

my_image = pygame.image.load(path_to_image)

Görüntüye giden yol göreli veya mutlak olabilir. Performansı artırmak için genellikle görüntünüzü ekranla aynı piksel biçimine dönüştürmek akıllıca olur. Bu, Surface yöntemi ‘convert()’ çağrılarak şu şekilde yapılabilir:

my_image = pygame.image.load(path_to_image).convert()

Resminiz şeffaflık (alfa değerleri) içeriyorsa, bunun yerine ‘convert_alpha()’ yöntemini çağırmanız yeterlidir:

my_image = pygame.image.load(path_to_image).convert_alpha()

Blitting #

Yüzeylerin görüntülenebilmesi için ekrana kırpılması gerekir. Blitting aslında pikselleri bir Yüzeyden diğerine kopyalamak anlamına gelir (ekran da bir Yüzeydir). Ayrıca, 2 elemanlı bir tamsayı dizisi veya bir Rect nesnesi olması gereken Yüzeyin konumunu da iletmeniz gerekir. Yüzeyin üst kısmı konuma yerleştirilecektir.

screen.blit(my_image, (0, 0))
pygame.display.update()  # or pygame.display.flip()

Ekrandan başka Yüzeylere blim yapmak mümkündür. Ekrana neyin kırıldığını görüntülemek için ‘pygame.display.update()’ veya ‘pygame.display.flip()’ çağırmanız gerekir.

Şeffaflık #

Pygame’de desteklenen 3 çeşit şeffaflık vardır: renk tuşları, Yüzey alfaları ve piksel başına alfalar.

Renk Tuşları #

Bir rengi tamamen şeffaf hale getirir veya daha doğru bir şekilde, bir rengin blit olmamasını sağlar. İçinde siyah bir dikdörtgen olan bir görüntünüz varsa, siyah rengin kararmasını önlemek için bir renk anahtarı ayarlayabilirsiniz.

BLACK = (0, 0, 0)
my_image.set_colorkey(BLACK)  # Black colors will not be blit.

Bir Yüzeyin yalnızca bir renk anahtarı olabilir. Başka bir renk anahtarı ayarlamak öncekinin üzerine yazacaktır. Renk tuşları farklı alfa değerlerine sahip olamaz, yalnızca bir rengi görünmez yapabilir.

Yüzey alfaları #

Tüm Yüzeyi bir alfa değeriyle saydam yapar. Bu yöntemle farklı alfa değerlerine sahip olabilirsiniz ancak bu tüm Yüzeyi etkileyecektir.

my_image.set_alpha(100)  # 0 is fully transparent and 255 fully opaque.

piksel başına alfa #

Yüzeydeki her pikseli ayrı bir alfa değeriyle saydam yapar. Bu size en fazla özgürlük ve esnekliği sağlar, ancak aynı zamanda en yavaş yöntemdir. Bu yöntem ayrıca Yüzeyin piksel başına alfa Yüzey olarak oluşturulmasını gerektirir ve renk bağımsız değişkenlerinin dördüncü bir alfa tamsayı içermesi gerekir.

size = width, height = (32, 32)
my_image = pygame.Surface(size, pygame.SRCALPHA)  # Creates an empty per-pixel alpha Surface.

Renk dördüncü alfa değerini içeriyorsa Yüzey şimdi saydamlık çizecektir.

BLUE = (0, 0, 255, 255)
pygame.draw.rect(my_image, BLUE, my_image.get_rect(), 10)

Diğer Yüzeylerin aksine, bu Yüzey varsayılan rengi siyah değil şeffaf olacaktır. Bu yüzden ortadaki siyah dikdörtgen kayboluyor.

Colorkey ve Surface alpha’yı birleştirin #

Renk tuşları ve Yüzey alfaları birleştirilebilir, ancak piksel başına alfa birleştirilemez. Piksel başına bir Yüzeyin daha yavaş performansını istemiyorsanız bu yararlı olabilir.

purple_image.set_colorkey(BLACK)
purple_image.set_alpha(50)

Tam kod #

Bunu boş bir dosyaya kopyalayın ve çalıştırın. Görüntülerin görünmesi için 1, 2, 3 veya 4 tuşlarına basın. Daha opak hale getirmek için 2, 3 veya 4’e birden çok kez basın.

import pygame
pygame.init()

WHITE = (255, 255, 255)
BLACK = (0, 0, 0)
RED = (255, 0, 0)
GREEN = (0, 255, 0)
BLUE = (0, 0, 255, 50)  # This color contains an extra integer. It's the alpha value.
PURPLE = (255, 0, 255)

screen = pygame.display.set_mode((200, 325))
screen.fill(WHITE)  # Make the background white. Remember that the screen is a Surface!
clock = pygame.time.Clock()

size = (50, 50)
red_image = pygame.Surface(size)
green_image = pygame.Surface(size)
blue_image = pygame.Surface(size, pygame.SRCALPHA)  # Contains a flag telling pygame that the Surface is per-pixel alpha
purple_image = pygame.Surface(size)

red_image.set_colorkey(BLACK)
green_image.set_alpha(50)
# For the 'blue_image' it's the alpha value of the color that's been drawn to each pixel that determines transparency.
purple_image.set_colorkey(BLACK)
purple_image.set_alpha(50)

pygame.draw.rect(red_image, RED, red_image.get_rect(), 10)
pygame.draw.rect(green_image, GREEN, green_image.get_rect(), 10)
pygame.draw.rect(blue_image, BLUE, blue_image.get_rect(), 10)
pygame.draw.rect(purple_image, PURPLE, purple_image.get_rect(), 10)

while True:
    clock.tick(60)

    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            quit()
        elif event.type == pygame.KEYDOWN:
            if event.key == pygame.K_1:
                screen.blit(red_image, (75, 25))
            elif event.key == pygame.K_2:
                screen.blit(green_image, (75, 100))
            elif event.key == pygame.K_3:
                screen.blit(blue_image, (75, 175))
            elif event.key == pygame.K_4:
                screen.blit(purple_image, (75, 250))

    pygame.display.update()
