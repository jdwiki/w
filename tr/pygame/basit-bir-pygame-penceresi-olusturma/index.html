Basit bir pygame penceresi oluşturma

  • Kodun tamamı

Bu sayfada

  • Kodun tamamı

Kodun tamamı #

import pygame

pygame.init()

WIDTH = 300
HEIGHT = 200
SCREEN = pygame.display.set_mode((WIDTH, HEIGHT))

pygame.display.set_caption('My Game')

WHITE = (255, 255, 255)
BLACK = (0, 0, 0)
RED = (255, 0, 0)
GREEN = (0, 255, 0)
BLUE = (0, 0, 255)
YELLOW = (255, 255, 255)

SCREEN.fill(RED)
pygame.display.flip()

is_running = True
while is_running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            is_running = False

pygame.quit()

Pygame’i içe aktarma ve başlatma

Python’daki herhangi bir modülde yaptığımız gibi, pygame’i içe aktarmamız gerekiyor:

import pygame

Daha sonra içe aktarılan tüm pygame modüllerini başlatıyoruz:

pygame.init()

Bu, tüm pygame modüllerini başlatmak için kullanılır. Bu olmadan modüller çalışmaz

Sabitleri Tanımlama

Daha sonra burada bazı sabitleri tanımlarız:

WIDTH = 300
HEIGHT = 200
SCREEN = pygame.display.set_mode((WIDTH, HEIGHT))

“WIDTH” ve “HEIGHT” sabitleri, 300 piksel genişliğe ve 200 piksel yüksekliğe sahip olacak bir pencere oluşturmak için kullanılır. ‘SCREEN’, ‘pygame.display.set_mode((WIDTH, HEIGHT))’ içinde kullanılan işlev, ekranın modunu ayarlayacak ve bir Yüzey nesnesi döndürecektir. Bu işlevin parametrelerinin daha önce tanımlanan “WIDTH” ve “HEIGHT” sabitleri olduğuna dikkat edin.

Pencerenin adını ayarlama

Daha sonra pencerenin adını Oyunum olarak değiştirmek için bu işlevi kullanırız:

pygame.display.set_caption('My Game')

Renklerin tanımlanması

Daha sonra penceremizde kullanılabilecek 6 renk tanımlıyoruz:

WHITE = (255, 255, 255)
BLACK = (0, 0, 0)
RED = (255, 0, 0)
GREEN = (0, 255, 0)
BLUE = (0, 0, 255)
YELLOW = (255, 255, 255)

Renkleri tanımlarken 0 ile 255 arasında değişen 3 değer koyarız. pygame.Color sınıfı normalde şu formatta çalışır:

COLOUR = (r, g, b, a)

r parametresinin rengin kırmızı değerini belirlediği yerde, g parametresi rengin yeşil değerini ve b parametresi rengin mavi değerini ayarlar. a parametresi rengin alfa değerini ayarlar.

Daha sonra şu komutu veriyoruz:

SCREEN.fill(RED)

Bu, ekranımız olan Surface nesnesini kırmızı renkle dolduran bir pygame.Surface.fill işlevidir.

pygame.display.flip() kullanarak

Daha sonra bu işlevi kullanırız

pygame.display.flip()

Bu temelde, Yüzeyde çizdiğimiz her şeyi görünür hale getirir ve tüm ekranın içeriğini günceller. Bu satır olmadan, kullanıcı pygame ekranında hiçbir şey görmez.

Oyun döngüsü

Sonraki birkaç satıra “oyun döngüsü” denir.

Bunu başlatmak için bir değişken yapıyoruz ve onu True yapıyoruz:

is_running = True

Böylece while döngümüze başlayabiliriz:

while is_running:

hangi tüm oyun boyunca çalışacak.

En temel biçiminde, pygame, örneğin bir düğmeye basma veya fare tıklaması gibi kullanıcı girdisi alan “olaylara” sahiptir. Pygame bu olayları bir olay kuyruğu aracılığıyla ele alır. Bu olayları olay kuyruğundan şu for döngüsüyle alabiliriz:

for event in pygame.event.get():

Temel olarak bir olaylar listesinden geçen olay kuyruğumuz. Bunlar sonraki 2 satır:

if event.type == pygame.QUIT:
    is_running = False

Bu, kullanıcı üst köşedeki çıkış düğmesine bastığında ‘pygame.QUIT’ türündeki olayın gerçekleşmesini sağlayacaktır.

“is_running” artık “False” olduğundan ve komut dosyası son satıra geçtiğinden bu, while döngüsünü sona erdirir:

pygame.quit()

Hangi pygame modüllerini başlatır.
