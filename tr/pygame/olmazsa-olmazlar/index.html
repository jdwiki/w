Çizim ve temel bir animasyon #

Bu program bazı şekiller ve ‘merhaba dünya!’ çizer ve pencerenin her köşesine bir görüntünün gitmesine izin verir.

kodun tamamı: #

import pygame,sys
from pygame.locals import *

pygame.init()

FPS = 30 #frames per second setting
fpsClock = pygame.time.Clock()

#set up the window
screen = pygame.display.set_mode((500,400), 0, 32)
pygame.display.set_caption('drawing')

#set up the colors
black = (  0,   0,   0)
white = (255, 255, 255)
red   = (255,   0,   0)
green = (  0, 255,   0)
blue  = (  0,   0, 255)

imageImg  = pygame.image.load('baddie.png')
imagex = 320
imagey = 220
direction = 'left'

fontObj = pygame.font.Font('freesansbold.ttf', 32)
text = fontObj.render('Hello World!', True, green, blue)
rect = text.get_rect()
rect.center = (200, 150)

# the main game loop
while True:
    screen.fill(white)

    # draw a green polygon onto the surface
    pygame.draw.polygon(screen, green, ((146, 0), (291, 106), (236, 277), (56, 277), (0, 106)))

    # draw some blue lines onto the surface
    pygame.draw.line(screen, blue, (60, 60), (120, 60), 4)
    pygame.draw.line(screen, blue, (120, 60), (60, 120))
    pygame.draw.line(screen, blue, (60, 120), (120, 120), 4)

    # draw a blue circle onto the surface
    pygame.draw.circle(screen, blue, (300, 50), 100, 0)

    # draw a red ellipse onto the surface
    pygame.draw.ellipse(screen, red, (300, 250, 80,80), 1)

    # draw a red rectangle onto the surface
    pygame.draw.rect(screen,red, (200, 150, 100, 50))

    # draw the text onto the surface
    screen.blit(text, rect)

    if direction == 'right':
        imagex += 5
        if imagex == 320:
            direction = 'down'
    elif direction == 'down':
        imagey += 5
        if imagey == 220:
            direction = 'left'
    elif direction == 'left':
        imagex -= 5
        if imagex == 20:
           direction = 'up'
    elif direction == 'up':
        imagey -= 5
        if imagey == 20:
           direction = 'right'
    screen.blit(imageImg, (imagex, imagey))

for event in pygame.event.get():
    if event.type == QUIT:
        pygame.quit()
        sys.exit()

pygame.display.update()
fpsClock.tick(FPS)

pygame ve pencerenin kurulumu: #

import pygame,sys
from pygame.locals import *

pygame.init()

#set up the window
screen = pygame.display.set_mode((500,400), 0, 32)
pygame.display.set_caption('drawing')

beyaz arka planı çizme: #

Bu fonksiyonda arka planın rengini tanımlarsınız.

screen.fill(white)

yeşil çokgen çizimi: #

Burada çokgenin her köşesinin (x ve y koordinatları) görüntü yüzeyini, rengini ve konumunu tanımlarsınız, bunu saat yönünde ve saat yönünün tersine yapabilirsiniz.

pygame.draw.polygon(screen, green, ((146, 0), (291, 106), (236, 277), (56, 277), (0, 106)))

mavi çizgilerin çizilmesi: #

Bu fonksiyonda, görüntü yüzeyini, rengi, ilk ve son noktayı ve çizginin genişliğini tanımlarsınız (genişlik vermezseniz, sadece 1’dir).

pygame.draw.line(screen, blue, (60, 60), (120, 60), 4)
pygame.draw.line(screen, blue, (120, 60), (60, 120))
pygame.draw.line(screen, blue, (60, 120), (120, 120), 4)

mavi dairenin çizilmesi: #

Bu fonksiyonda dairenin ekran yüzeyini, rengini, konumunu, yarıçapını ve genişliğini tanımlarsınız (genişlik için 0 verirseniz, bu düz bir dairedir).

pygame.draw.circle(screen, blue, (300, 50), 100, 0)

elips çizimi: #

Bu fonksiyonda ekran yüzeyini, rengi, konumu, yatay boyutu ve dikey boyutu ve genişliği tanımlarsınız.

pygame.draw.ellipse(screen, red, (300, 250, 80,80), 1)

dikdörtgenin çizilmesi: #

Bu fonksiyonda ekran yüzeyini, rengi, konumu ve yatay ve dikey boyutu tanımlarsınız.

pygame.draw.rect(screen,red, (200, 150, 100, 50))

metni tanımlama: #

İlk önce bu fonksiyonla metninizin türünü ve boyutunu tanımlarsınız:

fontObj = pygame.font.Font('freesansbold.ttf', 32)

Ardından asıl metni, eğer metin kalın ise rengini ve isterseniz işaretlemenin rengini tanımlarsınız. Bunu bu işlevle yapabilirsiniz:

text = fontObj.render('Hello World!', True, green, blue)

Metninizi işaretlemek istiyorsanız, pygame’e bu fonksiyonla şunu söylemelisiniz:

rect = text.get_rect()

Ve eğer metnin merkezinin konumunu tanımlamak istiyorsanız, bunu bu fonksiyonla yapabilirsiniz:

rect.center = (200, 150)

metni çizme: #

Bir işaretleme ve/veya merkez tanımladıysanız:

screen.blit(text, rect)

Aksi takdirde, metnin konumunu tanımlamanız gerekir, bu nedenle metni şu şekilde çizin:

screen.blit(text, (100,50))

resmin tanımlanması: #

Burada hangi görüntüyü kullanmak istediğinizi (bunu bu şekilde yaparsanız, görüntü dosyasının program dosyanızla aynı dizinde olması gerekir), başlangıç ​​konumunu (x ve y) ve görüntünün yönünü tanımlarsınız.

image  = pygame.image.load('image.png')
baddiex = 320
baddiey = 220
direction = 'left'

görüntüyü canlandırmak: #

Kodun bu kısmı ile görüntünün yönünü kontrol ediyoruz, eğer bir köşeye ulaştıysa yönünü değiştirin, değilse görüntüyü aynı yönde 5 piksel daha çizin.

if direction == 'right':
    imagex += 5
    if imagex == 360:
        direction = 'down'
elif direction == 'down':
    imagey += 5
    if imagey == 260:
        direction = 'left'
elif direction == 'left':
    imagex -= 5
    if imagex == 20:
       direction = 'up'
elif direction == 'up':
    imagey -= 5
    if imagey == 20:
       direction = 'right'
screen.blit(imageImg, (imagex, imagey))

not: Resmim 20x20 piksel, kullanıyorum if imagex == 360: ve if imagey == 260: çünkü resmim pencerenin kenarından 20 piksel uzaktadır , tıpkı diğer 2 köşe gibi. Resminizin boyutu farklıysa, muhtemelen bu sayıları değiştirmeniz gerekecektir.

bırakma kontrolü: #

Burada pygame penceresini kapatıp kapatmadığınızı kontrol ediyoruz ve eğer öyleyse, pencereyi kapatın, bunu programınızda bir yere yazmazsanız, muhtemelen pencereyi kapatamayacaksınız.

for event in pygame.event.get():
    if event.type == QUIT:
        pygame.quit()
        sys.exit()

ekranı güncelleme: #

Bu fonksiyon ile ekranı güncellersiniz, böylece çizdiğiniz her şey görünür hale gelir.

pygame.display.update()

#FPS ayarı: Bu fonksiyon ile pygame’e FPS ayarınıza saygı duyulması için yeterince uyumasını söylersiniz.

fpsClock.tick(FPS)

PIL ile kullanma #

Her ikisinde de eksik işlevler nedeniyle hem PIL hem de Pygame kullanmanız gerektiğinde, tercihen diske yazmadan Pygame Yüzeyleri ve PIL Görüntüleri arasında dönüştürme yapmak için bir yola ihtiyacınız vardır.

Bunun için her iki kitaplıkta da sağlanan “tostring” ve “fromstring” işlevlerini kullanabilirsiniz.

PIL’den Pygame’e dönüştürme:

strFormat = 'RGBA'
raw_str = image.tostring("raw", strFormat)
surface = pygame.image.fromstring(raw_str, image.size, strFormat)

Pygame’den PIL’e Dönüşüm:

strFormat = 'RGBA'
raw_str = pygame.image.tostring(surface, strFormat, False)
image = Image.frombytes(strFormat, surface.get_size(), raw_str)
