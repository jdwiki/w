React ve React Router ile Merhaba Dünya #

‘react’ ve ‘react-router’ı kurduktan sonra, ikisini birlikte kullanma zamanı.

Sözdizimi çok basittir, url açıldığında oluşturmak istediğiniz ‘url’yi ve ‘bileşeni’ belirtirsiniz.

<Route path="merhaba" component={ HelloComponent } />

Bunun anlamı, url yolu “merhaba” olduğunda, “HelloComponent” bileşenini oluştur

DOSYA ADI: “app.js”

'use strict';

import React from 'react';
import { render } from 'react-dom';
import { Router, browserHistory, Link } from 'react-router';

// These are just demo components which render different text.

let DashboardPage = () => (
  <div>
    <h1>Welcome User</h1>
    <p>This is your dashboard and I am an example of a stateless functional component.</p>
    <Link to="/settings">Goto Settings Page</Link>
  </div>
)

let SettingsPage = () => (
  <div>
    <h1>Manage your settings</h1>
    <p>display the settings form fields here...or whatever you want</p>
    <Link to="/">Back to Dashboard Page</Link>
  </div>
)

let AuthLoginPage = () => (
  <div>
    <h1>Login Now</h1>
    <div>
      <form action="">
        <input type="text" name="email" placeholder="email address" />
        <input type="password" name="password" placeholder="password" />
        <button type="submit">Login</button>
      </form>
    </div>
  </div>
)

let AuthLogoutPage = () => (
  <div>
    <h1>You have been successfully logged out.</h1>
    <div style={{ marginTop: 30 }}>
      <Link to="/auth/login">Back to login page</Link>
    </div>
  </div>
)

let ArticlePage = ({ params }) => (
  <h3>Article {params.id}</h3>
)

let PageNotFound = () => (
  <div>
    <h1>The page you're looking for doesn't exist.</h1>
  </div>
)

// Here we pass Router to the render function.
render( (
    <Router history={ browserHistory }>

      <Route path="/" component={ DashboardPage } />
      <Route path="settings" component={ SettingsPage } />

      <Route path="auth">
        <IndexRoute component={ AuthLoginPage } />
        <Route path="login" component={ AuthLoginPage } />
        <Route path="logout" component={ AuthLogoutPage } />
      </Route>
    
      <Route path="articles/:id" component={ ArticlePage } />

      <Route path="*" component={ PageNotFound } />

    </Router>
  ), document.body );

Rota Parametreleri : Yönlendirici yolu, parametrenin değerini bileşende okuyabilmemiz için parametreleri alacak şekilde yapılandırılabilir. <Route path="makaleler/:id" component={ ArticlePage } /> içindeki yolun bir /:id var. Bu “id” değişkeni, path parametresinin amacına hizmet eder ve “ArticlePage” bileşeninden “{props.params.id}” kullanılarak erişilebilir.

“http://localhost:3000/#/articles/123” adresini ziyaret edersek, “ArticlePage” bileşenindeki “{props.params.id}” 123 olarak çözülecektir. Ancak, “http://localhost:3000” adresini ziyaret edersek /#/articles`, id parametresi olmadığı için çalışmayacak.

route parametresi, bir çift parantez arasına yazılarak isteğe bağlı yapılabilir: <Route path="makaleler(/:id)" component={ ArticlePage } />

Alt yolları kullanmak istiyorsanız, şunları yapabilirsiniz:

<Route path="path" component={ PathComponent }>
  <Route path="subpath" component={ SubPathComponent } />
</Route>
  • /pathe erişildiğinde, PathComponent oluşturulacak
  • “/path/subpath"e erişildiğinde, “PathComponent” işlenecek ve “SubPathComponent” ona “props.children” olarak geçirilecektir.

Mevcut olmayan tüm rotaları yakalamak ve ‘404 sayfası bulunamadı’ sayfasını oluşturmak için path="*" kullanabilirsiniz.

Başlarken #

Bu başlarken, [create-react-app][1] veya Babel’i ve oradaki tüm güzellikleri kullanarak eşdeğer bir şeyle çalıştığınızı varsayar.

Ayrıca [buradan][2] harika belgelere de göz atın.

İlk önce, tepki-yönlendirici-dom’u kurun:

npm install tepki-yönlendirici-dom' veya yarn tepki-yönlendirici-dom ekle’.

Ardından, iki öğe ve temel sayfalar içeren temel bir Navbar’dan oluşan bir bileşen oluşturun:

import React from 'react'
import { BrowserRouter, Route, Link } from 'react-router-dom'

const Home = () => (
  <div>
    <p>We are now on the HOME page</p>
  </div>
)

const About = () => (
  <div>
    <p>We are now on the ABOUT page</p>
  </div>
)

const App = () => (
  <BrowserRouter>
    <div>
      <ul>
        <li><Link to="/">Home</Link></li>
        <li><Link to="/about">About</Link></li>
      </ul>
      <hr/>
      <Route path="/" component={Home}/>
      <Route path="/about" component={About}/>
    </div>
  </BrowserRouter>
)
export default App

Bu kodu adım adım inceleyelim:

  • React'i 'react'ten içe aktar: Reacti içe aktardığınızdan emin olun
  • react-router-dom'dan { BrowserRouter as Router, Route, Link } içe aktar bölünmüş:
  • “BrowserRouter” asıl yönlendiricinin kendisidir. Bileşeninizi “BrowserRouter” bileşeninin içine sardığınızdan emin olun.
  • “Rota”, gidilebilecek belirli bir rotadır.
  • “Bağlantı”, köprü olarak kullanabileceğiniz bir “” etiketi üreten bir bileşendir.
  • ‘const Home’, ana sayfayı döndüren bir işlevdir.
  • const Hakkında, Hakkında sayfasını döndüren bir işlevdir.

  • “const App” ana bileşendir:

  • “”, “” bileşenini kullanmak istediğiniz bileşenleri saran [JSX][3] bileşenidir.

  • ‘

    ’, ‘BrowserRouter’ içindeki tüm JSX’i sarmak için tek bir öğedir.

  • “

      ” Navbar’dır. Ana Sayfaya bir bağlantı ve Hakkında’ya bir bağlantı içerir.

    • Ana sayfaya <li><Link to="/">Ana Sayfa</Link></li> bağlantıları. Bağlantı “/” ile ilgili olduğundan, boş bir göreli yolun ana sayfayı oluşturduğunu görebilirsiniz.

    • Hakkında sayfasına <li><Link to="/about">Hakkında</Link></li> bağlantılar.

    • <Route path="/" component={Home}/>, göreli yol "/" ise hangi bileşenin oluşturulması gerektiğini açıklar.

    • <Route path="/about" component={About}/>, göreli yol "/about" ise hangi bileşenin oluşturulması gerektiğini açıklar.

    Buradan öğrenecek çok şey var, ama umarım bu temelleri açıklar, böylece buradan öğrenmeye devam edebilirsiniz.

    Kurulum ve Kurulum #

    React Router’ı kurmak için npm komutunu çalıştırmanız yeterlidir.

    npm install --save tepki-yönlendiriciyi kaydet

    Ve işin bitti. Reaksiyon yönlendiricisini kurmak için yapmanız gereken tam anlamıyla bu.

    Lütfen Dikkat: ’tepki-yönlendirici’, ’tepki’ye bağlıdır, bu nedenle ’tepki’yi de yüklediğinizden emin olun.

    Kurmak:

    babil gibi bir ES6 aktarıcı kullanarak

    import { Router, Route, Link } from 'react-router'

    ES6 aktarıcı kullanmamak

    var Router = require('react-router').Router
var Route = require('react-router').Route
var Link = require('react-router').Link

    UMD derlemesini kullanarak kurulum #

    npmcdn adresinde bir yapı da mevcuttur. Komut dosyasını şu şekilde ekleyebilirsiniz:

    <script src="https://npmcdn.com/react-router/umd/ReactRouter.min.js"></script>

    Kütüphane global olarak window.ReactRouter üzerinden erişilebilir olacaktır.
