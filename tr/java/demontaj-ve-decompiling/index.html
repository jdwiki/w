Sözdizimi #

  • javap [seçenekler] <sınıflar>

Parametreler #

İsim Açıklama
<sınıflar> Sökülecek sınıfların listesi. “package1.package2.Classname” biçiminde, * veya* “package1/package2/Classname” biçiminde olabilir. “.class” uzantısını dahil etmeyin**.
-help, --help, -? Bu kullanım mesajını yazdır
-sürüm Sürüm bilgileri
-v, -verbose Ek bilgileri yazdır
-l Satır numarası ve yerel değişken tablolarını yazdır
`-kamu' Yalnızca genel sınıfları ve üyeleri göster
-korumalı Korumalı/genel sınıfları ve üyeleri göster
`-paket' Paketi/korumalı/genel sınıfları ve üyeleri göster (varsayılan)
-p, -özel Tüm sınıfları ve üyeleri göster
-c Kodu sökün
-s Dahili tip imzaları yazdırın
-sysinfo İşlenmekte olan sınıfın sistem bilgilerini (yol, boyut, tarih, MD5 karması) göster
`-sabitler' Son sabitleri göster
-classpath <path> Kullanıcı sınıfı dosyalarının nerede bulunacağını belirtin
-cp <yol> Kullanıcı sınıfı dosyalarının nerede bulunacağını belirtin
-bootclasspath <path> Önyükleme sınıfı dosyalarının konumunu geçersiz kıl

javap ile bayt kodunu görüntüleme #

Bir Java programı için oluşturulan bayt kodunu görmek istiyorsanız, onu görüntülemek için sağlanan javap komutunu kullanabilirsiniz.

Aşağıdaki Java kaynak dosyasına sahip olduğumuzu varsayarsak:

package com.stackoverflow.documentation;

import org.springframework.stereotype.Service;

import java.io.IOException;
import java.io.InputStream;
import java.util.List;

@Service
public class HelloWorldService {

    public void sayHello() {
        System.out.println("Hello, World!");
    }

    private Object[] pvtMethod(List<String> strings) {
        return new Object[]{strings};
    }

    protected String tryCatchResources(String filename) throws IOException {
        try (InputStream inputStream = getClass().getResourceAsStream(filename)) {
            byte[] bytes = new byte[8192];
            int read = inputStream.read(bytes);
            return new String(bytes, 0, read);
        } catch (IOException | RuntimeException e) {
            e.printStackTrace();
            throw e;
        }
    }

    void stuff() {
        System.out.println("stuff");
    }
}
