Dizeyi diğer veri türlerine dönüştürme. #

Bir sayısal dizeyi aşağıdaki gibi çeşitli Java sayısal türlerine dönüştürebilirsiniz:

** int’ye dize:**

String number = "12";
int num = Integer.parseInt(number);

Kayan dize:

String number = "12.0";
float num = Float.parseFloat(number);

İkili için dize:

String double = "1.47";
double num = Double.parseDouble(double);

Boole için dize:

String falseString = "False";
boolean falseBool = Boolean.parseBoolean(falseString);   // falseBool = false 
    
String trueString = "True";
boolean trueBool = Boolean.parseBoolean(trueString);     // trueBool = true

** Uzun dize:**

String number = "47";
long num = Long.parseLong(number);

BigInteger için dize:

String bigNumber = "21";
BigInteger reallyBig = new BigInteger(bigNumber);

BigDecimal’e Dize:

String bigFraction = "17.21455";
BigDecimal reallyBig = new BigDecimal(bigFraction);

Dönüşüm İstisnaları:

Uygun şekilde biçimlendirilmemiş bir sayı olmayan veya hedef türü için aralık dışında olan bir dizeyi ayrıştırmaya çalışırsanız, yukarıdaki sayısal dönüşümlerin tümü (işaretlenmemiş) bir ‘NumberFormatException’ verir. İstisnalar konusu, bu tür istisnalarla nasıl başa çıkılacağını tartışır.

Bir dizgeyi ayrıştırabileceğinizi test etmek istiyorsanız, şunun gibi bir tryParse... yöntemini uygulayabilirsiniz:

boolean tryParseInt (String value) {  
    try {  
        String somechar = Integer.parseInt(value);
        return true;  
     } catch (NumberFormatException e) { 
        return false;  
     }  
}

Ancak, bu tryParse... yöntemini ayrıştırmadan hemen önce çağırmak (tartışmalı olarak) kötü bir uygulamadır. Sadece ‘parse…’ yöntemini çağırmak ve istisnayı ele almak daha iyi olurdu.

Baytlara / baytlardan dönüştürme #

Bir dizeyi bir bayt dizisine kodlamak için, herhangi bir Java çalışma zamanında mevcut olan standart karakter kümelerinden biriyle String#getBytes() yöntemini kullanabilirsiniz:

byte[] bytes = "test".getBytes(StandardCharsets.UTF_8);

ve kodunu çözmek için:

String testString = new String(bytes, StandardCharsets.UTF_8);

statik bir içe aktarma kullanarak aramayı daha da basitleştirebilirsiniz:

import static java.nio.charset.StandardCharsets.UTF_8;
...
byte[] bytes = "test".getBytes(UTF_8);

Daha az yaygın olan karakter kümeleri için karakter kümesini bir dize ile belirtebilirsiniz:

byte[] bytes = "test".getBytes("UTF-8");

ve tersi:

String testString = new String (bytes, "UTF-8");

ancak bu, işaretli ‘UnsupportedCharsetException’ ile ilgilenmeniz gerektiği anlamına gelir.

Aşağıdaki çağrı varsayılan karakter setini kullanacaktır. Varsayılan karakter seti platforma özeldir ve genellikle Windows, Mac ve Linux platformları arasında farklılık gösterir.

byte[] bytes = "test".getBytes();

ve tersi:

String testString = new String(bytes);

Geçersiz karakterlerin ve baytların bu yöntemlerle değiştirilebileceğini veya atlanabileceğini unutmayın. Daha fazla kontrol için - örneğin girişi doğrulamak için - “CharsetEncoder” ve “CharsetDecoder” sınıflarını kullanmanız önerilir.

Base64 Kodlama / Kod Çözme #

Bazen ikili verileri base64 kodlu bir dize olarak kodlamanız gerektiğini göreceksiniz.

Bunun için javax.xml.bind paketindeki DatatypeConverter sınıfını kullanabiliriz:

import javax.xml.bind.DatatypeConverter;
import java.util.Arrays;

// arbitrary binary data specified as a byte array
byte[] binaryData = "some arbitrary data".getBytes("UTF-8");

// convert the binary data to the base64-encoded string
String encodedData = DatatypeConverter.printBase64Binary(binaryData);
// encodedData is now "c29tZSBhcmJpdHJhcnkgZGF0YQ=="

// convert the base64-encoded string back to a byte array
byte[] decodedData = DatatypeConverter.parseBase64Binary(encodedData);

// assert that the original data and the decoded data are equal
assert Arrays.equals(binaryData, decodedData);

Apache commons-codec’i

Alternatif olarak, Apache commons-codec’ten ‘Base64’ü kullanabiliriz.

import org.apache.commons.codec.binary.Base64;

// your blob of binary as a byte array
byte[] blob = "someBinaryData".getBytes();

// use the Base64 class to encode
String binaryAsAString = Base64.encodeBase64String(blob);

// use the Base64 class to decode
byte[] blob2 = Base64.decodeBase64(binaryAsAString);

// assert that the two blobs are equal
System.out.println("Equal : " + Boolean.toString(Arrays.equals(blob, blob2)));

Bu programı çalışırken incelerseniz, ‘someBinaryData’nın, çok yönetilebilir bir UTF-8 String nesnesi olan ‘c29tZUJpbmFyeURhdGE=’ olarak kodlandığını göreceksiniz.

Aynısı için ayrıntılar Base64 adresinde bulunabilir.

// encode with padding
String encoded = Base64.getEncoder().encodeToString(someByteArray);

// encode without padding
String encoded = Base64.getEncoder().withoutPadding().encodeToString(someByteArray);

// decode a String
byte [] barr = Base64.getDecoder().decode(encoded);

Diğer veri türlerini String’e dönüştürme #

  • String sınıfının ‘valueOf’ yöntemlerinden birini kullanarak diğer ilkel veri türlerinin değerini String olarak alabilirsiniz.

    For example:

    int i = 42;
String string = String.valueOf(i);
//string now equals "42”.

    This method is also overloaded for other datatypes, such as float, double, boolean, and even Object.

  • Ayrıca herhangi bir Object (herhangi bir sınıfın herhangi bir örneğini) üzerinde .toString çağırarak bir String olarak alabilirsiniz. Bunun faydalı çıktı vermesi için sınıfın toString() öğesini geçersiz kılması gerekir. “Date” ve diğerleri gibi standart Java kitaplığı sınıflarının çoğu bunu yapar.

    For example:

    Foo foo = new Foo(); //Any class.
String stringifiedFoo = foo.toString().

    Here stringifiedFoo contains a representation of foo as a String.

Ayrıca herhangi bir sayı türünü aşağıdaki gibi kısa gösterimle String’e dönüştürebilirsiniz.

int i = 10;
String str = i + "";

Ya da sadece basit yol

String str = 10 + "";

Bir “InputStream"den “Dize” Alma #

Bir “Dize”, bayt dizisi oluşturucusu kullanılarak bir “InputStream"den okunabilir.

import java.io.*;

public String readString(InputStream input) throws IOException {
    byte[] bytes = new byte[50]; // supply the length of the string in bytes here
    input.read(bytes);
    return new String(bytes);
}

Bu, sistem varsayılan karakter kümesini kullanır, ancak alternatif bir karakter kümesi belirtilebilir:

return new String(bytes, Charset.forName("UTF-8"));

Dizeleri Sayısal Bir Değere Ayrıştırma #

İlkel sayısal türe veya sayısal sarmalayıcı türüne dize:

Her sayısal sarmalayıcı sınıfı, bir “Dize"yi karşılık gelen ilkel türe dönüştüren bir “parseXxx” yöntemi sağlar. Aşağıdaki kod, “Integer.parseInt” yöntemini kullanarak bir “Dize"yi “int"e dönüştürür:

String string = "59";
int primitive = Integer.parseInteger(string);

Bir “Dize"yi sayısal sarmalayıcı sınıfının bir örneğine dönüştürmek için sarmalayıcı sınıflarının aşırı yüklenmesi “valueOf” yöntemini kullanabilirsiniz:

String string = "59";
Integer wrapper = Integer.valueOf(string);

veya otomatik kutulamaya güvenin (Java 5 ve üstü):

String string = "59";
Integer wrapper = Integer.parseInteger(string);  // 'int' result is autoboxed

Yukarıdaki kalıp “byte”, “short”, “int”, “long”, “float” ve “double” ve karşılık gelen sarmalayıcı sınıfları (“Byte”, “Kısa”, “Tamsayı”, “Uzun”, ‘Yüzer’ ve ‘Çift’).

** Radix kullanarak Tamsayıya dize:**

String integerAsString = "0101"; // binary representation
int parseInt = Integer.parseInt(integerAsString,2);
Integer valueOfInteger = Integer.valueOf(integerAsString,2);
System.out.println(valueOfInteger); // prints 5 
System.out.println(parseInt); // prints 5

İstisnalar

Kabul edilebilir bir sayısal temsil olmayan veya bir değeri temsil eden bir dize için sayısal bir “valueOf(String)” veya “parseXxx(…)” yöntemi çağrıldığında, denetlenmeyen NumberFormatException istisnası atılır. menzil dışında.
