Hiçbir şey gerçekten rastgele değildir ve bu nedenle javadoc bu sayıları sözde rastgele olarak adlandırır. Bu sayılar bir sözde rasgele sayı üreteci ile oluşturulur.

Sözde Rastgele Sayılar #

Java, “utils” paketinin bir parçası olarak, uygun şekilde “Random” olarak adlandırılan temel bir sözde rastgele sayı üreteci sağlar. Bu nesne, yerleşik sayısal veri türlerinden (‘int’, ‘float’, vb.) herhangi biri olarak sözde rastgele bir değer oluşturmak için kullanılabilir. Rastgele bir Boole değeri veya rastgele bir bayt dizisi oluşturmak için de kullanabilirsiniz. Örnek bir kullanım aşağıdaki gibidir:

import java.util.Random;  

...
  
Random random = new Random();
int randInt = random.nextInt();
long randLong = random.nextLong();

double randDouble = random.nextDouble(); //This returns a value between 0.0 and 1.0
float randFloat = random.nextFloat(); //Same as nextDouble

byte[] randBytes = new byte[16];
random.nextBytes(randBytes); //nextBytes takes a user-supplied byte array, and fills it with random bytes. It returns nothing.

NOT: Bu sınıf yalnızca oldukça düşük kaliteli sözde rasgele sayılar üretir ve şifreleme işlemleri veya daha yüksek kaliteli rasgeleliğin kritik olduğu diğer durumlar için rasgele sayılar üretmek için asla kullanılmamalıdır (Bunun için ‘SecureRandom’u kullanmak istersiniz). sınıf, aşağıda belirtildiği gibi). “Güvenli” ve “güvensiz” rastgelelik arasındaki ayrım için bir açıklama bu örneğin kapsamı dışındadır.

Belirli Aralıktaki Sözde Rastgele Sayılar #

“Random"ın “nextInt(int bağlı)” yöntemi, bir üst özel sınırı, yani döndürülen rastgele değerin şundan küçük olması gereken bir sayıyı kabul eder. Ancak, yalnızca “nextInt” yöntemi bir sınırı kabul eder; nextLong, nextDouble vb. yok.

Random random = new Random();
random.nextInt(1000); // 0 - 999

int number = 10 + random.nextInt(100); // number is in the range of 10 to 109

Java 1.7’den başlayarak, ThreadLocalRandom (kaynak) öğesini de kullanabilirsiniz. Bu sınıf, iş parçacığı için güvenli bir PRNG (sözde rastgele sayı üreteci) sağlar. Bu sınıfın “nextInt” yönteminin hem üst hem de alt sınırı kabul ettiğini unutmayın.

import java.util.concurrent.ThreadLocalRandom;

// nextInt is normally exclusive of the top value,
// so add 1 to make it inclusive
ThreadLocalRandom.current().nextInt(min, max + 1);

Resmi belgelerde “nextInt(int bağlı)“nın “bağlı” olduğunda garip şeyler yapabileceğini belirttiğine dikkat edin. ` 230+1’e yakın (vurgu eklendi):

Algoritma biraz zor. Eşit olmayan dağılıma neden olacak değerleri reddeder (2^31’in n’ye bölünememesinden dolayı). Bir değerin reddedilme olasılığı n’ye bağlıdır. En kötü durum n=2^30+1’dir, bunun için reddetme olasılığı 1/2’dir ve döngü sona ermeden önce beklenen yineleme sayısı 2’dir.

Diğer bir deyişle, bir sınır belirtmek (biraz) ’nextInt’ yönteminin performansını düşürür ve ‘sınır’ maksimum int değerinin yarısına yaklaştıkça bu performans düşüşü daha belirgin hale gelir.

Kriptografik olarak güvenli sahte rasgele sayılar oluşturma #

“Random” ve “ThreadLocalRandom”, günlük kullanım için yeterince iyidir, ancak büyük bir sorunları vardır: Bunlar, doğrusal uyumlu bir oluşturucuya dayalıdır. çıktı oldukça kolay tahmin edilebilir. Bu nedenle, bu iki sınıf kriptografik kullanımlar (anahtar oluşturma gibi) için uygun değildir.

‘java.security.SecureRandom’, çıktısı tahmin edilmesi çok zor olan bir PRNG’nin gerekli olduğu durumlarda kullanılabilir. Bu sınıfın örnekleri tarafından oluşturulan rastgele sayıları tahmin etmek, sınıfı kriptografik olarak güvenli olarak etiketlemek için yeterince zordur.

import java.security.SecureRandom;
import java.util.Arrays;

public class Foo {
    public static void main(String[] args) {
        SecureRandom rng = new SecureRandom();
        byte[] randomBytes = new byte[64];
        rng.nextBytes(randomBytes); // Fills randomBytes with random bytes (duh)
        System.out.println(Arrays.toString(randomBytes));
    }
}

“SecureRandom”, kriptografik olarak güvenli olmasının yanı sıra, “Random” dönemi olan 248‘a kıyasla 2160 gibi devasa bir süreye sahiptir. “Random"dan ve Mersenne Twister ve [Xorshift](https://en.wikipedia. org/wiki/Xorshift), ancak.

SecureRandom uygulamasının hem platforma hem de sağlayıcıya bağlı olduğunu unutmayın. Varsayılan ‘SecureRandom’ ( ‘sun.security.provider.SecureRandom’ içinde ‘SUN’ sağlayıcısı tarafından verilir):

  • Unix benzeri sistemlerde, “/dev/random” ve/veya “/dev/urandom"dan gelen verilerle tohumlanmış.
  • Windows’ta, CryptoAPI içindeki “CryptGenRandom()” çağrılarıyla tohumlanmış.

Belirli Bir Tohumla Rastgele Sayılar Üretme #

//Creates a Random instance with a seed of 12345.
Random random = new Random(12345L);

//Gets a ThreadLocalRandom instance
ThreadLocalRandom tlr = ThreadLocalRandom.current();

//Set the instance's seed.
tlr.setSeed(12345L);

Rastgele sayılar üretmek için aynı çekirdeği kullanmak, her seferinde aynı sayıları döndürür, bu nedenle, yinelenen sayılarla sonuçlanmak istemiyorsanız, her “Random” örneği için farklı bir tohum ayarlamak iyi bir fikirdir.

Her çağrı için farklı olan bir “Uzun” elde etmenin iyi bir yöntemi System.currentTimeMillis()‘dir:

Random random = new Random(System.currentTimeMillis());
ThreadLocalRandom.current().setSeed(System.currentTimeMillis());

Yinelemeler olmadan rastgele sayılar seçin #

/**
 * returns a array of random numbers with no duplicates
 * @param range the range of possible numbers for ex. if 100 then it can be anywhere from 1-100
 * @param length the length of the array of random numbers
 * @return array of random numbers with no duplicates.
 */
public static int[] getRandomNumbersWithNoDuplicates(int range, int length){
    if (length<range){
        // this is where all the random numbers
        int[] randomNumbers = new int[length];
        
        // loop through all the random numbers to set them
        for (int q = 0; q < randomNumbers.length; q++){
            
            // get the remaining possible numbers
            int remainingNumbers = range-q;
            
            // get a new random number from the remainingNumbers
            int newRandSpot = (int) (Math.random()*remainingNumbers);
            
            newRandSpot++;
            
            // loop through all the possible numbers
            for (int t = 1; t < range+1; t++){
                
                // check to see if this number has already been taken
                boolean taken = false;
                for (int number : randomNumbers){
                    if (t==number){
                        taken = true;
                        break;
                    }
                }
                
                // if it hasnt been taken then remove one from the spots
                if (!taken){
                    newRandSpot--;
                    
                    // if we have gone though all the spots then set the value
                    if (newRandSpot==0){
                        randomNumbers[q] = t;
                    }
                }
            }
        }
        return randomNumbers;
    } else {
        // invalid can't have a length larger then the range of possible numbers
    }
    return null;
}

Yöntem, istenen uzunluğun boyutuna sahip olan ve olası sayıların kalan uzunluğunu bulan bir diziyi döngüye sokarak çalışır. Bu olası sayılardan ’newRandSpot’ rasgele bir sayısını ayarlar ve bu sayıyı kalan alınmamış sayı içinde bulur. Bunu, aralıkta dolaşarak ve bu numaranın daha önce alınmış olup olmadığını kontrol ederek yapar.

Örneğin, aralık 5 ve uzunluk 3 ise ve biz zaten 2 sayısını seçtik. O zaman geriye kalan 4 sayı var, bu nedenle 1 ile 4 arasında rastgele bir sayı elde ederiz ve herhangi bir sayıyı atlayarak aralık (5) boyunca döngü yaparız. daha önce kullandığımız (2).

Şimdi 1 ve 4 arasında seçilen bir sonraki sayının 3 olduğunu söyleyelim. İlk döngüde henüz alınmamış 1 elde ederiz, böylece 3’ten 1’i çıkararak 2’yi yapabiliriz. Şimdi ikinci döngüde alınmış olan 2’yi elde ederiz. bu yüzden hiçbir şey yapmıyoruz. 4’e ulaşana kadar bu kalıbı takip ediyoruz, burada 1’i kaldırdığımızda 0 oluyor, böylece yeni randomNumber’ı 4’e ayarladık.

apache-common lang3 kullanarak Rastgele sayı oluşturma #

Tek bir satır kullanarak rastgele sayılar üretmek için org.Apache.commons.lang3.RandomUtils’i kullanabiliriz.

int x = RandomUtils.nextInt(1, 1000);

nextInt(int startInclusive, int endExclusive) yöntemi bir aralık alır.

int dışında, bu sınıfı kullanarak rastgele ’long’, ‘double’, ‘float’ ve ‘bytes’ üretebiliriz.

static float nextFloat() //Returns a random float within 0 - Float.MAX_VALUE
