Bir list, sıralı bir değerler topluluğudur. Java’da listeler, Java Collections Framework‘in bir parçasıdır. Listeler, java.util.Collection öğesini genişleten java.util.List arabirimini uygular.

Sözdizimi #

  • ls.add(E öğesi); //Bir eleman ekler
  • ls.remove(E öğesi); //Bir öğeyi kaldırır
  • for(E element : ls){} //Her element üzerinde yinelenir
  • ls.toArray(yeni Dize[ls.uzunluk]); // Bir Dize Listesini bir Dize dizisine dönüştürür
  • ls.get(int dizini); //Belirtilen dizindeki öğeyi döndürür.
  • ls.set(int dizini, E öğesi); //Belirtilen bir konumdaki elemanı değiştirir.
  • ls.isEmpty(); //Dizi eleman içermiyorsa true, aksi halde false döndürür.
  • ls.indexOf(Nesne o); //Belirtilen o öğesinin ilk konumunun dizinini döndürür veya mevcut değilse -1 değerini döndürür.
  • ls.lastIndexOf(Nesne o); //Belirtilen o öğesinin son konumunun dizinini döndürür veya mevcut değilse -1 değerini döndürür.
  • ls.size(); //Listedeki eleman sayısını döndürür.

list, sıralı bir değerler koleksiyonunu depolayan bir nesnedir. “Sıralı”, değerlerin belirli bir sırada saklandığı anlamına gelir; bir öğe önce gelir, biri ikinci sırada gelir, vb. Bireysel değerlere genellikle “elemanlar” denir. Java listeleri genellikle şu özellikleri sağlar:

  • Listeler sıfır veya daha fazla eleman içerebilir.
  • Listeler yinelenen değerler içerebilir. Başka bir deyişle, bir öğe bir listeye birden fazla kez eklenebilir.
  • Listeler, öğelerini belirli bir sırayla saklar, yani bir öğe önce gelir, diğeri gelir, vb.
  • Her öğenin, liste içindeki konumunu gösteren bir index‘i vardır. İlk elemanın indeksi 0, sonrakinin indeksi 1 vb.
  • Listeler, listenin başına, sonuna veya herhangi bir dizine eleman eklenmesine izin verir.
  • Bir listenin belirli bir değer içerip içermediğinin test edilmesi, genellikle listedeki her bir öğenin incelenmesi anlamına gelir. Bu, bu kontrolü gerçekleştirme zamanının, listenin boyutuyla orantılı olarak O(n) olduğu anlamına gelir.

Bir listeye sondan başka bir noktada değer eklemek, aşağıdaki öğelerin tümünü “aşağı” veya “sağa” kaydıracaktır. Başka bir deyişle, n dizinine bir öğe eklemek, daha önce n dizininde olan öğeyi n+1 dizinine taşır ve bu böyle devam eder. Örneğin:

List<String> list = new ArrayList<>();
list.add("world");
System.out.println(list.indexOf("world"));      // Prints "0"
// Inserting a new value at index 0 moves "world" to index 1
list.add(0, "Hello");
System.out.println(list.indexOf("world"));      // Prints "1"
System.out.println(list.indexOf("Hello"));      // Prints "0"

Genel bir listeyi sıralama #

“Koleksiyonlar” sınıfı, bir listeyi sıralamak için iki standart statik yöntem sunar:

  • sort(List<T> list), T'nin Karşılaştırılabilir'i genişlettiği listelere uygulanabilir<? süper T> ve
  • sort(List<T> list, Comparator<?super T> c) her tür listeye uygulanabilir.

İlkini uygulamak, her zaman mümkün olmayan, sıralanan liste öğelerinin sınıfını değiştirmeyi gerektirir. Ayrıca, varsayılan sıralamayı sağlamasına rağmen, farklı durumlarda başka sıralama düzenleri gerekli olabileceğinden veya sıralama yalnızca bir defalık bir görev olduğundan istenmeyen bir durum olabilir.

Aşağıdaki sınıfın örnekleri olan nesneleri sıralama görevimiz olduğunu düşünün:

public class User {
    public final Long id;
    public final String username;

    public User(Long id, String username) {
        this.id = id;
        this.username = username;
    }

    @Override
    public String toString() {
        return String.format("%s:%d", username, id);
    }
}

‘Collections.sort(List list)’ kullanmak için, ‘Comparable’ arayüzünü uygulamak için ‘User’ sınıfını değiştirmemiz gerekiyor. Örneğin

public class User implements Comparable<User> {
    public final Long id;
    public final String username;

    public User(Long id, String username) {
        this.id = id;
        this.username = username;
    }

    @Override
    public String toString() {
        return String.format("%s:%d", username, id);
    }

    @Override
    /** The natural ordering for 'User' objects is by the 'id' field. */
    public int compareTo(User o) {
        return id.compareTo(o.id);
    }
}

(Bir yana: ‘String’, ‘Long’, ‘Integer’ gibi birçok standart Java sınıfı, ‘Comparable’ arabirimini uygular. Bu, bu öğelerin listelerini varsayılan olarak sıralanabilir hale getirir ve diğerlerinde ‘compare’ veya ‘compareTo’ uygulamasını basitleştirir. sınıflar.)

Yukarıdaki değişiklikle, doğal sıralama sınıflarına göre ‘Kullanıcı’ nesnelerinin bir listesini kolayca sıralayabiliriz. (Bu durumda, ‘id’ değerlerine göre sıralanmasını tanımladık). Örneğin:

List<User> users = Lists.newArrayList(
    new User(33L, "A"),
    new User(25L, "B"),
    new User(28L, ""));
Collections.sort(users);

System.out.print(users);
// [B:25, C:28, A:33]

Ancak, “Kullanıcı” nesnelerini “id” yerine “ad"a göre sıralamak istediğimizi varsayalım. Alternatif olarak, sınıfı ‘Karşılaştırılabilir’ uygulamak için değiştiremediğimizi varsayalım.

“Karşılaştırıcı” argümanıyla “sort” yönteminin yararlı olduğu yer burasıdır:

Collections.sort(users, new Comparator<User>() {
    @Override
    /* Order two 'User' objects based on their names. */
    public int compare(User left, User right) {
        return left.username.compareTo(right.username);
    }
});
System.out.print(users);
// [A:33, B:25, C:28]

Java 8’de anonim sınıf yerine lambda kullanabilirsiniz. İkincisi tek bir astara indirgenir:

Collections.sort(users, (l, r) -> l.username.compareTo(r.username));

Ayrıca, Java 8, ‘Liste’ arayüzüne, sıralamayı daha da basitleştiren varsayılan bir ‘sort’ yöntemi ekler.

users.sort((l, r) -> l.username.compareTo(r.username))
son sürüm

2 liste arasında ortak öğeleri bulma #

Diyelim ki iki listeniz var: A ve B ve her iki listede de bulunan öğeleri bulmanız gerekiyor.

Bunu sadece List.retainAll() yöntemini çağırarak yapabilirsiniz.

Örnek:

public static void main(String[] args) {
    List<Integer> numbersA = new ArrayList<>();
    List<Integer> numbersB = new ArrayList<>();
    numbersA.addAll(Arrays.asList(new Integer[] { 1, 3, 4, 7, 5, 2 }));
    numbersB.addAll(Arrays.asList(new Integer[] { 13, 32, 533, 3, 4, 2 }));

    System.out.println("A: " + numbersA);
    System.out.println("B: " + numbersB);
    List<Integer> numbersC = new ArrayList<>();
    numbersC.addAll(numbersA);
    numbersC.retainAll(numbersB);

    System.out.println("List A : " + numbersA);
    System.out.println("List B : " + numbersB);
    System.out.println("Common elements between A and B: " + numbersC);

}

Liste Oluşturma #

Listenize bir tür vermek

Bir liste oluşturmak için bir türe ihtiyacınız vardır (herhangi bir sınıf, örneğin String). Bu, ‘Listenizin’ türüdür. ‘Liste’ yalnızca belirtilen türdeki nesneleri depolayacaktır. Örneğin:

List<String> strings;

"string1", "merhaba dünya!", "hoşçakal", vb. depolayabilir, ancak “9.2"yi depolayamaz:

List<Double> doubles;

“9.2"yi depolayabilir, ancak “merhaba dünya!"‘yı depolayamaz.

Listenizi başlatma

Yukarıdaki listelere bir şey eklemeye çalışırsanız, bir NullPointerException elde edersiniz, çünkü “strings” ve “double” her ikisi de null‘a eşittir!

Bir listeyi başlatmanın iki yolu vardır:

Seçenek 1: Listeyi uygulayan bir sınıf kullanın

‘Liste’ bir arabirimdir, yani bir yapıcıya sahip değildir, bunun yerine bir sınıfın geçersiz kılması gereken yöntemlerdir. “ArrayList” en sık kullanılan “Liste"dir, ancak “LinkedList” de yaygındır. Böylece listemizi şu şekilde başlatıyoruz:

List<String> strings = new ArrayList<String>();

veya

List<String> strings = new LinkedList<String>();

Java SE 7’den başlayarak, bir diamond operatörü kullanabilirsiniz:

List<String> strings = new ArrayList<>();

veya

List<String> strings = new LinkedList<>();
son sürüm

Seçenek 2: Koleksiyonlar sınıfını kullanın

‘Collections’ sınıfı, ‘List’ değişkeni olmadan Listeler oluşturmak için iki kullanışlı yöntem sağlar:

  • emptyList(): boş bir liste döndürür.
  • singletonList(T): T türünde bir liste oluşturur ve belirtilen öğeyi ekler.

Ve verileri doldurmak için mevcut bir “Liste” kullanan bir yöntem:

  • addAll(L, T...): belirtilen tüm öğeleri ilk parametre olarak geçirilen listeye ekler.

Örnekler:

import java.util.List;
import java.util.Collections;

List<Integer> l = Collections.emptyList();
List<Integer> l1 = Collections.singletonList(42);
Collections.addAll(l1, 1, 2, 3);

Konumsal Erişim İşlemleri #

List API, konumsal erişim işlemleri için sekiz yönteme sahiptir:

  • ekle(T tipi)
  • add(int indeksi, T tipi)
  • kaldır(nesne o)
  • kaldır(int indeksi)
  • get(int indeksi)
  • set(int indeksi, E elementi)
  • int indexOf(Object o)
  • int lastIndexOf(Object o)

Öyleyse, bir Listemiz varsa:

List<String> strings = new ArrayList<String>();

Ve “Merhaba dünya!” dizelerini eklemek istedik. ve “Hoşçakal dünya!” ona göre şöyle yapardık:

strings.add("Hello world!");
strings.add("Goodbye world!");

Ve listemiz iki unsuru içerecektir. Şimdi “Program başlıyor!” eklemek istediğimizi varsayalım. listenin önünde. Bunu şöyle yapardık:

strings.add(0, "Program starting!");

NOT: İlk öğe 0’dır.

Şimdi, “Güle güle dünya!” kaldırmak isteseydik. satır, şöyle yapabiliriz:

strings.remove("Goodbye world!");

Ve eğer ilk satırı kaldırmak isteseydik (ki bu durumda “Program başlıyor!” olurdu), bunu şu şekilde yapabiliriz:

strings.remove(0);

Not:

  1. Liste öğelerinin eklenmesi ve çıkarılması listeyi değiştirir ve bu, liste aynı anda yineleniyorsa bir ConcurrentModificationExceptiona yol açabilir.

  2. Eleman ekleme ve çıkarma, liste sınıfına, kullanılan yönteme ve bir elemanın başında mı, sonunda mı yoksa sonunda mı eklediğinize/çıkardığınıza bağlı olarak ‘O(1)’ veya ‘O(N)’ olabilir. listenin ortasında.

Listedeki bir öğeyi belirli bir konumda almak için List API’sinin E get(int index); yöntemini kullanabilirsiniz. Örneğin:

strings.get(0);

listenin ilk öğesini döndürür.

set(int index, E element); öğesini kullanarak herhangi bir öğeyi belirtilen bir konumda değiştirebilirsiniz. Örneğin:

strings.set(0,"This is a replacement");

Bu, “Bu bir değiştirmedir” Dizesini listenin ilk öğesi olarak ayarlayacaktır.

Not: The set method will overwrite the element at the position 0. It will not add the new String at the position 0 and push the old one to the position 1.

int indexOf(Object o);, argüman olarak iletilen nesnenin ilk oluşumunun konumunu döndürür. Listede nesnenin hiçbir oluşumu yoksa, -1 değeri döndürülür. Bir önceki örneğin devamında aşağıdakileri ararsanız:

strings.indexOf("This is a replacement")

Listemizin 0 konumunda “Bu bir değiştirmedir” Dizesini ayarladığımızda 0’ın döndürülmesi bekleniyor. Listede birden fazla oluşum olması durumunda int indexOf(Object o); çağrıldığında belirtildiği gibi ilk oluşumun indeksi döndürülecektir. int lastIndexOf(Object o) öğesini çağırarak listedeki son oluşumun dizinini alabilirsiniz. Yani başka bir “Bu bir yedek” eklersek:

strings.add("This is a replacement");
strings.lastIndexOf("This is a replacement");

Bu sefer 0 değil 1 döndürülecek;

ArrayList’ten eleman oluşturma, ekleme ve çıkarma #

“ArrayList”, Java’daki yerleşik veri yapılarından biridir. Öğeleri (Nesneler) depolamak için dinamik bir dizidir (önce veri yapısının boyutunun bildirilmesi gerekmemektedir).

‘AbstractList’ sınıfını genişletir ve ‘List’ arabirimini uygular. Bir “ArrayList”, ekleme sırasını koruduğu yerde yinelenen öğeler içerebilir. ‘ArrayList’ sınıfının senkronize edilmediğine dikkat edilmelidir, bu nedenle ‘ArrayList’ ile eşzamanlılık işlenirken dikkatli olunmalıdır. ArrayList rasgele erişime izin verir çünkü dizi indeks bazında çalışır. Dizi listesinden bir öğe kaldırıldığında sıklıkla meydana gelen kaydırma nedeniyle “ArrayList"te manipülasyon yavaştır.

Bir ‘ArrayList’ aşağıdaki gibi oluşturulabilir:

List<T> myArrayList = new ArrayList<>();

Burada ‘T’ ( https://www.wikiod.com/tr/java/jenerik ) ‘ArrayList’ içinde depolanacak türdür.

‘ArrayList’ türü herhangi bir Nesne olabilir. Tür, ilkel bir tür olamaz (bunun yerine wrapper sınıflarını kullanın).

‘ArrayList’e bir öğe eklemek için ‘add()’ yöntemini kullanın:

myArrayList.add(element);

Veya belirli bir dizine öğe eklemek için:

myArrayList.add(index, element); //index of the element should be an int (starting from 0)

‘ArrayList’ten bir öğeyi kaldırmak için ‘remove()’ yöntemini kullanın:

myArrayList.remove(element);

Veya belirli bir dizinden bir öğeyi kaldırmak için:

myArrayList.remove(index); //index of the element should be an int (starting from 0)

Liste öğesinin yerinde değiştirilmesi #

Bu örnek, değiştirilen öğenin değiştirilen öğeyle aynı konumda olmasını sağlarken bir “Liste” öğesinin değiştirilmesiyle ilgilidir.

Bu, şu yöntemler kullanılarak yapılabilir:

  • set(int indeksi, T tipi)
  • int indexOf(T tipi)

“Program başlıyor!”, “Merhaba dünya!” öğelerini içeren bir “ArrayList” düşünün. ve “Hoşçakal dünya!”

List<String> strings = new ArrayList<String>();
strings.add("Program starting!");
strings.add("Hello world!");
strings.add("Goodbye world!");

Değiştirmek istediğimiz elemanın indeksini biliyorsak, basitçe aşağıdaki gibi set kullanabiliriz:

strings.set(1, "Hi world");

Dizini bilmiyorsak, önce onu arayabiliriz. Örneğin:

int pos = strings.indexOf("Goodbye world!");
if (pos >= 0) {
    strings.set(pos, "Goodbye cruel world!");
}

Notlar:

  1. ‘set’ işlemi bir ‘ConcurrentModificationException’a neden olmaz.
  2. ‘set’ işlemi, ‘ArrayList’ için hızlıdır ( ‘O(1)’ ), ancak ‘LinkedList’ için yavaştır ( ‘O(N)’).
  3. Bir ‘ArrayList’ veya ‘LinkedList’ üzerinde ‘indexOf’ araması yavaştır ( ‘O(N)’ ).

Listeyi değiştirilemez hale getirme #

Collections sınıfı, bir listeyi değiştirilemez hale getirmenin bir yolunu sağlar:

List<String> ls = new ArrayList<String>();
List<String> unmodifiableList = Collections.unmodifiableList(ls);

Tek öğeli değiştirilemez bir liste istiyorsanız, şunları kullanabilirsiniz:

List<String> unmodifiableList = Collections.singletonList("Only string in the list");

Listedeki nesneleri hareket ettirme #

Collections sınıfı, çeşitli yöntemler kullanarak nesneleri listede hareket ettirmenize olanak tanır (ls, List’tir):

Listeyi tersine çevirme:

Collections.reverse(ls);

Listedeki öğelerin dönen konumları

Döndürme yöntemi bir tamsayı argümanı gerektirir. Bu, onu çizgi boyunca hareket ettirmek için kaç noktadır. Bunun bir örneği aşağıdadır:

List<String> ls = new ArrayList<String>();
ls.add(" how");
ls.add(" are");
ls.add(" you?");
ls.add("hello,");
Collections.rotate(ls, 1);

for(String line : ls) System.out.print(line);
System.out.println();

Bu “merhaba, nasılsın?” yazacaktır.

Bir listedeki öğeleri karıştırma

Yukarıdaki aynı listeyi kullanarak, bir listedeki öğeleri karıştırabiliriz:

Collections.shuffle(ls);

Nesneleri noktalara rastgele yerleştirmek için kullandığı bir java.util.Random nesnesi de verebiliriz:

Random random = new Random(12); 
Collections.shuffle(ls, random);
