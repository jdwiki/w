Örneklerle süper anahtar kelime kullanımı #

süper anahtar kelime üç yerde önemli bir rol oynar

  1. Yapıcı Seviyesi
  2. Yöntem Düzeyi
  3. Değişken Seviye

Yapıcı Seviyesi #

‘super’ anahtar sözcüğü, üst sınıf yapıcısını çağırmak için kullanılır. Bu kurucu, varsayılan kurucu veya parametreli kurucu olabilir.

  • Varsayılan kurucu: super();

  • Parametreli kurucu: super(int no, double miktar, String name);

      class Parentclass
  {
     Parentclass(){
        System.out.println("Constructor of Superclass");
     }
  }
  class Subclass extends Parentclass
  {
     Subclass(){
      /* Compile adds super() here at the first line
       * of this constructor implicitly
       */
      System.out.println("Constructor of Subclass");
     }
     Subclass(int n1){
      /* Compile adds super() here at the first line
       * of this constructor implicitly
       */
      System.out.println("Constructor with arg");
     }
     void display(){
      System.out.println("Hello");
     }
     public static void main(String args[]){
      // Creating object using default constructor
      Subclass obj= new Subclass();
      //Calling sub class method 
         obj.display();
         //Creating object 2 using arg constructor
         Subclass obj2= new Subclass(10);
         obj2.display();
    }
  }

Not: super() yapıcıdaki ilk ifade olmalıdır, aksi takdirde derleme hata mesajını alırız.

Yöntem Düzeyi #

‘super’ anahtar sözcüğü, yöntemin geçersiz kılınması durumunda da kullanılabilir. ‘super’ anahtar sözcüğü, üst sınıf yöntemini çağırmak veya çağırmak için kullanılabilir.

class Parentclass
{
   //Overridden method
   void display(){
    System.out.println("Parent class method");
   }
}
class Subclass extends Parentclass
{
   //Overriding method
   void display(){
    System.out.println("Child class method");
   }
   void printMsg(){
    //This would call Overriding method
    display();
    //This would call Overridden method
    super.display();
   }
   public static void main(String args[]){        
    Subclass obj= new Subclass();
    obj.printMsg(); 
   }
}

Not:Yöntem geçersiz kılma yoksa, üst sınıf yöntemini çağırmak için ‘super’ anahtar sözcüğünü kullanmamız gerekmez.

Değişken Düzey #

“super”, hemen üst sınıf örnek değişkenini belirtmek için kullanılır. Kalıtım durumunda, temel sınıf olasılığı olabilir ve türetilmiş sınıf benzer veri üyelerine sahip olabilir. Temel/ana sınıfın veri üyesi ile türetilmiş/alt sınıfın veri üyesini ayırt etmek için, türetilmiş sınıf bağlamında temel sınıf verileri üyelerden önce ‘super’ anahtar kelimesi gelmelidir.

//Parent class or Superclass
class Parentclass
{
    int num=100;
}
//Child class or subclass
class Subclass extends Parentclass
{
    /* I am declaring the same variable 
     * num in child class too.
     */
    int num=110;
    void printNumber(){
     System.out.println(num); //It will print value 110
     System.out.println(super.num); //It will print value 100
    }
    public static void main(String args[]){
       Subclass obj= new Subclass();
       obj.printNumber();    
    }
}

Not: Temel sınıf veri üyesi adından önce ‘super’ anahtar sözcüğünü yazmıyorsak, o zaman mevcut sınıf veri üyesi olarak anılacaktır ve temel sınıf veri üyesi türetilmiş sınıf bağlamında gizlenir.
