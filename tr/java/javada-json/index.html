JSON (JavaScript Object Notation), insanlar ve makineler için okuması ve yazması kolay, hafif, metin tabanlı, dilden bağımsız bir veri alışverişi formatıdır. JSON, iki yapılandırılmış türü temsil edebilir: nesneler ve diziler. JSON genellikle Ajax uygulamalarında, konfigürasyonlarda, veritabanlarında ve RESTful web servislerinde kullanılır. JSON İşleme için Java API’si, JSON’u ayrıştırmak, oluşturmak, dönüştürmek ve sorgulamak için taşınabilir API’ler sağlar.

Bu örnek, Google Gson kitaplığı, Jackson Object Mapper ve diğerleri gibi çeşitli kitaplıkları kullanarak Java’da JSON’u ayrıştırmaya ve oluşturmaya odaklanır.

Diğer kitaplıkları kullanan örnekler burada bulunabilir: Java’da JSON nasıl ayrıştırılır

Jackson Nesne Eşleştiricisini Kullanma #

Pojo Modeli

public class Model {
    private String firstName;
    private String lastName;
    private int age;
    /* Getters and setters not shown for brevity */        
}

Örnek: Nesneye Dize

Model outputObject = objectMapper.readValue(
     "{\"firstName\":\"John\",\"lastName\":\"Doe\",\"age\":23}",
     Model.class);
System.out.println(outputObject.getFirstName());
//result: John

Örnek: String’e Nesne

String jsonString = objectMapper.writeValueAsString(inputObject));
//result: {"firstName":"John","lastName":"Doe","age":23}

Detaylar #

İthalat beyanı gerekli:

import com.fasterxml.jackson.databind.ObjectMapper;

Maven bağımlılığı: jackson-databind

‘ObjectMapper’ örneği #

//creating one
ObjectMapper objectMapper = new ObjectMapper();
  • ‘ObjectMapper’ iş parçacığı güvenlidir
  • önerilir: paylaşılan, statik bir örneğe sahip olun

Seri durumdan çıkarma: #

<T> T readValue(String content, Class<T> valueType)
  • “valueType” belirtilmelidir – dönüş bu türden olacaktır
  • atar
  • ‘IOException’ - düşük seviyeli bir G/Ç sorunu olması durumunda
  • JsonParseException - temel girdi geçersiz içerik içeriyorsa
  • JsonMappingException - giriş JSON yapısı nesne yapısıyla eşleşmiyorsa

Kullanım örneği (jsonString giriş dizesidir):

Model fromJson = objectMapper.readValue(jsonString, Model.class);

Serileştirme yöntemi: #

Dize writeValueAsString(Nesne değeri)

  • atar
  • Bir hata durumunda “JsonProcessingException”
  • Not: 2.1 sürümünden önce, IOException’ı içeren throws yan tümcesi; 2.1 kaldırdı.

Nesneye JSON (Gson Kitaplığı) #

Diyelim ki sadece ‘isim’ ile ‘Kişi’ adında bir sınıfınız var.

private class Person {
    public String name;

    public Person(String name) {
        this.name = name;
    }
}

Kod:

Gson gson = new Gson();
String json = "{\"name\": \"John\"}";

Person person = gson.fromJson(json, Person.class);
System.out.println(person.name); //John

Sınıf yolunuzda gson library olmalıdır.

Verileri JSON olarak kodlama #

Bir “JSONObject” oluşturmanız ve içine veri koymanız gerekiyorsa, aşağıdaki örneği göz önünde bulundurun:

// Create a new javax.json.JSONObject instance.
JSONObject first = new JSONObject();

first.put("foo", "bar");
first.put("temperature", 21.5);
first.put("year", 2016);

// Add a second object.
JSONObject second = new JSONObject();
second.put("Hello", "world");
first.put("message", second);

// Create a new JSONArray with some values
JSONArray someMonths = new JSONArray(new String[] { "January", "February" });
someMonths.put("March");
// Add another month as the fifth element, leaving the 4th element unset.
someMonths.put(4, "May");

// Add the array to our object
object.put("months", someMonths);

// Encode
String json = object.toString();

// An exercise for the reader: Add pretty-printing!
/* {
       "foo":"bar",
       "temperature":21.5,
       "year":2016,
       "message":{"Hello":"world"},
       "months":["January","February","March",null,"May"]
   }
*/

JSON verilerinin kodunu çözme #

Bir “JSONObject"den veri almanız gerekiyorsa, aşağıdaki örneği göz önünde bulundurun:

String json = "{\"foo\":\"bar\",\"temperature\":21.5,\"year\":2016,\"message\":{\"Hello\":\"world\"},\"months\":[\"January\",\"February\",\"March\",null,\"May\"]}";

// Decode the JSON-encoded string
JSONObject object = new JSONObject(json);

// Retrieve some values
String foo = object.getString("foo");
double temperature = object.getDouble("temperature");
int year = object.getInt("year");

// Retrieve another object
JSONObject secondary = object.getJSONObject("message");
String world = secondary.getString("Hello");

// Retrieve an array
JSONArray someMonths = object.getJSONArray("months");
// Get some values from the array
int nMonths = someMonths.length();
String february = someMonths.getString(1);

Tek öğeyi JSON’dan çıkarın #

String json = "{\"name\": \"John\", \"age\":21}";

JsonObject jsonObject = new JsonParser().parse(json).getAsJsonObject();

System.out.println(jsonObject.get("name").getAsString()); //John
System.out.println(jsonObject.get("age").getAsInt()); //21

JsonArray’den Java Listesine (Gson Kitaplığı) #

Java ArrayListe dönüştürmek istediğiniz basit bir JsonArray:

{
    "list": [
                "Test_String_1",
                "Test_String_2"
            ] 
}

Şimdi “JsonArray” “listesini”, karşılık gelen Java “ArrayList” döndüren aşağıdaki yönteme iletin:

public ArrayList<String> getListString(String jsonList){
    Type listType = new TypeToken<List<String>>() {}.getType();
    //make sure the name 'list' matches the name of 'JsonArray' in your 'Json'.
    ArrayList<String> list = new Gson().fromJson(jsonList, listType);    
    return list;
}

POM.xml dosyanıza aşağıdaki maven bağımlılığını eklemelisiniz:

<!-- https://mvnrepository.com/artifact/com.google.code.gson/gson -->
<dependency>
    <groupId>com.google.code.gson</groupId>
    <artifactId>gson</artifactId>
    <version>2.7</version>
</dependency>

Veya sınıf yolunuzda com.google.code.gson:gson:jar:<version> jar’ına sahip olmalısınız.

Jackson kullanarak JSON koleksiyonunu Nesnelerin koleksiyonuna seri hale getirin #

Bir pojo sınıfınız olduğunu varsayalım ‘Kişi’

public class Person {
    public String name;

    public Person(String name) {
        this.name = name;
    }
}

Ve onu bir JSON dizisine veya Person nesnelerinin bir haritasına ayrıştırmak istiyorsunuz. Tür silme nedeniyle, çalışma zamanında doğrudan ‘List’ ve ‘Map<String, Person>’ sınıflarını oluşturamazsınız (ve dolayısıyla bunları JSON’u seri durumdan çıkarmak için kullanabilirsiniz)_. Bu sınırlamanın üstesinden gelmek için jackson iki yaklaşım sunar - ‘TypeFactory’ ve ‘TypeReference’.

TürFabrika

Burada benimsenen yaklaşım, sizin için türünüzü oluşturmak için bir fabrikayı (ve onun statik yardımcı işlevini) kullanmaktır. Aldığı parametreler kullanmak istediğiniz koleksiyon (liste, set vb.) ve o koleksiyonda saklamak istediğiniz sınıftır.

Tür Referansı

Tip referansı yaklaşımı daha basit görünüyor çünkü sizi biraz yazmaktan kurtarıyor ve daha temiz görünüyor. TypeReference, istediğiniz tür Liste<Kişi> ilettiğiniz bir tür parametresini kabul eder. Bu TypeReference nesnesini başlatmanız ve onu tür kapsayıcınız olarak kullanmanız yeterlidir.

Şimdi JSON’unuzu bir Java nesnesine nasıl seri hale getireceğinize bakalım. JSON’unuz bir dizi olarak biçimlendirilmişse, onu bir Liste olarak seri durumdan çıkarabilirsiniz. Daha karmaşık bir iç içe yapı varsa, bir Haritaya seri durumdan çıkarmak isteyeceksiniz. Her ikisinin de örneklerine bakacağız.

JSON dizisinin seri durumdan çıkarılması #

String jsonString = "[{\"name\": \"Alice\"}, {\"name\": \"Bob\"}]"

TypeFabrika yaklaşımı #

CollectionType listType = 
    factory.constructCollectionType(List.class, Person.class);
List<Preson> list = mapper.readValue(jsonString, listType);

##TypeReferans yaklaşımı

TypeReference<Person> listType = new TypeReference<List<Person>>() {};
List<Person> list = mapper.readValue(jsonString, listType);

JSON haritasının seri durumdan çıkarılması #

String jsonString = "{\"0\": {\"name\": \"Alice\"}, \"1\": {\"name\": \"Bob\"}}"

TypeFabrika yaklaşımı #

CollectionType mapType = 
    factory.constructMapLikeType(Map.class, String.class, Person.class);
List<Person> list = mapper.readValue(jsonString, mapType);

##TypeReferans yaklaşımı

TypeReference<Person> mapType = new TypeReference<Map<String, Person>>() {};
Map<String, Person> list = mapper.readValue(jsonString, mapType);

Detaylar #

Kullanılan içe aktarma ifadesi:

import com.fasterxml.jackson.core.type.TypeReference;
import com.fasterxml.jackson.databind.ObjectMapper;
import com.fasterxml.jackson.databind.type.CollectionType;

Kullanılan örnekler:

ObjectMapper mapper = new ObjectMapper();
TypeFactory factory = mapper.getTypeFactory();

Not #

‘TypeReference’ yaklaşımı daha iyi görünse de birkaç dezavantajı vardır:

  1. ‘TypeReference’ anonim sınıf kullanılarak başlatılmalıdır
  2. Genel bir açıklık sağlamalısınız

Bunu yapmamak, seri durumdan çıkarma hatasına yol açacak genel tür bağımsız değişkeninin kaybına neden olabilir.
