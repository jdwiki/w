  • Deneyin … Yakalayın … Sonunda
  • İç içe ara / devam

Tüm kontrol yapıları, aksi belirtilmedikçe, blok ifadelerinden yararlanır. Bunlar küme parantezleri {} ile gösterilir.

Bu, kaşlı ayraçlar * gerektirmeyen* normal ifadelerden farklıdır, ancak aynı zamanda Yalnızca önceki ifadeyi hemen takip eden satırın dikkate alınacağı konusunda sert bir uyarı ile gelin.

Bu nedenle, bu kontrol yapılarından herhangi birini kaşlı ayraçlar olmadan yazmak tamamen geçerlidir, yalnızca bir ifade başlangıcı takip ettiği sürece, ancak buggy uygulamalarına yol açabileceğinden kesinlikle önerilmez veya bozuk kod.

Örnek:

// valid, but discouraged
Scanner scan = new Scanner(System.in);
int val = scan.nextInt();
if(val % 2 == 0)
    System.out.println("Val was even!");


// invalid; will not compile
// note the misleading indentation here
for(int i = 0; i < 10; i++)

do…while Döngüsü #

“do…while” döngüsü, en az bir kez yürütülmesinin garanti edilmesiyle diğer döngülerden farklıdır. Koşullu ifade ana döngü gövdesinden sonra yapıldığından “test sonrası döngü” yapısı olarak da adlandırılır.

int i = 0;
do {
    i++;
    System.out.println(i);
} while (i < 100); // Condition gets checked AFTER the content of the loop executes.

Bu örnekte, döngü “100” sayısı yazdırılana kadar çalışacaktır (koşul “i < 100” olmasa ve “i <= 100” olmasa da), çünkü döngü koşulu, döngü yürütüldükten sonra değerlendirilir.

En az bir yürütme garantisi ile, değişkenleri döngü dışında bildirmek ve bunları içeride başlatmak mümkündür.

String theWord;
Scanner scan = new Scanner(System.in);
do {
    theWord = scan.nextLine();
} while (!theWord.equals("Bird"));

System.out.println(theWord);

Bu bağlamda, ‘TheWord’ döngünün dışında tanımlanır, ancak doğal akışına dayalı bir değere sahip olması garanti edildiğinden ’theWord’ başlatılacaktır.

Her biri için #

Java 5 ve sonraki sürümlerde, gelişmiş for döngüleri olarak da bilinen for-each döngüleri kullanılabilir:

List strings = new ArrayList();
        
strings.add("Bu");
strings.add("is");
strings.add("her döngü için bir döngü");
        
for (Dize dizesi: dizeler) {
    System.out.println(string);
}

Her döngü Arrays ve ‘Iterable’ arabiriminin uygulamaları üzerinde yineleme yapmak için kullanılabilir, daha sonra ‘List’ veya ‘Set’ gibi Collections sınıflarını içerir.

Döngü değişkeni, kaynak türünden atanabilen herhangi bir türde olabilir.

“Yinelenebilir” veya “T[]” için geliştirilmiş bir for döngüsü için döngü değişkeni, aşağıdaki durumlarda “S” türünde olabilir:

  • `T, S’yi uzatır’
  • hem “T” hem de “S” ilkel türlerdir ve döküm olmadan atanabilir
  • “S” ilkel bir türdür ve “T”, kutudan çıkarma dönüşümünden sonra “S"ye atanabilir bir türe dönüştürülebilir.
  • T ilkel bir türdür ve otomatik kutulama dönüştürmesiyle Sye dönüştürülebilir.

Örnekler:

T elements = ...
for (S s : elements) {
}
T S Derler
int[] uzun evet
uzun[] int hayır
Yinelenebilir<Byte> uzun evet
Yinelenebilir<Dize> KarakterSırası evet
Yinelenebilir<CharSequence> dize hayır
int[] Uzun hayır
int[] Tamsayı evet

Anahtar deyimi #

“switch” ifadesi, Java’nın çok yönlü dal ifadesidir. Uzun if-else if-else zincirlerinin yerini almak ve onları daha okunaklı kılmak için kullanılır. Ancak, ‘if’ deyimlerinden farklı olarak, eşitsizlikler kullanılamaz; her değer somut olarak tanımlanmalıdır.

“switch” ifadesinin üç kritik bileşeni vardır:

  • ‘case’: Bu, ‘switch’ deyiminin argümanıyla denklik için değerlendirilen değerdir.
  • “varsayılan”: Bu, isteğe bağlı, tümünü yakalayan bir ifadedir ve “vaka” ifadelerinin hiçbiri “doğru” olarak değerlendirilmemelidir.
  • “vaka” ifadesinin aniden tamamlanması; genellikle ‘break’: Bu, daha fazla ‘vaka’ ifadesinin istenmeyen şekilde değerlendirilmesini önlemek için gereklidir.

“Devam” dışında, bir ifadenin aniden tamamlanmasına neden olacak herhangi bir ifade kullanmak mümkündür. .html#jls-14.1). Bu içerir:

  • “kır”
  • ‘dönüş’
  • “atmak”

Aşağıdaki örnekte, “varsayılan” dahil olmak üzere dört olası durumla tipik bir “switch” ifadesi yazılmıştır.

Scanner scan = new Scanner(System.in);
int i = scan.nextInt();
switch (i) {
    case 0:
        System.out.println("i is zero");
        break;
    case 1:
        System.out.println("i is one");
        break;
    case 2:
        System.out.println("i is two");
        break;
    default:
        System.out.println("i is less than zero or greater than two");
}

“Ara"yı veya ani bir şekilde tamamlanacak herhangi bir ifadeyi atlayarak, birkaç değere karşı değerlendirilen “sona erme” durumları olarak bilinen durumlardan yararlanabiliriz. Bu, bir değere karşı başarılı olacak aralıklar oluşturmak için kullanılabilir, ancak yine de eşitsizlikler kadar esnek değildir.

Scanner scan = new Scanner(System.in);
int foo = scan.nextInt();
switch(foo) {
    case 1:
        System.out.println("I'm equal or greater than one");
    case 2:
    case 3:    
        System.out.println("I'm one, two, or three");
        break;
    default:
        System.out.println("I'm not either one, two, or three");
}

‘foo == 1’ durumunda çıktı şöyle olacaktır:

I'm equal or greater than one
I'm one, two, or three

‘foo == 3’ durumunda çıktı şöyle olacaktır:

I'm one, two, or three

enum Option {

Java’da İfadeye Devam Et #

Devam ifadesi, geçerli yinelemede kalan adımları atlamak için kullanılır. ve sonraki döngü yinelemesi ile başlayın. Kontrol, “devam” ifadesinden, varsa adım değerine (artırma veya eksiltme) gider.

String[] programmers = {"Adrian", "Paul", "John", "Harry"};

    //john is not printed out
    for (String name : programmers) {
        if (name.equals("John"))
            continue;
        System.out.println(name);
    }

‘Continue’ ifadesi ayrıca programın kontrolünü, adlandırılmış bir döngünün (varsa) adım değerine kaydırabilir:

Outer: // The name of the outermost loop is kept here as 'Outer'
for(int i = 0; i < 5; )
{
    for(int j = 0; j < 5; j++)
    {
        continue Outer;
    }
}

Eğer / Else If / Else Kontrolü #

if (i < 2) {
  System.out.println("i is less than 2");
} else if (i > 2) {
  System.out.println("i is more than 2");
} else {
  System.out.println("i is not less than 2, and not more than 2");
}

“if” bloğu yalnızca “i” 1 veya daha az olduğunda çalışır.

“else if” koşulu, yalnızca ondan önceki tüm koşullar (önceki “else if” yapılarında ve üst “if” yapılarında) “false” olarak test edildiyse kontrol edilir. Bu örnekte, “else if” koşulu yalnızca “i” 2’den büyük veya 2’ye eşitse kontrol edilecektir.

Eğer sonucu ’true’ ise, bloğu çalıştırılır ve ondan sonraki tüm ’else if’ ve ’else’ yapıları atlanır.

Eğer ‘if’ ve ’else if’ koşullarından hiçbiri ’true’ olarak test edilmemişse, sonundaki ’else’ bloğu çalıştırılacaktır.

Döngüler İçin #

for (int i = 0; i < 100; i++) {
    System.out.println(i);
}

“for” döngüsünün üç bileşeni (’;’ ile ayrılmış) değişken bildirim/başlatma (burada ‘int i = 0’), koşul (burada ‘i < 100’) ve artış ifadesidir (burada ‘i++). ). Değişken bildirimi, ilk çalıştırmada "{" işaretinin hemen içine yerleştirilmiş gibi bir kez yapılır. Daha sonra koşul kontrol edilir, eğer 'doğru' ise döngünün gövdesi yürütülür, 'yanlış' ise döngü durur. Döngünün devam ettiğini varsayarsak, gövde yürütülür ve son olarak }` değerine ulaşıldığında, koşul tekrar kontrol edilmeden hemen önce artış ifadesi yürütülür.

Döngü yalnızca bir ifade içeriyorsa, küme parantezleri isteğe bağlıdır (noktalı virgülle bir satır yapabilirsiniz). Ancak, yanlış anlamaları ve hataları önlemek için her zaman parantez kullanılması önerilir.

‘for’ döngü bileşenleri isteğe bağlıdır. İş mantığınız bu bölümlerden birini içeriyorsa, ilgili bileşeni “for” döngünüzden çıkarabilirsiniz.

int i = obj.getLastestValue(); // i value is fetched from a method
    
for (; i < 100; i++) { // here initialization is not done
    System.out.println(i);
}

“for (;;) { function-body }” yapısı bir “while (true)” döngüsüne eşittir.

‘İç İçe Döngüler’

İç içe döngü adı verilen, içinde başka bir döngü ifadesi bulunan herhangi bir döngü ifadesi. Daha fazla iç döngüye sahip döngü için aynı yola ‘iç içe döngü’ denir.

    for(;;){
        //Outer Loop Statements
        for(;;){
            //Inner Loop Statements
        }
        //Outer Loop Statements
    }

Yuvalanmış for döngüsü, üçgen şeklindeki sayıları yazdırmak için gösterilebilir.

for(int i=9;i>0;i--){//Outer Loop
    System.out.println();
    for(int k=i;k>0;k--){//Inner Loop -1
        System.out.print(" ");
    }
    for(int j=i;j<=9;j++){//Inner Loop -2
        System.out.print(" "+j);
    }
 }

Üçlü operatör #

Bazen bir koşulu kontrol etmeniz ve bir değişkenin değerini ayarlamanız gerekir.

Örn.

String name;

if (A > B) {
    name = "Billy";
} else {
    name = "Jimmy";
}

Bu kolayca bir satırda şu şekilde yazılabilir:

String name = A > B ? "Billy" : "Jimmy";

Koşul doğruysa, değişkenin değeri koşuldan hemen sonraki değere ayarlanır. Koşul yanlışsa, değişkene ikinci değer verilir.

Döngüler #

int i = 0;
while (i < 100) { // condition gets checked BEFORE the loop body executes
    System.out.println(i);
    i++;
}

Bir ‘while’ döngüsü, parantez içindeki koşul ’true’ olduğu sürece çalışır. Bu aynı zamanda “ön test döngüsü” yapısı olarak da adlandırılır, çünkü ana döngü gövdesi her seferinde gerçekleştirilmeden önce koşullu ifadenin karşılanması gerekir.

Döngü yalnızca bir ifade içeriyorsa küme parantezleri isteğe bağlıdır, ancak bazı kodlama stili kuralları ne olursa olsun parantezleri kullanmayı tercih eder.

Eğer / Değilse #

int i = 2;
if (i < 2) {
  System.out.println("i is less than 2");
} else {
  System.out.println("i is greater than 2");
}

Bir “if” ifadesi, parantez içindeki koşulun sonucuna bağlı olarak kodu koşullu olarak yürütür. Parantez içindeki koşul doğru olduğunda, { ve } gibi küme parantezleri ile tanımlanan if ifadesinin bloğuna girilecektir. kapanış parantezi kadar açılış parantez if ifadesinin kapsamıdır.

“else” bloğu isteğe bağlıdır ve atlanabilir. “if” ifadesi “yanlış” ise çalışır ve “if” ifadesi doğruysa çalışmaz Çünkü bu durumda “if” ifadesi yürütülür.

Kırmak #

“break” ifadesi bir döngüyü (“for”, “while” gibi) veya bir switch ifadesinin değerlendirmesini sonlandırır.

Döngü:

while(true) {
   if(someCondition == 5) {
       break;
   }
}

Örnekteki döngü sonsuza kadar çalışır. Ancak yürütmenin bir noktasında “someCondition” “5"e eşit olduğunda, döngü sona erer.

Birden çok döngü basamaklandırılırsa, yalnızca en içteki döngü “break” kullanılarak sona erer.

Deneyin … Yakalayın … Sonunda #

try { ... } catch ( ... ) { ... } kontrol yapısı, İstisnaları işlemek için kullanılır.

String age_input = "abc";
try {
    int age = Integer.parseInt(age_input);
    if (age >= 18) {
        System.out.println("You can vote!");
    } else {
        System.out.println("Sorry, you can't vote yet.");
    }
} catch (NumberFormatException ex) {
    System.err.println("Invalid input.  '" + age_input + "' is not a valid integer.");
}

Bu yazdırır:

Geçersiz giriş. ‘abc’ geçerli bir tam sayı değil.

‘catch’den sonra ‘finally’ yan tümcesi eklenebilir. Bir istisna atılıp atılmadığına bakılmaksızın, ‘finally’ yan tümcesi her zaman yürütülür.

dene { ... } yakalama ( ... ) { ... } sonunda { ... }

String age_input = "abc";
try {
    int age = Integer.parseInt(age_input);
    if (age >= 18) {
        System.out.println("You can vote!");
    } else {
        System.out.println("Sorry, you can't vote yet.");
    }
} catch (NumberFormatException ex) {
    System.err.println("Invalid input.  '" + age_input + "' is not a valid integer.");
} finally {
    System.out.println("This code will always be run, even if an exception is thrown");
}

Bu yazdırır:

Geçersiz giriş. ‘abc’ geçerli bir tam sayı değil. Bir istisna atılsa bile bu kod her zaman çalıştırılacaktır.
