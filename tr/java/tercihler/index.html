Olay dinleyicileri ekleme #

Bir Preferences nesnesi tarafından yayılan iki tür olay vardır: PreferenceChangeEvent ve NodeChangeEvent.

TercihDeğiştirEtkinliği #

Düğümün anahtar/değer çiftlerinden biri her değiştiğinde, bir “Properties” nesnesi tarafından bir “PreferenceChangeEvent” yayınlanır. PreferenceChangeEventler bir PreferenceChangeListener ile dinlenebilir:

preferences.addPreferenceChangeListener(evt -> {
    String newValue = evt.getNewValue();
    String changedPreferenceKey = evt.getKey();
    Preferences changedNode = evt.getNode();
});

NodeChangeEvent #

Bu olay, bir “Özellikler” düğümünün alt düğümü eklendiğinde veya kaldırıldığında tetiklenir.

preferences.addNodeChangeListener(new NodeChangeListener() {
    @Override
    public void childAdded(NodeChangeEvent evt) {
        Preferences addedChild = evt.getChild();
        Preferences parentOfAddedChild = evt.getParent();
    }

    @Override
    public void childRemoved(NodeChangeEvent evt) {
        Preferences removedChild = evt.getChild();
        Preferences parentOfRemovedChild = evt.getParent();
    }
});

Tercihlerin alt düğümlerini alma #

“Tercihler” nesneleri her zaman bütün bir “Tercihler” ağacındaki belirli bir düğümü temsil eder, şuna benzer:

/userRoot
├── com
│   └── şirketim
│   └── uygulamam
│   ├── darkApplicationMode=true
│   ├── showExitConfirmation=false
│   └── pencereMaximized=true
└── org
    └── myorganization
        └── anotherapp
            ├── defaultFont=Helvetica
            ├── defaultSavePath=/home/matt/Documents
            └── exporting
                ├── defaultFormat=pdf
                └── openInBrowserAfterExport=false

/com/mycompany/myapp düğümünü seçmek için:

  1. Konvansiyonel olarak, bir sınıfın paketine göre:

    package com.mycompany.myapp;

// ...

// Because this class is in the com.mycompany.myapp package, the node
// /com/mycompany/myapp will be returned.
Preferences myApp = Preferences.userNodeForPackage(getClass());

  2. Göreceli yoldan:

    Preferences myApp = Preferences.userRoot().node("com/mycompany/myapp");

Göreceli bir yol (’/’ ile başlamayan bir yol) kullanmak, yolun çözümlendiği üst düğüme göre çözümlenmesine neden olur. Örneğin, aşağıdaki örnek “/one/two/ Three/com/mycompany/myapp” yolunun düğümünü döndürür:

   Preferences prefix = Preferences.userRoot().node("one/two/three");
   Preferences myAppWithPrefix = prefix.node("com/mycompany/myapp");
   // prefix          is /one/two/three
   // myAppWithPrefix is /one/two/three/com/mycompany/myapp

  1. Mutlak yoldan:

    Preferences myApp = Preferences.userRoot().node("/com/mycompany/myapp");

Kök düğümde mutlak bir yol kullanmak, göreli bir yol kullanmaktan farklı olmayacaktır. Aradaki fark, bir alt düğümde çağrılırsa yolun kök düğüme göre çözümlenmesidir.

   Preferences prefix = Preferences.userRoot().node("one/two/three");
   Preferences myAppWitoutPrefix = prefix.node("/com/mycompany/myapp");
   // prefix            is /one/two/three
   // myAppWitoutPrefix is /com/mycompany/myapp

Birden çok uygulama örneğinde tercih erişimini koordine etme #

Tüm “Tercihler” örnekleri, tek bir Java Sanal Makinesinin (JVM) iş parçacıkları arasında her zaman iş parçacığı için güvenlidir. “Tercihler” birden çok JVM’de paylaşılabildiğinden, değişiklikleri sanal makineler arasında eşitlemeyle ilgilenen özel yöntemler vardır.

Yalnızca tek bir örnekte çalışması gereken bir uygulamanız varsa, harici senkronizasyon gerekmez.

Tek bir sistemde birden çok örnekte çalışan bir uygulamanız varsa ve bu nedenle “Tercihler” erişiminin sistemdeki JVM’ler arasında koordine edilmesi gerekiyorsa, herhangi bir " Tercihler düğümü, “Tercihler” düğümündeki değişikliklerin sistemdeki diğer JVM’ler tarafından görülebilmesini sağlamak için kullanılabilir:

// Warning: don't use this if your application is intended
// to only run a single instance on a machine once
// (this is probably the case for most desktop applications)
try {
    preferences.sync();
} catch (BackingStoreException e) {
    // Deal with any errors while saving the preferences to the backing storage
    e.printStackTrace();
}

Tercih değerlerini alma #

Bir “Tercihler” düğümünün değeri, “Dize”, “boolean”, “byte[]”, “double”, “float”, “int” veya “long” türünde olabilir. Belirtilen değerin “Tercihler” düğümünde bulunmaması durumunda, tüm çağrılar varsayılan bir değer sağlamalıdır.

Preferences preferences = Preferences.userNodeForPackage(getClass());

String someString = preferences.get("someKey", "this is the default value");
boolean someBoolean = preferences.getBoolean("someKey", true);
byte[] someByteArray = preferences.getByteArray("someKey", new byte[0]);
double someDouble = preferences.getDouble("someKey", 887284.4d);
float someFloat = preferences.getFloat("someKey", 38723.3f);
int someInt = preferences.getInt("someKey", 13232);
long someLong = preferences.getLong("someKey", 2827637868234L);

Tercihleri ​​kullanma #

“Tercihler”, bir kullanıcının kişisel uygulama ayarlarını yansıtan kullanıcı ayarlarını saklamak için kullanılabilir; editör yazı tipi, uygulamanın tam ekran modunda başlatılmasını tercih edip etmedikleri, “bunu bir daha gösterme” onay kutusunu işaretleyip işaretlemedikleri ve bunun gibi şeyler.

public class ExitConfirmer {
    private static boolean confirmExit() {
        Preferences preferences = Preferences.userNodeForPackage(ExitConfirmer.class);
        boolean doShowDialog = preferences.getBoolean("showExitConfirmation", true); // true is default value

        if (!doShowDialog) {
            return true;
        }

        //
        // Show a dialog here...
        //
        boolean exitWasConfirmed = ...; // whether the user clicked OK or Cancel
        boolean doNotShowAgain = ...; // get value from "Do not show again" checkbox

        if (exitWasConfirmed && doNotShowAgain) {
            // Exit was confirmed and the user chose that the dialog should not be shown again
            // Save these settings to the Preferences object so the dialog will not show again next time
            preferences.putBoolean("showExitConfirmation", false);
        }

        return exitWasConfirmed;
    }

    public static void exit() {
        if (confirmExit()) {
            System.exit(0);
        }
    }
}
