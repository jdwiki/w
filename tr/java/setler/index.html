Başlatma #

Küme, yinelenen öğeler içeremeyen bir Koleksiyondur. Matematiksel küme soyutlamasını modeller.

‘Set’, ‘HashSet’, ‘TreeSet’, ‘LinkedHashSet’ gibi çeşitli sınıflarda uygulanmasına sahiptir.

Örneğin:

Karma Küme:

Set<T> set = new HashSet<T>();

Burada “T”, “Dize”, “Tamsayı” veya başka herhangi bir nesne olabilir. HashSet, O(1)‘in hızlı aranmasına izin verir, ancak buna eklenen verileri sıralamaz ve öğelerin ekleme sırasını kaybeder.

AğaçSeti:

O(lg(n)) alan temel işlemler için bazı hızlardan ödün vererek verileri sıralanmış bir şekilde depolar. Öğelerin ekleme sırasını korumaz.

TreeSet<T> sortedSet = new TreeSet<T>();

LinkedHashSet:

Bu, ‘HashSet’in bağlantılı bir liste uygulamasıdır. Once, öğeleri eklendikleri sırayla yineleyebilir. İçeriği için sıralama sağlanmamıştır. O(1) temel işlemler sağlanır, ancak yedek bağlantılı listeyi korumanın ‘HashSet’ten daha yüksek maliyeti vardır.

LinkedHashSet<T> linkedhashset = new LinkedHashSet<T>();

Kümenin Temelleri #

Set nedir?

Küme, kümedeki hiçbir iki öğenin eşit olmadığı önemli bir özelliğe sahip bir öğe kümesi içeren bir veri yapısıdır.

Set Çeşitleri:

  1. HashSet: Bir karma tablosu tarafından desteklenen bir küme (aslında bir HashMap örneği)
  2. Bağlantılı HashSet: Hash tablosu ve bağlantılı liste tarafından desteklenen, öngörülebilir yineleme sırasına sahip bir Küme
  3. TreeSet: TreeMap’e dayalı bir NavigableSet uygulaması.

Bir set oluşturma

Set<Integer> set = new HashSet<Integer>(); // Creates an empty Set of Integers

Set<Integer> linkedHashSet = new LinkedHashSet<Integer>(); //Creates a empty Set of Integers, with predictable iteration order

Bir Kümeye eleman ekleme

Öğeler, ‘add()’ yöntemi kullanılarak bir kümeye eklenebilir

 set.add(12); //  - Adds element 12 to the set
 set.add(13); //  - Adds element 13 to the set

Bu yöntemi uyguladıktan sonra setimiz:

set = [12,13]

Bir Kümenin tüm öğelerini silin

set.clear();  //Removes all objects from the collection.

Bu setten sonra:

set = []

Bir öğenin Kümenin parçası olup olmadığını kontrol edin

Kümedeki bir elemanın varlığı contains() yöntemi kullanılarak kontrol edilebilir.

set.contains(0);  //Returns true if a specified object is an element within the set.

Çıktı: “Yanlış”

Bir Setin boş olup olmadığını kontrol edin

Bir Kümenin boş olup olmadığını kontrol etmek için isEmpty() yöntemi kullanılabilir.

set.isEmpty();  //Returns true if the set has no elements

Çıktı: Doğru

Setten bir öğeyi kaldırın

 set.remove(0); // Removes first occurrence of a specified object from the collection

Set Boyutunu Kontrol Edin

set.size(); //Returns the number of elements in the collection

Çıkış: 0

Küme Türleri ve Kullanımı #

Genel olarak kümeler, benzersiz değerleri saklayan bir koleksiyon türüdür. Benzersizlik, “equals()” ve “hashCode()” yöntemleriyle belirlenir.

Sıralama, kümenin türüne göre belirlenir.

HashSet - Rastgele Sıralama #

Set<String> set = new HashSet<> ();
set.add("Banana");
set.add("Banana");
set.add("Apple");
set.add("Strawberry");

// Set Elements: ["Strawberry", "Banana", "Apple"]
son sürüm

LinkedHashSet - Kampanya Siparişi #

Set<String> set = new LinkedHashSet<> ();
set.add("Banana");
set.add("Banana");
set.add("Apple");
set.add("Strawberry");

// Set Elements: ["Banana", "Apple", "Strawberry"]
son sürüm

“TreeSet” - “CompareTo()” veya “Karşılaştırıcı” ile #

Set<String> set = new TreeSet<> ();
set.add("Banana");
set.add("Banana");
set.add("Apple");
set.add("Strawberry");

// Set Elements: ["Apple", "Banana", "Strawberry"]
son sürüm 
Set<String> set = new TreeSet<> ((string1, string2) -> string2.compareTo(string1));
set.add("Banana");
set.add("Banana");
set.add("Apple");
set.add("Strawberry");

// Set Elements: ["Strawberry", "Banana", "Apple"]
son sürüm

Değerler içeren bir HashSet bildirme #

HashSet’ten miras alan yeni bir sınıf oluşturabilirsiniz:

Set<String> h = new HashSet<String>() {{
    add("a");
    add("b");
}};

Tek satırlık çözüm:

Set<String> h = new HashSet<String>(Arrays.asList("a", "b"));

Guava’yı kullanma:

Sets.newHashSet("a", "b", "c")

Akışları Kullanma:

Set<String> set3 = Stream.of("a", "b", "c").collect(toSet());

Mevcut bir Setten bir liste oluşturun #

Yeni bir Liste kullanma

List<String> list = new ArrayList<String>(listOfElements);

List.addAll() yöntemini kullanma

    Set<String> set = new HashSet<String>();
    set.add("foo");
    set.add("boo");
    
    List<String> list = new ArrayList<String>();
    list.addAll(set);

Java 8 Steam API’sini kullanma

List<String> list = set.stream().collect(Collectors.toList());

Set kullanarak kopyaları ortadan kaldırma #

Bir koleksiyon “elemanlarınız” olduğunu ve aynı öğeleri içeren ancak tüm yinelenenler ortadan kaldırılmış olan başka bir koleksiyon oluşturmak istediğinizi varsayalım:

Collection<Type> noDuplicates = new HashSet<Type>(elements);

Örnek:

List<String> names = new ArrayList<>(
        Arrays.asList("John", "Marco", "Jenny", "Emily", "Jenny", "Emily", "John"));
Set<String> noDuplicates = new HashSet<>(names);
System.out.println("noDuplicates = " + noDuplicates);

Çıktı:

noDuplicates = [Marco, Emily, John, Jenny]
