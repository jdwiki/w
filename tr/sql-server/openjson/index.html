Anahtarı al:JSON metninden değer çiftleri #

OPENJSON işlevi JSON metnini ayrıştırır ve tüm anahtar:değer çiftlerini JSON’un ilk düzeyinde döndürür:

declare @json NVARCHAR(4000) = N'{"Name":"Joe","age":27,"skills":["C#","SQL"]}';
SELECT * FROM OPENJSON(@json);
anahtar değer tip
İsim Joe 1
yaş 27 2
beceriler [“C#”,“SQL”] 4

Sütun türü, değerin türünü tanımlar, yani boş(0), dize(1), sayı(2), boolean(3), dizi(4) ve nesne(5).

JSON dizisini satır kümesine dönüştürün #

OPENJSON işlevi, JSON nesnelerinin koleksiyonunu ayrıştırır ve JSON metninden satır kümesi olarak değerleri döndürür.

declare @json nvarchar(4000) = N'[
  {"Number":"SO43659","Date":"2011-05-31T00:00:00","Customer": "MSFT","Price":59.99,"Quantity":1},
  {"Number":"SO43661","Date":"2011-06-01T00:00:00","Customer":"Nokia","Price":24.99,"Quantity":3}
]'

SELECT    * 
FROM OPENJSON (@json)
    WITH (
          Number   varchar(200),
          Date     datetime,
          Customer varchar(200),
          Quantity int
  )

WITH yan tümcesinde, OPENJSON işlevinin dönüş şeması belirtilir. JSON nesnelerindeki anahtarlar, sütun adlarına göre alınır. JSON’daki bir anahtar, WITH yan tümcesinde belirtilmemişse (örneğin, bu örnekte Fiyat), yoksayılacaktır. Değerler otomatik olarak belirtilen türlere dönüştürülür.

Sayı Tarih Müşteri Miktar
SO43659 2011-05-31T00:00:00 MSFT 1
SO43661 2011-06-01T00:00:00 Nokia 3

Yuvalanmış JSON alanlarını satır kümesine dönüştürün #

OPENJSON işlevi, JSON nesnelerinin koleksiyonunu ayrıştırır ve JSON metninden satır kümesi olarak değerleri döndürür. Giriş nesnesindeki değerler iç içeyse, WITH yan tümcesindeki her sütunda ek eşleme parametresi belirtilebilir:

declare @json nvarchar(4000) = N'[
  {"data":{"num":"SO43659","date":"2011-05-31T00:00:00"},"info":{"customer":"MSFT","Price":59.99,"qty":1}},
  {"data":{"number":"SO43661","date":"2011-06-01T00:00:00"},"info":{"customer":"Nokia","Price":24.99,"qty":3}}
]'

SELECT    * 
FROM OPENJSON (@json)
    WITH (
          Number   varchar(200) '$.data.num',
          Date     datetime '$.data.date',
          Customer varchar(200) '$.info.customer',
          Quantity int '$.info.qty',
  )

WITH yan tümcesinde, OPENJSON işlevinin dönüş şeması belirtilir. Tür belirtildikten sonra, döndürülen değerin bulunması gereken JSON düğümlerinin yolu. JSON nesnelerindeki anahtarlar bu yollarla alınır. Değerler otomatik olarak belirtilen türlere dönüştürülür.

Sayı Tarih Müşteri Miktar
SO43659 2011-05-31T00:00:00 MSFT 1
SO43661 2011-06-01T00:00:00 Nokia 3

Dahili JSON alt nesnelerini çıkarma #

OPENJSON, JSON metninin içindeki JSON nesnelerinin parçalarını çıkarabilir. JSON alt nesnesine başvuran sütun tanımında nvarchar(max) türünü ve AS JSON seçeneğini ayarlayın:

declare @json nvarchar(4000) = N'[
  {"Number":"SO43659","Date":"2011-05-31T00:00:00","info":{"customer":"MSFT","Price":59.99,"qty":1}},
  {"Number":"SO43661","Date":"2011-06-01T00:00:00","info":{"customer":"Nokia","Price":24.99,"qty":3}}
]'

SELECT    * 
FROM OPENJSON (@json)
    WITH (
          Number   varchar(200),
          Date     datetime,
          Info nvarchar(max) '$.info' AS JSON
  )

Bilgi sütunu “Bilgi” nesnesine eşlenecektir. Sonuçlar:

Sayı Tarih Bilgi
SO43659 2011-05-31T00:00:00 {“müşteri”:“MSFT”,“Fiyat”:59,99,“adet”:1}
SO43661 2011-06-01T00:00:00 {“müşteri”:“Nokia”,“Fiyat”:24.99,“adet”:3}

Yuvalanmış JSON alt dizileriyle çalışma #

JSON, iç dizilerle karmaşık bir yapıya sahip olabilir. Bu örnekte, OrderItems’in iç içe alt dizisine sahip bir dizi siparişimiz var.

declare @json nvarchar(4000) = N'[
  {"Number":"SO43659","Date":"2011-05-31T00:00:00",
    "Items":[{"Price":11.99,"Quantity":1},{"Price":12.99,"Quantity":5}]},
  {"Number":"SO43661","Date":"2011-06-01T00:00:00",
    "Items":[{"Price":21.99,"Quantity":3},{"Price":22.99,"Quantity":2},{"Price":23.99,"Quantity":2}]}
]'

Öğeler dizisini JSON parçası olarak döndürecek olan OPENJSON kullanarak kök düzeyindeki özellikleri ayrıştırabiliriz. Daha sonra OPENJSON’u tekrar Items dizisine uygulayabilir ve iç JSON tablosunu açabiliriz. Birinci düzey tablo ve iç tablo, standart tablolar arasındaki JOIN’deki gibi “birleştirilir”:

SELECT    * 
FROM
    OPENJSON (@json)
    WITH (  Number varchar(200), Date datetime,
            Items nvarchar(max) AS JSON )
        CROSS APPLY 
            OPENJSON (Items)
            WITH ( Price float, Quantity int)

Sonuçlar:
