PostgreSQL, kullanıcılara sunulan zengin bir yerel veri türleri kümesine sahiptir. Kullanıcılar, CREATE TYPE komutunu kullanarak PostgreSQL’e yeni türler ekleyebilir.

https://www.postgresql.org/docs/9.6/static/datatype.html

Sayısal Türler #

İsim Depolama Boyutu Açıklama Aralık
“küçük” 2 bayt küçük aralıklı tam sayı -32768 ila +32767
’tamsayı' 4 bayt tamsayı için tipik seçim -2147483648 ila +2147483647
“büyük” 8 bayt geniş aralıklı tam sayı -9223372036854775808 - +9223372036854775807
“ondalık” değişken kullanıcı tanımlı kesinlik, kesin ondalık noktadan önce 131072 basamağa kadar; ondalık noktadan sonra 16383 basamağa kadar
‘sayısal’ değişken kullanıcı tanımlı kesinlik, kesin ondalık noktadan önce 131072 basamağa kadar; ondalık noktadan sonra 16383 basamağa kadar
“gerçek” 4 bayt değişken hassasiyetli, kesin olmayan 6 ondalık basamak hassasiyeti
çift hassasiyet 8 bayt değişken hassasiyetli, kesin olmayan 15 ondalık basamak hassasiyeti

‘int4range’ tamsayı aralığı
‘int8range’ bigint aralığı
“sayı” sayısal aralığı

Diziler #

PostgreSQL’de herhangi bir yerleşik, kullanıcı tanımlı veya enum türünden Diziler oluşturabilirsiniz. Varsayılan olarak bir Dizi için sınır yoktur, ancak bunu belirtebilirsiniz.

Dizi Bildirmek #

SELECT integer[];
SELECT integer[3];
SELECT integer[][];
SELECT integer[3][3];
SELECT integer ARRAY;
SELECT integer ARRAY[3];

Dizi Oluşturma #

SELECT '{0,1,2}';
SELECT '{{0,1},{1,2}}';
SELECT ARRAY[0,1,2];
SELECT ARRAY[ARRAY[0,1],ARRAY[1,2]];

Diziye Erişmek #

Varsayılan olarak PostgreSQL, diziler için tek tabanlı bir numaralandırma kuralı kullanır, yani n elemanlı bir dizi “dizi[1]” ile başlar ve “dizi[n]” ile biter.

--accesing a spefific element
WITH arr AS (SELECT ARRAY[0,1,2] int_arr) SELECT int_arr[1] FROM arr;

int_arr
---------
        0
(1 row)

--sclicing an array
WITH arr AS (SELECT ARRAY[0,1,2] int_arr) SELECT int_arr[1:2] FROM arr;

int_arr
---------
    {0,1}
(1 row)

Bir dizi hakkında bilgi alma #

--array dimensions (as text)
with arr as (select ARRAY[0,1,2] int_arr) select array_dims(int_arr) from arr;

array_dims
------------
       [1:3]
(1 row)

--length of an array dimension
 WITH arr AS (SELECT ARRAY[0,1,2] int_arr) SELECT array_length(int_arr,1) FROM arr;

 array_length
 --------------
              3
 (1 row)

--total number of elements across all dimensions
 WITH arr AS (SELECT ARRAY[0,1,2] int_arr) SELECT cardinality(int_arr) FROM arr;
 
 cardinality
 -------------
             3
 (1 row)
