Paketleyici ve Minifier Uzantısı #

Visual Studio ayrıca, bu işlemi sizin için gerçekleştirebilecek bir Bundler and Minifier Extension içerir. Uzantı, bir satır kod yazmadan ihtiyacınız olan dosyaları kolayca seçmenize ve paketlemenize olanak tanır.

Paketlerinizi Oluşturma #

Uzantıyı yükledikten sonra, bir pakete dahil etmek istediğiniz tüm belirli dosyaları seçin ve uzantıdan Dosyaları Paketle ve Küçült seçeneğini kullanın:

Bu, paketinize bir ad vermenizi ve onu kaydetmek için bir konum seçmenizi isteyecektir. Daha sonra projenizde aşağıdaki gibi görünen “bundleconfig.json” adlı yeni bir dosya göreceksiniz:

[
  {
    "outputFileName": "wwwroot/app/bundle.js",
    "inputFiles": [
      "wwwroot/lib/jquery/dist/jquery.js",
      "wwwroot/lib/bootstrap/dist/js/bootstrap.js",
      "wwwroot/lib/jquery-validation/dist/jquery.validate.js",
      "wwwroot/lib/jquery-validation-unobtrusive/jquery.validate.unobtrusive.js" 
    ]
  }
]

NOT: Dosyaların seçildiği sıra, paket içinde görünmelerini sipariş edin, yani eğer varsa bağımlılıklar, bunu hesaba kattığınızdan emin olun.

Paketlerinizi Küçültme #

Şimdi bir önceki adım dosyalarınızı paketleyecektir, eğer paketi küçültmek istiyorsanız, bunu bundleconfig.json dosyasında belirtmeniz gerekir. ** Küçültülmesini istediğinizi belirtmek için mevcut paketinize aşağıdaki gibi bir “küçült” bloğu eklemeniz yeterlidir :**

[
  {
    "outputFileName": "wwwroot/app/bundle.js",
    "inputFiles": [
      "wwwroot/lib/jquery/dist/jquery.js",
      "wwwroot/lib/bootstrap/dist/js/bootstrap.js",
      "wwwroot/lib/jquery-validation/dist/jquery.validate.js",
      "wwwroot/lib/jquery-validation-unobtrusive/jquery.validate.unobtrusive.js" 
    ],
    "minify": {
      "enabled": true
    }
  }
]

Paketlerinizi Otomatikleştirin #

Son olarak, bu işlemi otomatikleştirmek istiyorsanız, paketlerinizin uygulamanızdaki tüm değişiklikleri yansıtmasını sağlamak için uygulamanız oluşturulduğunda çalışacak şekilde bir görev zamanlayabilirsiniz.

Bunu yapmak için aşağıdakileri yapmanız gerekir:

  • Görev Yürütücü Gezgini’ni açın (Araçlar > Görev Yürütücü Gezgini aracılığıyla).
  • **bundleconfig.json altındaki Tüm Dosyaları Güncelle seçeneğine ** sağ tıklayın.
  • Bağlantılar bağlam menüsünden tercih ettiğiniz bağlamayı seçin.

Bunu yaptıktan sonra, paketleriniz seçtiğiniz tercih edilen adımda otomatik olarak güncellenmelidir.

dotnet paketi Komutu #

ASP.NET Core RTM sürümü, mevcut ASP.NET Core uygulamalarına kolayca entegre edilebilen ve herhangi bir harici uzantı veya komut dosyası gerektirmeyen yeni bir Bundling ve Minification aracı olan ‘BundlerMinifier.Core’u tanıttı.

BundlerMinifier.Core’u kullanma #

Bu aracı kullanmak için, aşağıda görüldüğü gibi mevcut ‘project.json’ dosyanızın ’tools’ bölümündeki ‘BundlerMinifier.Core’a bir referans ekleyin:

"tools": {
  "BundlerMinifier.Core": "2.0.238",
  "Microsoft.AspNetCore.Razor.Tools": "1.0.0-preview2-final",
  "Microsoft.AspNetCore.Server.IISIntegration.Tools": "1.0.0-preview2-final"
}

Paketlerinizi Yapılandırma #

Aracı ekledikten sonra, paketlerinize dahil etmek istediğiniz dosyaları yapılandırmak için kullanılacak projenize bir bundleconfig.json dosyası eklemeniz gerekecektir. Minimal bir konfigürasyon aşağıda görülebilir:

[
  {
    "outputFileName": "wwwroot/css/site.min.css",
    "inputFiles": [
      "wwwroot/css/site.css"
    ]
  },
  {
    "outputFileName": "wwwroot/js/site.min.js",
    "inputFiles": [
      "wwwroot/js/site.js"
    ],
    "minify": {
      "enabled": true,
      "renameLocals": true
    },
    "sourceMap": false
  },
  {
    "outputFileName": "wwwroot/js/semantic.validation.min.js",
    "inputFiles": [
      "wwwroot/js/semantic.validation.js"
    ],
    "minify": {
      "enabled": true,
      "renameLocals": true
    }
  }
]

Paket Oluşturma / Güncelleme #

Paketleriniz yapılandırıldıktan sonra, mevcut dosyalarınızı aşağıdaki komutla paketleyebilir ve küçültebilirsiniz:

dotnet bundle

Otomatik Paketleme #

Paketleme ve Küçültme işlemi, mevcut ‘project.json’ dosyanızın ön derleme bölümüne ‘dotnet paketi’ komutu eklenerek derleme işleminin bir parçası olarak otomatikleştirilebilir:

"scripts": {
  "precompile": [
    "dotnet bundle"
  ]
}

Kullanılabilir Komutlar #

Kullanılabilir tüm komutların bir listesini ve açıklamalarını aşağıda görebilirsiniz:

  • dotnet paketi - Belirtilen dosyalarınızı bir araya getirmek ve küçültmek için bundleconfig.json dosyasını kullanarak paket komutunu yürütür.
  • dotnet paket temizliği - Varolan tüm çıktı dosyalarını diskten temizler.
  • dotnet paket izleme - Dosyalarınızı gruplamak için ‘bundleconfig.json’ yapılandırmasından mevcut bir giriş dosyası her geldiğinde otomatik olarak ‘dotnet paketi’ çalıştıracak bir izleyici oluşturur.
  • dotnet paket yardımı - Komut satırı arayüzünü kullanmak için mevcut tüm yardım seçeneklerini ve talimatları görüntüler.

Grunt ve Gulp #

ASP.NET Core uygulamalarında, tasarım zamanı sırasında Gulp ve Grunt gibi üçüncü taraf araçları kullanarak istemci tarafı kaynakları paketler ve küçültürsünüz. .com/). Tasarım zamanı paketleme ve küçültme kullanılarak, uygulamanın dağıtımından önce küçültülmüş dosyalar oluşturulur. Dağıtımdan önce paketleme ve küçültme, azaltılmış sunucu yükü avantajı sağlar. Ancak, tasarım zamanı paketleme ve küçültmenin yapı karmaşıklığını artırdığını ve yalnızca statik dosyalarla çalıştığını bilmek önemlidir.

Bu, projenizdeki bir gulpfile.js dosyası aracılığıyla Gulp’u yapılandırarak ASP.NET Core’da yapılır:

// Defining dependencies
var gulp = require("gulp"),  
    rimraf = require("rimraf"),
    concat = require("gulp-concat"),
    cssmin = require("gulp-cssmin"),
    uglify = require("gulp-uglify");

// Define web root
var webroot = "./wwwroot/"

// Defining paths
var paths = {  
    js: webroot + "js/**/*.js",
    minJs: webroot + "js/**/*.min.js",
    css: webroot + "css/**/*.css",
    minCss: webroot + "css/**/*.min.css",
    concatJsDest: webroot + "js/site.min.js",
    concatCssDest: webroot + "css/site.min.css"
};

// Bundling (via concat()) and minifying (via uglify()) Javascript
gulp.task("min:js", function () {  
    return gulp.src([paths.js, "!" + paths.minJs], { base: "." })
        .pipe(concat(paths.concatJsDest))
        .pipe(uglify())
        .pipe(gulp.dest("."));
});

// Bundling (via concat()) and minifying (via cssmin()) Javascript
gulp.task("min:css", function () {  
    return gulp.src([paths.css, "!" + paths.minCss])
        .pipe(concat(paths.concatCssDest))
        .pipe(cssmin())
        .pipe(gulp.dest("."));
});

Bu yaklaşım, mevcut Javascript ve CSS dosyalarınızı, kullanılan dizinlere ve globbing modellerine göre uygun şekilde paketler ve küçültür.
