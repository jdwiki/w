Project json, Microsoft’un msbuild lehine csproj dosyalarına geri dönmeden önce, asp.net-core projeleri tarafından geçici olarak kullanılan bir proje yapılandırma dosyası yapısıdır.

Basit Kütüphane projesi örneği #

NETStandard 1.6 tabanlı bir kitaplık şöyle görünür:

{
  "version": "1.0.0",
  "dependencies": {
    "NETStandard.Library": "1.6.1", //nuget dependency
   },
  "frameworks": { //frameworks the library is build for
    "netstandard1.6": {}
  },
  "buildOptions": {
    "debugType": "portable"
  }
}

Basit başlangıç ​​projesi #

Bir .NetCore 1.1 Konsol Uygulaması için proje yapılandırmasının basit bir örneği

{
  "version": "1.0.0",
  "buildOptions": {
    "emitEntryPoint": true // make sure entry point is emitted.
  },
  "dependencies": {
  },
  "tools": {
  },
  "frameworks": {
    "netcoreapp1.1": { // run as console app
      "dependencies": {
        "Microsoft.NETCore.App": {
          "type": "platform",
          "version": "1.1.0"
        }
      },
      "imports": "dnxcore50"
    }
  },
}
