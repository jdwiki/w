Parametreler #

İsim Bilgi
asp eylemi Formun gönderileceği eylem yönteminin adı
asp denetleyicisi asp-action içinde belirtilen eylem yönteminin bulunduğu denetleyicinin adı
asp-route-* Form eylemi öznitelik değerine sorgu dizesi olarak eklemek istediğiniz özel rota değerleri. 8’i istediğiniz sorgu dizesi adıyla değiştirin

Form Etiketi Yardımcısı - Özel rota özellikleriyle #

<form asp-action="create" 
      asp-controller="Home" 
      asp-route-returnurl="dashboard" 
      asp-route-from="google">
     <!--Your form elements goes here--> 
</form>

Bu, aşağıdaki işaretlemeyi oluşturacaktır

<form action="/Home/create?returnurl=dashboard&amp;from=google" method="post">
    <!--Your form elements goes here--> 
</form>

Form Etiket Yardımcısı - Temel örnek #

<form asp-action="create" asp-controller="Home">
     <!--Your form elements goes here--> 
</form>

Giriş Etiketi Yardımcısı #

Görünümünüzün güçlü bir şekilde aşağıdaki gibi bir görünüm modeline yazıldığını varsayarsak

public class CreateProduct
{
   public string Name { set; get; }
}

Ve bunun bir nesnesini eylem yönteminizden görünüme geçiriyorsunuz.

@model CreateProduct
<form asp-action="create" asp-controller="Home" >

    <input type="text" asp-for="Name"/>   
    <input type="submit"/>

</form>

Bu, aşağıdaki işaretlemeyi oluşturacaktır.

<form action="/Home/create" method="post"> 

    <input type="text" id="Name" name="Name" value="" />
    <input type="submit"/>
    <input name="__RequestVerificationToken" type="hidden" value="ThisWillBeAUniqueToken" />

</form>

Giriş alanının varsayılan bir değerle oluşturulmasını istiyorsanız, eylem yönteminde görünüm modelinizin Ad özelliği değerini ayarlayabilirsiniz.

public IActionResult Create()
{
  var vm = new CreateProduct { Name="IPhone"};
  return View(vm);
}

Form gönderme ve Model bağlama

HttpPost eylem yöntemi parametreniz/‘ad’ adlı bir parametre olarak “CreateProduct” kullanırsanız model bağlama düzgün çalışır

Etiket Yardımcısı Seç #

Görünümünüzün böyle bir görünüm modeline güçlü bir şekilde yazıldığını varsayarsak

public class CreateProduct
{       
   public IEnumerable<SelectListItem> Categories { set; get; }
   public int SelectedCategory { set; get; }
}

Ve GET eylem yönteminizde, bu görünüm modelinin bir nesnesini oluşturuyor, Kategoriler özelliğini ayarlıyor ve görünüme gönderiyorsunuz.

public IActionResult Create()
{
    var vm = new CreateProduct();
    vm.Categories = new List<SelectListItem>
    {
        new SelectListItem {Text = "Books", Value = "1"},
        new SelectListItem {Text = "Furniture", Value = "2"}
    };
    return View(vm);
}

ve senin görüşüne göre

@model CreateProduct

<form asp-action="create" asp-controller="Home">
    <select asp-for="SelectedCategory" asp-items="@Model.Categories">
        <option>Select one</option>
    </select>
    <input type="submit"/>
</form>

Bu, aşağıdaki işaretlemeyi oluşturacaktır (yalnızca formun/alanların ilgili kısımlarını içerir)

<form action="/Home/create" method="post">  
    <select data-val="true" id="SelectedCategory" name="SelectedCategory">
        <option>Select one</option>
        <option value="1">Shyju</option>
        <option value="2">Sean</option>
    </select>
    <input type="submit"/>
</form>

Form gönderiminde seçilen açılır değerin alınması

HttpPost eylem yöntemi parametrenizle aynı görünüm modelini kullanabilirsiniz.

[HttpPost]
public ActionResult Create(CreateProduct model)
{
  //check model.SelectedCategory value
  / /to do : return something
}

Bir seçeneği seçili olarak ayarlayın

Bir seçeneği seçili seçenek olarak ayarlamak isterseniz, SelectedCategory özellik değerini ayarlamanız yeterlidir.

public IActionResult Create()
{
    var vm = new CreateProduct();
    vm.Categories = new List<SelectListItem>
    {
        new SelectListItem {Text = "Books", Value = "1"},
        new SelectListItem {Text = "Furniture", Value = "2"},
        new SelectListItem {Text = "Music", Value = "3"}
    };
    vm.SelectedCategory = 2;
    return View(vm);
}

Bir Çoklu seçim açılır menüsü/ListBox oluşturma

Bir çoklu seçim açılır menüsü oluşturmak istiyorsanız, görünümünüzde ‘asp-for’ özelliği için kullandığınız görünüm modeli özelliğinizi bir dizi türüne değiştirebilirsiniz.

public class CreateProduct
{       
   public IEnumerable<SelectListItem> Categories { set; get; }
   public int[] SelectedCategories { set; get; }
}

görünümde

@model CreateProduct

<form asp-action="create" asp-controller="Home" >
    <select asp-for="SelectedCategories" asp-items="@Model.Categories">
        <option>Select one</option>
    </select>
    <input type="submit"/>
</form>

Bu, ‘multiple’ özniteliği olan SELECT öğesini oluşturacaktır.

<form action="/Home/create" method="post">
     <select id="SelectedCategories" multiple="multiple" name="SelectedCategories">
        <option>Select one</option>
        <option value="1">Shyju</option>
       <option value="2">Sean</option>
     </select>
    <input type="submit"/>
</form>

Özel Etiket Yardımcısı #

‘ITagHelper’ uygulayarak veya ‘TagHelper’ kolaylık sınıfından türeterek kendi etiket yardımcılarınızı oluşturabilirsiniz.

  • Varsayılan kural, isteğe bağlı TagHelper son eki olmadan yardımcının adıyla eşleşen bir html etiketini hedeflemektir. Örneğin, “WidgetTagHelper”, bir “” etiketini hedefleyecektir.
  • “[HtmlTargetElement]” özelliği, hedeflenen etiketi daha fazla kontrol etmek için kullanılabilir
  • Sınıfın herhangi bir genel özelliğine, jilet işaretlemesinde nitelik olarak bir değer verilebilir. Örneğin bir kamu mülkü genel string Title {get; set;}, <widget title="my title"> şeklinde bir değer verilebilir.
  • Varsayılan olarak, etiket yardımcıları, etiket yardımcıları için Pascal kasalı C# sınıf adlarını ve özelliklerini küçük kebap durumuna çevirir. Örneğin, “[HtmlTargetElement]” kullanmayı atlarsanız ve sınıf adı “WidgetBoxTagHelper” ise, Razor’da “” yazarsınız.
  • “Process” ve “ProcessAsync”, oluşturma mantığını içerir. Her ikisi de, oluşturulmakta olan geçerli etiket hakkında bilgi içeren bir bağlam parametresi ve oluşturulan sonucu özelleştirmek için kullanılan bir çıktı parametresi alır.

Özel etiket yardımcıları içeren herhangi bir derlemenin _ViewImports.cshtml dosyasına eklenmesi gerekir (ad alanı değil, kaydedilmekte olan derlemedir):

@addTagHelper *, MyAssembly

Örnek Widget Özel Etiket Yardımcısı #

Aşağıdaki örnek, aşağıdaki gibi jilet işaretlemesini hedefleyecek özel bir widget etiketi yardımcısı oluşturur:

<widget-box title="My Title">This is my content: @ViewData["Message"]</widget-box>

Hangisi olarak işlenecek:

<div class="widget-box">
    <div class="widget-header">My Title</div>
    <div class="widget-body">This is my content: some message</div>
</div>

Böyle bir etiket yardımcısı oluşturmak için gereken kod şudur:

[HtmlTargetElement("widget-box")]
public class WidgetTagHelper : TagHelper
{
    public string Title { get; set; }

    public override async Task ProcessAsync(TagHelperContext context, TagHelperOutput output)
    {
        var outerTag = new TagBuilder("div");
        outerTag.Attributes.Add("class", output.TagName);
        output.MergeAttributes(outerTag);
        output.TagName = outerTag.TagName;

        //Create the header
        var header = new TagBuilder("div");
        header.Attributes.Add("class", "widget-header");
        header.InnerHtml.Append(this.Title);
        output.PreContent.SetHtmlContent(header);

        //Create the body and replace original tag helper content
        var body = new TagBuilder("div");
        body.Attributes.Add("class", "widget-body");
        var originalContents = await output.GetChildContentAsync();
        body.InnerHtml.Append(originalContents.GetContent());
        output.Content.SetHtmlContent(body);
    }
}

Etiket Etiket Yardımcısı #

Etiket Etiket Yardımcısı, bir model özelliği için “etiket” oluşturmak için kullanılabilir. MVC’nin önceki sürümlerinde Html.LabelFor yönteminin yerini alır.

Diyelim ki bir modeliniz var:

public class FormViewModel
{
    public string Name { get; set; }
}

Görünümde “label” HTML öğesini ve “asp-for” etiket yardımcısını kullanabilirsiniz:

<form>
    <label asp-for="Name"></label>
    <input asp-for="Name" type="text" />
</form>

Bu, MVC’nin önceki sürümlerinde aşağıdaki koda eşdeğerdir:

<form>
    @Html.LabelFor(x => x.Name)
    @Html.TextBoxFor(x => x.Name)
</form>

Yukarıdaki her iki kod parçacığı da aynı HTML’yi oluşturur:

<form>
    <label for="Name">Name</label>
    <input name="Name" id="Name" type="text" value="">
</form>

Bağlantı etiketi yardımcısı #

Bağlantı etiketi yardımcısı, belirli bir denetleyici eylemine veya MVC yoluna bağlanmak için href nitelikleri oluşturmak için kullanılır. Temel örnek

<a asp-controller="Products" asp-action="Index">Login</a>

Bazen, bağlandığınız denetleyici eylemi için ek parametreler belirtmemiz gerekir. Asp-route-prefix ile öznitelikler ekleyerek bu parametreler için değerler belirleyebiliriz.

<a asp-controller="Products" asp-action="Details" asp-route-id="@Model.ProductId">
   View Details
</a>
