Kurulum veya Kurulum #

Less, en popüler CSS Ön işlemcilerinden biri olmuştur ve ayrıca Bootstrap, Foundation, vb. gibi sayısız ön uç çerçevesinde yaygın olarak konuşlandırılmıştır. Less Compiler, bir İçerik Dağıtım Ağından edinilebilen JavaScript tabanlı bir derleyicidir. :

<script src="//cdnjs.cloudflare.com/ajax/libs/less.js/2.5.1/less.min.js"></script>

Az belgenizi JavaScript derleyicisi yüklenmeden önce, <link /> etiketini kullanarak eklemeniz gerekir. Derleyici ile birlikte Less stil sayfası şöyle görünür:

<link rel="stylesheet/less" type="text/css" href="main.less" />
<script src="//cdnjs.cloudflare.com/ajax/libs/less.js/2.5.1/less.min.js"></script>

Not: İstemci tarafında (veya tarayıcıda) Daha Az Derleme yapılması genellikle önerilmez. Yalnızca geliştirme için veya sunucu tarafı derlemeyi mümkün kılmayan dinamik ayarlar kullanılırken kullanılmalıdır.

Örnek Daha Az Sözdizimi #

Aşağıdaki örnek, değişkenlerin nasıl bildirildiğini ve kullanıldığını, karışımların Less içinde nasıl tanımlandığını ve çağrıldığını gösteren örnek bir Less dosyasıdır.

/* Variables */
@color-base: #87ceeb;

/* Simple mixin to set border */

.set-border(@width; @style; @color) {
  border: @width @style darken(@color, 10%);
}

/* Main CSS */
.class1 {
  background-color: @color-base;
  .set-border(1px; solid; @color-base);
  .class2 {
    background-color: #fff;
    color: @color-base;
    .set-border(1px; solid; #fff);
  }
}

Yukarıdaki kod derlendiğinde aşağıdaki CSS’yi üretecektir: (kısa olması için yorumlar kaldırılmıştır)

.class1 {
  background-color: #87ceeb;
  border: 1px solid #5bbce4;
}
.class1 .class2 {
  background-color: #fff;
  color: #87ceeb;
  border: 1px solid #e6e6e6;
}

Komut satırından Daha Az dosya derleme #

lessc [options] <source> [destination]

Yukarıdaki komut, komut satırında Daha Az dosya derlemek için kullanılır. Seçenekler, derleyicinin derleme sırasında veya derlemeden sonra kullanması gereken çeşitli ayarlardır. Seçenekler, çıktı CSS dosyasını sıkıştırmak veya küçültmek için ‘-x’ veya ‘–compress’, yalnızca parantez içindeki değerlere matematik işlemleri uygulamak için ‘-sm=on’ veya ‘–strict-math=on’ seçeneklerini içerir. sonra derlenmesi gereken kaynak Less dosyasının yolu gelir. Hedef, çıktı dosyasının yolu ve adıdır. Bu sağlanmazsa, çıktı komut satırı penceresinin kendisinde yazdırılır.

Aşağıdaki Daha az kodu düşünün

/* Filename: test.less */
#demo {
 color: @color;
 background: beige;
 width: 100% / 4;
}
@color: red;

Komut penceresinde derlenmiş CSS’yi yazdır:

Komut satırında aşağıdaki komut çalıştırıldığında, test.less dosyası derlenecek ve çıktı, hedef yol sağlanmadığı için doğrudan komut penceresinde yazdırılacaktır.

lessc test.less

Çıktı:

#demo {
  color: red;
  background: beige;
  width: 25%;
}

Bir CSS dosyası oluşturun ve dosyaya derlenmiş çıktı yazın:

Aynı dosya, aşağıdaki ifadeyle derlendiğinde, test.less dosyasıyla aynı yolda test.css adında bir dosya oluşturacak ve çıktıyı bu CSS dosyasına yazdıracak/yazacaktır.

lessc test.less > test.css

Bir CSS dosyası oluşturun ve dosyayı küçültün:

Aşağıdaki komut, çıktıyı bir CSS dosyasına yazdıracak/yazacak ve ayrıca sonunda sıkıştıracaktır.

lessc -x test.less > test.css

Çıktı:

#demo{color:red;background:beige;width:25%}

Strict Math seçeneği etkinken:

Kesin eşleşme seçeneği etkinleştirildiğinde, “width” değerleri parantez içine alınmadığından çıktı aşağıdaki gibi olacaktır.

lessc -sm=on test.less > test.css

Çıktı:

#demo {
  color: red;
  background: beige;
  width: 100% / 4;
}

Daha Az Yerleştirme #

Less’te çok daha basit CSS kuralları yazabilir ve bunları iyi biçimlendirilmiş tutabilirsiniz, bu nedenle bu kodu yazmak yerine:

CSS

.item {
  border: 1px solid;
  padding: 4px;
}
.item .content, .item .image {
  float: left;
}
.item .content {
  font-size: 12px;
}
.item .image {
  width: 300px;
}

sadece şunu yazabilirsiniz:

Az

.item {
  border: 1px solid;
  padding: 4px;
  .content, .image {
    float: left;
  }
  .content {
    font-size: 12px;
  }
  .image {
    width: 300px;
  }
}

ve Less bu kodu hepimizin bildiği normal CSS’de derleyecektir.

Dosyaları Birleştirme - İçe Aktarma #

@import ifadesi, başka bir dosyadan CSS/Less kodunu kendi CSS/Less dosyanıza eklemenize olanak tanır.

.foo {
  background: #900;
}
@import "my-other-css-file.css";
@import "my-other-less-file.less";
