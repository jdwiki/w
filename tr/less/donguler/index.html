Her bir döngü için yazma #

Less’teki her bir döngü için, aşağıdaki farklar dışında, for döngüsüyle aynı temel bileşenlere sahiptir:

  • Üzerinde yinelenmesi gereken öğelerin listesini içeren bir değişken.
  • Döngünün dizinine göre değişkendeki her bir öğeyi çıkarmak için bir ’extract()’ işlevi.
  • Dizinin uzunluğunu (yani, listedeki öğelerin sayısını) hesaplamak ve bunu birincil karıştırma çağrısında ("[başlatma]" için) kullanmak için bir “length()” işlevi.

Aşağıda, Less ile yazılmış, “@images” değişkenindeki her öğeyi yineleyen, “id” öğesinin öğe/resim adıyla aynı olduğu bir “#id” seçici oluşturan ve ayrıca arka planı ayarlayan her döngü için bir örnek verilmiştir. Bunun için görüntü özelliği.

@images: cat, dog, apple, orange; /* the array list of items */

.for-each-loop(@index) when (@index > 0) { /* recursive mixin call with guard - condition */
  
  @image: extract(@images, @index); /* extract function to fetch each item from the list */
  
 /* the statement */
  #@{image} {
    background-image: url("http://mysite.com/@{image}.png");
  }
  /* end of the statement */

  .for-each-loop(@index - 1); /* the next iteration's call - final-expression */
}

.for-loop(length(@images)); /* the primary call with length() function - initialization */

Derlenmiş CSS:

#orange {
  background-image: url("http://mysite.com/orange.png");
}
#apple {
  background-image: url("http://mysite.com/apple.png");
}
#dog {
  background-image: url("http://mysite.com/dog.png");
}
#cat {
  background-image: url("http://mysite.com/cat.png");
}

Basit bir for döngüsü yazma #

Döngülerin kullanımı, kodu DRY tutmanın ve tekrardan kaçınmanın mükemmel bir yoludur. Sass’tan farklı olarak, Less for write loops’da yerleşik ‘@for’ veya ‘@each’ yönergesi yoktur, ancak yine de özyinelemeli karışımlar kullanılarak yazılabilir. Özyinelemeli bir karışım, sürekli kendini çağıran bir karışımdan başka bir şey değildir.

Less kullanılarak yazılan bir döngünün dört temel bileşeni vardır ve bunlar aşağıdaki gibidir:

  • Muhafız ifadeleri ile bir karışım. Koruma, döngünün çıkış kriterleri karşılandığında döngüyü sonlandırmak için kullanılır. Döngü için JavaScript (for([initialization]; [condition]; [final-expression])) açısından, koruma [Koşul]dur.
  • İlk yinelemeyi yürütmek için karışıma birincil çağrı. Mixin’e yapılan bu birincil çağrı, bir seçici bloğun içinden (eğer mixin tüm içeriğini saran bir seçiciye sahip değilse) veya bir seçici bloğun dışından (eğer mixin içeriğini saran bir seçiciye sahipse) yapılabilir. Döngü için JavaScript açısından, bu birincil çağrı, sayaç benzeri değişken için temel değeri belirlediğinden “[başlatma]” işlevi görür.
  • Karıştırmayı özyinelemeli hale getirmek için kendi içinden bir çağrı. Bu çağrı, tipik olarak, argüman olarak sayaç değişkeninin artan veya azalan bir değerini iletir. Böylece sonraki yinelemeleri çağırır. JS for döngüsü açısından, bu, sonraki çağrıyla birlikte “[son ifade]“yi yapar.
  • Son olarak, tipik bir for döngüsü sözdizimindeki ‘ifadeye’ eşdeğer olan karışımın diğer içerikleri.

Aşağıda, birden çok “#img*” seçicisi (burada * bir sayıdır) oluşturan ve ayrıca “background-image” özelliğini “image*.png” olarak ayarlayan Less ile yazılmış basit bir for döngüsü verilmiştir.

.for-loop(@index) when (@index > 0) { /* recursive mixin with guard expression - condition */

  /* the statement */
  #img@{index} {
    background-image: url("http://mysite.com/image@{index}.png");
  }
  /* end of the statement */

  .for-loop(@index - 1); /* the next iteration's call - final-expression*/
}
.for-loop(3); /* the primary call - initialization */

Derlenmiş CSS:

#img3 {
  background-image: url("http://mysite.com/image3.png");
}
#img2 {
  background-image: url("http://mysite.com/image2.png");
}
#img1 {
  background-image: url("http://mysite.com/image1.png");
}
