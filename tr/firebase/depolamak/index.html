Firebase Storage, ağ kalitesinden bağımsız olarak Firebase uygulamalarınız için güvenli dosya yüklemeleri ve indirmeleri sağlar. Görüntüleri, sesleri, videoları veya kullanıcı tarafından oluşturulan diğer içeriği depolamak için kullanabilirsiniz. Firebase Storage, güçlü, basit ve uygun maliyetli bir nesne depolama hizmeti olan Google Cloud Storage tarafından desteklenir.

Firebase Storage, dosyalarınızı varsayılan Google App Engine uygulamasıyla paylaşılan bir Google Cloud Storage paketinde depolayarak hem Firebase hem de Google Cloud API’leri aracılığıyla erişilebilir hale getirir. Bu, Firebase aracılığıyla mobil istemcilerden dosya yükleme ve indirme esnekliğine ve Google Cloud Platform’u kullanarak görüntü filtreleme veya video kod dönüştürme gibi sunucu tarafı işlemleri yapmanıza olanak tanır. Firebase Storage otomatik olarak ölçeklenir, yani Firebase Storage’dan Google Cloud Storage’a veya başka bir sağlayıcıya geçiş yapmaya gerek yoktur.

Bu entegrasyon, dosyalara doğrudan Google Cloud Storage gcloud istemci kitaplıklarından erişilebilmesini sağlar, böylece Firebase Storage’ı favori sunucu tarafı dillerinizle birlikte kullanabilirsiniz. Daha fazla kontrol için Google Cloud Storage XML ve JSON API’lerini de kullanabilirsiniz.

Firebase Storage, kullanıcıları tanımlamak için Firebase Authentication ile sorunsuz bir şekilde bütünleşir ve tek tek dosyalar veya dosya grupları üzerinde erişim kontrolleri ayarlamanıza izin veren bildirime dayalı bir güvenlik dili sağlar, böylece dosyaları istediğiniz kadar genel veya özel yapabilirsiniz.

En güncel API’ler, örnekler ve örnek uygulamalar için Firebase Storage için genel belgelere bakın.

iOS’a başlarken #

Önkoşullar #

  1. Firebase Konsolunda yeni bir proje oluşturun ve bu projeye bir iOS uygulaması ekleyin.
  2. GoogleServices-Info.plist dosyasını indirin ve uygulamanıza ekleyin.

Uygulamanıza Firebase Storage ekleyin #

Projenizin ‘Podfile’ dosyasına aşağıdaki bağımlılığı ekleyin:

pod 'Firebase/Storage'

‘pod install’ı çalıştırın ve oluşturulan ‘.xcworkspace’ dosyasını açın.

Firebase’i CocoaPod’lar olmadan kurmak için bu talimatları izleyin

Firebase Storage’ı kurun #

Herhangi bir Firebase uygulama referansı oluşturulmadan önce Firebase’i başlatmalısınız veya Kullanılmış. Bunu başka bir Firebase özelliği için zaten yaptıysanız, atlayabilirsiniz aşağıdaki iki adım.

Firebase modülünü içe aktarın:

// Obj-C
@import Firebase;

// Swift
import Firebase

Genellikle uygulamanızın “application:didFinishLaunchingWithOptions:” yönteminde bir “FIRApp” paylaşılan örneğini yapılandırın:

// Obj-C
[FIRApp configure];  

// Swift
FIRApp.configure()

Varsayılan Firebase Uygulamasını kullanarak depolama hizmetine bir referans alın:

// Obj-C
FIRStorage *storage = [FIRStorage storage];  

// Swift
let storage = FIRStorage.storage()

Firebase Storage’da bir dosyaya referans oluşturun:

// Obj-C
FIRStorageReference *reference = [[storage reference] child:@"path/to/file.txt"];

// Swift
let reference = storage.reference().child("path/to/file.txt")

Firebase Storage’a bir dosya yükleyin:

// Obj-C
NSData *data = ... 
FIRStorageUploadTask *uploadTask = [riversRef putData:data metadata:nil completion:^(FIRStorageMetadata *metadata, NSError *error) {
  if (error != nil) {
    // Uh-oh, an error occurred!
  } else {
    // Metadata contains file metadata such as size, content-type, and download URL.
    NSURL downloadURL = metadata.downloadURL;
  }
}];

// Swift
let data: NSData! = ...
let uploadTask = riversRef.putData(data, metadata: nil) { metadata, error in
  if (error != nil) {
    // Uh-oh, an error occurred!
  } else {
    // Metadata contains file metadata such as size, content-type, and download URL.
    let downloadURL = metadata!.downloadURL
  }
}
