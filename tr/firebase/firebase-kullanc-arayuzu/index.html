Firebase Kullanıcı Arayüzü

  • FirebaseUI’ye Başlarken

Bu sayfada

  • FirebaseUI’ye Başlarken

Firebase, uygulamanızı geliştirmenize, etkileşimli bir kullanıcı tabanı oluşturmanıza ve daha fazla para kazanmanıza yardımcı olmak için tasarlanmış bir entegre ürünler paketidir. Gerçek zamanlı veritabanı, dosya depolama ve kullanıcı kimlik doğrulaması gibi uygulamanızı oluşturmanıza yardımcı olan araçların yanı sıra anında iletme bildirimleri, analizler, kilitlenme raporlaması ve dinamik bağlantılar gibi uygulamanızı büyütmenize ve uygulamanızdan para kazanmanıza yardımcı olacak araçları içerir.

Firebase’i şaheserinizi oluşturmak için kullanabileceğiniz bir dizi Lego tuğlası olarak düşünebilirsiniz. Tıpkı tuğlalar gibi, Firebase de nispeten fikirsizdir, çünkü parçaları birleştirmenin sonsuz sayıda yolu vardır ve size bazı yolların yanlış olduğunu söylemeyeceğiz :)

FirebaseUI, Firebase üzerine kurulmuştur ve geliştiricilere, standart kodu ortadan kaldırmak ve Google’ın en iyi uygulamalarını tanıtmak için Firebase ilkellerinin üzerine basit, özelleştirilebilir ve üretime hazır yerel mobil bağlamalar sağlar

Lego benzetmesinde, FirebaseUI, raftan kaldırabileceğiniz ve ihtiyaçlarınıza göre ayarlayabileceğiniz talimatlar içeren önceden oluşturulmuş bir kit setidir. FirebaseUI açık kaynak olduğundan, FirebaseUI’yi oluşturmak için Firebase’in ayrı bileşenlerini nasıl kullandığımızı görebilirsiniz. FirebaseUI’nin fikir sahibi olması gerekir - size tuğlaların nasıl bir arada olması gerektiğini düşündüğümüzü anlatıyoruz, bu yüzden bazı seçimler yapıyoruz. Ancak FirebaseUI açık kaynak olduğundan, bireysel ihtiyaçlarınıza daha iyi uyması için içeri girebilir ve yaptığımız şeyi değiştirebilirsiniz.

Bir Lego şehri inşa ediyorsanız, sıfırdan başlayıp her binayı elle tasarlamak yerine, önceden yapılmış bir koleksiyondan bir grup ev çekip ihtiyaçlarınıza göre biraz değiştirmeyi tercih edersiniz, değil mi?

FirebaseUI, tam olarak bunu yapmanıza izin verir, bu yüzden onu örnek uygulamalarımıza ve örneklerimize dahil ediyoruz. Geliştiriciler (kendimiz dahil) tembeldir - kodumuzun en iyi şekilde yeniden kullanılmasını ve en özlü örnekleri istiyoruz ve FirebaseUI, geliştirme maliyetinin çok altında gerçekten iyi kullanıcı deneyimlerine dönüşen gerçekten yüksek kaliteli örnekler sunmamıza izin veriyor.

FirebaseUI’ye Başlarken #

FirebaseUI, Android, iOS ve Web istemcileri sunar. Onlarla şu şekilde başlayabilirsiniz:

Android:

// app/build.gradle

dependencies {
    // Single target that includes all FirebaseUI libraries
    compile 'com.firebaseui:firebase-ui:0.5.2'

    // FirebaseUI Database only
    compile 'com.firebaseui:firebase-ui-database:0.5.2'

    // FirebaseUI Auth only
    compile 'com.firebaseui:firebase-ui-auth:0.5.2'
}

iOS:

# Podfile

# Pull in all Firebase UI features
pod 'FirebaseUI', '~> 0.5'

# Only pull in the "Database" FirebaseUI features
pod 'FirebaseUI/Database', '~> 0.5'

# Only pull in the "Auth" FirebaseUI features (including Facebook and Google)
pod 'FirebaseUI/Auth', '~> 0.5'

# Only pull in the "Facebook" login features
pod 'FirebaseUI/Facebook', '~> 0.5'

# Only pull in the "Google" login features
pod 'FirebaseUI/Google', '~> 0.5'

Ağ:

<!--Include FirebaseUI sources in HTML-->

<script src="https://www.gstatic.com/firebasejs/ui/live/0.5/firebase-ui-auth.js"></script>
<link type="text/css" rel="stylesheet" href="https://www.gstatic.com/firebasejs/ui/live/0.5/firebase-ui-auth.css" />
