Düğümde Hello World Firebase Gerçek Zamanlı Veritabanı #

Sistem gereksinimleri:

  • Düğüm JS

Başlarken

  1. Önce Firebase Konsolu’na gidin ve Yeni Proje Oluşturun.

  2. Projeyi oluşturduktan sonra, projede sol kenar çubuğunda proje Adının yanında ayarlar simgesine tıklayın ve İzinler’i seçin.

  3. İzinler Sayfasında, sol kenar çubuğundaki Hizmet hesapları’na ve ardından Hizmet Hesabı Oluştur’a tıklayın.

  4. Açılır pencerede hizmet hesap adınızı girin ve Hesap Rolü’nü seçin ve Yeni bir özel anahtar sağla’yı seçin ve ardından JSON’u seçin ve Oluştur’u tıklayın (Google Uygulaması Etki Alanı Genelinde Yetkilendirmeyi Etkinleştirmeyi İşaretsiz Bırakın).

  5. Oluştur’a tıkladığınızda, Hesap Kimlik Bilgilerinizle birlikte bir JSON dosyası indirecektir, dosyayı Sisteminizin Herhangi Bir Yerine kaydetmeniz yeterlidir.

  6. Sonraki adım, Firebase Konsolunuzda, Firebase Konsolu’na gidin ve sol kenar çubuğundaki Veritabanı’na tıklayın. Bundan sonra, bazı kukla değere sahip user_data Adı ile yeni bir Veritabanı Nesnesi oluşturun.

  7. Şimdi Firebase Veritabanı projeniz kuruldu, şimdi proje dizininize aşağıdaki kodu kopyalamanız yeterlidir.

    //Loading Firebase Package var firebase = require(“firebase”);

    /**

    • Update your Firebase Project
    • Credentials and Firebase Database
    • URL */ firebase.initializeApp({ serviceAccount: “”, databaseURL: “” }); //by adding your credentials, you get authorized to read and write from the database

    /**

    • Loading Firebase Database and refering
    • to user_data Object from the Database */ var db = firebase.database(); var ref = db.ref("/user_data"); //Set the current directory you are working in

    /**

    • Setting Data Object Value */ ref.set([ { id:20, name:“Jane Doe”, email:“jane@doe.com”, website:“https://jane.foo.bar” }, { id:21, name:“John doe”, email:“john@doe.com”, website:“https://foo.bar” } ]);

    /**

    • Pushing New Value
    • in the Database Object */ ref.push({ id:22, name:“Jane Doe”, email:“jane@doe.com”, website:“https://jane.foo.bar” });

    /**

    • Reading Value from
    • Firebase Data Object */ ref.once(“value”, function(snapshot) { var data = snapshot.val(); //Data is in JSON format. console.log(data); });

  8. JSON Credentials dosyası URL’sini değiştirmeniz yeterlidir (Yeni başlayanlar için, kimlik bilgilerini Aynı klasöre kopyalayın ve index.js dosyasına yalnızca kimlik bilgilerini Dosya Adını ekleyin).

  9. Sonraki adım, index.js’deki gerçek Firebase Veritabanı URL’sini değiştirmektir, bu URL’yi Firebase Konsolunda Veritabanı Sekmesinde bulabileceksiniz, URL, https://.firebaseio.com gibi olacaktır. /.

  10. Son adım yapmaktır

    npm install firebase

  11. Yukarıdaki komutu yürüttükten sonra NPM, Firebase için gerekli paketleri kuracaktır. Sonunda proje yürütmek ve test etmek için

    node index.js

Proje aslında ne yapıyor?

Proje, verileri bulut tabanlı Firebase Veritabanından yükler. Proje ayrıca bir Firebase Veri Nesnesinden nasıl veri Yazılacağını ve Okunacağını da gösterir.

Verilerinizin gerçek zamanlı olarak güncellendiğini görmek için konsolunuz‘a gidin, yaptığınız projeye tıklayın ve sol tarafta Veritabanı’na tıklayın. Burada, verilerinizin değerleriyle birlikte gerçek zamanlı olarak güncellendiğini görebilirsiniz.

Firebase kuyruğu ve çalışanı #

Görevleri veya verileri firebase gerçek zamanlı veritabanına aktarabilir ve bazı arka plan işlemlerini çalıştırmak için firebase kuyruğunu dinleyen bir çalışanı çalıştırabilirsiniz.

Firebase’i kurun

  1. Henüz yoksa, Firebase konsolunda bir Firebase projesi oluşturun. Uygulamanızla ilişkilendirilmiş mevcut bir Google projeniz varsa, Google Projesini İçe Aktar’ı tıklayın. Aksi takdirde, Yeni Proje Oluştur’a tıklayın..

  2. Ayarlar simgesine tıklayın ve İzinler’i seçin.

  3. Soldaki menüden Hizmet hesapları’nı seçin.

  4. Hizmet hesabı oluştur’a tıklayın.

    Enter a name for your service account.

    You can optionally customize the ID from the one automatically generated from the name.

    Choose Project > Editor from the Role dropdown.

    Select Furnish a new private key and leave the Key type as JSON.

    Leave Enable Google Apps Domain-wide Delegation unselected.

    Click Create

  5. Hizmet hesabını oluşturduğunuzda, hizmet hesabınızın kimlik bilgilerini içeren bir JSON dosyası sizin için indirilir. Sunucuda SDK’yı başlatmak için buna ihtiyacınız olacak.

Kurulum sunucusu

nodejs uygulamanızda npm kullanarak firebase-queue yükleyin

npm install firebase firebase-queue --save

Firebase ve firebase-queue’yu kurduktan sonra, yeni bir Queue oluşturarak ve bunu Firebase referansınız ve bir işleme fonksiyonu ile ileterek başlayabilirsiniz.

Şimdi, yeni bir kullanıcı oluşturulduğunda uygulamadan bir firebase kuyruk görevi oluşturalım ve işçiyi firebase-queue görevini dinleyecek ve oluşturulan kullanıcı postasına bir e-posta gönderecek şekilde ayarlayalım.

*server.js

var app=express();
var Queue = require('firebase-queue'),
    Firebase = require('firebase');

Firebase Proje Kimlik Bilgilerinizi ve Firebase Veritabanı URL’nizi güncelleyin

var firebase = Firebase.initializeApp({
    serviceAccount: "path/to/serviceAccountCredentials.json",
  databaseURL: "https://databaseName.firebaseio.com"
});

veya firebase kimlik bilgilerini doğrudan aşağıdaki gibi girebilirsiniz

var firebase = Firebase.initializeApp({ 
    serviceAccount: {
        projectId: "projectId",
        clientEmail: "foo@projectId.iam.gserviceaccount.com",
        privateKey: "-----BEGIN PRIVATE KEY-----\nkey\n-----END PRIVATE KEY-----\n"
    },
    databaseURL: "https://databaseName.firebaseio.com"
});


var refQueue = firebase.database().ref("queue/tasks");

createUser =  funtion(email, password){
    var user = {
        username: email,
        password: password
    };
    user = new db.users(user);
    user.save(function(err, user){
        if(!err){
            refQueue.push({case: "NEW_USER", data: user});
        }
    })
}

*işçi.js

var Queue = require('firebase-queue'),
    Firebase = require('firebase');

//Update your Firebase Project Credentials and Firebase Database URL by one of the way specified in server.js
var firebase = Firebase.initializeApp({
    serviceAccount: "path/to/serviceAccountCredentials.json",
  databaseURL: "https://databaseName.firebaseio.com"
});

var refQueue = firebase.database().ref("queue");

var queue = new Queue(refQueue, function(data, progress, resolve, reject) {
    switch(data.case){
        case "NEW_USER":
            sendMail(data.data.email);
            console.log("user created");
            //sendMail function is not an inbuilt function and will not work unless you define and implement the function
            break;

    // Finish the task asynchronously
    setTimeout(function() {
        resolve();
    }, 1000);
});
