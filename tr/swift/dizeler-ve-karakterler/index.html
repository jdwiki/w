Sözdizimi #

  • String.characters // String içindeki karakterlerin bir Dizisini döndürür
  • String.characters.count // Karakter sayısını döndürür
  • String.utf8 // Bir String.UTF8View, Dize’deki UTF-8 karakter noktalarını döndürür
  • String.utf16 // Bir String.UTF16View, Dize’deki UTF-16 karakter noktalarını döndürür
  • String.unicodeScalars // Bir String.UnicodeScalarView, String içindeki UTF-32 karakter noktalarını döndürür
  • String.isEmpty // Dize herhangi bir metin içermiyorsa true değerini döndürür
  • String.hasPrefix(String) // Dize argümanın önüne eklenmişse true değerini döndürür
  • String.hasSuffix(String) // Dize argümanın sonuna eklenmişse true değerini döndürür
  • String.startIndex // Dizedeki ilk karaktere karşılık gelen Dizini döndürür
  • String.endIndex // Dizedeki son karakterden sonraki noktaya karşılık gelen Dizini döndürür
  • String.components(separatedBy: String) // Verilen ayırıcı dizeyle ayrılmış alt dizeleri içeren bir dizi döndürür
  • String.append(Karakter) // Karakteri (argüman olarak verilen) String’e ekler

Swift’deki bir ‘String’, bir karakter koleksiyonudur ve buna bağlı olarak, bir Unicode skaler koleksiyonudur. Swift Dizeleri Unicode’a dayandığından, İngilizce ve emoji dışındaki diller de dahil olmak üzere herhangi bir Unicode skaler değeri olabilir.

İki skaler birleşerek tek bir karakter oluşturabileceğinden, bir String’deki skaler sayısı her zaman karakter sayısıyla aynı olmayabilir.

Dizeler hakkında daha fazla bilgi için Swift Programlama Dili ve [String Structure Reference](https: //developer.apple.com/library/tvos/documentation/Swift/Reference/Swift_String_Structure/index.html).

Uygulama ayrıntıları için bkz. “Swift String Design”

Dize ve Karakter Değişmezleri #

String Swift’deki değişmez değerler çift tırnak (") ile sınırlandırılır:

let greeting = "Hello!"  // greeting's type is String

Karakterler, değişmez değer yalnızca bir grafik kümesi içerdiği sürece dize değişmez değerlerinden başlatılabilir:

let chr: Character = "H" // valid
let chr2: Character = "😊" // valid
let chr3: Character = "abc" // invalid - multiple grapheme clusters

Dize İnterpolasyonu #

Dize enterpolasyonu izin verir bir ifadeyi doğrudan bir dize değişmezine enjekte etmek. Bu, dizeler, tam sayılar, kayan nokta sayıları ve daha fazlası dahil olmak üzere her tür değerle yapılabilir.

Sözdizimi bir ters eğik çizgidir ve ardından parantezler şu değeri sarar: \(değer). İşlev çağrıları da dahil olmak üzere herhangi bir geçerli ifade parantez içinde görünebilir.

let number = 5
let interpolatedNumber = "\(number)"  // string is "5"
let fortyTwo = "\(6 * 7)"             // string is "42"

let example = "This post has \(number) view\(number == 1 ? "" : "s")"
// It will output "This post has 5 views" for the above example.
// If the variable number had the value 1, it would output "This post has 1 view" instead.

Özel türler için, dize enterpolasyonunun varsayılan davranışı "\(myobj)" şeklindedir. , print(myobj) tarafından kullanılan gösterimin aynısı olan String(myobj) ile eşdeğerdir. Türünüz için CustomStringConvertible protokolünü uygulayarak bu davranışı özelleştirebilirsiniz.

Swift 3 için, SE-0089, String.init uyarınca <T>(_:), String.init<T>(decoding:) olarak yeniden adlandırıldı.

"\(myobj)" dize enterpolasyonu yeni String.init<T: LosslessStringConvertible>(_:) başlatıcısını tercih edecek, ancak değer “LosslessStringConvertible” değil.

son sürüm

Özel karakterler #

Belirli karakterlerin dize değişmezlerinde kullanılması için özel bir çıkış dizisi gerekir:

Karakter Anlam
\0 boş karakter
\\ düz bir ters eğik çizgi, \
\t sekme karakteri
\v dikey sekme
\r a [satır başı] (https://en.wikipedia.org/wiki/Newline)
\n a satır beslemesi (“yeni satır”)
\" çift tırnak, "
\' tek bir alıntı, ’’'
\u{n} Unicode kod noktası n (onaltılık olarak)

Örnek:

let message = "Then he said, \"I \u{1F496} you!\""

print(message) // Then he said, "I 💖 you!"

Dizeleri birleştir #

Yeni bir dize oluşturmak için dizeleri “+” operatörüyle birleştirin:

let name = "John"
let surname = "Appleseed"
let fullName = name + " " + surname  // fullName is "John Appleseed"

+= kullanarak bir [**mutable**](http://stackoverflow.com/questions/24002092/what-is-the-difference-between-let-and-var-in-swift) dizesine ekleyin bileşik atama operatörü veya bir yöntem kullanarak:

let str2 = "there"
var instruction = "look over"
instruction += " " + str2  // instruction is now "look over there"

var instruction = "look over"
instruction.append(" " + str2)  // instruction is now "look over there"

Değişken bir Dize’ye tek bir karakter ekleyin:

var greeting: String = "Hello"
let exclamationMark: Character = "!"
greeting.append(exclamationMark) 
// produces a modified String (greeting) = "Hello!"

Değişken bir Dize’ye birden çok karakter ekleyin

var alphabet:String = "my ABCs: "
alphabet.append(contentsOf: (0x61...0x7A).map(UnicodeScalar.init)
                                         .map(Character.init) )
// produces a modified string (alphabet) = "my ABCs: abcdefghijklmnopqrstuvwxyz"

appendContentsOf(_:), append(_:) olarak yeniden adlandırıldı.

son sürüm

joinWithSeparator(_:) kullanarak yeni bir dize oluşturmak için bir dizi dizeye katılın:

let words = ["apple", "orange", "banana"]
let str = words.joinWithSeparator(" & ")

print(str)   // "apple & orange & banana"

joinWithSeparator(_:), joined(separator:) olarak yeniden adlandırıldı.

“Ayırıcı” varsayılan olarak boş dizedir, dolayısıyla “[“a”, “b”, “c”].joined() == “abc”`.

son sürüm

Dize Kodlama ve Ayrıştırma #

Bir Swift String, Unicode‘dan yapılmıştır /standart.html) kod noktaları. Birkaç farklı şekilde ayrıştırılabilir ve kodlanabilir.

let str = "ที่👌①!"

Ayrıştırılan Dizeler #

Bir dizenin “karakterleri”, Unicode genişletilmiş grafik kümeleri:

Array(str.characters)  // ["ที่", "👌", "①", "!"]

“unicodeScalars”, bir dize oluşturan Unicode kod noktalarıdır (“ที่"nin bir grafik kümesi olduğuna dikkat edin, ancak 3 kod noktası — 3607, 3637 , 3656 — bu nedenle elde edilen dizinin uzunluğu “karakterler” ile aynı değildir):

str.unicodeScalars.map{ $0.value }  // [3607, 3637, 3656, 128076, 9312, 33]

Dizeleri [UTF-8] (https://en.wikipedia.org/wiki/UTF-8) (“UInt8” dizisi) veya [UTF-16] (https://en) olarak kodlayabilir ve ayrıştırabilirsiniz. .wikipedia.org/wiki/UTF-8). .wikipedia.org/wiki/UTF-16) (“UInt16” dizisi):

Array(str.utf8)   // [224, 184, 151, 224, 184, 181, 224, 185, 136, 240, 159, 145, 140, 226, 145, 160, 33]
Array(str.utf16)  // [3607, 3637, 3656, 55357, 56396, 9312, 33]

Dize Uzunluğu ve Yineleme #

Bir dizenin “karakterleri”, “unicodeScalars”, “utf8” ve “utf16"nın tümü Koleksiyon’lerdir, bu nedenle onların “sayılarını” alabilir ve bunlar üzerinde yineleyebilirsiniz:

// NOTE: These operations are NOT necessarily fast/cheap! 

str.characters.count     // 4
str.unicodeScalars.count // 6
str.utf8.count           // 17
str.utf16.count          // 7

for c in str.characters { // ...
for u in str.unicodeScalars { // ...
for byte in str.utf8 { // ...
for byte in str.utf16 { // ...

Dizeleri inceleyin ve karşılaştırın #

Bir dizenin boş olup olmadığını kontrol edin:

if str.isEmpty {
    // do something if the string is empty
}

// If the string is empty, replace it with a fallback:
let result = str.isEmpty ? "fallback string" : str

İki dizenin eşit olup olmadığını kontrol edin (Unicode kurallı eşdeğerlik anlamında):

"abc" == "def"          // false
"abc" == "ABC"          // false
"abc" == "abc"          // true

// "LATIN SMALL LETTER A WITH ACUTE" == "LATIN SMALL LETTER A" + "COMBINING ACUTE ACCENT"
"\u{e1}" == "a\u{301}"  // true

Bir dizenin başka bir dizeyle başlayıp bitmediğini kontrol edin:

"fortitude".hasPrefix("fort")      // true
"Swift Language".hasSuffix("age")  // true

let aString = "This is a test string."

// first, reverse the String's characters 
let reversedCharacters = aString.characters.reverse()

// then convert back to a String with the String() initializer
let reversedString = String(reversedCharacters)

print(reversedString) // ".gnirts tset a si sihT"

Dizenin Tanımlanmış Bir Kümeden Karakterler içerip içermediğini kontrol edin #

Edebiyat

let letters = CharacterSet.letters

let phrase = "Test case"
let range = phrase.rangeOfCharacter(from: letters)

// range will be nil if no letters is found
if let test = range {
   print("letters found")
}
else {
   print("letters not found")
}
son sürüm 
let letters = NSCharacterSet.letterCharacterSet()

let phrase = "Test case"
let range = phrase.rangeOfCharacterFromSet(letters)

// range will be nil if no letters is found
if let test = range {
   print("letters found")
}
else {
  print("letters not found")
}
son sürüm

Objective-C ‘NSCharacterSet’ sınıfına da köprülenen yeni ‘CharacterSet’ yapısı, önceden tanımlanmış birkaç kümeyi şu şekilde tanımlar:

  • “ondalık basamaklar”
  • büyük harfle yazılmış harfler
  • “alfanümerik”
  • ‘kontrol Karakterleri’
  • “yasadışı Karakterler”
  • ve daha fazlasını NSCharacterSet referansında bulabilirsiniz.

Ayrıca kendi karakter kümenizi de tanımlayabilirsiniz:

let phrase = "Test case"
let charset = CharacterSet(charactersIn: "t")

if let _ = phrase.rangeOfCharacter(from: charset, options: .caseInsensitive) {
   print("yes") 
}
else {
   print("no")
}
son sürüm 
let charset = NSCharacterSet(charactersInString: "t")

if let _ = phrase.rangeOfCharacterFromSet(charset, options: .CaseInsensitiveSearch, range: nil) {
   print("yes")
}
else {
    print("no")
}
son sürüm

Ayrıca aralığı da dahil edebilirsiniz:

let phrase = "Test case"
let charset = CharacterSet(charactersIn: "t")

if let _ = phrase.rangeOfCharacter(from: charset, options: .caseInsensitive, range: phrase.startIndex..<phrase.endIndex)) {
   print("yes") 
}
else {
   print("no")
}
son sürüm

Dize Yineleme #

let string = "My fantastic string"
var index = string.startIndex

while index != string.endIndex {
    print(string[index])
    index = index.successor()
}

Not: “endIndex” dizenin sonundan sonradır (yani “string[string.endIndex]” bir hatadır, ancak “string[string.startIndex]” uygundur). Ayrıca, boş bir dizede (""), string.startIndex == string.endIndex, truedur. Boş bir dizgede startIndex.successor() çağıramayacağınız için boş dizeleri kontrol ettiğinizden emin olun.

son sürüm

Swift 3’te, Dize dizinlerinde artık “successor()”, “predecessor()”, “advancedBy(:)”, “advancedBy(:limit:)” veya “distanceTo(_:)” yoktur.

Bunun yerine, bu işlemler artık indekslerini artırmak ve azaltmaktan sorumlu olan koleksiyona taşınır.

Kullanılabilir yöntemler .index(after:), .index(before:) ve .index(_:, offsetBy:) şeklindedir.

let string = "My fantastic string"
var currentIndex = string.startIndex

while currentIndex != string.endIndex {
    print(string[currentIndex])
    currentIndex = string.index(after: currentIndex)
}

Not: “.index” yöntemiyle karıştırılmaması için “currentIndex"i değişken adı olarak kullanıyoruz.

son sürüm

Ve örneğin, diğer tarafa gitmek istiyorsanız:

var index:String.Index? = string.endIndex.predecessor()

while index != nil {
    print(string[index!])
    if index != string.startIndex {
        index = index.predecessor()
    }
    else {
        index = nil
    }
}

(Ya da önce dizgiyi tersine çevirebilirsiniz, ancak dizginin sonuna kadar gitmeniz gerekmiyorsa, muhtemelen böyle bir yöntemi tercih edersiniz)

son sürüm 
var currentIndex: String.Index? = string.index(before: string.endIndex)

while currentIndex != nil {
    print(string[currentIndex!])
    if currentIndex != string.startIndex {
        currentIndex = string.index(before: currentIndex!)
    }
    else {
        currentIndex = nil
    }

Unicode #

Ayar değerleri #

Unicode’u doğrudan kullanma

var str: String = "I want to visit 北京, Москва, मुंबई, القاهرة, and 서울시. 😊"
var character: Character = "🌍"

Onaltılık değerler kullanma

var str: String = "\u{61}\u{5927}\u{1F34E}\u{3C0}" // a大🍎π
var character: Character = "\u{65}\u{301}" // é = "e" + accent mark

Swift ‘Karakterinin’ birden çok Unicode kod noktasından oluşabileceğini, ancak tek bir karakter gibi göründüğünü unutmayın. Buna Genişletilmiş Grafem Kümesi denir.

Dönüşümler #

Dize –> Altıgen

// Accesses views of different Unicode encodings of `str`
str.utf8
str.utf16
str.unicodeScalars // UTF-32

Onaltılı –> Dize

let value0: UInt8 = 0x61
let value1: UInt16 = 0x5927
let value2: UInt32 = 0x1F34E

let string0 = String(UnicodeScalar(value0)) // a
let string1 = String(UnicodeScalar(value1)) // 大
let string2 = String(UnicodeScalar(value2)) // 🍎

// convert hex array to String
let myHexArray = [0x43, 0x61, 0x74, 0x203C, 0x1F431] // an Int array
var myString = ""
for hexValue in myHexArray {
    myString.append(UnicodeScalar(hexValue))
}
print(myString) // Cat‼🐱

UTF-8 ve UTF-16 için dönüşümün her zaman bu kadar kolay olmadığını unutmayın çünkü emoji gibi şeyler tek bir UTF-16 değeriyle kodlanamaz. Bir vekil çift alır.

Dizeleri Biçimlendirme #

Baştaki Sıfırlar #

let number: Int = 7

let str2 = String(format: "%05d", number) // 00007

Ondalıktan Sonra Sayılar #

let number: Float = 3.14159

Ondalık - Onaltılık #

let number: Int = 13627
let str1 = String(format: "%2X", number) // 353B
let str2 = String(format: "%2x", number) // 353b (notice the lowercase b)

Alternatif olarak, aynı şeyi yapan özel başlatıcı kullanılabilir:

let number: Int = 13627
let str1 = String(number, radix: 16, uppercase: true) //353B
let str2 = String(number, radix: 16) // 353b

Rastgele sayı tabanına sahip bir sayının ondalığı #

let number: Int = 13627
let str1 = String(number, radix: 36) // aij

Radix, “[2, 36]” içindeki “Int"dir.

Baştaki ve sondaki WhiteSpace ve NewLine’ı kaldırın #

let someString = "  Swift Language  \n"
let trimmedString = someString.stringByTrimmingCharactersInSet(NSCharacterSet.whitespaceAndNewlineCharacterSet())
// "Swift Language"

stringByTrimmingCharactersInSet yöntemi, belirli bir karakter kümesinde bulunan String karakterlerinin her iki ucundan kaldırılarak oluşturulan yeni bir dize döndürür.

Ayrıca yalnızca boşluk veya yeni satırı kaldırabiliriz.

Yalnızca boşlukları kaldırma:

let trimmedWhiteSpace = someString.stringByTrimmingCharactersInSet(NSCharacterSet.whitespaceCharacterSet())
// "Swift Language  \n"

Yalnızca yeni satırın kaldırılması:

let trimmedNewLine = someString.stringByTrimmingCharactersInSet(NSCharacterSet.newlineCharacterSet())
// "  Swift Language  "
son sürüm 
let someString = "  Swift Language  \n"

let trimmedString = someString.trimmingCharacters(in: .whitespacesAndNewlines)
// "Swift Language"

let trimmedWhiteSpace = someString.trimmingCharacters(in: .whitespaces)
// "Swift Language  \n"

let trimmedNewLine = someString.trimmingCharacters(in: .newlines)
// "  Swift Language  "
son sürüm
