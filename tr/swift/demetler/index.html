Bir tanımlama grubu türü, parantez içine alınmış virgülle ayrılmış bir tür listesidir.

Bu türler listesi ayrıca öğelerin adlarına sahip olabilir ve bu adları tek tek öğelerin değerlerine atıfta bulunmak için kullanabilir.

Bir öğe adı, hemen ardından iki nokta üst üste (:) gelen bir tanımlayıcıdan oluşur.

Genel kullanım -

Fonksiyonun birden çok değer içeren tek bir Tuple döndürmesini sağlamak için bir fonksiyonun dönüş tipi olarak bir Tuple tipi kullanabiliriz.

Tuple’lar değer türleri olarak kabul edilir. Tuples hakkında daha fazla bilgi belgelerde bulunabilir: [developer.apple.com/library/content/documentation/Swift/Conceptual/Swift_Programming_Language/TheBasics.html](https://developer.apple.com/library/content/documentation/Swift/Conceptual/Swift_Programming_Language/TheBasics.html //apple_ref/doc/uid/TP40014097-CH5-ID329)

Tuple’lar nedir? #

Tuples, birden çok değeri tek bir bileşik değerde gruplandırır. Bir tanımlama grubu içindeki değerler herhangi bir türden olabilir ve birbirleriyle aynı türden olmaları gerekmez.

Tuple’lar, herhangi bir miktardaki değeri gruplayarak oluşturulur:

let tuple = ("one", 2, "three")

// Values are read using index numbers starting at zero
print(tuple.0) // one
print(tuple.1) // 2
print(tuple.2) // three

Ayrıca, tanımlama grubu tanımlandığında bireysel değerler de adlandırılabilir:

let namedTuple = (first: 1, middle: "dos", last: 3)

// Values can be read with the named property
print(namedTuple.first)  // 1
print(namedTuple.middle) // dos

// And still with the index number
print(namedTuple.2)      // 3

Değişken olarak kullanıldıklarında da adlandırılabilirler ve hatta isteğe bağlı değerlere sahip olma yeteneğine sahiptirler:

var numbers: (optionalFirst: Int?, middle: String, last: Int)?

//Later On
numbers = (nil, "dos", 3)

print(numbers.optionalFirst)// nil
print(numbers.middle)//"dos"
print(numbers.last)//3

Bireysel değişkenlere ayrıştırma #

Tuple’lar, aşağıdaki sözdizimi ile bireysel değişkenlere ayrıştırılabilir:

let myTuple = (name: "Some Name", age: 26)
let (first, second) = myTuple

print(first) // "Some Name"
print(second)  // 26

Bu sözdizimi, demetin adsız özelliklere sahip olup olmadığına bakılmaksızın kullanılabilir:

let unnamedTuple = ("uno", "dos")
let (one, two) = unnamedTuple
print(one) // "uno"
print(two) // "dos"

Belirli özellikler, alt çizgi (_) kullanılarak yoksayılabilir:

let longTuple = ("ichi", "ni", "san")
let (_, _, third) = longTuple
print(third) // "san"

Fonksiyonların Dönüş Değeri Olarak Tuples #

Fonksiyonlar tuple’ları döndürebilir:

func tupleReturner() -> (Int, String) {
    return (3, "Hello")
}

let myTuple = tupleReturner()
print(myTuple.0) // 3
print(myTuple.1) // "Hello"

Parametre adları atarsanız, bunlar dönüş değerinden kullanılabilir:

func tupleReturner() -> (anInteger: Int, aString: String) {
    return (3, "Hello")
}

let myTuple = tupleReturner()
print(myTuple.anInteger) // 3
print(myTuple.aString) // "Hello"

Tuple türünüzü adlandırmak için bir typealias kullanma #

Bazen aynı tanımlama grubunu kodunuz boyunca birden çok yerde kullanmak isteyebilirsiniz. Bu, özellikle demetiniz karmaşıksa, hızla dağınık olabilir:

// Define a circle tuple by its center point and radius
let unitCircle: (center: (x: CGFloat, y: CGFloat), radius: CGFloat) = ((0.0, 0.0), 1.0)

func doubleRadius(ofCircle circle: (center: (x: CGFloat, y: CGFloat), radius: CGFloat)) -> (center: (x: CGFloat, y: CGFloat), radius: CGFloat) {
    return (circle.center, circle.radius * 2.0)
}

Belirli bir demet türünü birden fazla yerde kullanıyorsanız, typealias kullanabilirsiniz. /apple_ref/doc/uid/TP40014097-CH34-ID361) tanımlama grubu türünüzü adlandırmak için anahtar kelime.

// Define a circle tuple by its center point and radius
typealias Circle = (center: (x: CGFloat, y: CGFloat), radius: CGFloat)

let unitCircle: Circle = ((0.0, 0.0), 1)

func doubleRadius(ofCircle circle: Circle) -> Circle {
    // Aliased tuples also have access to value labels in the original tuple type.
    return (circle.center, circle.radius * 2.0)
}

Ancak bunu çok sık yaptığınızı düşünüyorsanız, bunun yerine bir struct kullanmayı düşünmelisiniz.

Değerleri değiştirme #

Tuple’lar, geçici değişkenler kullanmadan 2 (veya daha fazla) değişken arasında değerleri değiştirmek için kullanışlıdır.

2 değişkenli örnek #

2 değişken verildi

var a = "Marty McFly"
var b = "Emmett Brown"

değerleri kolayca değiştirebiliriz

(a, b) = (b, a)

Sonuç:

print(a) // "Emmett Brown" 
print(b) // "Marty McFly"

4 değişkenli örnek #

var a = 0
var b = 1
var c = 2
var d = 3

(a, b, c, d) = (d, c, b, a)

print(a, b, c, d) // 3, 2, 1, 0

Anahtarda Kasa Olarak Tuples #

Bir anahtarda demetleri kullanın

let switchTuple = (firstCase: true, secondCase: false)
   
switch switchTuple {   
 case (true, false):
    // do something
 case (true, true):
    // do something
 case (false, true):
    // do something
 case (false, false):
    // do something
}

Veya bir Enum ile kombinasyon halinde Örneğin Beden Sınıfları ile:

    let switchTuple = (UIUserInterfaceSizeClass.Compact, UIUserInterfaceSizeClass.Regular)
    
    switch switchTuple {
    case (.Regular, .Compact):
        //statement
    case (.Regular, .Regular):
        //statement
    case (.Compact, .Regular):
        //statement
    case (.Compact, .Compact):
        //statement
    }
