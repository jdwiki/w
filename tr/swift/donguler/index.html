Sözdizimi #

  • dizideki sabitler için { ifadeler }
  • koşulun {ifadeler} olduğu sırayla sabit için
  • dizideki var değişken için { deyimler }
  • sırayla _ için { ifadeler }
  • durum için {ifadeler} dizisinde sabite izin ver
  • durum için, koşulun {ifadeler} olduğu sırayla sabit kalsın
  • dizideki durum var değişkeni için { ifadeler }
  • while koşulu {ifadeler}
  • koşul sırasında { ifadeleri } tekrarlayın
  • sequence.forEach(gövde: (Element) atışları -> Void)

For-in döngüsü #

içinde döngüsü, herhangi bir dizi üzerinde yineleme yapmanızı sağlar.

Bir aralıkta yineleme #

Hem yarı açık hem de kapalı aralıkları yineleyebilirsiniz:

for i in 0..<3 {
    print(i)
}

for i in 0...2 {
    print(i)
}

// Both print:
// 0
// 1
// 2

Bir dizi veya küme üzerinde yineleme #

let names = ["James", "Emily", "Miles"]

for name in names {
   print(name)
}

// James
// Emily
// Miles

Dizideki her bir öğe için indekse ihtiyacınız varsa, SequenceType üzerinde enumerate() yöntemini kullanabilirsiniz.

for (index, name) in names.enumerate() {
   print("The index of \(name) is \(index).")
}

// The index of James is 0.
// The index of Emily is 1.
// The index of Miles is 2.

“enumerate()”, 0’dan başlayarak ardışık “Int"lere sahip eleman çiftlerini içeren tembel bir dizi döndürür. Bu nedenle dizilerde, bu sayılar her bir elemanın verilen indeksine karşılık gelir – ancak bu diğer türlerde geçerli olmayabilir. koleksiyonlarından.

son sürüm

Swift 3’te, ’enumerate()’ ’enumerated()’ olarak yeniden adlandırıldı:

for (index, name) in names.enumerated() {
   print("The index of \(name) is \(index).")
}
son sürüm

Sözlük üzerinde yineleme #

let ages = ["James": 29, "Emily": 24]

for (name, age) in ages {
    print(name, "is", age, "years old.")
}

// Emily is 24 years old.
// James is 29 years old.

Tersine yineleme #

Herhangi bir sırayı tersten yinelemek için SequenceType üzerinde reverse() yöntemini kullanabilirsiniz:

for i in (0..<3).reverse() {
    print(i)
}

for i in (0...2).reverse() {
    print(i)
}

// Both print:
// 2
// 1
// 0

let names = ["James", "Emily", "Miles"]

for name in names.reverse() {
    print(name)
}

// Miles
// Emily
// James
son sürüm

Swift 3’te ‘reverse()’, [‘reversed()’][5] olarak yeniden adlandırıldı:

for i in (0..<3).reversed() {
    print(i)
}
son sürüm

Özel adımlarla aralıklar üzerinde yineleme #

[Strideable][8] üzerinde [stride(_:_:)][7] yöntemlerini kullanarak, özel bir adımla bir aralıkta yineleme yapabilirsiniz:

for i in 4.stride(to: 0, by: -2) {
    print(i)
}

// 4
// 2

for i in 4.stride(through: 0, by: -2) {
    print(i)
}

// 4
// 2
// 0
son sürüm

Swift 3’te, “Stridable” üzerindeki “stride(::)” yöntemleri global [stride(_:_:_:)][9] işlevleriyle değiştirildi:

for i in stride(from: 4, to: 0, by: -2) {
    print(i)
}

for i in stride(from: 4, through: 0, by: -2) {
    print(i)
}
son sürüm

[5]: http://swiftdoc.org/v3.0/protocol/Sequence/#func–ters çevrilmiş [6]: http://swiftdoc.org/v3.0/protocol/Sequence/ [7]: http://swiftdoc.org/v2.2/protocol/Strideable/#func–stride-through_by_ [8]: http://swiftdoc.org/v2.2/protocol/Strideable/ [9]: http://swiftdoc.org/v3.0/func/stride/#func-stride-t_-strideable-from_-t-to_-t-by_-t-stride

Tekrarlama döngüsü #

while döngüsüne benzer şekilde, döngüden sonra sadece kontrol ifadesi değerlendirilir. Bu nedenle, döngü her zaman en az bir kez yürütülür.

var i: Int = 0

repeat {
   print(i)
   i += 1
} while i < 3

// 0
// 1
// 2

Filtrelemeli for-in döngüsü #

  1. where maddesi

Bir “where” yan tümcesi ekleyerek, yinelemeleri verilen koşulu karşılayanlarla sınırlayabilirsiniz.

for i in 0..<5 where i % 2 == 0 {
    print(i)
}

// 0
// 2
// 4


let names = ["James", "Emily", "Miles"]

for name in names where name.characters.contains("s") {
    print(name)
}

// James
// Miles
  1. case maddesi

Yalnızca bazı kalıplarla eşleşen değerler üzerinden yineleme yapmanız gerektiğinde kullanışlıdır:

let points = [(5, 0), (31, 0), (5, 31)]
for case (_, 0) in points {
    print("point on x-axis")
}

//point on x-axis
//point on x-axis

Ayrıca isteğe bağlı değerleri filtreleyebilir ve uygunsa bunları sabit bağlamadan sonra ? işareti ekleyerek açabilirsiniz:

let optionalNumbers = [31, 5, nil]
for case let number? in optionalNumbers {
    print(number)
}

//31    
//5

Her blok için Sıra Türü #

SequenceType protokolüne uyan bir tür, bir kapatma içindeki öğelerini yineleyebilir:

collection.forEach { print($0) }

Aynısı, adlandırılmış bir parametreyle de yapılabilir:

collection.forEach { item in
    print(item)
}

*Not: Bu bloklarda kontrol akış deyimleri (break veya Continue gibi) kullanılamaz. Bir dönüş çağrılabilir ve çağrılırsa, mevcut yineleme için bloğu hemen döndürür (devam etmek gibi). Bir sonraki yineleme daha sonra yürütülür.

let arr = [1,2,3,4]

arr.forEach {
    
    // blocks for 3 and 4 will still be called
    if $0 == 2 {
        return
    }
}

döngü sırasında #

Koşul doğru olduğu sürece bir ‘while’ döngüsü yürütülür.

var count = 1

while count < 10 {
    print("This is the \(count) run of the loop")
    count += 1
}

Döngüyü kırmak #

Bir döngü, koşulu doğru kaldığı sürece yürütülür, ancak break anahtar sözcüğünü kullanarak onu manuel olarak durdurabilirsiniz. Örneğin:

var peopleArray = ["John", "Nicole", "Thomas", "Richard", "Brian", "Novak", "Vick", "Amanda", "Sonya"]
var positionOfNovak = 0

for person in peopleArray {
    if person == "Novak" { break }
    positionOfNovak += 1
}

print("Novak is the element located on position [\(positionOfNovak)] in peopleArray.")
//prints out: Novak is the element located on position 5 in peopleArray. (which is true)
