Downcasting #

Bir değişken, tür döküm operatörleri ‘as?’ ve ‘as!’ kullanılarak bir alt türe indirgenebilir.

‘as?’ operatörü bir alt türe dönüştürmeye çalışır. Başarısız olabilir, bu nedenle isteğe bağlı olarak döndürür.

let value: Any = "John"

let name = value as? String
print(name) // prints Optional("John")

let age = value as? Double
print(age) // prints nil

‘as!’ operatörü bir oyuncuyu zorlar. Bir isteğe bağlı döndürmez, ancak döküm başarısız olursa çöker.

let value: Any = "Paul"

let name = value as! String
print(name) // prints "Paul"

let age = value as! Double // crash: "Could not cast value…"

Koşullu açma ile tip döküm operatörlerinin kullanılması yaygındır:

let value: Any = "George"

if let name = value as? String {
    print(name) // prints "George"
}

if let age = value as? Double {
    print(age) // Not executed
}

Swift Dilinde döküm yazın #

Tip Döküm #

Tür atama, bir örneğin türünü kontrol etmenin veya bu örneği kendi sınıf hiyerarşisinde başka bir yerden farklı bir üst sınıf veya alt sınıf olarak ele almanın bir yoludur.

Swift’de tip döküm, is ve operatörleri ile uygulanır. Bu iki operatör, bir değerin türünü kontrol etmek veya bir değeri farklı bir türe dönüştürmek için basit ve anlamlı bir yol sağlar.

Downcasting #

Belirli bir sınıf türünün sabiti veya değişkeni, aslında perde arkasındaki bir alt sınıfın örneğine atıfta bulunabilir. Durumun böyle olduğuna inandığınız yerde, bir tür döküm operatörüyle (as?

Downcasting başarısız olabileceğinden, type cast operatörü iki farklı biçimde gelir. Koşullu form, as?, aşağı çekmeye çalıştığınız türün isteğe bağlı bir değerini döndürür. Zorlanmış biçim, as!, aşağılamayı dener ve sonucu tek bir bileşik eylem olarak zorla-açar.

Downcast’in başarılı olup olmayacağından emin değilseniz, type cast operatörünün (as?) koşullu formunu kullanın. Operatörün bu formu her zaman isteğe bağlı bir değer döndürür ve downcast mümkün değilse değer sıfır olur. Bu, başarılı bir downcast olup olmadığını kontrol etmenizi sağlar.

Tip döküm operatörünün (as!) zorunlu formunu sadece downcast’in her zaman başarılı olacağından emin olduğunuzda kullanın. Operatörün bu formu, yanlış bir sınıf türüne indirmeye çalışırsanız, bir çalışma zamanı hatasını tetikleyecektir. Daha fazla bilgi edinin.

Dizeden Int & Float’a dönüştürme : - #

     let numbers = "888.00"
     let intValue = NSString(string: numbers).integerValue
     print(intValue) // Output - 888

     
     let numbers = "888.00"
     let floatValue = NSString(string: numbers).floatValue
     print(floatValue) // Output : 888.0

Dize Dönüşümüne Kaydır #

    let numbers = 888.00
    let floatValue = String(numbers) 
    print(floatValue) // Output : 888.0

    // Get Float value at particular decimal point 
    let numbers = 888.00
    let floatValue = String(format: "%.2f", numbers) // Here %.2f will give 2 numbers after decimal points we can use as per our need
    print(floatValue) // Output : "888.00"

Tamsayıdan Dize değerine #

    let numbers = 888
    let intValue = String(numbers)
    print(intValue) // Output : "888"

Dize değerine kayar #

    let numbers = 888.00
    let floatValue = String(numbers)
    print(floatValue)

Dize için isteğe bağlı Float değeri #

    let numbers: Any = 888.00
    let floatValue = String(describing: numbers)
    print(floatValue) // Output : 888.0

İsteğe Bağlı Dizeden Int değerine #

    let hitCount = "100"
    let data :AnyObject = hitCount
    let score = Int(data as? String ?? "") ?? 0
    print(score)

JSON’dan aşağı yayın değerleri #

    let json = ["name" : "john", "subjects": ["Maths", "Science", "English", "C Language"]] as [String : Any]
    let name = json["name"] as? String ?? ""
    print(name) // Output : john
    let subjects = json["subjects"] as? [String] ?? []
    print(subjects) // Output : ["Maths", "Science", "English", "C Language"]

İsteğe bağlı JSON’dan aşağı yayın değerleri #

    let response: Any = ["name" : "john", "subjects": ["Maths", "Science", "English", "C Language"]]
    let json = response as? [String: Any] ?? [:]
    let name = json["name"] as? String ?? ""
    print(name) // Output : john
    let subjects = json["subjects"] as? [String] ?? []
    print(subjects) // Output : ["Maths", "Science", "English", "C Language"]

JSON Yanıtını koşullarla yönetin #

    let response: Any = ["name" : "john", "subjects": ["Maths", "Science", "English", "C Language"]] //Optional Response 
    
    guard let json = response as? [String: Any] else {
        // Handle here nil value
        print("Empty Dictionary")
        // Do something here
        return
    }
    let name = json["name"] as? String ?? ""
    print(name) // Output : john
    let subjects = json["subjects"] as? [String] ?? []
    print(subjects) // Output : ["Maths", "Science", "English", "C Language"]

Nil Response’u koşulla yönetin #

    let response: Any? = nil
    guard let json = response as? [String: Any] else {
        // Handle here nil value
        print("Empty Dictionary")
        // Do something here
        return
    }
    let name = json["name"] as? String ?? ""
    print(name) 
    let subjects = json["subjects"] as? [String] ?? []
    print(subjects)

Çıktı : Boş Sözlük #

Anahtarlı döküm #

“switch” ifadesi, farklı türlerde yayınlamayı denemek için de kullanılabilir:

func checkType(_ value: Any) -> String {
    switch value {

    // The `is` operator can be used to check a type
    case is Double:
        return "value is a Double"

    // The `as` operator will cast. You do not need to use `as?` in a `switch`.
    case let string as String:
        return "value is the string: \(string)"

    default:
        return "value is something else"
    }

}

checkType("Cadena")  // "value is the string: Cadena"
checkType(6.28)      // "value is a Double"
checkType(UILabel()) // "value is something else"

Yükseltme #

‘as’ operatörü bir üst tipe yayın yapacaktır. Başarısız olamayacağı için isteğe bağlı döndürmez.

let name = "Ringo"
let value = string as Any  // `value` is of type `Any` now

Alt sınıflamayı içeren bir işlev parametresinde bir downcast kullanma örneği #

Bir alt sınıfın kodunu ve üst sınıfının bir parametresini alan bir fonksiyonun içindeki verileri kullanmak için bir downcast kullanılabilir.

class Rat {
    var color = "white"
}

class PetRat: Rat {
    var name = "Spot"
}

func nameOfRat(🐭: Rat) -> String {
    guard let petRat = (🐭 as? PetRat) else {
        return "No name"
    }
    
    return petRat.name
}

let noName = Rat()
let spot = PetRat()

print(nameOfRat(noName))
print(nameOfRat(spot))
