Sayıları dizelere/dizelerden dönüştürün #

Sayıları dizelere dönüştürmek için Dize başlatıcıları kullanın:

String(1635999)                              // returns "1635999"
String(1635999, radix: 10)                   // returns "1635999"
String(1635999, radix: 2)                    // returns "110001111011010011111"
String(1635999, radix: 16)                   // returns "18f69f"
String(1635999, radix: 16, uppercase: true)  // returns "18F69F"
String(1635999, radix: 17)                   // returns "129gf4"
String(1635999, radix: 36)                   // returns "z2cf"

Veya basit durumlar için [dize enterpolasyonu](https://www.wikiod.com/tr/swift/dizeler-ve-karakterler#Dize ve Karakter Değişmezleri) kullanın:

let x = 42, y = 9001
"Between \(x) and \(y)"  // equivalent to "Between 42 and 9001"

Dizeleri sayılara dönüştürmek için sayısal türlerin başlatıcılarını kullanın:

if let num = Int("42") { /* ... */ }                // num is 42
if let num = Int("Z2cF") { /* ... */ }              // returns nil (not a number)
if let num = Int("z2cf", radix: 36) { /* ... */ }   // num is 1635999
if let num = Int("Z2cF", radix: 36) { /* ... */ }   // num is 1635999
if let num = Int8("Z2cF", radix: 36) { /* ... */ }  // returns nil (too large for Int8)

Sayı türleri ve değişmez değerler #

Swift’in yerleşik sayısal türleri şunlardır:

  • Word boyutunda (mimariye bağlı) imzalı Int ve imzasız UInt.
  • Sabit boyutlu işaretli tamsayılar Int8, Int16, Int32, Int64 ve işaretsiz tamsayılar UInt8, UInt16, UInt32, UInt64.
  • Kayan nokta türleri Float32/Float, [Float64/Double][Double] ve Float80 (yalnızca x86).

Değişmezler #

Sayısal bir hazır bilgi türü bağlamdan çıkarılır:

let x = 42    // x is Int by default
let y = 42.0  // y is Double by default

let z: UInt = 42     // z is UInt
let w: Float = -1    // w is Float
let q = 100 as Int8  // q is Int8

Sayısal değişmezlerdeki rakamları ayırmak için alt çizgi (_) kullanılabilir. Baştaki sıfırlar yoksayılır.

Kayan noktalı değişmezler anlamlı ve üslü kısımlar («anlamlı» e</i) kullanılarak belirtilebilir ondalık için b> «üs»; 0x «önemli» p  «üs» onaltılık sayı için).

Tamsayı değişmez sözdizimi #

let decimal = 10               // ten
let decimal = -1000            // negative one thousand
let decimal = -1_000           // equivalent to -1000
let decimal = 42_42_42         // equivalent to 424242
let decimal = 0755             // equivalent to 755, NOT 493 as in some other languages
let decimal = 0123456789

let hexadecimal = 0x10         // equivalent to 16
let hexadecimal = 0x7FFFFFFF
let hexadecimal = 0xBadFace
let hexadecimal = 0x0123_4567_89ab_cdef

let octal = 0o10               // equivalent to 8
let octal = 0o755              // equivalent to 493
let octal = -0o0123_4567             

let binary = -0b101010         // equivalent to -42
let binary = 0b111_101_101     // equivalent to 0o755
let binary = 0b1011_1010_1101  // equivalent to 0xB_A_D

Kayan noktalı değişmez sözdizimi #

let decimal = 0.0
let decimal = -42.0123456789
let decimal = 1_000.234_567_89

let decimal = 4.567e5               // equivalent to 4.567×10⁵, or 456_700.0
let decimal = -2E-4                 // equivalent to -2×10⁻⁴, or -0.0002
let decimal = 1e+0                  // equivalent to 1×10⁰, or 1.0

let hexadecimal = 0x1p0             // equivalent to 1×2⁰, or 1.0
let hexadecimal = 0x1p-2            // equivalent to 1×2⁻², or 0.25
let hexadecimal = 0xFEEDp+3         // equivalent to 65261×2³, or 522088.0
let hexadecimal = 0x1234.5P4        // equivalent to 0x12345, or 74565.0
let hexadecimal = 0x123.45P8        // equivalent to 0x12345, or 74565.0
let hexadecimal = 0x12.345P12       // equivalent to 0x12345, or 74565.0
let hexadecimal = 0x1.2345P16       // equivalent to 0x12345, or 74565.0
let hexadecimal = 0x0.12345P20      // equivalent to 0x12345, or 74565.0

Yuvarlama #

yuvarlak #

Değeri x.5 yukarı yuvarlayarak en yakın tam sayıya yuvarlar (ancak -x.5’in aşağı yuvarladığına dikkat edin).

round(3.000) // 3
round(3.001) // 3
round(3.499) // 3
round(3.500) // 4
round(3.999) // 4

round(-3.000) // -3
round(-3.001) // -3
round(-3.499) // -3
round(-3.500) // -4  *** careful here ***
round(-3.999) // -4

tavan #

Ondalık değeri olan herhangi bir sayıyı bir sonraki büyük tam sayıya yuvarlar.

ceil(3.000) // 3
ceil(3.001) // 4
ceil(3.999) // 4

ceil(-3.000) // -3
ceil(-3.001) // -3
ceil(-3.999) // -3

zemin #

Ondalık değeri olan herhangi bir sayıyı bir sonraki daha küçük tam sayıya yuvarlar.

floor(3.000) // 3
floor(3.001) // 3
floor(3.999) // 3

floor(-3.000) // -3
floor(-3.001) // -4
floor(-3.999) // -4

Int #

Herhangi bir ondalık değeri bırakarak “Double"ı “Int"e dönüştürür.

Int(3.000) // 3
Int(3.001) // 3
Int(3.999) // 3

Int(-3.000) // -3
Int(-3.001) // -3
Int(-3.999) // -3

Notlar #

  • ‘yuvarlak’, ‘ceil’ ve ‘floor’ hem 64 hem de 32 bit mimariyi işler.

Bir sayısal türü diğerine dönüştürün #

func doSomething1(value: Double) { /* ... */ }
func doSomething2(value: UInt) { /* ... */ }

let x = 42               // x is an Int
doSomething1(Double(x))  // convert x to a Double
doSomething2(UInt(x))    // convert x to a UInt

Tamsayı başlatıcılar, değer taşarsa veya taşarsa bir çalışma zamanı hatası üretir:

Int8(-129.0) // fatal error: floating point value cannot be converted to Int8 because it is less than Int8.min
Int8(-129)   // crash: EXC_BAD_INSTRUCTION / SIGILL
Int8(-128)   // ok
Int8(-2)     // ok
Int8(17)     // ok
Int8(127)    // ok
Int8(128)    // crash: EXC_BAD_INSTRUCTION / SIGILL
Int8(128.0)  // fatal error: floating point value cannot be converted to Int8 because it is greater than Int8.max

Kayan noktalı tamsayıya dönüştürme değerleri sıfıra yuvarlar:

Int(-2.2)  // -2
Int(-1.9)  // -1
Int(-0.1)  //  0
Int(1.0)   //  1
Int(1.2)   //  1
Int(1.9)   //  1
Int(2.0)   //  2

Tam sayıdan kayan sayıya dönüştürme kayıplı olabilir:

Int(Float(1_000_000_000_000_000_000))  // 999999984306749440

Rastgele sayı üretimi #

‘arc4random_uniform(someNumber: UInt32) -> UInt32’

Bu size “0” ile “someNumber - 1” aralığında rastgele tamsayılar verir.

“UInt32” için maksimum değer 4,294,967,295’tir (yani “2^32 - 1”).

Örnekler:

  • Yazı tura

      let flip = arc4random_uniform(2) // 0 or 1

  • Zar atma

      let roll = arc4random_uniform(6) + 1 // 1...6

  • Ekim ayında rastgele gün

      let day = arc4random_uniform(31) + 1 // 1...31

  • 1990’larda rastgele bir yıl

      let year = 1990 + arc4random_uniform(10)

Genel form:

let number = min + arc4random_uniform(max - min + 1)

burada “sayı”, “maks” ve “min”, “UInt32"dir.

Notlar #

  • ‘arc4random’ ile hafif bir modulo önyargısı vardır, bu nedenle ‘arc4random_uniform’ tercih edilir.
  • Bir ‘UInt32’ değerini bir ‘Int’e çevirebilirsiniz, ancak aralık dışına çıkmamaya dikkat edin.

Üs #

Swift’de, yerleşik ‘pow()’ yöntemiyle ‘Çift’leri üsselleştirebiliriz:

pow(BASE, EXPONENT)

Aşağıdaki kodda, taban (5), üs (2)’nin gücüne ayarlanmıştır:

let number = pow(5.0, 2.0) // Equals 25
