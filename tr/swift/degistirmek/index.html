Parametreler #

parametre Ayrıntılar
Test edilecek değer Karşılaştırılacak değişken

Girişinizin olası her değeri için bir vaka sağlayın. Belirtmek istemediğiniz kalan giriş değerlerini kapsamak için bir “varsayılan durum” kullanın. Varsayılan durum son durum olmalıdır.

Varsayılan olarak Swift’deki Switch’ler, bir vaka eşleştirildikten sonra diğer vakaları kontrol etmeye devam etmeyecektir.

Anahtar ve Opsiyonel Özellikler #

Sonucun isteğe bağlı olduğu bazı örnek durumlar.

var result: AnyObject? = someMethod()

switch result {
case nil:
    print("result is nothing")
case is String:
    print("result is a String")
case _ as Double:
    print("result is not nil, any value that is a Double")
case let myInt as Int where myInt > 0:
    print("\(myInt) value is not nil but an int and greater than 0")
case let a?:
    print("\(a) - value is unwrapped")
}

Temel Kullanım #

let number = 3
switch number {
case 1:
    print("One!")
case 2:
    print("Two!")
case 3:
    print("Three!")
default:
    print("Not One, Two or Three")
}

switch ifadeleri ayrıca tamsayılar dışındaki veri türleriyle de çalışır. Herhangi bir veri türüyle çalışırlar. İşte bir dizeyi değiştirmeye bir örnek:

let string = "Dog"
switch string {
case "Cat", "Dog":
  print("Animal is a house pet.")
default:
  print("Animal is not a house pet.")
}

Bu, aşağıdakileri yazdıracaktır:

Animal is a house pet.

Bir Aralığı Eşleştirme #

Switch deyimindeki tek bir durum, bir dizi değerle eşleşebilir.

let number = 20
switch number {
case 0:
    print("Zero")
case 1..<10:
    print("Between One and Ten")
case 10..<20:
    print("Between Ten and Twenty")
case 20..<30:
    print("Between Twenty and Thirty")
default:
    print("Greater than Thirty or less than Zero")
}

Bir anahtarda where ifadesini kullanma #

Where ifadesi, pozitif bir eşleşme için gerekli olan ek kriterleri eklemek için bir switch case eşleşmesinde kullanılabilir. Aşağıdaki örnek yalnızca aralığı değil, aynı zamanda sayının tek mi çift mi olduğunu kontrol eder:

switch (temperature) {
      case 0...49 where temperature % 2 == 0:
        print("Cold and even")

      case 50...79 where temperature % 2 == 0:
        print("Warm and even")

      case 80...110 where temperature % 2 == 0:
        print("Hot and even")

      default:
        print("Temperature out of range or odd")
}

Kısmi eşleştirme #

Switch deyimi kısmi eşleşmeyi kullanır.

let coordinates: (x: Int, y: Int, z: Int) = (3, 2, 5)

switch (coordinates) {
case (0, 0, 0): // 1
  print("Origin")
case (_, 0, 0): // 2
  print("On the x-axis.")
case (0, _, 0): // 3
  print("On the y-axis.")
case (0, 0, _): // 4
  print("On the z-axis.")
default:        // 5
  print("Somewhere in space")
}
  1. Değerin (0,0,0) olduğu durumla tam olarak eşleşir. Bu, 3B uzayın kökenidir.
  2. y=0, z=0 ve herhangi bir x değeriyle eşleşir. Bu, koordinatın x ekseni üzerinde olduğu anlamına gelir.
  3. x=0, z=0 ve herhangi bir y değeriyle eşleşir. Bu, koordinatın onlar ekseni üzerinde olduğu anlamına gelir.
  4. x=0, y=0 ve herhangi bir z değeriyle eşleşir. Bu, koordinatın z ekseni üzerinde olduğu anlamına gelir.
  5. Koordinatların kalanını eşleştirir.

Not: değeri umursamadığınızı belirtmek için alt çizgiyi kullanın.

Değeri yoksaymak istemiyorsanız, bunu switch ifadenizde şu şekilde kullanabilirsiniz:

let coordinates: (x: Int, y: Int, z: Int) = (3, 2, 5)

switch (coordinates) {
case (0, 0, 0):
  print("Origin")
case (let x, 0, 0):
  print("On the x-axis at x = \(x)")
case (0, let y, 0):
  print("On the y-axis at y = \(y)")
case (0, 0, let z):
  print("On the z-axis at z = \(z)")
case (let x, let y, let z):
  print("Somewhere in space at x = \(x), y = \(y), z = \(z)")
}

Burada eksen durumları, ilgili değerleri çıkarmak için let sözdizimini kullanır. kod daha sonra dizeyi oluşturmak için dize enterpolasyonunu kullanarak değerleri yazdırır.

Not: Bu switch deyiminde bir varsayılana ihtiyacınız yoktur. Bunun nedeni, son durumun esasen varsayılan olmasıdır - herhangi bir şeyle eşleşir, çünkü demetin herhangi bir bölümünde herhangi bir kısıtlama yoktur. Switch deyimi, vakalarıyla birlikte tüm olası değerleri tüketiyorsa, varsayılan gerekli değildir.

Daha karmaşık durumları eşleştirmek için let-where sözdizimini de kullanabiliriz. Örneğin:

let coordinates: (x: Int, y: Int, z: Int) = (3, 2, 5)

switch (coordinates) {
case (let x, let y, _) where y == x:
  print("Along the y = x line.")
case (let x, let y, _) where y == x * x:
  print("Along the y = x^2 line.")
default:
break
}

Burada “y eşittir x” ve “y eşittir x kare” çizgilerini eşleştiriyoruz.

Birden Çok Değeri Eşleştirme #

Bir switch ifadesindeki tek bir durum, birden çok değerde eşleşebilir.

let number = 3
switch number {
case 1, 2:
    print("One or Two!")
case 3:
    print("Three!")
case 4, 5, 6:
    print("Four, Five or Six!")
default:
    print("Not One, Two, Three, Four, Five or Six")
}

Anahtarı kullanarak birden çok kısıtlamadan birini yerine getirin #

Bir Tuple oluşturabilir ve şöyle bir anahtar kullanabilirsiniz:

var str: String? = "hi"
var x: Int? = 5

switch (str, x) {
case (.Some,.Some):
    print("Both have values")
case (.Some, nil):
    print("String has a value")
case (nil, .Some):
    print("Int has a value")
case (nil, nil):
    print("Neither have values")
}

Anahtar ve Numaralandırmalar #

Switch ifadesi, Enum değerleriyle çok iyi çalışır

enum CarModel {
    case Standard, Fast, VeryFast
}

let car = CarModel.Standard

switch car {
case .Standard: print("Standard")
case .Fast: print("Fast")
case .VeryFast: print("VeryFast")
}

Her olası araba değeri için bir durum sağladığımızdan, ‘varsayılan’ durumu atlıyoruz.

Anahtarlar ve demetler #

Anahtarlar, demetleri açabilir:

public typealias mdyTuple = (month: Int, day: Int, year: Int)

let fredsBirthday =   (month: 4, day: 3, year: 1973)




switch theMDY
{
//You can match on a literal tuple:
case (fredsBirthday):
  message = "\(date) \(prefix) the day Fred was born"

//You can match on some of the terms, and ignore others:
case (3, 15, _):
  message = "Beware the Ides of March"

//You can match on parts of a literal tuple, and copy other elements
//into a constant that you use in the body of the case:
case (bobsBirthday.month, bobsBirthday.day, let year) where year > bobsBirthday.year:
  message = "\(date) \(prefix) Bob's \(possessiveNumber(year - bobsBirthday.year))" +
    "birthday"

//You can copy one or more elements of the tuple into a constant and then
//add a where clause that further qualifies the case:
case (susansBirthday.month, susansBirthday.day, let year) 
  where year > susansBirthday.year:
  message = "\(date) \(prefix) Susan's " +
    "\(possessiveNumber(year - susansBirthday.year)) birthday"

//You can match some elements to ranges:.
case (5, 1...15, let year):
  message = "\(date) \(prefix) in the first half of May, \(year)"
}

Sınıfa dayalı eşleştirme - hazırlıkForSegue için harika #

Ayrıca, açtığınız şeyin sınıfına dayalı olarak bir switch ifadesi geçişi yapabilirsiniz.

Bunun yararlı olduğu bir örnek ‘prepareForSegue’ içindedir. Segue tanımlayıcısına göre geçiş yapardım, ama bu kırılgan. storyboard’unuzu daha sonra değiştirir ve segue tanımlayıcısını yeniden adlandırırsanız, kodunuzu bozar. Veya, aynı görünüm denetleyici sınıfını (ancak farklı film şeridi sahneleri) birden çok örneğe segue kullanırsanız, hedefin sınıfını bulmak için segue tanımlayıcısını kullanamazsınız.

Hızlı geçiş ifadeleri kurtarmaya.

Swift case let var as Class sözdizimini şu şekilde kullanın:

override func prepareForSegue(segue: UIStoryboardSegue, sender: AnyObject?) {
  switch segue.destinationViewController {
    case let fooViewController as FooViewController:
      fooViewController.delegate = self

    case let barViewController as BarViewController:
      barViewController.data = data

    default:
      break
  }
}
son sürüm

Swift 3’te sözdizimi biraz değişti:

override func prepare(for segue: UIStoryboardSegue, sender: AnyObject?) {       
  switch segue.destinationViewController {
    case let fooViewController as FooViewController:
      fooViewController.delegate = self

    case let barViewController as BarViewController:
      barViewController.data = data

    default:
      break
  }
}
son sürüm

Düşüşleri değiştir #

Swift’de, insanların aşina olduğu diğer dillerden farklı olarak, her case ifadesinin sonunda örtük bir mola olduğunu belirtmekte fayda var. Bir sonraki duruma geçmek için (yani birden fazla vakanın yürütülmesi) için “fallthrough” ifadesini kullanmanız gerekir.

switch(value) {
case 'one':
    // do operation one
    fallthrough
case 'two':
    // do this either independant, or in conjunction with first case
default:
    // default operation
}

bu, akışlar gibi şeyler için kullanışlıdır.
