Paylaşılan Bir Projede SQLite.NET Kullanma #

SQLite.NET, bir ‘Xamarin.Forms’ projesinde ‘SQLite’ sürüm 3’ü kullanarak yerel veritabanları desteği eklemeyi mümkün kılan açık kaynaklı bir kitaplıktır.

Aşağıdaki adımlar, bu bileşenin bir “Xamarin.Forms” Paylaşılan Projesine nasıl dahil edileceğini gösterir:

  1. SQLite.cs sınıfının en son sürümünü indirin ve ekleyin to the Shared Project.

  2. Veritabanına dahil edilecek her tablonun Shared Project’te bir sınıf olarak modellenmesi gerekmektedir. Bir tablo, sınıfa en az iki öznitelik eklenerek tanımlanır: “Table” (sınıf için) ve “PrimaryKey” (bir özellik için).

Bu örnek için, Shared Project’e aşağıdaki gibi tanımlanan “Song” adlı yeni bir sınıf eklenir:

using System;
using SQLite;

namespace SongsApp
{
    [Table("Song")]
    public class Song
    {
        [PrimaryKey]
        public string ID { get; set; }
        public string SongName { get; set; }
        public string SingerName { get; set; }
    }
}

  1. Ardından, ‘SQLiteConnection’ sınıfından (SQLite.cs’de bulunur) miras alan ‘Veritabanı’ adlı yeni bir sınıf ekleyin. Bu yeni sınıfta, her tablo için veritabanı erişimi, tablo oluşturma ve CRUD işlemleri için kod tanımlanmıştır. Örnek kod aşağıda gösterilmiştir:

    using System; using System.Linq; using System.Collections.Generic; using SQLite;

     {
         return Table<Song>().ToList();

     public Song GetSong(string id)

         return Table<Song>().Where(t => t.ID == id).First();
     }

     public bool AddSong(Song song)
     {
         Insert(song);
     }

     public bool UpdateSong(Song song)
     {
         Update(song);
     }

     public void DeleteSong(Song song)
     {
         Delete(song);
     }
 }

    }

  3. Veritabanı mantıkları için bir sınıf oluşturun ve aşağıdaki kod takip edilir.

SQLite.Net’i kullanarak; System.Collections.Generic kullanarak; System.Linq kullanarak; Xamarin.Forms kullanarak; ad alanı DatabaseEmployeeCreation.SqlLite.ViewModel { public class DatabaseLogic { static object locker = new object(); SQLiteConnection database;

    public DatabaseLogic()
    {
        database = DependencyService.Get<ISQLite>().GetConnection();
        // create the tables
        database.CreateTable<Employee>();
    }

    public IEnumerable<Employee> GetItems()
    {
        lock (locker)
        {
            return (from i in database.Table<Employee>() select i).ToList();
        }
    }

    public IEnumerable<Employee> GetItemsNotDone()
    {
        lock (locker)
        {
            return database.Query<Employee>("SELECT * FROM [Employee]");
        }
    }

    public Employee GetItem(int id)
    {
        lock (locker)
        {
            return database.Table<Employee>().FirstOrDefault(x => x.Eid == id);
        }
    }

    public int SaveItem(Employee item)
    {
        lock (locker)
        {
            if (item.Eid != 0)
            {
                database.Update(item);
                return item.Eid;
            }
            else
            {
                return database.Insert(item);
            }
        }
    }

    public int DeleteItem(int Eid)
    {
        lock (locker)
        {
            return database.Delete<Employee>(Eid);
        }
    }
}

}

         /// </summary>
         void ReadWriteStream(Stream readStream, Stream writeStream)
         {
             int Length = 256;
             Byte[] buffer = new Byte[Length];
             int bytesRead = readStream.Read(buffer, 0, Length);
             // write the required bytes
             while (bytesRead > 0)
             {
                 writeStream.Write(buffer, 0, bytesRead);
                 bytesRead = readStream.Read(buffer, 0, Length);
             }
             readStream.Close();
             writeStream.Close();
         }
     }
