TapGestureRecognizer ekleyerek bir Resmi dokunulabilir hale getirin #

Xamarin.Forms’da kullanılabilen birkaç varsayılan tanıyıcı vardır, bunlardan biri ‘TapGestureRecognizer’dır.

Bunları hemen hemen her görsel öğeye ekleyebilirsiniz. Bir “Görüntü"ye bağlanan basit bir uygulamaya bir göz atın. İşte kodda nasıl yapılacağı.

var tappedCommand = new Command(() =>
{
    //handle the tap
});

var tapGestureRecognizer = new TapGestureRecognizer { Command = tappedCommand };
image.GestureRecognizers.Add(tapGestureRecognizer);

Veya XAML’de:

<Image Source="tapped.jpg">
    <Image.GestureRecognizers>
        <TapGestureRecognizer
                Command="{Binding TappedCommand}"
                NumberOfTapsRequired="2" />
  </Image.GestureRecognizers>
</Image>

Burada komut, veri bağlama kullanılarak ayarlanır. Gördüğünüz gibi, harekete geçmeden önce daha fazla dokunma için etkinleştirmek için NumberOfTapsRequired ayarını da yapabilirsiniz. Varsayılan değer 1 dokunuştur.

Diğer hareketler Pinch ve Pan’dır.

Bir Görüntüyü Kıstırma hareketiyle yakınlaştırın #

Bir “Görüntüyü” (veya başka bir görsel öğeyi) yakınlaştırılabilir hale getirmek için ona bir “PinchGestureRecognizer” eklemeliyiz. İşte kodda nasıl yapılacağı:

var pinchGesture = new PinchGestureRecognizer();
pinchGesture.PinchUpdated += (s, e) => {
// Handle the pinch
};
    
image.GestureRecognizers.Add(pinchGesture);

Ancak XAML’den de yapılabilir:

<Image Source="waterfront.jpg">
  <Image.GestureRecognizers>
    <PinchGestureRecognizer PinchUpdated="OnPinchUpdated" />
  </Image.GestureRecognizers>
</Image>

Eşlik eden olay işleyicide, resminizi yakınlaştırmak için kodu sağlamalısınız. Elbette başka kullanımlar da uygulanabilir.

void OnPinchUpdated (object sender, PinchGestureUpdatedEventArgs e)
{
    // ... code here
}

Diğer hareketler ise Tap ve Pan’dır.

PanGestureRecognizer ile yakınlaştırılan tüm Görüntü içeriğini gösterin #

Yakınlaştırılmış bir ‘Görüntü’ (veya başka bir içeriğiniz) olduğunda, tüm içeriğini yakınlaştırılmış durumda göstermek için ‘Görüntü’ etrafında sürüklemek isteyebilirsiniz.

Bu, PanGestureRecognizer’ı uygulayarak başarılabilir. Koddan bu şöyle görünür:

var panGesture = new PanGestureRecognizer();
panGesture.PanUpdated += (s, e) => {
  // Handle the pan
};

image.GestureRecognizers.Add(panGesture);

Bu, XAML’den de yapılabilir:

<Image Source="MonoMonkey.jpg">
  <Image.GestureRecognizers>
    <PanGestureRecognizer PanUpdated="OnPanUpdated" />
  </Image.GestureRecognizers>
</Image>

Arka plan kod olayında artık kaydırma işlemini buna göre yapabilirsiniz. Bunu işlemek için bu yöntem imzasını kullanın:

void OnPanUpdated (object sender, PanUpdatedEventArgs e)
{
  // Handle the pan
}

Kullanıcının MR.Gestures ile ekrana dokunduğu yere bir iğne yerleştirin #

Hareket tanıyıcılarda yerleşik Xamarins, yalnızca çok temel dokunma kullanımı sağlar. Örneğin. dokunan bir parmağın konumunu almanın bir yolu yoktur. MR.Gestures, 14 farklı dokunma işleme olayı ekleyen bir bileşendir. Dokunma parmaklarının konumu, tüm MR.Gestures olaylarına geçirilen “EventArgs"ın bir parçasıdır.

Ekranda herhangi bir yere iğne yerleştirmek istiyorsanız, en kolay yol, “Tapping” olayını işleyen bir “MR.Gestures.AbsoluteLayout” kullanmaktır.

<mr:AbsoluteLayout x:Name="MainLayout" Tapping="OnTapping">
    ...
</mr:AbsoluteLayout>

Gördüğünüz gibi, Tapping="OnTapping" ayrıca iç içe geçmiş GestureRecognizers ile Xamarins sözdiziminden çok .NET’e benziyor. Bu sözdizimi iOS’tan kopyalandı ve .NET geliştiricileri için biraz kokuyor.

Arkadaki kodunuzda şöyle ‘OnTapping’ işleyicisini ekleyebilirsiniz:

private void OnTapping(object sender, MR.Gestures.TapEventArgs e)
{
    if (e.Touches?.Length > 0)
    {
        Point touch = e.Touches[0];
        var image = new Image() { Source = "pin" };
        MainLayout.Children.Add(image, touch);
    }
}

‘Tapping’ olayı yerine, ‘TappingCommand’ı kullanabilir ve ViewModel’inize bağlayabilirsiniz, ancak bu, bu basit örnekte işleri karmaşıklaştıracaktır.

MR.Gestures için daha fazla örnek, [GitHub’daki1 GestureSample uygulamasında ve MR.Gestures web sitesinde bulunabilir. Bunlar ayrıca olay işleyiciler, komutlar, MVVM, … ile diğer tüm dokunma olaylarının nasıl kullanılacağını gösterir.
