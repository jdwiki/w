‘DependencyService’i kullanırken genellikle 3 parçaya ihtiyacınız vardır:

  • Arayüz - Bu, soyutlamak istediğiniz işlevleri tanımlar.
  • Platform uygulaması - Her bir platforma özel projede, önceden tanımlanmış arayüzü uygulayan bir sınıf.
  • Kayıt - Her platforma özel uygulama sınıfı, bir meta veri özniteliği aracılığıyla ‘Bağımlılık Hizmeti’ne kaydedilmelidir. Bu, “DependencyService"in uygulamanızı çalışma zamanında bulmasını sağlar.

‘DependencyService’i kullanırken, hedeflediğiniz her platform için bir uygulama sağlamanız gerekir. Bir uygulama sağlanmadığında uygulama çalışma zamanında başarısız olur.

Android uygulaması #

Android’e özgü uygulama, sizi yerel bir “Java.Lang.Object” öğesinden devralmaya ve “IOnInitListener” arabirimini uygulamaya zorladığı için biraz daha karmaşıktır. Android, sunduğu birçok SDK yöntemi için geçerli bir Android bağlamı sağlamanızı gerektirir. Xamarin.Forms, bu gibi durumlarda kullanabileceğiniz bir Android bağlamı sağlayan bir “Forms.Context” nesnesi sunar.

using Android.Speech.Tts;
using Xamarin.Forms;
using System.Collections.Generic;
using DependencyServiceSample.Droid;

public class TextToSpeechAndroid : Java.Lang.Object, ITextToSpeech, TextToSpeech.IOnInitListener
{
    TextToSpeech _speaker;

    public TextToSpeechAndroid () {}

    public void Speak (string whatToSay)
    {
        var ctx = Forms.Context;

        if (_speaker == null) 
        {
            _speaker = new TextToSpeech (ctx, this);
        } 
        else 
        {
            var p = new Dictionary<string,string> ();
            _speaker.Speak (whatToSay, QueueMode.Flush, p);
        }
    }

    #region IOnInitListener implementation
    
    public void OnInit (OperationResult status)
    {
        if (status.Equals (OperationResult.Success)) 
        {
            var p = new Dictionary<string,string> ();
            _speaker.Speak (toSpeak, QueueMode.Flush, p);
        }
    }
    
    #endregion
}

Sınıfınızı oluşturduğunuzda, çalışma zamanında onu keşfetmek için DependencyServicei etkinleştirmeniz gerekir. Bu, sınıf tanımının üstüne ve herhangi bir ad alanı tanımlarının dışına bir “[montaj]” özniteliği eklenerek yapılır.

using Android.Speech.Tts;
using Xamarin.Forms;
using System.Collections.Generic;
using DependencyServiceSample.Droid;


[assembly: Xamarin.Forms.Dependency (typeof (TextToSpeechAndroid))]
namespace DependencyServiceSample.Droid {
    ...

Bu öznitelik, sınıfı “DependencyService” ile kaydeder, böylece “ITextToSpeech” arabiriminin bir örneğine ihtiyaç duyulduğunda kullanılabilir.

Arayüz #

Arayüz, DependencyService aracılığıyla ortaya çıkarmak istediğiniz davranışı tanımlar. Bir “DependencyService"in örnek bir kullanımı, Text-to-Speech hizmetidir. Şu anda Xamarin.Forms’da bu özellik için bir soyutlama yoktur, bu nedenle kendinizinkini oluşturmanız gerekir. Davranış için bir arayüz tanımlayarak başlayın:

public interface ITextToSpeech
{
       void Speak (string whatToSay);
}

Arayüzümüzü tanımladığımız için paylaşılan kodumuzdan ona karşı kod yazabiliriz.

Not: Arabirimi uygulayan sınıfların “DependencyService” ile çalışması için parametresiz bir kurucuya sahip olması gerekir.

iOS uygulaması #

Tanımladığınız arayüzün hedeflenen her platformda uygulanması gerekiyor. iOS için bu, ‘AVFoundation’ çerçevesi aracılığıyla yapılır. Aşağıdaki “ITextToSpeech” arabirimi uygulaması, belirli bir metnin İngilizce olarak söylenmesini işler.

using AVFoundation;

public class TextToSpeechiOS : ITextToSpeech
{
    public TextToSpeechiOS () {}

    public void Speak (string whatToSay)
    {
        var speechSynthesizer = new AVSpeechSynthesizer ();

        var speechUtterance = new AVSpeechUtterance (whatToSay) {
            Rate = AVSpeechUtterance.MaximumSpeechRate/4,
            Voice = AVSpeechSynthesisVoice.FromLanguage ("en-US"),
            Volume = 0.5f,
            PitchMultiplier = 1.0f
        };

        speechSynthesizer.SpeakUtterance (speechUtterance);
    }
}

Sınıfınızı oluşturduğunuzda, çalışma zamanında onu keşfetmek için DependencyServicei etkinleştirmeniz gerekir. Bu, sınıf tanımının üstüne ve herhangi bir ad alanı tanımlarının dışına bir “[montaj]” özniteliği eklenerek yapılır.

using AVFoundation;
using DependencyServiceSample.iOS;

[assembly: Xamarin.Forms.Dependency (typeof (TextToSpeechiOS))]
namespace DependencyServiceSample.iOS {
    public class TextToSpeechiOS : ITextToSpeech
...

Bu öznitelik, sınıfı “DependencyService” ile kaydeder, böylece “ITextToSpeech” arabiriminin bir örneğine ihtiyaç duyulduğunda kullanılabilir.

Paylaşılan kod #

Platforma özel sınıflarınızı oluşturup kaydettikten sonra, bunları paylaşılan kodunuza bağlamaya başlayabilirsiniz. Aşağıdaki sayfa, önceden tanımlanmış bir cümle kullanarak metinden sese işlevini tetikleyen bir düğme içerir. Yerel SDK’ları kullanarak çalışma zamanında ‘ITextToSpeech’in platforma özel uygulamasını almak için ‘DependencyService’i kullanır.

public MainPage ()
{
    var speakButton = new Button {
        Text = "Talk to me baby!",
        VerticalOptions = LayoutOptions.CenterAndExpand,
        HorizontalOptions = LayoutOptions.CenterAndExpand,
    };
    
    speakButton.Clicked += (sender, e) => {
        DependencyService.Get<ITextToSpeech>().Speak("Xamarin Forms likes eating cake by the ocean.");
    };
    
    Content = speakButton;
}

Bu uygulamayı bir iOS veya Android cihazında çalıştırdığınızda ve düğmeye dokunduğunuzda, uygulamanın verilen cümleyi söylediğini duyacaksınız.
