js düğümü altında temel Express web sunucusu nasıl kurulur ve Express yönlendiricisini Keşfetmek.

Son olarak, Express Router’ı kullanarak uygulamanızdaki yönlendirme özelliğini kullanabilirsiniz ve uygulaması kolaydır.

Ekspres Web Sunucusu Yönlendirme #

Hızlı Web Sunucusu Oluşturma

Ekspres sunucu kullanışlı oldu ve birçok kullanıcı ve topluluk aracılığıyla derinleşiyor. Popüler oluyor.

Bir Ekspres Sunucu oluşturalım. Paket Yönetimi ve Bağımlılık Esnekliği için NPM’yi (Düğüm Paket Yöneticisi) kullanacağız.

  1. Proje dizinine gidin ve package.json dosyasını oluşturun. package.json { “ad”: “expressRouter”, “sürüm”: “0.0.1”, “Kodlar”: { “start”: “node Server.js” }, “bağımlılıklar”: { “express”: “^4.12.3” } }

  2. Dosyayı kaydedin ve aşağıdaki npm install komutunu kullanarak ekspres bağımlılığı yükleyin. Bu, proje dizininizde gerekli bağımlılıkla birlikte node_modules yaratacaktır.

  3. Ekspres Web Sunucusunu oluşturalım. Go to the Project directory and create server.js file. sunucu.js

    var express = require(“express”); var app = express();

// Router() nesnesi oluşturuluyor

var yönlendirici = express.Router();

// Tüm rotaları burada belirtin, bu Ana sayfa içindir.

router.get("/",function(req,res){
res.json({"message" : "Hello World"});

});

app.use("/api",yönlendirici);

// Bu Portu Dinle

app.listen(3000,function(){ console.log(“Live at Port 3000”); });

For more detail on setting node server you can see [here][1].

  1. Aşağıdaki komutu yazarak sunucuyu çalıştırın.

    node server.js

    If Server runs successfully, you will se something like this..

  2. Şimdi tarayıcıya veya postacıya gidin ve bir istekte bulunun

    http://localhost:3000/api/

    The output will be .

Hepsi bu, Ekspres yönlendirmenin temeli.

Şimdi GET, POST vb. ile ilgilenelim.

yous server.js dosyasını aşağıdaki gibi değiştirin

var express = require("express");
var app = express();

//Creating Router() object

var router = express.Router();

// Router middleware, mentioned it before defining routes.

router.use(function(req,res,next) {
  console.log("/" + req.method);
  next();
});

// Provide all routes here, this is for Home page.

router.get("/",function(req,res){
  res.json({"message" : "Hello World"});
});


app.use("/api",router);


app.listen(3000,function(){
  console.log("Live at Port 3000");
});

Şimdi sunucuyu yeniden başlatırsanız ve istekte bulunursanız

http://localhost:3000/api/

gibi bir şey göreceksiniz

Yönlendirmede Parametreye Erişim

Parametreye url’den de erişebilirsiniz, örneğin http://example.com/api/:name/. Böylece name parametresine erişim sağlanabilir. Aşağıdaki kodu server.js’nize ekleyin

router.get("/user/:id",function(req,res){
  res.json({"message" : "Hello "+req.params.id});
});

Şimdi sunucuyu yeniden başlatın ve http://localhost:3000/api/user/Adem adresine gidin, çıktı gibi olacaktır.
