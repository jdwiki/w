Burada Node.js’nin tarihçesi, sürüm bilgileri ve mevcut durumu hakkında tartışacağız.

Her yılki önemli olaylar #

2009 #

  • 3 Mart : Projeye “node” adı verildi
  • 1 Ekim: Npm’nin ilk çok erken önizlemesi, Düğüm paketi yönetici
  • 8 Kasım: Ryan Dahl’ın (Node.js’nin Yaratıcısı) Orijinali JSConf 2009’da Node.js Konuşması

2010 #

  • Ekspres: Bir Node.js web geliştirme çerçevesi
  • Socket.io ilk sürümü
  • 28 Nisan: Heroku’da Deneysel Node.js Desteği
  • 28 Temmuz : Ryan Dahl’ın Node.js’deki Google Tech Talk
  • 20 Ağustos: Node.js 0.2.0 yayınlandı

2011 #

  • 31 Mart : Node.js Kılavuzu
  • 1 Mayıs: npm 1.0: Yayınlandı
  • 1 Mayıs : Ryan Dahl’ın Reddit’teki AMA’sı
  • 10 Temmuz : Node.js’ye giriş niteliğindeki Node Başlangıç ​​Kitabı tamamlandı.
    • A comprehensive Node.js tutorial for beginners.
  • 16 Ağustos: LinkedIn, Node.js kullanır
    • LinkedIn launched its completely overhauled mobile app with new features and new parts under the hood.
  • 5 Ekim : Ryan Dahl, Node.js’nin tarihini ve onu neden yarattığını anlatıyor
  • 5 Aralık: Uber’de üretimde olan Node.js
    • Uber Engineering Manager Curtis Chambers explains why his company completely re-engineered their application using Node.js to increase verimliliği artırın ve iş ortağı ve müşteri deneyimini iyileştirin.

2012 #

  • 30 Ocak : Node.js yaratıcısı Ryan Dahl, Node’un günlük hayatından uzaklaşıyor
  • 25 Haziran: Node.js v0.8.0 [kararlı] çıktı
  • 20 Aralık: Hapi, bir Node.js çerçevesi yayınlandı

#2013 #

  • 30 Nisan: ORTALAMA Yığın: MongoDB, ExpressJS, AngularJS ve Node.js
  • 17 Mayıs: eBay’in İlk Node.js Uygulamasını Nasıl Oluşturduk
  • 15 Kasım: PayPal, bir Node.js çerçevesi olan Kraken’i piyasaya sürdü
  • 22 Kasım: Walmart’ta Node.js Bellek Sızıntısı
    • Eran Hammer of Wal-Mart labs came to the Node.js core team complaining of a memory leak he had been tracking down for months.
  • 19 Aralık: Koa - Node.js için web çerçevesi

2014 #

  • 15 Ocak: TJ Fontaine, Node projesini devraldı
  • 23 Ekim : Node.js Danışma Kurulu
    • Joyent and several members of the Node.js community announced a proposal for a Node.js Advisory Board as a next step towards a fully open Node.js açık kaynak projesi için yönetim modeli.
  • 19 Kasım: Alev Grafiklerinde Node.js - Netflix
  • 28 Kasım : IO.js – V8 Javascript için Etkinleştirilmiş I/O

2015 #

Q1 #

  • 14 Ocak: IO.js 1.0.0
  • 10 Şubat : Joyent, Node.js Vakfını Kurmak İçin Hareket Ediyor
    • Joyent, IBM, Microsoft, PayPal, Fidelity, SAP and The Linux Foundation Join Forces to Support Node.js Community With Neutral and Open Governance
  • 27 Şubat : IO.js ve Node.js mutabakat teklifi

2. Çey #

  • 14 Nisan: npm Özel Modüller
  • 28 Mayıs : [Düğüm lideri TJ Fontaine istifa ediyor ve Joyent’ten ayrılıyor](http://venturebeat.com/2015/05/08/node-lead-tj-fontaine-is-stepping-down-and-leaving-joyent -fazla/)
  • 13 Mayıs: Node.js ve io.js, Node Foundation altında birleşiyor

Q3 #

  • 2 Ağustos : Trace - Node.js performans izleme ve hata ayıklama
    • Trace is a visualized microservice monitoring tool that gets you all the metrics you need when operating microservices.
  • 13 Ağustos: 4.0 yeni 1.0’dır

4.Ç #

  • 12 Ekim: Düğüm v4.2.0, ilk Uzun Süreli Destek sürümü
  • 8 Aralık: Apigee, RisingStack ve Yahoo, Node.js Vakfı’na katıldı
  • 8 ve 9 Aralık: İnteraktif Düğüm
    • The first annual Node.js conference by the Node.js Foundation

2016 #

Q1 #

  • 10 Şubat: Express kuluçkada bir proje haline geldi
  • 23 Mart: Sol taraf olayı
  • 29 Mart: Google Cloud Platform, Node.js Foundation’a katıldı

2. Çey #

  • 26 Nisan: npm’nin 210.000 kullanıcısı var

Q3 #

  • 18 Temmuz: CJ Silverio, npm’nin CTO’su oldu
  • 1 Ağustos: Trace, Node.js hata ayıklama çözümü genel kullanıma sunuldu
  • 15 Eylül : Avrupa’daki ilk İnteraktif Node

4.Ç #

  • 11 Ekim : İplik paket yöneticisi yayınlandı
  • 18 Ekim : Node.js 6, LTS sürümü olur
