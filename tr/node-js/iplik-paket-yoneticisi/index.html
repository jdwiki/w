Yarn, npm’ye benzer şekilde Node.js için bir paket yöneticisidir. Birçok ortak noktayı paylaşırken, Yarn ve npm arasında bazı önemli farklılıklar vardır.

Temel bir paket oluşturma #

“yarn init” komutu, paketiniz hakkında bazı bilgileri yapılandırmak için bir “package.json” dosyasının oluşturulmasında size yol gösterecektir. Bu, npm’deki “npm init” komutuna benzer.

Paketinizi tutmak için yeni bir dizin oluşturun ve bu dizine gidin ve ardından “yarn init” komutunu çalıştırın.

mkdir my-package && cd my-package
yarn init

CLI’de takip eden soruları yanıtlayın

question name (my-package): my-package
question version (1.0.0): 
question description: A test package
question entry point (index.js): 
question repository url: 
question author: StackOverflow Documentation
question license (MIT): 
success Saved package.json
✨  Done in 27.31s.

Bu, aşağıdakine benzer bir “package.json” dosyası oluşturacaktır.

{
  "name": "my-package",
  "version": "1.0.0",
  "description": "A test package",
  "main": "index.js",
  "author": "StackOverflow Documentation",
  "license": "MIT"
}

Şimdi bir bağımlılık eklemeyi deneyelim. Bunun için temel sözdizimi “iplik ekleme [paket-adı]” şeklindedir.

ExpressJS’yi yüklemek için aşağıdakileri çalıştırın

‘iplik ekspres ekleyin’

Bu, “package.json” dosyanıza bir “bağımlılıklar” bölümü ekleyecek ve ExpressJS ekleyecektir.

"dependencies": {
    "express": "^4.15.2"
}

İplik Kurulumu #

Bu örnek, işletim sisteminiz için Yarn’ı kurmanın farklı yöntemlerini açıklar.

Mac os işletim sistemi #

Ev yapımı #

brew update
brew install yarn

MacPort’lar #

sudo port install yarn

YOLunuza İplik Ekleme #

Aşağıdakileri tercih ettiğiniz kabuk profiline ekleyin (.profile,.bashrc,.zshrc vb.)

export PATH="$PATH:`yarn global bin`"

Pencereler #

Yükleyici #

Öncelikle Node.js kurulu değilse kurun.

Yarn yükleyiciyi İplik web sitesinden .msi olarak indirin.

çikolatalı #

choco install yarn

Linux #

Debian/Ubuntu #

Dağıtımınız için Node.js’nin kurulu olduğundan emin olun veya aşağıdakileri çalıştırın

curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
sudo apt-get install -y nodejs

YarnPkg deposunu yapılandırın

curl -sS https://dl.yarnpkg.com/debian/pubkey.gpg | sudo apt-key add -
echo "deb https://dl.yarnpkg.com/debian/ stable main" | sudo tee /etc/apt/sources.list.d/yarn.list

İplik yükleyin

sudo apt-get update && sudo apt-get install yarn

CentOS / Fedora / RHEL #

Henüz yüklü değilse Node.js’yi yükleyin

curl --silent --location https://rpm.nodesource.com/setup_6.x | bash -

İplik yükleyin

sudo wget https://dl.yarnpkg.com/rpm/yarn.repo -O /etc/yum.repos.d/yarn.repo
sudo yum install yarn

Kemer #

İpliği AUR ile yükleyin.

Yaourt kullanan örnek:

yaourt -S yarn

Sadece #

sudo eopkg install yarn

Tüm Dağıtımlar #

Aşağıdakileri tercih ettiğiniz kabuk profiline ekleyin (.profile,.bashrc,.zshrc vb.)

export PATH="$PATH:`yarn global bin`"

Alternatif Kurulum Yöntemi #

Kabuk komut dosyası #

curl -o- -L https://yarnpkg.com/install.sh | bash

veya yüklenecek bir sürüm belirtin

curl -o- -L https://yarnpkg.com/install.sh | bash -s -- --version [version]

Tarball #

cd /opt
wget https://yarnpkg.com/latest.tar.gz
tar zvxf latest.tar.gz

Npm #

Zaten npm yüklediyseniz, çalıştırın

npm install -g yarn

Yükleme Sonrası #

Çalıştırarak Yarn’ın kurulu sürümünü kontrol edin

yarn --version

İplik ile paketi kurun #

Yarn, npm ile aynı kayıt defterini kullanır. Bu, npm’de bulunan her paketin Yarn’da aynı olduğu anlamına gelir.

Bir paketi kurmak için yarn add package komutunu çalıştırın.

Paketin belirli bir sürümüne ihtiyacınız varsa, yarn add package@version kullanabilirsiniz.

Yüklemeniz gereken sürüm etiketlendiyse, yarn add package@tag kullanabilirsiniz.
