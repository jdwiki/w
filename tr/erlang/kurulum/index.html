FreeBSD’de Erlang/OTP Oluşturun ve Yükleyin #

Aşağıdaki örnekler, Erlang/OTP’yi FreeBSD’ye kurmak için 3 ana yöntemi göstermektedir.

Yöntem 1 - Önceden Oluşturulmuş İkili Paket #

Önceden oluşturulmuş ikili paketi kurmak için pkg kullanın:

$ pkg install erlang

Yeni kurulumunuzu test edin:

$ erl
Erlang/OTP 18 [erts-7.3.1] [source] [64-bit] [smp:2:2] [async-threads:10] [hipe] [kernel-poll:false]

Eshell V7.3.1  (abort with ^G)

Yöntem 2 - Bağlantı noktası koleksiyonunu kullanarak derleyin ve kurun (önerilir) #

Bağlantı noktasını her zamanki gibi oluşturun ve kurun:

$ make -C /usr/ports/lang/erlang install clean

Yeni kurulumunuzu test edin:

$ erl
Erlang/OTP 18 [erts-7.3.1] [source] [64-bit] [smp:2:2] [async-threads:10] [hipe] [kernel-poll:false]

Eshell V7.3.1  (abort with ^G)

Bu, resmi web sitesinden sürüm tarball’ını getirecek, gerekirse birkaç yama uygulayacak, sürümü oluşturacak ve kuracaktır. Açıkçası, biraz zaman alacak.

Yöntem 3 - Sürüm tarball’ından derleyin ve yükleyin #

Not: Sürümü manuel olarak oluşturmak işe yarar, ancak bağlantı noktası koleksiyonu, sürümü daha FreeBSD dostu hale getiren yamalar içerdiğinden, yukarıdaki iki yöntemin kullanılması tercih edilmelidir.

Yayın dosyasını indirin:

$ fetch 'http://erlang.org/download/otp_src_18.3.tar.gz'

MD5 toplamının doğru olup olmadığını kontrol edin:

$ fetch 'http://erlang.org/download/MD5'
MD5                                           100% of   24 kB  266 kBps 00m00s

$ grep otp_src_18.3.tar.gz MD5
MD5(otp_src_18.3.tar.gz)= 7e4ff32f97c36fb3dab736f8d481830b

$ md5 otp_src_18.3.tar.gz
MD5 (otp_src_18.3.tar.gz) = 7e4ff32f97c36fb3dab736f8d481830b

Tarball’ı çıkarın:

$ tar xzf otp_src_18.3.tar.gz

Yapılandır:

$ ./configure --disable-hipe

Erlang’ı HiPe ile oluşturmak istiyorsanız, bağlantı noktası koleksiyonundaki yamaları uygulamanız gerekir.

İnşa etmek:

$ gmake

Düzenlemek:

$ gmake install

Yeni kurulumunuzu test edin:

$ erl
Erlang/OTP 18 [erts-7.3] [source] [64-bit] [smp:2:2] [async-threads:10] [kernel-poll:false]

Eshell V7.3  (abort with ^G)

Kerl kullanarak derleyin ve kurun #

kerl, Erlang/OTP sürümlerini oluşturmanıza ve yüklemenize yardımcı olan bir araçtır.

Kıvrımı yükleyin:

$ make -C /usr/ports/ftp/curl install clean

İndir adam:

$ fetch 'https://raw.githubusercontent.com/kerl/kerl/master/kerl'
$ chmod +x kerl

Mevcut sürümlerin listesini güncelleyin:

$ ./kerl update releases
The available releases are:
R10B-0 R10B-10 R10B-1a R10B-2 R10B-3 R10B-4 R10B-5 R10B-6 R10B-7 R10B-8 R10B-9 R11B-0 R11B-1 R11B-2 R11B-3 R11B-4 R11B-5 R12B-0 R12B-1 R12B-2 R12B-3 R12B-4 R12B-5 R13A R13B01 R13B02-1 R13B02 R13B03 R13B04 R13B R14A R14B01 R14B02 R14B03 R14B04 R14B R14B_erts-5.8.1.1 R15B01 R15B02 R15B02_with_MSVCR100_installer_fix R15B03-1 R15B03 R15B R16A_RELEASE_CANDIDATE R16B01 R16B02 R16B03-1 R16B03 R16B 17.0-rc1 17.0-rc2 17.0 17.1 17.3 17.4 17.5 18.0 18.1 18.2 18.2.1 18.3 19.0

Gerekli sürümü oluşturun:

$ ./kerl build 18.3 erlang-18.3

Derlemenin derleme listesinde mevcut olup olmadığını kontrol edin:

$ ./kerl list builds
18.3,erlang-18.3

Yapıyı bir yere kurun:

$ ./kerl install erlang-18.3 ./erlang-18.3

Bash veya balık kabuğu çalıştırıyorsanız ‘activate’ dosyasını kaynaklayın. Bir cshell çalıştırıyorsanız, yapı bin dizinini PATH’e ekleyin:

$ setenv PATH "/some/where/erlang-18.3/bin/:$PATH"

Yeni kurulumunuzu test edin:

$ which erl
/some/where/erlang-18.3/bin//erl

$ erl
Erlang/OTP 18 [erts-7.3] [source] [64-bit] [smp:2:2] [async-threads:10] [hipe] [kernel-poll:false]

Eshell V7.3  (abort with ^G)

Diğer sürümler #

Erlang/OTP’nin başka bir sürümünü oluşturmak istiyorsanız, koleksiyondaki diğer bağlantı noktalarını arayın:

  • lang/erlang-runtime15
  • lang/erlang-runtime16
  • lang/erlang-runtime17
  • lang/erlang-runtime18

Ubuntu’da Erlang/OTP Oluşturun ve Yükleyin #

Aşağıdaki örnekler, Erlang/OTP’yi Ubuntu’ya kurmak için iki ana yöntemi göstermektedir.

Yöntem 1 - Önceden Oluşturulmuş İkili Paket #

Basitçe bu komutu çalıştırın ve Erlang Solutions adresinden en son kararlı Erlang sürümünü indirip kuracaktır.

$ sudo apt-get install erlang

Yöntem 2 - Kaynaktan Oluştur ve Yükle #

tar dosyasını indirin:

$ wget http://erlang.org/download/otp_src_19.0.tar.gz

tar dosyasını çıkarın:

$ tar -zxf otp_src_19.0.tar.gz

Ayıklanan dizini girin ve “ERL_TOP” öğesini geçerli yol olarak ayarlayın:

$ cd otp_src_19.0
$ export ERL_TOP=`pwd`

Şimdi yapıyı yapılandırmadan önce, Erlang’ı kurmak için ihtiyacınız olan tüm bağımlılıklara sahip olduğunuzdan emin olmak istiyorsunuz:

Temel bağımlılıklar:

$ sudo apt-get install autoconf libncurses-dev build-essential

Diğer uygulama bağımlılıkları

Uygulama Bağımlılık yükleme
Merhaba $ sudo apt-get install m4
ODBC $ sudo apt-get install unixodbc-dev
OpenSSL $ sudo apt-get install libssl-dev
wxWidget’lar $ sudo apt-get install libwxgtk3.0-dev libglu-dev
Belgeler $ sudo apt-get install fop xsltproc
Orber ve diğer C++ projeleri $ sudo apt-get install g++
jinterface $ sudo apt-get install default-jdk

Yapılandırın ve oluşturun: Kendi seçeneklerinizi ayarlayabilir veya varsayılan yapılandırmayı çalıştırmak için boş bırakabilirsiniz. Erlang/OTP için gelişmiş yapılandırma ve derleme.

$ ./configure [ options ]
$ make

Yapının test edilmesi:

$ make release_tests
$ cd release/tests/test_server
$ $ERL_TOP/bin/erl -s ts install -s ts smoke_test batch -s init stop

Bu komutları çalıştırdıktan sonra, web tarayıcınız ile $ERL_TOP/release/tests/test_server/index.html dosyasını açın ve herhangi bir başarısızlık olup olmadığını kontrol edin. Tüm testler başarılı olursa kuruluma devam edebiliriz.

Yükleme:

$ make install

OpenBSD’de Erlang/OTP Oluşturun ve Yükleyin #

OpenBSD’deki Erlang, şu anda ‘alpha’, ‘sparc’ ve ‘hppa’ mimarilerinde bozuk.

Yöntem 1 - Önceden Oluşturulmuş İkili Paket #

OpenBSD, sisteminize kurmak istediğiniz sürümü seçmenize izin verir:

######################################################################
# free-choice:
######################################################################
$ pkg_add erlang
# a       0: <None>
#   1: erlang-16b.03p10v0
#   2: erlang-17.5p6v0
#   3: erlang-18.1p1v0
#   4: erlang-19.0v0

######################################################################
# manual-choice:
######################################################################
pkg_add erlang%${version}
# example: 
pkg_add erlang%19

OpenBSD, Erlang’ın birden çok sürümünü destekleyebilir. Düşünmeyi daha kolay hale getirmek için, her ikili kendi adına Erlang versiyonu yüklenir. Yani, erlang-19.0v0’ı yüklediyseniz, erl ikili dosyanız erl19 olacaktır.

“erl” kullanmak istiyorsanız, bir sembolik bağlantı oluşturabilirsiniz:

ln -s /usr/local/bin/erl19 /usr/local/bin/erl

veya kabuk yapılandırma dosyanızda veya .profile dosyasında bir takma ad oluşturun:

echo 'alias erl="erl19"' >> ~/.profile
# or
echo 'alias erl="erl19"' >> ~/.shrc

Artık “erl” komutunu çalıştırabilirsiniz:

erl19
# or if you have an alias or symlink
erl
# Erlang/OTP 19 [erts-8.0] [source] [async-threads:10] [kernel-poll:false]
# 
# Eshell V8.0  (abort with ^G)

Yöntem 2 - Bağlantı noktalarını kullanarak derleyin ve kurun #

 RELEASE=OPENBSD_$(uname -r  | sed 's/\./_/g')
 cd /usr
 cvs -qz3 -danoncvs@anoncvs.openbsd.org:/cvs co -r${RELEASE}
 cd /usr/ports/lang/erlang
 ls -p 
 # 16/ 17/ 18/ 19/  CVS/  Makefile  Makefile.inc  erlang.port.mk
 cd 19
 make && make install

Yöntem 3 - Kaynaktan derleyin #

Kaynaktan derleme ek paketler gerektirir:

  • “git”
  • “gmake”
  • otokonf-2.59
pkg_add git gmake autoconf%2.59
git clone https://github.com/erlang/otp.git
cd otp
AUTOCONF_VERSION="2.59" ./build_build all
