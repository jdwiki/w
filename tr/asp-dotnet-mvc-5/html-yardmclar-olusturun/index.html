Html Yardımcıları Oluşturun

  • Basit bir yardımcı oluşturun - içinde metin bulunan bir div
  • Tek Kullanımlık Yardımcı (Html.BeginForm gibi)

Bu sayfada

  • Basit bir yardımcı oluşturun - içinde metin bulunan bir div
  • Tek Kullanımlık Yardımcı (Html.BeginForm gibi)

Html yardımcıları, MVC çerçevesini kullanarak görünümlerde html öğeleri oluşturmanın çok kullanışlı bir yoludur. Biraz zamanla ekibiniz bunları kullanmaktan gerçekten faydalanabilir. Kodu temiz tutmaya ve hataya açık tutmaya yardımcı olur.

Yardımcıları kullanmak için önce görünümün içine bir ‘@using’ yönergesi eklemeniz veya ‘Görünümler’ klasöründe bulunan ‘Web.config’ dosyasının içine ad alanını eklemeniz gerekir.

Basit bir yardımcı oluşturun - içinde metin bulunan bir div #

public static class MyHelpers 
{
   public static MvcHtmlString MyCustomDiv(this HtmlHelper htmlHelper, string text,
          object htmlAttributes = null)
   {
        var mainTag = new TagBuilder("div");
        mainTag.MergeAttributtes(htmlAttributes);
        mainTag.AddCssClass("some custom class");
        mainTag.SetInnerHtml(text);
        return MvcHtmlString.Create(mainTag.ToString());
   }
}

Görünümlerde kullanmak için:

@Html.MyCustomDiv("Test inside custom div");
@Html.MyCustomDiv("Test inside custom div", new {@class="some class for the div element"});

Tek Kullanımlık Yardımcı (Html.BeginForm gibi) #

  1. Önce tek kullanımlık bir sınıf oluşturun:

    public class MyDisposableHelper: IDisposable { private bool _disposed;
    private readonly ViewContext _viewContext;

     public MyDisposableHelper(ViewContext viewContext)
 {
     if (viewContext == null)
     {
         throw new ArgumentNullException(nameof(viewContext));
     }            
     _viewContext = viewContext;
 }

 public void Dispose()
 {
     Dispose(true);
     GC.SuppressFinalize(this);
 }

 protected virtual void Dispose(bool disposing)
 {
     if (_disposed)
         return;
     _disposed = true;            
     _viewContext.Writer.Write("</div>");
 }

 public void EndForm()
 {
     Dispose(true);
 }

    }

Bu sınıf, ‘IDisposable’ı devralır, çünkü yardımcıyı ‘Html.BeginForm(…)’ gibi kullanmak istiyoruz. Atıldığında, görünümde yardımcıyı çağırdığımızda oluşturulan ‘div’i kapatır.

  1. HtmlHelper uzantı yöntemini oluşturun:

    public static MyDisposableHelper BeginContainingHelper(this HtmlHelper htmlHelper) { var containingTag = new TagBuilder(“div”); //add default css classes, attributes as needed htmlHelper.ViewContext.Writer.Write(containingTag .ToString(TagRenderMode.StartTag)); return new MyDisposableHelper (htmlHelper.ViewContext); }

Burada dikkat edilmesi gereken şey, özel öğeden yanıt sayfasına yazmak için ‘Writer.Write’ çağrısıdır. ‘TagRenderMode.StartTag’, yazara div’i henüz kapatmamasını bildirmek için kullanılır, çünkü ‘MyDisposableHelper’ sınıfını atarken onu kapatacağız.

  1. Görünümde kullanmak için:

    @using (Html.BeginContainingHelper()) {

    element inside our custom element
    }
