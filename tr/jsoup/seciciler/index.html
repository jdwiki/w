Bir seçici, birleştiricilerle ayrılmış basit seçiciler zinciridir. Seçiciler büyük/küçük harfe duyarsızdır (elemanlara, niteliklere ve nitelik değerlerine karşı dahil).

Evrensel seçici (*), hiçbir öğe seçici sağlanmadığında örtüktür (yani *.header ve .header eşdeğerdir).

Desen Maçlar Örnek
* herhangi bir eleman *
“etiket” verilen etiket adına sahip öğeler ‘div’
ns|E ad alanında E tipi öğeler ns fb|name fb:name öğelerini bulur
#id “id” öznitelik kimliğine sahip öğeler div#wrap, #logo
.sınıf “sınıf” sınıf adına sahip öğeler div.left, .result
[attr] “attr” adlı bir özniteliğe sahip öğeler (herhangi bir değerle) a[href], [başlık]
[^attrÖnek] “attrPrefix” ile başlayan bir öznitelik adına sahip öğeler. HTML5 veri kümeleriyle öğeleri bulmak için kullanın [^veri-], div[^veri-]
[attr=val] “attr” adlı bir özniteliğe ve “val” değerine eşit olan öğeler img[width=500], a[rel=nofollow]
[attr="val"] “attr” adlı bir özniteliğe ve “val” değerine eşit olan öğeler span[hello="Cleveland"][elveda="Columbus"], a[rel="nofollow"]
[attr^=valÖnek] “attr” adlı bir özniteliğe ve “valPrefix” ile başlayan değere sahip öğeler a[href^=http:]
[attr$=valSuffix] “attr” adlı bir özniteliğe ve “valSuffix” ile biten değere sahip öğeler img[src$=.png]
[attr*=valContaining] “attr” adlı bir özniteliğe ve “valContaining” içeren değere sahip öğeler a[href*=/arama/]
[attr~=normal ifade] “attr” adlı bir özniteliğe ve normal ifadeyle eşleşen değere sahip öğeler img[src~=(?i)\.(png|jpe?g)]
Yukarıdakiler herhangi bir sırayla birleştirilebilir div.header[başlık]

CSS seçicileri kullanarak öğeleri seçme #

String html = "<!DOCTYPE html>" +
              "<html>" +
                "<head>" +
                  "<title>Hello world!</title>" +
                "</head>" +
                "<body>" +
                  "<h1>Hello there!</h1>" +
                  "<p>First paragraph</p>" +
                  "<p class=\"not-first\">Second paragraph</p>" +
                  "<p class=\"not-first third\">Third <a href=\"page.html\">paragraph</a></p>" +
                "</body>" +
              "</html>";

// Parse the document
Document doc = Jsoup.parse(html);

// Get document title
String title = doc.select("head > title").first().text();
System.out.println(title); // Hello world!

Element firstParagraph = doc.select("p").first();

// Get all paragraphs except from the first
Elements otherParagraphs = doc.select("p.not-first");
// Same as
otherParagraphs = doc.select("p");
otherParagraphs.remove(0);

// Get the third paragraph (second in the list otherParagraphs which
// excludes the first paragraph)
Element thirdParagraph = otherParagraphs.get(1);
// Alternative:
thirdParagraph = doc.select("p.third");

// You can also select within elements, e.g. anchors with a href attribute
// within the third paragraph.
Element link = thirdParagraph.select("a[href]");
// or the first <h1> element in the document body
Element headline = doc.select("body").first().select("h1").first();

Twitter İşaretlemesini Çıkarın #

    // Twitter markup documentation: 
    // https://dev.twitter.com/cards/markup
    String[] twitterTags = {
            "twitter:site", 
            "twitter:site:id", 
            "twitter:creator", 
            "twitter:creator:id", 
            "twitter:description", 
            "twitter:title", 
            "twitter:image", 
            "twitter:image:alt", 
            "twitter:player", 
            "twitter:player:width", 
            "twitter:player:height", 
            "twitter:player:stream", 
            "twitter:app:name:iphone", 
            "twitter:app:id:iphone", 
            "twitter:app:url:iphone", 
            "twitter:app:name:ipad", 
            "twitter:app:id:ipad", 
            "twitter:app:url:ipadt",
            "twitter:app:name:googleplay", 
            "twitter:app:id:googleplay", 
            "twitter:app:url:googleplay"        
    };
    
    // Connect to URL and extract source code
    Document doc = Jsoup.connect("http://stackoverflow.com/").get();
    
    for (String twitterTag : twitterTags) {
        
        // find a matching meta tag
        Element meta = doc.select("meta[name=" + twitterTag + "]").first();
        
        // if found, get the value of the content attribute
        String content = meta != null ? meta.attr("content") : "";
        
        // display results
        System.out.printf("%s = %s%n", twitterTag, content);
    }
