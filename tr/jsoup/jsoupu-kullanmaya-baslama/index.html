Bağlantıların URL’lerini ve başlıklarını çıkarın #

Jsoup, bir web sayfasındaki tüm bağlantıları kolayca çıkarmak için kullanılabilir. Bu durumda, Jsoup’u yalnızca istediğimiz belirli bağlantıları çıkarmak için kullanabiliriz, burada, bir sayfadaki “h3” başlığındakiler. Ayrıca bağlantıların metnini de alabiliriz.

Document doc = Jsoup.connect("http://stackoverflow.com").userAgent("Mozilla").get();
for (Element e: doc.select("a.question-hyperlink")) {
    System.out.println(e.attr("abs:href"));
    System.out.println(e.text());
    System.out.println();
}

Bu, aşağıdaki çıktıyı verir:

http://stackoverflow.com/questions/12920296/past-5-week-calculation-in-webi-bo-4-0
Past 5 week calculation in WEBI (BO 4.0)?

http://stackoverflow.com/questions/36303701/how-to-get-information-about-the-visualized-elements-in-listview
How to get information about the visualized elements in listview?

[...]

Burada neler oluyor:

  • İlk olarak, belirtilen URL’den HTML belgesini alıyoruz. Bu kod ayrıca isteğin Kullanıcı Aracısı başlığını “Mozilla” olarak ayarlar, böylece web sitesi genellikle tarayıcılara sunacağı sayfayı sunar.

  • Ardından, Yığın Taşması sorularına, bu durumda “soru-köprü” sınıfına sahip bağlantılara ilişkin tüm bağlantıları almak için “select(…)” ve bir for döngüsü kullanın.

  • Her bağlantının metnini .text() ve bağlantının hrefini attr("abs:href") ile yazdırın. Bu durumda, mutlak URL’yi almak için abs: kullanırız, yani. etki alanı ve protokol dahil.

Kısmi HTML’den tam URL’yi çıkarın #

Bir link:href’in yalnızca öznitelik değerinin seçilmesi, göreli URL’yi döndürür.

   String bodyFragment = 
          "<div><a href=\"/documentation\">Stack Overflow Documentation</a></div>";


    Document doc = Jsoup.parseBodyFragment(bodyFragment);
    String link = doc
            .select("div > a")
            .first()
            .attr("href");
    
    System.out.println(link);

Çıktı

/documentation

Temel URI’yi parse’a geçirerek yöntemi ve kullanımı yerine absUrl yöntemi attr, tam URL’yi çıkarabiliriz.

    Document doc = Jsoup.parseBodyFragment(bodyFragment, "http://stackoverflow.com");
    
    String link = doc
                .select("div > a")
                .first()
                .absUrl("href");
    
    System.out.println(link);

HTML belge dosyasından verileri çıkarın #

Jsoup, HTML içeren yerel bir dosyadan verileri işlemek veya çıkarmak için kullanılabilir. filePath diskteki bir dosyanın yoludur. ENCODING isteniyor Karakter Seti Adı ör. “Windows-31J”. İsteğe bağlı.

 // load file
    File inputFile = new File(filePath);
    // parse file as HTML document
    Document doc = Jsoup.parse(filePath, ENCODING);
    // select element by <a> 
    Elements elements = doc.select("a");
