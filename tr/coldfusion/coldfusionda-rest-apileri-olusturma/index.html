Coldfusion'da REST API'leri oluşturma

  • Arka uç oluşturma
  • Arayüz

Bu sayfada

  • Arka uç oluşturma
  • Arayüz

REST API’leri, verilere farklı diller (sunucu ve istemci tarafı) dahil olmak üzere her yerden erişilmesi gerektiğinde ilginçtir. Bu, veriden ve işlemeden ayrılmayı gerektirir.

Arka uç oluşturma #

<cfcomponent displayname="myAPI" output="false">
    <cffunction name="init" access="public" output="no">
        <!--- do some basic stuff --->
        <cfreturn this>
    </cffunction>

    <cffunction name="welcome">
        <cfreturn "Hello World!">
    </cffunction>
</cfcomponent>

Arayüz #

<cfscript>
    api_request = GetHttpRequestData();
    api = createObject("component","myAPI").init();
</cfscript>

<cfif api_request.method is 'GET'>
    <cfoutput>#api.welcome()#</cfoutput>
<cfelseif api_request.method is 'POST'>
    <cfheader statuscode="500" statustext="Internal Server Error" />
</cfif>
