Sunucuyla İletişim Kurmak için Basit jQuery Örneği #

jQuery.ajax API web sitesinden alınmıştır:

$.ajax({
    method: "POST",
    url: "some.php",
    data: {
        name: "John",
        location: "Boston"
    },
    success: function(msg) {
        alert("Data Saved: " + msg);
    },
    error: function(e) {
        alert("Error: " + e);
    }
});

Bu kod parçasının jQuery sayesinde okunması ve neler olduğunu anlaması kolaydır.

  • $.ajax - bu bit, jQuery’nin ajax işlevini çağırır.

  • method: "POST" - buradaki bu satır, sunucuyla iletişim kurmak için bir POST yöntemi kullanacağımızı bildirir. İstek türlerini okuyun!

  • ‘url’ - bu değişken, isteğin nereye SENT gönderileceğini bildirir. Bir yere TO isteği gönderiyorsunuz. Fikir bu.

  • “veri” - oldukça yalındır. Bu, isteğinizle birlikte gönderdiğiniz verilerdir.

  • success - Bu fonksiyon burada yazdığınız veri ile ne yapacağınıza karar vermek için geri msg! örneğin önerdiği gibi, şu anda yalnızca döndürülen “msg” ile bir uyarı oluşturuyor.

  • hata - buraya yazdığınız bu fonksiyon, hata mesajlarını görüntülemek veya ajax isteği hatalarla karşılaştığında çalışacak eylemleri sağlamak için yazıyorsunuz.

  • “.done"a bir alternatif

    success: function(result) {
    // do something
});

Kurulum veya Kurulum #

AJAX nedir? #

AJAX Asenkron JavaScript ve XML anlamına gelir. Özetle, öyle sunucu tarafı ile iletişim kurmak için XMLHttpRequest nesnesinin kullanımı komut dosyaları. Çeşitli biçimlerde bilgi göndermenin yanı sıra bilgi alabilir. JSON, XML, HTML ve hatta metin dosyaları dahil olmak üzere biçimler. -Mozilla Geliştirici Ağı 2016

AJAX’ı uygulamanın en kolay yolu, özellikle sunucularla iletişim kurmayı planlıyorsanız, jQuery kullanmaktır.

jQuery nedir? #

jQuery, hızlı, küçük ve zengin özelliklere sahip bir JavaScript kitaplığıdır. Yapar HTML belgesi geçişi ve manipülasyonu, olay işleme gibi şeyler, çalışan kullanımı kolay bir API ile animasyon ve Ajax çok daha basit çok sayıda tarayıcıda. -jquery.com

Çok fazla jQuery kullanmayanlar için, bunu hayatımızı kolaylaştırmak için kullanabileceğimiz işlevler olarak düşünün. Bu, aynı şeyi başarmak için yazmamız gereken kod miktarını azalttığı için AJAX ile kullanmak için mükemmeldir!
