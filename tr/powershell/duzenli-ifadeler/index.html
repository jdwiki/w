Sözdizimi #

  • ‘metin’ - ‘RegExPattern’ ile eşleşir
  • ‘metin’ -‘RegExPattern’, ‘yenideğer’ yerine
  • [regex]::Match(“metin”,“desen”) #Tek eşleşme
  • [regex]::Eşleşmeler(“metin”,“desen”) #Birden çok eşleşme
  • [regex]::Değiştir(“metin”,“desen”,“yenideğer”)
  • [regex]::Replace(“text”,“pattern”, {param($m) }) #MatchEvaluator
  • [regex]::Escape(“input”) #Escape özel karakterleri

Tek maç #

Regex’i kullanarak bir metnin belirli bir kalıp içerip içermediğini hızlı bir şekilde belirleyebilirsiniz. PowerShell’de Regex ile çalışmanın birden çok yolu vardır.

#Sample text
$text = @"
This is (a) sample
text, this is
a (sample text)
"@

#Sample pattern: Content wrapped in ()
$pattern = '\(.*?\)'

-Match operatörünü kullanma #

Yerleşik “-matches” operatörünü kullanarak bir dizgenin bir kalıpla eşleşip eşleşmediğini belirlemek için, “input” -match “pattern” sözdizimini kullanın. Bu, aramanın sonucuna bağlı olarak “doğru” veya “yanlış” döndürür. Eğer eşleşme varsa, $Matchler değişkenine erişerek eşleşmeyi ve grupları (kalıpta tanımlanmışsa) görüntüleyebilirsiniz.

> $text -match $pattern
True

> $Matches

Name Value
---- -----
0    (a)

Ayrıca, bir dizi diziyi filtrelemek ve yalnızca bir eşleşme içeren dizeleri döndürmek için “-match” kullanabilirsiniz.

> $textarray = @"
This is (a) sample
text, this is
a (sample text)
"@ -split "`n"

> $textarray -match $pattern
This is (a) sample
a (sample text)

Select-String’i Kullanma #

PowerShell 2.0, normal ifade kullanarak metin içinde arama yapmak için yeni bir cmdlet’i tanıttı. Bir eşleşme içeren metin girişi başına bir ‘MatchInfo’ nesnesi döndürür. Eşleşen grupları vb. bulmak için özelliklerine erişebilirsiniz.

> $m = Select-String -InputObject $text -Pattern $pattern

> $m

This is (a) sample
text, this is
a (sample text)

> $m | Format-List *

IgnoreCase : True
LineNumber : 1
Line       : This is (a) sample
             text, this is
             a (sample text)
Filename   : InputStream
Path       : InputStream
Pattern    : \(.*?\)
Context    : 
Matches    : {(a)}

‘-match’ gibi, ‘Select-String’ de bir dizi diziyi bir diziye aktararak filtrelemek için kullanılabilir. Bir eşleşme içeren dize başına bir “MatchInfo” nesnesi oluşturur.

> $textarray | Select-String -Pattern $pattern

This is (a) sample
a (sample text)

#You can also access the matches, groups etc.
> $textarray | Select-String -Pattern $pattern | fl *


IgnoreCase : True
LineNumber : 1
Line       : This is (a) sample
Filename   : InputStream
Path       : InputStream
Pattern    : \(.*?\)
Context    : 
Matches    : {(a)}

IgnoreCase : True
LineNumber : 3
Line       : a (sample text)
Filename   : InputStream
Path       : InputStream
Pattern    : \(.*?\)
Context    : 
Matches    : {(sample text)}

“Select-String”, “-SimpleMatch” anahtarını ekleyerek normal bir metin deseni (normal ifade yok) kullanarak da arama yapabilir.

[RegEx]::Match()‘i Kullanma #

.NET [RegEx] sınıfında bulunan statik Match() yöntemini de kullanabilirsiniz.

> [regex]::Match($text,$pattern)

Groups   : {(a)}
Success  : True
Captures : {(a)}
Index    : 8
Length   : 3
Value    : (a)

> [regex]::Match($text,$pattern) | Select-Object -ExpandProperty Value
(a)

Yer değiştirmek #

Normal ifade için ortak bir görev, bir kalıpla eşleşen metni yeni bir değerle değiştirmektir.

#Sample text
$text = @"
This is (a) sample
text, this is
a (sample text)
"@

#Sample pattern: Text wrapped in ()
$pattern = '\(.*?\)'

#Replace matches with:
$newvalue = 'test'

-Değiştir operatörünü kullanma #

PowerShell’deki ‘-replace’ operatörü, ‘‘input’ -replace ‘pattern’, ’newvalue’’ sözdizimini kullanarak bir kalıpla eşleşen metni yeni bir değerle değiştirmek için kullanılabilir.

> $text -replace $pattern, $newvalue
This is test sample
text, this is
a test

[RegEx]::Replace() yöntemini kullanma #

Eşleşmeleri değiştirme, ‘[RegEx]’ .NET sınıfındaki ‘Replace()’ yöntemi kullanılarak da yapılabilir.

[regex]::Replace($text, $pattern, 'test')
This is test sample
text, this is
a test

MatchEvalutor kullanarak metni dinamik değerle değiştirin #

Bazen bir kalıpla eşleşen değeri, o belirli eşleşmeye dayalı yeni bir değerle değiştirmeniz gerekir, bu da yeni değeri tahmin etmeyi imkansız hale getirir. Bu tür senaryolar için bir “MatchEvaluator” çok faydalı olabilir.

PowerShell’de, bir “MatchEvaluator”, geçerli eşleşme için bir “Match” nesnesi içeren tek bir parametreye sahip bir komut dosyası bloğu kadar basittir. Eylemin çıktısı, söz konusu eşleşme için yeni değer olacaktır. “MatchEvalutor”, “[Regex]::Replace()” statik yöntemiyle kullanılabilir.

Örnek: () içindeki metni uzunluğuyla değiştirme

#Sample text
$text = @"
This is (a) sample
text, this is
a (sample text)
"@
    
#Sample pattern: Content wrapped in ()
$pattern = '(?<=\().*?(?=\))'

$MatchEvalutor = {
    param($match)

    #Replace content with length of content
    $match.Value.Length

}

Çıktı:

> [regex]::Replace($text, $pattern, $MatchEvalutor)

This is 1 sample
text, this is
a 11

Örnek: ‘örnek’i büyük harf yap

#Sample pattern: "Sample"
$pattern = 'sample'

$MatchEvalutor = {
    param($match)

    #Return match in upper-case
    $match.Value.ToUpper()

}

Çıktı:

> [regex]::Replace($text, $pattern, $MatchEvalutor)

This is (a) SAMPLE
text, this is
a (SAMPLE text)

Özel karakterlerden kaçış #

Bir normal ifade kalıbı, bir kalıbı tanımlamak için birçok özel karakter kullanır. Ör., ., “herhangi bir karakter” anlamına gelir, `+’, “bir veya daha fazla” anlamına gelir vb.

Bu karakterleri bir kalıpta ., + vb. olarak kullanmak için, özel anlamlarını ortadan kaldırmak için onlardan kaçmanız gerekir. Bu, normal ifadede ters eğik çizgi \ olan kaçış karakteri kullanılarak yapılır. Örnek: “+” aramak için “+” kalıbını kullanırsınız.

Normal ifadedeki tüm özel karakterleri hatırlamak zor olabilir, bu nedenle aramak istediğiniz bir dizedeki her özel karakterden kaçmak için [RegEx]::Escape("input") yöntemini kullanabilirsiniz.

> [regex]::Escape("(foo)")
\(foo\)

> [regex]::Escape("1+1.2=2.2")
1\+1\.2=2\.2

Çoklu eşleşmeler #

Bir metindeki bir kalıp için tüm eşleşmeleri bulmanın birden çok yolu vardır.

#Sample text
$text = @"
This is (a) sample
text, this is
a (sample text)
"@

#Sample pattern: Content wrapped in ()
$pattern = '\(.*?\)'

Select-String’i Kullanma #

Tüm eşleşmeleri (global eşleşme), “-AllMatches” anahtarını “Select-String"e ekleyerek bulabilirsiniz.

> $m = Select-String -InputObject $text -Pattern $pattern -AllMatches

> $m | Format-List *

IgnoreCase : True
LineNumber : 1
Line       : This is (a) sample
             text, this is
             a (sample text)
Filename   : InputStream
Path       : InputStream
Pattern    : \(.*?\)
Context    : 
Matches    : {(a), (sample text)}

#List all matches
> $m.Matches

Groups   : {(a)}
Success  : True
Captures : {(a)}
Index    : 8
Length   : 3
Value    : (a)

Groups   : {(sample text)}
Success  : True
Captures : {(sample text)}
Index    : 37
Length   : 13
Value    : (sample text)

#Get matched text
> $m.Matches | Select-Object -ExpandProperty Value
(a)
(sample text)

[RegEx]::Match()‘i Kullanma #

.NET [regex]-sınıfındaki Matches()` yöntemi, birden çok eşleşme için genel arama yapmak için de kullanılabilir.

> [regex]::Matches($text,$pattern)

Groups   : {(a)}
Success  : True
Captures : {(a)}
Index    : 8
Length   : 3
Value    : (a)

Groups   : {(sample text)}
Success  : True
Captures : {(sample text)}
Index    : 37
Length   : 13
Value    : (sample text)

> [regex]::Matches($text,$pattern) | Select-Object -ExpandProperty Value

(a)
(sample text)
