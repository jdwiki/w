Nesneleri Güncelleme #

Özellik ekleme #

Mevcut bir nesneye özellikler eklemek isterseniz, Üye Ekle cmdlet’ini kullanabilirsiniz. PSObjects ile değerler “Not Özellikleri” türünde tutulur

$object = New-Object -TypeName PSObject -Property @{
        Name = $env:username
        ID = 12
        Address = $null
    }

Add-Member -InputObject $object -Name "SomeNewProp" -Value "A value" -MemberType NoteProperty

# Returns
PS> $Object
Name ID Address SomeNewProp
---- -- ------- -----------
nem  12         A value

Ayrıca Select-Object Cmdlet (hesaplanmış özellikler olarak adlandırılır) ile özellikler de ekleyebilirsiniz:

$newObject = $Object | Select-Object *, @{label='SomeOtherProp'; expression={'Another value'}}

# Returns
PS> $newObject
Name ID Address SomeNewProp SomeOtherProp
---- -- ------- ----------- -------------
nem  12         A value     Another value

Yukarıdaki komut şu şekilde kısaltılabilir:

$newObject = $Object | Select *,@{l='SomeOtherProp';e={'Another value'}}

Özelliklerin kaldırılması #

Bir nesneden özellikleri kaldırmak için Select-Object Cmdlet’i kullanabilirsiniz:

$object = $newObject | Select-Object * -ExcludeProperty ID, Address

# Returns
PS> $object
Name SomeNewProp SomeOtherProp
---- ----------- -------------
nem  A value     Another value

Yeni bir nesne oluşturma #

PowerShell, diğer bazı betik dillerinden farklı olarak, nesneleri ardışık düzen aracılığıyla gönderir. Bunun anlamı, bir komuttan diğerine veri gönderdiğinizde, nesneler oluşturabilmeniz, değiştirebilmeniz ve toplayabilmeniz önemlidir.

Bir nesne oluşturmak basittir. Oluşturduğunuz nesnelerin çoğu, PowerShell’de özel nesneler olacaktır ve bunun için kullanılacak tür PSObject’dir. PowerShell, .NET’te oluşturabileceğiniz herhangi bir nesneyi oluşturmanıza da olanak tanır.

İşte birkaç özelliğe sahip yeni nesneler oluşturmaya bir örnek:

Seçenek 1: Yeni Nesne #

$newObject = New-Object -TypeName PSObject -Property @{
    Name = $env:username
    ID = 12
    Address = $null
}

# Returns
PS> $newObject
Name ID Address
---- -- -------
nem  12

Komutun önüne $newObject = yazarak nesneyi bir değişkende saklayabilirsiniz.

Ayrıca nesne koleksiyonlarını da saklamanız gerekebilir. Bu, boş bir koleksiyon değişkeni oluşturarak ve koleksiyona aşağıdaki gibi nesneler ekleyerek yapılabilir:

$newCollection = @()
$newCollection += New-Object -TypeName PSObject -Property @{
    Name = $env:username
    ID = 12
    Address = $null
}

Daha sonra bu koleksiyon nesnesini nesne bazında yinelemek isteyebilirsiniz. Bunu yapmak için belgelerdeki Döngü bölümünü bulun.

Seçenek 2: Nesne Seç #

İnternette bulacağınız nesneler oluşturmanın daha az yaygın bir yolu şudur:

$newObject = 'unuseddummy' | Select-Object -Property Name, ID, Address
$newObject.Name = $env:username
$newObject.ID = 12

# Returns
PS> $newObject
Name ID Address
---- -- -------
nem  12

Seçenek 3: pscustomobject türü hızlandırıcı (PSv3+ gerekli) #

Sıralı tür hızlandırıcı, PowerShell’i özelliklerimizi tanımladığımız sırada tutmaya zorlar. “[PSCustomObject]” kullanmak için sıralı tür hızlandırıcıya ihtiyacınız yok:

$newObject = [PSCustomObject][Ordered]@{
    Name = $env:Username
    ID = 12
    Address = $null
}

# Returns
PS> $newObject
Name ID Address
---- -- -------
nem  12

Bir nesneyi incelemek #

Artık bir nesneniz olduğuna göre, onun ne olduğunu bulmak iyi olabilir. Bir nesnenin ne olduğunu ve ne içerdiğini görmek için Get-Member cmdlet’ini kullanabilirsiniz:

Get-Item c:\windows | Get-Member

Bu şunları sağlar:

TypeName: System.IO.DirectoryInfo

Ardından, nesnenin sahip olduğu özellikler ve yöntemler listesi gelir.

Bir nesnenin türünü almanın başka bir yolu da GetType yöntemini kullanmaktır, şöyle:

C:\> $Object = Get-Item C:\Windows
C:\> $Object.GetType()

IsPublic IsSerial Name                                     BaseType
-------- -------- ----                                     --------
True     True     DirectoryInfo                            System.IO.FileSystemInfo

Nesnenin sahip olduğu özelliklerin bir listesini değerleriyle birlikte görüntülemek için, Özellik parametresi : * (tümü anlamına gelir) olarak ayarlanmış Format-List cmdlet’ini kullanabilirsiniz.

[Nullable[datetime]]$nullableDate = Get-Date -Year 2012
$nullableDate
$nullableDate.GetType().FullName
$nullableDate = $null
$nullableDate

$aDate
$aDate.GetType().FullName
