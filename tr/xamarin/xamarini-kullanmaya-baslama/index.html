Xamarin Studio’yu OS X’e Yükleme #

Bir OS X makinesinde Xamarin geliştirmeyi başlatmanın ilk adımı, resmi web sitesinden Xamarin Studio Community sürümünü indirip yüklemektir. Aşağıdaki resimde gösterildiği gibi yükleyiciyi indirmek için birkaç alanın doldurulması gerekir.

Xamarin Unified yükleyicisi, Xamarin.Android, Xamarin.iOS ve Xamarin Studio’nun üzerine gerekli tüm Xamarin dışı bileşenleri (ör.

Xamarin.iOS uygulamaları geliştirmek için aşağıdaki ön koşulların karşılanması gerekir:

  • iOS geliştirici merkezinden en son iOS SDK’sı.
  • Mac App Store’dan veya Apple Developer Web Sitesinden en son Xcode sürümü.
  • Mac OS X Yosemite (10.10) ve üzeri

Kurulum süreci #

Ön koşullar karşılandıktan sonra, Xamarin logosuna çift tıklayarak Xamarin Yükleyicisini çalıştırın.

OS X Gatekeeper, indirilen uygulamayı açmak için sizden onay isteyen bir iletişim kutusu gösterebilir. Devam etmek için “aç” ı tıklayın.

Asıl yükleme işlemini başlatmak için Xamarin yazılımı lisans koşullarını okuyup kabul etmelisiniz. “Lisans koşullarını kabul ediyorum” onay kutusunu işaretleyin ve otomatik kullanım ve hata raporlama talebini not edin.

Kurulumdaki bir sonraki adım, kurulacak ürünleri seçmektir. Öğeler çoğunlukla açıklayıcıdır ancak Intel® HAXM bazı geliştiricilere yabancı olabilir. Intel® Hardware Accelerated Execution Manager anlamına gelir ve Android emülasyonunu daha hızlı hale getirir.

Sistemde zaten kurulu olan ürünler gösterilir ancak gri renkte gösterilir.

Ürünleri seçtikten sonra, Xamarin Unified yükleyicisi her bir yükleyiciyi otomatik olarak indirecek ve yürütecektir. Son adımda Xamarin.Android seçilmişse, Android SDK için kurulum konumunu seçmeniz istenir. Varsayılan konum çoğu durumda güvenli bir seçimdir, bu nedenle devam etmek için “Devam”a basın.

Son olarak, yükleyici neyin indirilip kurulacağının kısa bir özetini gösterecektir. Bu örnekte Xamarin.Android henüz yüklenmedi, bu nedenle diğer önkoşullarla birlikte listede gösteriliyor.

“Devam” düğmesine tıklayarak her ürün için indirme ve kurulum süreci başlar. Yükleyici, mevcut sistem kullanıcısının kullanıcı adı ve parolasını soran bir iletişim kutusu göstererek sistemde değişiklik yapmak için izin isteyebilir. Ayrıntıları girin ve kuruluma devam etmek için “Tamam” ı tıklayın.

Kurulum tamamlandıktan sonra Xamarin Studio başlatılabilir. Topluluk sürümü ücretsizdir ve oturum açmayı gerektirmez, ancak Enterprise özelliklerini kullanmak için bir hesap oluşturulması ve deneme sürümünün etkinleştirilmesi gerekir.

Xamarin Studio kullanarak Merhaba Dünya : Xamarin.Forms #

Xamarin Studio’yu OS X’e başarıyla yükledikten sonra. İlk Hello World Uygulamasının zamanı geldi.

Merhaba Dünya Uygulaması: Xamarin.Forms

Xamarin Formları Nedir :

Xamarin.Forms, tek bir paylaşılan C# kod tabanından iOS, Android ve Windows Phone için yerel kullanıcı arabirimleri oluşturmanıza olanak tanıyan yeni bir kitaplıktır. Çalışma zamanında yerel kontrollerle eşlenen 40’tan fazla platformlar arası kontrol ve düzen sağlar; bu, kullanıcı arayüzlerinizin tamamen yerel olduğu anlamına gelir.

Aşama 1:

Yeni bir çözüm oluşturun.

“Yeni Çözüm” e tıklayın

Adım 2: Forms Uygulamasını seçin ve İleri’ye tıklayın

Aşama 3: Uygulama adı ekleyin ve İleri’ye tıklayın

Çözüm oluşturulduğunda proje darlığı şu şekilde görünecektir:

App.xaml:

<?xml version="1.0" encoding="utf-8"?>
<Application xmlns="http://xamarin.com/schemas/2014/forms" 
    xmlns:x="http://schemas.microsoft.com/winfx/2009/xaml"
    x:Class="HelloXamarinForms.App">
    <Application.Resources>
        <!-- Application resource dictionary -->
    </Application.Resources>
</Application>

App.xaml.cs:

using Xamarin.Forms;

namespace HelloXamarinForms
{
    public partial class App : Application
    {
        public App()
        {
            InitializeComponent();

            MainPage = new HelloXamarinFormsPage();
        }

        protected override void OnStart()
        {
            // Handle when your app starts
        }

        protected override void OnSleep()
        {
            // Handle when your app sleeps
        }

        protected override void OnResume()
        {
            // Handle when your app resumes
        }
    }
}

MerhabaXamarinFormsPage.xaml

<?xml version="1.0" encoding="utf-8"?>
<ContentPage xmlns="http://xamarin.com/schemas/2014/forms"
    xmlns:x="http://schemas.microsoft.com/winfx/2009/xaml"
    xmlns:local="clr-namespace:HelloXamarinForms"
    x:Class="HelloXamarinForms.HelloXamarinFormsPage">

    <Label Text="Welcome to Xamarin Forms!" VerticalOptions="Center"
        HorizontalOptions="Center" />
</ContentPage>

MerhabaXamarinFormsPage.xaml.cs

using Xamarin.Forms;

namespace HelloXamarinForms
{
    public partial class HelloXamarinFormsPage : ContentPage
    {
        public HelloXamarinFormsPage()
        {
            InitializeComponent();
        }
    }
}
