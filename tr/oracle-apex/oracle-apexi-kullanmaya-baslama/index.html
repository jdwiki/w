Apex ile Oluşturulan Site #

Oracle application express ile oluşturulmuş uygulamaların listesini gösteren Apex ile Oluşturulan Site.

Merhaba Dünya uygulaması #

“Merhaba Dünya!” oluşturma adımları Oracle Application Express’te uygulama:

  1. Çalışma alanınızda oturum açın.
  2. Uygulama Oluşturucu‘u tıklayın. Uygulama Oluşturucu sayfası açılır.
  3. Oluştur‘u tıklayın. 4 uygulama tipine sahip sayfa görünür. Sayfa, her uygulama türü hakkında kısa yardım metni içerir.
  4. Masaüstü‘ye tıklayın. Bazı uygulama özelliklerine sahip sayfa görünür. Seçmek:
    • Schema: default database schema for the application.
    • Name: the name of your application. By default it will be displayed in the page header on all pages.
    • Application: ID number, which have to be unique through all workspaces. By default APEX suggests next free number.
    • Theme and Theme Style: define appearance of application pages.
  5. Uygulama Oluştur‘u tıklayın. Uygulama başlatmaya hazır.
  6. Uygulamayı Çalıştır‘ı tıklayın. Uygulamanızın giriş sayfası görüntülenir.

Oluşturulan masaüstü uygulamasının varsayılan olarak iki sayfası vardır: oturum açma sayfası ve Ana sayfa. Ana sayfa, APEX çalışma alanınız için giriş ve şifre girdiğini görmek için erişimi kısıtladı.

Ücretsiz deneyin #

Resmi sitede ücretsiz Oracle Application Express çalışma alanı talep edebilirsiniz: apex.oracle.com.

Kurulum veya Kurulum #

Oracle 11g veya 12cR1’i kurduğunuzda, Apex’in eski bir sürümü önceden yüklenir. En son sürüme yükseltmeniz şiddetle tavsiye edilir.

  1. apex.oracle.com adresine gidin ve Apex’in en son sürümünü indirin.

  2. Yüklemek için belirli sürümün belgelerini izleyin.

  3. Son Üç Sürüm İçin Önemli Belge Bağlantısı

  4. Oracle Uygulama Ekspres Sürümü 4.1

  5. Oracle Uygulama Ekspres Sürümü 4.2

  6. Oracle Uygulama Ekspres Sürümü 5

  7. Oracle Uygulama Ekspres Sürümü 5.1

  8. Oracle Apex Developer iseniz, birçok sorununuzu çözen Denes Kubicek Demo Uygulaması bir kez ziyaret edin.
