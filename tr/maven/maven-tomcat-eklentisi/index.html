maven eklentisini kullanarak Tomcat’i başlatın. #

Örnekte Tomcat 7’yi maven eklentisi kullanarak başlatacağız, isteğe bağlı olarak REST bitiş noktası için kullanıcı/şifre koruması ekleyeceğiz. Ayrıca bina savaşı özelliği de ekleniyor.

Tomcat için pom eklenti bölümüne aşağıdaki bölümü ekleyin

             <plugin>
                <groupId>org.apache.tomcat.maven</groupId>
                <artifactId>tomcat7-maven-plugin</artifactId>
                <version>2.2</version>
                <configuration>
                    <url>http://localhost:8090/manager</url>
                    <server>localhost</server>
                    <port>8191</port>
                    <path>/${project.build.finalName}</path>
                    <tomcatUsers>src/main/tomcatconf/tomcat-users.xml</tomcatUsers>
                </configuration>
            </plugin>

Maven war eklentisinin eklendiğinden ve web.xml’nin konumda bulunduğundan emin olun /src/main/webapp/WEB-INF/web.xml. Aşağıda savaş eklentisi örneği verilmiştir.

<plugin>
            <artifactId>maven-war-plugin</artifactId>
            <version>2.3</version>
            </plugin>
            <plugin>
                <artifactId>maven-compiler-plugin</artifactId>
                <version>3.1</version>
                <configuration>
                    <source>1.7</source>
                    <target>1.7</target>
                    <webResources>
                        <resource>
                            <!-- this is relative to the pom.xml directory -->
                            <directory>/src/main/webapp/WEB-INF/web.xml</directory>
                        </resource>
                    </webResources>
                </configuration>
            </plugin>

İsteğe bağlı olarak, tomcat-users.xml dosyasını src/main/tomcatconf konumuna ekleyin. Tomcat başladığında otomatik olarak kopyalanacaktır.

<tomcat-users>
    <user name="user" password="password" roles="admin" />
</tomcat-users>

İsteğe bağlı olarak, REST URL’sini korumak için web.xml’de aşağıdaki girişi ekleyin.

    <!-- tomcat user -->
    <security-constraint>
        <web-resource-collection>
            <web-resource-name>Wildcard means whole app requires authentication</web-resource-name>
            <url-pattern>/helloworld/*</url-pattern>
            <http-method>GET</http-method>
        </web-resource-collection>
        <auth-constraint>
            <role-name>admin</role-name>
        </auth-constraint>
        <user-data-constraint>
            <transport-guarantee>NONE</transport-guarantee>
        </user-data-constraint>
    </security-constraint>
    <login-config>
        <auth-method>BASIC</auth-method>
    </login-config>

Eclipse’den yeni maven yapısı oluşturun. Savaş projesini seçin ve Hedefler bölümünde aşağıdaki komutu ekleyin.

tomcat7:run

mesajı göreceksiniz.

[BİLGİ] — tomcat7-maven-plugin:2.2:run (varsayılan-cli) @ web-service-ldap2 — [BİLGİ] http://localhost:8191/ üzerinde savaş yürütüyor
