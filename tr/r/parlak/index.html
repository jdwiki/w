Bir uygulama oluşturun #

Shiny, RStudio tarafından geliştirilen ve R’de bir analizin sonuçlarını etkileşimli olarak görüntülemek için web sayfalarının oluşturulmasına izin veren bir R paketidir.

Shiny uygulaması oluşturmanın iki basit yolu vardır:

  • bir .R dosyasında veya
  • iki dosyada: “ui.R” ve “server.R”.

Shiny uygulaması iki bölüme ayrılmıştır:

  • ui: Düzeni ve görünümü kontrol eden bir kullanıcı arayüzü komut dosyası uygulamanın
  • sunucu: Uygulamanın tepki vermesine izin veren kodu içeren bir sunucu komut dosyası.

Bir dosya #

library(shiny)

# Create the UI
ui <- shinyUI(fluidPage(
    # Application title
    titlePanel("Hello World!")
))

# Create the server function
server <- shinyServer(function(input, output){})

# Run the app
shinyApp(ui = ui, server = server)

İki dosya #

‘ui.R’ dosyası oluştur #

library(shiny)

# Define UI for application
shinyUI(fluidPage(  
  # Application title
  titlePanel("Hello World!")
))

server.R dosyası oluşturun #

library(shiny)

# Define server logic
shinyServer(function(input, output){})

Onay Kutusu Grubu #

Birden çok seçeneği bağımsız olarak değiştirmek için kullanılabilecek bir onay kutusu grubu oluşturun. Sunucu, girişi seçilen değerlerin karakter vektörü olarak alacaktır.

library(shiny)

ui <- fluidPage(    
  checkboxGroupInput("checkGroup1", label = h3("This is a Checkbox group"), 
                 choices = list("1" = 1, "2" = 2, "3" = 3),
                 selected = 1),
  fluidRow(column(3, verbatimTextOutput("text_choice")))
  ) 


server <- function(input, output){
  output$text_choice <- renderPrint({
    return(paste0("You have chosen the choice ",input$checkGroup1))})
} 

shinyApp(ui = ui, server = server)

Ayarları değiştirmek mümkündür:

  • etiket: başlık
  • seçimler: seçilen değerler
  • seçili : Başlangıçta seçilen değer (seçim yokken NULL)
  • satır içi : yatay veya dikey
  • Genişlik

HTML eklemek de mümkündür.

Radyo düğmesi #

Listeden bir öğe seçmek için kullanılan bir dizi radyo düğmesi oluşturabilirsiniz.

Ayarları değiştirmek mümkündür:

  • seçili : Başlangıçta seçilen değer (seçim olmaması için karakter(0))
  • satır içi : yatay veya dikey
  • Genişlik

HTML eklemek de mümkündür.

library(shiny)

ui <- fluidPage(    
  radioButtons("radio", 
               label = HTML('<FONT color="red"><FONT size="5pt">Welcome</FONT></FONT><br> <b>Your favorite color is red ?</b>'),
               choices = list("TRUE" = 1, "FALSE" = 2),
               selected = 1,
               inline = T,
               width = "100%"),      
  fluidRow(column(3, textOutput("value")))) 


server <- function(input, output){
  output$value <- renderPrint({
    if(input$radio == 1){return('Great !')}
     else{return("Sorry !")}})}

shinyApp(ui = ui, server = server)

Seç kutusu #

Değer listesinden tek veya birden çok öğe seçmek için kullanılabilecek bir seçim listesi oluşturun.

library(shiny)

ui <- fluidPage(    
  selectInput("id_selectInput", 
          label = HTML('<B><FONT size="3">What is your favorite color ?</FONT></B>'), 
          multiple = TRUE,
          choices = list("red" = "red", "green" = "green", "blue" = "blue", "yellow" = "yellow"), 
          selected = NULL),
  br(), br(),
  fluidRow(column(3, textOutput("text_choice")))) 


server <- function(input, output){
  output$text_choice <- renderPrint({
    return(input$id_selectInput)})
}

shinyApp(ui = ui, server = server)

Ayarları değiştirmek mümkündür:

  • etiket: başlık
  • seçimler: seçilen değerler
  • seçili : Başlangıçta seçilen değer (seçim yokken NULL)
  • çoklu : DOĞRU veya YANLIŞ
  • Genişlik
  • boyut
  • selectize: DOĞRU veya YANLIŞ (selectize.js’yi kullanmak veya kullanmamak için ekranı değiştirin)

HTML eklemek de mümkündür.

Parlak bir uygulama başlatın #

Uygulamanızı nasıl oluşturduğunuza bağlı olarak, bir uygulamayı birkaç yolla başlatabilirsiniz. Uygulamanız “ui.R” ve “server.R” olmak üzere iki dosyaya bölünmüşse veya uygulamanızın tamamı tek bir dosyadaysa.

1. İki dosya uygulaması #

İki dosyanız ‘ui.R’ ve ‘server.R’ aynı klasörde olmalıdır. Ardından konsolda shinyApp() işlevini çalıştırarak ve Shiny uygulamasını içeren dizinin yolunu geçerek uygulamanızı başlatabilirsiniz.

shinyApp("path_to_the_folder_containing_the_files")

Bir “ui.R” veya “server.R” dosyası açtığınızda Rstudio’da görünen Uygulamayı Çalıştır düğmesine basarak uygulamayı doğrudan Rstudio’dan da başlatabilirsiniz.

Veya çalışma dizininiz Shiny App dizini ise konsola basitçe runApp() yazabilirsiniz.

2. Tek dosya uygulaması #

Eğer bir R dosyasında kendi dosyanızı oluşturursanız, onu shinyApp() işleviyle de başlatabilirsiniz.

  • kodunuzun içinde:

    library(shiny)

    ui <- fluidPage() #Create the ui server <- function(input, output){} #create the server

    shinyApp(ui = ui, server = server) #run the App

  • konsolda, “appFile” parametresiyle Shiny uygulamasını içeren bir “.R” dosyasına yol ekleyerek:

    shinyApp(appFile=“path_to_my_R_file_containig_the_app”)

Hata ayıklama #

debug() ve debugonce() çoğu Shiny hata ayıklama bağlamında iyi çalışmayacaktır. Bununla birlikte, kritik yerlere eklenen tarayıcı() ifadeleri, Shiny kodunuzun nasıl çalıştığına (çalışmadığına) dair size birçok fikir verebilir. Ayrıca bkz: [tarayıcı() kullanarak hata ayıklama](https://www.wikiod.com/tr/r/hata-ayklama#Tarayıcıyı kullanma)

Vitrin modu #

Vitrin modu, uygulamanızı onu oluşturan kodun yanında görüntüler ve server.R onları çalıştırırken kod satırlarını vurgular.

Vitrin modunu etkinleştirmenin iki yolu vardır:

  • Shiny uygulamasını display.mode = “showcase”, örneğin runApp("MyApp", display.mode = "showcase") argümanıyla başlatın.
  • Shiny uygulama klasörünüzde ‘DESCRIPTION’ adlı bir dosya oluşturun ve içine şu satırı ekleyin: DisplayMode: Showcase.

Reaktif Günlük Görüntüleyici #

Reaktif Günlük Görüntüleyici, uygulamanızdaki reaktif bağımlılıkları ve yürütmeyi görselleştirmek için tarayıcı tabanlı etkileşimli bir araç sağlar. Reaktif Günlük Görüntüleyiciyi etkinleştirmek için, R konsolunda options(shiny.reactlog=TRUE) komutunu çalıştırın ve veya bu kod satırını server.R dosyanıza ekleyin. Reactive Log Visualizer’ı başlatmak için, uygulamanız çalışırken Windows’ta Ctrl+F3 veya Mac’te Command+F3 tuşlarına basın. Reactive Log Visualizer’da gezinmek için sol ve sağ ok tuşlarını kullanın.

library(shiny)

# Create the UI
ui <- shinyUI(fluidPage(
  titlePanel("Basic widgets"),
  
  fluidRow(
    
    column(3,
           h3("Buttons"),
           actionButton("action", label = "Action"),
           br(),
           br(), 
           submitButton("Submit")),
    
    column(3,
           h3("Single checkbox"),
           checkboxInput("checkbox", label = "Choice A", value = TRUE)),
    
    column(3, 
           checkboxGroupInput("checkGroup", 
                              label = h3("Checkbox group"), 
                              choices = list("Choice 1" = 1, 
                                             "Choice 2" = 2, "Choice 3" = 3),
                              selected = 1)),
    
    column(3, 
           dateInput("date", 
                     label = h3("Date input"), 
                     value = "2014-01-01"))   
  ),
  
  fluidRow(
    
    column(3,
           dateRangeInput("dates", label = h3("Date range"))),
    
    column(3,
           fileInput("file", label = h3("File input"))),
    
    column(3, 
           h3("Help text"),
           helpText("Note: help text isn't a true widget,", 
                    "but it provides an easy way to add text to",
                    "accompany other widgets.")),
    
    column(3, 
           numericInput("num", 
                        label = h3("Numeric input"), 
                        value = 1))   
  ),
  
  fluidRow(
    
    column(3,
           radioButtons("radio", label = h3("Radio buttons"),
                        choices = list("Choice 1" = 1, "Choice 2" = 2,
                                       "Choice 3" = 3),selected = 1)),
    
    column(3,
           selectInput("select", label = h3("Select box"), 
                       choices = list("Choice 1" = 1, "Choice 2" = 2,
                                      "Choice 3" = 3), selected = 1)),
    
    column(3, 
           sliderInput("slider1", label = h3("Sliders"),
                       min = 0, max = 100, value = 50),
           sliderInput("slider2", "",
                       min = 0, max = 100, value = c(25, 75))
    ),
    
    column(3, 
           textInput("text", label = h3("Text input"), 
                     value = "Enter text..."))   
  )
  
))

# Create the server function
server <- shinyServer(function(input, output){})

# Run the app
shinyApp(ui = ui, server = server)
