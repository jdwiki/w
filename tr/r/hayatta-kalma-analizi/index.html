randomForestSRC ile Rastgele Orman Hayatta Kalma Analizi #

Rastgele orman algoritması regresyon ve sınıflandırma görevlerine uygulanabileceği gibi, hayatta kalma analizine de genişletilebilir.

Aşağıdaki örnekte, bir hayatta kalma modeli uygundur ve ‘randomForestSRC’ CRAN’dan paketi kullanılarak tahmin, puanlama ve performans analizi için kullanılır.

require(randomForestSRC)

set.seed(130948) #Other seeds give similar comparative results
x1   <- runif(1000)
y    <- rnorm(1000, mean = x1, sd = .3)
data <- data.frame(x1 = x1, y = y)
head(data)

x1 y 1 0.9604353 1.3549648 2 0.3771234 0.2961592 3 0.7844242 0.6942191 4 0.9860443 1.5348900 5 0.1942237 0.4629535 6 0.7442532 -0.0672639

(modRFSRC <- rfsrc(y ~ x1, data = data, ntree=500, nodesize = 5))

Örnek boyutu: 1000 Ağaç sayısı: 500 Minimum terminal düğüm boyutu: 5 Ortalama hayır. terminal düğüm sayısı: 208.258 Her bölmede denenen değişken sayısı: 1 Toplam numara değişken sayısı: 1 Analiz: RF-R Aile: reg Bölme kuralı: mse Açıklanan % varyans: 32.08 Hata oranı: 0.11

x1new   <- runif(10000)
ynew    <- rnorm(10000, mean = x1new, sd = .3)
newdata <- data.frame(x1 = x1new, y = ynew)

survival.results <- predict(modRFSRC, newdata = newdata)
survival.results

Test (tahmin) verilerinin örnek boyutu: 10000 Büyüyen ağaç sayısı: 500 Ortalama hayır. büyüme terminal düğümlerinin sayısı: 208.258 Toplam numara büyüme değişkenleri: 1 Analiz: RF-R Aile: reg Açıklanan % varyans: 34.97 Test seti hata oranı: 0.11

Giriş - hayatta kalma paketi ile parametrik hayatta kalma modellerinin temel uyumu ve çizimi #

“survival”, R’de hayatta kalma analizi için en yaygın kullanılan pakettir. Yerleşik “akciğer” veri kümesini kullanarak, “survreg()” işleviyle bir regresyon modeli uydurarak ve " ile bir eğri oluşturarak Hayatta Kalma Analizine başlayabiliriz. survfit()` ve bu paket için yeni verilerle “predict” yöntemini çağırarak tahmin edilen hayatta kalma eğrilerini çizme.

Aşağıdaki örnekte, etkisini görselleştirmek için tahmin edilen 2 eğriyi çiziyoruz ve 2 yeni veri seti arasında “cinsiyet"i değiştiriyoruz:

require(survival)
s <- with(lung,Surv(time,status))

sWei  <- survreg(s ~ as.factor(sex)+age+ph.ecog+wt.loss+ph.karno,dist='weibull',data=lung)

fitKM <- survfit(s ~ sex,data=lung)
plot(fitKM)

lines(predict(sWei, newdata = list(sex      = 1, 
                                   age      = 1, 
                                   ph.ecog  = 1, 
                                   ph.karno = 90,
                                   wt.loss  = 2),
                                 type = "quantile",
                                 p    = seq(.01, .99, by = .01)),
                                 seq(.99, .01, by        =-.01),
                                 col = "blue")

lines(predict(sWei, newdata = list(sex      = 2, 
                                   age      = 1, 
                                   ph.ecog  = 1, 
                                   ph.karno = 90,
                                   wt.loss  = 2),
                                 type = "quantile",
                                 p    = seq(.01, .99, by = .01)),
                                 seq(.99, .01, by        =-.01),
                                 col = "red")

survminer ile Kaplan Meier hayatta kalma eğrileri ve risk seti tabloları tahminleri #

temel arsa

install.packages('survminer')
source("https://bioconductor.org/biocLite.R")
biocLite("RTCGA.clinical") # data for examples
library(RTCGA.clinical)
survivalTCGA(BRCA.clinical, OV.clinical,
             extract.cols = "admin.disease_code") -> BRCAOV.survInfo
library(survival)
fit <- survfit(Surv(times, patient.vital_status) ~ admin.disease_code,
               data = BRCAOV.survInfo)
library(survminer)
ggsurvplot(fit, risk.table = TRUE)

Daha ileri

ggsurvplot(
   fit,                     # survfit object with calculated statistics.
   risk.table = TRUE,       # show risk table.
   pval = TRUE,             # show p-value of log-rank test.
   conf.int = TRUE,         # show confidence intervals for 
                            # point estimaes of survival curves.
   xlim = c(0,2000),        # present narrower X axis, but not affect
                            # survival estimates.
   break.time.by = 500,     # break X axis in time intervals by 500.
   ggtheme = theme_RTCGA(), # customize plot and risk table with a theme.
 risk.table.y.text.col = T, # colour risk table text annotations.
  risk.table.y.text = FALSE # show bars instead of names in text annotations
                            # in legend of risk table
)
