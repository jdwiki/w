Verileri doğrudan bir R oturumuna alın. R’nin güzel özelliklerinden biri veri toplama kolaylığıdır. R paketlerini kullanarak veri yaymanın birkaç yolu vardır.

Yerleşik veri kümeleri #

R, geniş bir yerleşik veri kümesi koleksiyonuna sahiptir. Genellikle, hızlı ve kolay tekrarlanabilir örnekler oluşturmak için öğretim amacıyla kullanılırlar. Yerleşik veri kümelerini listeleyen güzel bir web sayfası var:

https://vincentarelbundock.github.io/Rdatasets/datasets.html

Örnek #

İsviçre Doğurganlığı ve Sosyoekonomik Göstergeleri (1888) Verileri. Kırsallık ve Katolik nüfusun egemenliğine dayalı doğurganlık farkını kontrol edelim.

library(tidyverse) 

swiss %>% 
        ggplot(aes(x = Agriculture, y = Fertility, 
                   color = Catholic > 50))+
        geom_point()+
        stat_ellipse()

Açık veritabanlarına erişmek için paketler #

Bazı veritabanlarına erişmek için özel olarak çok sayıda paket oluşturulur. Bunları kullanmak, verileri okumak/biçimlendirmek için bir sürü zaman kazandırabilir.

Eurostat #

’eurostat’ paketinin ‘search_eurostat()’ işlevi olmasına rağmen, mevcut tüm ilgili veri kümelerini bulamıyor. Bu, bir veri kümesinin koduna Eurostat web sitesinde manuel olarak göz atmak için daha uygundur: Ülkeler Veritabanı veya Bölgesel Veritabanı. Otomatik indirme çalışmazsa, veriler Toplu İndirme Tesisi aracılığıyla manuel olarak alınabilir.

library(tidyverse)
library(lubridate)
library(forcats)
library(eurostat)
library(geofacet)
library(viridis)
library(ggthemes)
library(extrafont)

# download NEET data for countries
neet <- get_eurostat("edat_lfse_22")

neet %>% 
        filter(geo %>% paste %>% nchar == 2,
               sex == "T", age == "Y18-24") %>%
        group_by(geo) %>% 
        mutate(avg = values %>% mean()) %>% 
        ungroup() %>% 
        ggplot(aes(x = time %>% year(),
                   y = values))+
        geom_path(aes(group = 1))+
        geom_point(aes(fill = values), pch = 21)+
        scale_x_continuous(breaks = seq(2000, 2015, 5),
                           labels = c("2000", "'05", "'10", "'15"))+
        scale_y_continuous(expand = c(0, 0), limits = c(0, 40))+
        scale_fill_viridis("NEET, %", option = "B")+
        facet_geo(~ geo, grid = "eu_grid1")+
        labs(x = "Year",
             y = "NEET, %",
             title = "Young people neither in employment nor in education and training in Europe",
             subtitle = "Data: Eurostat Regional Database, 2000-2016",
             caption = "ikashnitsky.github.io")+
        theme_few(base_family =  "Roboto Condensed", base_size = 15)+
        theme(axis.text = element_text(size = 10),
              panel.spacing.x = unit(1, "lines"),
              legend.position = c(0, 0),
              legend.justification = c(0, 0))

Kısıtlanmış verilere erişmek için paketler #

İnsan Ölümleri Veritabanı #

İnsan Ölümleri Veritabanı, az çok güvenilir istatistiklerin mevcut olduğu ülkeler için insan ölüm verilerini toplayan ve ön işleyen Max Planck Nüfus Araştırmaları Enstitüsü’nin bir projesidir.

# load required packages
library(tidyverse) 
library(extrafont)
library(HMDHFDplus)

country <- getHMDcountries()

exposures <- list()
for (i in 1: length(country)) {
        cnt <- country[i]
        exposures[[cnt]] <- readHMDweb(cnt, "Exposures_1x1", user_hmd, pass_hmd)
        # let's print the progress
        paste(i,'out of',length(country)) 
} # this will take quite a lot of time

Lütfen, “user_hmd” ve “pass_hmd” argümanlarının İnsan Ölümleri Veritabanının web sitesindeki oturum açma kimlik bilgileri olduğunu unutmayın. Verilere erişmek için http://www.mortality.org/ adresinde bir hesap oluşturmalı ve readHMDweb() işlevine kendi kimlik bilgilerini sağlamalıdır.

sr_age <- list()

for (i in 1:length(exposures)) {
        di <- exposures[[i]]
        sr_agei <- di %>% select(Year,Age,Female,Male) %>% 
                filter(Year %in% 2012) %>%
                select(-Year) %>%
                transmute(country = names(exposures)[i],
                          age = Age, sr_age = Male / Female * 100)
        sr_age[[i]] <- sr_agei
}
sr_age <- bind_rows(sr_age)

# remove optional populations
sr_age <- sr_age %>% filter(!country %in% c("FRACNP","DEUTE","DEUTW","GBRCENW","GBR_NP"))

# summarize all ages older than 90 (too jerky)
sr_age_90 <- sr_age %>% filter(age %in% 90:110) %>% 
        group_by(country) %>% summarise(sr_age = mean(sr_age, na.rm = T)) %>%
        ungroup() %>% transmute(country, age=90, sr_age)

df_plot <- bind_rows(sr_age %>% filter(!age %in% 90:110), sr_age_90)

# finaly - plot
df_plot %>% 
        ggplot(aes(age, sr_age, color = country, group = country))+
        geom_hline(yintercept = 100, color = 'grey50', size = 1)+
        geom_line(size = 1)+
        scale_y_continuous(limits = c(0, 120), expand = c(0, 0), breaks = seq(0, 120, 20))+
        scale_x_continuous(limits = c(0, 90), expand = c(0, 0), breaks = seq(0, 80, 20))+
        xlab('Age')+
        ylab('Sex ratio, males per 100 females')+
        facet_wrap(~country, ncol=6)+
        theme_minimal(base_family = "Roboto Condensed", base_size = 15)+
        theme(legend.position='none',
              panel.border = element_rect(size = .5, fill = NA))

Paketler içindeki veri kümeleri #

Veri içeren veya özellikle veri kümelerini yaymak için oluşturulmuş paketler vardır. Böyle bir paket yüklendiğinde (kütüphane(pkg)), ekli veri kümeleri ya R nesneleri olarak kullanılabilir hale gelir; veya ‘data()’ işleviyle çağrılmaları gerekir.

Boş anılar #

Ülkelerin gelişimine dair güzel bir veri seti.

library(tidyverse)
library(gapminder)

gapminder %>% 
        ggplot(aes(x = year, y = lifeExp, 
                   color = continent))+
        geom_jitter(size = 1, alpha = .2, width = .75)+
        stat_summary(geom = "path", fun.y = mean, size = 1)+
        theme_minimal()

Dünya Nüfus Beklentileri 2015 - Birleşmiş Milletler Nüfus Dairesi #

Dünyanın 1950-2015 yılları arasında doğumda erkek yaşam beklentisinde nasıl bir araya geldiğini görelim.

library(tidyverse)
library(forcats)
library(wpp2015)
library(ggjoy)
library(viridis)
library(extrafont)

data(UNlocations)

countries <- UNlocations %>% 
        filter(location_type == 4) %>% 
        transmute(name = name %>% paste()) %>% 
        as_vector()

data(e0M) 

e0M %>% 
        filter(country %in% countries) %>% 
        select(-last.observed) %>% 
        gather(period, value, 3:15) %>% 
        ggplot(aes(x = value, y = period %>% fct_rev()))+
        geom_joy(aes(fill = period))+
        scale_fill_viridis(discrete = T, option = "B", direction = -1, 
                           begin = .1, end = .9)+
        labs(x = "Male life expectancy at birth",
             y = "Period",
             title = "The world convergence in male life expectancy at birth since 1950",
             subtitle = "Data: UNPD World Population Prospects 2015 Revision",
             caption = "ikashnitsky.github.io")+
        theme_minimal(base_family =  "Roboto Condensed", base_size = 15)+
        theme(legend.position = "none")
