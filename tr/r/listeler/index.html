Listelere Hızlı Giriş #

Genel olarak, bir kullanıcı olarak etkileşimde bulunacağınız nesnelerin çoğu bir vektör olma eğiliminde olacaktır; örneğin sayısal vektör, mantıksal vektör. Bu nesneler yalnızca tek bir değişken türü alabilir (sayısal bir vektörün içinde yalnızca sayılar olabilir).

Bir liste, içinde herhangi bir tür değişkeni saklayabilir, bu da onu, ihtiyacımız olan herhangi bir değişken türünü depolayabilen genel nesne haline getirir.

Liste başlatma örneği

exampleList1 <- list('a', 'b')
exampleList2 <- list(1, 2)
exampleList3 <- list('a', 1, 2)

Listede tanımlanan verileri anlamak için str fonksiyonunu kullanabiliriz.

str(exampleList1)
str(exampleList2)
str(exampleList3)

Listelerin alt kümelenmesi, listenin bir diliminin çıkarılması, yani orijinal listedeki öğelerin bir alt kümesini içeren bir listenin elde edilmesi ile tek bir öğenin çıkarılması arasında ayrım yapar. Vektörler için yaygın olarak kullanılan [ operatörünün kullanılması yeni bir liste oluşturur.

# Returns List
exampleList3[1]
exampleList3[1:2]

Tek bir öğe elde etmek için bunun yerine [[ kullanın.

# Returns Character
exampleList3[[1]]

Liste girişleri şu şekilde adlandırılabilir:

exampleList4 <- list(
    num = 1:3,
    numeric = 0.5,
    char = c('a', 'b')
)

Adlandırılmış listelerdeki girdilere dizinleri yerine adlarıyla erişilebilir.

exampleList4[['char']]

Alternatif olarak, adlandırılmış öğelere erişmek için $ operatörü kullanılabilir.

exampleList4$num

Bu, yazmanın daha hızlı olması ve okunması daha kolay olması avantajına sahiptir, ancak olası bir tuzaktan haberdar olmak önemlidir. $ operatörü, eşleşen liste öğelerini belirlemek için kısmi eşleştirme kullanır ve beklenmeyen sonuçlar üretebilir.

exampleList5 <- exampleList4[2:3]

exampleList4$num
# c(1, 2, 3)

exampleList5$num
# 0.5

exampleList5[['num']]
# NULL

Listeler, farklı uzunluklardaki ve çeşitli sınıflardaki nesneleri depolayabildikleri için özellikle yararlı olabilir.

## Numeric vector
exampleVector1 <- c(12, 13, 14)
## Character vector
exampleVector2 <- c("a", "b", "c", "d", "e", "f")
## Matrix
exampleMatrix1 <- matrix(rnorm(4), ncol = 2, nrow = 2)
## List
exampleList3 <- list('a', 1, 2)

exampleList6 <- list(
    num = exampleVector1, 
    char = exampleVector2,
    mat = exampleMatrix1, 
    list = exampleList3
)
exampleList6
#$num
#[1] 12 13 14
#
#$char
#[1] "a" "b" "c" "d" "e" "f"
#
#$mat
#          [,1]        [,2]
#[1,] 0.5013050 -1.88801542
#[2,] 0.4295266  0.09751379
#
#$list
#$list[[1]]
#[1] "a"
#
#$list[[2]]
#[1] 1
#
#$list[[3]]
#[1] 2

Listelere giriş #

Serileştirme: bilgileri iletmek için listeleri kullanma #

Farklı türlerdeki verileri bir araya getirmenin gerekli olduğu durumlar vardır. Örneğin Azure ML’de, bilgileri bir R betik modülünden diğerine özel olarak veri çerçeveleri aracılığıyla iletmek gerekir. Bir veri çerçevemiz ve bir numaramız olduğunu varsayalım:

> df
       name height        team fun_index title age         desc Y
1    Andrea    195       Lazio        97     6  33   eccellente 1
2      Paja    165  Fiorentina        87     6  31       deciso 1
3      Roro    190       Lazio        65     6  28       strano 0
4    Gioele     70       Lazio       100     0   2    simpatico 1
5     Cacio    170    Juventus        81     3  33         duro 0
6     Edola    171       Lazio        72     5  32     svampito 1
7    Salami    175       Inter        75     3  30  doppiopasso 1
8    Braugo    180       Inter        79     5  32          gjn 0
9     Benna    158    Juventus        80     6  28     esaurito 0
10   Riggio    182       Lazio        92     5  31     certezza 1
11 Giordano    185        Roma        79     5  29        buono 1

> number <- "42"

Bu bilgilere erişebiliriz:

> paste(df$name[4],"is a",df3$team[4], "supporter." )
[1] "Gioele is a  Lazio supporter."
> paste("The answer to THE question is", number )
[1] "The answer to THE question is 42"

Bir veri çerçevesine farklı veri türleri koymak için liste nesnesini ve serileştirmeyi kullanmalıyız. Özellikle verileri genel bir listeye koymamız ve ardından listeyi belirli bir veri çerçevesine koymamız gerekir:

l <- list(df,number)
dataframe_container <- data.frame(out2 = as.integer(serialize(l, connection=NULL)))

Bilgileri veri çerçevesinde depoladıktan sonra, kullanmak için seri hale getirmemiz gerekir:

#----- unserialize ----------------------------------------+
unser_obj <- unserialize(as.raw(dataframe_container$out2))
#----- taking back the elements----------------------------+
df_mod        <- unser_obj[1][[1]]  
number_mod    <- unser_obj[2][[1]]

Ardından, verilerin doğru şekilde aktarıldığını doğrulayabiliriz:

> paste(df_mod$name[4],"is a",df_mod$team[4], "supporter." )
[1] "Gioele is a  Lazio supporter."
> paste("The answer to THE question is", number_mod )

Listeleri kullanma nedenleri #

Ortalama bir R kullanıcısı için liste yapısı, işlenmesi daha karmaşık veri yapılarından biri gibi görünebilir. İçindeki tüm öğelerin aynı türden olduğuna dair hiçbir garanti yoktur; Listenin ne kadar karmaşık/karmaşık olmadığına dair garantili bir yapı yoktur (Listedeki bir öğe liste olabilir)

Ancak, işlevler arasında parametre iletmek için kullanmak için listelerin ne zaman kullanılacağının ana nedenlerinden biri.

# Function example which returns a single element numeric vector
exampleFunction1 <- function(num1, num2){
    result <- num1 + num2
    return(result)
}

# Using example function 1
exampleFunction1(1, 2)

# Function example which returns a simple numeric vector
exampleFunction2 <- function(num1, num2, multiplier){
    tempResult1 <- num1 + num2
    tempResult2 <- tempResult1 * multiplier
    result <- c(tempResult1, tempResult2)
    return(result) 
}

# Using example function 2
exampleFunction2(1, 2, 4)

Yukarıdaki örnekte, döndürülen sonuçlar yalnızca basit sayısal vektörlerdir. Bu kadar basit vektörleri geçmek için herhangi bir sorun yoktur.

Bu noktada genel olarak R işlevlerinin bir seferde yalnızca 1 sonuç döndürdüğünü belirtmek önemlidir (farklı sonuçlar döndürmek için if koşullarını kullanabilirsiniz). Ancak, bir dizi parametre alan ve sayısal bir vektör (ayar değeri) ve bir veri çerçevesi (hesaplamadan) gibi çeşitli türde sonuçlar döndüren bir işlev oluşturmayı düşünüyorsanız, tüm bu sonuçları bir listeye dökmeniz gerekir. iade etmeden önce.

# We will be using mtcars dataset here
# Function which returns a result that is supposed to contain multiple type of results
# This can be solved by putting the results into a list
exampleFunction3 <- function(dataframe, removeColumn, sumColumn){
    resultDataFrame <- dataframe[, -removeColumn]
    resultSum <- sum(dataframe[, sumColumn])
    resultList <- list(resultDataFrame, resultSum)
    return(resultList)
}

# Using example function 3
exampleResult <- exampleFunction3(mtcars, 2, 4)
exampleResult[[1]]
exampleResult[[2]]

Boş liste öğelerini korurken bir listeyi vektöre dönüştürün #

Bir listeyi bir vektöre veya data.frame nesnesine dönüştürmek istendiğinde, genellikle boş elemanlar bırakılır.

Bu, istenen uzunlukta bir listenin bazı boş değerlerle oluşturulduğunda sorunlu olabilir (örneğin, bir m x n matrisine, veri çerçevesine veya veri tablosuna eklenmek üzere n elemanlı bir liste oluşturulur). Ancak boş öğeleri koruyarak bir listeyi kayıpsız bir vektöre dönüştürmek mümkündür:
