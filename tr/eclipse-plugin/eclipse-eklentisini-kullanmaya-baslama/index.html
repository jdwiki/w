Selam Dünya #

Eclipse için bir Hello World eklentisi oluşturmak için tıklayın: Dosya ➜ Yeni ➜ Diğer…

Eklenti Projesi’ni seçin ve İleri’ye tıklayın >

Yeni Eklenti Projesi sihirbazı, yeni bir eklenti oluşturma seçeneklerinde size yol gösterecektir.

Bir proje adı girin (MerhabaDünya gibi) ve İleri >‘ye tıklayın

İçerik sayfasında, eklentinin ID, Sürüm, Ad ve Satıcı bilgilerini ayarlayabilirsiniz.

Sürüm, varsayılan olarak 1.0.0.qualifier olacaktır. Bunu olduğu gibi bırakabilirsiniz, ancak bunu anlamlı bir şeyle değiştirmek daha iyidir. Eclipse wiki vYYYYMMDD (yıl, ay gün) gibi bir sözdizimi önerir.

Şablonlar sayfasında, herhangi bir şablonu seçip İleri >‘yi tıklayarak eklentinizi oluşturmayı seçebilirsiniz. Alternatif olarak, Özel eklenti sihirbazı’nı seçerek bu şablonları birleştirebilir veya Şablonlardan birini kullanarak bir eklenti oluştur önündeki onay kutusunun seçimini kaldırarak şablonsuz yeni bir eklenti oluşturabilirsiniz.

Merhaba, World Command şablonu için ek ayarlar vardır: paket adı, İşleyici sınıfı adı ve mesaj kutusu metni.

Eklenti oluşturulduğunda plugin.xml ➜ Farklı Çalıştır ➜ Eclipse Uygulaması üzerine sağ tıklayarak çalıştırabilirsiniz.

Bu, eklentinizin yükleneceği yeni bir Eclipse örneğini (kendi çalışma alanıyla) başlatacaktır.

Bu Merhaba Dünya eklentisi, Eclipse GUI’ye 3 katkı sağlamış olacaktır:

1. Örnek Menü (Örnek Komutlu):

Eklenti.xml:

<extension
      point="org.eclipse.ui.menus">
   <menuContribution
         locationURI="menu:org.eclipse.ui.main.menu?after=additions">
      <menu
            label="Sample Menu"
            mnemonic="M"
            id="HelloWorld.menus.sampleMenu">
         <command
               commandId="HelloWorld.commands.sampleCommand"
               mnemonic="S"
               id="HelloWorld.menus.sampleCommand">
         </command>
      </menu>
   </menuContribution>
</extension>

2. Bir araç çubuğu simgesi:

Eklenti.xml:

<extension
      point="org.eclipse.ui.menus">
   <menuContribution
         locationURI="toolbar:org.eclipse.ui.main.toolbar?after=additions">
      <toolbar
            id="HelloWorld.toolbars.sampleToolbar">
         <command
               commandId="HelloWorld.commands.sampleCommand"
               icon="icons/sample.gif"
               tooltip="Say hello world"
               id="HelloWorld.toolbars.sampleCommand">
         </command>
      </toolbar>
   </menuContribution>
</extension>

3. Bir tuş kısayolu (Ctrl+6)

Eklenti.xml:

<extension
      point="org.eclipse.ui.bindings">
   <key
         commandId="HelloWorld.commands.sampleCommand"
         contextId="org.eclipse.ui.contexts.window"
         sequence="M1+6"
         schemeId="org.eclipse.ui.defaultAcceleratorConfiguration">
   </key>
</extension>

Bu 3’ten herhangi birini etkinleştirirken, Handler sınıfı yürütülür:

Eklenti.xml:

<extension
      point="org.eclipse.ui.commands">
   <category
         name="Sample Category"
         id="HelloWorld.commands.category">
   </category>
   <command
         name="Sample Command"
         categoryId="HelloWorld.commands.category"
         id="HelloWorld.commands.sampleCommand">
   </command>
</extension>
<extension
      point="org.eclipse.ui.handlers">
   <handler
         commandId="HelloWorld.commands.sampleCommand"
         class="helloworld.handlers.SampleHandler">
   </handler>
</extension>

SampleHandler.java:

package helloworld.handlers;

import org.eclipse.core.commands.AbstractHandler;
import org.eclipse.core.commands.ExecutionEvent;
import org.eclipse.core.commands.ExecutionException;
import org.eclipse.ui.IWorkbenchWindow;
import org.eclipse.ui.handlers.HandlerUtil;
import org.eclipse.jface.dialogs.MessageDialog;

/**
 * Our sample handler extends AbstractHandler, an IHandler base class.
 * @see org.eclipse.core.commands.IHandler
 * @see org.eclipse.core.commands.AbstractHandler
 */
public class SampleHandler extends AbstractHandler {

    @Override
    public Object execute(ExecutionEvent event) throws ExecutionException {
        IWorkbenchWindow window = HandlerUtil.getActiveWorkbenchWindowChecked(event);
        MessageDialog.openInformation(
                window.getShell(),
                "HelloWorld",
                "Hello, Eclipse world");
        return null;
    }
}

Hello World eklentisinin tek yaptığı bu.

Kurulum veya Kurulum #

Java Geliştiricileri için Eclipse IDE yüklü olduğunu varsayarak, Eclipse’i başlatın, “Yardım” -> “Yeni Yazılım Yükle…” öğesini tıklayın.

“Work with:” öğesinde “–Tüm Kullanılabilir Siteler–” öğesini seçin ve “Eclipse Plugin Development Tools” öğesine gidin. Önündeki onay kutusunu işaretleyerek “Eclipse Plug-in Development Environment” öğesini seçin.

Eclipse’in gereken bağımlılıkları kontrol etmesine izin vermek için “İleri“ye tıklayın. Kurulumu başlatmak için tekrar “İleri“ye tıklayın.

Bu bittiğinde, Eclipse’i yeniden başlatın.
