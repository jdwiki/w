OpenCV ile bir resim yükleyin ve görüntüleyin #

Bu örnekle, diskten renkli bir görüntünün nasıl yükleneceğini ve OpenCV’nin yerleşik işlevlerini kullanarak nasıl görüntüleneceğini göreceğiz. Bunu başarmak için C/C++, Python veya Java bağlamalarını kullanabiliriz.

C++‘da:

#include <opencv2/core.hpp>
#include <opencv2/highgui.hpp>

#include <iostream>

using namespace cv;

int main(int argc, char** argv) {
    // We'll start by loading an image from the drive
    Mat image = imread("image.jpg", CV_LOAD_IMAGE_COLOR);

    // We check that our image has been correctly loaded
    if(image.empty()) {
        std::cout << "Error: the image has been incorrectly loaded." << std::endl;
        return 0;
    }

    // Then we create a window to display our image
    namedWindow("My first OpenCV window");

    // Finally, we display our image and ask the program to wait for a key to be pressed
    imshow("My first OpenCV window", image);
    waitKey(0);

    return 0;
}

Python’da:

import sys
import cv2

# We load the image from disk
img = cv2.imread("image.jpg", cv2.CV_LOAD_IMAGE_COLOR)

# We check that our image has been correctly loaded
if img.size == 0
    sys.exit("Error: the image has not been correctly loaded.")

# We create a window to display our image
cv2.namedwindow("My first OpenCV window")

# We display our image and ask the program to wait until a key is pressed
cv2.imshow("My first OpenCV window", img)
cv2.waitKey(0)

# We close the window
cv2.destroyAllWindows()

Java’da:

import org.opencv.core.Core;
import org.opencv.core.Mat;
import org.opencv.core.CvType;
import org.opencv.highgui.Highgui;
public class Sample{
public static void main (String[] args) {

    //Load native opencv library
    System.loadLibrary(Core.NATIVE_LIBRARY_NAME);

    //Read image from file first param:file location ,second param:color space
    Mat img = imread("image.jpg",CV_LOAD_IMAGE_COLOR);

    //If the image is successfully read.
    if (img.size() == 0) {
        System.exit(1);
    }
}

HighGui’nin opencv java’da adlandırılmış pencereleri veya imshow eşdeğerleri yoktur. Resmi görüntülemek için salıncak veya swt kullanın.

OpenCV’yi kaynaktan oluşturun ve yükleyin #

Bu, OpenCV 3’ü kaynaktan Debian tabanlı bir Linux sistemine kurmak için adım adım bir kılavuzdur. Adımlar diğer dağıtımlar için aynı kalmalıdır, yapı için paketleri kurarken ilgili paket yöneticisi komutlarını değiştirmeniz yeterlidir.

Not: Bir şeyler inşa etmekle zaman kaybetmek istemiyorsanız veya terminali sevmiyorsanız, büyük olasılıkla OpenCV’yi Synaptic paket yöneticisi GUI’sinden yükleyebilirsiniz. Ancak, bu kütüphaneler genellikle güncel değildir.

Yapıya Hazırlanın #

Gerekli paketleri kurmak için terminalinizde aşağıdaki komutları verin:

sudo apt-get update
sudo apt-get install build-essential
sudo apt-get install cmake git libgtk2.0-dev pkg-config \
                     libavcodec-dev libavformat-dev libswscale-dev

Aşağıdaki paketler isteğe bağlıdır:

sudo apt-get install python-dev python-numpy libtbb2 libtbb-dev libjpeg-dev \
                     libpng-devlibtiff-dev libjasper-dev libdc1394-22-dev

OpenCV kaynak kodunu almak ve yapıyı hazırlamak için aşağıdaki komutu verin:

mkdir ~/src
cd ~/src
git clone https://github.com/opencv/opencv.git
cd opencv
mkdir build && cd build

Oluştur ve Yükle #

Örnekleri yapıya dahil ediyoruz, ancak bunları dışarıda bırakmaktan çekinmeyin. Ayrıca diğer bayrakları ayarlamaktan ve yapınızı uygun gördüğünüz şekilde özelleştirmekten çekinmeyin.

cmake -D CMAKE_BUILD_TYPE=RELEASE \
      -D CMAKE_INSTALL_PREFIX=/usr/local \
      -D INSTALL_PYTHON_EXAMPLES=ON \
      -D INSTALL_C_EXAMPLES=ON ..

CMake herhangi bir hata veya eksik kitaplık bildirmediyse, derlemeye devam edin.

make -j$(nproc)

Herhangi bir hata oluşmadıysa, sisteme OpenCV kurulumuna devam edebiliriz:

sudo make install

Şimdi OpenCV sisteminizde mevcut olmalıdır. OpenCV’nin nereye kurulduğunu ve hangi kitaplıkların kurulduğunu öğrenmek için aşağıdaki satırları kullanabilirsiniz:

pkg-config --cflags opencv  # get the include path (-I)
pkg-config --libs opencv    # get the libraries path (-L) and the libraries (-l)

Test Kurulumu #

Önce C++ örneklerini oluşturuyoruz:

cd ~/src/opencv/samples
cmake .
make

Herhangi bir hata üretilmediyse, herhangi bir numuneyi çalıştırın, örn.

./cpp/cpp-example-edge

Örnek çalışırsa, C++ kitaplıkları düzgün şekilde yüklenmiştir.

Ardından, Python bağlamalarını test edin:

python
>> import cv2
>> print cv2.__version__

Bu komutlar OpenCV’yi içe aktarır ve şikayet etmeden doğru sürümü yazdırırsa, Python bağlamaları düzgün şekilde yüklenir.

Tebrikler, az önce OpenCV oluşturdunuz ve kurdunuz. Mutlu programlama!

Mac için buraya bakın http://stackoverflow.com/questions/19671827/opencv-installation-on-mac-os-x

Windows’ta OpenCV 3.1’e Başlarken #

Windows’a OpenCV 3.1.0 kuruyoruz ve başlıyoruz. OpenCV’yi pencerelere kurmanın iki yolu vardır. Biri yükleyiciyi indirmek ve çalıştırmaktır. Diğer kaynaktan inşa etmektir.

Bu, OpenCV’yi kurmanın ve başlamanın en kolay yoludur. OpenCV, Windows’a buradan yüklemek için önceden oluşturulmuş ikili dosyalar sağlar. İndirmeyi bitirdikten sonra çıkarın ve seçilen yola yükleyin.

Uzman İpucu: OpenCV yolunuzun boşluk içermediğinden emin olun. Bu nedenle, C:\ veya D:\ kök dizinine kurmanız daha iyi olur.

Yukarıdaki yöntemle ilgili sorun, opencv_contrib modüllerini kullanamamanızdır. Ayrıca, tüm 3. parti araçlar ve kütüphanelerle birlikte gelmez. Yani, bunların hepsini kullanmak istiyorsanız, sadece takip edin.

OpenCV’yi kaynaktan kurmak için minimum minimumu açıklayacağım. Daha gelişmiş olanı için buraya bakın.

  • CMake‘i kurun.
  • OpenCV kaynağını https://github.com/Itseez/opencv.git adresinden boşluk içermeyen bir dizinde klonlayın. Buna “OpenCVdir” olarak değinelim.
  • Şimdi, CMake GUI’yi açın ve kaynak dizininizi (OpenCVdir) Kaynaklar menüsüne ekleyin ve dizini derleme menüsüne oluşturun. İpucu: Derleme dizini yoksa opencv klasörünüzde bir tane oluşturun.
  • Yapılandır’a tıklayın ve Visual Studio derleyici sürümünüzü seçin. Visual Studio 2013 Professional 32-bit’im vardı, bu yüzden Visual Studio 12 derleyicisini seçtim.

İpucu: Visual Studio 2013 Professional’ı buradan indirebilirsiniz. 30 ile geliyor gün deneme + oturum açtıktan sonra 90 gün uzatılmış iz.

  • Bitir’e basın, CMake tüm paketleri otomatik olarak yükleyecektir. ekleyebilir veya paketleri kaldırın. Yeniden Yapılandır’a basın.
  • Ekstra opencv_contrib modülleri ile derlemek istiyorsanız, bunları buradan indirmeniz gerekir. Ardından bunları ayıklayın ve opencv_contrib/modules dizinini klasörünüze ekleyin. CAşağıda gösterildiği gibi yapın.
  • Şimdi tekrar Yapılandır’a ve ardından Oluştur’a basın.
  • CMake’i kapatın. your_opencv\build klasörünüze gidin ve ‘OpenCV.sln’ dosyası adlı dosyayı açın. - Visual Studio’yu açacaktır. Şimdi, hem Hata Ayıklama modunda hem de Sürüm modunda çalıştırın.
  • Şimdi, Visual Studio’nuzun sağ üst köşesindeki çözüm gezgininde YÜKLE’yi seçin. projelendirin ve inşa edin.

Yaşasın!! OpenCV’nizin keyfini çıkarın.

OpenCV’yi Ortam Değişkenlerinin PATH değişkenine dahil etme dizini eklemek:

  • Sistem Özellikleri’ne gidin ve Gelişmiş Sistem Ayarları’na tıklayın.

  • Şimdi, Ortam Değişkenleri » Yol » Düzenle’ye tıklayın.

  • Burada, OpenCVdir/build/install/x86/vc**/bin içinde bulunan bin klasörünü bu değişkene ekleyin. Mevcut Path değerlerini değiştirmemeye dikkat edin.

  • Bundan sonra, Ortam değişkenlerinin değişmesi için sisteminizi yeniden başlatmanız gerekiyor ve şimdi gitmeye hazırsınız.

Ne ve Neden OPENCV? #

OpenCV (Open Source Computer Vision Library), açık kaynaklı bir bilgisayarlı görü ve makine öğrenimi yazılım kitaplığıdır. Makine öğrenimi, bilgisayarla görme, algoritma, matematiksel işlemler, video yakalama, görüntü işleme vb. gibi çeşitli amaçlar için inşa edilmiştir. Yıllar içinde farklı platformlarda (windows, Linux) desteğiyle araştırmacılar ve geliştiriciler arasında oldukça popüler hale gelmiştir. , android, ios). Ayrıca çeşitli ünlü programlama dillerinde sarıcıya sahiptir. Lisans sözleşmesi kapsamında, işletmelerin kodu kullanması ve değiştirmesi için erişimi vardır.

Kitaplık, performans ve hız açısından mükemmel doğruluğa sahip 2500’den fazla optimize edilmiş algoritma içerir. Bu algoritmalar yüzleri algılamak ve tanımak, nesneleri tanımlamak, videolarda insan eylemlerini sınıflandırmak, kamera hareketlerini izlemek, hareketli nesneleri izlemek, nesnelerin 3B modellerini çıkarmak, stereo kameralardan 3B nokta bulutları üretmek, yüksek çözünürlük elde etmek için görüntüleri bir araya getirmek için kullanılabilir. tüm bir sahnenin görüntüsü, bir görüntü veritabanından benzer görüntüler bulun, flaş kullanılarak çekilen görüntülerden kırmızı gözleri kaldırın, göz hareketlerini takip edin, manzarayı tanıyın ve artırılmış gerçeklikle kaplamak için işaretler oluşturun, vb. OpenCV’de kullanıcılar olarak dahil olan harika insanlar ve topluluk vardır. , geliştiriciler ve araştırmacılar, sayı 47 binden fazla ve tahmini indirme sayısı 7 milyonu aşıyor. Kütüphane, profesyonel şirketlerde, araştırma gruplarında ve diğer gruplarda kapsamlıdır.

Google, Yahoo, Microsoft, Intel, IBM, Sony, Honda, Toyota gibi kütüphaneyi kullanan birçok köklü şirket, OpenCV’yi yoğun olarak kullanan Applied Minds, VideoSurf ve Zeitera gibi birçok startup var. OpenCV’nin konuşlandırılan kullanımları, sokak görünümü görüntülerini bir araya getirmek, İsrail’deki güvenlik videolarındaki izinsiz girişleri tespit etmek, Çin’deki maden ekipmanlarını izlemek, robotların Willow Garage’da gezinmesine ve nesneleri almasına yardımcı olmak, Avrupa’da yüzme havuzunda boğulma kazalarını tespit etmek, interaktif sanat çalıştırmak gibi geniş bir aralığı kapsar. İspanya ve New York, Türkiye’de enkaz için pistleri kontrol ediyor, Japonya’da hızlı yüz algılamaya kadar dünyanın dört bir yanındaki fabrikalardaki ürünlerin etiketlerini inceliyor. C++, C, Python, Java ve MATLAB arayüzlerine sahiptir ve Windows, Linux, Android ve Mac OS’yi destekler. OpenCV, çoğunlukla gerçek zamanlı vizyon uygulamalarına yönelir ve mevcut olduğunda MMX ve SSE talimatlarından yararlanır. Tam özellikli bir CUDA ve OpenCL arayüzleri şu anda aktif olarak geliştirilmektedir. 500’den fazla algoritma ve bu algoritmaları oluşturan veya destekleyen işlev sayısı yaklaşık 10 kat daha fazladır. OpenCV yerel olarak C++ ile yazılmıştır ve STL kapsayıcılarıyla sorunsuz çalışan şablonlu bir arayüze sahiptir.

Web kamerasından görüntü alın #

Java, C/C++ ve Python ile OpenCV’nin VideoCapture sınıfını kullanarak bir web kamerasından alınan canlı video beslemesini görüntüleyin.

Java #

import org.opencv.core.Core;
import org.opencv.core.Mat;
import org.opencv.videoio.VideoCapture;
 
public class Camera {
    public static void main(String[] args) {
        // Load Native Library
        System.loadLibrary(Core.NATIVE_LIBRARY_NAME);
        // image container object
        Mat imageArray = new Mat();
        // Video device acces
        VideoCapture videoDevice = new VideoCapture();
        // 0:Start default video device 1,2 etc video device id
        videoDevice.open(0);
        // is contected
        if (videoDevice.isOpened()) {
        // Get frame from camera
            videoDevice.read(imageArray);
            // image array
            System.out.println(imageArray.toString());
            // Release video device
            videoDevice.release();
        } else {
            System.out.println("Error.");
        }
    }
}

C++ #

#include "opencv2/opencv.hpp"
#include "iostream"

int main(int, char**) {
    // open the first webcam plugged in the computer
    cv::VideoCapture camera(0);
    if (!camera.isOpened()) {
        std::cerr << "ERROR: Could not open camera" << std::endl;
        return 1;
    }

    // create a window to display the images from the webcam
    cv::namedWindow("Webcam", CV_WINDOW_AUTOSIZE);

    // this will contain the image from the webcam
    cv::Mat frame;
        
    // capture the next frame from the webcam
    camera >> frame;
    
    // display the frame until you press a key
    while (1) {
        // show the image on the window
        cv::imshow("Webcam", frame);
        // wait (10ms) for a key to be pressed
        if (cv::waitKey(10) >= 0)
            break;
    }
    return 0;
}

Python #

import numpy as np
import cv2

# Video source - can be camera index number given by 'ls /dev/video*
# or can be a video file, e.g. '~/Video.avi'
cap = cv2.VideoCapture(0)

while(True):
    # Capture frame-by-frame
    ret, frame = cap.read()

    # Our operations on the frame come here
    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)

    # Display the resulting frame
    cv2.imshow('frame',gray)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

# When everything done, release the capture
cap.release()
cv2.destroyAllWindows()
