Sözdizimi #

  • CascadeClassifier kademeli = new CascadeClassifier(“cascade.xml”); // cascade.xml’den bir kademeli sınıflandırıcı oluşturur
  • Mat görüntü = Imgcodecs.imread(“image.png”); // image.png’yi bir Mat (Matrix) nesnesine dönüştürür
  • MatOfRect algılamaları = yeni MatOfRect(); // Algılama sınıflarımız için çıktı olarak kullanılan boş bir MatOfRect (Dikdörtgenlerin Matrisi) dosyası oluşturur
  • algılamalar.toArray(); // Üzerinde yinelenebilen bir Rect nesneleri dizisini döndürür
  • Imgproc.rectangle(image, new Point(rect.x, rect.y), new Point(rect.x + rect.width, rect.y + rect.height), new Scalar(0, 255, 0)); // İlk Point’in x ve y konumlarından ikinci Point’in x ve y konumuna Mat nesnesi “image” üzerine yeşil çerçeveli bir dikdörtgen çizer. “rect”, genellikleDetects.toArray() tarafından sağlanan bir Rect nesnesidir. OpenCV’nin Point sınıfını kullanır.
  • Imgcodecs.imwrite(“çıktı.png”, resim); // “image” değiştirilmiş Mat nesnesini “output.png” dosyasına yazar
  • CascadeClassifier.detectMultiScale(görüntü, algılamalar); // Mat nesnesi “image” içindeki herhangi bir nesneyi algılar ve algılamaları MatOfRect nesnesi “algılamalarında” verir
  • CascadeClassifier.detectMultiScale(görüntü, algılamalar, scaleFactor, minNeighbors, flags, minSize, maxSize); // Ek parametrelerle bir algılama gerçekleştirir. Aşağıdaki ayrıntılara bakın.
  • Imgproc.ellipse(görüntü, merkez, eksenler, 0, 0, 360, yeni Skaler(255, 0, 255), kalınlık, çizgiTürü, 0); // Görüntüye ‘merkez’ noktasında bir elips çizer. OpenCV’nin Point sınıfını kullanır.

Parametreler #

Parametre Ayrıntılar
ölçekFaktörü Her görüntü ölçeğinde görüntü boyutunun ne kadar küçültüldüğü. Varsayılan = “1.1”
minKomşular Bir aday dikdörtgenin algılanan bir nesne olarak seçilmeden önce kaç komşusu olması gerekir. Varsayılan = 4
bayraklar Eski bayraklar. Çoğu durumda, bu “0” olarak ayarlanmalıdır. Varsayılan = 0
minBoyut Bir aday dikdörtgenin olabileceği minimum boyut. Bu, OpenCV’nin Size sınıfını kullanır. Algılama süresini ve CPU kullanımını azaltmak ve yanlış pozitifleri azaltmak için kullanılabilir.
maxBoyut Bir aday dikdörtgenin olabileceği maksimum boyut. Bu, OpenCV’nin Size sınıfını kullanır. Algılama süresini ve CPU kullanımını azaltmak ve yanlış pozitifleri azaltmak için kullanılabilir.
eksenler OpenCV’nin Size sınıfını kullanır. Elipsin genişliğini ve yüksekliğini tanımlar.
kalınlık Çizginin piksel cinsinden kalınlığı.
çizgi Türü Çeşitli parametrelere sahiptir. “0” düz çizgi, “8” 8 bağlantılı hat, “4” 4 bağlantılı hat ve “CV_AA” antialiased hat içindir. Varsayılan = 8

Statik bir görüntü alma, üzerindeki öğeleri algılama ve sonuçların çıktısını alma. #

Lütfen bu örneğin OpenCV 3.1 kullandığını unutmayın.

import org.opencv.core.Mat;
import org.opencv.core.MatOfRect;
import org.opencv.core.Point;
import org.opencv.core.Rect;
import org.opencv.core.Scalar;
import org.opencv.imgcodecs.Imgcodecs;
import org.opencv.imgproc.Imgproc;
import org.opencv.objdetect.CascadeClassifier;

public class Classifier {
    private CascadeClassifier diceCascade = new
        CascadeClassifier("res/newMethod/diceCascade.xml");
    private Mat image;
    private String loc = "path/to/image.png";
    private String output = "path/to/output.png";

    public void detImg() {
    
        Mat image = Imgcodecs.imread(loc); // Reads the image
    
        MatOfRect diceDetections = new MatOfRect(); // Output container
        diceCascade.detectMultiScale(image, diceDetections); // Performs the detection
    
        // Draw a bounding box around each detection.
        for (Rect rect : diceDetections.toArray()) {
            Imgproc.rectangle(image, new Point(rect.x, rect.y),
                new Point(rect.x + rect.width, rect.y + rect.height),
                new Scalar(0, 255, 0));
        }
    
        // Save the visualized detection.
        Imgcodecs.imwrite(output, image);
    
    }
}

‘diceDetections.toArray()’ tarafından döndürülen ‘Rect[]’ tekrarlanabilir. Dizi içindeki her bir “Düz” dört ana özelliğe sahip olacaktır: “x”, “y”, “genişlik” ve “yükseklik”. “x” ve “y”, dikdörtgenin sol üst konumunu tanımlar ve “width” ve “height”, dikdörtgenin genişlik ve yüksekliğinin bir “int” değerini döndürür. Bu, görüntülere dikdörtgen çizerken kullanılır. “Imgproc.rectangle” işlevinin minimum gerekli parametreleri aşağıdaki gibidir:

Imgproc.rectangle(Mat image, Point start, Point end, Scalar color);

Her iki “Nokta”, sol üst köşenin ve sağ alt köşenin konumları için kullanılır. Bu konumlar, birbirleri için değil, ilk parametre olarak sağlanan görüntü için her ikisi de mutlaktır. Bu nedenle, “bitiş” Noktasını doğru bir şekilde tanımlamak için “genişlik” veya “yükseklik"e ek olarak dikdörtgenin hem “x” hem de “y” konumunu eklemelisiniz.

Bu parametrelerde kullanılan ‘Point’ sınıfının, Java’nın standart kitaplığının ‘Point’ sınıfı değil olmadığına dikkat edin. Bunun yerine OpenCV’nin ‘Point’ sınıfını içe aktarmalısınız!

Bir video cihazından görüntüleri algılama #

Bu örnek, bir web kamerasından bir görüntü almak ve bir görüntüye kaydetmek için kullandığımız ‘VideoCapture’ sınıfını tanıtır.

import org.opencv.core.Mat;
import org.opencv.core.MatOfRect;
import org.opencv.core.Point;
import org.opencv.core.Rect;
import org.opencv.core.Scalar;
import org.opencv.imgcodecs.Imgcodecs;
import org.opencv.imgproc.Imgproc;
import org.opencv.objdetect.CascadeClassifier;
import org.opencv.videoio.VideoCapture;

public class Classifier {
    private CascadeClassifier diceCascade = new
        CascadeClassifier("res/newMethod/diceCascade.xml");
    private Mat image;
    private String loc = "path/to/image.png";
    private String output = "path/to/output.png";
    private VideoCapture vc = new VideoCapture();

    public void detImg() {
        vc.open(0); // Opens the video stream

        Mat image = new Mat(); // Creates an empty matrix
        vc.read(image); // Reads the image from the video stream and
            writes it to the image matrix.
    
        MatOfRect diceDetections = new MatOfRect(); // Output container
        diceCascade.detectMultiScale(image, diceDetections); // Performs the detection
    
        // Draw a bounding box around each detection.
        for (Rect rect : diceDetections.toArray()) {
            Imgproc.rectangle(image, new Point(rect.x, rect.y),
                new Point(rect.x + rect.width, rect.y + rect.height),
                new Scalar(0, 255, 0));
        }
    
        // Save the visualized detection.
        Imgcodecs.imwrite(output, image);

        vc.release(); // Closes the stream.
    
    }
}

Mat nesnesini BufferedImage nesnesine dönüştürme #

Daniel Baggio’nun bu örneği doğrudan bu StackExchange yanıtından alınmıştır, ancak görünürlük için yeniden yayınlanmıştır.

Bu sınıf bir Mat nesnesi alır ve “javax.swing” kitaplıkları tarafından kullanılan BufferedImage nesnesini döndürür. Bu, görüntüyü çizmek için bir “Grafik” nesnesi tarafından kullanılabilir.

private BufferedImage toBufferedImage(Mat m) {
    if (!m.empty()) {
        int type = BufferedImage.TYPE_BYTE_GRAY;
        if (m.channels() > 1) {
            type = BufferedImage.TYPE_3BYTE_BGR;
        }
        int bufferSize = m.channels() * m.cols() * m.rows();
        byte[] b = new byte[bufferSize];
        m.get(0, 0, b); // get all the pixels
        BufferedImage image = new BufferedImage(m.cols(), m.rows(), type);
        final byte[] targetPixels = ((DataBufferByte) image.getRaster().getDataBuffer()).getData();
        System.arraycopy(b, 0, targetPixels, 0, b.length);
        return image;
    }
    
    return null;
}

Algılamalar içindeki Algılamalar #

Bu örnek, nesnemiz olarak Zar ve zardaki siyah noktalar (pipler) kullanır. Örnek oldukça uzun olduğu için, önce bazı temel kavramları açıklamak, örneği anlamak için çok önemlidir.

İlk örneği anlamak, “Statik bir görüntü alma, üzerindeki öğeleri algılama ve sonuçların çıktısını alma”. özellikle OpenCV’nin dikdörtgenleri nasıl çizdiğini anlamak için bu örneği anlamak çok önemlidir.

Aşağıdaki resme bir göz atın:

Daha fazla algılama uygulamak için temel olarak algılanan bir alanı kullandığımız alt görüntüleme yöntemini kullanacağız. Bu, ancak bir nesne her zaman algılayabildiğimiz başka bir nesnenin içindeyse mümkündür, örneğin zarımızdaki tırtıllarımız gibi. Bu yöntemin çeşitli faydaları vardır:

  • Resmin tamamını taramak yerine, yalnızca nesnenin içinde olacağını bildiğimiz alanı taramamız gerekir.
  • Algılama alanı dışında herhangi bir yanlış pozitif olasılığını ortadan kaldırır.

Bunu, bize büyük nesnelerimizi içeren bir “MatOfRect” nesnesi vermek için tüm görüntü üzerinde bir kademeli sınıflandırıcı taraması uygulayarak yapıyoruz (bu durumda zar). Daha sonra ‘MatOfRect’ nesnesinden ’toArray()’ işlevi tarafından verilen ‘Rect[]’ dizisi üzerinde yineleniriz. Bu “Rect” nesnesi, orijinal görüntüden “Rect” nesnesinin özelliklerine (“x, y, genişlik, yükseklik”) “kırpılan” geçici bir “Mat” nesnesi oluşturmak için kullanılır; geçici “Mat” nesnesi. Başka bir deyişle, sınıflandırıcıya bunun yerine görüntünün yalnızca zar kısımlarında algılama yapmasını söylüyoruz ve görüntünün tamamında bir algılama gerçekleştirerek elde ettiğimiz ‘Rect’ nesnelerini kullanarak her zarın konumunu belirliyoruz.

Bununla birlikte, “Rect” nesneleri (pipler), görüntünün kendisine değil, zarlarına göre özelliklerine sahiptir. Bu sorunu çözmek için, asıl görüntüye tırtılların konumlarını gösteren dikdörtgenler çizmek istediğimizde, başlangıç ​​Noktasına hem dice.x hem de dice.y ekleriz.

import org.opencv.core.Mat;
import org.opencv.core.MatOfRect;
import org.opencv.core.Point;
import org.opencv.core.Rect;
import org.opencv.core.Scalar;
import org.opencv.core.Size;
import org.opencv.imgcodecs.Imgcodecs;
import org.opencv.imgproc.Imgproc;
import org.opencv.objdetect.CascadeClassifier;
import org.opencv.videoio.VideoCapture;

public class Classifier {

    private CascadeClassifier diceCascade =
        new CascadeClassifier("res/newMethod/diceCascade.xml");
    private CascadeClassifier pipCascade =
        new CascadeClassifier("res/newMethod/pipCascade6.xml");
    private VideoCapture vc = new VideoCapture();
    private Mat image;

    public void openVC(int index) {
        vc.open(index);
    }

    public void closeVC() {
        vc.close();
    }

    public Mat getNextImage() {
        image = new Mat();
        vc.read(image); // Sets the matrix to the current livestream frame.
        
        MatOfRect diceDetections = new MatOfRect(); // Output container
    
        // See syntax for explainations on addition parameters
        diceCascade.detectMultiScale(image, diceDetections, 1.1, 4, 0, new Size(20, 20),
            new Size(38, 38));
        
        // Iterates for every Dice ROI
        for (int i = 0; i < diceDetections.toArray().length; i++) {
            Rect diceRect = diceDetections.toArray()[i];
            
            // Draws rectangles around our detected ROI
            Point startingPoint = new Point(diceRect.x, diceRect.y);
            Point endingPoint = new Point(diceRect.x + diceRect.width,
                diceRect.y + diceRect.height);
            Imgproc.rectangle(image, startingPoint, endingPoint, new Scalar(255, 255, 0));
            
            MatOfRect pipDetections = new MatOfRect();
            
            pipCascade.detectMultiScale(image.submat(diceRect), pipDetections, 1.01, 4, 0,
                new Size(2, 2), new Size(10, 10));
            
            // Gets the number of detected pips and draws a cricle around the ROI
            for (int y = 0; y < pipDetections.toArray().length; y++) {
                // Provides the relative position of the pips to the dice ROI
                Rect pipRect = pipDetections.toArray()[y];
                
                // See syntax explaination
                // Draws a circle around our pips
                Point center = new Point(diceRect.x + pipRect.x + pipRect.width / 2,
                    diceRect.y + pipRect.y + pipRect.height / 2);
                Imgproc.ellipse(image, center, new Size(pipRect.width / 2, pipRect.height / 2),
                     0, 0, 360, new Scalar(255, 0, 255), 1, 0, 0);
            }
        }
        
        return image;
    }
}

‘getNextImage()’ işlevi, yayınlanan diğer örneklerle birlikte sürekli olarak çağrılabilen ve algılamaları gösteren bir canlı akış sağlamak için bir ‘BufferImage’a dönüştürülebilen bir ‘Mat’ nesnesi döndürür.
