Sözdizimi #

  1. Gauss Bulanıklığı Sözdizimi C++: void GaussianBlur(InputArray kaynağı, OutputArray dst, Size ksize, double sigmaX, double sigmaY=0, int borderType=BORDER_DEFAULT )

Parametreler #

Gauss Bulanıklığının Parametreleri Ayrıntılar
kaynak Girdi görüntüsü, görüntünün bağımsız olarak işlenen herhangi bir sayıda kanalı olabilir, ancak derinlik “CV_8U”, “CV_16U”, “CV_16S”, “CV_32F” veya “CV_64F” olmalıdır.
dst src ile aynı boyut ve tipte çıktı görüntüsü
beden Gauss çekirdek boyutu. “ksize.width” ve “ksize.height” farklı olabilir ancak her ikisi de pozitif ve tek olmalıdır. Veya sıfır olabilirler ve sonra sigma*‘dan hesaplanırlar.
sigmaX X yönünde Gauss çekirdek standart sapması.
sigmaY Y yönünde Gauss çekirdek standart sapması. ‘sigmaY’ sıfır ise, ‘sigmaX’e eşit olacak şekilde ayarlanır, her iki sigma da sıfırsa, ksize.width ve ksize.height’tan hesaplanır. Tüm bu anlambilimin gelecekteki olası değişikliklerinden bağımsız olarak sonucu tam olarak kontrol etmek için, “ksize”, “sigmaX” ve “sigmaY"nin tümünün belirtilmesi önerilir.
sınırTürü Piksel ekstrapolasyon yöntemi.

Gauss bulanıklığına özgü sözdizimi ve parametreleri bu yere koymanın mantıklı olduğunu düşünmüyorum çünkü konu çok geniş ve birçok başka örneği içermesi gerekiyor.

C++‘da Gauss Bulanıklığı ile Görüntüleri Düzeltme #

Bulanıklaştırma olarak da bilinen yumuşatma, Görüntü İşleme’de en sık kullanılan işlemlerden biridir.

Yumuşatma işleminin en yaygın kullanımı, daha sonraki işlemler için görüntüdeki gürültüyü azaltmaktır.

Düzgünleştirme işlemini gerçekleştirmek için birçok algoritma vardır.

Bir görüntüyü bulanıklaştırmak için en sık kullanılan filtrelerden biri olan Gauss Filtresi’ni OpenCV kitaplık işlevi ‘GaussianBlur()’ kullanarak inceleyeceğiz. Bu filtre, görüntülerden yüksek frekanslı gürültüyü çıkarmak için özel olarak tasarlanmıştır.

#include <opencv2/opencv.hpp>
#include <iostream>

using namespace std;
using namespace cv;

int main(int argc, char** argv){

    Mat image , blurredImage;

    // Load the image file
    image = imread(argv[1], CV_LOAD_IMAGE_COLOR);

    // Report error if image could not be loaded
    if(!image.data){
        cout<<"Error loading image" << "\n";
        return -1;
    }

    // Apply the Gaussian Blur filter. 
    // The Size object determines the size of the filter (the "range" of the blur)
    GaussianBlur( image, blurredImage, Size( 9, 9 ), 1.0);

    // Show the blurred image in a named window
    imshow("Blurred Image" , blurredImage);

    // Wait indefinitely untill the user presses a key
    waitKey(0);

    return 0;
}

Ayrıntılı matematiksel tanım ve diğer filtre türleri için orijinal belgelere göz atabilirsiniz.

Eşik #

Python’da:

import cv2
image_path= 'd:/contour.png'
img = cv2.imread(image_path)

#display image before thresholding
cv2.imshow('I am an image display window',img)
cv2.waitKey(0)

#convert image to gray scale - needed for thresholding
img_gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)


#apply threshold to gray image to obtain binary image

threshold=150 #value above which pixel values will be set to max_value
max_value=255  #value to which pixels above threshold will be set
threshold_stype=cv2.THRESH_BINARY #default threshold method

ret, img_binary = cv2.threshold(img_gray, threshold, max_value, threshold_stype)

#display image after thresholding
cv2.imshow('image after applying threshold',img_binary)
cv2.waitKey(0)

#save the binary image
cv2.imwrite('d:/binary.png',img_binary)
cv2.destroyAllWindows()
