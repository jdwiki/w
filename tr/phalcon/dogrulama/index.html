  • Doğrulama sınıfına ilişkin API referansı burada bulunabilir: https://docs.phalconphp.com/en/latest/api/Phalcon_Validation.html
  • \Phalcon\Validation içinde sağlanan bir varlık varsa, \Phalcon\Validation\Validator\Uniqueness içinde model anahtarını iletmenize gerek yoktur.

Yerleşik Doğrulayıcılar #

PresenceOf - Bir değerin boş veya boş dize olmadığını doğrular

$validator->add('name', new \Phalcon\Validation\Validator\PresenceOf([
   'message' => 'The name is required'
]));

E-posta - Bir değerin doğru e-posta biçimine sahip olup olmadığını kontrol eder

$validator->add('email', new \Phalcon\Validation\Validator\Email([
   'message' => 'The e-mail is not valid'
]));

Aynı - Bir değerin diğeriyle aynı olup olmadığını kontrol eder

$validator->add('terms', new \Phalcon\Validation\Validator\Identical([
   'accepted' => 'yes',
   'message' => 'Terms and conditions must be accepted'
]));

Url - Bir değerin url biçimine sahip olup olmadığını kontrol eder

$validator->add('url', new \Phalcon\Validation\Validator\Url([
   'message' => ':field must be a url'
]));

Onay - İki değerin aynı değere sahip olup olmadığını kontrol eder

$validator->add('password', new \Phalcon\Validation\Validator\Confirmation([
   'message' => 'Password doesn\'t match confirmation',
   'with' => 'confirmPassword'
]));

StringLength - Bir dizginin belirtilen maksimum ve minimum kısıtlamalara sahip olduğunu doğrular Bir dizginin uzunluğu L için min<=L<=maks, yani L en az min ve en çok maks.

$validation->add('name_last', new \Phalcon\Validation\Validator\StringLength([
      'max' => 50,
      'min' => 2,
      'messageMaximum' => 'We don\'t like really long names',
      'messageMinimum' => 'We want more than just their initials'
]));

Normal ifade - Bir alanın değerinin normal bir ifadeyle eşleşip eşleşmediğini doğrulamaya izin verir

$validator->add('created_at', new \Phalcon\Validation\Validator\Regex([
   'pattern' => '/^[0-9]{4}[-\/](0[1-9]|1[12])[-\/](0[1-9]|[12][0-9]|3[01])$/',
   'message' => 'The creation date is invalid'
]));

Kredi Kartı - Bir değerin geçerli bir kredi kartı numarasına sahip olup olmadığını kontrol eder

$validator->add('creditcard', new \Phalcon\Validation\Validator\CreditCard([
   'message' => 'The credit card number is not valid'
]));

Arasında - Bir değerin, iki değer içeren kapsamlı bir aralık arasında olduğunu doğrular. Bir x değeri için, minimum<=x<=maksimum ise test geçilir.

$validator->add('name', new \Phalcon\Validation\Validator\Between([
   'minimum' => 0,
   'maximum' => 100,
   'message' => 'The price must be between 0 and 100'
]));

ExclusionIn - Bir değerin değerler listesine eklenip eklenmediğini kontrol edin

$validator->add('status', new \Phalcon\Validation\Validator\ExclusionIn([
   'message' => 'The status must not be A or B',
   'domain' => ['A', 'B']
]));

InclusionIn - Bir değerin değerler listesine eklenip eklenmediğini kontrol edin

$validator->add('status', new \Phalcon\Validation\Validator\InclusionIn([
   'message' => 'The status must be A or B',
   'domain' => ['A', 'B']
]));

Benzersizlik - Bir değerin benzersiz olup olmadığını kontrol edin

$validator->add('login', new \Phalcon\Validation\Validator\Uniqueness([
    'message' => 'The login must be unique',
    'model' => new Users()
]));

Google reCaptcha özel doğrulama bileşeni #

Sınıf

use Phalcon\Validation\Validator;
use Phalcon\Validation\ValidatorInterface;
use Phalcon\Validation\Message;

class RecaptchaValidator extends Validator implements ValidatorInterface
{
    public function validate(\Phalcon\Validation $validation, $attribute)
    {
        $value = $validation->getValue('g-recaptcha-response');
        $ip = $validation->request->getClientAddress();
        if (!$this->verify($value, $ip)) {
            $validation->appendMessage(new Message($this->getOption('message'), $attribute, 'Recaptcha'));
            return false;
        }
        return true;
    }
    
    protected function verify($value, $ip)
    {
        $params = [
            'secret' => 'YOUR_RECAPTCHA_SECRET_KEY',
            'response' => $value,
            'remoteip' => $ip
        ];
        $response = json_decode(file_get_contents('https://www.google.com/recaptcha/api/siteverify?' . http_build_query($params)));
        return (bool) $response->success;
    }
}

Phalcon formunda örnek kullanım:

$reCaptchaField->addValidator(new \RecaptchaValidator([
    'message' => 'Your reCaptcha error message'
]));
