Ubuntu’da Theano’yu yükleme ve GPU’yu yapılandırma 14.04 #

Theano’yu kurmak ve GPU’yu yapılandırmak için aşağıdaki talimatları kullanabilirsiniz (yeni yüklenmiş bir Ubuntu 14.04 varsayın):

# Install Theano
sudo apt-get install python-numpy python-scipy python-dev python-pip python-nose g++ libopenblas-dev git
sudo pip install Theano

# Install Nvidia drivers, CUDA and CUDA toolkit, following some instructions from http://docs.nvidia.com/cuda/cuda-installation-guide-linux/index.html
wget http://developer.download.nvidia.com/compute/cuda/7.5/Prod/local_installers/cuda-repo-ubuntu1404-7-5-local_7.5-18_amd64.deb # Got the link at https://developer.nvidia.com/cuda-downloads
sudo dpkg -i cuda-repo-ubuntu1404-7-5-local_7.5-18_amd64.deb
sudo apt-get update
sudo apt-get install cuda

sudo reboot

Bu noktada, “nvidia-smi"yi çalıştırmak işe yarayacaktır, ancak “nvcc"yi çalıştırmak işe yaramaz.

# Execute in console, or (add in ~/.bash_profile then run "source ~/.bash_profile"):
export PATH=/usr/local/cuda-7.5/bin:$PATH
export LD_LIBRARY_PATH=/usr/local/cuda-7.5/lib64:$LD_LIBRARY_PATH

Bu noktada hem ’nvidia-smi’ hem de ’nvcc’ çalışması gerekir.

Theano’nun GPU’yu kullanıp kullanamayacağını test etmek için:

Aşağıdakileri “gpu_test.py” içine kopyalayıp yapıştırın:

# Start gpu_test.py
# From http://deeplearning.net/software/theano/tutorial/using_gpu.html#using-gpu
from theano import function, config, shared, sandbox
import theano.tensor as T
import numpy
import time

vlen = 10 * 30 * 768  # 10 x #cores x # threads per core
iters = 1000

rng = numpy.random.RandomState(22)
x = shared(numpy.asarray(rng.rand(vlen), config.floatX))
f = function([], T.exp(x))
print(f.maker.fgraph.toposort())
t0 = time.time()
for i in xrange(iters):
    r = f()
t1 = time.time()
print("Looping %d times took %f seconds" % (iters, t1 - t0))
print("Result is %s" % (r,))
if numpy.any([isinstance(x.op, T.Elemwise) for x in f.maker.fgraph.toposort()]):
    print('Used the cpu')
else:
    print('Used the gpu')
# End gpu_test.py

ve çalıştırın:

THEANO_FLAGS='mode=FAST_RUN,device=gpu,floatX=float32' python gpu_test.py

hangi dönmelidir:

f@f-Aurora-R4:~$ THEANO_FLAGS='mode=FAST_RUN,device=gpu,floatX=float32' python gpu_test.py
Using gpu device 0: GeForce GTX 690
[GpuElemwise{exp,no_inplace}(<CudaNdarrayType(float32, vector)>), HostFromGpu(GpuElemwise{exp,no_inplace}.0)]
Looping 1000 times took 0.658292 seconds
Result is [ 1.23178029  1.61879349  1.52278066 ...,  2.20771813  2.29967761
  1.62323296]
Used the gpu

cuDNN ekleme #

cuDNN eklemek için (talimatlar http://deeplearning.net/software/theano/library/sandbox/cuda/dnn.html adresinden):

  1. cuDNN’yi https://developer.nvidia.com/rdp/cudnn-download adresinden indirin (ücretsiz olan kayıt gerekir)
  2. tar -xvf cudnn-7.0-linux-x64-v3.0-prod.tgz
  3. Aşağıdakilerden birini yapın

Seçenek 1: “.h” dosyalarını “CUDA_ROOT/include” dizinine ve “.so*” dosyalarını “CUDA_ROOT/lib64” dizinine kopyalayın (varsayılan olarak “CUDA_ROOT”, Linux’ta “/usr/local/cuda"dır ).

sudo cp cuda/lib64/* /usr/local/cuda/lib64/
sudo cp cuda/include/cudnn.h /usr/local/cuda/include/

Seçenek 2:

export LD_LIBRARY_PATH=/home/user/path_to_CUDNN_folder/lib64:$LD_LIBRARY_PATH
export CPATH=/home/user/path_to_CUDNN_folder/include:$CPATH
export LIBRARY_PATH=/home/user/path_to_CUDNN_folder/lib64:$LD_LIBRARY_PATH

Varsayılan olarak Theano, cuDNN kullanıp kullanamayacağını algılayacaktır. Eğer öyleyse, onu kullanacaktır. Değilse, Theano optimizasyonları cuDNN operasyonlarını tanıtmayacaktır. Bu nedenle, kullanıcı bunları manuel olarak tanıtmadıysa Theano çalışmaya devam edecektir.

Theano cuDNN kullanamıyorsa bir hata almak için bu Theano bayrağını kullanın: optimizer_clude=cudnn.

Örnek:

THEANO_FLAGS='mode=FAST_RUN,device=gpu,floatX=float32,optimizer_including=cudnn' python gpu_test.py

cuDNN sürümünüzü öğrenmek için:

cat /usr/local/cuda/include/cudnn.h | grep CUDNN_MAJOR -A 2

CNMeM Ekleme #

CNMeM kitaplığı “Derin Öğrenme çerçevelerinin CUDA belleğini yönetmesine yardımcı olan basit bir kitaplıktır.”

# Build CNMeM without the unit tests
git clone https://github.com/NVIDIA/cnmem.git cnmem
cd cnmem
mkdir build
cd build
sudo apt-get install -y cmake
cmake ..
make

# Copy files to proper location
sudo cp ../include/cnmem.h /usr/local/cuda/include
sudo cp *.so /usr/local/cuda/lib64/
cd ../..

Theano ile kullanmak için lib.cnmem bayrağını eklemeniz gerekir. Örnek:

THEANO_FLAGS='mode=FAST_RUN,device=gpu,floatX=float32,lib.cnmem=0.8,optimizer_including=cudnn' python gpu_test.py

Komut dosyasının ilk çıktısı şöyle olmalıdır:

Using gpu device 0: GeForce GTX TITAN X (CNMeM is enabled with initial size: 80.0% of memory, cuDNN 5005)

’lib.cnmem=0.8’, GPU’nun %80’ine kadar kullanabileceği anlamına gelir.

CNMeM’in bazı ilginç hız iyileştirmeleri sağladığı bildirildi ve Theano, Torch ve Caffee tarafından destekleniyor.

Theano - kaynak 1:

Hızlanma, şekiller ve modelin kendisi gibi birçok faktöre bağlıdır. Hızlanma 0’dan 2 kat daha hızlı gider.

Theano - kaynak 2:

Theano bayrağı allow_gc’yi değiştirmezseniz GPU’da %20 hızlanma bekleyebilirsiniz. Bazı durumlarda (küçük modeller) %50 hızlanma gördük.

Ortak sorunlar:

  • Theano içe aktarılıyor: AttributeError: ‘module’ nesnesinin ‘find_graphviz’ özelliği yok

Kurulum veya Kurulum #

Theano kurulumu veya kurulumu hakkında ayrıntılı talimatlar.

Theano’yu birden fazla CPU çekirdeğinde çalıştırma #

Theano’yu OMP_NUM_THREADS=[number_of_cpu_cores] bayrağıyla birden fazla CPU çekirdeğinde çalıştırabilirsiniz.

Örnek:

OMP_NUM_THREADS=4 python gpu_test.py

theano/misc/check_blas.py komut dosyası, hangi BLAS’ın kullanıldığına ilişkin bilgileri verir:

cd [theano_git_directory]
OMP_NUM_THREADS=4 python theano/misc/check_blas.py

İlk theano programınız #

Bu örnekte, iki gerçek sayı verilen toplamı ve farkı hesaplayan fonksiyonları derleyeceğiz.

from __future__ import print_function
import theano
import theano.tensor as T

#define two symbolic scalar
s_x = T.fscalar()
s_y = T.fscalar()

#compute something
s_sum = s_x + s_y
s_diff = s_x - s_y

#compile a function that adds two number
#theano will call system compiler at here
fn_add = theano.function(inputs=[s_x, s_y], outputs=s_sum)
fn_diff = theano.function(inputs=[s_x, s_y], outputs=s_diff)

#call the compiled functions
print(fn_add(2., 2.)) #4.
print(fn_diff(2., 2.)) #0.
