Projeyi kurmak #

Proje bağımlılıklarını yüklemek için Düğüm Paket Yöneticisine ihtiyacınız var. İşletim sisteminiz için düğümü Nodejs.org adresinden indirin. Düğüm Paket Yöneticisi, düğümle birlikte gelir.

Düğüm ve npm sürümlerinizi daha iyi yönetmek için Düğüm Sürüm Yöneticisi‘yi de kullanabilirsiniz. Projenizi farklı düğüm sürümlerinde test etmek için harikadır. Ancak, üretim ortamı için önerilmez.

Sisteminize düğüm kurduktan sonra, Babel ve Webpack kullanarak ilk React projenizi patlatmak için bazı temel paketleri kurun.

Aslında terminalde komutları vurmaya başlamadan önce. Babel ve Webpack ne için kullanıldığına bir göz atın.

Terminalinizde npm init komutunu çalıştırarak projenize başlayabilirsiniz. İlk kurulumu izleyin. Bundan sonra, terminalinizde aşağıdaki komutları çalıştırın-

Bağımlılıklar:

npm install tepki tepki-dom --save

Geliştirici Bağımlılıkları:

npm babel-core babel-loader'ı yükle babel-preset-es2015 babel-preset-react babel-preset-stage-0 web paketi webpack-dev-server tepki-hot-loader --save-dev

İsteğe Bağlı Geliştirme Bağımlılıkları:

npm eslint eslint-eklenti-tepkisini kur babel-eslint --save-dev

Bu örnek paketine başvurabilirsiniz.json

Aşağıdaki içeriklerle proje kökünüzde .babelrc oluşturun:

{
  "presets": ["es2015", "stage-0", "react"]
}

İsteğe bağlı olarak aşağıdaki içeriklerle proje kökünüzde ‘.eslintrc’ oluşturun:

{
  "ecmaFeatures": {
    "jsx": true,
    "modules": true
  },
  "env": {
    "browser": true,
    "node": true
  },
  "parser": "babel-eslint",
  "rules": {
    "quotes": [2, "single"],
    "strict": [2, "never"],
  },
  "plugins": [
    "react"
  ]
}

Oluşturulan dosyaların git deponuza yüklenmesini önlemek için bir “.gitignore” dosyası oluşturun.

node_modules
npm-debug.log
.DS_Store
dist

Aşağıdaki minimum içerikle webpack.config.js dosyasını oluşturun.

var path = require('path');
var webpack = require('webpack');

module.exports = {
  devtool: 'eval',
  entry: [
    'webpack-dev-server/client?http://localhost:3000',
    'webpack/hot/only-dev-server',
    './src/index'
  ],
  output: {
    path: path.join(__dirname, 'dist'),
    filename: 'bundle.js',
    publicPath: '/static/'
  },
  plugins: [
    new webpack.HotModuleReplacementPlugin()
  ],
  module: {
    loaders: [{
      test: /\.js$/,
      loaders: ['react-hot', 'babel'],
      include: path.join(__dirname, 'src')
    }]
  }
};

Ve son olarak, aşağıdaki içeriklerle “npm start"ı çalıştırabilmek için bir “sever.js” dosyası oluşturun:

var webpack = require('webpack');
var WebpackDevServer = require('webpack-dev-server');
var config = require('./webpack.config');

new WebpackDevServer(webpack(config), {
  publicPath: config.output.publicPath,
  hot: true,
  historyApiFallback: true
}).listen(3000, 'localhost', function (err, result) {
  if (err) {
    return console.log(err);
  }

  console.log('Serving your awesome project at http://localhost:3000/');
});

React projenizin bir şeyler yaptığını görmek için src/app.js dosyası oluşturun.

import React, { Component } from 'react';

export default class App extends Component {
  render() {
    return (
      <h1>Hello, world.</h1>
    );
  }
}

“package.json” dosyanızda “start"ın ne anlama geldiğini tanımladıysanız, terminalde “node server.js” veya “npm start” komutunu çalıştırın.

tepki başlatıcı proje #

Bu Proje Hakkında

Bu basit bir kazan projesidir. Bu gönderi, ReactJs + Webpack + Bable ortamını kurmanız için size rehberlik edecektir.

Başlayalım

hızlı sunucuyu başlatmak ve proje boyunca bağımlılıkları yönetmek için düğüm paket yöneticisine ihtiyacımız olacak. düğüm paket yöneticisinde yeniyseniz, buradan kontrol edebilirsiniz. Not : Burada düğüm paket yöneticisinin yüklenmesi gerekir.

Uygun ada sahip bir klasör oluşturun ve terminalden veya GUI ile içine gidin. Ardından terminale gidin ve npm init yazın, bu bir package.json dosyası oluşturacak, Korkunç bir şey yok, size projenizin adı gibi birkaç soru soracak, sürüm, açıklama, giriş noktası, git deposu, yazar, lisans vb. Burada giriş noktası önemlidir çünkü projeyi çalıştırdığınızda düğüm başlangıçta onu arayacaktır. Sonunda, verdiğiniz bilgileri doğrulamanızı isteyecektir. evet yazabilir veya değiştirebilirsiniz. İşte bu kadar, package.json dosyamız hazır.

Hızlı sunucu kurulumu npm install express@4 –save komutunu çalıştırın. Bu proje için ihtiyacımız olan tüm bağımlılıklar bu. Burada kaydetme bayrağı önemlidir, onsuz package.js dosyası güncellenmeyecektir. package.json dosyasının ana görevi, bağımlılıkların listesini saklamaktır. Ekspres sürüm 4’ü ekleyecektir. package.json dosyanız gibi görünecektir. "dependencies": { "express": "^4.13.4", ............. },

İndirme işlemi tamamlandıktan sonra bağımlılıklarımızın node_modules klasörünün ve alt klasörünün olduğunu görebilirsiniz. Şimdi projenin kökünde yeni dosya server.js dosyası oluşturun. Şimdi ekspres sunucu ayarlıyoruz. Tüm kodu geçeceğim ve daha sonra açıklayacağım.

var express = require('express');
// Create our app
var app = express();

app.use(express.static('public'));

app.listen(3000, function () {
  console.log('Express server is using port:3000');
});

var express = require(’express’); bu size tüm ekspres API’ye erişim sağlayacaktır.

var app = express(); fonksiyon olarak ekspres kütüphaneyi çağırır. app.use(); ekspres uygulamanıza işlevsellik eklemesine izin verin. app.use(express.static(‘public’)); web sunucumuzda gösterilecek klasör adını belirleyecektir. app.listen(port, function(){}) burada bağlantı noktamız 3000 olacak ve çağırdığımız fonksiyon web sunucusunun düzgün çalıştığını doğrulayacaktır. Yani ekspres sunucu kuruldu.

Şimdi projemize gidin ve yeni bir public klasörü oluşturun ve index.html dosyası oluşturun. index.html, uygulamanız için varsayılan dosyadır ve Express sunucusu bu dosyayı arayacaktır. index.html, şuna benzeyen basit bir html dosyasıdır:

<!DOCTYPE html>
<html>

<head>
  <meta charset="UTF-8"/>
</head>

<body>
  <h1>hello World</h1>
</body>

</html>

Ve terminal üzerinden proje yoluna gidin ve node server.js yazın. Ardından * console.log(‘Express sunucusu bağlantı noktası kullanıyor:3000’);* ifadesini göreceksiniz.

Tarayıcıya gidin ve gezinme çubuğuna http://localhost:3000 yazın, merhaba Dünya göreceksiniz.

Şimdi ortak klasörün içine gidin ve yeni bir app.jsx dosyası oluşturun. JSX, JavaScript’inize XML sözdizimi ekleyen bir önişlemci adımıdır. React’i JSX olmadan kesinlikle kullanabilirsiniz, ancak JSX, React’i çok daha zarif hale getirir. İşte app.jsx için örnek kod

ReactDOM.render(
  <h1>Hello World!!!</h1>,
  document.getElementById('app')
);

Şimdi index.html adresine gidin ve kodu değiştirin, şöyle görünmelidir

<!DOCTYPE html>
<html>

<head>
  <meta charset="UTF-8"/>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/babel-core/5.8.23                                              /browser.min.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/react/0.14.7/react.js">                           </script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/react/0.14.7/react-dom.js">    </script>
</head>

<body>
  <div id="app"></div>

  <script type="text/babel" src="app.jsx"></script>
</body>

</html>

Bunu yerine yerleştirdikten sonra, işiniz bitti, umarım bunu basit bulursunuz.
