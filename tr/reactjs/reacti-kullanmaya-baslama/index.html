ReactJS nedir? #

ReactJS, uygulamanın yalnızca görünüm katmanından sorumlu, açık kaynaklı, bileşen tabanlı bir ön uç kitaplığıdır. Facebook tarafından bakımı yapılmaktadır.

ReactJS, HTML DOM’deki verileri (görünümleri) doldurmak için sanal DOM tabanlı mekanizma kullanır. Sanal DOM, her seferinde tam DOM’yi yeniden yüklemek yerine yalnızca tek tek DOM öğelerini değiştirdiği gerçeğine sahip olarak hızlı çalışır.

Bir React uygulaması, her biri küçük, yeniden kullanılabilir bir HTML parçasının çıktısını almaktan sorumlu olan birden çok bileşenden oluşur. Bileşenler, karmaşık uygulamaların basit yapı taşlarından oluşturulmasına izin vermek için diğer bileşenlerin içine yerleştirilebilir. Bir bileşen ayrıca dahili durumu da koruyabilir - örneğin, bir TabList bileşeni, o anda açık olan sekmeye karşılık gelen bir değişkeni depolayabilir.

React, JSX adı verilen alana özgü bir dil kullanarak bileşenler yazmamıza olanak tanır. JSX, JavaScript olaylarını karıştırırken bileşenlerimizi HTML kullanarak yazmamıza izin verir. React, bunu dahili olarak sanal bir DOM’ye dönüştürecek ve sonuçta bizim için HTML’mizi çıkaracaktır.

Sanal DOM’yi kullanarak bileşenleri HTML DOM’de yeniden oluşturmak için bileşenlerinizdeki değişiklikleri hızlı ve otomatik olarak belirtmek için “reacts” tepki verin. Sanal DOM, gerçek bir DOM’nin bellek içi temsilidir. React, işlemenin çoğunu doğrudan tarayıcının DOM’si yerine sanal DOM içinde yaparak hızlı hareket edebilir ve yalnızca son oluşturma döngüsü gerçekleştiğinden beri değişen bileşenleri ekler, günceller ve kaldırır.

Kurulum veya Kurulum #

ReactJS, herhangi bir HTML sayfasına dahil edilebilen tek bir “react-.js” dosyasında bulunan bir JavaScript kitaplığıdır. İnsanlar ayrıca ana React dosyasıyla birlikte react-dom-<version>.js React DOM kitaplığını da yükler:

Temel Katılım

<!DOCTYPE html>
<html>
    <head></head>
    <body>
    <script type="text/javascript" src="/path/to/react.js"></script>
    <script type="text/javascript" src="/path/to/react-dom.js"></script>
    <script type="text/javascript">
        // Use react JavaScript code here or in a separate file
    </script>
    </body>
</html>

JavaScript dosyalarını almak için resmi React belgelerinin kurulum sayfasına gidin.
