jQuery ile ReactJS #

Öncelikle, jquery library dosyasını içe aktarmanız gerekir. Dom’u değiştireceğimiz için findDOmNode’u da içe aktarmamız gerekiyor. Ve tabii ki React’i de ithal ediyoruz.

import React from 'react';
import { findDOMNode } from ‘react-dom’;
import $ from ‘jquery’;

Bir simgeye tıklandığında ateşlenecek bir ok işlevi ‘handleToggle’ ayarlıyoruz. Biz sadece bir simgenin üzerine tıklayın üzerinde ’toggle’ adında bir referansa sahip bir div’i gösteriyor ve saklıyoruz.

handleToggle = () => {
    const el = findDOMNode(this.refs.toggle);
    $(el).slideToggle();
};

Şimdi ’toggle’ referans adını ayarlayalım

<ul className=”profile-info additional-profile-info-list” ref=”toggle”>
  <li>
    <span className=”info-email”>Office Email</span>   me@shuvohabib.com
  </li>
</ul>

onClick’te ‘handleToggle’ı başlatacağımız div öğesi.

 <div className=”ellipsis-click” onClick={this.handleToggle}>
   <i className=”fa-ellipsis-h”/>
 </div>

Aşağıdaki kodun tamamını nasıl göründüğünü gözden geçirelim.

import React from ‘react’;
import { findDOMNode } from ‘react-dom’;
import $ from ‘jquery’;

export default class FullDesc extends React.Component {
    constructor() {
        super();
    }

    handleToggle = () => {
        const el = findDOMNode(this.refs.toggle);
        $(el).slideToggle();
    };

    render() {
        return (
            <div className=”long-desc”>
                <ul className=”profile-info”>
                    <li>
                        <span className=”info-title”>User Name : </span> Shuvo Habib
                    </li>
                </ul>

                <ul className=”profile-info additional-profile-info-list” ref=”toggle”>
                    <li>
                        <span className=”info-email”>Office Email</span> me@shuvohabib.com
                    </li>
                </ul>
                
                <div className=”ellipsis-click” onClick={this.handleToggle}>
                    <i className=”fa-ellipsis-h”/>
                </div>
            </div>
        );
    }
}

İşimiz bitti! jQuery’yi React bileşeninde bu şekilde kullanabiliriz.
