Parametreler #

parametre Ayrıntılar
değer Dönüştürülecek değer
hedefTürü Dönüştürülmekte olan tür
parametre Dönüştürmenin nasıl çalıştığını denetlemek için isteğe bağlı değer
kültür CultureInfo nesnesi - yerelleştirme gerekiyorsa gereklidir

‘Dönüştür’ yöntemi, kaynaktan (genellikle görünüm modeli) gelen değeri hedefe (genellikle bir kontrolün özelliği) dönüştürür.

‘ConvertBack’ yöntemi, hedefteki değeri kaynağa geri dönüştürür. Yalnızca bağlama “TwoWay” veya “OneWayToSource” ise gereklidir.

Bir “ConvertBack” desteklenmediğinde, yani dönüşüm öncesi değer ile dönüşüm sonrası değer arasında bire bir eşleme olmadığında, “ConvertBack” yönteminin “DependencyProperty.UnsetValue” döndürmesi yaygın bir uygulamadır. Beklenmeyen çalışma zamanı hatalarını önlediğinden, bir istisna (örneğin, “NotImplementedException”) oluşturmaktan daha iyi bir seçenektir. Ayrıca bağlamalar, bir dönüştürücü tarafından “DependencyProperty.UnsetValue” döndürüldüğünde “FallbackValue” özelliklerinden yararlanabilir.

Dönüştürücüler 101 #

XAML denetimleri, “DataContext” veya diğer denetimlerdeki nesnelere bağlanabilen bağımlılık özelliklerine sahip olabilir. Bağlanan nesnenin türü, hedef ‘DependencyProperty’ türünden farklı olduğunda, bir türü diğerine uyarlamak için bir dönüştürücü kullanılabilir.

Dönüştürücüler, “System.Windows.Data.IValueConverter” veya “System.Windows.Data.IMultiValueConverter” uygulayan sınıflardır; WPF, bazı hazır dönüştürücüler uygular, ancak geliştiriciler, sık sık olduğu gibi, özel uygulamalarda kullanım görebilir.

XAML’de bir dönüştürücü kullanmak için, “Kaynaklar” bölümünde bir örnek oluşturulabilir. Aşağıdaki örnek için System.Windows.Controls.BooleanToVisibilityConverter kullanılacaktır:

<UserControl.Resources>
    <BooleanToVisibilityConverter x:Key="BooleanToVisibilityConverter"/>
</UserControl.Resources>

Daha sonra bağlamada “BooleanToVisibilityConverter” örneğine başvurmak için kullanılan “x:Key” öğesinin tanımlandığına dikkat edin:

<TextBlock Text="This will be hidden if property 'IsVisible' is true"
           Visibility="{Binding IsVisible, 
                                Converter={StaticResource BooleanToVisibilityConverter}}"/>

Yukarıdaki örnekte, bir boolean “IsVisible” özelliği, “System.Windows.Visibility” numaralandırmasının bir değerine dönüştürülür; Doğruysa “Visibility.Visible”, aksi takdirde “Visibility.Collapsed”.

IsChecked Dönüştürücüye Dize #

XAML’de:

<RadioButton IsChecked="{Binding EntityValue, Mode=TwoWay,
                         Converter={StaticResource StringToIsCheckedConverter},
                         ConverterParameter=Male}"
             Content="Male"/>

<RadioButton IsChecked="{Binding EntityValue, Mode=TwoWay,
                         Converter={StaticResource StringToIsCheckedConverter},
                         ConverterParameter=Female}"
             Content="Female"/>

C# sınıfı:

public class StringToIsCheckedConverter : IValueConverter
{
    public object Convert(object value, Type targetType, object parameter, System.Globalization.CultureInfo culture)
    {
        string input = (string)value;
        string test = (string)parameter;
        return input == test;
    }

    public object ConvertBack(object value, Type targetType, object parameter, System.Globalization.CultureInfo culture)
    {
        if (value == null || !(value is bool))
        {
            return string.Empty;
        }
        if (parameter == null || !(parameter is string))
        {
            return string.Empty;
        }
        if ((bool)value)
        {
            return parameter.ToString();
        }
        else
        {
            return string.Empty;
        }
    }
}

Dönüştürücü Oluşturma ve Kullanma: BooleanToVisibilityConverter ve InvertibleBooleanToVisibilityConverter #

Bağlama deneyimini genişletmek ve genişletmek için, birini bir türün değerini başka bir türün değerine dönüştüren dönüştürücülere sahibiz. Bir Veri Bağlamada Dönüştürücülerden yararlanmak için önce uzantıları olan bir DataConverter sınıfı oluşturmanız gerekir

  • IValueConverter(WPF ve UWP)

veya

  • “IMultiValueDönüştürücü”(WPF)

birden fazla türü tek bir türe dönüştürmek istiyorsanız
Bu durumda, bir “boolean True/False” değerini karşılık gelen Görünürlükler “Visibility.Visible” ve “Visibility.Collapsed"e dönüştürmeye odaklanıyoruz:

public class BooleanToVisibilityConverter : IValueConverter
{
    public object Convert(object value, Type targetType, object parameter, string language)
    {
        return (value is bool && (bool) value) ? Visibility.Visible : Visibility.Collapsed;
    }

    public object ConvertBack(object value, Type targetType, object parameter, string language)
    {
        return (value is Visibility && (Visibility) value == Visibility.Visible);
    }
}

Convert yöntemi, ViewModel GET verisini FROM olarak değiştirdiğinizde çağrılır.
ConvertBack, BindingMode.TwoWay bağlamaları için ViewModel verileri TO SET ile çağrılır.

Elbette dönüştürücünüzdeki özellikleri de kullanabilirsiniz. Şuna bir göz atın:

public class InvertibleBooleanToVisibilityConverter : IValueConverter
{
    public bool Invert { get; set; } = false;

    public object Convert(object value, Type targetType, object parameter, string language)
    {
        return (value is bool && (bool) value != Invert) ? Visibility.Visible : Visibility.Collapsed;
    }

    public object ConvertBack(object value, Type targetType, object parameter, string language)
    {
        return (value is Visibility && ((Visibility) value == Visibility.Visible) != Invert);
    }
}

Bir ‘Binding’ içinde bir dönüştürücü kullanmak istiyorsanız, onu sayfanızda, pencerenizde veya başka bir öğede bir kaynak olarak ilan edin, ona bir anahtar verin ve potansiyel olarak ihtiyaç duyulan özellikleri sağlayın:

<Page ...
    xmlns:converters="using:MyNamespce.Converters">
<Page.Resources>
    <converters:InvertibleBooleanToVisibilityConverter 
        x:Key="BooleanToVisibilityConverter" 
        Invert="False" />
</Page.Resources>

ve onu bir bağlamada ‘StaticResource’ olarak kullanın:

<ProgressRing 
        Visibility="{Binding ShowBusyIndicator, 
            Converter={StaticResource BooleanToVisibilityConverter},
            UpdateSourceTrigger=PropertyChanged,
            Mode=OneWay}" />
