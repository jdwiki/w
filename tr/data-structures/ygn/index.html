Yığına Giriş #

Yığın, bir LIFO (son giren ilk çıkar) veri yapısıdır, yani yığına eklenen en son (veya “son giren”) öğe, kaldırılan ilk öğe (“ilk çıkar”) olacaktır.

Kutudaki kitap örneğini ele alalım. Bir seferde yalnızca bir kitap eklenebilir veya kutudan çıkarılabilir ve yalnızca üstten eklenebilir ve kaldırılabilir.

Şimdi, ilk iki kitabın olduğu kutu şuna benziyor:

 |--------|
| 2. kitap | ◀──── kutunun üstü
|-------|
| kitap 1 | ◀──── kutunun altı
|-------|
  1. kitabı eklersek, en üstte olacak. 3. kitaptan önce eklenen kitapların geri kalanı onun altında kalacak:
 |--------|
| kitap 3 | ◀──── kutunun üstü
|-------|
| 2. kitap |
|-------|
| kitap 1 | ◀──── kutunun altı
|-------|

Kutu bir yığın gibidir ve kitaplar veri öğeleridir. Kutuya kitap eklemek itme işlemidir, kutunun üstünden kitap çıkarmak/almak ise pop işlemidir. Pop işlemini yaparsak, 3. kitabı alacağız ve kutu, 3. kitabı basmadan önceki haline geri dönecektir. çıktı (LIFO). 1. kitabı almak için iki pop daha yapmamız gerekiyor: biri 2. kitabı kaldırmak, ikincisi 1. kitabı almak için.

Bir dizi kullanarak yığının uygulanması. Bunun için en üst konumu (tos) gösteren bir indekse ihtiyacımız var. Bir öğe yığına her itildiğinde, tos birer birer artar ve bir pop işlemi gerçekleştirildiğinde yığının tepesini (tos) gösteren indeks bir azalır.

İTMEK:

PUSH işleminden önce

                                  tos
                                   |
                                   |
        |-----|-----|-----|-----|-----|
        | i1  | i2  | i3  |  i4 |     |
        |-----|-----|-----|-----|-----|

void push(dataElment item){
    stack[top]=item; //item = i4
    top++;
    
}

PUSH işleminden sonra Yığın, yığının en üstüne bir işaretçiye sahiptir. Push işlemi her çağrıldığında, değeri en üste yerleştirir ve değeri günceller.

        tos -- Top of the stack
                            tos
                             |
                             |
        |-----|-----|-----|-----|
        | i1  | i2  | i3  |     |
        |-----|-----|-----|-----|

Pop : Pop işlemi, içeriği yığının tepesinden kaldırır ve tos değerini 1 azaltarak günceller.

Pop işleminden önce:

                                  tos
                                   |
                                   |
        |-----|-----|-----|-----|-----|
        | i1  | i2  | i3  |  i4 |     |
        |-----|-----|-----|-----|-----|

dataElment pop(){
    dataElment value = stack[tos--];
    return value;
}

Pop işleminden sonra:

                            tos
                             |
                             |
        |-----|-----|-----|-----|
        | i1  | i2  | i3  |  i4 |
        |-----|-----|-----|-----|
        When a push operation is performed it overwrites i4.

Palindromları bulmak için yığınları kullanma #

Palindrom, ‘kayak’ gibi her iki şekilde de okunabilen bir kelimedir.

Bu örnek, Python3 kullanılarak verilen bir kelimenin palindrom olup olmadığının nasıl bulunacağını gösterecektir.

İlk olarak, bir diziyi yığına çevirmemiz gerekiyor (dizileri yığın olarak kullanacağız).

str = "string"
stk = [c for c in str] #this makes an array: ["s", "t", "r", "i", "n", "g"]
stk.append("s") #adds a letter to the array
stk.pop() #pops the last element

Şimdi, kelimeyi tersine çevirmeliyiz.

def invert(str):
    stk = [c for c in str]
    stk2 = []
    while len(stk) > 0:
        stk2.append(stk.pop())
    #now, let's turn stk2 into a string
    str2 = ""
    for c in stk2:
        str2 += c
    return str2

Şimdi kelimeyi ve ters çevrilmiş formu karşılaştırabiliriz.

def palindrome(str):
    return str == invert(str)

Dizi ve Bağlantılı Liste kullanarak Yığın Uygulaması #

Dizi Uygulaması

#include<stdio.h>

#define MAX 100000                    //Global variables
int top = -1;
int a[MAX];

void push(int x){
    
    if(top == MAX-1){                         // Check whether stack is full
        printf("Stack Overflow\n");
        return;
    }
    
    a[++top] = x;                            // Otherwise increment top and insert x
}

void pop(){
    if(top == -1){                                     // if top = -1, empty stack
        printf("Empty stack\n");
        return;
    }
    top--;        
}

void print(){                         //printing stack values
    
    for(int i=0;i <= top;i++){
        printf("%d ",a[i]);        
    }        
    printf("\n");        
}

void topValue(){                        // Method can print the top value of a stack
    printf("%d",a[top]);
}
int main(){
        
    push(5);
    push(20);
    push(15);
    print();
    pop();
    print();
    push(35);
    print();
    topValue();    
    
}

Bağlantılı Liste uygulaması

#include<stdio.h>
#include<malloc.h>

 struct node{
    int data;
    node* link;
};
node* top = NULL;

void push(int data){
    
    node* temp = (struct node*)malloc(sizeof(struct node*));
    temp->data = data;
    temp->link = top;
    top = temp;
}

void pop(){
    
    if(top == NULL) return;
    node* temp = top;
    top = top->link;
    free(temp);
}
void print(){
    
    node* t = top;
    while(t != NULL){
        printf("%d ",t->data);
        t=t->link;
    }
    printf("\n");
}

void topValue(){
    printf("%d ",top->data);
}
int main(){
        
    push(5);
    push(20);
    push(15);
    print();
    pop();
    print();
    push(35);
    print();            
    topValue();    
}

Dengeli Parantezleri Kontrol Etme #

Köşeli ayraç şu karakterlerden herhangi biri olarak kabul edilir: (, ), {, }, [ veya ].

Bir açılış ayracı (yani, (, [ veya {) kapanış parantezinin (yani, ), ] veya }) tamamen aynı türden. Üç tür eşleşen parantez çifti vardır: [], {} ve ().

Eşleşen bir parantez çifti, içerdiği parantezler eşleşmezse dengelenmez. Örneğin, {[(])} dengeli değildir çünkü { ve } arasındaki içerikler dengeli değildir. Köşeli parantez çifti tek, dengesiz bir açma ayracını () içine alır ve parantez çifti tek, dengesiz bir köşeli ayraç ] içine alır.

Bu mantıkla, aşağıdaki koşullar yerine getirildiğinde bir parantez dizisinin dengeli kabul edildiğini söylüyoruz:

Eşleşmeyen parantez içermez.

Eşleşen bir çiftin sınırları içine alınmış parantezlerin alt kümesi of parantez aynı zamanda eşleşen bir parantez çiftidir.

Algoritma:

  1. Bir yığın bildirin (‘yığın’ deyin).
  2. Şimdi dize girişini geçin.
  • a) Geçerli karakter bir başlangıç ​​ayracıysa (yani, ( veya { veya [ ) ) istiflemek için itin.
  • b) Geçerli karakter bir kapanış parantez ise (yani ) veya } veya ]) sonra yığından çıkar. Açılan karakter eşleşen ise başlangıç ​​parantez sonra ince parantez ‘dengeli değil’.
  • c) Geçerli karakter bir kapatma ayracıysa (yani ) veya } veya ]) ve yığın boşsa, parantezler ‘dengelenmemiş’.
  1. Geçişi tamamladıktan sonra, yığında bir miktar başlangıç ​​ayracı varsa, o zaman dizge ‘dengeli değil’, yoksa ‘dengeli’ bir dizgemiz var.
