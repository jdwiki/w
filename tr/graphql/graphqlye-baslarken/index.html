GraphQL Sorgu Dili (GQL) #

Her veri kaynağı için URL uç noktaları tanımlamak yerine, GraphQL’de GraphQL sorgularını kabul eden tek bir uç nokta tanımlarsınız. Geleneksel veritabanı sorgu dillerinden farklı olarak, GraphQL Sorgu Dili (GQL), kök düzeyinde bir sorgu tarafından döndürülen verilerin bir projeksiyonudur. GraphQL şeması, veri modelini ve kök düzeyinde sorguları ve mutasyonları tanımlayacaktır. GQL bu sorguları yürütür ve hangi verilerin döndürüleceğini tanımlar.

Şema Tanımı #

GraphQL‘de Şema, verileriniz için kök yürütmeyi sorgular ve mutasyonlar ve ayrıca türleri tanımlar.

Şema Nesne Türü #

‘Kişi’ türünün iki alanı vardır, biri standart bir Skaler türdür ve diğeri ‘arkadaş’ olan diğer Kişi türlerinin bir listesiyle olan ilişkiyi temsil eder. GraphQL’i bu kadar güçlü yapan şey diğer türleri birbirine bağlamaktır. Artık GraphQL Sorgu Dili’nde (GQL), istemci herhangi bir ek kod veya gelişmiş sorgulama olmadan arkadaşlar grafiğini geçebilir.

type Person {
  id: ID
  name: String
  friends: [Person]
}

Şema Sorgusu #

“kişi” sorgusu, kimliğine göre tek bir kişiyi arar. Bu, GQL kullanan müşteriler için verilerinize giriş noktasıdır.

type Query {   
  person(id: ID!): Person
}

Nick’in Arkadaşının Arkadaşının Arkadaşlarını Sorgula #

Artık bir Kişi tipimiz ve bir kişi kök sorgumuz olduğuna göre, bir kişiyi ve kişinin arkadaş ağından istediğimiz kadar ayrılma derecesini arayabiliriz.

query {
  person(id: 'nick'){
    id
    name
    friends{
      id
      name
      friends{
        id
        name
        friends{
          id
          name
        }
      }
    }
  }
}

Sunucu Kurulumu ve Uygulaması #

GraphQL.js #

GraphQL.js, GraphQL için bir JavaScript referans uygulamasıdır. npm ile yükleyebilirsiniz:

  • Henüz yapmadıysanız, projenizde npm’yi başlatın: `npm init’
  • GraphQL.js’yi npm’den yükleyin: npm install --save graphql

Örnek Sunucu #

var { graphql, buildSchema } = require('graphql');

// Construct a schema, using GraphQL schema language
var schema = buildSchema(`
  type Query {
    hello: String
  }
`);

// The root provides a resolver function for each API endpoint
var root = {
  hello: () => {
    return 'Hello world!';
  },
};

// Run the GraphQL query '{ hello }' and print out the response
graphql(schema, '{ hello }', root).then((response) => {
  console.log(response);
});
