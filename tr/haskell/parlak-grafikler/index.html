Parlak Yükleme #

Parlaklık, Cabal aracı kullanılarak kolayca kurulur. Cabal’ı kurduktan sonra, Gloss’u kurmak için cabal install gloss çalıştırılabilir.

Alternatif olarak paket, kaynağı Hackage veya GitHub‘den indirerek ve aşağıdakileri yaparak kaynaktan oluşturulabilir:

  1. gloss/gloss-rendering/ dizinine girin ve cabal install yapın
  2. gloss/gloss/ dizinine girin ve bir kez daha cabal install yapın

Ekranda bir şey alma #

Gloss’ta, çok basit statik grafikler oluşturmak için ‘görüntüleme’ işlevi kullanılabilir.

Bunu kullanmak için önce iport Graphics.Gloss gerekir. Ardından kodda aşağıdakiler olmalıdır:

main :: IO ()
main = display window background drawing

“pencere”, iki şekilde oluşturulabilen “Görüntüleme” türündedir:

-- Defines window as an actual window with a given name and size
window = InWindow name (width, height) (0,0)

-- Defines window as a fullscreen window
window = FullScreen

Burada ‘InWindow’daki son ‘(0,0)’ argümanı sol üst köşenin konumunu işaretler.

1.11’den daha eski sürümler için: Gloss’un eski sürümlerinde “FullScreen”, çizilen çerçevenin boyutu anlamına gelen ve tam ekran boyutuna genişletilen başka bir argüman alır, örneğin: “Tam Ekran” (1024,768)`

“arka plan”, “Renk” türündedir. Arka plan rengini tanımlar, bu kadar basit:

background = white

Sonra çizimin kendisine geçiyoruz. Çizimler çok karmaşık olabilir. Bunların nasıl belirleneceği başka bir yerde ele alınacaktır (şu an için buna atıfta bulunulabilir), ancak yarıçapı 80 olan aşağıdaki daire kadar basit olabilir:

drawing = Circle 80

Özetleme örneği

Hackage ile ilgili belgelerde aşağı yukarı belirtildiği gibi, ekranda bir şey almak şu kadar kolaydır:

import Graphics.Gloss

main :: IO ()
main = display window background drawing
    where
      window = InWindow "Nice Window" (200, 200) (0, 0) 
      background = white 
      drawing = Circle 80
