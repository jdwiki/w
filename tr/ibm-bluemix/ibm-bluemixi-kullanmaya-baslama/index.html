IBM Bluemix Toolchain kullanarak GitHub aracılığıyla RESTful API’nin (Node.js) sürekli dağıtımı #

Adım 1: GitHub hesabınızı oluşturun #

Halihazırda bir GitHub hesabınız varsa, lütfen 2. Adıma geçin. Aksi takdirde, lütfen aşağıdaki adımları izleyin:

1.a Github sayfasına gidin.

1.b İstediğiniz kullanıcı adını, e-posta adresinizi ve ardından istediğiniz şifreyi girin. Ardından, GitHub’a Kaydol düğmesini tıklayın.

Adım 2: IBM Bluemix hesabınızı oluşturun #

Hesap oluşturma konusunda lütfen Bluemix Başlarken bölümüne bakın veya Bluemix hesabınızı nasıl oluşturacağınızla ilgili önceki örneği izleyin. bu sayfa.

Adım 3: Rest API paketinizi GitHub’a dağıtın #

3.a Git’i nasıl kullanacağınızı bilmiyorsanız, lütfen Git Eğitimi’ni okuyun veya GUI tabanlı bir Git uygulamasını SourceTree kullanabilirsiniz. SourceTree hakkında daha fazla bilgi edinmek için lütfen SourceTree Eğitimi okuyun.

3.b Rest API kodunuzu git push komutunu kullanarak GitHub’a yükleyin. Buna karşılık, Rest API‘imi GitHub’da çatallayabilir veya klonlayabilir, ardından gerektiğinde kendi değişikliklerinizi uygulayabilir ve ardından kodu GitHub hesabınıza yükleyebilirsiniz. Not: Örnek Rest API’m, MongoDB ve APICache’nin örnek bir uygulamasını içerir.

Adım 4: Toolchain (Sürekli Dağıtım aracı) kullanarak GitHub deponuzu IBM Bluemix’e dağıtın #

4.a Bluemix hesabınızda oturum açın.

4.b Diğer öğeleri göstermek için Menü’ye tıklayın.

Ardından “Hizmetler"i seçin.

Ardından “DevOps"a tıklayın.

4.c Araç Zincirlerini seçin. Ardından, Bir Araç Zinciri Oluştur düğmesini tıklayın.

4.d Diğer Şablonlar‘ı seçin

4.e İstediğiniz Araç Zinciri Adını girin, lütfen bunun API’ler için URL’nizde gösterileceğini unutmayın.

4.f Bluemix’in Alet Zincirinizi oluşturması için birkaç saniye bekleyin. Ardından, Araç Ekle‘ye tıklayın.

4.g GitHub‘ı seçin

4.h Şimdi GitHub’daki mevcut depomuzu bu Toolchain’e bağlamamız gerekiyor. Depo türü altında lütfen Mevcut‘u seçin. Ardından Rest API kaynağınız için doğru depo URL’sini seçin. Bundan sonra, depodaki değişiklikleri Toolchain üzerinden takip etmek isteyip istemediğiniz size kalmış. Bunu yapmak için lütfen Kod değişikliklerinin dağıtımını izle onay kutusunu işaretleyin.

4.i Şimdi ihtiyacımız olan şey, Rest API depomuzu otomatik olarak oluşturacak ve dağıtacak bir şey. Bu yüzden başka bir araç eklememiz gerekiyor.

4.j Teslimat Hattını ekleyelim

Ardından Oluştur ve Dağıt olarak adlandırın.

Birkaç saniye sonra yeni eklenen Teslimat Hattı görünecektir. Tıkla.

4.k Teslimat Hattımıza aşamalar eklememiz gerekiyor. Sahne Ekle‘yi tıklayın.

4.l Paketimizi otomatik olarak oluşturacak olan Derleme aşamasını oluşturun. Aşağıdaki şekildeki adımları izleyin:

npm install komutunu çalıştıracak bir JOB eklememiz gerekiyor.

Oluştur seçeneğini seçin.

Ardından, Oluşturucu Türü altında npm‘yi seçin. Ardından, Build Shell Command altında, npm install komutunu çalıştıracağını otomatik olarak yapılandırın.

Değişiklikleri kaydetmek için Kaydet düğmesini tıkladığınızdan emin olun.

4.m İnşa aşamasını oluşturduktan sonra, şimdi Dağıtım aşamasını oluşturmamız gerekiyor. Git ve Sahne Ekle‘yi tıklayarak başka bir aşama ekleyin.

Aşağıdaki şekilde tanımlanan adımları izleyin:

Ardından, İŞLER sekmesi altında İş Ekle‘yi tıklayın.

Ardından, Dağıt seçeneğini seçin.

Dağıtıcı Türü altında Cloud Foundry‘yi seçin (IBM Bluemix varsayılan Bulut Hizmeti).

Bu durumda dev alanını seçtim. Gerçek proje için, gerçek üretim kullanımınızı idare edebilecek daha iyi bir alan seçmek isteyebilirsiniz. Ardından, Kaydet düğmesini tıklayın.

4.n Artık tüm aşamalar yapılandırılmıştır. Varsayılan olarak, tüm aşamalar Dur durumundadır. Oynat beğen butonlarına tıklayarak tüm aşamalarımızı Çalıştırmalıyız.

4.o Aşağıda benzer bir şekil görüyorsanız, RESTful API’lerimizin Sürekli Dağıtımını ToolChain kullanarak IBM Bluemix aracılığıyla başarıyla yapılandırdığınız ve uyguladığınız anlamına gelir.

API’nin beklendiği gibi çalışıp çalışmadığını görmek için yukarıdaki şekilde tanımlandığı gibi bağlantıya tıklamayı deneyin.

Adım 5: PostMan kullanarak Rest API’sini test edin #

Postman, güçlü bir API test aracıdır ve bir uzantı olarak Google Chrome aracılığıyla yüklenebilir. Ekspres kodunuzda tanımladığınız rota kuralını takip ettiğinizden emin olun. Postacı aracının nasıl kullanılacağı hakkında daha fazla bilgi için lütfen Postacı Blogu‘a bakın.

Sorumluluk Reddi: #

Bu örnekte verilen herhangi bir harici bağlantıya sahip değilim. Bu bağlantılara sahip olanlara teşekkür ederiz.

Kurulum veya Kurulum #

ibm-bluemix’in kurulmasına veya yüklenmesine ilişkin ayrıntılı yönergeler.

Bir düğüm uygulamasını devreye almak için IBM Bluemix Setup’ı alma #

Adım 1: Bir bluemix hesabı oluşturun #

Bir hesap oluşturun https://console.ng.bluemix.net/registration/

Bu size 30 günlük bir deneme süresi ayarlayacaktır. Ücretsiz kaynaklar için herhangi bir ödeme yapmanız ve deneme sürenizin sonuna kadar faturalandırma ayarlamanız gerekmez (ancak tüm hizmetler kullanılamayabilir).

Adım 2: Bluemix ve Cloud Foundry Komut Satırı araçlarını kurun #

Bu isteğe bağlıdır, çoğu eylemi 1. adımdan itibaren web konsolu aracılığıyla gerçekleştirebilirsiniz. CLI yardımcı programları https://new-console.ng.bluemix.net/docs/starters/install_cli.html adresinden indirilebilir ve her ikisini de yüklemeniz gerekir. bluemix cli ve bulut dökümhane cli.

Adım 2a: Bluemix’e bağlanın #

$ bluemix api https://api.ng.bluemix.net

Adım 2b: Bluemix’te Oturum Açın #

$ bluemix login -u kullanıcı adı -o org_name -s space_name

Adım 1’de web konsolundan alabileceğiniz kullanıcı adınızı, kuruluş_adı ve boşluk_adı belirtmeniz gerekir.

Başarıyla oturum açtıktan sonra bluemix’i kurdunuz. http://clis.ng.bluemix.net/ui/home.html adresinden CLI hakkında daha fazla bilgi edinebilir ve ek eklentiler edinebilirsiniz.
