Toplu iş dosyalarındaki aritmetik sınırlamaları atlayın

  • Powershell’i kullanma
  • jscript kullanma
  • Kalem ve kağıt hesaplamalarını taklit etme, matematik fonksiyonları uygulamaları

Bu sayfada

  • Powershell’i kullanma
  • jscript kullanma
  • Kalem ve kağıt hesaplamalarını taklit etme, matematik fonksiyonları uygulamaları

Toplu dosyalar yalnızca 32 bit tamsayı hesaplamalarına izin verir, ancak bu farklı yaklaşımlarla atlanabilir.

Powershell’i kullanma #

Powershell, 7/2008’den itibaren her Windows sistemine varsayılan olarak yüklendiğinden, daha karmaşık hesaplamalar için kullanılabilir:

@echo off
set "expression=(2+3)*10/1000"
for /f %%# in ('"powershell %expression%"') do set result=%%#
echo %result%

Parantezlerin for komut sözdizimi ile çakışmasını önlemek için “for /f” içindeki ek çift tırnak işaretine dikkat edin.

Potansiyel sorun, powershell’in .net çerçevesinin yüklenmesi nedeniyle wsh/vbscript/jscript kullanmaktan çok daha yavaş olmasıdır.

jscript kullanma #

‘WSH/JScript’, NT’den beri her Windows sistemine kuruludur, bu nedenle onu daha karmaşık hesaplamalar için kullanmak onu oldukça taşınabilir hale getirir. JScript, toplu iş dosyasıyla birleştirmek için daha kolaydır:

@if (@codesection==@batch) @then
@echo off

set "expression=2*(2+3)/1000"
for /f %%# in ('cscript //nologo //e:jscript "%~f0" "%expression%"') do set 
result=%%#
echo %result%
:: more batch code

exit /b %errorlevel%
@end
WScript.Echo(eval(WScript.Arguments(0)));

Bu yaklaşımla tüm kodunuzu tek bir dosyaya koyabilirsiniz. Powershell kullanmaktan daha hızlıdır. Burada ve burada daha gelişmiş komut dosyaları bulunabilir (bunlar harici dosyalar olarak kullanılabilir).

Kalem ve kağıt hesaplamalarını taklit etme, matematik fonksiyonları uygulamaları #

  1. Burada en kapsamlı matematik kitaplığını bulabilirsiniz that emulates pen and paper calculations and allows working with bigger numbers.
  2. İşte başka bir kalem ve kağıt emülasyonu örneği: ADD , Comparison , Multiply
  3. Bazı matematik fonksiyon uygulamaları burada bulunabilir.
