Sözdizimi #

  • if [/i] StringToCompare1 == StringToCompare2 (commandA) else (commandB)

  • if errorlevel 1 (commandA) else (commandB)

  • if %errorlevel% == 1 (commandA) else (commandB)

  • varsa Dosya adı (commandA) başka (commandB)

  • eğer tanımlanmışsa DeğişkenAdı (commandA) başka (commandB)

Bir “if” deyiminde seçilebilecek birkaç sözdizimi vardır. Örnek olarak if string1==string2 kullanacağız.

1-Satır Sözdizimleri #

  • eğer string1==string2 komutA

  • eğer string1==string2 (komutA)

  • if string1==string2 (komutA) başka (komutB)

  • eğer string1==string2 (komutA) başka bir komutB

  • eğer string1==string2 (commandA)else (commandB)

  • eğer string1==string2 (commandA)else commandB

Çok Satırlı Sözdizimleri #

if string1==string2 (
    commandA
)

Veya

if string1==string2 (
    commandA
) else (
    commandB
)

Hala bazı ekstra sözdizimleri mevcut.

Hata Düzeyini Karşılaştırma #

If Errorlevel 1 (
    Echo Errorlevel is 1 or higher
    
    REM The phrase "1 or higher" is used because If Errorlevel 1 statement means:
    REM                                          If %Errorlevel% GEQ 1
    REM                                      Not If %Errorlevel% EQU 1
)

veya

If "%Errorlevel%"=="1" (
    Echo Errorlevel is 1
)

Yukarıdaki komut dosyası Errorlevel(yerleşik) değişkenini kontrol eder. ’not’ operatörü kullanılabilir.

Set "Test=%Errorlevel%"

If "%Test%" == "1" (
    Echo Errorlevel is 1
)

Bu da çalışıyor.

Lütfen bazı komutların hata seviyesini etkilemediğini unutmayın:

  • Kırmak
  • Eko
  • yerel
  • İçin
  • Eğer
  • Duraklat
  • Rem
  • Rd / Rmdir
  • Ayarlamak
  • Başlık

Aşağıdaki komutlar ayarlandı ancak hata seviyesini netleştirmedi:

  • Sınıflar
  • Git
  • Anahtarlar
  • Popd
  • Vardiya

Aşağıdaki komutlar çıkış kodlarını ayarlar ancak hata seviyesini ayarlamaz:

  • Rd / Rmdir

Aşağıdaki komutlar hata seviyesini ayarlar ancak çıkış kodlarını ayarlamaz:

  • Md / Mkdir

EĞER ifadesi ile sayıları karşılaştırma #

SET TEST=0

IF %TEST% == 0 (
    echo TEST FAILED
) ELSE IF %TEST% == 1 (
    echo TEST PASSED
) ELSE (
    echo TEST INVALID
)

Dizeleri karşılaştırma #

IF "%~1" == "-help" (
    ECHO "Hello"
)

burada ‘%1’ ilk komut satırı argümanına atıfta bulunur ve ‘~’ komut dosyası çağrıldığında dahil edilen tüm tırnak işaretlerini kaldırır.

Dosyanın var olup olmadığını kontrol edin #

If exist "C:\Foo\Bar.baz" (
    Echo File exist
)

Bu, C:\Foo\Bar.baz dosyasının varlığını kontrol eder. Bu varsa, Dosya var’ı yansıtır ‘Not’ operatörü de eklenebilir.

Değişken varsa / ayarla #

If Defined Foo (
    Echo Foo is defined
)

Bu, bir değişkenin tanımlanıp tanımlanmadığını kontrol eder. Yine, ‘Not’ operatörü kullanılabilir.
