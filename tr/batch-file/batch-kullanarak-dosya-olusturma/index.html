Toplu iş dosyalarının kullanışlı bir özelliği, onlarla dosya oluşturabilmektir. Bu bölüm, toplu iş kodu kullanılarak dosyaların nasıl oluşturulacağını gösterir.

Sözdizimi #

  • echo (buraya ne olmasını istiyorsanız yazın) » (dosya adı)
  • yankı (değişken adı) » (dosya adı)

Bir dosya varsa, > dosyanın üzerine yazar ve >> dosyanın sonuna eklenir. Bir dosya yoksa, her ikisi de yeni bir dosya oluşturacaktır.

Ayrıca, “echo” komutu, dizenizden sonra otomatik olarak yeni bir satır ekler.

Yani

echo 1 > num.txt
echo 1 > num.txt 
echo 2 >> num.txt

aşağıdaki dosyayı oluşturacaktır:

1
2

Bu değil:

1 1 2

veya

1 2

Ayrıca, bir metin dosyasındaki tek bir satırı değiştiremezsiniz. Tüm dosyayı okumanız, kodunuzda değiştirmeniz ve ardından tüm dosyaya yeniden yazmanız gerekir.

Dosya oluşturmak için yankı #

echo komutuyla dosya oluşturmanın yolları:

ECHO. > example.bat (creates an empty file called "example.bat")

ECHO message > example.bat (creates example.bat containing "message")
ECHO message >> example.bat (adds "message" to a new line in example.bat)
(ECHO message) >> example.bat (same as above, just another way to write it)

ECHO komutu ile bilgisayarınızdaki belirli bir dizinde bir dosya oluşturmak isterseniz, bir sorunla karşılaşabilirsiniz.

ECHO Hello how are you? > C:\Program Files\example.bat

(This will NOT make a file in the folder "Program Files", and might show an error message)

Ama sonra nasıl yapacağız? Aslında son derece basit… Adında boşluk olan bir yol veya dosya adı yazarken “tırnak işaretleri” kullanmayı unutmayın.

ECHO Hello how are you? > "C:\Program Files\example.bat"
(This will create "example.bat" in the folder "Program Files")

Peki ya yeni bir dosya çıktısı veren bir dosya yapmak isterseniz?

ECHO message > file1.bat > example.bat

(example.bat is NOT going to contain "message > file1.bat")
example.bat will just contain "message"... nothing else

O zaman bunu nasıl yaparız? Bunun için ^ sembolünü kullanıyoruz.

ECHO message ^> file1.bat > example.bat

(example.bat is going to contain "message > file1.bat")

Aynı şey toplu haldeki diğer şeyler için de geçerli

Bir sonraki adım, değişkenler ve ifadeler hakkında biraz bilgi sahibi olmanızı gerektirir:

değişkenler hakkında bilgi edinmek için burayı tıklayın | if ve else ifadeleri hakkında bilgi edinmek için burayı tıklayın

Değişkenler:

SET example="text"
ECHO %example% > file.bat
(This will output "text" to file.bat)

“metin” çıktısını istemiyorsak, ancak yalnızca düz %example% yazmasını istiyorsak:

ECHO ^%example^% > file.bat
(This will output "%example%" to file.bat)

IF/ELSE ifadeleri:

ELSE statements are written with "pipes" ||

IF ^%example^%=="Hello" ECHO True || ECHO False > file.bat

(example.bat is going to contain "if %example%=="Hello" echo True")
[it ignores everything after the ELSE statement]

tüm satırı çıktı almak için daha önce olduğu gibi yaparız.

IF ^%example^%=="Hello" ECHO True ^|^| ECHO False > file.bat

This will output:
IF %example%=="Hello" ECHO True || ECHO False

Değişken “Merhaba” ise “Doğru”, ELSE ise “Yanlış” diyecektir.

Yönlendirme #

Biçim:

[command] [> | >>] [filename]

> [komut] çıktısını [dosya adı] içine kaydeder.

>> [komut] çıktısını [dosya adı] içine * ekler*.

Örnekler:

  1. “echo Merhaba Dünya > myfile.txt” “Merhaba Dünya"yı dosyam.txt dosyasına kaydeder

  2. adınız %name% şeklinde yankılanır >> dosyam.txt dosyam.txt dosyasına “adınız xxxx” ifadesini ekler

  3. dir C:\ > directory.txt, C:\ dizinini directory.txt dosyasına kaydeder

Birçok komutun çıktısını kaydetme #

Bir toplu iş dosyası oluşturmak için birçok ‘ECHO’ komutunu kullanma:

(
  echo echo hi, this is the date today
  echo date /T
  echo echo created on %DATE%
  echo pause >nul
)>hi.bat

Komut yorumlayıcı, parantez içindeki tüm bölümü tek bir komut olarak değerlendirir ve ardından tüm çıktıyı hi.bat dosyasına kaydeder.

“hi.bat” artık şunları içeriyor:

echo hi, this is the date today
date /T
echo created on [date created]
pause >nul
