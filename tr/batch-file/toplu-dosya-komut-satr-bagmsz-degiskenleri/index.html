Toplu dosya komut satırı bağımsız değişkenleri

9’dan fazla bağımsız değişken içeren toplu dosyalar #

9’dan fazla argüman sağlandığında, “shift [/n]” komutu kullanılabilir, burada “/n” n. argümandan başlamak anlamına gelir, n sıfır ile sekiz arasındadır.

Argümanlar arasında döngü:

:args
set /a "i+=1"
set arg!i!=%~1
call echo arg!i! = %%arg!i!%%
shift
goto :args

Yukarıdaki örnekte gecikmeli genişletme değişkeni “i"nin değişkenler dizisine argüman değerleri atamak için kullanıldığını unutmayın. call komutu, bu tür değişken değerleri döngü içinde görüntülemeye izin verir.

Sayım bağımsız değişkenleri:

for %%i in (%*) do (set /a ArgCount+=1)
echo %ArgCount%

n’inci bağımsız değişkene bir değişken ayarlayın:

set i=5
call set "path%i%=%%~i"

Toplu iş dosyalarına sağlanan komut satırı bağımsız değişkenleri #

Toplu iş dosyası komut satırı bağımsız değişkenleri, toplu işlemi başlatırken gönderilen parametre değerleridir. Boşluk içeriyorsa tırnak içine alınmalıdır. Çalışan bir toplu iş dosyasında, argümanlar çeşitli amaçlar için kullanılır, yani “:labels"e yeniden yönlendirme, değişkenleri ayarlama veya komutları çalıştırma.

Argümanlara toplu iş dosyasında %1, %2, ..., %9 kullanılarak başvurulur.

@echo off
setlocal EnableDelayedExpansion
if not "%1"=="" (
    set "dir=%~1" & set "file=%~2"
    type !dir!\!file! | find /n /i "True" >nul^
     && echo Success! || echo Failure
)
exit /b

C:\Users\UserName> test.bat "C:\Temp\Test Results" "Latest.log"
Success!

Notlar:

  • Yukarıdaki örnekte, “%~1” bağımsız değişken değiştiricisi kullanılarak çift tırnak kaldırılmıştır.
  • Uzun dizeler ^ kullanılarak birkaç satıra bölünür ve sonraki satırda karakterden önce bir boşluk vardır.

Argümanları parantez içinde kaydırma #

Aşağıdaki “example.bat"ı alalım ve “1” , “2” ve “3” argümanlarıyla çağıralım:

@echo off

(
    shift
    shift
    echo %1
)

Son parantez bağlamına ulaşıldıktan sonra değişken genişletme değişeceğinden, çıktı şöyle olacaktır:

1

Bu, argümana erişmek için parantez içinde geçiş yaparken bir sorun olabileceğinden, çağrıyı kullanmanız gerekecek:

@echo off

(
    shift
    shift
    call echo %%1
)

şimdi çıktı “3” olacaktır. Bu teknikle ‘CALL’ komutu kullanıldığından (ek değişken genişlemesine yol açacaktır) bu teknikle erişilen argümanlar da parametrelendirilebilir:

@echo off

set argument=1

    shift
    shift
    call echo %%%argument%

gecikmeli genişleme ile:

@echo off
setlocal enableDelayedExpansion
set argument=1

    shift
    shift
    call echo %%!argument!

çıktı olacak

3
