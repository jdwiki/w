AlchemyLanguage, metninize veya HTML içeriğinize daha derin bir bakış açısı sağlayan bir metin analizi yöntemleri topluluğudur. AlchemyLanguage ve diğer Watson hizmetlerine nasıl başlayacağınızı öğrenmek için Başlarken konusuna bakın. Daha fazla AlchemyLanguage ayrıntısı ve örneği için [API referansı][2] ve [belgeler][3]’e bakın.

Boyut sınırları #

  • Metin temizlemeden önce HTML içeriği: 600 KB
  • Metin temizlemeden sonra kaynak metin: 50 KB
  • Özel Modelleri kullanan aramalar: 5 KB

Dil desteği #

Her bir işlev için hangi dillerin desteklendiğini görmek için [API referansı][4]’ndaki her bir işlevin girişine bakın.

Dil algılama #

AlchemyLanguage, varsayılan olarak kaynak metninizin dilini otomatik olarak algılar. language sorgu parametresi ile içeriğinizin dilini manuel olarak belirleyebilirsiniz. (ör. “dil=ispanyolca”)

Metin temizleme #

API’nin bir HTML veya URL işlevini kullandığınızda, AlchemyLanguage, kaynak metni analize hazırlamak için içeriği temizler. ‘sourceText’ parametresi, aşağıdaki seçeneklerle temizleme işlemini özelleştirmenize olanak tanır:

  • “cleaned_or_raw” (varsayılan) – Bağlantılar, reklamlar vb. gibi web sitesi öğelerini kaldırır. Temizleme başarısız olursa, ham web sayfası metni kullanılır
  • temizlenmiş– Bağlantılar, reklamlar vb. gibi web sitesi öğelerini kaldırır.
  • raw – Temizlenmemiş web sayfası metnini kullanır
  • “cquery” – “cquery” parametresinde belirttiğiniz görsel kısıtlamalar sorgusunu kullanır. Görsel kısıtlama sorguları hakkında ayrıntılar için [belgelere][5] bakın.
  • xpath – xpath parametresinde belirttiğiniz XPath sorgusunu kullanır
  • “xpath_or_raw” – XPath sorgusunun sonuçlarını kullanır, XPath sorgusu hiçbir şey döndürmezse düz metne döner
  • “cleaned_and_xpath” – Temizlenmiş web sayfası metninde bir XPath sorgusunun sonuçlarını kullanır

Birleşik Çağrı: tek bir API çağrısında birden çok işlevi kullanın (Node.js) #

Kombine Çağrı yöntemi, tek bir istekte birden fazla AlchemyLanguage işlevini kullanmanıza olanak tanır. Bu örnek, IBM web sitesinden varlıkları ve anahtar sözcükleri almak için Birleşik Çağrı kullanır ve her sonuç için duyarlılık bilgilerini döndürür.

Bu örnek, AlchemyLanguage hizmeti kimlik bilgileri ve [Node.js][2] gerektirir.

  1. [Watson Developer Cloud Node.js SDK][3]‘yi kurmak için bir komut satırı arayüzü kullanın:

    $ npm install watson-developer-cloud

  2. Aşağıdaki kodu aynı dizindeki bir app.js dosyasına kaydedin. “API_KEY” anahtarını AlchemyAPI anahtarınızla değiştirdiğinizden emin olun:

var AlchemyLanguageV1 = require('watson-developer-cloud/alchemy-language/v1');
var alchemy_language = AlchemyLanguageV1({
  api_key: 'API_KEY'
})

var parameters = {
  extract: 'entities,keywords',
  sentiment: 1, 
  url: 'https://www.ibm.com/us-en/'
};

alchemy_language.combined(parameters, function (err, response) {
  if (err)
    console.log('error:', err);
  else
    console.log(JSON.stringify(response, null, 2));
});

  1. Uygulamayı çalıştırın:

    $ node app.js

Duyarlılık Analizi: metindeki belirli ifadeler için duyarlılık bilgilerini alın (Node.js) #

AlchemyLanguage’ın Hedeflenen Duyarlılık özelliği, içeriğinizde hedef ifadeler arayabilir ve her bir sonuç için duygu bilgisi döndürebilir.

Bu örnek, AlchemyLanguage hizmet kimlik bilgileri ve [Node.js][2] gerektirir

  1. [Watson Developer Cloud Node.js SDK][3]‘yi kurmak için bir komut satırı arayüzü kullanın:

    $ npm install watson-developer-cloud

  2. Aşağıdaki kodu aynı dizindeki bir app.js dosyasına kaydedin. “API_KEY” anahtarını AlchemyAPI anahtarınızla değiştirdiğinizden emin olun:

var AlchemyLanguageV1 = require('watson-developer-cloud/alchemy-language/v1');
var alchemy_language = new AlchemyLanguageV1({
  api_key: 'API_KEY'
})

var parameters = {
  text: 'Grapes are the best! I hate peaches.',
  targets: [
    'grapes',
    'peaches'
  ]
};

alchemy_language.sentiment(parameters, function (err, response) {
  if (err)
    console.log('error:', err);
  else
    console.log(JSON.stringify(response, null, 2));
});

  1. Uygulamayı çalıştırın:

    $ node app.js

Kavramlar: bir web sayfasından kavramları tanımlayın (Node.js) #

AlchemyLanguage, içeriğinizde atıfta bulunulan genel kavramları algılayabilir. Hizmet, her kavram için Bağlantılı Veriler bağlantıları ve mümkün olduğunda ilgili bir web sitesine bir URL döndürür.

Bu örnek, [AlchemyLanguage hizmet kimlik bilgileri][2] ve [Node.js][3] gerektirir

  1. [Watson Developer Cloud Node.js SDK][4]‘ü kurmak için bir komut satırı arayüzü kullanın:

    $ npm install watson-developer-cloud

  2. Aşağıdaki kodu aynı dizindeki app.js dosyasına kaydedin. “API_KEY” anahtarını AlchemyAPI anahtarınızla değiştirdiğinizden emin olun.

var AlchemyLanguageV1 = require('watson-developer-cloud/alchemy-language/v1');
var alchemy_language = new AlchemyLanguageV1({
  api_key: 'API_KEY'
})

var parameters = {
  url: 'http://www.cnn.com'
};

alchemy_language.concepts(parameters, function (err, response) {
  if (err)
    console.log('error:', err);
  else
    console.log(JSON.stringify(response, null, 2));
});

  1. Uygulamayı çalıştırın:

    $ node app.js
