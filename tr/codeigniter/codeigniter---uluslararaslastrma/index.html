CodeIgniter - Uluslararasılaştırma

  • Örnek

Bu sayfada

  • Örnek

CodeIgniter’daki dil sınıfı, uluslararasılaştırma için birden çok dili desteklemenin kolay bir yolunu sunar. Bir dereceye kadar, metni birçok farklı dilde görüntülemek için farklı dil dosyalarını kullanabiliriz.

Örnek #

Dosya oluşturma Dil

Bir dil dosyası oluşturmak için onu _lang.php ile sonlandırmalısınız. Örneğin, Fransızca dili için bir dil dosyası oluşturmak istiyorsanız, onu french_lang.php ile kaydetmelisiniz. Bu dosya içerisinde tüm dil metinlerinizi aşağıda gösterildiği gibi anahtar, değer kombinasyonunda $lang dizisinde saklayabilirsiniz.

$lang['key'] = 'val';

Dil dosyası yükleniyor

Uygulamanızda herhangi bir dili kullanmak için, o dosyada saklanan çeşitli metinleri almak için önce o dilin dosyasını yüklemelisiniz. Dil dosyasını yüklemek için aşağıdaki kodu kullanabilirsiniz.

$this->lang->load('filename', 'language');

filename : Yüklemek istediğiniz dosyanın adıdır. Burada dosyanın uzantısını değil, sadece dosyanın adını kullanın. Dil : Onu içeren dil setidir.

Dil Metni Alınıyor

$this->lang->line('language_key');

Dil dosyasından bir satır almak için aşağıdaki kodu çalıştırmanız yeterlidir. Burada dil_anahtar, yüklenen dil dosyasındaki anahtarın değerini almak için kullanılan anahtar parametresidir.

Otomatik Yükleme Dilleri

Genel olarak bir dile ihtiyacınız varsa, bunu aşağıda gösterildiği gibi application/config/autoload.php dosyasına otomatik olarak yükleyebilirsiniz.

| -----------------------------------------------------------------------
|  Auto-load Language files
| -----------------------------------------------------------------------
| Prototype:
|   $autoload['language'] = array('lang1', 'lang2');
|
| NOTE: Do not include the "_lang" part of your file. For example
| "codeigniter_lang.php" would be referenced as array('codeigniter');
|
*/
$autoload['language'] = array();

Basitçe, CodeIgniter tarafından otomatik olarak yüklenecek farklı dilleri iletin.

“Lang_controller.php” adında bir denetleyici oluşturun ve onu “application/controller/Lang_controller.php” içine kaydedin.

<?php

Lang_controller sınıfı, CI_Controller’ı genişletir {

public function index(){
     //Load form helper
     $this->load->helper('form');

     //Get the selected language
     $language = $this->input->post('language');
    
     //Choose language file according to selected lanaguage
     if($language == "french")
        $this->lang->load('french_lang','french');
     else if($language == "german")
        $this->lang->load('german_lang','german');
     else
     $this->lang->load('english_lang','english');
    
     //Fetch the message from language file.
     $data['msg'] = $this->lang->line('msg');
    
     $data['language'] = $language;
     //Load the view file
     $this->load->view('lang_view',$data);
     }
  }
?>

’lang_view.php’ adlı bir görünüm dosyası oluşturun ve bunu ‘application/views/lang_view.php’ konumuna kaydedin.

<!DOCTYPE html>
<html lang = "en"> 
  <head>
     <meta charset = "utf-8">
     <title>CodeIgniter Internationalization Example</title>
  </head>  
  <body>
     <?php
        echo form_open('/lang');
     ?>
     <select name = "language" onchange = "javascript:this.form.submit();">
        <?php
           $lang = array('english'=>"English",'french'=>"French",'german'=>"German");
            
           foreach($lang as $key=>$val) {
              if($key == $language)
                 echo "<option value = '".$key."' selected>".$val."</option>";
              else
                 echo "<option value = '".$key."'>".$val."</option>";
           }           
        ?>          
     </select>
     <?php
        form_close();
        echo $msg;
     ?>     
  </body>
</html>

Aşağıdaki şekilde gösterildiği gibi uygulama/dil içinde İngilizce, Fransızca ve Almanca adlı üç klasör oluşturun.

Aşağıdaki kodu kopyalayın ve english_lang.php dosyasına application/language/english klasörüne kaydedin.

<?php
   $lang['msg'] = "CodeIgniter Internationalization example.";
?>

Aşağıda verilen kodu kopyalayın ve ‘application/language/French’ klasöründeki ‘french_lang.php’ dosyasına kaydedin.

<?php
   $lang['msg'] = "Exemple CodeIgniter internationalisation.";
?>

Aşağıdaki kodu kopyalayın ve application/language/german klasöründeki german_lang.php dosyasına kaydedin.

<?php
    $lang['msg'] = "CodeIgniter Internationalisierung Beispiel.";
?>

Yukarıdaki denetleyiciye yol eklemek için ‘application/config/routes.php’ içindeki ‘routes.php’ dosyasını değiştirin ve dosyanın sonuna aşağıdaki satırı ekleyin.

$route['lang'] = "Lang_controller";

Yukarıdaki örneği yürütmek için tarayıcıda aşağıdaki URL’yi yürütün.

http://yoursite.com/index.php/lang
