Kurulum veya Kurulum #

Windows’ta #

  • Windows için Visual Studio Code yükleyicisini indirin.
  • İndirildikten sonra yükleyiciyi çalıştırın (VSCodeSetup-version.exe). Bu sadece bir dakika sürecek.

Varsayılan olarak, 64 bit makine için VS Code, C:\Program Files (x86)\Microsoft VS Code altına yüklenir.

Not: VS Kodu için .NET Framework 4.5.2 gereklidir. Windows 7 kullanıyorsanız, lütfen .NET Framework 4.5.2’nin kurulu olduğundan emin olun.

İpucu: Kurulum, isteğe bağlı olarak %PATH% kodunuza Visual Studio Code ekler, böylece konsoldan ‘code ’ yazabilirsiniz. bu klasörde VS Kodunu açmak için. %PATH% çevresel değişkeninde yapılan değişikliğin etkili olması için kurulumdan sonra konsolunuzu yeniden başlatmanız gerekecek.

Mac’te #

  • Mac için Visual Studio Code İndirin.
  • İçeriği genişletmek için indirilen arşive çift tıklayın.
  • Visual Studio Code.app’i Uygulamalar klasörüne sürükleyerek Launchpad’de kullanılabilir hale getirin.
  • Simgeye sağ tıklayıp Seçenekler, Dock’ta Tut’u seçerek Dock’unuza VS Kodu ekleyin.

VS Code’u yola ekledikten sonra ‘code’ yazarak terminalden de çalıştırabilirsiniz:

  • VS Code’u başlatın.
  • Komut Paletini açın (Ctrl+Shift+P) ve Kabuk Komutunu bulmak için ‘kabuk komutu’ yazın: PATH komutunda ‘kod’ komutunu yükleyin.

Yeni $PATH değerinin etkili olması için terminali yeniden başlatın. ‘kod’ yazabileceksiniz. o klasördeki dosyaları düzenlemeye başlamak için herhangi bir klasörde.

Not: .bash_profile’nizde (veya eşdeğeri) eski bir VS Code sürümünden hala eski kod diğer adı varsa, bunu kaldırın ve Shell Komutu: ‘code’ komutunu PATH komutunda çalıştırarak değiştirin.

Linux’ta #

Debian ve Ubuntu tabanlı dağıtımlar #

Debian/Ubuntu tabanlı dağıtımları kurmanın en kolay yolu, .deb paketini (64-bit) indirmek ve varsa grafik yazılım merkezi aracılığıyla veya aşağıdaki komut satırı aracılığıyla kurmaktır:

sudo dpkg -i <file>.deb
sudo apt-get install -f # Install dependencies

.deb paketinin yüklenmesi, normal sistem mekanizmasını kullanarak otomatik güncellemeyi etkinleştirmek için apt deposunu ve imzalama anahtarını otomatik olarak yükleyecektir. İndirme sayfasında 32 bit ve .tar.gz ikili dosyalarının da mevcut olduğunu unutmayın.

Depo ve anahtar, aşağıdaki komut dosyasıyla manuel olarak da yüklenebilir:

curl https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > microsoft.gpg
sudo mv microsoft.gpg /etc/apt/trusted.gpg.d/microsoft.gpg
sudo sh -c 'echo "deb [arch=amd64] https://packages.microsoft.com/repos/vscode stable main" > /etc/apt/sources.list.d/vscode.list'

Ardından paket önbelleğini güncelleyin ve aşağıdakileri kullanarak paketi yükleyin:

sudo apt-get update
sudo apt-get install code # or code-insiders for insiders build

RHEL, Fedora ve CentOS tabanlı dağıtımlar #

Şu anda kararlı 64 bit VS Kodunu bir yum deposuna gönderiyoruz, aşağıdaki komut dosyası anahtarı ve depoyu yükleyecektir:

sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
sudo sh -c 'echo -e "[code]\nname=Visual Studio Code\nbaseurl=https://packages.microsoft.com/yumrepos/vscode\nenabled=1\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" > /etc/yum.repos.d/vscode.repo'

Ardından paket önbelleğini güncelleyin ve paketi dnf (Fedora 22 ve üzeri) kullanarak kurun:

dnf check-update
sudo dnf install code

Veya yum kullanan eski sürümlerde:

yum check-update
sudo yum install code

openSUSE ve SLE tabanlı dağıtımlar #

Yukarıdaki yum deposu, openSUSE ve SLE tabanlı sistemler için de çalışır, aşağıdaki komut dosyası anahtarı ve depoyu yükleyecektir:

sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
sudo sh -c 'echo -e "[code]\nname=Visual Studio Code\nbaseurl=https://packages.microsoft.com/yumrepos/vscode\nenabled=1\ntype=rpm-md\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" > /etc/zypp/repos.d/vscode.repo'

Ardından paket önbelleğini güncelleyin ve aşağıdakileri kullanarak paketi yükleyin:

sudo zypper refresh
sudo zypper install code

Arch Linux için AUR paketi #

VS Kodu için topluluk tarafından yönetilen bir Arch User Repository (AUR) paketi vardır.

.rpm paketini manuel olarak yükleme .rpm paketi (64-bit) ayrıca manuel olarak indirilebilir ve kurulabilir, ancak yukarıdaki veri havuzu kurulmadıkça otomatik güncelleme çalışmayacaktır. İndirildikten sonra paket yöneticiniz kullanılarak yüklenebilir, örneğin dnf ile:

sudo dnf install <file>.rpm

32-bit ve .tar.gz ikili dosyalarının indirme sayfasında da mevcut olduğunu unutmayın.

İlk Adımlar (C++): HelloWorld.cpp #

Herhangi bir dilde tipik olarak yazılan ilk program, “merhaba dünya” yazısıdır. Bu örnek, bu programın nasıl yazılacağını ve Visual Studio Code kullanılarak hatalarının nasıl ayıklanacağını gösterir (Bundan sonra Visual Studio Code’a VS Code olarak değineceğim).

Projeyi Oluştur

Adım 1, yeni bir proje oluşturmak olacaktır. Bu birkaç yolla yapılabilir. İlk yol doğrudan kullanıcı arayüzündendir.

  1. VS Code programını açın. Standart karşılama ekranıyla karşılaşacaksınız (görüntülerin bir Mac üzerinde çalışırken çekildiğini, ancak kurulumunuza benzer olmaları gerektiğini unutmayın):

  1. Başlat menüsünden Yeni dosya‘yı seçin. Bu, komut dosyamızı oluşturmaya başlayabileceğimiz yeni bir düzenleme penceresi açacaktır. Devam edin ve bu dosyayı kaydedin (bunu yapmak için File > Save menüsünü kullanabilirsiniz). Bu örnek için HelloWorld.cpp dosyasını çağıracağız ve onu VSC_HelloWorld/ olarak adlandıracağımız yeni bir dizine yerleştireceğiz.

  2. Programı yazın. Bu oldukça basit olmalıdır, ancak aşağıdakileri dosyaya kopyalamaktan çekinmeyin:

    #include

    int main() { std::cout « “Hello world!” « std::endl; return 0; }

Kodu Çalıştırın

Sonra betiği çalıştırmak ve çıktısını kontrol etmek istiyoruz. Bunu yapmanın çeşitli yolları var. En basiti bir terminal açmak ve oluşturduğumuz dizine gitmek. Artık betiği derleyebilir ve şunu yazarak gcc ile çalıştırabilirsiniz:

$ g++ HelloWorld.cpp -o helloworld
$ ./helloworld
Hello World!

Evet, program çalıştı! Ama gerçekten istediğimiz bu değil. Programı VSCode’un içinden çalıştırabilseydik çok daha iyi olurdu. Yine de şanslıyız! VSCode, “Görünüm” > “Entegre Terminal” menüsünden erişebileceğimiz yerleşik bir terminale sahiptir. Bu, pencerenin alt yarısında VSC_HelloWorld dizinine gidebileceğiniz ve yukarıdaki komutları çalıştırabileceğiniz bir terminal açacaktır.

Genellikle bunu bir Run Task yürüterek yaparız. Menüden “Görevler” > “Görevi Çalıştır…” öğesini seçin. Pencerenin üst kısmına yakın bir yerde bir hata mesajı içeren küçük bir açılır pencere gördüğünü fark edeceksiniz (

İlk program (C++): Merhaba World.cpp #

Bu örnek, C++‘da bir “merhaba dünya” programının nasıl yazılacağını göstererek size VS Kodunun temel işlevselliğini tanıtır. Devam etmeden önce “ms-vscode.cpptools” uzantısının kurulu olduğundan emin olun.

Projeyi Başlatın

İlk adım yeni bir proje oluşturmaktır. Bunu yapmak için VS Code programını yükleyin. Tipik karşılama ekranı ile karşılanacaksınız:

İlk programı oluşturmak için karşılama ekranından “Başlat” > “Yeni dosya” öğesini seçin. Bu yeni bir dosya penceresi açacaktır. Devam edin ve dosyayı ("Dosya" > “Kaydet”) yeni bir dizine kaydedin. Dizine istediğiniz herhangi bir ad verebilirsiniz, ancak bu örnek “VSC_HelloWorld” dizinini ve “HelloWorld.cpp” dosyasını çağıracaktır.

Şimdi asıl programı yazın (aşağıdaki metni kopyalamaktan çekinmeyin):

#include <iostream>

int main()
{
    // Output the hello world text
    std::cout << "Hello world!" << std::endl;
    return 0;
}

Harika! Ayrıca, “ms-vscode.cpptools” uzantısını yüklediğiniz için kod vurgulamanızın da oldukça iyi olduğunu fark edeceksiniz. Şimdi kodu çalıştırmaya geçelim.

Komut Dosyasını Çalıştırma (temel)

“HelloWorld.cpp” dosyasını VS Kodunun içinden çalıştırabiliriz. Böyle bir programı çalıştırmanın en basit yolu, entegre terminali ("Görünüm" > “Entegre Terminal”) açmaktır. Bu, görünümün alt kısmında bir terminal penceresi açar. Bu terminalin içinden oluşturduğumuz dizine gidebilir, yazdığımız betiği oluşturabilir ve yürütebiliriz. Burada kodu derlemek ve çalıştırmak için aşağıdaki komutları kullandık:

$ g++ HelloWorld.cpp -o hellowold
$ ./hellowold

Beklenen Merhaba Dünya! çıktısını aldığımıza dikkat edin.

Komut Dosyasını Çalıştırma (biraz daha gelişmiş)

Harika, ancak kodu oluşturmak ve yürütmek için VS Kodunu doğrudan kullanabiliriz. Bunun için öncelikle “VSC_HelloWorld” dizinini bir çalışma alanına çevirmemiz gerekiyor. Bu şu şekilde yapılabilir:

  1. Explorer menüsünü açma (en soldaki dikey menüde en üstteki öğe)
  2. Klasörü Aç düğmesini seçin
  3. Üzerinde çalıştığımız “VSC_HelloWorld” dizinini seçin. Not: VS Kodu içinde bir dizini açarsanız (örneğin “Dosya” > “Aç…” kullanarak) zaten bir çalışma alanında olursunuz.

Explorer menüsü artık dizinin içeriğini görüntüler.

Ardından, VS Kodunun çalıştırmasını istediğimiz gerçek görevleri tanımlamak istiyoruz. Bunu yapmak için “Görevler” > “Varsayılan Oluşturma Görevini Yapılandır” öğesini seçin. Açılır menüde “Diğer” öğesini seçin. Bu, bir görev için bazı varsayılan değerleri içeren “tasks.json” adlı yeni bir dosya açar. Bu değerleri değiştirmemiz gerekiyor. Bu dosyayı aşağıdakileri içerecek şekilde güncelleyin ve kaydedin:

{
    "version": "2.0.0",
    "tasks": [
        {
            "taskName": "build",
            "type": "shell",
            "command": "g++ HelloWorld.cpp -o helloworld"
        },
        {
            "taskName": "run",
            "type": "shell",
            "command": "${workspaceRoot}/helloworld"
        }
    ]
}

Yukarıdakilerin ayrıca çalışma dizinimizde gizli bir .vscode dizini oluşturduğunu unutmayın. Burası, VS Code’un projeye özel ayar dosyaları dahil olmak üzere yapılandırma dosyalarını koyduğu yerdir. Tasks hakkında daha fazla bilgiyi burada bulabilirsiniz.

Yukarıdaki örnekte, “${workspaceRoot}”, “VSC_HelloWorld” dizinimiz olan çalışma alanımızın en üst düzey dizinine başvurur. Şimdi, projeyi metodun içinden inşa etmek için “Tasks” > “Run Build Task…” öğesini seçin ve oluşturduğumuz “build” görevimizi seçin ve “Görevi taramadan Continue” öğesini seçin. Çıktı" görünen açılır menülerden. Ardından “Tasks” > “Run Task…” komutunu kullanarak ve oluşturduğumuz “run” görevini seçerek çalıştırılabilir dosyayı çalıştırabiliriz. Entegre terminal açıksa, “Merhaba Dünya!” metin orada yazdırılacaktır.

Siz çıktıyı göremeden terminalin kapanması mümkündür. Bu olursa, aşağıdaki gibi bir kod satırı ekleyebilirsiniz int i; std::cin >> i;, main() fonksiyonunun sonundaki return ifadesinden hemen önce. Ardından, herhangi bir sayı yazıp <Enter> tuşlarına basarak komut dosyasını sonlandırabilirsiniz.
