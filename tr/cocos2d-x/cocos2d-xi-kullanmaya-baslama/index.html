Yükleme veya Kurulum(Mac OS X) #

GENEL BAKIŞ #

cocos2d-x, açık kaynaklı, platformlar arası bir oyun motorudur. Geliştiricilerin iOS, Android, Windows Phone, Mac OS X, Windows Desktop ve Linux’ta C++, Lua ve Javascript dağıtımında kodlama yapmasına olanak tanır.

ÖNKOŞULLAR #

Yapı Gereksinimleri

  • Mac OS X 10.7+, Xcode 4.6+
  • Windows 7+, VS 2012+
  • Ubuntu 12.10+, CMake 2.6+
  • Python 2.7.5
  • NDK, JDK, ANT (Android geliştirme)

Çalışma Zamanı Gereksinimleri

  • iPhone / iPad oyunları için iOS 5.0+
  • Android oyunları için Android 2.3+
  • Windows Phone oyunları için Windows Phone 8+
  • Mac oyunları için OS X v10.6+
  • Windows oyunları için Windows 7+

Kurulum #

Aşağıdaki her şeyi indirin:

  • Cocos2d-x v3.x
  • JDK/SDK 1.6+
  • Android komut satırı araçları
  • GDR
  • Apache Karınca
  • Python 2.7.5

Yukarıdaki her şeyi indirdikten sonra,

  • Cocos2d-x’in sıkıştırmasını açın
  • JDK’yı yükleyin.
  • Android SDK ve NDK’yi aynı kök konuma açın. (belki: ~/AndroidDev)
  • Python 2.7’nin kurulu olduğunu ve erişilebilir olduğunu doğrulayın.
  • Apache Ant’i kurun ve çalıştığını doğrulayın.

Cocos2d-x dizininizin içinde şunu çalıştırın:

python setup.py

dikkat: ~ işaretini kullanmamalısınız. home dizininizin tam yolunu kullanın. Aksi takdirde, komut dosyaları hata yolu değeri nedeniyle başarısız olur.

  • Şimdi NDK_ROOT yolunu soracak. Buraya, çıkarılan NDK dizinini girin yol:

~/android-ndk-r10e

  • SDK_ROOT. Buraya Android SDK yolunu girin:

~/android-sdk-macosx

  • ANT_ROOT. Buraya Apache ANT bin yolunu girin:

~/apache-ant-1.9.6/bin

Yukarıdaki ortam değişkenlerini ayarladıktan sonra şunu çalıştırın:

source ~/.bash_profile

Vola, cocos2d-x sisteminize başarıyla kuruldu.
