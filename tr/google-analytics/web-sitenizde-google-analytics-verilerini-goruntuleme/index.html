Web Sitenizde Google Analytics verilerini görüntüleme #

Bu belge, Google Access belirteçlerinin nasıl alınacağını ve bunları Google Analytics verilerinin web sitelerimizde görüntülenmesi için nasıl kullanılacağını açıklar.

Örnek: Canlı bir örnek şurada mevcuttur:

https://newtonjoshua.com

not: Aşağıdaki tüm adımlar için aynı gmail hesabını kullanın.

  1. ADIM: Google Analytics’i Kurun

Google Analytics’i web sitenize kurmak için aşağıdaki adımları izleyin

  1. Analytics hesabınızda oturum açın.
  2. Yönetici sekmesini seçin.
  3. HESAP sütunundaki açılır menüden bir hesap seçin.
  4. MÜLK sütunundaki açılır menüden bir mülk seçin.
  5. MÜLK altında, İzleme Bilgileri > İzleme Kodu’nu tıklayın.
  6. Veri toplamak için Analytics izleme kodunu kopyalayıp yapıştırmanız gerekir. into the source code on every web page you wish to track.
  7. Sizin için Javascript izleme kodu parçacığına sahip olduğunuzda property, copy the snippet exactly without editing it.
  8. İzleme kodu snippet’inizi (tamamen, değiştirilmeden) önce yapıştırın. the closing tag on every web page on your site you wish to track.
  9. Analytics izlemeyi başarıyla yükledikten sonra, up to 24 hours for data such as traffic referral information, user characteristics, and browsing information to appear in your reports

bahsetmek,

  1. https://support.google.com/analytics/answer/1008080?hl=tr
  2. https://analytics.google.com

ADIM 2: Jetonları Alın #

Google Projesi:

Bir Google Cloud Platform projesi oluşturmak için Google Developers Console’u ( https://console.developers.google.com ) açın ve Proje Oluştur‘a tıklayın.

OAuth 2.0 API erişimini etkinleştirin:

Uygulamanızın, kullanıcı verilerine erişmesi ve sizin adınıza diğer Google hizmetleriyle iletişim kurması gerekir. Uygulamanıza API erişimi vermek için OAuth 2.0’ı kullanın.

Bunu etkinleştirmek için bir Müşteri Kimliğine ihtiyacınız var:

  1. Google API Konsolu Kimlik Bilgileri sayfasını açın (https://console.developers.google.com/apis/credentials).

  2. Projeden açılır menüden projenizi seçin.

  3. Kimlik bilgileri oluştur’u ve OAuth istemci kimliği’ni seçin.

  4. Uygulama türü altında, Web uygulaması’nı seçin, bir Ad girin ve

  5. ayarla the Restrictions by entering JavaScript origins, Redirect URIs to point the website where you are planning to display the data, then click Create.

  6. Not edin OAuth 2.0 client_id and client_secret. You will need them to configure the UI.

Onay Kodu Al:

Tarayıcıya girin,

https://accounts.google.com/o/oauth2/auth?scope=https://www.googleapis.com/auth/analytics.readonly&response_type=code&client_id={{ client_id}}&redirect_uri={{redirect_uri }} &approval_prompt=force&access_type=çevrimdışı

Şuraya yönlendirileceksiniz:

{{redirect_uri }}?code=={{yetkilendirme_kodu}}#

Yenileme Simgesini Alın:

Bir POST isteği gönderin, muhtemelen bir REST konsolu aracılığıyla

https://www.googleapis.com/oauth2/v3/token?code={{yetkilendirme_kodu}} &client_id={{client_id}}&client_secret={{client_secret}} &redirect_uri={{redirect_uri }}&grant_type=yetkilendirme_kodu

İle bir JSON yanıtı alacaksınız

{“refresh_token”: refresh_token}

Google API’lerine erişmek üzere erişim belirtecini almak için yenileme belirtecini kullanabilirsiniz.

Erişim izni almak:

POST isteği gönder,

https://www.googleapis.com/oauth2/v3/token?client_id={{client_id}} &client_secret={{client_id}} &grant_type=refresh_token&refresh_token={{refresh_token}}

Yanıtta access_token içeren bir JSON alacaksınız.

{access_token: {{access_token}}}

Örnek:

 var access_token = '';
function getAccessToken(){
    $.post('https://www.googleapis.com/oauth2/v3/token', {
            client_id: {{client_id}},
            client_secret: {{client_secret}},
            grant_type: 'refresh_token',
            refresh_token: {{refresh_token}}
        }, function (data, status) {
            if (status === 'success') {
                access_token = data.access_token;
                // Do something eith the access_token
            } else {
                console.error(status);
            }
        });
}

Token geçerliliğini kontrol edin:

POST isteği gönder,

https://www.googleapis.com/oauth2/v1/tokeninfo?access_token={{access_token}}

Örnek:

function checkValidity() {
    $.post('https://www.googleapis.com/oauth2/v1/tokeninfo', {
            access_token:{{access_token}}
        }).done(function (data, status) {
            if (status === 'success') {
                console.debug(data.expires_in);
                var check = false;
                check = data.hasOwnProperty('expires_in');
                if (check) {
                    // Token is valid
                }
                if (!check) {
                    getAccessToken();
                }
            } else {
                console.debug(status);
            }

        })
        .fail(function (data) {
            console.error(data);
            getAccessToken();
        });
}
  1. Adım: Verileri Getirin

Yerleştirme API’si:

GA Embed API, birkaç dakika içinde GA gösterge tablonuzu kolayca oluşturup web sitenize yerleştirmenize olanak tanıyan bir JavaScript kitaplığıdır.

https://developers.google.com/analytics/devguides/reporting/embed/v1/getting-started adresine bakın

Sorgu Gezgini: Embed API Query Explorer’ı ziyaret edin ve yetkilendirin

https://ga-dev-tools.appspot.com/query-explorer/

Verileri almak istediğiniz görünümü seçin.

Gerekli metrikleri ve boyutları seçin.

Örnek:

Ülke Verilerini Al (Her ülkeden web siteme erişen kullanıcıların sayısını bilmek istiyorum)

Bu verileri almak için metrikleri “kullanıcılar” ve boyutları “ülke” olarak seçin

Sorguyu Çalıştır‘a tıklayın

Bir tabloda görüntülenen sorgunun analitik verilerini bulacaksınız.

API Sorgu URI’sini kopyalayın. Ve uri’ye access_token={{access_token}} ekleyin

Örnek:

https://www.googleapis.com/analytics/v3/data/ga?ids={{ids}}&start-date=2015-07-01&end-date=today&metrics=ga%3Ausers&dimensions=ga%3Acountry &access_token={{access_token}}

Verileri tarayıcınıza almak için URI’lere POST isteği gönderin.

Örnek:

function gaGetCountry() {
    $.get('https://www.googleapis.com/analytics/v3/data/ga?' +
        'ids={{ids}}' +
        'start-date=2015-07-01&' +
        'end-date=today&' +
        'metrics=ga%3Ausers&' +
        'dimensions=ga%3Acountry&' +
        'sort=ga%3Ausers&' +
        'filters=ga%3Ausers%3E10&' +
        'max-results=50' +
        '&access_token=' + {{access_token}},
        function (data, status) {
            if (status === 'success') {

                // Display the Data
                drawRegionsMap(data.rows);

            } else {
                console.debug(status);
            }

        });
}
  1. Adım: Verileri Görüntüle

Şimdi verileri topladık. Sonunda onları web sitemizde diaplay yapmak zorundayız.

“Sitenizde canlı verileri görüntüleyin”, Google Charts’ın başlığıdır. Ve yapacağımız şey bu.

https://developers.google.com/chart/ adresine bakın

Aşağıdaki örnek, id=‘countryChart’ ile div’de bir GeoChart çizecektir.

 //Draw country Chart
 function drawRegionsMap(data) {

        var head = data[0];
        head[0] = 'Country';
        head[1] = 'Users';
        for (var i = 1; i < data.length; i++) {
            var d = data[i];
            d[1] = Number(d[1]);
        }

        var chartData = google.visualization.arrayToDataTable(data);
        var options = {
            title: 'My Website is viewed from,',
            domain: '{{Country Code eg: IN for India}}',
            tooltip: {
                textStyle: {
                    color: 'navy'
                },
                showColorCode: true
            },
            legend: {
                textStyle: {
                    color: 'navy',
                    fontSize: 12
                }
            },
            colorAxis: {
                colors: ['#00FFFF', '#0000FF']
            }
        };
    
        var chart = new google.visualization.GeoChart(document.getElementById('countryChart'));
    
        chart.draw(chartData, options); 
}
