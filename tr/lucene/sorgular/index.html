Boole Sorgusu #

BooleanQuery, diğer sorguları birleştirmek için kullanılır.

Üç BooleanClause.Occur parametresi kullanılarak birleştirilebilirler:

  • BooleanClause.Occur.MUST - Alt sorgu eşleşmelidir.
  • BooleanClause.Occur.SHOULD - Alt sorgu eşleşmeyebilir, ancak eşleşirse daha yüksek puan alır. MUST yan tümcesi yoksa, en az bir SHOULD yan tümcesi eşleştirilmelidir.
  • BooleanClause.Occur.MUST_NOT - Alt sorgu belgeyle eşleşmemelidir.

Bu örnekte, bir belge “önemli” ise ancak * “yasak” değilse* eşleşir ve aynı zamanda “yararlı” ise daha yüksek bir puan alır.

Query importantQuery = new TermQuery(new Term("fieldname", "important"));
Query optionalQuery = new TermQuery(new Term("fieldname", "helpful"));
Query forbidQuery = new TermQuery(new Term("fieldname", "forbidden"));
BooleanQuery query = new BooleanQuery.Builder()
        .add(importantQuery, BooleanClause.Occur.MUST) 
        .add(optionalQuery, BooleanClause.Occur.SHOULD) 
        .add(forbidQuery, BooleanClause.Occur.MUST_NOT) 
        .build();

Alternatif olarak, eşleşmesi gereken minimum tümce sayısını da belirtebilirsiniz:

Query query1 = new TermQuery(new Term("fieldname", "one"));
Query query2 = new TermQuery(new Term("fieldname", "two"));
Query query3 = new TermQuery(new Term("fieldname", "three"));
BooleanQuery query = new BooleanQuery.Builder()
        .add(query1, BooleanClause.Occur.SHOULD) 
        .add(query2, BooleanClause.Occur.SHOULD) 
        .add(query3, BooleanClause.Occur.SHOULD)
        .setMinimumNumberShouldMatch(2)
        .build();

Gotcha: BooleanClause.Occur.MUST_NOT içeren tümceler diğer her şeyle eşleşmez*, sadece eşleşmeleri elerler. BooleanQuery’nizde en az bir “MUST” veya “SHOULD” yan tümcesi olmalıdır, yoksa hiçbir şeyle eşleşmeyecektir. Bu, örneğin, ÇALIŞMAZ:

//***This does NOT work!***
Query forbidQuery = new TermQuery(new Term("fieldname", "forbidden"));
BooleanQuery getEverythingElseQuery = new BooleanQuery.Builder()
        .add(forbidQuery, BooleanClause.Occur.MUST_NOT) 
        .build();

Sorguları artırma #

Diğer alt sorgulara göre puanını artırmak için bir sorgu yükseltilebilir. Bu, bir BoostQuery ile sarılarak yapılır.

Query lessRelevantQuery = new TermQuery(new Term("fieldname", "ipsum"));
//Five times as interesting
Query highlyRelevantQuery = new BoostQuery(
         new TermQuery(new Term("fieldname", "lorem")), 
         5.0f);
BooleanQuery query = new BooleanQuery.Builder()
        .add(lessRelevantQuery, BooleanClause.Occur.SHOULD) 
        .add(highlyRelevantQuery, BooleanClause.Occur.SHOULD) 
        .build();

Tümce Sorgulama #

PhraseQuery, bir dizi terim aramak için kullanılır. Aşağıdakiler “Merhaba Dünya” ifadesiyle eşleşir (“StandardAnalyzer” ile dizine eklendikten sonra)

Query query = new PhraseQuery.Builder()
        .add(new Term("text", "hello"))
        .add(new Term("text", "world"))
        .build();

PhraseQuery, “setSlop” ile bir maksimum [düzenleme mesafesi] (https://en.wikipedia.org/wiki/Damerau%E2%80%93Levenshtein_distance) ayarlayarak “slop"u veya bir sorgu içindeki ek terimleri de işleyebilir. Bu “Lorem Ipsum Sit Amet Dolor” ile eşleşecek:

Query query = new PhraseQuery.Builder()
        .add(new Term("text", "lorem"))
        .add(new Term("text", "amet"))
        .setSlop(2)
        .build();

Tam konum artışlarını da ayarlayabilirsiniz:

Query query = new PhraseQuery.Builder()
        .add(new Term("text", "lorem"), 0)
        .add(new Term("text", "sit"), 2)
        .add(new Term("text", "dolor"), 4)
        .build();

DisjunctionMaxQuery #

Bu, sorguları, alt sorgularının en iyi (yani en yüksek puanı alan) eşleşmesinin nihai puana katkıda bulunacağı şekilde birleştirir.

List<Query> disjuncts = new ArrayList<Query>();
disjuncts.add(new TermQuery(new Term("fieldname", "hello")));
disjuncts.add(new TermQuery(new Term("fieldname", "world")));
Query query = new DisjunctionMaxQuery(disjuncts, 0.0f);

‘DisjunctionMaxQuery’ yapıcısının ikinci argümanı, sıfır olmadığında, bağları koparmak için maksimum olmayan eşleşmelerin puana küçük bir katkı yapmasına izin veren bir eşitlik bozucu değerdir. Genellikle küçük olmalıdır (0,1 mertebesinde).
