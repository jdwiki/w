Temel SFML programı #

Her şey doğru ayarlanmışsa, aşağıdaki kod parçası “SFML çalışıyor!” başlıklı bir pencere gösterecektir. yeşil bir daire ile:

#include <SFML/Graphics.hpp>

int main()
{
    sf::RenderWindow window(sf::VideoMode(200, 200), "SFML works!");
    sf::CircleShape shape(100.f);
    shape.setFillColor(sf::Color::Green);

    while (window.isOpen())
    {
        sf::Event event;
        while (window.pollEvent(event))
        {
            if (event.type == sf::Event::Closed)
                window.close();
        }

        window.clear();
        window.draw(shape);
        window.display();
    }

    return 0;
}

Kurulum - Linux #

Linux’ta SFML kurulumunun farklı yaklaşımları vardır:

  1. Doğrudan dağıtımınızın paket deposundan kurun
  2. Kaynak kodunu alın, oluşturun ve kurun
  3. Önceden derlenmiş SDK’yı indirin ve dosyaları manuel olarak kopyalayın

Seçenek 1 tercih edilendir; Yüklemek istediğiniz SFML sürümü resmi depoda mevcutsa, paket yöneticinizi kullanarak yükleyin. Örneğin, Debian’da şunları yapardınız:

sudo apt-get install libsfml-dev

Seçenek 2 daha fazla çalışma gerektirir: geliştirme başlıkları dahil tüm SFML bağımlılıklarının kullanılabilir olduğundan emin olmanız, CMake’in kurulu olduğundan emin olmanız ve bazı komutları manuel olarak yürütmeniz gerekir. Bu, sisteminize uyarlanmış bir paketle sonuçlanacaktır. Bu şekilde gitmek istiyorsanız, SFML’yi kendiniz oluşturmaya yönelik özel bir eğitim var.

Son olarak, SFML resmi bir paket olarak mevcut değilse, seçenek 3 hızlı kurulum için iyi bir seçimdir. İndirme sayfasından SDK’yı indirin, paketinden çıkarın ve dosyaları tercih ettiğiniz konuma kopyalayın: kişisel klasörünüzde ayrı bir yol (/home/me/sfml gibi) veya standart bir yol (/usr/local gibi).

SFML’nin daha eski bir sürümünü zaten yüklediyseniz, yeni sürümle çakışmayacağından emin olun!

Kurulum - Windows #

SFML’yi Windows’a kurmanın en yaygın yolu resmi SDK dosyasını indirmektir.

Ardından arşivi paketinden çıkarabilir ve tercih ettiğiniz ortamda kullanabilirsiniz.

vcpkg #

Hala geliştirme aşamasında olmasına rağmen, Visual studio 2017 veya daha yenisini kullanıyorsanız, SFML’yi görsel stüdyo ile entegre olan vcpkg aracılığıyla da yükleyebilirsiniz, bu da kurulum sürecini büyük ölçüde basitleştirir:

`vcpkg sfml’yi kurun'

Insallation - macOS #

Öncelikle SFML SDK’yı indirmeniz gerekiyor. Ardından, SFML uygulamaları geliştirmeye başlamak için aşağıdaki öğeleri yüklemeniz gerekir:

Başlık dosyaları ve kitaplıkları #

SFML, dylib’ler veya çerçeveler olarak mevcuttur. Her ikisi de aynı sisteme aynı anda kurulabilmesine rağmen, yalnızca bir tür ikili dosya gereklidir. Çerçeveleri kullanmanızı öneririz.

çerçeveler #

  • Çerçevelerin içeriğini /Library/Frameworks’e kopyalayın.

dil #

  • lib içeriğini /usr/local/lib dizinine kopyalayın ve include içeriğini /usr/local/include dizinine kopyalayın.

SFML bağımlılıkları #

SFML, Mac OS X’te birkaç harici kitaplığa bağlıdır. Extlib’lerin içeriğini /Library/Frameworks’e kopyalayın.

Xcode şablonları #

Xcode kullanıyorsanız, şablonları yüklemeniz şiddetle önerilir. SFML dizinini şablonlardan /Library/Developer/Xcode/Templates’e kopyalayın (henüz yoksa klasörleri oluşturun).

SFML Penceresinde Merhaba Dünya #

Bir pencere açacak küçük bir program yazalım ve ekrana “Merhaba Dünya” yazalım.

#include <SFML\Graphics.hpp>
#include <cassert>

int main() {
    sf::RenderWindow sfmlWin(sf::VideoMode(600, 360), "Hello World SFML Window");
    sf::Font font;
    //You need to pass the font file location
    if (!font.loadFromFile(/*
                           Put the filename that identify the font file you want to load*/"myfont.ttf")) {
        return -1;
    }
    sf::Text message("Hello, World !", font);

    while (sfmlWin.isOpen()) {

        sf::Event e;
        while (sfmlWin.pollEvent(e)) {

            switch (e.type) {
            case sf::Event::EventType::Closed:
                sfmlWin.close();
                break;
            }
        }

        sfmlWin.clear();
        sfmlWin.draw(message);
        sfmlWin.display();
    }
    return 0;
}

Orada ne yaptığımızı anlatalım.

İlk önce bir sf::Font nesnesi yarattık. Metni görüntülemek için kullanacağımız yazı tipi verilerini saklamak için bu nesneye ihtiyacımız var. Bundan sonra, fontu belleğe yüklemek için kullanılan loadFromFile yöntemini çağırdık. SFML’nin sistem yazı tiplerinizi bilmediğini unutmamalıyız, bu nedenle bir yazı tipi adı değil bir dosya adı sağlamanız gerekir.

Daha sonra bir sf::Text nesnesi oluşturduk. 3 parametreli bir kurucu çağırıyoruz:

  • Görüntülemek istediğiniz dize
  • Nesnenin kullanacağı yazı tipi
  • Buraya geçmediğimiz piksel cinsinden karakter boyutu, bu nedenle il varsayılan değere ayarlanacaktır: 30

sf::Text nesnesi hazır olduğundan, daha önce oluşturduğumuz sfmlWin oluşturma penceresi nesnesi üzerinde draw yöntemini çağırarak onu ana sfml döngüsünde çizmemiz yeterlidir.

Kurulum veya Kurulum #

  • İlk olarak, resmi sayfadan SFML’nin bir kopyasını indirin.
  • Bilgisayarınızda kolayca erişebileceğiniz herhangi bir yere kaydedin.
  • Codeblocks’u açın.
  • Proje->Yapı Seçenekleri->LinkerSettings sekmesine gidin.
  • Ekle düğmesine tıklayın ve SFML’nin bin klasörüne gidin ve orada bulunan tüm dosyaları seçin.
  • Şimdi Arama Dizinleri sekmesine gidin ve SFML’nin ‘include’ klasörünü ekleyin.
  • Aynı sekmede, Bağlayıcı Ayarları alt sekmesine tıklayın ve ‘bin’ klasörünü ekleyin.
