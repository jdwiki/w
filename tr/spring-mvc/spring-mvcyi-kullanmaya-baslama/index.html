İlk Bahar-MVC Projesi #

Dinamik Web projesi oluşturun, aşağıdaki bilgileri aşağıda belirtildiği gibi sağlayın

  1. Proje adı : DemoSpringMVCProject
  2. Hedef çalışma zamanı: Apache Tomcat v7.0 sunucusu olarak ayarlayın

Bitir’e tıklayın, başarılı bir şekilde dinamik web projesi oluşturduk.

Şimdi Spring-MVC çerçevesini kurmamız gerekiyor :

  1. ’ WebContent\WEB-INF\ ‘ klasörü altında web.xml oluşturun
<?xml version="1.0" encoding="UTF-8"?>
<web-app xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xmlns="http://java.sun.com/xml/ns/javaee" xmlns:web="http://java.sun.com/xml/ns/javaee/web-app_2_5.xsd" xsi:schemaLocation="http://java.sun.com/xml/ns/javaee http://java.sun.com/xml/ns/javaee/web-app_2_5.xsd" id="WebApp_ID" version="2.5">
<display-name>Demo9</display-name>

<servlet>
  <servlet-name>spring</servlet-name>
  <servlet-class>org.springframework.web.servlet.DispatcherServlet</servlet-class>
</servlet>

<servlet-mapping>
  <servlet-name>demo</servlet-name>
  <url-pattern>/</url-pattern>
</servlet-mapping>

</web-app>
  • DispatcherServlet sınıfının gelen isteği engellediği ve isteği hangi denetleyicinin işleyeceğini belirlediği yer.
  • servlet.xml oluştururken servlet-name ’ demo ’ kullanacağız
  1. ’ WebContent\WEB-INF\ ‘ klasörü altında demo-servlet.xml oluşturun
<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
xmlns:context="http://www.springframework.org/schema/context"
xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd
    http://www.springframework.org/schema/context http://www.springframework.org/schema/context/spring-context.xsd">

<context:component-scan base-package="com"></context:component-scan>

<bean class="org.springframework.web.servlet.view.InternalResourceViewResolver">
 <property name="prefix" value="/WEB-INF/jsp/"></property>
 <property name="suffix" value=".jsp"></property>
</bean>

</beans>

  • bağlam: bileşen taraması, ‘com’ paketi altında tanımlanan tüm denetleyicileri taramak için kullanılır.

  • ViewResolver Arayüzü Mantıksal ve gerçek görünümler arasındaki eşlemeyi yönetmek için kullanın. Görünümleri eşlemek için önceden tanımlanmış görünüm çözümleyici uygulaması mevcuttur. Ör: InternalResourceViewResolver, VelocityViewResolver.

  • Setter özelliğinden başka bir şey olmayan prefix tanımladığımız tüm jsp sayfalarını aramak için değeri ’/WEB-INF/jsp/’( klasör yolu ) olarak ayarlanır. Getter özelliğinden başka bir şey olmayan Sonek, değeri ’.jsp’ (.jsp uzantılı arama dosyası) olarak ayarlanmıştır.

Kontrolör ve jsp sayfalarına geçelim :

  1. src klasörü altında bir com.demo.controller paketi oluşturun.

  2. com.demo.controller paketi altında bir LoginController sınıfı oluşturun

    package com.demo.controller;

    import javax.servlet.http.HttpServletRequest; import org.springframework.stereotype.Controller; import org.springframework.web.bind.annotation.RequestMapping;

    @Controller public class LoginController {

    @RequestMapping("/") public String startPage(){ return “login”; }

    @RequestMapping("/signin") public String handleRequest(HttpServletRequest request){ String name = request.getParameter(“name”); String pass = request.getParameter(“password”); if(name.equals(pass)) { return “welcome”; }else{ return “login”; }

    }

    }

  3. ’ WebContent\WEB-INF\jsp\ ‘ altında bir login.jsp ve Welcome.jsp sayfası oluşturun

login.jsp
