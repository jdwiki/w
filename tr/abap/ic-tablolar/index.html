Dahili tablo türleri #

DATA: <TABLE NAME> TYPE <SORTED|STANDARD|HASHED> TABLE OF <TYPE NAME> 
      WITH <UNIQUE|NON-UNIQUE> KEY <FIELDS FOR KEY>.

Standart Tablo

Bu tablo, doğrusal bir şekilde depolanan tüm girdileri içerir ve kayıtlara doğrusal bir şekilde erişilir. Büyük tablo boyutları için tablo erişimi yavaş olabilir.

Sıralanmış Tablo

EŞSİZ ANAHTAR İLE|EŞSİZ ANAHTAR eklenmesini gerektirir. İkili arama gerçekleştirilmesi nedeniyle arama hızlıdır. Girişler, sıralama düzenini bozabileceğinden bu tabloya eklenemez, bu nedenle her zaman INSERT anahtar sözcüğü kullanılarak eklenirler.

Karma Tablo

EŞSİZ ANAHTAR İLE|EŞSİZ ANAHTAR eklenmesini gerektirir. Anahtar/değer çiftlerini korumak için özel bir karma algoritma kullanır. Teorik olarak aramalar “STANDART” tablo kadar yavaş olabilir, ancak pratikte, tablonun boyutundan bağımsız olarak sabit bir süre alan “SORTED” bir tablodan daha hızlıdır.

ABAP Dahili Tablolarının Beyanı #

Yerel Tip Tanımına Dayalı Dahili Tablo Beyanı #

" Declaration of type
TYPES: BEGIN OF ty_flightb,
         id        TYPE fl_id,
         dat       TYPE fl_date,
         seatno    TYPE fl_seatno,
         firstname TYPE fl_fname, 
         lastname  TYPE fl_lname,
         fl_smoke  TYPE fl_smoker,
         classf    TYPE fl_class,
         classb    TYPE fl_class,
         classe    TYPE fl_class,
         meal      TYPE fl_meal,
         service   TYPE fl_service,
         discout   TYPE fl_discnt,
       END OF lty_flightb.

" Declaration of internal table
DATA t_flightb TYPE STANDARD TABLE OF ty_flightb.

Veritabanı Tablosuna Dayalı Beyanname #

DATA t_flightb TYPE STANDARD TABLE OF flightb.

Satır İçi Tablo Bildirimi #

ABAP sürümü > 7.4 gerektirir

TYPES t_itab TYPE STANDARD TABLE OF i WITH EMPTY KEY.

DATA(t_inline) = VALUE t_itab( ( 1 ) ( 2 ) ( 3 ) ).

Başlık Satırları Bildirimli Dahili Tablo #

ABAP’de başlık satırlı tablolar ve başlık satırsız tablolar vardır. Başlık satırlı tablolar daha eski bir kavramdır ve yeni geliştirmede kullanılmamalıdır.

Dahili Tablo: Başlık satırlı / başlıksız Standart Tablo

Bu kod, mevcut ‘compc_str’ yapısıyla ‘i_compc_all’ tablosunu bildirir.

DATA: i_compc_all TYPE STANDARD TABLE OF compc_str WITH HEADER LINE.
DATA: i_compc_all TYPE STANDARD TABLE OF compc_str.

Dahili Tablo: Başlık satırı olan/olmayan Karma Tablo

DATA: i_map_rules_c TYPE HASHED TABLE OF /bic/ansdomm0100 WITH HEADER LINE
DATA: i_map_rules_c TYPE HASHED TABLE OF /bic/ansdomm0100

Başlıksız tablolar için çalışma alanı bildirimi

Bir çalışma alanı (genellikle wa olarak kısaltılır) tabloyla tamamen aynı yapıya sahiptir, ancak yalnızca bir satır içerebilir (WA, yalnızca tek boyutlu bir tablonun yapısıdır).

DATA: i_compc_all_line LIKE LINE OF i_compc_all.

Okuma, Yazma ve Dahili Tablolara Ekleme #

Başlık satırı ile dahili tablolara okuyun, yazın ve ekleyin:

" Read from table with header (using a loop):
LOOP AT i_compc_all.              " Loop over table i_compc_all and assign header line
  CASE i_compc_all-ftype.         " Read cell ftype from header line from table i_compc_all 
    WHEN 'B'.                     " Bill-to customer number transformation
      i_compc_bil = i_compc_all.  " Assign header line of table i_compc_bil with content of header line i_compc_all
      APPEND i_compc_bil.         " Insert header line of table i_compc_bil into table i_compc_bil
    " ... more WHENs
  ENDCASE.
ENDLOOP.

Hatırlatma: Nesne yönelimli bağlamlarda başlık satırlı dahili tablolar yasaktır. Başlık satırları olmayan dahili tabloların kullanılması her zaman önerilir.

Başlık satırı olmadan dahili tablolara okuyun, yazın ve ekleyin:

" Loop over table i_compc_all and assign current line to structure i_compc_all_line
LOOP AT i_compc_all INTO i_compc_all_line.      
  CASE i_compc_all_line-ftype.                " Read column ftype from current line (which as assigned into i_compc_all_line)
    WHEN 'B'.                                 " Bill-to customer number transformation
      i_compc_bil_line = i_compc_all_line.    " Copy structure
      APPEND i_compc_bil_line TO i_compc_bil. " Append structure to table
    " more WHENs ...
  ENDCASE.
ENDLOOP.


" Insert into table with Header:
INSERT TABLE i_sap_knb1.                      " insert into TABLE WITH HEADER: insert table header into it's content
insert i_sap_knb1_line into table i_sap_knb1. " insert into HASHED TABLE: insert structure i_sap_knb1_line into hashed table i_sap_knb1
APPEND p_t_errorlog_line to p_t_errorlog.     " insert into STANDARD TABLE: insert structure / wa p_t_errorlog_line into table p_t_errorlog_line
