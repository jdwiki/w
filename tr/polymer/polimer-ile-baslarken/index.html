Temel Eleman Yapısı #

Aşağıdaki çok temel öğe olan “my-element” öğesini “src/my-element.html” olarak kaydettik

<link rel="import" href="bower_components/polymer/polymer.html">

<dom-module id="my-element">

  <template>
    <style>
      /* local styles go here */
      :host {
        display: block;
      }
    </style>
    <!-- local DOM goes here -->
    <content></content>
  </template>

  <script>
    Polymer({
      /* this is the element's prototype */
      is: 'my-element'
    });
  </script>

</dom-module>
  • “”, bir HTML içe aktarma kullanan Polimer kitaplığını içerir.
  • “”, öğenin yerel DOM sarmalayıcısıdır (bu durumda, “my-element”).
  • “<şablon>”, gerçek yerel DOM tanımıdır.
