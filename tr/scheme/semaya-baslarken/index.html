TAVUK Şemasını Yükleme #

CHICKEN, “yumurta” adı verilen kendi genişletme modülü sistemine sahip bir Scheme yorumlayıcısı ve derleyicisidir. İlk önce Şemayı C’ye derleyerek Şemayı yerel koda derleyebilir.

Kurulum #

Debian veya Ubuntu veya diğer türetilmiş dağıtımlar: #

sudo apt-get install chicken-bin

Fedora / RHEL / CentOS: #

sudo yum install chicken-bin

Arch Linux: #

sudo pacman -S chicken

Gentoo: #

sudo emerge -av dev-scheme/chicken

Homebrew ile OS X: #

brew install chicken

OpenBSD #

doas pkg_add -vi chicken

Microsoft Windows #

  • [MSYS2]‘yi kurun(https://msys2.github.io)

  • MSYS2 MinGW-w64 Shell’i çalıştırın

  • Çalıştırarak bazı önkoşulları yükleyin:

    pacman -S mingw-w64-cross-toolchain base-devel mingw-w64-x86_64-gcc winpty wget

  • Aşağıdakileri yazarak en son sürüm tarball dosyasını indirin:

    wget https://code.call-cc.org/releases/current/chicken.tar.gz

  • tar xvf Chicken.tar.gz komutunu çalıştırarak tarball’ı çıkarın

  • Ayıklanan dizini girin, örneğin cd Chicken-4.11.0 yazarak

  • make PLATFORM=mingw-msys install komutunu çalıştırın

‘csi’yi çalıştırmakta sorun yaşıyorsanız, bunun yerine ‘winpty csi’yi çalıştırmayı deneyin

TAVUK Kullanımı #

CHICKEN Scheme REPL’i kullanmak için komut satırına ‘csi’ yazın.

CHICKEN kullanarak bir Scheme programı derlemek için, geçerli dizinde “program” adında bir yürütülebilir dosya oluşturacak olan “csc program.scm” dosyasını çalıştırın.

Modülleri yükleme #

Chicken Scheme’in yumurta dizininde göz atılabilecek birçok modülü vardır. Yumurtalar, indirilecek ve daha sonra tavuk şeması tarafından derlenecek şema modülleridir. Bazı durumlarda, normal paket yöneticinizi kullanarak harici bağımlılıklar kurmanız gerekebilir.

Seçilen yumurtaları şu komutla kurarsınız:

sudo chicken-install [name of egg]

REPL’den yararlanma #

‘csi’de satır düzenlemenin beklediğiniz gibi davranmasını sağlamak için REPL’nize ‘readline’ desteği eklemek isteyebilirsiniz.

Bunu yapmak için, “sudo Chicken-install readline” komutunu çalıştırın ve ardından aşağıdaki içeriğe sahip “~/.csirc” adlı bir dosya oluşturun:

(use readline)
(current-input-port (make-readline-port))
(install-history-file #f "/.csi.history")

mit şemasını yükleme #

MIT/GNU Scheme‘in nasıl kurulacağına ilişkin örnekler aşağıdadır:

Debian/Ubuntu kurulumu:

sudo apt-get install mit-scheme

Manuel kurulum:

Unix ikili dosyasını doğrudan GNU Projesi‘den indirin, ardından resmi web sayfasından talimatları izleyin:

# Unpack the tar file
tar xzf mit-scheme.tar.gz

# move into the directory
cd mit-scheme/src

# configure the software
./configure

Varsayılan olarak, yazılım /usr/local dizinine, bin ve lib alt dizinlerine kurulacaktır. Başka bir yere yüklenmesini istiyorsanız, örneğin /opt/mit-scheme, ./configure --prefix=/opt/mit-schemede olduğu gibi --prefix seçeneğini yapılandırma komut dosyasına iletin.

config betiği, bu tür betikler için tüm normal argümanları kabul eder ve ayrıca MIT/GNU Scheme’e özgü bazı argümanları da kabul eder. Tüm olası argümanları ve anlamlarını görmek için ./configure --help komutunu çalıştırın.

# build
make compile-microcode

# compile
make install # may require super-user permissions (Depending on configuration)

Windows 7:

Kendi kendine yüklenen yürütülebilir resmi web sitesinde bulunabilir.

MIT/GNU Scheme, kendi kendine yüklenen bir yürütülebilir dosya olarak dağıtılır. Yazılımın kurulumu basittir. İndirilen dosyayı çalıştırmanız ve yükleyicinin sorularını yanıtlamanız yeterlidir. Yükleyici, MIT/GNU Scheme’in kurulacağı dizini ve kısayolların yerleştirileceği klasörün adını seçmenize izin verecektir.
