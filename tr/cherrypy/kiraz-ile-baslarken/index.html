CherryPy’de merhaba dünya #

Virtualenv’iniz varsa ve CherryPy zaten kuruluysa, bir hello.py dosyası oluşturun:

#!/usr/bin/env python
# -*- coding: UTF-8 -*-

import cherrypy

class HelloWorld(object):
    @cherrypy.expose
    def index(self):
        return 'Hello World!'

    @cherrypy.expose
    def greet(self, name):
        return 'Hello {}!'.format(name)

cherrypy.quickstart(HelloWorld())

Ardından dosyayı yürütün: $ merhaba.py veya $ python merhaba.py. Şuna benzer bir çıktı görmelisiniz:

Kurulum Talimatları #

Ön koşullar #

  • Bu talimatlar, bash veya Git-bash Windows’lu herhangi bir Linux, Unix, Mac türüne sahip olduğunuzu varsayar.
  • Windows: Git-bash for Windows‘u indirip kurun, ardından komut satırından ‘bash’ komutunu çalıştırın.
  • Bash dışındaki diğer kabuklar da iyidir, aşağıdaki ‘activate’ komutunu ‘activate.csh’ veya Google ile değiştirin: “virtualenv Activate your-shell-name”.

Başlamadan önce Python, virtualenv ve pip’in kurulu olduğunu kontrol edin:

  • $ piton --sürüm
  • $ virtualenv --version
  • $ pip --version

Düzenlemek #

Web/uygulamanız ile bir dizin oluşturun, ortam oluşturun ve CherryPy paketini kurun.

  • $ mkdir /develop/myapp/
  • $ cd /develop/myapp/
  • $ sanalenv venv
  • $ kaynak venv/bin/etkinleştir
    • On Windows in Git-bash: $ source venv/Scripts/activate
  • `(venv) $ pip Cherrypy’yi kurun’
  • (venv) $python
Python 3.5.2 ...
>>> import cherrypy
>>> cherrypy
<module 'cherrypy' from '... venv/site-packages/cherrypy/__init__.py'>

Tebrikler! Artık ilk CherryPy uygulamanız için hazırsınız.

CherryPy ile dosya yükleme #

Bu örnek üç bölümden oluşmaktadır:

  • server.py - Bir dosyayı alıp kaydedebilen CherryPy uygulaması.
  • webpage.html - Bir web sayfasından server.py’ye dosya yükleme örneği.
  • cli.py - Komut satırı aracından server.py’ye dosya yükleme örneği.
  • Bonus - upload.txt - yükleyeceğiniz dosya.

sunucu.py #

#!/usr/bin/env python
# -*- coding: UTF-8 -*-

import os
import cherrypy

config = {
    'global' : {
        'server.socket_host' : '127.0.0.1',
        'server.socket_port' : 8080
    }
}


class App:

    @cherrypy.expose
    def upload(self, ufile):
        # Either save the file to the directory where server.py is
        # or save the file to a given path:
        # upload_path = '/path/to/project/data/'
        upload_path = os.path.dirname(__file__)

        # Save the file to a predefined filename
        # or use the filename sent by the client:
        # upload_filename = ufile.filename
        upload_filename = 'saved.txt'

        upload_file = os.path.normpath(
            os.path.join(upload_path, upload_filename))
        size = 0
        with open(upload_file, 'wb') as out:
            while True:
                data = ufile.file.read(8192)
                if not data:
                    break
                out.write(data)
                size += len(data)
        out = '''
File received.
Filename: {}
Length: {}
Mime-type: {}
''' .format(ufile.filename, size, ufile.content_type, data)
        return out


if __name__ == '__main__':
    cherrypy.quickstart(App(), '/', config)

#cli.py #

Bu örnek, Python request paketini gerektirir, ancak dosya sunucuya düz Python’da gönderilebilir.

#!/usr/bin/env python
# -*- coding: UTF-8 -*-

import requests

url = 'http://127.0.0.1:8080/upload'
files = {'ufile': open('file.txt', 'rb')}

r = requests.post(url, files=files)

print(r)
print(r.text)

yükleme.txt #

Hello! This file was uploaded to CherryPy.

Tarayıcıdan yükle #

  • $ server.py dosyasını çalıştırın
  • Web tarayıcınızda webpage.html dosyasını açın.
  • Sürücünüzden dosya seçip gönderdikten sonra saved.txt olarak kaydedilecektir.

Komut satırından yükle #

  • Bir konsol açın ve $ server.py dosyasını çalıştırın
  • Başka bir konsol açın ve $cli.py dosyasını çalıştırın
    • Note: Test file upload.txt should be in the same directory with cli.py
  • ‘upload.txt’ dosyası yüklenip ‘saved.txt’ olarak kaydedilmelidir.
