İpuçları:

  • Yükseltmeden önce her zaman /usr/ports/UPDATING dosyasını kontrol etmeyi unutmayın. Kullandığınız programlarda veya yapılandırmalarında mevcut kurulumunuzu bozacak bazı önemli değişiklikler olabilir.

Yazılım oluşturma ve yükleme #

Yazılımınızı port ağacında bulduysanız, şimdi onu oluşturma zamanı.

Manuel yapılandırma ile basit kurulum ve kurulum #

cd /usr/ports/www/apache24
make
make install

Otomatik yapılandırma ile basit kurulum ve kurulum #

cd /usr/ports/www/apache24
make BATCH=yes
make install

Bağlantı Noktaları ağacını alma #

Bağlantı noktası yakalama #

portsnap fetch
portsnap extract

port ağacını portsnap ile güncelleme #

portsnap update

günlük güncellemeler için cron işi planlayın #

0 3 * * * root /usr/sbin/portsnap cron

SVN #

kafa #

cd /usr/ports
svnlite checkout https://svnweb.freebsd.org/ports/head .

üç ayda bir #

FreeBSD Limanlar ekibi, her 3 ayda bir liman ağacını dondurur. Bu bağlantı noktası ağacını almak için bağlantı noktası dallarını kullanabilirsiniz:

cd /usr/ports
svnlite checkout https://svnweb.freebsd.org/ports/branches/2016Q4 .

Tarball (http veya ftp) #

cd /usr/ports
fetch http://ftp.freebsd.org/pub/FreeBSD/releases/amd64/11.0-RELEASE/ports.txz
tar xJvf ports.txz

Git #

git clone https://github.com/freebsd/freebsd-ports

Yazılım arama #

anahtar kelime araması #

cd /usr/ports
make search key=apache

isim arama #

cd /usr/ports
make search name=apache24

Fresport kullanma #

Resmi FreeBSD bağlantı noktaları web sitesi (http://freshports.org/), bağlantı noktalarını ve bununla ilgili tüm bilgileri bulmanız için size güzel bir yol sunar.

Yazılım kaynaklarını yapılandırma #

Bağlantı noktalarından özel yapılandırma istiyorsanız, bunu make config oluşturmadan önce yapılandırabilirsiniz. Tüm bağlantı noktaları yapılandırması `/var/db/ports/${CATEGORY_NAME}/options’ içinde makefile olarak saklanır.

www/apache24 yapılandırması #

cd /usr/ports/www/apache24
make config
make
make install

Bu yapılandırma /var/db/ports/www_apache24/options klasörüne kaydedilecektir.

Ambalajlama #

Manuel paketleme #

Port bazında kendi paketinizi oluşturabilirsiniz.

cd /usr/ports/www/apache24
make package BATCH=yes

Bu komut paketinizi /usr/ports/packages/All dizininde saklayacaktır.

’toz dergisi’ kullanma #

poudriere şu anda FreeBSD için resmi paket oluşturucudur.

Toz magazini yükleme #

pkg install poudriere
# or
cd /usr/ports/ports-mgmt/poudriere
make
make install

Toz magazinini yapılandırma #

‘poudriere’ yapılandırması ‘/usr/local/etc/poudriere.conf’ ve ‘/usr/local/etc/poudriere.d’ içinde saklanır

Barut fıçısı hapishanesini konuşlandırma #

poudriere jail -c -j myjail

poudriere hapishanesi güncelleniyor #

poudriere jail -u -j myjail

Barut fıçısı bağlantı noktası ağacının konuşlandırılması #

poudriere ports -c -p myports

poudriere bağlantı noktası ağacının güncellenmesi #

poudriere ports -u -p myports

Toplu yapı #

poudriere bulk -j myjail -p myports www/apache24
