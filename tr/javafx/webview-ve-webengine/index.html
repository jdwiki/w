WebView ve WebEngine

  • Sayfa yükleniyor
  • Bir Web Görünümünün sayfa geçmişini alın
  • görüntülenen web sayfasından Java uygulamaları günlüğüne Javascript uyarıları gönder.
  • Web sayfasında Java uygulaması ve Javascript arasındaki iletişim

Bu sayfada

  • Sayfa yükleniyor
  • Bir Web Görünümünün sayfa geçmişini alın
  • görüntülenen web sayfasından Java uygulamaları günlüğüne Javascript uyarıları gönder.
  • Web sayfasında Java uygulaması ve Javascript arasındaki iletişim

‘WebView’, JavaFX bileşen ağacına entegre edilmiş JavaFX Düğümü’dür. Bir “WebEngine"i yönetir ve içeriğini görüntüler.

‘WebEngine’, temelde tüm işi yapan temeldeki Tarayıcı Motorudur.

Sayfa yükleniyor #

WebView wv = new WebView();
WebEngine we = wv.getEngine();
we.load("https://stackoverflow.com");

WebView, WebEngine etrafındaki UI kabuğudur docs.oracle.com/javase/8/javafx/api/javafx/scene/web/WebEngine.html). Bir sayfayla UI dışı etkileşim için neredeyse tüm kontroller “WebEngine” sınıfı aracılığıyla yapılır.

Bir Web Görünümünün sayfa geçmişini alın #

WebHistory history = webView.getEngine().getHistory();

Tarih, temelde girişlerin bir listesidir. Her giriş, ziyaret edilen bir sayfayı temsil eder ve URL, başlık ve sayfanın en son ziyaret edildiği tarih gibi ilgili sayfa bilgilerine erişim sağlar.

Liste, getEntries() yöntemi kullanılarak elde edilebilir. “WebEngine” web’de gezinirken geçmiş ve ilgili giriş listesi değişir. Liste, tarayıcı eylemlerine bağlı olarak genişleyebilir veya küçülebilir. Bu değişiklikler, listenin sunduğu ObservableList API tarafından dinlenebilir.

Halihazırda ziyaret edilen sayfayla ilişkili geçmiş girişinin dizini, ‘currentIndexProperty()’ ile temsil edilir. Geçerli dizin, ‘go(int)’ yöntemini kullanarak geçmişteki herhangi bir girişe gitmek için kullanılabilir. maxSizeProperty(), geçmiş listesinin boyutu olan maksimum geçmiş boyutunu ayarlar.

Aşağıda, Web Geçmişi Öğeleri Listesinin nasıl elde edileceğine ve işleneceğine ilişkin bir örnek verilmiştir.

Geçmiş öğelerini depolamak için bir “ComboBox” (comboBox) kullanılır. “WebHistory” üzerinde bir “ListChangeListener” kullanarak “ComboBox”, mevcut “WebHistory” ile güncellenir. ‘ComboBox’ üzerinde, seçilen sayfaya yönlendiren bir ‘EventHandler’ bulunur.

final WebHistory history = webEngine.getHistory();

comboBox.setItems(history.getEntries());
comboBox.setPrefWidth(60);
comboBox.setOnAction(new EventHandler<ActionEvent>() {
    @Override
    public void handle(ActionEvent ev) {
        int offset =
                comboBox.getSelectionModel().getSelectedIndex()
                - history.getCurrentIndex();
        history.go(offset);
    }
});

history.currentIndexProperty().addListener(new ChangeListener<Number>() {

    @Override
    public void changed(ObservableValue<? extends Number> observable, Number oldValue, Number newValue) {
        // update currently selected combobox item
        comboBox.getSelectionModel().select(newValue.intValue());
    }
});

// set converter for value shown in the combobox:
//   display the urls
comboBox.setConverter(new StringConverter<WebHistory.Entry>() {

    @Override
    public String toString(WebHistory.Entry object) {
        return object == null ? null : object.getUrl();
    }

    @Override
    public WebHistory.Entry fromString(String string) {
        throw new UnsupportedOperationException();
    }
});

görüntülenen web sayfasından Java uygulamaları günlüğüne Javascript uyarıları gönder. #

private final Logger logger = Logger.getLogger(getClass().getCanonicalName());

WebView webView = new WebView();
webEngine = webView.getEngine();

webEngine.setOnAlert(event -> logger.warning(() -> "JS alert: " + event.getData()));

Web sayfasında Java uygulaması ve Javascript arasındaki iletişim #

Kendi özel web sayfanızı görüntülemek için bir WebView kullanırken ve bu web sayfası Javascript içeriyorsa, Java programı ile web sayfasındaki Javascript arasında iki yönlü bir iletişim kurmak gerekli olabilir.

Bu örnek, böyle bir iletişimin nasıl kurulacağını gösterir.

Web sayfasında bir giriş alanı ve bir düğme görüntülenecektir. Düğmeye tıklandığında, giriş alanındaki değer, onu işleyen Java uygulamasına gönderilir. Bir sonuç işlendikten sonra, sonucu web sayfasında görüntüleyen Javascript’e gönderilir.

Temel ilke, Javascript’ten Java’ya iletişim için Java’da web sayfasına ayarlanmış bir nesnenin oluşturulmasıdır. Ve diğer yön için, Javascript’te bir nesne oluşturulur ve web sayfasından çıkarılır.

Aşağıdaki kod Java bölümünü gösterir, hepsini tek bir dosyada tuttum:

package com.sothawo.test;

import javafx.application.Application;
import javafx.concurrent.Worker;
import javafx.scene.Scene;
import javafx.scene.web.WebEngine;
import javafx.scene.web.WebView;
import javafx.stage.Stage;
import netscape.javascript.JSObject;

import java.io.File;
import java.net.URL;

/**
 * @author P.J. Meisch (pj.meisch@sothawo.com).
 */
public class WebViewApplication extends Application {

    /** for communication to the Javascript engine. */
    private JSObject javascriptConnector;

    /** for communication from the Javascript engine. */
    private JavaConnector javaConnector = new JavaConnector();;

    @Override
    public void start(Stage primaryStage) throws Exception {
        URL url = new File("./js-sample.html").toURI().toURL();

        WebView webView = new WebView();
        final WebEngine webEngine = webView.getEngine();

        // set up the listener
        webEngine.getLoadWorker().stateProperty().addListener((observable, oldValue, newValue) -> {
            if (Worker.State.SUCCEEDED == newValue) {
                // set an interface object named 'javaConnector' in the web engine's page
                JSObject window = (JSObject) webEngine.executeScript("window");
                window.setMember("javaConnector", javaConnector);

                // get the Javascript connector object. 
                javascriptConnector = (JSObject) webEngine.executeScript("getJsConnector()");
            }
        });

        Scene scene = new Scene(webView, 300, 150);
        primaryStage.setScene(scene);
        primaryStage.show();

        // now load the page
        webEngine.load(url.toString());
    }

    public class JavaConnector {
        /**
         * called when the JS side wants a String to be converted.
         *
         * @param value
         *         the String to convert
         */
        public void toLowerCase(String value) {
            if (null != value) {
                javascriptConnector.call("showResult", value.toLowerCase());
            }
        }
    }
}

Sayfa yüklendiğinde, bu çağrılarla web sayfasına bir “JavaConnector” nesnesi (iç sınıf tarafından tanımlanan ve bir alan olarak oluşturulan) ayarlanır:

JSObject window = (JSObject) webEngine.executeScript("window");
window.setMember("javaConnector", javaConnector);

‘javascriptConnector’ nesnesi web sayfasından şu şekilde alınır:

javascriptConnector = (JSObject) webEngine.executeScript("getJsConnector()");

“JavaConnector"dan “toLowerCase(String)” yöntemi çağrıldığında, iletilen değer dönüştürülür ve ardından “javascriptConnector” nesnesi aracılığıyla geri gönderilir.

Ve bu html ve javascript kodudur:

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>Sample</title>
    </head>
    <body>
        <main>
    
            <div><input id="input" type="text"></div>
            <button onclick="sendToJava();">to lower case</button>
            <div id="result"></div>
    
        </main>
    
        <script type="text/javascript">
            function sendToJava () {
                var s = document.getElementById('input').value;
                javaConnector.toLowerCase(s);
            };
    
            var jsConnector = {
                showResult: function (result) {
                    document.getElementById('result').innerHTML = result;
                }
            };
    
            function getJsConnector() {
                return jsConnector;
            };
        </script>
    </body>
</html>

“sendToJava” işlevi, Java kodu tarafından ayarlanan “JavaConnector” yöntemini çağırır:

function sendToJava () {
    var s = document.getElementById('input').value;
    javaConnector.toLowerCase(s);
};

ve Java kodu tarafından “javascriptConnector"u almak için çağrılan işlev sadece “jsConnector” nesnesini döndürür:

var jsConnector = {
    showResult: function (result) {
        document.getElementById('result').innerHTML = result;
    }
};
    
function getJsConnector() {
    return jsConnector;
};

Java ve Javascript arasındaki çağrıların argüman türü, Strings ile sınırlı değildir. Olası türler ve dönüştürme hakkında daha fazla bilgi JSObject API belgesi içinde bulunur.
