Bu sayfada

  • FXML kullanan temel JavaFX projesi

JavaFX Scene Builder, kullanıcıların JavaFX uygulaması kullanıcı arabirimlerini kodlamadan hızlı bir şekilde tasarlamasını sağlayan bir görsel düzen aracıdır. FXML dosyaları oluşturmak için kullanılır.

JavaFX Scene Builder, kullanıcıların JavaFX uygulaması kullanıcı arabirimlerini kodlamadan hızlı bir şekilde tasarlamasını sağlayan bir görsel düzen aracıdır. Kullanıcılar, UI bileşenlerini bir çalışma alanına sürükleyip bırakabilir, özelliklerini değiştirebilir, stil sayfaları uygulayabilir ve oluşturdukları düzen için FXML kodu arka planda otomatik olarak oluşturulur. Sonuç, daha sonra kullanıcı arayüzünü uygulamanın mantığına bağlayarak bir Java projesiyle birleştirilebilen bir FXML dosyasıdır.

Model Görünümü Denetleyicisi (MVC) perspektifinden:

  • Kullanıcı arayüzünün açıklamasını içeren FXML dosyası görünümdür.
  • Denetleyici, isteğe bağlı olarak FXML dosyası için denetleyici olarak bildirilen Initializable sınıfını uygulayan bir Java sınıfıdır.
  • Model, Java tarafında tanımlanmış, denetleyici aracılığıyla görünüme bağlanabilen etki alanı nesnelerinden oluşur.

Sahne Oluşturucu Kurulumu

  1. Platformunuz için yükleyiciyi veya yürütülebilir kavanozu seçerek Gluon’un web sitesinden Scene Builder’ın en son sürümünü indirin.

  2. Yükleyici indirildiğinde, Scene Builder’ı sisteminize yüklemek için çift tıklayın. Güncellenmiş bir JRE dahildir.

  3. Bağımsız uygulama olarak çalıştırmak için Scene Builder simgesine çift tıklayın.

  4. IDE Entegrasyonu

    While Scene Builder is a standalone application, it produces FXML files that are integrated with a Java SE project. When creating this project on an IDE, it is convenient to include a link to the Scene Builder path, so FXML files can be edited.

    • NetBeans: On Windows go to NetBeans->Tools->Options->Java->JavaFX. On Mac OS X go to NetBeans->Preferences->Java->JavaFX. Provide the path for the Scene Builder Home.
    • IntelliJ: On Windows go to IntelliJ->Settings->Languages & Frameworks->JavaFX. On Mac OS X go to IntelliJ->Preferences->Languages & Frameworks->JavaFX. Provide the path for the Scene Builder Home.
    • Eclipse: On Windows go to Eclipse->Window->Preferences->JavaFX. On Mac OS X go to Eclipse->Preferences->JavaFX. Provide the path for the Scene Builder Home.

Biraz tarihçe

Scene Builder projesi Oracle tarafından JavaFX kullanılarak oluşturulmuştur ve OpenJFX projesi içinde açık kaynak kodludur.

Oracle sağlanan ikili dosyalar, Scene Builder v 2.0’a kadar, yalnızca Java SE 8u40’ın piyasaya sürülmesinden önceki JavaFX özelliklerini içerir, bu nedenle ‘Spinner’ kontrolleri gibi yeni özellikler dahil değildir.

Gluon ikili sürüm dağıtımını devraldı ve her platform için güncel bir Scene Builder 8+ buradan indirilebilir.

JavaFX’teki en son değişiklikleri ve ayrıca en son iyileştirmeleri ve hata düzeltmelerini içerir.

Açık kaynak projesi, sorunların, özellik isteklerinin ve çekme isteklerinin oluşturulabileceği burada bulunabilir.

Oracle eski ikili dosyaları hala buradan indirilebilir.

FXML kullanan temel JavaFX projesi #

Bu, NetBeans (Yeni Proje -> JavaFX -> JavaFX FXML Uygulaması) ile oluşturulmuş, FXML kullanan temel bir projedir. Sadece üç dosya içerir:

Ana Uygulama sınıfı

package org.stackoverflow;

import javafx.application.Application;
import javafx.fxml.FXMLLoader;
import javafx.scene.Parent;
import javafx.scene.Scene;
import javafx.stage.Stage;

public class BasicApplication extends Application {

    @Override
    public void start(Stage stage) throws Exception {
        Parent root = FXMLLoader.load(getClass().getResource("BasicFXML.fxml"));

        Scene scene = new Scene(root);

        stage.setScene(scene);
        stage.show();
    }

    public static void main(String[] args) {
        launch(args);
    }
}

FXML dosyası

<?xml version="1.0" encoding="UTF-8"?>

<?import java.lang.*?>
<?import java.util.*?>
<?import javafx.scene.*?>
<?import javafx.scene.control.*?>
<?import javafx.scene.layout.*?>

<AnchorPane id="AnchorPane" prefHeight="200" prefWidth="320" xmlns:fx="http://javafx.com/fxml/1" fx:controller="org.stackoverflow.BasicFXMLController">
    <children>
        <Button layoutX="126" layoutY="90" text="Click Me!" onAction="#handleButtonAction" fx:id="button" />
        <Label layoutX="126" layoutY="120" minHeight="16" minWidth="69" fx:id="label" />
    </children>
</AnchorPane>

Kontrolör

package org.stackoverflow;

import javafx.event.ActionEvent;
import javafx.fxml.FXML;
import javafx.scene.control.Label;

public class BasicFXMLController {

    @FXML
    private Label label;

    public void initialize() {
        // TODO
    }   

    @FXML
    private void handleButtonAction(ActionEvent event) {
        label.setText("Hello World!");
    }

}

Koşmak

Projeyi oluşturmak ve çalıştırmak, tıklanabilir bir düğme ile küçük bir pencere göstermelidir:

Nasıl çalışır

Kısaca, ana Application sınıfında, FXMLLoader, ‘FXMLLoader.load(getClass().getResource(“BasicFXML.fxml”))’ tarafından belirtildiği gibi jar/sınıf yolundan ‘basicFXML.fxml’ dosyasını yükler.

‘basicFXML.fxml’ yüklenirken, yükleyici, FXML’de ‘fx:controller=“org.stackoverflow.BasicFXMLController”’ tarafından belirtilen denetleyici sınıfının adını bulacaktır.

Ardından yükleyici, FXML’de “fx:id” olan ve denetleyici sınıfında “@FXML” notu ile işaretlenen tüm nesneleri enjekte etmeye çalışacağı o sınıfın bir örneğini oluşturacaktır.

Bu örnekte, FXMLLoader etiketi <Label ... fx:id="label"/> temelinde oluşturacak ve etiket örneğini @FXML private Label etiketine; enjekte edecektir.

Son olarak, tüm FXML yüklendiğinde, FXMLLoader denetleyicinin ‘initialize’ yöntemini çağıracak ve bir eylem işleyicisini düğmeyle kaydeden kod yürütülecektir.

Düzenleme

FXML dosyası IDE içinde düzenlenebilse de, IDE yalnızca temel sözdizimi denetimi ve otomatik tamamlama sağladığı, ancak görsel rehberlik sağlamadığı için önerilmez.

En iyi yaklaşım, tüm değişikliklerin dosyaya kaydedileceği FXML dosyasını Scene Builder ile açmaktır.

Dosyayı açmak için Scene Builder başlatılabilir:

Veya dosya, doğrudan IDE’den Scene Builder ile açılabilir:

  • NetBeans’ten proje sekmesinde dosyaya çift tıklayın veya sağ tıklayın ve ‘Aç’ı seçin.
  • IntelliJ’den proje sekmesinde dosyaya sağ tıklayın ve ‘Sahne Oluşturucuda Aç’ı seçin.
  • Eclipse’den proje sekmesinde dosyaya sağ tıklayın ve ‘Scene Builder ile Aç’ı seçin.

Scene Builder düzgün bir şekilde kurulmuşsa ve yolu IDE’ye eklenmişse (aşağıdaki Açıklamalara bakın), dosyayı açacaktır:

Sol bölmelerden yeni kaplar veya yeni kontroller sürüklenerek değişiklikler yapılabilir ve sağ bölmelerde özellikler ve düzen değerleri değiştirilebilir.

FXML’nin Java koduna enjekte edilmesini sağlayan kimlik etiketlerinden birinin “fx:id” olduğuna dikkat edin. “Kod” bölmesinde ayarlanabilir:

Değişiklikleri uyguladıktan sonra dosyayı kaydedin (Scene Builder -> Dosya -> Kaydet). Dosyayı IDE’den düzenlerken değişiklik yapılırsa, dosya kaydedilirken bunlar Scene Builder’da güncellenecektir.
