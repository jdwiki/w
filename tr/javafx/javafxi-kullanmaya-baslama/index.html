Merhaba Dünya programı #

Aşağıdaki kod, tıklandığında konsola bir “Dize” yazdıran tek bir “Düğme” içeren basit bir kullanıcı arabirimi oluşturur.

import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.Alert;
import javafx.scene.control.Alert.AlertType;
import javafx.scene.control.Button;
import javafx.scene.layout.StackPane;
import javafx.stage.Stage;

public class HelloWorld extends Application {

    @Override
    public void start(Stage primaryStage) {
        // create a button with specified text
        Button button = new Button("Say 'Hello World'");

        // set a handler that is executed when the user activates the button
        // e.g. by clicking it or pressing enter while it's focused
        button.setOnAction(e -> {
           //Open information dialog that says hello
           Alert alert = new Alert(AlertType.INFORMATION, "Hello World!?");
           alert.showAndWait();
        });

        // the root of the scene shown in the main window
        StackPane root = new StackPane();

        // add button as child of the root
        root.getChildren().add(button);

        // create a scene specifying the root and the size
        Scene scene = new Scene(root, 500, 300);

        // add scene to the stage
        primaryStage.setScene(scene);

        // make the stage visible
        primaryStage.show();
    }

    public static void main(String[] args) {
        // launch the HelloWorld application.

        // Since this method is a member of the HelloWorld class the first
        // parameter is not required
        Application.launch(HelloWorld.class, args);
    }

}

‘Application’ sınıfı, her JavaFX uygulamasının giriş noktasıdır. Yalnızca bir “Uygulama” başlatılabilir ve bu, kullanılarak yapılır.

Application.launch(HelloWorld.class, args);

Bu, parametre olarak geçirilen “Application” sınıfının bir örneğini oluşturur ve JavaFX platformunu başlatır.

Burada programcı için aşağıdakiler önemlidir:

  1. İlk “launch”, “Application” sınıfının yeni bir örneğini oluşturur (bu durumda “HelloWorld”). ‘Application’ sınıfı bu nedenle argümansız bir kurucuya ihtiyaç duyar.
  2. Oluşturulan ‘Application’ örneğinde ‘init()’ çağrılır. Bu durumda, “Uygulama"dan varsayılan uygulama hiçbir şey yapmaz.
  3. ‘Uygulama’ örneği için ‘start’ çağrılır ve birincil ‘Aşama’ (= pencere) yönteme iletilir. Bu yöntem, JavaFX Uygulama iş parçacığında (Platform iş parçacığı) otomatik olarak çağrılır.
  4. Uygulama, platform kapanma zamanının geldiğini belirleyene kadar çalışır. Bu, bu durumda son pencere kapatıldığında yapılır.
  5. “Application” örneğinde “stop” yöntemi çağrılır. Bu durumda, “Uygulama"dan uygulama hiçbir şey yapmaz. Bu yöntem, JavaFX Uygulama iş parçacığında (Platform iş parçacığı) otomatik olarak çağrılır.

‘start’ yönteminde sahne grafiği oluşturulur. Bu durumda 2 “Düğüm” içerir: Bir “Düğme” ve bir “Yığın Panosu”.

“Düğme”, kullanıcı arayüzündeki bir düğmeyi temsil eder ve “Yığın Panosu”, yerleşimini belirleyen “Düğme” için bir kapsayıcıdır.

Bu ‘Düğümleri’ görüntülemek için bir ‘Sahne’ oluşturulur. Son olarak, tüm kullanıcı arayüzünü gösteren pencere olan “Sahne"ye “Sahne” eklenir.

Kurulum veya Kurulum #

JavaFX API’leri, tam entegre bir özellik olarak mevcuttur. Java SE Runtime Environment (JRE) ve Java Development Kit (JDK). JDK tüm büyük masaüstü platformları için mevcut olduğundan (Windows, Mac OS X ve Linux), JDK 7 ve sonraki sürümlerde derlenmiş JavaFX uygulamaları ayrıca tüm büyük masaüstü platformlarında çalışır. ARM platformları için destek ayrıca JavaFX 8 ile kullanıma sunulmuştur. ARM için JDK şunları içerir: JavaFX’in temel, grafik ve kontrol bileşenleri.

JavaFX’i yüklemek için, Java Runtime ortamının seçtiğiniz sürümünü ve Java Development kit yükleyin.

JavaFX tarafından sunulan özellikler şunları içerir:

  1. Java API’leri.
  2. FXML ve Sahne Oluşturucu.
  3. Web Görünümü.
  4. Salıncak birlikte çalışabilirliği.
  5. Yerleşik UI kontrolleri ve CSS.
  6. Modena teması.
  7. 3D Grafik Özellikleri.
  8. Tuval API’si.
  9. Yazdırma API’si.
  10. Zengin Metin Desteği.
  11. Çoklu Dokunma Desteği.
  12. Hi-DPI desteği.
  13. Donanım hızlandırmalı grafik işlem hattı.
  14. Yüksek performanslı medya motoru.
  15. Bağımsız uygulama dağıtım modeli.
