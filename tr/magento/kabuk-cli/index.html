Temel bilgiler #

  • Kabuk betiklerini kullanmak için bir Linux komut satırına sahip olmanız veya sunucunuza SSH kullanarak bağlanmanız gerekir.
  • “MAGENTO_ROOT/kabuğunuza” gidin
  • Komut dosyası, i.e. yazarak çalıştırılabilir.
php -f indexer.php help

Dosyalara göre çekirdek kabuk yöntemleri #

  1. özet.php

  2. indeksleyici.php

  3. derleyici.php

  4. log.php

Özel php kabuk betikleri #

Bazen yürütme sürelerini azaltmak veya ön ucu etkilemeyecek farklı şeyler ayarlamak için Magento’ya bir web tarayıcısının dışında erişmemiz gerekir.

Magento’yu önyüklemenin 2 yolu vardır, ancak yalnızca biri Magento yoludur. Yukarıdaki örnekler bölümünde daha fazlasını okuyun.

Mage_Shell_Abstract’ı genişletmeden kabuğu kullanma #

Arayarak Magento’yu Önyükleme: #

require_once 'app/Mage.php';
Mage::app();
// Your code

Bu en basit yol ama aslında Magento yolu değil çünkü ‘Mage_Shell_Abstract’ı genişleten sınıfı kullanmıyoruz - genişletildiğinde bize komut satırı argümanlarını ayrıştırmak için araçlar sağlayan, yapıcısında ‘__applyPhpVariables()’ çağıran sınıf (işlev) .htaccess dosyalarını ayrıştırır ve kabuk komut dosyasına php ayarlarını uygular).

Kabuğu Magento yöntemiyle kullanma - Mage_Shell_Abstract’ı genişletin #

Magento yolu #

Dosya shell/custom.php içinde bulunuyor

<?php
require_once' abstract.php';

class Stackoverflow_Shell_Custom extends Mage_Shell_Abstract
{

protected $_argname = array();
 
    public function __construct() {
        parent::__construct();
 
        // Time limit to infinity
        set_time_limit(0);     
 
        // Get command line argument named "argname"
        // Accepts multiple values (comma separated)
        if($this->getArg('argname')) {
            $this->_argname = array_merge(
                $this->_argname,
                array_map(
                    'trim',
                    explode(',', $this->getArg('argname'))
                )
            );
        }
    }

 // Shell script point of entry
    public function run() {
 
    }
 
    // Usage help
    public function usageHelp()
    {
        return <<<USAGE
Usage:  php -f scriptname.php -- [options]
 
  --argname <argvalue>       Argument description
 
  help                   This help
 
USAGE;
    }
}
// Instantiate
$shell = new Stackoverflow_Shell_Custom();
 
// Initiate script
$shell->run();

}

php indexer.php status

php indexer.php reindexall
