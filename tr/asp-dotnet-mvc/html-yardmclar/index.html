HTML yardımcıları, bir görünümde HTML öğelerini oluşturmak için kullanılan yöntemlerdir. Bunlar “System.Web.Mvc.HtmlHelper” ad alanının bir parçasıdır.

HTML yardımcılarının farklı türleri vardır:

Standart HTML Yardımcıları: Normal HTML öğelerini oluşturmak için kullanılırlar, ör. Html.TextBox().

Kesinlikle Yazılan HTML Yardımcıları: Bu yardımcılar, HTML öğelerini model özelliklerine göre oluşturur, ör. Html.TextBoxFor().

Özel HTML Yardımcıları: Kullanıcı, MvcHtmlString döndüren özel yardımcı yöntemi oluşturabilir.

HTML Çıktıları ile Standart HTML Yardımcıları #

Html.TextBox()

  • @Html.TextBox("Ad", null, yeni { @class = "form-kontrol" }) output:<input class="form-control" id="Name"name="Name"type="text"value=""/>
  • @Html.TextBox("Ad", "Yığın Taşması", yeni { @class = "form-kontrol" }) output:<input class="form-control" id="Name"name="Name"type="text" value="Yığın Taşması"/>

Html.TextArea()

  • @Html.TextArea("Notlar", null, yeni { @class = "form-kontrol" }) çıktı:<textarea class="form-control" id="Notes" name="Notes" rows="2" cols="20"></textarea>
  • @Html.TextArea("Notlar", "Lütfen Notları Girin", yeni { @class = "form-kontrol" }) output:<textarea class="form-control" id="Notes" name="Notes" rows="2" cols="20" >Lütfen Notları girin</textarea>

Html.Etiket()

  • @Html.Label("Ad","FirstName") çıktı:<label for="Name"> Ad </label>
  • @Html.Label("Ad", "FirstName", yeni { @class = "NameClass" }) çıktı:<label for="Name" class="NameClass">FirstName</label>

Html.Gizli()

  • @Html.Hidden("Ad", "Değer") output:<input id="Ad" name="Ad" type="hidden" value="Değer" />

Html.CheckBox()

  • @Html.CheckBox("isStudent", doğru) output:<input check="checked" id="isStudent" name="isStudent" type="checkbox" value="true" />

Html.Parola()

  • @Html.Password("StudentPassword") output:<input id="StudentPassword" name="StudentPassword" type="password" value="" />

Özel Yardımcı - Etiketli Radyo Düğmesini Oluştur #

       public static MvcHtmlString RadioButtonLabelFor<TModel, TProperty>(this HtmlHelper<TModel> self, Expression<Func<TModel, TProperty>> expression, bool value, string labelText)
    {
        // Retrieve the qualified model identifier
        string name = ExpressionHelper.GetExpressionText(expression);
        string fullName = self.ViewContext.ViewData.TemplateInfo.GetFullHtmlFieldName(name);

        // Generate the base ID
        TagBuilder tagBuilder = new TagBuilder("input");
        tagBuilder.GenerateId(fullName);
        string idAttr = tagBuilder.Attributes["id"];

        // Create an ID specific to the boolean direction
        idAttr = string.Format("{0}_{1}", idAttr, value);

        // Create the individual HTML elements, using the generated ID
        MvcHtmlString radioButton = self.RadioButtonFor(expression, value, new { id = idAttr });
        MvcHtmlString label = self.Label(idAttr, labelText);

        return new MvcHtmlString(radioButton.ToHtmlString() + label.ToHtmlString());
    }

Örnek : @Html.RadioButtonLabelFor(m => m.IsActive, true, "Evet")

Özel HTML Yardımcısı - Görünen Ad #

/// <summary>
/// Gets displayName from DataAnnotations attribute
/// </summary>
/// <typeparam name="TModel"></typeparam>
/// <typeparam name="TProperty"></typeparam>
/// <param name="htmlHelper"></param>
/// <param name="expression"></param>
/// <returns></returns>
public static MvcHtmlString GetDisplayName<TModel, TProperty>(this HtmlHelper<TModel> htmlHelper, Expression<Func<TModel, TProperty>> expression)
{
    var metaData = ModelMetadata.FromLambdaExpression(expression, htmlHelper.ViewData);
    var value = metaData.DisplayName ?? (metaData.PropertyName ?? ExpressionHelper.GetExpressionText(expression));
    return MvcHtmlString.Create(value);
}

Özel Yardımcı - Gönder düğmesi oluştur #

/// <summary>
/// Creates simple button
/// </summary>
/// <param name="poHelper"></param>
/// <param name="psValue"></param>
/// <returns></returns>
public static MvcHtmlString SubmitButton(this HtmlHelper poHelper, string psValue)
{
    return new MvcHtmlString(string.Format("<input type=\"submit\" value=\"{0}\">", psValue));
}

HTML çıktısı dahil HtmlHelper örneklerinin kapsamlı listesi #

HtmlHelper.Action() #

  • @Html.Action(actionName: "Dizin") çıktı: Index() adlı bir eylem yöntemiyle oluşturulan HTML

  • @Html.Action(actionName: "Dizin", routeValues: yeni {id = 1}) çıktı: Index(int ​​id) adlı bir işlem yöntemiyle oluşturulan HTML

  • @(Html.Action("Index", routeValues: new RouteValueDictionary(new Dictionary<string, object>{ {"id", 1} }))) çıktı: Index(int ​​id) adlı bir işlem yöntemiyle oluşturulan HTML

  • @Html.Action(actionName: "Index", controllerName: "Home") çıktı: “HomeController"da “Index()” adlı bir eylem yöntemiyle oluşturulan HTML

  • @Html.Action(actionName: "Index", controllerName: "Home", routeValues: yeni {id = 1}) çıktı: “HomeController"da “Index(int ​​id)” adlı bir eylem yöntemiyle oluşturulan HTML

  • @Html.Action(actionName: "Index", controllerName: "Home", routeValues: new RouteValueDictionary(new Dictionary<string, object>{ {"id", 1} })) çıktı: “HomeController"da “Index(int ​​id)” adlı bir eylem yöntemiyle oluşturulan HTML

HtmlHelper.ActionLink() #

  • @Html.ActionLink(linkText: "Beni tıkla", actionName: "Dizin") çıktı: <a href="Home/Index">Beni tıklayın</a>

  • @Html.ActionLink(linkText: "Beni tıkla", actionName: "Index", routeValues: yeni {id = 1}) çıktı: <a href="Home/Index/1">Beni tıklayın</a>

  • @Html.ActionLink(linkText: "Beni tıkla", actionName: "Index", routeValues: yeni {id = 1}, htmlAttributes: new {@class = "btn btn-default", data_foo = "bar") çıktı: <a href="Home/Index/1" class="btn btn-default" data-foo="bar">Beni tıklayın</a>

  • @Html.ActionLink() çıktı: <a href=""></a>

@HtmlHelper.BeginForm() #

  • kullanarak (Html.BeginForm("MyAction", "MyController", FormMethod.Post, new {id="form1",@class = "form-horizontal"})) çıktı: <form action="/MyController/MyAction" class="form-horizontal" id="form1" method="post">

Özel Yardımcı - Tarih Saat Seçici #

public static MvcHtmlString DatePickerFor<TModel, TProperty>(this HtmlHelper<TModel> htmlHelper, Expression<Func<TModel, TProperty>> expression, object htmlAttributes)
{
    var sb = new StringBuilder();
    var metaData = ModelMetadata.FromLambdaExpression(expression, htmlHelper.ViewData);
    var dtpId = "dtp" + metaData.PropertyName;
    var dtp = htmlHelper.TextBoxFor(expression, htmlAttributes).ToHtmlString();
    sb.AppendFormat("<div class='input-group date' id='{0}'> {1} <span class='input-group-addon'><span class='glyphicon glyphicon-calendar'></span></span></div>", dtpId, dtp);
    return MvcHtmlString.Create(sb.ToString());
}

Örnek:

@Html.DatePickerFor(model => model.PublishedDate,  new { @class = "form-control" })

Bootstrap.v3.Datetimepicker kullanıyorsanız JavaScript’iniz aşağıdaki gibidir –

$('#dtpPublishedDate').datetimepicker({ format: 'MMM DD, YYYY' });
