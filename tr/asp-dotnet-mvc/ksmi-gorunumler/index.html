Kısmi görünüm, başka bir görünüm içinde oluşturulan bir görünümdür. Kısmi görünümler yeniden kullanılabilir ve böylece kod tekrarı önlenebilir. Html.Partial veya Html.RenderPartial tarafından işlenebilirler.

Sözdizimi #

  • @Html.Partial(“GörünümAdı”)

    @Html.Partial(“ViewName”,ViewModel)

    @{Html.RenderPartial(“ViewName”);}

Kısmi görünümünüz aşağıdakilerden farklı bir klasörde bulunuyorsa paylaşılan klasör, ardından görünümün tam yolunu şu şekilde belirtmeniz gerekir: aşağıda:

-@Html.RenderPartial("~/Areas/Admin/Views/Shared/partial/_subcat.cshtml")

Bir Dizeye Kısmi Görünüm - e-posta içeriği vb. için #

işlevi çağırmak

string InvoiceHtml = myFunction.RenderPartialViewToString("PartialInvoiceCustomer", ToInvoice); // ToInvoice is a model, you can pass parameters if needed

HTML oluşturma işlevi

public static string RenderPartialViewToString(string viewName, object model)
{
    using (var sw = new StringWriter())
    {
        BuyOnlineController controller = new BuyOnlineController(); // instance of the required controller (you can pass this as a argument if needed)

        // Create an MVC Controller Context
        var wrapper = new HttpContextWrapper(System.Web.HttpContext.Current);

        RouteData routeData = new RouteData();

        routeData.Values.Add("controller", controller.GetType().Name.ToLower().Replace("controller", ""));

        controller.ControllerContext = new ControllerContext(wrapper, routeData, controller);

        controller.ViewData.Model = model;

        var viewResult = ViewEngines.Engines.FindPartialView(controller.ControllerContext, viewName);

        var viewContext = new ViewContext(controller.ControllerContext, viewResult.View, controller.ViewData, controller.TempData, sw);
        viewResult.View.Render(viewContext, sw);

        return sw.ToString();
    }
}

Html.Partial Vs Html.RenderPartial #

Html.Partial bir dize döndürürken, Html.RenderPartial geçersiz döndürür.

Html.RenderPartial

Bu yöntem void döndürür ve sonuç doğrudan HTTP yanıt akışına yazılır. Bu, geçerli web sayfasında/şablonda kullanılan aynı TextWriter nesnesini kullandığı anlamına gelir. Bu nedenle, bu yöntem Kısmi yöntemden daha hızlıdır. Bu yöntem, kısmi görünümde görüntülenen veriler zaten ilgili görünüm modelinde olduğunda kullanışlıdır.

Örnek : Bir makalenin yorumlarını göstermek için bir blogda, görünüm modelinde yorum içeren bir makale bilgisi zaten doldurulduğundan RenderPartial yöntemini kullanmak istiyoruz.

@{Html.RenderPartial("_Comments");}

Html.Kısmi

Bu yöntem, HTML kodlu bir dize döndürür. Bu bir değişkende saklanabilir. RenderPartial yöntemi gibi, Kısmi yöntem de kısmi görünümde görüntülenen veriler zaten ilgili görünüm modelinde olduğunda kullanışlıdır.

Örnek: Bir makalenin yorumlarını göstermek için bir blogda, görünüm modelinde yorum içeren bir makale bilgisi zaten doldurulduğundan Kısmi yöntemini kullanabilirsiniz.

@Html.Partial("_Comments")
