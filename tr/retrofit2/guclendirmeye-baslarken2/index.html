Kurmak #

** Güçlendirme nedir?**

Resmi Güçlendirme sayfası kendisini şöyle tanımlar:

Android ve Java için güvenli bir REST istemcisi.

Bu kitaplık, bir web API’sinden JSON veya XML verilerini indirmeyi oldukça basit hale getirir. Veriler indirildikten sonra, burada listelenen bağdaştırıcı/ayrıştırıcıdan herhangi biri kullanılarak her istek için tanımlanan bir Düz Eski Java Nesnesine (POJO) ayrıştırılır.

Demo amacıyla GSON ayrıştırıcısını kullanırdık

Kurmak:

  1. manifest.xml’de internet izni ekleyin:
 <uses-permission android:name="android.permission.INTERNET" />
  1. build.gradle dosyanıza aşağıdakini ekleyin:
dependencies {
    compile 'com.squareup.retrofit2:retrofit:2.1.0'
    compile 'com.squareup.retrofit2:converter-gson:2.1.0'  
}

  1. Json yanıtınıza göre uygun POJO(Model) oluşturun:

    If your json response is:

{
    "CategoryModel": {
        "debug": "on",
        "window": {
            "title": "Sample Konfabulator Widget",
            "name": "main_window",
            "width": 500,
            "height": 500
        },
        "image": {
            "src": "Images/Sun.png",
            "name": "sun1",
            "hOffset": 250,
            "vOffset": 250,
            "alignment": "center"
        },
        "text": {
            "data": "Click Here",
            "size": 36,
            "style": "bold",
            "name": "text1",
            "hOffset": 250,
            "vOffset": 100,
            "alignment": "center",
            "onMouseUp": "sun1.opacity = (sun1.opacity / 100) * 90;"
        }
    }
}

Ardından, modelinizi oluşturmanıza yardımcı olacak json’unuzu biçimlendirmek için JsonOnlineEditor veya JsonView gibi bir web sitesini kullanabilir veya Json’unuzu [GSON][3 kullanarak POJO’ya dönüştürmek için jsonschema2pojo öğesini kullanabilirsiniz. ] ek açıklamalar :

public class CategoryModel {
   
    @SerializedName("debug")
    private String debug;

    @SerializedName("window")
    private Window window;

    @SerializedName("image")
    private Image image;

    @SerializedName("text")
    private Text text;
}

  1. Ardından, tüm istek ve yanıt için denetleyici görevi gören bir Retrofit örneğine ihtiyacımız var.

    Note : We prefer to create this controller as singleton which is very helpful if we want to set some additional property of the client .

public static final String BASE_URL = "http://test.com"

Retrofit retrofit = new Retrofit.Builder()
            .baseUrl(BASE_URL)
            .addConverterFactory(GsonConverterFactory.create())
            .build();
  1. Ardından, tüm API çağrılarını istek, yanıt tipi ve her çağrı için istek parametreleri ile tanımladığınız Arayüz sınıfını oluşturun.(GET,POST..vb. )
public interface IPlusService {
    @GET("/api/category")
    Call<CategoryModel> getAllCategory();
}
  1. Güçlendirme örneği ile ağ/istek istemcisi oluşturun:
IPlusService requestClient = retrofit.create(IPlusService.class);
  1. Fragment/Activity içinde web servisinizi arayın:
requestClient.getAllCategory().enqueue(new Callback<ResponseBody>() {
    @Override
    public void onResponse(Call<CategoryModel> call, Response<CategoryModel> response) {
        // DO success handling 
    }

    @Override
    public void onFailure(Call<CategoryModel> call, Throwable t) {
        // DO failure handling 
    }
});
