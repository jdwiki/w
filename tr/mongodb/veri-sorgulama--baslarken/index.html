Temel sorgulama örnekleri

Bulmak() #

bir koleksiyondaki tüm belgeleri al

db.collection.find({});

bir koşul kullanarak bir koleksiyondaki belgeleri alın (MYSQL’deki WHERE’ye benzer)

db.collection.find({key: value}); 
example
  db.users.find({email:"sample@email.com"});

Boolean koşullarını kullanarak bir koleksiyondaki belgeleri alın (Sorgu Operatörleri)

//AND
db.collection.find( {
    $and: [
     { key: value }, { key: value } 
    ] 
})
//OR
db.collection.find( {
    $or: [
     { key: value }, { key: value } 
    ] 
})
//NOT
db.inventory.find( { key: { $not: value } } )

daha fazla boole işlemi ve örneği bulunabilir burada

NOT: find(), bir belge eşleşmesi bulunsa bile koleksiyonu aramaya devam eder, bu nedenle büyük bir koleksiyonda kullanıldığında verimsizdir, ancak verilerinizi dikkatli bir şekilde modelleyerek ve/veya dizinleri kullanarak şunları yapabilirsiniz: find() işlevinin verimliliğini artırın

##Birini Bul() db.collection.findOne({});

sorgulama işlevi find() işlevine benzer, ancak bu, koşuluyla eşleşen bir belge bulduğu anda yürütmeyi sona erdirir, boş nesne ile birlikte kullanılırsa, ilk belgeyi alır ve döndürür. findOne() mongodb API belgeleri

find() yönteminin sonuçlarını sınırlayın, atlayın, sıralayın ve sayın #

Toplama yöntemlerine benzer şekilde find() yöntemiyle de sonuçları sınırlama, atlama, sıralama ve sayma olanağınız vardır. Diyelim ki aşağıdaki koleksiyonumuz var:

db.test.insertMany([
    {name:"Any", age:"21", status:"busy"}, 
    {name:"Tony", age:"25", status:"busy"}, 
    {name:"Bobby", age:"28", status:"online"}, 
    {name:"Sonny", age:"28", status:"away"}, 
    {name:"Cher", age:"20", status:"online"}
])

Koleksiyonu listelemek için:

db.test.find({})

Geri dönücek:

{ "_id" : ObjectId("592516d7fbd5b591f53237b0"), "name" : "Any", "age" : "21", "status" : "busy" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b1"), "name" : "Tony", "age" : "25", "status" : "busy" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b2"), "name" : "Bobby", "age" : "28", "status" : "online" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b3"), "name" : "Sonny", "age" : "28", "status" : "away" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b4"), "name" : "Cher", "age" : "20", "status" : "online" }

İlk 3 belgeyi atlamak için:

db.test.find({}).skip(3)

Geri dönücek:

{ "_id" : ObjectId("592516d7fbd5b591f53237b3"), "name" : "Sonny", "age" : "28", "status" : "away" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b4"), "name" : "Cher", "age" : "20", "status" : "online" }

Alan adına göre azalan şekilde sıralamak için:

db.test.find({}).sort({ "name" : -1})

Geri dönücek:

{ "_id" : ObjectId("592516d7fbd5b591f53237b1"), "name" : "Tony", "age" : "25", "status" : "busy" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b3"), "name" : "Sonny", "age" : "28", "status" : "away" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b4"), "name" : "Cher", "age" : "20", "status" : "online" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b2"), "name" : "Bobby", "age" : "28", "status" : "online" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b0"), "name" : "Any", "age" : "21", "status" : "busy" }

Artan şekilde sıralamak istiyorsanız, sadece -1’i 1 ile değiştirin.

Sonuçları saymak için:

db.test.find({}).count()

Geri dönücek:

5

Ayrıca bu yöntemlerin kombinasyonlarına da izin verilir. Örneğin, ilk 1’i atlayarak azalan sıralı koleksiyondan 2 belge alın:

db.test.find({}).sort({ "name" : -1}).skip(1).limit(2)

Geri dönücek:

{ "_id" : ObjectId("592516d7fbd5b591f53237b3"), "name" : "Sonny", "age" : "28", "status" : "away" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b4"), "name" : "Cher", "age" : "20", "status" : "online" }

Projeksiyonlu find() yöntemi #

find() yönteminin izdüşümlü temel sözdizimi aşağıdaki gibidir.

> db.COLLECTION_NAME.find({},{KEY:1});

Tüm belgeleri yaş alanı olmadan göstermek istiyorsanız, komut aşağıdaki gibidir.

db.people.find({},{age : 0});

Tüm belgelere yaş alanını göstermek istiyorsanız, komut aşağıdaki gibidir.

Projeksiyonlu Find() yöntemi #

MongoDB’de projeksiyon, bir belgenin tüm verilerini seçmek yerine yalnızca gerekli verileri seçmek anlamına gelir.

find() yönteminin izdüşümlü temel sözdizimi aşağıdaki gibidir.

> db.COLLECTION_NAME.find({},{KEY:1});

Tüm belgeleri yaş alanı olmadan göstermek istiyorsanız, komut aşağıdaki gibidir.

> db.people.find({},{age:0});

Sadece yaş alanını göstermek istiyorsanız komut aşağıdaki gibidir.

> db.people.find({},{age:1});

Not: ‘find()’ yöntemi yürütülürken ‘_id’ alanı her zaman görüntülenir, bu alanı istemiyorsanız, ‘0’ olarak ayarlamanız gerekir.

> db.people.find({},{name:1,_id:0});

Not: Alanı göstermek için “1”, alanları gizlemek için “0” kullanılır.
