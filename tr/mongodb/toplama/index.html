“Toplamalar” işlemleri veri kayıtlarını işler ve hesaplanan sonuçları döndürür. Toplama işlemleri, birden çok belgedeki değerleri birlikte gruplandırır ve tek bir sonuç döndürmek için gruplanmış veriler üzerinde çeşitli işlemler gerçekleştirebilir. MongoDB, toplama gerçekleştirmek için üç yol sağlar: toplama ardışık düzeni, harita azaltma işlevi ve tek amaçlı toplama yöntemleri.

Mongo kılavuzundan https://docs.mongodb.com/manual/aggregation/

Sözdizimi #

  • db.collection.aggregate(boru hattı, seçenekler)

Parametreler #

parametre Ayrıntılar
boru hattı dizi(bir dizi veri toplama işlemi veya aşaması)
seçenekler belge(isteğe bağlı, yalnızca ardışık düzen dizi olarak mevcutsa kullanılabilir)

MongoDB’deki toplama çerçevesi, SQL’in ortak ‘GROUP BY’ işlevselliğini elde etmek için kullanılır.

Her örnek için “işlemler” adlı koleksiyondaki aşağıdaki eklemeleri göz önünde bulundurun.

> db.transactions.insert({ cr_dr : "D", amount : 100, fee : 2});
> db.transactions.insert({ cr_dr : "C", amount : 100, fee : 2});
> db.transactions.insert({ cr_dr : "C", amount : 10,  fee : 2});
> db.transactions.insert({ cr_dr : "D", amount : 100, fee : 4});
> db.transactions.insert({ cr_dr : "D", amount : 10,  fee : 2});
> db.transactions.insert({ cr_dr : "C", amount : 10,  fee : 4});
> db.transactions.insert({ cr_dr : "D", amount : 100, fee : 2});

Saymak #

Borç ve Alacak işlemlerinin sayısını nasıl alırsınız? Bunu yapmanın bir yolu, aşağıdaki gibi count() işlevini kullanmaktır.

> db.transactions.count({cr_dr : "D"});

veya

> db.transactions.find({cr_dr : "D"}).length();

Ama ya cr_dr olası değerlerini önceden bilmiyorsanız. Burada Toplama çerçevesi devreye giriyor. Aşağıdaki Toplama sorgusuna bakın.

> db.transactions.aggregate( 
      [
          {
              $group : {
                  _id : '$cr_dr', // group by type of transaction
                // Add 1 for each document to the count for this type of transaction
                  count : {$sum : 1}
              }
          }
      ]
  );

Ve sonuç

{
    "_id" : "C",
    "count" : 3
}
{
    "_id" : "D",
    "count" : 5
}

Toplam #

‘Miktar’ toplamı nasıl alınır? Aşağıdaki toplu sorguya bakın.

> db.transactions.aggregate( 
     [
         {
             $group : {
                 _id : '$cr_dr',
                 count : {$sum : 1},    //counts the number
                 totalAmount : {$sum : '$amount'}    //sums the amount
             }
         }
     ] 
 );

Ve sonuç

{
    "_id" : "C",
    "count" : 3.0,
    "totalAmount" : 120.0
}
{
    "_id" : "D",
    "count" : 5.0,
    "totalAmount" : 410.0
}

“Tutar” ve “ücret"i toplayan başka bir versiyon.

> db.transactions.aggregate(
      [
          {
              $group : {
                  _id : '$cr_dr',
                  count : {$sum : 1},
                  totalAmount : {$sum : { $sum : ['$amount', '$fee']}}
              }
          }
      ] 
  );

Ve sonuç

{
    "_id" : "C",
    "count" : 3.0,
    "totalAmount" : 128.0
}    
{
    "_id" : "D",
    "count" : 5.0,
    "totalAmount" : 422.0
}

Ortalama #

Ortalama borç ve kredi işlemleri tutarı nasıl alınır?

> db.transactions.aggregate(
    [
        {
            $group : {
                _id : '$cr_dr', // group by type of transaction (debit or credit)
                count : {$sum : 1},    // number of transaction for each type
                totalAmount : {$sum : { $sum : ['$amount', '$fee']}},    // sum
                averageAmount : {$avg : { $sum : ['$amount', '$fee']}}   // average
            }
        }
    ] 
)

sonuç

{
    "_id" : "C", // Amounts for credit transactions
    "count" : 3.0,
    "totalAmount" : 128.0,
    "averageAmount" : 40.0
}
{
    "_id" : "D", // Amounts for debit transactions
    "count" : 5.0,
    "totalAmount" : 422.0,
    "averageAmount" : 82.0
}

Dizilerle işlemler. #

Dizilerdeki veri girişleriyle çalışmak istediğinizde önce diziyi çözmeniz gerekir. Çözme işlemi, dizideki her giriş için bir belge oluşturur. Büyük dizilere sahip çok sayıda belgeniz olduğunda, belge sayısında bir patlama göreceksiniz.

{ "_id" : 1, "item" : "myItem1", sizes: [ "S", "M", "L"] }
{ "_id" : 2, "item" : "myItem2", sizes: [ "XS", "M", "XL"] }
    
db.inventory.aggregate( [ { $unwind : "$sizes" }] )

Önemli bir uyarı, bir belge diziyi içermediğinde kaybolacağıdır. Mongo 3.2 ve üzeri sürümlerde, “preserveNullAndEmptyArrays” eklenmiş bir gevşeme seçeneği vardır. Bu seçenek, dizi eksik olduğunda belgenin korunmasını sağlar.

{ "_id" : 1, "item" : "myItem1", sizes: [ "S", "M", "L"] }
{ "_id" : 2, "item" : "myItem2", sizes: [ "XS", "M", "XL"] }
{ "_id" : 3, "item" : "myItem3" }
    
db.inventory.aggregate( [ { $unwind : { path: "$sizes", includeArrayIndex: "arrayIndex" } }] )

Kibrit #

Ortalama 70000 $ veya daha az kazanan çalışanların yaşının 35 veya daha büyük olduğu tüm departmanları almak için nasıl bir sorgu yazılır?

Bunun için 70000$ veya daha az maaşı olan çalışanları eşleştirmek için bir sorgu yazmamız gerekiyor. Ardından, çalışanları departmana göre gruplandırmak için toplu aşamayı ekleyin. Ardından, ör. $avg akümülatörünü kullanarak departman başına ortalama yaşı bulmak için ve mevcut $match ve $group toplamlarının altında ortalama_age, başka bir $match toplaması ekler, böylece yalnızca 35’e eşit veya daha büyük olan bir ortalama_age ile sonuçlar alıyoruz.

db.employees.aggregate([
  {"$match": {"salary": {"$lte": 70000}}},
  {"$group": {"_id": "$dept",
              "average_age": {"$avg": "$age"}
             }
  },
  {"$match": {"average_age": {"$gte": 35}}}
])

Sonuç:

{
  "_id": "IT",
  "average_age": 31
}
{
  "_id": "Customer Service",
  "average_age": 34.5
}
{
  "_id": "Finance",
  "average_age": 32.5
}

Koleksiyonda yinelenen alana sahip dokümanları kaldırın (tekilleştirme) #

allowDiskUse: true seçeneğinin isteğe bağlı olduğunu, ancak koleksiyon boyutunuz büyükse bu toplama işlemi bellek açısından yoğun bir işlem olabileceğinden bellek yetersizliği sorunlarının azaltılmasına yardımcı olacağını unutmayın - bu nedenle her zaman kullanmanızı öneririm.

var duplicates = [];

db.transactions.aggregate([
 { $group: { 
   _id: { cr_dr: "$cr_dr"},
   dups: { "$addToSet": "$_id" }, 
   count: { "$sum": 1 } 
 }
}, 
{ $match: { 
  count: { "$gt": 1 }   
}}
],allowDiskUse: true}
)              
.result          
.forEach(function(doc) {
  doc.dups.shift();   
  doc.dups.forEach( function(dupId){ 
    duplicates.push(dupId); 
  }
)    
})
// printjson(duplicates);     

// Remove all duplicates in one go    
db.transactions.remove({_id:{$in:duplicates}})
