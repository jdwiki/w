“Destination-over” ile mevcut şekillerin arkasına çizin #

context.globalCompositeOperation = "destination-over"

“destination-over” kompozisyonu yeni çizimi mevcut çizimlerin altına yerleştirir.

context.drawImage(rainy,0,0);
context.globalCompositeOperation='destination-over';  // sunny UNDER rainy
context.drawImage(sunny,0,0);

“Hedef dışı” ile mevcut şekilleri silin #

context.globalCompositeOperation = "destination-out"

“hedef dışı” birleştirme, mevcut çizimleri silmek için yeni şekiller kullanır.

Yeni şekil aslında çizilmez - sadece mevcut pikselleri silmek için “çerez kesici” olarak kullanılır.

context.drawImage(apple,0,0);
context.globalCompositeOperation = 'destination-out';   // bitemark erases
context.drawImage(bitemark,100,40);

Varsayılan birleştirme: Mevcut şekiller üzerine yeni şekiller çizilir #

context.globalCompositeOperation = "source-over"

“kaynak üzerinden” birleştirme [varsayılan], tüm yeni çizimleri mevcut çizimlerin üzerine yerleştirir.

context.globalCompositeOperation='source-over';  // the default
context.drawImage(background,0,0);
context.drawImage(parachuter,0,0);

“Destination-in” ile şekillerin içindeki görüntüleri kırpın #

context.globalCompositeOperation = "destination-in"

“destination-in”, mevcut çizimleri yeni bir şekil içinde birleştirir.

Not: Mevcut çizimin yeni çizimin dışında kalan herhangi bir kısmı silinir.

context.drawImage(picture,0,0);
context.globalCompositeOperation='destination-in';  // picture clipped inside oval
context.drawImage(oval,0,0);

“Kaynak girişi” ile şekillerin içindeki görüntüleri kırpın #

context.globalCompositeOperation = "source-in";

source-in birleştirme klipleri yeni çizimleri mevcut bir şekil içinde.

Not: Yeni çizimin mevcut çizimin dışında kalan herhangi bir kısmı silinir.

context.drawImage(oval,0,0);
context.globalCompositeOperation='source-in';  // picture clipped inside oval
context.drawImage(picture,0,0);

“source-atop” ile iç gölgeler #

context.globalCompositeOperation = 'source-atop'

‘source-atop’ birleştirme, yeni görüntüyü mevcut bir şeklin içinde birleştirir.

// gold filled rect
ctx.fillStyle='gold';
ctx.fillRect(100,100,100,75);
// shadow
ctx.shadowColor='black';
ctx.shadowBlur=10;
// restrict new draw to cover existing pixels
ctx.globalCompositeOperation='source-atop';
// shadowed stroke
// "source-atop" clips off the undesired outer shadow
ctx.strokeRect(100,100,100,75);
ctx.strokeRect(100,100,100,75);

Görüntüyü “fark” ile ters çevirin veya olumsuzlayın #

Bileşik işlemle bir görüntünün üzerinde beyaz bir dikdörtgen oluşturun

ctx.globalCompositeOperation = 'difference';

Efektin miktarı alfa ayarıyla kontrol edilebilir

// Render the image
ctx.globalCompositeOperation='source-atop';
ctx.drawImage(image, 0, 0);

// set the composite operation
ctx.globalCompositeOperation='difference';
ctx.fillStyle = "white";
ctx.globalAlpha = alpha;  // alpha 0 = no effect 1 = full effect
ctx.fillRect(0, 0, image.width, image.height);

“Renk” ile Siyah Beyaz #

aracılığıyla bir görüntüden rengi kaldırın

ctx.globalCompositeOperation = 'color';

Efektin miktarı alfa ayarıyla kontrol edilebilir

// Render the image
ctx.globalCompositeOperation='source-atop';
ctx.drawImage(image, 0, 0);

// set the composite operation
ctx.globalCompositeOperation='color';
ctx.fillStyle = "white";
ctx.globalAlpha = alpha;  // alpha 0 = no effect 1 = full effect
ctx.fillRect(0, 0, image.width, image.height);

“Doygunluk” ile renk kontrastını artırın #

Bir görüntünün doygunluk seviyesini şu şekilde artırın:

ctx.globalCompositeOperation = 'saturation';

Efektin miktarı, alfa ayarı veya dolgu kaplamasındaki doygunluk miktarı ile kontrol edilebilir.

// Render the image
ctx.globalCompositeOperation='source-atop';
ctx.drawImage(image, 0, 0);

// set the composite operation
ctx.globalCompositeOperation ='saturation';
ctx.fillStyle = "red";
ctx.globalAlpha = alpha;  // alpha 0 = no effect 1 = full effect
ctx.fillRect(0, 0, image.width, image.height);

“Parlaklık” özelliğine sahip Sepya FX #

ile renkli bir sepya FX oluşturun.

ctx.globalCompositeOperation = 'luminosity';

Bu durumda sepya rengi önce görüntüye dönüştürülür.

Efektin miktarı, alfa ayarı veya dolgu kaplamasındaki doygunluk miktarı ile kontrol edilebilir.

// Render the image
ctx.globalCompositeOperation='source-atop';
ctx.fillStyle = "#F80";  // the color of the sepia FX
ctx.fillRect(0, 0, image.width, image.height);

// set the composite operation
ctx.globalCompositeOperation ='luminosity';

ctx.globalAlpha = alpha;  // alpha 0 = no effect 1 = full effect
ctx.drawImage(image, 0, 0);

“globalAlpha” ile opaklığı değiştirin #

context.globalAlpha=0.50

“GlobalAlpha"yı 0.00 (tamamen şeffaf) ile 1.00 (tamamen opak) arasında bir değere ayarlayarak yeni çizimlerin opaklığını değiştirebilirsiniz.

Varsayılan “globalAlpha” 1.00’dır (tamamen opak).

Mevcut çizimler “globalAlpha"dan etkilenmez.

// draw an opaque rectangle
context.fillRect(10,10,50,50);

// change alpha to 50% -- all new drawings will have 50% opacity
context.globalAlpha=0.50;

// draw a semi-transparent rectangle
context.fillRect(100,10,50,50);
