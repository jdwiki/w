Tuval üzerinde fare konumunu algılama #

Bu örnek, HTML5 Kanvas’ın sol üst köşesi “(0,0)” olacak şekilde fare konumunun tuvale göre nasıl alınacağını gösterecektir. “e.clientX” ve “e.clientY”, fare konumlarını belgenin üst kısmına göre alır, bunu tuvalin üst kısmına göre değiştirmek için, ekranın “sol” ve “sağ” konumlarını çıkarırız. X ve Y istemcisinden tuval.

var canvas = document.getElementById("myCanvas");
var ctx = canvas.getContext("2d");
ctx.font = "16px Arial";

canvas.addEventListener("mousemove", function(e) { 
    var cRect = canvas.getBoundingClientRect();        // Gets CSS pos, and width/height
    var canvasX = Math.round(e.clientX - cRect.left);  // Subtract the 'left' of the canvas 
    var canvasY = Math.round(e.clientY - cRect.top);   // from the X/Y positions to make  
    ctx.clearRect(0, 0, canvas.width, canvas.height);  // (0,0) the top left of the canvas
    ctx.fillText("X: "+canvasX+", Y: "+canvasY, 10, 20);
});

Çalıştırılabilir Örnek

“Math.round” kullanımı, tuvalin sınırlayıcı dikdörtgeninin tamsayı konumlarına sahip olmayabileceğinden, “x,y” konumlarının tamsayı olmasını sağlamak içindir.

Tuval boyutu ve çözünürlüğü #

Bir tuvalin boyutu, sayfada kapladığı alandır ve CSS genişlik ve yükseklik özellikleriyle tanımlanır.

canvas {
   width : 1000px;
   height : 1000px;
}

Tuval çözünürlüğü içerdiği piksel sayısını tanımlar. Çözünürlük, tuval öğesi genişlik ve yükseklik özellikleri ayarlanarak ayarlanır. Belirtilmezse, tuval varsayılan olarak 300 x 150 piksel olur.

Aşağıdaki tuval yukarıdaki CSS boyutunu kullanacak ancak “genişlik” ve “yükseklik” belirtilmediği için çözünürlük 300’e 150 olacaktır.

<canvas id="my-canvas"></canvas>

Bu, her pikselin eşit olmayan bir şekilde gerilmesine neden olur. Piksel boyutu 1:2’dir. Tuval gerildiğinde tarayıcı bilinear filtrelemeyi kullanır. Bunun, uzatılan pikselleri bulanıklaştırma etkisi vardır.

Tuvali kullanırken en iyi sonuçları elde etmek için tuval çözünürlüğünün ekran boyutuyla eşleştiğinden emin olun.

Görüntüleme boyutuyla eşleştirmek için yukarıdaki CSS stilinden sonra tuvali, stilin tanımladığı piksel sayısına “width” ve “height” ayarlı olarak ekleyin.

<canvas id = "my-canvas" width = "1000" height = "1000"></canvas>

Html5 Kanvas Öğesi bir web sayfasına nasıl eklenir #

Html5-Tuval …

  • Bir Html5 öğesidir.
  • Çoğu modern tarayıcıda desteklenir (Internet Explorer 9+).
  • Varsayılan olarak saydam olan görünür bir öğedir
  • Varsayılan genişliği 300 piksel ve varsayılan yüksekliği 150 pikseldir.
  • Tüm içeriğin Canvas’a programlı olarak eklenmesi gerektiğinden JavaScript gerektirir.

Örnek: Hem Html5 işaretlemesini hem de JavaScript’i kullanarak bir Html5-Canvas öğesi oluşturun:

<!doctype html>
<html>
<head>
<style>
    body{ background-color:white; }
    #canvasHtml5{border:1px solid red; }
    #canvasJavascript{border:1px solid blue; }
</style>
<script>
window.onload=(function(){

    // add a canvas element using javascript
    var canvas=document.createElement('canvas');
    canvas.id='canvasJavascript'
    document.body.appendChild(canvas);

}); // end $(function(){});
</script>
</head>
<body>

    <!-- add a canvas element using html -->
    <canvas id='canvasHtml5'></canvas>

</body>
</html>

Ekran dışı tuval #

Çoğu zaman tuvalle çalışırken, bazı giriş piksel verilerini tutmak için bir tuvaliniz olması gerekir. Ekran dışı bir tuval oluşturmak, 2B bağlam elde etmek kolaydır. Ekran dışı bir tuval, işlemek için mevcut grafik donanımını da kullanır.

Aşağıdaki kod basitçe bir tuval oluşturur ve onu mavi piksellerle doldurur.

function createCanvas(width, height){
    var canvas = document.createElement("canvas"); // create a canvas element
    canvas.width = width;
    canvas.height = height;
    return canvas;
}

var myCanvas = createCanvas(256,256); // create a small canvas 256 by 256 pixels
var ctx = myCanvas.getContext("2d");
ctx.fillStyle = "blue";
ctx.fillRect(0,0,256,256);

Çoğu zaman ekran dışı tuval birçok görev için kullanılacaktır ve birçok tuvaliniz olabilir. Tuvalin kullanımını basitleştirmek için tuval bağlamını tuvale ekleyebilirsiniz.

function createCanvasCTX(width, height){
    var canvas = document.createElement("canvas"); // create a canvas element
    canvas.width = width;
    canvas.height = height;
    canvas.ctx = canvas.getContext("2d");
    return canvas;
}
var myCanvas = createCanvasCTX(256,256); // create a small canvas 256 by 256 pixels
myCanvas.ctx.fillStyle = "blue";
myCanvas.ctx.fillRect(0,0,256,256);

Html5 Canvas Yetenekleri ve Kullanımları için bir dizin #

Kanvasın Yetenekleri #

Canvas, web sayfanıza programlı olarak çizim yapmanızı sağlar:

  • Görüntüler,
  • Metinler,
  • Çizgiler ve Eğriler.

Tuval çizimleri kapsamlı bir şekilde şekillendirilebilir:

  • kontur genişliği,
  • kontur rengi,
  • şekil dolgu rengi,
  • [opaklık](https://www.wikiod.com/tr/html5-canvas/kompozisyon#Opaklığı “globalAlpha” ile değiştirin),
  • gölgeleme,
  • doğrusal gradyanlar ve [radyal gradyanlar](http: //stackoverflow.com/documentation/html5-canvas/3241/path-syntax-only/13476/createradialgradient-creates-a-path-styling-object),
  • yazı tipi yüzü,
  • yazı tipi boyutu,
  • metin hizalaması,
  • metin hem konturlu hem de dolgulu olabilir,
  • görüntü yeniden boyutlandırma,
  • görüntü kırpma,
  • birleştirme

Tuvalin Kullanım Alanları #

Çizimler birleştirilebilir ve tuval üzerinde herhangi bir yere yerleştirilebilir, böylece aşağıdakileri oluşturmak için kullanılabilir:

  • Boya / Eskiz uygulamaları,
  • Hızlı tempolu interaktif oyunlar,
  • Çizelge, grafik gibi veri analizleri,
  • Photoshop benzeri görüntüleme,
  • Flash benzeri reklam ve Gösterişli web içeriği.

Canvas, görüntülerin Kırmızı, Yeşil, Mavi ve Alfa bileşen renklerini değiştirmenize olanak tanır. Bu, tuvalin Photoshop’a benzer sonuçlarla görüntüleri işlemesine olanak tanır.

  • Bir görüntünün herhangi bir bölümünü piksel düzeyinde yeniden renklendirin (HSL kullanıyorsanız, önemli Aydınlatma ve Doygunluğu koruyarak bir görüntüyü yeniden renklendirebilirsiniz, böylece sonuç, görüntüye boyayla boyanmış gibi görünmez),
  • Bir görüntüdeki bir kişinin/öğenin etrafındaki arka planı “nakavt”,
  • Bir görüntünün bir kısmını Algıla ve Doldur (örneğin, kullanıcı tarafından tıklanan bir çiçek yaprağının rengini yeşilden sarıya değiştirin - sadece o tıklanan taç yaprağı!),
  • Perspektif çarpıtma yapın (örneğin, bir resmi bir bardağın eğrisine sarın),
  • Bir görüntüyü içerik açısından inceleyin (ör. yüz tanıma),
  • Bir resimle ilgili soruları yanıtlayın: Park yerimin bu resminde park etmiş bir araba var mı?,
  • Standart görüntü filtreleri uygulayın (gri tonlamalı, sepya, vb.)
  • Hayal edebileceğiniz herhangi bir egzotik görüntü filtresini uygulayın (Sobel Edge Detection),
  • Görüntüleri birleştirin. Sevgili Büyükanne Sue aile birleşimine gidemediyse, onu yeniden bir araya gelme görüntüsüne “photoshop"la. Kuzen Phil’i sevmiyorum - sadece “onu photoshopla,
  • Bir video oynatın / Bir videodan bir kare alın,
  • Tuval içeriğini .jpg olarak dışa aktarın | .png görüntü (isteğe bağlı olarak görüntüyü kırpabilir veya açıklama ekleyebilir ve sonucu yeni bir görüntü olarak dışa aktarabilirsiniz),

Tuval çizimlerini taşıma ve düzenleme hakkında (örneğin bir aksiyon oyunu oluşturmak için):

  • Tuval üzerine bir şey çizildikten sonra, mevcut çizim taşınamaz veya düzenlenemez. Kanvas çizimlerinin taşınabilir olduğuna dair bu yaygın yanılgı açıklığa kavuşturulmaya değer: Mevcut kanvas çizimleri düzenlenemez veya taşınamaz!
  • Kanvas çok, çok hızlı çizer. Kanvas, saniyenin çok kısa bir bölümünde yüzlerce resim, metin, çizgi ve eğri çizebilir. Çizimi hızlandırmak için uygun olduğunda GPU’yu kullanır.
  • Kanvas, bir şeyi hızlı ve tekrar tekrar çizerek ve ardından onu yeni bir konumda yeniden çizerek hareket yanılsaması yaratır. Televizyon gibi, bu sürekli yeniden çizim göze hareket yanılsaması verir.

Döndür #

2B bağlamın “rotate(r)” yöntemi, tuvali, orijin çevresinde belirtilen “r” radyan miktarı kadar döndürür.

HTML

<canvas id="canvas" width=240 height=240 style="background-color:#808080;">
</canvas>

<button type="button" onclick="rotate_ctx();">Rotate context</button>

Javascript

var canvas = document.getElementById("canvas");
var ctx = canvas.getContext("2d");
var ox = canvas.width / 2;
var oy = canvas.height / 2;
ctx.font = "42px serif";
ctx.textAlign = "center";
ctx.textBaseline = "middle";
ctx.fillStyle = "#FFF";
ctx.fillText("Hello World", ox, oy);

rotate_ctx = function() {
  // translate so that the origin is now (ox, oy) the center of the canvas
  ctx.translate(ox, oy);
  // convert degrees to radians with radians = (Math.PI/180)*degrees. 
  ctx.rotate((Math.PI / 180) * 15);
  ctx.fillText("Hello World", 0, 0);
  // translate back
  ctx.translate(-ox, -oy);
};

Tuvali resim dosyasına kaydet #

Tuval’ görüntü verisi için data URI değerini döndüren canvas.toDataURL() yöntemini kullanarak bir tuvali bir görüntü dosyasına kaydedebilirsiniz.

Yöntem iki isteğe bağlı parametre alabilir ‘canvas.toDataURL(type, encoderOptions)’: ’type’ görüntü formatıdır (atlanırsa varsayılan ‘image/png’ olur); “encoderOptions”, görüntü kalitesini belirten 0 ile 1 arasında bir sayıdır (varsayılan 0,92’dir).

Burada bir tuval çiziyoruz ve tuvalin veri URI’sini “myImage.jpg’ye indir” bağlantısına ekliyoruz.

HTML

<canvas id="canvas" width=240 height=240 style="background-color:#808080;">
</canvas>
<p></p>
<a id="download" download="myImage.jpg" href="" onclick="download_img(this);">Download to myImage.jpg</a>

Javascript

var canvas = document.getElementById("canvas");
var ctx = canvas.getContext("2d");
var ox = canvas.width / 2;
var oy = canvas.height / 2;
ctx.font = "42px serif";
ctx.textAlign = "center";
ctx.textBaseline = "middle";
ctx.fillStyle = "#800";
ctx.fillRect(ox / 2, oy / 2, ox, oy);

download_img = function(el) {
  // get image URI from canvas object
  var imageURI = canvas.toDataURL("image/jpg");
  el.href = imageURI;
};
