Sıkıştırmayı Etkinleştir (Apache 2.0+) #

Gzip sıkıştırmasını etkinleştirmek, aktarılan yanıtın boyutunu %90’a kadar azaltabilir, bu da kaynağı indirme süresini önemli ölçüde azaltabilir, istemci için veri kullanımını azaltabilir ve sayfalarınızın ilk kez oluşturulma süresini iyileştirebilir. — PageSpeed ​​Insights

Sıkıştırma bununla etkinleştirilebilir:

AddOutputFilterByType DEFLATE "text/html"/
                              "text/plain"/
                              "text/xml"/
                              "text/css"/
                              "text/javascript"/
                              "application/javascript"

Tarayıcı Önbelleğe Almadan Yararlanın (Apache 2.0+) #

Kaynakların ağ üzerinden alınması hem yavaş hem de pahalıdır: indirme işlemi, istemci ve sunucu arasında birden fazla gidiş-dönüş gerektirebilir, bu da işlemeyi geciktirebilir ve sayfa içeriğinin oluşturulmasını engelleyebilir ve ayrıca ziyaretçi için veri maliyetlerine neden olabilir. Tüm sunucu yanıtları, istemcinin önceden getirilen bir yanıtı yeniden kullanıp kullanamayacağını ve ne zaman kullanabileceğini belirlemesine yardımcı olmak için bir önbelleğe alma ilkesi belirtmelidir. — PageSpeed ​​Insights

Tarayıcı önbelleğinden şu şekilde yararlanabilirsiniz:

# Enable browser caching
ExpiresActive On

# Set the default caching duration
ExpiresDefault "access plus 1 week"

# Change the caching duration by file type
ExpiresByType text/html "access plus 2 weeks"

KeepAlive’ı Etkinleştir (Apache 2.0+) #

HTTP/1.0’ın Keep-Alive uzantısı ve HTTP/1.1’in kalıcı bağlantı özelliği, aynı TCP bağlantısı üzerinden birden çok isteğin gönderilmesine izin veren uzun ömürlü HTTP oturumları sağlar. Bazı durumlarda bunun, birçok resim içeren HTML belgeleri için gecikme sürelerinde neredeyse %50’lik bir hızlanma ile sonuçlandığı gösterilmiştir. Keep-Alive bağlantılarını etkinleştirmek için KeepAlive’ı Açık olarak ayarlayın. — Apache Belgeleri

# Enable KeepAlive
KeepAlive On

# OPTIONAL — limit the amount of requests per connection with 'MaxKeepAliveRequests'
# Example: MaxKeepAliveRequests 500

# OPTIONAL — limit the amount of time the server will wait before it closes 
# the connection with 'KeepAliveTimeout'
# Example: KeepAliveTimeout 500
