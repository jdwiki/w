Java Programınızı Netbeans IDE ile Çalıştırın #

Yani, kurulumu tamamladınız, şimdi programı onunla çalıştırmanın zamanı geldi. Netbeans, masaüstünüze bir kısayol oluşturdu, IDE’yi açmak için “Çift tıklayın”.

Yeni bir proje oluşturmak için

1. Bu düğmesini tıklayın VEYA

  • Git: dosyalar » Yeni Proje, ardından bu pencere açılır

Yukarıdaki resimde gösterildiği gibi kategorilerden (sol taraftan) Java ve projelerden (sağ taraftan) Java Uygulaması seçin.

2. İleri > düğmesine tıklayın, bu pencere açılır

Uygun bir “Proje Adı” sağlayın Projeye istediğiniz herhangi bir ad verebilirsiniz Ama işinizle ilgili olarak adlandırmak iyidir. Gözat… düğmesini tıklayarak proje klasörünüzün konumunu değiştirebilirsiniz. Netbeans’in sizin için otomatik olarak bir “sınıf” oluşturmasını istiyorsanız, “Ana Sınıf Oluştur” onay kutusunu işaretli tutun.

3. Bitir düğmesini tıklayın.

Bundan sonra IDE sizin için kod düzenleyici penceresini açacaktır. Mutlaka 3 şey yaratacaktır:

  1. paket adı
  2. sınıf adı
  3. ana yöntem

Böyle olmalı

package your_package_name;

public class your_class_name {
   
    public static void main(String[] args) {
        // TODO code application logic here
    }
    
}

Şu anda bunu dikkate almayan bazı yorumlar olabilir.

4. Şimdi değiştirin

     // TODO code application logic here

Bununla birlikte

  System.out.println("Welcome New Programmer");

5. Şimdi kod düzenleyicide “sağ tıklayın” ve Dosyayı Çalıştır‘ı seçin VEYA shift+f6 tuşlarına basın, yazdıracaktır

"Welcome New Programmer"

Gerekli Yazılım #

NetBeans IDE’yi yüklemek için Java SE Geliştirme Kiti (JDK) 7 Güncelleme 10 (veya üstü) veya JDK 8 gereklidir. JDK 7 ve JDK 8’in en son güncellemesini http://www.oracle.com/technetwork/java/javase/downloads adresinden indirebilirsiniz.

PHP ve C/C++ NetBeans paketleri yalnızca Java Runtime Environment (JRE) 7 veya 8’in kurulup çalıştırılmasını gerektirir. Java özelliklerinden herhangi birini kullanmayı planlıyorsanız JDK 7 veya 8 gereklidir.

NetBeans IDE 8.1’deki JavaFX 2.2 (veya daha yenisi) özelliklerini kullanmak için JDK 7 Güncelleme 6 (veya daha yenisi) gereklidir.

NetBeans IDE 8.1’de JavaFX 8 özelliklerini kullanmak için JDK 8 gereklidir.

Not:

  • JavaFX 2.2, Windows, Mac ve Linux için en son JDK 7 ile birlikte gelir.

  • JavaFX 8, Windows, Mac ve Linux için JDK 8 ile birlikte gelir.

  • NetBeans IDE’nin önceki sürümlerinin aksine, NetBeans IDE 8.1, IDE’de JavaFX desteğini kullanmak için “FX etkin” bir Java platformu kurmanızı gerektirmez. JDK 7 Güncelleme 6 (veya daha yenisi) veya JDK 8 olan herhangi bir standart Java platformu kurarsanız, IDE’de JavaFX projeleri geliştirebilirsiniz.

  • NetBeans IDE’de JavaFX özelliklerini kullanmak için, JDK sürümünüzü JDK 7 Güncelleme 10’a (veya daha yenisine) yükseltmenizi veya JDK 8’i kullanmanızı şiddetle tavsiye ederiz.

Kurulumunuzu Özelleştirme #

Tüm indirme seçeneğini indirirseniz, tam olarak hangi araçları ve çalışma zamanlarını yükleyeceğinizi seçebilirsiniz. Yükleyicinin Hoş Geldiniz sayfasında Özelleştir’e tıklayın ve kullanmak istediğiniz özellikleri seçin. Daha fazla bilgi için bkz. Yazılımı Yükleme.

NetBeans IDE yapılandırmanıza işlevsellik eklemek istiyorsanız, NetBeans Eklenti Yöneticisini kullanın. Örneğin, PHP paketini indirip kurarak başladığınızı varsayalım. Daha sonra C/C++ işlevselliğini denemek istediğinize karar verirsiniz. Bunu yapmak için IDE’den NetBeans Eklenti Yöneticisine gidin (Araçlar > Eklentiler’i seçin) ve mevcut kurulumunuza C/C++ paketini ekleyin.

Bazı NetBeans IDE özelliklerinin kurulumu ve yapılandırılması hakkında birkaç önemli not vardır:

  • Glassfish, Java Enterprise Edition’ın (EE) referans uygulaması olarak NetBeans IDE ile sağlanır. Üretim ortamlarında kullanılmak üzere tasarlanmamıştır.

  • Uygulama geliştiricileri, referans sürüm olarak Glassfish kullanılarak geliştirilen tüm uygulamaların güncel, güvenli kriptografik protokoller ve algoritmalar kullanmasını ve bu uygulamaları üretimde dağıtmadan önce buna göre test edilmesini sağlamalıdır.

Not, NetBeans IDE ile sağlanan Glassfish sürümünün güvenlik düzeltmeleriyle güncel olmadığını ve kolaylık sağlamak için verildiğini unutmayın. Sonuç olarak, NetBeans kullanıcıları, NetBeans IDE’yi dağıtırken Glassfish’in en son sürümüne yükseltmeli ve Glassfish dağıtımları yeni güvenlik düzeltmeleri içerdiğinde Glassfish’in yeni sürümlerine yükseltme yapmalıdır. Glassfish yükseltme talimatları, MOS notu 2073920.1’de verilmektedir.

  • Apache Tomcat 8.0.27, Java EE ve Tüm indirme seçeneklerine dahildir, ancak bu seçeneklerden herhangi birinden varsayılan olarak yüklenmez. Apache Tomcat’i kurmak için aşağıdaki uygun seçeneği seçin:

  • Java EE indirmesinden yükleyiciyi başlatın ve Hoş Geldiniz sayfasında Apache Tomcat 8.0.27 onay kutusunu seçin;

  • Tüm indirmeden yükleyiciyi başlatın ve Kurulumu Özelleştir iletişim kutusunda Apache Tomcat 8.0.27’yi seçin.

  • C/C++ geliştirme için (C/C++ ve Tüm indirmeler için geçerlidir), ek olarak derleyiciler ve araçlar yüklemeniz gerekir.

  • Mobil cihazlar (Android ve iOS) için HTML5 ve JavaScript geliştirme için, hedef geliştirme platformu için ek olarak SDK’ları ve diğer geliştirme araçlarını yüklemeniz gerekir. Daha fazla bilgi için Mobil Tarayıcılar wiki sayfasına bakın.

  • OS X için, NetBeans IDE kurulumunu Tümü ve Java EE indirmelerinden özelleştirebilirsiniz. Java SE, HTML5/JavaScript, PHP ve C/C++ indirmelerinden IDE kurulumu özelleştirilemez. You can also download a platform-independent zip file and run the NetBeans IDE executable file. Note that the GlassFish Server Open Source Edition and Apache Tomcat are only available with platform-specific installers.

NetBeans IDE’yi İndirme #

NetBeans IDE’yi indirmek için NetBeans sitesini ziyaret edin ve işletim sisteminize, Mimariye ve teknolojilere dayalı olarak IDE’nin uygun sürümünü indirin. Aşağıdaki teknolojiler arasından seçim yapabilirsiniz:

  • Java SE. Tüm standart Java SE geliştirme özelliklerinin yanı sıra NetBeans Platform geliştirme ve JavaFX 2.2 SDK (veya daha yenisi) veya JavaFX 8 SDK desteğini destekler.

  • Java EE. Java SE ve Java EE uygulamaları geliştirmeye yönelik araçların yanı sıra NetBeans Platform geliştirme ve JavaFX 2.2 SDK (veya daha yenisi) veya JavaFX 8 SDK desteği sağlar. Bu indirme seçeneği, GlassFish Server Açık Kaynak Sürümü 4.1.1 ve Apache Tomcat 8.0.27 yazılımını da içerir.

  • HTML5/JavaScript. HTML5/Javascript geliştirme için araçlar sağlar. Java Runtime Environment’ı içerir ve ayrı bir Java kurulumu gerektirmez.

  • PHP. PHP 5.x geliştirme, Zend ve Symfony Framework desteği için araçlar sağlar. Java Runtime Environment’ı içerir ve ayrı bir Java kurulumu gerektirmez.

  • C/C++. C, C++, Qt, Fortran ve Assembly dillerinde geliştirmeyi destekler. Java Runtime Environment’ı içerir ve ayrı bir Java kurulumu gerektirmez.

  • Herşey. Bu, NetBeans IDE için kullanılabilen tüm çalışma zamanlarını ve teknolojileri içeren tam bir indirme seçeneğidir.

Yükleyicinin boyutu, işletim sistemine ve ayrıca seçilen teknolojilere göre değişir. Java JDK’nın bilgisayarınıza yüklenmesi gerekir. Bir seçenek olarak, NetBeans IDE ile birlikte verilen JDK’yı da indirebilirsiniz.

Kurulumdan sonra, IDE’nin Güncelleme Merkezi’nde teknolojileri de kaldırabilir veya ekleyebilirsiniz.

Kurulum: Microsoft Windows ve Linux #

Yazılımı yüklemek için:

  1. İndirme işlemi tamamlandıktan sonra yükleyiciyi çalıştırın.

    • For Windows, the installer executable file has the .exe extension. Double-click the installer file to run it.

    • For Linux platforms, the installer file has the .sh extension. For these platforms, you need to make the installer files executable by using the following command: chmod +x . Type ./ to run the installer.

  2. Kurulum sihirbazının Hoş Geldiniz sayfasında aşağıdaki adımları gerçekleştirin:

    1. Click Customize. In the Customize Installation dialog box, make your selections.
    2. Click OK.
    3. At the Welcome page of the installation wizard, click Next.

  3. Lisans sözleşmesi sayfasında, lisans sözleşmesini gözden geçirin, kabul onay kutusuna tıklayın ve İleri’ye tıklayın.

  4. NetBeans IDE kurulum sayfasında aşağıdakileri yapın:

    1. Accept the default installation directory for the NetBeans IDE or specify another directory.
    **Note**: The installation directory must be empty and the user profile you are using to run the installer must have read/write permissions for this directory.
2. (Applicable only to All or Java EE bundle.) Accept the default JDK installation to use with the NetBeans IDE or select a different installation from the drop-down list. If the installation wizard did not find a compatible JDK installation to use with the NetBeans IDE, your JDK is not installed in the default location. In this case, specify the path to an installed JDK and click Next, or cancel the current installation. After installing the required JDK version you can restart the installation.

**Note**: If the JDK version is older than the recommended JDK 7 Update 10, download and install the latest JDK update from Java SE Downloads page and restart the NetBeans IDE installer.

  5. GlassFish Server Açık Kaynak Sürümü 4.1.1 kurulum sayfası açılırsa, varsayılan kurulum dizinini kabul edin veya başka bir kurulum konumu belirtin.

  6. Apache Tomcat’i kuruyorsanız, kurulum sayfasında varsayılan kurulum dizinini kabul edin veya başka bir kurulum konumu belirtin. Sonrakine tıkla.

    • At the Summary page, do the following: Verify that the list of components to be installed is correct. Select the Check for Updates check box if you want to check the Update Center for possible updates.
    • Verify that you have adequate space on your system for the installation. Click Install to begin the installation.
    • At the Setup Complete page, provide anonymous usage data if desired, and click Finish.

Not: Yazılım kurulumunu başarıyla tamamlarken sorunlarla karşılaşırsanız, kurulum sürecini etkileyebilecek çözülmemiş sorunlar için açıklamalar ve önerilen geçici çözümler için Sorun Giderme bölümüne bakın.

Kurulum: OS X #

Yazılımı yüklemek için:

  1. İndirme işlemi tamamlandıktan sonra yükleyiciyi çalıştırın. Yükleyici dosyası .dmg uzantısına sahiptir.
  2. Açılan panelde paket simgesine çift tıklayın. Paket .pkg uzantısına sahiptir. Kurulum sihirbazı başlar.
  3. “Bu paket, yazılımın yüklenip yüklenemeyeceğini belirlemek için bir program çalıştıracak” olduğunda Devam’ı tıklayın. iletişim kutusu görüntülenir.
  4. Kurulum sihirbazının Giriş sayfasında Devam’a tıklayın.

Not: JDK sürümü önerilen JDK 7 Güncelleme 10’dan daha eskiyse, Java SE İndirmeleri sayfasından en son JDK güncellemesini indirip yükleyin ve NetBeans IDE yükleyicisini yeniden başlatın. 5. Lisans sözleşmesini gözden geçirin ve Devam’a tıklayın. Lisansı kabul etmek için açılır pencerede Kabul Et’e tıklayın. 6. Bir Hedef Seçin sayfasında, sürücüyü seçin ve Devam’a tıklayın. 7. All veya Java EE paketini indirdiyseniz, kurulumunuzu özelleştirebilirsiniz. Son kurulum sihirbazı panelinde, panelin sol alt tarafındaki Özelleştir düğmesine basın. Ürün ağacı görüntülenir. 8. Yüklemek istediğiniz ürünleri seçin. 9. Sisteminiz için yöneticinin adını ve parolasını girin ve kurulumu başlatmak için Tamam’a tıklayın.

Kurulum: Platformdan Bağımsız Paket #

Yazılımı çalıştırmak için:

  1. Platformdan bağımsız ZIP dosyasının indirilmesi tamamlandıktan sonra, onu sisteminizdeki herhangi bir klasöre çıkarın.
  2. netbeans/bin dizininde bulunan yürütülebilir dosyayı çalıştırın.
  3. Lisans Sözleşmesini kabul edin.
  4. İleri’ye tıklayın. NetBeans IDE kurulumu başlar.

Not: Uyumlu bir JDK kurulumu bulunamazsa, aşağıdakileri yaparak JDK kurulum dizinine manuel olarak bir yol eklemeniz gerekebilir:

  1. netbeans/etc dizininde bulunan netbeans.conf dosyasını bir metin düzenleyicide açın.
  2. netbeans_jdkhome seçeneği için uyumlu bir JDK kurulumunun konumunu girin. Windows’ta varsayılan konum C:\Program Files\Java\jdk1.7.0_67 veya benzeridir.
  3. netbeans.conf dosyasını kaydedin ve yürütülebilir dosyayı netbeans/bin dizininde çalıştırın.

Not: GlassFish Server Açık Kaynak Sürümü ve Apache Tomcat, yalnızca platforma özel yükleyicilerle kullanılabilir.

Not: Makinenizde zaten yüklü olan GlassFish Server Açık Kaynak Sürümü ve/veya Apache Tomcat’i kullanmak için NetBeans IDE’de Araçlar menüsünde Sunucular’ı tıklayın. Bu, gerekli web sunucusunun bir örneğini NetBeans IDE ile kaydetmeye yardımcı olan Sunucu Örneği Ekleme Sihirbazı’nı görüntüler.
