Visual Studio Projelerinde Json.Net Nasıl Kurulur #

Json.Net’i Visual Studio Projenize 2 yoldan biriyle yükleyebilirsiniz.

Paket Yöneticisi Konsolunu kullanarak Json.Net’i kurun. #

  1. Hızlı Başlat kutusuna paket yöneticisi konsolu yazıp seçerek Visual Studio’da Paket Yöneticisi Konsolu penceresini açın ve seçin

veya Görünüm -> Diğer Pencereler -> Paket Yöneticisi Konsolu’nu tıklayarak.

  1. Paket Yönetici Konsolu görünür hale geldiğinde, Varsayılan Proje açılır menüsünden seçerek, Json.Net’i yüklemek istediğiniz projeyi çözümünüz içinde seçin.

  1. Şimdi aşağıdaki komutu yazın ve enter tuşuna basın.

“Yükleme Paketi Newtonsoft.Json”

  1. Bu, Json.Net’in en son sürümünü yükleyecektir. Paket Yöneticisi Konsolunda kurulum ilerlemesini göreceksiniz. Başarılı olursa, seçtiğiniz Projeye başarıyla yüklendiği mesajını alacaksınız.

  1. Başarılı bir şekilde kurulduktan sonra, Json.Net Assembly’yi seçtiğiniz Projedeki referanslarda görebileceksiniz. Json.Net derlemesinin adı ‘Newtonsoft.Json’

Json.Net artık projenizde kullanıma hazır!

Visual Studio Solution Explorer’ı kullanarak Json.Net’i kurun. #

json.net’i Visual Studio’daki Çözüm Gezgini’ni kullanarak da yükleyebilirsiniz.

  1. Projenizdeki Referanslar düğümüne sağ tıklayın ve Nuget Paketlerini Yönet… seçeneğine tıklayın.

  2. Nuget Paket Yöneticisi İletişim kutusunda, sol bölmede Çevrimiçi’nin seçili olduğundan emin olun. Sağ üstteki arama kutusuna Json.Net yazın. Bu, ortadaki arama sonuçları bölmesinde Json.Net Nuget Paketini görüntüler.

  3. Yükle düğmesini tıklayın.

  1. Açılan ilerleme penceresinde kurulum ilerlemesini göreceksiniz.

  1. Kurulum tamamlandığında, Nuget Paket Yöneticisi iletişim kutusunda Json.Net paketinin yanında yeşil bir onay işareti göreceksiniz.

  1. Seçtiğiniz proje için çözüm gezgininizdeki Referanslar düğümünde Newtonsoft.Json Derlemesini de göreceksiniz.

Bu, Json.Net kurulumunu tamamlar. Artık json verileri üzerinde çeşitli işlemleri gerçekleştirmek için projenizde kullanmaya hazırsınız.

C#’ta Json.Net kullanarak JSON verilerini Nesneye Serileştirme #

Aşağıdaki örnek, bir Nesneyi (yani bir sınıfın örneğini) içeren bir JSON dizesini nasıl seri durumdan çıkarabileceğinizi gösterir.

using System;
using System.Collections.Generic;
using Newtonsoft.Json;
                    
public class Program
{
    public class Employee
    {
        public string FirstName { get; set; }
        public string LastName { get; set; }
        public bool IsManager { get; set; }
        public DateTime JoinedDate { get; set; }
        public IList<string> Titles { get; set; }
    }
    
    public static void Main()
    {
        string json = @"{
                          'FirstName': 'Shiva',
                          'LastName': 'Kumar',
                          'IsManager': true,
                          'JoinedDate': '2014-02-10T00:00:00Z',
                          'Titles': [
                            'Sr. Software Engineer',
                            'Applications Architect'
                          ]
                        }";
        
        Employee employee = JsonConvert.DeserializeObject<Employee>(json);

        Console.WriteLine(employee.FirstName);
        Console.WriteLine(employee.LastName);
        Console.WriteLine(employee.JoinedDate);
        foreach (string title in employee.Titles)
        {
            Console.WriteLine("  {0}", title);    
        }
    }
}

Bu konsol programını çalıştırırsanız çeşitli Console.WriteLine deyimlerinin çıktısı aşağıdaki gibi olacaktır.

Shiva
Kumar
2/10/2014 12:00:00 AM
  Sr. Software Engineer
  Applications Architect

Dikkat edilecek birkaç şey

  1. Aşağıdaki satır, json dizesindeki verilerin “Employee” sınıfının “employee” nesne örneğine fiilen seri durumdan çıkarılmasını gerçekleştirir.

Çalışan çalışan = JsonConvert.DeserializeObject<Employee>(json);

  1. ’employee.Titles’ bir ‘Liste’ türü olduğundan, o ‘Liste’deki her bir öğe arasında dolaşmak için ‘foreach’ döngü yapısını kullanırız.

C#’ta Json.Net kullanarak bir Nesneyi JSON’a Serileştirme #

Aşağıdaki örnek, bir C# Nesnesinin örneğindeki verileri JSON dizesine seri hale getirmek için Json.Net’i nasıl kullanabileceğinizi gösterir.

using System;
using System.Collections.Generic;
using Newtonsoft.Json;

public class Employee
{
    public string FirstName { get; set; }
    public string LastName { get; set; }
    public bool IsManager { get; set; }
    public DateTime JoinedDate { get; set; }
    public IList<string> Titles { get; set; }
}
                    
public class Program
{
    public static void Main()
    {
        Employee employee = new Employee
        {
            FirstName = "Shiva",
            LastName = "Kumar",            
            IsManager = true,
            JoinedDate = new DateTime(2013, 1, 20, 0, 0, 0, DateTimeKind.Utc),
            Titles = new List<string>
            {
                "Sr. Software Engineer",
                "Applications Architect"
            }
        };

        string json = JsonConvert.SerializeObject(employee, Formatting.Indented);


Console.WriteLine(json);
        
    }
}

Bu konsol programını çalıştırırsanız, Console.WriteLine(json) çıktısı olacaktır.

{
  "FirstName": "Shiva",
  "LastName": "Kumar",
  "IsManager": true,
  "JoinedDate": "2013-01-20T00:00:00Z",
  "Titles": [
    "Sr. Software Engineer",
    "Applications Architect"
  ]
}

Dikkat edilecek birkaç şey

  1. Aşağıdaki satır, “employee” sınıfı örneğinin içindeki verilerin bir json dizesine gerçek serileştirmesini gerçekleştirir

    string json = JsonConvert.SerializeObject(employee, Formatting.Indented);

  2. ‘Formatting.Indented’ parametresi, Json.Net’e verileri girinti ve yeni satırlarla serileştirmesini söyler. Bunu yapmazsanız, serileştirilmiş dize, girinti veya satır sonu olmayan uzun bir dize olacaktır.
