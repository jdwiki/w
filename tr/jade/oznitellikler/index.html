&Öznitellikler #

“ve öznitelikler” olarak telaffuz edilen “& öznitelikler” sözdizimi, bir nesneyi bir öğenin özniteliklerine bölmek için kullanılabilir.

Kod: #

div#foo(data-bar="foo")&attributes({'data-foo': 'bar'})

Sonuç: #

<div id="foo" data-bar="foo" data-foo="bar"></div>

Nesnenin bir nesne değişmezi olması gerekmez. Ayrıca değeri olarak bir nesneye sahip bir değişken de olabilir (ayrıca bkz. Mixin Attributes)

Kod: #

- var attributes = {'data-foo': 'bar'};
div#foo(data-bar="foo")&attributes(attributes)

Sonuç: #

<div id="foo" data-bar="foo" data-foo="bar"></div>

Normal Nitelikler #

HTML Etiketi: #

Etiket özellikleri html’ye benzer, ancak değerleri yalnızca normal JavaScript’tir.

a(href='google.com') Google
a(class='button', href='google.com') Google

Sonuç: #

<a href="google.com">Google</a><a href="google.com" class="button">Google</a>

Değişken: #

Tüm normal JavaScript ifadeleri de iyi çalışıyor:

- var authenticated = true
body(class=authenticated ? 'authed' : 'anon')

Sonuç: #

<body class="authed"></body>

Birçok Özellik #

Çok sayıda özniteliğiniz varsa, bunları birçok satıra da yayabilirsiniz:

input(
  type='checkbox'
  name='agreement'
  checked
)

Sonuç: #

<input type="onay kutusu" name="anlaşma" kontrol edildi="kontrol edildi"/>

Çıkış Yapılmamış Nitelikler #

Varsayılan olarak, siteler arası komut dosyası çalıştırma gibi saldırıları önlemek için tüm öznitelikler çıkışlıdır (özel karakterleri kaçış dizileriyle değiştirir). Özel karakterler kullanmanız gerekiyorsa, = yerine != kullanabilirsiniz.

Kod: #

div(escaped="<code>")
div(unescaped!="<code>")

Sonuç: #

<div escaped="&lt;code&gt;"></div>
<div unescaped="<code>"></div>

Boole Nitelikleri #

Boole nitelikleri Jade tarafından yansıtılır ve “doğru” veya “yanlış” olarak da bilinen boolleri kabul eder. Hiçbir değer belirtilmediğinde true kabul edilir.

Kod: #

input(type='checkbox', checked)
input(type='checkbox', checked=true)
input(type='checkbox', checked=false)
input(type='checkbox', checked=true.toString())

Sonuç: #

<input type="checkbox" checked="checked"/>
<input type="checkbox" checked="checked"/>
<input type="checkbox"/>
<input type="checkbox" checked="true"/>

Doküman türü ‘html’ ise, jade özniteliği yansıtmaması gerektiğini bilir ve özlü stili kullanır (tüm tarayıcılar tarafından anlaşılır).

Kod: #

doctype html
input(type='checkbox', checked)
input(type='checkbox', checked=true)
input(type='checkbox', checked=false)
input(type='checkbox', checked=true && 'checked')

Sonuç: #

<!DOCTYPE html>
<input type="checkbox" checked>
<input type="checkbox" checked>
<input type="checkbox">
<input type="checkbox" checked="checked">

Stil Özellikleri #

‘style’ niteliği bir dizge olabilir (herhangi bir normal nitelik gibi) ama aynı zamanda bir nesne de olabilir, bu da stilin bölümleri JavaScript tarafından oluşturulduğunda kullanışlıdır.

Kod: #

a(style={renk: 'kırmızı', arka plan: 'yeşil'})

Sonuç: #

<a style="color:red;background:green"></a>

Sınıf Nitelikleri #

‘class’ niteliği bir dizge olabilir (herhangi bir normal nitelik gibi), ancak JavaScript’ten oluşturulduğunda kullanışlı olan bir dizi sınıf adı da olabilir.

Kod: #

- var classes = ['foo', 'bar', 'baz']
a(class=classes)
//- the class attribute may also be repeated to merge arrays
a.bing(class=classes class=['bing'])

Sonuç: #

<a class="foo bar baz"></a><a class="bing foo bar baz bing"></a>

Ayrıca, koşullu sınıfları uygulamak için yararlı olan, sınıf adlarını doğru veya yanlış değerlere eşleyen bir nesne olabilir.

Kod: #

- var currentUrl = '/about'
a(class={active: currentUrl === '/'} href='/') Home
a(class={active: currentUrl === '/about'} href='/about'

Sonuç: #

<a href="/">Ana Sayfa</a><a href="/about" class="active">Hakkında</a>

Sınıf Değişmezi #

Sınıflar bir “.classname” sözdizimi kullanılarak tanımlanabilir:

Kod: #

“a.düğmesi”

Sonuç: #

<a class="button"></a>

Div’ler çok yaygın bir etiket seçimi olduğundan, etiket adını çıkarmazsanız varsayılandır:

Kod: #

.içerik

Sonuç: #

<div class="content"></div>

Kimlik Değişmezi #

Kimlikler bir “#idname” sözdizimi kullanılarak tanımlanabilir:

Kod: #

“bir#ana bağlantı”

Sonuç: #

<a id="main-link"></a>

Div’ler çok yaygın bir etiket seçimi olduğundan, etiket adını çıkarmazsanız varsayılandır:

Kod: #

“#içerik”

Sonuç: #

<div id="content"></div>
