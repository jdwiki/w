Maksimum karı elde etmek için Çubuğu kesmek #

n inç uzunluğunda bir çubuk ve n’den küçük tüm parçaların fiyatlarını içeren bir dizi m fiyat verildi. Çubuğu kesip parçaları satarak elde edilebilecek maksimum değeri bulmalıyız. Örneğin, çubuğun uzunluğu 8 ise ve farklı parçaların değerleri aşağıdaki gibi veriliyorsa, elde edilebilecek maksimum değer 22‘dir.

       +---+---+---+---+---+---+---+---+
 (price)| 1 | 5 | 8 | 9 | 10| 17| 17| 20|
        +---+---+---+---+---+---+---+---+

Bir 2B dizi dp[m][n + 1] kullanacağız, burada n, çubuğun uzunluğu ve m, fiyat dizisinin uzunluğudur. Örneğimiz için dp[8][9]‘a ihtiyacımız olacak. Burada dp[i][j], j uzunluğundaki çubuğu satarak maksimum fiyatı gösterecektir. Bir bütün olarak maksimum j uzunluğunu elde edebilir veya karı maksimize etmek için uzunluğu kırabilirdik.

İlk başta, 0. sütun için hiçbir katkı sağlamayacaktır, bu nedenle tüm değerleri 0 olarak işaretlemek. Dolayısıyla 0. sütunun tüm değerleri 0 olacaktır. dp[0][1] için maksimum değer nedir? 1 uzunluğundaki çubuğu satarak elde edebiliriz. Bu 1 olacaktır. Benzer şekilde 2 dp[0][2] uzunluğundaki çubuk için de 2(1+1) alabiliriz. Bu, **dp[0][8]’e kadar devam eder. **.Yani ilk yinelemeden sonra dp[][] dizimiz şöyle görünecek.

Sonucu dp[n][m+1]‘de alacağız.

Java’da uygulama

public int getMaximumPrice(int price[],int n){
        int arr[][] = new int[n][price.length+1];
        
        for(int i=0;i<n;i++){
            for(int j=0;j<price.length+1;j++){
                if(j==0 || i==0)
                    arr[i][j] = 0;
                else if(j>=i){
                    arr[i][j] = Math.max(arr[i-1][j], price[i-1]+arr[i][j-i]);
                }else{
                    arr[i][j] = arr[i-1][j];
                }
            }
        }
        return arr[n-1][price.length];
    }
