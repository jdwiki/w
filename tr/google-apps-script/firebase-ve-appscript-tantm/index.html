Firebase Veritabanında Veri Okumak ve Yazmak için Firebase’i Google AppScript ile entegre edin.

Firebase, mobil, masaüstü ve tablet cihazlarda uygulamalar oluşturmaya ve barındırmaya yardımcı olmak için gerçek zamanlı veritabanı kullanan Google’ın bir NoSQL veritabanı sistemidir. NoSQL veritabanları, verileri yapılandırılmış biçimde depolamak için JSON nesnelerini kullanır.

GAS’ta bir Firebase projesine bağlanma ve Google E-Tablodan Firebase’e veri aktarma #

Firebase kaynağını AppScript’e yükleyin #

  • Bunu yapmak için Kaynaklar’a ve ardından Kitaplıklar’a tıklayın.
  • Firebase, AppScript’e yüklenmesi gereken benzersiz bir proje kitaplığı anahtarına sahiptir.
  • Kitaplıklar’a tıklayın The following pop-up appears. Enter the following project key in the textbox. MYeP8ZEEt1ylVDxS7uyg9plDOcoke7-2l This is the project library key for Firebase.
  • Şimdi sürümde kararlı genel sürüm sürümünü seçin.
  • Kaydet’e tıklayın. Artık Firebase, çalışmanız için AppScript’inize başarıyla yüklenmiştir.

Şimdi Firebase’den veri okumak ve yazmak için bir örnek alalım. #

  • Şimdi Google E-Tablolar’da tasarlanmış örnek bir tablo alıyoruz.

  • Şimdi, tablolardaki bu tabloyu kullanarak veritabanını Firebase’de oluşturmak için. AppScript’e aşağıdaki kodu ekleyin.

      function writeDataToFirebase() {
    var ss = SpreadsheetApp.openById("1LACsj0s3syAa9gvORdRWBhJ_YcXHybjQfHPgw3TLQ6g");
    var sheet = ss.getSheets()[0];
    var data = sheet.getDataRange().getValues();
    var dataToImport = {};
    for(var i = 1; i < data.length; i++) {
      var firstName = data[i][0];
      var lastName = data[i][1];
      dataToImport[firstName + '-' + lastName] = {
        firstName:firstName,
        lastName:lastName,
        emailAddress:data[i][2],
        semester:data[i][4],
        department:data[i][5],
      };
    }
    var firebaseUrl = "https://example-app.firebaseio.com/";
    var secret = "secret-key";
    var base = FirebaseApp.getDatabaseByUrl(firebaseUrl, secret);
    base.setData("", dataToImport);
  }

Elektronik tablo kimliğini ve firebaseURL’yi ve gizli anahtarı değiştirin.

firebaseURL ve gizli anahtar nasıl bulunur? #

  • Firebase Kontrol Panelinize gidin ve sol üst köşedeki ayarlar dişli kutusuna tıklayın. Proje Ayarları’na tıklayın.
  • VeritabanıURL’sini bulabileceğiniz Hizmet Hesapları bölümüne gidin. Bu, firebaseURL işlevi görür.
  • Şimdi Veritabanı Sırları sekmesine tıklayın ve gizli anahtarı bulabilirsiniz.

Artık firebaseURL’yi ve gizli anahtarı eklediniz. Artık gitmeye hazırsınız. AppScript motorunda çalıştırma kodunu tıklayın. #

  • Çalıştırdığınızda ilk kez İzinleri gözden geçirmenizi isteyecektir.
  • İzinleri Gözden Geçir ve İzin Ver’e tıklayın.
  • Artık fonksiyonunuzu çalıştırıyorsunuz ve Firebase Veritabanında oluşturulan tabloyu görebilirsiniz.

Veritabanını görmek için Firebase kontrol paneline gidin ve Veritabanına tıklayarak veritabanını görüntüleyebilirsiniz.

Okuma ve yazma uygulamak için birkaç işlev daha. #

1. Bağlantının çalışıp çalışmadığını test etmek için basit bir veri yazmak. #

function myFunction(){
 var firebaseUrl = "https://example-app.firebaseio.com/";
 var secret = "secret-key";
 var base = FirebaseApp.getDatabaseByUrl(firebaseUrl, secret);
 base.setData("test", "Hello Firebase"); 
}

2. Tüm Verileri okumak için #

function getAllData() {
         var firebaseUrl = "https://example-app.firebaseio.com/";
         var secret = "secret-key";
var base = FirebaseApp.getDatabaseByUrl(firebaseUrl, secret);
 var data = base.getData();
          for(var i in data) {
            Logger.log(data[i].firstName + ' ' + data[i].lastName);
          }
}

Okunan veriler Günlüklerde görüntülenir. Günlükleri kontrol etmek için Görünüm → Günlükler’e tıklayın veya sadece Control + Enter tuşlarını kullanın.

3. Belirli bir kaydı okumak için #

function getContact() {
  var firebaseUrl = "https://example-app.firebaseio.com/";
  var secret = "secret-key";
  var base = FirebaseApp.getDatabaseByUrl(firebaseUrl, secret);
  var contact = base.getData("Yash-Udasi");
  Logger.log(contact);
}

Okunan veriler Günlüklerde görüntülenir. Günlükleri kontrol etmek için Görünüm → Günlükler’e tıklayın veya sadece Control + Enter tuşlarını kullanın.

4. Belirli bir kaydı güncellemek için. #

function updateData() {
  var firebaseUrl = "https://example-app.firebaseio.com/";
  var secret = "secret-key";
  var base = FirebaseApp.getDatabaseByUrl(firebaseUrl, secret);
  base.updateData("Yash-Udasi/emailAddress", "yash.udasi@fyuff.com");
}
