Kutular Rust’ın çok önemli bir parçasıdır ve her Rustacean bunların ne olduğunu ve nasıl kullanılacağını bilmelidir.

Kutu Oluşturma #

Stabil Rust’ta Box::new fonksiyonunu kullanarak bir Kutu yaratırsınız.

let boxed_int: Box<i32> = Box::new(1);

Kutulu Değerleri Kullanma #

Kutular Deref<Target=T> uyguladığı için, kutulu değerleri içerdikleri değer gibi kullanabilirsiniz.

let boxed_vec = Box::new(vec![1, 2, 3]);
println!("{}", boxed_vec.get(0));

Kutulu bir değer üzerinde desen eşleştirmesi yapmak istiyorsanız, kutunun referansını manuel olarak kaldırmanız gerekebilir.

struct Point {
    x: i32,
    y: i32,
}

let boxed_point = Box::new(Point { x: 0, y: 0});
// Notice the *. That dereferences the boxed value into just the value
match *boxed_point {
    Point {x, y} => println!("Point is at ({}, {})", x, y),
}

Özyinelemeli Numaralandırmalar ve Yapılar Oluşturmak için Kutuları Kullanma #

Box kullanmadan Rust’ta özyinelemeli bir numaralandırma oluşturmaya çalışırsanız, numaralandırmanın boyutlandırılamayacağını söyleyen bir derleme zamanı hatası alırsınız.

// This gives an error!
enum List {
    Nil,
    Cons(i32, List)
}

Numaralandırmanın tanımlanmış bir boyuta sahip olması için, özyinelemeli olarak içerilen değer bir Kutuda olmalıdır.

// This works!
enum List {
    Nil,
    Cons(i32, Box<List>)
}

Bu işe yarar, çünkü Box, T ne olursa olsun her zaman aynı boyuta sahiptir, bu da Rust’ın List’e bir boyut vermesine izin verir.
