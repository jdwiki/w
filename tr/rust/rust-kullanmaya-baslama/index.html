println’in gelişmiş kullanımı! #

println! (ve kardeşi [print!](https://doc.rust-lang. org/std/macro.print!.html)) diğer birçok dilde bulunan “printf” işlev ailesine benzer, dinamik veriler içeren metin üretmek ve yazdırmak için uygun bir mekanizma sağlar. İlk argümanı, diğer argümanların metin olarak nasıl yazdırılacağını belirleyen bir format dizesidir. Biçim dizesi, bir değiştirmenin gerçekleşmesi gerektiğini belirtmek için yer tutucular ("{}" içine alınmış) içerebilir:

// No substitution -- the simplest kind of format string
println!("Hello World");
// Output: Hello World

// The first {} is substituted with a textual representation of
// the first argument following the format string. The second {}
// is substituted with the second argument, and so on.
println!("{} {} {}", "Hello", true, 42);
// Output: Hello true 42

Bu noktada şunu soruyor olabilirsiniz: “println!”, “true” boole değerini “true” dizesi olarak yazdırmayı nasıl bildi? “{}”, gerçekten, değerin Display özelliği kullanılarak metne dönüştürülmesi gerektiği konusunda biçimlendiriciye verilen bir talimattır. . Bu özellik, çoğu ilkel Rust türü (dizeler, sayılar, booleanlar, vb.) için uygulanır ve “kullanıcıya yönelik çıktı” içindir. Bu nedenle, 42 sayısı, dahili olarak bu şekilde depolandığı gibi ikili olarak değil, ondalık olarak 42 olarak yazdırılacaktır.

O halde, “Ekran"ı * uygulamayan* türleri nasıl yazdırırız, örnekler Dilimler (’[i32]’), vektörler (‘Vec’) veya seçeneklerdir (‘Option<&str>’)? Bunların kullanıcıya yönelik net bir metin gösterimi yoktur (yani, bir cümleye önemsiz bir şekilde ekleyebileceğiniz). Bu tür değerlerin yazdırılmasını kolaylaştırmak için Rust ayrıca Debug özelliğine ve ilgili {:?} yer tutucu. Belgelerden: “Debug çıktıyı programcıya yönelik, hata ayıklama bağlamında biçimlendirmelidir.” Hadi bazı örneklere bakalım:

println!("{:?}", vec!["a", "b", "c"]);
// Output: ["a", "b", "c"]

println!("{:?}", Some("fantastic"));
// Output: Some("fantastic")

println!("{:?}", "Hello");
// Output: "Hello"
// Notice the quotation marks around "Hello" that indicate
// that a string was printed.

“Hata ayıklama” ayrıca, iki nokta üst üste işaretinden sonra “#” değiştiricisini kullanarak etkinleştirebileceğiniz yerleşik bir güzel yazdırma mekanizmasına sahiptir:

println!("{:#?}", vec![Some("Hello"), None, Some("World")]);
// Output: [
//    Some(
//        "Hello"
//    ),
//    None,
//    Some(
//        "World"
//    )
// ]

Biçim dizeleri, oldukça karmaşık ikameleri ifade etmenize olanak tanır:

// You can specify the position of arguments using numerical indexes.
println!("{1} {0}", "World", "Hello");
// Output: Hello World

// You can use named arguments with format
println!("{greeting} {who}!", greeting="Hello", who="World");
// Output: Hello World

// You can mix Debug and Display prints:
println!("{greeting} {1:?}, {0}", "and welcome", Some(42), greeting="Hello");
// Output: Hello Some(42), and welcome

println! ve arkadaşlarınız, çalışma zamanında çökmek yerine işe yaramayacak bir şey yapmaya çalışıyorsanız sizi uyaracaktır:

// This does not compile, since we don't use the second argument.
println!("{}", "Hello World", "ignored");

// This does not compile, since we don't give the second argument.
println!("{} {}", "Hello");

// This does not compile, since Option type does not implement Display
println!("{}", Some(42));

Özünde, Rust yazdırma makroları basitçe format! makrosunun etrafındaki sarmalayıcılardır ve bu, birleştirerek bir dize oluşturmaya olanak tanır. farklı veri değerlerinin birlikte metinsel temsilleri. Bu nedenle, yukarıdaki tüm örneklerde, biçimlendirilmiş dizgiyi yazdırmak yerine saklamak için ‘format!’ yerine ‘println!’ kullanabilirsiniz:

let x: String = format!("{} {}", "Hello", 42);
assert_eq!(x, "Hello 42");

Makrolar olmadan konsol çıktısı #

// use Write trait that contains write() function
use std::io::Write;

fn main() {
    std::io::stdout().write(b"Hello, world!\n").unwrap();
}

  • ‘std::io::Write’ özelliği, bayt akışlarını kabul eden nesneler için tasarlanmıştır. Bu durumda, standart çıktı için bir tanıtıcı std::io::stdout() ile elde edilir.

  • Write::write(), bir byte-string değişmezi (b"<string>") ile oluşturulan bir bayt dilimini (&[u8]) kabul eder. “Write::write()”, yazılan bayt sayısını (başarı durumunda) veya hata değerini (başarısızlık durumunda) içeren bir “Sonuç<usize, IoError>” döndürür.

  • ‘Result::unwrap()’ çağrısı, çağrının başarılı olmasının beklendiğini belirtir (‘Result<usize, IoError> ->usize’) ve değer atılır.

Minimal örnek #

Geleneksel Hello World programını Rust’ta yazmak için, aşağıdaki kaynak kodunu içeren “hello.rs” adlı bir metin dosyası oluşturun:

fn main() {
    println!("Hello World!");
}

Bu, “main” adında, parametre almayan ve veri döndürmeyen yeni bir işlevi tanımlar. Bu, programınızın çalıştırıldığında yürütülmeye başladığı yerdir. İçinde, konsola metin yazdıran bir makro olan bir println! var.

İkili bir uygulama oluşturmak için, kaynak dosyanın adını ileterek Rust derleyicisini çağırın:

$ rustc hello.rs

Ortaya çıkan yürütülebilir dosya, ana kaynak modülle aynı ada sahip olacaktır, bu nedenle programı bir Linux veya MacOS sisteminde çalıştırmak için şunu çalıştırın:

$ ./hello
Hello World!

Bir Windows sisteminde şunu çalıştırın:

C:\Rust> hello.exe
Hello World!

Başlarken #

Kurulum #

Rust programlama dilini kullanarak herhangi bir şey yapmadan önce, onu Windows için veya terminalinizi Unix benzeri sistemlerde kullanarak edinmeniz gerekir; burada “$”, terminal:

$ curl https://sh.rustup.rs -sSf | sh

Bu, hangi sistemde olursanız olun, gerekli dosyaları alacak ve Rust’ın en son sürümünü sizin için kuracaktır. Daha fazla bilgi için bkz. proje sayfası.

_Not: Bazı Linux dağıtımları (ör. Arch Linux), bunun yerine kurulabilen bir paket olarak “rustup” sağlar. Ve birçok Unix benzeri sistem ‘rustc’ ve ‘cargo’yu ayrı paketler olarak sağlasa da, çoklu yayın kanallarını yönetmeyi ve çapraz derleme yapmayı çok daha kolay hale getirdiğinden, bunun yerine [‘rustup’] kullanılması önerilir. _

Pas Derleyici #

Artık, UNIX’teyse terminalimizde veya Windows’ta komut isteminde aşağıdaki komutu çalıştırarak Rust‘un bilgisayarlarımıza başarıyla yüklenip yüklenmediğini kontrol edebiliriz:

$ rustc --version

Bu komut başarılı olursa Rust‘un bilgisayarlarımızda kurulu olan derleyicisinin versiyonu gözlerimizin önünde görüntülenecektir.

Kargo #

Rust ile birlikte Rut paketlerinizi ve projelerinizi yönetmek için kullanılan bir yapı aracı olan Cargo gelir. Bunun da bilgisayarınızda mevcut olduğundan emin olmak için, hangi sistemde olduğunuza bağlı olarak terminal veya komut istemine atıfta bulunan konsolda aşağıdakileri çalıştırın:

$ cargo --version

Rust derleyicisinin eşdeğer komutu gibi, bu da Cargo’nun geçerli sürümünü döndürür ve görüntüler.

İlk Kargo projenizi oluşturmak için [Kargo][2]‘ye gidebilirsiniz.

Alternatif olarak, [Minimal örnek][3]’te gösterildiği gibi programları doğrudan “rustc” kullanarak derleyebilirsiniz.

[2]: https://www.wikiod.com/tr/rust/kargo#Yeni proje oluştur [3]: https://www.wikiod.com/tr/rust/rust-kullanmaya-baslama#Minimal örnek
