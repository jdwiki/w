kaset #

tape minimalist JavaScript test çerçevesidir, TAP-uyumlu işaretlemesi verir.

’npm’ run komutunu kullanarak ’tape’ yüklemek için

npm install --save-dev tape @types/tape

TypeScript ile ’tape’ kullanmak için, bu çalıştırma komutunu yapmak için ’ts-node’u global paket olarak kurmanız gerekir.

npm install -g ts-node

Artık ilk testinizi yazmaya hazırsınız.

//math.test.ts
import * as test from "tape";

test("Math test", (t) => {
    t.equal(4, 2 + 2);
    t.true(5 > 2 + 2);

    t.end();
});

Test çalıştırma komutunu yürütmek için

ts-node node_modules/tape/bin/tape math.test.ts

Çıktıda görmelisiniz

TAP version 13
# Math test
ok 1 should be equal
ok 2 should be truthy

1..2
# tests 2
# pass  2

# ok

İyi iş, az önce TypeScript testinizi yaptınız.

Birden fazla test dosyası çalıştırın

Yol joker karakterlerini kullanarak birden çok test dosyasını aynı anda çalıştırabilirsiniz. Tüm TypeScript testlerini ’testler’ dizini çalıştırma komutunda yürütmek için

ts-node node_modules/tape/bin/tape tests/**/*.ts

(ts-is) #

jest, TypeScript kodunu test etmek için ts-jest ile kullanılabilen, Facebook’un ağrısız JavaScript test çerçevesidir.

npm run komutunu kullanarak jest yüklemek için

npm install --save-dev jest @types/jest ts-jest typescript

Kullanım kolaylığı için “jest"i global paket olarak kurun

npm install -g jest

“jest"in TypeScript ile çalışması için “package.json"a yapılandırma eklemeniz gerekir.

//package.json
{
...
"jest": {
    "transform": {
      ".(ts|tsx)": "<rootDir>/node_modules/ts-jest/preprocessor.js"
    },
    "testRegex": "(/__tests__/.*|\\.(test|spec))\\.(ts|tsx|js)$",
    "moduleFileExtensions": ["ts", "tsx", "js"]
  }
}

Artık “jest” hazır. Test etmek için örnek fizz buzumuz olduğunu varsayalım

//fizzBuzz.ts
export function fizzBuzz(n: number): string {
    let output = "";
    for (let i = 1; i <= n; i++) {
        if (i % 5 && i % 3) {
            output += i + ' ';
        }
        if (i % 3 === 0) {
            output += 'Fizz ';
        }
        if (i % 5 === 0) {
            output += 'Buzz ';
        }
    }
    return output;
}

Örnek test şöyle görünebilir

//FizzBuzz.test.ts
/// <reference types="jest" />

import {fizzBuzz} from "./fizzBuzz";
test("FizzBuzz test", () =>{
    expect(fizzBuzz(2)).toBe("1 2 ");
    expect(fizzBuzz(3)).toBe("1 2 Fizz ");
});

Test çalıştırmasını yürütmek için

jest

Çıktıda görmelisiniz

 PASS  ./fizzBuzz.test.ts
  ✓ FizzBuzz test (3ms)

Test Suites: 1 passed, 1 total
Tests:       1 passed, 1 total
Snapshots:   0 total
Time:        1.46s, estimated 2s
Ran all test suites.

Kod kapsamı #

“jest”, kod kapsamı raporlarının oluşturulmasını destekler.

TypeScript ile kod kapsamını kullanmak için “package.json"a başka bir yapılandırma satırı eklemeniz gerekir.

{
...
  "jest": {
  ...
    "testResultsProcessor": "<rootDir>/node_modules/ts-jest/coverageprocessor.js"
  }
}

Kapsam raporu çalıştırma oluşturma ile testleri çalıştırmak için

jest --coverage

Örnek fizz buzz’ımızla birlikte kullanılıyorsa, görmelisiniz

Alsas #

Alsatian, TypeScript ile yazılmış bir birim test çerçevesidir. Test Durumlarının kullanımına izin verir ve TAP uyumlu işaretlemesini çıkarır.

Kullanmak için npmden yükleyin:

npm install alsatian --save-dev

Ardından bir test dosyası oluşturun:

import { Expect, Test, TestCase } from "alsatian";
import { SomeModule } from "../src/some-module";    

export SomeModuleTests {

    @Test()
    public statusShouldBeTrueByDefault() {
        let instance = new SomeModule();
        
        Expect(instance.status).toBe(true);
    }
    
    @Test("Name should be null by default")
    public nameShouldBeNullByDefault() {
        let instance = new SomeModule();
        
        Expect(instance.name).toBe(null);
    }
    
    @TestCase("first name")
    @TestCase("apples")
    public shouldSetNameCorrectly(name: string) {
        let instance = new SomeModule();
        
        instance.setName(name);
        
        Expect(instance.name).toBe(name);
    }
    
}

Tam belgeler için alsatian’ın GitHub deposuna bakın.

chai değişmez eklentisi #

  1. npm chai, chai-immutable ve ts-node’dan yükleyin

    npm install --save-dev chai chai-immutable ts-node

  2. Mocha ve chai için kurulum türleri

    npm install --save-dev @types/mocha @types/chai

  3. Basit bir test dosyası yazın:

      import {List, Set} from 'immutable';
  import * as chai from 'chai';
  import * as chaiImmutable from 'chai-immutable';

  chai.use(chaiImmutable);

  describe('chai immutable example', () => {
    it('example', () => {
      expect(Set.of(1,2,3)).to.not.be.empty;

      expect(Set.of(1,2,3)).to.include(2);
      expect(Set.of(1,2,3)).to.include(5);
    })
  })

  4. Konsolda çalıştırın:

    mocha --compilers ts:ts-node/register,tsx:ts-node/register 'test/**/*.spec.@(ts|tsx)'
