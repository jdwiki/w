TypeScript’i çalıştırmanın ve hata ayıklamanın iki yolu vardır:

JavaScript’e aktar, düğümde çalıştır ve TypeScript kaynak dosyalarına geri bağlanmak için eşlemeleri kullan

veya

ts-node kullanarak TypeScript’i doğrudan çalıştırın

Bu makalede, Visual Studio Code ve WebStorm kullanımının her iki yolu da açıklanmaktadır. Tüm örnekler, ana dosyanızın index.ts olduğunu varsaymaktadır.

WebStorm’da ts-düğümlü TypeScript #

Bu komut dosyasını package.json dosyanıza ekleyin:

"start:idea": "ts-node %NODE_DEBUG_OPTION% --ignore false index.ts",

Komut dosyasına sağ tıklayın ve Create ’test:idea’… öğesini seçin ve hata ayıklama yapılandırmasını oluşturmak için ‘OK’ ile onaylayın:

Bu yapılandırmayı kullanarak hata ayıklayıcıyı başlatın:

Visual Studio Kodunda ts-node ile TypeScript #

TypeScript projenize ts-node ekleyin:

npm i ts-node

package.json dosyanıza bir komut dosyası ekleyin:

"start:debug": "ts-node --inspect=5858 --debug-brk --ignore false index.ts"

“launch.json” dosyasının node2 türünü kullanacak şekilde yapılandırılması ve npm’yi “start:debug” komut dosyasını çalıştırarak başlatması gerekir:

{
    "version": "0.2.0",
    "configurations": [
        {
            "type": "node2",
            "request": "launch",
            "name": "Launch Program",
            "runtimeExecutable": "npm",
            "windows": {
                "runtimeExecutable": "npm.cmd"
            },
            "runtimeArgs": [
                "run-script",
                "start:debug"
            ],
            "cwd": "${workspaceRoot}/server",
            "outFiles": [],
            "port": 5858,
            "sourceMaps": true
        }
    ]
}

Visual Studio Kodunda SourceMaps ile JavaScript #

tsconfig.json kümesinde

"sourceMap": true,

tsc komutunu kullanarak TypeScript kaynaklarından js dosyalarıyla birlikte eşlemeler oluşturmak için.
launch.json dosyası:

{
    "version": "0.2.0",
    "configurations": [
        {
            "type": "node",
            "request": "launch",
            "name": "Launch Program",
            "program": "${workspaceRoot}\\index.js",
            "cwd": "${workspaceRoot}",
            "outFiles": [],
            "sourceMaps": true
        }
    ]
}

Bu, düğümü oluşturulan index.js (ana dosyanız index.ts ise) dosyası ve kesme noktalarında duran ve TypeScript kodunuzdaki değişken değerleri çözen Visual Studio Code’daki hata ayıklayıcı ile başlatır.
