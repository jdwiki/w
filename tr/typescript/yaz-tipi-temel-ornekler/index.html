Bu, genel bir araba sınıfını genişleten ve bir araba tanımlama yöntemini tanımlayan temel bir örnektir.

Burada daha fazla TypeScript örneği bulun - TypeScript Örnekleri GitRepo

Extends ve super anahtar sözcüğünü kullanan 1 temel sınıf kalıtım örneği #

Genel bir Car sınıfı, bazı araba özelliklerine ve bir açıklama yöntemine sahiptir.

class Car{
    name:string;
    engineCapacity:string;

    constructor(name:string,engineCapacity:string){
        this.name = name;
        this.engineCapacity = engineCapacity;
    }

    describeCar(){
        console.log(`${this.name} car comes with ${this.engineCapacity} displacement`);
    }
}

new Car("maruti ciaz","1500cc").describeCar();

HondaCar, mevcut jenerik otomobil sınıfını genişletiyor ve yeni özellikler ekliyor.

class HondaCar extends Car{
    seatingCapacity:number;

    constructor(name:string,engineCapacity:string,seatingCapacity:number){
        super(name,engineCapacity);
        this.seatingCapacity=seatingCapacity;
    }

    describeHondaCar(){
        super.describeCar();
        console.log(`this cars comes with seating capacity of ${this.seatingCapacity}`);
    }
}
new HondaCar("honda jazz","1200cc",4).describeHondaCar();

2 statik sınıf değişkeni örneği - kaç zaman yönteminin çağrıldığını sayın #

burada countInstance statik bir sınıf değişkenidir

class StaticTest{
    static countInstance : number= 0;
    constructor(){
        StaticTest.countInstance++;
    }
}

new StaticTest();
new StaticTest();
console.log(StaticTest.countInstance);
