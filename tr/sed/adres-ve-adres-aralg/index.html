  • Normal ifade kalıbıyla eşleşen satırlar
  • Belirli satır
  • Belirli satır aralığı
  • Hem sayı hem de desen kullanarak aralığı belirleme
  • Adres aralığını reddetme

Sed komutları adresler veya adres aralıkları kullanılarak yalnızca belirli satırlarda çalışacak şekilde belirtilebilir.

Normal ifade kalıbıyla eşleşen satırlar #

$ cat ip.txt 
address
range
substitution
pattern
sample
Add Sub Mul Div

  • Bir desenle eşleşen çizgiler

    $ sed '/add/d' ip.txt 
range
substitution
pattern
sample
Add Sub Mul Div

$ sed -n '/t/p' ip.txt 
substitution
pattern

$ sed -n '/[A-Z]/ s| |/|gp' ip.txt 
Add/Sub/Mul/Div

  • Desen aralığı

    $ sed -n '/add/,/sub/p' ip.txt 
address
range
substitution

$ sed -n '/a/,/e/p' ip.txt 
address
range
pattern
sample

Not

  • İkinci örnekte, iki aralığı eşleştirdi - “1,2” satırları ve “4,5” satırları
  • Yerine diğer karakterlerin nasıl kullanılacağı hakkında [Farklı sınırlayıcılar kullanma](https://www.wikiod.com/tr/sed/duzenli-ifadeler#Farklı sınırlayıcılar kullanma Different-delimiters#t=201611031024036829908) konusuna bakın. / kalıbı belirtmek için

  • Büyük/küçük harfe duyarsız eşleşme

    $ sed -n '/add/Ip' ip.txt 
address
Add Sub Mul Div

$ sed -n '/add/I,/sub/p' ip.txt 
address
range
substitution
Add Sub Mul Div
son sürüm

Belirli satır #

$ cat ip.txt
address
range
substitution
pattern
sample

  • Ninci satır

    $ sed -n '2p' ip.txt 
range

$ sed '3d' ip.txt 
address
range
pattern
sample

  • Son satır

    $ sed -n '$p' ip.txt 
sample

Belirli satır aralığı #

$ cat ip.txt 
address
range
substitution
pattern
sample

  • Belirtilen aralık, bu satır numaralarını içerir

    $ sed -n '2,4p' ip.txt 
range
substitution
pattern

  • $ son satırı belirtmek için kullanılabilir. Açıklık için adres ve komut arasında boşluk kullanılabilir

    $ sed -n '3,$ s/[aeiou]//gp' ip.txt 
sbstttn
pttrn
smpl

  • iinci satırdan i+jinci satıra

    $ sed '2,+2d' ip.txt 
address
sample

  • iinci satır ve i+j, i+2j, i+3j vb.

    $ sed -n '1~2p' ip.txt 
address
substitution
sample
son sürüm

Hem sayı hem de desen kullanarak aralığı belirleme #

$ cat ip.txt 
address
range
substitution
pattern
sample
Add Sub Mul Div

  • Satır numarasından satır eşleştirme desenine

    $ sed -n '2,/pat/p' ip.txt 
range
substitution
pattern

  • Satır numarasına satır eşleştirme deseni

    $ sed '/pat/,$d' ip.txt 
address
range
substitution

  • Satır eşleştirme deseni ve onu takip eden satır sayısı

    $ sed -n '/add/I,+1p' ip.txt 
address
range
Add Sub Mul Div

  • “0”, desen girişin ilk satırıyla eşleştiğinde aralığın sonunu belirtmek için başlangıç ​​satırı numarası olarak kullanılabilir

    $ sed -n '0,/r/p' ip.txt 
address

$ sed -n '1,/r/p' ip.txt 
address
range

$ sed -n '0,/u/p' ip.txt 
address
range
substitution
son sürüm

Adres aralığını reddetme #

$ cat ip.txt 
address
range
substitution
1234
search pattern
sample
Add Sub Mul Div

  • Belirtilen adres dışındaki satırların silinmesi

    $ sed '/[0-9]/!d' ip.txt 
1234

$ sed -n '/[0-9]/p' ip.txt 
1234

$ sed '$!d' ip.txt 
Add Sub Mul Div

$ sed -n '$p' ip.txt 
Add Sub Mul Div

  • Bir desenle eşleşmeyen satırlarda arama yapın ve değiştirin

    $ sed '/ /! s/^/#/' ip.txt 
#address
#range
#substitution
#1234
search pattern
#sample
Add Sub Mul Div

$ sed '/add/,/sub/! s/[aeiou]//gi' ip.txt 
address
range
substitution
1234
srch pttrn
smpl
dd Sb Ml Dv
