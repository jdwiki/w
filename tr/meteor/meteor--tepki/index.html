React, kullanıcı arayüzleri oluşturmak için bir JavaScript kitaplığıdır. Facebook tarafından geliştirilen ve sürdürülen açık kaynak. Meteor, React için üretime hazır desteğe sahiptir.

Kaynaklar:

  • React öğreticisi
  • Meteor + React öğreticisi

Kurulum ve “Merhaba Dünya” #

Projenize React ekleyin:

meteor npm install --save react react-dom react-mounter

Basit bir React bileşenini görüntülemek için ‘client/helloworld.jsx’ dosyasını oluşturun:

import React, { Component } from 'react';
import { mount } from 'react-mounter';

// This component only renders a paragraph containing "Hello World!"
class HelloWorld extends Component {
  render() {
    return <p>Hello World!</p>;
  }
}

// When the client application starts, display the component by mounting it to the DOM.
Meteor.startup(() => {
  mount(HelloWorld);
});

MongoDB koleksiyonunu görüntüleme #

Bu örnek, bir MongoDB koleksiyonunun bir React bileşeninde nasıl görüntülenebileceğini gösterir. Koleksiyon, sunucu ve istemci arasında sürekli olarak senkronize edilir ve veritabanı içeriği değiştikçe sayfa anında güncellenir.

React bileşenlerini ve Meteor koleksiyonlarını bağlamak için ‘react-meteor-data’ paketine ihtiyacınız olacak.

$ meteor add react-meteor-data
$ meteor npm install react-addons-pure-render-mixin

“both/collections.js” içinde basit bir koleksiyon bildirilir. ‘both’ dizinindeki her kaynak dosya hem istemci tarafı hem de sunucu tarafı kodudur:

import { Mongo } from 'meteor/mongo';

// This collection will contain a list of random numbers
export const Numbers = new Mongo.Collection("numbers");

Koleksiyonun sunucuda yayınlanması gerekiyor. “server/publications.js” içinde basit bir yayın oluşturun:

import { Meteor } from 'meteor/meteor';
import { Numbers } from '/both/collections.js';

// This publication synchronizes the entire 'numbers' collection with every subscriber
Meteor.publish("numbers/all", function() {
  return Numbers.find();
});

‘createComponent’ işlevini kullanarak (‘Numbers’ koleksiyonu gibi) reaktif değerleri bir React bileşenine iletebiliriz. “istemci/shownumbers.jsx”:

import React from 'react';
import { createContainer } from 'meteor/react-meteor-data';
import { Numbers } from '/both/collections.js';

// This stateless React component renders its 'numbers' props as a list
function _ShowNumbers({numbers}) {
  return <div>List of numbers:
    <ul>
      // note, that every react element created in this mapping requires 
      // a unique key - we're using the _id auto-generated by mongodb here
      {numbers.map(x => <li key={x._id}>{x.number}</li>)}
    </ul>
  </div>;
}

// Creates the 'ShowNumbers' React component. Subscribes to 'numbers/all' publication,
// and passes the contents of 'Numbers' as a React property.
export const ShowNumbers = createContainer(() => {
  Meteor.subscribe('numbers/all');
  return {
    numbers: Numbers.find().fetch(),
  };
}, _ShowNumbers);

Başlangıçta veritabanı muhtemelen boştur.

MongoDB’ye girişler ekleyin ve sayfanın otomatik olarak güncellenmesini izleyin.

$ meteor mongo 
MongoDB shell version: 3.2.6
connecting to: 127.0.0.1:3001/meteor

meteor:PRIMARY> db.numbers.insert({number: 5});
WriteResult({ "nInserted" : 1 })

meteor:PRIMARY> db.numbers.insert({number: 42});
WriteResult({ "nInserted" : 1 })
