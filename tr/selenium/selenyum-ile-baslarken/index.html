Java’da basit Selenyum testi #

Aşağıdaki kod, selenyum kullanan basit bir java programıdır. Aşağıdaki kodun yolculuğu

  1. Firefox tarayıcısını açın

  2. Google sayfasını açın

  3. Google sayfasının başlığını yazdırın

  4. Arama kutusu konumunu bulun

  5. Arama kutusuna değeri Selenium olarak iletin

  6. Formu gönderin

  7. Tarayıcıyı kapatın

    package org.openqa.selenium.example;

    import org.openqa.selenium.By; import org.openqa.selenium.WebDriver; import org.openqa.selenium.WebElement; import org.openqa.selenium.firefox.FirefoxDriver; import java.util.concurrent.TimeUnit;

    public class Selenium2Example { public static void main(String[] args) { // Create a new instance of the Firefox driver WebDriver driver = new FirefoxDriver();

         // An implicit wait is to tell WebDriver to poll the DOM for a certain amount of time 
     // when trying to find an element or elements if they are not immediately available. 
     // The default setting is 0. Once set, the implicit wait is set for the life of the WebDriver object instance.   
     driver.manage().timeouts().implicitlyWait(10, TimeUnit.SECONDS);

     // Maximize the browser window to fit into screen
     driver.manage().window().maximize();

     // Visit Google
     driver.get("http://www.google.com");

     // Check the title of the page
     System.out.println("Page title is: " + driver.getTitle());

     // Find the text input element by its name
     WebElement element = driver.findElement(By.name("q"));

     // Enter something to search for
     element.sendKeys("Selenium!");

     // Now submit the form. WebDriver will find the form for us from the element
     element.submit();

     //Close the browser
     driver.quit();
 }

    }

C#‘da Basit Selenyum Örneği #

//Create a new ChromeDriver
IWebDriver driver = new ChromeDriver();

//Navigate to www.google.com
driver.Navigate().GoToUrl("https://www.google.com");

//Find the WebElement of the search bar
IWebElement element = driver.FindElement(By.Name("q"));

//Type Hello World into search bar
element.SendKeys("Hello World");

//Submit the input
element.Submit();

//Close the browser
driver.Quit();

Terminal aracılığıyla python Selenium kurulumu (BASH) #

En kolay yol pip ve VirtualEnv kullanmaktır. Selenyum ayrıca python 3.* gerektirir.

Virtualenv’i aşağıdakileri kullanarak kurun:

$: pip install virtualenv

Selenium dosyalarınız için bir dizin oluşturun/girin:

$: cd my_selenium_project

Selenium dosyalarınız için dizinde yeni bir VirtualEnv oluşturun:

$: virtualenv -p /usr/bin/python3.0 venv

VirtualEnv’i etkinleştirin:

$: source venv/bin/active

Her bash satırının başında şimdi (venv) görmelisiniz. Pip kullanarak Selenium’u kurun:

$: pip install selenium

Selenium, varsayılan olarak FireFox sürücüsüyle birlikte gelir.
Selenium’u google chrome’da çalıştırmak istiyorsanız, şunu da yapın:

$: pip install chromedriver

Artık sürüm kontrollü bir VirtualEnv’iniz var. Her şeyin doğru ayarlandığından emin olmak için:

Python’u başlatın:

$: python

Çıktıları yazdırır:

Python 2.7.10 (default, Jul 14 2015, 19:46:27) 
[GCC 4.2.1 Compatible Apple LLVM 6.0 (clang-600.0.39)] on darwin
Type "help", "copyright", "credits" or "license" for more information.

Yeni bir web sürücüsü (bu durumda bir krom sürücü) oluşturun ve www.google.com adresine gidin:

>>> from selenium import webdriver
>>> driver = webdriver.Chrome()
>>> driver.get("https://www.google.com")

Sürücüyü ve python yorumlayıcısını kapatın:

>>> driver.quit()
>>> quit()

VirtualEnv’i devre dışı bırakın:

$: deactivate

driver = webdriver.Chrome() satırı hata veriyorsa:

  • Ayrıca krom tarayıcının kurulu olduğundan emin olun. Bunu yapmazsanız, Selenium krom sürücüsü Chrome ikili dosyasına erişemez.
  • webdriver.Chrome() ayrıca chromedriver konumunuz için bir parametre alabilir. Pip kullanarak yüklediyseniz, (mac üzerinde) driver = webdriver.Chrome("./venv/selenium/webdriver/chromedriver") deneyin.
