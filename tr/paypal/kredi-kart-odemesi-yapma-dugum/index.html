Parametreler #

parametre Ayrıntılar
kart_verileri işlem için ödeme bilgilerini içeren JSON nesnesi
kredi_kartı_detayları Kasaya atılmak üzere PayPal’a gönderilen kredi kartı verilerini içeren JSON nesnesi
client_id PayPal uygulama istemci kimliğiniz (OAuth 2 kimlik bilgileri)
paypal PayPal Düğüm SDK referansı
sır PayPal uygulama sırrınız (OAuth 2 kimlik bilgileri)
kullanıcı kimliği node-uuid paketine başvuru

Bu örnek, kullanıcıyı PayPal SDK’larını kullanarak basit bir kredi kartı işlemine kredilendirmeye götürür.

Düğüm Örneği #

NPM’den PayPal Node modülünü kurarak başlayın

npm install paypal-rest-sdk

Uygulama dosyanızda, SDK için yapılandırma bilgilerini ekleyin

var paypal = require('paypal-rest-sdk');

var client_id = 'YOUR CLIENT ID';
var secret = 'YOUR SECRET';

paypal.configure({
    'mode': 'sandbox', //sandbox or live
    'client_id': client_id,
    'client_secret': secret
});

SDK gereksinimini ekliyoruz, ardından bir uygulama oluştururken elde edilen istemci kimliği ve sır için değişkenler ayarlıyoruz. Daha sonra bu detayları kullanarak uygulamamızı yapılandırır ve çalıştığımız ortamı (canlı veya sanal alan) belirleriz.

Ardından, ödeyenin ödeme bilgilerini içeren JSON nesnesini kurduk.

  "intent": "sale",
  "payer": {
    "payment_method": "credit_card",
    "funding_instruments": [{
      "credit_card": {

        "billing_address": {
          "line1": "52 N Main ST",
          "city": "Johnstown",
          "state": "OH",
          "postal_code": "43210",
          "country_code": "US" }}}]},
  "transactions": [{
    "amount": {
      "total": "7.47",
      "currency": "USD",
      "details": {
        "subtotal": "7.41",
        "tax": "0.03",
        "shipping": "0.03"}},
    "description": "This is the payment transaction description." 
}]};

Bir “satış” “niyeti” ve “kredi_kartı” için bir “ödeme_yöntemi” ekleyin. Ardından, “finansman araçları” altında kredi kartı için kart ve adres bilgilerini ve “işlemler” altında tahsil edilecek tutarı ekleyin. Buraya birden fazla işlem nesnesi yerleştirilebilir.

Son olarak, ödemeyi işleme koymak için ‘card_data’ nesnemizi ileterek ‘payment.create(…)’ için bir talepte bulunuyoruz.

paypal.payment.create(card_data, function(error, payment){
  if(error){
    console.error(error);
  } else {
    console.log(payment);
  }
});

Kasalı Kredi Kartı ile Ödeme Yapma (Düğüm) #

Bu örnekte, PayPal kasasını kullanarak bir kredi kartının nasıl saklanacağını inceleyeceğiz, ardından bir kullanıcı için bir kredi kartı işlemini işlemek için o depolanan kredi kartına başvuracağız.

Kasayı kullanmak istememizin nedeni, hassas kredi kartı bilgilerini kendi sunucularımızda saklamak zorunda kalmamamızdır. Ödeme yöntemine yalnızca sağlanan bir kasa kimliği aracılığıyla başvuruyoruz, bu da kredi kartlarını kendimiz saklamakla ilgili birçok PCI uyumluluk düzenlemesiyle uğraşmak zorunda olmadığımız anlamına geliyor.

Önceki örneklerde olduğu gibi, ortamımızı kurmaya başlıyoruz.

var paypal = require('paypal-rest-sdk'),
    uuid = require('node-uuid');

var client_id = 'YOUR CLIENT ID';
var secret = 'YOUR SECRET';

paypal.configure({
  'mode': 'sandbox', //sandbox or live
  'client_id': client_id,
  'client_secret': secret
});

Buradaki önceki örneklerden tek fark, kartı saklarken ödeyenler için benzersiz UUID’ler oluşturmak için kullanılacak yeni bir ’node-uuid’ paketine ihtiyaç duymamızdır. Bu paketi şu yollarla yükleyebilirsiniz:

npm install node-uuid

Ardından, depolama için PayPal kasasına gönderilecek kredi kartı JSON nesnesini tanımlarız. Karttaki bilgilerin yanı sıra “node-uuid” kullanarak oluşturduğumuz benzersiz bir ödeme kimliği içerir. Bu benzersiz ‘payer_id’yi, kasalı kartla ödeme oluştururken kullanılacağı için kendi veritabanınızda saklamanız gerekir.

var create_card_details = {
    "type": "visa",
    "number": "4417119669820331",
    "expire_month": "11",
    "expire_year": "2018",
    "first_name": "John",
    "last_name": "Doe",
    "payer_id": uuid.v4()
};

Son olarak, kredi kartını saklamamız ve bu kartı kullanarak ödemeyi işlememiz gerekiyor. Bir kredi kartını kasaya koymak için, az önce oluşturduğumuz ‘credit_card_details’ nesnesini ileterek ‘credit_card.create(…)’ öğesini çağırırız. Her şey yolunda giderse, tonozlu kartla ilgili ayrıntıları içeren bir nesnenin bize geri dönmesi gerekir. Bu kartla ödeme yapmak için gerçekten sadece iki parça bilgiye ihtiyacımız var: halihazırda sakladığımız payer_id ve ayrıca kendi veritabanımızda referans olarak saklanması gereken kasa kimliği.

paypal.credit_card.create(create_card_details, function(error, credit_card){
    if(error){
        console.error(error);
    } else {
        var card_data = {
            "intent": "sale",
            "payer": {
                "payment_method": "credit_card",
                "funding_instruments": [{
                    "credit_card_token": {
                        "credit_card_id": credit_card.id,
                        "payer_id": credit_card.payer_id
                    }
                }]
            },
            "transactions": [{
                "amount": {
                    "total": "7.47",
                    "currency": "USD",
                    "details": {
                        "subtotal": "7.41",
                        "tax": "0.03",
                        "shipping": "0.03"
                    }
                },
                "description": "This is the payment transaction description." 
            }]
        };
        
        paypal.payment.create(card_data, function(error, payment){
            if(error){
                console.error(error);
            } else {
                console.log(JSON.stringify(payment));
            }
        });
    }
});

Kredi kartının başarılı bir şekilde kasaya alınmasını takip eden bölümde, bir önceki kredi kartı işleme örneğinde yaptığımız gibi, sadece kart detaylarını tanımlıyor ve ödemeyi gerçekleştiriyoruz. ‘card_data’ nesnesinin yapısındaki temel fark, ‘ödeyen’ altında tanımladığımız ‘funding_instruments’ bölümüdür. Kredi kartı bilgilerini tanımlamak yerine, kasa kimliği referansını ve ödeyen kimliğini içeren aşağıdaki nesneyi kullanırız:

"credit_card_token": {
    "credit_card_id": credit_card.id,
    "payer_id": credit_card.payer_id
}

Bir ödemeyi işlemek için kasalı bir kartı bu şekilde kullanırız.
