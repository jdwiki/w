Sözlükler #

Sözlükler bir Dict çekirdek kitaplığında uygulanır.

Benzersiz anahtarları değerlere eşleyen bir sözlük. Anahtarlar herhangi biri olabilir karşılaştırılabilir tür. Buna Int, Float, Time, Char, String ve kümeler veya karşılaştırılabilir türlerin listeleri.

Ekleme, kaldırma ve sorgulama işlemlerinin tümü O(log n) zaman alır.

Tuples ve Records’tan farklı olarak, Sözlükler yapılarını değiştirebilir, yani anahtar ekleyip çıkarmak mümkündür.

Bir değeri bir anahtarla güncellemek mümkündür.

Dinamik anahtarları kullanarak bir değere erişmek veya bir değeri güncellemek de mümkündür.

Değerlere erişim #

Bir “Dict.get” işlevini kullanarak Sözlükten bir değer alabilirsiniz.

“Dict.get” tipi tanımı:

get : comparable -> Dict comparable v -> Maybe v

Her zaman Belki v döndürür, çünkü var olmayan bir anahtarla bir değer elde etmeye çalışmak mümkündür.

import Dict

initialUsers =
      Dict.fromList [ (1, "John"), (2, "Brad") ]

getUserName id =
  initialUsers
  |> Dict.get id
  |> Maybe.withDefault "Anonymous"
  
getUserName 2 -- "Brad"
getUserName 0 -- "Anonymous"

Değerler güncelleniyor #

Bir Sözlükte güncelleme işlemi, istenen anahtar bulunmadığından Belki.map kullanılarak gerçekleştirilir.

import Dict

initialUsers =
  Dict.fromList [ (1, "John"), (2, "Brad") ]

updatedUsers =
  Dict.update 1 (Maybe.map (\name -> name ++ " Johnson")) initialUsers

Maybe.withDefault "No user" (Dict.get 1 updatedUsers) -- "John Johnson"

Kayıtlar #

Kayıt, bir dizi anahtar/değer çiftidir.

greeter =
    { isMorning: True
    , greeting: "Good morning!"
    }

Var olmayan bir anahtarla bir değere erişmek mümkün değildir.

Record’un yapısını dinamik olarak değiştirmek mümkün değildir.

Kayıtlar, değerleri yalnızca sabit anahtarlarla güncellemenize izin verir.

Değerlere erişim #

Olası çalışma zamanı hatalarını önlemek için değerlere dinamik anahtar kullanılarak erişilemez:

isMorningKeyName =
    "isMorning "


greeter[isMorningKeyName] -- Compiler error
greeter.isMorning -- True

Değere erişmek için kullanılan alternatif sözdizimi, Kayıttan geçerken değeri çıkarmanıza olanak tanır:

greeter
    |> .greeting
    |> (++) " Have a nice day!" -- "Good morning! Have a nice day!"

Genişletme Türleri #

Bazen, işlevlere ilettiğiniz kayıt türlerini sınırlamak için bir parametrenin imzasını istersiniz. Kayıt türlerini genişletmek, süper türler fikrini gereksiz kılar. Aşağıdaki örnek, bu konseptin nasıl uygulanabileceğini göstermektedir:

type alias Person =
    { name : String
    }


type alias Animal =
    { name : String
    }


peter : Person
peter =
    { name = "Peter" }


dog : Animal
dog =
    { name = "Dog" }


getName : { a | name : String } -> String
getName livingThing =
    livingThing.name


bothNames : String
bothNames =
    getName peter ++ " " ++ getName dog

Genişleyen kayıtları bir adım daha ileri götürebilir ve şöyle bir şey yapabiliriz:

type alias Named a = { a | name : String }
type alias Totalled a = { a | total : Int }


totallyNamed : Named ( Totalled { age : Int })
totallyNamed =
  { name = "Peter Pan"
  , total = 1337
  , age = 14
  }

Artık bu kısmi türleri işlevlerde aktarmanın yollarına sahibiz:

changeName : Named a -> String -> Named a
changeName a newName =
  { a | name = newName }
  
cptHook = changeName totallyNamed "Cpt. Hook" |> Debug.log "who?"

Değerler güncelleniyor #

Elm, Kayıt güncellemeleri için özel bir sözdizimine sahiptir:

model =
    { id: 1
    , score: 0
    , name: "John Doe"
    }


update model =
    { model
       | score = model.score + 100
       | name = "John Johnson"
    }

demetler #

Tuple’lar, herhangi bir türdeki değerlerin sıralı listeleridir.

(True, "Hello!", 42)

Bir Tuple’ın yapısını değiştirmek veya değeri güncellemek mümkün değildir.

Elm’deki Tuple’lar, ilkel bir veri türü olarak kabul edilir; bu, Tuple’ları kullanmak için herhangi bir modül içe aktarmanıza gerek olmadığı anlamına gelir.

Değerlere erişim #

Temel Bilgiler modülü, iki ( a, b ) uzunluğundaki bir Tuple değerlerine kalıp eşleştirme kullanmadan erişmek için iki yardımcı fonksiyona sahiptir:

fst (True, "Hello!") -- True
snd (True, "Hello!") -- "Hello!"

Daha büyük uzunluktaki demetlerin erişim değerleri, desen eşleştirme yoluyla yapılır.

Desen eşleştirme #

Tuple’lar, desen eşleştirme ile birlikte son derece kullanışlıdır:

toggleFlag: (Sting, Bool) -> (Sting, Bool)
toggleFlag (name, flag) =
    (name, not flag)
