Robot çerçevesi, Gömülü ürünlerin Otomasyon testlerinde yaygın olarak kullanılmaktadır. Örnek olarak Gömülü bir ürün alacağız ve Robot Framework kullanarak test senaryolarının nasıl otomatikleştirileceğini göreceğiz.

Kısaltma:

  • RPS - Uzak güç kaynağı
  • RF - Robot çerçeve çalışması

Uzaktan Güç Kaynağı Testi #

Uzak Güç kaynağı simülasyonu #

Gerçek bir uzak güç kaynağı donanımımız olmadığı için, onu simüle edeceğiz. python programını kullanarak.

RPS hakkında temel fikir #

  • Aslında uzak güç kaynağının bir http sunucusu vardır.
  • Kullanıcı, http isteğini kullanarak güç kaynağını AÇMAK/KAPATMAK için komutlar gönderebilir.

Aşağıdaki rps-server.py programını kullanarak uzak güç kaynağını simüle edeceğiz.

from flask import Flask, request
from flask_httpauth import HTTPBasicAuth


app = Flask(__name__)
auth = HTTPBasicAuth()

users = {
        'admin': '12345678'
}
app.url_map.strict_slashes = False

PINS = ['P60', 'P61', 'P62', 'P63']

PINS_STATUS = {'P60':'0', 'P61': '0', 'P62':'0', 'P63':'0'}

@auth.get_password
def get_pw(username):
        if username in users:
                return users.get(username)
        return None

@app.route('/')
@auth.login_required
def index():
        return "Hello, %s!" % auth.username()

def get_html_string():
        html_str = '<html>P60={}P61={}P62={}P63={}</html>'.format(PINS_STATUS['P60'],
                                                PINS_STATUS['P61'],
                                            PINS_STATUS['P62'],
                                            PINS_STATUS['P63'])
        return html_str

def parse_cmd_args(args):
        global current_status
        if str(args['CMD']) == 'SetPower':
                for key in args:
                        if key in PINS:
                                PINS_STATUS[key] = str(args[key])
                            
                return get_html_string()
    
        if str(args['CMD']) == 'GetPower':
                return get_html_string()
   

    @app.route('/SetCmd', methods=['GET','POST'])
    def rps():
        if request.method=="GET":
                args=request.args.to_dict()
                ret = parse_cmd_args(args)
                return ret

Yukarıdaki kod aslında uzak güç kaynağını kontrol etmek için http sunucusunu simüle eder.

RPS sunucusu Nasıl Çalıştırılır ? #

$ export FLASK_APP=rps-server.py
$ flask run

rps sunucusuna komutlar nasıl gönderilir? #

RPS’yi kontrol etmek için kullanılan iki komut aşağıdadır.

  1. Güç Ayarla
  2. GetPower

Varsayılan olarak sunucu 5000 numaralı bağlantı noktasını dinliyor olacaktır.

Güç kaynağı portları,

  1. P60
  2. P61
  3. S62
  4. P64

Limanların durumları,

  1. AÇIK - 1
  2. KAPALI - 0

Gereksinimler #

Uzak bir güç kaynağı oluşturmak için gereksinimler şunlardır:

  1. Uzak güç kaynağı, uzaktan AÇILABİLİR/KAPATILABİLİR
  2. Uzak güç kaynağı durumuna uzaktan erişilebilir.

Test senaryoları türetme #

Gereksinimlerden türetilen test senaryoları

  1. Güç kaynağı 2’yi uzaktan açın.
  2. Güç kaynağı 2’nin açık olduğunu doğrulayın.
  3. Güç kaynağı 2’yi uzaktan kapatın.
  4. Güç kaynağı 2’nin kapalı olduğunu doğrulayın.

Manuel Test #

  • rps sunucusunu çalıştırın.
  • Port 3’ü açmak için bir tarayıcı açın ve aşağıdaki URI’yi verin

http://admin:12345678@localhost:5000/SetCmd?CMD=SetPower&P62=1

  • Tüm portların durumunu almak için

http://admin:12345678@localhost:5000/SetCmd?CMD=GetPower

Test kitaplığı yazma #

Http isteğini kullanarak http komutları göndermek için python’da bir test kütüphanesi yazmamız gerekiyor. Daha sonra bu kütüphaneyi robot çerçeve çalışmasında anahtar kelime olarak kullanacağız.

komutlar.py #

SetPower ve GetPower göndermek için commands.py’den kitaplığı kullanacağız.

import requests
import re


class commands(object):

    ROBOT_LIBRARY_SCOPE = 'GLOBAL'
    def __init__(self, ip='localhost:5000'):
        self.ip_address = ip
        self.query = {}
        self.user = 'admin'
        self.passw = '12345678'

    def form_query(self, state, cmd, port):
        port = self.get_port_no(port)
        self.query = {port: state}
        return self.query

    def get_port_no(self, port_no):
        port = 'P6' + str(port_no)
        return port

    def clean_html(self, data):
        exp = re.compile('<.*?>')
        text = re.sub(exp, "", data)
        return text.rstrip()

    def send_cmds(self, cmd, port=None, state=None):
        url = 'http://{}:{}@{}/SetCmd?CMD={}'\
              .format(self.user,
                      self.passw,
                      self.ip_address,
                      cmd)
        print url
        if cmd == 'SetPower':
            self.form_query(state, cmd, port)
            self.req = requests.get(url, params=self.query)
            return True
        elif cmd == 'GetPower':
            self.req = requests.get(url)
            data = self.clean_html(self.req.text)
            return data
        else:
            return False

        return self.req.text

Python anahtar kelime dokümantasyonu #

  1. send_cmds(cmd, port=Yok, durum=Yok) kullanacağımız fonksiyondur.
  2. Robot anahtar sözcüğünde bu işlevi kullanırken, işlev adında _, Küçük Harf veya Büyük Harf ile uğraşmanıza gerek yoktur.

Anahtar kelime olarak kullanıldığında Python işlevi şöyle görünecektir,

Send Cmds       cmd   port  state

Test Anahtar Kelimeleri yazma #

Test takımında python anahtar kelimesi olarak Gönder Cmds kullanacağız.

  • RPS gönderme komutları, gücü ayarlamak için aşağıdaki dört argümanı kullanır

    • command = SetPower
    • port = 2
    • state = 1 for ON / 0 for off Bu komutu çağırdığımızda, güç kaynağını AÇAR/KAPATACAKTIR.

  • RPS alma gücü, tüm Güç kaynağı bağlantı noktalarının durumunu döndürür

    *** Keywords *** RPS send commands [Arguments] ${command} ${port} ${state} ${output}= Send cmds ${command} ${port} ${state} [return] ${output}

    RPS get Power [Arguments] ${command} ${output}= Send cmds ${command} [return] ${output}}

Güç kaynağını test etmek için algoritma #

  1. Gücü bir bağlantı noktasına ayarlayın
  2. cmd’nin durumunu kontrol edin
  3. Bağlantı noktasının durumunu öğrenin ve AÇIK/KAPALI olup olmadığını kontrol edin

Yukarıdaki anahtar kelimeleri kullanarak test senaryoları yazma #

Şimdi aşağıdaki iki anahtar kelimeyi kullanarak test senaryosu yazmaya hazırız

  • RPS komutları gönder - Bağlantı noktasının gücünü ayarlamak ve ayarlamak için

  • RPS güç alır - Tüm bağlantı noktasının durumunu almak için

    *** Settings *** Library commands.py

    *** Test Cases *** Turn on Power supply 2 remotely ${out}= RPS send commands SetPower 2 1 Should be equal ${out} ${True}

    Verify power supply 2 is on ${out}= RPS get power GetPower should contain ${out} P62=1

    Turn off Power supply 2 remotely ${out}= RPS send commands SetPower 2 0 Should be equal ${out} ${True}

    Verify power supply 2 is off ${out}= RPS get power GetPower should contain ${out} P62=0
