Temel kurulum #

Omurga, DOM manipülasyonu (Backbone.View kullanarak) ve RESTful kalıcılık için Alt Çizgi ve (isteğe bağlı olarak) jQuery gerektirir.

Backbone ile çalışmaya başlamanın en hızlı yolu, HTML ‘’ içinde basit komut dosyası etiketleriyle bir ‘index.html’ dosyası oluşturmaktır:

<html>
    <head>
        <script src="https://code.jquery.com/jquery-3.1.0.min.js"></script>
        <script src="https://cdnjs.cloudflare.com/ajax/libs/underscore.js/1.8.3/underscore-min.js"></script>

        <script src="https://cdnjs.cloudflare.com/ajax/libs/backbone.js/1.3.3/backbone-min.js"></script>
    </head>
    <body>
    </body>
</html>

Omurga artık sayfada kullanıma hazır.

Omurga Kullanma Örneği (Öncelikle Backbone.Model) #

Backbone.js dört ayrı bileşenden oluşur: Koleksiyonlar, Modeller, Yönlendiriciler ve Görünümler. Bunların her biri farklı amaçlara hizmet eder:

  • “Model” - tek bir veri nesnesini temsil eder, ancak olay sistemi gibi yerel JavaScript nesneleri tarafından sağlanmayan ek işlevler ve verileri alıp uzak bir sunucuya göndermek için daha uygun bir yol ekler.

  • “Koleksiyon” - bir Modeller kümesini veya “koleksiyonunu” temsil eder ve modellerini yönetmek için işlevler sağlar.

  • “Görünüm” - kullanıcı arayüzünün tek bir bölümünü temsil eder; her Görünüm bir HTML DOM öğesini sarar ve basit olay bağlama gibi kolaylık özelliklerinin yanı sıra bu öğeyle çalışmak için bir yapı sağlar.

  • “Yönlendirici” - URL değişikliklerine yanıt olarak bir uygulamanın farklı mantığı tetiklemesine (örneğin farklı sayfalar göstermesine) izin vererek “tek sayfalı bir uygulamayı” etkinleştirir.

Kendi lezzetlerinizi yaratın #

Bu bileşenlerin her birinin nasıl kullanılacağına bakmadan önce, Backbone’un sınıf sistemine hızlıca bir göz atalım. Bir Backbone sınıfının yeni bir alt sınıfını oluşturmak için, orijinal sınıfın “extend” yöntemini çağırmanız ve buna örnek özelliklerini ve (statik) sınıf özelliklerini nesneler olarak iletmeniz yeterlidir:

const MyModelClass = Backbone.Model.extend({
    instanceMethod: function() { console.log('Instance method!'); },
}, {
    staticMethod: function() { console.log('Static method!'); },
});

Diğer herhangi bir sınıf sisteminde olduğu gibi, örnek yöntemler sınıfın örneklerinde (nesnelerinde) çağrılabilirken, statik yöntemler doğrudan sınıfın kendisinde (kurucu) çağrılır:

var myInstance = new MyModelClass();

// Call an instance method on our instance
myInstance.instanceMethod(); // logs "Instance method!"

// Call a static method on our class
MyModelClass.staticMethod(); // logs "Static method!"

Bir sınıf kullanmak #

Şimdi, her bir sınıfı nasıl kullanabileceğinize dair hızlı bir örneğe bakalım. Bir kitabın ‘Modeli’ ile başlayacağız.

const Book = Backbone.Model.extend({
    idAttribute: 'isbn',
    urlRoot: '/book'
});

Az önce orada olanları parçalayalım. İlk olarak, ‘Model’in ‘Kitap’ alt sınıfını oluşturduk ve ona iki örnek özelliği verdik.

  • ‘idAttribute’, AJAX işlemlerini gerçekleştirirken Backbone’a modelin “isbn” niteliğini kimliği olarak kullanmasını söyler.
  • “urlRoot”, Backbone’a “www.example.com/book” adresinde kitap verilerini aramasını söyler.

Şimdi bir kitap örneği oluşturalım ve verilerini sunucudan alalım:

var huckleberryFinn = new Book({ isbn: '0486403491' });
huckleberryFinn.fetch({
    // the Backbone way
    success: (model, response, options) => {
       console.log(model.get('name')); // logs "Huckleberry Finn"
    }
}).done(() => console.log('the jQuery promise way'));

Yeni bir “Kitap” oluşturduğumuzda ona bir nesne ilettik ve Backbone bu nesneyi “Model"in ilk “nitelikleri” (verileri) olarak kullanır. Omurga, ‘idAttribute’ öğesinin ‘isbn’ olduğunu bildiği için, yeni Kitabımızın URL’sinin ‘/book/0486403491’ olduğunu bilir. Getirmesini söylediğimizde, Backbone kitap verileri için bir AJAX isteğinde bulunmak için jQuery kullanacak. “getir”, getirme işlemi tamamlandıktan sonra eylemleri tetiklemek için kullanabileceğiniz bir söz verir (tıpkı “$.ajax” gibi).

Niteliklere ‘get’ veya ‘set’ yöntemleri kullanılarak erişilebilir veya değiştirilebilir:

huckleberryFinn.get('numberOfPages'); // returns 64

huckleberryFinn.set('numberOfPages', 1); // changes numberOfPages to 1

“Modeller” ayrıca bir “Model"in başına bir şey geldiğinde tepki vermek için kullanabileceğiniz bir olay sistemine sahiptir. Örneğin, “numberOfPages” değiştiğinde bir iletiyi günlüğe kaydetmek için şunları yapabilirsiniz:

huckleberryFinn.on('change:numberOfPages', () => console.log('Page change!'));

Diğer Omurga sınıflarına daha ayrıntılı bir giriş için, kendi belge sayfalarına bakın.

Temel kavramları gösteren örnek #

Aşağıdaki örnek bir giriş niteliğindedir:

  • Alt çizgi kullanarak şablon derlemesi
  • Bir şablondaki değişkenlere erişme
  • Görünüm oluşturma
  • Görünüm oluşturma
  • Bir görünüm gösteriliyor
<html>
<head>
    <script src="https://code.jquery.com/jquery-3.1.0.min.js"></script>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/underscore.js/1.8.3/underscore-min.js"></script>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/backbone.js/1.3.3/backbone-min.js"></script>
</head>
<body>

    <div id="example_container"></div>

    <script type="text/template" id="example_template">
        <label><%= example_label %></label>
        <input type="text" id="example_input" />
        <input type="button" id="example_button" value="Search" />
    </script>
    <script type="text/javascript">
        var ExampleView = Backbone.View.extend({
            // Compile the template using underscore
            template: _.template($("#example_template").html()),
            events: {
                "click #example_button": "onButtonClick"
            },

            initialize: function(options) {
                this.customOption = options.customOption;
            },

            render: function() {
                // Load the compiled HTML into the Backbone "el"
                this.$el.html(this.template({
                    example_label: "My Search"
                }));

                return this; // for chaining, a Backbone's standard for render
            },

            onButtonClick: function(event) {
                // Button clicked, you can access the button that 
                // was clicked with event.currentTarget
                console.log("Searching for " + $("#example_input").val());
            }
        });
        $(function() {
            //show the view inside the div with id 'example_container'
            var exampleView = new ExampleView({
                el: $("#example_container"),
                customOption: 41,
            });
            exampleView.render();
        });
    </script>
</body>
</html>

Merhaba Web (Temel “Merhaba Dünya” tipi kurulum) #

<html>
    <head>
        <script src="https://code.jquery.com/jquery-3.1.0.min.js"></script>
        <script src="https://cdnjs.cloudflare.com/ajax/libs/underscore.js/1.8.3/underscore-min.js"></script>

        <script src="https://cdnjs.cloudflare.com/ajax/libs/backbone.js/1.3.3/backbone-min.js"></script>
        
        <script>
            $( function(){
                ( function(){
                    var View = Backbone.View.extend( {
                        "el": "body",
                        "template": _.template( "<p>Hello, Web!</p>" ),
    
                        "initialize": function(){
                            this.render();
                        },
                        "render": function(){
                            this.$el.html( this.template() );
                        }
                    } );
    
                    new View();
                })()
            } );
        </script>
    </head>
    <body>
    </body>
</html>
