Devam Eden ABL’de “CHARACTER” olarak tanımlananlar ve “LONGCHAR” olarak tanımlananlar olmak üzere iki tür dize vardır. 32K’dan daha büyük bir dosya bir ‘LONGCHAR’dır. Çoğu dize, başka bir şekilde belirtilmediği sürece büyük/küçük harfe duyarsızdır.

Unutmayın - tüm pozisyonlar pozisyon 1 ile başlar!

Birleştirme dizeleri #

‘+’ operatörünü kullanarak iki veya daha fazla diziyi kolayca birleştirebilirsiniz.

DEFINE VARIABLE cString AS CHARACTER   NO-UNDO.

cString = "HELLO".

cString = cString + " " + "GOODBYE".

DISPLAY cString FORMAT "X(20)".

VAKA DUYARLI dizeler #

Progress ABL’deki tüm dizeler, aksi belirtilmedikçe büyük/küçük harfe duyarlıdır.

Bu örnekte, dizelerin aynı olduğunu söyleyen bir mesaj kutusu görüntülenecektir.

DEFINE VARIABLE str1 AS CHARACTER   NO-UNDO.
DEFINE VARIABLE str2 AS CHARACTER   NO-UNDO.

str1 = "abc".
str2 = "ABC".

IF str1 = str2 THEN 
    MESSAGE "The strings are identical" VIEW-AS ALERT-BOX.

Bir dizenin büyük/küçük harfe duyarlı olduğunu belirtmek için ‘CASE-SENSITIVE’ özniteliğini eklemeniz yeterlidir.

DEFINE VARIABLE str1 AS CHARACTER   NO-UNDO CASE-SENSITIVE.
DEFINE VARIABLE str2 AS CHARACTER   NO-UNDO.

str1 = "abc".
str2 = "ABC".

IF str1 = str2 THEN 
    MESSAGE "The strings are identical" VIEW-AS ALERT-BOX.
ELSE 
    MESSAGE "There's a difference" VIEW-AS ALERT-BOX.

(Bu durumda dizelerden birinin buna sahip olması yeterlidir).

Bir dize tanımlama, atama ve görüntüleme #

Genel olarak, gerçekten gerekmedikçe tüm değişkenleri ve parametreleri her zaman ‘GERİ ALMA’ YOK olarak tanımlamanız gerekir.

DEFINE VARIABLE cString AS CHARACTER   NO-UNDO.

cString = "HELLO".

DISPLAY cString.

Dize manipülasyonu #

Dize ile çalışmak için birkaç kullanışlı yerleşik işlev vardır. Karakterlerin konumuyla çalışan tüm işlevler, birçok dilde yaygın olduğu gibi 0 ile değil, ilk karakter olarak dizin 1 ile başlar.

STRING - herhangi bir değeri bir dizgeye dönüştürür

Bu örnek, 2000 tamsayısını “2000” dizesine dönüştürür.

DEFINE VARIABLE i AS INTEGER     NO-UNDO.
DEFINE VARIABLE c AS CHARACTER   NO-UNDO.

i = 2000.

c = STRING(i).

DISPLAY c.

CHR ve ASC - tek karakterleri ascii’ye ve ascii’den dönüştürür.

CHR(tam sayı)

ascii kodu tamsayısının karakter temsilini döndürür

ASC(karakter)

Karakter için ascii tamsayı değerini döndürür

DEFINE VARIABLE ix     AS INTEGER   NO-UNDO.
DEFINE VARIABLE letter AS CHARACTER NO-UNDO FORMAT "X(1)" EXTENT 26.

DO ix = 1 TO 26:
  letter[ix] = CHR((ASC("A")) - 1 + ix).
END.

DISPLAY SKIP(1) letter WITH 2 COLUMNS NO-LABELS
  TITLE "T H E  A L P H A B E T".

UZUNLUK - bir dizenin uzunluğunu döndürür

UZUNLUK(dize). //Dizenin uzunluğuna sahip bir tamsayı döndürür.

DEFINE VARIABLE cString AS CHARACTER   NO-UNDO.

cString = "HELLO".

MESSAGE "The string " cString " is " LENGTH(cString) " characters long" VIEW-AS ALERT-BOX.

SUBSTRING - bir dizenin bir bölümünü döndürür veya atar

  • SUBSTRING(dize, başlangıç ​​konumu, uzunluk).

“Başlangıç-konumu” konumundan başlayarak “dizge"den “uzunluk” karakterlerini döndürür.

  • SUBSTRING(dize, başlangıç ​​konumu).

“Başlangıç ​​​​pozisyonu” konumundan başlayarak “dize"nin geri kalanını döndürür

DEFINE VARIABLE cString AS CHARACTER   NO-UNDO.

cString = "ABCDEFGH".

DISPLAY SUBSTRING(cString, 4, 2). //Displays "DE" 
DISPLAY SUBSTRING(cString, 4). //Displays "DEFGH"

Alt dize, bir dizenin bir bölümünün üzerine yazmak için de kullanılabilir. Aynı sözdizimini kullanın ancak bunun yerine bu alt dizeyi atayın:

DEFINE VARIABLE cString AS CHARACTER   NO-UNDO.

cString = "ABCDEFGH".
 
SUBSTRING(cString, 4, 2) = "XY". //Replaces position 4 and 5 with "XY" 

DISPLAY cString.

“OVERLAY” adlı benzer bir işlev de vardır. İlerleme belgesindeki bu örnek “OVERLAY” ve “SUBSTRING” arasındaki farkları kapsar:

/* This procedure illustrates the differences between the SUBSTRING and
   OVERLAY statements. */
DEFINE VARIABLE cOriginal  AS CHARACTER NO-UNDO INITIAL "OpenEdge".
DEFINE VARIABLE cSubstring AS CHARACTER NO-UNDO.
DEFINE VARIABLE cOverlay   AS CHARACTER NO-UNDO.
DEFINE VARIABLE cResults   AS CHARACTER NO-UNDO.

/* Default behavior without optional LENGTH. */
ASSIGN
 cSubstring              = cOriginal
 SUBSTRING(cSubstring,2) = "***"
 cOverlay                = cOriginal
 OVERLAY(cOverlay,2)     = "***"
 cResults                = "target = ~"OpenEdge~". ~n~n" 
  + "If you do not supply a length, SUBSTRING and OVERLAY default as follows:
  ~n~n" + "SUBSTRING(target,2) = ~"***~"  yields:  " + cSubstring + ". ~n" 
  + "OVERLAY(target,2)     = ~"***~"  yields:  " + cOverlay + ".".

/* Behavior with zero LENGTH. */
ASSIGN
 cSubstring                = cOriginal
 SUBSTRING(cSubstring,2,0) = "***"
 cOverlay                  = cOriginal
 OVERLAY(cOverlay,2,0)     = "***"
 cResults                  = cResults + "~n~n" 
  + "For a zero length, SUBSTRING and OVERLAY behave as follows:  ~n~n" 
  + "SUBSTRING(target,2,0) = ~"***~"  yields:  " + cSubstring + ". ~n" 
  + "OVERLAY(target,2,0)     = ~"***~"  yields:  " + cOverlay + ".".

/* Behavior with LENGTH < replacement. */
ASSIGN 
 cSubstring                = cOriginal
 SUBSTRING(cSubstring,2,1) = "***"
 cOverlay                  = cOriginal
 OVERLAY(cOverlay,2,1)     = "***"
 cResults                  = cResults + "~n~n" 
  + "For a length shorter than the replacement, SUBSTRING and OVERLAY behave
  as follows: ~n~n" + "SUBSTRING(target,2,1) = ~"***~"  yields:  " 
  + cSubstring + ". ~n" + "OVERLAY(target,2,1)     = ~"***~"  yields:  " 
  + cOverlay + ".".

/* Behavior with LENGTH = replacement. */
ASSIGN 
 cSubstring                = cOriginal
 SUBSTRING(cSubstring,2,3) = "***"
 cOverlay                  = cOriginal
 OVERLAY(cOverlay,2,3)     = "***"
 cResults                  = cResults + "~n~n" 
  + "For a length equal to the replacement, SUBSTRING and OVERLAY behave as
  follows:  ~n~n" + "SUBSTRING(target,2,3) = ~"***~"  yields:  " 
  + cSubstring + ". ~n" + "OVERLAY(target,2,3)     = ~"***~"  yields:  " 
  + cOverlay + ".".

/* Behavior with LENGTH > replacement. */
ASSIGN 
 cSubstring                = cOriginal
 SUBSTRING(cSubstring,2,6) = "***"
 cOverlay                  = cOriginal
 OVERLAY(cOverlay,2,6)     = "***"
 cResults                  = cResults + "~n~n" 
  + "For a length greater than the replacement, SUBSTRING and OVERLAY behave
  as follows:  ~n~n" + "SUBSTRING(target,2,6) = ~"***~"  yields:  " 
  + cSubstring + ". ~n" + "OVERLAY(target,2,6)     = ~"***~"  yields:  " 
  + cOverlay + ".".

MESSAGE cResults VIEW-AS ALERT-BOX.

INDEX - bir dizgedeki dizgenin konumunu döndürür.

R-INDEX aynı şeyi yapacak, ancak sağdan sola arama yapacak.

INDEX(kaynak, hedef)

Hedefi kaynak içinde (soldan sağa) arayın ve konumunu döndürün. Eksikse 0 döndürün.

INDEX(kaynak, hedef, başlangıç ​​konumu).

Yukarıdakiyle aynı ancak aramaya başlangıç ​​konumundan başlayın

DEFINE VARIABLE str AS CHARACTER   NO-UNDO.

str = "ABCDEFGH".

DISPLAY INDEX(str, "cd") INDEX(str, "cd", 4). //Will display 3 and 0

DEĞİŞTİR - bir dize içindeki bir dizeyi değiştirir.

DEĞİŞTİR(dize, dizeden dizeye)

string içindeki from-string’i to-string ile değiştirir. From-string ve to-string aynı uzunlukta olmak zorunda değildir, bir karakteri kaldırmak için to-string de hiçbir şey (””) olabilir.

DEFINE VARIABLE c AS CHARACTER   NO-UNDO.

c = "ELLO".

DISPLAY REPLACE(c, "E", "HE"). // Displays "HELLO"

c = "ABABABA".

DISPLAY REPLACE(c, "B", ""). // Remove all Bs

TRIM - baştaki ve sondaki boşlukları (veya diğer karakterleri) kaldırır.

Bu, verileri temizlerken faydalı olabilir.

TRIM(dize)

Baştaki ve sondaki tüm boşlukları, sekmeleri, satır beslemelerini, satır başlarını kaldırır.

TRIM(dize, karakter).

Baştaki ve sondaki tüm “karakterleri” kaldırır.

“SOL TRIM” ve “SAĞ TRIM” aynı şeyi yapar, ancak yalnızca önde veya arkadadır.

DEFINE VARIABLE c AS CHARACTER   NO-UNDO.

c = "__HELLO_WORLD_____".

DISPLAY TRIM(c, "_").
/*Displays HELLO_WORLD without all the leading and 
trailing underscores but leaves the one in the middle.
REPLACE would have removed that one as well */

SUBSTITUTE - bir dizgedeki parametreleri değiştirir.

SUBSTITUTE is a limited function for replacing up to nine preformatted parameters in a string.

İKAME(dize, param1, param2, …, param9).

Parametreler “&1” ile “&9” arasında olmalıdır.

Dizede bir ve işareti kullanmak istiyorsanız (ve onu parametre olarak kullanmıyorsanız) başka bir ve işaretiyle çıkış yapın: &&.

DEFINE VARIABLE str AS CHARACTER   NO-UNDO.
                                      
str = "&1 made &2 goals in &3 games playing for &4".

MESSAGE SUBSTITUTE(str, "Zlatan Ibrahimovic", 113, 122, "Paris Saint-Germain") VIEW-AS ALERT-BOX.
MESSAGE SUBSTITUTE(str, "Mats Sundin", 555, 1305, "Toronto Maple Leafs") VIEW-AS ALERT-BOX.

Bir parametre bir dizgede birden fazla görünebilir, tümü değiştirilecektir:

MESSAGE SUBSTITUTE("&1 &2 or not &1 &2", "To", "Be") VIEW-AS ALERT-BOX.

BAŞLAR ve MAÇLAR #

BAŞLAR - bir dize başka bir dizeyle başlarsa DOĞRU döndürür.

string1 string2 BAŞLIYOR

Eğer string1 string2 ile başlarsa (veya ona eşitse) bu true değerini döndürür. Aksi takdirde false döndürür. İkinci dize boşsa ("") her zaman true değerini döndürür.

BEGINS, örneğin bir ad gibi bir şeyin başlangıcını aramak istediğiniz sorgularda çok kullanışlıdır. Ama temelde dizeler üzerinde çalışan bir işlevdir.

DEFINE VARIABLE str AS CHARACTER   NO-UNDO.
DEFINE VARIABLE beg AS CHARACTER   NO-UNDO.

str = "HELLO".
beg = "HELLO".
DISPLAY str BEGINS beg. // yes

str = "HELLO".
beg = "H".
DISPLAY str BEGINS beg. // yes

str = "HELLO".
beg = "".
DISPLAY str BEGINS beg. // yes


str = "HELLO".
beg = "HELLO WORLD".
DISPLAY str BEGINS beg. // no

MATCHES, bir dizgede belirli joker karakter ölçütleri karşılanırsa true değerini döndürür.

string1 MATCHES ifadesi

string1 joker karakter ifadesiyle eşleşirse true döndürür:

* (yıldız) = 0 ila n karakter (temelde herhangi bir uzunluktaki herhangi bir dize)

. (nokta) = herhangi bir karakter için joker karakter (boş hariç)

DEFINE VARIABLE str AS CHARACTER   NO-UNDO.
DEFINE VARIABLE beg AS CHARACTER   NO-UNDO.

str = "HELLO".
beg = "HELLO".
DISPLAY str MATCHES beg. // yes

str = "HELLO".
beg = "H*".
DISPLAY str MATCHES beg. // yes

str = "HELLO".
beg = "*O".
DISPLAY str MATCHES beg. // yes

str = "HELLO WORLD".
beg = "HELLO.WORLD".
DISPLAY str MATCHES beg. // yes

str = "HELLO WORLD".
beg = "*WORL..".
DISPLAY str MATCHES beg. // no

str = "*HELLO WORLD".
beg = "WOR*LD".
DISPLAY str MATCHES beg. // no

Büyük ve küçük harfleri dönüştürme #

Daha önce belirtildiği gibi, dizeler normalde büyük/küçük harfe duyarlı değildir, ancak bu yalnızca dizelerin karşılaştırılmasıyla ilgilidir. Kasayı değiştirmek için yerleşik işlevler var.

BÜYÜK HARF (dize)

Dizeyi büyük harf yapar

LC(dize)

Dizeyi küçük harf yapar

DEFINE VARIABLE c AS CHARACTER   NO-UNDO.
DEFINE VARIABLE d AS CHARACTER   NO-UNDO.

c = "Hello".
d = "World".

DISPLAY CAPS(c) LC(d). // HELLO world

Dizelerin normalde büyük/küçük harfe duyarsız olduğunu unutmayın

DEFINE VARIABLE c AS CHARACTER   NO-UNDO.
DEFINE VARIABLE d AS CHARACTER   NO-UNDO.

c = "hello".
d = "hello".

DISPLAY CAPS(c) = LC(d). // yes

‘VAKA DUYARLI’ olarak belirtilmediği sürece

DEFINE VARIABLE c AS CHARACTER   NO-UNDO CASE-SENSITIVE.
DEFINE VARIABLE d AS CHARACTER   NO-UNDO.

c = "hello".
d = "hello".

DISPLAY CAPS(c) = LC(d). // no

Listeler #

Progress 4GL’de virgülle (veya diğer karakterlerle) ayrılmış listelerle çalışmak için bir dizi işlev ve yöntem vardır.

NUM-GİRİŞ Listedeki girişlerin sayısını döndürür. İsteğe bağlı olarak sınırlayıcı belirtebilirsiniz, virgül varsayılandır

NUM-GİRİŞ(dize [, sınırlayıcı])

Varsayılan sınırlayıcı olan virgül kullanarak:

DEFINE VARIABLE cList AS CHARACTER   NO-UNDO.

cList = "Goodbye,cruel,world!".

DISPLAY NUM-ENTRIES(cList). //3

Başka bir sınırlayıcı, noktalı virgül kullanarak:

DEFINE VARIABLE cList AS CHARACTER   NO-UNDO.

cList = "Goodbye;cruel;world!".

DISPLAY NUM-ENTRIES(cList, ";"). //3

ENTRY - işlev - bir listede belirtilen bir girişi döndürür

Her zamanki gibi başlangıç ​​pozisyonu 0 değil, 1’dir!

GİRİŞ( giriş, liste [, sınırlayıcı]).

DEFINE VARIABLE cList AS CHARACTER   NO-UNDO.

cList = "Goodbye,cruel,world!".

DISPLAY ENTRY(2, cList). //cruel

ENTRY - yöntem - bir listede belirtilen bir girişin değerini atama

GİRİŞ(giriş, liste [, sınırlayıcı]) = değer

DEFINE VARIABLE cList AS CHARACTER   NO-UNDO.

cList = "Goodbye,cruel,world!".

ENTRY(1, cList) = "Hello".
ENTRY(2, cList) = "nice".

MESSAGE REPLACE(cList, ",", " ") VIEW-AS ALERT-BOX. //Hello nice world!

ARA - belirli bir giriş için bir listeyi kontrol edin. Girişini döndürür.

Dize listede yoksa arama 0 döndürür

ARA(dize, liste [, sınırlayıcı])

DEFINE VARIABLE cList AS CHARACTER   NO-UNDO.

cList = "Hello,nice,world!".

MESSAGE LOOKUP("nice", cList) VIEW-AS ALERT-BOX. //2
MESSAGE LOOKUP("cruel", cList) VIEW-AS ALERT-BOX. //0

Özel karakterler (ve kaçan) #

Progress 4GL’de özel bir karakter yazmanın normal yolu, onun önüne bir tilde karakteri (~) koymaktır.

Bunlar varsayılan özel karakterlerdir

Sıra olarak yorumlandı Yorum
~" " “string” kullanılarak tanımlanan stringlerin içine " yazmak için kullanılır.
~’ ' ‘string’ kullanılarak tanımlanan dizgilerin içine ’ yazmak için kullanılır.
~~ ~ Örneğin, nasıl yorumlandığını değil, diziyi yazdırmak istiyorsanız.
~\ \
~{ { { önişlemcilerde kullanılır ve bazen kaçmak gerekir.
~nnn Tek bir karakter nnn, karakterin ascii değerini temsil eden sekizlik bir sayıdır.
~t sekme
~n Yeni satır/satır besleme
~r Taşıma iadesi
~E kaçış
~b Geri Al
~f Form besleme

Eğer tilde’yi hiç görüntülemek istiyorsanız, kaçılmalıdır!

MESSAGE "A single tilde: ~~" VIEW-AS ALERT-BOX.

MESSAGE "At sign: ~100" SKIP
        "Tab~tseparated~twords!" SKIP
        "A linefeed:~n"
        "Escaping a quote sign: ~"This is a quote!~"" SKIP VIEW-AS ALERT-BOX.
