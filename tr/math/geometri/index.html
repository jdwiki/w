Üç Noktadan Açıyı Hesapla #

Önce sorunu anlayalım, bu rakamı düşünün-

A, B & O bildiğimiz yerde ϴ hesaplamak istiyoruz.

Şimdi, ϴ elde etmek istiyorsak, önce α ve β‘yı bulmamız gerekiyor. Herhangi bir düz çizgi için, biliyoruz ki…

y = m * x + c

A = (ax, ay), B = (bx, by) ve O = (ox, oy) olsun. Yani OA- satırı için

oy = m1 * ox + c   ⇒ c = oy - m1 * ox   ...(eqn-1)

ay = m1 * ax + c   ⇒ ay = m1 * ax + oy - m1 * ox   [from eqn-1]
                   ⇒ ay = m1 * ax + oy - m1 * ox
                   ⇒ m1 = (ay - oy) / (ax - ox)
                   ⇒ tan α = (ay - oy) / (ax - ox)   [m = slope = tan ϴ]   ...(eqn-2)

Aynı şekilde OB- satırı için

tan β = (by - oy) / (bx - ox)   ...(eqn-3)

Şimdi ϴ = β - α ihtiyacımız var. Trigonometride bir formülümüz var-

tan (β-α) = (tan β + tan α) / (1 - tan β * tan α)   ...(eqn-4)

Denklem-4’te ’tan α’ (denk-2’den) ve ’tan b’ (denk-3’ten) değerlerini değiştirdikten ve sadeleştirme uyguladıktan sonra-

tan (β-α) = ( (ax-ox)*(by-oy)+(ay-oy)*(bx-ox) ) / ( (ax-ox)*(bx-ox)-(ay-oy)*(by-oy) )

Yani,

ϴ = β-α = tan^(-1) ( ((ax-ox)*(by-oy)+(ay-oy)*(bx-ox)) / ((ax-ox)*(bx-ox)-(ay-oy)*(by-oy)) )

İşte bu!

Şimdi, aşağıdaki şekli alın-

C# veya ardından Java yöntemi yukarıdaki teoriyi uygular-

    double calculateAngle(double P1X, double P1Y, double P2X, double P2Y,
            double P3X, double P3Y){
 
        double numerator = P2Y*(P1X-P3X) + P1Y*(P3X-P2X) + P3Y*(P2X-P1X);
        double denominator = (P2X-P1X)*(P1X-P3X) + (P2Y-P1Y)*(P1Y-P3Y);
        double ratio = numerator/denominator;

        double angleRad = Math.Atan(ratio);
        double angleDeg = (angleRad*180)/Math.PI;

        if(angleDeg<0){
            angleDeg = 180+angleDeg;
        }

        return angleDeg;
    }
