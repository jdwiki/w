‘MediaPlayer’ kullanımı esas olarak Durum diyagramına dayanmaktadır:

Bu, ses/video oynatmak için belirli bir sıra ile tanımlanmış bir eylem dizisinin gerçekleşmesi gerektiği anlamına gelir. Ayrıca hangi hangi durumda işlem yapılabileceğini belirtir.

MediaPlayer API, esneklikten yoksundur (özel kod çözücü ekleme ve işleme mantığı ekleme) ve HTTP üzerinden Dinamik Uyarlamalı Akış (DASH) ve SmoothStreaming için sSupport’tan yoksundur. Bunlar için ExoPlayer’e bakın.

Temel oluşturma ve oynama #

MediaPlayer sınıfı, ses/video dosyalarının ve akışların oynatılmasını kontrol etmek için kullanılabilir.

MediaPlayer nesnesinin oluşturulması üç tür olabilir:

  1. Yerel kaynaktan medya

     MediaPlayer mediaPlayer = MediaPlayer.create(context, R.raw.resource);
 mediaPlayer.start(); // no need to call prepare(); create() does that for you

  2. Yerel URI’den (ContentResolver’dan elde edilir)

     Uri myUri = ....; // initialize Uri here
 MediaPlayer mediaPlayer = new MediaPlayer();
 mediaPlayer.setAudioStreamType(AudioManager.STREAM_MUSIC);
 mediaPlayer.setDataSource(getApplicationContext(), myUri);
 mediaPlayer.prepare();
 mediaPlayer.start();

  3. Harici URL’den

     String url = "http://........"; // your URL here
 MediaPlayer mediaPlayer = new MediaPlayer();
 mediaPlayer.setAudioStreamType(AudioManager.STREAM_MUSIC);
 mediaPlayer.setDataSource(url);
 mediaPlayer.prepare(); // might take long! (for buffering, etc)
 mediaPlayer.start();

public class SoundActivity extends Activity  {
 
    private MediaPlayer mediaPlayer;
    ProgressBar progress_bar;
    
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_tool_sound);
        mediaPlayer = new MediaPlayer();
        mediaPlayer.setAudioStreamType(AudioManager.STREAM_MUSIC);
        progress_bar = (ProgressBar) findViewById(R.id.progress_bar);
   
        btn_play_stop.setEnabled(false);
        btn_play_stop.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View view) {
                if(mediaPlayer.isPlaying()) {
                    mediaPlayer.pause();
                    btn_play_stop.setImageResource(R.drawable.ic_pause_black_24dp);
                } else {
                    mediaPlayer.start();
                    btn_play_stop.setImageResource(R.drawable.ic_play_arrow_black_24px);
                }
            }
        });

        mediaPlayer.setDataSource(proxyUrl);
        mediaPlayer.setOnCompletionListener(new MediaPlayer.OnCompletionListener() {
            @Override
            public void onCompletion(MediaPlayer mp) {
                observer.stop();
                progress_bar.setProgress(mp.getCurrentPosition());
                // TODO Auto-generated method stub
                mediaPlayer.stop();
                mediaPlayer.reset();
            }
        });
        mediaPlayer.setOnBufferingUpdateListener(new MediaPlayer.OnBufferingUpdateListener() {
            @Override
            public void onBufferingUpdate(MediaPlayer mp, int percent) {
                progress_bar.setSecondaryProgress(percent);
            }
        });
        mediaPlayer.setOnPreparedListener(new MediaPlayer.OnPreparedListener() {
            @Override
            public void onPrepared(MediaPlayer mediaPlayer) {
                btn_play_stop.setEnabled(true);

            }
        });
        observer = new MediaObserver();
        mediaPlayer.prepare();
        mediaPlayer.start();
        new Thread(observer).start();
    }

   
    private MediaObserver observer = null;

    private class MediaObserver implements Runnable {
        private AtomicBoolean stop = new AtomicBoolean(false);

        public void stop() {
            stop.set(true);
        }

        @Override
        public void run() {
            while (!stop.get()) {
                progress_bar.setProgress((int)((double)mediaPlayer.getCurrentPosition() / (double)mediaPlayer.getDuration()*100));
                try {
                    Thread.sleep(200);
                } catch (Exception ex) {
                    Logger.log(ToolSoundActivity.this, ex);
                }

            }
        }
    }
 
    @Override
    protected void onDestroy() {
        super.onDestroy();
        mediaPlayer.stop();
    }
}

Asenkron hazırlık #

MediaPlayer$prepare() bir engelleme çağrısıdır ve yürütme tamamlanana kadar kullanıcı arayüzünü donduracaktır. Bu sorunu çözmek için MediaPlayer$prepareAsync() kullanılabilir.

mMediaPlayer = ... // Initialize it here
mMediaPlayer.setOnPreparedListener(new MediaPlayer.OnPreparedListener(){
    @Override
    public void onPrepared(MediaPlayer player) {
        // Called when the MediaPlayer is ready to play
        mMediaPlayer.start();
    }
}); // Set callback for when prepareAsync() finishes
mMediaPlayer.prepareAsync(); // Prepare asynchronously to not block the Main Thread

Senkronize işlemlerde, hatalar normalde bir istisna veya bir hata kodu ile bildirilir, ancak asenkron kaynakları kullandığınızda, uygulamanızın hatalardan uygun şekilde haberdar edildiğinden emin olmalısınız. MediaPlayer için,

mMediaPlayer.setOnErrorListener(new MediaPlayer.OnErrorListener(){
        @Override
        public boolean onError(MediaPlayer mp, int what, int extra) {
            // ... react appropriately ...
            // The MediaPlayer has moved to the Error state, must be reset!
            // Then return true if the error has been handled
        }
});

Sistem zil seslerini alma #

Bu örnek, sistem zil seslerinin (RingtoneManager.TYPE_RINGTONE) URI’lerinin nasıl getirileceğini gösterir:

private List<Uri> loadLocalRingtonesUris() {
        List<Uri> alarms = new ArrayList<>();
        try {
            RingtoneManager ringtoneMgr = new RingtoneManager(getActivity());
            ringtoneMgr.setType(RingtoneManager.TYPE_RINGTONE);

            Cursor alarmsCursor = ringtoneMgr.getCursor();
            int alarmsCount = alarmsCursor.getCount();
            if (alarmsCount == 0 && !alarmsCursor.moveToFirst()) {
                alarmsCursor.close();
                return null;
            }

            while (!alarmsCursor.isAfterLast() && alarmsCursor.moveToNext()) {
                int currentPosition = alarmsCursor.getPosition();
                alarms.add(ringtoneMgr.getRingtoneUri(currentPosition));
            }

        } catch (Exception ex) {
            ex.printStackTrace();
        }

        return alarms;
    }

Liste, istenen zil sesi türlerine bağlıdır. Olasılıklar şunlardır:

  • “RingtoneManager.TYPE_RINGTONE”
  • RingtoneManager.TYPE_NOTIFICATION
  • RingtoneManager.TYPE_ALARM
  • RingtoneManager.TYPE_ALL = TYPE_RINGTONE | TİP_BİLDİRİM | TYPE_ALARM


Zil Seslerini “android.media.Ringtone” olarak almak için her “Uri"nin “RingtoneManager” tarafından çözülmesi gerekir:

android.media.Ringtone osRingtone = RingtoneManager.getRingtone(context, uri);

Sesi çalmak için şu yöntemi kullanın:

public void setDataSource(Context context, Uri uri)

“android.media.MediaPlayer"dan. “MediaPlayer”, Durum şemasına göre başlatılmalı ve hazırlanmalıdır.

Sistem sesini alma ve ayarlama #

Ses akışı türleri #

Zil sesi akışlarının farklı profilleri vardır. Her birinin hacmi farklı.

Buradaki her örnek, ‘AudioManager.STREAM_RING’ akış türü için yazılmıştır. Ancak bu tek değil. Kullanılabilir akış türleri şunlardır:

  • “AKIŞ_ALARM”
  • “STREAM_DTMF”
  • “AKIM_MÜZİK”
  • “AKIŞ_BİLDİRİMİ”
  • “AKIŞ_RING”
  • AKIM_SİSTEM
  • “STREAM_VOICE_CALL”


Sesi ayarlama #

Belirli bir profilin hacmini almak için arayın:

AudioManager audio = (AudioManager) getActivity().getSystemService(Context.AUDIO_SERVICE);
int currentVolume = audioManager.getStreamVolume(AudioManager.STREAM_RING);

Bu değer, akış için maksimum değer bilinmediğinde çok az kullanışlıdır:

AudioManager audio = (AudioManager) getActivity().getSystemService(Context.AUDIO_SERVICE);
int streamMaxVolume = audioManager.getStreamMaxVolume(AudioManager.STREAM_RING);

Bu iki değerin oranı göreli bir hacim (0 < hacim < 1) verecektir:

float volume = ((float) currentVolume) / streamMaxVolume


Sesi tek adımda ayarlama #

Akışın sesini bir adım artırmak için, şunu arayın:

AudioManager audio = (AudioManager) getActivity().getSystemService(Context.AUDIO_SERVICE);
audio.adjustStreamVolume(AudioManager.STREAM_RING, AudioManager.ADJUST_RAISE, 0);

Akışın sesini bir adım düşürmek için şunu arayın:

AudioManager audio = (AudioManager) getActivity().getSystemService(Context.AUDIO_SERVICE);
audio.adjustStreamVolume(AudioManager.STREAM_RING, AudioManager.ADJUST_LOWER, 0);


MediaPlayer’ı belirli akış türünü kullanacak şekilde ayarlama #

Bunu yapmak için MediaPlayer sınıfından bir yardımcı fonksiyon vardır.
void setAudioStreamType(int streamtype) çağırmanız yeterlidir:

MediaPlayer mMedia = new MediaPlayer();
mMedia.setAudioStreamType(AudioManager.STREAM_RING);

Sesi androidstudio’ya aktarın ve oynatın #

Bu, bilgisayarınızda/dizüstü bilgisayarınızda bulunan bir ses dosyasının nasıl oynatılacağına dair bir örnektir. Önce res altında yeni bir dizin oluşturun ve bu şekilde raw olarak adlandırın.

Çalmak istediğiniz sesi bu klasöre kopyalayın. Bir .mp3 veya .wav dosyası olabilir.

Şimdi örneğin butona tıkladığınızda bu sesi çalmak istiyorsunuz, işte böyle yapılıyor

public class MainActivity extends AppCompatActivity {
@Override
public void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    setContentView(R.layout.aboutapp_activity);

    MediaPlayer song=MediaPlayer.create(this, R.raw.song);

    Button button=(Button)findViewById(R.id.button);
    button.setOnClickListener(new View.OnClickListener() {
        @Override
        public void onClick(View view) {
           song.start();
        }
    });
}
}

Bu, düğmeye tıklandığında şarkıyı yalnızca bir kez çalacaktır, şarkıyı her düğmede yeniden çalmak istiyorsanız, bunun gibi bir kod yazın tıklayın.

 public class MainActivity extends AppCompatActivity {
 @Override
 public void onCreate(Bundle savedInstanceState) {
   super.onCreate(savedInstanceState);
   setContentView(R.layout.aboutapp_activity);

   MediaPlayer song=MediaPlayer.create(this, R.raw.song);

   Button button=(Button)findViewById(R.id.button);
   button.setOnClickListener(new View.OnClickListener() {
    @Override
    public void onClick(View view) {
        if (song.isPlaying()) {
            song.reset();
            song= MediaPlayer.create(getApplicationContext(), R.raw.song);
       }
         song.start();
     }
   });
 }
}
