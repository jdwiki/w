Glide’ın resmi belgeleri çok daha iyi bir kaynaktır:

Glide v4 için bkz. http://bumptech.github.io/glide/. Glide v3 için bkz. https://github.com/bumptech/glide/wiki.

Glide, medya kod çözme, bellek ve disk önbelleğe alma ve kaynak havuzunu basit ve kullanımı kolay bir arayüze saran, Android için hızlı ve verimli bir açık kaynaklı medya yönetimi ve görüntü yükleme çerçevesidir.

Glide, video hareketsiz görüntüleri, görüntüleri ve hareketli GIF’leri getirmeyi, kodunu çözmeyi ve görüntülemeyi destekler. Glide, geliştiricilerin hemen hemen her ağ yığınına bağlanmasına olanak tanıyan esnek bir API içerir. Varsayılan olarak Glide, özel bir HttpUrlConnection tabanlı yığın kullanır, ancak bunun yerine Google’s Volley projesine veya Square’s OkHttp kitaplığına eklenti yardımcı program kitaplıkları da içerir.

Glide’ın birincil odak noktası, her türlü görüntü listesini kaydırmayı mümkün olduğunca sorunsuz ve hızlı hale getirmektir, ancak Glide, uzak bir görüntüyü getirmeniz, yeniden boyutlandırmanız ve görüntülemeniz gereken hemen hemen her durumda etkilidir.

Kaynak kodu ve daha fazla belge GitHub’da mevcuttur: https://github.com/bumptech/glide

Bir resim yükleme #

Resim görünümü #

Belirli bir URL’den, Uri’den, kaynak kimliğinden veya başka herhangi bir modelden bir “ImageView"a resim yüklemek için:

ImageView imageView = (ImageView) findViewById(R.id.imageView);
String yourUrl = "http://www.yoururl.com/image.png";

Glide.with(context)
    .load(yourUrl)
    .into(imageView);

Uris için, ‘yourUrl’yi Uri’nizle (‘content://media/external/images/1’) değiştirin. Drawable’lar için ‘yourUrl’ öğesini kaynak kimliğinizle (‘R.drawable.image’) değiştirin.

RecyclerView ve ListView #

ListView veya RecyclerView’da tamamen aynı satırları kullanabilirsiniz:

@Override
public void onBindViewHolder(RecyclerView.ViewHolder viewHolder, int position) {
    MyViewHolder myViewHolder = (MyViewHolder) viewHolder;
    String currentUrl = myUrls.get(position); 

    Glide.with(context)
        .load(currentUrl)
        .into(myViewHolder.imageView);
}

“onBindViewHolder"da bir yük başlatmak istemiyorsanız, “ImageView"ı değiştirmeden önce herhangi bir “ImageView” Glide’ın yönettiğinden “clear()” emin olun:

@Override
public void onBindViewHolder(RecyclerView.ViewHolder viewHolder, int position) {
    MyViewHolder myViewHolder = (MyViewHolder) viewHolder;
    String currentUrl = myUrls.get(position); 

    if (TextUtils.isEmpty(currentUrl)) {
        Glide.clear(viewHolder.imageView);
        // Now that the view has been cleared, you can safely set your own resource
        viewHolder.imageView.setImageResource(R.drawable.missing_image);
    } else {
        Glide.with(context)
            .load(currentUrl)
            .into(myViewHolder.imageView);
    }
}

Kaydırmalı daire dönüşümü (Görüntüyü dairesel bir ImageView’a yükleyin) #

Kaydırma ile bir daire görüntüsü oluşturun.

public class CircleTransform extends BitmapTransformation {

    public CircleTransform(Context context) {
        super(context);
    }

    @Override protected Bitmap transform(BitmapPool pool, Bitmap toTransform, int outWidth, int outHeight) {
        return circleCrop(pool, toTransform);
    }

    private static Bitmap circleCrop(BitmapPool pool, Bitmap source) {
        if (source == null) return null;

        int size = Math.min(source.getWidth(), source.getHeight());
        int x = (source.getWidth() - size) / 2;
        int y = (source.getHeight() - size) / 2;

        Bitmap squared = Bitmap.createBitmap(source, x, y, size, size);

        Bitmap result = pool.get(size, size, Bitmap.Config.ARGB_8888);
        if (result == null) {
            result = Bitmap.createBitmap(size, size, Bitmap.Config.ARGB_8888);
        }

        Canvas canvas = new Canvas(result);
        Paint paint = new Paint();
        paint.setShader(new BitmapShader(squared, BitmapShader.TileMode.CLAMP, BitmapShader.TileMode.CLAMP));
        paint.setAntiAlias(true);
        float r = size / 2f;
        canvas.drawCircle(r, r, r, paint);
        return result;
    }

    @Override public String getId() {
        return getClass().getName();
    }
}

Kullanım:

Glide.with(context)
    .load(yourimageurl)
    .transform(new CircleTransform(context))
    .into(userImageView);

Varsayılan dönüşümler #

Kayma, iki varsayılan dönüşüm içerir, merkeze sığdır ve merkez kırp.

Fit merkezi:

Glide.with(context)
    .load(yourUrl)
    .fitCenter()
    .into(yourView);

Fit center, Android’in ScaleType.FIT_CENTER ile aynı dönüşümü gerçekleştirir.

Orta kırpma:

Glide.with(context)
    .load(yourUrl)
    .centerCrop()
    .into(yourView);

Ortadan kırpma, Android’in ScaleType.CENTER_CROP ile aynı dönüşümü gerçekleştirir.

Daha fazla bilgi için Glide’ın wiki’sine bakın.

Özel Kaydırma hedefiyle yuvarlatılmış köşeler görüntüsünü kaydırın #

İlk önce yardımcı program sınıfı yapın veya bu yöntemi gerekli sınıfta kullanın

public class UIUtils {
public static BitmapImageViewTarget getRoundedImageTarget(@NonNull final Context context, @NonNull final ImageView imageView,
                                                        final float radius) {
    return new BitmapImageViewTarget(imageView) {
        @Override
        protected void setResource(final Bitmap resource) {
            RoundedBitmapDrawable circularBitmapDrawable =
                    RoundedBitmapDrawableFactory.create(context.getResources(), resource);
            circularBitmapDrawable.setCornerRadius(radius);
            imageView.setImageDrawable(circularBitmapDrawable);
        }
    };
}

Resim yükleniyor:

Glide.with(context)
     .load(imageUrl)
     .asBitmap()
     .into(UIUtils.getRoundedImageTarget(context, imageView, radius));

asBitmap() kullandığınız için animasyonlar kaldırılacaktır. animate() yöntemini kullanarak bu yerde kendi animasyonunuzu kullanabilirsiniz.

Varsayılan Glide animasyonuna benzer solmaya sahip örnek.

Glide.with(context)
     .load(imageUrl)
     .asBitmap()
     .animate(R.anim.abc_fade_in)
     .into(UIUtils.getRoundedImageTarget(context, imageView, radius));

Lütfen bu animasyonun destek kitaplığı özel kaynağı olduğunu unutmayın - değişebileceği ve hatta kaldırılabileceği için kullanılması önerilmez.

RoundedBitmapDrawableFactory kullanmak için destek kitaplığına da sahip olmanız gerektiğini unutmayın.

Yer tutucu ve Hata işleme #

Yükleme sırasında bir Drawable eklemek istiyorsanız, bir yer tutucu ekleyebilirsiniz:

Glide.with(context)
    .load(yourUrl)
    .placeholder(R.drawable.placeholder)
    .into(imageView);

Yük herhangi bir nedenle başarısız olursa bir Drawable’ın gösterilmesini istiyorsanız:

Glide.with(context)
    .load(yourUrl)
    .error(R.drawable.error)
    .into(imageView);

Boş bir model (URL, Uri, dosya yolu vb.) sağlarsanız bir Drawable’ın gösterilmesini istiyorsanız:

Glide.with(context)
    .load(maybeNullUrl)
    .fallback(R.drawable.fallback)
    .into(imageView);

Görüntüleri önceden yükleme #

Uzak görüntüleri önceden yüklemek ve görüntünün yalnızca bir kez indirildiğinden emin olmak için:

Glide.with(context)
    .load(yourUrl)
    .diskCacheStrategy(DiskCacheStrategy.SOURCE)
    .preload();

O zamanlar:

Glide.with(context)
    .load(yourUrl)
    .diskCacheStrategy(DiskCacheStrategy.SOURCE) // ALL works here too
    .into(imageView);

Yerel görüntüleri önceden yüklemek ve dönüştürülmüş bir kopyanın disk önbelleğinde (ve belki de bellek önbelleğinde) olduğundan emin olmak için:

Glide.with(context)
    .load(yourFilePathOrUri)
    .fitCenter() // Or whatever transformation you want
    .preload(200, 200); // Or whatever width and height you want

O zamanlar:

Glide.with(context)
    .load(yourFilePathOrUri)
    .fitCenter() // You must use the same transformation as above
    .override(200, 200) // You must use the same width and height as above
    .into(imageView);

Özel dönüşümler olmadan görüntüyü dairesel bir ImageView’e yükleyin #

Görüntüyü şuraya yüklemek için özel bir BitmapImageViewTarget oluşturun:

public class CircularBitmapImageViewTarget extends BitmapImageViewTarget
{
    private Context context;
    private ImageView imageView;

    public CircularBitmapImageViewTarget(Context context, ImageView imageView)
    {
        super(imageView);
        this.context = context;
        this.imageView = imageView;
    }
    
    @Override
    protected void setResource(Bitmap resource)
    {
        RoundedBitmapDrawable bitmapDrawable = RoundedBitmapDrawableFactory.create(context.getResources(), resource);
        bitmapDrawable.setCircular(true);
        imageView.setImageDrawable(bitmapDrawable);
    }
}

Kullanım:

Glide
    .with(context)
    .load(yourimageidentifier)
    .asBitmap()
    .into(new CircularBitmapImageViewTarget(context, imageView));
