Kayan eylem düğmesi, özel bir terfi eylemi türü için kullanılır, varsayılan olarak genişleyen bir malzeme parçası olarak ekranda canlanır. İçindeki simge canlandırılabilir, ayrıca FAB, göreceli önemlerinden dolayı diğer UI öğelerinden farklı hareket edebilir. Bir kayan eylem düğmesi, bir uygulamadaki yalnızca bir eylemi tetikleyebilen veya bir yerde gezinebilen birincil eylemi temsil eder.

Parametreler #

parametre Detay
android.support.design:yükseklik FAB için yükseklik değeri. “@[+][paket:]tür/ad” biçiminde başka bir kaynağa referans veya “?[paket:]tür/ad” biçiminde bir tema niteliği olabilir.
android.support.design:fabSize FAB için boyut.
android.support.design:rippleColor FAB için dalgalanma rengi.
android.support.design:useCompatPadding Uyumluluk dolgusunu etkinleştirin.

Kayan eylem düğmeleri, özel bir tür yükseltilmiş eylem için kullanılır. Kullanıcı arayüzünün üzerinde yüzen daire içine alınmış bir simge ile ayırt edilirler ve geçiş, fırlatma ve aktarma bağlantı noktası ile ilgili özel hareket davranışlarına sahiptirler.

En yaygın eylemi temsil etmek için ekran başına yalnızca bir kayan eylem düğmesi önerilir.

“FloatingActionButton"u kullanmadan önce, “build.gradle” dosyasına tasarım destek kitaplığı bağımlılığını eklemelisiniz:

dependencies {
    compile 'com.android.support:design:25.1.0'
}

Kaydırmada FloatingActionButton’u Göster ve Gizle #

Bir ‘FloatingActionButton’u varsayılan animasyonla göstermek ve gizlemek için ‘show()’ ve ‘hide()’ yöntemlerini çağırmanız yeterlidir. Bir Fragmente koymak yerine Aktivite düzeninde bir FloatingActionButton tutmak iyi bir uygulamadır, bu, gösterirken ve gizlerken varsayılan animasyonların çalışmasına izin verir.

İşte bir “ViewPager” ile bir örnek:

  • Üç Sekme
  • Birinci ve üçüncü Sekme için ‘FloatingActionButton’u göster
  • Orta Sekmedeki ‘FloatingActionButton’u gizleyin
public class MainActivity extends AppCompatActivity {

    FloatingActionButton fab;
    ViewPager viewPager;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        fab = (FloatingActionButton) findViewById(R.id.fab);
        viewPager = (ViewPager) findViewById(R.id.viewpager);

        // ...... set up ViewPager ............

        viewPager.addOnPageChangeListener(new ViewPager.OnPageChangeListener() {

            @Override
            public void onPageSelected(int position) {
                if (position == 0) {
                    fab.setImageResource(android.R.drawable.ic_dialog_email);
                    fab.show();
                } else if (position == 2) {
                    fab.setImageResource(android.R.drawable.ic_dialog_map);
                    fab.show();
                } else {
                    fab.hide();
                }
            }

            @Override
            public void onPageScrolled(int position, float positionOffset, int positionOffsetPixels) {}

            @Override
            public void onPageScrollStateChanged(int state) {}
        });

        // Handle the FloatingActionButton click event:
        fab.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View v) {
                int position = viewPager.getCurrentItem();
                if (position == 0) {
                    openSend();
                } else if (position == 2) {
                    openMap();
                }
            }
        });

    }
}

Kaydırmada FloatingActionButton’u Göster ve Gizle #

Destek Kitaplığı 22.2.1 sürümünden başlayarak, [show() özelliğinden yararlanan bir FloatingActionButton.Behavior alt sınıfını kullanarak bir FloatingActionButton öğesini kaydırma davranışından göstermek ve gizlemek mümkündür. ]3 ve hide() yöntemleri.

Bunun yalnızca RecyclerView ve NestedScrollView gibi İç İçe Kaydırmayı destekleyen iç Görünümlerle bağlantılı olarak bir CoordinatorLayout ile çalıştığını unutmayın.

Bu “ScrollAwareFABBehavior” sınıfı, Android Guides on Codepath (öznitelikli cc-wiki gerekli)‘dan gelir

public class ScrollAwareFABBehavior extends FloatingActionButton.Behavior {
    public ScrollAwareFABBehavior(Context context, AttributeSet attrs) {
        super();
    }

    @Override
    public boolean onStartNestedScroll(final CoordinatorLayout coordinatorLayout, final FloatingActionButton child,
                                       final View directTargetChild, final View target, final int nestedScrollAxes) {
        // Ensure we react to vertical scrolling
        return nestedScrollAxes == ViewCompat.SCROLL_AXIS_VERTICAL
                || super.onStartNestedScroll(coordinatorLayout, child, directTargetChild, target, nestedScrollAxes);
    }

    @Override
    public void onNestedScroll(final CoordinatorLayout coordinatorLayout, final FloatingActionButton child,
                               final View target, final int dxConsumed, final int dyConsumed,
                               final int dxUnconsumed, final int dyUnconsumed) {
        super.onNestedScroll(coordinatorLayout, child, target, dxConsumed, dyConsumed, dxUnconsumed, dyUnconsumed);
        if (dyConsumed > 0 && child.getVisibility() == View.VISIBLE) {
            // User scrolled down and the FAB is currently visible -> hide the FAB
            child.hide();
        } else if (dyConsumed < 0 && child.getVisibility() != View.VISIBLE) {
            // User scrolled up and the FAB is currently not visible -> show the FAB
            child.show();
        }
    }
}

FloatingActionButton düzeni xml’de, “ScrollAwareFABBehavior” tam nitelikli sınıf adıyla “app:layout_behavior” öğesini belirtin:

app:layout_behavior="com.example.app.ScrollAwareFABBehavior"

Örneğin bu düzen ile:

Renk

Bu görünümün arka plan rengi varsayılan olarak temanızın colorAccent’idir.

Yukarıdaki resimde src yalnızca + simgesini gösteriyorsa (varsayılan olarak 24x24 dp), tam dairenin arka plan rengini elde etmek için app:backgroundTint="@color/your_color" kullanabilirsiniz.

Koddaki rengi değiştirmek isterseniz kullanabilirsiniz,

myFab.setBackgroundTintList(ColorStateList.valueOf(int cinsinden renginiz));

FAB’nin rengini basılı durumda değiştirmek istiyorsanız,

mFab.setRippleColor(your color in int);

Konumlandırma

Mobil cihazlarda kenardan minimum 16dp ve tablet/masaüstünde minimum 24dp yerleştirilmesi önerilir.

Not : FloatingActionButton alanının tamamını kapsayacak şekilde bir src ayarladığınızda, en iyi sonucu elde etmek için o görüntünün doğru boyutuna sahip olduğunuzdan emin olun.

Varsayılan daire boyutu 56 x 56dp’dir

Mini daire boyutu : 40 x 40dp

Yalnızca İç Mekan simgesini değiştirmek istiyorsanız, varsayılan boyut için 24 x 24dp’lik bir simge kullanın

FloatingActionButton davranışını ayarlama #

FAB’nin davranışını XML olarak ayarlayabilirsiniz.

Örneğin:

<android.support.design.widget.FloatingActionButton    
   app:layout_behavior=".MyBehavior" />

Veya aşağıdakileri kullanarak programlı olarak ayarlayabilirsiniz:

CoordinatorLayout.LayoutParams p = (CoordinatorLayout.LayoutParams) fab.getLayoutParams();
p.setBehavior(xxxx);
fab.setLayoutParams(p);
