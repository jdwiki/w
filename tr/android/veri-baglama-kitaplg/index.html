Kurmak

Veri bağlamayı kullanmadan önce, build.gradle dosyanızda eklentiyi etkinleştirmelisiniz.

android {
    ....
    dataBinding {
        enabled = true
    }
}

Not: 1.5.0 sürümünde Android Gradle eklentisine veri bağlama eklendi

Sınıf adlarını bağlama

Veri bağlama eklentisi, mizanpajınızın dosya adını Pascal durumuna dönüştürerek ve sonuna “Binding” ekleyerek bir bağlayıcı sınıf adı oluşturur. Böylece “item_detail_activity.xml”, “ItemDetailActivityBinding” adında bir sınıf üretecektir.

Temel metin alanı bağlama #

Gradle (Modül:uygulama) Yapılandırması

android {
    ....
    dataBinding {
        enabled = true
    }
}

Veri örneği

public class Item {
    public String name;
    public String description;

    public Item(String name, String description) {
        this.name = name;
        this.description = description;
    }
}

Düzen XML

İlk adım, düzeninizi bir “” etiketine sarmak, bir “” öğesi eklemek ve veri modeliniz için bir “” öğesi eklemektir.

Ardından, “model” değişkenin adı ve “alanadı” erişmek istediğiniz alandır, “@{model.fieldname}” kullanarak veri modelindeki alanlara XML özniteliklerini bağlayabilirsiniz.

item_detail_activity.xml:

<?xml version="1.0" encoding="utf-8"?>
<layout xmlns:android="http://schemas.android.com/apk/res/android">
   <data>
       <variable name="item" type="com.example.Item"/>
   </data>

   <LinearLayout
       android:orientation="vertical"
       android:layout_width="match_parent"
       android:layout_height="match_parent">

       <TextView
           android:layout_width="wrap_content"
           android:layout_height="wrap_content"
           android:text="@{item.name}"/>

       <TextView
           android:layout_width="wrap_content"
           android:layout_height="wrap_content"
           android:text="@{item.description}"/>

   </LinearLayout>
</layout>

Bağlamalarla düzgün şekilde yapılandırılan her XML düzeni dosyası için, Android Gradle eklentisi karşılık gelen bir sınıf oluşturur: bağlamalar. item_detail_activity adlı bir düzenimiz olduğundan, ilgili oluşturulan bağlama sınıfına “ItemDetailActivityBinding” adı verilir.

Bu bağlama daha sonra şöyle bir Aktivitede kullanılabilir:

public class ItemDetailActivity extends Activity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
       super.onCreate(savedInstanceState);
       ItemDetailActivityBinding binding = DataBindingUtil.setContentView(this, R.layout.item_detail_activity);
       Item item = new Item("Example item", "This is an example item.");
       binding.setItem(item);
    }
}

<layout ...>

@={} ** Bağlama ifadesinin, iki yönlü Bağlama için gerekli olan ek bir =** içerdiğine dikkat edin. İki yönlü Binding ifadelerinde metot kullanmak mümkün değildir.

lambda ifadesini kullanan özel etkinlik #

Arayüz Tanımla

public interface ClickHandler {
    public void onButtonClick(User user);
}

Model sınıfı oluştur

public class User {
    private String name;

    public User(String name) {
        this.name = name;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }
}

Düzen XML

<layout xmlns:android="http://schemas.android.com/apk/res/android">

    <data>
        <variable
            name="handler"
            type="com.example.ClickHandler"/>

        <variable
            name="user"
            type="com.example.User"/>
    </data>

    <RelativeLayout
        android:layout_width="match_parent"
        android:layout_height="match_parent">

        <Button
            android:layout_width="wrap_content"
            android:layout_height="wrap_content"
            android:text="@{user.name}"
            android:onClick="@{() -> handler.onButtonClick(user)}"/>
    </RelativeLayout>
</layout>

Etkinlik kodu :

public class MainActivity extends Activity implements ClickHandler {

    private ActivityMainBinding binding;
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        binding = DataBindingUtil.setContentView(this,R.layout.activity_main);
        binding.setUser(new User("DataBinding User"));
        binding.setHandler(this);
    }

    @Override
    public void onButtonClick(User user) {
        Toast.makeText(MainActivity.this,"Welcome " + user.getName(),Toast.LENGTH_LONG).show();
    }
}

xml kodunda bulunmayan ancak java kodunda ayarlanabilen bazı görünüm dinleyicileri için özel bağlama ile bağlanabilir.

Veri Bağlamada varsayılan değer #

Önizleme bölmesi, veri bağlama ifadeleri için varsayılan değerleri görüntüler sağlanırsa.

Örneğin :

android:layout_height="@{@dimen/main_layout_height, default=wrap_content}"

Tasarım sırasında ‘wrap_content’ alacak ve önizleme bölmesinde ‘wrap_content’ olarak hareket edecektir.

Başka bir örnek

android:text="@{user.name, default=`Preview Text`}"

Önizleme bölmesinde ‘Önizleme Metni’ gösterecek, ancak onu cihazda/emülatörde çalıştırdığınızda, ona bağlanan gerçek metin görüntülenecektir.

Bir erişimci yöntemiyle bağlama #

Modelinizde özel yöntemler varsa, veri bağlama kitaplığı, yöntemin tam adını kullanmadan görünümünüzde bunlara erişmenize izin verir.

Veri örneği

public class Item {
    private String name;

    public String getName() {
        return name;
    }
}

Düzen XML

<?xml version="1.0" encoding="utf-8"?>
<layout xmlns:android="http://schemas.android.com/apk/res/android">
   <data>
       <variable name="item" type="com.example.Item"/>
   </data>

   <LinearLayout
       android:orientation="vertical"
       android:layout_width="match_parent"
       android:layout_height="match_parent">

       <!-- Since the "name" field is private on our data model,
            this binding will utilize the public getName() method instead. -->
       <TextView
           android:layout_width="wrap_content"
           android:layout_height="wrap_content"
           android:text="@{item.name}"/>

   </LinearLayout>
</layout>

public void doSomething() {
    DialogTestBinding binding = DataBindingUtil
            .inflate(LayoutInflater.from(context), R.layout.dialog_test, null, false);

    Dialog dialog = new Dialog(context);
    dialog.setContentView(binding.getRoot());
    dialog.show();
}

Referans sınıfları #

Veri örneği

public class Item {
    private String name;

    public String getName() {
        return name;
    }
}

Düzen XML

Başvurulan sınıfları tıpkı Java’da olduğu gibi içe aktarmalısınız.

<?xml version="1.0" encoding="utf-8"?>
<layout xmlns:android="http://schemas.android.com/apk/res/android">
   <data>
       <import type="android.view.View"/>
       <variable name="item" type="com.example.Item"/>
   </data>

   <LinearLayout
       android:orientation="vertical"
       android:layout_width="match_parent"
       android:layout_height="match_parent">

       <!-- We reference the View class to set the visibility of this TextView -->
       <TextView
           android:layout_width="wrap_content"
           android:layout_height="wrap_content"
           android:text="@{item.name}"
           android:visibility="@{item.name == null ? View.VISIBLE : View.GONE"/>

   </LinearLayout>
</layout>

Not: java.lang.* paketi sistem tarafından otomatik olarak içe aktarılır. (Aynı şey ‘Java’ için ‘JVM’ tarafından yapılır)

Bağdaştırıcı sınıfı #

public class ListItemAdapter extends RecyclerView.Adapter<RecyclerView.ViewHolder> {

    private Activity host;
    private List<Item> items;

    public ListItemAdapter(Activity activity, List<Item> items) {
        this.host = activity;
        this.items = items;
    }

    @Override
    public RecyclerView.ViewHolder onCreateViewHolder(ViewGroup parent, int viewType) {
        // inflate layout and retrieve binding
        ListItemBinding binding = DataBindingUtil.inflate(host.getLayoutInflater(),
                R.layout.list_item, parent, false);

        return new ItemViewHolder(binding);
    }

    @Override
    public void onBindViewHolder(RecyclerView.ViewHolder holder, int position) {
        Item item = items.get(position);

        ItemViewHolder itemViewHolder = (ItemViewHolder)holder;
        itemViewHolder.bindItem(item);
    }

    @Override
    public int getItemCount() {
        return items.size();
    }

    private static class ItemViewHolder extends RecyclerView.ViewHolder {
        ListItemBinding binding;

        ItemViewHolder(ListItemBinding binding) {
            super(binding.getRoot());
            this.binding = binding;
        }

        void bindItem(Item item) {
            binding.setItem(item);
            binding.executePendingBindings();
        }
    }
}

Binding ile dinleyiciyi tıklayın #

ClickHandler için arayüz oluştur

public interface ClickHandler {
    public void onButtonClick(View v);
}

Düzen XML

<?xml version="1.0" encoding="utf-8"?>
<layout xmlns:android="http://schemas.android.com/apk/res/android">

    <data>
        <variable
            name="handler"
            type="com.example.ClickHandler"/>
    </data>

    <RelativeLayout
        android:layout_width="match_parent"
        android:layout_height="match_parent">

        <Button
            android:layout_width="wrap_content"
            android:layout_height="wrap_content"
            android:text="click me"
            android:onClick="@{handler.onButtonClick}"/>
    </RelativeLayout>
</layout>

Etkinliğinizdeki etkinliği yönetin

public class MainActivity extends Activity implements ClickHandler {

    private ActivityMainBinding binding;
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        binding = DataBindingUtil.setContentView(this,R.layout.activity_main);
        binding.setHandler(this);
    }

    @Override
    public void onButtonClick(View v) {
        Toast.makeText(context,"Button clicked",Toast.LENGTH_LONG).show();
    }
}

Özel değişkenlerle DataBinding(int,boolean) #

Bazen tek bir değere dayalı görünümü gizle/göster gibi temel işlemleri yapmamız gerekiyor, o tek değişken için model oluşturamıyoruz veya bunun için model oluşturmak iyi bir uygulama değil. DataBinding, bu işlemleri gerçekleştirmek için temel veri türlerini destekler.

<layout xmlns:android="http://schemas.android.com/apk/res/android">

    <data>

        <import type="android.view.View" />

        <variable
            name="selected"
            type="Boolean" />

    </data>

    <RelativeLayout
        android:layout_width="match_parent"
        android:layout_height="match_parent">

        <TextView
            android:layout_width="wrap_content"
            android:layout_height="wrap_content"
            android:text="Hello World"
            android:visibility="@{selected ? View.VISIBLE : View.GONE}" />

    </RelativeLayout>
</layout>

ve değerini Java sınıfından ayarlayın.

binding.setSelected(true);

BindingAdapter yöntemi

@BindingAdapter({"imageUrl","progressbar"})
public static void loadImage(ImageView view, String imageUrl, ProgressBar progressBar){
    Glide.with(view.getContext()).load(imageUrl)
                        .listener(new RequestListener<String, GlideDrawable>() {
                    @Override
                    public boolean onException(Exception e, String model, Target<GlideDrawable> target, boolean isFirstResource) {
                        return false;
                    }

                    @Override
                    public boolean onResourceReady(GlideDrawable resource, String model, Target<GlideDrawable> target, boolean isFromMemoryCache, boolean isFirstResource) {
                        progressBar.setVisibility(View.GONE);
                        return false;
                    }
                }).into(view);
}
