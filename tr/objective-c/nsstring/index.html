NSString sınıfı, dizelerle (karakter dizileri) çalışmak için Foundation çerçevesinin bir parçasıdır. Ayrıca dizeleri karşılaştırma, arama ve değiştirme yöntemlerini de içerir.

Çeşitli nesne türlerini ve veri türlerini NSString’lere yerleştirmek için şuraya bakın: Objective-C, Format Specifiers

Kodlama ve Kod Çözme #

// decode
NSString *string = [[NSString alloc] initWithData:utf8Data
                                         encoding:NSUTF8StringEncoding];

// encode
NSData *utf8Data = [string dataUsingEncoding:NSUTF8StringEncoding];

Desteklenen bazı kodlamalar şunlardır:

  • NSASCIIStringEncoding
  • NSUTF8StringEncoding
  • NSUTF16StringEncoding (== NSUnicodeStringEncoding)

“utf8Data.bytes” öğesinin C dizeleri için gerekli olan bir sonlandırıcı boş karakter içermediğini unutmayın. Bir C dizesine ihtiyacınız varsa, “UTF8String"i kullanın:

const char *cString = [string UTF8String];
printf("%s", cString);

IP uzunluğu #

NSString, karakter sayısını almak için bir “uzunluk” özelliğine sahiptir.

NSString *string = @"example";
NSUInteger length = string.length;       // length equals 7

Bölme Örneği‘de olduğu gibi, “NSString"in karakterleri temsil etmek için UTF-16 kullandığını unutmayın. Uzunluk aslında sadece UTF-16 kod birimlerinin sayısıdır. Bu, kullanıcının karakter olarak algıladığından farklı olabilir.

İşte şaşırtıcı olabilecek bazı durumlar:

@"é".length == 1   // LATIN SMALL LETTER E WITH ACUTE (U+00E9)
@"é".length == 2   // LATIN SMALL LETTER E (U+0065) + COMBINING ACUTE ACCENT (U+0301)
@"❤️".length == 2  // HEAVY BLACK HEART (U+2764) + VARIATION SELECTOR-16 (U+FE0F)
@"🇮🇹".length == 4  // REGIONAL INDICATOR SYMBOL LETTER I (U+1F1EE) + REGIONAL INDICATOR SYMBOL LETTER T (U+1F1F9)

Teknik olarak “grapheme clusters” olarak bilinen, kullanıcı tarafından algılanan karakterlerin sayısını elde etmek için, [-enumerateSubstringsInRange:options:usingBlock:][eSIR⭕uB] ile dizeyi yinelemelisiniz. ve bir sayı tutun. Bu, [Nikolai Ruhe’un Stack Overflow’a verdiği yanıtta][NRanswer] gösterilmiştir.

[eSIR⭕uB]:https://developer.apple.com/library/ios/documentation/Cocoa/Reference/Foundation/Classes/NSString_Class/#//apple_ref/occ/instm/NSString/enumerateSubstringsInRange:options:usingBlock : [NRanswer]:http://stackoverflow.com/a/33539320/603977

Dizeleri Karşılaştırma #

isEqualToString: kullanılarak dizeler eşitlik açısından karşılaştırılır

== operatörü yalnızca nesne kimliğini test eder ve nesnelerin mantıksal değerlerini karşılaştırmaz, bu nedenle kullanılamaz:

NSString *stringOne = @"example";
NSString *stringTwo = [stringOne mutableCopy];

BOOL objectsAreIdentical = (stringOne == stringTwo);          // NO
BOOL stringsAreEqual = [stringOne isEqualToString:stringTwo]; // YES

(stringOne == stringTwo) ifadesi, iki dizenin bellek adreslerinin aynı olup olmadığını test eder, bu genellikle istediğimiz şey değildir.

Dize değişkenleri “nil” olabilirse, bu duruma da dikkat etmeniz gerekir:

BOOL equalValues = stringOne == stringTwo || [stringOne isEqualToString:stringTwo];

Bu koşul, dizeler eşit değerlere sahip olduğunda veya her ikisi de “nil” olduğunda “EVET” döndürür.

İki diziyi alfabetik olarak sıralamak için ‘karşılaştır’: kullanın.

NSComparisonResult result = [firstString compare:secondString];

‘NSComparisonResult’ şunlar olabilir:

  • NSOrderedAscending: İlk dize ikinci dizeden önce gelir.
  • NSOrderedSame: Dizeler eşittir.
  • NSOrderedDescending: İkinci dize ilk dizeden önce gelir.

İki dize eşitliğini karşılaştırmak için isEqualToString: kullanın.

BOOL result = [firstString isEqualToString:secondString];

Boş dizeyle (@"") karşılaştırmak için “uzunluk"u kullanmak daha iyidir.

BOOL result = string.length == 0;

Bölme #

Bir dizeyi, ayırıcı karakter ile bölünen bir dizi parçaya bölebilirsiniz.

NSString * yourString = @"Stack,Exchange,Network";
NSArray * yourWords = [yourString componentsSeparatedByString:@","]; 
// Output: @[@"Stack", @"Exchange", @"Network"]

Bir dizi birkaç farklı sınırlayıcı üzerinde bölme yapmanız gerekiyorsa, -[NSString componentSeparatedByCharactersInSet:] kullanın.

NSString * yourString = @"Stack Overflow+Documentation/Objective-C";
NSArray * yourWords = [yourString componentsSeparatedByCharactersInSet:
                      [NSCharacterSet characterSetWithCharactersInString:@"+/"]];
// Output: @[@"Stack Overflow", @"Documentation", @"Objective-C"]`

Bir dizeyi bireysel karakterlerine bölmeniz gerekiyorsa, döngü dizenin uzunluğu boyunca ve her karakteri yeni bir dizeye dönüştürün.

NSMutableArray * characters = [[NSMutableArray alloc] initWithCapacity:[yourString length]];
for (int i = 0; i < [myString length]; i++) {
    [characters addObject: [NSString stringWithFormat:@"%C", 
                                      [yourString characterAtIndex:i]];
}

[Uzunluk Örneği][Uzunluk]’ta olduğu gibi, burada bir “karakter"in bir UTF-16 kod birimi olduğunu, kullanıcının karakter olarak gördüğü gibi olmadığını unutmayın. Bu döngüyü @"🇮🇷" ile kullanırsanız dörde bölündüğünü göreceksiniz.

Kullanıcı tarafından algılanan karakterlerin bir listesini almak için [-enumerateSubstringsInRange:options:usingBlock:][eSIR⭕uB] kullanın.

NSMutableArray * characters = [NSMutableArray array];
[yourString enumerateSubstringsInRange:(NSRange){0, [yourString length]}
                               options:NSStringEnumerationByComposedCharacterSequences
                            usingBlock:^(NSString * substring, NSRange r, NSRange s, BOOL * b){
                                [characters addObject:substring];
                            }];

Bu, İtalyan bayrağı gibi grafik kümelerini tek bir alt dize olarak korur.

Oluşturma #

Basit:

NSString *newString = @"My String";

Birden çok dizeden:

NSString *stringOne = @"Hello";
NSString *stringTwo = @"world";
NSString *newString = [NSString stringWithFormat:@"My message: %@ %@",
                     stringOne, stringTwo];

Değişken Dize Kullanma

NSString *stringOne = @"Hello";
NSString *stringTwo = @"World";
NSMutableString *mutableString = [NSMutableString new];
[mutableString appendString:stringOne];
[mutableString appendString:stringTwo];

NSData’dan:

‘NSData’dan başlatılırken, ‘NSString’ karakterlerin ham veri akışında nasıl temsil edildiğini tahmin edemediği için açık bir kodlama sağlanmalıdır. Günümüzde en yaygın kodlama, JSON gibi belirli veriler için bile bir gereklilik olan UTF-8’dir.

Açıkça NULL ile sonlandırılmış bir C-dizesi beklediğinden +[NSString stringWithUTF8String:] kullanmaktan kaçının, bu -[NSData bytes] _sağlamaz.

NSString *newString = [[NSString alloc] initWithData:myData encoding:NSUTF8StringEncoding];

NSArray’den:

NSArray *myArray = [NSArray arrayWithObjects:@"Apple", @"Banana", @"Strawberry", @"Kiwi", nil];
NSString *newString = [myArray componentsJoinedByString:@" "];

Alt Dizi Arama #

Bir Dizenin alt dize içerip içermediğini aramak için aşağıdakileri yapın:

NSString *myString = @"This is for checking substrings";
NSString *subString = @"checking"; 

BOOL doesContainSubstring = [myString containsString:subString];  // YES

iOS 7 veya OS X 10.9’u (veya önceki sürümleri) hedefliyorsanız:

BOOL doesContainSubstring = ([myString rangeOfString:subString].location != NSNotFound);  // YES

Vaka Değiştirme #

Bir String’i büyük harfe dönüştürmek için “uppercaseString"i kullanın:

NSString *myString = @"Emphasize this";
NSLog(@"%@", [myString uppercaseString]; // @"EMPHASIZE THIS"

Bir String’i küçük harfe dönüştürmek için “lowercaseString"i kullanın:

NSString *myString = @"NORMALIZE this";
NSLog(@"%@", [myString lowercaseString]; // @"normalize this"

Bir dizedeki her kelimenin ilk harfini büyük harf yapmak için “capitalizedString” kullanın:

NSString *myString = @"firstname lastname";
NSLog(@"%@", [myString capitalizedString]); // @"Firstname Lastname"

Bir Dize Dizisine Katılma #

“NSString"in “NSArray"ini yeni bir “NSString” ile birleştirmek için:

NSArray *yourWords = @[@"Objective-C", @"is", @"just", @"awesome"];
NSString *sentence = [yourWords componentsJoinedByString:@" "];

// Sentence is now: @"Objective-C is just awesome"

Baştaki ve Sondaki Boşluğu Kaldırma #

NSString *someString = @"   Objective-C Language  \n";
NSString *trimmedString = [someString stringByTrimmingCharactersInSet:[NSCharacterSet whitespaceAndNewlineCharacterSet]];
//Output will be - "Objective-C Language"

stringByTrimmingCharactersInSet yöntemi, belirli bir karakter kümesinde bulunan Dize karakterlerinin her iki ucundan kaldırılarak oluşturulan yeni bir dize döndürür.

Ayrıca yalnızca boşluk veya yeni satırı kaldırabiliriz

// Removing only WhiteSpace
NSString *trimmedWhiteSpace = [someString stringByTrimmingCharactersInSet:[NSCharacterSet whitespaceCharacterSet]];
//Output will be - "Objective-C Language  \n"

// Removing only NewLine
NSString *trimmedNewLine = [someString stringByTrimmingCharactersInSet:[NSCharacterSet newlineCharacterSet]];
//Output will be - "   Objective-C Language  "

Biçimlendirme #

‘NSString’ biçimlendirmesi, ‘printf’ ANSI-C işlevinde bulunan tüm biçim dizelerini destekler. Dil tarafından yapılan tek ekleme, tüm Objective-C nesnelerini biçimlendirmek için kullanılan %@ sembolüdür.

Tam sayıları biçimlendirmek mümkündür

int myAge = 21;
NSString *formattedAge = [NSString stringWithFormat:@"I am %d years old", my_age];

Veya NSObject’ten alt sınıflanan herhangi bir nesne

NSDate *now = [NSDate date];
NSString *formattedDate = [NSString stringWithFormat:@"The time right now is: %@", now];

Format Belirticilerin tam listesi için lütfen bakınız: Objective-C, Format Specifiers, Syntax

C Dizeleriyle Çalışmak #

“NSString"i “const char"a dönüştürmek için -[NSString UTF8String] kullanın. .html#//apple_ref/occ/instm/NSString/UTF8String):

NSString *myNSString = @"Some string";
const char *cString = [myNSString UTF8String];

Ayrıca -[NSString cStringUsingEncoding:] kullanabilirsiniz. /occ/instm/NSString/cStringUsingEncoding:) dizeniz UTF-8’den farklı bir şeyle kodlanmışsa.

Ters yol için -[NSString stringWithUTF8String:] kullanın occ/clm/NSString/stringWithUTF8String:):

const *char cString = "Some string";
NSString *myNSString = [NSString stringWithUTF8String:cString];
myNSString = @(cString); // Equivalent to the above.

‘const char *’ elde ettiğinizde, onunla bir ‘chars’ dizisine benzer şekilde çalışabilirsiniz:

printf("%c\n", cString[5]);

Dizeyi değiştirmek istiyorsanız, bir kopyasını alın:

char *cpy = calloc(strlen(cString)+1, 1);
strncpy(cpy, cString, strlen(cString));
// Do stuff with cpy
free(cpy);
