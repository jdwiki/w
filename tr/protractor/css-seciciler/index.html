Sözdizimi #

  • by.css(‘css seçici’)
  • by.id(‘id’)
  • by.model(‘model’)
  • by.binding(‘bağlayıcı’)

Parametreler #

parametre Ayrıntılar
css seçici Sınıf adının temelinde öğeyi seçmek için '.class-name' gibi bir css seçici
kimlik dom öğesinin kimliği
model Dom elemanı için kullanılan model
bağlama Belirli bir öğeye bağlanmak için kullanılan bağlamanın adı

Css seçicileri nasıl yazılır?

CSS seçicileri yazmak için en önemli nitelikler dom sınıfı ve kimliğidir. Bir örnek için, bir html dom aşağıdaki örneğe benziyorsa:

  <form class="form-signin">
      <input type="text" id="email" class="form-control" placeholder="Email">
      <input type="password" id="password" class="form-control" placeholder="Password">
      <button class="btn btn-block" id="signin-button" type="submit">Sign in</button>
  </form>

Ardından e-posta giriş alanını seçmek için css seçiciyi şu şekilde yazabilirsiniz:

  1. Sınıf adını kullanma: css seçicideki sınıf adı özel karakter .(nokta) ile başlar. Bunun için css seçicisi bu “.form-control” gibi olacaktır.

    by.css('.form-control')

‘form-control’ sınıfı her iki girdi öğesi tarafından da paylaşıldığından, konum belirleyicilerde yineleme endişesi yaratır. Dolayısıyla böyle bir durumda id varsa, o zaman her zaman sınıf adı yerine id kullanmayı tercih etmelisiniz.

  1. Kimliği kullanma: css seçicideki kimlik, özel karakter # (karma) ile başlar. Bu nedenle, e-posta giriş öğesi için kimliği kullanan css seçici aşağıdaki gibi yazılacaktır:

    by.css('#email')

  2. Birden çok sınıf adı kullanma: Dom elemanının birden fazla sınıfı varsa, o zaman sınıfların kombinasyonunu css seçici olarak yapabilirsiniz. Örneğin dom öğesi şöyleyse:

    // css selector using multiple classes by.css('.username-class.form-control')

  3. Etiket adını diğer özniteliklerle kullanma : Etiket adını ve diğer öznitelikleri kullanarak css seçici yazmak için genel ifade tagname[attribute-type='attribute-vallue'] şeklindedir. Bu nedenle, oturum açma düğmesi için css bulucu ifadesinin ardından şu şekilde oluşturulabilir:

    by.css(“button[type=‘submit’]”) //or by.css(“button[id=‘signin-button’]”)

$ ve $$ CSS seçici bulucu kısayolları #

İletki API’si, CSS öğesi konum belirleyicilerinin jQuery benzeri kısayol gösterimi $() kullanmasına izin verir.

Normal CSS Öğe Bulucu:

element(by.css('h1.documentation-text[ng-bind="title"]'));
element(by.css('[ng-click="submit"]));

Kısayol $() CSS Öğe Bulucu:

$('h1.documentation-text[ng-bind="title"]');
$('[ng-click="submit"]');

Bir konumlandırıcı altında birden çok öğe bulmak için kısayol gösterimi $$() kullanın.

Normal CSS Öğe Bulucu:

element.all(by.css('h1.documentation-text[ng-bind="title"]'));
element.all(by.css('[ng-click="submit"]));

Kısayol $$() CSS Öğe Bulucu:

$$('h1.documentation-text[ng-bind="title"]');
$$('[ng-click="submit"]');

Tam bir HTML öznitelik değerine göre öğe seçin #

Tam bir HTML özniteliğine göre bir öğe seçmek için css bulucu desenini kullanın [attribute=value]

//selects the first element with href value '/contact'
element(by.css('[href="/contact"]')); 

//selects the first element with tag option and value 'foo'
element(by.css('option[value="foo"]')); 

//selects all input elements nested under the form tag with name attribute 'email'
element.all(by.css('form input[name="email"]'));

Belirtilen değeri içeren bir HTML niteliğine göre öğe seçin #

Belirtilen bir değeri içeren bir HTML özniteliğine göre bir öğe seçmek için css bulucu desenini kullanın [attribute*=value]

//selects the first element with href value that contains'cont'
element(by.css('[href*="cont"]')); 

//selects the first element with tag h1 and class attribute that contains 'fo'
element(by.css('h1[class*="fo"]')); 

//selects all li elements with a title attribute that contains 'users'
element.all(by.css('li[title*='users']'));

Konum belirleyicilere giriş #

İletki içindeki bir konum belirleyici, HTML dom öğeleri üzerinde eylem gerçekleştirmek için kullanılır. İletki’de kullanılan en yaygın ve en iyi konum belirleyiciler css, id, model ve bağlamadır. Örneğin, yaygın olarak kullanılan konum belirleyiciler şunlardır:

by.css('css-selector')
by.id('id')
