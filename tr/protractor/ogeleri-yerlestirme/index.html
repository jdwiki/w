Bir sayfayla etkileşime girebilmek için İletki’ye tam olarak hangi öğeyi arayacağını söylemeniz gerekir. Öğeleri seçmek için kullanılan temel, konum belirleyicilerdir. İletki, genel Selenium seçicileri içermesinin yanı sıra, daha sağlam ve değişikliklere karşı kalıcı olan Açısal’a özgü konum belirleyicilere de sahiptir. Ancak bazen Angular bir uygulamada bile normal konumlandırıcılar kullanılmalıdır.

Parametreler #

parametre Detay
seçici Seçicinin değerini belirten bir dize (yer belirleyiciye bağlıdır)

İletkiye özel konumlandırıcılar (Açısal tabanlı uygulamalar için) #

Bu konum belirleyiciler, bir uygulamadaki değişikliklere karşı daha kalıcı olduklarından, kolayca kırılabilen css veya xpath tabanlı konumlayıcılardan daha kalıcı olduklarından, mümkün olduğunda öncelikli olarak kullanılmalıdır.

bağlama bulucu #

Sözdizimi

by.binding('bind value')

Örnek #

Görüş

<span>{{user.password}}</span>
<span ng-bind="user.email"></span>

Konumlandırıcı

by.binding('user.password')
by.binding('user.email')

Kısmi eşleşmeleri de destekler

by.binding('email')

Tam Bağlama bulucu #

Kısmi eşleşmelere izin verilmemesi dışında “bağlama"ya benzer.

Sözdizimi

by.exactBinding('exact bind value')

Örnek #

Görüş

<span>{{user.password}}</span>

Konumlandırıcı

by.exactBinding('user.password')
by.exactBinding('password') // Will not work

Model bulucu #

Angular model yönergesi ile bir öğe seçer

Sözdizimi

by.model('model value')

Örnek #

Görüş

<input ng-model="user.username">

Konumlandırıcı

by.model('user.username')

Düğme metni bulucu #

Metnine göre bir düğme seçer. Yalnızca düğme metninin sık sık değişmesi beklenmiyorsa kullanılmalıdır.

Sözdizimi

by.buttonText('button text')

Örnek #

Görüş

<button>Sign In</button>

Konumlandırıcı

by.buttonText('Sign In')

Kısmi düğme metin bulucu #

“buttonText"e benzer, ancak kısmi eşleşmelere izin verir. Yalnızca düğme metninin sık sık değişmesi beklenmiyorsa kullanılmalıdır.

Sözdizimi

by.partialButtonText('partial button text')

Örnek

Görüş

<button>Register an account</button>

Konumlandırıcı

by.partialButtonText('Register')

Tekrarlayıcı bulucu #

Açısal tekrarlayıcı yönergesi ile bir öğe seçer

Sözdizimi

by.repeater('repeater value')

Örnek #

Görüş

<tbody ng-repeat="review in reviews">
    <tr>Movie was good</tr>
    <tr>Movie was ok</tr>
    <tr>Movie was bad</tr>
</tbody>

Konumlandırıcı

by.repeater('review in reviews')

Kısmi eşleşmeleri de destekler

by.repeater('reviews')

Tam tekrarlayıcı bulucu #

“Tekrarlayıcı"ya benzer, ancak kısmi eşleşmelere izin vermez

Sözdizimi

by.exactRepeater('exact repeater value')

Örnek #

Görüş

<tbody ng-repeat="review in reviews">
    <tr>Movie was good</tr>
    <tr>Movie was ok</tr>
    <tr>Movie was bad</tr>
</tbody>

Konumlandırıcı

by.exactRepeater('review in reviews')
by.exactRepeater('reviews') // Won't work

CSS ve metin bulucu #

Öğenin metin içeriğini de belirtebileceğiniz genişletilmiş bir CSS bulucu.

Sözdizimi

by.cssContainingText('css selector', 'text of css element')

Örnek #

Görüş

<ul>
    <li class="users">Mike</li>
    <li class="users">Rebecca</li>
</ul>

Konumlandırıcı

by.cssContainingText('.users', 'Rebecca') // Will return the second li only

Seçenekler bulucu #

Açısal seçenekler yönergesi ile bir öğe seçer

Sözdizimi

by.options('options value')

Örnek #

Görüş

<select ng-options="country.name for c in countries">
    <option>Canada</option>
    <option>United States</option>
    <option>Mexico</option>
</select>

Konumlandırıcı

by.options('country.name for c in countries')

Derin CSS bulucu #

gölge DOM içine uzanan CSS bulucu

Sözdizimi

by.deepCss('css selector')

Örnek #

Görüş

<div>
    <span id="outerspan">
    <"shadow tree">
        <span id="span1"></span>
        <"shadow tree">
            <span id="span2"></span>
        </>
    </>
</div>

Konumlandırıcı

by.deepCss('span') // Will select every span element

Konum bulmayla ilgili temel bilgiler #

Konumlandırıcılar kendi başlarına İletki’de etkileşime girebilecek bir öğe döndürmezler, yalnızca İletki’ye öğeyi nasıl bulacağını gösteren talimatlardır.

Öğenin kendisine erişmek için şu söz dizimini kullanın:

element(locator);
element.all(locator);

Not: Eleman(lar)a, üzerinde bir eylem gerçekleştirilinceye kadar gerçekte erişilmez - yani İletki, yalnızca öğe üzerinde getText() gibi bir eylem çağrıldığında öğeyi almaya gider.

Konumlandırıcı kullanarak yalnızca bir öğe seçmek istiyorsanız, “element” öğesini kullanın. Konum belirleyiciniz birden fazla öğeye işaret ediyorsa, “eleman” bulunan ilk öğeyi döndürür. “element”, bir “ElementFinder” döndürür.

Bir konumlandırıcı kullanarak birden fazla öğe seçmek istiyorsanız, “element.all” bulunan tüm öğeleri döndürür. “element.all” bir “ElementArrayFinder” döndürür ve dizideki her öğeye farklı yöntemler kullanılarak erişilebilir - örneğin, “map” işlevi.

element.all(locator).map(function(singleElement) {
        return singleElement.getText();
    }
});

Zincirleme konumlandırıcıları

Karmaşık bir uygulamada bir öğeyi seçmek için birden çok konumlandırıcıyı zincirleyebilirsiniz. “Locator” nesnelerini doğrudan zincirleyemezsiniz, “ElementFinders"ı zincirlemeniz gerekir:

element(by.repeater('movie in movies').element(by.linkText('Watch Frozen on Netflix')

Kaç tane zincir kullanabileceğinizin bir sınırı yoktur; sonunda, bulucularınıza bağlı olarak tek bir ‘ElementFinder’ veya ve ‘ElementArrayFinder’ alacaksınız.
