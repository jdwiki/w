Kaynaktan Diğer Düğümlere En Kısa Yolu Bulma #

Breadth-first-search (BFS), ağaç veya grafik veri yapılarının arasında gezinme veya arama için bir algoritmadır. Ağaç kökünde (veya bazen bir ‘arama anahtarı’ olarak adlandırılan bir grafiğin rastgele bir düğümünde) başlar ve bir sonraki seviye komşulara geçmeden önce komşu düğümleri araştırır. BFS, 1950’lerin sonlarında Edward Forrest Moore tarafından icat edildi ve onu bir labirentten çıkan en kısa yolu bulmak için kullandı ve bağımsız olarak C. Y. Lee tarafından keşfedildi. 1961’de bir kablo yönlendirme algoritması.

BFS algoritmasının süreçleri şu varsayımlar altında çalışır:

  1. Herhangi bir düğümü bir kereden fazla geçmeyeceğiz.
  2. Kaynak düğüm veya başladığımız düğüm, 0 düzeyinde bulunur.
  3. Kaynak düğümden doğrudan ulaşabileceğimiz düğümler 1. düzey düğümlerdir, 1. düzey düğümlerden doğrudan ulaşabildiğimiz düğümler 2. düzey düğümlerdir vb.
  4. Seviye, kaynaktan en kısa yolun mesafesini gösterir.

Bir örnek görelim:

Bu grafiğin birden fazla şehir arasındaki bağlantıyı temsil ettiğini, her bir düğümün bir şehri temsil ettiğini ve iki düğüm arasındaki bir kenarın onları birbirine bağlayan bir yol olduğunu gösterdiğini varsayalım. düğüm 1‘den düğüm 10‘a gitmek istiyoruz. Yani düğüm 1, seviye 0 olan kaynağımızdır. düğüm 1‘i ziyaret edildi olarak işaretliyoruz. Buradan düğüm 2, düğüm 3 ve düğüm 4’e gidebiliriz. Yani düzey (0+1) = düzey 1 düğümler olacaklar. Şimdi onları ziyaret edildi olarak işaretleyeceğiz ve onlarla çalışacağız.

Renkli düğümler ziyaret edilir. Şu anda üzerinde çalıştığımız düğümler pembe ile işaretlenecektir. Aynı düğümü iki kez ziyaret etmeyeceğiz. düğüm 2, düğüm 3 ve düğüm 4’ten düğüm 6, düğüm 7 ve düğüm 8’e gidebiliriz. Onları ziyaret edildi olarak işaretleyelim. Bu düğümlerin seviyesi seviye (1+1) = seviye 2 olacaktır.

Fark etmediyseniz, düğümlerin düzeyi yalnızca kaynağa olan en kısa yol mesafesini belirtir. Örneğin: düzey 2‘de düğüm 8’i bulduk. Yani kaynak ile düğüm 8 arasındaki mesafe 2‘dir.

Hedef düğümümüze, yani düğüm 10‘a henüz ulaşmadık. Öyleyse sonraki düğümleri ziyaret edelim. doğrudan düğüm 6, düğüm 7 ve düğüm 8‘den gidebiliriz.

Bunu görebiliyoruz, düğüm 10’u seviye 3’te bulduk. Yani source ile node 10 arasındaki en kısa yol 3. Grafiği seviye bazında araştırdık ve en kısa yolu bulduk. Şimdi kullanmadığımız kenarları silelim:

Kullanmadığımız kenarları çıkardıktan sonra BFS ağacı adında bir ağaç elde ediyoruz. Bu ağaç, kaynak ile diğer tüm düğümler arasındaki en kısa yolu gösterir.

Dolayısıyla görevimiz kaynak’tan seviye 1 düğümlere gitmek olacaktır. Daha sonra seviye 1‘den seviye 2 düğümlerine ve bu şekilde hedefimize ulaşana kadar. İşleyeceğimiz düğümleri saklamak için queue kullanabiliriz. Yani, birlikte çalışacağımız her düğüm için, doğrudan geçilebilen ve henüz kuyrukta geçilemeyen tüm diğer düğümleri iteceğiz.

Örneğimizin simülasyonu:

İlk önce kaynağı kuyruğa itiyoruz. Sıramız şöyle görünecek:

 front
+-----+
|  1  |
+-----+

1. düğüm düzeyi 0 olacaktır. düzey[1] = 0. Şimdi BFS’mize başlıyoruz. İlk önce kuyruğumuzdan bir düğüm açıyoruz. düğüm 1‘i alıyoruz. Bundan düğüm 4, düğüm 3 ve düğüm 2‘ye gidebiliriz. Bu düğümlere düğüm 1‘den ulaştık. Yani seviye[4] = seviye[3] = seviye[2] = seviye[1] + 1 = **1 **. Şimdi onları ziyaret edildi olarak işaretliyoruz ve kuyruğa itiyoruz.

                   front
+-----+  +-----+  +-----+
|  2  |  |  3  |  |  4  |
+-----+  +-----+  +-----+

Şimdi düğüm 4‘ü açıyoruz ve onunla çalışıyoruz. 4. düğümden **7. düğüme gidebiliriz. seviye[7] = seviye[4] + 1 = 2. 7. düğümü ziyaret edildi olarak işaretliyoruz ve onu kuyruğa itiyoruz.

                   front
+-----+  +-----+  +-----+
|  7  |  |  2  |  |  3  |
+-----+  +-----+  +-----+

3. düğümden, 7. düğüme ve 8. düğüme gidebiliriz. 7. düğümü ziyaret edilmiş olarak işaretlediğimiz için, 8. düğümü ziyaret edilmiş olarak işaretliyoruz, seviye[8] = seviye[3] + ** değiştiriyoruz 1** = 2. Kuyrukta düğüm 8‘i itiyoruz.

                   front
+-----+  +-----+  +-----+
|  6  |  |  7  |  |  2  |
+-----+  +-----+  +-----+

Bu işlem hedefimize ulaşana veya kuyruk boşalana kadar devam edecek. level dizisi bize kaynak’tan en kısa yolun mesafesini sağlayacaktır. level dizisini infinity değeriyle başlatabiliriz, bu da düğümlerin henüz ziyaret edilmediğini gösterir. Sözde kodumuz şöyle olacaktır:

Procedure BFS(Graph, source):
Q = queue();
level[] = infinity
level[source] := 0
Q.push(source)
while Q is not empty
    u -> Q.pop()
    for all edges from u to v in Adjacency list
        if level[v] == infinity
            level[v] := level[u] + 1
            Q.push(v)
        end if
    end for
end while
Return level

level dizisini yineleyerek, her bir düğümün kaynağa olan mesafesini bulabiliriz. Örneğin: 10. düğümün kaynağa olan uzaklığı düzey[10] içinde saklanacaktır.

Bazen sadece en kısa mesafeyi değil, aynı zamanda kaynak’tan hedef düğümümüze gidebileceğimiz yolu da yazdırmamız gerekebilir. Bunun için bir ebeveyn dizisi tutmamız gerekiyor. ana[kaynak] NULL olacaktır. level dizisindeki her güncelleme için, for döngüsü içindeki sözde kodumuza basitçe ‘parent[v] := u’ ekleyeceğiz. BFS’yi bitirdikten sonra yolu bulmak için, NULL değeriyle gösterilecek olan kaynağa ulaşana kadar ana diziyi geriye saracağız. Sözde kod şöyle olacaktır:

Procedure PrintPath(u):  //recursive    |   Procedure PrintPath(u):   //iterative
if parent[u] is not equal to null       |   S =  Stack()
    PrintPath(parent[u])                |   while parent[u] is not equal to null    
end if                                  |       S.push(u)
print -> u                              |       u := parent[u]
                                        |   end while
                                        |   while S is not empty
                                        |       print -> S.pop
                                        |   end while

Karmaşıklık:

Her düğümü bir kez ve her kenarı bir kez ziyaret ettik. Dolayısıyla karmaşıklık O(V + E) olacaktır; burada V düğüm sayısı ve E kenar sayısıdır.

2B grafikte Kaynaktan En Kısa Yolu Bulma #

Çoğu zaman, tek bir kaynaktan diğer tüm düğümlere veya 2B grafikteki belirli bir düğüme giden en kısa yolu bulmamız gerekecek. Örneğin, bir şövalyenin satranç tahtasında belirli bir kareye ulaşması için kaç hamle gerektiğini bulmak istiyoruz veya bazı hücrelerin engellendiği bir dizimiz var, bir hücreden diğerine en kısa yolu bulmalıyız. . Sadece yatay ve dikey olarak hareket edebiliriz. Çapraz hareketler bile mümkün olabilir. Bu durumlar için, düğümlerdeki kareleri veya hücreleri dönüştürebilir ve bu sorunları BFS kullanarak kolayca çözebiliriz. Artık ziyaret edilen, ana ve seviyemiz 2B diziler olacaktır. Her düğüm için olası tüm hareketleri ele alacağız. Belirli bir düğüme olan mesafeyi bulmak için hedefimize ulaşıp ulaşmadığımızı da kontrol edeceğiz.

Yön dizisi adı verilen ek bir şey olacak. Bu, gidebileceğimiz tüm olası yön kombinasyonlarını saklayacaktır. Diyelim ki yatay ve dikey hareketler için yön dizilerimiz şöyle olacak:

+----+-----+-----+-----+-----+
| dx |  1  |  -1 |  0  |  0  |
+----+-----+-----+-----+-----+
| dy |  0  |   0 |  1  |  -1 |
+----+-----+-----+-----+-----+

Burada dx x eksenindeki hareketi ve dy y eksenindeki hareketi temsil eder. Yine bu kısım isteğe bağlıdır. Ayrıca olası tüm kombinasyonları ayrı ayrı yazabilirsiniz. Ancak yön dizisini kullanarak işlemek daha kolaydır. Çapraz hareketler veya şövalye hareketleri için daha fazla ve hatta farklı kombinasyonlar olabilir.

Aklımızda tutmamız gereken ek kısım:

  • Hücrelerden herhangi biri bloke olursa, her olası hamle için hücrenin bloke olup olmadığını kontrol edeceğiz.
  • Ayrıca sınırların dışına çıkıp çıkmadığımızı, yani dizi sınırlarını geçip geçmediğimizi kontrol edeceğiz.
  • Satır ve sütun sayısı verilecektir.

Sözde kodumuz şöyle olacaktır:

Procedure BFS2D(Graph, blocksign, row, column):
for i from 1 to row
    for j from 1 to column
        visited[i][j] := false
    end for
end for
visited[source.x][source.y] := true
level[source.x][source.y] := 0
Q = queue()
Q.push(source)
m := dx.size
while Q is not empty
    top := Q.pop
    for i from 1 to m
        temp.x := top.x + dx[i]
        temp.y := top.y + dy[i]
        if temp is inside the row and column and top doesn't equal to blocksign
            visited[temp.x][temp.y] := true
            level[temp.x][temp.y] := level[top.x][top.y] + 1
            Q.push(temp)
        end if
    end for
end while
Return level

Daha önce tartıştığımız gibi, BFS yalnızca ağırlıksız grafikler için çalışır. Ağırlıklı grafikler için [Dijkstra](https://www.wikiod.com/tr/algorithm/dijkstra-algoritmas#Dijkstra’nın En Kısa Yol Algoritması)’nin algoritmasına ihtiyacımız olacak. Negatif kenar döngüleri için Bellman-Ford’nin algoritmasına ihtiyacımız var. Yine bu algoritma tek kaynaklı en kısa yol algoritmasıdır. Her bir düğümden diğer tüm düğümlere olan mesafeyi bulmamız gerekirse, Floyd-Warshall’nin algoritmasına ihtiyacımız olacak.

BFS Kullanan Yönsüz Grafiğin Bağlı Bileşenleri. #

BFS, bir yönsüz grafiğin bağlı bileşenlerini bulmak için kullanılabilir. Ayrıca verilen grafiğin bağlantılı olup olmadığını da bulabiliriz. Sonraki tartışmamız, yönlendirilmemiş grafiklerle uğraştığımızı varsayar. Bağlı bir grafiğin tanımı:

Her çift arasında bir yol varsa bir grafik bağlanır. köşeler.

Aşağıda bağlı bir grafik bulunmaktadır.

Aşağıdaki grafik bağlı değil ve 2 bağlı bileşene sahiptir:

  1. Bağlı Bileşen 1: {a,b,c,d,e}
  2. Bağlı Bileşen 2: {f}

BFS, bir grafik geçiş algoritmasıdır. Rastgele bir kaynak düğümden başlayarak, algoritmanın sonlandırılmasında tüm düğümler ziyaret edilirse, grafik bağlanır, aksi takdirde bağlı değildir.

Algoritma için PseudoCode.

boolean isConnected(Graph g)
{     
 BFS(v)//v is a random source node.
 if(allVisited(g))
 {
  return true;
 }
 else return false;
}

Yönlendirilmemiş bir grafiğin bağlı olup olmadığını bulmak için C uygulaması:

#include<stdio.h>
#include<stdlib.h>
#define MAXVERTICES 100    

void enqueue(int);
int deque();
int isConnected(char **graph,int noOfVertices);
void BFS(char **graph,int vertex,int noOfVertices);    
int count = 0;
//Queue node depicts a single Queue element
//It is NOT a graph node.
struct node
{
    int v;
    struct node *next;
};

typedef struct node Node;
typedef struct node *Nodeptr;

Nodeptr Qfront = NULL;
Nodeptr Qrear = NULL;
char *visited;//array that keeps track of visited vertices.

int main()
{
    int n,e;//n is number of vertices, e is number of edges.
    int i,j;
    char **graph;//adjacency matrix

    printf("Enter number of vertices:");
    scanf("%d",&n);

    if(n < 0 || n > MAXVERTICES)
    {
     fprintf(stderr, "Please enter a valid positive integer from 1 to %d",MAXVERTICES);
     return -1;
    }

    graph = malloc(n * sizeof(char *));
    visited = malloc(n*sizeof(char));

    for(i = 0;i < n;++i)
    {
        graph[i] = malloc(n*sizeof(int));
        visited[i] = 'N';//initially all vertices are not visited.
        for(j = 0;j < n;++j)
            graph[i][j] = 0;
    }

    printf("enter number of edges and then enter them in pairs:");
    scanf("%d",&e);

    for(i = 0;i < e;++i)
    {
        int u,v;
        scanf("%d%d",&u,&v);
        graph[u-1][v-1] = 1;
        graph[v-1][u-1] = 1;
    }    
    
    if(isConnected(graph,n))
        printf("The graph is connected");
    else printf("The graph is NOT connected\n");       
}

void enqueue(int vertex)
{
    if(Qfront == NULL)
    {
        Qfront = malloc(sizeof(Node));
        Qfront->v = vertex;
        Qfront->next = NULL;
        Qrear = Qfront;
    }
    else
    {
        Nodeptr newNode = malloc(sizeof(Node));
        newNode->v = vertex;
        newNode->next = NULL;
        Qrear->next = newNode;
        Qrear = newNode;
    }
}

int deque()
{
    if(Qfront == NULL)
    {
        printf("Q is empty , returning -1\n");
        return -1;
    }
    else
    {
        int v = Qfront->v;
        Nodeptr temp= Qfront;
        if(Qfront == Qrear)
        {
            Qfront = Qfront->next;
            Qrear = NULL;
        }
        else
            Qfront = Qfront->next;

        free(temp);
        return v;
    }
}

int isConnected(char **graph,int noOfVertices)
{
    int i;

    //let random source vertex be vertex 0;
    BFS(graph,0,noOfVertices);

    for(i = 0;i < noOfVertices;++i)
        if(visited[i] == 'N')
         return 0;//0 implies false;

    return 1;//1 implies true;
}

void BFS(char **graph,int v,int noOfVertices)
{        
    int i,vertex;
    visited[v] = 'Y';
    enqueue(v);    
    while((vertex = deque()) != -1)
    {            
        for(i = 0;i < noOfVertices;++i)
            if(graph[vertex][i] == 1 && visited[i] == 'N')
            {
                enqueue(i);
                visited[i] = 'Y';
            }
    }
}
