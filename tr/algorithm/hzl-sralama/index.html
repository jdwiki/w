Bazen Quicksort, Partition-Exchange sıralama olarak da bilinir.
Yardımcı Alan: O(n)
Zaman karmaşıklığı: en kötü O(n²), en iyiO(nlogn)

Hızlı Sıralama Temelleri #

Quicksort, bir öğe (“pivot”) seçen ve tüm öğeler pivot ondan önce gelir ve daha büyük tüm elementler ondan sonra gelir. Algoritma daha sonra liste sıralanıncaya kadar bölümlere özyinelemeli olarak uygulanır.

1. Lomuto bölme düzeni mekanizması :

Bu şema, tipik olarak dizideki son eleman olan bir pivot seçer. Algoritma, pivotu i değişkenine yerleştirmek için indeksi korur ve pivottan küçük veya ona eşit bir eleman bulduğu her seferinde, bu indeks artırılır ve bu eleman pivotun önüne yerleştirilir.

partition(A, low, high) is
pivot := A[high]
i := low
for j := low to high – 1 do
    if A[j] ≤ pivot then
        swap A[i] with A[j]
        i := i + 1
swap A[i] with A[high]
return i

Hızlı Sıralama mekanizması:

quicksort(A, low, high) is
if low < high then
    p := partition(A, low, high)
    quicksort(A, low, p – 1)
    quicksort(A, p + 1, high)

Hızlı sıralama örneği:

2. Hoare bölme şeması:

Bölünmekte olan dizinin uçlarından başlayan ve daha sonra bir ters çevirme algılayana kadar birbirlerine doğru hareket eden iki dizin kullanır: yanlış durumda olan biri pivottan büyük veya ona eşit, biri daha küçük veya eşit olan bir çift öğe birbirine göre sıralayın. Ters çevrilmiş elemanlar daha sonra değiştirilir. Endeksler buluştuğunda, algoritma durur ve son indeksi döndürür. Hoare’nin şeması, Lomuto’nun bölme şemasından daha verimlidir, çünkü ortalama olarak üç kat daha az takas yapar ve tüm değerler eşit olduğunda bile verimli bölümler oluşturur.

quicksort(A, lo, hi) is
if lo < hi then
    p := partition(A, lo, hi)
    quicksort(A, lo, p)
    quicksort(A, p + 1, hi)

bölüm :

partition(A, lo, hi) is
pivot := A[lo]
i := lo - 1
j := hi + 1
loop forever
    do:
        i := i + 1
    while A[i] < pivot do
    
    do:
        j := j - 1
    while A[j] > pivot do
    
    if i >= j then
        return j
    
    swap A[i] with A[j]

Python’da Hızlı Sıralama #

def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) / 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)

print quicksort([3,6,8,10,1,2,1])

“[1, 1, 2, 3, 6, 8, 10]” yazdırır #

Lomuto bölümü java uygulaması #

public class Solution {

public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    int n = sc.nextInt();
    int[] ar = new int[n];
    for(int i=0; i<n; i++)
      ar[i] = sc.nextInt();
     quickSort(ar, 0, ar.length-1);   
}

public static void quickSort(int[] ar, int low, int high)
 {
    if(low<high)
    {
        int p = partition(ar, low, high);
        quickSort(ar, 0 , p-1);
        quickSort(ar, p+1, high);
    }    
 }   
public static int partition(int[] ar, int l, int r)
 {
    int pivot = ar[r];
    int i =l;
    for(int j=l; j<r; j++)
     {
        if(ar[j] <= pivot)
         {
            int t = ar[j];
            ar[j] = ar[i];
            ar[i] = t;
            i++;
        }   
    } 
    int t = ar[i];
    ar[i] = ar[r];
    ar[r] = t;
 
    return i;
}

Haskell Uygulaması #

quickSort :: Ord a => [a] -> [a]
quickSort [] = []
quickSort (x:xs) = quickSort [ y | y <- xs, y <= x ] 
                   ++ [x] 
                   ++ quickSort [ z | z <- xs, z > x ]

C# Uygulaması #

public class QuickSort
{
    private static int Partition(int[] input, int low, int high)
    {
        var pivot = input[high];
        var i = low - 1;
        for (var j = low; j <= high - 1; j++)
        {
            if (input[j] <= pivot)
            {
                i++;
                var temp = input[i];
                input[i] = input[j];
                input[j] = temp;
            }
        }
        var tmp = input[i + 1];
        input[i + 1] = input[high];
        input[high] = tmp;
        return (i + 1);
    }

    private static void SortQuick(int[] input, int low, int high)
    {
        while (true)
        {
            if (low < high)
            {
                var pi = Partition(input, low, high);
                SortQuick(input, low, pi - 1);
                low = pi + 1;
                continue;
            }
            break;
        }
    }

    public static int[] Main(int[] input)
    {
        SortQuick(input, 0, input.Length - 1);
        return input;
    }
}
