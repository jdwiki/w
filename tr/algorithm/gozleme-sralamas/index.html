Krep Sıralaması Temel Bilgiler #

Gözleme Sıralaması, bir spatula yığının herhangi bir noktasına yerleştirilebildiği ve üzerindeki tüm krepleri çevirmek için kullanılabildiği zaman, düzensiz bir krep yığınını boyut sırasına göre sıralamaya ilişkin matematiksel problem için konuşma diline özgü bir terimdir. Krep sayısı, belirli sayıda krep için gereken minimum çevirme sayısıdır.

Mümkün olan en az karşılaştırma ile sıralamaya çalışan geleneksel bir sıralama algoritmasının aksine, amaç, diziyi mümkün olduğunca az tersine çevirerek sıralamaktır.

Fikir, Seçim Sıralamasına benzer bir şey yapmaktır. Maksimum elemanı birer birer en sona yerleştiriyoruz ve mevcut dizinin boyutunu birer birer küçültüyoruz.

Sorunu incelemek:

  1. Krepleri en küçüğünden (üstten) büyüğe (alttan) sipariş etmeniz gerekiyor, başlangıç ​​yığını herhangi bir sırayla düzenlenebilir.
  2. Sadece tüm yığını çevirme işlemini yapabilirim.
  3. Belirli bir gözlemeyi yığının altına çevirmek için, önce onu yukarı çevirmemiz gerekir (sonra tekrar aşağıya çevirmemiz gerekir).
  4. Her bir gözleme siparişi vermek için bir yukarı çevirme ve son konumuna bir aşağı çevirme gerekir.

Sezgisel Algoritma:

  1. En büyük bozuk gözlemeyi bulun ve aşağıya çevirin (önce onu yığının üstüne çevirmeniz gerekebilir).

  2. Yığın sipariş edilene kadar birinci adımı tekrarlayın.

  3. İşte bu, iki adımlı bir algoritma çalışacak.

Krep sıralama algoritması örneği:

Yardımcı Alan: O(1)
Zaman Karmaşıklığı: O(n^2)

C# Uygulaması #

public class PancakeSort
{
    private static void SortPancake(int[] input, int n)
    {
        for (var bottom = n - 1; bottom > 0; bottom--)
        {
            var index = bottom;
            var maxIndex = input[bottom];
            int i;
            for (i = bottom - 1; i >= 0; i--)
            {
                if (input[i] > maxIndex)
                {
                    maxIndex = input[i];
                    index = i;
                }
            }
            if (index == bottom) continue;
            var temp = new int[n];
            var j = 0;
            for (i = bottom; i > index; i--,j++)
            {
                temp[j] = input[i];
            }
            for (i = 0; i < index; i++, j++)
            {
                temp[j] = input[i];
            }
            if (temp.Length > j) temp[j] = input[index];
            for (i = 0; i <= bottom; i++)
            {
                input[i] = temp[i];
            }
        }
    }

    public static int[] Main(int[] input)
    {
        SortPancake(input, input.Length);
        return input;
    }
}
