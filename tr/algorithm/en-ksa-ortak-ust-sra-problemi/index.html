En Kısa Genel Üst Sıra Problemi Temel Bilgiler #

En Kısa Ortak Süper Dizi, bu görev için harici bir işlev olarak kullanabileceğiniz en uzun ortak alt diziyle yakından ilgili bir sorundur. En kısa ortak üst dizi sorunu, en uzun ortak alt dizi sorunuyla yakından ilişkili bir sorundur.

En kısa ortak üst dizi (scs), minimum uzunlukta ortak bir üst dizidir. En kısa ortak üst dizi probleminde, iki dizi ‘X’ ve ‘Y’ verilir ve görev bu dizilerin mümkün olan en kısa ortak üst dizisini bulmaktır. Genel olarak, bir scs benzersiz değildir.

“X = <x1,…,xm>” ve “Y = <y1,…,yn>” olmak üzere iki dizi verildiğinde, “U = <u1,…,uk>” dizisi ortak bir süper “U”, hem “X” hem de “Y"nin bir üst dizisiyse “X” ve “Y” dizisi. Başka bir deyişle, “x” ve “y” dizelerinin en kısa ortak üst dizisi, en kısa “z” dizesidir, öyle ki hem “x” hem de “y”, “z"nin alt dizileridir.

İki giriş dizisi için, en uzun ortak alt diziden (lcs) bir scs kolayca oluşturulabilir. Örneğin, “X[1..m]=abcbdab” ve “Y[1..n]=bdcaba” ise, lcs “Z[1..r]=bcba” olur. Sembol sırasını korurken lcs olmayan sembolleri ekleyerek scs’yi elde ederiz: U[1..t]=abdcabdab.

İki giriş dizisi için r+t=m+n olduğu oldukça açıktır. Ancak, üç veya daha fazla giriş dizisi için bu geçerli değildir. Ayrıca, lcs ve scs problemlerinin ikili problemler olmadığına dikkat edin.

Bir dizi “S1,S2,…,Sl” dizilerinin bir üst dizisi olan “S” dizisini bulmanın daha genel sorunu için, sorun NP-Complete’dir. Ayrıca, ortalama durum için iyi yaklaşımlar bulunabilir, ancak en kötü durum için bulunmaz.

En Kısa Ortak Üst Sıra Problemi Örneği:

Zaman Karmaşıklığı: O(max(m,n))

En Kısa Ortak Üst Sıra Probleminin C# ile Uygulanması #

public class ShortestCommonSupersequence
{
    private static int Max(int a, int b)
    {
        return a > b ? a : b;
    }
    
    private static int Lcs(string x, string y, int m, int n)
    {
        var l = new int[m + 1, n + 1];
        for (var i = 0; i <= m; i++)
        {
            for (var j = 0; j <= n; j++)
            {
                if (i == 0 || j == 0) l[i, j] = 0;
                else if (x[i - 1] == y[j - 1]) l[i, j] = l[i - 1, j - 1] + 1;
                else l[i, j] = Max(l[i - 1, j], l[i, j - 1]);
            }
        }
        return l[m, n];
    }

    private static int Scs(string x, string y)
    {
        int m = x.Length, n = y.Length;
        int l = Lcs(x, y, m, n);
        return m + n - l;
    }

    public static int Main(string x, string y)
    {
        return Scs(x, y);
    }
}
