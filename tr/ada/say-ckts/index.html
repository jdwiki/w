  • Ondalık Sabit Nokta Sayılarını Yazdır, yani Para
  • Tek Satırda Birden Çok Öğeyi Yazdırın

Ada’nın standart paketleri, tüm sayısal türlerin çıktısını sağlar. Çıktı formatı birçok şekilde ayarlanabilir.

Genel bir paketin sayısal bir türle nasıl başlatıldığına dikkat edin. Ayrıca, tüm örnek için ayarlanacak varsayılanlar ve ayrıca bu parametreyle “Put” çağrılırken “Genişlik"i geçersiz kılmanın yolları vardır.

16 Tabanını Kullanarak Tam Sayıları Yazdır (Onaltılık) #

“Ada.Text_IO.Integer_IO” örneğinde bir “Default_Base” ayar değişkeni ayarlanır; ayrıca, “Default_Width” çıktının başında boşluk olmayacak şekilde ayarlanır.

with Ada.Text_IO;   use Ada.Text_IO;

procedure Print_Hex is
    subtype Count is Integer range -1_000_000 .. 1_000_000;

    package Count_IO is new Integer_IO (Count);
    X : Count;
begin
    Count_IO.Default_Width := 1;
    Count_IO.Default_Base := 16;

    X := Count'First;
    while X < Count'Last loop
        Count_IO.Put (X);
        New_Line;

        X := X + 500_000;
    end loop;
end Print_Hex;

Sonuç #

-16#F4240#
-16#7A120#
16#0#
16#7A120#

Boşluğu cömertçe kullanarak tam sayıları yazdırın #

“Tamsayı_IO” örneklerinin, her çıktı numarasının alacağı karakter sayısı olan “Varsayılan_Genişlik” ayar değişkeni vardır.

with Ada.Text_IO;   use Ada.Text_IO;

procedure Print_Integer is
    subtype Count is Integer range -1_000_000 .. 1_000_000;

    package Count_IO is new Integer_IO (Count);
    X : Count;
begin
    Count_IO.Default_Width := 12;

    X := Count'First;
    while X < Count'Last loop
        Count_IO.Put (X);
        Count_IO.Put (X + 1);
        New_Line;

        X := X + 500_000;
    end loop;
end Print_Integer;

Sonuç #

    -1000000
     -500000
           0
      500000

Ondalık Sabit Nokta Sayılarını Yazdır, yani Para #

Ada.Text_IO.Editing, “resim dizileri” kullanarak ondalık sabit nokta değerlerinin biçimlendirilmesini sağlar. Bunlar, ayırıcılar, para birimi işaretleri vb. için “sihirli” karakterler kullanarak çıktıyı tanımlar.

with Ada.Text_IO.Editing;   use Ada.Text_IO;

procedure Print_Value is

    Max_Count      : constant := 1_000_000;

    type Fruit is (Banana, Orange, Pear);
    subtype Count is Integer range -Max_Count .. +Max_Count;

    type Money is delta 0.001 digits 10;

    package Fruit_IO is new Enumeration_IO (Fruit);
    package Money_IO is new Editing.Decimal_Output
      (Money,
       Default_Currency => "CHF",
       Default_Separator => ''');

    Inventory : constant array (Fruit) of Count :=
      (Banana => +27_420,
       Orange => +140_600,
       Pear   => -10_000);

    Price_List : constant array (Fruit) of Money :=
      (Banana => 0.07,
       Orange => 0.085,
       Pear   => 0.21);

    Format : constant Editing.Picture :=
      Editing.To_Picture ("<###BZ_ZZZ_ZZ9.99>");
begin
    Fruit_IO.Default_Width := 12;

    for F in Inventory'Range loop
        Fruit_IO.Put (F);
        Put          (" | ");
        Money_IO.Put (Item => Inventory (F) * Price_List (F),
                      Pic => Format);
        New_Line;
    end loop;
end Print_Value;

Sonuç #

BANANA       |  CHF     1'919.40 
ORANGE       |  CHF    11'951.00 
PEAR         | (CHF     2'100.00)

Tek Satırda Birden Çok Öğeyi Yazdırın #

‘_IO’ paketlerinin örneklerini birleştirin, sayısal türüyle doğru olanı kullanın.

with Ada.Text_IO;   use Ada.Text_IO;

procedure Print_Inventory is
    type Fruit is (Banana, Orange, Pear);
    subtype Count is Integer range -1_000_000 .. 1_000_000;

    package Fruit_IO is new Enumeration_IO (Fruit);
    package Count_IO is new Integer_IO (Count);

    Inventory : constant array (Fruit) of Count :=
      (Banana => 27_420,
       Orange => 140_600,
       Pear   => -10_000);

begin
    Fruit_IO.Default_Width := 12;

    for F in Inventory'Range loop
        Fruit_IO.Put (F);
        Put          (" | ");
        Count_IO.Put (Inventory (F));
        New_Line;
    end loop;
end Print_Inventory;

Sonuç #

BANANA       |    27420
ORANGE       |   140600
PEAR         |   -10000
