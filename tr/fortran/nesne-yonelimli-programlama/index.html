Türetilmiş tür tanımı #

Fortran 2003, nesne yönelimli programlama desteğini tanıttı. Bu özellik, modern programlama tekniklerinden yararlanmayı sağlar. Türetilmiş türler aşağıdaki formla tanımlanır:

TYPE [[, attr-list] :: ] name [(name-list)]
   [def-stmts]
   [PRIVATE statement or SEQUENCE statement]. . .
   [component-definition]. . .
   [procedure-part]
END TYPE [name]

nerede,

  • attr-list - nitelik belirteçlerinin listesi
  • ad - türetilmiş veri türünün adı
  • name-list - virgülle ayrılmış tür parametre adlarının listesi
  • def-stmts - ad listesinde belirtilen tür parametrelerinin bir veya daha fazla INTEGER bildirimi
  • bileşen-tanımı - türetilmiş türün bileşenini tanımlayan bir veya daha fazla tür bildirim ifadesi veya prosedür işaretçi ifadesi
  • prosedür bölümü - isteğe bağlı olarak bir PRIVATE ifadesi ve bir veya daha fazla prosedür bağlama ifadesi tarafından takip edilen bir CONTAINS ifadesi

Örnek:

type shape
    integer :: color
end type shape

Soyut türetilmiş türler #

Genişletilebilir türetilmiş bir tür abstract olabilir

type, abstract :: base_type
end type

Böyle bir türetilmiş tür hiçbir zaman somutlaştırılamaz, örneğin

type(base_type) t1
allocate(type(base_type) :: t2)

ancak polimorfik bir nesne, beyan edilen türü olarak buna sahip olabilir.

class(base_type), allocatable :: t1

veya

function f(t1)
  class(base_type) t1
end function

Soyut türlerin bileşenleri ve türe bağlı prosedürleri olabilir

type, abstract :: base_type
  integer i
contains
  procedure func
  procedure(func_iface), deferred :: def_func
end type

‘def_func’ prosedürü, ‘func_iface’ arabirimi ile ertelenmiş türe bağlı bir prosedürdür. Böyle bir ertelenmiş tip-sınırlı prosedür, her bir genişletme tipi tarafından uygulanmalıdır.

Tip Prosedürleri #

Sınıf benzeri davranış elde etmek için tip ve ilgili prosedürler (alt program ve fonksiyonlar) bir modüle yerleştirilmelidir:

Örnek:

module MShape
    implicit none
    private

    type, public :: Shape
    private
        integer :: radius
    contains
        procedure :: set   => shape_set_radius
        procedure :: print => shape_print
    end type Shape

contains
    subroutine shape_set_radius(this, value)
        class(Shape), intent(in out) :: self
        integer, intent(in)          :: value

        self%radius = value
    end subroutine shape_set_radius

    subroutine shape_print(this)
        class(Shape), intent(in) :: self

        print *, 'Shape: r = ', self%radius
    end subroutine shape_print
end module MShape

Daha sonra bir kodda bu Shape sınıfını aşağıdaki gibi kullanabiliriz:

! declare a variable of type Shape
type(Shape) :: shape

! call the type-bound subroutine
call shape%set(10)
call shape%print

Tür uzantısı #

Türetilmiş bir tür, ne “bağlama” özniteliğine ne de “sıra” özniteliğine sahip değilse _genişletilebilir_dir. Böyle bir tür başka bir tür tarafından genişletilebilir.

module mod

  type base_type
    integer i
  end type base_type

  type, extends(base_type) :: higher_type
    integer j
  end type higher_type

end module mod

‘base_type’ türü bildirilen bir polimorfik değişken, ‘higher_type’ türüyle uyumludur ve dinamik tür olarak buna sahip olabilir

class(base_type), allocatable :: obj
allocate(obj, source=higher_type(1,2))

Tür uyumluluğu, bir alt öğe zinciri boyunca iner, ancak bir tür yalnızca bir başka türü genişletebilir.

Genişleyen türetilmiş bir tür, üst öğeden türe bağlı prosedürleri devralır, ancak bu geçersiz kılınabilir

module mod

  type base_type
  contains
    procedure :: sub => sub_base
  end type base_type

  type, extends(base_type) :: higher_type
  contains
    procedure :: sub => sub_higher
  end type higher_type

contains

  subroutine sub_base(this)
    class(base_type) this
  end subroutine sub_base

  subroutine sub_higher(this)
    class(higher_type) this
  end subroutine sub_higher

end module mod

program prog
  use mod

  class(base_type), allocatable :: obj

  obj = base_type()
  call obj%sub

  obj = higher_type()
  call obj%sub

end program

Yapıcı yazın #

Tür adını aşırı yüklemek için bir arabirim kullanılarak türetilmiş türler için özel oluşturucular yapılabilir. Bu şekilde, bileşenlere karşılık gelmeyen anahtar kelime argümanları, bu türde bir nesne oluşturulurken kullanılabilir.

module ball_mod
  implicit none

  ! only export the derived type, and not any of the
  ! constructors themselves
  private
  public :: ball

  type :: ball_t
     real :: mass
  end type ball_t

  ! Writing an interface overloading 'ball_t' allows us to
  ! overload the type constructor
  interface ball_t
     procedure :: new_ball
  end interface ball_t

contains

  type(ball_t) function new_ball(heavy)
    logical, intent(in) :: heavy

    if (heavy) then
       new_ball%mass = 100
    else
       new_ball%mass = 1
    end if
    
  end function new_ball
  
end module ball_mod

program test
  use ball_mod
  implicit none

  type(ball_t) :: football
  type(ball_t) :: boulder
  
  ! sets football%mass to 4.5
  football = ball_t(4.5)
  ! calls 'ball_mod::new_ball'
  boulder = ball_t(heavy=.true.)
end program test

Bu, ayrı başlatma rutinleri kullanmaktan daha düzenli bir API yapmak için kullanılabilir:

subroutine make_heavy_ball(ball)
  type(ball_t), intent(inout) :: ball
  ball%mass = 100
end subroutine make_heavy_ball

...

call make_heavy_ball(boulder)
