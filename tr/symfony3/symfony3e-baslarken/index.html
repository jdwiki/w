3. Windows Sistemleri #

Yol ortam değişkeninize php eklemelisiniz. Tez adımlarını izleyin:

Windows 7 :

  • Bilgisayarım simgesine sağ tıklayın
  • Özellikler’i tıklayın
  • Sol gezinme panelinden Gelişmiş sistem ayarları’na tıklayın
  • Gelişmiş sekmesine tıklayın
  • Ortam Değişkenleri düğmesini tıklayın
  • Sistem Değişkenleri bölümünde Yol (büyük/küçük harf duyarsız) öğesini seçin ve Düzenle düğmesini tıklayın.
  • Dizenin sonuna noktalı virgül (;) ekleyin, ardından PHP kurulumunuzun tam dosya sistemi yolunu ekleyin (örn. C:\Program Files\PHP)
  • Tüm iletişim kutuları kaybolana kadar Tamam’ı tıklamaya devam edin
  • Komut isteminizi kapatın ve tekrar açın
  • Sıralanmış

Windows 8 ve 10

  • Ara’da şunu arayın ve seçin: Sistem (Denetim Masası)
  • Gelişmiş sistem ayarları bağlantısını tıklayın.
  • Ortam Değişkenleri’ne tıklayın.
  • Sistem Değişkenleri bölümünde PATH ortam değişkenini bulun ve seçin. Düzenle’yi tıklayın. PATH ortam değişkeni yoksa Yeni’ye tıklayın.
  • PHP kurulumunuzun tam dosya sistemi yolunu ekleyin (ör. C:\Program Files\PHP)

Bundan sonra, komut konsolunuzu açın ve aşağıdaki komutu yürütün:

c:\> php -r "readfile('https://symfony.com/installer');" > symfony

Ardından, indirilen symfony dosyasını projenizin dizinine taşıyın ve aşağıdaki gibi çalıştırın:

c:\> move symfony c:\projects
c:\projects\> php symfony

1. Symfony Installer’ı Yükleme #

Yükleyici için PHP 5.4 veya üstü gerekir. Hala eski PHP 5.3 sürümünü kullanıyorsanız, Symfony Installer’ı kullanamazsınız. Nasıl devam edeceğinizi öğrenmek için Yükleyici olmadan Symfony Uygulamaları Oluşturma bölümünü okuyun. - kaynak: http://symfony.com/doc/current/book/installation.html

5. Projenizi Belirli Bir Symfony Sürümüne Dayandırmak #

Projenizin belirli bir Symfony sürümüne dayalı olması gerekiyorsa, yeni komutun isteğe bağlı ikinci argümanını kullanın:

# use the most recent version in any Symfony branch
$ symfony new my_project_name 2.8
$ symfony new my_project_name 3.1

# use a specific Symfony version
$ symfony new my_project_name 2.8.1
$ symfony new my_project_name 3.0.2

# use a beta or RC version (useful for testing new Symfony versions)
$ symfony new my_project 3.0.0-BETA1
$ symfony new my_project 3.1.0-RC1

Yükleyici ayrıca mevcut en yeni Symfony LTS sürümünü yükleyen lts adlı özel bir sürümü de destekler:

$ symfony new my_project_name lts

Neden birkaç Symfony sürümü olduğunu ve projeleriniz için hangisini kullanacağınızı daha iyi anlamak için Symfony Sürüm sürecini okuyun.

Ayrıca yükleyici olmadan da symfony uygulamaları oluşturabilirsiniz, ancak bu iyi bir fikir değildi. Yine de isterseniz, bu bağlantıdaki orijinal öğreticiyi izleyin:

Resmi Symfony Docs, Symfony’yi yükleyici olmadan yapılandırma

2. Linux ve Mac OS X Sistemleri #

Komut konsolunuzu açın ve aşağıdaki komutları yürütün:

$ sudo curl -LsS https://symfony.com/installer -o /usr/local/bin/symfony
$ sudo chmod a+x /usr/local/bin/symfony

4. Symfony Uygulamasını Oluşturma #

Symfony Installer hazır olduğunda, yeni komutla ilk Symfony uygulamanızı oluşturun:

# Linux, Mac OS X
$ symfony new my_project_name

# Windows
c:\> cd projects/
c:\projects\> php symfony new my_project_name

Bu komut, mutlaka htdocs klasöründen değil, herhangi bir yerden çalıştırılabilir.

Bu komut, mevcut en son kararlı Symfony sürümüne dayalı yeni bir proje içeren my_project_name/ adlı yeni bir dizin oluşturur. Ayrıca yükleyici, sisteminizin Symfony uygulamalarını yürütmek için teknik gereksinimleri karşılayıp karşılamadığını kontrol eder. Değilse, bu gereksinimleri karşılamak için gereken değişikliklerin listesini görürsünüz.

Sayfa oluşturma #

Devam etmeden önce, Kurulum bölümünü okuduğunuzdan ve tarayıcıda yeni Symfony uygulamanıza erişebildiğinizden emin olun.

Şanslı (iyi, rastgele) bir sayı üreten ve yazdıran bir sayfa - /lucky/sayı - oluşturmak istediğinizi varsayalım. Bunu yapmak için, birisi /lucky/numaraya gittiğinde yürütülecek bir “Kontrolör sınıfı” ve bunun içinde bir “denetleyici” yöntemi oluşturun.

// src/AppBundle/Controller/LuckyController.php
namespace AppBundle\Controller;

use Sensio\Bundle\FrameworkExtraBundle\Configuration\Route;
use Symfony\Component\HttpFoundation\Response;

class LuckyController
{
    /**
     * @Route("/lucky/number")
     */
    public function numberAction()
    {
        $number = rand(0, 100);

        return new Response(
            '<html><body>Lucky number: '.$number.'</body></html>'
        );
    }
}
