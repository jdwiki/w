Bu sayfada

  • Ek açıklamaları kullanarak Symfony doğrulaması
  • YAML kullanarak Symfony doğrulaması

Aslında, form doğrulaması, “Validator Component” adlı bir bileşene dayanır.

Bir şablonda bir form göstermeniz gerekmiyorsa, genellikle özel hizmeti kullanabilirsiniz. API’ler gibi. Verileri aşağıdaki gibi aynı şekilde doğrulayabilirsiniz:

Örneğin, symfony doc temelinde:

$validator = $this->get('validator');
$errors = $validator->validate($author);

if (count($errors) > 0) {
    /*
     * Uses a __toString method on the $errors variable which is a
     * ConstraintViolationList object. This gives us a nice string
     * for debugging.
     */
    $errorsString = (string) $errors;
}

Ek açıklamaları kullanarak Symfony doğrulaması #

  • ‘app/config/config.yml’ dosyasındaki ek açıklamaları kullanarak doğrulamayı etkinleştirin

    framework: validation: { enable_annotations: true }

  • AppBundle/Entity dizininde bir Varlık oluşturun. Doğrulamalar @Assert notları ile yapılır.

    id; } /** * Set name * * @param string $name * * @return Car */ public function setName($name) { $this->name = $name; return $this; } /** * Get name * * @return string */ public function getName() { return $this->name; } /** * Set number * * @param integer $number * * @return Car */ public function setNumber($number) { $this->number = $number; return $this; } /** * Get number * * @return integer */ public function getNumber() { return $this->number; } }

  • AppBundle/Form dizininde yeni bir form oluşturun.

    add('name', TextType::class, ['label'=>'Name']) ->add('number', IntegerType::class, ['label'=>'Number']) ; } /** * @param OptionsResolver $resolver */ public function configureOptions(OptionsResolver $resolver) { $resolver->setDefaults(array( 'data_class' => 'AppBundle\Entity\Car' )); } public function setDefaultOptions(OptionsResolverInterface $resolver) { // TODO: Implement setDefaultOptions() method. } public function getName() { return 'car_form'; } }

  • AppBundle/Controller/DefaultController.php içinde yeni bir rota ve yeni bir eylem yöntemi oluşturun. Rota ayrıca açıklamalarla bildirilecektir, bu nedenle bu rotayı ana rota dosyasına (app/config/routing.yml) aktardığınızdan emin olun.

    createForm( CarType::class, $car, [ 'action' => $this->generateUrl('app_car'), 'method'=>'POST', 'attr'=>[ 'id'=>'form_car', 'class'=>'car_form' ] ] ); $form->handleRequest($request); return $this->render( 'AppBundle:Default:car.html.twig',[ 'form'=>$form->createView() ] ); } }

  • Görünümü AppBundle/Resources/views/Default/car.html.twig içinde oluşturun.

    {% extends ‘::base.html.twig’ %}

    {% block body %} {{ form_start(form, {‘attr’: {’novalidate’:’novalidate’}}) }} {{ form_row(form.name) }} {{ form_row(form.number) }} {{ form_end(form) }} {% endblock %}

  • Symfony’nin yerleşik sunucusunu (php bin/console server:run) başlatın ve tarayıcınızda 127.0.0.1:8000/car yoluna erişin. İki giriş kutusu ve bir gönder düğmesinden oluşan bir form olmalıdır. Giriş kutularına herhangi bir veri girmeden gönder düğmesine basarsanız, hata mesajları görüntülenecektir.

YAML kullanarak Symfony doğrulaması #

  • AppBundle/Entity dizininde bir Varlık oluşturun. Bunu manuel olarak veya Symfony’nin php bin/console doktrin:generate:entity komutunu kullanarak ve her adımda gerekli bilgileri doldurarak yapabilirsiniz. ‘Yapılandırma biçimi (yml, xml, php veya açıklama)’ adımında ‘yml’ seçeneğini belirtmelisiniz.

    id; } /** * Set name * * @param string $name * * @return Person */ public function setName($name) { $this->name = $name; return $this; } /** * Get name * * @return string */ public function getName() { return $this->name; } /** * Set age * * @param integer $age * * @return Person */ public function setAge($age) { $this->age = $age; return $this; } /** * Get age * * @return int */ public function getAge() { return $this->age; } }

  • Entity sınıfı için Entity eşleme bilgilerini oluşturun. Symfony’nin php bin/console doktrin:generate:entity komutunu kullanıyorsanız, aşağıdaki kod otomatik olarak oluşturulacaktır. Aksi halde komutu kullanmazsanız aşağıdaki kodu elle oluşturabilirsiniz.

    AppBundle/Resources/config/doctrine/Person.orm.yml #

    AppBundle\Entity\Person: type: entity table: persons repositoryClass: AppBundle\Repository\PersonRepository id: id: type: integer id: true generator: strategy: AUTO fields: name: type: string length: ‘50’ age: type: integer lifecycleCallbacks: { }

  • Entity sınıfı için doğrulama oluşturun.

    AppBundle/Resources/config/validation/person.yml #

    AppBundle\Entity\Person: properties: name: - NotBlank: message: “Name is required” - Length: min: 3 max: 50 minMessage: “Please use at least 3 chars” maxMessage: “Please use max 50 chars” - Regex: pattern: “/^[A-Za-z]+$/” message: “Please use only letters” age: - NotBlank: message: “Age is required” - Length: min: 1 max: 3 minMessage: “The age must have at least 1 number in length” maxMessage: “The age must have max 3 numbers in length” - Regex: pattern: “/^[0-9]+$/” message: “Please use only numbers”

  • AppBundle/Form dizininde yeni bir form oluşturun.

    add('name', TextType::class, ['label'=>'Name']) ->add('age', IntegerType::class, ['label'=>'Age']) ; } /** * @param OptionsResolver $resolver */ public function configureOptions(OptionsResolver $resolver) { $resolver->setDefaults(array( 'data_class' => 'AppBundle\Entity\Person' )); } public function setDefaultOptions(OptionsResolverInterface $resolver) { // TODO: Implement setDefaultOptions() method. } public function getName() { return 'person_form'; } }

  • AppBundle/Resources/config/routing.yml içinde yeni bir rota oluşturun

    app_person: path: /person defaults: { _controller: AppBundle:Default:person }
