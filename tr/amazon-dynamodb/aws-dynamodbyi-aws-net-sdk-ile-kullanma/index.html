AWS DynamoDb'yi AWS .NET SDK ile Kullanma

Amazon DynamoDB, Amazon Web Services (AWS) tarafından sunulan hızlı bir NoSQL veritabanı hizmetidir. DynamoDB, AWS SDK for .NET kullanılarak .NET uygulamalarından çağrılabilir. SDK, DynamoDB ile iletişim kurmak için üç farklı model sunar. Bu konu, her modeldeki çeşitli API’leri tanıtır.

Modeller #

SDK, DynamoDB ile iletişim kurmanın üç yolunu sunar. Her biri, kontrol ve kullanım kolaylığı arasında ödünleşimler sunar. Aşağıdaki API’lerle ilgili ayrıntılar için AWS .NET SDK Reference’e bakın.

  • Düşük seviye: Amazon.DynamoDBv2 ad alanı — Bu, DynamoDB hizmet çağrıları üzerinde ince bir sarıcıdır. Tüm hizmet özellikleriyle eşleşir. Her bir işlem hakkında daha fazla bilgi edinmek için hizmet belgelerine başvurabilirsiniz.

  • Belge Modeli: Amazon.DynamoDBv2.DocumentModel ad alanı — Bu, verilerle uğraşmak için daha basit bir arabirim sağlayan bir modeldir. DynamoDB tabloları “Tablo” nesneleri ile temsil edilirken, tek tek veri satırları “Belge” nesneleri ile temsil edilir. .NET nesnelerinin DynamoDB verilerine dönüştürülmesi, temel türler için otomatiktir.

  • Nesne Kalıcılık Modeli: Amazon.DynamoDBv2.DataModel ad alanı — Bu API kümesi, .NET nesnelerini DynamoDB’de depolamanıza ve yüklemenize olanak tanır. Hedef tabloyu ve karma/aralık anahtarlarını yapılandırmak için nesneler işaretlenmelidir. “DynamoDBContext”, işaretlenmiş nesnelere etki eder. DynamoDB verilerini depolamak ve yüklemek veya bir sorgu veya tarama işleminden .NET nesnelerini almak için kullanılır. Temel veri türleri otomatik olarak DynamoDB verilerine dönüştürülür ve dönüştürücüler, isteğe bağlı türlerin DynamoDB’de saklanmasına izin verir.

Üç model, hizmetle çalışmaya farklı yaklaşımlar sağlar. Düşük seviyeli yaklaşım daha fazla istemci tarafı kodu gerektirse de (kullanıcının sayılar ve tarihler gibi .NET türlerini DynamoDB destekli dizelere dönüştürmesi gerekir) tüm hizmet özelliklerine erişim sağlar. Karşılaştırıldığında, Nesne Kalıcılık Modeli yaklaşımı, kullanıcı çoğunlukla tanıdık .NET nesneleriyle çalıştığı için hizmeti kullanmayı kolaylaştırır, ancak tüm işlevleri sağlamaz. Örneğin Object Persistence Model ile koşullu Put çağrıları yapmak mümkün değildir.

.NET SDK Geliştirici Kılavuzu içinde .NET SDK kullanarak AWS’yi çalıştırma hakkında daha fazla bilgi edinin.

Not: Bu konu, orijinal olarak AWS .NET SDK blogunda yayınlanan bir blog gönderisinden izin alınarak uyarlanmıştır.

Düşük Düzey API Örneği #

var client = new AmazonDynamoDBClient();
 
// Store item
client.PutItem(new PutItemRequest
{
    TableName = "Books",
    Item = new Dictionary<string, AttributeValue>
    {
        { "Title", new AttributeValue { S = "Cryptonomicon" } },
        { "Id", new AttributeValue { N = "42" } },
        { "Authors", new AttributeValue {
            SS = new List<string> { "Neal Stephenson" } } },
        { "Price", new AttributeValue { N = "12.95" } }
    }
});
 
// Get item
Dictionary<string, AttributeValue> book = client.GetItem(new GetItemRequest
{
    TableName = "Books",
    Key = new Dictionary<string, AttributeValue>
    {
        { "Id", new AttributeValue { N = "42" } }
    }
}).Item;
 
Console.WriteLine("Id = {0}", book["Id"].S);
Console.WriteLine("Title = {0}", book["Title"].S);
Console.WriteLine("Authors = {0}",
    string.Join(", ", book["Authors"].SS));

Belge Modeli Örneği #

var client = new AmazonDynamoDBClient();
Table booksTable = Table.LoadTable(client, "Books");
 
// Store item
Document book = new Document();
book["Title"] = "Cryptonomicon";
book["Id"] = 42;
book["Authors"] = new List<string> { "Neal Stephenson" };
book["Price"] = 12.95;
booksTable.PutItem(book);
 
// Get item
book = booksTable.GetItem(42);
Console.WriteLine("Id = {0}", book["Id"]);
Console.WriteLine("Title = {0}", book["Title"]);
Console.WriteLine("Authors = {0}",
    string.Join(", ", book["Authors"].AsListOfString()));

Nesne Kalıcılığı Modeli Örneği #

Bu örnek iki bölümden oluşmaktadır: ilk olarak ‘Kitap’ türümüzü tanımlamalıyız; ikinci olarak, onu DynamoDBContext ile kullanırız.

[DynamoDBTable("Books")]
class Book
{
    [DynamoDBHashKey]
    public int Id { get; set; }
    public string Title { get; set; }
    public List<string> Authors { get; set; }
    public double Price { get; set; }
}

Şimdi, DynamoDBContext ile kullanın.

var client = new AmazonDynamoDBClient();
DynamoDBContext context = new DynamoDBContext(client);
 
// Store item
Book book = new Book
{
    Title = "Cryptonomicon",
    Id = 42,
    Authors = new List<string> { "Neal Stephenson" },
    Price = 12.95
};
context.Save(book);
 
// Get item
book = context.Load<Book>(42);
Console.WriteLine("Id = {0}", book.Id);
Console.WriteLine("Title = {0}", book.Title);
Console.WriteLine("Authors = {0}", string.Join(", ", book.Authors));
