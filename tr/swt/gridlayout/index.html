Bir “Composite” (veya “Composite"nin bir alt sınıfı) üzerinde bir “GridLayout” örneği ayarlandığında, tüm alt denetimler bir ızgara düzeninde düzenlenir. Birden çok sütun olduğunda, ızgara soldan sağa ve yukarıdan aşağıya doldurulur.

Sütun sayısını belirlemeye ek olarak, ızgara etrafındaki kenar boşluklarını ve ızgaradaki hücreler arasındaki aralığı çeşitli üye değişkenleri aracılığıyla isteğe bağlı olarak ayarlayabilirsiniz.

Sözdizimi #

  • GridLayout()
  • GridLayout(int numColumns, boolean makeColumnsEqualWidth)

Parametreler #

parametre Ayrıntılar
numColumns Kılavuzdaki sütun sayısı
makeColumnsEqualWidth Mizanpajdaki sütunların aynı genişlikte olması gerekip gerekmediği

Bir “Composite” (veya “Composite"nin bir alt sınıfı) üzerinde bir “GridLayout” kullanırken, alt kontroller, düzen verilerini GridData benzersiz bir örneğine ayarlamalıdır. .org/kepler/index.jsp?topic=%2Forg.eclipse.platform.doc.isv%2Freference%2Fapi%2Forg%2Feclipse%2Fswt%2Flayout%2FGridData.html) alt öğenin ebeveyn içinde nasıl görüntüleneceğini belirlemek için:

// ...
final Shell shell = new Shell(display);
shell.setLayout(new GridLayout());

final Composite child = new Composite(shell, SWT.NONE);
child.setLayoutData(new GridData(SWT.FILL, SWT.FILL, true, true));
// ...

Alt denetimlerde bir “GridData” nesnesinin ayarlanamaması, çocuğun istenen şekilde konumlandırılmamasına neden olabilir. Ayrıca, çocuk bir “Kompozit” ise (veya “Bileşik"in bir alt sınıfıysa), çocuk ve çocukları hiç görünmeyebilir.

Yanlış düzen verileri kullanılırsa (örneğin, “GridData” yerine “FormData”), çalışma zamanında sonuç bir “ClassCastException” olacaktır:

“main” iş parçacığında özel durum Java.lang.ClassCastException: org.Eclipse.swt.layout.FormData, org.Eclipse.swt.layout.GridData öğesine yayınlanamaz

Tek kolon #

Bu örnekte, tek sütunlu bir düzen oluşturmak için varsayılan, no-args GridLayout() yapıcısını kullanıyoruz.

public class SingleColumnGridLayoutExample {

    private final Display display;
    private final Shell shell;

    public SingleColumnGridLayoutExample() {
        display = new Display();
        shell = new Shell(display);

        // Create the layout and apply to the Shell
        shell.setLayout(new GridLayout());

        // Add the child controls to the Shell - in this case, in a single column
        final Button buttonA = new Button(shell, SWT.PUSH);
        buttonA.setLayoutData(new GridData(SWT.FILL, SWT.FILL, true, true));
        buttonA.setText("Button A");

        final Button buttonB = new Button(shell, SWT.PUSH);
        buttonB.setLayoutData(new GridData(SWT.FILL, SWT.FILL, true, true));
        buttonB.setText("Button B");

        final Button buttonC = new Button(shell, SWT.PUSH);
        buttonC.setLayoutData(new GridData(SWT.FILL, SWT.FILL, true, true));
        buttonC.setText("Button C");

        final Button buttonD = new Button(shell, SWT.PUSH);
        buttonD.setLayoutData(new GridData(SWT.FILL, SWT.FILL, true, true));
        buttonD.setText("Button D");
    }

    public void run() {
        shell.pack();
        shell.open();

        while (!shell.isDisposed()) {
            if (!display.readAndDispatch()) {
                display.sleep();
            }
        }
        display.dispose();
    }

    public static void main(final String... args) {
        new SingleColumnGridLayoutExample().run();
    }

}

Çoklu Sütun #

Yukarıdaki Tek Sütunlu Örneğe benzer şekilde, bunun yerine GridLayout(int, boolean) yapıcısını kullanırsak, birden çok sütunlu bir düzen oluşturabiliriz.

Bu durumda, her biri aynı genişlikte olan iki sütun oluştururuz.

public class MultiColumnGridLayoutExample {

    private final Display display;
    private final Shell shell;

    public MultiColumnGridLayoutExample() {
        display = new Display();
        shell = new Shell(display);

        // Create the layout and apply to the Shell
        shell.setLayout(new GridLayout(2, true));

        // Add the child controls to the Shell - in this case, in two columns
        final Button buttonA = new Button(shell, SWT.PUSH);
        buttonA.setLayoutData(new GridData(SWT.FILL, SWT.FILL, true, true));
        buttonA.setText("Button A");
        
        final Button buttonB = new Button(shell, SWT.PUSH);
        buttonB.setLayoutData(new GridData(SWT.FILL, SWT.FILL, true, true));
        buttonB.setText("Button B");
        
        final Button buttonC = new Button(shell, SWT.PUSH);
        buttonC.setLayoutData(new GridData(SWT.FILL, SWT.FILL, true, true));
        buttonC.setText("Button C");
        
        final Button buttonD = new Button(shell, SWT.PUSH);
        buttonD.setLayoutData(new GridData(SWT.FILL, SWT.FILL, true, true));
        buttonD.setText("Button D");
    }

    public void run() {
        shell.pack();
        shell.open();

        while (!shell.isDisposed()) {
            if (!display.readAndDispatch()) {
                display.sleep();
            }
        }
        display.dispose();
    }

    public static void main(final String... args) {
        new MultiColumnGridLayoutExample().run();
    }

}

Varsayılan Olmayan Aralık ve Kenar Boşlukları #

“GridLayout” örneğindeki bazı üye değişkenlerden yararlanarak, yerleşimin etrafındaki kenar boşluklarını ve hücreler arasındaki boşlukları değiştirebiliriz. Bu örnekte aşağıdakileri ayarlıyoruz:

  1. ‘verticalSpacing = 0’ - Hücreler arasındaki dikey aralığı ‘0px’ olarak ayarlar.
  2. “horizontalSpacing = 20” - Hücreler arasındaki yatay boşluğu “20 piksel” olarak ayarlar.
  3. ‘marginWidth = 10’ - Düzenin sol ve sağ kenar boşluklarını ‘10px’ olarak ayarlar.

Not: “marginHeight"ı değiştirmeyiz, bu nedenle varsayılan “5px” değerinde kalır.

public class GridLayoutExample {

    private final Display display;
    private final Shell shell;

    public GridLayoutExample() {
        display = new Display();
        shell = new Shell(display);

        // Create a layout with two columns of equal width
        final GridLayout shellLayout = new GridLayout(2, true);
        shellLayout.verticalSpacing = 0; // Vertical spacing between cells
        shellLayout.horizontalSpacing = 20; // Horizontal spacing between cells
        shellLayout.marginWidth = 10; // Horizontal margin around the layout
        shell.setLayout(shellLayout);

        final Button buttonA = new Button(shell, SWT.PUSH);
        buttonA.setLayoutData(new GridData(SWT.FILL, SWT.FILL, true, true));
        buttonA.setText("Button A");

        final Button buttonB = new Button(shell, SWT.PUSH);
        buttonB.setLayoutData(new GridData(SWT.FILL, SWT.FILL, true, true));
        buttonB.setText("Button B");

        final Button buttonC = new Button(shell, SWT.PUSH);
        buttonC.setLayoutData(new GridData(SWT.FILL, SWT.FILL, true, true));
        buttonC.setText("Button C");

        final Button buttonD = new Button(shell, SWT.PUSH);
        buttonD.setLayoutData(new GridData(SWT.FILL, SWT.FILL, true, true));
        buttonD.setText("Button D");
    }

    public void run() {
        shell.pack();
        shell.open();

        while (!shell.isDisposed()) {
            if (!display.readAndDispatch()) {
                display.sleep();
            }
        }
        display.dispose();
    }

    public static void main(final String... args) {
        new GridLayoutExample().run();
    }

}
