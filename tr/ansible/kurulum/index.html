Ansible’ı Ubuntu’ya Kurma #

Ansible, Ansible ikili dosyalarını kurmak için kullanılabilecek bir PPA deposu bulundurur:

sudo apt-add-repository ppa:ansible/ansible -y
sudo apt-get update && sudo apt-get install ansible -y

Belirli bir sürümü yüklemek için “pip” kullanın. PPA güncelliğini yitirmiş olabilir.

Kaynaktan yükleme #

Ansible, kasadan en iyi kullanılır.

Sizin gibi çalışır (kök değil) ve minimum python bağımlılığına sahiptir.

pip ile Python pip bağımlılığı kurulumu:

sudo pip install paramiko PyYAML Jinja2 httplib2 six

Ardından, Ansible deposunu GitHub’dan klonlayın:

cd ~/Documents
git clone git://github.com/ansible/ansible.git --recursive 
cd ansible

Son olarak, ansible başlatma komut dosyası satırını ~/.bashrc veya ~/.zshrc dosyanıza ekleyin:

source ~/Documents/ansible/hacking/env-setup

Terminal oturumunuzu yeniden başlatın ve ile test edin

ansible --version

Ansible’ı MacOS’a Yükleme #

Homebrew veya Pip paket yöneticisini kullanarak Ansible’ı OS X’e kurmanın iki ana yolu vardır.

Homebrew’unuz varsa, en son Ansible aşağıdaki komut kullanılarak yüklenebilir:

brew install ansible

Ansible 1.9.X dalını kurmak için aşağıdaki komutu kullanın:

brew install homebrew/versions/ansible19

Ansible 2.0.X dalını kurmak için aşağıdaki komutu kullanın:

brew install homebrew/versions/ansible20

Pip kullanarak kurulum yapmak için şu komutu kullanın: pip install ansible.

Belirli bir sürümü yüklemek için pip install ansible=<gerekli sürüm> kullanın.

Red Hat tabanlı sistemlere kurulum #

Ansible, CentOS veya diğer Red Hat tabanlı sistemlere kurulabilir. Öncelikle önkoşulları yüklemelisiniz:

sudo yum -y update
sudo yum -y install gcc libffi-devel openssl-devel python-pip python-devel

sonra Ansible’ı pip ile kurun:

sudo pip install ansible

Kurulumdan sonra kurulum araçlarını yükseltmenizi tavsiye edebilirim:

sudo pip install --upgrade setuptools

Ayrıca yerel Paket Yöneticisini de kullanabilirsiniz:

yum install ansible

git repo’dan Amazon Linux’a kurulum #

Amazon Linux bir RHEL çeşididir, bu nedenle Red Hat talimatları çoğunlukla işe yaramalıdır. Bununla birlikte, en az bir tutarsızlık vardır.

python-devel yerine python27-devel paketinin açıkça gerekli olduğu bir örnek vardı.

Burada kaynaktan kurulum yapacağız.

sudo yum -y update
sudo yum -y install python27 python27-devel openssl-devel libffi-devel gcc git

git clone https://github.com/ansible/ansible/<search the github for a preferable branch>

cd ansible
sudo python setup.py build
sudo python setup.py install

Sanal Kutu+Vagrant Kullanarak Herhangi Bir İşletim Sistemi(windows) Makinesine Ansible Kurulumu #

Dizüstü bilgisayarım Windows 10 kullanıyor. Burada Ansible’ı test etmek ve öğrenmek için izleyebileceğiniz adımları veriyorum.

BAZI TEORİ

Ansible için Playbook’u çalıştırmak için bir Kontrol Makinesine ve bir ana bilgisayara (veya ana bilgisayarlara) ihtiyacınız vardır.

  • Kontrol Makinesi Linux tabanlı veya MacOS (pencerelere izin verilmez) olmalı ve Python’a (2.6 veya daha yüksek sürüm) ihtiyaç duymalıdır. Burada Ansible kurulacak.
  • Hedef makine (ana bilgisayar/düğüm) Linux/MacOS/windows olabilir. Bunun için yalnızca Python’un yüklenmesi gerekir. Aracı yazılımı gerekmez.

KURMAK

  1. Adım: Virtual Box yükleyin

Sanal kutu, farklı işletim sistemlerinden sanal bilgisayarlar oluşturmak için bir yazılımdır. Bu, birden fazla bilgisayara veya farklı işletim sistemine ve farklı sürümlere sahip olmak gibidir.

Sisteminizdeki işletim sistemine göre Virtual Box indirin ve kurun.

  1. Adım: Vagrant‘ı yükleyin

Vagrant, sanal kutuda sanal makineler oluşturmak için Komut Satırı Arabirimidir. Bu işleri kolaylaştırır. Temel Vagrant komutlarını öğrenmeniz gerekiyor.

Adım 3: Sanal makinenizi istediğiniz yerde bir klasör oluşturun

Adım 4: Vagrant kullanarak Sanal Makine oluşturun

Terminali açın ve klasörü oluşturduğunuz yola gidin ve aşağıdaki iki komutu çalıştırın.

Sanal Kutu öğesini seçmeniz gerekir. Örneğin Ubuntu kuruyorum. Listeden herhangi bir şey seçebilirsiniz. Bu iki komutu “sanal kutu” kategorisi altında çalıştırmanız gerekir: vagrant init ubuntu/trusty64 ve vagrant up --provider virtualbox. Diğer kategoriler şunlar olabilir: hyperv, vmware_desktop vb. (gerekli dosyaları indireceği için bu biraz zaman alacaktır)

Adım 4: Ansible’ı kurun

UbuntuOS için: sudo apt-get install ansible

Alternatif çözüm: #

Ansible alıştırması yapmak için Katacoda kullanabilirsiniz. Hiçbir şey yüklemenize veya ayarlamanıza gerek yok. 2. adımda verilen iki komutu çalıştırın ve bundan sonra gitmeye hazırsınız.
