TensorFlow'da Bir Modeli Kaydetme ve Geri Yükleme

  • Modeli kaydetme
  • Modeli geri yükleme

Bu sayfada

  • Modeli kaydetme
  • Modeli geri yükleme

Tensorflow, bir grafikteki tüm değişkenlerin mevcut değerlerini kaydetme/geri yükleme ile gerçek grafik yapısını kaydetme/geri yükleme arasında ayrım yapar. Grafiği geri yüklemek için, Tensorflow’un işlevlerini kullanmakta özgürsünüz ya da ilk etapta grafiği oluşturan kod parçanızı tekrar çağırabilirsiniz. Grafiği tanımlarken, grafik kaydedildikten ve geri yüklendikten sonra hangi değişkenlerin/işlemlerin geri alınabileceğini de düşünmelisiniz.

Yukarıdaki model geri yükleme bölümünde, eğer doğru anladıysam, modeli oluşturuyorsunuz ve ardından değişkenleri geri yüklüyorsunuz. tf.add_to_collection() kullanarak kaydederken ilgili tensörleri/yer tutucuları eklediğiniz sürece modeli yeniden oluşturmanın gerekli olmadığına inanıyorum. Örneğin:

tf.add_to_collection('cost_op', cost_op)

Daha sonra kaydedilen grafiği geri yükleyebilir ve kullanarak “cost_op"a erişebilirsiniz.

with tf.Session() as sess:
    new_saver = tf.train.import_meta_graph('model.meta')` 
    new_saver.restore(sess, 'model')
    cost_op = tf.get_collection('cost_op')[0]

tf.add_to_collection() komutunu çalıştırmasanız bile, tensörlerinizi geri alabilirsiniz, ancak süreç biraz daha zahmetlidir ve bazı şeyler için doğru isimleri bulmak için biraz araştırma yapmanız gerekebilir. Örneğin:

bir tensorflow grafiği oluşturan bir komut dosyasında, ’lab_squeeze’ bazı tensörler kümesi tanımlarız:

...
with tf.variable_scope("inputs"):
    y=tf.convert_to_tensor([[0,1],[1,0]])
    split_labels=tf.split(1,0,x,name='lab_split')
    split_labels=[tf.squeeze(i,name='lab_squeeze') for i in split_labels]
...
with tf.Session().as_default() as sess:
    saver=tf.train.Saver(sess,split_labels)
    saver.save("./checkpoint.chk")

bunları daha sonra şöyle hatırlayabiliriz:

with tf.Session() as sess:
    g=tf.get_default_graph()
    new_saver = tf.train.import_meta_graph('./checkpoint.chk.meta')` 
    new_saver.restore(sess, './checkpoint.chk')
    split_labels=['inputs/lab_squeeze:0','inputs/lab_squeeze_1:0','inputs/lab_squeeze_2:0']

    split_label_0=g.get_tensor_by_name('inputs/lab_squeeze:0') 
    split_label_1=g.get_tensor_by_name("inputs/lab_squeeze_1:0")

Bir tensörün adını bulmanın birkaç yolu vardır - onu tensör panosundaki grafiğinizde bulabilir veya şöyle bir şeyle arayabilirsiniz:

sess=tf.Session()
g=tf.get_default_graph()
...
x=g.get_collection_keys()
[i.name for j in x for i in g.get_collection(j)] # will list out most, if not all, tensors on the graph

Modeli kaydetme #

Bir modeli tensorflow’ta kaydetmek oldukça kolaydır.

Diyelim ki “x” girdili doğrusal bir modeliniz var ve “y” çıktısını tahmin etmek istiyorsunuz. Buradaki kayıp, ortalama kare hatasıdır (MSE). Parti boyutu 16’dır.

# Define the model
x = tf.placeholder(tf.float32, [16, 10])  # input
y = tf.placeholder(tf.float32, [16, 1])   # output

w = tf.Variable(tf.zeros([10, 1]), dtype=tf.float32)

res = tf.matmul(x, w)
loss = tf.reduce_sum(tf.square(res - y))

train_op = tf.train.GradientDescentOptimizer(0.01).minimize(loss)

İşte birden fazla parametreye sahip olabilen Saver nesnesi geliyor (cf. doc).

# Define the tf.train.Saver object
# (cf. params section for all the parameters)    
saver = tf.train.Saver(max_to_keep=5, keep_checkpoint_every_n_hours=1)

Son olarak, modeli bir “tf.Session()” içinde, “1000” yineleme için eğitiriz. Modeli burada yalnızca her ‘100’ yinelemede bir kaydederiz.

# Start a session
max_steps = 1000
with tf.Session() as sess:
    # initialize the variables
    sess.run(tf.initialize_all_variables())

    for step in range(max_steps):
        feed_dict = {x: np.random.randn(16, 10), y: np.random.randn(16, 1)}  # dummy input
        _, loss_value = sess.run([train_op, loss], feed_dict=feed_dict)

        # Save the model every 100 iterations
        if step % 100 == 0:
            saver.save(sess, "./model", global_step=step)

Bu kodu çalıştırdıktan sonra dizininizdeki son 5 kontrol noktasını görmelisiniz:

  • “model-500” ve “model-500.meta”
  • “model-600” ve “model-600.meta”
  • “model-700” ve “model-700.meta”
  • “model-800” ve “model-800.meta”
  • “model-900” ve “model-900.meta”

Bu örnekte, “koruyucu” aslında hem değişkenlerin mevcut değerlerini bir kontrol noktası olarak hem de grafiğin yapısını (’*.meta’) kaydederken, örneğin model geri yüklendiğinde ‘x’ ve ‘y’ yer tutucuları. Örneğin. geri yükleme bu eğitim komut dosyasından başka bir yerde yapılırsa, geri yüklenen grafikten “x” ve “y"yi almak zahmetli olabilir (özellikle daha karmaşık modellerde). Bunu önlemek için, her zaman değişkenlerinize / yer tutucularınıza / operasyonlarınıza ad verin veya açıklamalardan birinde gösterildiği gibi tf.collections kullanmayı düşünün.

Modeli geri yükleme #

Geri yükleme de oldukça güzel ve kolaydır.

İşte kullanışlı bir yardımcı fonksiyon:

def restore_vars(saver, sess, chkpt_dir):
    """ Restore saved net, global score and step, and epsilons OR
    create checkpoint directory for later storage. """
    sess.run(tf.initialize_all_variables())

    checkpoint_dir = chkpt_dir 

    if not os.path.exists(checkpoint_dir):
        try:
            print("making checkpoint_dir")
            os.makedirs(checkpoint_dir)
            return False
        except OSError:
            raise

    path = tf.train.get_checkpoint_state(checkpoint_dir)
    print("path = ",path)
    if path is None:
        return False
    else:
        saver.restore(sess, path.model_checkpoint_path)
        return True

Ana kod:

path_to_saved_model = './'
max_steps = 1

# Start a session
with tf.Session() as sess:

    ... define the model here ...

    print("define the param saver")
    saver = tf.train.Saver(max_to_keep=5, keep_checkpoint_every_n_hours=1)

    # restore session if there is a saved checkpoint
    print("restoring model")
    restored = restore_vars(saver, sess, path_to_saved_model)
    print("model restored ",restored)

    # Now continue training if you so choose

    for step in range(max_steps):

        # do an update on the model (not needed)
        loss_value = sess.run([loss])
        # Now save the model
        saver.save(sess, "./model", global_step=step)
