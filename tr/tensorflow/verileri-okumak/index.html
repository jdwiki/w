Bir TXT dosyasından resimler ve etiketler nasıl yüklenir #

Tensorflow belgelerinde görüntülerin ve etiketlerin doğrudan bir TXT dosyasından nasıl yükleneceği açıklanmamıştır. Aşağıdaki kod, bunu nasıl başardığımı göstermektedir. Ancak bu, bunu yapmanın en iyi yolunun bu olduğu ve bu yolun sonraki adımlarda yardımcı olacağı anlamına gelmez.

Örneğin, belgeler bir sıcak vektör [0,1] kullanırken, etiketleri tek bir tamsayı değeri {0,1} olarak yüklüyorum.

# Learning how to import images and labels from a TXT file
#
# TXT file format
#
# path/to/imagefile_1 label_1
# path/to/imagefile_2 label_2
# ...                 ...
#
# where label_X is either {0,1}

#Importing Libraries
import os
import tensorflow as tf
import matplotlib.pyplot as plt
from tensorflow.python.framework import ops
from tensorflow.python.framework import dtypes

#File containing the path to images and the labels [path/to/images label]
filename = '/path/to/List.txt'

#Lists where to store the paths and labels
filenames = []
labels = []

#Reading file and extracting paths and labels
with open(filename, 'r') as File:
    infoFile = File.readlines() #Reading all the lines from File
    for line in infoFile: #Reading line-by-line
        words = line.split() #Splitting lines in words using space character as separator
        filenames.append(words[0])
        labels.append(int(words[1]))

NumFiles = len(filenames)

#Converting filenames and labels into tensors
tfilenames = ops.convert_to_tensor(filenames, dtype=dtypes.string)
tlabels = ops.convert_to_tensor(labels, dtype=dtypes.int32)

#Creating a queue which contains the list of files to read and the value of the labels
filename_queue = tf.train.slice_input_producer([tfilenames, tlabels], num_epochs=10, shuffle=True, capacity=NumFiles)

#Reading the image files and decoding them
rawIm= tf.read_file(filename_queue[0])
decodedIm = tf.image.decode_png(rawIm) # png or jpg decoder

#Extracting the labels queue
label_queue = filename_queue[1]

#Initializing Global and Local Variables so we avoid warnings and errors
init_op = tf.group(tf.local_variables_initializer() ,tf.global_variables_initializer())

#Creating an InteractiveSession so we can run in iPython
sess = tf.InteractiveSession()

with sess.as_default():
    sess.run(init_op)
    
    # Start populating the filename queue.
    coord = tf.train.Coordinator()
    threads = tf.train.start_queue_runners(coord=coord)

    for i in range(NumFiles): #length of your filenames list
        nm, image, lb = sess.run([filename_queue[0], decodedIm, label_queue])
        
        print image.shape
        print nm
        print lb
        
        #Showing the current image
        plt.imshow(image)
        plt.show()

    coord.request_stop()
    coord.join(threads)

Örnekleri rastgele karıştırma #

Örnekleri rastgele karıştırmak için, ’tf.train.batch’ yerine ’tf.train.shuffle_batch’ işlevini aşağıdaki gibi kullanabilirsiniz:

parsed_batch = tf.train.shuffle_batch([serialized_example],
    batch_size=100, capacity=1000,
    min_after_dequeue=200)

“tf.train.shuffle_batch” (aynı zamanda “tf.train.batch”) bir “tf.Queue” oluşturur ve ona “serileştirilmiş_örnekler” eklemeye devam eder.

“kapasite”, bir seferde Kuyrukta kaç öğenin depolanabileceğini ölçer. Daha büyük kapasite, daha büyük bellek kullanımına yol açar, ancak onu doldurmayı bekleyen iş parçacıklarının neden olduğu daha düşük gecikme süresi.

“min_after_dequeue”, kuyruktan öğeler alındıktan sonra kuyrukta bulunan minimum öğe sayısıdır. ‘shuffle_batch’ kuyruğu, öğeleri mükemmel bir şekilde homojen bir şekilde karıştırmaz - akılda kalıcı olmayan, büyük verilerle tasarlanmıştır. Bunun yerine, “min_after_dequeue” ve “kapasite” öğelerini okur, bunları bellekte saklar ve rastgele bir yığın seçer. Bundan sonra, sayısını “min_after_dequeue” ve “kapasite” arasında tutmak için bazı öğeleri daha sıraya koyar. Bu nedenle, “min_after_dequeue” değeri ne kadar büyükse, öğeler o kadar rastgele olur - “batch_size” öğelerinin seçiminin en az “min_after_dequeue” ardışık öğelerinden alınması garanti edilir, ancak “kapasite” ne kadar büyük ve o kadar uzun olmalıdır başlangıçta kuyruğu doldurmak için sürer.

Yığınlama ile n dönem için veri okuma #

Veri örneklerinizin zaten bir python değişkenine okunduğunu ve verilen boyuttaki gruplar halinde n kez okumak istediğinizi varsayalım:

import numpy as np
import tensorflow as tf
data = np.array([1, 2, 3, 4, 5])
n = 4

Verileri yığınlar halinde, muhtemelen rastgele karıştırma ile birleştirmek için, “tf.train.batch” veya “tf.train.batch_shuffle” kullanabilirsiniz, ancak ona tüm verileri n kez üretecek tensörü iletmeniz gerekir:

limited_tensor = tf.train.limit_epochs(data, n)
batch = tf.train.shuffle_batch([limited_tensor], batch_size=3, enqueue_many=True, capacity=4)

’limit_epochs’, numpy dizisini kaputun altındaki tensöre dönüştürür ve n kez üreten bir tensör döndürür ve ardından bir OutOfRangeError atar. “shuffle_batch” öğesine iletilen “enqueue_many=True” bağımsız değişkeni, “[limited_tensor]” tensör listesindeki her tensörün bir dizi örnek içerdiği şeklinde yorumlanması gerektiğini belirtir. Toplu işlem kuyruğunun kapasitesinin, tensördeki örnek sayısından daha küçük olabileceğini unutmayın.

Veriler her zamanki gibi işlenebilir:

with tf.Session() as sess:
  sess.run(tf.initialize_local_variables())
  tf.train.start_queue_runners()
  try:
    while True:
      data_batch = sess.run(batch)
      # process data
  except tf.errors.OutOfRangeError:
    pass

CSV dosyasındaki örnekleri sayın #

import tensorflow as tf
filename_queue = tf.train.string_input_producer(["file.csv"], num_epochs=1)
reader = tf.TextLineReader()
key, value = reader.read(filename_queue)
col1, col2 = tf.decode_csv(value, record_defaults=[[0], [0]])

with tf.Session() as sess:
  sess.run(tf.initialize_local_variables())
  tf.train.start_queue_runners()
  num_examples = 0
  try:
    while True:
      c1, c2 = sess.run([col1, col2])
      num_examples += 1
  except tf.errors.OutOfRangeError:
    print "There are", num_examples, "examples"

’num_epochs=1’, listedeki her dosyayı bir kez işledikten sonra ‘string_input_producer’ kuyruğunun kapanmasını sağlar. ’try:’ sırasında yakalanan ‘OutOfRangeError’ın yükselmesine yol açar. Varsayılan olarak, “string_input_producer” dosya adlarını sonsuz sayıda üretir.

’tf.initialize_local_variables()’, yürütüldüğünde ‘string_input_producer’ içindeki ’num_epoch’ local değişkenini başlatan bir tensorflow Op’dur.

tf.train.start_queue_runners(), sıralara eşzamansız olarak veri eklemeyi işleyen ekstra basamakları başlatır.

TFRecord dosyasını oku ve ayrıştır #

TFRecord dosyaları, verileri (tensörler) depolamak için yerel tensorflow ikili biçimidir. Dosyayı okumak için CSV örneğine benzer bir kod kullanabilirsiniz:

import tensorflow as tf
filename_queue = tf.train.string_input_producer(["file.tfrecord"], num_epochs=1)
reader = tf.TFRecordReader()
key, serialized_example = reader.read(filename_queue)

Ardından, “serialized_example” Queue’daki örnekleri ayrıştırmanız gerekir. Bunu, daha önce gruplama gerektiren ancak daha hızlı “tf.parse_example” veya “tf.parse_single_example” kullanarak yapabilirsiniz:

batch = tf.train.batch([serialized_example], batch_size=100)
parsed_batch = tf.parse_example(batch, features={
  "feature_name_1": tf.FixedLenFeature(shape=[1], tf.int64),
  "feature_name_2": tf.FixedLenFeature(shape=[1], tf.float32)
})

“tf.train.batch”, “[x, y, z]” şeklindeki verilen tensörlerin ardışık değerlerini “[batch_size, x, y, z]” şeklindeki tensörlerle birleştirir. “features” dict, özelliklerin adlarını tensorflow’un [özellikler] tanımlarıyla eşler(https://www.tensorflow.org/versions/r0.10/api_docs/python/io_ops.html#FixedLenFeature). “parse_single_example"ı benzer şekilde kullanırsınız:

parsed_example = tf.parse_single_example(serialized_example, {
  "feature_name_1": tf.FixedLenFeature(shape=[1], tf.int64),
  "feature_name_2": tf.FixedLenFeature(shape=[1], tf.float32)
})

’tf.parse_example’ ve ’tf.parse_single_example’, değerlerle tensöre bir sözlük eşleme özelliği adları döndürür.

‘parse_single_example’dan gelen toplu örnekler için, tensörleri dict’den çıkarmalı ve daha önce olduğu gibi ’tf.train.batch’ kullanmalısınız:

parsed_batch = dict(zip(parsed_example.keys(),
    tf.train.batch(parsed_example.values(), batch_size=100)

Verileri daha önce olduğu gibi okursunuz ve “sess.run” için değerlendirmek üzere tüm tensörlerin listesini geçirirsiniz:

with tf.Session() as sess:
  sess.run(tf.initialize_local_variables())
  tf.train.start_queue_runners()
  try:
    while True:
      data_batch = sess.run(parsed_batch.values())
      # process data
  except tf.errors.OutOfRangeError:
    pass
