Parametreler #

parametre Ayrıntılar
veri türü (dtype) özellikle tensorflow paketi tarafından sağlanan veri türlerinden biri. Örneğin. tensorflow.float32
veri şekli (şekil) Yer tutucunun liste veya demet olarak boyutları. “Yok”, bilinmeyen boyutlar için kullanılabilir. Örneğin. (Yok,30) bir (? x 30) boyut yer tutucusu tanımlar
isim (isim) İşlem için bir ad (isteğe bağlı).

Yer Tutucuların Temelleri #

Yer tutucular, değerleri bir tensorflow grafiğine beslemenize olanak tanır. Ek olarak, beslenen değerlerin boyutları ve veri türü ile ilgili kısıtlamaları belirlemenize izin verirler. Bu nedenle, yeni eğitim örneklerini beslemek için bir sinir ağı oluştururken kullanışlıdırlar.

Aşağıdaki örnek, 32 bitlik kayan noktalar olan (veya buna tip olarak yayınlanabilen) öğelere sahip 3’e 4 tensör için bir yer tutucu bildirir.

a = tf.placeholder(tf.float32, shape=[3,4], name='a')

Yer tutucular kendi başlarına herhangi bir değer içermeyecektir, bu nedenle bir oturum çalıştırırken onları değerlerle beslemek önemlidir, aksi takdirde bir hata mesajı alırsınız. Bu, “session.run()” çağrılırken “feed_dict” argümanı kullanılarak yapılabilir, örneğin:

# run the graph up to node b, feeding the placeholder `a` with values in my_array 
session.run(b, feed_dict={a: my_array})

İşte bir yer tutucuyu bildirme ve besleme sürecinin tamamını gösteren basit bir örnek.

import tensorflow as tf
import numpy as np

# Build a graph
graph = tf.Graph()
with graph.as_default():
    # declare a placeholder that is 3 by 4 of type float32
    a = tf.placeholder(tf.float32, shape=(3, 4), name='a')
    
    # Perform some operation on the placeholder
    b = a * 2
    
# Create an array to be fed to `a`
input_array = np.ones((3,4))

# Create a session, and run the graph
with tf.Session(graph=graph) as session:
    # run the session up to node b, feeding an array of values into a
    output = session.run(b, feed_dict={a: input_array})
    print(output)

Yer tutucu 3’e 4 bir birler dizisi alır ve bu tensör daha sonra b düğümünde 2 ile çarpılır, ardından aşağıdakileri döndürür ve yazdırır:

[[ 2.  2.  2.  2.]
 [ 2.  2.  2.  2.]
 [ 2.  2.  2.  2.]]

Varsayılan Yer Tutucu #

Genellikle, derin bir ağ eğitimi sırasında bir veya daha fazla doğrulama grubunu aralıklı olarak çalıştırmak ister. Tipik olarak eğitim verileri bir kuyruk tarafından beslenirken, doğrulama verileri “sess.run()” içindeki “feed_dict” parametresinden geçirilebilir. tf.placeholder_with_default() bu durumda iyi çalışacak şekilde tasarlanmıştır:

import numpy as np
import tensorflow as tf

IMG_SIZE = [3, 3]
BATCH_SIZE_TRAIN = 2
BATCH_SIZE_VAL = 1

def get_training_batch(batch_size):
    ''' training data pipeline '''
    image = tf.random_uniform(shape=IMG_SIZE)
    label = tf.random_uniform(shape=[])
    min_after_dequeue = 100
    capacity = min_after_dequeue + 3 * batch_size
    images, labels = tf.train.shuffle_batch(
        [image, label], batch_size=batch_size, capacity=capacity,
        min_after_dequeue=min_after_dequeue)
    return images, labels

# define the graph
images_train, labels_train = get_training_batch(BATCH_SIZE_TRAIN)
image_batch = tf.placeholder_with_default(images_train, shape=None)
label_batch = tf.placeholder_with_default(labels_train, shape=None)
new_images = tf.mul(image_batch, -1)
new_labels = tf.mul(label_batch, -1)

# start a session
with tf.Session() as sess:
    sess.run(tf.initialize_all_variables())
    coord = tf.train.Coordinator()
    threads = tf.train.start_queue_runners(sess=sess, coord=coord)

    # typical training step where batch data are drawn from the training queue
    py_images, py_labels = sess.run([new_images, new_labels])
    print('Data from queue:')
    print('Images: ', py_images)  # returned values in range [-1.0, 0.0]
    print('\nLabels: ', py_labels) # returned values [-1, 0.0]

    # typical validation step where batch data are supplied through feed_dict
    images_val = np.random.randint(0, 100, size=np.hstack((BATCH_SIZE_VAL, IMG_SIZE)))
    labels_val = np.ones(BATCH_SIZE_VAL)
    py_images, py_labels = sess.run([new_images, new_labels],
                      feed_dict={image_batch:images_val, label_batch:labels_val})
    print('\n\nData from feed_dict:')
    print('Images: ', py_images) # returned values are integers in range [-100.0, 0.0]
    print('\nLabels: ', py_labels) # returned values are -1.0

    coord.request_stop()
    coord.join(threads)

Bu örnekte ‘image_batch’ ve ’label_batch’, ‘sess.run()’ çağrısı sırasında karşılık gelen değerler ‘feed_dict’ parametresi olarak iletilmedikçe ‘get_training_batch()’ tarafından üretilir.
