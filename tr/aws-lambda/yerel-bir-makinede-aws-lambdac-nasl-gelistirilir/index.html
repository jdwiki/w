Sunucusuz hareketinin bir parçası olarak, AWS kısa süre önce AWS Lambda işlevleri için C# desteğini duyurdu. Bu yazımda size Visual Studio 2015 kullanarak basit bir C# AWS Lambda Function nasıl geliştireceğinizi, kuracağınızı ve çağıracağınızı göstereceğim.

Bu örneğe başlamadan önce, ilk olarak bu belgenin Açıklamalar bölümüne gidin ve gerekli tüm bileşenlerin kurulu olduğundan emin olun.

Gerekli Bileşenleri Yükleyin #

Bu yazının yazıldığı tarihte (Ocak 2017), aşağıdaki bileşenlerin bu sırayla makinenize yüklenmesi gerekecektir. Kullandığım makinede Windows 10 Home kuruluydu.

  1. Visual Studio 2015 güncellemesi 3
  2. Visual Studio 2015 için AWS Toolkit - sürüm 1.11.2.0
  3. Microsoft .NET Core Tools (Önizleme 2) - sürüm 14.1.21111.0

Bu bileşenlerin her biri aktif olarak geliştirilir ve yayınlanır, bu nedenle bağlantıları ve sürümleri iki kez kontrol edin ve gerektiğinde bu belgeyi güncelleyin.

Bunların her birinin yüklenmesi biraz zaman alabilir, bu yüzden sabırlı olun, hataları en aza indirmek için birer birer yapın.

Son olarak, geliştirmeye başlamadan önce Windows’u yeniden başlatmak üzücüdür, ancak bir Windows kutusunda büyük geliştirme aracı kurulumları yaptıktan sonra her zaman iyi bir fikirdir.

Adım 1. Visual Studio’da yeni bir çözüm oluşturun #

  1. Visual Studio’yu açın ve Dosya Seçin -> Yeni Proje
  2. Testli AWS Lambda Projesi’ni seçin (.NET Core)
  3. Ardından Taslak Seç ekranı görüntülenecektir. Boş İşlev‘i seçin ve Bitir düğmesini tıklayın:
  4. Araçlar -> NuGet Paket Yöneticisi -> Paket Yöneticisi Konsolu’na gidin.
  5. Konsol penceresinde Install-Package Amazon.Lambda.Core yazın
  6. Çözüm gezgininde projenize sağ tıklayın ve Paketleri Geri Yükle’yi seçin. Bu, kod yazmaya başlamadan önceki son hazırlık adımıdır.

Adım 2. Projenize kod ekleyin ve kodu değiştirin #

  1. Function.cs dosyasını açın ve sınıf kodunu aşağıdaki ile değiştirin:

     public class Function
 {

 /// <summary>
 /// A simple function that takes a birth date and returns Age in years
 /// </summary>
 /// <param name="input"></param>
 /// <returns>Age is years</returns>
 /// 
 [LambdaSerializer(typeof(SimpleSerializer))]
 public string FunctionHandler(Dictionary<string, int> input)
 {
     var defaultMessage = "Age could not be determined.";

     var birthDate = new DateTime(input["year"], input["month"], input["day"]);
     var ageInYears = DateTime.Today.Year - birthDate.Year;
     if (birthDate.DayOfYear > DateTime.Today.DayOfYear)
         ageInYears--;

     defaultMessage = $"Age in years: {ageInYears}";

     return defaultMessage;
 }
 }

En üste yakın bir yerde aşağıdaki using ifadelerini eklemeniz gerekecek:

using System.Collections.Generic;
using Amazon.Lambda.Core;

  3. Test Projesinde, “FunctionTest.cs” dosyasının 23. satırını aşağıdaki şekilde değiştirin:

  4. Çözümünüzü oluşturun - derleme hatanız olmamalıdır.

Adım 3 - Lambda İşlevinizi AWS’ye yükleyin #

  1. Projenize sağ tıklayın ve AWS Lambda’da Yayınla… öğesini seçin.

  2. AWS Lambda’ya Yükle ekranı görünecektir. Doğru bölgenin seçildiğinden emin olun. Tüm varsayılanları koru. Ardından yalnızca İşlev Adını girin: AWSLambdaFunctionAgeInYears ve ardından ileri’ye tıklayın.

  3. Sonraki sayfada, Rol Adı alanı için AWSLambdaRole’u seçin. Yükle’ye tıklayın ve işlev hatasız olarak yüklenmelidir.

Adım 4. AWS Lambda işlevinizi Visual Studio içinden çağırın #

  1. Yukarıdaki 3. Adımdan sonra, Visual Studio, işleviniz yüklenmiş olarak `İşlev Görüntüle’ penceresini açacaktır.

  2. Bu ekranın sol alt köşesindeki Örnek Giriş kutusuna aşağıdaki json’u girin:

    { “month”: “10”, “day”: “28”, “year”: “1979” }

  1. Son adım: Yeşil Çağrı düğmesini tıklayın. İşlev AWS’de çalışacak ve yanıt sağ alt Yanıt penceresinde görüntülenecektir.
