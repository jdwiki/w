Sözdizimi #

  • Seri durumdan çıkarılmış JSON tabanlı nesne
  • “Kayıtlar” anahtarında bir veya daha fazla gerçek olay var
  • Her alt olay nesnesi, neyin değiştiğini belirlemek için ihtiyacınız olan tüm bilgileri içerir

Parametreler #

parametre Ayrıntılar
Kayıtlar -> [] -> s3 -> kova -> ad S3 paketinin adı
Kayıtlar -> [] -> s3 -> nesne -> anahtar Dosyanın yolu ve adı.

Dosya adları #

Çoğu dosya yolunun aksine, S3 anahtar adı (JSON Şeması: $.Records[0].s3.object.key) başta eğik çizgi içermez. Bu nedenle, s3://mybucket/path/file.txt yolunda bir dosyanız varsa, anahtar path/file.txt olacaktır.

En azından Python’da anahtar alan değeri UTF-8 URL kodludur. Bu, dosya adı boşluk veya ascii olmayan karakterler içerdiğinde fark edilir. Alanın URL kodunun çözülmesi, ardından UTF-8 kodunun çözülmesi gerekir - Bkz. http://stackoverflow.com/questions/39465220/get-non-ascii-filename-from-s3-notification-event-in-lambda

Kayıt anahtarı #

Olayın Kayıtlar anahtarının içinde aynı (veya farklı) birden çok işlemin olması mümkündür; ancak pratikte, genellikle Lambda işlevinizin çağrılması başına bir olay görürsünüz.

Daha Fazla Örnek ve Test #

Eylemler -> Test Olayını Yapılandır’ı seçerseniz, Lambda konsolunda aslında örnek olaylar vardır. Ancak yukarıdaki örneklerde PUT işlemini görebilirsiniz.

İşlevinizin nasıl yanıt verdiğini görmek için test olaylarını AWS Lambda konsolundan değiştirebilir ve gönderebilirsiniz.
