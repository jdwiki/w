Python kullanarak Kurulum veya Kurulum #

1) scikit öğrenme

scikit-learn, SciPy üzerine inşa edilmiş ve 3-Clause BSD lisansı altında dağıtılmış, makine öğrenimi için bir Python modülüdür. Destek vektör makineleri, rastgele ormanlar, gradyan artırma, k-means ve DBSCAN dahil olmak üzere çeşitli sınıflandırma, regresyon ve kümeleme algoritmalarına sahiptir ve Python sayısal ve bilimsel kitaplıkları NumPy ve SciPy ile birlikte çalışmak üzere tasarlanmıştır.

scikit-learn’in mevcut kararlı sürümü gerektirir:

  • Python (>= 2.6 veya >= 3.3),
  • NumPy (>= 1.6.1),
  • SciPy (>= 0.9).

Çoğu kurulum için ‘pip’ python paket yöneticisi python’u ve tüm bağımlılıklarını yükleyebilir:

pip install scikit-learn

Ancak linux sistemlerinde olası derleme işlemlerinden kaçınmak için “conda” paket yöneticisinin kullanılması önerilir.

conda install scikit-learn

scikit-learn olup olmadığını kontrol etmek için Shell’de yürütün:

python -c 'import sklearn; print(sklearn.__version__)'

Windows ve Mac OSX Kurulumu:

Canopy ve Anaconda, Windows, Mac OSX (ayrıca Linux için de geçerlidir) için geniş bir bilimsel python kitaplığına ek olarak scikit-learn‘ün yeni bir sürümünü sunar.

Resmi kaynak kodu deposu: https://github.com/scikit-learn/scikit-learn

2) Akıllı Bilgi İşlem için Numenta Platformu

Numenta Akıllı Bilgi İşlem Platformu (NuPIC), HTM öğrenme algoritmalarını uygulayan bir makine zekası platformudur. HTM, neokorteksin ayrıntılı bir hesaplama teorisidir. HTM’nin merkezinde, uzamsal ve zamansal kalıpları depolayan ve geri çağıran zamana dayalı sürekli öğrenme algoritmaları bulunur. NuPIC, özellikle akış veri kaynaklarının anormallik tespiti ve tahmini olmak üzere çeşitli sorunlara uygundur.

NuPIC ikili dosyaları şunlar için kullanılabilir:

Linux x86 64bit OS X 10.9 OS X 10.10 Windows 64bit

NuPIC’i tüm işletim sistemlerine yüklemek için aşağıdaki bağımlılıklar gereklidir.

  • Python 2.7
  • pip>=8.1.2
  • kurulum araçları>=25.2.0
  • tekerlek>=0.29.0
  • dizi
  • gcc (4.8+) veya clang gibi C++ 11 derleyicisi

Ek OS X gereksinimleri:

  • Xcode komut satırı araçları

NuPIC’i yüklemek için aşağıdakileri çalıştırın:

pip install nupic

Resmi kaynak kodu deposu: https://github.com/numenta/nupic

3) yeni bilgi

Nilearn, NeuroImaging verileri üzerinde hızlı ve kolay istatistiksel öğrenme için bir Python modülüdür. Tahmine dayalı modelleme, sınıflandırma, kod çözme veya bağlantı analizi gibi uygulamalarla çok değişkenli istatistikler için scikit-learn Python araç kutusundan yararlanır.

Yazılımı kullanmak için gerekli bağımlılıklar şunlardır:

  • Python >= 2.6,
  • kurulum araçları
  • Numpy >= 1.6.1
  • Bilim >= 0.9
  • Scikit-öğren >= 0.14.1
  • Nibabel >= 1.1.0

Nilearn çizim işlevlerini kullanıyorsanız veya örnekleri çalıştırıyorsanız, matplotlib >= 1.1.1 gereklidir.

Testleri yapmak istiyorsanız, burun >= 1.2.1 ve kapsama alanı >= 3.6’ya ihtiyacınız var.

Öncelikle yukarıda listelenen tüm bağımlılıkları kurduğunuzdan emin olun. Ardından, komut isteminde aşağıdaki komutu çalıştırarak nilearn’i yükleyebilirsiniz:

pip install -U --user nilearn

Resmi kaynak kodu deposu: https://github.com/nilearn/nilearn/

4) Anakonda’yı Kullanma

Anaconda’da birçok bilimsel Python kitaplığı hazırdır. Kurulum dosyalarını buradan alabilirsiniz. Bir yandan Anaconda’yı kullanarak çok fazla paket kurmanız ve yapılandırmanız gerekmez, BSD lisanslıdır ve Python 3 ve Python 2 için mevcut olan önemsiz kurulum işlemlerine sahiptir, diğer yandan size daha az esneklik sağlar. Örnek olarak, bazı son teknoloji derin öğrenme python paketleri, Anaconda’nın kurulu olduğu farklı bir numpy sürümü kullanabilir. Ancak, bu dezavantaj, başka bir python kurulumunun ayrı olarak kullanılmasıyla ele alınabilir (örneğin, linux ve MAC’de varsayılanınız).

Anaconda kurulumu sizden kurulum yeri seçimine ve ayrıca PATH ekleme seçeneğine yönlendirir. Anaconda’yı PATH’inize eklerseniz, işletim sisteminizin varsayılan olarak Anaconda Python’u bulması beklenir. Bu nedenle, değişiklikler ve gelecekteki kurulumlar yalnızca bu Python sürümü için geçerli olacaktır.

Açıklığa kavuşturmak için, Anaconda’nın kurulumundan sonra ve yazarsanız, Ubuntu 14.04’ü terminal üzerinden kullanarak PATH’e eklersiniz.

python

Voila, Anaconda Python sizin varsayılan Python’unuzdur, birçok kütüphaneyi hemen kullanmaya başlayabilirsiniz. Ancak, eski Python’unuzu kullanmak istiyorsanız

/usr/bin/python

Uzun lafın kısası Anaconda, Python ile makine öğrenimi ve veri analizine başlamanın en hızlı yollarından biridir.

R Dilini Kullanarak Kurulum veya Kurulum #

[Paketler](https://www.wikiod.com/tr/r/paketleri-yukleme#Depolardan paketleri indirin ve yükleyin) R işlevleri, verileri ve derlenmiş koleksiyonlardır iyi tanımlanmış bir biçimde kodlayın. Genel (ve özel) depolar, R paketlerinin koleksiyonlarını barındırmak için kullanılır. En büyük R paketleri koleksiyonu CRAN’da mevcuttur. En popüler R makine öğrenimi paketlerinden bazıları, diğerleri arasında aşağıdaki gibidir:

1) bölüm

Açıklama: Sınıflandırma, regresyon ve hayatta kalma ağaçları için özyinelemeli bölümleme. Breiman, Friedman, Olshen ve Stone’un 1984 kitabının işlevselliğinin çoğunun bir uygulaması.

Aşağıdaki kod kullanılarak CRAN’dan yüklenebilir:

install.packages("rpart")

Paketi yükleyin:

library(rpart)

Resmi kaynak: https://cran.r-project.org/web/packages/rpart/index.html

2) e1071

Açıklama: Gizli sınıf analizi, kısa süreli Fourier dönüşümü, bulanık kümeleme, destek vektör makineleri, en kısa yol hesaplaması, torbalı kümeleme, saf Bayes sınıflandırıcısı vb. için işlevler.

CRAN’dan kurulum:

install.packages("e1071")

Paket yükleniyor:

library(e1071)

Resmi kaynak: https://cran.r-project.org/web/packages/e1071/index.html

3) rastgele Orman

Açıklama: Rastgele girdiler kullanan bir ağaç ormanına dayalı sınıflandırma ve regresyon.

CRAN’dan kurulum:

install.packages("randomForest")

Paket yükleniyor:

library(randomForest)

Resmi kaynak: https://cran.r-project.org/web/packages/randomForest/index.html

4) şapka

Açıklama: Sınıflandırma ve regresyon modellerini eğitmek ve çizmek için çeşitli işlevler.

CRAN’dan kurulum:

install.packages("caret")

Paket yükleniyor:

library(caret)

Resmi kaynak: https://cran.r-project.org/web/packages/caret/index.html
