Aktivasyon Fonksiyonları #

Aktarım işlevi olarak da bilinen etkinleştirme işlevleri, giriş düğümlerini belirli bir şekilde çıkış düğümlerine eşlemek için kullanılır.

Bir sinir ağı katmanının çıktısına doğrusal olmayanlık kazandırmak için kullanılırlar.

Yaygın olarak kullanılan bazı fonksiyonlar ve eğrileri aşağıda verilmiştir:

Sigmoid İşlevi #

Sigmoid, çıktısı “[0, 1]” aralığında olan bir ezme işlevidir.

’numpy’ ile türevi ile birlikte sigmoid uygulamak için kod aşağıda gösterilmiştir:

def sigmoid(z):
    return 1.0 / (1.0 + np.exp(-z))

def sigmoid_prime(z):
    return sigmoid(z) * (1 - sigmoid(z))

Hiperbolik Tanjant Fonksiyonu (tanh) #

Tanh ve sigmoid fonksiyonları arasındaki temel fark, tanh’ın 0 merkezli olması, girdileri [-1, 1] aralığına sıkıştırması ve hesaplanmasının daha verimli olmasıdır.

Gizli bir katmanın aktivasyonunu hesaplamak için ’np.tanh’ veya ‘math.tanh’ işlevlerini kolayca kullanabilirsiniz.

ReLU İşlevi #

Doğrultulmuş bir doğrusal birim basitçe “max(0,x)” yapar. Sinir ağı birimlerinin aktivasyon fonksiyonları için en yaygın seçeneklerden biridir.

ReLU’lar, sigmoid / hiperbolik tanjant birimlerinin kaybolan gradyan problemini ele alır, böylece derin ağlarda verimli gradyan yayılımına izin verir.

ReLU adı, Nair ve Hinton’un makalesinden gelmektedir, Rektifiye Edilmiş Doğrusal Birimler Kısıtlı Boltzmann Makinelerini İyileştirir.

Sızdıran ReLU’lar (LReLU’lar) ve Üstel Doğrusal Birimler (ELU’lar) gibi bazı varyasyonları vardır.

Vanilla ReLU’yu ’numpy’ ile türeviyle birlikte uygulama kodu aşağıda gösterilmiştir:

def relU(z):
    return z * (z > 0)

def relU_prime(z):
    return z > 0

Başlarken: Python ile Basit Bir YSA #

Aşağıdaki kod listesi, MNIST veri kümesinden el yazısı rakamları sınıflandırmaya çalışır. Rakamlar şöyle görünür:

Kod, bu rakamları önceden işleyerek her bir görüntüyü 0’lar ve 1’lerden oluşan 2B bir diziye dönüştürür ve ardından bu verileri %97’ye varan doğrulukla (50 dönem) bir sinir ağını eğitmek için kullanır.

"""
Deep Neural Net 

(Name: Classic Feedforward)

"""

import numpy as np
import pickle, json
import sklearn.datasets
import random
import time
import os


def sigmoid(z):
    return 1.0 / (1.0 + np.exp(-z))

def sigmoid_prime(z):
    return sigmoid(z) * (1 - sigmoid(z))

def relU(z):
    return np.maximum(z, 0, z)
    
def relU_prime(z):
    return z * (z <= 0)
    
def tanh(z):
    return np.tanh(z)

def tanh_prime(z):
    return 1 - (tanh(z) ** 2)

def transform_target(y):
    t = np.zeros((10, 1))
    t[int(y)] = 1.0
    return t

"""--------------------------------------------------------------------------------"""

class NeuralNet:

    def __init__(self, layers, learning_rate=0.05, reg_lambda=0.01):
        self.num_layers = len(layers)
        self.layers = layers
        self.biases = [np.zeros((y, 1)) for y in layers[1:]]
        self.weights = [np.random.normal(loc=0.0, scale=0.1, size=(y, x)) for x, y in zip(layers[:-1], layers[1:])]
        self.learning_rate = learning_rate
        self.reg_lambda = reg_lambda
        self.nonlinearity = relU
        self.nonlinearity_prime = relU_prime

    def __feedforward(self, x):
        """ Returns softmax probabilities for the output layer """
        for w, b in zip(self.weights, self.biases):
            x = self.nonlinearity(np.dot(w, np.reshape(x, (len(x), 1))) + b)

        return np.exp(x) / np.sum(np.exp(x))

    def __backpropagation(self, x, y):
        """
        :param x: input
        :param y: target
        """
        weight_gradients = [np.zeros(w.shape) for w in self.weights]
        bias_gradients = [np.zeros(b.shape) for b in self.biases]

        # forward pass
        activation = x
        hidden_activations = [np.reshape(x, (len(x), 1))]
        z_list = []

        for w, b in zip(self.weights, self.biases):    
            z = np.dot(w, np.reshape(activation, (len(activation), 1))) + b
            z_list.append(z)
            activation = self.nonlinearity(z)
            hidden_activations.append(activation)

        t = hidden_activations[-1] 
        hidden_activations[-1] = np.exp(t) / np.sum(np.exp(t))

        # backward pass
        delta = (hidden_activations[-1] - y) * (z_list[-1] > 0)
        weight_gradients[-1] = np.dot(delta, hidden_activations[-2].T)
        bias_gradients[-1] = delta

        for l in range(2, self.num_layers):
            z = z_list[-l]
            delta = np.dot(self.weights[-l + 1].T, delta) * (z > 0)
            weight_gradients[-l] = np.dot(delta, hidden_activations[-l - 1].T)
            bias_gradients[-l] = delta

        return (weight_gradients, bias_gradients)

    def __update_params(self, weight_gradients, bias_gradients):
        for i in xrange(len(self.weights)):
            self.weights[i] += -self.learning_rate * weight_gradients[i]
            self.biases[i] += -self.learning_rate * bias_gradients[i]

    def train(self, training_data, validation_data=None, epochs=10):
        bias_gradients = None
        for i in xrange(epochs):
            random.shuffle(training_data)
            inputs = [data[0] for data in training_data]
            targets = [data[1] for data in training_data]

            for j in xrange(len(inputs)):
                (weight_gradients, bias_gradients) = self.__backpropagation(inputs[j], targets[j])
                self.__update_params(weight_gradients, bias_gradients)

            if validation_data: 
                random.shuffle(validation_data)
                inputs = [data[0] for data in validation_data]
                targets = [data[1] for data in validation_data]

                for j in xrange(len(inputs)):
                    (weight_gradients, bias_gradients) = self.__backpropagation(inputs[j], targets[j])
                    self.__update_params(weight_gradients, bias_gradients)

            print("{} epoch(s) done".format(i + 1))

        print("Training done.")

    def test(self, test_data):
        test_results = [(np.argmax(self.__feedforward(x[0])), np.argmax(x[1])) for x in test_data]
        return float(sum([int(x == y) for (x, y) in test_results])) / len(test_data) * 100

    def dump(self, file):
        pickle.dump(self, open(file, "wb"))

"""--------------------------------------------------------------------------------"""

if __name__ == "__main__":
    total = 5000
    training = int(total * 0.7)
    val = int(total * 0.15)
    test = int(total * 0.15)

    mnist = sklearn.datasets.fetch_mldata('MNIST original', data_home='./data')

    data = zip(mnist.data, mnist.target)
    random.shuffle(data)
    data = data[:total]
    data = [(x[0].astype(bool).astype(int), transform_target(x[1])) for x in data]

    train_data = data[:training]
    val_data = data[training:training+val]
    test_data = data[training+val:]

    print "Data fetched"

    NN = NeuralNet([784, 32, 10]) # defining an ANN with 1 input layer (size 784 = size of the image flattened), 1 hidden layer (size 32), and 1 output layer (size 10, unit at index i will predict the probability of the image being digit i, where 0 <= i <= 9)  

    NN.train(train_data, val_data, epochs=5)

    print "Network trained"

    print "Accuracy:", str(NN.test(test_data)) + "%"

Bu, kendi kendine yeten bir kod örneğidir ve başka herhangi bir değişiklik yapılmadan çalıştırılabilir. Python sürümünüz için “numpy” ve “scikit” öğrenmenin kurulu olduğundan emin olun.

Geri Yayılım - Sinir Ağlarının Kalbi #

Geri yayılımın amacı, ağırlıkları optimize etmektir, böylece sinir ağı, keyfi girdileri çıktılara nasıl doğru bir şekilde eşleyeceğini öğrenebilir.

Her katmanın kendi ağırlıkları vardır ve bu ağırlıklar, verilen girdiyi doğru bir şekilde tahmin edebilmek için ayarlanmalıdır.

Geri yayılımın yüksek düzeyde bir genel görünümü aşağıdaki gibidir:

  1. İleri geçiş - girdi bir miktar çıktıya dönüştürülür. Her katmanda, aktivasyon, girdi ve ağırlıklar arasında bir nokta çarpım ile hesaplanır, ardından elde edilen sonuç, sapma ile toplanır. Son olarak, bu değer, bir sonraki katmanın girdisi olacak olan katmanın aktivasyonunu elde etmek için bir aktivasyon fonksiyonundan geçirilir.
  2. Son katmanda çıktı, o girdiye karşılık gelen gerçek etiketle karşılaştırılır ve hata hesaplanır. Genellikle, ortalama kare hatasıdır.
  3. Geriye geçiş - 2. adımda hesaplanan hata iç katmanlara geri yayılır ve tüm katmanların ağırlıkları bu hatayı hesaba katacak şekilde ayarlanır.

1. Ağırlıkları Başlatma #

Ağırlık başlatmanın basitleştirilmiş bir örneği aşağıda gösterilmiştir:

layers = [784, 64, 10] 
weights = np.array([(np.random.randn(y, x) * np.sqrt(2.0 / (x + y))) for x, y in zip(layers[:-1], layers[1:])])
biases = np.array([np.zeros((y, 1)) for y in layers[1:]])

  • Gizli katman 1, boyut ağırlığına [64, 784] ve boyut 64.

  • Çıktı katmanı, boyut ağırlığına [10, 64] ve boyut önyargısına sahiptir

Yukarıdaki kodda ağırlıkları başlatırken neler olduğunu merak ediyor olabilirsiniz. Buna Xavier başlatma denir ve ağırlık matrislerinizi rastgele başlatmaktan daha iyi bir adımdır. Evet, başlatma önemli. Başlatma işleminize bağlı olarak, gradyan inişi sırasında daha iyi bir yerel minimum bulabilirsiniz (geri yayılım, gradyan inişinin yüceltilmiş bir versiyonudur).

2. İleri Geçiş #

activation = x
hidden_activations = [np.reshape(x, (len(x), 1))]
z_list = []

for w, b in zip(self.weights, self.biases):    
    z = np.dot(w, np.reshape(activation, (len(activation), 1))) + b
    z_list.append(z)
    activation = relu(z)
    hidden_activations.append(activation)

t = hidden_activations[-1] 
hidden_activations[-1] = np.exp(t) / np.sum(np.exp(t))

Bu kod, yukarıda açıklanan dönüşümü gerçekleştirir. “hidden_activations[-1]” softmax olasılıklarını içerir - toplamı 1 olan tüm sınıfların tahminlerini içerir.

3. Geri Geçiş #

weight_gradients = [np.zeros(w.shape) for w in self.weights]
bias_gradients = [np.zeros(b.shape) for b in self.biases]

delta = (hidden_activations[-1] - y) * (z_list[-1] > 0) # relu derivative
weight_gradients[-1] = np.dot(delta, hidden_activations[-2].T)
bias_gradients[-1] = delta

for l in range(2, self.num_layers):
    z = z_list[-l]
    delta = np.dot(self.weights[-l + 1].T, delta) * (z > 0) # relu derivative
    weight_gradients[-l] = np.dot(delta, hidden_activations[-l - 1].T)
    bias_gradients[-l] = delta

İlk 2 satır degradeleri başlatır. Bu gradyanlar hesaplanır ve daha sonra ağırlıkları ve sapmaları güncellemek için kullanılacaktır.

Sonraki 3 satır, tahmini hedeften çıkararak hatayı hesaplar. Hata daha sonra iç katmanlara geri yayılır.

Şimdi, döngünün çalışmasını dikkatlice izleyin. 2. ve 3. satırlar, hatayı “katman[i]“den “katman[i - 1]“e dönüştürür. Anlamak için çarpılmakta olan matrislerin şekillerini takip edin.

4. Ağırlıklar/Parametre Güncellemesi #

for i in xrange(len(self.weights)):
    self.weights[i] += -self.learning_rate * weight_gradients[i]
    self.biases[i] += -self.learning_rate * bias_gradients[i]

“self.learning_rate”, ağın öğrendiği hızı belirtir. Çok hızlı öğrenmesini istemezsiniz çünkü yakınsamayabilir. İyi bir minimum bulmak için yumuşak bir iniş tercih edilir. Genellikle “0.01” ile “0.1” arasındaki oranlar iyi olarak kabul edilir.

Softmax İşlevi #

Softmax regresyon (veya çok terimli lojistik regresyon), birden çok sınıfı işlemek istediğimiz duruma lojistik regresyonun genelleştirilmesidir. İkili olmayan sınıflandırma uygulamak istediğimiz sinir ağları için özellikle yararlıdır. Bu durumda basit lojistik regresyon yeterli değildir. Softmax’ın bize verdiği tüm etiketler arasında bir olasılık dağılımına ihtiyacımız var.

Softmax aşağıdaki formülle hesaplanır:

______________Nereye sığar? ______________________________ #

Bir vektöre softmax işlevini “numpy” ile uygulayarak normalleştirmek için şunu kullanın:

np.exp(x) / np.sum(np.exp(x))

Burada “x”, YSA’nın son katmanından etkinleştirmedir.
