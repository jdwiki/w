Visual Studio 2010’da Yeni Bir Projeye Başlamak #

Visual Studio 2010’da yeni bir proje başlatmak için:

  • Dosya Menüsü’ne tıklayın
  • Proje Seç

Ardından, listelenmiş proje türlerimiz ve mevcut dillerimiz var:

  • En soldaki panelden mevcut Dilleri seçin [Aşağıda Visual C#, VB, F# vb. gibi dillerimiz var]
  • Kurulan şablonlardan dilleri seçtiğimizde, listelenmiş proje/uygulama türlerimiz var.
  • Proje türlerini seçin, projenize bir isim verin ve tamam’a tıklayın.

Proje türlerinin bazı yaygın örnekleri:

Windows Forms Uygulamaları :

Windows Forms Kullanıcı arabirimi ile uygulama oluşturma projesi.

WPF Uygulamaları :

Windows Presentation Foundation İstemci uygulaması.

Konsol Uygulamaları :

Komut satırı uygulamaları. Grafiksel Kullanıcı Arayüzü (GUI) yok.

ASP.NET:

Web kullanıcı arayüzü uygulamaları, ör. internet uygulamaları

Silverlight Uygulamaları:

Silverlight kullanan zengin internet uygulamaları

Mevcut Bir Projeyi Açma #

  • Visual Studio’yu açın ve Dosya Menüsü -> Aç -> Proje‘yi seçin

  • Ardından, proje klasöründen Çözüm Dosyasını seçin

  • Proje dosyaları Çözüm Gezgini’nde gösterilecektir.

Dosya Komutları

  • Dosya.Yeni Proje :

CTRL + SHIFT + N

Yeni Proje iletişim kutusunu görüntüler.

  • Dosya.OpenProject:

CTRL + SHIFT + O

Mevcut projelerin oluşturulabileceği Projeyi Aç iletişim kutusunu görüntüler. çözeltiye eklendi.

  • Proje.Sınıf Ekle:

SHIFT + ALT + C

Yeni Öğe Ekle iletişim kutusunu görüntüler ve Sınıf şablonunu şu şekilde seçer: varsayılan.

  • Proje.AddExistingItem :

SHIFT + ALT + A

Mevcut dosyaların eklenebileceği Varolan Öğe Ekle iletişim kutusunu görüntüler. mevcut projeye eklenebilir.

  • Proje.Yeni Öğe Ekle:

CTRL + SHIFT + A

CTRL + K, D veya CTRL + E, D

CTRL + H

Bul ve Değiştir sekmesinin Hızlı Değiştir sekmesinde değiştirme seçeneklerini görüntüler. Değiştir iletişim kutusu.

Tüm etkin aramaların bir listesini görüntüleyen Çağrı Yığını penceresini görüntüler. geçerli yürütme iş parçacığı için yöntemler veya yığın çerçeveleri.
