Modeller doğru şekilde ilişkiliyse, EntityFramework kullanarak ilgili verileri kolayca yükleyebilirsiniz. Seçebileceğiniz üç seçeneğiniz var: yavaş yükleme, istekli yükleme ve açık yükleme.

Örneklerde kullanılan modeller:

public class Company
{
    public int Id { get; set; }
    public string FullName { get; set; }
    public string ShortName { get; set; }

    // Navigation properties
    public virtual Person Founder { get; set; }
    public virtual ICollection<Address> Addresses { get; set; }
}

public class Address 
{        
    public int Id { get; set; }
    public int CompanyId { get; set; }
    public int CountryId { get; set; }
    public int CityId { get; set; }
    public string Street { get; set; }

    // Navigation properties
    public virtual Company Company { get; set; }
    public virtual Country Country { get; set; }
    public virtual City City { get; set; }
}

İstekli yükleme #

Eager yükleme, gerekli tüm varlıklarınızı bir kerede yüklemenizi sağlar. Tüm varlıklarınızı tek bir veritabanı çağrısında çalıştırmayı tercih ediyorsanız, o zaman Hevesli yükleme yapmanız gereken yoldur. Ayrıca birden fazla seviye yüklemenize izin verir.

İlgili varlıkları yüklemek için iki seçeneğiniz vardır, Include yönteminin kesinlikle yazılmış veya dize aşırı yüklemelerini seçebilirsiniz.

Kesinlikle yazılmış. #

// Load one company with founder and address details
int companyId = ...;
Company company = context.Companies
    .Include(m => m.Founder)
    .Include(m => m.Addresses)
    .SingleOrDefault(m => m.Id == companyId);

// Load 5 companies with address details, also retrieve country and city
// information of addresses
List<Company> companies = context.Companies
    .Include(m => m.Addresses.Select(a => a.Country));
    .Include(m => m.Addresses.Select(a => a.City))
    .Take(5).ToList();

Bu yöntem Entity Framework 4.1’den beri mevcuttur. using System.Data.Entity; referansını ayarladığınızdan emin olun.

Dize aşırı yüklenmesi. #

// Load one company with founder and address details
int companyId = ...;
Company company = context.Companies
    .Include("Founder")
    .Include("Addresses")
    .SingleOrDefault(m => m.Id == companyId);

// Load 5 companies with address details, also retrieve country and city
// information of addresses
List<Company> companies = context.Companies
    .Include("Addresses.Country");
    .Include("Addresses.City"))
    .Take(5).ToList();

Açık yükleme #

Tembel yükleme özelliğini kapattıktan sonra, girişler için açıkça Load yöntemini çağırarak varlıkları tembelce yükleyebilirsiniz. Referans, tek gezinme özelliklerini yüklemek için kullanılırken, koleksiyonları almak için Koleksiyon kullanılır.

Company company = context.Companies.FirstOrDefault();
// Load founder
context.Entry(company).Reference(m => m.Founder).Load();
// Load addresses
context.Entry(company).Collection(m => m.Addresses).Load();

Hevesli yükleme üzerinde olduğu gibi, varlıkları adlarına göre yüklemek için yukarıdaki yöntemlerin aşırı yüklemelerini kullanabilirsiniz:

Company company = context.Companies.FirstOrDefault();
// Load founder
context.Entry(company).Reference("Founder").Load();
// Load addresses
context.Entry(company).Collection("Addresses").Load();

İlgili varlıkları filtreleyin. #

Query yöntemini kullanarak yüklenen ilgili varlıkları filtreleyebiliriz:

Company company = context.Companies.FirstOrDefault();
// Load addresses which are in Baku
context.Entry(company)
    .Collection(m => m.Addresses)
    .Query()
    .Where(a => a.City.Name == "Baku")
    .Load();

Yavaş yüklenme #

Tembel yükleme varsayılan olarak etkindir. Tembel yükleme, türetilmiş proxy sınıfları yaratılarak ve sanal gezinme özelliklerini geçersiz kılarak elde edilir. Tembel yükleme, özelliğe ilk kez erişildiğinde gerçekleşir.

int companyId = ...;
Company company = context.Companies
    .First(m => m.Id == companyId);
Person founder = company.Founder; // Founder is loaded
foreach (Address address in company.Addresses)
{
    // Address details are loaded one by one.        
}

Belirli gezinme özellikleri için Tembel yüklemeyi kapatmak için, sanal anahtar kelimeyi özellik bildiriminden kaldırmanız yeterlidir:

public Person Founder { get; set; } // "virtual" keyword has been removed

Geç yüklemeyi tamamen kapatmak istiyorsanız, örneğin Bağlam yapıcısı: Yapılandırma’yı değiştirmeniz gerekir.

public class MyContext : DbContext        
{
    public MyContext(): base("Name=ConnectionString")
    {
        this.Configuration.LazyLoadingEnabled = false;
    }
}

Not: Serileştirme kullanıyorsanız lütfen tembel yüklemeyi *kapatmayı unutmayın. Serileştiriciler her özelliğe eriştiği için hepsini veritabanından yükleyeceksiniz. Ek olarak, gezinme özellikleri arasında döngüye girebilirsiniz.

Projeksiyon Sorguları #

Denormalize edilmiş bir türde ilgili verilere ihtiyaç duyulursa veya ör. yalnızca bir sütun alt kümesi, projeksiyon sorgularını kullanabilir. Fazladan bir tür kullanmak için bir neden yoksa, değerleri bir anonim tür olarak birleştirme olasılığı vardır.

var dbContext = new MyDbContext();
var denormalizedType = from company in dbContext.Company
                       where company.Name == "MyFavoriteCompany"
                       join founder in dbContext.Founder
                       on company.FounderId equals founder.Id
                       select new 
                       {
                           CompanyName = company.Name,
                           CompanyId = company.Id,
                           FounderName = founder.Name,
                           FounderId = founder.Id
                       };

Veya sorgu sözdizimi ile:

var dbContext = new MyDbContext();
var denormalizedType = dbContext.Company
                       .Join(dbContext.Founder, 
                                    c => c.FounderId,
                                    f => f.Id ,
                                    (c, f) => new 
                                    {
                                        CompanyName = c.Name,
                                        CompanyId = c.Id,
                                        FounderName = f.Name,
                                        FounderId = f.Id
                                    })
                       .Select(cf => cf);
