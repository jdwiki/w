Nasıl etkinleştirilir/devre dışı bırakılır #

Belge düzeyi

Varsayılan olarak açıktır, ancak kod dosyasının en üstüne Option Infer On|Off koyarak ayarlayabilirsiniz. Seçenek tüm belgeye uygulanacaktır.

Proje seviyesi

Visual Studio’daki menüden açıp kapatabilirsiniz:

Proje > [Proje] Özellikleri > Derleme Sekmesi > Seçenek çıkarımı

Açılır menüden “Açık|Kapalı"yı seçin. Seçenek tüm belgeye uygulanacaktır.

Tüm yeni projeler

Aşağıdakileri seçerek tüm yeni Projeler için varsayılan olarak Açık duruma getirebilirsiniz:

Araçlar > Seçenekler > Projeler ve Çözümler > VB varsayılanları > Option Infer

Açılır menüden “Açık|Kapalı"yı seçin.

Bu ne? #

Değişkenleri bildirirken yerel tür çıkarımının kullanılmasını sağlar.

Tür çıkarımı nedir?

Açıkça bir veri türü belirtmeden yerel değişkenleri bildirebilirsiniz. Derleyici, bir değişkenin veri türünü, başlatma ifadesinin türünden çıkarır.

Seçenek Çıkarımı Açık:

Dim aString  = "1234" '--> Will be treated as String by the compiler
Dim aNumber  = 4711   '--> Will be treated as Integer by the compiler

vs. açık tür bildirimi:

'State a type explicitly
Dim aString as String = "1234"
Dim aNumber as Integer = 4711

Seçenek Çıkarım Kapalı: ‘Option Infer Off’ ile derleyici davranışı ‘Option Strict’ ayarına bağlıdır bu zaten burada belgelenmiştir.

  • Seçenek Çıkarım Kapalı - Seçenek Kesin Kapalı Açık tip bildirimleri olmayan tüm değişkenler ‘Object’ olarak bildirilir.

    Dim aString  = "1234" '--> Will be treated as Object by the compiler

  • Seçenek Çıkarım Kapalı - Seçenek Kesin Açık The compiler won´t let you declare a variable without an explicit type.

     'Dim aString  = "1234" '--> Will not compile due to missing type in declaration

Tür çıkarımı ne zaman kullanılır #

Temel olarak, mümkün olduğunda tür çıkarımını kullanabilirsiniz. Ancak, ‘Option Infer Off’ ve ‘Option Strict Off’u birleştirirken dikkatli olun, çünkü bu istenmeyen çalışma zamanı davranışına yol açabilir:

Dim someVar = 5
someVar.GetType.ToString() '--> System.Int32
someVar = "abc"
someVar.GetType.ToString() '--> System.String

Anonim Tür Anonim türler yalnızca ‘Option Infer On’ ile bildirilebilir. Genellikle LINQ ile uğraşırken kullanılırlar:

Dim countryCodes = New List(Of String)
countryCodes.Add("en-US")
countryCodes.Add("en-GB")
countryCodes.Add("de-DE")
countryCodes.Add("de-AT")

Dim q = From code In countryCodes
        Let split = code.Split("-"c)
        Select New With {.Language = split(0), .Country = split(1)}
  • Seçenek Çıkarımı Açık Derleyici anonim türü tanıyacaktır:
  • Seçenek Çıkarım Kapalı Derleyici bir hata verecektir (Option Strict On ile) veya “q"yi “nesne” türü olarak kabul eder (“Seçenek Kesin Kapalı” ile). Her iki durum da anonim türü kullanamayacağınız sonucunu üretecektir.

Çiftler/Ondalık Sayılar Ondalık basamaklı sayısal değişkenler varsayılan olarak “Çift” olarak çıkarılacaktır:

Dim aNumber = 44.11 '--> Will be treated as type `Double` by the compiler

“Decimal” gibi başka bir tür isteniyorsa, değişkeni başlatan değerin işaretlenmesi gerekir:

Dim mDecimal = 47.11D '--> Will be treated as type `Decimal` by the compiler
