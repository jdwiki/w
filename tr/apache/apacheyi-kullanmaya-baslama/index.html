Kurulum veya Kurulum #

Apache’nin kurulması veya yüklenmesiyle ilgili ayrıntılı talimatlar.

Ubuntu Kurulumu #

sudo apt-get install apache2

Windows Kurulumu #

WAMP yığınına göz atın. WAMP, Windows, Apache, MySQL, PhpMyAdmin’in kısaltmasıdır.

CentOS Kurulumu #

Apache 2.2, CentOS6 ile birlikte gelirken, 2.4, herhangi bir işletim sistemine yüklemek için CentOS7 ile birlikte gelir.

yum -y install httpd

macOS Kurulumu #

macOS, Apache önceden yüklenmiş olarak gelir, ancak Apache’yi Homebrew aracılığıyla yükleyebilir

Zaten çalışan yerleşik Apache’niz varsa, önce kapatılması ve otomatik olarak yüklenen tüm komut dosyalarının kaldırılması gerekir.

$ sudo apachectl stop
$ sudo launchctl unload -w /System/Library/LaunchDaemons/org.apache.httpd.plist 2>/dev/null
$ brew install httpd24 --with-privileged-ports --with-http2

[Ubuntu] Basit Merhaba Dünya Örneği #

Bu örnek, Hello World HTML sayfası sunan bir arka uç oluşturma konusunda size rehberlik edecektir.

Yükleme Gereksinimleri #

Bu adım için sipariş önemlidir!

  • `sudo apt-get apache2’yi kurun'

HTML’yi ayarlama #

Apache dosyaları /var/www/html/ içinde yaşar. Hızlıca oraya gidelim. Önce kök dizininizde, cd, ardından cd /var/www/html/ olduğundan emin olun.

Bu html dizini, tüm web sitesi dosyalarınızın yaşayacağı yerdir. Hızlı bir şekilde basit bir Hello World dosyası oluşturalım.

Favori metin düzenleyicinizi kullanarak aşağıdakini yazın

<!DOCTYPE html>
<html>
<head>
    <title>Hello World!</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>

Bu dosyayı geçerli dizinde index.html olarak kaydedin ve hazırsınız!

Web Sayfanızı Ziyaret Edin #

Az önce oluşturduğunuz sayfayı ziyaret etmek için tercih ettiğiniz tarayıcıda ’localhost’a gidin. Bu işe yaramazsa, 127.0.0.1 deneyin. “Merhaba Dünya!” “h1” olarak. Sen bittin!

Sunucunun açık olduğundan emin olmak için. #

Tarayıcının sunucuya bağlanamadığına dair bir mesaj alırsanız, önce sunucunun açık olduğundan emin olun.

$ ps -aef | grep httpd

Apache çalışır durumdaysa birkaç httpd işlemi görmelisiniz.
