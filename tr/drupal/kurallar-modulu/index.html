Kurallar modülü, site yöneticilerinin program aracılığıyla veya önceden belirlenmiş olaylara yanıt olarak koşullu olarak yürütülecek eylemleri otomatikleştirmesine olanak tanıyan bir motordur.

Kurallar, bir kullanıcının oturum açması gibi bir Drupal sitesinde meydana gelen Kural Olaylarına tepki verebilir. bazı Kural Koşulları karşılanırsa koşullu olarak yürütülür.

Kaynaklar #

  • Video eğitimleri: Johan Falk, 30’dan fazla video ve ilgili videodan oluşan bir dizi “Kural Çerçevesini Öğrenin” için etkileyici bir eğitim seti oluşturarak Drupal 7 gününün başlarında harika bir iş çıkardı. “nodeone.se” adresinde barındırılan blog gönderileri (lisans = Attribution-Noncommercial-Share Alike 3.0).

Ancak nodeone.se etki alanı artık onları barındırmıyor. Kuralları Öğrenin, bu değerli blog gönderilerini (ilgili videolara ilgili bağlantılarla birlikte) kurtarma girişimidir.

  • Küçük Kurallar Kitabı, Kurallar modülü hakkında (15 sayfa) bir hızlı başlangıçtır.

Kurallar Kullanıcı Arayüzü kullanılarak gösterilen özel bir kural #

Kural Dışa Aktarma biçiminde gösterilen özel bir kural #

Kural dışa aktarma biçiminde bir Kural örneği:

{ "rules_display_userpoints_after_updating_content" : {
    "LABEL" : "Display userpoints after updating content",
    "PLUGIN" : "reaction rule",
    "OWNER" : "rules",
    "REQUIRES" : [ "userpoints_rules", "rules", "rules_conditional" ],
    "ON" : { "node_update" : [] },
    "DO" : [
      { "userpoints_rules_get_current_points" : {
          "USING" : { "user" : [ "site:current-user" ], "tid" : "all" },
          "PROVIDE" : { "loaded_points" : { "total_points" : "Number of points in all categories together" } }
        }
      },
      { "drupal_message" : { "message" : "You now have [total-points:value] points" } },
      { "CONDITIONAL" : [
          {
            "IF" : { "NOT data_is" : { "data" : [ "total-points" ], "op" : "\u003C", "value" : "20" } },
            "DO" : [
              { "drupal_message" : { "message" : "You have sufficient points (you still have [total-points:value] ...)." } }
            ]
          },
          { "ELSE" : [
              { "drupal_message" : { "message" : "You DO NOT have sufficient points (you only have [total-points:value] ...)." } }
            ]
          }
        ]
      }
    ]
  }
}

İlk Kural Eylemi olarak (Kural Koşulu değil!) bir kullanıcının mevcut kullanıcı puanı miktarını alır. Miktar en az 20 ise, “Yeterli puanınız var …” ile başlayan bir mesaj, aksi takdirde mesaj “Yeterli puanınız YOK…” ile başlar.

Kurallarla alan toplama öğelerini işleme #

Kurallar ile Field collection öğeyi işlemek gerçekten eğlenceli! Bu Kurala bir göz atın (Kurallar dışa aktarma biçiminde):

{ "rules_calculate_sum_of_prices_in_all_field_collection_items" : {
    "LABEL" : "Calculate sum of prices in all field collection items",
    "PLUGIN" : "reaction rule",
    "OWNER" : "rules",
    "REQUIRES" : [ "rules" ],
    "ON" : { "node_view--article" : { "bundle" : "article" } },
    "IF" : [
      { "entity_has_field" : { "entity" : [ "node" ], "field" : "field_article_details" } }
    ],
    "DO" : [
      { "drupal_message" : { "message" : "\u003Cstrong\u003EDrupal calculator\u003C\/strong\u003E started ..." } },
      { "variable_add" : {
          "USING" : { "type" : "decimal", "value" : "0" },
          "PROVIDE" : { "variable_added" : { "total_price" : "Price total" } }
        }
      },
      { "LOOP" : {
          "USING" : { "list" : [ "node:field-article-details" ] },
          "ITEM" : { "article_details_item" : "Article details item" },
          "DO" : [
            { "data_calc" : {
                "USING" : {
                  "input_1" : [ "total-price" ],
                  "op" : "+",
                  "input_2" : [ "article-details-item:field-price" ]
                },
                "PROVIDE" : { "result" : { "calculation_result" : "Calculation result" } }
              }
            },
            { "data_set" : { "data" : [ "total-price" ], "value" : [ "calculation-result" ] } },
            { "drupal_message" : { "message" : "After adding a price of \u003Cstrong\u003E[article-details-item:field-price]\u003C\/strong\u003E for field collection item with id \u003Cstrong\u003E[article-details-item:item-id]\u003C\/strong\u003E, subtotal is \u003Cstrong\u003E[calculation-result:value]\u003C\/strong\u003E." } }
          ]
        }
      },
      { "drupal_message" : { "message" : "The \u003Cstrong\u003ETotal price\u003C\/strong\u003E for all prices included as field collection items is \u003Cstrong\u003E[total-price:value]\u003C\/strong\u003E." } },
      { "drupal_message" : { "message" : "\u003Cstrong\u003EDrupal calculator\u003C\/strong\u003E ended ..." } }
    ]
  }
}

Bu kuralla ilgili daha fazla ayrıntı aşağıdadır …

Kurallar Etkinliği: #

İçerik görüntülenir (Makale türünden), “makale” içerik türünün makine adını uygun olana uyarlayın (veya uygun olan başka bir Kural Olayı kullanın).

Kurallar Koşul: #

Varlık alanı var, varlık “düğüm” ve alan toplama alanımın makine adı “field_article_details” (bu makine adını uygun olan her şeye uyarlayın, ancak alan toplama alanının kendisini kullandığınızdan emin olun).

Kural İşlemleri: #

Uyanın, sihrin (eğlencenin?) olacağı yer burası… Bunlar ilgili Kural Eylemleri:

  1. Sitede bir mesaj göster, şunun gibi bir mesajla:

Drupal hesap makinesi başladı…

  1. Değişken ekle, oysa ’toplam_fiyat’, ondalık (2 basamaklı), başlangıç ​​değeri 0 olan bir değişkendir.

  2. Bir döngü ekleyin, alan toplama alanımın her bir öğesi üzerinde (‘field_article_details’ makine adıyla) yineleme yapmak ve her yineleme için şu Kural Eylemlerini gerçekleştirmek:

  • ’toplam_fiyat’ (yukarıdaki 2. Kural Eyleminde tanımlanmıştır) ve ‘makale-ayrıntıları-öğesi:alan-fiyatı’ (bu, içindeki alanın makine adıdır) toplamını hesaplayan bir değer hesaplayın. fiyatları içeren, 2 basamaklı ondalık sayı) ve sonucu (toplamı) “hesaplama_sonucu” değişkeninde saklayan alan koleksiyonu.

  • Bir veri değeri ayarlayın, bu, “hesaplama_sonucu” değişkeninde depolanan değeri “toplam_fiyat"ta (yukarıdaki Kural Eylemi 2’de tanımlanmıştır) basitçe kopyalar. Açıklama: emin değilim (test edilmedi), ancak belki bu “hesaplama_sonucu” değişkeni doğrudan “toplam_fiyat” (önceki işlemde) ile değiştirilebilir, böylece bu işleme ihtiyacınız olmaz.

  • Sitede bir mesaj göster, şöyle bir mesajla:

    After adding a price of 3.40 for field collection item with id 3, subtotal is 15.00.

  1. Aşağıdaki gibi bir mesajla Sitede bir mesaj göster:

Alan toplama kalemleri olarak dahil edilen tüm fiyatlar için toplam fiyat 26,23’tür.

  1. Sitede bir mesaj göster, şunun gibi bir mesajla:

Drupal hesap makinesi sona erdi…

Açıkçası, bu kural daha çok bir prototip. Olması gerektiği gibi çalıştığına ikna olduktan sonra, Sitede bir mesaj göster ile tüm Kural Eylemlerini kaldırmanız yeterlidir. Böylece, yalnızca 2. ve 3. madde (son alt madde işareti olmadan) Kural Eylemleri olarak bırakılır.

Gösteri zamanı … #
