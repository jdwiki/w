Uygulama oluşturma #

Grails uygulaması oluşturmak için ‘grails create-app’ komutunu kullanın. Aşağıdaki komut, geçerli dizinde “myapp” adlı bir Grails uygulaması oluşturur:

grails create-app fancy-app

Çalıştırmak, yeni oluşturulan uygulama dizinini ziyaret etmek kadar basittir:

cd fancy-app

ve daha sonra

grails run-app 
// in order to run the app on a different port, e.g. 8888, use this instead
grails run-app -port 8888
// in order to run the app with a secure communication
grails run-app -https

Grails Kurulumu #

Not: GRAILS, Grails’i kurmadan önce sisteminizde yüklü bir Java JDK (JRE çalışma zamanı ortamı yeterli değildir) gerektirir. Lütfen [JDK nasıl kurulur][1] bölümüne bakın. Bu yazı itibariyle, en son JDK’yı yüklemeniz önerilir.

Mac OSX, Linux, Cygwin, Solaris ve FreeBSD için:

Grails sürümlerini yönetmenin en basit yolu sdkman kullanmaktır. sdkman kuruluysa, kullanarak Grails’in herhangi bir sürümünü yükleyebilirsiniz.

sdk install grails [version]

Bu, bunu doğru yapmak için tüm adımları halledecektir. ‘Sürüm’ü atlarsanız, en son grails sürümü yüklenecektir. sdkman kullanımı hakkında daha fazla bilgi için [sdkman kullanım sayfasına[2] bakın.

Linux için:

    GRAILS_HOME=$HOME/bin/grails/current
    # abbreviating it using "..." for brevity
    PATH=$GRAILS_HOME/bin:$JAVA_HOME/bin: ... :$PATH

Pencereler için:

  1. [Oracle][3]’ten bir Java JDK indirin ve Windows makinenize kurun. Kurulum klasörünü not alın.
  2. Grails’in bir sürümünü İndirilenler sayfasından manuel olarak indirin.
  3. Grails dosyasını istediğiniz yere çıkarın.
  4. Önemli: 2 yeni ortam değişkeni JAVA_HOME ve GRAILS_HOME kurmalısınız (Windows 10 için *Denetim Masası \ Sistem ve Güvenlik \ Sistem \ Gelişmiş Sistem ayarları \ Gelişmiş sekmesi \ Ortam Değişkenleri altında bulunur) * , çıkarılan dizinlere işaret ederek, ör.

İsim: JAVA_HOME

Değer: C:\Programs\Java\jdk1.8.0_31

İsim: GRAILS_HOME

Değer: c:\grails\grails-3.2.4

  1. Önemli: Windows PATH değişkenini hem JAVA_HOME hem de GRAILS_HOME içerecek şekilde genişletmelisiniz. Yol değişkeni ayrıca kontrol panelinde bulunur (bkz. 4), örn. sonuna şunu ekleyin:

;C:\Programs\Java\jdk1.8.0_31\bin;c:\grails\grails-3.2.4;c:\grails\grails-3.2.4\bin

  1. Kurulumunuzun doğru olduğunu doğrulamak için bir Komut İstemi açın ve GRAILS -VERSION yazın. Şunun gibi bir şey almalısın:
| Grails Version: 3.2.4
| Groovy Version: 2.4.6
| JVM Version: 1.8.0_65

[1]: https://www.wikiod.com/tr/java/java-dilini-kullanmaya-baslama#Windows’a yükledikten sonra %PATH% ve %JAVA_HOME% ayarı [2]: http://sdkman.io/usage.html [3]: http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

Bir Uygulamayı Test Etme #

Grails’deki create-* komutları, sizin için src/test/groovy dizini içinde otomatik olarak birim veya entegrasyon testleri oluşturur. Bu testleri, Birim ve entegrasyon testleri bölümünde bulabileceğiniz geçerli test mantığıyla doldurmak elbette size kalmıştır.

Testleri yürütmek için test-app komutunu aşağıdaki gibi çalıştırın:

grails test-app

Model Oluşturma #

Grails’deki bir model (bkz: Model-View-Controller modeli), Etki Alanı Sınıfı ile temsil edilir. Etki alanı sınıfları, bilgilerin kâselerde hem kalıcılığını hem de sunumunu tanımlayabilir. Etki alanı sınıfları ayrıca doğrulamalar içerebilir.

Grails uygulamanızda bir araba filosunu yönetmek için, filonuzdaki çeşitli arabaları tanımlamak, depolamak ve temsil etmek üzere bir etki alanı sınıfı tanımlayabilirsiniz.

Bir etki alanı sınıfı için saplama oluşturmak için uygulama klasörünüzde aşağıdaki komutu yürütün:

grails create-domain-class org.fleetmanager.Car

Ardından, oluşturulan car.groovy dosyasını açın ve etki alanı sınıfınızı aşağıdaki gibi düzenleyin:

package org.fleetmanager

class Car {
   String      manufacturer
   String      model
   String      color
   Integer     year
   Date        acquisitionDate
   Boolean     isElectric
}

Son olarak, aşağıdaki Grails komutunu kullanarak araç etki alanınız için bir denetleyici ve bir görünüm oluşturun:

grails generate-all org.fleetmanager.Car

Artık uygulamalarınızı çalıştırabilir, araç kumandasını seçebilir ve filonuzu yönetebilirsiniz.
