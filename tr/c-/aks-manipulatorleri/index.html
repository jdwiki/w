  • Akış manipülatörleri
  • Çıkış akışı manipülatörleri
  • Giriş akışı manipülatörleri

Manipülatörler, “operatör »” veya “operatör «” kullanarak giriş ve çıkış akışlarını kontrol etmeye yardımcı olan özel yardımcı işlevlerdir.

Hepsi #include <iomanip> ile dahil edilebilir.

Manipülatörler başka bir şekilde kullanılabilir. Örneğin:

  1. ‘os.width(n);’, ‘os « std::setw(n);’
    değerine eşittir is.width(n);, is >> std::setw(n); değerine eşittir

  2. ‘os.precision(n);’, ‘os « std::setprecision(n);’
    ‘a eşittir is.precision(n);, is >> std::setprecision(n); değerine eşittir

  3. os.setfill(c);, os << std::setfill(c); değerine eşittir

  4. str >> std::setbase(base); veya str << std::setbase(base); eşittir

str.setf(base ==  8 ? std::ios_base::oct :
            base == 10 ? std::ios_base::dec :
                base == 16 ? std::ios_base::hex :
                     std::ios_base::fmtflags(0),
         std::ios_base::basefield);

  1. os.setf(std::ios_base::flag);, os << std::flag;
    değerine eşittir is.setf(std::ios_base::flag);, is >> std::flag; değerine eşittir

    os.unsetf(std::ios_base::flag); eşittir os << std::no ## flag;
    is.unsetf(std::ios_base::flag); eşittir is >> std::no ## flag;

    (burada ## - birleştirme operatörüdür)

    sonraki bayraklar için: boolalpha, showbase, showpoint, showpos, skipws, büyük harf.

  2. std::ios_base::basefield.
    “bayrak” için: “ara”, “hex” ve “okt”:

  • os.setf(std::ios_base::flag, std::ios_base::basefield); eşittir os << std::flag;
    is.setf(std::ios_base::flag, std::ios_base::basefield); eşittir is >> std::flag;
    ( 1 )

  • str.unsetf(std::ios_base::flag, std::ios_base::basefield);, str.setf(std::ios_base::fmtflags(0), std::ios_base::basefield) değerine eşittir ;
    ( 2 )
  1. std::ios_base::adjustfield.
    “bayrak"lar için: “sol”, “sağ” ve “iç”:
  • os.setf(std::ios_base::flag, std::ios_base::adjustfield); eşittir os << std::flag;
    is.setf(std::ios_base::flag, std::ios_base::adjustfield); eşittir is >> std::flag;
    ( 1 )

  • str.unsetf(std::ios_base::flag, std::ios_base::adjustfield);, str.setf(std::ios_base::fmtflags(0), std::ios_base::adjustfield) değerine eşittir ;
    ( 2 )

( 1 ) Daha önce ayarlanmış ilgili alanın bayrağının ayarı ‘unsetf’ ile kaldırılmışsa.
( 2 ) “bayrak” ayarlanmışsa.

  1. std::ios_base::floatfield.
  • os.setf(std::ios_base::flag, std::ios_base::floatfield); eşittir os << std::flag;
    is.setf(std::ios_base::flag, std::ios_base::floatfield); eşittir is >> std::flag;

    “bayrak"lar için: “sabit” ve “bilimsel”.

  • os.setf(std::ios_base::fmtflags(0), std::ios_base::floatfield); eşittir os << std::defaultfloat;
    is.setf(std::ios_base::fmtflags(0), std::ios_base::floatfield); eşittir is >> std::defaultfloat;

  1. str.setf(std::ios_base::fmtflags(0), std::ios_base::flag);, str.unsetf(std::ios_base::flag) değerine eşittir

    “bayrak"lar için: “basefield”, “adjustfield”, “floatfield”.

  2. os.setf(mask) eşittir os << setiosflags(mask);
    is.setf(mask) eşittir is >> setiosflags(mask);

    ‘os.unsetf(mask)’, ‘os « resetiosflags(mask);’
    değerine eşittir is.unsetf(mask), is >> resetiosflags(mask); değerine eşittir

    ‘std::ios_base::fmtflags’ türündeki hemen hemen tüm ‘maskeler’ için.

Akış manipülatörleri #

std::boolalpha ve std::noboolalpha - booleanların metinsel ve sayısal gösterimi arasında geçiş yapın.

std::cout << std::boolalpha << 1;
// Output: true

std::cout << std::noboolalpha << false;
// Output: 0

bool boolValue;
std::cin >> std::boolalpha >> boolValue;
std::cout << "Value \"" << std::boolalpha << boolValue
          << "\" was parsed as " << std::noboolalpha << boolValue;
// Input: true
// Output: Value "true" was parsed as 0

std::showbase ve std::noshowbase - sayısal tabanı belirten önek kullanılıp kullanılmadığını kontrol edin.

std::dec (ondalık), std::hex (onaltılık) ve std::oct (sekizlik) - tabanı değiştirmek için kullanılır tamsayılar için.

#include <sstream>

std::cout << std::dec << 29 << ' - '
          << std::hex << 29 << ' - '
          << std::showbase << std::oct << 29 << ' - '
          << std::noshowbase << 29  '\n';
int number;
std::istringstream("3B") >> std::hex >> number;
std::cout << std::dec << 10;
// Output: 22 - 1D - 35 - 035
// 59

Varsayılan değerler std::ios_base::noshowbase ve std::ios_base::dec şeklindedir.

‘std::istringstream’ hakkında daha fazlasını görmek istiyorsanız <sstream> başlığına bakın.


std::uppercase ve std::nouppercase - kayan nokta ve onaltılık tamsayı çıktısında büyük harf karakterlerin kullanılıp kullanılmayacağını kontrol edin. Giriş akışları üzerinde hiçbir etkisi yoktur.

std::cout << std::hex << std::showbase
              << "0x2a with nouppercase: " << std::nouppercase << 0x2a << '\n'
              << "1e-10 with uppercase: " << std::uppercase << 1e-10 << '\n'
}
// Output: 0x2a with nouppercase: 0x2a
// 1e-10 with uppercase: 1E-10

Varsayılan “std::nouppercase"dir.


std::setw(n) - sonraki giriş/çıkış alanının genişliğini tam olarak n olarak değiştirir.

Bazı işlevler çağrıldığında “n” genişlik özelliği “0"a sıfırlanıyor (tam liste burada).

std::cout << "no setw:" << 51 << '\n'
          << "setw(7): " << std::setw(7) << 51 << '\n'
          << "setw(7), more output: " << 13
          << std::setw(7) << std::setfill('*') << 67 << ' ' << 94 << '\n';

char* input = "Hello, world!";
char arr[10];
std::cin >> std::setw(6) >> arr;
std::cout << "Input from \"Hello, world!\" with setw(6) gave \"" << arr << "\"\n";

// Output: 51
// setw(7):      51
// setw(7), more output: 13*****67 94

// Input: Hello, world!
// Output: Input from "Hello, world!" with setw(6) gave "Hello"

Varsayılan std::setw(0) şeklindedir.

// Output:

//   $: $3.67**********

          << "flt: " << std::setw(15) << -9.87  << '\n'

          << "usd: " << std::setw(15)
          << std::setfill(' ') << std::put_money(367, true) << '\n';

//   $: **********$3.67

std::fixed, std::scientific, std::hexfloat [C++11] ve [std::defaultfloat] 7 [C++11] - kayan noktalı giriş/çıkış için biçimlendirmeyi değiştirin.

std::fixed std::ios_base::floatfield öğesini std::ios_base::fixed olarak ayarlar,
std::scientific' - [std::ios_base::scientific][4]'e, <br> std::hexfloat-std::ios_base::fixed’e | std::ios_base::bilimsel ve <br> 'std::defaultfloat' - 'std::ios_base::fmtflags(0)'a. <br><br> [fmtflags`](http://en.cppreference.com/w/cpp/io/ios_base/fmtflags)

#include <sstream>
...

std::cout << '\n'

          << "The number 0.07 in hexfloat:   " << std::hexfloat << 0.01 << '\n'

double f;

std::showpoint ve std::noshowpoint - ondalık noktanın her zaman kayan nokta gösterimine dahil edilip edilmediğini kontrol edin. Giriş akışları üzerinde hiçbir etkisi yoktur.

std::cout << "7.0 with showpoint: " << std::showpoint << 7.0 << '\n'
          << "7.0 with noshowpoint: " << std::noshowpoint << 7.0 << '\n';

// 1.0 with noshowpoint: 7

Varsayılan ‘std::showpoint’ şeklindedir.


std::showpos ve std::noshowpos - negatif olmayan çıktıda ‘+’ işaretinin görüntülenmesini kontrol eder. Giriş akışları üzerinde hiçbir etkisi yoktur.

std::cout << "With showpos: " << std::showpos
          << 0 << ' ' << -2.718 << ' ' << 17 << '\n'
          << "Without showpos: " << std::noshowpos
          << 0 << ' ' << -2.718 << ' ' << 17 << '\n';
// Output: With showpos: +0 -2.718 +17
// Without showpos: 0 -2.718 17

“std::noshowpos” ise varsayılan.


std::unitbuf, std::nounitbuf - her işlemden sonra çıkış akışının boşaltılmasını kontrol eder. Giriş akışı üzerinde hiçbir etkisi yoktur. std::unitbuf kızarmaya neden olur.


std::setbase(base) - akışın sayısal tabanını ayarlar.

“std::setbase(8)”, “std::ios_base::basefield” öğesinin “std::ios_base::oct”,

“Temel”, “8”, “10” veya “16"dan farklıysa, o zaman “std::ios_base::basefield”, “std::ios_base::fmtflags(0)” olarak ayarlanmıştır. Ondalık çıktı ve önek bağımlı girdi anlamına gelir.

Varsayılan olarak ‘std::ios_base::basefield’ ‘std::ios_base::dec’ ve ardından varsayılan olarak ‘std::setbase(10)’ şeklindedir.

std::setiosflags(mask) ve [std::resetiosflags(mask)](http://en .cppreference.com/w/cpp/io/manip/resetiosflags) - ‘std::ios_base::fmtflags’ türündeki ‘mask’ içinde belirtilen bayrakları ayarlayın ve temizleyin.

#include <sstream>
...

in >> std::resetiosflags(std::ios_base::basefield) >> num1 >> num2;
std::cout << "Parsing \"10 010\" with autodetect gives: " << num1 << ' ' << num2 << '\n';
// Parsing "10 010" with autodetect gives: 10 8

std::cout << std::setiosflags(std::ios_base::hex |

                              std::ios_base::showbase) << 42 << '\n';

std::skipws ve std::noskipws - biçimlendirilmiş giriş işlevleri tarafından baştaki boşlukların atlanmasını kontrol eder. Çıkış akışları üzerinde hiçbir etkisi yoktur.

#include <sstream>
...

char c1, c2, c3;
std::istringstream("a b c") >> c1 >> c2 >> c3;

std::cout << "noskipws behavior:  c1 = " << c1 << "  c2 = " << c2 << "  c3 = " << c3 << '\n';

// noskipws behavior:  c1 = a  c2 =    c3 = b

Varsayılan std::ios_base::skipws şeklindedir.


std::quoted(s[, delim[, escape]]) [C++14] - ekler veya alıntılar gömülü boşluklarla alıntılanan dizeler.

s' - eklenecek veya ayıklanacak dize.<br> selim- sınırlayıcı olarak kullanılacak karakter, varsayılan olarak"`.
“escape” - kaçış karakteri olarak kullanılacak karakter, varsayılan olarak “".

#include <sstream>
...
 
std::stringstream ss;

Çıkış akışı manipülatörleri #

std::ends - çıkış akışına '\0' boş karakteri ekler. Daha resmi olarak bu manipülatörün beyanı şuna benzer:

template <class charT, class traits>
std::basic_ostream<charT, traits>& ends(std::basic_ostream<charT, traits>& os);

ve bu manipülatör,
os << std::ends; ifadesinde kullanıldığında os.put(charT()) öğesini çağırarak karakteri yerleştirir.


std::endl ve [std::flush](http://en.cppreference.com/w /cpp/io/manip/flush) her ikisi de çıkış akışını ‘out.flush()’ çağırarak ‘out’ akıtır. Hemen çıktı üretmeye neden olur. Ancak ‘std::endl’, temizlemeden önce satır sonu ‘\n’’ sembolünü ekler.

std::cout << "First line." << std::endl << "Second line. " << std::flush
          << "Still second line.";
// Output: First line.
// Second line. Still second line.

std::setfill(c) - dolgu karakterini “c” olarak değiştirir. Genellikle ‘std::setw’ ile birlikte kullanılır.

std::cout << "\nDefault fill: " << std::setw(10) << 79 << '\n'
          << "setfill('#'): " << std::setfill('#')
          << std::setw(10) << 42 << '\n';
// Output:
// Default fill:         79
// setfill('#'): ########79

std::put_money(mon[, intl]) [C++11]. “out « std::put_money(mon, intl)” ifadesinde, “mon” (“long double” veya “std::basic_string” türündeki) parasal değerini ile belirtilen karakter temsiline dönüştürür. std::money_put yerel ayarın şu anda ‘out’ içine yerleştirilmiş yönü. “intl” “true” ise uluslararası para birimi dizelerini kullanın, aksi takdirde para birimi simgelerini kullanın.

long double money = 123.45;
// or std::string money = "123.45";
 
std::cout.imbue(std::locale("en_US.utf8"));
std::cout << std::showbase << "en_US: " << std::put_money(money)
          << " or " << std::put_money(money, true) << '\n';
// Output: en_US: $1.23 or USD  1.23
 
std::cout.imbue(std::locale("ru_RU.utf8"));
std::cout << "ru_RU: " << std::put_money(money)
          << " or " << std::put_money(money, true) << '\n';
// Output: ru_RU: 1.23 руб or 1.23 RUB 
 
std::cout.imbue(std::locale("ja_JP.utf8"));
std::cout << "ja_JP: " << std::put_money(money)
          << " or " << std::put_money(money, true) << '\n';
// Output: ja_JP: ￥123 or JPY  123

std::put_time(tmb, fmt) [C++11] - biçimlendirir ve bir tarih/saat değeri verir ‘std::tm’, belirtilen ‘fmt’ biçimine göre.

“tmb” - “localtime()” veya “gmtime()“dan elde edilen “const std::tm*” takvim zaman yapısının işaretçisi.
“fmt” - dönüştürme biçimini belirten boş sonlandırılmış bir “const CharT*” dizesinin işaretçisi.

#include <ctime>
...

std::time_t t = std::time(nullptr);
std::tm tm = *std::localtime(&t);

std::cout.imbue(std::locale("ru_RU.utf8"));
std::cout << "\nru_RU: " << std::put_time(&tm, "%c %Z") << '\n';
// Possible output:
// ru_RU: Вт 04 июл 2017 15:08:35 UTC

std::ws - giriş akışında baştaki boşlukları kullanır. ‘std::skipws’den farklıdır.

#include <sstream>

std::string str;

std::get_money(mon[, uluslararası]) [C++11]. in >> std::get_money(mon, intl) ifadesinde, karakter girişini std::money_get tarafından belirtildiği gibi parasal bir değer olarak ayrıştırır. /cpp/locale/money_get) yerel ayarın şu anda “in” içine yerleştirilmiş yüzüdür ve değeri “mon” (“uzun çift” veya “std::basic_string” türünden) içinde depolar. Manipülatör, “intl” “true” ise gerekli uluslararası para birimi dizelerini bekler, aksi takdirde isteğe bağlı para birimi simgelerini bekler.

#include <sstream>
#include <locale>
...

// Output:
// "$1,234.56 2.22 USD  3.33" parsed as: 123456, 222, 333

std::get_time(tmb, fmt) [C++11] - ` içinde saklanan bir tarih/saat değerini ayrıştırır ‘fmt’ biçiminde belirtilen tmb’.

’tmb’ - sonucun depolanacağı ‘const std::tm*’ nesnesi için geçerli işaretçi.
“fmt” - dönüştürme biçimini belirten boş sonlandırılmış bir “const CharT*” dizesinin işaretçisi.

#include <sstream>

...
