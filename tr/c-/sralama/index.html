‘std::sort’ işlev ailesi, ‘algoritma’ kitaplığında bulunur.

Sıralama ve sıralama kapsayıcıları #

Standart kitaplık başlığı ‘algoritması’nda bulunan ‘std::sort’, bir çift yineleyici tarafından tanımlanan bir dizi değeri sıralamak için standart bir kitaplık algoritmasıdır. ‘std::sort’ iki değeri karşılaştırmak için kullanılan son parametre olarak bir işlev alır; sıralamayı bu şekilde belirler. ‘std::sort’un kararlı olmadığına dikkat edin.

Karşılaştırma işlevi , öğelere bir Sıkı, Zayıf Sıralama uygulamalıdır. Basit bir küçüktür (veya büyüktür) karşılaştırması yeterli olacaktır.

Rastgele erişim yineleyicileri olan bir kap, ‘std::sort’ algoritması kullanılarak sıralanabilir:

#include <vector>
#include <algorithm>

std::vector<int> MyVector = {3, 1, 2}

//Default comparison of <
std::sort(MyVector.begin(), MyVector.end());
son sürüm

‘std::sort’, yineleyicilerinin rastgele erişim yineleyicileri olmasını gerektirir. Sıra kapsayıcıları “std::list” ve “std::forward_list” (C++11 gerektirir) rastgele erişim yineleyicileri sağlamaz, bu nedenle “std::sort” ile kullanılamazlar. Ancak, kendi yineleyici türleriyle çalışan bir sıralama algoritması uygulayan “sıralama” üye işlevlerine sahiptirler.

#include <list>
#include <algorithm>

std::list<int> MyList = {3, 1, 2}

//Default comparison of <
//Whole list only.
MyList.sort();
son sürüm

Üye “sıralama” işlevleri her zaman tüm listeyi sıralar, böylece bir alt öğe aralığını sıralayamazlar. Bununla birlikte, ’list’ ve ‘ileri_liste’ hızlı birleştirme işlemlerine sahip olduğundan, sıralanacak öğeleri listeden çıkarabilir, sıralayabilir ve ardından oldukça verimli bir şekilde oldukları yere geri yerleştirebilirsiniz:

void sort_sublist(std::list<int>& mylist, std::list<int>::const_iterator start, std::list<int>::const_iterator end) {
    //extract and sort half-open sub range denoted by start and end iterator 
    std::list<int> tmp;
    tmp.splice(tmp.begin(), list, start, end);
    tmp.sort();
    //re-insert range at the point we extracted it from
    list.splice(end, tmp);
}

Belirtilen sıralamaya sahip sıralama kapsayıcıları #

Bir kaptaki değerlerde belirli operatörler zaten aşırı yüklenmişse, artan veya azalan düzende sıralamak için özel işlevlerle ‘std::sort’ kullanılabilir:

#include <vector>
#include <algorithm>
#include <functional>

std::vector<int> v = {5,1,2,4,3};

//sort in ascending order (1,2,3,4,5)
std::sort(v.begin(), v.end(), std::less<int>());

// Or just:
std::sort(v.begin(), v.end());

//sort in descending order (5,4,3,2,1)
std::sort(v.begin(), v.end(), std::greater<int>());

//Or just:
std::sort(v.rbegin(), v.rend());
son sürüm

C++14’te, karşılaştırma işlevi nesneleri için şablon argümanı sağlamamıza gerek yoktur ve bunun yerine nesnenin ne iletildiğine bağlı olarak çıkarım yapmasına izin veririz:

std::sort(v.begin(), v.end(), std::less<>());     // ascending order
std::sort(v.begin(), v.end(), std::greater<>());  // descending order
son sürüm

Dizi kapsayıcılarını aşırı yüklenmiş daha az operatöre göre sıralama #

Hiçbir sıralama işlevi iletilmezse, “std::sort”, öğe çiftlerinde “işleç<” öğesini çağırarak öğeleri sıralar; bu, bağlamsal olarak “bool"a (veya yalnızca “bool”) dönüştürülebilir bir tür döndürmesi gerekir. Temel türler (tamsayılar, kayan noktalar, işaretçiler vb.) karşılaştırma operatörlerinde zaten oluşturulmuştur.

Varsayılan “sıralama” çağrısının kullanıcı tanımlı türlerde çalışmasını sağlamak için bu operatörü aşırı yükleyebiliriz.

// Include sequence containers
#include <vector>
#include <deque>
#include <list>

// Insert sorting algorithm
#include <algorithm>    

class Base {
 public:

    // Constructor that set variable to the value of v
    Base(int v): variable(v) {
    }
    
    // Use variable to provide total order operator less
    //`this` always represents the left-hand side of the compare.
    bool operator<(const Base &b) const { 
        return this->variable < b.variable;
    }
    
    int variable;
};

int main() {
    std::vector <Base> vector;
    std::deque <Base> deque;
    std::list <Base> list;
    
    // Create 2 elements to sort
    Base a(10);
    Base b(5);
    
    // Insert them into backs of containers
    vector.push_back(a);
    vector.push_back(b);
    
    deque.push_back(a);
    deque.push_back(b);
    
    list.push_back(a);
    list.push_back(b);
    
    // Now sort data using operator<(const Base &b) function
    std::sort(vector.begin(), vector.end());
    std::sort(deque.begin(), deque.end());
    // List must be sorted differently due to its design
    list.sort();

    return 0;
}

Karşılaştırma işlevini kullanarak sıralama kapsayıcıları #

// Include sequence containers
#include <vector>
#include <deque>
#include <list>

// Insert sorting algorithm
#include <algorithm>

class Base {
 public:

    // Constructor that set variable to the value of v
    Base(int v): variable(v) {
    }
       
    int variable;
};

bool compare(const Base &a, const Base &b) {
    return a.variable < b.variable;
}

int main() {
    std::vector <Base> vector;
    std::deque <Base> deque;
    std::list <Base> list;
    
    // Create 2 elements to sort
    Base a(10);
    Base b(5);
    
    // Insert them into backs of containers
    vector.push_back(a);
    vector.push_back(b);
    
    deque.push_back(a);
    deque.push_back(b);
    
    list.push_back(a);
    list.push_back(b);
    
    // Now sort data using comparing function
    std::sort(vector.begin(), vector.end(), compare);
    std::sort(deque.begin(), deque.end(), compare);
    list.sort(compare);

    return 0;
}

Lambda ifadeleri kullanarak sıra kapsayıcılarını sıralama (C++11) #

// Include sequence containers
#include <vector>
#include <deque>
#include <list>
#include <array>
#include <forward_list>

// Include sorting algorithm
#include <algorithm>

class Base {
 public:

    // Constructor that set variable to the value of v
    Base(int v): variable(v) {
    }
    
    int variable;
};


int main() {
    // Create 2 elements to sort
    Base a(10);
    Base b(5);
    
    // We're using C++11, so let's use initializer lists to insert items.
    std::vector <Base> vector = {a, b};
    std::deque <Base> deque = {a, b};
    std::list <Base> list = {a, b};
    std::array <Base, 2> array = {a, b};
    std::forward_list<Base> flist = {a, b};
    
    // We can sort data using an inline lambda expression
    std::sort(std::begin(vector), std::end(vector),
      [](const Base &a, const Base &b) { return a.variable < b.variable;});

    // We can also pass a lambda object as the comparator
    // and reuse the lambda multiple times
    auto compare = [](const Base &a, const Base &b) {
                     return a.variable < b.variable;};
    std::sort(std::begin(deque), std::end(deque), compare);
    std::sort(std::begin(array), std::end(array), compare);
    list.sort(compare);
    flist.sort(compare);

    return 0;
}

std::map ile sıralama (artan ve azalan) #

Bu örnek, öğeleri bir harita kullanarak bir anahtarın artan düzeninde sıralar. Aşağıdaki örnekte ‘std::string’ yerine sınıf dahil herhangi bir türü kullanabilirsiniz.

#include <iostream>
#include <utility>
#include <map>

int main()
{
    std::map<double, std::string> sorted_map;
    // Sort the names of the planets according to their size
    sorted_map.insert(std::make_pair(0.3829, "Mercury"));
    sorted_map.insert(std::make_pair(0.9499, "Venus"));
    sorted_map.insert(std::make_pair(1,      "Earth"));
    sorted_map.insert(std::make_pair(0.532,  "Mars"));
    sorted_map.insert(std::make_pair(10.97,  "Jupiter"));
    sorted_map.insert(std::make_pair(9.14,   "Saturn"));
    sorted_map.insert(std::make_pair(3.981,  "Uranus"));
    sorted_map.insert(std::make_pair(3.865,  "Neptune"));

    for (auto const& entry: sorted_map)
    {
        std::cout << entry.second << " (" << entry.first << " of Earth's radius)" << '\n';
    }
}

Çıktı:

Mercury (0.3829 of Earth's radius)
Mars (0.532 of Earth's radius)
Venus (0.9499 of Earth's radius)
Earth (1 of Earth's radius)
Neptune (3.865 of Earth's radius)
Uranus (3.981 of Earth's radius)
Saturn (9.14 of Earth's radius)
Jupiter (10.97 of Earth's radius)

Eşit anahtarlara sahip girişler mümkünse, ‘harita’ yerine ‘multimap’ kullanın (aşağıdaki örnekte olduğu gibi).

Öğeleri azalan şekilde sıralamak için, haritayı uygun bir karşılaştırma işleviyle (std::greer<>) bildirin:

#include <iostream>
#include <utility>
#include <map>

int main()
{
    std::multimap<int, std::string, std::greater<int>> sorted_map;
    // Sort the names of animals in descending order of the number of legs
    sorted_map.insert(std::make_pair(6,   "bug"));
    sorted_map.insert(std::make_pair(4,   "cat"));
    sorted_map.insert(std::make_pair(100, "centipede"));
    sorted_map.insert(std::make_pair(2,   "chicken"));
    sorted_map.insert(std::make_pair(0,   "fish"));
    sorted_map.insert(std::make_pair(4,   "horse"));
    sorted_map.insert(std::make_pair(8,   "spider"));

    for (auto const& entry: sorted_map)
    {
        std::cout << entry.second << " (has " << entry.first << " legs)" << '\n';
    }
}

Çıktı

centipede (has 100 legs)
spider (has 8 legs)
bug (has 6 legs)
cat (has 4 legs)
horse (has 4 legs)
chicken (has 2 legs)
fish (has 0 legs)

Yerleşik dizileri sıralama #

“Sort” algoritması, iki yineleyici tarafından tanımlanan bir diziyi sıralar. Bu, yerleşik (c stili olarak da bilinir) bir diziyi sıralamak için yeterlidir.

int arr1[] = {36, 24, 42, 60, 59};

// sort numbers in ascending order
sort(std::begin(arr1), std::end(arr1));

// sort numbers in descending order
sort(std::begin(arr1), std::end(arr1), std::greater<int>());

C++ 11’den önce, dizinin sonunun, dizinin boyutu kullanılarak “hesaplanması” gerekiyordu:

// Use a hard-coded number for array size
sort(arr1, arr1 + 5);

// Alternatively, use an expression
const size_t arr1_size = sizeof(arr1) / sizeof(*arr1);
sort(arr1, arr1 + arr1_size);
