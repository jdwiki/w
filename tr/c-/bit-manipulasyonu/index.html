std::bitset kullanmak için <bitset> üstbilgi eklemeniz gerekir.

#include <bitset>

‘std::bitset’, bit kümelerinin c-tarzı işlenmesiyle aynı kullanıma izin vermek için tüm operatör işlevlerini aşırı yükler.

Referanslar

  • Bit Twiddling Hacks

Biraz geçiş yapmak #

C tarzı bit manipülasyonu #

XOR operatörü (^) kullanılarak bir bit değiştirilebilir.

// Bit x will be the opposite value of what it is currently
number ^= 1LL << x;

std::bitset’i kullanma #

std::bitset<4> num(std::string("0100"));
num.flip(2); // num is now 0000
num.flip(0); // num is now 0001
num.flip();  // num is now 1110 (flips all bits)

Biraz kontrol etmek #

C tarzı bit manipülasyonu #

Bitin değeri, sayıyı “x” kez sağa kaydırarak ve ardından bit düzeyinde VE (’&’) gerçekleştirerek elde edilebilir:

(number >> x) & 1LL;  // 1 if the 'x'th bit of 'number' is set, 0 otherwise

Sağa kaydırma işlemi, aritmetik (işaretli) kaydırma veya mantıksal (imzasız) kaydırma olarak uygulanabilir. “sayı » x” ifadesindeki “sayı”, işaretli bir türe ve negatif bir değere sahipse, elde edilen değer uygulama tanımlıdır.

Bu bitin değerine doğrudan yerinde ihtiyacımız varsa, bunun yerine maskeyi sola kaydırabiliriz:

(number & (1LL << x));  // (1 << x) if the 'x'th bit of 'number' is set, 0 otherwise

Her ikisi de koşullu olarak kullanılabilir, çünkü sıfırdan farklı tüm değerler doğru kabul edilir.

std::bitset’i kullanma #

std::bitset<4> num(std::string("0010"));
bool bit_val = num.test(1);  // bit_val value is set to true;

Tüm bitleri ayarla #

C tarzı bit manipülasyonu #

x = -1; // -1 == 1111 1111 ... 1111b

(Bunun neden işe yaradığına ve aslında en iyi yaklaşım olduğuna dair bir açıklama için buraya bakın.)

std::bitset’i kullanma #

std::bitset<10> x;
x.set(); // Sets all bits to '1'

En sağdaki küme bitini kaldır #

C tarzı bit manipülasyonu #

template <typename T>
T rightmostSetBitRemoved(T n)
{
    // static_assert(std::is_integral<T>::value && !std::is_signed<T>::value, "type should be unsigned"); // For c++11 and later
    return n & (n - 1);
}

Açıklama

  • ’n’ sıfır ise, sıfır olan ‘0 & 0xFF..FF’ elde ederiz.
  • yoksa ’n’, ‘0bxxxxxx1..00’ olarak yazılabilir ve ’n - 1’, ‘0bxxxxxx011..11’, yani ’n & (n - 1)’, ‘0bxxxxxx000..00’ şeklindedir.

Biraz ayarlama #

C tarzı bit işleme #

Bitsel VEYA operatörü (|) kullanılarak bir bit ayarlanabilir.

// Bit x will be set
number |= 1LL << x;

std::bitset’i kullanma #

“set(x)” veya “set(x,true)” - “x” konumundaki biti “1” olarak ayarlar.

std::bitset<5> num(std::string("01100"));
num.set(0);      // num is now 01101
num.set(2);      // num is still 01101
num.set(4,true); // num is now 11110

Biraz temizleme #

C tarzı bit manipülasyonu #

Bitsel AND operatörü (&) kullanılarak bir bit temizlenebilir.

// Bit x will be cleared
number &= ~(1LL << x);

std::bitset’i kullanma #

“reset(x)” veya “set(x,false)” - “x” konumundaki biti temizler.

std::bitset<5> num(std::string("01100"));
num.reset(2);     // num is now 01000
num.reset(0);     // num is still 01000
num.set(3,false); // num is now 00000

n. biti x olarak değiştirme #

C tarzı bit manipülasyonu #

// Bit n will be set if x is 1 and cleared if x is 0.
number ^= (-x ^ number) & (1LL << n);

std::bitset’i kullanma #

set(n,val) - n bitini val değerine ayarlar.

std::bitset<5> num(std::string("00100"));
num.set(0,true);  // num is now 00101
num.set(2,false); // num is now 00001

Sayma bitleri seti #

Bir bit dizisinin popülasyon sayımına kriptografide ve diğer uygulamalarda sıklıkla ihtiyaç duyulur ve sorun geniş çapta incelenmiştir.

Saf yol, bit başına bir yineleme gerektirir:

unsigned value = 1234;
unsigned bits = 0;  // accumulates the total number of bits set in `n`

for (bits = 0; value; value >>= 1)
  bits += value & 1;

Güzel bir numara ( https://www.wikiod.com/tr/docs/c%2b%2b/3016/bit-manipulation/17299/remove-rightmost-set-bit temel alınarak):

unsigned bits = 0;  // accumulates the total number of bits set in `n`

for (; value; ++bits)
  value &= value - 1;

Ayarlanmış bitler kadar yinelemeden geçer, bu nedenle “değer"in sıfırdan farklı bitlere sahip olması beklendiğinde iyidir.

Yöntem ilk olarak Peter Wegner tarafından önerildi (CACM 3 / 322 - 1960’da) ve Brian W. Kernighan ve Dennis M. Ritchie tarafından C Programlama Dili’nde göründüğü için iyi biliniyor.

Bu, biri multication olan 12 aritmetik işlem gerektirir:

unsigned popcount(std::uint64_t x)
{
  const std::uint64_t m1  = 0x5555555555555555;  // binary: 0101...
  const std::uint64_t m2  = 0x3333333333333333;  // binary: 00110011..
  const std::uint64_t m4  = 0x0f0f0f0f0f0f0f0f;  // binary: 0000111100001111

  x -= (x >> 1) & m1;             // put count of each 2 bits into those 2 bits
  x = (x & m2) + ((x >> 2) & m2); // put count of each 4 bits into those 4 bits 
  x = (x + (x >> 4)) & m4;        // put count of each 8 bits into those 8 bits 
  return (x * h01) >> 56;  // left 8 bits of x + (x<<8) + (x<<16) + (x<<24) + ... 
}

Bu tür bir uygulama en kötü durum davranışına sahiptir (daha fazla ayrıntı için bkz. [Hamming ağırlığı2).

Çoğu CPU’nun belirli bir talimatı vardır (x86’nın ‘popcnt’si gibi) ve derleyici belirli bir (standart dışı) yerleşik bir işlev sunabilir. Örneğin. g++ ile şunlar vardır:

int __builtin_popcount (unsigned x);

Bir tamsayının 2’nin katı olup olmadığını kontrol edin #

“n & (n - 1)” numarası da (bkz. https://www.wikiod.com/tr/docs/c%2b%2b/3016/bit-manipulation/17299/remove-rightmost-set-bit) yararlıdır bir tamsayının 2’nin kuvveti olup olmadığını belirlemek için:

bool power_of_2 = n && !(n & (n - 1));

Kontrolün ilk kısmı (n &&) olmadan, 0 yanlış bir şekilde 2’nin kuvveti olarak kabul edilir.

Bit Manipülasyon Uygulaması: Küçükten Büyük Harflere #

Bit manipülasyonunun çeşitli uygulamalarından biri, bir maske ve uygun bir bit işlemi seçerek bir harfi küçükten büyük harfe dönüştürmektir. Örneğin, a harfinin bu ikili gösterimi ‘01(1)00001’ iken sermaye karşılığı ‘01(0)00001’dir. Sadece parantez içinde biraz farklılık gösterirler. Bu durumda, a harfini küçükten büyük harfe dönüştürmek, temelde parantez içindeki biti bire ayarlamaktır. Bunu yapmak için aşağıdakileri yaparız:

/****************************************
convert small letter to captial letter.
========================================
     a: 01100001
  mask: 11011111  <-- (0xDF)  11(0)11111
      :---------
a&mask: 01000001  <-- A letter
*****************************************/

Bir harfi A harfine dönüştürme kodu

#include <cstdio>

int main()
{
    char op1 = 'a';  // "a" letter (i.e. small case)
    int mask = 0xDF; // choosing a proper mask

    printf("a (AND) mask = A\n");
    printf("%c   &   0xDF = %c\n", op1, op1 & mask);
    
    return 0;
}

sonuç

$ g++ main.cpp -o test1
$ ./test1
a (AND) mask = A
a   &   0xDF = A
