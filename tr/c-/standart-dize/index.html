Dizeler, karakter dizilerini temsil eden nesnelerdir. Standart “string” sınıfı, metin ve diğer karakter dizileriyle uğraşırken açık “char” dizilerini kullanmaya basit, güvenli ve çok yönlü bir alternatif sağlar. C++ “string” sınıfı, “std” ad alanının bir parçasıdır ve 1998’de standartlaştırılmıştır.

Sözdizimi #

  • // Boş dize bildirimi

    std::string s;

  • // const karakterden oluşturma* (c-string)

std::string s(“Merhaba”);

std::string s = “Merhaba”;

  • // Kopya yapıcıyı kullanarak oluşturma

std::string s1(“Merhaba”);

std::string s2(s1);

  • // alt diziden oluşturma

std::string s1(“Merhaba”);

std::string s2(s1, 0, 4); // s1’in 0 konumundan 4 karakteri s2’ye kopyalayın

  • // Bir karakter arabelleğinden oluşturma

std::string s1(“Merhaba Dünya”); std::string s2(s1, 5); // s1’in ilk 5 karakterini s2’ye kopyalayın

  • // Fill yapıcısını kullanarak inşa edin (yalnızca char)

std::string s(5, ‘a’); // s aaa içerir

  • // Aralık oluşturucu ve yineleyici kullanarak oluşturun

std::string s1(“Merhaba Dünya”);

std::string s2(s1.begin(), s1.begin()+5); // s1’in ilk 5 karakterini s2’ye kopyalayın

‘std::string’i kullanmadan önce, diğer başlıkların (örneğin ‘iostream’) içermediği işlevleri/operatörleri/aşırı yüklemeleri içerdiğinden, ‘string’ başlığını eklemelisiniz.

const char* yapıcısını nullptr ile kullanmak tanımsız davranışa yol açar.

std::string oops(nullptr);
std::cout << oops << "\n";

“index >= size()” ise “at” yöntemi bir “std::out_of_range” istisnası atar.

“İşleç[]“in davranışı biraz daha karmaşıktır, her durumda “index > size()” ise tanımsız davranışı vardır, ancak “index == size()” olduğunda:

  1. const olmayan bir dizede davranış tanımsız;
  2. Bir const dizesinde, CharT() (null karakteri) değerine sahip bir karaktere yapılan başvuru döndürülür.
son sürüm
  1. “CharT()” (null karakteri) değerine sahip bir karaktere yapılan başvuru döndürülür.
  2. Bu referansı değiştirmek tanımsız davranıştır.

std::wstring’e dönüştürme #

C++‘da, karakter dizileri, ‘std::basic_string’ sınıfının yerel bir karakter türüyle özelleştirilmesiyle temsil edilir. Standart kitaplık tarafından tanımlanan iki ana koleksiyon ‘std::string’ ve ‘std::wstring’dir:

#include <string>
#include <codecvt>
#include <locale>

std::string input_str = "this is a -string-, which is a sequence based on the -char- type.";
std::wstring input_wstr = L"this is a -wide- string, which is based on the -wchar_t- type.";

// conversion
std::wstring str_turned_to_wstr = std::wstring_convert<std::codecvt_utf8<wchar_t>>().from_bytes(input_str);

std::string wstr_turned_to_str = std::wstring_convert<std::codecvt_utf8<wchar_t>>().to_bytes(input_wstr);

Kullanılabilirliği ve/veya okunabilirliği geliştirmek için dönüştürmeyi gerçekleştirecek işlevleri tanımlayabilirsiniz:

#include <string>
#include <codecvt>
#include <locale>

using convert_t = std::codecvt_utf8<wchar_t>;
std::wstring_convert<convert_t, wchar_t> strconverter;

std::string to_string(std::wstring wstr)
{
    return strconverter.to_bytes(wstr);
}

std::wstring to_wstring(std::string str)
{
    return strconverter.from_bytes(str);
}

std::wstring a_wide_string = to_wstring("Hello World!");

Başlangıçta/sonda karakterleri kırpma #

Bu örnek, <algorithm>, <locale> ve <utility> başlıklarını gerektirir.

Bir diziyi veya dizeyi kırpmak, belirli bir yüklemle eşleşen tüm önde gelen ve sondaki öğeleri (veya karakterleri) kaldırmak anlamına gelir. Önce sondaki öğeleri kırpıyoruz, çünkü bu herhangi bir öğeyi hareket ettirmeyi içermiyor ve ardından önde gelen öğeleri kırpıyoruz. Aşağıdaki genellemelerin tüm “std::basic_string” türleri (ör. “std::string” ve “std::wstring”) için ve yanlışlıkla dizi kapsayıcıları (ör. ::liste`).

template <typename Sequence, // any basic_string, vector, list etc.
          typename Pred>     // a predicate on the element (character) type
Sequence& trim(Sequence& seq, Pred pred) {
    return trim_start(trim_end(seq, pred), pred);
}

Sondaki öğeleri kırpmak, yüklemle eşleşmeyen son öğesini bulmayı ve oradan silmeyi içerir:

template <typename Sequence, typename Pred>
Sequence& trim_end(Sequence& seq, Pred pred) {
    auto last = std::find_if_not(seq.rbegin(),
                                 seq.rend(),
                                 pred);
    seq.erase(last.base(), seq.end());
    return seq;
}

Önde gelen öğeleri kırpmak, yüklemle eşleşmeyen ilk öğeyi bulmayı ve oraya kadar silmeyi içerir:

template <typename Sequence, typename Pred>
Sequence& trim_start(Sequence& seq, Pred pred) {
    auto first = std::find_if_not(seq.begin(),
                                  seq.end(),
                                  pred);
    seq.erase(seq.begin(), first);
    return seq;
}

Bir “std::string” içindeki boşlukları kırpmak için yukarıdakileri uzmanlaştırmak için, yüklem olarak std::isspace() işlevini kullanabiliriz:

std::string& trim(std::string& str, const std::locale& loc = std::locale()) {
    return trim(str, [&loc](const char c){ return std::isspace(c, loc); });
}

std::string& trim_start(std::string& str, const std::locale& loc = std::locale()) {
    return trim_start(str, [&loc](const char c){ return std::isspace(c, loc); });
}

std::string& trim_end(std::string& str, const std::locale& loc = std::locale()) {
    return trim_end(str, [&loc](const char c){ return std::isspace(c, loc); });
}

Benzer şekilde, std::wstring vb. için std::iswspace() işlevini kullanabiliriz.

Kesilmiş bir kopya olan yeni bir dizi oluşturmak istiyorsanız, ayrı bir işlev kullanabilirsiniz:

template <typename Sequence, typename Pred>
Sequence trim_copy(Sequence seq, Pred pred) { // NOTE: passing seq by value
    trim(seq, pred);
    return seq;
}

Sözlüksel karşılaştırma #

İki std::string, ==, !=, <, <=, > ve >= operatörleri kullanılarak sözlükbilimsel olarak karşılaştırılabilir:

std::string str1 = "Foo";
std::string str2 = "Bar";

assert(!(str1 < str2));
assert(str > str2);
assert(!(str1 <= str2));
assert(str1 >= str2);
assert(!(str1 == str2));
assert(str1 != str2);

Tüm bu işlevler, karşılaştırmayı gerçekleştirmek için temeldeki ‘std::string::compare()’ yöntemini kullanır ve kolaylık sağlamak için boole değerleri döndürür. Bu işlevlerin çalışması, fiili uygulamadan bağımsız olarak aşağıdaki gibi yorumlanabilir:

  • operatör==:

    If str1.length() == str2.length() and each character pair matches, then returns true, otherwise returns false.

  • operatör!=:

    If str1.length() != str2.length() or one character pair doesn’t match, returns true, otherwise it returns false.

  • operatör< veya operatör>:

    Finds the first different character pair, compares them then returns the boolean result.

  • operatör<= veya operatör>=:

    Finds the first different character pair, compares them then returns the boolean result.

Not: karakter çifti terimi, aynı konumların her iki dizisindeki karşılık gelen karakterler anlamına gelir. Daha iyi anlamak için, iki örnek dize “str1” ve “str2” ise ve uzunlukları sırasıyla “n” ve “m” ise, her iki dizenin karakter çiftleri her biri “str1[i]” ve “str2[i] anlamına gelir. ` i = 0, 1, 2, …, max(n,m) olan çiftler. Karşılık gelen karakterin bulunmadığı herhangi bir i için, yani i ’n’ veya ’m’den büyük veya ona eşitse, bu en düşük değer olarak kabul edilir.

İşte “<” kullanımına bir örnek:

std::string str1 = "Barr";
std::string str2 = "Bar";

assert(str2 < str1);

Adımlar aşağıdaki gibidir:

  1. İlk karakterleri karşılaştırın, 'B' == 'B' - devam edin.
  2. İkinci karakterleri karşılaştırın, `‘a’ == ‘a’’ - devam edin.
  3. Üçüncü karakterleri karşılaştırın, 'r' == 'r' - devam edin.
  4. ‘str2’ aralığı artık tükendi, ‘str1’ aralığı hala karakterler içeriyor. Böylece str2 < str1.

            pos += with.length();

std::string hello = "Hello";
std::string world = "world";
hello += world; // "Helloworld"

Dize değişmezleri de dahil olmak üzere C dizeleri de ekleyebilirsiniz:

std::string hello = "Hello";
std::string world = "world";
const char *comma = ", ";
std::string newhelloworld = hello + comma + world + "!"; // "Hello, world!"

Ayrıca, tek tek ‘char’ları geri itmek için ‘push_back()’ kullanabilirsiniz:

std::string s = "a, b, ";
s.push_back('c'); // "a, b, c"

Ayrıca, ‘+=’ ile hemen hemen aynı olan ‘append()’ vardır:

std::string app = "test and ";
app.append("test"); // "test and test"

Tam sayılara/kayan nokta türlerine dönüştürme #

Bir sayı içeren bir “std::string”, dönüştürme işlevleri kullanılarak bir tamsayı türüne veya bir kayan nokta türüne dönüştürülebilir.

Bütün bu işlevlerin sayısal olmayan bir karakterle karşılaştıklarında giriş dizesini ayrıştırmayı bıraktığını unutmayın, bu nedenle "123abc" 123e dönüştürülür.

‘std::ato*’ işlev ailesi, C stili dizeleri (karakter dizileri) tamsayı veya kayan nokta türlerine dönüştürür:

std::string ten = "10";

double num1 = std::atof(ten.c_str());
int num2 = std::atoi(ten.c_str());
long num3 = std::atol(ten.c_str());

long long num4 = std::atoll(ten.c_str());
son sürüm

Ancak, bu işlevlerin kullanılması önerilmez, çünkü dizeyi ayrıştıramazlarsa ‘0’ döndürürler. Bu kötüdür, çünkü örneğin giriş dizesi “0” ise, “0” da geçerli bir sonuç olabilir, bu nedenle dönüştürmenin gerçekten başarısız olup olmadığını belirlemek imkansızdır.

Daha yeni ‘std::sto*’ işlev ailesi, ‘std::string’leri tamsayı veya kayan nokta türlerine dönüştürür ve girdilerini ayrıştıramazlarsa istisnalar atar. Mümkünse bu işlevleri kullanmalısınız:

std::string ten = "10";

int num1 = std::stoi(ten);
long num2 = std::stol(ten);
long long num3 = std::stoll(ten);

float num4 = std::stof(ten);
double num5 = std::stod(ten);
long double num6 = std::stold(ten);

Ayrıca, bu işlevler ayrıca std::ato* ailesinden farklı olarak sekizli ve onaltılı dizeleri de işler. İkinci parametre, giriş dizesindeki (burada gösterilmemiştir) ilk dönüştürülmemiş karaktere yönelik bir işaretçidir ve üçüncü parametre, kullanılacak temeldir. “0”, sekizli (“0” ile başlayan) ve onaltılı (“0x” veya “0X” ile başlayan) otomatik algılamadır ve diğer herhangi bir değer kullanılacak temeldir.

std::string ten = "10";
std::string ten_octal = "12";
std::string ten_hex = "0xA";

int num1 = std::stoi(ten, 0, 2); // Returns 2
int num2 = std::stoi(ten_octal, 0, 8); // Returns 10
long num3 = std::stol(ten_hex, 0, 16);  // Returns 10
long num4 = std::stol(ten_hex);  // Returns 0
long num5 = std::stol(ten_hex, 0, 0); // Returns 10 as it detects the leading 0x
son sürüm

Karakter kodlamaları arasında dönüştürme #

Kodlamalar arasında dönüştürme C++11 ile kolaydır ve çoğu derleyici bununla <codecvt> ve <locale> başlıkları aracılığıyla platformlar arası bir şekilde başa çıkabilir.

#include <iostream>
#include <codecvt>
#include <locale>
#include <string>
using namespace std;

int main() {
    // converts between wstring and utf8 string
    wstring_convert<codecvt_utf8_utf16<wchar_t>> wchar_to_utf8;
    // converts between u16string and utf8 string
    wstring_convert<codecvt_utf8_utf16<char16_t>, char16_t> utf16_to_utf8;
    
    wstring wstr = L"foobar";
    string utf8str = wchar_to_utf8.to_bytes(wstr);
    wstring wstr2 = wchar_to_utf8.from_bytes(utf8str);
    
    wcout << wstr << endl;
    cout << utf8str << endl;
    wcout << wstr2 << endl;
    
    u16string u16str = u"foobar";
    string utf8str2 = utf16_to_utf8.to_bytes(u16str);
    u16string u16str2 = utf16_to_utf8.from_bytes(utf8str2);
    
    return 0;
}

Visual Studio 2015’in bu dönüştürmeler için destek sağladığını, ancak kitaplık uygulamalarında bir hata, “char16_t” ile uğraşırken “wstring_convert” için farklı bir şablon kullanmayı gerektirdiğini unutmayın:

using utf16_char = unsigned short;
wstring_convert<codecvt_utf8_utf16<utf16_char>, utf16_char> conv_utf8_utf16;

void strings::utf16_to_utf8(const std::u16string& utf16, std::string& utf8)
{
  std::basic_string<utf16_char> tmp;
  tmp.resize(utf16.length());
  std::copy(utf16.begin(), utf16.end(), tmp.begin());
  utf8 = conv_utf8_utf16.to_bytes(tmp);
}
void strings::utf8_to_utf16(const std::string& utf8, std::u16string& utf16)
{ 
  std::basic_string<utf16_char> tmp = conv_utf8_utf16.from_bytes(utf8);
  utf16.clear();
  utf16.resize(tmp.length());
  std::copy(tmp.begin(), tmp.end(), utf16.begin());
}

Bir dizgede karakter(ler) bulma #

Bir karakter veya başka bir dize bulmak için std::string::find kullanabilirsiniz. İlk eşleşmenin ilk karakterinin konumunu döndürür. Hiçbir eşleşme bulunamazsa, işlev std::string::npos değerini döndürür.

std::string str = "Curiosity killed the cat";
auto it = str.find("cat");

if (it != std::string::npos)
    std::cout << "Found at position: " << it << '\n';
else
    std::cout << "Not found!\n";

Bulunduğu konum: 21

Arama fırsatları aşağıdaki işlevlerle daha da genişletilir:

find_first_of     // Find first occurrence of characters 
find_first_not_of // Find first absence of characters 
find_last_of      // Find last occurrence of characters 
find_last_not_of  // Find last absence of characters

Bu işlevler, dizenin sonundaki karakterleri aramanıza ve olumsuz durumu (yani dizede olmayan karakterler) bulmanıza olanak tanır. İşte bir örnek:

std::string str = "dog dog cat cat";
std::cout << "Found at position: " << str.find_last_of("gzx") << '\n';

Konumda bulundu: 6

Not: Yukarıdaki işlevlerin alt dizeleri aramadığını, bunun yerine arama dizesinde bulunan karakterleri aradığını unutmayın. Bu durumda, “g"nin son tekrarı “6” konumunda bulundu (diğer karakterler bulunamadı).
