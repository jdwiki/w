C++ dosya G/Ç, streams aracılığıyla yapılır. Anahtar soyutlamalar şunlardır:

Metin okumak için `std::istream'.

Metin yazmak için `std::ostream'.

Karakterleri okumak veya yazmak için std::streambuf.

Biçimlendirilmiş girdi, operatör>> kullanır.

Biçimlendirilmiş çıktı, operatör<< kullanır.

Akışlar, örneğin biçimlendirmenin ayrıntıları ve harici kodlamalar ile dahili kodlama arasındaki çeviri için “std::locale” kullanır.

Akışlar hakkında daha fazla bilgi: https://www.wikiod.com/tr/docs/c%2b%2b/7660/iostream-library

Bir dosyaya yazma #

Bir dosyaya yazmanın birkaç yolu vardır. En kolay yol, akış ekleme operatörü (<<) ile birlikte bir çıktı dosyası akışını (ofstream) kullanmaktır:

std::ofstream os("foo.txt");
if(os.is_open()){
    os << "Hello World!";
}

“«” yerine, çıktı dosyası akışının “write()” üye işlevini de kullanabilirsiniz:

std::ofstream os("foo.txt");
if(os.is_open()){
    char data[] = "Foo";

    // Writes 3 characters from data -> "Foo".
    os.write(data, 3);
}

Bir akışa yazdıktan sonra, işlemin başarısız olup olmadığını gösterdiğinden, “badbit” hata durum bayrağının ayarlanıp ayarlanmadığını her zaman kontrol etmelisiniz. Bu, çıktı dosyası akışının üye işlevi “bad()” çağrılarak yapılabilir:

os << "Hello Badbit!"; // This operation might fail for any reason.
if (os.bad())
    // Failed to write!

Dosya açma #

Bir dosyanın açılması 3 dosya akışının tümü için aynı şekilde yapılır (“ifstream”, “ofstream” ve “fstream”).

Dosyayı doğrudan yapıcıda açabilirsiniz:

std::ifstream ifs("foo.txt");  // ifstream: Opens file "foo.txt" for reading only.

std::ofstream ofs("foo.txt");  // ofstream: Opens file "foo.txt" for writing only.

std::fstream iofs("foo.txt");  // fstream:  Opens file "foo.txt" for reading and writing.

Alternatif olarak, dosya akışının open() üye fonksiyonunu kullanabilirsiniz:

std::ifstream ifs;
ifs.open("bar.txt");           // ifstream: Opens file "bar.txt" for reading only.

std::ofstream ofs;
ofs.open("bar.txt");           // ofstream: Opens file "bar.txt" for writing only.

std::fstream iofs;
iofs.open("bar.txt");          // fstream:  Opens file "bar.txt" for reading and writing.

Bir dosyanın başarıyla açılıp açılmadığını daima kontrol etmelisiniz (yazarken bile). Hatalar şunları içerebilir: dosya mevcut değil, dosya doğru erişim haklarına sahip değil, dosya zaten kullanılıyor, disk hataları oluştu, sürücü bağlantısı kesildi … Kontrol şu şekilde yapılabilir:

// Try to read the file 'foo.txt'.
std::ifstream ifs("fooo.txt");  // Note the typo; the file can't be opened.

// Check if the file has been opened successfully.
if (!ifs.is_open()) {
    // The file hasn't been opened; take appropriate actions here.
    throw CustomException(ifs, "File could not be opened");
}

Dosya yolu ters eğik çizgi içerdiğinde (örneğin, Windows sisteminde), onlardan düzgün bir şekilde kaçmalısınız:

// Open the file 'c:\folder\foo.txt' on Windows.
std::ifstream ifs("c:\\folder\\foo.txt"); // using escaped backslashes

veya ham değişmezi kullanın:

// Open the file 'c:\folder\foo.txt' on Windows.
std::ifstream ifs(R"(c:\folder\foo.txt)"); // using raw literal
son sürüm

veya bunun yerine eğik çizgi kullanın:

// Open the file 'c:\folder\foo.txt' on Windows.
std::ifstream ifs("c:/folder/foo.txt");

Şu anda Windows’ta yolunda ASCII olmayan karakterler içeren dosyayı açmak istiyorsanız, standart dışı geniş karakterli yol bağımsız değişkenini kullanabilirsiniz:

// Open the file 'пример\foo.txt' on Windows.
std::ifstream ifs(LR"(пример\foo.txt)"); // using wide characters with raw literal
son sürüm

Bir dosyadan okuma #

Bir dosyadan veri okumanın birkaç yolu vardır.

Verilerin nasıl biçimlendirildiğini biliyorsanız, akış çıkarma operatörünü (>>) kullanabilirsiniz. Aşağıdaki verileri içeren foo.txt adlı bir dosyanız olduğunu varsayalım:

John Doe 25 4 6 1987
Jane Doe 15 5 24 1976

Ardından, bu verileri dosyadan okumak için aşağıdaki kodu kullanabilirsiniz:

// Define variables.
std::ifstream is("foo.txt");
std::string firstname, lastname;
int age, bmonth, bday, byear;

// Extract firstname, lastname, age, bday month, bday day, and bday year in that order.
// Note: '>>' returns false if it reached EOF (end of file) or if the input data doesn't
// correspond to the type of the input variable (for example, the string "foo" can't be
// extracted into an 'int' variable).
while (is >> firstname >> lastname >> age >> bmonth >> bday >> byear)
    // Process the data that has been read.

Akış çıkarma operatörü >> her karakteri çıkarır ve saklanamayan bir karakter bulursa veya bu özel bir karakterse durur:

  • Dize türleri için operatör bir boşlukta ( ) veya yeni bir satırda (\n) durur.
  • Sayılar için operatör, sayı olmayan bir karakterde durur.

Bu, foo.txt dosyasının aşağıdaki sürümünün de önceki kod tarafından başarıyla okunacağı anlamına gelir:

John 
Doe 25
4 6 1987


Jane
Doe 
15 5
24
1976

Akış çıkarma operatörü >> her zaman kendisine verilen akışı döndürür. Bu nedenle, verileri art arda okumak için birden fazla operatör birbirine zincirlenebilir. Ancak, bir akış Boole ifadesi olarak da kullanılabilir (önceki koddaki ‘while’ döngüsünde gösterildiği gibi). Bunun nedeni, akış sınıflarının “bool” türü için bir dönüştürme operatörüne sahip olmasıdır. Bu “bool()” operatörü, akışta hata olmadığı sürece “true” değerini döndürür. Bir akış bir hata durumuna girerse (örneğin, daha fazla veri çıkarılamadığından), o zaman “bool()” operatörü “false” değerini döndürür. Bu nedenle, girdi dosyası sonuna kadar okunduktan sonra önceki koddaki ‘while’ döngüsünden çıkılacaktır.

Bir dosyanın tamamını bir dize olarak okumak istiyorsanız, aşağıdaki kodu kullanabilirsiniz:

// Opens 'foo.txt'.
std::ifstream is("foo.txt");
std::string whole_file;

// Sets position to the end of the file.
is.seekg(0, std::ios::end);

// Reserves memory for the file.
whole_file.reserve(is.tellg());

// Sets position to the start of the file.
is.seekg(0, std::ios::beg);

// Sets contents of 'whole_file' to all characters in the file.
whole_file.assign(std::istreambuf_iterator<char>(is),
  std::istreambuf_iterator<char>());

Bu kod, gereksiz bellek tahsislerini azaltmak için “dize” için yer ayırır.

Bir dosyayı satır satır okumak istiyorsanız, getline() işlevini kullanabilirsiniz:

std::ifstream is("foo.txt");   

// The function getline returns false if there are no more lines.
for (std::string str; std::getline(is, str);) {
    // Process the line that has been read.
}

Sabit sayıda karakter okumak istiyorsanız, akışın read() üye fonksiyonunu kullanabilirsiniz:

std::ifstream is("foo.txt");
char str[4];

// Read 4 characters from the file.
is.read(str, 4);

Bir okuma komutu yürüttükten sonra, işlemin başarısız olup olmadığını gösterdiğinden, hata durumu bayrağının ‘failbit’ ayarlanıp ayarlanmadığını her zaman kontrol etmelisiniz. Bu, dosya akışının üye işlevi fail() çağrılarak yapılabilir:

is.read(str, 4); // This operation might fail for any reason.

if (is.fail())
    // Failed to read!

Açılış modları #

Bir dosya akışı oluştururken bir açılış modu belirleyebilirsiniz. Açılış modu, temelde akışın dosyayı nasıl açacağını kontrol eden bir ayardır.

(Tüm modlar std::ios ad alanında bulunabilir.)

Bir dosya akışının yapıcısına veya onun open() üye işlevine ikinci parametre olarak bir açılış modu sağlanabilir:

std::ofstream os("foo.txt", std::ios::out | std::ios::trunc);

std::ifstream is;
is.open("foo.txt", std::ios::in | std::ios::binary);

Doğru bir varsayılan değere sahip olmalarına rağmen iostream üyeleri tarafından örtülü olarak ayarlanmadıkları için diğer bayrakları ayarlamak istiyorsanız ‘ios::in’ veya ‘ios::out’u ayarlamanız gerektiğine dikkat edilmelidir.

Bir açılış modu belirtmezseniz aşağıdaki varsayılan modlar kullanılır:

  • “ifstream” - “in”
  • “akım dışı” - “dışarı”
  • “fstream” - “in” ve “out”

Tasarım gereği belirtebileceğiniz dosya açma modları şunlardır:

Mod Anlamı için Tanım
“uygulama” ekle Çıktı Dosyanın sonuna veri ekler.
ikili ikili Giriş/Çıkış Giriş ve çıkış ikili olarak yapılır.
‘in’ giriş Giriş Dosyayı okumak için açar.
‘dışarı’ çıktı Çıktı Dosyayı yazmak için açar.
’trunc’ kesmek Giriş/Çıkış Açılırken dosyanın içeriğini kaldırır.
‘yedik’ sonunda Giriş Açılırken dosyanın sonuna gider.
Not: ‘İkili’ modun ayarlanması, verilerin tam olarak olduğu gibi okunmasını/yazılmasını sağlar; Ayarlamamak, yeni satır `’\n’’ karakterinin platforma özel bir satır sonu dizisine/sırasından çevrilmesini sağlar.

Örneğin Windows’ta satır sonu sırası CRLF’dir ("\r\n"). Şunu yazın: "\n" => "\r\n" Okuyun: "\r\n" => "\n"

Bir ASCII dosyasını std::string’e okuma #

std::ifstream f("file.txt");

if (f)
{
  std::stringstream buffer;
  buffer << f.rdbuf();
  f.close();

  // The content of "file.txt" is available in the string `buffer.str()`
}

rdbuf() yöntemi, stringstream::operator<< üye işlevi aracılığıyla “buffer"a aktarılabilen bir streambuf öğesine bir işaretçi döndürür .

Başka bir olasılık (Scott Meyers tarafından Etkili STL’te popüler hale getirilmiştir):

std::ifstream f("file.txt");

if (f)
{
  std::string str((std::istreambuf_iterator<char>(f)),
                  std::istreambuf_iterator<char>());

  // Operations on `str`...
}

Bu güzeldir çünkü az kod gerektirir (ve bir dosyayı yalnızca dizelere değil, doğrudan herhangi bir STL kapsayıcısına okumaya izin verir) ancak büyük dosyalar için yavaş olabilir.

NOT: dize oluşturucunun ilk argümanının etrafındaki fazladan parantezler, en can sıkıcı ayrıştırma sorununu önlemek için gereklidir.

Sonuncu ama bir o kadar önemli:

std::ifstream f("file.txt");

if (f)
{
  f.seekg(0, std::ios::end);
  const auto size = f.tellg();

  std::string str(size, ' ');
  f.seekg(0);
  f.read(&str[0], size); 
  f.close();

  // Operations on `str`...
}

bu muhtemelen en hızlı seçenektir (önerilen üç seçenek arasında).

Standart olmayan yerel ayarlarla dosya yazma #

Varsayılana farklı yerel ayarlar kullanarak bir dosya yazmanız gerekiyorsa, std::locale ve std: kullanabilirsiniz. :basic_ios::imbue() belirli bir dosya akışı için bunu yapmak için:

Kullanım kılavuzu:

  • Dosyayı açmadan önce her zaman bir akışa yerel uygulamanız gerekir.
  • Akış yüklendikten sonra yerel ayarı değiştirmemelisiniz.

Kısıtlamaların Nedenleri: Geçerli yerel ayar durumdan bağımsız değilse veya dosyanın başlangıcını göstermiyorsa, bir dosya akışını bir yerel ayar ile doldurmanın tanımsız davranışı vardır.

UTF-8 akışları (ve diğerleri) eyaletten bağımsız değildir. Ayrıca UTF-8 yerel ayarına sahip bir dosya akışı, açıldığında dosyadan Malzeme Listesi işaretçisini okumayı deneyebilir; bu yüzden sadece dosyayı açmak dosyadan karakterleri okuyabilir ve başlangıçta olmaz.

#include <iostream>
#include <fstream>
#include <locale>

int main()
{
  std::cout << "User-preferred locale setting is "
            << std::locale("").name().c_str() << std::endl;

  // Write a floating-point value using the user's preferred locale.
  std::ofstream ofs1;
  ofs1.imbue(std::locale(""));
  ofs1.open("file1.txt");
  ofs1 << 78123.456 << std::endl;

  // Use a specific locale (names are system-dependent)
  std::ofstream ofs2;
  ofs2.imbue(std::locale("en_US.UTF-8"));
  ofs2.open("file2.txt");
  ofs2 << 78123.456 << std::endl;

  // Switch to the classic "C" locale
  std::ofstream ofs3;
  ofs3.imbue(std::locale::classic());
  ofs3.open("file3.txt");
  ofs3 << 78123.456 << std::endl;
}

Açıkça klasik “C” yerel ayarına geçmek, programınız farklı bir varsayılan yerel ayar kullanıyorsa ve dosyaları okumak ve yazmak için sabit bir standart sağlamak istiyorsanız kullanışlıdır. “C” tercih edilen bir yerel ayar ile örnek şöyle yazar:

78,123.456
78,123.456
78123.456

Örneğin, tercih edilen yerel ayar Almancaysa ve bu nedenle farklı bir sayı biçimi kullanıyorsa, örnek şöyle yazar:

78 123,456
78,123.456
78123.456

(ilk satırdaki ondalık virgüle dikkat edin).

Bir akışı temizleme #

Dosya akışları, diğer birçok akış türü gibi varsayılan olarak arabelleğe alınır. Bu, akışa yazma işleminin temel alınan dosyanın hemen değişmesine neden olmayabileceği anlamına gelir. Tüm arabelleğe alınan yazmaların hemen gerçekleşmesini sağlamak için akışı temizleyebilirsiniz. Bunu doğrudan “flush()” yöntemini çağırarak veya “std::flush” akış manipülatörü aracılığıyla yapabilirsiniz:

std::ofstream os("foo.txt");
os << "Hello World!" << std::flush;

char data[3] = "Foo";
os.write(data, 3);
os.flush();

Yeni satır yazmayı akışı temizleme ile birleştiren bir akış manipülatörü std::endl vardır:

// Both following lines do the same thing
os << "Hello World!\n" << std::flush;
os << "Hello world!" << std::endl;

Arabelleğe alma, bir akışa yazma performansını iyileştirebilir. Bu nedenle çok fazla yazı yazan uygulamalar gereksiz yere flushing yapmaktan kaçınmalıdır. Aksine, G/Ç seyrek olarak yapılırsa, verilerin akış nesnesinde sıkışmasını önlemek için uygulamalar sık ​​sık boşaltmayı düşünmelidir.

Bir dosyayı bir kapsayıcıya okuma #

Aşağıdaki örnekte, dosyadan öğeleri okumak için std::string ve operator>> kullanıyoruz.

    std::ifstream file("file3.txt");

    std::vector<std::string>  v;

    std::string s;
    while(file >> s) // keep reading until we run out
    {
        v.push_back(s);
    }

Yukarıdaki örnekte, “operatör»” kullanarak her seferinde bir “öğe” okuyan dosyayı tekrarlıyoruz. Aynı etki, akıştan bir seferde bir “öğe” okuyan bir giriş yineleyicisi olan ‘std::istream_iterator’ kullanılarak elde edilebilir. Ayrıca çoğu kap, iki yineleyici kullanılarak oluşturulabilir, böylece yukarıdaki kodu şu şekilde basitleştirebiliriz:

    std::ifstream file("file3.txt");

    std::vector<std::string>  v(std::istream_iterator<std::string>{file},
                                std::istream_iterator<std::string>{});

Bunu, std::istream_iterator’a şablon parametresi olarak okumak istediğimiz nesneyi belirterek, sevdiğimiz herhangi bir nesne türünü okuyacak şekilde genişletebiliriz. Böylece yukarıdakileri basitçe şu şekilde satırları (kelimeler yerine) okuyacak şekilde genişletebiliriz:

// Unfortunately there is  no built in type that reads line using >>
// So here we build a simple helper class to do it. That will convert
// back to a string when used in string context.
struct Line
{
    // Store data here
    std::string data;
    // Convert object to string
    operator std::string const&() const {return data;}
    // Read a line from a stream.
    friend std::istream& operator>>(std::istream& stream, Line& line)
    {
        return std::getline(stream, line.data);
    }
};


    std::ifstream file("file3.txt");

    // Read the lines of a file into a container.
    std::vector<std::string>  v(std::istream_iterator<Line>{file},
                                std::istream_iterator<Line>{});

Bir döngü koşulu içinde dosyanın sonunu kontrol etme, kötü uygulama? #

eof yalnızca dosyanın sonunu okuduktan sonra “true” değerini döndürür. Bir sonraki okumanın akışın sonu olacağını göstermez.

while (!f.eof())
{
  // Everything is OK

  f >> buffer;

  // What if *only* now the eof / fail bit is set?

  /* Use `buffer` */
}

Doğru yazabilirsiniz:

while (!f.eof()) 
{  
  f >> buffer >> std::ws;

  if (f.fail())
    break;

  /* Use `buffer` */
}

ancak

while (f >> buffer)
{
  /* Use `buffer` */
}

daha basit ve daha az hataya açık.

Diğer referanslar:

  • std::ws: bir giriş akışından baştaki boşlukları atar
  • std::basic_ios::fail: ilişkili akışta bir hata meydana gelirse true döndürür

Bir dosyayı kapatma #

C++‘da bir dosyayı açıkça kapatmak nadiren gereklidir, çünkü bir dosya akışı ilişkili dosyayı kendi yıkıcısında otomatik olarak kapatacaktır. Ancak, dosya tanıtıcısını gereğinden uzun süre açık tutmaması için bir dosya akışı nesnesinin ömrünü sınırlamaya çalışmalısınız. Örneğin, bu, tüm dosya işlemlerini kendi kapsamına ({}) koyarak yapılabilir:

std::string const prepared_data = prepare_data();
{
    // Open a file for writing.
    std::ofstream output("foo.txt");

    // Write data.
    output << prepared_data;
}  // The ofstream will go out of scope here.
   // Its destructor will take care of closing the file properly.

‘close()’ işlevini açıkça çağırmak, yalnızca aynı ‘fstream’ nesnesini daha sonra yeniden kullanmak istiyorsanız ancak dosyayı aşağıdakiler arasında açık tutmak istemiyorsanız gereklidir:

// Open the file "foo.txt" for the first time.
std::ofstream output("foo.txt");

// Get some data to write from somewhere.
std::string const prepared_data = prepare_data();

// Write data to the file "foo.txt".
output << prepared_data;

// Close the file "foo.txt".
output.close();

// Preparing data might take a long time. Therefore, we don't open the output file stream
// before we actually can write some data to it.
std::string const more_prepared_data = prepare_complex_data();

// Open the file "foo.txt" for the second time once we are ready for writing.
output.open("foo.txt");

// Write the data to the file "foo.txt".
output << more_prepared_data;

// Close the file "foo.txt" once again.
output.close();

Biçimlendirilmiş bir metin dosyasından bir “yapı” okuma. #

struct info_type
{
    std::string name;
    int age;
    float height;
    
    // we define an overload of operator>> as a friend function which
    // gives in privileged access to private data members 
    friend std::istream& operator>>(std::istream& is, info_type& info)
    {
        // skip whitespace
        is >> std::ws;
        std::getline(is, info.name);
        is >> info.age;
        is >> info.height;
        return is;
    }
};

void func4()
{
    auto file = std::ifstream("file4.txt");

    std::vector<info_type> v;

    for(info_type info; file >> info;) // keep reading until we run out
    {
        // we only get here if the read succeeded
        v.push_back(info);
    }

    for(auto const& info: v)
    {
        std::cout << "  name: " << info.name << '\n';
        std::cout << "   age: " << info.age << " years" << '\n';
        std::cout << "height: " << info.height << "lbs" << '\n';
        std::cout << '\n';
    }
}

std::fileystem::copy_file("source_filename", "dest_filename");
