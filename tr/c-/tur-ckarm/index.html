Tür Çıkarımı

Bu sayfada

  • Veri Türü: Otomatik
  • Otomatik lambda
  • Döngüler ve otomatik

Bu konu, C++11’de bulunan “auto” anahtar sözcüğünü içeren tür çıkarımı hakkında tartışır.

Kopyalama veya mutasyonlar gibi istenmeyen davranışları önlemek gerekirse, “auto"yu her kullandığınızda “const”, “&” ve “constexpr” belirtmek genellikle daha iyidir. Bu ek ipuçları, derleyicinin başka herhangi bir çıkarım biçimi oluşturmamasını sağlar. Ayrıca ‘auto’nun aşırı kullanılması tavsiye edilmez ve yalnızca gerçek bildirim çok uzun olduğunda, özellikle STL şablonlarında kullanılmalıdır.

Veri Türü: Otomatik #

Bu örnek, derleyicinin gerçekleştirebileceği temel tür çıkarımlarını gösterir.

auto a = 1;        //    a = int
auto b = 2u;       //    b = unsigned int
auto c = &a;       //    c = int*
const auto  d = c; //    d = const int*
const auto& e = b; //    e = const unsigned int& 

auto x = a + b     //    x = int, #compiler warning unsigned and signed

auto v = std::vector<int>;    //    v = std::vector<int>

Ancak, auto anahtar sözcüğü her zaman “&” veya “const” veya “constexpr” için ek ipuçları olmadan beklenen tür çıkarımını gerçekleştirmez.

//    y = unsigned int, 
//    note that y does not infer as const unsigned int&
//    The compiler would have generated a copy instead of a reference value to e or b
auto y = e;

Otomatik lambda #

Veri türü auto anahtar sözcüğü, programcıların lambda işlevlerini bildirmesi için uygun bir yoldur. Programcıların bir işlev işaretçisi bildirmek için yazmaları gereken metin miktarını kısaltarak yardımcı olur.

auto DoThis = [](int a, int b) { return a + b; };
//    Do this is of type (int)(*DoThis)(int, int)
//    else we would have to write this long
int(*pDoThis)(int, int)= [](int a, int b) { return a + b; };

auto c = Dothis(1, 2);    //    c = int
auto d = pDothis(1, 2);   //    d = int

//    using 'auto' shortens the definition for lambda functions

Varsayılan olarak, lambda işlevlerinin dönüş türü tanımlanmadıysa, dönüş ifadesi türlerinden otomatik olarak çıkarılacaktır.

Bu 3 temelde aynı şey

[](int a, int b) -> int  { return a + b; };
[](int a, int b) -> auto { return a + b; };
[](int a, int b) { return a + b; };

Döngüler ve otomatik #

Bu örnek, for döngüleri için tür bildirimini kısaltmak için auto öğesinin nasıl kullanılabileceğini gösterir.

std::map<int, std::string> Map;
for (auto pair : Map)            //    pair = std::pair<int, std::string>
for (const auto pair : Map)      //    pair = const std::pair<int, std::string>    
for (const auto& pair : Map)     //    pair = const std::pair<int, std::string>&
for (auto i = 0; i < 1000; ++i)        //    i = int
for (auto i = 0; i < Map.size(); ++i)  //    Note that i = int and not size_t
for (auto i = Map.size(); i > 0; --i)  //    i = size_t
