int #

tarafından önerilen doğal boyuta sahip işaretli bir tamsayı türünü belirtir. “Yürütme ortamının mimarisi”, aralığı en az -32767 ila +32767 (dahil) içerir.

int x = 2;
int y = 3;
int z = x + y;

Diğer tamsayı türlerini elde etmek için “imzasız”, “kısa”, “uzun” ve “uzun uzun” (q.v.) ile birleştirilebilir.

bool #

Değeri “doğru” veya “yanlış” olabilen bir tamsayı türü.

bool is_even(int x) {
    return x%2 == 0;
}
const bool b = is_even(47); // false

karakter #

“Uygulamanın temel öğelerinin herhangi bir üyesini depolayacak kadar büyük” bir tamsayı türü. “char” karakterinin imzalı mı (ve en az -127 ila +127 arasında bir aralığa sahip olduğu) veya imzasız mı (ve en az 0 ila 255 arasında bir aralığa sahip olduğu) uygulama tarafından tanımlanır.

const char zero = '0';
const char one = zero + 1;
const char newline = '\n';
std::cout << one << newline; // prints 1 followed by a newline

karakter16_t #

“uint_least16_t” ile aynı boyuta ve hizalamaya sahip, bu nedenle bir UTF-16 kod birimini tutacak kadar büyük, işaretsiz bir tamsayı türü.

const char16_t message[] = u"你好，世界\n";            // Chinese for "hello, world\n"
std::cout << sizeof(message)/sizeof(char16_t) << "\n"; // prints 7
son sürüm

char32_t #

“uint_least32_t” ile aynı boyuta ve hizalamaya sahip, bu nedenle bir UTF-32 kod birimini tutacak kadar büyük, işaretsiz bir tamsayı türü.

const char32_t full_house[] = U"🂣🂳🂨🂸🃈";               // non-BMP characters
std::cout << sizeof(full_house)/sizeof(char32_t) << "\n"; // prints 6
son sürüm

batmadan yüzmek #

Kayan nokta türü. C++‘daki üç kayan nokta türünden en dar aralığa sahiptir.

float area(float radius) {
    const float pi = 3.14159f;
    return pi*radius*radius;
}

çift #

Kayan nokta türü. Menzili ‘float’ aralığını içerir. “Uzun” ile birleştirildiğinde, aralığı “çift” olan “uzun çift” kayan nokta türünü belirtir.

double area(double radius) {
    const double pi = 3.141592653589793;
    return pi*radius*radius;
}

uzun #

En az “int” uzunluğunda ve aralığı en az -2147483647 ila +2147483647 (yani, -(2^31 - 1) ila +(2^31 - 1) dahil) olan işaretli bir tamsayı türünü belirtir. ). Bu tür aynı zamanda ’long int’ olarak da yazılabilir.

const long approx_seconds_per_year = 60L*60L*24L*365L;

“Uzun çift” kombinasyonu, üç kayan nokta türünden en geniş aralığa sahip olan bir kayan nokta türünü belirtir.

long double area(long double radius) {
    const long double pi = 3.1415926535897932385L;
    return pi*radius*radius;
}

“long” belirteci “long long"da olduğu gibi iki kez geçtiğinde, en az “long” kadar uzun olan ve aralığı en az -9223372036854775807 - +9223372036854775807 (yani, -(2^63 - 1) ila +(2^63 - 1)).

// support files up to 2 TiB
const long long max_file_size = 2LL << 40;
son sürüm

kısa #

En az “char” uzunluğunda ve aralığı en az -32767 - +32767 dahil olmak üzere, işaretli bir tamsayı türünü belirtir. Bu tür aynı zamanda short int olarak da yazılabilir.

// (during the last year)
short hours_worked(short days_worked) {
    return 8*days_worked;
}

geçersiz #

Eksik tip; bir nesnenin “void” tipine sahip olması mümkün değildir ve “void” dizileri veya “void” referansları da yoktur. Hiçbir şey döndürmeyen işlevlerin dönüş türü olarak kullanılır.

Ayrıca, bir işlev, ‘void’ türünde tek bir parametreyle gereksiz olarak bildirilebilir; bu, parametresiz bir işlev bildirmekle eşdeğerdir (örneğin, “int main()” ve “int main(void)” aynı işlevi bildirir). Bu sözdizimine, C ile uyumluluk için izin verilir (burada işlev bildirimlerinin C++‘dakinden farklı bir anlamı vardır).

‘void*’ (“void’e işaretçi”) türü, herhangi bir nesne işaretçisinin kendisine dönüştürülebileceği ve geri döndürülebileceği ve aynı işaretçiyle sonuçlanabileceği özelliğine sahiptir. Bu özellik, “void*” türünü, örneğin C tarzı API’lerdeki (örneğin “qsort”, “pthread_create”) genel bağlamlar gibi belirli türde (güvenli olmayan) tür silme arabirimleri için uygun hale getirir.

Herhangi bir ifade, “void” türünde bir ifadeye dönüştürülebilir; buna atılan değer ifadesi: denir

static_cast<void>(std::printf("Hello, %s!\n", name));  // discard return value

Bu, bir ifadenin değerinin ilgi çekici olmadığını ve ifadenin yalnızca yan etkileri için değerlendirileceğini açıkça belirtmek için faydalı olabilir.

wchar_t #

Geniş karakter kümesi olarak da bilinen, desteklenen en büyük genişletilmiş karakter kümesinin tüm karakterlerini temsil edecek kadar büyük bir tamsayı türü. (“wchar_t"nin UTF-16 gibi herhangi bir özel kodlamayı kullandığı varsayımını yapmak taşınabilir değildir.)

Normalde ASCII 255 üzerindeki karakterleri depolamanız gerektiğinde kullanılır, çünkü ‘char’ karakter türünden daha büyük bir boyuta sahiptir.

const wchar_t message_ahmaric[] = L"ሰላም ልዑል\n"; //Ahmaric for "hello, world\n"
const wchar_t message_chinese[] = L"你好，世界\n";// Chinese for "hello, world\n"
const wchar_t message_hebrew[]  = L"שלום עולם\n"; //Hebrew for "hello, world\n"
const wchar_t message_russian[] = L"Привет мир\n";  //Russian for "hello, world\n"
const wchar_t message_tamil[]   = L"ஹலோ உலகம்\n"; //Tamil for "hello, world\n"
