std::setprecision #

‘out « setprecision(n)’ veya ‘in » setprecision(n)’ ifadesinde kullanıldığında, dışarı veya içeri akışının kesinlik parametresini tam olarak n’ye ayarlar. Bu işlevin parametresi, kesinlik için yeni bir değer olan tamsayıdır.

Örnek:

#include <iostream>
#include <iomanip>
#include <cmath>
#include <limits>
int main()
{
    const long double pi = std::acos(-1.L);
    std::cout << "default precision (6): " << pi << '\n'    
              << "std::precision(10):    " << std::setprecision(10) << pi << '\n'
              << "max precision:         "
              << std::setprecision(std::numeric_limits<long double>::digits10 + 1)
              << pi << '\n';
}
//Output
//default precision (6): 3.14159
//std::precision(10):    3.141592654
//max precision:         3.141592653589793239

std::setfill #

Bir ‘out « setfill(c)’ ifadesinde kullanıldığında, dışarı akışının dolgu karakterini ‘c’ olarak ayarlar.

Not: Geçerli dolgu karakteri std::ostream::fill ile elde edilebilir.

Örnek:

#include <iostream>
#include <iomanip>
int main()
{
    std::cout << "default fill: " << std::setw(10) << 42 << '\n'
          << "setfill('*'): " << std::setfill('*')
                              << std::setw(10) << 42 << '\n';
}
//output::
//default fill:         42
//setfill('*'): ********42

std::setiosflags #

out << setiosflags(mask) veya in >> setiosflags(mask) ifadesinde kullanıldığında, akışın tüm format bayraklarını maske tarafından belirtildiği şekilde dışarı veya içeri ayarlar.

Tüm std::ios_base::fmtflags listesi:

  • “dec” - tamsayı G/Ç için ondalık taban kullan
  • “oct” - tamsayı G/Ç için sekizlik tabanı kullan
  • “hex” - tamsayı G/Ç için onaltılık taban kullan
  • basefield - dec|oct|hex|0 maskeleme işlemleri için kullanışlıdır
  • sol - sol ayar(sağa dolgu karakterleri ekleyin)
  • “sağ” - sağ ayar (sola dolgu karakterleri ekler)
  • “dahili” - dahili ayarlama (dahili olarak belirtilen noktaya dolgu karakterleri ekler)
  • “adjustfield” - “sol|sağ|dahili”. Maskeleme işlemleri için kullanışlıdır
  • “bilimsel” - bilimsel gösterimi veya sabit ile birleştirilirse onaltılık gösterimi kullanarak kayan nokta türleri oluşturur
  • “sabit” - sabit notasyon veya bilimsel ile birleştirilirse onaltılık notasyon kullanarak kayan nokta türleri oluşturur
  • floatfield - bilimsel|sabit|(bilimsel|sabit)|0. Maskeleme işlemleri için kullanışlıdır
  • “boolalpha” - “bool” türünü alfasayısal biçimde ekler ve çıkarır
  • “showbase” - tamsayı çıktısı için sayısal tabanı belirten bir önek oluşturur, parasal G/Ç’de para birimi göstergesini gerektirir
  • showpoint - kayan noktalı sayı çıktısı için koşulsuz olarak bir ondalık basamak karakteri oluşturur
  • “showpos” - negatif olmayan sayısal çıktı için bir “+” karakteri oluşturur
  • skipws - ​​belirli giriş işlemlerinden önce baştaki boşlukları atla
  • “unitbuf” her çıktı işleminden sonra çıktıyı temizler
  • “büyük harf” - belirli küçük harfleri büyük harfleriyle değiştirin belirli çıktı çıktı işlemlerinde eşdeğerler

Manipülatörlere örnek:

    #include <iostream>
    #include <string>
    #include<iomanip>
    int main()
    {
      int l_iTemp = 47;
      std::cout<<  std::resetiosflags(std::ios_base::basefield);
      std::cout<<std::setiosflags( std::ios_base::oct)<<l_iTemp<<std::endl;
      //output: 57
      std::cout<<  std::resetiosflags(std::ios_base::basefield);
      std::cout<<std::setiosflags( std::ios_base::hex)<<l_iTemp<<std::endl;
      //output: 2f
      std::cout<<std::setiosflags( std::ios_base::uppercase)<<l_iTemp<<std::endl;
      //output 2F
      std::cout<<std::setfill('0')<<std::setw(12);
      std::cout<<std::resetiosflags(std::ios_base::uppercase);
      std::cout<<std::setiosflags( std::ios_base::right)<<l_iTemp<<std::endl;
      //output: 00000000002f
      
      std::cout<<std::resetiosflags(std::ios_base::basefield|std::ios_base::adjustfield);
      std::cout<<std::setfill('.')<<std::setw(10);
      std::cout<<std::setiosflags( std::ios_base::left)<<l_iTemp<<std::endl;
      //output: 47........
      
      std::cout<<std::resetiosflags(std::ios_base::adjustfield)<<std::setfill('#');
      std::cout<<std::setiosflags(std::ios_base::internal|std::ios_base::showpos);
      std::cout<<std::setw(10)<<l_iTemp<<std::endl;
      //output +#######47
      
      double l_dTemp = -1.2;
      double pi = 3.14159265359;
      std::cout<<pi<<"    "<<l_dTemp<<std::endl;
      //output +3.14159   -1.2
      std::cout<<std::setiosflags(std::ios_base::showpoint)<<l_dTemp<<std::endl;
      //output -1.20000
      std::cout<<setiosflags(std::ios_base::scientific)<<pi<<std::endl;
      //output: +3.141593e+00
      std::cout<<std::resetiosflags(std::ios_base::floatfield);
      std::cout<<setiosflags(std::ios_base::fixed)<<pi<<std::endl;
      //output: +3.141593
      bool b = true;
      std::cout<<std::setiosflags(std::ios_base::unitbuf|std::ios_base::boolalpha)<<b;
      //output: true
      return 0;
    }

standart::setw #

int val = 10;
// val will be printed to the extreme left end of the  output console:
std::cout << val << std::endl;
 // val will be printed in an output field of length 10 starting from right end of the field:
std::cout << std::setw(10) << val << std::endl;

Bu çıktı:

10
        10
1234567890

(karakter ofsetlerini görmeye yardımcı olmak için son satırın orada olduğu yer).

Bazen çıktı alanının genişliğini ayarlamamız gerekir, genellikle çıktıyı bazı yapılandırılmış ve uygun düzende almamız gerektiğinde. Bu, std::iomanip içindeki std::setw kullanılarak yapılabilir.

‘std::setw’ için sözdizimi şöyledir:

std::setw(int n)

n, ayarlanacak çıktı alanının uzunluğudur
