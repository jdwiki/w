Dize işlevleri, dize değerleri üzerinde işlemler gerçekleştirir ve sayısal ya da dize değerleri döndürür.

Dize işlevlerini kullanarak, örneğin verileri birleştirebilir, bir alt dizeyi çıkarabilir, dizeleri karşılaştırabilir veya bir dizeyi tüm büyük veya küçük harflere dönüştürebilirsiniz.

Sözdizimi #

  • BİRLEŞTİR ( string_value1, string_value2 [, string_valueN ] )
  • LTRIM ( karakter_ifadesi )
  • RTRIM ( karakter_ifadesi )
  • SUBSTRING ( ifade , başlangıç ​​, uzunluk )
  • ASCII (karakter_ifadesi)
  • REPLICATE ( string_expression ,integer_expression )
  • TERSİNE (string_expression)
  • BÜYÜK (karakter_ifadesi)
  • TRIM ([karakterler FROM] dizesi)
  • STRING_SPLIT (dize, ayırıcı)
  • STUFF (karakter_ifadesi, başlangıç, uzunluk, replaceWith_expression)
  • DEĞİŞTİR (string_expression, string_pattern, string_replacement)

Birleştir #

(Standart ANSI/ISO) SQL’de, dize birleştirme operatörü || şeklindedir. Bu sözdizimi, SQL Server dışındaki tüm büyük veritabanları tarafından desteklenir:

SELECT 'Hello' || 'World' || '!'; --returns HelloWorld!

Birçok veritabanı, dizeleri birleştirmek için bir “BAĞLANTI” işlevini destekler:

SELECT CONCAT('Hello', 'World'); --returns 'HelloWorld'

Bazı veritabanları, ikiden fazla dizgeyi birleştirmek için ‘CONCAT’ kullanmayı destekler (Oracle şunları yapmaz):

SELECT CONCAT('Hello', 'World', '!'); --returns 'HelloWorld!'

Bazı veritabanlarında, dize olmayan türlerin yayınlanması veya dönüştürülmesi gerekir:

SELECT CONCAT('Foo', CAST(42 AS VARCHAR(5)), 'Bar'); --returns 'Foo42Bar'

Bazı veritabanları (örneğin, Oracle) örtük kayıpsız dönüşümler gerçekleştirir. Örneğin, bir “CLOB” ve “NCLOB” üzerindeki “CONCAT”, bir “NCLOB” verir. Bir sayı üzerinde bir “CONCAT” ve bir “varchar2”, bir “varchar2” ile sonuçlanır, vb.:

SELECT CONCAT(CONCAT('Foo', 42), 'Bar') FROM dual; --returns Foo42Bar

Bazı veritabanları standart olmayan “+” operatörünü kullanabilir (ancak çoğu durumda “+” yalnızca sayılar için çalışır):

SELECT 'Foo' + CAST(42 AS VARCHAR(5)) + 'Bar';

CONCAT’ın desteklenmediği SQL Server < 2012’de, dizeleri birleştirmenin tek yolu ‘+‘dır.

Uzunluk #

SQL Server

LEN sondaki boşluğu saymaz.

SELECT LEN('Hello') -- returns 5

SELECT LEN('Hello '); -- returns 5

DATALENGTH, sondaki boşluğu sayar.

SELECT DATALENGTH('Hello') -- returns 5

SELECT DATALENGTH('Hello '); -- returns 6

Yine de, DATALENGTH öğesinin, dizeyi depolamak için kullanılan karakter kümesine bağlı olan, dizenin temeldeki bayt temsilinin uzunluğunu döndürdüğü belirtilmelidir.

DECLARE @str varchar(100) = 'Hello ' --varchar is usually an ASCII string, occupying 1 byte per char
SELECT DATALENGTH(@str) -- returns 6

DECLARE @nstr nvarchar(100) = 'Hello ' --nvarchar is a unicode string, occupying 2 bytes per char
SELECT DATALENGTH(@nstr) -- returns 12

Oracle

Sözdizimi: Uzunluk ( char )

Örnekler:

SELECT Length('Bible') FROM dual; --Returns 5
SELECT Length('righteousness') FROM dual; --Returns 13
SELECT Length(NULL) FROM dual; --Returns NULL

Ayrıca Bakınız: UzunlukB, UzunlukC, Uzunluk2, Uzunluk4

Boş alanları kırpın #

Kırp, seçimin başındaki veya sonundaki yazma alanını kaldırmak için kullanılır

MSSQL’de tek bir TRIM() yoktur

SELECT LTRIM('  Hello  ') --returns 'Hello  '
SELECT RTRIM('  Hello  ') --returns '  Hello'
SELECT LTRIM(RTRIM('  Hello  ')) --returns 'Hello'

MySql ve Oracle

SELECT TRIM('  Hello  ') --returns 'Hello'

Büyük ve küçük harf #

SELECT UPPER('HelloWorld') --returns 'HELLOWORLD'
SELECT LOWER('HelloWorld') --returns 'helloworld'

Bölmek #

Bir karakter ayırıcı kullanarak bir dize ifadesini böler. “STRING_SPLIT()” öğesinin tablo değerli bir işlev olduğuna dikkat edin.

SELECT value FROM STRING_SPLIT('Lorem ipsum dolor sit amet.', ' ');

value

Yer değiştirmek #

Sözdizimi:

REPLACE( Aranacak dize , Aranacak ve değiştirilecek dize , Orijinal dizeye yerleştirilecek dize )

Örnek:

SELECT REPLACE( 'Peter Steve Tom', 'Steve', 'Billy' ) --Return Values: Peter Billy Tom

Alt dize #

Sözdizimi: SUBSTRING ( string_expression, start, length ). SQL dizelerinin 1 dizinli olduğunu unutmayın.

SELECT SUBSTRING('Hello', 1, 2) --returns 'He'
SELECT SUBSTRING('Hello', 3, 3) --returns 'llo'

Bu, uzunluğu bilinmeyen bir dizgenin son “n” karakterlerini almak için genellikle “LEN()” işleviyle birlikte kullanılır.

DECLARE @str1 VARCHAR(10) = 'Hello', @str2 VARCHAR(10) = 'FooBarBaz';
SELECT SUBSTRING(@str1, LEN(@str1) - 2, 3) --returns 'llo'
SELECT SUBSTRING(@str2, LEN(@str2) - 2, 3) --returns 'Baz'

Şey #

Belirli bir konumda 0 veya daha fazla karakteri değiştirerek bir dizeyi diğerine doldurun.

Not: “başlangıç” konumu 1 dizinlidir (dizin oluşturmaya 0’dan değil, 1’den başlarsınız).

Sözdizimi:

STUFF ( character_expression , start , length , replaceWith_expression )

Örnek:

SELECT STUFF('FooBarBaz', 4, 3, 'Hello') --returns 'FooHelloBaz'

REGEXP #

Bir dizenin normal bir ifadeyle (başka bir dize tarafından tanımlanan) eşleşip eşleşmediğini kontrol eder.

SELECT 'bedded' REGEXP '[a-f]' -- returns True

SELECT 'beam' REGEXP '[a-f]' -- returns False
son sürüm

SOL SAĞ #

Sözdizimi:
LEFT ( string ifade , tamsayı ) RIGHT ( string-expression , integer )

SELECT LEFT('Hello',2)  --return He  
SELECT RIGHT('Hello',2) --return lo

Oracle SQL, SOL ve SAĞ işlevlerine sahip değildir. SUBSTR ve LENGTH ile öykünebilirler.
SUBSTR ( string-expression, 1, integer )
SUBSTR ( string-expression, length(string-expression)-integer+1, integer)

SELECT SUBSTR('Hello',1,2)  --return He  
SELECT SUBSTR('Hello',LENGTH('Hello')-2+1,2) --return lo

TERSİ #

Sözdizimi: REVERSE ( string ifade )

SELECT REVERSE('Hello') --returns olleH

TEKRARLAMAK #

“REPLICATE” işlevi, bir dizeyi belirtilen sayıda kendisiyle birleştirir.

Sözdizimi: REPLICATE ( string ifade , tamsayı )

SELECT REPLICATE ('Hello',4) --returns 'HelloHelloHelloHello'

SQL’de işlevi değiştir Seç ve Güncelle sorgusu #

SQL’deki Replace işlevi, bir dizenin içeriğini güncellemek için kullanılır. İşlev çağrısı MySQL, Oracle ve SQL Server için REPLACE( ) şeklindedir.

Değiştir işlevinin sözdizimi şöyledir:

REPLACE (str, find, repl)

Aşağıdaki örnek, Çalışanlar tablosunda ‘Güney’ tekrarlarını ‘Güney’ ile değiştirir:

Ad Adres
James Güney New York
John Güney Boston
Michael Güney San Diego

Bildirimi Seçin :

Aşağıdaki Değiştir işlevini uygularsak:

SELECT 
    FirstName, 
    REPLACE (Address, 'South', 'Southern') Address
FROM Employees 
ORDER BY FirstName

Sonuç:

Ad Adres
James Güney New York
John Güney Boston
Michael Güney San Diego

Güncelleme Bildirimi :

Aşağıdaki yaklaşımla tablomuzda kalıcı değişiklikler yapmak için bir replace işlevini kullanabiliriz.

Update Employees 
Set city = (Address, 'South', 'Southern');

Daha yaygın bir yaklaşım, bunu, bunun gibi bir WHERE yan tümcesi ile birlikte kullanmaktır:

Update Employees 
Set Address = (Address, 'South', 'Southern')
Where Address LIKE 'South%';

PARSENAME #

VERİTABANI : SQL Sunucusu

PARSENAME işlevi, verilen dizenin (nesne adı) belirli bir bölümünü döndürür. nesne adı, nesne adı, sahip adı, veritabanı adı ve sunucu adı gibi dizeler içerebilir.

Daha fazla ayrıntı MSDN:PARSENAME

Sözdizimi

PARSENAME('NameOfStringToParse',PartIndex)

Örnek

Nesne adını almak için ‘1’ parça indeksini kullanın

SELECT PARSENAME('ServerName.DatabaseName.SchemaName.ObjectName',1)  // returns `ObjectName`
SELECT PARSENAME('[1012-1111].SchoolDatabase.school.Student',1)     // returns `Student`

Şema adını almak için ‘2’ parça dizinini kullanın

SELECT PARSENAME('ServerName.DatabaseName.SchemaName.ObjectName',2)  // returns `SchemaName`
SELECT PARSENAME('[1012-1111].SchoolDatabase.school.Student',2)     // returns `school`

Veritabanı adını almak için ‘3’ parça dizinini kullanın

SELECT PARSENAME('ServerName.DatabaseName.SchemaName.ObjectName',3) // returns `DatabaseName`
SELECT PARSENAME('[1012-1111].SchoolDatabase.school.Student',3)    // returns `SchoolDatabase`

Sunucu adını almak için ‘4’ parça dizinini kullanın

SELECT PARSENAME('ServerName.DatabaseName.SchemaName.ObjectName',4)  // returns `ServerName`
SELECT PARSENAME('[1012-1111].SchoolDatabase.school.Student',4)     // returns `[1012-1111]`

PARSENAME, null değerini döndürür, belirtilen nesne adı dizesinde parça yok

INSTR #

Bir alt dizenin ilk oluşumunun dizinini döndür (bulunmazsa sıfır)

Sözdizimi: INSTR ( dize, alt dize )

SELECT INSTR('FooBarBar', 'Bar') -- return 4
SELECT INSTR('FooBarBar', 'Xar') -- return 0
