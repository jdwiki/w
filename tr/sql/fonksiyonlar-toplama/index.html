Sözdizimi #

  • işlev([DISTINCT] ifadesi) -DISTINCT isteğe bağlı bir parametredir
  • AVG ( [ TÜMÜ | DISTINCT ] ifadesi )
  • COUNT( { [ALL | DISTINCT ] ifadesi ] | * } )
  • GRUPLAMA(<column_expression>)
  • MAX ( [ TÜMÜ | DISTINCT ] ifadesi )
  • MIN ( [ TÜMÜ | DISTINCT ] ifadesi )
  • SUM ( [ ALL | DISTINCT ] ifadesi )
  • VAR ([ALL | DISTINCT] ifadesi) AŞIRI ( [ partition_by_clause ] order_by_clause )
  • VARP ([ALL | DISTINCT] ifadesi) AŞIRI ( [ partition_by_clause ] order_by_clause
  • STDEV ([ALL | DISTINCT] ifadesi) AŞIRI ( [ partition_by_clause ] order_by_clause )
  • STDEVP ([ALL | DISTINCT] ifadesi) AŞIRI ( [ partition_by_clause ] order_by_clause )

Veritabanı yönetiminde bir toplama işlevi, bir küme, bir torba veya bir liste gibi daha önemli anlam veya ölçüme sahip tek bir değer oluşturmak için birden çok satırın değerlerinin belirli kriterlere göre girdi olarak birlikte gruplandırıldığı bir işlevdir.

MIN        returns the smallest value in a given column
MAX        returns the largest value in a given column
SUM        returns the sum of the numeric values in a given column
AVG        returns the average value of a given column
COUNT      returns the total number of values in a given column
COUNT(*)   returns the number of rows in a table
GROUPING   Is a column or an expression that contains a column in a GROUP BY clause.
STDEV      returns the statistical standard deviation of all values in the specified expression.
STDEVP     returns the statistical standard deviation for the population for all values in the specified expression.
VAR        returns the statistical variance of all values in the specified expression. may be followed by the OVER clause.
VARP       returns the statistical variance for the population for all values in the specified expression.

Toplama işlevleri, bir “döndürülen sütun"a karşı hesaplamak için kullanılır. ‘SELECT’ ifadenizden sayısal veriler”. Temel olarak özetlerler belirli bir seçili veri sütununun sonuçları. - SQLCourse2.com

Tüm toplama işlevleri NULL değerleri yok sayar.

Liste Birleştirme #

bu SO yanıtına kısmi kredi.

Liste Birleştirme, değerleri her grup için tek bir dizede birleştirerek bir sütunu veya ifadeyi toplar. Her değeri sınırlandırmak için bir dize (boş veya atlandığında virgül) ve sonuçtaki değerlerin sırası belirtilebilir. SQL standardının bir parçası olmasa da, her büyük ilişkisel veritabanı satıcısı onu kendi yöntemleriyle destekler.

MySQL #

SELECT ColumnA
     , GROUP_CONCAT(ColumnB ORDER BY ColumnB SEPARATOR ',') AS ColumnBs
  FROM TableName
 GROUP BY ColumnA
 ORDER BY ColumnA;

Oracle ve DB2 #

SELECT ColumnA
     , LISTAGG(ColumnB, ',') WITHIN GROUP (ORDER BY ColumnB) AS ColumnBs
  FROM TableName
 GROUP BY ColumnA
 ORDER BY ColumnA;

PostgreSQL #

SELECT ColumnA
     , STRING_AGG(ColumnB, ',' ORDER BY ColumnB) AS ColumnBs
  FROM TableName
 GROUP BY ColumnA
 ORDER BY ColumnA;

SQL Server #

SQL Server 2016 ve önceki sürümler #

(CTE [KURU ilkesini2 teşvik etmek için dahil edilmiştir)

  WITH CTE_TableName AS (
       SELECT ColumnA, ColumnB
         FROM TableName)
SELECT t0.ColumnA
     , STUFF((
       SELECT ',' + t1.ColumnB
         FROM CTE_TableName t1
        WHERE t1.ColumnA = t0.ColumnA
        ORDER BY t1.ColumnB
          FOR XML PATH('')), 1, 1, '') AS ColumnBs
  FROM CTE_TableName t0
 GROUP BY t0.ColumnA
 ORDER BY ColumnA;

SQL Server 2017 ve SQL Azure #

SELECT ColumnA
     , STRING_AGG(ColumnB, ',') WITHIN GROUP (ORDER BY ColumnB) AS ColumnBs
  FROM TableName
 GROUP BY ColumnA
 ORDER BY ColumnA;

SQLite #

sipariş vermeden:

SELECT ColumnA
     , GROUP_CONCAT(ColumnB, ',') AS ColumnBs
  FROM TableName
 GROUP BY ColumnA
 ORDER BY ColumnA;

sipariş, bir alt sorgu veya CTE gerektirir:

  WITH CTE_TableName AS (
       SELECT ColumnA, ColumnB
         FROM TableName
        ORDER BY ColumnA, ColumnB)
SELECT ColumnA
     , GROUP_CONCAT(ColumnB, ',') AS ColumnBs
  FROM CTE_TableName
 GROUP BY ColumnA
 ORDER BY ColumnA;

TOPLA #

“Sum” işlevi, gruptaki tüm satırların değerini toplar. Group by cümlesi atlanırsa, tüm satırları toplar.

select sum(salary) TotalSalary
from employees;
Toplam Maaş

select DepartmentId, sum(salary) TotalSalary
from employees
group by DepartmentId;

AVG() #

AVG() toplama işlevi, genellikle bir sütundaki sayısal değerler olmak üzere belirli bir ifadenin ortalamasını döndürür. Dünyadaki şehirlerdeki yıllık nüfus hesaplamasını içeren bir tablomuz olduğunu varsayalım. New York City kayıtları aşağıdakilere benziyor:

ÖRNEK TABLO #

şehir_adı nüfus yıl
New York 8,550,405 2015
New York … …
New York 8,000,906 2005

Şehir adlarını, nüfus ölçümlerini ve son on yıla ait ölçüm yıllarını içeren bir tablodan New York, ABD’nin ortalama nüfusunu seçmek için:

SORGU #

select city_name, AVG(population) avg_population
from city_population
where city_name = 'NEW YORK CITY';

Nüfusun zaman içinde ortalaması alındığından, ölçüm yılının nasıl sorguda bulunmadığına dikkat edin.

SONUÇLAR #

şehir_adı ort_population
New York Şehri 8.250.754

Not: AVG() işlevi, değerleri sayısal türlere dönüştürür. Bu, tarihlerle çalışırken akılda tutulması özellikle önemlidir.

Saymak #

Satır sayısını sayabilirsiniz:

SELECT count(*) TotalRows
FROM employees;
Toplam Satır
4

Veya departman başına çalışanları sayın:

SELECT DepartmentId, count(*) NumEmployees
FROM employees
GROUP BY DepartmentId;
DepartmanKimliği SayıÇalışanlar
1 3
2 1

“NULL” değerlerini saymayacak efektle bir sütunu/ifadeyi sayabilirsiniz:

SELECT count(ManagerId) mgr
FROM EMPLOYEES;
yönetici
3

(Bir boş değer managerID sütunu vardır)

DISTINCT‘yi, COUNT gibi başka bir işlevin içinde de kullanabilirsiniz ve işlemin gerçekleştirileceği kümenin yalnızca DISTINCT üyelerini bulabilirsiniz.

Örneğin:

 SELECT COUNT(ContinentCode) AllCount
 ,      COUNT(DISTINCT ContinentCode) SingleCount
 FROM Countries;

Farklı değerler döndürür. SingleCount, tek tek Kıtaları yalnızca bir kez Sayacak, AllCount ise kopyaları içerecektir.

Maks #

Sütunun maksimum değerini bulun:

select max(age) from employee;

Yukarıdaki örnek, “çalışan” tablosunun “yaş” sütunu için en büyük değeri döndürecektir.

Sözdizimi:

SELECT MAX(column_name) FROM table_name;

Min #

Sütunun en küçük değerini bulun:

 select min(age) from employee;

Yukarıdaki örnek, “çalışan” tablosunun “yaş” sütunu için en küçük değeri döndürecektir.

Sözdizimi:

 SELECT MIN(column_name) FROM table_name;
