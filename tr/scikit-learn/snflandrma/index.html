Rastgele OrmanSınıflandırıcı #

Rastgele bir orman, veri kümesinin çeşitli alt örneklerinde bir dizi karar ağacı sınıflandırıcısına uyan ve tahmin doğruluğunu iyileştirmek ve aşırı uydurmayı kontrol etmek için ortalamayı kullanan bir meta tahmin edicidir.

Basit bir kullanım örneği:

İçe aktarmak:

from sklearn.ensemble import RandomForestClassifier

Tren verilerini ve hedef verileri tanımlayın:

train = [[1,2,3],[2,5,1],[2,1,7]]
target = [0,1,0]

“Hedef” içindeki değerler, tahmin etmek istediğiniz etiketi temsil eder.

Bir RandomForest nesnesi başlatın ve öğrenme (sığdır) işlemini gerçekleştirin:

rf = RandomForestClassifier(n_estimators=100)
rf.fit(train, target)

Tahmin etmek:

test = [2,2,3]
predicted = rf.predict(test)

Sınıflandırma Raporlarını Analiz Etme #

Kesinlik ve hatırlama, f1-skor (kesinlik ve hatırlamanın harmonik ortalaması) ve destek (gözlem sayısı) dahil olmak üzere ana sınıflandırma metriklerini gösteren bir metin raporu oluşturun. eğitim setindeki o sınıfın).

“sklearn” belgeler‘den örnek:

from sklearn.metrics import classification_report
y_true = [0, 1, 2, 2, 2]
y_pred = [0, 0, 2, 2, 1]
target_names = ['class 0', 'class 1', 'class 2']
print(classification_report(y_true, y_pred, target_names=target_names))

Çıktı -

         precision    recall  f1-score   support

class 0       0.50      1.00      0.67         1
class 1       0.00      0.00      0.00         1
class 2       1.00      0.67      0.80         3

avg / total   0.70      0.60      0.61         5

GradientBoostingClassifier #

Sınıflandırma için Gradient Boost. Gradient Boosting Classifier, art arda yinelemelerde (veya aşamalarda) hatası, artıkları düzelten (önceki aşamanın hatası) Regresyon Ağaçlarının eklenmesiyle düzeltilen bir temel modelin toplamsal bir topluluğudur.

İçe aktarmak:

from sklearn.ensemble import GradientBoostingClassifier

Oyuncak sınıflandırma verileri oluşturun

from sklearn.datasets import load_iris

iris_dataset = load_iris()

X, y = iris_dataset.data, iris_dataset.target

Bu verileri eğitim ve test setine ayıralım.

from sklearn.model_selection import train_test_split
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.2, random_state=0)

Varsayılan parametreleri kullanarak bir “GradientBoostingClassifier” modeli oluşturun.

gbc = GradientBoostingClassifier()
gbc.fit(X_train, y_train)

Test setinde puanlayalım

# We are using the default classification accuracy score
>>> gbc.score(X_test, y_test)
1

Varsayılan olarak yerleşik 100 tahmin edici vardır

>>> gbc.n_estimators
100

Bu, başlatma süresi sırasında “n_tahminciler"i farklı bir değere ayarlayarak kontrol edilebilir.

Destek Vektör Makinelerini Kullanma #

Destek vektör makineleri, iki etiketli nokta kümesi arasında (muhtemelen yüksek boyutlu) bir hiperdüzlem geçirmeye çalışan bir algoritma ailesidir, öyle ki, noktaların uzaklığı uçaktan bir anlamda optimaldir. SVM’ler sınıflandırma veya regresyon için kullanılabilir ([sklearn.svm.SVC](http://scikit-learn.org/stable/modules/generated/sklearn.svm.SVC.html’ye karşılık gelir) ve sklearn. svm.SVR.

Örnek:

Diyelim ki 2 boyutlu bir alanda çalışıyoruz. İlk olarak, bazı veriler oluşturuyoruz:

import numpy as np

Şimdi x ve y: oluşturuyoruz

x0, x1 = np.random.randn(10, 2), np.random.randn(10, 2) + (1, 1)
x = np.vstack((x0, x1))

y = [0] * 10 + [1] * 10

x‘in iki Gauss’tan oluştuğuna dikkat edin: biri (0, 0) çevresinde ve diğeri (1, 1) çevresinde ortalanmış.

Bir sınıflandırıcı oluşturmak için şunları kullanabiliriz:

from sklearn import svm

svm.SVC(kernel='linear').fit(x, y)

(0, 0): tahminini kontrol edelim.

>>> svm.SVC(kernel='linear').fit(x, y).predict([[0, 0]])
array([0])

Tahmin, sınıfın 0 olmasıdır.

Regresyon için benzer şekilde şunları yapabiliriz:

svm.SVR(kernel='linear').fit(x, y)

Bir Karar Ağacı #

Karar ağacı, kolayca anlaşılabilen bir dizi ayrıntılı kural (a>7 gibi) kullanan bir sınıflandırıcıdır.

Aşağıdaki örnek, uzunluğu 3 olan üç özellik vektörü kullanarak bir karar ağacı sınıflandırıcısını eğitir ve ardından test vektörü olarak adlandırılan, şimdiye kadar bilinmeyen bir dördüncü özellik vektörü için sonucu tahmin eder.

from sklearn.tree import DecisionTreeClassifier

# Define training and target set for the classifier
train = [[1,2,3],[2,5,1],[2,1,7]]
target = [10,20,30]

# Initialize Classifier. 
# Random values are initialized with always the same random seed of value 0 
# (allows reproducible results)
dectree = DecisionTreeClassifier(random_state=0)
dectree.fit(train, target)

# Test classifier with other, unknown feature vector
test = [2,2,3]
predicted = dectree.predict(test)

print predicted

Çıktı aşağıdakiler kullanılarak görselleştirilebilir:

import pydot
import StringIO

dotfile = StringIO.StringIO()
tree.export_graphviz(dectree, out_file=dotfile)
(graph,)=pydot.graph_from_dot_data(dotfile.getvalue())
graph.write_png("dtree.png")
graph.write_pdf("dtree.pdf")

Lojistik Regresyon Kullanarak Sınıflandırma #

LR Sınıflandırıcısında, tek bir denemenin olası sonuçlarını tanımlayan olasılıklar, bir lojistik fonksiyon kullanılarak modellenir. ’linear_model’ kitaplığında uygulanır.

from sklearn.linear_model import LogisticRegression

Sklearn LR uygulaması, isteğe bağlı L2 veya L1 düzenlemesi ile ikili, Bire-Duran veya çok terimli lojistik regresyona uyabilir. Örneğin, örnek bir sklearn veri setinde ikili bir sınıflandırmayı ele alalım.

from sklearn.datasets import make_hastie_10_2

X,y = make_hastie_10_2(n_samples=1000)

Burada X bir “n_samples X 10” dizisidir ve y, -1 veya +1 hedef etiketleridir.

Girdi verilerini eğitim ve test setlerine (%70-%30) bölmek için tren-test bölünmesini kullanın

from sklearn.model_selection import train_test_split 
#sklearn.cross_validation in older scikit versions

data_train, data_test, labels_train, labels_test = train_test_split(X,y, test_size=0.3)

LR Sınıflandırıcısını kullanmak diğer örneklere benzer

# Initialize Classifier. 
LRC = LogisticRegression()
LRC.fit(data_train, labels_train)

# Test classifier with the test data
predicted = LRC.predict(data_test)

Sonuçları görselleştirmek için Karışıklık matrisini kullanın

from sklearn.metrics import confusion_matrix

confusion_matrix(predicted, labels_test)
