Java’da komut kalıbı örneği #

[wikipedia] tanım:

Komut kalıbı, bir eylemi gerçekleştirmek veya daha sonra bir olayı tetiklemek için gereken tüm bilgileri kapsüllemek için bir nesnenin kullanıldığı davranışsal bir tasarım kalıbıdır.

dofactory‘den UML diyagramı:

Temel bileşenler ve iş akışı:

  1. “Komut”, “execute()” gibi soyut komutlar için bir arayüz bildirir.
  2. ‘Alıcı’ belirli bir komutu nasıl yürüteceğini bilir
  3. ‘Invoker’, yürütülmesi gereken ‘Somut Komut’u tutar
  4. “Müşteri”, “Somut Komut” oluşturur ve “Alıcı"yı atar
  5. “SomutKomut”, “Komut” ve “Alıcı” arasındaki bağlantıyı tanımlar

Bu şekilde, Komut düzeni Gönderen‘i (İstemci) Alıcı‘dan Çağlayan‘a ayırır. Invoker, hangi Command‘ın yürütüleceği konusunda tam bilgiye sahiptir ve Command, belirli bir işlemi yürütmek için hangi Alıcı’nın çağrılacağını bilir.

Kod parçacığı:

interface Command {
    void execute();
}
class Receiver {
    public void switchOn(){
        System.out.println("Switch on from:"+this.getClass().getSimpleName());
    }
}
class OnCommand implements Command{
    private Receiver receiver;

    public OnCommand(Receiver receiver){
        this.receiver = receiver;
    }
    public void execute(){
        receiver.switchOn();
    }
}
class Invoker {
    private Command command;
  
    public Invoker(Command command){
        this.command = command;
    }
    public void execute(){
        this.command.execute();
    }
}

class TV extends Receiver{
    
    public String toString(){
        return this.getClass().getSimpleName();
    }
}
class DVDPlayer extends Receiver{
    
    public String toString(){
        return this.getClass().getSimpleName();
    }
}

public class CommandDemoEx{
    public static void main(String args[]){
        // On command for TV with same invoker 
        Receiver receiver = new TV();
        Command onCommand = new OnCommand(receiver);
        Invoker invoker = new Invoker(onCommand);
        invoker.execute();
        
        // On command for DVDPlayer with same invoker 
        receiver = new DVDPlayer();
        onCommand = new OnCommand(receiver);
        invoker = new Invoker(onCommand);
        invoker.execute();            
    }
}

çıktı:

Switch on from:TV
Switch on from:DVDPlayer

Açıklama:

Bu örnekte,

  1. Komut arabirimi, ’execute()’ yöntemini tanımlar.
  2. OnCommand, execute() yöntemini uygulayan ConcreteCommand‘dır.
  3. Alıcı temel sınıftır.
  4. TV ve DVDPlayer, OnCommand gibi ConcreteCommand’a iletilen iki tür Alıcıdır.
  5. Invoker, Komut‘u içerir. Göndericiyi Alıcı ile ayırmanın anahtarıdır.
  6. Invoker, bu komutu yürütmek için Alıcı (TV)‘yi çağıran OnCommand -> alır.

Invoker’ı kullanarak TV ve DVDPlayer’ı açabilirsiniz. Bu programı genişletirseniz hem TV’yi hem de DVDPlayer’ı kapatırsınız.

  1. Geri arama mekanizmasını uygulamak
  2. Geri alma ve yineleme işlevini uygulamak için
  3. Komutların geçmişini korumak için
