Birden Çok Alt Noktada Paylaşılan Tek Gösterge #

Bazen bir alt nokta ızgaranız olur ve tek bir efsaneye sahip olmak istersiniz. aşağıdaki resimde olduğu gibi alt noktaların her biri için tüm satırları açıklar.

Bunu yapabilmek için şekil için küresel bir efsane oluşturmanız gerekecek. eksenler düzeyinde bir gösterge oluşturmak yerine (ayrı bir her alt nokta için efsane). Bu, olabildiğince fig.legend() çağırılarak elde edilir. aşağıdaki kodun kodunda görülüyor.

fig, (ax1, ax2, ax3) = plt.subplots(1, 3, figsize=(10,4))
fig.suptitle('Example of a Single Legend Shared Across Multiple Subplots')

# The data
x =  [1, 2, 3]
y1 = [1, 2, 3]
y2 = [3, 1, 3]
y3 = [1, 3, 1]
y4 = [2, 2, 3]

# Labels to use in the legend for each line
line_labels = ["Line A", "Line B", "Line C", "Line D"]

# Create the sub-plots, assigning a different color for each line.
# Also store the line objects created
l1 = ax1.plot(x, y1, color="red")[0]
l2 = ax2.plot(x, y2, color="green")[0]
l3 = ax3.plot(x, y3, color="blue")[0]
l4 = ax3.plot(x, y4, color="orange")[0] # A second line in the third subplot

# Create the legend
fig.legend([l1, l2, l3, l4],     # The line objects
           labels=line_labels,   # The labels for each line
           loc="center right",   # Position of legend
           borderaxespad=0.1,    # Small spacing around legend box
           title="Legend Title"  # Title for the legend
           )

# Adjust the scaling factor to fit your legend text completely outside the plot
# (smaller value results in more space being made for the legend)
plt.subplots_adjust(right=0.85)

plt.show()

Yukarıdaki örnekle ilgili dikkat edilmesi gereken bir şey şudur:

l1 = ax1.plot(x, y1, color="red")[0]

plot() çağrıldığında, line2D nesnelerinin bir listesini döndürür. bu durumda ile ayıklanan tek bir line2D nesnesi içeren bir liste döndürür. “[0]” dizine ekleniyor ve “l1” içinde saklanıyor.

Dahil etmekle ilgilendiğimiz tüm line2D nesnelerinin bir listesi. efsanenin fig.legend() için ilk argüman olarak iletilmesi gerekir. İkinci fig.legend() argümanı da gereklidir. bir listesi olması gerekiyordu göstergedeki her satır için etiket olarak kullanılacak dizeler.

fig.legend() öğesine iletilen diğer argümanlar tamamen isteğe bağlıdır ve sadece efsanenin estetiğine ince ayar yapılmasına yardımcı olur.

Basit Efsane #

Aynı çizimde, her biri farklı renkte birden fazla satırınız olduğunu varsayalım ve Her satırın neyi temsil ettiğini anlatan bir efsane yapmak istiyorsunuz. Bunu yapabilirsiniz “plot()“u çağırdığınızda her satıra bir etiket ileterek, ör. aşağıdaki satır “My Line 1” olarak etiketlenecektir.

ax.plot(x, y1, color="red", label="My Line 1")

Bu, o satırın açıklamasında görünecek metni belirtir. Şimdi gerçek efsaneyi görünür kılmak için ax.legend() çağırabiliriz

Varsayılan olarak, sağ üst köşedeki bir kutunun içinde bir efsane oluşturacaktır. arsa. Özelleştirmek için argümanları ’legend()’ öğesine iletebilirsiniz. Örneğin çerçeve kutusu olmadan sağ alt köşeye yerleştirebiliriz çevreleyen ve aşağıdakileri çağırarak lejand için bir başlık oluşturun:

ax.legend(loc="lower right", title="Legend Title", frameon=False)

Aşağıda bir örnek verilmiştir:

import matplotlib.pyplot as plt

# The data
x =  [1, 2, 3]
y1 = [2,  15, 27]
y2 = [10, 40, 45]
y3 = [5,  25, 40]

# Initialize the figure and axes
fig, ax = plt.subplots(1, figsize=(8, 6))

# Set the title for the figure
fig.suptitle('Simple Legend Example ', fontsize=15)

# Draw all the lines in the same plot, assigning a label for each one to be
# shown in the legend
ax.plot(x, y1, color="red", label="My Line 1")
ax.plot(x, y2, color="green", label="My Line 2")
ax.plot(x, y3, color="blue", label="My Line 3")

# Add a legend with title, position it on the lower right (loc) with no box framing (frameon)
ax.legend(loc="lower right", title="Legend Title", frameon=False)

# Show the plot
plt.show()

Konunun Dışına Yerleştirilen Efsane #

Bazen efsaneyi arsa dışına yerleştirmek gerekli veya arzu edilir. Aşağıdaki kod nasıl yapılacağını gösterir.

import matplotlib.pylab as plt
fig, ax = plt.subplots(1, 1, figsize=(10,6)) # make the figure with the size 10 x 6 inches
fig.suptitle('Example of a Legend Being Placed Outside of Plot')

# The data
x =  [1, 2, 3]
y1 = [1, 2, 4]
y2 = [2, 4, 8]
y3 = [3, 5, 14]

# Labels to use for each line
line_labels = ["Item A", "Item B", "Item C"]

# Create the lines, assigning different colors for each one.
# Also store the created line objects
l1 = ax.plot(x, y1, color="red")[0]
l2 = ax.plot(x, y2, color="green")[0]
l3 = ax.plot(x, y3, color="blue")[0]

fig.legend([l1, l2, l3],              # List of the line objects
           labels= line_labels,       # The labels for each line
           loc="center right",        # Position of the legend
           borderaxespad=0.1,         # Add little spacing around the legend box
           title="Legend Title")      # Title for the legend

# Adjust the scaling factor to fit your legend text completely outside the plot
# (smaller value results in more space being made for the legend)
plt.subplots_adjust(right=0.85)

plt.show()

Göstergeyi arsa dışına yerleştirmenin başka bir yolu, aşağıdaki örnekte gösterildiği gibi ‘bbox_to_anchor’ + ‘bbox_extra_artists’ + ‘bbox_inches=‘tight’’ kullanmaktır:

import matplotlib.pyplot as plt

# Data 
all_x = [10,20,30]
all_y = [[1,3], [1.5,2.9],[3,2]]

# Plot
fig = plt.figure(1)
ax = fig.add_subplot(111)
ax.plot(all_x, all_y)

# Add legend, title and axis labels
lgd = ax.legend([ 'Lag ' + str(lag) for lag in all_x], loc='center right', bbox_to_anchor=(1.3, 0.5))
ax.set_title('Title')
ax.set_xlabel('x label')
ax.set_ylabel('y label')    

fig.savefig('image_output.png',
            dpi=300, 
            format='png', 
            bbox_extra_artists=(lgd,),
            bbox_inches='tight')

Aynı Eksenlerde Birden Fazla Efsane #

plt.legend() veya ax.legend() i birden fazla çağırırsanız, ilk lejand kaldırılır ve yenisi çizilir. Resmi belgelere:

Bu, efsaneyi Eksenlerdeki en son tutamaçlara güncellemek için tekrar tekrar legend() çağırmak mümkün olacak şekilde yapılmıştır.

Yine de korkmayın: Bir eksene ikinci bir efsane (veya üçüncü veya dördüncü…) eklemek hala oldukça basittir. Buradaki örnekte, iki satır çiziyoruz, ardından ilgili maksimum ve minimum işaretçileri çiziyoruz. Bir gösterge çizgiler içindir, diğeri ise işaretçiler içindir.

import matplotlib.pyplot as plt
import numpy as np

# Generate data for plotting:  
x = np.linspace(0,2*np.pi,100)
y0 = np.sin(x)
y1 = .9*np.sin(.9*x)
# Find their maxima and minima and store
maxes = np.empty((2,2))
mins = np.empty((2,2))
for k,y in enumerate([y0,y1]):
    maxloc = y.argmax()
    maxes[k] = x[maxloc], y[maxloc]
    minloc = y.argmin()
    mins[k] = x[minloc], y[minloc]

# Instantiate figure and plot
fig = plt.figure()
ax = fig.add_subplot(111)
ax.plot(x,y0, label='y0')
ax.plot(x,y1, label='y1')
# Plot maxima and minima, and keep references to the lines
maxline, = ax.plot(maxes[:,0], maxes[:,1], 'r^')
minline, = ax.plot(mins[:,0], mins[:,1], 'ko')

# Add first legend:  only labeled data is included
leg1 = ax.legend(loc='lower left')
# Add second legend for the maxes and mins.
# leg1 will be removed from figure
leg2 = ax.legend([maxline,minline],['max','min'], loc='upper right')
# Manually add the first legend back
ax.add_artist(leg1)

Anahtar, gösterge nesnelerine referanslarınız olduğundan emin olmaktır. İlk somutlaştırdığınız (leg1), ikincisini eklediğinizde şekilden kaldırılır, ancak leg1 nesnesi hala mevcuttur ve ax.add_artist ile geri eklenebilir.

Gerçekten harika olan şey, hala her iki efsaneyi de manipüle edebilmenizdir. Örneğin, yukarıdaki kodun altına aşağıdakini ekleyin:

leg1.get_lines()[0].set_lw(8)
leg2.get_texts()[1].set_color('b')

Son olarak, örnekte, çizildiğinde yalnızca satırlara etiket verildiğini belirtmekte fayda var, yani ‘ax.legend()’ yalnızca bu satırları ’leg1’e ekler. Bu nedenle, belirteçlerin açıklaması (’leg2’) somutlaştırıldığında argüman olarak satırları ve etiketleri gerektiriyordu. Alternatif olarak, işaretçilere de çizildiklerinde etiketler verebilirdik. Ama o zaman, “ax.legend"e yapılan her iki çağrı, her bir lejandın sadece bizim istediğimiz öğeleri içermesi için bazı ekstra argümanlar gerektirebilirdi.
