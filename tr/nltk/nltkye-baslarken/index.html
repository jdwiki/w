NLTK’nın indirme işlevi #

NLTK’yı pip (pip install nltk) üzerinden kurabilirsiniz. Kurulduktan sonra birçok bileşen mevcut olmayacak ve NLTK’nın bazı özelliklerini kullanamayacaksınız.

UI kullanarak kurmak istediğiniz ek paketleri seçmek için Python kabuğunuzdan ntlk.download() işlevini çalıştırın. Alternatif olarak, python -m nltk.downloader [paket_adı] kullanabilirsiniz.

  • Mevcut tüm paketleri indirmek için.

    nltk.download(‘all’)

  • Belirli paketi indirmek için.

    nltk.download(‘package-name’)

  • Belirli bir klasörün tüm paketlerini indirmek için.

    import nltk

    dwlr = nltk.downloader.Downloader()

    chunkers, corpora, grammars, help, misc, #

    models, sentiment, stemmers, taggers, tokenizers #

    for pkg in dwlr.packages(): if pkg.subdir== ’taggers’: dwlr.download(pkg.id)

  • Corpora Klasörü dışındaki tüm paketleri indirmek için.

    import nltk

    dwlr = nltk.downloader.Downloader()

    for pkg in dwlr.corpora(): dwlr._status_cache[pkg.id] = ‘installed’

    dwlr.download(‘all’)

Kurulum veya Kurulum #

NLTK, Python sürümleri 2.7 veya 3.4+ gerektirir.

Bu talimatlar “python” sürümünü dikkate alır - 3.5

  • Mac/Unix :

    1. Install NLTK: run sudo pip install -U nltk
    2. Install Numpy (optional): run sudo pip install -U numpy
    3. Test installation: run python then type import nltk

    NOTE : For older versions of Python it might be necessary to install setuptools (see http://pypi.python.org/pypi/setuptools) and to install pip (sudo easy_install pip).

  • Pencereler :

    These instructions assume that you do not already have Python installed on your machine.

*32-bit binary installation*

1. Install Python 3.5: http://www.python.org/downloads/ (avoid the 64-bit versions)
2. Install Numpy (optional): http://sourceforge.net/projects/numpy/files/NumPy/ (the version that specifies pythnon3.5)
3. Install NLTK: http://pypi.python.org/pypi/nltk
4. Test installation: `Start>Python35`, then type `import nltk`

  • Üçüncü Taraf Yazılımı Yükleme :

    Please see: https://github.com/nltk/nltk/wiki/Installing-Third-Party-Software

Referans : http://www.nltk.org/install.html

Conda ile NLTK kurulumu. #

NLTK’yi Continuum’un anaconda / conda ile kurmak için.

Anaconda kullanıyorsanız, büyük olasılıkla nltk kökte zaten indirilmiş olacaktır (yine de çeşitli paketleri manuel olarak indirmeniz gerekebilir).

“conda” kullanma:

conda install nltk

“conda” kullanarak “nltk"yi yükseltmek için:

conda update nltk

“anakonda” ile:

Anaconda’da birden fazla python ortamı kullanıyorsanız, önce nltk’yi kurmak istediğiniz ortamı etkinleştirin. Komutu kullanarak aktif ortamı kontrol edebilirsiniz.

conda info --envs

Dizin yolundan önce * işareti olan ortam etkin ortamdır. Aktif ortam kullanımını değiştirmek için

activate <python_version>
for eg. activate python3.5

Şimdi bu ortamda kurulu paketlerin listesini commnad kullanarak kontrol edin.

conda list

Listede ’nltk’ bulamazsanız,

conda install -c anaconda nltk=3.2.1

Daha fazla bilgi için https://anaconda.org/anaconda/nltk adresini ziyaret edebilirsiniz.

Mini-conda a.k.a yüklemek için “conda”: http://conda.pydata.org/docs/install/quick.html

‘anaconda’yı kurmak için: https://docs.continuum.io/anaconda/install

NLTK ile #

Cümle sınırı tespiti yapmak için NLTK’yı (özellikle nltk.tokenize paketini) kullanabilirsiniz:

import nltk
text = "This is a test. Let's try this sentence boundary detector."
text_output = nltk.tokenize.sent_tokenize(text)
print('text_output: {0}'.format(text_output))

Çıktı:

text_output: ['This is a test.', "Let's try this sentence boundary detector."]
