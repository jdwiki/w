Süreçler Arası İletişim (IPC)

Süreçler arası iletişim (IPC) mekanizmaları, farklı bağımsız süreçlerin birbirleriyle iletişim kurmasına izin verir. Standart C herhangi bir IPC mekanizması sağlamaz. Bu nedenle, bu tür tüm mekanizmalar ana bilgisayar işletim sistemi tarafından tanımlanır. POSIX, kapsamlı bir IPC mekanizması seti tanımlar; Windows başka bir küme tanımlar; ve diğer sistemler kendi varyantlarını tanımlar.

Semaforlar #

Semaforlar, iki veya daha fazla işlem arasındaki işlemleri senkronize etmek için kullanılır. POSIX, iki farklı semafor işlevi kümesi tanımlar:

  1. ‘Sistem V IPC’ — semctl(), semop(), semget().
  2. ‘POSIX Semaforları’ — sem_close(), sem_destroy(), sem_getvalue(), sem_init(), sem_open(), sem_post(), sem_trywait(), sem_unlink().

Bu bölüm, Unix System V ile ortaya çıktıkları için bu şekilde adlandırılan System V IPC semaforlarını açıklar.

İlk olarak, gerekli başlıkları eklemeniz gerekir. POSIX’in eski sürümleri gerekli #include <sys/types.h>; modern POSIX ve çoğu sistem buna ihtiyaç duymaz.

#include <sys/sem.h>

Ardından, hem ebeveyn hem de çocuk için bir anahtar tanımlamanız gerekir.

#define KEY 0x1111

Bu anahtarın her iki programda da aynı olması gerekir, aksi takdirde aynı IPC yapısına atıfta bulunmazlar. Değerini kodlamadan üzerinde anlaşmaya varılmış bir anahtar oluşturmanın yolları vardır.

Ardından, derleyicinize bağlı olarak, şu adımı yapmanız gerekebilir veya gerekmeyebilir: semafor işlemleri amacıyla bir birlik ilan edin.

union semun {
    int val;
    struct semid_ds *buf;
    unsigned short  *array;
};

Ardından, try (semwait) ve raise (semsignal) yapılarınızı tanımlayın. P ve V adları Hollandaca kökenlidir.

struct sembuf p = { 0, -1, SEM_UNDO}; # semwait
struct sembuf v = { 0, +1, SEM_UNDO}; # semsignal

Şimdi, IPC semaforunuzun kimliğini alarak başlayın.

int id;
// 2nd argument is number of semaphores
// 3rd argument is the mode (IPC_CREAT creates the semaphore set if needed)
if ((id = semget(KEY, 1, 0666 | IPC_CREAT) < 0) {
    /* error handling code */
}

Üst öğede, semaforu 1 sayacı olacak şekilde başlatın.

union semun u;
u.val = 1;
if (semctl(id, 0, SETVAL, u) < 0) { // SETVAL is a macro to specify that you're setting the value of the semaphore to that specified by the union u
    /* error handling code */
}

Artık semaforu istediğiniz gibi azaltabilir veya artırabilirsiniz. Kritik bölümünüzün başlangıcında, semop() işlevini kullanarak sayacı azaltırsınız:

if (semop(id, &p, 1) < 0) {
    /* error handling code */
}

Semaforu artırmak için ‘&p’ yerine ‘&v’ kullanırsınız:

if (semop(id, &v, 1) < 0) {
    /* error handling code */
}

Her işlevin başarı durumunda “0” ve başarısızlık durumunda “-1” döndürdüğünü unutmayın. Bu iade durumlarını kontrol etmemek, yıkıcı sorunlara neden olabilir.

Örnek 1.1: İpliklerle Yarış #

Aşağıdaki program, bir çocuğu ‘çatallama’ işlemine sahip olacak ve hem ebeveyn hem de çocuk, herhangi bir senkronizasyon olmadan karakterleri terminale yazdırmaya çalışacak.

#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>
#include <string.h>

int main()
{
    int pid;
    pid =  fork();
    srand(pid);
    if(pid < 0)
    {
        perror("fork"); exit(1);
    }
    else if(pid)
    {
        char *s = "abcdefgh";
        int l = strlen(s);
        for(int i = 0; i < l; ++i)
        {
            putchar(s[i]);
            fflush(stdout);
            sleep(rand() % 2);
            putchar(s[i]);
            fflush(stdout);
            sleep(rand() % 2);
        }
    }
    else
    {
        char *s = "ABCDEFGH";
        int l = strlen(s);
        for(int i = 0; i < l; ++i)
        {
            putchar(s[i]);
            fflush(stdout);
            sleep(rand() % 2);
            putchar(s[i]);
            fflush(stdout);
            sleep(rand() % 2);
        }
    }
}

Çıktı (1. çalıştırma):

aAABaBCbCbDDcEEcddeFFGGHHeffgghh

(2. çalıştırma):

aabbccAABddBCeeCffgDDghEEhFFGGHH

Bu programı derlemek ve çalıştırmak size her seferinde farklı bir çıktı vermelidir.

Örnek 1.2: Semaforlarla Yarışmaktan Kaçının #

Semaforları kullanmak için Örnek 1.1‘i değiştirerek şunları elde ederiz:

#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>
#include <string.h>
#include <sys/types.h>
#include <sys/ipc.h>
#include <sys/sem.h>

#define KEY 0x1111

union semun {
    int val;
    struct semid_ds *buf;
    unsigned short  *array;
};

struct sembuf p = { 0, -1, SEM_UNDO};
struct sembuf v = { 0, +1, SEM_UNDO};

int main()
{
    int id = semget(KEY, 1, 0666 | IPC_CREAT);
    if(id < 0)
    {
        perror("semget"); exit(11);
    }
    union semun u;
    u.val = 1;
    if(semctl(id, 0, SETVAL, u) < 0)
    {
        perror("semctl"); exit(12);
    }
    int pid;
    pid =  fork();
    srand(pid);
    if(pid < 0)
    {
        perror("fork"); exit(1);
    }
    else if(pid)
    {
        char *s = "abcdefgh";
        int l = strlen(s);
        for(int i = 0; i < l; ++i)
        {
            if(semop(id, &p, 1) < 0)
            {
                perror("semop p"); exit(13);
            }
            putchar(s[i]);
            fflush(stdout);
            sleep(rand() % 2);
            putchar(s[i]);
            fflush(stdout);
            if(semop(id, &v, 1) < 0)
            {
                perror("semop p"); exit(14);
            }

            sleep(rand() % 2);
        }
    }
    else
    {
        char *s = "ABCDEFGH";
        int l = strlen(s);
        for(int i = 0; i < l; ++i)
        {
            if(semop(id, &p, 1) < 0)
            {
                perror("semop p"); exit(15);
            }
            putchar(s[i]);
            fflush(stdout);
            sleep(rand() % 2);
            putchar(s[i]);
            fflush(stdout);
            if(semop(id, &v, 1) < 0)
            {
                perror("semop p"); exit(16);
            }

            sleep(rand() % 2);
        }
    }
}

Çıktı:

aabbAABBCCccddeeDDffEEFFGGHHgghh

Bu programı derlemek ve çalıştırmak size her seferinde aynı çıktıyı verecektir.
