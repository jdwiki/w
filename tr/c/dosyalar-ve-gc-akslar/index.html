Mod dizeleri: #

fopen() ve freopen() içindeki mod dizeleri onlinepubs/9699919799/functions/freopen.html) şu değerlerden biri olabilir:

  • "r": Dosyayı, imleç dosyanın başına gelecek şekilde salt okunur modda açın.
  • "r+": İmleç dosyanın başına ayarlıyken dosyayı okuma-yazma modunda açın.
  • "w": Dosyayı içeriği 0 bayt olacak şekilde salt yazma modunda açın veya oluşturun. İmleç dosyanın başına ayarlanır.
  • "w+": Dosyayı içeriği 0 bayta kısaltılmış olarak okuma-yazma modunda açın veya oluşturun. İmleç dosyanın başına ayarlanır.
  • "a": İmleç dosyanın sonuna ayarlanmış olarak dosyayı salt yazma modunda açın veya oluşturun.
  • "a+": Dosyayı, okuma imleci dosyanın başına ayarlıyken okuma-yazma modunda açın veya oluşturun. Ancak çıktı her zaman dosyanın sonuna eklenecektir.

Bu dosya modlarının her birinde, ilk harften sonra eklenen bir “b” olabilir (ör. ““rb”” veya “a+b”’ veya ““ab+"). "b", farkın olduğu sistemlerde dosyanın metin dosyası yerine ikili dosya olarak ele alınması gerektiği anlamına gelir. Unix benzeri sistemlerde fark yaratmaz; Windows sistemlerinde önemlidir. (Ayrıca, Windows [fopen`](https://msdn.microsoft.com/en-us/library/yeby3zcb(v=vs.140).aspx) “b” yerine açık bir “t"nin belirtilmesine izin verir. ‘metin dosyası’ - ve platforma özel çok sayıda başka seçenek.)

  • "wx": Salt yazma modunda bir metin dosyası oluşturun. Dosya mevcut olmayabilir.
  • "wbx": Salt yazma modunda bir ikili dosya oluşturun. Dosya mevcut olmayabilir.

“x”, varsa, mod dizesindeki son karakter olmalıdır.

Aç ve dosyaya yaz #

#include <stdio.h>   /* for perror(), fopen(), fputs() and fclose() */
#include <stdlib.h>  /* for the EXIT_* macros */
 
int main(int argc, char **argv)
{
    int e = EXIT_SUCCESS;

    /* Get path from argument to main else default to output.txt */
    char *path = (argc > 1) ? argv[1] : "output.txt";

    /* Open file for writing and obtain file pointer */
    FILE *file = fopen(path, "w");
    
    /* Print error message and exit if fopen() failed */
    if (!file) 
    {
        perror(path);
        return EXIT_FAILURE;
    }

    /* Writes text to file. Unlike puts(), fputs() does not add a new-line. */
    if (fputs("Output in file.\n", file) == EOF)
    {
        perror(path);
        e = EXIT_FAILURE;
    }

    /* Close file */
    if (fclose(file)) 
    {
        perror(path);
        return EXIT_FAILURE;
    }
    return e;
}

Bu program, argümanda verilen adla dosyayı ana olarak açar, herhangi bir argüman verilmezse varsayılan olarak ‘output.txt’ olur. Aynı ada sahip bir dosya zaten varsa, içeriği atılır ve dosya yeni bir boş dosya olarak değerlendirilir. Dosyalar zaten mevcut değilse, fopen() çağrısı onu yaratır.

“fopen()” çağrısı herhangi bir nedenle başarısız olursa, bir “NULL” değeri döndürür ve global “errno” değişken değerini ayarlar. Bu, programın fopen() çağrısından sonra döndürülen değeri test edebileceği ve fopen() başarısız olursa perror() kullanabileceği anlamına gelir.

“fopen()” çağrısı başarılı olursa, geçerli bir “FILE” işaretçisi döndürür. Bu işaretçi daha sonra üzerinde fclose() çağrılıncaya kadar bu dosyaya referans vermek için kullanılabilir.

fputs() işlevi, verilen metni dosyanın önceki içeriğini değiştirerek açılan dosyaya yazar. “fopen()” işlevine benzer şekilde, “fputs()” işlevi de başarısız olursa “errno” değerini ayarlar, ancak bu durumda işlev başarısızlığı belirtmek için “EOF” döndürür (aksi halde negatif olmayan bir değer döndürür) .

fclose() işlevi tüm arabellekleri temizler, dosyayı kapatır ve FILE * ile gösterilen belleği boşaltır. Dönüş değeri, tıpkı “fputs()” gibi (başarılıysa “0” döndürmesine rağmen) tamamlanmayı gösterir, yine bir başarısızlık durumunda “errno” değerini ayarlar.

İşlemi çalıştır #

#include <stdio.h>

void print_all(FILE *stream)
{
    int c;
    while ((c = getc(stream)) != EOF)
        putchar(c);
}
int main(void)
{
    FILE *stream;

    /* call netstat command. netstat is available for Windows and Linux */
    if ((stream = popen("netstat", "r")) == NULL)
        return 1;
  
    print_all(stream);
    pclose(stream);
    return 0;
}

Bu program bir işlemi (netstat) popen() aracılığıyla çalıştırır ve süreçten tüm standart çıktıyı okur ve bunu standart çıktıya yansıtır.

Not: popen() standart C kitaplığında yoktur, ancak daha çok POSIX C‘ün bir parçasıdır)

fprintf #

Bir dosyada fprintf’yi, tıpkı bir konsolda olduğu gibi, “printf” ile kullanabilirsiniz. Örneğin, oyun kazançlarını, kayıplarını ve beraberliklerini takip etmek için yazabilirsiniz.

/* saves wins, losses and, ties */
void savewlt(FILE *fout, int wins, int losses, int ties)
{
    fprintf(fout, "Wins: %d\nTies: %d\nLosses: %d\n", wins, ties, losses);
}

Bir yan not: Bazı sistemler (kötü şöhretli Windows), çoğu programcının “normal” satır sonları dediği şeyi kullanmaz. UNIX benzeri sistemler satırları sonlandırmak için \n kullanırken, Windows bir çift karakter kullanır: \r (satır başı) ve \n (satır besleme). Bu diziye yaygın olarak CRLF denir. Bununla birlikte, C kullanırken, bu son derece platforma bağlı ayrıntılar için endişelenmenize gerek yoktur. Her \n örneğini doğru platform satırı sonuna dönüştürmek için bir C derleyicisi gerekir. Böylece bir Windows derleyicisi \n’yi \r\n’ye dönüştürür, ancak bir UNIX derleyicisi onu olduğu gibi tutar.

getline() kullanarak bir dosyadan satırları alın #

POSIX C kitaplığı, getline() işlevini tanımlar. Bu işlev, satır içeriğini tutmak için bir arabellek ayırır ve yeni satırı, satırdaki karakter sayısını ve arabelleğin boyutunu döndürür.

“example.txt"den her satırı alan örnek program:

#include <stdlib.h>
#include <stdio.h>  


#define FILENAME "example.txt"

int main(void)
{
  /* Open the file for reading */
  char *line_buf = NULL;
  size_t line_buf_size = 0;
  int line_count = 0;
  ssize_t line_size;
  FILE *fp = fopen(FILENAME, "r");
  if (!fp)
  {
    fprintf(stderr, "Error opening file '%s'\n", FILENAME);
    return EXIT_FAILURE;
  }

  /* Get the first line of the file. */
  line_size = getline(&line_buf, &line_buf_size, fp);

  /* Loop through until we are done with the file. */
  while (line_size >= 0)
  {
    /* Increment our line count */
    line_count++;

    /* Show the line details */
    printf("line[%06d]: chars=%06zd, buf size=%06zu, contents: %s", line_count,
        line_size, line_buf_size, line_buf);

    /* Get the next line */
    line_size = getline(&line_buf, &line_buf_size, fp);
  }

  /* Free the allocated line buffer */
  free(line_buf);
  line_buf = NULL;

  /* Close the file now that we are done with it */
  fclose(fp);

  return EXIT_SUCCESS;
}

Örnekte, “getline()” başlangıçta hiçbir arabellek ayrılmadan çağrılır. Bu ilk çağrı sırasında, ‘getline()’ bir arabellek ayırır, ilk satırı okur ve satırın içeriğini yeni arabelleğe yerleştirir. Sonraki çağrılarda, ‘getline()’ aynı ara belleği günceller ve yalnızca tüm satıra sığacak kadar büyük olmadığında arabelleği yeniden tahsis eder. Dosyayla işimiz bittiğinde geçici arabellek serbest bırakılır.

Başka bir seçenek de getdelim()dir. Bu, satır sonu karakterini belirtmeniz dışında ‘getline()’ ile aynıdır. Bu, yalnızca dosya türünüz için satırın son karakteri ‘\n’ değilse gereklidir. getline() Windows metin dosyalarıyla bile çalışır, çünkü çok baytlı satır sonu ("\r\n") ile ’\n’` hala satırdaki son karakterdir.

getline() uygulamasının örnek uygulaması #

#include <stdlib.h>
#include <stdio.h>
#include <errno.h>
#include <stdint.h>


#if !(defined _POSIX_C_SOURCE)
typedef long int ssize_t;
#endif

/* Only include our version of getline() if the POSIX version isn't available. */

#if !(defined _POSIX_C_SOURCE) || _POSIX_C_SOURCE < 200809L

#if !(defined SSIZE_MAX)
#define SSIZE_MAX (SIZE_MAX >> 1)
#endif

ssize_t getline(char **pline_buf, size_t *pn, FILE *fin)
{
  const size_t INITALLOC = 16;
  const size_t ALLOCSTEP = 16;
  size_t num_read = 0;

  /* First check that none of our input pointers are NULL. */
  if ((NULL == pline_buf) || (NULL == pn) || (NULL == fin))
  {
    errno = EINVAL;
    return -1;
  }

  /* If output buffer is NULL, then allocate a buffer. */
  if (NULL == *pline_buf)
  {
    *pline_buf = malloc(INITALLOC);
    if (NULL == *pline_buf)
    {
      /* Can't allocate memory. */
      return -1;
    }
    else
    {
      /* Note how big the buffer is at this time. */
      *pn = INITALLOC;
    }
  }

  /* Step through the file, pulling characters until either a newline or EOF. */

  {
    int c;
    while (EOF != (c = getc(fin)))
    {
      /* Note we read a character. */
      num_read++;

      /* Reallocate the buffer if we need more room */
      if (num_read >= *pn)
      {
        size_t n_realloc = *pn + ALLOCSTEP;
        char * tmp = realloc(*pline_buf, n_realloc + 1); /* +1 for the trailing NUL. */
        if (NULL != tmp)
        {
          /* Use the new buffer and note the new buffer size. */
          *pline_buf = tmp;
          *pn = n_realloc;
        }
        else
        {
          /* Exit with error and let the caller free the buffer. */
          return -1;
        }

        /* Test for overflow. */
        if (SSIZE_MAX < *pn)
        {
          errno = ERANGE;
          return -1;
        }
      }

      /* Add the character to the buffer. */
      (*pline_buf)[num_read - 1] = (char) c;

      /* Break from the loop if we hit the ending character. */
      if (c == '\n')
      {
        break;
      }
    }

    /* Note if we hit EOF. */
    if (EOF == c)
    {
      errno = 0;
      return -1;
    }
  }

  /* Terminate the string by suffixing NUL. */
  (*pline_buf)[num_read] = '\0';

  return (ssize_t) num_read;
}

#endif

İkili akış, dahili verileri şeffaf bir şekilde kaydedebilen sıralı bir karakter dizisidir. Bu modda, program ile dosya arasında herhangi bir yorum yapılmadan baytlar yazılır.

Neyse ki, imzasız * bir dönüştürmenin iki tamamlayıcı kullanması garanti edilir. İkili bir dosyaya imzalı bir tamsayı yazma kodu bu nedenle biraz şaşırtıcıdır.

/* write a 16-bit little endian integer */
int fput16le(int x, FILE *fp)
{
    unsigned int rep = x;
    int e1, e2;

    e1 = fputc(rep & 0xFF, fp);
    e2 = fputc((rep >> 8) & 0xFF, fp);

    if(e1 == EOF || e2 == EOF)
        return EOF;
    return 0;  
}

Bir dosyadan satırları oku #

‘stdio.h’ başlığı, ‘fgets()’ işlevini tanımlar. Bu işlev, bir akıştan bir satır okur ve onu belirtilen bir dizede saklar. İşlev, n - 1 karakterleri okunduğunda, satırsonu karakteri (\n') okunduğunda veya dosyanın sonuna (EOF) ulaşıldığında akıştan metin okumayı durdurur.

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

#define MAX_LINE_LENGTH 80

int main(int argc, char **argv)
{
    char *path;
    char line[MAX_LINE_LENGTH] = {0};
    unsigned int line_count = 0;
    
    if (argc < 1)
        return EXIT_FAILURE;
    path = argv[1];
    
    /* Open file */
    FILE *file = fopen(path, "r");
    
    if (!file)
    {
        perror(path);
        return EXIT_FAILURE;
    }
    
    /* Get each line until there are none left */
    while (fgets(line, MAX_LINE_LENGTH, file))
    {
        /* Print each line */
        printf("line[%06d]: %s", ++line_count, line);
        
        /* Add a trailing newline to lines that don't already have one */
        if (line[strlen(line) - 1] != '\n')
            printf("\n");
    }
    
    /* Close file */
    if (fclose(file))
    {
        return EXIT_FAILURE;
        perror(path);
    }
}
