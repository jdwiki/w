<ctype.h> — karakter sınıflandırma ve dönüştürme

Giriiş #

“ctype.h” başlığı, standart C kitaplığının bir parçasıdır. Karakterleri sınıflandırmak ve dönüştürmek için işlevler sağlar.

Bu işlevlerin tümü, EOF olması veya işaretsiz bir karakter olarak gösterilebilmesi gereken bir “int” parametresi alır.

Sınıflandırma işlevlerinin adlarının önüne ‘is’ eklenir. Her biri, kendisine iletilen karakter ilgili koşulu sağlıyorsa, sıfır olmayan bir tamsayı (DOĞRU) döndürür. Koşul sağlanmazsa, işlev sıfır değeri (YANLIŞ) döndürür.

Bu sınıflandırma işlevleri, varsayılan C yerel ayarının varsayıldığı şekilde gösterildiği gibi çalışır:

int a;
int c = 'A';
a = isalpha(c); /* Checks if c is alphabetic (A-Z, a-z), returns non-zero here. */
a = isalnum(c); /* Checks if c  is alphanumeric (A-Z, a-z, 0-9), returns non-zero here. */
a = iscntrl(c); /* Checks is c is a control character (0x00-0x1F, 0x7F), returns zero here. */
a = isdigit(c); /* Checks if c is a digit (0-9), returns zero here. */
a = isgraph(c); /* Checks if c has a graphical representation (any printing character except space), returns non-zero here. */
a = islower(c); /* Checks if c is a lower-case letter (a-z), returns zero here. */
a = isprint(c); /* Checks if c is any printable character (including space), returns non-zero here. */
a = isupper(c); /* Checks if c is a upper-case letter (a-z), returns zero here. */
a = ispunct(c); /* Checks if c is a punctuation character, returns zero here. */
a = isspace(c); /* Checks if c is a white-space character, returns zero here. */
a = isupper(c); /* Checks if c is an upper-case letter (A-Z), returns non-zero here. */
a = isxdigit(c); /* Checks if c is a hexadecimal digit (A-F, a-f, 0-9), returns non-zero here. */

a = isblank(c); /* Checks if c is a blank character (space or tab), returns non-zero here. */
son sürüm

İki dönüştürme işlevi vardır. Bunlar ’to’ öneki kullanılarak adlandırılır. Bu işlevler, yukarıdakilerle aynı argümanı alır. Ancak dönüş değeri basit bir sıfır veya sıfır olmayan bir değer değildir, ancak iletilen argüman bir şekilde değişmiştir.

Bu dönüştürme işlevleri, varsayılan C yerel ayarının varsayıldığı şekilde gösterildiği gibi çalışır:

int a;
int c = 'A';

/* Converts c to a lower-case letter (a-z). 
 * If conversion is not possible the unchanged value is returned. 
 * Returns 'a' here. 
 */
a = tolower(c);

/* Converts c to an upper-case letter (A-Z). 
 * If conversion is not possible the unchanged value is returned. 
 * Returns 'A' here. 
 */
a = toupper(c);

Bir akıştan okunan karakterleri sınıflandırma #

#include <ctype.h>
#include <stdio.h>

typedef struct {
  size_t space;
  size_t alnum;
  size_t punct;
} chartypes;

chartypes classify(FILE *f) {
  chartypes types = { 0, 0, 0 };
  int ch;

  while ((ch = fgetc(f)) != EOF) {
    types.space += !!isspace(ch);
    types.alnum += !!isalnum(ch);
    types.punct += !!ispunct(ch);
  }

  return types;
}

‘sınıflandırma’ işlevi, bir akıştan karakterleri okur ve boşluk, alfasayısal ve noktalama karakterlerinin sayısını sayar. Birkaç tuzaktan kaçınır.

  • Bir akıştan bir karakter okurken, sonuç bir ‘int’ olarak kaydedilir, aksi takdirde ‘EOF’ (dosya sonu işaretleyici) okuması ile aynı bit modeline sahip bir karakter arasında bir belirsizlik olurdu.
  • Karakter sınıflandırma işlevleri (ör. “isspace”), bağımsız değişkenlerinin “işaretsiz karakter” veya “EOF” makrosunun değeri olarak gösterilebilir olmasını bekler. Bu tam olarak fgetc’nin döndürdüğü şey olduğundan, burada dönüştürmeye gerek yoktur.
  • Karakter sınıflandırma işlevlerinin dönüş değeri yalnızca sıfır (“yanlış” anlamına gelir) ve sıfır olmayan (“doğru” anlamına gelir) arasında ayrım yapar. Oluşma sayısını saymak için bu değerin 1 veya 0’a dönüştürülmesi gerekir, bu çift olumsuzlama !! ile yapılır.

Bir dizeden karakterleri sınıflandırma #

#include <ctype.h>
#include <stddef.h>

typedef struct {
  size_t space;
  size_t alnum;
  size_t punct;
} chartypes;

chartypes classify(const char *s) {
  chartypes types = { 0, 0, 0 };
  const char *p;
  for (p= s; p != '\0'; p++) {
    types.space += !!isspace((unsigned char)*p);
    types.alnum += !!isalnum((unsigned char)*p);
    types.punct += !!ispunct((unsigned char)*p);
  }

  return types;
}

“sınıflandırma” işlevi, bir dizedeki tüm karakterleri inceler ve boşluk, alfasayısal ve noktalama karakterlerinin sayısını sayar. Birkaç tuzaktan kaçınır.

  • Karakter sınıflandırma işlevleri (ör. “isspace”), bağımsız değişkenlerinin “işaretsiz karakter” veya “EOF” makrosunun değeri olarak gösterilebilir olmasını bekler.
  • “*p” ifadesi “char” türündedir ve bu nedenle yukarıdaki ifadeyle eşleşecek şekilde dönüştürülmelidir.
  • “char” türü, “imzalı karakter” veya “imzasız karakter” ile eşdeğer olarak tanımlanmıştır.
  • ‘char’, ‘unsigned char’ ile eşdeğer olduğunda, ‘char’ türünün olası her değeri ‘unsigned char’ olarak gösterilebildiğinden sorun yoktur.
  • ‘char’, ‘signed char’ ile eşdeğer olduğunda, karakter sınıflandırma işlevlerine geçmeden önce ‘unsigned char’a dönüştürülmesi gerekir. Ve bu dönüşüm nedeniyle karakterin değeri değişebilse de, bu tam olarak bu işlevlerin beklediği şeydir.
  • Karakter sınıflandırma işlevlerinin dönüş değeri yalnızca sıfır (“yanlış” anlamına gelir) ve sıfır olmayan (“doğru” anlamına gelir) arasında ayrım yapar. Oluşma sayısını saymak için bu değerin 1 veya 0’a dönüştürülmesi gerekir, bu çift olumsuzlama !! ile yapılır.
