laravel-5.2'ye giriş

  • Kurulum veya Kurulum
  • Laravel 5.1 Framework’ü Ubuntu 16.04, 14.04 ve LinuxMint’e yükleyin

Bu sayfada

  • Kurulum veya Kurulum
  • Laravel 5.1 Framework’ü Ubuntu 16.04, 14.04 ve LinuxMint’e yükleyin

Laravel, paketler, geçişler ve Artisan CLI içeren bir MVC çerçevesidir. Laravel, CodeIgniter, Yii, ASP.NET MVC, Ruby on Rails, Sinatra ve diğerleri gibi çerçevelerin en iyi özelliklerinin çoğunu içeren sağlam bir araç seti ve bir uygulama mimarisi sunar. Laravel bir Açık Kaynak çerçevesidir. Web Geliştirme hızını artıracak çok zengin özelliklere sahiptir. Core PHP ve Advanced PHP ile aşina iseniz, Laravel işinizi kolaylaştıracaktır. Çok zaman kazandıracak.

Bu bölüm, laravel-5.1’in ne olduğuna ve bir geliştiricinin onu neden kullanmak isteyebileceğine ilişkin bir genel bakış sağlar.

Ayrıca laravel-5.1 içindeki herhangi bir büyük konudan bahsetmeli ve ilgili konulara bağlantı vermelidir. laravel-5.1 için Belgeler yeni olduğundan, bu ilgili konuların ilk sürümlerini oluşturmanız gerekebilir.

Kurulum veya Kurulum #

Bir Linux/Mac/Unix Makinesine Laravel 5.1 kurulumuna ilişkin talimatlar.

Kuruluma başlamadan önce aşağıdaki gereksinimlerin karşılanıp karşılanmadığını kontrol edin:

  • PHP >= 5.5.9
  • OpenSSL PHP Uzantısı -PDO PHP Uzantısı
  • Mbstring PHP Uzantısı
  • Tokenizer PHP Uzantısı

Kuruluma başlayalım:

  1. Besteci yükleyin. Besteci Belgeleri
  2. composer create-project laravel/laravel <klasör-adı> "5.1.*" komutunu çalıştırın
  3. storage klasörünün ve bootstrap/cache klasörünün yazılabilir olduğundan emin olun.
  4. .env dosyasını açın ve veritabanı kimlik bilgileri, hata ayıklama durumu, uygulama ortamı vb. gibi yapılandırma bilgilerini ayarlayın.
  5. “php artisan serve"yi çalıştırın ve tarayıcınızı “http://localhost:8000"e yönlendirin. Her şey yolundaysa o zaman sayfayı almalısın

Laravel 5.1 Framework’ü Ubuntu 16.04, 14.04 ve LinuxMint’e yükleyin #

Adım 1 – LAMP’yi kurun

Laravel ile başlamak için önce çalışan bir LAMP sunucusu kurmamız gerekiyor. Zaten LAMP yığını çalıştırıyorsanız, bu adımı atlayın, aksi takdirde Ubuntu sisteminde lambayı kurmak için aşağıdaki komutları kullanın.

PHP 5.6’yı yükleyin

$ sudo apt-get install python-software-properties
$ sudo add-apt-repository ppa:ondrej/php
$ sudo apt-get update
$ sudo apt-get install -y php5.6 php5.6-mcrypt php5.6-gd

Apache2’yi yükleyin

$ apt-get install apache2 libapache2-mod-php5

MySQL’i yükleyin

$ apt-get install mysql-server php5.6-mysql

Adım 2 – Composer’ı yükleyin

Laravel bağımlılıklarını kurmak için Composer gereklidir. Bu yüzden sistemimizde komut olarak indirip kullanmak için aşağıdaki komutları kullanın.

$ curl -sS https://getcomposer.org/installer | php
$ sudo mv composer.phar /usr/local/bin/composer
$ sudo chmod +x /usr/local/bin/composer

Adım 3 – Laravel’i kurun

Laravel’in en son sürümünü indirmek için, github’dan laravel’in ana deposunu klonlamak için aşağıdaki komutu kullanın.

$ cd /var/www
$ git clone https://github.com/laravel/laravel.git

Laravel kod dizinine gidin ve Laravel çerçevesi için gereken tüm bağımlılıkları kurmak için besteciyi kullanın.

$ cd /var/www/laravel
$ sudo composer install

Bağımlılıkların yüklenmesi biraz zaman alacaktır. Dosyalara uygun izinleri ayarladıktan sonra.

$ chown -R www-data.www-data /var/www/laravel
$ chmod -R 755 /var/www/laravel
$ chmod -R 777 /var/www/laravel/app/storage

Adım 4 – Şifreleme Anahtarını Ayarlayın

Şimdi, Illuminate şifreleme hizmeti tarafından kullanılan 32 bit uzunluğundaki rastgele sayı şifreleme anahtarını ayarlayın.

$ php artisan key:generate

Application key [uOHTNu3Au1Kt7Uloyr2Py9blU0J5XQ75] set successfully.

Şimdi config/app.php konfigürasyon dosyasını düzenleyin ve yukarıda oluşturulan uygulama anahtarını aşağıdaki gibi güncelleyin. Ayrıca şifrenin doğru ayarlandığından emin olun.

'key' => env('APP_KEY', 'uOHTNu3Au1Kt7Uloyr2Py9blU0J5XQ75'),

'cipher' => 'AES-256-CBC',

Adım 5 – Apache VirtualHost oluşturun

Şimdi web tarayıcısından Laravel çerçevesine erişmek için Apache yapılandırma dosyanıza bir Sanal Ana Bilgisayar ekleyin. /etc/apache2/sites-available/ dizini altında Apache yapılandırma dosyası oluşturun ve aşağıdaki içeriği ekleyin.

$ vim /etc/apache2/sites-available/laravel.example.com.conf

Bu, Sanal Ana Bilgisayar dosya yapısıdır.

<VirtualHost *:80>

        ServerName laravel.example.com
        DocumentRoot /var/www/laravel/public

        <Directory />
                Options FollowSymLinks
                AllowOverride None
        </Directory>
        <Directory /var/www/laravel>
                AllowOverride All
        </Directory>

        ErrorLog ${APACHE_LOG_DIR}/error.log
        LogLevel warn
        CustomLog ${APACHE_LOG_DIR}/access.log combined
</VirtualHost>

Son olarak, web sitesini etkinleştirelim ve aşağıdaki komutu kullanarak Apache hizmetini yeniden yükleyelim.

$ a2ensite laravel.example.com
$ sudo service apache2 reload

Adım 6 – Laravel’e Erişim

Bu noktada sisteminizde Laravel 5 PHP frameworkünü başarıyla tamamladınız. Şimdi web tarayıcısında Laravel uygulamanıza erişmek için ana bilgisayar dosyası girişi yapın. ‘127.0.0.1’i sunucu ipinizle ve laravel.example.com’u Apache’de yapılandırılmış alan adınızla değiştirin.

$ sudo echo "127.0.0.1  laravel.example.com" >> /etc/hosts
