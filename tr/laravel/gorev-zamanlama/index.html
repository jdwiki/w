Görev oluşturma #

Artisan kullanarak Laravel’de bir görev (Konsol Komutu) oluşturabilirsiniz. Komut satırınızdan:

php artisan make:console MyTaskName

Bu, dosyayı app/Console/Commands/MyTaskName.php içinde oluşturur. Bunun gibi görünecek:

<?php

namespace App\Console\Commands;

use Illuminate\Console\Command;

class MyTaskName extends Command
{
    /**
     * The name and signature of the console command.
     *
     * @var string
     */
    protected $signature = 'command:name';

    /**
     * The console command description.
     *
     * @var string
     */
    protected $description = 'Command description';

    /**
     * Create a new command instance.
     *
     * @return void
     */
    public function __construct()
    {
        parent::__construct();
    }

    /**
     * Execute the console command.
     *
     * @return mixed
     */
    public function handle()
    {
        //
    }
}

Bu tanımın bazı önemli kısımları şunlardır:

  • “$signature” özelliği, komutunuzu tanımlayan şeydir. Bu komutu daha sonra php artisan command:name komutunu çalıştırarak Artisan kullanarak komut satırından çalıştırabileceksiniz (command:name komutunuzun $signature ile eşleştiği yerde)
  • $description özelliği, kullanılabilir olduğunda komutunuzun yanında Artisan’ın yardım/kullanımı görüntülenir.
  • handle() yöntemi, komutunuz için kodu yazdığınız yerdir.

Sonunda, göreviniz Artisan aracılığıyla komut satırına sunulacaktır. Bu sınıftaki protected $signature = 'command:name'; özelliği, onu çalıştırmak için kullanacağınız şeydir.

Bir görevi kullanılabilir hale getirme #

app/Console/Kernel.php dosyasında bir görevi Artisan ve uygulamanız için kullanılabilir hale getirebilirsiniz.

“Kernel” sınıfı, komutlarınızı uygulamanız için kullanılabilir hale getiren “$commands” adlı bir dizi içerir.

Artisan ve uygulamanız için kullanılabilir hale getirmek için komutunuzu bu diziye ekleyin.

<?php

namespace App\Console;

use Illuminate\Console\Scheduling\Schedule;
use Illuminate\Foundation\Console\Kernel as ConsoleKernel;

class Kernel extends ConsoleKernel
{
    /**
     * The Artisan commands provided by your application.
     *
     * @var array
     */
    protected $commands = [
        Commands\Inspire::class,
        Commands\MyTaskName::class // This line makes MyTaskName available
    ];

    /**
     * Define the application's command schedule.
     *
     * @param  \Illuminate\Console\Scheduling\Schedule  $schedule
     * @return void
     */
    protected function schedule(Schedule $schedule)
    {
    
    }
}

Bu yapıldıktan sonra, artık Artisan’ı kullanarak komut satırı üzerinden komutunuza erişebilirsiniz. Komutunuzun ‘$signature’ özelliğinin ‘my:task’ olarak ayarlandığını varsayarsak, görevinizi yürütmek için aşağıdaki komutu çalıştırabilirsiniz:

php artisan my:task

Görevinizi zamanlama #

Komutunuz uygulamanız için kullanıma sunulduğunda, tıpkı bir CRON’da yaptığınız gibi önceden tanımlanmış aralıklarla çalışacak şekilde programlamak için Laravel’i kullanabilirsiniz.

app/Console/Kernel.php dosyasında, görevinizi planlamak için kullanabileceğiniz bir “program” yöntemi bulacaksınız.

<?php

namespace App\Console;

use Illuminate\Console\Scheduling\Schedule;
use Illuminate\Foundation\Console\Kernel as ConsoleKernel;

class Kernel extends ConsoleKernel
{
    /**
     * The Artisan commands provided by your application.
     *
     * @var array
     */
    protected $commands = [
        Commands\Inspire::class,
        Commands\MyTaskName::class
    ];

    /**
     * Define the application's command schedule.
     *
     * @param  \Illuminate\Console\Scheduling\Schedule  $schedule
     * @return void
     */
    protected function schedule(Schedule $schedule)
    {
        $schedule->command('my:task')->everyMinute();
        // $schedule->command('my:task')->everyFiveMinutes();
        // $schedule->command('my:task')->daily();
        // $schedule->command('my:task')->monthly();
        // $schedule->command('my:task')->sundays();
    }
}

Görevinizin ‘$imza’sının ‘benim:görevim’ olduğunu varsayarak, ‘Schedule $schedule’ nesnesini kullanarak yukarıda gösterildiği gibi zamanlayabilirsiniz. Laravel, yukarıdaki açıklama satırlarında gösterildiği gibi, komutunuzu programlamak için birçok farklı yol sunar.

Zamanlayıcıyı çalışacak şekilde ayarlama #

Zamanlayıcı şu komut kullanılarak çalıştırılabilir:

php artisan schedule:run

Zamanlayıcının doğru çalışması için her dakika çalıştırılması gerekir. Bunu, zamanlayıcıyı her dakika arka planda çalıştıran aşağıdaki satırla bir cron işi oluşturarak ayarlayabilirsiniz.

* * * * * php /path/to/artisan schedule:run >> /dev/null 2>&1
