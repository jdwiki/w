2D Sprite Animasyonu #

Sprite animasyonu, mevcut bir resim veya kare dizisini göstermekten oluşur.

Önce bir dizi görüntüyü varlık klasörüne içe aktarın. Ya sıfırdan bazı görüntüler oluşturun ya da bazılarını Asset Store’dan indirin. (Bu örnek, bu ücretsiz öğeyi kullanır.)

Tek bir animasyonun her bir görüntüsünü varlıklar klasöründen sahne görünümüne sürükleyin. Unity, yeni animasyon klibini adlandırmak için bir iletişim kutusu gösterecektir.

Bu, aşağıdakiler için kullanışlı bir kısayoldur:

  • yeni oyun nesneleri yaratmak
  • iki bileşen atama (bir Sprite Renderer ve bir Animatör)
  • animasyon denetleyicileri oluşturma (ve yeni Animator bileşenini bunlara bağlama)
  • seçilen karelerle animasyon klipleri oluşturma

Oynat’ı tıklayarak animasyon sekmesinde oynatmayı önizleyin:

Aynı oyun nesnesi için yeni animasyonlar oluşturmak ve ardından yeni oyun nesnesini ve animasyon denetleyicisini silmek için aynı yöntem kullanılabilir. Yeni animasyon klibini, 3B animasyonda olduğu gibi o nesnenin animasyon denetleyicisine ekleyin.

Koşu için Temel Animasyon #

Bu kod, Unity’de basit bir animasyon örneği gösterir.

Bu örnek için 2 animasyon klibiniz olmalıdır; Çalıştır ve Boşta. Bu animasyonlar Yerinde Durma hareketleri olmalıdır. Animasyon klipleri seçildiğinde, bir Animatör Denetleyicisi oluşturun. Bu Denetleyiciyi canlandırmak istediğiniz oyuncuya veya oyun nesnesine ekleyin.

Animatör penceresini Windows seçeneğinden açın. 2 animasyon klibini Animatör penceresine sürükleyin ve 2 durum oluşturulacaktır. Oluşturduktan sonra, Her ikisi de bool olarak 2 parametre eklemek için sol parametreler sekmesi. Birini “PerformRun” ve diğerini “PerformIdle” olarak adlandırın. “PerformIdle” öğesini true olarak ayarlayın.

Boşta durumundan Çalıştır’a ve Çalıştır’dan boşta durumuna geçiş yapın (Resme bakın). Boşta->Çalıştır geçişine tıklayın ve Denetçi penceresinde HasExit’in seçimini kaldırın. Aynısını diğeri için yap geçiş. Boşta->Çalıştır geçişi için bir koşul ekleyin: PerformIdle. Run->Idle için bir koşul ekleyin: PerformRun. Aşağıda verilen C# betiğini oyun nesnesine ekleyin. Yukarı tuşu ile animasyonlu çalışmalı, Sol ve Sağ tuşları ile dönmelidir.

using UnityEngine;
using System.Collections;

public class RootMotion : MonoBehaviour {

//Public Variables
[Header("Transform Variables")]
public float RunSpeed = 0.1f;
public float TurnSpeed = 6.0f;


Animator animator;

void Start()
{
    /**
    * Initialize the animator that is attached on the current game object i.e. on which you will attach this script.
    */
    animator = GetComponent<Animator>();
}

void Update()
{
    /**
    * The Update() function will get the bool parameters from the animator state machine and set the values provided by the user.
    * Here, I have only added animation for Run and Idle. When the Up key is pressed, Run animation is played. When we let go, Idle is played.
    */
    
    if (Input.GetKey (KeyCode.UpArrow)) {
            animator.SetBool ("PerformRun", true);
            animator.SetBool ("PerformIdle", false);
        } else {
            animator.SetBool ("PerformRun", false);
            animator.SetBool ("PerformIdle", true);
        }
}

void OnAnimatorMove()
    {
        /**
         * OnAnimatorMove() function will shadow the "Apply Root Motion" on the animator. Your game objects psoition will now be determined 
         * using this fucntion.
         */
        if (Input.GetKey (KeyCode.UpArrow)){
            transform.Translate (Vector3.forward * RunSpeed);
            if (Input.GetKey (KeyCode.RightArrow)) {
                transform.Rotate (Vector3.up * Time.deltaTime * TurnSpeed);
            }
            else if (Input.GetKey (KeyCode.LeftArrow)) {
                transform.Rotate (-Vector3.up * Time.deltaTime * TurnSpeed);
            }
        }

    }


   
}

Animasyon Klipleri Oluşturma ve Kullanma #

Bu örnek, oyun nesneleri veya oyuncular için animasyon kliplerinin nasıl oluşturulacağını ve kullanılacağını gösterecektir.

Bu örnekte kullanılan modellerin Unity Asset Store’dan indirildiğini unutmayın. Oynatıcı şu bağlantıdan indirildi: https://www.assetstore.unity3d.com/en/#!/content/21874.

Animasyon oluşturmak için önce Animasyon Penceresini açın. Pencere ve Animasyon Seç’e tıklayarak veya Ctrl+6 tuşlarına basarak açabilirsiniz. Hangi oyun nesnesini seçin Hiyerarşi Penceresinden animasyon klibini uygulamak istiyorsanız, ardından Animasyon Penceresinde Oluştur düğmesine tıklayın.

Animasyonunuza bir ad verin (IdlePlayer, SprintPlayer, DyingPlayer vb.) ve kaydedin. Şimdi, Animasyon Penceresinden Özellik Ekle düğmesine tıklayın. Bu, oyun nesnesinin veya oyuncunun özelliğini zamana göre değiştirmenize olanak tanır. Bu, döndürme, konum ve ölçek gibi Dönüşümleri ve oyun nesnesine ekli diğer özellikleri içerebilir; Çarpıştırıcı, Mesh Oluşturucu vb.

Oyun nesnesi için çalışan bir animasyon oluşturmak için insansı bir 3D modele ihtiyacınız olacak. Modeli yukarıdaki linkten indirebilirsiniz. oluşturmak için yukarıdaki adımları izleyin yeni bir animasyon. Bir Dönüştürme özelliği ekleyin ve karakter ayaklarından biri için Döndür’ü seçin.

Şu anda, oyun nesnesi özelliğindeki Oynat düğmeniz ve Döndürme değerleriniz kırmızıya dönecekti. Döndürme X, Y ve Z değerlerini görmek için açılır oka tıklayın. Varsayılan animasyon süresi 1 saniye olarak ayarlanmıştır. Animasyonlar enterpolasyon yapmak için anahtar kareleri kullanır değerler arasında. Canlandırmak için, zaman içinde farklı noktalarda anahtarlar ekleyin ve Denetçi Penceresinden dönüş değerlerini değiştirin. Örneğin dönme değeri zaman 0.0s 0.0 olabilir. 0.5s zamanında değer X için 20.0 olabilir. 1.0s zamanında değer 0.0 olabilir. 1.0s’de animasyonumuzu sonlandırabiliriz.

Animasyon uzunluğunuz, Animasyona eklediğiniz son Tuşlara bağlıdır. Enterpolasyonu daha pürüzsüz hale getirmek için daha fazla anahtar ekleyebilirsiniz.

Animasyon Eğrileri #

Animasyon eğrileri, animasyon oynatılırken bir kayan nokta parametresini değiştirmenize olanak tanır. Örneğin, bir animasyon varsa 60 saniye uzunluğunda ve bir kayan nokta değeri/parametresi istiyorsunuz, buna X deyin, animasyonda değişiklik yapın (animasyon zamanında olduğu gibi = 0.0s; X = 0.0 , animasyon zamanında = 30.0s; X = 1.0, animasyon zamanında = 60.0s; X = 0.0).

Float değerine sahip olduğunuzda, onu çevirmek, döndürmek, ölçeklemek veya başka herhangi bir şekilde kullanmak için kullanabilirsiniz.

Örneğimde, çalışan bir oyuncu oyun nesnesini göstereceğim. Çalıştırma animasyonu oynatıldığında, oyuncunun çeviri hızı animasyon ilerledikçe artmalıdır. Animasyon sonuna ulaştığında, çeviri hızı düşmelidir.

Çalışan bir animasyon klibi oluşturdum. Klibi seçin ve ardından denetçi penceresinde Düzenle’ye tıklayın.

Bir kez orada, Eğriler’e ilerleyin. Bir eğri eklemek için + işaretine tıklayın. Eğriyi adlandırın, ör. İleriRunEğrisi. Minyatür eğriye tıklayın doğru. İçinde varsayılan bir eğri bulunan küçük bir pencere açacaktır.

Yükseldiği ve sonra düştüğü parabolik şekilli bir eğri istiyoruz. Varsayılan olarak, çizgide 2 nokta vardır. Eğriyi çift tıklayarak daha fazla nokta ekleyebilirsiniz. Aşağıdakine benzer bir şekil oluşturmak için noktaları sürükleyin.

Animatör Penceresinde, çalışan klibi ekleyin. Ayrıca, ForwardRunCurve gibi eğriyle aynı ada sahip bir kayan nokta parametresi ekleyin.

Animasyon oynatıldığında, kayan değer eğriye göre değişecektir. Aşağıdaki kod, kayan değerin nasıl kullanılacağını gösterecektir:

using UnityEngine;
using System.Collections;

public class RunAnimation : MonoBehaviour {

Animator animator;
float curveValue;

void Start()
{
    animator = GetComponent<Animator>();
}

void Update()
{
    curveValue = animator.GetFloat("ForwardRunCurve");

    transform.Translate (Vector3.forward * curveValue);
}

}

eğriValue değişkeni, herhangi bir zamanda eğrinin(ForwardRunCruve) değerini tutar. Bu değeri çevirinin hızını değiştirmek için kullanıyoruz. Bu komut dosyasını oyuncu oyun nesnesine ekleyebilirsiniz.
