Okuma Tuşuna Basma ve GetKey, GetKeyDown ve GetKeyUp arasındaki fark #

Giriş, Güncelleme işlevinden okunmalıdır.

Mevcut tüm Keycode enum için referans.

1. Input.GetKey ile okuma tuşuna basın:

“Input.GetKey”, kullanıcı belirtilen anahtarı basılı tutarken tekrar tekrar “true” değerini döndürür. Bu, belirtilen tuşu basılı tutarken bir silahı tekrar ateşlemek için kullanılabilir. Aşağıda, Space tuşu basılı tutulduğunda merminin otomatik olarak ateşlenmesine bir örnek verilmiştir. Oyuncunun tuşa tekrar tekrar basıp bırakması gerekmez.

public GameObject bulletPrefab;
public float shootForce = 50f;

void Update()
{
    if (Input.GetKey(KeyCode.Space))
    {
        Debug.Log("Shooting a bullet while SpaceBar is held down");

        //Instantiate bullet
        GameObject bullet = Instantiate(bulletPrefab, transform.position, transform.rotation) as GameObject;

        //Get the Rigidbody from the bullet then add a force to the bullet
        bullet.GetComponent<Rigidbody>().AddForce(bullet.transform.forward * shootForce);
    }
}

2.Input.GetKeyDown ile okuma tuşuna basın:

Input.GetKeyDown, belirtilen tuşa basıldığında yalnızca bir kez doğru olacaktır. “Input.GetKey” ve “Input.GetKeyDown” arasındaki temel fark budur. Kullanımının bir örnek kullanımı, bir kullanıcı arayüzünü veya el fenerini veya bir öğeyi açıp kapatmaktır.

public Light flashLight;
bool enableFlashLight = false;

void Update()
{
    if (Input.GetKeyDown(KeyCode.Space))
    {
        //Toggle Light 
        enableFlashLight = !enableFlashLight;
        if (enableFlashLight)
        {
            flashLight.enabled = true;
            Debug.Log("Light Enabled!");
        }
        else
        {
            flashLight.enabled = false;
            Debug.Log("Light Disabled!");
        }
    }
}

3.Input.GetKeyUp ile okuma tuşuna basın:

Bu, “Input.GetKeyDown"ın tam tersidir. Tuşa basıldığında/kaldırıldığında algılamak için kullanılır. Tıpkı “Input.GetKeyDown” gibi, yalnızca bir kez “true” değerini döndürür. Örneğin, Input.GetKeyDown tuşu basılı tutulduğunda ışığı etkinleştirebilir, ardından Input.GetKeyUp ile tuş bırakıldığında ışığı devre dışı bırakabilirsiniz.

public Light flashLight;
void Update()
{
    //Disable Light when Space Key is pressed
    if (Input.GetKeyDown(KeyCode.Space))
    {
        flashLight.enabled = true;
        Debug.Log("Light Enabled!");
    }

    //Disable Light when Space Key is released
    if (Input.GetKeyUp(KeyCode.Space))
    {
        flashLight.enabled = false;
        Debug.Log("Light Disabled!");
    }
}

Fare Düğmesini Oku (Sol, Orta, Sağ) Tıklamalar #

Bu işlevler, Fare Düğmesi Tıklamalarını kontrol etmek için kullanılır.

  • Input.GetMouseButton(int düğmesi);
  • Input.GetMouseButtonDown(int düğmesi);
  • Input.GetMouseButtonUp(int düğmesi);

Hepsi aynı parametreyi alır.

  • 0 = Sol Fare Tıklaması.
  • 1 = Sağ Fare Tıklaması.
  • 2 = Orta Fare Tıklaması.

“GetMouseButton”, fare düğmesinin sürekli basılı tutulduğunu algılamak için kullanılır. Belirtilen fare düğmesi basılı tutulurken ’true’ değerini döndürür.

void Update()
{
    if (Input.GetMouseButton(0))
    {
        Debug.Log("Left Mouse Button Down");
    }

    if (Input.GetMouseButton(1))
    {
        Debug.Log("Right Mouse Button Down");
    }

    if (Input.GetMouseButton(2))
    {
        Debug.Log("Middle Mouse Button Down");
    }
}

GetMouseButtonDown fare tıklaması olduğunda algılamak için kullanılır. Bir kez basılırsa ’true’ değerini döndürür. Fare düğmesi bırakılıp tekrar basılana kadar tekrar ’true’ döndürmez.

void Update()
{
    if (Input.GetMouseButtonDown(0))
    {
        Debug.Log("Left Mouse Button Clicked");
    }

    if (Input.GetMouseButtonDown(1))
    {
        Debug.Log("Right Mouse Button Clicked");
    }

    if (Input.GetMouseButtonDown(2))
    {
        Debug.Log("Middle Mouse Button Clicked");
    }
}

GetMouseButtonUp, belirtilen fare düğmesinin ne zaman bırakıldığını algılamak için kullanılır. Bu, yalnızca belirtilen fare düğmesi bırakıldığında ’true’ değerini döndürür. Tekrar true döndürmek için tekrar basılmalı ve bırakılmalıdır.

void Update()
{
    if (Input.GetMouseButtonUp(0))
    {
        Debug.Log("Left Mouse Button Released");
    }

    if (Input.GetMouseButtonUp(1))
    {
        Debug.Log("Right Mouse Button Released");
    }

    if (Input.GetMouseButtonUp(2))
    {
        Debug.Log("Middle Mouse Button Released");
    }
}

İvmeölçer Sensörünü Oku (Temel) #

İvmeölçer sensörünü okumak için Input.acceleration kullanılır. 3B alanda ‘x’, ‘y’ ve ‘z’ eksen değerlerini içeren bir sonuç olarak ‘Vector3’ döndürür.

void Update()
{
    Vector3 acclerometerValue = rawAccelValue();
    Debug.Log("X: " + acclerometerValue.x + "  Y: " + acclerometerValue.y + "  Z: " + acclerometerValue.z);
}

Vector3 rawAccelValue()
{
    return Input.acceleration;
}

İvmeölçer Sensörünü Oku (İleri) #

Bir GameObject’i hareket ettirmek veya döndürmek için doğrudan ivmeölçer sensöründen gelen ham değerleri kullanmak, sarsıntılı hareketler veya titreşimler gibi sorunlara neden olabilir. Kullanmadan önce değerleri düzeltmeniz önerilir. Aslında, ivmeölçer sensöründen gelen değerler, kullanımdan önce daima düzeltilmelidir. Bu, düşük geçişli bir filtre ile gerçekleştirilebilir ve Vector3.Lerp devreye girdiği yer burasıdır.

//The lower this value, the less smooth the value is and faster Accel is updated. 30 seems fine for this
const float updateSpeed = 30.0f;

float AccelerometerUpdateInterval = 1.0f / updateSpeed;
float LowPassKernelWidthInSeconds = 1.0f;
float LowPassFilterFactor = 0;
Vector3 lowPassValue = Vector3.zero;

void Start()
{
    //Filter Accelerometer
    LowPassFilterFactor = AccelerometerUpdateInterval / LowPassKernelWidthInSeconds;
    lowPassValue = Input.acceleration;
}

void Update()
{

    //Get Raw Accelerometer values (pass in false to get raw Accelerometer values)
    Vector3 rawAccelValue = filterAccelValue(false);
    Debug.Log("RAW X: " + rawAccelValue.x + "  Y: " + rawAccelValue.y + "  Z: " + rawAccelValue.z);

    //Get smoothed Accelerometer values (pass in true to get Filtered Accelerometer values)
    Vector3 filteredAccelValue = filterAccelValue(true);
    Debug.Log("FILTERED X: " + filteredAccelValue.x + "  Y: " + filteredAccelValue.y + "  Z: " + filteredAccelValue.z);
}

//Filter Accelerometer
Vector3 filterAccelValue(bool smooth)
{
    if (smooth)
        lowPassValue = Vector3.Lerp(lowPassValue, Input.acceleration, LowPassFilterFactor);
    else
        lowPassValue = Input.acceleration;

    return lowPassValue;
}

İvmeölçer Sensörünü Oku (Hassas) #

İvmeölçer Sensörünü hassas bir şekilde okuyun.

Bu örnek bellek ayırır:

void Update()
{
    //Get Precise Accelerometer values 
    Vector3 accelValue = preciseAccelValue();
    Debug.Log("PRECISE X: " + accelValue.x + "  Y: " + accelValue.y + "  Z: " + accelValue.z);

}

Vector3 preciseAccelValue()
{
    Vector3 accelResult = Vector3.zero;
    foreach (AccelerationEvent tempAccelEvent in Input.accelerationEvents)
    {
        accelResult = accelResult + (tempAccelEvent.acceleration * tempAccelEvent.deltaTime);
    }
    return accelResult;
}

Bu örnek bellek ayırmaz:

void Update()
{
    //Get Precise Accelerometer values 
    Vector3 accelValue = preciseAccelValue();
    Debug.Log("PRECISE X: " + accelValue.x + "  Y: " + accelValue.y + "  Z: " + accelValue.z);

}

Vector3 preciseAccelValue()
{
    Vector3 accelResult = Vector3.zero;
    for (int i = 0; i < Input.accelerationEventCount; ++i)
    {
        AccelerationEvent tempAccelEvent = Input.GetAccelerationEvent(i);
        accelResult = accelResult + (tempAccelEvent.acceleration * tempAccelEvent.deltaTime);
    }
    return accelResult;
}

Bunun filtrelenmediğini unutmayın. Gürültüyü gidermek için ivmeölçer değerlerinin nasıl düzgünleştirileceğini öğrenmek için lütfen buraya bakın.
