İlgili Kayıtlardan Veri Arama

Belirli bir kaydı işlerken, ilgili kayıtlardan birinden veri almanız gerekecektir. Örneğin, belirli bir Satış Siparişi ile çalışırken ilgili Satış Temsilcisinden veri almanız gerekebilir. SuiteScript terminolojisinde buna arama denir.

Arama işlevi, SuiteScript 1.0’daki ’nlapiLookupField’ global işlevi ve SuiteScript 2.0’daki ‘N/search’ modülünün ’lookupFields’ yöntemi tarafından sağlanır.

Sözdizimi #

  • nlapiLookupField(recordType, recordId, sütunlar);

Parametreler #

parametre Ayrıntılar
kayıtTürü “Dize” - Aranan kayıt türünün dahili kimliği (ör. “satış siparişi”, “çalışan”)
kayıtKimliği “Dize” veya “Sayı” - Aranan kaydın dahili kimliği
sütunlar “Dize” veya “Dize[]” - Kayıttan alınacak alanların listesi. Alan Kimliklerine, Kayıt Tarayıcısının “Arama Sütunları” bölümünden başvurulabilir. Birleştirilen alanlar nokta sözdizimi kullanılarak alınabilir (ör. salesrep.email)

Verim #

Arama, sonuç için tek bir kaydın dahili kimliğine filtre uygulayan bir arama gerçekleştirmenin kısa yoludur. Başlık altında, aramalar aslında bir arama gerçekleştirir, bu nedenle performans, tek bir kayıt döndüren bir aramaya benzer olacaktır.

Bu aynı zamanda bir aramanın aynı bilgiyi almak için kaydı yüklemekten daha hızlı gerçekleştireceği anlamına gelir.

Sınırlamalar #

Aramalar yalnızca gövde alanı verilerini almak için kullanılabilir. Bir arama kullanarak ilgili kaydın alt listelerinden veri alamazsınız. Alt liste verilerine ihtiyacınız varsa, ya bir arama yapmanız ya da ilgili kaydı yüklemeniz gerekecektir.

[1.0] Birleştirilmiş Alanları Ara #

/**
 * An example of nlapiLookupField to retrieve joined fields from a related record
 */

var repId = nlapiGetFieldValue("salesrep");

// Retrieve multiple fields from the associated Sales Rep
var repData = nlapiLookupField("employee", repId, ["email", "firstname", "department.name"]);

console.log(repData);
console.log(repData.firstname + "'s email address is " + repData.email);
console.log(repData.firstname + "'s department is " + repData["department.name"]);

 */

// Get the Sales Rep record ID

[2.0] Tek Alan Ara #

require(["N/search", "N/currentRecord"], function (s, cr) {

    /**
     * An example of N/search#lookupFields to retrieve a single field from a related record
     */
    (function () {

        var record = cr.get();

        // Get the Sales Rep record ID
        var repId = record.getValue({
            "fieldId": "salesrep"
        });

        // Get the name of the Sales Rep
        var repName = record.getText({
            "fieldId": "salesrep"
        });

        // Retrieve the email address from the associated Sales Rep
        var repData = s.lookupFields({
            "type": "employee",
            "id": repId,
            "columns": ["email"]
        });

        console.log(repData);
        console.log(repName + "'s email address is " + repData.email);
    })();
});
