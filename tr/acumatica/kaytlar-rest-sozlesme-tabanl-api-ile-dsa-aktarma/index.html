Kayıtları REST Sözleşme Tabanlı API ile Dışa Aktarma

  • Tek Bir REST Çağrısında Veri Dışa Aktarma
  • Çoklu REST İsteklerinde Sayfalandırma Uygulaması

Bu sayfada

  • Tek Bir REST Çağrısında Veri Dışa Aktarma
  • Çoklu REST İsteklerinde Sayfalandırma Uygulaması

Bu konu, kayıtların Acumatica ERP’den REST Sözleşme Tabanlı API aracılığıyla nasıl dışa aktarılacağını gösterecektir. Acumatica ERP’nin Ekran Tabanlı API’sinin aksine, Sözleşme Tabanlı API, hem SOAP hem de REST arayüzleri sağlar. Sözleşmeye Dayalı API hakkında daha fazla bilgi için bkz. Acumatica ERP Belgeleri

Acumatica ERP’nin REST Sözleşme Tabanlı API’si ile iletişim kurmak için istemci uygulamanız her zaman aşağıdaki 3 adımı gerçekleştirmelidir:

  1. Acumatica ERP örneğine giriş yapın ve kullanıcı oturum bilgileriyle çerez alın

  2. Acumatica ERP örneğinde bulunan Sözleşme Tabanlı API uç noktalarından biriyle etkileşim kurun

  3. Kullanıcı oturumunu kapatmak için Acumatica ERP’den çıkış yapın

Bu konuda sağlanan tüm örnekler, her zaman standart Acumatica ERP kurulum sürecinin bir parçası olarak dağıtılan Varsayılan uç noktası ile oluşturulmuştur. Web Hizmeti Uç Noktaları ekranında (SM.20.70.60) mevcut uç noktaların ayrıntılarını görüntüleyebilir veya Acumatica ERP sözleşme tabanlı web hizmetlerinin özel uç noktalarını yapılandırabilirsiniz:

Acumatica ERP’nin Sözleşmeye Dayalı web hizmetiyle etkileşim kurmak için yukarıdaki tüm örneklerde kullanılan RestService sınıfının uygulaması aşağıda verilmiştir:

public class RestService : IDisposable
{
    private readonly HttpClient _httpClient;
    private readonly string _acumaticaBaseUrl;
    private readonly string _acumaticaEndpointUrl;

    public RestService(string acumaticaBaseUrl, string endpoint,
        string userName, string password,
        string company, string branch)
    {
        _acumaticaBaseUrl = acumaticaBaseUrl;
        _acumaticaEndpointUrl = _acumaticaBaseUrl + "/entity/" + endpoint + "/";
        _httpClient = new HttpClient(
            new HttpClientHandler
            {
                UseCookies = true,
                CookieContainer = new CookieContainer()
            })
        {
            BaseAddress = new Uri(_acumaticaEndpointUrl),
            DefaultRequestHeaders =
            {
                Accept = {MediaTypeWithQualityHeaderValue.Parse("text/json")}
            }
        };

        var str = new StringContent(
            new JavaScriptSerializer()
                .Serialize(
                    new
                    {
                        name = userName,
                        password = password,
                        company = company,
                        branch = branch
                    }),
                    Encoding.UTF8, "application/json");

        _httpClient.PostAsync(acumaticaBaseUrl + "/entity/auth/login", str)
            .Result.EnsureSuccessStatusCode();
    }

    void IDisposable.Dispose()
    {
        _httpClient.PostAsync(_acumaticaBaseUrl + "/entity/auth/logout",
            new ByteArrayContent(new byte[0])).Wait();
        _httpClient.Dispose();
    }

    public string GetList(string entityName)
    {
        var res = _httpClient.GetAsync(_acumaticaEndpointUrl + entityName)
            .Result.EnsureSuccessStatusCode();

        return res.Content.ReadAsStringAsync().Result;
    }

    public string GetList(string entityName, string parameters)
    {
        var res = _httpClient.GetAsync(_acumaticaEndpointUrl + entityName + "?" + parameters)
            .Result.EnsureSuccessStatusCode();

        return res.Content.ReadAsStringAsync().Result;
    }
}

Tek Bir REST Çağrısında Veri Dışa Aktarma #

Bu örnekte, aşağıdaki verileri Acumatica ERP’den REST Sözleşme Tabanlı API aracılığıyla tek bir çağrıda nasıl dışa aktaracağınızı keşfedeceksiniz:

  • uygulamada bulunan tüm stok kalemleri
  • IN türündeki tüm satış siparişi

Acumatica ERP’den kayıtları dışa aktarmanız gerekiyorsa, aşağıdaki URL’yi kullanın: `http://<Acumatica ERP örneği URL’si>/entity/<Uç nokta adı>/<Uç nokta sürümü>/<Üst düzey varlık>'

“<Üst düzey varlık>”, dışa aktaracağınız varlığın adıdır

Tüm stok kalemlerini tek bir REST çağrısında dışa aktarmak için: #

6.00.001 sürümünün Varsayılan uç noktasını kullanarak yerel bir “AcumaticaERP” örneğinden stok öğesi kayıtlarını dışa aktarmak için şu URL’yi kullanmalısınız: http://localhost/AcumaticaERP /entity/Default/6.0.0001/StockItem

Aşağıda, 6.00.001 sürümünün Varsayılan uç noktasına tek bir REST çağrısı göndererek tüm stok kalemlerini dışa aktarmak için C# ile yazılmış örnek kod verilmiştir:

using (RestService rs = new RestService(
    @"http://localhost/AcumaticaERP/", "Default/6.00.001",
    username, password, company, branch))
{
    string stockItems = rs.GetList("StockItem");
}

IN türündeki tüm satış siparişlerini tek bir REST çağrısında dışa aktarmak için: #

6.00.001 sürümünün Varsayılan uç noktasını kullanarak yerel bir “AcumaticaERP” örneğinden IN türündeki satış siparişlerini dışa aktarmak için aşağıdaki URL’yi kullanmanız gerekir. : `http://localhost/AcumaticaERP/entity/Default/6.00.001/SalesOrder?$filter=OrderType eq ‘IN’'

Aşağıda, 6.00.001 sürümünün Varsayılan uç noktasına tek bir REST çağrısı göndererek IN türündeki tüm satış siparişlerini dışa aktarmak için C# ile yazılmış örnek kod verilmiştir. :

using (RestService rs = new RestService(
    @"http://localhost/StackOverflow/", "Default/6.00.001",
    username, password, company, branch))
{
    var parameters = "$filter=OrderType eq 'IN'";
    string inSalesOrders = rs.GetList("SalesOrder", parameters);
}

Çoklu REST İsteklerinde Sayfalandırma Uygulaması #

Bu örnekte, aşağıdaki verileri Acumatica ERP’den REST Sözleşme Tabanlı API aracılığıyla gruplar halinde nasıl dışa aktaracağınızı keşfedeceksiniz:

  • 10 kayıtlık gruplar halinde uygulamada bulunan stok kalemleri
  • 100 kayıtlık gruplar halinde tüm satış siparişleri

Birden fazla REST çağrısı ile 10 kayıtlık gruplar halinde stok kalemlerini dışa aktarmak için: #

6.00.001 sürümünün Varsayılan uç noktasını kullanarak yerel bir “AcumaticaERP” örneğinden ilk 10 stok öğesini dışa aktarmak için şu URL’yi kullanmalısınız: http://localhost/ AcumaticaERP/varlık/Varsayılan/6.0.0001/StockItem?$top=10

Buna göre, 10’dan 20’ye kadar stok kalemi talep etmek için, yukarıdaki URL’yi filter parametresiyle genişletmeniz yeterlidir: `http://localhost/AcumaticaERP/entity/Default/6.0.0001/StockItem?$top=10& $filter=EnvanterKimliği gt ‘<EnvanterKimliği>’'

“”, önceki bir REST çağrısıyla dışa aktarılan son stok öğesinin kimliğidir

Aşağıda, 6.00.001 sürümünün Varsayılan uç noktasına birden çok REST çağrısı göndererek tüm stok kalemlerini 10 kayıtlık gruplar halinde dışa aktarmak için C# ile yazılmış örnek kod verilmiştir:

using (RestService rs = new RestService(
    @"http://localhost/StackOverflow/", "Default/6.00.001",
    username, password, company, branch))
{
    var json = new JavaScriptSerializer();
    string parameters = "$top=10";
    string items = rs.GetList("StockItem", parameters);
    var records = json.Deserialize<List<Dictionary<string, object>>>(items);

    while (records.Count == 10)
    {
        var inventoryID = records[records.Count - 1]["InventoryID"] as Dictionary<string, object>;
        var inventoryIDValue = inventoryID.Values.First();
        string nextParameters = parameters + "&" + 
            "$filter=" + string.Format("InventoryID gt '{0}'", inventoryIDValue);
        items = rs.GetList("StockItem", nextParameters);
        records = json.Deserialize<List<Dictionary<string, object>>>(items);
    }
}

Tüm satış siparişlerini birden fazla REST çağrısıyla 100 kayıtlık gruplar halinde dışa aktarmak için: #

6.00.001 sürümünün Varsayılan uç noktasını kullanarak yerel bir “AcumaticaERP” örneğinden ilk 100 satış siparişini dışa aktarmak için şu URL’yi kullanmalısınız: “http://localhost/ AcumaticaERP/varlık/Varsayılan/6.0.0001/Sipariş?$top=100`

Satış Siparişi varlığının birincil anahtarı Sipariş Türü ve Sipariş Numarası tarafından oluşturulduğundan, bu örnekte *filtre kombinasyonunu kullanacaksınız. * Sipariş Tipi ve Sipariş Numarası alanları için parametreler:

  • SO türünden 100’den 200’e kadar satış siparişi istemek için şu URL’yi kullanmalısınız: `http://localhost/AcumaticaERP/entity/Default/6.0.0001/SalesOrder?$top=100& $filter=OrderType eq ‘SO’ ve OrderNbr gt ‘’’

“”, önceki bir REST çağrısıyla dışa aktarılan son satış siparişinin numarasıdır.

  • buna göre, SO türünün yanındaki ilk 100 satış siparişini istemek için şu URL’yi kullanmalısınız: http://localhost/AcumaticaERP/entity/Default/6.0.0001/SalesOrder?$top =100&$filter=OrderType gt 'SO' ve OrderNbr gt ''

Aşağıda, tüm satış siparişlerini birden çok REST çağrısıyla 100 kayıtlık gruplar halinde 6.00.001 sürümünün Varsayılan uç noktasına dışa aktarmak için C# ile yazılmış örnek kod verilmiştir:

using (RestService rs = new RestService(
    @"http://localhost/StackOverflow/", "Default/6.00.001",
    username, password, company, branch))
{
    var json = new JavaScriptSerializer();
    string parameters = "$top=100";
    string items = rs.GetList("SalesOrder", parameters);
    var records = json.Deserialize<List<Dictionary<string, object>>>(items);

    bool sameOrderType = true;
    while (records.Count > 0 && (records.Count == 100 || !sameOrderType))
    {
        var orderType = records[records.Count - 1]["OrderType"] as Dictionary<string, object>;
        var orderTypeValue = orderType.Values.First();
        var orderNbr = records[records.Count - 1]["OrderNbr"] as Dictionary<string, object>;
        var orderNbrValue = orderNbr.Values.First();

        string nextParameters = parameters + "&" + "$filter=" +
            string.Format("OrderType {0} '{1}'", sameOrderType ? "eq" : "gt", orderTypeValue) + " and " +
            string.Format("OrderNbr gt '{0}'", sameOrderType ? orderNbrValue : "''" );
        items = rs.GetList("SalesOrder", nextParameters);
        records = json.Deserialize<List<Dictionary<string, object>>>(items);
        sameOrderType = records.Count == 100;
    }
}
