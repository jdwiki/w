  • PXStringListAttribute ardılında belirtilen öğeleri değiştirmek için

Bu konuda, PXStringList veya PXIntList özniteliklerinden devralınan alan özniteliklerinin nasıl değiştirileceğini öğreneceksiniz. Gösterilen yaklaşım, özelleştirmelerinizi daha yeni bir Acumatica sürümüne yükseltirken, temel Acumatica ERP ürününün işlevselliğini bozmamasını ve varsa minimum bakım gerektirmemesini sağlayacaktır.

Yukarıdaki tüm örneklerde, hem ‘_AllowedValues’ hem de ‘_AllowedLabels’ dizilerinde değişiklikler yaptınız. Göreviniz bir açılır öğenin yalnızca etiketini (harici değeri) değiştirmekse, Çeviri Sözlüklerini kullanmayı düşünün. Çeviri Sözlükleri hakkında daha fazla bilgi için bkz. Acumatica ERP Belgeleri

Medeni Durumları Değiştirme #

Bu örnekte, Kişiler formunda (CR302000) bulunan Medeni Durum açılır listesini değiştireceksiniz:

PXStringListAttribute halefine yeni öğeler eklemek için #

PXStringList veya PXIntList özniteliğinden devralınan bir öznitelik içinde sabit kodlanmış açılır öğeleri genişletmenin en iyi yolu, özel alan özniteliğinizin yapıcısındaki “_AllowedValues” ve “_AllowedLabels” dizilerinin boyutunu artırmaktır:

[PXLocalizable(Messages.Prefix)]
public static class MaritalStatusesMessages
{
    public const string CommonLaw = "Living common law";
    public const string Separated = "Separated (not living common law)";
    public const string DivorcedNoCommonLaw = "Divorced (not living common law)";
    public const string NeverMarried = "Never Married";
}

public class MaritalStatusesCst1Attribute : MaritalStatusesAttribute
{
    public const string CommonLaw = "L";
    public const string Separated = "P";
    public const string NeverMarried = "N";

    public MaritalStatusesCst1Attribute()
    {
        Array.Resize(ref _AllowedValues, _AllowedValues.Length + 3);
        _AllowedValues[_AllowedValues.Length - 3] = CommonLaw;
        _AllowedValues[_AllowedValues.Length - 2] = Separated;
        _AllowedValues[_AllowedValues.Length - 1] = NeverMarried;
        Array.Resize(ref _AllowedLabels, _AllowedLabels.Length + 3);
        _AllowedLabels[_AllowedLabels.Length - 3] = MaritalStatusesMessages.CommonLaw;
        _AllowedLabels[_AllowedLabels.Length - 2] = MaritalStatusesMessages.Separated;
        _AllowedLabels[_AllowedLabels.Length - 1] = MaritalStatusesMessages.NeverMarried;
    }
}

Yukarıdaki örnekte, Medeni Durum açılır listesine 3 ek öğe eklemek için “_AllowedValues” ve “_AllowedLabels” dizilerinin boyutunu artırdınız. ‘_AllowedValues’ dizisinde saklanan dahili değerler DAC alanlarına atanır ve veritabanına kaydedilir ve ‘_AllowedValues’ dizisindeki harici değerler, kullanıcı arayüzündeki dahili değerleri temsil eder.

Sonuçları test etmek için Contact DAC uzantısında MaritalStatus alanını “MaritalStatusesCst1Attribute” ile süsleyin:

public class ContactExt : PXCacheExtension<Contact>
{
    [PXRemoveBaseAttribute(typeof(MaritalStatusesAttribute))]
    [PXMergeAttributes(Method = MergeMethod.Append)]
    [MaritalStatusesCst1]
    public string MaritalStatus { get; set; }
}

PXStringListAttribute ardılında belirtilen öğeleri kaldırmak için #

PXStringList veya PXIntList özniteliğinden devralınan bir öznitelik içinde sabit kodlanmış belirli bir açılır öğeyi kaldırmak için, özel alan özniteliğinizin yapıcısındaki “_AllowedValues” ve “_AllowedLabels” dizilerinin boyutunu küçültmeniz gerekir:

public class MaritalStatusesCst2Attribute : MaritalStatusesCst1Attribute
{
    public MaritalStatusesCst2Attribute()
    {
        string[] allowedValues = new string[_AllowedValues.Length - 1];
        string[] allowedLabels = new string[_AllowedLabels.Length - 1];
        Array.Copy(_AllowedValues, 1, allowedValues, 0, _AllowedValues.Length - 1);
        Array.Copy(_AllowedLabels, 1, allowedLabels, 0, _AllowedValues.Length - 1);
        _AllowedValues = allowedValues;
        _AllowedLabels = allowedLabels;
    }
}

Yukarıdaki örnekte, Medeni Durum açılır listesinden Single öğesini kaldırmak için “_AllowedValues” ve “_AllowedLabels” dizilerinin boyutunu küçülttünüz.

Sonuçları test etmek için Contact DAC uzantısında MaritalStatus alanını “MaritalStatusesCst2Attribute” ile süsleyin:

public class ContactExt : PXCacheExtension<Contact>
{
    [PXRemoveBaseAttribute(typeof(MaritalStatusesAttribute))]
    [PXMergeAttributes(Method = MergeMethod.Append)]
    [MaritalStatusesCst2]
    public string MaritalStatus { get; set; }
}

PXStringListAttribute ardılında belirtilen öğeleri değiştirmek için #

Orijinal olarak PXStringList veya PXIntList özniteliğinden devralınan bir öznitelik içinde sabit kodlanmış belirli açılır öğeyi değiştirmek için, özel alan özniteliğinizin yapıcısındaki “_AllowedValues” ve “_AllowedLabels” dizilerindeki uygun değeri güncellemeniz gerekir:

public class MaritalStatusesCst3Attribute : MaritalStatusesCst2Attribute
{
    public const string DivorcedNoCommonLaw = "V";

    public MaritalStatusesCst3Attribute()
    {
        _AllowedValues[Array.IndexOf(_AllowedValues, Divorced)] = DivorcedNoCommonLaw;
        _AllowedLabels[Array.IndexOf(_AllowedLabels, Messages.Divorced)] = MaritalStatusesMessages.DivorcedNoCommonLaw;
    }
}

Yukarıdaki örnekte, _AllowedValues içindeki D - Boşanmış maddesini V - Boşanmış (yaşamayan ortak hukuk) ile değiştirmişsinizdir. ve sırasıyla ‘_AllowedLabels’ dizileri. Bunu yaparak, bir açılır öğenin hem dahili hem de harici değerlerini değiştiririz.

Sonuçları test etmek için Contact DAC uzantısında MaritalStatus alanını “MaritalStatusesCst3Attribute” ile süsleyin:

public class ContactExt : PXCacheExtension<Contact>
{
    [PXRemoveBaseAttribute(typeof(MaritalStatusesAttribute))]
    [PXMergeAttributes(Method = MergeMethod.Append)]
    [MaritalStatusesCst3]
    public string MaritalStatus { get; set; }
}
