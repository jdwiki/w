  • İki nokta arasındaki açıyı alma
  • İki nokta arasındaki mesafeyi alma
  • Bir noktanın dikdörtgen alan içinde olup olmadığını belirleyin

İki nokta arasındaki açıyı alma #

Vanilya matematiğini kullanma:

var from:Point = new Point(100, 50);
var to:Point = new Point(80, 95);

var angle:Number = Math.atan2(to.y - from.y, to.x - from.x);

İlk ikisi arasındaki farkı temsil eden yeni bir vektör kullanma:

var difference:Point = to.subtract(from);

var angle:Number = Math.atan2(difference.y, difference.x);

Not: atan2() derece değil radyan döndürür.

İki nokta arasındaki mesafeyi alma #

Vanilya matematiğini kullanma:

var from:Point = new Point(300, 10);
var to:Point = new Point(75, 40);

var a:Number = to.x - from.x;
var b:Number = to.y - from.y;

var distance:Number = Math.sqrt(a * a + b * b);

Point’nin dahili işlevselliğini kullanma:

var distance:Number = to.subtract(from).length; // or
var distance:Number = Point.distance(to, from);

Bir noktanın dikdörtgen alan içinde olup olmadığını belirleyin #

Bir noktanın bir dikdörtgenin içinde olup olmadığını Rectangle.containsPoint() kullanarak test edebilirsiniz. ()):

var point:Point = new Point(5, 5);
var rectangle:Rectangle = new Rectangle(0, 0, 10, 10);

var contains:Boolean = rectangle.containsPoint(point); // true

Radyanları dereceye dönüştürme #

var degrees:Number = radians * 180 / Math.PI;

Dereceleri radyana dönüştürme #

var radians:Number = degrees / 180 * Math.PI;

Bir noktayı bir açı boyunca hareket ettirme #

Taşımak istediğiniz açıya ve taşımak istediğiniz “x” ve “y” değerlerine sahip bir nesneye sahip olduğunuzu varsayarsak:

var position:Point = new Point(10, 10);
var angle:Number = 1.25;

“Math.cos” ile “x” ekseni boyunca hareket edebilirsiniz:

position.x += Math.cos(angle);

Ve “Math.sin” ile “y” ekseni:

position.y += Math.sin(angle);

Elbette ‘Math.cos’ ve ‘Math.sin’ sonucunu seyahat etmek istediğiniz mesafeyle çarpabilirsiniz:

var distance:int = 20;

position.x += Math.cos(angle) * distance;
position.y += Math.sin(angle) * distance;

Not: Giriş açısı radyan cinsinden olmalıdır.

Derece ve radyan cinsinden bir dairenin değeri #

  • Tam bir daire “360” derece veya “Math.PI * 2” radyandır.
  • Bu değerlerin yarısı “180” derece veya “Math.PI” radyandır.
  • Çeyrek, “90” derece veya “Math.PI / 2” radyandır.

Tüm dairenin radyan cinsinden yüzdesi olarak bir segment elde etmek için:

function getSegment(percent:Number):Number {
    return Math.PI * 2 * percent;
}

var tenth:Number = getSegment(0.1); // One tenth of a circle in radians.
