Selam Dünya! (SQLAlchemy Çekirdeği) #

Bu örnek, SQLAlchemy Core kullanarak bir tablonun nasıl oluşturulacağını, veri ekleneceğini ve veritabanından nasıl seçileceğini gösterir. Yeniden bilgi için: SQLAlchemy ORM, buraya bakın.

İlk olarak, veritabanımıza bağlanma yapmamız gerekecek.

from sqlalchemy import create_engine

engine = create_engine('sqlite://')

Motor, herhangi bir SQLAlchemy uygulamasının başlangıç ​​noktasıdır. Bu, gerçek veritabanı ve onun DBAPI’si için bir “ana üs” olup, bir SQLAlchemy uygulamasına bir bağlantı havuzu ve belirli bir veritabanı/DBAPI kombinasyonuyla nasıl konuşulacağını açıklayan bir lehçe aracılığıyla teslim edilir. Motor, hem DBAPI’nin modül işlevlerini hem de veritabanının davranışını yorumlayan hem bir diyalekt hem de bir bağlantı havuzuna başvurur.

Motorumuzu oluşturduktan sonra, tablolarımızı tanımlayın ve oluşturun.

from sqlalchemy import Column, Integer, Text, MetaData, Table

metadata = MetaData()
messages = Table(
    'messages', metadata,
    Column('id', Integer, primary_key=True),
    Column('message', Text),
)

messages.create(bind=engine)

MetaData nesnesini daha fazla açıklamak için aşağıdaki belgelere bakın:

Tablo nesneleri ve bunlarla ilişkili alt nesneler topluluğu, veritabanı meta verileri olarak adlandırılır.

Tablolarımızı, normal SQL CREATE TABLE ifadelerine benzeyen Tablo yapısını kullanarak MetaData adlı bir katalog içinde tanımlarız.

Artık tablolarımızı tanımlayıp oluşturduğumuza göre, veri eklemeye başlayabiliriz! Ekleme iki adımdan oluşur. insert yapısını oluşturma ve son sorguyu yürütme.

insert_message = messages.insert().values(message='Hello, World!')
engine.execute(insert_message)

Artık verilerimiz olduğuna göre, verilerimizi sorgulamak için seç işlevini kullanabiliriz. Sütun nesneleri, Tablo nesnesinde c özniteliğinin adlandırılmış öznitelikleri olarak bulunur, bu da sütunları doğrudan seçmeyi kolaylaştırır. Bu select ifadesinin çalıştırılması, fetchone(), fetchall() ve fetchmany() gibi birkaç yönteme erişimi olan bir “ResultProxy” nesnesi döndürür ve bunların tümü geri döner select ifademizde sorgulanan bir dizi veritabanı satırı.

from sqlalchemy import select
stmt = select([messages.c.message]) 
message, = engine.execute(stmt).fetchone()
print(message)

Hello, World!

Selam Dünya! (SQLAlchemy ORM) #

Bu örnek, bir tablonun nasıl oluşturulacağını, veri ekleneceğini ve SQLAlchemy ORM kullanılarak veritabanından nasıl seçileceğini gösterir. Bilgi için: SQLAlchemy Core, buraya bakın.

Öncelikle SQLAlchemy Core (Core) kullanarak nasıl bağlanacağımızla aynı olan veritabanımıza bağlanmamız gerekiyor.

from sqlalchemy import create_engine

engine = create_engine('sqlite://')

Motorumuzu bağlayıp oluşturduktan sonra tablolarımızı tanımlayıp oluşturmamız gerekiyor. SQLAlchemy ORM dilinin Core’dan büyük ölçüde farklılaşmaya başladığı yer burasıdır. ORM’de tablo oluşturma ve tanımlama süreci, tabloları ve bu tablolara eşlemek için kullanacağımız sınıfları tanımlayarak başlar. Bu işlem, SQLAlchemy’nin Declarative sistemi olarak adlandırdığı ORM’de tek adımda yapılır.

from sqlalchemy.ext.declarative import declarative_base
Base = declarative_base()

Artık temel eşleyicimiz bildirildiğine göre, bildirimsel eşlemelerimizi veya ‘model’imizi oluşturmak için ondan alt sınıf oluşturabiliriz.

from sqlalchemy import Column, Integer, String

class Message(Base):
    __tablename__ = 'messages'
    
    id = Column(Integer, primary_key=True)
    message = Column(String)

Bildirim temel sınıfını kullanarak, bir “Table” ve “Mapper” nesnesi oluşturuyoruz. Dokümanlardan:

Table nesnesi, MetaData olarak bilinen daha büyük bir koleksiyonun üyesidir. Deklaratif kullanırken, bu nesne bildirim temelli sınıfımızın .metadata özniteliği kullanılarak kullanılabilir.

Bunu göz önünde bulundurarak, henüz var olmayan tüm tabloları oluşturmak için SQLAlchemy Core’un MetaData kayıt defterini kullanan aşağıdaki komutu çağırabiliriz.

Base.metadata.create_all(engine)

Artık tablolarımız eşleştirilip oluşturulduğuna göre veri ekleyebiliriz! Ekleme, eşleştirici örneklerinin oluşturulması aracılığıyla yapılır.

message = Message(message="Hello World!")
message.message # 'Hello World!

Bu noktada elimizdeki tek şey ORM soyutlama düzeyinde bir mesaj örneğidir, ancak henüz veri tabanına hiçbir şey kaydedilmemiştir. Bunu yapmak için önce bir session oluşturmamız gerekiyor.

from sqlalchemy.orm import sessionmaker

Session = sessionmaker(bind=engine)
session = Session()

Bu oturum nesnesi bizim veritabanı işleyicimizdir. SQLAlchemy belgelerine göre:

Motor tarafından sağlanan bir bağlantı havuzundan bir bağlantı alır ve biz tüm değişiklikleri yapana ve/veya oturum nesnesini kapatana kadar onu tutar.

Artık oturumumuz olduğuna göre, oturuma yeni mesajımızı ekleyebilir ve değişikliklerimizi veritabanına uygulayabiliriz.

session.add(message)
session.commit()

Artık verilerimiz olduğuna göre, verilerimizi çekmek için ORM sorgu dilinden yararlanabiliriz.

query = session.query(Message)
instance = query.first()
print (instance.message) # Hello World!

Ama bu sadece başlangıç! “filtre”, “order_by” ve çok daha fazlası gibi sorgu oluşturmak için kullanılabilecek çok daha fazla özellik vardır. Daha fazla örnek ve bilgi için SQLAlchemy ORM Eğitimi’ne bakın.

Kurulum veya Kurulum #

pip install sqlalchemy

En yaygın uygulamalar, özellikle web uygulamaları için, genellikle yeni başlayanların ‘flask-sqlalchemy’ gibi ek bir kitaplık kullanmayı düşünmeleri önerilir.

pip install flask-sqlalchemy
