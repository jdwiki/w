Sözdizimi #

  • getir(url, seçenekler)[.then(…)[.catch(…)]]

  • Getirme API’si, HTTP istekleri için en yaygın kullanılan API’dir. Moderndir, esnektir ve vaatleri kullanır.

  • XMLHttpRequest API, HTTP istekleri için de kullanılır ve geliştiricilerin ApiSauce gibi mevcut favori kitaplıklarını kullanabilmeleri için temel olarak dahil edilmiştir.

  • Websocket API, sohbet uygulamaları gibi gerçek zamanlı senaryolarda “canlı” veriler için kullanılabilir.

Vaatleri getirme API’si ve Redux ile kullanma #

Redux, React-Native ile kullanılan en yaygın durum yönetimi kitaplığıdır. Aşağıdaki örnek, getirme API’sinin nasıl kullanılacağını ve redux-thunk kullanarak uygulamanızın durum düşürücüsünde değişiklikleri nasıl göndereceğinizi gösterir.

export const fetchRecipes = (action) => {
  return (dispatch, getState) => {
    fetch('/recipes', {
        method: 'POST',
        headers: {
          'Accept': 'application/json',
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({
          recipeName,
          instructions,
          ingredients
        })
    })
    .then((res) => {
      // If response was successful parse the json and dispatch an update
      if (res.ok) {
        res.json().then((recipe) => {
          dispatch({
            type: 'UPDATE_RECIPE',
            recipe
          });
        });
      } else {
        // response wasn't successful so dispatch an error
        res.json().then((err) => {
          dispatch({
            type: 'ERROR_RECIPE',
            message: err.reason,
            status: err.status
          });
        });
      }
    })
    .catch((err) => {
      // Runs if there is a general JavaScript error.
      dispatch(error('There was a problem with the request.'));
    });
  };
};

Getirme API’si ile HTTP #

Fetch ‘in ilerleme geri aramalarını desteklemediğine dikkat edilmelidir. Bakınız: https://github.com/github/fetch/issues/89.

Alternatif, XMLHttpRequest https://developer.mozilla.org/en-US/docs/Web/Events/progress kullanmaktır.

fetch('https://mywebsite.com/mydata.json').then(json => console.log(json));

fetch('/login', {
  method: 'POST',
  body: form,
  mode: 'cors',
  cache: 'default',
}).then(session => onLogin(session), failure => console.error(failure));

Getirme hakkında daha fazla ayrıntı MDN adresinde bulunabilir.

XMLHttpRequest ile Ağ Oluşturma #

var request = new XMLHttpRequest();
request.onreadystatechange = (e) => {
  if (request.readyState !== 4) {
    return;
  }

  if (request.status === 200) {
    console.log('success', request.responseText);
  } else {
    console.warn('error');
  }
};

request.open('GET', 'https://mywebsite.com/endpoint/');
request.send();

WebSockets #

var ws = new WebSocket('ws://host.com/path');

ws.onopen = () => {
  // connection opened

  ws.send('something'); // send a message
};

ws.onmessage = (e) => {
  // a message was received
  console.log(e.data);
};

ws.onerror = (e) => {
  // an error occurred
  console.log(e.message);
};

ws.onclose = (e) => {
  // connection closed
  console.log(e.code, e.reason);
};

eksenli HTTP #

Yapılandır

Web isteği için axios kitaplığını da kullanabilirsiniz.

Yapılandırması kolaydır. Bu amaçla, örneğin axios.js dosyasını oluşturabilirsiniz:

import * as axios from 'axios';

var instance = axios.create();
instance.defaults.baseURL = serverURL;
instance.defaults.timeout = 20000;]
//...
//and other options

export { instance as default };

ve sonra istediğiniz herhangi bir dosyada kullanın.

Talepler

Arka ucunuzdaki her hizmet için ‘İsviçre bıçağı’ kalıbını kullanmaktan kaçınmak için, entegrasyon işlevselliği için klasör içinde bunun yöntemleriyle ayrı bir dosya oluşturabilirsiniz:

import axios from '../axios';
import { 
    errorHandling
} from '../common';

const UserService = {
        getCallToAction() {
        return axios.get('api/user/dosomething').then(response => response.data)
            .catch(errorHandling);
    },
}
export default UserService;

Test yapmak

Axios’ları test etmek için özel bir kütüphane vardır: axios-mock-adapter.

Bu lib ile, test etmek için istediğiniz herhangi bir yanıtı axios’a ayarlayabilirsiniz. Ayrıca axois yöntemleriniz için bazı özel hatalar yapılandırabilirsiniz. Önceki adımda oluşturduğunuz axios.js dosyanıza ekleyebilirsiniz:

import MockAdapter from 'axios-mock-adapter';

var mock = new MockAdapter(instance);
mock.onAny().reply(500);

örneğin.

Redux Mağazası

Bazen, muhtemelen redux mağazanızda sakladığınız, yetkilendirme belirtecini başlıklara eklemeniz gerekir.

Bu durumda başka bir dosyaya ihtiyacınız olacak, bu işleve sahip interceptors.js:

export function getAuthToken(storeContainer) {
    return config => {
        let store = storeContainer.getState();
        config.headers['Authorization'] = store.user.accessToken;
        return config;
    };
}

Ardından, kök bileşeninizin yapıcısında şunu ekleyebilirsiniz:

axios.interceptors.request.use(getAuthToken(this.state.store));

ve ardından tüm istekleriniz yetkilendirme jetonunuzla takip edilecektir.

Gördüğünüz gibi axios çok basit, yapılandırılabilir ve tepki-yerel tabanlı uygulamalar için kullanışlı bir kütüphanedir.

Socket.io ile Web Soketi #

socket.io-client‘ı kurun

npm i socket.io-client --save

İçe aktarma modülü

import SocketIOClient from 'socket.io-client/dist/socket.io.js'

Yapıcınızda başlat

constructor(props){
    super(props);
    this.socket = SocketIOClient('http://server:3000');
  }

Şimdi soket bağlantınızı doğru bir şekilde kullanabilmek için, fonksiyonlarınızı yapıcıda da bağlamanız gerekir. Her 5 saniyede bir soket üzerinden bir sunucuya ping gönderecek (bunu ping olarak kabul edecek) basit bir uygulama oluşturmamız gerektiğini varsayalım ve ardından uygulama sunucudan bir yanıt alacaktır. Bunu yapmak için önce şu iki işlevi oluşturalım:

_sendPing(){
    //emit a dong message to socket server
    socket.emit('ding');
}

_getReply(data){
    //get reply from socket server, log it to console
    console.log('Reply from server:' + data);
}

Şimdi, bu iki işlevi yapıcımızda bağlamamız gerekiyor:

constructor(props){
    super(props);
    this.socket = SocketIOClient('http://server:3000');

    //bind the functions
    this._sendPing = this._sendPing.bind(this);
    this._getReply = this._getReply.bind(this);
}

Bundan sonra, soket sunucusundan mesajı alabilmek için _getReply işlevini de sokete bağlamamız gerekiyor. Bunu yapmak için _getReply fonksiyonumuzu socket nesnesine eklememiz gerekiyor. Oluşturucumuza aşağıdaki satırı ekleyin:

this.socket.on('dong', this._getReply);

Şimdi, ne zaman soket sunucusu ‘dong’ ile yayın yaparsa, uygulamanız onu alabilecektir.
