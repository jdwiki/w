Yaylı Önyükleme + JDBC

  • schema.sql dosyası
  • İlk JdbcTemplate Önyükleme Uygulaması
  • veri.sql

Bu sayfada

  • schema.sql dosyası
  • İlk JdbcTemplate Önyükleme Uygulaması
  • veri.sql

Spring Boot, bir SQL İlişkisel Veri Tabanı oluşturmak ve sürdürmek için kullanılabilir. Spring Boot kullanarak bellekteki bir H2 Veritabanına bağlanmayı seçebilir veya belki de MySql Veritabanına bağlanmayı seçebilirsiniz, bu tamamen sizin seçiminizdir. DB’nize karşı CRUD işlemleri yapmak istiyorsanız bunu JdbcTemplate bean kullanarak yapabilirsiniz, bu bean otomatik olarak Spring Boot tarafından sağlanacaktır. Spring Boot, JDBC ile ilgili yaygın olarak kullanılan bazı çekirdeklerin otomatik konfigürasyonunu sağlayarak size yardımcı olacaktır.

Başlamak için, sts Eclipse’inizde new –> Spring Starter Project –> Maven koordinatlarınızı doldurun –> seçeneğine gidin ve sonraki bağımlılıkları ekleyin:

SQL sekmesi altında –> JDBC ekleyin + MySql ekleyin (eğer seçiminiz MySql ise).

Mysql için MySql Java Connector’ı da eklemeniz gerekir.

Spring Boot application.properties dosyanızda (Spring Boot yapılandırma dosyanız), Veri Kaynağı kimlik bilgilerinizi MySql DB’ye yapılandırmanız gerekir:

  1. yay.veri kaynağı.url
  2. yay.veri kaynağı.kullanıcı adı
  3. yay.verikaynağı.şifre
  4. spring.datasource.driver-class-name

örneğin:

spring.datasource.url=jdbc:mysql://localhost/test
spring.datasource.username=dbuser
spring.datasource.password=dbpass
spring.datasource.driver-class-name=com.mysql.jdbc.Driver

Kaynaklar klasörünün altına sonraki iki dosyayı ekleyin:

  1. schema.sql –> uygulamanızı her çalıştırdığınızda Spring Boot bu dosyayı çalıştıracak, içinde DB şemanızı yazmanız, tabloları ve bunların ilişkilerini tanımlamanız gerekiyor.

  2. data.sql –> uygulamanızı her çalıştırdığınızda Spring Boot bu dosyayı çalıştıracaktır, bunun içinde ilk başlatma olarak tablonuza eklenecek verileri yazmanız gerekiyor.

Spring Boot size otomatik olarak JdbcTemplate bean sağlayacak, böylece anında şu şekilde kullanabilirsiniz:

@Autowired
private JdbcTemplate template;

başka konfigürasyonlar olmadan.

schema.sql dosyası #

CREATE SCHEMA IF NOT EXISTS `backgammon`;
USE `backgammon`;

DROP TABLE IF EXISTS `user_in_game_room`;
DROP TABLE IF EXISTS `game_users`;
DROP TABLE IF EXISTS `user_in_game_room`;

CREATE TABLE `game_users`
(
    `user_id` BIGINT NOT NULL AUTO_INCREMENT,
    `first_name` VARCHAR(255) NOT NULL,
    `last_name` VARCHAR(255) NOT NULL,
    `email` VARCHAR(255) NOT NULL UNIQUE,
    `user_name` VARCHAR(255) NOT NULL UNIQUE,
    `password` VARCHAR(255) NOT NULL,
    `role` VARCHAR(255) NOT NULL,
    `last_updated_date` DATETIME NOT NULL,
    `last_updated_by` BIGINT NOT NULL,
    `created_date` DATETIME NOT NULL,
    `created_by` BIGINT NOT NULL,
    PRIMARY KEY(`user_id`)
);

DROP TABLE IF EXISTS `game_rooms`;

CREATE TABLE `game_rooms`
(
    `game_room_id` BIGINT NOT NULL AUTO_INCREMENT,
    `name` VARCHAR(255) NOT NULL,
    `private` BIT(1) NOT NULL,
    `white` BIGINT DEFAULT NULL,
    `black` BIGINT DEFAULT NULL,
    `opened_by` BIGINT NOT NULL,
    `speed` BIT(3) NOT NULL,
    `last_updated_date` DATETIME NOT NULL,
    `last_updated_by` BIGINT NOT NULL,
    `created_date` DATETIME NOT NULL,
    `created_by` BIGINT NOT NULL,
    `token` VARCHAR(255) AS (SHA1(CONCAT(`name`, "This is a qwe secret 123", `created_by`, `created_date`))),
    PRIMARY KEY(`game_room_id`)
);

CREATE TABLE `user_in_game_room`
(
    `user_id` BIGINT NOT NULL,
    `game_room_id` BIGINT NOT NULL,
    `last_updated_date` DATETIME NOT NULL,
    `last_updated_by` BIGINT NOT NULL,
    `created_date` DATETIME NOT NULL,
    `created_by` BIGINT NOT NULL,
    PRIMARY KEY(`user_id`, `game_room_id`),
    FOREIGN KEY (`user_id`) REFERENCES `game_users`(`user_id`),
    FOREIGN KEY (`game_room_id`) REFERENCES `game_rooms`(`game_room_id`)
);

İlk JdbcTemplate Önyükleme Uygulaması #

@SpringBootApplication
@RestController
public class SpringBootJdbcApplication {

    @Autowired
    private JdbcTemplate template;
    
    @RequestMapping("/cars")
    public List<Map<String,Object>> stocks(){
        return template.queryForList("select * from car");
    }
    
    public static void main(String[] args) {
        SpringApplication.run(SpringBootJdbcApplication.class, args);
    }
}
