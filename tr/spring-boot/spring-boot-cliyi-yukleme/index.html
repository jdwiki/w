Spring Boot CLI'yi Yükleme

  • HomeBrew ile Mac OSX’e yükleyin
  • Manuel Kurulum
  • MacPorts ile Mac OSX’e yükleyin
  • SDKMAN ile herhangi bir işletim sistemine yükleyin!

Spring Boot CLI, komut satırından Spring Boot uygulamalarını kolayca oluşturmanıza ve bunlarla çalışmanıza olanak tanır.

Spring Boot CLI kurulduktan sonra spring komutu kullanılarak çalıştırılabilir:

Komut satırı yardımı almak için:

$ spring help

İlk Spring Boot Projenizi oluşturmak ve çalıştırmak için:

$ spring init my-app
$ cd my-app
$ spring run my-app

Tarayıcınızı localhost:8080 olarak açın:

$ open http://localhost:8080

Uygulamanıza henüz herhangi bir kaynak eklemediğiniz için beyaz etiket hata sayfasını alırsınız, ancak yalnızca aşağıdaki dosyalarla çalışmaya hazırsınız:

my-app/
├── mvnw
├── mvnw.cmd
├── pom.xml
└── src/
    ├── main/
    │   ├── java/
    │   │   └── com/
    │   │       └── example/
    │   │           └── DemoApplication.java
    │   └── resources/
    │       └── application.properties
    └── test/
        └── java/
            └── com/
                └── example/
                    └── DemoApplicationTests.java
  • mvnw ve mvnw.cmd - Maven’i (gerekirse) ilk kullanımda indirecek ve kuracak Maven sarmalayıcı komut dosyaları.
  • pom.xml - Maven proje tanımı
  • DemoApplication.java - Spring Boot uygulamanızı başlatan ana sınıf.
  • application.properties - Haricileştirilmiş konfigürasyon özellikleri için bir dosya. (’.yml’ uzantısı da verilebilir.)
  • DemoApplicationTests.java - Spring Boot uygulama bağlamının başlatılmasını doğrulayan bir birim testi.

HomeBrew ile Mac OSX’e yükleyin #

$ brew tap pivotal/tap
$ brew install springboot

MacPorts ile Mac OSX’e yükleyin #

$ sudo port install spring-boot-cli

SDKMAN ile herhangi bir işletim sistemine yükleyin! #

SDKMAN! Java için Yazılım Geliştirme Seti Yöneticisidir. Spring Boot CLI sürümlerinin yanı sıra Java, Maven, Gradle ve daha fazlasını yüklemek ve yönetmek için kullanılabilir.

$ sdk install springboot
