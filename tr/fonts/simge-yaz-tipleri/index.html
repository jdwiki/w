Bir web sayfasına bir simge yazı tipi dahil etme #

CSS’nize bir simge yazı tipi eklemek için aşağıdaki kodu ekleyebilirsiniz:

@font-face {
    font-family: 'myfont';
    src:url('fonts/myfont.eot?-td2xif');
    src:url('fonts/myfont.eot?#iefix-td2xif') format('embedded-opentype'),
        url('fonts/myfont.woff?-td2xif') format('woff'),
        url('fonts/myfont.ttf?-td2xif') format('truetype'),
        url('fonts/myfont.svg?-td2xif#myfont') format('svg');
    // Different URLs are required for optimal browser support
    // Make sure to :
    // 1) replace the URLs with your font's URLs
    // 2) replace `#myfont` with the name of your font
    font-weight: normal; // To avoid the font inherits boldness
    font-style: normal; // To avoid font inherits obliqueness or italic
}

.icon {
    font-family: 'myfont', Verdana, Arial, sans-serif; // Use regular fonts as fallback
    speak: none; // To avoid screen readers trying to read the content
    font-style: normal; // To avoid font inherits obliqueness or italic
    font-weight: normal; // To avoid the font inherits boldness
    font-variant: normal; // To avoid the font inherits small-caps
    text-transform: none; // To avoid the font inherits capitalization/uppercase/lowercase
    line-height: 1; // To avoid the font inherits an undesired line-height
    -webkit-font-smoothing: antialiased; // For improved readability on Webkit
    -moz-osx-font-smoothing: grayscale; // For improved readability on OSX + Mozilla
}

Bir web sayfasında bir simge yazı tipini kullanma #

HTML’nizde bir simge kullanmak için aşağıdakilerin her birini yapabilirsiniz:

<!-- Method 1 -->
<!--- * * * * * * * * * * * * -->
<!-- Set a font-family for an entire HTML element -->
<!-- Define your icon fonts in your CSS font-family after your regular fonts  -->
<!-- This means that regular characters are default. Icons are a fallback  -->
<!-- Use UTF-8 characters directly in your HTML for improved human readability -->
<div class="rate"><p>I rate this movie ★★★★☆!!</p></div>

<!-- Method 2 -->
<!--- * * * * * * * * * * * * -->
<!-- Set a font-family for an entire HTML element -->
<!-- Define your icon fonts in your CSS font-family after your regular fonts  -->
<!-- This means that regular characters are default. Icons are a fallback  -->
<!-- Use entity codes in your HTML when UTF-8 support is uncertain -->
<div class="rate"><p>I rate this movie &#9733;&#9733;&#9733;&#9733;&#9734;!!</p></div>

<!-- Method 3 -->
<!--- * * * * * * * * * * * * -->
<!-- Set a font-family only for the icons but not the HTML elements that include them -->
<!-- Define your icon fonts in your CSS font-family before your regular fonts  -->
<!-- This means that icons are default. Regular characters are a fallback  -->
<!-- Use UTF-8 characters directly in your HTML for improved human readability -->
<p>I rate this movie <span class="icon">★★★★☆</span>!!</p>

<!-- Method 4 -->
<!--- * * * * * * * * * * * * -->
<!-- Set a font-family only for the icons but not the HTML elements that include them -->
<!-- Define your icon fonts in your CSS font-family before your regular fonts  -->
<!-- This means that icons are default. Regular characters are a fallback  -->
<!-- Use entity codes in your HTML when UTF-8 support is uncertain -->
<p>I rate this movie <span class="icon">&#9733;&#9733;&#9733;&#9733;&#9734;</span>!!</p>

<!-- Method 5 -->
<!--- * * * * * * * * * * * * -->
<!-- Set a font-family only for the icons and use a separate HTML tag for each icon -->
<!-- Define your icon fonts in your CSS font-family before your regular fonts  -->
<!-- This means that icons are default. Regular characters are a fallback  -->
<!-- Use UTF-8 characters directly in your HTML for improved human readability -->
<p>I rate this movie
    <span class="icon">★</span>
    <span class="icon">★</span>
    <span class="icon">★</span>
    <span class="icon">★</span>
    <span class="icon">☆</span>
    !!
</p>

<!-- Method 6 -->
<!--- * * * * * * * * * * * * -->
<!-- Set a font-family only for the icons and use a separate HTML tag for each icon -->
<!-- Define your icon fonts in your CSS font-family before your regular fonts  -->
<!-- This means that icons are default. Regular characters are a fallback  -->
<!-- Use entity codes in your HTML when UTF-8 support is uncertain -->
<p>I rate this movie
    <span class="icon">&#9733;</span>
    <span class="icon">&#9733;</span>
    <span class="icon">&#9733;</span>
    <span class="icon">&#9733;</span>
    <span class="icon">&#9734;</span>
    !!
</p>

<!-- Method 7-->
<!--- * * * * * * * * * * * * -->
<!-- Set a font-family only for the icons and use a separate HTML tag for each icon -->
<!-- Define your icon fonts in your CSS font-family before your regular fonts  -->
<!-- This means that icons are default. Regular characters are a fallback  -->
<!-- Use the 'content' style rule with a ':before selector' in your CSS -->
<p>I rate this movie
    <span class="icon icon-star"></span>
    <span class="icon icon-star"></span>
    <span class="icon icon-star"></span>
    <span class="icon icon-star"></span>
    <span class="icon icon-star-unfilled"></span>
    !!
</p>

Yöntem 7’yi seçmek istiyorsanız, bazı ek CSS kodlarına ihtiyacınız olacak. Bu CSS kodu şöyle görünür:

.icon-star:before {
    content: "\2605";
}

.icon-star-unfilled:before {
    content: "\2606";
}

İkonik, Font Awesome veya Glyphicons gibi simge yazı tiplerinin tümü genellikle yöntem 7’yi kullanır. - bir hile sayfasından özel karakterler yapıştırın veya HTML varlıklarını kullanmaya zorlayın.

Bununla birlikte, birkaç dezavantajı olan bir yöntemdir. Her şeyden önce, :before CSS seçicisi için destek ve UNICODE karakterleri için bir kaçış dizisinin kullanılmasını gerektirir. Ne IE6-7 ne de Webkit’in bazı sürümleri bu desteği sağlamaz.

Diğer bir dezavantajı ise her simge için ayrı bir HTML etiketi kullanmak zorunda olmanız ve her etiketin simge yazı tipinden bir karaktere karşılık gelmesidir. Diğer yöntemlerden farklı olarak, yöntem 7 ile HTML etiketi içinde birden fazla simge görüntülemek mümkün değildir.

Diğer yöntemlerin de kendi dezavantajları vardır. Yöntem 1, 3 ve 5, karakteri bir hile sayfasından kopyalayıp yapıştırmanızı veya karakterin kendisini kodunuza yerleştirmek için araçlar kullanmanızı gerektirir. Simge yazı tipi standart olmayan bir eşleme kullanıyorsa, kod düzenleyiciniz karakteri görüntüleyemeyebilir veya simge yazı tipinizdeki karakterden farklı bir karakter görüntüleyebilir.

Yöntem 1, 3 ve 5 ayrıca tarayıcınızın doğru karakteri görüntülemek için uygun kodlamayı kullanmasını gerektirir. UNICODE karakterleri için bu, ASCII karakterleri için olduğu kadar açık değildir. Ancak bu, HTML belgenizin “head” kısmına “” meta etiketini ekleyerek sağlanmalıdır.

Yöntem 2, 4 ve 6, karakteri kopyalayıp yapıştırmanızı gerektirmez, ancak kodunuzu insanlar tarafından daha az okunabilir hale getirir ve kodda yapılan herhangi bir değişikliği insan hatasına daha açık hale getirir. Ayrıca, kullanmak istediğiniz simgelerin her biri için HTML varlık kodunu aramanız veya bunları ezberlemeniz gerekeceğinden. Aynısı açıkça yöntem 7’de kullanılan sınıflar için de geçerli olsa da, bu sınıfların ezberlenmesi bir HTML varlık kodundan çok daha kolaydır.

Bir web sayfasına belirli bir sembol ekleme #

Aşağı bakan bir üçgen düşünün.

Bu sembolü bir web sayfasında göstermenin birkaç doğru yolu vardır.

Yöntem 1: ondalık HTML varlığı kullanın #

HTML :

&#9660;

Yöntem 2: onaltılık HTML varlığı kullanın #

HTML :

&#x25BC;

Yöntem 3: karakteri doğrudan kullanın #

HTML :

▼

Yöntem 4 : CSS kullanın #

HTML :

<span class='icon-down'></span>

CSS :

.icon-down:before {
    content: "\25BC";
}

Bu üç yöntemin her biri aynı çıktıya sahip olmalıdır. Diğer semboller için aynı üç seçenek mevcuttur. Hatta bazılarının dize tabanlı bir referans kullanmanıza izin veren dördüncü bir seçeneği bile vardır (örneğin, ♥’yı görüntülemek için &hearts;).

UNICODE’da hangi simgelerin desteklendiğini ve hangi kodlara karşılık geldiklerini bulmak için Unicode-table.com gibi bir referans web sitesini kullanabilirsiniz. Örneğin, aşağıyı gösteren üçgenin değerlerini http://unicode-table.com/en/25BC/ adresinde bulabilirsiniz.

Bu yöntemlerin yalnızca varsayılan olarak her tarayıcıda bulunan simgeler için yeterli olduğunu unutmayın. ☃,❄,★,☂,☭,⎗ veya ⎘ gibi semboller için bu durum çok daha az olasıdır. Diğer UNICODE simgeleri için tarayıcılar arası destek sağlamak mümkün olsa da, bir simge yazı tipi edinmeniz veya kendi yazı tipinizi oluşturmanız gerekir. Kendi yazı tipinizi nasıl oluşturacağınızla ilgili daha fazla bilgi için **https://www.wikiod.com/tr/fonts/kendi-yaz-tipinizi-olusturma adresine bakın.
