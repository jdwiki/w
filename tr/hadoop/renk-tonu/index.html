Hue, HDFS, Hive, Spark, Hbase, Sqoop, Impala, Pig, Oozie vb. gibi yaygın olarak kullanılan Bigdata teknolojilerinin çoğuna bağlanmak ve bunlarla çalışmak için kullanılan bir Kullanıcı Arabirimidir. Hue, İlişkisel veritabanlarına karşı sorgu çalıştırmayı da destekler.

Bir Django web uygulaması olan Hue, öncelikle Hive sorgularını çalıştırmak için bir çalışma tezgahı olarak oluşturulmuştur. Daha sonra, Hue’nun işlevselliği, Hadoop Ekosisteminin farklı bileşenlerini desteklemek için artırıldı. Apache Lisansı altında açık kaynaklı yazılım olarak mevcuttur.

Ubuntu’da Hue Kurulumu #

Bu kurulum, “hadoop"un “hadoop” kullanıcısı altında önceden yüklenmiş olduğunu varsayar.

Ön koşullar:

Ton, aşağıdaki paketlere bağlıdır

  1. gcc
  2. g++
  3. libxml2-dev
  4. libxlst-dev
  5. libsasl2-dev
  6. libsasl2-modülleri-gssapi-mit
  7. libmysqlclient-dev
  8. python-dev
  9. python-kurulum araçları
  10. libsqlite3-dev
  11. karınca
  12. libkrb5-dev
  13. özgürlük-0.99-0
  14. libldap2-dev
  15. libssl-dev
  16. libgmp3-dev

Tüm paketleri yükleme

sudo apt-get update
sudo apt-get install gcc g++ libxml2-dev libxslt-dev libsasl2-dev libsasl2-modules-gssapi-mit libmysqlclient-dev python-dev python-setuptools libsqlite3-dev ant libkrb5-dev libtidy-0.99-0 libldap2-dev libssl-dev libgmp3-dev

Kurulum ve Yapılandırma

‘hadoop’ kullanıcısı olarak kurulum gerçekleştiriyoruz.

su - hadoop

  1. Gethue.com’dan Hue’yu indirin (bu bağlantı Hue web sitesinden alınan bir örnektir)

    wget https://dl.dropboxusercontent.com/u/730827/hue/releases/3.9.0/hue-3.9.0.tgz

  2. İndirilen tarball’ı çıkarın

    tar -xvf hue-3.9.0.tgz

  3. Kurulum komutunu yürütün

     cd hue-3.9.0
 PREFIX=/home/hadoop/ make install

  4. Yukarıdaki işlem tamamlandıktan sonra,

    Update ~/.bashrc file,

     export HUE_HOME=/home/hadoop/hue
 export PATH=$PATH:$HUE_HOME/build/env/bin

    source after adding the entries, source ~/.bashrc

  5. Hue’yu yapılandırın (düzenlenecek 3 dosya)

`cd $HUE_HOME/masaüstü/conf’

  • hue.ini

    [desktop]
 server_user=hadoop
 server_group=hadoop
 default_user=hadoop
 default_hdfs_superuser=hadoop

cd $HADOOP_CONF_DIR

  • çekirdek-site.xml

      <property>
      <name>hadoop.proxyuser.hadoop.hosts</name>
      <value>*</value> 
  </property> 
  <property>
      <name>hadoop.proxyuser.hadoop.groups</name>
      <value>*</value> 
  </property>

  • hdfs-site.xml

      <property>
      <name>dfs.webhdfs.enabled</name>
      <value>true</value> 
  </property>

  1. Hue’yu Başlatın (Henüz başlamadıysa Hadoop arka plan programlarını başlatın)

    nohup supervisor &

  2. Hue Web Arayüzüne giriş yapın: http://localhost:8888

    username: hadoop

    password: user_choice
