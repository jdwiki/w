Go ile verileri nasıl şifreleyeceğinizi ve şifresini nasıl çözeceğinizi öğrenin. Bunun kriptografi ile ilgili bir kurs olmadığını, bunun Go ile nasıl başarılacağını unutmayın.

Şifreleme ve şifre çözme #

Önsöz #

Bu, Go ile verilerin nasıl şifreleneceği ve şifresinin çözüleceği hakkında ayrıntılı bir örnektir. Kullanım kodu kısaltılmıştır, ör. hata işlemeden bahsedilmiyor. Hata işleme ve kullanıcı arayüzüne sahip tam çalışma projesi Github buradan bulunabilir.

Şifreleme #

Giriş ve veriler #

Bu örnek, Go’da tam çalışan bir şifrelemeyi ve şifre çözmeyi açıklar. Bunun için bir veriye ihtiyacımız var. Bu örnekte, kendi veri yapımızı “gizli” kullanıyoruz:

type secret struct {
    DisplayName       string
    Notes             string
    Username          string
    EMail             string
    CopyMethod        string
    Password          string
    CustomField01Name string
    CustomField01Data string
    CustomField02Name string
    CustomField02Data string
    CustomField03Name string
    CustomField03Data string
    CustomField04Name string
    CustomField04Data string
    CustomField05Name string
    CustomField05Data string
    CustomField06Name string
    CustomField06Data string
}

Sonra, böyle bir “sır"ı şifrelemek istiyoruz. Tam çalışma örneği burada (Github’a bağlantı) bulunabilir. Şimdi, adım adım süreç:

Aşama 1 #

Her şeyden önce, sırrı korumak için bir tür ana şifreye ihtiyacımız var: masterPassword := "PASS"

Adım 2 #

Dizeler yerine baytlarla çalışan tüm kripto yöntemleri. Böylece, sırrımızdaki verilerle bir bayt dizisi oluşturuyoruz.

secretBytesDecrypted := []byte(fmt.Sprintf("%s\n%s\n%s\n%s\n%s\n%s\n%s\n%s\n%s\n%s\n%s\n%s\n%s\n%s\n%s\n%s\n%s\n%s\n",
    artifact.DisplayName,
    strings.Replace(artifact.Notes, "\n", string(65000), -1),
    artifact.Username,
    artifact.EMail,
    artifact.CopyMethod,
    artifact.Password,
    artifact.CustomField01Name,
    artifact.CustomField01Data,
    artifact.CustomField02Name,
    artifact.CustomField02Data,
    artifact.CustomField03Name,
    artifact.CustomField03Data,
    artifact.CustomField04Name,
    artifact.CustomField04Data,
    artifact.CustomField05Name,
    artifact.CustomField05Data,
    artifact.CustomField06Name,
    artifact.CustomField06Data,
))

Aşama 3 #

Gökkuşağı sofra saldırılarını önlemek için biraz tuz oluşturuyoruz, bkz. Wikipedia: saltBytes := uuid.NewV4().Bytes(). Burada öngörülemeyen bir UUID v4 kullanıyoruz.

Adım 4 #

Şimdi, RFC 2898 ile ilgili olarak ana paroladan ve rastgele tuzdan bir anahtar ve bir vektör türetebiliriz:

keyLength := 256
rfc2898Iterations := 6

keyVectorData := pbkdf2.Key(masterPassword, saltBytes, rfc2898Iterations, (keyLength/8)+aes.BlockSize, sha1.New)
keyBytes := keyVectorData[:keyLength/8]
vectorBytes := keyVectorData[keyLength/8:]

Adım 5 #

İstenen CBC modu tüm bloklarla çalışır. Bu nedenle, verilerimizin tam bir bloğa hizalanıp hizalanmadığını kontrol etmeliyiz. Değilse, doldurmamız gerekir:

if len(secretBytesDecrypted)%aes.BlockSize != 0 {
    numberNecessaryBlocks := int(math.Ceil(float64(len(secretBytesDecrypted)) / float64(aes.BlockSize)))
    enhanced := make([]byte, numberNecessaryBlocks*aes.BlockSize)
    copy(enhanced, secretBytesDecrypted)
    secretBytesDecrypted = enhanced
}

Adım 6 #

Şimdi bir AES şifresi oluşturuyoruz: aesBlockEncrypter, aesErr := aes.NewCipher(keyBytes)

Adım 7 #

Şifrelenmiş veriler için gerekli belleği ayırıyoruz: encryptedData := make([]byte, len(secretBytesDecrypted)). AES-CBC durumunda, şifrelenmiş veriler şifrelenmemiş verilerle aynı uzunluğa sahipti.

Adım 8 #

Şimdi, şifreleyiciyi oluşturmalı ve verileri şifrelemeliyiz:

aesEncrypter := cipher.NewCBCEncrypter(aesBlockEncrypter, vectorBytes)
aesEncrypter.CryptBlocks(encryptedData, secretBytesDecrypted)

Şimdi, şifrelenmiş veriler “encryptedData” değişkeninin içindedir.

Adım 9 #

Şifrelenmiş veriler saklanmalıdır. Ancak yalnızca veriler değil: Tuz olmadan şifrelenmiş verilerin şifresi çözülemezdi. Bu nedenle, bunu yönetmek için bir tür dosya formatı kullanmalıyız. Burada şifrelenmiş veriyi base64 olarak kodluyoruz, bkz. Vikipedi:

encodedBytes := make([]byte, base64.StdEncoding.EncodedLen(len(encryptedData)))
base64.StdEncoding.Encode(encodedBytes, encryptedData)

Daha sonra dosya içeriğimizi ve kendi dosya formatımızı tanımlıyoruz. Biçim şuna benzer: “salt[0x10]base64 içeriği”. İlk önce tuzu saklıyoruz. Base64 içeriğinin başlangıcını işaretlemek için “10” baytını depolarız. Bu işe yarar, çünkü base64 bu değeri kullanmaz. Bu nedenle, dosyanın sonundan başına kadar ‘10’un ilk oluşumunu arayarak base64’ün başlangıcını bulabiliriz.

fileContent := make([]byte, len(saltBytes))
copy(fileContent, saltBytes)
fileContent = append(fileContent, 10)
fileContent = append(fileContent, encodedBytes...)

Adım 10 #

Son olarak dosyamızı yazabiliriz: writeErr := ioutil.WriteFile("my secret.data", fileContent, 0644).

Şifre çözme #

Giriş ve veriler #

Şifrelemeye gelince, çalışmak için bazı verilere ihtiyacımız var. Böylece, şifreli bir dosyamız olduğunu ve söz konusu yapının `gizli’ olduğunu varsayıyoruz. Amaç, dosyadan şifrelenmiş verileri okumak, şifresini çözmek ve yapının bir örneğini oluşturmaktır.

Aşama 1 #

İlk adım şifreleme ile aynıdır: Sırrın şifresini çözmek için bir tür ana şifreye ihtiyacımız var: masterPassword := "PASS".

Adım 2 #

Şimdi, şu dosyadan şifrelenmiş verileri okuyoruz: encryptedFileData, bytesErr := ioutil.ReadFile(filename).

Aşama 3 #

Daha önce bahsedildiği gibi, tuz ve şifrelenmiş verileri sınırlayıcı bayt ‘10’ ile bölebilir, baştan sona geriye doğru aranabilir:

for n := len(encryptedFileData) - 1; n > 0; n-- {
    if encryptedFileData[n] == 10 {
        saltBytes = encryptedFileData[:n]
        encryptedBytesBase64 = encryptedFileData[n+1:]
        break
    }
}

Adım 4 #

Ardından, base64 ile kodlanmış baytların kodunu çözmeliyiz:

decodedBytes := make([]byte, len(encryptedBytesBase64))
countDecoded, decodedErr := base64.StdEncoding.Decode(decodedBytes, encryptedBytesBase64)
encryptedBytes = decodedBytes[:countDecoded]

Adım 5 #

Şimdi, RFC 2898 ile ilgili olarak ana paroladan ve rastgele tuzdan bir anahtar ve bir vektör türetebiliriz:

keyLength := 256
rfc2898Iterations := 6

keyVectorData := pbkdf2.Key(masterPassword, saltBytes, rfc2898Iterations, (keyLength/8)+aes.BlockSize, sha1.New)
keyBytes := keyVectorData[:keyLength/8]
vectorBytes := keyVectorData[keyLength/8:]

Adım 6 #

Bir AES şifresi oluşturun: aesBlockDecrypter, aesErr := aes.NewCipher(keyBytes).

Adım 7 #

Şifresi çözülen veriler için gerekli belleği ayırın: decryptedData := make([]byte, len(encryptedBytes)). Tanım olarak, şifrelenmiş verilerle aynı uzunluğa sahiptir.

Adım 8 #

Şimdi, şifre çözücüyü oluşturun ve verilerin şifresini çözün:

aesDecrypter := cipher.NewCBCDecrypter(aesBlockDecrypter, vectorBytes)
aesDecrypter.CryptBlocks(decryptedData, encryptedBytes)

Adım 9 #

Okunan baytları dizeye dönüştürün: decryptedString := string(decryptedData). Satırlara ihtiyacımız olduğundan, dizeyi bölün: lines := strings.Split(decryptedString, "\n").

Adım 10 #

Satırlardan bir “sır” oluşturun:

artifact := secret{}
artifact.DisplayName = lines[0]
artifact.Notes = lines[1]
artifact.Username = lines[2]
artifact.EMail = lines[3]
artifact.CopyMethod = lines[4]
artifact.Password = lines[5]
artifact.CustomField01Name = lines[6]
artifact.CustomField01Data = lines[7]
artifact.CustomField02Name = lines[8]
artifact.CustomField02Data = lines[9]
artifact.CustomField03Name = lines[10]
artifact.CustomField03Data = lines[11]
artifact.CustomField04Name = lines[12]
artifact.CustomField04Data = lines[13]
artifact.CustomField05Name = lines[14]
artifact.CustomField05Data = lines[15]
artifact.CustomField06Name = lines[16]
artifact.CustomField06Data = lines[17]

Son olarak, notlar alanında satır sonlarını yeniden oluşturun: artifact.Notes = strings.Replace(artifact.Notes, string(65000), "\n", -1).
