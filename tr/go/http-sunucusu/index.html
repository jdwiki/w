Selam Dünya #

Golang’da web sunucuları yazmaya başlamanın tipik yolu, standart kütüphane net/http modülünü kullanmaktır.

Bunun için bir öğretici de var burada.

Aşağıdaki kod da bunu kullanır. İşte mümkün olan en basit HTTP sunucusu uygulaması. Herhangi bir HTTP isteğine "Merhaba Dünya" yanıtını verir.

Aşağıdaki kodu çalışma alanlarınızda bir server.go dosyasına kaydedin.

package main

import (
    "log"
    "net/http"
)

func main() {
    // All URLs will be handled by this function
    // http.HandleFunc uses the DefaultServeMux
    http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
        w.Write([]byte("Hello, world!"))
    })

    // Continue to process new requests until an error occurs
    log.Fatal(http.ListenAndServe(":8080", nil))
}

Sunucuyu aşağıdakileri kullanarak çalıştırabilirsiniz:

$ go run server.go

Veya derleyip çalıştırabilirsiniz.

$ go build server.go
$ ./server

Sunucu belirtilen bağlantı noktasını (:8080) dinleyecektir. Herhangi bir HTTP istemcisi ile test edebilirsiniz. İşte “cURL” ile bir örnek:

curl -i http://localhost:8080/
HTTP/1.1 200 OK
Date: Wed, 20 Jul 2016 18:04:46 GMT
Content-Length: 13
Content-Type: text/plain; charset=utf-8

Hello, world!

İşlemi durdurmak için Ctrl+C tuşlarına basın.

HTTPS Sunucusu Oluşturun #

Sertifika oluştur #

Bir HTTPS sunucusunu çalıştırmak için bir sertifika gereklidir. ‘openssl’ ile kendinden imzalı bir sertifika oluşturmak, şu komut yürütülerek yapılır:

openssl req -x509 -newkey rsa:4096 -sha256 -nodes -keyout key.pem -out cert.pem -subj "/CN=example.com" -days 3650`

Parametreler:

  • req Sertifika talep aracını kullanın
  • `x509’ Kendinden imzalı bir sertifika oluşturur
  • newkey rsa:4096 4096 bit anahtar uzunluğuna sahip RSA algoritmalarını kullanarak yeni bir anahtar ve sertifika oluşturur
  • sha256 Büyük tarayıcıların güvenli olarak kabul ettiği SHA256 karma algoritmalarını zorlar (2017 yılında)
  • düğümler Özel anahtar için parola korumasını devre dışı bırakır. Bu parametre olmadan, sunucunuz her başlatıldığında sizden şifre istemek zorunda kaldı.
  • keyout Anahtarın yazılacağı dosyaya isim verir
  • out Sertifikanın yazılacağı dosyayı adlandırır
  • subj Bu sertifikanın geçerli olduğu alan adını tanımlar
  • days Bu sertifika kaç gün geçerli olmalıdır? ‘3650’ yakl. 10 yıl.

Not: Kendinden imzalı bir sertifika kullanılabilir; dahili projeler, hata ayıklama, test etme vb. için. Dışarıdaki herhangi bir tarayıcı bu sertifikanın güvenli olmadığını belirtecektir. Bunu önlemek için sertifikanın bir sertifika yetkilisi tarafından imzalanması gerekir. Çoğunlukla, bu ücretsiz olarak mevcut değildir. Bir istisna “Let’s Encrypt” hareketidir: https://letsencrypt.org

Gerekli Go kodu #

Sunucu için yapılandırma TLS’sini aşağıdaki kodla işleyebilirsiniz. “cert.pem” ve “key.pem”, yukarıdaki komutla oluşturulan SSL sertifikanız ve anahtarınızdır.

package main

import (
    "log"
    "net/http"
)

func main() {
    http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
        w.Write([]byte("Hello, world!"))
    })

    log.Fatal(http.ListenAndServeTLS(":443","cert.pem","key.pem", nil))
}

Özel sunucu ve mux ile HTTP Hello World #

package main

import (
    "log"
    "net/http"
)

func main() {

    // Create a mux for routing incoming requests
    m := http.NewServeMux()

    // All URLs will be handled by this function
    m.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
        w.Write([]byte("Hello, world!"))
    })

    // Create a server listening on port 8000
    s := &http.Server{
        Addr:    ":8000",
        Handler: m,
    }

    // Continue to process new requests until an error occurs
    log.Fatal(s.ListenAndServe())
}

İşlemi durdurmak için Ctrl+C tuşlarına basın.

Şablonları Kullanarak Bir HTTP İsteğine Yanıt Verme #

Yanıtlar, Go’daki şablonlar kullanılarak bir “http.ResponseWriter"a yazılabilir. Dinamik sayfalar oluşturmak istiyorsanız bu kullanışlı bir araç olduğunu kanıtlıyor.

(Go’da Şablonların nasıl çalıştığını öğrenmek için lütfen Go Templates Documentation sayfasını ziyaret edin.)

Bir HTTP İsteğine yanıt vermek için “html/şablonu” kullanmak için basit bir örnekle devam edelim:

package main

import(
    "html/template"
    "net/http"
    "log"
)

func main(){
    http.HandleFunc("/",WelcomeHandler)
    http.ListenAndServe(":8080",nil)
}

type User struct{
    Name string
    nationality string //unexported field.
}

func check(err error){
    if err != nil{
        log.Fatal(err)
    }
}

func WelcomeHandler(w http.ResponseWriter, r *http.Request){
    if r.Method == "GET"{
        t,err := template.ParseFiles("welcomeform.html")
        check(err)
        t.Execute(w,nil)
    }else{
        r.ParseForm()
        myUser := User{}
        myUser.Name = r.Form.Get("entered_name")
        myUser.nationality = r.Form.Get("entered_nationality")
        t, err := template.ParseFiles("welcomeresponse.html")
        check(err)
        t.Execute(w,myUser)
    }
}

İçeriği nerede,

  1. “welcomeform.html” şunlardır:

    Help us greet you
    Enter Name: Enter Nationality:

  2. “welcomeresponse.html” şunlardır:

    Greetings, {{.Name}} Greetings, {{.Name}}.
    We know you are a {{.nationality}}!

Not:

  1. .html dosyalarının doğru dizinde olduğundan emin olun.

  2. Sunucu başlatıldıktan sonra http://localhost:8080/ ziyaret edilebildiğinde.

  3. Form gönderildikten sonra görüldüğü gibi, yapının dışa aktarılmamış milliyet alanı beklendiği gibi şablon paketi tarafından ayrıştırılamadı.

ServeMux kullanarak içerik sunma #

Basit bir statik dosya sunucusu şöyle görünür:

package main

import (
    "net/http"
)

func main() {
    muxer := http.NewServeMux()
    fileServerCss := http.FileServer(http.Dir("src/css"))
    fileServerJs := http.FileServer(http.Dir("src/js"))
    fileServerHtml := http.FileServer(http.Dir("content"))
    muxer.Handle("/", fileServerHtml)
    muxer.Handle("/css", fileServerCss)
    muxer.Handle("/js", fileServerJs)
    http.ListenAndServe(":8080", muxer)
}

Bir işleyici işlevi kullanma #

HandleFunc, sunucu mux’ında (yönlendirici) verilen model için işleyici işlevini kaydeder.

Temel Hello World örneğinde gördüğümüz gibi, anonim bir işlev tanımlamayı geçebilirsiniz:

http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintln(w, "Hello, world!")
}

Ancak bir HandlerFunc türünü de iletebiliriz. Başka bir deyişle, aşağıdaki imzaya uyan herhangi bir işlevi iletebiliriz:

func FunctionName(w http.ResponseWriter, req *http.Request)

Referansı önceden tanımlanmış bir ‘HandlerFunc’a ileterek önceki örneği yeniden yazabiliriz. İşte tam örnek:

package main

import (
    "fmt"
    "net/http"
)

// A HandlerFunc function
// Notice the signature of the function
func RootHandler(w http.ResponseWriter, req *http.Request) {
    fmt.Fprintln(w, "Hello, world!")
}

func main() {
    // Here we pass the reference to the `RootHandler` handler function
    http.HandleFunc("/", RootHandler)
    panic(http.ListenAndServe(":8080", nil))
}

Elbette, farklı yollar için birkaç işlev işleyicisi tanımlayabilirsiniz.

package main

import (
    "fmt"
    "log"
    "net/http"
)

func FooHandler(w http.ResponseWriter, req *http.Request) {
    fmt.Fprintln(w, "Hello from foo!")
}

func BarHandler(w http.ResponseWriter, req *http.Request) {
    fmt.Fprintln(w, "Hello from bar!")
}

func main() {
    http.HandleFunc("/foo", FooHandler)
    http.HandleFunc("/bar", BarHandler)

    log.Fatal(http.ListenAndServe(":8080", nil))
}

http yöntemini kullanma, sorgu dizelerine ve istek gövdesine erişme #

API geliştirme, isteğin HTTP Yöntemi arasında ayrım yapma, sorgu dizesi değerlerine erişme ve istek gövdesine erişme ile ilgili bazı yaygın görevlere ilişkin basit bir örnek.

Kaynaklar

  • http.Handler arayüzü

  • http.ResponseWriter

  • http.Request

  • Kullanılabilir Yöntem ve Durum sabitleri

    package main

    import ( “fmt” “io/ioutil” “log” “net/http” )

    type customHandler struct{}

    // ServeHTTP implements the http.Handler interface in the net/http package func (h customHandler) ServeHTTP(w http.ResponseWriter, r *http.Request) {

      // ParseForm will parse query string values and make r.Form available
  r.ParseForm()

  // r.Form is map of query string parameters
  // its' type is url.Values, which in turn is a map[string][]string
  queryMap := r.Form

  switch r.Method {
  case http.MethodGet:
      // Handle GET requests
      w.WriteHeader(http.StatusOK)
      w.Write([]byte(fmt.Sprintf("Query string values: %s", queryMap)))
      return
  case http.MethodPost:
      // Handle POST requests
      body, err := ioutil.ReadAll(r.Body)
      if err != nil {
          // Error occurred while parsing request body
          w.WriteHeader(http.StatusBadRequest)
          return
      }
      w.WriteHeader(http.StatusOK)
      w.Write([]byte(fmt.Sprintf("Query string values: %s\nBody posted: %s", queryMap, body)))
      return
  }

  // Other HTTP methods (eg PUT, PATCH, etc) are not handled by the above
  // so inform the client with appropriate status code
  w.WriteHeader(http.StatusMethodNotAllowed)

    }

    func main() { // All URLs will be handled by this function // http.Handle, similarly to http.HandleFunc // uses the DefaultServeMux http.Handle(”/", customHandler{})

      // Continue to process new requests until an error occurs
  log.Fatal(http.ListenAndServe(":8080", nil))

    }
