Haritalar, sıralanmamış anahtar/değer çiftlerini depolamak için kullanılan veri türleridir, böylece belirli bir anahtarla ilişkili değeri aramak çok verimli olur. Anahtarlar benzersizdir. Temel veri yapısı, yeni öğeleri barındırmak için gerektiği gibi büyür, bu nedenle programcının bellek yönetimi konusunda endişelenmesine gerek kalmaz. Diğer dillerin karma tablolar, sözlükler veya ilişkisel diziler dediği şeye benzerler.

Sözdizimi #

  • var mapName map[KeyType]ValueType // bir Harita tanımla
  • var mapName = map[KeyType]ValueType{} // boş bir Harita bildir ve ata
  • var mapName = map[KeyType]ValueType{key1: val1, key2: val2} // bir Harita bildir ve ata
  • mapName := make(map[KeyType]ValueType) // varsayılan boyut haritasını bildir ve başlat
  • mapName := make(map[KeyType]ValueType, length) // uzunluk boyut haritasını bildir ve başlat
  • mapName := map[KeyType]ValueType{} // otomatik olarak bildir ve := ile boş bir Harita ata
  • mapName := map[KeyType]ValueType{key1: value1, key2: value2} // otomatik olarak bildir ve := ile bir Harita atayın
  • değer := haritaAdı[anahtar] // Anahtara göre değer alın
  • değer, hasKey := mapName[key] // Anahtara göre değer alın, haritada anahtar varsa ‘hasKey’ ’true’ olur
  • mapName[anahtar] = değer // Değeri anahtara göre ayarla

Go, bir hash tablosu uygulayan yerleşik bir “harita” türü sağlar. Haritalar, Go’nun yerleşik ilişkisel veri türüdür (diğer dillerde hashes veya dictionaries olarak da adlandırılır).

Bir haritayı bildirme ve başlatma #

“map” anahtar sözcüğünü ve ardından anahtarlarının türlerini ve değerlerini kullanarak bir harita tanımlarsınız:

// Keys are ints, values are ints.
var m1 map[int]int // initialized to nil

// Keys are strings, values are ints.
var m2 map[string]int // initialized to nil

Haritalar referans türleridir ve bir kez tanımlandıktan sonra [sıfır değeri “nil” olur](https://www.wikiod.com/tr/go/haritalar#Bir haritanın sıfır değeri 201607220859253128466). Sıfır haritalara yazma işlemleri panik ve okumalar her zaman sıfır değerini döndürür.

Bir haritayı başlatmak için make işlevini kullanın:

m := make(map[string]int)

İki parametreli “make” formuyla, varsayılan kapasiteyi geçersiz kılarak harita için bir başlangıç ​​giriş kapasitesi belirlemek mümkündür:

m := make(map[string]int, 30)

Alternatif olarak, bir harita bildirebilir, onu sıfır değerine başlatabilir ve daha sonra ona bir değişmez değer atayabilirsiniz; bu, yapıyı json’a sıralarsanız ve dönüşte boş bir harita üretirseniz yardımcı olur.

m := make(map[string]int, 0)

Ayrıca bir harita yapabilir ve başlangıç ​​değerini küme parantezleri ({}) ile ayarlayabilirsiniz.

var m map[string]int = map[string]int{"Foo": 20, "Bar": 30}

fmt.Println(m["Foo"]) // outputs 20

Aşağıdaki tüm ifadeler, değişkenin aynı değere bağlı olmasına neden olur.

// Declare, initializing to zero value, then assign a literal value.
var m map[string]int
m = map[string]int{}

// Declare and initialize via literal value.
var m = map[string]int{}

// Declare via short variable declaration and initialize with a literal value.
m := map[string]int{}

[Bazı başlangıç ​​anahtar/değer çiftleriyle yeni bir harita oluşturmak] için bir map değişmezi de kullanabiliriz(https://www.wikiod.com/tr/go/haritalar#Harita oluşturma) .

Anahtar türü, herhangi bir karşılaştırılabilir türü olabilir; özellikle, işlevleri, haritaları ve dilimleri içermez. Değer türü, özel türler veya “arayüz{}” dahil olmak üzere herhangi bir tür olabilir.

type Person struct {
    FirstName string
    LastName  string
}

// Declare via short variable declaration and initialize with make.
m := make(map[string]Person)

// Declare, initializing to zero value, then assign a literal value.
var m map[string]Person
m = map[string]Person{}

// Declare and initialize via literal value.
var m = map[string]Person{}

// Declare via short variable declaration and initialize with a literal value.
m := map[string]Person{}

Haritada öğe olup olmadığını kontrol edin #

Haritadan bir değer elde etmek için 00 gibi bir şey yapmanız yeterlidir.

value := mapName[ key ]

Harita anahtarı içeriyorsa, karşılık gelen değeri döndürür. Değilse, haritanın değer türünün sıfır değerini döndürür (“int” değerlerinin haritasıysa “0”, “dize” değerlerinin haritasıysa “”"…)

m  := map[string]string{"foo": "foo_value", "bar": ""}
k  := m["foo"]  // returns "foo_value" since that is the value stored in the map
k2 := m["bar"]  // returns "" since that is the value stored in the map
k3 := m["nop"]  // returns "" since the key does not exist, and "" is the string type's zero value

Boş değerler ve var olmayan anahtarlar arasında ayrım yapmak için, harita erişiminin ikinci döndürülen değerini kullanabilirsiniz (like value, hasKey := map["key"] kullanarak).

Bu ikinci değer “boolean” türündedir ve şöyle olacaktır:

  • değer haritada olduğunda ’true’,
  • harita verilen anahtarı içermediğinde ‘false’.

Aşağıdaki örneğe bakın:

value, hasKey = m[ key ]
if hasKey {
    // the map contains the given key, so we can safely use the value
    // If value is zero-value, it's because the zero-value was pushed to the map
} else {
    // The map does not have the given key
    // the value will be the zero-value of the map's type
}

Harita öğelerini sayma #

Yerleşik işlev len bir “harita” içindeki öğelerin sayısını döndürür

m := map[string]int{}
len(m) // 0

m["foo"] = 1
len(m) // 1

Bir değişken bir “nil” haritasını gösteriyorsa, “len” 0 değerini döndürür.

var m map[string]int
len(m) // 0

Bir haritanın sıfır değeri #

Bir “harita"nın sıfır değeri “sıfır"dır ve uzunluğu “0"dır.

var m map[string]string
fmt.Println(m == nil) // true
fmt.Println(len(m) ==0) // true

Bir “nil” haritasının anahtarı yoktur ve anahtar eklenemez. Bir “nil” haritası, okunursa boş bir harita gibi davranır, ancak yazıldığında çalışma zamanı paniğine neden olur.

var m map[string]string

// reading
m["foo"] == "" // true. Remember "" is the zero value for a string
_, ok = m["foo"] // ok == false

// writing
m["foo"] = "bar" // panic: assignment to entry in nil map

Sıfır değerli bir haritadan okumaya veya yazmaya çalışmamalısınız. Bunun yerine, kullanmadan önce haritayı (“make” veya atama ile) başlatın.

var m map[string]string
m = make(map[string]string) // OR m = map[string]string{}
m["foo"] = "bar"

Eşzamanlı Harita Erişimi #

Hareket halindeki haritalar eşzamanlılık için güvenli değildir. Onlara aynı anda erişecekseniz, bunları okumak ve yazmak için bir kilit almalısınız. Genellikle en iyi seçenek sync.RWMutex kullanmaktır çünkü okuma ve yazma kilitlerine sahip olabilirsiniz. Ancak, bir ‘sync.Mutex’ de kullanılabilir.

type RWMap struct {
    sync.RWMutex
    m map[string]int
}

// Get is a wrapper for getting the value from the underlying map
func (r RWMap) Get(key string) int {
    r.RLock()
    defer r.RUnlock()
    return r.m[key]
}

// Set is a wrapper for setting the value of a key in the underlying map
func (r RWMap) Set(key string, val int) {
    r.Lock()
    defer r.Unlock()
    r.m[key] = val
}

// Inc increases the value in the RWMap for a key.
//   This is more pleasant than r.Set(key, r.Get(key)++)
func (r RWMap) Inc(key string) {
    r.Lock()
    defer r.Unlock()
    r.m[key]++
}

func main() {

    // Init
    counter := RWMap{m: make(map[string]int)}

    // Get a Read Lock
    counter.RLock()
    _ = counter.["Key"]
    counter.RUnlock()

    // the above could be replaced with
    _ = counter.Get("Key")

    // Get a write Lock
    counter.Lock()
    counter.m["some_key"]++
    counter.Unlock()

    // above would need to be written as 
    counter.Inc("some_key")
}

Sarmalayıcı işlevlerinin değiş tokuşu, temel alınan haritanın genel erişimi ile uygun kilitlerin doğru kullanılması arasındadır.

Bir Haritayı Kopyala #

Dilimler gibi, haritalar da temel bir veri yapısına referanslar içerir. Yani değerini başka bir değişkene atayarak sadece referans iletilecektir. Haritayı kopyalamak için başka bir harita oluşturmak ve her bir değeri kopyalamak gerekir:

// Create the original map
originalMap := make(map[string]int)
originalMap["one"] = 1
originalMap["two"] = 2

// Create the target map
targetMap := make(map[string]int)

// Copy from the original map to the target map
for key, value := range originalMap {
  targetMap[key] = value
}

Bir haritanın öğelerini yineleme #

import fmt

people := map[string]int{
  "john": 30,
  "jane": 29,
  "mark": 11,
}

for key, value := range people {
  fmt.Println("Name:", key, "Age:", value)
}

Aralık döngüsüne sahip bir harita üzerinde yineleme yaparken, yineleme sırasının belirtilmediğini ve bir yinelemeden aynı olmasının garanti edilmediğini unutmayın. bir sonrakine.

Ayrıca, yalnızca [anahtarları kapmak](https://www.wikiod.com/tr/go/haritalar#Bir haritanın tuşlarını yineleme) arıyorsanız, haritanın anahtarlarını veya değerlerini de atabilirsiniz. -a-map#t=201607220900510314955) veya sadece değerleri alın.

Bir haritanın tuşlarını yineleme #

people := map[string]int{
  "john": 30,
  "jane": 29,
  "mark": 11,
}

for key, _ := range people {
  fmt.Println("Name:", key)
}

Anahtarları arıyorsanız, ilk değer oldukları için alt çizgiyi bırakabilirsiniz:

for key := range people {
  fmt.Println("Name:", key)
}

Aralık döngüsüne sahip bir harita üzerinde yineleme yaparken, yineleme sırasının belirtilmediğini ve bir yinelemeden aynı olmasının garanti edilmediğini unutmayın. bir sonrakine.

Harita oluşturma #

composite literal kullanılarak tek bir ifadede bir harita bildirilebilir ve başlatılabilir.

Otomatik tip Kısa değişken bildirimi kullanma:

mapIntInt := map[int]int{10: 100, 20: 100, 30: 1000}
mapIntString := map[int]string{10: "foo", 20: "bar", 30: "baz"}
mapStringInt := map[string]int{"foo": 10, "bar": 20, "baz": 30}
mapStringString := map[string]string{"foo": "one", "bar": "two", "baz": "three"}

Aynı kod, ancak Değişken türleriyle:

var mapIntInt = map[int]int{10: 100, 20: 100, 30: 1000}
var mapIntString = map[int]string{10: "foo", 20: "bar", 30: "baz"}
var mapStringInt = map[string]int{"foo": 10, "bar": 20, "baz": 30}
var mapStringString = map[string]string{"foo": "one", "bar": "two", "baz": "three"}

Bir haritaya kendi yapılarınızı da ekleyebilirsiniz:

Değer olarak özel türleri kullanabilirsiniz:

// Custom struct types
type Person struct {
  FirstName, LastName string
}

var mapStringPerson = map[string]Person{
  "john": Person{"John", "Doe"},
  "jane": Person{"Jane", "Doe"}}
mapStringPerson := map[string]Person{
  "john": Person{"John", "Doe"},
  "jane": Person{"Jane", "Doe"}}

Yapınız ayrıca haritanın key anahtarı olabilir:

type RouteHit struct {
    Domain string
    Route  string
}

var hitMap = map[RouteHit]int{
  RouteHit{"example.com","/home"}: 1,
  RouteHit{"example.com","/help"}: 2}
hitMap := map[RouteHit]int{
  RouteHit{"example.com","/home"}: 1,
  RouteHit{"example.com","/help"}: 2}

{} köşeli parantezleri içine herhangi bir değer girmeden boş bir harita oluşturabilirsiniz.

mapIntInt := map[int]int{}
mapIntString := map[int]string{}
mapStringInt := map[string]int{}
mapStringString := map[string]string{}
mapStringPerson := map[string]Person{}

Bir değişkene atamanıza gerek kalmadan doğrudan bir harita oluşturabilir ve kullanabilirsiniz. Ancak, hem beyanı hem de içeriği belirtmeniz gerekecektir.

// using a map as argument for fmt.Println()
fmt.Println(map[string]string{
  "FirstName": "John",
  "LastName": "Doe",
  "Age": "30"})

// equivalent to
data := map[string]string{
  "FirstName": "John",
  "LastName": "Doe",
  "Age": "30"}
fmt.Println(data)

Bir harita öğesini silme #

delete yerleşik işlevi, belirtilen anahtara sahip öğeyi bir haritadan kaldırır.

people := map[string]int{"john": 30, "jane": 29}
fmt.Println(people) // map[john:30 jane:29]

delete(people, "john")
fmt.Println(people) // map[jane:29]

“harita” “nil” ise veya böyle bir öğe yoksa, “sil"in bir etkisi yoktur.

people := map[string]int{"john": 30, "jane": 29}
fmt.Println(people) // map[john:30 jane:29]

delete(people, "notfound")
fmt.Println(people) // map[john:30 jane:29]

var something map[string]int
delete(something, "notfound") // no-op

Bir haritayı set olarak kullanma #

Bazı dillerin kümeler için yerel bir yapısı vardır. Go’da bir küme oluşturmak için, kümenin değer türünden boş bir yapıya (map[Type]struct{}) bir harita kullanmak en iyi uygulamadır.

Örneğin, dizelerle:

// To initialize a set of strings:
greetings := map[string]struct{}{
    "hi":    {},
    "hello": {},
}

// To delete a value:
delete(greetings, "hi")

// To add a value:
greetings["hey"] = struct{}{}

// To check if a value is in the set:
if _, ok := greetings["hey"]; ok {
    fmt.Println("hey is in greetings")
}

Bir haritanın değerlerini yineleme #

people := map[string]int{
  "john": 30,
  "jane": 29,
  "mark": 11,
}

for _, value := range people {
  fmt.Println("Age:", value)
}

Aralık döngüsüne sahip bir harita üzerinde yineleme yaparken, yineleme sırasının belirtilmediğini ve bir yinelemeden aynı olmasının garanti edilmediğini unutmayın. bir sonrakine.

Değer olarak dilimlerle haritalar oluşturma #

m := make(map[string][]int)

Var olmayan bir anahtara erişmek, değer olarak sıfır bir dilim döndürür. Sıfır dilimler, “append” veya diğer yerleşik işlevlerle kullanıldığında sıfır uzunlukta dilimler gibi davrandığından, normalde bir anahtarın olup olmadığını kontrol etmeniz gerekmez:

// m["key1"] == nil && len(m["key1"]) == 0
m["key1"] = append(m["key1"], 1)
// len(m["key1"]) == 1

Haritadan bir anahtarın silinmesi, anahtarı sıfır dilime geri döndürür:

delete(m, "key1")
// m["key1"] == nil
