Sözdizimi #

  • func Marshal(v interface{}) ([]bayt, hata)
  • func Unmarshal(veri []bayt, v arayüzü{}) hatası

["encoding/json"``][1] json paketi paketi, Go` içinde JSON nesnelerinin kodlanmasını ve kodunun çözülmesini uygular.

JSON’daki türler, Go’daki karşılık gelen somut türleriyle birlikte:

JSON Türü Git Beton Tipi
boole bool
sayılar float64 veya int
dize dize
boş sıfır

Temel JSON kod çözme #

"encoding/json" paketindeki json.Unmarshal bir JSON değerinin kodunu verilen değişken tarafından gösterilen değere çözer.

Parametreler, “[]bayt” cinsinden kodu çözülecek değer ve seri durumdan çıkarılmış değer için depolama olarak kullanılacak bir değişkendir. Döndürülen değer bir hatadır (başarısızlık durumunda).

encodedValue := []byte(`{"London":18,"Rome":30}`)

// generic storage for the decoded JSON
var data map[string]interface{}

// decode the value into data
// notice that we must pass the pointer to data using &data
err := json.Unmarshal(encodedValue, &data)

// check if the decoding is successful
if err != nil {
    panic(err)
}

fmt.Println(data)
map[London:18 Rome:30]

Oyun Alanı

Yukarıdaki örnekte hem anahtarın türünü hem de değeri önceden nasıl bildiğimize dikkat edin. Ancak bu her zaman böyle değildir. Aslında, çoğu durumda JSON, karma değer türleri içerir.

encodedValue := []byte(`{"city":"Rome","temperature":30}`)

// generic storage for the decoded JSON
var data map[string]interface{}

// decode the value into data
if err := json.Unmarshal(encodedValue, &data); err != nil {
    panic(err)
}

// if you want to use a specific value type, we need to cast it
temp := data["temperature"].(float64)
fmt.Println(temp) // 30
city := data["city"].(string)
fmt.Println(city) // "Rome"

Oyun Alanı

Yukarıdaki son örnekte, kodu çözülen değeri depolamak için genel bir harita kullandık. Bir map[string]interface{} kullanmalıyız çünkü anahtarların dizgeler olduğunu biliyoruz, ancak değerlerinin türünü önceden bilmiyoruz.

Bu çok basit bir yaklaşımdır, ancak aynı zamanda son derece sınırlıdır. Gerçek dünyada, genellikle [bir JSON’un kodunu özel olarak tanımlanmış bir “yapı” türüne dönüştürürsünüz](https://www.wikiod.com/tr/go/json =201607220810357745507).

Kod çözme için anonim yapıları kullanma #

Anonim yapıları kullanmanın amacı, uygulamamızı yalnızca tek bir işlevde kullanılan türlerle doldurmadan yalnızca önemsediğimiz bilgilerin kodunu çözmektir.

jsonBlob := []byte(`
  {
    "_total": 1,
    "_links": {
      "self": "https://api.twitch.tv/kraken/channels/foo/subscriptions?direction=ASC&limit=25&offset=0",
      "next": "https://api.twitch.tv/kraken/channels/foo/subscriptions?direction=ASC&limit=25&offset=25"
    },
    "subscriptions": [
      {
        "created_at": "2011-11-23T02:53:17Z",
        "_id": "abcdef0000000000000000000000000000000000",
        "_links": {
          "self": "https://api.twitch.tv/kraken/channels/foo/subscriptions/bar"
        },
        "user": {
          "display_name": "bar",
          "_id": 123456,
          "name": "bar",
          "staff": false,
          "created_at": "2011-06-16T18:23:11Z",
          "updated_at": "2014-10-23T02:20:51Z",
          "logo": null,
          "_links": {
            "self": "https://api.twitch.tv/kraken/users/bar"
          }
        }
      }
    ]
  }
`)

var js struct {
    Total int `json:"_total"`
    Links struct {
        Next string `json:"next"`
    } `json:"_links"`
    Subs []struct {
        Created string `json:"created_at"`
        User    struct {
            Name string `json:"name"`
            ID   int    `json:"_id"`
        } `json:"user"`
    } `json:"subscriptions"`
}

err := json.Unmarshal(jsonBlob, &js)
if err != nil {
    fmt.Println("error:", err)
}
fmt.Printf("%+v", js)

Çıktı: {Toplam:1 Bağlantı:{Sonraki:https://api.twitch.tv/kraken/channels/foo/subscriptions?direction=ASC&limit=25&offset=25} Abonelikler:[{Oluşturuldu:2011-11-23T02:53: 17Z Kullanıcısı:{Ad:bar Kimliği:123456}}]}

Oyun Alanı

Genel durum için ayrıca bakınız: https://www.wikiod.com/tr/go/json

Temel JSON Kodlaması #

"encoding/json" paketindeki json.Marshal JSON’a bir değer kodlar.

Parametre, kodlanacak değerdir. Döndürülen değerler, JSON kodlu girişi (başarı durumunda) ve bir hatayı (başarısızlık durumunda) temsil eden bir bayt dizisidir.

decodedValue := []string{"foo", "bar"}

// encode the value
data, err := json.Marshal(decodedValue)

// check if the encoding is successful
if err != nil {
    panic(err)
}

// print out the JSON-encoded string
// remember that data is a []byte
fmt.Println(string(data))
// "["foo","bar"]"

Oyun Alanı

Yerleşik veri türleri için bazı temel kodlama örnekleri:

var data []byte

data, _ = json.Marshal(1)
fmt.Println(string(data))
// 1

data, _ = json.Marshal("1")
fmt.Println(string(data))
// "1"

data, _ = json.Marshal(true)
fmt.Println(string(data))
// true

data, _ = json.Marshal(map[string]int{"London": 18, "Rome": 30})
fmt.Println(string(data))
// {"London":18,"Rome":30}

Oyun Alanı

Basit değişkenleri kodlamak, JSON kodlamasının Go’da nasıl çalıştığını anlamanıza yardımcı olur. Ancak gerçek dünyada, büyük olasılıkla yapılarda depolanan daha karmaşık verileri kodlarsınız.

Bir dosyadan JSON verilerinin kodunu çözme #

JSON verileri dosyalardan da okunabilir.

Aşağıdaki içeriğe sahip “data.json” adında bir dosyamız olduğunu varsayalım:

[
    {
      "Name" : "John Doe",
      "Standard" : 4
    },
    {
      "Name" : "Peter Parker",
      "Standard" : 11
    },
    {
      "Name" : "Bilbo Baggins",
      "Standard" : 150
    }
]

Aşağıdaki örnek dosyayı okur ve içeriğin kodunu çözer:

package main

import (
    "encoding/json"
    "fmt"
    "log"
    "os"
)

type Student struct {
    Name     string
    Standard int `json:"Standard"`
}

func main() {
    // open the file pointer
    studentFile, err := os.Open("data.json")
    if err != nil {
        log.Fatal(err)
    }
    defer studentFile.Close()

    // create a new decoder
    var studentDecoder *json.Decoder = json.NewDecoder(studentFile)
    if err != nil {
        log.Fatal(err)
    }

    // initialize the storage for the decoded data
    var studentList []Student
    
    // decode the data
    err = studentDecoder.Decode(&studentList)
    if err != nil {
        log.Fatal(err)
    }

    for i, student := range studentList {
        fmt.Println("Student", i+1)
        fmt.Println("Student name:", student.Name)
        fmt.Println("Student standard:", student.Standard)
    }
}

“data.json” dosyası, Go yürütülebilir programıyla aynı dizinde olmalıdır. Go’da dosyalarla nasıl çalışılacağı hakkında daha fazla bilgi için Go File I/O belgeleri bölümünü okuyun.

JSON yapı alanlarını yapılandırma #

Aşağıdaki örneği göz önünde bulundurun:

type Company struct {
    Name     string
    Location string
}

Belirli Alanları Gizle/Atla #

“Gelir” ve “Satışlar"ı dışa aktarmak, ancak bunları kodlama/kod çözme işleminden gizlemek için “json:”-"` kullanın veya değişkeni küçük harfle başlayacak şekilde yeniden adlandırın. Bunun, değişkenin paketin dışında görünmesini engellediğini unutmayın.

type Company struct {
    Name     string `json:"name"`
    Location string `json:"location"`
    Revenue  int    `json:"-"`
    sales    int
}

Boş Alanları Yoksay #

Sıfır değerine ayarlandığında “Location"ın JSON’a dahil edilmesini önlemek için “json” etiketine “,omitempty” ekleyin.

type Company struct {
    Name     string `json:"name"`
    Location string `json:"location,omitempty"`
}

Oyun Alanındaki Örnek

Yapıları özel alanlarla sıralama #

İyi bir geliştirici olarak, hem dışa aktarılan hem de aktarılmayan alanlar ile aşağıdaki yapıyı oluşturdunuz:

type MyStruct struct {
    uuid string    
    Name string
}

Oyun Alanındaki Örnek: https://play.golang.org/p/Zk94Il2ANZ

Şimdi bu yapıyı etcd gibi bir şeyde depolamak için geçerli JSON’a Marshal() yapmak istiyorsunuz. Ancak, “uuid” dışa aktarılmadığından, “json.Marshal()” bunu atlar. Ne yapalım? Anonim bir yapı ve json.MarshalJSON() arayüzünü kullanın! İşte bir örnek:

type MyStruct struct {
    uuid string    
    Name string
}

func (m MyStruct) MarshalJSON() ([]byte, error {
    j, err := json.Marshal(struct {
        Uuid string
        Name string
    }{
        Uuid: m.uuid,
        Name: m.Name,
    })
    if err != nil {
           return nil, err
    }
    return j, nil
}

Oyun Alanındaki Örnek: https://play.golang.org/p/Bv2k9GgbzE

Go yapılarını kullanarak kodlama/kod çözme #

Bir “Şehir” türünü tanımlayan aşağıdaki “yapı"ya sahip olduğumuzu varsayalım:

type City struct {  
    Name string  
    Temperature int  
}

encoding/json paketini kullanarak City değerlerini kodlayabilir/kodunu çözebiliriz.

Her şeyden önce, kodlayıcıya yapı alanları ve JSON anahtarları arasındaki yazışmayı söylemek için Go meta verilerini kullanmamız gerekiyor.

type City struct {  
    Name string `json:"name"`  
    Temperature int `json:"temp"`  
    // IMPORTANT: only exported fields will be encoded/decoded  
    // Any field starting with a lower letter will be ignored  
}

Bu örneği basit tutmak için alanlar ve anahtarlar arasında açık bir yazışma bildireceğiz. Ancak, belgelerde açıklandığı gibi “json:” meta verilerinin çeşitli türevlerini kullanabilirsiniz.

ÖNEMLİ: Yalnızca [dışa aktarılan alanlar](https://www.wikiod.com/tr/go/yaplar#Dışa Aktarılan ve Dışa Aktarılmayan Alanlar (Özel ve Genel)) (büyük harfli alanlar) seri hale getirilecek/seri hale getirilecektir. Örneğin, alanı tsıcaklık olarak adlandırırsanız, “json” meta verilerini ayarlasanız bile yok sayılır.

Kodlama #

Bir “City” yapısını kodlamak için, temel örnekte olduğu gibi “json.Marshal” kullanın:

// data to encode  
city := City{Name: "Rome", Temperature: 30}  
 
// encode the data  
bytes, err := json.Marshal(city)  
if err != nil {  
    panic(err)  
}  
 
fmt.Println(string(bytes))  
// {"name":"Rome","temp":30}

Oyun Alanı

Kod çözme #

Bir “City” yapısının kodunu çözmek için, temel örnekte olduğu gibi “json.Unmarshal” öğesini kullanın:

// data to decode  
bytes := []byte(`{"name":"Rome","temp":30}`)  
 
// initialize the container for the decoded data  
var city City  
 
// decode the data  
// notice the use of &city to pass the pointer to city  
if err := json.Unmarshal(bytes, &city); err != nil {  
    panic(err)  
}  
 
fmt.Println(city)  
// {Rome 30}

Oyun Alanı
