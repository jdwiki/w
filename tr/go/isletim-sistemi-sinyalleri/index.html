Sözdizimi #

  • func Notify(c chan<- os.Signal, sig …os.Signal)

Parametreler #

Parametre Ayrıntılar
c chan<- os.Signal Özellikle “os.Signal” türünden “kanal” alma; sigChan := make(chan os.Signal) ile kolayca oluşturulur
işaret …os.Sinyal Bu “kanal"ı yakalamak ve göndermek için “os.Signal” türlerinin listesi. Daha fazla seçenek için https://golang.org/pkg/syscall/#pkg-constants adresine bakın.

Bir kanala sinyal atama #

Çoğu zaman, programınıza işletim sistemi tarafından durdurulması ve durumu korumak veya uygulamanızı temizlemek için bazı eylemler gerçekleştirmesi söylendiğinde yakalamak için bir nedeniniz olacaktır. Bunu başarmak için standart kitaplıktan ‘os/signal’ paketini kullanabilirsiniz. Aşağıda, sistemden gelen tüm sinyalleri bir kanala atamanın basit bir örneği ve ardından bu sinyallere nasıl tepki verileceği yer almaktadır.

package main

import (
    "fmt"
    "os"
    "os/signal"
)

func main() {
    // create a channel for os.Signal
    sigChan := make(chan os.Signal)

    // assign all signal notifications to the channel 
    signal.Notify(sigChan)

    // blocks until you get a signal from the OS
    select {
    // when a signal is received
    case sig := <-sigChan:
        // print this line telling us which signal was seen
        fmt.Println("Received signal from OS:", sig)
    }
}

Yukarıdaki betiği çalıştırdığınızda, bir kanal oluşturacak ve ardından o kanal bir sinyal alana kadar engelleyecektir.

$ go run signals.go 
^CReceived signal from OS: interrupt

Yukarıdaki “^C”, “SIGINT” sinyalini gönderen “CTRL+C” klavye komutudur.
