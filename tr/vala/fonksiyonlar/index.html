Fonksiyonlar, programınızın diğer fonksiyonları tarafından çalıştırılabilen kod parçalarıdır.

Programınız her zaman ‘ana’ işleviyle başlar.

Ayrıca bkz. eşzamansız işlevler.

Yöntemler, işlevle tamamen aynıdır, ancak bir nesne örneği üzerinde hareket ederler.

Temel fonksiyonlar #

Bir işlev, en azından dönüş türü ve benzersiz bir adla tanımlanır.

void say_hello () {
    print ("Hello, world!\n");   
}

Ardından, onu çağırmak için sadece fonksiyonun adını ve ardından bir parantez kullanın.

say_hello ();

İşlevler ayrıca parantezler arasında türlerine ve adlarına göre tanımlanmış ve virgülle ayrılmış parametrelere sahip olabilir. O zaman bunları fonksiyonunuzda normal değişkenler olarak kullanabilirsiniz.

int greet (string name, string family_name) {
    print ("Hello, %s %s!\n", name, family_name);
}

Parametreli bir işlevi çağırmak için parantezler arasına bir değişken veya değer koymanız yeterlidir.

string name = "John";
greet (name, "Doe");

Ayrıca return anahtar kelimesiyle bir değişkene atanabilecek bir değer de döndürebilirsiniz.

int add (int a, int b) {
    return a + b;
}

int sum = add (24, 18);

Tüm kod yolu bir “return” ifadesi ile bitmelidir. Örneğin, aşağıdaki kod geçersiz.

int positive_sub (int a, int b) {
    if (a >= b) {
        return a - b;
    } else {
        // Nothing is returned in this case.
        print ("%d\n", b - a);
    }
}

İsteğe bağlı parametreler #

Parametreler, varsayılan bir değer verilerek isteğe bağlı olarak işaretlenebilir. İşlev çağrılırken isteğe bağlı parametreler atlanabilir.

string greet (string name, string language = "English") {
    if (language == "English") {
        return @"Hello, $name!";
    } else {
        return @"Sorry $name, I don't speak $language";
    }
}

greet ("John");
greet ("Jane", "Italian");

Out ve Ref parametreleri #

Değer türleri (yapılar ve numaralandırmalar) değerlere göre işlevlere iletilir: değişkene bir referans değil, işleve bir kopya verilecektir. Yani aşağıdaki fonksiyon hiçbir şey yapmaz.

void add_three (int x) {
    x += 3;
}

int a = 39;
add_three (a);
assert (a == 39); // a is still 39

Bu davranışı değiştirmek için ‘ref’ anahtar sözcüğünü kullanabilirsiniz.

// Add it to the function declaration
void add_three (ref int x) {
    x += 3;
}

int a = 39;
add_three (ref a); // And when you call it
assert (a == 42); // It works!

‘out’ aynı şekilde çalışır, ancak işlevin bitiminden önce bu değişkene bir değer ayarlamanız gerekir.

string content;
FileUtils.get_contents ("file.txt", out content);

// OK even if content was not initialized, because
// we are sure that it got a value in the function above.
print (content);

Sözleşme programlama #

Parametrelerin belirli değerlere sahip olduğunu ‘requires’ ile belirtebilirsiniz.

int fib (int i) requires (i > 0) {
    if (i == 1) {
        return i;
    } else {
        return fib (i - 1) + fib (i - 2);
    }
}

fib (-1);

Derleme sırasında herhangi bir hata almazsınız, ancak programınızı çalıştırırken bir hata alırsınız ve fonksiyon çalışmaz.

Ayrıca, dönüş değerinin belirli bir koşulla eşleştiğini “sağlar” ile de iddia edebilirsiniz.

int add (int a, int b) ensures (result >= a && result >= b) {
    return a + b;
}

İstediğiniz kadar “gereksinim” ve “sağlar” a sahip olabilirsiniz.

Değişken bağımsız değişkenler #

int sum (int x, ...) {
    int result = x;
    va_list list = va_list ();
    for (int? y = list.arg<int?> (); y != null; y = list.arg<int?> ()) {
        result += y;
    }
    return result;
}

int a = sum (1, 2, 3, 36);

Bu fonksiyon ile istediğiniz kadar int geçirebilirsiniz. Başka bir şey iletirseniz, beklenmedik bir değer veya segmentasyon hatası alırsınız.
