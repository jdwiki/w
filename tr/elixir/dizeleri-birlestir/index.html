Dize İnterpolasyonunu Kullanma #

iex(1)> [x, y] = ["String1", "String2"]
iex(2)> "#{x} #{y}"
# "String1 String2"

IO Listesini Kullanma #

["String1", " ", "String2"] |> IO.iodata_to_binary 
# "String1 String2"

Bu, bellekte kopyalanmayan dizeler olarak bazı performans artışları sağlayacaktır. Alternatif yöntem:

iex(1)> IO.puts(["String1", " ", "String2"])
# String1 String2

Enum.join’i kullanma #

Enum.join(["String1", "String2"], " ")  
# "String1 String2"
