Elixir'de fonksiyonel programlama

  • Harita
  • Azaltmak

Bu sayfada

  • Harita
  • Azaltmak

İksir kullanarak harita ve azaltma gibi temel üst düzey işlevleri uygulamaya çalışalım.

Harita #

Harita, bir dizi ve bir işlev alacak ve bu işlevi o listedeki her öğeye uyguladıktan sonra bir dizi döndürecek bir işlevdir.

defmodule MyList do
  def map([], _func) do
    []
  end

  def map([head | tail], func) do
    [func.(head) | map(tail, func)]
  end
end

‘iex’ içine yapıştırmayı kopyalayın ve yürütün:

MyList.map [1,2,3], fn a -> a * 5 end

Steno sözdizimi MyList.map [1,2,3], &(&1 * 5) şeklindedir.

Azaltmak #

Azaltma, bir dizi, işlev ve akümülatör alacak ve bir sonraki akümülatörü vermek için ilk öğeyle yinelemeyi başlatmak için *** akümülatörü tohum olarak kullanacak ve yineleme dizideki tüm öğeler için devam edecek bir işlevdir*** (aşağıya bakın) örnek)

defmodule MyList do
  def reduce([], _func, acc) do
    acc
  end

  def reduce([head | tail], func, acc) do
    reduce(tail, func, func.(acc, head))
  end
end

Yukarıdaki pasajı kopyalayıp iex’e yapıştırın:

  1. Bir dizideki tüm sayıları eklemek için: `MyList.reduce [1,2,3,4], fn acc, element -> acc + element end, 0’
  2. Bir dizideki tüm sayıları çarpmak için: `MyList.reduce [1,2,3,4], fn acc, element -> acc * element end, 1'

Örnek 1: için açıklama

Iteration 1 => acc = 0, element = 1 ==> 0 + 1 ===> 1 = next accumulator
Iteration 2 => acc = 1, element = 2 ==> 1 + 2 ===> 3 = next accumulator
Iteration 3 => acc = 3, element = 3 ==> 3 + 3 ===> 6 = next accumulator
Iteration 4 => acc = 6, element = 4 ==> 6 + 4 ===> 10 = next accumulator = result(as all elements are done)

Küçült’ü kullanarak listeyi filtreleyin

MyList.reduce [1,2,3,4], fn acc, element -> if rem(element,2) == 0 do acc else acc ++ [element] end end, []
