Dizeyi büyük harfe dönüştürme #

’toupper’ işlevi bir dizeyi büyük harfe (büyük harflere) dönüştürür. Örneğin:

BEGIN {
    greeting = "hello"
    loud_greeting = toupper(greeting)
    print loud_greeting
}

Bu kod çalıştırıldığında “HELLO” çıktısını verecektir.

Bir dizgenin karmasını hesaplama #

awk‘da standart karma algoritmalardan birini uygulamak muhtemelen sıkıcı bir iş olsa da, metin belgelerinin tanıtıcısı olarak kullanılabilecek bir karma işlevi tanımlamak çok daha izlenebilir. Böyle bir işlevin yararlı olduğu pratik bir durum, örneğin test durumları gibi açıklamaları verilen öğelere kısa kimlikler atamaktır, böylece kısa kimlik, uzun açıklamasını sağlamak yerine kullanıcı tarafından öğeye referans olarak verilebilir.

hash işlevinin karakterleri, betiğin başlangıcında başlatılan bir arama tablosu kullanılarak gerçekleştirilen sayısal kodlara dönüştürmesi gerekir. hash işlevi daha sonra, karmaların hesaplanmasında çok klasik bir yaklaşım olan modüler aritmetik dönüşümler kullanılarak hesaplanır.

Gösteri amacıyla, giriş satırlarını hash’leriyle süslemek için bir kural ekliyoruz, ancak işlevi kullanmak için bu kurala gerek yok:

BEGIN{
  for(n=0;n<256;n++) {
    ord[sprintf("%c",n)] = n
  }
}

function hash(text, _prime, _modulo, _ax, _chars, _i)
{
  _prime = 104729;
  _modulo = 1048576;
  _ax = 0;
  split(text, _chars, "");
  for (_i=1; _i <= length(text); _i++) {
    _ax = (_ax * _prime + ord[_chars[_i]]) % _modulo;
  };
  return sprintf("%05x", _ax)
}

# Rule to demonstrate the function
#  These comments and the following line are not relevant
#  to the definition of the hash function but illustrate
#  its use.

{ printf("%s|%s\n", hash($0), $0) }

Animal Farm
EOF

Dize Birleştirme #

Dize birleştirme, herhangi bir operatör olmadan ifadelerin yan yana yazılmasıyla yapılır. Örneğin:

BEGIN {
   user = "root"
   print "Hello "user "!"
}

yazdıracak: Merhaba kök!

İfadelerin boşlukla ayrılması gerekmediğini unutmayın.

Dize metni değiştirme #

SUB işlevi, awk içindeki metni değiştirmeye izin verir

Dizeyi küçük harfe dönüştür #

AWK genellikle bir dizi dizi içeren dosyaların tamamını işlemek için kullanılır. Diyelim ki awk_test_file.txt dosyası şunları içeriyor:

First String
Second String
Third String

Tüm dizeleri küçük harfe dönüştürmek için yürütün:

awk '{ print tolower($0) }' awk_test_file.txt

Bunun sonucu:

first string
second string
third string

Alt dize çıkarma #

GNU awk, bir ana dizeden sabit uzunlukta bir karakter dizisi döndürmek için bir alt dize çıkarma işlevini destekler. sözdizimi

*substr(string, start [, length ])*

burada, “dize” kaynak dizedir ve “başlangıç”, isteğe bağlı uzunluktaki “uzunluk” karakterleri için çıkarmanın yapılmasını istediğiniz alt dize konumunun başlangıcını işaretler. Uzunluk belirtilmemişse, dizenin sonuna kadar çıkarma yapılır.

Dizenin ilk karakteri bir numaralı karakter olarak kabul edilir.

awk '
BEGIN {
    testString = "MyTESTstring"
    substring  =  substr(testString, 3, 4)    # Start at character 3 for a length of 4 characters
    print substring
}'

‘TEST’ alt dizesini çıkaracaktır.

awk '
BEGIN {
    testString = "MyTESTstring"
    substring  =  substr(testString, 3)    # Start at character 3 till end of the string
    print substring
}'

bu, alt dizeyi karakter konumu 3’ten tüm dizenin sonuna kadar çıkararak ‘TESTstring’ değerini döndürür.

Not:-

  • “start"a negatif bir değer verilirse, “GNU” awk tüm dizeyi yazdırır ve “uzunluk"a sıfır olmayan bir değer verilirse “GNU” awk davranışı bir “boş” dize döndürür ve davranış, farklı uygulamalar arasında değişir. “awk”.
