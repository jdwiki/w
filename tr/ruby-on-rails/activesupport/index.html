ActiveSupport, Rails çerçevesinin geri kalanı tarafından kullanılan genel amaçlı araçlardan oluşan bir yardımcı program mücevheridir.

Bu araçları sağlamanın başlıca yollarından biri, Ruby’nin yerel türlerini eşleme yapmaktır. Bunlara Çekirdek Uzantılar denir.

Çekirdek Uzantıları: Dize Erişimi #

String#at #

Bir dize nesnesinin alt dizesini döndürür. “Dize#[]” ile aynı arayüz.

str = "hello"
str.at(0)      # => "h"
str.at(1..3)   # => "ell"
str.at(-2)     # => "l"
str.at(-2..-1) # => "lo"
str.at(5)      # => nil
str.at(5..-1)  # => ""

[Dize#den][den] #

Verilen konumdan dizenin sonuna kadar bir alt dize döndürür.

str = "hello"
str.from(0)  # => "hello"
str.from(3)  # => "lo"
str.from(-2) # => "lo"

Dize#to #

Dizenin başından verilen konuma bir alt dize döndürür. Konum negatifse, dizenin sonundan sayılır.

str = "hello"
str.to(0)  # => "h"
str.to(3)  # => "hell"
str.to(-2) # => "hell"

“from” ve “to” birlikte kullanılabilir.

str = "hello"
str.from(0).to(-1) # => "hello"
str.from(1).to(-2) # => "ell"

Dize#ilk #

İlk karakteri veya dizenin uzunluğuna kadar belirli sayıda karakteri döndürür.

str = "hello"
str.first    # => "h"
str.first(1) # => "h"
str.first(2) # => "he"
str.first(0) # => ""
str.first(6) # => "hello"

Dize#son #

Geriye doğru sayarak dizenin sonundaki son karakteri veya belirli sayıda karakteri döndürür.

str = "hello"
str.last    # => "o"
str.last(1) # => "o"
str.last(2) # => "lo"
str.last(0) # => ""
str.last(6) # => "hello"

Çekirdek Uzantıları: Dize - Tarih/Saat Dönüşümü #

Dize#to_time #

Bir dizeyi bir Zaman değerine dönüştürür. “form” parametresi “:utc” veya “:local” olabilir, varsayılan olarak “:local” olur.

"13-12-2012".to_time               # => 2012-12-13 00:00:00 +0100
"06:12".to_time                    # => 2012-12-13 06:12:00 +0100
"2012-12-13 06:12".to_time         # => 2012-12-13 06:12:00 +0100
"2012-12-13T06:12".to_time         # => 2012-12-13 06:12:00 +0100
"2012-12-13T06:12".to_time(:utc)   # => 2012-12-13 06:12:00 UTC
"12/13/2012".to_time               # => ArgumentError: argument out of range

[Dize#to_date][to_date] #

Bir dizeyi bir Tarih değerine dönüştürür.

"1-1-2012".to_date   # => Sun, 01 Jan 2012
"01/01/2012".to_date # => Sun, 01 Jan 2012
"2012-12-13".to_date # => Thu, 13 Dec 2012
"12/13/2012".to_date # => ArgumentError: invalid date

String#to_datetime #

Bir dizeyi DateTime değerine dönüştürür.

"1-1-2012".to_datetime            # => Sun, 01 Jan 2012 00:00:00 +0000
"01/01/2012 23:59:59".to_datetime # => Sun, 01 Jan 2012 23:59:59 +0000
"2012-12-13 12:50".to_datetime    # => Thu, 13 Dec 2012 12:50:00 +0000
"12/13/2012".to_datetime          # => ArgumentError: invalid date

Çekirdek Uzantıları: Dize Hariç Tutma #

Dize#hariç tut? #

String#include? ifadesinin tersi

"hello".exclude? "lo" # => false
"hello".exclude? "ol" # => true
"hello".exclude? ?h   # => false

Çekirdek Uzantıları: Dize Filtreleri #

Dize#squish #

Verilen dizenin başında veya sonunda boşluk bırakmadan bir sürümünü döndürür ve iç kısımdaki tüm ardışık boşlukları tek boşluklarla birleştirir. Yıkıcı sürüm squish! doğrudan dize örneğinde çalışır.

Hem ASCII hem de Unicode boşluklarını işler.

%{ Multi-line
   string }.squish                   # => "Multi-line string"
" foo   bar    \n   \t   boo".squish # => "foo bar boo"

Dize#kaldır #

Tüm kalıpların kaldırıldığı yeni bir dize döndürür. Yıkıcı sürüm remove! doğrudan verilen dizi üzerinde çalışır.

str = "foo bar test"
str.remove(" test")                 # => "foo bar"
str.remove(" test", /bar/)          # => "foo "

[Dize#kes][kes] #

Dize uzunluğundan daha uzunsa, belirli bir dizenin belirli bir uzunlukta kesilmiş bir kopyasını döndürür.

'Once upon a time in a world far far away'.truncate(27)
# => "Once upon a time in a wo..."

Doğal bir molada kesmek için bir dize veya regexp :separator iletin

'Once upon a time in a world far far away'.truncate(27, separator: ' ')
# => "Once upon a time in a..."

'Once upon a time in a world far far away'.truncate(27, separator: /\s/)
# => "Once upon a time in a..."

Dize#truncate_words #

Belirli sayıda sözcükten sonra kesilen bir dize döndürür.

'Once upon a time in a world far far away'.truncate_words(4)
# => "Once upon a time..."

Farklı bir sözcük ayırıcısı belirtmek için bir dize veya normal ifade iletin

'Once<br>upon<br>a<br>time<br>in<br>a<br>world'.truncate_words(5, separator: '<br>')
# => "Once<br>upon<br>a<br>time<br>in..."

Son karakterler :omission dizesiyle değiştirilecektir (varsayılanı “…” şeklindedir)

'And they found that many people were sleeping better.'.truncate_words(5, omission: '... (continued)')
# => "And they found that many... (continued)"

String#strip_heredoc #

Heredocs’ta girintileri çıkarır. En az girintili boş olmayan satırı arar ve bu miktardaki baştaki boşlukları kaldırır.

if options[:usage]
  puts <<-USAGE.strip_heredoc
    This command does such and such.

    Supported options are:
      -h         This message
      ...
  USAGE
end

kullanıcı görecekti

This command does such and such.

Supported options are:
-h         This message
...

Çekirdek Uzantıları: Dize Bükümü #

Dize#çoğullaştır #

Dizenin çoğul biçimini döndürür. İsteğe bağlı olarak bir ‘count’ parametresi alır ve ‘count == 1’ ise tekil formu döndürür. Ayrıca dile özgü çoğullaştırma için bir “yerel” parametreyi kabul eder.

'post'.pluralize             # => "posts"
'octopus'.pluralize          # => "octopi"
'sheep'.pluralize            # => "sheep"
'words'.pluralize            # => "words"
'the blue mailman'.pluralize # => "the blue mailmen"
'CamelOctopus'.pluralize     # => "CamelOctopi"
'apple'.pluralize(1)         # => "apple"
'apple'.pluralize(2)         # => "apples"
'ley'.pluralize(:es)         # => "leyes"
'ley'.pluralize(1, :es)      # => "ley"

[String#singularize][tekilize] #

Dizenin tekil formunu döndürür. İsteğe bağlı bir “yerel ayar” parametresini kabul eder.

'posts'.singularize            # => "post"
'octopi'.singularize           # => "octopus"
'sheep'.singularize            # => "sheep"
'word'.singularize             # => "word"
'the blue mailmen'.singularize # => "the blue mailman"
'CamelOctopi'.singularize      # => "CamelOctopus"
'leyes'.singularize(:es)       # => "ley"

[Dize#sabitleştir][sabitleştir] #

Dizede belirtilen ada sahip bildirilen bir sabit bulmaya çalışır. Ad CamelCase’de olmadığında veya başlatılmadığında bir ‘NameError’ oluşturur.

'Module'.constantize  # => Module
'Class'.constantize   # => Class
'blargle'.constantize # => NameError: wrong constant name blargle

String#safe_constantize #

Bir “constantize” gerçekleştirir ancak “NameError"u yükseltmek yerine “nil” döndürür.

'Module'.safe_constantize  # => Module
'Class'.safe_constantize   # => Class
'blargle'.safe_constantize # => nil

[Dize#camelize][develeştirme] #

Dizeleri varsayılan olarak UpperCamelCase’e dönüştürür, eğer :lower param olarak verilirse bunun yerine altCamelCase’e dönüştürür.

takma ad: “deve çantası”

Not: ayrıca yolları ad alanlarına dönüştürmek için yararlı olan / karakterini :: biçimine dönüştürür.

'active_record'.camelize                # => "ActiveRecord"
'active_record'.camelize(:lower)        # => "activeRecord"
'active_record/errors'.camelize         # => "ActiveRecord::Errors"
'active_record/errors'.camelize(:lower) # => "activeRecord::Errors"

[Dize#başlıklandırma][başlıklandırma] #

Tüm kelimeleri büyük harf yapar ve daha güzel görünen bir başlık oluşturmak için dizedeki bazı karakterleri değiştirir.

takma ad: “başlık”

'man from the boondocks'.titleize # => "Man From The Boondocks"
'x-men: the last stand'.titleize  # => "X Men: The Last Stand"

Dize#alt çizgi #

Dizedeki ifadeden altı çizili, küçük harfli bir form oluşturur. ‘develeştirme’nin tersi.

Not: “alt çizgi” ayrıca ad alanlarını yollara dönüştürmek için “::” öğesini “/” olarak değiştirir.

'ActiveModel'.underscore         # => "active_model"
'ActiveModel::Errors'.underscore # => "active_model/errors"

[Dize#dasherize][tire] #

Dizedeki alt çizgileri kısa çizgilerle değiştirir.

'puni_puni'.dasherize # => "puni-puni"

String#demodulize #

Dizedeki sabit ifadeden modül bölümünü kaldırır.

'ActiveRecord::CoreExtensions::String::Inflections'.demodulize # => "Inflections"
'Inflections'.demodulize                                       # => "Inflections"
'::Inflections'.demodulize                                     # => "Inflections"
''.demodulize                                                  # => ''

Dize#deconstantize #

Dizedeki sabit ifadeden en sağdaki segmenti kaldırır.

'Net::HTTP'.deconstantize   # => "Net"
'::Net::HTTP'.deconstantize # => "::Net"
'String'.deconstantize      # => ""
'::String'.deconstantize    # => ""
''.deconstantize            # => ""

String#parameterize #

‘Güzel’ bir URL’nin parçası olarak kullanılabilmesi için bir dizedeki özel karakterleri değiştirir.

"Donald E. Knuth".parameterize # => "donald-e-knuth"

Bir dizgedeki karakterlerin büyük/küçük harf durumunu :preserve_case argümanıyla koruyun.

"Donald E. Knuth".parameterize(preserve_case: true) # => "Donald-E-Knuth"

‘parameterize’ için çok yaygın bir kullanım durumu, daha açıklayıcı url sümüklü böceklerini desteklemek için bir ActiveRecord modelinin ’to_param’ yöntemini geçersiz kılmaktır.

class Person < ActiveRecord::Base
  def to_param
    "#{id}-#{name.parameterize}"
  end
end

Person.find(1).to_param # => "1-donald-e-knuth"

Dize#tableize #

Modellerin tablo adlarına yönelik Rails’in yaptığı gibi bir tablonun adını oluşturur. Dizedeki son sözcüğü çoğul hale getirir.

'RawScaledScorer'.tableize # => "raw_scaled_scorers"
'ham_and_egg'.tableize     # => "ham_and_eggs"
'fancyCategory'.tableize   # => "fancy_categories"

[Dize#sınıflandır][sınıflandır] #

Rails’in tablo adları için modellere yaptığı gibi çoğul bir tablo adından bir sınıf adı dizesi döndürür.

'ham_and_eggs'.classify # => "HamAndEgg"
'posts'.classify        # => "Post"

Dize#insanlaştır #

İlk sözcüğü büyük harf yapar, alt çizgileri boşluklara dönüştürür ve varsa sonuna bir “_id” çıkarır.

'employee_salary'.humanize              # => "Employee salary"
'author_id'.humanize                    # => "Author"
'author_id'.humanize(capitalize: false) # => "author"
'_id'.humanize                          # => "Id"

String#upcase_first #

Yalnızca ilk karakteri büyük harfe dönüştürür.

'what a Lovely Day'.upcase_first # => "What a Lovely Day"
'w'.upcase_first                 # => "W"
''.upcase_first                  # => ""

Dize#yabancı_anahtar #

Bir sınıf adından bir yabancı anahtar adı oluşturur. Ad ile “id” arasına “_” eklenmesini devre dışı bırakmak için “false” parametresini iletin.

'Message'.foreign_key        # => "message_id"
'Message'.foreign_key(false) # => "messageid"
