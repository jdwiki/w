Rails ile kimlik doğrulama kimlik doğrulaması_with_http_token #

authenticate_with_http_token do |token, options|
  @user = User.find_by(auth_token: token)
end

Bu bitiş noktasını ‘curl’ ile aşağıdaki gibi bir istekte bulunarak test edebilirsiniz.

curl -IH "Authorization: Token token=my-token" http://localhost:3000
