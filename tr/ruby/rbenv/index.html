1. Ruby sürümlerini rbenv ile kurun ve yönetin #

Ruby’nin çeşitli sürümlerini rbenv ile kurmanın ve yönetmenin en kolay yolu, Ruby-build eklentisini kullanmaktır.

Önce rbenv deposunu ana dizininize kopyalayın:

$ git clone https://github.com/rbenv/rbenv.git ~/.rbenv

Ardından Ruby-build eklentisini klonlayın:

$ git clone https://github.com/rbenv/ruby-build.git ~/.rbenv/plugins/ruby-build

Bunu .bash_profile veya .zshrc dosyanıza ekleyerek kabuk oturumunuzda rbenv’nin başlatıldığından emin olun:

type rbenv > /dev/null
if [ "$?" = "0" ]; then
    eval "$(rbenv init -)"
fi

(Bu, esasen önce ‘rbenv’in mevcut olup olmadığını kontrol eder ve onu başlatır).

Muhtemelen kabuk oturumunuzu yeniden başlatmanız veya yeni bir Terminal penceresi açmanız gerekecektir.

Not: OSX üzerinde çalışıyorsanız, aşağıdakilerle birlikte Mac OS Komut Satırı Araçlarını da yüklemeniz gerekir:

$ xcode-select --install

“rbenv"i kaynaktan derlemek yerine Homebrew kullanarak da yükleyebilirsiniz:

$ brew update
$ brew install rbenv

Ardından verilen talimatları izleyin:

$ rbenv init

Ruby’nin yeni bir sürümünü yükleyin:

Kullanılabilir sürümleri listeleyin:

$ rbenv install --list

Bir sürüm seçin ve şununla yükleyin:

$ rbenv install 2.2.0

Yüklü sürümü genel sürüm olarak işaretleyin - yani, sisteminizin varsayılan olarak kullandığı sürüm:

$ rbenv global 2.2.0

Global sürümünüzün neyle olduğunu kontrol edin:

$ rbenv global
=> 2.2.0

Aşağıdakilerle yerel bir proje sürümü belirtebilirsiniz:

$ rbenv local 2.1.2
=> (Creates a .ruby-version file at the current directory with the specified version)

Dipnotlar:

[1]: YOLU Anlamak

Ruby’yi Kaldırma #

Belirli bir Ruby sürümünü kaldırmanın iki yolu vardır. En kolayı, dizini ~/.rbenv/versions dizininden kaldırmaktır:

$ rm -rf ~/.rbenv/versions/2.1.0

Alternatif olarak, tamamen aynı şeyi yapan kaldırma komutunu da kullanabilirsiniz:

$ rbenv uninstall 2.1.0

Bu sürüm bir yerde kullanılıyorsa, global veya yerel sürümünüzü güncellemeniz gerekir. Yolunuzdaki ilk sürüme (genellikle sisteminiz tarafından sağlanan varsayılan) geri dönmek için şunu kullanın:

$ rbenv global system
