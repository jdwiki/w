Sayı hiyerarşisi #

Ruby, sayıları temsil etmek için birkaç yerleşik sınıf içerir:

Numeric
  Integer
    Fixnum    # 1
    Bignum    # 10000000000000000000
  Float       # 1.0
  Complex     # (1+0i)
  Rational    # Rational(2, 3) == 2/3
  BigDecimal  # not loaded by default

En yaygın olanları:

  • Örneğin pozitif ve negatif tam sayıları temsil etmek için “Fixnum”
  • Kayan nokta sayılarını temsil etmek için “Kayan”

“BigDecimal”, varsayılan olarak yüklenmeyen tek şeydir. Şunlarla yükleyebilirsiniz:

require "bigdecimal"

Ruby 2.4+ sürümünde “Fixnum” ve “Bignum"un birleşik olduğuna dikkat edin; all tamsayılar artık sadece ‘Tamsayı’ sınıfının üyeleridir. Geriye dönük uyumluluk için, `Fixnum == Bignum == Tamsayı’.

Bir Dizeyi Tam Sayıya Dönüştürme #

Bir “Dize"yi “Tamsayı"ya dönüştürmek için “Tamsayı” yöntemini kullanabilirsiniz:

Integer("123")      # => 123
Integer("0xFF")     # => 255
Integer("0b100")    # => 4
Integer("0555")     # => 365

Belirli bir tabandan sayıları dönüştürmek için “Tamsayı” yöntemine bir temel parametre de iletebilirsiniz.

Integer('10', 5)    # => 5
Integer('74', 8)    # => 60
Integer('NUM', 36)  # => 30910

Parametre dönüştürülemiyorsa yöntemin bir “ArgumentError” oluşturduğunu unutmayın:

Integer("hello")
# raises ArgumentError: invalid value for Integer(): "hello"
Integer("23-hello")
# raises ArgumentError: invalid value for Integer(): "23-hello"

‘String#to_i’ yöntemini de kullanabilirsiniz. Ancak, bu yöntem biraz daha serbesttir ve “Tamsayı"dan farklı bir davranışa sahiptir:

"23".to_i         # => 23
"23-hello".to_i   # => 23
"hello".to_i      # => 0

“String#to_i”, sayıyı şu şekilde yorumlamak için temel olan bir argümanı kabul eder:

"10".to_i(2) # => 2
"10".to_i(3) # => 3
"A".to_i(16) # => 10

Tam Sayı Oluşturma #

0       # creates the Fixnum 0
123     # creates the Fixnum 123
1_000   # creates the Fixnum 1000. You can use _ as separator for readability

Varsayılan olarak gösterim 10 tabanıdır. Ancak, farklı tabanlar için başka yerleşik gösterimler de vardır:

0xFF    # Hexadecimal representation of 255, starts with a 0x
0b100   # Binary representation of 4, starts with a 0b
0555    # Octal representation of 365, starts with a 0 and digits

Yuvarlama Sayıları #

“Yuvarla” yöntemi, ondalık basamağından sonraki ilk rakam 5 veya daha büyükse bir sayıyı yukarı, bu rakam 4 veya daha düşükse aşağı yuvarlar. Bu, aradığınız hassasiyet için isteğe bağlı bir argüman alır.

4.89.round        # => 5
4.25.round        # => 4
3.141526.round(1) # => 3.1
3.141526.round(2) # => 3.14
3.141526.round(4) # => 3.1415

Kayan noktalı sayılar, “zemin” yöntemiyle sayıdan küçük olan en yüksek tam sayıya da yuvarlanabilir.

4.9999999999999.floor # => 4

Ayrıca, “tavan” yöntemi kullanılarak sayıdan daha büyük en düşük tam sayıya yuvarlanabilirler.

4.0000000000001.ceil  # => 5

Çift ve Tek Sayılar #

Bir sayının çift olup olmadığını belirlemek için ’even?’ yöntemi kullanılabilir.

4.even?      # => true
5.even?      # => false

Bir sayının tek olup olmadığını belirlemek için odd? yöntemi kullanılabilir.

4.odd?       # => false
5.odd?       # => true

Sayıyı dizeye dönüştürme #

Fixnum#to_s, isteğe bağlı bir temel argüman alır ve o tabandaki verilen sayıyı temsil eder:

2.to_s(2)   # => "10"
3.to_s(2)   # => "11"
3.to_s(3)   # => "10"
10.to_s(16) # => "a"

Argüman sağlanmazsa, 10 tabanındaki sayıyı temsil eder.

2.to_s # => "2"
10423.to_s # => "10423"

Rasyonel sayılar #

‘Rational’, pay ve payda olarak rasyonel bir sayıyı temsil eder:

r1 = Rational(2, 3)
r2 = 2.5.to_r
r3 = r1 + r2
r3.numerator   # => 19
r3.denominator # => 6
Rational(2, 4) # => (1/2)

Rational oluşturmanın diğer yolları

Rational('2/3')  # => (2/3)
Rational(3)      # => (3/1)
Rational(3, -5)  # => (-3/5)
Rational(0.2)    # => (3602879701896397/18014398509481984)
Rational('0.2')  # => (1/5)
0.2.to_r         # => (3602879701896397/18014398509481984)
0.2.rationalize  # => (1/5)
'1/4'.to_r       # => (1/4)

Karışık sayılar #

1i     # => (0+1i)
1.to_c # => (1+0i)
rectangular = Complex(2, 3)  # => (2+3i)
polar       = Complex('1@2') # => (-0.4161468365471424+0.9092974268256817i)

polar.rectangular # => [-0.4161468365471424, 0.9092974268256817]
rectangular.polar # => [3.605551275463989, 0.982793723247329]
rectangular + polar # => (1.5838531634528576+3.909297426825682i)

İki sayıyı bölme #

İki sayıyı bölerken karşılığında istediğiniz türe dikkat edin. İki tamsayıyı bölmenin tamsayı bölümünü çağıracağını unutmayın. Amacınız float bölümünü çalıştırmaksa, parametrelerden en az biri ‘float’ türünde olmalıdır.

Tamsayı bölümü:

3 / 2 # => 1

Şamandıra bölümü

3 / 3.0 # => 1.0

16 / 2 / 2    # => 4
16 / 2 / 2.0  # => 4.0
16 / 2.0 / 2  # => 4.0
16.0 / 2 / 2  # => 4.0
