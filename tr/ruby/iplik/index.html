Temel Konu Semantiği #

Ana iş parçacığının yürütülmesinden ayrı yeni bir iş parçacığı, ‘Thread.new’ kullanılarak oluşturulabilir.

thr = Thread.new {
  sleep 1 # 1 second sleep of sub thread
  puts "Whats the big deal"
}

Bu, yeni iş parçacığının yürütülmesini otomatik olarak başlatacaktır.

Ana iş parçacığının yürütülmesini yeni iş parçacığı durana kadar dondurmak için ‘join’ kullanın:

thr.join #=> ... "Whats the big deal"

Join’i çağırdığınızda Thread’in zaten bitmiş olabileceğini unutmayın, bu durumda yürütme normal şekilde devam edecektir. Bir alt iş parçacığı hiçbir zaman birleştirilmezse ve ana iş parçacığı tamamlanırsa, alt iş parçacığı kalan kodu yürütmez.

Paylaşılan kaynaklara erişim #

Birden çok iş parçacığından erişilen bir değişkene erişimi senkronize etmek için bir muteks kullanın:

counter = 0
counter_mutex = Mutex.new

# Start three parallel threads and increment counter
3.times.map do |index|
  Thread.new do
    counter_mutex.synchronize { counter += 1 }
  end
end.each(&:join) # Wait for all threads to finish before killing the process

Aksi takdirde, şu anda bir iş parçacığı tarafından görülebilen ‘sayaç’ değeri başka bir iş parçacığı tarafından değiştirilebilir.

Örnek “Mutex"siz (bkz. ör. “Önce” ve “Sonra"nın “1"den fazla farklı olduğu “İş parçacığı 0”):

2.2.0 :224 > counter = 0; 3.times.map { |i| Thread.new { puts "[Thread #{i}] Before: #{counter}"; counter += 1; puts "[Thread #{i}] After: #{counter}"; } }.each(&:join)
[Thread 2] Before: 0
[Thread 0] Before: 0
[Thread 0] After: 2
[Thread 1] Before: 0
[Thread 1] After: 3
[Thread 2] After: 1

Örnek **“Mutex” ile:

2.2.0 :226 > mutex = Mutex.new; counter = 0; 3.times.map { |i| Thread.new { mutex.synchronize { puts "[Thread #{i}] Before: #{counter}"; counter += 1; puts "[Thread #{i}] After: #{counter}"; } } }.each(&:join)
[Thread 2] Before: 0
[Thread 2] After: 1
[Thread 1] Before: 1
[Thread 1] After: 2
[Thread 0] Before: 2
[Thread 0] After: 3

Bir iş parçacığı nasıl öldürülür #

‘Thread.kill’ veya ‘Thread.terminate’ kullanımını çağırırsınız:

thr = Thread.new { ... }
Thread.kill(thr)

Bir Konuyu Sonlandırma #

Bir iş parçacığı, kod bloğunun sonuna ulaşırsa sona erer. Bir iş parçacığını erken sonlandırmanın en iyi yolu, onu kod bloğunun sonuna gelmeye ikna etmektir. Bu şekilde, iş parçacığı ölmeden önce temizleme kodunu çalıştırabilir.

Örnek değişkeni devam ederken bu iş parçacığı bir döngü çalıştırır. Bu değişkeni false olarak ayarlayın, iş parçacığı doğal bir ölümle ölecek:

require 'thread'

class CounterThread < Thread
  def initialize
    @count = 0
    @continue = true

    super do
      @count += 1 while @continue
      puts "I counted up to #{@count} before I was cruelly stopped."
    end
  end

  def stop
    @continue = false
  end
end

counter = CounterThread.new
sleep 2
counter.stop
