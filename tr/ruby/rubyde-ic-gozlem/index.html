İç gözlem nedir?

İçgözlem, içeriyi bilmek için içe bakmaktır. Bu, iç gözlemin basit bir tanımıdır.

Programlamada ve genel olarak Ruby’de… iç gözlem, nesneye, sınıfa… çalışma zamanında bunu bilmek için bakma yeteneğidir.

Hadi bazı örneklere bakalım #

Örnek:

s = "Hello"  # s is a string

Sonra s hakkında bir şeyler öğreniriz. Hadi başlayalım:

Yani çalışma zamanında s sınıfının ne olduğunu bilmek ister misiniz?

irb(main):055:0* s.class
=> String

Sınıfın iç gözlemi #

Aşağıdaki sınıf tanımını yapalım

class A
   def a; end
end

module B
   def b; end
end

class C < A
   include B
   def c; end
end

‘C’nin örnek yöntemleri nelerdir?

C.instance_methods # [:c, :b, :a, :to_json, :instance_of?...]

Yalnızca ‘C’ üzerinde bildiren örnek yöntemleri nelerdir?

C.instance_methods(false) # [:c]

“C” sınıfının ataları nelerdir?

C.atalar # [C, B, A, Nesne,...]

“C"nin üst sınıfı mı?

“C.superclass # A”
