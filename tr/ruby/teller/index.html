Sözdizimi #

  • ‘Bir dize’ // tek tırnaklı değişmez değer yoluyla bir dize oluşturur
  • “Bir dize” // çift tırnaklı değişmez değer yoluyla bir dize oluşturur
  • String.new(“Bir dize”)
  • %q(A dize) // tek tırnaklı dizeler oluşturmak için alternatif sözdizimi
  • %Q(A dize) // çift tırnaklı dizeler oluşturmak için alternatif sözdizimi

Tek tırnaklı ve çift tırnaklı String değişmezleri arasındaki fark #

Temel fark, çift tırnaklı “Dize” değişmez değerlerinin dize enterpolasyonlarını ve tüm kaçış dizilerini desteklemesidir.

Örneğin, enterpolasyon yoluyla keyfi Ruby ifadeleri içerebilirler:

# Single-quoted strings don't support interpolation
puts 'Now is #{Time.now}'
# Now is #{Time.now}

# Double-quoted strings support interpolation
puts "Now is #{Time.now}"
# Now is 2016-07-21 12:43:04 +0200

Çift tırnaklı dizeler, “\n”, dahil tüm kaçış dizileri kümesini de destekler. “\t”`…

puts 'Hello\nWorld'
# Hello\nWorld

puts "Hello\nWorld"
# Hello
# World

… tek tırnaklı dizeler *kaçış dizilerini hayır desteklerken, tek tırnaklı dizelerin yararlı olması için gereken minimum kümeyi engeller: Sırasıyla değişmez tek tırnaklar ve ters eğik çizgiler, ‘’'’ ve ‘’\’’.

Bir Dize Oluşturma #

Ruby, bir String nesnesi yaratmanın birkaç yolunu sunar. En yaygın yol, bir “dize değişmezi” oluşturmak için tek veya çift tırnak kullanmaktır:

s1 = 'Hello'
s2 = "Hello"

Temel fark, çift tırnaklı dize değişmezlerinin enterpolasyonu ve bazı ters eğik çizgi kaçış dizilerini destekledikleri için biraz daha esnek olmalarıdır.

Rastgele dize sınırlayıcıları kullanarak bir dize değişmezi oluşturmanın başka olası yolları da vardır. Rastgele bir dize sınırlayıcı, bir “%” ve ardından eşleşen bir sınırlayıcı çiftidir:

%(A string)
%{A string}
%<A string>
%|A string|
%!A string!

Son olarak, "" ve """ ile eşdeğer olan “%q” ve “%Q” dizisini kullanabilirsiniz:

puts %q(A string)
# A string
puts %q(Now is #{Time.now})
# Now is #{Time.now}

puts %Q(A string)
# A string
puts %Q(Now is #{Time.now})
# Now is 2016-07-21 12:47:45 +0200

“%q” ve “%Q” dizileri, dize tek tırnak, çift tırnak veya her ikisinin karışımını içerdiğinde kullanışlıdır. Bu şekilde, içerikten kaçmanıza gerek kalmaz:

%Q(<a href="/profile">User's profile<a>)

Eşleşen bir çift olduğu sürece birkaç farklı sınırlayıcı kullanabilirsiniz:

%q(A string)
%q{A string}
%q<A string>
%q|A string|
%q!A string!

Vaka manipülasyonu #

"string".upcase     # => "STRING"
"STRING".downcase   # => "string"
"String".swapcase   # => "sTRING"
"string".capitalize # => "String"

Bu dört yöntem orijinal alıcıyı değiştirmez. Örneğin,

str = "Hello"
str.upcase # => "HELLO"
puts str   # => "Hello"

Aynı eylemleri gerçekleştiren ancak orijinal alıcıyı değiştiren dört benzer yöntem vardır.

"string".upcase!     # => "STRING"
"STRING".downcase!   # => "string"
"String".swapcase!   # => "sTRING"
"string".capitalize! # => "String"

Örneğin,

str = "Hello"
str.upcase!  # => "HELLO"
puts str     # => "HELLO"

Notlar:

  • Ruby 2.4’ten önce bu yöntemler unicode’u işlemez.

Dize birleştirme #

Dizeleri “+” operatörüyle birleştirin:

s1 = "Hello"
s2 = " "
s3 = "World"

puts s1 + s2 + s3
# => Hello World

s = s1 + s2 + s3
puts s
# => Hello World

Veya << operatörüyle:

s = 'Hello'
s << ' '
s << 'World'
puts s
# => Hello World

<< operatörünün sol taraftaki nesneyi değiştirdiğini unutmayın.

Ayrıca dizeleri çoğaltabilirsiniz, örn.

"wow" * 3
# => "wowwowwow"

Bir Dizeyi Bölme #

“Dize#split”, bir “Dize"yi bir sınırlayıcıya dayalı olarak bir “Dizi"ye böler.

"alpha,beta".split(",")
# => ["alpha", "beta"]

Boş bir “Dize”, boş bir “Dizi” ile sonuçlanır:

"".split(",")
# => []

Eşleşmeyen bir sınırlayıcı, tek bir öğe içeren bir “Dizi” ile sonuçlanır:

"alpha,beta".split(".")
# => ["alpha,beta"]

Bir dizgiyi normal ifadeler kullanarak da bölebilirsiniz:

"alpha, beta,gamma".split(/, ?/)
# => ["alpha", "beta", "gamma"]

Sınırlayıcı isteğe bağlıdır, varsayılan olarak bir dize boşlukta bölünür:

"alpha beta".split
# => ["alpha", "beta"]

Konumlandırma dizeleri #

Ruby’de dizeler sola dayalı, sağa dayalı veya ortalanmış olabilir

Dizeyi sola yaslamak için ’ljust’ yöntemini kullanın. Bu, yeni dizenin karakter sayısını temsil eden bir tamsayı ve doldurulacak kalıbı temsil eden bir dize olmak üzere iki parametre alır.

Tamsayı, orijinal dizenin uzunluğundan büyükse, yeni dize, kalan boşluğu alan isteğe bağlı dize parametresiyle sola dayalı olacaktır. string parametresi verilmezse, string boşluklarla doldurulur.

str ="abcd"
str.ljust(4)          => "abcd"
str.ljust(10)         => "abcd      "

Bir dizgiyi sağa yaslamak için “rjust” yöntemini kullanın. Bu, yeni dizenin karakter sayısını temsil eden bir tamsayı ve doldurulacak kalıbı temsil eden bir dize olmak üzere iki parametre alır.

Tamsayı orijinal dizenin uzunluğundan büyükse, yeni dize, kalan alanı isteğe bağlı dize parametresi alarak sağa yaslanır. string parametresi verilmezse, string boşluklarla doldurulur.

str = "abcd"
str.rjust(4)          => "abcd"
str.rjust(10)         => "      abcd"

Bir dizgiyi ortalamak için “center” yöntemini kullanın. Bu, yeni dizenin genişliğini temsil eden bir tamsayı ve orijinal dizenin doldurulacağı bir dize olmak üzere iki parametre alır. Dize merkeze hizalanacaktır.

str = "abcd"
str.center(4)          => "abcd"
str.center(10)         => "   abcd   "

Dizeleri Birleştirme #

“Array#join”, bir sınırlayıcıya dayalı olarak bir “Dizi"yi bir “Dize” ile birleştirir:

["alpha", "beta"].join(",")
# => "alpha,beta"

Sınırlayıcı isteğe bağlıdır ve varsayılan olarak boş bir “Dize"dir.

["alpha", "beta"].join
# => "alphabeta"

Boş bir “Dizi”, hangi sınırlayıcı kullanılırsa kullanılsın boş bir “Dize” ile sonuçlanır.

[].join(",")
# => ""

Dize şununla başlar #

Bir dizgenin bir kalıpla başlayıp başlamadığını bulmak için ‘start_with?’ yöntemi işe yarar

str = "zebras are cool"
str.start_with?("zebras")      => true

Ayrıca ‘index’ ile kalıbın konumunu kontrol edebilirsiniz.

str = "zebras are cool"
str.index("zebras").zero?      => true

Dize enterpolasyonu #

Çift tırnaklı sınırlayıcı "”” ve “%Q” dizisi, “#{ruby_expression}” kullanılarak dize enterpolasyonunu destekler:

puts "Now is #{Time.now}"
# Now is Now is 2016-07-21 12:47:45 +0200

puts %Q(Now is #{Time.now})
# Now is Now is 2016-07-21 12:47:45 +0200

Dize şununla biter #

Bir dizgenin bir kalıpla bitip bitmediğini bulmak için ’end_with?’ yöntemi işe yarar

str = "I like pineapples"
str.end_with?("pineaaples")      => false

Çok satırlı dizeler #

Çok satırlı bir dize oluşturmanın en kolay yolu, tırnak işaretleri arasında yalnızca birden çok satır kullanmaktır:

address = "Four score and seven years ago our fathers brought forth on this
continent, a new nation, conceived in Liberty, and dedicated to the
proposition that all men are created equal."

Bu teknikle ilgili temel sorun, dize bir alıntı içeriyorsa, dize sözdizimini bozacak olmasıdır. Sorunu çözmek için bunun yerine bir heredoc kullanabilirsiniz:

puts <<-RAVEN
  Once upon a midnight dreary, while I pondered, weak and weary, 
  Over many a quaint and curious volume of forgotten lore— 
      While I nodded, nearly napping, suddenly there came a tapping, 
  As of some one gently rapping, rapping at my chamber door. 
  "'Tis some visitor," I muttered, "tapping at my chamber door— 
              Only this and nothing more." 
  RAVEN

Ruby, burada <<EOT içeren kabuk stili belgeleri destekler, ancak sonlandırma metni satırı başlatmalıdır. Bu, kod girintisini bozar, bu yüzden bu stili kullanmak için çok fazla neden yoktur. Ne yazık ki, kodun kendisinin nasıl girintilendiğine bağlı olarak dizede girintiler olacaktır.

Ruby 2.3, fazla önde gelen boşlukları ortadan kaldıran <<~ ekleyerek sorunu çözer:

<!-- if version [gte 2.3] -->
def build_email(address)
  return (<<~EMAIL)
    TO: #{address}

    To Whom It May Concern:

    Please stop playing the bagpipes at sunrise!
                     
    Regards,
    Your neighbor               
  EMAIL
end
<!-- end version if -->

Yüzde Dizeler ayrıca çok satırlı dizeler oluşturmak için de çalışır:

%q(
HAMLET        Do you see yonder cloud that's almost in shape of a camel?
POLONIUS        By the mass, and 'tis like a camel, indeed.
HAMLET        Methinks it is like a weasel.
POLONIUS        It is backed like a weasel.
HAMLET        Or like a whale?
POLONIUS        Very like a whale
)

Enterpolasyon ve kaçış dizilerinden kaçınmanın birkaç yolu vardır:

  • Çift tırnak yerine tek tırnak: `’\n bir satır başıdır.’’

  • Yüzde dizesinde küçük “q” harfi: “%q[#{değişken değil}]”

  • Bir heredoc’ta terminal dizesini tek alıntı yapın:

    <<-'CODE'
   puts 'Hello world!'
CODE

Biçimlendirilmiş dizeler #

Ruby, herhangi bir yer tutucuyu sağlanan dizideki değerlerle değiştirerek bir dizi değeri bir dizeye enjekte edebilir.

"Hello %s, my name is %s!" % ['World', 'br3nt']
# => Hello World, my name is br3nt!

Yer tutucular iki %s ile temsil edilir ve değerler ['Merhaba', 'br3nt'] dizisi tarafından sağlanır. % operatörü, dizeye dizinin değerlerini enjekte etmesi talimatını verir.

Dize karakter değiştirmeleri #

tr yöntemi, birinci argümanın karakterlerinin ikinci argümanın karakterleriyle değiştirildiği bir dizgenin bir kopyasını döndürür.

"string".tr('r', 'l') # => "stling"

Bir kalıbın yalnızca ilk oluşumunu başka bir ifadeyle değiştirmek için “sub” yöntemini kullanın.

"string ring".sub('r', 'l') # => "stling ring"

Bir kalıbın tüm tekrarlarını bu ifadeyle değiştirmek istiyorsanız, “gsub” kullanın.

"string ring".gsub('r','l') # => "stling ling"

Karakterleri silmek için ikinci parametre için boş bir dize girin

Tüm bu yöntemlerde düzenli ifadeler de kullanabilirsiniz.

Bu yöntemlerin yalnızca bir dizenin yeni bir kopyasını döndüreceğini ve dizeyi yerinde değiştirmeyeceğini unutmamak önemlidir. Bunun için sırasıyla tr!, sub! ve gsub! yöntemlerini kullanmanız gerekir.

Bir dizedeki verileri anlama #

Ruby’de bir dize, nasıl yorumlayabileceğinizi belirten bir kodlamanın adı (“UTF-8”, “US-ASCII”, “ASCII-8BIT” gibi) ile birlikte yalnızca bir bayt dizisidir. bu baytlar karakter olarak.

Ruby dizeleri metni (temelde bir karakter dizisi) tutmak için kullanılabilir, bu durumda genellikle UTF-8 kodlaması kullanılır.

"abc".bytes  # => [97, 98, 99]
"abc".encoding.name  # => "UTF-8"

Ruby dizeleri, ikili verileri (bir bayt dizisi) tutmak için de kullanılabilir; bu durumda, genellikle ASCII-8BIT kodlaması kullanılır.

[42].pack("i").encoding  # => "ASCII-8BIT"

Bir dizedeki bayt dizisinin kodlamayla eşleşmemesi, dizeyi kullanmaya çalışırsanız hatalarla sonuçlanması mümkündür.

"\xFF \xFF".valid_encoding? # => false
"\xFF \xFF".split(' ')      # ArgumentError: invalid byte sequence in UTF-8

Dize Değiştirme #

p "This is %s" % "foo"
# => "This is foo"

p "%s %s %s" % ["foo", "bar", "baz"]
# => "foo bar baz"

p "%{foo} == %{foo}" % {:foo => "foo" }
# => "foo == foo"

Daha fazla ayrıntı için String % docs ve Kernel::sprintf bölümüne bakın.
