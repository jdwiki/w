Anahtar Kelime Bağımsız Değişkenleri

  • Uyarı operatörüyle rastgele anahtar kelime bağımsız değişkenleri kullanma
  • Anahtar kelime argümanlarını kullanma
  • Gerekli anahtar kelime bağımsız değişkenleri

Bu sayfada

  • Uyarı operatörüyle rastgele anahtar kelime bağımsız değişkenleri kullanma
  • Anahtar kelime argümanlarını kullanma
  • Gerekli anahtar kelime bağımsız değişkenleri

Anahtar kelime argümanları Ruby 2.0’da tanıtıldı ve Ruby 2.1’de required anahtar kelime argümanlarının eklenmesiyle geliştirildi.

Anahtar kelime argümanına sahip basit bir yöntem aşağıdakine benzer:

def say(message: "Hello World")
  puts message
end

say
# => "Hello World"

say message: "Today is Monday"
# => "Today is Monday"

Bir hatırlatma olarak, anahtar kelime argümanı olmadan aynı yöntem şöyle olurdu:

def say(message = "Hello World")
  puts message
end

say
# => "Hello World"

say "Today is Monday"
# => "Today is Monday"

Bir Hash parametresi kullanarak önceki Ruby sürümlerinde anahtar kelime argümanını simüle edebilirsiniz. Bu, özellikle 2.0 öncesi Ruby sürümleriyle uyumluluk sağlayan kitaplıklarda hala çok yaygın bir uygulamadır:

def say(options = {})
  message = options.fetch(:message, "Hello World")
  puts 
end

say
# => "Hello World"

say message: "Today is Monday"
# => "Today is Monday"

Uyarı operatörüyle rastgele anahtar kelime bağımsız değişkenleri kullanma #

double splat (**) operatörünü kullanarak rastgele sayıda anahtar kelime bağımsız değişkenini kabul etmek için bir yöntem tanımlayabilirsiniz:

def say(**args)
  puts args
end

say foo: "1", bar: "2"
# {:foo=>"1", :bar=>"2"}

Argümanlar bir ‘Hash’ içinde yakalanır. Örneğin, istenen argümanları çıkarmak için ‘Hash’i değiştirebilirsiniz.

def say(**args)
  puts args[:message] || "Message not found"
end

say foo: "1", bar: "2", message: "Hello World"
# Hello World

say foo: "1", bar: "2"
# Message not found

Anahtar kelime argümanlarıyla bir uyarı operatörü kullanmak, anahtar kelime argümanı doğrulamasını önleyecektir, bilinmeyen anahtar kelime durumunda yöntem asla bir ‘ArgumentError’ oluşturmaz.

Standart uyarı operatörüne gelince, bir yöntem için bir “Hash"i anahtar kelime argümanlarına yeniden dönüştürebilirsiniz:

def say(message: nil, before: "<p>", after: "</p>")
  puts "#{before}#{message}#{after}"
end

args = { message: "Hello World", after: "</p><hr>" }
say(**args)
# <p>Hello World</p><hr>

args = { message: "Hello World", foo: "1" }
say(**args)
# => ArgumentError: unknown keyword: foo

Bu genellikle, gelen bağımsız değişkenleri değiştirmeniz ve bunları temel alınan bir yönteme aktarmanız gerektiğinde kullanılır:

def inner(foo:, bar:)
  puts foo, bar
end

def outer(something, foo: nil, bar: nil, baz: nil)
  puts something
  params = {}
  params[:foo] = foo || "Default foo"
  params[:bar] = bar || "Default bar"
  inner(**params)
end

outer "Hello:", foo: "Custom foo"
# Hello:
# Custom foo
# Default bar

Anahtar kelime argümanlarını kullanma #

Yöntem tanımında adı belirterek bir yöntemde bir anahtar kelime bağımsız değişkeni tanımlarsınız:

def say(message: "Hello World")
  puts message
end

say
# => "Hello World"

say message: "Today is Monday"
# => "Today is Monday"

Birden çok anahtar kelime bağımsız değişkeni tanımlayabilirsiniz, tanım sırası önemsizdir:

def say(message: "Hello World", before: "<p>", after: "</p>")
  puts "#{before}#{message}#{after}"
end

say
# => "<p>Hello World</p>"

say message: "Today is Monday"
# => "<p>Today is Monday</p>"

say after: "</p><hr>", message: "Today is Monday"
# => "<p>Today is Monday</p><hr>"

Anahtar kelime bağımsız değişkenleri, konumsal bağımsız değişkenlerle karıştırılabilir:

def say(message, before: "<p>", after: "</p>")
  puts "#{before}#{message}#{after}"
end

say "Hello World", before: "<span>", after: "</span>"
# => "<span>Hello World</span>"

Anahtar kelime argümanını konumsal argümanla karıştırmak Ruby 2.1’den önce çok yaygın bir yaklaşımdı, çünkü [gerekli anahtar kelime argümanlarını] tanımlamak mümkün değildi(https://www.wikiod.com/tr/ruby/anahtar-kelime-bagmsz-degiskenleri#Gerekli anahtar kelime bağımsız değişkenleri anahtar kelime-argümanlar#t=201608060918529843522).

Ayrıca, Ruby < 2.0’da, isteğe bağlı argümanlar için bir yöntem tanımının sonuna bir ‘Hash’ eklemek çok yaygındı. Sözdizimi, anahtar kelime argümanlarına çok benzer, o kadar ki ‘Hash’ yoluyla isteğe bağlı argümanlar Ruby 2 anahtar kelime argümanlarıyla uyumludur.

def say(message, options = {})
  before = option.fetch(:before, "<p>")
  after  = option.fetch(:after, "</p>")
  puts "#{before}#{message}#{after}"
end

# The method call is syntactically equivalent to the keyword argument one
say "Hello World", before: "<span>", after: "</span>"
# => "<span>Hello World</span>"

Tanımlanmamış bir anahtar kelime bağımsız değişkeni iletmeye çalışmanın bir hatayla sonuçlanacağını unutmayın:

def say(message: "Hello World")
  puts message
end

say foo: "Hello"
# => ArgumentError: unknown keyword: foo

Gerekli anahtar kelime bağımsız değişkenleri #

Gerekli anahtar sözcük bağımsız değişkenleri, anahtar sözcük bağımsız değişkenlerine yönelik bir iyileştirme olarak Ruby 2.1’de sunulmuştur.

Bir anahtar kelime argümanını gerektiği gibi tanımlamak için, argümanı varsayılan bir değer olmadan bildirmeniz yeterlidir.

def say(message:)
  puts message
end

say
# => ArgumentError: missing keyword: message

say message: "Hello World"
# => "Hello World"

Ayrıca gerekli ve gerekli olmayan anahtar kelime bağımsız değişkenlerini karıştırabilirsiniz:

def say(before: "<p>", message:, after: "</p>")
  puts "#{before}#{message}#{after}"
end

say
# => ArgumentError: missing keyword: message

say message: "Hello World"
# => "<p>Hello World</p>"

say message: "Hello World", before: "<span>", after: "</span>"
# => "<span>Hello World</span>"
