Her biri #

Ruby’nin birçok numaralandırıcı türü vardır, ancak başlamak için ilk ve en basit numaralandırıcı türü “her"dir. “Her” öğesinin nasıl çalıştığını göstermek için “1” ile “10” arasındaki her sayı için “çift” veya “tek” yazdıracağız.

Temel olarak, “blok” denen şeyi geçmenin iki yolu vardır. Bir “blok”, çağrılan yöntem tarafından yürütülecek olan geçirilen bir kod parçasıdır. “each” yöntemi, çağrıldığı nesne koleksiyonunun her öğesi için çağırdığı bir “blok” alır.

Bir bloğu bir metoda geçirmenin iki yolu vardır:

Yöntem 1: Satır içi #

(1..10).each { |i| puts i.even? ? 'even' : 'odd' }

Bu, bunu çözmenin çok sıkıştırılmış ve ruby bir yoludur. Bunu parça parça inceleyelim.

  1. “(1..10)”, “1” ile “10” arasında bir aralıktır. ‘1’den ‘10’a özel olmasını isteseydik, ‘(1…10)’ yazardık.
  2. “.each”, üzerinde işlem yaptığı nesnedeki “her” öğenin üzerinde numaralandıran bir numaralandırıcıdır. Bu durumda, aralıktaki “her” sayıya göre hareket eder.
  3. `{ |i| i.even koyar mı? ? ’even’ : ‘odd’ }’, kendisi daha fazla parçalanabilecek olan ’each’ ifadesinin bloğudur.
    1. |i| this means that each element in the range is represented within the block by the identifier i.
    2. puts is an output method in Ruby that has an automatic line break after each time it prints. (We can use print if we don’t want the automatic line break)
    3. i.even? checks if i is even. We could have also used i % 2 == 0; however, it is preferable to use built in methods.
    4. ? "even" : "odd" this is ruby’s ternary operator. The way a ternary operator is constructed is expression ? a : b. This is short for
    if expression
  a
else
  b
end

Bir satırdan uzun kodlar için “blok”, “çok satırlı blok” olarak geçirilmelidir.

Yöntem 2: Çok satırlı #

(1..10).each do |i|
  if i.even?
    puts 'even'
  else
    puts 'odd'
  end
end

Bir ‘çok satırlı blokta’, ‘do’, açılış parantezinin yerini alır ve ’end’, ‘satır içi’ stildeki kapanış dirseğinin yerini alır.

Ruby, reverse_each’i de destekler. Diziyi geriye doğru yineleyecektir.

@arr = [1,2,3,4]
puts @arr.inspect # output is [1,2,3,4]

print "Reversed array elements["
@arr.reverse_each do |val|
        print " #{val} " # output is 4 3 2 1
end
print "]\n"

Bir sınıfta uygulama #

Karmaşık nesneler üzerinde yineleme #

Diziler

İç içe diziler üzerinde yineleme yapabilirsiniz:

[[1, 2], [3, 4]].each { |(a, b)| p "a: #{ a }", "b: #{ b }" }

Aşağıdaki sözdizimine de izin verilir:

[[1, 2], [3, 4]].each { |a, b| "a: #{ a }", "b: #{ b }" }

Üretecek:

"a: 1"
"b: 2"
"a: 3"
"b: 4"

karmalar

Anahtar/değer çiftlerini yineleyebilirsiniz:

{a: 1, b: 2, c: 3}.each { |pair| p "pair: #{ pair }" }

Üretecek:

"pair: [:a, 1]"
"pair: [:b, 2]"
"pair: [:c, 3]"

Anahtarları ve değerleri aynı anda yineleyebilirsiniz:

{a: 1, b: 2, c: 3}.each { |(k, v)| p "k: #{ k }", "v: #{ k }" }

Üretecek:

"k: a"
"v: a"
"k: b"
"v: b"
"k: c"
"v: c"

Yineleyici için #

Bu, 4’ten 13’e (dahil) yinelenir.

for i in 4..13
    puts "this is #{i}.th number"
end

Ayrıca for kullanarak dizileri yineleyebiliriz.

names = ['Siva', 'Charan', 'Naresh', 'Manish']

for name in names
    puts name
end

Dizin ile yineleme #

Bazen bir numaralandırıcı üzerinde yineleme yaparken mevcut öğenin konumunu (index) bilmek istersiniz. Bu amaçla Ruby, “with_index” yöntemini sağlar. Tüm numaralandırıcılara uygulanabilir. Temel olarak, bir numaralandırmaya “with_index” ekleyerek, bu numaralandırmayı numaralandırabilirsiniz. Dizin, ikinci argüman olarak bir bloğa iletilir.

[2,3,4].map.with_index { |e, i| puts "Element of array number #{i} => #{e}" }
#Element of array number 0 => 2
#Element of array number 1 => 3
#Element of array number 2 => 4
#=> [nil, nil, nil]

“with_index” isteğe bağlı bir bağımsız değişkene sahiptir - varsayılan olarak “0” olan ilk dizin:

[2,3,4].map.with_index(1) { |e, i| puts "Element of array number #{i} => #{e}" }
#Element of array number 1 => 2
#Element of array number 2 => 3
#Element of array number 3 => 4
#=> [nil, nil, nil]

’each_with_index’ için özel bir yöntem vardır. Onunla “each.with_index” arasındaki tek fark, buna bir argüman iletemeyeceğinizdir, bu nedenle ilk indeks her zaman “0"dır.

[2,3,4].each_with_index { |e, i| puts "Element of array number #{i} => #{e}" }
#Element of array number 0 => 2
#Element of array number 1 => 3
#Element of array number 2 => 4
#=> [2, 3, 4]

Harita #

Değiştirilen nesneyi döndürür, ancak orijinal nesne olduğu gibi kalır. Örneğin:

arr = [1, 2, 3]
arr.map { |i| i + 1 } # => [2, 3, 4]
arr # => [1, 2, 3]

harita! orijinal nesneyi değiştirir:

arr = [1, 2, 3]
arr.map! { |i| i + 1 } # => [2, 3, 4]
arr # => [2, 3, 4]

Not: Aynı şeyi yapmak için “toplama"yı da kullanabilirsiniz.
