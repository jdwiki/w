Scala Koleksiyonlarını Java Koleksiyonlarına Dönüştürme ve tersi #

Bir koleksiyonu bir Java yöntemine aktarmanız gerektiğinde:

import scala.collection.JavaConverters._

val scalaList = List(1, 2, 3)
JavaLibrary.process(scalaList.asJava)

Java kodu bir Java koleksiyonu döndürürse, onu benzer şekilde bir Scala koleksiyonuna dönüştürebilirsiniz:

import scala.collection.JavaConverters._

val javaCollection = JavaLibrary.getList
val scalaCollection = javaCollection.asScala

Bunların dekoratör olduğuna dikkat edin, bu nedenle yalnızca temeldeki koleksiyonları bir Scala veya Java koleksiyon arabiriminde sararlar. Bu nedenle, “.asJava” ve “.asScala” çağrıları koleksiyonları kopyalamaz.

Diziler #

Diziler, değişmez olarak kabul edildikleri ve özel kuruculara ve örtük dönüşümlere sahip oldukları bir bükülme ile düzenli JVM dizileridir. Bunları ’new’ anahtar kelimesi olmadan oluşturun.

 val a = Array("element")

Şimdi “a”, “Array[String]” türüne sahip.

 val acs: Array[CharSequence] = a
 //Error: type mismatch;  found   : Array[String]  required: Array[CharSequence]

“Dize”, “CharSequence"e dönüştürülebilir olsa da, “Array[String]”, “Array[CharSequence]” öğesine dönüştürülemez.

“TraversableLike” “ArrayOps"a örtük bir dönüştürme sayesinde, diğer koleksiyonlar gibi bir “Array” kullanabilirsiniz:

 val b: Array[Int] = a.map(_.length)

Scala koleksiyonlarının çoğu (“TraversableOnce”), sonuç dizisini oluşturmak için örtük bir “ClassTag” alan bir “toArray” yöntemine sahiptir:

 List(0).toArray
 //> res1: Array[Int] = Array(0)

Bu, Scala kodunuzda herhangi bir ‘TraversableOnce’ kullanmayı ve ardından onu bir dizi bekleyen Java koduna geçirmeyi kolaylaştırır.

Scala ve Java tipi dönüşümler #

Scala, JavaConverters nesnesindeki tüm ana koleksiyon türleri arasında örtük dönüştürmeler sunar.

Aşağıdaki tür dönüşümleri çift yönlüdür.

Ölçek Türü Java Türü
yineleyici java.util.Iterator
yineleyici java.util.Sayılandırma
yineleyici java.util.Yinelenebilir
yineleyici java.util.Koleksiyon
değişken.Tampon java.util.List
değişken.Set java.util.Set
değişken.Harita java.util.Harita
değişebilir.ConcurrentMap java.util.concurrent.ConcurrentMap

Diğer bazı Scala koleksiyonları da Java’ya dönüştürülebilir, ancak orijinal Scala türüne geri dönüşleri yoktur:

Ölçek Türü Java Türü
Sıra java.util.List
değişken.Seq java.util.List
Ayarla java.util.Set
Harita java.util.Harita

Scala işlevleri için İşlevsel Arayüzler - scala-java8-compat #

Scala için bir Java 8 uyumluluk kiti.

Çoğu örnek Readme adresinden kopyalanmıştır.

scala.FunctionN ve java.util.function arasındaki dönüştürücüler

import java.util.function._
import scala.compat.java8.FunctionConverters._

val foo: Int => Boolean = i => i > 7
def testBig(ip: IntPredicate) = ip.test(9)
println(testBig(foo.asJava))  // Prints true

val bar = new UnaryOperator[String]{ def apply(s: String) = s.reverse }
List("cod", "herring").map(bar.asScala)    // List("doc", "gnirrih")

def testA[A](p: Predicate[A])(a: A) = p.test(a)
println(testA(asJavaPredicate(foo))(4))  // Prints false

Scala.Option ve java.util sınıfları arasında, İsteğe Bağlı, İsteğe Bağlı Double, İsteğe BağlıInt ve İsteğe BağlıLong sınıfları arasındaki dönüştürücüler.

import scala.compat.java8.OptionConverters._

    class Test {
      val o = Option(2.7)
      val oj = o.asJava        // Optional[Double]
      val ojd = o.asPrimitive  // OptionalDouble
      val ojds = ojd.asScala   // Option(2.7) again
    }

Scala koleksiyonlarından Java 8 Akışlarına Dönüştürücüler

import java.util.stream.IntStream

import scala.compat.java8.StreamConverters._
import scala.compat.java8.collectionImpl.{Accumulator, LongAccumulator}


  val m = collection.immutable.HashMap("fish" -> 2, "bird" -> 4)
  val parStream: IntStream = m.parValueStream
  val s: Int = parStream.sum
  // 6, potientially computed in parallel
  val t: List[String] = m.seqKeyStream.toScala[List]
  // List("fish", "bird")
  val a: Accumulator[(String, Int)] = m.accumulate // Accumulator[(String, Int)]

  val n = a.stepper.fold(0)(_ + _._1.length) +
    a.parStream.count // 8 + 2 = 10

  val b: LongAccumulator = java.util.Arrays.stream(Array(2L, 3L, 4L)).accumulate
  // LongAccumulator
  val l: List[Long] = b.to[List] // List(2L, 3L, 4L)
