Bu özellik Scala 2.10.0 ve üzeri sürümlerde mevcuttur.

İfadeyi dize değişmezlerinde kullanma #

İfadeleri dize değişmez değerlerine enterpolasyon yapmak için küme parantezlerini kullanabilirsiniz:

def f(x: String) = x + x
val a = "A"

s"${a}"    // "A"
s"${f(a)}" // "AA"

Parantezler olmadan, scala yalnızca tanımlayıcıyı “$” (bu durumda “f”)‘den sonra enterpolasyon yapar. “f"den “Dize"ye örtük bir dönüşüm olmadığından, bu, bu örnekte bir istisnadır:

s"$f(a)"  // compile-time error (missing argument list for method f)

Merhaba Dize İnterpolasyonu #

s enterpolatörü, bir dizge içinde değişkenlerin kullanımına izin verir.

val name = "Brian"
println(s"Hello $name")

çalıştırıldığında konsola “Merhaba Brian” yazdırır.

f İnterpolatörünü Kullanarak Biçimlendirilmiş Dize İnterpolasyonu #

val num = 42d

f kullanarak “num” için iki ondalık basamak yazdırın

println(f"$num%2.2f")
42.00

e kullanarak bilimsel gösterimi kullanarak “num"u yazdırın

println(f"$num%e")
4.200000e+01

a ile onaltılık olarak “num"u yazdırın

println(f"$num%a")
0x1.5p5

Diğer biçim dizeleri https://docs.Oracle.com/javase/6/docs/api/java/util/Formatter.html#detail adresinde bulunabilir.

Çıkarıcı olarak dize enterpolatörleri #

Ayrıca, belki de en ünlüsü Scala makrolarının quasiquotes API‘sinde kullanıldığı gibi, ayrıntılı çıkarıcılar (kalıp eşleştiriciler) oluşturmak için Scala’nın dize enterpolasyon özelliğini kullanmak da mümkündür.

“n"p0${i0}p1” öğesinin "new StringContext("p0", "p1").n(i0) işlevinin şekerini düşürdüğü göz önüne alındığında, çıkarıcı işlevinin ``dan örtük bir dönüştürme sağlayarak etkinleştirilmesi belki de şaşırtıcı değildir. StringContext’, bir “unapply” veya “unapplySeq” yöntemini tanımlayan türde “n” özelliğine sahip bir sınıfa.

Örnek olarak, ‘StringContext’ bölümlerinden düzenli bir ifade oluşturarak yol parçalarını çıkaran aşağıdaki çıkarıcıyı düşünün. Ardından, ağır kaldırmanın çoğunu, elde edilen scala.util.matching.Regex tarafından sağlanan “unapplySeq” yöntemine devredebiliriz:

implicit class PathExtractor(sc: StringContext) {
  object path {
    def unapplySeq(str: String): Option[Seq[String]] =
      sc.parts.map(Regex.quote).mkString("^", "([^/]+)", "$").r.unapplySeq(str)      
  }
}

"/documentation/scala/1629/string-interpolation" match {
  case path"/documentation/${topic}/${id}/${_}" => println(s"$topic, $id")
  case _ => ???
}

“path” nesnesinin, normal bir enterpolatör gibi davranmak için bir “apply” yöntemini de tanımlayabileceğine dikkat edin.

Özel dize enterpolatörleri #

Yerleşik olanlara ek olarak özel dize enterpolatörleri tanımlamak mümkündür.

my"foo${bar}baz"

Derleyici tarafından şu şekilde genişletilir:

new scala.StringContext("foo", "baz").my(bar)

“scala.StringContext”, “my” yöntemine sahip değildir, bu nedenle örtük dönüştürme ile sağlanabilir. Yerleşik s enterpolatörüyle aynı davranışa sahip özel bir enterpolatör daha sonra aşağıdaki gibi uygulanacaktır:

implicit class MyInterpolator(sc: StringContext) {
  def my(subs: Any*): String = {
    val pit = sc.parts.iterator
    val sit = subs.iterator
    // Note parts.length == subs.length + 1
    val sb = new java.lang.StringBuilder(pit.next())
    while(sit.hasNext) {
      sb.append(sit.next().toString)
      sb.append(pit.next())          
    }
    sb.toString
  }
}

Ve my"foo${bar}baz" enterpolasyonu aşağıdakileri azaltır:

new MyInterpolation(new StringContext("foo", "baz")).my(bar)

Enterpolasyon işlevinin bağımsız değişkenleri veya dönüş türü üzerinde herhangi bir kısıtlama olmadığını unutmayın. Bu bizi, aşağıdaki örnekte gösterildiği gibi, enterpolasyon sözdiziminin rastgele nesneler oluşturmak için yaratıcı bir şekilde kullanılabileceği karanlık bir yola götürür:

case class Let(name: Char, value: Int)

implicit class LetInterpolator(sc: StringContext) {
  def let(value: Int): Let = Let(sc.parts(0).charAt(0), value)
}

let"a=${4}" // Let(a, 4)
let"b=${"foo"}" // error: type mismatch
let"c=" // error: not enough arguments for method let: (value: Int)Let

Ham Dize İnterpolasyonu #

Bir Dize’nin olduğu gibi ve değişmez değerlerden kaçmadan yazdırılmasını istiyorsanız, ham enterpolatörünü kullanabilirsiniz.


println(raw"Merhaba Dünya İngilizce ve Fransızca\nİngilizce:\tMerhaba Dünya\nFransızca:\t\tBonjour Le Monde")

Ham enterpolatörünün kullanımıyla, konsolda aşağıdakilerin yazdırıldığını görmelisiniz:


İngilizce ve Fransızca Merhaba Dünya\nİngilizce:\tHello World\nFransızca:\t\tBonjour Le Monde

Ham enterpolatörü olmasaydı, \n ve \t kaçardı.


println("Merhaba Dünya İngilizce ve Fransızca\nİngilizce:\tMerhaba Dünya\nFransızca:\t\tBonjour Le Monde")

Baskılar:


Merhaba Dünya İngilizce ve Fransızca
İngilizce: Merhaba Dünya
İngilizce: Merhaba Dünya
