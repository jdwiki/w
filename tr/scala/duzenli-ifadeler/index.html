Normal ifadeleri bildirme #

scala.collection.immutable.StringOps aracılığıyla dolaylı olarak sağlanan “r” yöntemi, konu dizesinden bir scala.util.matching.Regex örneği üretir. Java’da olduğu gibi ters eğik çizgilerden kaçmak zorunda olmadığınız için Scala’nın üç tırnaklı dize sözdizimi burada yararlıdır:

val r0: Regex = """(\d{4})-(\d${2})-(\d{2})""".r     // :)
val r1: Regex = "(\\d{4})-(\\d{2})-(\\d{2})".r // :(

scala.util.matching.Regex, java.util.regex.Pattern üzerinde bir sarmalayıcı olarak Scala için deyimsel bir normal ifade API’si uygular ve desteklenen sözdizimi aynıdır. Bununla birlikte, Scala’nın çok satırlı dize değişmezleri desteği, ‘x’ işaretini önemli ölçüde daha kullanışlı hale getirerek yorumları etkinleştirir ve kalıp boşluklarını yok sayar:

val dateRegex = """(?x:
   (\d{4}) # year
  -(\d{2}) # month
  -(\d{2}) # day
)""".r

Kalıp yakalamalarınıza grup adları atamanıza izin veren, ‘r’, ‘def r(names: String*): Regex’in aşırı yüklenmiş bir sürümü vardır. Adlar yakalamalardan ayrıldığından bu biraz kırılgandır ve yalnızca normal ifade birden çok konumda kullanılacaksa kullanılmalıdır:

"""(\d{4})-(\d{2})-(\d{2})""".r("y", "m", "d").findFirstMatchIn(str) match {
  case Some(matched) =>
    val y = matched.group("y").toInt
    val m = matched.group("m").toInt
    val d = matched.group("d").toInt
    java.time.LocalDate.of(y, m, d)
  case None => ???
}

Bir dizgede bir kalıbın tekrar eşleştirilmesi #

val re = """\((.*?)\)""".r

val str = "(The)(example)(of)(repeating)(pattern)(in)(a)(single)(string)(I)(had)(some)(trouble)(with)(once)"

re.findAllMatchIn(str).map(_.group(1)).toList
res2: List[String] = List(The, example, of, repeating, pattern, in, a, single, string, I, had, some, trouble, with, once)
