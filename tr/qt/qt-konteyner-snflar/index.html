Qt, kendi şablon kapsayıcı sınıflarını sağlar. Hepsi dolaylı olarak paylaşılıyor. İki tür yineleyici sağlarlar (Java stili ve STL stili.)

Qt sıralı kaplar şunları içerir: QVector, QList, QLinkedList, QStack, QQueue.

Qt ilişkisel kapsayıcılar şunları içerir: QMap, QMultiMap, QHash, QMultiHash, QSet.

QStack kullanımı #

QStack<T>, yığın sağlayan bir şablon Qt sınıfıdır. STL’deki analogu ‘std::stack’ şeklindedir. Son giren ilk çıkar yapısıdır (LIFO).

QStack<QString> stack;
stack.push("First");
stack.push("Second");
stack.push("Third");
while (!stack.isEmpty())
{
    cout << stack.pop() << endl;
}

Çıktı verecek: Üçüncü, İkinci, Birinci.

“QStack”, “QVector"dan miras alır, dolayısıyla uygulaması STL’den oldukça farklıdır. STL’de ‘std::stack’, bir şablon argümanı (varsayılan olarak deque) olarak geçirilen tür için bir sarmalayıcı olarak uygulanır. Yine de ana işlemler “QStack” ve “std::stack” için aynıdır.

QVektör kullanımı #

QVector<T> dinamik dizi şablonu sınıfı sağlar. Çoğu durumda QList<T>den daha iyi performans sağlar, bu nedenle ilk tercih olmalıdır.

Çeşitli şekillerde başlatılabilir:

QVector<int> vect;
vect << 1 << 2 << 3;

QVector<int> v {1, 2, 3, 4};

En son, başlatma listesini içerir.

QVector<QString> stringsVector;
stringsVector.append("First");
stringsVector.append("Second");

Vektörün i-inci elemanını şu şekilde elde edebilirsiniz:

“v[i]” veya “at[i]”

‘i’nin geçerli konum olduğundan emin olun, ‘at(i)’ bile bir kontrol yapmaz, bu ‘std::vector’dan bir farktır.

QLinkedList kullanımı #

Qt’de, bağlı liste uygulamanız gerektiğinde QLinkedList’i kullanmalısınız.

“QLinkedList” - O(1) içine öğe eklemek, başa eklemek, eklemek hızlıdır, ancak dizin araması “QList” veya “QVector” - O(n)‘den daha yavaştır. Bu, bağlantılı listede bir şey bulmak için düğümler arasında yineleme yapmanız gerektiğini dikkate alarak normaldir.

Tam algoritmik karmaşıklık tablosu burada bulunabilir.

Sadece bazı öğeleri ‘QLinkedList’ içine eklemek için ‘«()’ operatörünü kullanabilirsiniz:

QLinkedList<QString> list;
list << "string1" << "string2" << "string3";

Öğeleri “QLinkedList"in ortasına eklemek veya öğelerinin tümünü veya bir kısmını değiştirmek için Java stili veya STL stili yineleyicileri kullanabilirsiniz. İşte, “QLinkedList"in tüm öğelerini 2 ile nasıl çarptığımıza dair basit bir örnek:

QLinkedList<int> integerList {1, 2, 3};
QLinkedList<int>::iterator it;
for (it = integerList.begin(); it != integerList.end(); ++it)
{
    *it *= 2;
}

QListesi #

“QList” sınıfı, listeler sağlayan bir şablon sınıfıdır. Öğeleri, hızlı dizin tabanlı erişim ve dizin tabanlı eklemeler ve çıkarmalar sağlayan bir listede saklar.

Listeye öğe eklemek için operatör<<(), insert(), append() veya prepend() kullanabilirsiniz. Örneğin:

operatör<<()

QList<QString> list;
list << "one" << "two" << "three";

insert()

QList<QString> list;
list << "alpha" << "beta" << "delta";
list.insert(2, "gamma");

ekle()

QList<QString> list;
list.append("one");
list.append("two");
list.append("three");

prepend()

QList<QString> list;
list.prepend("one");
list.prepend("two");
list.prepend("three");

Öğeye belirli bir dizin konumunda erişmek için operator[]() veya at() kullanabilirsiniz. at(), operator[]() dan daha hızlı olabilir, asla kapsayıcının derin kopyasına neden olmaz ve sabit zamanda çalışmalıdır. İkisi de argüman kontrolü yapmaz. Örnekler:

if (list[0] == "mystring")
    cout << "mystring found" << endl;

Veya

if (list.at(i) == "mystring")
    cout << "mystring found at position " << i << endl;

Öğeleri kaldırmak için, ‘removeAt()’, ’takeAt()’, ’takeFirst()’, ’takeLast()’, ‘removeFirst()’, ‘removeLast()’ veya ‘removeOne()’ gibi işlevler vardır. `. Örnekler:

takeFirst()

// takeFirst() removes the first item in the list and returns it
QList<QWidget *> list;
...
while (!list.isEmpty())
    delete list.takeFirst();

removeOne()

// removeOne() removes the first occurrence of value in the list
QList<QString> list;
list << "sun" << "cloud" << "sun" << "rain";
list.removeOne("sun");

Bir listedeki belirli bir değerin tüm oluşumlarını bulmak için “indexOf()” veya “lastIndexOf()” kullanabilirsiniz. Örnek:

indexOf()

int i = list.indexOf("mystring");
if (i != -1)
    cout << "First occurrence of mystring is at position " << i << endl;
