Atom’u (ve bazı faydalı paketleri) yükleyin #

https://atom.io/ adresine gidin ve atom düzenleyicisini yükleyin.

Ardından, daha kolay Titanyum kodlaması için bazı Atom paketleri kurun:

|Ad | Tip | Özellikler | |——————————— |————— |– ———— | titanyum dili javascript | Dil | JS Otomatik Tamamlama (alaşımsız)| | Titanyum Alaşımı | eklenti| Hepsi bir arada paket
Tanıma atla
İlgili aç
TSS Vurgula| | Ti-Create |eklenti| Projeler, kontrolör, modüller oluşturun| | Titanium-Build| eklenti| Simülatörde çalıştır (silme)|

Titanyum olmayan diğer faydalı paketler/eklentiler:

Ad Özellikler
Atom Güzelleştir Kod güzelleştirici (tss, xml, js desteği)
mini harita Tam kaynak kodunun bir önizlemesi.
minimap-highlight-selected Vurgulanan seçili paket için bir mini harita bağlama
vurgu seçili Çift tıklatıldığında seçilen geçerli sözcüğü vurgular
pigmentler Proje ve dosyalarda renkleri görüntülemek için bir paket.
Linter Cow Powers ile bir Base Linter çekirdeği (kendi başına hiçbir şey yapmaz, bir API tabanıdır)
Linter-jshint JavaScript için Linter eklentisi (bu, JS kodunuzu kontrol eder)
DocBlockr Belge yazmak için bir yardımcı paket
Terminal-plus Atom için temalar ve daha fazlasıyla eksiksiz bir terminal paketi
Proje Yöneticisi Proje yöneticisi

Kurulum veya Kurulum #

Appcelerator Titanium’u Yükleme #

İlk başta Titanium’u kurmamız gerekiyor:

  • uygulamaları derlemek için komut satırı araçları (CLI)
  • MVC çerçevesi Alloy
  • SDK

Ana parçalar, node.js paket yöneticisi ’npm’ kullanılarak kurulur. Yüklemeniz gerekiyorsa https://nodejs.org/ adresini kontrol edin.

Linux (Fedora) #

Fedora 23 kullanıyorsanız, gerekli kitaplıkları almak için aşağıdaki komutları çalıştırabilirsiniz:

# install tools and libraries needed for android sdk
dnf install nodejs npm git gcc glibc.i686 glibc-devel.i686 libstdc++.i686 zlib-devel.i686 ncurses-devel.i686 libX11-devel.i686 libXrender.i686 libXrandr.i686

# intall npm version 4.2.2
npm install -g npm
npm install n -g
n 4.2.2

# install cli tools
npm install -g titanium alloy appcelerator tisdk
  • Java JDK 8’i yükleyin: http://www.if-not-true-then-false.com/2014/install-oracle-Java-8-on-fedora-centos-rhel/
  • Android SDK’yı indirin (yalnızca SDK Araçları): https://developer.android.com/sdk/index.html#Other
  • Android SDK’yı açın ve SDK’yı yüklemek için android’i çalıştırın
  • .bash_profile’inizi ayarlayın:
echo " PATH=$PATH:$HOME/.local/bin:$HOME/android-sdk-linux/tools:$HOME/android-sdk-linux/platform-tools:/usr/java/latest/bin"
echo " export ANDROID_SDK=$HOME/android-sdk-linux"
echo " export JAVA_HOME=/usr/java/latest"
echo "export PATH"

OSX / Windows #

  • YAPILACAKLAR: düğümü/npm’yi Windows / OSX’e kurun*

Bir konsol açın ve araçları yüklemek için aşağıdaki komutu çalıştırın:

npm kurulum -g titanyum alaşımı tisdk

Titanyum SDK #

Bundan sonra SDK’yı kurmamız gerekiyor. Bunu yapmak için David Bankier’den tisdk aracını kullanacağız (https://github.com/dbankier/tisdk):

# mevcut titanyum sdk'leri listele
tisdk listesi

Çıktı böyle bir şey olacak

4.1.0.GA
4.1.0.Beta
4.0..RC5
4.0..RC4
4.0.0.RC3
4.0.0.RC2
4.0.0.RC
4.0.0.GA
...

Bu listeden en son GA’yı (4.1.0) seçip kuruyoruz.

tisdk 4.1.0.GA yükleyin

bu komutla titanyumun sdk’yi bulup bulmadığını kontrol edebilirsiniz:

ti sdk listesi

Ve birlikte

sen bilgi

bir şeyin eksik olup olmadığını görebilirsiniz (JDK nasıl kurulur ve Android SDK bunu takip eder)

Şimdi titanyum/alaşım projeleri oluşturmaya ve bunları derlemeye hazırsınız! Düzenleyiciyi kurma zamanı

en yeni SDK’yı edinin #

En yeni SDK, tisdk ile ikili dosya olarak mevcut değildir. Şunlarla derlemeniz gerekir:

tisdk derleme 5.0.0.GA

Daha fazla bilgi için https://github.com/dbankier/tisdk adresini ziyaret edin ve “Manuel yapılar"a bakın.

öbür metodlar #

  • Codexcast, “Titanium Mobile OSS ile Kurulumu Alma: SDK’yı derleme dahil hakkında bir video yayınladı -mobil-oss-dahil-sdk-derleme)”
  • resmi olmayan gece derlemelerini http://builds.appcelerator.com.s3.amazonaws.com/index.html#master adresinden edinin

kaynak: https://github.com/m1ga/titanium_with_atom#installing-appcelerator-titanium

İlk uygulamanızı oluşturun #

CLI ve Atom kullanarak boş bir Alloy uygulaması oluşturuyoruz.

Yeni bir terminal açın ve aşağıdakileri ekleyin:

--id com.test -d oluştur. -n UYGULAMA ADI -p tüm -t uygulama -u http://migaweb.de
cd UYGULAMA ADI/
alaşım yeni

Bu, temel bir uygulama (ad: APPNAME, paket tanımlayıcı: com.test, type: app, platform: all) oluşturacak ve ardından bunu bir Alloy projesine dönüştürecektir.

Atom paketini ti-create de kullanabilirsiniz.

Ağaç görünümünde açık olan klasörün içinde yeni bir proje oluşturacaktır. ‘Denetleyici/widget oluştur’ yalnızca mevcut bir Alloy projesinde çalışır (“Klasörü aç” — proje klasörünü seçin).

kaynak: https://github.com/m1ga/titanium_with_atom#create-your-first-app

Uygulamanızı derleyin #

Uygulamanızı derlemenin birkaç yolu vardır. Simülatörü/emülatörü kullanabilir, cihazınıza dağıtabilir veya mağaza apk’leri/ipa’ları oluşturabilirsiniz. Derleyiciyi beklerken size çok zaman kazandıran canlı bir test aracı (TiShadow) da vardır.

klişe yolu #

# androidden cihaza
ti build -p android -T cihazı

# android depolamak/dosyalamak için
ti build -p android -K /home/user/keyfile.keystore -T dist-playstore

# iOS simülatörü - boyutu/cihazı seçmek için bir menü gösterecek
build -p ios -C ?

# iOS'tan ipa'ya - tuşları seçmek için bir menü gösterecek
ti build -p ios --deploy-type üretim --ios-version 9.0 --keychain --target dist-adhoc --output-dir .
iOS ile ilgili #

Tüm dağıtım adlarını listelemek için şunları kullanabilirsiniz:

güvenlik bul-kimliği -v -p kod tasarımı
