Kurulum veya Kurulum #

Lütfen Sublime Text’in sürekli kullanımının bir lisans satın almanızı gerektirdiğini ve sizden [şartlar ve koşulları2 dikkate almanızın istendiğini unutmayın.

Sublime Text’i yükleme süreci her platform için farklıdır, ancak her durumda indirme sayfasını ziyaret etmeniz gerekir.

ST3’ü kurduktan sonra, paket yöneticisini Paket Kontrolü kurmak yaygındır.

Mac #

Mac için, OS X için Sublime Text’in yalnızca bir sürümü vardır.

  1. .dmg dosyasını indirin

  2. .dmg dosyasını açın

  3. Sublime Text 3 paketini Uygulamalar klasörüne sürükleyin

  4. Komut satırında kullanılacak sembolik bir bağlantı oluşturmak için following command at the terminal:

    ln -s “/Applications/Sublime Text.app/Contents/SharedSupport/bin/subl” /usr/local/bin/subl

Pencereler #

Windows için hem 64-bit hem de 32-bit sürümleri mevcuttur, taşınabilir ve taşınabilir değildir. Windows’un modern bir sürümünü kullanıyorsanız 64 bit sürümünü çalıştırabilmelisiniz. 64 bit sürümü çalıştırmakta sorun yaşıyorsanız, 32 bit sürümünü deneyin.

Taşınabilir veya Taşınabilir Değil mi? Sublime Text, Windows için iki şekilde gelir: normal ve taşınabilir. Çoğu kullanıcıya normal bir kurulum daha iyi hizmet vermelidir. Taşınabilir sürümü * yalnızca ihtiyacınız olduğunu biliyorsanız kullanın.*

Normal kurulumlar verileri iki klasör arasında ayırır: uygun kurulum klasörü ve veri dizini (veriler için kullanıcıya özel dizin). Normal kurulumlar ayrıca Sublime Text’i Windows Explorer’ın içerik menüsüyle bütünleştirir.

Taşınabilir kurulumlar, Sublime Text’in ihtiyaç duyduğu tüm dosyaları tek bir klasörde tutar. Bu klasör hareket ettirilebilir ve düzenleyici çalışmaya devam eder.

Sublime Text’in Normal Versiyonu Nasıl Kurulur

  1. Yükleyiciyi indirin
  2. Yükleyiciye çift tıklayın

Sublime Text’in Taşınabilir Sürümü Nasıl Yüklenir

  1. Sıkıştırılmış dosyaları indirin

  2. Sıkıştırılmış dosyaları istediğiniz bir klasöre açın

Bu klasörün içinde yürütülebilir sublime_text.exe dosyasını bulacaksınız.

Linux #

Linux için, işletim sisteminizin türünü kontrol etmek için terminalinizde bu komutu çalıştırın:

uname -m

Paketi indirebilir ve manuel olarak açabilirsiniz. Alternatif olarak, komut satırını kullanabilirsiniz.

Ubuntu -i386 için

  cd ~
  wget http://c758482.r82.cf2.rackcdn.com/sublime-text_build-3083_i386.deb

  • x64 için

    cd ~
wget http://c758482.r82.cf2.rackcdn.com/sublime-text_build-3083_amd64.deb

Diğer Linux Dağıtımları -i386 için

  cd ~
  wget http://c758482.r82.cf2.rackcdn.com/sublime_text_3_build_3083_x32.tar.bz2
  tar vxjf sublime_text_3_build_3083_x32.tar.bz2

  • x64 için

    cd ~
wget http://c758482.r82.cf2.rackcdn.com/sublime_text_3_build_3083_x64.tar.bz2
tar vxjf sublime_text_3_build_3083_x64.tar.bz2

Şimdi sıkıştırılmamış dosyaları uygun bir konuma taşımalıyız.

sudo mv Sublime\ Text\ 3 /opt/
Lastly, we create a symbolic link to use at the command line.

sudo ln -s /opt/Sublime\ Text\ 3/sublime_text /usr/bin/sublime

Ubuntu’da, Unity başlatıcısına Sublime Text eklemek istiyorsanız aşağıdakileri yapın:

İlk önce yeni bir dosya oluşturun.

sudo sublime /usr/share/applications/sublime.desktop

Aşağıdakileri içine kopyalayın.

[Desktop Entry]
Version=1.0
Name=Sublime Text 3
# Only KDE 4 seems to use GenericName, so we reuse the KDE strings.
# From Ubuntu's language-pack-kde-XX-base packages, version 9.04-20090413.
GenericName=Text Editor

Exec=sublime
Terminal=false
Icon=/opt/Sublime Text 3/Icon/48x48/sublime_text.png
Type=Application
Categories=TextEditor;IDE;Development
X-Ayatana-Desktop-Shortcuts=NewWindow

[NewWindow Shortcut Group]
Name=New Window
Exec=sublime -n
TargetEnvironment=Unity

Sublime Text kopyanızı kaydettiyseniz, ancak her açtığınızda lisansınızı girmeniz istenirse, bu komutu çalıştırmayı denemelisiniz.

sudo chown -R username:username /home/username/.config /sublime-text-3

Kullanıcı adını hesabınızın kullanıcı adıyla değiştirin. Bu, lisansı ilk girdiğinizde Sublime Text’i root olarak açmanız durumunda izin hatasını düzeltmelidir.

Referans

Yüce Eğitmen ile başlayın #

Bu eğitim, klasik vimtutor’dan ilham almıştır. Sublime Text 3 ile çalışmak için bazı kullanışlı kısayolları öğreneceksiniz. Bu eğitimin sonunda, ST’nin en önemli ve sık kullanılan kısayollarına ve özelliklerine aşina olacaksınız.

Kurulum #

Paket Kontrolü aracılığıyla:

  1. Henüz yüklü değilse Paket Kontrolü yükleyin: https://packagecontrol.io/installation#st3
  2. Komut paletini getirmek için Cmd+Shift+P tuşlarına basın önünde
  3. Install Package yazın ve enter tuşuna basın.
  4. Sublime Tutoru arayın ve eklentiyi kurmak için enter tuşuna basın.

Manuel Kurulum: #

  1. git‘in zaten kurulu olduğundan emin olun.
  2. Sublime Text’in “Packages” dizinine “cd”. 3. Mac’te genellikle şu yolda bulunur: ~/Library/Application Support/Sublime Text 3 /Paketler/. Alternatif olarak bir menü öğesi aracılığıyla da açabilirsiniz: Tercihler > Paketlere Gözat...
  3. Paketler dizinine girdikten sonra bu depoyu klonlayın: git klon git@github.org:jai/sublimetutor.git. Alternatif olarak indirin ve platformunuz için en son sürümü buradan çıkarın: https://github.com/jaipandya/SublimeTutor/releases
  4. Sublime Text’i Yeniden Başlatın

Başlarken #

Henüz yapmadıysanız, kurulum adımlarını kullanarak Sublime Tutor’u kurun yukarıda verilen.

Sublime Tutor yüklendikten sonra, Ctrl+Option+K tuşlarına basın Bu dosyayı Sublime Text’de açmak için klavye kısayolu. Başka bir seçenek de gitmek Bunu açmak için Help > Sublime Tutor menü seçeneği.

Komut Paleti ile:

  1. Komut paletini almak için Cmd+Shift+P ön.
  2. Sublime Tutor yazın, programı başlatmak için gelen ilk komutu seçin. etkileşimli rehber.
