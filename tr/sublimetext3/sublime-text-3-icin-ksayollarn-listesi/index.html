Sublime Text 3 için Kısayolların Listesi

  • Windows Kısayolları

Bu sayfada

  • Windows Kısayolları

Bu, Sublime Text sürüm 3’te yaygın olarak kullanılan kısayolların bir listesidir. Başkalarını biliyorsanız, bir düzenleme önermekten çekinmeyin, ancak bildiğim [Windows] ile başlayacağım. Yüce metin belgelerinde bir kısayol listesi vardır (http://docs.sublimetext.info/en/latest/reference/keyboard_shortcuts_win.html), ancak sayfanın üst kısmında bazılarının çalışmayabileceğine dair bir not vardır [ Bildiğim kadarıyla burada listelenenler işe yarıyor!]. Bağlantının OSX sayfasına bağlandığını da unutmayın.

Windows Kısayolları #

Genel Açıklama
CTRL + ` Göster Gizle Konsolu
CTRL + K veya CTRL + B Kenar Çubuğunu Göster/Gizle
CTRL + SHIFT + P Komut Paleti
CTRL + SPACE ‘Otomatik tamamlama’ önerisini seçin
CTRL + Y Son klavye kısayolunu yeniden yap
Metin Seçme Kısayol Eylemi
CTRL + D Bir kelime seçer/ ‘sonraki yinelemeyi seç’ için kullanılır
CTRL + U Sonraki yinelemenin seçimini kaldırın
CTRL + L Bir satır seçer
CTRL + SHIFT + J İçeriği Girinti
CTRL + ALT + UP Sütun seçimi yukarı
CTRL + ALT + DOWN Sütun seçimi aşağı
Windows Denetimi Windows Denetimi
F11 Tam ekran
SHIFT + F11 Dikkat dağıtmayan mod
CTRL + N Yeni Sekme
CTRL +SHIFT + N Yeni Pencere
CTRL + W Sekmeyi Kapat
ALT + (sekme no) Sekmeyi Değiştir
CTRL + PGUP Önceki Sekmeye Git
CTRL + PGDOWN Sonraki Sekmeye Git
CTRL + (bölme no) Bölme Numarasına Geç
CTRL + SHIFT + (bölme no) Sekmeyi bölmeye taşı Numara
Bul ve Değiştir Açıklama
CTRL + F Bul
F3 Sonrakini Bul
SHIFT + F3 Önceki Bul
CTRL + H Değiştir
CTRL + SHIFT + F Dosyalarda Bul
Tipografi Açıklama
CTRL + K Veya CTRL + U Büyük Harfe Dönüştür
CTRL + K Veya CTRL + L Küçük Harfe Dönüştür
CTRL + ARTI(+) Yazı tipi boyutunu büyüt
CTRL + EKSİ(-) Yazı tipi boyutunu küçült
Yer imleri Açıklama
CTRL + F2 Yeni Yer İşareti Oluştur
F2 Sonraki yer işaretine git
SHIFT + F2 Önceki Yer İşaretine Git
CTRL + SHIFT + F2 Tüm yer imlerini temizle
