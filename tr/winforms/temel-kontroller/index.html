Buton #

Düğmeler en basit kontrollerden biridir ve çoğunlukla kullanıcı istediği zaman bazı kodları çalıştırmak için kullanılır.

Burada gerçekten basit bir durumumuz var, bir düğmeye tıklandığında bir Mesaj kutusu göster. Bir forma bir düğme ekleriz, nesneyi tanımlamak için kodda kullanıldığı gibi cmdShowMessage olarak adlandırırız ve düğme metnini Mesajı Göster olarak ayarlarız.

Sadece görsel tasarımcı üzerindeki düğmeye çift tıklamamız gerekiyor ve Visual Studio, tıklama Etkinliği için kod üretecek. Şimdi sadece MessageBox kodunu buraya eklememiz gerekiyor:

private void cmdShowMessage_Click(object sender, EventArgs e)
{
  MessageBox.Show("Hello world!");
}

Programı şimdi çalıştırırsak ve butona tıklarsak aşağıdaki mesajı göreceğiz:

Metin Kutuları, kullanıcının programa veri girmesine izin verir.

Formu değiştireceğiz ve bir metin kutusu ekleyeceğiz, böylece mesaj kutusu bize kullanıcının istediği mesajı gösterecek. Şimdi formumuz şuna benziyor:

Ardından, metin kutusunun metnini kullanmak için düğme tıklama olayını değiştirin:

private void cmdShowMessage_Click(object sender, EventArgs e)
{
  string UserText = txtUserMessage.Text;
  MessageBox.Show(UserText);
}

Açılan kutu #

ComboBox’lar, kullanıcının geliştirici tarafından sağlanan çeşitli seçeneklerden birini seçmesine izin verir.

Formu değiştireceğiz ve bir birleşik giriş kutusu ekleyeceğiz, böylece mesaj kutusu bize sunacağımız bir listeden kullanıcının istediği mesajı gösterecek.

Combo’yu forma ekledikten sonra, şimdi combo’ya bir seçenekler listesi ekliyoruz. Bunu yapmak için “Öğeler” özelliğini değiştirmemiz gerekiyor:

Şimdi click olayının kodunu değiştirmemiz gerekiyor:

private void cmdShowMessage_Click(object sender, EventArgs e)
{
  string OptionText = cboSelectOption.SelectedItem.ToString();
  MessageBox.Show(OptionText);
}

Gördüğünüz gibi SelectedItem özelliğini kullanıyoruz, seçili seçeneğin nesnesini içeriyor. Gösterilecek bir dizgeye ihtiyacımız olduğundan ve derleyici nesnenin bir dizge olup olmadığını bilmediğinden, ToString() yöntemini kullanmamız gerekiyor.

Programı çalıştırırsak, tercih ettiğimiz seçeneği seçebileceğiz ve butona tıkladığımızda mesaj kutusu bunu gösterecek:

Bir kullanıcı açılan kutudan bir öğe seçtiğinde bilgilendirilmek için SelectionChangeCommitted olayını kullanın. SelectedIndexChanged olayını kullanabilirdik, ancak bu, birleşik giriş kutusundaki select öğesini programlı olarak değiştirdiğimizde de ortaya çıkıyor.

Onay Kutusu #

Onay kutusu, kullanıcının “İyi misin?” gibi belirli bir soru için kullanıcıdan “boolean” değerleri almasını sağlayan bir kontroldür.

“check” özelliği değiştirildiğinde ortaya çıkan “CheckedChanged” adlı bir olaya sahiptir.

İşte “Kontrol Edildi mi?” sorusu olan bir CheckBox.

Bu ‘MessageBox’ı ‘CheckedChanged’ olayından aldık,

private void checkBox1_CheckedChanged(object sender, EventArgs e)
        {
            bool IsChecked = checkBox1.Checked; 
            MessageBox.Show(IsChecked.ToString());
        }

CheckBox işaretliyse -> ‘IsChecked’ değişkeni ’true’ olacaktır.

CheckBox işaretli değilse -> IsChecked değişkeni false olacaktır.

Liste kutusu #

‘Listbox’, nesnelerin koleksiyonunu içerebilen bir kontroldür. ‘Liste kutusu’, ‘Combobox’a benzer ancak ‘Combobox’ içindedir; Yalnızca seçilen öğeler kullanıcı tarafından görülebilir. ‘Liste Kutusu’nda; tüm öğeler kullanıcı tarafından görülebilir.

ListBox’a nasıl öğe eklenir?

private void Form3_Load(object sender, EventArgs e)
        {
            string test = "Trial";
            string test2 = "45";
            int numberTest = 43;
            decimal decimalTest = 130;
            
            listBox1.Items.Add(test);
            listBox1.Items.Add(test2);
            listBox1.Items.Add(numberTest);
            listBox1.Items.Add(decimalTest);
        }

SayısalYukarıAşağı #

NumericUpDown, TextBox’a benzeyen kontroldür. Bu kontrol, kullanıcının bir aralıktan sayı görüntülemesini/seçmesini sağlar. Yukarı ve Aşağı okları metin kutusu değerini güncelliyor.

Kontrol görünümü;

Form_Load aralığında ayarlanabilir.

private void Form3_Load(object sender, EventArgs e)
    {
        numericUpDown1.Maximum = 10;
        numericUpDown1.Minimum = -10; 
    }

Çıktı;

UpDownAlign okların konumunu belirler;

 private void Form3_Load(object sender, EventArgs e)
        {
            numericUpDown1.UpDownAlign = LeftRightAlignment.Left;
            
        }

Çıktı;

UpButton() Yöntemi, kontrol sayısını artırır. (her yerden aranabilir. Onu çağırmak için bir ‘düğme’ kullandım.)

private void button1_Click(object sender, EventArgs e)
        {
            numericUpDown1.UpButton();
        }

DownButton() Yöntemi, kontrol sayısını azaltır. (her yerden aranabilir. Tekrar aramak için bir ‘düğme’ kullandım.)

 private void button2_Click(object sender, EventArgs e)

Değer Değişti;

Bu olay Yukarı veya Aşağı ok tıklandığında çalışacaktır.

private void numericUpDown1_ValueChanged(object sender, EventArgs e)
        {
            decimal result = numericUpDown1.Value; // it will get the current value
            if (result == numericUpDown1.Maximum) // if value equals Maximum value that we set in Form_Load.
            {
                label1.Text = result.ToString() + " MAX!"; // it will add "MAX" at the end of the label 
            }
            else if (result == numericUpDown1.Minimum) //  if value equals Minimum value that we set in Form_Load.
            {
                label1.Text = result.ToString() + " MIN!"; // it will add "MIN" at the end of the label 
            }
            else
            {
                label1.Text = result.ToString(); // If Not Max or Min, it will show only the number.
            }
        }
