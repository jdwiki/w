Hive Kurulumu(linux) #

https://hive.apache.org/downloads.html adresinden en son kararlı sürümü indirerek başlayın.

-> Şimdi dosyayı şu şekilde açın:

$ tar -xvf kovan-2.x.y-bin.tar.gz

-> /usr/local/ içinde bir dizin oluşturun

$ sudo mkdir /usr/local/hive

-> Dosyayı kök dizinine taşıyın

$ mv ~/Downloads/Hive-2.x.y /usr/local/Hive

-> .bashrc’de hadoop ve Hive için ortam değişkenlerini düzenleyin

$ gedit ~/.bashrc

bunun gibi

HIVE_HOME=/usr/local/Hive/Apache-Hive-2.0.1-bin/ dışa aktar

dışa aktar YOL=$YOL:$HIVE_HOME/bin

CLASSPATH’i dışa aktar=$CLASSPATH:/usr/local/Hadoop/lib/*:.

CLASSPATH’i dışa aktar=$CLASSPATH:/usr/local/Hive/apache-hive-2.0.1-bin/lib/*:.

-> Şimdi, zaten çalışmıyorsa hadoop’u başlatın. Ve çalıştığından ve güvenli modda olmadığından emin olun.

$ hadoop fs -mkdir /kullanıcı/kovan/depo

“Depo” dizini, kovanla ilgili tablo veya verilerin saklanacağı yerdir.

$ hadoop fs -mkdir /tmp

Geçici dizin “tmp”, işlemenin ara sonucunu depolamak için geçici konumdur.

-> Bu klasörlerde okuma/yazma izinlerini ayarlayın.

$ hadoop fs -chmod g+w /kullanıcı/kovan/depo

$ hadoop fs -chmod g+w /user/tmp

-> Şimdi konsolda bu komutla HIVE’ı çalıştırın

$ kovan

Hive’da Kelime Sayısı Örneği #

Dokümanlar dosyası (Giriş Dosyası)

Mary küçük bir kuzuya sahipti

yapağı kar gibi beyazdı

ve Mary’nin gittiği her yerde

kuzunun gideceği kesindi.

** Kovan Sorgusu **

CREATE TABLE FILES (line STRING);

LOAD DATA INPATH 'docs' OVERWRITE INTO TABLE FILES;

CREATE TABLE word_counts AS
SELECT word, count(1) AS count FROM
(SELECT explode(split(line, ' ')) AS word FROM FILES) w
GROUP BY word
ORDER BY word;

Linux’ta Harici Metastore ile Hive Kurulumu #

Ön koşullar:

  1. Java 7
  2. Hadoop (Hadoop Kurulumu için [buraya][1] bakın)
  3. Mysql Sunucusu ve İstemcisi

Kurulum:

Adım 1: En son Hive tarball’ı [indirilenler][2] sayfasından indirin.

  1. Adım: İndirilen tarball’ı çıkarın (Varsayım: Tarball $HOME içinde indirilir)

    tar -xvf /home/username/apache-hive-x.y.z-bin.tar.gz

  2. Adım: Ortam dosyasını güncelleyin (~/.bashrc)

    export HIVE_HOME=/home/username/apache-hive-x.y.z-bin export PATH=$HIVE_HOME/bin:$PATH yeni ortam değişkenlerini ayarlamak için dosyayı kaynaklayın.

    source ~/.bashrc

Adım 4: Mysql için JDBC bağlayıcısını [buradan][3] indirin ve çıkarın.

tar -xvf mysql-connector-java-a.b.c.tar.gz

Çıkarılan dizin, mysql-connector-java-a.b.c.jar bağlayıcı jar dosyasını içerir. $HIVE_HOME libine kopyalayın

cp mysql-connector-java-a.b.c.jar $HIVE_HOME/lib/

Yapılandırma:

$HIVE_HOME/conf/ dizini altında Hive-site.xml kovan yapılandırma dosyasını oluşturun ve metastore ile ilgili aşağıdaki özellikleri ekleyin.

<configuration>
   <property>
      <name>javax.jdo.option.ConnectionURL</name>
      <value>jdbc:mysql://localhost/hive_meta</value>
      <description>JDBC connect string for a JDBC metastore</description>
   </property> 
   
   <property>
      <name>javax.jdo.option.ConnectionDriverName</name>    
      <value>com.mysql.jdbc.Driver</value>
      <description>Driver class name for a JDBC metastore</description>
   </property> 
   
   <property>
       <name>javax.jdo.option.ConnectionUserName</name> 
       <value>mysqluser</value>
       <description>username to use against metastore database</description>
   </property> 
   
   <property>
       <name>javax.jdo.option.ConnectionPassword</name> 
       <value>mysqlpass</value>
       <description>password to use against metastore database</description>
   </property> 

   <property>
       <name>datanucleus.autoCreateSchema</name>
       <value>false</value> 
   </property> 

   <property>
       <name>datanucleus.fixedDatastore</name>
       <value>true</value>
   </property>
</configuration>

Özelliklerde MySQL “kullanıcı adı” ve “şifre” değerlerini uygun şekilde güncelleyin.

Metastore Şemasını Oluşturun:

Metastore şema komut dosyaları $HIVE_HOME/scripts/metastore/upgrade/mysql/ altında mevcuttur.

MySQL’e giriş yapın ve şemayı kaynaklayın,

mysql -u username -ppassword

mysql> create database hive_meta;
mysql> use hive_meta;
mysql> source hive-schema-x.y.z.mysql.sql;
mysql> exit;

Metastore’u Başlatma:

hive --service metastore

Arka planda çalıştırmak için,

nohup hive --service metastore &

HiveServer2’yi Başlatma: (Gerekirse kullanın)

hiveserver2

Arka planda çalıştırmak için,

nohup hiveserver2 metastore &

Not: Bu yürütülebilir dosyalar $HIVE_HOME/bin/ altında mevcuttur.

Bağlamak:

Hive ile bağlantı kurmak için “Hive”, “beeline” veya [Hue][4] kullanın.

Hive CLI kullanımdan kaldırılmıştır, Beeline veya Hue kullanılması önerilir.

Hue için Ek Yapılandırmalar:

Bu değeri $HUE_HOME/desktop/conf/hue.ini içinde güncelleyin

[beeswax] 
    hive_conf_dir=/home/username/apache-hive-x.y.z-bin/conf
