Sass exentsion Compass’ı kullanmaya başlama kılavuzu. Pusula, CSS3 özelliklerini kullanan her tarayıcıyı desteklemek için 1 satır yazmak için mixins sağladığı için CSS3 ile uğraşırken çok kullanışlıdır. Sprite görüntüleri eklemek de harika.

Ortamı ayarla #

Komut satırınızı açın

Ruby kullanarak kurulum #

gem update --system

gem install compass

Proje Oluştur #

compass create <myproject>

Bu bir pusula projesini başlatacaktır. adlı bir klasör ekleyecektir. Klasör aşağıdaki yapıya sahip gibi görünecektir:

Dosya/Klasör açıklama
küstah/ sass/scss dosyalarını bu klasöre koyun
stil sayfaları/ Bu klasörde derlenmiş css’iniz saklanacaktır
yapılandırma.rb Pusulayı yapılandırın - ör. klasör yolu, sass derlemesi

Pusulayı kullan #

compass watch

Bu, sass dosyalarınızı her değiştirdiğinizde derleyecektir. sass klasör yolu config.rb içinde değiştirilebilir

CSS3’ü pusula ile kullanma #

Bu sayfada hangi CSS3 bileşenlerinin desteklendiğine dair eksiksiz bir referans bulabilirsiniz.

CSS3’ü projenizde kullanabilmeniz için Compass, her tarayıcıda CSS3 özelliklerini destekleyen karışımlar sağlar. Sass/Scss dosyanızın üstünde pusula kullanmak istediğinizi belirtmelisiniz.

@import "compass/css3";

Sınır yarıçapı #

Sass dosyanıza sınır yarıçapını pusula ile ekleyin:

div {
    @include border-radius(4px);
}

CSS çıktısı

div {
  -moz-border-radius: 4px;
  -webkit-border-radius: 4px;
  border-radius: 4px;
}

Gördüğünüz gibi normal CSS adını kullanabilirsiniz. Önüne @include koyun ve değerinizi ayarlamak için ( ) kullanın.

Flexbox Örneği #

.row {
  @include display-flex;
  @include flex-direction(row);
}

CSS Çıktısı

.row {
  display: -webkit-flex;
  display: flex;
  -webkit-flex-direction: row;
  flex-direction: row;
}

Çözüm #

Bu sadece iki örnek. Pusula çok daha fazla CSS3 karışımı sağlar. Pusula kullanmak çok kullanışlıdır ve belirli bir tarayıcı için bir CSS3 bileşeni tanımlamayı unuttuğunuz için endişelenmenize gerek yoktur. Tarayıcı CSS3 özelliğini destekliyorsa pusula sizin için tanımlayacaktır.
