Aşağıda bir veri pompası içe/dışa aktarma oluşturma adımları verilmiştir:

Datapump işlerini izleyin #

Datapump işleri aşağıdakiler kullanılarak izlenebilir:

1. veri sözlüğü görünümleri:

   select * from dba_datapump_jobs; 
   SELECT * FROM DBA_DATAPUMP_SESSIONS; 
   select username,opname,target_desc,sofar,totalwork,message from V$SESSION_LONGOPS where username = 'bkpadmin';

2. Veri pompası durumu:

  • İçe/dışa aktarma günlüklerinden veya veri sözlüğü adından iş adını not edin ve

  • ekle komutunu çalıştırın:

  • İçe/Dışa Aktar komut istemine durumu yazın

    impdp / attach=<SYS_IMPORT_SCHEMA_01> Import> status İçe/Dışa Aktar komut isteminden çıkmak için CTRL+C tuşlarına basın

Adım 3/6 : Dizin oluşturun #

create or replace directory DATAPUMP_REMOTE_DIR as '/oracle/scripts/expimp';

Adım 7 : Komutları Dışa Aktarın #

Komutlar:

expdp <bkpadmin>/<bkp123>  parfile=<exp.par>

*Lütfen <> içindeki verileri ortamınıza göre uygun değerlerle değiştirin. Gereksinimlerinize göre parametreler ekleyebilir/değiştirebilirsiniz. Yukarıdaki örnekte, kalan tüm parametreler aşağıda belirtildiği gibi parametre dosyalarına eklenir: *

  • Dışa Aktarma Türü : Kullanıcı Dışa Aktarma

  • Tüm şemayı dışa aktar

  • Parametre dosyası ayrıntıları [exp.par deyin] :

    schemas= directory= DATAPUMP_REMOTE_DIR dumpfile=.dmp logfile=exp_.log

  • Dışa Aktarma Türü : Büyük şema için Kullanıcı Dışa Aktarma

  • Büyük veri kümeleri için tüm şemayı dışa aktarın: Burada dışa aktarma döküm dosyaları bölünecek ve sıkıştırılacaktır. Burada paralellik kullanılmıştır (Not : Paralellik eklemek sunucudaki CPU yükünü artıracaktır)

  • Parametre dosyası ayrıntıları [exp.par deyin] :

    schemas= directory= DATAPUMP_REMOTE_DIR dumpfile=%U.dmp logfile=exp__.log compression = all parallel=5

  • Dışa Aktarma Türü : Tablo Dışa Aktarma [ Tablo kümesini dışa aktar]

  • Parametre dosyası ayrıntıları [exp.par deyin] :

    tables= tname1, tname2, tname3 directory= DATAPUMP_REMOTE_DIR dumpfile=.dmp logfile=exp_.log

Adım 9 : Komutları İçe Aktar #

Ön koşul:

  • Kullanıcı içe aktarımından önce, içe aktarılan şemayı veya tabloyu bırakmak iyi bir uygulamadır.

Komutlar:

impdp <bkpadmin>/<bkp123>  parfile=<imp.par>

*Lütfen <> içindeki verileri ortamınıza göre uygun değerlerle değiştirin. Gereksinimlerinize göre parametreler ekleyebilir/değiştirebilirsiniz. Yukarıdaki örnekte, kalan tüm parametreler aşağıda belirtildiği gibi parametre dosyalarına eklenir: *

  • İçe Aktarma Türü : Kullanıcı İçe Aktarma

  • Tüm şemayı içe aktar

  • Parametre dosyası ayrıntıları [imp.par deyin] :

    schemas= directory= DATAPUMP_REMOTE_DIR dumpfile=.dmp logfile=imp_.log

  • İçe Aktarma Türü : Büyük şema için Kullanıcı İçe Aktarma

  • Büyük veri kümeleri için tüm şemayı içe aktarın: Burada paralellik kullanılır (Not: Paralellik eklemek sunucudaki CPU yükünü artıracaktır)

  • Parametre dosyası ayrıntıları [imp.par deyin] :

    schemas= directory= DATAPUMP_REMOTE_DIR dumpfile=%U.dmp logfile=imp__.log parallel=5

  • İçe Aktarma Türü : Tablo İçe Aktarma [ Tablo kümesini içe aktarma]

  • Parametre dosyası ayrıntıları [imp.par deyin] :

    tables= tname1, tname2, tname3 directory= DATAPUMP_REMOTE_DIR dumpfile=.dmp logfile=exp_.log TABLE_EXISTS_ACTION= <APPEND /SKIP /TRUNCATE /REPLACE>

1. Veri pompası adımları #

Kaynak Sunucu [Verileri Dışa Aktar] Hedef Sunucu [Verileri İçe Aktar]
1. Dışa aktarma döküm dosyalarını içerecek bir veri pompası klasörü oluşturun 4. İçe aktarma döküm dosyalarını içeren bir veri pompası klasörü oluşturun
2. Dışa aktarmayı gerçekleştirecek veritabanı şemasına giriş yapın. 5. İçe aktarmayı gerçekleştirecek veritabanı şemasına giriş yapın.
3. Adım 1’e işaret eden bir dizin oluşturun. 6. Adım 4’ü gösteren bir dizin oluşturun.
7. Dışa Aktarma İfadelerini Çalıştırın.
8. Döküm dosyalarını Hedef Sunucuya kopyalayın/SCP yapın.
9. Import deyimlerini çalıştır
10. verileri kontrol edin, geçersiz nesneleri derleyin ve ilgili hibeleri sağlayın

Tabloları farklı şemalar ve tablo alanları arasında kopyalayın #

 expdp <bkpadmin>/<bkp123> directory=DATAPUMP_REMOTE_DIR dumpfile=<customer.dmp>


 impdp <bkpadmin>/<bkp123> directory=DATAPUMP_REMOTE_DIR dumpfile=<customer.dmp> remap_schema=<source schema>:<target schema> remap_tablespace=<source tablespace>:<target tablespace>
