Kurulum veya Kurulum #

Tkinter, Mac OS X ve Windows platformu için Python yükleyici ikili dosyalarıyla önceden yüklenmiş olarak gelir. Bu nedenle, Python’u Mac OS X veya Windows platformu için resmi ikililerden yüklerseniz, Tkinter’ı kullanabilirsiniz.

Linux’un Debian sürümleri için aşağıdaki komutları kullanarak manuel olarak kurmanız gerekir.

Python 3 için

sudo apt-get install python3-tk

Python 2.7 için

sudo apt-get install python-tk

Yum yükleyicili Linux dağıtımları, şu komutu kullanarak tkinter modülünü kurabilir:

yum tkinter’ı kurun

Yüklemeyi Doğrulama

Tkinter’ı başarıyla kurup kurmadığınızı doğrulamak için Python konsolunuzu açın ve aşağıdaki komutu yazın:

import tkinter as tk # for Python 3 version

veya

import Tkinter as tk # for Python 2.x version

Yukarıdaki komut hatasız bir şekilde yürütülürse, Tkinter’ı başarıyla yüklediniz.

Tkinter sürümünü kontrol etmek için Python REPL’inize aşağıdaki komutları yazın:

piton 3.X için

import tkinter as tk
tk._test()

piton 2.X için

import Tkinter as tk
tk._test()

Not: “Tkinter as tk"yi içe aktarmak gerekli değildir, ancak sürümler arasında işlerin tutarlı kalmasına yardımcı olduğu için iyi bir uygulamadır.

Selam Dünya! (modüler, nesne yönelimli) #

import tkinter as tk

class HelloWorld(tk.Frame):
    def __init__(self, parent):
        super(HelloWorld, self).__init__(parent)

        self.label = tk.Label(self, text="Hello, World!")
        self.label.pack(padx=20, pady=20)
        
if __name__ == "__main__":
    root = tk.Tk()

    main = HelloWorld(root)
    main.pack(fill="both", expand=True)

    root.mainloop()

Not: Aşağıdakiler de dahil olmak üzere hemen hemen tüm tkinter widget’larından miras almak mümkündür. kök pencere. “tkinter.Frame"den devralma en azından tartışmalıdır birden çok belge arabirimini (MDI) desteklemesi bakımından en esnek olanı, tek belge arabirimleri (SDI), tek sayfa uygulamaları ve çok sayfalı uygulamalar.

Selam Dünya! (en az) #

Klasik “Merhaba, Dünya!“yı yaratarak temel tkinter bilgimizi test edelim. programı.

İlk olarak, tkinter’ı içe aktarmalıyız, bu sürüme göre değişecektir (“Python 2 ve 3 arasındaki farklar” hakkındaki açıklamalar bölümüne bakın)

Python 3’te tkinter modülü küçük t harfine sahiptir:

import tkinter as tk

Python 2’de Tkinter modülü büyük T harfine sahiptir:

import Tkinter as tk

as tk kullanmak kesinlikle gerekli değildir, ancak bu örneğin geri kalanı her iki sürüm için de aynı şekilde çalışacak şekilde kullanacağız.

artık tkinter modülünü içe aktardığımıza göre, Tk sınıfını kullanarak uygulamamızın kökünü oluşturabiliriz:

root = tk.Tk()

Bu, uygulamamız için pencere görevi görecektir. ( additional pencerelerinin bunun yerine “Üst düzey” örnekler olması gerektiğini unutmayın)

Artık bir penceremiz olduğuna göre, ona bir `Etiket’ ile metin ekleyelim.

label = tk.Label(root, text="Hello World!") # Create a text label
label.pack(padx=20, pady=20) # Pack it into the window

Uygulama hazır olduğunda, ‘mainloop’ yöntemiyle başlatabiliriz (main event loop‘u girin)

root.mainloop()

Bu, pencere kapatılana veya root.destroy() gibi geri aramalardan (daha sonra tartışılacaktır) çıkan fonksiyonları çağırarak durdurulana kadar uygulamayı açacak ve çalıştıracaktır.

Hepsini bir araya koy:

import tkinter as tk # Python 3.x Version
#import Tkinter as tk # Python 2.x Version

root = tk.Tk()

label = tk.Label(root, text="Hello World!") # Create a text label
label.pack(padx=20, pady=20) # Pack it into the window

root.mainloop()
