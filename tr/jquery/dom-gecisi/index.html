Öğenin alt öğelerini seçin #

Bir öğenin alt öğelerini seçmek için children() yöntemini kullanabilirsiniz.

<div class="parent">
    <h2>A headline</h2>
    <p>Lorem ipsum dolor sit amet...</p>
    <p>Praesent quis dolor turpis...</p>
</div>

“.parent” öğesinin tüm alt öğelerinin rengini değiştirin:

$('.parent').children().css("color", "green");

Yöntem, döndürülen öğeleri filtrelemek için kullanılabilecek isteğe bağlı bir “seçici” argümanını kabul eder.

// Only get "p" children
$('.parent').children("p").css("color", "green");

jQuery öğeleri listesi üzerinde yineleme #

jQuery öğelerinin listesini yinelemeniz gerektiğinde.

Bu DOM yapısını düşünün:

<div class="container">
    <div class="red one">RED 1 Info</div>
    <div class="red two">RED 2 Info</div>
    <div class="red three">RED 3 Info</div>
</div>

Tüm “div” öğelerinde bulunan metni “red” sınıfıyla yazdırmak için:

$(".red").each(function(key, ele){
    var text = $(ele).text();
    console.log(text);
});

İpucu: “anahtar”, şu anda üzerinde yinelediğimiz “div.red” öğesinin üst öğesi içinde dizinidir. “ele” HTML öğesidir, dolayısıyla “$()” veya “jQuery()” kullanarak ondan bir jQuery nesnesi oluşturabiliriz, örneğin: “$(ele)”. Daha sonra, nesne üzerinde css() veya hide() vb. gibi herhangi bir jQuery yöntemini çağırabiliriz. Bu örnekte, sadece nesnenin metnini çekiyoruz.

Kardeş seçme #

Bir öğenin kardeşlerini seçmek için .siblings() yöntemini kullanabilirsiniz.

Bir öğenin kardeşlerini değiştirmek istediğiniz tipik bir örnek, bir menüdedir:

<ul class="menu">
    <li class="selected">Home</li>
    <li>Blog</li>
    <li>About</li>
</ul>

Kullanıcı bir menü öğesini tıkladığında, ‘seçili’ sınıf tıklanan öğeye eklenmeli ve kardeşlerinden kaldırılmalıdır:

$(".menu").on("click", "li", function () {
    $(this).addClass("selected");
    $(this).siblings().removeClass("selected");
});

Yöntem, seçmek istediğiniz kardeş türlerini daraltmanız gerektiğinde kullanılabilecek isteğe bağlı bir “seçici” argümanı alır:

$(this).siblings("li").removeClass("selected");

en yakın() yöntemi #

Seçiciyle eşleşen ilk öğeyi, öğeden başlayıp DOM ağacında yukarı doğru hareket ettirerek döndürür.

HTML

<div id="abc" class="row">
    <div id="xyz" class="row">
    </div>
    <p id="origin">
      Hello
    </p>
</div>

jQuery

var target = $('#origin').closest('.row');
console.log("Closest row:", target.attr('id') );

var target2 = $('#origin').closest('p');
console.log("Closest p:", target2.attr('id') );

ÇIKTI

"Closest row: abc"
"Closest p: origin"

ilk() yöntemi : İlk yöntem, eşleşen öğeler kümesinden ilk öğeyi döndürür.

HTML

<div class='.firstExample'>
   <p>This is first paragraph in a div.</p>
   <p>This is second paragraph in a div.</p>
   <p>This is third paragraph in a div.</p>
   <p>This is fourth paragraph in a div.</p>
   <p>This is fifth paragraph in a div.</p>
</div>

JQuery

var firstParagraph = $("div p").first();
console.log("First paragraph:", firstParagraph.text());

Çıktı:

First paragraph: This is first paragraph in a div.

Sonraki öğeyi al #

Bir sonraki öğeyi almak için .next() yöntemini kullanabilirsiniz.

<ul>
    <li>Mark</li>
    <li class="anna">Anna</li>
    <li>Paul</li>
</ul>

“Anna” öğesinin üzerinde duruyorsanız ve bir sonraki öğe olan “Paul"u almak istiyorsanız, .next() yöntemi bunu yapmanıza izin verecektir.

// "Paul" now has green text
$(".anna").next().css("color", "green");

Yöntem, bir sonraki öğenin belirli bir tür öğe olması gerektiğinde kullanılabilecek isteğe bağlı bir “seçici” argümanı alır.

// Next element is a "li", "Paul" now has green text
$(".anna").next("li").css("color", "green");

Bir sonraki eleman ‘seçici’ türünde değilse, boş bir küme döndürülür ve değişiklikler hiçbir şey yapmaz.

// Next element is not a ".mark", nothing will be done in this case
$(".anna").next(".mark").css("color", "green");

Önceki öğeyi al #

Önceki öğeyi almak için .prev() yöntemini kullanabilirsiniz.

<ul>
    <li>Mark</li>
    <li class="anna">Anna</li>
    <li>Paul</li>
</ul>

“Anna” öğesinin üzerinde duruyorsanız ve önceki öğe olan “Mark"ı almak istiyorsanız, .prev() yöntemi bunu yapmanıza izin verecektir.

// "Mark" now has green text
$(".anna").prev().css("color", "green");

Yöntem, önceki öğenin belirli bir tür öğe olması gerektiğinde kullanılabilen isteğe bağlı bir “seçici” argümanı alır.

// Previous element is a "li", "Mark" now has green text
$(".anna").prev("li").css("color", "green");

Önceki eleman ‘seçici’ türünde değilse, boş bir küme döndürülür ve değişiklikler hiçbir şey yapmaz.

// Previous element is not a ".paul", nothing will be done in this case
$(".anna").prev(".paul").css("color", "green");

Bir seçimi filtrele #

Bir seçimi filtrelemek için .filter() yöntemini kullanabilirsiniz.

Yöntem bir seçimde çağrılır ve yeni bir seçim döndürür. Filtre bir öğeyle eşleşirse, döndürülen seçime eklenir, aksi takdirde yoksayılır. Hiçbir öğe eşleşmezse boş bir seçim döndürülür.

HTML #

Kullanacağımız HTML bu.

<ul>
    <li class="zero">Zero</li>
    <li class="one">One</li>
    <li class="two">Two</li>
    <li class="three">Three</li>
</ul>

seçici #

Seçicileri kullanarak filtreleme, bir seçimi filtrelemenin daha basit yollarından biridir.

$("li").filter(":even").css("color", "green"); // Color even elements green
$("li").filter(".one").css("font-weight", "bold"); // Make ".one" bold

İşlev #

Bir seçimi işlev kullanarak filtrelemek, seçicilerin kullanılması mümkün değilse yararlıdır.

İşlev, seçimdeki her öğe için çağrılır. Bir “true” değeri döndürürse, öğe döndürülen seçime eklenir.

var selection = $("li").filter(function (index, element) {
    // "index" is the position of the element
    // "element" is the same as "this"
    return $(this).hasClass("two");
});
selection.css("color", "green"); // ".two" will be colored green

Elementler #

DOM öğelerine göre filtreleyebilirsiniz. DOM öğeleri seçimdeyse, döndürülen seçime dahil edilirler.

var three = document.getElementsByClassName("three");
$("li").filter(three).css("color", "green");

seçim #

Bir seçimi başka bir seçime göre de filtreleyebilirsiniz. Bir öğe her iki seçimde de varsa, döndürülen seçime dahil edilecektir.

var elems = $(".one, .three");
$("li").filter(elems).css("color", "green");

bul() yöntemi #

.find() yöntemi, DOM ağacında bu öğelerin alt öğelerini aramamıza ve eşleşen öğelerden yeni bir jQuery nesnesi oluşturmamıza olanak tanır.

HTML

     <div class="parent">
        <div class="children" name="first">
            <ul>
                <li>A1</li>
                <li>A2</li>
                <li>A3</li>
            </ul>
        </div>
        <div class="children" name="second">
            <ul>
                <li>B1</li>
                <li>B2</li>
                <li>B3</li>
            </ul>
        </div>
       </div>

jQuery

 $('.parent').find('.children[name="second"] ul li').css('font-weight','bold');
