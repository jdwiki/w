Bir jQuery seçicileri, bir HTML belgesinde bir DOM (belge nesne modeli) öğesini seçer veya bulur. Kimlik, ad, türler, nitelikler, sınıf vb. bazında HTML öğelerini seçmek için kullanılır. Mevcut CSS seçicilerini temel alır.

Sözdizimi #

  • Etiket: İşaretleyici yok, etiketi doğrudan kullanın
  • Kimlik: “#kim”
  • Sınıf: .sınıfAdı
  • Özellik: [attributeName]
  • Değere sahip nitelik: [attributeName ='value']
  • Özellik ^= değeriyle başlar: [attributeName ^= 'değer']
  • Özellik “$= değeriyle biter: "[attributeName $='value']
  • Nitelik, *= değerini içeriyor: [attributeName *= 'value']
  • Sözde seçici: :sözde seçici
  • Herhangi bir soyundan gelen: Seçiciler arasındaki boşluk
  • Doğrudan çocuklar: > seçiciler arasında
  • İlkini izleyen komşu kardeş: `+'
  • İlkini izleyen komşu olmayan kardeş: ~
  • Veya: , ​​(virgül) seçici arasında

gibi bazı özel karakterler içeren “sınıf” veya “kimlik” veya “öznitelik” için “seçiciler” yazarken

! " # $ % & ``` ( ) * + , . / : ; < = > ? @ [ \ ] ^ { | } ~

karakterlerden iki ters eğik çizgi \\ kullanılarak kaçılması gerekir.

Örneğin.

<span id="temp.foobar"><span>

seçiciler şöyle çerçevelenecek,

$('#temp\\.foobar')

Seçici Türleri #

jQuery’de, aşağıdakiler de dahil olmak üzere, öğenin birçok çeşitli özelliğini kullanarak bir sayfadaki öğeleri seçebilirsiniz:

  • Tip
  • Sınıf
  • kimlik
  • Niteliğe Sahip Olmak
  • Özellik Değeri
  • Dizine Alınmış Seçici
  • Sözde durum

CSS seçicileri biliyorsanız, jQuery’deki seçicilerin aynı olduğunu fark edeceksiniz (küçük istisnalar dışında).

Örneğin aşağıdaki HTML’yi alın:

<a href="index.html"></a>                  <!-- 1 -->
<a id="second-link"></a>                   <!-- 2 -->
<a class="example"></a>                    <!-- 3 -->
<a class="example" href="about.html"></a>  <!-- 4 -->
<span class="example"></span>              <!-- 5 -->

Türlere Göre Seçme:

Aşağıdaki jQuery seçici, 1, 2, 3 ve 4 dahil tüm “” öğelerini seçecektir.

$("a")

Sınıfa Göre Seçme

Aşağıdaki jQuery seçici, 3, 4 ve 5 olan “example” sınıfının (a olmayan öğeler dahil) tüm öğelerini seçecektir.

$(".example")

Kimliğe göre seçme

Aşağıdaki jQuery seçici, 2 olan verilen kimliğe sahip öğeyi seçecektir.

$("#second-link")

Özellik Sahipliğine Göre Seçme

Aşağıdaki jQuery seçici, 1 ve 4 dahil olmak üzere tanımlanmış bir “href” özniteliğine sahip tüm öğeleri seçecektir.

$("[href]")

Özellik Değerine Göre Seçme

Aşağıdaki jQuery seçicisi, ‘href’ özniteliğinin bulunduğu tüm öğeleri, yalnızca 1 olan ‘index.html’ değeriyle seçecektir.

$("[href='index.html']")

İndekslenmiş Pozisyona Göre Seçim (İndekslenmiş Seçici)

Aşağıdaki jQuery seçicisi yalnızca 1’i seçecektir, ikinci <a> yani. “ikinci bağlantı”, çünkü sağlanan dizin “eq(1)” gibi “1"dir (dizinin “0"dan başladığını, dolayısıyla ikincinin burada seçildiğini unutmayın!).

$("a:eq(1)")

Dizinli Hariç Tutma ile Seçme

Bir öğeyi :not(:eq()) dizinini kullanarak hariç tutmak için

Aşağıdakiler, 1 olan “example” sınıfı hariç “” öğelerini seçer.

$("a").not(":eq(0)")

Hariç Tutarak Seçme

Bir öğeyi seçimden çıkarmak için :not() kullanın

Aşağıdaki, 1 ve 2 olan “example” sınıfına sahip olanlar hariç “” öğelerini seçer.

$("a:not(.example)")

Sözde duruma göre seçme

Ayrıca jQuery’de ‘:first-child’, ‘:last-child’, ‘:first-of-type’, ‘:last-of-type’, vb. dahil olmak üzere sözde durumları kullanarak seçim yapabilirsiniz.

Aşağıdaki jQuery seçici yalnızca ilk “” öğesini seçecektir: sayı 1.

$("a:first-of-type")

jQuery seçicilerini birleştirme

Birden çok jQuery seçiciyi birleştirerek özgünlüğünüzü de artırabilirsiniz; herhangi bir sayıda veya hepsini birleştirebilirsiniz. Aynı anda birden fazla sınıf, nitelik ve durum da seçebilirsiniz.

$("a.class1.class2.class3#someID[attr1][attr2='something'][attr3='something']:first-of-type:first-child")

Bu, aşağıdaki özelliklere sahip bir “” öğesini seçer:

  • Şu sınıflara sahiptir: ‘class1, class2 ve class3’
  • Şu kimliğe sahip: “someID”
  • Şu Özelliğe sahiptir: “attr1”
  • Şu Niteliklere ve değerlere sahiptir: “bir şey” değerine sahip “attr2”, “bir şey” değerine sahip “attr3”
  • Şu durumlara sahiptir: “birinci çocuk” ve “türünün ilk örneği”

Ayrıca farklı seçicileri virgülle ayırabilirsiniz:

$("a, .class1, #someID")

Bu şunları seçer:

  • Tüm “” öğeleri
  • ‘class1’ sınıfına sahip tüm öğeler
  • “#someID” kimliğine sahip bir öğe

Çocuk ve Kardeş seçimi

jQuery seçicileri, genellikle çocukları ve kardeşleri aynı şekilde seçmenize olanak tanıyan CSS ile aynı kurallara uyar.

  • Doğrudan olmayan bir çocuk seçmek için bir boşluk kullanın
  • Doğrudan bir çocuk seçmek için bir > kullanın
  • İlkinden sonra komşu bir kardeş seçmek için bir “+” kullanın
  • İlkinden sonra komşu olmayan bir kardeş seçmek için bir ~ kullanın

Joker karakter seçimi

Tüm öğeleri seçmek istediğimiz durumlar olabilir, ancak seçilecek ortak bir özellik yoktur (sınıf, öznitelik vb.). Bu durumda, tüm öğeleri basitçe seçen * seçicisini kullanabiliriz:

$('#wrapper *')    // Select all elements inside #wrapper element

Genel Bakış #

Öğeler, jQuery tarafından jQuery Selectors kullanılarak seçilebilir. İşlev, bir öğe veya bir öğe listesi döndürür.

Temel seçiciler #

$("*")                       // All elements
$("div")                     // All <div> elements
$(".blue")                   // All elements with class=blue
$(".blue.red")               // All elements with class=blue AND class=red
$(".blue,.red")              // All elements with class=blue OR class=red
$("#headline")               // The (first) element with id=headline
$("[href]")                  // All elements with an href attribute
$("[href='example.com']")    // All elements with href=example.com

İlişkisel operatörler #

$("div span")          // All <span>s that are descendants of a <div>
$("div > span")        // All <span>s that are a direct child of a <div>
$("a ~ span")          // All <span>s that are siblings following an <a>
$("a + span")          // All <span>s that are immediately after an <a>

Önbelleğe Alma Seçicileri #

jQuery’de bir seçiciyi her kullandığınızda, DOM, sorgunuzla eşleşen öğeler için aranır. Bunu çok sık veya tekrar tekrar yapmak performansı düşürür. Belirli bir seçiciye birden fazla başvurursanız, onu bir değişkene atayarak önbelleğe eklemelisiniz:

var nav = $('#navigation');
nav.show();

Bunun yerine:

$('#navigation').show();

Web sitenizin bu öğeyi sık sık göstermesi/gizlemesi gerekiyorsa, bu seçiciyi önbelleğe almak yardımcı olabilir. Aynı seçiciye sahip birden fazla öğe varsa, değişken şu öğelerin bir dizisi olur:

<div class="parent">
    <div class="child">Child 1</div>
    <div class="child">Child 2</div>
</div>

<script>
    var children = $('.child');
    var firstChildText = children[0].text();
    console.log(firstChildText); 
    
    // output: "Child 1"
</script>

NOT: Öğe, bir değişkene atandığı sırada DOM’de bulunmalıdır. DOM’da “child” adında bir sınıfa sahip bir öğe yoksa, o değişkende boş bir dizi depolayacaksınız.

<div class="parent"></div>

<script>
    var parent   = $('.parent');
    var children = $('.child');
    console.log(children);

    // output: []

    parent.append('<div class="child">Child 1</div>');
    children = $('.child');
    console.log(children[0].text());

    // output: "Child 1"
</script>

Bu seçiciyle DOM’ye öğe ekledikten/kaldırdıktan sonra seçiciyi değişkene yeniden atamayı unutmayın.

Not: Seçicileri önbelleğe alırken, birçok geliştirici değişkenin adını şu şekilde bir jQuery nesnesi olduğunu belirtmek için bir “$” ile başlatır:

var $nav = $('#navigation');
$nav.show();

Seçicileri birleştirme #

DOM Yapısını takip etmeyi düşünün

<ul class="parentUl">
    <li> Level 1
        <ul class="childUl">
            <li>Level 1-1 <span> Item - 1 </span></li>
            <li>Level 1-1 <span> Item - 2 </span></li>
        </ul>
    </li>
    <li> Level 2
        <ul class="childUl">
            <li>Level 2-1 <span> Item - 1 </span></li>
            <li>Level 2-1 <span> Item - 1 </span></li>
        </ul>
    </li>
</ul>

Alt ve alt seçiciler #

Bir ebeveyne <ul> - parentUl verildiğinde, onun soyundan gelenleri (<li>) bulun,

  1. Basit $('ebeveyn çocuk')

>> $('ul.parentUl li')

Bu, belirtilen atanın tüm eşleşen torunlarını tüm seviyeler aşağı alır.

  1. > - $('ebeveyn > çocuk')

>> $('ul.parentUl > li')

Bu, eşleşen tüm çocukları bulur (yalnızca 1. seviye aşağı).

  1. Bağlam tabanlı seçici - $('child','parent')

>> $('li','ul.parentUl')

Bu, yukarıdaki 1. ile aynı şekilde çalışır.

  1. find() - $('parent').find('child')

>> $('ul.parentUl').find('li')

Bu, yukarıdaki 1. ile aynı şekilde çalışır.

  1. çocuklar() - $('ebeveyn').find('çocuk')

>> $('ul.parentUl').children('li')

Bu, yukarıdaki 2. ile aynı şekilde çalışır.

Diğer birleştiriciler #

Grup Seçici : “,” #

Tüm “

    ” öğelerini VE tüm “
  • ” öğelerini VE tüm “” öğelerini seçin:

    $('ul, li, span')

    Çoklu seçici : "” (karaktersiz) #

    “parentUl” sınıfına sahip tüm “

      ” öğelerini seçin:

      $('ul.parentUl')

      Komşu Kardeş Seçici : “+” #

      Başka bir “

    • ” öğesinin hemen arkasına yerleştirilen tüm “
    • ” öğelerini seçin:

      $('li + li')

      Genel Kardeş Seçici : “~” #

      Diğer “

    • ” öğelerinin kardeşleri olan tüm “
    • ” öğelerini seçin:

      $('li ~ li')

      Seçici olarak DOM Öğeleri #

      jQuery çok çeşitli parametreleri kabul eder ve bunlardan biri gerçek bir DOM öğesidir. Bir DOM öğesini jQuery’ye geçirmek, [jQuery nesnesinin](https://www.wikiod.com/tr/jquery/jqueryyi-kullanmaya-baslama#jQuery Nesnesi) altında yatan dizi benzeri yapıya neden olur ) bu öğeyi tutmak için.

      jQuery, nodeType’ı inceleyerek bağımsız değişkenin bir DOM öğesi olduğunu algılayacaktır.

      Bir DOM öğesinin en yaygın kullanımı, jQuery API’sine erişim elde etmek için mevcut öğenin jQuery yapıcısına iletildiği geri aramalardadır.

      “each” geri aramasında olduğu gibi (not: her biri bir yineleyici işlevidir).

      $(".elements").each(function(){
    //the current element is bound to `this` internally by jQuery when using each
    var currentElement = this;

    //at this point, currentElement (or this) has access to the Native API
    
    //construct a jQuery object with the currentElement(this)
    var $currentElement = $(this);

    //now $currentElement has access to the jQuery API
});

      seçici olarak HTML dizeleri #

      jQuery, çok çeşitli parametreleri “seçici” olarak kabul eder ve bunlardan biri bir HTML dizesidir. Bir HTML dizesini jQuery’ye iletmek, [jQuery nesnesinin](https://www.wikiod.com/tr/jquery/jqueryyi-kullanmaya-baslama#jQuery Nesnesi) altında yatan dizi benzeri yapıya neden olur ) elde edilen oluşturulmuş HTML’yi tutmak için.

      jQuery, yapıcıya iletilen dizenin bir HTML dizesi olup olmadığını ve ayrıca “<” ile başlaması gerektiğini belirlemek için regex kullanır. Bu normal ifade rquickExpr = /^(?:\s*(<[\w\W]+>)[^>]*|#([\w-]*))$/ olarak tanımlanır (açıklama regex101.com).

      Bir HTML dizesinin seçici olarak en yaygın kullanımı, DOM öğeleri kümelerinin yalnızca kodda oluşturulması gerektiğidir, bu genellikle kitaplıklar tarafından Modal açılır pencereler gibi şeyler için kullanılır.

      Örneğin, bir div içine sarılmış bir bağlantı etiketini şablon olarak döndüren bir işlev

      function template(href,text){
    return $("<div><a href='" + href + "'>" + text + "</a></div>");
}

      Tutarak bir jQuery nesnesi döndürürdü

      <div>
    <a href="google.com">Google</a>
</div>

      template("google.com","Google") olarak adlandırılırsa.
