DOM öğeleri oluşturma #

“jQuery” işlevi (genellikle “$” olarak adlandırılır) hem öğeleri seçmek hem de yeni öğeler oluşturmak için kullanılabilir.

var myLink = $('<a href="http://stackexchange.com"></a>');

İsteğe bağlı olarak, öğe niteliklerine sahip ikinci bir argüman iletebilirsiniz:

var myLink = $('<a>', { 'href': 'http://stackexchange.com' });

`’’’ –> İlk argüman, oluşturmak istediğiniz DOM öğesinin türünü belirtir. Bu örnekte bu bir anchor ama bu listedeki herhangi bir şey olabilir. ‘a’ öğesinin referansı için şartname‘ye bakın.

{ 'href': 'http://stackexchange.com' } –> ikinci argüman, öznitelik adı/değer çiftlerini içeren bir JavaScript Nesnesi.

‘ad’:‘değer’ çiftleri, ilk argümanın ‘<’ ‘>’ arasında görünecektir, örneğin ‘<a ad:değer>’, bizim örneğimiz için ‘`

Öğe sınıflarını değiştirme #

Sayfanın aşağıdaki gibi bir HTML öğesi içerdiğini varsayarsak:

<p class="small-paragraph">
  This is a small <a href="https://en.wikipedia.org/wiki/Paragraph">paragraph</a>
  with a <a class="trusted" href="http://stackexchange.com">link</a> inside.
</p>

jQuery, DOM sınıflarını işlemek için yararlı işlevler sağlar, özellikle ‘hasClass()’, ‘addClass()’, ‘removeClass()’ ve ’toggleClass()’. Bu işlevler, eşleşen öğelerin “sınıf” özniteliğini doğrudan değiştirir.

$('p').hasClass('small-paragraph'); // true
$('p').hasClass('large-paragraph'); // false

// Add a class to all links within paragraphs
$('p a').addClass('untrusted-link-in-paragraph');

// Remove the class from a.trusted
$('a.trusted.untrusted-link-in-paragraph')
.removeClass('untrusted-link-in-paragraph')
.addClass('trusted-link-in-paragraph');

Bir sınıfı değiştir

Örnek işaretleme göz önüne alındığında, ilk .toggleClass() ile bir sınıf ekleyebiliriz:

$(".small-paragraph").toggleClass("pretty");

Şimdi bu “true” değerini döndürür: $(".small-paragraph").hasClass("güzel")

“toggleClass”, aşağıdaki gibi daha az kodla aynı etkiyi sağlar:

if($(".small-paragraph").hasClass("pretty")){
   $(".small-paragraph").removeClass("pretty");}
else {
   $(".small-paragraph").addClass("pretty"); }

İki sınıf arasında geçiş yap:

$(".small-paragraph").toggleClass("pretty cool");

Sınıf eklemek/kaldırmak için Boolean:

$(".small-paragraph").toggleClass("pretty",true); //cannot be truthy/falsey

$(".small-paragraph").toggleClass("pretty",false);

Sınıf geçişi işlevi (bir sorunu önlemek için aşağıdaki örneğe bakın)

$( "div.surface" ).toggleClass(function() {
  if ( $( this ).parent().is( ".water" ) ) {
    return "wet";
  } else {
    return "dry";
  }
});

Örneklerde kullanılır:

// functions to use in examples
function stringContains(myString, mySubString) {
  return myString.indexOf(mySubString) !== -1;
}
function isOdd(num) { return num % 2;}
var showClass = true; //we want to add the class

Örnekler:

Tek/çift sınıfları arasında geçiş yapmak için öğe dizinini kullanın

$( "div.gridrow" ).toggleClass(function(index,oldClasses, false), showClass ) {
  showClass
  if ( isOdd(index) ) {
    return "wet";
  } else {
    return "dry";
  }
});

Basit bir ızgara işaretlemesi verilen daha karmaşık “toggleClass” örneği

<div class="grid">
  <div class="gridrow">row</div>
  <div class="gridrow">row</div>
  <div class="gridrow">row</div>
  <div class="gridrow">row</div>
  <div class="gridrow">row</div>
  <div class="gridrow gridfooter">row but I am footer!</div>
</div>

Örneklerimiz için basit fonksiyonlar:

function isOdd(num) {
  return num % 2;
}

function stringContains(myString, mySubString) {
  return myString.indexOf(mySubString) !== -1;
}
var showClass = true; //we want to add the class

“Gridrow” sınıfına sahip öğelere tek/çift bir sınıf ekleyin

$("div.gridrow").toggleClass(function(index, oldClasses, showThisClass) {
  if (isOdd(index)) {
    return "odd";
  } else {
    return "even";
  }
  return oldClasses;
}, showClass);

Satırda “gridfooter” sınıfı varsa, tek/çift sınıfları kaldırın, gerisini saklayın.

$("div.gridrow").toggleClass(function(index, oldClasses, showThisClass) {
  var isFooter = stringContains(oldClasses, "gridfooter");
  if (isFooter) {
    oldClasses = oldClasses.replace('even', ' ').replace('odd', ' ');
    $(this).toggleClass("even odd", false);
  }
  return oldClasses;
}, showClass);

Döndürülen sınıflar, etkilenenlerdir. Burada, bir elemanın “ızgara altbilgisi” yoksa, çift/tek için bir sınıf ekleyin. Bu örnek, OLD sınıf listesinin geri dönüşünü göstermektedir. Eğer bu ’else return oldClasses;’ kaldırılırsa, yalnızca yeni sınıflar eklenir, bu nedenle ‘gridfooter’ sınıfına sahip satır, eğer biz o eskileri döndürmemiş olsaydık tüm sınıfları kaldırmış olurdu - aksi takdirde bunlar değiştirilirdi (kaldırılırdı).

$("div.gridrow").toggleClass(function(index, oldClasses, showThisClass) {
  var isFooter = stringContains(oldClasses, "gridfooter");
  if (!isFooter) {
    if (isOdd(index)) {
      return "oddLight";
    } else {
      return "evenLight";
    }
  } else return oldClasses;
}, showClass);

Diğer API Yöntemleri #

jQuery, DOM manipülasyonu için kullanılabilecek çeşitli yöntemler sunar.

Birincisi .empty() yöntemidir.

Aşağıdaki işaretlemeyi hayal edin:

<div id="content">
  <div>Some text</div>
</div>

$('#content').empty(); çağrıldığında, iç div kaldırılır. Bu, $('#content').html(''); kullanılarak da gerçekleştirilebilir.

Bir başka kullanışlı işlev de .closest() işlevidir:

<tr id="row_1">
  <td><button type="button" class="delete">Delete</button>
</tr>

Satır hücrelerinden birinde tıklanan bir düğmeye en yakın satırı bulmak istiyorsanız, bunu yapabilirsiniz:

$('.delete').click(function() {
  $(this).closest('tr');
});

Muhtemelen her biri kendi ‘sil’ düğmelerine sahip birden fazla satır olacağından, kapsamı gerçekten tıkladığımız düğmeyle sınırlamak için .click() işlevi içinde ‘$(this)’ kullanırız.

Tıkladığınız ‘Sil’ düğmesini içeren satırın ‘id’sini almak istiyorsanız, şöyle bir şey yapabilirsiniz:

$('.delete').click(function() {
  var $row = $(this).closest('tr');
  var id = $row.attr('id');
});

Değişkenin ne olduğunu netleştirmek için jQuery nesnelerini içeren değişkenlere bir “$” (dolar işareti) eklemek genellikle iyi bir uygulama olarak kabul edilir.

.closest() yöntemine bir alternatif, .parents() yöntemidir:

$('.delete').click(function() {
  var $row = $(this).parents('tr');
  var id = $row.attr('id');
});

ve ayrıca bir .parent() işlevi de vardır:

$('.delete').click(function() {
  var $row = $(this).parent().parent();
  var id = $row.attr('id');
});

.parent() DOM ağacının yalnızca bir düzeyine çıkar, bu nedenle oldukça esnek değildir, örneğin bir span içinde yer alacak silme düğmesini değiştirirseniz, jQuery seçici bozulur.

Öğeleri sıralama #

Öğeleri verimli bir şekilde (tümü aynı anda ve minimum DOM kesintisi ile) sıralamak için şunları yapmamız gerekir:

  1. Öğeleri bul

  2. Sıralama, belirli bir koşula göre

  3. DOM’a Ekle

    • Red
    • Green
    • Purple
    • Orange

  4. Onları bulun - .children() veya .find()

Bu bize oynamak için Array benzeri bir nesneyi geri verecektir.

   var $myColorList = $('#my-color-list');

   // Elements one layer deep get .children(), any deeper go with .find()
   var $colors = $myColorList.children('li');
  1. Bunları yeniden düzenleyin - Array.prototype.sort()

Bu, şu anda öğeleri HTML içeriğine (diğer bir deyişle renklerine) göre Artan düzende döndürmek için ayarlanmıştır.

   /**
    * Bind $colors to the sort method so we don't have to travel up
    * all these properties more than once.
    */
   var sortList = Array.prototype.sort.bind($colors);

   sortList(function ( a, b ) {

       // Cache inner content from the first element (a) and the next sibling (b)
       var aText = a.innerHTML;
       var bText = b.innerHTML;
    
       // Returning -1 will place element `a` before element `b`
       if ( aText < bText ) {
           return -1;
       }

       // Returning 1 will do the opposite
       if ( aText > bText ) {
           return 1;
       }

       // Returning 0 leaves them as-is
       return 0;
   });

  1. Bunları ekleyin - .append()

    Note that we don’t need to detach the elements first - append() will move elements that already exist in the DOM, so we won’t have extra copies

    // Put it right back where we got it
$myColorList.append($colors);

Sıralama yönünün başlangıç ​​değerini ayarlayın #

var ascending = true;

DOM işlememizi en aza indirmek için DOM öğelerimizi ve sortList()i buradan önbelleğe alın

var $myColorList = $('#my-color-list');
var $colors = $myColorList.children('li');
var sortList = Array.prototype.sort.bind($colors);

Her şeyi bir doSort() işlevine sarın #

// Put the sortList() and detach/append calls in this portable little thing
var doSort = function ( ascending ) {

    sortList(function ( a, b ) {
        // ...
    });

    $myColorList.append($colors);
};

$('#btn-sort') için tıklama işleyicisi ekleyin #

$('#btn-sort').on('click', function () {
    // Run the sort and pass in the direction value
    doSort(ascending);

    // Toggle direction and save
    ascending = !ascending;
});

Şimdi hep beraber #

var ascending = true;

var $myColorList = $('#my-color-list');
var $colors = $myColorList.children('li');
var sortList = Array.prototype.sort.bind($colors);

var doSort = function ( ascending ) {
    
    sortList(function ( a, b ) {

        var aText = a.innerHTML;
        var bText = b.innerHTML;

        if ( aText < bText ) {
            return ascending ? -1 : 1;
        }

        if ( aText > bText ) {
            return ascending ? 1 : -1;
        }

        return 0;
    });
    
    $myColorList.append($colors);

};

$('#btn-sort').on('click', function () {
    doSort(ascending);
    ascending = !ascending;
});

Çok seviyeli sıralama (sıralanmış öğeleri gruplama) #

// ...

var doSort = function ( ascending ) {

    sortList(function ( a, b ) {

        // ...initial sorting...

    }).sort(function ( a, b ) {
        
        // We take the returned items and sort them once more
        var aClass = a.className;
        var bClass = b.className;
        
        // Let's group the disabled items together and put them at the end

        /**
         * If the two elements being compared have the same class
         * then there's no need to move anything.
         */
        if ( aClass !== bClass ) {
            return aClass === 'disabled' ? 1 : -1;
        }
        return 0;
    });

    // And finalize with re-insert
    $myColorList.append($colors);
};

// ...

.html() #

Seçici içindeki tüm HTML’yi değiştirmek için bu yöntemi kullanabilirsiniz. Bunun gibi bir html öğeniz olduğunu varsayarsak

<div class="row">
    <div class="col-md-12">
        <div id="information">
            <p>Old message</p>
        </div>
    </div>
</div>

.html() kullanabilirsiniz. Kullanıcıları tek bir satırla uyarmak için bir uyarı veya bilgi metni kaldırmak ve eklemek için.

$("#information").html("<p>This is the new alert!</p>");
