Sözdizimi #

  • çevir(kayan x, kayan y, kayan z)
  • döndürmeX (kayan açı)
  • döndürY(kayan açı)
  • döndürZ(yüzer açı)
  • kutu (şamandıra boyutu)
  • kutu(yüzer w, kayan h, kayan d)

Parametreler #

parametreler Ayrıntılar
açı açı radyan cinsindendir
boyut tüm boyutları için kullanılacak kutunun boyutu
w “x ekseni"ndeki kutunun boyutu
h “y ekseni"ndeki kutunun boyutu
gün “z ekseni"ndeki kutunun boyutu

3D Çeviri #

Nesneleri P3D’de nasıl çevireceğiniz aşağıda açıklanmıştır:

size(200, 200, P3D); //Starting P3D renderer
fill(255, 0, 0, 150); //transparent red
rect(10, 10, 100, 100); //first rectangle
fill(0, 0, 255, 150); //transparent blue
translate(50, 50, 50); //translate x, y and z by 50 pixels
rect(0, 0, 100, 100); //second rectangle (same dimensions as the first one)

Kırmızı: ilk dikdörtgen Mavi: ikinci dikdörtgen

Yukarıdaki çizimden görülebileceği gibi, ikinci dikdörtgen yalnızca birinciden daha büyük görünüyor, gerçekte dikdörtgenin “50” pikselinin “z ekseni boyunca” çevrilmesinin bir sonucu olarak ekrana “daha yakın” olduğu zaman. ’ (ve elbette, dikdörtgen ‘x’ ve ‘y’ eksenleri boyunca çevrilmiştir).

3D Döndürme #

3B döndürme için üç işlev vardır: “açı"nın radyan cinsinden olduğu ilgili eksenlerinde döndürme için “döndürX(açı)”, “döndürY(açı)” ve “döndürZ(açı)”.

size(200, 200, P3D); //Starting P3D renderer
fill(255, 0, 0, 150); //transparent red
translate(width/2, height/2);//translate to centre, ie (100, 100)
rectMode(CENTER);//This makes the rectangle centre in (100, 100)
rect(0, 0, 100, 100); //first rectangle
fill(0, 0, 255, 150); //transparent blue
rotateX(PI/4); //rotate in the x-axis by PI/4 radians (45 degrees)
rect(0, 0, 100, 100); //second rectangle (same dimensions as the first one)


rotateY(radians(45)); //rotate in the y-axis by passing the radians conversion of 45 degrees


rotateZ(3*PI/4); //rotate in the z-axis by 3*PI/4 radians (270 degrees)

Not: dönüşümler (çevirmeler ve döndürmeler gibi) önceki dönüşüme eklenir.

Küboid çizme #

Bir küboid çizmek için, boyutlarını parametre olarak vererek box() fonksiyonunu kullanmanız gerekir.

size(200, 200, P3D); //Starting the P3D renderer
translate(width/2, height/2); //Translating to the centre of the sketch
rotateY(PI/4); //rotate so that...
rotateX(PI/6); //... it will be easy to see the box
noFill(); //disabling the box's fill, so that we will be able to see its edges
box(100, 50, 75); //the box function requires its dimensions as its parameters

box() fonksiyonunun parametre olarak konumunu kabul etmediğini* unutmayın.

Ayrıca sadece bir parametre ile box() fonksiyonunu çağırmanın bir yolu vardır. Bu durumda, bir küp olacaktır.

stroke(0, 100, 255); //change the edges' colour
fill(0, 0, 255); //fill the `box` in a blue colour
box(100); //draw a cube
