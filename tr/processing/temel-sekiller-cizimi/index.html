İşleme’de, şekil çizmek programın anahtarıdır. Aksi takdirde ekranda hiçbir şey görünmez. Bu bölüm size temel şekillerin nasıl çizildiğini anlatacaktır.

Sözdizimi #

  • satır(kayan x1, kayan y1, kayan x2, kayan y2)
  • satır(kayan x1, kayan y1, kayan z1, kayan x2, kayan y2, kayan z2)
  • elips(kayan x, kayan y, kayan w, kayan h)
  • rect(kayan x, kayan y, kayan w, kayan h)
  • üçgen(kayan x1, kayan y1, kayan x2, kayan y2, kayan x3, kayan y3)

Parametreler #

parametre Ayrıntılar
x1 ilk noktanın x koordinatı
y1 ilk noktanın y koordinatı
z1 ilk noktanın z koordinatı
x2 ikinci noktanın x koordinatı
y2 ikinci noktanın y koordinatı
z2 ikinci noktanın z koordinatı
x3 üçüncü noktanın x koordinatı
y3 üçüncü noktanın y koordinatı
x x koordinatı
ve y koordinatı
w genişlik
h yükseklik

Processing’in temeli hakkında bir referans bulabilirsiniz.

İşleniyor ana sayfası

Çizgi Çizmek #

İşleme, ekranda bir çizgi çizmek için line() adlı bir yöntem sağlar. Bu kod siyah zemin üzerine 10 piksellik beyaz bir çizgi çizer.

void setup() {
    size(500, 500);
    background(0);
    stroke(255);
    strokeWeight(10);
}

void draw() {
    line(0, 0, 500, 500);
}

’line()’ yönteminin imzası şudur.

line(x1, y1, x2, y2);

“x1” ve “y1”, başlangıç ​​noktasının bir koordinatıdır. “x2” ve “y2”, bitiş noktasının bir koordinatıdır.

Çizeceğiniz çizginin rengini belirtmek için stroke() metodu kullanılır.

Çizeceğiniz çizginin kalınlığını belirtmek için strokeWeight() metodu kullanılır. (piksel olarak)

Dikdörtgen Çizmek #

İşleme, bir dikdörtgen çizmek için “rect()” yöntemini sağlar. Bu kod, siyah zemin üzerine beyaz bir 50 X 50 dikdörtgen çizer.

void setup() {
    size(500, 500);
    background(0);
    fill(255);
    noStroke();
}

void draw() {
    rect(225, 225, 50, 50);
}

rect() metodunun imzası şudur.

rect(x, y, w, h);

“x” ve “y” dikdörtgenin koordinatıdır. “w” ve “h” dikdörtgenin genişliği ve yüksekliğidir.

“fill()” yöntemi, dikdörtgenin ve elips, üçgen, çokgen gibi diğer şekillerin dolgu rengini belirtmek için kullanılır.

Dikdörtgenin çevresinde hiçbir kontur olmadığını belirtmek için “noStroke()” yöntemi kullanılır. Bu yöntem elips, üçgen, çokgen gibi diğer şekilleri de etkiler.

Elips Çizimi #

İşleme, elips çizmek için ’elips’ yöntemini sağlar. Bu kod, yarıçapı 25 piksel olan beyaz bir daire çizer.

void setup() {
    size(500, 500);
    background(0);
    fill(255);
    noStroke();
}

void draw() {
    ellipse(225, 225, 50, 50);
}

’ellipse()’ yönteminin imzası şudur.

ellipse(x, y, w, h);

“x” ve “y” elipsin koordinatıdır. “w” ve “h” elipsin genişliği ve yüksekliğidir.

Üçgen Çizmek #

İşleme, bir üçgen çizmek için “üçgen” yöntemini sağlar. Aşağıdaki kod, her bir tanımlayıcı nokta arasına yaklaşık 25 piksellik bir eşkenar üçgen çizer.

void setup() {
    size(500, 500);
    background(0);
}
void draw() {
    triangle(0, 0, 25, 0, 12, 12);
}

“Üçgen"in imzası şu şekildedir:

triangle(x1, y1, x2, y2, x3, y3);

Her “x” noktası noktanın x eksenine ve “y” y eksenine karşılık gelir. Üç nokta bir üçgen oluşturacak şekilde birleştirilecektir.

Üçgen Çizmek #

İşleme, bir üçgen çizmek için triangle() yöntemini sağlar. Bu kod, siyah zemin üzerine beyaz bir üçgen çizer.

void setup() {
    size(500, 500);
    background(0);
    fill(255);
    noStroke();
}

void draw() {
    triangle(250, 225, 225, 275, 275, 275);
}
