Birkaç örnekle kıvrımlara giriş #

Kıvrımlar, öğe dizileriyle birlikte kullanılan (üst düzey) işlevlerdir. b herhangi bir tür olduğunda seq<'a> b içine daraltırlar (muhtemelen hala 'a). Bu biraz soyut yani somut pratik örneklere geçelim.

Tüm sayıların toplamını hesaplama #

Bu örnekte 'a bir intdir. Bir sayı listemiz var ve tüm sayıların toplamını hesaplamak istiyoruz. Listedeki sayıları toplamak için [1; 2; 3] yazıyoruz

List.fold (fun x y -> x + y) 0 [1; 2; 3] // val it : int = 6

Açıklayayım, listelerle uğraştığımız için ‘List’ modülünde ‘fold’ kullanıyoruz, dolayısıyla ‘List.fold’. ‘katlama’nın aldığı ilk argüman ikili bir işlevdir, klasör. İkinci argüman başlangıç ​​değeridir. ‘katlama’, ilk değerden ve ilk öğeden başlayarak listedeki öğelere klasör işlevini arka arkaya uygulayarak listeyi katlamaya başlar. Liste boşsa, başlangıç ​​değeri döndürülür!

Bir yürütme örneğinin şematik genel görünümü şöyle görünür:

let add x y = x + y // this is our folder (a binary function, takes two arguments)
List.fold add 0 [1; 2; 3;]
=> List.fold add (add 0 1) [2; 3] 
// the binary function is passed again as the folder in the first argument
// the initial value is updated to add 0 1 = 1 in the second argument
// the tail of the list (all elements except the first one) is passed in the third argument
// it becomes this:
List.fold add 1 [2; 3]
// Repeat untill the list is empty -> then return the "inital" value
List.fold add (add 1 2) [3]
List.fold add 3 [3] // add 1 2 = 3
List.fold add (add 3 3) [] 
List.fold add 6 [] // the list is empty now -> return 6
6

‘List.sum’ işlevi kabaca ‘List.fold add LanguagePrimitives.GenericZero’ işlevidir, burada genel sıfır tamsayılar, kayan noktalar, büyük tamsayılar vb. ile uyumlu hale gelir.

Listedeki öğeleri sayma (count uygulaması) #

Bu, hemen hemen yukarıdakiyle aynı şekilde yapılır, ancak listedeki öğenin gerçek değerini yok sayarak ve bunun yerine 1 ekleyerek yapılır.

List.fold (fun x y -> x + 1) 0 [1; 2; 3] // val it : int = 3

Bu şu şekilde de yapılabilir:

[1; 2; 3]
|> List.map (fun x -> 1) // turn every elemet into 1, [1; 2; 3] becomes [1; 1; 1]
|> List.sum // sum [1; 1; 1] is 3

Böylece “count"u şu şekilde tanımlayabilirsiniz:

let count xs = 
    xs 
    |> List.map (fun x -> 1) 
    |> List.fold (+) 0 // or List.sum

Maksimum listeyi bulma #

Bu sefer, karşılaştırdığımız değerlerin türünün ne olduğunu bilmediğimiz bu durumda olduğu gibi, ‘List.fold’a benzeyen ancak bir başlangıç ​​değeri olmayan ‘List.reduce’ kullanacağız:

let max x y = if x > y then x else y
// val max : x:'a -> y: 'a -> 'a when 'a : comparison, so only for types that we can compare
List.reduce max [1; 2; 3; 4; 5] // 5
List.reduce max ["a"; "b"; "c"] // "c", because "c" > "b" > "a"
List.reduce max [true; false] // true, because true > false

Bir listenin minimumunu bulma #

Tıpkı maksimumu bulurken olduğu gibi, klasör farklıdır

let min x y = if x < y then x else y
List.reduce min [1; 2; 3; 4; 5] // 1
List.reduce min ["a"; "b"; "c"] // "a"
List.reduce min [true; false] // false

Birleştirici listeler #

İşte listelerin listesini alıyoruz Klasör işlevi @ operatörüdür

// [1;2] @ [3; 4] = [1; 2; 3; 4]
let merge xs ys = xs @ ys
List.fold merge [] [[1;2;3]; [4;5;6]; [7;8;9]] // [1;2;3;4;5;6;7;8;9]

Veya klasör işleviniz olarak ikili operatörleri kullanabilirsiniz:

List.fold (@) [] [[1;2;3]; [4;5;6]; [7;8;9]] // [1;2;3;4;5;6;7;8;9]
List.fold (+) 0 [1; 2; 3] // 6
List.fold (||) false [true; false] // true, more on this below
List.fold (&&) true [true; false] // false, more on this below
// etc...

Bir sayının faktöriyelini hesaplama #

Sayıları toplamakla aynı fikir, ama şimdi onları çarpıyoruz. n faktöriyelini istiyorsak [1 .. n] listesindeki tüm elemanları çarparız. Kod şöyle olur:

// the folder
let times x y = x * y
let factorial n = List.fold times 1 [1 .. n]
// Or maybe for big integers
let factorial n = List.fold times 1I [1I .. n]

‘forall’, ‘vardır’ ve ‘içerir’ uygulama #

“forall” işlevi, bir dizideki tüm öğelerin bir koşulu karşılayıp karşılamadığını kontrol eder. “var”, listedeki en az bir öğenin koşulu karşılayıp karşılamadığını kontrol eder. İlk önce bir “bool” değerleri listesini nasıl daraltacağımızı bilmemiz gerekiyor. Eh, elbette kıvrımlar kullanıyoruz! boolean operatörleri klasör fonksiyonlarımız olacak.

Bir listedeki tüm öğelerin ’true’ olup olmadığını kontrol etmek için, onları ‘&&’ işleviyle ’true’ başlangıç ​​değeri olarak daraltırız.

List.fold (&&) true [true; true; true] // true
List.fold (&&) true [] // true, empty list -> return inital value
List.fold (&&) true [false; true] // false

Benzer şekilde, bir liste booleanında bir elemanın “true” olup olmadığını kontrol etmek için, onu “||” operatörüyle “false” başlangıç ​​değeri olarak daraltırız:

List.fold (||) false [true; false] // true
List.fold (||) false [false; false] // false, all are false, no element is true
List.fold (||) false [] // false, empty list -> return inital value

“Forall” ve “var” a geri dön. Burada herhangi bir türün bir listesini alıyoruz, tüm öğeleri boole değerlerine dönüştürmek için koşulu kullanıyoruz ve sonra onu daraltıyoruz:

let forall condition elements = 
    elements 
    |> List.map condition // condition : 'a -> bool
    |> List.fold (&&) true

let exists condition elements = 
    elements
    |> List.map condition
    |> List.fold (||) false

[1’deki tüm öğelerin olup olmadığını kontrol etmek için; 2; 3; 4] 5’ten küçüktür:

forall (fun n -> n < 5) [1 .. 4] // true

“var” ile “içerir” yöntemini tanımlayın:

let contains x xs = exists (fun y -> y = x) xs

Ya da

let contains x xs = exists ((=) x) xs

Şimdi [1 .. 5] listesinin 2 değerini içerip içermediğini kontrol edin:

contains 2 [1..5] // true

“ters” uygulama: #

let reverse xs = List.fold (fun acc x -> x :: acc) [] xs
reverse [1 .. 5] // [5; 4; 3; 2; 1]

‘harita’ ve ‘filtre’ uygulama #

let map f = List.fold (fun acc x -> List.append acc [f x]) List.empty
// map (fun x -> x * x) [1..5] -> [1; 4; 9; 16; 25]   

let filter p = Seq.fold (fun acc x -> seq { yield! acc; if p(x) then yield x }) Seq.empty
// filter (fun x -> x % 2 = 0) [1..10] -> [2; 4; 6; 8; 10]

Bir listenin tüm öğelerinin toplamını hesaplama #

Bir sayı listesinin terimlerinin toplamını (şamandıra, int veya büyük tamsayı türünden) hesaplamak için List.sum kullanılması tercih edilir. Diğer durumlarda List.fold, böyle bir toplamı hesaplamak için en uygun fonksiyondur.

  1. Karmaşık sayıların toplamı

Bu örnekte, karmaşık sayıların bir listesini bildiriyoruz ve listedeki tüm terimlerin toplamını hesaplıyoruz.

Programın başında System.Numerics’e bir referans ekleyin.

System.Numerics’i açın

Toplamı hesaplamak için akümülatörü karmaşık sayı 0 ile başlatırız.

let clist = [new Complex(1.0, 52.0); new Complex(2.0, -2.0); new Complex(0.0, 1.0)]

let sum = List.fold (+) (new Complex(0.0, 0.0)) clist

Sonuç:

(3, 51)
  1. Birleşim türü sayılarının toplamı

Bir listenin birleşim (float veya int) türündeki sayılardan oluştuğunu ve bu sayıların toplamını hesaplamak istediğini varsayalım.

Aşağıdaki sayı türünden önce bildirin:

type number = 
| Float of float
| Int of int

Bir listenin tip numarasının sayılarının toplamını hesaplayın:

let list = [Float(1.3); Int(2); Float(10.2)]

let sum = List.fold (
                         fun acc elem -> 
                                        match elem with
                                        | Float(elem) -> acc + elem
                                        | Int(elem) -> acc + float(elem)
                        ) 0.0 list

Sonuç:

13.5

Fonksiyonun akümülatörü temsil eden ilk parametresi float tipindedir ve listedeki bir öğeyi temsil eden ikinci parametre ise number tipindedir. Ancak eklemeden önce, eleman Int türündeyken float tipine uygun bir kalıp eşleştirme ve yayın kullanmamız gerekir.
