Herhangi bir türden bir listenin tersine çevrilmesi #

Bir listeyi tersine çevirmek için, liste öğelerinin ne tür olduğu değil, yalnızca hangi sırada oldukları önemlidir. Bu, genel bir işlev için mükemmel bir adaydır, bu nedenle, hangi listeden geçilirse geçilsin aynı ters çevirme işlevi kullanılabilir.

let rev list =
    let rec loop acc = function
        | []           -> acc
        | head :: tail -> loop (head :: acc) tail
    loop [] list

Kod, öğelerin türleri hakkında hiçbir varsayımda bulunmaz. Derleyici (veya F# interaktif) size bu fonksiyonun tip imzasının `‘T listesi -> ‘T listesi’ olduğunu söyleyecektir. ‘T’ size bunun herhangi bir kısıtlaması olmayan genel bir tür olduğunu söyler. Ayrıca “T” yerine “a” harfini de görebilirsiniz - harf önemsizdir çünkü yalnızca genel bir yer tutucudur. Bir “int listesi” veya “dize listesi” iletebiliriz ve her ikisi de sırasıyla bir “int listesi” veya bir “dize listesi” döndürerek başarılı bir şekilde çalışır.

Örneğin, F# interaktif’te:

> let rev list = ...
val it : 'T list -> 'T list
> rev [1; 2; 3; 4];;
val it : int list = [4; 3; 2; 1]
> rev ["one", "two", "three"];;
val it : string list = ["three", "two", "one"]

Bir listeyi farklı bir türe eşleme #

let map f list =
    let rec loop acc = function
        | []           -> List.rev acc
        | head :: tail -> loop (f head :: acc) tail
    loop [] list

Bu işlevin imzası, olabilecek en genel olan `(‘a -> ‘b) -> ‘a list -> ‘b list’ şeklindedir. Bu, “a"nın “b” ile aynı tür olmasını engellemez, aynı zamanda farklı olmalarına da izin verir. Burada, ‘f’ fonksiyonunun parametresi olan ‘a’ tipinin, ’list’ parametresinin tipi ile eşleşmesi gerektiğini görebilirsiniz. Bu işlev hala geneldir, ancak girişlerde bazı küçük kısıtlamalar vardır - türler eşleşmezse bir derleme hatası olacaktır.

Örnekler:

> let map f list = ...
val it : ('a -> 'b) -> 'a list -> 'b list
> map (fun x -> float x * 1.5) [1; 2; 3; 4];;
val it : float list = [1.5; 3.0; 4.5; 6.0]
> map (sprintf "abc%.1f") [1.5; 3.0; 4.5; 6.0];;
val it : string list = ["abc1.5"; "abc3.0"; "abc4.5"; "abc6.0"]
> map (fun x -> x + 1) [1.0; 2.0; 3.0];;
error FS0001: The type 'float' does not match the type 'int'
