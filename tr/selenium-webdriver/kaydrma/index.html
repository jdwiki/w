Java’yı farklı şekillerde kullanarak Farklı Kaydırma #

Aşağıda verilen çözüm, bazı sözdizimi değişiklikleriyle desteklenen diğer programlama dillerinde de kullanılabilir

  1. Özel kaydırma çubuğu varken (Tarayıcı kaydırması değil) web sayfasındaki aşağı kaydırma sayfasını/bölümü/bölümü yapmak. Demo için Buraya Tıklayın ve kaydırma çubuğunun bağımsız öğesi olup olmadığını kontrol edin.

Aşağıda verilen kodda kaydırma çubuğu öğenizi iletin ve kaydırma noktaları gerektirir.

    public static boolean scroll_Page(WebElement webelement, int scrollPoints)
    {
    try
    {
        System.out.println("---------------- Started - scroll_Page ----------------");
        driver = ExecutionSetup.getDriver();
        dragger = new Actions(driver);

        // drag downwards
        int numberOfPixelsToDragTheScrollbarDown = 10;
        for (int i = 10; i < scrollPoints; i = i + numberOfPixelsToDragTheScrollbarDown)
        {
            dragger.moveToElement(webelement).clickAndHold().moveByOffset(0, numberOfPixelsToDragTheScrollbarDown).release(webelement).build().perform();
        }
        Thread.sleep(500);
        System.out.println("---------------- Ending - scroll_Page ----------------");
        return true;
    }
    catch (Exception e)
    {
        System.out.println("---------------- scroll is unsucessfully done in scroll_Page ----------------");
        e.printStackTrace();
        return false;
    }
  }
  1. Özel kaydırma çubuğu varken (Tarayıcı kaydırma değil) web sayfasında Yukarı Kaydır sayfası/bölümü/bölümü yapmak. Demo için Buraya Tıklayın ve kaydırma çubuğunun bağımsız öğesi olup olmadığını kontrol edin.

Aşağıda verilen kodda kaydırma çubuğu öğenizi iletin ve kaydırma noktaları gerektirir.

public static boolean scroll_Page_Up(WebElement webelement, int scrollPoints)
{
    try
    {
        System.out.println("---------------- Started - scroll_Page_Up ----------------");
        driver = ExecutionSetup.getDriver();
        dragger = new Actions(driver);
        // drag upwards
        int numberOfPixelsToDragTheScrollbarUp = -10;
        for (int i = scrollPoints; i > 10; i = i + numberOfPixelsToDragTheScrollbarUp)
        {
            dragger.moveToElement(webelement).clickAndHold().moveByOffset(0, numberOfPixelsToDragTheScrollbarUp).release(webelement).build().perform();
        }
        System.out.println("---------------- Ending - scroll_Page_Up ----------------");
        return true;
    }
    catch (Exception e)
    {
        System.out.println("---------------- scroll is unsucessfully done in scroll_Page_Up----------------");
        e.printStackTrace();
        return false;
    }
}
  1. Çoklu tarayıcı kaydırma (Dahili tarayıcı) olduğunda aşağı kaydırma yapmak için ve Sayfa Aşağı Tuşu ile aşağı kaydırmak istediğinizde. Demo için buraya tıklayın

Aşağıda verilen kodda, kaydırma alanı öğenizi <div> gibi iletin ve sayfa aşağı tuşu gerektirir.

 public static boolean pageDown_New(WebElement webeScrollArea, int iLoopCount)
 {
    try
    {
        System.out.println("---------------- Started - pageDown_New ----------------");
        driver = ExecutionSetup.getDriver();
        dragger = new Actions(driver);

        for (int i = 0; i <= iLoopCount; i++)
        {
            dragger.moveToElement(webeScrollArea).click().sendKeys(Keys.PAGE_DOWN).build().perform();

        System.out.println"---------------- Ending - pageDown_New ----------------");
        return true;
    }
    catch (Exception e)
    {
        System.out.println("---------------- Not able to do page down ----------------");
        return false;
    }
}

  1. Yukarı/Aşağı kaydırma yapmak için tarayıcı kaydırıldığında (Dahili tarayıcı) ve Yalnızca sabit nokta ile Yukarı/Aşağı kaydırmak istediğinizde. Demo için buraya tıklayın

Aşağıda verilen kodda kaydırma noktanızı iletin. Olumlu, aşağı, olumsuz, yukarı kaydırma anlamına gelir.

public static boolean scroll_without_WebE(int scrollPoint)
{
    JavascriptExecutor jse;
    try
    {
        System.out.println("---------------- Started - scroll_without_WebE ----------------");

        driver = ExecutionSetup.getDriver();
        jse = (JavascriptExecutor) driver;
        jse.executeScript("window.scrollBy(0," + scrollPoint + ")", "");

        System.out.println("---------------- Ending - scroll_without_WebE ----------------");
        return true;
    }
    catch (Exception e)
    {
        System.out.println("---------------- scroll is unsucessful in scroll_without_WebE ----------------");
        e.printStackTrace();
        return false;
    }
}
  1. Yukarı/Aşağı kaydırma yapmak için tarayıcı kaydırma (Dahili tarayıcı) ve Yukarı/Aşağı kaydırma yapmak istediğinizde Görünür alanda öğe yapmak veya dinamik kaydırma için. Demo için buraya tıklayın

Aşağıda verilen kodda öğenizi iletin.

public static boolean scroll_to_WebE(WebElement webe)
{
    try
    {
        System.out.println("---------------- Started - scroll_to_WebE ----------------");

        driver = ExecutionSetup.getDriver();
        ((JavascriptExecutor) driver).executeScript("arguments[0].scrollIntoView();", webe);

        System.out.println("---------------- Ending - scroll_to_WebE ----------------");
        return true;
    }
    catch (Exception e)
    {
        System.out.println("---------------- scroll is unsucessful in scroll_to_WebE ----------------");
        e.printStackTrace();
        return false;
    }
}

Python kullanarak kaydırma #

1. “Eylemler” ile hedef öğeye (sayfanın alt kısmındaki “ŞABLONLARA GÖZAT” düğmesi) kaydırma*

from selenium import webdriver
from selenium.webdriver.common.action_chains import ActionChains

driver = webdriver.Chrome()
driver.get('http://www.w3schools.com/')
target = driver.find_element_by_link_text('BROWSE TEMPLATES')
actions = ActionChains(driver)
actions.move_to_element(target)
actions.perform()

2. “JavaScript” ile hedef öğeye (sayfanın altındaki “ŞABLONLARA GÖZAT” düğmesi) kaydırma*

from selenium import webdriver

driver = webdriver.Chrome()
driver.get('http://www.w3schools.com/')
target = driver.find_element_by_link_text('BROWSE TEMPLATES')
driver.execute_script('arguments[0].scrollIntoView(true);', target)

3. Yerleşik yöntemle hedef öğeye kaydırma (sayfanın altındaki “ŞABLONLARA GÖZAT” düğmesi)

from selenium import webdriver

    driver = webdriver.Chrome()
    driver.get('http://www.w3schools.com/')
    target = driver.find_element_by_link_text('BROWSE TEMPLATES')
    target.location_once_scrolled_into_view

“location_once_scrolled_into_view” öğesinin kaydırmadan sonra öğenin “x”, “y” koordinatlarını da döndürdüğünü unutmayın

4. Keys ile sayfanın en altına kaydırma

from selenium import webdriver
from selenium.webdriver.common.keys import Keys 

driver = webdriver.Chrome()
driver.get('http://www.w3schools.com/')
driver.find_element_by_tag_name('body').send_keys(Keys.END) # Use send_keys(Keys.HOME) to scroll up to the top of page

send_keys(Keys.DOWN)/send_keys(Keys.UP) ve send_keys(Keys.PAGE_DOWN)/send_keys(Keys.PAGE_UP) öğelerinin de kaydırma için kullanılabileceğini unutmayın.
