Selenyum Web Sürücüsü Nedir? #

Selenium, tarayıcıları otomatikleştirmek için tasarlanmış bir dizi araçtır. Birden çok platformda web uygulaması testleri için yaygın olarak kullanılır. Selenium şemsiyesi altında Selenium WebDriver (ex-Selenium RC), Selenium IDE ve Selenium Grid gibi birkaç araç bulunmaktadır.

WebDriver, web sayfalarındaki DOM öğelerini değiştirmenize ve ayrıca kullanıcı aracılarının davranışını komuta etmenize olanak tanıyan bir uzaktan kumanda arayüzüdür. Bu arayüz, aşağıdakiler gibi çeşitli platformlar için uygulanmış, dilden bağımsız bir kablo protokolü sağlar:

  • GeckoDriver (Mozilla Firefox)
  • ChromeDriver (Google Chrome)
  • SafariDriver (Apple Safari)
  • InternetExplorerDriver (MS InternetExplorer)
  • MicrosoftWebDriver veya EdgeDriver (MS Edge)
  • OperaChromiumDriver (Opera tarayıcısı)

ve diğer uygulamalar:

  • EventFiringWebSürücüsü
  • HtmlUnitSürücüsü
  • PhantomJSSürücü
  • UzakWeb Sürücüsü

Selenium WebDriver, daha fazla kontrol ve standart yazılım geliştirme uygulamalarının uygulanması için çeşitli dillerde Nesne Yönelimli API’ler sağlayan Selenium araçlarından biridir. Bir kullanıcının bir web uygulamasıyla etkileşim kurma şeklini doğru bir şekilde simüle etmek için, “Sentezlenmiş JavaScript olayları"nın aksine “Yerel İşletim Sistemi Düzeyindeki Olayları” kullanır.

Başvurulacak bağlantılar:

  • http://www.seleniumhq.org/
  • http://www.aosabook.org/en/selenium.html
  • https://www.w3.org/TR/webdriver/

Kurulum veya Kurulum #

WebDriver’ı kullanmaya başlamak için Selenium sitesinden ilgili Sürücüyü edinmeniz gerekir: Selenium HQ İndirmeleri. Buradan, WebDriver’ı çalıştırmaya çalıştığınız tarayıcı(lar) ve/veya platform(lar) ile ilgili sürücüyü indirmeniz gerekir, örn. Chrome’da test yapıyorsanız, Selenium sitesi sizi şu adrese yönlendirecektir:

https://sites.google.com/a/chromium.org/chromedriver/

chromedriver.exe dosyasını indirmek için.

Son olarak, WebDriver’ı kullanabilmek için ilgili dil bağlantılarını indirmeniz gerekir, örn. C# kullanıyorsanız, gerekli .dll dosyalarını edinmek için Selenium HQ İndirmeleri sayfasından indirmeye erişebilir veya alternatif olarak bunları NuGet paket yöneticisi aracılığıyla Visual Studio’da paketler halinde indirebilirsiniz.

Gerekli dosyalar şimdi indirilmelidir, WebDriver’ı kullanmaya nasıl başlayacağınız hakkında bilgi için diğer ‘selenium-webdriver’ belgelerine bakın.

Selenium WebDriver’ı kurmanın en kolay yolu bir NuGet paket yöneticisi kullanmaktır.

Projenizde, “Referanslar"a sağ tıklayın ve gösterildiği gibi “NuGet Paketlerini Yönet"e tıklayın:

Ardından, “webdriver” arama kutusuna yazın. Daha sonra şöyle bir şey görmelisiniz:

Sağ taraftaki Yükle düğmesine tıklayarak “Selenium.WebDriver” ve “Selenium.Support” (Destek paketi Bekle gibi ek kaynaklar içerir) yükleyin.

Ardından, kullanmak istediğiniz Web Sürücülerinizi aşağıdakilerden biri gibi yükleyebilirsiniz:

  • Selenium.WebDriver.ChromeDriver (Google Chrome)

  • PhantomJS (başsız)

Java için kurulum veya kurulum #

Selenium Webdriver ve Java’yı programlama dili olarak kullanarak testler yazmak için Selenium Webdriver’ın JAR dosyalarını Selenium web sitesinden indirmeniz gerekecektir.

Selenium web sürücüsü için bir Java projesi kurmanın birden çok yolu vardır, bunların en kolaylarından biri Maven kullanmaktır. Maven, tüm bağımlılıklar dahil olmak üzere Selenium web sürücüsü için gerekli Java bağlamalarını indirir. Diğer yol ise JAR dosyalarını indirip projenize aktarmaktır.

Maven kullanarak Selenium Webdriver projesini kurma adımları:

  1. Bu belgeyi izleyerek maven’i Windows kutusuna kurun: https://maven.Apache.org/install.html

  2. ‘Selenium-learing’ adında bir klasör oluşturun

  3. pom.xml adlı herhangi bir metin düzenleyiciyi kullanarak yukarıdaki klasöre bir dosya oluşturun.

  4. Aşağıdaki içeriği pom.xml dosyasına kopyalayın

     <project xmlns="http://maven.apache.org/POM/4.0.0"
      xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
      xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/xsd/maven-4.0.0.xsd">
         <modelVersion>4.0.0</modelVersion>
         <groupId>SeleniumLearning</groupId>
         <artifactId>SeleniumLearning</artifactId>
         <version>1.0</version>
         <dependencies>
             <dependency>
                 <groupId>org.seleniumhq.selenium</groupId>
                 <artifactId>selenium-learning</artifactId>
                 <version>3.0.0-beta1</version>
             </dependency>
         </dependencies>
 </project>

Not: Yukarıda belirttiğiniz sürümün en son sürüm olduğundan emin olun. En son sürümü buradan kontrol edebilirsiniz: http://docs.seleniumhq.org/download/maven.jsp

  1. Komut satırını kullanarak proje dizinine aşağıdaki komutu çalıştırın.

    mvn clean install

Yukarıdaki komut, gerekli tüm bağımlılıkları indirecek ve ardından projeye ekleyecektir. 6. Eclipse IDE’ye aktarabileceğiniz bir Eclipse projesi oluşturmak için aşağıdaki komutu yazın.

`mvn eclipse:eclipse`
  1. Projeyi Eclipse ide’ye aktarmak için aşağıdaki adımları takip edebilirsiniz.

Elipse’i Aç -> Dosya -> İçe Aktar -> Genel -> Çalışma Alanına Mevcut Proje -> İleri -> Gözat -> pom.xml içeren klasörü bulun -> Tamam -> Bitir

Projenize sağ tıklayarak m2Eclipse eklentisini kurun ve Maven -> Bağımlılık Yönetimini Etkinleştir’i seçin.

** Jar dosyalarını kullanarak Selenium Webdriver projesini kurma adımları**

  1. Aşağıdaki adımları izleyerek Eclipse’de yeni bir proje oluşturun.

Elipse’i Aç -> Dosya -> Yeni -> Java Projesi -> Bir ad verin (selenyum öğrenme) -> Bitir

  1. Jar dosyalarını http://www.seleniumhq.org/download/ adresinden indirin. Hem Selenium Standalone Server hem de Selenium Client & WebDriver Language Bindings‘i indirmeniz gerekir. Bu belge Java’dan bahsettiği için Java bölümünden sadece jar indirmeniz gerekiyor. Ekli ekran görüntüsüne bir göz atın.

Not: Selenium Bağımsız Sunucu, yalnızca testleri çalıştırmak için uzak sunucu kullanmak istiyorsanız gereklidir. Bu belge, projeyi kurmanın tamamen üstünde olduğundan, her şeyin yerinde olması daha iyidir.

  1. Kavanozlar zip dosyasına indirilecek, sıkıştırılmış halden çıkarılacak. .jar dosyasını doğrudan görebilmeniz gerekir.
  1. Eclipse’de 1. adımda oluşturduğunuz projeye sağ tıklayın ve aşağıdaki adımları izleyin.

Özellikler ->Java Yapı Yolu -> Kitaplıkları Seç sekmesi -> Dış Kavanoz Ekle’ye tıklayın -> Yukarıda indirdiğiniz sıkıştırılmamış jar klasörünü bulun -> lib klasöründeki tüm kavanozları seçin -> Tamam’a tıklayın -> Tekrar Ekle’ye tıklayın Harici Jars -> Aynı sıkıştırılmamış klasörü bulun -> lib klasörünün dışındaki kavanozu seçin (client-combined-3.0.0-beta1-nodeps.jar) -> Tamam

Benzer şekilde yukarıdaki adımı izleyerek ‘Selenium Bağımsız Sunucu’yu ekleyin.

  1. Artık projenize selenyum kodu yazmaya başlayabilirsiniz.

Not: Yukarıdaki belgeler Selenium-3.0.0 beta sürümüne dayanmaktadır, bu nedenle belirtilen jar dosyalarının adları sürümle değişebilir.
