PhantomJS [C#] #

PhantomJS JavaScript destekli** tam özellikli başsız bir web tarayıcısıdır.

Başlamadan önce bir PhantomJS sürücüsü indirmeniz ve bunu kodunuzun başına koymanız gerekir:

using OpenQA.Selenium;
using OpenQA.Selenium.PhantomJS;

Harika, şimdi başlatmaya geçiyoruz:

var driver = new PhantomJSDriver();

Bu, PhantomJSDriver sınıfının yeni bir örneğini oluşturacaktır. Daha sonra, aşağıdakiler gibi her Web Sürücüsü ile aynı şekilde kullanabilirsiniz:

using (var driver = new PhantomJSDriver())
{
    driver.Navigate().GoToUrl("http://stackoverflow.com/");

    var questions = driver.FindElements(By.ClassName("question-hyperlink"));

    foreach (var question in questions)
    {
        // This will display every question header on StackOverflow homepage.
        Console.WriteLine(question.Text);
    }
}

Bu iyi çalışıyor. Ancak, muhtemelen karşılaştığınız sorun, UI ile çalışırken, PhantomJS çoğu durumda gerçekten istenmeyen yeni bir konsol penceresi açar. Neyse ki, pencereyi gizleyebiliriz ve hatta PhantomJSOptions ve PhantomJSDriverService kullanarak performansı biraz iyileştirebiliriz. Aşağıdaki tam çalışma örneği:

// Options are used for setting "browser capabilities", such as setting a User-Agent
// property as shown below:
var options = new PhantomJSOptions();
options.AddAdditionalCapability("phantomjs.page.settings.userAgent", 
"Mozilla/5.0 (Windows NT 6.1; Win64; x64; rv:25.0) Gecko/20100101 Firefox/25.0");

// Services are used for setting up the WebDriver to your likings, such as
// hiding the console window and restricting image loading as shown below:
var service = PhantomJSDriverService.CreateDefaultService();
service.HideCommandPromptWindow = true;
service.LoadImages = false;

// The same code as in the example above:
using (var driver = new PhantomJSDriver(service, options))
{
    driver.Navigate().GoToUrl("http://stackoverflow.com/");

    var questions = driver.FindElements(By.ClassName("question-hyperlink"));

    foreach (var question in questions)
    {
        // This will display every question header on StackOverflow homepage.
        Console.WriteLine(question.Text);
    }
}

Profesyonel ipucu: Bir sınıfa tıklayın (örneğin, PhantomJSDriverService) ve ne yaptıklarının kısa bir açıklamasıyla birlikte tam olarak ne içerdiklerini görmek için F12’ye basın.

Basit Tarayıcı [C#] #

“SimpleBrowser”, JavaScript desteği olmayan hafif bir Web Sürücüsüdür.

Yukarıda bahsedilen “PhantomJS"den çok daha hızlıdır, ancak işlevsellik söz konusu olduğunda, süslü özellikleri olmayan basit görevlerle sınırlıdır.

İlk olarak, SimpleBrowser.WebDriver paketini indirmeniz ve ardından bu kodu başa koymanız gerekecek:

using OpenQA.Selenium;
using SimpleBrowser.WebDriver;

Şimdi, nasıl kullanılacağına dair kısa bir örnek:

using (var driver = new SimpleBrowserDriver())
{
    driver.Navigate().GoToUrl("http://stackoverflow.com/");

    var questions = driver.FindElements(By.ClassName("question-hyperlink"));

    foreach (var question in questions)
    {
        // This will display every question header on StackOverflow homepage.
        Console.WriteLine(question.Text);
    }
}

HTMLUnitSürücüsü #

HTMLUnitDriver, HtmlUnit’e dayalı Webdriver için başsız (GUI’siz) tarayıcının en hafif uygulamasıdır. HTML belgelerini modeller ve tıpkı normal tarayıcınızda yaptığınız gibi sayfaları çağırmanıza, formları doldurmanıza, bağlantıları tıklamanıza vb. izin veren bir API sağlar. JavaScript’i destekler ve AJAX kitaplıklarıyla çalışır. Test etmek ve web sitesinden veri almak için kullanılır.

Örnek: “http://stackoverflow.com/“dan soru listesi almak için HTMLUnitDriver kullanımı.

import java.util.List;
import java.util.concurrent.TimeUnit;
import org.openqa.selenium.By;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.htmlunit.HtmlUnitDriver;
        
class testHeadlessDriver{
            private void getQuestions() {
                    WebDriver driver = new HtmlUnitDriver();
                    driver.get("http://stackoverflow.com/");
                    driver.manage().timeouts().implicitlyWait(60, TimeUnit.SECONDS);
                    List<WebElement> questions = driver.findElements(By.className("question-hyperlink"));
                    questions.forEach((question) -> {
                        System.out.println(question.getText());
                    });
                   driver.close();
                }
    }

Diğer tarayıcılarla aynıdır (Mozilla Firefox, Google Chrome, IE), ancak GUI’si yoktur, yürütme ekranda görünmez.
