Sözdizimi #

  • ByChained([] bys tarafından parametreler)

Öğeler, bulucular ve ‘By’ sınıfı kullanılarak Selenium’da bulunur. Selenium ile sağlam bir otomasyon projesi yapabilmek için Web Elements yer belirleyicilerini akıllıca kullanmak gerekir. Konum belirleyiciler açıklayıcı, benzersiz ve değişme olasılığı düşük olmalıdır, böylece örneğin testlerde yanlış pozitifler almazsınız. Öncelik kullanmaktır:

  1. Kimlik - benzersiz olduğu ve tam olarak istediğiniz öğeyi alacağınız için.
  2. Sınıf Adı - Açıklayıcıdır ve belirli bir bağlamda benzersiz olabilir.
  3. CSS (xpath’ten daha iyi performans) - Daha karmaşık seçiciler için.
  4. XPATH - CSS’nin kullanılamadığı durumlarda (XPATH Ekseni), ör. “div::ebeveyn”.

Konum belirleyicilerin geri kalanı değişime veya işlemeye yatkındır ve tercihen bundan kaçınılır.

Temel kural: kodunuz belirli bir öğeyi bulamıyorsa bunun bir nedeni, kodunuzun tüm DOM öğelerinin indirilmesini beklememiş olması olabilir. Söz konusu öğeyi aramadan önce programınıza kısa bir süre “beklemesini” (3-5 saniye deneyin ve ardından gerektiği gibi yavaşça artırın) söylemeyi düşünün. İşte Python’da bu soru‘den alınan bir örnek:

from selenium import webdriver
import time

browser = webdriver.Firefox()
browser.get("https://app.website.com")

reports_element = browser.find_element_by_xpath("//button[contains(text(), 'Reports')]")

# Element not found! Try giving time for the browser to download all DOM elements:
time.sleep(10)

reports_element = browser.find_element_by_xpath("//button[contains(text(), 'Reports')]")
# This returns correct value!

WebDriver kullanarak sayfa öğelerini bulma #

Bir web sayfasında WebElements ile etkileşim kurmak için önce öğenin konumunu belirlememiz gerekir.

By, selenyumda bulunan anahtar kelimedir.

Elemanları şuradan bulabilirsiniz..

  1. ** tarafından** kimlik
  2. Tarafından Sınıf Adı
  3. Tarafından EtiketAdı
  4. Tarafından Ad
  5. Tarafından Bağlantı Metni
  6. Tarafından Kısmi Bağlantı Metni
  7. Tarafından CSS Seçici
  8. Tarafından XPath
  9. Kullanma JavaScript

Aşağıdaki komut dosyası örneğini düşünün

<form name="loginForm">
Login Username: <input id="username" name="login" type="text" />
Password: <input id="password" name="password" type="password" />
<input name="login" type="submit" value="Login" />

Yukarıdaki kodda kullanıcı adı ve şifre id’ler kullanılarak belirlenir. Şimdi öğeleri id ile tanımlayacaksınız.

driver.findElement(By.id(username));

driver.findElement(By.id(password));

Selenyum 7 farklı dili desteklediğinden, bu belge size tüm dillerdeki öğeleri bulma konusunda bir fikir verir.

Kimliğe göre #

Kimlik kullanarak bir öğenin nasıl bulunacağına ilişkin örnek:

<div id="coolestWidgetEvah">...</div>

Java       -  WebElement element = driver.findElement(By.id("coolestWidgetEvah"));
C#         -  IWebElement element = driver.FindElement(By.Id("coolestWidgetEvah"));
Python     -  element = driver.find_element_by_id("coolestWidgetEvah")
Ruby       -  element = driver.find_element(:id, "coolestWidgetEvah")
JavaScript/Protractor -  var elm = element(by.id("coolestWidgetEvah"));

Sınıf Adına Göre #

Sınıf adını kullanarak bir öğenin nasıl bulunacağına ilişkin örnek:

<div class="cheese"><span>Cheddar</span></div>

Java       -  WebElement element = driver.findElement(By.className("cheese"));
C#         -  IWebElement element = driver.FindElement(By.ClassName("cheese"));
Python     -  element = driver.find_element_by_class_name("cheese")
Ruby       -  cheeses = driver.find_elements(:class, "cheese")
JavaScript/Protractor -  var elm = element(by.className("cheese"));

Etiket Adına Göre #

Etiket adını kullanarak bir öğenin nasıl bulunacağına ilişkin örnek:

<iframe src="..."></iframe>

Java       -  WebElement element = driver.findElement(By.tagName("iframe"));
C#         -  IWebElement element = driver.FindElement(By.TagName("iframe"));
Python     -  element = driver.find_element_by_tag_name("iframe")
Ruby       -  frame = driver.find_element(:tag_name, "iframe")
JavaScript/Protractor -  var elm = element(by.tagName("iframe"));

İsimle #

Ad kullanarak bir öğenin nasıl bulunacağına ilişkin örnek:

<input name="cheese" type="text"/>

Java       -  WebElement element = driver.findElement(By.name("cheese"));
C#         -  IWebElement element = driver.FindElement(By.Name("cheese"));
Python     -  element = driver.find_element_by_name("cheese")
Ruby       -  cheese = driver.find_element(:name, "cheese")
JavaScript/Protractor -  var elm = element(by.name("cheese"));

Bağlantı Metnine Göre #

Bağlantı metnini kullanarak bir öğenin nasıl bulunacağına ilişkin örnek:

<a href="http://www.google.com/search?q=cheese">cheese</a>>

Java       -  WebElement element = driver.findElement(By.linkText("cheese"));
C#         -  IWebElement element = driver.FindElement(By.LinkText("cheese"));
Python     -  element = driver.find_element_by_link_text("cheese")
Ruby       -  cheese = driver.find_element(:link, "cheese")
JavaScript/Protractor -  var elm = element(by.linkText("cheese"));

Kısmi Bağlantı Metni ile #

Kısmi bağlantı metni kullanarak bir öğenin nasıl bulunacağına ilişkin örnek:

<a href="http://www.google.com/search?q=cheese">search for cheese</a>>

Java       -  WebElement element = driver.findElement(By.partialLinkText("cheese"));
C#         -  IWebElement element = driver.FindElement(By.PartialLinkText("cheese"));
Python     -  element = driver.find_element_by_partial_link_text("cheese")
Ruby       -  cheese = driver.find_element(:partial_link_text, "cheese")
JavaScript/Protractor -  var elm = element(by.partialLinkText("cheese"));

CSS Seçicilere Göre #

CSS Seçicileri kullanarak bir öğenin nasıl bulunacağına ilişkin örnek:

<div id="food" class="dairy">milk</span>

Java       -  WebElement element = driver.findElement(By.cssSelector("#food.dairy")); //# is used to indicate id and . is used for classname.
C#         -  IWebElement element = driver.FindElement(By.CssSelector("#food.dairy"));
Python     -  element = driver.find_element_by_css_selector("#food.dairy")
Ruby       -  cheese = driver.find_element(:css, "#food span.dairy.aged")
JavaScript/Protractor -  var elm = element(by.css("#food.dairy"));

İşte CSS Seçicileri oluşturma hakkında bir makale: http://www.w3schools.com/cssref/css_selectors.asp

XPath tarafından #

XPath kullanarak bir öğenin nasıl bulunacağına ilişkin örnek:

<input type="text" name="example" />

Java       -  WebElement element = driver.findElement(By.xpath("//input"));
C#         -  IWebElement element = driver.FindElement(By.XPath("//input"));
Python     -  element = driver.find_element_by_xpath("//input")
Ruby       -  inputs = driver.find_elements(:xpath, "//input")
JavaScript/Protractor -  var elm = element(by.xpath("//input"));

İşte XPath hakkında bir makale: http://www.w3schools.com/xsl/xpath_intro.asp

JavaScript’i kullanma #

Bir öğeyi bulmak için isteğe bağlı javascript çalıştırabilirsiniz ve bir DOM Öğesi döndürdüğünüz sürece, otomatik olarak bir WebElement nesnesine dönüştürülür.

jQuery’nin yüklü olduğu bir sayfada basit bir örnek:

Java       -  WebElement element = (WebElement) 
((JavascriptExecutor)driver).executeScript("return $('.cheese')[0]");

C#         -  IWebElement element = (IWebElement)
((IJavaScriptExecutor)driver).ExecuteScript("return $('.cheese')[0]");

Python     -  element = driver.execute_script("return $('.cheese')[0]");
Ruby       -  element = driver.execute_script("return $('.cheese')[0]")

Birden çok kritere göre seçme [C#] #

Seçicileri birlikte kullanmak da mümkündür. Bu, OpenQA.Selenium.Support.PageObjects.ByChained nesnesi kullanılarak yapılır:

element  = driver.FindElement(new ByChained(By.TagName("input"), By.ClassName("class"));

Herhangi bir sayıda “By” zincirlenebilir ve AND türü seçimi olarak kullanılır (yani tüm “By"ler eşleştirilir)

Sayfanın yüklenmesini durdurmadan önce öğeleri seçme #

driver.Navigate().GoToUrl(url); çağrılırken, sayfa tamamen yüklenene kadar kod yürütme durur. Bu, yalnızca veri çıkarmak istediğinizde bazen gereksizdir.

Not: Aşağıdaki kod örnekleri hack olarak kabul edilebilir. Bunu yapmanın “resmi” bir yolu yoktur.

Yeni bir iş parçacığı oluştur #

Bir web sayfasını yüklemek için bir iş parçacığı oluşturun ve başlatın, ardından Bekleyin‘i kullanın.

C#

using (var driver = new ChromeDriver())
{
    new Thread(() =>
    {
        driver.Navigate().GoToUrl("http://stackoverflow.com");
    }).Start();

    new WebDriverWait(driver, TimeSpan.FromSeconds(10))
      .Until(ExpectedConditions.ElementIsVisible(By.XPath("//div[@class='summary']/h3/a")));
}

Zaman Aşımlarını Kullan #

Bir WebDriverTimeout kullanarak bir sayfa yükleyebilirsiniz ve belirli bir süre sonra, sayfanın yüklenmesini durduracak bir istisna atar. Catch bloğunda Wait kullanabilirsiniz.

C#

using (var driver = new ChromeDriver())
{
    driver.Manage().Timeouts().SetPageLoadTimeout(TimeSpan.FromSeconds(5));

    try
    {
        driver.Navigate().GoToUrl("http://stackoverflow.com");
    }
    catch (WebDriverTimeoutException)
    {
        new WebDriverWait(driver, TimeSpan.FromSeconds(10))
          .Until(ExpectedConditions.ElementIsVisible
            (By.XPath("//div[@class='summary']/h3/a")));
    }
}

Sorun: Zaman aşımını çok kısa ayarladığınızda, istediğiniz öğenin mevcut olup olmadığına bakılmaksızın sayfanın yüklenmesi duracaktır. Zaman aşımını çok uzun ayarladığınızda, performans avantajını ortadan kaldıracaksınız.
