Komut satırından bazı Perl kodlarını yürütün #

Basit tek satırlar, “-e” anahtarı kullanılarak Perl için komut satırı argümanları olarak belirtilebilir (“yürüt” düşünün):

perl -e'print "Hello, World!\n"'

Windows alıntı kuralları nedeniyle, tek tırnaklı dizeleri kullanamazsınız, ancak şu değişkenlerden birini kullanmanız gerekir:

perl -e"print qq(Hello, World!\n)"
perl -e"print \"Hello, World!\n\""

Eski kodun kırılmasını önlemek için, ‘-e’ ile yalnızca Perl 5.8.x’e kadar mevcut olan sözdiziminin kullanılabileceğini unutmayın. Perl sürümünüzün destekleyebileceği daha yeni bir şey kullanmak için bunun yerine -E kullanın. Örneğin. 5.10.0’dan itibaren kullanılabilen ‘say’ı ve >=v5.14.0’dan Unicode 6.0’ı kullanmak için (‘STDOUT’un UTF-8’i yazdırdığından emin olmak için ‘-CO’ kullanır):

perl -CO -E'say "\N{PILE OF POO}"'
son sürüm

Windows tek satırlarında çift tırnaklı dizeleri kullanma #

Windows, komut satırı parametrelerini sarmak için yalnızca çift tırnak kullanır. Perl tek satırda çift tırnak kullanmak için (yani, enterpolasyonlu bir değişkenle bir dize yazdırmak için), onlardan ters eğik çizgi ile kaçmanız gerekir:

perl -e "my $greeting = 'Hello'; print \"$greeting, world!\n\""

Okunabilirliği artırmak için bir qq() operatörü kullanabilirsiniz:

perl -e "my $greeting = 'Hello'; print qq($greeting, world!\n)"

Bir desenle eşleşen satırları yazdır (PCRE grep) #

perl -ne'print if /foo/' file.txt

Büyük/küçük harfe duyarsız:

perl -ne'print if /foo/i' file.txt

Bir alt dizeyi başka bir alt dizeyle değiştirin (PCRE sed) #

perl -pe"s/foo/bar/g" file.txt

Veya yerinde:

perl -i -pe's/foo/bar/g' file.txt

Windows’ta:

perl -i.bak -pe"s/foo/bar/g" file.txt

Yalnızca belirli alanları yazdır #

perl -lane'print "$F[0] $F[-1]"' data.txt
# prints the first and the last fields of a space delimited record

CSV örneği:

perl -F, -lane'print "$F[0] $F[-1]"' data.csv

5 ile 10 arasındaki satırları yazdır #

perl -ne'print if 5..10' file.txt

Dosyayı yerinde düzenleyin #

Yedek kopya olmadan (Windows’ta desteklenmez)

perl -i -pe's/foo/bar/g' file.txt

Bir yedek kopya ile file.txt.bak

perl -i.bak -pe's/foo/bar/g' file.txt

‘backup’ alt dizinindeki ‘old_file.txt.orig’ yedek kopyasıyla (ikincisinin mevcut olması koşuluyla):

perl -i'backup/old_*.orig' -pe's/foo/bar/g' file.txt

Tüm dosyayı bir dize olarak okumak #

perl -0777 -ne'print "The whole file as a string: --->$_<---\n"'

Not: “-0777” yalnızca bir kuraldır. Herhangi bir “-0400” ve üstü aynı olacaktır.

Mojolicious’a dosya yükle #

perl -Mojo -E 'p("http://localhost:3000" => form => {Input_Type => "XML", Input_File => {file => "d:/xml/test.xml"}})'

’d:/xml/test.xml’ dosyası, ’localhost:3000’ üzerindeki bağlantıları dinleyen sunucuya yüklenecektir (Kaynak)

Bu örnekte:

-Mmodule, programınızı çalıştırmadan önce use modülünü; yürütür -E komut satırı bir program satırı girmek için kullanılır Eğer ‘ojo’ modülünüz yoksa, onu kurmak için ‘cpanm ojo’ komutunu kullanabilirsiniz.

Perl’in nasıl çalıştırılacağı hakkında daha fazla bilgi için ‘perldoc perlrun’ komutunu kullanın veya burayı okuyun.
