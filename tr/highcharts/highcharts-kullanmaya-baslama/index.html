Kurulum veya Kurulum #

Highchart almanın yolları:

  • Doğrudan Highcharts CDN‘den yükleyin

  • Resmi web sitesinden İndirin dosyalar.

  • npm aracılığıyla;

    npm install highcharts

  • Bower aracılığıyla;

    bower install highcharts

Ana kitaplığı satıcının CDN’sinden yüklemek için aşağıdakini kodunuza eklemeniz yeterlidir:

<script src="https://code.highcharts.com/highcharts.js"></script>

Dışa aktarma modülü gibi ek kitaplıklar, “highcharts.js” bildiriminizden sonra yüklenmelidir.

Kütüphaneleri doğrudan Highcharts’tan aramak size en güncel sürümü sağlayacaktır. Ancak, belirli bir sürümle en iyi şekilde çalışan belirli çizelgeleriniz varsa veya web sitenizin performansını optimize etmek istiyorsanız, dosyaları yerel olarak saklamayı düşünebilirsiniz.

Aşağıdaki kaynaklar, Highcharts’ı yükleme ve yapılandırma hakkında ayrıntılı bilgilerin yanı sıra, grafiklerinizi varsayılan yüklemenin ötesinde özelleştirmek için kullanabileceğiniz ek kitaplıklar ve modüller sunar.

  • Yüksek Grafikler için kurulum talimatları
  • Dışa aktarma modülüne genel bakış
  • Tüm modüller ve sürümler dahil olmak üzere Highcharts dosya hizmeti

Selam Dünya #

“index.html” dosyanıza “highcharts.js"yi ekleyerek başlayın

<html>
  <head>
    <script src="http://code.highcharts.com/highcharts.js"></script>
  </head>

Grafiğinizi içerecek bir “

” ekleyin

  <body>
    <div id="chart">
      <!-- Chart goes here -->
    </div>

Grafiği oluşturmak için yapılandırmayı belirtin. Bir grafik oluşturmak için gereken minimum yapılandırma:

  • Grafik nereye gidiyor? - chart.renderTo

  • Çizilecek veri nedir? - Çizilecek verileri beslemenin birkaç yolu vardır; aralarında en yaygın olanı serisi nesnesidir.

      var chartOptions = {
      chart: {
          renderTo: 'chart'  
      },
      series: [{      
          data: [1, 2]
      }]  
  };
  var chartHandle = Highcharts.Chart(chartOptions);

Bu, aşağıdaki gibi bir arsa oluşturur - keman.

Grafiğe eklenebilecek çok sayıda yapılandırma seçeneği vardır, bunlardan birkaçı,

  • chart.type - Ne tür bir grafik çizmek istiyorsunuz?
  • title - Grafiğin başlığı için yapılandırma
  • xAxis/yAxis - Grafiğin x ekseni/y ekseni için yapılandırma

Tüm yapılandırma seçeneklerinin tam listesi burada bulunabilir.

    <script>
    var chartOptions = {
      chart: {
        renderTo: 'chart',
        type: 'bar'
      },
      title: {
        text: 'Hello Highcharts'
      },
      xAxis: {
        categories: ['Hello', 'World']
      },
      yAxis: {
        title: 'Value'
      },
      series: [{
        name: 'Highcharts Intro',
        data: [1, 2]
      }]
    };

    var chart = new Highcharts.Chart(chartOptions);
    
    </script>
  </body>
</html>
