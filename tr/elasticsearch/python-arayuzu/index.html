Parametreler #

parametre Ayrıntılar
ev sahipleri “Host” ve “port” anahtarlarını içeren nesne biçimindeki ana bilgisayarlar dizisi. Varsayılan “host”, “localhost” ve “port” 9200’dür. Örnek bir giriş şuna benzer: “[{“host”: “ip of es server”, “port”: 9200}]`
sniff_on_start Boolean, istemcinin başlangıçta düğümleri koklamasını istiyorsanız, koklama, elastik arama kümesindeki düğümlerin listesini almak anlamına gelir
sniff_on_connection_fail İstemci etkinken bağlantı başarısız olursa koklamayı tetiklemek için Boolean
sniffer_timeout her koklama arasında saniye cinsinden zaman farkı
sniff_timeout saniyeler içinde tek bir koklama isteği için süre
retry_on_timeout İstemcinin zaman aşımının farklı bir elastik arama düğümüyle bağlantı kurmasını tetikleyip tetiklemeyeceği veya yalnızca hata atıp atmayacağı konusunda Booelan
http_auth Temel http kimlik doğrulaması burada kullanıcıadı:şifre

Bir Belgeyi İndeksleme (yani bir örnek ekleme) #

Gerekli Python Kitaplığını şu şekilde kurun:

$ pip install elasticsearch

Elasticsearch’e bağlanın, Belge Oluşturun (ör. veri girişi) ve Elasticsearch’ü kullanarak belgeyi “İndeksleyin”.

from datetime import datetime
from elasticsearch import Elasticsearch

# Connect to Elasticsearch using default options (localhost:9200)
es = Elasticsearch()

# Define a simple Dictionary object that we'll index to make a document in ES
doc = {
    'author': 'kimchy',
    'text': 'Elasticsearch: cool. bonsai cool.',
    'timestamp': datetime.now(),
}

# Write a document
res = es.index(index="test-index", doc_type='tweet', id=1, body=doc)
print(res['created'])

# Fetch the document
res = es.get(index="test-index", doc_type='tweet', id=1)
print(res['_source'])

# Refresh the specified index (or indices) to guarantee that the document
#  is searchable (avoid race conditions with near realtime search)
es.indices.refresh(index="test-index")

# Search for the document
res = es.search(index="test-index", body={"query": {"match_all": {}}})
print("Got %d Hits:" % res['hits']['total'])

# Show each "hit" or search response (max of 10 by default)
for hit in res['hits']['hits']:
    print("%(timestamp)s %(author)s: %(text)s" % hit["_source"])

Boş bir dizin oluşturma ve eşlemeyi ayarlama #

Bu örnekte, eşlemesini tanımlayarak boş bir dizin oluşturuyoruz (içinde hiçbir belgeyi dizinlemiyoruz).

İlk önce bir ElasticSearch örneği oluşturuyoruz ve ardından tercih ettiğimiz eşlemeyi tanımlıyoruz. Daha sonra indeksin var olup olmadığını kontrol ediyoruz, yoksa indeks adını ve mappingin gövdesini içeren ‘index’ ve ‘body’ parametrelerini belirterek oluşturuyoruz.

from elasticsearch import Elasticsearch

# create an ElasticSearch instance
es = Elasticsearch()
# name the index
index_name = "my_index"
# define the mapping
mapping = {
    "mappings": {
        "my_type": {
                "properties": {
                    "foo": {'type': 'text'},
                    "bar": {'type': 'keyword'}
                }
            }
        }
    }
    
# create an empty index with the defined mapping - no documents added
if not es.indices.exists(index_name):
    res = es.indices.create(
        index=index_name,
        body=mapping
    )
    # check the response of the request
    print(res)
    # check the result of the mapping on the index
    print(es.indices.get_mapping(index_name))

Kısmi Güncelleme ve Sorguya Göre Güncelleme #

Kısmi Güncelleme: Kısmi bir belge güncellemesi yapılması gerektiğinde kullanılır, yani aşağıdaki örnekte “doc_id” kimlikli belgenin “name” alanı “John” olarak güncellenecektir. Alan eksikse, yalnızca belgeye ekleneceğini unutmayın.

doc = {
    "doc": {
        "name": "John"
    }
}
es.update(index='index_name',
          doc_type='doc_name',
          id='doc_id',
          body=doc)

Sorguya göre güncelle: Bir koşulu karşılayan belgeleri güncellemek için gerektiğinde kullanılır, yani aşağıdaki örnekte, “ad” alanı “John” ile eşleşen belgelerin yaşını güncelliyoruz.

q = {
  "script": {
    "inline": "ctx._source.age=23",
    "lang": "painless"
  },
  "query": {
    "match": {
        "name": "John"
    }
  }
}

es.update_by_query(body=q, 
                   doc_type='doc_name', 
                   index='index_name')
