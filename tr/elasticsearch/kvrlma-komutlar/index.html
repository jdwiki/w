Sözdizimi #

  • curl -X<VERB> ‘<PROTOKOL>://<HOST>:<PORT>/<YOL>?<QUERY_STRING>’ -d ‘<BODY>’

  • Neresi:

  • VERB: Uygun HTTP yöntemi veya fiili: GET, POST, PUT, HEAD veya DELETE

  • PROTOKOL: http veya https (Elasticsearch’ün önünde bir https proxy’niz varsa.)

  • HOST: Elasticsearch kümenizdeki herhangi bir düğümün ana bilgisayar adı veya yerel makinenizdeki bir düğüm için yerel ana bilgisayar adı.

  • PORT: Elasticsearch HTTP hizmetini çalıştıran ve varsayılan olarak 9200 olan port.

  • YOL: API Bitiş Noktası (örneğin _count, kümedeki belge sayısını döndürür). Yol, _cluster/stats veya _nodes/stats/jvm gibi birden çok bileşen içerebilir.

  • QUERY_STRING: İsteğe bağlı herhangi bir sorgu dizesi parametresi (örneğin, ?pretty, okumayı kolaylaştırmak için JSON yanıtını güzel bir şekilde yazdırır.)

  • BODY: JSON kodlu bir istek gövdesi (istek gerekiyorsa.)

  • Referans: Elasticsearch ile Konuşmak: Elasticsearch Docs

Kümedeki belge sayısını saymak için Curl Komutu #

curl -XGET 'http://www.example.com:9200/myIndexName/_count?pretty'

Çıktı:

{
  "count" : 90,
  "_shards" : {
    "total" : 6,
    "successful" : 6,
    "failed" : 0
  }
}

Dizin içinde 90 belge var.

Referans Bağlantısı: Buradan

Kimliğe göre bir belge alın #

curl -XGET 'http://www.example.com:9200/myIndexName/myTypeName/1'

Çıktı:

{
    "_index" : "myIndexName",
    "_type" : "myTypeName",
    "_id" : "1",
    "_version" : 1,
    "found": true,
    "_source" : {
        "user" : "mrunal",
        "postDate" : "2016-07-25T15:48:12",
        "message" : "This is test document!"
    }
}

Referans Bağlantısı: Buradan

Dizin Oluştur #

curl -XPUT 'www.example.com:9200/myIndexName?pretty'

Çıktı:

{
  "acknowledged" : true
}

Referans Bağlantısı: Buradan

Tüm dizinleri listele #

curl 'www.example.com:9200/_cat/indices?v'

çıktı:

health status index               pri rep docs.count docs.deleted store.size pri.store.size 
green  open   logstash-2016.07.21   5   1       4760            0      4.8mb          2.4mb

green  open   logstash-2016.07.22   5   1      93528            0    103.6mb           52mb 
green  open   logstash-2016.07.25   5   1      20683            0     41.5mb         21.1mb

Referans Bağlantısı: Buradan

Bir Dizini Sil #

curl -XDELETE 'http://www.example.com:9200/myIndexName?pretty'

çıktı:

{
  "acknowledged" : true
}

Referans Bağlantısı: Buradan

Bir dizindeki tüm belgeleri listeleyin #

curl -XGET http://www.example.com:9200/myIndexName/_search?pretty=true&q=*:*

Bu, “Ara” API’sini kullanır ve “myIndexName” dizini altındaki tüm girişleri döndürür.
