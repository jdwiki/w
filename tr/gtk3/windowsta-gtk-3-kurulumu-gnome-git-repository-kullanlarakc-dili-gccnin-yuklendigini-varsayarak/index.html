GTK+3’ü İndirme (diğer sürümler için de uygundur) ve Kurulum #

Paketi İndirme:

Gerekli GTK Paketini indirmenin en kolay yolu, bu bağlantıyı kullanarak onu aramak ve indirmektir: https://git.gnome.org/browse/gtk+ (GNOME GIT Deposu)

GNOME GIT Deposu farklı sürümler için paketler sağlar ve listeyi kaydırarak istediğiniz sürümü kolayca bulabilirsiniz. Bunlar diğer kullanıcılar veya yazarlar tarafından yüklenir ve sık sık yeni paketlerin geldiğini göreceksiniz.

Alternatifler:

Paketi elde etmek için diğer seçenek veya alternatif yöntem, MSYS2’yi kullanmaktır. Bu yöntem biraz karmaşıktır ve iyi açıklanmamıştır veya belgelenmemiştir. En iyi yol, paketi doğrudan harici bir dosya barındırma web sitesinden indirmektir.

MSYS2 Yöntemi-

  1. http://www.gtk.org/download/windows.php (Paketi edinmeyle ilgili Resmi Web Sitesi Kaynağı)

  2. https://msys2.github.io/ (MSYS2’yi indirin)

Not : Bu Belgelerin Geri Kalanı, GTK Paketinin GNOME GIT Deposundan İndirildiği Varsayımına Dayanmaktadır. MSYS2 Yöntemi ile Test Edilmemiştir. Paket Evrensel Kaldığından, Kullanılan Yöntem Sorun Olmamalıdır. Kolaylığa Dayalı Olarak, Paketi Elde Etmenin Şiddetle Tavsiye Edilen Yolu, GNOME GIT Deposunu T0 Kullanmaktır.

Paketi İstenen Bir Konumda Çıkarma ve Saklama (C: Drive kullanılarak açıklanmıştır).

  1. C: Drive içinde gtk adında bir Klasör oluşturun. Klasör için başka adlar seçebilirsiniz, ancak bu ad, içeriği oldukça kolay tanımlamamıza yardımcı olur ve adı hatırlamak daha kolay hale gelir. Kolaylık olması için Gtk klasör konumunu %GD% olarak adlandıralım ve bu durumda C:\gtk (sizin durumunuzda farklı olabilir).

  2. Windows Ortam Değişkenlerinde Gtk ikilisinin Yolunu Ayarlama. Bu, Programları Derlemek ve çalıştırmak için “Cmd"yi kullanırken gerekli “Gtk” kitaplıklarına erişmek ve bunları elde etmek için önemlidir. Yol Ortamı Değişkenini kurmak için:

  • Windows XP : "Bilgisayarım" -> "Özellikler" üzerine sağ tıklayın.
  • Windows Vista/7/8/10 : "Bilgisayar" -> "Özellikler" -> "Gelişmiş sistem ayarları" üzerine sağ tıklayın. "Gelişmiş sekmesi" -> "Ortam" üzerine tıklayın değişkenler".
  • Açılan diyalog kutusunda mevcut olan Path değişkenine yeni Gtk bin file Path (%GD%\bin) ekleyin (Sistem Değişkenleri Altında veya Kullanıcı Değişkenleri Bölümüne Path adında bir değişken ekleyin ve içindeki değeri ekleyin) . Ayrıca “PKG_CONFIG_PATH” adında yeni bir değişken eklemeli ve ona şu değeri vermelisiniz - “C:\gtk\lib\pkgconfig” (çünkü gtk klasörüm C: sürücüsündedir).
Değişken Adı Değer
Yol %GD%\bin
PKG_CONFIG_PATH %GD%\lib\pkgconfig

%GD%, Gtk Klasörünün konumudur Şimdi Ortam Değişkenlerini Çıkarma ve ayarlama ile işimiz bitti. Kullanıcı Değişkenleri bölümünün yanı sıra Sistem değişkenleri bölümlerinde Path değişkenine değer eklediğimi göreceksiniz. Bu bölümlerden herhangi birine eklemek fazlasıyla yeterli.

3.Kurulumu test etme (Sisteminizde önceden GCC kurulu olmalıdır-MinGW tercih edilir (http://www.mingw.org/): Cmd’de aşağıdaki komutları çalıştırın:

pango-querymodules:%GTKDIR%\etc\pango\pango.modules

gdk-pixbuf-query-loaders:%GTKDIR%\lib\gdk-pixbuf-2.0\2.10.0\loaders.cache

gtk-query-immodules-3.0: %GTKDIR%\lib\gtk-3.0\3.0.0\immodules.cache

  • Konsolda, “pkg-config” komutunun şunu yazarak makul bir şey yazdırdığını doğrulayın: pkg-config --cflags --libs gtk+-3.0 Aşağıdaki resimde gösterilene benzer bir şey yazdırmalıdır.[! Komut Sonucu]4
