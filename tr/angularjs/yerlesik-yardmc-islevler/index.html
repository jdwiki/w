açısal.eşittir #

“angular.equals” işlevi, 2 nesnenin veya değerin eşit olup olmadığını karşılaştırır ve belirler, “angular.equals”, derin bir karşılaştırma yapar ve yalnızca aşağıdaki koşullardan en az 1 tanesi karşılanırsa doğru değerini döndürür.

açısal.eşittir(değer1, değer2)

  1. Nesneler veya değerler === karşılaştırmasını geçerse
  2. Her iki nesne veya değer aynı türdeyse ve tüm özellikleri ‘angular.equals’ kullanılarak eşitse
  3. Her iki değer de “NaN” değerine eşittir
  4. Her iki değer de aynı normal ifadenin sonucunu temsil eder.

Bu işlev, nesneleri veya dizileri yalnızca referanslar yerine değerlerine veya sonuçlarına göre derinlemesine karşılaştırmanız gerektiğinde yararlıdır.

Örnekler

angular.equals(1, 1) // true
angular.equals(1, 2) // false
angular.equals({}, {}) // true, note that {}==={} is false
angular.equals({a: 1}, {a: 1}) // true
angular.equals({a: 1}, {a: 2}) // false
angular.equals(NaN, NaN) // true

açısal.toJson #

‘angular.toJson’ işlevi bir nesneyi alır ve onu JSON formatlı bir dizeye seri hale getirir.

Yerel JSON.stringify işlevinin aksine, bu işlev $$ ile başlayan tüm özellikleri kaldırır (açısal genellikle dahili özelliklerin önüne $$ koyduğu için)

angular.toJson(object)

Bir ağdan geçmeden önce verilerin serileştirilmesi gerektiğinden, bu işlev, iletmek istediğiniz herhangi bir veriyi JSON’a dönüştürmek için kullanışlıdır.

Bu işlev, “.toString” yöntemine benzer şekilde çalıştığı için hata ayıklama için de kullanışlıdır.

Örnekler:

angular.toJson({name: "barf", occupation: "mog", $$somebizzareproperty: 42})
// "{"name":"barf","occupation":"mog"}"
angular.toJson(42)
// "42"
angular.toJson([1, "2", 3, "4"])
// "[1,"2",3,"4"]"
var fn = function(value) {return value}
angular.toJson(fn)
// undefined, functions have no representation in JSON

açısal.kopya #

“angular.copy” işlevi bir nesneyi, diziyi veya değeri alır ve bunun derin bir kopyasını oluşturur.

açısal.kopya()

Örnek:

nesneler:

let obj = {name: "vespa", occupation: "princess"};
let cpy = angular.copy(obj);
cpy.name = "yogurt"
// obj = {name: "vespa", occupation: "princess"}
// cpy = {name: "yogurt", occupation: "princess"}

diziler:

var w = [a, [b, [c, [d]]]];
var q = angular.copy(w);
// q = [a, [b, [c, [d]]]]

Yukarıdaki örnekte, “.equals” değere göre eşitliği test ettiği için “angular.equals(w, q)” doğru olarak değerlendirilecektir. ancak ‘w === q’ yanlış olarak değerlendirilecektir çünkü nesneler ve diziler arasında kesin karşılaştırma referansla yapılır.

açısal.isString #

‘angular.isString’ işlevi, kendisine verilen nesne veya değer ‘string’ türündeyse true değerini döndürür.

angular.isString(değer1)

Örnekler

angular.isString("hello") // true
angular.isString([1, 2]) // false
angular.isString(42) // false

Bu, gerçekleştirmenin eşdeğeridir

typeof someValue === "string"

açısal.isArray #

“angular.isArray” işlevi, yalnızca kendisine iletilen nesne veya değerin “Array” türünde olması durumunda true değerini döndürür.

açısal.isArray(değer);

Örnekler

angular.isArray([]) // true
angular.isArray([2, 3]) // true
angular.isArray({}) // false
angular.isArray(17) // false

Bu eşdeğerdir

Array.isArray(someValue)

açısal.merge #

angular.merge işlevi, hedef nesneyi derinlemesine genişletmek için kaynak nesneden tüm numaralandırılabilir özellikleri alır.

İşlev, artık genişletilmiş hedef nesneye bir başvuru döndürür

angular.merge(hedef, kaynak)

Örnekler

angular.merge({}, {}) // {} 
angular.merge({name: "king roland"}, {password: "12345"})
// {name: "king roland", password: "12345"}
angular.merge({a: 1}, [4, 5, 6]) // {0: 4, 1: 5, 2: 6, a: 1}
angular.merge({a: 1}, {b: {c: {d: 2}}}) // {"a":1,"b":{"c":{"d":2}}}

angular.isDefined ve angular.isUndefined #

‘angular.isDefined’ işlevi, bir değeri tanımlanmışsa test eder

angular.isDefined(someValue)

Bu, gerçekleştirmenin eşdeğeridir

value !== undefined; // will evaluate to true is value is defined

Örnekler

angular.isDefined(42) // true
angular.isDefined([1, 2]) // true
angular.isDefined(undefined) // false
angular.isDefined(null) // true

“angular.isUndefined” işlevi, bir değerin tanımsız olup olmadığını test eder (“angular.isDefined” işlevinin tam tersidir)

angular.isUndefined(someValue)

Bu, gerçekleştirmenin eşdeğeridir

value === undefined; // will evaluate to true is value is undefined

Ya da sadece

!angular.isDefined(value)

Örnekler

angular.isUndefined(42) // false
angular.isUndefined(undefined) // true

açısal.isDate #

“angular.isDate” işlevi, yalnızca kendisine iletilen nesnenin Date türünde olması durumunda true değerini döndürür.

angular.isDate(değer)

Örnekler

angular.isDate("lone star") // false
angular.isDate(new Date()) // true

angular.isFunction #

‘angular.isFunction’ işlevi, yalnızca ve ancak iletilen değerin bir işleve referans olması durumunda belirler ve true değerini döndürür.

İşlev, artık genişletilmiş hedef nesneye bir başvuru döndürür

angular.isFunction(fn)

Örnekler

var onClick = function(e) {return e};
angular.isFunction(onClick); // true

var someArray = ["pizza", "the", "hut"];
angular.isFunction(someArray ); // false

açısal.noop #

‘angular.noop’ hiçbir işlem yapmayan bir işlevdir, hiçbir şey yapmayacak bir işlev argümanı sağlamanız gerektiğinde ‘angular.noop’ iletirsiniz.

açısal.noop()

‘angular.noop’ için yaygın bir kullanım, bir işleve bir işlevden başka bir şey iletildiğinde hata verecek olan bir işleve boş bir geri çağırma sağlamak olabilir.

Örnek:

$scope.onSomeChange = function(model, callback) {
    updateTheModel(model);
    if (angular.isFunction(callback)) {
        callback();
    } else {
        throw new Error("error: callback is not a function!");
    }
};

$scope.onSomeChange(42, function() {console.log("hello callback")});
// will update the model and print 'hello callback'
$scope.onSomeChange(42, angular.noop);
// will update the model

Ek örnekler:

angular.noop() // undefined
angular.isFunction(angular.noop) // true

angular.isElement #

‘angular.isElement’, kendisine iletilen argüman bir DOM Öğesi veya jQuery ile sarılmış bir Öğe ise true değerini döndürür.

angular.isElement(eleman)

Bu işlev, bu şekilde işlenmeden önce iletilen bir bağımsız değişkenin bir öğe olup olmadığını kontrol etmek için kullanışlıdır.

Örnekler:

angular.isElement(document.querySelector("body"))
// true
angular.isElement(document.querySelector("#some_id"))
// false if "some_id" is not using as an id inside the selected DOM
angular.isElement("<div></div>")
// false

açısal.isNumara #

“angular.isNumber” işlevi, yalnızca kendisine iletilen nesne veya değerin Number türünde olması durumunda true değerini döndürür, buna +Infinity, -Infinity ve NaN dahildir

angular.isNumber(değer)

Bu işlev, aşağıdaki gibi bir tür zorlamaya neden olmaz.

"23" == 23 // true

Örnekler

angular.isNumber("23") // false
angular.isNumber(23) // true
angular.isNumber(NaN) // true
angular.isNumber(Infinity) // true

Bu işlev, aşağıdaki gibi bir tür zorlamaya neden olmaz.

"23" == 23 // true

açısal.fromJson #

“angular.fromJson” işlevi, geçerli bir JSON dizesini seri durumdan çıkarır ve bir Nesne veya Dizi döndürür.

angular.fromJson(string|nesne)

Bu işlevin yalnızca dizelerle sınırlı olmadığını, kendisine iletilen herhangi bir nesnenin bir temsilini çıkaracağını unutmayın.

Örnekler:

angular.fromJson("{\"yogurt\": \"strawberries\"}")
// Object {yogurt: "strawberries"}
angular.fromJson('{jam: "raspberries"}')
// will throw an exception as the string is not a valid JSON
angular.fromJson(this)
// Window {external: Object, chrome: Object, _gaq: Y, angular: Object, ng339: 3…}
angular.fromJson([1, 2])
// [1, 2]
typeof angular.fromJson(new Date())
// "object"

açısal.isObject #

“angular.isObject”, yalnızca kendisine iletilen argüman bir nesneyse true döndürür, bu işlev ayrıca bir Array için true döndürür ve “typeof null” “object” olsa bile “null” için false döndürür.

angular.isObject(değer)

Bu işlev, işlenecek tanımlı bir nesneye ihtiyacınız olduğunda tür denetimi için kullanışlıdır.

Örnekler:

angular.isObject({name: "skroob", job: "president"})
// true
angular.isObject(null)
// false
angular.isObject([null])
// true
angular.isObject(new Date())
// true
angular.isObject(undefined)
// false

açısal.kimlik #

“angular.identity” işlevi, kendisine iletilen ilk bağımsız değişkeni döndürür.

angular.identity(argüman)

Bu işlev işlevsel programlama için kullanışlıdır, beklenen bir işlevin geçilmemesi durumunda bu işlevi varsayılan olarak sağlayabilirsiniz.

Örnekler:

angular.identity(42) // 42

var mutate = function(fn, num) {
    return angular.isFunction(fn) ? fn(num) : angular.identity(num)
}

mutate(function(value) {return value-7}, 42) // 35
mutate(null, 42) // 42
mutate("mount. rushmore", 42) // 42

açısal.forEach #

‘angular.forEach’ bir nesneyi ve yineleyici işlevini kabul eder. Daha sonra nesnenin her bir numaralandırılabilir özelliği/değeri üzerinde yineleyici işlevini çalıştırır. Bu işlev dizilerde de çalışır.

‘Array.prototype.forEach’ öğesinin JS sürümü gibi, işlev devralınan özellikler (prototip özellikleri) üzerinde yineleme yapmaz, ancak işlev bir “boş” veya “tanımsız” bir değeri işlemeye çalışmaz ve yalnızca onu döndürür.

angular.forEach(nesne, fonksiyon(değer, anahtar) { // fonksiyon});

Örnekler:

angular.forEach({"a": 12, "b": 34}, (value, key) => console.log("key: " + key + ", value: " + value))
// key: a, value: 12
// key: b, value: 34
angular.forEach([2, 4, 6, 8, 10], (value, key) => console.log(key))
// will print the array indices: 1, 2, 3, 4, 5
angular.forEach([2, 4, 6, 8, 10], (value, key) => console.log(value))
// will print the array values: 2, 4, 6, 7, 10
angular.forEach(undefined, (value, key) => console.log("key: " + key + ", value: " + value))
// undefined
