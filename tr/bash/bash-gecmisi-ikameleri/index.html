Hızlı referans #

Tarihle etkileşim #

# List all previous commands
history

# Clear the history, useful if you entered a password by accident
history -c

Etkinlik tanımlayıcıları #

# Expands to line n of bash history
!n

# Expands to last command
!!

# Expands to last command starting with "text"
!text

# Expands to last command containing "text"
!?text

# Expands to command n lines ago
!-n

# Expands to last command with first occurrence of "foo" replaced by "bar"
^foo^bar^

# Expands to the current command
!#

Kelime tanımlayıcıları #

Bunlar, atıfta bulundukları olay tanımlayıcısından : ile ayrılır. İşaretçi sözcüğü bir sayı ile başlamıyorsa iki nokta üst üste atlanabilir: !^, !:^ ile aynıdır.

# Expands to the first argument of the most recent command
!^

# Expands to the last argument of the most recent command (short for !!:$)
!$

# Expands to the third argument of the most recent command
!:3

# Expands to arguments x through y (inclusive) of the last command
# x and y can be numbers or the anchor characters ^ $
!:x-y

# Expands to all words of the last command except the 0th
# Equivalent to :^-$
!*

Değiştiriciler #

Bunlar, önceki olayı veya kelime tanımlayıcısını değiştirir.

# Replacement in the expansion using sed syntax
# Allows flags before the s and alternate separators
:s/foo/bar/ #substitutes bar for first occurrence of foo
:gs|foo|bar| #substitutes bar for all foo

# Remove leading path from last argument ("tail")
:t

# Remove trailing path from last argument ("head")
:h

# Remove file extension from last argument
:r

“HISTCONTROL” Bash değişkeni “ignorespace” veya “ignoreboth” içeriyorsa (veya alternatif olarak “HISTIGNORE”, “[ ]*” kalıbını içeriyorsa, komutlarınızın başına bir karakter ekleyerek komutlarınızın Bash geçmişinde saklanmasını önleyebilirsiniz. Uzay:

# This command won't be saved in the history
 foo

# This command will be saved
bar

Sudo ile önceki komutu tekrarlayın #

$ apt-get install r-base
E: Could not open lock file /var/lib/dpkg/lock - open (13: Permission denied)
E: Unable to lock the administration directory (/var/lib/dpkg/), are you root?
$ sudo !!
sudo apt-get install r-base
[sudo] password for <user>:

Komut geçmişinde desene göre arama yapın #

kontrolr‘ye basın ve bir kalıp yazın.

Örneğin, yakın zamanda man 5 crontab çalıştırdıysanız, “crontab” yazmaya *başlayarak hızlı bir şekilde bulabilirsiniz. İstem şu şekilde değişecektir:

(reverse-i-search)`cr': man 5 crontab

`cr' şimdiye kadar yazdığım dize var. Bu artımlı bir aramadır, bu nedenle yazmaya devam ettikçe arama sonucu, kalıbı içeren en son komutla eşleşecek şekilde güncellenir.

Çalıştırmadan önce eşleşen komutu düzenlemek için sol veya sağ ok tuşlarına basın, veya komutu çalıştırmak için enter tuşuna basın.

Varsayılan olarak arama, kalıpla eşleşen en son yürütülen komutu bulur. Geçmişte daha geriye gitmek için tekrar kontrolr‘ye basın. İstediğiniz komutu bulana kadar tekrar tekrar basabilirsiniz.

!#:N ile yeni oluşturulan dizine geç #

$ mkdir backup_download_directory && cd !#:1
mkdir backup_download_directory && cd backup_download_directory

Bu, geçerli komutun Nth argümanının yerini alacaktır. Örnekte !#:1 ilk argümanla, yani backup_download_directory ile değiştirilmiştir.

!$‘ı kullanma #

Komut satırını kullanırken tekrarı azaltmak için !$ kullanabilirsiniz:

$ echo ping
ping
$ echo !$
ping

Ayrıca tekrar üzerine inşa edebilirsiniz

$ echo !$ pong
ping pong
$ echo !$, a great game
pong, a great game

Son örnekte, önceki komuta iletilen son argümanın ‘pong’ olması nedeniyle ‘ping pong, harika bir oyun’ almadığımıza dikkat edin, tırnak işaretleri ekleyerek bu gibi sorunları önleyebiliriz. Örnekle devam edersek, son argümanımız “oyun” idi:

$ echo "it is !$ time"
it is game time
$ echo "hooray, !$!"
hooray, it is game time!

Önceki komutu bir ikame ile tekrarlayın #

$ mplayer Lecture_video_part1.mkv
$ ^1^2^
mplayer Lecture_video_part2.mkv

Bu komut, daha önce yürütülen komutta ‘1’ yerine ‘2’ yazacaktır. Yalnızca dizenin ilk oluşumunu değiştirir ve !!:s/1/2/ ile eşdeğerdir.

tüm oluşumları değiştirmek istiyorsanız, !!:gs/1/2/ veya !!:as/1/2/ kullanmanız gerekir.
