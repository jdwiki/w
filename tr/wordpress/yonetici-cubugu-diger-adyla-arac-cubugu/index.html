Yönetici Çubuğu (diğer adıyla Araç Çubuğu)

  • Yönetici Araç Çubuğunu Yöneticiler dışındaki herkesten kaldırma
  • Filtreleri kullanarak Yönetici araç çubuğunu kaldırma
  • WordPress Logosunu Yönetici Çubuğundan Kaldırma
  • Yönetici giriş sayfasına özel logonuzu ve özel bağlantınızı ekleyin

Bu sayfada

  • Yönetici Araç Çubuğunu Yöneticiler dışındaki herkesten kaldırma
  • Filtreleri kullanarak Yönetici araç çubuğunu kaldırma
  • WordPress Logosunu Yönetici Çubuğundan Kaldırma

WordPress Yönetici Araç Çubuğu, sürüm 3.1’de eklenmiştir ve genel yönetim görevlerine ve ayrıca kullanıcı profiline ve diğer WordPress bilgilerine bağlantılar içerir. Ancak, birçok site sahibi, araç çubuğunun varsayılan olarak tüm oturum açmış kullanıcılara gösterilmesinden hoşlanmaz ve/veya buna kendi seçeneklerini eklemek ister.

Yönetici Araç Çubuğunu Yöneticiler dışındaki herkesten kaldırma #

Yönetici kullanıcı seviyesi dışındaki herkesten kaldırmak için aşağıdaki kodu functions.php dosyasına ekleyin:

add_action('after_setup_theme', 'no_admin_bar');

    function no_admin_bar() {
        if (!current_user_can('administrator') && !is_admin()) {
          show_admin_bar(false);
        }
    }

Filtreleri kullanarak Yönetici araç çubuğunu kaldırma #

Yönetici çubuğunu gizlemenin başka bir yolu da eklemektir.

if ( !current_user_can( 'manage_options' ) ) {
    add_filter( 'show_admin_bar', '__return_false' , 1000 );
}

Ayarlar sayfasına erişim yetkisi olmayan kullanıcılar yönetici çubuğunu göremezler.

WordPress Logosunu Yönetici Çubuğundan Kaldırma #

Geliştiriciler, öğeleri WordPress yönetici çubuğundan veya araç çubuğundan kaldırmak için admin_bar_menu eylemini kullanabilir.

add_action('admin_bar_menu', 'remove_wp_logo_from_admin_bar', 999);
function remove_wp_logo_from_admin_bar( $wp_admin_bar ) {
    $wp_admin_bar->remove_node('wp-logo');
}

Yukarıdaki kod, WordPress logosunu yönetici çubuğundan kaldırır. Tek yapmanız gereken kodu function.php dosyanızın içine yapıştırmak.

remove_node yöntemine iletilen parametre, kaldırmak istediğiniz düğümün kimliğidir. Kimlikler, üzerinde bir Araç Çubuğu bulunan WordPress sayfasının HTML kaynak kodunda bulunabilir. Örneğin, Araç Çubuğunda soldaki WordPress Logosu için li öğesinin kimliği “wp-admin-bar-wp-logo” şeklindedir:

<li id="wp-admin-bar-wp-logo" class="menupop"> … </li>

Düğümün kimliğini almak için li’nin kimliğinden “wp-admin-bar-” öğesini kaldırın. Bu örnekte düğüm kimliği “wp-logo” dur.

Yönetici çubuğunuzdaki çeşitli öğelerin veya düğümlerin düğüm kimliklerini bulmak için tarayıcı denetçisi araçlarını kullanabilirsiniz.

Yönetici giriş sayfasına özel logonuzu ve özel bağlantınızı ekleyin #

Kendi logonuzu eklemek için aşağıdaki kancaları ve varsayılan wordpress logosunu değiştirmek için bağlantıyı ekleyebilirsiniz.

Özel logo eklemek için

function custom_login_logo() {
echo '<style type="text/css">
h1 a { background-image: url('.get_bloginfo('template_directory').'/images/custom-logo.png) !important; background-size : 100% !important; width: 300px !important; height : 100px !important;}
</style>';
}
add_action('login_head', 'custom_login_logo');

Özel logo bağlantısı eklemek için

add_filter( 'login_headerurl', 'custom_loginlogo_url' );
function custom_loginlogo_url($url) {
    return home_url();
}
