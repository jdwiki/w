add_submenu_page()

  • Sözdizimi
  • Parametreler
  • “Alt Menü Sayfası"nı “Araçlar"ın alt sayfası olarak gezinme çubuğuna ekleme

Bu sayfada

  • Sözdizimi
  • Parametreler
  • “Alt Menü Sayfası"nı “Araçlar"ın alt sayfası olarak gezinme çubuğuna ekleme

Bu işlev, yönetici panelleri gezinme çubuğundaki mevcut bir öğeye bir alt öğe eklemek içindir.

Sözdizimi #

  • add_submenu_page( $parent_slug, $page_title, $menu_title, $yetenek, $menu_slug, $function )

Parametreler #

parametre Ayrıntılar
$parent_slug (string) Üst menünün bilgi adı (veya standart bir WordPress yönetici sayfasının dosya adı).
$sayfa_başlığı (string) Menü seçildiğinde sayfanın başlık etiketlerinde görüntülenecek metin.
$menü_başlığı (string) Menü için kullanılacak metin.
$yetenek (string) Bu menünün kullanıcıya gösterilmesi için gereken yetenek.
$menu_slug (string) Bu menüye atıfta bulunulacak bilgi adı (bu menü için benzersiz olmalıdır).
$fonksiyon (çağrılabilir) (İsteğe bağlı) Bu sayfanın içeriğini çıkarmak için çağrılacak işlev.

$parent_slug için bir sümüklü böcek listesi burada

  • Pano: “index.php”
  • Gönderiler: ’edit.php'
  • Medya: ‘upload.php’
  • Sayfalar: ’edit.php?post_type=page'
  • Yorumlar: ’edit-comments.php'
  • Özel Gönderi Türleri: ’edit.php?post_type=your_post_type'
  • Görünüm: “themes.php”
  • Eklentiler: ‘plugins.php’
  • Kullanıcılar: ‘users.php’
  • Araçlar: “tools.php”
  • Ayarlar: ‘options-general.php’
  • Ağ Ayarları: ‘settings.php’

“Alt Menü Sayfası"nı “Araçlar"ın alt sayfası olarak gezinme çubuğuna ekleme #

Kod

add_action('admin_menu', 'register_my_custom_submenu_page');
 
function register_my_custom_submenu_page() {
    add_submenu_page(
        'tools.php',
        'Submenu Page',
        'My Custom Submenu Page',
        'manage_options',
        'my-custom-submenu-page',
        'my_custom_submenu_page_content' );
}
 
function my_custom_submenu_page_content() {
    echo '<div class="wrap">';
        echo '<h2>Page Title</h2>';
    echo '</div>';
}

Çıktı

Açıklama

Kodda, `register_my_custom_submenu_page’ adında bir fonksiyon oluşturduk ve öğeyi Araçlar sayfası olan tools.php’nin bir çocuğu olarak navbar’a eklemek için ‘add_submenu_page’ kullandık.

Geçirdiğimiz argümanlar hakkında bilgi edinmek için lütfen bu sayfadaki parametreler bölümünü kontrol edin. Ardından, register_my_custom_submenu_page fonksiyonumuzu çalıştırmak için add_action kullandık. Son olarak, sayfadaki içerikleri görüntülemek için my_custom_submenu_page_content fonksiyonunu oluşturduk.
