  • dizi = yeni Dizi(değer, değer, …)
  • dizi = Array.of(değer, değer, …)
  • dizi = Array.from(arrayLike)

Özet: JavaScript’teki diziler, oldukça basit, gelişmiş bir prototipe sahip değiştirilmiş “Nesne” örnekleridir ve listeyle ilgili çeşitli görevleri gerçekleştirebilir. ECMAScript 1st Edition’a eklendiler ve diğer prototip yöntemler ECMAScript 5.1 Edition’a geldi.

Uyarı: new Array() yapıcısında n adında bir sayısal parametre belirtilirse, o zaman n elemanlı bir dizi bildirir, değeri olan 1 elemanlı bir dizi ilan etmez n!

console.log(new Array(53)); // This array has 53 'undefined' elements!

Bununla birlikte, bir dizi bildirirken her zaman [] kullanmalısınız:

console.log([53]); // Much better!

Dizi Benzeri Nesneler nelerdir? #

JavaScript, Uzunluk özelliğine sahip Dizilerin Nesne temsilleri olan “Dizi benzeri Nesnelere” sahiptir. Örneğin:

var realArray = ['a', 'b', 'c'];
var arrayLike = {
  0: 'a',
  1: 'b',
  2: 'c',
  length: 3
};

Dizi benzeri Nesnelerin yaygın örnekleri, işlevlerdeki arguments nesnesi ve [document.getElementsByTagName] gibi yöntemlerden döndürülen HTMLCollection veya NodeList nesneleridir. 4 veya document.querySelectorAll.

Ancak, Diziler ve Dizi benzeri Nesneler arasındaki temel farklardan biri, Dizi benzeri nesnelerin ‘Array.prototype’ yerine ‘Object.prototype’ öğesinden miras almasıdır. Bu, Dizi benzeri Nesnelerin forEach(), push(), [map()] gibi ortak Dizi prototip yöntemlerine erişemeyeceği anlamına gelir. 11, filter() ve dilim():

var parent = document.getElementById('myDropdown');
var desiredOption = parent.querySelector('option[value="desired"]');
var domList = parent.children;

domList.indexOf(desiredOption); // Error! indexOf is not defined.
domList.forEach(function() { 
  arguments.map(/* Stuff here */) // Error! map is not defined.
}); // Error! forEach is not defined.

function func() {
  console.log(arguments);
}
func(1, 2, 3);   // → [1, 2, 3]

Değerleri azaltma #

‘reduce()’ yöntemi, bir akümülatöre ve dizinin her değerine (soldan sağa) tek bir değere indirgemek için bir işlev uygular.

Dizi Toplamı #

Bu yöntem, bir dizinin tüm değerlerini tek bir değerde toplamak için kullanılabilir:

[1, 2, 3, 4].reduce(function(a, b) {
  return a + b;
});
// → 10

İsteğe bağlı ikinci parametre reduce() öğesine geçirilebilir. Değeri, geri aramaya yapılan ilk çağrı (“işlev(a, b)” olarak belirtilir) için ilk argüman (“a” olarak belirtilir) olarak kullanılacaktır.

[2].reduce(function(a, b) {
  console.log(a, b); // prints: 1 2
  return a + b;
}, 1);
// → 3
son sürüm

Küçültme Kullanarak Harita #

initial value parametresini kullanmanın başka bir örneği olarak, bir dizi öğe üzerinde bir işlevi çağırma ve sonuçları yeni bir dizide döndürme görevini düşünün. Diziler sıradan değerler olduğundan ve liste birleştirme sıradan bir işlev olduğundan, aşağıdaki örnekte gösterildiği gibi bir listeyi toplamak için ‘reduce’ kullanabiliriz:

function map(list, fn) {
  return list.reduce(function(newList, item) {
    return newList.concat(fn(item));
  }, []);
}

// Usage:
map([1, 2, 3], function(n) { return n * n; });
// → [1, 4, 9]

Bunun yalnızca örnekleme amaçlı olduğunu unutmayın (ilk değer parametresinin), liste dönüşümleriyle çalışmak için yerel ‘haritayı’ kullanın (ayrıntılar için Eşleme değerleri’ne bakın).

Eşleme değerleri #

Mevcut bir dizinin değerlerine dayalı olarak yeni bir dizi oluşturmak genellikle gereklidir.

Örneğin, bir dizi diziden bir dizi dizi uzunluğu oluşturmak için:

['one', 'two', 'three', 'four'].map(function(value, index, arr) {
  return value.length;
});
// → [3, 3, 5, 4]
son sürüm 
['one', 'two', 'three', 'four'].map(value => value.length);
// → [3, 3, 5, 4]
son sürüm

Bu örnekte, “map()” işlevine anonim bir işlev sağlanmıştır ve harita işlevi, dizideki her öğe için onu çağırarak aşağıdaki parametreleri bu sırayla sağlar:

  • Elemanın kendisi
  • Elemanın dizini (0, 1…)
  • Tüm dizi

Ek olarak, “map()”, eşleme işlevinde “this"in değerini ayarlamak için isteğe bağlı ikinci bir parametre sağlar. Yürütme ortamına bağlı olarak, “this"in varsayılan değeri değişebilir:

Bir tarayıcıda, “bu"nun varsayılan değeri her zaman “pencere"dir:

['one', 'two'].map(function(value, index, arr) {
  console.log(this); // window (the default value in browsers)
  return value.length;
});

Bunu, bunun gibi herhangi bir özel nesneyle değiştirebilirsiniz:

['one', 'two'].map(function(value, index, arr) {
  console.log(this); // Object { documentation: "randomObject" }
  return value.length;
}, {
  documentation: 'randomObject'
});

Dizileri Sıralama #

.sort() yöntemi, bir dizinin öğelerini sıralar. Varsayılan yöntem, diziyi Unicode kod noktalarına göre sıralayacaktır. Bir diziyi sayısal olarak sıralamak için “.sort()” yönteminin kendisine bir “compareFunction” iletilmesi gerekir.

Not:** .sort() yöntemi saf değildir. .sort(), diziyi yerinde sıralar, yani orijinal dizinin sıralanmış bir kopyasını oluşturmak yerine orijinal diziyi yeniden sıralar ve döndürür.

Varsayılan Sıralama

Diziyi UNICODE düzeninde sıralar.

['s', 't', 'a', 34, 'K', 'o', 'v', 'E', 'r', '2', '4', 'o', 'W', -1, '-4'].sort();

Sonuçlar:

[-1, '-4', '2', 34, '4', 'E', 'K', 'W', 'a', 'l', 'o', 'o', 'r', 's', 't', 'v']

Not:** Büyük harfler küçük harflerin üzerine taşınmıştır. Dizi alfabetik sırada değil ve sayılar sayısal sırada değil.

Alfabetik Sıralama

['s', 't', 'a', 'c', 'K', 'o', 'v', 'E', 'r', 'f', 'l', 'W', '2', '1'].sort((a, b) => {
  return a.localeCompare(b);        
});

Sonuçlar:

['1', '2', 'a', 'c', 'E', 'f', 'K', 'l', 'o', 'r', 's', 't', 'v', 'W']

Not:** Yukarıdaki sıralama, herhangi bir dizi öğesi bir dize değilse bir hata verir. Dizinin dize olmayan öğeler içerebileceğini biliyorsanız, aşağıdaki güvenli sürümü kullanın.

['s', 't', 'a', 'c', 'K', 1, 'v', 'E', 'r', 'f', 'l', 'o', 'W'].sort((a, b) => {
  return a.toString().localeCompare(b);      
});

["zebras", "dogs", "elephants", "penguins"].sort(function(a, b) {
  return b.length - a.length;
});

Sonuçlar

["elephants", "penguins", "zebras", "dogs"];

Yineleme #

Geleneksel bir “for” döngüsü #

Geleneksel bir “for” döngüsünün üç bileşeni vardır:

  1. Başlatma: görünüm bloğu ilk kez yürütülmeden önce yürütülür
  2. Koşul: döngü bloğu yürütülmeden önce her seferinde bir koşulu kontrol eder ve yanlışsa döngüden çıkar
  3. Sonradan düşünülen: döngü bloğu yürütüldükten sonra her seferinde gerçekleştirilir

Bu üç bileşen birbirinden bir ; sembolü ile ayrılır. Bu üç bileşenin her birinin içeriği isteğe bağlıdır; bu, aşağıdakilerin mümkün olan en düşük “for” döngüsü olduğu anlamına gelir:

for (;;) {
    // Do stuff
}

Elbette, bir if(condition === true) { break; eklemeniz gerekecek. } veya bir if(koşul === doğru) { dönüş; } çalışmasını durdurmak için bu for-döngüsünün içinde bir yere.

Genellikle, başlatma bir dizini bildirmek için kullanılır, koşul bu dizini minimum veya maksimum değerle karşılaştırmak için kullanılır ve sonraki düşünce dizini artırmak için kullanılır:

for (var i = 0, length = 10; i < length; i++) {
    console.log(i);
}

Bir dizide dolaşmak için geleneksel bir “for” döngüsünü kullanma #

Bir dizide dolaşmanın geleneksel yolu şudur:

for (var i = 0, length = myArray.length; i < length; i++) {
    console.log(myArray[i]);
}

Veya geriye doğru döngü yapmayı tercih ederseniz, şunu yaparsınız:

for (var i = myArray.length - 1; i > -1; i--) {
    console.log(myArray[i]);
}

Bununla birlikte, örneğin bunun gibi birçok varyasyon mümkündür:

for (var key = 0, value = myArray[key], length = myArray.length; key < length; value = myArray[++key]) {
    console.log(value);
}

… ya da bu …

var i = 0, length = myArray.length;
for (; i < length;) {
    console.log(myArray[i]);
    i++;
}

… ya da bu:

var key = 0, value;
for (; value = myArray[key++];){
    console.log(value);
}

Hangisi en iyi sonucu verirse, büyük ölçüde hem kişisel zevk hem de uyguladığınız özel kullanım durumu meselesidir.

Bu varyasyonların her birinin, çok çok eski olanlar da dahil olmak üzere tüm tarayıcılar tarafından desteklendiğini unutmayın!

Bir “while” döngüsü #

“for” döngüsünün bir alternatifi “while” döngüsüdür. Bir dizide dolaşmak için şunu yapabilirsiniz:

var key = 0;
while(value = myArray[key++]){
    console.log(value);
}

Geleneksel “for” döngüleri gibi, “while” döngüleri en eski tarayıcılar tarafından bile desteklenir.

Ayrıca, her while döngüsünün bir ‘for’ döngüsü olarak yeniden yazılabileceğini unutmayın. Örneğin, yukarıdaki “while” döngüsü, bu “for” döngüsüyle tamamen aynı şekilde davranır:

for(var key = 0; value = myArray[key++];){
    console.log(value);
}

için...içinde #

JavaScript’te bunu da yapabilirsiniz:

for (i in myArray) {
    console.log(myArray[i]);
}

Bununla birlikte, her durumda geleneksel bir “for” döngüsüyle aynı şekilde davranmadığından ve dikkate alınması gereken potansiyel yan etkiler bulunduğundan, bu dikkatli kullanılmalıdır. Daha fazla ayrıntı için https://stackoverflow.com/questions/500504/why-is-using-for-in-with-array-iteration-such-a-bad-idea konusuna bakın.

Array.prototype.forEach() #

.forEach(...) yöntemi, ES 5 ve üzeri sürümlerde bir seçenektir. Tüm modern tarayıcıların yanı sıra Internet Explorer 9 ve sonraki sürümleri tarafından desteklenir.

[1, 2, 3, 4].forEach(function(value, index, arr) {
  var twoValue = value * 2;
  console.log("2 * value is: %d", twoValue);
});
son sürüm

Geleneksel ‘for’ döngüsüyle karşılaştırıldığında, ‘.forEach()’ içindeki döngüden çıkamayız. Bu durumda, “for” döngüsünü kullanın veya aşağıda sunulan kısmi yinelemeyi kullanın.

JavaScript’in tüm sürümlerinde, geleneksel bir C stili “for” döngüsü kullanarak bir dizinin dizinleri arasında yineleme yapmak mümkündür.

var myArray = [1, 2, 3, 4];
for(var i = 0; i < myArray.length; ++i) {
  var twoValue = myArray[i] * 2;
  console.log("2 * value is: %d", twoValue);
}

“while” döngüsünü kullanmak da mümkündür:

var myArray = [1, 2, 3, 4],
    i = 0, sum = 0;
while(i++ < myArray.length) {
  sum += i;
}
console.log(sum);

Array.prototype.every #

ES5’ten beri, bir dizinin bir kısmı üzerinde yineleme yapmak istiyorsanız, biz ‘false’ döndürene kadar yinelenen Array.prototype.every kullanabilirsiniz:

// [].every() stops once it finds a false result
// thus, this iteration will stop on value 7 (since 7 % 2 !== 0)
[2, 4, 7, 9].every(function(value, index, arr) {
  console.log(value);
  return value % 2 === 0; // iterate until an odd number is found
});
son sürüm

Herhangi bir JavaScript sürümünde eşdeğerdir:

var arr = [2, 4, 7, 9];
for (var i = 0; i < arr.length && (arr[i] % 2 !== 0); i++) { // iterate until an odd number is found
  console.log(arr[i]);
}

  console.log(value);

son sürüm

Dizi karşılaştırma #

Basit dizi karşılaştırması için JSON stringify’ı kullanabilir ve çıktı dizelerini karşılaştırabilirsiniz:

JSON.stringify(array1) === JSON.stringify(array2)

Not: bunun yalnızca her iki nesne de JSON serileştirilebilirse ve döngüsel referanslar içermiyorsa işe yarayacaktır. ‘TypeError: Dairesel yapıyı JSON’a dönüştürme’ hatası verebilir.

Dizileri karşılaştırmak için özyinelemeli bir işlev kullanabilirsiniz.

function compareArrays(array1, array2) { 
  var i, isA1, isA2;
  isA1 = Array.isArray(array1);
  isA2 = Array.isArray(array2);
  
  if (isA1 !== isA2) { // is one an array and the other not?
    return false;      // yes then can not be the same
  }
  if (! (isA1 && isA2)) {      // Are both not arrays 
    return array1 === array2;  // return strict equality
  }
  if (array1.length !== array2.length) { // if lengths differ then can not be the same
    return false;
  }
  // iterate arrays and compare them
  for (i = 0; i < array1.length; i += 1) {
    if (!compareArrays(array1[i], array2[i])) { // Do items compare recursively
      return false;
    }           
  }
  return true; // must be equal
}

UYARI: Yukarıdaki işlevi kullanmak tehlikelidir ve dizinin döngüsel referansları (kendisine referans içeren bir diziye referansı) olma ihtimalinden şüpheleniyorsanız bir ‘dene’ ‘yakala’ya sarılmalıdır.

a = [0] ;
a[1] = a;
b = [0, a]; 
compareArrays(a, b); // throws RangeError: Maximum call stack size exceeded

Not:** İşlev, dizi olmayan öğeleri karşılaştırmak için === katı eşitlik operatörünü kullanır {a: 0} === {a: 0} false

Yayılma operatörü #

ES6 ile, tek tek öğeleri virgülle ayrılmış bir sözdiziminde ayırmak için formaları kullanabilirsiniz:

let arr = [1, 2, 3, ...[4, 5, 6]];  // [1, 2, 3, 4, 5, 6]

// in ES < 6, the operations above are equivalent to
arr = [1, 2, 3];
arr.push(4, 5, 6);

Yayılma operatörü ayrıca, her bir karakteri yeni bir dize öğesine ayırarak dizeler üzerinde de hareket eder. Bu nedenle, bunları tamsayılara dönüştürmek için bir dizi işlevi kullanarak, yukarıda oluşturulan dizi aşağıdakine eşdeğerdir:

let arr = [1, 2, 3, ...[..."456"].map(x=>parseInt(x))]; // [1, 2, 3, 4, 5, 6]

Veya tek bir dize kullanarak bu, şu şekilde basitleştirilebilir:

let arr = [..."123456"].map(x=>parseInt(x)); // [1, 2, 3, 4, 5, 6]

Eşleme gerçekleştirilmezse, o zaman:

let arr = [..."123456"]; // ["1", "2", "3", "4", "5", "6"]

Spread operatörü ayrıca argümanları bir fonksiyona yaymak için de kullanılabilir:

function myFunction(a, b, c) { }
let args = [0, 1, 2];

myFunction(...args);

// in ES < 6, this would be equivalent to:
myFunction.apply(null, args);

Bir nesnenin Dizi olup olmadığını kontrol etme #

‘Array.isArray(obj)’, nesne bir ‘Array’ ise ’true’, aksi takdirde ‘false’ değerini döndürür.

Array.isArray([])           // true

Array.isArray(1)            // false

Çoğu durumda, bir nesnenin bir “Array” olup olmadığını kontrol etmek için “instanceof” seçeneğini kullanabilirsiniz.

[] instanceof Array; // true
{} instanceof Array; // false

“Array.isArray”, dizinin prototipi değiştirilse bile yine “true” döndüreceği ve dizi olmayan bir prototip değiştirildiğinde “false” döndüreceği için bir “instanceof” denetimi kullanmaya göre bir avantaja sahiptir. “Dizi” prototipi.

var arr = [];
Object.setPrototypeOf(arr, null);
Array.isArray(arr);   // true
arr instanceof Array; // false

Belirli bir dizindeki diziye bir öğe ekle #

Basit öğe ekleme, Array.prototype.splice yöntemiyle yapılabilir:

arr.splice(index, 0, item);

Birden çok argüman ve zincirleme desteği ile daha gelişmiş varyant:

/* Syntax:
   array.insert(index, value1, value2, ..., valueN) */

Array.prototype.insert = function(index) {
  this.splice.apply(this, [index, 0].concat(
    Array.prototype.slice.call(arguments, 1)));
  return this;
};

["a", "b", "c", "d"].insert(2, "X", "Y", "Z").slice(1, 6);  // ["b", "X", "Y", "Z", "c"]

Ve dizi tipi argümanlar birleştirme ve zincirleme desteği ile:

/* Syntax:
   array.insert(index, value1, value2, ..., valueN) */

Array.prototype.insert = function(index) {
  index = Math.min(index, this.length);
  arguments.length > 1
    && this.splice.apply(this, [index, 0].concat([].pop.call(arguments)))
    && this.insert.apply(this, arguments);
  return this;
};

["a", "b", "c", "d"].insert(2, "V", ["W", "X", "Y"], "Z").join("-");  // "a-b-V-W-X-Y-Z-c-d"

Bir dizideki nesneleri yeniden biçimlendirmek için haritayı kullanma #

Array.prototype.map(): Orijinal dizideki her öğede sağlanan bir işlevi çağırmanın sonuçlarını içeren bir yeni dizi döndürür.

Aşağıdaki kod örneği, bir dizi kişiyi alır ve ‘fullName’ özelliğine sahip kişileri içeren yeni bir dizi oluşturur.

var personsArray = [

    id: 2,
    firstName: "Kaylee",
    lastName: "Frye"
  }, {
    id: 3,
    firstName: "Jayne",
    lastName: "Cobb"
  }
];

// Returns a new array of objects made up of full names.
var reformatPersons = function(persons) {
  return persons.map(function(person) {
    // create a new object to store full name.
    var newObj = {};
    newObj["fullName"] = person.firstName + " " + person.lastName;

    // return our new object.
    return newObj;
  });
};
