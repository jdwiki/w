console.log()‘u kullanma #

Giriiş #

Tüm modern web tarayıcıları, NodeJ’ler ve hemen hemen tüm diğer JavaScript ortamları, bir dizi günlük kaydı yöntemi kullanarak bir konsola mesaj yazmayı destekler. Bu yöntemlerden en yaygın olanı console.log() yöntemidir.

Bir tarayıcı ortamında, “console.log()” işlevi ağırlıklı olarak hata ayıklama amacıyla kullanılır.

Başlarken #

JavaScript Konsolunu tarayıcınızda açın, aşağıdakini yazın ve Enter’a basın :

console.log("Hello, World!");

Bu, aşağıdakileri konsola kaydeder:

Yukarıdaki örnekte, ‘console.log()’ işlevi konsola ‘Merhaba, Dünya!’ yazdırır ve ’tanımsız’ döndürür (yukarıda konsol çıktı penceresinde gösterilmiştir). Bunun nedeni, ‘console.log()’ öğesinin açık bir dönüş değerine sahip olmamasıdır.

Değişkenleri günlüğe kaydetme #

console.log() herhangi bir türdeki değişkenleri günlüğe kaydetmek için kullanılabilir; sadece dizeler değil. Konsolda görüntülenmesini istediğiniz değişkeni girmeniz yeterlidir, örneğin:

var foo = "bar";
console.log(foo);

Bu, aşağıdakileri konsola kaydeder:

İki veya daha fazla değeri kaydetmek istiyorsanız, bunları virgülle ayırmanız yeterlidir. Birleştirme sırasında her argüman arasına otomatik olarak boşluk eklenir:

var thisVar = 'first value';
var thatVar = 'second value';
console.log("thisVar:", thisVar, "and thatVar:", thatVar);

Bu, aşağıdakileri konsola kaydeder:

Yer tutucular #

“console.log()“u yer tutucularla birlikte kullanabilirsiniz:

var greet = "Hello", who = "World";
console.log("%s, %s!", greet, who);

Günlük Nesneleri #

Aşağıda bir nesneyi kaydetmenin sonucunu görüyoruz. Bu genellikle API çağrılarından gelen JSON yanıtlarını günlüğe kaydetmek için kullanışlıdır.

console.log({
    'Email': '', 
    'Groups': {},
    'Id': 33,
    'IsHiddenInUI': false,
    'IsSiteAdmin': false,
    'LoginName': 'i:0#.w|virtualdomain\\user2',
    'PrincipalType': 1,
    'Title': 'user2'
});

DOM API’sini kullanma #

DOM Document Object Model anlamına gelir. XML ve HTML gibi yapılandırılmış belgelerin nesne yönelimli bir temsilidir.

Bir “Element"in “textContent” özelliğini ayarlamak, bir web sayfasında metin çıktısı almanın bir yoludur.

Örneğin, aşağıdaki HTML etiketini göz önünde bulundurun:

<p id="paragraph"></p>

textContent özelliğini değiştirmek için aşağıdaki JavaScript’i çalıştırabiliriz:

document.getElementById("paragraph").textContent = "Hello, World";

Bu, “paragraf” kimliğine sahip öğeyi seçecek ve metin içeriğini “Merhaba, Dünya” olarak ayarlayacaktır:

<p id="paragraph">Hello, World</p>

(Bu demoya da bakın)

Programlı olarak yeni bir HTML öğesi oluşturmak için JavaScript’i de kullanabilirsiniz. Örneğin, aşağıdaki gövdeye sahip bir HTML belgesi düşünün:

<body>
   <h1>Adding an element</h1>
</body>

JavaScript’imizde, ’textContent’ özelliğine sahip yeni bir ‘

’ etiketi oluşturuyoruz ve bunu html gövdesinin sonuna ekliyoruz:

var element = document.createElement('p');
element.textContent = "Hello, World";
document.body.appendChild(element); //add the newly created element to the DOM

Bu, HTML gövdenizi aşağıdaki şekilde değiştirecektir:

<body>
   <h1>Adding an element</h1>
   <p>Hello, World</p>
</body>
