Kullanıcı İstemleri Hakkında #

Kullanıcı İstemleri, Web Uygulaması API’sinin bir parçasıdır. html5/webappapis.html#webappapis), onay veya giriş gibi bir kullanıcı eylemi isteyen Tarayıcı modellerini çağırmak için kullanılır.

window.alert(mesaj)

Kullanıcıya bir mesaj içeren kalıcı bir açılır pencere gösterin. Kullanıcının kapatmak için [Tamam]‘a tıklamasını gerektirir.

alert("Hello World");

Aşağıdaki “Uyarı () kullanma” bölümünde daha fazla bilgi.

boolean = window.confirm(message)

Sağlanan mesajla birlikte kalıcı bir açılır pencere gösterin. Sırasıyla ’true’ / ‘false’ boole değeriyle yanıt verecek [Tamam] ve [İptal] düğmelerini sağlar.

confirm("Delete this comment?");

sonuç = window.prompt(message, defaultValue)

Sağlanan mesajla kalıcı bir açılır pencere ve isteğe bağlı önceden doldurulmuş bir değere sahip bir giriş alanı gösterin. Kullanıcı tarafından sağlanan giriş değerini “sonuç” olarak döndürür.

prompt("Enter your website address", "http://");

Daha fazla bilgiyi aşağıda “Stem() Kullanımı” bölümünde bulabilirsiniz.


window.print()

Belge yazdırma seçenekleriyle bir modal açar.

print();

Kalıcı İstem Modalı #

İstemi kullanırken, kullanıcı her zaman İptal’e tıklayabilir ve hiçbir değer döndürülmez. Boş değerleri önlemek ve daha kalıcı yapmak için:

<h2>Welcome <span id="name"></span>!</h2>

<script>
// Persistent Prompt modal
var userName;
while(!userName) {
  userName = prompt("Enter your name", "");
  if(!userName) {
    alert("Please, we need your name!");
  } else {
    document.getElementById("name").innerHTML = userName;
  }
}
</script>

Öğeyi Silmeyi Onayla #

“onayla()“yı kullanmanın bir yolu, bazı kullanıcı arayüzü eylemlerinin sayfada bazı yıkıcı değişiklikler yapması ve bir bildirim ve bir kullanıcı onayı ile daha iyi eşlik etmesidir - örneğin, bir gönderi mesajını silmeden önce. :

<div id="post-102">
  <p>I like Confirm modals.</p>
  <a data-deletepost="post-102">Delete post</a>
</div>
<div id="post-103">
  <p>That's way too cool!</p>
  <a data-deletepost="post-103">Delete post</a>
</div>

// Collect all buttons
var deleteBtn = document.querySelectorAll("[data-deletepost]");

function deleteParentPost(event) {
  event.preventDefault(); // Prevent page scroll jump on anchor click
  
  if( confirm("Really Delete this post?") ) {
    var post = document.getElementById( this.dataset.deletepost );
    post.parentNode.removeChild(post);
    // TODO: remove that post from database
  } // else, do nothing
  
}

// Assign click event to buttons
[].forEach.call(deleteBtn, function(btn) {
  btn.addEventListener("click", deleteParentPost, false);
});

uyarı kullanımı() #

“Pencere” nesnesinin “alert()” yöntemi, belirtilen bir mesaj ve bir Tamam veya İptal düğmesi içeren bir alert kutusu görüntüler. Bu düğmenin metni tarayıcıya bağlıdır ve değiştirilemez.

Sözdizimi

alert("Hello world!");
// Or, alternatively...
window.alert("Hello world!");

Üretir

Bilginin kullanıcıya iletildiğinden emin olmak istiyorsanız, genellikle bir uyarı kutusu kullanılır.

Not: Uyarı kutusu, odağı geçerli pencereden uzaklaştırır ve tarayıcıyı mesajı okumaya zorlar. Kutu kapanana kadar kullanıcının sayfanın diğer bölümlerine erişmesini engellediği için bu yöntemi aşırı kullanmayın. Ayrıca, kullanıcı Tamam‘yi tıklayana kadar sonraki kod yürütmesini durdurur. (özellikle, ‘setInterval()’ veya ‘setTimeout()’ ile ayarlanan zamanlayıcılar da işaretlemez). Uyarı kutusu yalnızca tarayıcılarda çalışır ve tasarımı değiştirilemez.

parametre Açıklama
mesaj Gerekli. Uyarı kutusunda görüntülenecek metni veya dizeye dönüştürülerek görüntülenen bir nesneyi belirtir.

İstem() kullanımı #

İstem, kullanıcıya girişlerini isteyen bir iletişim kutusu görüntüler. Metin alanının üstüne yerleştirilecek bir mesaj sağlayabilirsiniz. Dönüş değeri, kullanıcı tarafından sağlanan girişi temsil eden bir dizedir.

var name = prompt("What's your name?");
console.log("Hello, " + name);

Ayrıca, istemin metin alanında varsayılan metin olarak görüntülenecek olan ikinci bir parametre “prompt()” iletebilirsiniz.

var name = prompt('What\'s your name?', ' Name...');
console.log('Hello, ' + name);
parametre Açıklama
mesaj Gerekli. Bilgi isteminin metin alanının üzerinde görüntülenecek metin.
varsayılan İsteğe bağlı. İstem görüntülendiğinde metin alanında görüntülenecek varsayılan metin.
