  • Bir dizi JavaScript ifadesini değerlendirin

JavaScript’te “eval” işlevi, bir dizeyi JavaScript koduymuş gibi değerlendirir. Dönüş değeri, değerlendirilen dizenin sonucudur, ör. ’eval(‘2 + 2’)’, ‘4’ değerini döndürür.

’eval’ global kapsamda mevcuttur. Değerlendirmenin sözcüksel kapsamı, dolaylı olarak çağrılmadığı sürece yerel kapsamdır (ör. var geval = eval; geval(s);).

“eval” kullanımı kesinlikle önerilmez. Ayrıntılar için Açıklamalar bölümüne bakın.

Sözdizimi #

  • değerlendir(dize);

Parametreler #

parametre Ayrıntılar
dize Değerlendirilecek JavaScript.

’eval’ kullanımı kesinlikle önerilmez; birçok senaryoda bir güvenlik açığı sunar.

eval(), iletilen kodu arayanın ayrıcalıklarıyla yürüten tehlikeli bir işlevdir. Eval()‘i kötü niyetli bir taraftan etkilenebilecek bir dize ile çalıştırırsanız, web sayfanızın/uzantınızın izinleriyle kullanıcının makinesinde kötü amaçlı kod çalıştırabilirsiniz. Daha da önemlisi, üçüncü taraf kodu, eval() öğesinin çağrıldığı kapsamı görebilir ve bu, benzer İşlevin duyarlı olmadığı şekillerde olası saldırılara yol açabilir.

MDN JavaScript Referansı

Bunlara ek olarak:

  • JavaScript’in eval() yöntemini kullanma
  • JavaScript’te “eval()” ile ilgili güvenlik sorunları nelerdir?

Giriiş #

JavaScript’i her zaman kendi içinden çalıştırabilirsiniz, ancak sunduğu güvenlik açıkları nedeniyle bu kesinlikle önerilmez (ayrıntılar için bkz. Açıklamalar.

JavaScript’i JavaScript’in içinden çalıştırmak için aşağıdaki işlevi kullanmanız yeterlidir:

eval("var a = 'Hello, World!'");

Değerlendirme ve Matematik #

Aşağıdaki koda benzer bir şey kullanarak ’eval()’ işleviyle bir değişkeni bir şeye ayarlayabilirsiniz:

var x = 10;
var y = 20;
var a = eval("x * y") + "<br>";
var b = eval("2 + 2") + "<br>";
var c = eval("x + 17") + "<br>";

var res = a + b + c;

“res” değişkeninde saklanan sonuç şöyle olacaktır:

200
4
27

“eval” kullanımı kesinlikle önerilmez. Ayrıntılar için Açıklamalar bölümüne bakın.

var x = 5;
var str = "if (x == 5) {console.log('z is 42'); z = 42;} else z = 0; ";

console.log("z is ", eval(str));
