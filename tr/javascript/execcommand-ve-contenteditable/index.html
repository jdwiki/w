Contenteditable Değişikliklerini Dinlemek #

Çoğu form öğesiyle çalışan olaylar (örneğin, “change”, “keydown”, “keyup”, “keypress”) “contenteditable” ile çalışmaz.

Bunun yerine, “input” olayı ile “contenteditable” içeriğindeki değişiklikleri dinleyebilirsiniz. “contenteditableHtmlElement” öğesinin “contenteditable” olan bir JS DOM nesnesi olduğunu varsayarsak:

contenteditableHtmlElement.addEventListener("input", function() {
    console.log("contenteditable element changed");
});

Başlarken #

“contenteditable” HTML özelliği, bir HTML öğesini kullanıcı tarafından düzenlenebilir bir alana dönüştürmek için basit bir yol sağlar.

<div contenteditable>You can <b>edit</b> me!</div>

Yerel Zengin Metin düzenleme

JavaScript ve execCommandW3C kullanarak, o anda odaklanmış olan ‘contenteditable’a ek olarak daha fazla düzenleme özelliği aktarabilirsiniz. ` öğesi (özellikle şapka pozisyonunda veya seçiminde).

“execCommand” işlev yöntemi 3 bağımsız değişkeni kabul eder

document.execCommand(commandId, showUI, value)
  • “commandId” Dizesi. listesinden kullanılabilir commandId’ler (bkz: Parametreler→commandId)
  • showUI Boolean (uygulanmadı. ‘yanlış’)
  • değer Dize Bir komut ise komutla ilgili bir Dize değer bekler, aksi takdirde "". (bkz: Parametreler→değer)

"bold" komut ve "formatBlock" (burada bir değer bekleniyor) kullanan örnek:

document.execCommand("bold", false, "");          // Make selected text bold
document.execCommand("formatBlock", false, "H2"); // Make selected text Block-level <h2>

Hızlı Başlangıç ​​Örneği:

<button data-edit="bold"><b>B</b></button>
<button data-edit="italic"><i>I</i></button>
<button data-edit="formatBlock:p">P</button>
<button data-edit="formatBlock:H1">H1</button>
<button data-edit="insertUnorderedList">UL</button>
<button data-edit="justifyLeft">&#8676;</button>
<button data-edit="justifyRight">&#8677;</button>
<button data-edit="removeFormat">&times;</button>

<div contenteditable><p>Edit me!</p></div>

[].forEach.call(document.querySelectorAll("[data-edit]"), function(btn) {
  btn.addEventListener("click", edit, false);
});

function edit(event) {
  event.preventDefault();
  var cmd_val = this.dataset.edit.split(":");
  document.execCommand(cmd_val[0], false, cmd_val[1]);
}

jsFiddle demosu Temel Zengin Metin düzenleyici örneği (Modern tarayıcılar)

Son düşünceler Uzun süredir mevcut olmasına rağmen (IE6), “execCommand"ın uygulamaları ve davranışları tarayıcıdan tarayıcıya değişiklik göstererek “Tam özellikli ve tarayıcılar arası uyumlu bir WYSIWYG düzenleyicisi oluşturmayı” deneyimli JavaScript geliştiricileri için zor bir görev haline getirir. Henüz tam olarak standardize edilmemiş olsa bile Chrome, Firefox, Edge gibi daha yeni tarayıcılarda oldukça iyi sonuçlar bekleyebilirsiniz. Diğer tarayıcılar için daha iyi desteğe ve HTMLTable düzenleme vb. gibi daha fazla özelliğe ihtiyacınız varsa, temel bir kural, zaten var olan ve sağlam bir Zengin Metin düzenleyicisi aramaktır.

Biçimlendirme #

Kullanıcılar, biçimlendirme için yaygın klavye kısayolları (kalın için Ctrl-B, Ctrl-I</kbd) gibi tarayıcılarının özelliklerini kullanarak “içerik düzenlenebilir” belgelere veya öğelere biçimlendirme ekleyebilir > italik vb. için) veya panodan resimleri, bağlantıları veya işaretlemeyi sürükleyip bırakarak.

Ayrıca geliştiriciler, geçerli seçime (vurgulanan metin) biçimlendirme uygulamak için JavaScript’i kullanabilir.

document.execCommand('bold', false, null); // toggles bold formatting
document.execCommand('italic', false, null); // toggles italic formatting
document.execCommand('underline', false, null); // toggles underline

execCommand(“copy”) kullanarak textarea’dan panoya kopyala #

Örnek:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title></title>
</head>
<body>
    <textarea id="content"></textarea>
    <input type="button" id="copyID" value="Copy" />
    <script type="text/javascript">
        var button = document.getElementById("copyID"),
            input = document.getElementById("content");

        button.addEventListener("click", function(event) {
            event.preventDefault();
            input.select();
            document.execCommand("copy");
        });
    </script>
</body>
</html>

document.execCommand("kopyala") geçerli seçimi panoya kopyalar
