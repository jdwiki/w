Kalan / Modül (%) #

Kalan / modül operatörü (%), (tamsayı) bölme işleminden sonra kalanı döndürür.

console.log( 42 %  10); //  2
console.log( 42 % -10); //  2
console.log(-42 %  10); // -2
console.log(-42 % -10); // -2
console.log(-40 %  10); // -0
console.log( 40 %  10); //  0

Bu operatör, bir işlenen ikinci bir işlenene bölündüğünde kalan değeri döndürür. İlk işlenen negatif bir değer olduğunda, dönüş değeri her zaman negatif olacaktır ve pozitif değerler için bunun tersi olacaktır.

Yukarıdaki örnekte, işareti değiştirmeden tekrar çıkarmaya yetecek kadar kalmadan önce ‘42’den ‘10’ dört kez çıkarılabilir. Böylece kalan: 42 - 4 * 10 = 2.

Kalan operatör aşağıdaki problemler için faydalı olabilir:

  1. Bir tamsayının başka bir sayıya bölünüp bölünemeyeceğini test edin:

     x % 4 == 0 // true if x is divisible by 4
 x % 2 == 0 // true if x is even number
 x % 2 != 0 // true if x is odd number

‘0 === -0’ olduğundan, bu aynı zamanda ‘x <= -0’ için de geçerlidir.

  1. “[0, n)” aralığı içinde döngüsel değer artışı/azaltma uygulayın.

Tamsayı değerini “0"dan “n"ye (ancak dahil değil) artırmamız gerektiğini varsayalım, böylece “n-1"den sonraki değer “0” olur. Bu, böyle bir sözde kodla yapılabilir:

var n = ...; // given n
var i = 0;
function inc() {
    i = (i + 1) % n;
}
while (true) {
    inc();
    // update something with i
}

Şimdi yukarıdaki problemi genelleştirin ve bu değeri “0"dan (dahil değil) “n"ye hem artırmaya hem de azaltmaya izin vermemiz gerektiğini varsayalım, böylece “n-1"den sonraki değer “0” olur ve önceki değer “0"dan önce “n-1” olur.

var n = ...; // given n
var i = 0;
function delta(d) { // d - any signed integer
    i = (i + d + n) % n; // we add n to (i+d) to ensure the sum is positive
}

Artık pozitif ve negatif herhangi bir tamsayıyı geçen delta() fonksiyonunu delta parametresi olarak çağırabiliriz.

Bir sayının kesirli kısmını elde etmek için modülü kullanma #

 var myNum = 10 / 4;       // 2.5
 var fraction = myNum % 1; // 0.5
 myNum = -20 / 7;          // -2.857142857142857
 fraction = myNum % 1;     // -0.857142857142857

Yuvarlama #

var a = Math.round(2.3);       // a is now 2

Ondalık basamağa yuvarlama #

“Math.floor”, “Math.ceil()” ve “Math.round()” birkaç ondalık basamağa yuvarlamak için kullanılabilir

2 ondalık basamağa yuvarlamak için:

 var myNum = 2/3;               // 0.6666666666666666
 var multiplier = 100;
 var a = Math.round(myNum * multiplier) / multiplier;  // 0.67
 var b = Math.ceil (myNum * multiplier) / multiplier;  // 0.67
 var c = Math.floor(myNum * multiplier) / multiplier;  // 0.66

Ayrıca birkaç basamağa yuvarlayabilirsiniz:

 var myNum = 10000/3;           // 3333.3333333333335
 var multiplier = 1/100;
 var a = Math.round(myNum * multiplier) / multiplier;  // 3300
 var b = Math.ceil (myNum * multiplier) / multiplier;  // 3400
 var c = Math.floor(myNum * multiplier) / multiplier;  // 3300

Daha kullanışlı bir işlev olarak:

 // value is the value to round
 // places if positive the number of decimal places to round to
 // places if negative the number of digits to round to
 function roundTo(value, places){
     var power = Math.pow(10, places);
     return Math.round(value * power) / power;
 }
 var myNum = 10000/3;    // 3333.3333333333335
 roundTo(myNum, 2);  // 3333.33
 roundTo(myNum, 0);  // 3333
 roundTo(myNum, -2); // 3300

Ve “tavan” ve “zemin” varyantları:

 function ceilTo(value, places){
     var power = Math.pow(10, places);
     return Math.ceil(value * power) / power;
 }
 function floorTo(value, places){
     var power = Math.pow(10, places);
     return Math.floor(value * power) / power;
 }

Artırma (++) #

Artırma operatörü (++) işlenenini bir artırır.

  • Sonek olarak kullanılırsa, artırmadan önceki değeri döndürür.
  • Ön ek olarak kullanılırsa, artırıldıktan sonra değeri döndürür.
//postfix
var a = 5,    // 5
    b = a++,  // 5
    c = a     // 6

Bu durumda, ‘b’ ayarlandıktan sonra ‘a’ artırılır. Yani, “b” 5 ve “c” 6 olacaktır.

//prefix
var a = 5,    // 5
    b = ++a,  // 6
    c = a     // 6

Bu durumda, “b” ayarlanmadan önce “a” artırılır. Yani, “b” 6 olacak ve “c” 6 olacak.

Artırma ve eksiltme operatörleri yaygın olarak “for” döngülerinde kullanılır, örneğin:

for(var i = 0; i < 42; ++i)
{
  // do something awesome!
}

prefix varyantının nasıl kullanıldığına dikkat edin. Bu, geçici bir değişkenin gereksiz yere yaratılmamasını sağlar (işlemden önceki değeri döndürmek için).

Bitsel operatörler #

Tüm bitsel işlemlerin, herhangi bir işleneni dahili işleve ToInt32 ileterek 32 bit tam sayılar üzerinde çalıştığını unutmayın.

Bitsel veya #

var a;
a = 0b0011 | 0b1010; // a === 0b1011
// truth table
// 1010 | (or)
// 0011  
// 1011  (result)

Bitsel ve #

a = 0b0011 & 0b1010; // a === 0b0010
// truth table
// 1010 & (and)
// 0011  
// 0010  (result)

Bit düzeyinde not #

a = ~0b0011; // a === 0b1100
// truth table
// 10 ~(not)
// 01  (result)

Bitsel xor (özel veya) #

a = 0b1010 ^ 0b0011; // a === 0b1001
// truth table
// 1010 ^ (xor)
// 0011  
// 1001  (result)

Bitsel sol kaydırma #

a = 0b0001 << 1; // a === 0b0010
a = 0b0001 << 2; // a === 0b0100
a = 0b0001 << 3; // a === 0b1000

Sola kaydırma, “Math.pow(2, n)” ile tamsayı çarpmasına eşdeğerdir. Tamsayılı matematik yaparken shift, bazı matematik işlemlerinin hızını önemli ölçüde artırabilir.

var n = 2;
var a = 5.4;
var result = (a << n) === Math.floor(a) * Math.pow(2,n);
// result is true
a = 5.4 << n; // 20

Bitsel sağ kaydırma >> (İşaret yayma kaydırma) >>> (Sıfır dolgulu sağa kaydırma) #

a = 0b1001 >> 1; // a === 0b0100
a = 0b1001 >> 2; // a === 0b0010
a = 0b1001 >> 3; // a === 0b0001

a = 0b1001 >>> 1; // a === 0b0100
a = 0b1001 >>> 2; // a === 0b0010
a = 0b1001 >>> 3; // a === 0b0001

Negatif bir 32 bit değer her zaman en soldaki bit üzerindedir:

a = 0b11111111111111111111111111110111 | 0;   
console.log(a); // -9
b = a >> 2;     // leftmost bit is shifted 1 to the right then new left most bit is set to on (1)
console.log(b); // -3
b = a >>> 2;    // leftmost bit is shifted 1 to the right. the new left most bit is set to off (0)
console.log(b); // 2147483643

Bir >>> işleminin sonucu her zaman pozitiftir. Bir >> sonucu her zaman kaydırılan değerle aynı işarettir.

Pozitif sayılar üzerinde sağa kaydırma, “Math.pow(2,n)” ile bölmeye ve sonucu döşemeye eşdeğerdir:

a = 256.67;
n = 4;
result = (a >> n) === Math.floor( Math.floor(a) / Math.pow(2,n) );
// result is true
a = a >> n; //  16

result = (a >>> n) === Math.floor( Math.floor(a) / Math.pow(2,n) );
// result is true
a = a >>> n; //  16

Negatif sayılarda sağa kaydırma sıfır dolgusu (>>>) farklıdır. JavaScript, bit işlemleri yaparken imzasız girişlere dönüşmediğinden, operasyonel eşdeğeri yoktur:

a = -256.67;
result = (a >>> n) === Math.floor( Math.floor(a) / Math.pow(2,n) );
// result is false

Bitsel atama operatörleri #

Not (~) dışında, yukarıdaki tüm bitsel operatörler atama operatörleri olarak kullanılabilir:

a |= b;   // same as: a = a | b;
a ^= b;   // same as: a = a ^ b;
a &= b;   // same as: a = a & b;
a >>= b;  // same as: a = a >> b;
a >>>= b; // same as: a = a >>> b;
a <<= b;  // same as: a = a << b;

Uyarı: Javascript, tam sayıları depolamak için Big Endian’ı kullanır. Bu her zaman cihazın/işletim sisteminin Endian’ı ile eşleşmeyecektir. Bit uzunlukları 8 bitten büyük olan yazılan dizileri kullanırken, bitsel işlemleri uygulamadan önce ortamın Little Endian veya Big Endian olup olmadığını kontrol etmelisiniz.

Uyarı: & ve | gibi bitsel operatörler, mantıksal operatörlerle [&& (ve)] aynı değildir](https://www.wikiod.com/tr/docs/javascript 208/boolean-logic/779/logic-operators-with-booleans#t=201606101417337061178) ve [|| (veya)](https://www.wikiod.com/tr/docs/javascript208/boolean-logic/779 /logic-operators-with-booleans#t=201606101417337061178). Mantıksal operatörler olarak kullanılırlarsa yanlış sonuçlar verirler. ^ operatörü, güç operatörü (ab) değildir üs-matematik-güç-veya).

Ekleme (+) #

Toplama operatörü (+) sayıları ekler.

var a = 9,
    b = 3,
    c = a + b;

‘c’ şimdi 12 olacak

Bu işlenen, tek bir atamada birden çok kez de kullanılabilir:

    b = 3,
    c = 8,
    d = a + b + c;

d şimdi 20 olacak.

Her iki işlenen de ilkel türlere dönüştürülür. Ardından, biri bir dize ise, ikisi de dizelere dönüştürülür ve birleştirilir. Aksi takdirde, ikisi de sayılara dönüştürülür ve eklenir.

null + null;      // 0
null + undefined; // NaN
null + {};        // "null[object Object]"
null + '';        // "null"

İşlenenler bir dize ve bir sayıysa, sayı bir dizeye dönüştürülür ve ardından birleştirilir; bu, sayısal görünen dizelerle çalışırken beklenmeyen sonuçlara yol açabilir.

"123" + 1;        // "1231" (not 124)

true + 1;         // 2
false + 5;        // 5
null + 1;         // 1
undefined + 1;    // NaN

Bir dize değeriyle birlikte bir boole değeri verilirse, bunun yerine boole değeri bir dizeye dönüştürülür:

true + "1";        // "true1"
false + "bar";     // "falsebar"

Farklı olasılıklara sahip olayları simüle etme #

Bazen sadece iki sonucu olan, belki farklı olasılıkları olan bir olayı simüle etmeniz gerekebilir, ancak kendinizi farklı olasılıklara sahip birçok olası sonucu gerektiren bir durumda bulabilirsiniz. Altı eşit olası sonucu olan bir olayı simüle etmek istediğinizi varsayalım. Bu oldukça basit.

function simulateEvent(numEvents) {
    var event = Math.floor(numEvents*Math.random());
    return event;
}

// simulate fair die
console.log("Rolled a "+(simulateEvent(6)+1));  // Rolled a 2

Ancak, eşit derecede olası sonuçlar istemeyebilirsiniz. Yüzdeler veya olasılık katları olarak bir dizi olasılık olarak temsil edilen üç sonuçtan oluşan bir listeniz olduğunu varsayalım. Böyle bir örnek ağırlıklı bir kalıp olabilir. Böyle bir olayı simüle etmek için önceki işlevi yeniden yazabilirsiniz.

function simulateEvent(chances) {
    var sum = 0;
    chances.forEach(function(chance) {
        sum+=chance;
    });
    var rand = Math.random();
    var chance = 0;
    for(var i=0; i<chances.length; i++) {
        chance+=chances[i]/sum;
        if(rand<chance) {
            return i;
        }
    }
    
    // should never be reached unless sum of probabilities is less than 1
    // due to all being zero or some being negative probabilities
    return -1;
}

// simulate weighted dice where 6 is twice as likely as any other face
// using multiples of likelihood
console.log("Rolled a "+(simulateEvent([1,1,1,1,1,2])+1));  // Rolled a 1

// using probabilities
console.log("Rolled a "+(simulateEvent([1/7,1/7,1/7,1/7,1/7,2/7])+1));  // Rolled a 6

Muhtemelen fark ettiğiniz gibi, bu işlevler bir dizin döndürür, böylece bir dizide saklanan daha açıklayıcı sonuçlara sahip olabilirsiniz. İşte bir örnek.

var rewards = ["gold coin","silver coin","diamond","god sword"];
var likelihoods = [5,9,1,0];
// least likely to get a god sword (0/15 = 0%, never),
// most likely to get a silver coin (9/15 = 60%, more than half the time)

// simulate event, log reward
console.log("You get a "+rewards[simulateEvent(likelihoods)]);  // You get a silver coin

var isLittleEndian = true;

Gauss dağılımıyla rastgele #

Math.random() işlevi 0’a yaklaşan bir standart sapmaya sahip rastgele sayılar vermelidir. Bir deste karttan seçim yaparken veya bir zar atışını simüle ederken istediğimiz budur.

Ancak çoğu durumda bu gerçekçi değildir. Gerçek dünyada rastgelelik, ortak bir normal değer etrafında toplanma eğilimindedir. Bir grafik üzerinde çizilirse, klasik çan eğrisini veya gauss dağılımını elde edersiniz.

Bunu Math.random() işleviyle yapmak nispeten basittir.

var randNum = (Math.random() + Math.random()) / 2; 
var randNum = (Math.random() + Math.random() + Math.random()) / 3; 
var randNum = (Math.random() + Math.random() + Math.random() + Math.random()) / 4;

Sonuna rastgele bir değer eklemek, rastgele sayıların varyansını arttırır. Eklediğiniz sayıya bölmek, sonucu 0–1 aralığına normalleştirir

Birkaç rastgeleden fazlasını eklemek karmaşık olduğundan, basit bir işlev, istediğiniz bir varyansı seçmenize olanak tanır.

// v is the number of times random is summed and should be over >= 1
// return a random number between 0-1 exclusive
function randomG(v){ 
    var r = 0;
    for(var i = v; i > 0; i --){
        r += Math.random();
    }
    return r / v;
}

Görüntü, farklı v değerleri için rastgele değerlerin dağılımını gösterir. Sol üst standart tek Math.random() çağrısıdır, sağ alttaki 8 kez toplanan Math.random()dur. Bu, Chrome kullanılarak 5.000.000 örnekten alınmıştır

Bu yöntem en çok “v<5” konumunda etkilidir

Tavan ve Zemin #

ceil()

‘ceil()’ yöntemi, bir sayıyı upwards en yakın tam sayıya yuvarlar ve sonucu döndürür.

Sözdizimi:

Math.ceil(n);

Örnek:

console.log(Math.ceil(0.60)); //  1
console.log(Math.ceil(0.40)); //  1
console.log(Math.ceil(5.1));  //  6
console.log(Math.ceil(-5.1)); // -5
console.log(Math.ceil(-5.9)); // -5

zemin()

“floor()” yöntemi, bir sayıyı _aşağıya doğru en yakın tam sayıya yuvarlar ve sonucu döndürür.

Sözdizimi:

Math.floor(n);

Örnek:

console.log(Math.ceil(0.60)); //  0
console.log(Math.ceil(0.40)); //  0
console.log(Math.ceil(5.1));  //  5
console.log(Math.ceil(-5.1)); // -6
console.log(Math.ceil(-5.9)); // -6

Math.atan2 yön bulmak için #

Vektörlerle veya çizgilerle çalışıyorsanız, bir aşamada bir vektörün veya çizginin yönünü almak isteyeceksiniz. Veya bir noktadan başka bir noktaya yön.

“Math.atan(yComponent, xComponent)”, “-Math.PI” ile “Math.PI” ("-180” ila “180” derece) aralığında yarıçap cinsinden açıyı döndürür

Vektörün yönü #

var vec = {x : 4, y : 3};
var dir = Math.atan2(vec.y, vec.x);  // 0.6435011087932844

Çizginin yönü #

var line = {
    p1 : { x : 100, y : 128},
    p2 : { x : 320, y : 256}
}
// get the direction from p1 to p2
var dir = Math.atan2(line.p2.y - line.p1.y, line.p2.x - line.p1.x); // 0.5269432271894297

Bir noktadan başka bir noktaya yön #

var point1 = { x: 123, y : 294};
var point2 = { x: 354, y : 284};
// get the direction from point1 to point2
var dir = Math.atan2(point2.y - point1.y, point2.x - point1.x); // -0.04326303140726714

Sin & Cos, yön ve mesafe verilen bir vektör oluşturmak için #

Kutup biçiminde (yön ve mesafe) bir vektörünüz varsa, onu x ve y bileşeni olan bir kartezyen vektöre dönüştürmek isteyeceksiniz. Referans için ekran koordinat sistemi, soldan sağa 0 derecelik noktalar, ekranın aşağısında 90 (PI/2) nokta ve saat yönünde bu şekilde devam eden yönlere sahiptir.

var dir = 1.4536; // direction in radians
var dist = 200; // distance
var vec = {};
vec.x = Math.cos(dir) * dist; // get the x component
vec.y = Math.sin(dir) * dist; // get the y component

“dir” yönünde normalleştirilmiş (1 birim uzunluğunda) bir vektör oluşturmak için mesafeyi de göz ardı edebilirsiniz.

var dir = 1.4536; // direction in radians
var vec = {};
vec.x = Math.cos(dir); // get the x component
vec.y = Math.sin(dir); // get the y component

Koordinat sisteminizde y kadar yukarıdaysa, o zaman cos ve sin arasında geçiş yapmanız gerekir. Bu durumda pozitif bir yön, x ekseninden saat yönünün tersi yönündedir.

// get the directional vector where y points up
var dir = 1.4536; // direction in radians
var vec = {};
vec.x = Math.sin(dir); // get the x component
vec.y = Math.cos(dir); // get the y component

Matematik.hipot #

İki nokta arasındaki uzaklığı bulmak için, aralarındaki vektörün bileşeninin karesinin toplamının karekökünü almak için pisagorları kullanırız.

var v1 = {x : 10, y :5};
var v2 = {x : 20, y : 10};
var x = v2.x - v1.x;
var y = v2.y - v1.y;
var distance = Math.sqrt(x * x + y * y); // 11.180339887498949

ECMAScript 6 ile aynı şeyi yapan Math.hypot geldi

var v1 = {x : 10, y :5}; 
var v2 = {x : 20, y : 10}; 
var x = v2.x - v1.x; 
var y = v2.y - v1.y;
var distance = Math.hypot(x,y); // 11.180339887498949

Artık, kodun bir değişken karmaşası haline gelmesini durdurmak için ara değişkenleri tutmanız gerekmez.

var v1 = {x : 10, y :5};
var v2 = {x : 20, y : 10};
var distance = Math.hypot(v2.x - v1.x, v2.y - v1.y); // 11.180339887498949

Math.hypot herhangi bir sayıda boyut alabilir

// find distance in 3D
var v1 = {x : 10, y : 5, z : 7};
var v2 = {x : 20, y : 10, z : 16};
var dist = Math.hypot(v2.x - v1.x, v2.y - v1.y, v2.z - v1.z); // 14.352700094407325

// find length of 11th dimensional vector
var v = [1,3,2,6,1,7,3,7,5,3,1]; 
var i = 0;
dist = Math.hypot(v[i++],v[i++],v[i++],v[i++],v[i++],v[i++],v[i++],v[i++],v[i++],v[i++],v[i++]);

Math.sin kullanan periyodik fonksiyonlar #

“Math.sin” ve “Math.cos”, 2*PI radyan (360 derece) periyoduyla döngüseldirler, -1 ila 1 aralığında 2 genlikli bir dalga üretirler.

Sinüs ve kosinüs fonksiyonunun grafiği: (Vikipedi’nin nezaketiyle) [](https://en.wikipedia.org/wiki /Sinüs dalgası)

Her ikisi de ses dalgaları oluşturmaktan animasyonlara ve hatta görüntü verilerini kodlamak ve kodunu çözmek gibi birçok türde periyodik hesaplama için çok kullanışlıdır.

Bu örnek, periyot/frekans, faz, genlik ve ofset üzerinde kontrole sahip basit bir günah dalgasının nasıl oluşturulacağını gösterir.

Kullanılan zaman birimi saniyedir. Yalnızca frekans üzerinde kontrole sahip en basit biçim.

// time is the time in seconds when you want to get a sample
// Frequency represents the number of oscillations per second
function oscillator(time, frequency){  
    return Math.sin(time * 2 * Math.PI * frequency);
}

Hemen hemen her durumda, döndürülen değerde bazı değişiklikler yapmak isteyeceksiniz. Değişiklikler için ortak terimler

  • Faz: Salınımların başlangıcından itibaren frekans cinsinden sapma. 0 ile 1 aralığında bir değerdir, burada 0,5 değeri dalgayı frekansının yarısı kadar zamanda ileri taşır. 0 veya 1 değeri hiçbir değişiklik yapmaz.
  • Genlik: Bir çevrim sırasında en düşük değerden ve en yüksek değerden olan mesafe. 1 genliği 2 aralığına sahiptir. En düşük nokta (dip) -1 ila en yüksek (tepe) 1. Frekans 1’e sahip bir dalga için tepe 0,25 saniyede ve dip noktası 0,75’tedir.
  • Ofset: tüm dalgayı yukarı veya aşağı hareket ettirir.

Tüm bunları işleve dahil etmek için:

function oscillator(time, frequency = 1, amplitude = 1, phase = 0, offset = 0){
    var t = time * frequency * Math.PI * 2; // get phase at time
    t += phase * Math.PI * 2; // add the phase offset
    var v = Math.sin(t); // get the value at the calculated position in the cycle
    v *= amplitude; // set the amplitude
    v += offset; // add the offset
    return v;
}

Veya daha kompakt (ve biraz daha hızlı biçimde):

function oscillator(time, frequency = 1, amplitude = 1, phase = 0, offset = 0){
    return Math.sin(time * frequency * Math.PI * 2 + phase * Math.PI * 2) * amplitude + offset; 
}

Zaman dışındaki tüm argümanlar isteğe bağlıdır

Maksimum ve minimum alma #

Math.max() işlevi, sıfır veya daha fazla sayıdan en büyüğünü döndürür.

Math.max(4, 12);   //  12
Math.max(-1, -15); // -1

Math.min() işlevi, sıfır veya daha fazla sayıdan en küçüğünü döndürür.

Math.min(4, 12);   //  4
Math.min(-1, -15); // -15

Bir diziden maksimum ve minimum alma: #

var arr = [1, 2, 3, 4, 5, 6, 7, 8, 9],
    max = Math.max.apply(Math, arr),
    min = Math.min.apply(Math, arr);

console.log(max); // Logs: 9
console.log(min); // Logs: 1

ECMAScript 6 yayılma operatörü, bir dizinin maksimum ve minimumunu alma:

var arr = [1, 2, 3, 4, 5, 6, 7, 8, 9],
    max = Math.max(...arr),
    min = Math.min(...arr);

console.log(max); // Logs: 9
console.log(min); // Logs: 1

Sayıyı Min/Maks Aralığıyla Sınırla #

Belirli bir aralık sınırı içinde tutmak için bir sayıyı sıkıştırmanız gerekirse

function clamp(min, max, val) {
    return Math.min(Math.max(min, +val), max);
}

console.log(clamp(-10, 10, "4.30")); // 4.3
console.log(clamp(-10, 10, -8));     // -8
console.log(clamp(-10, 10, 12));     // 10
console.log(clamp(-10, 10, -15));    // -10
