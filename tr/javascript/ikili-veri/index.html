Bir görüntü dosyasının ikili gösterimini alma #

Bu örnek, bu sorudan esinlenmiştir.

Dosya API’sini kullanarak dosya yüklemeyi bildiğinizi varsayacağız.

// preliminary code to handle getting local file and finally printing to console
// the results of our function ArrayBufferToBinary().
var file = // get handle to local file.
var reader = new FileReader();
reader.onload = function(event) {
    var data = event.target.result;
    console.log(ArrayBufferToBinary(data));
};
reader.readAsArrayBuffer(file); //gets an ArrayBuffer of the file

Şimdi bir DataView kullanarak dosya verilerinin gerçek dönüşümünü 1’lere ve 0’lara yapıyoruz:

function ArrayBufferToBinary(buffer) {
   // Convert an array buffer to a string bit-representation: 0 1 1 0 0 0...
   var dataView = new DataView(buffer);
   var response = "", offset = (8/8); 
   for(var i = 0; i < dataView.byteLength; i += offset) {
       response += dataView.getInt8(i).toString(2); 
   }
   return response;
}

DataView sayısal verileri okumanıza/yazmanıza izin verir; “getInt8”, bayt konumundan gelen verileri dönüştürür - burada “0”, iletilen değerdir - “ArrayBuffer"da imzalı 8 bitlik tamsayı gösterimine ve “toString(2)”, 8 bitlik tamsayıyı ikili gösterime dönüştürür biçim (yani, 1’ler ve 0’lardan oluşan bir dize).

Dosyalar bayt olarak kaydedilir. ‘Sihirli’ ofset değeri, bayt olarak saklanan dosyaları, yani 8-bit tamsayılar olarak aldığımızı ve 8-bit tamsayı gösteriminde okuduğumuzu not ederek elde edilir. Bayt kayıtlı (yani 8 bit) dosyalarımızı 32 bit tam sayılara okumaya çalışıyor olsaydık, 32/8 = 4’ün bayt ofset değerimiz olan bayt boşluklarının sayısı olduğunu not ederdik.

Bu görev için DataViewler aşırıya kaçıyor. Tipik olarak, verilerin endianlığı veya heterojenliği ile karşılaşıldığı durumlarda kullanılırlar (örneğin, farklı temellerde kodlanmış başlıkları olan ve bu değeri anlamlı bir şekilde çıkarmak istediğimiz PDF dosyalarını okurken). Sadece metinsel bir temsil istediğimiz için, heterojenliği umursamıyoruz çünkü asla buna gerek yok.

Çok daha iyi - ve daha kısa - bir çözüm, “ArrayBuffer"ın tamamını işaretsiz 8 bitlik tam sayılardan oluşan olarak ele alan “UInt8Array” türünde bir dizi kullanılarak bulunabilir:

function ArrayBufferToBinary(buffer) {
    var uint8 = new Uint8Array(buffer);
    return uint8.reduce((binary, uint8) => binary + uint8.toString(2), "");
}

Bloblar ve ArrayBuffers arasında dönüştürme #

JavaScript’in tarayıcıda ikili verileri temsil etmenin iki ana yolu vardır. ArrayBuffers/TypedArrays, doğrudan değiştirebileceğiniz değiştirilebilir (yine de sabit uzunlukta) ikili veriler içerir. Bloblar, yalnızca eşzamansız Dosya arabirimi aracılığıyla erişilebilen değişmez ikili veriler içerir.

Bir “Blob"u “ArrayBuffer"a dönüştürün (eşzamansız) #

var blob = new Blob(["\x01\x02\x03\x04"]),
    fileReader = new FileReader(),
    array;

fileReader.onload = function() {
    array = this.result;
    console.log("Array contains", array.byteLength, "bytes.");
};

fileReader.readAsArrayBuffer(blob);

Bir “Söz” (eşzamansız) kullanarak bir “Blob"u “ArrayBuffer"a dönüştürün #

var blob = new Blob(["\x01\x02\x03\x04"]);

var arrayPromise = new Promise(function(resolve) {
    var reader = new FileReader();

    reader.onloadend = function() {
        resolve(reader.result);
    };

    reader.readAsArrayBuffer(blob);
});

arrayPromise.then(function(array) {
    console.log("Array contains", array.byteLength, "bytes.");
});
son sürüm

Bir “ArrayBuffer” veya yazılan diziyi “Blob"a dönüştürün #

var array = new Uint8Array([0x04, 0x06, 0x07, 0x08]);

var blob = new Blob([array]);

ArrayBuffer’ları DataView ile Değiştirme #

DataView’ler, her şeyi tek tip bir dizi olarak görüntülemek yerine, bir ArrayBuffer’dan tek tek değerleri okumak ve yazmak için yöntemler sağlar. Burada iki baytı ayrı ayrı ayarlıyoruz, sonra bunları birlikte 16 bitlik işaretsiz bir tamsayı olarak yorumluyoruz, önce büyük endian, sonra küçük endian.

var buffer = new ArrayBuffer(2);
var view = new DataView(buffer);

view.setUint8(0, 0xFF);
view.setUint8(1, 0x01);

console.log(view.getUint16(0, false)); // 65281
console.log(view.getUint16(0, true));  // 511

Base64 dizesinden TypedArray oluşturma #

var data = 
   'iVBORw0KGgoAAAANSUhEUgAAAAUAAAAFCAYAAACN' +
   'byblAAAAHElEQVQI12P4//8/w38GIAXDIBKE0DHx' +
   'gljNBAAO9TXL0Y4OHwAAAABJRU5ErkJggg==';

var characters = atob(data);

var array = new Uint8Array(characters.length);

for (var i = 0; i < characters.length; i++) {
  array[i] = characters.charCodeAt(i);
}

TypedArrays’i Kullanma #

TypedArrays, sabit uzunluktaki değişken ikili ArrayBuffer’lara farklı görünümler sağlayan bir tür kümesidir. Çoğunlukla, atanan tüm değerleri belirli bir sayısal türe zorlayan Diziler gibi davranırlar. Verilerinin yeni bir görünümünü oluşturmak için bir ArrayBuffer örneğini TypedArray yapıcısına iletebilirsiniz.

var buffer = new ArrayBuffer(8);
var byteView = new Uint8Array(buffer);
var floatView = new Float64Array(buffer);

console.log(byteView);  // [0, 0, 0, 0, 0, 0, 0, 0]
console.log(floatView); // [0]
byteView[0] = 0x01;
byteView[1] = 0x02;
byteView[2] = 0x04;
byteView[3] = 0x08;
console.log(floatView); // [6.64421383e-316]

ArrayBuffer’lar, .slice(...) yöntemi kullanılarak doğrudan veya bir TypedArray görünümü aracılığıyla kopyalanabilir.

var byteView2 = byteView.slice();
var floatView2 = new Float64Array(byteView2.buffer);
byteView2[6] = 0xFF;
console.log(floatView);  // [6.64421383e-316]
console.log(floatView2); // [7.06327456e-304]

arrayBuffer aracılığıyla yineleme #

Bir arrayBuffer üzerinden yineleme yapmanın uygun bir yolu için, başlık altında “DataView” yöntemlerini uygulayan basit bir yineleyici oluşturabilirsiniz:

var ArrayBufferCursor = function() {
  var ArrayBufferCursor = function(arrayBuffer) {
    this.dataview = new DataView(arrayBuffer, 0);
    this.size = arrayBuffer.byteLength;
    this.index = 0;
  }

  ArrayBufferCursor.prototype.next = function(type) {
    switch(type) {
      case 'Uint8':
        var result = this.dataview.getUint8(this.index);
        this.index += 1;
        return result;
      case 'Int16':
        var result = this.dataview.getInt16(this.index, true);
        this.index += 2;
        return result;
      case 'Uint16':
        var result = this.dataview.getUint16(this.index, true);
        this.index += 2;
        return result;
      case 'Int32':
        var result = this.dataview.getInt32(this.index, true);
        this.index += 4;
        return result;
      case 'Uint32':
        var result = this.dataview.getUint32(this.index, true);
        this.index += 4;
        return result;
      case 'Float':
      case 'Float32':
        var result = this.dataview.getFloat32(this.index, true);
        this.index += 4;
        return result;
      case 'Double':
      case 'Float64':
        var result = this.dataview.getFloat64(this.index, true);
        this.index += 8;
        return result;
      default:
        throw new Error("Unknown datatype");
    }
  };

  ArrayBufferCursor.prototype.hasNext = function() {
    return this.index < this.size;
  }

  return ArrayBufferCursor;
});

Daha sonra şöyle bir yineleyici oluşturabilirsiniz:

var cursor = new ArrayBufferCursor(arrayBuffer);

Hala öğe olup olmadığını kontrol etmek için hasNexti kullanabilirsiniz.

for(;cursor.hasNext();) {
    // There's still items to process
}

Sonraki değeri almak için “next” yöntemini kullanabilirsiniz:

var nextValue = cursor.next('Float');

Böyle bir yineleyici ile, ikili verileri işlemek için kendi ayrıştırıcınızı yazmak oldukça kolay hale gelir.
