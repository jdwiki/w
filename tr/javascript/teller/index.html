Temel Bilgiler ve Dize Birleştirme #

JavaScript’teki dizeler, Şablon Değişmezlerinde (backticks) `hello` Tek tırnak işareti hello, çift tırnak "Hello" ve (ES2015, ES6’dan) içine alınabilir.

var hello  = "Hello";
var world  = 'world';
var helloW = `Hello World`;              // ES2015 / ES6

Dizeler, String() işlevi kullanılarak diğer türlerden oluşturulabilir.

var intString = String(32); // "32"
var booleanString = String(true); // "true"
var nullString = String(null); // "null"

Veya, Sayıları, Booleanları veya Nesneleri Dizelere dönüştürmek için toString() kullanılabilir.

var intString = (5232).toString(); // "5232"
var booleanString = (false).toString(); // "false"
var objString = ({}).toString(); // "[object Object]"

Dizeler ayrıca String.fromCharCode yöntemi kullanılarak da oluşturulabilir.

String.fromCharCode(104,101,108,108,111) //"hello"

’new’ anahtar sözcüğünü kullanarak bir String nesnesi oluşturmaya izin verilir, ancak ilkel dizelerin aksine Objects gibi davrandığı için önerilmez.

var objectString = new String("Yes, I am a String object");
typeof objectString;//"object"
typeof objectString.valueOf();//"string"

Birleştirme Dizeleri #

Dize bitiştirme, “+” birleştirme operatörüyle veya String nesne prototipindeki yerleşik “concat()” yöntemiyle yapılabilir.

var foo = "Foo";
var bar = "Bar";
console.log(foo + bar);        // => "FooBar"
console.log(foo + " " + bar);  // => "Foo Bar"

foo.concat(bar)            // => "FooBar"
"a".concat("b", " ", "d")  // => "ab d"

Dizeler, dize olmayan değişkenlerle birleştirilebilir, ancak dize olmayan değişkenleri dizelere tip-dönüştürür.

var string = "string";
var number = 32;
var boolean = true;

console.log(string + number + boolean); // "string32true"

Dize Şablonları #

Dizeler, şablon değişmezleri (backticks) `hello` kullanılarak oluşturulabilir.

var greeting = `Hello`;

Şablon değişmez değerleriyle, şablon değişmezlerinin içindeki ${variable} kullanarak dize enterpolasyonu yapabilirsiniz:

var place = `World`;
var greet = `Hello ${place}!`

console.log(greet); // "Hello World!"

Ters eğik çizgilerin değişiklik yapmadan dizgede olmasını sağlamak için String.raw’ı kullanabilirsiniz.

`a\\b` // =  a\b
String.raw`a\\b` // = a\\b

Ters Dize #

JavaScript’te bir dizgiyi tersine çevirmenin en “popüler” yolu, oldukça yaygın olan aşağıdaki kod parçasıdır:

function reverseString(str) {
    return str.split('').reverse().join('');
}

reverseString('string');    // "gnirts"

Ancak bu, yalnızca ters çevrilen dize vekil çiftler içermediği sürece işe yarar. Astral semboller, yani temel çok dilli düzlemin dışındaki karakterler, iki kod birimi ile temsil edilebilir ve bu naif tekniğin yanlış sonuçlar üretmesine yol açacaktır. Ayrıca, birleştirme işaretlerine sahip karakterler (örn. çift taraflı), birleştirildiği orijinal karakter yerine mantıksal “sonraki” karakterde görünecektir.

'𝌆■.'.split('').reverse().join(''); //fails

Yöntem çoğu dil için iyi çalışsa da, dize ters çevirme için gerçekten doğru, kodlamaya saygı gösteren bir algoritma biraz daha fazla yer alır. Böyle bir uygulama Esrever adlı küçük bir kitaplıktır ve bu kitaplık, mükemmel bir şekilde tersine çevirmeyi gerçekleştirmek için birleştirme işaretlerini ve vekil çiftleri eşleştirmek için düzenli ifadeler kullanır.

Açıklama #

Bölüm Açıklama Sonuç
str Giriş dizesi "dize"
String.prototype.split( sınırlayıcı ) str dizesini bir diziye böler. "" parametresi, her karakter arasında bölme anlamına gelir. ["s","t","r","i","n","g"]
Array.prototype.reverse() Diziyi, öğeleri ters sırada olacak şekilde bölünmüş dizeden döndürür. ["g","n","i","r","t","s"]
Array.prototype.join( sınırlayıcı ) Dizideki öğeleri bir dizgede birleştirir. "" parametresi, boş bir sınırlayıcı anlamına gelir (yani, dizinin öğeleri yan yana yerleştirilir). "gnirts"

Sürüm operatörünü kullanma

function reverseString(str) {
    return [...String(str)].reverse().join('');    
}

console.log(reverseString('stackoverflow'));  // "wolfrevokcats"
console.log(reverseString(1337));             // "7331"
console.log(reverseString([1, 2, 3]));        // "3,2,1"
son sürüm

Özel “ters ()” işlevi

function reverse(string) {
    var strRev = "";
    for (var i = string.length - 1; i >= 0; i--) {
        strRev += string[i];
    }
    return strRev; 
}

reverse("zebra");  // "arbez"

Dizeleri Sözlüksel Olarak Karşılaştırma #

Dizeleri alfabetik olarak karşılaştırmak için localeCompare() kullanın. Bu, referans dizesi sözlükbilimsel olarak (alfabetik olarak) karşılaştırılan dizeden (parametre) önceyse negatif bir değer, daha sonra gelirse pozitif bir değer ve eşitlerse “0” değeri döndürür.

var a = "hello";
var b = "world";

console.log(a.localeCompare(b)); // -1

> ve < operatörleri aynı zamanda dizeleri sözlükbilimsel olarak karşılaştırmak için de kullanılabilir, ancak sıfır değeri döndüremezler (bu, == eşitlik operatörü ile test edilebilir). Sonuç olarak, localeCompare() fonksiyonunun bir formu şu şekilde yazılabilir:

function strcmp(a, b) {
    if(a === b) {
        return 0;
    }

    if (a > b) {
        return 1;
    }

    return -1;
}

console.log(strcmp("hello", "world")); // -1
console.log(strcmp("hello", "hello")); //  0
console.log(strcmp("world", "hello")); //  1

Bu, özellikle dönüş değerinin işaretine göre karşılaştırma yapan bir sıralama işlevi kullanırken (“sıralama” gibi) kullanışlıdır.

var arr = ["bananas", "cranberries", "apples"];
arr.sort(function(a, b) {
    return a.localeCompare(b);
});
console.log(arr); // [ "apples", "bananas", "cranberries" ]

Dizindeki dizindeki erişim karakteri #

Dizede belirtilen dizinde bir karakter almak için charAt() kullanın.

var string = "Hello, World!";
console.log( string.charAt(4) ); // "o"

Alternatif olarak, dizeler diziler gibi ele alınabileceğinden, braket notasyonu aracılığıyla dizini kullanın.

var string = "Hello, World!";
console.log( string[4] ); // "o"

Belirtilen dizindeki karakterin karakter kodunu almak için charCodeAt() kullanın. .

var string = "Hello, World!";
console.log( string.charCodeAt(4) ); // 111

Kaçan alıntılar #

Dizeniz tek tırnak içine alınmışsa (yani) içteki değişmez alıntıdan ters eğik çizgi `' ile çıkmanız gerekir.

var text = 'L\'albero means tree in Italian';
console.log( text ); \\ "L'albero means tree in Italian"

Aynısı çift tırnak için de geçerlidir:

var text = "I feel \"high\"";

HTML temsillerini bir Dize içinde saklıyorsanız, alıntılardan kaçan alıntılara özel dikkat gösterilmelidir, çünkü HTML dizeleri alıntılardan, yani niteliklerden büyük ölçüde yararlanır:

var content = "<p class=\"special\">Hello World!</p>";        // valid String
var hello   = '<p class="special">I\'d like to say "Hi"</p>'; // valid String

HTML dizelerindeki tırnak işaretleri, tek tırnak olarak “'” (veya “'”) ve çift tırnak olarak “"” (veya “"”) kullanılarak da gösterilebilir.

var hi    = "<p class='special'>I'd like to say &quot;Hi&quot;</p>"; // valid String
var hello = '<p class="special">I&apos;d like to say "Hi"</p>';      // valid String

Not: “'” ve “"” kullanımı, tarayıcıların özellik tırnaklarına otomatik olarak yerleştirebileceği çift tırnakların üzerine yazmaz. Örneğin, <p class="özel"> için <p class="özel"> yapılması, &quot; kullanılması <p class=""özel"">ne yol açabilir, burada \" <p class="özel"> olun.

Boşluğu kırp #

Bir dizenin kenarlarındaki boşlukları kesmek için String.prototype.trim kullanın:

"    some whitespaced string  ".trim();  // "some whitespaced string"

Birçok JavaScript motoru, ancak Internet Explorer değil, standart olmayan “trimLeft"i uyguladı ve “trimRight” yöntemleri. Standartlaştırılmış ’trimStart’ ve ’trimEnd’ yöntemleri için şu anda sürecin 1. Aşamasında olan bir teklif vardır. uyumluluk için trimLeftvetrimRight`.

// Stage 1 proposal
"    this is me    ".trimStart();  // "this is me    "
"    this is me    ".trimEnd();  // "    this is me"

// Non-standard methods, but currently implemented by most engines
"    this is me    ".trimLeft();  // "this is me    "
"    this is me    ".trimRight();  // "    this is me"

Bir dizeyi diziye bölme #

Dizelerden bölünmüş alt dizeler dizisine gitmek için .split kullanın:

var s = "one, two, three, four, five"
s.split(", ");  // ["one", "two", "three", "four", "five"]

Bir dizgeye geri dönmek için dizi yöntemini .join kullanın:

s.split(", ").join("--");  // "one--two--three--four--five"

Bir dize algılama #

Bir parametrenin ilkel bir dize olup olmadığını saptamak için “typeof” kullanın:

var aString = "my string";
var anInt = 5;
var anObj = {};
typeof aString === "string";   // true
typeof anInt === "string";     // false
typeof anObj === "string";     // false

Eğer new String("somestr") aracılığıyla bir String nesneniz varsa, yukarıdakiler çalışmayacaktır. Bu durumda, “instanceof” kullanabiliriz:

var aStringObj = new String("my string");
aStringObj instanceof String;    // true

Her iki durumu da kapsamak için basit bir yardımcı fonksiyon yazabiliriz:

var isString = function(value) {
    return typeof value === "string" || value instanceof String;
};

var aString = "Primitive String";
var aStringObj = new String("String Object");
isString(aString); // true
isString(aStringObj); // true
isString({}); // false
isString(5); // false

Veya ‘Object’in ’toString’ fonksiyonunu kullanabiliriz. Bu yöntem, tıpkı ’typeof’ gibi diğer veri türlerini de desteklediğinden, bir switch ifadesinde söylendiği gibi diğer türleri de kontrol etmemiz gerekiyorsa bu yararlı olabilir.

var pString = "Primitive String";
var oString = new String("Object Form of String");
Object.prototype.toString.call(pString);//"[object String]"
Object.prototype.toString.call(oString);//"[object String]"
