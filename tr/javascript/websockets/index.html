  • Sözdizimi

WebSocket, bir istemci ve sunucu arasında iki yönlü iletişimi sağlayan bir protokoldür:

WebSocket’in amacı, tarayıcı tabanlı bir mekanizma sağlamaktır. yapan sunucularla iki yönlü iletişime ihtiyaç duyan uygulamalar birden fazla HTTP bağlantısı açmaya güvenmeyin. (RFC 6455)

WebSocket, HTTP protokolü üzerinden çalışır.

Sözdizimi #

  • yeni WebSocket(url)
  • ws.binaryType /* alınan mesajın teslim türü: “arraybuffer” veya “blob” */
  • ws.kapat()
  • ws.send(veri)
  • ws.onmessage = fonksiyon(mesaj) { /* … */ }
  • ws.onopen = function() { /* … */ }
  • ws.onerror = function() { /* … */ }
  • ws.onclose = function() { /* … */ }

Parametreler #

parametre Ayrıntılar
url Bu web soket bağlantısını destekleyen sunucu url’si.
veri Ana bilgisayara gönderilecek içerik.
mesaj Ana bilgisayardan alınan mesaj.

Dize mesajlarıyla çalışma #

var wsHost = "ws://my-sites-url.com/path/to/echo-web-socket-handler";
var ws = new WebSocket(wsHost);
var value = "an example message";

//onmessage : Event Listener - Triggered when we receive message form server
ws.onmessage = function(message) {
    if (message === value) {
        console.log("The echo host sent the correct message.");
    } else {
        console.log("Expected: " + value);
        console.log("Received: " + message);
    }
};

//onopen : Event Listener - event is triggered when websockets readyState changes to open which means now we are ready to send and receives messages from server
ws.onopen = function() {
    //send is used to send the message to server
    ws.send(value);
};

Bir web soket bağlantısı kurun #

var wsHost = "ws://my-sites-url.com/path/to/web-socket-handler";
var ws = new WebSocket(wsHost);

İkili mesajlarla çalışma #

var wsHost = "http://my-sites-url.com/path/to/echo-web-socket-handler";
var ws = new WebSocket(wsHost);
var buffer = new ArrayBuffer(5); // 5 byte buffer
var bufferView = new DataView(buffer);

bufferView.setFloat32(0, Math.PI);
bufferView.setUint8(4, 127);

ws.binaryType = 'arraybuffer';

ws.onmessage = function(message) {
    var view = new DataView(message.data);
    console.log('Uint8:', view.getUint8(4), 'Float32:', view.getFloat32(0))
};

ws.onopen = function() {
    ws.send(buffer);
};

Güvenli bir web soket bağlantısı yapma #

var sck = "wss://site.com/wss-handler";
var wss = new WebSocket(sck);

Bu, HTTP yerine HTTPS kullanan güvenli bir web soketi bağlantısı kurmak için “ws” yerine “wss"yi kullanır.
