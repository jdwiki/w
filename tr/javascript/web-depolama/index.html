Sözdizimi #

  • localStorage.setItem(ad, değer);

  • localStorage.getItem(ad);

  • localStorage.name = değer;

  • localStorage.name;

  • localStorage.clear()

  • localStorage.removeItem(ad);

Parametreler #

parametre Açıklama
isim Öğenin anahtarı/adı
değer Öğenin değeri

Web Depolama API’si WHATWG HTML Yaşam Standardında belirtilmiştir.

localStorage’ı kullanma #

localStorage nesnesi, dizelerin kalıcı (ancak kalıcı değil - aşağıdaki sınırlara bakın) anahtar/değer depolamasını sağlar. Herhangi bir değişiklik, aynı kaynaktan gelen diğer tüm pencerelerde/çerçevelerde hemen görülebilir. Kullanıcı kaydedilen verileri temizlemedikçe veya bir son kullanma sınırı yapılandırmadıkça, saklanan değerler süresiz olarak kalıcıdır. localStorage, değerleri almak ve ayarlamak için harita benzeri bir arabirim kullanır.

localStorage.setItem('name', "John Smith");
console.log(localStorage.getItem('name')); // "John Smith"

localStorage.removeItem('name');
console.log(localStorage.getItem('name')); // null

Basit yapılandırılmış verileri depolamak istiyorsanız, depolama için dizelere ve dizelerden seri hale getirmek için JSON kullanabilirsiniz.

var players = [{name: "Tyler", score: 22}, {name: "Ryan", score: 41}];
localStorage.setItem('players', JSON.stringify(players));

console.log(JSON.parse(localStorage.getItem('players')));
// [ Object { name: "Tyler", score: 22 }, Object { name: "Ryan", score: 41 } ]

Depolamayı işlemenin daha basit yolu #

’localStorage’, ‘sessionStorage’ JavaScript Nesneleridir ve onlara bu şekilde davranabilirsiniz. .getItem(), .setItem() vb. gibi Depolama Yöntemleri kullanmak yerine… işte daha basit bir alternatif:

// Set
localStorage.greet = "Hi!"; // Same as: window.localStorage.setItem("greet", "Hi!");

// Get
localStorage.greet;         // Same as: window.localStorage.getItem("greet");

// Remove item
delete localStorage.greet;  // Same as: window.localStorage.removeItem("greet");

// Clear storage
localStorage.clear();

Örnek:

// Store values (Strings, Numbers)
localStorage.hello = "Hello";
localStorage.year  = 2017;    

// Store complex data (Objects, Arrays)
var user = {name:"John", surname:"Doe", books:["A","B"]};
localStorage.user = JSON.stringify( user );

// Important: Numbers are stored as String
console.log( typeof localStorage.year ); // String

// Retrieve values
var someYear = localStorage.year; // "2017"

// Retrieve complex data
var userData = JSON.parse( localStorage.user );
var userName = userData.name; // "John"

// Remove specific data
delete localStorage.year; 

// Clear (delete) all stored data
localStorage.clear();

Depolama etkinlikleri #

localStorage içinde bir değer ayarlandığında, aynı kaynaktan diğer tüm “pencerelerde” bir “storage” olayı gönderilir. Bu, durumu yeniden yüklemeden veya bir sunucuyla iletişim kurmadan farklı sayfalar arasında durumu senkronize etmek için kullanılabilir. Örneğin, bir girdi öğesinin değerini başka bir pencerede paragraf metni olarak yansıtabiliriz:

İlk Pencere #

var input = document.createElement('input');
document.body.appendChild(input);

input.value = localStorage.getItem('user-value');

input.oninput = function(event) {
  localStorage.setItem('user-value', input.value);
};

İkinci Pencere #

var output = document.createElement('p');
document.body.appendChild(output);

output.textContent = localStorage.getItem('user-value');

window.addEventListener('storage', function(event) {
  if (event.key === 'user-value') {
    output.textContent = event.newValue;
  }
});

Notlar #

Etki alanı komut dosyası aracılığıyla değiştirilmişse, etkinlik başlatılmaz veya Chrome, Edge ve Safari altında yakalanamaz.

İlk pencere #

// page url: http://sub.a.com/1.html
document.domain = 'a.com';

var input = document.createElement('input');
document.body.appendChild(input);

input.value = localStorage.getItem('user-value');

input.oninput = function(event) {
  localStorage.setItem('user-value', input.value);
};

İkinci Pencere #

// page url: http://sub.a.com/2.html
document.domain = 'a.com';

var output = document.createElement('p');
document.body.appendChild(output);

// Listener will never called under Chrome(53), Edge and Safari(10.0).
window.addEventListener('storage', function(event) {
  if (event.key === 'user-value') {
    output.textContent = event.newValue;
  }
});

oturumDepolama #

sessionStorage nesnesi, localStorage ile aynı Depolama arabirimini uygular. Ancak, aynı kaynaktaki tüm sayfalarla paylaşılmak yerine, sessionStorage verileri her pencere/sekme için ayrı olarak depolanır. Depolanan veriler, açık olduğu sürece o penceredeki/sekmedeki sayfalar arasında kalır, ancak başka hiçbir yerde görünmez.

var audio = document.querySelector('audio');

// Maintain the volume if the user clicks a link then navigates back here.
audio.volume = Number(sessionStorage.getItem('volume') || 1.0);
audio.onvolumechange = function(event) {
  sessionStorage.setItem('volume', audio.volume);
};

Verileri sessionStorage’a kaydedin

sessionStorage.setItem('key', 'value');

sessionStorage’dan kayıtlı verileri alın

var data = sessionStorage.getItem('key');

Kaydedilmiş verileri sessionStorage’dan kaldırın

sessionStorage.removeItem('key')

Depolama alanını temizleme #

Depoyu temizlemek için, sadece çalıştırın

localStorage.clear();

Hata koşulları #

Çoğu tarayıcı, çerezleri engelleyecek şekilde yapılandırıldığında “localStorage"ı da engeller. Bunu kullanma girişimleri bir istisna ile sonuçlanacaktır. Bu vakaları yönetmeyi unutmayın.

var video = document.querySelector('video')
try {
    video.volume = localStorage.getItem('volume')
} catch (error) {
    alert('If you\'d like your volume saved, turn on cookies')
}
video.play()

Hata işlenmediyse, program düzgün çalışmayı durduracaktır.

Depolama Öğesini Kaldır #

Belirli bir öğeyi tarayıcı Deposundan kaldırmak için (“setItem"in tersi) “removeItem"i kullanın

localStorage.removeItem("greet");

Örnek:

localStorage.setItem("greet", "hi");
localStorage.removeItem("greet");

console.log( localStorage.getItem("greet") ); // null

(Aynı sessionStorage için de geçerlidir)

localDepolama uzunluğu #

“localStorage.length” özelliği, “localStorage” içindeki öğelerin sayısını belirten bir tamsayı döndürür

Örnek:

Öğeleri Ayarla

localStorage.setItem('StackOverflow', 'Documentation');
localStorage.setItem('font', 'Helvetica');
localStorage.setItem('image', 'sprite.svg');

uzunluk al

localStorage.length; // 3
