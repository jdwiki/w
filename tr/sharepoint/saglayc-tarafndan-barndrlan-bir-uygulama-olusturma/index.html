Geliştirme ortamını ayarlama #

Uygulama Geliştirmeye başlamak için Visual studio 2013 veya daha yüksek bir sürüme ihtiyacımız var. En son topluluk veya ifade sürümünü buradan indirin > https://www.visualstudio.com/products/free-developer-offers-vs

Bir kez indirilip kurulduktan sonra

Aç ve Yeni proje oluştur‘a tıklayın

Office/SharePoint bölümünü genişletin, aşağıda gösterildiği gibi Uygulama için bir seçenek görmelisiniz.

Uygulama seçeneği mevcut değilse VS’yi kapatın, Microsoft Office Developer Tools‘u indirin ve yükleyin https://www.visualstudio.com/en-us/features/office-tools-vs.aspx

Geliştirici sitesi için hazırlanıyor #

Görsel stüdyomuz olduğunda, uygulamaları SharePoint’e dağıtmak için bir geliştirici sitesine ihtiyacımız var. Almanın en basit yolu şudur: > Ücretsiz, bir yıllık Office 365 geliştirici hesabı için kaydolun https://profile.microsoft.com/RegSysProfileCenter/wizardnp.aspx?wizid=14b845d0-938c-45af-b061-f798fbb4d170&lcid=1033

Kayıt işlemi tamamlandıktan sonra, tüm Uygulamalarınız için https://www.office.com/ center URL’si

Visual Studio’da Uygulama Oluşturun #

İlk uygulamamızı oluşturmaya başlayalım

  1. Görsel stüdyoyu açın ve > yeni proje oluşturun

  2. Adı ve Konumu Girin

  3. Önceki adımda oluşturduğunuz geliştirici site URL’nizi girin ve Sağlayıcı tarafından barındırılan’ı seçin.

  1. Giriş için olduğu gibi açılır pencere açılacaktır

  2. Bir sonraki adımda uygulama tipine göre MVC veya Webform’u seçin. Burada MCV’yi seçiyorum

  3. Eklentinizin kimlik doğrulamasını nasıl istiyorsunuz? altında, Windows Azure Erişim Denetim Hizmetini Kullan’ı seçin ve Bitir’e tıklayın.

  4. Solution Explorer’da 2 projenin oluşturulduğunu görebiliriz. Biri SharePoint uygulama bölümü, diğeri ise asp.net web uygulaması

Kodlamaya başlayalım #

Burada temel bir haber uygulaması örneğini alıyorum

  1. SharePoint geliştirici sitesini açın ve bilgilerimizi depolamak için bir liste oluşturun. news articles

  2. Özel bir liste oluşturun ve Body, Summery, ThumbnailImageUrl 3 sütun daha ekleyin

  3. SharePoint uygulamamıza geri dönün, AppManifest.xml dosyasını açın, izin Sekmesine tıklayın ve site koleksiyonuna Okuma izni verin ve kaydedin.

  4. HomeController’ı web uygulamasından açın, benim durumumda bu bir MVC uygulaması. Bir web formu uygulaması oluşturuyorsanız, kodunuz default.aspx.cs sayfasında olmalıdır.

  5. Listeden en son haberleri almak için kod parçacığı aşağıdadır. Dizin sayfamız böyle görünmelidir.

     [SharePointContextFilter]
 public ActionResult Index()
 {
     User spUser = null;

     var spContext = SharePointContextProvider.Current.GetSharePointContext(HttpContext);
     List<NewsList> newsList = new List<NewsList>();
     using (var clientContext = spContext.CreateUserClientContextForSPHost())
     {
         if (clientContext != null)
         {
             spUser = clientContext.Web.CurrentUser;

             clientContext.Load(spUser, user => user.Title);

             clientContext.ExecuteQuery();

             ViewBag.UserName = spUser.Title;

             List lst = clientContext.Web.Lists.GetByTitle("News");
             CamlQuery queryNews = CamlQuery.CreateAllItemsQuery(10);
             ListItemCollection newsItems = lst.GetItems(queryNews);
             clientContext.Load(newsItems, includes => includes.Include(i => i.Id, i => i.DisplayName, i => i["ThumbnailImageUrl"], i => i["Summery"]));

             clientContext.ExecuteQuery();

             if (newsItems != null)
             {
                 foreach (var lstProductItem in newsItems)
                 {
                     newsList.Add(
                         new NewsList
                         {
                             Id = Convert.ToInt32(lstProductItem.Id.ToString()),
                             Title = lstProductItem.DisplayName.ToString(),
                             Summery = lstProductItem["Summery"].ToString(),
                             Thumbnail = lstProductItem["ThumbnailImageUrl"].ToString()
                         });
                 }
             }
         }
     }

     return View(newsList);
 }

    Tam makale sayfası oluşturma #

    Tüm haber makalelerini gösterecek olan ilk sayfayı zaten oluşturduk. Bu sayfa Komple makaleyi gösterecektir.

    1. HomeController’a Bir Eylem Yöntemi Daha Ekleyin

       [SharePointContextFilter]
 public ActionResult Aticle(int ArticleId)
 {
     User spUser = null;

     var spContext = SharePointContextProvider.Current.GetSharePointContext(HttpContext);
     FullArticle article = new FullArticle();
     using (var clientContext = spContext.CreateUserClientContextForSPHost())
     {
         if (clientContext != null)
         {
             spUser = clientContext.Web.CurrentUser;

             clientContext.Load(spUser, user => user.Title);

             clientContext.ExecuteQuery();

             ViewBag.UserName = spUser.Title;

             List lst = clientContext.Web.Lists.GetByTitle("News");
             CamlQuery queryNews = new CamlQuery();
             queryNews.ViewXml = @"<View><Query><Where><Eq><FieldRef Name='ID'/>" + "<Value Type='Number'>" + ArticleId + "</Value></Eq></Where></Query>" +
                 "<ViewFields><FieldRef Name='ID'/><FieldRef Name='Title'/><FieldRef Name='Body'/></ViewFields></View>";//
             ListItemCollection newsItems = lst.GetItems(queryNews);
             clientContext.Load(newsItems, includes => includes.Include(i => i.Id, i => i.DisplayName, i => i["Body"]));

             clientContext.ExecuteQuery();

             if (newsItems != null)
             {
                 foreach (var lstProductItem in newsItems)
                 {
                     article.Id = Convert.ToInt32(lstProductItem.Id.ToString());
                     article.Title = lstProductItem.DisplayName.ToString();
                     article.Body = lstProductItem["Body"].ToString();
                 }
             }
         }
     }
     return View(article);
 }
