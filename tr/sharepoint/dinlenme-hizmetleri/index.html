REST Hizmet Bitiş Noktası URL'leri

REST istemci erişim API’si ilk olarak SharePoint 2010’da tanıtıldı, ancak SharePoint 2013’te büyük ölçüde genişletildi. REST API [SharePoint 2010](https://msdn.microsoft.com/en-us/library/office/ff521587(v) =office.14).aspx), /_vti_bin/ListData.svc url’sindeki ListData web hizmeti aracılığıyla erişilir. SharePoint 2013 biraz farklı davranan “/_api/lists/” ve “/_api/web” uç nokta URL’lerini tanıttı .

Yukarıdaki uç nokta URL’lerinden önce “http://sunucu/site” gelmelidir; burada “sunucu”, sunucunun adını ve “site”, belirli sitenin adını veya yolunu temsil eder.

Örnek URL… 2010 SharePoint 2013
Liste Alma: /_vti_bin/ListData.svc/ListName /_api/lists('ListGuid')
Öğe Getirme: /_vti_bin/ListData.svc/ListName(1) /_api/lists('ListGuid')/items(1)
Web Alma: (eşdeğeri yok) /_api/web

Listelere ve liste öğelerine erişimdeki farklılıklara rağmen, bu sonuçlarla çalışmak her iki sürümde de çok benzer.

‘ListData.svc’ hizmetinin geriye dönük uyumluluk için SharePoint 2013’te hala mevcut olduğunu unutmayın.

REST İstekleri Gönderme

Bir REST isteği, yerel bir JavaScript XMLHttpRequest aracılığıyla veya jQuery AJAX sarmalayıcı yapısı aracılığıyla gönderilebilir.

XMLHttpRequest Sözdizimi

var xhr = new XMLHttpRequest();
xhr.open(verb, url, true);
xhr.setRequestHeader("Content-Type","application/json");
xhr.send(data);

jQuery AJAX Sözdizimi

$.ajax({
    method: verb,
    url: url,
    headers: { "Content-Type":"application/json" },
    data: data
});

AJAX aracılığıyla istek gönderme hakkında daha fazla ayrıntı için JavaScript AJAX belgelerine bakın.

Listelerle Çalışma #

Liste Öğelerini Alma

Bu örnek, tüm liste öğelerinin nasıl alınacağını ve bunlar arasında nasıl yineleneceğini gösterir. Belirli sayıda sonuç istemek için “top” parametresini kullanabilirsiniz. Belirli alanları ($select=id, Title, uri) seçmek için select parametresini de kullanabilirsiniz.

JavaScript

function GetListItems(){
    $.ajax({
        url: "../_api/web/lists/getbytitle('List Title')/items?$top=50"
        contentType: "application/json;odata=verbose",
        method: "GET",
        headers: { "accept": "application/json;odata=verbose" },
        success: function (data) {
            $.each(data.d.results, function(index,item){
                //use item to access the individual list item
                console.log(item.Id);
            });
        },
        error: function(error){
            console.log(error);
        }
    });
    }

Arama Sütunlarıyla Liste Öğelerini Alın #

Bazen şuna benzeyen bir liste yapınız olabilir:

İsim Tip Açıklama
Başlık Dize (Metin) Hayvanın adı
Yaş Sayı hayvan kaç yaşında
Değer para Hayvanın değeri
Tip Arama (Hayvan Türleri Tablosu) Arama Alanı (Hayvan Türleri Tablosundaki seçeneklerin açılır listesini verir)

Hayvan Türleri Tablosu #

İsim Tip Açıklama
Başlık Dize (Metin) Türün/hayvan türünün adı (örn. Domuz)
NumLegs Sayı Hayvanın bacak sayısı

Muhtemelen en ciddi örnek değil ama buradaki sorun hala geçerli. SharePoint listesinden değerleri almak için olağan isteği kullandığınızda, hayvanın ‘Türü’ için JSON yanıtında yalnızca ‘TypeId’ adlı bir alanı geri alırsınız. Bu öğeleri yalnızca tek bir AJAX çağrısında genişletmek için, URL parametrelerinde bazı ekstra işaretlemeler gerekir.

Bu örnek, arama sütunlarından daha fazlası için de geçerlidir. “Kişiler/Gruplar” sütunlarını kullandığınızda, bunlar aslında yalnızca aramalardır, böylece “Başlık”, “E-posta” ve diğerleri gibi öğeleri kolayca çekebilirsiniz.

Örnek Kod #

Önemli Not: Arama sütunlarından geri dönmek istediğiniz alanları tanımlarken, orijinal tablodaki alan adının önüne arama alanı adını eklemelisiniz. Örneğin, arama sütunundan ‘NumLegs’ özniteliğini geri almak istiyorsanız, ‘Type/NumLegs’ yazmalısınız.

JavaScript

// webUrl: The url of the site (ex. https://www.contoso.com/sites/animals)
// listTitle: The name of the list you want to query
// selectFields: the specific fields you want to get back
// expandFields: the name of the fields that need to be pulled from lookup tables
// callback: the name of the callback function on success
function getItems(webUrl,listTitle,selectFields, expandFields, callback){
    var endpointUrl = webUrl + "/_api/web/lists/getbytitle('" + listTitle + "')/items";
    endpointUrl+= '?$select=' + selectFields.join(",");
    endpointUrl+= '&$expand=' + expandFields.join(",");
    return executeRequest(endpointUrl,'GET', callback);
}

function executeRequest(url,method,callback,headers,payload) 
{
    if (typeof headers == 'undefined'){
        headers = {};
    }
    headers["Accept"] = "application/json;odata=verbose";
    if(method == "POST") {
        headers["X-RequestDigest"] = $("#__REQUESTDIGEST").val();
    }   

    var ajaxOptions = 
    {       
    url: url,   
    type: method,  
    contentType: "application/json;odata=verbose",
    headers: headers,
    success: function (data) { callback(data) }
    };
    if(method == "POST") {
    ajaxOptions.data = JSON.stringify(payload);
    }  

    return $.ajax(ajaxOptions);
}

// Setup the ajax request by setting all of the arguments to the getItems function
function getAnimals() {
    var url = "https://www.contoso.com/sites/animals";
    var listTitle = "AnimalListing";
    
    var selectFields = [
        "Title",
        "Age",
        "Value",
        "Type/Title",
        "Type/NumLegs"
    ];

    var expandFields = [
        "Type/Title",
        "Type/NumLegs"
    ];

    getItems(url, listTitle, selectFields, expandFields, processAnimals);
}

// Callback function
// data: returns the data given by SharePoint
function processAnimals(data) {
    console.log(data);
    // Process data here
}

// Start the entire process
getAnimals();

Bir sorgudan döndürülen çağrı listesi öğeleri #

REST kullanarak sayfalamayı simüle etmek için aşağıdakileri yapabilirsiniz:

  1. ‘$orderby’ parametresine göre ilk ’n’ girişlerini atlamak için ‘$skip=n’ parametresini kullanın

  2. “$orderby” ve “$atla” parametrelerine göre en üstteki “n” girişlerini döndürmek için “$top=n” parametresini kullanın.

    var endpointUrl = “/_api/lists(‘guid’)/items”; // SP2010: “/_vti_bin/ListData.svc/ListName”; $.getJSON( endpointUrl + “?$orderby=Id&$top=1000”, function(data){ processData(data); // you can do something with the results here var count = data.d.results.length; getNextBatch(count, processData, onComplete); // fetch next page } );

    function getNextBatch(totalSoFar, processResults, onCompleteCallback){ $.getJSON( endpointUrl + “?$orderby=Id&$skip="+totalSoFar+”&$top=1000", function(data){ var count = data.d.results.length; if(count > 0){ processResults(data); // do something with results getNextBatch(totalSoFar+count, callback); // fetch next page }else{ onCompleteCallback(); } } ); }

SharePoint listesinde yeni oluşturulan öğenin kimliğini alın #

Bu örnek, SharePoint REST API kullanılarak yeni oluşturulan bir öğenin kimliğinin nasıl alınacağını gösterir.

Not :

listName - Bu değişken listenizin adını içerir.

newItemBody - Bu, listeye yeni öğe eklemek için istek gövdeniz olacaktır.

Örneğin. var newItemBody = { __metadata: { ’type’: ‘SP.Data.MyListNameItem’ }, Title: ‘Some title value’ };

function CreateListItemWithDetails(listName, newItemBody) {

    var item = newItemBody;
    return $.ajax({
        url: _spPageContextInfo.siteAbsoluteUrl + "/_api/web/lists/getbytitle('" + listName + "')/items",
        type: "POST",
        contentType: "application/json;odata=verbose",
        data: JSON.stringify(item),
        headers: {
            "Accept": "application/json;odata=verbose",
            "X-RequestDigest": $("#__REQUESTDIGEST").val(),
            "content-Type": "application/json;odata=verbose"
        }

    });
}

CreateListItemWithDetails(listName, newItemBody)
    .then(function(data){
        //success callback
        var NewlyCreatedItemId = data.d.ID;
    }, function(data){
        //failure callback
    });

SharePoint 2010 REST Arayüzü kullanılarak CRUD işlemleri nasıl yapılır #

Oluşturmak

REST üzerinden Create işlemi gerçekleştirmek için aşağıdaki işlemleri gerçekleştirmelisiniz:

‘POST’ fiilini kullanarak bir HTTP isteği oluşturun. POST hedefi olarak bir varlık eklemek istediğiniz listenin hizmet URL’sini kullanın. İçerik türünü ‘application/json’ olarak ayarlayın. Yeni liste öğelerinizi bir dize olarak temsil eden JSON nesnelerini seri hale getirin ve bu değeri istek gövdesine ekleyin JavaScript örneği:

function createListItem(webUrl,listName, itemProperties, success, failure) {

    $.ajax({
        url: webUrl + "/_vti_bin/listdata.svc/" + listName,
        type: "POST",
        processData: false,
        contentType: "application/json;odata=verbose",
        data: JSON.stringify(itemProperties),
        headers: {
            "Accept": "application/json;odata=verbose"
        },
        success: function (data) {
            success(data.d);
        },
        error: function (data) {
            failure(data.responseJSON.error);
        }
    });
}

kullanım

var taskProperties = {
    'TaskName': 'Order Approval',
    'AssignedToId': 12
};

createListItem('https://contoso.sharepoint.com/project/','Tasks',taskProperties,function(task){
    console.log('Task' + task.TaskName + ' has been created'); 
  },
  function(error){
    console.log(JSON.stringify(error));
  }
);

Okumak

REST üzerinden bir Okuma işlemi gerçekleştirmek için aşağıdaki işlemleri gerçekleştirmelisiniz:

‘GET’ fiilini kullanarak bir HTTP isteği oluşturun. GET için hedef olarak bir varlık eklemek istediğiniz liste öğesinin hizmet URL’sini kullanın. İçerik türünü ‘application/json’ olarak ayarlayın. JavaScript örneği:

function getListItemById(webUrl,listName, itemId, success, failure) {
    var url = webUrl + "/_vti_bin/listdata.svc/" + listName + "(" + itemId + ")";
    $.ajax({
        url: url,
        method: "GET",
        headers: { "Accept": "application/json; odata=verbose" },
        success: function (data) {
            success(data.d);
        },
        error: function (data) {
            failure(data.responseJSON.error);
        }
    });
}

kullanım

getListItemById('https://contoso.sharepoint.com/project/','Tasks',2,function(taskItem){
    console.log(taskItem.TaskName); 
  },
  function(error){
    console.log(JSON.stringify(error));
  }
);

Güncelleme

Mevcut bir varlığı güncellemek için aşağıdaki eylemleri gerçekleştirmelisiniz:

‘POST’ fiilini kullanarak bir HTTP isteği oluşturun. “MERGE” değerine sahip bir “X-HTTP-Method” başlığı ekleyin. POST hedefi olarak güncellemek istediğiniz liste öğesinin hizmet URL’sini kullanın Varlığın orijinal ETag değeri veya * ile bir “If-Match” başlığı ekleyin. JavaScript örneği:

function updateListItem(webUrl,listName,itemId,itemProperties,success, failure)
{
   getListItemById(webUrl,listName,itemId,function(item){

      $.ajax({
         type: 'POST',
         url: item.__metadata.uri,
         contentType: 'application/json',
         processData: false,
         headers: {
                "Accept": "application/json;odata=verbose",
                "X-HTTP-Method": "MERGE",
                "If-Match": item.__metadata.etag
         },
         data: Sys.Serialization.JavaScriptSerializer.serialize(itemProperties),
         success: function (data) {
                success(data);
         },
         error: function (data) {
                failure(data);
         }
      });

   },
   function(error){
       failure(error);
   });

}

kullanım

var taskProperties = {
    'TaskName': 'Approval',
    'AssignedToId': 12  
};


updateListItem('https://contoso.sharepoint.com/project/','Tasks',2,taskProperties,function(item){
    console.log('Task has been updated'); 
  },
  function(error){
    console.log(JSON.stringify(error));
  }
);

Silmek

Bir varlığı silmek için aşağıdaki eylemleri gerçekleştirmelisiniz:

‘POST’ fiilini kullanarak bir HTTP isteği oluşturun. “DELETE” değerine sahip bir “X-HTTP-Method” başlığı ekleyin. POST hedefi olarak güncellemek istediğiniz liste öğesinin hizmet URL’sini kullanın Varlığın orijinal ETag değerine sahip bir “If-Match” başlığı ekleyin. JavaScript örneği:

function deleteListItem(webUrl, listName, itemId, success, failure) {
    getListItemById(webUrl,listName,itemId,function(item){
        $.ajax({
            url: item.__metadata.uri,
            type: "POST",
            headers: {
                "Accept": "application/json;odata=verbose",
                "X-Http-Method": "DELETE",
                "If-Match": item.__metadata.etag
            },
            success: function (data) {
                success();
            },
            error: function (data) {
                failure(data.responseJSON.error);
            }
        });
    },
   function (error) {
       failure(error);
   });
}

kullanım

deleteListItem('https://contoso.sharepoint.com/project/','Tasks',3,function(){
    console.log('Task has been deleted'); 
  },
  function(error){
    console.log(JSON.stringify(error));
  }
);
