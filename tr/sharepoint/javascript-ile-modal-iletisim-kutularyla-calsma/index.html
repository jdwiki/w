Sözdizimi #

  • var options = SP.UI.$create_DialogOptions();

  • var modalDialog = SP.UI.ModalDialog.showModalDialog(seçenekler);

Parametreler #

seçenekler Özellik Açıklama
başlık İletişim kutusunun başlığını içeren bir dize
url İletişim kutusunda görünen sayfanın URL’sini içeren bir dize. url veya html belirtilmelidir. url, html‘den önceliklidir.
html İletişim kutusunda görüntülenecek bir HTML öğesi.
x Bir tamsayı değeri olarak iletişim kutusunun x-ofseti.
y Bir tamsayı değeri olarak iletişim kutusunun y-kayması.
genişlik Bir tamsayı değeri olarak iletişim kutusunun genişliği. Belirtilmemişse ve otomatik boyutlandırma yanlış ise genişlik 768 piksel olarak ayarlanır
yükseklik Bir tamsayı değeri olarak iletişim kutusunun yüksekliği. Belirtilmemişse ve otomatik boyutlandırma yanlış ise, yükseklik 576 piksel olarak ayarlanır
allowMaximize Büyüt düğmesinin gösterilip gösterilmeyeceğini belirten bir Boole değeri.
gösterMaksimumlaştırılmış İletişim kutusunun ekranı kaplamış olarak açılıp açılmayacağını belirten bir Boole değeri.
gösterKapat İletişim kutusunda Kapat düğmesinin görünüp görünmeyeceğini belirten bir Boole değeri.
otomatikBoyut İletişim platformunun iletişim kutusu boyutlandırmasını otomatik olarak işleyip işlemediğini belirten bir Boole değeri.
dialogReturnValueCallback Geri arama işlevini belirten bir işlev işaretçisi. İşlev iki parametre alır: SP.UI.DialogResult Enumeration türünde bir dialogResult ve iletişim kutusu tarafından döndürülen verileri içeren bir returnValue nesnesi.
argümanlar İletişim kutusuna iletilen verileri içeren bir nesne.

“SP.UI.ModalDialog” ad alanı, SharePoint 2010 ile JavaScript Nesne Modeli ile tanıtıldı ve sonraki SharePoint 2013, Office365 ve 2016 sürümlerinde kullanılabilir.

Ek referans malzemeleri:

  • [SP.UI.ModalDialog.showModalDialog(options) için MSDN Referansı](https://msdn.microsoft.com/en-us/library/office/ff410058%28v=office.14%29.aspx?f=255&MSPPError =-2147217396)
  • SP.UI.DialogResult Enumeration için MSDN Referansı

Bir İletişim Kutusu Kapatıldığında Bir Eylem Gerçekleştirin #

SP.SOD.executeOrDelayUntilScriptLoaded(showDialog,"sp.js");

function showDialog(){
    var options = SP.UI.$create_DialogOptions();
    options.url = "/mySite/lists/myList/NewForm.aspx";
    options.dialogReturnValueCallback = myCallBackFunction;
    SP.UI.ModalDialog.showModalDialog(options);    
    function myCallBackFunction(result,data){
        switch(result){
            case SP.UI.DialogResult.invalid: 
                alert("The dialog result was invalid"); 
                break;
            case SP.UI.DialogResult.cancel: 
                alert("You clicked cancel or close"); 
                break;
            case SP.UI.DialogResult.OK: 
                alert("You clicked OK, creating an item in the list."); 
                break;
        }
    }
}

Bir İletişim Kutusunda Mevcut Bir Sayfayı Göster #

SP.SOD.executeOrDelayUntilScriptLoaded(showDialog,"sp.js");

function showDialog(){
    SP.UI.ModalDialog.showModalDialog(
        { url: "/org/it/web/wik/Lists/ExampleCode/DispForm.aspx?ID=6" }
    );
}

Özel Bir İletişim Kutusu Göster #

SP.SOD.executeOrDelayUntilScriptLoaded(showDialog,"sp.js");

function showDialog(){
    var dialogOptions = SP.UI.$create_DialogOptions();
    dialogOptions.title = "Your Title Here!";
    var dummyElement = document.createElement("div");
    dummyElement.style.textAlign = "center";
    dummyElement.appendChild(document.createElement("br"));
    dummyElement.appendChild(document.createTextNode("Some beautifully crafted text."));
    dummyElement.appendChild(document.createElement("br"));
    dialogOptions.html = dummyElement;
    SP.UI.ModalDialog.showModalDialog(dialogOptions);    
}
