Dizileri Kopyalama #

= operatörünü kullanarak bir VBA dizisini aynı türdeki bir diziye kopyalayabilirsiniz. Diziler aynı türde olmalıdır, aksi takdirde kod “Diziye atanamıyor” derleme hatası verir.

Dim source(0 to 2) As Long
Dim destinationLong() As Long
Dim destinationDouble() As Double

destinationLong = source     ' copies contents of source into destinationLong
destinationDouble = source   ' does not compile

Kaynak dizi sabit veya dinamik olabilir, ancak hedef dizi dinamik olmalıdır. Sabit bir diziye kopyalamaya çalışmak, “Dizeye atanamıyor” derleme hatası verecektir. Alıcı dizide önceden var olan tüm veriler kaybolur ve sınırları ve boyutları kaynak diziyle aynı şekilde değiştirilir.

Dim source() As Long
ReDim source(0 To 2)

Dim fixed(0 To 2) As Long
Dim dynamic() As Long

fixed = source   ' does not compile
dynamic = source ' does compile

Dim dynamic2() As Long
ReDim dynamic2(0 to 6, 3 to 99) 

dynamic2 = source ' dynamic2 now has dimension (0 to 2)

Kopyalama yapıldıktan sonra iki dizi bellekte ayrıdır, yani iki değişken aynı temel verilere referans değildir, bu nedenle bir dizide yapılan değişiklikler diğerinde görünmez.

Dim source(0 To 2) As Long
Dim destination() As Long

source(0) = 3
source(1) = 1
source(2) = 4

destination = source
destination(0) = 2

Debug.Print source(0); source(1); source(2)                ' outputs: 3 1 4
Debug.Print destination(0); destination(1); destination(2) ' outputs: 2 1 4

Nesne Dizilerini Kopyalama #

Nesne dizileriyle, nesnelerin kendileri değil, bu nesnelere referanslar kopyalanır. Bir dizideki bir nesnede değişiklik yapılırsa, diğer dizide de değiştirilmiş gibi görünecektir - ikisi de aynı nesneye başvuruyor. Ancak, bir dizideki bir öğeyi farklı bir nesneye ayarlamak, onu diğer dizideki o nesneye ayarlamaz.

Dim source(0 To 2) As Range
Dim destination() As Range

Set source(0) = Range("A1"): source(0).Value = 3
Set source(1) = Range("A2"): source(1).Value = 1
Set source(2) = Range("A3"): source(2).Value = 4

destination = source

Set destination(0) = Range("A4")   'reference changed in destination but not source

destination(0).Value = 2           'affects an object only in destination
destination(1).Value = 5           'affects an object in both source and destination

Debug.Print source(0); source(1); source(2)                  ' outputs 3 5 4
Debug.Print destination(0); destination(1); destination(2)   ' outputs 2 5 4

Dizi İçeren Varyantlar #

Ayrıca bir değişken değişkenine ve değişkeninden bir dizi kopyalayabilirsiniz. Bir varyanttan kopyalarken, alıcı diziyle aynı türde bir dizi içermelidir, aksi takdirde “Tür uyuşmazlığı” çalışma zamanı hatası verir.

Dim var As Variant
Dim source(0 To 2) As Range
Dim destination() As Range

var = source
destination = var

var = 5
destination = var  ' throws runtime error

Fonksiyonlardan Dizi Döndürme #

Normal bir modüldeki bir işlev (ancak bir Sınıf modülü değil), veri türünden sonra () koyarak bir dizi döndürebilir.

Function arrayOfPiDigits() As Long()
    Dim outputArray(0 To 2) As Long
    
    outputArray(0) = 3
    outputArray(1) = 1
    outputArray(2) = 4

    arrayOfPiDigits = outputArray
End Function

Fonksiyonun sonucu daha sonra aynı tipte veya bir değişkende dinamik bir diziye yerleştirilebilir. Öğelere doğrudan ikinci bir parantez seti kullanılarak da erişilebilir, ancak bu her seferinde işlevi çağıracaktır, bu nedenle sonuçları bir kereden fazla kullanmayı planlıyorsanız sonuçları yeni bir dizide saklamak en iyisidir.

Sub arrayExample()

    Dim destination() As Long
    Dim var As Variant
    
    destination = arrayOfPiDigits()
    var = arrayOfPiDigits
    
    Debug.Print destination(0)          ' outputs 3
    Debug.Print var(1)                  ' outputs 1
    Debug.Print arrayOfPiDigits()(2)    ' outputs 4
    
End Sub

Döndürülen şeyin aslında bir referans değil, işlevin içindeki dizinin bir kopyası olduğunu unutmayın. Bu nedenle, işlev bir Statik dizinin içeriğini döndürürse, verileri çağırma prosedürüyle değiştirilemez.

Çıktı argümanı ile bir Dizi çıktısı almak #

Bir prosedürün argümanlarının girdi olması ve dönüş değeri aracılığıyla çıktılanması normalde iyi bir kodlama uygulamasıdır. Bununla birlikte, VBA’nın sınırlamaları bazen bir prosedürün bir “ByRef” argümanı aracılığıyla veri çıkışı yapmasını gerekli kılar.

Sabit bir diziye çıktı alma #

Sub threePiDigits(ByRef destination() As Long)
    destination(0) = 3
    destination(1) = 1
    destination(2) = 4
End Sub

Sub printPiDigits()
    Dim digits(0 To 2) As Long
    
    threePiDigits digits
    Debug.Print digits(0); digits(1); digits(2) ' outputs 3 1 4
End Sub

Sınıf yönteminden bir Dizi çıktısı almak #

Bir Class modülündeki bir yöntem/prosedürden bir dizi çıktısı almak için bir çıktı argümanı da kullanılabilir.

' Class Module 'MathConstants'
Sub threePiDigits(ByRef destination() As Long)
    ReDim destination(0 To 2)
    
    destination(0) = 3
    destination(1) = 1
    destination(2) = 4
End Sub

' Standard Code Module
Sub printPiDigits()
    Dim digits() As Long
    Dim mathConsts As New MathConstants
    
    mathConsts.threePiDigits digits
    Debug.Print digits(0); digits(1); digits(2) ' outputs 3 1 4
End Sub

Dizileri Prosedürlere Geçirme #

Dizi değişkeninin adından sonra () koyarak diziler işlemlere geçirilebilir.

Function countElements(ByRef arr() As Double) As Long
    countElements = UBound(arr) - LBound(arr) + 1
End Function

Diziler başvuru yoluyla iletilmelidir. Geçiş mekanizması belirtilmemişse, örn. myFunction(arr()), ardından VBA varsayılan olarak ByRef varsayacaktır, ancak bunu açık hale getirmek iyi bir kodlama uygulamasıdır. Bir diziyi değere göre geçirmeye çalışmak, ör. myFunction(ByVal arr()) bir “Dizi argümanı ByRef olmalıdır” derleme hatasıyla (veya VBE seçeneklerinde “Otomatik Sözdizimi Kontrolü” işaretlenmemişse “Sözdizimi hatası” derleme hatasıyla) sonuçlanır.

Referans yoluyla geçmek, dizide yapılan herhangi bir değişikliğin çağrı işleminde korunacağı anlamına gelir.

Sub testArrayPassing()
    Dim source(0 To 1) As Long
    source(0) = 3
    source(1) = 1
    
    Debug.Print doubleAndSum(source)  ' outputs 8
    Debug.Print source(0); source(1)  ' outputs 6 2
End Sub

Function doubleAndSum(ByRef arr() As Long)
    arr(0) = arr(0) * 2
    arr(1) = arr(1) * 2
    doubleAndSum = arr(0) + arr(1)
End Function

Orijinal diziyi değiştirmekten kaçınmak istiyorsanız, herhangi bir öğeyi değiştirmemesi için işlevi yazmaya dikkat edin.

Function doubleAndSum(ByRef arr() As Long)
    doubleAndSum = arr(0) * 2 + arr(1) * 2
End Function

Alternatif olarak dizinin çalışan bir kopyasını oluşturun ve kopyayla çalışın.

Function doubleAndSum(ByRef arr() As Long)
    Dim copyOfArr() As Long
    copyOfArr = arr
    
    copyOfArr(0) = copyOfArr(0) * 2
    copyOfArr(1) = copyOfArr(1) * 2
    
    doubleAndSum = copyOfArr(0) + copyOfArr(1)
End Function
