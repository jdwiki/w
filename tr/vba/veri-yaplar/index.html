Bağlantılı liste #

Bu bağlantılı liste örneği, Soyut veri türünü ayarla işlemlerini uygular.

SinglyLinkedNode sınıfı

Option Explicit

Private Value As Variant
Private NextNode As SinglyLinkedNode '"Next" is a keyword in VBA and therefore is not a valid variable name

LinkedList sınıfı

Option Explicit

Private head As SinglyLinkedNode

'Set type operations

Public Sub Add(value As Variant)
    Dim node As SinglyLinkedNode
        
    Set node = New SinglyLinkedNode
    node.value = value
    Set node.nextNode = head
    
    Set head = node
End Sub

Public Sub Remove(value As Variant)
    Dim node As SinglyLinkedNode
    Dim prev As SinglyLinkedNode
    
    Set node = head
    
    While Not node Is Nothing
        If node.value = value Then
            'remove node
            If node Is head Then
                Set head = node.nextNode
            Else
                Set prev.nextNode = node.nextNode
            End If
            Exit Sub
        End If
        Set prev = node
        Set node = node.nextNode
    Wend

End Sub

Public Function Exists(value As Variant) As Boolean
    Dim node As SinglyLinkedNode
    
    Set node = head
    While Not node Is Nothing
        If node.value = value Then
            Exists = True
            Exit Function
        End If
        Set node = node.nextNode
    Wend
End Function

Public Function Count() As Long
    Dim node As SinglyLinkedNode
    
    Set node = head
    
    While Not node Is Nothing
        Count = Count + 1
        Set node = node.nextNode
    Wend
    
End Function

İkili ağaç #

Bu, dengesiz bir ikili arama ağacına bir örnektir. İkili bir ağaç, kavramsal olarak, her düğümün iki çocuğu olan sol ve sağ olmak üzere ortak bir kökten aşağı doğru inen bir düğüm hiyerarşisi olarak yapılandırılmıştır. Örneğin, 7, 5, 9, 3, 11, 6, 12, 14 ve 15 sayılarının BinaryTree’ye eklendiğini varsayalım. Yapı aşağıdaki gibi olacaktır. Bu ikili ağacın dengeli olmadığını unutmayın; bu, arama performansını garanti etmek için istenen bir özellik olabilir - bkz. [AVL ağaçları](https:/ /en.wikipedia.org/wiki/AVL_tree) kendi kendini dengeleyen bir ikili arama ağacı örneği için.

             7
            / \
           5   9
          / \   \
         3   6   11
                   \ 
                    12
                      \
                       14
                        \
                         15
