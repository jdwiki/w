Scripting.FileSystemObject, bu konudaki eski yöntemlerden çok daha sağlamdır. Hemen her durumda tercih edilmelidir.

Klasörlerin ve Dosyaların Var Olup Olmadığını Belirleme #

Dosyalar:

Bir dosyanın var olup olmadığını belirlemek için, dosya adını Dir$ işlevine iletin ve bir sonuç döndürüp döndürmediğini test edin. “Dir$“ın joker karakterleri desteklediğine dikkat edin, bu nedenle belirli bir dosyayı test etmek için geçirilen “pathName"in, bunları içermediğinden emin olmak için test edilmesi gerekir. Aşağıdaki örnekte bir hata ortaya çıkıyor - bu istenen davranış değilse, işlev ‘Yanlış’ döndürecek şekilde değiştirilebilir.

Public Function FileExists(pathName As String) As Boolean
    If InStr(1, pathName, "*") Or InStr(1, pathName, "?") Then
        'Exit Function   'Return False on wild-cards.
        Err.Raise 52    'Raise error on wild-cards.
    End If
    FileExists = Dir$(pathName) <> vbNullString
End Function

Klasörler (Dir$ yöntemi):

‘Dir$()’ işlevi, isteğe bağlı ‘attributes’ parametresi için ‘vbDirectory’ iletilerek bir klasörün var olup olmadığını belirlemek için de kullanılabilir. Bu durumda, geçirilen pathName değeri bir yol ayırıcıyla (\) bitmelidir, çünkü dosya adları ile eşleşme yanlış pozitiflere neden olacaktır. Joker karakterlere yalnızca son yol ayırıcıdan sonra izin verildiğini unutmayın, bu nedenle aşağıdaki örnek işlev, giriş bir joker karakter içeriyorsa 52 - “Hatalı dosya adı veya numarası” çalışma zamanı hatası verecektir. Bu istenen davranış değilse, işlevin üst kısmındaki ‘On Error Resume Next’i kaldırın. Ayrıca, Dir$ öğesinin göreceli dosya yollarını desteklediğini (yani ..\Foo\Bar) da unutmayın, bu nedenle sonuçların yalnızca geçerli çalışma dizini değiştirilmediği sürece geçerli olması garanti edilir.

Public Function FolderExists(ByVal pathName As String) As Boolean
    'Uncomment the "On Error" line if paths with wild-cards should return False
    'instead of raising an error.
    'On Error Resume Next    
    If pathName = vbNullString Or Right$(pathName, 1) <> "\" Then
        Exit Function
    End If
    FolderExists = Dir$(pathName, vbDirectory) <> vbNullString
End Function

Klasörler (ChDir yöntemi):

ChDir ifadesi, bir klasörün var olup olmadığını test etmek için de kullanılabilir. Bu yöntemin, VBA’nın çalıştığı ortamı geçici olarak değiştireceğini unutmayın; bu nedenle, dikkate alınması gereken buysa, bunun yerine Dir$ yöntemi kullanılmalıdır. Parametresi ile çok daha az bağışlayıcı olma avantajına sahiptir. Bu yöntem aynı zamanda göreli dosya yollarını da destekler, dolayısıyla Dir$ yöntemiyle aynı uyarıya sahiptir.

Public Function FolderExists(ByVal pathName As String) As Boolean
    'Cache the current working directory
    Dim cached As String
    cached = CurDir$

    On Error Resume Next
    ChDir pathName
    FolderExists = Err.Number = 0
    On Error GoTo 0
    'Change back to the cached working directory.
    ChDir cached
End Function

Dosya Klasörleri Oluşturma ve Silme #

NOT: Aşağıdaki örneklerde, bu konudaki Klasörlerin ve Dosyaların Var Olup Olmadığını Belirleme örneğindeki FolderExists işlevi kullanılmaktadır.

MkDir ifadesi yeni bir klasör oluşturmak için kullanılabilir. Sürücü harflerini (C:\Foo), UNC adlarını (\\Server\Foo), göreli yolları (..\Foo) veya geçerli çalışma dizinini (Foo) içeren yolları kabul eder.

Sürücü veya UNC adı atlanırsa (yani \Foo), klasör geçerli sürücüde oluşturulur. Bu, geçerli çalışma dizini ile aynı sürücü olabilir veya olmayabilir.

Public Sub MakeNewDirectory(ByVal pathName As String)
    'MkDir will fail if the directory already exists.
    If FolderExists(pathName) Then Exit Sub
    'This may still fail due to permissions, etc.
    MkDir pathName
End Sub

RmDir ifadesi mevcut klasörleri silmek için kullanılabilir. MkDir ile aynı formlardaki yolları kabul eder ve mevcut çalışma dizini ve sürücü ile aynı ilişkiyi kullanır. İfadenin Windows rd kabuk komutuna benzer olduğunu unutmayın, bu nedenle hedef dizin boş değilse bir çalışma zamanı hatası 75: “Yol/Dosya erişim hatası” verecektir.

Public Sub DeleteDirectory(ByVal pathName As String)
    If Right$(pathName, 1) <> "\" Then
        pathName = pathName & "\"
    End If
    'Rmdir will fail if the directory doesn't exist.
    If Not FolderExists(pathName) Then Exit Sub
    'Rmdir will fail if the directory contains files.
    If Dir$(pathName & "*") <> vbNullString Then Exit Sub
    
    'Rmdir will fail if the directory contains directories.
    Dim subDir As String
    subDir = Dir$(pathName & "*", vbDirectory)
    Do
        If subDir <> "." And subDir <> ".." Then Exit Sub
        subDir = Dir$(, vbDirectory)
    Loop While subDir <> vbNullString
    
    'This may still fail due to permissions, etc.
    RmDir pathName
End Sub
