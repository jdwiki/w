Çok Boyutlu Diziler #

Çok Boyutlu Diziler #

Adından da anlaşılacağı gibi, çok boyutlu diziler, genellikle iki veya üç olmak üzere birden fazla boyut içeren ancak 32’ye kadar boyuta sahip olabilen dizilerdir.

Bir çoklu dizi, çeşitli düzeylerde bir matris gibi çalışır, örneğin bir, iki ve üç Boyut arasında bir karşılaştırma yapın.

One Dimension, tipik dizinizdir, bir öğe listesi gibi görünür.

Dim 1D(3) as Variant

*1D - Visually*
(0)
(1)
(2)

İki Boyut, bir Sudoku Izgarası veya bir Excel sayfası gibi görünür, diziyi başlatırken, dizinin kaç satır ve sütuna sahip olacağını tanımlarsınız.

Dim 2D(3,3) as Variant
'this would result in a 3x3 grid 

*2D - Visually*
(0,0) (0,1) (0,2)
(1,0) (1,1) (1,2)
(2,0) (2,1) (2,2)

Üç Boyut, Rubik Küpü gibi görünmeye başlar, diziyi başlatırken, dizinin sahip olacağı satırları, sütunları ve katmanları/derinlikleri tanımlarsınız.

Dim 3D(3,3,2) as Variant
'this would result in a 3x3x3 grid

*3D - Visually*
       1st layer                 2nd layer                  3rd layer
         front                     middle                     back
(0,0,0) (0,0,1) (0,0,2) ¦ (1,0,0) (1,0,1) (1,0,2) ¦ (2,0,0) (2,0,1) (2,0,2)
(0,1,0) (0,1,1) (0,1,2) ¦ (1,1,0) (1,1,1) (1,1,2) ¦ (2,1,0) (2,1,1) (2,1,2)
(0,2,0) (0,2,1) (0,2,2) ¦ (1,2,0) (1,2,1) (1,2,2) ¦ (2,2,0) (2,2,1) (2,2,2)

Diğer boyutlar 3B’nin çarpımı olarak düşünülebilir, dolayısıyla bir 4B(1,3,3,3) iki yan yana 3B dizi olacaktır.

İki Boyutlu Dizi #

Oluşturma

Aşağıdaki örnek, bir çalışan listesinin bir derlemesi olacaktır, her çalışanın listede bir takım bilgileri olacaktır (Ad, Soyad, Adres, E-posta, Telefon …), örnek esas olarak dizide saklanacaktır ( çalışan,bilgi) (0,0) olmak üzere ilk çalışanın adıdır.

Dim Bosses As Variant
'set bosses as Variant, so we can input any data type we want

Bosses = [{"Jonh","Snow","President";"Ygritte","Wild","Vice-President"}]
'initialise a 2D array directly by filling it with information, the redult wil be a array(1,2) size 2x3 = 6 elements

Dim Employees As Variant
'initialize your Employees array as variant
'initialize and ReDim the Employee array so it is a dynamic array instead of a static one, hence treated differently by the VBA Compiler
ReDim Employees(100, 5)
'declaring an 2D array that can store 100 employees with 6 elements of information each, but starts empty
'the array size is 101 x 6 and contains 606 elements

For employee = 0 To UBound(Employees, 1)
'for each employee/row in the array, UBound for 2D arrays, which will get the last element on the array
'needs two parameters 1st the array you which to check and 2nd the dimension, in this case 1 = employee and 2 = information
    For information_e = 0 To UBound(Employees, 2)
    'for each information element/column in the array
        
        Employees(employee, information_e) = InformationNeeded ' InformationNeeded would be the data to fill the array
        'iterating the full array will allow for direct attribution of information into the element coordinates
    Next
Next

Yeniden boyutlandırma

Tek Boyutlu bir dizinin normu gibi bir Çoklu Diziyi yeniden boyutlandırma veya ‘ReDim Koru’ hata alır, bunun yerine bilgilerin orijinalle aynı boyutta ve eklenecek satır/sütun sayısıyla aynı olan bir Geçici diziye aktarılması gerekir. Ekle. Aşağıdaki örnekte, bir Temp Dizisini nasıl başlatacağımızı, bilgileri orijinal diziden nasıl aktaracağımızı, kalan boş öğeleri nasıl dolduracağımızı ve geçici diziyi orijinal diziyle nasıl değiştireceğimizi göreceğiz.

Dim TempEmp As Variant
'initialise your temp array as variant
ReDim TempEmp(UBound(Employees, 1) + 1, UBound(Employees, 2))
'ReDim/Resize Temp array as a 2D array with size UBound(Employees)+1 = (last element in Employees 1st dimension) + 1,
'the 2nd dimension remains the same as the original array. we effectively add 1 row in the Employee array

'transfer
For emp = LBound(Employees, 1) To UBound(Employees, 1)
    For info = LBound(Employees, 2) To UBound(Employees, 2)
        'to transfer Employees into TempEmp we iterate both arrays and fill TempEmp with the corresponding element value in Employees
        TempEmp(emp, info) = Employees(emp, info)
    
    Next
Next

'fill remaining
'after the transfers the Temp array still has unused elements at the end, being that it was increased
'to fill the remaining elements iterate from the last "row" with values to the last row in the array
'in this case the last row in Temp will be the size of the Employees array rows + 1, as the last row of Employees array is already filled in the TempArray

For emp = UBound(Employees, 1) + 1 To UBound(TempEmp, 1)
    For info = LBound(TempEmp, 2) To UBound(TempEmp, 2)
        
        TempEmp(emp, info) = InformationNeeded & "NewRow"
    
    Next
Next

'erase Employees, attribute Temp array to Employees and erase Temp array
Erase Employees
Employees = TempEmp
Erase TempEmp

Eleman Değerlerini Değiştirme

Belirli bir öğedeki değerleri değiştirmek/değiştirmek, değiştirilecek koordinatı çağırarak ve ona yeni bir değer vererek yapılabilir: Employees(0, 0) = "NewValue"

Alternatif olarak, gerekli parametrelere karşılık gelen değerleri eşleştirmek için koordinatları kullanın:

For emp = 0 To UBound(Employees)
    If Employees(emp, 0) = "Gloria" And Employees(emp, 1) = "Stephan" Then
    'if value found
        Employees(emp, 1) = "Married, Last Name Change"
        Exit For
        'don't iterate through a full array unless necessary
    End If
Next

Okuma

Dizideki öğelere erişim, Yuvalanmış Döngü (her öğeyi yineleyen), Döngü ve Koordinat (Satırları yineleyin ve sütunlara doğrudan erişerek) veya her iki koordinatla doğrudan erişim ile yapılabilir.

'nested loop, will iterate through all elements
For emp = LBound(Employees, 1) To UBound(Employees, 1)
    For info = LBound(Employees, 2) To UBound(Employees, 2)
        Debug.Print Employees(emp, info)
    Next
Next

'loop and coordinate, iteration through all rows and in each row accessing all columns directly
For emp = LBound(Employees, 1) To UBound(Employees, 1)
    Debug.Print Employees(emp, 0)
    Debug.Print Employees(emp, 1)
    Debug.Print Employees(emp, 2)
    Debug.Print Employees(emp, 3)
    Debug.Print Employees(emp, 4)
    Debug.Print Employees(emp, 5)
Next

'directly accessing element with coordinates
Debug.Print Employees(5, 5)

Unutmayın, Çok Boyutlu dizileri kullanırken bir dizi haritası tutmak her zaman kullanışlıdır, bunlar kolayca kafa karıştırabilir.

Üç Boyutlu Dizi #

3B dizi için, yalnızca Çalışan ve Bilgileri depolamakla kalmayıp aynı zamanda içinde çalıştıkları Binayı da ekleyerek 2B diziyle aynı önermeyi kullanacağız.

3B dizi, bir excel belgesinde farklı sayfalar olarak düşünülebilecek Çalışanlar (Satırlar olarak düşünülebilir), Bilgi (Sütunlar) ve Binaya sahip olacaktır, aralarında aynı boyuta sahiptirler, ancak her sayfanın bir hücrelerinde/öğelerinde farklı bilgi kümesi. 3B dizi, n adet 2B dizi içerecektir.

Oluşturma

Bir 3B dizinin başlatılması için 3 koordinat gerekir Dim 3Darray(2,5,5) As Variant Dizideki ilk koordinat Bina/Yaprak sayısı (farklı satır ve sütun kümeleri) olacaktır, ikinci koordinat Satırları tanımlayacaktır ve üçüncü Sütunlar. Yukarıdaki “Dim”, 108 elemanlı (‘366’) bir 3B diziyle sonuçlanacak ve 3 farklı 2B dizi setine sahip olacaktır.

Dim ThreeDArray As Variant
'initialise your ThreeDArray array as variant
ReDim ThreeDArray(1, 50, 5)
'declaring an 3D array that can store two sets of 51 employees with 6 elements of information each, but starts empty
'the array size is 2 x 51 x 6 and contains 612 elements

For building = 0 To UBound(ThreeDArray, 1)
    'for each building/set in the array
    For employee = 0 To UBound(ThreeDArray, 2)
    'for each employee/row in the array
        For information_e = 0 To UBound(ThreeDArray, 3)
        'for each information element/column in the array
            
            ThreeDArray(building, employee, information_e) = InformationNeeded ' InformationNeeded would be the data to fill the array
        'iterating the full array will allow for direct attribution of information into the element coordinates
        Next
    Next
Next

Yeniden boyutlandırma

Bir 3B diziyi yeniden boyutlandırmak, 2B’yi yeniden boyutlandırmaya benzer, önce artırılacak parametrenin koordinatına bir tane ekleyerek orijinal ile aynı boyutta bir Geçici dizi oluşturun, ilk koordinat dizideki küme sayısını artıracak, ikincisi ve üçüncü koordinatlar, her kümedeki Satır veya Sütun sayısını artıracaktır.

Aşağıdaki örnek, her kümedeki Satır sayısını birer birer artırır ve son eklenen öğeleri yeni bilgilerle doldurur.

Dim TempEmp As Variant
'initialise your temp array as variant
ReDim TempEmp(UBound(ThreeDArray, 1), UBound(ThreeDArray, 2) + 1, UBound(ThreeDArray, 3))
'ReDim/Resize Temp array as a 3D array with size UBound(ThreeDArray)+1 = (last element in Employees 2nd dimension) + 1,
'the other dimension remains the same as the original array. we effectively add 1 row in the for each set of the 3D array

'transfer
For building = LBound(ThreeDArray, 1) To UBound(ThreeDArray, 1)
    For emp = LBound(ThreeDArray, 2) To UBound(ThreeDArray, 2)
        For info = LBound(ThreeDArray, 3) To UBound(ThreeDArray, 3)
            'to transfer ThreeDArray into TempEmp by iterating all sets in the 3D array and fill TempEmp with the corresponding element value in each set of each row
            TempEmp(building, emp, info) = ThreeDArray(building, emp, info)
        
        Next
    Next
Next

'fill remaining
'to fill the remaining elements we need to iterate from the last "row" with values to the last row in the array in each set, remember that the first empty element is the original array Ubound() plus 1
For building = LBound(TempEmp, 1) To UBound(TempEmp, 1)
    For emp = UBound(ThreeDArray, 2) + 1 To UBound(TempEmp, 2)
        For info = LBound(TempEmp, 3) To UBound(TempEmp, 3)
            
            TempEmp(building, emp, info) = InformationNeeded & "NewRow"
        
        Next
    Next
Next

'erase Employees, attribute Temp array to Employees and erase Temp array
Erase ThreeDArray
ThreeDArray = TempEmp
Erase TempEmp

Eleman Değerlerini Değiştirme ve Okuma

3B dizideki öğeleri okumak ve değiştirmek, 2B diziyi yaptığımıza benzer şekilde yapılabilir, sadece döngülerdeki ve koordinatlardaki ekstra seviye için ayarlayın.

Do
' using Do ... While for early exit
    For building = 0 To UBound(ThreeDArray, 1)
        For emp = 0 To UBound(ThreeDArray, 2)
            If ThreeDArray(building, emp, 0) = "Gloria" And ThreeDArray(building, emp, 1) = "Stephan" Then
            'if value found
                ThreeDArray(building, emp, 1) = "Married, Last Name Change"
                Exit Do
                'don't iterate through all the array unless necessary
            End If
        Next
    Next
Loop While False

'nested loop, will iterate through all elements
For building = LBound(ThreeDArray, 1) To UBound(ThreeDArray, 1)
    For emp = LBound(ThreeDArray, 2) To UBound(ThreeDArray, 2)
        For info = LBound(ThreeDArray, 3) To UBound(ThreeDArray, 3)
            Debug.Print ThreeDArray(building, emp, info)
        Next
    Next
Next

'loop and coordinate, will iterate through all set of rows and ask for the row plus the value we choose for the columns
For building = LBound(ThreeDArray, 1) To UBound(ThreeDArray, 1)
    For emp = LBound(ThreeDArray, 2) To UBound(ThreeDArray, 2)
        Debug.Print ThreeDArray(building, emp, 0)
        Debug.Print ThreeDArray(building, emp, 1)
        Debug.Print ThreeDArray(building, emp, 2)
        Debug.Print ThreeDArray(building, emp, 3)
        Debug.Print ThreeDArray(building, emp, 4)
        Debug.Print ThreeDArray(building, emp, 5)
    Next
Next

'directly accessing element with coordinates
Debug.Print Employees(0, 5, 5)

Dinamik Diziler (Dizi Yeniden Boyutlandırma ve Dinamik İşleme) #

Dinamik Diziler #

Bir diziye dinamik olarak değişkenler eklemek ve azaltmak, işlediğiniz bilgilerin belirli sayıda değişkene sahip olmadığı durumlarda büyük bir avantajdır.

Dinamik Olarak Değerler Ekleme #

Diziyi ‘ReDim’ İfadesi ile basitçe yeniden boyutlandırabilirsiniz, bu diziyi yeniden boyutlandırır, ancak dizide zaten depolanmış bilgileri saklamak istiyorsanız, ‘Koru’ bölümüne ihtiyacınız olacaktır.

Aşağıdaki örnekte bir dizi oluşturuyoruz ve dizideki değerleri koruyarak her yinelemede bir değişken daha artırıyoruz.

Dim Dynamic_array As Variant
' first we set Dynamic_array as variant

For n = 1 To 100

    If IsEmpty(Dynamic_array) Then
        'isempty() will check if we need to add the first value to the array or subsequent ones
    
        ReDim Dynamic_array(0)
        'ReDim Dynamic_array(0) will resize the array to one variable only
        Dynamic_array(0) = n

    Else
        ReDim Preserve Dynamic_array(0 To UBound(Dynamic_array) + 1)
        'in the line above we resize the array from variable 0 to the UBound() = last variable, plus one effectivelly increeasing the size of the array by one
        Dynamic_array(UBound(Dynamic_array)) = n
        'attribute a value to the last variable of Dynamic_array
    End If

Next

Dinamik Olarak Değerleri Kaldırma #

Diziyi azaltmak için aynı mantığı kullanabiliriz. Örnekte “last” değeri diziden kaldırılacaktır.

Dim Dynamic_array As Variant
Dynamic_array = Array("first", "middle", "last")
    
ReDim Preserve Dynamic_array(0 To UBound(Dynamic_array) - 1)
' Resize Preserve while dropping the last value

Diziyi Sıfırlama ve Dinamik Olarak Yeniden Kullanma #

Oluşturduğumuz dizileri, bellekte çok fazla yer kalmayacak şekilde yeniden kullanabiliriz, bu da çalışma süresini yavaşlatır. Bu, çeşitli boyutlardaki diziler için kullanışlıdır. Diziyi yeniden kullanmak için kullanabileceğiniz bir pasaj, diziyi yeniden “(0)” konumuna “ReDim” yapmak, diziye bir değişken atamak ve diziyi serbestçe yeniden artırmaktır.

Aşağıdaki snippet’te 1’den 40’a kadar değerlere sahip bir dizi oluşturuyorum, diziyi boşaltıyorum ve diziyi 40’tan 100’e kadar değerlerle yeniden dolduruyorum, tüm bunlar dinamik olarak yapılıyor.

Dim Dynamic_array As Variant

For n = 1 To 100

    If IsEmpty(Dynamic_array) Then
        ReDim Dynamic_array(0)
        Dynamic_array(0) = n
    
    ElseIf Dynamic_array(0) = "" Then
        'if first variant is empty ( = "") then give it the value of n
        Dynamic_array(0) = n
    Else
        ReDim Preserve Dynamic_array(0 To UBound(Dynamic_array) + 1)
        Dynamic_array(UBound(Dynamic_array)) = n
    End If
    If n = 40 Then
        ReDim Dynamic_array(0)
        'Resizing the array back to one variable without Preserving,
        'leaving the first value of the array empty
    End If

Next

Pürüzlü Diziler (Dizi Dizileri) #

Pürüzlü Diziler Çok Boyutlu Diziler DEĞİL #

Diziler (Keskin Diziler), görsel olarak düşünürseniz Çok Boyutlu Diziler ile aynı değildir. farklı aylardaki günler gibi farklı sayıda öğeye sahip iç dizilere sahip takvim.

Çentikli Diziler, iç içe seviyeleri nedeniyle oldukça dağınık ve kullanımı zor olsalar ve fazla tür güvenliğine sahip olmasalar da, çok esnektirler, farklı veri türlerini oldukça kolay bir şekilde manipüle etmenize izin verirler ve kullanılmayan veya boş elemanlar.

Pürüzlü Dizi Oluşturma #

Aşağıdaki örnekte, biri Names, diğeri Numbers için iki dizi içeren ve ardından her birinin bir öğesine erişen pürüzlü bir dizi başlatacağız.

Dim OuterArray() As Variant
Dim Names() As Variant
Dim Numbers() As Variant
'arrays are declared variant so we can access attribute any data type to its elements

Names = Array("Person1", "Person2", "Person3")
Numbers = Array("001", "002", "003")

OuterArray = Array(Names, Numbers)
'Directly giving OuterArray an array containing both Names and Numbers arrays inside

Debug.Print OuterArray(0)(1)
Debug.Print OuterArray(1)(1)
'accessing elements inside the jagged by giving the coordenades of the element

Dinamik Olarak Jagged Diziler Oluşturma ve Okuma #

Dizileri oluşturmak için yaklaşık olarak daha dinamik olabiliriz, excel’de bir müşteri veri sayfamız olduğunu ve müşteri ayrıntılarını çıkarmak için bir dizi oluşturmak istediğimizi hayal edin.

   Name -   Phone   -  Email  - Customer Number 
Person1 - 153486231 - 1@STACK - 001
Person2 - 153486242 - 2@STACK - 002
Person3 - 153486253 - 3@STACK - 003
Person4 - 153486264 - 4@STACK - 004
Person5 - 153486275 - 5@STACK - 005

Dinamik olarak bir Header dizisi ve bir Customers dizisi oluşturacağız, Header sütun başlıklarını içerecek ve Customers dizisi her müşterinin/satırın bilgilerini diziler olarak içerecek.

Dim Headers As Variant
' headers array with the top section of the customer data sheet
    For c = 1 To 4
        If IsEmpty(Headers) Then
            ReDim Headers(0)
            Headers(0) = Cells(1, c).Value
        Else
            ReDim Preserve Headers(0 To UBound(Headers) + 1)
            Headers(UBound(Headers)) = Cells(1, c).Value
        End If
    Next
    
Dim Customers As Variant
'Customers array will contain arrays of customer values
Dim Customer_Values As Variant
'Customer_Values will be an array of the customer in its elements (Name-Phone-Email-CustNum)
    
    For r = 2 To 6
    'iterate through the customers/rows
        For c = 1 To 4
        'iterate through the values/columns
            
            'build array containing customer values
            If IsEmpty(Customer_Values) Then
                ReDim Customer_Values(0)
                Customer_Values(0) = Cells(r, c).Value
            ElseIf Customer_Values(0) = "" Then
                Customer_Values(0) = Cells(r, c).Value
            Else
                ReDim Preserve Customer_Values(0 To UBound(Customer_Values) + 1)
                Customer_Values(UBound(Customer_Values)) = Cells(r, c).Value
            End If
        Next
        
        'add customer_values array to Customers Array
        If IsEmpty(Customers) Then
            ReDim Customers(0)
            Customers(0) = Customer_Values
        Else
            ReDim Preserve Customers(0 To UBound(Customers) + 1)
            Customers(UBound(Customers)) = Customer_Values
        End If
        
        'reset Custumer_Values to rebuild a new array if needed
        ReDim Customer_Values(0)
    Next

    Dim Main_Array(0 To 1) As Variant
    'main array will contain both the Headers and Customers
    
    Main_Array(0) = Headers
    Main_Array(1) = Customers

Dinamik olarak tek boyutlu bir dizi oluşturmanın yolunu daha iyi anlamak için lütfen Diziler belgelerinde Dinamik Dizileri (Dizi Yeniden Boyutlandırma ve Dinamik İşleme) kontrol edin.

Yukarıdaki snippet’in Sonucu, biri 4 elemanlı, 2 girinti düzeyine sahip dizilerden biri ve diğerinin kendisi, her biri 4 elemanlı 5 dizi ve 3 girinti seviyesi içeren başka bir Pürüzlü Dizi olan iki dizili bir Jagged Dizidir, yapının altına bakın:

Main_Array(0) - Headers - Array("Name","Phone","Email","Customer Number")
          (1) - Customers(0) - Array("Person1",153486231,"1@STACK",001)
                Customers(1) - Array("Person2",153486242,"2@STACK",002)
                ...
                Customers(4) - Array("Person5",153486275,"5@STACK",005)

Bilgiye erişmek için oluşturduğunuz Pürüzlü Dizinin yapısını aklınızda bulundurmanız gerekir, yukarıdaki örnekte “Ana Dizi"nin bir “Başlıklar” Dizisi ve bir Dizi Dizisi (“Müşteriler”) içerdiğini görebilirsiniz. ) bu nedenle öğelere erişmenin farklı yolları ile.

Şimdi ‘Ana Dizi’nin bilgilerini okuyacağız ve Müşterilerin her bir bilgisini ‘Bilgi Türü: Bilgi’ olarak yazdıracağız.

For n = 0 To UBound(Main_Array(1))
    'n to iterate from fisrt to last array in Main_Array(1)
    
    For j = 0 To UBound(Main_Array(1)(n))
        'j will iterate from first to last element in each array of Main_Array(1)
        
        Debug.Print Main_Array(0)(j) & ": " & Main_Array(1)(n)(j)
        'print Main_Array(0)(j) which is the header and Main_Array(0)(n)(j) which is the element in the customer array
        'we can call the header with j as the header array has the same structure as the customer array
    Next
Next

Jagged Array’inizin yapısını takip etmeyi UNUTMAYIN, yukarıdaki örnekte bir müşterinin Adına erişmek için üç seviye olan Main_Array -> Customers -> CustomerNumber -> Name seçeneğine erişerek "Person4" döndürün. Main_Array’de Customers’ın konumuna, ardından Customers Jagged dizisindeki dördüncü müşterinin Konumuna ve son olarak ihtiyacınız olan öğenin konumuna, bu durumda ‘Main_Array(1)(3)(0)‘a ihtiyacınız olacak. Main_Array(Müşteriler)(MüşteriNumarası)(Ad).

VBA’da Dizi Bildirme #

Bir dizi bildirmek, bir değişken bildirmekle çok benzer, ancak Dizinin adının hemen ardından boyutunu bildirmeniz gerekir:

Dim myArray(9) As String 'Declaring an array that will contain up to 10 strings

Varsayılan olarak, VBA’daki Diziler SIFIR’dan dizine eklenir, bu nedenle parantez içindeki sayı dizinin boyutuna değil son öğenin dizinine atıfta bulunur

Öğelere Erişim #

Dizinin bir öğesine erişim, Dizi adının ardından parantez içinde öğenin dizini kullanılarak yapılır:

myArray(0) = "first element"
myArray(5) = "sixth element"
myArray(9) = "last element"

Dizi İndeksleme #

Bu satırı bir modülün en üstüne yerleştirerek Dizi indekslemesini değiştirebilirsiniz:

Option Base 1

Bu satırla, modülde bildirilen tüm Diziler ONE’dan dizine alınacaktır.

Belirli Dizin #

Ayrıca, To anahtar sözcüğünü ve alt ve üst sınırını (= dizin) kullanarak her Diziyi kendi dizini ile bildirebilirsiniz:

Dim mySecondArray(1 To 12) As String 'Array of 12 strings indexed from 1 to 12
Dim myThirdArray(13 To 24) As String 'Array of 12 strings indexed from 13 to 24

Dinamik Bildirim #

Dizinizin bildiriminden önce boyutunu bilmiyorsanız, dinamik bildirimi ve ReDim anahtar sözcüğünü kullanabilirsiniz:

Dim myDynamicArray() As Strings 'Creates an Array of an unknown number of strings
ReDim myDynamicArray(5) 'This resets the array to 6 elements

ReDim anahtar sözcüğünü kullanmanın, Dizinizin önceki tüm içeriğini sileceğini unutmayın. Bunu önlemek için “ReDim"den sonra “Preserve” anahtar sözcüğünü kullanabilirsiniz:

Dim myDynamicArray(5) As String
myDynamicArray(0) = "Something I want to keep"

ReDim Preserve myDynamicArray(8) 'Expand the size to up to 9 strings
Debug.Print myDynamicArray(0) ' still prints the element

Bir dizeden dizi oluşturmak için Böl’ü kullanma #

Bölünmüş İşlev

belirtilen sayıda alt dize içeren sıfır tabanlı, tek boyutlu bir dizi döndürür.

Sözdizimi

Split(ifade [, sınırlayıcı [, sınır [, karşılaştır]]])

Bölüm Açıklama
ifade Gerekli. Alt dizeler ve sınırlayıcılar içeren dize ifadesi. ifade sıfır uzunluklu bir dize (”” veya vbNullString) ise, Split hiçbir öğe ve veri içermeyen boş bir dizi döndürür. Bu durumda, döndürülen dizinin LBound değeri 0 ve UBound değeri -1 olacaktır.
sınırlayıcı İsteğe bağlı. Alt dize sınırlarını belirlemek için kullanılan dize karakteri. Atlanırsa, boşluk karakterinin (" “) sınırlayıcı olduğu varsayılır. sınırlayıcı sıfır uzunluklu bir dize ise, tüm ifade dizesini içeren tek elemanlı bir dizi döndürülür.
sınır İsteğe bağlı. Döndürülecek alt dizi sayısı; -1, tüm alt dizelerin döndürüldüğünü gösterir.
karşılaştır İsteğe bağlı. Alt dizeleri değerlendirirken kullanılacak karşılaştırma türünü gösteren sayısal değer. Değerler için Ayarlar bölümüne bakın.

Ayarlar

karşılaştır bağımsız değişkeni aşağıdaki değerlere sahip olabilir:

Sabit Değer Açıklama
Açıklama -1 Option Compare ifadesinin ayarını kullanarak bir karşılaştırma yapar.
vbBinaryCompare 0 İkili karşılaştırma yapar.
vbTextCompare 1 Metin karşılaştırması yapar.
vbDatabaseCompare 2 Yalnızca Microsoft Access. Veritabanınızdaki bilgilere dayalı olarak bir karşılaştırma yapar.

Örnek

Bu örnekte, birkaç stil göstererek Split’in nasıl çalıştığı gösterilmiştir. Yorumlar, gerçekleştirilen farklı Bölme seçeneklerinin her biri için sonuç kümesini gösterecektir. Son olarak, döndürülen dize dizisi üzerinde nasıl döngü yapılacağı gösterilmiştir.

Sub Test
    
    Dim textArray() as String

    textArray = Split("Tech on the Net")
    'Result: {"Tech", "on", "the", "Net"}

    textArray = Split("172.23.56.4", ".")
    'Result: {"172", "23", "56", "4"}

    textArray = Split("A;B;C;D", ";")
    'Result: {"A", "B", "C", "D"}

    textArray = Split("A;B;C;D", ";", 1)
    'Result: {"A;B;C;D"}

    textArray = Split("A;B;C;D", ";", 2)
    'Result: {"A", "B;C;D"}

    textArray = Split("A;B;C;D", ";", 3)
    'Result: {"A", "B", "C;D"}

    textArray = Split("A;B;C;D", ";", 4)
    'Result: {"A", "B", "C", "D"}

    'You can iterate over the created array
    Dim counter As Long

    For counter = LBound(textArray) To UBound(textArray)
        Debug.Print textArray(counter)
    Next
 End Sub

Bir dizinin öğelerini yineleme #

İçin…Sonraki #

Dizin öğelerini yinelemenin en hızlı yolu dizin numarası olarak yineleyici değişkeni kullanmaktır:

Dim items As Variant
items = Array(0, 1, 2, 3)

Dim index As Integer
For index = LBound(items) To UBound(items)
    'assumes value can be implicitly converted to a String:
    Debug.Print items(index) 
Next

Çok boyutlu dizileri yinelemek için iç içe döngüler kullanılabilir:

Dim items(0 To 1, 0 To 1) As Integer
items(0, 0) = 0
items(0, 1) = 1
items(1, 0) = 2
items(1, 1) = 3

Dim outer As Integer
Dim inner As Integer
For outer = LBound(items, 1) To UBound(items, 1)
    For inner = LBound(items, 2) To UBound(items, 2)
        'assumes value can be implicitly converted to a String:
        Debug.Print items(outer, inner)
    Next
Next

Her Biri İçin…Sonraki #

Performans önemli değilse, dizileri yinelemek için “For Each…Next” döngüsü de kullanılabilir:

Dim items As Variant
items = Array(0, 1, 2, 3)

Dim item As Variant 'must be variant
For Each item In items
    'assumes value can be implicitly converted to a String:
    Debug.Print item
Next

Bir “ForEvery” döngüsü, tüm boyutları dıştan içe doğru yineleyecektir (öğelerin bellekte düzenlendiği sıra ile aynı sırayla), bu nedenle iç içe döngülere gerek yoktur:

Dim items(0 To 1, 0 To 1) As Integer
items(0, 0) = 0
items(1, 0) = 1
items(0, 1) = 2
items(1, 1) = 3

Dim item As Variant 'must be Variant
For Each item In items
    'assumes value can be implicitly converted to a String:
    Debug.Print item
Next

Performans önemliyse, “Her Bir” döngülerinin en iyi “Koleksiyon” nesnelerini yinelemek için kullanıldığını unutmayın.
