Zend Expressive'a Giriş

  • Basit bir Merhaba Dünya

Bu sayfada

  • Basit bir Merhaba Dünya

Basit bir Merhaba Dünya #

Besteci kullanarak, uygulamanın kurulacağı dizinde şu komutu yürütün: besteci oluştur-proje zendframework/zend-expressive-skeleton expressive-skeleton.

Yükleme işlemi sırasında çeşitli kararlar vermeniz istenecektir.

  1. Varsayılan kurulum sorusu için hayır (`n’) deyin;
  2. Yönlendirici için Aura Router’ı (#1) kullanalım;
  3. Konteyner için Zend ServiceManager’ı (#3) kullanalım;
  4. Şablon için Zend View (#3) kullanalım;
  5. Son olarak, hata işleyici için Whoops (#1) kullanalım.

Kurulduktan sonra, kendinizi ’expressive-skeleton’ kök dizinine alın, yerleşik PHP CLI sunucusunu başlatın: ‘php -S 0.0.0.0:8080 -t public public/index.php’. Tarayıcınızla http://localhost:8080/ adresine gidin, uygulamanız şimdi çalışır durumda olmalıdır.

Yeni bir ara katman yazılımı için yeni bir yol yapılandıralım. İlk olarak, config/autoload/routes.global.php içinde router yapılandırma dosyasını açın ve aşağıdaki satırları ekleyin:

<?php

return [
    'dependencies' => [
        ...
    ],

    'routes' => [
        [
            'dependencies' => [
                'invokables' => [
                    ...
                ],
                'factories' => [
                    ...
                    // Add the following line
                    App\Action\HelloWorldAction::class => App\Action\HelloWorldFactory::class,
                ],
            ],
        ],
        // Following lines should be added
        [
            'name' => 'hello-world',
            'path' => '/hello-world',
            'middleware' => App\Action\HelloWorldAction::class,
            'allowed_methods' => ['GET'],
        ],
    ],
];

Aşağıdaki içeriği src/App/Action/HelloWorldFactory.php içine yerleştirin:

<?php

namespace App\Action;

use Interop\Container\ContainerInterface;
use Zend\Expressive\Template\TemplateRendererInterface;

class HelloWorldFactory
{
    public function __invoke(ContainerInterface $container)
    {
        $template = ($container->has(TemplateRendererInterface::class))
            ? $container->get(TemplateRendererInterface::class)
            : null;

        return new HelloWorldAction($template);
    }
}

Ardından, src/App/Action/HelloWorldAction.php içindeki bu içerik:

<?php

namespace App\Action;

use Psr\Http\Message\ResponseInterface;
use Psr\Http\Message\ServerRequestInterface;
use Zend\Diactoros\Response\HtmlResponse;
use Zend\Diactoros\Response\JsonResponse;
use Zend\Expressive\Template;
use Zend\Expressive\Plates\PlatesRenderer;
use Zend\Expressive\Twig\TwigRenderer;
use Zend\Expressive\ZendView\ZendViewRenderer;

class HelloWorldAction
{
    private $template;

    public function __construct(Template\TemplateRendererInterface $template = null)
    {
        $this->template = $template;
    }

    public function __invoke(ServerRequestInterface $request, ResponseInterface $response, callable $next = null)
    {
        $data = [];

        return new HtmlResponse($this->template->render('app::hello-world'));
    }
}

Ardından, son olarak, aşağıdakileri “templates/app/hello-world.phtml” içine koymanız yeterlidir:

<?php echo 'Hello World'; ?>

İşimiz bitti ! http://localhost:8080/hello-world’e gidin ve Zend Expressive’a “merhaba” deyin!
