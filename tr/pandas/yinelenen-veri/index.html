Çoğaltmayı bırak #

drop_duplicates kullanın:

In [216]: df = pd.DataFrame({'A':[1,2,3,3,2],
     ...:                    'B':[1,7,3,0,8]})

In [217]: df
Out[217]: 
   A  B
0  1  1
1  2  7
2  3  3
3  3  0
4  2  8

# keep only the last value
In [218]: df.drop_duplicates(subset=['A'], keep='last')
Out[218]: 
   A  B
0  1  1
3  3  0
4  2  8

# keep only the first value, default value
In [219]: df.drop_duplicates(subset=['A'], keep='first')
Out[219]: 
   A  B
0  1  1
1  2  7
2  3  3

# drop all duplicated values
In [220]: df.drop_duplicates(subset=['A'], keep=False)
Out[220]: 
   A  B
0  1  1

Bir veri çerçevesinin bir kopyasını almak değil de mevcut olanı değiştirmek istediğinizde:

In [221]: df = pd.DataFrame({'A':[1,2,3,3,2],
     ...:                    'B':[1,7,3,0,8]})

In [222]: df.drop_duplicates(subset=['A'], inplace=True)

In [223]: df
Out[223]: 
   A  B
0  1  1
1  2  7
2  3  3

Çoğaltmayı seç #

“0” değerini “B” sütununa ayarlamanız gerekiyorsa, “A” sütununda çoğaltılan veriler ilk önce Series.duplicated ile maske oluşturun oluşturulan/pandas.Series.duplicated.html) ve ardından DataFrame.ix veya Series.mask:

In [224]: df = pd.DataFrame({'A':[1,2,3,3,2],
     ...:                    'B':[1,7,3,0,8]})

In [225]: mask = df.A.duplicated(keep=False)

In [226]: mask
Out[226]: 
0    False
1     True
2     True
3     True
4     True
Name: A, dtype: bool

In [227]: df.ix[mask, 'B'] = 0

In [228]: df['C'] = df.A.mask(mask, 0)

In [229]: df
Out[229]: 
   A  B  C
0  1  1  1
1  2  0  0
2  3  0  0
3  3  0  0
4  2  0  0

Maskeyi ters çevirmeye ihtiyacınız varsa ~ kullanın:

In [230]: df['C'] = df.A.mask(~mask, 0)

In [231]: df
Out[231]: 
   A  B  C
0  1  1  0
1  2  0  2
2  3  0  3
3  3  0  3
4  2  0  2

Benzersiz öğeleri sayma ve alma #

Bir dizideki benzersiz öğelerin sayısı:

In [1]: id_numbers = pd.Series([111, 112, 112, 114, 115, 118, 114, 118, 112])
In [2]: id_numbers.nunique()
Out[2]: 5

Bir dizide benzersiz öğeler edinin:

In [3]: id_numbers.unique()
Out[3]: array([111, 112, 114, 115, 118], dtype=int64)

In [4]: df = pd.DataFrame({'Group': list('ABAABABAAB'), 
                           'ID': [1, 1, 2, 3, 3, 2, 1, 2, 1, 3]})

In [5]: df
Out[5]: 
  Group  ID
0     A   1
1     B   1
2     A   2
3     A   3
4     B   3
5     A   2
6     B   1
7     A   2
8     A   1
9     B   3

Her gruptaki benzersiz öğelerin sayısı:

In [6]: df.groupby('Group')['ID'].nunique()
Out[6]: 
Group
A    3
B    2
Name: ID, dtype: int64

Her grupta benzersiz öğelerden yararlanın:

In [7]: df.groupby('Group')['ID'].unique()
Out[7]: 
Group
A    [1, 2, 3]
B       [1, 3]
Name: ID, dtype: object

Bir sütundan benzersiz değerler alın. #

In [15]: df = pd.DataFrame({"A":[1,1,2,3,1,1],"B":[5,4,3,4,6,7]})

In [21]: df
Out[21]: 
   A  B
0  1  5
1  1  4
2  2  3
3  3  4
4  1  6
5  1  7

A ve B sütunlarında benzersiz değerler elde etmek için.

In [22]: df["A"].unique()
Out[22]: array([1, 2, 3])

In [23]: df["B"].unique()
Out[23]: array([5, 4, 3, 6, 7])

A sütunundaki benzersiz değerleri bir liste olarak almak için (‘unique()’nin iki farklı şekilde kullanılabileceğini unutmayın)

In [24]: pd.unique(df['A']).tolist()
Out[24]: [1, 2, 3]

İşte daha karmaşık bir örnek. ‘A’nın 1’e eşit olduğu ‘B’ sütunundaki benzersiz değerleri bulmak istediğimizi varsayalım.

İlk önce, nasıl çalıştığını görebilmeniz için bir kopya tanıtalım. ‘4’ satırındaki ‘B’ sütunundaki 6’yı 4 ile değiştirelim:

In [24]: df.loc['4', 'B'] = 4    
Out[24]:    
   A  B
0  1  5
1  1  4
2  2  3
3  3  4
4  1  4
5  1  7

Şimdi verileri seçin:

In [25]: pd.unique(df[df['A'] == 1 ]['B']).tolist()
Out[25]: [5, 4, 7]

Bu, önce iç DataFrame’i düşünerek parçalanabilir:

df['A'] == 1

Bu, A sütununda 1’e eşit olan değerleri bulur ve bunlara Doğru veya Yanlış uygular. Daha sonra bunu DataFrame’in ‘B’ sütunundaki değerleri seçmek için kullanabiliriz (dış DataFrame seçimi)

Karşılaştırma için, benzersiz kullanmıyorsak liste burada. ‘A’ sütununun 1 olduğu ‘B’ sütunundaki her değeri alır

In [26]: df[df['A'] == 1]['B'].tolist()
Out[26]: [5, 4, 4, 7]
