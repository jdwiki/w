Veri çerçevelerinin Boole indekslemesi

  • Veri çerçevesine boole maskesi uygulama
  • Boole dizini ile DataFrame’e erişme
  • Sütun değerine göre verileri maskeleme
  • Dizin değerine dayalı verileri maskeleme

Bu sayfada

  • Veri çerçevesine boole maskesi uygulama
  • Boole dizini ile DataFrame’e erişme
  • Sütun değerine göre verileri maskeleme
  • Dizin değerine dayalı verileri maskeleme

“.ix”, “.loc”, “.iloc” DataFrame dizin oluşturucu nesnelerini kullanarak bir veri çerçevesindeki satırlara erişme ve bir boole maskesini kullanmaktan nasıl farklılaştığı.

Veri çerçevesine boole maskesi uygulama #

Bu bizim örnek veri çerçevemiz olacak:

  color      name   size
0   red      rose    big
1  blue    violet    big
2   red     tulip  small
3  blue  harebell  small

Sihirli ‘getitem’ veya ‘[]’ erişimcisini kullanma. Veri çerçevesiyle aynı uzunlukta bir Doğru ve Yanlış listesi vermek size şunları verecektir:

df[[True, False, True, False]]
  color   name   size
0   red   rose    big
2   red  tulip  small

Boole dizini ile DataFrame’e erişme #

Bu bizim örnek veri çerçevemiz olacak:

df = pd.DataFrame({"color": ['red', 'blue', 'red', 'blue']},
                  index=[True, False, True, False])
      color
True    red
False  blue
True    red
False  blue

.loc ile erişim

df.loc[True]
     color
True   red
True   red

.iloc ile erişim

df.iloc[True]
>> TypeError

df.iloc[1]
color    blue
dtype: object

Önemli olan, eski panda sürümlerinin boole ve tamsayı girişi arasında ayrım yapmamasıdır, bu nedenle ‘.iloc[True]’, ‘.iloc[1]’ ile aynı değeri döndürür.

.ix ile erişim

df.ix[True]
     color
True   red
True   red

df.ix[1]
color    blue
dtype: object

Gördüğünüz gibi, .ix iki davranışa sahiptir. Bu, kodda çok kötü bir uygulamadır ve bundan kaçınılmalıdır. Lütfen daha açık olmak için .iloc veya .loc kullanın.

Sütun değerine göre verileri maskeleme #

Bu bizim örnek veri çerçevemiz olacak:

  color      name   size
0   red      rose    big
1  blue    violet  small
2   red     tulip  small
3  blue  harebell  small

Bir veri çerçevesinden tek bir sütuna erişerek, sütundaki her öğeyi verilen değişkenle karşılaştırmak için basit bir karşılaştırma ‘==’ kullanabiliriz ve True ve False’dan bir ‘pd.Series’ üretebiliriz.

df['size'] == 'small'
0    False
1     True
2     True
3     True
Name: size, dtype: bool

Bu ‘pd.Series’, basit bir ’liste’nin bir uzantısı olan np.array’in bir uzantısıdır, Böylece bunu yukarıdaki örnekte olduğu gibi ‘getitem’ veya ‘[]’ erişimcisine verebiliriz.

size_small_mask = df['size'] == 'small'
df[size_small_mask]
  color      name   size
1  blue    violet  small
2   red     tulip  small
3  blue  harebell  small

Dizin değerine dayalı verileri maskeleme #

Bu bizim örnek veri çerçevemiz olacak:

         color   size
name                 
rose       red    big
violet    blue  small
tulip      red  small
harebell  blue  small

Tıpkı bir sütun değerinde olduğu gibi, indeks değerlerine dayalı bir maske oluşturabiliriz.

rose_mask = df.index == 'rose'
df[rose_mask]
     color size
name           
rose   red  big

Ama bunu yapmak neredeyse aynı

df.loc['rose']
color    red
size     big
Name: rose, dtype: object

Önemli fark, “.loc” dizinde eşleşen yalnızca bir satırla karşılaştığında, bir “pd.Series” döndürür, eşleşen daha fazla satırla karşılaşırsa bir “pd.DataFrame” döndürür. Bu, bu yöntemi oldukça kararsız hale getirir.

Bu davranış, ‘.loc’ dosyasına tek bir girdi listesi verilerek kontrol edilebilir. Bu onu bir veri çerçevesi döndürmeye zorlar.

df.loc[['rose']]
         color   size
name                 
rose       red    big
