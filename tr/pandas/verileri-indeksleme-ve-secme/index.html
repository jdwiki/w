Yola Bağlı Dilimleme #

Bir sonraki öğenin veya sonraki satırın önceden seçilen öğeye veya satıra bağlı olacağı şekilde bir serinin öğelerini veya bir veri çerçevesinin satırlarını çaprazlamak gerekli olabilir. Buna yol bağımlılığı denir.

Aşağıdaki düzensiz frekanslı s zaman serilerini göz önünde bulundurun.

#starting python community conventions
import numpy    as np
import pandas   as pd

# n is number of observations
n = 5000

day = pd.to_datetime(['2013-02-06'])
# irregular seconds spanning 28800 seconds (8 hours)
seconds = np.random.rand(n) * 28800 * pd.Timedelta(1, 's')
# start at 8 am
start = pd.offsets.Hour(8)
# irregular timeseries
tidx = day + start + seconds
tidx = tidx.sort_values()

s = pd.Series(np.random.randn(n), tidx, name='A').cumsum()
s.plot();

Yola bağlı bir koşul varsayalım. Serinin ilk üyesinden başlayarak, sonraki her bir elemanı, o eleman ile mevcut eleman arasındaki mutlak fark ‘x’ten büyük veya ona eşit olacak şekilde almak istiyorum.

Bu sorunu python üreteçlerini kullanarak çözeceğiz.

Jeneratör işlevi

def mover(s, move_size=10):
    """Given a reference, find next value with
    an absolute difference >= move_size"""
    ref = None
    for i, v in s.iteritems():
        if ref is None or (abs(ref - v) >= move_size):
            yield i, v
            ref = v

Sonra yeni bir dizi ‘hareket’ tanımlayabiliriz.

moves = pd.Series({i:v for i, v in mover(s, move_size=10)},
                  name='_{}_'.format(s.name))

Her ikisini de planlamak

moves.plot(legend=True)
s.plot(legend=True)

Veri çerçeveleri için analog şöyle olacaktır:

def mover_df(df, col, move_size=2):
    ref = None
    for i, row in df.iterrows():
        if ref is None or (abs(ref - row.loc[col]) >= move_size):
            yield row
            ref = row.loc[col]

df = s.to_frame()
moves_df = pd.concat(mover_df(df, 'A', 10), axis=1).T

moves_df.A.plot(label='_A_', legend=True)
df.A.plot(legend=True)

Boole indeksleme #

Boolean dizileri kullanılarak bir veri çerçevesinin satırları ve sütunları seçilebilir.

import pandas as pd
import numpy as np
np.random.seed(5)
df = pd.DataFrame(np.random.randint(100, size=(5, 5)), columns = list("ABCDE"), 
                  index = ["R" + str(i) for i in range(5)])
print (df)
#      A   B   C   D   E
# R0  99  78  61  16  73
# R1   8  62  27  30  80
# R2   7  76  15  53  80
# R3  27  44  77  75  65
# R4  47  30  84  86  18

mask = df['A'] > 10
print (mask)
# R0     True
# R1    False
# R2    False
# R3     True
# R4     True
# Name: A, dtype: bool

print (df[mask])
#      A   B   C   D   E
# R0  99  78  61  16  73
# R3  27  44  77  75  65
# R4  47  30  84  86  18

print (df.ix[mask, 'C'])
# R0    61
# R3    77
# R4    84
# Name: C, dtype: int32

print(df.ix[mask, ['C', 'D']])
#      C   D
# R0  61  16
# R3  77  75
# R4  84  86

Pozisyona göre seç #

‘iloc’ ( integer location‘in kısaltması) yöntemi, konum indekslerine göre bir veri çerçevesinin satırlarını seçmeye izin verir. Bu şekilde, Python’un liste dilimlemesinde olduğu gibi veri çerçeveleri dilimlenebilir.

df = pd.DataFrame([[11, 22], [33, 44], [55, 66]], index=list("abc"))

df
# Out:
#     0   1
# a  11  22
# b  33  44
# c  55  66

df.iloc[0]  # the 0th index (row)
# Out:
# 0    11
# 1    22
# Name: a, dtype: int64

df.iloc[1]  # the 1st index (row)
# Out:
# 0    33
# 1    44
# Name: b, dtype: int64

df.iloc[:2] # the first 2 rows
#     0   1
# a  11  22
# b  33  44

df[::-1]    # reverse order of rows
#     0   1
# c  55  66
# b  33  44
# a  11  22

Satır konumu, sütun konumu ile birleştirilebilir

df.iloc[:, 1]  # the 1st column
# Out[15]:
# a    22
# b    44
# c    66
# Name: 1, dtype: int64

Veri çerçevesinde farklı satırlar seçin #

İzin vermek

df = pd.DataFrame({'col_1':['A','B','A','B','C'], 'col_2':[3,4,3,5,6]})
df
# Output:
#   col_1  col_2
# 0     A      3
# 1     B      4
# 2     A      3
# 3     B      5
# 4     C      6

“col_1” içindeki farklı değerleri elde etmek için Series.unique() kullanabilirsiniz.

df['col_1'].unique()
# Output:
# array(['A', 'B', 'C'], dtype=object)

Ancak Series.unique() yalnızca tek bir sütun için çalışır.

SQL’in seçim benzersiz sütun_1, sütun_2 öğesini simüle etmek için DataFrame.drop_duplicates() kullanabilirsiniz. ):

df.drop_duplicates()
#   col_1  col_2
# 0     A      3
# 1     B      4
# 3     B      5
# 4     C      6

Bu size veri çerçevesindeki tüm benzersiz satırları alacaktır. Yani eğer

df = pd.DataFrame({'col_1':['A','B','A','B','C'], 'col_2':[3,4,3,5,6], 'col_3':[0,0.1,0.2,0.3,0.4]})
df
# Output:
#   col_1  col_2  col_3
# 0     A      3    0.0
# 1     B      4    0.1
# 2     A      3    0.2
# 3     B      5    0.3
# 4     C      6    0.4

df.drop_duplicates()
#   col_1  col_2  col_3
# 0     A      3    0.0
# 1     B      4    0.1
# 2     A      3    0.2
# 3     B      5    0.3
# 4     C      6    0.4

Benzersiz kayıtları seçerken dikkate alınacak sütunları belirtmek için bunları bağımsız değişken olarak iletin

df = pd.DataFrame({'col_1':['A','B','A','B','C'], 'col_2':[3,4,3,5,6], 'col_3':[0,0.1,0.2,0.3,0.4]})
df.drop_duplicates(['col_1','col_2'])
# Output:
#   col_1  col_2  col_3
# 0     A      3    0.0
# 1     B      4    0.1
# 3     B      5    0.3
# 4     C      6    0.4

# skip last column
# df.drop_duplicates(['col_1','col_2'])[['col_1','col_2']]
#   col_1  col_2
# 0     A      3
# 1     B      4
# 3     B      5
# 4     C      6

Etiketlerle dilimleme #

Etiketleri kullanırken, hem başlangıç ​​hem de bitiş, sonuçlara dahil edilir.

import pandas as pd
import numpy as np
np.random.seed(5)
df = pd.DataFrame(np.random.randint(100, size=(5, 5)), columns = list("ABCDE"), 
                  index = ["R" + str(i) for i in range(5)])

# Out: 
#      A   B   C   D   E
# R0  99  78  61  16  73
# R1   8  62  27  30  80
# R2   7  76  15  53  80
# R3  27  44  77  75  65
# R4  47  30  84  86  18

“R0” ile “R2” arasındaki satırlar:

df.loc['R0':'R2']
# Out: 
#      A   B   C   D   E
# R0   9  41  62   1  82
# R1  16  78   5  58   0
# R2  80   4  36  51  27

“loc"un [“iloc"dan](https://www.wikiod.com/tr/pandas/verileri-indeksleme-ve-secme#Pozisyona göre seç) ne kadar farklı olduğuna dikkat edin, çünkü “iloc” sonu hariç tutar dizin

df.loc['R0':'R2'] # rows labelled R0, R1, R2
# Out: 
#      A   B   C   D   E
# R0   9  41  62   1  82
# R1  16  78   5  58   0
# R2  80   4  36  51  27


# df.iloc[0:2] # rows indexed by 0, 1
#      A   B   C   D   E
# R0  99  78  61  16  73
# R1   8  62  27  30  80

“C” ile “E” arasındaki sütunlar:

df.loc[:, 'C':'E']
# Out: 
#      C   D   E
# R0  62   1  82
# R1   5  58   0
# R2  36  51  27
# R3  68  38  83
# R4   7  30  62

Bir veri çerçevesinin ilk/son n satırını alın #

Bir veri çerçevesinin ilk veya son birkaç kaydını görüntülemek için “head” ve “tail” yöntemlerini kullanabilirsiniz.

İlk n satırı döndürmek için DataFrame.head([n]) kullanın

df.head(n)

Son n satırı döndürmek için DataFrame.tail([n]) kullanın

df.tail(n)

n bağımsız değişkeni olmadan, bu işlevler 5 satır döndürür.

“head”/“tail” için dilim gösteriminin şöyle olacağını unutmayın:

df[:10]  # same as df.head(10)
df[-10:] # same as df.tail(10)

Eksik veri içeren satırları filtreleyin (NaN, None, NaT) #

Eksik veri içeren bir veri çerçeveniz varsa (NaN, pd.NaT, None) tamamlanmamış satırları filtreleyebilirsiniz.

df = pd.DataFrame([[0,1,2,3],
                  [None,5,None,pd.NaT],
                  [8,None,10,None],
                  [11,12,13,pd.NaT]],columns=list('ABCD'))
df
# Output:    
#     A   B   C     D
# 0   0   1   2     3
# 1 NaN   5 NaN   NaT
# 2   8 NaN  10  None
# 3  11  12  13   NaT

DataFrame.dropna eksik veri içeren en az bir alan içeren tüm satırları bırakır

df.dropna()
# Output:
#    A  B  C  D
# 0  0  1  2  3

Belirtilen sütunlarda veri eksik olan satırları bırakmak için “alt küme"yi kullanın.

df.dropna(subset=['C'])
# Output:
#     A   B   C     D
# 0   0   1   2     3
# 2   8 NaN  10  None
# 3  11  12  13   NaT

Filtrelenmiş çerçeveyle yerinde değiştirme için “inplace = True” seçeneğini kullanın.
