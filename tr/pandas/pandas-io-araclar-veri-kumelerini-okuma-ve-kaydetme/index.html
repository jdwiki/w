Pandaların resmi belgeleri, IO Araçları üzerinde dosya okumak ve yazmak için ilgili işlevlerin yanı sıra bazı işlevlerin listesini içeren bir sayfa içerir. örnekler ve ortak parametreler.

csv dosyasını DataFrame’e okuma #

“data_file.csv” dosyasını okumak için örnek, örneğin:

Dosya: #

index,header1,header2,header3
1,str_data,12,1.4
3,str_data,22,42.33
4,str_data,2,3.44
2,str_data,43,43.34

7, str_data, 25, 23.32

Kod: #

pd.read_csv('data_file.csv')

Çıktı: #

   index    header1  header2  header3
0      1   str_data       12     1.40
1      3   str_data       22    42.33
2      4   str_data        2     3.44
3      2   str_data       43    43.34
4      7   str_data       25    23.32

Bazı yararlı argümanlar: #

  • ay The default field delimiter is a comma ,. Use this option if you need a different delimiter, for instance pd.read_csv('data_file.csv', sep=';')

  • index_colindex_col = n (n bir tamsayı) ile pandalara DataFrame’i indekslemek için n sütununu kullanmalarını söylersiniz. Yukarıdaki örnekte:

     pd.read_csv('data_file.csv',  index_col=0)

    Output:

               header1  header2  header3
 index
  1       str_data       12     1.40
  3       str_data       22    42.33
  4       str_data        2     3.44
  2       str_data       43    43.34
  7       str_data       25    23.32

  • skip_blank_lines Varsayılan olarak boş satırlar atlanır. Boş satırlar eklemek için skip_blank_lines=False kullanın (NaN değerleriyle doldurulacaklar)

       pd.read_csv('data_file.csv',  index_col=0,skip_blank_lines=False)

    Output:

                header1  header2  header3
   index
    1      str_data       12     1.40
    3      str_data       22    42.33
    4      str_data        2     3.44
    2      str_data       43    43.34
   NaN          NaN      NaN      NaN
    7      str_data       25    23.32

  • parse_dates Tarih verilerini ayrıştırmak için bu seçeneği kullanın.

    File:

   date_begin;date_end;header3;header4;header5
   1/1/2017;1/10/2017;str_data;1001;123,45
   2/1/2017;2/10/2017;str_data;1001;67,89
   3/1/2017;3/10/2017;str_data;1001;0

Code to parse columns `0` and `1` as dates:

   pd.read_csv('f.csv', sep=';', parse_dates=[0,1])

Output:

     date_begin   date_end   header3  header4 header5
   0 2017-01-01 2017-01-10  str_data     1001  123,45
   1 2017-02-01 2017-02-10  str_data     1001   67,89
   2 2017-03-01 2017-03-10  str_data     1001       0

By default, the date format is inferred. If you want to specify a date format you can use for instance

   dateparse = lambda x: pd.datetime.strptime(x, '%d/%m/%Y')
   pd.read_csv('f.csv', sep=';',parse_dates=[0,1],date_parser=dateparse)

Output:

     date_begin   date_end   header3  header4 header5
   0 2017-01-01 2017-10-01  str_data     1001  123,45
   1 2017-01-02 2017-10-02  str_data     1001   67,89
   2 2017-01-03 2017-10-03  str_data     1001       0

CSV dosyasına kaydet #

Varsayılan parametrelerle kaydet:

df.to_csv(file_name)

Belirli sütunları yazın:

df.to_csv(file_name, columns =['col'])

Zor sınırlayıcı ‘,’ - değiştirmek için:

df.to_csv(file_name,sep="|")

Başlık olmadan yazın:

df.to_csv(file_name, header=False)

Belirli bir başlıkla yazın:

df.to_csv(file_name, header = ['A','B','C',...]

Belirli bir kodlama kullanmak için (ör. ‘utf-8’) kodlama bağımsız değişkenini kullanın:

df.to_csv(dosya_adı, kodlama=‘utf-8’)

read_csv ile tarih sütunlarını ayrıştırma #

Nginx erişim günlüğünü oku (birden çok alıntı karakteri) #

Birden çok alıntı için sep yerine normal ifade kullanın:

df = pd.read_csv(log_file,
              sep=r'\s(?=(?:[^"]*"[^"]*")*[^"]*$)(?![^\[]*\])',
              engine='python',
              usecols=[0, 3, 4, 5, 6, 7, 8],
              names=['ip', 'time', 'request', 'status', 'size', 'referer', 'user_agent'],
              na_values='-',
              header=None
                )

raw_data = {'first_name': ['John', 'Jane', 'Jim'],

df.to_csv('data_file.csv')
