Tabloyu DataFrame’e oku #

Üstbilgi, altbilgi, satır adları ve dizin sütunu içeren tablo dosyası: #

dosya: tablo.txt

This is a header that discusses the table file
to show space in a generic table file

index  name     occupation
1      Alice    Salesman
2      Bob      Engineer
3      Charlie  Janitor  

This is a footer because your boss does not understand data files

kod:

import pandas as pd
# index_col=0 tells pandas that column 0 is the index and not data 
pd.read_table('table.txt', delim_whitespace=True, skiprows=3, skipfooter=2, index_col=0)

çıktı:

          name occupation
index
1        Alice   Salesman
2          Bob   Engineer
3      Charlie    Janitor

Satır adları veya dizin içermeyen tablo dosyası: #

dosya: tablo.txt

Alice    Salesman
Bob      Engineer
Charlie  Janitor

kod:

import pandas as pd 
pd.read_table('table.txt', delim_whitespace=True, names=['name','occupation'])

çıktı:

      name occupation
0    Alice   Salesman
1      Bob   Engineer
2  Charlie    Janitor

Tüm seçenekler panda belgelerinde bulunabilir burada

Virgül yerine noktalı virgülle ayrılmış başlık içeren veriler #

dosya: tablo.csv

index;name;occupation
1;Alice;Saleswoman
2;Bob;Engineer
3;Charlie;Janitor

kod:

import pandas as pd
pd.read_csv('table.csv', sep=';', index_col=0)

çıktı:

          name occupation
index

2          Bob   Engineer
3      Charlie    Janitor

Google e-tablo verilerini pandaların veri çerçevesine toplayın #

Bazen google e-tablolarından veri toplamamız gerekir. Google elektronik tablolarından veri toplamak için gspread ve oauth2client kitaplıklarını kullanabiliriz. Veri toplamak için bir örnek:

Kod:

from __future__ import print_function
import gspread
from oauth2client.client import SignedJwtAssertionCredentials
import pandas as pd
import json

scope = ['https://spreadsheets.google.com/feeds']

credentials = ServiceAccountCredentials.from_json_keyfile_name('your-authorization-file.json', scope)

gc = gspread.authorize(credentials)

work_sheet = gc.open_by_key("spreadsheet-key-here")
sheet = work_sheet.sheet1
data = pd.DataFrame(sheet.get_all_records()) 

print(data.head())
