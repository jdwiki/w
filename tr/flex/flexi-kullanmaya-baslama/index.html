Kurulum veya Kurulum (Apache Flex) #

Apache Flex SDK’yı kurmanın iki yolu vardır. İşlemi otomatikleştiren bir Adobe AIR uygulaması olan Apache Flex SDK Installer’ı kullanabilirsiniz (Windows veya OS X/macOS’ta). Ya da manüel olarak kurabilirsiniz ki bu açıkça platformunuzda daha fazla konfor gerektirir ancak kurulum detaylarında daha fazla esneklik sağlar.

Apache Flex Yükleyicisini Kullanarak Apache Flex SDK’yı Yükleme #

  1. İşletim sisteminiz için installer ikili dosyasını indirin (Window, Mac or Linux).
  2. Yükleyiciyi çalıştırın
  3. İndirmek istediğiniz SDK sürümünü seçin. Apache Flex projesi, en son yayınlanan sürümü indirmenizi önerir.
  4. Flash Player sürümünü seçin. Flex projesi, onu varsayılan olarak bırakmanızı önerir. Sonrakine tıkla**.
  5. SDK’nın yüklenmesini istediğiniz dizini seçin. Sonrakine tıkla**.
  6. Tüm lisans sözleşmelerini kontrol edin. Yükle‘yi tıklayın.
  7. Yükleyicinin Apache Flex SDK’yı indirip kurmasını bekleyin.

Apache Flex SDK’yı Manuel Olarak Yükleme #

NOT: Bu uzun ve ilgili bir süreç olabilir.

Sisteminizi hazırlayın #

  1. En son Java JDK’yı indirin ve yükleyin
  2. Ant’ın en son sürümünü indirin ve yükleyin
  3. Git’in en son sürümünü indirin ve yükleyin
  4. En son JUnit’i indirin ve yükleyin

Klasör Yapısını Ayarlayın #

  1. Flex SDK için bilgisayarınızda, genellikle kök veya Belgeler dizininde depolanan aşağıdaki dizin yapısını oluşturun.
/ApacheFlex
/ApacheFlex/dependencies
/ApacheFlex/source

İndirme Kaynağı #

  1. Bir terminal açın ve dizini şu şekilde değiştirin:
/ApacheFlex/source
  1. Flex SDK kaynağını indirmek için aşağıdaki git komutunu çalıştırın:
git clone https://git-wip-us.apache.org/repos/asf/flex-sdk.git flex-sdk
  1. TLF kaynağını indirmek için aşağıdaki git komutunu çalıştırın:
git clone https://git-wip-us.apache.org/repos/asf/flex-tlf.git flex-tlf
  1. flex-sdk dizinine geçin:
/ApacheFlex/source/flex-sdk
  1. ‘geliştir’ dalına geçin (İsteğe bağlı):
git checkout develop

Ortam Özelliklerini Düzenleyin #

Derleme komut dosyalarının çalışması için Ortam Değişkenlerini ayarlamanız gerekir. Ortam özellikleri dosyası, işlem boyunca kullanacağınız bağımlılıkların konumlarına işaret eden ortam değişkenlerini içerir. Bunlar, SDK’yı çalıştırmak ve derlemek için çeşitli derleme komut dosyaları tarafından kullanılır. Bu öğreticinin geri kalanı boyunca ek ortam değişkenleri ekleyeceksiniz. Derleme komut dosyasını daha sonra çalıştırdığınızda, komut dosyası sağlanan yolları doğrular ve yanlışlarsa sizi uyarır.

  1. Ortam özellikleri dosyasını oluşturun:

“flex-sdk” dizininde “env-template.properties” dosyasının bir kopyasını oluşturun ve “env.properties” olarak adlandırın.

Java ve Ant’ı zaten yüklediğiniz için, “env.properties” dosyasının sonuna aşağıdakini ekleyerek bunların yolunu belirleyin:

env.JAVA_HOME=/System/Library/Frameworks/JavaVM.framework/Versions/Current/Commands/java

env.ANT_HOME=/Applications/Ant/bin
  1. Sonra yolu TLF dizinine ekleyin:
env.TLF_HOME=/ApacheFlex/source/flex-tlf

Gerekli Bağımlılıkları Ekle #

Adobe AIR SDK’sı #

  1. Air SDK Flex Edition’ı indirin (Flex için özel olarak yapılmış sürümü indirdiğinizden emin olun).
  2. Çıkartın:
/ApacheFlex/dependencies/AIRSDK/4.0

burada son dizin adı, indirdiğiniz AIR SDK sürümünü belirtir.

  1. ‘AIR_HOME’ ortam değişkenini önceki adımda seçtiğiniz AIR SDK yoluna ayarlayın. Örneğin,
env.ANT_HOME=/ApacheFlex/dependencies/AIRSDK/4.0

Adobe Flash Player Projektör #

  1. Flash Player Projektörünü İndirin
  2. Yükleyiciyi çalıştırın. Projector Content Debugger kurulum programı hiçbir yere kurulmaz (en azından benim için). Yükleyiciyi çalıştırdıktan sonra, istediğiniz konuma kopyalamanız gerekir. Bu durumda, /ApacheFlex/dependencies/player dizinine kopyalayın. Not: OSX’te, Flash Player’ın yoluna ‘.app/Contents/MacOS/Flash Player Debugger’ öğesini ekleyin.
/ApacheFlex/dependencies/player/Flash Player.app/Contents/MacOS/Flash Player Debugger
  1. ‘FLASHPLAYER_DEBUGGER’ ortam değişkenini projektörün kurulum yoluna ayarlayın.
env.FLASHPLAYER_DEBUGGER=/ApacheFlex/dependencies/player/Flash Player.app/Contents/MacOS/Flash Player Debugger

Flash Player Eklentisini de kullanabilirsiniz. Genellikle burada bulunur: /Library/Internet Plug-ins/Flash Player/ Tam yol şöyle olacaktır: /Library/Internet Plug-ins/Flash Player.app/Contents/MacOS/Flash Player Debugger

Adobe Flash Player playerglobal.swc #

  1. playerglobal.swc’yi indirin - Bu, az önce indirdiğiniz içerik hata ayıklayıcı ile aynı sayfadadır.
  2. Şuraya kopyalayın:
/ApacheFlex/dependencies/PlayerGlobal/player/12.0

sayı, indirdiğiniz sürüme karşılık gelir. 3. Kopyalanan dosyayı ‘playerglobal.swc’ olarak yeniden adlandırın 4. “PLAYERGLOBAL_HOME” ortam değişkenini şu şekilde ayarlayın:

env.PLAYERGLOBAL_HOME=/ApacheFlex/Dependencies/PlayerGlobal/player

Adobe Pixel Bender Araç Seti #

  1. Windows veya Mac için Pixel Bender’ı indirin
  2. Pixel Bender’ı yükleyin. Programı kurduğunuzda size kurulum yerini seçme seçeneği sunar. Yüklemeden önce bu konumu kopyalayın.
  3. PIXELBENDER_HOME ortam değişkenini kurulum dizinine ayarlayın.

Sistemi Hazırla #

Test için, Adobe Flash Player’ın “mm.cfg” dosyasının uygun özelliklerin ayarlanmış olması ve bir “FlashPlayerTrust” dosyasının yerel SWF’lerin yerel dosyalara erişmesine izin vermesi gerekir.

  1. mm.cfg dosyasını bir düzenleyicide açın. mm.cfg dosyasının konumu işletim sisteminize bağlıdır. Bu sayfaya bakın.
  2. Mevcut değilse, Adobe Flash Player Projektör içerik hata ayıklayıcısını manuel olarak başlatın ve o sırada mm.cfg dosyasını oluşturmalıdır.
  3. Dosyada aşağıdaki girişlerin olduğunu ekleyin veya doğrulayın:
ErrorReportingEnable=1
TraceOutputFileEnable=1
  1. mm.cfg dosyasıyla aynı dizinde FlashPlayerTrust dizini bulunur.
  2. Bu dizini açın ve ApacheFlex.cfg adlı yeni bir metin dosyası oluşturun.
  3. Bu dosyaya Apache Flex SDK’nın üst dizinini ekleyin. Örneğin:
#Path to Apache Flex Source
/ApacheFlex/source
  1. Dosyayı kaydedin.

Yapı Özelliklerini Ayarlayın #

  1. flex-sdk dizininde yapı özellikleri dosyası bulunur. Bu, indirdiğiniz yazılımın sürüm numaralarını içerir.
  2. Bu dosyayı açın ve kullandığınız Flash Player ve AIR SDK sürümüne aşağıdaki değerleri ayarlayın:
# Flash player version for which player global swc to use
playerglobal.version = 12.0

# AIR version number
air.version = 4.0

Bu değerler, SDK’ya giden yolun oluşturulmasında kullanılır. Yani playerglobal.swc dosyasını aşağıdaki dizine kaydettiyseniz: /ApacheFlex/dependencies/PlayerGlobal/player/12.0 playerglobal.version’un değeri 12.0 olacaktır.

Not: Bu değer yolu oluşturduğundan, yolunuz bunu içeriyorsa 12 yerine tam değeri 12.0 kullandığınızdan emin olun.

Not: Üzerinde çalıştığınız Flex SDK sürümünde henüz bahsedilmeyen yeni bir Flash Player sürümü kullanıyorsanız, şu mesajı alabilirsiniz: Error: configuration variable ‘swf-version’ value contains unknown token ‘playerglobal.swfversion’ /ApacheFlex/source/flex-sdk/frameworks/build.xml dosyasını güncellenmiş bir sürüm kontrol koduyla güncellemeniz gerekecek.
