Selam Dünya! (Ayriyeten) #

# not all of this is required, but just here for reference
bl_info = {
    "name": "Hello World",                          # name of the add-on
    "author": "Blender developer",                  # name of the author
    "version": (1, 0),                              # version number for the add-on
    "blender": (2, 78, 0),                          # version of Blender the add-on is compatible with
    "location": "Space > Hello World",              # where the user can find the add-on
    "description": "Greets something",              # add-on description
    "warning": "Beta version",                      # whatever the user needs to be warned about
    "wiki_url": "",                                 # documentation link
    "category": "Development",                      # add-on category
    }

# the blender python module
import bpy
# this is just for convenience - could just use as bpy.props.StringProperty, but there are normally lots of properties
from bpy.props import StringProperty


class HelloWorld(bpy.types.Operator):
    """Says hello to the world."""       # python docstring 
    bl_idname = "greetings.hello_world"  # this will be callable with bpy.ops.greetings.hello_world()
    bl_label = "Hello World"             # the user-friendly name for this operator (e.g., in a button)
    bl_options = {'REGISTER', 'UNDO'}    # 'UNDO' is only here for reference (you can't actually take back what you say)

    name = StringProperty(
            name="name",
            default="world",
            description="Who to say hello to",
            )

    def execute(self, context):
        # Note: The execute method is called when the user clicks on the operator or calls it from python.

        message = "Hello, " + self.name + "!"
        
        # print to console
        print(message)
        
        # show a popup that automatically goes away (in info area's header)
        self.report({'INFO'}, message)
        
        # show a popup under the cursor that doesn't go away automatically
        self.report({'ERROR'}, message)

        # generally return {'FINISHED'} or {'CANCELLED'} at the end of the execute method
        return {'FINISHED'}


# this is automatically called when the add-on is enabled
def register():
    # simply register the class
    bpy.utils.register_class(HelloWorld)
    

# this is automatically called when the add-on is disabled
def unregister():
    # simply unregister the class
    bpy.utils.unregister_class(HelloWorld)


# common "pythonic" approach to main()...call register() automatically
if __name__ == "__main__":
    register()

Bunu bir python dosyasına (.py) kaydedin, ardından Blender’da normal bir eklenti olarak yükleyin. Blender’da hemen hemen herhangi bir alana “boşluk” yazın ve oluşturduğunuz operatörü bulmak için “Merhaba Dünya” yazın.

Görünüm alanı ve pencereleri #

Blender’ın görünüm alanı birçok farklı pencereden oluşan dinamik, değiştirilebilir bir arayüzdür. Varsayılan olarak çalışan program ile görünüm alanı 5 farklı pencereden oluşur. Windows, üst veya sol alt köşedeki küçük kare gösterge simgeleri aranarak tanımlanabilir. Şunlara benzeyebilirler:

(3D görünüm simgesi)

(zaman çizelgesi simgesi)

(özellikler simgesi)

Bu küçük resimler, bağlı oldukları pencerenin türünü belirtir ve pencere türü, bunlardan birine tıklanarak ve başka bir pencere seçilerek değiştirilebilir.

Tüm pencereler yeniden boyutlandırılabilir ve bölünebilir, yani her biri iki parçaya bölünebilir, boyutları değiştirilebilir veya tek bir pencerede birleştirilebilir.

Bu işlevi denemek için önce, menü düğmelerinden hemen sonra ekranın en üstünde görünen ekran düzeni seçicinin konumuna dikkat edin:

Bu seçici, istediğiniz zaman varsayılan pencere düzenine geri dönmenize izin verir ve (Blender’ın birçok seçicisi gibi) dinamik bir liste gibi davranır. Bu, denemeye başlamadan önce bu düzeni kaydetmek istiyorsanız, düzeni kopyalamak için + düğmesini tıklayın ve ardından değişikliklerinizi yapın. Düzen adına tıklayarak ve varsayılanı yeniden seçerek düzene dönün.

Artık düzene geri dönülebildiğine göre, ana pencerenin köşesindeki tutamaçlardan birini sürükleyin - şöyle görünür:

İmleciniz artı işaretine dönüşecek ve pencere ikiye bölünecektir.

İki pencereyi birleştirmek, pencerenin karşı köşesindeki tutamak ile yapılabilir. Yukarıdaki resimde, pencereyi bölmek için kullanılan tutamak sol alt köşedeydi: bu nedenle, pencereyi bir başkasıyla birleştirmek için kullanılan tutucu sağ üsttekidir.

Pencereyi bir başkasıyla birleştirmek için, bu ikinci tutamacını birleştirmek istediğiniz yöne sürüklemeniz yeterlidir. İçe doğru daraltmak istiyorsanız, önce onu kendisinden uzaklaştırmanız gerekebilir, şöyle:

Kurulum veya Kurulum #

  • https://www.blender.org/download/ adresine gidin

  • İşletim sisteminizi seçin

  • İşletim sisteminize uygun blender sürümünün yanındaki uygun aynaya tıklayın. Genellikle geçerli konumunuza en yakın aynaya tıklayabilirsiniz. (daha fazla bilgi)

  • Ayrıca, sayfanın alt kısmında günlük deneysel yapılara ve kaynak koduna bağlantılar da bulunmaktadır. Bu, en son özelliklere erişmenizi sağlayabilir (kararlılık pahasına).

  • Blender’ı başarıyla indirdiniz!

Blender’ı indirdikten sonra, açmak için sıkıştırmayı açmanız ve ardından blender.exe veya blender.app dosyasını açmanız yeterlidir.
