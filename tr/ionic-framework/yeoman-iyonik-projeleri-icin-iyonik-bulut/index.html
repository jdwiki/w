Yo (Yeoman) İyonik Projeler için İyonik Platform (İyonik Bulut) #

İyonik Platform:

Ionic uygulamalarınızı kolay yoldan derleyin, aktarın, dağıtın ve ölçeklendirin.

Başlık Açıklaması:
Ionic Platform, platformlar arası mobil uygulamaları yönetmek ve ölçeklendirmek için bir bulut platformudur. Entegre hizmetler, sizin ve ekibinizin uygulamalarınızı verimli bir şekilde oluşturmasına, dağıtmasına ve büyütmesine olanak tanır.

Belge Hedefi:
İyonik Platform, standart İyonik projelerle iyi çalışır. Ancak herhangi bir Standart Olmayan Dizin Yapısını izleyen projeler birkaç engelle karşılaşabilir. Bu belge, Yeoman kullanılarak oluşturulan Ionic projelerinde Ionic Platform’u kullanma adımlarını sağlar.

Belge Kapsamı:
Bu belge, Yeoman kullanarak bir Ionic projesi oluşturmak ve bunu Ionic Platform Web Client kullanarak Ionic Platform ile entegre etmek için temel adımları kapsar. Bu belge, Ionic Deploy, Ionic Analytics ve Ionic Push’u kullanmanın temel adımlarını kapsar.

Hedef kitlesi:
Bu belgenin hedeflenen hedef kitlesi, aşağıdaki Ön Koşullara aşina olan, hem başlangıç ​​düzeyinde hem de uzman düzeyinde uzmanlığa sahip Web/Mobil Uygulama geliştiricileridir.

Ön koşullar:
Bu belgeyi denemeden önce aşağıdaki çerçevelere/araçlara aşina olmalısınız.

  • AngularJs: https://docs.angularjs.org/guide
  • IonicFramework: http://ionicframework.com/docs/guide
  • Yeoman: http://yeoman.io/codelab/index.html
  • İyonik Jeneratör: https://github.com/diegonetto/generator-ionic
  • İyonik Platform: https://ionic.io/platform

İyonik Çerçeve oluşturucu #

Modern web uygulamaları için Web’in İskele aracı olan Yeoman’dan Ionic Framework için bir jeneratör

Node.js, Chrome’un V8 JavaScript motoru üzerine kurulmuş bir JavaScript çalışma zamanıdır. npm, JavaScript için paket yöneticisidir. Düğümü (ve npm’yi) http://nodejs.org adresinden indirin ve yükleyin

$ npm install npm –g
$ npm install -g yo

Yeoman, üretken kalmanıza yardımcı olacak en iyi uygulamaları ve araçları önererek yeni projelere başlamanıza yardımcı olur.

$ yo ionic [app-name]

package.json dosyasında devDependencies’e aşağıdakileri ekleyin

"grunt-string-replace": "^1.2.1"

bower.json dosyasında aşağıdakileri bağımlılıklara dahil edin

"ionic-platform-web-client": "^0.7.1"

Gruntfile.js dosyasında scripts klasörünü ‘js’ olarak değiştirin. Gerekirse index.html dosyasını da değiştirin.

grunt.initConfig({   yeoman: {…………
    scripts: 'js',
    ………… } })

O zaman koş

$ bower install && npm install
$ grunt
$ grunt serve

$ cordova platform add android 
$ grunt build:android --debug

iyonik-platform-web-istemcisi #

Ionic platformuyla etkileşimler sağlayan bir web istemcisi.

Uygulamanızın Ionic Platform ile konuşmasına izin vermek için bazı kodlara ihtiyacımız var. Ionic uygulamasının eklentiler ve Ionic.io platformuyla arayüz oluşturması için Ionic platform web istemcisini eklememiz gerekiyor.

$ ionic io init

app.js dosyanıza ‘ionic.service.core’ modül bağımlılığını ekleyin. Gruntfile.js dosyasına, aşağıda verildiği gibi ‘ionicSettings’ adlı grunt görevini ekleyin.

grunt.initConfig({
ionicSettings: JSON.stringify(grunt.file.readJSON('./.io-config.json')),

ionicIoBundlePath: 'www/bower_components/ionic-platform-web-client/dist/ionic.io.bundle.min.js',

'string-replace': {
  ionicSettings: {
    files: {
      '<%= ionicIoBundlePath %>': '<%= ionicIoBundlePath %>',
    },
    options: {
      replacements: [
        {
          pattern: 
        '"IONIC_SETTINGS_STRING_START";"IONIC_SETTINGS_STRING_END"',
          replacement: 
        '"IONIC_SETTINGS_STRING_START";var settings =<%= ionicSettings %>; return { get: function(setting) { if (settings[setting]) { return settings[setting]; } return null; } };"IONIC_SETTINGS_STRING_END";'
        }
      ]
    }
  }
},
       copy: {
    ionicPlatform:{
                expand: true,
                cwd: 'app/bower_components/ionic-platform-web-client/dist/',
                src: ['**'],
                dest: 'www/bower_components/ionic-platform-web-client/dist'
               }
    }
});

grunt.registerTask('ionicSettings', ['copy:ionicPlatform','string-replace:ionicSettings']);
