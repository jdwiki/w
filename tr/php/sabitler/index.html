Sözdizimi #

  • tanımla ( string $isim , karışık $değer [, bool $case_inresponsive = false ] )
  • const CONSTANT_NAME = DEĞER;

Sabitler daha sonra değiştirilmemesi gereken değerleri saklamak için kullanılır. Ayrıca, özellikle ortamı (geliştirme/üretim) tanımlayan konfigürasyon parametrelerini depolamak için sıklıkla kullanılırlar.

Sabitlerin değişkenler gibi türleri vardır, ancak tüm türler bir sabiti başlatmak için kullanılamaz. Nesneler ve kaynaklar, sabitler için hiçbir şekilde değer olarak kullanılamaz. Diziler, PHP 5.6’dan başlayarak sabitler olarak kullanılabilir.

Bazı sabit adları PHP tarafından ayrılmıştır. Bunlara “doğru”, “yanlış”, “boş” ve ayrıca modüle özgü birçok sabit dahildir.

Sabitler genellikle büyük harflerle adlandırılır.

Sabitleri tanımlama #

Sabitler, “const” ifadesi veya “define” işlevi kullanılarak oluşturulur. Kural, sabit adlar için BÜYÜK HARF kullanmaktır.

Açık değerler kullanarak sabiti tanımlayın #

const PI = 3.14; // float
define("EARTH_IS_FLAT", false); // boolean
const "UNKNOWN" = null; // null
define("APP_ENV", "dev"); // string
const MAX_SESSION_TIME = 60 * 60; // integer, using (scalar) expressions is ok

const APP_LANGUAGES = ["de", "en"]; // arrays


define("BETTER_APP_LANGUAGES", ["lu", "de"]); // arrays

Başka bir sabit kullanarak sabiti tanımlayın #

bir sabitiniz varsa, ona göre başka bir tane tanımlayabilirsiniz:

const TAU = PI * 2;
define("EARTH_IS_ROUND", !EARTH_IS_FLAT);
define("MORE_UNKNOWN", UNKNOWN);
define("APP_ENV_UPPERCASE", strtoupper(APP_ENV)); // string manipulation is ok too
// the above example (a function call) does not work with const:
// const TIME = time(); # fails with a fatal error! Not a constant scalar expression
define("MAX_SESSION_TIME_IN_MINUTES", MAX_SESSION_TIME / 60);

const APP_FUTURE_LANGUAGES = [-1 => "es"] + APP_LANGUAGES; // array manipulations


define("APP_BETTER_FUTURE_LANGUAGES", array_merge(["fr"], APP_BETTER_LANGUAGES));

Ayrılmış sabitler #

Bazı sabit adları PHP tarafından ayrılmıştır ve yeniden tanımlanamaz. Tüm bu örnekler başarısız olacaktır:

define("true", false); // internal constant
define("false", true); // internal constant
define("CURLOPT_AUTOREFERER", "something"); // will fail if curl extension is loaded

Ve bir Bildirim verilecektir:

Constant ... already defined in ...

koşullu tanımlar #

Aynı değişkeni tanımlayabileceğiniz birkaç dosyanız varsa (örneğin, ana yapılandırmanız ve ardından yerel yapılandırmanız), aşağıdaki sözdizimi çakışmaları önlemeye yardımcı olabilir:

defined("PI") || define("PI", 3.1415); // "define PI if it's not yet defined"

“const” ve “tanımla” #

‘define’ bir çalışma zamanı ifadesi, ‘const’ ise bir derleme zamanı ifadesidir.

Böylece ‘define’ dinamik değerlere (yani fonksiyon çağrıları, değişkenler vb.) ve hatta dinamik isimlere ve koşullu tanımlamaya izin verir. Ancak her zaman kök ad alanına göre tanımlar.

“const” statiktir (yalnızca diğer sabitler, skalerler veya diziler ve bunların yalnızca sınırlı bir kümesiyle, sözde _sabit skaler ifadeler, yani aritmetik, mantıksal ve karşılaştırma operatörleri ile dizi referansını kaldırmaya izin veren işlemlere izin verir), ancak bunlar o anda etkin olan ad alanı ile otomatik olarak ad alanı öneki.

“const” yalnızca diğer sabitleri ve skaleri değerler olarak destekler ve işlem yapmaz.

Sınıf Sabitleri #

Sabitler, bir “const” anahtar sözcüğü kullanılarak sınıfların içinde tanımlanabilir.

class Foo {
    const BAR_TYPE = "bar";

    // reference from inside the class using self::
    public function myMethod() {
        return self::BAR_TYPE;
    }
}

// reference from outside the class using <ClassName>::
echo Foo::BAR_TYPE;

Bu, öğe türlerini saklamak için kullanışlıdır.

<?php

class Logger {
    const LEVEL_INFO = 1;
    const LEVEL_WARNING = 2;
    const LEVEL_ERROR = 3;

    // we can even assign the constant as a default value
    public function log($message, $level = self::LEVEL_INFO) {
        echo "Message level " . $level . ": " . $message;
    }
}

$logger = new Logger();
$logger->log("Info"); // Using default value
$logger->log("Warning", $logger::LEVEL_WARNING); // Using var
$logger->log("Error", Logger::LEVEL_ERROR); // using class

Sabitin tanımlı olup olmadığı kontrol ediliyor #

Basit kontrol #

Sabitin tanımlı olup olmadığını kontrol etmek için ’tanımlı’ fonksiyonunu kullanın. Bu işlevin sabitin değeriyle ilgilenmediğini, yalnızca sabitin var olup olmadığını umursadığını unutmayın. Sabitin değeri “boş” veya “yanlış” olsa bile, işlev yine de “doğru” değerini döndürür.

<?php

define("GOOD", false);

if (defined("GOOD")) {
    print "GOOD is defined" ; // prints "GOOD is defined"

    if (GOOD) {
        print "GOOD is true" ; // does not print anything, since GOOD is false
    }
}

if (!defined("AWESOME")) {
   define("AWESOME", true); // awesome was not defined. Now we have defined it 
}

Sabitin kodunuzda yalnızca onu tanımladığınız satırdan sonra “görünür” hale geldiğini unutmayın:

<?php

if (defined("GOOD")) {
   print "GOOD is defined"; // doesn't print anyhting, GOOD is not defined yet.
}

define("GOOD", false);

if (defined("GOOD")) {
   print "GOOD is defined"; // prints "GOOD is defined"
}

Tüm tanımlanmış sabitleri alma #

PHP tarafından oluşturulanlar da dahil olmak üzere tüm tanımlanmış sabitleri almak için get_defined_constants fonksiyonunu kullanın:

<?php

$constants = get_defined_constants();
var_dump($constants); // pretty large list

Yalnızca uygulamanız tarafından tanımlanan sabitleri almak için, betiğinizin başında ve sonunda (normalde önyükleme işleminden sonra) işlevi çağırın:

<?php

$constants = get_defined_constants();

define("HELLO", "hello"); 
define("WORLD", "world"); 

$new_constants = get_defined_constants();

$myconstants = array_diff_assoc($new_constants, $constants);
var_export($myconstants); 
   
/* 
Output:

array (
  'HELLO' => 'hello',
  'WORLD' => 'world',
) 
*/

Bazen hata ayıklamak için yararlıdır

Sabit diziler #

Diziler, PHP 5.6 sürümünden itibaren düz sabitler ve sınıf sabitleri olarak kullanılabilir:

Sınıf sabiti örneği #

class Answer {
    const C = [2,4];
}

print Answer::C[1] . Answer::C[0]; // 42

Düz sabit örnek #

const ANSWER = [2,4];
print ANSWER[1] . ANSWER[0]; // 42

Ayrıca PHP 7.0 sürümünden bu işlevsellik, düz sabitler için define işlevine taşınmıştır.

define('VALUES', [2, 3]);
define('MY_ARRAY', [
    1,
    VALUES,
]);

print MY_ARRAY[1][1]; // 3

Sabitleri kullanma #

Sabiti kullanmak için adını kullanmanız yeterlidir:

if (EARTH_IS_FLAT) {
    print "Earth is flat";
}

print APP_ENV_UPPERCASE;

veya sabitin adını önceden bilmiyorsanız, “sabit” işlevini kullanın:

// this code is equivalent to the above code
$const1 = "EARTH_IS_FLAT";
$const2 = "APP_ENV_UPPERCASE";

if (constant($const1)) {
    print "Earth is flat";
}

print constant($const2);
