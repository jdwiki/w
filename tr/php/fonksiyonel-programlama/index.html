PHP’nin işlevsel programlaması, işlevlere dayanır. PHP’deki işlevler, bir dizi eylemi gerçekleştirmek için düzenli, yeniden kullanılabilir kod sağlar. Fonksiyonlar kodlama sürecini basitleştirir, gereksiz mantığı önler ve kodun takip edilmesini kolaylaştırır. Bu konu, PHP’de işlevlerin, bağımsız değişkenlerin, parametrelerin, dönüş ifadelerinin ve kapsamın bildirilmesini ve kullanımını açıklar.

Kapanışlar #

Kapatma, kapsam dışına erişemeyen anonim bir işlevdir.

Anonim bir işlevi bu şekilde tanımlarken, o işlev için bir “ad alanı” yaratırsınız. Şu anda yalnızca bu ad alanına erişimi vardır.

$externalVariable = "Hello";
$secondExternalVariable = "Foo"; 
$myFunction = function() { 
  
  var_dump($externalVariable, $secondExternalVariable); // returns two error notice, since the variables aren´t defined 

}

Herhangi bir harici değişkene erişimi yoktur. Bu ad alanına harici değişkenlere erişim izni vermek için, onu kapanışlar (use()) aracılığıyla tanıtmanız gerekir.

$myFunction = function() use($externalVariable, $secondExternalVariable) {
   var_dump($externalVariable, $secondExternalVariable); // Hello Foo
}

Bu, büyük ölçüde PHP’nin sıkı değişken kapsamına atfedilir - Eğer bir değişken kapsam içinde tanımlanmadıysa veya ‘global’ ile getirilmediyse, o zaman mevcut değildir.

Ayrıca not edin:

Değişkenleri üst kapsamdan devralmak, değişkenleri kullanmakla aynı şey değildir. küresel değişkenler. Global değişkenler global kapsamda bulunur; hangi işlev yürütülürse yürütülsün aynı.

Bir öğenin ana kapsamı kapatma, kapatmanın bildirildiği işlevdir (değil mutlaka çağrıldığı işlev).

[Anonim İşlevler için PHP Belgelerinden1 alınmıştır.

PHP’de, kapanışlar erken bağlayıcı bir yaklaşım kullanır. Bu, “use” anahtar sözcüğü kullanılarak kapatmanın ad alanına iletilen değişkenlerin, kapatma tanımlandığında aynı değerlere sahip olacağı anlamına gelir.

Bu davranışı değiştirmek için referans değişkenini iletmelisiniz.

$rate = .05;

// Exports variable to closure's scope
$calculateTax = function ($value) use ($rate) {
    return $value * $rate;
};

$rate = .1; 

print $calculateTax(100); // 5

$rate = .05;

// Exports variable to closure's scope
$calculateTax = function ($value) use (&$rate) { // notice the & before $rate
    return $value * $rate;
};

$rate = .1;

print $calculateTax(100); // 10

Kapaklı/kapanışsız anonim işlevler tanımlanırken varsayılan bağımsız değişkenler örtük olarak gerekli değildir.

$message = 'Im yelling at you';

$yell = function() use($message) {
    echo strtoupper($message);
};

$yell(); // returns: IM YELLING AT YOU

Değişkenlere atama #

Anonim işlevler, bir geri aramanın beklendiği durumlarda parametreler olarak kullanılmak üzere değişkenlere atanabilir:

$uppercase = function($data) {
    return strtoupper($data);
};

$mixedCase = ["Hello", "World"];
$uppercased = array_map($uppercase, $mixedCase);
print_r($uppercased);

Bu değişkenler, bağımsız işlev çağrıları olarak da kullanılabilir:

echo $uppercase("Hello world!"); // HELLO WORLD!

İşlev olarak nesneler #

class SomeClass {
    public function __invoke($param1, $param2) {
        // put your code here
    }
}

$instance = new SomeClass();
$instance('First', 'Second'); // call the __invoke() method

‘__invoke’ yöntemine sahip bir nesne, tam olarak başka herhangi bir işlev gibi kullanılabilir.

‘__invoke’ yöntemi, nesnenin tüm özelliklerine erişebilecek ve herhangi bir yöntemi çağırabilecektir.

Dış değişkenleri kullanma #

‘use’ yapısı, değişkenleri anonim işlevin kapsamına almak için kullanılır:

$divisor = 2332;
$myfunction = function($number) use ($divisor) {
    return $number / $divisor;
};

echo $myfunction(81620); //Outputs 35

Değişkenler ayrıca referans yoluyla içe aktarılabilir:

$collection = [];

$additem = function($item) use (&$collection) {
    $collection[] = $item;
};

$additem(1);
$additem(2);

//$collection is now [1,2]

Yerleşik işlevleri geri arama olarak kullanma #

Argüman olarak ‘çağrılabilir’ alan işlevlerde, yerleşik PHP işlevine sahip bir dize de koyabilirsiniz. Dizideki tüm dizelerden baştaki ve sondaki boşlukları kaldırmak için “array_map” parametresi olarak “trim"in kullanılması yaygındır.

$arr = ['   one  ', 'two   ', '   three'];
var_dump(array_map('trim', $arr));

// array(3) {
//   [0] =>
//   string(3) "one"
//   [1] =>
//   string(3) "two"
//   [2] =>
//   string(5) "three"
// }

Anonim işlev #

Anonim bir işlev, adı olmayan bir işlevdir.

// Anonymous function
function() {
    return "Hello World!";
};

PHP’de anonim bir işlev bir ifade gibi ele alınır ve bu nedenle noktalı virgül ; ile sonlandırılmalıdır.

Anonim bir işlev, bir değişkene atanmış olmalıdır.

// Anonymous function assigned to a variable
$sayHello = function($name) {
    return "Hello $name!";
};

print $sayHello('John'); // Hello John

Veya başka bir işlevin parametresi olarak iletilmelidir.

$users = [
    ['name' => 'Alice', 'age' => 20], 
    ['name' => 'Bobby', 'age' => 22], 
    ['name' => 'Carol', 'age' => 17]
];

// Map function applying anonymous function
$userName = array_map(function($user) {
    return $user['name'];
}, $users);

print_r($usersName); // ['Alice', 'Bobby', 'Carol']

Veya başka bir işlevden geri döndürüldü.

Kendi kendine çalışan anonim işlevler:

// For PHP 7.x
(function () {
    echo "Hello world!";
})();

// For PHP 5.x
call_user_func(function () {
    echo "Hello world!";
});

Kendi kendini yürüten anonim işlevlere bir argüman iletmek:

// For PHP 7.x
(function ($name) {
    echo "Hello $name!";
})('John');

// For PHP 5.x
call_user_func(function ($name) {
    echo "Hello $name!";
}, 'John');

Saf işlevler #

Saf işlev, aynı girdi verildiğinde her zaman aynı çıktıyı döndüren ve yan etkisi olmayan bir işlevdir.

// This is a pure function
function add($a, $b) {
    return $a + $b;
}

Bazı yan etkiler dosya sistemini değiştirme, veritabanlarıyla etkileşim, ekrana yazdırma şeklindedir.

// This is an impure function
function add($a, $b) {
    echo "Adding...";
    return $a + $b;
}

PHP’deki yaygın işlevsel yöntemler #

Haritalama #

Bir dizinin tüm öğelerine bir işlev uygulamak:

array_map('strtoupper', $array);

Geri aramanın ilk geldiği listenin tek yönteminin bu olduğunu unutmayın.

Küçültme (veya katlama) #

Bir diziyi tek bir değere düşürme:

$sum = array_reduce($numbers, function ($carry, $number) {
   return $carry + $number;
});

Filtreleme #

Yalnızca geri aramanın “true” döndürdüğü dizi öğelerini döndürür:

$onlyEven = array_filter($numbers, function ($number) {
    return ($number % 2) === 0;
});

Dürbün #

PHP’de anonim bir işlevin diğer PHP işlevleri gibi kendi kapsamı vardır.

JavaScript’te anonim bir işlev, kapsam dışındaki bir değişkene erişebilir. Ancak PHP’de buna izin verilmez.

$name = 'John';

// Anonymous function trying access outside scope
$sayHello = function() {
    return "Hello $name!";
}

print $sayHello('John'); // Hello !
// With notices active, there is also an Undefined variable $name notice

Bir geri arama işlevini parametre olarak iletmek #

Kullanıcı tanımlı geri arama işlevlerini parametre olarak kabul eden birkaç PHP işlevi vardır, örneğin: call_user_func(), usort() ve [array_map()](https://secure.php.net/manual /en/function.array-map.php).

Kullanıcı tanımlı geri arama işlevinin nerede tanımlandığına bağlı olarak, bunları iletmenin farklı yolları vardır:

Prosedürel stil: #

function square($number)
{
    return $number * $number;
}

$initial_array = [1, 2, 3, 4, 5];
$final_array = array_map('square', $initial_array);
var_dump($final_array); // prints the new array with 1, 4, 9, 16, 25

Nesne Yönelimli stil: #

class SquareHolder
{
    function square($number)
    {
        return $number * $number;
    }
}

$squaredHolder = new SquareHolder();
$initial_array = [1, 2, 3, 4, 5];
$final_array = array_map([$squaredHolder, 'square'], $initial_array);

Statik bir yöntem kullanan Nesne Yönelimli stil: #

class StaticSquareHolder
{
    public static function square($number)
    {
        return $number * $number;
    }
}

$initial_array = [1, 2, 3, 4, 5];
$final_array = array_map(['StaticSquareHolder', 'square'], $initial_array);
// or:
$final_array = array_map('StaticSquareHolder::square', $initial_array); // for PHP >= 5.2.3

var_dump($final_array); // prints the new array with 1, 4, 9, 16, 25
