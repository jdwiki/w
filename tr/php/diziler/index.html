Dizi, rastgele sayıda değeri tek bir değerde saklayan bir veri yapısıdır. PHP’deki bir dizi aslında sıralı bir haritadır; burada map, değerleri anahtarlarla ilişkilendiren bir türdür.

Sözdizimi #

  • $dizi = dizi(‘Değer1’, ‘Değer2’, ‘Değer3’); // Tuşlar varsayılan olarak 0, 1, 2, …,
  • $dizi = dizi(‘Değer1’, ‘Değer2’, ); // İsteğe bağlı sondaki virgül
  • $dizi = dizi(‘anahtar1’ => ‘Değer1’, ‘anahtar2’ => ‘Değer2’, ); // Açık anahtarlar
  • $dizi = dizi(‘anahtar1’ => ‘Değer1’, ‘Değer2’, ); // Dizi ( [‘key1’] => Değer1 1 => ‘Değer2’)
  • $dizi = [‘anahtar1’ => ‘Değer1’, ‘anahtar2’ => ‘Değer2’, ]; // PHP 5.4+ steno
  • $dizi[] = ‘ValueX’; // Dizinin sonuna ‘ValueX’ ekleyin
  • $dizi[‘keyX’] = ‘ValueX’; // ‘keyX’ anahtarına ‘valueX’ atayın
  • $dizi += [‘keyX’ => ‘valueX’, ‘keyY’ => ‘valueY’]; // Mevcut bir diziye eleman ekleme/üzerine yazma

Parametreler #

parametre Detay
Anahtar Anahtar, bir dizinin benzersiz tanımlayıcısı ve dizinidir. Bir “dize” veya “tamsayı” olabilir. Bu nedenle, geçerli anahtarlar 'foo', '5', 10, 'a2b', ... olacaktır.
Değer Her bir “anahtar” için karşılık gelen bir değer vardır (aksi halde “boş” ve erişim üzerine bir bildirim gönderilir). Değerin giriş türü üzerinde herhangi bir kısıtlaması yoktur.

Dizi Başlatma #

Bir dizi boş olarak başlatılabilir:

// An empty array
$foo = array();

// Shorthand notation available since PHP 5.4
$foo = [];

Bir dizi başlatılabilir ve değerlerle önceden ayarlanabilir:

// Creates a simple array with three strings
$fruit = array('apples', 'pears', 'oranges');

// Shorthand notation available since PHP 5.4
$fruit = ['apples', 'pears', 'oranges'];

Bir dizi, özel dizinlerle de başlatılabilir (ilişkisel dizi olarak da adlandırılır):

// A simple associative array
$fruit = array(
   'first'  => 'apples',
   'second' => 'pears', 
   'third'  => 'oranges'
);

// Key and value can also be set as follows
$fruit['first'] = 'apples';

// Shorthand notation available since PHP 5.4
$fruit = [
    'first'  => 'apples', 
    'second' => 'pears', 
    'third'  => 'oranges'
];

Değişken daha önce kullanılmadıysa, PHP onu otomatik olarak oluşturacaktır. Uygun olsa da, bu kodun okunmasını zorlaştırabilir:

$foo[] = 1;     // Array( [0] => 1 )
$bar[][] = 2;   // Array( [0] => Array( [0] => 2 ) )

Dizin genellikle kaldığınız yerden devam eder. PHP, sayısal dizeleri tamsayı olarak kullanmaya çalışacaktır:

$foo = [2 => 'apple', 'melon'];  // Array( [2] => apple, [3] => melon )
$foo = ['2' => 'apple', 'melon']; // same as above
$foo = [2 => 'apple', 'this is index 3 temporarily', '3' => 'melon']; // same as above! The last entry will overwrite the second!

Sabit boyutlu bir dizi başlatmak için SplFixedArray kullanabilirsiniz:

$array = new SplFixedArray(3);

$array[0] = 1;
$array[1] = 2;
$array[2] = 3;
$array[3] = 4; // RuntimeException

// Increase the size of the array to 10
$array->setSize(10);

Not: “SplFixedArray” kullanılarak oluşturulan bir dizi, büyük veri kümeleri için azaltılmış bellek ayak izine sahiptir, ancak anahtarlar tamsayı olmalıdır.


Dinamik boyutta ancak “n” boş olmayan öğeleri (örneğin bir yer tutucu) olan bir diziyi başlatmak için aşağıdaki gibi bir döngü kullanabilirsiniz:

$myArray = array();
$sizeOfMyArray = 5;
$fill = 'placeholder';

for ($i = 0; $i < $sizeOfMyArray; $i++) {
    $myArray[] = $fill;
}

// print_r($myArray); results in the following:
// Array ( [0] => placeholder [1] => placeholder [2] => placeholder [3] => placeholder [4] => placeholder )

Tüm yer tutucularınız aynıysa, array_fill() işlevini kullanarak da oluşturabilirsiniz:

PHP 5.5 ve sonraki sürümlerde, array_column() öğesini array_search() ile birlikte kullanabilirsiniz.

Bu özellikle ilişkisel dizide bir değer olup olmadığını kontrol etmek için kullanışlıdır:

$userdb = [
    [
        "uid" => '100',
        "name" => 'Sandra Shush',
        "url" => 'urlof100',
    ],
    [
        "uid" => '5465',
        "name" => 'Stefanie Mcmohn',
        "pic_square" => 'urlof100',
    ],
    [
        "uid" => '40489',
        "name" => 'Michael',
        "pic_square" => 'urlof40489',
    ]
];

$key = array_search(40489, array_column($userdb, 'uid'));

ArrayAccess ve Yineleyici Arayüzleri #

Bir başka kullanışlı özellik de özel nesne koleksiyonlarınıza PHP’de diziler olarak erişmektir. Bunu desteklemek için PHP (>=5.0.0) çekirdeğinde iki arayüz mevcuttur: ArrayAccess ve Iterator. İlki, özel nesnelerinize dizi olarak erişmenizi sağlar.

Dizi Erişimi

Bir kullanıcı sınıfımız ve tüm kullanıcıları depolayan bir veritabanı tablomuz olduğunu varsayalım. Aşağıdakileri yapacak bir “UserCollection” sınıfı oluşturmak istiyoruz:

  1. belirli bir kullanıcıya benzersiz kullanıcı adı tanımlayıcısı ile hitap etmemize izin verin
  2. kullanıcı koleksiyonumuzda temel (tüm CRUD değil, en azından Oluşturma, Alma ve Silme) işlemleri gerçekleştirin

Aşağıdaki kaynağı göz önünde bulundurun (bundan böyle 5.4 sürümünden beri kullanılabilen kısa dizi oluşturma sözdizimi “[]” kullanıyoruz):

class UserCollection implements ArrayAccess {
    protected $_conn;
    
    protected $_requiredParams = ['username','password','email'];
    
    public function __construct() {
        $config = new Configuration();

        $connectionParams = [
            //your connection to the database
        ];
        
        $this->_conn = DriverManager::getConnection($connectionParams, $config);
    }
    
    protected function _getByUsername($username) {
        $ret = $this->_conn->executeQuery('SELECT * FROM `User` WHERE `username` IN (?)',
            [$username]
        )->fetch();
        
        return $ret;
    }
    
    // START of methods required by ArrayAccess interface
    public function offsetExists($offset) {
        return (bool) $this->_getByUsername($offset);
    }

    public function offsetGet($offset) {
        return $this->_getByUsername($offset);
    }

    public function offsetSet($offset, $value) {
        if (!is_array($value)) {
            throw new \Exception('value must be an Array');
        }

        $passed = array_intersect(array_values($this->_requiredParams), array_keys($value));
        if (count($passed) < count($this->_requiredParams)) {
            throw new \Exception('value must contain at least the following params: ' . implode(',', $this->_requiredParams));
        }
        $this->_conn->insert('User', $value);
    }

    public function offsetUnset($offset) {
        if (!is_string($offset)) {
            throw new \Exception('value must be the username to delete');
        }
        if (!$this->offsetGet($offset)) {
            throw new \Exception('user not found');
        }
        $this->_conn->delete('User', ['username' => $offset]);
    }
    // END of methods required by ArrayAccess interface
}

o zaman şunları yapabiliriz:

$users = new UserCollection();

var_dump(empty($users['testuser']),isset($users['testuser']));
$users['testuser'] = ['username' => 'testuser', 
                      'password' => 'testpassword',
                      'email'    => 'test@test.com'];
var_dump(empty($users['testuser']), isset($users['testuser']), $users['testuser']);
unset($users['testuser']);
var_dump(empty($users['testuser']), isset($users['testuser']));

kodu başlatmadan önce “testuser” olmadığını varsayarak aşağıdaki çıktıyı verir:

bool(true)
bool(false)
bool(false)
bool(true)
array(17) {
  ["username"]=>
  string(8) "testuser"
  ["password"]=>
  string(12) "testpassword"
  ["email"]=>
  string(13) "test@test.com"
}
bool(true)
bool(false)

ÖNEMLİ: ‘array_key_exists’ işleviyle bir anahtarın varlığını kontrol ettiğinizde ‘offsetExists’ çağrılmaz. Bu nedenle, aşağıdaki kod iki kez “false” çıktısı verecektir:

var_dump(array_key_exists('testuser', $users));
$users['testuser'] = ['username' => 'testuser', 
                      'password' => 'testpassword',
                      'email'    => 'test@test.com'];
var_dump(array_key_exists('testuser', $users));

Yineleyici

‘Foreach’ ve ‘while’ ile yinelemeye izin vermek için ‘Iterator’ arayüzünden birkaç fonksiyonla sınıfımızı yukarıdan genişletelim.

İlk olarak, mevcut yineleyici indeksimizi tutan bir özellik eklememiz gerekiyor, onu sınıf özelliklerine $_position olarak ekleyelim:

// iterator current position, required by Iterator interface methods
protected $_position = 1;

İkinci olarak, sınıfımız tarafından uygulanmakta olan arayüzler listesine İteratör arayüzünü ekleyelim:

class UserCollection implements ArrayAccess, Iterator {

daha sonra arayüz işlevlerinin gerektirdiğini ekleyin:

// START of methods required by Iterator interface
public function current () {
    return $this->_getById($this->_position);
}
public function key () {
    return $this->_position;
}
public function next () {
    $this->_position++;
}
public function rewind () {
    $this->_position = 1;
}
public function valid () {
    return null !== $this->_getById($this->_position);
}
// END of methods required by Iterator interface

Bu örneğin mükemmel olmadığını unutmayın, çünkü veritabanındaki kimlikler sıralı olmayabilir, ancak bu sadece size ana fikri vermek için yazılmıştır: ‘ArrayAccess’ ve ‘Iterator’ uygulayarak nesne koleksiyonlarınızı mümkün olan herhangi bir şekilde ele alabilirsiniz. arayüzler:

class UserCollection implements ArrayAccess, Iterator {
    // iterator current position, required by Iterator interface methods
    protected $_position = 1;
    
    // <add the old methods from the last code snippet here>
    
    // START of methods required by Iterator interface
    public function current () {
        return $this->_getById($this->_position);
    }
    public function key () {
        return $this->_position;
    }
    public function next () {
        $this->_position++;
    }
    public function rewind () {
        $this->_position = 1;
    }
    public function valid () {
        return null !== $this->_getById($this->_position);
    }
    // END of methods required by Iterator interface
}
