Bir diziyi filtreleme #

Bir diziden değerleri filtrelemek ve filtre koşulunu sağlayan tüm değerleri içeren yeni bir dizi elde etmek için ‘array_filter’ işlevini kullanabilirsiniz.

Boş olmayan değerleri filtreleme #

En basit filtreleme durumu, tüm “boş” değerleri kaldırmaktır:

$my_array = [1,0,2,null,3,'',4,[],5,6,7,8];
$non_empties = array_filter($my_array); // $non_empties will contain [1,2,3,4,5,6,7,8];

Geri aramaya göre filtreleme #

Bu sefer kendi filtreleme kuralımızı tanımlıyoruz. Yalnızca çift sayıları almak istediğimizi varsayalım:

$my_array = [1,2,3,4,5,6,7,8];

$even_numbers = array_filter($my_array, function($number) {
    return $number % 2 === 0;   
});

“array_filter” işlevi, filtrelenecek diziyi ilk argümanı olarak ve filtre yüklemini ikinci argümanı olarak tanımlayan bir geri çağırmayı alır.

Dizine göre filtreleme #

‘array_filter’ işlevine, geri çağırmaya hangi değerlerin iletileceğini ayarlamaya izin veren üçüncü bir parametre sağlanabilir. Bu parametre, “ARRAY_FILTER_USE_KEY” veya “ARRAY_FILTER_USE_BOTH” olarak ayarlanabilir; bu, geri aramanın dizideki her öğenin değeri yerine anahtarı almasına veya argümanları olarak hem değer hem de anahtar almasına neden olur. Örneğin, değerler yerine indekslerle uğraşmak istiyorsanız:

$numbers = [16,3,5,8,1,4,6];

$even_indexed_numbers = array_filter($numbers, function($index) {
    return $index % 2 === 0;
}, ARRAY_FILTER_USE_KEY);
son sürüm

Filtrelenmiş dizideki dizinler #

“array_filter” öğesinin orijinal dizi anahtarlarını koruduğunu unutmayın. Yaygın bir hata, filtrelenmiş dizi üzerinde “for” döngüsünü kullanmayı denemek olacaktır:

<?php

$my_array = [1,0,2,null,3,'',4,[],5,6,7,8];
$filtered = array_filter($my_array); 

error_reporting(E_ALL); // show all errors and notices

// innocently looking "for" loop
for ($i = 0; $i < count($filtered); $i++) {
   print $filtered[$i];
}

/*
Output:
1
Notice: Undefined offset: 1
2
Notice: Undefined offset: 3
3
Notice: Undefined offset: 5
4
Notice: Undefined offset: 7
*/

Bunun nedeni, 1 (“0” vardı), 3 (“boş”), 5 (boş dize “”") ve 7 (boş dizi “[]”) konumlarındaki değerlerin karşılık gelenleriyle birlikte kaldırılmasıdır. dizin anahtarları.

Dizine alınmış bir dizideki bir filtrenin sonucu arasında döngü yapmanız gerekiyorsa, doğru dizinlere sahip yeni bir dizi oluşturmak için önce “array_filter” sonucunda “array_values” öğesini çağırmalısınız:

$my_array = [1,0,2,null,3,'',4,[],5,6,7,8];
$filtered = array_filter($my_array); 
$iterable = array_values($filtered);

error_reporting(E_ALL); // show all errors and notices

for ($i = 0; $i < count($iterable); $i++) {
   print $iterable[$i];
}

// No warnings!

Bir diziden öğeleri kaldırma #

Bir dizinin içindeki bir öğeyi kaldırmak için, ör. 1 indeksli eleman.

$fruit = array("bananas", "apples", "peaches");
unset($fruit[1]);

Bu, elmaları listeden kaldıracaktır, ancak “unset” öğesinin kalan öğelerin dizinlerini değiştirmediğine dikkat edin. Yani ‘$fruit’ şimdi ‘0’ ve ‘2’ dizinlerini içeriyor.

İlişkisel dizi için şu şekilde kaldırabilirsiniz:

$fruit = array('banana', 'one'=>'apple', 'peaches');

print_r($fruit);
/*
    Array
    (
        [0] => banana
        [one] => apple
        [1] => peaches
    )
*/

unset($fruit['one']);

Şimdi $meyve

print_r($fruit);

/*
Array
(
    [0] => banana
    [1] => peaches
)
*/

Dikkat

unset($fruit);

değişkeni kaldırır ve böylece tüm diziyi kaldırır, yani öğelerinin hiçbirine artık erişilemez.

Terminal elemanlarının çıkarılması #

array_shift() - Bir öğeyi dizinin başından kaydırır.

Örnek:

  $fruit = array("bananas", "apples", "peaches");
  array_shift($fruit);
  print_r($fruit);

Çıktı:

 Array
(
    [0] => apples
    [1] => peaches
)

array_pop() - Öğeyi dizinin sonundan çıkar.

Örnek:

  $fruit = array("bananas", "apples", "peaches");
  array_pop($fruit);
  print_r($fruit);

Çıktı:

 Array
(
    [0] => bananas
    [1] => apples
)

Dizi Sıralama #

PHP’de diziler için birkaç sıralama işlevi vardır:

çeşit() #

Bir diziyi değere göre artan düzende sıralayın.

$fruits = ['Zitrone', 'Orange', 'Banane', 'Apfel'];
sort($fruits);
print_r($fruits);

sonuçlanır

Array
(
    [0] => Apfel
    [1] => Banane
    [2] => Orange
    [3] => Zitrone
)

çeşit() #

Bir diziyi değere göre azalan düzende sıralayın.

$fruits = ['Zitrone', 'Orange', 'Banane', 'Apfel'];
rsort($fruits);
print_r($fruits);

sonuçlanır

Array
(
    [0] => Zitrone
    [1] => Orange
    [2] => Banane
    [3] => Apfel
)

asort() #

Bir diziyi değere göre artan düzende sıralayın ve indisleri koruyun.

$fruits = [1 => 'lemon', 2 => 'orange',  3 => 'banana', 4 => 'apple'];
asort($fruits);
print_r($fruits);

sonuçlanır

Array
(
    [4] => apple
    [3] => banana
    [1] => lemon
    [2] => orange
)

arsort() #

Bir diziyi değere göre azalan düzende sıralayın ve indisleri koruyun.

$fruits = [1 => 'lemon', 2 => 'orange',  3 => 'banana', 4 => 'apple'];
arsort($fruits);
print_r($fruits);

sonuçlanır

Array
(
    [2] => orange
    [1] => lemon
    [3] => banana
    [4] => apple
)

ksort() #

Bir diziyi tuşa göre artan düzende sıralama

$fruits = ['d'=>'lemon', 'a'=>'orange', 'b'=>'banana', 'c'=>'apple'];
ksort($fruits);
print_r($fruits);

sonuçlanır

Array
(
    [a] => orange
    [b] => banana
    [c] => apple
    [d] => lemon
)

krssort() #

Bir diziyi tuşa göre azalan düzende sıralayın.

$fruits = ['d'=>'lemon', 'a'=>'orange', 'b'=>'banana', 'c'=>'apple'];
krsort($fruits);
print_r($fruits);

sonuçlanır

Array
(
    [d] => lemon
    [c] => apple
    [b] => banana
    [a] => orange
)

natsort() #

Bir diziyi bir insanın yapacağı şekilde sıralayın (doğal düzen).

$files = ['File8.stack', 'file77.stack', 'file7.stack', 'file13.stack', 'File2.stack'];
natsort($files);
print_r($files);

sonuçlanır

Array
(
    [4] => File2.stack
    [0] => File8.stack
    [2] => file7.stack
    [3] => file13.stack
    [1] => file77.stack
)

natcasesort() #

Bir diziyi bir insanın yapacağı şekilde (doğal düzen) sıralayın, ancak büyük/küçük harf yoğun

$files = ['File8.stack', 'file77.stack', 'file7.stack', 'file13.stack', 'File2.stack'];
natcasesort($files);
print_r($files);

sonuçlanır

Array
(
    [4] => File2.stack
    [2] => file7.stack
    [0] => File8.stack
    [3] => file13.stack
    [1] => file77.stack
)

Karıştır() #

Bir diziyi karıştırır (rastgele sıralanır).

$array = ['aa', 'bb', 'cc'];
shuffle($array);
print_r($array);

Açıklamada yazıldığı gibi rastgeledir, bu yüzden burada neyle sonuçlanabileceğine dair sadece bir örnek

Array
(
    [0] => cc
    [1] => bb
    [2] => aa
)

usort() #

Kullanıcı tanımlı karşılaştırma işleviyle bir diziyi sıralayın.

function compare($a, $b)
{
    if ($a == $b) {
        return 0;
    }
    return ($a < $b) ? -1 : 1;
}

$array = [3, 2, 5, 6, 1];
usort($array, 'compare');
print_r($array);

sonuçlanır

Array
(
    [0] => 1
    [1] => 2
    [2] => 3
    [3] => 5
    [4] => 6
)

usort() #

Kullanıcı tanımlı karşılaştırma işleviyle bir diziyi sıralayın ve anahtarları koruyun.

function compare($a, $b)
{
    if ($a == $b) {
        return 0;
    }
    return ($a < $b) ? -1 : 1;
}

$array = ['a' => 1, 'b' => -3, 'c' => 5, 'd' => 3, 'e' => -5];
uasort($array, 'compare');
print_r($array);

sonuçlanır

Array
(
    [e] => -5
    [b] => -3
    [a] => 1
    [d] => 3
    [c] => 5
)

uksort() #

Kullanıcı tanımlı karşılaştırma işleviyle bir diziyi tuşlara göre sıralayın.

function compare($a, $b)
{
    if ($a == $b) {
        return 0;
    }
    return ($a < $b) ? -1 : 1;
}

$array = ['ee' => 1, 'g' => -3, '4' => 5, 'k' => 3, 'oo' => -5];

uksort($array, 'compare');
print_r($array);

Dizinin başlangıcına eleman ekleme #

Bazen bir dizinin başlangıcına dizi içindeki mevcut öğelerden (sıra) hiçbirini değiştirmeden bir öğe eklemek istersiniz. Bu durumda, array_unshift() kullanabilirsiniz.

array_unshift(), geçirilen öğeleri dizinin önüne ekler. Öğe listesinin bir bütün olarak başa eklendiğine dikkat edin, böylece başa eklenen öğeler aynı sırada kalır. Tüm sayısal dizi tuşları değişmez anahtarlar olmayacakken sıfırdan saymaya başlayacak şekilde değiştirilecektir dokunulmak.

array_unshift() için PHP belgelerinden alınmıştır.

Bunu başarmak istiyorsanız, yapmanız gereken tek şey şunlar:

$myArray = array(1, 2, 3);

array_unshift($myArray, 4);

Bu şimdi dizinizdeki ilk öğe olarak ‘4’ ekleyecektir. Bunu şu şekilde doğrulayabilirsiniz:

print_r($myArray);

Bu, şu sırayla bir dizi döndürür: 4, 1, 2, 3.

array_unshift diziyi yeni öğe olarak anahtar-değer çiftlerini sıfırlamaya zorladığı için, aşağıdaki girişlerin n+1 anahtarlarına sahip olmasına izin verdiğinden, yeni bir dizi oluşturmak ve mevcut diziyi yeni diziye eklemek daha akıllıdır oluşturulan dizi.

Örnek:

$myArray = array('apples', 'bananas', 'pears');
$myElement = array('oranges');
$joinedArray = $myElement;

foreach ($myArray as $i) {
  $joinedArray[] = $i;
}

Çıktı ($joinedArray):

Array ( [0] => oranges [1] => apples [2] => bananas [3] => pears )

Yalnızca bazı dizi anahtarlarını beyaz listeye alın #

Dizilerinizde yalnızca belirli anahtarlara izin vermek istediğinizde, özellikle dizi istek parametrelerinden geldiğinde, ‘array_intersect_key’ ile ‘array_flip’ birlikte kullanabilirsiniz.

$parameters = ['foo' => 'bar', 'bar' => 'baz', 'boo' => 'bam'];
$allowedKeys = ['foo', 'bar'];
$filteredParameters = array_intersect_key($parameters, array_flip($allowedKeys));

// $filteredParameters contains ['foo' => 'bar', 'bar' => 'baz]

“parameters” değişkeni izin verilen herhangi bir anahtar içermiyorsa, “filteredParameters” değişkeni boş bir diziden oluşacaktır.

PHP 5.6’dan beri, ARRAY_FILTER_USE_KEY bayrağını üçüncü parametre olarak geçirerek bu görev için array_filter öğesini de kullanabilirsiniz:

$parameters  = ['foo' => 1, 'hello' => 'world'];
$allowedKeys = ['foo', 'bar'];
$filteredParameters = array_filter(
    $parameters,
    function ($key) use ($allowedKeys) {
        return in_array($key, $allowedKeys);
    },
    ARRAY_FILTER_USE_KEY
);

“array_filter” kullanmak, anahtara karşı isteğe bağlı bir test gerçekleştirme esnekliği sağlar, örn. “$allowedKeys”, düz dizeler yerine normal ifade kalıpları içerebilir. Ayrıca, “array_flip()” ile birlikte “array_intersect_key()” yerine kodun amacını daha açık bir şekilde belirtir.

Anahtarlarla değerleri değiştirin #

‘array_flip’ işlevi, tüm anahtarları öğeleriyle değiştirir.

$colors = array(
    'one' => 'red',
    'two' => 'blue',
    'three' => 'yellow',
);

array_flip($colors); //will output

array(
    'red' => 'one',
    'blue' => 'two',
    'yellow' => 'three'
)

İki diziyi tek bir dizide birleştir #

$a1 = array("red","green");
$a2 = array("blue","yellow");
print_r(array_merge($a1,$a2));

/*
    Array ( [0] => red [1] => green [2] => blue [3] => yellow )
*/

İlişkisel dizi:

$a1=array("a"=>"red","b"=>"green");
$a2=array("c"=>"blue","b"=>"yellow");
print_r(array_merge($a1,$a2));
/*
    Array ( [a] => red [b] => yellow [c] => blue )
*/
  1. Bir veya daha fazla dizinin öğelerini, birinin değerleri öncekinin sonuna eklenecek şekilde birleştirir. Elde edilen diziyi döndürür.
  2. Giriş dizileri aynı dize anahtarlarına sahipse, o anahtarın sonraki değeri öncekinin üzerine yazılır. Bununla birlikte, diziler sayısal tuşlar içeriyorsa, sonraki değer orijinal değerin üzerine yazılmaz, ancak eklenir.
  3. Sayısal tuşlu giriş dizisindeki değerler, sonuç dizisinde sıfırdan başlayarak artan tuşlarla yeniden numaralandırılacaktır.
