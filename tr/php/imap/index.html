Posta kutusuna bağlanma #

Bir IMAP hesabıyla herhangi bir şey yapmak için önce ona bağlanmanız gerekir. Bunu yapmak için bazı gerekli parametreleri belirtmeniz gerekir:

  • Posta sunucusunun sunucu adı veya IP adresi
  • Bağlanmak istediğiniz bağlantı noktası
    • IMAP is 143 or 993 (secure)
    • POP is 110 or 995 (secure)
    • SMTP is 25 or 465 (secure)
    • NNTP is 119 or 563 (secure)
  • Bağlantı bayrakları (aşağıya bakın)
bayrak Açıklama Seçenekler Varsayılan
/servis=servis Hangi hizmet kullanılacak imap, pop3, nntp, smtp imap
/kullanıcı=kullanıcı sunucuda oturum açmak için uzak kullanıcı adı
/authuser=kullanıcı uzaktan kimlik doğrulama kullanıcısı; belirtilmişse bu, parolası kullanılan kullanıcı adıdır (örn. yönetici)
/anonim anonim kullanıcı olarak uzaktan erişim
/hata ayıklama uygulamanın hata ayıklama günlüğüne protokol telemetrisini kaydetme engelli
/güvenli ağ üzerinden düz metin parolası iletmeyin
/norsh önceden doğrulanmış bir IMAP oturumu oluşturmak için rsh veya ssh kullanmayın
/ssl oturumu şifrelemek için Güvenli Yuva Katmanını kullanın
/validate-cert TLS/SSL sunucusundan sertifikalar etkin
/novalidate-cert sunucu kendinden imzalı sertifikalar kullanıyorsa gerekli olan TLS/SSL sunucusundan gelen sertifikaları doğrulamayın. DİKKATLİ KULLANIN engelli
/tls oturumu şifrelemek için start-TLS kullanımını zorlamak ve onu desteklemeyen sunuculara bağlantıyı reddetmek
/notls onu destekleyen sunucularda bile oturumu şifrelemek için start-TLS yapmayın
/salt okunur salt okunur posta kutusu açma isteği (yalnızca IMAP; NNTP’de yoksayılır ve SMTP ve POP3 ile ilgili bir hata)

Bağlantı dizginiz şöyle görünecek:

{imap.example.com:993/imap/tls/secure}

Bağlantı dizenizdeki karakterlerden herhangi biri ASCII değilse, bunun [utf7_encode($string)](https://php.net/manual/en/function.imap-utf7-encode.php ile kodlanması gerektiğini unutmayın. ).

Posta kutusuna bağlanmak için, bir akışa işaret eden bir kaynak değeri döndüren imap_open komutunu kullanırız:

<?php
$mailbox = imap_open("{imap.example.com:993/imap/tls/secure}", "username", "password");
if ($mailbox === false) {
    echo "Failed to connect to server";
}

IMAP uzantısını yükleyin #

PHP’de IMAP işlevlerini kullanmak için IMAP uzantısını yüklemeniz gerekir:

PHP5 ile Debian/Ubuntu

sudo apt-get install php5-imap
sudo php5enmod imap

PHP7 ile Debian/Ubuntu

sudo apt-get install php7.0-imap

YUM tabanlı dağıtım

sudo yum install php-imap

php5.6 ile Mac OS X

brew reinstall php56 --with-imap

Posta kutusundaki tüm klasörleri listele #

Posta kutunuza bağlandıktan sonra, içine bir göz atmak isteyeceksiniz. İlk kullanışlı komut imap_list. İlk parametre ‘imap_open’dan edindiğiniz kaynaktır, ikincisi posta kutusu dizginiz ve üçüncüsü bir bulanık arama dizgisidir ("*" herhangi bir kalıpla eşleştirmek için kullanılır).

$folders = imap_list($mailbox, "{imap.example.com:993/imap/tls/secure}", "*");
if ($folders === false) {
    echo "Failed to list folders in mailbox";
} else {
    print_r($folders);
}

Çıktı buna benzer görünmelidir

Array
(
    [0] => {imap.example.com:993/imap/tls/secure}INBOX
    [1] => {imap.example.com:993/imap/tls/secure}INBOX.Sent
    [2] => {imap.example.com:993/imap/tls/secure}INBOX.Drafts
    [3] => {imap.example.com:993/imap/tls/secure}INBOX.Junk
    [4] => {imap.example.com:993/imap/tls/secure}INBOX.Trash
)

Bu sonuçları şu şekilde filtrelemek için üçüncü parametreyi kullanabilirsiniz:

$folders = imap_list($mailbox, "{imap.example.com:993/imap/tls/secure}", "*.Sent");

Ve şimdi sonuç sadece adında .Sent olan girişleri içeriyor:

Array
(
    [0] => {imap.example.com:993/imap/tls/secure}INBOX.Sent
)

Not: Bulanık bir arama olarak * kullanılması tüm eşleşmeleri yinelemeli olarak döndürür. % kullanırsanız, yalnızca belirtilen geçerli klasördeki eşleşmeleri döndürür.

Posta kutusundaki mesajları bulma #

imap_headers kullanarak bir posta kutusundaki tüm iletilerin listesini döndürebilirsiniz.

<?php
$headers = imap_headers($mailbox);

Sonuç, aşağıdaki desene sahip bir dizi dizidir:

[FLAG] [MESSAGE-ID])[DD-MM-YYY] [FROM ADDRESS] [SUBJECT TRUNCATED TO 25 CHAR] ([SIZE] chars)

İşte her satırın nasıl görünebileceğinin bir örneği:

A     1)19-Aug-2016 someone@example.com Message Subject (1728 chars)
D     2)19-Aug-2016 someone@example.com RE: Message Subject (22840 chars)
U     3)19-Aug-2016 someone@example.com RE: RE: Message Subject (1876 chars)
N     4)19-Aug-2016 someone@example.com RE: RE: RE: Message Subje (1741 chars)
Sembol bayrak Anlamı
bir Yanıtlandı Mesaja cevap verildi
D Silindi Mesaj silindi (ancak kaldırılmadı)
F İşaretli İleti işaretlendi/dikkat için işaretlendi
N Yeni Mesaj yeni ve görülmedi
sağ Son Mesaj yeni ve görüldü
U okunmadı Mesaj okunmadı
X Taslak Mesaj bir taslaktır

Bu çağrının yürütülmesinin oldukça uzun sürebileceğini ve çok büyük bir liste döndürebileceğini unutmayın.

Bir alternatif, tek tek mesajları ihtiyaç duyduğunuzda yüklemektir. E-postalarınızın her birine 1’den (en eski) imap_num_msg($mailbox) değerine kadar bir kimlik atanır ).
