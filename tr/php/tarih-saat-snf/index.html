getZaman damgası #

“getTimeStemp”, bir tarih saat nesnesinin bir unix temsilidir.

$date = new DateTime();
echo $date->getTimestamp();

bu, 1 Ocak 1970 Perşembe, 00:00:00 UTC’den bu yana geçen saniyelerin bir tamsayı göstergesini koyacaktır.

tarih ayarla #

“setDate”, DateTime nesnesindeki tarihi ayarlar.

$date = new DateTime();
$date->setDate(2016, 7, 25);

bu örnek tarihi yirmi beş Temmuz 2015 olarak ayarlar, aşağıdaki sonucu verir:

2016-07-25 17:52:15.819442

Tarih Aralıkları Ekleme veya Çıkarma #

Bir DateTime nesnesinde bir aralık eklemek veya çıkarmak için DateInterval sınıfını kullanabiliriz.

7 günlük bir aralık eklediğimiz ve ekrana bir mesaj yazdırdığımız aşağıdaki örneğe bakın:

$now = new DateTime();// empty argument returns the current date
$interval = new DateInterval('P7D');//this objet represents a 7 days interval
$lastDay = $now->add($interval); //this will return a DateTime object
$formatedLastDay = $lastDay->format('Y-m-d');//this method format the DateTime object and returns a String
echo "Samara says: Seven Days. You'll be happy on $formatedLastDay.";

Bu çıktı (1 Ağustos 2016’da çalışıyor):

Samara diyor ki: Yedi Gün. 2016-08-08’de mutlu olacaksınız.

Tarihleri ​​çıkarmak için sub yöntemini benzer şekilde kullanabiliriz.

$now->sub($interval);
echo "Samara says: Seven Days. You were happy last on $formatedLastDay.";

Bu çıktı (1 Ağustos 2016’da çalışıyor):

Samara diyor ki: Yedi Gün. En son 2016-07-25’te mutluydun.

Özel biçimden DateTime oluşturun #

PHP, bir dizi tarih biçimini ayrıştırabilir. Standart olmayan bir formatı ayrıştırmak veya kodunuzun kullanılacak formatı açıkça belirtmesini istiyorsanız, statik DateTime::createFromFormat kullanabilirsiniz. /en/datetime.createfromformat.php) yöntemi:

Nesne yönelimli stil

$format = "Y,m,d";
$time = "2009,2,26";
$date = DateTime::createFromFormat($format, $time);

Prosedür stili

$format = "Y,m,d";
$time = "2009,2,26";
$date = date_create_from_format($format, $time);

DateTimes Yazdırma #

PHP 4+, bir DateTime nesnesini istenen biçimde bir dizeye dönüştüren bir yöntem, biçim sağlar. PHP Kılavuzuna göre, bu nesne yönelimli işlevdir:

public string DateTime::format ( string $format )

date() işlevi bir parametre alır - bir dizge olan bir biçim

Biçim #

Biçim bir dizedir ve biçimi tanımlamak için tek karakterler kullanır:

  • Y: yılın dört haneli gösterimi (örn: 2016)
  • y: yılın iki basamaklı gösterimi (örn: 16)
  • a: ay, sayı olarak (01 - 12)
  • A: ay, üç harf olarak (Ocak, Şubat, Mart vb.)
  • j: başında sıfır olmadan ayın günü (1 - 31)
  • D: haftanın günü, üç harf olarak (Pzt, Sal, Çar, vb.)
  • s: saat (12 saatlik biçim) (01 - 12)
  • H: saat (24 saat formatı) (00 - 23)
  • A: AM veya PM
  • i: dakika, başında sıfırlar (00 - 59)
  • s: saniye, baştaki sıfırlarla (00 ila 59)
  • Listenin tamamı burada bulunabilir

Kullanım #

Bu karakterler, saatleri hemen hemen her formatta görüntülemek için çeşitli kombinasyonlarda kullanılabilir. İşte bazı örnekler:

$date = new DateTime('2000-05-26T13:30:20'); /* Friday, May 26, 2000 at 1:30:20 PM */

$date->format("H:i");
/* Returns 13:30 */

$date->format("H i s");
/* Returns 13 30 20 */

$date->format("h:i:s A");
/* Returns 01:30:20 PM */

$date->format("j/m/Y");
/* Returns 26/05/2000 */

$date->format("D, M j 'y - h:i A");
/* Returns Fri, May 26 '00 - 01:30 PM */

Prosedürel #

Prosedür biçimi benzerdir:

Nesne odaklı #

$date->format($format)

Prosedür Eşdeğeri #

date_format($date, $format)

PHP 5.6’dan önceki Mutable’dan DateTime’ın Değişmez sürümünü oluşturun #

PHP 5.6+ sürümünde “\DateTimeImmutable” oluşturmak için şunu kullanın:

\DateTimeImmutable::createFromMutable($concrete);

PHP 5.6’dan önce şunları kullanabilirsiniz:

\DateTimeImmutable::createFromFormat(\DateTime::ISO8601, $mutable->format(\DateTime::ISO8601), $mutable->getTimezone());
