Sözdizimi #

  • for ($i = 0; $i < sayı($dizi); $i++) { incremental_iteration(); }
  • for ($i = sayım($dizi) - 1; $i >= 0; $i–) { reverse_iteration(); }
  • foreach ($veri olarak $veri) { }
  • foreach ($veri olarak $anahtar => $veri) { }
  • foreach ($veri olarak &$veri) { }

Bir diziyi yineleme yöntemlerinin karşılaştırılması #

Yöntem Avantaj
‘foreach’ Bir diziyi yinelemenin en basit yöntemi.
referansa göre ‘foreach’ Bir dizinin öğelerini yinelemek ve değiştirmek için basit yöntem.
artan indeksli for Dizinin serbest bir sırayla yinelenmesine izin verir, ör. birden çok öğeyi atlama veya tersine çevirme
Dahili dizi işaretçileri Artık bir döngü kullanmak gerekli değildir (böylece her işlev çağrısı, sinyal alımı vb. bir kez yinelenebilir)

Birden çok diziyi birlikte yineleme #

Bazen aynı uzunluktaki iki dizinin birlikte yinelenmesi gerekir, örneğin:

$people = ['Tim', 'Tony', 'Turanga'];
$foods = ['chicken', 'beef', 'slurm'];

“array_map” bunu gerçekleştirmenin en basit yoludur:

array_map(function($person, $food) {
    return "$person likes $food\n";
}, $people, $foods);

hangi çıktı:

Tim likes chicken
Tony likes beef
Turanga likes slurm

Bu, ortak bir dizin aracılığıyla yapılabilir:

assert(count($people) === count($foods));
for ($i = 0; $i < count($people); $i++) {
    echo "$people[$i] likes $foods[$i]\n";
}

İki dizinin artımlı anahtarları yoksa, “dizi_değerleri($dizi)[$i]”, “$dizi[$i]” yerine kullanılabilir.

Her iki dizi de aynı anahtar sırasına sahipse, dizilerden birinde anahtarlı foreach döngüsü de kullanabilirsiniz:

foreach ($people as $index => $person) {
    $food = $foods[$index];
    echo "$person likes $food\n";
}

Ayrı diziler, yalnızca aynı uzunlukta ve aynı anahtar adına sahiplerse döngüye alınabilirler. Bu, bir anahtar sağlamazsanız ve numaralandırılmışsa, sorun olmaz veya anahtarları adlandırır ve her dizide aynı sıraya koyarsanız sorun olmaz.

Ayrıca “array_combine” kullanabilirsiniz.

$combinedArray = array_combine($people, $foods);
// $combinedArray = ['Tim' => 'chicken', 'Tony' => 'beef', 'Turanga' => 'slurm'];

Ardından, öncekiyle aynı şeyi yaparak bunun arasında dolaşabilirsiniz:

foreach ($combinedArray as $person => $meal) {
    echo "$person likes $meal\n";
}

Artımlı bir dizin kullanma #

Bu yöntem, bir tamsayıyı 0’dan dizideki en büyük dizine artırarak çalışır.

$colors = ['red', 'yellow', 'blue', 'green'];
for ($i = 0; $i < count($colors); $i++) {
    echo 'I am the color ' . $colors[$i] . '<br>';
}

Bu ayrıca, dizi büyükse ek yüke neden olabilecek “array_reverse” kullanılmadan bir dizinin ters sırada yinelenmesine izin verir.

$colors = ['red', 'yellow', 'blue', 'green'];
for ($i = count($colors) - 1; $i >= 0; $i--) {
    echo 'I am the color ' . $colors[$i] . '<br>';
}

Bu yöntemi kullanarak dizini kolayca atlayabilir veya geri alabilirsiniz.

$array = ["alpha", "beta", "gamma", "delta", "epsilon"];
for ($i = 0; $i < count($array); $i++) {
    echo $array[$i], PHP_EOL;
    if ($array[$i] === "gamma") {
        $array[$i] = "zeta";
        $i -= 2;
    } elseif ($array[$i] === "zeta") {
        $i++;
    }
}

Çıktı:

alpha
beta
gamma
beta
zeta
epsilon

Artımlı indeksleri olmayan diziler için (indisleri ters sırada olan diziler dahil, ör. [1 => "foo", 0 => "bar"], ["foo" => "f", "bar" => "b"]), bu doğrudan yapılamaz. Bunun yerine “array_values” veya “array_keys” kullanılabilir:

$array = ["a" => "alpha", "b" => "beta", "c" => "gamma", "d" => "delta"];
$keys = array_keys($array);
for ($i = 0; $i < count($array); $i++) {
    $key = $keys[$i];
    $value = $array[$key];
    echo "$value is $key\n";
}

Dahili dizi işaretçilerini kullanma #

Her dizi örneği bir dahili işaretçi içerir. Bu işaretçiyi manipüle ederek, aynı çağrıdan farklı zamanlarda bir dizinin farklı öğeleri alınabilir.

each kullanma #

“each()” öğesine yapılan her çağrı, geçerli dizi öğesinin anahtarını ve değerini döndürür ve dahili dizi işaretçisini artırır.

$array = ["f" => "foo", "b" => "bar"];
while (list($key, $value) = each($array)) {
    echo "$value begins with $key";
}

sonraki kullanımı #

$array = ["Alpha", "Beta", "Gamma", "Delta"];
while (($value = next($array)) !== false) {
    echo "$value\n";
}

Bu örneğin, dizideki hiçbir öğenin boolean “false” ile aynı olmadığını varsaydığını unutmayın. Böyle bir varsayımı önlemek için, dahili işaretçinin dizinin sonuna ulaşıp ulaşmadığını kontrol etmek için anahtar kullanın:

$array = ["Alpha", "Beta", "Gamma", "Delta"];
while (key($array) !== null) {
    echo current($array) . PHP_EOL;
    next($array);
}

Bu aynı zamanda bir diziyi doğrudan döngü olmadan yinelemeyi de kolaylaştırır:

class ColorPicker {
    private $colors = ["#FF0064", "#0064FF", "#64FF00", "#FF6400", "#00FF64", "#6400FF"];
    public function nextColor() : string {
        $result = next($colors);
        // if end of array reached
        if (key($colors) === null) {
            reset($colors);
        }
        return $result;
    }
}

foreach’ı kullanma #

Doğrudan döngü #

foreach ($colors as $color) {
    echo "I am the color $color<br>";
}

Tuşlarla döngü #

$foods = ['healthy' => 'Apples', 'bad' => 'Ice Cream'];
foreach ($foods as $key => $food) {
    echo "Eating $food is $key";
}

Referansa göre döngü #

Yukarıdaki örneklerdeki “foreach” döngülerinde, değeri ("$color" veya “$food”) doğrudan değiştirmek, dizideki değerini değiştirmez. Değerin dizideki öğeye bir referans işaretçisi olması için “&” operatörü gereklidir.

$years = [2001, 2002, 3, 4];
foreach ($years as &$year) {
    if ($year < 2000) $year += 2000;
}

Bu şuna benzer:

$years = [2001, 2002, 3, 4];
for($i = 0; $i < count($years); $i++) { // these two lines
    $year = &$years[$i];                // are changed to foreach by reference
    if($year < 2000) $year += 2000;
}

Eşzamanlılık #

PHP dizileri, yineleme sırasında eşzamanlılık sorunları olmadan herhangi bir şekilde değiştirilebilir (örneğin, Java ‘List’lerinden farklı olarak). Dizi başvuruya göre yineleniyorsa, sonraki yinelemeler dizideki değişikliklerden etkilenecektir. Aksi takdirde, dizideki değişiklikler sonraki yinelemeleri etkilemez (sanki bunun yerine dizinin bir kopyasını yineliyormuşsunuz gibi). Döngüyü değere göre karşılaştırın:

$array = [0 => 1, 2 => 3, 4 => 5, 6 => 7];
foreach ($array as $key => $value) {
    if ($key === 0) {
        $array[6] = 17;
        unset($array[4]);
    }
    echo "$key => $value\n";
}

Çıktı:

0 => 1
2 => 3
4 => 5
6 => 7

Ancak dizi referansla yinelenirse,

$array = [0 => 1, 2 => 3, 4 => 5, 6 => 7];
foreach ($array as $key => &$value) {
    if ($key === 0) {
        $array[6] = 17;
        unset($array[4]);
    }
    echo "$key => $value\n";
}

Çıktı:

0 => 1
2 => 3
6 => 17

“4 => 5” anahtar/değer kümesi artık yinelenmez ve “6 => 7”, “6 => 17” olarak değiştirilir.

ArrayObject Yineleyiciyi Kullanma #

Php arrayiterator, diziler ve nesneler üzerinde yineleme yaparken değerleri değiştirmenize ve ayarlamanızı kaldırmanıza olanak tanır.

Örnek:

$array = ['1' => 'apple', '2' => 'banana', '3' => 'cherry'];

$arrayObject = new ArrayObject($array);

$iterator = $arrayObject->getIterator();

for($iterator; $iterator->valid(); $iterator->next()) {
    echo $iterator->key() . ' => ' . $iterator->current() . "</br>";
}

Çıktı:

1 => apple
2 => banana
3 => cherry
