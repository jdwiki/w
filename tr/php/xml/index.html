DomDocument kullanarak bir XML oluşturun #

DOMDocument kullanarak bir XML oluşturmak için, temel olarak, tüm etiketleri ve nitelikleri ‘createElement()’ ve ‘createAttribute()’ yöntemlerini kullanarak oluşturmamız ve bunların XML yapısını ‘appendChild()’ ile oluşturmamız gerekir.

Aşağıdaki örnek, ikinci etiket için etiketler, nitelikler, bir CDATA bölümü ve farklı bir ad alanı içerir:

$dom = new DOMDocument('1.0', 'utf-8');
$dom->preserveWhiteSpace = false;
$dom->formatOutput = true;

//create the main tags, without values
$books = $dom->createElement('books');
$book_1 = $dom->createElement('book');

// create some tags with values
$name_1 = $dom->createElement('name', 'PHP - An Introduction');
$price_1 = $dom->createElement('price', '$5.95');
$id_1 = $dom->createElement('id', '1');

//create  and append an attribute
$attr_1 = $dom->createAttribute('version');
$attr_1->value = '1.0';
//append the attribute
$id_1->appendChild($attr_1);

//create the second tag book with different namespace
$namespace = 'www.example.com/libraryns/1.0';

//include the namespace prefix in the books tag
$books->setAttributeNS('http://www.w3.org/2000/xmlns/', 'xmlns:ns', $namespace);
$book_2 = $dom->createElementNS($namespace,'ns:book');
$name_2 = $dom->createElementNS($namespace, 'ns:name');

//create a CDATA section (that is another DOMNode instance) and put it inside the name tag
$name_cdata = $dom->createCDATASection('PHP - Advanced');
$name_2->appendChild($name_cdata);
$price_2 = $dom->createElementNS($namespace, 'ns:price', '$25.00');
$id_2 = $dom->createElementNS($namespace, 'ns:id', '2');

//create the XML structure
$books->appendChild($book_1);
$book_1->appendChild($name_1);
$book_1->appendChild($price_1);
$book_1->appendChild($id_1);
$books->appendChild($book_2);
$book_2->appendChild($name_2);
$book_2->appendChild($price_2);
$book_2->appendChild($id_2);

$dom->appendChild($books);

//saveXML() method returns the XML in a String
print_r ($dom->saveXML());

Bu, aşağıdaki XML’i çıkaracaktır:

<?xml version="1.0" encoding="utf-8"?>
<books xmlns:ns="www.example.com/libraryns/1.0">
  <book>
    <name>PHP - An Introduction</name>
    <price>$5.95</price>
    <id version="1.0">1</id>
  </book>
  <ns:book>
    <ns:name><![CDATA[PHP - Advanced]]></ns:name>
    <ns:price>$25.00</ns:price>
    <ns:id>2</ns:id>
  </ns:book>
</books>

DOMDocument ile bir XML belgesi okuyun #

SimpleXML’ye benzer şekilde, bir dizeden veya bir XML dosyasından XML’i ayrıştırmak için DOMDocument’i kullanabilirsiniz.

1. Bir dizeden

$doc = new DOMDocument();
$doc->loadXML($string);

2. Bir dosyadan

$doc = new DOMDocument();
$doc->load('books.xml');// use the actual file path. Absolute or relative

Ayrıştırma örneği

Aşağıdaki XML göz önüne alındığında:

<?xml version="1.0" encoding="UTF-8"?>
<books>
   <book>
      <name>PHP - An Introduction</name>
      <price>$5.95</price>
      <id>1</id>
   </book>
   <book>
      <name>PHP - Advanced</name>
      <price>$25.00</price>
      <id>2</id>
   </book>
</books>

Bu ayrıştırmak için bir örnek koddur

$books = $doc->getElementsByTagName('book');
foreach ($books as $book) {
    $title = $book->getElementsByTagName('name')->item(0)->nodeValue;
    $price = $book->getElementsByTagName('price')->item(0)->nodeValue;
    $id = $book->getElementsByTagName('id')->item(0)->nodeValue;
    print_r ("The title of the book $id is $title and it costs $price." . "\n");
}

Bu çıktı:

  1. kitabın adı PHP - Giriş’tir ve fiyatı 5,95$‘dır.

  2. kitabın adı PHP - Advanced ve 25.00$‘a mal oluyor.

XMLWriter kullanarak bir XML dosyası oluşturun #

Bir XMLWriter nesnesi örneği oluşturun:

$xml = new XMLWriter();

Ardından yazmak istediğiniz dosyayı açın. Örneğin, /var/www/example.com/xml/output.xml adresine yazmak için şunu kullanın:

$xml->openUri('file:///var/www/example.com/xml/output.xml');

Belgeyi başlatmak için (XML açık etiketini oluşturun):

$xml->startDocument('1.0', 'utf-8');

Bu çıktı:

<?xml version="1.0" encoding="UTF-8"?>

Artık öğeleri yazmaya başlayabilirsiniz:

$xml->writeElement('foo', 'bar');

Bu, XML’i oluşturacaktır:

<foo>bar</foo>

Basit değerlere sahip düğümlerden biraz daha karmaşık bir şeye ihtiyacınız varsa, bir öğeyi “başlatabilir” ve kapatmadan önce ona nitelikler ekleyebilirsiniz:

$xml->startElement('foo');
$xml->writeAttribute('bar', 'baz');
$xml->writeCdata('Lorem ipsum');
$xml->endElement();

Bu çıktı:

<foo bar="baz"><![CDATA[Lorem ipsum]]></foo>

SimpleXML ile bir XML belgesi okuyun #

XML’i bir dizeden veya bir XML dosyasından ayrıştırabilirsiniz.

1. Bir dizeden

$xml_obj = simplexml_load_string($string);

2. Bir dosyadan

$xml_obj = simplexml_load_file('books.xml');

Ayrıştırma örneği

Aşağıdaki XML göz önüne alındığında:

<?xml version="1.0" encoding="UTF-8"?>
<books>
   <book>
      <name>PHP - An Introduction</name>
      <price>$5.95</price>
      <id>1</id>
   </book>
   <book>
      <name>PHP - Advanced</name>
      <price>$25.00</price>
      <id>2</id>
   </book>
</books>

Bu ayrıştırmak için bir örnek koddur

$xml = simplexml_load_string($xml_string);
$books = $xml->book;
foreach ($books as $book) {
    $id = $book->id;
    $title = $book->name;
    $price = $book->price;
    print_r ("The title of the book $id is $title and it costs $price." . "\n");
}

Bu çıktı:

  1. kitabın adı PHP - Giriş’tir ve fiyatı 5,95 ABD dolarıdır.
  2. kitabın adı PHP - Advanced ve 25.00$‘a mal oluyor.

PHP’nin SimpleXML Kitaplığı ile XML’den Yararlanma #

SimpleXML, XML dizelerini kullanımı kolay bir PHP nesnesine dönüştüren güçlü bir kitaplıktır.

Aşağıdaki, aşağıdaki gibi bir XML yapısını varsayar.

<?xml version="1.0" encoding="UTF-8"?>
<document>
    <book>
        <bookName>StackOverflow SimpleXML Example</bookName>
        <bookAuthor>PHP Programmer</bookAuthor>
    </book>
    <book>
        <bookName>Another SimpleXML Example</bookName>
        <bookAuthor>Stack Overflow Community</bookAuthor>
        <bookAuthor>PHP Programmer</bookAuthor>
        <bookAuthor>FooBar</bookAuthor>
    </book>
</document>

Verilerimizi SimpleXML’de okuyun

Başlamak için verilerimizi SimpleXML’de okumamız gerekiyor. Bunu 3 farklı şekilde yapabiliriz. İlk olarak, verilerimizi bir DOM düğümünden yükleyebiliriz.

$xmlElement = simplexml_import_dom($domNode);

Sonraki seçeneğimiz verilerimizi bir XML dosyasından yüklemek.

$xmlElement = simplexml_load_file($filename);

Son olarak, verilerimizi bir değişkenden yükleyebiliriz.

$xmlString = '<?xml version="1.0" encoding="UTF-8"?>
<document>
    <book>
        <bookName>StackOverflow SimpleXML Example</bookName>
        <bookAuthor>PHP Programmer</bookAuthor>
    </book>
    <book>
        <bookName>Another SimpleXML Example</bookName>
        <bookAuthor>Stack Overflow Community</bookAuthor>
        <bookAuthor>PHP Programmer</bookAuthor>
        <bookAuthor>FooBar</bookAuthor>
    </book>
</document>';
$xmlElement = simplexml_load_string($xmlString);

bir DOM Öğesinden, bir dosyadan veya bir dizeden yüklemeyi seçmiş olsanız da, artık $xmlElement adında bir SimpleXMLElement değişkenine sahip olacaksınız. Artık PHP’de XML’imizi kullanmaya başlayabiliriz.

SimpleXML Verilerimize Erişim

SimpleXMLElement nesnemizdeki verilere erişmenin en basit yolu özellikleri doğrudan çağırmaktır. İlk kitabımız olan `StackOverflow SimpleXML Örneği’ne erişmek istiyorsak, aşağıdaki gibi ona erişebiliriz.

echo $xmlElement->book->bookName;

Bu noktada SimpleXML, hangi kitabı istediğimizi açıkça söylemediğimiz için ilkini istediğimizi varsayacaktır. Ancak, Başka Bir SimpleXML Örneği yerine ilkini istemediğimize karar verirsek, aşağıdaki gibi ona erişebiliriz.

echo $xmlElement->book[1]->bookName;

“[0]” kullanmanın, kullanmamakla aynı şekilde çalıştığını belirtmekte fayda var, bu nedenle

$xmlElement->book

aynı şekilde çalışır

$xmlElement->book[0]

XML’mizde dolaşmak

Bizim durumumuzda bir web sayfasında görüntülemek istediğimiz çok sayıda öğeniz, kitabınız olması gibi XML üzerinden döngü yapmak isteyebileceğiniz birçok neden vardır. Bunun için, [SimpleXMLElement’in sayım işlevinden] yararlanarak bir foreach döngüsü veya standart bir for döngüsü kullanabiliriz.]8.

foreach ( $xmlElement->book as $thisBook ) {
    echo $thisBook->bookName
}

veya

$count = $xmlElement->count();
for ( $i=0; $i<$count; $i++ ) {
    echo $xmlElement->book[$i]->bookName;
}

Hataları İşleme

Şimdiye kadar geldik, sadece insan olduğumuzu ve eninde sonunda bir hatayla karşılaşacağımızı anlamak önemlidir - özellikle sürekli farklı XML dosyalarıyla oynuyorsak. Ve böylece, bu hataları ele almak isteyeceğiz.

Bir XML dosyası oluşturduğumuzu düşünün. Fark edeceksiniz ki, bu XML daha önce sahip olduğumuza çok benzer olsa da, bu XML dosyasındaki problem son kapanış etiketinin /document yerine /doc olmasıdır.

<?xml version="1.0" encoding="UTF-8"?>
<document>
    <book>
        <bookName>StackOverflow SimpleXML Example</bookName>
        <bookAuthor>PHP Programmer</bookAuthor>
    </book>
    <book>
        <bookName>Another SimpleXML Example</bookName>
        <bookAuthor>Stack Overflow Community</bookAuthor>
        <bookAuthor>PHP Programmer</bookAuthor>
        <bookAuthor>FooBar</bookAuthor>
    </book>
</doc>

libxml_use_internal_errors(true);
$xmlElement = simplexml_load_file($file);
if ( $xmlElement === false ) {
    $errors = libxml_get_errors();
    foreach ( $errors as $thisError ) {
        switch ( $thisError->level ) {
            case LIBXML_ERR_FATAL:
                echo "FATAL ERROR: ";
                break;
            case LIBXML_ERR_ERROR:
                echo "Non Fatal Error: ";
                break;
            case LIBXML_ERR_WARNING:
                echo "Warning: ";
                break;
        }
        echo $thisError->code . PHP_EOL .
            'Message: ' . $thisError->message . PHP_EOL .
            'Line: ' . $thisError->line . PHP_EOL .
            'Column: ' . $thisError->column . PHP_EOL .
            'File: ' . $thisError->file;
    }
    libxml_clear_errors();
} else {
    echo 'Happy Days';
}
