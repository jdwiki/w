Sözdizimi #

  • preg_replace($desen, $değiştirme, $konu, $limit = -1, $sayım = 0);
  • preg_replace_callback($desen, $geri arama, $konu, $limit = -1, $sayım = 0);
  • preg_match($pattern, $subject, &$matches, $flags = 0, $ofset = 0);
  • preg_match_all($desen, $konu, &$eşleşmeler, $flags = PREG_PATTERN_ORDER, $ofset = 0);
  • preg_split($desen, $konu, $limit = -1, $bayraklar = 0)

Parametreler #

parametre Ayrıntılar
$desen normal ifadeli bir dize (PCRE deseni)

PHP düzenli ifadeleri, Perl düzenli ifadelerinden türetilen PCRE model standartlarını takip eder.

PHP’deki tüm PCRE dizeleri sınırlayıcılarla kapatılmalıdır. Sınırlayıcı, alfasayısal olmayan, ters eğik çizgi olmayan, boşluk olmayan herhangi bir karakter olabilir. Örneğin, popüler sınırlayıcılar ~, /, % şeklindedir.

PCRE kalıpları grupları, karakter sınıflarını, karakter gruplarını, ileriye/arkaya bakma iddialarını ve çıkış karakterlerini içerebilir.

$pattern dizgisinde PCRE değiştiricileri kullanmak mümkündür. Yaygın olarak kullanılanlardan bazıları “i” (büyük/küçük harfe duyarlı değildir), “m” (çok satırlı) ve “s"dir (nokta metakarakteri yeni satırları içerir). “g” (genel) değiştiricisine izin verilmiyor, bunun yerine “preg_match_all” işlevini kullanacaksınız.

PCRE dizeleriyle eşleşmeler, $ öneki numaralı dizilerle yapılır:

<?php

$replaced = preg_replace('%hello ([a-z]+) world%', 'goodbye $1 world', 'hello awesome world');

echo $replaced; // 'goodbye awesome world'

Küresel Normal İfade eşleşmesi #

‘preg_match_all’ kullanılarak bir global RegExp eşleşmesi gerçekleştirilebilir. “preg_match_all”, konu dizesindeki tüm eşleşen sonuçları döndürür (yalnızca ilkini döndüren “preg_match"in aksine).

“preg_match_all” işlevi eşleşme sayısını döndürür. Üçüncü parametre $matches, dördüncü parametrede verilebilen bayraklar tarafından kontrol edilen formatta eşleşmeleri içerecektir.

Bir dizi verilirse, “$eşleşmeler”, “preg_match” ile alacağınız benzer biçimde dizi içerecektir, ancak “preg_match” ilk eşleşmede durur, burada “preg_match_all”, dize tamamen tüketilene ve geri dönene kadar dize üzerinde yinelenir. dördüncü argümanda bayrak tarafından kontrol edilebilen çok boyutlu bir dizideki her yinelemenin sonucu.

Dördüncü argüman olan “$flags”, “$eşleşmeler” dizisinin yapısını kontrol eder. Varsayılan mod “PREG_PATTERN_ORDER” ve olası bayraklar “PREG_SET_ORDER” ve “PREG_PATTERN_ORDER” şeklindedir.

Aşağıdaki kod, “preg_match_all” kullanımını gösterir:

$subject = "a1b c2d3e f4g";
$pattern = '/[a-z]([0-9])[a-z]/';

var_dump(preg_match_all($pattern, $subject, $matches, PREG_SET_ORDER)); // int(3)
var_dump($matches);
preg_match_all($pattern, $subject, $matches); // the flag is PREG_PATTERN_ORDER by default
var_dump($matches);
// And for reference, same regexp run through preg_match()
preg_match($pattern, $subject, $matches);
var_dump($matches);

“PREG_SET_ORDER"dan gelen ilk var_dump şu çıktıyı verir:

array(3) {
  [0]=>
  array(2) {
    [0]=>
    string(3) "a1b"
    [1]=>
    string(1) "1"
  }
  [1]=>
  array(2) {
    [0]=>
    string(3) "c2d"
    [1]=>
    string(1) "2"
  }
  [2]=>
  array(2) {
    [0]=>
    string(3) "f4g"
    [1]=>
    string(1) "4"
  }
}

“$matches” iç içe üç diziye sahiptir. Her dizi, “preg_match"in dönüş sonucuyla aynı formatta olan bir eşleşmeyi temsil eder.

İkinci var_dump (PREG_PATTERN_ORDER) şu çıktıyı verir:

array(2) {
  [0]=>
  array(3) {
    [0]=>
    string(3) "a1b"
    [1]=>
    string(3) "c2d"
    [2]=>
    string(3) "f4g"
  }
  [1]=>
  array(3) {
    [0]=>
    string(1) "1"
    [1]=>
    string(1) "2"
    [2]=>
    string(1) "4"
  }

Aynı normal ifade “preg_match” üzerinden çalıştırıldığında, aşağıdaki dizi döndürülür:

array(2) {
  [0] =>
  string(3) "a1b"
  [1] =>
  string(1) "1"
}

Normal ifadelerle dize eşleştirme #

“preg_match”, bir dizenin normal ifadeyle eşleşip eşleşmediğini kontrol eder.

$string = 'This is a string which contains numbers: 12345';

$isMatched = preg_match('%^[a-zA-Z]+: [0-9]+$%', $string);
var_dump($isMatched); // bool(true)

Üçüncü bir parametre girerseniz, normal ifadenin eşleşen verileriyle doldurulur:

preg_match('%^([a-zA-Z]+): ([0-9]+)$%', 'This is a string which contains numbers: 12345', $matches);
// $matches now contains results of the regular expression matches in an array.
echo json_encode($matches); // ["numbers: 12345", "numbers", "12345"]

“$matches”, tüm eşleşmenin bir dizisini, ardından açık parantezin ofset sırasına göre parantezlerle sınırlanmış normal ifadede alt dizeleri içerir. Bunun anlamı, normal ifade olarak ‘/z(a(b))/’ varsa, dizin 0 tüm “zab” alt dizesini içerir, dizin 1, dış parantezler “ab” ile sınırlanan alt dizeyi içerir ve dizin 2 şunları içerir: iç parantezler “b”.

Dizeyi düzenli bir ifadeyle diziye ayırın #

$string = "0| PHP 1| CSS 2| HTML 3| AJAX 4| JSON";

//[0-9]: Any single character in the range 0 to 9
// +   : One or more of 0 to 9
$array = preg_split("/[0-9]+\|/", $string, -1, PREG_SPLIT_NO_EMPTY);
//Or
// []  : Character class
// \d  : Any digit
//  +  : One or more of Any digit
$array = preg_split("/[\d]+\|/", $string, -1, PREG_SPLIT_NO_EMPTY);

Çıktı:

Array
(
    [0] =>  PHP
    [1] =>  CSS 
    [2] =>  HTML 
    [3] =>  AJAX 
    [4] =>  JSON
)

Bir dizeyi bir diziye bölmek için, eşleşmesi ve araması için dizeyi ve preg_split(); için bir regexp iletmeniz yeterlidir, üçüncü bir parametre (limit) eklemek, gerçekleştirilecek “eşleşme” sayısını ayarlamanıza, kalan string dizinin sonuna eklenecektir.

Dördüncü parametre (‘flags’) burada dizimizin boş anahtarlar/değerler içermesini engelleyen ‘PREG_SPLIT_NO_EMPTY’ kullanıyoruz.

Dizenin normal ifadeyle değiştirilmesi #

$string = "a;b;c\nd;e;f";
// $1, $2 and $3 represent the first, second and third capturing groups
echo preg_replace("(^([^;]+);([^;]+);([^;]+)$)m", "$3;$2;$1", $string);

çıktılar

c;b;a
f;e;d

Noktalı virgüller arasındaki her şeyi arar ve sırayı tersine çevirir.

Geri arama ile dize değiştir #

“preg_replace_callback”, eşleşen her yakalama grubunu tanımlanan geri aramaya göndererek çalışır ve onu geri aramanın dönüş değeriyle değiştirir. Bu, herhangi bir mantığa dayalı olarak dizeleri değiştirmemizi sağlar.

$subject = "He said 123abc, I said 456efg, then she said 789hij";
$regex = "/\b(\d+)\w+/";

// This function replaces the matched entries conditionally 
// depending upon the first character of the capturing group
function regex_replace($matches){
    switch($matches[1][0]){
        case '7':
            $replacement = "<b>{$matches[0]}</b>";
            break;
        default:
            $replacement = "<i>{$matches[0]}</i>";
    }
    return $replacement;
}

$replaced_str = preg_replace_callback($regex, "regex_replace", $subject);

print_r($replaced_str);
# He said <i>123abc</i>, I said <i>456efg</i>, then she said <b>789hij</b>
