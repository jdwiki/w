Sözdizimi #

  • ifade1 ise ?o zaman? body1 başka bir ifade2 ise ?o zaman? body2 … ?başka? ?vücutN?
  • sonraki vücut testini başlatmak için
  • test gövdesi sırasında
  • foreach varlist1 list1 ?varlist2 list2 …? gövde

Belgeler: break, for, foreach, if, switch, uplevel, while

if / while / for #

if ifade1 ?sonra? body1 yoksa ifade2 ?o zaman? vücut2 … ?başka? ?vücutN?

exprN, bir boole değeri olarak değerlendirilen bir ifadedir. bodyN bir komut listesidir.

set i 5
if {$i < 10} {
  puts {hello world}
} elseif {$i < 70} {
  puts {enjoy world}
} else {
  puts {goodbye world}
}

için sonraki vücudu test etmeye başla

start, next ve body komut listeleridir. test, bir boole değeri olarak değerlendirilen bir ifadedir.

break komutu döngüden çıkar. devam komutu, döngünün bir sonraki yinelemesine atlayacaktır.

Yaygın kullanım:

for {set i 0} {$i < 5} {incr i} {
  puts "$i: hello world"
}

start ve next komut listeleri olduğundan, herhangi bir komut mevcut olabilir.

for {set i 0; set j 5} {$i < 5} {incr i; incr j -1} {
  puts "i:$i j:$j"
}

while test gövdesi

test, bir boole değeri olarak değerlendirilen herhangi bir ifadedir. test true iken body yürütülür.

set x 0
while {$x < 5} {
  puts "hello world"
  incr x
}

break komutu döngüden çıkar. devam komutu, döngünün bir sonraki yinelemesine atlayacaktır.

set lineCount 0
while {[gets stdin line] >= 0} {
  puts "[incr lineCount]: $line"
  if { $line eq "exit" } {
    break
  }  
}

Liste yinelemesi: foreach #

foreach varlist1 list1 ?varlist2 list2 …? gövde

foreach, bir liste veya birden çok liste üzerinde döngüye izin veren güçlü bir kontrol yapısıdır.

set alpha [list a b c d e f]
foreach {key} $alpha {
   puts "key: $key"
}

Birden çok değişken adı belirtilebilir.

set alphaindexes [list a 1 b 2 c 3 d 4 e 5 f 6]
foreach {key num} $alphaindexes {
   puts "key:$key num:$num"
}

Aynı anda birden fazla liste tekrar edilebilir.

set alpha [list a b c d e f]
set indexes [list 1 2 3 4 5 6]
foreach {key} $alpha {idx} $indexes {
   puts "key: $key idx:$idx"
}

Tcl’ye yeni bir kontrol yapısı ekleme #

Tcl’de bir kontrol yapısı temelde sadece başka bir komuttur. Bu, bir “do … while” / “do … … kadar” kontrol yapısının olası bir uygulamasıdır.

proc do {body keyword expression} {
    uplevel 1 $body
    switch $keyword {
        while {uplevel 1 [list while $expression $body]}
        until {uplevel 1 [list while !($expression) $body]}
        default {
            return -code error "unknown keyword \"$keyword\": must be until or while"
        }
    }
}

Her iki tür “do” döngüsü için ortak olan, “body” adlı komut dosyasının her zaman en az bir kez çalıştırılmasıdır, bu yüzden bunu hemen yaparız. “uplevel 1 $body” çağrısı, “komut dosyasını arayanın yığın düzeyinde yürütün” anlamına gelir. Bu şekilde, komut dosyası tarafından kullanılan tüm değişkenler görünür olacak ve üretilen tüm sonuçlar arayan düzeyinde kalacaktır. Ardından komut dosyası, “anahtar kelime” parametresine dayalı olarak, bir koşul doğruyken mi yoksa yanlış olana kadar mı yineleyeceğini seçer; bu, koşulun mantıksal olumsuzlaması doğruyken yinelemeyle aynıdır. Beklenmeyen bir anahtar kelime verilirse, bir hata mesajı üretilir.
