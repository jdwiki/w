Kurulum veya Kurulum #

Pencereler #

  1. Doğru JDK’ya sahip olduğunuzu doğrulayın. Komut istemini açarak kontrol edebilirsiniz (windows tuşuna basın ve cmd yazın). Komut istemine “javac -version” yazın, bu, eşleştirmenizde yüklü olan JDK’nın geçerli sürümünü veya Java eksikse bir hatayı* gösterir. JDK mevcut değilse veya sürüm 1.8’den düşükse, Java SE Development Kit 8‘i indirin.

  2. En son Android Studio sürümünü indirin.

  3. İndirilen .exe dosyasını başlatın.

  4. Android Studio’yu kurmak için sihirbazı takip edin

  5. Kurulum tamamlandıktan sonra, kurulum işlemi sırasında Masaüstünde yapılan kısayoldan Android Studio’yu açın.

  6. Android Studio’yu ilk kez açtığınızda, önceki ayarlarınızı kopyalamanızı isteyebilir, ancak ilk kez olduğu için hiçbir şeyi kopyalamamayı seçebilirsiniz.

  7. Ardından Android Studio, Android uygulamaları geliştirmek için gerekli API’leri indirmenizi ister. Bunları indirin, indirdikten sonra IDE açılacak ve ilk uygulamanızı yazabileceksiniz.

Mac #

  1. Doğru JDK’ya sahip olduğunuzu doğrulayın. Terminali açarak kontrol edebilirsiniz (komut + boşluk tuşlarına basın ve terminal yazın). Komut satırına javac -version yazın, bu, eşleştirmenizde yüklü olan JDK’nın geçerli sürümünü veya Java eksikse bir hatayı* gösterir. JDK mevcut değilse veya sürüm 1.8’den düşükse, Java SE Development Kit 8.‘yi indirin **

  2. En son Android Studio sürümünü indirin.

  3. İndirilen .dmg dosyasını başlatın.

  4. Android Studio’yu Uygulamalar klasörüne sürükleyip bırakın, ardından Android Studio’yu başlatın.

  5. Android Studio’yu açın.

  6. Android Studio’yu ilk kez açtığınızda, önceki ayarlarınızı kopyalamanızı isteyebilir, ancak ilk kez olduğu için hiçbir şeyi kopyalamamayı seçebilirsiniz.

  7. Ardından Android Studio, Android uygulamaları geliştirmek için gerekli API’leri indirmenizi ister. Bunları indirin, indirdikten sonra IDE açılacak ve ilk uygulamanızı yazabileceksiniz.

Linux #

  1. Doğru JDK’ya sahip olduğunuzu doğrulayın. Terminali açarak kontrol edebilirsiniz (command + boşluk tuşlarına basın ve ’terminal’ yazın). Komut satırına “javac -version” yazın, bu, eşleştirmenizde yüklü olan JDK’nın geçerli sürümünü veya Java eksikse bir hatayı* gösterir. JDK mevcut değilse veya sürüm 1.8’den düşükse, Java SE Development Kit 8‘i indirin. **

  2. En son Android Studio sürümünü indirin.

  3. Android Studio’yu belirli bir klasörde açın/çıkartın.

  4. Terminali açın ve Android Studio’yu çıkardığınız yola gidin. (Ardından Android Studio klasörüne girmek için cd komutunu kullanın.) İçeri girdikten sonra bin klasörüne girmemiz gerekiyor, bu yüzden tekrar cd bin komutunu kullanın ve girin.

  5. Şimdi gerekli dosyamızın modunu değiştirmemiz gerekiyor yani studio.sh bunun için sudo chmod 777 -R studio.sh komutunu girin, enter tuşuna basın ve (varsa) şifrenizi yazın ve girin. (Ayrıca ls komutu ile bin içinde bulunan dosyaların listesini görebilirsiniz.).

  6. Modu değiştirdikten sonra .studio.sh dosyasını çalıştırmamız gerekiyor, bunu yapmak için ./studio.sh komutunu girin.

  7. Android Studio başlatma komutunu ilk kez çalıştırdığınızda, önceki ayarlarınızı kopyalamanızı isteyebilir, ancak ilk kez olduğu için hiçbir şeyi kopyalamamayı seçebilirsiniz.

  8. Ardından Android Studio, Android uygulamaları geliştirmek için gerekli API’leri indirmenizi ister. Bunları indirin, indirdikten sonra IDE açılacak ve ilk uygulamanızı yazabileceksiniz.

64 bit ubuntu işletim sisteminde android stüdyo kurarken “mksdcard SDK aracı çalıştırılamıyor” ile karşılaşabilirsiniz çünkü stüdyo bazı 32 ikili dosya gerektirir. Bu hatanın üstesinden gelmek için Android Studio’yu bitirin ve kapatın ve terminale gidin ve sudo apt-get install lib32z1 lib32ncurses5 lib32bz2-1.0 lib32stdc++6 komutunu çalıştırın. Bu ikili dosyalar için kurulum tamamlandıktan sonra tekrar 6. adıma dönün ve kurulum işlemine devam edin.

Notlar #

  • JDK’yı zaten yüklediyseniz ve hala alıyorsanız, ‘Sistem Değişkenleri’nde ‘JAVA_HOME’u ayarladığınızdan emin olun. Birinin nasıl kurulacağıyla ilgili bu cevaba bakabilirsiniz.

  • Bununla birlikte, Mac’te Android Studio’da JDK 1.8 kullanılırken bilinen kararlılık sorunları vardır. Bu sorunlar çözülene kadar, JDK’nızı daha eski bir sürüme (ancak JDK 1.6’dan daha düşük olmayacak şekilde) düşürerek kararlılığı artırabilirsiniz.

  • Android Studio indirme işlemi tamamlanırken, JDK’nın hangi sürümüne sahip olduğunuzu doğrulayın: bir komut satırı açın ve “javac -version” yazın. JDK mevcut değilse veya sürüm 1.8‘den düşükse, [Java SE Development Kit 8]‘i indirin(http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads- 2133151.html).

  • Ubuntu’nun 64 bit sürümünü çalıştırıyorsanız, aşağıdaki komutlarla bazı 32 bit kitaplıkları yüklemeniz gerekir:

    sudo dpkg --add-architecture i386
    sudo apt-get update
    sudo apt-get install lib32z1 lib32ncurses5 libbz2-1.0:i386 lib32stdc++6

  • 64-bit Fedora çalıştırıyorsanız, komut şudur:

    sudo yum install zlib.i686 ncurses-libs.i686 bzip2-libs.i686

İndirmeyle ilgili sorun #

  • Sisteminizin Android Studio ile uyumlu olmadığı mesajı gibi bir sorunla karşılaşırsanız (Chrome’dan farklı bir web tarayıcısı kullandığınızda olabilir) http://tools.android.com/ adresinden ‘Android Studio’yu indirin.

Bu kadar!

1. Farklı Cihazları Önizleyin #

Android stüdyosunun sağ tarafında bir ön izleme paneli bulunmaktadır. Bu panelde, uygulamanızın ‘UI’sini şu şekilde önizlediğiniz cihaz adına sahip bir düğme var.

Bunun küçük açılır göstergesine tıklayın ve önceden tanımlanmış tüm cihazlarla birlikte kayan bir panel görünecektir. Ekran boyutlarından bahseden farklı cihazlarla Uygulama ‘UI’nizi önizlemek için bunlardan herhangi birini seçebilirsiniz.
