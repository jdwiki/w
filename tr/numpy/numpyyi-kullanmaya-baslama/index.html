Temel İçe Aktarma #

Herhangi bir bölümünü kullanmak için numpy modülünü içe aktarın.

import numpy as np

Çoğu örnek, numpy’nin kısaltması olarak “np"yi kullanır. Kod örneklerinde “np"nin “numpy” anlamına geldiğini varsayın.

x = np.array([1,2,3,4])

Linux’ta kurulum #

NumPy, çoğu popüler Linux dağıtımının varsayılan depolarında bulunur ve bir Linux dağıtımındaki paketlerin genellikle kurulduğu şekilde kurulabilir.

Bazı Linux dağıtımlarının Python 2.x ve Python 3.x için farklı NumPy paketleri vardır. Ubuntu ve Debian’da, APT paket yöneticisini kullanarak sistem düzeyinde “numpy"yi kurun:

sudo apt-get install python-numpy  
sudo apt-get install python3-numpy

Diğer dağıtımlar için zypper (Suse), yum (Fedora) vb. gibi paket yöneticilerini kullanın.

“numpy”, Python 2 için Python’un paket yöneticisi “pip” ve Python 3 için “pip3” ile de kurulabilir:

pip install numpy  # install numpy for Python 2
pip3 install numpy  # install numpy for Python 3

“pip”, en popüler Linux dağıtımlarının varsayılan depolarında bulunur ve aşağıdakiler kullanılarak Python 2 ve Python 3 için yüklenebilir:

sudo apt-get install python-pip  # pip for Python 2
sudo apt-get install python3-pip  # pip for Python 3

Kurulumdan sonra, Python paketlerini kurmak için pip kullanmak için Python 2 için ‘pip’ ve Python 3 için ‘pip3’ kullanın. Ancak, kaynaktan numpy oluşturmak için gerekli olan birçok bağımlılık kurmanız gerekebileceğini unutmayın (geliştirme paketleri, derleyiciler, fortran vb. dahil).

Sistem düzeyinde ’numpy’ kurmanın yanı sıra, ‘virtualenv’ gibi popüler Python paketlerini kullanarak ’numpy’yi sanal ortamlarda kurmak da yaygındır (belki de şiddetle tavsiye edilir). Ubuntu’da ‘virtualenv’ aşağıdakiler kullanılarak kurulabilir:

sudo apt-get install virtualenv

Ardından, Python 2 veya Python 3 için bir sanalenv oluşturup etkinleştirin ve ardından “numpy"yi yüklemek için “pip"i kullanın:

virtualenv venv  # create virtualenv named venv for Python 2
virtualenv venv -p python3  # create virtualenv named venv for Python 3
source venv/bin/activate  # activate virtualenv named venv
pip install numpy  # use pip for Python 2 and Python 3; do not use pip3 for Python3

Windows’ta Kurulum #

pypi (pip tarafından kullanılan varsayılan paket dizini) aracılığıyla Numpy yüklemesi genellikle Windows bilgisayarlarda başarısız olur. Windows’a yüklemenin en kolay yolu, önceden derlenmiş ikili dosyaları kullanmaktır.

Pek çok paketin önceden derlenmiş tekerlekleri için bir kaynak Christopher Gohkle’nin sitesi. Python sürümünüze ve sisteminize göre bir sürüm seçin. 64 bit sistemde Python 3.5 için bir örnek:

  1. “numpy-1.11.1+mkl-cp35-cp35m-win_amd64.whl” dosyasını buradan indirin
  2. Bir Windows terminali açın (cmd veya powershell)
  3. pip install C:\path_to_download\numpy-1.11.1+mkl-cp35-cp35m-win_amd64.whl komutunu yazın

Tek paketlerle uğraşmak istemiyorsanız, çoğu paketi bir araya toplayan ve çalışmak için sınırlı bir ortam sağlayan Winpython dağıtımını kullanabilirsiniz. Benzer şekilde, Anaconda Python dağıtımı numpy ve çok sayıda diğer yaygın paketlerle önceden yüklenmiş olarak gelir.

Diğer bir popüler kaynak, sanal ortamları da destekleyen conda paket yöneticisi‘dir.

  1. conda indirin ve kurun.
  2. Bir Windows terminali açın.
  3. conda install numpy komutunu yazın

Mac’te Kurulum #

NumPy’yi Mac’te kurmanın en kolay yolu pip kullanmaktır.

pip install numpy

Conda kullanarak kurulum. Conda, Windows, Mac ve Linux için kullanılabilir

  1. Conda’yı yükleyin. Conda’yı Anaconda (Tam paket, numpy dahil) veya Miniconda (herhangi bir ek paket olmadan yalnızca Conda,Python ve bağlı oldukları paketler) ile kurmanın iki yolu vardır. Hem Anaconda hem de Miniconda aynı Conda’yı kurar.
  2. Miniconda için ek komut, conda install numpy komutunu yazın

Rackspace tarafından barındırılan Geçici Jupyter Notebook #

Jupyter Notebooks etkileşimli, tarayıcı tabanlı bir geliştirme ortamıdır. Başlangıçta python hesaplamasını çalıştırmak için geliştirildiler ve bu nedenle numpy ile çok iyi oynuyorlar. Bir Jupyter not defterinde herhangi birinin yerel sistemine tam yükleme yapmadan numpy’yi denemek için Rackspace, tmpnb.org adresinde ücretsiz geçici not defterleri sağlar.

Not: Bunun, herhangi bir türde yukarı satış içeren tescilli bir hizmet olmadığını. Jupyter, UC Berkeley ve Cal Poly San Luis Obispo tarafından geliştirilen tamamen açık kaynaklı bir teknolojidir. Rackspace, bu hizmeti geliştirme sürecinin bir parçası olarak bağışlar.

tmpnb.org’da “numpy"yi denemek için:

  1. tmpnb.org adresini ziyaret edin
  2. “Python.ipynb’ye Hoş Geldiniz"i seçin veya

  4. Yeni » Python 3
