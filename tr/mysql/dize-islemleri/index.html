STR_TO_DATE - Dizeyi tarihe dönüştür #

[the string] “07/25/2016” gibi bir değere sahip “my_date_field” adlı dize türlerinden birinin sütunuyla, aşağıdaki ifade “STR_TO_DATE” işlevinin kullanımını gösterir:

SELECT STR_TO_DATE(my_date_field, '%m/%d/%Y') FROM my_table;

Bu işlevi ‘WHERE’ yan tümcesinin bir parçası olarak da kullanabilirsiniz.

LOWER() / LCASE() #

Dize bağımsız değişkenini küçük harfe dönüştürün

Sözdizimi: LOWER(str)

LOWER('fOoBar') -- 'foobar'
LCASE('fOoBar') -- 'foobar'

YER DEĞİŞTİRMEK() #

Dize bağımsız değişkenini küçük harfe dönüştürün

Sözdizimi: REPLACE(str, from_str, to_str)

REPLACE('foobarbaz', 'bar', 'BAR') -- 'fooBARbaz'
REPLACE('foobarbaz', 'zzz', 'ZZZ') -- 'foobarbaz'

SUBSTRING() #

SUBSTRING (veya eşdeğeri: SUBSTR), belirtilen konumdan başlayarak ve isteğe bağlı olarak belirtilen uzunlukta alt dizeyi döndürür

Sözdizimi: SUBSTRING(str, start_position)

SELECT SUBSTRING('foobarbaz', 4); -- 'barbaz'
SELECT SUBSTRING('foobarbaz' FROM 4); -- 'barbaz'

-- using negative indexing
SELECT SUBSTRING('foobarbaz', -6); -- 'barbaz'
SELECT SUBSTRING('foobarbaz' FROM -6); -- 'barbaz'

Sözdizimi: SUBSTRING(str; start_position; uzunluk)

SELECT SUBSTRING('foobarbaz', 4, 3); -- 'bar'
SELECT SUBSTRING('foobarbaz', FROM 4 FOR 3); -- 'bar'

-- using negative indexing
SELECT SUBSTRING('foobarbaz', -6, 3); -- 'bar'
SELECT SUBSTRING('foobarbaz' FROM -6 FOR 3); -- 'bar'

BÜYÜK() / UCASE() #

Dize bağımsız değişkenini büyük harfe dönüştür

Sözdizimi: UPPER(str)

UPPER('fOoBar') -- 'FOOBAR'
UCASE('fOoBar') -- 'FOOBAR'

UZUNLUK() #

Dizenin uzunluğunu bayt cinsinden döndürür. Bazı karakterler birden fazla bayt kullanılarak kodlanabileceğinden, karakter uzunluğunu istiyorsanız CHAR_LENGTH() bölümüne bakın.

Sözdizimi: LENGTH(str)

LENGTH('foobar') -- 6
LENGTH('fööbar') -- 8 -- contrast with CHAR_LENGTH(...) = 6

CHAR_LENGTH() #

Dizedeki karakter sayısını döndür

Sözdizimi: CHAR_LENGTH(str)

CHAR_LENGTH('foobar') -- 6
CHAR_LENGTH('fööbar') -- 6 -- contrast with LENGTH(...) = 8

HEX(str) #

Argümanı onaltılık sayıya dönüştürün. Bu dizeler için kullanılır.

HEX('fööbar') -- 66F6F6626172 -- in "CHARACTER SET latin1" because "F6" is hex for ö
HEX('fööbar') -- 66C3B6C3B6626172  -- in "CHARACTER SET utf8 or utf8mb4" because "C3B6" is hex for ö
