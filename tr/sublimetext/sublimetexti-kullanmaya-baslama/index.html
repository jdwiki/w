Kullanıcı Arayüzünü/Temayı Özelleştirme #

Sublime Text’i özelleştirmek için (temalar, sözdizimi vurgulama vb. dahil) paket kontrolünün kurulu olması gerekir.

Paket kontrolünü kurmak için www.packagecontrol.io/installation adresini ziyaret edin.

Yukarıdaki bağlantıyı takip etmek yerine, yüklemek için Sublime konsolunu açabilirsiniz. Konsola ‘ctrl+’ kısayolu veya ‘Görünüm > Konsolu Göster’ menüsü aracılığıyla erişilir. Açıldıktan sonra, Sublime Text sürümünüz için aşağıdaki Python kodunu konsola yapıştırın.

import urllib.request,os,hashlib; h = 'df21e130d211cfc94d9b0905775a7c0f' + '1e3d39e33b79698005270310898eea76'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by)

Bu kod sizin için Installed Packages klasörünü oluşturur ve ardından Package Control.sublime-package klasörünü içine indirir. İndirme, Python standart kitaplık sınırlamaları nedeniyle HTTPS yerine HTTP üzerinden yapılacaktır, ancak dosya SHA-256 kullanılarak doğrulanacaktır.

Bu bittikten sonra, Sublime Text’i açın ve paket arama işlevini açmak için OSX’te ‘shift-command-p’ veya pencerelerde ‘control-p’ tuşlarına basın.

“Paket Kontrolü” yazmaya başlayın ve “Paket Kontrolü: Paketi Yükle"yi seçin

Bu yüklendikten sonra, her paketi/temayı arayın ve bir tane yüklemek için çift tıklayın.

Bu yüklendikten sonra, arama işlevini yeniden açın (OSX’te ‘shift-command-p’ veya pencerelerde ‘control-p’) ve yeni yüklediğiniz paketi/temayı arayın.

Çoğu paket otomatik etkinleştirmeyle gelir; örneğin, “Boxy” teması “Boxy Teması: Etkinleştirme"yi gösterir. Temayı yüklemek için bunu seçmeniz yeterlidir.

Kurulum #

Sublime Text’i indirmek için web sitelerindeki download bölümünü ziyaret edin.

Aşağıdakiler dahil olmak üzere farklı işletim sistemleri için çeşitli yapılar vardır:

  • ÖKÜZLER
  • Pencereler
  • Ücretsiz (64 ve 32 bit)

Sublime Text başarıyla indirildikten sonra, kurulum işlemini başlatmak için .dmg dosyasını açmanız yeterlidir.

Kurulum işlemi tamamlandıktan sonra Uygulamalar klasörüne gidin ve Sublime Text’i açın.

Sublime Text’in ücretsiz sürümünü başarıyla yüklediniz!

Lisans almak için, bu satın alma bağlantısını ziyaret ederek Sublime Text satın almalısınız.
