Tür güvenli oluşturucu bir dil özelliğinden ziyade bir kavramdır, bu nedenle kesin olarak resmileştirilmemiştir.

Tip güvenli bir oluşturucunun tipik yapısı #

Tek bir oluşturucu işlevi genellikle 3 adımdan oluşur:

  1. Bir nesne oluşturun.
  2. Nesneyi başlatmak için lambda yürütün.
  3. Nesneyi yapıya ekleyin veya iade edin.

Kotlin kitaplıklarında tip güvenli inşaatçılar #

Tip güvenli oluşturucular kavramı, bazı Kotlin kitaplıklarında ve çerçevelerinde yaygın olarak kullanılmaktadır, örneğin:

  • Anko
  • Wasabi
  • Hangisi
  • Özellik

Tip güvenli ağaç yapısı oluşturucu #

Oluşturucular, alıcılarla birlikte lambda ifadelerini argüman olarak alan bir dizi uzantı işlevi olarak tanımlanabilir. Bu örnekte, bir “JFrame” menüsü oluşturuluyor:

import javax.swing.*

fun JFrame.menuBar(init: JMenuBar.() -> Unit) {
    val menuBar = JMenuBar()
    menuBar.init()
    setJMenuBar(menuBar)
}

fun JMenuBar.menu(caption: String, init: JMenu.() -> Unit) {
    val menu = JMenu(caption)
    menu.init()
    add(menu)
}

fun JMenu.menuItem(caption: String, init: JMenuItem.() -> Unit) {
    val menuItem = JMenuItem(caption)
    menuItem.init()
    add(menuItem)
}

Bu işlevler daha sonra nesnelerin ağaç yapısını kolay bir şekilde oluşturmak için kullanılabilir:

class MyFrame : JFrame() {
    init {
        menuBar {
            menu("Menu1") {
                menuItem("Item1") {
                    // Initialize MenuItem with some Action
                }
                menuItem("Item2") {}
            }
            menu("Menu2") {
                menuItem("Item3") {}
                menuItem("Item4") {}
            }
        }
    }
}
