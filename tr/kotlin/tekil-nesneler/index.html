Tekil nesneler

  • Java’nın statik yöntemlerinin/alanlarının değiştirilmesi olarak kullanın
  • Singleton olarak kullanın

Bu sayfada

  • Java’nın statik yöntemlerinin/alanlarının değiştirilmesi olarak kullanın
  • Singleton olarak kullanın

Bir nesne, “nesne” anahtar sözcüğü kullanılarak bildirilebilen özel bir sınıf türüdür. Nesneler, Java’daki Singletons’a (bir tasarım deseni) benzer. Ayrıca java’nın statik parçası olarak da işlev görür. Java’dan kotlin’e geçiş yapan yeni başlayanlar, bu özelliği static veya singletons yerine büyük ölçüde kullanabilir.

Java’nın statik yöntemlerinin/alanlarının değiştirilmesi olarak kullanın #

object CommonUtils {

    var anyname: String ="Hello"

    fun dispMsg(message: String) {
        println(message)
    }
}

Başka herhangi bir sınıftan, değişkeni ve işlevleri şu şekilde çağırmanız yeterlidir:

CommonUtils.anyname
CommonUtils.dispMsg("like static call")

Singleton olarak kullanın #

Kotlin nesneleri aslında yalnızca tekil nesnelerdir. Birincil avantajı, singleton örneğini almak için `SomeSingleton.INSTANCE’ kullanmak zorunda olmamanızdır.

Java’da singleton’unuz şöyle görünür:

public enum SharedRegistry {
    INSTANCE;
    public void register(String key, Object thing) {}
}

public static void main(String[] args) {
    SharedRegistry.INSTANCE.register("a", "apple");
    SharedRegistry.INSTANCE.register("b", "boy");
    SharedRegistry.INSTANCE.register("c", "cat");
    SharedRegistry.INSTANCE.register("d", "dog");
}

Kotlin’de eşdeğer kod

object SharedRegistry {
    fun register(key: String, thing: Object) {}
}

fun main(Array<String> args) {
    SharedRegistry.register("a", "apple")
    SharedRegistry.register("b", "boy")
    SharedRegistry.register("c", "cat")
    SharedRegistry.register("d", "dog")
}
