Kotlin’de döngüler, mümkün olan her yerde optimize edilmiş döngülere derlenir. Örneğin, bir sayı aralığı üzerinde yinelerseniz, nesne oluşturma ek yükünü önlemek için bayt kodu, düz int değerlerine dayalı olarak karşılık gelen bir döngüye derlenir.

Yinelenebilirler üzerinde döngü #

Standart for-loop’u kullanarak herhangi bir yinelenebilir öğe üzerinde döngü oluşturabilirsiniz:

val list = listOf("Hello", "World", "!")
for(str in list) {
    print(str)
}

Sayı aralıkları gibi Kotlin’deki pek çok şey yinelenebilir:

for(i in 0..9) {
    print(i)
}

Yineleme sırasında bir dizine ihtiyacınız varsa:

for((index, element) in iterable.withIndex()) {
    print("$element at index $index")
}

Ayrıca, forEach işlevini kullanarak, görünür dil yapıları olmadan standart kitaplığa dahil edilen yinelemeye işlevsel bir yaklaşım da vardır:

iterable.forEach {
    print(it.toString())
}

Bu örnekte “it”, geçerli öğeyi örtük olarak tutar, bkz. Lambda İşlevleri

1: https://www.wikiod.com/tr/kotlin/fonksiyonlar#Lambda Fonksiyonları

Bir eylemi x kez tekrarla #

repeat(10) { i ->
    println("This line will be printed 10 times")
    println("We are on the ${i + 1}. loop iteration")
}

Ara ve devam et #

Break ve Continue anahtar sözcükleri diğer dillerde olduğu gibi çalışır.

while(true) {
    if(condition1) {
        continue // Will immediately start the next iteration, without executing the rest of the loop body
    }
    if(condition2) {
        break // Will exit the loop completely
    }
}

İç içe döngüleriniz varsa, döngü deyimlerini etiketleyebilir ve hangi döngüye devam etmek veya kesmek istediğinizi belirtmek için break ve Continue deyimlerini niteleyebilirsiniz:

outer@ for(i in 0..10) {
    inner@ for(j in 0..10) {
        break       // Will break the inner loop
        break@inner // Will break the inner loop
        break@outer // Will break the outer loop
    }
}

Ancak bu yaklaşım, işlevsel ‘forEach’ yapısı için çalışmayacaktır.

Kotlin’de bir Harita üzerinde yineleme #

//iterates over a map, getting the key and value at once

var map = hashMapOf(1 to "foo", 2 to "bar", 3 to "baz")

for ((key, value) in map) {
    println("Map[$key] = $value")
}

Özyineleme #

Çoğu programlama dilinde olduğu gibi Kotlin’de özyineleme yoluyla döngü yapmak da mümkündür.

fun factorial(n: Long): Long = if (n == 0) 1 else n * factorial(n - 1)

println(factorial(10)) // 3628800

Yukarıdaki örnekte, verilen koşul sağlanana kadar ‘faktöriyel’ işlevi kendi başına tekrar tekrar çağrılacaktır.

Döngüler #

while ve do-while döngüleri diğer dillerde olduğu gibi çalışır:

while(condition) {
    doSomething()
}

do {
    doSomething()
} while (condition)

Do-while döngüsünde koşul bloğunun, döngü gövdesinde bildirilen değerlere ve değişkenlere erişimi vardır.

Yineleme için işlevsel yapılar #

Kotlin Standart Kitaplığı ayrıca koleksiyonlar üzerinde yinelemeli olarak çalışmak için çok sayıda yararlı işlev sağlar.

Örneğin, bir öğe listesini dönüştürmek için map işlevi kullanılabilir.

val numbers = listOf(1, 2, 3, 4, 5, 6, 7, 8, 9, 0)
val numberStrings = numbers.map { "Number $it" }

Bu tarzın birçok avantajından biri de benzer bir şekilde zincirleme işlemlere izin vermesidir. Yukarıdaki listenin çift sayılar için filtrelenmesi gerekseydi, sadece küçük bir değişiklik gerekli olurdu. filtre işlevi kullanılabilir.

val numbers = listOf(1, 2, 3, 4, 5, 6, 7, 8, 9, 0)
val numberStrings = numbers.filter { it % 2 == 0 }.map { "Number $it" }
