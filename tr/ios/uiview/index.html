Sözdizimi #

  1. // Amaç-C

  2. [UIView new] // Ayrılmış ve başlatılmış bir görünüm nesnesi alın

  3. [[UIView alloc] initWithFrame:(Pass CGRect)] // Ayrılan ve bir çerçeve ile başlatılan görünümü alın

  4. [[UIView alloc] init] // Ayrılmış ve başlatılmış bir görünüm nesnesi alın

  5. // Hızlı

  6. UIView() // CGRect.zero çerçeveli bir UIView örneği oluşturur

  7. UIView(frame: CGRect) // Çerçeveyi belirten bir UIView örneği oluşturur

  8. UIView.addSubview(UIView) // alt görünüm olarak başka bir UIView örneği ekleyin

  9. UIView.hidden // Görünümün görünürlüğünü alın veya ayarlayın

  10. UIView.alpha // Görünümün opaklığını alın veya ayarlayın

  11. UIView.setNeedsLayout() // Görünümü düzenini güncellemeye zorlar

UIView sınıfı, ekranda dikdörtgen bir alan ve bu alandaki içeriği yönetmek için arayüzler tanımlar. Çalışma zamanında, bir görünüm nesnesi, alanındaki herhangi bir içeriğin işlenmesini ve ayrıca bu içerikle olan tüm etkileşimleri yönetir.

Görünümü yuvarlatın #

Yuvarlatılmış bir “UIView” yapmak için, görünümün “katmanı” için bir “cornerRadius” belirtin.

Bu aynı zamanda ‘UIImageView’ gibi ‘UIView’dan devralan herhangi bir sınıf için de geçerlidir.

Programlı olarak #

Swift kodu

someImageView.layoutIfNeeded()
someImageView.clipsToBounds = true
someImageView.layer.cornerRadius = 10

Hedef-C Kodu

[someImageView layoutIfNeeded];
someImageView.clipsToBounds = YES;
someImageView.layer.cornerRadius = 10;

Örnek

//Swift code
topImageView.layoutIfNeeded()
bottomImageView.layoutIfNeeded()
topImageView.clipsToBounds = true
topImageView.layer.cornerRadius = 10
bottomImageView.clipsToBounds = true
bottomImageView.layer.cornerRadius = bottomImageView.frame.width / 2

//Objective-C code
[topImageView layoutIfNeeded]
[bottomImageView layoutIfNeeded];
topImageView.clipsToBounds = YES;
topImageView.layer.cornerRadius = 10;
bottomImageView.clipsToBounds = YES;
bottomImageView.cornerRadius = CGRectGetWidth(bottomImageView.frame) / 2;

Storyboard Yapılandırması #

Storyboard‘da karşılık gelen özellikleri ayarlayarak “programatik olmayan” bir yuvarlak görünüm efekti de elde edilebilir.

“layer” özellikleri Storyboard’da gösterilmediğinden, “cornerRadius” özniteliğini Kullanıcı Tanımlı Çalışma Zamanı Nitelikleri bölümü aracılığıyla değiştirmeniz gerekir.

Swift Uzantısı #

Bu kullanışlı uzantıyı, aynı genişlik ve yüksekliğe sahip olduğu sürece yuvarlak görünüm uygulamak için kullanabilirsiniz.

extension UIView {
    @discardableResult
    public func setAsCircle() -> Self {
        self.clipsToBounds = true
        let frameSize = self.frame.size
        self.layer.cornerRadius = min(frameSize.width, frameSize.height) / 2.0
        return self
    }
}

Kullanmak için:

yourView.setAsCircle()

IBInspectable ve IBDesignable’ı Kullanma #

Son Xcode sürümlerindeki en havalı yeni özelliklerden biri (veya ikisi) ‘IBInspectable’ özellikleri ve ‘IBDesignable’ ‘UIView’ özellikleridir. Bunların uygulamanızın işlevselliği ile ilgisi yoktur, bunun yerine Xcode’daki geliştirici deneyimini etkiler. Amaç, iOS uygulamanızdaki özel görünümleri çalıştırmadan görsel olarak inceleyebilmektir. Bu nedenle, “UIView"dan devralan, yaratıcı bir şekilde “CustomView” adlı özel bir görünümünüz olduğunu varsayalım. Bu özel görünümde, belirlenmiş bir renge sahip bir metin dizisi görüntüleyecektir. Ayrıca herhangi bir metni görüntülememeyi de seçebilirsiniz. Üç özelliğe ihtiyacımız olacak:

var textColor: UIColor = UIColor.blackColor()
var text: String?
var showText: Bool = true

Daha sonra sınıftaki ‘drawRect’ işlevini geçersiz kılabiliriz:

if showText {
    if let text = text {
        let s = NSString(string: text)
        s.drawInRect(rect,
            withAttributes: [
                NSForegroundColorAttributeName: textColor,
                NSFontAttributeName: UIFont(name: "Helvetica Neue", size: 18)!
            ])
    }
}

text özelliğinin ayarlandığını varsayarsak, bu, uygulama çalıştırıldığında görünümün sol üst köşesine bir dize çizecektir. Sorun şu ki, uygulamayı çalıştırmadan neye benzediğini bilemeyiz. İşte burada ‘IBInspectable’ ve ‘IBDesignable’ devreye girer. ‘IBInspectable’, tıpkı yerleşik kontrollerde olduğu gibi Xcode’da görünümün özellik değerlerini görsel olarak ayarlamamızı sağlar. “IBDesignable” bize storyboard’da görsel bir önizleme gösterecek. İşte sınıfın nasıl görünmesi gerektiği:

@IBDesignable
class CustomView: UIView {
    @IBInspectable var textColor: UIColor = UIColor.blackColor()
    @IBInspectable var text: String?
    @IBInspectable var showText: Bool = true

    override func drawRect(rect: CGRect) {
        // ...
    }
}

Veya Amaç C’de:

IB_DESIGNABLE
@interface CustomView: UIView

@property (nonatomic, strong) IBInspectable UIColor* textColor;
@property (nonatomic, strong) IBInspectable NSString* text;
@property (nonatomic, assign) IBInspectable BOOL showText;

@end

@implementation CustomView

- (instancetype)init {
    if(self = [super init]) {
        self.textColor = [UIColor blackColor];
        self.showText = YES;
    }
    return self;
}

- (void)drawRect:(CGRect)rect {
    //...
}

@end

Sonraki ekran görüntüleri Xcode’da ne olduğunu gösterir. Birincisi, revize edilmiş sınıfı ekledikten sonra olan şeydir. Üç özellik için üç yeni UI öğesi olduğuna dikkat edin. Metin Rengi bir renk seçici görüntüler, Metin yalnızca bir giriş kutusudur ve Metni Göster bize sırasıyla “yanlış” ve “doğru” olan “Kapalı” ve “Açık” seçeneklerini sunar.

Sonraki, renk seçiciyi kullanarak Metin Rengini kırmızıya değiştirdikten sonradır. Ayrıca, ‘drawRect’ işlevinin onu göstermesi için bazı metinler sağlanmıştır. Arayüz Oluşturucu’daki görünümün de güncellendiğine dikkat edin.

Son olarak, özellik denetçisinde Metin Göster‘i ‘Kapalı’ olarak ayarlamak, Arayüz Oluşturucu’daki metin görüntüsünün kaybolmasını sağlar.

Ancak, ‘Storyboard’unuzda birden çok görünümde yuvarlak ‘UIView’ oluşturmamız gerektiğinde hepimiz bir durumla karşı karşıyayız. ‘Storyboard’un her görünümüne ‘IBDesignable’ bildirmek yerine, ‘UIView’un bir ‘Uzantısı’ oluşturmak daha iyidir ve köşe yarıçapını ayarlayarak yuvarlak görünüm oluşturmak için proje boyunca her ‘UIView’ için oluşturulmuş bir kullanıcı arabirimi edinin. Film şeridinde oluşturduğunuz herhangi bir UIView üzerinde yapılandırılabilir bir sınır yarıçapı.

extension UIView {
    
    @IBInspectable var cornerRadius:CGFloat {
        set {
            layer.cornerRadius = newValue
            clipsToBounds = newValue > 0
        }
        get {
            return layer.cornerRadius
        }
    }
    
}

Anlık görüntü alma #

Bunun gibi bir ‘UIView’dan anlık görüntü alabilirsiniz:

Süratli

let snapshot = view.snapshotView(afterScreenUpdates: true)

Hedef-C

UIView *snapshot = [view snapshotViewAfterScreenUpdates: YES];

Bir UIView oluşturun #

Hedef-C

CGRect myFrame = CGRectMake(0, 0, 320, 35)
UIView *view = [[UIView alloc] initWithFrame:myFrame];

//Alternative way of defining the frame
UIView *view = [[UIView alloc] init];
CGRect myFrame = view.frame;
myFrame.size.width = 320;
myFrame.size.height = 35;
myFrame.origin.x = 0;
myFrame.origin.y = 0;
view.frame = myFrame;

Süratli

let myFrame = CGRect(x: 0, y: 0, width: 320, height: 35)
let view = UIView(frame: myFrame)

Bir Görünümü Sallayın #

extension UIView {
    func shake() {
        let animation = CAKeyframeAnimation(keyPath: "transform.translation.x")
        animation.timingFunction = CAMediaTimingFunction(name: kCAMediaTimingFunctionLinear)
        animation.duration = 0.6
        animation.values = [-10.0, 10.0, -7.0, 7.0, -5.0, 5.0, 0.0 ]
        layer.add(animation, forKey: "shake")
    }
}

Bu işlev, biraz sallayarak belirli bir görünüme dikkat çekmek için kullanılabilir.

İç İçerik Boyutunu Kullanma #

Bir UIView alt sınıfı oluştururken, gerçek içerik boyutu, sabit kodlanmış yükseklik ve genişlik kısıtlamalarının ayarlanmasını önlemeye yardımcı olur

bir sınıfın bunu nasıl kullanabileceğine dair temel bir bakış

class ImageView: UIView {
    var image: UIImage {
        didSet {
            invalidateIntrinsicContentSize()
        }
    }
    // omitting initializers
    // convenience init(image: UIImage)

    override func intrinsicContentSize() -> CGSize {
        return CGSize(width: image.size.width, height: image.size.height)
    }
}

Yalnızca bir boyut sağlamak istiyorsanız, yok saymak istediğiniz değer için “UIViewNoIntrinsicMetric” değerini sağlayabilirsiniz.

 override func intrinsicContentSize() -> CGSize {
     return CGSize(width: UIViewNoIntrinsicMetric, height: image.size.width)   
 }

AutoLayout ve Interface Builder ile kullanıldığında avantajlar

Biri bu ImageView (veya UIImageView) alabilir ve yatay hizalamayı denetleme merkezi X’e ve dikey hizalamayı denetleme merkezi Y’ye ayarlayabilir.

Başka bir UIView’a ve başka bir UIView’dan UIView ekleme ve silme işlemlerini programlı olarak yönetin #

Programlı olarak yeni bir “alt Görünüm” eklemek istediğiniz bir “parentView"a sahip olduğunuzu varsayalım (örneğin, bir “UIViewController"ın görünümüne bir “UIImageView” eklemek istediğinizde), bunu aşağıdaki gibi yapabilirsiniz.

Hedef-C

[parentView addSubview:subView];

Süratli

parentView.addSubview(subView)

Alt Görünümü, aşağıdaki kodu kullanarak zaten parentView’ın bir alt görünümü olan başka bir “subView2"nin altına da ekleyebilirsiniz:

Hedef-C

[parentView insertSubview:subView belowSubview:subView2];

Süratli

parentView.insertSubview(subView, belowSubview: subView2)

SubView2’nin üstüne eklemek isterseniz, bunu şu şekilde yapabilirsiniz:

Hedef-C

[parentView insertSubview:subView aboveSubview:subView2];

Süratli

parentView.insertSubview(subView, aboveSubview: subView2)

Kodunuzda bir yerde belirli bir “alt Görünüm"ü öne getirmeniz gerekiyorsa, bu nedenle diğerlerinin hepsinden önce “parentView"ün alt görünümlerini şu şekilde yapabilirsiniz:

Hedef-C

[parentView bringSubviewToFront:subView];

Süratli

parentView.bringSubviewToFront(subView)

Son olarak, ‘parentView’dan ‘subView’ öğesini kaldırmak istiyorsanız, aşağıdakileri yapabilirsiniz:

Hedef-C

[subView removeFromSuperview];

Süratli

subView.removeFromSuperview()

Bir UIView’u Canlandırma #

let view = UIView(frame: CGRect(x: 0, y: 0, width: 100, height: 100))
view.backgroundColor = UIColor.orange
self.view.addSubview(view)
UIView.animate(withDuration: 0.75, delay: 0.5, options: .curveEaseIn, animations: {
    //This will cause view to go from (0,0) to
    // (self.view.frame.origin.x,self.view.frame.origin.y)
    view.frame.origin.x = self.view.frame.origin.x
    view.frame.origin.y = self.view.frame.origin.y
}) { (finished) in
    view.backgroundColor = UIColor.blueColor()
}

Boyut ve çerçeve özellikleri için UIView uzantısı #

Görünümün orijininin x koordinatını almak istiyorsak, şöyle yazmamız gerekir:

view.frame.origin.x

Genişlik için şunu yazmamız gerekiyor:

view.frame.size.width

Ancak bir ‘UIView’a basit bir uzantı eklersek, tüm özellikleri çok basit bir şekilde elde edebiliriz, örneğin:

view.x
view.y
view.width
view.height

Ayrıca şu özelliklerin ayarlanmasına da yardımcı olacaktır:

view.x = 10
view.y = 10
view.width = 100
view.height = 200

Ve basit uzantı şöyle olurdu:

extension UIView {
    
    var x: CGFloat {
        get {
            return self.frame.origin.x
        }
        set {
            self.frame = CGRect(x: newValue, y: self.frame.origin.y, width: self.frame.size.width, height: self.frame.size.height)
        }
    }
    
    var y: CGFloat {
        get {
            return self.frame.origin.y
        }
        set {
            self.frame = CGRect(x: self.frame.origin.x, y: newValue, width: self.frame.size.width, height: self.frame.size.height)
        }
    }
    
    var width: CGFloat {
        get {
            return self.frame.size.width
        }
        set {
            self.frame = CGRect(x: self.frame.origin.x, y: self.frame.origin.y, width: newValue, height: self.frame.size.height)
        }
    }
    
    var height: CGFloat {
        get {
            return self.frame.height
        }
        set {
            self.frame = CGRect(x: self.frame.origin.x, y: self.frame.origin.y, width: self.frame.size.width, height: newValue)
        }
    }
}

Bu sınıf dosyasını bir projeye eklememiz gerekiyor ve proje boyunca kullanıma hazır olacak!

Otomatik Düzeni kullanarak UIView oluşturun #

UIView *view = [[UIView alloc] init];

[self.view addSubview:view];

//Use the function if you want to use height as constraint
[self addView:view onParentView:self.view withHeight:200.f];


//Use this function if you want to add view with respect to parent and should resize with it
[self addFullResizeConstraintForSubview:view addedOnParentView:self.view];

İşlevler

Otomatik yerleşim kısıtlamalarını kullanarak sabit yükseklikte görünüm ekleme işlevi

-(void)addView:(UIView*)subView onParentView:(UIView*)parentView withHeight:(CGFloat)height{

subView.translatesAutoresizingMaskIntoConstraints = NO;
    
NSLayoutConstraint *trailing =[NSLayoutConstraint
                                   constraintWithItem:subView
                                   attribute:NSLayoutAttributeTrailing
                                   relatedBy:NSLayoutRelationEqual
                                   toItem:parent
                                   attribute:NSLayoutAttributeTrailing
                                   multiplier:1.0
                                   constant:10.f];


NSLayoutConstraint *top = [NSLayoutConstraint
                               constraintWithItem:subView
                               attribute:NSLayoutAttributeTop
                               relatedBy:NSLayoutRelationEqual
                               toItem:parent
                               attribute:NSLayoutAttributeTop
                               multiplier:1.0
                               constant:10.f];
    
NSLayoutConstraint *leading = [NSLayoutConstraint
                                   constraintWithItem:subView
                                   attribute:NSLayoutAttributeLeading
                                   relatedBy:NSLayoutRelationEqual
                                   toItem:parent
                                   attribute:NSLayoutAttributeLeading
                                   multiplier:1.0
                                   constant:10.f];
[parent addConstraint:trailing];
[parent addConstraint:top];
[parent addConstraint:leading];

NSLayoutConstraint *heightConstraint =[NSLayoutConstraint
                                           constraintWithItem:subView
                                           attribute:NSLayoutAttributeHeight
                                           relatedBy:NSLayoutRelationEqual
                                           toItem:nil
                                           attribute:0
                                           multiplier:0.0
                                           constant:height];
    
[subView addConstraint:heightConstraint];

}

İşlev, oluşturulan UIView için tam yeniden boyutlandırma kısıtlaması ekler.

-(void)addFullResizeConstraintForSubview:(UIView*)subView addedOnParentView:(UIView*)parentView{

subView.translatesAutoresizingMaskIntoConstraints = NO;
    
NSLayoutConstraint *trailing =[NSLayoutConstraint
                                   constraintWithItem:subView
                                   attribute:NSLayoutAttributeTrailing
                                   relatedBy:NSLayoutRelationEqual
                                   toItem:parent
                                   attribute:NSLayoutAttributeTrailing
                                   multiplier:1.0
                                   constant:10.f];


NSLayoutConstraint *top = [NSLayoutConstraint
                               constraintWithItem:subView
                               attribute:NSLayoutAttributeTop
                               relatedBy:NSLayoutRelationEqual
                               toItem:parent
                               attribute:NSLayoutAttributeTop
                               multiplier:1.0
                               constant:10.f];
    
NSLayoutConstraint *leading = [NSLayoutConstraint
                                   constraintWithItem:subView
                                   attribute:NSLayoutAttributeLeading
                                   relatedBy:NSLayoutRelationEqual
                                   toItem:parent
                                   attribute:NSLayoutAttributeLeading
                                   multiplier:1.0
                                   constant:10.f];

NSLayoutConstraint *bottom =[NSLayoutConstraint
                                 constraintWithItem:subView
                                 attribute:NSLayoutAttributeBottom
                                 relatedBy:NSLayoutRelationEqual
                                 toItem:parent
                                 attribute:NSLayoutAttributeBottom
                                 multiplier:1.0
                                 constant:0.f];
[parent addConstraint:trailing];
[parent addConstraint:top];
[parent addConstraint:leading];
[parent addConstraint:bottom];
}
