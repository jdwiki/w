Sözdizimi #

  • kSecClassGenericPassword // İnternet dışı bir parolayı temsil eden bir değer anahtarı
  • kSecClassInternetPassword // İnternet şifresini temsil eden bir değer anahtarı
  • kSecClassCertificate // Sertifikayı temsil eden bir değer anahtarı
  • kSecClassCertificate // Bir anahtarı temsil eden bir değer anahtarı
  • kSecClassIdentity // Sertifika artı bir anahtar olan bir kimliği temsil eden bir değer anahtarı

iOS, parolalar, şifreleme anahtarları, sertifikalar ve kimlikler gibi özel bilgileri Anahtar Zinciri adı verilen güvenli bir depolama alanında saklar. Bu depolama alanı, uygulama işlemcisinin içine gömülü olan Secure Enclave adlı bir yardımcı işlemci tarafından tamamen yönetilir. Anahtarlık iOS’ta korumalı alan olduğundan, anahtarlık öğeleri yalnızca onları oraya koyan uygulama tarafından alınabilir.

Bazı durumlarda, hataları önlemek için Xcode yeteneklerinde Anahtar Zinciri Paylaşımını açmanız gerekir.

Anahtarlık ile etkileşim kurmak için Anahtarlık Hizmetleri adı verilen bir c çerçevesi kullanıyoruz. Daha fazla bilgi için Apple’ın Anahtar Zinciri Hizmetleri Programlama Kılavuzuna bakın. ).

Anahtar Zinciri Hizmetleri ‘Foundation’ seviyesinin altında olduğundan, ‘CoreFoundation’ türlerini kullanmakla sınırlıdır. Sonuç olarak, çoğu nesne dahili olarak “CFDictionary” anahtarları olarak “CFString"leri tutan ve değerleri olarak çeşitli “CoreFoundation” türleri olarak temsil edilir.

Anahtarlık Hizmetleri, “Güvenlik” çerçevesinin bir parçası olarak dahil edilmiş olsa da, arka uçta bazı yardımcı işlevler içerdiğinden, “Vakıf"ı içe aktarmak genellikle iyi bir seçenektir.

Ayrıca, doğrudan Anahtar Zinciri Hizmetleri ile uğraşmak istemiyorsanız, Apple Genel Anahtarlık sağlar. /doc/uid/DTS40007797) Arka planda Anahtar Zinciri Hizmetlerini kullanan Swift türleri sağlayan Swift örnek projesi.

Anahtarlığa Parola Ekleme #

Her Anahtarlık Öğesi çoğunlukla bir “CFDictionary” olarak temsil edilir. Bununla birlikte, Objective-C’de ‘NSDictionary’yi kullanabilir ve köprülemeden yararlanabilirsiniz veya Swift’de ‘Sözlük’ kullanabilir ve açıkça ‘CFDictionary’ye aktarabilirsiniz.

Aşağıdaki sözlükle bir şifre oluşturabilirsiniz:

Süratli #

var dict = [String : AnyObject]()

İlk olarak, Anahtarlık’ın bunun bir parola olduğunu bilmesini sağlayan bir anahtar/değer çiftine ihtiyacınız var. Dikt anahtarımız bir “Dize” olduğundan, herhangi bir “CFString"i Swift 3’te açıkça bir “Dize"ye atamamız gerektiğine dikkat edin. CFString, Hashable olmadığı için Swift Sözlüğünün anahtarı olarak kullanılamaz.

Süratli #

dict[kSecClass as String] = kSecClassGenericPassword

Daha sonra, parolamızın onu tanımlayacak ve daha sonra bulmamıza yardımcı olacak bir dizi özelliği olabilir. Genel şifreler için özelliklerin bir listesi.

Süratli #

// The password will only be accessible when the device is unlocked
dict[kSecAttrAccessible as String] = kSecAttrAccessibleWhenUnlocked
// Label may help you find it later
dict[kSecAttrLabel as String] = "com.me.myapp.myaccountpassword" as CFString
// Username
dict[kSecAttrAccount as String] = "My Name" as CFString
// Service name
dict[kSecAttrService as String] = "MyService" as CFString

Son olarak, gerçek özel verilerimize ihtiyacımız var. Bunu bellekte çok uzun süre tutmadığınızdan emin olun. Bu “CFData” olmalıdır.

Süratli #

dict[kSecValueData as String] = "my_password!!".data(using: .utf8) as! CFData

Son olarak, Anahtarlık Hizmetleri ekleme işlevi, yeni oluşturulan anahtarlık öğesini nasıl döndürmesi gerektiğini bilmek ister. Verileri bellekte çok uzun süre tutmamanız gerektiğinden, yalnızca öznitelikleri şu şekilde döndürebilirsiniz:

Süratli #

dict[kSecReturnAttributes as String] = kCFBooleanTrue

Şimdi eşyamızı oluşturduk. Ekleyelim:

Süratli #

var result: AnyObject?
let status = withUnsafeMutablePointer(to: &result) {
    SecItemAdd(dict as CFDictionary, UnsafeMutablePointer($0))
}
let newAttributes = result as! Dictionary<String, AnyObject>

Bu, yeni öznitelikleri dict ‘sonuç’ içine yerleştirir. SecItemAdd, oluşturduğumuz sözlüğün yanı sıra sonucumuzun nerede olmasını istediğimize dair bir işaretçi alır. Ardından işlev, başarıyı veya bir hata kodunu belirten bir “OSStatus” döndürür. Sonuç kodları burada açıklanmıştır.

Anahtar Zinciri Erişim Kontrolü (parola geri dönüşlü TouchID) #

Anahtarlık, öğelerin yalnızca kullanıcının Touch ID (veya bu tür bir geri dönüşe izin veriliyorsa parola) ile kimliği doğrulandıktan sonra Anahtarlıktan öğe alınmasına izin verecek özel SecAccessControl özniteliği ile kaydedilmesine olanak tanır. Uygulamaya yalnızca kimlik doğrulamanın başarılı olup olmadığı bildirilir, tüm kullanıcı arayüzü iOS tarafından yönetilir.

İlk olarak, SecAccessControl nesnesi oluşturulmalıdır:

Süratli #

let error: Unmanaged<CFError>?

guard let accessControl = SecAccessControlCreateWithFlags(kCFAllocatorDefault, kSecAttrAccessibleWhenPasscodeSetThisDeviceOnly, .userPresence, &error) else {
    fatalError("Something went wrong")
}

Ardından, kSecAttrAccessControl anahtarıyla (diğer örneklerde kullanmakta olduğunuz kSecAttrAccessible anahtarıyla birbirini dışlayan) sözlüğe ekleyin:

Süratli #

var dictionary = [String : Any]()

dictionary[kSecClass as String] = kSecClassGenericPassword
dictionary[kSecAttrLabel as String] = "com.me.myapp.myaccountpassword" as CFString
dictionary[kSecAttrAccount as String] = "My Name" as CFString
dictionary[kSecValueData as String] = "new_password!!".data(using: .utf8) as! CFData
dictionary[kSecAttrAccessControl as String] = accessControl

Ve daha önce yaptığınız gibi kaydedin:

Süratli #

let lastResultCode = SecItemAdd(query as CFDictionary, nil)

Depolanan verilere erişmek için bir anahtar için Anahtarlık’ı sorgulamanız yeterlidir. Anahtar Zinciri Hizmetleri, kullanıcıya kimlik doğrulama diyaloğunu sunacak ve uygun parmak izinin sağlanıp sağlanmadığına veya parola eşleşip eşleşmediğine bağlı olarak veri veya sıfır döndürecektir.

İsteğe bağlı olarak, bilgi istemi dizesi belirtilebilir:

Süratli #

var query = [String: Any]()

query[kSecClass as String] = kSecClassGenericPassword
query[kSecReturnData as String] = kCFBooleanTrue
query[kSecAttrAccount as String] = "My Name" as CFString
query[kSecAttrLabel as String] = "com.me.myapp.myaccountpassword" as CFString
query[kSecUseOperationPrompt as String] = "Please put your fingers on that button" as CFString

var queryResult: AnyObject?
let status = withUnsafeMutablePointer(to: &queryResult) {
    SecItemCopyMatching(query as CFDictionary, UnsafeMutablePointer($0))
}

Kullanıcının yetkilendirmeyi reddetmesi, iptal etmesi veya başarısız olması durumunda ‘status’un ’err’ olacağına dikkat edin.

Süratli #

if status == noErr {
    let password = String(data: queryResult as! Data, encoding: .utf8)!
    print("Password: \(password)")
} else {
    print("Authorization not passed")
}

Anahtar Zincirinde Parola Bulma #

Bir sorgu oluşturmak için onu bir ‘CFDictionary’ olarak temsil etmemiz gerekir. Ayrıca Objective-C’de ‘NSDictionary’ veya Swift’de ‘Sözlük’ kullanabilir ve ‘CFDictionary’ye aktarabilirsiniz.

Bir sınıf anahtarına ihtiyacımız var:

Süratli #

var dict = [String : AnyObject]()
dict[kSecClass as String] = kSecClassGenericPassword

Ardından, aramamızı daraltmak için nitelikler belirleyebiliriz:

Süratli #

// Label
dict[kSecAttrLabel as String] = "com.me.myapp.myaccountpassword" as CFString
// Username
dict[kSecAttrAccount as String] = "My Name" as CFString
// Service name
dict[kSecAttrService as String] = "MyService" as CFString

Burada açıklanan özel arama değiştirici anahtarları da belirtebiliriz.

Son olarak, verilerimizin nasıl iade edilmesini istediğimizi söylememiz gerekiyor. Aşağıda, yalnızca özel parolanın kendisinin bir “CFData” nesnesi olarak döndürülmesini isteyeceğiz:

Süratli #

dict[kSecReturnData as String] = kCFBooleanTrue

Şimdi arayalım:

Süratli #

var queryResult: AnyObject?
let status = withUnsafeMutablePointer(to: &queryResult) {
    SecItemCopyMatching(dict as CFDictionary, UnsafeMutablePointer($0))
}
// Don't keep this in memory for long!!
let password = String(data: queryResult as! Data, encoding: .utf8)!

Burada, “SecItemCopyMatching” bir sorgu sözlüğü ve sonucun nereye gitmesini istediğinize dair bir işaretçi alır. Sonuç kodlarıyla birlikte bir “OSStatus” döndürür. Burada olasılıklardır.

Anahtar Zincirinde Parola Güncelleme #

Her zamanki gibi, güncellemek istediğimiz öğeyi temsil edecek bir ‘CFDictionary’ye ihtiyacımız var. Bu, eski özel veriler de dahil olmak üzere öğenin tüm eski değerlerini içermelidir. Ardından, değiştirmek istediğiniz herhangi bir özelliğin veya verilerin kendisinin bir ‘CFDictionary’sini alır.

İlk önce, bir sınıf anahtarı ve bir nitelik listesi oluşturalım. Bu öznitelikler aramamızı daraltabilir, ancak değiştirecekseniz tüm öznitelikleri ve orada eski değerleri eklemelisiniz.

Süratli #

var dict = [String : AnyObject]()
dict[kSecClass as String] = kSecClassGenericPassword
// Label
dict[kSecAttrLabel as String] = "com.me.myapp.myaccountpassword" as CFString
// Username
dict[kSecAttrAccount as String] = "My Name" as CFString

Şimdi eski verileri eklemeliyiz:

Süratli #

dict[kSecValueData as String] = "my_password!!".data(using: .utf8) as! CFData

Şimdi aynı öznitelikleri ancak farklı bir şifre oluşturalım:

Süratli #

var newDict = [String : AnyObject]()
newDict[kSecClass as String] = kSecClassGenericPassword
// Label
newDict[kSecAttrLabel as String] = "com.me.myapp.myaccountpassword" as CFString
// Username
newDict[kSecAttrAccount as String] = "My Name" as CFString
// New password
newDict[kSecValueData as String] = "new_password!!".data(using: .utf8) as! CFData

Şimdi, onu Anahtarlık Hizmetlerine iletiyoruz:

Süratli #

let status = SecItemUpdate(dict as CFDictionary, newDict as CFDictionary)

SecItemUpdate bir durum kodu döndürür. Sonuçlar burada açıklanmıştır.

Anahtar Zincirinden Parolayı Kaldırma #

Anahtar Zincirinden bir öğeyi silmek için tek bir şeye ihtiyacımız var: silinecek öğeleri tanımlayan özniteliklere sahip bir “CFDictionary”. Sorgu sözlüğüyle eşleşen tüm öğeler kalıcı olarak silinecektir, bu nedenle yalnızca tek bir öğeyi silmeyi planlıyorsanız, sorgunuzla ilgili spesifik olduğunuzdan emin olun. Her zaman olduğu gibi, Objective-C’de bir ‘NSDictionary’ kullanabiliriz veya Swift’de bir ‘Sözlük’ kullanabilir ve ardından ‘CFDictionary’ye aktarabiliriz.

Bu bağlamda bir sorgu sözlüğü, yalnızca öğenin ne olduğunu açıklamak için bir sınıf anahtarı ve öğeyle ilgili bilgileri açıklamak için öznitelikler içerir. “kSecMatchCaseInSense” gibi arama kısıtlamalarının dahil edilmesine izin verilmez.

Süratli #

var dict = [String : AnyObject]()
dict[kSecClass as String] = kSecClassGenericPassword
// Label
dict[kSecAttrLabel as String] = "com.me.myapp.myaccountpassword" as CFString
// Username
dict[kSecAttrAccount as String] = "My Name" as CFString

Ve şimdi basitçe kaldırabiliriz:

Süratli #

let status = SecItemDelete(dict as CFDictionary)

SecItemDelete bir OSStatus döndürür. Sonuç kodları burada açıklanmıştır.

Anahtarlık.h

#import <Foundation/Foundation.h>
typedef void (^KeychainOperationBlock)(BOOL successfulOperation, NSData *data, OSStatus status);

@interface Keychain : NSObject

-(id) initWithService:(NSString *) service_ withGroup:(NSString*)group_;

-(void)insertKey:(NSString *)key withData:(NSData *)data withCompletion:(KeychainOperationBlock)completionBlock;
-(void)updateKey:(NSString*)key withData:(NSData*) data withCompletion:(KeychainOperationBlock)completionBlock;
-(void)removeDataForKey:(NSString*)key withCompletionBlock:(KeychainOperationBlock)completionBlock;
-(void)findDataForKey:(NSString*)key withCompletionBlock:(KeychainOperationBlock)completionBlock;

@end

Anahtarlık.m

#import "Keychain.h"
#import <Security/Security.h>

@implementation Keychain

{
    NSString * keychainService;
    NSString * keychainGroup;
}

-(id) initWithService:(NSString *)service withGroup:(NSString*)group
{
    self =[super init];
    if(self) {
        keychainService = [NSString stringWithString:service];
        if(group) {
            keychainGroup = [NSString stringWithString:group];
        }
    }
    
    return  self;
}

-(void)insertKey:(NSString *)key
        withData:(NSData *)data
  withCompletion:(KeychainOperationBlock)completionBlock
{
    NSMutableDictionary * dict =[self prepareDict:key];
    [dict setObject:data forKey:(__bridge id)kSecValueData];
    [dict setObject:keychainService forKey:(id)kSecAttrService];
    
    OSStatus status = SecItemAdd((__bridge CFDictionaryRef)dict, NULL);
    if(errSecSuccess != status) {
        DLog(@"Unable add item with key =%@ error:%d",key,(int)status);
        if (completionBlock) {
            completionBlock(errSecSuccess == status, nil, status);
        }
    }
    if (status == errSecDuplicateItem) {
        [self updateKey:key withData:data withCompletion:^(BOOL successfulOperation, NSData *updateData, OSStatus updateStatus) {
            if (completionBlock) {
                completionBlock(successfulOperation, updateData, updateStatus);
            }
            DLog(@"Found duplication item -- updating key with data");
        }];
    }
}

-(void)findDataForKey:(NSString *)key
  withCompletionBlock:(KeychainOperationBlock)completionBlock
{
    NSMutableDictionary *dict = [self prepareDict:key];
    [dict setObject:(__bridge id)kSecMatchLimitOne forKey:(__bridge id)kSecMatchLimit];
    [dict setObject:keychainService forKey:(id)kSecAttrService];
    [dict setObject:(id)kCFBooleanTrue forKey:(__bridge id)kSecReturnData];
    CFTypeRef result = NULL;
    OSStatus status = SecItemCopyMatching((__bridge CFDictionaryRef)dict,&result);
    
    if( status != errSecSuccess) {
        DLog(@"Unable to fetch item for key %@ with error:%d",key,(int)status);
        if (completionBlock) {
            completionBlock(errSecSuccess == status, nil, status);
        }
    } else {
        if (completionBlock) {
            completionBlock(errSecSuccess == status, (__bridge NSData *)result, status);
        }
    }
}

-(void)updateKey:(NSString *)key
        withData:(NSData *)data
  withCompletion:(KeychainOperationBlock)completionBlock
{
    NSMutableDictionary * dictKey =[self prepareDict:key];
    
    NSMutableDictionary * dictUpdate =[[NSMutableDictionary alloc] init];
    [dictUpdate setObject:data forKey:(__bridge id)kSecValueData];
    [dictUpdate setObject:keychainService forKey:(id)kSecAttrService];
    OSStatus status = SecItemUpdate((__bridge CFDictionaryRef)dictKey, (__bridge CFDictionaryRef)dictUpdate);
    if( status != errSecSuccess) {
        DLog(@"Unable to remove item for key %@ with error:%d",key,(int)status);
    }
    if (completionBlock) {
        completionBlock(errSecSuccess == status, nil, status);
    }
}

-(void)removeDataForKey:(NSString *)key
    withCompletionBlock:(KeychainOperationBlock)completionBlock {
    NSMutableDictionary *dict = [self prepareDict:key];
    OSStatus status = SecItemDelete((__bridge CFDictionaryRef)dict);
    if( status != errSecSuccess) {
        DLog(@"Unable to remove item for key %@ with error:%d",key,(int)status);
    }
    if (completionBlock) {
        completionBlock(errSecSuccess == status, nil, status);
    }
}

#pragma mark Internal methods

-(NSMutableDictionary*) prepareDict:(NSString *) key {
    
    NSMutableDictionary *dict = [[NSMutableDictionary alloc] init];
    [dict setObject:(__bridge id)kSecClassGenericPassword forKey:(__bridge id)kSecClass];
    
    NSData *encodedKey = [key dataUsingEncoding:NSUTF8StringEncoding];
    [dict setObject:encodedKey forKey:(__bridge id)kSecAttrGeneric];
    [dict setObject:encodedKey forKey:(__bridge id)kSecAttrAccount];
    [dict setObject:keychainService forKey:(__bridge id)kSecAttrService];
    [dict setObject:(__bridge id)kSecAttrAccessibleAlwaysThisDeviceOnly forKey:(__bridge id)kSecAttrAccessible];
    
    //This is for sharing data across apps
    if(keychainGroup != nil) {
        [dict setObject:keychainGroup forKey:(__bridge id)kSecAttrAccessGroup];
    }
    
    return  dict;
}
