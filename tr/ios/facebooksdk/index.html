Kendi özel “Facebook ile Giriş Yap” düğmenizi oluşturma #

Bazen, FacebookSDK ile birlikte gelen orijinal buton yerine “Facebook ile Giriş Yap” butonu için kendi UI’mizi tasarlamak istiyoruz.

  1. Film şeridinizde, UIButton’unuzu sürükleyin ve olmasını istediğiniz gibi ayarlayın.
  2. Ctrl + düğmenizi IBAction olarak görünüm denetleyicinize sürükleyin.
  3. IBAction yönteminin içinde, aşağıdaki gibi gerçek Facebook düğmesine bir dokunuşu simüle edeceksiniz:

Süratli:

    let loginButton = FBSDKLoginButton()
    loginButton.delegate = self
    // Your Custom Permissions Array
    loginButton.readPermissions =
    [
                             "public_profile",
                             "email",
                             "user_about_me",
                             "user_photos"
    ]
    // Hiding the button
    loginButton.hidden = true
    self.view.addSubview(loginButton)
    // Simulating a tap for the actual Facebook SDK button
    loginButton.sendActionsForControlEvents(UIControlEvents.TouchUpInside)

Hedef-C:

FBSDKLoginButton *FBButton = [FBSDKLoginButton new];

// Your Custom Permissions Array
FBButton.readPermissions = @[@"public_profile",
                             @"email",
                             @"user_about_me",
                             @"user_photos"
                             ];
FBButton.loginBehavior = FBSDKLoginBehaviorNative;
[FBButton setDelegate:self];
[FBButton setHidden:true];
[loginButton addSubview:FBButton];

[FBButton sendActionsForControlEvents:UIControlEventTouchUpInside];

Sen bittin.

FacebookSDK Entegrasyonu #

Adım 1: SDK’yı yükleyin

SDK’yı manuel olarak veya CocoaPods aracılığıyla yükleyebilirsiniz. İkinci seçenek şiddetle tavsiye edilir.

Bu satırları ‘Podfile’ içine koyun:

target 'MyApp' do
  use_frameworks!

  pod 'FBSDKCoreKit'
  pod 'FBSDKLoginKit'
  pod 'FBSDKShareKit'
end

Terminalde “pod install” komutunu çalıştırın ve ardından “.xcodeproj” yerine “.xcworkspace” dosyasını açın.

“FBSDKLoginKit” ve “FBSDKShareKit” isteğe bağlıdır. Onlara ihtiyacınız olabilir veya olmayabilir.

2. Adım: Facebook’ta bir uygulama oluşturun

Bir uygulama oluşturmak için Hızlı Başlangıçlar - Geliştiriciler için Facebook‘a gidin.

Facebook, uygulamayı oluşturduktan sonra SDK’yı indirmenizi isteyecektir. SDK’yı zaten CocoaPod’lar aracılığıyla kurduysanız bu bölümü atlayabilirsiniz.

Adım 3: “.plist” dosyasını düzenleyin

a. Uygulamanızın Facebook ile “iletişim kurabilmesi” için, .plist dosyanıza bazı ayarlar koymanız gerekir. Facebook, Hızlı Başlangıçlar sayfasında size özelleştirilmiş snippet’i verecektir.

b. .plist dosyanızı kaynak kodu olarak düzenleyin.

c. Özelleştirilmiş snippet’inizi kaynak koduna yapıştırın. Dikkatli olun! Snippet tam olarak “” etiketinin alt öğesi olmalıdır. Kaynak kodunuz şöyle bir şey olmalıdır:

<plist version="1.0">
<dict>
    // ...

        </dict>

    <key>FacebookAppID</key>
    <string>{FBAppId}</string>

        <string>fbapi</string>
        <string>fb-messenger-api</string>
        <string>fbauth2</string>

    </array>
<key>NSAppTransportSecurity</key>
<dict>

Snippet’i yanlış bir yere yapıştırırsanız, sorunla karşılaşırsınız.

enum FacebookParametesField : String

if FBSDKAccessToken.currentAccessToken() != nil

FBSDKGraphRequest(graphPath: "me", 
                 parameters: [FacebookParametesField.FIELDS_KEY.rawValue : FacebookParametesField.FIELDS_VALUE.rawValue])
