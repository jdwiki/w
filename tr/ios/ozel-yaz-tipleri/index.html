Özel yazı tiplerini gömme #

Özel Yazı Tipi Desteği
Özel yazı tiplerini kullanmak isteyen uygulamalar artık bu yazı tiplerini uygulama paketlerine dahil edebilir ve bu yazı tiplerini Info.plist dosyalarına UIAppFonts anahtarını ekleyerek sisteme kaydedebilir. Bu anahtarın değeri, uygulamanın paketindeki yazı tipi dosyalarını tanımlayan bir dizi dizidir. Sistem anahtarı gördüğünde, belirtilen yazı tiplerini yükler ve bunları uygulamanın kullanımına sunar.

Yazı tipleri Info.plist içinde ayarlandıktan sonra, özel yazı tiplerinizi IB’deki herhangi bir yazı tipi olarak veya programlı olarak kullanabilirsiniz.

  1. Yazı tipinizi Xcode Supporting Files klasörüne sürükleyip bırakın. Uygulamanızı “Hedeflere ekle” bölümünde işaretlemeyi unutmayın. Bu andan itibaren bu fontu IB’de kullanabilir ve font paletinden seçebilirsiniz.

  1. Bu yazı tipini cihazda kullanılabilir hale getirmek için ‘Info.plist’i açın ve ‘Uygulama anahtarı tarafından sağlanan yazı tiplerini’ (UIAppFonts) ekleyin. Öğe 0 anahtarına değer olarak yazı tipi adını ekleyin. Not: Yazı tipi adı, yazı tipi dosya adınızdan farklı olabilir.

  2. Aşağıdaki pasajı kullanarak özel eklenen yazı tipi adını alın

[Hızlı 3]

for family in UIFont.familyNames {
            print("\(family)")

            for name in UIFont.fontNames(forFamilyName: family) {
                print("   \(name)")
            }
        }

[Hedef - C]

for (NSString *familyName in [UIFont familyNames]){
        NSLog(@"Family name: %@", familyName);
        for (NSString *fontName in [UIFont fontNamesForFamilyName:familyName]) {
            NSLog(@"--Font name: %@", fontName);
        }
    }

Storyboard ile Özel Yazı Tipleri #

Film şeridinden UI bileşenleri için Özel Yazı Tipleri, film şeridinde Kullanıcı Tanımlı Çalışma Zamanı Nitelikleri ve Kategoriler ile kolayca elde edilebilir.

Avantajlar şöyle,

  • ui öğesi için çıkışları tanımlamaya gerek yok
  • Öğeler için yazı tipini programlı olarak ayarlamanıza gerek yok.

İzlenecek adımlar

  1. Yazı Tipi Dosyası: Yazı Tipi dosyasını (.ttf) uygulama paketine ekleyin ve yazı tipi girişini Info.plist’te Yazı tipi tarafından sağlanan altına ekleyin. uygulama özel yazı tiplerinin bu belgelerinde olduğu gibi.

  2. Kategorileri Tanımlayın: UIKit+IBExtensions gibi bir dosya ekleyin ve aşağıdakiler için UILabel, UIButton vb. gibi UI öğeleri için kategoriler ekleyin. özel yazı tipini ayarlamak istediğiniz. Tüm kategoriler bir özel mülk fontName der. Bu özel yazı tipini ayarlamak için daha sonra storyboard (4. adımda olduğu gibi).

UIKit+IBExtensions.h #

#import <UIKit/UIKit.h>

//Category extension for UILabel
@interface UILabel (IBExtensions)

@property (nonatomic, copy) NSString *fontName;
@end

// Category extension for UITextField
@interface UITextField (IBExtensions)

@property (nonatomic, copy) NSString *fontName;
@end

// Category extension for UIButton
@interface UIButton (IBExtensions)

@property (nonatomic, copy) NSString *fontName;
@end
  1. Alıcılar ve Ayarlayıcılar: Eklenen her kategori için fontName özelliği için alıcıları ve ayarlayıcıları tanımlayın.

UIKit+IBExtensions.m #

#import "UIKit+IBExtensions.h"

@implementation UILabel (IBExtensions)

- (NSString *)fontName {
    return self.font.fontName;
}

- (void)setFontName:(NSString *)fontName {
    self.font = [UIFont fontWithName:fontName size:self.font.pointSize];
}
@end

@implementation UITextField (IBExtensions)

- (NSString *)fontName {
    return self.font.fontName;
}

- (void)setFontName:(NSString *)fontName {
    self.font = [UIFont fontWithName:fontName size:self.font.pointSize];
}
@end

@implementation UIButton (IBExtensions)

- (NSString *)fontName {
    return self.titleLabel.font.fontName;
}

- (void)setFontName:(NSString *)fontName{
    self.titleLabel.font = [UIFont fontWithName:fontName size:self.titleLabel.font.pointSize];
}
@end
  1. Film şeridinde yazı tipini ayarlama: KeyPath olarak fontName ve Custom’ınız ile Kullanıcı Tanımlı Çalışma Zamanı Niteliklerine bir giriş ekleyin Yazı Tipinin Adı, gösterildiği gibi Dize olarak tür ile değer olarak.

Bu, uygulamayı çalıştırırken özel yazı tipinizi ayarlayacaktır.

Notlar:

  • Lato-Regular, kullandığım özel yazı tipidir.
  • Pakete eklenen .ttf dosyasındaki aynı isim storyboard’da uzantısız kullanılmalıdır.
  • Yazı tipi boyutu, UI öğesinin nitelik denetçisinde tanımlananla aynı olacaktır.

Notlar (Uyarılar) #

Bunun Xcode 7.1.1+ üzerinde çalıştığı doğrulandı. (Aşağıdaki Örneklere bakın)

  1. Yazı tipi etkinleşmeden önce gezinme çubuğu renk tonunu değiştirmeniz gerekir (Xcode’da bir hata gibi görünüyor; varsayılana geri döndürebilirsiniz ve yazı tipi yapışacaktır)
  2. Bir sistem yazı tipi seçerseniz ~ Boyutun 0.0 (Otherwise the new font will be ignored)

  1. Görünürde yalnızca bir NavBar olduğunda bu sorunsuz çalışıyor gibi görünüyor hierarchy. It appears that secondary NavBars in the same stack are ignored. (Note that if you show the master navigation controller’s navBar all the other custom navBar settings are ignored).

Gotchas (iki) #

Bunlardan bazıları tekrar edilmiştir, bu da büyük olasılıkla kayda değer oldukları anlamına gelir.

  1. Bazen storyboard xml bozulur. Bu, şunları yapmanızı gerektirir: review the structure in Storyboard as Source Code mode (right click the storyboard file > Open As …)
  2. Bazı durumlarda, kullanıcı tanımlı çalışma zamanı özniteliğiyle ilişkilendirilen navigationItem etiketi, bir xml olarak ayarlanmıştır. child of the view tag instead of the view controller tag. If so remove it from between the tags for proper operation.
  3. Özel yazı tipinin doğru olduğundan emin olmak için NavBar Tonunu değiştirin. used.
  4. Dinamik bir yazı tipi kullanmadığınız sürece yazı tipinin boyut parametresini doğrulayın style
  5. Görünüm hiyerarşisi, ayarları geçersiz kılar. Görünüşe göre bir yazı tipi per stack is possible.

  3. app-info.plist’te, tarafından sağlanan Yazı Tipleri adlı bir anahtar ekleyin. application.It’s an array type , add all your font file names to the array,note:including the file extension.
