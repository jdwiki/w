iOS’ta yerelleştirme #

Her dil için ayrı bir ‘Localizable.strings’ dosyası oluşturun. Sağ taraf her dil için farklı olacaktır. Bunu bir anahtar/değer çifti olarak düşünün:

"str" = "str-language";

Objective-C’deki str’ye erişin:

//Try to provide description on the localized string to be able to create a proper documentation if needed
NSString *str = NSLocalizedString(@"string", @"description of the string");

Swift’de str’ye erişin:

let str = NSLocalizedString("string", comment: "language");
