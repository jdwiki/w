UIColor Oluşturma #

Bir “UIColor” oluşturmanın birçok yolu vardır:

Süratli

  • Önceden tanımlanmış renklerden birini kullanarak:

    let redColor = UIColor.redColor()
let blueColor: UIColor = .blueColor()

// In Swift 3, the "Color()" suffix is removed:
let redColor = UIColor.red
let blueColor: UIColor = .blue

Derleyici, değişkenin bir “UIColor” örneği olduğunu zaten biliyorsa, türü hep birlikte atlayabilirsiniz:

  let view = UIView()
  view.backgroundColor = .yellowColor()

  • Gri tonlamalı değeri ve alfayı kullanma:

    let grayscaleColor = UIColor(white: 0.5, alpha: 1.0)

  • Ton, doygunluk, parlaklık ve alfa kullanımı:

    let hsbColor = UIColor(
    hue: 0.4,
    saturation: 0.3,
    brightness: 0.7,
    alpha: 1.0
)

  • RGBA değerlerini kullanarak:

    let rgbColor = UIColor(
    red: 30.0 / 255, 
    green: 70.0 / 255, 
    blue: 200.0 / 255, 
    alpha: 1.0
)

  • Bir desen resmi kullanma:

    let patternColor = UIColor(patternImage: UIImage(named: "myImage")!)

Hedef-C

  • Önceden tanımlanmış renklerden birini kullanarak:

    UIColor *redColor = [UIColor redColor];

  • Gri tonlamalı değeri ve alfayı kullanma:

    UIColor *grayscaleColor = [UIColor colorWithWhite: 0.5 alpha: 1.0];

  • Ton, doygunluk, parlaklık ve alfa kullanımı:

    UIColor *hsbColor = [UIColor
    colorWithHue: 0.4
    saturation: 0.3
    brightness: 0.7
    alpha: 1.0
];

  • RGBA değerlerini kullanarak:

    UIColor *rgbColor = [UIColor
    colorWithRed: 30.0 / 255.0
    green: 70.0 / 255.0
    blue: 200.0 / 255.0 
    alpha: 1.0
];

  • Bir desen resmi kullanma:

    UIColor *pattenColor = [UIColor colorWithPatternImage:[UIImage imageNamed:@"myImage.png"]];

Onaltılık sayı veya dizeden bir UIColor oluşturma #

Onaltılık bir sayı veya dizeden bir “UIColor” oluşturabilirsiniz, ör. 0xff00cc, “#FFFFFF”

Süratli

Int Değeri

extension UIColor {
    convenience init(hex: Int, alpha: CGFloat = 1.0) {
        let r = CGFloat((hex >> 16) & 0xff) / 255
        let g = CGFloat((hex >> 08) & 0xff) / 255
        let b = CGFloat((hex >> 00) & 0xff) / 255
        self.init(red: r, green: g, blue: b, alpha: alpha)
    }
}

Örnek:

let color = UIColor(hex: 0xff00cc, alpha: 1.0)

“alfa” için varsayılan “1.0” değerinin sağlandığını unutmayın, bu nedenle aşağıdaki gibi kullanılabilir:

let color = UIColor(hex: 0xff00cc)

Dize değeri

extension UIColor {
    convenience init(hexCode: String) {
        let hex = hexCode.stringByTrimmingCharactersInSet(NSCharacterSet.alphanumericCharacterSet().invertedSet)
        var int = UInt32()
        NSScanner(string: hex).scanHexInt(&int)
        let a, r, g, b: UInt32

        switch hex.characters.count {
        case 3:
            (a, r, g, b) = (255, (int >> 8) * 17, (int >> 4 & 0xF) * 17, (int & 0xF) * 17)
        case 6:
            (a, r, g, b) = (255, int >> 16, int >> 8 & 0xFF, int & 0xFF)
        case 8:
            (a, r, g, b) = (int >> 24, int >> 16 & 0xFF, int >> 8 & 0xFF, int & 0xFF)
        default:
            (a, r, g, b) = (1, 1, 1, 0)
        }
    
        self.init(red: CGFloat(r) / 255, green: CGFloat(g) / 255, blue: CGFloat(b) / 255, alpha: CGFloat(a) / 255)
    }
}

Örnek Kullanım:

Alfa ile altıgen

let color = UIColor("#80FFFFFF")

Alfa içermeyen altıgen (“renkli” alfa 1.0’a eşit olacaktır)

let color = UIColor("#FFFFFF")
let color = UIColor("#FFF")

Hedef-C

Int Değeri

@interface UIColor (Hex)
+ (UIColor *)colorWithHex:(NSUInteger)hex alpha:(CGFloat)alpha;
@end

@implementation UIColor (Hex)
+ (UIColor *)colorWithHex:(NSUInteger)hex alpha:(CGFloat)alpha {
    return [UIColor colorWithRed:((CGFloat)((hex & 0xFF0000) >> 16))/255.0
                           green:((CGFloat)((hex & 0xFF00) >> 8))/255.0
                            blue:((CGFloat)(hex & 0xFF))/255.0
                           alpha:alpha];
}
@end

Örnek:

UIColor *color = [UIColor colorWithHex:0xff00cc alpha:1.0];

Dize değeri

- (UIColor*) hex:(NSString*)hexCode {
    
    NSString *noHashString = [hexCode stringByReplacingOccurrencesOfString:@"#" withString:@""];
    NSScanner *scanner = [NSScanner scannerWithString:noHashString];
    [scanner setCharactersToBeSkipped:[NSCharacterSet symbolCharacterSet]];
    
    unsigned hex;
    if (![scanner scanHexInt:&hex]) return nil;
    int a;
    int r;
    int g;
    int b;

    switch (noHashString.length) {
        case 3:
            a = 255;
            r = (hex >> 8) * 17;
            g = ((hex >> 4) & 0xF) * 17;
            b = ((hex >> 0) & 0xF) * 17;
            break;
        case 6:
            a = 255;
            r = (hex >> 16);
            g = (hex >> 8) & 0xFF;
            b = (hex) & 0xFF;
            break;
        case 8:
            a = (hex >> 24);
            r = (hex >> 16) & 0xFF;
            g = (hex >> 8) & 0xFF;
            b = (hex) & 0xFF;
            break;
            
        default:
            a = 255.0;
            r = 255.0;
            b = 255.0;
            g = 255.0;
            break;
    }

    return [UIColor colorWithRed:r / 255.0f green:g / 255.0f blue:b / 255.0f alpha:a / 255];
}

Örnek kullanım:

Alfa ile altıgen

UIColor* color = [self hex:@"#80FFFFFF"];

Alfa içermeyen altıgen (“renkli” alfa 1’e eşittir)

UIColor* color = [self hex:@"#FFFFFF"];
UIColor* color = [self hex:@"#FFF"];

Alfa bileşenli renk #

“init(red:,green:,blue:,alpha:)” başlatıcısını kullanarak yeni bir tane oluşturmadan opaklığı belirli bir “UIColor"a ayarlayabilirsiniz.

Süratli #

let colorWithAlpha = UIColor.redColor().colorWithAlphaComponent(0.1)

Hızlı 3 #

//In Swift Latest Version
_ colorWithAlpha = UIColor.red.withAlphaComponent(0.1)

Amaç-C #

UIColor * colorWithAlpha = [[UIColor redColor] colorWithAlphaComponent:0.1];

Belgelenmemiş Yöntemler #

“UIColor” üzerinde, alternatif renkleri veya işlevleri ortaya çıkaran çeşitli belgelenmemiş yöntemler vardır. Bunlar UIColor özel başlık dosyasında bulunabilir. İki özel yöntemin kullanımını belgeleyeceğim, styleString() ve _systemDestructiveTintColor().

‘stilDizesi’ #

iOS 2.0’dan bu yana, RGB alanı dışındaki “whiteColor” gibi renkler için bile, rengin RGB veya RGBA dize temsilini döndüren “styleString” adlı “UIColor” üzerinde özel bir örnek yöntemi vardır.

Amaç-C:

@interface UIColor (Private)

- (NSString *)styleString;

@end

// ...

[[UIColor whiteColor] styleString]; // rgb(255,255,255)
[[UIColor redColor] styleString]; // rgb(255,0,0)
[[UIColor lightTextColor] styleString]; // rgba(255,255,255,0.600000)

Swift’de arayüzü ortaya çıkarmak için bir köprüleme başlığı kullanabilirsiniz. Saf Swift ile, özel yöntemle bir “@objc” protokolü ve protokolle “unsafeBitCast” “UIColor” oluşturmanız gerekir:

@objc protocol  UIColorPrivate {
    func styleString() -> String
}

let white = UIColor.whiteColor()
let red = UIColor.redColor()
let lightTextColor = UIColor.lightTextColor()

let whitePrivate = unsafeBitCast(white, UIColorPrivate.self)
let redPrivate = unsafeBitCast(red, UIColorPrivate.self)
let lightTextColorPrivate = unsafeBitCast(lightTextColor, UIColorPrivate.self)

whitePrivate.styleString() // rgb(255,255,255)
redPrivate.styleString() // rgb(255,0,0)
lightTextColorPrivate.styleString() // rgba(255,255,255,0.600000)

_systemDestructiveTintColor() #

‘UIColor’ üzerinde ‘_systemDestructiveTintColor’ adında, yıkıcı sistem düğmeleri tarafından kullanılan kırmızı rengi döndürecek belgelenmemiş bir sınıf yöntemi vardır:

let red = UIColor.performSelector("_systemDestructiveTintColor").takeUnretainedValue()

Renk sahipliği kendi nesnemize aktarılmadığından .takeUnretainedValue() üzerinde çağırmanız gereken yönetilmeyen bir nesne döndürür.

Belgelenmemiş herhangi bir API’de olduğu gibi, bu yöntemi kullanmaya çalışırken dikkatli olmalısınız:

if UIColor.respondsToSelector("_systemDestructiveTintColor") {
    if let red = UIColor.performSelector("_systemDestructiveTintColor").takeUnretainedValue() as? UIColor {
        // use the color
    }
}

veya bir protokol kullanarak:

@objc protocol UIColorPrivateStatic {
    func _systemDestructiveTintColor() -> UIColor
}

let privateClass = UIColor.self as! UIColorPrivateStatic
privateClass._systemDestructiveTintColor() // UIDeviceRGBColorSpace 1 0.231373 0.188235 1

Bir görüntü deseninden UIColor #

‘UIColor(patternImage:_)’ yöntemini kullanarak bir görüntü deseni kullanarak bir ‘UIColor’ nesnesi oluşturabilirsiniz.

btn.backgroundColor = UIColor(patternImage: UIImage(named: "image")!)

Kullanıcı tanımlı niteliklerin CGColor veri tipini uygulamasını sağlayın #

Varsayılan olarak, Arayüz Oluşturucu “CGColor” veri türünü kabul etmez, bu nedenle arayüz oluşturucuda kullanıcı tanımlı öznitelikleri kullanarak bir “CGColor” eklenmesine izin verir; şöyle bir uzantı kullanmak isteyebilirsiniz:

Swift Uzantısı:

extension CALayer {
    func borderUIColor() -> UIColor? {
        return borderColor != nil ? UIColor(CGColor: borderColor!) : nil
    }
    
    func setBorderUIColor(color: UIColor) {
        borderColor = color.CGColor
    }
}

Yeni kullanıcı tanımlı öznitelik (borderUIColor) sorunsuz bir şekilde tanınacak ve uygulanacaktır.

Belirli bir UIColor’ın Daha Açık ve Daha Koyu Tonu #

Aşağıdaki kod örneği, dinamik temalara sahip uygulamalarda yararlı olan belirli bir rengin daha açık ve daha koyu tonlarını nasıl elde edebileceğinizi gösterir.

Daha Koyu Renk için

+ (UIColor *)darkerColorForColor:(UIColor *)c
{
    CGFloat r, g, b, a;
    if ([c getRed:&r green:&g blue:&b alpha:&a])
        return [UIColor colorWithRed:MAX(r - 0.2, 0.0)
                               green:MAX(g - 0.2, 0.0)
                                blue:MAX(b - 0.2, 0.0)
                               alpha:a];
    return nil;
}

Daha Açık Renk için

+ (UIColor *)lighterColorForColor:(UIColor *)c
{
    CGFloat r, g, b, a;
    if ([c getRed:&r green:&g blue:&b alpha:&a])
        return [UIColor colorWithRed:MIN(r + 0.2, 1.0)
                               green:MIN(g + 0.2, 1.0)
                                blue:MIN(b + 0.2, 1.0)
                               alpha:a];
    return nil;
}

UIColor’dan Ayarlanmış Renk Parlaklığı #

Aşağıdaki kod örneği, daha yüksek bir yüzdenin daha parlak ve daha düşük bir yüzdenin daha koyu olacağı o rengin ayarlanmış bir sürümünü verecektir.

Hedef-C

+ (UIColor *)adjustedColorForColor:(UIColor *)c : (double)percent
{
    if (percent < 0) percent = 0;
    
    CGFloat r, g, b, a;
    if ([c getRed:&r green:&g blue:&b alpha:&a])
        return [UIColor colorWithRed:MAX(r * percent, 0.0)
                               green:MAX(g * percent, 0.0)
                                blue:MAX(b * percent, 0.0)
                               alpha:a];
    return nil;
}

Süratli

func adjustedColorForColor( c: UIColor, var percent: CGFloat) -> UIColor {
    if percent < 0 {
        percent = 0
    }

    var r,g,b,a: CGFloat
    r = 0.0
    g = 0.0
    b = 0.0
    a = 0.0

    if c.getRed(&r, green: &g, blue: &b, alpha: &a) {
        return UIColor(red: max(r * percent, 0.0), green: max(g * percent, 0.0), blue: max(b * percent, 0.0), alpha: a)
    }

    return UIColor()
}
