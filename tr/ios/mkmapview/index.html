Harita türünü değiştir #

5 farklı türü vardır (MKMapType), MKMapView görüntüleyebilir.

.standart #

Tüm yolların konumunu ve bazı yol adlarını gösteren bir sokak haritası görüntüler.

Hızlı 2 #

mapView.mapType = .Standard

Hızlı 3 #

mapView.mapType = .standard

Amaç-C #

_mapView.mapType = MKMapTypeStandard;

son sürüm

.uydu #

Bölgenin uydu görüntülerini görüntüler.

Hızlı 2 #

mapView.mapType = .Satellite

Hızlı 3 #

mapView.mapType = .satellite

Amaç-C #

_mapView.mapType = MKMapTypeSatellite;

son sürüm

.satelliteFlyover #

Varsa, üstgeçit verileriyle birlikte alanın uydu görüntüsünü görüntüler.

Hızlı 2 #

mapView.mapType = .SatelliteFlyover

Hızlı 3 #

mapView.mapType = .satelliteFlyover

Amaç-C #

_mapView.mapType = MKMapTypeSatelliteFlyover;
son sürüm

.karma #

Üstte yol ve yol adı bilgilerinin bulunduğu bölgenin uydu görüntüsünü görüntüler.

Hızlı 2 #

mapView.mapType = .Hybrid

Hızlı 3 #

mapView.mapType = .hybrid

Amaç-C #

_mapView.mapType = MKMapTypeHybrid;

son sürüm

.hybridFlyover #

Varsa, geçiş verileriyle birlikte bir hibrit uydu görüntüsü görüntüler.

Hızlı 2 #

mapView.mapType = .HybridFlyover

Hızlı 3 #

mapView.mapType = .hybridFlyover

Amaç-C #

_mapView.mapType = MKMapTypeHybridFlyover;
son sürüm

MKMapView’ı ekleyin #

Süratli

let mapView = MKMapView(frame: CGRect(x: 0, y: 0, width: 320, height: 500))

Daha karmaşık uygulamalarda erişmek isteyebileceğiniz için mapView’ı içeren ‘ViewController’ın bir özelliği olarak saklamanız önerilir.

Hedef C

self.map = [[MKMapView alloc]initWithFrame:CGRectMake(0, 0, self.view.frame.size.width, self.view.frame.size.height)];
[self.view addSubview:self.map];

Harita için Yakınlaştırmayı/Bölgeyi Ayarla #

Bir yakınlaştırma seviyesi ayarlamak için, diyelim ki kullanıcı konumu merkez ve 2 km’lik alan yarıçap olarak olacak şekilde kullanıcının konumunu yakınlaştırmak istiyoruz. Ardından, aşağıdaki kodu kullanıyoruz

MKUserLocation *userLocation = _mapView.userLocation;
MKCoordinateRegion region = MKCoordinateRegionMakeWithDistance (userLocation.location.coordinate, 2000, 2000);
[_mapView setRegion:region animated:NO];

MKLocalSearch kullanarak yerel arama uygulaması #

MKLocalSearch, kullanıcıların “spor salonu” gibi doğal dil dizelerini kullanarak konum aramasına olanak tanır. Arama tamamlandıktan sonra sınıf, belirtilen bir bölge içindeki arama dizesiyle eşleşen konumların bir listesini döndürür.

Arama sonuçları, MKLocalSearchResponse nesnesi içinde MKMapItem biçimindedir.

örnekle deneyelim

MKLocalSearchRequest *request = 
    [[MKLocalSearchRequest alloc] init];//initialising search request
request.naturalLanguageQuery = @”Gym”; // adding query
request.region = _mapView.region; //setting region
MKLocalSearch *search = 
     [[MKLocalSearch alloc]initWithRequest:request];//initiate search

[search startWithCompletionHandler:^(MKLocalSearchResponse 
     *response, NSError *error) 
{
   if (response.mapItems.count == 0)
       NSLog(@"No Matches");
   else
       for (MKMapItem *item in response.mapItems)
       {
           NSLog(@"name = %@", item.name);
           NSLog(@"Phone = %@", item.phoneNumber);
        }
}];

OpenStreetMap Döşeme Yerleşimi #

Bazı durumlarda, Apple’ın sağladığı varsayılan haritaları kullanmak istemeyebilirsiniz.

Örneğin OpenStreetMap‘den özel döşemeler içeren “mapView” dosyanıza bir kaplama ekleyebilirsiniz.

Diyelim ki, ‘self.mapView’, ‘ViewController’ınıza önceden eklemiş olduğunuz ‘MKMapView’ınızdır.

İlk başta, “ViewController"ınızın “MKMapViewDelegate” protokolüne uyması gerekir.

class MyViewController: UIViewController, MKMapViewDelegate

O zaman ‘ViewController’ı ‘mapView’ temsilcisi olarak ayarlamanız gerekir.

mapView.delegate = self

Ardından, harita için yer paylaşımını yapılandırırsınız. Bunun için bir URL şablonuna ihtiyacınız olacak. URL, tüm döşeme sunucularında buna benzer olmalıdır ve harita verilerini çevrimdışı olarak depolasanız bile: http://tile.openstreetmap.org/{z}/{x}/{y}.png

let urlTeplate = "http://tile.openstreetmap.org/{z}/{x}/{y}.png"
let overlay = MKTileOverlay(urlTemplate: urlTeplate)
overlay.canReplaceMapContent = true

Bindirmeyi yapılandırdıktan sonra, onu ‘mapView’ınıza eklemelisiniz.

mapView.add(overlay, level: .aboveLabels)

Özel haritaları kullanmak için, “seviye” için “.aboveLabels” kullanılması önerilir. Aksi takdirde, varsayılan etiketler özel haritanızda görünür olacaktır. Varsayılan etiketleri görmek istiyorsanız, burada .aboveRoads öğesini seçebilirsiniz.

Projenizi şimdi yürütürseniz, haritanızın hala varsayılan haritayı göstereceğini anlarsınız:

[

Bunun nedeni, “mapView"a henüz bindirmenin nasıl oluşturulacağını söylememiş olmamızdır. Bu nedenle, delegeyi daha önce ayarlamak zorunda kaldınız. Şimdi görünüm denetleyicinize func mapView(_ mapView: MKMapView, rendererFor overlay: MKOverlay) -> MKOverlayRenderer ekleyebilirsiniz:

func mapView(_ mapView: MKMapView, rendererFor overlay: MKOverlay) -> MKOverlayRenderer {
    if overlay is MKTileOverlay {
        let renderer = MKTileOverlayRenderer(overlay: overlay)
        return renderer
    } else {
        return MKTileOverlayRenderer()
    }
}

Haritaya Pin/Nokta Açıklaması Ekleme #

Harita üzerinde bir ilgi çekici noktaya açıklama eklemek için pin açıklamasını kullanırız. Şimdi, önce açıklama nesnesi oluşturarak başlayın.

MKPointAnnotation *pointAnnotation = [[MKPointAnnotation alloc] init];

Şimdi, nokta Annotation’a koordinat sağlayın,

CLLocationCoordinate2D coordinate = CLLocationCoordinate2DMake(23.054625,72.534562);
pointAnnotation.coordinate = coordinate;

Şimdi, ek açıklamaya başlık ve alt başlık sağlayın,

pointAnnotation.title = @"XYZ Point";
pointAnnotation.subtitle = @"Ahmedabad Area";

Şimdi, bu ek açıklamayı haritaya ekleyin.

[self.mapView addAnnotation:pointAnnotation];

Evet.. Yaşasın.. işi yaptın. Artık verilen koordinatta nokta açıklamasını (kırmızı renkli pin) görebilirsiniz.

Ama şimdi, pinin rengini değiştirmek isterseniz ne olur(mevcut 3 renk - Mor, kırmızı ve yeşil). Ardından bu adımı izleyin.

mapview’in temsilcisini kendine ayarla,

self.mapView.delegate = self;

MKMapViewDelegate uygulamasını ekleyin. Şimdi aşağıdaki yöntemi ekleyin, sonra

- (MKAnnotationView *)mapView:(MKMapView *)mapView viewForAnnotation:(id <MKAnnotation>)annotation
{
    // If it's the user location, just return nil, because it have user location's own annotation, if you want to change that, then use this object;
    if ([annotation isKindOfClass:[MKUserLocation class]])
        return nil;

    if ([annotation isKindOfClass:[MKPointAnnotation class]])
    {
        //Use dequed pin if available
        MKAnnotationView *pinView = [mapView dequeueReusableAnnotationViewWithIdentifier:@"PinAnnotationView"];
    
        if (!pinView)
        {
            // If not dequed, then create new.
            pinView = [[MKAnnotationView alloc] initWithAnnotation:annotation reuseIdentifier:@"PinAnnotationView"];
            pinView.canShowCallout = YES;
            pinView.image = [UIImage imageNamed:@"abc.png"];
            pinView.calloutOffset = CGPointMake(0, 32);
        } else {
            pinView.annotation = annotation;
        }
        return pinView;
    }
    return nil;
}

Koordinat ve yakınlaştırma düzeyi için kaydırın #

Kullanıcılarınıza bir konum gösterdiğinizde, gösterilecek bir bölge ayarlamak yerine MKMapView uygulamasının yakınlaştırma düzeyinde bir koordinat görüntülemesini isteyebilirsiniz. Bu işlevsellik varsayılan olarak uygulanmaz, bu nedenle, karmaşık hesaplamayı bir koordinat ve yakınlaştırma düzeyinden bir ‘MKKoordinatBölgesi’ne yapan yöntemlerle ‘MKMapView’ı genişletmeniz gerekir.

let MERCATOR_OFFSET = 268435456.0
let MERCATOR_RADIUS = 85445659.44705395
let DEGREES = 180.0

public extension MKMapView {
    
    //MARK: Map Conversion Methods
    
    private func longitudeToPixelSpaceX(longitude:Double)->Double{
        return round(MERCATOR_OFFSET + MERCATOR_RADIUS * longitude * M_PI / DEGREES)
    }
    
    private func latitudeToPixelSpaceY(latitude:Double)->Double{
        return round(MERCATOR_OFFSET - MERCATOR_RADIUS * log((1 + sin(latitude * M_PI / DEGREES)) / (1 - sin(latitude * M_PI / DEGREES))) / 2.0)
    }
    
    private func pixelSpaceXToLongitude(pixelX:Double)->Double{
        return ((round(pixelX) - MERCATOR_OFFSET) / MERCATOR_RADIUS) * DEGREES / M_PI
    }
    
    private func pixelSpaceYToLatitude(pixelY:Double)->Double{
        return (M_PI / 2.0 - 2.0 * atan(exp((round(pixelY) - MERCATOR_OFFSET) / MERCATOR_RADIUS))) * DEGREES / M_PI
    }
    
    private func coordinateSpanWithCenterCoordinate(centerCoordinate:CLLocationCoordinate2D, zoomLevel:Double)->MKCoordinateSpan{
        // convert center coordiate to pixel space
        let centerPixelX = longitudeToPixelSpaceX(longitude: centerCoordinate.longitude)
        let centerPixelY = latitudeToPixelSpaceY(latitude: centerCoordinate.latitude)
        print(centerCoordinate)
        // determine the scale value from the zoom level
        let zoomExponent:Double = 20.0 - zoomLevel
        let zoomScale:Double = pow(2.0, zoomExponent)
        // scale the map’s size in pixel space
        let mapSizeInPixels = self.bounds.size
        let scaledMapWidth = Double(mapSizeInPixels.width) * zoomScale
        let scaledMapHeight = Double(mapSizeInPixels.height) * zoomScale
        // figure out the position of the top-left pixel
        let topLeftPixelX = centerPixelX - (scaledMapWidth / 2.0)
        let topLeftPixelY = centerPixelY - (scaledMapHeight / 2.0)
        // find delta between left and right longitudes
        let minLng = pixelSpaceXToLongitude(pixelX: topLeftPixelX)
        let maxLng = pixelSpaceXToLongitude(pixelX: topLeftPixelX + scaledMapWidth)
        let longitudeDelta = maxLng - minLng
        let minLat = pixelSpaceYToLatitude(pixelY: topLeftPixelY)
        let maxLat = pixelSpaceYToLatitude(pixelY: topLeftPixelY + scaledMapHeight)
        let latitudeDelta = -1.0 * (maxLat - minLat)
        return MKCoordinateSpan(latitudeDelta: latitudeDelta, longitudeDelta: longitudeDelta)
    }
    
    /**
     Sets the center of the `MKMapView` to a `CLLocationCoordinate2D` with a custom zoom-level. There is no nee to set a region manually. :-)
     
     - author: Mylene Bayan (on GitHub)
     */
    public func setCenter(_ coordinate:CLLocationCoordinate2D, zoomLevel:Double, animated:Bool){
        // clamp large numbers to 28
        var zoomLevel = zoomLevel
        zoomLevel = min(zoomLevel, 28)
        // use the zoom level to compute the region
        print(coordinate)
        let span = self.coordinateSpanWithCenterCoordinate(centerCoordinate: coordinate, zoomLevel: zoomLevel)
        let region = MKCoordinateRegionMake(coordinate, span)
        if region.center.longitude == -180.00000000{
            print("Invalid Region")
        }
        else{
            self.setRegion(region, animated: animated)
        }
    }
}

(Mylene Bayan tarafından hazırlanan orijinal Swift 2 sürümü GitHub‘de bulunabilir)

Bu “uzantıyı” uyguladıktan sonra, merkez koordinatını aşağıdaki gibi ayarlayabilirsiniz:

let centerCoordinate = CLLocationCoordinate2DMake(48.136315, 11.5752901) //latitude, longitude
mapView?.setCenter(centerCoordinate, zoomLevel: 15, animated: true)

“zoomLevel”, genellikle “0” ile “21” (çok yüksek bir yakınlaştırma düzeyidir) arasında bir “Çift” değerdir, ancak “28"e kadar değerlere izin verilir.

Ek Açıklama ile Çalışma #

Tüm Ek Açıklamaları Alın

//following method returns all annotations object added on map
NSArray *allAnnotations = mapView.annotations;

Ek Açıklama Görünümünü Alın

for (id<MKAnnotation> annotation in mapView.annotations)
{
    MKAnnotationView* annotationView = [mapView viewForAnnotation:annotation];
    if (annotationView)
    {
       // Do something with annotation view 
       // for e.g change image of annotation view
       annotationView.image = [UIImage imageNamed:@"SelectedPin.png"];
    }
}

Tüm Ek Açıklamaları Kaldır

[mapView removeAnnotations:mapView.annotations]

Tek Açıklamayı Kaldır

//getting all Annotation
NSArray *allAnnotations = self.myMapView.annotations;

if (allAnnotations.count > 0)
{
    //getting first annoation
    id <MKAnnotation> annotation=[allAnnotations firstObject];
    
    //removing annotation
    [mapView removeAnnotation:annotation];
    
}
