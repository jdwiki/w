Parametreler #

yaygın olarak kullanılan Genel Yöntem Kullanım
SetTitle(String) İletişim kutusunun başlığını ayarlar
SetIcon(Çizilebilir) Uyarı iletişim kutusu için Simge Ayarla
SetMesajı(dize) Görüntülenecek mesajı ayarlayın.
SetNegativeButton(String, EventHandler) İletişim kutusunun negatif düğmesine basıldığında çağrılacak bir dinleyici ayarlayın.
SetPositiveButton(String, EventHandler) İletişimin pozitif düğmesine basıldığında çağrılacak bir dinleyici ayarlayın.
SetNeutralButton(String, EventHandler) İletişimin nötr düğmesine basıldığında çağrılacak bir dinleyici ayarlayın.
SetOnCancelListener(IDialogInterfaceOnCancelListener) Diyalog iptal edildiğinde çağrılacak geri aramayı ayarlar.
SetOnDismissListener(IDialogInterfaceOnDismissListener) Diyalog herhangi bir nedenle kapatıldığında çağrılacak geri aramayı ayarlar.
Show() Bu oluşturucuya sağlanan bağımsız değişkenlerle ve Dialog.Show’un iletişim kutusuyla bir AlertDialog oluşturur.

Gereksinimler

Ad alanı: Android.App

Montaj: Mono.Android (Mono.Android.dll’de)

Montaj Sürümleri: 0.0.0.0

Genel İnşaatçılar

AlertDialog.Builder(Bağlam): -

Bu oluşturucu ve oluşturduğu AlertDialog için bir bağlam kullanan yapıcı.

AlertDialog.Builder(Context, Int32):-

Bu oluşturucu ve oluşturduğu AlertDialog için bir bağlam ve tema kullanan yapıcı.

Materyal Tasarım AlertDialog’u Kullanma

Modern AlertDialog’u kullanmak için:

  1. NuGet paketlerinden Support v7 AppCompat kitaplığını yükleyin

  2. AlertDialog’u Android.Support.V7.App.AlertDialog ile değiştirin veya following statement at the top to make your dialog shine.

         using AlertDialog = Android.Support.V7.App.AlertDialog;

Uyarı İletişim Kutusu #

// 1. Instantiate an AlertDialog.Builder with its constructor
// the parameter this is the context (usually your activity)
AlertDialog.Builder builder = new AlertDialog.Builder(this);

// 2. Chain together various setter methods to set the dialog characteristics 
builder.SetMessage(Resource.String.dialog_message)
       .SetTitle(Resource.String.dialog_title);

// 3. Get the AlertDialog from create()
AlertDialog dialog = builder.Create();

dialog.Show();

Basit Uyarı İletişim Kutusu Örneği #

Xamarin.Android’de basit bir Uyarı İletişim Kutusu oluşturacağız

Şimdi, belgelerdeki başlangıç ​​kılavuzunu gözden geçirdiğinizi düşünün.

Proje yapısına şu şekilde sahip olmalısınız:

Ana Faaliyetiniz şöyle görünmelidir:

 public class MainActivity : Activity
 {
 int count = 1;
 
 protected override void OnCreate(Bundle bundle)
 {
 base.OnCreate(bundle);
 
 // Set our view from the "main" layout resource
 SetContentView(Resource.Layout.Main);
 
 // Get our button from the layout resource,
 // and attach an event to it
 Button button = FindViewById<Button>(Resource.Id.MyButton);
 
 button.Click += delegate { button.Text = string.Format("{0} clicks!", count++); };
 }
 }

Şimdi yapacağımız şey, butona tıklandığında sayaca bir tane eklemek yerine, basit bir Alert Dialog’da kullanıcıya bir tane eklemek veya çıkarmak isteyip istemediğini soracağız.

Ve Pozitif veya Negatif butonuna tıklandığında eylemi gerçekleştireceğiz.

 button.Click += delegate {
 AlertDialog.Builder alert = new AlertDialog.Builder(this);
 alert.SetTitle("Specify Action");
 alert.SetMessage("Do you want to add or substract?");

 alert.SetPositiveButton("Add", (senderAlert, args) =>
 {
 count++;
 button.Text = string.Format("{0} clicks!", count);
  });

  alert.SetNegativeButton("Substract", (senderAlert, args) =>
  {
  count--;
  button.Text = string.Format("{0} clicks!", count);
  });

  Dialog dialog = alert.Create();
      dialog.Show();
 };
