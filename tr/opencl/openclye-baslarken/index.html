OpenCL nedir? #

OpenCL, Open Ccomputing Language’ın kısaltmasıdır. OpenCL, CPU’lardan GPU’lara, FPGA’lar gibi daha özel platformlara kadar, hesaplama cihazları adı verilen heterojen platformlar arasında paralel programlama için bir Çerçevedir. OpenCL, bu hesaplama cihazlarında paralel hesaplama için standart bir arabirim ve ayrıca cihazlar arası paralellik sağlar. C99’a dayalı bir programlama dilini ve OpenCL özellikli cihazlarda uygulanan temel işlevlerin minimum gereksinimlerini belirtir. OpenCL ayrıca, farklı platformlar arasında kodun yeniden kullanımını basit hale getirmek için mümkün olduğunca genel olan soyut bir hesaplama ve bellek modelini tanımlar.

Önkoşullar #

Makinenizin içinde modern bir CPU veya grafik kartı (GPU) varsa, OpenCL’deki ilk adımlar için her şeye hazır olma ihtimaliniz vardır. İşlemcinizin OpenCL uyumlu olup olmadığını öğrenmek genellikle üreticinin ana sayfasından yapılabilir.

https://www.khronos.org/conformance/adopters/conformant-products#opencl

OpenCL nedir? #

Açık Bilgi İşlem Dili (OpenCL), CPU’lar, GPU’lar ve diğer paralel işlemciler ve hızlandırıcılar üzerinde çalışan programlar yazmak için bir çerçevedir.

OpenCL, adlandırılmış çip üzerindeki belleğe erişim sağlayan bir programlama dili (C’ye dayalı), görevleri paralel olarak yürütmek için bir model ve bu görevleri senkronize etme becerisini belirtir.

OpenCL 1.2’nin C# uygulaması: 64 bit pencerelerde AMD sistemi için platform sayısı #

OpenCL düşük seviyeli api’dir, bu yüzden önce “C alanı"nda uygulanmalıdır. Bunun için, Khronos’un sitesinden başlık dosyalarını indirmeniz gerekiyor. Donanımım AMD ve sürüm 1.2’ye uygun, indirme

opencl.h 
cl_platform.h 
cl.h 
cl_ext.h 
cl_egl.h 
cl_dx9_media_sharing.h 
cl_d3d10.h 
cl_d3d11.h 
cl_gl.h 
cl_gl_ext.h 
cl.hpp

bu sayfadan

C++ bağlamaları için yeterli olmalıdır, bu nedenle bu dosyaları projenize ekledikten ve uygun ikili (ve kitaplık) dosya konumlarını ayarladıktan sonra (

64-bit amd kitaplığı için $(AMDAPPSDKROOT)\lib\x86_64 (amd app sdk kitaplıkları tercih edilir)

,

64-bit opencl.dll için C:\Windows\SysWOW64 (ICD bir Linux sistemine aitse .so dosyası)

örneğin, ancak Intel-Nvidia için farklı), uygun sürücüleri (amd için crimson gibi) yükledikten sonra bir platform listesi (amd,intel,xilinx,nvidia) sorgulamaya başlayabilirsiniz. Sürücüler, opencl uygulamasını (ICD kullanarak) çalıştırmak içindir, kitaplıklar ve başlık dosyaları kısaca geliştirme içindir.

Platformları sorgulamak için:

#define __CL_ENABLE_EXCEPTIONS
#include "stdafx.h"
#include <vector>
#include <CL/cl.hpp>

extern "C"
    {
       // when this class is created, it contains a list of platforms in "platforms" field.
       class OpenClPlatformList
       {
           public:
               std::vector<cl::Platform> platforms;
               int platformNum;
               OpenClPlatformList()
               {
                   platforms= std::vector< cl::Platform>();
                   cl::Platform::get(&platforms);
                   platformNum= platforms.size();
               }
        };


        // this is seen from C# when imported. Creates an object in memory.
        __declspec(dllexport)
            OpenClPlatformList * createPlatformList()
        {
            return new OpenClPlatformList();
        }

        __declspec(dllexport)
            int platformNumber(OpenClPlatformList * hList)
        {
            return hList->platformNum;
        }


        __declspec(dllexport)
            void deletePlatformList(OpenClPlatformList * p)
        {
            if (p != NULL)
                delete p;
            p = NULL;
        }


    }

bir dll içine yerleştirilebilir (OCLImplementation.dll gibi)

ve C# tarafından kullanmak için,

using System;
using System.Collections.Generic;
using System.Runtime.InteropServices;


namespace WrapperCSharp
{
    public class WrapperCSharp
    {
        [DllImport("OCLImplementation", CallingConvention = CallingConvention.Cdecl)]
        private static extern IntPtr createPlatformList();

        [DllImport("OCLImplementation", CallingConvention = CallingConvention.Cdecl)]
        private static extern int platformNumber(IntPtr hList);

        [DllImport("OCLImplementation", CallingConvention = CallingConvention.Cdecl)]
        private static extern void deletePlatformList(IntPtr hList);
    }
}

Tabii ki dll, C# projesi tarafından görülmelidir, basitçe onu projenin yürütülebilir dosyasının yanına koymak onu çözer.

Şimdi, örnek bilgisayarın en az bir opencl özellikli platformu varsa,

IntPtr platformList = createPlatformList(); // just an address in C-space
int totalPlatforms = platformNumber(platformList); // AMD+NVIDIA systems should have "2"
deletePlatformList(platformList); //

totalPlatforms değişkeni en az “1” değerine sahip olmalıdır. Ardından, CPU, GPU gibi tüm cihazları ve phi veya bazı fpga gibi özel amaçlı hızlandırıcıları sorgulamak için tüm platformlarda yineleme yapmak için ek işlevleri kullanarak C-alanında platformlar değişkenini kullanabilirsiniz.

Zaman açısından kritik projeler için tüm bu C++‘ları C# sarmalayıcılarına yazmak yeterli değildir. C#, Java ve diğer diller için yazılmış birçok sarmalayıcı vardır. Java için, “opencl-c’ye Java bayt kodu” dönüştürücü api olan “Aparapi” vardır; bu, tamamen Java’da yazdıklarınızı anında bir gpu-paralel sürümüne alır, böylece biraz taşınabilir.

OpenCL ve C# #

C# için OpenCL ile iletişim kurmak için bir arayüz sunan birçok paketleyici vardır.

  • OpenCL.NET: Bu, oradaki en düşük seviyeli paketleyicilerden biridir. Hiçbir soyutlama eklemeden C# için OpenCL API’sinin eksiksiz bir uygulamasını sunar. Böylece C\C++ örnekleri bu kitaplık için kolayca taşınabilir. Tek proje sayfası şu anda 15.12.2017’de kapanan ancak paket NuGet’te bulunan kodeks üzerindedir.

https://openclnet.codeplex.com/

  • NOpenCL: Bu kitaplık, C# ve OpenCL arasında soyut bir arayüz sunar.

Kısa vadeli hedef, performanstan ödün vermeden OpenCL’nin tam kapasitesine erişim sağlayan, kullanımı kolay bir soyut katman sağlamaktır.

https://github.com/tunnelvisionlabs/NOpenCL

  • Cloo:

Cloo, .NET/Mono uygulamalarının OpenCL çerçevesinden tam olarak yararlanmasını sağlayan açık kaynaklı, kullanımı kolay, yönetilen bir kitaplıktır.

https://sourceforge.net/projects/cloo/
