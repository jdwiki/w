Yönetilen Kitaplık Bağımlılığı Ekle #

’libraryDependency’, sağlanan sürümlerle eşleşen, otomatik olarak indirilen bağımlılıklar olan ‘yönetilen’ kitaplık bağımlılıklarını işleyen ‘SettingKey’dir. Tek bir bağımlılık eklemek için:

libraryDependencies += "com.typesafe.slick" %% "slick" % "3.2.0-M1"

İlk kısım olan "com.typesafe.slick", kitaplık paketini gösterir. İkinci kısım olan "kaygan", söz konusu kütüphanedir. Son kısım olan "3.2.0-M1", sürümdür. Kitaplığa %% ile katıldığından, scalaVersion ayar anahtarı tarafından sağlanan Scala sürümü kullanılacaktır.

++= kullanarak aynı anda birden çok kitaplık ekleyebilirsiniz:

libraryDependencies ++= Seq(
  "com.typesafe.slick" %% "slick" % "3.2.0-M1" % "compile",
  "com.typesafe.slick" %% "slick-hikaricp" % "3.2.0-M1",
  "mysql" % "mysql-connector-java" % "latest.release"
)

Bağımlılıkları hesaplamanıza izin veren Scala’nın işlevsel doğasını hatırlayın. Sadece bir “Seq” döndürmeyi unutmayın:

libraryDependencies ++= {
  lazy val liftVersion = "3.0-RC3" //Version of a library being used
  lazy val liftEdition = liftVersion.substring(0,3) //Compute a value
  Seq(
    "net.liftweb" %% "lift-webkit" % liftVersion % "compile",  // Use var in Seq
    "net.liftmodules" %% ("ng_" + liftEdition) % "0.9.2" % "compile",  // Use computed var in Seq
  )  // Because this is the last statement, the Seq is returned and appended to libraryDependencies
}

Bir Depo Ekle #

Depo, SBT’nin ’libraryDependencies’i aradığı bir yerdir. Derleme bir bağımlılık bulamadığından şikayet ederse, doğru depodan yoksun olabilir. SBT içinde, depolar “çözümleyiciler” Ayar Anahtarında listelenir:

resolvers += "Flyway" at "https://flywaydb.org/repo"

Bu, ‘url konumunda’ ‘Depo adı’ sözdizimini takip eder.

Kitaplığı Scala’nın Proje Sürümüne Sabitleme #

Projenizde bu varsa:

scalaVersion := 2.11  // Replace '2.11' with the version of Scala your project is running on

Ardından, projenin kullandığı Scala sürümüne göre derlenen kitaplık sürümünü otomatik olarak almak için %% kullanabilirsiniz:

libraryDependencies += "com.typesafe.slick" %% "slick" % "3.2.0-M1"

Yukarıdaki iki satıra sahip olmanın, bu satıra sahip olmakla eşdeğer olduğunu unutmayın:

libraryDependencies += "com.typesafe.slick" % "slick_2.11" % "3.2.0-M1"

Kitaplığı Scala’nın Belirli Sürümüne Sabitleme #

Bir kitaplık, “groupId” ve “artifactId” (bir kitaplık bağımlılığındaki ilk iki dize) arasındaki “%” operatörü kullanılarak belirli bir Scala sürümüne “sabitlenebilir”. Bu örnekte, kütüphaneyi “slick"in “artifactId"sine sahip Scala sürüm “2.10"a sabitliyoruz:

 libraryDependencies += "com.typesafe.slick" % "slick_2.10" % "3.2.0-M1"
