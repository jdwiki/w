yii2’deki oturum #

Oturum Sınıfını içe aktar #

use  yii\web\Session;

Bir oturum oluşturun #

$session = Yii::$app->session;
$session->open(); // open a session
$session->close();  // close a session

Değeri oturum değişkeninde saklayın. #

$session = Yii::$app->session;

$session->set('name', 'stack');
OR
$session['name'] = 'stack';
OR
$_SESSION['name'] = 'stack';

Oturum değişkeninden değeri alın. #

$name = $session->get('name');
OR
$name = $session['name'];

Oturum değişkenini kaldırın #

$session->remove('name');
OR
unset($session['name']);
OR
unset($_SESSION['name']);

$session->destroy(); // destroy all session

Tüm oturum değişkenlerini kaldırın #

$session->removeAll();

Oturum değişkenini kontrol edin #

$session->has('name')
OR
isset($session['name'])
//both function return boolean value [true or false]

Oturum Flaşı #

Oturum flaşını ayarla

$session = Yii::$app->session;
$session->setFlash('error', 'Error in login');

Oturum flaşını al

echo $session->getFlash('error');

Oturum flaşını kontrol edin

$result = $session->hasFlash('error');

Oturum flaşını kaldır

$session->removeFlash('error');

Tüm oturum flaş değişkenlerini kaldırın

$session->removeAllFlashes();

Doğrudan oturum değişkenini kullan #

Oturum değişkenini ayarlayın ve alın

\Yii::$app->session->set('name','stack');
\Yii::$app->session->get('name');

Oturum flaşı

\Yii::$app->getSession()->setFlash('flash_msg','Message');
\Yii::$app->getSession()->getFlash('flash_msg');

Daha sonra tekrar ziyaret etmek için URL’yi hatırla #

Kullanım durumu: Farklı (ilgili) bir denetleyiciye yeni bir kayıt ekledikten sonra geri dönülecek geçerli URL’yi hatırlayın, örneğin, düzenlenmekte olan bir faturaya eklemek için yeni bir ilgili kişi oluşturun.

InvoiceController / actionUpdate:

Url::remember(Url::current(), 'returnInvoice');

ContactController / actionCreate:

if ($model->save()) {
    $return = Url::previous('returnInvoice');
    if ($return) {
        return $this->redirect($return);
    }
    // ...
}

İşiniz bittiğinde hatırlanan URL’yi sıfırlayabilirsiniz:

InvoiceController / actionUpdate:

if ($model->save()) {
    Url::remember(null, 'returnInvoice');
    // ...
}

Anahtar adı - bu örnekte “returnInvoice” - isteğe bağlıdır.

Diziler olan oturum değişkenleri oluşturma ve düzenleme #

Oturum değişkenini bir değişken olarak kaydedin.

$session = Yii::$app->session;

$sess = $session['keys'];

Ardından istediğiniz dizi değerini oluşturun veya güncelleyin

$sess['first'] = 'abc';

Ve son olarak oturum değişkenine kaydedin

$session['keys'] = $sess
