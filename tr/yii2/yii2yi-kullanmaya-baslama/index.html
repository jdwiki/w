Kurulum veya Kurulum #

Yii2 iki şekilde kurulabilir. Bunlar

  1. Composer aracılığıyla yükleme
  2. Bir Arşiv Dosyasından Kurulum

Composer ile Kurulum #

Composer’ı Yükleme #

Eğer Composer kurulu değilse, bunu getcomposer.org adresindeki talimatları izleyerek yapabilirsiniz. Linux ve Mac OS X’te aşağıdaki komutları çalıştıracaksınız:

curl -sS https://getcomposer.org/installer | php
mv composer.phar /usr/local/bin/composer

Windows için composer-setup.exe dosyasını indirip yüklemeniz yeterlidir. Github API hız sınırını aşmak için github API erişim belirtecini yapılandırmanız gerekebilir.

Yii’yi yükleme #

Composer yüklendiğinde, Web’den erişilebilen bir klasör altında aşağıdaki komutları çalıştırarak Yii’yi yükleyebilirsiniz:

composer global require "fxp/composer-asset-plugin:^1.2.0"
composer create-project --prefer-dist yiisoft/yii2-app-basic basic

ardından Yii2’yi temel şablonla kurmak için aşağıdaki komutu çalıştırın.

composer create-project --prefer-dist --stability=dev yiisoft/yii2-app-basic basic

Yii2’yi gelişmiş şablon çalıştırma ile kurmak için

composer create-project --prefer-dist --stability=dev yiisoft/yii2-app-advanced advanced
cd advanced
php init

Bundan sonra yeni bir veritabanı oluşturun ve common/config/main-local.php içindeki [‘db’] bileşenlerinin konfigürasyonunu buna göre ayarlayın. sonra aşağıdaki komutu çalıştırın

php yii migrate

Arşiv Dosyasından Kurulum #

  1. Arşiv dosyasını Yii-download adresinden indirin

  2. İndirilen dosyayı Web’den erişilebilen bir klasöre açın.

  3. cookieValidationKey yapılandırma öğesi için gizli bir anahtar girerek config/web.php dosyasını değiştirin

    You can add any type of key you want:

     'cookieValidationKey' => '',

 For example : xyctuyvibonp

 'cookieValidationKey' => 'xyctuyvibonp',

    //insert a secret key in the following (if it is empty) - this is required by cookie validation ‘cookieValidationKey’ => ’enter your secret key here’,

ubuntu’da gelişmiş Yii2’yi yükleyin #

İlk önce besteciyi kurmamız gerekiyor. Besteci yükleme adımları Composer’ı yükleyin.

curl -sS https://getcomposer.org/installer | php

Şimdi dizini değiştirin:

sudo mv composer.phar /usr/local/bin/composer

Bestecinin çalıştığını kontrol edin

composer

Şimdi Composer yüklendi.

Yii2 ilerlemesini yüklemenin iki yolu vardır.

1.Bir Arşiv Dosyasından Yükleme

Aşağıdaki linkten zip dosyasını alın.

Hedef dizine açın, ör. /var/www/html.

https://github.com/yiisoft/yii2/releases/download/2.0.8/yii-advanced-app-2.0.8.tgz

“Gelişmiş” klasörün içine gidin. El ile hareket ettirin veya aşağıdaki komutu yazın.

cd advanced

Aşağıdaki komutu çalıştırın.

php init

2.Composer aracılığıyla yükleme

Besteci aracılığıyla yükleme, github kimlik doğrulama belirtecini gerektirir. Jeton için GitHub’a kaydolmanız gerekir.

Kaydolduktan sonra jetonunuzu oluşturabilirsiniz:

Bir jeton oluşturma adımları

  1. Herhangi bir sayfanın sağ üst köşesinde profil fotoğrafınızı ve ardından Ayarlar’ı tıklayın.
  2. Kullanıcı ayarları kenar çubuğunda Kişisel erişim belirteçleri’ni tıklayın.
  3. Yeni belirteç oluştur’a tıklayın.
  4. Simgenize açıklayıcı bir ad verin.
  5. Bu simgeye vermek istediğiniz kapsamları seçin.
  6. Belirteç oluştur’a tıklayın.
  7. Belirteci panonuza kopyalayın. Güvenlik nedeniyle, gezindikten sonra bu sayfanın dışında hiç kimse jetonu bir daha göremeyecek.

Referans: https://help.github.com/articles/creating-an-access-token-for-command-line-use/

Belirteç oluşturduktan sonra kopyalayın

Dizini değiştir

cd /var/www/html/

Aşağıdaki komutu çalıştırın

composer config -g github-oauth.github.com <AuthToken>

örnek:

composer config -g github-oauth.github.com f1eefb8f188c22dd6467f1883cb2615c194d1ce1

yii2’yi yükleyin

composer create-project --prefer-dist yiisoft/yii2-app-advanced advanced

“Gelişmiş” klasörün içine gidin. El ile hareket ettirin veya aşağıya yazın emretmek.

cd advanced

Aşağıdaki komutu çalıştırın.

php init

Bitti!

Şimdi kontrol edebilirsiniz.

http://localhost/gelişmiş/ön uç/web

ve

http://localhost/gelişmiş/arka uç/web
