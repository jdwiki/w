Yii2 sorgu oluşturucuyu kullanma #

Yii2, veri tabanından veri almak için verimli yollar sağlar. emp_id, emp_name ve emp_salary alanlarına sahip basit bir çalışan tablosu örneğini düşünün. Çalışan isimlerini ve maaşlarını almak için sorguyu kullanıyoruz.

select emp_name,emp_salary from employee

Yukarıdaki sorguyu Yii2’de oluşturmak için birçok yöntem vardır. Yöntemlerden biri bir yii\db\Query nesnesi kullanmaktır.

//creates a new \yii\db\Query() object
$query=new \yii\db\Query(); 

$rows=$query->select(['emp_name','emp_salary']) //specify required columns in an array
             ->from('employee') //specify table name
             ->all(); //returns an array of rows with each row being an associative array of name-value pairs.

$rows dizisindeki her bir ad-değer çifti arasında dolaşmak için bir foreach döngüsü kullanabiliriz.

foreach ($rows as $row) {
    echo "Employee Name: ".$row['emp_name'].",Employee Salary: ".$row['emp_salary']."<br>";
}

Bu çıktı

Çalışan Adı: Kiran,Çalışan Maaş: 25000

Çalışan Adı: Midhun, Çalışan Maaşı: 50000

Çalışan Adı: Jishnu, Çalışan Maaşı: 20000

Çalışan Adı: Ajith,Çalışan Maaşı: 25000

Çalışan Adı: Akshay,Çalışan Maaş: 750000

Daha fazla örnek

Diyelim ki maaşı 25000 olan çalışanların isimlerini bulmamız gerekiyor. Sorguyu sql olarak şu şekilde yazabiliriz.

select emp_name from employee where salary=25000

Yii2’de, yukarıdaki sorguyu oluşturma kodu

$query=new \yii\db\Query(); 

$rows=$query->select(['emp_name']) 
            ->from('employee')
            ->where(['emp_salary'=>25000]) //specify the condition as an associative array where key is column name
            ->all();

Maaşı 25000’den fazla olan çalışan isimlerini bulmamız gerekirse, Yii2’deki kodu şöyle yazabiliriz.

  $rows=$query->select(['emp_name']) 
        ->from('employee')
        ->where(['>','emp_salary', 25000]) 
//Here first element in the above array specify relational operator used, second element specify the table name and third the value itself.
        ->all();

where() kullanarak daha fazla durum kontrolü #

Aşağıdaki gibi where() yöntemi kullanılarak birden çok koşul yazılabilir.

// Creates a new \yii\db\Query() object
$query = new \yii\db\Query();
$rows = $query->select(['emp_name','emp_salary']) 
        ->from('employee')
        ->where(['emp_name' => 'Kiran', 'emp_salary' => 25000]) // Specify multiple conditions
        ->one(); // Returns the first row of the result

Yukarıdaki kod, adı kiran ve maaşı 25000 olan bir çalışanı getirecektir. Birden fazla çalışan yukarıdaki koşulu karşılıyorsa, one() çağrısı yalnızca ilk sonucun alınmasını sağlar. Tüm sonuçları almak için all() kullanmalısınız.

all() işlevini kullanırsanız sonucun her zaman bir dizi olacağını unutmayın; Sadece bir veya sıfır sonuç olsa bile. Bu dizi, tüm sonuçları dizi olarak içerir veya hiçbir kayıt eşleşmediğinde boştur. one() çağrısı, sonuç dizisini doğrudan döndürür veya sorgu hiçbir şey döndürmezse false döndürür.

SQL’deki eşdeğer kod aşağıda verilmiştir.

select emp_name, emp_salary from employee where emp_name = 'Kiran' and emp_salary = 25000 limit 1;

Yukarıdaki sorguyu Yii2’de yazmanın alternatif bir yolu aşağıda verilmiştir.

$rows = $query->select(['emp_name', 'emp_salary']) 
    ->from('employee')
    ->where(['emp_name' => 'Kiran'])
    ->andWhere(['emp_salary' => 25000])
    ->one();

Ek koşullar kümesi andWhere kullanılarak belirtilebilir. Bu, sorguya daha sonra ek koşul denetimi eklememiz gerekirse yararlı olacaktır.

Birden çok koşul belirtmenin bir başka yolu da işleç biçimi where() yöntemini kullanmaktır. Yukarıdaki sorgu aşağıda verildiği gibi de yazılabilir.

 $rows = $query->select(['emp_name','emp_salary']) 
    ->from('employee')
    ->where(['and', 'emp_name="kiran"', 'emp_salary=25000'])        
    ->one();

Burada dizideki ilk eleman olarak ‘ve’ operatörünü belirtiyoruz. Benzer şekilde, ‘veya’, ‘arasında’, ‘not arasında’, ‘in’, ‘not in’, ‘** kullanabiliriz. gibi**’, ‘veya beğen’, ‘beğenmedim’, ‘veya beğenmedim’, ‘var’, ‘yok’, Operatör olarak ‘>’, ‘<=’ vb.

‘in’ ve ‘beğen’ kullanımına örnekler

Diyelim ki maaşları 20000, 25000 ve 50000 olan çalışanları bulmamız gerekiyor. Normal sql’de sorguyu şöyle yazardık

select * from employee where salary in (20000,25000,50000)

Yii2’de bunu aşağıdaki gibi yazabiliriz.

$rows = $query->from('employee')
        ->where(['emp_salary' => [20000,25000,50000]]) 
        ->all();

Aynı koşulu belirtmenin başka bir yolu da

$rows = $query->from('employee')
    ->where(['in', 'emp_salary', [20000,25000,50000]]) // Making use of operator format of where() method
    ->all();

Benzer şekilde, 20000, 25000 ve 50000 maaşı olmayan tüm çalışanları almak istiyorsak, ‘in’ yerine ‘not in’ belirtilebilir.

Şimdi where() koşulunda ‘like’ kullanımına ilişkin bazı örnekler görelim. Adında ‘gopal’ dizgisine sahip tüm çalışanları bulmamız gerektiğini varsayalım. İsimler venugopal, rajagopal, gopalakrishnan vb olabilir. Sql sorgusu aşağıda verilmiştir.

select * from employee where emp_name like '%gopal%'

Yii2’de bunu şöyle yazacağız

 $rows = $query->from('employee')
        ->where(['like', 'emp_name', 'gopal']) // Making use of operator format of where() method
        ->all();

Adlarında ‘gopal’ ve ‘nair’ dizesine sahip tüm çalışanları bulmamız gerekirse. olarak yazabiliriz

   $rows = $query->from('employee')
        ->where(['like', 'emp_name', ['gopal','nair']]) // Making use of operator format of where() method
        ->all();

Bu olarak değerlendirir

‘%gopal%’ ve ‘%nair%’ gibi emp_name olan çalışandan * seçin

Benzer şekilde, adlarında ‘gopal’ ve ‘nair’ dizgisine sahip olmayan tüm çalışanları belirtmek için ‘beğenmedim’ ifadesini kullanabiliriz.

orderBy() kullanma #

orderBy() yöntemi, bir SQL sorgusunun ORDER BY parçasını belirtir.Örneğin, emp_id, emp_first_name, emp_last_name ve emp_salary alanlarına sahip çalışan tablomuzu düşünün.Sonucu, çalışan maaşlarının sırasını artırarak sıralamamız gerektiğini varsayalım. aşağıda verildiği gibi sql’de yapabilir.

Select * from employee order by emp_salary

yii2’de sorguyu aşağıdaki gibi oluşturabiliriz.

 //creates a new \yii\db\Query() object
    $query=new \yii\db\Query();

    $rows= $query->from('employee')->orderBy([
    'emp_salary' => SORT_ASC //specify sort order ASC for ascending DESC for descending      
    ])->all();

Çalışanları önce isimleri artan sonra maaşlarını azalan şekilde sıralamamız gerekirse düz sql olarak aşağıdaki gibi yazabiliriz.

Select * from employee order by emp_first_name ASC, emp_salary DESC

Eşdeğer sql, yii2 kullanılarak aşağıdaki gibi oluşturulabilir.

//creates a new \yii\db\Query() object
    $query=new \yii\db\Query();

    $rows= $query->from('employee')->orderBy([
    'emp_first_name' => SORT_ASC
    'emp_salary' => SORT_DESC      
    ])->all();

Ayrıca, ham SQL deyimleri yazarken yaptığınız gibi, bir dize kullanarak ORDER BY belirtebilirsiniz. Örneğin yukarıdaki sorgu aşağıdaki gibi de oluşturulabilir.

//creates a new \yii\db\Query() object
$query=new \yii\db\Query();
$rows=$query->from('employee')->orderBy('emp_first_name ASC, emp_salary DESC')->all();

ORDER BY parçasına ek sütunlar eklemek için addOrderBy() öğesini çağırabilirsiniz. Örneğin

//creates a new \yii\db\Query() object
$query=new \yii\db\Query();
$rows=$query->from('employee')->orderBy('emp_first_name ASC')
    ->addOrderBy('emp_salary DESC')->all();
