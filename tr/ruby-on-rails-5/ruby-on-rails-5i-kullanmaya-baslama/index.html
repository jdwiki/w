İlk merhaba dünya sayfanızı oluşturma #

Windows’ta komuttan ve Linux’ta Terminal’den yeni bir Rails uygulaması ‘merhaba dünya’ oluşturun.

rails new hello-world

Şimdi yeni uygulama dizinine gidin

cd hello-world

Şimdi bir denetleyici oluşturun

rails generate controller hello_world index

Burada “index”, “hello_world” denetleyicisindeki yöntemin adıdır. Uygulama dizininizde app/controllers/hello_world_controller.rb dosyasını açarak kontrol edebilirsiniz. Kod şöyle görünür:

class HelloWorldController < ApplicationController
  def index
  end
end

“config/routes.rb” dosyanıza, yönteminize işaret eden bir “route” otomatik olarak eklenir. routes.rb dosyanızdaki koda bakın.

    Rails.application.routes.draw do
      get 'hello_world/index'

      # For details on the DSL available within this file, see http://guides.rubyonrails.org/routing.html
   end

Şimdi app/views/hello_world/index.html.rb dosyasını açın Tüm içeriği temizle ve yaz

Merhaba, bu benim ilk Rails sayfam.

Rails sunucusunu başlat:

rails server

Bu url’yi tarayıcınızda açın:

http://localhost:3000/hello_world/

Görmelisin:

Hello, this is my first rails page

Yeni sayfanızı, ana sayfanız yapın. config klasöründeki route.rb dosyasında get 'hello_world/index' satırını kaldırın ve şunu ekleyin:

root 'hello_world#index'

Şimdi aç: http://localhost:3000/ Göreceksiniz: ‘Merhaba, bu benim ilk raylarım’ Bitirdiniz.

Ubuntu 17.04 Zesty Zapus’ta Ruby On Rails Kurulumu #

Bu yaklaşık 30 dakika sürecektir. Ubuntu 16.10 Yakkety Yak’ta Ruby on Rails Geliştirme Ortamı ayarlayacağız. En son Masaüstü sürümünü buradan indirmek isteyeceksiniz: http://releases.ubuntu.com/17.04/

Ctrl + Alt + T tuşlarını kullanarak terminalinizi açın.

Ruby’yi Kurmak #

İlk adım Ruby için Bağımlılıkları kurmaktır.

sudo apt-get update
sudo apt-get install git-core curl zlib1g-dev build-essential libssl-dev libreadline-dev libyaml-dev libsqlite3-dev sqlite3 libxml2-dev libxslt1-dev libcurl4-openssl-dev python-software-properties libffi-dev nodejs

Ruby 2.4.0 sürümünü kullanacağız ve bu tavsiye edilir. Ruby’yi üç yöntemden birini kullanarak kurun. Her birinin kendi faydaları vardır, çoğunlukla insanlar rbenv’i tercih eder, ancak rvm’ye aşina iseniz, bunun için gidin.

Not: Yalnızca ‘rbenv’ kullanıcılarının, ‘bundler’ı kurduktan sonra ‘rbenv rehash’i çalıştırması gerekir.

rbenv ile yükleme: #

cd
git clone https://github.com/rbenv/rbenv.git ~/.rbenv
echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc
echo 'eval "$(rbenv init -)"' >> ~/.bashrc
exec $SHELL

git clone https://github.com/rbenv/ruby-build.git ~/.rbenv/plugins/ruby-build
echo 'export PATH="$HOME/.rbenv/plugins/ruby-build/bin:$PATH"' >> ~/.bashrc
exec $SHELL

rbenv install 2.4.0
rbenv global 2.4.0
ruby -v

Ardından Bundler’ı yükleyin:

gem install bundler

Paketleyiciyi kurduktan sonra ‘rbenv rehash’ kullanın.

rvm ile yükleme: #

sudo apt-get install libgdbm-dev libncurses5-dev automake libtool bison libffi-dev
gpg --keyserver hkp://keys.gnupg.net --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3
curl -sSL https://get.rvm.io | bash -s stable
source ~/.rvm/scripts/rvm
rvm install 2.4.0
rvm use 2.4.0 --default
ruby -v

Ardından Bundler’ı yükleyin:

gem install bundler

Kaynakla yükleme: #

cd
wget http://ftp.ruby-lang.org/pub/ruby/2.4/ruby-2.4.0.tar.gz
tar -xzvf ruby-2.4.0.tar.gz
cd ruby-2.4.0/
./configure
make
sudo make install
ruby -v

Ardından Son adım Bundler’ı kurmaktır:

gem install bundler

Rayların Kurulması #

Rails bugünlerde o kadar çok bağımlılıkla geliyor ki, NodeJS gibi bir JavaScript çalışma zamanı kurmamız gerekecek. Bu, daha hızlı bir üretim ortamı sağlamak için JavaScript’inizi birleştiren ve küçülten Rails’de Coffee-script ve Asset Pipeline kullanmanıza olanak tanır.

Resmi depoyu kullanarak NodeJS’yi kurun:

curl -sL https://deb.nodesource.com/setup_4.x | sudo -E bash -
sudo apt-get install -y nodejs

Şimdi ‘Rails 5.0.1’i yükleyin:

gem install rails -v 5.0.1

‘rbenv’ kullanıyorsanız, Rails’i çalıştırılabilir hale getirmek için aşağıdaki komutu çalıştırın:

 rbenv rehash

Şimdi Raylar Kuruldu, Rayların doğru şekilde kurulduğundan emin olmak için rails -v komutunu çalıştırın:

rails -v
# Rails 5.0.1

Herhangi bir nedenle farklı bir sonuç alıyorsanız, ortamınız düzgün kurulmamış olabilir.

PostgreSQL’i Ayarlama #

Rails, varsayılan veritabanı olarak sqlite3 ile birlikte gelir. Diskte basit bir dosya olarak saklandığı için muhtemelen kullanmak istemeyeceksiniz. Muhtemelen MySQL veya PostgreSQL gibi daha sağlam bir şey isteyeceksiniz.

Her ikisi hakkında da çok sayıda belge var, bu yüzden daha rahat edeceğiniz gibi görünen birini seçebilirsiniz.

sudo sh -c "echo 'deb http://apt.postgresql.org/pub/repos/apt/ xenial-pgdg main' > /etc/apt/sources.list.d/pgdg.list"
wget --quiet -O - http://apt.postgresql.org/pub/repos/apt/ACCC4CF8.asc | sudo apt-key add -
sudo apt-get update
sudo apt-get install postgresql-common
sudo apt-get install postgresql-9.5 libpq-dev

PostgreSQL kurulumu sizin için bir kullanıcı kurmaz, bu nedenle veritabanları oluşturma izni olan bir kullanıcı oluşturmak için bu adımları izlemeniz gerekir. “Hizqeel"i kullanıcı adınızla değiştirmekten çekinmeyin.

sudo -u postgres createuser hizqeel -s

# If you would like to set a password for the user, you can do the following
sudo -u postgres psql
postgres=# \password hizqeel

Kurulum veya Kurulum #

Windows Platformunda

Adım 1: Ruby Kurulumu

Ruby’yi bilgisayarınıza zaten yüklediyseniz, bu adımı atlayabilirsiniz.

  • rubyinstaller.org’un indirme sayfasına gidin ve windows platformunuzun mimarisine karşılık gelen Ruby’nin kararlı sürümlerinden birini indirin.
  • Ruby sürümünüz için geliştirme kitini tekrar indirin [buradan] 1.
  • Şimdi Ruby’yi kurun.
  • Ruby kurulumunuzu Ruby geliştirme kiti ile geliştirmenin tam zamanı. Bu, bazı mücevherlerin bilgisayarınızda derlenmesi için gereklidir.
  • Ruby geliştirme kiti dosyanızı, Ruby kurulum yolunuz C:\Ruby-** yakınındaki bir klasöre çıkarın. Örneğin, “C:\rubyDevkit”.
  • Şimdi cmd.exe’yi açın ve devkit’inizi çıkardığınız dizine gidin.
  • Ardından ruby dk.rb init komutunu çalıştırın. Bu, kurulumunuzu başlatacaktır. Ardından ruby dk.rb install komutunu çalıştırın.

Adım 2: Rails’i Kurun

  • Ruby’nin başarılı kurulumundan sonra. Bir sonraki adım rayları kurmaktır. Rails’i kurmadan önce, komut istemi pencerenizden `gem install bundler’ı çalıştıran paketleyici gem’i kurun.

  • Paketleyiciyi kurduktan sonra şimdi “gem install Rails -v version_of_rails” örneğini “-v 5.0.0.1” çalıştırın.

  • Tüm işlemler hatasız tamamlandıysa rayları başarıyla kurdunuz.
