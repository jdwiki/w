Birçok programlama dili, süreçler arasında veya cihazlar arasında iletişim kurmak için soketleri kullanır. Bu konu, ortak ağ protokolleri üzerinden veri göndermeyi ve almayı kolaylaştırmak için Python’daki soket modülünün doğru kullanımını açıklar.

Parametreler #

parametre Açıklama
soket.AF_UNIX UNIX Soketi
soket.AF_INET IPv4
soket.AF_INET6 IPv6
soket.SOCK_STREAM TCP
soket.SOCK_DGRAM UDP

UDP aracılığıyla veri gönderme #

UDP bağlantısız bir protokoldür. Diğer işlemlere veya bilgisayarlara mesajlar herhangi bir bağlantı kurulmadan gönderilir. Mesajınız alındıysa otomatik onay yoktur. UDP genellikle gecikmeye duyarlı uygulamalarda veya ağ çapında yayınlar gönderen uygulamalarda kullanılır.

Aşağıdaki kod, UDP kullanarak yerel ana bilgisayar bağlantı noktası 6667’yi dinleyen bir işleme bir mesaj gönderir

UDP bağlantısız olduğundan, gönderme işleminden sonra soketi “kapatmaya” gerek olmadığını unutmayın.

from socket import socket, AF_INET, SOCK_DGRAM
s = socket(AF_INET, SOCK_DGRAM)
msg = ("Hello you there!").encode('utf-8')  # socket.sendto() takes bytes as input, hence we must encode the string first.
s.sendto(msg, ('localhost', 6667))

UDP üzerinden veri alma #

UDP bağlantısız bir protokoldür. Bu, mesaj gönderen eşlerin mesaj göndermeden önce bağlantı kurmasını gerektirmediği anlamına gelir. socket.recvfromböylece bir demet döndürür (msg [soketin aldığı mesaj], addr [gönderenin adresi])

Yalnızca “soket” modülünü kullanan bir UDP sunucusu:

from socket import socket, AF_INET, SOCK_DGRAM
sock = socket(AF_INET, SOCK_DGRAM)
sock.bind(('localhost', 6667))

while True:
    msg, addr = sock.recvfrom(8192)  # This is the amount of bytes to read at maximum
    print("Got message from %s: %s" % (addr, msg))

Aşağıda “socketserver.UDPServer” kullanan alternatif bir uygulama bulunmaktadır:

from socketserver import BaseRequestHandler, UDPServer

class MyHandler(BaseRequestHandler):
    def handle(self):
        print("Got connection from: %s" % self.client_address)
        msg, sock = self.request
        print("It said: %s" % msg)
        sock.sendto("Got your message!".encode(), self.client_address) # Send reply

serv = UDPServer(('localhost', 6667), MyHandler)
serv.serve_forever()

Varsayılan olarak, “sockets” bloğu. Bu, betiğin yürütülmesinin soket veri alana kadar bekleyeceği anlamına gelir.

TCP üzerinden veri gönderme #

İnternet üzerinden veri göndermek, çoklu modüller kullanılarak mümkün hale getirilmiştir. Soket modülü, diğer bilgisayarlardan veya işlemlerden veri göndermekten veya almaktan sorumlu olan temel İşletim Sistemi işlemlerine düşük düzeyde erişim sağlar.

Aşağıdaki kod, b'Hello' bayt dizesini ana bilgisayar yerel ana bilgisayarındaki 6667 numaralı bağlantı noktasını dinleyen bir TCP sunucusuna gönderir ve bittiğinde bağlantıyı kapatır:

from socket import socket, AF_INET, SOCK_STREAM
s = socket(AF_INET, SOCK_STREAM)
s.connect(('localhost', 6667))  # The address of the TCP server listening
s.send(b'Hello')
s.close()

Soket çıkışı varsayılan olarak engellenir, bu, programın bağlantıda bekleyeceği ve eylem ’tamamlanana’ kadar çağrı göndereceği anlamına gelir. Bağlanmak için bu, sunucunun bağlantıyı gerçekten kabul ettiği anlamına gelir. Göndermek için yalnızca işletim sisteminin daha sonra gönderilecek verileri kuyruğa almak için yeterli arabellek alanına sahip olduğu anlamına gelir.

Kullanımdan sonra prizler daima kapatılmalıdır.

Çok iş parçacıklı TCP Soket Sunucusu #

Argümansız çalıştırıldığında, bu program ‘5000’ bağlantı noktasındaki ‘127.0.0.1’ bağlantılarını dinleyen bir TCP soket sunucusunu başlatır. Sunucu, her bağlantıyı ayrı bir iş parçacığında işler.

-c argümanı ile çalıştırıldığında, bu program sunucuya bağlanır, istemci listesini okur ve çıktısını alır. İstemci listesi bir JSON dizesi olarak aktarılır. İstemci adı -n argümanı iletilerek belirtilebilir. Farklı isimler geçilerek müşteri listesindeki etkisi gözlemlenebilir.

client_list.py

import argparse
import json
import socket
import threading

def handle_client(client_list, conn, address):
    name = conn.recv(1024)
    entry = dict(zip(['name', 'address', 'port'], [name, address[0], address[1]]))
    client_list[name] = entry
    conn.sendall(json.dumps(client_list))
    conn.shutdown(socket.SHUT_RDWR)
    conn.close()

def server(client_list):
    print "Starting server..."
    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    s.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1)
    s.bind(('127.0.0.1', 5000))
    s.listen(5)
    while True:
        (conn, address) = s.accept()
        t = threading.Thread(target=handle_client, args=(client_list, conn, address))
        t.daemon = True
        t.start()

def client(name):
    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    s.connect(('127.0.0.1', 5000))
    s.send(name)
    data = s.recv(1024)
    result = json.loads(data)
    print json.dumps(result, indent=4)

def parse_arguments():
    parser = argparse.ArgumentParser()
    parser.add_argument('-c', dest='client', action='store_true')
    parser.add_argument('-n', dest='name', type=str, default='name')
    result = parser.parse_args()
    return result

def main():
    client_list = dict()
    args = parse_arguments()
    if args.client:
        client(args.name)
    else:
        try:
            server(client_list)
        except KeyboardInterrupt:
            print "Keyboard interrupt"

if __name__ == '__main__':
    main()

Sunucu Çıkışı

$ python client_list.py
Starting server...

İstemci Çıktısı

$ python client_list.py -c -n name1
{
    "name1": {
        "address": "127.0.0.1", 
        "port": 62210, 
        "name": "name1"
    }
}

Alma arabellekleri 1024 bayt ile sınırlıdır. İstemci listesinin JSON dize gösterimi bu boyutu aşarsa, kesilecektir. Bu, aşağıdaki istisnanın ortaya çıkmasına neden olur:

ValueError: Unterminated string starting at: line 1 column 1023 (char 1022)

Önce ağ kartınızın otomatik sağlama toplamını devre dışı bırakırsınız:

sudo ethtool -K eth1 tx off

Ardından paketinizi bir SOCK_RAW soketi kullanarak gönderin:

#!/usr/bin/env python
from socket import socket, AF_PACKET, SOCK_RAW
s = socket(AF_PACKET, SOCK_RAW)
s.bind(("eth1", 0))

# We're putting together an ethernet frame here, 
# but you could have anything you want instead
# Have a look at the 'struct' module for more 
# flexible packing/unpacking of binary data
# and 'binascii' for 32 bit CRC
src_addr = "\x01\x02\x03\x04\x05\x06"
dst_addr = "\x01\x02\x03\x04\x05\x06"
payload = ("["*30)+"PAYLOAD"+("]"*30)
checksum = "\x1a\x2b\x3c\x4d"
ethertype = "\x08\x01"

s.send(dst_addr+src_addr+ethertype+payload+checksum)
