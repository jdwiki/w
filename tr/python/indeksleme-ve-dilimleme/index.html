Sözdizimi #

  • nesne[başlat:durdur:adım]
  • dilim (durdur)
  • dilim(başla, durdur[, adım])

Parametreler #

parametre Tanım
“nesne” İçinden bir “alt nesne” çıkarmak istediğiniz nesne
“başlat” Alt nesnenin başlamasını istediğiniz “obj” dizini (Python’un sıfır dizinli olduğunu, yani “obj” öğesinin ilk öğesinin “0” dizinine sahip olduğunu unutmayın). Atlanırsa, varsayılan olarak “0” olur.
“dur” Alt nesnenin bitmesini istediğiniz “nesne"nin (dahil olmayan) dizini. Atlanırsa, varsayılan olarak “len(obj)” olur.
“adım” Yalnızca her “adım” öğesini seçmenizi sağlar. Atlanırsa, varsayılan olarak “1” olur.

Bir unicode karakterin 1 uzunluğunda bir dizeyle temsil edildiği uyarısıyla, dizeleri değişmez bir karakter koleksiyonu olarak görüntüleyerek dizeleri dilimleme kavramını diğer dizileri dilimleme konseptiyle birleştirebilirsiniz.

Matematiksel gösterimde, yarı açık bir “[başlangıç, bitiş)” aralığı kullanmak için dilimlemeyi düşünebilirsiniz, yani başlangıç ​​dahil ancak son dahil değildir. Aralığın yarı açık doğası, ’len(x[:n])’ = ’n’, burada ’len(x)’ > = ’n’ avantajına sahiptir, ancak aralığın başlangıçta kapalı olması avantajlıdır. bu ‘x[n:n+1]’ = ‘[x[n]]’ burada ‘x’, ’len(x) >= n’ içeren bir listedir, böylece indeksleme ve dilimleme gösterimi arasında tutarlılık sağlanır.

Temel Dilimleme #

Herhangi bir yinelenebilir için (örneğin bir dize, liste vb. için), Python, verilerinin bir alt dizesini veya alt listesini dilimlemenize ve döndürmenize olanak tanır.

Dilimleme biçimi:

iterable_name[start:stop:step]

nerede,

  • “start”, dilimin ilk dizinidir. Varsayılan olarak 0’dır (ilk öğenin dizini)
  • dilimin son dizinini geçtikten sonra “dur”. Varsayılan olarak len(yinelenebilir)
  • “adım” adım boyutudur (aşağıdaki örneklerle daha iyi açıklanmıştır)

Örnekler:

a = "abcdef"
a            # "abcdef" 
             # Same as a[:] or a[::] since it uses the defaults for all three indices
a[-1]        # "f"
a[:]         # "abcdef" 
a[::]        # "abcdef" 
a[3:]        # "def" (from index 3, to end(defaults to size of iterable)) 
a[:4]        # "abcd" (from beginning(default 0) to position 4 (excluded)) 
a[2:4]       # "cd" (from position 2, to position 4 (excluded))

Ek olarak, yukarıdakilerden herhangi biri tanımlanan adım boyutuyla kullanılabilir:

a[::2]       # "ace" (every 2nd element)
a[1:4:2]     # "bd" (from index 1, to index 4 (excluded), every 2nd element)

Endeksler negatif olabilir, bu durumda dizinin sonundan hesaplanırlar.

a[:-1]     # "abcde" (from index 0 (default), to the second last element (last element - 1))
a[:-2]     # "abcd" (from index 0 (default), to the third last element (last element -2))
a[-1:]     # "f" (from the last element to the end (default len())

Adım boyutları da negatif olabilir, bu durumda dilim listede ters sırada yinelenir:

a[3:1:-1]   # "dc" (from index 2 to None (default), in reverse order)

Bu yapı, yinelenebilir bir durumu tersine çevirmek için kullanışlıdır.

a[::-1]     # "fedcba" (from last element (default len()-1), to first, in reverse order(-1))

Negatif adımlar için varsayılan “end_index” değerinin “Yok” olduğuna dikkat edin (bkz. http://stackoverflow.com/a/12521981)

a[5:None:-1] # "fedcba" (this is equivalent to a[::-1])
a[5:0:-1]    # "fedcb" (from the last element (index 5) to second element (index 1)

Bir nesneyi ters çevirme #

Bir “str”, “list” veya “tuple"ı (veya temel olarak step parametresiyle dilimleme uygulayan herhangi bir koleksiyon nesnesini) çok kolay bir şekilde tersine çevirmek için dilimleri kullanabilirsiniz. Aşağıda, yukarıda listelenen diğer türler için eşit derecede geçerli olmasına rağmen, bir dizgiyi tersine çevirmenin bir örneği verilmiştir:

s = 'reverse me!'
s[::-1]    # '!em esrever'

Hızlıca sözdizimine bakalım. “[::-1]”, dilimin dizenin başından sonuna kadar olması gerektiği anlamına gelir (çünkü “start” ve “end” atlanmıştır) ve bir “-1” adımı, dilimin dizi boyunca hareket etmesi gerektiği anlamına gelir. ters dize.

Dilim atama #

Dilimleri kullanan bir başka güzel özellik de dilim atamadır. Python, tek bir işlemde bir listenin eski dilimlerini değiştirmek için yeni dilimler atamanıza izin verir.

Bu, bir listeniz varsa, tek bir atamada birden çok üyeyi değiştirebileceğiniz anlamına gelir:

lst = [1, 2, 3]
lst[1:3] = [4, 5]
print(lst) # Out: [1, 4, 5]

Atama boyut olarak da eşleşmemelidir, bu nedenle eski bir dilimi farklı boyutta yeni bir dilimle değiştirmek isterseniz şunları yapabilirsiniz:

lst = [1, 2, 3, 4, 5]
lst[1:4] = [6]
print(lst) # Out: [1, 6, 5]

Tüm listeyi değiştirmek gibi şeyler yapmak için bilinen dilimleme sözdizimini kullanmak da mümkündür:

lst = [1, 2, 3]
lst[:] = [4, 5, 6]
print(lst) # Out: [4, 5, 6]

Veya sadece son iki üye:

lst = [1, 2, 3]
lst[-2:] = [4, 5, 6]
print(lst) # Out: [1, 4, 5, 6]

Bir dizinin sığ bir kopyasını oluşturma #

Bir dizinin bir kopyasını oluşturmanın hızlı bir yolu (orijinal diziye başka bir referansla bir değişken atamanın aksine):

arr[:]

Sözdizimini inceleyelim. “[:]”, “başlangıç”, “bitiş” ve “dilim” öğelerinin atlandığı anlamına gelir. Sırasıyla ‘0’, ’len(arr)’ ve ‘1’ varsayılanlarıdır, yani talep ettiğimiz alt dizi, baştan sona kadar tüm ‘dizi’ öğelerine sahip olacaktır.

Pratikte, bu şuna benzer:

arr = ['a', 'b', 'c']
copy = arr[:]
arr.append('d')
print(arr)    # ['a', 'b', 'c', 'd']
print(copy)   # ['a', 'b', 'c']

Gördüğünüz gibi, ‘arr.append(’d’)’, ‘arr’a ’d’yi ekledi, ancak ‘kopyalama’ değişmeden kaldı!

Bunun sığ bir kopya oluşturduğuna ve “arr.copy()” ile aynı olduğuna dikkat edin.

Özel sınıfları indeksleme: getitem, setitem ve delitem #

class MultiIndexingList:
    def __init__(self, value):
        self.value = value
        
    def __repr__(self):
        return repr(self.value)
    
    def __getitem__(self, item):
        if isinstance(item, (int, slice)):
            return self.__class__(self.value[item])
        return [self.value[i] for i in item]
    
    def __setitem__(self, item, value):
        if isinstance(item, int):
            self.value[item] = value
        elif isinstance(item, slice):
            raise ValueError('Cannot interpret slice with multiindexing')
        else:
            for i in item:
                if isinstance(i, slice):
                    raise ValueError('Cannot interpret slice with multiindexing')
                self.value[i] = value
    
    def __delitem__(self, item):
        if isinstance(item, int):
            del self.value[item]
        elif isinstance(item, slice):
            del self.value[item]
        else:
            if any(isinstance(elem, slice) for elem in item):
                raise ValueError('Cannot interpret slice with multiindexing')
            item = sorted(item, reverse=True)
            for elem in item:
                del self.value[elem]

Bu, eleman erişimi için dilimleme ve indeksleme sağlar:

a = MultiIndexingList([1,2,3,4,5,6,7,8])
a
# Out: [1, 2, 3, 4, 5, 6, 7, 8]
a[1,5,2,6,1]
# Out: [2, 6, 3, 7, 2]
a[4, 1, 5:, 2, ::2]
# Out: [5, 2, [6, 7, 8], 3, [1, 3, 5, 7]]
#       4|1-|----50:---|2-|-----::2-----   <-- indicated which element came from which index

Öğeleri ayarlarken ve silerken yalnızca virgülle ayrılmış tamsayı dizinlemeye izin verilir (dilimleme yok):

a[4] = 1000
a
# Out: [1, 2, 3, 4, 1000, 6, 7, 8]
a[2,6,1] = 100
a
# Out: [1, 100, 100, 4, 1000, 6, 100, 8]
del a[5]
a
# Out: [1, 100, 100, 4, 1000, 100, 8]
del a[4,2,5]
a
# Out: [1, 100, 4, 8]

Temel Dizin Oluşturma #

Python listeleri 0 tabanlıdır yani listedeki ilk öğeye “0” dizini ile erişilebilir

arr = ['a', 'b', 'c', 'd']
print(arr[0])
>> 'a'

Listedeki ikinci öğeye “1” dizini ile, üçüncü öğeye “2” dizini vb. ile erişebilirsiniz:

print(arr[1])
>> 'b'
print(arr[2])
>> 'c'

Listenin sonundaki öğelere erişmek için negatif endeksleri de kullanabilirsiniz. Örneğin. “-1” dizini size listenin son öğesini verir ve “-2” dizini size listenin sondan ikinci öğesini verir:

print(arr[-1])
>> 'd'
print(arr[-2])
>> 'c'

Listede olmayan bir dizine erişmeye çalışırsanız, bir ‘IndexError’ ortaya çıkar:

print arr[6]
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list index out of range

Nesneleri dilimleyin #

Dilimler kendi içlerinde nesnelerdir ve yerleşik slice() işleviyle değişkenlerde saklanabilir. Dilim değişkenleri, kodunuzu daha okunabilir kılmak ve yeniden kullanımı teşvik etmek için kullanılabilir.

>>> programmer_1 = [ 1956, 'Guido', 'van Rossum', 'Python', 'Netherlands']
>>> programmer_2 = [ 1815, 'Ada', 'Lovelace', 'Analytical Engine', 'England']
>>> name_columns = slice(1, 3)
>>> programmer_1[name_columns]
['Guido', 'van Rossum']
>>> programmer_2[name_columns]
['Ada', 'Lovelace']
