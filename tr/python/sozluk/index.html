Parametreler #

parametre Ayrıntılar
anahtar Aramak için istenen tuş
değer Ayarlanacak veya döndürülecek değer

Sözlük oluştururken hatırlanması gereken faydalı öğeler:

  • Her anahtar benzersiz olmalıdır (aksi takdirde geçersiz kılınır)
  • Her anahtar hashable olmalıdır (karma için ‘hash’ işlevini kullanabilir; aksi takdirde ‘TypeError’ atılır)
  • Anahtarlar için belirli bir sıra yoktur.

Sözlüğe Giriş #

Sözlük, Python’da Mapping olarak da bilinen bir key value store örneğidir. Bir anahtara başvurarak öğeleri saklamanıza ve almanıza olanak tanır. Sözlüklere anahtar ile atıfta bulunulduğundan, çok hızlı aramaları vardır. Esas olarak öğelere anahtara göre referans vermek için kullanıldıklarından, sıralanmazlar.

bir dict oluşturma #

Sözlükler birçok şekilde başlatılabilir:

değişmez sözdizimi #

d = {}                        # empty dict
d = {'key': 'value'}          # dict with initial values

# Also unpacking one or multiple dictionaries with the literal syntax is possible

# makes a shallow copy of otherdict
d = {**otherdict}
# also updates the shallow copy with the contents of the yetanotherdict.
d = {**otherdict, **yetanotherdict}
son sürüm

dikte anlama #

d = {k:v for k,v in [('key', 'value',)]}

ayrıca bkz: Anlayışlar

yerleşik sınıf: dict() #

d = dict()                    # emtpy dict
d = dict(key='value')         # explicit keyword arguments
d = dict([('key', 'value')])  # passing in a list of key/value pairs
# make a shallow copy of another dict (only possible if keys are only strings!)
d = dict(**otherdict)

bir dict değiştirme #

Bir sözlüğe öğe eklemek için, değeri olan yeni bir anahtar oluşturmanız yeterlidir:

d['newkey'] = 42

Değer olarak “list” ve “sözlük” eklemek de mümkündür:

d['new_list'] = [1, 2, 3]
d['new_dict'] = {'nested_dict': 1}

Bir öğeyi silmek için anahtarı sözlükten silin:

del d['newkey']

KeyError İstisnalarından Kaçınma #

Sözlükleri kullanırken yaygın olarak görülen bir tuzak, var olmayan bir anahtara erişmektir. Bu genellikle bir “KeyError” istisnasıyla sonuçlanır

mydict = {}
mydict['not there']

Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
KeyError: 'not there'

Anahtar hatalarından kaçınmanın bir yolu, anahtarın olmaması durumunda döndürülecek varsayılan bir değer belirlemenize olanak tanıyan ‘dict.get’ yöntemini kullanmaktır.

value = mydict.get(key, default_value)

Bu, varsa “mydict[key]” döndürür, aksi takdirde “varsayılan_değer” döndürür. Bunun “mydict"e “anahtar” eklemediğini unutmayın. Bu nedenle, bu anahtar değer çiftini korumak istiyorsanız, anahtar değer çiftini * tutan* mydict.setdefault(key, default_value) kullanmalısınız.

mydict = {}
print(mydict)
# {}
print(mydict.get("foo", "bar"))
# bar
print(mydict)
# {}
print(mydict.setdefault("foo", "bar"))
# bar
print(mydict)
# {'foo': 'bar'}

Sorunla başa çıkmanın alternatif bir yolu, istisnayı yakalamaktır.

try:
    value = mydict[key]
except KeyError:
    value = default_value

Anahtarın sözlükte olup olmadığını da kontrol edebilirsiniz.

if key in mydict:
    value = mydict[key]
else:
    value = default_value

Bununla birlikte, çok iş parçacıklı ortamlarda, kontrol ettikten sonra anahtarın sözlükten kaldırılmasının ve istisnanın hala atılabileceği bir yarış koşulu oluşturmanın mümkün olduğunu unutmayın.

Başka bir seçenek de, bir new_key verildiğinde dict içinde yeni girişler oluşturmak için default_factory olan bir dict alt sınıfını, collections.defaultdict kullanmaktır.

Bir Sözlük Üzerinde Yineleme #

Yineleyici olarak bir sözlük kullanırsanız (örneğin, bir “for” ifadesinde), sözlüğün anahtarlarını geçer. Örneğin:

d = {'a': 1, 'b': 2, 'c':3}
for key in d:
    print(key, d[key])
# c 3
# b 2
# a 1

Aynı şey bir anlamada kullanıldığında da geçerlidir.

print([key for key in d])
# ['c', 'b', 'a']

items() yöntemi, hem anahtar hem de değer üzerinde aynı anda döngü yapmak için kullanılabilir:

for key, value in d.items():
    print(key, value)
# c 3
# b 2
# a 1

“values()” yöntemi, beklendiği gibi, yalnızca değerler üzerinde yineleme yapmak için kullanılabilir:

for key, value in d.values():
    print(key, value)
    # 3
    # 2
    # 1
son sürüm

Burada, keys(), values() ve items() yöntemleri geri döner ve yineleyicileri döndürmek için iterkeys() itervalues() ve iteritems() üç ekstra yöntemi vardır. .

son sürüm

Varsayılan değerlere sahip sözlük #

Standart kitaplıkta defaultdict olarak bulunur

from collections import defaultdict

d = defaultdict(int)
d['key']                         # 0
d['key'] = 5
d['key']                         # 5

d = defaultdict(lambda: 'empty')
d['key']                         # 'empty'
d['key'] = 'full'
d['key']                         # 'full'

[*] Alternatif olarak, yerleşik ‘dict’ sınıfını kullanmanız gerekiyorsa, ‘dict.setdefault()’ kullanmak, daha önce var olmayan bir anahtara her eriştiğinizde bir varsayılan oluşturmanıza olanak tanır:

>>> d = {}
{}
>>> d.setdefault('Another_key', []).append("This worked!")
>>> d
{'Another_key': ['This worked!']}

Eklenecek çok sayıda değeriniz varsa, “dict.setdefault()” her çağrıldığında başlangıç ​​değerinin yeni bir örneğini (bu örnekte bir “[]”) oluşturacağını ve bunun gereksiz iş yükleri yaratabileceğini unutmayın.

[*] Python Yemek Kitabı, 3. baskı, David Beazley ve Brian K. Jones (O’Reilly). Telif hakkı 2013 David Beazley ve Brian Jones, 978-1-449-34037-7.

Sözlükleri birleştirme #

Aşağıdaki sözlükleri göz önünde bulundurun:

>>> fish = {'name': "Nemo", 'hands': "fins", 'special': "gills"}
>>> dog = {'name': "Clifford", 'hands': "paws", 'color': "red"}

Python 3.5+ #

>>> fishdog = {**fish, **dog}
>>> fishdog
{'hands': 'paws', 'color': 'red', 'name': 'Clifford', 'special': 'gills'}

Bu örneğin gösterdiği gibi, yinelenen anahtarlar en son değerlerine eşlenir (örneğin, “Clifford”, “Nemo"yu geçersiz kılar).

Python 3.3+ #

>>> from collections import ChainMap
>>> dict(ChainMap(fish, dog))
{'hands': 'fins', 'color': 'red', 'special': 'gills', 'name': 'Nemo'}

Bu teknikle, belirli bir anahtar için en önemli değer, sondan ziyade önceliklidir (“Clifford”, “Nemo” lehine atılır).

Python 2.x, 3.x #

>>> from itertools import chain
>>> dict(chain(fish.items(), dog.items()))
{'hands': 'paws', 'color': 'red', 'name': 'Clifford', 'special': 'gills'}

Bu, birleştirme için “**” tabanlı teknikte olduğu gibi en son değeri kullanır (“Clifford”, “Nemo"yu geçersiz kılar).

>>> fish.update(dog)
>>> fish
{'color': 'red', 'hands': 'paws', 'name': 'Clifford', 'special': 'gills'}

‘dict.update’ öncekinin üzerine yazmak için ikinci dict’i kullanır.

Bir sözlüğün değerlerine erişme #

dictionary = {"Hello": 1234, "World": 5678}
print(dictionary["Hello"])

Yukarıdaki kod “1234” yazdıracaktır.

Bu örnekte "Merhaba" dizgisine key adı verilir. Anahtarı köşeli parantez içine alarak ‘dict’ içinde bir değer aramak için kullanılır.

“dict” tanımında ilgili iki nokta üst üste işaretinden sonra “1234” sayısı görülür. Buna, bu “dikte"de “Merhaba"nın eşleştiği değer denir.

Var olmayan bir anahtarla böyle bir değeri aramak, yakalanmadığı takdirde yürütmeyi durduran bir ‘KeyError’ istisnasına neden olur. KeyError riskini almadan bir değere erişmek istiyorsak, dictionary.get yöntemini kullanabiliriz. Varsayılan olarak, anahtar yoksa, yöntem ‘Yok’ değerini döndürür. Başarısız bir arama durumunda ‘Yok’ yerine döndürülecek ikinci bir değer iletebiliriz.

w = dictionary.get("whatever")
x = dictionary.get("whatever", "nuh-uh")

Bu örnekte ‘w’, ‘Yok’ değerini alacak ve ‘x’, ‘“nuh-uh”’ değerini alacak.

Anahtarlara ve değerlere erişme #

Sözlüklerle çalışırken, sözlükteki tüm anahtarlara ve değerlere, bir “for” döngüsünde, bir liste kavrayışında veya yalnızca düz bir liste olarak erişmek genellikle gereklidir.

Aşağıdaki gibi bir sözlük verildi:

mydict = {
    'a': '1',
    'b': '2'
}

“keys()” yöntemini kullanarak bir anahtar listesi alabilirsiniz:

print(mydict.keys())
# Python2: ['a', 'b']
# Python3: dict_keys(['b', 'a'])

Bunun yerine bir değerler listesi istiyorsanız, values() yöntemini kullanın:

print(mydict.values())
# Python2: ['1', '2']
# Python3: dict_values(['2', '1'])

Hem anahtarla hem de ona karşılık gelen değerle çalışmak istiyorsanız, items() yöntemini kullanabilirsiniz:

print(mydict.items())
# Python2: [('a', '1'), ('b', '2')]
# Python3: dict_items([('b', '2'), ('a', '1')])

NOT: Bir “dict” sıralanmamış olduğundan, “keys()”, “values()” ve “items()” öğelerinin sıralama düzeni yoktur. Bu yöntemlerin döndürdüğü sırayı önemsiyorsanız, sort(), sorted() veya bir OrderedDict kullanın.

Python 2/3 Farkı: Python 3’te, bu yöntemler listeler değil, özel yinelenebilir nesneler döndürür ve Python 2 ‘iterkeys()’, ‘itervalues()’ ve ‘iteritems()’ ile eşdeğerdir. yöntemler. Bu nesneler, bazı farklılıklar olsa da, çoğunlukla listeler gibi kullanılabilir. Daha fazla ayrıntı için PEP 3106 adresine bakın.

Sıralı bir sözlük oluşturma #

Sözlükteki tuşlar üzerinde yineleme yaparken belirli bir sırayı takip edecek sıralı bir sözlük oluşturabilirsiniz.

Şuradan ‘OrderedDict’ kullanın “koleksiyonlar” modülü. Bu, yinelendiğinde her zaman sözlük öğelerini orijinal ekleme sırasına döndürür.

from collections import OrderedDict

d = OrderedDict()
d['first'] = 1
d['second'] = 2
d['third'] = 3
d['last'] = 4

# Outputs "first 1", "second 2", "third 3", "last 4"
for key in d:
    print(key, d[key])

** operatörünü kullanarak sözlükleri açma #

Bir sözlükteki anahtar/değer çiftlerini bir işlevin bağımsız değişkenlerine teslim etmek için `**’ anahtar sözcük bağımsız değişkenini açma operatörünü kullanabilirsiniz. Resmi belgelerden basitleştirilmiş bir örnek:

>>>
>>> def parrot(voltage, state, action):
...     print("This parrot wouldn't", action, end=' ')
...     print("if you put", voltage, "volts through it.", end=' ')
...     print("E's", state, "!")
...
>>> d = {"voltage": "four million", "state": "bleedin' demised", "action": "VOOM"}
>>> parrot(**d)

This parrot wouldn't VOOM if you put four million volts through it. E's bleedin' demised !

Python 3.5’ten itibaren, bu sözdizimini rastgele sayıda “dict” nesnesini birleştirmek için de kullanabilirsiniz.

>>> fish = {'name': "Nemo", 'hands': "fins", 'special': "gills"}
>>> dog = {'name': "Clifford", 'hands': "paws", 'color': "red"}
>>> fishdog = {**fish, **dog}
>>> fishdog

{'hands': 'paws', 'color': 'red', 'name': 'Clifford', 'special': 'gills'}

Bu örneğin gösterdiği gibi, yinelenen anahtarlar en son değerlerine eşlenir (örneğin, “Clifford”, “Nemo"yu geçersiz kılar).

Sözlük oluşturma #

Sözlük oluşturma kuralları:

  • Her anahtar benzersiz olmalıdır (aksi takdirde geçersiz kılınır)
  • Her anahtar hashable olmalıdır (karma için ‘hash’ işlevini kullanabilir; aksi takdirde ‘TypeError’ atılır)
  • Anahtarlar için belirli bir sıra yoktur.
# Creating and populating it with values
stock = {'eggs': 5, 'milk': 2}

# Or creating an empty dictionary
dictionary = {}

# And populating it after
dictionary['eggs'] = 5
dictionary['milk'] = 2

# Values can also be lists
mydict = {'a': [1, 2, 3], 'b': ['one', 'two', 'three']}

# Use list.append() method to add new elements to the values list
mydict['a'].append(4)   # => {'a': [1, 2, 3, 4], 'b': ['one', 'two', 'three']}
mydict['b'].append('four')  # => {'a': [1, 2, 3, 4], 'b': ['one', 'two', 'three', 'four']}

# We can also create a dictionary using a list of two-items tuples
iterable = [('eggs', 5), ('milk', 2)]
dictionary = dict(iterables)

# Or using keyword argument:
dictionary = dict(eggs=5, milk=2)

# Another way will be to use the dict.fromkeys:
dictionary = dict.fromkeys((milk, eggs))  # => {'milk': None, 'eggs': None}
dictionary = dict.fromkeys((milk, eggs), (2, 5))  # => {'milk': 2, 'eggs': 5}

dict() yapıcısı #

“dict()” yapıcısı, anahtar sözcük bağımsız değişkenlerinden veya tek bir yinelenebilir anahtar/değer çiftlerinden veya tek bir sözlük ve anahtar sözcük bağımsız değişkenlerinden sözlükler oluşturmak için kullanılabilir.

dict(a=1, b=2, c=3)                   # {'a': 1, 'b': 2, 'c': 3}
dict([('d', 4), ('e', 5), ('f', 6)])  # {'d': 4, 'e': 5, 'f': 6}
dict([('a', 1)], b=2, c=3)            # {'a': 1, 'b': 2, 'c': 3}
dict({'a' : 1, 'b' : 2}, c=3)         # {'a': 1, 'b': 2, 'c': 3}

Sondaki virgül #

Listeler ve demetler gibi, sözlüğünüze sonunda bir virgül ekleyebilirsiniz.

role = {"By day": "A typical programmer",
        "By night": "Still a typical programmer", }

PEP 8, sondaki virgül ile kapanış ayracı arasında bir boşluk bırakmanız gerektiğini belirtir.

Sözlük değerlerinin tüm kombinasyonları #

options = {
    "x": ["a", "b"],
    "y": [10, 20, 30]
}

Yukarıda gösterilen gibi bir sözlük verildiğinde, burada bir liste var karşılık gelen anahtar için keşfedilecek bir dizi değeri temsil eder. Sanmak "x"="a" "y"=10 ile, ardından "x"="a" ile"y"=10 ile keşfetmek istiyorsunuz ve böylece tüm olası kombinasyonları keşfedene kadar devam edin.

Aşağıdaki kodu kullanarak bu tür tüm değer kombinasyonlarını döndüren bir liste oluşturabilirsiniz.

import itertools

options = {
    "x": ["a", "b"],
    "y": [10, 20, 30]}

keys = options.keys()
values = (options[key] for key in keys)
combinations = [dict(zip(keys, combination)) for combination in itertools.product(*values)]
print combinations

Bu bize “kombinasyonlar” değişkeninde saklanan aşağıdaki listeyi verir:

[{'x': 'a', 'y': 10},
 {'x': 'b', 'y': 10},
 {'x': 'a', 'y': 20},
 {'x': 'b', 'y': 20},
 {'x': 'a', 'y': 30},
 {'x': 'b', 'y': 30}]
