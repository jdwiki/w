Sözdizimi #

  • zip dosyasını içe aktar
  • class zipfile.ZipFile(file, mode=‘r’, sıkıştırma=ZIP_STORED, allowZip64=True)

ZIP dosyası olmayan bir dosyayı açmaya çalışırsanız, zipfile.BadZipFile istisnası ortaya çıkar.

Python 2.7’de bu, ‘zipfile.BadZipfile’ olarak yazılmıştır ve bu eski ad Python 3.2+ sürümünde yenisiyle birlikte korunur.

Zip Dosyalarını Açma #

Başlamak için ‘zipfile’ modülünü içe aktarın ve dosya adını ayarlayın.

import zipfile
filename = 'zipfile.zip'

Zip arşivleriyle çalışmak dosyalarla çalışmaya çok benzer, zip dosyasını açarak nesneyi yaratırsınız, bu da dosyayı tekrar kapatmadan önce üzerinde çalışmanıza izin verir.

zip = zipfile.ZipFile(filename)
print(zip)
# <zipfile.ZipFile object at 0x0000000002E51A90>
zip.close()

Python 2.7’de ve 3.2’den daha yüksek Python 3 sürümlerinde ‘with’ bağlam yöneticisini kullanabiliriz. Dosyayı “oku” modunda açıyoruz ve ardından dosya adlarının bir listesini yazdırıyoruz:

with zipfile.ZipFile(filename, 'r') as z:
    print(zip)
    # <zipfile.ZipFile object at 0x0000000002E51A90>

Zip Dosyası İçeriğinin İncelenmesi #

Bir zip dosyasının içeriğini incelemenin birkaç yolu vardır. Sadece ‘stdout’a gönderilen çeşitli bilgileri almak için ‘printdir’i kullanabilirsiniz.

with zipfile.ZipFile(filename) as zip:
    zip.printdir()

    # Out:
    # File Name                                             Modified             Size
    # pyexpat.pyd                                    2016-06-25 22:13:34       157336
    # python.exe                                     2016-06-25 22:13:34        39576
    # python3.dll                                    2016-06-25 22:13:34        51864
    # python35.dll                                   2016-06-25 22:13:34      3127960
    # etc.

Ayrıca ’namelist’ yöntemiyle dosya adlarının bir listesini de alabiliriz. Burada sadece listeyi yazdırıyoruz:

with zipfile.ZipFile(filename) as zip:
    print(zip.namelist())

# Out: ['pyexpat.pyd', 'python.exe', 'python3.dll', 'python35.dll', ... etc. ...]

“namelist” yerine, her dosya hakkında örneğin zaman damgası ve dosya boyutu gibi ek bilgiler içeren “ZipInfo” nesnelerinin bir listesini döndüren “infolist” yöntemini çağırabiliriz:

with zipfile.ZipFile(filename) as zip:
    info = zip.infolist()
    print(zip[0].filename)
    print(zip[0].date_time)
    print(info[0].file_size)

# Out: pyexpat.pyd
# Out: (2016, 6, 25, 22, 13, 34)
# Out: 157336

Zip dosyası içeriğini bir dizine çıkarma #

Bir zip dosyasının tüm dosya içeriğini çıkarın

import zipfile
with zipfile.ZipFile('zipfile.zip','r') as zfile:
    zfile.extractall('path')

Tek dosyaları ayıklamak istiyorsanız, çıkarma yöntemini kullanın, giriş parametresi olarak isim listesi ve yolu alır.

import zipfile
f=open('zipfile.zip','rb')
zfile=zipfile.ZipFile(f)
for cont in zfile.namelist():
    zfile.extract(cont,path)

Yeni arşivler oluşturma #

Yeni arşiv oluşturmak için zip dosyasını yazma moduyla açın.

import zipfile
new_arch=zipfile.ZipFile("filename.zip",mode="w")

Bu arşive dosya eklemek için write() yöntemini kullanın.

new_arch.write('filename.txt','filename_in_archive.txt') #first parameter is filename and second parameter is filename in archive by default filename will taken if not provided
new_arch.close()

Arşive bayt dizisi yazmak istiyorsanız writestr() yöntemini kullanabilirsiniz.

str_bytes="string buffer"
new_arch.writestr('filename_string_in_archive.txt',str_bytes)
new_arch.close()
