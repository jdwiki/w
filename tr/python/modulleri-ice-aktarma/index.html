Sözdizimi #

  • modül_adı içe aktar
  • module_name.submodule_name dosyasını içe aktarın
  • module_name‘den içe aktarma *
  • module_name‘den import submodule_name [, class_name, function_name, …etc]
  • module_name‘den some_name‘yi new_name olarak içe aktarın
  • module_name.submodule_name‘den sınıf_adı [, işlev_adı, …vb] içe aktarın

Bir modülü içe aktarmak, Python’un bu modüldeki tüm üst düzey kodu değerlendirmesini sağlar, böylece modülün içerdiği tüm işlevleri, sınıfları ve değişkenleri _öğrenir. Bir modülünüzün başka bir yere aktarılmasını istediğinizde, üst düzey kodunuza dikkat edin ve modül geldiğinde yürütülmesini istemiyorsanız onu if __name__ == '__main__': içine alın. ithal.

all özel değişkeni #

Modüller, ‘from mymodule import *’ kullanılırken hangi değişkenlerin içe aktarılacağını kısıtlamak için ‘all’ adlı özel bir değişkene sahip olabilir.

Aşağıdaki modül verildiğinde:

# mymodule.py

__all__ = ['imported_by_star']

imported_by_star = 42
not_imported_by_star = 21

“modülümden içe aktarma *” kullanılırken yalnızca “imported_by_star” içe aktarılır:

>>> from mymodule import *
>>> imported_by_star
42
>>> not_imported_by_star
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
NameError: name 'not_imported_by_star' is not defined

Ancak, “not_imported_by_star” açıkça içe aktarılabilir:

>>> from mymodule import not_imported_by_star
>>> not_imported_by_star
21

Bir modülü içe aktarma #

“import” ifadesini kullanın:

>>> import random
>>> print(random.randint(1, 10))
4

“Modülü içe aktar”, bir modülü içe aktarır ve ardından “module.name” sözdizimini kullanarak, örneğin değerler, işlevler ve sınıflar gibi nesnelerine başvurmanıza izin verir. Yukarıdaki örnekte, “randint” işlevini içeren “rastgele” modül içe aktarılır. Böylece ‘random’u içe aktararak ‘random.randint’ ile ‘randint’i çağırabilirsiniz.

Bir modülü içe aktarabilir ve onu farklı bir adla atayabilirsiniz:

>>> import random as rn
>>> print(rn.randint(1, 10))
4

Python dosyanız “main.py”, “custom.py” ile aynı klasördeyse. Bunu şu şekilde içe aktarabilirsiniz:

import custom

Bir modülden bir işlevi içe aktarmak da mümkündür:

>>> from math import sin
>>> sin(1)
0.8414709848078965

Belirli işlevleri bir modülün derinliklerine aktarmak için, nokta operatörü yalnızca ‘import’ anahtar sözcüğünün sol tarafında kullanılabilir:

from urllib.request import urlopen

Python’da üst seviyeden fonksiyon çağırmanın iki yolu vardır. Biri “ithalat” ve diğeri “dan”. İsim çakışması ihtimalimiz olduğunda ‘import’ kullanmalıyız. Diyelim ki ‘hello.py’ dosyamız ve ‘function’ adında aynı işleve sahip ‘world.py’ dosyalarımız var. Sonra “import” ifadesi işe yarayacaktır.

from hello import function
from world import function

function() #world's function will be invoked. Not hello's

Genel olarak “import” size bir ad alanı sağlayacaktır.

import hello
import world

hello.function() # exclusively hello's function will be invoked 
world.function() # exclusively world's function will be invoked

Ancak, yeterince eminseniz, tüm projenizde aynı fonksiyon adına sahip olmanın hiçbir yolu yoktur, `from’ ifadesini kullanmalısınız.

Aynı satırda birden fazla içe aktarma yapılabilir:

>>> # Multiple modules
>>> import time, sockets, random
>>> # Multiple functions
>>> from math import sin, cos, tan
>>> # Multiple constants
>>> from math import pi, e

>>> print(pi)
3.141592653589793
>>> print(cos(45))
0.5253219888177297
>>> print(time.time())
1482807222.7240417

Yukarıda gösterilen anahtar kelimeler ve sözdizimi kombinasyonlarda da kullanılabilir:

>>> from urllib.request import urlopen as geturl, pathname2url as path2url, getproxies
>>> from math import factorial as fact, gamma, atan as arctan
>>> import random.randint, time, sys

>>> print(time.time())
1482807222.7240417
>>> print(arctan(60))
1.554131203080956
>>> filepath = "/dogs/jumping poodle (december).png"
>>> print(path2url(filepath))
/dogs/jumping%20poodle%20%28december%29.png

İsteğe bağlı bir dosya sistemi konumundan modülleri içe aktarın #

Python Standard Library‘de yerleşik modül olarak veya yan paket olarak halihazırda mevcut olmayan bir modülü içe aktarmak istiyorsanız, bunu modülünüzün bulunduğu dizine yolu ekleyerek yapabilirsiniz. sys.path konumunda bulunur. Bu, bir ana bilgisayarda birden çok python ortamının bulunduğu durumlarda yararlı olabilir.

import sys
sys.path.append("/path/to/directory/containing/your/module")
import mymodule

Modülün yolunu değil, “mymodule"ün bulunduğu dizine giden yolu eklemeniz önemlidir.

Bir modülden tüm adları içe aktarma #

from module_name import *

örneğin:

from math import *
sqrt(2)    # instead of math.sqrt(2)
ceil(2.7)  # instead of math.ceil(2.7)

Bu, “matematik” modülünde tanımlanan tüm adları, alt çizgi ile başlayan adlar dışında (yazıcının yalnızca dahili kullanım için olduğunu hissettiğini gösterir) genel ad alanına içe aktaracaktır.

Uyarı: Aynı ada sahip bir işlev önceden tanımlanmış veya içe aktarılmışsa, üzerine yazılır. Neredeyse her zaman yalnızca belirli adları “matematik içe aktarma sqrt, ceil"den içe aktarmak önerilen yoldur:

def sqrt(num):
    print("I don't know what's the square root of {}.".format(num))

sqrt(4)
# Output: I don't know what's the square root of 4.

from math import * 
sqrt(4)
# Output: 2.0

Yıldızlı içe aktarmalara yalnızca modül düzeyinde izin verilir. Bunları sınıf veya işlev tanımlarında gerçekleştirme girişimleri ‘SyntaxError’ ile sonuçlanır.

def f():
    from math import *

ve

class A:
    from math import *

her ikisi de başarısız olur:

SyntaxError: import * only allowed at module level

Programlı içe aktarma #

Bir işlev çağrısı yoluyla bir modülü içe aktarmak için, “importlib” modülünü kullanın (sürüm 2.7’den itibaren Python’da bulunur):

import importlib
random = importlib.import_module("random")

importlib.import_module() işlevi ayrıca bir paketin alt modülünü de doğrudan içe aktaracaktır:

collections_abc = importlib.import_module("collections.abc")
son sürüm

Python’un eski sürümleri için “imp” modülünü kullanın.

Programlı bir içe aktarma gerçekleştirmek için “imp.find_module” ve “imp.load_module” işlevlerini kullanın.

standart kitaplık belgelerinden alınmıştır

import imp, sys
def import_module(name):
    fp, pathname, description = imp.find_module(name)
    try:
        return imp.load_module(name, fp, pathname, description)
    finally:
        if fp:
            fp.close()
son sürüm

Modülleri programlı olarak içe aktarmak için __import__() ** KULLANMAYIN**! “sys.modules”, “fromlist” argümanı vb. ile ilgili, hangi “importlib.import_module()“nin sizin için işlediğini gözden kaçırmak kolay olan ince ayrıntılar vardır.

İthalat için PEP8 kuralları #

İçe aktarma için önerilen bazı PEP8 stil yönergeleri:

  1. İthalatlar ayrı satırlarda olmalıdır:

     from math import sqrt, ceil      # Not recommended
 from math import sqrt            # Recommended
 from math import ceil

  2. Modülün üst kısmındaki ithalatları aşağıdaki gibi sipariş edin:

    • Standard library imports
    • Related third party imports
    • Local application/library specific imports

  3. Geçerli ad alanındaki adlarda karışıklığa yol açacağı için joker karakter içe aktarmalarından kaçınılmalıdır. Modül içe aktarmadan * yaparsanız, kodunuzdaki belirli bir adın “modül"den gelip gelmediği belirsiz olabilir. Birden çok from module import *-type deyiminiz varsa, bu iki kat doğrudur.

  4. Göreceli ithalat kullanmaktan kaçının; bunun yerine açık ithalat kullanın.

Bir modülden belirli adları içe aktarma #

Modülün tamamını içe aktarmak yerine yalnızca belirtilen adları içe aktarabilirsiniz:

from random import randint # Syntax "from MODULENAME import NAME1[, NAME2[, ...]]"
print(randint(1, 10))      # Out: 5

‘dan rastgele’ gereklidir, çünkü python yorumlayıcısının bir işlevi veya sınıfı hangi kaynaktan içe aktaracağını bilmesi gerekir ve ‘import randint’ işlevin veya sınıfın kendisini belirtir.

Aşağıdaki başka bir örnek (yukarıdakine benzer):

from math import pi
print(pi)                  # Out: 3.14159265359

Aşağıdaki örnek, bir modülü içe aktarmadığımız için bir hataya neden olacaktır:

random.randrange(1, 10)    # works only if "import random" has been run before

Çıktılar:

NameError: name 'random' is not defined

Python yorumlayıcısı ‘rastgele’ ile ne demek istediğinizi anlamıyor. Örneğe ‘import rasgele’ eklenerek bildirilmesi gerekiyor:

import random
random.randrange(1, 10)

Alt modülleri içe aktarma #

from module.submodule import function

Bu, “fonksiyonu” “module.submodule"den içe aktarır.

import() işlevi #

__import__() işlevi, adın yalnızca çalışma zamanında bilindiği modülleri içe aktarmak için kullanılabilir.

if user_input == "os":
    os = __import__("os")

# equivalent to import os

Bu işlev ayrıca bir modülün dosya yolunu belirtmek için de kullanılabilir.

mod = __import__(r"C:/path/to/file/anywhere/on/computer/module.py")

Bir modülü yeniden içe aktarma #

Etkileşimli yorumlayıcıyı kullanırken bir modülü yeniden yüklemek isteyebilirsiniz. Bu, bir modülü düzenliyorsanız ve en yeni sürümü içe aktarmak istiyorsanız veya mevcut bir modülün bir öğesine maymun yama uyguladıysanız ve değişikliklerinizi geri almak istiyorsanız yararlı olabilir.

Geri almak için modülü tekrar alamayacağınızı unutmayın:

import math
math.pi = 3
print(math.pi)    # 3
import math
print(math.pi)    # 3

Bunun nedeni, yorumlayıcının içe aktardığınız her modülü kaydetmesidir. Ve bir modülü yeniden içe aktarmaya çalıştığınızda, yorumlayıcı onu kayıtta görür ve hiçbir şey yapmaz. Bu nedenle, yeniden içe aktarmanın zor yolu, ilgili öğeyi kayıttan çıkardıktan sonra “içe aktarma"yı kullanmaktır:

print(math.pi)    # 3
import sys
if 'math' in sys.modules:  # Is the ``math`` module in the register?
    del sys.modules['math']  # If so, remove it.
import math
print(math.pi)    # 3.141592653589793

Ancak daha basit ve basit bir yol var.

Python 2 #

“Yeniden yükle” işlevini kullanın:

import math
math.pi = 3
print(math.pi)    # 3
reload(math)
print(math.pi)    # 3.141592653589793
son sürüm

Python 3 #

“yeniden yükle” işlevi “importlib"e taşındı:

import math
math.pi = 3
print(math.pi)    # 3
from importlib import reload
reload(math)
print(math.pi)    # 3.141592653589793
son sürüm
