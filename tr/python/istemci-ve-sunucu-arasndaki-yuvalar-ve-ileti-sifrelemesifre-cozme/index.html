Kriptografi güvenlik amacıyla kullanılır. Python’da IDEA şifreleme MODE TO’sunu kullanan çok fazla Şifreleme/Şifre Çözme örneği yoktur. Bu belgenin amacı :

İstasyondan istasyona iletişimde RSA Dijital İmza şemasını genişletin ve uygulayın. Mesajın bütünlüğü için Hashing’i kullanma, yani SHA-1. Basit Anahtar Aktarımı protokolü üretin. Anahtarı IDEA şifrelemesi ile şifreleyin. Blok Şifre Modu Sayaç Modudur

Kullanılan Dil: Python 2.7 (İndirme Bağlantısı: https://www.python.org/downloads/ )

Kullanılan Kitaplık:

*PyCrypto (İndirme Bağlantısı: https://pypi.python.org/pypi/pycrypto )

*PyCryptoPlus (İndirme Bağlantısı: https://github.com/doegox/python-cryptoplus )

Kütüphane Kurulumu:

PyCrypto: Dosyayı açın. Dizine gidin ve linux(alt+ctrl+t) için terminali açın ve Windows için CMD(shift+sağ tıklama+burada aç komut istemini seç). Bundan sonra python setup.py install yazın (Python Ortamının Windows işletim sisteminde doğru şekilde ayarlandığından emin olun)

PyCryptoPlus: Son kitaplıkla aynı.

Görevlerin Uygulanması: Görev iki bölüme ayrılmıştır. Biri el sıkışma süreci diğeri ise iletişim sürecidir. Soket Kurulumu:

  • Genel ve özel anahtarların oluşturulması ve ortak anahtarın hash edilmesi olarak ihtiyacımız var. Şimdi soketi kurmak için. Soketi kurmak için, “import socket” ile başka bir modülü içe aktarıp (client için) veya kullanıcıdan gelen soketin IP adresini ve portu (sunucu için) bağlamamız (sunucu için) gerekir.

    ———-Client Side———-

      server = socket.socket(socket.AF_INET,socket.SOCK_STREAM)

  except BaseException: print "-----Check Server Address or Port-----"

“ socket.AF_INET,socket.SOCK_STREAM” kabul() işlevini ve mesajlaşma temellerini kullanmamıza izin verecektir. Bunun yerine “ socket.AF_INET,socket.SOCK_DGRAM” da kullanabiliriz ama o zaman setblocking(value) kullanmamız gerekecek.

El Sıkışma Süreci:

  • (MÜŞTERİ)İlk görev, genel ve özel anahtar oluşturmaktır. Özel oluşturmak için ve ortak anahtar, bazı modülleri içe aktarmamız gerekiyor. Bunlar: Crypto’dan Random’u içe aktar ve Crypto.PublicKey’den RSA’yı içe aktar. Anahtarları oluşturmak için birkaç basit kod satırı yazmamız gerekiyor:

    random_generator = Random.new().read key = RSA.generate(1024,random_generator) public = key.publickey().exportKey()

random_generator, “Kripto alma Random” modülünden türetilmiştir. Anahtar, rastgele karakterler üreterek 1024 boyutunda özel bir anahtar oluşturacak olan “Crypto.PublicKey import RSA”dan türetilmiştir. Genel, önceden oluşturulmuş özel anahtardan genel anahtarı dışa aktarıyor.

  • (İSTEMCİ) Genel ve özel anahtarı oluşturduktan sonra, SHA-1 hash kullanarak sunucuya göndermek için ortak anahtarı hash etmemiz gerekiyor. SHA-1 hash’ini kullanmak için “import hashlib” yazarak başka bir modülü import etmemiz gerekiyor. Public key’i hash etmek için iki satır kod yazdık:

      hash_object = hashlib.sha1(public) 
  hex_digest = hash_object.hexdigest()

Burada hash_object ve hex_digest değişkenimizdir. Bundan sonra, istemci sunucuya hex_digest ve public gönderecek ve Sunucu, istemciden alınan hash ile ortak anahtarın yeni hash değerini karşılaştırarak bunları doğrulayacaktır. Yeni hash ve istemciden gelen hash eşleşirse, bir sonraki prosedüre geçecektir. İstemciden gönderilen public string şeklinde olduğu için sunucuda key olarak kullanılamayacaktır. yan. Bunu önlemek ve string public key’i rsa public key’e çevirmek için server_public_key = RSA.importKey(getpbk) yazmamız gerekiyor, burada getpbk clienttan gelen public key.

  • (SUNUCU) Bir sonraki adım bir oturum anahtarı oluşturmaktır. Burada “os” modülünü kullanarak rastgele bir “key = os.urandom(16)” anahtarı oluşturdum ve bu bize 16bit uzunluğunda bir anahtar verecek ve daha sonra o anahtarı “AES.MODE_CTR” de şifreleyip tekrar hash ettim. SHA-1 ile:

      #encrypt CTR MODE session key
  en = AES.new(key_128,AES.MODE_CTR,counter = lambda:key_128) encrypto = en.encrypt(key_128)
  #hashing sha1
  en_object = hashlib.sha1(encrypto)
  en_digest = en_object.hexdigest()

Yani en_digest bizim oturum anahtarımız olacak.

  • (SUNUCU) El sıkışma işleminin son kısmı, istemciden alınan açık anahtarı ve oluşturulan oturum anahtarını şifrelemektir. sunucu tarafı.

      #encrypting session key and public key
  E = server_public_key.encrypt(encrypto,16)

Şifrelemeden sonra sunucu anahtarı istemciye string olarak gönderir.

  • (MÜŞTERİ) Sunucudan şifrelenmiş (genel ve oturum anahtarı) dizisini aldıktan sonra, istemci Özel Anahtar kullanarak bunların şifresini çözecektir. bu, daha önce ortak anahtarla birlikte oluşturuldu. şifreli olarak (genel ve oturum anahtarı) dize biçimindeydi, şimdi onu almalıyız eval() kullanarak bir anahtar olarak geri dönün. Şifre çözme işlemi yapılırsa, her iki taraf da onayladığını teyit ettiği için el sıkışma süreci de tamamlanmıştır. aynı tuşları kullanıyor. Şifresini çözmek için:

      en = eval(msg)
  decrypt = key.decrypt(en)
  # hashing sha1
  en_object = hashlib.sha1(decrypt) en_digest = en_object.hexdigest()

Çıktıda okunabilmesi için SHA-1’i burada kullandım.

İletişim süreci:

İletişim işlemi için, IDEA şifreleme MODE_CTR için ANAHTAR olarak her iki taraftaki oturum anahtarını kullanmalıyız. Her iki taraf da oturum anahtarını kullanarak mesajları IDEA.MODE_CTR ile şifreleyecek ve şifresini çözecektir.

  • (Şifreleme) IDEA şifrelemesi için çağrılabilir olması gerektiği gibi 16bit boyutunda ve sayıcı anahtarına ihtiyacımız var. MODE_CTR’de sayaç zorunludur. Şifrelediğimiz ve hash ettiğimiz oturum anahtarı artık 40 boyutundadır ve bu, IDEA şifrelemesinin sınır anahtarını aşacaktır. Bu nedenle, oturum anahtarının boyutunu küçültmemiz gerekiyor. Azaltma için, dizge[değer:değer] işlevinde yerleşik normal python kullanabiliriz. Değer, kullanıcının seçimine göre herhangi bir değer olabilir. Bizim durumumuzda, “anahtar[:16]” yaptım anahtardan 0 ile 16 arasında değer alacağı yer. Bu dönüştürme, key[1:17] veya key[16:] gibi birçok yolla yapılabilir. Sonraki kısım, işleme için 3 argüman alacak olan IDEA.new() yazarak yeni IDEA şifreleme fonksiyonu oluşturmaktır. İlk argüman ANAHTAR olacak, ikinci argüman IDEA şifrelemesinin modu olacak (bizim durumumuzda IDEA.MODE_CTR) ve üçüncü argüman mutlaka çağrılabilir bir fonksiyon olan counter= olacaktır. sayaç=, işlev tarafından döndürülecek bir dize boyutu tutacaktır. counter= öğesini tanımlamak için makul bir değer kullanmamız gerekir. Bu durumda lambda tanımlayarak KEY boyutunu kullandım. Lambda kullanmak yerine, counter= için rastgele değer üreten Counter.Util’i kullanabiliriz. Counter.Util’i kullanmak için, kriptodan sayaç modülünü içe aktarmamız gerekiyor. Bu nedenle, kod şöyle olacaktır:

      ideaEncrypt = IDEA.new(key, IDEA.MODE_CTR, counter=lambda : key)

“ideaEncrypt”i IDEA şifreleme değişkenimiz olarak tanımladıktan sonra, herhangi bir mesajı şifrelemek için yerleşik şifreleme işlevini kullanabiliriz.

eMsg = ideaEncrypt.encrypt(whole)
#converting the encrypted message to HEXADECIMAL to readable eMsg =         
eMsg.encode("hex").upper()

Bu kod segmentinde bütün, şifrelenecek mesajdır ve eMsg, şifrelenmiş mesajdır. Mesajı şifreledikten sonra, okunabilir hale getirmek için onu HEXADECIMAL’e dönüştürdüm ve üst() karakterleri büyük harf yapmak için yerleşik işlevdir. Bundan sonra, bu şifreli mesaj, şifrenin çözülmesi için karşı istasyona gönderilecektir.

  • (Şifre Çözme)

Şifrelenmiş mesajların şifresini çözmek için, aynı argümanları ve aynı anahtarı kullanarak başka bir şifreleme değişkeni oluşturmamız gerekecek, ancak bu sefer değişken şifreli mesajların şifresini çözecek. Bunun kodu geçen seferkiyle aynı. Ancak mesajların şifresini çözmeden önce mesajı onaltılıdan deşifre etmemiz gerekiyor çünkü şifreleme bölümümüzde şifrelenmiş mesajı okunabilir hale getirmek için onaltılık olarak kodladık. Dolayısıyla, kodun tamamı şöyle olacaktır:

decoded = newmess.decode("hex")
ideaDecrypt = IDEA.new(key, IDEA.MODE_CTR, counter=lambda: key) 
dMsg = ideaDecrypt.decrypt(decoded)

Bu işlemler, şifreleme ve şifre çözme için hem sunucu hem de istemci tarafında yapılacaktır.

    # creating session key
    key_128 = os.urandom(16)
    #encrypt CTR MODE session key
    en = AES.new(key_128,AES.MODE_CTR,counter = lambda:key_128)
    encrypto = en.encrypt(key_128)
    #hashing sha1
    en_object = hashlib.sha1(encrypto)
    en_digest = en_object.hexdigest()

    print ("\n-----SESSION KEY-----\n"+en_digest)

    #encrypting session key and public key

import time
import socket
import threading
import hashlib
import itertools
import sys
from Crypto import Random
from Crypto.PublicKey import RSA
from CryptoPlus.Cipher import IDEA

#animating loading
done = False
def animate():
    for c in itertools.cycle(['....','.......','..........','............']):
        if done:
            break
        sys.stdout.write('\rCONFIRMING CONNECTION TO SERVER '+c)
        sys.stdout.flush()
        time.sleep(0.1)

#public key and private key
random_generator = Random.new().read
key = RSA.generate(1024,random_generator)
public = key.publickey().exportKey()
private = key.exportKey()

#hashing the public key
hash_object = hashlib.sha1(public)
hex_digest = hash_object.hexdigest()

#Setting up socket
server = socket.socket(socket.AF_INET,socket.SOCK_STREAM)

#host and port input user
host = raw_input("Server Address To Be Connected -> ")
port = int(input("Port of The Server -> "))
#binding the address and port
server.connect((host, port))
# printing "Server Started Message"
thread_load = threading.Thread(target=animate)
thread_load.start()

time.sleep(4)
done = True

def send(t,name,key):
    mess = raw_input(name + " : ")
    key = key[:16]
    #merging the message and the name
    whole = name+" : "+mess
    ideaEncrypt = IDEA.new(key, IDEA.MODE_CTR, counter=lambda : key)
    eMsg = ideaEncrypt.encrypt(whole)
    #converting the encrypted message to HEXADECIMAL to readable
    eMsg = eMsg.encode("hex").upper()
    if eMsg != "":
        print ("ENCRYPTED MESSAGE TO SERVER-> "+eMsg)
    server.send(eMsg)
def recv(t,key):
    newmess = server.recv(1024)
    print ("\nENCRYPTED MESSAGE FROM SERVER-> " + newmess)
    key = key[:16]
    decoded = newmess.decode("hex")
    ideaDecrypt = IDEA.new(key, IDEA.MODE_CTR, counter=lambda: key)
    dMsg = ideaDecrypt.decrypt(decoded)
    print ("\n**New Message From Server**  " + time.ctime(time.time()) + " : " + dMsg + "\n")

while True:
    server.send(public)
    confirm = server.recv(1024)
    if confirm == "YES":
        server.send(hex_digest)

    #connected msg
    msg = server.recv(1024)
    en = eval(msg)
    decrypt = key.decrypt(en)
    # hashing sha1
    en_object = hashlib.sha1(decrypt)
    en_digest = en_object.hexdigest()

    print ("\n-----ENCRYPTED PUBLIC KEY AND SESSION KEY FROM SERVER-----")
    print (msg)
    print ("\n-----DECRYPTED SESSION KEY-----")
    print (en_digest)
    print ("\n-----HANDSHAKE COMPLETE-----\n")
    alais = raw_input("\nYour Name -> ")

    while True:
        thread_send = threading.Thread(target=send,args=("------Sending Message------",alais,en_digest))
        thread_recv = threading.Thread(target=recv,args=("------Recieving Message------",en_digest))
        thread_send.start()
        thread_recv.start()

        thread_send.join()
        thread_recv.join()
        time.sleep(0.5)
    time.sleep(60)
    server.close()
