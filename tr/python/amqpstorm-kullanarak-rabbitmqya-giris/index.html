AMQPStorm‘in en son sürümü pypi adresinde mevcuttur veya pip kullanarak yükleyebilirsiniz.

pip install amqpstorm

RabbitMQ’dan gelen mesajlar nasıl tüketilir #

Kitaplığı içe aktararak başlayın.

from amqpstorm import Connection

Mesajları tüketirken, önce gelen mesajları işlemek için bir fonksiyon tanımlamamız gerekir. Bu herhangi bir çağrılabilir işlev olabilir ve bir mesaj nesnesi veya bir mesaj demeti (start_tüketicide tanımlanan ‘‘’to_tuple’’` parametresine bağlı olarak) alması gerekir.

Gelen mesajdaki verileri işlemenin yanı sıra, mesajı Kabul Etmemiz veya Reddetmemiz de gerekecektir. Bu önemlidir, çünkü RabbitMQ’ya mesajı düzgün bir şekilde aldığımızı ve işlediğimizi bildirmemiz gerekir.

def on_message(message):
    """This function is called on message received.

    :param message: Delivered message.
    :return:
    """
    print("Message:", message.body)

    # Acknowledge that we handled the message without any issues.
    message.ack()

    # Reject the message.
    # message.reject()

    # Reject the message, and put it back in the queue.
    # message.reject(requeue=True)

Ardından RabbitMQ sunucusuna bağlantı kurmamız gerekiyor.

connection = Connection('127.0.0.1', 'guest', 'guest')

Bundan sonra bir kanal kurmamız gerekiyor. Her bağlantının birden fazla kanalı olabilir ve genel olarak çok iş parçacıklı görevleri gerçekleştirirken, iş parçacığı başına bir tane olması önerilir (ancak gerekli değildir).

channel = connection.channel()

Kanalımızı kurduktan sonra RabbitMQ’ya mesajları tüketmeye başlamak istediğimizi bildirmeliyiz. Bu durumda, tüketilen tüm mesajlarımızı işlemek için önceden tanımlı ``on_message``` fonksiyonumuzu kullanacağız.

RabbitMQ sunucusunda dinleyeceğimiz sıra ````simple_queue``` olacak ve ayrıca RabbitMQ’ya, onlarla işimiz bittiğinde tüm gelen mesajları kabul edeceğimizi söylüyoruz.

channel.basic.consume(callback=on_message, queue='simple_queue', no_ack=False)

Son olarak, RabbitMQ sunucusu tarafından iletilen mesajları işlemeye başlamak için IO döngüsünü başlatmamız gerekiyor.

channel.start_consuming(to_tuple=False)

RabbitMQ’da mesajlar nasıl yayınlanır #

Kitaplığı içe aktararak başlayın.

from amqpstorm import Connection
from amqpstorm import Message

Ardından RabbitMQ sunucusuna bir bağlantı açmamız gerekiyor.

connection = Connection('127.0.0.1', 'guest', 'guest')

Bundan sonra bir kanal kurmamız gerekiyor. Her bağlantının birden fazla kanalı olabilir ve genel olarak çok iş parçacıklı görevleri gerçekleştirirken, iş parçacığı başına bir tane olması önerilir (ancak gerekli değildir).

channel = connection.channel()

Kanalımızı kurduktan sonra mesajımızı hazırlamaya başlayabiliriz.

# Message Properties.
properties = {
    'content_type': 'text/plain',
    'headers': {'key': 'value'}
}

# Create the message.
message = Message.create(channel=channel, body='Hello World!', properties=properties)

Artık sadece yayınla```yı çağırarak ve bir ''routing_key' sağlayarak mesajı yayınlayabiliriz. Bu durumda mesajı simple_queue adlı bir kuyruğa göndereceğiz.

message.publish(routing_key='simple_queue')

RabbitMQ’da gecikmeli kuyruk nasıl oluşturulur #

İlk önce, biri ana kuyruk için diğeri gecikme kuyruğu için olmak üzere iki temel kanal kurmamız gerekiyor. Sondaki örneğimde, gerekli olmayan ancak kodu daha güvenilir kılan birkaç ek işaret ekledim; “teslimi onayla”, “teslimat_modu” ve “dayanıklı” gibi. Bunlarla ilgili daha fazla bilgiyi RabbitMQ manual içinde bulabilirsiniz.

Kanalları kurduktan sonra, gecikme kanalından ana kuyruğumuza mesaj göndermek için kullanabileceğimiz ana kanala bir bağlama ekliyoruz.

channel.queue.bind(exchange='amq.direct', routing_key='hello', queue='hello')

Daha sonra gecikme kanalımızı, mesajların süresi dolduktan sonra ana kuyruğa iletecek şekilde yapılandırmamız gerekiyor.

delay_channel.queue.declare(queue='hello_delay', durable=True, arguments={
    'x-message-ttl': 5000,
    'x-dead-letter-exchange': 'amq.direct',
    'x-dead-letter-routing-key': 'hello'
})
  • x-message-ttl (Mesaj - Yaşam Süresi)

Bu normalde eski mesajları otomatik olarak kaldırmak için kullanılır. belirli bir süre sonra sıraya gireriz, ancak iki isteğe bağlı argüman ekleyerek bu davranışı değiştirebilir ve bunun yerine bu parametrenin belirlemesini sağlayabilir milisaniye cinsinden mesajların gecikme kuyruğunda ne kadar kalacağı.

  • x-ölü harf-yönlendirme anahtarı

Bu değişken, mesajı farklı bir kuyruğa aktarmamızı sağlar. süresi dolduğunda, varsayılan kaldırma davranışı yerine tamamen.

  • x-ölü-mektup değişimi

    This variable determines which Exchange used to transfer the message from hello_delay to hello queue.

Gecikme kuyruğunda yayınlama

Tüm temel Pika parametrelerini ayarlamayı bitirdiğimizde, temel yayınlamayı kullanarak gecikme kuyruğuna bir mesaj göndermeniz yeterlidir.

delay_channel.basic.publish(exchange='',
                            routing_key='hello_delay',
                            body='test',
                            properties={'delivery_mod': 2})

Komut dosyasını çalıştırdıktan sonra, RabbitMQ yönetim modülünüzde oluşturulan aşağıdaki kuyrukları görmelisiniz.

Örnek.

from amqpstorm import Connection

connection = Connection('127.0.0.1', 'guest', 'guest')

# Create normal 'Hello World' type channel.
channel = connection.channel()
channel.confirm_deliveries()
channel.queue.declare(queue='hello', durable=True)

# We need to bind this channel to an exchange, that will be used to transfer
# messages from our delay queue.
channel.queue.bind(exchange='amq.direct', routing_key='hello', queue='hello')

# Create our delay channel.
delay_channel = connection.channel()
delay_channel.confirm_deliveries()

# This is where we declare the delay, and routing for our delay channel.
delay_channel.queue.declare(queue='hello_delay', durable=True, arguments={
    'x-message-ttl': 5000, # Delay until the message is transferred in milliseconds.
    'x-dead-letter-exchange': 'amq.direct', # Exchange used to transfer the message from A to B.
    'x-dead-letter-routing-key': 'hello' # Name of the queue we want the message transferred to.
})

delay_channel.basic.publish(exchange='',
                            routing_key='hello_delay',
                            body='test',
                            properties={'delivery_mode': 2})

print("[x] Sent")
