İnsanların görmesi için verileri depolarken ve dönüştürürken, dize biçimlendirme çok önemli hale gelebilir. Python, bu konuda ana hatları verilen çok çeşitli dize biçimlendirme yöntemleri sunar.

Sözdizimi #

  • “{}".format(42) ==> “42”

  • “{0}".format(42) ==> “42”

  • “{0:.2f}".format(42) ==> “42.00”

  • “{0:.0f}".format(42.1234) ==> “42”

  • “{answer}".format(no_answer=41, answer=42) ==> “42”

  • “{answer:.2f}".format(no_answer=41, answer=42) ==> “42.00”

  • “{[anahtar]}".format({‘anahtar’: ‘değer’}) ==> “değer”

  • “{1}".format([‘sıfır’, ‘bir’, ‘iki’]) ==> “bir”

  • “{answer} = {answer}".format(answer=42) ==> “42 = 42”

  • ’ ‘.join([‘yığın’, ’taşma’]) ==> “yığın taşması”

  • Nasıl çalıştığına dair çok kapsamlı ve nazik bir giriş/açıklama için PyFormat.info adresine bakın.

Dize Biçimlendirmenin Temelleri #

foo = 1
bar = 'bar'
baz = 3.14

Çıktıyı biçimlendirmek için str.format’ı kullanabilirsiniz. Parantez çiftleri, argümanların iletildiği sırayla argümanlarla değiştirilir:

print('{}, {} and {}'.format(foo, bar, baz))
# Out: "1, bar and 3.14"

İndeksler parantez içinde de belirtilebilir. Sayılar, “str.format” işlevine (0 tabanlı) iletilen bağımsız değişkenlerin dizinlerine karşılık gelir.

print('{0}, {1}, {2}, and {1}'.format(foo, bar, baz))
# Out: "1, bar, 3.14, and bar"
print('{0}, {1}, {2}, and {3}'.format(foo, bar, baz))
# Out: index out of range error

Adlandırılmış bağımsız değişkenler de kullanılabilir:

print("X value is: {x_val}. Y value is: {y_val}.".format(x_val=2, y_val=3))
# Out: "X value is: 2. Y value is: 3."

str.format’a aktarıldığında nesne niteliklerine başvurulabilir:

class AssignValue(object):
    def __init__(self, value):
        self.value = value
my_value = AssignValue(6)
print('My value is: {0.value}'.format(my_value))  # "0" is optional
# Out: "My value is: 6"

Sözlük tuşları da kullanılabilir:

my_dict = {'key': 6, 'other_key': 7}
print("My other key is: {0[other_key]}".format(my_dict))  # "0" is optional
# Out: "My other key is: 7"

Aynısı liste ve demet indeksleri için de geçerlidir:

my_list = ['zero', 'one', 'two']
print("2nd element is: {0[2]}".format(my_list))  # "0" is optional
# Out: "2nd element is: two"

Not: ‘str.format’a ek olarak, Python aynı zamanda dize biçimlendirme veya interpolasyon operatörü olarak da bilinen modülo operatörü ‘%’ sağlar (bkz. PEP 3101)– dizeleri biçimlendirmek için. “str.format”, “%“nin halefidir ve örneğin birden çok yer değiştirmeyi kolaylaştırarak daha fazla esneklik sunar.

Argüman dizinlerine ek olarak, küme parantezlerinin içine bir format belirtimi de ekleyebilirsiniz. Bu, özel kurallara uyan bir ifadedir ve başında iki nokta üst üste (:) olması gerekir. Biçim belirtiminin tam açıklaması için belgeler’e bakın. Biçim belirtimine bir örnek, hizalama yönergesi :~^20dir (^ orta hizalama, toplam genişlik 20, ~ karakteriyle doldurma anlamına gelir):

'{:~^20}'.format('centered')
# Out: '~~~~~~centered~~~~~~'

“format”, “%” ile mümkün olmayan davranışlara izin verir, örneğin argümanların tekrarı:

t = (12, 45, 22222, 103, 6)
print '{0} {2} {1} {2} {3} {2} {4} {2}'.format(*t)    
# Out: 12 22222 45 22222 103 22222 6 22222

“Format” bir fonksiyon olduğu için diğer fonksiyonlarda argüman olarak kullanılabilir:

number_list = [12,45,78]
print map('the number is {}'.format, number_list)
# Out: ['the number is 12', 'the number is 45', 'the number is 78']   
    

from datetime import datetime,timedelta
    
once_upon_a_time = datetime(2010, 7, 1, 12, 0, 0)
delta = timedelta(days=13, hours=8,  minutes=20)
    
gen = (once_upon_a_time + x * delta for x in xrange(5))
    
print '\n'.join(map('{:%Y-%m-%d %H:%M:%S}'.format, gen))
#Out: 2010-07-01 12:00:00
#     2010-07-14 20:20:00
#     2010-07-28 04:40:00
#     2010-08-10 13:00:00
#     2010-08-23 21:20:00

Hizalama ve dolgu #

Dizenin hizalamasını değiştirmek için “format()” yöntemi kullanılabilir. Bunu, “:[fill_char][align_operator][width]” biçiminde bir biçim ifadesi ile yapmanız gerekir, burada “align_operator” şunlardan biridir:

  • “<”, alanı “genişlik” içinde sola hizalanmaya zorlar.
  • >, alanı genişlik içinde sağa hizalanmaya zorlar.
  • “^”, alanı “genişlik” içinde ortalanmaya zorlar.
  • =, dolgunun işaretten sonra yerleştirilmesini zorlar (yalnızca sayısal türler).

“fill_char” (eğer atlanmış varsayılan boşluk ise) dolgu için kullanılan karakterdir.

'{:~<9s}, World'.format('Hello')
# 'Hello~~~~, World'

'{:~>9s}, World'.format('Hello')
# '~~~~Hello, World'

'{:~^9s}'.format('Hello')
# '~~Hello~~'

'{:0=6d}'.format(-123)
# '-00123'
son sürüm

Not: ’ljust()’, ‘rjust()’, ‘center()’, ‘zfill()’ dize işlevlerini kullanarak aynı sonuçları elde edebilirsiniz, ancak bu işlevler sürüm 2.5’ten beri kullanımdan kaldırılmıştır.

Biçim değişmezleri (f-string) #

Değişmez biçim dizeleri, PEP 498 (Python3.6 ve üzeri)‘de tanıtıldı ve bir dosyanın başına “f"yi eklemenize izin verdi. Geçerli kapsamdaki tüm değişkenlerle birlikte “.format"ı etkin bir şekilde uygulamak için dize değişmezi.

>>> foo = 'bar'
>>> f'Foo is {foo}'
'Foo is bar'

Bu, hizalama ve nokta gösterimi dahil olmak üzere daha gelişmiş biçim dizeleriyle de çalışır.

>>> f'{foo:^7s}'
'  bar  '

Not: “f””, python2’de “bayt” için “b”” veya “unicode” için “u”” gibi belirli bir türü ifade etmez. Biçimlendirme hemen uygulanır ve normal bir karıştırma sağlanır.

Biçim dizeleri ayrıca yuvalanmış:

>>> price = 478.23
>>> f"{f'${price:0.2f}':*>20s}"
'*************$478.23'

Bir f-string’deki ifadeler soldan sağa sırayla değerlendirilir. Bu, yalnızca ifadelerin yan etkileri varsa saptanabilir:

>>> def fn(l, incr):
...    result = l[0]
...    l[0] += incr
...    return result
...
>>> lst = [0]
>>> f'{fn(lst,2)} {fn(lst,3)}'
'0 2'
>>> f'{fn(lst,2)} {fn(lst,3)}'
'5 7'
>>> lst
[10]

Kayan biçimlendirme #

>>> '{0:.0f}'.format(42.12345)
'42'

>>> '{0:.1f}'.format(42.12345)
'42.1'

>>> '{0:.3f}'.format(42.12345)
'42.123'

>>> '{0:.5f}'.format(42.12345)
'42.12345'

>>> '{0:.7f}'.format(42.12345)
'42.1234500'

Başka bir referans yolu için de aynı tutuş:

>>> '{:.3f}'.format(42.12345)
'42.123'

>>> '{answer:.3f}'.format(answer=42.12345)
'42.123'

Kayan nokta sayıları bilimsel gösterim veya yüzde olarak da biçimlendirilebilir:

>>> '{0:.3e}'.format(42.12345)
'4.212e+01'

>>> '{0:.0%}'.format(42.12345)
'4212%'

{0} ve {name} notasyonlarını da birleştirebilirsiniz. Bu, özellikle tüm değişkenleri önceden belirlenmiş sayıda ondalık sayıya 1 bildirimle yuvarlamak istediğinizde kullanışlıdır:

>>> s = 'Hello'
>>> a, b, c = 1.12345, 2.34567, 34.5678
>>> digits = 2

>>> '{0}! {1:.{n}f}, {2:.{n}f}, {3:.{n}f}'.format(s, a, b, c, n=digits)
'Hello! 1.12, 2.35, 34.57'

Datetime ile dize biçimlendirme #

Herhangi bir sınıf, ‘format’ yöntemiyle kendi dize biçimlendirme sözdizimini yapılandırabilir. Standart Python kitaplığında bundan kullanışlı bir şekilde yararlanan bir tür, “strftime” benzeri biçimlendirme kodlarının doğrudan “str.format” içinde kullanılabileceği “datetime” türüdür:

>>> from datetime import datetime
>>> 'North America: {dt:%m/%d/%Y}.  ISO: {dt:%Y-%m-%d}.'.format(dt=datetime.now())
'North America: 07/21/2016.  ISO: 2016-07-21.'

Tarih-saat biçimlendiricilerinin tam listesi resmi belgelerde bulunabilir.

Adlandırılmış yer tutucular #

Biçim dizeleri, “biçim” için anahtar sözcük bağımsız değişkenleri kullanılarak enterpolasyon yapılan adlandırılmış yer tutucular içerebilir.

Sözlük kullanma (Python 2.x) #

>>> data = {'first': 'Hodor', 'last': 'Hodor!'}
>>> '{first} {last}'.format(**data)
'Hodor Hodor!'

Sözlük kullanma (Python 3.2+) #

>>> '{first} {last}'.format_map(data)
'Hodor Hodor!'

str.format_map sözlükleri önce paketlerini açmadan kullanmanıza olanak tanır. Ayrıca, yeni doldurulmuş bir “dict” yerine “data” sınıfı (özel bir tür olabilir) kullanılır.

Sözlük olmadan: #

>>> '{first} {last}'.format(first='Hodor', last='Hodor!')
'Hodor Hodor!'

Getitem ve Getattr kullanarak biçimlendirme #

getitem‘yi destekleyen herhangi bir veri yapısı, iç içe geçmiş yapılarını biçimlendirebilir:

person = {'first': 'Arthur', 'last': 'Dent'} 
'{p[first]} {p[last]}'.format(p=person) 
# 'Arthur Dent'

Nesne niteliklerine getattr() kullanılarak erişilebilir:

class Person(object):
    first = 'Zaphod'
    last = 'Beeblebrox'

'{p.first} {p.last}'.format(p=Person())
# 'Zaphod Beeblebrox'

Sayısal Değerleri Biçimlendirme #

“.format()” yöntemi, bir sayıyı aşağıdaki gibi farklı biçimlerde yorumlayabilir:

>>> '{:c}'.format(65)    # Unicode character
'A'

>>> '{:d}'.format(0x0a)  # base 10
'10'

>>> '{:n}'.format(0x0a)  # base 10 using current locale for separators
'10'

Tam sayıları farklı tabanlara göre biçimlendirin (onaltılı, oct, ikili)

>>> '{0:x}'.format(10) # base 16, lowercase - Hexadecimal
'a'

>>> '{0:X}'.format(10) # base 16, uppercase - Hexadecimal
'A'

>>> '{:o}'.format(10) # base 8 - Octal
'12'

>>> '{:b}'.format(10) # base 2 - Binary
'1010'

>>> '{0:#b}, {0:#o}, {0:#x}'.format(42) # With prefix
'0b101010, 0o52, 0x2a'

>>> '8 bit: {0:08b}; Three bytes: {0:06x}'.format(42) # Add zero padding
'8 bit: 00101010; Three bytes: 00002a'

Bir RGB kayan nokta grubunu renkli bir onaltılı dizeye dönüştürmek için biçimlendirmeyi kullanın:

>>> r, g, b = (1.0, 0.4, 0.0)
>>> '#{:02X}{:02X}{:02X}'.format(int(255 * r), int(255 * g), int(255 * b))
'#FF6600'

Yalnızca tamsayılar dönüştürülebilir:

>>> '{:x}'.format(42.0)
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: Unknown format code 'x' for object of type 'float'

İç içe biçimlendirme #

Bazı biçimler, biçimlendirilmiş dizenin genişliği veya hizalama gibi ek parametreler alabilir:

>>> '{:.>10}'.format('foo')
'.......foo'

Bunlar, “{}” içine daha fazla “{}” yerleştirilerek “biçim” parametresi olarak da sağlanabilir:

>>> '{:.>{}}'.format('foo', 10)
'.......foo'
'{:{}{}{}}'.format('foo', '*', '^', 15)
'******foo******'

İkinci örnekte, '{:{}{}}' biçim dizgisi daha önce '{:*^15}' (yani “orta ve * ile toplam uzunluk 15 olacak şekilde altlık”) olarak değiştirilir. bu şekilde biçimlendirilmek üzere gerçek 'foo' dizgisine uygulamak.

Bu, örneğin tablo verilerini hizalarken, parametrelerin önceden bilinmediği durumlarda yararlı olabilir:

>>> data = ["a", "bbbbbbb", "ccc"]
>>> m = max(map(len, data))
>>> for d in data:
...     print('{:>{}}'.format(d, m))
      a
bbbbbbb
    ccc

Dizeleri doldurma ve kesme, birleştirilmiş #

Değişkenleri 3 karakterlik bir sütunda yazdırmak istediğinizi varsayalım.

Not: { ve } ikiye katlandığında onlardan kaçar.

s = """

pad
{{:3}}             :{a:3}:

truncate
{{:.3}}            :{e:.3}:

combined
{{:>3.3}}          :{a:>3.3}:
{{:3.3}}           :{a:3.3}:
{{:3.3}}           :{c:3.3}:
{{:3.3}}           :{e:3.3}:
"""

print (s.format(a="1"*1, c="3"*3, e="5"*5))

Çıktı:

{:3}             :1  :

truncate
{:.3}            :555:

{:>3.3}          :  1:
{:3.3}           :1  :
{:3.3}           :333:
{:3.3}           :555:

Bir sınıf için özel biçimlendirme #

Not:

Aşağıdaki her şey, “str.format” yönteminin yanı sıra “biçim” işlevi. Aşağıdaki metinde ikisi birbirinin yerine kullanılabilir.

Python, “format” işlevine iletilen her değer için bu argüman için bir “format” yöntemi arar. Bu nedenle, kendi özel sınıfınız, ‘format’ işlevinin sınıfınızı ve niteliklerini nasıl görüntüleyeceğini ve biçimlendireceğini belirlemek için kendi ‘format’ yöntemine sahip olabilir.

Bu, ‘str’ yönteminden farklıdır, ‘format’ yönteminde olduğu gibi, hizalama, alan genişliği vb. dahil biçimlendirme dilini hesaba katabilir ve hatta (isterseniz) kendi biçim belirteçlerinizi ve kendi biçimlendirmenizi uygulayabilirsiniz. dil uzantıları.1

object.__format__(self, format_spec)

Örneğin :

# Example in Python 2 - but can be easily applied to Python 3

class Example(object):
    def __init__(self,a,b,c):
        self.a, self.b, self.c = a,b,c

    def __format__(self, format_spec):
        """ Implement special semantics for the 's' format specifier """
        # Reject anything that isn't an s
        if format_spec[-1] != 's':
            raise ValueError('{} format specifier not understood for this object', format_spec[:-1])

        # Output in this example will be (<a>,<b>,<c>)
        raw = "(" + ",".join([str(self.a), str(self.b), str(self.c)]) + ")"
        # Honor the format language by using the inbuilt string format
        # Since we know the original format_spec ends in an 's' 
        # we can take advantage of the str.format method with a 
        # string argument we constructed above
        return "{r:{f}}".format( r=raw, f=format_spec )

inst = Example(1,2,3)
print "{0:>20s}".format( inst )
# out :              (1,2,3)
# Note how the right align and field width of 20 has been honored.

Not:

Özel sınıfınız özel bir __format__ yöntemine sahip değilse ve sınıfın örneği “format” işlevine iletilir, Python2 her zaman ‘str’ veya ‘repr’ yönteminin dönüş değerini kullanır ne yazdırılacağını belirleme yöntemi (ve hiçbiri yoksa, o zaman varsayılan “repr” kullanılacaktır) ve “s” biçimini kullanmanız gerekecektir. bunu biçimlendirmek için belirteç. Python3 ile özel sınıfınızı geçmek için ‘format’ işlevi için, üzerinde ‘format’ yöntemini tanımlamanız gerekir özel sınıfınız.
