jQuery UI komut dosyasını ve temel kullanımı ekleme #

jQuery UI kitaplığına başlamak için jQuery betiğini, jQuery UI betiğini ve jQuery UI stil sayfasını HTML’nize eklemeniz gerekir.

İlk olarak, indirin jQuery UI; indirme sayfasında ihtiyacınız olan özellikleri seçin. İndirdiğiniz dosyayı açın ve “jquery-ui.css” ve “jquery-ui.js” (ve “jquery.js”) dosyalarını HTML’nizden (ör. diğer komut dosyaları ve stil sayfalarınızla) kullanabileceğiniz bir klasöre koyun.

jQuery UI, jQuery’ye bağlıdır, bu nedenle “jquery-ui.js"den önce “jquery.js"yi eklemeyi unutmayın.

<link rel="stylesheet" href="stylesheets/jquery-ui.css">
<script src="scripts/jquery.js"></script>
<script src="scripts/jquery-ui.js"></script>

Bu kadar! Artık jQuery UI’yi kullanabilirsiniz. Örneğin, veri seçiciyi aşağıdaki HTML ile kullanın:

<input type="text" name="date" id="date">

Ardından aşağıdaki JavaScript’i kullanın:

$("#date").datepicker();

İlk Kez Örnek için jQuery Kullanıcı Arayüzü Ayarlama #

jQuery UI çerçevesi, jQuery JavaScript kitaplığı için Kullanıcı Arayüzü kontrollerini genişletmeye ve artırmaya yardımcı olur.

jQuery UI’yi kullanmak istediğinizde, bu kitaplıkları HTML’nize eklemeniz gerekecektir. Başlamanın hızlı bir yolu, İçerik Dağıtım Ağı’ndaki mevcut kod kaynaklarını kullanmaktır:

jQuery Kitaplıkları

https://code.jquery.com/jquery-3.1.0.js
https://code.jquery.com/ui/1.12.0/jquery-ui.js

jQuery UI için birçok farklı tema seçebilir ve hatta Kendi Temanızı Yuvarlayabilirsiniz. Bu örnek için ‘Smoothness’ kullanacağız. Temayı CSS ile eklersiniz.

jQuery UI CSS’si

https://code.jquery.com/ui/1.12.0/themes/smoothness/jquery-ui.css

Hepsini bir araya koy

CDN’nizi indirdiğinizde veya seçtiğinizde, şimdi bu kitaplıkları ve stil sayfalarını HTML’nize eklemek isteyeceksiniz, böylece web sayfanız artık jQuery ve jQuery UI’yi kullanabilir. Kütüphaneleri yüklediğiniz sıra önemlidir. Önce jQuery kitaplığını, ardından jQuery UI kitaplığınızı arayın. jQuery UI, jQuery’yi genişlettiğinden, daha sonra çağrılmalıdır. HTML’niz aşağıdaki gibi görünebilir.

<html>
<head>
  <title>My First UI</title>
  <link rel="stylesheet" href="https://code.jquery.com/ui/1.12.0/themes/smoothness/jquery-ui.css">
  <script src="https://code.jquery.com/jquery-3.1.0.js"></script>
  <script src="https://code.jquery.com/ui/1.12.0/jquery-ui.js"></script>
  <script>
  $( function() {
    $( "#sortable" ).sortable();
    $( "#sortable" ).disableSelection();
  } );
  </script>
</head>
<body>
 
<ul id="sortable">
  <li class="ui-state-default">Item 1</li>
  <li class="ui-state-default">Item 2</li>
  <li class="ui-state-default">Item 3</li>
  <li class="ui-state-default">Item 4</li>
  <li class="ui-state-default">Item 5</li>
  <li class="ui-state-default">Item 6</li>
  <li class="ui-state-default">Item 7</li>
</ul>
 
</body>
</html>
