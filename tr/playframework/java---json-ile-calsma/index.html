Manuel JSON oluşturma #

import play.libs.Json;


public JsonNode createJson() {
    // {"id": 33, "values": [3, 4, 5]}
    ObjectNode rootNode = Json.newObject();
    ArrayNode listNode = Json.newArray();

    long values[] = {3, 4, 5};
    for (long val: values) {
        listNode.add(val);
    }
    
    rootNode.put("id", 33);
    rootNode.set("values", listNode);
    return rootNode;
}

Dize/dosyadan json yükleniyor #

import play.libs.Json;
// (...)

Ortak klasörünüzden bir dosya yükleme #

// Note: "app" is an play.Application instance
JsonNode node = Json.parse(app.resourceAsStream("public/myjson.json"));

Bir dizeden yükle #

String myStr = "{\"name\": \"John Doe\"}";
JsonNode node = Json.parse(myStr);

Bir JSON belgesini çaprazlama #

Aşağıdaki örneklerde “json”, aşağıdaki verilere sahip bir JSON nesnesi içerir:

[
  {
    "name": "John Doe",
    "work": {
      "company": {
        "name": "ASDF INC",
        "country": "USA"
      },
      "cargo": "Programmer"
    },
    "tags": ["java", "jvm", "play"]
  },
  {
    "name": "Bob Doe",
    "work": {
      "company": {
        "name": "NOPE INC",
        "country": "AUSTRALIA"
      },
      "cargo": "SysAdmin"
    },
    "tags": ["puppet", "ssh", "networking"],
    "active": true
  }
]

Bir kullanıcının adını alın (güvenli değil) #

JsonNode node = json.get(0).get("name"); // --> "John Doe"
// This will throw a NullPointerException, because there is only two elements
JsonNode node = json.get(2).get("name"); // --> *crash*

Kullanıcı adını alın (güvenli yol) #

JsonNode node1 = json.at("/0/name"); // --> TextNode("John Doe")
JsonNode node2 = json.at("/2/name"); // --> MissingNode instance
if (! node2.isMissingNode()) {
    String name = node2.asText();    
}

İlk kullanıcının çalıştığı ülkeyi alın #

JsonNode node2 = json.at("/0/work/company/country"); // TextNode("USA")

Her ülkeyi al #

List<JsonNode> d = json.findValues("country"); // List(TextNode("USA"), TextNode("AUSTRALIA"))

“Etkin” özniteliğini içeren her kullanıcıyı bulun #

List<JsonNode> e = json.findParents("active"); // List(ObjectNode("Bob Doe"))

JSON ve Java nesneleri arasında dönüştürme (temel) #

Varsayılan olarak, Jackson (Play JSON’un kullandığı kitaplık), her genel alanı aynı ada sahip bir json alanıyla eşleştirmeye çalışır. Nesnenin alıcıları/ayarlayıcıları varsa, onlardan adı çıkaracaktır. Bu nedenle, ISBN’yi saklamak için özel alana sahip bir ‘Kitap’ sınıfınız ve ‘getISBN/setISBN’ adlı get/set yöntemleriniz varsa, Jackson

  • Java’dan JSON’a dönüştürürken “ISBN” alanıyla bir JSON nesnesi oluşturun
  • Java nesnesindeki isbn alanını tanımlamak için “setISBN” yöntemini kullanın (JSON nesnesinin bir “ISBN” alanı varsa).

JSON’dan Java nesnesi oluşturun #

public class Person {
    String id, name;
}

JsonNode node = Json.parse("{\"id\": \"3S2F\", \"name\", \"Salem\"}");
Person person = Json.fromJson(node, Person.class);
System.out.println("Hi " + person.name); // Hi Salem

Java nesnesinden JSON nesnesi oluşturun #

// "person" is the object from the previous example
JsonNode personNode = Json.toJson(person)

JSON nesnesinden JSON dizesi oluşturma #

// personNode comes from the previous example
String json = personNode.toString();
// or 
String json = Json.stringify(json);

JSON güzel baskı #

System.out.println(personNode.toString());
/* Prints:
{"id":"3S2F","name":"Salem"}
*/   

System.out.println(Json.prettyPrint(personNode));
/* Prints:
{
  "id" : "3S2F",
  "name" : "Salem"
}
*/
