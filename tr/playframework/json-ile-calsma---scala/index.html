build.sbt içinde "com.typesafe.play" % "play-json_2.11" % "2.5.3", bkz.

  • https://mvnrepository.com/artifact/com.typesafe.play/play-json_2.11
  • http://stackoverflow.com/questions/19436069/adding-play-json-library-to-sbt

Manuel olarak JSON oluşturma #

Manuel olarak bir JSON nesne ağacı (bir “JsValue”) oluşturabilirsiniz.

import play.api.libs.json._

val json = JsObject(Map(
  "name" -> JsString("Jsony McJsonface"),
  "age" -> JsNumber(18),
  "hobbies" -> JsArray(Seq(
    JsString("Fishing"),
    JsString("Hunting"),
    JsString("Camping")
  ))
))

Veya birkaç örtük dönüşüme dayanan daha kısa eşdeğer sözdizimi ile:

import play.api.libs.json._

val json = Json.obj(
  "name" -> "Jsony McJsonface",
  "age" -> 18,
  "hobbies" -> Seq(
    "Fishing",
    "Hunting",
    "Camping"
  )
)

JSON dizesini almak için:

json.toString
// {"name":"Jsony McJsonface","age":18,"hobbies":["Fishing","Hunting","Camping"]}
Json.prettyPrint(json)  
//  {
//    "name" : "Jsony McJsonface",
//    "age" : 18,
//    "hobbies" : [ "Fishing", "Hunting", "Camping" ]
//  }

Scala: JSON’u manuel olarak okuma #

Size bir JSON dizesi verilirse:

val str =
    """{
    |    "name" : "Jsony McJsonface",
    |    "age" : 18,
    |    "hobbies" : [ "Fishing", "Hunting", "Camping" ],
    |    "pet" : {
    |        "name" : "Doggy",
    |        "type" : "dog"
    |    }
    |}""".stripMargin

JSON ağacını temsil eden bir JsValue elde etmek için onu ayrıştırabilirsiniz.

val json = Json.parse(str)

Ve belirli değerleri aramak için ağaçta gezinin:

(json \ "name").as[String]          // "Jsony McJsonface"

faydalı yöntemler #

  • \ bir JSON nesnesindeki belirli bir anahtara gitmek için

  • Bir JSON nesnesindeki belirli bir anahtarın tüm oluşumlarına gitmek için `\’ iç içe nesnelerde yinelemeli olarak arama yapmak

  • .apply(idx) (yani (idx)) dizideki bir dizine gitmek için

  • “.as[T]” kesin bir alt türe atamak için

  • ‘.asOpt[T]’ kesin bir alt türe dönüştürme girişiminde bulunur, yanlış türse Yok değerini döndürür

  • “.validate[T]”, bir JsSuccess veya JsError döndürerek kesin bir alt türe bir JSON değeri atmayı denemek için

    (json \ “name”).as[String] // “Jsony McJsonface” (json \ “pet” \ “name”).as[String] // “Doggy” (json \ “name”).map(_.as[String]) // List(“Jsony McJsonface”, “Doggy”) (json \ “type”)(0).as[String] // “dog” (json \ “wrongkey”).as[String] // throws JsResultException (json \ “age”).as[Int] // 18 (json \ “hobbies”).as[Seq[String]] // List(“Fishing”, “Hunting”, “Camping”) (json \ “hobbies”)(2).as[String] // “Camping” (json \ “age”).asOpt[String] // None (json \ “age”).validate[String] // JsError containing some error detail

Vaka sınıflarına/dan itibaren otomatik olarak eşleme #

Genel olarak JSON ile çalışmanın en kolay yolu, doğrudan JSON ile eşlenen bir vaka sınıfına sahip olmaktır. (aynı alan adı, eşdeğer türler, vb.).

case class Person(
  name: String,
  age: Int,
  hobbies: Seq[String],
  pet: Pet
)

case class Pet(
  name: String,
  `type`: String
)

// these macros will define automatically the conversion to/from JSON
// based on the cases classes definition
implicit val petFormat = Json.format[Pet]
implicit val personFormat = Json.format[Person]

Json’a dönüştürme #

val person = Person(
  "Jsony McJsonface",
  18,
  Seq("Fishing", "Hunting", "Camping"),
  Pet("Doggy", "dog")
)

Json.toJson(person).toString
// {"name":"Jsony McJsonface","age":18,"hobbies":["Fishing","Hunting","Camping"],"pet":{"name":"Doggy","type":"dog"}}

Json’dan dönüştürme #

val str =
    """{
    |    "name" : "Jsony McJsonface",
    |    "age" : 18,
    |    "hobbies" : [ "Fishing", "Hunting", "Camping" ],
    |     "pet" : {
    |       "name" : "Doggy",
    |       "type" : "dog"
    |     }
    |}""".stripMargin


Json.parse(str).as[Person]
// Person(Jsony McJsonface,18,List(Fishing, Hunting, Camping),Pet(Doggy,dog))
