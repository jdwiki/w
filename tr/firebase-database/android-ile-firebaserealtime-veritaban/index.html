Firebase’den değer/değerler alın #

Android’de Gerçek Zamanlı Veritabanını Ekleyin #

  1. Uygulamanızı Firebase’e bağlamak için Yükleme ve kurulum işlemini tamamlayın. Bu, projeyi Firebase’de yaratacaktır.

  2. Modül düzeyindeki “build.gradle” dosyanıza Firebase Realtime Database bağımlılığını ekleyin:

    compile ‘com.google.firebase:firebase-database:9.2.1’

  3. Firebase Veritabanı Kurallarını Yapılandırın

Artık Android’de Gerçek Zamanlı Veritabanı ile çalışmaya hazırsınız.

Örneğin, ‘message’ tuşunun altındaki veritabanına ‘Merhaba Dünya’ mesajı yazıyorsunuz.

// Write a message to the database
FirebaseDatabase database = FirebaseDatabase.getInstance();
DatabaseReference myRef = database.getReference("message");

myRef.setValue("Hello, World!");

Verileri kaydetmek için setValue kullanma #

SetValue() yöntemi, tüm alt düğümler dahil, belirtilen konumdaki verilerin üzerine yazar.

Bu yöntemi aşağıdakiler için kullanabilirsiniz:

  1. Mevcut JSON türlerine karşılık gelen türleri aşağıdaki gibi iletin:
  • Sicim
  • Uzun
  • Çift Kişilik
  • Boole
  • Harita<Dize, Nesne>
  • Liste
  1. Özel bir Java nesnesini iletin, eğer onu tanımlayan sınıf, hiçbir argüman almayan ve atanacak özellikler için genel alıcılara sahip bir varsayılan kurucuya sahipse.

Bu, CustomObject ile bir örnektir. Önce nesneyi tanımlayın.

@IgnoreExtraProperties
public class User {

    public String username;
    public String email;

    public User() {
        // Default constructor required for calls to DataSnapshot.getValue(User.class)
    }

    public User(String username, String email) {
        this.username = username;
        this.email = email;
    }

Ardından Veritabanı referansını alın ve değeri ayarlayın:

   User user = new User(name, email);
   DatabaseReference mDatabase mDatabase = FirebaseDatabase.getInstance().getReference();
   mDatabase.child("users").child(userId).setValue(user);
