Temel şablon kullanma #

base.twig.html #

<!DOCTYPE html>
<html>
    <head>
        <title>{{ title | default('Hello World') }}</title>
        <link rel="stylesheet" type="text/css" href="theme.css">
        {% block css %}
        {% endblock %}
    </head>
    <body>
    {% block body %}
        <nav>
        {% block navigation %}
            <a href="#">Link</a>
            <a href="#">Link</a>
            <a href="#">Link</a>
        {% endblock navigation %}
        </nav>
        <section id="container">
            <section id="content">
            {% block content %}
            <p>
            Lorem ipsum dolor sit amet.
            </p>
        </section>
        {% endblock content %}
    </section>
    {% endblock body %}
    </body>
</html>

sayfa.twig.html

{% extends base.twig.html %}
{% block navigation %}
<a href="page2.html">Page 2</a>
<a href="page3.html">Page 3</a>
<a href="page4.html">Page 4</a>
{% endblock %}
{% block content %}
This is my first page
{% endblock content %}

Dahil edilen bir şablonun içeriğini değiştirin #

makale.twig.html #

<article>
    <h1>{{ article.title }}</h1>
    {% block content %}
    <p>{{ article.content }}</p>
    {% endblock %}
</article>

makaleler.twig.html #

{# use default template for article #}
{% for article in articles %}
    {% include "article.twig.html" %}
{% endfor %}

{# use custom template for article #}
{% for article in articles %}
    {% embed "article.twig.html" %}
        {% block content %}
            <img src="{{ article.image }}" alt="{{ article.title }}" />
            {{ parent() }}
        {% endblock %}
    {% endembed %}
{% endfor %}

Değişken devralma #

ebeveyn.html.twig #

<!DOCTYPE html>
<html>
    <head>
        <title>{{ title_variable | default('Normal Page') }}</title>
    </head>
    <body>
         <h1>This is the {{ title_variable }} Page</h1>    
    </body>
</html>

alt.html.twig #

{% extends "parent.html.twig" %}
{% set title_variable = "Child" %}

Dahil Etme, Genişletme, Kullanım, Makro, Gömme Karşılaştırması #

Twig’de çeşitli türde kalıtım ve kod yeniden kullanımı vardır:

Katmak #

Ana hedef kod yeniden kullanımıdır. Örnek olarak “base.html.twig” içinde “header.html.twig” ve “footer.html.twig” kullanmayı düşünün.

header.html.twig

<nav>
   <div>Homepage</div>
   <div>About</div>
</nav>

base.html.twig

{% include 'header.html.twig' %}
<main>{% block main %}{% endblock %}</main>

Genişler #

Ana hedef dikey kalıtımdır. Örnek olarak “homepage.html.twig” ve “about.html.twig” içindeki “base.html.twig"i genişletmeyi düşünün.

base.html.twig

{% include 'header.html.twig' %}
<main>{% block main %}{% endblock %}</main>

anasayfa.html.twig

{% extends 'base.html.twig' %}

{% block main %}
<p>You are at the homepage</p>
{% endblock %}

hakkında.html.twig

{% extends 'base.html.twig' %}

{% block main %}
<p>You are at the about page</p>
{% endblock %}

Kullanmak #

Ana hedef yatay yeniden kullanımdır. “single.product.html.twig” (“product.layout.html.twig” öğesini genişletir) ve “single.service.html.twig” (“service.layout’u genişletir)” içinde “sidebar.product.html.twig” kullanmayı düşünün. html.page’) sayfaları. (makrolar gibi, ancak bloklar için)

kenar çubuğu.html.twig

<aside>{% block sidebar %}{% endblock %}</aside>

single.product.html.twig

{% extends 'product.layout.html.twig' %}

{% use 'sidebar.html.twig' %}
{% block main %}
<p>You are at the product page for product number 123</p>
{% endblock %}

single.service.html.twig

{% extends 'service.layout.html.twig' %}

{% use 'sidebar.html.twig' %}
{% block main %}
<p>You are at the service page for service number 456</p>
{% endblock %}

makro #

Ana hedef, değişkenli birçok şablonda yeniden kullanılabilir işaretlemeye sahip olmaktır. Bazı değişkenleri alan ve bazı işaretlemeler veren bir fonksiyon düşünün.

form.html.twig

{% makro girişi(ad, değer, tür) %} <input type=”{{ type|default(’text’) }}" name="{{ name }}" value="{{ value|e }}" }}" /> {% son makro %} #

profil.hizmet.html.twig

{% import "forms.html.twig" as forms %}

<div>{{ forms.input('username') }}</div>

Göm #

Ana hedef blok geçersiz kılmaktır. Hem “Kullan” hem de “Dahil Et” işlevlerine sahiptir. “page.html.twig"i “product.table.html.twig” ve “service.table.html.twig” içine yerleştirmeyi düşünün.

pagination.html.twig

<div>
    <div>{% block first %}{% endblock %}</div>
    {% for i in (min + 1)..(max - 1) %}
        <div>{{ i }}</div>
    {% endfor %}
    <div>{% block last %}{% endblock %}</div>
</div>

ürün.tablo.html.twig

{% set min, max = 1, products.itemPerPage %}

{% embed 'pagination.html.twig' %}
    {% block first %}First Product Page{% endblock %}
    {% block last %}Last Product Page{% endblock %}
{% endembed %}

service.table.html.twig

{% set min, max = 1, services.itemPerPage %}

{% embed 'pagination.html.twig' %}
    {% block first %}First Service Page{% endblock %}
    {% block last %}Last Service Page{% endblock %}
{% endembed %}

Lütfen gömülü dosyanın (burada “pagination.html.twig”) mevcut bağlama erişimi olduğunu unutmayın (burada “min”, “maks” değişkenleri). Ayrıca gömülü dosyaya fazladan değişkenler iletebilirsiniz:

pagination.html.twig

<p>{{ count }} items</p>
<div>
    <div>{% block first %}{% endblock %}</div>
    {% for i in (min + 1)..(max - 1) %}
        <div>{{ i }}</div>
    {% endfor %}
    <div>{% block last %}{% endblock %}</div>
</div>

ürün.tablo.html.twig

{% set min, max = 1, products|length %}

{% embed 'pagination.html.twig' with {'count': products|length } %}
    {% block first %}First Product Page{% endblock %}
    {% block last %}Last Product Page{% endblock %}
{% endembed %}
