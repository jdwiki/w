Kurulum/Kurulum #

İndiriliyor:

  • GitHub deposundan Bootstrap doğrudan veya klon vb. indirin
  • Bootstrap’in özelleştirilmiş sürümünü resmi belgeler adresinden indirin
  • Bower ile yükleyin: bower install bootstrap
  • npm ile kurun: npm install bootstrap
  • Besteci ile yükleyin: `besteci twbs/bootstrap gerektirir'

Dosya Yapısı

Yükleme:

HTML sayfanıza Bootstrap’in CSS’sini, JS’sini ve jQuery’nin bağımlılığını (ön sürüm 3, en azından en son Bootstrap sürümünden itibaren) ekleyin. Bootstrap’ın JavaScript özelliklerini kullanmayı planlıyorsanız, jQuery referansınızın HTML’nizdeki bootstrap.js referansınızdan önce gelmesi gerektiğini lütfen unutmayın.

Yüklü Bootstrap dosyalarınızı yukarıdaki bölümden kullanabilir veya Bootstrap üreticileri tarafından sağlanan bir CDN’ye başvurabilirsiniz (bağlantılar Bootstrap ile Başlarken’ten alınmıştır):

<!-- Latest compiled and minified CSS -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css" integrity="sha384-BVYiiSIFeK1dGmJRAkycuHAHRg32OmUcww7on3RYdg4Va+PmSTsz/K68vbdEjh4u" crossorigin="anonymous">

<!-- Optional theme -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap-theme.min.css" integrity="sha384-rHyoN1iRsVXV4nD0JutlnGaslCJuC7uwjduW9SVrLvRYooPp2bWYgmgJQIXwl/Sp" crossorigin="anonymous">

<!-- Latest compiled and minified JavaScript -->
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js" integrity="sha384-Tc5IQib027qvyjSMfHjOMaLkfuWVxZxUPnCJA7l2mCWNIpG9mGCD8wGNIcPD7Txa" crossorigin="anonymous"></script>

Çok basit bir Bootstrap web sayfası:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <!-- The above 3 meta tags *must* come first in the head; any other head content must come *after* these tags -->
    <title>Bootstrap 101 Template</title>

    <!-- Bootstrap -->
    <link href="css/bootstrap.min.css" rel="stylesheet">

    <!-- HTML5 shim and Respond.js for IE8 support of HTML5 elements and media queries -->
    <!-- WARNING: Respond.js doesn't work if you view the page via file:// -->
    <!--[if lt IE 9]>
      <script src="https://oss.maxcdn.com/html5shiv/3.7.2/html5shiv.min.js"></script>
      <script src="https://oss.maxcdn.com/respond/1.4.2/respond.min.js"></script>
    <![endif]-->
  </head>
  <body>
    <h1>Hello, world!</h1>

    <!-- jQuery (necessary for Bootstrap's JavaScript plugins) -->
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>
    <!-- Include all compiled plugins (below), or include individual files as needed -->
    <script src="js/bootstrap.min.js"></script>
  </body>
</html>

Bootstrap ne zaman kullanılır? #

Bootstrap, HTML, CSS ve Javascript için düşünülmüş bir çerçevedir. Form öğeleri, düğmeler, kalıcı pencereler ve gezinme öğeleri gibi kabul edilen [Kullanıcı Arayüzü] öğeleri için temel stil ve işlevsellik içerir.

Bootstrap duyarlı bir web çerçevesidir, yani mobil cihazlar, tabletler ve masaüstü bilgisayarlar gibi büyük ve küçük ekran boyutları için düzeni ve tasarımı tek bir kod tabanında uyarlamak için tasarlanmıştır.

Bootstrap’in temel kavramlarından biri ızgara çerçevesidir. HTML öğelerine sınıflar uygulayarak, on iki sütundan oluşan temel bir ızgara kullanarak karmaşık düzenler oluşturmak mümkündür. Örneğin, dört sütunlu bir düzen, tablet cihazlarda iki sütuna ve mobil cihazlarda bir sütuna uyarlanabilir. Izgara, bunu başarmak için belirli ekran boyutlarını hedeflemeye yönelik bir CSS yöntemi olan “medya sorgularını” kullanır.

Bootstrap özellikle aşağıdaki durumlarda iyi performans gösterir:

  • Özel tasarım öncelikli değildir
  • HTML’yi düzenlemek ve sınıf eklemek, özel CSS oluşturmaktan daha rahatsınız
  • Diğer birçok web sitesiyle pek çok görsel benzerliğe sahip olacak bir çerçeve kullanmakta rahatsınız.

Bootstrap, belgeler mükemmel olduğundan HTML, CSS ve Javascript’te yeni olanlar tarafından kullanılabilir. Ancak, Bootstrap tarafından kullanılan üç temel teknolojiye (HTML, CSS ve Javascript) tam olarak hakim olmayanlar için bir öğrenme eğrisi vardır.

Bootstrap’in stilini veya işlevselliğini değiştirmek için Bootstrap temaları satın almak veya indirmek mümkündür. CSS ve Javascript’in özelleştirilmesiyle Bootstrap’i başlangıç ​​noktası olarak kullanmak da mümkündür.
