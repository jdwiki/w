Sintaxe #

  • localStorage.setItem(nome, valor);

  • localStorage.getItem(nome);

  • localStorage.name = valor;

  • localStorage.name;

  • localStorage.clear()

  • localStorage.removeItem(nome);

Parâmetros #

Parâmetro Descrição
nome A chave/nome do item
valor O valor do item

A API do Web Storage é especificada no WHATWG HTML Living Standard.

Usando localStorage #

O objeto localStorage fornece armazenamento persistente (mas não permanente - veja os limites abaixo) de valores-chave de strings. Quaisquer alterações são imediatamente visíveis em todas as outras janelas/quadros da mesma origem. Os valores armazenados persistem indefinidamente, a menos que o usuário limpe os dados salvos ou configure um limite de expiração. localStorage usa uma interface semelhante a um mapa para obter e definir valores.

localStorage.setItem('name', "John Smith");
console.log(localStorage.getItem('name')); // "John Smith"

localStorage.removeItem('name');
console.log(localStorage.getItem('name')); // null

Se você deseja armazenar dados estruturados simples, você pode usar JSON para serializá-los de e para strings para armazenamento.

var players = [{name: "Tyler", score: 22}, {name: "Ryan", score: 41}];
localStorage.setItem('players', JSON.stringify(players));

console.log(JSON.parse(localStorage.getItem('players')));
// [ Object { name: "Tyler", score: 22 }, Object { name: "Ryan", score: 41 } ]

Manuseio mais simples de armazenamento #

localStorage, sessionStorage são objetos JavaScript e você pode tratá-los como tal. Em vez de usar Métodos de Armazenamento como .getItem(), .setItem(), etc, o… aqui está uma alternativa mais simples:

// Set
localStorage.greet = "Hi!"; // Same as: window.localStorage.setItem("greet", "Hi!");

// Get
localStorage.greet;         // Same as: window.localStorage.getItem("greet");

// Remove item
delete localStorage.greet;  // Same as: window.localStorage.removeItem("greet");

// Clear storage
localStorage.clear();

Exemplo:

// Store values (Strings, Numbers)
localStorage.hello = "Hello";
localStorage.year  = 2017;    

// Store complex data (Objects, Arrays)
var user = {name:"John", surname:"Doe", books:["A","B"]};
localStorage.user = JSON.stringify( user );

// Important: Numbers are stored as String
console.log( typeof localStorage.year ); // String

// Retrieve values
var someYear = localStorage.year; // "2017"

// Retrieve complex data
var userData = JSON.parse( localStorage.user );
var userName = userData.name; // "John"

// Remove specific data
delete localStorage.year; 

// Clear (delete) all stored data
localStorage.clear();

Eventos de armazenamento #

Sempre que um valor for definido em localStorage, um evento storage será despachado em todas as outras windows da mesma origem. Isso pode ser usado para sincronizar o estado entre diferentes páginas sem recarregar ou se comunicar com um servidor. Por exemplo, podemos refletir o valor de um elemento de entrada como texto de parágrafo em outra janela:

Primeira janela #

var input = document.createElement('input');
document.body.appendChild(input);

input.value = localStorage.getItem('user-value');

input.oninput = function(event) {
  localStorage.setItem('user-value', input.value);
};

Segunda Janela #

var output = document.createElement('p');
document.body.appendChild(output);

output.textContent = localStorage.getItem('user-value');

window.addEventListener('storage', function(event) {
  if (event.key === 'user-value') {
    output.textContent = event.newValue;
  }
});

Notas #

O evento não é acionado ou detectável no Chrome, Edge e Safari se o domínio foi modificado por meio de script.

Primeira janela #

// page url: http://sub.a.com/1.html
document.domain = 'a.com';

var input = document.createElement('input');
document.body.appendChild(input);

input.value = localStorage.getItem('user-value');

input.oninput = function(event) {
  localStorage.setItem('user-value', input.value);
};

Segunda Janela #

// page url: http://sub.a.com/2.html
document.domain = 'a.com';

var output = document.createElement('p');
document.body.appendChild(output);

// Listener will never called under Chrome(53), Edge and Safari(10.0).
window.addEventListener('storage', function(event) {
  if (event.key === 'user-value') {
    output.textContent = event.newValue;
  }
});

sessãoArmazenamento #

O objeto sessionStorage implementa a mesma interface Storage que localStorage. No entanto, em vez de serem compartilhados com todas as páginas da mesma origem, os dados do sessionStorage são armazenados separadamente para cada janela/guia. Os dados armazenados persistem entre as páginas nessa janela/guia enquanto estiverem abertos, mas não são visíveis em nenhum outro lugar.

var audio = document.querySelector('audio');

// Maintain the volume if the user clicks a link then navigates back here.
audio.volume = Number(sessionStorage.getItem('volume') || 1.0);
audio.onvolumechange = function(event) {
  sessionStorage.setItem('volume', audio.volume);
};

Salvar dados em sessionStorage

sessionStorage.setItem('key', 'value');

Obtenha dados salvos do sessionStorage

var data = sessionStorage.getItem('key');

Remova os dados salvos do sessionStorage

sessionStorage.removeItem('key')

Limpando o armazenamento #

Para limpar o armazenamento, basta executar

localStorage.clear();

Condições de erro #

A maioria dos navegadores, quando configurados para bloquear cookies, também bloquearão localStorage. Tentativas de usá-lo resultarão em uma exceção. Não se esqueça de gerenciar esses casos.

var video = document.querySelector('video')
try {
    video.volume = localStorage.getItem('volume')
} catch (error) {
    alert('If you\'d like your volume saved, turn on cookies')
}
video.play()

Se o erro não fosse tratado, o programa pararia de funcionar corretamente.

Remover item de armazenamento #

Para remover um item específico do armazenamento do navegador (o oposto de setItem) use removeItem

localStorage.removeItem("greet");

Exemplo:

localStorage.setItem("greet", "hi");
localStorage.removeItem("greet");

console.log( localStorage.getItem("greet") ); // null

(O mesmo se aplica a sessionStorage)

localComprimento do armazenamento #

A propriedade localStorage.length retorna um número inteiro indicando o número de elementos no localStorage

Exemplo:

Definir itens

localStorage.setItem('StackOverflow', 'Documentation');
localStorage.setItem('font', 'Helvetica');
localStorage.setItem('image', 'sprite.svg');

Obter comprimento

localStorage.length; // 3
