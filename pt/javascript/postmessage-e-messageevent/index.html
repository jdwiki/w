.postMessage() e MessageEvent

Sintaxe #

  • windowObject.postMessage(message, targetOrigin, [transfer]);
  • window.addEventListener("message", receiveMessage);

Parâmetros #

Parâmetros  
mensagem  
targetOrigin  
transferência opcional

Começando #

O que é .postMessage(), quando e por que o usamos #

.postMessage() é uma forma de permitir a comunicação segura entre scripts de origem cruzada.

Normalmente, duas páginas diferentes só podem se comunicar diretamente usando JavaScript quando estão sob a mesma origem, mesmo que uma delas esteja incorporada a outra (por exemplo, iframes) ou uma seja aberta de dentro da outra (por exemplo, window .abrir()). Com .postMessage(), você pode contornar essa restrição mantendo-se seguro.

Você só pode usar .postMessage() quando tiver acesso ao código JavaScript de ambas as páginas. Como o destinatário precisa validar o remetente e processar a mensagem de acordo, você só pode usar este método para se comunicar entre dois scripts aos quais você tem acesso.

Construiremos um exemplo para enviar mensagens para uma janela filho e fazer com que as mensagens sejam exibidas na janela filho. A página pai/remetente será assumida como http://sender.com e a página filha/receptor será considerada como http://receiver.com para o exemplo.

#Enviando mensagens

Para enviar mensagens para outra janela, você precisa ter uma referência ao seu objeto window. window.open() retorna o objeto de referência da janela recém-aberta. Para outros métodos para obter uma referência a um objeto de janela, veja a explicação no parâmetro otherWindow aqui.

var childWindow = window.open("http://receiver.com", "_blank");

Adicione uma textarea e um botão de envio que serão usados ​​para enviar mensagens para a janela filha.

<textarea id="text"></textarea>
<button id="btn">Send Message</button>

Envie o texto de textarea usando .postMessage(message, targetOrigin) quando o button for clicado.

var btn = document.getElementById("btn"),
    text = document.getElementById("text");

btn.addEventListener("click", function () {
    sendMessage(text.value);
    text.value = "";
});

function sendMessage(message) {
    if (!message || !message.length) return;
    childWindow.postMessage(JSON.stringify({
        message: message,
        time: new Date()
    }), 'http://receiver.com');
}

Para enviar e receber objetos JSON em vez de uma simples string, os métodos JSON.stringify() e JSON.parse() podem ser usados. Um Transfarable Object pode ser fornecido como o terceiro parâmetro opcional do método .postMessage(message, targetOrigin, transfer), mas o suporte do navegador ainda está faltando mesmo em navegadores modernos.

Para este exemplo, como nosso receptor é assumido como sendo a página http://receiver.com, nós inserimos sua url como targetOrigin. O valor deste parâmetro deve corresponder ao origin do objeto childWindow para a mensagem a ser enviada. É possível usar * como um curinga, mas é altamente recomendado evitar o uso do curinga e sempre definir este parâmetro para a origem específica do receptor por motivos de segurança.

Recebendo, Validando e Processando Mensagens #

O código desta parte deve ser colocado na página do receptor, que é http://receiver.com para nosso exemplo.

Para receber mensagens, o evento de mensagem da janela deve ser escutado.

window.addEventListener("message", receiveMessage);

Quando uma mensagem é recebida, há algumas etapas que devem ser seguidas para garantir a segurança o máximo possível.

  • Validar o remetente
  • Validar a mensagem
  • Processar a mensagem

O remetente deve sempre ser validado para garantir que a mensagem seja recebida de um remetente confiável. Depois disso, a própria mensagem deve ser validada para garantir que nada malicioso seja recebido. Após essas duas validações, a mensagem pode ser processada.

function receiveMessage(ev) {
    //Check event.origin to see if it is a trusted sender.
    //If you have a reference to the sender, validate event.source
    //We only want to receive messages from http://sender.com, our trusted sender page.
    if (ev.origin !== "http://sender.com" || ev.source !== window.opener)
        return;

    //Validate the message
    //We want to make sure it's a valid json object and it does not contain anything malicious 
    var data;
    try {
        data = JSON.parse(ev.data);
        //data.message = cleanseText(data.message) 
    } catch (ex) {
        return;
    }

    //Do whatever you want with the received message
    //We want to append the message into our #console div
    var p = document.createElement("p");
    p.innerText = (new Date(data.time)).toLocaleTimeString() + " | " + data.message;
    document.getElementById("console").appendChild(p);
}
