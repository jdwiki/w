Especificação ECMAScript 2015 19.4 Símbolos

Noções básicas de tipo primitivo de símbolo #

Symbol é um novo tipo primitivo no ES6. Os símbolos são usados ​​principalmente como chaves de propriedade, e uma de suas principais características é que são únicos, mesmo que tenham a mesma descrição. Isso significa que eles nunca terão um conflito de nome com qualquer outra chave de propriedade que seja um símbolo ou string.

const MY_PROP_KEY = Symbol();
const obj = {};

obj[MY_PROP_KEY] = "ABC";
console.log(obj[MY_PROP_KEY]);

Neste exemplo, o resultado de console.log seria ABC.

Você também pode ter símbolos nomeados como:

const APPLE    = Symbol('Apple');
const BANANA   = Symbol('Banana');
const GRAPE    = Symbol('Grape');

Cada um desses valores é exclusivo e não pode ser substituído.

Fornecer um parâmetro opcional (description) ao criar símbolos primitivos pode ser usado para depuração, mas não para acessar o símbolo em si (mas veja o [Symbol.for()](https://www.wikiod.com/pt/docs/javascript /2764/symbols/19176/using-symbol-for-to-create-global-shared-symbols) para uma maneira de registrar/pesquisar símbolos compartilhados globais).

Usando Symbol.for() para criar símbolos globais e compartilhados #

O método Symbol.for permite que você registre e procure símbolos globais por nome. Na primeira vez que é chamado com uma determinada chave, ele cria um novo símbolo e o adiciona ao registro.

let a = Symbol.for('A');

A próxima vez que você chamar Symbol.for('A'), o mesmo símbolo será retornado em vez de um novo (em contraste com Symbol('A') que criaria um símbolo novo e único que acontece de ter a mesma descrição).

a === Symbol.for('A') // true

mas

a === Symbol('A') // false

Convertendo um símbolo em uma string #

Ao contrário da maioria dos outros objetos JavaScript, os símbolos não são convertidos automaticamente em uma string ao realizar a concatenação.

let apple = Symbol('Apple') + ''; // throws TypeError!

Em vez disso, eles precisam ser explicitamente convertidos em uma string quando necessário (por exemplo, para obter uma descrição textual do símbolo que pode ser usado em uma mensagem de depuração) usando o método toString ou o construtor String.

const APPLE = Symbol('Apple');
let str1 = APPLE.toString(); // "Symbol(Apple)"
let str2 = String(APPLE);    // "Symbol(Apple)"
