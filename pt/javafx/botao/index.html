Adicionando um gráfico a um botão #

Criar um botão #

A criação de um Button é simples:

Button sampleButton = new Button();

Isso criará um novo Button sem nenhum texto ou gráfico dentro.

Se você quiser criar um Button com um texto, basta usar o construtor que recebe um String como parâmetro (que define a textProperty do Button):

Button sampleButton = new Button("Click Me!");

Se você deseja criar um Button com um gráfico dentro ou qualquer outro Node, use este construtor:

Button sampleButton = new Button("I have an icon", new ImageView(new Image("icon.png")));

Adicionando um ouvinte de ação #

Os botões disparam eventos de ação quando são ativados (por exemplo, clicados, uma combinação de teclas para o botão é pressionada, …).

button.setOnAction(new EventHandler<ActionEvent>() {
    @Override
    public void handle(ActionEvent event) {
        System.out.println("Hello World!");
    }
});

Se você estiver usando Java 8+, poderá usar lambdas para ouvintes de ação.

button.setOnAction((ActionEvent a) -> System.out.println("Hello, World!"));
// or
button.setOnAction(a -> System.out.println("Hello, World!"));

Botões Padrão e Cancelar #

A API Button fornece uma maneira fácil de atribuir atalhos de teclado comuns a botões sem a necessidade de acessar a lista de aceleradores atribuída a Scene ou ouvir explicitamente os eventos de tecla. Ou seja, dois métodos de conveniência são fornecidos: setDefaultButton e setCancelButton:

  • Definir setDefaultButton para true fará com que o Button seja acionado toda vez que receber um evento KeyCode.ENTER.

  • Definir setCancelButton para true fará com que o Button seja acionado toda vez que receber um evento KeyCode.ESCAPE.

O exemplo a seguir cria uma Scene com dois botões que são acionados quando as teclas Enter ou Escape são pressionadas, independentemente de estarem focadas ou não.

FlowPane root = new FlowPane();
        
Button okButton = new Button("OK");
okButton.setDefaultButton(true);
okButton.setOnAction(e -> {
    System.out.println("OK clicked.");
});
        
Button cancelButton = new Button("Cancel");            
cancelButton.setCancelButton(true);
cancelButton.setOnAction(e -> {
    System.out.println("Cancel clicked.");
});
        
root.getChildren().addAll(okButton, cancelButton);
Scene scene = new Scene(root);

O código acima não funcionará se esses KeyEvents forem consumidos por qualquer Node pai:

scene.setOnKeyPressed(e -> {
    e.consume();
});
