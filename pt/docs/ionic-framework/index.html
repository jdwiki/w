Tutorial de estrutura iônica

Ionic é um framework para desenvolvimento de aplicativos móveis com HTML, CSS e JavaScript. Os aplicativos Ionic são executados como aplicativos nativos e têm uma “aparência e comportamento” nativos.

O Ionic é construído na estrutura AngularJS e fornece uma solução completa para projetar, construir e empacotar aplicativos móveis. O design é realizado com uma coleção de ferramentas de modelo e uma biblioteca de ícones personalizados. O Ionic fornece componentes CSS/SASS personalizados, bem como Javascript UI Extensions. Os aplicativos Ionic podem ser criados, emulados e empacotados com sua Command Line Interface (CLI).

Os modelos Ionic são dinâmicos e responsivos e se adaptam ao seu ambiente para fornecer uma “aparência e comportamento” nativo. Essa adaptação inclui layout, estilo e ícones. O Ionic também disponibiliza [personalização de plataforma] 2 independente. Como os aplicativos Ionic usam tecnologia web de front-end, eles também podem ser visualizados em um navegador para um desenvolvimento mais rápido.

Os aplicativos Ionic são criados em cima do Apache Cordova por padrão. Eles têm acesso a todos os Plugins Cordova que permitem que você use funcionalidades nativas, como notificações push, câmera, acelerômetro etc. Os aplicativos Cordova funcionam em várias plataformas e dispositivos (telefones, tablets, etc.). ) com muito pouco esforço extra. O Cordova pode ser trocado por outras tecnologias de plataforma cruzada, como trigger.io.
