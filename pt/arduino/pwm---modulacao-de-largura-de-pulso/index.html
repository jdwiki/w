Controle um motor DC através da porta Serial usando PWM #

Neste exemplo pretendemos realizar uma das tarefas mais comuns: Tenho um pequeno motor DC por aí, como uso meu Arduino para controlá-lo? Fácil, com PWM e comunicação serial, usando a função analogWrite() e a biblioteca Serial.

O básico #

Pulse Width Modulation ou PWM para abreviar é uma técnica para imitar sinais analógicos usando saída digital. Como é que isso funciona? Usando um trem de pulsos cuja relação D (duty cycle) entre o tempo em alto nível (digital 1, geralmente 5V) e o tempo em baixo nível (digital 0, 0V) em cada período pode ser modificada para produzir uma tensão média entre esses dois níveis:

Usando a função analogWrite(pin,value) do Arduino, podemos variar o value do ciclo de trabalho da saída do pin. Observe que o pin deve ser colocado no modo de saída e o value deve estar entre 0 (0V) e 255 (5V). Qualquer valor intermediário simulará uma saída analógica intermediária proporcional.

No entanto, a finalidade dos sinais analógicos geralmente está relacionada ao controle de sistemas mecânicos que exigem mais tensão e corrente do que a placa Arduino sozinha é capaz. Neste exemplo, aprenderemos como amplificar os recursos PWM do Arduino.

Para isso é usado um diodo MOSFET. Em essência, este diodo atua como um interruptor. Permite ou interrompe o fluxo elétrico entre seus terminais fonte e dreno. Mas em vez de um interruptor mecânico, possui um terceiro terminal chamado gate. Uma corrente muito pequena (<1mA) “abrirá” esta porta e permitirá que a corrente flua. Isso é muito conveniente, pois podemos enviar a saída PWM do Arduino para esta porta, criando assim outro trem de pulsos PWM com o mesmo ciclo de trabalho através do MOSFET, o que permite tensões e correntes que destruiriam o Arduino.

Lista de materiais: o que você precisa para construir este exemplo #

  • Diodo MOSFET: por exemplo, o popular BUZ11
  • Diodo de proteção para o motor: Schottky SB320
  • Resistor: qualquer coisa 10K ~ 1M Ohm
  • Motor: Um motor pequeno típico (um típico pode ser de 12V)
  • Uma fonte de alimentação compatível com o motor que você selecionou
  • Uma tábua de pão
  • Cabos coloridos!
  • Um Arduino, mas você já sabia disso.

A construção #

Junte tudo! Ligue os trilhos da placa de ensaio e coloque o diodo MOSFET nele. Conecte o motor entre o trilho positivo e o dreno MOSFET. Conecte o diodo de proteção da mesma forma: entre o dreno do MOSFET e o trilho positivo. Conecte a fonte do MOSFET ao trilho de aterramento comum. Por fim, conecte o pino PWM (estamos usando o pino 10 neste exemplo) ao portão do MOSFET e também ao terra comum através do resistor (precisamos de corrente muito baixa!).

O código #

Agora podemos conectar o Arduino a um computador, fazer o upload do código e controlar o motor, enviando valores através da comunicação serial. Lembre-se de que esses valores devem ser inteiros entre 0 e 255. O código real deste exemplo é muito simples. Uma explicação é fornecida em cada linha.

int in = 0;                   // Variable to store the desired value
byte pinOut = 10;             // PWM output pin

void setup() {                // This executes once
  Serial.begin(9600);             // Initialize serial port
  pinMode(pinOut, OUTPUT);        // Prepare output pin
}

void loop() {                 // This loops continuously
  if(Serial.available()){         // Check if there's data
    in = Serial.read();           // Read said data into the variable "in"
    analogWrite(pinOut, in);      // Pass the value of "in" to the pin
  }
}

E é isso! Agora você pode usar os recursos PWM do Arduino para controlar aplicativos que exigem sinais analógicos mesmo quando os requisitos de energia excedem os limites da placa.

PWM com um TLC5940 #

O TLC5940 é um item útil quando você fica sem portas PWM no Arduino. Possui 16 canais, cada um controlável individualmente com 12 bits de resolução (0-4095). Uma biblioteca existente está disponível em http://playground.arduino.cc/Learning/TLC5940. É útil para controlar vários servos ou LEDs RGB. Basta ter em mente que os LEDs devem ser ânodo comum para funcionar. Além disso, os chips são encadeados em série, permitindo ainda mais portas PWM.

Exemplo:

// Include the library
#include <Tlc5940.h>

void setup() {
    // Initialize
    Tlc.init();
    Tlc.clear(); 
}

unsigned int level = 0;
void loop() {
    // Set all 16 outputs to same value
    for (int i = 0; i < 16; i++) {
        Tlc.set(i, level);
    }
    level = (level + 1) % 4096;
    // Tell the library to send the values to the chip
    Tlc.update();
    delay(10);
}
