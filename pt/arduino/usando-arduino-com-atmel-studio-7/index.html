Usando Arduino com Atmel Studio 7

  • Exemplo de esboço importado do Atmel Studio 7

Nesta página

  • Exemplo de esboço importado do Atmel Studio 7

Configurar #

  • Baixe e instale o Atmel Studio 7 de aqui.
  • Compre um depurador. Você pode se dar bem com um programador ISP, mas se quiser recursos de depuração, que é uma das grandes vantagens de usar o Atmel Studio, você precisará de um depurador. Eu recomendo o Atmel ICE, pois ele fornece recursos de depuração para arduinos baseados em AVR (como o Uno, pro mini, etc) e os Arduinos baseados em ARM, como o Zero e o Due. Se você estiver em um orçamento, você pode obtê-lo sem a caixa de plástico e tome cuidado para não chocá-lo.

Conexões #

  • Para o Uno, use o cabo ICSP de 6 pinos. Conecte um lado ao Uno conforme mostrado. Conecte o outro lado na porta AVR do depurador.

Para o Arduino Pro Mini, use o mini cabo squid conforme mostrado, novamente conectando do outro lado a porta AVR do depurador.

Considerações de depuração #

Para depurar com o Uno, você precisará cortar o rastreamento de habilitação de Reset (você sempre pode soldá-lo de volta para usar com o Arduino IDE):

Usando o Pro Mini, se você pretende conectar a porta serial ao seu computador usando uma placa FTDI, não conecte a linha DTR, pois isso interferirá na interface Serial Wire Debug (SWD) da Atmel. Eu simplesmente conecto energia, terra, Tx e Rx como mostrado aqui abaixo. Rx e Tx no Arduino vão para Tx e Rx, respectivamente na placa FTDI. Algumas placas FTDI são rotuladas de forma diferente, então se a porta serial não funcionar, troque Rx e Tx.

Você terá que fornecer energia separadamente ao Arduino porque o depurador não o alimentará. Isso pode ser feito no Pro Mini através da placa FTDI como mostrado acima, ou com um cabo USB ou adaptador AC no Uno.

Configuração do software #

Conecte o Atmel ICE ao seu computador, inicie o Atmel Studio e agora você pode importar um projeto Arduino existente.

No Atmel Studio, vá em File -> New -> Project e selecione “Create project from Arduino sketch”. Preencha as opções, incluindo menus suspensos de placa e dispositivo.

Vá para Project -> yourProjectName Properties, clique em Tool, selecione Atmel ICE em debugger/programmer e debugWire em interface. Vá para Depurar -> Iniciar depuração e interromper. Você deve ver um aviso e ser perguntado se deseja configurar o fusível DWEN. Escolha OK, desconecte o Arduino da energia e conecte-o novamente. Você pode interromper a depuração clicando no botão quadrado vermelho e começar clicando no botão triângulo verde. Para retornar o Arduino a um estado que possa ser usado no Arduino IDE, enquanto você estiver depurando, escolha Debug -> disable debugWIRE e feche.

Observe que todas as funções que você adicionar também devem incluir um protótipo de função (loop e configuração não precisam deles). Você pode ver as que o Atmel Studio adicionou na parte superior do esboço se houver alguma função quando importou seu projeto para o Atmel Studio (veja o código de exemplo, por exemplo).

O suporte a C++11 é habilitado por padrão no Arduino 1.6.6 e superior. Isso fornece mais recursos da linguagem C++ e ativá-la pode aumentar a compatibilidade com o sistema Arduinio. Para habilitar o C++11 no Atmel Studio 7, clique com o botão direito do mouse em seu arquivo de projeto, selecione propriedades, clique em ToolChain à esquerda, clique em Miscellaneous em AVR/GNU C++ Compiler e coloque ```-std=c++11` `` no campo Outros sinalizadores.

Para incluir bibliotecas em seu esboço #

Copie o arquivo de biblioteca .cpp em C:\Users\YourUserName\Documents\Atmel Studio\7.0\YourSolutionName\YourProjectName\ArduinoCore\src\core, então no Atmel Studio, abra a janela Solution Explorer clique com o botão direito do mouse em na pasta Arduino Core/src/core, escolha adicionar -> item existente e escolha o arquivo que você adicionou. Faça o mesmo com o arquivo de biblioteca .h e a pasta YourProjectName/Dependancies.

Para adicionar a janela do terminal #

Você sempre pode ter o IDE do Android aberto e usar essa janela Serial (basta selecionar a porta serial correta), no entanto, para adicionar uma janela Serial incorporada ao Atmel Studio, vá para Ferramentas -> Extensões e atualizações, clique em Downloads disponíveis e procure por Terminal Window ou Terminal para Atmel Studio e instale-o. Uma vez instalado, vá para View -> Terminal Window.

Benefícios #

Programar o Arduino com um IDE moderador como o Atmel Studio 7 oferece inúmeras vantagens sobre o IDE do Arduino, incluindo depuração, preenchimento automático, salto para definição e declaração, navegação para frente/para trás, marcadores e opções de refatoração, para citar alguns.

Você pode configurar as combinações de teclas indo em Ferramentas -> Opções -> Ambiente -> Teclado. Alguns que realmente aceleram o desenvolvimento são:

  • Edit.CommentSelection, Edit.UncommentSelection
  • View.NavigateForward, View.NavigateBackward
  • Edit.MoveSelectedLinesUp, Edit.MoveSelectedLinesDown
  • Edit.GoToDefinition

Exemplo de esboço importado do Atmel Studio 7 #

Este é um exemplo da aparência de um esboço simples do Arduino após ser importado para o Atmel Studio. O Atmel Studio adicionou as seções geradas automaticamente na parte superior. O resto é idêntico ao código original do Arduino. Se você expandir o projeto ArduinoCore que foi criado e procurar na pasta src -> core, você encontrará main.cpp, o ponto de entrada para o programa. Lá você pode ver a chamada para a função de configuração do Arduino e um loop for sem fim que chama a função de loop do Arduino repetidamente.

/* Begining of Auto generated code by Atmel studio */
#include <Arduino.h>
/* End of auto generated code by Atmel studio */


// Beginning of Auto generated function prototypes by Atmel Studio
void printA();
// End of Auto generated function prototypes by Atmel Studio

void setup() {
  Serial.begin(9600);
}

void loop() {
    printA();
}

void printA() {
    Serial.println("A");
}
