Apenas o mínimo #

Aqui está o esboço do Arduino ‘mínimo’. Isso pode ser carregado no Arduino IDE escolhendo File > Examples > 01. Basics > Bare Minimum.

void setup() {
  // put your setup code here, to run once
}

void loop() {
  // put your main code here, to run repeatedly
}

O código na função setup() será executado uma vez quando o programa for iniciado. Isso é útil para configurar pinos de E/S, inicializar variáveis, etc. O código na função loop() será executado repetidamente até que o Arduino seja desligado ou um novo programa seja carregado. Efetivamente, o código acima se parece com isso dentro da biblioteca de tempo de execução do Arduino:

setup();
while(1) {
  loop();
}

Ao contrário dos programas executados em seu computador, o código do Arduino nunca pode ser encerrado. Isso ocorre porque o microcontrolador tem apenas um programa carregado nele. Se este programa fosse encerrado, não haveria nada para dizer ao microcontrolador o que fazer.

Configuração inicial #

Software necessário: Arduino IDE

Configurar #

A maioria das placas compatíveis com Arduino tem uma porta USB e vem com um cabo USB. Conecte o Arduino usando o cabo USB e inicie o Arduino IDE.

A IDE do Arduino começará com um novo sketch, normalmente com funções vazias setup() e loop(). Isso é suficiente para fazer o upload para uma placa Arduino, mas não fará nada. O esboço de exemplo “Blink” funciona como um teste simples ao usar uma placa Arduino pela primeira vez. Vá para Arquivo → Exemplos → 01.Fundamentos → Piscar. Isso abrirá uma nova janela com o esboço Blink.

Selecione sua placa. Vá para Ferramentas → Conselho → [nome da sua placa Arduino].

Selecione a porta COM da sua placa. A maioria das placas compatíveis com Aurduino criará uma porta COM falsa, que é usada para comunicação serial (depuração) e para programação da placa. COM 1 geralmente já está presente, e seu quadro criará um novo, por exemplo COM 4. Selecione em Ferramentas → Porta → COM 4 (ou outro número COM).

Algumas placas possuem configurações adicionais no menu Ferramentas, como velocidade do clock. Eles variam de placa para placa, mas geralmente um conjunto aceitável de padrões já está selecionado.

Envio #

Agora você está pronto para fazer o upload do Blink. Clique no botão Carregar ou selecione Esboço → Envio. O esboço será compilado e, em seguida, enviado para sua placa Arduino. Se tudo funcionou, o LED on-board começará a piscar a cada segundo.

#Monitor de série

No Arduino IDE ypu hava um monitor serial. Para abri-lo use o botão serial monitor no lado direito da janela.

Certifique-se de que o código foi carregado antes de abrir o monitor. O upload e o monitor não serão executados ao mesmo tempo!

Piscar #

Aqui está um pequeno exemplo que demonstra as funções setup() e loop(). Isso pode ser carregado no Arduino IDE escolhendo File > Examples > 01. Basics > Blink. (Nota: A maioria das placas Arduino tem um LED já conectado ao pino 13, mas pode ser necessário adicionar um LED externo para ver os efeitos deste esboço.)

// the setup function runs once when you press reset or power the board
void setup() {
  // initialize digital pin 13 as an output.
  pinMode(13, OUTPUT);
}

// the loop function runs over and over again forever
void loop() {
  digitalWrite(13, HIGH);   // turn the LED on (HIGH is the voltage level)
  delay(1000);              // wait for a second
  digitalWrite(13, LOW);    // turn the LED off by making the voltage LOW
  delay(1000);              // wait for a second
}

O trecho acima:

  1. Define a função setup(). A função setup() é chamada primeiro na execução em cada programa Arduino.

    1. Sets pin 13 as an output.

      Without this, it might be set to an input, which would make the LED not work; however once it is set as an output it will stay that way so this only needs to be done once when the program starts.

  2. Define a função loop(). A função loop() é chamada repetidamente enquanto o programa estiver em execução.

    1. digitalWrite(13, HIGH); turns the LED on.
    2. delay(1000); waits one second (1000 milliseconds).
    3. digitalWrite(13, LOW); turns the LED off.
    4. delay(1000); waits one second (1000 milliseconds).

Como o loop() é executado repetidamente enquanto o programa estiver em execução, o LED piscará em um período de 2 segundos (1 segundo ligado, 1 segundo desligado). Este exemplo é baseado no Arduino Uno e em qualquer outra placa que já tenha um LED conectado ao pino 13. Se a placa que está sendo usada não tiver um LED on-board conectado a esse pino, pode-se conectar externamente.

Mais sobre tempo (por exemplo, atrasos e medição de tempo): https://www.wikiod.com/pt/arduino/gerenciamento-de-tempo

LED - Com controle de botão #

Você também pode usar este código para configurar um LED com um interruptor de botão com um resistor pull-up, isso pode ser preferencialmente com a próxima etapa após a configuração do controlador de LED inicial

int buttonState = 0; // variable for reading the pushbutton status

void setup()
{
    // initialize the LED pin as an output:
    pinMode(13, OUTPUT); // You can set it just using its number
    // initialize the pushbutton pin as an input:
    pinMode(2, INPUT);
}

void loop()
{
    // read the state of the pushbutton value:
    buttonState = DigitalRead(2);

    // check if the pushbutton is pressed.
    // If it's not, the buttonState is HIGH : if (buttonState == HIGH)
    {
        // turn LED off:
        digitalWrite(13, LOW);
    }
    else
    {
        // turn LED off:
        digitalWrite(13, HIGH);
    }
}
