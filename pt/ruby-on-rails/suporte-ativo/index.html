ActiveSupport é uma joia utilitária de ferramentas de uso geral usadas pelo resto do framework Rails.

Uma das principais maneiras de fornecer essas ferramentas é fazendo o monkeypatch dos tipos nativos do Ruby. Elas são chamadas de Extensões principais.

Extensões principais: acesso à string #

String#at #

Retorna uma substring de um objeto string. Mesma interface que String#[].

str = "hello"
str.at(0)      # => "h"
str.at(1..3)   # => "ell"
str.at(-2)     # => "l"
str.at(-2..-1) # => "lo"
str.at(5)      # => nil
str.at(5..-1)  # => ""

[String#from][from] #

Retorna uma substring da posição especificada até o final da string.

str = "hello"
str.from(0)  # => "hello"
str.from(3)  # => "lo"
str.from(-2) # => "lo"

[String#to][to] #

Retorna uma substring do início da string até a posição especificada. Se a posição for negativa, é contada a partir do final da string.

str = "hello"
str.to(0)  # => "h"
str.to(3)  # => "hell"
str.to(-2) # => "hell"

from e to podem ser usados ​​em conjunto.

str = "hello"
str.from(0).to(-1) # => "hello"
str.from(1).to(-2) # => "ell"

[String#first][first] #

Retorna o primeiro caractere ou um determinado número de caracteres até o comprimento da string.

str = "hello"
str.first    # => "h"
str.first(1) # => "h"
str.first(2) # => "he"
str.first(0) # => ""
str.first(6) # => "hello"

[String#last][last] #

Retorna o último caractere ou um determinado número de caracteres do final da string contando para trás.

str = "hello"
str.last    # => "o"
str.last(1) # => "o"
str.last(2) # => "lo"
str.last(0) # => ""
str.last(6) # => "hello"

Extensões principais: conversão de string para data/hora #

String#to_time #

Converte uma string em um valor Time. O parâmetro form pode ser :utc ou :local, o padrão é :local.

"13-12-2012".to_time               # => 2012-12-13 00:00:00 +0100
"06:12".to_time                    # => 2012-12-13 06:12:00 +0100
"2012-12-13 06:12".to_time         # => 2012-12-13 06:12:00 +0100
"2012-12-13T06:12".to_time         # => 2012-12-13 06:12:00 +0100
"2012-12-13T06:12".to_time(:utc)   # => 2012-12-13 06:12:00 UTC
"12/13/2012".to_time               # => ArgumentError: argument out of range

String#to_date #

Converte uma string em um valor de Data.

"1-1-2012".to_date   # => Sun, 01 Jan 2012
"01/01/2012".to_date # => Sun, 01 Jan 2012
"2012-12-13".to_date # => Thu, 13 Dec 2012
"12/13/2012".to_date # => ArgumentError: invalid date

String#to_datetime #

Converte uma string em um valor DateTime.

"1-1-2012".to_datetime            # => Sun, 01 Jan 2012 00:00:00 +0000
"01/01/2012 23:59:59".to_datetime # => Sun, 01 Jan 2012 23:59:59 +0000
"2012-12-13 12:50".to_datetime    # => Thu, 13 Dec 2012 12:50:00 +0000
"12/13/2012".to_datetime          # => ArgumentError: invalid date

Extensões principais: exclusão de string #

[String#exclude?][exclude] #

O inverso de String#include?

"hello".exclude? "lo" # => false
"hello".exclude? "ol" # => true
"hello".exclude? ?h   # => false

Extensões principais: filtros de string #

String#squish #

Retorna uma versão da string fornecida sem espaços em branco à esquerda ou à direita e combina todos os espaços em branco consecutivos no interior com espaços simples. A versão destrutiva squish! opera diretamente na instância da string.

Lida com espaços em branco ASCII e Unicode.

%{ Multi-line
   string }.squish                   # => "Multi-line string"
" foo   bar    \n   \t   boo".squish # => "foo bar boo"

[String#remove][remove] #

Retorna uma nova string com todas as ocorrências dos padrões removidas. A versão destrutiva remove! opera diretamente na string fornecida.

str = "foo bar test"
str.remove(" test")                 # => "foo bar"
str.remove(" test", /bar/)          # => "foo "

String#truncar #

Retorna uma cópia de uma determinada string truncada em um determinado comprimento se a string for maior que o comprimento.

'Once upon a time in a world far far away'.truncate(27)
# => "Once upon a time in a wo..."

Passe uma string ou regexp :separator para truncar em uma quebra natural

'Once upon a time in a world far far away'.truncate(27, separator: ' ')
# => "Once upon a time in a..."

'Once upon a time in a world far far away'.truncate(27, separator: /\s/)
# => "Once upon a time in a..."

[String#truncate_words][words] #

Retorna uma string truncada após um determinado número de palavras.

'Once upon a time in a world far far away'.truncate_words(4)
# => "Once upon a time..."

Passe uma string ou regexp para especificar um separador de palavras diferente

'Once<br>upon<br>a<br>time<br>in<br>a<br>world'.truncate_words(5, separator: '<br>')
# => "Once<br>upon<br>a<br>time<br>in..."

Os últimos caracteres serão substituídos pela string :omission (o padrão é “…”)

'And they found that many people were sleeping better.'.truncate_words(5, omission: '... (continued)')
# => "And they found that many... (continued)"

String#strip_heredoc #

Recuo de tiras em heredocs. Procura a linha não vazia menos recuada e remove essa quantidade de espaço em branco à esquerda.

if options[:usage]
  puts <<-USAGE.strip_heredoc
    This command does such and such.

    Supported options are:
      -h         This message
      ...
  USAGE
end

o usuário veria

This command does such and such.

Supported options are:
-h         This message
...

Extensões principais: flexão de string #

String#pluralize #

Retorna da forma plural da string. Opcionalmente, recebe um parâmetro count e retorna a forma singular se count == 1. Também aceita um parâmetro locale para pluralização de idioma específico.

'post'.pluralize             # => "posts"
'octopus'.pluralize          # => "octopi"
'sheep'.pluralize            # => "sheep"
'words'.pluralize            # => "words"
'the blue mailman'.pluralize # => "the blue mailmen"
'CamelOctopus'.pluralize     # => "CamelOctopi"
'apple'.pluralize(1)         # => "apple"
'apple'.pluralize(2)         # => "apples"
'ley'.pluralize(:es)         # => "leyes"
'ley'.pluralize(1, :es)      # => "ley"

String#singularize #

Retorna a forma singular da string. Aceita um parâmetro opcional locale.

'posts'.singularize            # => "post"
'octopi'.singularize           # => "octopus"
'sheep'.singularize            # => "sheep"
'word'.singularize             # => "word"
'the blue mailmen'.singularize # => "the blue mailman"
'CamelOctopi'.singularize      # => "CamelOctopus"
'leyes'.singularize(:es)       # => "ley"

[String#constantize][constantize] #

Tenta encontrar uma constante declarada com o nome especificado na string. Ele gera um NameError quando o nome não está no CamelCase ou não é inicializado.

'Module'.constantize  # => Module
'Class'.constantize   # => Class
'blargle'.constantize # => NameError: wrong constant name blargle

[String#safe_constantize][safe] #

Executa um constantize mas retorna nil em vez de gerar NameError.

'Module'.safe_constantize  # => Module
'Class'.safe_constantize   # => Class
'blargle'.safe_constantize # => nil

String#camelize #

Converte strings para UpperCamelCase por padrão, se :lower for fornecido como parâmetro converte para lowerCamelCase.

alias: camelcase

Observação: também converterá / para ::, o que é útil para converter caminhos em namespaces.

'active_record'.camelize                # => "ActiveRecord"
'active_record'.camelize(:lower)        # => "activeRecord"
'active_record/errors'.camelize         # => "ActiveRecord::Errors"
'active_record/errors'.camelize(:lower) # => "activeRecord::Errors"

String#titleize #

Coloca todas as palavras em maiúsculas e substitui alguns caracteres na string para criar um título mais bonito.

alias: titlecase

'man from the boondocks'.titleize # => "Man From The Boondocks"
'x-men: the last stand'.titleize  # => "X Men: The Last Stand"

String#sublinhado #

Cria uma forma sublinhada e minúscula a partir da expressão na string. O inverso de camelize.

Observação: underscore também mudará :: para / para converter namespaces em caminhos.

'ActiveModel'.underscore         # => "active_model"
'ActiveModel::Errors'.underscore # => "active_model/errors"

String#dasherize #

Substitui sublinhados por traços na string.

'puni_puni'.dasherize # => "puni-puni"

String#demodulize #

Remove a parte do módulo da expressão constante na string.

'ActiveRecord::CoreExtensions::String::Inflections'.demodulize # => "Inflections"
'Inflections'.demodulize                                       # => "Inflections"
'::Inflections'.demodulize                                     # => "Inflections"
''.demodulize                                                  # => ''

String#desconstantize #

Remove o segmento mais à direita da expressão constante na string.

'Net::HTTP'.deconstantize   # => "Net"
'::Net::HTTP'.deconstantize # => "::Net"
'String'.deconstantize      # => ""
'::String'.deconstantize    # => ""
''.deconstantize            # => ""

String#parameterize #

Substitui caracteres especiais em uma string para que ela possa ser usada como parte de uma URL ‘bonita’.

"Donald E. Knuth".parameterize # => "donald-e-knuth"

Preserve o caso dos caracteres em uma string com o argumento :preserve_case.

"Donald E. Knuth".parameterize(preserve_case: true) # => "Donald-E-Knuth"

Um caso de uso muito comum para parameterize é substituir o método to_param de um modelo ActiveRecord para suportar slugs de URL mais descritivos.

class Person < ActiveRecord::Base
  def to_param
    "#{id}-#{name.parameterize}"
  end
end

Person.find(1).to_param # => "1-donald-e-knuth"

String#tableize #

Cria o nome de uma tabela como o Rails faz para modelos para nomes de tabelas. Pluraliza a última palavra na string.

'RawScaledScorer'.tableize # => "raw_scaled_scorers"
'ham_and_egg'.tableize     # => "ham_and_eggs"
'fancyCategory'.tableize   # => "fancy_categories"

String#classificar #

Retorna uma string de nome de classe de um nome de tabela plural como o Rails faz para nomes de tabelas para modelos.

'ham_and_eggs'.classify # => "HamAndEgg"
'posts'.classify        # => "Post"

[String#humanize][humanize] #

Capitaliza a primeira palavra, transforma sublinhados em espaços e remove um _id final, se presente.

'employee_salary'.humanize              # => "Employee salary"
'author_id'.humanize                    # => "Author"
'author_id'.humanize(capitalize: false) # => "author"
'_id'.humanize                          # => "Id"

String#upcase_first #

Converte apenas o primeiro caractere para maiúsculo.

'what a Lovely Day'.upcase_first # => "What a Lovely Day"
'w'.upcase_first                 # => "W"
''.upcase_first                  # => ""

[String#chave_estrangeira][chave_estrangeira] #

Cria um nome de chave estrangeira a partir de um nome de classe. Passe o parâmetro false para desabilitar a adição de _ entre name e id.

'Message'.foreign_key        # => "message_id"
'Message'.foreign_key(false) # => "messageid"
'Admin::Post'.foreign_key    # => "post_id"
