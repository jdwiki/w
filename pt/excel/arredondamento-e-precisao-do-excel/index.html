Arredondamento e precisão do Excel

Várias fórmulas do Excel lidam com arredondamento e precisão de números não inteiros. Isso é diferente do uso de formatação de célula que afeta a exibição de dados numéricos. Em alguns casos, apenas a formatação de células é suficiente, mas em cálculos complexos, são necessárias regras rígidas de arredondamento e precisão para obter resultados consistentes e corretos.

Sintaxe #

  • =ARRED(número, num_dígitos)
  • =ROUNDUP(número, num_dígitos)
  • =ARREDONDO(número, num_dígitos)
  • =MROUND(número, múltiplo)
  • =TRUNC(número, [num_dígitos])
  • =INT(número)
  • =TETO(número, significância)
  • =PISO(número, significância)
  • =PAR(número)
  • =ODD(número)
  • =CORRIGIDO(número, [decimais], [sem vírgulas])

Parâmetros #

Parâmetros Detalhes
número número a ser arredondado. Pode ser uma célula como B2 ou uma constante como 3.14159
num_dígitos qual a casa a ser arredondada 2. Omitido ou 0 significa arredondar para um número inteiro. 1 ou 2 significa arredondar para décimos ou centésimos. -1 ou -3 significa arredondar para dezenas ou milhares.
múltiplo O múltiplo para o qual você deseja arredondar o número.
significado O múltiplo para o qual você deseja arredondar o número.
decimais O número de dígitos à direita do ponto decimal. (Opcional - o padrão é 2)
no_commas Um valor lógico que, se TRUE, impede que FIXED inclua vírgulas no texto retornado. (Opcional - o padrão é FALSE)
[ … ] Os parâmetros em [colchetes] são opcionais.

Os valores exibidos ao usuário podem ser apresentados com formatação específica que não afeta os valores reais dos dados. Por exemplo, os dados exibidos podem ser formatados como uma porcentagem. Consulte https://www.wikiod.com/pt/excel/formatacao-da-celula para obter detalhes.

Usando as funções TETO E PISO #

A função CEILING arredonda um número para cima, a partir de zero, para o múltiplo de significância mais próximo. A função FLOOR faz o mesmo arredondando o número para zero.

Um exemplo de quando CEILING pode ser usado é se você quiser evitar o uso de centavos em seus preços e seu produto custa $ 4,42, use a fórmula =CEILING(4.42,0.05) para arredondar os preços para o níquel mais próximo .

Por exemplo:

=CEILING(2.2, 1)
=FLOOR(2.2, 1)
=CEILING(-4.8, 2)
=FLOOR(-4.8, 2)
=CEILING(0.456, 0.01)
=FLOOR(0.456, 0.01)

Retornaria:

3
2
-4
-6
0.46
0.45

Usando a função ROUND #

A função ROUND arredonda um valor. O número de casas decimais para arredondar é especificado por um valor positivo no parâmetro num_digits. Um valor negativo para o num_digits arredondará a parte inteira do valor à esquerda do ponto decimal, por exemplo, para os 10 mais próximos (para -1) ou para os 1000 mais próximos (para -3).

Aqui está uma tabela mostrando como a rodada pode ser usada.

Começando com ROUND(b,2) ARRED(b,1) RODADA(b) ARRED(b,-1)
23.10651375 23.11 23.1 23 20
19.16818924 19,17 19.2 19 20
3.92748883 3,93 3.9 4 0
31.38208409 31,38 31,4 31 30
38.34235561 38,34 38,3 38 40
7.682632495 7,68 7,7 8 10
35.39315416 35,39 35,4 35 40
20.47004449 20.47 20,5 20 20
20.49775276 20,5 20,5 20 20
2.288822497 2,29 2.3 2 0

Funções similares adicionais também estão disponíveis para controlar a direção do arredondamento:

  • ROUNDUP - Sempre arredonda um número para cima, longe de zero.
  • ROUNDDOWN - Sempre arredonda um número para baixo, até zero.

Usando as funções TRUNC & INT #

A fórmula do Excel TRUNC é usada para truncar um número para um determinado número de casas decimais, especificado pelo parâmetro opcional num_digits. Se este parâmetro for definido como um valor negativo, ele truncará a parte inteira do valor. Se o parâmetro for omitido, o valor padrão será ‘0’, que remove a parte decimal do número.

A função INT funciona de forma semelhante ao TRUNC, pois remove a parte decimal de um número arredondando-o para baixo para deixar a parte inteira. A diferença entre os dois é ao realizar a operação em um número negativo; TRUNC removerá o decimal, porém INT arredondará o valor para baixo a partir de zero.

Por exemplo:

=TRUNC(123.456,2)
=TRUNC(123.4357,-1)
=TRUNC(-123.123)
=INT(567.89)
=INT(-567.89)

Exibirá:

123.45
120.00
-123.00
567.00
-568.00

Usando a função MROUND #

A função do Excel MROUND é usada para arredondar um número para um intervalo diferente de uma potência de 10.

Esses exemplos mostram MROUND para o quarto mais próximo e para o número par mais próximo.

Começando com MROUND(b,0,25) MROUND(b,2)
23.93195211 24h00 24
2.793135388 2,75 2
21.93903064 22h00 22
13.74193739 13,75 14
16.77047412 16,75 16
13.03922302 13h00 14
17.06132896 17h00 18
16.11741694 16h00 16
33.48249592 33,50 34
37.29656687 37,25 38

Um resultado semelhante pode ser obtido usando as funções EVEN e ODD que arredondam um número para cima para o número par ou ímpar mais próximo, respectivamente.

Usando a função FIXED #

A função FIXED arredonda um número para o número especificado de decimais definido pelo parâmetro decimals, formata o número em formato decimal usando uma vírgula como separador, a menos que especificado como não obrigatório definido pelo parâmetro no_commas, e retorna o resultado como texto. O parâmetro decimals é opcional e tem como padrão duas casas decimais. O parâmetro no_commas também é opcional e o padrão é FALSE.

Por exemplo:

=FIXED(1234.567, 1)
=FIXED(1234.567, -1)
=FIXED(1234.567, 1, TRUE)
=FIXED(1234.567)

Retornaria:

1,234.6
1,230
1234.6
1234.57
