Limites máximos de armazenamento de memória e tamanho de arquivo para pastas de trabalho do Modelo de Dados ambiente de 32 bits está sujeito a 2 gigabytes (GB) de espaço de endereço virtual, compartilhado pelo Excel, a pasta de trabalho e suplementos executados no mesmo processo. O compartilhamento do espaço de endereço de um modelo de dados pode ser de 500 a 700 megabytes (MB), mas pode ser menor se outros modelos de dados e suplementos forem carregados. O ambiente de 64 bits não impõe limites rígidos ao tamanho do arquivo. O tamanho da pasta de trabalho é limitado apenas pela memória disponível e pelos recursos do sistema. O Excel 2016 oferece a funcionalidade de reconhecimento de endereço grande que permite que o Excel 2016 de 32 bits consuma o dobro da memória quando os usuários trabalham em um sistema operacional Windows de 64 bits. Para obter mais informações, consulte Alteração do recurso de reconhecimento de endereço grande para Excel. Observação: Adicionar tabelas ao modelo de dados aumenta o tamanho do arquivo. Se você não planeja criar relacionamentos complexos de Modelo de Dados usando muitas fontes de dados e tipos de dados em sua pasta de trabalho, desmarque a caixa Adicionar esses dados ao Modelo de Dados ao importar ou criar tabelas, tabelas dinâmicas ou conexões de dados.

Pastas de trabalho com a configuração “Permitir alterações por mais de um usuário…” habilitada Se a configuração Permitir alterações por mais de um usuário… estiver ativada para uma pasta de trabalho, as informações a seguir se aplicarão. Essa configuração pode ser acessada clicando na guia Revisão > Compartilhar pasta de trabalho. Observe que nas versões mais recentes do Excel, o botão Compartilhar pasta de trabalho foi ocultado. Para reexibir, clique em Arquivo > Opções > Barra de Ferramentas de Acesso Rápido. Abra a lista em Escolher comandos de e selecione Todos os comandos. Role para baixo nessa lista até ver Compartilhar pasta de trabalho (legado). Selecione esse item e clique em Adicionar. Clique OK. O botão Compartilhar pasta de trabalho agora está na parte superior da Janela do Excel e se parece com isso: Compartilhar pasta de trabalho
