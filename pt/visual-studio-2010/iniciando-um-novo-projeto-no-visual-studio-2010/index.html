Iniciando um novo projeto no Visual Studio 2010 #

Para iniciar um novo projeto no Visual Studio 2010:

  • Clique no menu Arquivo
  • Selecionar projeto

Em seguida, temos os tipos de projeto e os idiomas disponíveis listados:

  • Selecione Idiomas disponíveis no painel mais à esquerda [Abaixo temos idiomas como Visual C#, VB, F# etc]
  • Uma vez que selecionamos os idiomas dos modelos instalados, temos os tipos de projeto/aplicativo listados.
  • Selecione os tipos de projeto, dê um nome ao seu projeto e clique em ok.

Alguns exemplos comuns de tipos de projeto:

Aplicativos do Windows Forms:

Projeto para criação de aplicativo com interface de usuário do Windows Forms.

Aplicativos WPF:

Aplicativo cliente do Windows Presentation Foundation.

Aplicativos de console:

Aplicativos de linha de comando. Sem interface gráfica do usuário (GUI).

ASP.NET:

Aplicativos de interface de usuário da Web, ou seja. Aplicativos da web

Aplicações Silverlight:

Aplicativos avançados de internet usando Silverlight

Abrindo um projeto existente #

  • Abra o Visual Studio e selecione Menu Arquivo -> Abrir -> Projeto

  • Em seguida, selecione o arquivo de solução na pasta do projeto

  • Os arquivos do projeto serão mostrados no Solution Explorer.

Microsoft Visual Studion 2010 Microsoft® Visual C#® Associações de teclas padrão #

Comandos de arquivo

  • Arquivo.NovoProjeto:

CTRL + SHIFT + N

Exibe a caixa de diálogo Novo Projeto.

  • Arquivo.OpenProject:

CTRL + SHIFT + O

Exibe a caixa de diálogo Abrir Projeto, onde os projetos existentes podem ser adicionado à solução.

  • Projeto.AddClass:

SHIFT + ALT + C

Exibe a caixa de diálogo Adicionar novo item e seleciona o modelo de classe como padrão.

  • Projeto.AddExistingItem:

SHIFT +ALT + A

Exibe a caixa de diálogo Adicionar item existente, onde os arquivos existentes podem ser adicionado ao projeto atual.

  • Projeto.AdicionarNovoItem:

CTRL + SHIFT + A

Exibe a caixa de diálogo Adicionar novo item, onde um novo arquivo pode ser adicionado o projeto atual.

Editando

  • Edit.HideSelection:

CTRL + M , CTRL + H

Recolhe a seleção atual para uma região de contorno.

  • Editar.ToggleAllOutlining:

CTRL + M , L

Alterna todas as regiões de contorno recolhidas anteriormente entre recolhidas e estados expandidos.

  • Edit.ToggleOutliningExpansion:

CTRL + M , M

Alterna a região recolhida atualmente selecionada entre as e estado expandido.

  • Editar.Parar delineamento:

CTRL + M , P

Remove todas as informações de contorno de todo o documento.

  • Editar.CommentSelection:

CTRL + K , C ou CTRL + E , C

Insere // no início da linha atual ou em cada linha do seleção atual.

Editar.DescomentarSeleção CTRL + K, U ou CTRL + E, U Remove o // no início de a linha atual ou cada linha do seleção atual.

  • Editar.FormatarDocumento:

CTRL + K, D ou CTRL + E, D

Formata o documento atual de acordo com o recuo e o código configurações de formatação especificadas no painel Formatação em: Ferramentas | Opções | Editor de Texto | C#.

  • Editar.FormatSelection:

CTRL + K,F ou CTRL + E, F

Formata a seleção atual de acordo com o recuo e o código configurações de formatação especificadas no painel Formatação em Ferramentas | Opções | Editor de Texto | C#.

  • Editar.Inserir Trecho:

CTRL + K, X

Exibe o Seletor de trecho de código. O trecho de código selecionado será inserido na posição do cursor.

Edit.SurroundWith:

CTRL + K, S

Exibe o Seletor de Fragmentos de Código. O snippet de código selecionado será enrolado em torno do texto selecionado.

  • Edit.InvokeSnippetFromShortcut:

TAB

Insere o trecho de código expandido do nome do atalho.

  • Edit.CycleClipboardRing:

CTRL +SHIFT +V

Cola o texto do anel da área de transferência no local do cursor no arquivo. O uso subsequente da tecla de atalho itera pelos itens no Anel da área de transferência.

  • Editar.Substituir:

CTRL + H

Exibe as opções de substituição na guia Substituição rápida da tela Localizar e Caixa de diálogo Substituir.

  • Editar.ReplaceInFiles:

CTRL + SHIFT + H

Exibe as opções de substituição na guia Substituir em Arquivos do Localizar e caixa de diálogo Substituir.

Comandos da janela:

  • Visualizar.ClassView:

CTRL + W, C

Exibe a janela Visualização da turma.

  • View.CodeDefinitionWindow:

CTRL + W, D

Exibe a janela de definição de código.

  • Visualizar.ErrorList:

CTRL + W, E

Exibe a janela Lista de Erros.

  • Visualizar.ObjectBrowser:

CTRL + W, J

Exibe o Pesquisador de objetos.

  • Visualizar.Saída:

CTRL + W, O

Exibe a janela Saída, onde as mensagens de status podem ser visualizadas em tempo de execução.

  • Visualizar.Janela de Propriedades:

CTRL + W, P

Exibe a janela Propriedades, que lista as propriedades de tempo de design e eventos para o item atualmente selecionado.

  • Visualizar.SolutionExplorer:

CTRL + W, S

Exibe o Solution Explorer, que lista os projetos e arquivos no solução atual.

  • Visualizar.Lista de Tarefas:

CTRL + W, T

Exibe a janela Lista de Tarefas, que exibe tarefas personalizadas, comentários, atalhos, avisos e mensagens de erro.

  • Visualizar.Caixa de ferramentas:

CTRL + W, X

Exibe a caixa de ferramentas, que contém controles que podem ser incluídos ou usado com seu código.

  • Visualizar.ServerExplorer:

CTRL + W, L

Exibe o Server Explorer, que permite visualizar e manipular o banco de dados servidores, logs de eventos, filas de mensagens, serviços da Web e outros serviços do sistema.

  • Janela.FecharJanela:

SHIFT + ESC

Fecha a janela de ferramentas atual.

  • Dados.ShowDataSources:

SHIFT + ALT + D

Exibe a janela Fontes de Dados.

  • Janela.FecharDocumentoJanela:

CTRL + F4

Fecha a aba atual

Comando de depuração:

  • Depuração.Automóveis

CTRL+D, A

Exibe a janela Autos, que exibe as variáveis ​​usadas no linha de código atual e a linha de código anterior.

  • Depuração.CallStack:

CTRL+D,C

Exibe a janela Call Stack, que exibe uma lista de todos os ativos métodos ou quadros de pilha para o thread de execução atual.

  • Depuração.Imediata:

CTRL+D, I

Exibe a janela Imediata, onde as expressões podem ser avaliadas.

  • Depuração.Locals:

CTRL+D, L

Exibe a janela Locais, que exibe as variáveis ​​locais e seus valores para cada método no quadro de pilha atual.

  • Depuração.QuickWatch:

CTRL+D, Q

Exibe a caixa de diálogo QuickWatch que contém o valor atual do expressão selecionada.

  • Depurar.Iniciar:

F5

Inicia o aplicativo no depurador com base nas configurações do projeto de inicialização. Quando no modo Break, invocando este comando executará o aplicativo até o próximo ponto de interrupção.

  • Depuração.Iniciar semDepuração:

CTRL+F5

Inicia o aplicativo sem chamar o depurador.

  • Debug.StepInto:

F11

Executa o código uma instrução por vez, seguindo a execução em chamadas de métodos.

  • Debug.StepIntoSpecic:

SHIFT+ALT+F11

Pede para selecionar em qual subexpressão entrar, independentemente a configuração de opções de ferramentas para pular propriedades e operadores.

  • Depuração.StepOut:

SHIFT+F11

Executa as linhas restantes de um método no qual o o ponto de execução está localizado.

  • Depuração.Passar Acima:

F10

Executa a próxima linha de código, mas não segue a execução qualquer chamada de método.

  • Depuração.Parar depuração:

SHIFT+F5

Interrompe a execução do aplicativo atual no depurador.

  • Debug.ToggleBreakpoint:

F9

Define ou remove um ponto de interrupção na linha atual.

  • Depurar.Assistir:

CTRL+D, W

Exibe a janela Watch, que exibe os valores dos variáveis ​​ou expressões de observação.

  • Depuração.Ativar ponto de interrupção:

CTRL+F9

Alterna o ponto de interrupção entre desabilitado e habilitado.

  • Teste.RunAllTestsInSolution:

CTRL+R, A

Executa todos os testes de unidade na solução.

Comandos de construção:

  • Construir.BuildSolution:

F6 ou CTRL+SHIFT+B

Constrói todos os projetos na solução.

  • Construir.BuildSelection:

SHIFT+F6

Compila o projeto selecionado e suas dependências.
