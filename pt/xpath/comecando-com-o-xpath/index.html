Testando Xpaths no console do navegador #

Uma maneira rápida de testar seu xpath é no console da ferramenta de desenvolvedor do navegador.

O formato é

   $x('//insert xpath here')

$ - especifica que é um seletor.

x - especifica que está usando xpaths

Exemplo:

$x("//button[text() ='Submit']")

Quando este comando for digitado ele retornará todas as ocorrências de elementos que são botões com texto igual a Enviar.

Selecionar texto #

Para o XML de amostra (sem namespaces):

Este XPath,

/r/f/text()

irá selecionar o nó de texto com este valor de string:

"Text 1"

E este XPath,

string(/r/f)

retornará o valor da string de f, que também é:

"Text 1"

Selecione um elemento #

Para o XML de amostra (sem namespaces):

Este XPath,

/r/e

irá selecionar este elemento:

<e a="1"/>

Operações HTML comuns #

Se o HTML DOM de entrada for

<html>
    <body>
        <a>link</a>
        <div class='container' id='divone'>
            <p class='common' id='enclosedone'>Element One</p>
            <p class='common' id='enclosedtwo'>Element Two</p>
        </div>
    </body>
</html>

Encontre um elemento com um id específico em toda a página

//*[@id='divone'] # Returns <div class='container' id='divone'>

Encontre um elemento com um id específico em um caminho específico

/html/body/div/p[@id='enclosedone'] # Returns <p class='common' id='enclosedone'>Element One</p>

Selecione um elemento com um determinado id e classe

//p[@id='enclosedone' and @class='common'] # Returns <p class='common' id='enclosedone'>Element One</p>

Selecione o texto de um elemento específico

//*[@id='enclosedone']/text() # Returns Element One

XML de exemplo (sem namespaces) #

Aqui está um exemplo de XML no qual XPaths de exemplo podem ser escritos:

<r>
  <e a="1"/>
  <f a="2" b="1">Text 1</f>
  <f/>
  <g>
    <i c="2">Text 2</i>
    Text 3
    <j>Text 4</j>
  </g>
</r>
