Sintaxe #

  1. Todos os ancestrais de um nó
    • /path to the node/ancestor::node()
  2. Um ancestral específico de um nó
    • /path to the node/ancestor::ancestor_name
  3. Pai de um nó
    • /path to the node/parent::node()
  4. Seguindo irmãos de um nó
    • /path to the node/following-sibling::node()
  5. Um irmão específico seguindo um nó
    • /path to the node/following-sibling::sibling_name
  6. Irmãos precedentes de um nó
    • /path to the node/preceding-sibling::node()
  7. Um irmão específico precedendo um nó
    • /path to the node/preceding-sibling::sibling_name
  8. Todos os nós filhos imediatos de um nó
    • /path to the node/child::node()
  9. Um nó filho imediato específico de um nó
    • /path to the node/child::chid_name
  10. Todos os descendentes de um nó
    • /path to the node/descendant::node()
  11. Todos os descendentes específicos de um nó
    • /path the to node/descendant::descendant_name

Parâmetros #

Eixo seleciona
ancestral todos os nós ancestrais
pai nó pai
seguinte-irmão irmãos seguindo o nó
irmão-antecessor irmãos que precedem o nó
criança crianças imediatas
descendente todos os descendentes, independentemente do nível de aninhamento

Esses eixos podem ser usados ​​em combinação com outras funções para atender às nossas necessidades.

Encontrar meus ancestrais #

XML

<GrandFather name="Bardock" gender="male" spouse="Gine">
    <Dad name="Goku" gender="male" spouse="Chi Chi">
        <Me name="Gohan" gender="male"/>
        <brother name="Goten" gender="male"/>
    </Dad>
</GrandFather>

XPATH

//Me/ancestor::node()

RESULTADO

<GrandFather name="Bardock" gender="male" spouse="Gine">
  <Dad name="Goku" gender="male" spouse="Chi Chi">
    <Me name="Gohan" gender="male" />
    <brother name="Goten" gender="male" />
  </Dad>
</GrandFather>
<Dad name="Goku" gender="male" spouse="Chi Chi">
  <Me name="Gohan" gender="male" />
  <brother name="Goten" gender="male" />
</Dad>

Encontrar meu pai #

XML

<GrandFather name="Bardock" gender="male" spouse="Gine">
    <Dad name="Goku" gender="male" spouse="Chi Chi">
        <Me name="Gohan" gender="male"/>
        <brother name="Goten" gender="male"/>
    </Dad>
</GrandFather>

XPATH

//Me/ancestor::Dad

ou

//Me/parent::node()

RESULTADO

<Dad name="Goku" gender="male" spouse="Chi Chi">
  <Me name="Gohan" gender="male" />
  <brother name="Goten" gender="male" />
</Dad>

Encontre meu avô #

XML

<GrandFather name="Bardock" gender="male" spouse="Gine">
  <Dad name="Goku" gender="male" spouse="Chi Chi">
    <Me name="Gohan" gender="male" />
    <brother name="Goten" gender="male" />
  </Dad>
</GrandFather>

XPATH

//Me/ancestor::GrandFather

ou

//Me/parent::node()/parent::node()

RESULTADO

<GrandFather name="Bardock" gender="male" spouse="Gine">
  <Dad name="Goku" gender="male" spouse="Chi Chi">
    <Me name="Gohan" gender="male" />
    <brother name="Goten" gender="male" />
  </Dad>
</GrandFather>

Encontre meu irmão #

XML

<GrandFather name="Bardock" gender="male" spouse="Gine">
  <Dad name="Goku" gender="male" spouse="Chi Chi">
    <brother name="Goten" gender="male" />
    <Me name="Gohan" gender="male" />
    <brother name="Goten" gender="male" />
  </Dad>
</GrandFather>

XPATH

//Me/following-sibling::brother

RESULTADO

<brother name="Goten" gender="male" />

Obtenha todos os avatares antes de Parashurama #

XML

<Dashavatar>
    <Avatar name="Matsya"/>
    <Avatar name="Kurma"/>
    <Avatar name="Varaha"/>
    <Avatar name="Narasimha"/>
    <Avatar name="Vamana"/>
    <Avatar name="Balabhadra"/>
    <Avatar name="Parashurama"/>
    <Avatar name="Rama"/>
    <Avatar name="Krishna"/>
    <Avatar name="Kalki"/>
</Dashavatar>

XPATH

//Avatar[@name='Parashurama']/preceding-sibling::node()

RESULTADO

<Avatar name="Matsya"/>
<Avatar name="Kurma"/>
<Avatar name="Varaha"/>
<Avatar name="Narasimha"/>
<Avatar name="Vamana"/>
<Avatar name="Balabhadra"/>

Obtenha todos os avatares após Parashurama #

XML

<Dashavatar>
    <Avatar name="Matsya"/>

    <Avatar name="Krishna"/>
    <Avatar name="Kalki"/>
</Dashavatar>

//Avatar[@name='Parashurama']/following-sibling::node()

RESULTADO

<Avatar name="Kalki" />

Obtenha todos os detalhes (nós filhos) de House #

XML

<House>
    <Rooms>10</Rooms>
    <People>4</People>
    <TVs>4</TVs>
    <Floors>2</Floors>
</House>

XPATH

/House/child::node()

RESULTADO

<Rooms>10</Rooms>
<People>4</People>
<TVs>4</TVs>
<Floors>2</Floors>

Obtenha todos os quartos (crianças imediatas chamadas Quarto) na Casa #

XML

<House>
    <numRooms>4</numRooms>
    <Room name="living"/>
    <Room name="master bedroom"/>
    <Room name="kids' bedroom"/>
    <Room name="kitchen"/>
</House>

XPATH

/House/child::Room

ou

/House/*[local-name()='Room']

RESULTADO

<Room name="living" />
<Room name="master bedroom" />
<Room name="kids' bedroom" />
<Room name="kitchen" />

Obtenha todos os quartos (independentemente da posição) na Casa #

XML

<House>
    <numRooms>4</numRooms>
    <Floor number="1">
        <Room name="living"/>
        <Room name="kitchen"/>
    </Floor>
    <Floor number="2">
        <Room name="master bedroom"/>
        <Room name="kids' bedroom"/>    
    </Floor>
</House>

XPATH

/House/descendant::Room

RESULTADO

<Room name="living" />
<Room name="kitchen" />
<Room name="master bedroom" />
<Room name="kids' bedroom" />
