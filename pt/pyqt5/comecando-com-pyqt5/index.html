Exemplo de Olá Mundo #

Este exemplo cria uma janela simples com um botão e uma linha de edição em um layout. Ele também mostra como conectar um sinal a um slot, de modo que clicar no botão adiciona algum texto à edição de linha.

import sys
from PyQt5.QtWidgets import QApplication, QWidget


if __name__ == '__main__':
    
    app = QApplication(sys.argv)

    w = QWidget()
    w.resize(250, 150)
    w.move(300, 300)
    w.setWindowTitle('Hello World')
    w.show()
    
    sys.exit(app.exec_())

Análise

app = QtWidgets.QApplication(sys.argv)

Todo aplicativo PyQt5 deve criar um objeto de aplicativo. O parâmetro sys.argv é uma lista de argumentos de uma linha de comando. Os scripts Python podem ser executados a partir do shell.

w = QWidget()

O widget QWidget é a classe base de todos os objetos da interface do usuário no PyQt5. Nós fornecemos o construtor padrão para QWidget. O construtor padrão não tem pai. Um widget sem pai é chamado de janela.

w.resize(250, 150)

O método resize() redimensiona o widget. Tem 250px de largura e 150px de altura.

w.move(300, 300)

O método move() move o widget para uma posição na tela nas coordenadas x=300, y=300.

w.setWindowTitle('Hello World')

Aqui definimos o título para nossa janela. O título é mostrado na barra de título.

w.show()

O método show() exibe o widget na tela. Um widget é criado primeiro na memória e depois mostrado na tela.

sys.exit(app.exec_())

Por fim, entramos no mainloop da aplicação. A manipulação de eventos começa a partir deste ponto. O mainloop recebe eventos do sistema de janelas e os despacha para os widgets do aplicativo. O mainloop termina se chamarmos o método exit() ou o widget principal for destruído. O método sys.exit() garante uma saída limpa. O ambiente será informado como a aplicação terminou.

O método exec_() tem um sublinhado. É porque o exec é uma palavra-chave do Python. E assim, exec_() foi usado no lugar.

Adicionando um ícone de aplicativo #

import sys
from PyQt5.QtWidgets import QApplication, QWidget
from PyQt5.QtGui import QIcon


class Example(QWidget):
    
    def __init__(self):
        super().__init__()
        
        self.initUI()
        
        
    def initUI(self):
        
        self.setGeometry(300, 300, 300, 220)
        self.setWindowTitle('Icon')
        self.setWindowIcon(QIcon('web.png'))        
    
        self.show()
        
        
if __name__ == '__main__':
    
    app = QApplication(sys.argv)
    ex = Example()
    sys.exit(app.exec_())

Análise

Argumentos de função em Python

Em Python, funções definidas pelo usuário podem receber quatro tipos diferentes de argumentos.

  1. Argumentos padrão:
  • Definição de função

def defaultArg( nome, msg = "Olá!"):

  • Chamada de função

    defaultArg( name)

  1. Argumentos necessários:

  • Definição de função

    def requiredArg (str,num):

  • Chamada de função:

    requiredArg ("Hello",12)

  1. Argumentos de palavra-chave:

  • Definição de função

    def keywordArg( name, role ):

  • Chamada de função

    keywordArg( name = "Tom", role = "Manager")

    or

    keywordArg( role = "Manager", name = "Tom")

  1. Número variável de argumentos:

  • Definição de função

    def varlengthArgs(*varargs):

  • Chamada de função

    varlengthArgs(30,40,50,60)

    class Example(QWidget):

      def __init__(self):
      super().__init__()
      ...

Três coisas importantes na programação orientada a objetos são classes, dados e métodos. Aqui criamos uma nova classe chamada Example. A classe Example herda da classe QWidget. Isso significa que chamamos dois construtores: o primeiro para a classe Example e o segundo para a classe herdada. O método super() retorna o objeto pai da classe Example e chamamos seu construtor. A variável self se refere ao próprio objeto.

Por que usamos __init__?

Veja isso:

class A(object):
    def __init__(self):
        self.lst = []

class B(object):
    lst = []

e agora tente:

>>> x = B()
>>> y = B()
>>> x.lst.append(1)
>>> y.lst.append(2)
>>> x.lst
[1, 2]
>>> x.lst is y.lst
True

e isto:

>>> x = A()
>>> y = A()
>>> x.lst.append(1)
>>> y.lst.append(2)
>>> x.lst
[1]
>>> x.lst is y.lst
False

Isso significa que x na classe B é estabelecido antes da instanciação?

Sim, é um atributo de classe (é compartilhado entre instâncias). Enquanto na classe A é um atributo de instância.

self.initUI()

A criação da GUI é delegada ao método initUI().

self.setGeometry(300, 300, 300, 220)
self.setWindowTitle('Icon')
self.setWindowIcon(QIcon('web.png'))

Todos os três métodos foram herdados da classe QWidget. O setGeometry() faz duas coisas: localiza a janela na tela e define seu tamanho. Os dois primeiros parâmetros são as posições xey da janela. A terceira é a largura e a quarta é a altura da janela. Na verdade, ele combina os métodos resize() e move() em um método. O último método define o ícone do aplicativo. Para fazer isso, criamos um objeto QIcon. O QIcon recebe o caminho para o nosso ícone a ser exibido.

if __name__ == '__main__':
    
    app = QApplication(sys.argv)
    ex = Example()
    sys.exit(app.exec_())

O aplicativo e os objetos de exemplo são criados. O loop principal é iniciado.

Mostrando uma dica de ferramenta #

import sys
from PyQt5.QtWidgets import (QWidget, QToolTip, 
    QPushButton, QApplication)
from PyQt5.QtGui import QFont    


class Example(QWidget):
    
    def __init__(self):
        super().__init__()
        
        self.initUI()
        
        
    def initUI(self):
        
        QToolTip.setFont(QFont('SansSerif', 10))
        
        self.setToolTip('This is a <b>QWidget</b> widget')
        
        btn = QPushButton('Button', self)
        btn.setToolTip('This is a <b>QPushButton</b> widget')
        btn.resize(btn.sizeHint())
        btn.move(50, 50)       
        
        self.setGeometry(300, 300, 300, 200)
        self.setWindowTitle('Tooltips')    
        self.show()
        
        
if __name__ == '__main__':
    
    app = QApplication(sys.argv)
    ex = Example()
    sys.exit(app.exec_())

Análise

QToolTip.setFont(QFont('SansSerif', 10))

Este método estático define uma fonte usada para renderizar dicas de ferramentas. Usamos uma fonte SansSerif de 10px.

self.setToolTip('This is a <b>QWidget</b> widget')

Para criar uma dica de ferramenta, chamamos o método setTooltip(). Podemos usar a formatação de texto rico.

btn = QPushButton('Button', self)
btn.setToolTip('This is a <b>QPushButton</b> widget')

Criamos um widget de botão e definimos uma dica de ferramenta para ele.

btn.resize(btn.sizeHint())
btn.move(50, 50)

O botão está sendo redimensionado e movido na janela. O método sizeHint() fornece um tamanho recomendado para o botão.

Empacote seu projeto em executável/instalador #

cx_Freeze - uma ferramenta pode empacotar seu projeto para executável/instalador

  • depois de instalá-lo pelo pip, para o pacote demo.py, precisamos do setup.py abaixo.
import sys
from cx_Freeze import setup, Executable

# Dependencies are automatically detected, but it might need fine tuning.
build_exe_options = {
    "excludes": ["tkinter"],
    "include_files":[('./platforms','./platforms')] # need qwindows.dll for qt5 application
}

# GUI applications require a different base on Windows (the default is for a
# console application).
base = None
if sys.platform == "win32":
    base = "Win32GUI"

setup(  name = "demo",
        version = "0.1",
        description = "demo",
        options = {"build_exe": build_exe_options},
        executables = [Executable("demo.py", base=base)])
  • então construa
python .\setup.py build
  • então dist
python .\setup.py bdist_msi

Instalação ou configuração #

  1. Instale o Anaconda (PyQt5 é embutido), especialmente para usuários do Windows.
  2. Integrar QtDesigner e QtUIConvert no PyCharm (Ferramentas Externas)
    • Open PyCharm Settings > Tools > External Tools
    • Create Tool(QtDesigner) - used to edit *.ui files
    • Create Tool(PyUIConv) - used to convert *.ui to *.py
  3. Gravar demonstração
    • new window.ui by external tool(QtDesigner)
    • convert to window.py by external tool(PyUIConv)
    • demo 
      import sys
from PyQt5.QtWidgets import QApplication,QMainWindow
from window import Ui_MainWindow

if __name__ == '__main__':
    app = QApplication(sys.argv)
    w = QMainWindow()
    ui = Ui_MainWindow()
    ui.setupUi(w)
    w.show()
    sys.exit(app.exec_())
