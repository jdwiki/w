Instalação ou configuração #

Por padrão, todas as bibliotecas necessárias para compilar aplicativos ASP.NET são incluídas durante a instalação do Visual Studio. Se for lançada uma versão mais recente do ASP.NET que não foi incluída no Visual Studio, você poderá baixar a biblioteca SDK apropriada da Microsoft, que incluirá todas as bibliotecas necessárias para essa versão.

Da mesma forma, o sistema operacional Windows vem pré-instalado com uma versão mais recente do ASP.NET e é registrado automaticamente no IIS para configuração e execução. Da mesma forma, se uma versão mais recente do ASP.NET estiver disponível, você pode instalar o SDK para a versão que você precisa e então usar a ferramenta aspnet_regiis para registrar a estrutura com o IIS para uso.

Também deve ser observado que, para implantações de servidor, também existe um pacote redistribuível do ASP.NET SDK. Esta versão é uma versão simplificada do SDK, com apenas as bibliotecas essenciais e não possui as ferramentas e integrações com o Visual Studio.

Visão geral do ASP.NET #

ASP.NET é um modelo de desenvolvimento Web unificado que inclui os serviços necessários para você construir aplicativos Web de classe empresarial com um mínimo de codificação. O ASP.NET faz parte do .NET Framework e, ao codificar aplicativos ASP.NET, você tem acesso a classes no .NET Framework.

Você pode codificar seus aplicativos em qualquer linguagem compatível com o Common Language Runtime (CLR), incluindo Microsoft Visual Basic, C#, JScript .NET e J#. Essas linguagens permitem que você desenvolva aplicativos ASP.NET que se beneficiam do Common Language Runtime, segurança de tipo, herança e assim por diante.

ASP.NET inclui:

  • Uma estrutura de página e controles
  • O compilador ASP.NET
  • Infraestrutura de segurança
  • Instalações de gestão estatal
  • Configuração do aplicativo
  • Monitoramento de saúde e recursos de desempenho
  • Suporte a depuração
  • Uma estrutura de serviços Web XML
  • Ambiente de hospedagem extensível e gerenciamento do ciclo de vida do aplicativo
  • Um ambiente de designer extensível

Introdução simples do ASP.NET #

Asp.net é uma estrutura de aplicativo da Web desenvolvida pela Microsoft para criar aplicativos da Web e WebServices dinâmicos orientados a dados.

Asp.net é basicamente um subconjunto de uma estrutura .NET mais ampla. Um framework nada mais é do que uma coleção de classes.

No .NET Framework você pode construir o aplicativo Console. Aplicativo da Web, aplicativo de janela, aplicativo móvel. Portanto, para o aplicativo da Web, o ASP.net está sendo usado.

ASP.NET é o sucessor do ASP clássico (Active Server Page).

O que é Aplicativo da Web?

Um aplicativo da web é um aplicativo que é acessado por usuários usando um navegador da web, como: -Microsoft Internet Explorer. -Google Chrome

  • Mozila Fire Fox
  • Safari de maçã

Olá Mundo com OWIN #

Use o gerenciador de pacotes para instalar o Microsoft.Owin.SelfHost

install-packet Microsoft.Owin.SelfHost

Código para um aplicativo web HelloWorld mínimo em execução a partir de uma janela de console:

namespace HelloOwin
{
    using System;
    using Owin;

    class Program
    {
        static readonly string baseUrl = "http://localhost:8080";

        static void Main(string[] args)
        {
            using (Microsoft.Owin.Hosting.WebApp.Start<Startup>(baseUrl))
            {
                Console.WriteLine("Prease any key to quit.");
                Console.ReadKey();
            }
        }

        
    }

    public class Startup
    {
        public void Configuration(IAppBuilder app)
        {
            app.Run(ctx =>
            {
                return ctx.Response.WriteAsync("Hello World");
            });
        }
    }
}
