Ambientes de Desenvolvimento Integrado #

O desenvolvimento geralmente começa com um editor ou um Ambiente de Desenvolvimento Integrado. Os seguintes IDEs são conhecidos por suportarem o Meteor até certo ponto:

  • Atom - IDE Javascript que pode aproveitar totalmente a estrutura javascript isomórfica do Meteor. Se você quiser hackear seu próprio editor, este é o único a escolher.
  • Cloud9 - A mais nova oferta de desenvolvimento em nuvem que suporta Meteor, com um tutorial.
  • MeteorDevTools - Extensão do Chrome para Blaze, DDP e Minimongo.
  • Sublime - Editor de texto leve e popular.
  • WebStorm - O IDE mais completo atualmente disponível para Meteor.

Depuradores #

A maioria da depuração acontece no terminal ou nas ferramentas de desenvolvimento do Chrome ou Safari, que são bastante sofisticadas o suficiente para 99% de suas necessidades. No entanto, se você deseja depurar no Firefox ou precisa de uma funcionalidade extra de depuração do servidor, existem alguns utilitários extras que você pode usar.

  • Firefox - Firebug
  • Node-Inspector
  • Meteor Toys ou adicione diretamente- meteor add meteortoys:allthings

Utilitários de colaboração remota para desenvolvedores distribuídos #

Desenvolver aplicativos Meteor geralmente significa desenvolver reatividade multicliente, o que requer ferramentas de colaboração. As seguintes ferramentas provaram ser populares dentro da comunidade Meteor.

  • Google Hangouts - Videoconferência e bate-papo.
  • Zenhub.io - Quadros Kanban para GitHub.
  • InVision - Wireframing e prototipagem colaborativos.
  • Meeting Hero - Planejamento de reuniões colaborativas.
  • Hackpad - Edição colaborativa de documentos.
  • Slack - Feeds colaborativos de rastreamento de projetos.
  • MadEye - Editor web colaborativo.
  • Screenhero - Compartilhamento de tela colaborativo.
  • Proto.io - Wireframing e prototipagem.
  • HuBoard - Quadros Kanban para GitHub.
  • Zapier - Os melhores aplicativos. Juntos.
  • Teamwork.com - Gerenciamento tradicional de projetos e gráficos de Gannt.
  • Sprint.ly - Mais quadros kanban e planejamento de sprint que funciona com o GitHub.
  • LucidChart - Alternativa online ao Visio.
  • Waffle.io - Alternativa Trello/ZenHub que se integra ao GitHub.

Clientes REST #

Se você deseja integrar o Meteor com uma API externa, é provável que ele seja exposto como uma interface REST. Costumamos usar os seguintes aplicativos do Chrome para testar APIs REST.

  • Carteiro
  • DHC Rest Client

Ferramentas on-line:

  • Hurl.it
  • RequestBin

Ferramentas de banco de dados #

Depois de passar pelo aplicativo ‘Hello World’, você precisará começar a prestar atenção à sua coleção e esquemas de documentos e precisará de algumas ferramentas para gerenciar seu banco de dados.

  • Robomongo - Um favorito de longa data da comunidade para gerenciar o Mongo. Altamente recomendado.
  • JSON Generator - Utilitário inestimável para gerar conjuntos de dados de amostra.
  • Página de preferências do MacOSX Mongo - GUI de preferências para MacOSX.
  • MongoHub - Outra GUI do Mongo, semelhante ao RoboMongo. Apenas MacOSX.
  • Mongo3 - Uma das poucas ferramentas de gerenciamento de cluster disponíveis. Capaz de visualizar conjuntos de replicação. A única desvantagem é que é construído em Ruby.
  • Mongo Monitoring Service - Quando você estiver pronto para colocar algo em produção, o MMS é inestimável. Agora conhecido como MongoDB Atlas.
  • Mongo Express - Interface de administração MongoDB baseada na Web, escrita com Node.js e express

Codificação móvel no iOS #

Texttastic Code Editor - Editor de código com destaque de sintaxe para dispositivos iOS.

Working Copy - Clone repositórios do Github para o seu iPad e codifique em qualquer lugar.

CodeHub - Navegue e mantenha seus repositórios do GitHub. Ferramenta de gerenciamento.

iOctocat - Utilitário social para seguir projetos do Github.

iMockups para iPad - Wireframes e maquetes. Suporta wireframes para desktops e dispositivos móveis.

Blueprint - wireframes e maquetes do iOS. Principalmente para desenvolvimento iOS, mas um pouco utilizável para aplicativos da web.

JSON Designer - Arquitetura de dados e design de esquema de dados.
