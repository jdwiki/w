Function é uma série de instruções entre as instruções “Function” e “End Function”.

A Função executa uma atividade e retorna o controle ao chamador. Quando ele retorna o controle, ele também retorna um valor para o código de chamada.

Você pode definir uma função em uma classe, estrutura e módulo. Por padrão é Público. Isso significa que você pode chamá-lo de qualquer lugar em seu aplicativo que tenha acesso à classe, estrutura ou módulo em que você o definiu.

Sintaxe #

  • [Modifiers] Function Name_Of_The_Function [(Arg_List)] As Return_Type

  • [Declarações]

  • Função Final

  • Os dois Modificadores de Função usados ​​neste exemplo são Público e Privado. Esses Modificadores definem o escopo da Função.

  • Funções com escopo Privado só podem ser chamadas a partir do arquivo de origem de onde foram definidas. No nosso caso, pode ser chamado no módulo. E não pode ser chamado fora do Módulo.

  • Funções com escopo Público podem ser chamadas tanto fora quanto dentro do Módulo. Simplesmente podemos dizer como “Podemos chamá-lo em qualquer lugar do programa”.

  • O modificador padrão da função é público.

  • Por padrão, os argumentos da função são passados ​​por referência (em um tópico separado, isso será explicado em detalhes).

Criando e chamando uma função #

Este exemplo usando projeto EXE padrão com adição de um arquivo de módulo.

  • Criar novo projeto “EXE padrão”. Então aqui, um formulário será adicionado ao projeto por padrão.
  • Adicionar um arquivo de módulo ao projeto
  • Coloque um botão de comando no formulário
  • Criar evento de clique do botão de comando.

Código do módulo

Criou duas Funções no Módulo. Uma é uma Função Pública (FnAdd). São necessários dois argumentos inteiros val_1 e val_2. Ele retorna um inteiro. Esta função adiciona os dois argumentos e retorna o valor para o chamador. Antes da adição, os dois argumentos passam por um processo em outra Função. Que é uma função privada. Característica/Regras da Função Pública e Privada explicadas na seção Observações.

Public Function FnAdd(val_1 As Integer, val_2 As Integer) As Integer

'Calling private function
val_1 = FnMultiplyBy5(val_1)

'Calling private function
val_2 = FnMultiplyBy5(val_2)

'Function return statement
FnAdd = val_1 + val_2

End Function

Abaixo está a função Privada no Módulo. Leva um argumento inteiro val. Ele retorna um inteiro. Esta função multiplica um valor 5 com o argumento e retorna o resultado para o chamador.

Private Function FnMultiplyBy5(Val As Integer) As Integer

'Function return statement
FnMultiplyBy5 = Val * 5

End Function

Código do formulário

No botão de comando clique em Evento. Aqui estamos chamando a função Module Public “FnAdd”

Private Sub Command1_Click()
Debug.Print FnAdd(3, 7)
End Sub

Resultado na janela imediata
