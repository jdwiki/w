Parâmetros #

Parâmetro Detalhes
valor O valor a ser convertido de
targetType O tipo que está sendo convertido para
parâmetro Valor opcional para controlar como a conversão funciona
cultura Objeto CultureInfo - obrigatório se a localização for necessária

O método Convert converte o valor da fonte (geralmente o modelo de exibição) para o destino (geralmente uma propriedade de um controle).

O método ConvertBack converte o valor do destino de volta para a origem. Só é necessário se a ligação for TwoWay ou OneWayToSource.

Quando um ConvertBack não é suportado, ou seja, não há mapeamento um-para-um entre o valor pré-conversão e o valor pós-conversão, é uma prática comum fazer com que o método ConvertBack retorne DependencyProperty.UnsetValue. É uma opção melhor do que lançar uma exceção (por exemplo, NotImplementedException), pois evita erros inesperados de tempo de execução. Além disso, as vinculações podem se beneficiar de seu FallbackValue quando DependencyProperty.UnsetValue é retornado por um conversor.

Conversores 101 #

Os controles XAML podem ter propriedades de dependência que podem ser associadas a objetos de DataContext ou outros controles. Quando o tipo do objeto que está sendo vinculado é diferente do tipo de destino DependencyProperty, um conversor pode ser usado para adaptar um tipo a outro.

Conversores são classes que implementam System.Windows.Data.IValueConverter ou System.Windows.Data.IMultiValueConverter; O WPF implementa alguns conversores prontos para uso, mas os desenvolvedores podem ver uso em implementações personalizadas, como é frequentemente o caso.

Para usar um conversor em XAML, uma instância pode ser instanciada na seção Resources. Para o exemplo abaixo, System.Windows.Controls.BooleanToVisibilityConverter será usado:

<UserControl.Resources>
    <BooleanToVisibilityConverter x:Key="BooleanToVisibilityConverter"/>
</UserControl.Resources>

Observe o elemento x:Key definido, que é então usado para referenciar a instância de BooleanToVisibilityConverter na ligação:

<TextBlock Text="This will be hidden if property 'IsVisible' is true"
           Visibility="{Binding IsVisible, 
                                Converter={StaticResource BooleanToVisibilityConverter}}"/>

No exemplo acima, uma propriedade booleana IsVisible é convertida em um valor da enumeração System.Windows.Visibility; Visibility.Visible se verdadeiro, ou Visibility.Collapsed caso contrário.

Conversor de String para IsChecked #

Em XAML:

<RadioButton IsChecked="{Binding EntityValue, Mode=TwoWay,
                         Converter={StaticResource StringToIsCheckedConverter},
                         ConverterParameter=Male}"
             Content="Male"/>

<RadioButton IsChecked="{Binding EntityValue, Mode=TwoWay,
                         Converter={StaticResource StringToIsCheckedConverter},
                         ConverterParameter=Female}"
             Content="Female"/>

A classe C#:

public class StringToIsCheckedConverter : IValueConverter
{
    public object Convert(object value, Type targetType, object parameter, System.Globalization.CultureInfo culture)
    {
        string input = (string)value;
        string test = (string)parameter;
        return input == test;
    }

    public object ConvertBack(object value, Type targetType, object parameter, System.Globalization.CultureInfo culture)
    {
        if (value == null || !(value is bool))
        {
            return string.Empty;
        }
        if (parameter == null || !(parameter is string))
        {
            return string.Empty;
        }
        if ((bool)value)
        {
            return parameter.ToString();
        }
        else
        {
            return string.Empty;
        }
    }
}

Criando e usando um conversor: BooleanToVisibilityConverter e InvertibleBooleanToVisibilityConverter #

Para estender e expandir a experiência de vinculação, temos conversores para converter um valor de um tipo em outro valor de outro tipo. Para alavancar os conversores em um Databinding, primeiro você precisa criar uma classe DataConverter tht extens

  • IValueConverter(WPF e UWP)

ou

  • IMultiValueConverter(WPF)

se você quiser converter vários tipos em um tipo
Neste caso, focamos na conversão de um valor booleano True/False para as Visibilidades correspondentes Visibility.Visible e Visibility.Collapsed:

public class BooleanToVisibilityConverter : IValueConverter
{
    public object Convert(object value, Type targetType, object parameter, string language)
    {
        return (value is bool && (bool) value) ? Visibility.Visible : Visibility.Collapsed;
    }

    public object ConvertBack(object value, Type targetType, object parameter, string language)
    {
        return (value is Visibility && (Visibility) value == Visibility.Visible);
    }
}

O método Convert é chamado sempre que você GET dados FROM o ViewModel.
O ConvertBack é chamado em SET ing data TO o ViewModel para ligações BindingMode.TwoWay.

Claro que você também pode utilizar propriedades dentro do seu conversor. Dê uma olhada neste:

public class InvertibleBooleanToVisibilityConverter : IValueConverter
{
    public bool Invert { get; set; } = false;

    public object Convert(object value, Type targetType, object parameter, string language)
    {
        return (value is bool && (bool) value != Invert) ? Visibility.Visible : Visibility.Collapsed;
    }

    public object ConvertBack(object value, Type targetType, object parameter, string language)
    {
        return (value is Visibility && ((Visibility) value == Visibility.Visible) != Invert);
    }
}

Se você quiser usar um conversor em um Binding, simplesmente declare-o como um recurso em sua página, janela ou outro elemento, dê a ele uma chave e forneça as propriedades potencialmente necessárias:

<Page ...
    xmlns:converters="using:MyNamespce.Converters">
<Page.Resources>
    <converters:InvertibleBooleanToVisibilityConverter 
        x:Key="BooleanToVisibilityConverter" 
        Invert="False" />
</Page.Resources>

e use-o como um StaticResource em uma ligação:

<ProgressRing 
        Visibility="{Binding ShowBusyIndicator, 
            Converter={StaticResource BooleanToVisibilityConverter},
            UpdateSourceTrigger=PropertyChanged,
            Mode=OneWay}" />
