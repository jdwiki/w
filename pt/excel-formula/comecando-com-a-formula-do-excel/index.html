Introdução #

As fórmulas do Excel são usadas para processar equações simples ou complexas e enviar o resultado para uma célula específica dentro de uma planilha.

_(Existe uma exceção para isso onde a classe WorksheetFunction é usada na programação VBA, no entanto, o VBA está fora do escopo desta tag - consulte a documentação excel-vba ou vba para artigos relacionados ao VBA.) _

As fórmulas podem ser usadas para converter, manipular ou avaliar muitos tipos diferentes de expressões usando todos os tipos de dados. No entanto, existem algumas restrições em torno das fórmulas e do que elas são capazes de alcançar.

  • As fórmulas não podem alterar nenhuma parte de uma planilha ou o conteúdo de outra célula.
  • Fórmulas só podem retornar números precisos de até 15 dígitos.
  • As fórmulas não podem conter mais de 8.192 caracteres.
  • As fórmulas não podem fazer referência a mais de 2.048 outros intervalos.
  • Fórmulas não podem usar mais de 255 argumentos em qualquer função.
  • As datas usadas nos cálculos devem estar no intervalo de 1900-01-01 a 9999-12-31

Instalação ou Configuração #

Instruções detalhadas sobre a(s) fórmula(s) do Excel.

Existem muitas fórmulas que você pode escolher. Eles são divididos em 7 categorias diferentes e estão na guia FÓRMULAS no Excel. As categorias são:

  1. Financeiro
  2. Lógico
  3. Texto
  4. Data e Hora
  5. Pesquisa e Referência
  6. Matemática e trigonometria
  7. Mais funções

Depois, há também seções para

  1. Usado recentemente
  2. AutoSoma

Se você não tiver certeza de qual usar, pode usar o botão ‘Inserir Função’ para ajudar a determinar qual usar.

Você não precisa usar apenas uma fórmula por vez, você pode combiná-las para obter os resultados necessários ou combiná-las com outras coisas como Intervalos Nomeados.

Named Ranges também estão na guia FORMULAS na seção Define Name. Você pode usar o Name Manager para criar ou modificar os intervalos e os nomes dos intervalos. Há também três outros botões para Definir Nome para definir e aplicar nomes, Usar na Fórmula para escolher um nome definido e inseri-lo em uma fórmula e Criar da Seleção que criará um intervalo nomeado com o nome com base na célula mais à esquerda no intervalo.

Há também a seção Formula Auditing. Esta seção ajudará na solução de problemas de uma fórmula. Aqui você pode traçar a fórmula e ver exatamente o que ela está fazendo.

Por fim, há a seção Cálculo que permitirá ativar/desativar o cálculo automático de todas as fórmulas e calcular manualmente quaisquer fórmulas, independentemente de o cálculo automático estar ativado ou não.

Veja o Excel #

Não há necessidade de instalação adicional para excel-formula além do que já é necessário para excel. Consulte a documentação do Excel.

Referências de células do Excel #

Você pode usar referências de célula sem números de linha.

Por exemplo, a fórmula =sum(C:C) somará todos os valores na coluna C da planilha ativa. Isso é útil quando você está adicionando e removendo linhas, mas não deseja atualizar suas fórmulas a cada vez.

Há alguns casos em que usar essa referência de coluna inteira não é uma boa ideia. Há um bom artigo aqui. Ele discute muitas variáveis ​​diferentes e casos de teste para explicar quando seria benéfico usar algo como =sum(C:C) ou usar algo como =sum(C1:C1000).
