Representando Linhas

Uma linha geométrica é uma linha reta que se estende até o infinito em ambas as direções. Isso o distingue do segmento ou raio, que termina em algum ponto, e também da curva ou polígono, que não precisa ser reto.

Existem diferentes maneiras de como uma linha pode ser apresentada. Cada um vem com suas próprias vantagens e desvantagens.

Ponto e direção #

Pode-se descrever uma linha em dimensões arbitrárias como

X = A + t*D

onde A e D são ambos vetores de muitas dimensões adequadas. Então em 2d isso seria

x = Ax + t*Dx
y = Ay + t*Dy

Agora, como t assume qualquer valor real, esta equação produzirá todos os pontos ao longo da linha. A representação não é única, porém: qualquer ponto ao longo da linha pode ser usado como ponto de partida ‘A’, e qualquer múltiplo do vetor ‘D’ representa a mesma direção.

Par de pontos #

Dado um par de pontos ‘A’ e ‘B’ em um espaço vetorial de dimensão arbitrária, pode-se descrever a linha entre eles como

X = A + t*(B - A) = (1 - t)*A + t*B

então em 2d isso seria

x = Ax + t*(Bx - Ax) = (1 - t)*Ax + t*Bx
y = Ay + t*(By - Ay) = (1 - t)*Ay + t*By

Como t assume qualquer valor real, isso produzirá todos os pontos ao longo da linha. A representação não é única, pois qualquer par de pontos distintos ao longo da linha descreverá a mesma linha. É fácil alternar entre linha completa e segmento de linha usando essa representação, pois restringir t ao intervalo [0, 1] produzirá um segmento de linha.

Forma normal / equação homogênea #

Uma linha no plano pode ser descrita como

a*x + b*y + c = 0

Isso usa um vetor de parâmetro de três elementos [a, b, c] para descrever a linha. Às vezes, o termo constante c é movido para o lado direito da equação. A representação não é única, pois o comprimento desse vetor é arbitrário (desde que não seja zero). Esse vetor é chamado de homogêneo.

O vetor [a, b] é perpendicular à linha, daí o nome “forma normal”. Se o comprimento desse vetor for um, isso é chamado de forma normal de Hesse que pode ser usada para calcular facilmente a distância de pontos arbitrários até a linha em questão . Mesmo isso não torna a representação totalmente única, pois pode-se negar todos os três parâmetros e obter o mesmo conjunto de pontos que satisfaçam a equação.

O conceito se generaliza para dimensões mais altas, mas não descreve mais uma linha, mas sim um plano em 3d e um hiperplano em geral.

Função Simples #

Pode-se descrever uma linha no plano como

y = a*x + b

para que a linha seja essencialmente controlada por dois parâmetros a, b. Para uma determinada linha, a escolha desses parâmetros é única. Mas as linhas verticais não podem ser descritas assim.
