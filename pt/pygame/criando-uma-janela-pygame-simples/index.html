Criando uma janela pygame simples

  • O código completo

Nesta página

  • O código completo

O código completo #

import pygame

pygame.init()

WIDTH = 300
HEIGHT = 200
SCREEN = pygame.display.set_mode((WIDTH, HEIGHT))

pygame.display.set_caption('My Game')

WHITE = (255, 255, 255)
BLACK = (0, 0, 0)
RED = (255, 0, 0)
GREEN = (0, 255, 0)
BLUE = (0, 0, 255)
YELLOW = (255, 255, 255)

SCREEN.fill(RED)
pygame.display.flip()

is_running = True
while is_running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            is_running = False

pygame.quit()

Importando e inicializando pygame

Como fazemos com qualquer módulo em python, precisamos importar pygame:

import pygame

Em seguida, inicializamos todos os módulos pygame importados:

pygame.init()

Isso é usado para inicializar todos os módulos pygame. Sem isso os módulos não funcionariam

Definindo Constantes

Em seguida, definimos algumas constantes aqui:

WIDTH = 300
HEIGHT = 200
SCREEN = pygame.display.set_mode((WIDTH, HEIGHT))

As constantes WIDTH e HEIGHT são usadas para criar uma janela, que teria uma largura de 300 pixels e uma altura de 200 pixels. A função usada em SCREEN, pygame.display.set_mode((WIDTH, HEIGHT)), irá definir o modo de exibição e retornar um Surface object. Observe como os parâmetros para esta função são as constantes WIDTH e HEIGHT definidas anteriormente.

Definindo o nome da janela

Em seguida, usamos esta função para alterar o nome da janela para Meu Jogo:

pygame.display.set_caption('My Game')

Definindo cores

Depois definimos 6 cores que podem ser usadas em nossa janela:

WHITE = (255, 255, 255)
BLACK = (0, 0, 0)
RED = (255, 0, 0)
GREEN = (0, 255, 0)
BLUE = (0, 0, 255)
YELLOW = (255, 255, 255)

Ao definir as cores colocamos 3 valores que variam entre 0 e 255. A classe pygame.Color normalmente segue este formato:

COLOUR = (r, g, b, a)

Onde o parâmetro r define o valor vermelho da cor, o parâmetro g define o valor verde da cor e o parâmetro b define o valor azul da cor. O parâmetro a define o valor alfa da cor.

Damos então este comando:

SCREEN.fill(RED)

Esta é uma função pygame.Surface.fill que preenche o objeto Surface, nossa tela, com a cor vermelha.

Usando pygame.display.flip()

Usamos então esta função

pygame.display.flip()

Isso basicamente faz com que tudo o que desenhamos na superfície da tela fique visível e atualiza o conteúdo de toda a tela. Sem essa linha, o usuário não veria nada na tela do pygame.

O loop do jogo

As próximas linhas são o que chamamos de “loop de jogo”.

Para começar, criamos uma variável e a tornamos True:

is_running = True

Para que possamos iniciar nosso loop while:

while is_running:

que será executado durante todo o jogo.

Em sua forma mais básica, o pygame tem “eventos” que recebem a entrada do usuário, por exemplo, um botão ou clique do mouse. O Pygame lida com esses eventos por meio de uma fila de eventos. Podemos obter esses eventos da fila de eventos com este loop for:

for event in pygame.event.get():

Que basicamente passa por uma lista de eventos, nossa fila de eventos. Estas são as próximas 2 linhas:

if event.type == pygame.QUIT:
    is_running = False

Isso fará com que quando o usuário pressionar o botão de saída no canto superior, ocorra o evento com o tipo pygame.QUIT.

Isso então termina o loop while, já que is_running agora é False e ​​o script segue para a linha final:

pygame.quit()

O que desinicializa os módulos pygame.
