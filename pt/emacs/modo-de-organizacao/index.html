Org é um modo para fazer anotações, manter listas de tarefas e planejamento de projetos com um sistema de texto simples rápido e eficaz. Também é um sistema de autoria com suporte exclusivo para programação alfabetizada e pesquisa reproduzível.

site oficial do modo org

Sintaxe de marcação #

Org fornece uma linguagem de marcação completa que ajuda a estruturar o documento e é refletida com a maior precisão possível ao exportar para outros formatos (como HTML ou LaTeX).

Estrutura #

Título do documento #

#+TITLE: This is the title of the document

Seccionando #

* First level
** Second level

Listas #

Ordered list (items can also be numbered like '1)', with a perenthesis):
1. foo
2. bar
3. baz

Unordered list (items can also start with '+' or '*'):
- foo
- bar
- baz

Description
- lorem ipsum :: this is example text
- foo bar :: these are placeholder words

Caixas de seleção #

Cada item em uma lista simples pode ser transformado em uma caixa de seleção, iniciando-a com a string ‘[ ]’.

* TODO [2/4] (or [50%])
  - [-] call people [1/3]
    - [ ] Peter
      - [X] Sarah
      - [ ] Sam
  - [X] order food
  - [ ] think about what music to play
  - [X] talk to the neighbors
  • C-c C-c org-toggle-checkbox
  • C-c C-x C-b org-toggle-checkbox
  • M-S- org-insert-todo-heading
  • C-c C-x o org-toggle-ordered-property
  • C-c # org-update-statistics-cookies

Ênfase e monoespaço #

You can make words *bold*, /italic/, _underlined_, =verbatim=
and ~code~, and, if you must, ‘+strike-through+’.

O texto no código e a string literal não são processados ​​para a sintaxe específica do modo Org, são exportados literalmente.

Links e referências #

Links #

O modo de organização reconhecerá os formatos de URL e os ativará como links clicáveis. No entanto, os links podem ser declarados explicitamente assim:

You will find more information in the [[http://orgmode.org/org.html][Org Manual]].

ou alternativamente :

The org manual is located here: [[http://orgmode.org/org.html]]

Notas de rodapé #

As notas de rodapé podem ser nomeadas:

See the org manual[fn:manual] to get more details.
...
[fn:manual] You will find it here: http://orgmode.org/org.html

ou anônimo e inline:

See the org manual[fn:: You will find it here: http://orgmode.org/org.html]
to get more details.

Associações de teclas básicas #

Para alternar o nível de contorno mostrado:

  • Tab Nível de contorno do ciclo para um título
  • Shift-Tab Nível de contorno do ciclo para todo o documento

Para percorrer os estados TODO:

  • Shift-Seta para a direita
  • Shift-Seta para a esquerda

Para aumentar ou diminuir o nível hierárquico de um título

  • Meta-Seta para a direita Torna o nível mais baixo (“aumentar recuo”)
  • Meta-Seta Esquerda Torna o nível mais alto (“diminuir recuo”)

Para alternar a prioridade para um determinado título:

  • Seta para cima
  • Seta Shift-Down

Para mover um título para cima ou para baixo:

  • Seta Meta-Up
  • Seta Meta-Baixo

(Meta refere-se a teclas diferentes em teclados diferentes. Na maioria das vezes é Alt ou ⌘).

Blocos de código #

Para adicionar um bloco de código, envolva-o com #+BEGIN_SRC language e #+END_SRC. idioma deve corresponder ao modo principal do idioma em questão, por exemplo, o modo principal para o Emacs Lisp é emacs-lisp-mode, então escreva #+BEGIN_SRC emacs-lisp.

#+BEGIN_SRC emacs-lisp
(defun hello-world ()
  (interactive)
  (message "hello world"))
#+END_SRC

#+BEGIN_SRC python
print "hello world"
#+END_SRC

Você pode abrir o bloco de código em um buffer separado digitando C-c' (para org-edit-special). Se você não tiver o modo principal para o idioma especificado, isso dará uma mensagem de erro como No such language mode: foo-mode.

Se o conteúdo que você deseja colocar no bloco não estiver em nenhuma linguagem de programação, você pode usar #+BEGIN_EXAMPLE e #+END_EXAMPLE.

#+BEGIN_EXAMPLE
output from a command I just ran
#+END_EXAMPLE

Existem modelos fáceis para ambos. No início da linha, digite <s ou <e e pressione TAB. Ele se expandirá em um bloco com marcadores de início e fim para SRC ou EXAMPLE, respectivamente.

Esses marcadores não diferenciam maiúsculas de minúsculas, então você pode escrever #+begin_src etc se preferir.

Tabelas #

| Name  | Phone | Age |
|-------+-------+-----|
| Peter | 1234  | 17  |
| Anna  | 4321  | 25  |

Para adicionar uma Tabela no modo org, basta colocar uma barra entre suas colunas (|)

| column1 | column2 | this column is wider |

Quando você pressiona Return de dentro de uma coluna, o org-mode cria automaticamente uma nova linha com as barras.

  • Tab e Return moverão respectivamente para a próxima célula ou linha (ou criará uma nova se não houver nenhuma)
  • Você pode trocar as linhas e colunas com M-ArrowKey
  • M-S-Down e M-S-Right criarão respectivamente uma linha (acima da atual) e uma coluna (à esquerda da atual)
  • M-S-Up e M-S-Left removerão respectivamente a linha atual e a coluna atual
  • C-c i cria um separador
