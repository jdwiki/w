Sair do Emacs #

Você pode sair do Emacs com a seguinte combinação de teclas:

C-x C-c

Onde C é a tecla control.

Suspender o Emacs

Você pode suspender o Emacs usando a seguinte combinação de teclas:

C-z

Isso leva você de volta ao seu shell. Se você quiser retomar sua sessão do emacs, digite fg no seu terminal.

Aborta o comando atual #

Muitas vezes, você entrará em um estado em que tem uma sequência de comandos parcialmente digitada em andamento, mas deseja abortá-la. Você pode abortá-lo com uma das seguintes combinações de teclas:

C-g

EscEscEsc

Tampões #

  • Exemplo de uma lista de buffers

    CRM Buffer                Size  Mode              Filename[/Process]
. * .emacs                3294  Emacs-Lisp        ~/.emacs
 %  *Help*                 101  Help
    search.c             86055  C                 ~/cvs/emacs/src/search.c
 %  src                  20959  Dired by name     ~/cvs/emacs/src/
  * *mail*                  42  Mail
 %  HELLO                 1607  Fundamental       ~/cvs/emacs/etc/HELLO
 %  NEWS                481184  Outline           ~/cvs/emacs/etc/NEWS
    *scratch*              191  Lisp Interaction
  * *
Messages*            1554  Messages

    The first field of a line indicates:

    • ‘.’ the buffer is current.
    • ‘%’ a read-only buffer.
    • ‘*’ the buffer is modified.

  • Selecione buffer. Você pode selecionar qualquer buffer aberto com a seguinte combinação de teclas:

C-xb

Você será solicitado a informar o nome do buffer para o qual deseja alternar.

  • Listar buffers:

    C-x C-b

  • Save-some-buffer, dando a escolha de qual buffer salvar ou não:

    C-x s

  • Mate um buffer:

    C-x k

  • Operações em buffers marcados:

    <kbd>S</kbd>  Save the marked buffers                                                                  

<kbd>A</kbd>  View the marked buffers in this frame.                                                   

<kbd>H</kbd>  View the marked buffers in another frame.                                                

<kbd>V</kbd> Revert the marked buffers.                                                               

<kbd>T</kbd> Toggle read-only state of marked buffers.                                                

<kbd>D</kbd> Kill the marked buffers.                                                                 

<kbd>M-s a C-s</kbd> Do incremental search in the marked buffers.                                     

<kbd>M-s a C-M-s</kbd> Isearch for regexp in the marked buffers.                                      

<kbd>U</kbd> Replace by regexp in each of the marked buffers.                                                                                                                  

<kbd>Q</kbd> Query replace in each of the marked buffers.                                             

<kbd>I</kbd> As above, with a regular expression.                                                     

<kbd>P</kbd> Print the marked buffers.                                                                

<kbd>O</kbd> List lines in all marked buffers which match a given regexp (like the function `occur').                                                                                    

<kbd>X</kbd> Pipe the contents of the marked buffers to a shell command.                                                   

<kbd>N</kbd> Replace the contents of the marked buffers with the output of a shell command.  

<kbd>!</kbd> Run a shell command with the buffer's file as an argument.

    C-x s

  • Mude para o próximo buffer:

C-x DIREITA

  • Mudar para o buffer anterior:

C-x ESQUERDA

Manipulação de arquivos #

  • Salve novamente o arquivo aberto com o mesmo nome de arquivo (Salvar):

    C-x C-s

  • Escreva como filename (Salvar como):

    C-x C-w filename

O novo nome do arquivo será solicitado no minibuffer.

  • Criar novo arquivo ou carregar arquivo existente (Novo/Carregar):

    C-x C-f filename

    With the mnemonic here for f meaning file. You will be prompted for a file path in the minibuffer.

  • Visite o arquivo alternativo

    C-x C-f

    If the file does not exist yet, you will be prompted the path of the file to create in the minibuffer.

Movimento do cursor (ponto) #

Além dos movimentos do cursor usando as teclas de seta, Home, End, Page up e Page down, o emacs define um número de pressionamentos de tecla que podem mover o cursor sobre pedaços menores ou maiores de texto:

Por caractere:

  • Caractere para trás: C-b
  • Caractere para frente: C-f

Por palavra

  • Palavra para trás: M-b (i.e. Alt b, ou Meta b)
  • Encaminhar palavra: M-f

Por linha:

  • Início da linha atual: C-a
  • Início da linha atual primeiro caractere (sem espaço):M-m
  • Fim da linha atual: C-e
  • Linha anterior: C-p
  • Próxima linha: C-n

Todo o buffer:

  • Início do buffer: M-«/kbd>
  • Fim do buffer: M->

Por ‘bloco’, dependendo do contexto (modo):

Associações de teclas típicas:

  • Frase/declaração invertida: M-a
  • Encaminhar frase/declaração: M-e
  • Início da função: M-C-a
  • Fim da função: M-C-e

Argumentos de prefixo

Para mover vários ‘passos’ de uma vez, os comandos de movimento podem receber um argumento de prefixo pressionando ESC ou C-u e um número antes das teclas listadas. Para C-u, o número é opcional e o padrão é 4. Ex. ESC 3 C-n move 3 linhas para baixo, enquanto C-u M-f move 4 palavras para frente.

Notação de ligações de teclas #

A documentação do Emacs usa uma notação consistente para todas as ligações de teclas, que é explicada aqui:

Acordes principais #

Um “acorde de tecla” é obtido pressionando duas ou mais teclas simultaneamente. Os acordes de teclas são indicados separando todas as teclas por traços (-). Eles geralmente envolvem teclas modificadoras, que são colocadas na frente:

  • C-: controle;
  • S-: deslocamento;
  • M-: alt (o “M” significa “Meta” por razões históricas).

Outras chaves são simplesmente denotadas por seus nomes, como:

  • a: a chave a;
  • esquerda: a tecla de seta para a esquerda;
  • SPC: a tecla de espaço;
  • RET: a tecla de retorno.

Exemplos de acordes chave incluem:

  • C-a: pressionando control e a simultaneamente;
  • S-direita: pressionando shift e direita simultaneamente;
  • C-M-a: pressionando control, alt e a simultaneamente.

Sequências de teclas #

“Sequências de teclas” são sequências de teclas (ou acordes de teclas), que devem ser digitadas uma após a outra. Eles são denotados separando todas as notações de tom (ou acorde) por um espaço.

Exemplos incluem:

  • C-x b: pressionando controle e x simultaneamente, depois soltando-os e pressionando b;
  • C-x C-f: pressionando control e x simultaneamente, depois soltando x e pressionando f (uma vez que ambos os acordes envolvem o modificador control, não é necessário liberá-lo).

Usando ESC em vez de Alt #

Os acordes de teclas usando o modificador Alt também podem ser inseridos como uma sequência de teclas começando com ESC. Isso pode ser útil ao usar o Emacs em uma conexão remota que não transmite acordes de teclas Alt, ou quando essas combinações de teclas são capturadas, por exemplo, por um gerenciador de janelas.

Exemplo:

M-x pode ser inserido como ESC x.

Descrevendo ligações de teclas em arquivos lisp do Emacs #

A mesma notação descrita aqui pode ser usada ao definir associações de teclas em arquivos lisp do Emacs.

Exemplo:

(global-set-key (kbd “C-x C-b”) ‘buffer-menu) liga a sequência de teclas C-x C-b ao comando buffer-menu

Múltiplas janelas ou quadros #

“Janela” no Emacs refere-se ao que poderia ser chamado de “painel” ou “divisão de tela”. Alguns comandos de manipulação de janela incluem:

  • Dividir a janela atual horizontalmente: C-x 2
  • Dividir a janela atual verticalmente: C-x 3
  • Selecione a próxima janela: C-x o
  • Fechar janela atual: C-x 0
  • Feche todas as outras janelas, exceto a atual: C-x 1

Um “frame” no Emacs é o que poderia ser chamado de “janela”. Os quadros são manipulados usando estes comandos:

  • Criar novo quadro: C-x 5 2
  • Excluir quadro atual: C-x 5 0
  • Excluir outros quadros: C-x 5 1

A troca de janelas pode ser conseguida usando

  • S-esquerda, S-direita, S-up, S-down (isto é, Shift em conjunto com uma tecla de seta) para alternar para a janela vizinha em uma direção, ou
  • C-x o para mudar para a próxima janela.

Região - Recortar, Copiar, Colar #

  • Definir marca na localização do cursor:

    C-space or C-@

  • Matar região (Cortar):

    C-w

  • Copie a região para matar o anel:

    M-w or Esc-w

  • Yank (Colar) morto mais recentemente:

    C-y

  • Yank (Colar) próximo último morto:

    M-y or Esc-y

Matar #

killé o comando usado pelo Emacs para a exclusão de texto. O comando kill é análogo ao comando cut no Windows. Existem vários comandos que ‘mata’ uma palavra (M-d), o resto da linha (C-k), ou blocos de texto maiores. O texto apagado é adicionado ao kill-ring, do qual pode ser arrancado posteriormente.

Selecionar e cortar (matar) #

Matar e arrancar Semelhante ao recurso select-and-cut no Windows, aqui temos o C-spc. A tecla de ligação C-spc iniciará a seleção, o usuário pode mover a marca com a ajuda das teclas de seta ou outro comando para fazer uma seleção. Quando a seleção estiver completa - pressione o C-w para matar o texto selecionado

Alguns comandos básicos que podem ser usados ​​como referência rápida para o comando kill (retirado do tutorial do Emacs) <pré> M-DEL Kill the word immediately before the cursor M-d Kill the next word after the cursor

C-k         Kill from the cursor position to end of line
M-k         Kill to the end of the current sentence

Puxão #

yankdescreve a inserção de texto apagado anteriormente, por exemplo usando C-y que arranca o texto apagado mais recentemente. O comando Yank é análogo ao comando paste no Windows.

Texto Yank morto anteriormente #

Sabemos que o comando kill adiciona o texto morto a um kill-ring. Para recuperar o texto apagado do kill-ring use o comando M-y repetidamente até que o texto desejado seja puxado.

(Nota: Para a tecla M-y funcionar o comando anterior deve ser um YANK caso contrário não funcionaria)

Desfazer #

Para desfazer algo que você acabou de fazer:

C-_ ou C-x u ou C-/

Pesquisar e substituir #

No Emacs, a ferramenta de pesquisa básica (I-Search) permite pesquisar antes ou depois da localização do cursor.

  • Para procurar sometext após a localização do seu cursor (search-forward) pressione C-s sometext. Se você quiser ir para a próxima ocorrência de sometext, apenas pressione C-s novamente (e assim por diante para as próximas ocorrências). Quando o cursor estiver no local correto, pressione Enter para sair do prompt de pesquisa.

  • Para pesquisar antes da localização do seu cursor (search-backward), use C-r da mesma forma que usou antes.

  • Para mudar de search-backward para search-forward, pressione 2 vezes C-s. E pressione 2 vezes C-r para pesquisar para trás quando estiver no prompt procurar para frente.

  • Pesquise e substitua:

    M-% (or Esc-% ) oldtext Enter newtext Enter

    • Confirm: y
    • Skip: n
    • Quit: q
    • Replace all: !

Caso #

  • Palavra maiúscula: M-c

  • Converter palavra para maiúscula: M-u

  • Converter palavra para minúscula: M-l
