Janelas #

  1. Verifique se você tem o nodeJS LTS mais recente instalado
  2. Abra o prompt de comando e digite $ npm install -g nativescript
  3. No prompt de comando, digite $ @powershell -NoProfile -ExecutionPolicy Bypass -Command "iex ((new-object net.webclient).DownloadString('https://www.nativescript.org/setup/win'))" - isso pode demorar um pouco
  4. Para verificar se o acima funcionou, digite $ tns doctor no prompt de comando (seu cmd)
  5. Se houver algum erro, consulte o guia de solução de problemas

Instalação ou configuração #

Instruções detalhadas sobre como configurar ou instalar o Nativescript.

Os exemplos a seguir mostram as etapas necessárias para configurar um sistema Windows ou OSX e, em seguida, assinar os guias de solução de problemas caso você tenha algum problema.

Além disso, há exemplos de como configurar fluxos de trabalho, IDEs e emuladores recomendados.

Mac OS #

  1. Certifique-se de ter o mais recente Node.js LTS instalado. Se você usa Homebrew isso pode ser feito com brew install node4-lts.
  2. Abra o Terminal e digite npm install -g nativescript. Se você receber um erro EACCES, use sudo npm install -g nativescript.
  3. No prompt de comando digite ruby -e "$(curl -fsSL https://www.nativescript.org/setup/mac)". (Isso pode demorar um pouco.)
  4. Para verificar se o procedimento acima funcionou, digite tns doctor no Terminal.
  5. Se houver algum erro, consulte o guia de solução de problemas.

Usando o Visual Studio Code para desenvolvimento NativeScript #

Visual Studio Code é um editor de código de código aberto e rico em recursos da Microsoft. Para configurá-lo para desenvolvimento NativeScript, abra a Paleta de Comandos (F1 ou ⌘+Shift+P) e digite ext install NativeScript.

Depois que a extensão NativeScript for instalada, o depurador deverá permitir que você defina pontos de interrupção em seu código. Quando um dispositivo está conectado ou um emulador está em execução, você pode iniciar seu aplicativo na guia Depurar.

Seu primeiro programa Hello World #

$ mkdir hello-world
$ cd hello-world
$ tns create hello-world --ng
$ tns platform add android #You can only add ios on an OSX machine

Em seguida, verifique se você tem um dispositivo conectado ou um emulador em execução (se não tiver, o emulador padrão deve iniciar ou um erro será gerado. Eu recomendaria o genymotion para Android).

$ tns run android

Se você quiser usar o emulador Android padrão, adicione o sinalizador --emulator.

A partir do tns 2.5 livesync agora é a ação padrão para tns run <platform>, que irá recompilar automaticamente quando você salvar as alterações do arquivo. Isso pode melhorar drasticamente seu tempo de desenvolvimento, no entanto, se você fizer alterações em seus plugins, precisará recompilar corretamente.

Como depurar o aplicativo nativo-android via WiFi (sem Root) #

1-Você precisa conectar seu dispositivo ao seu computador via cabo USB. Verifique se a depuração USB está funcionando. Você pode verificar se ele aparece ao executar adb devices(ou tns device).

2-Execute o adb tcpip 5555

3-Desligue o seu aparelho (retire o cabo USB).

4- Vá em Configurações -> Sobre o telefone -> Status para visualizar o endereço IP do seu telefone.

5-Execute adb connect <endereço IP do seu dispositivo>:5555

6-Se você executar adb devices (ou tns device) novamente, você deverá ver seu dispositivo.

7- Agora você pode usar os comandos tns run android, tns livesync android.

NOTAS :

1-quando a rede WiFi muda, você não precisa repetir as etapas 1 a 3 (estas configuram seu telefone no modo de depuração wifi). Você precisa se conectar ao seu telefone novamente executando as etapas 4 a 6.

2-Os telefones Android perdem o modo de depuração wifi ao reiniciar. Assim, se sua bateria morreu, você deve começar de novo. Caso contrário, se você ficar de olho na bateria e não reiniciar o telefone, poderá viver sem cabo por semanas!

AVISO :

deixar a opção habilitada é perigoso, qualquer pessoa da sua rede pode se conectar ao seu aparelho em depuração, mesmo estando em rede de dados. Faça isso apenas quando estiver conectado a um Wi-Fi confiável e lembre-se de desconectá-lo quando terminar!
