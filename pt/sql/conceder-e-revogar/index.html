CONCEDER E REVOGAR

  • Sintaxe
  • Conceder/revogar privilégios

Nesta página

  • Sintaxe
  • Conceder/revogar privilégios

Sintaxe #

  • GRANT [privilégio1] [, [privilégio2] … ] ON [tabela] PARA [concedido1] [, [concedido2] … ] [ COM OPÇÃO GRANT ]
  • REVOKE [privilégio1] [, [privilégio2] … ] ON [tabela] DE [beneficiário1] [, [beneficiário2] … ]

Conceda permissões aos usuários. Se a opção WITH GRANT OPTION for especificada, o beneficiário também ganha o privilégio de conceder a permissão fornecida ou revogar as permissões concedidas anteriormente.

Conceder/revogar privilégios #

GRANT SELECT, UPDATE
ON Employees
TO User1, User2;

Conceda permissão User1 e User2 para executar as operações SELECT e UPDATE na tabela Employees.

REVOKE SELECT, UPDATE
ON Employees
FROM User1, User2;

Revogue de User1 e User2 a permissão para realizar operações SELECT e UPDATE na tabela Employees.
