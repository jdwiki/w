Iniciando um aplicativo Ionic #

Iniciando um aplicativo Ionic #

$ ionic start myapp [template]

Os modelos iniciais podem vir de um modelo nomeado, de um repositório do Github, de um Codepen ou de um diretório local. Um modelo inicial é o que se torna o diretório www dentro do projeto Cordova.

Iniciadores de modelos nomeados #

  • tabs (Padrão)

  • sidemenu

  • em branco

Iniciadores do Github Repo #

  • Qualquer URL de repositório do Github, ex: https://github.com/driftyco/ionic-starter-tabs
  • Modelos nomeados são simplesmente aliases para repositórios iniciais Ionic

Iniciadores de URL Codepen #

  • Qualquer URL do Codepen, ex: http://codepen.io/ionic/pen/odqCz

  • Demos do Ionic Codepen

Iniciadores do diretório local: #

  • Caminho relativo ou absoluto para um diretório local

Sinalizadores/opções de linha de comando: #

--appname, -a ....... Human readable name for the app (Use quotes around the name) 

--id, -i ............ Package name set in the <widget id> config ex: com.mycompany.myapp 

--no-cordova, -w .... Do not create an app targeted for Cordova
