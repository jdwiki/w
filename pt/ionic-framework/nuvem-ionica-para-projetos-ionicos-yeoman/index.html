Ionic Platform (Ionic Cloud) para Yo (Yeoman) Ionic Projects #

Plataforma Iônica:

Crie, envie, implante e dimensione seus aplicativos Ionic da maneira mais fácil.

Descrição do título:
A Ionic Platform é uma plataforma em nuvem para gerenciar e dimensionar aplicativos móveis multiplataforma. Os serviços integrados permitem que você e sua equipe criem, implantem e desenvolvam seus aplicativos com eficiência.

Objetivo do documento:
A Ionic Platform funciona bem com os projetos Ionic padrão. Mas os projetos que seguem qualquer Estrutura de Diretório Não Padrão podem enfrentar alguns obstáculos. Este documento fornece as etapas para usar o Ionic Platform nos projetos Ionic criados usando Yeoman.

Escopo do documento:
Este documento cobre as etapas básicas para criar um projeto Ionic usando Yeoman e integrá-lo com a Ionic Platform usando o Ionic Platform Web Client. Este documento abrange as etapas básicas para utilizar Ionic Deploy, Ionic Analytics e Ionic Push.

Audiência pretendida:
O público-alvo deste documento são desenvolvedores de aplicativos da Web/Mobile, com experiência de nível iniciante e especialista, que estão familiarizados com os pré-requisitos abaixo.

Pré-requisitos:
Você deve estar familiarizado com as seguintes estruturas/ferramentas antes de tentar este documento.

  • AngularJs: https://docs.angularjs.org/guide
  • IonicFramework: http://ionicframework.com/docs/guide
  • Yeoman: http://yeoman.io/codelab/index.html
  • Gerador Iônico: https://github.com/diegonetto/generator-ionic
  • Plataforma Ionic: https://ionic.io/platform

Gerador de estrutura iônica #

Um gerador para o Ionic Framework da Yeoman, a ferramenta Scaffolding da Web para webapps modernos

Node.js é um runtime JavaScript construído no mecanismo JavaScript V8 do Chrome. npm é o gerenciador de pacotes para JavaScript. Baixe e instale o Node (e npm) de http://nodejs.org

$ npm install npm –g
$ npm install -g yo

Yeoman ajuda você a iniciar novos projetos, prescrevendo as melhores práticas e ferramentas para ajudá-lo a se manter produtivo.

$ yo ionic [app-name]

Em package.json inclua o seguinte em devDependencies

"grunt-string-replace": "^1.2.1"

Em bower.json inclua o seguinte nas dependências

"ionic-platform-web-client": "^0.7.1"

Em Gruntfile.js altere a pasta scripts para ‘js’. Altere em index.html também, se necessário.

grunt.initConfig({   yeoman: {…………
    scripts: 'js',
    ………… } })

Então corra

$ bower install && npm install
$ grunt
$ grunt serve

$ cordova platform add android 
$ grunt build:android --debug

ionic-platform-web-client #

Um cliente web que fornece interações com a plataforma Ionic.

Precisamos de algum código para permitir que seu aplicativo se comunique com a Ionic Platform. Precisamos adicionar o cliente web da plataforma Ionic para que o aplicativo Ionic faça interface com os plugins e a plataforma Ionic.io.

$ ionic io init

Em seu app.js, adicione a dependência do módulo ‘ionic.service.core’. Em Gruntfile.js adicione a tarefa grunt ‘ionicSettings’ conforme indicado abaixo.

grunt.initConfig({
ionicSettings: JSON.stringify(grunt.file.readJSON('./.io-config.json')),

ionicIoBundlePath: 'www/bower_components/ionic-platform-web-client/dist/ionic.io.bundle.min.js',

'string-replace': {
  ionicSettings: {
    files: {
      '<%= ionicIoBundlePath %>': '<%= ionicIoBundlePath %>',
    },
    options: {
      replacements: [
        {
          pattern: 
        '"IONIC_SETTINGS_STRING_START";"IONIC_SETTINGS_STRING_END"',
          replacement: 
        '"IONIC_SETTINGS_STRING_START";var settings =<%= ionicSettings %>; return { get: function(setting) { if (settings[setting]) { return settings[setting]; } return null; } };"IONIC_SETTINGS_STRING_END";'
        }
      ]
    }
  }
},
       copy: {
    ionicPlatform:{
                expand: true,
                cwd: 'app/bower_components/ionic-platform-web-client/dist/',
                src: ['**'],
                dest: 'www/bower_components/ionic-platform-web-client/dist'
               }
    }
});

grunt.registerTask('ionicSettings', ['copy:ionicPlatform','string-replace:ionicSettings']);
