O Ionic tem muitos componentes CSS declarados prontos para facilitar sua vida ao codificar seu próximo aplicativo móvel híbrido. Esses componentes variam de um sistema de grade básico ao estilo de seus formulários. Esses componentes estão em seu uso se você optar por instalar o Ionic com as folhas de estilo CSS predefinidas.

Uma das funções básicas que o Ionic CSS traz à sua mão é que ele vem com um conjunto de cores para começar, mas como regra geral as cores devem ser substituídas. Cores utilitárias são adicionadas para ajudar a definir uma convenção de nomenclatura. Você poderia chamá-lo de um tema básico do aplicativo. Para personalizar as cores, você pode simplesmente substituir as que vêm do arquivo CSS ionic.css. Além disso, como o Ionic é construído usando Sass, para mais poder e flexibilidade, você também pode modificar e estender as variáveis ​​de cor dentro do arquivo _variables.scss.

Você pode configurar um projeto Ionic para usar SASS com muita facilidade executando o comando ionic setup sass em seu terminal.

Você pode encontrar a documentação oficial sobre Ionic CSS aqui: http://ionicframework.com/docs/components/

Sintaxe básica do sistema de grade #

Grade básica #

No Ionic você pode declarar linhas definindo a classe row para um elemento. As linhas serão elementos alinhados horizontalmente e qualquer coisa dentro desse elemento pertencerá à linha. Dentro da linha, você pode declarar colunas de largura diferentes. Você tem uma escolha das seguintes declarações.

Classe Largura
.col-10 10%
.col-20 20%
.col-25 25%
.col-33 33,3333%
.col-50 50%
.col-67 66,6666%
.col-75 75%
.col-80 80%
.col-90 90%

O valor máximo que as colunas podem ter dentro de uma linha é 100. Aqui estão alguns exemplos da grade básica.

<div class="row">
  <div class="col col-50">.col.col-50</div>
  <div class="col">.col</div>
  <div class="col">.col</div>
</div>

<div class="row">
  <div class="col col-75">.col.col-75</div>
  <div class="col">.col</div>
</div>

<div class="row">
  <div class="col">.col</div>
  <div class="col col-75">.col.col-75</div>
</div>

<div class="row">
  <div class="col">.col</div>
  <div class="col">.col</div>
</div>

Grades de deslocamento #

Você também pode definir col-offset-<value> para as colunas. No exemplo abaixo, um terço de uma coluna de largura tem um terço do deslocamento de largura, o que significa que terá um terço de largura e será centralizado na página devido ao deslocamento.

<div class="row">
  <div class="col col-33 col-offset-33">.col</div>
  <div class="col">.col</div>
</div>

Alinhar colunas #

Alinhar colunas verticalmente é possivelmente definindo o col-<align_value> para uma coluna separadamente como esta.

<div class="row">
  <div class="col col-top">.col</div>
  <div class="col col-center">.col</div>
  <div class="col col-bottom">.col</div>
  <div class="col">1<br>2<br>3<br>4</div>
</div>

O acima irá alinhar cada coluna por conta própria. Se você deseja alinhar todas as colunas dentro da linha, você pode definir o valor de alinhamento para a própria linha. No exemplo abaixo, todas as colunas dentro desta linha se alinharão verticalmente no centro da linha.

<div class="row row-center">
  <div class="col">.col</div>
  <div class="col">.col</div>
  <div class="col">.col</div>
  <div class="col">1<br>2<br>3<br>4</div>
</div>

Grade responsiva #

Você também pode querer que as colunas sejam responsivas, pois elas serão empilhadas umas sobre as outras em alguma largura da janela de visualização. Você tem três escolhas.

Classe Ponto de interrupção (aproximadamente)
.responsive-sm Menor do que o telefone paisagem
.responsive-md Menor do que o tablet retrato
.responsive-lg Menor do que o tablet paisagem

Neste exemplo, essas colunas serão empilhadas na largura de aproximadamente um telefone em modo paisagem.

<div class="row responsive-sm">
  <div class="col">.col</div>
  <div class="col">.col</div>
  <div class="col">.col</div>
  <div class="col">.col</div>
</div>

Você também pode fazer suas próprias consultas de mídia, é claro, se esses pontos de interrupção não forem adequados para você e/ou você precisar de pontos de interrupção mais específicos.

Sintaxe básica do item de lista #

Quase todos os aplicativos têm algum tipo de lista. O Ionic tem suas próprias declarações CSS de item de lista prontas para uso para facilitar a criação de listas dentro de seu aplicativo. Você pode usar elementos HTML e declarar uma classe para o ou usar a diretiva ion-list para criá-los. Exemplo de uma diretiva está na parte inferior.

Sintaxe CSS básica do item de lista:

<ul class="list">
  <li class="item"></li>
</ul>

Lista com divisórias:

<div class="list">
  <a class="item" href="#">
    List item
  </a>
  <div class="item item-divider">
    Divider that looks a bit different from items
  </div>
  <a class="item" href="#">
    Another list item
  </a>
</div>

Listar itens com ícones:

<div class="list">
  <a class="item item-icon-left" href="#">
    <i class="icon ion-trash-b"></i>
    List item with a trashcan icon on the left
  </a>
</div>

Você também pode definir ícones em ambos os lados dos itens com a seguinte sintaxe:

<div class="list">
  <a class="item item-icon-left item-icon-right" href="#">
    <i class="icon ion-trash-b"></i>
    List item with a trashcan icon on the left and a briefcase icon on the right
    <i class="icon ion-briefcase"></i>
  </a>
</div>

Um item de lista com botão ou botões pode ser criado assim:

<div class="list">
  <div class="item item-button-right">
    Item with a button on the right that has a clock icon in it
    <button class="button button-positive">
      <i class="icon ion-clock"></i>
    </button>
  </div>
</div>

Também é possível criar itens de lista com avatares, miniaturas e inserções que criarão preenchimento ao redor dos itens da lista. O Ionic também lida com a configuração de ícones, etc., nos itens da lista, definindo o preenchimento de acordo com os itens da lista.

Ionic também tem suas próprias diretivas para checkboxes, radio buttons, etc. Aqui está um exemplo de uma lista de checkbox com Ionic.

<ion-list>
  <ion-checkbox ng-model="choice1">Choice 1</ion-checkbox>
  <ion-checkbox ng-model="choice2">Choice 2</ion-checkbox>
</ion-list>

Uso básico de cores utilitárias #

O CSS Ionic predefinido terá um tema e cores predefinidas para ele. Você pode modificar ou substituir os valores básicos no ionic.css ou em seu arquivo CSS personalizado. Você também pode defini-los com SASS e para usar SASS no Ionic você só precisa executar o comando ionic setup sass em seu terminal.

Uso básico de cores em um botão. A classe button-<phrase> fará com que o fundo do botão e as bordas sejam da cor do tema definido.

<button class="button button-positive">
  button-positive
</button>

<button class="button button-calm">
  button-calm
</button>

<button class="button button-balanced">
  button-balanced
</button>

Suas opções de prefixo CSS são as seguintes:

  • <elemento>-luz

  • <elemento>-estável

  • <elemento>-positivo

  • <elemento>-calma

  • <elemento>-balanceado

  • <elemento>-energizado

  • <elemento>-assertivo

  • <elemento>-royal

  • <elemento>-escuro

Essas classes também podem ser adicionadas, por exemplo, em crachás, etc. Aqui está um exemplo de crachá:

<span class="badge badge-positive">Positive badge</span>
