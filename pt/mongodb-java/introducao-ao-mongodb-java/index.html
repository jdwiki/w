Instalação ou configuração #

Pré-requisito

Um servidor MongoDB em execução, mais detalhes sobre isso aqui.

Configurando um projeto Java

O MongoDB Java Driver é fornecido como um arquivo JAR e pode ser incluído em um projeto como qualquer outro JAR. Por exemplo:

  • Maven: adicione uma entrada na seção <dependencies> do seu pom.xml.

<dependência> org.mongodb mongo-java-driver 3.4.2 </dependência>

  • Gradle: adicione a seguinte dependência ao seu build.gradle

compilar 'org.mongodb:mongo-java-driver:3.4.2'

  • Inclusão Manual: baixe o arquivo JAR aqui e adicione-o ao classpath do seu projeto.

Nota: a versão mostrada nos exemplos acima é a versão estável mais recente no momento da redação, mas você pode escolher qualquer versão disponível [aqui](http://search.maven.org/#search%7Cga%7C1%7Corg.mongodb %3A%3Amongo-java-driver).

Usando o Driver Java MongoDB

Vamos criar um MongoClient, a invocação mais simples é:

MongoClient mongoClient = new MongoClient(new ServerAddress("localhost", 27017));

Isso pressupõe que o servidor MongoDB está sendo executado no host local na porta padrão e que não é seguro. Existem inúmeras variantes disso para acesso autenticado, opções de conexão personalizadas, conexão a conjuntos de réplicas e clusters fragmentados, todos abordados em detalhes nos [documentos do Mongo DB Java Driver](http://mongodb.github.io/mongo-java-driver/ 3.4/driver/tutorials/connect-to-mongodb/).

Agora que estamos conectados, vamos criar um banco de dados e uma coleção. O MongoDB cria implicitamente uma coleção quando o primeiro documento é salvo nela, então vamos fazer isso:

MongoDatabase database = mongoClient.getDatabase("starter");
MongoCollection<Document> collection = database.getCollection("superheroes");
Document document = new Document("name", "Superman");
collection.insertOne(document);

Nota: você pode criar explicitamente uma coleção, mas o uso acima é mais comum e certamente se encaixa melhor com um tópico ‘começando’.

Agora vamos ver se podemos encontrar o documento que criamos:

FindIterable<Document> documents = purchases.find(Filters.eq("name", "Superman"));
for (Document found : documents) {
    System.out.println(found.toJson());
}

O código acima produzirá o seguinte (o campo _id é a chave de documento gerada automaticamente pelo MongoDB):

Documento encontrado: { “_id” : { “$oid” : “5981bd586d47c203904a9cf9” }, “name” : “Superman” }

Agora que verificamos que criamos um documento com sucesso, vamos excluí-lo:

DeleteResult deleteResult = purchases.deleteOne(Filters.eq("name", "Superman"));
System.out.println(String.format("Deleted %s document(s)", deleteResult.getDeletedCount()));

O código acima produzirá o seguinte:

Excluiu 1 documento(s)

Exemplo Completo

Juntando as várias peças obtemos:

import com.google.common.collect.Lists;
import com.mongodb.MongoClient;
import com.mongodb.MongoCredential;
import com.mongodb.ServerAddress;
import com.mongodb.client.FindIterable;
import com.mongodb.client.MongoCollection;
import com.mongodb.client.model.Filters;
import com.mongodb.client.result.DeleteResult;
import org.bson.Document;

import java.util.List;

public class IntroducingTheMongoDBJavaDriver {

    public static void main(String... args) throws Exception {
        MongoClient mongoClient = new MongoClient(new ServerAddress("localhost", 27017));

        MongoCollection<Document> superheroes = mongoClient.getDatabase("starter").getCollection("superheroes");

        Document document = new Document("name", "Superman");
        superheroes.insertOne(document);

        FindIterable<Document> documents = superheroes.find(Filters.eq("name", "Superman"));
        for (Document found : documents) {
            System.out.println(String.format("Found document: %s", found.toJson()));
        }

        DeleteResult deleteResult = superheroes.deleteOne(Filters.eq("name", "Superman"));
        System.out.println(String.format("Deleted %s document(s)", deleteResult.getDeletedCount()));

        long count = superheroes.count();
        System.out.println(String.format("Number of superheroes: %s", count));
    }
}
