Instalação ou Configuração #

No Windows #

  • Baixe o Visual Studio Code instalador para Windows.
  • Uma vez baixado, execute o instalador (VSCodeSetup-version.exe). Isso levará apenas um minuto.

Por padrão, o VS Code é instalado em C:\Program Files (x86)\Microsoft VS Code para uma máquina de 64 bits.

Observação: o .NET Framework 4.5.2 é necessário para o VS Code. Se você estiver usando o Windows 7, certifique-se de que o .NET Framework 4.5.2 esteja instalado.

Dica: a instalação adicionará opcionalmente o Visual Studio Code ao seu %PATH%, portanto, no console, você pode digitar ‘code .’ para abrir o VS Code nessa pasta. Você precisará reiniciar seu console após a instalação para que a alteração na variável de ambiente %PATH% tenha efeito.

No Mac #

  • Baixe o código do Visual Studio para Mac.
  • Clique duas vezes no arquivo baixado para expandir o conteúdo.
  • Arraste o Visual Studio Code.app para a pasta Aplicativos, disponibilizando-o no Launchpad.
  • Adicione o VS Code ao seu Dock clicando com o botão direito do mouse no ícone e escolhendo Opções, Manter no Dock.

Você também pode executar o VS Code no terminal digitando ‘code’ depois de adicioná-lo ao caminho:

  • Inicie o VS Code.
  • Abra a Paleta de Comandos (Ctrl+Shift+P) e digite ‘shell command’ para encontrar o Shell Command: Instale o comando ‘code’ no comando PATH.

Reinicie o terminal para que o novo valor $PATH tenha efeito. Você poderá digitar ‘código’. em qualquer pasta para começar a editar arquivos nessa pasta.

Observação: se você ainda tiver o alias de código antigo em seu .bash_profile (ou equivalente) de uma versão anterior do VS Code, remova-o e substitua-o executando o comando Shell: Instalar o comando ‘code’ no comando PATH.

No Linux #

Distribuições baseadas em Debian e Ubuntu #

A maneira mais fácil de instalar para distribuições baseadas em Debian/Ubuntu é baixar e instalar o pacote .deb (64 bits) através do centro gráfico de software se estiver disponível ou através da linha de comando com:

sudo dpkg -i <file>.deb
sudo apt-get install -f # Install dependencies

A instalação do pacote .deb instalará automaticamente o repositório apt e a chave de assinatura para habilitar a atualização automática usando o mecanismo normal do sistema. Observe que os binários de 32 bits e .tar.gz também estão disponíveis na página de download.

O repositório e a chave também podem ser instalados manualmente com o seguinte script:

curl https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > microsoft.gpg
sudo mv microsoft.gpg /etc/apt/trusted.gpg.d/microsoft.gpg
sudo sh -c 'echo "deb [arch=amd64] https://packages.microsoft.com/repos/vscode stable main" > /etc/apt/sources.list.d/vscode.list'

Em seguida, atualize o cache do pacote e instale o pacote usando:

sudo apt-get update
sudo apt-get install code # or code-insiders for insiders build

Distribuições baseadas em RHEL, Fedora e CentOS #

Atualmente, enviamos o VS Code estável de 64 bits em um repositório yum, o script a seguir instalará a chave e o repositório:

sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
sudo sh -c 'echo -e "[code]\nname=Visual Studio Code\nbaseurl=https://packages.microsoft.com/yumrepos/vscode\nenabled=1\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" > /etc/yum.repos.d/vscode.repo'

Em seguida, atualize o cache do pacote e instale o pacote usando dnf (Fedora 22 e superior):

dnf check-update
sudo dnf install code

Ou em versões mais antigas usando yum:

yum check-update
sudo yum install code

Distribuições baseadas em openSUSE e SLE #

O repositório yum acima também funciona para sistemas baseados em openSUSE e SLE, o seguinte script instalará a chave e o repositório:

sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
sudo sh -c 'echo -e "[code]\nname=Visual Studio Code\nbaseurl=https://packages.microsoft.com/yumrepos/vscode\nenabled=1\ntype=rpm-md\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" > /etc/zypp/repos.d/vscode.repo'

Em seguida, atualize o cache do pacote e instale o pacote usando:

sudo zypper refresh
sudo zypper install code

Pacote AUR para Arch Linux #

Existe um [pacote Arch User Repository (AUR)] mantido pela comunidade] 4 para o VS Code.

Instalando o pacote .rpm manualmente O pacote .rpm (64 bits) também pode ser baixado e instalado manualmente, porém a atualização automática não funcionará a menos que o repositório acima esteja instalado. Uma vez baixado, pode ser instalado usando seu gerenciador de pacotes, por exemplo, com dnf:

sudo dnf install <file>.rpm

Observe que os binários de 32 bits e .tar.gz também estão disponíveis na página de download.

Primeiros passos (C++): HelloWorld.cpp #

O primeiro programa que se escreve em qualquer linguagem é o script “hello world”. Este exemplo demonstra como escrever este programa e depurá-lo usando Visual Studio Code (vou me referir ao Visual Studio Code como VS Code a partir de agora).

Crie o projeto

O passo 1 será criar um novo projeto. Isso pode ser feito de várias maneiras. A primeira maneira é diretamente da interface do usuário.

  1. Abra o programa VS Code. Você será recebido com a tela de boas-vindas padrão (observe que as imagens são tiradas enquanto trabalha em um Mac, mas devem ser semelhantes à sua instalação):

  1. No menu Iniciar, selecione Novo arquivo. Isso abrirá uma nova janela de edição onde podemos começar a construir nosso script. Vá em frente e salve este arquivo (você pode usar o menu File > Save para fazer isso). Para este exemplo, chamaremos o arquivo HelloWorld.cpp e o colocaremos em um novo diretório que chamaremos de VSC_HelloWorld/.

  2. Escreva o programa. Isso deve ser bastante simples, mas sinta-se à vontade para copiar o seguinte no arquivo:

    #include

    int main() { std::cout « “Hello world!” « std::endl; return 0; }

Execute o código

Em seguida, queremos executar o script e verificar sua saída. Existem várias maneiras de fazer isso. O mais simples é abrir um terminal e navegar até o diretório que criamos. Agora você pode compilar o script e executá-lo com gcc digitando:

$ g++ HelloWorld.cpp -o helloworld
$ ./helloworld
Hello World!

Eba, o programa funcionou! Mas isso não é realmente o que queremos. Seria muito melhor se pudéssemos executar o programa de dentro do próprio VSCode. Mas estamos com sorte! O VSCode possui um terminal integrado que podemos acessar através do menu “View” > “Integrated Terminal”. Isso abrirá um terminal na metade inferior da janela a partir do qual você pode navegar até o diretório VSC_HelloWorld e executar os comandos acima.

Normalmente fazemos isso executando um Run Task. No menu, selecione “Tarefas” > “**Executar tarefa…”. Você notará um pequeno pop-up próximo ao topo da janela com uma mensagem de erro (algo como

Primeiro programa (C++): Hello World.cpp #

Este exemplo apresenta a funcionalidade básica do VS Code demonstrando como escrever um programa “hello world” em C++. Antes de continuar, verifique se a extensão “ms-vscode.cpptools” está instalada.

Iniciar o projeto

O primeiro passo é criar um novo projeto. Para fazer isso, carregue o programa VS Code. Você deve ser saudado com a tela de boas-vindas típica:

Para criar o primeiro programa, selecione “Iniciar” > “Novo arquivo” na tela de boas-vindas. Isso abrirá uma nova janela de arquivo. Vá em frente e salve o arquivo ("File" > “Save”) em um novo diretório. Você pode nomear o diretório como quiser, mas este exemplo chamará o diretório “VSC_HelloWorld” e o arquivo “HelloWorld.cpp”.

Agora escreva o programa real (sinta-se à vontade para copiar o texto abaixo):

#include <iostream>

int main()
{
    // Output the hello world text
    std::cout << "Hello world!" << std::endl;
    return 0;
}

Excelente! Você também notará que, como instalou a extensão “ms-vscode.cpptools”, você também tem um realce de código bastante. Agora vamos seguir para a execução do código.

Executando o script (básico)

Podemos executar “HelloWorld.cpp” de dentro do próprio VS Code. A maneira mais simples de executar tal programa é abrir o terminal integrado ("Exibir" > “Terminal Integrado”). Isso abre uma janela de terminal na parte inferior da visualização. De dentro deste terminal, podemos navegar para nosso diretório criado, construir e executar o script que escrevemos. Aqui usamos os seguintes comandos para compilar e executar o código:

$ g++ HelloWorld.cpp -o hellowold
$ ./hellowold

Observe que obtemos a saída esperada Hello World!.

Executando o Script (um pouco mais avançado)

Ótimo, mas podemos usar o VS Code diretamente para construir e executar o código também. Para isso, primeiro precisamos transformar o diretório “VSC_HelloWorld” em um espaço de trabalho. Isso pode ser feito por:

  1. Abrindo o menu Explorer (item mais alto no menu vertical na extrema esquerda)
  2. Selecione o botão Abrir pasta
  3. Selecione o diretório “VSC_HelloWorld” no qual estamos trabalhando. Observação: se você abrir um diretório dentro do VS Code (usando “File” > “Open…” por exemplo), você já estará em um workspace.

O menu Explorer agora exibe o conteúdo do diretório.

Em seguida, queremos definir as tarefas reais que queremos que o VS Code execute. Para fazer isso, selecione “Tasks” > “Configure Default Build Task”. No menu suspenso, selecione “Outro”. Isso abre um novo arquivo chamado “tasks.json” que contém alguns valores padrão para uma tarefa. Precisamos mudar esses valores. Atualize este arquivo para conter o seguinte e salve-o:

{
    "version": "2.0.0",
    "tasks": [
        {
            "taskName": "build",
            "type": "shell",
            "command": "g++ HelloWorld.cpp -o helloworld"
        },
        {
            "taskName": "run",
            "type": "shell",
            "command": "${workspaceRoot}/helloworld"
        }
    ]
}

Observe que o acima também cria um diretório .vscode oculto em nosso diretório de trabalho. É aqui que o VS Code coloca os arquivos de configuração, incluindo qualquer arquivo de configuração específico do projeto. Você pode descobrir mais sobre Tasks aqui.

No exemplo acima, ${workspaceRoot} faz referência ao diretório de nível superior do nosso espaço de trabalho, que é nosso diretório “VSC_HelloWorld”. Agora, para construir o projeto de dentro do método selecione “Tasks” > “Run Build Task…” e selecione nossa tarefa “build” criada e “Continue without scan the task output” nos menus suspensos que aparecem. Em seguida, podemos executar o executável usando “Tasks” > “Run Task…” e selecionando a tarefa “run” que criamos. Se você tiver o terminal integrado aberto, notará que a mensagem “Hello World!” texto será impresso lá.

É possível que o terminal feche antes que você possa visualizar a saída. Se isso acontecer, você pode inserir uma linha de código como esta int i; std::cin >> i; logo antes da instrução return no final da função main(). Você pode encerrar o script digitando qualquer número e pressionando <Enter>.

E é isso! Agora você pode começar a escrever e executar seus scripts C++ a partir do VS Code.
