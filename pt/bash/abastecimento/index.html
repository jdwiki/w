Obtendo um arquivo #

O fornecimento de um arquivo é diferente da execução, pois todos os comandos são avaliados no contexto da sessão atual do bash - isso significa que quaisquer variáveis, funções ou aliases definidos persistirão durante toda a sessão.

Crie o arquivo que você deseja como fonte sourceme.sh

#!/bin/bash

export A="hello_world"
alias sayHi="echo Hi"
sayHello() {
    echo Hello
}

Da sua sessão, origine o arquivo

$ source sourceme.sh

A partir de agora, você tem todos os recursos do arquivo de origem disponíveis

$ echo $A
hello_world

$ sayHi
Hi

$ sayHello
Hello

Observe que o comando . é sinônimo de source, de modo que você pode simplesmente usar

$ . sourceme.sh

Fornecimento de um ambiente virtual #

Ao desenvolver vários aplicativos em uma máquina, torna-se útil separar as dependências em ambientes virtuais.

Com o uso de virtualenv, esses ambientes são originados em seu shell para que, quando você executar um comando, ele venha desse ambiente virtual.

Isso é mais comumente instalado usando pip.

pip install https://github.com/pypa/virtualenv/tarball/15.0.2

Crie um novo ambiente

virtualenv --python=python3.5 my_env

Ative o ambiente

source my_env/bin/activate
