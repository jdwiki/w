Sintaxe #

  • ls [OPÇÃO]… [ARQUIVO]…

Parâmetros #

Opção Descrição
-a, --all Listar todas as entradas, incluindo aquelas que começam com um ponto
-A, --quase-tudo Listar todas as entradas excluindo . e ..
-c Ordenar arquivos por tempo de mudança
-d, --diretório Listar entradas de diretório
-h, --human-legível Mostrar tamanhos em formato legível por humanos (ou seja, K, M)
-H Igual ao anterior apenas com potências de 1000 em vez de 1024
-l Mostrar conteúdo em formato de listagem longa
-o Formato de listagem longa sem informações de grupo
-r, --reverse Mostrar conteúdo em ordem inversa
-s, --size Tamanho de impressão de cada arquivo em blocos
-S Classificar por tamanho de arquivo
--sort=PALAVRA Classifique o conteúdo por uma palavra. (ou seja, tamanho, versão, status)
-t Ordenar por hora de modificação
-u Ordenar por hora do último acesso
-v Ordenar por versão
-1 Listar um arquivo por linha

Listar arquivos em um formato de listagem longo #

A opção -l do comando ls imprime o conteúdo de um diretório especificado em um formato de listagem longa. Se nenhum diretório for especificado, por padrão, o conteúdo do diretório atual será listado.

ls -l /etc

Saída de exemplo:

total 1204
drwxr-xr-x  3 root root    4096 Apr 21 03:44 acpi
-rw-r--r--  1 root root    3028 Apr 21 03:38 adduser.conf
drwxr-xr-x  2 root root    4096 Jun 11 20:42 alternatives
...

A saída primeiro exibe total, que indica o tamanho total em blocos de todos os arquivos no diretório listado. Em seguida, ele exibe oito colunas de informações para cada arquivo no diretório listado. Abaixo estão os detalhes de cada coluna na saída:

Coluna nº Exemplo Descrição
1.1 d Tipo de arquivo (veja a tabela abaixo)
1.2 rwxr-xr-x String de permissão
2 3 Número de links físicos
3 raiz Nome do proprietário
4 root Grupo de proprietários
5 4096 Tamanho do arquivo em bytes
6 21 de abril 03:44 Hora da modificação
7 acpi Nome do arquivo

Tipo de arquivo #

O tipo de arquivo pode ser um dos seguintes caracteres.

Personagem Tipo de arquivo
- Arquivo normal
b Bloquear arquivo especial
c Arquivo especial de caractere
C Arquivo de alto desempenho (“dados contíguos”)
d Diretório
D Porta (arquivo IPC especial apenas no Solaris 2.5+)
l Link simbólico
M Arquivo off-line (“migrado”) (Cray DMF)
n Arquivo especial de rede (HP-UX)
p FIFO (pipe nomeado)
P Porta (arquivo de sistema especial apenas no Solaris 10+)
s Soquete
? Algum outro tipo de arquivo

Liste os dez arquivos modificados mais recentemente #

A seguir, listará até dez dos arquivos modificados mais recentemente no diretório atual, usando um formato de listagem longa (-l) e ordenado por tempo (-t).

ls -lt | head

Lista todos os arquivos, incluindo arquivos Dot #

Um dotfile é um arquivo cujos nomes começam com um .. Eles são normalmente ocultos por ls e não listados, a menos que solicitados.

Por exemplo, a seguinte saída de ls:

$ ls
bin  pki

A opção -a ou --all listará todos os arquivos, incluindo dotfiles.

$ ls -a
.   .ansible       .bash_logout   .bashrc  .lesshst  .puppetlabs  .viminfo
..  .bash_history  .bash_profile  bin      pki       .ssh

A opção -A ou --almost-all listará todos os arquivos, incluindo dotfiles, mas não listará . e .. implícitos. Observe que . é o diretório atual e .. é o diretório pai.

$ ls -A 
.ansible       .bash_logout   .bashrc  .lesshst  .puppetlabs  .viminfo
.bash_history  .bash_profile  bin      pki       .ssh

Listar arquivos #

O comando ls lista o conteúdo de um diretório especificado, excluindo os dotfiles. Se nenhum diretório for especificado, por padrão, o conteúdo do diretório atual será listado.

Os arquivos listados são classificados em ordem alfabética, por padrão, e alinhados em colunas se não couberem em uma linha.

$ ls
apt  configs  Documents  Fonts    Music      Programming  Templates  workspace
bin  Desktop  eclipse    git      Pictures   Public       Videos

Listar arquivos sem usar ls #

Use a [expansão de nome de arquivo] do shell Bash(https://www.gnu.org/software/bash/manual/bashref.html#Filename-Expansion) e [expansão de chaves](https://www.gnu.org/software /bash/manual/bashref.html#Brace-Expansion) para obter os nomes dos arquivos:

# display the files and directories that are in the current directory
printf "%s\n" *

# display only the directories in the current directory
printf "%s\n" */

# display only (some) image files
printf "%s\n" *.{gif,jpg,png}

Para capturar uma lista de arquivos em uma variável para processamento, normalmente é uma boa prática usar um bash array:

files=( * )

# iterate over them
for file in "${files[@]}"; do
    echo "$file"
done

Listar arquivos em formato de árvore #

O comando tree lista o conteúdo de um diretório especificado em um formato de árvore. Se nenhum diretório for especificado, por padrão, o conteúdo do diretório atual será listado.

Saída de exemplo:

$ tree /tmp
/tmp
├── 5037
├── adb.log
└── evince-20965
    └── image.FPWTJY.png

Use a opção -L do comando tree para limitar a profundidade de exibição e a opção -d para listar apenas diretórios.

Saída de exemplo:

$ tree -L 1 -d /tmp
/tmp
└── evince-20965

Listar arquivos classificados por tamanho #

A opção -S do comando ls classifica os arquivos em ordem decrescente de tamanho de arquivo.

$ ls -l -S ./Fruits
total 444
-rw-rw-rw- 1 root root 295303 Jul 28 19:19 apples.jpg
-rw-rw-rw- 1 root root 102283 Jul 28 19:19 kiwis.jpg
-rw-rw-rw- 1 root root  50197 Jul 28 19:19 bananas.jpg

Quando usado com a opção -r, a ordem de classificação é invertida.

$ ls -l -S -r /Fruits
total 444
-rw-rw-rw- 1 root root  50197 Jul 28 19:19 bananas.jpg
-rw-rw-rw- 1 root root 102283 Jul 28 19:19 kiwis.jpg
-rw-rw-rw- 1 root root 295303 Jul 28 19:19 apples.jpg
