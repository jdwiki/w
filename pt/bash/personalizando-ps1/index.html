Colorir e personalizar o prompt do terminal #

É assim que o autor define sua variável pessoal PS1:

gitPS1(){
    gitps1=$(git branch 2>/dev/null | grep '*')
    gitps1="${gitps1:+ (${gitps1/#\* /})}"
    echo "$gitps1"
}
#Please use the below function if you are a mac user
gitPS1ForMac(){
    git branch 2> /dev/null | sed -e '/^[^*]/d' -e 's/* \(.*\)/ (\1)/'
}
timeNow(){
    echo "$(date +%r)"
}
if [ "$color_prompt" = yes ]; then
  if [ x$EUID = x0 ]; then
    PS1='\[\033[1;38m\][$(timeNow)]\[\033[00m\] \[\033[1;31m\]\u\[\033[00m\]\[\033[1;37m\]@\[\033[00m\]\[\033[1;33m\]\h\[\033[00m\] \[\033[1;34m\]\w\[\033[00m\]\[\033[1;36m\]$(gitPS1)\[\033[00m\] \[\033[1;31m\]:/#\[\033[00m\] '
  else
    PS1='\[\033[1;38m\][$(timeNow)]\[\033[00m\] \[\033[1;32m\]\u\[\033[00m\]\[\033[1;37m\]@\[\033[00m\]\[\033[1;33m\]\h\[\033[00m\] \[\033[1;34m\]\w\[\033[00m\]\[\033[1;36m\]$(gitPS1)\[\033[00m\] \[\033[1;32m\]:/$\[\033[00m\] '
  fi
else
    PS1='[$(timeNow)] \u@\h \w$(gitPS1) :/$ '
fi

Mostra o nome do branch git no prompt do terminal #

Você pode ter funções na variável PS1, apenas certifique-se de aspas simples ou use escape para caracteres especiais:

gitPS1(){
    gitps1=$(git branch 2>/dev/null | grep '*')
    gitps1="${gitps1:+ (${gitps1/#\* /})}"
    echo "$gitps1"
}

PS1='\u@\h:\w$(gitPS1)$ '

Ele lhe dará um prompt como este:

user@Host:/path (master)$

Notas:

  • Faça as alterações no arquivo ~/.bashrc ou /etc/bashrc ou ~/.bash_profile ou ~./profile (dependendo do sistema operacional) e salve-o.
  • Execute source ~/.bashrc (específico da distro) após salvar o arquivo.

Alterar prompt do PS1 #

Para alterar o PS1, basta alterar o valor da variável shell PS1. O valor pode ser definido no arquivo ~/.bashrc ou /etc/bashrc, dependendo da distribuição. PS1 pode ser alterado para qualquer texto simples como:

PS1="hello "

Além do texto simples, vários caracteres especiais com escape de barra invertida são suportados:

Formato Ação
\a um caractere de sino ASCII (07)
\d a data no formato “Weekday Month Date” (por exemplo, “Tue May 26”)
\D{formato} o formato é passado para strftime(3) e o resultado é inserido na string do prompt; um formato vazio resulta em uma representação de hora específica de localidade. As chaves são necessárias
\e um caractere de escape ASCII (033)
\h o nome do host até o primeiro ‘.’
\H o nome do host
\j o número de trabalhos atualmente gerenciados pelo shell
\l o nome base do nome do dispositivo terminal do shell
\n nova linha
\r retorno de carro
\s o nome do shell, o nome base de $0 (a parte após a barra final)
\t a hora atual no formato HH:MM:SS de 24 horas
\T a hora atual no formato HH:MM:SS de 12 horas
\@ a hora atual no formato am/pm de 12 horas
\A a hora atual no formato HH:MM de 24 horas
\u o nome de usuário do usuário atual
\v a versão do bash (por exemplo, 2.00)
\V o lançamento do bash, versão + nível de patch (por exemplo, 2.00.0)
\w o diretório de trabalho atual, com $HOME abreviado com um til
\W o nome base do diretório de trabalho atual, com $HOME abreviado com um til
\! o número do histórico deste comando
\# o número do comando deste comando
\$ se o UID efetivo for 0, um #, caso contrário, um $
\nnn* o caractere correspondente ao número octal nnn
\ uma barra invertida
\[ iniciar uma sequência de caracteres não imprimíveis, que podem ser usados ​​para incorporar uma sequência de controle de terminal no prompt
\] terminar uma sequência de caracteres não imprimíveis

Então, por exemplo, podemos definir PS1 para:

PS1="\u@\h:\w\$ "

E vai sair:

usuário@máquina:~$

Mostre um branch git usando PROMPT_COMMAND #

Se você estiver dentro de uma pasta de um repositório git, pode ser bom mostrar o branch atual em que você está. Em ~/.bashrc ou /etc/bashrc adicione o seguinte (git é necessário para que isso funcione):

function prompt_command {
    # Check if we are inside a git repository
    if git status > /dev/null 2>&1; then
        # Only get the name of the branch
        export GIT_STATUS=$(git status | grep 'On branch' | cut -b 10-)
    else
        export GIT_STATUS=""
    fi
}
# This function gets called every time PS1 is shown
PROMPT_COMMAND=prompt_command

PS1="\$GIT_STATUS \u@\h:\w\$ "

Se estivermos em uma pasta dentro de um repositório git, isso resultará:

branch user@machine:~$

E se estivermos dentro de uma pasta normal:

usuário@máquina:~$

Mostra a hora no prompt do terminal #

timeNow(){
    echo "$(date +%r)"
}
PS1='[$(timeNow)] \u@\h:\w$ '

Ele lhe dará um prompt como este:

[05:34:37 PM] user@Host:/path$

Notas:

  • Faça as alterações no arquivo ~/.bashrc ou /etc/bashrc ou ~/.bash_profile ou ~./profile (dependendo do sistema operacional) e salve-o.
  • Execute source ~/.bashrc (específico da distro) após salvar o arquivo.

Mostra o status e a hora de retorno do comando anterior #

Às vezes, precisamos de uma dica visual para indicar o status de retorno do comando anterior. O trecho a seguir coloca-o na cabeça do PS1.

Observe que a função __stat() deve ser chamada toda vez que um novo PS1 for gerado, caso contrário, ela permanecerá no status de retorno do último comando do seu .bashrc ou .bash_profile.

# -ANSI-COLOR-CODES- #
Color_Off="\033[0m"
###-Regular-###
Red="\033[0;31m"
Green="\033[0;32m"
Yellow="\033[0;33m"
####-Bold-####

function __stat() {
    if [ $? -eq 0 ]; then
        echo -en "$Green ✔ $Color_Off "
    else
        echo -en "$Red ✘ $Color_Off "
    fi
}

PS1='$(__stat)'
PS1+="[\t] "
PS1+="\e[0;33m\u@\h\e[0m:\e[1;34m\w\e[0m \n$ "

export PS1
