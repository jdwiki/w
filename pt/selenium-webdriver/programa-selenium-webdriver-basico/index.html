Este tópico tem como objetivo mostrar o programa básico de driver web em linguagens suportadas por selênio como C#, Groovy, Java, Perl, PHP, Python e Ruby.

A jornada inclui abrir o driver do navegador -> Página do Google -> desligar o navegador

##C#

using OpenQA.Selenium;
using OpenQA.Selenium.Chrome;

namespace BasicWebdriver
{
    class WebDriverTest
    {
        static void Main()
        {
            using (var driver = new ChromeDriver())
            {
                driver.Navigate().GoToUrl("http://www.google.com");
            }
        }
    }
}

O ‘programa’ acima irá navegar para a página inicial do Google e, em seguida, fechará o navegador após carregar totalmente a página.

using (var driver = new ChromeDriver())

Isso instancia um novo objeto WebDriver usando a interface IWebdriver e cria uma nova instância de janela do navegador. Neste exemplo, estamos usando o ChromeDriver (embora isso possa ser substituído pelo driver apropriado para qualquer navegador que queiramos usar). Estamos envolvendo isso com uma instrução using, porque IWebDriver implementa IDisposable, portanto, não precisa digitar explicitamente driver.Quit();.

Caso você não tenha baixado seu WebDriver usando [NuGet][1], então você precisará passar um argumento na forma de um caminho para o diretório onde o próprio driver “chromedriver.exe” está localizado.

Navegação #

driver.Navigate().GoToUrl("http://www.google.com");

e

driver.Url = "http://www.google.com";

Ambas as linhas fazem a mesma coisa. Eles instruem o driver a navegar para uma URL específica e esperar até que a página seja carregada antes de passar para a próxima instrução.

Existem outros métodos vinculados à navegação, como Back(), Forward() ou Refresh().

Depois disso, o bloco using é encerrado com segurança e descarta o objeto.

[1]: https://www.wikiod.com/pt/selenium-webdriver/introducao-ao-selenium-webdriver#Instalação ou configuração

##Java

O código abaixo é tudo sobre 3 etapas.

  1. Abrindo um navegador Chrome
  2. Abrindo a página do Google
  3. Desligue o navegador
import org.openqa.selenium;
import org.openqa.selenium.chrome;

public class WebDriverTest {
    public static void main(String args[]) {
        System.setProperty("webdriver.chrome.driver", "C:\\path\\to\\chromedriver.exe");
        WebDriver driver = new ChromeDriver();

        driver.get("http://www.google.com");
        driver.quit();
    }
}

O ‘programa’ acima navegará para a página inicial do Google e, em seguida, fechará o navegador antes de concluir.

System.setProperty("webdriver.chrome.driver", "C:\\path\\to\\chromedriver.exe");
WebDriver driver = new ChromeDriver();

A primeira linha informa ao sistema onde encontrar o executável ChromeDriver (chromedriver.exe). Em seguida, criamos nosso objeto de driver chamando o construtor ChromeDriver(), novamente poderíamos estar chamando nosso construtor aqui para qualquer navegador/plataforma.

driver.get("http://www.google.com");

Isso diz ao nosso driver para navegar até o URL especificado: http://www.google.com. A API Java WebDriver fornece o método get() diretamente na interface do WebDriver, embora outros métodos de navegação possam ser encontrados através do método navigate(), ex. driver.navigate.back().

Assim que a página terminar de carregar, chamamos imediatamente:

driver.quit();

Isso diz ao driver para fechar todas as janelas abertas do navegador e descartar o objeto do driver, pois não temos outro código após essa chamada, isso efetivamente encerra o programa.

driver.close();

É uma instrução (não mostrada aqui) para o driver fechar apenas a janela ativa, neste caso como temos apenas uma única janela as instruções causariam resultados idênticos ao chamar quit().

Pitão #

from selenium import webdriver
from selenium.webdriver.common.keys import Keys

def set_up_driver():
    path_to_chrome_driver = 'chromedriver'
    return webdriver.Chrome(executable_path=path_to_chrome_driver)

def get_google():
    driver = set_up_driver()
    driver.get('http://www.google.com')
    tear_down(driver)

def tear_down(driver):
    driver.quit()

if '__main__' == __name__:
    get_google()

O ‘programa’ acima navegará para a página inicial do Google e, em seguida, fechará o navegador antes de concluir.

if '__main__' == __name__:
    get_google()

Primeiro temos nossa função principal, nosso ponto de entrada no programa, que chama get_google().

def get_google():
    driver = set_up_driver()

get_google() então começa criando nossa instância driver via set_up_driver():

def set_up_driver():
    path_to_chrome_driver = 'chromedriver'
    return webdriver.Chrome(executable_path=path_to_chrome_driver)

Por meio do qual informamos onde o chromedriver.exe está localizado e instanciamos nosso objeto de driver com este caminho. O restante de get_google() navega para o Google:

driver.get('http://www.google.com')

E então chama tear_down() passando o objeto driver:

tear_down(driver)

tear_down() simplesmente contém uma linha para desligar nosso objeto de driver:

driver.quit()

Isso diz ao driver para fechar todas as janelas abertas do navegador e descartar o objeto do navegador, pois não temos outro código após essa chamada, isso efetivamente encerra o programa.

Classes de página: Página de login e página inicial Classe da página de login:

import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.support.FindBy;

public class LoginPage {

    WebDriver driver;
    public LoginPage(WebDriver driver){
        this.driver = driver;
    }
    @FindBy(id="email")
    private WebElement loginTextBox;

    @FindBy(id="pass")
    private WebElement passwordTextBox;

    @FindBy(xpath = ".//input[@data-testid='royal_login_button']")
    private WebElement loginBtn;

    public void enterUserName(String userName){
        if(loginTextBox.isDisplayed()) {
            loginTextBox.clear();
            loginTextBox.sendKeys(userName);
        }
        else{
            System.out.println("Element is not loaded");
        }
    }
    public void enterPassword(String password){
        if(passwordTextBox.isDisplayed()) {
            passwordTextBox.clear();
            passwordTextBox.sendKeys(password);
        }
        else{
            System.out.println("Element is not loaded");
        }
    }
    public HomePage clickLogin(){
        if(loginBtn.isDisplayed()) {
            loginBtn.click();
        }
        return new HomePage(driver);
    }
}

Classe da página inicial:

import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.support.FindBy;

public class HomePage {

    WebDriver driver;
    public HomePage(WebDriver driver){
        this.driver = driver;
    }

    @FindBy(xpath=".//a[@data-testid='blue_bar_profile_link']/span")
    private WebElement userName;

    public String getUserName(){
        if(userName.isDisplayed()) {
            return userName.getText();
        }
        else {
            return "Username is not present";
        }
    }

}
