Rolagem diferente usando java de maneiras diferentes #

Abaixo, a solução também pode ser usada em outras linguagens de programação suportadas com algumas alterações de sintaxe

  1. Para fazer Rolar para baixo página/seção/divisão na página da web enquanto houver barra de rolagem personalizada (não rolagem do navegador). Clique aqui para demonstração e verifique se a barra de rolagem tem seu elemento independente.

No código abaixo, passe o elemento da barra de rolagem e exija pontos de rolagem.

    public static boolean scroll_Page(WebElement webelement, int scrollPoints)
    {
    try
    {
        System.out.println("---------------- Started - scroll_Page ----------------");
        driver = ExecutionSetup.getDriver();
        dragger = new Actions(driver);

        // drag downwards
        int numberOfPixelsToDragTheScrollbarDown = 10;
        for (int i = 10; i < scrollPoints; i = i + numberOfPixelsToDragTheScrollbarDown)
        {
            dragger.moveToElement(webelement).clickAndHold().moveByOffset(0, numberOfPixelsToDragTheScrollbarDown).release(webelement).build().perform();
        }
        Thread.sleep(500);
        System.out.println("---------------- Ending - scroll_Page ----------------");
        return true;
    }
    catch (Exception e)
    {
        System.out.println("---------------- scroll is unsucessfully done in scroll_Page ----------------");
        e.printStackTrace();
        return false;
    }
  }

  1. Para rolar para baixo quando rolam vários navegadores (navegador incorporado) e você deseja rolar para baixo com a Tecla Page Down. Clique aqui para demonstração

No código abaixo, passe o elemento da área de rolagem como <div> e exija a tecla page down.

 public static boolean pageDown_New(WebElement webeScrollArea, int iLoopCount)
 {
    try
    {
        System.out.println("---------------- Started - pageDown_New ----------------");
        driver = ExecutionSetup.getDriver();
        dragger = new Actions(driver);

        for (int i = 0; i <= iLoopCount; i++)
        {
            dragger.moveToElement(webeScrollArea).click().sendKeys(Keys.PAGE_DOWN).build().perform();
        }
        System.out.println"---------------- Ending - pageDown_New ----------------");
        return true;
    }
    catch (Exception e)
    {
        System.out.println("---------------- Not able to do page down ----------------");
        return false;
    }
}

  1. Para rolar para cima/para baixo quando rolagem do navegador (navegador embutido) e você deseja rolar para cima/para baixo com Apenas ponto fixo. Clique aqui para demonstração

No código abaixo, passe seu ponto de rolagem. Positivo significa para baixo e negativo significa rolar para cima.

public static boolean scroll_without_WebE(int scrollPoint)
{
    JavascriptExecutor jse;
    try
    {
        System.out.println("---------------- Started - scroll_without_WebE ----------------");

        driver = ExecutionSetup.getDriver();
        jse = (JavascriptExecutor) driver;
        jse.executeScript("window.scrollBy(0," + scrollPoint + ")", "");

        System.out.println("---------------- Ending - scroll_without_WebE ----------------");
        return true;
    }
    catch (Exception e)
    {
        System.out.println("---------------- scroll is unsucessful in scroll_without_WebE ----------------");
        e.printStackTrace();
        return false;
    }
}

public static boolean scroll_to_WebE(WebElement webe)
{
    try
    {
        System.out.println("---------------- Started - scroll_to_WebE ----------------");

        driver = ExecutionSetup.getDriver();

Rolagem usando Python #

1. Rolando para o elemento de destino (botão “BROWSE TEMPLATES” na parte inferior da página) com Actions

from selenium import webdriver
from selenium.webdriver.common.action_chains import ActionChains

driver = webdriver.Chrome()
driver.get('http://www.w3schools.com/')
target = driver.find_element_by_link_text('BROWSE TEMPLATES')
actions = ActionChains(driver)
actions.move_to_element(target)
actions.perform()

2. Rolando para o elemento de destino (botão “BROWSE TEMPLATES” na parte inferior da página) com JavaScript

from selenium import webdriver

driver = webdriver.Chrome()
driver.get('http://www.w3schools.com/')
target = driver.find_element_by_link_text('BROWSE TEMPLATES')
driver.execute_script('arguments[0].scrollIntoView(true);', target)

3. Rolagem para o elemento de destino (botão “BROWSE TEMPLATES” na parte inferior da página) com método integrado

from selenium import webdriver

    driver = webdriver.Chrome()
    driver.get('http://www.w3schools.com/')
    target = driver.find_element_by_link_text('BROWSE TEMPLATES')
    target.location_once_scrolled_into_view

Observe que location_once_scrolled_into_view também retorna as coordenadas x, y do elemento após a rolagem

4. Rolando até o final da página com Keys

from selenium import webdriver
from selenium.webdriver.common.keys import Keys 

driver = webdriver.Chrome()
driver.get('http://www.w3schools.com/')
driver.find_element_by_tag_name('body').send_keys(Keys.END) # Use send_keys(Keys.HOME) to scroll up to the top of page

Observe que send_keys(Keys.DOWN)/send_keys(Keys.UP) e send_keys(Keys.PAGE_DOWN)/send_keys(Keys.PAGE_UP) também podem ser usados ​​para rolar
