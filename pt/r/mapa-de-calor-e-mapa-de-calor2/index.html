stats:: mapa de calor #

Exemplo 1 (uso básico) #

require(graphics); require(grDevices)
x  <- as.matrix(mtcars)
rc <- rainbow(nrow(x), start = 0, end = .3)
cc <- rainbow(ncol(x), start = 0, end = .3)
hv <- heatmap(x, col = cm.colors(256), scale = "column",
              RowSideColors = rc, ColSideColors = cc, margins = c(5,10),
              xlab = "specification variables", ylab =  "Car Models",
              main = "heatmap(<Mtcars data>, ..., scale = \"column\")")


utils::str(hv) # the two re-ordering index vectors

Ajustando parâmetros em heatmap.2 #

Dado:

x  <- as.matrix(mtcars)

Pode-se usar heatmap.2 - uma versão otimizada mais recente do heatmap, carregando a seguinte biblioteca:

require(gplots)
heatmap.2(x)

Para adicionar um título, rótulo x ou y ao seu mapa de calor, você precisa definir o main, xlab e ylab:

heatmap.2(x, main = "My main title: Overview of car features", xlab="Car features", ylab = "Car brands")

Se você deseja definir sua própria paleta de cores para seu mapa de calor, você pode definir o parâmetro col usando a função colorRampPalette:

heatmap.2(x, trace="none", key=TRUE, Colv=FALSE,dendrogram = "row",col =  colorRampPalette(c("darkblue","white","darkred"))(100))

Como você pode notar, os rótulos no eixo y (os nomes dos carros) não se encaixam na figura. Para corrigir isso, o usuário pode ajustar o parâmetro margins:

heatmap.2(x, trace="none", key=TRUE,col = colorRampPalette(c("darkblue","white","darkred"))(100), margins=c(5,8))

Além disso, podemos alterar as dimensões de cada seção do nosso mapa de calor (o histograma chave, os dendogramas e o próprio mapa de calor), ajustando lhei e lwid :

Se quisermos mostrar apenas um dendograma de linha (ou coluna), precisamos definir Colv=FALSE (ou Rowv=FALSE) e ajustar o parâmetro dendogram:

heatmap.2(x, trace="none", key=TRUE, Colv=FALSE, dendrogram = "row", col = colorRampPalette(c("darkblue","white","darkred"))(100), margins=c(5,8), lwid = c(5,15), lhei = c(3,15))

Para alterar o tamanho da fonte do título da legenda, rótulos e eixo, o usuário precisa definir cex.main, cex.lab, cex.axis na lista par:

par(cex.main=1, cex.lab=0.7, cex.axis=0.7)
heatmap.2(x, trace="none", key=TRUE, Colv=FALSE, dendrogram = "row", col = colorRampPalette(c("darkblue","white","darkred"))(100), margins=c(5,8), lwid = c(5,15), lhei = c(5,15))
