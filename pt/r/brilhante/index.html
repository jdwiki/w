Crie um aplicativo #

Shiny é um pacote R desenvolvido por RStudio que permite a criação de páginas web para exibir interativamente os resultados de uma análise em R.

Existem duas maneiras simples de criar um aplicativo Shiny:

  • em um arquivo .R, ou
  • em dois arquivos: ui.R e server.R.

Um aplicativo Shiny é dividido em duas partes:

  • ui: um script de interface do usuário, controlando o layout e a aparência do aplicativo.
  • servidor: Um script de servidor que contém código para permitir que o aplicativo reaja.

Um arquivo #

library(shiny)

# Create the UI
ui <- shinyUI(fluidPage(
    # Application title
    titlePanel("Hello World!")
))

# Create the server function
server <- shinyServer(function(input, output){})

# Run the app
shinyApp(ui = ui, server = server)

Dois arquivos #

Criar arquivo ui.R #

library(shiny)

# Define UI for application
shinyUI(fluidPage(  
  # Application title
  titlePanel("Hello World!")
))

Cria o arquivo server.R #

library(shiny)

# Define server logic
shinyServer(function(input, output){})

Grupo de caixas de seleção #

Crie um grupo de caixas de seleção que podem ser usadas para alternar várias opções de forma independente. O servidor receberá a entrada como vetor de caracteres dos valores selecionados.

library(shiny)

ui <- fluidPage(    
  checkboxGroupInput("checkGroup1", label = h3("This is a Checkbox group"), 
                 choices = list("1" = 1, "2" = 2, "3" = 3),
                 selected = 1),
  fluidRow(column(3, verbatimTextOutput("text_choice")))
  ) 


server <- function(input, output){
  output$text_choice <- renderPrint({
    return(paste0("You have chosen the choice ",input$checkGroup1))})
} 

shinyApp(ui = ui, server = server)

É possível alterar as configurações:

  • etiqueta: título
  • escolhas: valores selecionados
  • selected : O valor inicialmente selecionado (NULL para nenhuma seleção)
  • em linha: horizontal ou vertical
  • largura

Também é possível adicionar HTML.

Botao de radio #

Você pode criar um conjunto de botões de opção usados ​​para selecionar um item de uma lista.

É possível alterar as configurações:

  • selected : O valor inicialmente selecionado (caractere(0) para nenhuma seleção)
  • em linha: horizontal ou vertical
  • largura

Também é possível adicionar HTML.

library(shiny)

ui <- fluidPage(    
  radioButtons("radio", 
               label = HTML('<FONT color="red"><FONT size="5pt">Welcome</FONT></FONT><br> <b>Your favorite color is red ?</b>'),
               choices = list("TRUE" = 1, "FALSE" = 2),
               selected = 1,
               inline = T,
               width = "100%"),      
  fluidRow(column(3, textOutput("value")))) 


server <- function(input, output){
  output$value <- renderPrint({
    if(input$radio == 1){return('Great !')}
     else{return("Sorry !")}})}

shinyApp(ui = ui, server = server)

Selecione a caixa #

Crie uma lista de seleção que pode ser usada para escolher um ou vários itens de uma lista de valores.

library(shiny)

ui <- fluidPage(    
  selectInput("id_selectInput", 
          label = HTML('<B><FONT size="3">What is your favorite color ?</FONT></B>'), 
          multiple = TRUE,
          choices = list("red" = "red", "green" = "green", "blue" = "blue", "yellow" = "yellow"), 
          selected = NULL),
  br(), br(),
  fluidRow(column(3, textOutput("text_choice")))) 


server <- function(input, output){
  output$text_choice <- renderPrint({
    return(input$id_selectInput)})
}

shinyApp(ui = ui, server = server)

É possível alterar as configurações:

  • etiqueta: título
  • escolhas: valores selecionados
  • selected : O valor inicialmente selecionado (NULL para nenhuma seleção)
  • múltiplo: VERDADEIRO ou FALSO
  • largura
  • Tamanho
  • selectize: TRUE ou FALSE (para usar ou não selectize.js, altere o display)

Também é possível adicionar HTML.

Inicie um aplicativo Shiny #

Você pode iniciar um aplicativo de várias maneiras, dependendo de como você cria seu aplicativo. Se seu aplicativo estiver dividido em dois arquivos ui.R e server.R ou se todo o seu aplicativo estiver em um arquivo.

1. Aplicativo de dois arquivos #

Seus dois arquivos ui.R e server.R devem estar na mesma pasta. Você pode então iniciar seu aplicativo executando no console a função shinyApp() e passando o caminho do diretório que contém o aplicativo Shiny.

shinyApp("path_to_the_folder_containing_the_files")

Você também pode iniciar o aplicativo diretamente do Rstudio pressionando o botão Run App que aparece no Rstudio quando um arquivo ui.R ou server.R é aberto.

Ou você pode simplesmente escrever runApp() no console se seu diretório de trabalho for o diretório Shiny App.

2. Um aplicativo de arquivo #

Se você criar o seu em um arquivo R, você também pode iniciá-lo com a função shinyApp().

  • dentro do seu código:

    library(shiny)

    ui <- fluidPage() #Create the ui server <- function(input, output){} #create the server

    shinyApp(ui = ui, server = server) #run the App

  • no console adicionando o caminho para um arquivo .R contendo a aplicação Shiny com o parâmetro appFile:

    shinyApp(appFile=“path_to_my_R_file_containig_the_app”)

Depuração #

debug() e debugonce() não funcionarão bem no contexto da maioria das depurações do Shiny. No entanto, instruções browser() inseridas em locais críticos podem fornecer muitas informações sobre como seu código Shiny está (não) funcionando. Veja também: [Depurando usando browser()](https://www.wikiod.com/pt/r/depuracao#Usando o navegador)

#Modo vitrine

Modo de demonstração exibe seu aplicativo junto com o código que o gera e destaca as linhas de código em server.R conforme ele as executa.

Há duas maneiras de ativar o modo Showcase:

  • Inicie o aplicativo Shiny com o argumento display.mode = “showcase”, por exemplo, runApp("MyApp", display.mode = "showcase").
  • Crie um arquivo chamado DESCRIPTION na pasta do seu aplicativo Shiny e adicione esta linha nele: DisplayMode: Showcase.

Visualizador de log reativo #

Reactive Log Visualizer fornece uma ferramenta interativa baseada em navegador para visualizar dependências reativas e execução em seu aplicativo. Para habilitar o Reactive Log Visualizer, execute options(shiny.reactlog=TRUE) no console R e ou adicione essa linha de código em seu arquivo server.R. Para iniciar o Reactive Log Visualizer, pressione Ctrl+F3 no Windows ou Command+F3 no Mac quando seu aplicativo estiver em execução. Use as teclas de seta para a esquerda e para a direita para navegar no Reactive Log Visualizer.

Widgets de controle #

Função Widget
actionButton Botão de ação
checkboxGroupInput Um grupo de caixas de seleção
checkboxInput Uma única caixa de seleção
dateInput Um calendário para auxiliar na seleção de datas
dateRangeInput Um par de calendários para selecionar um intervalo de datas
fileInput Um assistente de controle de upload de arquivo
helpText Texto de ajuda que pode ser adicionado a um formulário de entrada
numericInput Um campo para inserir números
radioButtons Um conjunto de botões de rádio
selectInput Uma caixa com opções para selecionar
sliderInput Uma barra deslizante
submitButton Um botão de envio
textInput Um campo para inserir texto 
library(shiny)

# Create the UI
ui <- shinyUI(fluidPage(
  titlePanel("Basic widgets"),
  
  fluidRow(
    
    column(3,
           h3("Buttons"),
           actionButton("action", label = "Action"),
           br(),
           br(), 
           submitButton("Submit")),
    
    column(3,
           h3("Single checkbox"),
           checkboxInput("checkbox", label = "Choice A", value = TRUE)),
    
    column(3, 
           checkboxGroupInput("checkGroup", 
                              label = h3("Checkbox group"), 
                              choices = list("Choice 1" = 1, 
                                             "Choice 2" = 2, "Choice 3" = 3),
                              selected = 1)),
    
    column(3, 
           dateInput("date", 
                     label = h3("Date input"), 
                     value = "2014-01-01"))   
  ),
  
  fluidRow(
    
    column(3,
           dateRangeInput("dates", label = h3("Date range"))),
    
    column(3,
           fileInput("file", label = h3("File input"))),
    
    column(3, 
           h3("Help text"),
           helpText("Note: help text isn't a true widget,", 
                    "but it provides an easy way to add text to",
                    "accompany other widgets.")),
    
    column(3, 
           numericInput("num", 
                        label = h3("Numeric input"), 
                        value = 1))   
  ),
  
  fluidRow(
    
    column(3,
           radioButtons("radio", label = h3("Radio buttons"),
                        choices = list("Choice 1" = 1, "Choice 2" = 2,
                                       "Choice 3" = 3),selected = 1)),
