Uso básico de aov() #

A Análise de Variância (aov) é usada para determinar se as médias de dois ou mais grupos diferem significativamente entre si. As respostas são consideradas independentes umas das outras, normalmente distribuídas (dentro de cada grupo), e as variações dentro do grupo são consideradas iguais.

Para concluir a análise, os dados devem estar em formato longo (consulte o tópico remodelagem de dados) . aov() é um wrapper em torno da função lm(), usando a notação de fórmula de Wilkinson-Rogers y~f onde y é a variável de resposta (independente) e f é uma variável de fator (categórica) representando a associação do grupo. Se f for numérico em vez de uma variável de fator, aov() relatará os resultados de uma regressão linear no formato ANOVA, o que pode surpreender usuários inexperientes.

A função aov() usa Soma de Quadrados Tipo I (sequencial). Este tipo de Soma de Quadrados testa todos os efeitos (principal e de interação) sequencialmente. O resultado é que o primeiro efeito testado também recebe variância compartilhada entre ele e outros efeitos no modelo. Para que os resultados de tal modelo sejam confiáveis, os dados devem ser balanceados (todos os grupos são do mesmo tamanho).

Quando as premissas para a Soma de Quadrados Tipo I não se confirmarem, a Soma de Quadrados Tipo II ou Tipo III pode ser aplicável. A Soma de Quadrados Tipo II testa cada efeito principal após cada outro efeito principal e, portanto, controla qualquer variação sobreposta. No entanto, a Soma de Quadrados Tipo II não assume nenhuma interação entre os efeitos principais.

Por fim, a Soma de Quadrados Tipo III testa cada efeito principal após cada outro efeito principal e cada interação. Isso torna a Soma de Quadrados Tipo III uma necessidade quando uma interação está presente.

As Somas de Quadrados Tipo II e Tipo III são implementadas na função Anova().

Usando o conjunto de dados mtcars como exemplo.

mtCarsAnovaModel <- aov(wt ~ factor(cyl), data=mtcars)

Para visualizar o resumo do modelo ANOVA:

summary(mtCarsAnovaModel)

Pode-se também extrair os coeficientes do modelo lm() subjacente:

coefficients(mtCarsAnovaModel)

Uso básico de Anova() #

Ao lidar com um desenho desequilibrado e/ou contrastes não ortogonais, são necessárias a Soma de Quadrados Tipo II ou Tipo III. A função Anova() do pacote car implementa isso. Tipo II Soma de Quadrados não assume nenhuma interação entre os efeitos principais. Se as interações forem assumidas, a Soma de Quadrados do Tipo III é apropriada.

A função Anova() envolve a função lm().

Usando os conjuntos de dados mtcars como exemplo, demonstrando a diferença entre o Tipo II e o Tipo III quando uma interação é testada.
