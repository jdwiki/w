Parâmetros #

|Parâmetro|Detalhes | |———|———————————– —————| |x |variável do eixo x. Pode fornecer data$variablex ou data[,x]| |y |variável do eixo y. Pode fornecer data$variabley ou data[,y]| |main |Título principal do enredo | |sub |Subtítulo opcional da trama | |xlab |Etiqueta para eixo x | |ylab |Etiqueta para o eixo y | |pch |Inteiro ou caractere indicando símbolo de plotagem | |col |Inteiro ou string indicando cor | |tipo |Tipo de plotagem. "p" para pontos, "l" para linhas, "b" para ambos, "c" apenas para a parte de linhas de "b", "o" para ambos ‘overplotted ‘, "h" para linhas verticais semelhantes a ‘histograma’ (ou ‘alta densidade’), "s" para degraus de escada, "S" para outros degraus, "n" para não plotagem |

Os itens listados na seção “Parâmetros” são uma pequena fração dos parâmetros possíveis que podem ser modificados ou definidos pela função par. Veja par para uma lista mais completa. Além disso, todos os dispositivos gráficos, incluindo os dispositivos gráficos interativos específicos do sistema, terão um conjunto de parâmetros que podem personalizar a saída.

Combinando parcelas #

Muitas vezes é útil combinar vários tipos de plotagem em um gráfico (por exemplo, um Barplot ao lado de um Scatterplot.) R torna isso fácil com a ajuda das funções par() e layout().

par() #

par usa os argumentos mfrow ou mfcol para criar uma matriz de nrows e ncols c(nrows, ncols) que servirá como uma grade para seus gráficos. O exemplo a seguir mostra como combinar quatro gráficos em um gráfico:

par(mfrow=c(2,2))
plot(cars, main="Speed vs. Distance")
hist(cars$speed, main="Histogram of Speed")
boxplot(cars$dist, main="Boxplot of Distance")
boxplot(cars$speed, main="Boxplot of Speed")

layout() #

O layout() é mais flexível e permite especificar a localização e a extensão de cada plot dentro do gráfico combinado final. Esta função espera um objeto de matriz como entrada:

layout(matrix(c(1,1,2,3), 2,2, byrow=T))
hist(cars$speed, main="Histogram of Speed")
boxplot(cars$dist, main="Boxplot of Distance")
boxplot(cars$speed, main="Boxplot of Speed")

Gráfico de densidade #

Um acompanhamento muito útil e lógico para histogramas seria traçar a função de densidade suavizada de uma variável aleatória. Um gráfico básico produzido pelo comando

plot(density(rnorm(100)),main="Normal density",xlab="x")

pareceria

Você pode sobrepor um histograma e uma curva de densidade com

x=rnorm(100)
hist(x,prob=TRUE,main="Normal density + histogram")
lines(density(x),lty="dotted",col="red")

que dá

Trama Básica #

Um gráfico básico é criado chamando plot(). Aqui usamos o quadro de dados interno cars que contém a velocidade dos carros e as distâncias percorridas para parar na década de 1920. (Para saber mais sobre o conjunto de dados, use help(cars)).

plot(x = cars$speed, y = cars$dist, pch = 1, col = 1, 
     main = "Distance vs Speed of Cars", 
     xlab = "Speed", ylab = "Distance")

Podemos usar muitas outras variações no código para obter o mesmo resultado. Também podemos alterar os parâmetros para obter resultados diferentes.

with(cars, plot(dist~speed, pch = 2, col = 3, 
     main = "Distance to stop vs Speed of Cars", 
     xlab = "Speed", ylab = "Distance"))

Recursos adicionais podem ser adicionados a este gráfico chamando points(), text(), mtext(), lines(), grid(), etc.

plot(dist~speed, pch = "*", col = "magenta", data=cars,
     main = "Distance to stop vs Speed of Cars", 
     xlab = "Speed", ylab = "Distance")
mtext("In the 1920s.")
grid(,col="lightblue")

Começando com R_Plots #

  • Gráfico de dispersão

Você tem dois vetores e quer plotá-los.

x_values <- rnorm(n = 20 , mean = 5 , sd = 8) #20 values generated from Normal(5,8)
y_values <- rbeta(n = 20 , shape1 = 500 , shape2 = 10) #20 values generated from Beta(500,10)

Se você quiser fazer um gráfico que tenha os y_values no eixo vertical e os x_values no eixo horizontal, você pode usar os seguintes comandos:

plot(x = x_values, y = y_values, type = "p") #standard scatter-plot
plot(x = x_values, y = y_values, type = "l") # plot with lines
plot(x = x_values, y = y_values, type = "n") # empty plot

Você pode digitar ?plot() no console para ler mais opções.

  • Boxplot

Você tem algumas variáveis ​​e deseja examinar suas distribuições

#boxplot is an easy way to see if we have some outliers in the data.
   
z<- rbeta(20 , 500 , 10) #generating values from beta distribution
z[c(19 , 20)] <- c(0.97 , 1.05) # replace the two last values with outliers      
boxplot(z) # the two points are the outliers of variable z.
  • Histogramas

Maneira fácil de desenhar histogramas

hist(x = x_values) # Histogram for x vector
hist(x = x_values, breaks = 3) #use breaks to set the numbers of bars you want
  • Gráfico de setores

Se você quiser visualizar as frequências de uma variável basta desenhar pie

Primeiro temos que gerar dados com frequências, por exemplo:

P <- c(rep('A' , 3) , rep('B' , 10) , rep('C' , 7) )
t <- table(P) # this is a frequency matrix of variable P
pie(t) # And this is a visual version of the matrix above

Histogramas #

Os histogramas permitem um pseudo-plot da distribuição subjacente dos dados.

hist(ldeaths)


hist(ldeaths, breaks = 20, freq = F, col = 3)

Matplot #

matplot é útil para plotar rapidamente vários conjuntos de observações do mesmo objeto, principalmente de uma matriz, no mesmo gráfico.

Aqui está um exemplo de uma matriz contendo quatro conjuntos de sorteios aleatórios, cada um com uma média diferente.

xmat <- cbind(rnorm(100, -3), rnorm(100, -1), rnorm(100, 1), rnorm(100, 3))
head(xmat)
#          [,1]        [,2]       [,3]     [,4]
# [1,] -3.072793 -2.53111494  0.6168063 3.780465
# [2,] -3.702545 -1.42789347 -0.2197196 2.478416
# [3,] -2.890698 -1.88476126  1.9586467 5.268474
# [4,] -3.431133 -2.02626870  1.1153643 3.170689
# [5,] -4.532925  0.02164187  0.9783948 3.162121
# [6,] -2.169391 -1.42699116  0.3214854 4.480305

Uma maneira de plotar todas essas observações no mesmo gráfico é fazer uma chamada plot seguida por mais três chamadas points ou lines.

plot(xmat[,1], type = 'l')
lines(xmat[,2], col = 'red')
lines(xmat[,3], col = 'green')
lines(xmat[,4], col = 'blue')

No entanto, isso é tedioso e causa problemas porque, entre outras coisas, por padrão, os limites dos eixos são fixados por plot para caber apenas na primeira coluna.

Muito mais conveniente nesta situação é usar a função matplot, que requer apenas uma chamada e automaticamente cuida dos limites dos eixos e alterando a estética de cada coluna para torná-las distinguíveis.

matplot(xmat, type = 'l')

Observe que, por padrão, matplot varia tanto a cor (col) quanto o tipo de linha (lty) porque isso aumenta o número de combinações possíveis antes que elas sejam repetidas. No entanto, qualquer uma (ou ambas) dessas estéticas pode ser fixada em um único valor…

matplot(xmat, type = 'l', col = 'black')

…ou um vetor personalizado (que reciclará para o número de colunas, seguindo as regras padrão de reciclagem de vetor R).

matplot(xmat, type = 'l', col = c('red', 'green', 'blue', 'orange'))

Parâmetros gráficos padrão, incluindo main, xlab, xmin, funcionam exatamente da mesma maneira que para plot. Para saber mais sobre isso, veja ?par.

Como plot, se dado apenas um objeto, matplot assume que é a variável y e usa os índices para x. No entanto, x e y podem ser especificados explicitamente.

matplot(x = seq(0, 10, length.out = 100), y = xmat, type='l')

Na verdade, tanto x quanto y podem ser matrizes.

xes <- cbind(seq(0, 10, length.out = 100),
             seq(2.5, 12.5, length.out = 100),
             seq(5, 15, length.out = 100),
             seq(7.5, 17.5, length.out = 100))
matplot(x = xes, y = xmat, type = 'l')

Função de distribuição cumulativa empírica #

Um acompanhamento muito útil e lógico para histogramas e gráficos de densidade seria a Função de Distribuição Cumulativa Empírica. Podemos usar a função ecdf() para este propósito. Um gráfico básico produzido pelo comando

plot(ecdf(rnorm(100)),main="Cumulative distribution",xlab="x")
