Criação manual de JSON #

import play.libs.Json;


public JsonNode createJson() {
    // {"id": 33, "values": [3, 4, 5]}
    ObjectNode rootNode = Json.newObject();
    ArrayNode listNode = Json.newArray();

    long values[] = {3, 4, 5};
    for (long val: values) {
        listNode.add(val);
    }
    
    rootNode.put("id", 33);
    rootNode.set("values", listNode);
    return rootNode;
}

Carregando json de string/arquivo #

import play.libs.Json;
// (...)

Carregando um arquivo da sua pasta pública #

// Note: "app" is an play.Application instance
JsonNode node = Json.parse(app.resourceAsStream("public/myjson.json"));

Carregar de uma string #

String myStr = "{\"name\": \"John Doe\"}";
JsonNode node = Json.parse(myStr);

Transversando um documento JSON #

Nos exemplos a seguir, json contém um objeto JSON com os seguintes dados:

[
  {
    "name": "John Doe",
    "work": {
      "company": {
        "name": "ASDF INC",
        "country": "USA"
      },
      "cargo": "Programmer"
    },
    "tags": ["java", "jvm", "play"]
  },
  {
    "name": "Bob Doe",
    "work": {
      "company": {
        "name": "NOPE INC",
        "country": "AUSTRALIA"
      },
      "cargo": "SysAdmin"
    },
    "tags": ["puppet", "ssh", "networking"],
    "active": true
  }
]

Pega o nome de algum usuário (inseguro) #

JsonNode node = json.get(0).get("name"); // --> "John Doe"
// This will throw a NullPointerException, because there is only two elements
JsonNode node = json.get(2).get("name"); // --> *crash*

Obtenha o nome de usuário (maneira segura) #

JsonNode node1 = json.at("/0/name"); // --> TextNode("John Doe")
JsonNode node2 = json.at("/2/name"); // --> MissingNode instance
if (! node2.isMissingNode()) {
    String name = node2.asText();    
}

Obtenha o país onde o primeiro usuário trabalha #

JsonNode node2 = json.at("/0/work/company/country"); // TextNode("USA")

Obtenha todos os países #

List<JsonNode> d = json.findValues("country"); // List(TextNode("USA"), TextNode("AUSTRALIA"))

Encontre todos os usuários que contenham o atributo “ativo” #

List<JsonNode> e = json.findParents("active"); // List(ObjectNode("Bob Doe"))

Conversão entre objetos JSON e Java (básico) #

Por padrão, Jackson (a biblioteca que o Play JSON usa) tentará mapear todos os campos públicos para um campo json com o mesmo nome. Se o objeto tiver getters/setters, ele inferirá o nome deles. Então, se você tem uma classe Book com um campo privado para armazenar o ISBN e tem métodos get/set chamados getISBN/setISBN, Jackson irá

  • Crie um objeto JSON com o campo “ISBN” ao converter de Java para JSON
  • Use o método setISBN para definir o campo isbn no objeto Java (se o objeto JSON tiver um campo “ISBN”).

Criar objeto Java a partir de JSON #

public class Person {
    String id, name;
}

JsonNode node = Json.parse("{\"id\": \"3S2F\", \"name\", \"Salem\"}");
Person person = Json.fromJson(node, Person.class);
System.out.println("Hi " + person.name); // Hi Salem

Criar objeto JSON a partir do objeto Java #

// "person" is the object from the previous example
JsonNode personNode = Json.toJson(person)

Criando uma string JSON a partir de um objeto JSON #

// personNode comes from the previous example
String json = personNode.toString();
// or 
String json = Json.stringify(json);

Impressão bonita em JSON #

System.out.println(personNode.toString());
/* Prints:
{"id":"3S2F","name":"Salem"}
*/   

System.out.println(Json.prettyPrint(personNode));
/* Prints:
{
  "id" : "3S2F",
  "name" : "Salem"
}
*/
