Criar Objeto Gráfico #

Existem três maneiras de criar um objeto gráfico

  1. Do Evento de Pintura

Toda vez que o controle é redesenhado (redimensionado, atualizado…) esse evento é chamado, use desta forma se você quiser que o controle desenhe consistentemente no controle

   'this will work on any object's paint event, not just the form
   Private Sub Form1_Paint(sender as Object, e as PaintEventArgs) Handles Me.Paint
     Dim gra as Graphics
     gra = e.Graphics
   End Sub
  1. Criar gráfico

Isso é usado com mais frequência quando você deseja criar um gráfico de uso único no controle ou não deseja que o controle seja repintado

   Dim btn as New Button
   Dim g As Graphics = btn.CreateGraphics
  1. De um Gráfico existente

Use este método quando quiser desenhar e alterar um gráfico existente

   'The existing image can be from a filename, stream or Drawing.Graphic
   Dim image = New Bitmap("C:\TempBit.bmp")
   Dim gr As Graphics = Graphics.FromImage(image)

Desenhar Formas #

Para começar a desenhar uma forma você precisa definir um objeto de caneta A Pen aceita dois parâmetros:

  1. Cor da caneta ou pincel
  2. Largura da caneta

O Objeto Caneta é usado para criar um contorno do objeto que você deseja desenhar

Depois de definir a caneta, você pode definir propriedades específicas da caneta

   Dim pens As New Pen(Color.Purple)
   pens.DashStyle = DashStyle.Dash 'pen will draw with a dashed line
   pens.EndCap = LineCap.ArrowAnchor 'the line will end in an arrow
   pens.StartCap = LineCap.Round 'The line draw will start rounded
   '*Notice* - the Start and End Caps will not show if you draw a closed shape

Em seguida, use o objeto gráfico que você criou para desenhar a forma

  Private Sub GraphicForm_Paint(sender As Object, e As PaintEventArgs) Handles MyBase.Paint
    Dim pen As New Pen(Color.Blue, 15) 'Use a blue pen with a width of 15
    Dim point1 As New Point(5, 15) 'starting point of the line
    Dim point2 As New Point(30, 100) 'ending point of the line
    e.Graphics.DrawLine(pen, point1, point2)

    e.Graphics.DrawRectangle(pen, 60, 90, 200, 300) 'draw an outline of the rectangle

Por padrão, a largura da caneta é igual a 1

    Dim pen2 as New Pen(Color.Orange) 'Use an orange pen with width of 1
    Dim origRect As New Rectangle(90, 30, 50, 60) 'Define bounds of arc
    e.Graphics.DrawArc(pen2, origRect, 20, 180) 'Draw arc in the rectangle bounds

End Sub

Preencher Formas #

Graphics.FillShapes desenha uma forma e a preenche com a cor fornecida. As formas de preenchimento podem usar

  1. Ferramenta Brush - para preencher a forma com uma cor sólida

    Dim rect As New Rectangle(50, 50, 50, 50)
e.Graphics.FillRectangle(Brushes.Green, rect) 'draws a rectangle that is filled with green

e.Graphics.FillPie(Brushes.Silver, rect, 0, 180) 'draws a half circle that is filled with silver

  2. Ferramenta HatchBrush - para preencher a forma com um padrão

    Dim hBrush As New HatchBrush(HatchStyle.ZigZag, Color.SkyBlue, Color.Gray) ‘creates a HatchBrush Tool with a background color of blue, foreground color of gray, ‘and will fill with a zigzag pattern Dim rectan As New Rectangle(100, 100, 100, 100) e.Graphics.FillRectangle(hBrush, rectan)

  3. LinearGradientBrush - para preencher a forma com um gradiente

    Dim lBrush As New LinearGradientBrush(point1, point2, Color.MediumVioletRed, Color.PaleGreen)
 Dim rect As New Rectangle(50, 50, 200, 200)
 e.Graphics.FillRectangle(lBrush, rect)

  4. TextureBrush - para preencher a forma com uma imagem

Você pode escolher uma imagem de recursos, um Bitmap já definido ou um nome de arquivo

   Dim textBrush As New TextureBrush(New Bitmap("C:\ColorPic.jpg"))
    Dim rect As New Rectangle(400, 400, 100, 100)
    e.Graphics.FillPie(textBrush, rect, 0, 360)

Tanto a Ferramenta Hatch Brush quanto LinearGradientBrush importam a seguinte instrução: Imports System.Drawing.Drawing2D

Texto #

Para desenhar texto no formulário use o método DrawString

Quando você desenha uma corda você pode usar qualquer um dos 4 pincéis listados acima

Dim lBrush As New LinearGradientBrush(point1, point2, Color.MediumVioletRed, Color.PaleGreen)
e.Graphics.DrawString("HELLO", New Font("Impact", 60, FontStyle.Bold), lBrush, New Point(40, 400))
'this will draw the word "Hello" at the given point, with a linearGradient Brush

Como você não pode definir a largura ou a altura do texto, use Measure Text para verificar o tamanho do texto

Dim lBrush As New LinearGradientBrush(point1, point2, Color.MediumVioletRed, Color.PaleGreen)
Dim TextSize =  e.Graphics.MeasureString("HELLO", New Font("Impact", 60, FontStyle.Bold), lBrush)
'Use the TextSize to determine where to place the string, or if the font needs to be smaller

Ex: Você precisa desenhar a palavra “Teste” no topo do formulário. A largura do formulário é 120. Use este loop para diminuir o tamanho da fonte até que caiba na largura do formulário

Dim FontSize as Integer = 80
Dim TextSize = e.graphics.measeString("Test", New Font("Impact",FontSize, FontStyle.Bold), new Brush(colors.Blue, 10)    
Do while TextSize.Width >120
FontSize = FontSize -1
TextSize = e.graphics.measeString("Test", New Font("Impact",FontSize, FontStyle.Bold), new Brush(colors.Blue, 10)  
Loop
