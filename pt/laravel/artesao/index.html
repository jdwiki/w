Introdução #

Artisan é um utilitário que pode ajudá-lo a realizar tarefas repetitivas específicas com comandos bash. Abrange muitas tarefas, incluindo: trabalhar com banco de dados migrações e seeding, limpar cache, criar arquivos necessários para configuração de Autenticação, criando novos controladores, modelos, classes de eventos e muito mais.

Artisan é o nome da interface de linha de comando incluída no Laravel. Ele fornece vários comandos úteis para seu uso durante o desenvolvimento de seu aplicativo.

Para ver uma lista de todos os comandos Artisan disponíveis, você pode usar o comando list:

php artisan list

Para saber mais sobre qualquer comando disponível, basta preceder seu nome com a palavra-chave help:

php artisan help [command-name]

Lista todas as rotas registradas filtradas por vários métodos #

php artisan route:list --method=GET --method=POST

Isso incluirá todas as rotas que aceitam os métodos GET e POST simultaneamente.

Criando e registrando novo comando de artesão #

Você pode criar novos comandos via

php artesão make:command [commandName]

Portanto, isso criará a classe de comando [commandName] dentro do diretório app/Console/Commands.

dentro desta classe você encontrará as variáveis ​​protected $signature e protected $description, ela representa o nome e descrição do seu comando que será usado para descrever seu comando.

após criar o comando você pode registrar seu comando dentro da classe app/Console/Kernel.php onde você encontrará a propriedade commands.

para que você possa adicionar seu comando dentro do array $ command como:

protected $commands = [
    Commands\[commandName]::class
];

e então eu posso usar meu comando via console.

então como exemplo eu nomeei meu comando como

protected $signature = 'test:command';

Então sempre que eu vou correr

php artisan test:command

ele chamará o método handle dentro da classe com assinatura test:command.

Executando comandos Laravel Artisan usando código PHP #

Você também pode usar os comandos Laravel Artisan de suas rotas ou controladores.

Para executar um comando usando código PHP:

Artisan::call('command-name');

Por exemplo,

Artisan::call('db:seed');
