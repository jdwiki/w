Etapas iniciais #

Múltiplas conexões de banco de dados, de qualquer tipo, podem ser definidas dentro do arquivo de configuração do banco de dados (provavelmente app/config/database.php). Por exemplo, para extrair dados de 2 bancos de dados MySQL, defina-os separadamente:

<?php
return array(

    'default' => 'mysql',

    'connections' => array(

        # Our primary database connection
        'mysql' => array(
            'driver'    => 'mysql',
            'host'      => 'host1',
            'database'  => 'database1',
            'username'  => 'user1',
            'password'  => 'pass1'
            'charset'   => 'utf8',
            'collation' => 'utf8_unicode_ci',
            'prefix'    => '',
        ),

        # Our secondary database connection
        'mysql2' => array(
            'driver'    => 'mysql',
            'host'      => 'host2',
            'database'  => 'database2',
            'username'  => 'user2',
            'password'  => 'pass2'
            'charset'   => 'utf8',
            'collation' => 'utf8_unicode_ci',
            'prefix'    => '',
        ),
    ),
);

A conexão padrão ainda está definida como mysql. Isso significa que, a menos que especificado de outra forma, o aplicativo usa a conexão mysql.

Usando Eloquente #

Existem várias maneiras de definir qual conexão usar nos modelos Eloquent. Uma maneira é definir a variável $connection no modelo:

<?php

class SomeModel extends Eloquent {

    protected $connection = 'mysql2';

}

A conexão também pode ser definida em tempo de execução através do método setConnection.

<?php

class SomeController extends BaseController {

    public function someMethod()
    {
        $someModel = new SomeModel;

        $someModel->setConnection('mysql2');

        $something = $someModel->find(1);

        return $something;
    }
}

Usando o construtor de esquema #

No Schema Builder, use a fachada Schema com qualquer conexão. Execute o método connection() para especificar qual conexão usar:

Schema::connection('mysql2')->create('some_table', function($table)
{
    $table->increments('id'):
});

Usando o construtor de consultas de banco de dados #

Semelhante ao Schema Builder, defina uma conexão no Query Builder:

$users = DB::connection('mysql2')->select(...);

Da documentação do Laravel #

Cada conexão individual pode ser acessada através do método de conexão na fachada DB, mesmo quando há várias conexões definidas. O name passado para o método connection deve corresponder a uma das conexões listadas no arquivo de configuração config/database.php:

$users = DB::connection('foo')->select(...);

O raw também pode ser acessado, a instância PDO subjacente usando o método getPdo em uma instância de conexão:

$pdo = DB::connection()->getPdo();

https://laravel.com/docs/5.4/database#using-multiple-database-connections
