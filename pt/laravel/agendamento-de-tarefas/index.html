Criando uma tarefa #

Você pode criar uma tarefa (Comando do Console) no Laravel usando o Artisan. Da sua linha de comando:

php artisan make:console MyTaskName

Isso cria o arquivo em app/Console/Commands/MyTaskName.php. Isso parecerá assim:

<?php

namespace App\Console\Commands;

use Illuminate\Console\Command;

class MyTaskName extends Command
{
    /**
     * The name and signature of the console command.
     *
     * @var string
     */
    protected $signature = 'command:name';

    /**
     * The console command description.
     *
     * @var string
     */
    protected $description = 'Command description';

    /**
     * Create a new command instance.
     *
     * @return void
     */
    public function __construct()
    {
        parent::__construct();
    }

    /**
     * Execute the console command.
     *
     * @return mixed
     */
    public function handle()
    {
        //
    }
}

Algumas partes importantes desta definição são:

  • A propriedade $signature é o que identifica seu comando. Você poderá executar este comando mais tarde através da linha de comando usando Artisan executando php artisan command:name (onde command:name corresponde ao $signature do seu comando)
  • A propriedade $description é a ajuda/uso do Artisan ao lado do seu comando quando ele é disponibilizado.
  • O método handle() é onde você escreve o código para seu comando.

Eventualmente, sua tarefa será disponibilizada para a linha de comando através do Artisan. A propriedade protected $signature = 'command:name'; nesta classe é o que você usaria para executá-la.

Disponibilizando uma tarefa #

Você pode disponibilizar uma tarefa ao Artisan e ao seu aplicativo no arquivo app/Console/Kernel.php.

A classe Kernel contém um array chamado $commands que disponibiliza seus comandos para sua aplicação.

Adicione seu comando a este array, para torná-lo disponível para Artisan e seu aplicativo.

<?php

namespace App\Console;

use Illuminate\Console\Scheduling\Schedule;
use Illuminate\Foundation\Console\Kernel as ConsoleKernel;

class Kernel extends ConsoleKernel
{
    /**
     * The Artisan commands provided by your application.
     *
     * @var array
     */
    protected $commands = [
        Commands\Inspire::class,
        Commands\MyTaskName::class // This line makes MyTaskName available
    ];

    /**
     * Define the application's command schedule.
     *
     * @param  \Illuminate\Console\Scheduling\Schedule  $schedule
     * @return void
     */
    protected function schedule(Schedule $schedule)
    {
    
    }
}

Feito isso, agora você pode acessar seu comando pela linha de comando, usando o Artisan. Supondo que seu comando tenha a propriedade $signature definida como my:task, você pode executar o seguinte comando para executar sua tarefa:

php artisan my:task

Agendando sua tarefa #

Quando seu comando é disponibilizado para sua aplicação, você pode usar o Laravel para agendá-lo para ser executado em intervalos pré-definidos, assim como faria com um CRON.

No arquivo app/Console/Kernel.php você encontrará um método schedule que você pode usar para agendar sua tarefa.

<?php

namespace App\Console;

use Illuminate\Console\Scheduling\Schedule;
use Illuminate\Foundation\Console\Kernel as ConsoleKernel;

class Kernel extends ConsoleKernel
{
    /**
     * The Artisan commands provided by your application.
     *
     * @var array
     */
    protected $commands = [
        Commands\Inspire::class,
        Commands\MyTaskName::class
    ];

    /**
     * Define the application's command schedule.
     *
     * @param  \Illuminate\Console\Scheduling\Schedule  $schedule
     * @return void
     */
    protected function schedule(Schedule $schedule)
    {
        $schedule->command('my:task')->everyMinute();
        // $schedule->command('my:task')->everyFiveMinutes();
        // $schedule->command('my:task')->daily();
        // $schedule->command('my:task')->monthly();
        // $schedule->command('my:task')->sundays();
    }
}

Assumindo que a $signature da sua tarefa é my:task, você pode agendá-la como mostrado acima, usando o objeto Schedule $schedule. O Laravel fornece várias maneiras diferentes de agendar seu comando, conforme mostrado nas linhas comentadas acima.

Configurando o agendador para ser executado #

O agendador pode ser executado usando o comando:

php artisan schedule:run

O agendador precisa ser executado a cada minuto para funcionar corretamente. Você pode configurar isso criando um cron job com a seguinte linha, que executa o agendador a cada minuto em segundo plano.

* * * * * php /path/to/artisan schedule:run >> /dev/null 2>&1
