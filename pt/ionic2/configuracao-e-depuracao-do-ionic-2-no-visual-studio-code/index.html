Visual Studio é um IDE de código aberto que fornece intellisense e facilidade de edição para código. Este IDE suporta muitas linguagens como (Ionic, C, C#, AngularJs, TypeScript, Android e assim por diante). Essas linguagens são capazes de executar o código adicionando suas extensões no VSCode. Ao usar o VSCode, podemos executar e depurar o código de diferentes linguagens.

Crie e adicione seu projeto Ionic no VSCode #

O VsCode não pode criar o projeto iônico porque é um editor de código. Assim, você pode criar seu projeto iônico por CLI ou cmd. crie seu projeto pelo comando abaixo

$ ionic start appName blank

O comando acima usa para criar um aplicativo iônico de modelo em branco. O Ionic2 fornece três tipos de modelos em branco, guias e menu lateral. Então. você pode substituir o modelo em branco por quaisquer outros dois modelos conforme sua necessidade.

Agora, seu projeto Ionic foi criado. Assim, você pode adicionar seu projeto no VSCode para editar. Para adicionar seu projeto siga os pontos abaixo.

  1. Vá para o menu Arquivo no VScode.
  2. Clique em Abrir pasta dentro do menu Arquivo.
  3. Encontre e abra a pasta do seu projeto.

Você pode abrir a pasta diretamente usando a tecla de atalho ctrl+O ou ctrl+k

`

Instalação do VSCode #

Primeiro você precisa baixar e instalar o VSCode. Esta versão mais recente do VSCode está disponível para download em seu Site oficial . Após baixar o VSCode você deve instalá-lo e abri-lo.

 Introduction of Extensions in VSCode

O VSCode é um editor aberto, portanto, fornece editor para todos os idiomas, mas para executar um código, você precisa adicionar a extensão para esse idioma específico. Para executar e editar seu código iônico, você deve adicionar a extensão ionic2-vscode em yourVSCode. No lado esquerdo do VSCode Editor existem 5 ícones nos quais o ícone mais baixo é usado para extensão. As extensões que você pode obter usando tecla de atalho (ctrl+shift+X).

Add Extension for Ionic2 in VsCode

Ao pressionar ctrl+shift+X você mostra a parte da extensão onde no topo três pontos são mostrados … esses pontos são conhecidos como mais ícone. Ao clicar nele, uma caixa de diálogo é exibida abre e mostra o número de opções para escolher .você pode escolher a opção conforme sua necessidade mas para obter todas as extensões você deve selecionar Mostrar Extensão Recomendada .na lista de todas as extensões você pode instalar sua Extensão (ionic2- vscode),npm

Execute e depure seu projeto Ionic #

> Executar e depurar no Chrome

Para executar o projeto iônico, use o comando abaixo no terminal ou cmd ou CLI

 $ ionic serve

Para depurar o projeto ionic, primeiro você deve adicionar a extensão (Debugger for chrome) e então configurar o arquivo launch.json como este.

 {
        "version": "0.2.0",
        "configurations": [
           
            {
          "name": "Launch in Chrome",
          "type": "chrome",
          "request": "launch",
          "url": "http://localhost:8100",
          "sourceMaps": true,
          "webRoot": "${workspaceRoot}/src"
        }
]
}

****** Executar e depurar no Android****

Para executar o projeto iônico no Android, você deve adicionar a plataforma Android pelo comando abaixo no terminal ou cmd ou CLI:

$ ionic cordova platform add android

Compile o Android por este comando

$ ionic cordova build android

Executar comando para plataforma Android

$ ionic cordova run android

Agora, seu aplicativo é executado em um dispositivo Android real.
