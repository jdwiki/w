“Olá Mundo!” Programa #

Para começar a usar Vue.js, certifique-se de ter o arquivo de script incluído em seu HTML. Por exemplo, adicione o seguinte ao seu HTML.

<script src="https://npmcdn.com/vue/dist/vue.js"></script>

Exemplo Simples #

Modelo HTML #

<div id="app">
  {{ message }}
</div>

##Javascript

new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue.js!'
  }
})

Veja uma demonstração ao vivo deste exemplo.

Você também pode conferir o exemplo “Hello World” feito pelo Vue.js.

Hello World no Vue 2 (do jeito JSX) #

JSX não deve ser interpretado pelo navegador. Ele deve primeiro ser transpilado em Javascript padrão. Para usar JSX você precisa instalar o plugin para babel babel-plugin-transform-vue-JSX

Execute o Comando abaixo:

npm install babel-plugin-syntax-jsx babel-plugin-transform-vue-jsx babel-helper-vue-jsx-merge-props --save-dev

e adicione-o ao seu .babelrc assim:

{
  "presets": ["es2015"],
  "plugins": ["transform-vue-jsx"]
}

Código de exemplo com VUE JSX:

import Vue from 'vue'  
import App from './App.vue'

new Vue({  
  el: '#app',
  methods: {
    handleClick () {
      alert('Hello!')
    }
  },
  render (h) {
    return (
      <div>
        <h1 on-click={this.handleClick}>Hello from JSX</h1>
        <p> Hello World </p>
      </div>
    )
  }
})

Ao usar o JSX, você pode escrever estruturas concisas do tipo HTML/XML no mesmo arquivo em que escreve o código JavaScript.

Parabéns, você terminou :)

Manipulando a entrada do usuário #

O VueJS também pode ser usado para manipular facilmente a entrada do usuário, e a ligação bidirecional usando o v-model facilita muito a alteração de dados com facilidade.

HTML:

<script src="https://unpkg.com/vue/dist/vue.js"></script>
<div id="app">
    {{message}}
<input v-model="message"> 
</div>

JS:

new Vue({
  el: '#app',
  data: {
    message: 'Hello Vue.js!'
  }
})

É muito fácil fazer uma ligação bidirecional no VueJS usando a diretiva v-model.
