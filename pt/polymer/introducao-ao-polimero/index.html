Estrutura de elementos básicos #

Temos o seguinte elemento muito básico my-element salvo como src/my-element.html

<link rel="import" href="bower_components/polymer/polymer.html">

<dom-module id="my-element">

  <template>
    <style>
      /* local styles go here */
      :host {
        display: block;
      }
    </style>
    <!-- local DOM goes here -->
    <content></content>
  </template>

  <script>
    Polymer({
      /* this is the element's prototype */
      is: 'my-element'
    });
  </script>

</dom-module>
  • O <link> inclui a biblioteca Polymer usando uma importação HTML.
  • O <dom-module> é o wrapper DOM local para o elemento (neste caso, my-element).
  • O <template> é a definição real do DOM local.
  • O <style> dentro do <template> permite definir estilos com escopo para este elemento e seu DOM local e não afetará mais nada no documento.
  • O <content> conterá qualquer coisa que você colocar dentro do seu elemento.
  • A pseudo classe :host corresponde ao elemento personalizado (my-element).
  • A chamada Polymer registra o elemento.
  • A propriedade is é o nome do elemento (ele tem para corresponder ao <dom-module> do id)

Você pode importá-lo em seu aplicativo usando:

<link rel="import" href="src/my-element.html">

E use-o como uma tag:

<my-element>Content</my-element>

Configurando seu primeiro aplicativo de polímero a partir de um modelo #

Vamos nos preparar para construir seu próprio Progressive Web App incrível com Polymer!

Antes de começar a instalar o Polymer, você precisa do seguinte:

  • Node.js - confira a StackOverflow Installing Node.js Documentation

  • Bower - você pode instalar o Bower usando o Node Package Manager instalado com o Node.js:

    npm install -g bower

Instalando a Interface de Linha de Comando do Polymer #

O Polymer CLI fornece todas as ferramentas necessárias para Projetos Polymer:

npm install -g polymer-cli

Inicialize seu aplicativo a partir de um modelo de aplicativo #

Use polymer init para inicializar seu aplicativo a partir de um modelo de aplicativo.

Um modelo legal é o --app-drawer-template. Vamos usar isso:

polymer init app-drawer-template

Sirva seu aplicativo #

Não há necessidade de construção para servir seu primeiro aplicativo Polymer incrível. Basta servir:

polymer serve --open

Isso abrirá o aplicativo em seu navegador padrão em http://localhost:8080.

Olá Mundo #

Este exemplo cria um elemento Polymer chamado x-foo, cujo modelo se vincula a uma propriedade de string, chamada “message”. O HTML do elemento é importado para o documento principal, o que permite o uso de tags <x-foo> em <body>.

x-foo.html

<dom-module id="x-foo">
  <template>
    <span>{{message}}</span>
  </template>

  <script>
    Polymer({
      is: 'x-foo',
      properties : {
        message: {
          type: String,
          value: "Hello world!"
        }
      }
    });
  </script>
</dom-module>

index.html

<head>
  <!-- PolyGit used here as CDN for demo purposes only. In production,
       it's recommended to import Polymer and Web Components locally
       from Bower. -->
  <base href="https://polygit.org/polymer+1.6.0/components/">

  <script src="webcomponentsjs/webcomponents-lite.min.js"></script>
  <link rel="import" href="polymer/polymer.html">

  <link rel="import" href="x-foo.html">
</head>
<body>
  <x-foo></x-foo>
</body>

Usando elementos do catálogo de polímeros #

O Polymer fornece muitos elementos bem construídos para você usar em seu aplicativo.

Procure-os em seu Catálogo de Elementos.

Vamos passar pelo fluxo de trabalho de usar um elemento incluindo paper-input (Documentação)

Baixe o elemento #

Para baixar um elemento existem duas maneiras:

Bower #

A maneira conveniente é usar a linha de comando usando o comando bower install:

bower install --save PolymerElements/paper-input

Observação: --save adiciona o elemento como uma dependência ao bower.json do seu aplicativo.

arquivo ZIP #

A outra maneira é adicionar o elemento selecionado (paper-input neste caso) à sua coleção (no Polymer Catalog) usando “Adicionar à coleção” na navegação e baixar sua coleção usando o ícone de estrela no canto superior direito .

Isso gerará um arquivo .zip que contém o elemento e todas as suas dependências. Você pode então copiar a pasta bower_components dentro de .zip/components para o diretório raiz do seu aplicativo.

Importe o elemento em seu aplicativo #

Para importar o elemento que você acabou de instalar, importe o arquivo .html correspondente:

<link rel="import" href="bower_components/paper-input/paper-input.html">

Use o elemento #

Agora você pode usar paper-input dentro do documento para o qual o importou:

<paper-input></paper-input>
