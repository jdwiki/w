Sintaxe #

  • $(função(){ $( “#selecter” ).accordion(); });
  • $(função(){ $( “#selecter” ).accordion({ active: 2 }); });
  • $(função(){ $( “#selecter” ).accordion({ animate: 200 }); });
  • $(função(){ $( “#selecter” ).accordion({ collapsible: true }); });

Parâmetros #

Parâmetro Detalhe
ativo Digite Boolean ou Integer, Boolean requer recolhível para ser verdadeiro
animar Digite Boolean, Number, String ou Object
dobrável Tipo Booleano
evento Tipo String
cabeçalho Elemento seletor de tipo
alturaEstilo Tipo String
ícones Digite objeto de ícone da interface do usuário do jQuery

Mais informações podem ser encontradas aqui: http://api.jqueryui.com/accordion/

Uso básico do acordeão #

Para usar um acordeão, deve-se ter cabeçalhos e conteúdo dentro dos cabeçalhos em seu HTML. Então deve-se instanciar o método accordion() do jQuery UI.

<script>
$(function() {
    $( "#accordion" ).accordion();
});
</script>

No HTML:

<div id="accordion">
    <h3>First header</h3>
        <div>First content panel</div>
    <h3>Second header</h3>
        <div>Second content panel</div>
</div>

Acordeão destrói o uso #

$( "#accordion" ).accordion( "destroy" );

Isso removerá completamente a funcionalidade de acordeão e mostrará o HTML padrão removendo todos os elementos jQuery-UI.

Este método não aceita argumentos.

Desativar o uso do acordeão #

$( "#accordion" ).accordion( "disable" );

Este método irá desabilitar o acordeão, ou seja, os cabeçalhos não são selecionáveis ​​tornando o conteúdo somente leitura e estático.

Este método não aceita argumentos.

Acordeão habilitar Uso #

$( "#accordion" ).accordion( "enable" );

Este método habilitará um acordeão. Isso habilitará um acordeão desabilitado ou simplesmente não fará nada em um acordeão já habilitado.

Este método não aceita argumentos.

Uso da opção de acordeão #

var options = $( "#accordion" ).accordion( "option" );

Isso retornará um PlainObject fornecendo todas as opções que representam o acordeão selecionado. Isso conterá todos os valores das chaves que são explicados na seção Parâmetros.

Este método recebe parâmetros que são os optionNames básicos explicados no parâmetro. Pode-se definir as opções assim:

$( "#accordion" ).accordion( "option", "disabled", true );

Uso de atualização de acordeão #

$( "#accordion" ).accordion( "refresh" );

Este método recalcula a altura dos painéis sanfonados se cabeçalhos ou conteúdo foram adicionados ou removidos no DOM.

Uso de widget de acordo com #

var widget = $( "#accordion" ).accordion( "widget" );

Este método retorna um objeto jQuery contendo o acordeão.
