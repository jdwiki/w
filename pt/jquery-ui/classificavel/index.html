Sintaxe #

  • $("#sortable").sortable({ /*Opções aqui*/ }); // Inicializa Classificável

  • $("#sortable").sortable(“option”, “option_name”, option_value); //Definir opção fora do inicializador

  • valor var = $("#sortable").sortable(“option”, “option_name”); // Obtém o valor de uma opção

Parâmetros #

Parâmetro Descrição
Opções  
appendTo (jQuery, Element, Selector, String) [Padrão: “pai”] O elemento ao qual o auxiliar é adicionado
eixo (String) [Padrão: false] As direções que o item pode ser arrastado (x ou y)
cancelar (Seletor) [Padrão: “input,textarea,button,select,option”] Não inicia a ordenação se você iniciar em um elemento que corresponda ao seletor
aulas (Objeto) [Padrão: {}] Especifique classes adicionais para adicionar aos elementos classificáveis ​​quando as classes estruturais forem adicionadas. ({ui-sortable-helper: custom_class})
conectarCom (Seletor) [Padrão: false] Permite que itens de um classificável sejam arrastados para outro
contenção (Elemento, Seletor, String) [Padrão: false] O elemento ao qual os itens são restritos
cursor (String) [Padrão: “auto”] Define o tipo de cursor a ser mostrado na ordenação
cursorAt (Objeto) [Padrão: false] Define a posição em que o auxiliar parece estar sendo movido
desativado (Boolean) [Padrão: false] Desativa a classificação se true
dropOnEmpty (Boolean) [Padrão: true] Se falsos itens deste classificável não puderem ser colocados em classificáveis ​​vazios
forceHelperSize (Boolean) [Padrão: false] Força o auxiliar a ter um tamanho
forcePlaceholderSize (Boolean) [Padrão: false] Força o espaço reservado a ter um tamanho
grade (Array) [Padrão: false] Define uma grade para encaixar o auxiliar ([ x,y ])
manusear (Seletor, Elemento) [Padrão: false] Define os elementos nos quais a classificação pode começar. Oposto a cancelar
ajudante (String, Function) [Padrão: “original”] String “original” ou “clone”, ou função que retorna o elemento a ser usado como auxiliar.
artigos (Seletor) [Padrão: “> *”] Define os itens que devem ser classificáveis ​​
opacidade (Número 0.01 a 1) [Padrão: false] Define a opacidade do auxiliar
marcador de posição (String) [Padrão: false] Define uma classe ou classes a serem aplicadas ao espaço reservado
reverter (Boolean, Number) [Padrão: false] O tempo que leva para o auxiliar deslizar para sua nova posição
rolar (Boolean) [Padrão: true] Se deve rolar quando estiver nas bordas da página
scrollSensibilidade (Número) [Padrão: 20] Define o quão próximo da borda da página o cursor precisa estar para iniciar a rolagem
velocidade de rolagem (Número) [Padrão: 20] A velocidade na qual rolar
tolerância (String) [Padrão: “intersect”] Define qual modo usar ao calcular quando um item está sobre outro (“intersect” ou “ponteiro”)
zIndex (Inteiro) [Padrão: 1000] Define o z-index do auxiliar ao ordenar
Métodos  
cancelar() Cancela a classificação atual e retorna os elementos à sua posição anterior ao início da classificação
destruir() Remove a funcionalidade classificável e retorna o elemento ao seu estado pré-inicialização
desativar() Desativa o classificável
habilitar() Habilita o classificável
instância() Retorna o objeto de instância classificável
opção() Obtém pares de valores-chave de todas as opções para o classificável
opção(String) Obtém o valor de uma opção
opção(String, Qualquer) Define o valor da opção especificada pela String
opção(Objeto) Define uma ou mais opções com o objeto sendo pares de valores-chave de opções
atualizar() Atualiza as opções classificáveis ​​recarregando todos os itens classificáveis. Isso faz com que novos itens sejam reconhecidos
atualizarPosições() Refreca as posições em cache dos itens classificáveis ​​
serialize(Objeto) Serializa os ids dos itens (por padrão) em uma string que pode ser enviada ou anexada a um url Object options: {key: define a chave na string serializada, attribute:[Default “id”] define o atributo a ser observado, expressão :[Padrão: “/(.+)-=_/”] regex para dividir o atributo em pares de valores-chave}
toArray(Objeto) Serializa o id dos itens classificáveis ​​em uma matriz. O objeto pode conter um atributo de parâmetro que tem o atributo para colocar no array o padrão é id
widget() Retorna um objeto jQuery do elemento classificável
Eventos  
ativar(evento, ui) Acionado quando a lista conectada, todas as listas conectadas no início do arrasto a recebem
beforeStop(evento, ui) Os acionadores antes da classificação param quando o auxiliar é deslocado para a mesma posição que o espaço reservado
change(evento, ui) Acionado quando os elementos mudam de posição, ou seja. quando o espaço reservado se move
create(evento, ui) Acionado quando o classificável é criado
deactivate(evento, ui) Acionado quando a classificação é interrompida. Isso vai para todas as listas conectadas também
out(evento, ui) Acionado quando o item é movido para fora da lista classificável
over(evento, ui) Acionado quando o item é movido para uma lista classificável
receber(evento, ui) Acionado quando um item de uma lista conectada é colocado em outra. O destino é a lista de recebimento
remove(evento, ui) Acionado quando um item de uma lista conectada é colocado em outra. Alvo é a lista de doação
sort(evento, ui) Acionado durante a classificação
start(evento, ui) Acionado quando a classificação é iniciada
stop(evento, ui) Acionado quando a classificação é interrompida
update(event, ui) Disparado quando a ordenação é interrompida e a posição DOM está sendo atualizada

Grade classificável com layout flexível #

Isso usou o layout flexível com o classificável para criar uma grade de caixas responsivas que podem ser movidas arrastando e soltando.

HTML

<div id="sortable">
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
  <div>5</div>
</div>

JS

$(function(){
    $('#sortable').sortable({
        //pass all options in here
    });
});

CSS

#sortable{
    width: 500px;
    display: flex;
    flex-wrap: wrap;
}
#sortable div {
    margin: 10px;
    background-color: #f00;
    flex-basis: 100px;
    height: 100px;
}

Itens estacionários ao arrastar #

Este exemplo usa uma classe no espaço reservado para transformá-lo em uma linha e fazer com que não ocupe espaço.

HTML

<div id="sortable">
    <div>1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
</div>

JS

$("#sortable").sortable({
    placeholder: 'placeholder',
    helper: 'clone',
    start: function(event, ui){
        ui.item.show();
    }
});

CSS

#sortable div{
  background-color: #f00;
  width: 50px;
  height: 50px;
  margin: 10px;
  padding: 0px;
}
#sortable div.placeholder{
  height: 4px;
  margin: -7px 10px;
}

Classificável - Animar reversão de item não aceito #

Exemplo de trabalho: https://jsfiddle.net/Twisty/4f5yh3pa/7/

Cancelar e reverter um classificável não está fortemente documentado. A ajuda mostra como mover um item de uma lista para outra lista conectada pode ser cancelado condicionalmente. por padrão, isso não é animado por classificável, este exemplo inclui uma animação.

Resultado: a lista nº 2 aceita apenas itens que tenham uma classe de aceitável. Ambas as listas podem ser classificadas naturalmente de outra forma.

HTML

<div class="ui-widget">
  <ul id="sortable1" class="connectedSortable">
    <li class="ui-state-default acceptable">Item 1</li>
    <li class="ui-state-default">Item 2</li>
    <li class="ui-state-default">Item 3</li>
    <li class="ui-state-default">Item 4</li>
    <li class="ui-state-default">Item 5</li>
  </ul>
  <ul id="sortable2" class="connectedSortable">
    <li class="ui-state-default">Item 6</li>
    <li class="ui-state-default acceptable">Item 7</li>
  </ul>
</div>

CSS

.ui-widget {
  position: relative;
}

.connectedSortable {
  border: 1px solid #eee;
  width: 142px;
  min-height: 20px;
  list-style-type: none;
  margin: 0;
  padding: 5px 0 0 0;
  float: left;
  margin-right: 10px;
}

#sortable1 {
  background: #fff;
}

#sortable2 {
  background: #999;
}

.connectedSortable li {
  margin: 0 5px 5px 5px;
  padding: 5px;
  font-size: 1.2em;
  width: 120px;
}

JavaScript

$(function() {
  $(".connectedSortable").sortable({
    connectWith: ".connectedSortable",
    receive: function(e, ui) {
      var $self = $(this);
      var $item = ui.item;
      var $sender = ui.sender;
      // Restrict condition to only one specific list if desired
      if ($(e.target).attr("id") == "sortable2") {
        if ($item.hasClass("acceptable")) {
          // Item Accepted
          console.log($self.attr("id") + " accepted item from: #" + $sender.attr("id") + " > " + $item.text());
        } else {
          // Item Rejected
          console.log($self.attr("id") + " rejected item from: #" + $sender.attr("id") + " > " + $item.text());
          // Animate the return of the items position
          $item.css("position", "absolute").animate(ui.originalPosition, "slow", function() {
            // Return the items position control to it's parent
            $item.css("position", "inherit");
            // Cancel the sortable action to return it to it's origin
            $sender.sortable("cancel");
          });
        }
      }
    }
  }).disableSelection();
});

Exemplo Simples #

Pegue qualquer lista e adicione um identificador ao wrapper externo (ul, div)

<ul id="sortable">
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
<li>Item 4</li>
</ul>

Na sua jquery:

$(function(){
    $('#sortable').sortable({
        //pass all options in here
    });
});

Isso permitirá que todos os li no wrapper #sortable sejam arrastados e soltos na lista
