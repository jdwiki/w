Soma de Gauss: 1 + 2 + 3 + … + n #

A soma

1 + 2 + 3 + … + n

Simplifica para

n(n+1)/2.

Observe que esta quantidade é Θ(n2).

Este atalho surge frequentemente na análise de algoritmos como ordenação por inserção ou ordenação por seleção.

Números da forma n(n+1)/2 são chamados os números triangulares.

Soma de uma série geométrica: r^0 + r^1 + r^2 + … #

A soma da série geométrica

r0 + r1 + r2 + … + rn-1

No caso em que r ≠ 1, simplifica para (rn - 1) / (r - 1). Se r < 1, esta soma é limitada a partir de cima por 1 / (1 - r).

Se r = 1, esta soma é rn.

Somas dos Números de Fibonacci #

Os números de Fibonacci são definidos indutivamente como

  • F0 = 0
  • F1 = 1
  • Fn+2 = Fn + Fn+1

A soma dos primeiros n+1 números de Fibonacci é dada por

F0 + F1 + F2 + … + Fn = Fn +2 - 1.

Esse somatório surge, entre outros lugares, na análise de heaps de Fibonacci, onde é usado para fornecer um limite inferior no número de nós em cada árvore no heap.

Somas de potências de dois: 1 + 2 + 4 + 8 + 16 + … #

A soma

20 + 21 + 22 + … + 2n-1

simplifica para 2n - 1. Isso explica por que o valor máximo que pode ser armazenado em um inteiro de 32 bits sem sinal é 232 - 1.

Somas de postes de vedação #

Aqui consideramos somas da forma

a + b + a + b + … a

vs.

a + b + a + b + … b

Para visualizar essas somas, imagine uma seção de cerca alternando entre postes e trilhos. Três cenários são possíveis.

  1. Imagine uma seção de cerca com postes em cada extremidade, conectados por trilhos. n trilhos requerem n+1 postes. Por outro lado, os postes p são unidos por trilhos p-1.

    |—|—|—|

  1. Imagine uma seção de cerca com um poste em uma extremidade, mas um trilho aberto na outra. n trilhos requerem n postes.

    |—|—|—

    or

    —|—|—|

  1. Imagine uma seção de cerca com trilhos abertos em ambas as extremidades. n trilhos requerem n-1 postes. Por outro lado, p postes são unidos por trilhos p+1.

    —|—|—

Cálculos como este surgem em situações como o layout de objetos gráficos onde os tamanhos dos objetos devem ser somados e os espaços entre os objetos também devem ser somados. Em tais situações é importante estar ciente se a intenção é ou não ter espaços em cada extremidade.

A largura total de tal cerca será sempre:

(largura do poste) x (número de postes) + (largura do trilho) x (número de trilhos)

Mas deve-se ter cuidado ao calcular o número de postes e o número de trilhos para evitar o chamado erro off-by-one. Erros desse tipo são comuns.

Somas dos recíprocos: 1/1 + 1/2 + 1/3 + 1/4 + … #

A soma

1/1 + 1/2 + 1/3 + 1/4 + … + 1/n

é igual ao enésimo número harmônico, denotado Hn. O enésimo número harmônico obedece às desigualdades

ln(n+1) & and; H<so ≤ (lnn) +

e, portanto, Hn = Θ(log n). Os números harmônicos geralmente surgem na análise de algoritmos, sendo o quicksort aleatório um exemplo particularmente interessante.

Somas de quadrados recíprocos: 1/1 + 1/4 + 1/9 + 1/16 + 1/25 + … #

A soma

1/1 + 1/4 + 1/9 + 1/16 + …

out to infinito converge para π2 / 6 e, portanto, qualquer soma da forma

1/1 + 1/4 + 1/9 + 1/16 + … + 1/n2

é &Teta;(1).
