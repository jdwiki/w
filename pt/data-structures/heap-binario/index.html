Heap Binário

  • Exemplo

Nesta página

  • Exemplo

Um heap binário é uma árvore binária completa que satisfaz a propriedade de ordenação do heap. A ordenação pode ser de dois tipos: a propriedade min-heap: o valor de cada nó é maior ou igual ao valor de seu pai, com o elemento de valor mínimo na raiz.

Exemplo #

Min-Heap

        1
      /   \
     2     3
    / \   / \
   4   5 6   7

A árvore acima é um Min-Heap, pois a raiz é o mínimo entre todos os nós presentes na árvore. A mesma propriedade é seguida por todos os nós da árvore.

Pilha Máxima

        7
      /   \
     6     5
    / \   / \
   4   3 2   1

A árvore acima é um Max-Heap, pois a raiz é o máximo entre todos os nós presentes na árvore. A mesma propriedade é seguida por todos os nós da árvore.

Operações suportadas por Min-Heap

  1. getMin() - Retorna o elemento raiz. Como é o primeiro elemento em um array podemos recuperar o elemento mínimo em O(1)

  2. extractMin() - Remove o elemento mínimo do heap. Como após a remoção do elemento a árvore deve satisfazer a propriedade Min-Heap para que seja realizada uma operação (heapifying) para manter a propriedade tree .Isso leva O(logn)

  3. dereaseKey() - Diminui o valor da chave. A complexidade de tempo para esta operação é O(logn)

  4. insert() - A chave é sempre inserida no final da árvore. Se a chave adicionada não seguir a propriedade heap, precisamos percolar para que a árvore satisfaça a propriedade heap. Este passo leva leva tempo O(logn).

  5. delete() - Esta etapa leva tempo O(logn). Para excluir uma chave, primeiro precisamos diminuir o valor da chave para um valor mínimo e depois extrair esse valor mínimo.

Heap é usado em muitos dos algoritmos de grafo como Dijkstra's Shortest Path e Prim's Minimum Spanning Tree.

Implementação em Java

public class MinHeap {
    
    int hArr[];
    int capacity;
    int heapSize;
    
    public MinHeap(int capacity){
        this.heapSize = 0;
        this.capacity = capacity;
        hArr = new int[capacity];
    }
    
    public int getparent(int i){
        return (i-1)/2;
    }
    
    public int getLeftChild(int i){
        return 2*i+1;
    }
    
    public int getRightChild(int i){
        return 2*i+2;
    }
    
    public void insertKey(int k){
        if(heapSize==capacity)
            return;
        
        heapSize++;
        int i = heapSize-1;
        hArr[i] = k;
        
        while(i!=0 && hArr[getparent(i)]>hArr[i]){
            swap(hArr[i],hArr[getparent(i)]);
            i = getparent(i);
        }
    }
    
    public int extractMin(){
        if(heapSize==0)
            return Integer.MAX_VALUE;
        
        if(heapSize==1){
            heapSize--;
            return hArr[0];
        }
        
        int root = hArr[0];
        hArr[0] = hArr[heapSize-1];
        heapSize--;
        MinHeapify(0);
        
        return root;
    }
    
    public void decreaseKey(int i , int newVal){
        hArr[i] = newVal;
        while(i!=0 && hArr[getparent(i)]>hArr[i]){
            swap(hArr[i],hArr[getparent(i)]);
            i = getparent(i);
        }
    }
    
    public void deleteKey(int i){
        decreaseKey(i, Integer.MIN_VALUE);
        extractMin();
    }
    
    public void MinHeapify(int i){
        int l = getLeftChild(i);
        int r = getRightChild(i);
        int smallest = i;
        if(l<heapSize && hArr[l] < hArr[i])
            smallest = l;
        if(l<heapSize && hArr[r] < hArr[smallest])
            smallest = r;
        
        if(smallest!=i){
            swap(hArr[i], hArr[smallest]);
            MinHeapify(smallest);
        }
    }
    
    public void swap(int x, int y){
        int temp = hArr[x];
        hArr[x] = hArr[y];
        hArr[y] = temp;
    }
}
