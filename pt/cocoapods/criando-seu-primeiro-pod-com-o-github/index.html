Criando seu primeiro pod com o Github #

Abrir repositório

  1. Crie novo repositório (Meu conselho primeiro adicione pods em um repositório temporário antes de adicionar ao seu projeto original)
  2. Preencha o nome do repositório com o nome do seu repositório. Usarei “myrepo” para este tutorial e usarei os símbolos “<” e “>” para as partes que você deve alterar para as suas.
  3. Clique em “Inicializar este repositório com um README”
  4. Adicione uma licença. Vou usar “MIT” para este tutorial.

Puxe seu repositório para o local

  1. Abra o Terminal.
  2. Vá onde seu repositório é cd </your/path/<myrepo> ou crie-o com cd </your/path>. mkdir <myrepo>. cd <myrepo>
  3. Adicione git à sua pasta com git init
  4. Vincule seu repositório com git remote add origin https://github.com/<username>/<myrepo>.git. (Você pode verificar se está funcionando com git remote -v (Não git remove -v))
  5. Puxe seu repositório do github com git pull
  6. Digite: git pull origin master
  7. Digite git commit

Adicione Cocoapods

  1. Instale os cocoapods: sudo gem install cocoapods. Escreva pod --help para verificar se está funcionando. (Pule se você já tem cocoapods)
  2. Crie <myrepo>.podspec com touch <myrepo>.podspec
  3. Abra <myrepo>.podspec com um editor. Use Xcode ou Sublime Text.
  4. Cole o seguinte. Você pode conferir este exemplo. Para obter detalhes, consulte a documentação
Pod::Spec.new do |s|
s.name = "<myrepo>"
s.versão = "0.1"
s.summary = "<Meu resumo>"
s.description = "<Deve ser maior que o resumo>"
s.homepage = "https://github.com/<username>/<myrepo>"
s.license = 'MEU'
s.author = { "<username>" => "<useremail>" }
s.source = { :git => "https://github.com/<username>/<myrepo>.git", :tag => s.version.to_s }
s.platform = :ios, '8.0'
s.requires_arc = true

# Se houver mais de um arquivo de origem: https://guides.cocoapods.org/syntax/podspec.html#source_files
s.source_files = '<myrepo>.swift'

fim
  1. Escreva pod lib lint <myrepo>.podspec no terminal. Se você tiver algum erro escreva pod lib lint <myrepo>.podspec --verbose. Resolva seus erros (resolver seus erros pode levar algum tempo, não pode ajudar aqui).
  2. Adicione novas alterações ao git com git add . e confirme com git commit -m "Pods adicionados"
  3. Envie suas alterações para seu repositório remoto com git push
  4. Crie uma nova versão https://github.com// Clique em Releases -> Clique em New Version
  5. Escreva “0.1” na Versão do Tag. (Opcional você pode preencher o “Título do Release”). Certifique-se de que a versão seja idêntica ao s.version dentro de <myrepo>.podspec
  6. Escreva pod trunk register name@example.org 'Your Name' --description='macbook pro'
  7. Após o seu novo lançamento, abra o terminal e escreva pod trunk push <myrepo>.podspec. Isso mudará as CocoaPods Specs que permitirá que você escreva pod '<myrepo>' dentro do seu podfile
