Calendário #

VBA suporta 2 calendários: Gregorian e Hijri

A propriedade Calendar é usada para modificar ou exibir o calendário atual.

Os 2 valores para o Calendário são:

Valor Constante Descrição
0 vbCalGreg Calendário gregoriano (padrão)
1 vbCalHijri calendário islâmico

Exemplo #

Sub CalendarExample()
    'Cache the current setting.
    Dim Cached As Integer
    Cached = Calendar

    ' Dates in Gregorian Calendar
    Calendar = vbCalGreg
    Dim Sample As Date
    'Create sample date of 2016-07-28
    Sample = DateSerial(2016, 7, 28)

    Debug.Print "Current Calendar : " & Calendar
    Debug.Print "SampleDate = " & Format$(Sample, "yyyy-mm-dd")
    
    ' Date in Hijri Calendar
    Calendar = vbCalHijri
    Debug.Print "Current Calendar : " & Calendar
    Debug.Print "SampleDate = " & Format$(Sample, "yyyy-mm-dd")
    
    'Reset VBA to cached value.
    Cached = Calendar
End Sub

Este Sub imprime o seguinte;

Current Calendar : 0
SampleDate = 2016-07-28
Current Calendar : 1
SampleDate = 1437-10-23

Funções básicas #

Recuperar DataHora do Sistema #

O VBA suporta 3 funções internas para recuperar a data e/ou hora do relógio do sistema.

Função Tipo de retorno Valor de retorno
Agora Data Retorna a data e hora atuais
Data Data Retorna a parte da data da data e hora atuais
Hora Data Retorna a parte da hora da data e hora atuais 
Sub DateTimeExample()

    ' -----------------------------------------------------
    ' Note : EU system with default date format DD/MM/YYYY
    ' -----------------------------------------------------
    
    Debug.Print Now   ' prints 28/07/2016 10:16:01 (output below assumes this date and time)
    Debug.Print Date  ' prints 28/07/2016
    Debug.Print Time  ' prints 10:16:01
    
    ' Apply a custom format to the current date or time
    Debug.Print Format$(Now, "dd mmmm yyyy hh:nn")  ' prints 28 July 2016 10:16
    Debug.Print Format$(Date, "yyyy-mm-dd")         ' prints 2016-07-28
    Debug.Print Format$(Time, "hh") & " hour " & _
                Format$(Time, "nn") & " min " & _
                Format$(Time, "ss") & " sec "       ' prints 10 hour 16 min 01 sec
    
End Sub

Função de temporizador #

A função Timer retorna um Single representando o número de segundos decorridos desde a meia-noite. A precisão é de um centésimo de segundo.

Sub TimerExample()

    Debug.Print Time    ' prints 10:36:31  (time at execution)
    Debug.Print Timer   ' prints 38191,13  (seconds since midnight)

End Sub

Como as funções Now e Time são precisas apenas em segundos, Timer oferece uma maneira conveniente de aumentar a precisão da medição do tempo:

Sub GetBenchmark()
       
    Dim StartTime  As Single
    StartTime = Timer       'Store the current Time
    
    Dim i As Long
    Dim temp As String
    For i = 1 To 1000000    'See how long it takes Left$ to execute 1,000,000 times
        temp = Left$("Text", 2)
    Next i
    
    Dim Elapsed As Single
    Elapsed = Timer - StartTime
    Debug.Print "Code completed in " & CInt(Elapsed * 1000) & " ms"

End Sub

IsDate() #

IsDate() testa se uma expressão é uma data válida ou não. Retorna um Booleano.

Sub IsDateExamples()

    Dim anything As Variant
       
    anything = "September 11, 2001"

    Debug.Print IsDate(anything)    'Prints True
            
    anything = #9/11/2001#

    Debug.Print IsDate(anything)    'Prints True
   
    anything = "just a string"

    Debug.Print IsDate(anything)    'Prints False

    anything = vbNull
    
    Debug.Print IsDate(anything)    'Prints False
    
End Sub

Funções de extração #

Essas funções usam um Variant que pode ser convertido em um Date como parâmetro e retorna um Integer representando uma parte de uma data ou hora. Se o parâmetro não puder ser convertido em uma Data, resultará em um erro de tempo de execução 13: Tipo incompatível.

Função Descrição Valor devolvido
Ano() Retorna a parte do ano do argumento de data. Inteiro (100 a 9999)
Mês() Retorna a parte do mês do argumento de data. Inteiro (1 a 12)
Dia() Retorna a parte do dia do argumento de data. Inteiro (1 a 31)
Dia da Semana() Retorna o dia da semana do argumento de data. Aceita um segundo argumento opcional definindo o primeiro dia da semana Inteiro (1 a 7)
Hora() Retorna a parte da hora do argumento de data. Inteiro (0 a 23)
Minuto() Retorna a parte do minuto do argumento de data. Inteiro (0 a 59)
Segundo() Retorna a segunda parte do argumento de data. Inteiro (0 a 59)

Exemplos:

Sub ExtractionExamples()

    Dim MyDate As Date
    
    MyDate = DateSerial(2016, 7, 28) + TimeSerial(12, 34, 56)

    Debug.Print Format$(MyDate, "yyyy-mm-dd hh:nn:ss") ' prints 2016-07-28 12:34:56

    Debug.Print Year(MyDate)                           ' prints 2016
    Debug.Print Month(MyDate)                          ' prints 7
    Debug.Print Day(MyDate)                            ' prints 28
    Debug.Print Hour(MyDate)                           ' prints 12
    Debug.Print Minute(MyDate)                         ' prints 34
    Debug.Print Second(MyDate)                         ' prints 56
    
    Debug.Print Weekday(MyDate)                        ' prints 5
    'Varies by locale - i.e. will print 4 in the EU and 5 in the US
    Debug.Print Weekday(MyDate, vbUseSystemDayOfWeek)
    Debug.Print Weekday(MyDate, vbMonday)              ' prints 4
    Debug.Print Weekday(MyDate, vbSunday)              ' prints 5
    
End Sub

Função DatePart() #

DatePart() também é uma função que retorna uma parte de uma data, mas funciona de forma diferente e permite mais possibilidades do que as funções acima. Ele pode, por exemplo, retornar o Trimestre do ano ou a Semana do ano.

Sintaxe:

DatePart ( interval, date  [, firstdayofweek] [, firstweekofyear] )

interval argumento pode ser:

Intervalo Descrição
“aaaa” Ano (100 a 9999)
“s” Dia do ano (1 a 366)
“m” Mês (1 a 12)
“q” Trimestre (1 a 4)
“ww” Semana (1 a 53)
“w” Dia da semana (1 a 7)
“d” Dia do mês (1 a 31)
“h” Hora (0 a 23)
“n” Minuto (0 a 59)
“s” Segundo (0 a 59)

primeiro diadasemana é opcional. é uma constante que especifica o primeiro dia da semana. Se não for especificado, vbSunday é assumido.

primeira semana do ano é opcional. é uma constante que especifica a primeira semana do ano. Se não for especificado, a primeira semana será considerada a semana em que 1º de janeiro ocorre.

Exemplos:

Sub DatePartExample()

    Dim MyDate As Date
    
    MyDate = DateSerial(2016, 7, 28) + TimeSerial(12, 34, 56)

    Debug.Print Format$(MyDate, "yyyy-mm-dd hh:nn:ss") ' prints 2016-07-28 12:34:56
    
    Debug.Print DatePart("yyyy", MyDate)              ' prints 2016
    Debug.Print DatePart("y", MyDate)                 ' prints 210
    Debug.Print DatePart("h", MyDate)                 ' prints 12
    Debug.Print DatePart("Q", MyDate)                 ' prints 3
    Debug.Print DatePart("w", MyDate)                 ' prints 5
    Debug.Print DatePart("ww", MyDate)                ' prints 31

End Sub

Funções de cálculo #

DateDiff() #

DateDiff() retorna um Long representando o número de intervalos de tempo entre duas datas especificadas.

Sintaxe

DateDiff ( interval, date1, date2  [, firstdayofweek] [, firstweekofyear] )
  • interval pode ser qualquer um dos intervalos definidos na função [DatePart()](https://www.wikiod.com/pt/vba/manipulacao-de-data-e-hora#Funções de extração)
  • date1 e date2 são as duas datas que você deseja usar no cálculo
  • firstdayofweek e firstweekofyear são opcionais. Consulte a função [DatePart()](https://www.wikiod.com/pt/vba/manipulacao-de-data-e-hora#Funções de extração) para obter explicações

Exemplos

Sub DateDiffExamples()

    ' Check to see if 2016 is a leap year.
    Dim NumberOfDays As Long
    NumberOfDays = DateDiff("d", #1/1/2016#, #1/1/2017#)
    
    If NumberOfDays = 366 Then
        Debug.Print "2016 is a leap year."              'This will output.
    End If
           
    ' Number of seconds in a day
    Dim StartTime As Date
    Dim EndTime As Date
    StartTime = TimeSerial(0, 0, 0)
    EndTime = TimeSerial(24, 0, 0)
    Debug.Print DateDiff("s", StartTime, EndTime)       'prints 86400

End Sub

DataAdicionar() #

DateAdd() retorna uma Date à qual uma data ou intervalo de tempo especificado foi adicionado.

Sintaxe

DateAdd ( interval, number, date  )
  • interval pode ser qualquer um dos intervalos definidos na função [DatePart()](https://www.wikiod.com/pt/vba/manipulacao-de-data-e-hora#Funções de extração)
  • number Expressão numérica que é o número de intervalos que você deseja adicionar. Pode ser positivo (para obter datas no futuro) ou negativo (para obter datas no passado).
  • data é uma Data ou literal que representa a data à qual o intervalo é adicionado

Exemplos :

Sub DateAddExamples()

    Dim Sample As Date
    'Create sample date and time of 2016-07-28 12:34:56
    Sample = DateSerial(2016, 7, 28) + TimeSerial(12, 34, 56)
    
    ' Date 5 months previously (prints 2016-02-28):
    Debug.Print Format$(DateAdd("m", -5, Sample), "yyyy-mm-dd")
    
    ' Date 10 months previously (prints 2015-09-28):
    Debug.Print Format$(DateAdd("m", -10, Sample), "yyyy-mm-dd")
    
    ' Date in 8 months (prints 2017-03-28):
    Debug.Print Format$(DateAdd("m", 8, Sample), "yyyy-mm-dd")

    ' Date/Time 18 hours previously (prints 2016-07-27 18:34:56):
    Debug.Print Format$(DateAdd("h", -18, Sample), "yyyy-mm-dd hh:nn:ss")
    
    ' Date/Time in 36 hours (prints 2016-07-30 00:34:56):
    Debug.Print Format$(DateAdd("h", 36, Sample), "yyyy-mm-dd hh:nn:ss")

End Sub

Conversão e Criação #

CDate() #

CDate() converte algo de qualquer tipo de dados para um tipo de dados Date

Sub CDateExamples()

    Dim sample As Date

    ' Converts a String representing a date and time to a Date
    sample = CDate("September 11, 2001 12:34")
    Debug.Print Format$(sample, "yyyy-mm-dd hh:nn:ss")      ' prints 2001-09-11 12:34:00
    
    ' Converts a String containing a date to a Date
    sample = CDate("September 11, 2001")
    Debug.Print Format$(sample, "yyyy-mm-dd hh:nn:ss")      ' prints 2001-09-11 00:00:00

    ' Converts a String containing a time to a Date
    sample = CDate("12:34:56")
    Debug.Print Hour(sample)                        ' prints 12
    Debug.Print Minute(sample)                      ' prints 34
    Debug.Print Second(sample)                      ' prints 56
    
    ' Find the 10000th day from the epoch date of 1899-12-31
    sample = CDate(10000)
    Debug.Print Format$(sample, "yyyy-mm-dd")       ' prints 1927-05-18
    
End Sub

Observe que o VBA também tem um CVDate() vagamente tipado que funciona da mesma forma que a função CDate() exceto retornar uma data digitada Variant em vez de uma Data fortemente tipada. A versão CDate() deve ser preferida ao passar para um parâmetro Date ou atribuir a uma variável Date, e a versão CVDate() deve ser preferida ao passar para um parâmetro Variant ou atribuir a uma variável Variante. Isso evita a conversão implícita de tipo.

DataSerial() #

A função DateSerial() é usada para criar uma data. Ele retorna uma Data para um ano, mês e dia especificados.

Sintaxe:

DateSerial ( year, month, day )

Com argumentos de ano, mês e dia válidos Inteiros (Ano de 100 a 9999, Mês de 1 a 12, Dia de 1 a 31).

Exemplos

Sub DateSerialExamples()

    ' Build a specific date
    Dim sample As Date
    sample = DateSerial(2001, 9, 11)
    Debug.Print Format$(sample, "yyyy-mm-dd")                   ' prints 2001-09-11
    
    ' Find the first day of the month for a date.
    sample = DateSerial(Year(sample), Month(sample), 1)
    Debug.Print Format$(sample, "yyyy-mm-dd")                   ' prints 2001-09-11
    
    ' Find the last day of the previous month.
    sample = DateSerial(Year(sample), Month(sample), 1) - 1
    Debug.Print Format$(sample, "yyyy-mm-dd")                   ' prints 2001-09-11
    
End Sub

Observe que DateSerial() aceitará datas “inválidas” e calculará uma data válida a partir delas. Isso pode ser usado criativamente para o bem:

Exemplo Positivo

Sub GoodDateSerialExample()

    'Calculate 45 days from today
    Dim today As Date
    today = DateSerial (2001, 9, 11)
    Dim futureDate As Date
    futureDate = DateSerial(Year(today), Month(today), Day(today) + 45)
    Debug.Print Format$(futureDate, "yyyy-mm-dd")            'prints 2009-10-26

Finalizar Sub

No entanto, é mais provável que cause problemas ao tentar criar uma data a partir de uma entrada de usuário não validada:

Exemplo Negativo

Sub BadDateSerialExample()

    'Allow user to enter unvalidate date information
    Dim myYear As Long
    myYear = InputBox("Enter Year")                                         
            'Assume user enters 2009
    Dim myMonth As Long
    myMonth = InputBox("Enter Month")                                       
            'Assume user enters 2
    Dim myDay As Long
    myDay = InputBox("Enter Day")                                           
            'Assume user enters 31
    Debug.Print Format$(DateSerial(myYear, myMonth, myDay), "yyyy-mm-dd")   
            'prints  2009-03-03

Finalizar Sub
