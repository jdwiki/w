Você deve adicionar o Microsoft Scripting Runtime ao projeto VBA por meio do comando Ferramentas → Referências do VBE para implementar a ligação antecipada do objeto Scripting Dictionary. Esta referência de biblioteca é transportada com o projeto; ele não precisa ser referenciado novamente quando o projeto VBA é distribuído e executado em outro computador.

Propriedades e métodos #

Um objeto de dicionário de script armazena informações em pares de chave/item. As Chaves devem ser exclusivas e não uma matriz, mas os Itens associados podem ser repetidos (sua exclusividade é mantida pela Chave complementar) e podem ser de qualquer tipo de variante ou objeto.

Um dicionário pode ser pensado como um banco de dados na memória de dois campos com um índice único primário no primeiro ‘campo’ (a Chave). Esse índice exclusivo na propriedade Keys permite ‘pesquisas’ muito rápidas para recuperar o valor de Item associado a uma chave.

Propriedades

nome|leitura/gravação | tipo | descrição |————|:——————-:|:————- ——–:|—— CompareMode | ler/gravar | Constante CompareMode | A configuração do CompareMode só pode ser realizada em um dicionário vazio. Os valores aceitos são 0 (vbBinaryCompare), 1 (vbTextCompare), 2 (vbDatabaseCompare). Contagem | somente leitura | inteiro longo não assinado | Uma contagem baseada em um dos pares de chave/item no objeto de dicionário de script. Chave | ler/gravar | variante sem matriz | Cada chave única individual no dicionário. Item(Chave) | ler/gravar | qualquer variante | Propriedade padrão. Cada item individual associado a uma chave no dicionário. Observe que tentar recuperar um item com uma chave que não existe no dicionário adicionará implicitamente a chave passada.

Métodos

nome              |descrição |—————————–|——————- —————– Add(Chave,Item) | Adiciona uma nova Chave e Item ao dicionário. A nova chave não deve existir na coleção de chaves atual do dicionário, mas um item pode ser repetido entre muitas chaves exclusivas. Existe(Chave) | Teste booleano para determinar se uma chave já existe no dicionário. Chaves | Retorna a matriz ou coleção de chaves exclusivas. Itens | Retorna a matriz ou coleção de itens associados. Remover(Chave) | Remove uma chave de dicionário individual e seu item associado. Remover Tudo | Limpa todas as chaves e itens de um objeto de dicionário.

Código de amostra

'Populate, enumerate, locate and remove entries in a dictionary that was created
'with late binding
Sub iterateDictionaryLate()
    Dim k As Variant, dict As Object
    
    Set dict = CreateObject("Scripting.Dictionary")
    dict.CompareMode = vbTextCompare          'non-case sensitive compare model
    
    'populate the dictionary
    dict.Add Key:="Red", Item:="Balloon"
    dict.Add Key:="Green", Item:="Balloon"
    dict.Add Key:="Blue", Item:="Balloon"
    
    'iterate through the keys
    For Each k In dict.Keys
        Debug.Print k & " - " & dict.Item(k)
    Next k

    'locate the Item for Green
    Debug.Print dict.Item("Green")
    
    'remove key/item pairs from the dictionary
    dict.Remove "blue"      'remove individual key/item pair by key
    dict.RemoveAll          'remove all remaining key/item pairs

End Sub

'Populate, enumerate, locate and remove entries in a dictionary that was created
'with early binding (see Remarks)
Sub iterateDictionaryEarly()
    Dim d As Long, k As Variant
    Dim dict As New Scripting.Dictionary
    
    dict.CompareMode = vbTextCompare          'non-case sensitive compare model
    
    'populate the dictionary
    dict.Add Key:="Red", Item:="Balloon"
    dict.Add Key:="Green", Item:="Balloon"
    dict.Add Key:="Blue", Item:="Balloon"
    dict.Add Key:="White", Item:="Balloon"
    
    'iterate through the keys
    For Each k In dict.Keys
        Debug.Print k & " - " & dict.Item(k)
    Next k

    'iterate through the keys by the count
    For d = 0 To dict.Count - 1
        Debug.Print dict.Keys(d) & " - " & dict.Items(d)
    Next d
    
    'iterate through the keys by the boundaries of the keys collection
    For d = LBound(dict.Keys) To UBound(dict.Keys)
        Debug.Print dict.Keys(d) & " - " & dict.Items(d)
    Next d
    
    'locate the Item for Green
    Debug.Print dict.Item("Green")
    'locate the Item for the first key
    Debug.Print dict.Item(dict.Keys(0))
    'locate the Item for the last key
    Debug.Print dict.Item(dict.Keys(UBound(dict.Keys)))
    
    'remove key/item pairs from the dictionary
    dict.Remove "blue"                         'remove individual key/item pair by key
    dict.Remove dict.Keys(0)                   'remove first key/item by index position
    dict.Remove dict.Keys(UBound(dict.Keys))   'remove last key/item by index position
    dict.RemoveAll                             'remove all remaining key/item pairs

End Sub

Obtendo valores exclusivos com Scripting.Dictionary #

O Dicionário permite obter um conjunto único de valores de forma muito simples. Considere a seguinte função:

Function Unique(values As Variant) As Variant()
    'Put all the values as keys into a dictionary
    Dim dict As New Scripting.Dictionary
    Dim val As Variant
    For Each val In values
        dict(val) = 1 'The value doesn't matter here
    Next
    Unique = dict.Keys
End Function

que você poderia então chamar assim:

Dim duplicates() As Variant
duplicates = Array(1, 2, 3, 1, 2, 3)
Dim uniqueVals() As Variant
uniqueVals = Unique(duplicates)

e uniqueVals conteria apenas {1,2,3}.

Nota: Esta função pode ser usada com qualquer objeto enumerável.

Agregando dados com Scripting.Dictionary (Maximum, Count) #

Os dicionários são ótimos para gerenciar informações onde ocorrem várias entradas, mas você só se preocupa com um único valor para cada conjunto de entradas — o primeiro ou último valor, o valor mínimo ou máximo, uma média, uma soma etc.

Considere uma pasta de trabalho que contém um log da atividade do usuário, com um script que insere o nome de usuário e a data de edição sempre que alguém edita a pasta de trabalho:

** Planilha Log**

A B
bob 12/10/2016 9:00
alice 13/10/2016 13:00
bob 13/10/2016 13:30
alice 13/10/2016 14:00
alice 14/10/2016 13:00

Digamos que você queira gerar a última hora de edição para cada usuário, em uma planilha chamada Resumo.

Observações: 1. Supõe-se que os dados estejam em ActiveWorkbook. 2. Estamos usando um array para extrair os valores da planilha; isso é mais eficiente do que iterar em cada célula. 3. O Dicionário é criado usando a vinculação antecipada.

Sub LastEdit()
Dim vLog as Variant, vKey as Variant
Dim dict as New Scripting.Dictionary
Dim lastRow As Integer, lastColumn As Integer
Dim i as Long
Dim anchor As Range

With ActiveWorkbook
    With .Sheets("Log")
        'Pull entries in "log" into a variant array
        lastRow = .Range("a" & .Rows.Count).End(xlUp).Row
        vlog = .Range("a1", .Cells(lastRow, 2)).Value2

        'Loop through array
        For i = 1 to lastRow
            Dim username As String
            username = vlog(i, 1)
            Dim editDate As Date
            editDate = vlog(i, 2)

            'If the username is not yet in the dictionary:
            If Not dict.Exists(username) Then
                dict(username) = editDate
            ElseIf dict(username) < editDate Then
                dict(username) = editDate
            End If
        Next
    End With

    With .Sheets("Summary")
        'Loop through keys
        For Each vKey in dict.Keys
            'Add the key and value at the next available row
            Anchor = .Range("A" & .Rows.Count).End(xlUp).Offset(1,0)
            Anchor = vKey
            Anchor.Offset(0,1) = dict(vKey)
        Next vKey
    End With
End With
End Sub

e a saída ficará assim:

** Planilha Resumo**

A B
bob 13/10/2016 13:30
alice 14/10/2016 13:00

Se, por outro lado, você deseja exibir quantas vezes cada usuário editou a pasta de trabalho, o corpo do loop For deve ficar assim:

        'Loop through array
        For i = 1 to lastRow
            Dim username As String
            username = vlog(i, 1)

            'If the username is not yet in the dictionary:
            If Not dict.Exists(username) Then
                dict(username) = 1
            Else
                dict(username) = dict(username) + 1
            End If
        Next

e a saída ficará assim:

** Planilha Resumo**

A B
bob 2
alice 3
