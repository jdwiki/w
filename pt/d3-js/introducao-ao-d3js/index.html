Instalação #

Há várias maneiras de baixar e usar o D3.

Download direto do script #

  1. Baixe e extraia d3.zip
  2. Copie a pasta resultante para onde você manterá as dependências do seu projeto
  3. Referencie d3.js (para desenvolvimento) ou d3.min.js (para produção) em seu HTML: <script type="text/javascript" src="scripts/d3/d3.js"></script>

NPM #

  1. Inicialize o NPM em seu projeto, caso ainda não o tenha feito: npm init
  2. NPM instala D3: npm install --save d3
  3. Referencie d3.js (para desenvolvimento) ou d3.min.js (para produção) em seu HTML: <script type="text/javascript" src="node_modules/d3/build/d3.js"></script>

CDN #

  1. Faça referência a d3.js (para desenvolvimento) ou d3.min.js (para produção) em seu HTML: <script type="text/javascript" src="https://cdnjs.cloudflare.com/ajax/libs /d3/4.1.1/d3.js"></script>

GITHUB #

  1. Obtenha qualquer versão do d3.js (para desenvolvimento) ou d3.min.js (para produção) do Github : <script type="text/javascript" src="https://raw.githubusercontent.com/d3/d3/v3.5.16/d3.js"></script>

Para vincular diretamente à versão mais recente, copie este snippet:

<script src="https://d3js.org/d3.v4.min.js"></script>

O que é D3? Documentos baseados em dados. #

Estamos tão acostumados com o nome D3.js que é possível esquecer que D3 é na verdade DDD (Data-Driven Documents). E é isso que o D3 faz bem, uma abordagem orientada a dados para manipulação do DOM (Document Object Model): o D3 vincula dados a elementos DOM e manipula esses elementos com base nos dados limitados.

Vamos ver um recurso muito básico do D3 neste exemplo. Aqui, não anexaremos nenhum elemento SVG. Em vez disso, usaremos um SVG já presente na página, algo assim:

<svg width="400" height="400">
    <circle cx="50" cy="50" r="10"></circle>
    <circle cx="150" cy="50" r="10"></circle>
    <circle cx="210" cy="320" r="10"></circle>
    <circle cx="210" cy="30" r="10"></circle>
    <circle cx="180" cy="200" r="10"></circle>
</svg>

Este é um SVG bem básico, com 5 círculos. No momento, esses círculos não estão “vinculados” a nenhum dado. Vamos verificar esta última alegação:

Em nosso código, escrevemos:

var svg = d3.select("svg");
var circles = svg.selectAll("circle");
console.log(circles.nodes());

Aqui, d3.select("svg") retorna um objeto d3 contendo a tag <svg width="400" height="400"></svg> e todas as tags filhas, os <circle>s. Observe que, se houver várias tags svg na página, apenas a primeira será selecionada. Se você não quiser isso, você também pode selecionar pelo id da tag, como d3.select("#my-svg"). O objeto d3 tem propriedades e métodos embutidos que usaremos muito mais tarde.

svg.selectAll("circle") cria um objeto de todos os elementos <circle></circle> de dentro da tag <svg>. Ele pode pesquisar em várias camadas, portanto, não importa se as tags são filhas diretas.

circles.nodes() retorna as tags circle com todas as suas propriedades.

Se olharmos para o console e escolhermos o primeiro círculo, veremos algo assim:

Primeiro, temos attributes, depois childNodes, depois children, e assim por diante… mas sem dados.

Vamos vincular alguns dados

Mas, o que acontece se vincularmos dados a esses elementos DOM?

Em nosso código, existe uma função que cria um objeto com duas propriedades, x e y, com valores numéricos (este objeto está dentro de um array, verifique o violino abaixo). Se ligarmos esses dados aos círculos…

circles.data(data);

Isto é o que veremos se inspecionarmos o console:

Temos algo novo antes de attributes! Algo chamado __data__… e veja: os valores de x e y estão lá!

Podemos, por exemplo, alterar a posição dos círculos com base nesses dados. Dê uma olhada neste violino.

Isso é o que o D3 faz melhor: vincular dados a elementos DOM e manipular esses elementos DOM com base nos dados limitados.

chart.js #

// Sample dataset. In a real application, you will probably get this data from another source such as AJAX.
var dataset = [5, 10, 15, 20, 25]

// Sizing variables for our chart. These are saved as variables as they will be used in calculations.
var chartWidth = 300
var chartHeight = 100
var padding = 5

// We want our our bars to take up the full height of the chart, so, we will apply a scaling factor to the height of every bar.
var heightScalingFactor = chartHeight / getMax(dataset)

// Here we are creating the SVG that will be our chart.
var svg = d3
  .select('#my-chart')           // I'm starting off by selecting the container.
    .append('svg')               // Appending an SVG element to that container.
    .attr('width', chartWidth)   // Setting the width of the SVG.
    .attr('height', chartHeight) // And setting the height of the SVG.

// The next step is to create the rectangles that will make up the bars in our bar chart.
svg
  .selectAll('rect')                                          // I'm selecting all of the rectangles in the SVG (note that at this point, there actually aren't any, but we'll be creating them in a couple of steps).
  .data(dataset)                                              // Then I'm mapping the dataset to those rectangles.
  .enter()                                                    // This step is important in that it allows us to dynamically create the rectangle elements that we selected previously.
    .append('rect')                                           // For each element in the dataset, append a new rectangle.
      .attr('x', function (value, index) {                    // Set the X position of the rectangle by taking the index of the current item we are creating, multiplying it by the calculated width of each bar, and adding a padding value so we can see some space between bars.
          return (index * (chartWidth / dataset.length)) + padding
        })
      .attr('y', function (value, index) {                    // Set the rectangle by subtracting the scaled height from the height of the chart (this has to be done becuase SVG coordinates start with 0,0 at their top left corner).
        return chartHeight - (value * heightScalingFactor)
      })
      .attr('width', (chartWidth / dataset.length) - padding) // The width is dynamically calculated to have an even distribution of bars that take up the entire width of the chart.
      .attr('height', function (value, index) {               // The height is simply the value of the item in the dataset multiplied by the height scaling factor.
        return value * heightScalingFactor
      })
      .attr('fill', 'pink')                                   // Sets the color of the bars.

/**
 *  Gets the maximum value in a collection of numbers.
 */
function getMax(collection) {
  var max = 0

  collection.forEach(function (element) {
    max = element > max ? element : max
  })

  return max
}

Olá Mundo! #

Crie um arquivo .html contendo este snippet:

<!DOCTYPE html>
<meta charset="utf-8">
<body>
<script src="//d3js.org/d3.v4.min.js"></script>
<script>

d3.select("body").append("span")
    .text("Hello, world!");

</script>
