Instalação ou Configuração (Mac OS X) #

VISÃO GLOBAL #

cocos2d-x é um mecanismo de jogo multiplataforma de código aberto. Ele permite que os desenvolvedores codifiquem em C++, Lua e implantação de Javascript em iOS, Android, Windows Phone, Mac OS X, Windows Desktop e Linux.

PRÉ-REQUISITOS #

Requisitos de construção

  • Mac OS X 10.7+, Xcode 4.6+
  • Windows 7+, VS 2012+
  • Ubuntu 12.10+, CMake 2.6+
  • Python 2.7.5
  • NDK, JDK, ANT (desenvolvimento Android)

Requisitos de tempo de execução

  • iOS 5.0+ para jogos para iPhone/iPad
  • Android 2.3+ para jogos Android
  • Windows Phone 8+ para jogos do Windows Phone
  • OS X v10.6+ para jogos Mac
  • Windows 7+ para jogos do Windows

Instalação #

Baixe tudo abaixo:

  • Cocos2d-x v3.x
  • JDK/SDK 1.6+
  • Ferramentas de linha de comando do Android
  • RDA
  • Apache Ant
  • Python 2.7.5

Depois de baixar tudo acima,

  • descompacte Cocos2d-x
  • instale o JDK.
  • descompacte o Android SDK e o NDK no mesmo local raiz. (pode ser: ~/AndroidDev)
  • verifique se o Python 2.7 está instalado e acessível.
  • instale o Apache Ant e verifique se funciona.

Dentro do seu diretório Cocos2d-x execute:

python setup.py

cuidado: Você não deve usar o sinal ~. Use o caminho completo para seu diretório home. Caso contrário, os scripts falharão devido ao valor do caminho de erro.

  • Agora ele pedirá o caminho NDK_ROOT. Aqui, insira o diretório NDK extraído caminho:

~/android-ndk-r10e

  • SDK_ROOT. Aqui, insira o caminho do Android SDK:

~/android-sdk-macosx

  • ANT_ROOT. Aqui, insira o caminho do bin do Apache ANT:

~/apache-ant-1.9.6/bin

Depois de definir as variáveis ​​de ambiente acima, execute:

source ~/.bash_profile

Vola, o cocos2d-x foi configurado com sucesso em seu sistema.

Instalação ou Configuração (Windows) #

VISÃO GLOBAL #

cocos2d-x é um mecanismo de jogo multiplataforma de código aberto. Ele permite que os desenvolvedores codifiquem em C++, Lua e implantação de Javascript em iOS, Android, Windows Phone, Mac OS X, Windows Desktop e Linux.

PRÉ-REQUISITOS #

Requisitos de construção

  • Mac OS X 10.7+, Xcode 4.6+
  • Windows 7+, VS 2012+
  • Ubuntu 12.10+, CMake 2.6+
  • Python 2.7.5
  • NDK, JDK, ANT (desenvolvimento Android)

Requisitos de tempo de execução

  • iOS 5.0+ para jogos para iPhone/iPad
  • Android 2.3+ para jogos Android
  • Windows Phone 8+ para jogos do Windows Phone
  • OS X v10.6+ para jogos Mac
  • Windows 7+ para jogos do Windows

Instalação #

Baixe tudo abaixo:

  • Cocos2d-x v3.x
  • JDK/SDK 1.6+
  • Ferramentas de linha de comando do Android
  • RDA
  • Apache Ant
  • Python 2.7.5

Depois de baixar tudo acima,

  • descompacte Cocos2d-x
  • instale o JDK e ao mesmo tempo crie uma nova variável chamada: JAVA_HOME e dê o caminho para onde você instalou o JDK acima.
  • descompacte o Android SDK e o NDK no mesmo local raiz. (pode ser: ~/AndroidDev)
  • instale o Python e anote onde você o colocou.
  • extraia o Apache Ant e coloque a pasta inteira onde deseja mantê-la. Anote onde você colocou.
  • adicione os caminhos para Apache Ant e Python à sua variável PATH.
  • reinício
  • agora, teste seu ambiente antes de continuar. Inicie um prompt de comando e execute:

formiga

`python`

Se tudo funcionar, é hora de executar setup.py para configurar seu ambiente de desenvolvimento Android. Isso definirá as variáveis ​​de ambiente necessárias. Se você não configurou esse ambiente antes, será solicitado a inserir caminhos para variáveis ​​que não foram encontradas. Você executa o setup.py no diretório em que o Cocos2d-x está.

Dentro do seu diretório Cocos2d-x, execute:

python setup.py

cuidado: Você não deve usar o sinal ~. Use o caminho completo para seu diretório home. Caso contrário, os scripts falharão devido ao valor do caminho de erro.

  • Agora ele pedirá o caminho NDK_ROOT. Aqui, insira o diretório NDK extraído caminho:

~/android-ndk-r10e

  • SDK_ROOT. Aqui, insira o caminho do Android SDK:

~/android-sdk-macosx

  • ANT_ROOT. Aqui, insira o caminho do bin do Apache ANT:

~/apache-ant-1.9.6/bin

  • Reinício

Vola, o cocos2d-x foi configurado com sucesso em seu sistema.
