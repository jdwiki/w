Convertendo string para maiúscula #

A função toupper converterá uma string para maiúscula (letras maiúsculas). Por exemplo:

BEGIN {
    greeting = "hello"
    loud_greeting = toupper(greeting)
    print loud_greeting
}

Este código produzirá “HELLO” quando executado.

Calculando um hash de uma string #

Embora implementar um dos algoritmos de hash padrão no awk seja provavelmente uma tarefa tediosa, definir uma função hash que pode ser usada como um identificador para documentos de texto é muito mais tratável. Uma situação prática em que tal função é útil é atribuir ids curtos a itens de acordo com sua descrição, por exemplo, casos de teste, para que o id curto possa ser fornecido como referência ao item pelo usuário em vez de fornecer sua descrição longa.

A função hash precisa converter caracteres em códigos numéricos, o que é feito usando uma tabela de pesquisa inicializada no início do script. A função hash é então calculada usando transformações aritméticas modulares, uma abordagem muito clássica para o cálculo de hashes.

Para fins de demonstração, adicionamos uma regra para decorar as linhas de entrada com seu hash, mas essa regra não é necessária para usar a função:

BEGIN{
  for(n=0;n<256;n++) {
    ord[sprintf("%c",n)] = n
  }
}

function hash(text, _prime, _modulo, _ax, _chars, _i)
{
  _prime = 104729;
  _modulo = 1048576;
  _ax = 0;
  split(text, _chars, "");
  for (_i=1; _i <= length(text); _i++) {
    _ax = (_ax * _prime + ord[_chars[_i]]) % _modulo;
  };
  return sprintf("%05x", _ax)
}

# Rule to demonstrate the function
#  These comments and the following line are not relevant
#  to the definition of the hash function but illustrate
#  its use.

{ printf("%s|%s\n", hash($0), $0) }

Salvamos o programa acima no arquivo hash.awk e o demonstramos em uma pequena lista de títulos de livros clássicos em inglês:

Alice's Adventures in Wonderland
Animal Farm
EOF

Concatenação de Strings #

A concatenação de strings é feita simplesmente escrevendo expressões próximas umas das outras sem nenhum operador. Por exemplo:

BEGIN {
   user = "root"
   print "Hello "user "!"
}

imprimirá: Olá root!

Observe que as expressões não precisam ser separadas por espaços em branco.

Substituição de texto de string #

A função SUB permite substituir o texto dentro do awk

sub(regexp, substituição, destino)

onde regexp pode ser uma expressão regular completa

$ cat file
AAAAA
BBBB
CCCC
DDDD
EEEE
FFFF
GGGG
$ awk '{sub("AAA","XXX", $0); print}' file
XXXAA
BBBB
CCCC
DDDD
EEEE
FFFF
GGGG

Converte string para minúscula #

AWK frequentemente usado para manipular arquivos inteiros contendo uma lista de strings. Digamos que o arquivo awk_test_file.txt contenha:

First String
Second String
Third String

Para converter todas as strings para letras minúsculas, execute:

awk '{ print tolower($0) }' awk_test_file.txt

Isso resultará:

first string
second string
third string

Extração de substring #

GNU awk suporta uma função de extração de substring para retornar uma seqüência de caracteres de comprimento fixo de uma string principal. A sintaxe é

*substr(string, start [, length ])*

onde, string é a string de origem e start marca o início da posição da substring que você deseja que a extração seja feita para um comprimento opcional de caracteres length. Se o comprimento não for especificado, a extração é feita até o final da string.

O primeiro caractere da string é tratado como o caractere número um.

awk '
BEGIN {
    testString = "MyTESTstring"
    substring  =  substr(testString, 3, 4)    # Start at character 3 for a length of 4 characters
    print substring
}'

produzirá a substring TEST.

awk '
BEGIN {
    testString = "MyTESTstring"
    substring  =  substr(testString, 3)    # Start at character 3 till end of the string
    print substring
}'

isso extrai a substring da posição do caractere 3 até o final da string inteira, retornando TESTstring

Observação:-

  • Se start recebe um valor negativo, GNU awk imprime a string inteira e se length recebe um valor diferente de zero, GNU awk behavior retorna uma string null e o comportamento varia entre diferentes implementações de awk.
