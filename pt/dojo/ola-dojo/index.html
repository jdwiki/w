Um simples Hello Dojo como o Hello World em outras linguagens de programação. O Dojo é simples de instalar/configurar e usar. Você pode baixar a versão mais recente do Dojo em http://dojotoolkit.org/download/ ou pode usar o que é chamado de CDN em seu projeto. Eu prefiro baixar e usar. A versão mais recente no momento deste tópico é 1.12.1 e o trabalho no Dojo 2.0 está em andamento.

Olá Mundo #

Crie um arquivo HTML como abaixo. Na tag de script, você pode usar o link CDN ou do seu local como

<script type="text/javascript" src="dojo/dojo/dojo.js"
        data-dojo-config="async:true">

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>Tutorial: Hello Dojo!</title>
</head>
<body>
    <h1 id="greeting">Hello</h1>
    <!-- load Dojo -->
    <script src="//ajax.googleapis.com/ajax/libs/dojo/1.10.4/dojo/dojo.js"
            data-dojo-config="async: true"></script>
</body>
</html>

É isso. Ao executar este HTML, você deverá ver Hello. Sim, ainda não usamos nenhum código específico do Dojo neste exemplo. Mas isso mostra como você pode criar uma página simples usando o Dojo. A seguir, veremos como podemos aplicar o Dojo a esta página simples.

Dojo AMD #

No próximo exemplo, vamos usar os recursos do Dojo e entender o que significa AMD (Asynchronous Module Definition).

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>Tutorial: Hello Dojo!</title>
</head>
<body>
    <h1 id="greeting">Hello</h1>
    <!-- load Dojo -->
    <script src="//ajax.googleapis.com/ajax/libs/dojo/1.10.4/dojo/dojo.js"
            data-dojo-config="async: true"></script>

    <script>
        require([
            'dojo/dom',
            'dojo/dom-construct'
        ], function (dom, domConstruct) {
            var greetingNode = dom.byId('greeting');
            domConstruct.place('<em> Dojo!</em>', greetingNode);
        });
    </script>
</body>
</html>

A diferença do exemplo anterior é que há uma tag de script extra e, dentro dela, há poucos recursos do dojo sendo usados. Vamos ver cada um deles

requer Em termos leigos, require é semelhante a importar ou usar instruções em outras linguagens nas quais você importa algumas bibliotecas OOTB (no dojo você as chama de módulos). Os módulos existentes são carregados usando a palavra-chave ‘require’ e os novos módulos são criados usando a palavra-chave ‘define’. Aprenderemos mais sobre os módulos na seção posterior. Para este exemplo, usamos dois módulos OOTB ‘dojo/dom’ e ‘dojo/dom-construct’. dojo/dom (dom) é o DOM principal (modelo de objeto de documento) que pode ser usado para obter um nó de html. Para desenvolvedores de javascript, é semelhante a document.getElementById(’’) de fato, internamente o dojo usa o mesmo método. dojo/dom-construct(domConstruct) é usado para criar nós como div, li, ul etc. É uma API de construção DOM e também pode ser usada para inserir um nó no DOM em qualquer posição. Digamos que você tenha uma div ‘abc’ e queira criar outra div ‘xyz’ e colocá-la depois de ‘abc’. Você pode fazer isso como

domConstruct.create("div", { id:"xyz",innerHTML: "<p>new DIV</p>" });
domConstruct.place(dojo.byId("xyz"), dojo.byId("abc"), "after");

Voltando ao nosso exemplo, temos

require([
            'dojo/dom',
            'dojo/dom-construct'
        ], function (dom, domConstruct) {
            var greetingNode = dom.byId('greeting');
        domConstruct.place('<em> Dojo!</em>', greetingNode);
        });

dentro da função, você vê dom e domConstruct. É assim que nos referimos a dojo/dom e dojo/dom-construct. Você pode usar qualquer convenção de nomenclatura que desejar

require([
                'dojo/dom',
                'dojo/dom-construct'
            ], function (hi, bye) {
                var greetingNode = hi.byId('greeting');
                bye.place('<em> Dojo!</em>', greetingNode);
            });

Mas é uma boa prática ter nomes significativos como para dojo/dom use dom e para dojo/dom-construct, user domConstruct.

Agora dentro da função, temos

var greetingNode = hi.byId('greeting');

O que isso faz é pesquisar o a dom (div neste caso) com id=‘greeting’. variável saudaçãoNode, terá o nó dom real. Então nós temos,

domConstruct.place('<em> Dojo!</em>', greetingNode);

Então, aqui estamos anexando o Dojo! para o nó saudaçãoNode. Isto é como Olá+Dojo! e a saída será Hello Dojo!
