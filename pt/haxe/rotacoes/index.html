Sintaxe #

  • for (identificador de variável em coleção iterativa) { expressão }
  • while (condição) { expressão }
  • do { expressão } while (condição);
  • parar;
  • Prosseguir;

Por #

Loops for iterar em uma coleção iterativa. Uma coleção iterativa é qualquer classe que unifica estruturalmente com os tipos Iterator<T> ou Iterable<T> da biblioteca padrão Haxe.

Um loop for que registra números no intervalo de 0 a 10 (exclusivo) pode ser escrito da seguinte forma:

for (i in 0...10) {
    trace(i);
}

O identificador de variável i contém o valor individual dos elementos na coleção iterativa. Esse comportamento é semelhante ao for-each em outros idiomas.

Um loop for que registra elementos em uma matriz pode, portanto, ser escrito da seguinte maneira:

for (char in ['a', 'b', 'c', 'd']) {
    trace(char);
}

Experimente o exemplo em try.haxe.org.

Referências #

  • “Para”, Haxe manual
  • “Iteradores”, manual Haxe

Enquanto #

While-loops executa uma expressão de corpo desde que a condição do loop seja avaliada como true.

Um loop while que registra números no intervalo de 9 a 0 (inclusive) pode ser escrito da seguinte forma:

var i = 10;
while (i-- > 0) {
    trace(i);
}

Experimente o exemplo em try.haxe.org.

Referências #

  • “Enquanto”, Haxe manual

Fazer enquanto #

Do-while-loops executa uma expressão de corpo pelo menos uma vez e continua a executá-la enquanto a condição do loop for avaliada como true.

Um loop do-while que registra números no intervalo de 10 a 0 (inclusive) pode ser escrito da seguinte maneira:

var i = 10;
do {
    trace(i);
} while (i-- > 0);

Experimente o exemplo em try.haxe.org.

Referências #

  • “Faça enquanto”, Haxe manual

Controle de fluxo #

O fluxo ou execução de um loop pode ser controlado pelo uso das expressões break e continue.

Parar #

break sai do loop atual. Caso o loop esteja aninhado dentro de outro loop, o loop pai não é afetado.

for (i in 0...10) {
    for (j in 0...10) {
        if (j == 5) break;
        trace(i, j);
    }
}

Experimente o exemplo em try.haxe.org.

Continuar #

continue pula a iteração atual do loop no ponto da expressão. Caso o loop esteja aninhado dentro de outro loop, o loop pai não é afetado.

for (i in 0...10) {
    for (j in 0...10) {
        if (j == 5) continue;
        trace(i, j);
    }
}

Experimente o exemplo em try.haxe.org.

Referências #

  • “Break”, Haxe manual
  • “Continue”, Haxe manual
