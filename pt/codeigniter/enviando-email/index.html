No CodeIgniter 3 você deve incluir o parâmetro:

$config['newline'] = "\r\n";

Simplesmente não funciona sem ele.

Se você não se importa com novas linhas e está usando o CodeIgniter 2, este parâmetro de configuração é opcional.

Carregar a biblioteca de e-mail #

Primeiro você precisa carregar a biblioteca de e-mail.

Faça isso no arquivo do controlador que enviará o e-mail:

$this->load->library('email');

Ou carregue-o globalmente no arquivo autoload.php na pasta config:

$autoload['libraries'] = array('email');

Enquanto estiver lá, você pode carregar o auxiliar de e-mail se quiser usar alguns dos atalhos internos do CodeIgniter:

$autoload['helper'] = array('email');

O auxiliar de email pode ser carregado no arquivo Controller de maneira semelhante à biblioteca de email:

$this->load->helper('email');

Defina seus parâmetros de configuração de e-mail #

Crie um novo arquivo na pasta application/config chamado email.php

Defina os parâmetros para enviar e-mail. Eles serão carregados quando você enviar seu e-mail.

$config['newline'] = "\r\n"; //You must use double quotes on this one
$config['protocol'] = 'smtp';
$config['smtp_host'] = 'ssl://smtp.gmail.com'; //Change for your specific needs
$config['smtp_port'] = 465; //Change for your specific needs
$config['smtp_user'] = 'test@test.com'; //Change for your specific needs
$config['smtp_pass'] = 'yourpassword'; //Change for your specific needs
$config['charset'] = 'iso-8859-1';
$config['mailtype'] = 'text'; //This can be set as 'html' too

Crie seu e-mail #

$this->email->from('accounts@yourwebsite.com', 'Tom Webmaster');
$this->email->to('fred@fake.com', 'Freddie Fakeperson');
$this->email->subject('Your Account Is Active');
$this->email->message('Welcome to our new site!');

No método ‘de’, o primeiro parâmetro é o endereço de e-mail do qual você está enviando, o segundo parâmetro é o nome que você deseja que o destinatário veja.

No método ‘para’, você define para quem o email está sendo enviado.

O método ‘assunto’ define o assunto do e-mail.

O método ‘message’ define o que estará no corpo do seu email.

Qualquer um deles pode ser um dado enviado ao seu site por um usuário. Então você pode ter uma variável aqui que contém dados postados. Então, eles podem se parecer mais com isso:

$this->email->to($email, $username);

Envie seu e-mail #

$sent = $this->email->send();


//This is optional - but good when you're in a testing environment.
if(isset($sent)){
    echo "It sent!";
}else{
    echo "It did not send.";
}

Envie um e-mail HTML #

Mas você não quer apenas um e-mail de texto simples. Você quer um e-mail html bonito.

Defina seu arquivo de configuração como html:

$config['mailtype'] = 'html';

Se você quiser passar dados (como um nome de usuário, por exemplo) para o email html, coloque-os em um array:

$data = array('name' => $name, 
              'email' => $email,
              'phone' => $phone,
              'date' => $date);

Então, ao enviar, aponte sua ‘mensagem’ para uma vista. Em seguida, passe seu array de dados para ele:

$this->email->message($this->load->view('new_user',$data, true));

Na sua pasta application/view, crie sua view.

Neste caso é chamado ’new_user.php’.

Você pode estilizar isso da maneira que quiser. Aqui está um exemplo rápido:

<html>
<head>
    <style type='text/css'>
        body {background-color: #CCD9F9;
             font-family: Verdana, Geneva, sans-serif}

        h3 {color:#4C628D}

        p {font-weight:bold}
    </style>
</head>
<body>

    <h3>Hi <?php echo $name;?>,</h3>
    <h3>Thanks for contacting us.</h3> 

    <p>You've taken your first step into a larger world.</p>   
    <p>We really appreciate your interest.</p>

</body>
</html>

Formulário de Contato #

Controlador (Pages.php)

public function contact()
{
    
    $this->load->library('email');
    $this->load->library('form_validation');

    //Set form validation
    $this->form_validation->set_rules('name', 'Name', 'trim|required|min_length[4]|max_length[16]');
    $this->form_validation->set_rules('email', 'Email', 'trim|required|valid_email|min_length[6]|max_length[60]');
    $this->form_validation->set_rules('message', 'Message', 'trim|required|min_length[12]|max_length[200]');

    //Run form validation
    if ($this->form_validation->run() === FALSE)
    {
        $this->load->view('contact');
    } else {

        //Get the form data
        $name = $this->input->post('name');
        $from_email = $this->input->post('email');
        $subject = $this->input->post('subject');
        $message = $this->input->post('message');

        //Web master email
        $to_email = 'admin@domain.com'; //Webmaster email, who receive mails

        //Mail settings
        $config['protocol'] = 'smtp';
        $config['smtp_host'] = 'ssl://smtp.gmail.com';
        $config['smtp_port'] = '465';
        $config['smtp_user'] = 'mail@domain.com'; // Your email address
        $config['smtp_pass'] = 'mailpassword'; // Your email account password
        $config['mailtype'] = 'html'; // or 'text'
        $config['charset'] = 'iso-8859-1';
        $config['wordwrap'] = TRUE; //No quotes required
        $config['newline'] = "\r\n"; //Double quotes required

        $this->email->initialize($config);                        

        //Send mail with data
        $this->email->from($from_email, $name);
        $this->email->to($to_email);
        $this->email->subject($subject);
        $this->email->message($message);
        
        if ($this->email->send())
        {
            $this->session->set_flashdata('msg','<div class="alert alert-success">Mail sent!</div>');

            redirect('contact');
        } else {
            $this->session->set_flashdata('msg','<div class="alert alert-danger">Problem in sending</div>');
            $this->load->view('contact');
        }

    }
