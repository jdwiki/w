CodeIgniter - Internacionalização

  • Exemplo

Nesta página

  • Exemplo

A classe de idioma no CodeIgniter fornece uma maneira fácil de oferecer suporte a vários idiomas para internacionalização. Até certo ponto, podemos usar arquivos de idiomas diferentes para exibir texto em vários idiomas diferentes.

Exemplo #

Criação de arquivos Idioma

Para criar um arquivo de idioma, você deve terminá-lo com _lang.php. Por exemplo, você deseja criar um arquivo de idioma para o idioma francês, então você deve salvá-lo com french_lang.php. Dentro deste arquivo você pode armazenar todos os seus textos de idioma na combinação de chave e valor no array $lang como mostrado abaixo.

$lang['key'] = 'val';

Carregando arquivo de idioma

Para usar qualquer idioma em seu aplicativo, você deve primeiro carregar o arquivo desse idioma específico para recuperar vários textos armazenados nesse arquivo. Você pode usar o código a seguir para carregar o arquivo de idioma.

$this->lang->load('filename', 'language');

filename : É o nome do arquivo que você deseja carregar. Não use a extensão do arquivo aqui, mas apenas o nome do arquivo. Language : É o conjunto de idiomas que o contém.

Buscando o texto do idioma

$this->lang->line('language_key');

Para buscar uma linha do arquivo de linguagem simplesmente execute o seguinte código. Em que language_key é o parâmetro de chave usado para buscar o valor da chave no arquivo de idioma carregado.

Idiomas de carregamento automático

Se você precisar de algum idioma globalmente, então você pode carregá-lo automaticamente no arquivo application/config/autoload.php como mostrado abaixo.

| -----------------------------------------------------------------------
|  Auto-load Language files
| -----------------------------------------------------------------------
| Prototype:
|   $autoload['language'] = array('lang1', 'lang2');
|
| NOTE: Do not include the "_lang" part of your file. For example
| "codeigniter_lang.php" would be referenced as array('codeigniter');
|
*/
$autoload['language'] = array();

Simplesmente, passe os diferentes idiomas para serem carregados automaticamente pelo CodeIgniter.

Crie um controlador chamado Lang_controller.php e salve-o em application/controller/Lang_controller.php

<?php

class Lang_controller estende CI_Controller {

public function index(){
     //Load form helper
     $this->load->helper('form');

     //Get the selected language
     $language = $this->input->post('language');
    
     //Choose language file according to selected lanaguage
     if($language == "french")
        $this->lang->load('french_lang','french');
     else if($language == "german")
        $this->lang->load('german_lang','german');
     else
     $this->lang->load('english_lang','english');
    
     //Fetch the message from language file.
     $data['msg'] = $this->lang->line('msg');
    
     $data['language'] = $language;
     //Load the view file
     $this->load->view('lang_view',$data);
     }
  }
?>

Crie um arquivo de visualização chamado lang_view.php e salve-o em application/views/lang_view.php

<!DOCTYPE html>
<html lang = "en"> 
  <head>
     <meta charset = "utf-8">
     <title>CodeIgniter Internationalization Example</title>
  </head>  
  <body>
     <?php
        echo form_open('/lang');
     ?>
     <select name = "language" onchange = "javascript:this.form.submit();">
        <?php
           $lang = array('english'=>"English",'french'=>"French",'german'=>"German");
            
           foreach($lang as $key=>$val) {
              if($key == $language)
                 echo "<option value = '".$key."' selected>".$val."</option>";
              else
                 echo "<option value = '".$key."'>".$val."</option>";
           }           
        ?>          
     </select>
     <?php
        form_close();
        echo $msg;
     ?>     
  </body>
</html>

Crie três pastas chamadas inglês, francês e alemão em aplicativo/idioma conforme mostrado na figura abaixo.

Copie o código abaixo e salve-o no arquivo english_lang.php na pasta application/language/english.

<?php
   $lang['msg'] = "CodeIgniter Internationalization example.";
?>

Copie o código abaixo e salve-o no arquivo french_lang.php na pasta application/language/French.

<?php
   $lang['msg'] = "Exemple CodeIgniter internationalisation.";
?>

Copie o código abaixo e salve-o no arquivo german_lang.php na pasta application/language/german.

<?php
    $lang['msg'] = "CodeIgniter Internationalisierung Beispiel.";
?>

Altere o arquivo routes.php em application/config/routes.php para adicionar a rota para o controlador acima e adicione a seguinte linha no final do arquivo.

$route['lang'] = "Lang_controller";

Execute a seguinte URL no navegador para executar o exemplo acima.

http://yoursite.com/index.php/lang

Em seguida, verifique em seu navegador. Obrigada.
