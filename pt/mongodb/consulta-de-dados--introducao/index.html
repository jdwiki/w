Exemplos básicos de consulta

Achar() #

recupere todos os documentos em uma coleção

db.collection.find({});

recupere documentos em uma coleção usando uma condição (semelhante a WHERE no MYSQL)

db.collection.find({key: value}); 
example
  db.users.find({email:"sample@email.com"});

recuperar documentos em uma coleção usando condições booleanas (operadores de consulta)

//AND
db.collection.find( {
    $and: [
     { key: value }, { key: value } 
    ] 
})
//OR
db.collection.find( {
    $or: [
     { key: value }, { key: value } 
    ] 
})
//NOT
db.inventory.find( { key: { $not: value } } )

mais operações booleanas e exemplos podem ser encontrados aqui

OBSERVAÇÃO: find() continuará pesquisando a coleção mesmo se uma correspondência de documento for encontrada , portanto, é ineficiente quando usado em uma coleção grande . No entanto, modelando cuidadosamente seus dados e/ou usando índices, você pode aumentar a eficiência de find()

EncontrarUm() #

db.collection.findOne({});

a funcionalidade de consulta é semelhante a find() mas isso terminará a execução no momento em que encontrar um documento que corresponda à sua condição , se usado com um objeto vazio , ele buscará o primeiro documento e o retornará . findOne() documentação da API do mongodb

limite, pule, classifique e conte os resultados do método find() #

Semelhante aos métodos de agregação também pelo método find() você tem a possibilidade de limitar, pular, classificar e contar os resultados. Digamos que temos a seguinte coleção:

db.test.insertMany([
    {name:"Any", age:"21", status:"busy"}, 
    {name:"Tony", age:"25", status:"busy"}, 
    {name:"Bobby", age:"28", status:"online"}, 
    {name:"Sonny", age:"28", status:"away"}, 
    {name:"Cher", age:"20", status:"online"}
])

Para listar a coleção:

db.test.find({})

Retornará:

{ "_id" : ObjectId("592516d7fbd5b591f53237b0"), "name" : "Any", "age" : "21", "status" : "busy" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b1"), "name" : "Tony", "age" : "25", "status" : "busy" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b2"), "name" : "Bobby", "age" : "28", "status" : "online" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b3"), "name" : "Sonny", "age" : "28", "status" : "away" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b4"), "name" : "Cher", "age" : "20", "status" : "online" }

Para pular os 3 primeiros documentos:

db.test.find({}).skip(3)

Retornará:

{ "_id" : ObjectId("592516d7fbd5b591f53237b3"), "name" : "Sonny", "age" : "28", "status" : "away" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b4"), "name" : "Cher", "age" : "20", "status" : "online" }

Para classificar decrescente pelo nome do campo:

db.test.find({}).sort({ "name" : -1})

Retornará:

{ "_id" : ObjectId("592516d7fbd5b591f53237b1"), "name" : "Tony", "age" : "25", "status" : "busy" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b3"), "name" : "Sonny", "age" : "28", "status" : "away" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b4"), "name" : "Cher", "age" : "20", "status" : "online" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b2"), "name" : "Bobby", "age" : "28", "status" : "online" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b0"), "name" : "Any", "age" : "21", "status" : "busy" }

Se você quiser classificar em ordem crescente, basta substituir -1 por 1

Para contar os resultados:

db.test.find({}).count()

Retornará:

5

Também são permitidas combinações destes métodos. Por exemplo, obtenha 2 documentos de uma coleção ordenada decrescente pulando o primeiro 1:

db.test.find({}).sort({ "name" : -1}).skip(1).limit(2)

Retornará:

{ "_id" : ObjectId("592516d7fbd5b591f53237b3"), "name" : "Sonny", "age" : "28", "status" : "away" }
{ "_id" : ObjectId("592516d7fbd5b591f53237b4"), "name" : "Cher", "age" : "20", "status" : "online" }

Método Find() com projeção #

No MongoDB, projeção significa selecionar apenas os dados necessários em vez de selecionar todos os dados de um documento.

A sintaxe básica do método find() com projeção é a seguinte

> db.COLLECTION_NAME.find({},{KEY:1});

Se você deseja mostrar todos os documentos sem o campo de idade, o comando é o seguinte

> db.people.find({},{age:0});

Se você deseja mostrar apenas o campo de idade, o comando é o seguinte

> db.people.find({},{age:1});

Nota: campo _id é sempre exibido durante a execução do método find(), se você não quiser este campo, então você precisa defini-lo como 0.

> db.people.find({},{name:1,_id:0});

Observação: 1 é usado para mostrar o campo enquanto 0 é usado para ocultar os campos.
