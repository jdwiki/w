Criando e inicializando arrays simples #

Introdução #

Um array é um objeto container que contém vários valores. Na imagem a seguir você pode ver um array com tamanho 10, o primeiro elemento indexado 1 e o último elemento 10.

Autohotkey oferece algumas maneiras de definir e criar arrays.

  • Matriz := []
  • Matriz := Matriz()

Criando e inicializando arrays com N número de itens #

Array := [Item1, Item2, ..., ItemN]
Array := Array(Item1, Item2, ..., ItemN)

No Autohotkey, é possível ter arrays sem itens:

Array := [] ; works fine.

E os elementos podem ser posteriormente atribuídos a ele:

Array[0] := 1

O tamanho do array pode ser determinado usando um método chamado length:

msgbox % array.length()  ; shows 1 in this case.

Se a matriz não estiver vazia, MinIndex e MaxIndex/Length retornarão o índice mais baixo e mais alto atualmente em uso na matriz. Como o índice mais baixo é quase sempre 1, MaxIndex geralmente retorna o número de itens. No entanto, se não houver chaves inteiras, MaxIndex retornará uma string vazia, enquanto Length retornará 0.

Criando e inicializando arrays multidimensionais #

Você pode criar uma matriz multidimensional da seguinte maneira:

Array[1, 2] := 3

Você pode criar e inicializar ao mesmo tempo, e os arrays internos não precisam ter o mesmo tamanho.

Array := [[4,5,6],7,8]

Arrays como este também são chamados de arrays de arrays.

Preenchendo um array #

; Assign an item:
Array[Index] := Value

; Insert one or more items at a given index:
Array.InsertAt(Index, Value, Value2, ...)

; Append one or more items:
Array.Push(Value, Value2, ...)

O valor de um índice para um elemento de matriz também pode ser um número inteiro negativo (-1, 0, 1, 2, 3, 4, …)

Removendo elementos de um array #

; Remove an item:
RemovedValue := Array.RemoveAt(Index)

; Remove the last item:
RemovedValue := Array.Pop()

Adicionando métodos personalizados substituindo a função Array() #

AutoHotkey é uma linguagem de programação baseada em protótipos, o que significa que você pode substituir qualquer função/objeto embutido a qualquer momento. Este exemplo demonstra a substituição da função Array() para adicionar métodos em um objeto de classe personalizado.

; Overrides Array()
Array(Args*) {
    Args.Base := _Array
    Return Args
}

; Custom Class Object with Methods
Class _Array {
    
    ; Reverses the order of the array.
    Reverse() {
        Reversed := []
        Loop % This.MaxIndex()
            Reversed.Push(This.Pop())
        Return Reversed
    }    
    
    ; Sums all Integers in Array
    Sum(Sum=0) {
        For Each, Value In This
            Sum += Value Is Integer ? Value : 0
        Return Sum
    }
}

; Arr == ["Hello, World!", 4, 3, 2, 1]
Arr := [1, 2, 3, 4, "Hello, World!"].Reverse() 
                    
; SumOfArray == 10
SumOfArray := Arr.Sum()
