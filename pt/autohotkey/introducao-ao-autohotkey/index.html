Olá Mundo #

Mostre um “Olá, mundo!” na caixa de mensagem.

MsgBox, Hello World!

Mostre um “Olá, mundo!” na dica de ferramenta.

#Persistent
Tooltip, Hello World!

Mostre um “Olá, mundo!” mensagem na edição da barra de bandeja.

#Persistent
TrayTip,, Hello World!

Imprime “Hello, World” na saída padrão (stdout).

FileAppend, % "Hello, World", *

Instalação ou Configuração #

De Documentação do site Autohotkey

  1. Vá para a Página inicial do AutoHotkey.
  2. Clique em Download, uma vez baixado execute o executável.
  3. Durante a instalação do AutoHotkey, você será solicitado a escolher entre UNICODE ou ANSI. Em suma, você provavelmente gostaria de escolher UNICODE. Tem suporte para letras e números não ingleses (caracteres).
  4. Continue até ver um botão Instalar.
  5. Uma vez feito, ótimo!

Use como software portátil

  1. Vá para a página de download do AutoHotkey.
  2. Encontre a seção Portátil, escolha entre UNICODE 32, 64 ou ANSI e baixe.
  3. Ao escolher a pasta de destino, escolha qualquer dispositivo armazenado externo correto ou não.
  4. Agora você pode optar por associar arquivos .ahk com Autohotkey.exe
  5. Crie um arquivo de texto simples e dê a extensão .ahk
  6. Em seguida, clique com o botão direito do mouse no arquivo .ahk no explorer e clique em Propriedades.
  7. Nas Propriedades do arquivo, clique no botão Alterar ao lado da opção “Abre com”.
    • After clicking Change, you’ll be given a list of programs to open the file, select the program you want to use and then click OK or Apply.
    • If the program you want to select is not listed, click the Browse button and find the Autohotkey executable (.exe) file and click OK to select that program.
  8. Agora os arquivos .ahk serão executados como se o autohotkey estivesse instalado, ótimo!

Se você tiver o chocolatey instalado, execute o seguinte comando como usuário administrador

choco instalar autohotkey

Alternativamente, ele pode ser construído a partir do código-fonte. Veja aqui para detalhes: https://github.com/Lexikos/AutoHotkey_L/

Mostrar “Hello World” em uma GUI #

Gui, Add, Text,, Hello World!
Gui, Show, w200 h200
return

GuiClose:
ExitApp

Obtenha um efeito semelhante ao SplashTextOn #

Gui, +AlwaysOnTop +Disabled -SysMenu +Owner  ; +Owner avoids a taskbar button.
Gui, Add, Text,, Some text to display.
Gui, Show, NoActivate, Title of Window  ; NoActivate avoids deactivating the currently active window.

Como criar um script #

Depois de instalar o AutoHotkey, você provavelmente desejará que ele faça coisas. O AutoHotkey não é mágico, todos desejamos que fosse, mas não é. Então, precisaremos dizer a ele o que fazer. Esse processo é chamado de “Script”.

  1. Clique com o botão direito do mouse na área de trabalho.
  2. Encontre “Novo” no menu.
  3. Clique em “Script AutoHotkey” dentro do menu “Novo”.
  4. Dê um novo nome ao script. Nota: Deve terminar com uma extensão .ahk. Ex. MeuScript.ahk
  5. Encontre o arquivo recém-criado em sua área de trabalho e clique com o botão direito nele.
  6. Clique em “Editar script”.
  7. Uma janela deve ter aparecido, provavelmente o Bloco de Notas. Se sim, SUCESSO!

Agora que você criou um script, precisamos adicionar coisas ao arquivo. Para obter uma lista de todos os comandos, funções e variáveis ​​integrados, consulte a seção 5. Aqui está um script muito básico contendo uma tecla de atalho que digita texto usando o comando Enviar quando a tecla de atalho é pressionada.

^j::
   Send, My First Script
Return

Vamos nos aprofundar mais adiante. Até então, aqui está uma explicação do código acima.

  • A primeira linha. ^j:: é a tecla de atalho. ^ significa CTRL, j é a letra j. Qualquer coisa à esquerda de :: são as teclas que você precisa pressionar.
  • A segunda linha. Send, My First Script é como você SEND pressiona as teclas. SEND é o comando, qualquer coisa após a vírgula (,) será digitada.
  • A terceira linha. Retornar. O retorno se tornará seu melhor amigo. Ele literalmente PARE o código de ir mais longe, para as linhas abaixo. Isso evitará muitos problemas quando você começar a ter muitas coisas em seus scripts.
  1. Salve o arquivo.
  2. Clique duas vezes no arquivo/ícone na área de trabalho para executá-lo. Abra o bloco de notas ou (qualquer coisa que você possa digitar) e pressione Ctrl e J.
  3. Hip Hip Viva! Seu primeiro script está pronto. Vá pegar alguns lanches de recompensa e volte a ler o resto deste tutorial.
