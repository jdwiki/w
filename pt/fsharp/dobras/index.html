Introdução às dobras, com alguns exemplos #

Folds são funções (de ordem superior) usadas com sequências de elementos. Eles colapsam seq<'a> em 'b onde 'b é qualquer tipo (possivelmente ainda 'a). Isso é um pouco abstrato, então vamos entrar em exemplos práticos concretos.

Calculando a soma de todos os números #

Neste exemplo, 'a é um int. Temos uma lista de números e queremos calcular a soma de todos os números dela. Para somar os números da lista [1; 2; 3] escrevemos

List.fold (fun x y -> x + y) 0 [1; 2; 3] // val it : int = 6

Deixe-me explicar, porque estamos lidando com listas, usamos fold no módulo List, daí List.fold. o primeiro argumento que fold leva é uma função binária, a pasta. O segundo argumento é o valor inicial. fold começa a dobrar a lista aplicando consecutivamente a função de pasta aos elementos da lista começando com o valor inicial e o primeiro elemento. Se a lista estiver vazia, o valor inicial é retornado!

A visão geral esquemática de um exemplo de execução se parece com isso:

let add x y = x + y // this is our folder (a binary function, takes two arguments)
List.fold add 0 [1; 2; 3;]
=> List.fold add (add 0 1) [2; 3] 
// the binary function is passed again as the folder in the first argument
// the initial value is updated to add 0 1 = 1 in the second argument
// the tail of the list (all elements except the first one) is passed in the third argument
// it becomes this:
List.fold add 1 [2; 3]
// Repeat untill the list is empty -> then return the "inital" value
List.fold add (add 1 2) [3]
List.fold add 3 [3] // add 1 2 = 3
List.fold add (add 3 3) [] 
List.fold add 6 [] // the list is empty now -> return 6
6

A função List.sum é aproximadamente List.fold add LanguagePrimitives.GenericZero onde o zero genérico a torna compatível com inteiros, floats, big integers etc.

Contando elementos em uma lista (implementando count) #

Isso é feito quase da mesma forma que acima, mas ignorando o valor real do elemento na lista e adicionando 1.

List.fold (fun x y -> x + 1) 0 [1; 2; 3] // val it : int = 3

Isso também pode ser feito assim:

[1; 2; 3]
|> List.map (fun x -> 1) // turn every elemet into 1, [1; 2; 3] becomes [1; 1; 1]
|> List.sum // sum [1; 1; 1] is 3

Então você pode definir count da seguinte forma:

let count xs = 
    xs 
    |> List.map (fun x -> 1) 
    |> List.fold (+) 0 // or List.sum

Encontrando o máximo da lista #

Desta vez vamos usar List.reduce que é como List.fold mas sem um valor inicial como neste caso onde não sabemos qual é o tipo dos valores que estamos comparando:

let max x y = if x > y then x else y
// val max : x:'a -> y: 'a -> 'a when 'a : comparison, so only for types that we can compare
List.reduce max [1; 2; 3; 4; 5] // 5
List.reduce max ["a"; "b"; "c"] // "c", because "c" > "b" > "a"
List.reduce max [true; false] // true, because true > false

Encontrando o mínimo de uma lista #

Assim como ao encontrar o max, a pasta é diferente

let min x y = if x < y then x else y
List.reduce min [1; 2; 3; 4; 5] // 1
List.reduce min ["a"; "b"; "c"] // "a"
List.reduce min [true; false] // false

Concatenando listas #

Aqui estamos pegando uma lista de listas A função de pasta é o operador @

// [1;2] @ [3; 4] = [1; 2; 3; 4]
let merge xs ys = xs @ ys
List.fold merge [] [[1;2;3]; [4;5;6]; [7;8;9]] // [1;2;3;4;5;6;7;8;9]

Ou você pode usar operadores binários como sua função de pasta:

List.fold (@) [] [[1;2;3]; [4;5;6]; [7;8;9]] // [1;2;3;4;5;6;7;8;9]
List.fold (+) 0 [1; 2; 3] // 6
List.fold (||) false [true; false] // true, more on this below
List.fold (&&) true [true; false] // false, more on this below
// etc...

Calculando o fatorial de um número #

Mesma ideia de somar os números, mas agora nós os multiplicamos. se quisermos o fatorial de n, multiplicamos todos os elementos da lista [1 .. n]. Código se torna:

// the folder
let times x y = x * y
let factorial n = List.fold times 1 [1 .. n]
// Or maybe for big integers
let factorial n = List.fold times 1I [1I .. n]

Implementando forall, exists e contains #

a função forall verifica se todos os elementos em uma seqüência satisfazem uma condição. exists verifica se pelo menos um elemento na lista satisfaz a condição. Primeiro precisamos saber como recolher uma lista de valores bool. Bem, nós usamos dobras, é claro! operadores booleanos serão nossas funções de pasta.

Para verificar se todos os elementos em uma lista são true, nós os recolhemos com a função && com true como valor inicial.

List.fold (&&) true [true; true; true] // true
List.fold (&&) true [] // true, empty list -> return inital value
List.fold (&&) true [false; true] // false

Da mesma forma, para verificar se um elemento é true em uma lista booleana, nós o recolhemos com o operador || com false como valor inicial:

List.fold (||) false [true; false] // true
List.fold (||) false [false; false] // false, all are false, no element is true
List.fold (||) false [] // false, empty list -> return inital value

Voltar para forall e exists. Aqui, pegamos uma lista de qualquer tipo, usamos a condição para transformar todos os elementos em valores booleanos e, em seguida, recolhemos:

let forall condition elements = 
    elements 
    |> List.map condition // condition : 'a -> bool
    |> List.fold (&&) true

let exists condition elements = 
    elements
    |> List.map condition
    |> List.fold (||) false

Para verificar se todos os elementos em [1; 2; 3; 4] são menores que 5:

forall (fun n -> n < 5) [1 .. 4] // true

defina o método contains com exists:

let contains x xs = exists (fun y -> y = x) xs

Ou mesmo

let contains x xs = exists ((=) x) xs

Agora verifique se a lista [1 .. 5] contém o valor 2:

contains 2 [1..5] // true

Implementando reverse: #

let reverse xs = List.fold (fun acc x -> x :: acc) [] xs
reverse [1 .. 5] // [5; 4; 3; 2; 1]

Implementando map e filter #

let map f = List.fold (fun acc x -> List.append acc [f x]) List.empty
// map (fun x -> x * x) [1..5] -> [1; 4; 9; 16; 25]   

let filter p = Seq.fold (fun acc x -> seq { yield! acc; if p(x) then yield x }) Seq.empty
// filter (fun x -> x % 2 = 0) [1..10] -> [2; 4; 6; 8; 10]

Existe alguma coisa que o fold não pode fazer? eu realmente não sei

Calculando a soma de todos os elementos de uma lista #

Para calcular a soma dos termos (do tipo float, int ou big integer) de uma lista numérica, é preferível usar List.sum Em outros casos, List.fold é a função mais adequada para calcular essa soma.

  1. Soma de números complexos

Neste exemplo, declaramos uma lista de números complexos e calculamos a soma de todos os termos da lista.

No início do programa, adicione uma referência a System.Numerics

abra System.Numerics

Para calcular a soma, inicializamos o acumulador com o número complexo 0.

let clist = [new Complex(1.0, 52.0); new Complex(2.0, -2.0); new Complex(0.0, 1.0)]

let sum = List.fold (+) (new Complex(0.0, 0.0)) clist

Resultado:

(3, 51)
  1. Soma dos números do tipo união

Suponha que uma lista seja composta por números do tipo união (float ou int) e queira calcular a soma desses números.

Declare antes do seguinte tipo de número:

type number = 
| Float of float
| Int of int

Calcular a soma dos números do número do tipo de uma lista:

let list = [Float(1.3); Int(2); Float(10.2)]

let sum = List.fold (
                         fun acc elem -> 
                                        match elem with
                                        | Float(elem) -> acc + elem
                                        | Int(elem) -> acc + float(elem)
                        ) 0.0 list

Resultado:

13.5

O primeiro parâmetro da função, que representa o acumulador, é do tipo float e o segundo parâmetro, que representa um item da lista, é do tipo número. Mas antes de adicionarmos, precisamos usar uma correspondência de padrões e converter para o tipo float quando elem for do tipo Int.
