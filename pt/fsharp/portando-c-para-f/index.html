Classe Implementando uma Interface #

As classes implementam uma interface para atender ao contrato da interface. Por exemplo, uma classe C# pode implementar IDisposable,

public class Resource : IDisposable
{ 
    private MustBeDisposed internalResource;
    
    public Resource() 
    { 
        internalResource = new MustBeDisposed();
    }

    public void Dispose()
    {
        Dispose(true);
        GC.SuppressFinalize(this);
    }

    protected virtual void Dispose(bool disposing){
        if (disposing){
            if (resource != null) internalResource.Dispose();
        }
    }
}

Para implementar uma interface em F#, use interface na definição de tipo,

type Resource() = 
    let internalResource = new MustBeDisposed()

    interface IDisposable with
        member this.Dispose(): unit = 
            this.Dispose(true)
            GC.SuppressFinalize(this)

    member __.Dispose disposing = 
        match disposing with 
        | true  -> if (not << isNull) internalResource then internalResource.Dispose()
        | false -> ()

Alguns #

Alguns dos tipos mais simples de classes são POCOs.

// C#
public class Person
{
    public string FirstName { get; set; }
    public string LastName { get; set; }
    public DateTime Birthday { get; set; }
}

No F# 3.0, propriedades automáticas semelhantes às propriedades automáticas do C# foram introduzidas,

// F#
type Person() = 
    member val FirstName = "" with get, set
    member val LastName = "" with get, set
    member val BirthDay = System.DateTime.Today with get, set

A criação de uma instância de qualquer um é semelhante,

// C#
var person = new Person { FirstName = "Bob", LastName = "Smith", Birthday = DateTime.Today }; 
// F#
let person = new Person(FirstName = "Bob", LastName = "Smith")

Se você pode usar valores imutáveis, um tipo de registro é muito mais idiomático F#.

type Person = { 
    FirstName:string; 
    LastName:string; 
    Birthday:System.DateTime 
}

E este registro pode ser criado:

let person = { FirstName = "Bob"; LastName = "Smith"; Birthday = System.DateTime.Today }

Os registros também podem ser criados com base em outros registros, especificando o registro existente e adicionando with e, em seguida, uma lista de campos a serem substituídos:

let formal = { person with FirstName = "Robert" }
