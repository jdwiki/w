Reversão de uma lista de qualquer tipo #

Para reverter uma lista, não é importante o tipo dos elementos da lista, apenas a ordem em que eles estão. Este é um candidato perfeito para uma função genérica, então a mesma função de reversão pode ser usada não importa qual lista seja passada.

let rev list =
    let rec loop acc = function
        | []           -> acc
        | head :: tail -> loop (head :: acc) tail
    loop [] list

O código não faz suposições sobre os tipos dos elementos. O compilador (ou F# interativo) lhe dirá que a assinatura de tipo desta função é 'T list -> 'T list. O 'T informa que é um tipo genérico sem restrições. Você também pode ver 'a em vez de 'T - a letra não é importante porque é apenas um espaço reservado genérico. Podemos passar uma int list ou uma string list, e ambas funcionarão com sucesso, retornando uma int list ou uma string list respectivamente.

Por exemplo, em F# interativo:

> let rev list = ...
val it : 'T list -> 'T list
> rev [1; 2; 3; 4];;
val it : int list = [4; 3; 2; 1]
> rev ["one", "two", "three"];;
val it : string list = ["three", "two", "one"]

Mapeando uma lista em um tipo diferente #

let map f list =
    let rec loop acc = function
        | []           -> List.rev acc
        | head :: tail -> loop (f head :: acc) tail
    loop [] list

A assinatura desta função é ('a -> 'b) -> 'a list -> 'b list, que é o mais genérico que pode ser. Isso não impede que 'a seja do mesmo tipo que 'b, mas também permite que sejam diferentes. Aqui você pode ver que o tipo 'a que é o parâmetro para a função f deve corresponder ao tipo do parâmetro list. Esta função ainda é genérica, mas há algumas pequenas restrições nas entradas - se os tipos não corresponderem, haverá um erro de compilação.

Exemplos:

> let map f list = ...
val it : ('a -> 'b) -> 'a list -> 'b list
> map (fun x -> float x * 1.5) [1; 2; 3; 4];;
val it : float list = [1.5; 3.0; 4.5; 6.0]
> map (sprintf "abc%.1f") [1.5; 3.0; 4.5; 6.0];;
val it : string list = ["abc1.5"; "abc3.0"; "abc4.5"; "abc6.0"]
> map (fun x -> x + 1) [1.0; 2.0; 3.0];;
error FS0001: The type 'float' does not match the type 'int'
