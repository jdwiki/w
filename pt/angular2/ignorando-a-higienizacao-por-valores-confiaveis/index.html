Parâmetros #

Params Detalhes
seletor nome da tag que você referencia seu componente no html
template(templateUrl) uma string que representa html que será inserida onde quer que a tag <selector> esteja. templateUrl é um caminho para um arquivo html com o mesmo comportamento
pipes um array de pipes que são usados ​​por este componente.

SUPER IMPORTANTE! #

DESATIVAR A SANITIZAÇÃO DEIXA VOCÊ EM RISCO DE XSS (Cross-Site Scripting) E OUTROS VETORES DE ATAQUE. POR FAVOR, CERTIFIQUE-SE DE CONFIAR NO QUE ESTÁ OBTENDO 100%

O uso de Pipes relega você a apenas alterar os valores de atributo da seguinte forma:

<tag [attribute]="expression or variable reference | pipeName">

você não pode usar pipes dessa maneira:

<tag attribute="expression or variable reference | pipeName">

ou desta forma

<tag attribute={{expression or variable reference | pipeName}}

Ignorando Sanitização com tubos (para reutilização de código) #

O projeto está seguindo a estrutura do guia de início rápido do Angular2 aqui.

RootOfProject
|
+-- app
|   |-- app.component.ts
|   |-- main.ts
|   |-- pipeUser.component.ts
|   \-- sanitize.pipe.ts
|
|-- index.html
|-- main.html
|-- pipe.html

main.ts

import { bootstrap } from '@angular/platform-browser-dynamic';
import { AppComponent } from './app.component';

bootstrap(AppComponent);

Isso encontra o arquivo index.html na raiz do projeto e se baseia nisso.

app.component.ts

import { Component } from '@angular/core';
import { PipeUserComponent } from './pipeUser.component';

@Component({
    selector: 'main-app',
    templateUrl: 'main.html',
    directives: [PipeUserComponent]
})

export class AppComponent { }

Este é o componente de nível superior que agrupa outros componentes que são usados.

pipeUser.component.ts

import { Component } from '@angular/core';
import { IgnoreSanitize } from "./sanitize.pipe";

@Component({
    selector: 'pipe-example',
    templateUrl: "pipe.html",
    pipes: [IgnoreSanitize]
})

export class PipeUserComponent{
    constructor () { }        
    unsafeValue: string = "unsafe/picUrl?id=";
    docNum: string;

    getUrl(input: string): any {
        if(input !== undefined) {
            return this.unsafeValue.concat(input);
            // returns : "unsafe/picUrl?id=input"
        } else {
            return "fallback/to/something";
        }
    }
}

Este componente fornece a visualização para o Pipe trabalhar.

sanitize.pipe.ts

import { Pipe, PipeTransform } from '@angular/core';
import { DomSanitizationService } from '@angular/platform-browser';

@Pipe({
    name: 'sanitaryPipe'
})
export class IgnoreSanitize implements PipeTransform {

   constructor(private sanitizer: DomSanitizationService){}

   transform(input: string) : any {
       return this.sanitizer.bypassSecurityTrustUrl(input);
   }

}
