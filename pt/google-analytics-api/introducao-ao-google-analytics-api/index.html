Introdução #

As APIs do Google Analytics permitem que você acesse dados no Google Analytics. Não deve ser confundido com o protocolo de medição que é usado para inserir dados no Google Analytics.

A API do Google Analytics é dividida em partes de serviço.

APIs de relatórios do Google Analytics

  1. A Google Analytics Core Reporting API v3 fornece acesso à maioria dos dados de relatório no Google Analytics. Com a API de relatórios principais, você pode:

    • Build custom dashboards to display Google Analytics data.
    • Save time by automating complex reporting tasks.
    • Integrate your Google Analytics data with other business applications.

  2. A API de relatórios do Google Analytics V4 é o método programático mais avançado para acessar dados de relatórios no Google Analytics. Com a API de relatórios do Google Analytics, você pode:

    • Build custom dashboards to display Google Analytics data.
    • Automate complex reporting tasks to save time.
    • Integrate your Google Analytics data with other business applications.

  3. A API de relatórios em tempo real permite solicitar dados em tempo real—por exemplo, atividade em tempo real em uma visualização—para um usuário autenticado. Você pode usar a API de relatórios em tempo real para:

    • Display active viewers of a page and create a sense of urgency for users looking at an item with finite inventory.
    • Display the most popular content such as the top 10 active pages.
    • Create and display a real time dashboard.

A API de relatórios em tempo real, em versão beta limitada, está disponível para somente visualização do desenvolvedor. Inscreva-se para acessar a API.

  1. A API de relatórios de funis multicanal permite solicitar dados de funis multicanal para um usuário autenticado. Os dados são derivados dos dados do caminho de conversão, que mostram as interações do usuário com várias origens de tráfego em várias sessões antes da conversão. Isso permite que você analise como vários canais de marketing influenciam as conversões ao longo do tempo. Para obter mais detalhes sobre quais dados estão disponíveis, leia Sobre os funis multicanais, bem como Sobre os dados dos funis multicanais. Com a API de relatórios de funis multicanal, você pode:

    • Create custom reports using Multi-Channel Funnels data. For example, you could use the Top Conversion Paths data to report on attributes such as relative position of interactions in a conversion path.
    • Integrate Multi-Channel Funnels data with your business data. For example, you could correlate online conversion data with offline sales data and media cost data to arrive at a more complete picture of marketing ROI.
    • Display Multi-Channel Funnels in new environments. For instance, you could create visualizations and other presentations of the data that communicate the value of different marketing channels in driving conversions.

Ajudantes

  1. A API de metadados retorna a lista e os atributos das colunas (ou seja, dimensões e métricas) expostas nas APIs de relatórios do Google Analytics. Os atributos retornados incluem nome da interface do usuário, descrição, suporte a segmentos e muito mais. Você pode usar a API de metadados para:

    • Automatically discover new columns.
    • Access all dimensions and metrics attributes for Google Analytics reporting APIs.

Observação: isso retorna apenas metadados para a Core Reporting API e o API de relatórios. Não metadados em tempo real.

  1. A Google Analytics Embed API é uma biblioteca JavaScript que permite criar e incorporar facilmente um painel em um site de terceiros em questão de minutos. Ele oferece um conjunto de componentes conectáveis ​​que podem trabalhar juntos para construir ferramentas complexas, tornando-o simples e poderoso ao mesmo tempo.

APIs de configuração

  1. A API de gerenciamento do Analytics permite acesso programático aos dados de configuração do Google Analytics. Você pode criar aplicativos para gerenciar com mais eficiência contas grandes ou complexas do Analytics. Grandes empresas com muitas propriedades podem automatizar a configuração da conta. Mesmo se você estiver criando um aplicativo de relatórios, a API de gerenciamento fornece ferramentas para navegar em sua conta. Você pode usar a API de gerenciamento do Google Analytics para:

    • List all the Account, Property and View information for a user.
    • Manage Properties, Views, and Goals.
    • Manage user permissions for an account hierarchy.
    • Retrieve a View ID to use with the Core Reporting API.
    • Determine which goals are active and access their configured names.
    • Manage Links between Analytics properties and AdWords accounts.
    • Manage Remarketing Audiences.

Operações de gravação na API de gerenciamento (por exemplo, criar, atualizar, excluir, patch) para recursos de propriedade da Web, vista da propriedade (perfil) e meta é atualmente disponível como uma prévia do desenvolvedor em beta limitada. Se você está interessado em usar esses recursos, solicite acesso ao beta.

  1. A API de aprovisionamento pode ser usada para criar novas contas do Google Analytics e ativar o Google Analytics para seus clientes em grande escala. Destina-se a prestadores de serviços qualificados e grandes parceiros. Por exemplo, você pode usar a API de aprovisionamento como parte de um processo de integração de um novo usuário para criar uma nova conta do Google Analytics para um cliente e, em seguida, usar recursos adicionais da API de gerenciamento para configurar programaticamente a conta e vinculá-la ao Google AdWords. Tudo isso pode ser automatizado e iniciado a partir de sua própria interface de administração ou de relatórios.

A API de aprovisionamento está disponível apenas por convite. Nós não somos atualmente aceitando novos projetos.

Acessando as APIs do Google Analytics #

Você pode acessar tecnicamente as APIs do Google Analytics usando qualquer linguagem de programação que possa lidar com uma solicitação HTTP Post ou HTTP Get.

Dito isto, o Google também criou várias bibliotecas de clientes padrão oficiais para ajudá-lo com isso. Usar uma biblioteca cliente padrão para a linguagem de programação escolhida pode ser muito mais fácil do que codificá-la do zero.

Bibliotecas de cliente OFFICAL com suporte à API do Google Analytics:

  1. Biblioteca cliente PHP de APIs do Google - GitHub
  2. Biblioteca de cliente .Net de APIs do Google - GitHub NuGet
  3. Biblioteca cliente Python das APIs do Google - GitHub
  4. Biblioteca cliente Java das APIs do Google - link
  5. Biblioteca Objective-C de APIs do Google - GitHub
