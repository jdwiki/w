Iron é um framework web popular para Rust (baseado na biblioteca de nível inferior Hyper) que promove a ideia de extensibilidade através de * middleware*. Grande parte da funcionalidade necessária para criar um site útil pode ser encontrada no middleware da Iron e não na própria biblioteca.

Servidor simples ‘Olá’ #

Este exemplo envia uma resposta codificada para o usuário quando ele envia uma solicitação do servidor.

extern crate iron;

use iron::prelude::*;
use iron::status;

// You can pass the handler as a function or a closure. In this
// case, we've chosen a function for clarity.
// Since we don't care about the request, we bind it to _.
fn handler(_: &mut Request) -> IronResult<Response> {
    Ok(Response::with((status::Ok, "Hello, Stack Overflow")))
}

fn main() {
    Iron::new(handler).http("localhost:1337").expect("Server failed!")
}

Ao criar um novo servidor Iron neste exemplo, espere detectar quaisquer erros com uma mensagem de erro mais descritiva. Em aplicativos de produção, trate o erro produzido (consulte a documentação para http()).

Roteamento Simples com Ferro #

Este exemplo fornecerá roteamento básico da Web usando o Iron.

Para começar, você precisará adicionar a dependência Iron ao seu arquivo Cargo.toml.

[dependencies]
iron = "0.4.*"

Usaremos a própria biblioteca de roteadores do Iron. Para simplificar, o projeto Iron fornece essa biblioteca como parte da biblioteca principal do Iron, eliminando qualquer necessidade de adicioná-la como uma dependência separada. Em seguida, referenciamos a biblioteca Iron e a biblioteca Router.

extern crate iron;
extern crate router;

Em seguida, importamos os objetos necessários para nos permitir gerenciar o roteamento e retornar uma resposta ao usuário.

use iron::{Iron, Request, Response, IronResult};
use iron::status;
use router::{Router};

Neste exemplo, vamos simplificar escrevendo a lógica de roteamento dentro de nossa função main(). É claro que, à medida que seu aplicativo cresce, você desejará separar roteamento, registro, preocupações de segurança e outras áreas de seu aplicativo da web. Por enquanto, este é um bom ponto de partida.

fn main() {
    let mut router = Router::new();
    router.get("/", handler, "handler");
    router.get("/:query", query_handler, "query_handler");

Vamos rever o que conseguimos até agora. Nosso programa atualmente instancia um novo objeto Iron Router e anexa dois “manipuladores” a dois tipos de solicitação de URL: o primeiro ("/") é a raiz do nosso domínio, e o segundo ("/:query ") é qualquer caminho na raiz.

Ao usar um ponto e vírgula antes da palavra “query”, estamos dizendo ao Iron para pegar essa parte do caminho da URL como uma variável e passá-la para nosso manipulador.

A próxima linha de código é como instanciamos o Iron, designando nosso próprio objeto roteador para gerenciar nossas solicitações de URL. O domínio e a porta são codificados neste exemplo para simplificar.

    Iron::new(router).http("localhost:3000").unwrap();

Em seguida, declaramos duas funções inline que são nossos manipuladores, handler e query_handler. Ambos são usados ​​para demonstrar URLs fixos e URLs variáveis.

Na segunda função, pegamos a variável "query" da URL mantida pelo objeto de solicitação e a enviamos de volta ao usuário como resposta.

    fn handler(_: &mut Request) -> IronResult<Response> {
        Ok(Response::with((status::Ok, "OK")))
    }

    fn query_handler(req: &mut Request) -> IronResult<Response> {
        let ref query = req.extensions.get::<Router>()
            .unwrap().find("query").unwrap_or("/");
        Ok(Response::with((status::Ok, *query)))
    }
}

Se executarmos este exemplo, poderemos ver o resultado no navegador web em localhost:3000. A raiz do domínio deve responder com "OK", e qualquer coisa abaixo da raiz deve repetir o caminho de volta.

A próxima etapa deste exemplo pode ser a separação do roteamento e a veiculação de páginas estáticas.

Instalando o Ferro #

Adicione esta dependência ao arquivo Cargo.toml:

[dependencies]
iron = "0.4.0"

Execute o cargo build e o Cargo fará o download e instalará a versão especificada do Iron.
