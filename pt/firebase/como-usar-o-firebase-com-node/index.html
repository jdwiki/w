Olá mundo Firebase Realtime Database in Node #

Requisitos de sistema:

  • Nó JS

Começando

  1. Primeiro, vá para Firebase Console e crie um novo projeto.

  2. Após criar o projeto, no projeto clique no ícone de configurações ao lado do nome do projeto na barra lateral esquerda e selecione Permissões.

  3. Na página de permissões, clique em contas de serviço na barra lateral esquerda e clique em criar conta de serviço

  4. Na janela pop-up, digite o nome da sua conta de serviço e escolha Função da conta e selecione Fornecer uma nova chave privada e depois selecione JSON e clique em Criar (Deixe Ativar delegação em todo o domínio do Google App desmarcada).

  5. Ao clicar em criar ele irá baixar um arquivo JSON com suas Credenciais de Conta, basta salvar o arquivo em Qualquer Lugar do seu Sistema.

  6. A próxima etapa é criar um banco de dados no Firebase Console para o qual vá para o Firebase Console e clique em Database na barra lateral esquerda. Depois disso, basta criar um novo objeto de banco de dados com o nome user_data com algum valor fictício.

  7. Agora que seu projeto do Firebase Database está configurado, basta copiar o código a seguir no diretório do seu projeto.

    //Loading Firebase Package var firebase = require(“firebase”);

    /**

    • Update your Firebase Project
    • Credentials and Firebase Database
    • URL */ firebase.initializeApp({ serviceAccount: “”, databaseURL: “” }); //by adding your credentials, you get authorized to read and write from the database

    /**

    • Loading Firebase Database and refering
    • to user_data Object from the Database */ var db = firebase.database(); var ref = db.ref("/user_data"); //Set the current directory you are working in

    /**

    • Setting Data Object Value */ ref.set([ { id:20, name:“Jane Doe”, email:“jane@doe.com”, website:“https://jane.foo.bar” }, { id:21, name:“John doe”, email:“john@doe.com”, website:“https://foo.bar” } ]);

    /**

    • Pushing New Value
    • in the Database Object */ ref.push({ id:22, name:“Jane Doe”, email:“jane@doe.com”, website:“https://jane.foo.bar” });

    /**

    • Reading Value from
    • Firebase Data Object */ ref.once(“value”, function(snapshot) { var data = snapshot.val(); //Data is in JSON format. console.log(data); });

  8. Basta alterar com a URL do arquivo de credenciais JSON (Para começar basta copiar o arquivo de credenciais na mesma pasta e no arquivo index.js basta adicionar o nome do arquivo de credenciais).

  9. O próximo passo é alterar o in index.js com o URL real do Firebase Database, você poderá encontrar este URL no Firebase Console na guia Database, o URL será como https://.firebaseio.com /.

  10. O passo final é fazer

    npm install firebase

  11. Após executar o comando acima, o NPM instalará os pacotes necessários para o Firebase. Finalmente para executar e testar o projeto execute

    node index.js

O que o projeto realmente faz?

O projeto carrega os dados do Firebase Database baseado em nuvem. O projeto também demonstra como gravar e ler dados de um Firebase Data Object.

Para ver seus dados atualizados em tempo real, vá para seu console clique no projeto que você fez e, à esquerda, clique em Banco de Dados. Lá, você pode ver seus dados atualizados em tempo real, juntamente com seus valores.

Fila e trabalhador do Firebase #

Você pode enviar tarefas ou dados para o banco de dados em tempo real do Firebase e executar um trabalhador que escuta a fila do Firebase para executar alguns processos em segundo plano

Configurar o Firebase

  1. Crie um projeto do Firebase no console do Firebase, caso ainda não tenha um. Se você já tiver um projeto do Google associado ao seu aplicativo, clique em Importar projeto do Google. Caso contrário, clique em Criar novo projeto..

  2. Clique no ícone de configurações e selecione Permissões.

  3. Selecione Contas de serviço no menu à esquerda.

  4. Clique em Criar conta de serviço.

    Enter a name for your service account.

    You can optionally customize the ID from the one automatically generated from the name.

    Choose Project > Editor from the Role dropdown.

    Select Furnish a new private key and leave the Key type as JSON.

    Leave Enable Google Apps Domain-wide Delegation unselected.

    Click Create

  5. Quando você cria a conta de serviço, um arquivo JSON contendo as credenciais da sua conta de serviço é baixado para você. Você precisará disso para inicializar o SDK no servidor.

Configurar servidor

Instale o firebase-queue usando npm em seu aplicativo nodejs

npm install firebase firebase-queue --save

Depois de instalar o firebase e o firebase-queue, você pode começar criando uma nova Queue e transmitindo sua referência do Firebase e uma função de processamento.

Agora vamos criar uma tarefa de fila do Firebase a partir do aplicativo quando um novo usuário é criado e definir o trabalhador para escutar a tarefa de fila do Firebase e enviar um e-mail para os usuários criados.

*servidor.js

var app=express();
var Queue = require('firebase-queue'),
    Firebase = require('firebase');

Atualize suas credenciais de projeto do Firebase e o URL do banco de dados do Firebase

var firebase = Firebase.initializeApp({
    serviceAccount: "path/to/serviceAccountCredentials.json",
  databaseURL: "https://databaseName.firebaseio.com"
});

ou você pode inserir as credenciais do Firebase diretamente como abaixo

var firebase = Firebase.initializeApp({ 
    serviceAccount: {
        projectId: "projectId",
        clientEmail: "foo@projectId.iam.gserviceaccount.com",
        privateKey: "-----BEGIN PRIVATE KEY-----\nkey\n-----END PRIVATE KEY-----\n"
    },
    databaseURL: "https://databaseName.firebaseio.com"
});


var refQueue = firebase.database().ref("queue/tasks");

createUser =  funtion(email, password){
    var user = {
        username: email,
        password: password
    };
    user = new db.users(user);
    user.save(function(err, user){
        if(!err){
            refQueue.push({case: "NEW_USER", data: user});
        }
    })
}

createUser("abc@abc.com", "password");

*trabalhador.js

var Queue = require('firebase-queue'),
    Firebase = require('firebase');

//Update your Firebase Project Credentials and Firebase Database URL by one of the way specified in server.js
var firebase = Firebase.initializeApp({
    serviceAccount: "path/to/serviceAccountCredentials.json",
  databaseURL: "https://databaseName.firebaseio.com"
});

var refQueue = firebase.database().ref("queue");

var queue = new Queue(refQueue, function(data, progress, resolve, reject) {
    switch(data.case){
        case "NEW_USER":
            sendMail(data.data.email);
            console.log("user created");
            //sendMail function is not an inbuilt function and will not work unless you define and implement the function
            break;

    // Finish the task asynchronously
    setTimeout(function() {
        resolve();
    }, 1000);
});

execute o servidor e o trabalhador separadamente e teste com a fila do Firebase

node server.js

node worker.js
