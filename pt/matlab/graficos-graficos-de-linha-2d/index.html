Sintaxe #

  • plot(Y)

  • plot(Y,LineSpec)

  • plot(X,Y)

  • plot(X,Y,LineSpec)

  • plot(X1,Y1, X2,Y2, …, Xn,Yn)

  • plot(X1,Y1,LineSpec1, X2,Y2,LineSpec2, …, Xn,Yn,LineSpecn)

  • plot(___, Nome, Valor)

  • h = plot(___)

Parâmetros #

Parâmetro Detalhes
X valores x
S valores y
LineSpec Estilo de linha, símbolo de marcador e cor, especificados como uma string
Nome, Valor Pares opcionais de argumentos nome-valor para personalizar as propriedades da linha
h identificador para objeto gráfico de linha

http://www.mathworks.com/help/matlab/ref/plot.html

Várias linhas em um único gráfico #

Neste exemplo, vamos plotar várias linhas em um único eixo. Além disso, escolhemos uma aparência diferente para as linhas e criamos uma legenda.

% create sample data
x = linspace(-2,2,100);                 % 100 linearly spaced points from -2 to 2
y1 = x.^2;
y2 = 2*x.^2;
y3 = 4*x.^2;

% create plot
figure;                                 % open new figure
plot(x,y1, x,y2,'--', x,y3,'-.');       % plot lines
grid minor;                             % add minor grid
title('Quadratic functions with different curvatures');
xlabel('x');
ylabel('f(x)');
legend('f(x) = x^2', 'f(x) = 2x^2', 'f(x) = 4x^2', 'Location','North');

No exemplo acima, plotamos as linhas com um único comando plot. Uma alternativa é usar comandos separados para cada linha. Precisamos manter o conteúdo de uma figura com hold on o mais recente antes de adicionarmos a segunda linha. Caso contrário, as linhas traçadas anteriormente desaparecerão da figura. Para criar o mesmo gráfico acima, podemos usar os seguintes comandos:

figure; hold on;
plot(x,y1);
plot(x,y2,'--');
plot(x,y3,'-.');

Pedidos personalizados de cor e estilo de linha #

No MATLAB, podemos definir novas ordens personalizadas default, como uma ordem de cores e uma ordem de estilo de linha. Isso significa que novos pedidos serão aplicados a qualquer figura criada após a aplicação dessas configurações. As novas configurações permanecem até que a sessão do MATLAB seja fechada ou novas configurações tenham sido feitas.

Ordem padrão de cor e estilo de linha

Por padrão, o MATLAB usa algumas cores diferentes e apenas um estilo de linha sólida. Portanto, se plot for chamado para desenhar várias linhas, o MATLAB alterna entre uma ordem de cores para desenhar linhas em cores diferentes.

Podemos obter a ordem de cores padrão chamando get com um identificador global 0 seguido por este atributo DefaultAxesColorOrder:

>> get(0, 'DefaultAxesColorOrder')
ans =
         0    0.4470    0.7410
    0.8500    0.3250    0.0980
    0.9290    0.6940    0.1250
    0.4940    0.1840    0.5560
    0.4660    0.6740    0.1880
    0.3010    0.7450    0.9330
    0.6350    0.0780    0.1840

Cor personalizada e ordem de estilo de linha

Uma vez que tenhamos decidido definir uma ordem de cor personalizada E uma ordem de estilo de linha, o MATLAB deve alternar entre ambos. A primeira mudança que o MATLAB aplica é uma cor. Quando todas as cores se esgotarem, o MATLAB aplica o próximo estilo de linha de uma ordem de estilo de linha definida e define um índice de cores para 1. Isso significa que o MATLAB começará a alternar entre todas as cores novamente, mas usando o próximo estilo de linha em sua ordem. Quando todos os estilos de linha e cores estão esgotados, obviamente o MATLAB começa a circular desde o início usando a primeira cor e o primeiro estilo de linha.

Para este exemplo, defini um vetor de entrada e uma função anônima para facilitar um pouco a plotagem de figuras:

F = @(a,x) bsxfun(@plus, -0.2*x(:).^2, a);
x = (-5:5/100:5-5/100)';

Para definir uma nova cor ou um novo estilo de linha, chamamos a função set com um handle global 0 seguido por um atributo DefaultAxesXXXXXXX; XXXXXXX pode ser ColorOrder ou LineStyleOrder. O comando a seguir define uma nova ordem de cores para preto, vermelho e azul, respectivamente:

set(0, 'DefaultAxesColorOrder', [0 0 0; 1 0 0; 0 0 1]);
plot(x, F([1 2 3 4 5 6],x));

Como você pode ver, o MATLAB alterna apenas entre cores porque a ordem do estilo de linha é definida como uma linha sólida por padrão. Quando um conjunto de cores está esgotado, o MATLAB começa a partir da primeira cor na ordem de cores.

Os comandos a seguir definem as ordens de cor e estilo de linha:

set(0, 'DefaultAxesColorOrder', [0 0 0; 1 0 0; 0 0 1]);
set(0, 'DefaultAxesLineStyleOrder', {'-' '--'});
plot(x, F([1 2 3 4 5 6],x));

Agora, o MATLAB alterna entre diferentes cores e diferentes estilos de linha usando a cor como atributo mais frequente.
