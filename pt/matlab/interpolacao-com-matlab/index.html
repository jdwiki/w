Sintaxe #

  1. zy = interp1(x,y);
  2. zy = interp1(x,y,‘método’);
  3. zy = interp1(x,y,‘método’,’extrapolação’);
  4. zy = interp1(x,y,zx);
  5. zy = interp1(x,y,zx,‘método’);
  6. zy = interp1(x,y,zx,‘método’,’extrapolação’);

Interpolação por partes bidimensional #

Inicializamos os dados:

[X,Y] = meshgrid(1:2:10);
Z = X.*cos(Y) - Y.*sin(X);

A superfície se parece com o seguinte.

Agora definimos os pontos onde queremos interpolar:

[Vx,Vy] = meshgrid(1:0.25:10);

Agora podemos realizar a interpolação mais próxima,

Vz = interp2(X,Y,Z,Vx,Vy,'nearest');

interpolação linear,

Vz = interp2(X,Y,Z,Vx,Vy,'linear');

interpolação cúbica

Vz = interp2(X,Y,Z,Vx,Vy,'cubic');

ou interpolação de spline:

Vz = interp2(X,Y,Z,Vx,Vy,'spline');

Interpolação por partes 1 dimensional #

Usaremos os seguintes dados:

x = 1:5:50;
y = randi([-10 10],1,10);

Aqui x e y são as coordenadas dos pontos de dados e z são os pontos sobre os quais precisamos de informações.

z = 0:0.25:50;

Uma maneira de encontrar os valores de y de z é a interpolação linear por partes.

z_y = interp1(x,y,z,'linear');

Assim, calcula-se a linha entre dois pontos adjacentes e obtém-se z_y assumindo que o ponto seria um elemento dessas linhas.

interp1 também oferece outras opções como interpolação mais próxima,

z_y = interp1(x,y,z, 'nearest');

próxima interpolação,

z_y = interp1(x,y,z, 'next');

interpolação anterior,

z_y = interp1(x,y,z, 'previous');

Interpolação cúbica por partes que preserva a forma,

z_y = interp1(x,y,z, 'pchip');

convolução cúbica, z_y = interp1(x,y,z, ‘v5cubic’);

e interpolação de spline

z_y = interp1(x,y,z, 'spline');

Aqui estão interpolações constantes por partes de interpolação próxima, próxima e anterior.

Interpolação polinomial #

Inicializamos os dados que queremos interpolar:

x = 0:0.5:10;
y = sin(x/2);

Isso significa que a função subjacente para os dados no intervalo [0,10] é senoidal. Agora os coeficientes dos polinômios de aproximação estão sendo calculados:

p1 = polyfit(x,y,1);
p2 = polyfit(x,y,2);
p3 = polyfit(x,y,3);
p5 = polyfit(x,y,5);
p10 = polyfit(x,y,10);

Aqui é x o valor x e y o valor y de nossos pontos de dados e o terceiro número é a ordem/grau do polinômio. Agora definimos a grade na qual queremos calcular nossa função de interpolação:

zx = 0:0.1:10;

e calcule os valores de y:

zy1 = polyval(p1,zx);
zy2 = polyval(p2,zx);
zy3 = polyval(p3,zx);
zy5 = polyval(p5,zx);
zy10 = polyval(p10,zx);

Pode-se ver que o erro de aproximação para a amostra fica menor quando o grau do polinômio aumenta.

Embora a aproximação da linha reta neste exemplo tenha erros maiores, o polinômio de ordem 3 aproxima a função sinusal neste intervalo de forma relativamente boa.

A interpolação com polinômios de ordem 5 e ordem 10 tem quase nenhum erro de aproximação.

No entanto, se considerarmos o desempenho fora da amostra, vemos que pedidos muito altos tendem a se ajustar demais e, portanto, apresentam um desempenho ruim fora da amostra. Montamos

zx = -10:0.1:40;
p10 = polyfit(X,Y,10);
p20 = polyfit(X,Y,20);

e

zy10 = polyval(p10,zx);
zy20 = polyval(p20,zx);

Se dermos uma olhada no gráfico, veremos que o desempenho fora da amostra é melhor para o pedido 1

e continua piorando com o aumento do grau.
