  • O texto de ajuda pode ser localizado antes ou depois da linha function, desde que não haja código entre a linha de função e o início do texto de ajuda.
  • A capitalização do nome da função apenas coloca o nome em negrito e não é obrigatória.
  • Se uma linha for prefixada com Veja também, quaisquer nomes na linha que correspondam ao nome de uma classe ou função no caminho de pesquisa serão automaticamente vinculados à documentação dessa classe/função.
  • Funções globais podem ser referenciadas aqui precedendo seu nome com um \. Caso contrário, os nomes tentarão primeiro resolver para funções de membro.
  • Hiperlinks no formato <a href="matlab:web('url')">Nome</a> são permitidos.

Documentação de funções simples #

function output = mymult(a, b)
% MYMULT Multiply two numbers.
%   output = MYMULT(a, b) multiplies a and b.
%
%   See also fft, foo, sin.
%
%   For more information, see <a href="matlab:web('https://google.com')">Google</a>.
    output = a * b;
end

help mymult fornece:

mymult Multiplique dois números.

output = mymult(a, b) multiplica a e b.

Veja também fft, foo, sin.

Para obter mais informações, consulte Google.

fft e sin são vinculados automaticamente a seus respectivos textos de ajuda, e o Google é um link para google.com. foo não irá vincular a nenhuma documentação neste caso, desde que não haja uma função/classe documentada com o nome de foo no caminho de busca.

Obtendo uma assinatura de função #

Muitas vezes, é útil fazer com que o MATLAB imprima a 1st linha de uma função, pois geralmente contém a assinatura da função, incluindo entradas e saídas:

dbtype <functionName> 1

Exemplo:

>> dbtype fit 1

1  function [fitobj,goodness,output,warnstr,errstr,convmsg] = fit(xdatain,ydatain,fittypeobj,varargin)

Documentação da função local #

Neste exemplo, a documentação para a função local baz (definida em foo.m) pode ser acessada pelo link resultante em help foo, ou diretamente através de help foo>baz.

function bar = foo
%This is documentation for FOO.
% See also foo>baz

% This wont be printed, because there is a line without % on it.
end

function baz
% This is documentation for BAZ.
end

Documentando uma função com um script de exemplo #

Para documentar uma função, geralmente é útil ter um script de exemplo que use sua função. A função de publicação no Matlab pode ser usada para gerar um arquivo de ajuda com imagens, código, links incorporados, etc. A sintaxe para documentar seu código pode ser encontrada aqui.

A função Esta função usa uma FFT “corrigida” no Matlab.

function out_sig = myfft(in_sig)

out_sig = fftshift(fft(ifftshift(in_sig)));

end

O script de exemplo Este é um script separado que explica as entradas, saídas e dá um exemplo explicando por que a correção é necessária. Graças a Wu, Kan, o autor original desta função.

%% myfft
% This function uses the "proper" fft in matlab. Note that the fft needs to
% be multiplied by dt to have physical significance.
% For a full description of why the FFT should be taken like this, refer
% to: Why_use_fftshift(fft(fftshift(x)))__in_Matlab.pdf included in the
% help folder of this code. Additional information can be found:
% <https://www.mathworks.com/matlabcentral/fileexchange/25473-why-use-fftshift-fft-fftshift-x----in-matlab-instead-of-fft-x-->
%
%% Inputs
% *in_sig* - 1D signal
% 
%% Outputs
% *out_sig* - corrected FFT of *in_sig*
% 
%% Examples
% Generate a signal with an analytical solution. The analytical solution is
% then compared to the fft then to myfft. This example is a modified
% version given by Wu, Kan given in the link aboce.
%%
% Set parameters
fs = 500;           %sampling frequency
dt = 1/fs;          %time step
T=1;                %total time window
t = -T/2:dt:T/2-dt; %time grids
df = 1/T;           %freq step
Fmax = 1/2/dt;      %freq window
f=-Fmax:df:Fmax-df; %freq grids, not used in our examples, could be used by plot(f, X)
%%
% Generate Gaussian curve
Bx = 10; A = sqrt(log(2))/(2*pi*Bx);    %Characteristics of Gaussian curve
x = exp(-t.^2/2/A^2);                   %Create Gaussian Curve
%% 
% Generate Analytical solution
Xan = A*sqrt(2*pi)*exp(-2*pi^2*f.^2*A^2); %X(f), real part of the analytical Fourier transform of x(t)

%%
% Take FFT and corrected FFT then compare
Xfft = dt *fftshift(fft(x));    %FFT
Xfinal = dt * myfft(x);         %Corrected FFT
hold on
plot(f,Xan);
plot(f,real(Xfft));
plot(f,real(Xfinal),'ro');
title('Comparison of Corrected and Uncorrected FFT');
legend('Analytical Solution','Uncorrected FFT','Corrected FFT');
xlabel('Frequency'); ylabel('Amplitude');
DT = max(f) - min(f);
xlim([-DT/4,DT/4]);

A saída A opção de publicação pode ser encontrada na guia “Publicar”, destacada na imagemhttps://www.wikiod.com/pt/matlab/funcoes-de-documentacao#Documentação de Função Simples abaixo.

O Matlab executará o script e salvará as imagens exibidas, bem como o texto gerado pela linha de comando. A saída pode ser salva em muitos tipos diferentes de formatos, incluindo HTML, Latex e PDF.
