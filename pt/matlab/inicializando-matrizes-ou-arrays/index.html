O Matlab tem três funções importantes para criar matrizes e definir seus elementos como zeros, uns ou a matriz identidade. (A matriz identidade tem uns na diagonal principal e zeros em outros lugares.)

Sintaxe #

  • Z = zeros (sz, tipo de dados, tipo de matriz)
  • X = uns(sz, tipo de dados)
  • I = olho(sz, tipo de dados)

Parâmetros #

Parâmetro Detalhes
sz n (para uma matriz n x n)
sz n, m (para uma matriz n x m)
sz m,n,…,k (para uma matriz m-por-n-por-…-por-k)
tipo de dados ‘double’ (padrão), ‘single’, ‘int8’, ‘uint8’, ‘int16’, ‘uint16’, ‘int32’, ‘uint32’, ‘int64’ ou ‘uint64’
tipo de matriz ‘distribuído’
tipo de matriz ‘codistribuído’
tipo de matriz ‘gpuArray’

Essas funções criarão uma matriz de duplas, por padrão.

Criando uma matriz de 0s #

z1 = zeros(5); % Create a 5-by-5 matrix of zeroes
z2 = zeros(2,3); % Create a 2-by-3 matrix

Criando uma matriz de 1s #

o1 = ones(5); % Create a 5-by-5 matrix of ones
o2 = ones(1,3); % Create a 1-by-3 matrix / vector of size 3

Criando uma matriz de identidade #

i1 = eye(3); % Create a 3-by-3 identity matrix
i2 = eye(5,6); % Create a 5-by-6 identity matrix
