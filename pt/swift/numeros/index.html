Converte números de/para strings #

Use inicializadores de String para converter números em strings:

String(1635999)                              // returns "1635999"
String(1635999, radix: 10)                   // returns "1635999"
String(1635999, radix: 2)                    // returns "110001111011010011111"
String(1635999, radix: 16)                   // returns "18f69f"
String(1635999, radix: 16, uppercase: true)  // returns "18F69F"
String(1635999, radix: 17)                   // returns "129gf4"
String(1635999, radix: 36)                   // returns "z2cf"

Ou use [interpolação de strings](https://www.wikiod.com/pt/swift/strings-e-personagens#Literais de Strings e Caracteres) para casos simples:

let x = 42, y = 9001
"Between \(x) and \(y)"  // equivalent to "Between 42 and 9001"

Use inicializadores de tipos numéricos para converter strings em números:

if let num = Int("42") { /* ... */ }                // num is 42
if let num = Int("Z2cF") { /* ... */ }              // returns nil (not a number)
if let num = Int("z2cf", radix: 36) { /* ... */ }   // num is 1635999
if let num = Int("Z2cF", radix: 36) { /* ... */ }   // num is 1635999
if let num = Int8("Z2cF", radix: 36) { /* ... */ }  // returns nil (too large for Int8)

Tipos de números e literais #

Os tipos numéricos integrados do Swift são:

  • Tamanho do Word (dependente da arquitetura) assinado Int e não assinado UInt.
  • Números inteiros assinados de tamanho fixo Int8, Int16, Int32, Int64 e inteiros não assinados UInt8, UInt16, UInt32, UInt64.
  • Tipos de ponto flutuante Float32/Float, [Float64/Double][Double] e Float80 (somente x86).

Literais #

O tipo de um literal numérico é inferido do contexto:

let x = 42    // x is Int by default
let y = 42.0  // y is Double by default

let z: UInt = 42     // z is UInt
let w: Float = -1    // w is Float
let q = 100 as Int8  // q is Int8

Os sublinhados (_) podem ser usados ​​para separar dígitos em literais numéricos. Os zeros à esquerda são ignorados.

Literais de ponto flutuante podem ser especificados usando significand e partes expoentes («significand» e</ b> «expoente» para decimal; 0x «significativo» p  «expoente» para hexadecimal).

Sintaxe literal de inteiro #

let decimal = 10               // ten
let decimal = -1000            // negative one thousand
let decimal = -1_000           // equivalent to -1000
let decimal = 42_42_42         // equivalent to 424242
let decimal = 0755             // equivalent to 755, NOT 493 as in some other languages
let decimal = 0123456789

let hexadecimal = 0x10         // equivalent to 16
let hexadecimal = 0x7FFFFFFF
let hexadecimal = 0xBadFace
let hexadecimal = 0x0123_4567_89ab_cdef

let octal = 0o10               // equivalent to 8
let octal = 0o755              // equivalent to 493
let octal = -0o0123_4567             

let binary = -0b101010         // equivalent to -42
let binary = 0b111_101_101     // equivalent to 0o755
let binary = 0b1011_1010_1101  // equivalent to 0xB_A_D

Sintaxe literal de ponto flutuante #

let decimal = 0.0
let decimal = -42.0123456789
let decimal = 1_000.234_567_89

let decimal = 4.567e5               // equivalent to 4.567×10⁵, or 456_700.0
let decimal = -2E-4                 // equivalent to -2×10⁻⁴, or -0.0002
let decimal = 1e+0                  // equivalent to 1×10⁰, or 1.0

let hexadecimal = 0x1p0             // equivalent to 1×2⁰, or 1.0
let hexadecimal = 0x1p-2            // equivalent to 1×2⁻², or 0.25
let hexadecimal = 0xFEEDp+3         // equivalent to 65261×2³, or 522088.0
let hexadecimal = 0x1234.5P4        // equivalent to 0x12345, or 74565.0
let hexadecimal = 0x123.45P8        // equivalent to 0x12345, or 74565.0
let hexadecimal = 0x12.345P12       // equivalent to 0x12345, or 74565.0
let hexadecimal = 0x1.2345P16       // equivalent to 0x12345, or 74565.0
let hexadecimal = 0x0.12345P20      // equivalent to 0x12345, or 74565.0

Arredondamento #

redondo #

Arredonda o valor para o número inteiro mais próximo com x.5 arredondando para cima (mas observe que -x.5 arredonda para baixo).

round(3.000) // 3
round(3.001) // 3
round(3.499) // 3
round(3.500) // 4
round(3.999) // 4

round(-3.000) // -3
round(-3.001) // -3
round(-3.499) // -3
round(-3.500) // -4  *** careful here ***
round(-3.999) // -4

teto #

Arredonda qualquer número com um valor decimal até o próximo número inteiro maior.

ceil(3.000) // 3
ceil(3.001) // 4
ceil(3.999) // 4

ceil(-3.000) // -3
ceil(-3.001) // -3
ceil(-3.999) // -3

piso #

Arredonda qualquer número com um valor decimal para baixo para o próximo número inteiro menor.

floor(3.000) // 3
floor(3.001) // 3
floor(3.999) // 3

floor(-3.000) // -3
floor(-3.001) // -4
floor(-3.999) // -4

Int #

Converte um Double em um Int, descartando qualquer valor decimal.

Int(3.000) // 3
Int(3.001) // 3
Int(3.999) // 3

Int(-3.000) // -3
Int(-3.001) // -3
Int(-3.999) // -3

Notas #

  • round, ceil e floor lidam com arquiteturas de 64 e 32 bits.

Converte um tipo numérico para outro #

func doSomething1(value: Double) { /* ... */ }
func doSomething2(value: UInt) { /* ... */ }

let x = 42               // x is an Int
doSomething1(Double(x))  // convert x to a Double
doSomething2(UInt(x))    // convert x to a UInt

Os inicializadores inteiros produzem um erro de tempo de execução se o valor exceder ou não:

Int8(-129.0) // fatal error: floating point value cannot be converted to Int8 because it is less than Int8.min
Int8(-129)   // crash: EXC_BAD_INSTRUCTION / SIGILL
Int8(-128)   // ok
Int8(-2)     // ok
Int8(17)     // ok
Int8(127)    // ok
Int8(128)    // crash: EXC_BAD_INSTRUCTION / SIGILL
Int8(128.0)  // fatal error: floating point value cannot be converted to Int8 because it is greater than Int8.max

Conversão de flutuante para inteiro arredonda os valores para zero:

Int(-2.2)  // -2
Int(-1.9)  // -1
Int(-0.1)  //  0
Int(1.0)   //  1
Int(1.2)   //  1
Int(1.9)   //  1
Int(2.0)   //  2

A conversão de inteiro para flutuante pode ser com perdas:

Int(Float(1_000_000_000_000_000_000))  // 999999984306749440

Geração de números aleatórios #

arc4random_uniform(someNumber: UInt32) -> UInt32

Isso fornece números inteiros aleatórios no intervalo de 0 a someNumber - 1.

O valor máximo para UInt32 é 4.294.967.295 (ou seja, 2^32 - 1).

Exemplos:

  • Troca de moedas

      let flip = arc4random_uniform(2) // 0 or 1

  • Rolar os dados

      let roll = arc4random_uniform(6) + 1 // 1...6

  • Dia aleatório em outubro

      let day = arc4random_uniform(31) + 1 // 1...31

  • Ano aleatório na década de 1990

      let year = 1990 + arc4random_uniform(10)

Forma geral:

let number = min + arc4random_uniform(max - min + 1)

onde number, max e min são UInt32.

Notas #

  • Há um ligeiro viés de módulo com arc4random, então arc4random_uniform é o preferido.
  • Você pode converter um valor UInt32 para um Int, mas apenas tome cuidado para não sair do alcance.

Exponenciação #

Em Swift, podemos exponenciar Doubles com o método interno pow():

pow(BASE, EXPONENT)

No código abaixo, a base (5) é definida como a potência do expoente (2):

let number = pow(5.0, 2.0) // Equals 25
