Seu primeiro programa Swift #

Escreva seu código em um arquivo chamado hello.swift:

print("Hello, world!")
  • Para compilar e executar um script em uma única etapa, use swift do terminal (em um diretório onde este arquivo está localizado):

Para iniciar um terminal, pressione CTRL+ALT+T no Linux, ou encontre-o no Launchpad no macOS. Para alterar o diretório, digite cddirectory_name (ou cd .. para voltar)

$ rápido olá.rápido
Olá, mundo!

Um compilador é um programa de computador (ou um conjunto de programas) que transforma o código-fonte escrito em uma linguagem de programação (a linguagem de origem) em outra linguagem de computador (a linguagem de destino), sendo que esta última geralmente tem uma forma binária conhecido como código objeto. (Wikipedia)

  • Para compilar e executar separadamente, use swiftc:
$ swiftc olá.swift

Isso compilará seu código em um arquivo hello. Para executá-lo, digite ./, seguido por um nome de arquivo.

$ ./olá
Olá, mundo!
  • Ou use o swift REPL (Read-Eval-Print-Loop), digitando swift na linha de comando, e depois digitando seu código no interpretador:

Código:

<pré> func saudação(nome: String, sobrenome: String) { print(“Greetings (name) (surname)”) }

let meuNome = “Homer” let meuSobrenome = “Simpson”

saudar(nome: meuNome, sobrenome: meuSobrenome)

Vamos quebrar este código grande em pedaços:

  • func greet(name: String, surname: String) { // corpo da função } - cria uma função que recebe um nome e um sobrenome.

  • print("Saudações \(nome) \(sobrenome)") - Isso imprime no console “Saudações”, então nome, então sobrenome. Basicamente \(variable_name) imprime o valor dessa variável.

  • let myName = "Homer" e let mySurname = "Simpson" - crie constantes (variáveis ​​cujo valor você não pode alterar) usando let com nomes: myName, mySurname e valores : "Homer", "Simpson" respectivamente.

  • greet(name: myName, surname: mySurname) - chama uma função que criamos anteriormente fornecendo os valores de constants myName, mySurname.

Executando usando REPL:

$ rápido
Bem-vindo ao Apple Swift. Digite :help para obter assistência.
1> func saudação(nome: String, sobrenome: String) { 
2. print("Saudações \(nome) \(sobrenome)") 
3. } 
4>
5> deixe meuNome = "Homer"
meuNome: String = "Homer"
6> deixe meuSobrenome = "Simpson"
meuSobrenome: String = "Simpson"
7> saudação(nome: meuNome, sobrenome: meuSobrenome)
Saudações Homer Simpson
8> ^D

Pressione CTRL+D para sair do REPL.

Seu primeiro programa em Swift em um Mac (usando um Playground) #

No seu Mac, baixe e instale o Xcode da Mac App Store seguindo este link.

Após a conclusão da instalação, abra o Xcode e selecione Get started with a Playground:

No próximo painel, você pode dar um nome ao seu Playground ou pode deixá-lo como MyPlayground e pressionar Next:

Selecione um local para salvar o Playground e pressione Criar:

O Playground será aberto e sua tela deverá ficar assim:

Agora que o Playground está na tela, pressione ⇧ + cmd + Y para mostrar a Área de Depuração.

Por fim, exclua o texto dentro do Playground e digite:

print("Hello world")

Você deverá ver “Hello world” na Área de depuração e “Hello world\n” na Barra Lateral direita:

Parabéns! Você criou seu primeiro programa em Swift!

Instalando o Swift #

Primeiro, download o compilador e os componentes.

Em seguida, adicione Swift ao seu caminho. No macOS, o local padrão da cadeia de ferramentas para download é /Library/Developer/Toolchains. Execute o seguinte comando no Terminal:

export PATH=/Library/Developer/Toolchains/swift-latest.xctoolchain/usr/bin:"${PATH}"

No Linux, você precisará instalar o clang:

$ sudo apt-get install clang

Se você instalou a cadeia de ferramentas do Swift em um diretório diferente da raiz do sistema, você precisará executar o seguinte comando, usando o caminho real da sua instalação do Swift:

$ export PATH=/path/to/Swift/usr/bin:"${PATH}"

Você pode verificar se tem a versão atual do Swift executando este comando:

$ swift --version

Seu primeiro programa no aplicativo Swift Playgrounds no iPad #

O aplicativo Swift Playgrounds é uma ótima maneira de começar a codificar Swift em movimento. Para usá-lo:

1- Baixe Swift Playgrounds para iPad na App Store.

2- Abra o aplicativo.

3- Na guia Meus Playgrounds, toque em + no canto superior esquerdo e selecione Modelo em branco.

4- Digite seu código.

5- Toque em Executar meu código para executar seu código.

6- Na frente de cada linha, o resultado será armazenado em um pequeno quadrado. Toque nele para revelar o resultado.

7- Para percorrer lentamente o código para rastreá-lo, toque no botão ao lado de Executar meu código.

Valor opcional e enumeração opcional #

Tipo Optionals, que trata da ausência de um valor. Os opcionais dizem “há um valor e é igual a x” ou “não há valor algum”.

Um Optional é um tipo próprio, na verdade, um dos novos enums superpoderosos do Swift. Ele tem dois valores possíveis, None e Some(T), onde T é um valor associado do tipo de dado correto disponível no Swift.

Vamos dar uma olhada neste pedaço de código por exemplo:

let x: String? = "Hello World"

if let y = x {
   print(y)
}

Na verdade, se você adicionar uma instrução print(x.dynamicType) no código acima, você verá isso no console:

Optional<String>

Corda? é, na verdade, açúcar sintático para Optional e Optional é um tipo por si só.

Aqui está uma versão simplificada do cabeçalho de Optional, que você pode ver clicando na palavra Optional em seu código do Xcode:

enum Optional<Wrapped> {

 /// The absence of a value.
 case none

 /// The presence of a value, stored as `Wrapped`.
 case some(Wrapped)
}

Opcional é na verdade uma enumeração, definida em relação a um tipo genérico Wrapped. Tem dois casos: .none para representar a ausência de um valor, e .some para representar a presença de um valor, que é armazenado como seu valor associado do tipo Wrapped.

Deixe-me passar por isso novamente: String? não é uma String, mas um Optional<String>. O fato de que Optional é um tipo significa que ele tem seus próprios métodos, por exemplo map e flatMap.
