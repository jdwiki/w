HMAC com MD5, SHA1, SHA224, SHA256, SHA384, SHA512 (Swift 3) #

Essas funções farão hash de entrada de String ou Data com um dos oito algoritmos de hash criptográfico.

O parâmetro name especifica o nome da função hash como uma String As funções suportadas são MD5, SHA1, SHA224, SHA256, SHA384 e SHA512

Este exemplo requer Criptografia Comum É necessário ter um cabeçalho de ponte para o projeto: #import <CommonCrypto/CommonCrypto.h> Adicione o Security.framework ao projeto.

Essas funções recebem um nome de hash, uma mensagem a ser hash, uma chave e retornam um resumo:

<pré> hashName: nome de uma função hash como String mensagem: mensagem como dados chave: chave como dados retorna: digerir como dados func hmac(hashName:String, message:Data, key:Data) -> Data? { let algos = [“SHA1”: (kCCHmacAlgSHA1, CC_SHA1_DIGEST_LENGTH), “MD5”: (kCCHmacAlgMD5, CC_MD5_DIGEST_LENGTH), “SHA224”: (kCCHmacAlgSHA224, CC_SHA224_DIGEST_LENGTH), “SHA256”: (kCCHmacAlgSHA256, CC_SHA256_DIGEST_LENGTH), “SHA384”: (kCCHmacAlgSHA384, CC_SHA384_DIGEST_LENGTH), “SHA512”: (kCCHmacAlgSHA512, CC_SHA512_DIGEST_LENGTH)] guard let (hashAlgorithm, length) = algos[hashName] else { return nil } var macData = Data(count: Int(length))

    macData.withUnsafeMutableBytes {macBytes in
        message.withUnsafeBytes {messageBytes in
            key.withUnsafeBytes {keyBytes in
                CCHmac(CCHmacAlgorithm(hashAlgorithm),
                       keyBytes,     key.count,
                       messageBytes, message.count,
                       macBytes)
            }
        }
    }
    return macData
}

<pré> hashName: nome de uma função hash como String mensagem: mensagem como String chave: chave como String retorna: digerir como dados 

func hmac(hashName:String, message:String, key:String) -> Data? {
    let messageData = message.data(using:.utf8)!
    let keyData = key.data(using:.utf8)!
    return hmac(hashName:hashName, message:messageData, key:keyData)
}

MD2, MD4, MD5, SHA1, SHA224, SHA256, SHA384, SHA512 (Swift 3) #

Essas funções farão hash de entrada de String ou Data com um dos oito algoritmos de hash criptográfico.

O parâmetro name especifica o nome da função hash como uma String As funções suportadas são MD2, MD4, MD5, SHA1, SHA224, SHA256, SHA384 e SHA512

Este exemplo requer Criptografia Comum É necessário ter um cabeçalho de ponte para o projeto: #import <CommonCrypto/CommonCrypto.h> Adicione o Security.framework ao projeto.

Esta função recebe um nome de hash e Data para ser hash e retorna um Data:

<pré> name: Um nome de uma função hash como uma String data: os dados a serem hash retorna: o resultado com hash como Data 

func hash(name:String, data:Data) -> Data? {
    let algos = ["MD2":    (CC_MD2,    CC_MD2_DIGEST_LENGTH),
                 "MD4":    (CC_MD4,    CC_MD4_DIGEST_LENGTH),
                 "MD5":    (CC_MD5,    CC_MD5_DIGEST_LENGTH),
                 "SHA1":   (CC_SHA1,   CC_SHA1_DIGEST_LENGTH),
                 "SHA224": (CC_SHA224, CC_SHA224_DIGEST_LENGTH),
                 "SHA256": (CC_SHA256, CC_SHA256_DIGEST_LENGTH),
                 "SHA384": (CC_SHA384, CC_SHA384_DIGEST_LENGTH),
                 "SHA512": (CC_SHA512, CC_SHA512_DIGEST_LENGTH)]
    guard let (hashAlgorithm, length) = algos[name]  else { return nil }
    var hashData = Data(count: Int(length))

    _ = hashData.withUnsafeMutableBytes {digestBytes in
        data.withUnsafeBytes {messageBytes in
            hashAlgorithm(messageBytes, CC_LONG(data.count), digestBytes)
        }
    }
    return hashData
}

Esta função recebe um nome de hash e uma String para ser hash e retorna um Data:

<pré> name: Um nome de uma função hash como uma String string: A String a ser hash retorna: o resultado com hash como Data func hash(name:String, string:String) -> Data? { let data = string.data(using:.utf8)! return hash(name:name, data:data) }

Exemplos:

let clearString = "clearData0123456"
let clearData   = clearString.data(using:.utf8)!
print("clearString: \(clearString)")
print("clearData: \(clearData as NSData)")

let hashSHA256 = hash(name:"SHA256", string:clearString)
print("hashSHA256: \(hashSHA256! as NSData)")

let hashMD5 = hash(name:"MD5", data:clearData)
print("hashMD5: \(hashMD5! as NSData)")
