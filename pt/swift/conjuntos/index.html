Declarando conjuntos #

Conjuntos são coleções não ordenadas de valores únicos. Os valores exclusivos devem ser do mesmo tipo.

var colors = Set<String>()

Você pode declarar um conjunto com valores usando a sintaxe literal de matriz.

var favoriteColors: Set<String> = ["Red", "Blue", "Green", "Blue"]
// {"Blue", "Green", "Red"}

Executando operações em conjuntos #

Valores comuns de ambos os conjuntos:

Você pode usar o método intersect(_:) para criar um novo conjunto contendo todos os valores comuns aos dois conjuntos.

let favoriteColors: Set = ["Red", "Blue", "Green"]
let newColors: Set = ["Purple", "Orange", "Green"]

let intersect = favoriteColors.intersect(newColors) // a AND b
// intersect = {"Green"}

Todos os valores de cada conjunto:

Você pode usar o método union(_:) para criar um novo conjunto contendo todos os valores exclusivos de cada conjunto.

let union = favoriteColors.union(newColors) // a OR b
// union = {"Red", "Purple", "Green", "Orange", "Blue"}

Observe como o valor “Verde” aparece apenas uma vez no novo conjunto.

Valores que não existem nos dois conjuntos:

Você pode usar o método exclusiveOr(_:) para criar um novo conjunto contendo os valores exclusivos de qualquer um, mas não de ambos.

let exclusiveOr = favoriteColors.exclusiveOr(newColors) // a XOR b
// exclusiveOr = {"Red", "Purple", "Orange", "Blue"}

Observe como o valor “Verde” não aparece no novo conjunto, pois estava nos dois conjuntos.

Valores que não estão em um conjunto:

Você pode usar o método subtract(_:) para criar um novo conjunto contendo valores que não estão em um conjunto específico.

let subtract = favoriteColors.subtract(newColors) // a - (a AND b)
// subtract = {"Blue", "Red"}

Observe como o valor “Verde” não aparece no novo conjunto, pois também estava no segundo conjunto.

##Conjunto Contado

Swift 3 apresenta a classe CountedSet (é a versão Swift da classe Objective-C NSCountedSet).

CountedSet, como sugerido pelo nome, registra quantas vezes um valor está presente.

let countedSet = CountedSet()
countedSet.add(1)
countedSet.add(1)
countedSet.add(1)
countedSet.add(2)

countedSet.count(for: 1) // 3
countedSet.count(for: 2) // 1

Modificando valores em um conjunto #

var favoriteColors: Set = ["Red", "Blue", "Green"]
//favoriteColors = {"Blue", "Green", "Red"}

Você pode usar o método insert(_:) para adicionar um novo item em um conjunto.

favoriteColors.insert("Orange")
//favoriteColors = {"Red", "Green", "Orange", "Blue"}

Você pode usar o método remove(_:) para remover um item de um conjunto. Retorna o valor contendo opcional que foi removido ou nil se o valor não estava no conjunto.

let removedColor = favoriteColors.remove("Red")
//favoriteColors = {"Green", "Orange", "Blue"}
// removedColor = Optional("Red")

let anotherRemovedColor = favoriteColors.remove("Black")
// anotherRemovedColor = nil

Verificando se um conjunto contém um valor #

var favoriteColors: Set = ["Red", "Blue", "Green"]
//favoriteColors = {"Blue", "Green", "Red"}

Você pode usar o método contains(_:) para verificar se um conjunto contém um valor. Ele retornará true se o conjunto contiver esse valor.

if favoriteColors.contains("Blue") {
    print("Who doesn't like blue!")
}
// Prints "Who doesn't like blue!"

Adicionando valores do meu próprio tipo a um Set #

Para definir um Set de seu próprio tipo, você precisa adequar seu tipo a Hashable

struct Starship: Hashable {
    let name: String
    var hashValue: Int { return name.hashValue }
}

func ==(left:Starship, right: Starship) -> Bool {
    return left.name == right.name
}

Agora você pode criar um Set de Starship(s)

let ships : Set<Starship> = [Starship(name:"Enterprise D"), Starship(name:"Voyager"), Starship(name:"Defiant") ]
