Sintaxe #

  • String.characters // Retorna um Array dos caracteres na String
  • String.characters.count // Retorna o número de caracteres
  • String.utf8 // Um ​​String.UTF8View, retorna os pontos de caractere UTF-8 na String
  • String.utf16 // Um ​​String.UTF16View, retorna os pontos de caractere UTF-16 na String
  • String.unicodeScalars // Um ​​String.UnicodeScalarView, retorna os pontos de caractere UTF-32 na String
  • String.isEmpty // Retorna true se a String não contiver nenhum texto
  • String.hasPrefix(String) // Retorna true se a String for prefixada com o argumento
  • String.hasSuffix(String) // Retorna true se a String for sufixada com o argumento
  • String.startIndex // Retorna o índice que corresponde ao primeiro caractere na string
  • String.endIndex // Retorna o índice que corresponde ao ponto após o último caractere na string
  • String.components(separatedBy: String) // Retorna um array contendo as substrings separadas pela string separadora fornecida
  • String.append(Character) // Adiciona o caractere (dado como argumento) à String

Uma String em Swift é uma coleção de caracteres e, por extensão, uma coleção de escalares Unicode. Como as Strings Swift são baseadas em Unicode, elas podem ser qualquer valor escalar Unicode, incluindo idiomas diferentes do inglês e emojis.

Como dois escalares podem se combinar para formar um único caractere, o número de escalares em uma String não é necessariamente sempre o mesmo que o número de caracteres.

Para obter mais informações sobre Strings, consulte The Swift Programming Language e a [String Structure Reference](https: //developer.apple.com/library/tvos/documentation/Swift/Reference/Swift_String_Structure/index.html).

Para detalhes de implementação, consulte “Swift String Design”

Literais de Strings e Caracteres #

String literais em Swift são delimitados com aspas duplas ("):

let greeting = "Hello!"  // greeting's type is String

Characters pode ser inicializado a partir de literais de string, desde que o literal contenha apenas um cluster de grafemas:

let chr: Character = "H" // valid
let chr2: Character = "😊" // valid
let chr3: Character = "abc" // invalid - multiple grapheme clusters

Interpolação de Strings #

Interpolação de strings permite injetando uma expressão diretamente em um literal de string. Isso pode ser feito com todos os tipos de valores, incluindo strings, inteiros, números de ponto flutuante e muito mais.

A sintaxe é uma barra invertida seguida por parênteses envolvendo o valor: \(valor). Qualquer expressão válida pode aparecer entre parênteses, incluindo chamadas de função.

let number = 5
let interpolatedNumber = "\(number)"  // string is "5"
let fortyTwo = "\(6 * 7)"             // string is "42"

let example = "This post has \(number) view\(number == 1 ? "" : "s")"
// It will output "This post has 5 views" for the above example.
// If the variable number had the value 1, it would output "This post has 1 view" instead.

Para tipos personalizados, o comportamento padrão da interpolação de string é "\(myobj)" é equivalente a String(myobj), a mesma representação usada por print(myobj). Você pode personalizar esse comportamento implementando o protocolo CustomStringConvertible para seu tipo.

Para Swift 3, de acordo com SE-0089, String.init <T>(_:) foi renomeado para String.init<T>(descrevendo:).

A interpolação de string "\(myobj)" preferirá o novo inicializador String.init<T: LosslessStringConvertible>(_:), mas retornará para init<T>(descrevendo:) se o valor for não LosslessStringConvertible.

Caracteres especiais #

Certos caracteres requerem uma sequência de escape especial para serem usados ​​em literais de string:

Personagem Significado
\0 o caractere nulo
\\ uma barra invertida simples, \
\t um caractere de tabulação
\v uma guia vertical
\r a [retorno de carro] (https://en.wikipedia.org/wiki/Newline)
\n um alimentação de linha (“nova linha”)
\" aspas duplas, "
\' uma aspa simples, '
\u{n} o ponto de código Unicode n (em hexadecimal)

Exemplo:

let message = "Then he said, \"I \u{1F496} you!\""

print(message) // Then he said, "I 💖 you!"

Concatenar strings #

Concatene strings com o operador + para produzir uma nova string:

let name = "John"
let surname = "Appleseed"
let fullName = name + " " + surname  // fullName is "John Appleseed"

Anexe a uma string mutable usando o += operador de atribuição composto, ou usando um método:

let str2 = "there"
var instruction = "look over"
instruction += " " + str2  // instruction is now "look over there"

var instruction = "look over"
instruction.append(" " + str2)  // instruction is now "look over there"

Anexe um único caractere a uma String mutável:

var greeting: String = "Hello"
let exclamationMark: Character = "!"
greeting.append(exclamationMark) 
// produces a modified String (greeting) = "Hello!"

Anexar vários caracteres a uma string mutável

var alphabet:String = "my ABCs: "
alphabet.append(contentsOf: (0x61...0x7A).map(UnicodeScalar.init)
                                         .map(Character.init) )
// produces a modified string (alphabet) = "my ABCs: abcdefghijklmnopqrstuvwxyz"

appendContentsOf(_:) foi renomeado para append(_:).

Junte uma sequência de strings para formar uma nova string usando joinWithSeparator(_:):

let words = ["apple", "orange", "banana"]
let str = words.joinWithSeparator(" & ")

print(str)   // "apple & orange & banana"

joinWithSeparator(_:) foi renomeado para joined(separator:).

O separador é a string vazia por padrão, então ["a", "b", "c"].joined() == "abc".

Codificação e decomposição de strings #

Um Swift String é feito de [Unicode](http://www.unicode.org/standard /standard.html) pontos de código. Ele pode ser decomposto e codificado de várias maneiras diferentes.

let str = "ที่👌①!"

Decompondo Strings #

Os caracteres de uma string são Unicode clusters de grafema estendidos:

Array(str.characters)  // ["ที่", "👌", "①", "!"]

Os unicodeScalars são os [pontos de código] Unicode (https://en.wikipedia.org/wiki/Code_point) que compõem uma string (observe que ที่ é um cluster de grafemas, mas 3 pontos de código — 3607, 3637 , 3656 — então o comprimento do array resultante não é o mesmo de characters):

str.unicodeScalars.map{ $0.value }  // [3607, 3637, 3656, 128076, 9312, 33]

Você pode codificar e decompor strings como [UTF-8] (https://en.wikipedia.org/wiki/UTF-8) (uma sequência de UInt8s) ou [UTF-16] (https://en .wikipedia.org/wiki/UTF-8). .wikipedia.org/wiki/UTF-16) (uma sequência de UInt16s):

Array(str.utf8)   // [224, 184, 151, 224, 184, 181, 224, 185, 136, 240, 159, 145, 140, 226, 145, 160, 33]
Array(str.utf16)  // [3607, 3637, 3656, 55357, 56396, 9312, 33]

Comprimento da String e Iteração #

Os characters, unicodeScalars, utf8 e utf16 de uma string são todos Collections, então você pode obter sua count e iterar sobre eles:

// NOTE: These operations are NOT necessarily fast/cheap! 

str.characters.count     // 4
str.unicodeScalars.count // 6
str.utf8.count           // 17
str.utf16.count          // 7

for c in str.characters { // ...
for u in str.unicodeScalars { // ...
for byte in str.utf8 { // ...
for byte in str.utf16 { // ...

Examinar e comparar strings #

Verifique se uma string está vazia:

if str.isEmpty {
    // do something if the string is empty
}

// If the string is empty, replace it with a fallback:
let result = str.isEmpty ? "fallback string" : str

Verifique se duas strings são iguais (no sentido de equivalência canônica Unicode):

"abc" == "def"          // false
"abc" == "ABC"          // false
"abc" == "abc"          // true

// "LATIN SMALL LETTER A WITH ACUTE" == "LATIN SMALL LETTER A" + "COMBINING ACUTE ACCENT"
"\u{e1}" == "a\u{301}"  // true

Verifique se uma string começa/termina com outra string:

"fortitude".hasPrefix("fort")      // true
"Swift Language".hasSuffix("age")  // true

Invertendo Strings #

let aString = "This is a test string."

// first, reverse the String's characters 
let reversedCharacters = aString.characters.reverse()

// then convert back to a String with the String() initializer
let reversedString = String(reversedCharacters)

print(reversedString) // ".gnirts tset a si sihT"

let reversedCharacters = aString.characters.reversed()
let reversedString = String(reversedCharacters)

Verifique se String contém caracteres de um conjunto definido #

Cartas

let letters = CharacterSet.letters

let phrase = "Test case"
let range = phrase.rangeOfCharacter(from: letters)

// range will be nil if no letters is found
if let test = range {
   print("letters found")
}
else {
   print("letters not found")
}

let letters = NSCharacterSet.letterCharacterSet()

let phrase = "Test case"
let range = phrase.rangeOfCharacterFromSet(letters)

// range will be nil if no letters is found
if let test = range {
   print("letters found")
}
else {
  print("letters not found")
}

A nova estrutura CharacterSet que também é conectada à classe Objective-C NSCharacterSet define vários conjuntos predefinidos como:

  • Dígitos Decimais
  • Letras maiúsculas
  • alfanumérico
  • controlCharacters
  • caracteres ilegais
  • e mais você pode encontrar na referência NSCharacterSet.

Você também pode definir seu próprio conjunto de caracteres:

let phrase = "Test case"
let charset = CharacterSet(charactersIn: "t")

if let _ = phrase.rangeOfCharacter(from: charset, options: .caseInsensitive) {
   print("yes") 
}
else {
   print("no")
}

let charset = NSCharacterSet(charactersInString: "t")

if let _ = phrase.rangeOfCharacterFromSet(charset, options: .CaseInsensitiveSearch, range: nil) {
   print("yes")
}
else {
    print("no")
}

Você também pode incluir intervalo:

let phrase = "Test case"
let charset = CharacterSet(charactersIn: "t")

if let _ = phrase.rangeOfCharacter(from: charset, options: .caseInsensitive, range: phrase.startIndex..<phrase.endIndex)) {
   print("yes") 
}
else {
   print("no")
}

Iteração de String #

let string = "My fantastic string"
var index = string.startIndex

while index != string.endIndex {
    print(string[index])
    index = index.successor()
}

Nota: endIndex vem após o final da string (ou seja, string[string.endIndex] é um erro, mas string[string.startIndex] está correto). Além disso, em uma string vazia (""), string.startIndex == string.endIndex é true. Certifique-se de verificar se há strings vazias, já que você não pode chamar startIndex.successor() em uma string vazia.

No Swift 3, os índices String não têm mais successor(), predecessor(), advancedBy(_:), advancedBy(_:limit:) ou distanceTo(_:).

Em vez disso, essas operações são movidas para a coleção, que agora é responsável por incrementar e decrementar seus índices.

Os métodos disponíveis são .index(after:), .index(before:) e .index(_:, offsetBy:).

let string = "My fantastic string"
var currentIndex = string.startIndex

while currentIndex != string.endIndex {
    print(string[currentIndex])
    currentIndex = string.index(after: currentIndex)
}

Observação: estamos usando currentIndex como um nome de variável para evitar confusão com o método .index.

E, por exemplo, se você quiser ir para o outro lado:

var index:String.Index? = string.endIndex.predecessor()

while index != nil {
    print(string[index!])
    if index != string.startIndex {
        index = index.predecessor()
    }
    else {
        index = nil
    }
}

(Ou você pode simplesmente inverter a string primeiro, mas se você não precisar percorrer todo o caminho, provavelmente preferiria um método como este)

var currentIndex: String.Index? = string.index(before: string.endIndex)

while currentIndex != nil {
    print(string[currentIndex!])
    if currentIndex != string.startIndex {
        currentIndex = string.index(before: currentIndex!)
    }
    else {
        currentIndex = nil
    }
}

Observe que Index é um tipo de objeto, e não um Int. Você não pode acessar um caractere de string da seguinte forma:

let string = "My string"
string[2] // can't do this
string.characters[2] // and also can't do this

Mas você pode obter um índice específico da seguinte forma:

index = string.startIndex.advanceBy(2)

currentIndex = string.index(string.startIndex, offsetBy: 2)

E pode retroceder assim:

index = string.endIndex.advancedBy(-2)

currentIndex = string.index(string.endIndex, offsetBy: -2)

Se você pode exceder os limites da string ou deseja especificar um limite, pode usar:

index = string.startIndex.advanceBy(20, limit: string.endIndex)

currentIndex = string.index(string.startIndex, offsetBy: 20, limitedBy: string.endIndex)

Alternativamente, pode-se apenas iterar pelos caracteres em uma string, mas isso pode ser menos útil dependendo do contexto:

for c in string.characters {
    print(c)
}

Unicode #

Configurando valores #

Usando Unicode diretamente

var str: String = "I want to visit 北京, Москва, मुंबई, القاهرة, and 서울시. 😊"
var character: Character = "🌍"

Usando valores hexadecimais

var str: String = "\u{61}\u{5927}\u{1F34E}\u{3C0}" // a大🍎π
var character: Character = "\u{65}\u{301}" // é = "e" + accent mark

Observe que o Character do Swift pode ser composto de vários pontos de código Unicode, mas parece ser um único caractere. Isso é chamado de cluster de grafema estendido.

Conversões #

String –> Hex

// Accesses views of different Unicode encodings of `str`
str.utf8
str.utf16
str.unicodeScalars // UTF-32

Hex –> Cadeia

let value0: UInt8 = 0x61
let value1: UInt16 = 0x5927
let value2: UInt32 = 0x1F34E

let string0 = String(UnicodeScalar(value0)) // a
let string1 = String(UnicodeScalar(value1)) // 大
let string2 = String(UnicodeScalar(value2)) // 🍎

// convert hex array to String
let myHexArray = [0x43, 0x61, 0x74, 0x203C, 0x1F431] // an Int array
var myString = ""
for hexValue in myHexArray {
    myString.append(UnicodeScalar(hexValue))
}
print(myString) // Cat‼🐱

Observe que para UTF-8 e UTF-16 a conversão nem sempre é tão fácil porque coisas como emoji não podem ser codificadas com um único valor UTF-16. É preciso um par substituto.

Dividindo uma String em um Array #

No Swift, você pode facilmente separar uma String em um array cortando-a em um determinado caractere:

let startDate = "23:51"

let startDateAsArray = startDate.components(separatedBy: ":") // ["23", "51"]`

let startDate = "23:51"

let startArray = startDate.componentsSeparatedByString(":") // ["23", "51"]`

Ou quando o separador não estiver presente:

let myText = "MyText"

let myTextArray = myText.components(separatedBy: " ") // myTextArray is ["MyText"]

let myText = "MyText"

let myTextArray = myText.componentsSeparatedByString(" ") // myTextArray is ["MyText"]

Strings Maiúsculas e Minúsculas #

Para tornar todos os caracteres em uma String maiúsculas ou minúsculas:

let text = "AaBbCc"
let uppercase = text.uppercaseString // "AABBCC"
let lowercase = text.lowercaseString // "aabbcc"

let text = "AaBbCc"
let uppercase = text.uppercased() // "AABBCC"
let lowercase = text.lowercased() // "aabbcc"

Formatando Strings #

Zeros à esquerda #

let number: Int = 7
let str1 = String(format: "%03d", number) // 007
let str2 = String(format: "%05d", number) // 00007

Números após Decimal #

let number: Float = 3.14159
let str1 = String(format: "%.2f", number) // 3.14
let str2 = String(format: "%.4f", number) // 3.1416 (rounded)

Decimal para Hexadecimal #

let number: Int = 13627
let str1 = String(format: "%2X", number) // 353B
let str2 = String(format: "%2x", number) // 353b (notice the lowercase b)

Alternativamente, pode-se usar inicializador especializado que faz o mesmo:

let number: Int = 13627
let str1 = String(number, radix: 16, uppercase: true) //353B
let str2 = String(number, radix: 16) // 353b

Decimal para um número com base arbitrária #

let number: Int = 13627
let str1 = String(number, radix: 36) // aij

Radix é Int em [2, 36].

Convertendo uma string Swift para um tipo de número #

Int("123") // Returns 123 of Int type
Int("abcd") // Returns nil
Int("10") // Returns 10 of Int type
Int("10", radix: 2) // Returns 2 of Int type
Double("1.5") // Returns 1.5 of Double type
Double("abcd") // Returns nil

Observe que isso retorna um valor Optional, que deve ser [unwrapped](https://www.wikiod.com/pt/docs/ /swift/247/optionals/913/unwrapping-an-optional#t=201607230855233123287) antes de ser usado.

Converter String para e de Data / NSData #

Para converter String de e para Data / NSData, precisamos codificar essa string com uma codificação específica. O mais famoso é o UTF-8 que é uma representação de 8 bits de caracteres Unicode, adequada para transmissão ou armazenamento por sistemas baseados em ASCII. Aqui está uma lista de todas as String Encodings

String para Data/NSData

let data = string.data(using: .utf8)

let data = string.dataUsingEncoding(NSUTF8StringEncoding)

Data/NSData para String

let string = String(data: data, encoding: .utf8)

let string = String(data: data, encoding: NSUTF8StringEncoding)

Contar ocorrências de um caractere em uma string #

Dado um String e um Character

let text = "Hello World"
let char: Character = "o"

Podemos contar o número de vezes que o Character aparece na String usando

let sensitiveCount = text.characters.filter { $0 == char }.count // case-sensitive
let insensitiveCount = text.lowercaseString.characters.filter { $0 == Character(String(char).lowercaseString) } // case-insensitive

Remova WhiteSpace e NewLine à esquerda e à direita #

let someString = "  Swift Language  \n"
let trimmedString = someString.stringByTrimmingCharactersInSet(NSCharacterSet.whitespaceAndNewlineCharacterSet())
// "Swift Language"

O método stringByTrimmingCharactersInSet retorna uma nova string feita removendo de ambas as extremidades os caracteres String contidos em um determinado conjunto de caracteres.

Também podemos remover apenas espaços em branco ou nova linha.

Removendo apenas espaços em branco:

let trimmedWhiteSpace = someString.stringByTrimmingCharactersInSet(NSCharacterSet.whitespaceCharacterSet())
// "Swift Language  \n"

Removendo apenas a nova linha:

let trimmedNewLine = someString.stringByTrimmingCharactersInSet(NSCharacterSet.newlineCharacterSet())
// "  Swift Language  "

let someString = "  Swift Language  \n"

let trimmedString = someString.trimmingCharacters(in: .whitespacesAndNewlines)
// "Swift Language"

let trimmedWhiteSpace = someString.trimmingCharacters(in: .whitespaces)
// "Swift Language  \n"

let trimmedNewLine = someString.trimmingCharacters(in: .newlines)
// "  Swift Language  "

Nota: todos esses métodos pertencem à Foundation. Use import Foundation se o Foundation ainda não tiver sido importado por meio de outras bibliotecas como Cocoa ou UIKit.
