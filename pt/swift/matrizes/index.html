Array é um tipo de coleção ordenada de acesso aleatório.

As matrizes são um dos tipos de dados mais usados ​​em um aplicativo. Usamos o tipo Array para armazenar elementos de um único tipo, o tipo Element do array. Um array pode armazenar qualquer tipo de elemento — de inteiros a strings e classes.

Sintaxe #

  • Matriz<Elemento> // O tipo de um array com elementos do tipo Element
  • [Element] // Açúcar sintático para o tipo de um array com elementos do tipo Element
  • [element0, element1, element2, … elementN] // Um ​​literal de array
  • [Element]() // Cria um novo array vazio do tipo [Element]
  • Array(count:repeatedValue:) // Cria um array de elementos count, cada um inicializado para repeatedValue
  • Array(_:) // Cria um array a partir de uma sequência arbitrária

Arrays são uma coleção ordenada de valores. Os valores podem ser repetidos, mas devem ser do mesmo tipo.

Noções básicas de matrizes #

Array é um tipo de coleção ordenada na biblioteca padrão do Swift. Ele fornece acesso aleatório O(1) e realocação dinâmica. Array é um tipo genérico, então o tipo de valores que ele contém são conhecidos em tempo de compilação.

Como Array é um tipo de valor, sua mutabilidade é definida pelo fato de ser anotado como var (mutável) ou let (imutável).

O tipo [Int] (significando: um array contendo Ints) é syntactic sugar para Array<T>.

Leia mais sobre arrays em [The Swift Programming Language](https://developer.apple.com/library/ios/documentation/Swift/Conceptual/Swift_Programming_Language/CollectionTypes.html#//apple_ref/doc/uid/TP40014097-CH8- ID107).

Matrizes vazias #

As três declarações a seguir são equivalentes:

// A mutable array of Strings, initially empty.

var arrayOfStrings: [String] = []      // type annotation + array literal
var arrayOfStrings = [String]()        // invoking the [String] initializer
var arrayOfStrings = Array<String>()   // without syntactic sugar

Literais de matriz #

Um literal de matriz é escrito com colchetes em torno de elementos separados por vírgula:

// Create an immutable array of type [Int] containing 2, 4, and 7
let arrayOfInts = [2, 4, 7]

O compilador geralmente pode inferir o tipo de uma matriz com base nos elementos do literal, mas anotações de tipo explícitas podem substituir o padrão:

let arrayOfUInt8s: [UInt8] = [2, 4, 7]   // type annotation on the variable
let arrayOfUInt8s = [2, 4, 7] as [UInt8] // type annotation on the initializer expression
let arrayOfUInt8s = [2 as UInt8, 4, 7]   // explicit for one element, inferred for the others

Arrays com valores repetidos #

// An immutable array of type [String], containing ["Example", "Example", "Example"]
let arrayOfStrings = Array(repeating: "Example",count: 3)

Criando arrays de outras sequências #

let dictionary = ["foo" : 4, "bar" : 6]

// An immutable array of type [(String, Int)], containing [("bar", 6), ("foo", 4)]
let arrayOfKeyValuePairs = Array(dictionary)

Matrizes multidimensionais #

Em Swift, um array multidimensional é criado aninhando arrays: um array bidimensional de Int é [[Int]] (ou Array<Array<Int>>).

let array2x3 = [
    [1, 2, 3],
    [4, 5, 6]
]
// array2x3[0][1] is 2, and array2x3[1][2] is 6.

Para criar uma matriz multidimensional de valores repetidos, use chamadas aninhadas do inicializador de matriz:

var array3x4x5 = Array(repeating: Array(repeating: Array(repeating: 0,count: 5),count: 4),count: 3)

Extraindo valores de um determinado tipo de um Array com flatMap(_:) #

O Array things contém valores do tipo Any.

let things: [Any] = [1, "Hello", 2, true, false, "World", 3]

Podemos extrair valores de um determinado tipo e criar um novo Array desse tipo específico. Digamos que queremos extrair todos os Int(s) e colocá-los em um array Int de maneira segura.

let numbers = things.flatMap { $0 as? Int }

Agora numbers é definido como [Int]. A função flatMap descarta todos os elementos nil e o resultado contém apenas os seguintes valores:

[1, 2, 3]

Achatando o resultado de uma transformação Array com flatMap(_:) #

Além de ser capaz de criar um array filtrando nil dos elementos transformados de uma sequência, há também uma versão de flatMap(_:) que espera a transformação closure para retornar uma sequência S.

extension SequenceType {
    public func flatMap<S : SequenceType>(transform: (Self.Generator.Element) throws -> S) rethrows -> [S.Generator.Element]
}

Cada sequência da transformação será concatenada, resultando em um array contendo os elementos combinados de cada sequência – [S.Generator.Element].

Combinando os caracteres em um array de strings #

Por exemplo, podemos usá-lo para pegar um array de strings primos e combinar seus caracteres em um único array:

let primes = ["2", "3", "5", "7", "11", "13", "17", "19"]
let allCharacters = primes.flatMap { $0.characters }
// => "["2", "3", "5", "7", "1", "1", "1", "3", "1", "7", "1", "9"]"

Decompondo o exemplo acima:

  1. primes é um [String] (Como um array é uma sequência, podemos chamar flatMap(_:) nele).
  2. O fechamento da transformação inclui um dos elementos de primes, uma String (Array<String>.Generator.Element).
  3. A closure então retorna uma seqüência do tipo String.CharacterView.
  4. O resultado é então um array contendo os elementos combinados de todas as seqüências de cada uma das chamadas de fechamento de transformação – [String.CharacterView.Generator.Element].

Achatando um array multidimensional #

Como flatMap(_:) irá concatenar as sequências retornadas das chamadas de fechamento de transformação, ele pode ser usado para achatar um array multidimensional – como um array 2D em um array 1D, um array 3D em um array 2D etc.

Isso pode ser feito simplesmente retornando o dado elemento $0 (um array aninhado) na closure:

// A 2D array of type [[Int]]
let array2D = [[1, 3], [4], [6, 8, 10], [11]]

// A 1D array of type [Int]
let flattenedArray = array2D.flatMap { $0 }

print(flattenedArray) // [1, 3, 4, 6, 8, 10, 11]

Combinando os elementos de um Array com reduce(_:combine:) #

reduce(_:combine:) pode ser usado para combinar os elementos de uma sequência em um único valor. É necessário um valor inicial para o resultado, bem como um closure para aplicar a cada elemento – que retornará o novo valor acumulado.

Por exemplo, podemos usá-lo para somar uma matriz de números:

let numbers = [2, 5, 7, 8, 10, 4]

let sum = numbers.reduce(0) {accumulator, element in
    return accumulator + element
}

print(sum) // 36

Estamos passando 0 para o valor inicial, pois esse é o valor inicial lógico para uma soma. Se passássemos um valor de N, a soma resultante seria N + 36. A closure passada para reduce tem dois argumentos. acumulador é o valor acumulado atual, ao qual é atribuído o valor que o fechamento retorna em cada iteração. element é o elemento atual na iteração.

Como neste exemplo, estamos passando um fechamento (Int, Int) -> Int para reduce, que é simplesmente a saída da adição das duas entradas - podemos realmente passar o operador + diretamente, como operadores são funções em Swift:

let sum = numbers.reduce(0, combine: +)

Filtrando nil de uma transformação Array com flatMap(_:) #

Você pode usar flatMap(_:) de maneira semelhante a map(_:) para criar um array aplicando uma transformação aos elementos de uma sequência.

extension SequenceType {
    public func flatMap<T>(@noescape transform: (Self.Generator.Element) throws -> T?) rethrows -> [T]
}

A diferença com esta versão do flatMap(_:) é que ela espera que a transformação closure retorne um valor Optional T? para cada um dos elementos. Ele então desempacotará com segurança cada um desses valores opcionais, filtrando nil – resultando em uma matriz de [T].

Por exemplo, você pode fazer isso para transformar um [String] em um [Int] usando o inicializador String falível do [Int] 5, filtrando quaisquer elementos que não possam ser convertidos:

let strings = ["1", "foo", "3", "4", "bar", "6"]

let numbersThatCanBeConverted = strings.flatMap { Int($0) }

print(numbersThatCanBeConverted) // [1, 3, 4, 6]

Você também pode usar a habilidade do flatMap(_:) para filtrar nil para simplesmente converter um array de opcionais em um array de não-opcionais:

let optionalNumbers : [Int?] = [nil, 1, nil, 2, nil, 3]

let numbers = optionalNumbers.flatMap { $0 }

print(numbers) // [1, 2, 3]

Achatando preguiçosamente um Array multidimensional com flatten() #

Podemos usar flatten() para preguiçosamente reduzir o aninhamento de uma sequência multidimensional.

Por exemplo, achatamento preguiçoso de um array 2D em um array 1D:

// A 2D array of type [[Int]]
let array2D = [[1, 3], [4], [6, 8, 10], [11]]

// A FlattenBidirectionalCollection<[[Int]]>
let lazilyFlattenedArray = array2D.flatten()

print(lazilyFlattenedArray.contains(4)) // true

No exemplo acima, flatten() retornará uma FlattenBidirectionalCollection, que aplicará lentamente o achatamento do array. Portanto, contains(_:) exigirá apenas que os dois primeiros arrays aninhados de array2D sejam achatados - pois causará um curto-circuito ao encontrar o elemento desejado.

Subscrevendo um array com um intervalo #

Pode-se extrair uma série de elementos consecutivos de um Array usando um Range.

let words = ["Hey", "Hello", "Bonjour", "Welcome", "Hi", "Hola"]
let range = 2...4
let slice = words[range] // ["Bonjour", "Welcome", "Hi"]

Subscrever um Array com um Range retorna um ArraySlice. É uma subsequência do Array.

Em nosso exemplo, temos um Array of Strings, então retornamos ArraySlice<String>.

Embora um ArraySlice esteja em conformidade com CollectionType e possa ser usado com sort, filter, etc, seu propósito não é para armazenamento de longo prazo, mas para cálculos transitórios: ele deve ser convertido novamente em um Array assim que você terminei de trabalhar com ele.

Para isso, use o inicializador Array():

let result = Array(slice)

Para resumir em um exemplo simples sem etapas intermediárias:

let words = ["Hey", "Hello", "Bonjour", "Welcome", "Hi", "Hola"]
let selectedWords = Array(words[2...4]) // ["Bonjour", "Welcome", "Hi"]

Removendo elemento de um array sem saber seu índice #

Geralmente, se queremos remover um elemento de um array, precisamos saber seu índice para que possamos removê-lo facilmente usando a função remove(at:).

Mas e se não soubermos o índice, mas soubermos o valor do elemento a ser removido!

Então aqui está a extensão simples para um array que nos permitirá remover um elemento do array facilmente sem saber seu índice:

Rápido3 #

extension Array where Element: Equatable {

    mutating func remove(_ element: Element) {
        _ = index(of: element).flatMap {
            self.remove(at: $0)
        }
    }
}

por exemplo.

    var array = ["abc", "lmn", "pqr", "stu", "xyz"]
    array.remove("lmn")
    print("\(array)")    //["abc", "pqr", "stu", "xyz"]
    
    array.remove("nonexistent")
    print("\(array)")    //["abc", "pqr", "stu", "xyz"]
    //if provided element value is not present, then it will do nothing!

Além disso, se, por engano, fizemos algo assim: array.remove(25) ou seja, fornecemos valor com tipo de dados diferente, o compilador lançará um erro mencionando- não é possível converter o valor para o tipo de argumento esperado

Filtrando uma matriz #

Você pode usar o método filter(_:) em SequenceType para criar um novo array contendo os elementos da sequência que satisfazem um determinado predicado, que pode ser fornecido como a fechamento.

Por exemplo, filtrando números pares de um [Int]:

let numbers = [22, 41, 23, 30]

let evenNumbers = numbers.filter { $0 % 2 == 0 }

print(evenNumbers)  // [22, 30]

Filtrando uma [Person], com idade inferior a 30:

struct Person {
    var age : Int
}

let people = [Person(age: 22), Person(age: 41), Person(age: 23), Person(age: 30)]

let peopleYoungerThan30 = people.filter { $0.age < 30 }

print(peopleYoungerThan30) // [Person(age: 22), Person(age: 23)]

Classificando um Array de Strings #

A maneira mais simples é usar sorted():

let words = ["Hello", "Bonjour", "Salute", "Ahola"]
let sortedWords = words.sorted()
print(sortedWords) // ["Ahola", "Bonjour", "Hello", "Salute"]

ou sort()

var mutableWords = ["Hello", "Bonjour", "Salute", "Ahola"]
mutableWords.sort()
print(mutableWords) // ["Ahola", "Bonjour", "Hello", "Salute"]

Você pode passar um encerramento como um argumento para classificação:

let words = ["Hello", "Bonjour", "Salute", "Ahola"]
let sortedWords = words.sorted(isOrderedBefore: { $0 > $1 })
print(sortedWords) // ["Salute", "Hello", "Bonjour", "Ahola"]

Sintaxe alternativa com um encerramento à direita:

let words = ["Hello", "Bonjour", "Salute", "Ahola"]
let sortedWords = words.sorted() { $0 > $1 }
print(sortedWords) // ["Salute", "Hello", "Bonjour", "Ahola"]

Mas haverá resultados inesperados se os elementos na matriz não forem consistentes:

let words = ["Hello", "bonjour", "Salute", "ahola"]
let unexpected = words.sorted()
print(unexpected) // ["Hello", "Salute", "ahola", "bonjour"]

Para resolver esse problema, classifique em uma versão minúscula dos elementos:

let words = ["Hello", "bonjour", "Salute", "ahola"]
let sortedWords = words.sorted { $0.lowercased() < $1.lowercased() }
print(sortedWords) // ["ahola", "bonjour", "Hello", "Salute"]

Ou import Foundation e use os métodos de comparação do NSString como caseInsensitiveCompare:

let words = ["Hello", "bonjour", "Salute", "ahola"]
let sortedWords = words.sorted { $0.caseInsensitiveCompare($1) == .orderedAscending }
print(sortedWords) // ["ahola", "bonjour", "Hello", "Salute"]

Como alternativa, use localizedCaseInsensitiveCompare, que pode gerenciar sinais diacríticos.

Para classificar corretamente as Strings pelo valor numérico que elas contêm, use compare com a opção .numeric:

let files = ["File-42.txt", "File-01.txt", "File-5.txt", "File-007.txt", "File-10.txt"]
let sortedFiles = files.sorted() { $0.compare($1, options: .numeric) == .orderedAscending }
print(sortedFiles) // ["File-01.txt", "File-5.txt", "File-007.txt", "File-10.txt", "File-42.txt"]

Acessando índices com segurança #

Adicionando a seguinte extensão aos índices de matriz pode ser acessado sem saber se o índice está dentro dos limites.

extension Array {
    subscript (safe index: Int) -> Element? {
        return indices ~= index ? self[index] : nil
    }
}

exemplo:

if let thirdValue = array[safe: 2] {
    print(thirdValue)
}

Determinar se uma matriz está vazia ou retornar seu tamanho

var exampleArray = [1,2,3,4,5]
exampleArray.isEmpty //false
exampleArray.count //5

Inverter uma matriz Observação: o resultado não é executado na matriz em que o método é chamado e deve ser colocado em sua própria variável.

exampleArray = exampleArray.reverse()
//exampleArray = [9, 8, 7, 6, 5, 3, 2]

Classificando uma matriz #

var array = [3, 2, 1]

Criando um novo array ordenado #

Como Array está em conformidade com SequenceType, podemos gerar um novo array dos elementos ordenados usando um método de ordenação embutido.

No Swift 2, isso é feito com o método sort().

let sorted = array.sort()  // [1, 2, 3]

A partir do Swift 3, ele foi renomeado para sorted().

let sorted = array.sorted()  // [1, 2, 3]

Classificando um array existente no local #

Como Array está em conformidade com MutableCollectionType, podemos classificar seus elementos no lugar.

No Swift 2, isso é feito usando o método sortInPlace().

array.sortInPlace() // [1, 2, 3]

A partir do Swift 3, foi renomeado para sort().

array.sort() // [1, 2, 3]

Nota: Para usar os métodos acima, os elementos devem estar em conformidade com o protocolo Comparable.

Classificando um array com uma ordenação personalizada #

Você também pode classificar um array usando um closure para definir se um elemento deve ser ordenado antes do outro – o que não é restrito a arrays onde os elementos devem ser Comparable. Por exemplo, não faz sentido que um Landmark seja Comparable – mas você ainda pode classificar uma matriz de pontos de referência por altura ou nome.

struct Landmark {
    let name : String
    let metersTall : Int
}

var landmarks = [Landmark(name: "Empire State Building", metersTall: 443),
                 Landmark(name: "Eifell Tower", metersTall: 300),
                 Landmark(name: "The Shard", metersTall: 310)]

// sort landmarks by height (ascending)
landmarks.sortInPlace {$0.metersTall < $1.metersTall}

print(landmarks) // [Landmark(name: "Eifell Tower", metersTall: 300), Landmark(name: "The Shard", metersTall: 310), Landmark(name: "Empire State Building", metersTall: 443)]

// create new array of landmarks sorted by name
let alphabeticalLandmarks = landmarks.sort {$0.name < $1.name}

print(alphabeticalLandmarks) // [Landmark(name: "Eifell Tower", metersTall: 300), Landmark(name: "Empire State Building", metersTall: 443), Landmark(name: "The Shard", metersTall: 310)]

// sort landmarks by height (ascending)
landmarks.sort {$0.metersTall < $1.metersTall}

// create new array of landmarks sorted by name
let alphabeticalLandmarks = landmarks.sorted {$0.name < $1.name}

Nota: A comparação de strings pode gerar resultados inesperados se as strings forem inconsistentes, veja [Classificando uma matriz de strings][10].

[10]: https://www.wikiod.com/pt/swift/matrizes#Classificando um array de strings

Transformando os elementos de um Array com map(_:) #

Como Array está em conformidade com SequenceType, podemos usar map(_:) para transformar um array de A em um array de B usando um fechamento do tipo (A) lança -> B.

Por exemplo, poderíamos usá-lo para transformar um array de Ints em um array de Strings assim:

let numbers = [1, 2, 3, 4, 5]
let words = numbers.map { String($0) }
print(words) // ["1", "2", "3", "4", "5"]

map(_:) irá iterar através do array, aplicando o encerramento dado a cada elemento. O resultado desse encerramento será usado para preencher um novo array com os elementos transformados.

Como String tem um inicializador que recebe um Int, também podemos usar esta sintaxe mais clara:

let words = numbers.map(String.init)

Uma transformação map(_:) não precisa alterar o tipo do array – por exemplo, ela também pode ser usada para multiplicar um array de Ints por dois:

let numbers = [1, 2, 3, 4, 5]
let numbersTimes2 = numbers.map {$0 * 2}
print(numbersTimes2) // [2, 4, 6, 8, 10]

Encontrando o elemento mínimo ou máximo de um Array #

Você pode usar os métodos minElement() e maxElement() para encontrar o elemento mínimo ou máximo em uma determinada sequência. Por exemplo, com uma matriz de números:

let numbers = [2, 6, 1, 25, 13, 7, 9]

let minimumNumber = numbers.minElement() // Optional(1)
let maximumNumber = numbers.maxElement() // Optional(25)

A partir do Swift 3, os métodos foram renomeados para min() e max() respectivamente:

let minimumNumber = numbers.min() // Optional(1)
let maximumNumber = numbers.max() // Optional(25)

Os valores retornados desses métodos são Opcional para refletir o fato de que o array pode estar vazio – se estiver, nil será retornado.

Nota: Os métodos acima requerem que os elementos estejam em conformidade com o protocolo Comparable.

Encontrando o elemento mínimo ou máximo com um pedido personalizado #

Você também pode usar os métodos acima com um closure personalizado, definindo se um elemento deve ser ordenado antes do outro, permitindo que você encontre o elemento mínimo ou máximo em uma matriz onde os elementos não são necessariamente Comparable.

Por exemplo, com uma matriz de vetores:

struct Vector2 {
    let dx : Double
    let dy : Double
    
    var magnitude : Double {return sqrt(dx*dx+dy*dy)}
}

let vectors = [Vector2(dx: 3, dy: 2), Vector2(dx: 1, dy: 1), Vector2(dx: 2, dy: 2)]

// Vector2(dx: 1.0, dy: 1.0)
let lowestMagnitudeVec2 = vectors.minElement { $0.magnitude < $1.magnitude } 

// Vector2(dx: 3.0, dy: 2.0)
let highestMagnitudeVec2 = vectors.maxElement { $0.magnitude < $1.magnitude } 

let lowestMagnitudeVec2 = vectors.min { $0.magnitude < $1.magnitude }
let highestMagnitudeVec2 = vectors.max { $0.magnitude < $1.magnitude }

Semântica de valor #

Copiar um array copiará todos os itens dentro do array original.

Alterar o novo array não mudará o array original.

var originalArray = ["Swift", "is", "great!"]
var newArray = originalArray
newArray[2] = "awesome!"
//originalArray = ["Swift", "is", "great!"]
//newArray = ["Swift", "is", "awesome!"]

As matrizes copiadas compartilharão o mesmo espaço na memória que as originais até serem alteradas. Como resultado disso, há um impacto no desempenho quando a matriz copiada recebe seu próprio espaço na memória quando é alterada pela primeira vez.

Acessando valores de matriz #

Os exemplos a seguir usarão essa matriz para demonstrar o acesso aos valores

var exampleArray:[Int] = [1,2,3,4,5]
//exampleArray = [1, 2, 3, 4, 5]

Para acessar um valor em um índice conhecido, use a seguinte sintaxe:

let exampleOne = exampleArray[2]
//exampleOne = 3

Observação: o valor em índice dois é o terceiro valor no Array. Arrays usam um índice baseado em zero, o que significa que o primeiro elemento no Array está no índice 0.

let value0 = exampleArray[0]
let value1 = exampleArray[1]
let value2 = exampleArray[2]
let value3 = exampleArray[3]
let value4 = exampleArray[4]
//value0 = 1
//value1 = 2 
//value2 = 3
//value3 = 4
//value4 = 5

Acesse um subconjunto de um Array usando o filtro:

var filteredArray = exampleArray.filter({ $0 < 4 })
//filteredArray = [1, 2, 3]

Os filtros podem ter condições complexas, como filtrar apenas números pares:

var evenArray = exampleArray.filter({ $0 % 2 == 0 })
//evenArray = [2, 4]

Também é possível retornar o índice de um determinado valor, retornando nil caso o valor não tenha sido encontrado.

exampleArray.indexOf(3) // Optional(2)

Existem métodos para o primeiro, último, valor máximo ou mínimo em um Array. Esses métodos retornarão nil se o Array estiver vazio.

exampleArray.first // Optional(1)
exampleArray.last // Optional(5)
exampleArray.maxElement() // Optional(5)
exampleArray.minElement() // Optional(1)

Agrupando valores de matriz #

Se tivermos uma estrutura como esta

struct Box {
    let name: String
    let thingsInside: Int
}

e uma matriz de Box(es)

let boxes = [
    Box(name: "Box 0", thingsInside: 1),
    Box(name: "Box 1", thingsInside: 2),
    Box(name: "Box 2", thingsInside: 3),
    Box(name: "Box 3", thingsInside: 1),
    Box(name: "Box 4", thingsInside: 2),
    Box(name: "Box 5", thingsInside: 3),
    Box(name: "Box 6", thingsInside: 1)
]

podemos agrupar as caixas pela propriedade thingsInside para obter um Dicionário onde a chave é o número de coisas e o valor é um array de caixas.

let grouped = boxes.reduce([Int:[Box]]()) { (res, box) -> [Int:[Box]] in
    var res = res
    res[box.thingsInside] = (res[box.thingsInside] ?? []) + [box]
    return res
}

Agora agrupado é um [Int:[Box]] e tem o seguinte conteúdo

[
    2: [Box(name: "Box 1", thingsInside: 2), Box(name: "Box 4", thingsInside: 2)], 
    3: [Box(name: "Box 2", thingsInside: 3), Box(name: "Box 5", thingsInside: 3)],
    1: [Box(name: "Box 0", thingsInside: 1), Box(name: "Box 3", thingsInside: 1), Box(name: "Box 6", thingsInside: 1)]
]

Comparando 2 Arrays com zip #

A função zip aceita 2 parâmetros do tipo SequenceType e retorna um Zip2Sequence onde cada elemento contém um valor da primeira sequência e um da segunda sequência.

Exemplo

let nums = [1, 2, 3]
let animals = ["Dog", "Cat", "Tiger"]
let numsAndAnimals = zip(nums, animals)

nomsAndAnimals agora contém os seguintes valores

sequência1 sequência1
1 "Cão"
2 "Gato"
3 "Tigre"

Isso é útil quando você deseja realizar algum tipo de comparação entre o n-ésimo elemento de cada Array.

Exemplo

Dadas 2 matrizes de Int(s)

let list0 = [0, 2, 4]
let list1 = [0, 4, 8]

você quer verificar se cada valor em list1 é o dobro do valor relacionado em list0.

let list1HasDoubleOfList0 = !zip(list0, list1).filter { $0 != (2 * $1)}.isEmpty
