Interface de linha de comando do Redis #

redis-cli é o programa de interface de linha de comando do Redis que permite enviar comandos para o Redis e ler as respostas enviadas pelo servidor, diretamente do terminal. O uso básico da linha de comando está abaixo:

Acesso ao redis:

$ redis-cli
127.0.0.1:6379>

Acesso ao redis com autenticação:

$ redis-cli -a myPassword
127.0.0.1:6379>

Selecione o banco de dados e mostre o tamanho do banco de dados (o número padrão do banco de dados é 0):

127.0.0.1:6379> dbsize
(integer) 2
127.0.0.1:6379> select 1
OK
127.0.0.1:6379[1]> dbsize
(integer) 20

Instale o Redis usando o Docker #

É simples começar a usar o Redis usando o docker:

docker pull redis
docker run -p 6379:6379 --rm --name redis redis

Agora você está executando a instância na porta 6397

Atenção: Todos os dados serão excluídos, quando o Redis for interrompido.

Para conectar o redis-cli, inicie outra janela de encaixe:

docker run -it --link redis:redis --rm redis redis-cli -h redis -p 6379

Agora você pode brincar com seu docker redis.

Visão geral #

O Redis é um banco de dados remoto na memória que oferece alto desempenho, replicação e um modelo de dados exclusivo para produzir uma plataforma para solucionar problemas. Redis é uma estrutura de dados na memória de código aberto (licenciada BSD), usada como banco de dados, cache e corretor de mensagens. Ele é categorizado como um armazenamento de valor-chave NoSQL. Ele suporta estruturas de dados como strings, hashes, listas, conjuntos, conjuntos ordenados com consultas de intervalo, bitmaps, hiperloglogs e índices geoespaciais com consultas de raio. Suportando cinco tipos diferentes de estruturas de dados,

  1. STRING (Operar em toda a string, partes, inteiros e floats)
  2. LISTA (Empurre ou puxe itens de ambas as extremidades)
  3. SET (Adicionar, buscar, remover, verificar, cruzar, união, diferença etc.)
  4. HASH (armazenar, pegar, remover em hash)
  5. ZSET (igual ao conjunto, mas de forma ordenada)
  6. GEO (Adicionar, atualizar, excluir latitude e longitude, chegar dentro de um dado redius)

O Redis possui replicação integrada, scripts Lua, despejo de LRU, transações e diferentes níveis de persistência em disco (sincronização/assíncrona).

Antes da versão 3, o Redis funcionava no modo mestre-escravo e exigia que o Redis-Sentinel fornecesse alta disponibilidade. Somente o mestre aceita gravações e sincroniza dados com seus escravos por bifurcação.

A partir da versão 3, o Redis funciona e recomenda o modo multimestre em que os recursos de failover, fragmentação/particionamento e reestilhaçamento são incorporados. O Redis-Sentinel não é necessário a partir da versão 3. Para que o cluster redis opere, são necessários no mínimo 3 nós/processos mestres.

Recursos adicionais são replicação, persistência e fragmentação do lado do cliente. O Redis acomoda uma ampla variedade de problemas que podem ser mapeados naturalmente no que o Redis oferece, permitindo que você resolva seus problemas sem ter que realizar o trabalho conceitual exigido por outros bancos de dados.

Redis “Olá Mundo” #

Primeiro você precisa instalar e iniciar seu servidor Redis, verifique o link abaixo que pode ajudá-lo a instalar o redis em seu servidor ou máquina local.

https://www.wikiod.com/pt/redis/instalacao-e-configuracao

Agora abra seu prompt de comando e execute o comando redis-cli:

Para salvar o primeiro conjunto > SET ‘keyname’ e depois ‘value’

127.0.0.1:6379> SET hkey "Hello World!"

Pressione Enter, você deve ver

OK

Em seguida, digite:

GET hkey

Você deveria ver:

"Hello World!"

Instalação do Redis no Windows, com exemplo do Node.js #

O Redis possui uma porta Windows fornecida pela ‘Microsoft Open Technologies’. Você pode usar o instalador msi encontrado em: https://github.com/MSOpenTech/redis/releases

Após a conclusão da instalação, você pode ver que ‘Redis’ é um serviço do Windows (e seu status deve ser “Iniciado”)

Para escrever um exemplo ‘Hello world’ que usa Redis no Node.js (no Windows também), você pode usar o seguinte módulo npm: https://www.npmjs.com/package/redis

exemplo de código:

var redis = require('redis'),
    client = redis.createClient();

client.set('mykey', 'Hello World');
client.get('mykey', function(err,res){
    console.log(res);
});
