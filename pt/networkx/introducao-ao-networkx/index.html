Instalação ou configuração #

Bem-vindo ao mundo dos gráficos. Se você conectou dados, pode precisar de um dos tipos de gráficos para modelar esses padrões. Existem várias coisas que podem ser feitas com Gráficos, como mapeamento de padrões de tráfego, gerenciamento de redes de distribuição de água, análise de mídia social, etc.

NetworkX é uma biblioteca que lida com criação/importação/exportação/manipulação/algoritmos/plotagem de banco de dados gráfico. Você pode começar a usar vários tipos de gráficos de rede. Por exemplo: o Facebook pode colocar seus usuários em um Graph() simples

import networkx as nx
facebook = nx.Graph()  
facebook.add_node('you')
facebook.add_node('me')
if both_agree_to_be_friends('you','me'):
    facebook.add_edge('you','me') #order isn't important here.

O Facebook usaria um Graph() regular porque não há nada de especial sobre a borda entre os nós. Este exemplo do Facebook só pode ter uma vantagem (amizade) entre os nós.

Outro tipo de gráfico seria um gráfico dirigido. O Twitter usaria um gráfico direcionado porque os nós têm uma direção. No Twitter eu posso te seguir, mas você não precisa me seguir. Então, poderíamos representar isso com este código:

import networkx as nx
twitter = nx.DiGraph()
twitter.add_node('you')
twitter.add_node('me')
twitter.add_edge('me','you') #order is important here.

É assim que o Twitter pode configurar ’eu’ para seguir ‘você’, mas não o contrário.

Também existem MultiGraphs() e MultiDiGraphs() caso você queira mais de uma aresta entre dois nós. Esses quatro tipos cobrem uma ampla variedade de problemas que podem ser representados com Gráficos. Se você quiser, pode adicionar um dicionário de dados a nós e arestas. A documentação deste módulo é incrível. Cada algoritmo é bem pesquisado e bem implementado. Espero que goste de trabalhar com isso.

Instruções de instalação: As instruções detalhadas sobre a instalação do NetworkX estão disponíveis aqui.

Como em qualquer outro pacote python, o NetworkX pode ser instalado usando pip, Miniconda/Anaconda e a partir do código-fonte.

Instalando com pip

pip install networkx

Será feita uma tentativa de localizar e instalar uma versão apropriada do NetworkX que corresponda ao seu sistema operacional e à versão do Python.

Para usar o pip, você precisa ter o setuptools instalado.

Se você deseja instalar a versão de desenvolvimento do GitHub, use o comando

pip install git://github.com/networkx/networkx.git#egg=networkx

Miniconda e Anaconda usam conda para instalação/atualização de software.

O NetworkX está atualmente instalado com Anaconda. Miniconda não vem com o NetworkX por padrão.

Você pode atualizar/instalar o NetworkX para a versão mais recente com:

conda install networkx

ou se você deseja atualizar a instalação do NetworkX, então

conda update networkx

Instalando da fonte

Arquivo do arquivo de origem

  1. Baixe a fonte de https://pypi.python.org/pypi/networkx/ or get the latest version.
  2. Desembale e troque directory to the source directory (it should have the files README.txt and setup.py).
  3. Execute python setup.py install para compilar e install
  4. (Opcional) Execute os testes de nariz para executar os testes se você tiver nose installed.

Instalando do GitHub

  1. Clone o repositório NetworkX (consulte https://github.com/networkx/networkx/ para opções)

    git clone https://github.com/networkx/networkx.git

  2. Altere o diretório para NetworkX

  3. Execute python setup.py install para compilar e instalar

  4. (Opcional) Execute o nosetests para executar os testes se você tiver o nose instalado.

Se você não tem permissão para instalar software em seu sistema, você pode instalar em outro diretório usando os sinalizadores --user, --prefix ou --home para setup.py.

Requisitos Para usar o NetworkX você precisa do Python 2.7, 3.3 ou posterior

Pacotes opcionais

  1. NumPy: Fornece representação matricial de gráficos e é usado em alguns algoritmos de gráficos para cálculos de matrizes de alto desempenho. (http://scipy.org/Download)
  2. SciPy: Fornece representação matricial esparsa de gráficos e muitas ferramentas científicas numéricas. (http://scipy.org/Download)
  3. Matplotlib: Fornece desenho flexível de gráficos. (http://matplotlib.sourceforge.net/)
  4. GraphViz em conjunto com PyGraphviz (http://pygraphviz.github.io) ou pydotplus (https://github.com/carlos-jenkins/pydotplus): fornece desenho gráfico e algoritmos de layout de gráfico. (http://graphviz.org)
  5. PyYAML: Necessário para leitura e escrita no formato YAML. (http://pyyaml.org)

Programa básico para exibir nós no matplotlib usando networkx #

import networkx as nx  # importing networkx package
import matplotlib.pyplot as plt # importing matplotlib package and pyplot is for displaying the graph on canvas 
b=nx.Graph()
b.add_node('helloworld')
b.add_node(1)
b.add_node(2)
'''Node can be called by any python-hashable obj like string,number etc'''
nx.draw(b)  #draws the networkx graph containing nodes which are declared till before
plt.show()  # displays the networkx graph on matplotlib canvas

Esclarecimentos adicionais:

nx.draw(b,nodelist=[1,'helloworld']) #displays the particular nodes which are given by nodelist only 
nx.draw_networkx(b,nodelist=[1,'helloworld']) #displays the node along with its name given by us i.e 1, hello  respectively
