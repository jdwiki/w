O componente cursor nos permite interagir com entidades clicando e olhando.

Sintaxe #

  • <a-cursor de entidade=""></a-cursor>
  • <a-cursor></a-cursor>

Parâmetros #

Propriedade Descrição
fusível Se o cursor é baseado em fusíveis. Valor padrão: false no desktop, true no celular
fuseTimeout Quanto tempo esperar (em milissegundos) antes de acionar um evento de clique baseado em fusível. Valor padrão: 1500

O cursor é uma aplicação específica do componente raycaster na medida em que

  • Escuta cliques do mouse e fusíveis baseados em olhar
  • Captura apenas a primeira entidade interceptada
  • Emite eventos especiais de mouse e hover (por exemplo, relacionados ao mouse para baixo/para cima/entrar/sair)
  • Tem mais estados para pairar.

Quando o mouse clica, a entidade visível mais próxima que cruza o cursor, se houver, emitirá um evento de clique. Observe que o componente cursor aplica apenas o comportamento de raycasting. Para fornecer uma forma ou aparência ao cursor, você pode aplicar a geometria e os componentes do material.

Eventos #

Evento Descrição
clique Emitido no cursor e na entidade interceptada se uma entidade atualmente interceptada for clicada (seja pelo mouse ou pelo fusível).
fusão Emitido no cursor e na entidade de interseção quando o cursor baseado em fusível inicia a contagem regressiva.
mousedown Emitido no cursor e na entidade de interseção (se houver) ao clicar com o mouse no elemento de tela.
mouseenter Emitido no cursor e na entidade interceptada (se houver) quando o cursor cruza com uma entidade.
mouseleave Emitido no cursor e na entidade interceptada (se houver) quando o cursor não cruza mais com a entidade interceptada anteriormente.
mouseup Emitido no cursor e na entidade de interseção (se houver) ao clicar com o mouse no elemento de tela.

Cursor padrão #

Por exemplo, podemos criar um cursor em forma de anel fixado no centro da tela. Para fixar o cursor na tela para que o cursor esteja sempre presente, não importa para onde olhemos, nós o colocamos como filho da entidade câmera ativa. Nós o colocamos na frente da câmera, colocando-o no eixo Z negativo. Quando o cursor clica na caixa, podemos ouvir o evento de clique.

<a-entity camera>
  <a-entity cursor="fuse: true; fuseTimeout: 500"
            position="0 0 -1"
            geometry="primitive: ring; radiusInner: 0.02; radiusOuter: 0.03"
            material="color: black; shader: flat">
  </a-entity>
</a-entity>

<a-entity id="box" cursor-listener geometry="primitive: box" material="color: blue">
</a-entity>

// Component to change to random color on click.
AFRAME.registerComponent('cursor-listener', {
  init: function () {
    var COLORS = ['red', 'green', 'blue'];
    this.el.addEventListener('click', function (evt) {
      var randomIndex = Math.floor(Math.random() * COLORS.length);
      this.setAttribute('material', 'color', COLORS[randomIndex]);
      console.log('I was clicked at: ', evt.detail.intersection.point);
    });
  }
});

Interações baseadas em olhar com componente de cursor #

Vamos primeiro examinar as interações baseadas no olhar. As interações baseadas no olhar dependem de girar nossas cabeças e olhar para os objetos para interagir com eles. Esse tipo de interação é para fones de ouvido sem controle. Mesmo com um controlador apenas de rotação (Daydream, GearVR), a interação ainda é semelhante. Como o A-Frame fornece controles de arrastar do mouse por padrão, o modo de olhar pode ser usado na área de trabalho para visualizar a interação arrastando a rotação da câmera.

Para adicionar interação baseada em olhar, precisamos adicionar ou implementar um componente. O A-Frame vem com um componente de cursor que fornece interação baseada em olhar se conectado à câmera:

  1. Defina explicitamente a entidade . Anteriormente, o A-Frame fornecia a câmera padrão.

  2. Adicione a entidade a-cursor como um elemento filho abaixo da entidade camera.

  3. Opcionalmente, configure o raycaster usado pelo cursor.

um primitivo de cursor #

A primitiva do cursor é uma retícula que permite o clique e a interatividade básica com uma cena em dispositivos que não possuem um controlador manual. A aparência padrão é uma geometria de anel. O cursor geralmente é colocado como filho da câmera.

<a-scene>
  <a-camera>
    <a-cursor></a-cursor>
  </a-camera>
  <a-box></a-box>
</a-scene>

Cursor baseado em fusível #

Também conhecido como cursor baseado em olhar. Se configurarmos o cursor para ser baseado em fusível, o cursor acionará um clique se o usuário olhar para uma entidade por um determinado período de tempo. Imagine um laser preso à cabeça do usuário e o laser se estende para a cena. Se o usuário olhar para uma entidade por tempo suficiente (ou seja, o fuseTimeout), o cursor acionará um clique.

A vantagem das interações baseadas em fusíveis para VR é que não requer dispositivos de entrada extras além do fone de ouvido. O cursor baseado em fusível destina-se principalmente a aplicativos do Google Cardboard. A desvantagem das interações baseadas em fusíveis é que exige que o usuário vire muito a cabeça.

Configurando o Cursor através do Componente Raycaster #

O cursor é construído em cima e depende do componente raycaster. Se quisermos personalizar as peças de raycasting do cursor, podemos fazer alterando as propriedades do componente raycaster. Digamos que queremos definir uma distância máxima, verificar interseções com menos frequência e definir quais objetos são clicáveis:

<a-entity cursor raycaster="far: 20; interval: 1000; objects: .clickable"></a-entity>

Adicionando feedback visual #

Para adicionar feedback visual ao cursor para mostrar quando o cursor está clicando ou fundindo, podemos usar o sistema de animação. Quando o cursor cruzar a entidade, ele emitirá um evento, e o sistema de animação pegará o evento com o atributo begin:

<a-entity cursor="fuse: true; fuseTimeout: 500"
          position="0 0 -1"
          geometry="primitive: ring"
          material="color: black; shader: flat">
  <a-animation begin="click" easing="ease-in" attribute="scale"
               fill="backwards" from="0.1 0.1 0.1" to="1 1 1"></a-animation>
  <a-animation begin="cursor-fusing" easing="ease-in" attribute="scale"
               fill="forwards" from="1 1 1" to="0.1 0.1 0.1"></a-animation>
</a-entity>

Cursor do mouse #

Observação: Para este exemplo, você precisa adicionar um pacote npm externo.

Se você quiser usar um cursor do mouse do seu computador, você precisa adicionar aframe-mouse-cursor-component. Depois se você deve incluir o script usando este código:

import 'aframe';
import 'aframe-mouse-cursor-component';

// or this 

require('aframe');
require('aframe-mouse-cursor-component');

E na sua câmera você precisa adicionar o componente mouse-cursor.

<a-scene>
  <a-entity camera look-controls mouse-cursor>
</a-scene>
