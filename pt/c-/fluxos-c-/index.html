O construtor padrão de std::istream_iterator constrói um iterador que representa o final do fluxo. Assim, std::copy(std::istream_iterator<int>(ifs), std::istream_iterator<int>(), .... significa copiar da posição atual em ifs para o final.

Streams de string #

Imprimindo coleções com iostream #

Impressão básica #

std::ostream_iterator permite imprimir o conteúdo de um contêiner STL para qualquer fluxo de saída sem loops explícitos. O segundo argumento do construtor std::ostream_iterator define o delimitador. Por exemplo, o seguinte código:

std::vector<int> v = {1,2,3,4};
std::copy(v.begin(), v.end(), std::ostream_iterator<int>(std::cout, " ! "));

vai imprimir

1 ! 2 ! 3 ! 4 !

Conversão de tipo implícito #

std::ostream_iterator permite converter o tipo de conteúdo do container implicitamente. Por exemplo, vamos ajustar std::cout para imprimir valores de ponto flutuante com 3 dígitos após o ponto decimal:

std::cout << std::setprecision(3);
std::fixed(std::cout);

e instancia std::ostream_iterator com float, enquanto os valores contidos permanecem int:

std::vector<int> v = {1,2,3,4};
std::copy(v.begin(), v.end(), std::ostream_iterator<float>(std::cout, " ! "));

então o código acima rende

1.000 ! 2.000 ! 3.000 ! 4.000 !

apesar de std::vector conter ints.

Geração e transformação #

As funções std::generate, std::generate_n e std::transform fornecem uma ferramenta muito poderosa para manipulação de dados on-the-fly. Por exemplo, tendo um vetor:

std::vector<int> v = {1,2,3,4,8,16};

podemos facilmente imprimir o valor booleano da declaração “x is even” para cada elemento:

std::boolalpha(std::cout); // print booleans alphabetically
std::transform(v.begin(), v.end(), std::ostream_iterator<bool>(std::cout, " "),
[](int val) {
    return (val % 2) == 0;
});

ou imprima o elemento quadrado:

std::transform(v.begin(), v.end(), std::ostream_iterator<int>(std::cout, " "),
[](int val) {
    return val * val;
});

Imprimindo N números aleatórios delimitados por espaço:

const int N = 10;
std::generate_n(std::ostream_iterator<int>(std::cout, " "), N, std::rand);

Matrizes #

Como na seção sobre leitura de arquivos de texto, quase todas essas considerações podem ser aplicadas a arrays nativos. Por exemplo, vamos imprimir valores quadrados de um array nativo:

int v[] = {1,2,3,4,8,16};
std::transform(v, std::end(v), std::ostream_iterator<int>(std::cout, " "),
[](int val) {
    return val * val;
});

Lendo um arquivo até o final #

Lendo um arquivo de texto linha por linha #

Uma maneira adequada de ler um arquivo de texto linha por linha até o final geralmente não é clara na documentação do ifstream. Vamos considerar alguns erros comuns cometidos por programadores C++ iniciantes e uma maneira adequada de ler o arquivo.

Linhas sem caracteres de espaço em branco #

Para simplificar, vamos supor que cada linha no arquivo não contenha símbolos de espaço em branco.

ifstream tem operator bool(), que retorna true quando um stream não tem erros e está pronto para ser lido. Além disso, ifstream::operator >> retorna uma referência ao próprio fluxo, para que possamos ler e verificar EOF (assim como erros) em uma linha com uma sintaxe muito elegante:

std::ifstream ifs("1.txt");
std::string s;
while(ifs >> s) {
    std::cout << s << std::endl;
}

Linhas com caracteres de espaço em branco #

ifstream::operator >> lê o fluxo até que qualquer caractere de espaço em branco ocorra, então o código acima imprimirá as palavras de uma linha em linhas separadas. Para ler tudo até o final da linha, use std::getline em vez de ifstream::operator >>. getline retorna a referência ao thread com o qual trabalhou, então a mesma sintaxe está disponível:

while(std::getline(ifs, s)) {
    std::cout << s << std::endl;
}

Obviamente, std::getline também deve ser usado para ler um arquivo de linha única até o final.

Lendo um arquivo em um buffer de uma só vez #

Finalmente, vamos ler o arquivo do início ao fim sem parar em nenhum caractere, incluindo espaços em branco e novas linhas. Se soubermos que o tamanho exato do arquivo ou o limite superior do comprimento é aceitável, podemos redimensionar a string e ler:

s.resize(100);
std::copy(std::istreambuf_iterator<char>(ifs), std::istreambuf_iterator<char>(),
    s.begin());

Caso contrário, precisamos inserir cada caractere no final da string, então std::back_inserter é o que precisamos:

std::copy(std::istreambuf_iterator<char>(ifs), std::istreambuf_iterator<char>(),
    std::back_inserter(s));

Alternativamente, é possível inicializar uma coleção com dados de fluxo, usando um construtor com argumentos de intervalo do iterador:

std::vector v(std::istreambuf_iterator<char>(ifs),
    std::istreambuf_iterator<char>());

Observe que esses exemplos também são aplicáveis ​​se ifs for aberto como arquivo binário:

std::ifstream ifs("1.txt", std::ios::binary);

Copiando fluxos #

Um arquivo pode ser copiado para outro arquivo com fluxos e iteradores:

std::ofstream ofs("out.file");
std::copy(std::istreambuf_iterator<char>(ifs), std::istreambuf_iterator<char>(),
    std::ostream_iterator<char>(ofs));
ofs.close();

ou redirecionado para qualquer outro tipo de stream com interface compatível. Por exemplo, fluxo de rede Boost.Asio:

boost::asio::ip::tcp::iostream stream;
stream.connect("example.com", "http");
std::copy(std::istreambuf_iterator<char>(ifs), std::istreambuf_iterator<char>(),
    std::ostream_iterator<char>(stream));
stream.close();

Matrizes #

Como os iteradores podem ser considerados uma generalização de ponteiros, os contêineres STL nos exemplos acima podem ser substituídos por matrizes nativas. Aqui está como analisar números em array:

int arr[100];
std::copy(std::istream_iterator<char>(ifs), std::istream_iterator<char>(), arr);

Cuidado com o estouro de buffer, pois os arrays não podem ser redimensionados rapidamente após serem alocados. Por exemplo, se o código acima for alimentado com um arquivo que contém mais de 100 números inteiros, ele tentará escrever fora do array e terá um comportamento indefinido.

Analisando arquivos #

Analisando arquivos em contêineres STL #

istream_iterators são muito úteis para ler sequências de números ou outros dados analisáveis ​​em contêineres STL sem loops explícitos no código.

Usando o tamanho do contêiner explícito:

std::vector<int> v(100);
std::copy(std::istream_iterator<int>(ifs), std::istream_iterator<int>(),
    v.begin());

ou com a inserção de iterador:

std::vector<int> v;
std::copy(std::istream_iterator<int>(ifs), std::istream_iterator<int>(),
    std::back_inserter(v));

Observe que os números no arquivo de entrada podem ser divididos por qualquer número de caracteres de espaço em branco e novas linhas.

Analisando tabelas de texto heterogêneas #

Como istream::operator>> lê o texto até um símbolo de espaço em branco, ele pode ser usado na condição while para analisar tabelas de dados complexas. Por exemplo, se tivermos um arquivo com dois números reais seguidos de uma string (sem espaços) em cada linha:

1.12 3.14 foo
2.1 2.2 barr

pode ser analisado assim:

std::string s;
double a, b;
while(ifs >> a >> b >> s) {
    std::cout << a << " " << b << " " << s << std::endl;
}

Transformação #

Qualquer função de manipulação de intervalo pode ser usada com intervalos std::istream_iterator. Um deles é o std::transform, que permite processar dados on-the-fly. Por exemplo, vamos ler valores inteiros, multiplicá-los por 3,14 e armazenar o resultado em um contêiner de ponto flutuante:

std::vector<double> v(100);
std::transform(std::istream_iterator<int>(ifs), std::istream_iterator<int>(),
v.begin(),
[](int val) {
    return val * 3.14;
});
