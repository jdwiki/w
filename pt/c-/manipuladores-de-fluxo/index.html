Manipuladores são funções auxiliares especiais que ajudam a controlar fluxos de entrada e saída usando operator >> ou operator <<.

Todos eles podem ser incluídos por #include <iomanip>.

Manipuladores podem ser usados ​​de outra maneira. Por exemplo:

  1. os.width(n); é igual a os << std::setw(n);
    is.width(n); é igual a is >> std::setw(n);

  2. os.precision(n); é igual a os << std::setprecision(n);
    is.precision(n); é igual a is >> std::setprecision(n);

  3. os.setfill(c); é igual a os << std::setfill(c);

  4. str >> std::setbase(base); ou str << std::setbase(base); igual a

str.setf(base ==  8 ? std::ios_base::oct :
            base == 10 ? std::ios_base::dec :
                base == 16 ? std::ios_base::hex :
                     std::ios_base::fmtflags(0),
         std::ios_base::basefield);

  1. os.setf(std::ios_base::flag); é igual a os << std::flag;
    is.setf(std::ios_base::flag); é igual a is >> std::flag;

    os.unsetf(std::ios_base::flag); é igual a os << std::no ## flag;
    is.unsetf(std::ios_base::flag); é igual a is >> std::no ## flag;

    (onde ## - é operador de concatenação)

    para os próximos flags: boolalpha, showbase, showpoint, showpos, skipws, maiúsculas.

  2. std::ios_base::basefield.
    Para flags: dec, hex e oct:

  • os.setf(std::ios_base::flag, std::ios_base::basefield); é igual a os << std::flag;
    is.setf(std::ios_base::flag, std::ios_base::basefield); igual a is >> std::flag;
    (1)

  • str.unsetf(std::ios_base::flag, std::ios_base::basefield); é igual a str.setf(std::ios_base::fmtflags(0), std::ios_base::basefield) ;
    (2)
  1. std::ios_base::adjustfield.
    Para flags: left, right e internal:
  • os.setf(std::ios_base::flag, std::ios_base::adjustfield); é igual a os << std::flag;
    is.setf(std::ios_base::flag, std::ios_base::adjustfield); é igual a is >> std::flag;
    (1)

  • str.unsetf(std::ios_base::flag, std::ios_base::adjustfield); é igual a str.setf(std::ios_base::fmtflags(0), std::ios_base::adjustfield) ;
    (2)

( 1 ) Se o sinalizador do campo correspondente definido anteriormente já foi desmarcado por unsetf.
( 2 ) Se flag estiver definido.

  1. std::ios_base::floatfield.
  • os.setf(std::ios_base::flag, std::ios_base::floatfield); é igual a os << std::flag;
    is.setf(std::ios_base::flag, std::ios_base::floatfield); é igual a is >> std::flag;

    para flags: fixed e scientific.

  • os.setf(std::ios_base::fmtflags(0), std::ios_base::floatfield); é igual a os << std::defaultfloat;
    is.setf(std::ios_base::fmtflags(0), std::ios_base::floatfield); é igual a is >> std::defaultfloat;

  1. str.setf(std::ios_base::fmtflags(0), std::ios_base::flag); é igual a str.unsetf(std::ios_base::flag)

    para flags: basefield, adjustfield, floatfield.

  2. os.setf(mask) é igual a os << setiosflags(mask);
    is.setf(mask) é igual a is >> setiosflags(mask);

    os.unsetf(mask) é igual a os << resetiosflags(mask);
    is.unsetf(mask) é igual a is >> resetiosflags(mask);

    Para quase todas as mask do tipo std::ios_base::fmtflags.

Manipuladores de fluxo #

std::boolalpha e std::noboolalpha - alterna entre representação textual e numérica de booleanos.

std::cout << std::boolalpha << 1;
// Output: true

std::cout << std::noboolalpha << false;
// Output: 0

bool boolValue;
std::cin >> std::boolalpha >> boolValue;
std::cout << "Value \"" << std::boolalpha << boolValue
          << "\" was parsed as " << std::noboolalpha << boolValue;
// Input: true
// Output: Value "true" was parsed as 0

std::showbase e std::noshowbase - controlam se o prefixo que indica a base numérica é usado.

std::dec (decimal), std::hex (hexadecimal) e std::oct (octal) - são usados ​​para alterar a base para inteiros.

#include <sstream>

std::cout << std::dec << 29 << ' - '
          << std::hex << 29 << ' - '
          << std::showbase << std::oct << 29 << ' - '
          << std::noshowbase << 29  '\n';
int number;
std::istringstream("3B") >> std::hex >> number;
std::cout << std::dec << 10;
// Output: 22 - 1D - 35 - 035
// 59

Os valores padrão são std::ios_base::noshowbase e std::ios_base::dec.

Se você quiser ver mais sobre std::istringstream confira o cabeçalho <sstream>.


std::uppercase e std::nouppercase - controlam se caracteres maiúsculos são usados ​​na saída de ponto flutuante e inteiro hexadecimal. Não tem efeito nos fluxos de entrada.

std::cout << std::hex << std::showbase
              << "0x2a with nouppercase: " << std::nouppercase << 0x2a << '\n'
              << "1e-10 with uppercase: " << std::uppercase << 1e-10 << '\n'
}
// Output: 0x2a with nouppercase: 0x2a
// 1e-10 with uppercase: 1E-10

O padrão é std::nouppercase.


std::setw(n) - altera a largura do próximo campo de entrada/saída para exatamente n.

A propriedade width n é redefinida para 0 quando algumas funções são chamadas (a lista completa está aqui).

std::cout << "no setw:" << 51 << '\n'
          << "setw(7): " << std::setw(7) << 51 << '\n'
          << "setw(7), more output: " << 13
          << std::setw(7) << std::setfill('*') << 67 << ' ' << 94 << '\n';

char* input = "Hello, world!";
char arr[10];
std::cin >> std::setw(6) >> arr;
std::cout << "Input from \"Hello, world!\" with setw(6) gave \"" << arr << "\"\n";

// Output: 51
// setw(7):      51
// setw(7), more output: 13*****67 94

// Input: Hello, world!
// Output: Input from "Hello, world!" with setw(6) gave "Hello"

O padrão é std::setw(0).


std::left, std::right e std::internal - modifique a posição padrão dos caracteres de preenchimento definindo std: :ios_base::adjustfield para std::ios_base::left, std::ios_base::right e std::ios_base::internal correspondentemente. std::left e std::right aplicam-se a qualquer saída, std::internal - para saída inteira, de ponto flutuante e monetária. Não tem efeito nos fluxos de entrada.

#include <locale>
...

std::cout.imbue(std::locale("en_US.utf8"));

std::cout << std::left << std::showbase << std::setfill('*')
          << "flt: " << std::setw(15) << -9.87  << '\n'
          << "hex: " << std::setw(15) << 41 << '\n'
          << "  $: " << std::setw(15) << std::put_money(367, false) << '\n'
          << "usd: " << std::setw(15) << std::put_money(367, true) << '\n'
          << "usd: " << std::setw(15)
          << std::setfill(' ') << std::put_money(367, false) << '\n';
// Output:
// flt: -9.87**********
// hex: 41*************
//   $: $3.67**********
// usd: USD *3.67******
// usd: $3.67          

std::cout << std::internal << std::showbase << std::setfill('*')
          << "flt: " << std::setw(15) << -9.87  << '\n'
          << "hex: " << std::setw(15) << 41 << '\n'
          << "  $: " << std::setw(15) << std::put_money(367, false) << '\n'
          << "usd: " << std::setw(15) << std::put_money(367, true) << '\n'
          << "usd: " << std::setw(15)
          << std::setfill(' ') << std::put_money(367, true) << '\n';
// Output:
// flt: -**********9.87
// hex: *************41
//   $: $3.67**********
// usd: USD *******3.67
// usd: USD        3.67

std::cout << std::right << std::showbase << std::setfill('*')
          << "flt: " << std::setw(15) << -9.87  << '\n'
          << "hex: " << std::setw(15) << 41 << '\n'
          << "  $: " << std::setw(15) << std::put_money(367, false) << '\n'
          << "usd: " << std::setw(15) << std::put_money(367, true) << '\n'
          << "usd: " << std::setw(15)
          << std::setfill(' ') << std::put_money(367, true) << '\n';
// Output:
// flt: **********-9.87
// hex: *************41
//   $: **********$3.67
// usd: ******USD *3.67
// usd:       USD  3.67

O padrão é std::left.


std::fixed, std::scientific, std::hexfloat [C++11] e [std::defaultfloat] 7 [C++11] - altera a formatação para entrada/saída de ponto flutuante.

std::fixed define o std::ios_base::floatfield para std::ios_base::fixed,
std::cientific - para std::ios_base::scientific,
std::hexfloat - para std::ios_base::fixed | std::ios_base::cientific e
std::defaultfloat - para std::ios_base::fmtflags(0).

fmtflags

#include <sstream>
...

std::cout << '\n'
          << "The number 0.07 in fixed:      " << std::fixed << 0.01 << '\n'
          << "The number 0.07 in scientific: " << std::scientific << 0.01 << '\n'
          << "The number 0.07 in hexfloat:   " << std::hexfloat << 0.01 << '\n'
          << "The number 0.07 in default:    " << std::defaultfloat << 0.01 << '\n';

double f;
std::istringstream is("0x1P-1022");
double f = std::strtod(is.str().c_str(), NULL);
std::cout << "Parsing 0x1P-1022 as hex gives " << f << '\n';

// Output:
// The number 0.01 in fixed:      0.070000
// The number 0.01 in scientific: 7.000000e-02
// The number 0.01 in hexfloat:   0x1.1eb851eb851ecp-4
// The number 0.01 in default:    0.07
// Parsing 0x1P-1022 as hex gives 2.22507e-308

O padrão é std::ios_base::fmtflags(0).

Existe um bug em alguns compiladores que causa

double f;
std::istringstream("0x1P-1022") >> std::hexfloat >> f;
std::cout << "Parsing 0x1P-1022 as hex gives " << f << '\n';
// Output: Parsing 0x1P-1022 as hex gives 0

std::showpoint e std::noshowpoint - controlam se o ponto decimal é sempre incluído na representação de ponto flutuante. Não tem efeito nos fluxos de entrada.

std::cout << "7.0 with showpoint: " << std::showpoint << 7.0 << '\n'
          << "7.0 with noshowpoint: " << std::noshowpoint << 7.0 << '\n';
// Output: 1.0 with showpoint: 7.00000
// 1.0 with noshowpoint: 7

O padrão é std::showpoint.


std::showpos e std::noshowpos - controlam a exibição do sinal + na saída não negativa. Não tem efeito nos fluxos de entrada.

std::cout << "With showpos: " << std::showpos
          << 0 << ' ' << -2.718 << ' ' << 17 << '\n'
          << "Without showpos: " << std::noshowpos
          << 0 << ' ' << -2.718 << ' ' << 17 << '\n';
// Output: With showpos: +0 -2.718 +17
// Without showpos: 0 -2.718 17

Padrão se std::noshowpos.


std::unitbuf, std::nonitbuf - controla a descarga do fluxo de saída após cada operação. Não tem efeito no fluxo de entrada. std::unitbuf causa flushing.


std::setbase(base) - define a base numérica do fluxo.

std::setbase(8) é igual a definir std::ios_base::basefield para std::ios_base::oct,
std::setbase(16) - para std:: ios_base::hex,
std::setbase(10) - para std::ios_base::dec.

Se base for diferente de 8, 10 ou 16, então std::ios_base::basefield será definido como std::ios_base::fmtflags(0). Significa saída decimal e entrada dependente de prefixo.

Como padrão std::ios_base::basefield é std::ios_base::dec então por padrão std::setbase(10).


std::setprecision(n) - altera a precisão do ponto flutuante.

#include <cmath>
#include <limits>
...

typedef std::numeric_limits<long double> ld;
const long double pi = std::acos(-1.L);

std::cout << '\n'
          << "default precision (6):   pi: " << pi << '\n'
          << "                       10pi: " << 10 * pi << '\n'
          << "std::setprecision(4):  10pi: " << std::setprecision(4) << 10 * pi << '\n'
          << "                    10000pi: " << 10000 * pi << '\n'
          << "std::fixed:         10000pi: " << std::fixed << 10000 * pi << std::defaultfloat << '\n'
          << "std::setprecision(10):   pi: " << std::setprecision(10) << pi << '\n'
          << "max-1 radix precicion:   pi: " << std::setprecision(ld::digits - 1) << pi << '\n'
          << "max+1 radix precision:   pi: " << std::setprecision(ld::digits + 1) << pi << '\n'
          << "significant digits prec: pi: " << std::setprecision(ld::digits10) << pi << '\n';

// Output:
// default precision (6):   pi: 3.14159
//                        10pi: 31.4159
// std::setprecision(4):  10pi: 31.42
//                     10000pi: 3.142e+04
// std::fixed:         10000pi: 31415.9265
// std::setprecision(10):   pi: 3.141592654
// max-1 radix precicion:   pi: 3.14159265358979323851280895940618620443274267017841339111328125
// max+1 radix precision:   pi: 3.14159265358979323851280895940618620443274267017841339111328125
// significant digits prec: pi: 3.14159265358979324

O padrão é std::setprecision(6).


std::setiosflags(mask) e [std::resetiosflags(mask)](http://en .cppreference.com/w/cpp/io/manip/resetiosflags) - defina e limpe os sinalizadores especificados em mask do tipo std::ios_base::fmtflags.

#include <sstream>
...

std::istringstream in("10 010 10 010 10 010");
int num1, num2;

in >> std::oct >> num1 >> num2;
std::cout << "Parsing \"10 010\" with std::oct gives:   " << num1 << ' ' << num2 << '\n';
// Output: Parsing "10 010" with std::oct gives:   8 8

in >> std::dec >> num1 >> num2;
std::cout << "Parsing \"10 010\" with std::dec gives:   " << num1 << ' ' << num2 << '\n';
// Output: Parsing "10 010" with std::oct gives:   10 10

in >> std::resetiosflags(std::ios_base::basefield) >> num1 >> num2;
std::cout << "Parsing \"10 010\" with autodetect gives: " << num1 << ' ' << num2 << '\n';
// Parsing "10 010" with autodetect gives: 10 8

std::cout << std::setiosflags(std::ios_base::hex |
                              std::ios_base::uppercase |
                              std::ios_base::showbase) << 42 << '\n';
// Output: OX2A

std::skipws e std::noskipws - controlam o salto do espaço em branco inicial pelas funções de entrada formatadas. Não tem efeito nos fluxos de saída.

#include <sstream>
...

char c1, c2, c3;
std::istringstream("a b c") >> c1 >> c2 >> c3;
std::cout << "Default  behavior:  c1 = " << c1 << "  c2 = " << c2 << "  c3 = " << c3 << '\n';

std::istringstream("a b c") >> std::noskipws >> c1 >> c2 >> c3;
std::cout << "noskipws behavior:  c1 = " << c1 << "  c2 = " << c2 << "  c3 = " << c3 << '\n';
// Output: Default  behavior:  c1 = a  c2 = b  c3 = c
// noskipws behavior:  c1 = a  c2 =    c3 = b

O padrão é std::ios_base::skipws.


std::quoted(s[, delim[, escape]]) [C++14] - inserções ou extrações strings entre aspas com espaços incorporados.

s - a string a ser inserida ou extraída.
delim - o caractere a ser usado como delimitador, " por padrão.
escape - o caractere a ser usado como caractere de escape, \ por padrão.

#include <sstream>
...
 
std::stringstream ss;
std::string in = "String with spaces, and embedded \"quotes\" too";
std::string out;
 
ss << std::quoted(in);
std::cout << "read in     [" << in << "]\n"
          << "stored as   [" << ss.str() << "]\n";
 
ss >> std::quoted(out);
std::cout << "written out [" << out << "]\n";
// Output:
// read in     [String with spaces, and embedded "quotes" too]
// stored as   ["String with spaces, and embedded \"quotes\" too"]
// written out [String with spaces, and embedded "quotes" too]

Manipuladores de fluxo de saída #

std::ends - insere um caractere nulo '\0' no fluxo de saída. Mais formalmente, a declaração deste manipulador parece

template <class charT, class traits>
std::basic_ostream<charT, traits>& ends(std::basic_ostream<charT, traits>& os);

e este manipulador coloca o caractere chamando os.put(charT()) quando usado em uma expressão
os << std::ends;


std::endl e [std::flush](http://en.cppreference.com/w /cpp/io/manip/flush) ambos liberam o fluxo de saída out chamando out.flush(). Causa imediatamente a produção de saída. Mas std::endl insere o símbolo de fim de linha '\n' antes da descarga.

std::cout << "First line." << std::endl << "Second line. " << std::flush
          << "Still second line.";
// Output: First line.
// Second line. Still second line.

std::setfill(c) - altera o caractere de preenchimento para c. Freqüentemente usado com std::setw.

std::cout << "\nDefault fill: " << std::setw(10) << 79 << '\n'
          << "setfill('#'): " << std::setfill('#')
          << std::setw(10) << 42 << '\n';
// Output:
// Default fill:         79
// setfill('#'): ########79

std::put_money(mon[, intl]) [C++11]. Em uma expressão out << std::put_money(mon, intl), converte o valor monetário mon (do tipo long double ou std::basic_string) para sua representação de caractere conforme especificado pelo std::money_put faceta da localidade atualmente imbuída em out. Use strings de moeda internacional se intl for true, caso contrário, use símbolos de moeda.

long double money = 123.45;
// or std::string money = "123.45";
 
std::cout.imbue(std::locale("en_US.utf8"));
std::cout << std::showbase << "en_US: " << std::put_money(money)
          << " or " << std::put_money(money, true) << '\n';
// Output: en_US: $1.23 or USD  1.23
 
std::cout.imbue(std::locale("ru_RU.utf8"));
std::cout << "ru_RU: " << std::put_money(money)
          << " or " << std::put_money(money, true) << '\n';
// Output: ru_RU: 1.23 руб or 1.23 RUB 
 
std::cout.imbue(std::locale("ja_JP.utf8"));
std::cout << "ja_JP: " << std::put_money(money)
          << " or " << std::put_money(money, true) << '\n';
// Output: ja_JP: ￥123 or JPY  123

std::put_time(tmb, fmt) [C++11] - formata e gera um valor de data/hora para std::tm de acordo com o formato especificado fmt.

tmb - ponteiro para a estrutura de tempo do calendário const std::tm* como obtido de localtime() ou gmtime().
fmt - ponteiro para uma string terminada em nulo const CharT* especificando o formato de conversão.

#include <ctime>
...

std::time_t t = std::time(nullptr);
std::tm tm = *std::localtime(&t);

std::cout.imbue(std::locale("ru_RU.utf8"));
std::cout << "\nru_RU: " << std::put_time(&tm, "%c %Z") << '\n';
// Possible output:
// ru_RU: Вт 04 июл 2017 15:08:35 UTC

Manipuladores de fluxo de entrada #

std::ws - consome os principais espaços em branco no fluxo de entrada. É diferente de std::skipws.

#include <sstream>
...

std::string str;
std::istringstream("  \v\n\r\t    Wow!There   is no whitespaces!") >> std::ws >> str;
std::cout << str;
// Output: Wow!There   is no whitespaces!

std::get_money(mon[, intl]) [C++11]. Em uma expressão in >> std::get_money(mon, intl) analisa a entrada de caracteres como um valor monetário, conforme especificado pelo [std::money_get](http://en.cppreference.com/w /cpp/locale/money_get) do locale atualmente imbuído em in, e armazena o valor em mon (do tipo long double ou std::basic_string). Manipulador espera strings de moeda internacional necessárias se intl for true, espera símbolos de moeda opcionais caso contrário.

#include <sstream>
#include <locale>
...
 
std::istringstream in("$1,234.56 2.22 USD  3.33");
long double v1, v2;
std::string v3;

in.imbue(std::locale("en_US.UTF-8"));
in >> std::get_money(v1) >> std::get_money(v2) >> std::get_money(v3, true);
if (in) {
    std::cout << std::quoted(in.str()) << " parsed as: "
              << v1 << ", " << v2 << ", " << v3 << '\n';
}
// Output:
// "$1,234.56 2.22 USD  3.33" parsed as: 123456, 222, 333

std::get_time(tmb, fmt) [C++11] - analisa um valor de data/hora armazenado em tmb do formato especificado fmt.

tmb - ponteiro válido para o objeto const std::tm* onde o resultado será armazenado.
fmt - ponteiro para uma string terminada em nulo const CharT* especificando o formato de conversão.

#include <sstream>
#include <locale>
...

std::tm t = {};
std::istringstream ss("2011-Februar-18 23:12:34");

ss.imbue(std::locale("de_DE.utf-8"));
ss >> std::get_time(&t, "%Y-%b-%d %H:%M:%S");
if (ss.fail()) {
    std::cout << "Parse failed\n";
}
else {
    std::cout << std::put_time(&t, "%c") << '\n';
}
// Possible output:
// Sun Feb 18 23:12:34 2011
