Para usar std::bitset você terá que incluir <bitset> header.

#include <bitset>

std::bitset sobrecarrega todas as funções do operador para permitir o mesmo uso que a manipulação de bitsets no estilo c.

Referências

  • Truques pouco giratórios

Alternando um pouco #

Manipulação de bits estilo C #

Um bit pode ser alternado usando o operador XOR (^).

// Bit x will be the opposite value of what it is currently
number ^= 1LL << x;

Usando std::bitset #

std::bitset<4> num(std::string("0100"));
num.flip(2); // num is now 0000
num.flip(0); // num is now 0001
num.flip();  // num is now 1110 (flips all bits)

Verificando um pouco #

Manipulação de bits estilo C #

O valor do bit pode ser obtido deslocando o número para a direita x vezes e, em seguida, executando bit a bit AND (&) nele:

(number >> x) & 1LL;  // 1 if the 'x'th bit of 'number' is set, 0 otherwise

A operação de deslocamento à direita pode ser implementada como um deslocamento aritmético (sinalizado) ou um deslocamento lógico (sem sinal). Se number na expressão number >> x tiver um tipo com sinal e um valor negativo, o valor resultante será definido pela implementação.

Se precisarmos do valor desse bit diretamente no local, poderíamos deslocar a máscara para a esquerda:

(number & (1LL << x));  // (1 << x) if the 'x'th bit of 'number' is set, 0 otherwise

Qualquer um pode ser usado como condicional, pois todos os valores diferentes de zero são considerados verdadeiros.

Usando std::bitset #

std::bitset<4> num(std::string("0010"));
bool bit_val = num.test(1);  // bit_val value is set to true;

Definir todos os bits #

Manipulação de bits estilo C #

x = -1; // -1 == 1111 1111 ... 1111b

(Veja aqui para uma explicação de por que isso funciona e é realmente a melhor abordagem.)

Usando std::bitset #

std::bitset<10> x;
x.set(); // Sets all bits to '1'

Remova o bit definido mais à direita #

Manipulação de bits estilo C #

template <typename T>
T rightmostSetBitRemoved(T n)
{
    // static_assert(std::is_integral<T>::value && !std::is_signed<T>::value, "type should be unsigned"); // For c++11 and later
    return n & (n - 1);
}

Explicação

  • se n for zero, temos 0 & 0xFF..FF que é zero
  • senão n pode ser escrito 0bxxxxxx10..00 e n - 1 é 0bxxxxxx011..11, então n & (n - 1) é 0bxxxxxx000..00.

Configurando um pouco #

Manipulação de bits estilo C #

Um bit pode ser definido usando o operador OR bit a bit (|).

// Bit x will be set
number |= 1LL << x;

Usando std::bitset #

set(x) ou set(x,true) - define o bit na posição x para 1.

std::bitset<5> num(std::string("01100"));
num.set(0);      // num is now 01101
num.set(2);      // num is still 01101
num.set(4,true); // num is now 11110

Limpando um pouco #

Manipulação de bits estilo C #

Um bit pode ser apagado usando o operador AND bit a bit (&).

// Bit x will be cleared
number &= ~(1LL << x);

Usando std::bitset #

reset(x) ou set(x,false) - limpa o bit na posição x.

std::bitset<5> num(std::string("01100"));
num.reset(2);     // num is now 01000
num.reset(0);     // num is still 01000
num.set(3,false); // num is now 00000

Mudando o enésimo bit para x #

Manipulação de bits estilo C #

// Bit n will be set if x is 1 and cleared if x is 0.
number ^= (-x ^ number) & (1LL << n);

Usando std::bitset #

set(n,val) - define o bit n para o valor val.

std::bitset<5> num(std::string("00100"));
num.set(0,true);  // num is now 00101
num.set(2,false); // num is now 00001

Conjunto de bits de contagem #

A contagem de população de um bitstring é frequentemente necessária em criptografia e outras aplicações e o problema tem sido amplamente estudado.

A maneira ingênua requer uma iteração por bit:

unsigned value = 1234;
unsigned bits = 0;  // accumulates the total number of bits set in `n`

for (bits = 0; value; value >>= 1)
  bits += value & 1;

Um bom truque (baseado em https://www.wikiod.com/pt/docs/c%2b%2b/3016/bit-manipulation/17299/remove-rightmost-set-bit ) é:

unsigned bits = 0;  // accumulates the total number of bits set in `n`

for (; value; ++bits)
  value &= value - 1;

Ele passa por tantas iterações quantos forem os bits definidos, então é bom quando se espera que value tenha poucos bits diferentes de zero.

O método foi proposto pela primeira vez por Peter Wegner (em CACM 3 / 322 - 1960) e é bem conhecido desde que aparece em C Programming Language por Brian W. Kernighan e Dennis M. Ritchie.

Isso requer 12 operações aritméticas, uma das quais é uma multicação:

unsigned popcount(std::uint64_t x)
{
  const std::uint64_t m1  = 0x5555555555555555;  // binary: 0101...
  const std::uint64_t m2  = 0x3333333333333333;  // binary: 00110011..
  const std::uint64_t m4  = 0x0f0f0f0f0f0f0f0f;  // binary: 0000111100001111

  x -= (x >> 1) & m1;             // put count of each 2 bits into those 2 bits
  x = (x & m2) + ((x >> 2) & m2); // put count of each 4 bits into those 4 bits 
  x = (x + (x >> 4)) & m4;        // put count of each 8 bits into those 8 bits 
  return (x * h01) >> 56;  // left 8 bits of x + (x<<8) + (x<<16) + (x<<24) + ... 
}

Esse tipo de implementação tem o melhor comportamento no pior caso (consulte Peso de Hamming para obter mais detalhes).

Muitas CPUs têm uma instrução específica (como o popcnt do x86) e o compilador pode oferecer uma função específica (não padrão) incorporada. Por exemplo. com g++ existe:

int __builtin_popcount (unsigned x);

Verifica se um inteiro é uma potência de 2 #

O truque n & (n - 1) (consulte https://www.wikiod.com/pt/docs/c%2b%2b/3016/bit-manipulation/17299/remove-rightmost-set-bit) também é útil para determinar se um inteiro é uma potência de 2:

bool power_of_2 = n && !(n & (n - 1));

Observe que sem a primeira parte da verificação (n &&), 0 é incorretamente considerado uma potência de 2.

Aplicação de manipulação de bits: letras minúsculas para maiúsculas #

Uma das várias aplicações da manipulação de bits é converter uma letra minúscula para maiúscula ou vice-versa, escolhendo uma máscara e uma operação de bits adequada. Por exemplo, a letra a tem essa representação binária 01(1)00001 enquanto sua contraparte maiúscula tem 01(0)00001. Eles diferem apenas no bit entre parênteses. Nesse caso, converter a letra a de minúscula para maiúscula é basicamente definir o bit entre parênteses para um. Para isso, fazemos o seguinte:

/****************************************
convert small letter to captial letter.
========================================
     a: 01100001
  mask: 11011111  <-- (0xDF)  11(0)11111
      :---------
a&mask: 01000001  <-- A letter
*****************************************/

O código para converter uma letra para uma letra A é

#include <cstdio>

int main()
{
    char op1 = 'a';  // "a" letter (i.e. small case)
    int mask = 0xDF; // choosing a proper mask

    printf("a (AND) mask = A\n");
    printf("%c   &   0xDF = %c\n", op1, op1 & mask);
    
    return 0;
}

O resultado é

$ g++ main.cpp -o test1
$ ./test1
a (AND) mask = A
a   &   0xDF = A
