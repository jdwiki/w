##Enquanto loop A diretiva @while fará um loop sobre um bloco de código até que a condição especificada se torne falsa. No exemplo a seguir, esse loop será executado até $font-size <= 18 enquanto incrementa o valor de $font-size em 2.

$font-size: 12;

@while $font-size <= 18 {
    .font-size-#{$font-size} {
        font-size: ($font-size * 1px);
    }

    $font-size: $font-size + 2;
}

Saída do código acima

.font-size-12 {
  font-size: 12px;
}

.font-size-14 {
  font-size: 14px;
}

.font-size-16 {
  font-size: 16px;
}

.font-size-18 {
  font-size: 18px;
}

para loop #

A diretiva @for permite que você percorra algum código para uma quantidade definida de iterações e tem duas formas:

  • @for <var> de <start> até <end> {}
  • @for <var> de <start> a <end> {}

A diferença nas duas formas é o através e o para; a palavra-chave through incluirá o <end> no loop onde to não incluirá; usar através é o equivalente a usar >= ou <= em outras linguagens, como C++, JavaScript ou PHP.

Notas

  • Ambos <start> e <end> devem ser inteiros ou funções que retornam inteiros.
  • Quando <start> for maior que <end>, o contador diminuirá em vez de aumentar.

Exemplo de SCSS

@for $i from 1 through 3 {
  .foo-#{$i} { width: 10px * $i; }
}

// CSS output
.foo-1 { width: 10px; }
.foo-2 { width: 20px; }
.foo-3 { width: 30px; }

Diretiva condicional (if) #

A diretiva de controle @if avalia uma dada expressão e se ela retornar algo diferente de false, ela processa seu bloco de estilos.

Exemplo Sass

$test-variable: true !default

=test-mixin
  @if $test-variable
    display: block
  @else
    display: none

.test-selector
  +test-mixin

Exemplo de SCSS

$test-variable: true !default

@mixin test-mixin() {
  @if $test-variable {
    display: block;
  } @else {
    display: none;
  }
}

.test-selector {
  @include test-mixin();
}

Os exemplos acima produzem o seguinte CSS:

.test-selector {
  display: block;
}

Cada laço #

A diretiva @each permite iterar em qualquer lista ou mapa. Ele assume a forma de @each $var ou <list or map> {} onde $var pode ser qualquer nome de variável e <list or map> pode ser qualquer coisa que retorne uma lista ou mapa.

No exemplo a seguir, o loop irá percorrer a lista $authors atribuindo cada item a $author, processará seu bloco de estilos usando esse valor de $author e prosseguirá para o próximo item na lista.

Exemplo de SCSS

$authors: "adam", "steve", "john";
@each $author in $authors {
    .photo-#{$author} {
      background: image-url("avatars/#{$author}.png") no-repeat
    }
}

Saída CSS

.photo-adam {
  background: image-url("avatars/adam.png") no-repeat;
}
.photo-steve {
  background: image-url("avatars/steve.png") no-repeat;
}
.photo-john {
  background: image-url("avatars/john.png") no-repeat;
}

Atribuição múltipla #

A atribuição múltipla permite que você obtenha acesso fácil a todas as variáveis ​​declarando várias variáveis ​​na diretiva @each.

Listas aninhadas

Para ter acesso fácil a todos os elementos aninhados, você pode declarar variáveis ​​separadas para corresponder a cada elemento aninhado. Certifique-se de ter a quantidade correta de variáveis ​​e elementos aninhados. No exemplo a seguir, um loop each está iterando por meio de uma lista de três elementos, cada um contendo três elementos aninhados. Ter a quantidade errada de variáveis ​​declaradas resultará em um erro do compilador.

@each $animal, $color, $cursor in (puma, black, default),
                                  (sea-slug, blue, pointer),
                                  (egret, white, move) {
  .#{$animal}-icon {
    background-image: url('/images/#{$animal}.png');
    border: 2px solid $color;
    cursor: $cursor;
  }
}

Mapas

A atribuição múltipla também funciona para o Maps, mas é limitada a apenas duas variáveis, uma variável para acessar a chave e uma variável para acessar o valor. Os nomes $key e $value são arbitrários no exemplo a seguir:

@each $key, $value in ('first': 1, 'second': 2, 'third': 3) {
  .order-#{$key} {
    order: $value;
  }
}

Cada Loop com valores de mapas/listas #

No exemplo abaixo, o valor no mapa $color-array é tratado como uma lista de pares.

Entrada SCSS

$color-array:(
  black: #4e4e4e,                       
  blue: #0099cc,
  green: #2ebc78
);
@each $color-name, $color-value in $color-array {
  .bg-#{$color-name} {
    background: $color-value;
  }
}

Saída CSS

.bg-black {
  background: #4e4e4e;
}

.bg-blue {
  background: #0099cc;
}

.bg-green {
  background: #2ebc78;
}
