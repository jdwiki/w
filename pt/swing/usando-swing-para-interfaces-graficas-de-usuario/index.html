Saindo do aplicativo ao fechar a janela #

É fácil esquecer de sair do aplicativo quando a janela é fechada. Lembre-se de adicionar a seguinte linha.

frame.setDefaultCloseOperation(WindowConstants.EXIT_ON_CLOSE); //Quit the application when the JFrame is closed

Criando uma janela vazia (JFrame) #

Criando o JFrame #

Criar uma janela é fácil. Você só precisa criar um JFrame.

JFrame frame = new JFrame();

Intitular a janela #

Você pode querer dar um título à sua janela. Você pode fazer isso passando uma string ao criar o JFrame, ou chamando frame.setTitle(String title).

JFrame frame = new JFrame("Super Awesome Window Title!");
//OR
frame.setTitle("Super Awesome Window Title!");

Configurando o tamanho da janela #

A janela será tão pequena quanto possível quando for criada. Para torná-lo maior, você pode definir seu tamanho explicitamente:

frame.setSize(512, 256);

Ou você pode ter o próprio tamanho do quadro baseado no tamanho de seu conteúdo com o método pack().

frame.pack();

Os métodos setSize() e pack() são mutuamente exclusivos, então use um ou outro.

O que fazer ao fechar a janela #

Observe que o aplicativo não será encerrado quando a janela for fechada. Você pode sair do aplicativo após a janela ser fechada dizendo ao JFrame para fazer isso.

frame.setDefaultCloseOperation(WindowConstants.EXIT_ON_CLOSE);

Alternativamente, você pode dizer à janela para fazer outra coisa quando estiver fechada.

WindowConstants.DISPOSE_ON_CLOSE //Get rid of the window
WindowConstants.EXIT_ON_CLOSE //Quit the application
WindowConstants.DO_NOTHING_ON_CLOSE //Don't even close the window
WindowConstants.HIDE_ON_CLOSE //Hides the window - This is the default action

Criando um painel de conteúdo #

Uma etapa opcional é criar um painel de conteúdo para sua janela. Isso não é necessário, mas se você quiser fazer isso, crie um JPanel e chame frame.setContentPane(Component component).

JPanel pane = new JPanel();
frame.setContentPane(pane);

Mostrando a janela #

Depois de criá-lo, você desejará criar seus componentes e mostrar a janela. Mostrar a janela é feito como tal.

frame.setVisible(true);

Exemplo #

Para aqueles que gostam de copiar e colar, aqui está um código de exemplo.

JFrame frame = new JFrame("Super Awesome Window Title!"); //Create the JFrame and give it a title
frame.setSize(512, 256); //512 x 256px size
frame.setDefaultCloseOperation(WindowConstants.EXIT_ON_CLOSE); //Quit the application when the JFrame is closed

JPanel pane = new JPanel(); //Create the content pane
frame.setContentPane(pane); //Set the content pane

frame.setVisible(true); //Show the window

Adicionando componentes #

Um componente é algum tipo de elemento da interface do usuário, como um botão ou um campo de texto.

Criando um componente #

A criação de componentes é quase idêntica à criação de uma janela. Em vez de criar um JFrame no entanto, você cria esse componente. Por exemplo, para criar um JButton, faça o seguinte.

JButton button = new JButton();

Muitos componentes podem ter parâmetros passados ​​a eles quando criados. Por exemplo, um botão pode receber algum texto para exibição.

JButton button = new JButton("Super Amazing Button!");

Se você não quiser criar um botão, uma lista de componentes comuns pode ser encontrada em outro exemplo nesta página.

Os parâmetros que podem ser passados ​​a eles variam de componente para componente. Uma boa maneira de verificar o que eles podem aceitar é observar os parâmetros dentro do seu IDE (se você usar um). Os atalhos padrão estão listados abaixo.

  • IntelliJ IDEA - Windows / Linux: CTRL + P
  • IntelliJ IDEA - OS X / macOS: CMD + P
  • Eclipse: CTRL + SHIFT + Espaço
  • NetBeans: CTRL + P

Mostrando o componente #

Após a criação de um componente, você normalmente definiria seus parâmetros. Depois disso, você precisa colocá-lo em algum lugar, como em seu JFrame, ou em seu painel de conteúdo, se você criou um.

frame.add(button); //Add to your JFrame
//OR
pane.add(button); //Add to your content pane
//OR
myComponent.add(button); //Add to whatever

Exemplo #

Aqui está um exemplo de como criar uma janela, definir um painel de conteúdo e adicionar um botão a ele.

JFrame frame = new JFrame("Super Awesome Window Title!"); //Create the JFrame and give it a title
frame.setSize(512, 256); //512 x 256px size
frame.setDefaultCloseOperation(WindowConstants.EXIT_ON_CLOSE); //Quit the application when the JFrame is closed

JPanel pane = new JPanel(); //Create the content pane
frame.setContentPane(pane); //Set the content pane

JButton button = new JButton("Super Amazing Button!"); //Create the button
pane.add(button); //Add the button to the content pane

frame.setVisible(true); //Show the window

Configurando Parâmetros para Componentes #

Os componentes têm vários parâmetros que podem ser definidos para eles. Eles variam de componente para componente, portanto, uma boa maneira de ver quais parâmetros podem ser definidos para componentes é começar a digitar componentName.set e deixar o preenchimento automático do seu IDE (se você usar um IDE) sugerir métodos. O atalho padrão em muitos IDEs, se não aparecer automaticamente, é CTRL + Space.

Criando interfaces de usuário interativas #

Ter um botão lá é muito bom, mas qual é o sentido se clicar nele não faz nada? ActionListeners são usados ​​para dizer ao seu botão ou outro componente para fazer algo quando ele é ativado.

Adicionar ActionListeners é feito como tal.

buttonA.addActionListener(new ActionListener() {
    @Override
    public void actionPerformed(ActionEvent e) {
        //Code goes here...
        System.out.println("You clicked the button!");
    }
});

Ou, se você estiver usando Java 8 ou superior…

buttonA.addActionListener(e -> {
    //Code
    System.out.println("You clicked the button!");
});

Exemplo (Java 8 e superior) #

JFrame frame = new JFrame("Super Awesome Window Title!"); //Create the JFrame and give it a title
frame.setSize(512, 256); //512 x 256px size
frame.setDefaultCloseOperation(WindowConstants.EXIT_ON_CLOSE); //Quit the application when the JFrame is closed

JPanel pane = new JPanel(); //Create a pane to house all content
frame.setContentPane(pane);

JButton button = new JButton("Click me - I know you want to.");
button.addActionListener(e -> {
    //Code goes here
    System.out.println("You clicked me! Ouch.");
});
pane.add(buttonA);

frame.setVisible(true); //Show the window

Organizando o Layout do Componente #

Adicionar componentes um após o outro resulta em uma interface de usuário difícil de usar, porque todos os componentes estão em algum lugar. Os componentes são ordenados de cima para baixo, cada componente em uma “linha” separada.

Para remediar isso e fornecer a você como desenvolvedor a possibilidade de layout de componentes facilmente, o Swing possui LayoutManagers.

Esses LayoutManagers são abordados com mais detalhes em Introdução aos gerenciadores de layout bem como os tópicos separados do Gerenciador de Layout:

  • Layout do GridBag
