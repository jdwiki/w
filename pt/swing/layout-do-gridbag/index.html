Sintaxe #

  • frame.setLayout(novo GridBagLayout()); //Definir GridBagLayout para frame
  • pane.setLayout(novo GridBagLayout()); //Definir GridBagLayout para Painel
  • Painel JPanel = new JPanel(new GridBagLayout()); //Definir GridBagLayout para Painel
  • GridBagConstraints c = new GridBagConstraints() //Inicializa um GridBagConstraint

Como funciona o GridBagLayout? #

Os layouts são usados ​​sempre que você deseja que seus componentes não sejam exibidos apenas um ao lado do outro. O GridBagLayout é útil, pois divide sua janela em linhas e colunas, e você decide em qual linha e coluna colocar os componentes, bem como quantas linhas e colunas o componente tem.

Tomemos esta janela como exemplo. As linhas de grade foram marcadas para mostrar o layout.

Aqui, criei 6 componentes, dispostos usando um GridBagLayout.

Componente Posição Tamanho
JLabel: “Meu incrível aplicativo Swing” 0, 0 3, 1
JButton: “Botão A” 0, 1 1, 1
JButton: “Botão B” 1, 1 1, 1
JButton: “Botão C” 2, 1 1, 1
JSlider 0, 2 3, 1
JScrollBar 0, 3 3, 1

Observe que a posição 0, 0 está no canto superior esquerdo: os valores x (coluna) aumentam da esquerda para a direita, os valores y (linha) aumentam de cima para baixo.

Para iniciar o layout dos componentes em um GridBagLayout, primeiro defina o layout do seu JFrame ou painel de conteúdo.

frame.setLayout(new GridBagLayout());
//OR
pane.setLayout(new GridBagLayout());
//OR
JPanel pane = new JPanel(new GridBagLayout()); //Add the layout when creating your content pane

Observe que você nunca define o tamanho da grade. Isso é feito automaticamente à medida que você adiciona seus componentes.

Depois, você precisará criar um objeto GridBagConstraints.

GridBagConstraints c = new GridBagConstraints();

Para garantir que seus componentes preencham o tamanho da janela, você pode definir o peso de todos os componentes como 1. O peso é usado para determinar como distribuir o espaço entre colunas e linhas.

c.weightx = 1;
c.weighty = 1;

Outra coisa que você pode querer fazer é certificar-se de que os componentes ocupem o máximo de espaço horizontal possível.

c.fill = GridBagConstraints.HORIZONTAL;

Você também pode definir outras opções de preenchimento, se desejar.

GridBagConstraints.NONE //Don't fill components at all
GridBagConstraints.HORIZONTAL //Fill components horizontally
GridBagConstraints.VERTICAL //Fill components vertically
GridBagConstraints.BOTH //Fill components horizontally and vertically

Ao criar componentes, você desejará definir onde na grade ele deve ir e quantos blocos de grade devem ser usados. Por exemplo, para colocar um botão na 3ª linha da 2ª coluna e ocupar um espaço de grade de 5 x 5, faça o seguinte. Tenha em mente que a grade começa em 0, 0, não em 1, 1.

JButton button = new JButton("Fancy Button!");
c.gridx = 2;
c.gridy = 1;
c.gridwidth = 5;
c.gridheight = 5;
pane.add(buttonA, c);

Ao adicionar componentes à sua janela, lembre-se de passar as restrições como parâmetro. Isso pode ser visto na última linha do exemplo de código acima.

Você pode reutilizar o mesmo GridBagConstraints para cada componente - alterá-lo após adicionar um componente não altera o componente adicionado anteriormente.

Exemplo #

Aqui está o código para o exemplo no início desta seção.

JFrame frame = new JFrame("Super Awesome Window Title!"); //Create the JFrame and give it a title
frame.setSize(512, 256); //512 x 256px size
frame.setDefaultCloseOperation(WindowConstants.EXIT_ON_CLOSE); //Quit the application when the JFrame is closed

JPanel pane = new JPanel(new GridBagLayout()); //Create a pane to house all content, and give it a GridBagLayout
frame.setContentPane(pane);

GridBagConstraints c = new GridBagConstraints();
c.weightx = 1;
c.weighty = 1;
c.fill = GridBagConstraints.HORIZONTAL;

JLabel headerLabel = new JLabel("My Amazing Swing Application");
c.gridx = 0;
c.gridwidth = 3;
c.gridy = 0;
pane.add(headerLabel, c);

JButton buttonA = new JButton("Button A");
c.gridx = 0;
c.gridwidth = 1;
c.gridy = 1;
pane.add(buttonA, c);

JButton buttonB = new JButton("Button B");
c.gridx = 1;
c.gridwidth = 1;
c.gridy = 1;
pane.add(buttonB, c);

JButton buttonC = new JButton("Button C");
c.gridx = 2;
c.gridwidth = 1;
c.gridy = 1;
pane.add(buttonC, c);

JSlider slider = new JSlider(0, 100);
c.gridx = 0;
c.gridwidth = 3;
c.gridy = 2;
pane.add(slider, c);

JScrollBar scrollBar = new JScrollBar(JScrollBar.HORIZONTAL, 20, 20, 0, 100);
c.gridx = 0;
c.gridwidth = 3;
c.gridy = 3;
pane.add(scrollBar, c);

frame.setVisible(true); //Show the window
