Crie uma instância UIWebView #

Rápido

let webview = UIWebView(frame: CGRect(x: 0, y: 0, width: 320, height: 480))

Objetivo-C

UIWebView *webview = [[UIWebView alloc] initWithFrame:CGRectMake(0, 0, 320, 480)];

//Alternative way of defining frame for UIWebView
UIWebView *webview = [[UIWebView alloc] init];
CGRect webviewFrame = webview.frame;
webviewFrame.size.width = 320;
webviewFrame.size.height = 480;
webviewFrame.origin.x = 0;
webviewFrame.origin.y = 0;
webview.frame = webviewFrame;

Determinando o tamanho do conteúdo #

Em muitos casos, por exemplo, ao usar exibições da Web em células de exibição de tabela, é importante determinar o tamanho do conteúdo da página HTML renderizada. Depois de carregar a página, isso pode ser calculado no método delegado UIWebViewDelegate:

- (void) webViewDidFinishLoad:(UIWebView *) aWebView {
    CGRect frame = aWebView.frame;
    frame.size.height = 1;
    aWebView.frame = frame;
    CGSize fittingSize = [aWebView sizeThatFits:CGSizeZero];
    frame.size = fittingSize;
    aWebView.frame = frame;

    NSLog(@"size: %f, %f", fittingSize.width, fittingSize.height);
}

O código emprega um truque adicional de definir rapidamente a altura da visualização da web para 1 antes de medir o tamanho adequado. Caso contrário, ele simplesmente relataria o tamanho do quadro atual. Após a medição, definimos imediatamente a altura para a altura real do conteúdo.

Fazendo uma solicitação de URL #

Carregar conteúdo no webview do url

Rápido

webview.loadRequest(NSURLRequest(URL: NSURL(string: "http://www.google.com")!))

Objetivo-C

[webview loadRequest:[NSURLRequest requestWithURL:[NSURL URLWithString:@"http://www.google.com"]]];

Carregar string HTML #

As visualizações da Web são úteis para carregar strings HTML geradas localmente.

NSString *html = @"<!DOCTYPE html><html><body>Hello World</body></html>";
[webView loadHTMLString:html baseURL:nil];

Rápido

  let htmlString = "<h1>My First Heading</h1><p>My first paragraph.</p>"
  webView.loadHTMLString(htmlString, baseURL: nil)

Uma URL base local pode ser especificada. Isso é útil para fazer referência a imagens, folhas de estilo ou scripts do pacote de aplicativos:

NSString *html = @"<!DOCTYPE html><html><head><link href='style.css' rel='stylesheet' type='text/css'></head><body>Hello World</body></html>";
[self loadHTMLString:html baseURL:[NSURL fileURLWithPath:[[NSBundle mainBundle] resourcePath]]];

Neste caso, o style.css é carregado localmente do diretório de recursos do aplicativo. Claro que também é possível especificar um URL remoto.

Carregar JavaScript #

Podemos executar JavaScript personalizado em um UIWebView usando o método stringByEvaluatingJavaScriptFromString(). Este método retorna o resultado da execução do script JavaScript passado no parâmetro script, ou nil se o script falhar.

Rápido

Carregar script de String

webview.stringByEvaluatingJavaScriptFromString("alert('This is JavaScript!');")

Carregar script do arquivo local

//Suppose you have javascript file named "JavaScript.js" in project.
let filePath = NSBundle.mainBundle().pathForResource("JavaScript", ofType: "js")
        do {
            let jsContent = try String.init(contentsOfFile: filePath!, encoding: NSUTF8StringEncoding)
            webview.stringByEvaluatingJavaScriptFromString(jsContent)
        }
        catch let error as NSError{
            print(error.debugDescription)
        }

Objetivo-C

Carregar script de String

[webview stringByEvaluatingJavaScriptFromString:@"alert('This is JavaScript!');"];

Carregar script do arquivo local

//Suppose you have javascript file named "JavaScript.js" in project.
NSString *filePath = [[NSBundle mainBundle] pathForResource:@"JavaScript" ofType:@"js"];
NSString *jsContent = [NSString stringWithContentsOfFile:filePath encoding:NSUTF8StringEncoding error:nil];
[webview stringByEvaluatingJavaScriptFromString:jsContent];

Observação O método stringByEvaluatingJavaScriptFromString: aguarda de forma síncrona a conclusão da avaliação do JavaScript. Se você carregar conteúdo da Web cujo código JavaScript não foi verificado, invocar esse método pode travar seu aplicativo. A melhor prática é adotar a classe WKWebView e usar seu método evaluateJavaScript:completionHandler:. Mas o WKWebView está disponível a partir do iOS 8.0 e posterior.

Carregar arquivos de documentos como .pdf, .txt, .doc etc. #

Em vez de páginas da web, também podemos carregar os arquivos do documento no iOS WebView como .pdf, .txt, .doc etc. O método loadData é usado para carregar NSData no webview.

Rápido

//Assuming there is a text file in the project named "home.txt".
let localFilePath = NSBundle.mainBundle().pathForResource("home", ofType:"txt");
let data = NSFileManager.defaultManager().contentsAtPath(localFilePath!);
webview.loadData(data!, MIMEType: "application/txt", textEncodingName:"UTF-8" , baseURL: NSURL())

Objetivo-C

//Assuming there is a text file in the project named "home.txt".
NSString *localFilePath = [[NSBundle mainBundle] pathForResource:@"home" ofType:@"txt"];
NSData *data = [[NSFileManager defaultManager] contentsAtPath:localFilePath];
[webview loadData:data MIMEType:@"application/txt" textEncodingName:@"UTF-8" baseURL:[NSURL new]];

Faça links que dentro do UIWebview sejam clicáveis #

Em vc.h

@interface vc : UIViewController<UIWebViewDelegate>

em vc.m

- (BOOL)webView:(UIWebView *)webView shouldStartLoadWithRequest:(NSURLRequest *)request navigationType:(UIWebViewNavigationType)navigationType{
    
    
    if (navigationType == UIWebViewNavigationTypeLinkClicked){
        //open it on browser if you want to open it in same web view remove return NO;
        NSURL *url = request.URL;
        if ([[UIApplication sharedApplication] canOpenURL:url]) {
            [[UIApplication sharedApplication] openURL:url];
        }
        return NO;
        
    }
    
    return YES;
    
}

Carregar arquivo HTML local no webView #

Primeiro, adicione o arquivo HTML ao seu projeto (se você for solicitado a escolher opções para adicionar o arquivo, selecione Copiar itens, se necessário)

A linha de código a seguir carrega o conteúdo do arquivo HTML no webView

webView.loadRequest(NSURLRequest(URL: NSURL(fileURLWithPath: NSBundle.mainBundle().pathForResource("YOUR HTML FILE", ofType: "html")!})
  • Se seu arquivo HTML for chamado index.html, substitua SEU ARQUIVO HTML por index
  • Você pode usar este código em viewDidLoad() ou viewDidAppear() ou qualquer outra função
