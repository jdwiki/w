UIImage mascarada com Label #

Isso torna a imagem mascarada no formato das letras do rótulo:

Objetivo-C #

self.maskImage.layer.mask = self.maskLabel.layer;
self.maskImage.layer.masksToBounds = YES;

Swift 3 #

maskImageView.mask = maskLabel
maskImageView.masksToBounds = true

Fazendo uma imagem em um círculo ou arredondado #

Este exemplo mostra como fazer um UIView ou UIImageView, arredondado com algum raio como este:

Objetivo-C #

someImageView.layer.cornerRadius = CGRectGetHeight(someImageView.frame) / 2;
someImageView.clipsToBounds = YES;

Rápido #

someImageView.layer.cornerRadius = someImageView.frame.height/2
// this should alleviate the performance hit that adding transparency may cause - see http://stackoverflow.com/a/6254531/189804
// Be sure to check scrolling performance with Instruments if you take this approach.
someImageView.layer.shouldRasterize = true
someImageView.clipsToBounds = true // All parts of the image that are outside its bounds (the frame) are cut out (makes the rounded corners visible)

Sugere-se que se você usar autolayout coloque o código someImageView.layer.cornerRadius em viewDidLayoutSubviews. Isso permitirá que o cornerRadius da imagem seja atualizado se a imagem mudar de tamanho.

override func viewDidLayoutSubviews() {
    super.viewDidLayoutSubviews()
    someImageView.layer.cornerRadius = someImageView.frame.size.width/2
    someImageView.layer.masksToBounds = true
}

Como a propriedade Mode afeta uma imagem #

A propriedade do modo de conteúdo de uma visualização informa como seu conteúdo deve ser apresentado. No Interface Builder, os vários modos podem ser selecionados no Attributes Inspector.

Vamos usar duas visualizações de imagem para ver como os vários modos funcionam.

Escala para Preencher #

As alturas e larguras da imagem são esticadas para corresponder ao tamanho do UIImageView.

Ajuste de Aspecto #

O lado mais longo (altura ou largura) da imagem é esticado para corresponder à visualização. Isso torna a imagem o maior possível enquanto ainda mostra a imagem inteira e não distorce a altura ou a largura. (Defino o plano de fundo UIImageView para azul para que seu tamanho fique claro.)

Preenchimento de Aspecto #

O lado mais curto (altura ou largura) da imagem é esticado para corresponder à visualização. Como no “Aspect Fit”, as proporções da imagem não são distorcidas em relação à proporção original.

Redesenhar #

Redraw é apenas para visualizações personalizadas que precisam fazer seu próprio dimensionamento e redimensionamento. Não estamos usando uma visualização personalizada, portanto, não devemos usar o Redraw. Observe que aqui UIImageView apenas nos dá o mesmo resultado que Scale to Fill, mas está fazendo mais trabalho nos bastidores.

Sobre o Redraw, a documentação da Apple diz:

Os modos de conteúdo são bons para reciclar o conteúdo de sua visualização, mas você também pode definir o modo de conteúdo para o valor UIViewContentModeRedraw quando desejar especificamente que suas visualizações personalizadas sejam redesenhadas durante as operações de dimensionamento e redimensionamento. Definir o modo de conteúdo da sua visualização para esse valor força o sistema a chamar o método drawRect: da sua visualização em resposta a alterações na geometria. Em geral, você deve evitar usar esse valor sempre que possível e certamente não deve usá-lo com as visualizações do sistema padrão.

Centro #

Certo #

Animando um UIImageView #

Você pode animar um UIImageView exibindo rapidamente imagens nele em uma sequência usando as propriedades de animação do UIImageView:

imageView.animationImages = [UIImage(named: "image1")!,
                             UIImage(named: "image2")!,
                             UIImage(named: "image3")!,
                             UIImage(named: "image4")!,
                             UIImage(named: "image5")!,
                             UIImage(named: "image6")!,
                             UIImage(named: "image7")!,
                             UIImage(named: "image8")!] 
imageView.animationDuration = 0.3
imageView.animationRepeatCount = 1

A propriedade animationImages é um Array de UIImages que é executado de cima para baixo quando a animação é acionada.

A propriedade animationDuration é um Double dizendo por quantos segundos a animação será executada.

A propriedade animationRepeatCount é um Int que diz quantas vezes a animação será executada.

Para iniciar e parar a animação, você pode chamar os métodos apropriados para isso:

imageView.startAnimating()
imageView.stopAnimating()

Existe o método isAnimating() que retorna um valor Boolean indicando se a animação está sendo executada em um momento ou não.

Observe que essa não é uma maneira muito eficiente de criar animações: é bastante lenta e consome recursos. Considere usar Layers ou Sprites para obter melhores resultados

Crie uma UIImageView #

Para criar um UIImageView programaticamente, tudo que você precisa fazer é criar uma instância de UIImageView:

//Swift
let imageView = UIImageView()

//Objective-C
UIImageView *imageView = [[UIImageView alloc] init];

Você pode definir o tamanho e a posição do UIImageView com um CGRect:

//Swift
imageView.frame = CGRect(x: 0, y: 0, width: 200, height: 200)

//Objective-C
imageView.frame = CGRectMake(0,0,200,200);

Ou você pode definir o tamanho durante a inicialização:

//Swift
UIImageView(frame: CGRect(x: 0, y: 0, width: 200, height: 200))

//Objective-C
UIImageView *imageView = [[UIImageView alloc] initWithFrame:CGRectMake(0,0,200,200);

//Alternative way of defining frame for UIImageView
UIImageView *imageView = [[UIImageView alloc] init];
CGRect imageViewFrame = imageView.frame;
imageViewFrame.size.width = 200;
imageViewFrame.size.height = 200;
imageViewFrame.origin.x = 0;
imageViewFrame.origin.y = 0;
imageView.frame = imageViewFrame;

Nota: Você deve importar o UIKit para usar um UIImageView.

Atribuindo uma imagem a um UIImageView #

Você pode atribuir uma imagem a um UIImageView durante a inicialização ou posteriormente usando a propriedade image:

//Swift
UIImageView(image: UIImage(named: "image1"))

UIImageView(image: UIImage(named: "image1"), highlightedImage: UIImage(named: "image2"))

imageView.image = UIImage(named: "image1") 

//Objective-C
[[UIImageView alloc] initWithImage:[UIImage imageNamed:@"image1"];

[[UIImageView alloc] initWithImage:[UIImage imageNamed:@"image1"] highlightedImage:[UIImage imageNamed:@"image2"]];

imageView.image = [UIImage imageNamed:@"image1"];

Alterar a cor de uma imagem #

//Swift
imageView.tintColor = UIColor.redColor()
imageView.image = imageView.image?.imageWithRenderingMode(.AlwaysTemplate)

//Swift 3
imageView.tintColor = UIColor.red
imageView.image = imageView.image?.withRenderingMode(.alwaysTemplate)

//Objective-C
imageView.tintColor = [UIColor redColor];
imageView.image = [imageView.image imageWithRenderingMode:UIImageRenderingModeAlwaysTemplate]
