Criando uma UIColor #

Há muitas maneiras de criar um UIColor:

Rápido

  • Usando uma das cores predefinidas:

    let redColor = UIColor.redColor()
let blueColor: UIColor = .blueColor()

// In Swift 3, the "Color()" suffix is removed:
let redColor = UIColor.red
let blueColor: UIColor = .blue

Se o compilador já sabe que a variável é uma instância de UIColor você pode pular o tipo todos juntos:

  let view = UIView()
  view.backgroundColor = .yellowColor()

  • Usando o valor em tons de cinza e o alfa:

    let grayscaleColor = UIColor(white: 0.5, alpha: 1.0)

  • Usando matiz, saturação, brilho e alfa:

    let hsbColor = UIColor(
    hue: 0.4,
    saturation: 0.3,
    brightness: 0.7,
    alpha: 1.0
)

  • Usando os valores RGBA:

    let rgbColor = UIColor(
    red: 30.0 / 255, 
    green: 70.0 / 255, 
    blue: 200.0 / 255, 
    alpha: 1.0
)

  • Usando uma imagem padrão:

    let patternColor = UIColor(patternImage: UIImage(named: "myImage")!)

Objetivo-C

  • Usando uma das cores predefinidas:

    UIColor *redColor = [UIColor redColor];

  • Usando o valor em tons de cinza e o alfa:

    UIColor *grayscaleColor = [UIColor colorWithWhite: 0.5 alpha: 1.0];

  • Usando matiz, saturação, brilho e alfa:

    UIColor *hsbColor = [UIColor
    colorWithHue: 0.4
    saturation: 0.3
    brightness: 0.7
    alpha: 1.0
];

  • Usando os valores RGBA:

    UIColor *rgbColor = [UIColor
    colorWithRed: 30.0 / 255.0
    green: 70.0 / 255.0
    blue: 200.0 / 255.0 
    alpha: 1.0
];

  • Usando uma imagem padrão:

    UIColor *pattenColor = [UIColor colorWithPatternImage:[UIImage imageNamed:@"myImage.png"]];

Criando uma UIColor a partir de um número hexadecimal ou string #

Você pode criar um UIColor a partir de um número ou string hexadecimal, por exemplo 0xff00cc, “#FFFFFF”

Rápido

Valor Int

extension UIColor {
    convenience init(hex: Int, alpha: CGFloat = 1.0) {
        let r = CGFloat((hex >> 16) & 0xff) / 255
        let g = CGFloat((hex >> 08) & 0xff) / 255
        let b = CGFloat((hex >> 00) & 0xff) / 255
        self.init(red: r, green: g, blue: b, alpha: alpha)
    }
}

Exemplo:

let color = UIColor(hex: 0xff00cc, alpha: 1.0)

Observe que para alpha o valor padrão de 1.0 é fornecido, então ele pode ser usado da seguinte forma:

let color = UIColor(hex: 0xff00cc)

Valor da string

extension UIColor {
    convenience init(hexCode: String) {
        let hex = hexCode.stringByTrimmingCharactersInSet(NSCharacterSet.alphanumericCharacterSet().invertedSet)
        var int = UInt32()
        NSScanner(string: hex).scanHexInt(&int)
        let a, r, g, b: UInt32

        switch hex.characters.count {
        case 3:
            (a, r, g, b) = (255, (int >> 8) * 17, (int >> 4 & 0xF) * 17, (int & 0xF) * 17)
        case 6:
            (a, r, g, b) = (255, int >> 16, int >> 8 & 0xFF, int & 0xFF)
        case 8:
            (a, r, g, b) = (int >> 24, int >> 16 & 0xFF, int >> 8 & 0xFF, int & 0xFF)
        default:
            (a, r, g, b) = (1, 1, 1, 0)
        }
    
        self.init(red: CGFloat(r) / 255, green: CGFloat(g) / 255, blue: CGFloat(b) / 255, alpha: CGFloat(a) / 255)
    }
}

- (UIColor*) hex:(NSString*)hexCode {
    
    NSString *noHashString = [hexCode stringByReplacingOccurrencesOfString:@"#" withString:@""];
    NSScanner *scanner = [NSScanner scannerWithString:noHashString];
    [scanner setCharactersToBeSkipped:[NSCharacterSet symbolCharacterSet]];
    
    unsigned hex;
    if (![scanner scanHexInt:&hex]) return nil;
    int a;
    int r;
    int g;
    int b;

    switch (noHashString.length) {
        case 3:
            a = 255;
            r = (hex >> 8) * 17;
            g = ((hex >> 4) & 0xF) * 17;
            b = ((hex >> 0) & 0xF) * 17;
            break;
        case 6:
            a = 255;
            r = (hex >> 16);
            g = (hex >> 8) & 0xFF;
            b = (hex) & 0xFF;
            break;
        case 8:
            a = (hex >> 24);
            r = (hex >> 16) & 0xFF;
            g = (hex >> 8) & 0xFF;
            b = (hex) & 0xFF;
            break;
            
        default:
            a = 255.0;
            r = 255.0;
            b = 255.0;
            g = 255.0;
            break;
    }

    return [UIColor colorWithRed:r / 255.0f green:g / 255.0f blue:b / 255.0f alpha:a / 255];
}

Métodos não documentados #

Há uma variedade de métodos não documentados em UIColor que expõem cores ou funcionalidades alternativas. Eles podem ser encontrados no [arquivo de cabeçalho privado UIColor] 1. Vou documentar o uso de dois métodos privados, styleString() e _systemDestructiveTintColor().

styleString #

Desde o iOS 2.0 existe um método de instância privada em UIColor chamado styleString que retorna uma representação de string RGB ou RGBA da cor, mesmo para cores como whiteColor fora do espaço RGB.

Objetivo-C:

@interface UIColor (Private)

- (NSString *)styleString;

@end

// ...

[[UIColor whiteColor] styleString]; // rgb(255,255,255)
[[UIColor redColor] styleString]; // rgb(255,0,0)
[[UIColor lightTextColor] styleString]; // rgba(255,255,255,0.600000)

No Swift, você pode usar um cabeçalho de ponte para expor a interface. Com Swift puro, você precisará criar um protocolo @objc com o método privado e unsafeBitCast UIColor com o protocolo:

@objc protocol  UIColorPrivate {
    func styleString() -> String
}

let white = UIColor.whiteColor()
let red = UIColor.redColor()
let lightTextColor = UIColor.lightTextColor()

let whitePrivate = unsafeBitCast(white, UIColorPrivate.self)
let redPrivate = unsafeBitCast(red, UIColorPrivate.self)
let lightTextColorPrivate = unsafeBitCast(lightTextColor, UIColorPrivate.self)

whitePrivate.styleString() // rgb(255,255,255)
redPrivate.styleString() // rgb(255,0,0)
lightTextColorPrivate.styleString() // rgba(255,255,255,0.600000)

_systemDestructiveTintColor() #

Existe um método de classe não documentado em UIColor chamado _systemDestructiveTintColor que retornará a cor vermelha usada pelos botões destrutivos do sistema:

let red = UIColor.performSelector("_systemDestructiveTintColor").takeUnretainedValue()

Ele retorna um objeto não gerenciado, no qual você deve chamar .takeUnretainedValue(), já que a propriedade da cor não foi transferida para nosso próprio objeto.

Como acontece com qualquer API não documentada, você deve ter cuidado ao tentar usar este método:

if UIColor.respondsToSelector("_systemDestructiveTintColor") {
    if let red = UIColor.performSelector("_systemDestructiveTintColor").takeUnretainedValue() as? UIColor {
        // use the color
    }
}

ou usando um protocolo:

@objc protocol UIColorPrivateStatic {
    func _systemDestructiveTintColor() -> UIColor
}

let privateClass = UIColor.self as! UIColorPrivateStatic
privateClass._systemDestructiveTintColor() // UIDeviceRGBColorSpace 1 0.231373 0.188235 1

Faz com que os atributos definidos pelo usuário apliquem o tipo de dados CGColor #

Por padrão, o Interface Builder não aceita o tipo de dados CGColor, então para permitir adicionar um CGColor usando atributos definidos pelo usuário no construtor de interface; pode-se querer usar uma extensão como esta:

Extensão Swift:

extension CALayer {
    func borderUIColor() -> UIColor? {
        return borderColor != nil ? UIColor(CGColor: borderColor!) : nil
    }
    
    func setBorderUIColor(color: UIColor) {
        borderColor = color.CGColor
    }
}

O novo atributo definido pelo usuário (borderUIColor) será reconhecido e aplicado sem problemas.

Tonalidade mais clara e mais escura de uma determinada UIColor #

O exemplo de código abaixo demonstra como você pode obter um tom mais claro e mais escuro de uma determinada cor, útil em aplicativos com temas dinâmicos

Para Cor mais escura

+ (UIColor *)darkerColorForColor:(UIColor *)c
{
    CGFloat r, g, b, a;
    if ([c getRed:&r green:&g blue:&b alpha:&a])
        return [UIColor colorWithRed:MAX(r - 0.2, 0.0)
                               green:MAX(g - 0.2, 0.0)
                                blue:MAX(b - 0.2, 0.0)
                               alpha:a];
    return nil;
}

Para Cor mais clara

+ (UIColor *)lighterColorForColor:(UIColor *)c
{
    CGFloat r, g, b, a;
    if ([c getRed:&r green:&g blue:&b alpha:&a])
        return [UIColor colorWithRed:MIN(r + 0.2, 1.0)
                               green:MIN(g + 0.2, 1.0)
                                blue:MIN(b + 0.2, 1.0)
                               alpha:a];
    return nil;
}

Brilho de cor ajustado de UIColor #

O exemplo de código abaixo fornecerá uma versão ajustada dessa cor, onde uma porcentagem maior será mais clara e uma porcentagem menor será mais escura.

Objetivo-C

+ (UIColor *)adjustedColorForColor:(UIColor *)c : (double)percent
{
    if (percent < 0) percent = 0;
    
    CGFloat r, g, b, a;
    if ([c getRed:&r green:&g blue:&b alpha:&a])
        return [UIColor colorWithRed:MAX(r * percent, 0.0)
                               green:MAX(g * percent, 0.0)
                                blue:MAX(b * percent, 0.0)
                               alpha:a];
    return nil;
}

Rápido

func adjustedColorForColor( c: UIColor, var percent: CGFloat) -> UIColor {
    if percent < 0 {
        percent = 0
    }

    var r,g,b,a: CGFloat
    r = 0.0
    g = 0.0
    b = 0.0
    a = 0.0

    if c.getRed(&r, green: &g, blue: &b, alpha: &a) {
        return UIColor(red: max(r * percent, 0.0), green: max(g * percent, 0.0), blue: max(b * percent, 0.0), alpha: a)
    }

    return UIColor()
}
