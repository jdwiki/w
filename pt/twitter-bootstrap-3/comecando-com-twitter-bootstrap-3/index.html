Instalação #

  • Download direto - link de download
  • CDN - chegue aqui
  • Bower - bower install bootstrap [ler]
  • NPM - npm install bootstrap [ler]
  • Compositor - compositor requer twbs/bootstrap [ler]
  • Personalizar - Se você tiver sua própria configuração, você pode personalizar aqui.
  • Sass - Para projetos relacionados ao Sass, você pode obtê-lo aqui.

Uso

<head>
  <link rel="stylesheet" href="path/to/bootstrap.min.css">
</head>

A referência ao arquivo bootstrap js é feita usando a tag script logo acima da tag body (veja abaixo). Observe também que o bootstrap está usando jQuery para a maioria dos widgets - como o carrossel de acordeão etc., então faça referência ao arquivo bootstrap js abaixo do arquivo jQuery js.

**Amostra** 
<!DOCTYPE html>
<html lang="en">

<head>

  <title>Form Bootstrap Example</title>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">
</head>

<body>

  <div class="container">
    <h2>Form Email</h2>
    <form role="form">
      <div class="form-group">
        <label for="email">Email:</label>
        <input type="email" class="form-control" id="email" placeholder="Enter email">
      </div>
      <div class="form-group">
        <label for="pwd">Password:</label>
        <input type="password" class="form-control" id="pwd" placeholder="Enter password">
      </div>
      <div class="checkbox">
        <label><input type="checkbox"> Remember me</label>
      </div>
      <button type="submit" class="btn btn-default">Submit</button>
    </form>
  </div>

  <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js"></script>

</body>

</html>

A página contém uma barra de navegação de largura total com links de exemplo e um controle suspenso. A barra de navegação aproveita os recursos mobile first do Bootstrap. Ele começa recolhido em visualizações móveis e se torna horizontal à medida que a largura da janela de visualização disponível aumenta.

Instalação e introdução do Bootstrap #

Introdução
Então você quer começar a usar bootstrap para o seu projeto? Excelente! então vamos começar agora mesmo!.

O que é bootstrap? Bootstrap é uma biblioteca de código aberto que você pode usar para fazer projetos incríveis responsivos usando design responsivo e código simples. Design Responsivo é uma filosofia de design onde no design do sistema (a representação e o layout) responde/adapta dependendo do layout do dispositivo. A principal razão para manter seu design responsivo é aumentar o alcance de seu aplicativo para uma base de usuários maior usando uma variedade de dispositivos.
Instalação

Informações básicas
Então você instalou o bootstrap em seu projeto. E agora é hora de começar a usar as grandes vantagens do bootstrap. Primeiro vou falar um pouco sobre o uso básico do bootstrap, depois mostrarei alguns pequenos exemplos e no final darei um exemplo de código de inicialização que você pode usar como modelo inicial!

O sistema de grade
Bootstrap usa um sistema de grade. Este sistema de grade normalmente consiste em 12 colunas. Cada uma dessas 12 colunas tem a mesma largura, mas pode ter alturas diferentes.

Portanto, temos um sistema de grade que consiste em 12 colunas. Podemos usar essas colunas para construir nosso site básico. digamos que queremos alcançar o seguinte layout:

menu - largura total
barra lateral - 1/3 da tela
Conteúdo principal 2/3 da tela
Rodapé - largura total

O cardápio
Primeiro vamos olhar para o menu. Como sabemos o sistema de grade funciona com 12 colunas. Como queremos o menu na largura total, temos que colocar o menu em todas as 12 colunas. Ficará como o exemplo abaixo

<div class="col-lg-12 col-md-12 col-sm-12 col-xs-12">
    Menu
</div>

Ao dar ao menu uma classe col-lg-12 indicamos o seguinte:
col - O col no nome da classe significa colunas.
lg - O lg no nome da classe representa a largura da tela, neste caso grande.
12 - o 12 no nome da classe representa o número de colunas que queremos que nosso menu possua. já que neste caso queremos o menu em toda a largura, obtemos todas as 12 colunas (12/12)

Como usamos 12 das 12 colunas para nosso menu, tudo depois do menu será definido em uma nova linha.

A barra lateral
O segundo item que queremos adicionar ao nosso modelo é a barra lateral. Agora, como eu disse, queremos que a barra lateral seja 1/3 da tela. Então o que vamos fazer é dividir as 12 colunas em 3. que é 4. Agora sabemos quantas colunas queremos preencher para chegar a 1/3 da tela. Siga o código abaixo.

<div class="col-lg-12 col-md-12 col-sm-12 col-xs-12">
    The menu
</div>
<div class="col-lg-4 col-md-4 col-sm-4 col-xs-4">
    The sidebar
</div>

O mesmo que com o menu só que agora nosso número de col’s difere do menu.
col - O col no nome da classe significa colunas.
lg - O lg no nome da classe representa a largura da tela, neste caso grande.
4 - o 12 no nome da classe representa o número de colunas que queremos que nossa barra lateral possua. já que neste caso queremos que a barra lateral preencha 1/3 da tela, então pegaremos apenas 4 das 12 colunas (4/12)

O conteúdo
Agora nesta linha ainda temos 8 colunas ao lado de nossa barra lateral. Então agora vamos preenchê-los com nosso conteúdo. Veja o código de exemplo abaixo

<div class="col-lg-12 col-md-12 col-sm-12 col-xs-12">
    The menu
</div>
<div class="col-lg-4 col-md-4 col-sm-4 col-xs-4">
    The sidebar
</div>
<div class="col-lg-8 col-md-8 col-sm-8 col-xs-8">
    The main content
</div>

Agora, uma vez que preenchemos as 8 colunas restantes de nossas 12 colunas nesta linha, a próxima seção começará novamente em uma nova linha com 12 colunas.

O rodapé
O rodapé é, novamente, como o menu será um bloco de largura total na tela, então pegaremos todas as 12 colunas nesta linha para o nosso rodapé, veja o código de exemplo abaixo.

<div class="col-lg-12 col-md-12 col-sm-12 col-xs-12">
    The menu
</div>
<div class="col-lg-4 col-md-4 col-sm-4 col-xs-4">
    The sidebar
</div>
<div class="col-lg-8 col-md-8 col-sm-8 col-xs-8">
    The main content
</div>
<div class="col-lg-12 col-md-12 col-sm-12 col-xs-12">
    The footer
</div>i are described below.

Então agora nós criamos, com apenas um pequeno arquivo html nosso primeiro template bootstrap. Mas isso é o básico. Normalmente, formaríamos esse código um pouco mais para fornecer a experiência completa de bootstrap. Algumas dessas experiências são descritas abaixo.

Usando linhas e contêineres Como eu disse no exemplo simples acima, o bootstrap usa linhas de 12 colunas. quando uma linha é preenchida com 12 colunas, o bootstrap começará em uma nova linha. a melhor maneira de mostrar essas linhas é usando classes de linha. Também usaremos uma classe de contêiner. isso é como uma tag body, nesse container vamos colocar todo o nosso código. Você pode escolher entre um contêiner ou uma classe de fluido de contêiner. A única diferença é que a classe container-fluid usa a largura total de uma tela e a classe container não. Um exemplo desses recursos básicos está abaixo.

<div class="container-fluid">
    <div class="row">
        <div class="col-lg-12 col-md-12 col-sm-12 col-xs-12">
            The menu consisting of 12 columns
        </div>
    </div>
    <div class="row">
        <div class="col-lg-4 col-md-4 col-sm-4 col-xs-4">
            The sidebar consisting of 4 columns
        </div>
        <div class="col-lg-8 col-md-8 col-sm-8 col-xs-8">
            The main content consisting of 8 columns
        </div>
    </div>
    <div class="row">
        <div class="col-lg-12 col-md-12 col-sm-12 col-xs-12">
            The footer consisting of 12 columns
        </div>
    </div>
</div>

Então, agora criamos um modelo de página inteira com bootstrap. É realmente muito simples, mas comece pelo básico e mais tarde você poderá usar todos os tipos de classes e funcionalidades de bootstrap. Última coisa. Os nomes das colunas lg - md - sm e xs nas classes correspondem, como eu disse para a largura da página. Grande, Médio, Pequeno e XPequeno. Não se esqueça de que você pode usar isso para estilizar colunas de maneira diferente em larguras diferentes, alterando a quantidade de colunas. Lembre-se de que uma linha completa consiste em 12 colunas!
