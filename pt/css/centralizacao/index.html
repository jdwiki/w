Usando o Flexbox #

HTML:

<div class="container">
  <img src="http://lorempixel.com/400/200" />
</div>

CSS:

html, body, .container {
  height: 100%;
}    
.container {
  display: flex;
  justify-content: center; /* horizontal center */
}
img {
  align-self: center; /* vertical center */
}

Ver resultado

HTML:

<img src="http://lorempixel.com/400/200" />

CSS:

html, body {
  height: 100%;
}   
body {
  display: flex;
  justify-content: center; /* horizontal center */
  align-items: center;     /* vertical center */
}

Ver resultado

Consulte [centralização vertical e horizontal dinâmica](https://www.wikiod.com/pt/css/layout-de-caixa-flexivel-flexbox#Centralização vertical e horizontal dinâmica (alinhar itens, justificar conteúdo) #t=201607231920579121055) na documentação do Flexbox para obter mais detalhes sobre o flexbox e o significado dos estilos.

Suporte ao navegador

Flexbox é suportado por todos os principais navegadores, exceto versões do IE anteriores a 10.

Algumas versões recentes de navegadores, como Safari 8 e IE10, exigem prefixos de fornecedor.

Para uma maneira rápida de gerar prefixos existe o Autoprefixer, uma ferramenta de terceiros.

Para navegadores mais antigos (como IE 8 e 9) um Polyfill está disponível.

Para uma visão mais detalhada do suporte ao navegador flexbox, consulte esta resposta.

Usando transformação CSS #

As transformações CSS são baseadas no tamanho dos elementos, portanto, se você não souber a altura ou a largura do seu elemento, poderá posicioná-lo absolutamente 50% da parte superior e esquerda de um contêiner relativo e traduzi-lo por 50% para a esquerda e para cima para centralizá-lo verticalmente e horizontalmente.

Tenha em mente que com esta técnica, o elemento pode acabar sendo renderizado em um limite de pixel não inteiro, fazendo com que pareça desfocado. Consulte esta resposta em SO para obter uma solução alternativa.

HTML

<div class="container">
  <div class="element"></div>
</div>

CSS

.container {
  position: relative;
}

.element {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

Ver exemplo no JSFiddle

COMPATIBILIDADE ENTRE NAVEGADORES #

A propriedade transform precisa que os prefixos sejam suportados por navegadores mais antigos. Prefixos são necessários para Chrome<=35, Safari<=8, Opera<=22, navegador Android<=4.4.4 e IE9. As transformações CSS não são suportadas pelo IE8 e versões anteriores.

Aqui está uma declaração de transformação comum para o exemplo anterior:

-webkit-transform: translate(-50%, -50%); /* Chrome, Safari, Opera, Android */
    -ms-transform: translate(-50%, -50%); /* IE 9 */
        transform: translate(-50%, -50%);

Para obter mais informações, consulte canIuse.

MAIS INFORMAÇÕES #

  • O elemento está sendo posicionado de acordo com o primeiro pai não estático (position: relative, absolute ou fixed). Explore mais neste fiddle e neste tópico de documentação.

  • Para centralização somente horizontal, use left: 50% e transform: translateX(-50%). O mesmo vale para centralização somente vertical: center com top: 50% e transform: translateY(-50%).

  • Usar elementos de largura/altura não estáticos com este método de centralização pode fazer com que o elemento centralizado pareça esmagado. Isso acontece principalmente com elementos contendo texto e pode ser corrigido adicionando: margin-right: -50%; e margin-bottom: -50%;. Veja este fiddle para mais informações.

5: https://www.wikiod.com/pt/css/posicionamento#Posição Absoluta 6: https://jsfiddle.net/4xxmxca0/

Usando margem: 0 auto; #

Os objetos podem ser centralizados usando margin: 0 auto; se forem elementos de bloco e tiverem uma largura definida.

HTML

<div class="containerDiv">
    <div id="centeredDiv"></div>
</div>

<div class="containerDiv">
    <p id="centeredParagraph">This is a centered paragraph.</p>
</div>

<div class="containerDiv">
    <img id="centeredImage" src="https://i.kinja-img.com/gawker-media/image/upload/s--c7Q9b4Eh--/c_scale,fl_progressive,q_80,w_800/qqyvc3bkpyl3mfhr8all.jpg" />
</div>

CSS

.containerDiv {
    width: 100%;
    height: 100px;
    padding-bottom: 40px;
}

#centeredDiv {
    margin: 0 auto;
    width: 200px;
    height: 100px;
    border: 1px solid #000;
}

#centeredParagraph {
    width: 200px;
    margin: 0 auto;
}

#centeredImage {
    display: block;
    width: 200px;
    margin: 0 auto;
}

Resultado:

Exemplo JSFiddle: Centralizando objetos com margem: 0 auto;

Usando alinhamento de texto #

O tipo de centralização mais comum e fácil é o de linhas de texto em um elemento. CSS tem a regra text-align: center para este propósito:

HTML

<p>Lorem ipsum</p>

CSS

p {
    text-align: center;
}

Isso não funciona para centralizar elementos de bloco inteiros. text-align controla apenas o alinhamento do conteúdo embutido como texto em seu elemento de bloco pai.

Veja mais sobre text-align na seção Typography.

Usando posição: absoluto #

Trabalhando em navegadores antigos (IE >= 8)

As margens automáticas, combinadas com valores de zero para os deslocamentos left e right ou top e bottom, centralizarão um elemento absolutamente posicionado dentro de seu pai.

Ver resultado

HTML

<div class="parent">
  <img class="center" src="http://lorempixel.com/400/200/" />
</div>

CSS

.parent {
  position: relative;
  height: 500px;
}

.center {
  position: absolute;
  margin: auto;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
}

Elementos que não possuem largura e altura implícitas como as imagens, precisarão desses valores definidos.

Outros recursos: Centralização absoluta em CSS

Usando calc() #

A função calc() é parte de uma nova sintaxe em CSS3 na qual você pode calcular (matematicamente) que tamanho/posição seu elemento ocupa usando uma variedade de valores como pixels, porcentagens, etc. Nota:- Sempre que você usar esta função , sempre cuide do espaço entre dois valores calc(100% - 80px).

CSS

.center {
    position: absolute;
    height: 50px;
    width: 50px;
    background: red;
    top: calc(50% - 50px / 2); /* height divided by 2*/
    left: calc(50% - 50px / 2); /* width divided by 2*/
}

HTML

<div class="center"></div>

Alinha verticalmente qualquer coisa com 3 linhas de código #

Suportado pelo IE11+

Ver resultado

Use essas 3 linhas para alinhar verticalmente praticamente tudo. Apenas certifique-se de que a div/imagem à qual você aplica o código tenha um pai com altura.

CSS

div.vertical {
  position: relative;
  top: 50%;
  transform: translateY(-50%);
}

HTML

<div class="vertical">Vertical aligned text!</div>

Usando altura da linha #

Você também pode usar line-height para centralizar verticalmente uma única linha de texto dentro de um contêiner:

CSS

div {
    height: 200px;
    line-height: 200px;
}

Isso é muito feio, mas pode ser útil dentro de um elemento <input />. A propriedade line-height funciona apenas quando o texto a ser centralizado abrange uma única linha. Se o texto for dividido em várias linhas, a saída resultante não será centralizada.

Técnica do elemento fantasma (hack de Michał Czernow) #

Essa técnica funciona mesmo quando as dimensões do contêiner são desconhecidas.

Configure um elemento “fantasma” dentro do contêiner a ser centralizado com 100% de altura e, em seguida, use vertical-align: middle tanto nele quanto no elemento a ser centralizado.

CSS

/* This parent can be any width and height */
.block {
  text-align: center;

  /* May want to do this if there is risk the container may be narrower than the element inside */
  white-space: nowrap;
}
 
/* The ghost element */
.block:before {
  content: '';
  display: inline-block;
  height: 100%;
  vertical-align: middle;

  /* There is a gap between ghost element and .centered,
  caused by space character rendered. Could be eliminated by
  nudging .centered (nudge distance depends on font family),
  or by zeroing font-size in .parent and resetting it back
  (probably to 1rem) in .centered. */
  margin-right: -0.25em;
}

/* The element to be centered, can also be of any width and height */ 
.centered {
  display: inline-block;
  vertical-align: middle;
  width: 300px;
  white-space: normal; /* Resetting inherited nowrap behavior */
}

HTML

<div class="block">
  <div class="centered"></div>
</div>

Centralização em relação a outro item #

Veremos como centralizar o conteúdo com base na altura de um elemento próximo.

Compatibilidade: IE8+, todos os outros navegadores modernos.

HTML

<div class="content">
  <div class="position-container">
    <div class="thumb">
      <img src="http://lorempixel.com/400/200/">
    </div>
    <div class="details">
      <p class="banner-title">text 1</p>
      <p class="banner-text">content content content content content content content content content content content content content content</p>
      <button class="btn">button</button>
    </div>
  </div>
</div>

CSS

.content * {
  box-sizing: border-box;
}
.content .position-container {
  display: table;
}
.content .details {
  display: table-cell;
  vertical-align: middle;
  width: 33.333333%;
  padding: 30px;
  font-size: 17px;
  text-align: center;
}
.content .thumb {
  width: 100%;
}
.content .thumb img {
  width: 100%;
}

Link para JSFiddle

Os pontos principais são os 3 containers .thumb, .details e .position-container:

  • O .position-container deve ter display: table.
  • O .details deve ter a largura real definida como width: .... e display: table-cell, vertical-align: middle.
  • O .thumb deve ter width: 100% se você quiser que ocupe todo o espaço restante e será influenciado pela largura do .details.
  • A imagem (se você tiver uma imagem) dentro de .thumb deve ter width: 100%, mas não é necessário se você tiver as proporções corretas.

Alinha verticalmente uma imagem dentro de div #

HTML

<div class="wrap">
    <img src="http://lorempixel.com/400/200/" />
</div>

CSS

.wrap {
    height: 50px;/* max image height */
    width: 100px;
    border: 1px solid blue;
    text-align: center;
}
.wrap:before {
  content:"";
  display: inline-block;
  height: 100%;
  vertical-align: middle;
  width: 1px;
}

img {
    vertical-align: middle;
}

Centralização vertical e horizontal sem se preocupar com altura ou largura #

A técnica a seguir permite adicionar seu conteúdo a um elemento HTML e centralizá-lo horizontal e verticalmente sem se preocupar com sua altura ou largura.

O recipiente externo #

  • deve ter display: table;

O contêiner interno #

  • deve ter display: table-cell;
  • deve ter vertical-align: middle;
  • deve ter text-align: center;

A caixa de conteúdo #

  • deve ter display: inline-block;
  • deve reajustar o alinhamento horizontal do texto para, por exemplo. text-align: left; ou text-align: right;, a menos que você queira que o texto seja centralizado

Demonstração #

HTML

<div class="outer-container">
   <div class="inner-container">
     <div class="centered-content">
        You can put anything here!
     </div>
   </div>
</div>

CSS

body {
    margin : 0;
}

.outer-container {
    position : absolute;
    display: table;
    width: 100%; /* This could be ANY width */
    height: 100%; /* This could be ANY height */
    background: #ccc;
}

.inner-container {
    display: table-cell;
    vertical-align: middle;
    text-align: center;
}

.centered-content {
    display: inline-block;
    text-align: left;
    background: #fff;
    padding: 20px;
    border: 1px solid #000;
}

Centralização com tamanho fixo #

Se o tamanho do seu conteúdo for fixo, você pode usar o posicionamento absoluto para 50% com margin que reduz metade da largura e altura do seu conteúdo:

HTML

<div class="center">
    Center vertically and horizontally
</div>

CSS

.center {
    position: absolute;
    background: #ccc;

    left: 50%;
    width: 150px;
    margin-left: -75px;  /* width * -0.5 */

    top: 50%;
    height: 200px;
    margin-top: -100px;  /* height * -0.5 */
}

Centralização horizontal apenas com largura fixa #

Você pode centralizar o elemento horizontalmente mesmo que não saiba a altura do conteúdo:

HTML

<div class="center">
    Center only horizontally
</div>

CSS

.center {
    position: absolute;
    background: #ccc;

    left: 50%;
    width: 150px;
    margin-left: -75px;  /* width * -0.5 */
}

Centragem vertical com altura fixa #

Você pode centralizar o elemento verticalmente se souber a altura do elemento:

HTML

<div class="center">
    Center only vertically
</div>

CSS

.center {
    position: absolute;
    background: #ccc;

    top: 50%;
    height: 200px;
    margin-top: -100px;  /* width * -0.5 */
}

Alinha verticalmente elementos dinâmicos de altura #

Aplicar CSS intuitivamente não produz os resultados desejados porque

  • vertical-align:middle não é aplicável para elementos de nível de bloco
  • margin-top:auto e margin-bottom:auto valores usados seria computado como zero
  • margin-top:-50% valores de margem baseados em porcentagem são calculado em relação à largura do bloco contendo

Para um suporte mais amplo ao navegador, uma solução alternativa com elementos auxiliares:

HTML

<div class="vcenter--container">
  <div class="vcenter--helper">
    <div class="vcenter--content">
      <!--stuff-->
    </div>
  </div>
</div>

CSS

.vcenter--container {
  display: table;
  height: 100%;
  position: absolute;
  overflow: hidden;
  width: 100%;
}
.vcenter--helper {
  display: table-cell;
  vertical-align: middle;
}
.vcenter--content {
  margin: 0 auto;
  width: 200px;
}

jsfiddle da pergunta original. Esta abordagem

  • trabalha com elementos dinâmicos de altura
  • respeita o fluxo de conteúdo
  • é suportado por navegadores legados

Centralização horizontal e vertical usando o layout da mesa #

Pode-se facilmente centralizar um elemento filho usando a propriedade de exibição table.

HTML

<div class="wrapper">
    <div class="parent">
        <div class="child"></div>
    </div>
</div>

CSS

.wrapper { 
    display: table; 
    vertical-align: center; 
    width: 200px; 
    height: 200px; 
    background-color: #9e9e9e; 
}
.parent { 
    display: table-cell; 
    vertical-align: middle; 
    text-align: center; 
}
.child { 
    display: inline-block; 
    vertical-align: middle; 
    text-align: center; 
    width: 100px; 
    height: 100px; 
    background-color: teal; 
}
