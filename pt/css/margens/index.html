Sintaxe #

  • margem: <superior & direita & inferior & esquerda>;
  • margem: <top>, <left & right>, <bottom>;
  • margem: <superior e inferior>, <esquerda e direita>;
  • margem: <top>, <right>, <bottom>, <left>;
  • margem superior: <top>;
  • margem direita: <direita>;
  • margem inferior: <fundo>;
  • margem esquerda: <esquerda>;

Parâmetros #

Parâmetro Detalhes
0 definir margem para nenhum
automóvel usado para centrar, definindo valores uniformemente em cada lado
unidades (por exemplo, px) veja a seção de parâmetros em Unidades para uma lista de unidades válidas
herdar herdar valor de margem do elemento pai
inicial restaurar ao valor inicial

Mais sobre “Recolher margens”: aqui.

Recolhimento de margem #

Quando duas margens se tocam verticalmente, elas são recolhidas. Quando duas margens se tocam horizontalmente, elas não colapsam.

Exemplo de margens verticais adjacentes:

Considere os seguintes estilos e marcação:

div{
    margin: 10px;
}

<div>
    some content
</div>
<div>
    some more content
</div>

Eles estarão separados por 10px, pois as margens verticais colapsam sobre um e outro. (O espaçamento não será a soma de duas margens.)

Exemplo de margens horizontais adjacentes:

Considere os seguintes estilos e marcação:

span{
    margin: 10px;
}

<span>some</span><span>content</span>

Eles estarão separados por 20px, pois as margens horizontais não colapsam sobre uma e outra. (O espaçamento será a soma de duas margens.)

Sobreposição com tamanhos diferentes

.top{
    margin: 10px;
}
.bottom{
    margin: 15px;
}

<div class="top">
    some content
</div>
<div class="bottom">
    some more content
</div>

Esses elementos serão espaçados verticalmente em 15px. As margens se sobrepõem o máximo possível, mas a margem maior determinará o espaçamento entre os elementos.

Margem sobreposta, pegadinha

.outer-top{
    margin: 10px;
}
.inner-top{
    margin: 15px;
}
.outer-bottom{
    margin: 20px;
}
.inner-bottom{
    margin: 25px;
}

<div class="outer-top">
    <div class="inner-top">
        some content
    </div>
</div>
<div class="outer-bottom">
    <div class="inner-bottom">
        some more content
    </div>
</div>

Qual será o espaçamento entre os dois textos? (passe o mouse para ver a resposta)

!O espaçamento será de 25px. Como todas as quatro margens estão se tocando, elas entrarão em colapso, usando a maior margem das quatro.

Agora, e se adicionarmos algumas bordas à marcação acima.

div{
    border: 1px solid red;
}

Qual será o espaçamento entre os dois textos? (passe o mouse para ver a resposta)

!O espaçamento será de 59px! Agora, apenas as margens de .outer-top e .outer-bottom se tocam e são as únicas margens recolhidas. As margens restantes são separadas pelas bordas. Então temos 1px + 10px + 1px + 15px + 20px + 1px + 25px + 1px. (Os 1px são as bordas…)

Recolher margens entre elementos pai e filho:

HTML:

<h1>Title</h1>
<div>
  <p>Paragraph</p>
</div>

CSS

h1 {
  margin: 0;
  background: #cff;
}
div {
  margin: 50px 0 0 0;
  background: #cfc;
}
p {
  margin: 25px 0 0 0;
  background: #cf9;
}

No exemplo acima, apenas a maior margem se aplica. Você pode ter esperado que o parágrafo estaria localizado a 60px do h1 (já que o elemento div tem um topo de margem de 40px e o p tem um topo de margem de 20px). Isso não acontece porque as margens colapsam juntas para formar uma margem.

Aplicar margem em um determinado lado #

Propriedades específicas de direção #

CSS permite que você especifique um determinado lado para aplicar a margem. As quatro propriedades fornecidas para este fim são:

  • margem esquerda
  • margem direita
  • margem superior
  • margin-bottom

O código a seguir aplicaria uma margem de 30 pixels ao lado esquerdo da div selecionada. Ver resultado

HTML

<div id="myDiv"></div>

CSS

#myDiv {
    margin-left: 30px;
    height: 40px;
    width: 40px;
    background-color: red;
}
Parâmetro Detalhes
margem esquerda A direção na qual a margem deve ser aplicada.
30px A largura da margem.

Especificando a direção usando a propriedade abreviada #

A propriedade padrão margin pode ser expandida para especificar larguras diferentes para cada lado dos elementos selecionados. A sintaxe para fazer isso é a seguinte:

margin: <top> <right> <bottom> <left>;

O exemplo a seguir aplica uma margem de largura zero na parte superior da div, uma margem de 10px no lado direito, uma margem de 50px no lado esquerdo e uma margem de 100px no lado esquerdo. Ver resultado

HTML

<div id="myDiv"></div>

CSS

#myDiv {
    margin: 0 10px 50px 100px;
    height: 40px;
    width: 40px;
    background-color: red;
}

Simplificação da propriedade de margem #

p {
    margin:1px;                /* 1px margin in all directions */
    
    /*equals to:*/
    
    margin:1px 1px;
    
    /*equals to:*/
    
    margin:1px 1px 1px;
    
    /*equals to:*/
    
    margin:1px 1px 1px 1px;
}

Outro exemplo:

p{
    margin:10px 15px;        /* 10px margin-top & bottom And 15px margin-right & left*/
    
    /*equals to:*/
    
    margin:10px 15px 10px 15px;
    
    /*equals to:*/
    
    margin:10px 15px 10px;
    /* margin left will be calculated from the margin right value (=15px) */
}

Centralize horizontalmente os elementos em uma página usando a margem #

Desde que o elemento seja um bloco e tenha um valor de largura definido explicitamente, as margens podem ser usadas para centralizar os elementos do bloco em uma página horizontalmente.

Adicionamos um valor de largura menor que a largura da janela e a propriedade auto de margin distribui o espaço restante para a esquerda e para a direita:

#myDiv {
 width:80%;
 margin:0 auto;
}

No exemplo acima, usamos a declaração abreviada margin para primeiro definir 0 para os valores das margens superior e inferior (embora possa ser qualquer valor) e, em seguida, usamos auto para permitir que o navegador aloque o espaço automaticamente para o valores das margens esquerda e direita.

No exemplo acima, o elemento #myDiv está definido para 80% de largura, o que deixa o uso de 20% de sobra. O navegador distribui esse valor para os lados restantes assim:

(100% - 80%) / 2 = 10%

Margens negativas #

Margin é uma das poucas propriedades CSS que podem ser definidas para valores negativos. Esta propriedade pode ser usada para sobrepor elementos sem posicionamento absoluto.

div{
  display: inline;
}

#over{
  margin-left: -20px;
}

<div>Base div</div>
<div id="over">Overlapping div</div>
