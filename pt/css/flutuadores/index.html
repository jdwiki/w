Sintaxe #

  • claro: nenhum | esquerda | direito | ambos | início em linha | fim em linha;
  • flutuar: esquerda | direito | nenhum | início em linha | fim em linha;

Como o float implica o uso do layout do bloco, ele modifica o valor calculado dos valores de exibição em alguns casos 1

1: https://developer.mozilla.org/en-US/docs/Web/CSS/float MDN

Flutuar uma imagem dentro do texto #

O uso mais básico de um float é ter texto em volta de uma imagem. O código abaixo produzirá dois parágrafos e uma imagem, com o segundo parágrafo fluindo ao redor da imagem. Observe que é sempre conteúdo depois do elemento flutuante que flui ao redor do elemento flutuante.

HTML:

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio. Praesent libero. Sed cursus ante dapibus diam. Sed nisi. Nulla quis sem at nibh elementum imperdiet. Duis sagittis ipsum. Praesent mauris. Fusce nec tellus sed augue semper porta. Mauris massa. Vestibulum lacinia arcu eget nulla. </p>

<img src="http://lorempixel.com/200/100/" />

<p>Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos. Curabitur sodales ligula in libero. Sed dignissim lacinia nunc. Curabitur tortor. Pellentesque nibh. Aenean quam. In scelerisque sem at dolor. Maecenas mattis. Sed convallis tristique sem. Proin ut ligula vel nunc egestas porttitor. Morbi lectus risus, iaculis vel, suscipit quis, luctus non, massa. Fusce ac turpis quis ligula lacinia aliquet. </p>

CSS:

img {
  float:left;
  margin-right:1rem;
}

limpar propriedade #

A propriedade clear está diretamente relacionada aos floats. Valores de propriedade:

  • nenhum - Padrão. Permite elementos flutuantes em ambos os lados

  • left - Não são permitidos elementos flutuantes no lado esquerdo

  • right - Nenhum elemento flutuante permitido no lado direito

  • ambos - Não são permitidos elementos flutuantes no lado esquerdo ou direito

  • inicial - Define esta propriedade para seu valor padrão. Leia sobre inicial

  • inherit - Herda esta propriedade de seu elemento pai. Leia sobre herdar

Clearfix #

O hack do clearfix é uma forma popular de conter floats (N. Gallagher aka @necolas)

Não deve ser confundido com a propriedade [clear](https://www.wikiod.com/pt/css/flutuadores#limpar propriedade), clearfix é um conceito (que também está relacionado a floats, daí a possível confusão). Para conter floats, você deve adicionar a classe .cf ou .clearfix no container (o pai) e estilizar esta classe com algumas regras descritas abaixo.

3 versões com efeitos ligeiramente diferentes (fontes :A new micro clearfix hack por N. Gallagher e [clearfix reloaded](http://yuiblog.com/ blog/2010/09/27/clearfix-reloaded-overflowhidden-desmystified/) por T. J. Koblentz):

Clearfix (com o colapso da margem superior dos floats contidos ainda ocorrendo) #

.cf:after {
    content: "";
    display: table;
}

.cf:after {
    clear: both;
}

Clearfix também previne o colapso da margem superior de floats contidos #

/**
 * For modern browsers
 * 1. The space content is one way to avoid an Opera bug when the
 *    contenteditable attribute is included anywhere else in the document.
 *    Otherwise it causes space to appear at the top and bottom of elements
 *    that are clearfixed.
 * 2. The use of `table` rather than `block` is only necessary if using
 *    `:before` to contain the top-margins of child elements.
 */
.cf:before,
.cf:after {
    content: " "; /* 1 */
    display: table; /* 2 */
}

.cf:after {
    clear: both;
}

Clearfix com suporte a navegadores desatualizados IE6 e IE7 #

.cf:before,
.cf:after {
    content: " ";
    display: table;
}

.cf:after {
    clear: both;
}

/**
 * For IE 6/7 only
 * Include this rule to trigger hasLayout and contain floats.
 */
.cf {
    *zoom: 1;
}

Codepen mostrando efeito clearfix

Outro recurso: Tudo o que você sabe sobre clearfix está errado (clearfix e BFC - Block Formatting Context enquanto hasLayout está relacionado a navegadores desatualizados IE6 talvez 7)

DIV em linha usando float #

O div é um elemento de nível de bloco, ou seja, ocupa toda a largura da página e os irmãos são colocados um abaixo do outro, independentemente de sua largura.

<div>
    <p>This is DIV 1</p>
</div>
<div>
    <p>This is DIV 2</p>
</div>

A saída do código a seguir será

Podemos torná-los in-line adicionando uma propriedade CSS float ao div.

HTML:

<div class="outer-div">
    <div class="inner-div1">
        <p>This is DIV 1</p>
    </div>
    <div class="inner-div2">
        <p>This is DIV 2</p>
    </div>
</div>

CSS

.inner-div1 {
    width: 50%;
    margin-right:0px;
    float:left;
    background : #337ab7;
    padding:50px 0px;
}
 
.inner-div2 {
    width: 50%;
    margin-right:0px;
    float:left;
    background : #dd2c00;
    padding:50px 0px;
}
 
p {
    text-align:center;
}

Layout simples de duas colunas de largura fixa #

Um layout simples de duas colunas consiste em dois elementos flutuantes de largura fixa. Observe que a barra lateral e a área de conteúdo não têm a mesma altura neste exemplo. Essa é uma das partes complicadas com layouts de várias colunas usando flutuadores e requer soluções alternativas para fazer com que várias colunas pareçam ter a mesma altura.

HTML:

<div class="wrapper">

<div class="sidebar">
  <h2>Sidebar</h2>

  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio.</p>
</div>

<div class="content">
  <h1>Content</h1>

  <p>Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos. Curabitur sodales ligula in libero. Sed dignissim lacinia nunc. Curabitur tortor. Pellentesque nibh. Aenean quam. In scelerisque sem at dolor. Maecenas mattis. Sed convallis tristique sem. Proin ut ligula vel nunc egestas porttitor. Morbi lectus risus, iaculis vel, suscipit quis, luctus non, massa. Fusce ac turpis quis ligula lacinia aliquet. </p>
</div>

</div>

CSS:

.wrapper {
  width:600px;
  padding:20px;
  background-color:pink;

  /* Floated elements don't use any height. Adding "overflow:hidden;" forces the
     parent element to expand to contain its floated children. */
  overflow:hidden;
}

.sidebar {
  width:150px;
  float:left;
  background-color:blue;
}

.content {
  width:450px;
  float:right;
  background-color:yellow;
}

Layout simples de três colunas de largura fixa #

HTML:

<div class="wrapper">
  <div class="left-sidebar">
    <h1>Left Sidebar</h1>
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. </p>
  </div>
  <div class="content">
    <h1>Content</h1>
    <p>Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos. Curabitur sodales ligula in libero. Sed dignissim lacinia nunc. Curabitur tortor. Pellentesque nibh. Aenean quam. In scelerisque sem at dolor. Maecenas mattis. Sed convallis tristique sem. Proin ut ligula vel nunc egestas porttitor. Morbi lectus risus, iaculis vel, suscipit quis, luctus non, massa. </p>
  </div>
  <div class="right-sidebar">
    <h1>Right Sidebar</h1>
    <p>Fusce ac turpis quis ligula lacinia aliquet.</p>
  </div>
</div>

CSS:

.wrapper {
  width:600px;
  background-color:pink;
  padding:20px;

  /* Floated elements don't use any height. Adding "overflow:hidden;" forces the
     parent element to expand to contain its floated children. */
  overflow:hidden;
}

.left-sidebar {
  width:150px;
  background-color:blue;
  float:left;
}

.content {
  width:300px;
  background-color:yellow;
  float:left;
}

.right-sidebar {
  width:150px;
  background-color:green;
  float:right;
}

Layout preguiçoso/ganancioso de duas colunas #

Esse layout usa uma coluna flutuante para criar um layout de duas colunas sem larguras definidas. Neste exemplo, a barra lateral esquerda é “preguiçosa”, pois ocupa apenas o espaço necessário. Outra maneira de dizer isso é que a barra lateral esquerda é “encolhida”. A coluna de conteúdo à direita é “gananciosa”, pois ocupa todo o espaço restante.

HTML:

<div class="sidebar">
<h1>Sidebar</h1>
<img src="http://lorempixel.com/150/200/" />
</div>

<div class="content">
<h1>Content</h1>
<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio. Praesent libero. Sed cursus ante dapibus diam. Sed nisi. Nulla quis sem at nibh elementum imperdiet. Duis sagittis ipsum. Praesent mauris. Fusce nec tellus sed augue semper porta. Mauris massa. Vestibulum lacinia arcu eget nulla. </p>
<p>Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos. Curabitur sodales ligula in libero. Sed dignissim lacinia nunc. Curabitur tortor. Pellentesque nibh. Aenean quam. In scelerisque sem at dolor. Maecenas mattis. Sed convallis tristique sem. Proin ut ligula vel nunc egestas porttitor. Morbi lectus risus, iaculis vel, suscipit quis, luctus non, massa. Fusce ac turpis quis ligula lacinia aliquet. Mauris ipsum. Nulla metus metus, ullamcorper vel, tincidunt sed, euismod in, nibh. </p>
</div>

CSS:

.sidebar {
  /* `display:table;` shrink-wraps the column */
  display:table;
  float:left;
  background-color:blue;
}

.content {
  /* `overflow:hidden;` prevents `.content` from flowing under `.sidebar` */
  overflow:hidden;
  background-color:yellow;
}

Uso da propriedade overflow para limpar floats #

Definir o valor de overflow para hidden,auto ou scroll para um elemento, limpará todos os floats dentro desse elemento.

Observação: usar overflow:scroll sempre mostrará a caixa de rolagem
