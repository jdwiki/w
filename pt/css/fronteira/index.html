raio da borda #

A propriedade border-radius permite alterar a forma do modelo de caixa básico.

Cada canto de um elemento pode ter até dois valores, para o raio vertical e horizontal desse canto (para um máximo de 8 valores).

O primeiro conjunto de valores define o raio horizontal. O segundo conjunto opcional de valores, precedido por um ‘/’ , define o raio vertical. Se apenas um conjunto de valores for fornecido, ele será usado para o raio vertical e horizontal.

border-radius: 10px 5% / 20px 25em 30px 35em;

O 10px é o raio horizontal do canto superior esquerdo e inferior direito. E o 5% é o raio horizontal do canto superior direito e inferior esquerdo. Os outros quatro valores após ‘/’ são os raios verticais para canto superior esquerdo, canto superior direito, canto inferior direito e canto inferior esquerdo.

Tal como acontece com muitas propriedades CSS, atalhos podem ser usados ​​para qualquer ou todos valores possíveis. Portanto, você pode especificar qualquer coisa de um a oito valores. A abreviação a seguir permite definir o raio horizontal e vertical de cada canto com o mesmo valor:

HTML:

<div class='box'></div>

CSS:

.box {
    width: 250px;
    height: 250px;
    background-color: black;
    border-radius: 10px;
}

Border-radius é mais comumente usado para converter elementos de caixa em círculos. Ao definir o raio da borda para metade do comprimento de um elemento quadrado, um elemento circular é criado:

.circle {
    width: 200px;
    height: 200px;
    border-radius: 100px;
}

Como o border-radius aceita porcentagens, é comum usar 50% para evitar o cálculo manual do valor do border-radius:

.circle {
    width: 150px;
    height: 150px;
    border-radius: 50%;
}

Se as propriedades de largura e altura não forem iguais, a forma resultante será uma oval em vez de um círculo.

Exemplo de raio de borda específico do navegador:

  -webkit-border-top-right-radius: 4px;
  -webkit-border-bottom-right-radius: 4px;
  -webkit-border-bottom-left-radius: 0;
  -webkit-border-top-left-radius: 0;
  -moz-border-radius-topright: 4px;
  -moz-border-radius-bottomright: 4px;
  -moz-border-radius-bottomleft: 0;
  -moz-border-radius-topleft: 0;
  border-top-right-radius: 4px;
  border-bottom-right-radius: 4px;
  border-bottom-left-radius: 0;
  border-top-left-radius: 0;

estilo de borda #

A propriedade border-style define o estilo da borda de um elemento. Esta propriedade pode ter de um a quatro valores (para cada lado do elemento um valor).

Exemplos:

border-style: dotted;

border-style: dotted solid double dashed;

border-style também pode ter os valores none e hidden. Eles têm o mesmo efeito, exceto que hidden funciona para resolução de conflitos de borda para elementos <table>. Em uma <table> com várias bordas, none tem a prioridade mais baixa (significando em um conflito, a borda seria exibida) e hidden tem a prioridade mais alta (ou seja, em um conflito, a borda não seria exibida) .

Várias bordas #

Usando contorno:

.div1{
  border: 3px solid black; 
  outline: 6px solid blue;
  width: 100px;
  height: 100px;
  margin: 20px;
}

Usando box-shadow:

.div2{
  border: 5px solid green; 
  box-shadow: 0px 0px 0px 4px #000;
  width: 100px;
  height: 100px;
  margin: 20px;
}

Usando um pseudo elemento:

.div3 {
  position: relative;
  border: 5px solid #000;
  width: 100px;
  height: 100px;
  margin: 20px;
}
.div3:before {
  content: " ";
  position: absolute;
  border: 5px solid blue;
  z-index: -1;
  top: 5px;
  left: 5px;
  right: 5px;
  bottom: 5px;
}

borda (abreviaturas) #

Na maioria dos casos você deseja definir várias propriedades de borda (border-width, border-style e border-color) para todos os lados de um elemento.

Em vez de escrever:

border-width: 1px;
border-style: solid;
border-color: #000;

Você pode simplesmente escrever:

border: 1px solid #000;

Esses atalhos também estão disponíveis para todos os lados de um elemento: border-top, border-left, border-right e border-bottom. Então você pode fazer:

border-top: 2px double #aaaaaa;

imagem-borda #

Com a propriedade border-image você tem a possibilidade de definir uma imagem a ser usada ao invés de estilos de borda normais.

Uma border-image consiste essencialmente em um

  • border-image-source: O caminho para a imagem a ser usada
  • border-image-slice: especifica o deslocamento usado para dividir a imagem em nove regiões (quatro cantos, quatro bordas e um meio)
  • border-image-repeat: Especifica como as imagens para os lados e o meio da imagem da borda são dimensionadas

Considere o seguinte exemplo onde border.png é uma imagem de 90x90 pixels:

border-image: url("border.png") 30 trecho;

A imagem será dividida em nove regiões com 30x30 pixels. As bordas serão usadas como os cantos da borda, enquanto o lado será usado no meio. Se o elemento for maior/mais largo que 30px, esta parte da imagem será esticada. A parte do meio da imagem padrão é transparente.

colapso de borda #

A propriedade border-collapse aplica-se apenas a tables (e elementos exibidos como display: table ou inline-table) e define se as bordas da tabela são recolhidas em uma única borda ou desanexadas como no HTML padrão.

table {
  border-collapse: separate; /* default */
  border-spacing: 2px; /* Only works if border-collapse is separate */
}

Consulte também a entrada de documentação [Tabelas - colapso de borda](https://www.wikiod.com/pt/css/tabelas#colapso de fronteira)

Criando uma borda multicolorida usando border-image #

CSS #

.bordered {
  border-image: linear-gradient(to right, red 20%, green 20%, green 40%, blue 40%, blue 60%, maroon 60%, maroon 80%, chocolate 80%); /* gradient with required colors */
  border-image-slice: 1;
}

HTML #

<div class='bordered'>Border on all sides</div>

O exemplo acima produziria uma borda composta por 5 cores diferentes. As cores são definidas através de um linear-gradient (você pode encontrar mais informações sobre gradientes nos [docs](https://www.wikiod.com/pt/css/planos-de-fundo#Gradientes de fundo )). Você pode encontrar mais informações sobre a propriedade border-image-slice no exemplo border-image na mesma página .

(Nota: Propriedades adicionais foram adicionadas ao elemento para fins de apresentação.)

Você deve ter notado que a borda esquerda tem apenas uma cor (a cor inicial do gradiente), enquanto a borda direita também tem apenas uma cor (a cor final do gradiente). Isso ocorre devido à maneira como a propriedade da imagem da borda funciona. É como se o gradiente fosse aplicado a toda a caixa e, em seguida, as cores fossem mascaradas das áreas de preenchimento e conteúdo, fazendo com que pareça que apenas a borda possui o gradiente.

Quais bordas têm uma única cor depende da definição do gradiente. Se o gradiente for um gradiente to right, a borda esquerda seria a cor inicial do gradiente e a borda direita seria a cor final. Se fosse um gradiente ‘para baixo’, a borda superior seria a cor inicial do gradiente e a borda inferior seria a cor final. Abaixo está a saída de um gradiente de 5 cores to bottom.

Se a borda for necessária apenas em lados específicos do elemento, a propriedade border-width pode ser usada como qualquer outra borda normal. Por exemplo, adicionar o código abaixo produziria uma borda apenas na parte superior do elemento.

border-width: 5px 0px 0px 0px;

Observe que, qualquer elemento que tenha a propriedade border-image não respeitará o border-radius (isto é, a borda não será curva). Isso é baseado na declaração abaixo na especificação:

Os fundos de uma caixa, mas não sua imagem de borda, são recortados na curva apropriada (conforme determinado por ‘background-clip’).

borda-[esquerda|direita|superior|inferior] #

A propriedade border-[left|right|top|bottom] é usada para adicionar uma borda a um lado específico de um elemento.

Por exemplo, se você quiser adicionar uma borda ao lado esquerdo de um elemento, você pode fazer:

#element {
    border-left: 1px solid black;
}
