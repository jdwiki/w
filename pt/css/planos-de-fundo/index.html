Com CSS você pode definir cores, gradientes e imagens como plano de fundo de um elemento.

É possível especificar várias combinações de imagens, cores e gradientes e ajustar o tamanho, posicionamento e repetição (entre outros) destes.

Sintaxe #

  • cor de fundo: cor | transparente | inicial | herdar;
  • imagem de fundo: url | nenhum | inicial | herdar;
  • posição de fundo: valor;
  • tamanho do plano de fundo: <bg-size> [<bg-size>]
  • <tamanho bg>: auto | comprimento | capa | conter | inicial | herdar;
  • repetição de fundo: repetir | repetir-x | repetir-y | sem repetição | inicial | herdar;
  • origem-fundo: caixa de preenchimento | caixa-borda | caixa de conteúdo | inicial | herdar;
  • background-clip: border-box | caixa de enchimento | caixa de conteúdo | inicial | herdar;
  • anexo de fundo: scroll | fixo | locais | inicial | herdar;
  • fundo: bg-color bg-image position / bg-size bg-repeat bg-origin bg-clip bg-attachment inicial | herdar;
  • Os gradientes CSS3 não funcionarão em versões do Internet Explorer inferiores a 10.

Cor de fundo #

A propriedade background-color define a cor de fundo de um elemento usando um valor de cor ou por meio de palavras-chave, como transparent, inherit ou initial.

  • transparent, especifica que a cor de fundo deve ser transparente. Este é o padrão.

  • inherit, herda esta propriedade de seu elemento pai.

  • initial, configura esta propriedade para seu valor padrão.

Isso pode ser aplicado a todos os elementos e ::first-letter/::first-line pseudo-elements.

As cores em CSS podem ser especificadas por diferentes métodos.

Nomes de cores #

CSS

div {
  background-color: red;  /* red */
}

HTML

<div>This will have a red background</div>
  • O exemplo usado acima é uma das várias maneiras que o CSS tem para representar uma única cor.

Códigos de cores hexadecimais #

O código hexadecimal é usado para denotar componentes RGB de uma cor em notação hexadecimal de base 16. #ff0000, por exemplo, é vermelho brilhante, onde o componente vermelho da cor é 256 bits (ff) e as partes verdes e azuis correspondentes da cor são 0 (00).

Se ambos os valores em cada um dos três pares RGB (R, G e B) forem iguais, o código de cor pode ser reduzido em três caracteres (o primeiro dígito de cada par). #ff0000 pode ser abreviado para #f00, e #ffffff pode ser abreviado para #fff.

A notação hexadecimal não diferencia maiúsculas de minúsculas.

body {
  background-color: #de1205; /* red */
}

.main {
  background-color: #00f; /* blue */
}

RGB / RGBa #

Outra maneira de declarar uma cor é usar RGB ou RGBa.

RGB significa Vermelho, Verde e Azul, e requer três valores separados entre 0 e 255, colocados entre colchetes, que correspondem aos valores decimais de cor para respectivamente vermelho, verde e azul.

RGBa permite adicionar um parâmetro alfa adicional entre 0,0 e 1,0 para definir a opacidade.

header {
  background-color: rgb(0, 0, 0); /* black */
}

footer {
  background-color: rgba(0, 0, 0, 0.5); /* black with 50% opacity */
}

HSL / HSLa #

Outra maneira de declarar uma cor é usar HSL ou HSLa e é semelhante a RGB e RGBa.

HSL significa matiz, saturação e leveza, e também é frequentemente chamado de HLS:

  • Matiz é um grau na roda de cores (de 0 a 360).
  • A saturação é uma porcentagem entre 0% e 100%.
  • A leveza também é uma porcentagem entre 0% e 100%.

O HSLa permite adicionar um parâmetro alfa adicional entre 0,0 e 1,0 para definir a opacidade.

li a {
  background-color: hsl(120, 100%, 50%); /* green */
}

#p1 {
  background-color: hsla(120, 100%, 50%, .3); /* green with 30% opacity */
}

Interação com a imagem de fundo #

As seguintes afirmações são todas equivalentes:

body {
  background: red;
  background-image: url(partiallytransparentimage.png);
}

body {
  background-color: red;
  background-image: url(partiallytransparentimage.png);
}

body {
  background-image: url(partiallytransparentimage.png);
  background-color: red;
}

body {
  background: red url(partiallytransparentimage.png);
}

Todos eles farão com que a cor vermelha seja mostrada abaixo da imagem, onde as partes da imagem são transparentes, ou a imagem não está aparecendo (talvez como resultado de background-repeat).

Observe que o seguinte não é equivalente:

body {
  background-image: url(partiallytransparentimage.png);
  background: red;
}

Aqui, o valor de background substitui sua background-image.

Para obter mais informações sobre a propriedade background, consulte https://www.wikiod.com/pt/css/planos-de-fundo#Abreviação de fundo

Gradientes de fundo #

Gradientes são novos tipos de imagem, adicionados em CSS3. Como uma imagem, os gradientes são definidos com a propriedade [background-image](https://www.wikiod.com/pt/css/planos-de-fundo#Imagem de fundo) ou a abreviação background.

Existem dois tipos de funções de gradiente, lineares e radiais. Cada tipo tem uma variante não repetitiva e uma variante repetitiva:

  • gradiente linear()
  • repeating-linear-gradient()
  • gradiente radial()
  • repetindo-radial-gradiente()

Gradiente linear() #

Um gradiente linear tem a seguinte sintaxe

background: linear-gradient( <direction>?, <color-stop-1>, <color-stop-2>, ...);
Valor Significado
<direção> Pode ser um argumento como to top, to bottom, to right ou to left; ou um ângulo como 0deg, 90deg… . O ângulo começa de cima para baixo e gira no sentido horário. Pode ser especificado em deg, grad, rad ou turn . Se omitido, o gradiente flui de cima para baixo
<lista de paradas de cores> Lista de cores, opcionalmente seguida de cada uma por uma porcentagem ou comprimento para exibi-la. Por exemplo, amarelo 10%, rgba(0,0,0,.5) 40px, #fff 100%…

Por exemplo, isso cria um gradiente linear que começa da direita e faz a transição de vermelho para azul

.linear-gradient {
  background: linear-gradient(to left, red, blue); /* you can also use 270deg */
}

Você pode criar um gradiente diagonal declarando uma posição inicial horizontal e vertical.

.diagonal-linear-gradient {
  background: linear-gradient(to left top, red, yellow 10%);
}

É possível especificar qualquer número de paradas de cor em um gradiente, separando-as com vírgulas. Os exemplos a seguir criarão um gradiente com 8 paradas de cor

.linear-gradient-rainbow {
  background: linear-gradient(to left, red, orange, yellow, green, blue, indigo, violet)
}

radial-gradiente() #

.radial-gradient-simple {
  background: radial-gradient(red, blue);
}

.radial-gradient {
  background: radial-gradient(circle farthest-corner at top left, red, blue);
}
Valor Significado
círculo Forma de gradiente. Os valores são círculo ou elipse, o padrão é elipse.
canto mais distante Palavras-chave que descrevem quão grande deve ser a forma final. Os valores são closest-side, farthest-side, closest-corner, farthest-corner
superior esquerdo Define a posição do centro do gradiente, da mesma forma que background-position.

Repetindo gradientes #

As funções de gradiente repetidas usam os mesmos argumentos dos exemplos acima, mas colocam o gradiente lado a lado no plano de fundo do elemento.

.bullseye {
  background: repeating-radial-gradient(red, red 10%, white 10%, white 20%);
}
.warning {
  background: repeating-linear-gradient(-45deg, yellow, yellow 10%, black 10%, black 20% );
}
Valor Significado
-45deg Unidade angular. O ângulo começa de cima para baixo e gira no sentido horário. Pode ser especificado em deg, grad, rad ou turn .
à esquerda Direção do gradiente, o padrão é para baixo. Sintaxe: to [eixo y(topo OR inferior)] [eixo x(esquerdo OU direito)] ou seja, topo direito
amarelo 10% Cor, opcionalmente seguida por uma porcentagem ou comprimento para exibi-la. Repetido duas ou mais vezes.

Observe que os códigos de cores HEX, RGB, RGBa, HSL e HSLa podem ser usados ​​em vez de nomes de cores. Os nomes das cores foram usados ​​para fins de ilustração. Observe também que a sintaxe do gradiente radial é muito mais complexa do que o gradiente linear, e uma versão simplificada é mostrada aqui. Para obter uma explicação completa e especificações, consulte os MDN Docs

Imagem de fundo #

A propriedade background-image é usada para especificar uma imagem de fundo a ser aplicada a todos os elementos correspondentes. Por padrão, esta imagem é lado a lado para cobrir todo o elemento, excluindo a margem.

.myClass {
  background-image: url('/path/to/image.jpg');
}

Para usar várias imagens como background-image, defina url() separado por vírgulas

.myClass {
  background-image: url('/path/to/image.jpg'),
                    url('/path/to/image2.jpg');
}

As imagens serão empilhadas de acordo com sua ordem com a primeira imagem declarada em cima das outras e assim por diante.

Valor | Resultado —————————-|——————— ———– url('/path/to/image.jpg') | Especifique o(s) caminho(s) da imagem de fundo ou um recurso de imagem especificado com esquema de URI de dados (apóstrofos podem ser omitidos), separe os múltiplos por vírgula nenhum | Sem imagem de fundo inicial | Valor padrão herdar | Herdar o valor dos pais

Mais CSS para imagem de fundo

Esses atributos a seguir são muito úteis e quase essenciais também.

background-size:     xpx ypx | x% y%;
background-repeat:   no-repeat | repeat | repeat-x | repeat-y;
background-position: left offset (px/%) right offset (px/%) | center center | left top | right bottom;

Abreviação de fundo #

A propriedade background pode ser usada para definir uma ou mais propriedades relacionadas ao plano de fundo:

Valor Descrição Versão CSS
imagem de fundo Imagem de fundo para usar 1+
cor de fundo Cor de fundo para aplicar 1+
posição de fundo Posição da imagem de fundo 1+
tamanho de fundo Tamanho da imagem de fundo 3+
repetição de fundo Como repetir a imagem de fundo 1+
origem de fundo Como o fundo é posicionado (ignorado quando background-attachment é fixo) 3+
background-clip Como o plano de fundo é pintado em relação à content-box, border-box ou à padding-box 3+
fundo-anexo Como a imagem de fundo se comporta, se ela rola junto com o bloco que a contém ou tem uma posição fixa dentro da viewport 1+
inicial Configura a propriedade para valor como padrão 3+
herdar Herda o valor da propriedade do pai 2+

A ordem dos valores não importa e cada valor é opcional

Sintaxe #

A sintaxe da declaração abreviada em segundo plano é:

background: [<background-image>] [<background-color>]  [<background-position>]/[<background-size>] [<background-repeat>] [<background-origin>] [<background-clip>] [<background-attachment>] [<initial|inherit>];

Exemplos #

background: red;

Basta definir uma background-color com o valor red.

background: border-box red;

Definir um background-clip para border-box e um background-color para red.

background: no-repeat center url("somepng.jpg");

Define um background-repeat para não repetir, background-origin para o centro e uma background-image para uma imagem.

background: url('pattern.png') green;

Neste exemplo, o background-color do elemento seria definido como green com pattern.png, se estiver disponível, sobreposto na cor, repetindo quantas vezes forem necessárias para preencher o elemento. Se o pattern.png incluir qualquer transparência, a cor verde será visível atrás dela.

background: #000000 url("picture.png") top left / 600px auto no-repeat;

Neste exemplo temos um fundo preto com uma imagem ‘picture.png’ na parte superior, a imagem não se repete em nenhum dos eixos e é posicionada no canto superior esquerdo. O / após a posição deve incluir o tamanho da imagem de fundo que neste caso é definida como 600px largura e auto para a altura. Este exemplo pode funcionar bem com uma imagem de recurso que pode desbotar em uma cor sólida.

OBSERVAÇÃO: o uso da propriedade abreviada de plano de fundo redefine todos os valores de propriedade de plano de fundo definidos anteriormente, mesmo que um valor não seja fornecido. Se você deseja apenas modificar um valor de propriedade de plano de fundo definido anteriormente, use uma propriedade longa.

Tamanho do plano de fundo #

Visao geral #

A propriedade background-size permite controlar a escala da background-image. Leva até dois valores, que determinam a escala/tamanho da imagem resultante na direção vertical e horizontal. Se a propriedade estiver ausente, ela será considerada auto em width e height.

auto manterá a proporção da imagem, se puder ser determinada. A altura é opcional e pode ser considerada auto. Portanto, em uma imagem de 256 px × 256 px, todas as configurações de background-size a seguir produziriam uma imagem com altura e largura de 50 px:

background-size: 50px;
background-size: 50px auto; /* same as above */
background-size: auto 50px;
background-size: 50px 50px;

Então, se começarmos com a imagem a seguir (que tem o tamanho mencionado de 256 px × 256 px),

terminaremos com um 50 px × 50 px na tela do usuário, contido no fundo do nosso elemento:

Pode-se também usar valores percentuais para dimensionar a imagem em relação ao elemento. O exemplo a seguir produziria uma imagem desenhada de 200 px × 133 px:

#withbackground {
    background-image: url(to/some/background.png);

    background-size: 100% 66%;
    
    width: 200px;
    height: 200px;

    padding: 0;
    margin: 0;
}

O comportamento depende da background-origin.

Mantendo a proporção #

O último exemplo na seção anterior perdeu sua proporção original. O círculo se transformou em uma elipse, o quadrado em um retângulo, o triângulo em outro triângulo.

A abordagem de comprimento ou porcentagem não é flexível o suficiente para manter a proporção em todos os momentos. auto não ajuda, pois você pode não saber qual dimensão do seu elemento será maior. No entanto, para cobrir completamente certas áreas com uma imagem (e proporção correta) ou para conter uma imagem com proporção correta completamente em uma área de plano de fundo, os valores contain e cover fornecem a funcionalidade adicional.

Eggsplanation para contain e cover #

Desculpe o trocadilho ruim, mas vamos usar uma foto do dia de Biswarup Ganguly para demonstração. Vamos dizer que esta é a sua tela e a área cinza está fora da tela visível. Para demonstração, vamos supor uma proporção de 16 × 9.

Queremos usar a imagem do dia acima mencionada como pano de fundo. No entanto, cortamos a imagem para 4x3 por algum motivo. Poderíamos definir a propriedade background-size para algum tamanho fixo, mas vamos nos concentrar em contain e cover. Observe que também assumo que não desmontamos a largura e/ou a altura de body.

conter #

conter Dimensione a imagem, preservando sua proporção intrínseca (se houver), para o tamanho maior, de modo que tanto a largura quanto a altura possam caber dentro da área de posicionamento do plano de fundo.

Isso garante que a imagem de fundo esteja sempre completamente contida na área de posicionamento do plano de fundo, no entanto, pode haver algum espaço vazio preenchido com sua cor de fundo neste caso:

capa #

capa Dimensione a imagem, preservando sua proporção intrínseca (se houver), para o menor tamanho, de modo que tanto a largura quanto a altura possam cobrir completamente a área de posicionamento do plano de fundo.

Isso garante que a imagem de fundo esteja cobrindo tudo. Não haverá ‘cor de fundo’ visível, no entanto, dependendo da proporção da tela, grande parte da sua imagem pode ser cortada:

Demonstração com código real #

div > div {
  background-image: url(http://i.stack.imgur.com/r5CAq.jpg);
  background-repeat: no-repeat;
  background-position: center center;
  background-color: #ccc;
  border: 1px solid;
  width: 20em;
  height: 10em;
}
div.contain {
  background-size: contain;
}
div.cover {
  background-size: cover;
}

Posição de fundo #

A propriedade background-position é usada para especificar a posição inicial de uma imagem de fundo ou gradiente

.myClass {
  background-image: url('path/to/image.jpg');
  background-position: 50% 50%;
}

A posição é definida usando uma coordenada X e Y e pode ser definida usando qualquer uma das unidades usadas no CSS.

Unidade Descrição
valor% valor% Uma porcentagem para o deslocamento horizontal é relativa a (largura da área de posicionamento do plano de fundo - largura da imagem de plano de fundo).
Uma porcentagem para o deslocamento vertical é relativa a (altura da área de posicionamento do plano de fundo - altura da imagem de plano de fundo)
O tamanho da imagem é o tamanho dado por background-size.
valorpx valorpx Desloca a imagem de fundo por um comprimento dado em pixels em relação ao canto superior esquerdo da área de posicionamento de fundo

Unidades em CSS podem ser especificadas por diferentes métodos (veja aqui).

Propriedades de posição de fundo à mão #

Além da propriedade abreviada acima, também é possível usar as propriedades de fundo longhand background-position-x e background-position-y. Estes permitem que você controle as posições x ou y separadamente.

NOTA: Isso é suportado em todos os navegadores, exceto Firefox (versões 31-48) 2. O Firefox 49, a ser lançado em setembro de 2016, irá oferecer suporte a essas propriedades. Até lá, há um hack do Firefox nesta resposta do Stack Overflow.

A propriedade background-origin #

A propriedade background-origin especifica onde a imagem de fundo está posicionada.

Nota: Se a propriedade background-attachment for definida como fixed, esta propriedade não terá efeito.

Valores possíveis:

  • padding-box - A posição é relativa à caixa de preenchimento
  • border-box - A posição é relativa à caixa de borda

Anexo de fundo #

A propriedade background-attachment define se uma imagem de fundo é fixa ou rola com o restante da página.

body { 
  background-image: url('img.jpg');
  background-attachment: fixed;
}
Valor Descrição
rolar O plano de fundo rola junto com o elemento. Este é o padrão.
fixo O plano de fundo é fixo em relação à viewport.
locais O plano de fundo rola junto com o conteúdo do elemento.
inicial Configura esta propriedade para seu valor padrão.
herdar Herda esta propriedade de seu elemento pai.

Exemplos #

anexo em segundo plano: scroll #

O comportamento padrão, quando o corpo é rolado, o plano de fundo rola com ele:

body {
  background-image: url('image.jpg');
  background-attachment: scroll;
}

anexo em segundo plano: fixo #

A imagem de fundo será fixa e não se moverá quando o corpo for rolado:

body {
  background-image: url('image.jpg');
  background-attachment: fixed;
}

anexo em segundo plano: local #

A imagem de fundo da div rolará quando o conteúdo da div for rolado.

div {
  background-image: url('image.jpg');
  background-attachment: local;
}

Várias imagens de fundo #

Em CSS3, podemos empilhar vários planos de fundo no mesmo elemento.

#mydiv {
  background-image: url(img_1.png), /* top image */
                    url(img_2.png), /* middle image */
                    url(img_3.png); /* bottom image */
  background-position: right bottom,
                       left top,
                       right top;
  background-repeat: no-repeat,
                     repeat,
                     no-repeat;
}

As imagens serão empilhadas umas sobre as outras com o primeiro plano de fundo na parte superior e o último plano de fundo na parte de trás. img_1 estará na parte superior, o img_2 e img_3 na parte inferior.

Também podemos usar a propriedade abreviada de fundo para isso:

#mydiv {
  background: url(img_1.png) right bottom no-repeat,
              url(img_2.png) left top repeat,
              url(img_3.png) right top no-repeat;
}

Também podemos empilhar imagens e gradientes:

#mydiv {
  background: url(image.png) right bottom no-repeat,
              linear-gradient(to bottom, #fff 0%,#000 100%);
}

Clipe de fundo #

Definição e Uso: A propriedade background-clip especifica a área de pintura do plano de fundo.

Valor padrão: border-box

Valores

  • border-box é o valor padrão. Isso permite que o plano de fundo se estenda até a borda externa da borda do elemento.

Fundo de repetição #

A propriedade background-repeat define se/como uma imagem de fundo será repetida.

Por padrão, uma imagem de fundo é repetida verticalmente e horizontalmente.

div {
  background-image: url("img.jpg");
  background-repeat: repeat-y;
}

Cor de fundo com opacidade #

Se você definir opacity em um elemento, isso afetará todos os seus elementos filhos. Para definir uma opacidade apenas no fundo de um elemento, você terá que usar cores RGBA. O exemplo a seguir terá um fundo preto com opacidade de 0,6.

/* Fallback for web browsers that don't support RGBa */
background-color: rgb(0, 0, 0);

/* RGBa with 0.6 opacity */
background-color: rgba(0, 0, 0, 0.6);

/* For IE 5.5 - 7*/
filter: progid:DXImageTransform.Microsoft.gradient(startColorstr=#99000000, endColorstr=#99000000);

/* For IE 8*/
-ms-filter: "progid:DXImageTransform.Microsoft.gradient(startColorstr=#99000000, endColorstr=#99000000)";
