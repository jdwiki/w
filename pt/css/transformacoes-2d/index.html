Sintaxe #

  • Girar Transformar
  • transformar: girar(<ângulo>)
  • Traduzir transformação
  • transform: translate(<comprimento-ou-porcentagem> [, <comprimento-ou-porcentagem>]?)
  • transform: translateX(<comprimento-ou-porcentagem>)
  • transform: traduzirY(<comprimento-ou-porcentagem>)
  • Transformação inclinada
  • transform: skew(<ângulo> [, <ângulo>]?)
  • transform: skewX(<ângulo>)
  • transform: skewY(<ângulo>)
  • Transformação de escala
  • transform: scale(<fator de escala> [, <fator de escala>]?)
  • transform: scaleX(<fator de escala>)
  • transform: escalaY(<fator de escala>)
  • Transformação de matriz
  • transform: matriz(<número> [, <número> ]{5,5})

Parâmetros #

Função/Parâmetro Detalhes
girar(x) Define uma transformação que move o elemento em torno de um ponto fixo no eixo Z
translate(x,y) Move a posição do elemento nos eixos X e Y
translateX(x) Move a posição do elemento no eixo X
translateY(y) Move a posição do elemento no eixo Y
escala(x,y) Modifica o tamanho do elemento nos eixos X e Y
scaleX(x) Modifica o tamanho do elemento no eixo X
escalaY(y) Modifica o tamanho do elemento no eixo Y
inclinar(x,y) Mapeamento de cisalhamento, ou transvecção, distorcendo cada ponto de um elemento por um certo ângulo em cada direção
inclinaçãoX(x) Mapeamento de cisalhamento horizontal distorcendo cada ponto de um elemento em um determinado ângulo na direção horizontal
skewY(y) Mapeamento de cisalhamento vertical distorcendo cada ponto de um elemento em um determinado ângulo na direção vertical
matriz() Define uma transformação 2D na forma de uma matriz de transformação.
ângulo O ângulo pelo qual o elemento deve ser girado ou inclinado (dependendo da função com a qual é usado). O ângulo pode ser fornecido em graus (deg), gradianos (grad), radianos (rad) ou voltas (turn). Na função skew(), o segundo ângulo é opcional. Se não for fornecido, não haverá (0) inclinação no eixo Y.
comprimento ou porcentagem A distância expressa como um comprimento ou uma porcentagem pela qual o elemento deve ser traduzido. Na função translate(), o segundo comprimento ou porcentagem é opcional. Se não for fornecido, não haverá tradução (0) no eixo Y.
fator de escala Um número que define quantas vezes o elemento deve ser dimensionado no eixo especificado. Na função scale(), o segundo fator de escala é opcional. Se não for fornecido, o primeiro fator de escala também será aplicado ao eixo Y.

Sistema de coordenação 2D #

As transformações são feitas de acordo com um sistema de coordenação 2D X/Y. O eixo X vai da direita para a esquerda e o eixo Y vai para baixo conforme mostrado na imagem a seguir:

Assim, um translateY() positivo vai para baixo e um translateX() positivo vai para a direita.

Suporte e prefixos do navegador #

  • O IE suporta esta propriedade desde o IE9 com o prefixo -ms-. Versões mais antigas e Edge não precisam do prefixo
  • Firefox suporta desde a versão 3.5 e precisa do prefixo -moz- até a versão 15
  • O Chrome desde a versão 4 e até a versão 34 precisa do prefixo -webkit-
  • O Safari precisa do prefixo -webkit- até a versão 8
  • O Opera precisa do prefixo -o- para a versão 11.5 e o prefixo -webkit- da versão 15 para 22
  • O Android precisa do prefixo -webkit- da versão 2.1 a 4.4.4

Exemplo de transformação prefixada: #

-webkit-transform: rotate(45deg);
    -ms-transform: rotate(45deg);
        transform: rotate(45deg);

Girar #

HTML

<div class="rotate"></div>

CSS

.rotate {
    width: 100px;
    height: 100px;
    background: teal;
    transform: rotate(45deg);
}

Este exemplo irá girar o div em 45 graus no sentido horário. O centro de rotação está no centro da div, 50% da esquerda e 50% da parte superior. Você pode alterar o centro de rotação configurando a propriedade transform-origin.

transform-origin: 100% 50%;

O exemplo acima irá definir o centro de rotação para o meio da extremidade do lado direito.

Escala #

HTML

<div class="scale"></div>

CSS

.scale {
    width: 100px;
    height: 100px;
    background: teal;
    transform: scale(0.5, 1.3);
}

Este exemplo dimensionará o div para 100px * 0.5 = 50px no eixo X e para 100px * 1.3 = 130px no eixo Y.

O centro da transformação está no centro da div, 50% da esquerda e 50% da parte superior.

Inclinar #

HTML

<div class="skew"></div>

CSS

.skew {
    width: 100px;
    height: 100px;
    background: teal;
    transform: skew(20deg, -30deg);
}

Este exemplo irá inclinar a div em 20 graus no eixo X e em -30 graus no eixo Y. O centro da transformação está no centro da div, 50% da esquerda e 50% da parte superior.

Veja o resultado aqui.

Transformações múltiplas #

Várias transformações podem ser aplicadas a um elemento em uma propriedade como esta:

transform: rotate(15deg) translateX(200px);

Isso irá girar o elemento 15 graus no sentido horário e depois traduzi-lo 200px para a direita.

Em transformações encadeadas, o sistema de coordenadas se move com o elemento. Isso significa que a translação não será horizontal, mas em um eixo gire 15 graus no sentido horário, conforme mostrado na imagem a seguir:

Alterar a ordem das transformações alterará a saída. O primeiro exemplo será diferente de

transform: translateX(200px) rotate(15deg);

<div class="transform"></div>

.transform {
  transform: rotate(15deg) translateX(200px);
}

Como mostrado nesta imagem:

Traduzir #

HTML

<div class="translate"></div>

CSS

.translate {
    width: 100px;
    height: 100px;
    background: teal;
    transform: translate(200px, 50%);
}

Este exemplo moverá o div em 200px no eixo X e em 100px * 50% = 50px no eixo Y.

Você também pode especificar traduções em um único eixo.

No eixo X:

.translate {
    transform: translateX(200px);
}

No eixo Y:

.translate {
    transform: translateY(50%);
}

Origem da transformação #

As transformações são feitas em relação a um ponto que é definido pela propriedade transform-origin.

A propriedade tem 2 valores: transform-origin: X Y;

No exemplo a seguir, o primeiro div (.tl) é girado em torno do canto superior esquerdo com transform-origin: 0 0; e o segundo (.tr) é transformado em torno do canto superior direito com transform- origem: 100% 0. A rotação é aplicada ao passar o mouse :

HTML:

<div class="transform originl"></div>
<div class="transform origin2"></div>

CSS:

.transform {
    display: inline-block;
    width: 200px;
    height: 100px;
    background: teal;
    transition: transform 1s;
}

.origin1 {
    transform-origin: 0 0;
}

.origin2 {
    transform-origin: 100% 0;
}

.transform:hover {
    transform: rotate(30deg);
}

O valor padrão para a propriedade transform-origin é 50% 50%, que é o centro do elemento.
