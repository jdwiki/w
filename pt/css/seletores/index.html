Seletores CSS identificam elementos HTML específicos como alvos para estilos CSS. Este tópico aborda como os seletores CSS visam elementos HTML. Os seletores usam uma ampla variedade de mais de 50 métodos de seleção oferecidos pela linguagem CSS, incluindo elementos, classes, IDs, pseudoelementos e pseudoclasses e padrões.

Sintaxe #

  • #Eu iria

  • .nome da classe

  • :pseudo-classname

  • ::pseudo-elementname

  • [attr] /* tem o atributo attr. */

  • [attr="value"] /* tem o atributo attr e seu valor é exatamente “value”. */

  • [attr~="value"] /* tem o atributo attr, e seu valor, quando dividido em whitespace, contém “value”. */

  • [attr|="value"] /* tem o atributo attr, e seu valor é exatamente “value”, ou seu valor começa com “value-” . */

  • [attr^="value"] /* tem o atributo attr e seu valor começa com “value”. */

  • [attr$="value"] /* tem o atributo attr e seu valor termina com “value”. */

  • [attr*="value"] /* tem o atributo attr e seu valor contém “value”. */

  • nome do elemento

  • Às vezes você verá dois pontos duplos (::) em vez de apenas um (:). Esta é uma maneira de separar pseudo-classes de pseudo-elementos.

  • Navegadores antigos, como o Internet Explorer 8, somente suportam dois pontos (:) para definir pseudo-elementos.

  • Ao contrário das pseudo-classes, apenas um pseudoelemento pode ser usado por seletor, se presente deve aparecer após a sequência de seletores simples que representa os assuntos do seletor (uma versão futura da especificação W3C pode permitir vários pseudo-elementos por seletor).

Seletores de atributos #

Visão geral #

Os seletores de atributo podem ser usados ​​com vários tipos de operadores que alteram os critérios de seleção de acordo. Eles selecionam um elemento usando a presença de um determinado atributo ou valor de atributo.

Seletor(1) Elemento correspondente Seleciona elementos… Versão CSS
[attr] <div attr> Com atributo attr 2
[attr='val'] <div attr="val"> Onde o atributo attr tem o valor val 2
[attr~='val'] <div attr="val val2 val3"> Onde val aparece na
lista separada por espaços em branco de attr		 2
[attr^='val'] <div attr="val1 val2"> Onde o valor de attr começa com val 3
[attr$='val'] <div attr="sth aval"> Onde o valor do attr termina com val 3
[attr*='val'] <div attr="somevalhere"> Onde attr contém val em qualquer lugar 3
[attr|=‘val’] <div attr="val-sth etc"> Onde o valor de attr é exatamente val,
ou começa com val e imediatamente
seguido por - (U+002D)		 2
[attr=‘val’ i] <div attr=“val”> Onde attr tem o valor val,
ignorando as letras maiúsculas de val.		 4(2)

Notas:

  1. O valor do atributo pode estar entre aspas simples ou aspas duplas. Nenhuma citação também pode funcionar, mas não é válida de acordo com o padrão CSS e é desencorajada.

  2. Não existe uma especificação CSS4 única e integrada, pois ela é dividida em módulos separados. No entanto, existem módulos de “nível 4”. Consulte o suporte do navegador.

Detalhes #

[atributo] #

Seleciona elementos com o atributo fornecido.

div[data-color] {
  color: red;
}

<div data-color="red">This will be red</div>
<div data-color="green">This will be red</div>
<div data-background="red">This will NOT be red</div>

Demonstração ao vivo no JSBin

[attribute="value"] #

Seleciona elementos com o atributo e valor fornecidos.

div[data-color="red"] {
  color: red;
}

<div data-color="red">This will be red</div>
<div data-color="green">This will NOT be red</div>
<div data-color="blue">This will NOT be red</div>

Demonstração ao vivo no JSBin

[atributo*="valor"] #

Seleciona elementos com o atributo e valor fornecidos onde o atributo fornecido contém o valor fornecido em qualquer lugar (como uma substring).

[class*="foo"] {
  color: red;
}

<div class="foo-123">This will be red</div>
<div class="foo123">This will be red</div>
<div class="bar123foo">This will be red</div>
<div class="barfooo123">This will be red</div>
<div class="barfo0">This will NOT be red</div>

Demonstração ao vivo no JSBin

[atributo~="valor"] #

Seleciona elementos com o atributo e valor fornecidos onde o valor fornecido aparece em uma lista separada por espaços em branco.

[class~="color-red"] {
  color: red;
}

<div class="color-red foo-bar the-div">This will be red</div>
<div class="color-blue foo-bar the-div">This will NOT be red</div>

Demonstração ao vivo no JSBin

[attribute^="value"] #

Seleciona elementos com o atributo e o valor fornecidos, onde o atributo fornecido começa com o valor.

[class^="foo-"] {
  color: red;
}

<div class="foo-123">This will be red</div>
<div class="foo-234">This will be red</div>
<div class="bar-123">This will NOT be red</div>

Demonstração ao vivo no JSBin

[attribute$="value"] #

Seleciona elementos com o atributo e o valor fornecidos, onde o atributo especificado termina com o valor fornecido.

[class$="file"] {
  color: red;
}

<div class="foobar-file">This will be red</div>
<div class="foobar-file">This will be red</div>
<div class="foobar-input">This will NOT be red</div>

Demonstração ao vivo no JSBin

[atributo|="valor"] #

Seleciona elementos com um determinado atributo e valor onde o valor do atributo é exatamente o valor dado ou é exatamente o valor dado seguido por - (U+002D)

[lang|="EN"] {
  color: red;
}

<div lang="EN-us">This will be red</div>
<div lang="EN-gb">This will be red</div>
<div lang="PT-pt">This will NOT be red</div>

Demonstração ao vivo no JSBin

[attribute="valor" i] #

Seleciona elementos com um determinado atributo e valor onde o valor do atributo pode ser representado como Value, VALUE, vAlUe ou qualquer outra possibilidade que não diferencia maiúsculas de minúsculas.

[lang="EN" i] {
  color: red;
}

<div lang="EN">This will be red</div>
<div lang="en">This will be red</div>
<div lang="PT">This will NOT be red</div>

Demonstração ao vivo no JSBin

Especificidade dos seletores de atributos #

0-1-0 #

O mesmo que seletor de classe e pseudoclasse.

*[type=checkbox] // 0-1-0

Observe que isso significa que um seletor de atributo pode ser usado para selecionar um elemento por seu ID em um nível de especificidade mais baixo do que se ele fosse selecionado com um seletor de ID: [id="my-ID"] targets o mesmo elemento que #my-ID, mas com menor especificidade.

Consulte a Seção de Sintaxe para obter mais detalhes.

2: https://www.wikiod.com/pt/css/seletores#Seletores de ID

Seletores básicos #

Seletor Descrição
* Seletor universal (todos os elementos)
div Seletor de tags (todos os elementos <div>)
.azul Seletor de classe (todos os elementos com classe blue)
.azul.vermelho Todos os elementos com classe blue e red (um tipo de seletor composto)
#título Seletor de ID (o elemento com o atributo “id” definido como headline)
:pseudo-classe Todos os elementos com pseudo-classe
::pseudo-elemento Elemento que corresponde ao pseudoelemento
:lang(pt) Elemento que corresponde à declaração :lang, por exemplo <lang span="en">
div > p seletor filho

Observação: o valor de um ID deve ser exclusivo em uma página da web. É uma violação do padrão HTML usar o valor de um ID mais de uma vez na mesma árvore de documentos .

Uma lista completa de seletores pode ser encontrada na especificação de seletores CSS nível 3.

3: https://www.wikiod.com/pt/css/seletores#Seletores de atributos 4: https://www.wikiod.com/pt/css/seletores#Combinadores 5: https://www.wikiod.com/pt/css/pseudo-elementos

Combinadores #

Visão geral #

Seletor Descrição
div span Seletor descendente (todos os <span>s que são descendentes de um <div>)
div > intervalo Seletor filho (todos os <span>s que são filhos diretos de um <div>)
um ~ intervalo General Sibling selector (todos os <span>s que são irmãos após um <a>)
a + intervalo Seletor de irmãos adjacentes (todos os <span>s que estão imediatamente após um <a>)

Observação: seletores irmãos direcionam os elementos que vêm depois deles no documento de origem. O CSS, por sua natureza (em cascata), não pode segmentar elementos anteriores ou pais. No entanto, usando a propriedade flex order, um seletor irmão anterior pode ser simulado em mídia visual.

Combinador Descendente: selector selector #

Um combinador descendente, representado por pelo menos um caractere de espaço ( ), seleciona elementos que são descendentes do elemento definido. Este combinador seleciona todos descendentes do elemento (de elementos filhos para baixo).

div p {
  color:red;
}

<div>
  <p>My text is red</p>
  <section>
    <p>My text is red</p>
  </section>
</div>

<p>My text is not red</p>

Demonstração ao vivo no JSBin

No exemplo acima, os dois primeiros elementos <p> são selecionados, pois ambos são descendentes do <div>.

Combinador filho: seletor > seletor #

O combinador filho (>) é usado para selecionar elementos que são filhos ou descendentes diretos do elemento especificado.

div > p {
  color:red;
}

<div>
  <p>My text is red</p>
  <section>
    <p>My text is not red</p>
  </section>
</div>

Demonstração ao vivo no JSBin

O CSS acima seleciona apenas o primeiro elemento <p>, pois é o único parágrafo diretamente descendente de um <div>.

O segundo elemento <p> não está selecionado porque não é um filho direto do <div>.

Combinador de irmãos adjacentes: seletor + seletor #

O combinador adjacente (+) seleciona um elemento irmão que segue imediatamente um elemento especificado.

p + p {
  color:red;
}

<p>My text is not red</p>
<p>My text is red</p>
<p>My text is red</p>
<hr>
<p>My text is not red</p>

Demonstração ao vivo no JSBin

O exemplo acima seleciona apenas os elementos <p> que são diretamente precedidos por outro elemento <p>.

Combinador Geral de Irmãos: selector ~ selector #

O combinador irmão geral (~) seleciona todos os irmãos que seguem o elemento especificado.

p ~ p {
  color:red;
}

<p>My text is not red</p>
<p>My text is red</p>
<hr>
<h1>And now a title</h1>
<p>My text is red</p>

Demonstração ao vivo no JSBin

O exemplo acima seleciona todos os elementos <p> que são precedidos por outro elemento <p>, sejam ou não imediatamente adjacentes.

Pseudo-classes #

Pseudo-classes são palavras-chave que permitem a seleção com base em informações que estão fora da árvore do documento ou que não podem ser expressas por outros seletores ou combinadores. Esta informação pode ser associada a um determinado estado (pseudoclasses state e dynamic), a localizações (pseudoclasses estrutural e target), a negações da primeira (negação pseudo-classe) ou para linguagens (lang pseudo-classe). Exemplos incluem se um link foi seguido ou não (:visited), o mouse está sobre um elemento (:hover), uma caixa de seleção está marcada (:checked), etc.

Sintaxe #

selector:pseudo-class { 
    property: value;
}

Lista de pseudo-classes: #

Nome Descrição
:ativo Aplica-se a qualquer elemento sendo ativado (ou seja, clicado) pelo usuário.
:qualquer Permite construir conjuntos de seletores relacionados criando grupos aos quais os
itens incluídos irão corresponder. Esta é uma alternativa para repetir um seletor inteiro.
:alvo Seleciona o elemento #news ativo atual (clicado em uma URL
que contém o nome da âncora)
:checked Aplica-se a elementos de rádio, caixa de seleção ou opção que estão marcados
ou alternados para um estado “ligado”.
:padrão Representa qualquer elemento da interface do usuário que é o padrão entre um grupo de
elementos semelhantes.
:desativado Aplica-se a qualquer elemento de interface do usuário que esteja em um estado desabilitado.
:vazio Aplica-se a qualquer elemento que não tenha filhos.
:ativado Aplica-se a qualquer elemento de interface do usuário que esteja em um estado ativado.
:primeiro Usado em conjunto com a regra @page, seleciona a primeira página em um
documento impresso.
:primeiro filho Representa qualquer elemento que seja o primeiro elemento filho de seu pai.
:first-of-type Aplica-se quando um elemento é o primeiro do tipo de elemento selecionado
dentro de seu pai. Este pode ou não ser o primeiro filho.
:foco Aplica-se a qualquer elemento que tenha o foco do usuário. Isso pode ser fornecido pelo teclado do usuário, eventos do mouse ou outras formas de entrada.
:foco-dentro Pode ser usado para destacar uma seção inteira quando um elemento dentro dela estiver focado. Ele corresponde a qualquer elemento que a pseudoclasse :focus corresponda ou que tenha um descendente focado.
:tela cheia Aplica-se a qualquer elemento exibido no modo de tela cheia. Ele seleciona toda a pilha
de elementos e não apenas o elemento de nível superior.
:hover Aplica-se a qualquer elemento que esteja passando o mouse pelo dispositivo apontador do usuário, mas
não ativado.
:indeterminado Aplica elementos de interface de usuário de rádio ou caixa de seleção que não estão marcados nem
desmarcados, mas estão em um estado indeterminado. Isso pode ser devido a um atributo
de elemento ou manipulação do DOM.
:dentro do intervalo A pseudo-classe CSS :in-range corresponde quando um elemento tem
seu atributo value dentro das limitações de intervalo especificadas para este elemento.
Permite que a página forneça um feedback de que o valor atualmente definido< br />usar o elemento está dentro dos limites do intervalo.
:inválido Aplica-se a elementos <input> cujos valores são inválidos de acordo com
o tipo especificado no atributo type=.
:lang Aplica-se a qualquer elemento que esteja envolvendo o elemento <body> com um atributo lang= corretamente
designado. Para que a pseudoclasse seja válida, ela deve
conter um código de idioma válido de duas ou três letras.
:último filho Representa qualquer elemento que seja o último elemento filho de seu pai.
:last-of-type Aplica-se quando um elemento é o último do tipo de elemento selecionado dentro
de seu pai. Este pode ou não ser o último filho.
:esquerda Usado em conjunto com a regra @page, isso seleciona todas as páginas
esquerdas em um documento impresso.
:link Aplica-se a quaisquer links que não tenham sido visitados pelo usuário.
:not() Aplica-se a todos os elementos que não correspondem ao valor passado para
(:not(p) ou :not(.class-name) por exemplo. Deve ter um valor a ser< br />válido e pode conter apenas um seletor. No entanto, você pode encadear vários seletores :not juntos.
:nth-child Aplica-se quando um elemento é o elemento n-th de seu pai, onde n
pode ser um inteiro, uma expressão matemática (por exemplo, n+3) ou as palavras-chave
odd ou ‘mesmo’.
:nº do tipo Aplica-se quando um elemento é o elemento n-th de seu pai do
mesmo tipo de elemento, onde n pode ser um inteiro, uma expressão
matemática (por exemplo, n+3) ou as palavras-chave “ímpar” ou “par”.
:filho único A pseudo-classe CSS :only-child representa qualquer elemento
que seja o único filho de seu pai. Isso é o mesmo que
:first-child:last-child ou :nth-child(1):nth-last-child(1),
mas com uma especificidade menor.
:opcional A pseudo-classe CSS :optional representa qualquer elemento
que não tenha o atributo obrigatório definido nele. Isso permite
formulários indicar facilmente campos opcionais e estilizá-los adequadamente.
:fora do intervalo A pseudoclasse CSS :out-of-range corresponde quando um elemento tem seu
atributo de valor fora das limitações de intervalo especificadas para este elemento.
Ela permite que a página forneça um feedback de que o valor atual definido usando o elemento
está fora dos limites do intervalo. Um valor pode estar fora de um intervalo se for
menor ou maior que os valores máximo e mínimo definidos.
:placeholder-mostrado Experimental. Aplica-se a qualquer elemento de formulário que esteja exibindo texto de espaço reservado.
:somente leitura Aplica-se a qualquer elemento que não seja editável pelo usuário.
:ler-escrever Aplica-se a qualquer elemento editável por um usuário, como elementos <input>.
:certo Usado em conjunto com a regra @page, isso seleciona todas as páginas corretas em um
documento impresso.
:root corresponde ao elemento raiz de uma árvore que representa o documento.
:escopo A pseudo-classe CSS corresponde aos elementos que são um ponto de
referência para os seletores corresponderem.
:alvo Seleciona o elemento #news ativo atual (clicado em uma URL
que contém o nome da âncora)
:visitou Aplica-se a quaisquer links que tenham sido visitados pelo usuário.

Seletores de nome de classe #

O seletor de nome de classe seleciona todos os elementos com o nome de classe de destino. Por exemplo, o nome da classe .warning selecionaria o seguinte elemento <div>:

<div class="warning">
    <p>This would be some warning copy.</p>
</div>

Você também pode combinar nomes de classe para elementos de destino mais especificamente. Vamos construir no exemplo acima para mostrar uma seleção de classe mais complicada.

CSS

.important {
    color: orange;
}
.warning {
    color: blue;
}
.warning.important {
    color: red;
}

HTML

<div class="warning">
    <p>This would be some warning copy.</p>
</div>

<div class="important warning">
    <p class="important">This is some really important warning copy.</p>
</div>

Neste exemplo, todos os elementos com a classe .warning terão uma cor de texto azul, elementos com a classe .important terão uma cor de texto laranja e todos os elementos que tiverem ambos o .important e o O nome da classe .warning terá uma cor de texto vermelha.

Observe que dentro do CSS, a declaração .warning.important não tinha nenhum espaço entre os dois nomes de classe. Isso significa que ele só encontrará elementos que contenham ambos os nomes de classe warning e important em seu atributo class. Esses nomes de classe podem estar em qualquer ordem no elemento.

Se um espaço fosse incluído entre as duas classes na declaração CSS, ele selecionaria apenas elementos que possuem elementos pai com nomes de classe .warning e elementos filhos com nomes de classe .important.

Esse truque ajuda você a selecionar um elemento usando o ID como valor para um seletor de atributo para evitar a alta especificidade do seletor de ID.

HTML:

Booleano global com checkbox:checked e ~ (combinador geral de irmãos) #

Com o seletor ~, você pode implementar facilmente um booleano acessível globalmente sem usar JavaScript.

Adiciona boolean como checkbox #

No início do seu documento, adicione quantos booleanos quiser com um único id e o conjunto de atributos hidden:

<input type="checkbox" id="sidebarShown" hidden />
<input type="checkbox" id="darkThemeUsed" hidden />

<!-- here begins actual content, for example: -->
<div id="container">
    <div id="sidebar">
        <!-- Menu, Search, ... -->
    </div>

    <!-- Some more content ... -->
</div>

<div id="footer">
    <!-- ... -->
</div>

Muda o valor do booleano #

Você pode alternar o booleano adicionando um label com o atributo for definido:

<label for="sidebarShown">Show/Hide the sidebar!</label>

Acessando valor booleano com CSS #

O seletor normal (como .color-red) especifica as propriedades padrão. Eles podem ser substituídos seguindo os seletores true / false:

/* true: */
<checkbox>:checked ~ [sibling of checkbox & parent of target] <target>

/* false: */
<checkbox>:not(:checked) ~ [sibling of checkbox & parent of target] <target>

Observe que <checkbox>, [sibling ...] e <target> devem ser substituídos pelos seletores apropriados. [sibling ...] pode ser um seletor específico, (geralmente se você for preguiçoso) simplesmente * ou nada se o alvo já for um irmão da caixa de seleção.

Exemplos para a estrutura HTML acima seriam:

#sidebarShown:checked ~ #container #sidebar {
    margin-left: 300px;
}

#darkThemeUsed:checked ~ #container,
#darkThemeUsed:checked ~ #footer {
    background: #333;
}

O seletor :last-of-type #

O :last-of-type seleciona o elemento que é o último filho, de um tipo específico, de seu pai. No exemplo abaixo, o css seleciona o último parágrafo e o último título h1.

p:last-of-type { 
  background: #C5CAE9; 
}
h1:last-of-type { 
  background: #CDDC39; 
}

Seletores de ID #

Os seletores de ID selecionam elementos DOM com o ID de destino. Para selecionar um elemento por um ID específico em CSS, o prefixo # é usado.

Por exemplo, o seguinte elemento HTML div…

<div id="exampleID">
    <p>Example</p>
</div>

…pode ser selecionado por #exampleID em CSS como mostrado abaixo:

#exampleID {
    width: 20px;
}

Observação: as especificações HTML não permitem vários elementos com o mesmo ID

O exemplo de seletor de pseudo-classe :only-child #

A pseudo-classe :only-child CSS representa qualquer elemento que seja o único filho de seu pai.

HTML:

<div>
  <p>This paragraph is the only child of the div, it will have the color blue</p>
</div>

<div>
  <p>This paragraph is one of the two children of the div</p>
  <p>This paragraph is one of the two children of its parent</p>
</div>

CSS:

p:only-child {
  color: blue;
}

O exemplo acima seleciona o elemento <p> que é o filho único de seu pai, neste caso um <div>.
