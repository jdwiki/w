Nesta página

Grid layout é um novo e poderoso sistema de layout CSS que permite dividir o conteúdo de uma página da web em linhas e colunas de maneira fácil.

Módulo de Layout de Grade CSS Nível 1 é, desde 9 de setembro de 2016, uma Recomendação Candidata do W3C. Considera-se que está em fase de teste (https://www.w3.org/Style/CSS/current-work).

A partir de 3 de julho de 2017, os navegadores Internet Explorer 10 e 11 e Edge da Microsoft suportam apenas uma versão mais antiga da especificação usando um prefixo de fornecedor.

Exemplo básico #

Propriedade Valores possíveis
exibição grade / grade em linha

O CSS Grid é definido como uma propriedade de exibição. Aplica-se apenas a um elemento pai e seus filhos imediatos.

Considere a seguinte marcação:

<section class="container">
  <div class="item1">item1</div>
  <div class="item2">item2</div>
  <div class="item3">item3</div>
  <div class="item4">item4</div>
</section>

A maneira mais fácil de definir a estrutura de marcação acima como uma grade é simplesmente definir sua propriedade display como grid:

.container {
  display: grid;   
}

No entanto, fazer isso invariavelmente fará com que todos os elementos filho sejam colapsados ​​uns sobre os outros. Isso ocorre porque as crianças atualmente não sabem como se posicionar dentro da grade. Mas podemos dizer-lhes explicitamente.

Primeiro precisamos dizer ao elemento de grade .container quantas linhas e colunas irão compor sua estrutura e podemos fazer isso usando as propriedades grid-columns e grid-rows (observe a pluralização):

.container {
  display: grid;
  grid-columns: 50px 50px 50px;
  grid-rows: 50px 50px;
}

No entanto, isso ainda não nos ajuda muito porque precisamos dar uma ordem para cada elemento filho. Podemos fazer isso especificando os valores grid-row e grid-column que informarão onde ele fica na grade:

.container .item1 {
  grid-column: 1;
  grid-row: 1;
}
.container .item2 {
  grid-column: 2;
  grid-row: 1;
}
.container .item3 {
  grid-column: 1;
  grid-row: 2;
}
.container .item4 {
  grid-column: 2;
  grid-row: 2;
}

Ao dar a cada item um valor de coluna e linha, ele identifica a ordem dos itens dentro do contêiner.

Veja um exemplo de trabalho em JSFiddle. Você precisará visualizar isso no IE10, IE11 ou Edge para que funcione, pois atualmente são os únicos navegadores que suportam Grid Layout (com o prefixo do fornecedor -ms-) ou habilitar um sinalizador no Chrome, Opera e Firefox de acordo com caniuse para testar com eles.
