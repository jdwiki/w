Sintaxe #

  • fonte: [font-style] [font-variant] [font-weight] font-size [/line-height] font-family;
  • estilo da fonte: estilo da fonte
  • variante de fonte: variante de fonte
  • peso da fonte: peso da fonte;
  • tamanho da fonte: tamanho da fonte;
  • altura-linha: altura-linha;
  • font-family: font-family;
  • cor: cor;
  • aspas: nenhum|string|inicial|herdar;
  • esticar fonte: * esticar fonte *;
  • alinhamento de texto: alinhamento de texto;
  • recuo de texto: comprimento|inicial|herdar;
  • estouro de texto: clip|ellipsis|string|initial|herit;
  • transformação de texto: nenhum|maiúsculo|maiúsculo|minúsculo|inicial|herdado;
  • text-shadow: h-shadow v-shadow blur-radius color|none|initial|herit;
  • ajuste de tamanho da fonte: número|nenhum|inicial|herdar;
  • font-stretch: ultra-condensado|extra-condensado|condensado|semi-condensado|normal|semi-expandido|expandido|extra-expandido|ultra-expandido|inicial|herdado;
  • hifens: nenhum | manuais | auto;
  • tamanho da guia: número|comprimento|inicial|herdar;
  • espaçamento entre letras: normal|comprimento|inicial|herdar;
  • espaçamento entre palavras: normal|comprimento|inicial|herdar;

Parâmetros #

Parâmetro Detalhes
estilo de fonte itálico ou oblíquo
variante da fonte normal ou minúsculas
peso da fonte normal, negrito ou numérico de 100 a 900.
tamanho da fonte O tamanho da fonte fornecido em %, px, em, ou qualquer outra medida CSS válida
altura da linha A altura da linha dada em %, px, em, ou qualquer outra medida CSS válida
font-family Isto é para definir o nome da família.
cor Qualquer [representação de cores CSS](https://www.wikiod.com/pt/css/cores válida), como red, #00FF00, hsl(240, 100%, 50%) etc.
estiramento de fonte Se deve ou não usar uma face confinada ou expandida da fonte. Os valores válidos são normal, ultra-condensed, extra-condensed, condensed, semi-condensed, semi-expanded, expanded, extra-expanded ou ultra-expanded
text-align start, end, left, right, center, justify, match-parent
decoração de texto none, underline, overline, line-through, initial, herit;
  • A propriedade text-shadow não é suportada por versões do Internet Explorer anteriores a 10.

A abreviação da fonte #

Com a sintaxe:

element {
    font: [font-style] [font-variant] [font-weight] [font-size/line-height] [font-family];
}

Você pode ter todos os seus estilos relacionados a fontes em uma declaração com a abreviação font. Basta usar a propriedade font e colocar seus valores na ordem correta.

Por exemplo, para tornar todos os elementos p em negrito com um tamanho de fonte de 20px e usando Arial como a família de fontes, normalmente você o codificaria da seguinte forma:

p {
    font-weight: bold;
    font-size: 20px;
    font-family: Arial, sans-serif;
}

No entanto, com a abreviação da fonte, ela pode ser condensada da seguinte maneira:

p {
    font: bold 20px Arial, sans-serif;
}

Observação: como font-style, font-variant, font-weight e line-height são opcionais, os três são ignorados neste exemplo. É importante observar que usando o atalho reinicia os demais atributos não informados. Outro ponto importante é que os dois atributos necessários para que o atalho de fonte funcione são font-size e font-family. Se ambos não estiverem incluídos, o atalho será ignorado.

Valor inicial para cada uma das propriedades:

  • fonte-estilo: normal;
  • variante da fonte: normal;
  • peso da fonte: normal;
  • font-stretch: normal;
  • tamanho da fonte: médio;
  • altura da linha: normal;
  • font-family – depende do agente do usuário

Citações #

A propriedade quotes é usada para personalizar as aspas de abertura e fechamento da tag <q>.

q {
    quotes: "«" "»";
}

Tamanho da fonte #

HTML:

<div id="element-one">Hello I am some text.</div>
<div id="element-two">Hello I am some smaller text.</div>

CSS:

#element-one {
    font-size: 30px;   
}

#element-two {
    font-size: 10px;
}

O texto dentro de #element-one terá tamanho de 30px, enquanto o texto em #element-two terá tamanho de 10px.

Direção do texto #

div {
    direction: ltr; /* Default, text read read from left-to-right */
}
.ex {
    direction: rtl; /* text read from right-to-left */
}
.horizontal-tb {
    writing-mode: horizontal-tb; /* Default, text read from left-to-right and top-to-bottom. */
}
.vertical-rtl {
    writing-mode: vertical-rl; /* text read from right-to-left and top-to-bottom */
}
.vertical-ltr {
    writing-mode: vertical-rl; /* text read from left-to-right and top to bottom */
}

A propriedade direction é usada para alterar a direção horizontal do texto de um elemento.

Sintaxe: direção: ltr | rtl | inicial | herdar;

A propriedade writing-mode altera o alinhamento do texto para que possa ser lido de cima para baixo ou da esquerda para a direita, dependendo do idioma.

Sintaxe: direção: horizontal-tb | vertical-rl | vertical-lr;

Pilhas de fontes #

font-family: 'Segoe UI', Tahoma, sans-serif;

O navegador tentará aplicar a fonte “Segoe UI” aos caracteres dentro dos elementos visados ​​pela propriedade acima. Se essa fonte não estiver disponível ou a fonte não contiver um glifo para o caractere necessário, o navegador retornará ao Tahoma e, se necessário, a qualquer fonte sans-serif no computador do usuário. Observe que qualquer nome de fonte com mais de uma palavra, como “Segoe UI”, precisa ter aspas simples ou duplas.

font-family: Consolas, 'Courier New', monospace;

O navegador tentará aplicar a fonte “Consolas” aos caracteres dentro dos elementos visados ​​pela propriedade acima. Se essa fonte não estiver disponível ou a fonte não contiver um glifo para o caractere necessário, o navegador retornará para “Courier New” e, se necessário, qualquer fonte monoespaçada no computador do usuário.

Transformação de texto #

A propriedade text-transform permite que você altere a capitalização do texto. Os valores válidos são: maiúsculas, maiúsculas, minúsculas, inicial, herdar e nenhum

CSS:

.example1 {
    text-transform: uppercase;
}
.example2 {
    text-transform: capitalize;
}
.example3 {
    text-transform: lowercase;
}

HTML

<p class="example1">
    all letters in uppercase  <!-- "ALL LETTERS IN UPPERCASE" -->
</p>
<p class="example2">
    all letters in capitalize <!-- "All Letters In Capitalize (Sentence Case)" -->
</p>
<p class="example3">
    all letters in lowercase  <!-- "all letters in lowercase" -->
</p>

Estouro de texto #

A propriedade text-overflow trata de como o conteúdo transbordado deve ser sinalizado aos usuários. Neste exemplo, o ellipsis representa o texto recortado.

.text {
  overflow: hidden;
  text-overflow: ellipsis;
}

Infelizmente, text-overflow: ellipsis só funciona em uma única linha de texto. Não há como oferecer suporte a reticências na última linha no CSS padrão, mas isso pode ser alcançado com a implementação de flexboxes somente para webkit não padrão.

.giveMeEllipsis {
   overflow: hidden;
   text-overflow: ellipsis;
   display: -webkit-box;
   -webkit-box-orient: vertical;
   -webkit-line-clamp: N; /* number of lines to show */
   line-height: X;        /* fallback */
   max-height: X*N;       /* fallback */
}

Exemplo (abrir no Chrome ou Safari):

http://jsfiddle.net/csYjC/1131/

Recursos:

https://www.w3.org/TR/2012/WD-css3-ui-20120117/#text-overflow0

Sombra de texto #

Para adicionar sombras ao texto, use a propriedade text-shadow. A sintaxe é a seguinte:

text-shadow: horizontal-offset vertical-offset blur color;

Sombra sem raio de desfoque #

h1 {
  text-shadow: 2px 2px #0000FF;
}

Isso cria um efeito de sombra azul em torno de um título

Sombra com raio de desfoque #

Para adicionar um efeito de desfoque, adicione um argumento de opção blur radius

h1 {
  text-shadow: 2px 2px 10px #0000FF;
}

Várias sombras #

Para dar a um elemento várias sombras, separe-as com vírgulas

h1 {
    text-shadow: 0 0 3px #FF0000, 0 0 5px #0000FF;
}

Espaçamento entre letras #

h2 {
  /* adds a 1px space horizontally between each letter;
     also known as tracking */
  letter-spacing: 1px;
}

A propriedade de espaçamento entre letras é usada para especificar o espaço entre os caracteres em um texto.

! o espaçamento entre letras também suporta valores negativos:

p {
  letter-spacing: -1px;
}

Recursos: https://developer.mozilla.org/en-US/docs/Web/CSS/letter-spacing

Recuo de Texto #

p {
    text-indent: 50px;
}

A propriedade text-indent especifica quanto espaço horizontal o texto deve ser movido antes do início da primeira linha do conteúdo de texto de um elemento.

Recursos:

  • http://stackoverflow.com/questions/5856952/indenting-only-the-first-line-of-text-in-a-paragraph
  • https://www.w3.org/TR/CSS21/text.html#propdef-text-indent
  • https://developer.mozilla.org/en-US/docs/Web/CSS/text-indent

Decoração de Texto #

A propriedade text-decoration é usada para definir ou remover decorações do texto.

h1 { text-decoration: none; }
h2 { text-decoration: overline; }
h3 { text-decoration: line-through; }
h4 { text-decoration: underline; }

text-decoration pode ser usado em combinação com text-decoration-style e text-decoration-color como uma propriedade abreviada:

.title { text-decoration: underline dotted blue; }

Esta é uma versão abreviada de

.title {
  text-decoration-style: dotted;
  text-decoration-line: underline;
  text-decoration-color: blue;
}

Deve-se notar que as seguintes propriedades são suportadas apenas no Firefox

  • texto-decoração-cor
  • linha de decoração de texto
  • estilo de decoração de texto
  • texto-decoração-pular

Espaçamento entre palavras #

A propriedade word-spacing especifica o comportamento de espaçamento entre tags e palavras.

Valores possíveis

  • um comprimento positivo ou negativo (usando em px vh cm etc.) ou porcentagem (usando %)
  • a palavra-chave normal usa o espaçamento de palavras padrão da fonte
  • a palavra-chave inherit pega o valor do elemento pai

CSS

.normal    { word-spacing: normal; }
.narrow    { word-spacing: -3px; }
.extensive { word-spacing: 10px; }

HTML

<p>
  <span class="normal">This is an example, showing the effect of "word-spacing".</span><br>
  <span class="narrow">This is an example, showing the effect of "word-spacing".</span><br>
  <span class="extensive">This is an example, showing the effect of "word-spacing".</span><br>
</p>

Variante de fonte #

Atributos:

normal

Atributo padrão de fontes.

minúsculas

Define todas as letras como maiúsculas, mas torna as letras minúsculas (do texto original) menores em tamanho do que as letras originalmente maiúsculas.

CSS:

.smallcaps{
   font-variant: small-caps;
}

HTML:

<p class="smallcaps">
    Documentation about CSS Fonts
    <br>
    aNd ExAmpLe
</p>

RESULTADO:

Observação: A propriedade font-variant é uma abreviação para as propriedades: font-variant-caps, font-variant-numeric, font-variant-alternates, font-variant-ligatures e font-variant-east-asian.
