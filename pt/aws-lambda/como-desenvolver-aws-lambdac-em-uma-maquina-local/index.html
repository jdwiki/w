Como parte do movimento Serverless, a AWS anunciou recentemente o suporte C# para funções do AWS Lambda. Neste artigo, mostrarei como desenvolver, instalar e chamar uma função simples do AWS Lambda em C# usando o Visual Studio 2015.

Antes de iniciar este exemplo, primeiro vá para a seção Comentários deste documento e certifique-se de ter todos os componentes necessários instalados.

Instale os componentes necessários #

No momento da redação deste artigo (janeiro de 2017), os seguintes componentes precisarão ser instalados em sua máquina nesta ordem. A máquina que usei tinha o Windows 10 Home instalado.

  1. [Atualização 3 do Visual Studio 2015] 2
  2. Kit de ferramentas da AWS para Visual Studio 2015 - versão 1.11.2.0
  3. Ferramentas do Microsoft .NET Core (Visualização 2) - versão 14.1.21111.0

Cada um desses componentes é desenvolvido e lançado ativamente, portanto, verifique os links e as versões e atualize este documento quando necessário.

Cada um desses pode levar algum tempo para instalar, então seja paciente, faça um de cada vez para minimizar os erros.

Por fim, reiniciar o Windows antes de iniciar o desenvolvimento é triste, mas sempre uma boa ideia depois de fazer grandes instalações de ferramentas de desenvolvimento em uma caixa do Windows.

Etapa 1. Crie uma nova solução no Visual Studio #

  1. Abra o Visual Studio e selecione Arquivo -> Novo Projeto
  2. Selecione Projeto AWS Lambda com testes (.NET Core)
  3. Em seguida, a tela Select Blueprint será exibida. Selecione Função vazia e clique no botão Concluir:
  4. Vá para Ferramentas -> Gerenciador de Pacotes NuGet -> Console do Gerenciador de Pacotes.
  5. Na janela do console, digite Install-Package Amazon.Lambda.Core
  6. Clique com o botão direito do mouse em seu projeto no gerenciador de soluções e selecione Restaurar Pacotes. Esta é a etapa final de preparação antes de começar a escrever o código.

Etapa 2. Adicione e altere o código em seu projeto #

  1. Abra Function.cs e substitua o código da classe pelo seguinte:

     public class Function
 {

 /// <summary>
 /// A simple function that takes a birth date and returns Age in years
 /// </summary>
 /// <param name="input"></param>
 /// <returns>Age is years</returns>
 /// 
 [LambdaSerializer(typeof(SimpleSerializer))]
 public string FunctionHandler(Dictionary<string, int> input)
 {
     var defaultMessage = "Age could not be determined.";

     var birthDate = new DateTime(input["year"], input["month"], input["day"]);
     var ageInYears = DateTime.Today.Year - birthDate.Year;
     if (birthDate.DayOfYear > DateTime.Today.DayOfYear)
         ageInYears--;

     defaultMessage = $"Age in years: {ageInYears}";

     return defaultMessage;
 }
 }

Você precisará adicionar as seguintes instruções using próximas ao topo:

using Amazon.Lambda.Core;

  3. No projeto de teste, altere a linha 23 do FunctionTest.cs para o seguinte:

             var upperCase = function.FunctionHandler(null);

  4. Construa sua solução - você não deve ter erros de construção.

Etapa 3 - Instale sua função Lambda na AWS #

  1. Clique com o botão direito do mouse em seu projeto e selecione Publish to AWS Lambda…

  2. A tela Upload to AWS Lambda será exibida. Certifique-se de que a região correta esteja selecionada. Mantenha todos os padrões. Em seguida, insira apenas o Nome da Função: AWSLambdaFunctionAgeInYears e clique em próximo.

  3. Na próxima página, selecione AWSLambdaRole para o campo Nome da função. Clique em Carregar e a função deve carregar sem erros.

Etapa 4. Invoque sua função AWS Lambda de dentro do Visual Studio #

  1. Após a Etapa 3 acima, o Visual Studio abrirá a janela View Function com sua função carregada.

  2. No canto inferior esquerdo desta tela, digite o seguinte json na caixa Sample Input:

    { “month”: “10”, “day”: “28”, “year”: “1979” }

  1. Última etapa: Clique no botão verde Invoke. A função será executada na AWS e a resposta será exibida no canto inferior direito da janela Response.
