Adicionando campos a uma tabela dinâmica #

Duas coisas importantes a serem observadas ao adicionar campos a uma Tabela Dinâmica são Orientação e Posição. Às vezes, um desenvolvedor pode supor onde um campo é colocado, portanto, é sempre mais claro definir explicitamente esses parâmetros. Essas ações afetam apenas a tabela dinâmica fornecida, não o cache dinâmico.

Dim thisPivot As PivotTable
Dim ptSheet As Worksheet
Dim ptField As PivotField

Set ptSheet = ThisWorkbook.Sheets("SheetNameWithPivotTable")
Set thisPivot = ptSheet.PivotTables(1)

With thisPivot
    Set ptField = .PivotFields("Gender")
    ptField.Orientation = xlRowField
    ptField.Position = 1
    Set ptField = .PivotFields("LastName")
    ptField.Orientation = xlRowField
    ptField.Position = 2
    Set ptField = .PivotFields("ShirtSize")
    ptField.Orientation = xlColumnField
    ptField.Position = 1
    Set ptField = .AddDataField(.PivotFields("Cost"), "Sum of Cost", xlSum)
    .InGridDropZones = True
    .RowAxisLayout xlTabularRow
End With

Criando uma tabela dinâmica #

Um dos recursos mais poderosos do Excel é o uso de Tabelas Dinâmicas para classificar e analisar dados. Usar o VBA para criar e manipular os Pivots é mais fácil se você entender a relação das Tabelas Dinâmicas com os Caches Dinâmicos e como fazer referência e usar as diferentes partes das Tabelas.

Em sua forma mais básica, seus dados de origem são uma área de dados Range em uma Worksheet. Esta área de dados DEVE identificar as colunas de dados com uma linha de cabeçalho como a primeira linha do intervalo. Uma vez criada a Tabela Dinâmica, o usuário pode visualizar e alterar os dados de origem a qualquer momento. No entanto, as alterações podem não ser refletidas automaticamente ou imediatamente na própria Tabela Dinâmica porque há uma estrutura de armazenamento de dados intermediária chamada Cache Dinâmica que está diretamente conectada à própria Tabela Dinâmica.

Se forem necessárias várias Tabelas Dinâmicas, com base nos mesmos dados de origem, o Cache Dinâmico poderá ser reutilizado como o armazenamento de dados interno para cada uma das Tabelas Dinâmicas. Essa é uma boa prática porque economiza memória e reduz o tamanho do arquivo do Excel para armazenamento.

Como exemplo, para criar uma Tabela Dinâmica com base nos dados de origem mostrados nas Figuras acima:

Sub test()
    Dim pt As PivotTable
    Set pt = CreatePivotTable(ThisWorkbook.Sheets("Sheet1").Range("A1:E15"))
End Sub

Function CreatePivotTable(ByRef srcData As Range) As PivotTable
    '--- creates a Pivot Table from the given source data and
    '    assumes that the first row contains valid header data
    '    for the columns
    Dim thisPivot As PivotTable
    Dim dataSheet As Worksheet
    Dim ptSheet As Worksheet
    Dim ptCache As PivotCache
    
    '--- the Pivot Cache must be created first...
    Set ptCache = ThisWorkbook.PivotCaches.Create(SourceType:=xlDatabase, _
                                                  SourceData:=srcData)
    '--- ... then use the Pivot Cache to create the Table
    Set ptSheet = ThisWorkbook.Sheets.Add
    Set thisPivot = ptCache.CreatePivotTable(TableDestination:=ptSheet.Range("A3"))
    Set CreatePivotTable = thisPivot
End Function

Formatando os dados da tabela dinâmica #

Este exemplo altera/configura vários formatos na área de intervalo de dados (DataBodyRange) da Tabela Dinâmica fornecida. Todos os parâmetros formatados em um Range padrão estão disponíveis. A formatação dos dados afeta apenas a própria Tabela Dinâmica, não o Cache Dinâmico.

NOTA: a propriedade é denominada TableStyle2 porque a propriedade TableStyle não é membro das propriedades do objeto PivotTable.

Dim thisPivot As PivotTable
Dim ptSheet As Worksheet
Dim ptField As PivotField

Set ptSheet = ThisWorkbook.Sheets("SheetNameWithPivotTable")
Set thisPivot = ptSheet.PivotTables(1)

With thisPivot
    .DataBodyRange.NumberFormat = "_($* #,##0.00_);_($* (#,##0.00);_($* "-"??_);_(@_)"
    .DataBodyRange.HorizontalAlignment = xlRight
    .ColumnRange.HorizontalAlignment = xlCenter
    .TableStyle2 = "PivotStyleMedium9"
End With
