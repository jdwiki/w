Criando um gráfico com intervalos e um nome fixo #

Os gráficos podem ser criados trabalhando diretamente com o objeto Series que define os dados do gráfico. Para chegar à Series sem um gráfico existente, você cria um ChartObject em uma determinada Worksheet e então obtém o objeto Chart dele. A vantagem de trabalhar com o objeto Series é que você pode definir os Values e XValues referindo-se aos objetos Range. Essas propriedades de dados definirão corretamente a Série com referências a esses intervalos. A desvantagem dessa abordagem é que a mesma conversão não é tratada ao definir o Name; é um valor fixo. Ele não se ajustará aos dados subjacentes no Range original. Verificando a fórmula SERIES e é óbvio que o nome está fixo. Isso deve ser tratado criando a fórmula SERIES diretamente.

Código usado para criar o gráfico

Observe que este código contém declarações de variáveis ​​extras para o Chart e a Worksheet. Estes podem ser omitidos se não forem usados. No entanto, eles podem ser úteis se você estiver modificando o estilo ou qualquer outra propriedade do gráfico.

Sub CreateChartWithRangesAndFixedName()

    Dim xData As Range
    Dim yData As Range
    Dim serName As Range
    
    'set the ranges to get the data and y value label
    Set xData = Range("B3:B12")
    Set yData = Range("C3:C12")
    Set serName = Range("C2")
    
    'get reference to ActiveSheet
    Dim sht As Worksheet
    Set sht = ActiveSheet
    
    'create a new ChartObject at position (48, 195) with width 400 and height 300
    Dim chtObj As ChartObject
    Set chtObj = sht.ChartObjects.Add(48, 195, 400, 300)
    
    'get reference to chart object
    Dim cht As Chart
    Set cht = chtObj.Chart
    
    'create the new series
    Dim ser As Series
    Set ser = cht.SeriesCollection.NewSeries
    
    ser.Values = yData
    ser.XValues = xData
    ser.Name = serName
    
    ser.ChartType = xlXYScatterLines

End Sub

Dados/intervalos originais e Chart resultante após a execução do código

Observe que a fórmula SERIES inclui um "B" para o nome da série em vez de uma referência ao Range que a criou.

Criando um gráfico vazio #

O ponto de partida para a grande maioria do código de gráficos é criar um Chart vazio. Observe que este Gráfico está sujeito ao modelo de gráfico padrão que está ativo e pode não estar realmente vazio (se o modelo tiver sido modificado).

A chave para o ChartObject é determinar sua localização. A sintaxe para a chamada é ChartObjects.Add(Left, Top, Width, Height). Uma vez que o ChartObject é criado, você pode usar seu objeto Chart para realmente modificar o gráfico. O ChartObject se comporta mais como um Shape para posicionar o gráfico na planilha.

Código para criar um gráfico vazio

Sub CreateEmptyChart()
    
    'get reference to ActiveSheet
    Dim sht As Worksheet
    Set sht = ActiveSheet
    
    'create a new ChartObject at position (0, 0) with width 400 and height 300
    Dim chtObj As ChartObject
    Set chtObj = sht.ChartObjects.Add(0, 0, 400, 300)
    
    'get refernce to chart object
    Dim cht As Chart
    Set cht = chtObj.Chart
    
    'additional code to modify the empty chart
    '...

End Sub

Crie um gráfico modificando a fórmula SÉRIE #

Para controle completo sobre um novo objeto Chart e Series (especialmente para um nome dinâmico Series), você deve modificar a fórmula SERIES diretamente. O processo para configurar os objetos Range é direto e o principal obstáculo é simplesmente a construção da string para a fórmula SERIES.

A fórmula SERIES usa a seguinte sintaxe:

=SERIES(Name,XValues,Values,Order)

Esses conteúdos podem ser fornecidos como referências ou como valores de matriz para os itens de dados. Order representa a posição da série no gráfico. Observe que as referências aos dados não funcionarão a menos que sejam totalmente qualificadas com o nome da planilha. Para obter um exemplo de uma fórmula funcional, clique em qualquer série existente e verifique a barra de fórmulas.

Código para criar um gráfico e configurar dados usando a fórmula SERIES

Observe que a construção da string para criar a fórmula SERIES usa .Address(,,,True). Isso garante que a referência de intervalo externa seja usada para que um endereço totalmente qualificado com o nome da planilha seja incluído. Você receberá um erro se o nome da planilha for excluído.

Sub CreateChartUsingSeriesFormula()

    Dim xData As Range
    Dim yData As Range
    Dim serName As Range
    
    'set the ranges to get the data and y value label
    Set xData = Range("B3:B12")
    Set yData = Range("C3:C12")
    Set serName = Range("C2")
    
    'get reference to ActiveSheet
    Dim sht As Worksheet
    Set sht = ActiveSheet
    
    'create a new ChartObject at position (48, 195) with width 400 and height 300
    Dim chtObj As ChartObject
    Set chtObj = sht.ChartObjects.Add(48, 195, 400, 300)
    
    'get refernce to chart object
    Dim cht As Chart
    Set cht = chtObj.Chart
    
    'create the new series
    Dim ser As Series
    Set ser = cht.SeriesCollection.NewSeries
    
    'set the SERIES formula
    '=SERIES(name, xData, yData, plotOrder)
    
    Dim formulaValue As String
    formulaValue = "=SERIES(" & _
        serName.Address(, , , True) & "," & _
        xData.Address(, , , True) & "," & _
        yData.Address(, , , True) & ",1)"
    
    ser.Formula = formulaValue
    ser.ChartType = xlXYScatterLines

End Sub

Dados originais e gráfico resultante

Observe que, para este gráfico, o nome da série está definido corretamente com um intervalo para a célula desejada. Isso significa que as atualizações serão propagadas para o Chart.

Organizando gráficos em uma grade #

Uma tarefa comum com gráficos no Excel é padronizar o tamanho e o layout de vários gráficos em uma única planilha. Se feito manualmente, você pode manter pressionado ALT enquanto redimensiona ou move o gráfico para “aderir” aos limites da célula. Isso funciona para alguns gráficos, mas uma abordagem VBA é muito mais simples.

Código para criar uma grade

Esse código criará uma grade de gráficos começando em uma determinada posição (Superior, Esquerda), com um número definido de colunas e um tamanho de gráfico comum definido. Os gráficos serão colocados na ordem em que foram criados e enrolados na borda para formar uma nova linha.

Sub CreateGridOfCharts()

    Dim int_cols As Integer
    int_cols = 3
    
    Dim cht_width As Double
    cht_width = 250
        
    Dim cht_height As Double
    cht_height = 200
    
    Dim offset_vertical As Double
    offset_vertical = 195
    
    Dim offset_horz As Double
    offset_horz = 40

    Dim sht As Worksheet
    Set sht = ActiveSheet

    Dim count As Integer
    count = 0
    
    'iterate through ChartObjects on current sheet
    Dim cht_obj As ChartObject
    For Each cht_obj In sht.ChartObjects
        
        'use integer division and Mod to get position in grid
        cht_obj.Top = (count \ int_cols) * cht_height + offset_vertical
        cht_obj.Left = (count Mod int_cols) * cht_width + offset_horz
        cht_obj.Width = cht_width
        cht_obj.Height = cht_height

        count = count + 1

    Next cht_obj
End Sub
