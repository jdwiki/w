CustomDocumentProperties na prática

  • Organizando novos números de fatura

  • Organizando novos números de fatura

Usar CustomDocumentProperties (CDPs) é um bom método para armazenar valores definidos pelo usuário de maneira relativamente segura dentro da mesma pasta de trabalho, mas evitando mostrar valores de células relacionadas simplesmente em uma planilha desprotegida *).

Observação: os CDPs representam uma coleção separada comparável a BuiltInDocumentProperties, mas permitem criar seus próprios nomes de propriedade definidos pelo usuário em vez de uma coleção fixa.

*) Como alternativa, você pode inserir valores também em uma pasta de trabalho oculta ou “muito oculta”.

Organizando novos números de fatura #

Incrementar um número de fatura e salvar seu valor é uma tarefa frequente. Usar CustomDocumentProperties (CDPs) é um bom método para armazenar esses números de maneira relativamente segura dentro da mesma pasta de trabalho, mas evitando mostrar valores de células relacionadas simplesmente em uma planilha desprotegida.

Dica adicional:

Alternativamente, você pode inserir valores também em uma planilha oculta ou mesmo em uma planilha “muito oculta” (consulte https://www.wikiod.com/pt/excel-vba/dicas-e-truques-do-excel-vba /using-xlveryhidden-sheets#t=201707311517265257827). Claro, é possível salvar dados também em arquivos externos (por exemplo, arquivo ini, csv ou qualquer outro tipo) ou no registro.

Exemplo de conteúdo:

O exemplo abaixo mostra

  • uma função NextInvoiceNo que define e retorna o próximo número da fatura,
  • um procedimento DeleteInvoiceNo, que exclui completamente a fatura CDP, bem como
  • um procedimento showAllCDPs listando a coleção completa de CDPs com todos os nomes. Não usando VBA, você também pode listá-los através das informações da pasta de trabalho: Info | Propriedades [DropDown:] | Propriedades Avançadas | Personalizado

Você pode obter e definir o próximo número da fatura (último não mais um) simplesmente chamando a função mencionada acima, retornando um valor de string para facilitar a adição de prefixos. “InvoiceNo” é usado implicitamente como nome CDP em todos os procedimentos.

Dim sNumber As String
sNumber = NextInvoiceNo ()

Código de exemplo:

Option Explicit

Sub Test()
  Dim sNumber As String
  sNumber = NextInvoiceNo()
  MsgBox "New Invoice No: " & sNumber, vbInformation, "New Invoice Number"
End Sub

Function NextInvoiceNo() As String
' Purpose: a) Set Custom Document Property (CDP) "InvoiceNo" if not yet existing
'          b) Increment CDP value and return new value as string
' Declarations
  Dim prop As Object
  Dim ret  As String
  Dim wb   As Workbook
' Set workbook and CDPs
  Set wb = ThisWorkbook
  Set prop = wb.CustomDocumentProperties

  ' -------------------------------------------------------
  ' Generate new CDP "InvoiceNo" if not yet existing
  ' -------------------------------------------------------
    If Not CDPExists("InvoiceNo") Then
    '  set temporary starting value "0"
       prop.Add "InvoiceNo", False, msoPropertyTypeString, "0"
    End If

  ' --------------------------------------------------------
  ' Increment invoice no and return function value as string
  ' --------------------------------------------------------
       ret = Format(Val(prop("InvoiceNo")) + 1, "0")
  ' a) Set CDP "InvoiceNo" = ret
       prop("InvoiceNo").value = ret
  ' b) Return function value 
       NextInvoiceNo = ret
End Function

Private Function CDPExists(sCDPName As String) As Boolean
' Purpose: return True if custom document property (CDP) exists
' Method: loop thru CustomDocumentProperties collection and check if name parameter exists
' Site: cf. http://stackoverflow.com/questions/23917977/alternatives-to-public-variables-in-vba/23918236#23918236
' vgl.: https://answers.microsoft.com/en-us/msoffice/forum/msoffice_word-mso_other/using-customdocumentproperties-with-vba/91ef15eb-b089-4c9b-a8a7-1685d073fb9f
' Declarations
  Dim cdp As Variant      ' element of CustomDocumentProperties Collection
  Dim boo As Boolean      ' boolean value showing element exists
  For Each cdp In ThisWorkbook.CustomDocumentProperties
    If LCase(cdp.Name) = LCase(sCDPName) Then
       boo = True      ' heureka
       Exit For        ' exit loop
    End If
  Next
  CDPExists = boo          ' return value to function
End Function


Sub DeleteInvoiceNo()
' Declarations
  Dim wb     As Workbook
  Dim prop   As Object
' Set workbook and CDPs
  Set wb = ThisWorkbook
  Set prop = wb.CustomDocumentProperties

' ----------------------
' Delete CDP "InvoiceNo"
' ----------------------
 If CDPExists("InvoiceNo") Then
    prop("InvoiceNo").Delete
 End If

Finalizar Sub

Sub showAllCDPs()
' Purpose: Show all CustomDocumentProperties (CDP) and values (if set)
' Declarations
  Dim wb      As Workbook
  Dim cdp     As Object

  Dim i       As Integer
  Dim maxi   As Integer
  Dim s       As String
' Set workbook and CDPs
  Set wb = ThisWorkbook
  Set cdp = wb.CustomDocumentProperties
' Loop thru CDP getting name and value
  maxi = cdp.Count
  For i = 1 To maxi
    On Error Resume Next    ' necessary in case of unset value
    s = s & Chr(i + 96) & ") " & _
            cdp(i).Name & "=" & cdp(i).value & vbCr
  Next i
' Show result string
  Debug.Print s
End Sub
