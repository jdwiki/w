Sintaxe #

  • Set - O operador usado para definir uma referência a um objeto, como um Range
  • For Each - O operador costumava percorrer todos os itens em uma coleção

Observe que os nomes das variáveis ​​r, cell e outros podem ser nomeados como você quiser, mas devem ser nomeados adequadamente para que o código seja mais fácil de entender para você e para os outros.

Maneiras de se referir a uma única célula #

A maneira mais simples de se referir a uma única célula na planilha atual do Excel é simplesmente colocar o formulário A1 de sua referência entre colchetes:

[a3] = "Hello!"

Observe que colchetes são apenas convenientes syntactic sugar para o método Evaluate do objeto Application, portanto, tecnicamente, isso é idêntico ao seguinte código:

Application.Evaluate("a3") = "Hello!"

Você também pode chamar o método Cells que recebe uma linha e uma coluna e retorna uma referência de célula.

Cells(3, 1).Formula = "=A1+A2"

Lembre-se que sempre que você passa uma linha e uma coluna para o Excel do VBA, a linha é sempre a primeira, seguida pela coluna, o que é confuso porque é o oposto da notação A1 comum onde a coluna aparece primeiro.

Em ambos os exemplos, não especificamos uma planilha, então o Excel usará a planilha ativa (a planilha que está na frente da interface do usuário). Você pode especificar a planilha ativa explicitamente:

ActiveSheet.Cells(3, 1).Formula = "=SUM(A1:A2)"

Ou você pode fornecer o nome de uma planilha específica:

Sheets("Sheet2").Cells(3, 1).Formula = "=SUM(A1:A2)"

Há uma grande variedade de métodos que podem ser usados ​​para ir de um intervalo para outro. Por exemplo, o método Rows pode ser usado para acessar as linhas individuais de qualquer intervalo, e o método Cells pode ser usado para acessar células individuais de uma linha ou coluna, portanto, o código a seguir se refere à célula C1:

ActiveSheet.Rows(1).Cells(3).Formula = "hi!"

Criando um intervalo #

Um Range não pode ser criado ou preenchido da mesma forma que uma string:

Sub RangeTest()
    Dim s As String
    Dim r As Range 'Specific Type of Object, with members like Address, WrapText, AutoFill, etc.
    
    ' This is how we fill a String:
    s = "Hello World!"

    ' But we cannot do this for a Range:
    r = Range("A1") '//Run. Err.:  91 Object variable or With block variable not set//

    ' We have to use the Object approach, using keyword Set:
    Set r = Range("A1")
End Sub

É considerado uma prática recomendada [qualificar suas referências](https://www.wikiod.com/pt/excel-vba/praticas-recomendadas-de-vba#Sempre defina e defina referências a todas as pastas de trabalho e planilhas workbooks-and-sheets#t=201607261348479429911), então a partir de agora usaremos a mesma abordagem aqui. Mais sobre Criar variáveis ​​de objeto (por exemplo, intervalo) no MSDN . Mais sobre Definir declaração no MSDN.

Existem diferentes maneiras de criar o mesmo intervalo:

Sub SetRangeVariable()
    Dim ws As Worksheet
    Dim r As Range

    Set ws = ThisWorkbook.Worksheets(1) ' The first Worksheet in Workbook with this code in it
    
    ' These are all equivalent:
    Set r = ws.Range("A2")
    Set r = ws.Range("A" & 2)
    Set r = ws.Cells(2, 1) ' The cell in row number 2, column number 1
    Set r = ws.[A2] 'Shorthand notation of Range.
    Set r = Range("NamedRangeInA2") 'If the cell A2 is named NamedRangeInA2. Note, that this is Sheet independent.
    Set r = ws.Range("A1").Offset(1, 0) ' The cell that is 1 row and 0 columns away from A1
    Set r = ws.Range("A1").Cells(2,1) ' Similar to Offset. You can "go outside" the original Range.

    Set r = ws.Range("A1:A5").Cells(2) 'Second cell in bigger Range.
    Set r = ws.Range("A1:A5").Item(2) 'Second cell in bigger Range.
    Set r = ws.Range("A1:A5")(2) 'Second cell in bigger Range.
End Sub

Observe no exemplo que Cells(2, 1) é equivalente a Range(“A2”). Isso ocorre porque Cells retorna um objeto Range. Algumas fontes: Chip Pearson-Cells Within Ranges; MSDN-Range Object; John Walkenback-referindo-se a intervalos em seu código VBA.

Observe também que em qualquer instância em que um número é usado na declaração do intervalo, e o próprio número está fora das aspas, como Range(“A” & 2), você pode trocar esse número por uma variável que contém um inteiro/longo. Por exemplo:

Sub RangeIteration()
    Dim wb As Workbook, ws As Worksheet
    Dim r As Range

    Set wb = ThisWorkbook
    Set ws = wb.Worksheets(1)

    For i = 1 To 10
        Set r = ws.Range("A" & i)
        ' When i = 1, the result will be Range("A1")
        ' When i = 2, the result will be Range("A2")
        ' etc.
        ' Proof:
        Debug.Print r.Address
    Next i
End Sub

Se você estiver usando loops duplos, Cells é melhor:

Sub RangeIteration2()
    Dim wb As Workbook, ws As Worksheet
    Dim r As Range

    Set wb = ThisWorkbook
    Set ws = wb.Worksheets(1)

    For i = 1 To 10
        For j = 1 To 10
            Set r = ws.Cells(i, j)
            ' When i = 1 and j = 1, the result will be Range("A1")
            ' When i = 2 and j = 1, the result will be Range("A2")
            ' When i = 1 and j = 2, the result will be Range("B1")
            ' etc.
            ' Proof:
            Debug.Print r.Address
        Next j
    Next i
End Sub

Propriedade de compensação #

  • Offset(Rows, Columns) - O operador usado para referenciar estaticamente outro ponto da célula atual. Muitas vezes usado em loops. Deve-se entender que os números positivos na seção de linhas se movem para a direita, enquanto os negativos se movem para a esquerda. Com a seção de colunas, os positivos descem e os negativos sobem.

ou seja

Private Sub this()
    ThisWorkbook.Sheets("Sheet1").Range("A1").Offset(1, 1).Select
    ThisWorkbook.Sheets("Sheet1").Range("A1").Offset(1, 1).Value = "New Value"
    ActiveCell.Offset(-1, -1).Value = ActiveCell.Value
    ActiveCell.Value = vbNullString
End Sub

Esse código seleciona B2, coloca uma nova string lá e, em seguida, move essa string de volta para A1 depois de limpar B2.

Salvando uma referência a uma célula em uma variável #

Para salvar uma referência a uma célula em uma variável, você deve usar a sintaxe Set, por exemplo:

Dim R as Range
Set R = ActiveSheet.Cells(3, 1)

mais tarde…

R.Font.Color = RGB(255, 0, 0)

Por que a palavra-chave Set é necessária? Set informa ao Visual Basic que o valor no lado direito do = deve ser um objeto.

Como transpor intervalos (horizontal para vertical e vice-versa) #

Sub TransposeRangeValues()
    Dim TmpArray() As Variant, FromRange as Range, ToRange as Range

    set FromRange = Sheets("Sheet1").Range("a1:a12")         'Worksheets(1).Range("a1:p1")
    set ToRange = ThisWorkbook.Sheets("Sheet1").Range("a1")  'ThisWorkbook.Sheets("Sheet1").Range("a1")

    TmpArray = Application.Transpose(FromRange.Value)
    FromRange.Clear
    ToRange.Resize(FromRange.Columns.Count,FromRange.Rows.Count).Value2 = TmpArray
End Sub

Nota: Copy/PasteSpecial também tem uma opção Paste Transpose que atualiza também as fórmulas das células transpostas.
