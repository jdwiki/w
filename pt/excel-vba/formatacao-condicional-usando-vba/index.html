Você não pode definir mais de três formatos condicionais para um intervalo. Use o método Modify para modificar um formato condicional existente ou use o método Delete para excluir um formato existente antes de adicionar um novo.

FormatConditions.Add #

Sintaxe: #

FormatConditions.Add(Type, Operator, Formula1, Formula2)

Parâmetros: #

Nome Obrigatório / Opcional Tipo de dados
Tipo Obrigatório XlFormatConditionType
Operador Opcional Variante
Fórmula1 Opcional Variante
Fórmula2 Opcional Variante

Enumeração XlFormatConditionType: #

Nome Descrição
xlAboveAverageCondition Condição acima da média
xlBlanksCondição Condição de espaços em branco
xlCellValue Valor da célula
xlColorScale Escala de cores
xlDatabar Barra de dados
xlErrorsCondition Condição de erros
xlExpressão Expressão
XlIconSet Conjunto de ícones
xlNoBlanksCondition Sem condição de espaços em branco
xlNoErrorsCondition Sem condição de erros
xlTextString Cadeia de texto
xlTimePeríodo Período de tempo
xlTop10 10 principais valores
xlValores únicos Valores únicos

Formatação por valor de célula: #

With Range("A1").FormatConditions.Add(xlCellValue, xlGreater, "=100")
    With .Font
        .Bold = True
        .ColorIndex = 3
     End With
End With

Operadores: #

Nome
xlEntre
xlIgual
xlMaior
xlMaiorIgual
xlMenos
xlLessEqual
xlNotBetween
xlNotEqual
Se Type for xlExpression, o argumento Operator será ignorado.

A formatação por texto contém: #

With Range("a1:a10").FormatConditions.Add(xlTextString, TextOperator:=xlContains, String:="egg")
    With .Font
        .Bold = True
        .ColorIndex = 3
    End With
End With

Operadores: #

Nome Descrição
xlComeçaCom Começa com um valor especificado.
xlContém Contém um valor especificado.
xlNão contém Não contém o valor especificado.
xlEndsWith Termina com o valor especificado

Formatação por período #

With Range("a1:a10").FormatConditions.Add(xlTimePeriod, DateOperator:=xlToday)
    With .Font
        .Bold = True
        .ColorIndex = 3
    End With
End With

Remover formato condicional #

Remova todo o formato condicional no intervalo: #

Range("A1:A10").FormatConditions.Delete

Remova todo o formato condicional na planilha: #

Cells.FormatConditions.Delete

FormatConditions.AddUniqueValues #

Destacando valores duplicados #

With Range("E1:E100").FormatConditions.AddUniqueValues
   .DupeUnique = xlDuplicate
   With .Font
       .Bold = True
       .ColorIndex = 3
   End With
End With

Destacando valores únicos #

With Range("E1:E100").FormatConditions.AddUniqueValues
   With .Font
       .Bold = True
       .ColorIndex = 3
   End With
End With

FormatConditions.AddTop10 #

Destacando os 5 principais valores #

With Range("E1:E100").FormatConditions.AddTop10
    .TopBottom = xlTop10Top
    .Rank = 5
    .Percent = False
    With .Font
        .Bold = True
        .ColorIndex = 3
    End With
End With

FormatConditions.AddAboveAverage #

With Range("E1:E100").FormatConditions.AddAboveAverage
    .AboveBelow = xlAboveAverage
    With .Font
        .Bold = True
        .ColorIndex = 3
    End With
End With

Operadores: #

Nome Descrição
XlAcima da Média Acima da média
XlAboveStdDev Acima do desvio padrão
XlAbaixo da média Abaixo da média
XlBelowStdDev Abaixo do desvio padrão
XlEqualAboveAverage Igual acima da média
XlEqualBelowAverage Igual abaixo da média

FormatConditions.AddIconSetCondition #

Range("a1:a10").FormatConditions.AddIconSetCondition
With Selection.FormatConditions(1)
    .ReverseOrder = False
    .ShowIconOnly = False
    .IconSet = ActiveWorkbook.IconSets(xl3Arrows)
End With

With Selection.FormatConditions(1).IconCriteria(2)
    .Type = xlConditionValuePercent
    .Value = 33
    .Operator = 7
End With

With Selection.FormatConditions(1).IconCriteria(3)
    .Type = xlConditionValuePercent
    .Value = 67
    .Operator = 7
End With
