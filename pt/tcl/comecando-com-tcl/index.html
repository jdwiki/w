Instalação #

Instalando o Tcl 8.6.4 no Windows:

  1. A maneira mais fácil de obter o Tcl em uma máquina Windows é instalar o ActiveTcl distribution from ActiveState.

  2. Navegue até www.activestate.com e siga os links para download the Free Community Edition of ActiveTcl for Windows (choose 32/64 bit version appropriately).

  3. Execute o instalador que resultará em uma nova instalação do ActiveTcl usually in the C:\Tcl directory.

  4. Abra um prompt de comando para testar a instalação, digite “tclsh” que should open an interactive tcl console. Enter “info patchlevel” to check the version of tcl that was installed and it should display an output of the form “8.6.x” depending on the edition of ActiveTcl that has been downloaded.

  • Você também pode adicionar “C:\Tcl\bin” ou seu equivalente à variável PATH do seu ambiente.

    C:>tclsh % info patchlevel 8.6.4

O programa Olá, mundo em Tcl (e Tk) #

O código a seguir pode ser inserido em um shell Tcl (tclsh), ou em um arquivo de script e execute através de um shell Tcl:

puts "Hello, world!"

Ele fornece o argumento string Hello, world! para o comando puts. O comando puts escreve seu argumento na saída padrão (seu terminal no modo interativo) e adiciona uma nova linha depois.

Em um shell habilitado para Tk, essa variação pode ser usada:

pack [button .b -text "Hello, world!" -command exit]

Ele cria um botão gráfico com o texto Hello, world! e o adiciona à janela do aplicativo. Quando pressionado, o aplicativo sai.

Um shell habilitado para Tk é iniciado como: wish Ou usando tclsh junto com a seguinte declaração:

package require Tk

Recursos do Tcl #

  • Portabilidade entre plataformas
  • Funciona em Windows, Mac OS X, Linux e praticamente todas as variantes do unix.
  • Programação orientada a eventos
  • Acionar eventos com base na variável leitura/gravação/desativação.
  • Acionar eventos quando um comando é inserido ou deixado.
  • Dispara eventos quando um canal de E/S (arquivo ou rede) se torna legível/gravável.
  • Crie seus próprios eventos.
  • Acionar um comando baseado em um temporizador.
  • Programação Orientada a Objetos
  • Mixins.
  • Superclasses e subclasses.
  • Gramática Simples
  • Suporte completo a unicode
  • Ele simplesmente funciona. Nenhum comando especial é necessário para lidar com strings unicode.
  • Converta de e para diferentes sistemas de codificação com facilidade.
  • Flexível
  • Criar novas estruturas de controle e comandos.
  • Variáveis ​​de acesso no contexto do procedimento de chamada.
  • Executar código no contexto do procedimento de chamada.
  • Poderosas capacidades de introspecção.
  • Muitos depuradores Tcl foram escritos em Tcl.
  • Interface da biblioteca
  • Integrar bibliotecas C existentes e fornecer uma interface Tcl para a biblioteca.
  • Os “stubs” da biblioteca não estão vinculados a nenhuma versão específica do Tcl e serão still work after a Tcl upgrade.
  • API completa
  • Incorpore um intérprete Tcl em seu idioma favorito.
  • Python, Ruby, R, Java e outros incluem uma API Tcl.
  • Biblioteca bigint incorporada.
  • Nenhuma ação especial é necessária para lidar com números muito grandes.
  • Intérpretes seguros
  • Criar sandboxes nas quais o código do usuário pode ser executado.
  • Habilitar e desabilitar comandos específicos para o intérprete.
  • Expressões regulares
  • Um poderoso e rápido mecanismo de expressão regular escrito por Henry Spencer (criador de regex).

Instalando pacotes através do teacup #

Hoje em dia, muitos idiomas oferecem suporte ao servidor de arquivo para instalar seus pacotes em sua máquina local. TCL também tendo o mesmo servidor de arquivamento que chamamos de Teacup

teacup version
teacup search <packageName>

Exemplo

teacup install Expect
