Introdução aos comandos do Unix

Este tópico fornecerá uma cobertura abrangente dos comandos básicos do Unix.

Uma lista não exaustiva de comandos Unix #

$man <command>

Exibe as páginas de manual on-line para o comando

$clear

Limpa a tela do terminal

$pwd

Retorna o nome do diretório de trabalho

$echo <string>

Grava a string na saída padrão

$printf <string>

Formate e imprima a string Exemplo: imprimir $PATH $printf “%s\n” $PATH

$uptime

Mostrar há quanto tempo o sistema está em execução

$which <program>

Localize um arquivo de programa no caminho do usuário

$whereis <program>

Verifica os diretórios binários padrão para os programas especificados, imprimindo o

ARQUIVOS

$cd [directory]

Alterar diretório Símbolos de diretório comumente usados:

  • . : Diretório atual

  • .. : Diretório Parental

  • ~ : Diretório inicial

  • / : diretório raiz

    $ls

  • $ls -a : Mostra arquivos ocultos

  • $ls -l : Mostrar lista longa

  • $ls -1 : Mostra apenas o nome do arquivo por linha

  • $ls -h : formato legível por humanos

    $file Determinar o tipo de arquivo (por exemplo, gzip)

LEITURA DE ARQUIVOS

$more

Exibir o conteúdo de um arquivo uma tela de cada vez
barra de espaço: Role para a próxima tela; b=tela anterior
enter : Rolar uma linha
h : Ajuda para mais
q : Sair da ajuda

$less <file>

Less é um programa semelhante a more, mas que permite o movimento para trás no arquivo, bem como o movimento para frente

$cat <file>

Lê arquivos sequencialmente, gravando-os na saída padrão

$head [-number] <file>

Exibir as primeiras linhas de um arquivo

$tail [-number] <file>

Exibe o conteúdo do arquivo ou, por padrão, sua entrada padrão, para a saída padrão

  • $tail -f : Visualize as alterações no arquivo em tempo real

    $touch Define os tempos de modificação e acesso dos arquivos. Se algum arquivo não existir, ele será criado com permissões padrão

    $tee Copia a entrada padrão para a saída padrão. Pressione ctrl-d para parar de adicionar conteúdo

  • $tee -a : Anexe a saída aos arquivos em vez de sobrescrevê-los

    $mkdir Crie um diretório

    $wc Exibe o número de linhas, palavras e bytes contidos em cada arquivo de entrada ou entrada padrão

  • wc -l : Contar linhas

  • wc -w : Contar palavras

  • wc -m : Contar caracteres

    $diff Compare dois arquivos linha por linha. Irá imprimir apenas as diferentes linhas.

    $locate Localizar arquivos no disco

  • $locate -q : suprimir erros

    $find Pesquise arquivos por nome ou conteúdo

  • $find -name : Localizar por nome de arquivo

  • $find -size <+/-n> Exemplo: Encontre arquivos no diretório atual com mais de 10k $find . -size +10

    $rm Excluir <arquivo ou diretório>

  • rm -f : Pular confirmação

  • rm -i : Aprova cada exclusão

  • rm -r : Recursivo

Exemplo: Excluir o <diretório> e seu conteúdo $rm -r 

$mv <source_file> <target_file>

Renomeia um arquivo

$mv <source_file> <target_directory>

Move um arquivo

  • $mv -i : não substitui os arquivos existentes
  • $mv -r : Recursivo

Exemplo: mover o diretório para cima na hierarquia $mv ..

$cp

Copie um arquivo/diretório dentro da mesma máquina (Use o comando scp para copiar para uma máquina remota)

  • $cp -i : Não sobrescreve os arquivos existentes
  • $cp -r :recursivo

Exemplo: Copiar e renomear um arquivo

$cp <file_name> <new_file_name>

Exemplo: Copiar para o diretório

$cp <file_name> <directory_name>

Exemplo: Copiar e renomear um diretório

$cp -R <directory> <new_directory>

Exemplo: Copie todos os arquivos de tipo específico para um diretório

$cp *.txt <directory>


$ln -s <file> <link name>

Criar um alias (link) para um arquivo

  • $ln -s : Cria um soft link (um link que funciona entre máquinas)

    $sort Classificar o conteúdo de um arquivo -r reverse sorts -n numeric sort

Exemplo: classifique o e grave o resultado em sorted.txt $sort | uniq -u > sorted.txt

$uniq [-ucd] filename(s)

Procura linhas duplicadas. Os dados devem ser classificados primeiro

  • $uniq -d : mostra apenas uma cópia das linhas duplicadas

  • $uniq -u : mostra apenas as linhas que não são duplicadas

  • $uniq -c : Saída de cada linha precedida por uma contagem de ocorrências Exemplo: mostrar usuários que estão conectados mais de uma vez $who | cut -d’ ‘ -f1 | sort | uniq -d

    $grep <file_name> Imprime linhas que contêm uma correspondência para um padrão.

  • $grep -i : executa correspondência insensível a maiúsculas e minúsculas

  • $grep -v : Imprime todas as linhas que não contêm o regex

  • $grep -r : Pesquisa recursivamente nos subdiretórios listados e imprime os nomes dos arquivos com a ocorrência do padrão

  • $grep -I : Excluir arquivos binários

    $tr “string1” [“string 2”] Ferramenta de pesquisa e substituição. tr só aceita sua entrada de pipes e redirecionamentos. ele não aceita arquivos como entrada.

  • tr -d : Excluir todas as ocorrências de todos os CHARACTERS na string1

Exemplo: Imprima um arquivo a.txt na tela após excluir todas as ocorrências de “;” $cat a.txt | tr -d “;”

  • tr -s : Substitui ocorrências por um único caractere

Exemplo: $echo “SSSS SS” | tr -s “S” “S”

$tar

Cria e manipula arquivos de streaming de arquivos. Esta implementação pode extrair de imagens tar, pax, cpio, zip, jar, ar e ISO e pode criar tar, pax, cpio, ar e shar.


USO DE DISCO

$du [file or directory]

Exiba o uso do bloco do sistema de arquivos para cada arquivo ou diretório. Se nenhum arquivo/diretório for especificado, o uso do bloco do diretório atual será exibido.

  • $du -a : Arquivos e diretórios (o padrão é apenas diretórios)

  • $du -h : formato legível por humanos Exemplo: exibir o uso do disco, ordenado, apenas arquivos MB $du -h | grep -i “m\t” | sort -n Exemplo: Descubra os 10 maiores arquivos/diretórios $du -a /var | sort -n -r | head -n 10

    $df Exibir espaço livre em disco

  • $df -h : formato legível por humanos


REDIREÇÕES E TUBOS

> Redirecionar a saída padrão. Não sobrescreva o arquivo se ele existir

>! Redirecionar a saída padrão. Substituir o arquivo se ele existir

>& Redirecionar saída padrão e erro padrão

Exemplo: Redirecionar a saída do comando para um arquivo

$ls > result.txt

Use o arquivo > /dev/null para descartar a mensagem de erros

Exemplo: Encontre um arquivo chamado my_file_name e imprima o resultado em ~/find.txt; Ocultar erros (por exemplo, “permissões negadas”)

$find / -name my_file_name.* > /dev/null > ~/find.txt

< Redirecionar entrada padrão

>> Anexar saída padrão <command> » : Anexa a saída ao final de um existente

<command> < : Redireciona a entrada para um comando de um arquivo

| Redirecionar a saída padrão para outro comando (pipe)

Exemplo: Mostrar detalhes paginados de processos em execução

$ps -ex | more

| : Saída do canal do comando1 para ser a entrada do comando2 (se um arquivo de saída for desejado no meio de um canal, use o comando tee)

Exemplo: contar o número de usuários conectados

$who | wc -l



PROCESSOS

$ps

Mostrar processos ativos

  • $ps -e : Mostra informações sobre o processo
  • $ps -x : Mostra processos ocultos

Exemplo: encontrar processos por nome $grep -l <process_name_regex>

$kill [-signal] pid

Mate um processo.

Alguns dos sinais mais usados:

  • 3 : SAIR (sair)

  • 9 : KILL (morte não capturável, não ignorável)

  • 15 : TERM (sinal de terminação de software)

    $top Exibir e atualizar informações classificadas sobre processos Consulte as páginas do manual para obter uma lista de chaves possíveis. chaves comuns são: cpu, threads, portas

  • $top -o

    $htop Exibir e atualizar informações classificadas sobre processos


    USUÁRIO E PERMISSÕES

    $sudo Execute o comando como um superusuário

    $su (usuário substituto) abre uma sessão como administrador

    $exit Sair da raiz

    $whoami Exibir ID de usuário efetivo

    $who Imprima todos os nomes de usuários conectados

    $passwd Mudar senha

    $chmod Alterar o acesso do proprietário/grupo a um arquivo ou diretório

Quem: u usuário; grupo g; o outro; um tudo acima

Operação: + adicionar; - remover; = set (significa redefinir para nada e definir apenas o que foi especificado)

Permissões: r w x

Exemplo: adiciona permissões de leitura/execução ao grupo

$chmod g +rx <file>

Exemplo:

$chmod 743 <file>

Observe que para $ cd em um diretório você precisa das permissões x
