Este tópico abordaria como o projeto angular-cli está pronto para a compilação de produção, quais são todas as etapas necessárias antes da implantação, como criar o arquivo war para implantação do projeto e, finalmente, como configurar o tomcat apache para implantação do projeto angular-cli.

Etapas necessárias antes de implantar o projeto angular-cli para compilação de produção. #

Antes de implantar o projeto angular no servidor, precisamos construir o projeto angular para produção. Também precisamos alterar o caminho de roteamento no arquivo index.html de <base href="/"> para <base href="/"> se não for feito, seu projeto não será carregado corretamente, haverá ser algum erro de roteamento dizendo que o arquivo 404 não foi encontrado.

Comando de compilação Angular-cli para compilar o pacote de projeto para implantação de produção #

ng build –prod

O comando acima fornecido com uma opção extra como –prod geraria o pacote do projeto de compilação de produção. Assim que o comando acima for executado no diretório raiz do seu projeto, aparecerá um diretório chamado dist. No qual todo o pacote de compilação de produção do seu projeto aparece nele.

Criando o arquivo war para implantação de produção do projeto angular-cli no servidor apache tomcat #

Quando o diretório dist estiver pronto com seus pacotes de produção criados. Basta abrir o diretório dist e abrir o prompt de comando digite o seguinte comando para criar o arquivo war para implantar seu projeto no servidor apache tomcat.

   jar cvf dist.war .

Uma vez que os comandos jar acima sejam executados. Isso geraria um arquivo dist.war dentro do diretório dist.

Configurando o tomcat do apache para implantação do projeto angular-cli. #

  1. Corte/Copie o arquivo dist.war do diretório dist e coloque-o no diretório apache tomcat webapp.
  2. Vá para a pasta apache tomcat bin e clique duas vezes no arquivo startup.bat.
  3. Agora o servidor tomcat executará o arquivo dist.war e inicializará o servidor tomcat catalina.
  4. Assim que o servidor tomcat catalina for iniciado, abra o navegador da web e digite localhost:8080/dist e toque na tecla Enter, seu projeto será executado na janela do navegador da web.
