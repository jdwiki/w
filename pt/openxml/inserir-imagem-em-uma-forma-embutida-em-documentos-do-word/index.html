Insira uma imagem em formas de documento do MS Word, como retângulos e ovais.

Esta documentação pressupõe que você saiba como inserir uma imagem em um documento do Word, abrir e fechar um documento do Word usando OpenXML

Adicione os seguintes namespaces OpenXML à sua classe #

using System;
using System.Collections.Generic;
using System.Linq;
using DocumentFormat.OpenXml;
using A = DocumentFormat.OpenXml.Drawing;
using DW = DocumentFormat.OpenXml.Drawing.Wordprocessing;
using PIC = DocumentFormat.OpenXml.Drawing.Pictures;
using DocumentFormat.OpenXml.Drawing.Wordprocessing;
using Wps = DocumentFormat.OpenXml.Office2010.Word.DrawingShape;

Abra o documento e adicione o objeto imagePart para referenciar a imagem que você deseja inserir na forma #

Agora abra o documento usando OpenXML, você deve adicionar um imagePart que referencia o objeto de imagem para o objeto MainDocumentPart usando um fluxo de arquivo e obter o ID da imagem

    string temp;
MainDocumentPart mainPart = document.MainDocumentPart;
                               ImagePart imagePart = mainPart.AddImagePart(ImagePartType.Bmp);

                               using (FileStream stream = new FileStream(barcodepath, FileMode.Open))
                               {
                                  imagePart.FeedData(stream);
                               }

                               temp = mainPart.GetIdOfPart(imagePart);

Obtém uma referência de um objeto Blip #

No escritório OpenXML, uma imagem inserida em um documento do Word é considerada um objeto ou elemento “Blip”. A classe é derivada de DocumentFormat.OpenXml.Drawing o Blip deve ter um valor Embed que seja um ID de imagem. O objeto Blip então vai dentro de um BlipFill Object/element, e que também vai dentro de um graphicData Object/element e que por sua vez vai para um elemento de objeto graphic. Tenho certeza que agora você percebeu que tudo funciona como uma árvore XML. Exemplo de árvore Open XML abaixo.

<a:graphic xmlns:a="http://schemas.openxmlformats.org/drawingml/2006/main">
                       <a:graphicData uri="http://schemas.microsoft.com/office/word/2010/wordprocessingShape">
                         <wps:wsp>
                           <wps:cNvSpPr>
                             <a:spLocks noChangeArrowheads="1" />
                           </wps:cNvSpPr>
                           <wps:spPr bwMode="auto">
                             <a:xfrm>
                               <a:off x="0" y="0" />
                               <a:ext cx="1234440" cy="1234440" />
                             </a:xfrm>
                             <a:prstGeom prst="roundRect">
                               <a:avLst>
                                 <a:gd name="adj" fmla="val 16667" />
                               </a:avLst>
                             </a:prstGeom>
                             <a:blipFill dpi="0" rotWithShape="1">
                               <a:blip r:embed="Raade88ffea8d4c1b" />
                               <a:stretch>
                                 <a:fillRect l="10000" t="10000" r="10000" b="10000" />
                               </a:stretch>
                             </a:blipFill>
                           </wps:spPr>
                           <wps:bodyPr rot="0" vert="horz" wrap="square" lIns="91440" tIns="45720" rIns="91440" bIns="45720" anchor="t" anchorCtr="0" upright="1">
                             <a:noAutofit />
                           </wps:bodyPr>
                         </wps:wsp>
                       </a:graphicData>
                     </a:graphic>

Adicionando referência da imagem as formas no documento modelo. #

Agora você tem uma referência da imagem. Insira a imagem nas formas no documento de modelo. Para fazer isso, você terá que usar algum LINQ para percorrer o documento e obter uma referência a todas as formas no documento. O elemento wps:spPr que você vê no código XML acima é o elemento xml para as formas em um documento. A classe C# equivalente é WordprocessingShape

IEnumerable<DocumentFormat.OpenXml.Office2010.Word.DrawingShape.WordprocessingShape> shapes2 = document.MainDocumentPart.document.Body.Descendants<DocumentFormat.OpenXml.Office2010.Word.DrawingShape.WordprocessingShape>();

Com uma referência de coleção de todas as formas, percorra a coleção. #

Agora que você tem uma coleção de todas as referências de forma no documento. Percorra a coleção com um foreach e, em cada iteração, crie um objeto Blip. Defina o valor de incorporação do objeto Blip para a referência de ID da imagem que você capturou anteriormente da parte da imagem. Crie também um objeto Stretch e um objeto FillRectangle (estes não são realmente necessários, eles são usados ​​apenas para o alinhamento adequado da imagem). E anexe cada um ao seu objeto pai como o equivalente da árvore XML.

foreach (DocumentFormat.OpenXml.Office2010.Word.DrawingShape.WordprocessingShape sp in shapes2)
                              {
                                  //Wps.ShapeProperties shapeProperties1 =

                                  A.Blip blip1 = new A.Blip() { Embed = temp };

                                  A.Stretch stretch1 = new A.Stretch();
                                  A.FillRectangle fillRectangle1 = new A.FillRectangle() { Left = 10000, Top = 10000, Right = 10000, Bottom = 10000 };
                                  Wps.WordprocessingShape wordprocessingShape1 = new Wps.WordprocessingShape();
