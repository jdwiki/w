Detectando a posição do mouse na tela #

Este exemplo mostrará como obter a posição do mouse em relação à tela, de modo que (0,0) seja o canto superior esquerdo da tela HTML5. O e.clientX e o e.clientY obterão as posições do mouse em relação ao topo do documento, para mudar isso para ser baseado no topo da tela subtraímos as posições left e right do canvas do cliente X e Y.

var canvas = document.getElementById("myCanvas");
var ctx = canvas.getContext("2d");
ctx.font = "16px Arial";

canvas.addEventListener("mousemove", function(e) { 
    var cRect = canvas.getBoundingClientRect();        // Gets CSS pos, and width/height
    var canvasX = Math.round(e.clientX - cRect.left);  // Subtract the 'left' of the canvas 
    var canvasY = Math.round(e.clientY - cRect.top);   // from the X/Y positions to make  
    ctx.clearRect(0, 0, canvas.width, canvas.height);  // (0,0) the top left of the canvas
    ctx.fillText("X: "+canvasX+", Y: "+canvasY, 10, 20);
});

Exemplo executável

O uso de Math.round é para garantir que as posições x,y sejam inteiras, pois o retângulo delimitador da tela pode não ter posições inteiras.

Tamanho e resolução da tela #

O tamanho de uma tela é a área que ela ocupa na página e é definido pelas propriedades de largura e altura do CSS.

canvas {
   width : 1000px;
   height : 1000px;
}

A resolução da tela define o número de pixels que ela contém. A resolução é definida definindo as propriedades de largura e altura do elemento de tela. Se não for especificado, o padrão da tela será de 300 por 150 pixels.

A tela a seguir usará o tamanho CSS acima, mas como width e height não são especificados, a resolução será de 300 por 150.

<canvas id="my-canvas"></canvas>

Isso fará com que cada pixel seja esticado de forma desigual. O aspecto de pixel é 1:2. Quando a tela for esticada, o navegador usará filtragem bilinear. Isso tem um efeito de desfoque de pixels que são esticados.

Para obter os melhores resultados ao usar a tela, certifique-se de que a resolução da tela corresponda ao tamanho da tela.

Seguindo o estilo CSS acima para corresponder ao tamanho de exibição, adicione a tela com o width e o height definidos para a mesma contagem de pixels que o estilo define.

<canvas id = "my-canvas" width = "1000" height = "1000"></canvas>

Como adicionar o elemento Html5 Canvas a uma página da web #

Html5-Canvas …

  • É um elemento Html5.
  • É suportado na maioria dos navegadores modernos (Internet Explorer 9+).
  • É um elemento visível que é transparente por padrão
  • Tem uma largura padrão de 300px e uma altura padrão de 150px.
  • Requer JavaScript porque todo o conteúdo deve ser adicionado programaticamente ao Canvas.

Exemplo: crie um elemento Html5-Canvas usando marcação Html5 e JavaScript:

<!doctype html>
<html>
<head>
<style>
    body{ background-color:white; }
    #canvasHtml5{border:1px solid red; }
    #canvasJavascript{border:1px solid blue; }
</style>
<script>
window.onload=(function(){

    // add a canvas element using javascript
    var canvas=document.createElement('canvas');
    canvas.id='canvasJavascript'
    document.body.appendChild(canvas);

}); // end $(function(){});
</script>
</head>
<body>

    <!-- add a canvas element using html -->
    <canvas id='canvasHtml5'></canvas>

</body>
</html>

Tela fora da tela #

Muitas vezes, ao trabalhar com a tela, você precisará ter uma tela para armazenar alguns dados de pixel de entrada. É fácil criar uma tela fora da tela, obter um contexto 2D. Uma tela fora da tela também usará o hardware gráfico disponível para renderizar.

O código a seguir simplesmente cria uma tela e a preenche com pixels azuis.

function createCanvas(width, height){
    var canvas = document.createElement("canvas"); // create a canvas element
    canvas.width = width;
    canvas.height = height;
    return canvas;
}

var myCanvas = createCanvas(256,256); // create a small canvas 256 by 256 pixels
var ctx = myCanvas.getContext("2d");
ctx.fillStyle = "blue";
ctx.fillRect(0,0,256,256);

Muitas vezes a tela fora da tela será usada para muitas tarefas e você pode ter muitas telas. Para simplificar o uso da tela, você pode anexar o contexto da tela à tela.

function createCanvasCTX(width, height){
    var canvas = document.createElement("canvas"); // create a canvas element
    canvas.width = width;
    canvas.height = height;
    canvas.ctx = canvas.getContext("2d");
    return canvas;
}
var myCanvas = createCanvasCTX(256,256); // create a small canvas 256 by 256 pixels
myCanvas.ctx.fillStyle = "blue";
myCanvas.ctx.fillRect(0,0,256,256);

Capacidades do Canvas #

O Canvas permite desenhar programaticamente em sua página da web:

  • Imagens,
  • Textos,
  • Linhas e curvas.

Os desenhos em tela podem ser extensivamente estilizados:

  • largura do traço,
  • cor do traço,
  • cor de preenchimento da forma,
  • [opacity](https://www.wikiod.com/pt/html5-canvas/composicao#Altere a opacidade com “globalAlpha”),
  • sombreamento,
  • gradientes lineares e [gradientes radiais](http: //stackoverflow.com/documentation/html5-canvas/3241/path-syntax-only/13476/createraradialgradient-creates-a-path-styling-object),
  • [face da fonte](https://www.wikiod.com/pt/html5-canvas/texto#Formatação de texto),
  • [tamanho da fonte](https://www.wikiod.com/pt/html5-canvas/texto#Formatação de texto),
  • [alinhamento de texto](https://www.wikiod.com/pt/html5-canvas/texto#Formatação de texto),
  • [o texto pode ser traçado, preenchido ou ambos traçado e preenchido](https://www.wikiod.com/pt/html5-canvas/texto#Texto de desenho),
  • redimensionamento de imagem,
  • recorte de imagem,
  • composição

Usos do Canvas #

Os desenhos podem ser combinados e posicionados em qualquer lugar na tela para que possam ser usados ​​para criar:

  • Aplicativos de pintura/esboço,
  • Jogos interativos rápidos,
  • Análises de dados como gráficos, gráficos,
  • Imagens semelhantes ao Photoshop,
  • Publicidade tipo Flash e conteúdo da web chamativo.

O Canvas permite que você manipule as cores dos componentes Vermelho, Verde, Azul e Alfa das imagens. Isso permite que a tela manipule imagens com resultados semelhantes ao Photoshop.

  • Recolorir qualquer parte de uma imagem no nível de pixel (se você usar HSL, você pode até recolorir uma imagem mantendo a importante Iluminação e Saturação para que o resultado não pareça que alguém colocou tinta na imagem),
  • “Knockout” o fundo em torno de uma pessoa/item em uma imagem,
  • Detecte e preencha parte de uma imagem (por exemplo, altere a cor de uma pétala de flor clicada pelo usuário de verde para amarelo - apenas essa pétala clicada!),
  • Faça distorção de perspectiva (por exemplo, enrole uma imagem ao redor da curva de um copo),
  • Examinar o conteúdo de uma imagem (por exemplo, reconhecimento facial),
  • Responda a perguntas sobre uma imagem: Há um carro estacionado nesta imagem da minha vaga de estacionamento?,
  • Aplique filtros de imagem padrão (escala de cinza, sépia, etc)
  • Aplique qualquer filtro de imagem exótico que você possa imaginar (Sobel Edge Detection),
  • Combine imagens. Se a querida vovó Sue não pôde comparecer à reunião de família, apenas “photoshop” ela na imagem da reunião. Não gosto do primo Phil - apenas “photoshop-lo”
  • Reproduza um vídeo / Pegue um quadro de um vídeo,
  • Exporte o conteúdo da tela como um .jpg | imagem .png (você pode até opcionalmente cortar ou anotar a imagem e exportar o resultado como uma nova imagem),

Sobre mover e editar desenhos de tela (por exemplo, para criar um jogo de ação):

  • Depois que algo foi desenhado na tela, esse desenho existente não pode ser movido ou editado. Vale a pena esclarecer este equívoco comum de que os desenhos em tela são móveis: Os desenhos em tela existentes não podem ser editados ou movidos!
  • Canvas desenha muito, muito rapidamente. O Canvas pode desenhar centenas de imagens, textos, linhas e curvas em uma fração de segundo. Ele usa a GPU quando disponível para acelerar o desenho.
  • A tela cria a ilusão de movimento desenhando rapidamente e repetidamente algo e, em seguida, redesenhando-o em uma nova posição. Como a televisão, esse redesenho constante dá ao olho a ilusão de movimento.

Girar #

O método rotate(r) do contexto 2D gira a tela pela quantidade especificada r de radianos em torno da origem.

HTML

<canvas id="canvas" width=240 height=240 style="background-color:#808080;">
</canvas>

<button type="button" onclick="rotate_ctx();">Rotate context</button>

Javascript

var canvas = document.getElementById("canvas");
var ctx = canvas.getContext("2d");
var ox = canvas.width / 2;
var oy = canvas.height / 2;
ctx.font = "42px serif";
ctx.textAlign = "center";
ctx.textBaseline = "middle";
ctx.fillStyle = "#FFF";
ctx.fillText("Hello World", ox, oy);

rotate_ctx = function() {
  // translate so that the origin is now (ox, oy) the center of the canvas
  ctx.translate(ox, oy);
  // convert degrees to radians with radians = (Math.PI/180)*degrees. 
  ctx.rotate((Math.PI / 180) * 15);
  ctx.fillText("Hello World", 0, 0);
  // translate back
  ctx.translate(-ox, -oy);
};

Salvar tela em arquivo de imagem #

Você pode salvar uma tela em um arquivo de imagem usando o método canvas.toDataURL(), que retorna o data URI para os dados da imagem da tela.

O método pode receber dois parâmetros opcionais canvas.toDataURL(type, encoderOptions): type é o formato da imagem (se omitido o padrão é image/png); encoderOptions é um número entre 0 e 1 que indica a qualidade da imagem (o padrão é 0,92).

Aqui desenhamos uma tela e anexamos o URI de dados da tela ao link “Download to myImage.jpg”.

HTML

<canvas id="canvas" width=240 height=240 style="background-color:#808080;">
</canvas>
<p></p>
<a id="download" download="myImage.jpg" href="" onclick="download_img(this);">Download to myImage.jpg</a>

Javascript

var canvas = document.getElementById("canvas");
var ctx = canvas.getContext("2d");
var ox = canvas.width / 2;
var oy = canvas.height / 2;
ctx.font = "42px serif";
ctx.textAlign = "center";
ctx.textBaseline = "middle";
ctx.fillStyle = "#800";
ctx.fillRect(ox / 2, oy / 2, ox, oy);

download_img = function(el) {
  // get image URI from canvas object
  var imageURI = canvas.toDataURL("image/jpg");
  el.href = imageURI;
};
