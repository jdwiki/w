Introdução ao “context.getImageData” #

O Html5 Canvas oferece a capacidade de buscar e alterar a cor de qualquer pixel na tela.

Você pode usar a manipulação de pixels do Canvas para:

  • Crie um seletor de cores para uma imagem ou selecione uma cor em uma roda de cores.
  • Crie filtros de imagem complexos, como desfoque e detecção de bordas.
  • Recolorir qualquer parte de uma imagem no nível de pixel (se você usar HSL, você pode até recolorir uma imagem mantendo a importante Iluminação e Saturação para que o resultado não pareça que alguém colocou tinta na imagem). Nota: O Canvas agora possui Blend Compositing que também pode recolorir uma imagem em alguns casos.
  • “Knockout” o fundo em torno de uma pessoa/item em uma imagem,
  • Crie uma ferramenta de balde de tinta para detectar e preencher parte de uma imagem (por exemplo, alterar a cor de uma pétala de flor clicada pelo usuário de verde para amarelo).
  • Examine uma imagem para o conteúdo (por exemplo, reconhecimento facial).

Problemas comuns:

  • Por motivos de segurança, getImageData é desabilitado se você desenhou uma imagem originada em um domínio diferente da própria página web.
  • getImageData é um método relativamente caro porque cria uma grande matriz de dados de pixel e porque não usa a GPU para auxiliar seus esforços. Nota: Canvas agora tem composição de mistura que pode fazer algumas das mesmas manipulações de pixel que getImageData faz.
  • Para imagens .png, getImageData pode não relatar exatamente as mesmas cores do arquivo .png original porque o navegador tem permissão para fazer correção de gama e pré-multiplicação alfa ao desenhar imagens na tela.

Obtendo cores de pixel

Use getImageData para buscar as cores de pixel para todo ou parte do conteúdo da tela.

O método getImageData retorna um objeto imageData

O objeto imageData possui uma propriedade .data que contém as informações de cor do pixel.

A propriedade data é um Uint8ClampedArray contendo os dados de cor Vermelho, Verde, Azul e Alfa (opacidade) para todos os pixels solicitados.

// determine which pixels to fetch (this fetches all pixels on the canvas)
var x=0;
var y=0;
var width=canvas.width;
var height=canvas.height;

// Fetch the imageData object 
var imageData = context.getImageData(x,y,width,height);

// Pull the pixel color data array from the imageData object
var pixelDataArray = imageData.data;

Você pode obter a posição de qualquer pixel [x,y] dentro da matriz data assim:

// the data[] array position for pixel [x,y]
var n = y * canvas.width + x;

E então você pode buscar os valores vermelho, verde, azul e alfa desse pixel assim:

// the RGBA info for pixel [x,y]
var red=data[n];
var green=data[n+1];
var blue=data[n+2];
var alpha=data[n+3];
