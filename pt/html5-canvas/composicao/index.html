Desenhe atrás de formas existentes com “destination-over” #

context.globalCompositeOperation = "destination-over"

A composição “destino sobre” coloca um novo desenho sob os desenhos existentes.

context.drawImage(rainy,0,0);
context.globalCompositeOperation='destination-over';  // sunny UNDER rainy
context.drawImage(sunny,0,0);

Apague formas existentes com “destination-out” #

context.globalCompositeOperation = "destination-out"

a composição “destination-out” usa novas formas para apagar desenhos existentes.

A nova forma não é realmente desenhada - é apenas usada como um “cortador de biscoitos” para apagar os pixels existentes.

context.drawImage(apple,0,0);
context.globalCompositeOperation = 'destination-out';   // bitemark erases
context.drawImage(bitemark,100,40);

Composição padrão: novas formas são desenhadas sobre formas existentes #

context.globalCompositeOperation = "source-over"

composição “source-over” [padrão], coloca todos os novos desenhos sobre quaisquer desenhos existentes.

context.globalCompositeOperation='source-over';  // the default
context.drawImage(background,0,0);
context.drawImage(parachuter,0,0);

Recorte imagens dentro de formas com “destination-in” #

context.globalCompositeOperation = "destination-in"

A composição “destination-in” recorta os desenhos existentes dentro de uma nova forma.

Observação: qualquer parte do desenho existente que esteja fora do novo desenho será apagada.

context.drawImage(picture,0,0);
context.globalCompositeOperation='destination-in';  // picture clipped inside oval
context.drawImage(oval,0,0);

Recorte imagens dentro de formas com “source-in” #

context.globalCompositeOperation = "source-in";

A composição source-in recorta novos desenhos dentro de uma forma existente.

Observação: qualquer parte do novo desenho que estiver fora do desenho existente será apagada.

context.drawImage(oval,0,0);
context.globalCompositeOperation='source-in';  // picture clipped inside oval
context.drawImage(picture,0,0);

Sombras internas com “source-atop” #

context.globalCompositeOperation = 'source-atop'

A composição source-atop recorta uma nova imagem dentro de uma forma existente.

// gold filled rect
ctx.fillStyle='gold';
ctx.fillRect(100,100,100,75);
// shadow
ctx.shadowColor='black';
ctx.shadowBlur=10;
// restrict new draw to cover existing pixels
ctx.globalCompositeOperation='source-atop';
// shadowed stroke
// "source-atop" clips off the undesired outer shadow
ctx.strokeRect(100,100,100,75);
ctx.strokeRect(100,100,100,75);

Inverter ou Negar imagem com “diferença” #

Renderize um retângulo branco sobre uma imagem com a operação de composição

ctx.globalCompositeOperation = 'difference';

A quantidade do efeito pode ser controlada com a configuração alfa

// Render the image
ctx.globalCompositeOperation='source-atop';
ctx.drawImage(image, 0, 0);

// set the composite operation
ctx.globalCompositeOperation='difference';
ctx.fillStyle = "white";
ctx.globalAlpha = alpha;  // alpha 0 = no effect 1 = full effect
ctx.fillRect(0, 0, image.width, image.height);

Preto e branco com “cor” #

Remova a cor de uma imagem por meio de

ctx.globalCompositeOperation = 'color';

A quantidade do efeito pode ser controlada com a configuração alfa

// Render the image
ctx.globalCompositeOperation='source-atop';
ctx.drawImage(image, 0, 0);

// set the composite operation
ctx.globalCompositeOperation='color';
ctx.fillStyle = "white";
ctx.globalAlpha = alpha;  // alpha 0 = no effect 1 = full effect
ctx.fillRect(0, 0, image.width, image.height);

Aumente o contraste de cores com “saturação” #

Aumente o nível de saturação de uma imagem com

ctx.globalCompositeOperation = 'saturation';

A quantidade do efeito pode ser controlada com a configuração alfa ou a quantidade de saturação na sobreposição de preenchimento

// Render the image
ctx.globalCompositeOperation='source-atop';
ctx.drawImage(image, 0, 0);

// set the composite operation
ctx.globalCompositeOperation ='saturation';
ctx.fillStyle = "red";
ctx.globalAlpha = alpha;  // alpha 0 = no effect 1 = full effect
ctx.fillRect(0, 0, image.width, image.height);

Sépia FX com “luminosidade” #

Crie um FX sépia colorido com

ctx.globalCompositeOperation = 'luminosity';

Neste caso, a cor sépia é renderizada primeiro na imagem.

A quantidade do efeito pode ser controlada com a configuração alfa ou a quantidade de saturação na sobreposição de preenchimento

// Render the image
ctx.globalCompositeOperation='source-atop';
ctx.fillStyle = "#F80";  // the color of the sepia FX
ctx.fillRect(0, 0, image.width, image.height);

// set the composite operation
ctx.globalCompositeOperation ='luminosity';

ctx.globalAlpha = alpha;  // alpha 0 = no effect 1 = full effect
ctx.drawImage(image, 0, 0);

Altere a opacidade com “globalAlpha” #

context.globalAlpha=0.50

Você pode alterar a opacidade de novos desenhos definindo o globalAlpha para um valor entre 0,00 (totalmente transparente) e 1,00 (totalmente opaco).

O padrão globalAlpha é 1,00 (totalmente opaco).

Os desenhos existentes não são afetados pelo globalAlpha.

// draw an opaque rectangle
context.fillRect(10,10,50,50);

// change alpha to 50% -- all new drawings will have 50% opacity
context.globalAlpha=0.50;

// draw a semi-transparent rectangle
context.fillRect(100,10,50,50);
