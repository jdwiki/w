Sintaxe #

  • void clearRect(x, y, largura, altura)
  • ImageData createImageData (largura, altura)

Nenhum desses métodos produzirá pixels transparentes se o contexto foi criado com o parâmetro alpha: false.

Retângulos #

Você pode usar o método clearRect para limpar qualquer seção retangular da tela.

// Clear the entire canvas
ctx.clearRect(0, 0, canvas.width, canvas.height);

Nota: clearRect depende da matriz de transformação.

Para lidar com isso, é possível redefinir a matriz de transformação antes de limpar a tela.

ctx.save();                                       // Save the current context state
ctx.setTransform(1, 0, 0, 1, 0, 0);               // Reset the transformation matrix
ctx.clearRect(0, 0, canvas.width, canvas.height); // Clear the canvas
ctx.restore();                                    // Revert context state including 
                                                  // transformation matrix

Observação: ctx.save e ctx.restore são necessários apenas se você deseja manter o estado de contexto 2D da tela. Em algumas situações, salvar e restaurar podem ser lentos e geralmente devem ser evitados se não forem necessários.

Dados brutos da imagem #

É possível escrever diretamente nos dados da imagem renderizada usando putImageData. Ao criar novos dados de imagem e atribuí-los à tela, você limpará a tela inteira.

var imageData = ctx.createImageData(canvas.width, canvas.height);
ctx.putImageData(imageData, 0, 0);

Observação: putImageData não é afetado por nenhuma transformação aplicada ao contexto. Ele gravará dados diretamente na região de pixel renderizada.

Formas complexas #

É possível limpar regiões de formato complexo alterando a propriedade globalCompositeOperation.

// All pixels being drawn will be transparent
ctx.globalCompositeOperation = 'destination-out';

// Clear a triangular section
ctx.globalAlpha = 1;    // ensure alpha is 1
ctx.fillStyle = '#000'; // ensure the current fillStyle does not have any transparency
ctx.beginPath();
ctx.moveTo(10, 0);
ctx.lineTo(0, 10);
ctx.lineTo(20, 10);
ctx.fill();

// Begin drawing normally again
ctx.globalCompositeOperation = 'source-over';

Limpar tela com gradiente. #

Em vez de usar clearRect, que torna todos os pixels transparentes, você pode querer um plano de fundo.

Para limpar com um gradiente

// create the background gradient once
var bgGrad = ctx.createLinearGradient(0,0,0,canvas.height);
bgGrad.addColorStop(0,"#0FF");
bgGrad.addColorStop(1,"#08F");

// Every time you need to clear the canvas
ctx.fillStyle = bgGrad;
ctx.fillRect(0,0,canvas.width,canvas.height);

Isso é cerca de metade da velocidade 0,008ms do clearRect 0,004ms, mas os 4 milhões de segundos não devem afetar negativamente qualquer animação em tempo real. (Os tempos variam consideravelmente dependendo do dispositivo, resolução, navegador e configuração do navegador. Os tempos são apenas para comparação)

Limpar tela usando operação composta #

Limpe a tela usando a operação de composição. Isso limpará a tela independente de transformações, mas não é tão rápido quanto clearRect().

ctx.globalCompositeOperation = 'copy';

qualquer coisa desenhada a seguir limpará o conteúdo anterior.
