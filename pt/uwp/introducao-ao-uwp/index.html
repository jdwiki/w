Criando seu primeiro aplicativo UWP #

Este exemplo demonstra como desenvolver um aplicativo UWP simples.

Na criação de um projeto “Blank App (Universal Windows)” existem muitos arquivos essenciais que são criados em sua solução.

Todos os arquivos em seu projeto podem ser vistos no Solution Explorer.

Alguns dos arquivos cruciais em seu projeto são:

  • App.xaml e App.xaml.cs - App.xaml é usado para declarar recursos que estão disponíveis no aplicativo e App.xaml.cs é o código de back-end para ele. App.xaml.cs é o ponto de entrada padrão do aplicativo
  • MainPage.xaml - Esta é a página de inicialização padrão do seu aplicativo (você também pode alterar a página de inicialização do aplicativo em App.xaml.cs)
  • Package.appxmanifest - Este arquivo contém informações importantes do seu aplicativo, como nome de exibição, ponto de entrada, recursos visuais, lista de recursos, informações de embalagem etc.

Começando #

  • Adicionando um botão à sua página

Para adicionar qualquer elemento ou ferramenta de interface do usuário à sua página, basta arrastar e soltar o elemento da janela da caixa de ferramentas à esquerda. Procure uma ferramenta “Botão” na caixa de ferramentas e solte-a na página do seu aplicativo.

  • Personalizando a interface do usuário

Todas as propriedades de uma ferramenta específica são mostradas na janela de propriedades no lado inferior direito.

Aqui vamos mudar o texto dentro do botão para “Fale!”. Para fazer isso, primeiro toque no botão para selecioná-lo e, em seguida, percorra a janela de propriedades para encontrar Conteúdo e altere o texto para a string desejada (“Fale!”).

Também mudaremos a cor de fundo da página. Cada página tem um elemento pai (geralmente uma grade) que contém todos os outros elementos. Assim, vamos mudar a cor da grade pai. Para fazer isso, toque na grade e altere Brush > Background da janela de propriedades para a cor desejada.

A interface do usuário será semelhante a esta depois de personalizá-la.

  • Código por trás

    Now lets do something on click of our button!

    Clicking on a button triggers an event and we need to handle the event to do something useful when the button is clicked.

    Adding event handler

Para adicionar um manipulador de eventos de clique ao seu botão, selecione o botão vá para a janela de propriedades e selecione o ícone de raio. Esta janela consiste em todos os eventos que estão disponíveis para o elemento que selecionamos (o botão no nosso caso). Em seguida, clique duas vezes na caixa de texto ao lado do evento “Click” para gerar automaticamente o manipulador para o evento de clique do botão.

Depois disso, você será redirecionado para uma página c# (MainPage.xaml.cs). Adicione o seguinte código ao seu método de manipulador de eventos:

 MediaElement mediaElement = new MediaElement();
        var synth = new Windows.Media.SpeechSynthesis.SpeechSynthesizer();
        Windows.Media.SpeechSynthesis.SpeechSynthesisStream stream = await synth.SynthesizeTextToStreamAsync("Hello, World!");
        mediaElement.SetSource(stream, stream.ContentType);
        mediaElement.Play();

Em seguida, adicione a palavra-chave async ao seu manipulador de eventos.

Depois de adicionar o código acima, sua classe deve ficar assim:

public sealed partial class MainPage : Page
{
    string speakIt = "Hello, World!";
    public MainPage()
    {
        this.InitializeComponent();
    }

    private async void button_Click(object sender, RoutedEventArgs e)
    {
        MediaElement mediaElement = new MediaElement();
        var synth = new Windows.Media.SpeechSynthesis.SpeechSynthesizer();
        Windows.Media.SpeechSynthesis.SpeechSynthesisStream stream = await synth.SynthesizeTextToStreamAsync(speakIt);
        mediaElement.SetSource(stream, stream.ContentType);
        mediaElement.Play();

    }
}

  • Inicie seu aplicativo!

    Your application is ready to be launched. You can launch your application by pressing F5 or Select your device on which you want to deploy and debug your application and click on start button.

Depois de compilado, seu aplicativo será implantado em seu dispositivo. Dependendo da resolução do seu dispositivo e do tamanho da tela, o aplicativo configurará automaticamente seu layout. (Você pode redimensionar a janela para ver como ela funciona perfeitamente)

  • Indo além

Agora que você fez sua primeira aplicação, vamos dar um passo adiante!

Adicione uma caixa de texto à sua página e, ao clicar no botão, o aplicativo falará o que estiver escrito na caixa de texto.

Comece arrastando e soltando um controle TextBox da caixa de ferramentas para seu layout. Em seguida, dê um nome ao seu TextBox no menu de propriedades. (por que precisamos especificar um nome? para que possamos usar facilmente esse controle)

O Visual Studio, por padrão, dá um nome ao seu controle, mas é um bom hábito nomear os controles de acordo com o que eles fazem ou algo relevante.

Estou nomeando minha caixa de texto - “speakText”.

    private async void button_Click(object sender, RoutedEventArgs e)
    {
        //checking if the text provided in the textbox is null or whitespace
        if (!string.IsNullOrWhiteSpace(speakText.Text))
            speakIt = speakText.Text;
        else
            speakIt = "Please enter a valid string!";

        MediaElement mediaElement = new MediaElement();
        var synth = new Windows.Media.SpeechSynthesis.SpeechSynthesizer();
        Windows.Media.SpeechSynthesis.SpeechSynthesisStream stream = await synth.SynthesizeTextToStreamAsync(speakIt);
        mediaElement.SetSource(stream, stream.ContentType);
        mediaElement.Play();

    }

Agora implante seu código!!

Seu aplicativo agora pode falar qualquer string válida que você fornecer a ele !!

Parabéns! Você construiu com sucesso seu próprio aplicativo UWP !!

Instalação ou Configuração #

Instruções detalhadas sobre como configurar ou instalar o UWP.

Requisitos

  1. Windows 10
  2. Visual Studio 2015

Passos

  • Faça o download e instalação personalizada do Visual Studio 2015, certificando-se de que Ferramentas de Desenvolvimento de Aplicativos Universais do Windows esteja selecionada junto com suas subopções:-
    a) Tools and Windows SDK
    b) Emulator for Windows Phone
  • Certifique-se de Ativar modo de desenvolvedor no dispositivo de desenvolvimento e implantação.
  • Selecione o modelo com base no idioma que deseja usar:
    C#, Visual Basic, C++ or JavaScript.
  • Em seguida, crie um aplicativo em branco (Windows universal).
  • Selecione a versão de destino e mínima do Windows 10 adequada para seu aplicativo.

Clique aqui se não tiver certeza de quais versões você deve escolher ou simplesmente deixe as opções em seus valores padrão e clique em ‘OK’ para começar!
