Por ID #

document.getElementById('uniqueID')

vai recuperar

<div id="uniqueID"></div>

Desde que exista um elemento com o ID fornecido, document.getElementById retornará apenas esse elemento. Caso contrário, retornará null.

Observação: os IDs devem ser exclusivos. Vários elementos não podem ter o mesmo ID.

Por Nome da Tag #

document.getElementsByTagName('b')

vai recuperar

<b>All</b>
<b>of</b>
<b>the b elements.</b>

Se não existir nenhum elemento com o nome de tag fornecido, uma coleção vazia será retornada.

Por seletor de CSS #

Considere seguir o código html

<ul>
  <li id=“one” class=“main”>Item 1</li>
  <li id=“two” class=“main”>Item 2</li>
  <li id=“three” class=“main”>Item 3</li>
  <li id=“four”>Item 4</li>
</ul>

A seguinte árvore dom será construída com base no código html acima

                      ul
    
                      |
    
     |          |         |         |
    
    li         li        li        li
     |          |         |         |
Item 1        Item 2    Item 3    Item 4

Podemos selecionar elementos da árvore DOM com a ajuda de seletores CSS. Isso é possível por meio de dois métodos javascript: querySelector() e querySelectorAll().

querySelector() retorna o primeiro elemento que corresponde ao seletor css fornecido do DOM.

document.querySelector('li.main')

retorna o primeiro elemento li cuja classe é main

document.querySelector('#two')

retorna o elemento com id two

NOTA: Se nenhum elemento for encontrado, null será retornado. Se a string do seletor contiver um pseudo-elemento CSS, o retorno será null.

querySelectorAll() retorna todos os elementos que correspondem ao seletor css fornecido do DOM.

document.querySelectorAll('li.main')

retorna uma lista de nós contendo todos os elementos li cuja classe é main.

NOTA: Se nenhum elemento for encontrado, uma lista de nós vazia será retornada. Se a string de seletores contiver um pseudo-elemento CSS, a elementList retornada estará vazia

Seletores de consulta #

Nos navegadores modernos [1], é possível usar um seletor tipo CSS para consultar elementos em um documento – da mesma forma que sizzle.js (usado por jQuery ).

querySeletor #

Retorna o primeiro Element no documento que corresponde à consulta. Se não houver correspondência, retorna null.

// gets the element whose id="some-id"
var el1 = document.querySelector('#some-id');
 
// gets the first element in the document containing "class-name" in attribute class
var el2 = document.querySelector('.class-name');
 
// gets the first anchor element in the document
var el2 = document.querySelector('a');
 
// gets the first anchor element inside a section element in the document
var el2 = document.querySelector('section a');

querySelectorAll #

Retorna uma NodeList contendo todos os elementos do documento que correspondem à consulta. Se nenhum corresponder, retorna um NodeList vazio.

// gets all elements in the document containing "class-name" in attribute class
var el2 = document.querySelectorAll('.class-name');
 
// gets all anchor elements in the document
var el2 = document.querySelectorAll('a');

// gets all anchor elements inside any section element in the document
var el2 = document.querySelectorAll('section a');

Por nome de classe #

document.getElementsByClassName('class-name')

vai recuperar

<a class="class-name">Any</a>
<b class="class-name">tag</b>
<div class="class-name an-extra-class">with that class.</div>

Se nenhum elemento existente contiver a classe fornecida, uma coleção vazia será retornada.

Exemplo:

<p class="my-class">I will be matched</p>
<p class="my-class another-class">So will I</p>
<p class="something-else">I won't</p>

var myClassElements = document.getElementByClassName('my-class');
console.log(myClassElements.length); // 2
var nonExistentClassElements = document.getElementByClassName('nope');
console.log(nonExistentClassElements.length); // 0
