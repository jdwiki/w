Exemplo de mínimo absoluto #

Qualquer extensão do Chrome começa como uma extensão descompactada: uma pasta contendo os arquivos da extensão.

Um arquivo que deve conter é manifest.json, que descreve as propriedades básicas da extensão. Muitas das propriedades nesse arquivo são opcionais, mas aqui está um arquivo manifest.json mínimo absoluto:

{
  "manifest_version": 2,
  "name": "My Extension",
  "version": "1.0"
}

Crie uma pasta (por exemplo, myExtension) em algum lugar, adicione manifest.json conforme listado acima.

Em seguida, você precisa carregar a extensão no Chrome.

  1. Abra a página chrome://extensions/, acessível através de Menu > Mais ferramentas > Extensões.
  2. Ative o Modo de desenvolvedor com uma caixa de seleção no canto superior direito, caso ainda não esteja ativado.
  3. Clique no botão Load unpacked extension… e selecione a pasta myExtension criada.

É isso! Sua primeira extensão é carregada pelo Chrome:

Claro, ele ainda não faz nada, então é um bom momento para ler uma visão geral da arquitetura de extensão para começar a adicionar as partes que você precisa.

Importante: ao fazer alterações em sua extensão, não se esqueça de retornar a chrome://extensions/ e pressionar o link Recarregar para sua extensão depois de fazer as alterações. No caso de scripts de conteúdo, recarregue também a página de destino.

Crie uma nova guia #

No código de extensão, você pode usar qualquer API chrome.* se tiver decalizado as permissões necessárias. Além disso, algumas APIs funcionam apenas a partir de páginas de fundo e algumas APIs funcionam apenas a partir de scripts de conteúdo.

Você pode usar a maioria dos métodos chrome.tabs declarando quaisquer permissões. Agora nos concentramos em chrome.tabs.create

Observação: a nova guia será aberta sem nenhum aviso popup.

chrome.tabs.create({
       url:"http://stackoverflow.com",
       selected:false  // We open the tab in the background
})

Você pode aprender mais sobre o objeto guia, no desenvolvedor oficial do Chrome
