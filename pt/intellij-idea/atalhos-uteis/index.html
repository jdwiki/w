OS X / macOS: Cmd + <número da janela da ferramenta> #

Por exemplo, mudando o foco para a janela do projeto #

Windows: Alt + 1 #

OS X / macOS: Cmd + 1 #

pop-up de arquivos recentes #

OS X / macOS: Cmd + E #

Windows: Ctrl + Shift + A #

OS X / macOS: Cmd + Shift + A #

Navegar para #

  • Arquivo : #

    • Windows: Ctrl + Shift + N #

    • OS X / macOS: Cmd + Shift + N #

  • Classe : #

    • Windows: Ctrl + N #

    • OS X / macOS: Cmd + N #

  • Símbolo (classe/método/variável/nome da constante): #

    • Windows: Ctrl + Alt + Shift + N #

    • OS X / macOS: Cmd + Option + Shift + N #

    Note that you can use class name to narrow down the method/variable/constant search, for example to find symbol usersCollection in class UserDAO type:

      UserDAO.usersCollection

  • Em toda parte : #

    • Windows: Shift + Shift #

    • OS X / macOS: Shift + Shift #

Para pesquisar algo que tenha várias palavras, por exemplo, InetAddressCachePolicy você pode simplesmente digitar InAddCacPo ou algo semelhante que contenha partes de palavras em todo o nome.

Seleção #

Seleção com escopo crescente

Isso é útil quando você deseja selecionar um bloco para extrair uma variável / método etc., não há necessidade de fazer uma correspondência precisa de colchetes, basta colocar o acento circunflexo em algum lugar na instrução e continuar fazendo isso

Windows: Ctrl + W #

OS X / macOS: Cmd + W #

Seleção com escopo decrescente

Windows: Ctrl + Shift + W #

OS X / macOS: Cmd + Shift + W #

Esse recurso também é muito útil ao editar/jogar com documentos json em seu IDE.

Seleção vertical

pressione e segure

Windows: Alt #

OS X / macOS: Opt #

e selecione normalmente usando o mouse / trackpad (a maneira como você seleciona uma palavra em uma linha etc)

É assim que deve ser

Vários sinais de intercalação

pressione e segure

Windows: Alt + Shift #

OS X / macOS: Opt + Shift #

e clique onde tudo que você deseja colocar um acento circunflexo. Você pode optar por colocar vários acentos circunflexos em uma única linha ou entre linhas em diferentes posições.

Agora você pode realizar todas as operações que você poderia realizar em uma única palavra selecionada (mantenha pressionada Ctrl (windows) ou option (mac OS) e use Teclas esquerda ou Direita para pular palavras) e todas elas afetarão todas as posições do cursor.

Você pode até cortar/colar várias seleções de um lugar para outro.

Ter vários acentos circunflexos é muito útil quando você deseja alterar a estrutura do texto em muitas linhas / muitas posições na mesma linha.

Selecionando ocorrências duplicadas

Selecione algum texto e pressione

Windows: Alt + J #

OS X / macOS: ctrl + G #

para selecionar a próxima ocorrência do mesmo texto.

Você obtém um acento circunflexo em cada ocorrência selecionada que pode ser usada para alterar cada ocorrência simultaneamente.
