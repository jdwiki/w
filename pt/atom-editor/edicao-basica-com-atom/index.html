Árvore de arquivos interativa #

Para acompanhar a estrutura de arquivos de seus projetos, o Atom, como muitos editores de texto e IDEs, usa um modelo de árvore de arquivos. Essas árvores mostram os locais e os nomes de seus arquivos e diretórios. Para alternar a árvore entre visível e oculta, as teclas Ctrl+\ podem ser usadas (⌘+\ para Mac OS). Essa árvore também inclui muitas operações para arquivos e diretórios, conforme mostrado abaixo:

Os arquivos ocultos (a menos que definido de outra forma nas configurações do Atom) serão exibidos com nomes de arquivos sombreados. Um exemplo comum são os dados de configuração do repositório do GitHub no diretório .git.

Abrindo arquivos e diretórios #

Junto com outros editores de texto mais avançados, o Atom permite que os desenvolvedores abram um único arquivo ou diretório.

Abrindo Arquivos #

Para abrir arquivos com o Atom, use File > Open File… no menu conforme mostrado abaixo:

ou use o atalho de teclado mais rápido: Ctrl+O (Para Mac OS: ⌘+O). Isso abrirá um explorador de arquivos (Finder para Mac OS) a partir do qual você pode selecionar um arquivo para abrir, ou para selecionar vários arquivos, use o Ctrl (Para Mac ⌘) enquanto clica em outros arquivos ou segure a tecla Shift enquanto seleciona outros arquivos para selecionar um intervalo. Quando você tiver selecionado os arquivos que deseja abrir, pressione o botão ‘Abrir’ no explorador de arquivos. O Atom, como editor de texto, opta apenas por manipular arquivos com menos de 2 megabytes.

Localizar e substituir #

O recurso localizar e substituir no Atom funciona de duas maneiras, uma opera localmente apenas no arquivo em que você está e a outra em um conjunto de arquivos ou diretórios.

Para abrir a ferramenta localizar e substituir para um único arquivo, pressione Ctrl+F (Para Mac OS use ⌘+F kbd>). Digite no primeiro espaço a string que você está procurando. Pressione a tecla Enter para encontrar todas as instâncias da string. À direita do botão Localizar estão os botões de opção regex, diferencia maiúsculas de minúsculas, na seleção e palavra inteira. O botão Usar Regex permite pesquisar caracteres regex como \n, \t, \r e instruções regex /^[a-z0-9_-]{3,16} $/. O botão Case Sensitive - quando ativo - só encontrará strings com o mesmo caso (maiúsculas). A opção Somente na seleção só encontrará instâncias da string nas seções destacadas do arquivo. A opção Whole Word só encontrará instâncias delimitadas, não quando a string fizer parte de uma porção maior. Clicar no botão Substituir pegará a primeira instância encontrada com o método Localizar e a substituirá pelo conteúdo do campo de substituição (mesmo que esteja vazio). Clicar no botão Substituir tudo substituirá todas as instâncias encontradas pelo método Localizar e todas de uma vez pelo conteúdo do campo de substituição.
