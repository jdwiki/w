O que é átomo? #

Atom é um editor de texto hackeável criado pelo GitHub e desenvolvido em cima da plataforma de aplicativos de desktop Electron.

Isso significa que ele pode ser usado como um editor de texto para programação básica até um IDE de tamanho completo. Também é extremamente personalizável, fornece milhares de pacotes feitos pela comunidade (destaque de sintaxe, UX aprimorado etc.) e temas para atender às necessidades de todos. Também está disponível para Windows, MacOS e Linux.

  • Abrindo diretórios
  • Várias guias de edição
  • Painéis de edição lado a lado
  • Vários cursores de edição
  • Comutação de linha
  • Gerenciamento de árvore de arquivos e diretórios

Executando um “Olá, Mundo!” programa em Python usando Atom do zero #

O Atom é um editor de texto versátil e flexível e possui centenas de pacotes de código aberto criados pela comunidade que podem compilar e executar arquivos de origem, para vários idiomas. Este guia mostrará como é fácil codificar Python com o editor Atom.

Este guia pressupõe que você não tenha o Python nem o Atom instalados em seu sistema.

Etapa 1: Instalando o Python #

O Python pode ser instalado a partir do [site oficial] (https://www.python.org/), ou se você estiver usando Linux, por meio de gerenciadores de pacotes (no entanto, o Python geralmente vem pré-instalado de qualquer maneira).

Se você é um usuário do Windows, não se esqueça de definir python.exe para o seu %PATH%.

Etapa 2: Instalando o Atom #

Você pode instalar o editor Atom do site oficial ou por meio de gerenciadores de pacotes.

Etapa 3: Configurando o Atom #

Para obter mais informações sobre como instalar pacotes e temas, leia este tópico dedicado.

Para compilar e executar programas, a comunidade Atom fornece pacotes para atender a essa necessidade. Para este exemplo, usaremos script para executar nosso programa.

Vá para Arquivo > Configurações > Instalar.

Digite script na barra de pesquisa e instale-o. Quando instalado, deve ser listado em “Pacotes” na barra de configurações. Deve-se notar que script não é capaz de entrada do usuário.

Se você estiver usando MacOS ou Linux, você pode usar o gerenciador de pacotes apm para instalar pacotes.

Passo 4: Programando e executando #

Escolha um diretório onde você gostaria de armazenar seu arquivo de origem PY.

  1. Certifique-se de que consegue ver o painel Tree View; se você não puder ver este painel, você pode alterná-lo indo em Exibir > Alternar exibição em árvore.

  2. Vá para File > Add Project Folder e selecione um diretório que será definido como seu diretório raiz para um projeto.

  3. Clique com o botão direito do mouse na pasta e clique em Novo arquivo, digite hello-world.py e digite o seguinte código:

    print("Hello, World!")

  4. Pressione CTRL+SHIFT+B para executar o script. Alternativamente, você pode ir para View > Toggle Command Palette e digitar Script: Run.

O script deve retornar:

   Hello, World!
   [Finished in 0.125s]
