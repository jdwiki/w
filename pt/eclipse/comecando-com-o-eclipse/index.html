Instalação e configuração #

Para instalar o Eclipse, acesse a página da Web Eclipse Downloads onde geralmente há um link direto para fazer download da versão mais recente do Eclipse. Iniciando o Eclipse Mars (versão 4.5), pode-se fazer download de um instalador que o orienta através do procedimento de instalação, em vez de fazer download de toda a instalação como um arquivo compactado (esta opção ainda está disponível, no entanto). Também há links para download de pacotes antigos do Eclipse.

O Eclipse vem em vários pacotes diferentes que visam usuários diferentes, conforme mostrado na captura de tela abaixo do instalador. Por exemplo, o Eclipse IDE for Java Developers contém ferramentas básicas que dão suporte ao desenvolvimento, depuração e criação de aplicativos Java, bem como suporte básico para controle de versão, como um plug-in que permite o controle de versão de projetos usando o Git, enquanto o Eclipse for Android Developers fornece um ambiente para a criação de aplicativos Android.

Depois que um pacote é selecionado, a próxima página permite que o usuário selecione o diretório de instalação, juntamente com outras opções. A captura de tela a seguir ilustra o procedimento em uma máquina Windows.

Clicar no botão Instalar iniciará a instalação do pacote Eclipse nesse diretório.

Se a máquina já tiver o Java instalado corretamente, o Eclipse deve iniciar normalmente e normalmente nenhuma configuração ou configuração é necessária. No entanto, é uma boa prática alterar algumas opções de configuração do Eclipse, por exemplo, para especificar em qual JVM o Eclipse deve ser executado e configurar a memória mínima e máxima para essa JVM. Para isso, existe um arquivo chamado eclipse.ini no diretório de instalação, onde esta configuração de inicialização está localizada. Esta página contém detalhes sobre como configurar o Eclipse usando esse arquivo.

Importando projetos existentes #

No menu Arquivo, escolha a opção ‘Importar…’.

Isso abre a caixa de diálogo Importar, que solicita o tipo de projeto/arquivo que você deseja importar. Para um projeto Java básico, escolha ‘Existing Projects into Workspace’ na pasta ‘General’.

Em seguida, selecione o diretório onde o(s) projeto(s) está(ão) localizado(s) usando o botão ‘Procurar’. Todos os projetos que podem ser importados para o Eclipse aparecerão na seção ‘Projetos:’. Se o projeto já foi importado, ele ainda será exibido, mas a caixa de seleção será desativada.

Você também pode importar projetos diretamente de um arquivo compactado escolhendo ‘Selecionar arquivo’ e clicando no botão ‘Procurar’.

Depois de clicar em ‘Concluir’, o projeto agora está visível no seu Project Explorer e pronto para uso.

Instale o Marketplace no Eclipse #

Poucas das versões clássicas do eclipse não vêm pré-instaladas com o marketplace, isso pode ser instalado usando as seguintes etapas:

  1. Vá para Ajuda → Instalar novo software
  2. Adicionar novo Repositório (site especificado abaixo)
  3. Ferramentas de uso geral → Cliente do Marketplace
  4. Clique em Concluir e pronto.

Sites de atualização do mercado:

Oxygen - http://download.eclipse.org/releases/oxygen/
Neon - http://download.eclipse.org/releases/neon/
Mars - http://download.eclipse.org/releases/mars/
Luna - http://download.eclipse.org/mpc/luna
Helios - http://download.eclipse.org/releases/helios
Juno - http://download.eclipse.org/releases/juno/

Atalhos de teclado úteis #

Gerenciar arquivos e projetos #

  • Ctrl+Shift+R: Abrir recurso (arquivo, pasta ou projeto)
  • Ctrl+Shift+S: Salva todos os arquivos
  • Ctrl+W: Fecha o arquivo atual
  • Ctrl+Shift+W: Fecha todos os arquivos

Janela do Editor #

  • F12: Ir para a janela do editor
  • Ctrl+E : Mostra a lista de Editores abertos. Use as setas e enter para alternar
  • Ctrl+Page Down/Up : Alterna para o próximo editor / alterna para o editor anterior
  • Ctrl+M : Maximiza ou minimiza a janela do editor atual

Navegue no Editor #

  • Ctrl+L : Ir para a linha
  • Ctrl+Q: Ir para o último local editado
  • Ctrl+Shift+P : Com um colchete selecionado: pule para o colchete de fechamento ou abertura correspondente
  • Ctrl+Seta para baixo/para cima: Editor de rolagem sem alterar a posição do cursor

Editar texto #

  • Ctrl+D: Excluir linha
  • Alt+Shift+Y: quebra o texto selecionado (ajusta a largura do texto à tela)
  • Alt+Shift+S: opções do menu Open Source
  • Alt+Shift+R: refatorar a palavra destacada em todos os arquivos
  • Ctrl+Alt+Up/Ctrl+Alt+Down : Copia as linhas selecionadas para cima/baixo.

Pesquisar e substituir #

  • Ctrl+J : Digite um termo para pesquisar e use Ctrl+J / Ctrl+shift+J para subir/voltar
  • Ctrl+K/Ctrl+Shift+K : Ctrl+H e feche a janela de busca. Em seguida, encontre a ocorrência anterior / seguinte do termo de pesquisa.

Move um bloco de código #

  • Shift+tab : Move para a esquerda
  • Alt+Up/Alt+Down : Move para cima/baixo.

Criando e executando um programa Java HelloWorld #

Na barra de ferramentas, abra a Perspectiva Java.

Crie um novo projeto Java #

Clique com o botão direito do mouse no Package Explorer, e no menu selecione New -> Java Project

Na próxima caixa de diálogo, insira um nome de projeto e clique em Finish.

Agora você tem o novo projeto em seu espaço de trabalho.

Crie uma nova classe Java #

Clique com o botão direito do mouse em seu projeto e, no menu, selecione New -> Class.

Na caixa de diálogo digite o nome da classe (deve começar com letra maiúscula), marque também a caixa de seleção public static void main(String[] args), em seguida, clique em Concluir.

Agora você tem o primeiro arquivo Java em seu projeto. O editor abrirá automaticamente este novo arquivo.

Dentro do método main digite algum código para imprimir Hello world!.

Execute sua classe Java #

Clique com o botão direito do mouse em sua classe Java e, no menu, selecione Executar como -> Aplicação Java.

Voila, você vê a saída do seu programa Java no Console.
