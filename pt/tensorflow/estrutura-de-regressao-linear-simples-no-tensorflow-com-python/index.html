Um modelo amplamente utilizado em estatística tradicional é o modelo de regressão linear. Neste artigo, o objetivo é acompanhar passo a passo a implementação deste tipo de modelos. Vamos representar uma estrutura de regressão linear simples.

Para nosso estudo, analisaremos a idade das crianças no eixo x e a altura das crianças no eixo y. Tentaremos prever a altura das crianças, usando sua idade, aplicando regressão linear simples.[no TF encontrando os melhores W e b]

trem_X matriz np com dimensão x de informações
trem_Y np array com y dimensão de informação

Estrutura de código de função de regressão simples #

Definição da função:

def run_training(train_X, train_Y):

Variáveis ​​de entrada:

X = tf.placeholder(tf.float32, [m, n])
Y = tf.placeholder(tf.float32, [m, 1])

Representação de peso e viés

W = tf.Variable(tf.zeros([n, 1], dtype=np.float32), name="weight")
b = tf.Variable(tf.zeros([1], dtype=np.float32), name="bias")

Modelo linear:

with tf.name_scope("linear_Wx_b") as scope:
    activation = tf.add(tf.matmul(X, W), b)

Custo:

with tf.name_scope("cost") as scope:
    cost = tf.reduce_sum(tf.square(activation - Y)) / (2 * m)
    tf.summary.scalar("cost", cost)

Treinamento:

with tf.name_scope("train") as scope:
    optimizer = tf.train.GradientDescentOptimizer(0.07).minimize(cost)

Sessão do TensorFlow:

with tf.Session() as sess:
    merged = tf.summary.merge_all()
    writer = tf.summary.FileWriter(log_file, sess.graph)

Observação: mesclado e gravador fazem parte da estratégia do TensorBoard para rastrear o comportamento do modelo.

    init = tf.global_variables_initializer()
    sess.run(init)

Repetindo 1,5k vezes o loop de treinamento:

    for step in range(1500):
       result, _ = sess.run([merged, optimizer], feed_dict={X: np.asarray(train_X), Y: np.asarray(train_Y)})
       writer.add_summary(result, step)

Custo do treinamento de impressão:

    training_cost = sess.run(cost, feed_dict={X: np.asarray(train_X), Y: np.asarray(train_Y)})
    print "Training Cost: ", training_cost, "W=", sess.run(W), "b=", sess.run(b), '\n'

Previsão concreta com base no modelo treinado:

    print "Prediction for 3.5 years"

Rotina Principal #

def main():
    train_X, train_Y = read_data()
    train_X = feature_normalize(train_X)
    run_training(train_X, train_Y)

Nota: lembre-se das dependências das funções de revisão. read_data, feature_normalize e run_training

Rotina de Normalização #

def feature_normalize(train_X):
    global mean, std
    mean = np.mean(train_X, axis=0)
    std = np.std(train_X, axis=0)

    return np.nan_to_num((train_X - mean) / std)
