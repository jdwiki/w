Exemplo básico #

Atualização: o TensorFlow agora oferece suporte à convolução 1D desde a versão r0.11, usando [tf.nn.conv1d](https://www.tensorflow.org/versions/r0.11/api_docs/python/nn. html#conv1d).

Considere um exemplo básico com uma entrada de comprimento 10 e dimensão 16. O tamanho do lote é 32. Portanto, temos um espaço reservado com forma de entrada [batch_size, 10, 16].

batch_size = 32
x = tf.placeholder(tf.float32, [batch_size, 10, 16])

Em seguida, criamos um filtro com largura 3 e tomamos canais 16 como entrada e saída também canais 16.

filter = tf.zeros([3, 16, 16])  # these should be real values, not 0

Finalmente aplicamos tf.nn.conv1d com um stride e um padding:

  • passo: inteiro s
  • preenchimento: funciona como em 2D, você pode escolher entre SAME e VALID. SAME produzirá o mesmo comprimento de entrada, enquanto VALID não adicionará preenchimento zero.

Para o nosso exemplo, damos um passo de 2 e um preenchimento válido.

output = tf.nn.conv1d(x, filter, stride=2, padding="VALID")

A forma de saída deve ser [batch_size, 4, 16]. Com padding="SAME", teríamos uma forma de saída de [batch_size, 5, 16].

Para versões anteriores do TensorFlow, você pode usar apenas convoluções 2D enquanto define a altura das entradas e dos filtros como 1.

Matemática por trás da convolução 1D com exemplos avançados em TF #

`Para calcular a convolução 1D manualmente, você desliza seu kernel sobre a entrada, calcula as multiplicações elementares e as soma.

A maneira mais fácil é padding=0, stride=1 #

Então, se sua input = [1, 0, 2, 3, 0, 1, 1] e kernel = [2, 1, 3] o resultado da convolução é `[8, 11, 7, 9, 4]», que é calculado da seguinte forma:

  • 8 = 1 * 2 + 0 * 1 + 2 * 3
  • 11 = 0 * 2 + 2 * 1 + 3 * 3
  • 7 = 2 * 2 + 3 * 1 + 0 * 3
  • 9 = 3 * 2 + 0 * 1 + 1 * 3
  • 4 = 0 * 2 + 1 * 1 + 1 * 3

A função conv1d do TF calcula convoluções em lotes, portanto, para fazer isso no TF, precisamos fornecer os dados no formato correto (o documento explica que a entrada deve estar em [batch, in_width, in_channels], também explica como o kernel deve se parecer). Então

import tensorflow as tf
i = tf.constant([1, 0, 2, 3, 0, 1, 1], dtype=tf.float32, name='i')
k = tf.constant([2, 1, 3], dtype=tf.float32, name='k')

print i, '\n', k, '\n'

data   = tf.reshape(i, [1, int(i.shape[0]), 1], name='data')
kernel = tf.reshape(k, [int(k.shape[0]), 1, 1], name='kernel')

print data, '\n', kernel, '\n'

res = tf.squeeze(tf.nn.conv1d(data, kernel, 1, 'VALID'))
with tf.Session() as sess:
    print sess.run(res)

que lhe dará a mesma resposta que calculamos anteriormente: [ 8. 11. 7. 9. 4.]

Convolução com preenchimento #

O preenchimento é apenas uma maneira elegante de dizer anexar e preceder sua entrada com algum valor. Na maioria dos casos, esse valor é 0, e é por isso que na maioria das vezes as pessoas o chamam de preenchimento de zero. O TF suporta ‘VALID’ e ‘SAME’ zero-padding, para um preenchimento arbitrário você precisa usar tf.pad(). O preenchimento ‘VÁLIDO’ significa nenhum preenchimento, onde o mesmo significa que a saída terá o mesmo tamanho da entrada. Vamos calcular a convolução com padding=1 no mesmo exemplo (observe que para nosso kernel este é ‘SAME’ padding). Para fazer isso, apenas anexamos nosso array com 1 zero no início/fim: input = [0, 1, 0, 2, 3, 0, 1, 1, 0].

Aqui você pode notar que você não precisa recalcular tudo: todos os elementos permanecem os mesmos, exceto o primeiro/último que são:

  • 1 = 0 * 2 + 1 * 1 + 0 * 3
  • 3 = 1 * 2 + 1 * 1 + 0 * 3

Portanto, o resultado é [1, 8, 11, 7, 9, 4, 3] que é o mesmo calculado com TF:

res = tf.squeeze(tf.nn.conv1d(data, kernel, 1, 'SAME'))
with tf.Session() as sess:
    print sess.run(res)

Convolução com passos largos #

As passadas permitem que você pule elementos enquanto desliza. Em todos os nossos exemplos anteriores deslizamos 1 elemento, agora você pode deslizar elementos s de cada vez. Como usaremos um exemplo anterior, há um truque: deslizar por n elementos é equivalente a deslizar por 1 elemento e selecionar cada n-ésimo elemento.

Então, se usarmos nosso exemplo anterior com padding=1 e mudarmos stride para 2, você apenas pega o resultado anterior [1, 8, 11, 7, 9, 4, 3] e deixa cada 2º elemento, que resultará em [1, 11, 9, 3]. Você pode fazer isso no TF da seguinte maneira:

res = tf.squeeze(tf.nn.conv1d(data, kernel, 2, 'SAME'))
with tf.Session() as sess:
    print sess.run(res)
