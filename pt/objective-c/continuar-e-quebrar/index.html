Continuar e quebrar a declaração #

A instrução continue na linguagem de programação Objective-C funciona um pouco como a instrução break. Em vez de forçar o término, no entanto, continue força a próxima iteração do loop a ocorrer, pulando qualquer código no meio.

Para o loop for, a instrução continue faz com que as partes de teste condicional e de incremento do loop sejam executadas. Para os loops while e do…while, a instrução continue faz com que o controle do programa passe para os testes condicionais.

#import <Foundation/Foundation.h>
 
int main ()
{
   /* local variable definition */
   int a = 10;

   /* do loop execution */
   do
   {
      if( a == 15)
      {
         /* skip the iteration */
         a = a + 1;
         continue;
      }
      NSLog(@"value of a: %d\n", a);
      a++;
     
   }while( a < 20 );
 
   return 0;
}
