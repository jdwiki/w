A classe NSString faz parte do framework Foundation para trabalhar com strings (séries de caracteres). Também inclui métodos para comparar, pesquisar e modificar strings.

Para aninhar vários tipos de objetos e tipos de dados em NSStrings, consulte: Objective-C, Format Specifiers

Codificação e Decodificação #

// decode
NSString *string = [[NSString alloc] initWithData:utf8Data
                                         encoding:NSUTF8StringEncoding];

// encode
NSData *utf8Data = [string dataUsingEncoding:NSUTF8StringEncoding];

Algumas codificações suportadas são:

  • NSASCIIStringEncoding
  • NSUTF8StringEncoding
  • NSUTF16StringEncoding (== NSUnicodeStringEncoding)

Observe que utf8Data.bytes não inclui um caractere nulo de terminação, que é necessário para strings C. Se você precisar de uma string C, use UTF8String:

const char *cString = [string UTF8String];
printf("%s", cString);

Comprimento da string #

NSString tem uma propriedade length para obter o número de caracteres.

NSString *string = @"example";
NSUInteger length = string.length;       // length equals 7

Como no Exemplo de divisão, lembre-se de que NSString usa UTF-16 para representar caracteres. O comprimento é, na verdade, apenas o número de unidades de código UTF-16. Isso pode diferir do que o usuário percebe como caracteres.

Aqui estão alguns casos que podem ser surpreendentes:

@"é".length == 1   // LATIN SMALL LETTER E WITH ACUTE (U+00E9)
@"é".length == 2   // LATIN SMALL LETTER E (U+0065) + COMBINING ACUTE ACCENT (U+0301)
@"❤️".length == 2  // HEAVY BLACK HEART (U+2764) + VARIATION SELECTOR-16 (U+FE0F)
@"🇮🇹".length == 4  // REGIONAL INDICATOR SYMBOL LETTER I (U+1F1EE) + REGIONAL INDICATOR SYMBOL LETTER T (U+1F1F9)

Para obter o número de caracteres percebidos pelo usuário, conhecidos tecnicamente como “grapheme clusters”, você deve iterar sobre a string com [-enumerateSubstringsInRange:options:usingBlock:][eSIR⭕uB] e manter uma contagem. Isso é demonstrado em [uma resposta de Nikolai Ruhe no Stack Overflow][NRanswer].

[eSIR⭕uB]:https://developer.apple.com/library/ios/documentation/Cocoa/Reference/Foundation/Classes/NSString_Class/#//apple_ref/occ/instm/NSString/enumerateSubstringsInRange:options:usingBlock : [NRanswer]:http://stackoverflow.com/a/33539320/603977

Comparando Strings #

Strings são comparadas para igualdade usando isEqualToString:

O operador == apenas testa a identidade do objeto e não compara os valores lógicos dos objetos, portanto, não pode ser usado:

NSString *stringOne = @"example";
NSString *stringTwo = [stringOne mutableCopy];

BOOL objectsAreIdentical = (stringOne == stringTwo);          // NO
BOOL stringsAreEqual = [stringOne isEqualToString:stringTwo]; // YES

A expressão (stringOne == stringTwo) testa se os endereços de memória das duas strings são os mesmos, o que geralmente não é o que queremos.

Se as variáveis ​​de string podem ser nil, você também deve tomar cuidado com este caso:

BOOL equalValues = stringOne == stringTwo || [stringOne isEqualToString:stringTwo];

Esta condição retorna YES quando as strings têm valores iguais ou ambos são nil.

Para ordenar duas strings em ordem alfabética, use compare:.

NSComparisonResult result = [firstString compare:secondString];

NSComparisonResult pode ser:

  • NSOrderedAscending: A primeira string vem antes da segunda string.
  • NSOrderedSame: As strings são iguais.
  • NSOrderedDescending: A segunda string vem antes da primeira string.

Para comparar a igualdade de duas strings, use isEqualToString:.

BOOL result = [firstString isEqualToString:secondString];

Para comparar com a string vazia (@""), é melhor usar length.

BOOL result = string.length == 0;

Divisão #

Você pode dividir uma string em uma matriz de partes, divididas por um caractere separador.

NSString * yourString = @"Stack,Exchange,Network";
NSArray * yourWords = [yourString componentsSeparatedByString:@","]; 
// Output: @[@"Stack", @"Exchange", @"Network"]

Se você precisar dividir em um conjunto de vários delimitadores diferentes, use -[NSString componentsSeparatedByCharactersInSet:].

NSString * yourString = @"Stack Overflow+Documentation/Objective-C";
NSArray * yourWords = [yourString componentsSeparatedByCharactersInSet:
                      [NSCharacterSet characterSetWithCharactersInString:@"+/"]];
// Output: @[@"Stack Overflow", @"Documentation", @"Objective-C"]`

Se você precisar quebrar uma string em seus caracteres individuais, faça um loop sobre o comprimento da string e converter cada caractere em uma nova string.

NSMutableArray * characters = [[NSMutableArray alloc] initWithCapacity:[yourString length]];
for (int i = 0; i < [myString length]; i++) {
    [characters addObject: [NSString stringWithFormat:@"%C", 
                                      [yourString characterAtIndex:i]];
}

Como no [Exemplo de Comprimento][Comprimento], lembre-se de que um “caractere” aqui é uma unidade de código UTF-16, não necessariamente o que o usuário vê como um caractere. Se você usar esse loop com @"🇮🇹", verá que ele está dividido em quatro partes.

Para obter uma lista dos caracteres percebidos pelo usuário, use [-enumerateSubstringsInRange:options:usingBlock:][eSIR⭕uB].

NSMutableArray * characters = [NSMutableArray array];
[yourString enumerateSubstringsInRange:(NSRange){0, [yourString length]}
                               options:NSStringEnumerationByComposedCharacterSequences
                            usingBlock:^(NSString * substring, NSRange r, NSRange s, BOOL * b){
                                [characters addObject:substring];
                            }];

Isso preserva grupos de grafemas como a bandeira italiana como uma única substring.

Criação #

Simples:

NSString *newString = @"My String";

De várias strings:

NSString *stringOne = @"Hello";
NSString *stringTwo = @"world";
NSString *newString = [NSString stringWithFormat:@"My message: %@ %@",
                     stringOne, stringTwo];

Usando String Mutável

NSString *stringOne = @"Hello";
NSString *stringTwo = @"World";
NSMutableString *mutableString = [NSMutableString new];
[mutableString appendString:stringOne];
[mutableString appendString:stringTwo];

De NSData:

Ao inicializar a partir de NSData, uma codificação explícita deve ser fornecida, pois NSString não é capaz de adivinhar como os caracteres são representados no fluxo de dados brutos. A codificação mais comum hoje em dia é UTF-8, que é até um requisito para certos dados como JSON.

Evite usar +[NSString stringWithUTF8String:] pois espera uma string C explicitamente terminada em NULL, que -[NSData bytes] não fornece.

NSString *newString = [[NSString alloc] initWithData:myData encoding:NSUTF8StringEncoding];

Do NSArray:

NSArray *myArray = [NSArray arrayWithObjects:@"Apple", @"Banana", @"Strawberry", @"Kiwi", nil];
NSString *newString = [myArray componentsJoinedByString:@" "];

Procurando por uma Substring #

Para pesquisar se uma String contém uma substring, faça o seguinte:

NSString *myString = @"This is for checking substrings";
NSString *subString = @"checking"; 

BOOL doesContainSubstring = [myString containsString:subString];  // YES

Se segmentar iOS 7 ou OS X 10.9 (ou anterior):

BOOL doesContainSubstring = ([myString rangeOfString:subString].location != NSNotFound);  // YES

Caso de mudança #

Para converter uma String para maiúscula, use uppercaseString:

NSString *myString = @"Emphasize this";
NSLog(@"%@", [myString uppercaseString]; // @"EMPHASIZE THIS"

Para converter uma String para minúscula, use lowercaseString:

NSString *myString = @"NORMALIZE this";
NSLog(@"%@", [myString lowercaseString]; // @"normalize this"

Para capitalizar a primeira letra de cada palavra em uma string, use capitalizedString:

NSString *myString = @"firstname lastname";
NSLog(@"%@", [myString capitalizedString]); // @"Firstname Lastname"

Unindo um array de strings #

Para combinar um NSArray de NSString em um novo NSString:

NSArray *yourWords = @[@"Objective-C", @"is", @"just", @"awesome"];
NSString *sentence = [yourWords componentsJoinedByString:@" "];

// Sentence is now: @"Objective-C is just awesome"

Removendo espaços em branco à esquerda e à direita #

NSString *someString = @"   Objective-C Language  \n";
NSString *trimmedString = [someString stringByTrimmingCharactersInSet:[NSCharacterSet whitespaceAndNewlineCharacterSet]];
//Output will be - "Objective-C Language"

O método stringByTrimmingCharactersInSet retorna uma nova string feita pela remoção de ambas as extremidades dos caracteres String contidos em um determinado conjunto de caracteres.

Também podemos remover apenas espaços em branco ou nova linha

// Removing only WhiteSpace
NSString *trimmedWhiteSpace = [someString stringByTrimmingCharactersInSet:[NSCharacterSet whitespaceCharacterSet]];
//Output will be - "Objective-C Language  \n"

// Removing only NewLine
NSString *trimmedNewLine = [someString stringByTrimmingCharactersInSet:[NSCharacterSet newlineCharacterSet]];
//Output will be - "   Objective-C Language  "

Formatação #

A formatação NSString suporta todas as strings de formato disponíveis na função printf ANSI-C. A única adição feita pela linguagem é o símbolo %@ usado para formatar todos os objetos Objective-C.

É possível formatar inteiros

int myAge = 21;
NSString *formattedAge = [NSString stringWithFormat:@"I am %d years old", my_age];

Ou qualquer objeto subclassificado de NSObject

NSDate *now = [NSDate date];
NSString *formattedDate = [NSString stringWithFormat:@"The time right now is: %@", now];

Para obter uma lista completa de especificadores de formato, consulte: Objective-C, Format Specifiers, Syntax

Trabalhando com Strings C #

Para converter NSString para const char use [-[NSString UTF8String]](http://developer.apple.com/mac/library/documentation/Cocoa/Reference/Foundation/Classes/NSString_Class/Reference/NSString .html#//apple_ref/occ/instm/NSString/UTF8String):

NSString *myNSString = @"Some string";
const char *cString = [myNSString UTF8String];

Você também pode usar [-[NSString cStringUsingEncoding:]](http://developer.apple.com/mac/library/documentation/Cocoa/Reference/Foundation/Classes/NSString_Class/Reference/NSString.html#//apple_ref /occ/instm/NSString/cStringUsingEncoding:) se sua string estiver codificada com algo diferente de UTF-8.

Para o caminho reverso, use [-[NSString stringWithUTF8String:]](https://developer.apple.com/library/mac/documentation/Cocoa/Reference/Foundation/Classes/NSString_Class/index.html#//apple_ref/ occ/clm/NSString/stringWithUTF8String:):

const *char cString = "Some string";
NSString *myNSString = [NSString stringWithUTF8String:cString];
myNSString = @(cString); // Equivalent to the above.

Depois de ter o const char *, você pode trabalhar com ele de forma semelhante a um array de chars:

printf("%c\n", cString[5]);

Se você quiser modificar a string, faça uma cópia:

char *cpy = calloc(strlen(cString)+1, 1);
strncpy(cpy, cString, strlen(cString));
// Do stuff with cpy
free(cpy);
