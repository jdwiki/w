Aqui vamos discutir sobre o histórico do Node.js, informações de versão e seu status atual.

Principais eventos em cada ano #

#2009#

  • 3 de março: O projeto foi nomeado como “nó”
  • 1º de outubro: Primeira prévia do npm, o pacote Node gerente
  • 8 de novembro: Ryan Dahl’s (Criador do Node.js) Original Palestra Node.js no JSConf 2009

#2010#

  • Express: Uma estrutura de desenvolvimento web Node.js
  • Versão inicial do Socket.io
  • 28 de abril: Suporte experimental Node.js no Heroku
  • 28 de julho: Google Tech Talk de Ryan Dahl no Node.js
  • 20 de agosto: Node.js 0.2.0 lançado

#2011#

  • 31 de março: Guia Node.js
  • 1º de maio: npm 1.0: lançado
  • 1º de maio: AMA de Ryan Dahl no Reddit
  • 10 de julho: O Node Beginner Book, uma introdução ao Node.js, é concluído.
    • A comprehensive Node.js tutorial for beginners.
  • 16 de agosto: LinkedIn usa Node.js
    • LinkedIn launched its completely overhauled mobile app with new features and new parts under the hood.
  • 5 de outubro: Ryan Dahl fala sobre a história do Node.js e porque ele o criou
  • 5 de dezembro: Node.js em produção na Uber
    • Uber Engineering Manager Curtis Chambers explains why his company completely re-engineered their application using Node.js to increase eficiência e melhorar a experiência do parceiro e do cliente.

#2012#

  • 30 de janeiro: o criador do Node.js, Ryan Dahl, se afasta do dia-a-dia do Node
  • 25 de junho: Node.js v0.8.0 [stable] está disponível
  • 20 de dezembro: Hapi, um framework Node.js é lançado

#2013#

  • 30 de abril: The MEAN Stack: MongoDB, ExpressJS, AngularJS e Node.js
  • 17 de maio: Como construímos o primeiro aplicativo Node.js do eBay
  • 15 de novembro: PayPal lança Kraken, um framework Node.js
  • 22 de novembro: Node.js Memory Leak no Walmart
    • Eran Hammer of Wal-Mart labs came to the Node.js core team complaining of a memory leak he had been tracking down for months.
  • 19 de dezembro: Koa - Web framework para Node.js

#2014#

  • 15 de janeiro: TJ Fontaine assume o projeto Node
  • 23 de outubro: Conselho Consultivo do Node.js
    • Joyent and several members of the Node.js community announced a proposal for a Node.js Advisory Board as a next step towards a fully open modelo de governança para o projeto de código aberto Node.js.
  • 19 de novembro: Node.js in Flame Graphs - Netflix
  • 28 de novembro: IO.js – E/S com eventos para Javascript V8

#2015#

Q1 #

  • 14 de janeiro: IO.js 1.0.0
  • 10 de fevereiro: Joyent se move para estabelecer a fundação Node.js
    • Joyent, IBM, Microsoft, PayPal, Fidelity, SAP and The Linux Foundation Join Forces to Support Node.js Community With Neutral and Open Governance
  • 27 de fevereiro: proposta de reconciliação de IO.js e Node.js

Q2 #

  • 14 de abril: módulos privados npm
  • 28 de maio: [o líder do nó TJ Fontaine está deixando o Joyent](http://venturebeat.com/2015/05/08/node-lead-tj-fontaine-is-stepping-down-and-leaving-joyent -também/)
  • 13 de maio: Node.js e io.js estão se fundindo sob a Node Foundation

Q3 #

  • 2 de agosto: Trace - monitoramento e depuração de desempenho do Node.js
    • Trace is a visualized microservice monitoring tool that gets you all the metrics you need when operating microservices.
  • 13 de agosto: 4.0 é o novo 1.0

Q4 #

  • 12 de outubro: Node v4.2.0, primeira versão de suporte de longo prazo
  • 8 de dezembro: Apigee, RisingStack e Yahoo se unem à Node.js Foundation
  • 8 e 9 de dezembro: Node Interactive
    • The first annual Node.js conference by the Node.js Foundation

#2016#

Q1 #

  • 10 de fevereiro: Express se torna um projeto incubado
  • 23 de março: O incidente do leftpad
  • 29 de março: Google Cloud Platform se junta à Node.js Foundation

Q2 #

  • 26 de abril: npm tem 210.000 usuários

Q3 #

  • 18 de julho: CJ Silverio torna-se o CTO da npm
  • 1º de agosto: Trace, a solução de depuração Node.js torna-se geralmente disponível
  • 15 de setembro: O primeiro Node Interactive na Europa

Q4 #

  • 11 de outubro: O gerenciador de pacotes yarn foi lançado
  • 18 de outubro: Node.js 6 se torna a versão LTS
