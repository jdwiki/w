Coloque o pin do usuário no mapa. #

Observação, se você não estiver familiarizado com a API do Google Maps, poderá ler o exemplo anterior (básico) para entender este pequeno exemplo.

  • Primeiro, inicialize o mapa.

Você pode adicionar um elemento de mapa em seu código HTML e um pouco de CSS como este:

<!DOCTYPE html>
<html>
<head>
<style>
  /* Always set the map height explicitly to define the size of the div
   * element that contains the map. */
  #map {
    height: 100%;
  }
  /* Optional: Makes the sample page fill the window. */
  html, body {
    height: 100%;
    margin: 0;
    padding: 0;
  }
</style>
</head>
<body>
<div id="map"></div>
</body>
</html>

Agora, você precisa adicionar a biblioteca do Google Maps ao seu código com um script balise como este:

<script src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap"
async defer></script>

Você pode substituir YOUR_API_KEY no código por uma chave de API do Google. O é um link para obter uma chave. #

Em seguida, você deve escrever em seu código uma função que servirá como retorno de chamada (ou função de inicialização) para seu mapa. Aqui, apenas adicionamos uma pequena função que você pode encontrar no google aqui:

function initMap() {
    map = new google.maps.Map(document.getElementById('map'), {
      center: {lat: -34.397, lng: 150.644},
      zoom: 8
    });
  }

Agora você normalmente tem um mapa básico na tela. Você pode encontrar o código completo em google. #

  • Segundo, encontre a posição do usuário.

Para solicitar a posição do usuário, existe uma função muito simples que é fornecida pelo navegador:

navigator.geolocation.getCurrentPosition(showPosition);

Observe que esta função aceita um parâmetro. É uma função a ser chamada se a geolocalização for bem-sucedida.

Temos que criar esta função. :)

function showPosition(position) {
    alert (position);    
}

Esta função é muito simples e teremos que atualizá-la depois para traçar um marcador na posição do usuário.

A função de geolocalização que usamos aqui é muito simples. Você pode ter a documentação completa em w3schools. #

Em seu ponto, o código se parece com isso:

<!DOCTYPE html>
<html>

<head>
  <title>Simple Map</title>
  <meta name="viewport" content="initial-scale=1.0">
  <meta charset="utf-8">
  <style>
    /* Always set the map height explicitly to define the size of the div
      * element that contains the map. */
    
    #map {
      height: 100%;
    }
    /* Optional: Makes the sample page fill the window. */
    
    html,
    body {
      height: 100%;
      margin: 0;
      padding: 0;
    }
  </style>
</head>

<body>
  <div id="map"></div>
  <script>
    var map;
    navigator.geolocation.getCurrentPosition(showPosition);
    function initMap() {
      map = new google.maps.Map(document.getElementById('map'), {
        center: { lat: -34.397, lng: 150.644 },
        zoom: 8
      });
    }
    function showPosition(position) {
      console.log(position);
    }
  </script>
  <script src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap" async defer></script>
</body>

</html>
  • E terceiro, exiba a posição do usuário no mapa com um marcador.

Para exibir um marcador no mapa, você pode usar a função no exemplo ‘básico’:

// Create a marker and place it on the map
var marker = new google.maps.Marker({
    position: position,
    map: map
});

Não vou detalhar essas linhas de códigos com muita precisão. Você só pode agora que ao criar um marcador com este código: new google.maps.Marker({});, você passa as ‘opções do marcador’ entre os abraços. Você pode consultar a documentação do google aqui.

Observe também que você pode especificar a posição do marcador com muita facilidade com o parâmetro position.

Agora temos que modificar a função showPosition.

Você pode acessar simplesmente a lat e lng da variável position assim:

  var markerPosition={};
  markerPosition.lat=position.coords.latitude;
  markerPosition.lng=position.coords.longitude;

Assim, o google entende como simplesmente acessar o valor lat e lng.

Agora adicionamos para modificar a função showPosition para adicionar um marcador na posição do usuário.

function showPosition(position) {
    var markerPosition={};
    markerPosition.lat=position.coords.latitude;
    markerPosition.lng=position.coords.longitude;
    // Create a marker and place it on the map
    var marker = new google.maps.Marker({
        position: markerPosition,
        map: map
    });
}
