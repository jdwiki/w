Execute algum código Perl a partir da linha de comando #

One-liners simples podem ser especificados como argumentos de linha de comando para perl usando a opção -e (pense em “executar”):

perl -e'print "Hello, World!\n"'

Devido às regras de cotação do Windows, você não pode usar strings com aspas simples, mas precisa usar uma destas variantes:

perl -e"print qq(Hello, World!\n)"
perl -e"print \"Hello, World!\n\""

Note que para evitar quebrar o código antigo, somente a sintaxe disponível até Perl 5.8.x pode ser usada com -e. Para usar qualquer coisa mais recente que sua versão perl possa suportar, use -E em vez disso. Por exemplo. para usar say disponível de 5.10.0 em mais Unicode 6.0 de >=v5.14.0 (também usa -CO para garantir que STDOUT imprima UTF-8):

perl -CO -E'say "\N{PILE OF POO}"'

Usando strings com aspas duplas em one-liners do Windows #

O Windows usa apenas aspas duplas para agrupar parâmetros de linha de comando. Para usar aspas duplas em perl de uma linha (ou seja, para imprimir uma string com uma variável interpolada), você deve escapá-los com barras invertidas:

perl -e "my $greeting = 'Hello'; print \"$greeting, world!\n\""

Para melhorar a legibilidade, você pode usar um operador qq():

perl -e "my $greeting = 'Hello'; print qq($greeting, world!\n)"

Imprime linhas que correspondem a um padrão (PCRE grep) #

perl -ne'print if /foo/' file.txt

Não diferencia maiúsculas de minúsculas:

perl -ne'print if /foo/i' file.txt

Substitua uma substring por outra (PCRE sed) #

perl -pe"s/foo/bar/g" file.txt

Ou no local:

perl -i -pe's/foo/bar/g' file.txt

No Windows:

perl -i.bak -pe"s/foo/bar/g" file.txt

Imprime apenas alguns campos #

perl -lane'print "$F[0] $F[-1]"' data.txt
# prints the first and the last fields of a space delimited record

Exemplo CSV:

perl -F, -lane'print "$F[0] $F[-1]"' data.csv

Imprima as linhas 5 a 10 #

perl -ne'print if 5..10' file.txt

Editar arquivo no local #

Sem uma cópia de backup (não compatível com Windows)

perl -i -pe's/foo/bar/g' file.txt

Com uma cópia de backup file.txt.bak

perl -i.bak -pe's/foo/bar/g' file.txt

Com uma cópia de backup old_file.txt.orig no subdiretório backup (desde que exista):

perl -i'backup/old_*.orig' -pe's/foo/bar/g' file.txt

Lendo o arquivo inteiro como uma string #

perl -0777 -ne'print "The whole file as a string: --->$_<---\n"'

Nota: O -0777 é apenas uma convenção. Qualquer -0400 e acima seria o mesmo.

Carregar arquivo no mojolicious #

perl -Mojo -E 'p("http://localhost:3000" => form => {Input_Type => "XML", Input_File => {file => "d:/xml/test.xml"}})'

O arquivo d:/xml/test.xml será carregado para o servidor que escuta conexões em localhost:3000 (Fonte)

Neste exemplo:

-Mmodule executa use module; antes de executar seu programa -E commandline é usado para inserir uma linha de programa Se você não tem o módulo ojo você pode usar o comando cpanm ojo para instalá-lo
