O que é interpolado #

Perl interpola nomes de variáveis:

my $name = 'Paul';
print "Hello, $name!\n"; # Hello, Paul!

my @char = ('a', 'b', 'c');
print "$char[1]\n"; # b

my %map = (a => 125, b => 1080, c => 11);
print "$map{a}\n"; # 125

Arrays podem ser interpolados como um todo, seus elementos são separados por espaços:

my @char = ('a', 'b', 'c');
print "My chars are @char\n"; # My chars are a b c

Perl não interpola hashes como um todo:

my %map = (a => 125, b => 1080, c => 11);
print "My map is %map\n"; # My map is %map

e chamadas de função (incluindo constantes):

use constant {
    PI => '3.1415926'
};
print "I like PI\n";         # I like PI
print "I like " . PI . "\n"; # I like 3.1415926

Perl interpola sequências de escape começando com \:

\t                  horizontal tab
\n                  newline
\r                  return
\f                  form feed
\b                  backspace
\a                  alarm (bell)
\e                  escape

A interpolação de \n depende do sistema onde o programa está funcionando: ele produzirá um caractere(s) de nova linha de acordo com as convenções atuais do sistema.

Perl não interpola \v, o que significa tabulação vertical em C e outras linguagens.

O personagem pode ser endereçado usando seus códigos:

\x{1d11e}     𝄞 by hexadecimal code
\o{350436}    𝄞 by octal code
\N{U+1d11e}   𝄞 by Unicode code point

ou nomes Unicode:

\N{MUSICAL SYMBOL G CLEF}

Caracteres com códigos de 0x00 a 0xFF na codificação nativa podem ser endereçados de forma mais curta:

\x0a     hexadecimal
\012     octal

O caractere de controle pode ser endereçado usando sequências de escape especiais:

\c@      chr(0)
\ca      chr(1)
\cb      chr(2)
...
\cz      chr(26)
\c[      chr(27)
\c\      chr(28) # Cannot be used at the end of a string
                 # since backslash will interpolate the terminating quote
\c]      chr(29)
\c^      chr(30)
\c_      chr(31)
\c?      chr(127)

Letras maiúsculas têm o mesmo significado: "\cA" == "\ca".

A interpretação de todas as seqüências de escape, exceto para \N{...}, pode depender da plataforma, pois eles usam códigos dependentes da plataforma e da codificação.

Interpolação básica #

Interpolação significa que o interpretador Perl substituirá os valores das variáveis ​​por seus nomes e alguns símbolos (que são impossíveis ou difíceis de digitar diretamente) por sequências especiais de caracteres (também conhecido como escape). A distinção mais importante é entre aspas simples e duplas: aspas duplas interpolam a string incluída, mas aspas simples não.

my $name = 'Paul';
my $age = 64;
print "My name is $name.\nI am $age.\n"; # My name is Paul.
                                         # I am 64.

Mas:

print 'My name is $name.\nI am $age.\n'; # My name is $name.\nI am $age.\n

Você pode usar q{} (com qualquer delimitador) em vez de aspas simples e qq{} em vez de aspas duplas. Por exemplo, q{I'm 64} permite usar um apóstrofo dentro de uma string não interpolada (caso contrário, terminaria a string).

Declarações:

print qq{$name said: "I'm $age".}; # Paul said: "I'm 64".
print "$name said: \"I'm $age\"."  # Paul said: "I'm 64".

faça a mesma coisa, mas no primeiro você não precisa escapar aspas duplas dentro da string.

Se o nome da sua variável colidir com o texto ao redor, você pode usar a sintaxe ${var} para desambiguar:

my $decade = 80;
print "I like ${decade}s music!"  # I like 80s music!
