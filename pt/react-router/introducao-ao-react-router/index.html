Hello World com React e React Router #

Depois de instalar o react e o react-router, é hora de usar os dois juntos.

A sintaxe é muito simples, você especifica a url e o component que deseja renderizar quando essa url for aberta

<Route path="hello" component={ HelloComponent } />

Isso significa que quando o caminho da url for hello, renderizará o componente HelloComponent

FILENAME: app.js

'use strict';

import React from 'react';
import { render } from 'react-dom';
import { Router, browserHistory, Link } from 'react-router';

// These are just demo components which render different text.

let DashboardPage = () => (
  <div>
    <h1>Welcome User</h1>
    <p>This is your dashboard and I am an example of a stateless functional component.</p>
    <Link to="/settings">Goto Settings Page</Link>
  </div>
)

let SettingsPage = () => (
  <div>
    <h1>Manage your settings</h1>
    <p>display the settings form fields here...or whatever you want</p>
    <Link to="/">Back to Dashboard Page</Link>
  </div>
)

let AuthLoginPage = () => (
  <div>
    <h1>Login Now</h1>
    <div>
      <form action="">
        <input type="text" name="email" placeholder="email address" />
        <input type="password" name="password" placeholder="password" />
        <button type="submit">Login</button>
      </form>
    </div>
  </div>
)

let AuthLogoutPage = () => (
  <div>
    <h1>You have been successfully logged out.</h1>
    <div style={{ marginTop: 30 }}>
      <Link to="/auth/login">Back to login page</Link>
    </div>
  </div>
)

let ArticlePage = ({ params }) => (
  <h3>Article {params.id}</h3>
)

let PageNotFound = () => (
  <div>
    <h1>The page you're looking for doesn't exist.</h1>
  </div>
)

// Here we pass Router to the render function.
render( (
    <Router history={ browserHistory }>

      <Route path="/" component={ DashboardPage } />
      <Route path="settings" component={ SettingsPage } />

      <Route path="auth">
        <IndexRoute component={ AuthLoginPage } />
        <Route path="login" component={ AuthLoginPage } />
        <Route path="logout" component={ AuthLogoutPage } />
      </Route>
    
      <Route path="articles/:id" component={ ArticlePage } />

      <Route path="*" component={ PageNotFound } />

    </Router>
  ), document.body );

Route Parameters : O caminho do roteador pode ser configurado para receber parâmetros para que possamos ler o valor do parâmetro no componente. O caminho em <Route path="articles/:id" component={ ArticlePage } /> tem um /:id. Esta variável id serve ao propósito do parâmetro path e pode ser acessada no componente ArticlePage usando {props.params.id}.

Se visitarmos http://localhost:3000/#/articles/123 então {props.params.id} no componente ArticlePage será resolvido para 123. Mas visitando o url http://localhost:3000 /#/articles, não funcionará porque não há parâmetro id.

O parâmetro route pode ser tornado opcional escrevendo-o entre um par de parênteses: <Route path="articles(/:id)" component={ ArticlePage } />

Se você quiser usar subrotas, você pode fazer

<Route path="path" component={ PathComponent }>
  <Route path="subpath" component={ SubPathComponent } />
</Route>
  • quando /path for acessado, PathComponent será renderizado
  • quando /path/subpath for acessado, PathComponent será renderizado e SubPathComponent será passado para ele como props.children

Você pode usar path="*" para pegar todas as rotas que não existem e renderizar a página 404 page not found.

Começando #

Este iniciando pressupõe que você esteja trabalhando com [create-react-app][1], ou algo equivalente usando Babel e todos os recursos disponíveis.

Confira também a excelente documentação [bem aqui][2].

Primeiro, instale react-router-dom:

npm install react-router-dom ou yarn add react-router-dom.

Em seguida, crie um componente que exista de uma Navbar básica com dois itens e páginas básicas:

import React from 'react'
import { BrowserRouter, Route, Link } from 'react-router-dom'

const Home = () => (
  <div>
    <p>We are now on the HOME page</p>
  </div>
)

const About = () => (
  <div>
    <p>We are now on the ABOUT page</p>
  </div>
)

const App = () => (
  <BrowserRouter>
    <div>
      <ul>
        <li><Link to="/">Home</Link></li>
        <li><Link to="/about">About</Link></li>
      </ul>
      <hr/>
      <Route path="/" component={Home}/>
      <Route path="/about" component={About}/>
    </div>
  </BrowserRouter>
)
export default App

Vamos passo a passo através deste código:

  • import React from 'react': Certifique-se de importar React
  • import { BrowserRouter as Router, Route, Link } de 'react-router-dom' dividido:
  • BrowserRouter é o próprio roteador. Certifique-se de envolver seu componente dentro do componente BrowserRouter.
  • Route é uma rota particular que pode ser navegada para
  • Link é um componente que produz uma tag <a href="...">, que você pode usar como um hiperlink.
  • const Home é uma função que retorna a página inicial.
  • const About é uma função que retorna a página About.

  • const App é o componente principal:

  • <BrowserRouter> é o componente [JSX][3] que envolve os componentes nos quais você deseja usar o componente <Route>.

  • ‘

    é um elemento único para envolver todo JSX dentro doBrowserRouter`.

  • <ul> é a barra de navegação. Ele contém um link para Home e um link para About.

  • links <li><Link to="/">Home</Link></li> para a página inicial. Você pode ver que, como o link se refere a “/”, um caminho relativo vazio renderiza a página inicial.

  • links <li><Link to="/about">Sobre</Link></li> para a página Sobre.

  • <Route path="/" component={Home}/> descreve qual componente deve ser renderizado se o caminho relativo for "/".

  • <Route path="/about" component={About}/> descreve qual componente deve ser renderizado se o caminho relativo for "/about".

Muito o que aprender aqui, mas espero que isso explique os fundamentos, para que você possa continuar seus aprendizados.

Instalação e configuração #

Para instalar o React Router, basta executar o comando npm

npm install --save react-router

E você está feito. Isso é literalmente tudo o que você precisa fazer para instalar o roteador react.

Observe: react-router depende de react, então certifique-se de instalar react também.

Para configurar:

usando um transpilador ES6, como babel

import { Router, Route, Link } from 'react-router'

não usando um transpilador ES6

var Router = require('react-router').Router
var Route = require('react-router').Route
var Link = require('react-router').Link

Instalação usando a compilação UMD #

Uma compilação também está disponível em npmcdn. Você pode incluir o script assim:

<script src="https://npmcdn.com/react-router/umd/ReactRouter.min.js"></script>

A biblioteca estará disponível globalmente em window.ReactRouter.
