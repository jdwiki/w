Sintaxe #

  • numpy.cross(a, b) # produto cruzado de a e b (ou vetores em a e b)
  • numpy.cross(a, b, axisa=-1) #produto cruzado de vetores em a com b, s.t. vetores em a são dispostos ao longo do eixo axisa
  • numpy.cross(a, b, axisa=-1, axisb=-1, axisc=-1) # produtos cruzados de vetores em a e b, vetores de saída dispostos ao longo do eixo especificado por *axisc *
  • numpy.cross(a, b, axis=None) # produtos cruzados de vetores em a e b, vetores em a, b e na saída disposta ao longo do eixo axis

Parâmetros #

Coluna Coluna
a,b No uso mais simples, a e b são dois vetores de 2 ou 3 elementos. Eles também podem ser matrizes de vetores (ou seja, matrizes bidimensionais). Se a é um array e ‘b’ é um vetor, cross(a,b) retorna um array cujos elementos são os produtos cruzados de cada vetor em a com o vetor b. O b é um array e a é um único vetor, cross(a,b) retorna um array cujos elementos são os produtos cruzados de a com cada vetor em b. a e b podem ser arrays se tiverem a mesma forma. Neste caso, cross(a,b) retorna cross(a[0],b[0]), cross(a[1], b[1]), ...
axisa/b Se a for uma matriz, ela pode ter vetores dispostos no eixo de variação mais rápida, no eixo de variação mais lenta ou algo entre eles. axisa diz cross() como os vetores são dispostos em a. Por padrão, ele assume o valor do eixo de variação mais lenta. axisb funciona da mesma forma com a entrada b. Se a saída de cross() for um array, os vetores de saída podem ser dispostos em diferentes eixos de array; axisc diz cross como dispor os vetores em seu array de saída. Por padrão, axisc indica o eixo de variação mais lenta.
eixo Um parâmetro de conveniência que define axisa, axisb e axisc todos com o mesmo valor, se desejado. Se axis e qualquer um dos outros parâmetros estiverem presentes na chamada, o valor de axis substituirá os outros valores.

Produto cruzado de dois vetores #

O Numpy fornece uma função cross para calcular produtos cruzados vetoriais. O produto vetorial dos vetores [1, 0, 0] e [0, 1, 0] é [0, 0, 1]. Numpy nos diz:

>>> a = np.array([1, 0, 0])
>>> b = np.array([0, 1, 0])
>>> np.cross(a, b)
array([0, 0, 1])

como esperado.

Enquanto os produtos cruzados são normalmente definidos apenas para vetores tridimensionais. No entanto, qualquer um dos argumentos para a função Numpy pode ser dois vetores de elemento. Se o vetor c for fornecido como [c1, c2], o Numpy atribui zero à terceira dimensão: [c1, c2, 0]. Então,

>>> c = np.array([0, 2])
>>> np.cross(a, c)
array([0, 0, 2])

Ao contrário de dot que existe como uma função Numpy e um método de ndarray, cross existe apenas como uma função autônoma:

>>> a.cross(b)
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AttributeError: 'numpy.ndarray' object has no attribute 'cross'

Vários produtos cruzados com uma chamada #

Qualquer entrada pode ser uma matriz de vetores de 3 (ou 2) elementos.

>>> a=np.array([[1,0,0],[0,1,0],[0,0,1]])
>>> b=np.array([1,0,0])
>>> np.cross(a,b)
array([[ 0,  0,  0],
       [ 0,  0, -1],
       [ 0,  1,  0]])

O resultado neste caso é array([np.cross(a[0],b), np.cross(a1,b), np.cross(a2,b)])

b também pode ser uma matriz de vetores de 3 (ou 2-) elementos, mas deve ter a mesma forma que a. Caso contrário, o cálculo falha com um erro de “incompatibilidade de forma”. Então podemos ter

>>> b=np.array([[0,0,1],[1,0,0],[0,1,0]])
>>> np.cross(a,b)
array([[ 0, -1,  0],
       [ 0,  0, -1],
       [-1,  0,  0]])

e agora o resultado é array([np.cross(a[0],b[0]), np.cross(a[1],b[1]), np.cross(a[2],b[ 2])])

Mais flexibilidade com vários produtos cruzados #

Em nossos dois últimos exemplos, numpy assumiu que a[0,:] era o primeiro vetor, a[1,:] o segundo e a[2,:] o terceiro. Numpy.cross tem um argumento opcional axisa que nos permite especificar qual eixo define os vetores. Então,

>>> a=np.array([[1,1,1],[0,1,0],[1,0,-1]])
>>> b=np.array([0,0,1])
>>> np.cross(a,b)
array([[ 1, -1,  0],
       [ 1,  0,  0],
       [ 0, -1,  0]])
>>> np.cross(a,b,axisa=0)
array([[ 0, -1,  0],
       [ 1, -1,  0],
       [ 0, -1,  0]])
>>> np.cross(a,b,axisa=1)
array([[ 1, -1,  0],
       [ 1,  0,  0],
       [ 0, -1,  0]])

O resultado axisa=1 e o resultado padrão são ambos (np.cross([1,1,1],b), np.cross([0,1,0],b), np.cross([ 1,0,-1],b))». Por padrão, axisasempre indica o último eixo (variando mais lentamente) do array. O resultadoaxisa=0é(np.cross([1,0,1],b), np.cross([1,1,0],b), np.cross([1,0,- 1], b))».

Um parâmetro opcional semelhante, axisb, executa a mesma função para a entrada b, se também for um array bidimensional.

Os parâmetros axisa e axisb informam ao numpy como distribuir os dados de entrada. Um terceiro parâmetro, axisc, informa ao numpy como distribuir a saída se a ou b for multidimensional. Usando as mesmas entradas a e b acima, obtemos

>>> np.cross(a,b,1)
array([[ 1, -1,  0],
       [ 1,  0,  0],
       [ 0, -1,  0]])
>>> np.cross(a,b,1,axisc=0)
array([[ 1,  1,  0],
       [-1,  0, -1],
       [ 0,  0,  0]])
>>> np.cross(a,b,1,axisc=1)
array([[ 1, -1,  0],
       [ 1,  0,  0],
       [ 0, -1,  0]])

Portanto, axisc=1 e o axisc padrão dão o mesmo resultado, ou seja, os elementos de cada vetor são contíguos no índice de movimento mais rápido do array de saída. axisc é por padrão o último eixo da matriz. axisc=0 distribui os elementos de cada vetor pela dimensão de variação mais lenta da matriz.

Se você quiser que axisa, axisb e axisc tenham todos o mesmo valor, você não precisa definir todos os três parâmetros. Você pode definir um quarto parâmetro, axis, para o valor único necessário e os outros três parâmetros serão definidos automaticamente. axis substitui axisa, axisb ou axisc se algum deles estiver presente na chamada da função.
