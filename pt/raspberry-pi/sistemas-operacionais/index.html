Esta seção contém informações sobre vários sistemas operacionais disponíveis para o Raspberry Pi.

Raspbian #

O sistema operacional mais popular para o Raspberry Pi é um Raspbian baseado no Debian.

É oficialmente suportado pela Fundação Raspberry Pi.

O Raspbian pode ser baixado do site oficial do Raspberry Pi em uma das duas variantes:

  • Com ambiente de trabalho
  • Lite - imagem mínima

A partir de setembro de 2016, o Raspbian é fornecido com o ambiente de desktop PIXEL (“Pi Improved Xwindows Environment, Lightweight”)

Sistemas operacionais baseados em Yocto #

É possível criar uma distribuição Linux própria usando o Yocto Project.

Para Raspberry Pi- existe uma camada específica meta-raspberrypi que precisa ser usada para criar uma imagem.

Windows 10 IoT Core #

O Windows 10 IoT Core está disponível apenas para Raspberry Pi 2 e 3. É importante observar que esta não é uma versão completa do Windows, como se poderia esperar, ela foi projetada especificamente para aplicativos IoT (Internet das Coisas). Ele não executará a maioria dos aplicativos padrão do Windows e não possui um menu Iniciar. Ele inicializa em uma página com alguns exemplos de código e vídeos e possui uma interface de linha de comando.

OSMC #

OSMC é um media player gratuito e de código aberto baseado em Linux.

FreeELEC #

Distribuição Linux com recursos necessários para executar Kodi.

Kali #

Um sistema operacional completo projetado em torno de testes de penetração.

Link para Download:

https://www.offensive-security.com/kali-linux-arm-images/

Requisitos:

  1. Cartão SD Classe 10 Mínimo de 8 GB
  2. Cartão SD Classe 10 de 16 GB recomendado

Login padrão:

Username: root
Password: toor

Notas

  1. Instale o gparted e expanda a partição para o tamanho completo
  2. Execute apt-get install kali-linux-full para obter todas as ferramentas padrão disponíveis
  3. Executa o Ambiente de Trabalho XFCE
