Usando processamento com texto sublime #

Para adicionar um idioma ao Sublime Text, você usa Package Control. Você pode fazer isso colando o código Python correto (disponível no site Package Control no link acima) ou baixando o arquivo .sublime-package (também disponível para download no site). Depois de configurar o Package Control, reinicie o Sublime Text.

Para instalar o pacote de processamento do Package Control, siga as etapas a seguir:

  • Executar Ferramentas | Instale o processing-java para instalar a ferramenta processing-java. Esta ferramenta é necessária para construir o esboço do Processing na linha de comando e editores como Sublime Text e Atom.
  • Abra a Paleta de Comandos (Ctrl+Shift+P ou Cmd+Shift+P)
  • Procure por ‘Package Control: Install Package’
  • Procure por ‘Processing’ e instale o pacote
  • Reinicie o Sublime Text

Depois de seguir essas etapas, você poderá selecionar Processamento como idioma. Isso facilitará a codificação em Processing in Sublime Text.

Usando o Processing com o editor Atom #

Existem vários pacotes que podem executar esboços de processamento no editor Atom. Estas instruções usam o pacote Script. Há também pacotes disponíveis para sintaxe realce e autocomplete, que são necessários para Script para identificar os tipos de arquivos de processamento.

Instale o plug-in de script. Ou executando o apm install script na linha de comando ou procure o pacote ‘script’ por rgbkrk na aba Install nas configurações do Atom (atalho command + , ou ctrl + ,).

Assim que o Script estiver instalado, você precisará instalar o processing-java. Esta ferramenta vem com o software principal do Processing e é necessária para construir esboços do Processing na linha de comando e nos editores:

  • MacOS: Execute Ferramentas > Instalar "processing-java".

  • Linux: Adicione o diretório Processing à sua variável de ambiente PATH (substitua /path/to/processing pelo caminho onde o Processing está instalado):

    sudo ln -s /path/to/processing/processing-java /usr/local/bin/

  • Windows: Adicione o diretório Processing à sua variável de ambiente PATH:

  • Abra as Configurações Avançadas do Sistema executando sysdm.cpl ou pesquisando no Painel de Controle.

  • Clique no botão Variável de ambiente na guia Avançado.

  • Edite a variável PATH para incluir o diretório Processing nas variáveis ​​de usuário (apenas para sua conta) ou variáveis ​​de sistema (para todos os usuários).

Agora, você pode executar esboços de processamento executando Packages > Script > Run Script. O atalho padrão é command + shift + b ou ctrl + shift + b, mas para diminuir ainda mais a dor da transição, você pode vincular o atalho do Processing IDE para executar o esboço. A fim de fazer isso:

  • Abra o arquivo Atom Keymap executando File > Keymap

  • Cole as seguintes linhas no final do arquivo (sinta-se à vontade para alterar a ligação para o que quiser).

    'atom-text-editor':
  'ctrl-r': 'script:run'

Usando Processamento com Eclipse #

Para usar o Processing no Eclipse, comece criando um novo projeto Java. Em seguida, selecione File > Import e escolha General > File System para localizar o arquivo core.jar. Ele pode ser encontrado em PATH_TO_PROCESSING/core/library/ para Windows ou /Applications/Processing 3.app/Contents/Java/core/library/ para Mac. Quando isso estiver concluído, clique com o botão direito do mouse em core.jar e adicione-o ao caminho de construção.

O clichê para usar o Processing no Eclipse é o seguinte:

import processing.core.PApplet;

public class UsingProcessing extends PApplet {

    // The argument passed to main must match the class name
    public static void main(String[] args) {
        PApplet.main("UsingProcessing");
    }

    // method used only for setting the size of the window
    public void settings(){
        
    }

    // identical use to setup in Processing IDE except for size()
    public void setup(){
        
    }

    // identical use to draw in Prcessing IDE
    public void draw(){

    }
}

O método settings() é usado para definir o tamanho da janela. Por exemplo, para criar uma janela de 400 x 400, escreva o seguinte:

 public void settings(){
    size(400,400);   
 }

Todo o resto como descrito na documentação do Hello World em termos do uso de setup() e draw() se aplica aqui.

Como exemplo final, aqui está o código do [Exemplo de desenho de uma linha][2] se ele fosse escrito no Eclipse:

import processing.core.PApplet;

public class UsingProcessing extends PApplet {

    // The argument passed to main must match the class name
    public static void main(String[] args) {
        PApplet.main("UsingProcessing");
    }

    // method for setting the size of the window
    public void settings(){
        size(500, 500);
    }

    // identical use to setup in Processing IDE except for size()
    public void setup(){
        background(0);
        stroke(255);
        strokeWeight(10);
    }

    // identical use to draw in Prcessing IDE
    public void draw(){
        line(0, 0, 500, 500);
    }
}
