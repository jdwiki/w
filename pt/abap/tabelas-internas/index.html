Tipos de tabelas internas #

DATA: <TABLE NAME> TYPE <SORTED|STANDARD|HASHED> TABLE OF <TYPE NAME> 
      WITH <UNIQUE|NON-UNIQUE> KEY <FIELDS FOR KEY>.

Tabela Padrão

Esta tabela possui todas as entradas armazenadas de forma linear e os registros são acessados ​​de forma linear. Para tamanhos de mesa grandes, o acesso à mesa pode ser lento.

Tabela Classificada

Requer a adição WITH UNIQUE|NON-UNIQUE KEY. A pesquisa é rápida devido à execução de uma pesquisa binária. As entradas não podem ser anexadas a esta tabela, pois isso pode quebrar a ordem de classificação, portanto, elas sempre são inseridas usando a palavra-chave INSERT.

Tabela com hash

Requer a adição WITH UNIQUE|NON-UNIQUE KEY. Usa um algoritmo de hash proprietário para manter os pares chave-valor. Teoricamente, as pesquisas podem ser tão lentas quanto uma tabela ‘STANDARD’, mas na prática elas são mais rápidas do que uma tabela ‘SORTED’, levando um tempo constante, independentemente do tamanho da tabela.

Declaração de Tabelas Internas ABAP #

Declaração de tabela interna com base na definição de tipo local #

" Declaration of type
TYPES: BEGIN OF ty_flightb,
         id        TYPE fl_id,
         dat       TYPE fl_date,
         seatno    TYPE fl_seatno,
         firstname TYPE fl_fname, 
         lastname  TYPE fl_lname,
         fl_smoke  TYPE fl_smoker,
         classf    TYPE fl_class,
         classb    TYPE fl_class,
         classe    TYPE fl_class,
         meal      TYPE fl_meal,
         service   TYPE fl_service,
         discout   TYPE fl_discnt,
       END OF lty_flightb.

" Declaration of internal table
DATA t_flightb TYPE STANDARD TABLE OF ty_flightb.

Declaração baseada em Tabela de Banco de Dados #

DATA t_flightb TYPE STANDARD TABLE OF flightb.

Declaração de Tabela Interna Inline #

Requer versão ABAP > 7.4

TYPES t_itab TYPE STANDARD TABLE OF i WITH EMPTY KEY.

DATA(t_inline) = VALUE t_itab( ( 1 ) ( 2 ) ( 3 ) ).

Tabela Interna com Declaração de Linhas de Cabeçalho #

Em ABAP existem tabelas com linhas de cabeçalho e tabelas sem linhas de cabeçalho. Tabelas com linhas de cabeçalho são um conceito antigo e não devem ser usadas em novos desenvolvimentos.

Tabela Interna: Tabela Padrão com/sem linha de cabeçalho

Este código declara a tabela i_compc_all com a estrutura existente de compc_str.

DATA: i_compc_all TYPE STANDARD TABLE OF compc_str WITH HEADER LINE.
DATA: i_compc_all TYPE STANDARD TABLE OF compc_str.

Tabela interna: Tabela com hash com/sem linha de cabeçalho

DATA: i_map_rules_c TYPE HASHED TABLE OF /bic/ansdomm0100 WITH HEADER LINE
DATA: i_map_rules_c TYPE HASHED TABLE OF /bic/ansdomm0100

Declaração de área de trabalho para tabelas sem cabeçalho

Uma área de trabalho (comumente abreviada wa) tem exatamente a mesma estrutura que a tabela, mas pode conter apenas uma linha (uma WA é uma estrutura de uma tabela com apenas uma dimensão).

DATA: i_compc_all_line LIKE LINE OF i_compc_all.

Ler, gravar e inserir em tabelas internas #

Leia, escreva e insira em tabelas internas com uma linha de cabeçalho:

" Read from table with header (using a loop):
LOOP AT i_compc_all.              " Loop over table i_compc_all and assign header line
  CASE i_compc_all-ftype.         " Read cell ftype from header line from table i_compc_all 
    WHEN 'B'.                     " Bill-to customer number transformation
      i_compc_bil = i_compc_all.  " Assign header line of table i_compc_bil with content of header line i_compc_all
      APPEND i_compc_bil.         " Insert header line of table i_compc_bil into table i_compc_bil
    " ... more WHENs
  ENDCASE.
ENDLOOP.

Lembrete: Tabelas internas com linhas de cabeçalho são proibidas em contextos orientados a objetos. O uso de tabelas internas sem linhas de cabeçalho é sempre recomendado.

Leia, escreva e insira em tabelas internas sem uma linha de cabeçalho:

" Loop over table i_compc_all and assign current line to structure i_compc_all_line
LOOP AT i_compc_all INTO i_compc_all_line.      
  CASE i_compc_all_line-ftype.                " Read column ftype from current line (which as assigned into i_compc_all_line)
    WHEN 'B'.                                 " Bill-to customer number transformation
      i_compc_bil_line = i_compc_all_line.    " Copy structure
      APPEND i_compc_bil_line TO i_compc_bil. " Append structure to table
    " more WHENs ...
  ENDCASE.
ENDLOOP.


" Insert into table with Header:
INSERT TABLE i_sap_knb1.                      " insert into TABLE WITH HEADER: insert table header into it's content
insert i_sap_knb1_line into table i_sap_knb1. " insert into HASHED TABLE: insert structure i_sap_knb1_line into hashed table i_sap_knb1
APPEND p_t_errorlog_line to p_t_errorlog.     " insert into STANDARD TABLE: insert structure / wa p_t_errorlog_line into table p_t_errorlog_line
