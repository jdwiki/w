Avaliando expressões regulares com uma função de predicado #

A função de predicado matches pode ser usada para avaliar strings em tempo real sem o uso de nenhuma declaração de objeto.

IF matches( val = 'Not a hex string'
            regex = '[0-9a-f]*' ).
  cl_demo_output=>display( 'This will not display' ).
ELSEIF matches( val = '6c6f7665'
                regex = '[0-9a-f]*' ).
  cl_demo_output=>display( 'This will display' ).
ENDIF.

Substituindo #

A instrução REPLACE pode funcionar diretamente com expressões regulares:

DATA(lv_test) = 'The quick brown fox'.
REPLACE ALL OCCURRENCES OF REGEX '\wo' IN lv_test WITH 'XX'.

A variável lv_test será avaliada como The quick bXXwn XXx.

Expressões regulares orientadas a objetos #

Para operações regex mais avançadas, é melhor usar CL_ABAP_REGEX e suas classes relacionadas.

DATA: lv_test  TYPE string,
      lo_regex TYPE REF TO cl_abap_regex.

lv_test = 'The quick brown fox'.
CREATE OBJECT lo_regex
  EXPORTING
    pattern = 'q(...)\w'.

DATA(lo_matcher) = lo_regex->create_matcher( text = lv_test ).
WRITE: / lo_matcher->find_next( ).      " X
WRITE: / lo_matcher->get_submatch( 1 ). " uic
WRITE: / lo_matcher->get_offset( ).     " 4

Obtendo subcorrespondências usando expressões regulares OO #

Usando o método GET_SUBMATCH da classe CL_ABAP_MATCHER, podemos obter os dados nos grupos/subgrupos.

Objetivo: obter o token à direita da palavra-chave ‘Tipo’.

DATA: lv_pattern TYPE string VALUE 'type\s+(\w+)',
      lv_test TYPE string VALUE 'data lwa type mara'.

CREATE OBJECT ref_regex
  EXPORTING
        pattern     = lv_pattern
        ignore_case = c_true.

ref_regex->create_matcher(
    EXPORTING
        text    = lv_test
    RECEIVING
        matcher = ref_matcher
       ).

ref_matcher->get_submatch(
            EXPORTING
                 index = 0
            RECEIVING
                 submatch = lv_smatch.

A variável resultante lv_smatch contém o valor MARA.

Procurando #

A instrução FIND pode trabalhar diretamente com expressões regulares:

DATA(lv_test) = 'The quick brown fox'.

FIND REGEX '..ck' IN lv_test.
" sy-subrc == 0

FIND REGEX 'a[sdf]g' IN lv_test.
" sy-subrc == 4
