Selecione linhas específicas #

Esta consulta retornará todas as colunas da tabela sales onde os valores na coluna amount forem maiores que 10 e os dados na coluna region em “US”.

SELECT * FROM sales WHERE amount > 10 AND region = "US"

Você pode usar expressões regulares para selecionar as colunas que deseja obter. A instrução a seguir obterá os dados da coluna name e todas as colunas que começam com o prefixo address.

SELECT name, address.* FROM Employees

Você também pode usar a palavra-chave LIKE (combinada com o caractere ‘%’) para combinar strings que começam ou terminam com uma substring específica. A consulta a seguir retornará todas as linhas em que a coluna city começa com “New”

SELECT name, city FROM Employees WHERE city LIKE 'New%'

Você pode usar a palavra-chave RLIKE para usar Java expressões regulares. A consulta a seguir retornará linhas cuja coluna name contém as palavras “smith” ou “son”.

SELECT name, address FROM Employee WHERE name RLIKE '.*(smith|son).*'

Você pode aplicar funções aos dados retornados. A frase a seguir retornará todos os nomes em maiúsculas.

SELECT upper(name) FROM Employees

Você pode usar diferentes funções matemáticas, funções de coleção, funções de conversão de tipo, funções de data, funções condicionais ou funções de string.

Para limitar o número de linhas fornecidas no resultado, você pode usar a palavra-chave LIMIT. A instrução a seguir retornará apenas dez linhas.

SELECT * FROM Employees LIMIT 10

Selecionar todas as linhas #

SELECT é usado para recuperar linhas de dados de uma tabela. Você pode especificar quais colunas serão recuperadas:

SELECT Name, Position
FROM Employees;

Ou apenas use * para obter todas as colunas:

SELECT *
FROM Employees;

Select: Colunas selecionadas do projeto #

Estrutura da tabela de amostra (digamos, Funcionário)

Nome da coluna Tipo de dados
ID INT
F_Name STRING
L_Name STRING
Telefone STRING
Endereço STRING

Projetar todas as colunas

Use o curinga * para projetar todas as colunas. por exemplo.

Select * from Employee

Colunas selecionadas do projeto (digamos, ID, Nome)

Use o nome das colunas na lista de projeção. por exemplo.

Select ID, Name from Employee

Descarte 1 coluna da lista de projeção

Exibe todas as colunas, exceto 1 coluna. por exemplo.

Select `(ID)?+.+` from Employee

Descartar colunas correspondentes ao padrão

Rejeite todas as colunas que correspondem ao padrão. por exemplo. Rejeitar todas as colunas que terminam com NAME

Select `(.*NAME$)?+.+` from Employee
