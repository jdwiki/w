Sessões de depuração simultâneas com PhpStorm #

Uso #

Às vezes, você pode ter que depurar código em outro projeto PhpStorm, você precisa atualizar a configuração.

Configuração do PHP #

Em php.ini, edite o arquivo e coloque xdebug.remote_autostart = 1

Configuração do PhpStorm #

Você também precisa configurar seu IDE:

Na configuração do phpStorm, Max. conexões simultâneas devem ser definidas com um valor maior que um.

Documentação oficial #

Mais informações na documentação oficial

Configuração #

Dê uma olhada no seu arquivo de configuração php.ini e habilite o Xdebug, adicione a seguinte declaração:

[Xdebug]
zend_extension=<full_path_to_xdebug_extension>
xdebug.remote_enable=1
xdebug.remote_host=<the host where PhpStorm is running (e.g. localhost)>
xdebug.remote_port=<the port to which Xdebug tries to connect on the host where PhpStorm is running (default 9000)>

por exemplo com Wamp configurado eu tenho:

; XDEBUG Extension

zend_extension = "d:/wamp/bin/php/php5.5.12/zend_ext/php_xdebug-2.2.5-5.5-vc11.dll"
;
[xdebug]
xdebug.remote_enable = 1
xdebug.profiler_enable = off
xdebug.profiler_enable_trigger = off
xdebug.profiler_output_name = cachegrind.out.%t.%p
xdebug.profiler_output_dir = "d:/wamp/tmp"
xdebug.show_local_vars=0
xdebug.remote_host=localhost
xdebug.remote_port=9000

Reiniciar o Apache

Então em um shell execute este comando para verificar se o Xdebug está rodando corretamente:

php --versão

Depois que seu projeto PHP for criado, se você quiser depurá-lo, será necessário configurar seu intérprete php e Xdebug em configurações:

Ajuste de configuração se php-fpm for usado. #

Se for, a porta Xdebug 9000 entra em conflito com a porta padrão php-fpm.

Você deve escolher uma porta diferente, seja em php-fpm ou Xdebug.

Além disso, quando php-fpm for usado, você TAMBÉM deverá reiniciar o php-fpm para que quaisquer alterações nos arquivos php-ini tenham efeito.

Uso de um arquivo Xdebug.log #

Em alguns casos, você precisará manter em logs o que está acontecendo no seu servidor. Um arquivo de log pode ajudá-lo a resolver seus problemas.

A documentação oficial ajudará você a executar essa funcionalidade

Pode ajudar muito na hora de tentar descobrir o que está quebrado na instalação.
