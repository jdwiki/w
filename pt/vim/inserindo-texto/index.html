Executando comandos normais no modo de inserção #

Enquanto estiver no modo de inserção, pressione <C-o> para sair temporariamente do modo de inserção e execute um comando normal único.

Exemplo #

<C-o>2w salta para a segunda palavra à esquerda e retorna ao modo de inserção.

Observação: Repetir com . só repetirá as ações de retornar ao modo de inserção

Isso permite alguns mapeamentos úteis, por exemplo.

inoremap <C-f> <Right>
inoremap <C-b> <Left>
inoremap <C-a> <C-o>^
inoremap <C-e> <C-o>$

Agora ctrl+a colocará o cursor no início da linha e ctrl+e - no final da linha. Esses mapeamentos são usados ​​por padrão em readline, portanto, podem ser úteis para pessoas que desejam consistência.

Diferentes maneiras de entrar no modo de inserção #

Comando Descrição
a Anexar texto após a posição atual do cursor
A Acrescentar texto no final da linha atual
eu Inserir texto antes da posição atual do cursor
’eu' Inserir texto antes do primeiro caractere não em branco da linha atual
gI Inserir texto na primeira coluna da linha do cursor
gi Insira o texto na mesma posição em que foi deixado da última vez no modo Inserir
‘O’ Abra uma nova linha acima da linha atual e adicione texto lá (CAPITAL O)
o Abra uma nova linha abaixo da linha atual e adicione texto lá (minúsculo o)
s ou cl Excluir caractere sob o cursor e alternar para o modo de inserção
S ou cc Exclua a linha inteira e mude para o modo Inserir
C Excluir da posição do cursor até o final da linha e iniciar o modo de inserção
c{movimento} Exclua {motion} e inicie o modo de inserção (consulte https://www.wikiod.com/pt/vim/movimento#Movimento Básico)

Inserir atalhos do modo #

Comando Descrição
<C-w> Excluir palavra antes do cursor
<C-t> Recuar a linha atual com um shiftwidth
<C-d> Unindent a linha atual com um shiftwidth
<C-f> reindentar a linha, (mover o cursor para a posição de recuo automático)
<C-a> Inserir texto inserido anteriormente
<C-e> Insira o caractere abaixo
<C-h> Excluir um caractere para trás
<C-y> Insira o caractere acima
<C-r>{registro} Insira o conteúdo de {register}
<C-o>{comando de modo normal} execute {comando de modo normal} sem sair do modo de inserção
<C-n> Próxima opção de autocompletar para a palavra atual
<C-p> Opção de preenchimento automático anterior para a palavra atual
<C-v> Insira um caractere pelo seu valor ASCII em decimal
<C-v>x Insira um caractere pelo seu valor ASCII em hexadecimal
<C-v>u Insira um caractere por seu valor unicode em hexadecimal
<C-k> Insira um dígrafo

Saindo do modo de inserção #

Comando Descrição
<Esc> Sai do modo de inserção, aciona comandos automáticos e abreviações
<C-[> Sinônimo exato de <Esc>
<C-c> Sai do modo de inserção, não aciona comandos automáticos

Algumas pessoas gostam de usar um par de caracteres relativamente incomum como jk como atalho para <Esc> ou <C-[> que pode ser difícil de alcançar em alguns teclados:

inoremap jk <Esc>l

Inserir texto em várias linhas de uma só vez #

Pressione Ctrl + v para entrar no modo de bloqueio visual.

Use ↑ / ↓ / j / k para selecionar várias linhas.

Pressione Shift + i e comece a digitar o que quiser.

Após pressionar Esc, o texto será inserido em todas as linhas selecionadas.

Lembre-se que Ctrl+c não é 100% equivalente a Esc e não funcionará nesta situação!

Existem pequenas variações de Shift + i que você pode pressionar enquanto estiver no modo de bloqueio visual:

Chave Descrição
c ou s Exclua o bloco selecionado e entre no modo de inserção
C Exclua as linhas selecionadas (do cursor até o final) e entre no modo de inserção
R Excluir linhas selecionadas e entrar no modo de inserção
A Anexar às linhas selecionadas, com a coluna no final da primeira linha

Observe também que pressionar . após uma operação de bloco visual repetirá essa operação onde o cursor estiver!

Cole o texto usando o comando “colar” do terminal #

Se você usar o comando paste do seu programa emulador de terminal, o Vim interpretará o fluxo de caracteres como se eles fossem digitados. Isso causará todos os tipos de efeitos indesejáveis, particularmente o mal recuo.

Para corrigir isso, no modo de comando:

:set paste

Em seguida, passe para o modo de inserção, com i, por exemplo. Observe que o modo agora é -- INSERT (colar) --. Agora cole com o comando do emulador de terminal ou com o mouse. Quando terminar vá para o modo de comando, com Esc e execute:

:set nopaste

Existe uma maneira mais simples, quando se quer colar apenas uma vez. Coloque isso no seu .vimrc (ou use o plugin unimpaired.vim):

function! s:setup_paste() abort
  set paste
  augroup unimpaired_paste
    autocmd!
    autocmd InsertLeave *
      \ set nopaste |
      \ autocmd! unimpaired_paste
  augroup end
endfunction

nnoremap <silent> yo :call <SID>setup_paste()<CR>o
nnoremap <silent> yO :call <SID>setup_paste()<CR>O

Agora pode-se simplesmente pressionar yo para colar o código sob o cursor, e então <Esc> para voltar ao modo normal/sem colar.

Colando de um registrador no modo de inserção #

Enquanto estiver no modo de inserção, você pode usar <C-r> para colar de um registro, que é especificado pelo próximo pressionamento de tecla. <C-r>" por exemplo cola do registrador sem nome (").

Veja :registros de ajuda.

Comandos e atalhos avançados de inserção #

Aqui está uma referência rápida para comandos/atalhos avançados de inserção, formatação e filtragem.

Comando/Atalho Resultado
g + ? + m Execute a codificação rot13, no movimento m
n + ctrl + a +n para o número sob o cursor
n + ctrl + x -n para o número sob o cursor
g + q+ m Formate linhas de movimento m para largura fixa
:rce w Linhas centrais no intervalo r à largura w
:rle i Linhas alinhadas à esquerda no intervalo r com recuo i
:rri w Linhas alinhadas à direita no intervalo r à largura w
!mc Filtrar linhas de movimento m através do comando c
n!!c Filtrar n linhas através do comando c
:r!c Filtra as linhas r do intervalo através do comando c

Desative o recuo automático para colar o código #

Ao colar texto por meio de um emulador de terminal, o recurso de recuo automático pode destruir o recuo do texto colado.

Por exemplo:

function () {
    echo 'foo'
    echo 'bar'
    echo 'baz'
}

será colado como:

function () {
    echo 'foo'
        echo 'bar'
            echo 'baz'
            }

Nesses casos, use a opção paste/nopaste para desabilitar/habilitar o recurso de recuo automático:

:set paste
:set nopaste

Além disso, há uma abordagem mais simples para o problema: Adicione a seguinte linha em seu .vimrc:

set pastetoggle=<F3>

E se você quiser colar como está da área de transferência. Basta pressionar F3 no modo insert e colar. Pressione F3 novamente para sair do modo colar.
