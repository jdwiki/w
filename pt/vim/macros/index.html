Gravando uma macro #

Uma maneira de criar uma macro é gravá-la.

Comece a gravar uma macro e salve-a em um registro (neste exemplo, usaremos a, mas pode ser qualquer registro para o qual você normalmente poderia puxar o texto):

qa

Em seguida, execute os comandos que deseja gravar na macro (aqui, cercaremos o conteúdo de uma linha com as tags <li>):

I<li><ESC>A</li>

Quando terminarmos com os comandos que queremos gravar na macro, pare a gravação:

q

Agora, sempre que quisermos executar a sequência gravada de comandos armazenados em a, use:

@a

e o vim repetirá a sequência gravada.

Da próxima vez que você quiser repetir a última macro que foi usada, você pode digitar @ duas vezes:

@@

E como um bônus extra, é bom lembrar que se você colocar um número antes de um comando, ele o repetirá muitas vezes. Então, você repete a macro salva no registro a 20 vezes com:

20@a

Editando uma macro do vim #

Às vezes, você cometerá um erro com uma macro longa, mas preferirá editá-la do que regravá-la inteiramente. Você pode fazer isso usando o seguinte processo:

  1. Coloque a macro em uma linha vazia com "<register>p.

Se sua macro está salva no registro a, o comando é "ap.

  1. Edite a macro conforme necessário.

  2. Puxe a macro para o registro correto movendo o cursor para o início da linha e usando "<register>y$.

Você pode reutilizar o registro original ou usar outro. Se você quiser usar o registro b, o comando é "by$. ou usando "<registro>d$ (exclui a linha não utilizada)

O que é uma macro? #

Uma macro é uma série de pressionamentos de tecla destinados a serem “reproduzidos” pelo Vim sem nenhum atraso. As macros podem ser armazenadas em registradores ou variáveis, vinculadas a teclas ou executadas na linha de comando.

Aqui está uma macro simples que coloca em maiúscula a terceira ‘palavra’ em uma linha:

0wwgUiw

Essa macro pode ser gravada no registrador q:

qq         start recording into register q
0wwgUiw    
q          stop recording

ou salvo diretamente no registrador q:

:let @q = '0wwgUiw'

para ser reproduzido com:

@q

Mas também pode ser digitado diretamente na linha de comando:

:normal 0wwgUiw

para reprodução instantânea através do comando :normal.

Ou coloque em uma variável:

:let myvar = '0wwgUiw'

para ser reproduzido com:

@=myvar

Ou salvo como um mapeamento:

nnoremap <key> 0wwgUiw

para ser reproduzido pressionando <tecla>.

Se você deseja armazenar uma macro para reutilização posterior, digite no modo de inserção:

<C-r>q

Isso insere a macro no registro q (neste exemplo: 0wwgUiw). Você pode usar isso saída, por exemplo para definir a macro em seu vimrc:

let @q='0wwgUiw'

Fazendo isso, o registro q é inicializado com esta macro toda vez que você inicia o vim.

Macros recursivas #

As macros do Vim também podem ser recursivas. Isso é útil quando você precisa agir em cada linha (ou outro objeto de texto) até o final do arquivo.

Para gravar uma macro recursiva, comece com um registro vazio. (Um registro pode ser esvaziado usando q<register>q.)

Escolha um ponto de partida consistente em cada linha para começar e terminar.

Antes de terminar a gravação, invoque a própria macro como último comando. (É por isso que o registro deve estar vazio: então não fará nada, pois a macro ainda não existe).

Exemplo, dado o texto:

line 1
line 2
line 3
foo bar
more random text
.
.
.
line ???

No modo normal, com o cursor na primeira linha e um registrador vazio a, pode-se gravar esta macro:

qaI"<Esc>A"<Esc>j@aq

Então, com uma única invocação de @a, todas as linhas do arquivo estariam agora entre aspas duplas.

Gravar e reproduzir ação (macros) #

com o comando q podemos simplificar muito trabalho tedioso no vim.

exemplo 1. gerar sequência de matriz (1 a 20).

PASSO 1. pressione i para entrar no modo de inserção, insira 1

1

ETAPA 2. Grave a seguinte ação: “anexe o último número à próxima linha e incremente o número”

  1. digite esc para sair do modo de entrada
  2. digite qa para entrar no modo de gravação, usando o buffer a
  3. digite yy e p para copiar a linha atual e colá-la como a próxima linha
  4. digite ctrl + a para incrementar o número
  5. digite q novamente para finalizar o registro

2

digite 18@a para reproduzir a ação 3 e a ação 4 na etapa 2.
