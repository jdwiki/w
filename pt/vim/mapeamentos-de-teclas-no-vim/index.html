A atualização dos mapeamentos de teclas do Vim permite resolver dois tipos de problemas: reatribuir comandos de teclas a letras que são mais fáceis de memorizar ou acessíveis e criar comandos de teclas para funções que não possuem nenhum. Aqui você aprenderá sobre as várias maneiras de [re]mapear comandos de tecla e o contexto ao qual eles se aplicam (ou seja, modos vim)

Mapeamento básico #

Visão geral do mapa #

Uma sequência de teclas pode ser remapeada para outra sequência de teclas usando uma das variantes map.

Como exemplo, o seguinte map típico sairá do modo Inserir quando você pressionar jk em sequência rápida:

:inoremap jk <Esc>

Operador de mapa #

Existem várias variantes de :map para diferentes modos.

Comandos | Modos ——————|—————– ———————— :map, :noremap, :unmap | Modo Normal, Visual e Operador pendente :map!, :noremap!, :unmap! | Modo de inserção e linha de comando :nmap, :nnoremap, :nunmap | Modo normal :imap, :inoremap, :iunmap | Modo Inserir e Substituir :vmap, :vnoremap, :vunmap | Modo Visual e Selecionar :xmap, :xnoremap, :xunmap | Modo visual :smap, :snoremap, :sunmap | Modo de seleção :cmap, :cnoremap, :cunmap | Modo de linha de comando :omap, :onoremap, :ounmap | Modo pendente do operador

Normalmente, você deve usar as variantes :noremap; ele torna o mapeamento imune a remapeamento e recursão.

comando map #

  • Você pode exibir todos os mapeamentos usando :map (ou uma das variações acima).
  • Para exibir o mapeamento atual para uma sequência de teclas específica, use :map <key> onde <key> é uma sequência de teclas
  • Teclas especiais como Esc são mapeadas usando notação especial <>, como <Esc>. Para obter a lista completa de códigos de chave, consulte http://vimdoc.sourceforge.net/htmldoc/intro.html#keycodes
  • :nmapclear - Limpa todos os mapas de modo normal
  • :nunmap - Desmapeia um mapa de modo normal
  • Você pode configurar o tempo máximo entre as teclas de uma sequência alterando as variáveis ​​timeout e ttimeout

Exemplos #

  • imap jk <Esc>: digitar jk no modo de inserção o levará de volta ao modo normal
  • nnoremap tt :tabnew<CR>: digitar tt no modo normal abrirá uma nova página de guia
  • nnoremap <C-j> <C-w>j: digitar <C-j> no modo normal fará você pular para a janela abaixo e para a esquerda
  • vmap <C-c> \cc: digitar <C-c> no modo visual executará \cc (comando NERDCommenter para comentar a linha). Como isso depende de um mapeamento de plugin, você não pode usar :vnoremap aqui!

Combinação de teclas do líder do mapa #

A chave líder pode ser usada como uma maneira de criar um mapeamento com uma associação de teclas que pode ser substituída pelo usuário final.

O líder é a chave \ por padrão. Para substituí-lo, o usuário final teria que executar :let g:mapleader='somekey(s)' antes de definir o mapeamento.

Em um cenário típico, o mapleader é definido no .vimrc, e os plugins usam <Leader> na parte de combinação de teclas de seus mapeamentos para personalizá-los.

No plugin, definiríamos mapeamentos com:

:nnoremap <Leader>a somecomplexaction

Isso mapearia a ação somecomplexaction para a combinação de teclas \+a.

A ação a sem o líder não muda.

Também é possível usar <Plug>Mappings para deixar mais espaço para personalizar as combinações de teclas dos plugins.

Ilustração do mapeamento básico (atalhos úteis). #

Na maioria dos editores de texto, o atalho padrão para salvar o documento atual é Ctrl+S (ou Cmd+S no macOS).

O Vim não possui esse recurso por padrão, mas isso pode ser mapeado para facilitar as coisas. Adicionar as seguintes linhas no arquivo .vimrc fará o trabalho.

nnoremap <c-s> :w<CR>
inoremap <c-s> <c-o>:w<CR>

O comando nnoremap mapeia Ctrl+s para o comando :w (gravar conteúdo atual no arquivo) enquanto o comando inoremap mapeia o comando Ctrl</ kbd>+S para o comando :w e retorna ao modo de inserção (<c-o> entra no modo normal para um comando e retorna ao modo de inserção posteriormente, sem alterar a posição do cursor que outros soluções como <esc>:w<cr>a não podem garantir).

De forma similar,

" This is commented, as Ctrl+Z is used in terminal emulators to suspend the ongoing program/process.
" nnoremap <c-z> :u<CR>

" Thus, Ctrl+Z can be used in Insert mode
inoremap <c-z> <c-o>:u<CR>

" Enable Ctrl+C for copying selected text in Visual mode
vnoremap <c-c> <c-o>:y<CR>

PS: No entanto, deve-se notar que Ctrl+S pode não funcionar como esperado ao usar ssh (ou PuTTY). A solução para isso não está no escopo deste documento, mas pode ser encontrada Aqui.
