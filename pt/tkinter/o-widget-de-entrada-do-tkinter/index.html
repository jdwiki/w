Sintaxe #

  • entrada = tk.Entry(pai, **kwargs)
  • entrada.get()
  • entry.insert(index, “valor”)
  • entry.delete(start_index, end_index)
  • entry.bind(evento, retorno de chamada)

Parâmetros #

Parâmetro Descrição
pai Os widgets tkinter existem em uma hierarquia. Com exceção da janela raiz, todos os widgets têm um pai. Alguns tutoriais online chamam isso de “mestre”. Quando o widget é adicionado à tela com pack, place ou grid, ele aparecerá dentro deste widget pai
largura A largura especifica a largura desired do widget com base em uma largura média de caracteres. Para fontes de largura variável, isso é baseado na largura do caractere zero (0). O padrão é 20. Observe que a largura real pode ser maior ou menor dependendo de como ele é adicionado à tela.

Esses exemplos assumem que tkinter foi importado com import tkinter as tk (python 3) ou import Tkinter as tk (python 2).

Obtendo o valor de um widget de entrada #

O valor de um widget de entrada pode ser obtido com o método get do widget:

name_entry = tk.Entry(parent)
...
name = name_entry.get()

Opcionalmente, você pode associar uma instância de uma StringVar e recuperar o valor da StringVar em vez do widget:

name_var = tk.StringVar()
name_entry = tk.Entry(parent, textvariable=name_var)
...
name = name_var.get()

Criando um widget de entrada e definindo um valor padrão #

entry = tk.Entry(parent, width=10)
entry.insert(0, "Hello, World!")

Adicionando validação a um widget de entrada #

Para restringir os caracteres que podem ser digitados em um widget de entrada, apenas números, por exemplo, um comando de validação pode ser adicionado à entrada. Um comando de validação é uma função que retorna True se a alteração for aceita, False caso contrário. Esta função será chamada cada vez que o conteúdo da entrada for modificado. Vários argumentos podem ser passados ​​para esta função, como o tipo de alteração (inserção, exclusão), o texto inserido, …

def only_numbers(char):
    return char.isdigit()

validation = parent.register(only_numbers)
entry = Entry(parent, validate="key", validatecommand=(validation, '%S'))

A opção validate determina o tipo de evento que aciona a validação, aqui, é qualquer pressionamento de tecla na entrada. O '%S' na opção validatecommand significa que o caractere inserido ou excluído é passado como argumento para a função only_numbers. A lista completa de possibilidades pode ser encontrada aqui.

Obtendo int do widget de entrada #

Ao usar o método .get(), o que estiver no widget de entrada será convertido em uma string. Por exemplo, independentemente do tipo de entrada (pode ser um número ou frase), o resultado resultante será uma string. Se o usuário digitar 4 a saída será “4” como em uma string. Para obter um int de um Entry Widget, primeiro chame o método .get().

What_User_Wrote = Entry.get()

Agora convertemos essa string em um int assim:

Convert_To_Int = int(What_User_Wrote)

Da mesma forma, se você quiser economizar tempo, basta fazer:

Convert_To_Int = int(Entry.get())

Você pode usar o método acima se não quiser converter str para int.
