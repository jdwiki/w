  • Aqui está um exemplo de como transformar botões de opção em caixas de botão:
  • Crie um grupo de botões de opção

Sintaxe #

  • radiobutton = tk.Radiobutton(pai, **kwargs)

Parâmetros #

Parâmetro Descrição
pai Os widgets tkinter existem em uma hierarquia. Com exceção da janela raiz, todos os widgets têm um pai. Alguns tutoriais online chamam isso de “mestre”. Quando o widget for adicionado à tela com pack, place ou grid, ele aparecerá dentro deste widget pai.
comando função chamada cada vez que o usuário altera o estado do botão de opção
indicador 1 ou True para botões de opção, 0 ou False para caixas de botão
texto Texto a ser exibido ao lado do botão de opção.
valor Quando o botão de opção é selecionado, a variável de controle associada é definida como valor.
variável Variável de controle que o radiobutton compartilha com o outro radiobutton do grupo.

Esses exemplos assumem que tkinter foi importado com import tkinter as tk (python 3) ou import Tkinter as tk (python 2).

Referência:

Para transformar o exemplo acima em uma “caixa de botão” em vez de um conjunto de botões de opção, defina a opção do indicador para 0. Nesse caso, não há um indicador de botão de opção separado e o botão selecionado é desenhado como SUNKEN em vez de RAISED:

-effbot

Aqui está um exemplo de como transformar botões de opção em caixas de botão: #

import tkinter as tk
root = tk.Tk()

rbvar = StringVar()
rbvar.set(" ")

rb1 = tk.Radiobutton(root, text="Option 1", variable=rbvar, value='a', indicatoron=0)
rb1.pack()

rb2 = tk.Radiobutton(root, text="Option 2", variable=rbvar, value='b', indicatoron=0)
rb2.pack()

Crie um grupo de botões de opção #

Tal grupo é feito de botões de rádio que compartilham uma variável de controle para que não mais de um possa ser selecionado.

# control variable
var = tk.IntVar(parent, 0)

# group of radiobuttons
for i in range(1,4):
    tk.Radiobutton(parent, text='Choice %i' % i, value=i, variable=var).pack()

tk.Button(parent, text='Print choice', command=lambda: print(var.get())).pack()
