Integre o Firebase ao Google AppScript para ler e gravar dados no Firebase Database.

Firebase é um sistema de banco de dados NoSQL do Google que usa banco de dados em tempo real para ajudar a criar e hospedar aplicativos em dispositivos móveis, desktops e tablets. Os bancos de dados NoSQL usam os objetos JSON para armazenar os dados em formato estruturado.

Conectando-se a um projeto do Firebase no GAS e transferindo dados do Google Spreadsheet para o Firebase #

Instale o recurso Firebase no AppScript #

  • Para isso clique em Recursos e depois em Bibliotecas.
  • O Firebase tem uma chave de biblioteca de projeto exclusiva que precisa ser instalada no AppScript.
  • Clique em Bibliotecas The following pop-up appears. Enter the following project key in the textbox. MYeP8ZEEt1ylVDxS7uyg9plDOcoke7-2l This is the project library key for Firebase.
  • Agora na versão escolha a versão de lançamento público estável.
  • Clique em Salvar. Agora o Firebase está instalado com sucesso no seu AppScript para você trabalhar.

Agora vamos dar um exemplo de leitura e gravação de dados do Firebase. #

  • Agora pegamos uma tabela de amostra projetada no Planilhas Google.

  • Agora vamos construir o banco de dados no Firebase usando essa tabela nas planilhas. Adicione o seguinte código no AppScript.

      function writeDataToFirebase() {
    var ss = SpreadsheetApp.openById("1LACsj0s3syAa9gvORdRWBhJ_YcXHybjQfHPgw3TLQ6g");
    var sheet = ss.getSheets()[0];
    var data = sheet.getDataRange().getValues();
    var dataToImport = {};
    for(var i = 1; i < data.length; i++) {
      var firstName = data[i][0];
      var lastName = data[i][1];
      dataToImport[firstName + '-' + lastName] = {
        firstName:firstName,
        lastName:lastName,
        emailAddress:data[i][2],
        semester:data[i][4],
        department:data[i][5],
      };
    }
    var firebaseUrl = "https://example-app.firebaseio.com/";
    var secret = "secret-key";
    var base = FirebaseApp.getDatabaseByUrl(firebaseUrl, secret);
    base.setData("", dataToImport);
  }

Substitua o ID da planilha e o firebaseURL e a chave secreta.

Como encontrar o firebaseURL e a chave secreta? #

  • Vá para o Firebase Dashboard e clique na engrenagem de configurações no canto superior esquerdo. Clique em Configurações do projeto.
  • Vá para a seção Contas de serviço, você pode encontrar o databaseURL. Isso serve como firebaseURL.
  • Agora clique na guia Segredos do Banco de Dados e você poderá encontrar a chave secreta.

Agora você inseriu o firebaseURL e a chave secreta. Agora você está pronto para ir. Clique no código de execução no mecanismo AppScript. #

  • Ele pedirá para revisar as permissões pela primeira vez quando você executar.
  • Clique em Revisar Permissões e Permitir.
  • Agora você executa sua função e pode ver a tabela criada no Firebase Database.

Para ver o banco de dados, acesse o painel do Firebase e clique no banco de dados para visualizar o banco de dados.

Mais algumas funções para implementar leitura e escrita. #

1. Para escrever um dado simples para testar se a conexão está funcionando ou não. #

function myFunction(){
 var firebaseUrl = "https://example-app.firebaseio.com/";
 var secret = "secret-key";
 var base = FirebaseApp.getDatabaseByUrl(firebaseUrl, secret);
 base.setData("test", "Hello Firebase"); 
}

2. Para ler todos os dados #

function getAllData() {
         var firebaseUrl = "https://example-app.firebaseio.com/";
         var secret = "secret-key";
var base = FirebaseApp.getDatabaseByUrl(firebaseUrl, secret);
 var data = base.getData();
          for(var i in data) {
            Logger.log(data[i].firstName + ' ' + data[i].lastName);
          }
}

Os dados lidos são exibidos nos Logs. Para verificar os logs clique em View → Logs ou simplesmente use Control + Enter.

3. Para ler um registro específico #

function getContact() {
  var firebaseUrl = "https://example-app.firebaseio.com/";
  var secret = "secret-key";
  var base = FirebaseApp.getDatabaseByUrl(firebaseUrl, secret);
  var contact = base.getData("Yash-Udasi");
  Logger.log(contact);
}

Os dados lidos são exibidos nos Logs. Para verificar os logs clique em View → Logs ou simplesmente use Control + Enter.

4. Para atualizar um registro específico. #

function updateData() {
  var firebaseUrl = "https://example-app.firebaseio.com/";
  var secret = "secret-key";
  var base = FirebaseApp.getDatabaseByUrl(firebaseUrl, secret);
  base.updateData("Yash-Udasi/emailAddress", "yash.udasi@fyuff.com");
}
