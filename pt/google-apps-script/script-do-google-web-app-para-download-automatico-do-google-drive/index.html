Script do Google Web App para download automático do Google Drive

  • código.gs
  • Como funciona

Nesta página

  • código.gs
  • Como funciona

Este Script Web Simples do Google App (autônomo) permite que o Google Drive seja repetidamente pesquisado para que os arquivos sejam baixados para o PC local do usuário.

Mostra como usar um script de aplicativo para fornecer a função de:

  1. Interface do usuário (uma simples neste exemplo)
  2. A página de download do arquivo.

Para uma explicação mais completa de como funciona, leia o Exemplo “Como funciona”.

O Web Script deve ser publicado para funcionar.

Os pop-ups devem estar ativados para https://script.google.com
