Visão geral da instalação #

O ActionScript 3 pode ser usado instalando o Adobe AIR SDK ou Apache Flex SDK ou como parte do produto Adobe Animate CC (anteriormente conhecido como Flash Professional).

Adobe Animate CC é uma solução de software profissional que pode ser usada para criar projetos AS3 usando ferramentas visuais - uma vez instalado, não são necessárias mais etapas para começar a criar projetos AS3.

O AIR SDK e o Flex SDK podem ser usados ​​com ferramentas de linha de comando ou com vários IDEs de terceiros.

Além do Adobe Animate CC, existem outros quatro IDEs populares capazes de trabalhar com AS3. Esses IDEs têm suas próprias instruções sobre como começar.

  • Flash Builder (da Adobe - baseado no Eclipse)
  • IntelliJ IDEA (Por Jetbrains)
  • FlashDesenvolvimento
  • FDT (Plugin Eclipse)

Olá Mundo #

Um exemplo de classe de documento que imprime “Hello, World” no console de depuração quando instanciado.

import flash.display.Sprite;

public class Main extends Sprite {

    public function Main() {
        super();

        trace("Hello, World");
    }

}

Instalação de desenvolvimento em Flash #

FlashDevelop é um IDE de código aberto multiplataforma criado em 2005 para desenvolvedores Flash. Sem custo, é uma maneira muito popular de começar a desenvolver com AS3.

Para instalar o FlashDevelop:

  1. Baixe o arquivo de instalação e execute o instalador
  2. Quando a instalação estiver concluída, execute o FlashDevelop. Na primeira inicialização, a janela App Man deve aparecer solicitando que você escolha quais SDKs e ferramentas instalar.

Se o AppMan não abrir automaticamente ou você quiser adicionar algo mais tarde, abra-o escolhendo ‘Instalar software’ no menu ‘Ferramentas’.

Verifique o item AIR SDK+ ACS 2.0 (na seção ‘Compiler’) e o item Flash Player (SA) na seção ‘Runtimes’ (além de qualquer outra coisa que você queira instalar). Clique no botão instalar.

  1. Uma vez instalado o SDK, vamos testar criando um projeto hello world. Comece criando um novo projeto (no menu Project)

  2. Escolha AIR AS3 Projector na lista e dê a ele um nome/local.

  3. No painel do gerenciador de projetos (escolha ‘Gerenciador de Projetos’ no menu de visualização, se ainda não estiver visível), expanda a pasta src e abra o arquivo Main.as.

  4. No arquivo Main.as, agora você pode criar um primeiro programa de exemplo como https://www.wikiod.com/pt/actionscript-3

  5. Execute seu projeto clicando no ícone de reprodução, ou pressionando F5, ou Ctrl+Enter. O projeto irá compilar e quando terminar uma janela em branco deve aparecer (este é o seu aplicativo). Na janela de saída do FlashDevelop, você deverá ver as palavras: Hello World.

Agora você está pronto para começar a desenvolver aplicativos AS3 com FlashDevelop!

Instalação do Apache Flex #

de http://flex.apache.org/doc-getstarted.html

  1. Faça o download do instalador do SDK
  2. Execute o instalador do SDK. A primeira pergunta que será feita é o diretório de instalação.
  • em um Mac, use /Applications/Adobe Flash Builder 4.7/sdks/4.14.0/
  • em um PC, use C:\Program Files(x86)\Adobe Flash Builder 4.7\sdks\4.14.0

Você precisará criar as pastas 4.14.0. Pressione Avançar. Aceite as licenças do SDK e instale.

Instruções específicas do IDE para configuração do Apache Flex:

  • Construtor de Flash
  • IntelliJ IDEA
  • FlashDesenvolvimento
  • FDT

Construindo projetos Flex ou Flash na linha de comando usando mxmlc #

O compilador Flex (mxmlc) é uma das partes mais importantes do Flex SDK. Você pode editar o código AS3 em qualquer editor de texto que desejar. Crie um arquivo de classe principal que se estenda de DisplayObject.

Você pode acionar compilações na linha de comando da seguinte maneira:

mxmlc -source-path="." -default-size [width in pixels] [height in pixels] -default-frame-rate [fps] -o "outputPath.swf" "mainClass.as"

Se você precisar compilar um projeto Flash (em oposição ao Flex), você pode adicionar uma referência à biblioteca Flash da seguinte forma (você precisará ter o Adobe Animate IDE instalado):

mxmlc -source-path="." -library-path+="/Applications/Adobe Animate CC 2015.2/Adobe Animate CC 2015.2.app/Contents/Common/Configuration/ActionScript 3.0/libs" -static-link-runtime-shared-libraries=true -default-size [width in pixels] [height in pixels] -default-frame-rate [fps] -o "outputPath.swf" "mainClass.as"

Ou no Windows:

mxmlc -source-path="." -library-path+="C:\Program Files\Adobe\Adobe Animate CC 2015.2\Common\Configuration\ActionScript 3.0\libs" -static-link-runtime-shared-libraries=true -default-size [width in pixels] [height in pixels] -default-frame-rate [fps] -o "outputPath.swf" "mainClass.as"

Um exemplo “Hello World” exibido #

package {
    import flash.text.TextField;
    import flash.display.Sprite;

    public class TextHello extends Sprite {
        public function TextHello() {
            var tf:TextField = new TextField();
            tf.text = "Hello World!"
            tf.x = 50;
            tf.y = 40;
            addChild(tf);
        }
    }
}

Esta classe usa a classe TextField para exibir o texto.
