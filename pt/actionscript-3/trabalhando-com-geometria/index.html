Obtendo o ângulo entre dois pontos #

Usando a matemática vanilla:

var from:Point = new Point(100, 50);
var to:Point = new Point(80, 95);

var angle:Number = Math.atan2(to.y - from.y, to.x - from.x);

Usando um novo vetor representando a diferença entre os dois primeiros:

var difference:Point = to.subtract(from);

var angle:Number = Math.atan2(difference.y, difference.x);

Nota: atan2() retorna radianos, não graus.

Obtendo a distância entre dois pontos #

Usando a matemática vanilla:

var from:Point = new Point(300, 10);
var to:Point = new Point(75, 40);

var a:Number = to.x - from.x;
var b:Number = to.y - from.y;

var distance:Number = Math.sqrt(a * a + b * b);

Usando a funcionalidade incorporada do Point:

var distance:Number = to.subtract(from).length; // or
var distance:Number = Point.distance(to, from);

Determine se um ponto está dentro de uma área retangular #

Você pode testar se um ponto está dentro de um retângulo usando Rectangle.containsPoint():

var point:Point = new Point(5, 5);
var rectangle:Rectangle = new Rectangle(0, 0, 10, 10);

var contains:Boolean = rectangle.containsPoint(point); // true

Convertendo radianos para graus #

var degrees:Number = radians * 180 / Math.PI;

Convertendo graus para radianos #

var radians:Number = degrees / 180 * Math.PI;

Movendo um ponto ao longo de um ângulo #

Supondo que você tenha o ângulo em que gostaria de se mover e um objeto com valores x e y que deseja mover:

var position:Point = new Point(10, 10);
var angle:Number = 1.25;

Você pode se mover ao longo do eixo x com Math.cos:

position.x += Math.cos(angle);

E o eixo y com Math.sin:

position.y += Math.sin(angle);

É claro que você pode multiplicar o resultado de Math.cos e Math.sin pela distância que deseja percorrer:

var distance:int = 20;

position.x += Math.cos(angle) * distance;
position.y += Math.sin(angle) * distance;

Nota: O ângulo de entrada deve estar em radianos.

O valor de um círculo em graus e radianos #

  • Um círculo inteiro é 360 graus ou Math.PI * 2 radianos.
  • Metade desses valores segue para ser 180 graus ou Math.PI radianos.
  • Um quarto é então 90 graus ou Math.PI / 2 radianos.

Para obter um segmento como uma porcentagem de um círculo inteiro em radianos:

function getSegment(percent:Number):Number {
    return Math.PI * 2 * percent;
}

var tenth:Number = getSegment(0.1); // One tenth of a circle in radians.
