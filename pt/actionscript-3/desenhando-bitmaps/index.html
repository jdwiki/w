Desenha um objeto de exibição em dados de bitmap #

Uma função auxiliar para criar uma cópia de bitmap de um objeto. Isso pode ser usado para converter objetos vetoriais, texto ou Sprites aninhados complexos em um bitmap achatado.

function makeBitmapCopy(displayObj:IBitmapDrawable, transparent:Boolean = false, bgColor:uint = 0x00000000, smooth:Boolean = true):Bitmap {

    //create an empty bitmap data that matches the width and height of the object you wish to draw
    var bmd:BitmapData = new BitmapData(displayObj.width, displayObj.height, transparent, bgColor);
            
    //draw the object to the bitmap data
    bmd.draw(displayObj, null, null, null, null, smooth);
            
    //assign that bitmap data to a bitmap object
    var bmp:Bitmap = new Bitmap(bmd, "auto", smooth);
            
    return bmp;
}

Uso:

var txt:TextField = new TextField();
txt.text = "Hello There";

var bitmap:Bitmap = makeBitmapCopy(txt, true); //second param true to keep transparency
addChild(bitmap);

Desenha um objeto de exibição com quaisquer coordenadas do ponto de registro #

    public function drawDisplayObjectUsingBounds(source:DisplayObject):BitmapData {
        var bitmapData:BitmapData;//declare a BitmapData
        var bounds:Rectangle = source.getBounds(source);//get the source object actual size
        //round bounds to integer pixel values (to aviod 1px stripes left off)
        bounds = new Rectangle(Math.floor(bounds.x), Math.floor(bounds.y), Math.ceil(bounds.width), Math.ceil(bounds.height));

        //to avoid Invalid BitmapData error which occures if width or height is 0
        //(ArgumentError: Error #2015)
        if((bounds.width>0) && (bounds.height>0)){
            //create a BitmapData
            bitmapData = new BitmapData(bounds.width, bounds.height, true, 0x00000000);                
            var matrix:Matrix = new Matrix();//create a transform matrix
            //translate if to fit the upper-left corner of the source
            matrix.translate(-bounds.x, -bounds.y);
            bitmapData.draw(source, matrix);//draw the source
            return bitmapData;//return the result (exit point)
        }
        //if no result is created - return an empty BitmapData
        return new BitmapData(1, 1, true, 0x00000000);
    }

Uma nota lateral: Para getBounds() retornar valores válidos, o objeto precisa ter acesso ao palco pelo menos uma vez, caso contrário, os valores são falsos. Um código pode ser adicionado para garantir que o source passado tenha um estágio e, caso contrário, ele pode ser adicionado ao estágio e removido novamente.

Animando uma folha de sprite #

Uma folha de sprite por definição é um bitmap que contém uma certa animação. Jogos antigos usam sprite sheet do tipo grid, ou seja, cada quadro ocupa uma região igual, e os quadros são alinhados pelas bordas para formar um retângulo, provavelmente com alguns espaços desocupados. Mais tarde, para minimizar o tamanho do bitmap, as folhas de sprite começam a ser “embaladas” removendo o espaço em branco extra ao redor do retângulo que contém cada quadro, mas ainda assim cada quadro é um retângulo para simplificar as operações de cópia.

Para animar uma folha de sprite, duas técnicas podem ser usadas. Primeiro, você pode usar BitmapData.copyPixels() para copiar uma certa região de sua folha de sprite para um Bitmap exibido, produzindo um personagem animado. Essa abordagem é melhor se você usar um único Bitmap exibido que hospeda a imagem inteira.

var spriteSheet:BitmapData;
var frames:Vector.<Rectangle>; // regions of spriteSheet that represent frames
function displayFrameAt(frame:int,buffer:BitmapData,position:Point):void {
    buffer.copyPixels(spriteSheet,frames[frame],position,null,null,true);
}

A segunda técnica pode ser usada se você tiver muitos ‘Sprites’ ou ‘Bitmap’s na lista de exibição, e eles compartilharem a mesma folha de sprite. Aqui, o buffer de destino não é mais um único objeto, mas cada objeto tem seu próprio buffer, então a estratégia adequada é manipular a propriedade bitmapData dos bitmaps. Antes de fazer isso, no entanto, a folha de sprite deve ser cortada em quadros individuais.

public class Stuff extends Bitmap {
    static var spriteSheet:Vector.<BitmapData>;
    function displayFrame(frame:int) {
        this.bitmapData=spriteSheet[frame]; 
    }
    // ...
}
