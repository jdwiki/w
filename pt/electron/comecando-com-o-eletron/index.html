Instalação de Elétron #

Dependências #

Para instalar o electron você deve primeiro instalar o Node.js, que vem com o npm.

Como instalá-lo? #

Use npm:

# Install the `electron` command globally in your $PATH
npm install electron -g

# OR

# Install as a development dependency
npm install electron --save-dev

Olá Mundo! #

Configurar #

Uma estrutura de projeto Electron geralmente se parece com isso:

hello-world-app/
├── package.json
├── index.js
└── index.html

Agora vamos criar os arquivos e inicializar nosso package.json.

$ mkdir hello-world-app && cd hello-world-app
$ touch index.js
$ touch index.html
$ npm init

Nota: Se o parâmetro main não for especificado em package.json, o Electron usará index.js como o ponto de entrada padrão.

O Processo Principal #

No Electron, o processo que executa o script principal do package.json é chamado de processo principal. Aqui podemos exibir uma GUI criando instâncias BrowserWindow.

Adicione o seguinte a index.js:

const { app, BrowserWindow } = require('electron')

// Global reference to the window object
let win

// This method will be called when Electron has finished
// initialization and is ready to create browser windows
app.on('ready', function(){
    // Create the window
    win = new BrowserWindow({width: 800, height: 600})

    // Open and load index.html to the window
    win.loadURL('file://' + __dirname + '/index.html')

    // Emitted when the window is closed.
    win.on('closed', () => {
        // Dereference the window object
        win = null
    });
})

// Quit the app if all windows are closed
app.on('window-all-closed', () => {
    app.quit()
})

Modelo HTML e processo de renderização #

Em seguida, criamos a GUI para o aplicativo. Electron usa páginas da web como sua GUI, cada uma rodando em seu próprio processo chamado renderer process.

Adicione o seguinte código ao index.html:

<!DOCTYPE html>
<html>
<head>
    <title>Hello World</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>

Executando o aplicativo #

Existem várias maneiras de executar um aplicativo Electron.

Com electron-prebuilt instalado globalmente #

Primeiro, certifique-se de ter [electron-prebuilt instalado](https://www.wikiod.com/pt/electron/comecando-com-o-eletron#Instalação de Elétron -e-executá-lo).

Agora podemos testar o aplicativo usando este comando:

$ electron .

Método 2 - Sem electron-prebuilt instalado globalmente #

Primeiro, teremos que entrar na pasta do seu aplicativo (a pasta onde está package.json).

Lá, abra uma janela Terminal/Prompt de Comando e digite npm install para instalar o necessário na pasta desse aplicativo.

Depois, digite npm start para executar o aplicativo. Tenha em mente que seu package.json ainda precisa especificar um script ‘start’.

Se tudo funcionou corretamente, você deve ver algo assim:

Parabéns! Você criou com sucesso seu primeiro aplicativo Electron.
