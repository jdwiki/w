Baixando GTK+3 (também adequado para outras versões) e Configurando #

Baixando o pacote:

A maneira mais fácil de baixar o pacote GTK necessário é pesquisá-lo e baixá-lo usando este link: https://git.gnome.org/browse/gtk+ (GNOME GIT Repository)

GNOME GIT Repository fornece os pacotes para diferentes versões e você pode encontrar facilmente a versão desejada percorrendo a lista. Eles são enviados por outros usuários ou autores e você encontrará frequentemente novos pacotes.

Alternativas:

A outra opção ou método alternativo para adquirir o bundle é usar o MSYS2. Este método é um pouco complicado e não está bem explicado ou documentado. A melhor maneira é baixar diretamente o pacote de um site de hospedagem de arquivos externo.

Método MSYS2-

  1. http://www.gtk.org/download/windows.php (Recurso do site oficial sobre como adquirir o pacote)

  2. https://msys2.github.io/ (baixar MSYS2)

Observação: O restante desta documentação é baseado na suposição de que o pacote GTK é baixado do repositório GNOME GIT. Não testado com o método MSYS2. Como o pacote permanece universal, o método usado não deve ser um problema. Com base na facilidade, a maneira altamente recomendada de obter o pacote é T0 usando o repositório GNOME GIT.

Extrair e armazenar o pacote em um local desejado (explicado usando a unidade C:).

  1. Crie uma pasta chamada gtk no C: Drive. Você pode escolher outros nomes para a pasta, mas esse nome nos ajuda a identificar o conteúdo com bastante facilidade e lembrar o nome fica mais fácil. Vamos chamar a localização da pasta Gtk como %GD% para facilitar e neste caso é C:\gtk (pode ser diferente no seu caso).

  2. Configurando o Caminho para o binário Gtk nas Variáveis ​​de Ambiente do Windows. Isso é importante para acessar e obter as bibliotecas Gtk necessárias ao usar o Cmd para compilar e executar os Programas. Para configurar a variável de ambiente do caminho:

  • Windows XP : clique com o botão direito em "Meu Computador" -> "Propriedades".
  • Windows Vista/7/8/10 : clique com o botão direito do mouse em "Computador" -> "Propriedades" -> "Configurações avançadas do sistema". Clique na guia "Avançado" -> "Ambiente variáveis".
  • Adicione o novo Gtk bin file Path (%GD%\bin) à variável Path (em System Variables ou adicione uma variável chamada Path à seção User Variables e adicione o valor nela) disponível na caixa de diálogo que se abre . Você também deve adicionar uma nova variável chamada PKG_CONFIG_PATH e dar a ela o seguinte valor- C:\gtk\lib\pkgconfig (já que minha pasta gtk está localizada na unidade C:).
Nome da Variável Valor
Caminho %GD%\bin
PKG_CONFIG_PATH %GD%\lib\pkgconfig

%GD% é a localização da pasta Gtk Agora terminamos de extrair e configurar as variáveis ​​de ambiente. Você descobrirá que adicionei o valor à variável Path nas seções de variáveis ​​do sistema, bem como na seção de variáveis ​​do usuário. Adicionar a qualquer uma dessas seções é mais que suficiente.

3.Testar a configuração (Você deve ter o GCC instalado em seu sistema já-MinGW preferível (http://www.mingw.org/): Execute os seguintes comandos no cmd:

pango-querymodules:%GTKDIR%\etc\pango\pango.modules

gdk-pixbuf-query-loaders:%GTKDIR%\lib\gdk-pixbuf-2.0\2.10.0\loaders.cache

gtk-query-immodules-3.0: %GTKDIR%\lib\gtk-3.0\3.0.0\immodules.cache

  • No console, verifique se “pkg-config” imprime algo razoável digitando: pkg-config --cflags --libs gtk+-3.0 Deve imprimir algo semelhante ao mostrado na imagem a seguir.[! Resultado do Comando]4

É isso. Você baixou e configurou o GTK em seu sistema Windows.
