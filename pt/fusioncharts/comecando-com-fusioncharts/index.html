Instalação #

Como obter o FusionCharts:

  • Download do site oficial do FusionCharts

  • Instalar via npm:

    Install fusioncharts package #

    npm install fusioncharts

    Install fusionmaps package #

    npm install fusionmaps

  • Instalar via bower:

    Install fusioncharts package #

    bower install fusioncharts

    Install fusionmaps package #

    bower install fusionmaps

Gráficos, widgets e mapas #

O FusionCharts Suite XT é uma biblioteca de gráficos JavaScript abrangente que inclui mais de 90 gráficos e 1150 mapas. Todos esses gráficos e mapas são distribuídos em 4 pacotes diferentes que são nomeados como:

  • Fusion Charts XT
  • FusionWidgets XT
  • PowerCharts XT
  • Fusion Maps XT

Os detalhes de cada um desses pacotes são fornecidos abaixo.

FusionCharts XT

Ele inclui todos os gráficos de uso geral, desde gráficos básicos, como gráficos de barras, colunas, áreas, linhas e empilhados, até gráficos avançados, como gráficos de combinação, rolagem, Marimekko, gráfico XY e gráficos de linhas de zoom.

Veja exemplos ao vivo dos gráficos incluídos no FusionCharts XT

FusionWidgets XT

Inclui uma ampla variedade de medidores e gráficos, como medidor angular, medidor linear, medidor de bulbo, gráfico de Gantt, gráficos de pirâmide e funil, gráficos de tamanho de palavras e gráficos em tempo real. Esses medidores e gráficos podem ser usados ​​para representar KPIs e dados em tempo real em painéis, monitores e relatórios.

Veja exemplos ao vivo dos gráficos e medidores no FusionWidgets XT

PowerCharts XT

Ele inclui gráficos avançados, como o gráfico de mapa de árvore, o gráfico de mapa de calor, o gráfico de velas, o gráfico de nó de arrasto, gráficos de spline, gráficos de erro e gráficos visualmente editáveis ​​ou selecionáveis. Esses gráficos podem ser usados ​​para uso específico do domínio, como análise de desempenho, planejamento financeiro e plotagem de estruturas hierárquicas.

Veja exemplos ao vivo dos gráficos incluídos no PowerCharts XT

FusionMaps XT

Inclui mais de 1150 mapas interativos para todos os continentes e principais países. Esses mapas podem ser usados ​​para plotar dados geográficos, bem como criar marcadores para localizar localizações exatas.

Veja exemplos ao vivo dos mapas incluídos no FusionMaps XT
