Olá mundo em CherryPy #

Se você tem um virtualenv e o CherryPy já está instalado nele, crie um arquivo hello.py:

#!/usr/bin/env python
# -*- coding: UTF-8 -*-

import cherrypy

class HelloWorld(object):
    @cherrypy.expose
    def index(self):
        return 'Hello World!'

    @cherrypy.expose
    def greet(self, name):
        return 'Hello {}!'.format(name)

cherrypy.quickstart(HelloWorld())

Em seguida, execute o arquivo: $ hello.py ou $ python hello.py. Você deve ver uma saída semelhante a esta:

Instruções de instalação #

Pré-condições #

  • Estas instruções supõem que você tenha qualquer tipo de Linux, Unix, Mac com bash ou Git-bash Windows.
  • Windows: Baixe e instale Git-bash for Windows, então execute ‘bash’ na linha de comando.
  • Outros shells além do bash também funcionam, basta substituir o comando activate abaixo por activate.csh ou Google: “virtualenv activate your-shell-name”.

Antes de começar, verifique se Python, virtualenv e pip estão instalados:

  • $ python --version
  • $ virtualenv --version
  • $ pip --version

#Instalar#

Crie um diretório com seu web/app, crie um ambiente e instale o pacote CherryPy.

  • $ mkdir /develop/myapp/
  • $ cd /develop/myapp/
  • $ virtualenv venv
  • $ source venv/bin/activate
    • On Windows in Git-bash: $ source venv/Scripts/activate
  • (venv) $ pip install cherrypy
  • (venv) $python
Python 3.5.2 ...
>>> import cherrypy
>>> cherrypy
<module 'cherrypy' from '... venv/site-packages/cherrypy/__init__.py'>

Parabéns! Agora você está pronto para seu primeiro aplicativo CherryPy.

Upload de arquivo com CherryPy #

Este exemplo consiste em três partes:

  • server.py - aplicativo CherryPy que pode receber e salvar um arquivo.
  • webpage.html - Exemplo de como fazer upload de um arquivo para server.py de uma página da web.
  • cli.py - Exemplo de como fazer upload de um arquivo para server.py a partir de uma ferramenta de linha de comando.
  • Bônus - upload.txt - arquivo que você enviará.

#servidor.py#

#!/usr/bin/env python
# -*- coding: UTF-8 -*-

import os
import cherrypy

config = {
    'global' : {
        'server.socket_host' : '127.0.0.1',
        'server.socket_port' : 8080
    }
}


class App:

    @cherrypy.expose
    def upload(self, ufile):
        # Either save the file to the directory where server.py is
        # or save the file to a given path:
        # upload_path = '/path/to/project/data/'
        upload_path = os.path.dirname(__file__)

        # Save the file to a predefined filename
        # or use the filename sent by the client:
        # upload_filename = ufile.filename
        upload_filename = 'saved.txt'

        upload_file = os.path.normpath(
            os.path.join(upload_path, upload_filename))
        size = 0
        with open(upload_file, 'wb') as out:
            while True:
                data = ufile.file.read(8192)
                if not data:
                    break
                out.write(data)
                size += len(data)
        out = '''
File received.
Filename: {}
Length: {}
Mime-type: {}
''' .format(ufile.filename, size, ufile.content_type, data)
        return out


if __name__ == '__main__':
    cherrypy.quickstart(App(), '/', config)

</form>

#cli.py#

Este exemplo requer o pacote Python requests, no entanto, o arquivo pode ser enviado para o servidor em Python simples.

#!/usr/bin/env python
# -*- coding: UTF-8 -*-

import requests

url = 'http://127.0.0.1:8080/upload'
files = {'ufile': open('file.txt', 'rb')}

r = requests.post(url, files=files)

print(r)
print(r.text)

upload.txt #

Hello! This file was uploaded to CherryPy.

Carregar do navegador #

  • Execute $ server.py
  • Abra webpage.html em seu navegador web.
  • Depois de selecionar o arquivo da sua unidade e enviá-lo, ele será salvo como saved.txt.

Upload da linha de comando #

  • Abra um console e execute $ server.py
  • Abra outro console e execute $ cli.py
    • Note: Test file upload.txt should be in the same directory with cli.py
  • O arquivo upload.txt deve ser carregado e salvo como saved.txt.
