Inicialização #

Um Set é uma coleção que não pode conter elementos duplicados. Ele modela a abstração do conjunto matemático.

Set tem sua implementação em várias classes como HashSet, TreeSet, LinkedHashSet.

Por exemplo:

HashSet:

Set<T> set = new HashSet<T>();

Aqui T pode ser String, Integer ou qualquer outro objeto. HashSet permite a pesquisa rápida de O(1), mas não classifica os dados adicionados a ele e perde a ordem de inserção dos itens.

Árvore:

Ele armazena dados de maneira ordenada, sacrificando alguma velocidade para operações básicas que levam O(lg(n)). Não mantém a ordem de inserção dos itens.

TreeSet<T> sortedSet = new TreeSet<T>();

LinkedHashSet:

É uma implementação de lista vinculada de HashSet Uma vez que pode iterar sobre os itens na ordem em que foram adicionados. A classificação não é fornecida para seu conteúdo. O(1) operações básicas são fornecidas, no entanto, há um custo maior do que HashSet na manutenção da lista vinculada de apoio.

LinkedHashSet<T> linkedhashset = new LinkedHashSet<T>();

Noções básicas de conjunto #

O que é um conjunto?

Um conjunto é uma estrutura de dados que contém um conjunto de elementos com uma propriedade importante de que não há dois elementos no conjunto iguais.

Tipos de Conjunto:

  1. HashSet: um conjunto apoiado por uma tabela de hash (na verdade, uma instância de HashMap)
  2. Linked HashSet: um conjunto apoiado por tabela de hash e lista vinculada, com ordem de iteração previsível
  3. TreeSet: Uma implementação NavigableSet baseada em um TreeMap.

Criando um conjunto

Set<Integer> set = new HashSet<Integer>(); // Creates an empty Set of Integers

Set<Integer> linkedHashSet = new LinkedHashSet<Integer>(); //Creates a empty Set of Integers, with predictable iteration order

Adicionando elementos a um conjunto

Elementos podem ser adicionados a um conjunto usando o método add()

 set.add(12); //  - Adds element 12 to the set
 set.add(13); //  - Adds element 13 to the set

Nosso conjunto depois de executar este método:

set = [12,13]

Excluir todos os elementos de um conjunto

set.clear();  //Removes all objects from the collection.

Após este conjunto será:

set = []

Verifique se um elemento faz parte do Conjunto

A existência de um elemento no conjunto pode ser verificada usando o método contains()

set.contains(0);  //Returns true if a specified object is an element within the set.

Saída: Falso

Verifique se um Set está vazio

O método isEmpty() pode ser usado para verificar se um Set está vazio.

set.isEmpty();  //Returns true if the set has no elements

Saída: Verdadeiro

Remover um elemento do Conjunto

 set.remove(0); // Removes first occurrence of a specified object from the collection

Verifique o tamanho do conjunto

set.size(); //Returns the number of elements in the collection

Saída: 0

Tipos e uso de conjuntos #

Geralmente, os conjuntos são um tipo de coleção que armazena valores exclusivos. A exclusividade é determinada pelos métodos equals() e hashCode().

A classificação é determinada pelo tipo de conjunto.

HashSet - Ordenação Aleatória #

Set<String> set = new HashSet<> ();
set.add("Banana");
set.add("Banana");
set.add("Apple");
set.add("Strawberry");

// Set Elements: ["Strawberry", "Banana", "Apple"]

LinkedHashSet - Pedido de inserção #

Set<String> set = new LinkedHashSet<> ();
set.add("Banana");
set.add("Banana");
set.add("Apple");
set.add("Strawberry");

// Set Elements: ["Banana", "Apple", "Strawberry"]

TreeSet - Por compareTo() ou Comparator #

Set<String> set = new TreeSet<> ();
set.add("Banana");
set.add("Banana");
set.add("Apple");
set.add("Strawberry");

// Set Elements: ["Apple", "Banana", "Strawberry"]

Set<String> set = new TreeSet<> ((string1, string2) -> string2.compareTo(string1));
set.add("Banana");
set.add("Banana");
set.add("Apple");
set.add("Strawberry");

// Set Elements: ["Strawberry", "Banana", "Apple"]

Declarando um HashSet com valores #

Você pode criar uma nova classe que herda de HashSet:

Set<String> h = new HashSet<String>() {{
    add("a");
    add("b");
}};

Solução de uma linha:

Set<String> h = new HashSet<String>(Arrays.asList("a", "b"));

Usando goiaba:

Sets.newHashSet("a", "b", "c")

Usando fluxos:

Set<String> set3 = Stream.of("a", "b", "c").collect(toSet());

Crie uma lista de um conjunto existente #

Usando uma nova lista

List<String> list = new ArrayList<String>(listOfElements);

Usando o método List.addAll()

    Set<String> set = new HashSet<String>();
    set.add("foo");
    set.add("boo");
    
    List<String> list = new ArrayList<String>();
    list.addAll(set);

Usando Java 8 Steam API

List<String> list = set.stream().collect(Collectors.toList());

Eliminando duplicatas usando Set #

Suponha que você tenha uma coleção elements e queira criar outra coleção contendo os mesmos elementos, mas com todas as duplicatas eliminadas:

Collection<Type> noDuplicates = new HashSet<Type>(elements);

Exemplo:

List<String> names = new ArrayList<>(
        Arrays.asList("John", "Marco", "Jenny", "Emily", "Jenny", "Emily", "John"));
Set<String> noDuplicates = new HashSet<>(names);
System.out.println("noDuplicates = " + noDuplicates);

Resultado:

noDuplicates = [Marco, Emily, John, Jenny]
