Conectando o código com a interface do usuário #

Como o iOS onde você usa @IBOutlet e @IBAction, aqui você também pode usá-los.

Digamos que temos um botão que, quando clicado, altera o texto do rótulo para outra coisa.

Para começar:

  1. Adicione um WKInterfaceLabel e um WKInterfaceLabel ao InterfaceController.

  2. Ctrl-Arraste de WKInterfaceLabel para InterfaceController.swift e insira os detalhes conforme mostrado na figura a seguir para adicionar uma propriedade de saída:

  1. Ctrl-Arraste de WKInterfaceButton para InterfaceController.swift e insira os detalhes conforme mostrado na figura a seguir para adicionar um método de ação:

  1. Preencha o método de ação:

Rápido #

outputLabel.setText("Button Tapped!")

Objetivo-C #

[[self outputLabel] setText:@"Button Tapped!"]
  1. Execute o programa e toque no botão para ver o resultado.

Criando um novo projeto watchOS #

Para desenvolver um aplicativo para watchOS, você deve começar com o Xcode. O Xcode é executado apenas no macOS. No momento da escrita, a versão mais recente é o Xcode 8.3.

Se você deseja iniciar um novo projeto do zero:

  1. Inicialize seu Mac e instale o Xcode da App Store se ainda não estiver instalado.

  2. Escolha criar um novo projeto.

  3. Nos modelos, escolha watchOS e depois “iOS App with WatchKit App”.

  1. Preencha os detalhes do seu projeto e escolha um local.

Se você já tem um projeto iOS e deseja adicionar um destino watchOS:

  1. Vá para Arquivo -> Novo -> Destino.

  1. Escolha o aplicativo WatchKit.

  1. Preencha os detalhes do seu destino e escolha um local.

Fazendo um simples “Olá, Mundo!” aplicativo #

Cada destino watchOS inclui um aplicativo e uma extensão. O aplicativo contém o material da interface do usuário e a extensão contém a lógica real (semelhante a exibições e modelos na arquitetura MVC no iOS).

Cada App WatchKit tem um arquivo Interface.storyboard no qual você projeta o aplicativo e um arquivo Assets.xcassets para colocar seus ativos.

Cada extensão WatchKit tem um arquivo InterfaceController.swift (na verdade uma subclasse WKInterfaceController) que é semelhante ao arquivo ViewController no iOS.

Para criar um aplicativo Hello World:

  1. Abra o Interface.storyboard.

  2. Localize o InterfaceController principal.

  1. Na biblioteca do painel direito, adicione um WKInterfaceLabel.

  1. Arraste o rótulo e defina seu texto no painel direito como “Hello, World!”.

  1. Selecione o esquema correto (de acordo com a próxima figura) e execute o projeto tocando no botão executar na barra superior, usando o menu Produto, pressionando Cmd-R ou tocando em executar na Touch Bar.

O simulador do Apple Watch eventualmente aparecerá com seu aplicativo em execução.
