Criando um novo projeto (C#/XAML) #

Após iniciar o Visual Studio 2015, vá para Arquivo → Novo → Projeto. Na caixa de diálogo New Project, navegue na árvore de templates até Visual C# → Windows → Universal e selecione Blank App (Universal Windows).

Em seguida, precisamos preencher o formulário para descrever o Aplicativo:

  1. Nome: é o nome do aplicativo que será exibido ao usuário. Defina-o como HelloWorld ou use um título personalizado.
  2. Local: indica onde o projeto será armazenado
  3. Nome da solução: é uma espécie de container de projetos que agrupa vários projetos relacionados a uma mesma aplicação (por exemplo, uma solução pode ser composta por um projeto de UI e um projeto modelo). Você pode colocar o mesmo Name do seu projeto inicial.

Conteúdo do projeto padrão #

Você obterá um projeto com os seguintes arquivos:

  1. Package.appxmanifest: descreve as propriedades do seu aplicativo. Ele contém algumas configurações de interface do usuário, como seu nome disply, seu logotipo, as rotações suportadas. E também contém configurações técnicas como o ponto de entrada do aplicativo (que é a classe App por padrão). Por fim, também lista as autorizações exigidas pelo seu aplicativo na guia Capabilities; por exemplo, se você quiser usar a webcam em seu aplicativo, você terá que verificar os recursos correspondentes.
  2. App.xaml / App.xaml.cs: a classe App é o ponto de entrada padrão do seu aplicativo. Os arquivos xaml podem conter recursos compartilhados em todo o aplicativo, como configuração de estilos ou instância de uma classe que você deseja compartilhar, como um localizador ViewModel. Os arquivos code-behind contêm todo o código de inicialização do aplicativo. Por padrão, ele implementa o método OnLaunched que é invocado pelo usuário final. Ele inicializa a janela e navega para a primeira página da aplicação (por padrão a classe MainPage).
  3. MainPage.xaml / MainPage.xaml.cs: esta é a página inicial do nosso aplicativo. Ele contém apenas uma Grade vazia que é um controle de layout.

Modifica a visualização #

Abra o MainPage.xaml e substitua o controle Grid por

<Grid Background="{ThemeResource ApplicationPageBackgroundThemeBrush}">
        <Grid.ColumnDefinitions>
            <ColumnDefinition Width="auto" />
            <ColumnDefinition Width="*" />
        </Grid.ColumnDefinitions>
        <Button Click="Button_Click">Say Hello !</Button>
        <TextBlock Grid.Column="1"
                   VerticalAlignment="Center"
                   x:Name="myText"
                   Text="Click the button." />
</Grid>

Isso criará uma grade com duas colunas. A primeira coluna tem uma largura definida como auto, o que significa que ela será definida automaticamente em função do tamanho de seus filhos. A segunda coluna será esticada para preencher o espaço restante na janela.

Esta grade contém dois elementos:

  • um Button que fica na primeira coluna. O evento click é vinculado ao método Button_Click no code-behind e seu texto de legenda é “Say Hello!”.
  • um TextBlock que fica na segunda coluna. Seu texto está definido como “Clique no botão.”. E nós definimos um nome para este controle com a ajuda do atributo x:Name. Isso é necessário para poder usar o controle no code-behind.

Em MainPage.xaml.cs, adicione o seguinte código:

private void Button_Click(object sender, RoutedEventArgs e)
{
    this.myText.Text = "Hello World!";
}

Este é o método que será chamado quando o usuário clicar (ou tocar) no botão. E ele atualizará o TextBlock e definirá seu texto como “Hello World!”.

Executando o aplicativo #

Para executar o aplicativo, você pode usar o menu Debug → Start Debugging ou o atalho F5. Por padrão, ele executará o aplicativo em sua máquina local.

Instalação ou configuração #

Se você não tiver o Visual Studio 2015, acesse https://www.visualstudio.com/en-us/downloads/download-visual-studio-vs.aspx e selecione Visual Studio Community. É uma versão gratuita que possui todos os recursos necessários para desenvolver o aplicativo Windows 10. Você pode instalar a Enterprise Edition se tiver a licença necessária. Em seguida, inicie o instalador.

Se você já possui o Visual Studio 2015 instalado, no Painel de Controle do Windows, vá em Programa e Recursos e na lista clique com o botão direito do mouse em Microsoft Visual Studi0 2015 (este nome pode mudar em função da versão instalada do Visual Studio 2015). Studio 2015) e selecione Alterar.

Na caixa de diálogo Instalação do Visual Studio, clique em Modify e em Windows and Web Development instale todas as Universal Windows App Development Tools. Ele instalará o SDK do Windows 10, um emulador móvel do Windows 10 (alimentado pelo Hyper-V) e o Visual Studio necessário para criar aplicativos do Windows 10.

Quando a instalação estiver concluída, você poderá criar um novo projeto do Windows 10.

Requisitos de sistema: Windows 10

Visual Studio 2015 (recurso de ferramentas universais de desenvolvimento de aplicativos do Windows)

Instruções para habilitar o recurso Universal Windows App Development Tools durante a instalação durante o visual studio 2015.

i.Durante a instalação, é necessário selecionar personalizar e verificar as Ferramentas Universais de Desenvolvimento de Aplicativos do Windows.

ii. Se você já tem o Visual Studio 2015 , agora pode adicionar essas ferramentas à sua instalação existente do Visual Studio. Você pode executar o instalador ou abrir Programas e Recursos no Painel de Controle, selecionar Visual Studio e clicar em Alterar. Em seguida, na configuração, clique em Modificar e selecione Ferramentas para Aplicativos Universais do Windows.

iii.Após a instalação das ferramentas, pode-se criar um projeto usando novos modelos de projeto (Windows->Universal) no Visual Studio 2015 em uma linguagem de sua escolha – C#, VB, C++ ou JavaScript.

Para começar com o desenvolvimento de aplicativos universais do Windows 10, você precisa ter o Visual Studio 2015 em um computador com Windows 10. Você pode comprar o Windows 10 aqui. Você pode baixar o Visual Studio 2015 aqui. Você pode usar as versões Community, Professional ou Enterprise do Visual Studio. Alguns recursos serão limitados na versão Express.

Você pode encontrar um guia detalhado sobre como instalar o Windows 10 aqui.

Ao instalar o Visual Studio, escolha Personalizar e, em ‘Windows and Web Development’, selecione ‘Universal Windows App Development Tools’ na seleção de recursos.

Quando terminar, você estará pronto para desenvolver aplicativos universais do Windows!
