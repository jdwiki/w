Configuração de um ‘Olá, mundo!’ exemplo #

Guice é uma biblioteca Java. Para usá-lo, você deve adicionar um arquivo JAR no caminho de classe do seu projeto Java.

Aulas de exemplo #

Abaixo estão várias classes para um “Olá, mundo!” exemplo.

Uma interface de um “serviço” hello:

public interface HelloWorldService {
    public void sayHello();
}

A implementação do serviço:

public class HelloWorldServiceImpl implements HelloWorldService {
    @Override
    public void sayHello() {
        System.out.println("Hello, world!");
    }
}

Um módulo de nomenclatura do Guice. É necessário instruir Guice que HelloWorldServiceImpl será injetado onde um serviço hello for necessário.

import com.google.inject.AbstractModule;

public class HelloWorldModule extends AbstractModule {
    protected void configure() {
        bind(HelloWorldService.class).to(HelloWorldServiceImpl.class);
    }
}

Uma classe principal onde ocorre a injeção real de um serviço hello:

import javax.inject.Inject;

import com.google.inject.Guice;
import com.google.inject.Injector;
import com.google.inject.Module;

public class Main {

    @Inject
    private HelloWorldService service;//hello service
    
    public static void main(String[] args) {
        
        Main main = new Main();
        
        Module module = new HelloWorldModule();
        Injector injector = Guice.createInjector(module);
        injector.injectMembers(main);//injects the implementation of the service
        
        main.testGuice();
    }

    public void testGuice()
    {
        service.sayHello();//usage of the service
    }
}

Executar com Gradle #

Para configurar e executar rapidamente com Gradle 2.2.+ e Java 8:

  1. Instale o gradle se ainda não estiver instalado

  2. Crie um diretório vazio e navegue até ele com um shell habilitado para gradle

  3. Crie um projeto Java vazio:

    gradle init --type java-library

  4. No build.gradle gerado automaticamente:

  • altere apply plugin: 'java' para apply plugin: 'application'

  • adicione a seguinte linha

    mainClassName = 'Main'

  • na seção de dependências, adicione uma dependência a uma versão do guice, por exemplo:

    dependencies {
  ...
  compile group: 'com.google.inject', name: 'guice', version: '4.1.0'
  ...
}

  1. Adicione as classes mostradas acima no pacote padrão em src/main/java, cada uma em seu próprio arquivo

  2. Corra e divirta-se

    ..> gradlew run
    :compileJava
    :processResources UP-TO-DATE
    :classes
    :run
    Hello, world!

    BUILD SUCCESSFUL

    Total time: 3.595 secs

Executar com Maven #

Para configurar e executar rapidamente com Maven 3+ e Java 8:

  1. Instale o maven se ainda não estiver instalado

  2. Crie um diretório vazio e navegue até ele com um shell habilitado para maven

  3. Crie um projeto Java vazio:

    mvn archetype:generate -DgroupId=com.example -DartifactId=guice -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false

  4. Mude para o subdiretório guice

  5. No pom.xml gerado automaticamente:

  • no elemento dependencies adicione uma dependência ao guice:

    com.google.inject guice 4.1.0

  • adicione o seguinte plugin ao seu projeto (permite um teste fácil)

    ..... org.codehaus.mojo exec-maven-plugin 1.5.0 Main

  1. Adicione as classes mostradas acima no pacote padrão em src/main/java, cada uma em seu próprio arquivo

  2. Corra e divirta-se
