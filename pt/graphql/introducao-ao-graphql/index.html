Linguagem de consulta GraphQL (GQL) #

Em vez de definir endpoints de URL para cada recurso de dados, no GraphQL você define um único endpoint que aceita consultas do GraphQL. Ao contrário das linguagens de consulta de banco de dados tradicionais, a GraphQL Query Language (GQL) é uma projeção de dados retornados por uma consulta de nível raiz. O esquema GraphQL definirá o modelo de dados e as consultas e mutações no nível da raiz. O GQL executa essas consultas e define quais dados retornar.

Definição do esquema #

Em GraphQL o Schema define a execução raiz consultas e mutações bem como os tipos para seus dados.

Tipo de objeto de esquema #

O tipo Person tem dois campos, um é um tipo Scalar padrão e o outro representa um relacionamento com uma lista de outros tipos Person que são ‘amigos’. Vincular outros tipos é o que torna o GraphQL tão poderoso. Agora em GraphQL Query Language (GQL), o cliente pode percorrer o gráfico de amigos sem nenhum código adicional ou consulta avançada.

type Person {
  id: ID
  name: String
  friends: [Person]
}

Consulta de esquema #

A consulta person procura uma única pessoa pelo seu id. Este é o ponto de entrada para seus dados para clientes que usam GQL.

type Query {   
  person(id: ID!): Person
}

Consultar os amigos do amigo do Nick #

Agora que temos um tipo de pessoa e uma consulta de raiz de pessoa, podemos procurar uma pessoa e quantos graus de separação quisermos da rede de amigos da pessoa.

query {
  person(id: 'nick'){
    id
    name
    friends{
      id
      name
      friends{
        id
        name
        friends{
          id
          name
        }
      }
    }
  }
}

Instalação e implementação do servidor #

GraphQL.js #

GraphQL.js é uma implementação de referência JavaScript para GraphQL. Você pode instalá-lo via npm:

  • Inicialize o npm em seu projeto se ainda não o fez: npm init
  • Instale o GraphQL.js do npm: npm install --save graphql

Servidor de exemplo #

var { graphql, buildSchema } = require('graphql');

// Construct a schema, using GraphQL schema language
var schema = buildSchema(`
  type Query {
    hello: String
  }
`);

// The root provides a resolver function for each API endpoint
var root = {
  hello: () => {
    return 'Hello world!';
  },
};

// Run the GraphQL query '{ hello }' and print out the response
graphql(schema, '{ hello }', root).then((response) => {
  console.log(response);
});
