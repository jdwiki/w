Carregar vários arquivos CSV do mesmo formato de uma pasta

  • Parâmetros
  • Etapas para carregar dados

Nesta página

  • Parâmetros
  • Etapas para carregar dados

Neste guia, você pode descobrir as etapas para carregar vários arquivos CSV/TXT de uma pasta para a tabela do banco de dados.

Parâmetros #

Parâmetro/Disponível Detalhes
SourceFolder É um parâmetro de projeto somente leitura disponível e configurável na implantação. Exemplo de parâmetro do projeto são Strings de Conexão, senhas, porta não, usuários etc
CompleteSourceFilePath É leitura escrita variável de usuário disponível apenas dentro do pacote como variáveis ​​locais em linguagens de programação

Etapas para carregar dados #

Para atingir este objetivo, o que precisamos é

  1. Foreach Loop Container: Iterar sobre um diretório para selecionar arquivos.
  2. Tarefa de fluxo de dados: para carregar dados do CSV (Flat File Source) para o database table (OLE DB Destination).
  3. Fonte de arquivo simples: Para arquivos de texto ou csv.
  4. Destino OLE DB: Para selecionar a tabela de destino com a qual queremos preencher.

Passos

  1. Primeiro, arraste e solte um Foreach Loop Container da seção container da caixa de ferramentas do SSIS.
  2. Agora clique duas vezes em Project.params no Solution Explorer e crie uma variável SourceFolder como string. No campo de valor, digite o caminho do qual você deseja selecionar os arquivos. Estamos criando este caminho como parâmetro do projeto para que possa ser configurado após a implantação.
  3. Crie uma variável de usuário clicando no ícone Variables à direita e crie a variável CompleteSourceFilePath do tipo string. Esta variável manterá o valor retornado do contêiner de loop Foreach.
  4. Agora clique duas vezes no Foreach loop Container e selecione Collection no lado esquerdo. No lado direito, selecione Foreach File Enumerator. Agora, para Expressão, clique nos três pontos à direita que abrirão um editor de propriedades, selecione Directory na seção de propriedades e selecione @[$Project:SourceFolder] como seu valor. Clique OK.
  5. Na janela Foreach Loop Editor para Arquivos, digite *.txt ou *.csv, qualquer que seja a extensão de arquivo necessária.
  6. No lado esquerdo do Foreach Loop Editor selecione Variable Mappings, à direita selecione User::CompleteSourceFilePath que atribuirá automaticamente o Índice 0. Clique em OK.
  7. Na caixa de ferramentas do SSIS, arraste e solte Data Flow Task da seção de favoritos dentro do Foreach Loop Container. Cada nome de arquivo retornado pelo Contêiner de loop Foreach na variável CompleteSourceFilePath será usado na tarefa de fluxo de dados. ![Tarefa de fluxo de dados dentro de cada]]4
  8. Agora clique duas vezes na tarefa Data Flow que nos levará para o Data flow. Arraste e solte uma Flat File Source da outra seção Source da caixa de ferramentas.
  9. Na parte inferior da tela, na seção Gerenciadores de conexões, clique com o botão direito do mouse e selecione New Flat File Connections. Clique no botão Procurar e selecione um dos arquivos que deseja processar, defina outras propriedades como Qualificador de texto (como aspas duplas). Clique OK.
  10. Clique na nova Conexão de arquivo simples criada na seção Gerenciador de conexões e vá para a janela Propriedades. Encontre a propriedade Expressões e clique nos três pontos à direita. Na seção Property selecione ConnectionString e na Expression selecione a variável @[User::CompleteSourceFilePath]. Clique OK.
  11. Selecione um OLE DB Destination (de acordo com o banco de dados) e configure-o para a tabela que deseja carregar.
  12. Clique com o botão direito do mouse no nome do pacote (solution explorer) e clique em Executar pacote para testar o pacote.
