Crie um arquivo CSV e grave do SQL Server nesse arquivo

  • Etapas para importar dados

Nesta página

  • Etapas para importar dados

O guia ajuda a entender como importar dados da tabela do SQL Server para um arquivo CSV/txt.

  1. Clique com o botão direito do mouse na Tarefa de Fluxo de Dados e selecione a propriedade. As propriedades DefaultBufferMaxRows e DefaultBufferSize podem ser alteradas para melhorar o desempenho do carregamento de dados.
  2. A tarefa de fluxo de dados múltiplo pode ser executada em paralelo para melhor desempenho.
  3. Cada Tarefa tem dois fluxos de sucesso e falha. É importante lidar com o fluxo de falhas para tornar o pacote mais robusto.
  4. Dentro da Tarefa de Fluxo de Dados, clique com o botão direito na seta azul e selecione Ativar Visualizador de Dados para verificar o fluxo de dados em tempo de execução.
  5. Se alguma coluna for excluída na origem ou no destino, verifique qual coluna será excluída. Dentro do Data Flow clique com o botão direito na seta azul e selecione Resolve References, na nova janela podemos ver as colunas Unmapped Output (esquerda) e Unmapped Input Columns (direita).

Etapas para importar dados #

Aqui está o que é necessário para completar este objetivo.

  1. Tarefa de Fluxo de Dados: Dentro desta tarefa iremos realizar a importação de dados.
  2. Origem OLE DB: Para selecionar a fonte de dados, ou seja, a tabela de banco de dados do servidor SQL.
  3. Destino do Arquivo Simples: Destino no qual queremos carregar os dados.

Passos

  1. Arraste e solte uma Tarefa de fluxo de dados da caixa de ferramentas SSIS da seção Favoritos.
  2. Clique duas vezes em Data Flow Task no Control Flow que nos levará ao Data Flow.
  3. Arraste e solte um OLE DB Source, por padrão aparecerá uma cruz nele, significa que não está configurado com uma conexão. Dê um duplo clique na tarefa OLE DB Source, clique em New.
  4. Na janela Configurar Gerenciador de Conexão OLE DB, clique em Novo. Agora, na janela Connection Manager, selecione o Nome do Servidor ao qual você deseja se conectar. Selecione Autenticação do Windows se o seu servidor estiver em sua máquina, caso contrário, use a Autenticação do SQL Server e insira o nome de usuário e a senha. Clique em Test Connection no botão esquerdo para verificar a validade da credencial inserida. Clique em OK e novamente em OK.
  5. No OLE DB Source Editor, selecione o nome da tabela ou visualização e clique em Preview para verificar os dados. Clique em Fechar e depois em OK.
  6. Arraste e solte uma tarefa Flat File Destination da caixa de ferramentas do SSIS na seção outros destinos. Conecte o OLE DB Source ao Flat File Destination.
  7. Dê um duplo clique em Flat File Destination, clique em New para abrir a janela Flat File Format. Selecione Delimitado se desejar especificar o separador, qualificador de texto, fim de linha etc. Clique em OK.
  8. No Editor do Gerenciador de Conexão de Arquivo Simples, clique no botão Procurar, selecione o caminho para o arquivo e digite o nome do arquivo, clique em Abrir. Mesmo que não tenhamos selecionado nenhum arquivo, acabamos de inserir o nome do arquivo, o arquivo será criado.
  9. Agora selecione a Página de Código, Qualificador de Texto, etc. Lembre-se de marcar a caixa de seleção nome da coluna na primeira linha de dados. No lado esquerdo, selecione Colunas aqui, você pode especificar o separador de dados como vírgula ou barra vertical (|). Clique OK.
  10. No Editor de Destino de Arquivo Simples substituir dados no arquivo está selecionado, atualize-o de acordo com o requisito. À esquerda, selecione Mapeamentos e verifique se as colunas estão mapeadas corretamente. Clique OK.
  11. No Solution Explorer clique com o botão direito do mouse no nome do pacote e execute-o para verificar.
