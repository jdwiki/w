Como atualizar pacotes com o gerenciador de pacotes apt #

A **ferramenta Advanced **Package T, apropriadamente chamada de gerenciador de pacotes ‘apt’ pode lidar com a instalação e remoção de software no Debian, Slackware e outras distribuições Linux. Abaixo estão alguns exemplos simples de uso:

atualizar Esta opção recupera e verifica os arquivos Packages.gz, para que as informações sobre pacotes novos e atualizados estejam disponíveis. Para isso, digite o seguinte comando:

sudo apt-get update

melhoria Esta opção é usada para instalar as versões mais recentes de todos os pacotes atualmente instalados no sistema. Os pacotes atualmente instalados com novas versões disponíveis são recuperados e atualizados; sob nenhuma circunstância os pacotes instalados atualmente são removidos, ou os pacotes ainda não instalados são recuperados e instalados. Para atualizar, digite o seguinte comando:

sudo apt-get upgrade

dist-upgrade Além de desempenhar a função de atualização, o dist-upgrade também trata de forma inteligente as dependências em mudança com novas versões de pacotes. Ele tentará atualizar os pacotes mais importantes em detrimento dos menos importantes, se necessário. Para isso, digite o seguinte comando:

sudo apt-get dist-upgrade

Como atualizar pacotes com o gerenciador de pacotes pacman #

Para atualizar um programa específico:

 sudo pacman -S <programName>

Para atualizar todo o sistema:

sudo pacman -Syu

Como instalar um pacote com o gerenciador de pacotes pacman #

Para pesquisar pacotes no banco de dados, pesquisando tanto nos nomes quanto nas descrições dos pacotes:

 pacman -Ss string1 string2 ...

Para instalar um único pacote ou lista de pacotes (incluindo dependências), emita o seguinte comando:

sudo pacman -S package_name1 package_name2 ...

Como atualizar pacotes com o yum #

Yellowdog Updater, Modified, um dos últimos vestígios restantes do Yellow Dog Linux, é o gerenciador de pacotes usado pelos sistemas Red Hat, Fedora e CentOS e seus derivados. Ele pode lidar com a instalação e remoção de software empacotado como rpms para essas distribuições Linux. Abaixo estão alguns exemplos simples de uso:

procurar Este comando tentará localizar pacotes de software nos repositórios de software configurados que correspondam aos critérios de pesquisa fornecidos e exibirá o nome/versão/local do repositório das correspondências encontradas. Para usá-lo, digite o seguinte comando:

yum search <queryString>

instalar Este comando tentará localizar e instalar o software nomeado a partir dos repositórios de software configurados, localizando e instalando recursivamente qualquer software de pré-requisito necessário. Para usá-lo, digite o seguinte comando:

sudo yum install <packageName>

atualizar Esta opção é usada para instalar as versões mais recentes de todos os pacotes atualmente instalados no sistema. Os pacotes atualmente instalados com novas versões disponíveis são recuperados e atualizados; novos pré-requisitos também são recuperados e instalados conforme necessário, e os pacotes substituídos ou obsoletos são removidos. Para atualizar, digite o seguinte comando:

sudo yum update

Ao contrário do apt, a maioria dos comandos yum também verificará automaticamente se há atualizações nos metadados do repositório se uma verificação não tiver sido feita recentemente (ou se for forçada a fazê-lo) e recuperará e verificará metadados atualizados para que as informações sobre pacotes novos e atualizados estão disponíveis antes que a operação solicitada seja executada.
