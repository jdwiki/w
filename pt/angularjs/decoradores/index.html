Sintaxe #

  • decorador(nome, decorador);

Decorador é uma função que permite um serviço, fábrica, diretiva ou filtro a ser modificado antes de seu uso. O decorador está acostumado a substituir ou modificar o comportamento do serviço. O valor de retorno de a função de decorador pode ser o serviço original ou um novo serviço que substitui ou envolve e delega ao serviço original.

Qualquer decoração deve ser feita na fase config da aplicação angular injetando $provide e usando sua função $provide.decorator.

A função decorator tem um objeto $delegate injetado para fornecer acesso ao serviço que corresponde ao seletor no decorador. este $delegate será o serviço que você está decorando. O valor de retorno de a função prestada ao decorador terá lugar do serviço, diretiva, ou filtro sendo decorado.

Deve-se considerar o uso do decorador apenas se qualquer outra abordagem não for apropriada ou se revelar muito tediosa. Se um aplicativo grande estiver usando o mesmo serviço e uma parte estiver alterando o comportamento do serviço, é fácil criar confusão e/ou bugs no processo.

O caso de uso típico seria quando você tem uma dependência de terceiros que não pode ser atualizada, mas precisa que funcione de maneira um pouco diferente ou estenda-a.

Serviço de decoração, fábrica #

Abaixo está um exemplo de decorador de serviço, substituindo a data null retornada pelo serviço.

angular.module('app', [])
  .config(function($provide) {
    $provide.decorator('myService', function($delegate) {
      $delegate.getDate = function() { // override with actual date object
        return new Date();
      };
      return $delegate;
    });
  })
  .service('myService', function() {
    this.getDate = function() {
      return null; // w/o decoration we'll be returning null
    };
  })
  .controller('myController', function(myService) {
    var vm = this;
    vm.date = myService.getDate();
  });

<body ng-controller="myController as vm">
  <div ng-bind="vm.date | date:'fullDate'"></div>
</body>

Diretiva de decoração #

As diretivas podem ser decoradas como serviços e podemos modificar ou substituir qualquer uma de suas funcionalidades. Observe que a própria diretiva é acessada na posição 0 no array $delegate e o parâmetro name no decorador deve incluir o sufixo Directive (diferencia maiúsculas de minúsculas).

Então, se a diretiva é chamada myDate, ela pode ser acessada usando myDateDirective usando $delegate[0].

Abaixo está um exemplo simples onde a diretiva mostra a hora atual. Vamos decorá-lo para atualizar a hora atual em intervalos de um segundo. Sem decoração, sempre mostrará ao mesmo tempo.

<body>
  <my-date></my-date>
</body>

angular.module('app', [])
  .config(function($provide) {
    $provide.decorator('myDateDirective', function($delegate, $interval) {
      var directive = $delegate[0]; // access directive

      directive.compile = function() { // modify compile fn
        return function(scope) {
          directive.link.apply(this, arguments);
          $interval(function() {
            scope.date = new Date(); // update date every second
          }, 1000);
        };
      };
      
      return $delegate;
    });
  })
  .directive('myDate', function() {
    return {
      restrict: 'E',
      template: '<span>Current time is {{ date | date:\'MM:ss\' }}</span>',
      link: function(scope) {
        scope.date = new Date(); // get current date
      }
    };
  });

Decore o filtro #

Ao decorar filtros, o parâmetro de nome deve incluir o sufixo Filter (diferencia maiúsculas de minúsculas). Se o filtro for chamado repeat, o parâmetro decorador será repeatFilter. Abaixo, vamos decorar um filtro personalizado que repete qualquer string n vezes para que o resultado seja revertido. Você também pode decorar os filtros internos do angular da mesma maneira, embora não seja recomendado, pois pode afetar a funcionalidade do framework.

<body>
  <div ng-bind="'i can haz cheeseburger ' | repeat:2"></div>
</body>

angular.module('app', [])
  .config(function($provide) {
    $provide.decorator('repeatFilter', function($delegate) {
      return function reverse(input, count) {
        // reverse repeated string
        return ($delegate(input, count)).split('').reverse().join(''); 
      };
    });
  })
  .filter('repeat', function() {
    return function(input, count) {
      // repeat string n times
      return (input || '').repeat(count || 1);
    };
  });
