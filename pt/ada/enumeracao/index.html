Sintaxe #

  • função Enumeration’Image (Argumento: Enumeration’Base) return String;
  • função Enumeração’Img return String; – GNAT
  • função Enumeration’Val (Argumento: Universal_Integer) return Enumeration’Base;
  • função Enumeration’Pos (Argumento: Enumeration’Base) return Universal_Integer;
  • função Enumeration’Enum_Rep (Argumento: Enumeration’Base) return Universal_Integer;
  • função Literal‘Enum_Rep return Universal_Integer; – GNAT
  • função Literal‘Address return System.Address;
  • para uso de Enumeração (Literal_1 => Universal_Integer, Literal_n => Universal_Integer);
  • (Literal em Enumeration) return Boolean;

Iterando literais #

Um literal dentro de uma enumeração é um tipo discreto, então podemos usar o atributo Image para descobrir qual literal ele é na forma de texto. Observe que isso imprime a mesma palavra que no código (mas em maiúsculas).

with Ada.Text_IO; use Ada.Text_IO;

procedure Main is
   type Fruit is (Banana, Pear, Orange, Melon);
begin
   for I in Fruit loop
      Put (Fruit'Image (I));
      New_Line;
   end loop;
end;

Resultado #

BANANA
PEAR
ORANGE
MELON

Usando o pacote Enumeration_IO #

Em vez do atributo Image e Ada.Text_IO.Put em literais de enumeração, só podemos usar o pacote genérico Ada.Text_IO.Enumeration_IO para imprimir o literais.

with Ada.Text_IO; use Ada.Text_IO;

procedure Main is
   type Fruit is (Banana, Pear, Orange, Melon);
   package Fruit_IO is new Enumeration_IO (Fruit); use Fruit_IO;
begin
   for I in Fruit loop
      Put (I);
      New_Line;
   end loop;
end;

Resultado #

BANANA
PEAR
ORANGE
MELON

O primeiro caractere maiúsculo descansa literais minúsculos #

O atributo Image coloca em maiúscula todos os caracteres de literais de enumeração. A função Case_Rule_For_Names aplica maiúsculas para o primeiro caractere e torna o restante em minúsculas.

with Ada.Text_IO; use Ada.Text_IO;
with Ada.Strings.Maps.Constants; use Ada.Strings.Maps.Constants;
with Ada.Strings.Fixed; use Ada.Strings.Fixed;

procedure Main is
   type Fruit is (Banana, Pear, Orange, Melon);
   function Case_Rule_For_Names (Item : String) return String is
   begin
      return Translate (Item (Item'First .. Item'First), Upper_Case_Map) & Translate (Item (Item'First + 1 .. Item'Last), Lower_Case_Map);
   end;
begin
   for I in Fruit loop
      Put (Case_Rule_For_Names (Fruit'Image (I)));
      New_Line;
   end loop;
end;

Resultado #

Banana
Pear
Orange
Melon

Maiúsculas e minúsculas, usando Enumeration_IO, para um subintervalo #

Combinando [alteração de maiúsculas e minúsculas de caractere][.enumC] com [Enumeration_IO][.enumIO] e usando um buffer de texto para a imagem. O primeiro caractere é manipulado no lugar.

with Ada.Text_IO; use Ada.Text_IO;
with Ada.Characters.Handling; use Ada.Characters.Handling;

procedure Main is
    type Fruit is (Banana, Pear, Orange, Melon);
    package Fruit_IO is new Enumeration_IO (Fruit);
    Buffer : String (1 .. Fruit'Width);
begin
   for I in Fruit range Pear .. Fruit'Last loop
       Fruit_IO.Put (To => Buffer,
                     Item => I,
                     Set => Lower_Case);
       Buffer (Buffer'First) := To_Upper (Buffer (Buffer'First));
       Put_Line (Buffer);
   end loop;
end;

Resultado #

Pear  
Orange
Melon
