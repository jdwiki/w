Os atributos de subtipo 'Image e 'Value receberão, respectivamente, um valor escalar e uma string e retornarão, respectivamente, uma string e um valor escalar. O resultado de 'Image pode ser inserido em 'Value para obter o valor original. A recíproca também é verdadeira.

O atributo __Scalar_Object__'Image pode ser usado diretamente em objetos (desde Ada 2012-TC-1).

Sintaxe #

  • função Scalar’Image (Argumento: Scalar’Base) return String;
  • função Discrete’Image (Argumento: Discrete’Base) return String;
  • função Integer’Image (Argumento: Integer’Base) return String;
  • função Enumeration’Image (Argumento: Enumeration’Base) return String;
  • função Real’Image (Argumento: Real’Base) return String;
  • função Numeric’Image (Argumento: Numeric’Base) return String;
  • função Scalar’Value (Argumento: String) return Scalar’Base;
  • função Discrete’Value (Argumento: String) return Discrete’Base;
  • função Integer’Value (Argumento: String) return Integer’Base;
  • função Enumeration’Value (Argumento: String) return Enumeration’Base;
  • função Real’Value (Argumento: String) return Real’Base;
  • função Scalar_Object‘Imagem return String;

Note que 'Image pode gerar resultados definidos pela implementação (RM 3.5), ou seja, quando alguns caracteres gráficos necessários para o resultado String não são definidos em Character. Considere os repertórios maiores de 'Wide_Image e 'Wide_Wide_Image.

A permissão para usar o atributo __Scalar_Object__'Image diretamente em um objeto foi adicionada na Ada 2012-TC-1 (abril de 2016).

Imprima float usando o atributo Image #

with Ada.Text_IO;

procedure Main is
   type Some_Float digits 8 range 0.0 .. 10.0;
   X : Some_Float := 2.71;
begin
   Ada.Text_IO.Put_Line (X'Image);
end Main;

Resultado #

2.71000E+00

Imprima um inteiro usando o atributo Image #

with Ada.Text_IO;

procedure Main is
   type Some_Integer is range -42 .. 42;
   X : Some_Integer := 17;
begin
   Ada.Text_IO.Put_Line (X'Image);
end Main;

Resultado #

17

Imprima a enumeração usando o atributo Image #

with Ada.Text_IO;

procedure Main is
   type Fruit is (Banana, Orange, Pear);
   X : Fruit := Orange;
begin
   Ada.Text_IO.Put_Line (X'Image);
   Ada.Text_IO.Put_Line (Pear'Image);
end Main;

Resultado #

ORANGE
PEAR

Imprima a enumeração usando o atributo Image #

with Ada.Text_IO;

procedure Main is
   type Fruit is (Banana, Orange, Pear);
   X : Fruit := Orange;
begin
   Ada.Text_IO.Put_Line (Fruit'Image (X));
end Main;

Resultado #

ORANGE

Ases Reverso #

with Ada.Text_IO;

procedure Image_And_Value is
   type Fruit is (Banana, Orange, Pear);
   X  : Fruit := Orange;
begin
   Ada.Text_IO.Put_Line (Boolean'Image
      (Fruit'Value (Fruit'Image (X)) = X
          and
       Fruit'Image (Fruit'Value ("ORANGE")) = "ORANGE"));
end Image_And_Value;

Resultado #

TRUE
