Encadeamento #

Isso funciona, mas há algumas coisas que precisamos fazer para que nosso plugin sobreviva no mundo real. Um dos recursos do jQuery é o encadeamento, quando você vincula cinco ou seis ações em um seletor. Isso é feito fazendo com que todos os métodos do objeto jQuery retornem o objeto jQuery original novamente (há algumas exceções: .width() chamado sem parâmetros retorna a largura do elemento selecionado e não pode ser encadeado). Tornar nosso método de plug-in encadeável leva uma linha de código:

$.fn.greenify = function() {
    this.css( "color", "green" );
    // return the reference for chaining
    return this;
}
 
$( "a" ).greenify().addClass( "greenified" );

Instalação ou configuração #

Plugins jQuery são normalmente instalados via NPM ou Yarn (se hospedados lá), ou referenciando um arquivo de script externo contendo o plugin, seja de um diretório relativo ou de um CDN.

<script type="text/javascript" src="/path/to/plugin.jquery.js"></script>

Plugin básico que muda a cor do texto para verde. #

// plugin initialization
$.fn.greenify = function() {
    // within the function you can use any of the jQuery methods
    // and `this` context refers to jQuery object
    this.css( "color", "green" );
};
 
// apply plugin
$( "a" ).greenify();

Opções de suporte com padrões #

Você pode tornar seu plugin personalizável aceitando opções.

$.fn.colourize = function(options) {

    // This is one method to support default options
    var style = $.extend({
        color: "green",
        backgroundColor: "white"
    }, options);

    // Set the colours on the current selection based on the option parameters
    return this.css({
        color: style.color,
        backgroundColor: style.backgroundColor
    });
};

Exemplo de uso:

$("button").colourize({
    color: "orange"
});

O valor padrão para a opção de cor “verde” é substituído por $.extend() para ser “laranja”.

Estrutura típica de plugin jQuery #

Embora escrever plugins jQuery seja simples, queremos incluir nossos plugins em um escopo local. Isso evitará conflitos de namespace, além de poluir o namespace global, além de garantir que o jQuery seja carregado antes que nosso plugin o estenda.

// Encapsulate our plugins in a local scope
(function($) {

    // Plugin definition
    $.fn.colourize = function() {

        // Plugin code

    };

// Pass the jQuery object into our local scope
}(jQuery));

O wrapper de escopo local pode ser omitido em outros exemplos para mantê-los simples e concisos.

Usando o método each() #

Seu objeto jQuery típico conterá referências a qualquer número de elementos DOM, e é por isso que os objetos jQuery são frequentemente chamados de coleções. Se você quiser fazer qualquer manipulação com elementos específicos (por exemplo, obter um atributo de dados, calcular posições específicas), então você precisa usar .each() para percorrer os elementos.

$.fn.myNewPlugin = function() { 
    return this.each(function() {
        // Do something to each element here.
    });
 
    // return the reference for chaining
    return this;
};
