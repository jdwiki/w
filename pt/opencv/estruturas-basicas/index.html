Este tópico cobre estruturas básicas no OpenCV. As estruturas que serão discutidas neste tópico são DataType, Point, Vec, Size, Rect, Scalar, Ptr e Mat.

Tipo de dados #

Os tipos primitivos no OpenCV são unsigned char, bool, signed char, unsigned short, signed short, int, float, double. Qualquer tipo de dado no OpenCV é definido como CV_<bit-depth>{U|S|F}C(<number_of_channels>) onde U: unsigned, S:signed e F:floating point.

Por exemplo, CV_32FC2 é uma estrutura de 32 bits, ponto flutuante e 2 canais. e a definição de tipos básicos de um canal são

#define CV_8U   0
#define CV_8S   1
#define CV_16U  2
#define CV_16S  3
#define CV_32S  4
#define CV_32F  5
#define CV_64F  6
#define CV_USRTYPE1 7

Os outros tipos com canal superior são produzidos a partir destes pela seguinte definição:

#define CV_MAKETYPE(depth,cn) (CV_MAT_DEPTH(depth) + (((cn)-1) << CV_CN_SHIFT))

Usando esses tipos de dados, outras estruturas podem ser criadas.

Tapete #

Mat (Matrix) é um array n-dimensional que pode ser usado para armazenar vários tipos de dados, como imagens RGB, HSV ou em tons de cinza, vetores com valores reais ou complexos, outras matrizes etc.

Um Mat contém as seguintes informações: width, height, type, channels, data, flags, datastart, dataend e assim por diante.

Possui vários métodos, alguns deles são: create, copyTo, convertTo, isContinious etc.

Há muitas maneiras de criar uma variável Mat. Considere que eu quero criar uma matriz com 100 linhas, 200 colunas, digite CV_32FC3:

int R = 100, C = 200;
Mat m1; m1.create(R,C,CV_32FC3);//creates empty matrix
Mat m2(cv::Size(R, C), CV_32FC3); // creates a matrix with R rows, C columns with data type T where R and C are integers, 
Mat m3(R,C,CV_32FC3); // same as m2

Inicializando o tapete:

Mat m1 = Mat::zeros(R,C,CV_32FC3); // This initialized to zeros, you can use one, eye or cv::randn etc.
Mat m2(R,C,CV_32FC3);
for (int i = 0; i < m2.rows; i++)
    for (int j = 0; j < m2.cols; j++)
        for (int k = 0; k < m2.channels(); k++)
            m2.at<Vec3f>(i,j)[k] = 0;
//Note that, because m2 is a float type and has 3 channels, we used Vec3f, for more info see Vec 

Mat m3(3, out, CV_32FC1, cv::Scalar(0));

Vec #

Vec (Vector) é uma classe de modelo para valores numéricos. Ao contrário dos vetores c++, geralmente armazena vetores curtos (apenas alguns elementos).

A maneira como um Vec é definido é a seguinte:

 typedef Vec<type, channels> Vec< channels>< one char for the type>;

onde type é um de uchar, short, int, float, double e os caracteres para cada tipo são b, s, i, f, d, respectivamente.

Por exemplo, Vec3b indica um vetor de caracteres sem sinal de 3 canais. Cada índice em uma imagem RGB está neste formato.

Mat rgb = imread('path/to/file', CV_LOAD_IMAGE_COLOR);  
cout << rgb.at<Vec3b>(0,0); //The output is [r g b] values as ASCII character.
// To print integer values of RED value
cout << (int)rgb.at<Vec3b>(0,0)[0]; //The output will be an integer in [0, 255].

Na classe Vec os seguintes operadores são definidos

v1 = v2 + v3
v1 = v2 - v3
v1 = v2 * scale
v1 = scale * v2
v1 = -v2
v1 += v2 and other augmenting operations
v1 == v2, v1 != v2
