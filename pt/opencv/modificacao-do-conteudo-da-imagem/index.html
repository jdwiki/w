Modificação da cor da imagem no OpenCV - kmeans(). Para digitalizar todos os pixels de uma imagem e substituir os valores de pixel por cores genéricas. #

<código> #include opencv2/opencv.hpp> #include vetor> usando namespace std; usando namespace cv; int main() { Mat3b img = imread(“test.jpg”);

imshow("Original", img);

// Cluster

int K = 8;
int n = img.rows * img.cols;
Mat data = img.reshape(1, n);
data.convertTo(data, CV_32F);

Mat labels;
Mat1f colors;
kmeans(data, K, labels, cv::TermCriteria(), 1, cv::KMEANS_PP_CENTERS, colors);

for (int i = 0; i < n; ++i)
{
    data.at<float>(i, 0) = colors(labels.at<int>(i), 0);
    data.at<float>(i, 1) = colors(labels.at<int>(i), 1);
    data.at<float>(i, 2) = colors(labels.at<int>(i), 2);
}

Mat reduced = data.reshape(3, img.rows);
reduced.convertTo(reduced, CV_8U);



imshow("Reduced", reduced);
waitKey(0);

return 0;

}

Modificação de pixel por pixel de imagens #

No OpenCV, as imagens podem ser RGB/BGR, HSV, escala de cinza, preto-branco e assim por diante. É crucial conhecer o tipo de dados antes de lidar com imagens.

Os tipos de dados de imagem são principalmente CV_8UC3 (Matriz de uchar com 3 canais) e CV_8U (Matriz de uchar com 1 canal), no entanto, a conversão para outros tipos como CV_32FC3, CV_64F também é possível. (consulte [tipos de dados][1])

Considere que a imagem é uma imagem RGB que é lida pela função imread.

Mat rgb = imread('path/to/rgb/image', CV_LOAD_IMAGE_COLOR);
//to set RED pixel value of (i,j)th to X,
rgb.at<Vec3b>(i,j)[0] = X;

Da mesma forma, se a imagem estiver em escala de cinza,

gray.at<uchar>(i,j) = X;

Observe que, no OpenCV, as imagens em preto e branco são armazenadas como tipo CV_8U com os valores 0 e 255. Portanto, alterar imagens em preto e branco é o mesmo que imagens em cinza.

[1]: https://www.wikiod.com/pt/opencv/estruturas-basicas#Tipo de dados

Defina a imagem inteira para uma cor sólida #

Dado um cv::Mat img não vazio de algum tamanho, podemos preenchê-lo com uma cor sólida de várias maneiras:

img = cv::Scalar(blueVal,greenVal,redVal);

ou, o mais geral, suporte de máscara, cv::Mat::setTo():

img.setTo(cv::Scalar(blueVal,greenVal,redVal));

Se você estiver usando a API OpenCV C mais antiga com IplImage* img:

Usar:

cvSet(img, CV_RGB(redVal,greenVal,blueVal));
