Carregar e exibir uma imagem com OpenCV #

Com este exemplo, veremos como carregar uma imagem colorida do disco e exibi-la usando as funções internas do OpenCV. Podemos usar as ligações C/C++, Python ou Java para fazer isso.

Em C++:

#include <opencv2/core.hpp>
#include <opencv2/highgui.hpp>

#include <iostream>

using namespace cv;

int main(int argc, char** argv) {
    // We'll start by loading an image from the drive
    Mat image = imread("image.jpg", CV_LOAD_IMAGE_COLOR);

    // We check that our image has been correctly loaded
    if(image.empty()) {
        std::cout << "Error: the image has been incorrectly loaded." << std::endl;
        return 0;
    }

    // Then we create a window to display our image
    namedWindow("My first OpenCV window");

    // Finally, we display our image and ask the program to wait for a key to be pressed
    imshow("My first OpenCV window", image);
    waitKey(0);

    return 0;
}

Em Python:

import sys
import cv2

# We load the image from disk
img = cv2.imread("image.jpg", cv2.CV_LOAD_IMAGE_COLOR)

# We check that our image has been correctly loaded
if img.size == 0
    sys.exit("Error: the image has not been correctly loaded.")

# We create a window to display our image
cv2.namedwindow("My first OpenCV window")

# We display our image and ask the program to wait until a key is pressed
cv2.imshow("My first OpenCV window", img)
cv2.waitKey(0)

# We close the window
cv2.destroyAllWindows()

Em Java:

import org.opencv.core.Core;
import org.opencv.core.Mat;
import org.opencv.core.CvType;
import org.opencv.highgui.Highgui;
public class Sample{
public static void main (String[] args) {

    //Load native opencv library
    System.loadLibrary(Core.NATIVE_LIBRARY_NAME);

    //Read image from file first param:file location ,second param:color space
    Mat img = imread("image.jpg",CV_LOAD_IMAGE_COLOR);

    //If the image is successfully read.
    if (img.size() == 0) {
        System.exit(1);
    }
}

HighGui não tem janelas nomeadas ou equivalentes imshow em opencv java. Use swing ou swt para exibir a imagem.

Compile e instale o OpenCV a partir da fonte #

Este é um guia passo a passo para instalar o OpenCV 3 em um sistema Linux baseado em Debian a partir da fonte. As etapas devem permanecer as mesmas para outras distribuições, apenas substitua os comandos relevantes do gerenciador de pacotes ao instalar os pacotes para a compilação.

Nota: Se você não sentir vontade de perder tempo construindo coisas ou não gostar do terminal, provavelmente poderá instalar o OpenCV a partir da GUI do gerenciador de pacotes Synaptic. No entanto, essas bibliotecas geralmente estão desatualizadas.

Prepare-se para a compilação #

Emita os seguintes comandos em seu terminal para instalar os pacotes necessários:

sudo apt-get update
sudo apt-get install build-essential
sudo apt-get install cmake git libgtk2.0-dev pkg-config \
                     libavcodec-dev libavformat-dev libswscale-dev

Os seguintes pacotes são opcionais:

sudo apt-get install python-dev python-numpy libtbb2 libtbb-dev libjpeg-dev \
                     libpng-devlibtiff-dev libjasper-dev libdc1394-22-dev

Emita o seguinte comando para obter o código-fonte do OpenCV e preparar a compilação:

mkdir ~/src
cd ~/src
git clone https://github.com/opencv/opencv.git
cd opencv
mkdir build && cd build

Construir e Instalar #

Incluímos os exemplos na compilação, mas fique à vontade para deixá-los de fora. Sinta-se também à vontade para definir outros sinalizadores e personalizar sua compilação como achar melhor.

cmake -D CMAKE_BUILD_TYPE=RELEASE \
      -D CMAKE_INSTALL_PREFIX=/usr/local \
      -D INSTALL_PYTHON_EXAMPLES=ON \
      -D INSTALL_C_EXAMPLES=ON ..

Se o CMake não relatou nenhum erro ou biblioteca ausente, continue com a compilação.

make -j$(nproc)

Se nenhum erro foi produzido, podemos continuar instalando o OpenCV no sistema:

sudo make install

Agora o OpenCV deve estar disponível para o seu sistema. Você pode usar as seguintes linhas para saber onde o OpenCV foi instalado e quais bibliotecas foram instaladas:

pkg-config --cflags opencv  # get the include path (-I)
pkg-config --libs opencv    # get the libraries path (-L) and the libraries (-l)

Instalação de teste #

Primeiro construímos os exemplos C++:

cd ~/src/opencv/samples
cmake .
make

Se nenhum erro foi produzido, execute qualquer amostra, por exemplo.

./cpp/cpp-example-edge

Se o exemplo for executado, as bibliotecas C++ estão instaladas corretamente.

Em seguida, teste as ligações do Python:

python
>> import cv2
>> print cv2.__version__

Se esses comandos importarem o OpenCV e imprimirem a versão correta sem reclamar, as ligações do Python serão instaladas corretamente.

Parabéns, você acabou de construir e instalar o OpenCV. Boa programação!

Para Mac consulte aqui http://stackoverflow.com/questions/19671827/opencv-installation-on-mac-os-x

Introdução ao OpenCV 3.1 no Windows #

Instalamos o OpenCV 3.1.0 no Windows e começamos. Existem duas maneiras de instalar o OpenCV no Windows. Uma delas é baixar o instalador e executá-lo. Outro é construir a partir da fonte.

Esta é a maneira mais fácil de instalar o OpenCV e começar. O OpenCV fornece binários pré-compilados para instalar no Windows aqui. Após terminar o download, extraia-o e instale no caminho escolhido.

ProDica: certifique-se de que seu caminho OpenCV não inclua espaços. Então, é melhor se você apenas instalá-lo no diretório raiz C:\ ou D:\

O problema com o método acima é que você não pode usar os módulos opencv_contrib. Além disso, ele não vem com todas as ferramentas e bibliotecas de terceiros. Então, se você quiser usar tudo isso, basta seguir adiante.

Vou explicar o mínimo mínimo para instalar o OpenCV a partir da fonte. Para um mais avançado, consulte aqui.

  • Instale CMake.
  • Clone a fonte OpenCV de https://github.com/Itseez/opencv.git em algum diretório que não tenha espaços. Vamos nos referir a ele como “OpenCVdir”.
  • Agora, abra a GUI do CMake e adicione seu diretório de origem (OpenCVdir) ao menu Sources e construa o diretório no menu de compilação. Dica: se não houver um diretório de compilação, crie um em sua pasta opencv.
  • Clique em Configurar e selecione a versão do compilador do Visual Studio. Eu tinha o Visual Studio 2013 Professional 32 bits, então escolhi o compilador Visual Studio 12.

Dica: você pode baixar o Visual Studio 2013 Professional aqui. Ele vem com 30 dias de teste + 90 dias de trilha estendida após o login.

  • Pressione Concluir e o CMake carregará todos os pacotes automaticamente. Você pode adicionar ou remover pacotes. Pressione Configurar novamente.
  • Se você quiser construir com módulos opencv_contrib extras, você precisa baixá-los aqui. Em seguida, extraia-os e adicione o diretório opencv_contrib/modules ao seu CMake como mostrado abaixo.
  • Agora pressione Configurar novamente e depois pressione Gerar.
  • Feche o CMake. Vá para a pasta your_opencv\build e abra o arquivo chamado ‘OpenCV.sln’. - Abrirá o Visual Studio. Agora, execute-o no modo Debug e no modo Release .
  • Agora, no gerenciador de soluções no canto superior direito do Visual Studio, selecione INSTALAR projetar e construí-lo.

Viva!! Aproveite o seu OpenCV.

Adicionando o diretório de inclusão do OpenCV à variável PATH das variáveis ​​de ambiente:

  • Vá para Propriedades do sistema e clique em Configurações avançadas do sistema.

  • Agora, clique em Variáveis ​​de Ambiente » Caminho » Editar.

  • Aqui, adicione a pasta bin localizada em seu OpenCVdir/build/install/x86/vc**/bin a esta variável. Tenha cuidado para não substituir os valores de Caminho existentes.

  • Depois disso, você precisa reiniciar seu sistema para que as variáveis ​​de ambiente mudem e agora você está pronto para começar.

O que e por que OPENCV? #

OpenCV (Open Source Computer Vision Library) é uma biblioteca de software de visão computacional e aprendizado de máquina de código aberto. Foi construído para diversas finalidades como aprendizado de máquina, visão computacional, algoritmo, operações matemáticas, captura de vídeo, processamento de imagens etc. , android, ios). Também possui wrapper em várias linguagens de programação de renome. Sob o contrato de licença, ele tem acesso para as empresas utilizarem e modificarem o código.

A biblioteca contém mais de 2500 algoritmos otimizados, que possuem excelente precisão em desempenho e velocidade. Esses algoritmos podem ser usados ​​para detectar e reconhecer rostos, identificar objetos, classificar ações humanas em vídeos, rastrear movimentos de câmeras, rastrear objetos em movimento, extrair modelos 3D de objetos, produzir nuvens de pontos 3D de câmeras estéreo, unir imagens para produzir uma alta resolução imagem de uma cena inteira, encontrar imagens semelhantes de um banco de dados de imagens, remover olhos vermelhos de imagens tiradas com flash, seguir os movimentos dos olhos, reconhecer cenários e estabelecer marcadores para sobrepô-los com realidade aumentada, etc. OpenCV tem ótimas pessoas e comunidade envolvida como usuários , desenvolvedores e pesquisadores, o número é superior a 47 mil e número estimado de downloads superior a 7 milhões. A biblioteca é extensivamente em empresas profissionais, grupos de pesquisa e outros grupos.

Muitas empresas bem estabelecidas como Google, Yahoo, Microsoft, Intel, IBM, Sony, Honda, Toyota que empregam a biblioteca, existem muitas startups como Applied Minds, VideoSurf e Zeitera, que fazem uso extensivo do OpenCV. Os usos implantados do OpenCV abrangem desde a junção de imagens de vista da rua, detecção de intrusões em vídeo de vigilância em Israel, monitoramento de equipamentos de minas na China, ajuda a robôs a navegar e pegar objetos na Willow Garage, detecção de acidentes de afogamento em piscinas na Europa, execução de arte interativa em Espanha e Nova York, verificando pistas em busca de detritos na Turquia, inspecionando rótulos de produtos em fábricas ao redor do mundo para detecção rápida de rostos no Japão. Possui interfaces C++, C, Python, Java e MATLAB e suporta Windows, Linux, Android e Mac OS. O OpenCV se inclina principalmente para aplicativos de visão em tempo real e aproveita as instruções MMX e SSE quando disponíveis. Uma interface CUDA e OpenCL com todos os recursos está sendo desenvolvida ativamente agora. Existem mais de 500 algoritmos e cerca de 10 vezes mais funções que compõem ou suportam esses algoritmos. O OpenCV é escrito nativamente em C++ e possui uma interface de modelo que funciona perfeitamente com contêineres STL.

Obter imagem da webcam #

Exiba um feed de vídeo ao vivo obtido de uma webcam usando a classe VideoCapture do OpenCV com Java, C/C++ e Python.

#Java

import org.opencv.core.Core;
import org.opencv.core.Mat;
import org.opencv.videoio.VideoCapture;
 
public class Camera {
    public static void main(String[] args) {
        // Load Native Library
        System.loadLibrary(Core.NATIVE_LIBRARY_NAME);
        // image container object
        Mat imageArray = new Mat();
        // Video device acces
        VideoCapture videoDevice = new VideoCapture();
        // 0:Start default video device 1,2 etc video device id
        videoDevice.open(0);
        // is contected
        if (videoDevice.isOpened()) {
        // Get frame from camera
            videoDevice.read(imageArray);
            // image array
            System.out.println(imageArray.toString());
            // Release video device
            videoDevice.release();
        } else {
            System.out.println("Error.");
        }
    }
}

#C++

#include "opencv2/opencv.hpp"
#include "iostream"

int main(int, char**) {
    // open the first webcam plugged in the computer
    cv::VideoCapture camera(0);
    if (!camera.isOpened()) {
        std::cerr << "ERROR: Could not open camera" << std::endl;
        return 1;
    }

    // create a window to display the images from the webcam
    cv::namedWindow("Webcam", CV_WINDOW_AUTOSIZE);

    // this will contain the image from the webcam
    cv::Mat frame;
        
    // capture the next frame from the webcam
    camera >> frame;
    
    // display the frame until you press a key
    while (1) {
        // show the image on the window
        cv::imshow("Webcam", frame);
        // wait (10ms) for a key to be pressed
        if (cv::waitKey(10) >= 0)
            break;
    }
    return 0;
}

Pitão #

import numpy as np
import cv2

# Video source - can be camera index number given by 'ls /dev/video*
# or can be a video file, e.g. '~/Video.avi'
cap = cv2.VideoCapture(0)

while(True):
    # Capture frame-by-frame
    ret, frame = cap.read()

    # Our operations on the frame come here
    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)

    # Display the resulting frame
    cv2.imshow('frame',gray)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

# When everything done, release the capture
cap.release()
cv2.destroyAllWindows()
