Embora a maioria das tags HTML seja usada para criar elementos, o HTML também fornece tags de formatação no texto para aplicar estilos específicos relacionados ao texto a partes do texto. Este tópico inclui exemplos de formatação de texto HTML, como realce, negrito, sublinhado, subscrito e texto riscado.

Sintaxe #

  • <abbr>Abreviação</abbr>
  • <b>Texto em negrito</b>
  • <del>Texto excluído</del>
  • <em>Texto enfatizado</em>
  • <i>Texto em itálico</i>
  • <ins>Texto inserido</ins>
  • <mark>Texto marcado (ou realçado)</mark>
  • <s>Texto riscado</s>
  • <strong>Texto forte</strong>
  • <sub>Texto Subscrito</sub>
  • <sup>Texto sobrescrito</sup>
  • <u>Texto Sublinhado</u>

Destacando #

O elemento <mark> é novo no HTML5 e é usado para marcar ou destacar texto em um documento “devido à sua relevância em outro contexto”.1

O exemplo mais comum seria nos resultados de uma pesquisa onde o usuário inseriu uma consulta de pesquisa e os resultados são mostrados destacando a consulta desejada.

<p>Here is some content from an article that contains the <mark>searched query</mark>
that we are looking for. Highlighting the text will make it easier for the user to 
find what they are looking for.</p>

Resultado:

Uma formatação padrão comum é o texto preto em um fundo amarelo, mas isso pode ser alterado com CSS.

Negrito, Itálico e Sublinhado #

Texto em negrito #

Para texto em negrito, use as tags <strong> ou <b>:

<strong>Bold Text Here</strong>

ou

<b>Bold Text Here</b>

Qual é a diferença? Semântica. <strong> é usado para indicar que o texto é fundamental ou semanticamente importante para o texto ao redor, enquanto <b> indica que não há tal importância e simplesmente representa o texto que deve estar em negrito.

Se você usasse <b> um programa de conversão de texto em fala não diria a(s) palavra(s) de forma diferente de qualquer uma das outras palavras ao redor - você está simplesmente chamando a atenção para elas sem adicionar nenhuma importância adicional. Ao usar <strong>, no entanto, o mesmo programa gostaria de falar essas palavras com um tom de voz diferente para transmitir que o texto é importante de alguma forma.

Texto em itálico #

Para colocar o texto em itálico, use as tags <em> ou <i>:

<em>Italicized Text Here</em>

ou

<i>Italicized Text Here</i>

Qual é a diferença? Semântica. <em> é usado para indicar que o texto deve ter uma ênfase extra que deve ser enfatizada, enquanto <i> simplesmente representa o texto que deve ser separado do texto normal ao seu redor.

Por exemplo, se você quiser enfatizar a ação dentro de uma frase, pode-se fazê-lo enfatizando-a em itálico via <em>: “Você poderia apenas enviar a edição já?”

Mas se você estivesse identificando um livro ou jornal que normalmente colocaria em itálico estilisticamente, você simplesmente usaria <i>: “Eu fui forçado a ler Romeu e Julieta no ensino médio.

Texto sublinhado #

Enquanto o próprio elemento <u> foi descontinuado no HTMl 4, ele foi reintroduzido com significado semântico alternativo no HTML 5 - para representar uma anotação não textual não articulada. Você pode usar essa renderização para indicar texto com erros ortográficos na página ou para uma marca de nome próprio em chinês.

<p>This paragraph contains some <u>mispelled</u> text.</p>

Abreviatura #

Para marcar alguma expressão como uma abreviação, use a tag <abbr>:

<p>I like to write <abbr title="Hypertext Markup Language">HTML</abbr>!</p>

Se presente, o atributo title é utilizado para apresentar a descrição completa de tal abreviatura.

Inserido, Excluído ou Atingido #

Para marcar o texto como inserido, use a tag <ins>:

<ins>New Text</ins>

Para marcar o texto como excluído, use a tag <del>:

<del>Deleted Text</del>

Para riscar o texto, use a tag <s>:

<s>Struck-through text here</s>

Sobrescrito e Subscrito #

Para deslocar o texto para cima ou para baixo, você pode usar as tags <sup> e <sub>.

Para criar sobrescrito:

<sup>superscript here</sup>

Para criar subscrito:

<sub>subscript here</sub>
