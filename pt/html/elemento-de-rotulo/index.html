Sintaxe #

  • <label>Exemplo <input type="radio" name="r"></label> // Envolvendo um elemento de controle
  • <label for="rad1">Exemplo</label> <input id="rad1" type="radio" name="r"> // Usando o atributo for

Parâmetros #

Atributos Descrição
para Referência ao Elemento de ID de destino. Ou seja: for="sobrenome"
formulário HTML5, [Obsolete] Referência ao formulário que contém o Elemento de Destino. Elementos de rótulo são esperados dentro de um elemento <form>. Se o form="someFormId" for fornecido, isso permitirá que você coloque o rótulo em qualquer lugar do documento.

Sobre o rótulo #

O elemento <label> é usado para referenciar um elemento de ação de formulário. No escopo da Interface do usuário, é usado para facilitar o destino/seleção de elementos como Type radio ou checkbox.

<rótulo> como wrapper

Pode incluir o elemento de ação desejado

<label>
    <input type="checkbox" name="Cats">
    I like Cats!
</label>

(Clicar no texto do destino input alternará seu estado / valor)

<rótulo> como referência

Usando o atributo for você não precisa colocar o elemento de controle como descendente de label - mas o valor for deve corresponder ao seu ID

<input id="cats" type="checkbox" name="Cats">
<label for="cats" >I like Cats!</label>

Observação Não use mais de um elemento de controle dentro de um elemento <label>

Uso Básico #

Formulário simples com rótulos…

<form action="/login" method="POST">
    
    <label for="username">Username:</label>
    <input id="username" type="text" name="username" />

    <label for="pass">Password:</label>
    <input id="pass" type="password" name="pass" />

    <input type="submit" name="submit" />

</form>

<form id="my-form" action="/login" method="POST">
    
    <input id="username" type="text" name="username" />

    <label for="pass">Password:</label>
    <input id="pass" type="password" name="pass" />

    <input type="submit" name="submit" />

</form>

<label for="username" form="my-form">Username:</label>
