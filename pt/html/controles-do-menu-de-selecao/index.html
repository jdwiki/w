Sintaxe #

  • <select name=""></select>
  • <datalist id=""></datalist>
  • <optgroup label="Option Group"></optgroup>
  • <option value="">Opção</option>

Selecione o menu #

O elemento <select> gera um menu suspenso no qual o usuário pode escolher uma opção.

<select name="">
  <option value="1">One</option>
  <option value="2">Two</option>
  <option value="3">Three</option>
  <option value="4">Four</option>
</select>

Mudando o tamanho

Você pode alterar o tamanho do menu de seleção com o atributo size. Um tamanho de 0 ou 1 exibe o menu de estilo suspenso padrão. Um tamanho maior que 1 converterá a lista suspensa em uma caixa exibindo esse número de linhas, com uma opção por linha e uma barra de rolagem para percorrer as opções disponíveis.

<select name="" size="4"></select>

Menus de seleção de várias opções

Por padrão, os usuários podem selecionar apenas uma única opção. Adicionar o atributo multiple permite que os usuários selecionem várias opções de uma vez e enviem todas as opções selecionadas com o formulário. O uso do atributo multiple converte automaticamente o menu suspenso em uma caixa como se tivesse um tamanho definido. O tamanho padrão quando isso ocorre é determinado pelo navegador específico que você está usando e não é possível alterá-lo de volta para um menu de estilo suspenso ao permitir várias seleções.

<select name="" multiple></select>

Ao usar o atributo multiple, há uma diferença entre usar 0 e 1 para o tamanho, enquanto não existe diferença quando não se usa o atributo. O uso de 0 fará com que o navegador se comporte de qualquer maneira padrão para o qual está programado. O uso de 1 definirá explicitamente o tamanho da caixa resultante para apenas uma linha de altura.

Opções #

As opções dentro de um menu de seleção são o que o usuário selecionará. A sintaxe normal para uma opção é a seguinte:

<option>Some Option</option>

No entanto, é importante notar que o texto dentro do elemento <option> nem sempre é usado, e essencialmente se torna o valor padrão para atributos que não são especificados.

Os atributos que controlam a aparência e função reais da opção são value e label. O rótulo representa o texto que será exibido no menu suspenso (o que você está vendo e clicará para selecioná-lo). O valor representa o texto que será enviado junto com o envio do formulário. Se um desses valores for omitido, ele usará o texto dentro do elemento como o valor. Portanto, o exemplo que demos acima pode ser “expandido” para isso:

<option label="Some Option" value="Some Option">

Observe a omissão do texto interno e da tag final, que não são necessários para construir uma opção dentro do menu. Se eles fossem incluídos, o texto interno seria ignorado porque ambos os atributos já estão especificados e o texto não é necessário. No entanto, você provavelmente não verá muitas pessoas escrevendo dessa maneira. A maneira mais comum de escrever é com um valor que será enviado para o servidor, junto com o texto interno que eventualmente se tornará o atributo label, assim:

<option value="option1">Some Option</option>

Selecionando uma opção por padrão

Você também pode especificar uma determinada opção a ser selecionada no menu por padrão, anexando o atributo selected a ela. Por padrão, se nenhuma opção for especificada como selecionada no menu, a primeira opção do menu será selecionada quando renderizada. Se mais de uma opção tiver o atributo selected anexado, então a última opção presente no menu com o atributo será a selecionada por padrão.

<option value="option1" selected>Some option</option>

Se você estiver usando o atributo em um menu de seleção de várias opções, todas as opções com o atributo serão selecionadas por padrão e nenhuma será selecionada se nenhuma opção tiver o atributo.

<select multiple>
  <option value="option1" selected>Some option</option>
  <option value="option2" selected>Some option</option>   
</select>

Grupos de opções #

Você pode agrupar suas opções dentro de um menu de seleção para fornecer um layout mais estruturado em uma longa lista de opções usando o elemento <optgroup>.

A sintaxe é bem básica, bastando usar o elemento com um atributo label para identificar o título do grupo e conter zero ou mais opções que deveriam estar dentro desse grupo.

<select name="">
  <option value="milk">Milk</option>
  <optgroup label="Fruits">
    <option value="banana">Bananas</option>
    <option value="strawberry">Strawberries</option>
  </optgroup>
  <optgroup label="Vegetables" disabled>
    <option value="carrot">Carrots</option>
    <option value="zucchini">Zucchini</option>
  </optgroup>
</select>

Ao usar grupos de opções, nem todas as opções precisam estar contidas em um grupo. Além disso, desabilitar um grupo de opções desabilitará todas as opções dentro do grupo e não é possível reativar manualmente uma única opção dentro de um grupo desabilitado.

Lista de dados #

A tag <datalist> especifica uma lista de opções predefinidas para um elemento <input>. Ele fornece um recurso de “autocompletar” em elementos <input>. Os usuários verão uma lista suspensa de opções enquanto escrevem.

<input list="Languages">

<datalist id="Languages">
  <option value="PHP">
  <option value="Perl">
  <option value="Python">
  <option value="Ruby">
  <option value="C+">
</datalist>
