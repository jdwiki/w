SVG significa Scalable Vector Graphics. SVG é usado para definir gráficos para a Web

O elemento HTML <svg> é um contêiner para gráficos SVG.

O SVG tem vários métodos para desenhar caminhos, caixas, círculos, texto e imagens gráficas.

SVG é uma linguagem baseada em XML para criar imagens vetoriais escaláveis. Ele pode ser escrito diretamente em um documento HTML ou incorporado a partir de arquivos SVG externos. O SVG embutido pode ser reestilizado e modificado usando CSS e JavaScript, respectivamente.

O suporte do navegador para SVG varia, mas pode ser verificado aqui.

Para obter informações mais abrangentes, consulte a documentação SVG.

SVG embutido #

SVG pode ser escrito diretamente em um documento HTML. O SVG embutido pode ser estilizado e manipulado usando CSS e JavaScript.

<body>
    <svg class="attention" xmlns="http://www.w3.org/2000/svg" xmlns:xlink="http://www.w3.org/1999/xlink" viewBox="0 0 1000 1000" >
        <path id="attention" d="m571,767l0,-106q0,-8,-5,-13t-12,-5l-108,0q-7,0,-12,5t-5,13l0,106q0,8,5,13t12,6l108,0q7,0,12,-6t5,-13Zm-1,-208l10,-257q0,-6,-5,-10q-7,-6,-14,-6l-122,0q-7,0,-14,6q-5,4,-5,12l9,255q0,5,6,9t13,3l103,0q8,0,13,-3t6,-9Zm-7,-522l428,786q20,35,-1,70q-10,17,-26,26t-35,10l-858,0q-18,0,-35,-10t-26,-26q-21,-35,-1,-70l429,-786q9,-17,26,-27t36,-10t36,10t27,27Z" />
    </svg>
</body>

O SVG embutido acima pode ser estilizado usando a classe CSS correspondente:

.attention {
    fill: red;
    width: 50px;
    height: 50px;
}

Incorporando arquivos SVG externos em HTML #

Você pode usar os elementos <img> ou <object> para incorporar elementos SVG externos. Definir a altura e a largura é opcional, mas é altamente recomendado.

Usando o elemento de imagem #

<img src="attention.svg" width="50" height="50">

Usar <img> não permite estilizar o SVG usando CSS ou manipulá-lo usando JavaScript.

Usando o elemento objeto #

<object type="image/svg+xml" data="attention.svg" width="50" height="50">

Ao contrário do <img>, o <object> importa diretamente o SVG para o documento e, portanto, pode ser manipulado usando Javascript e CSS.

Incorporando SVG usando CSS #

Você pode adicionar arquivos SVG externos usando a propriedade background-image, assim como faria com qualquer outra imagem.

HTML:

<div class="attention"></div>

CSS:

.attention {
    background-image: url(attention.svg);
    background-size: 100% 100%;
    width: 50px;
    height: 50px;
}
