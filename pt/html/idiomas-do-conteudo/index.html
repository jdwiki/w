Sintaxe #

  • <element lang="code_language">   <!– O código do idioma deve estar no formato [ISO 639-1](https://en.wikipedia.org/wiki/ISO_639-1 ) –>

O valor do atributo lang deve ser uma tag de idioma BCP 47 válida ou a string vazia (se o idioma for desconhecido).

As tags de idioma BCP 47 estão listadas no [IANA Language Subtag Registry](http://www.iana.org/assignments/language -subtag-registro).

Acessibilidade #

Os Critérios de Sucesso relevantes das WCAG 2.0 são:

  • 3.1.1 Idioma da página
  • 3.1.2 Idioma das peças

As Técnicas WCAG 2.0 relacionadas são:

  • H57: Usando atributos de idioma no elemento html
  • H58: Usando atributos de linguagem para identificar mudanças na linguagem humana

Idioma do documento base #

É uma boa prática declarar o idioma principal do documento no elemento html:

<html lang="en">

Se nenhum outro atributo lang for especificado no documento, significa que tudo (ou seja, conteúdo do elemento e valores de texto do atributo) está nesse idioma.

Se o documento contém partes em outros idiomas, essas partes devem obter seus próprios atributos lang para “sobrescrever” a declaração do idioma.

Linguagem do elemento #

O atributo lang é usado para especificar o idioma do conteúdo do elemento e os valores de texto do atributo:

<p lang="en">The content of this element is in English.</p>

<p lang="en" title="The value of this attribute is also in English.">The content of this element is in English.</p>

A declaração de idioma é herdada:

<div lang="en">
  <p>This element contains English content.</p>
  <p title="This attribute, too.">Same with this element.</p>
</div>

Elementos com vários idiomas #

Você pode “sobrescrever” uma declaração de idioma:

<p lang="en">This English sentence contains the German word <span lang="de">Hallo</span>.</p>

Manipulando Atributos com Diferentes Idiomas #

Você pode “sobrescrever” a declaração de idioma de um elemento pai introduzindo qualquer elemento além de applet, base, basefont, br, frame, frameset, hr, iframe, meta , param, script (do HTML 4.0) com um atributo próprio lang:

<p lang="en" title="An English paragraph">
    <span lang="de" title="A German sentence">Hallo Welt!</span>
</p>

URLs regionais #

É possível adicionar o atributo hreflang aos elementos <a> e <area> que criam hiperlinks. Tal especifica o idioma do recurso vinculado. O idioma definido deve ser uma tag de idioma BCP 471 válida.

<p>
    <a href="example.org" hreflang="en">example.org</a> is one of IANA's example domains.
</p>
