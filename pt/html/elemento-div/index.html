Elemento Div

  • Sintaxe
  • Uso básico
  • Aninhamento

Nesta página

  • Sintaxe
  • Uso básico
  • Aninhamento

O elemento div em HTML é um elemento container que encapsula outros elementos e pode ser usado para agrupar e separar partes de uma página da web. Um div por si só não representa nada inerentemente, mas é uma ferramenta poderosa em web design. Este tópico abrange a finalidade e as aplicações do elemento div.

Sintaxe #

  • <div>div de exemplo</div>

Uso básico #

O elemento <div> normalmente não tem significado semântico específico por si só, simplesmente representando uma divisão, e é normalmente usado para agrupar e encapsular outros elementos dentro de um documento HTML e separar aqueles de outros grupos de conteúdo. Como tal, cada <div> é melhor descrito por seu conteúdo.

<div>
  <p>Hello! This is a paragraph.</p>
</div>

O elemento div é tipicamente um elemento em nível de bloco, o que significa que ele separa um bloco de um documento HTML e ocupa a largura máxima da página. Os navegadores geralmente têm a seguinte regra CSS padrão:

div {
  display: block;
}

É fortemente encorajado pelo The World Wide Web Consortium (W3C) ver o elemento div como um elemento de último recurso, para quando nenhum outro elemento for adequado. O uso de elementos mais apropriados em vez do elemento div leva a uma melhor acessibilidade para os leitores e uma manutenção mais fácil para os autores.

Por exemplo, um post de blog seria marcado usando <article>, um capítulo usando <section>, os auxílios de navegação de uma página usando <nav> e um grupo de controles de formulário usando <fieldset>.

Os elementos div podem ser úteis para fins estilísticos ou para envolver vários parágrafos em uma seção que devem ser anotados de maneira semelhante.

1: https://www.wikiod.com/pt/css/controle-de-layout#A propriedade de exibição

Aninhamento #

É uma prática comum colocar múltiplos <div> dentro de outro <div>. Isso geralmente é chamado de elementos de “aninhamento” e permite dividir ainda mais os elementos em subseções ou ajudar os desenvolvedores com o estilo CSS.

O <div class="outer-div"> é usado para agrupar dois elementos <div class="inner-div">; cada um contendo um elemento <p>.

<div class="outer-div">
  <div class="inner-div">
    <p>This is a paragraph</p>
  </div>
  <div class="inner-div">
    <p>This is another paragraph</p>
  </div>
</div>

Isso produzirá o seguinte resultado (estilos CSS aplicados para maior clareza):

Aninhamento de elementos inline e de bloco Ao aninhar elementos, você deve ter em mente que existem elementos em linha e em bloco. enquanto os elementos do bloco “adicionam uma quebra de linha no fundo”, ou seja, outros elementos aninhados são mostrados na próxima linha automaticamente, os elementos embutidos podem ser posicionados próximos uns dos outros por padrão

Evite o aninhamento profundo de <div>

Um layout de contêiner aninhado profundo e frequentemente usado mostra um estilo de codificação ruim.

Cantos arredondados ou algumas funções semelhantes geralmente criam esse código HTML. Para a maioria dos navegadores de última geração, existem contrapartes CSS3. Tente usar o mínimo possível de elementos HTML para aumentar a proporção de conteúdo para tag e reduzir o carregamento da página, resultando em uma melhor classificação nos mecanismos de pesquisa.

Seção div O elemento não deve ser aninhado com mais de 6 camadas.
