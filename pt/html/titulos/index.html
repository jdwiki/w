HTML fornece não apenas tags de parágrafo simples, mas seis tags de cabeçalho separadas para indicar cabeçalhos de vários tamanhos e espessuras. Enumerado como título 1 a título 6, o título 1 tem o texto maior e mais grosso, enquanto o título 6 é o menor e mais fino, até o nível do parágrafo. Este tópico detalha o uso adequado dessas tags.

Sintaxe #

  • <h1>...</h1>

  • <h2>...</h2>

  • <h3>...</h3>

  • <h4>...</h4>

  • <h5>...</h5>

  • <h6>...</h6>

  • Um elemento h1–h6 deve ter uma tag inicial e uma tag final.1

  • Os elementos h1–h6 são elementos de nível de bloco por padrão (estilo CSS: display: block).2

  • Elementos h1–h6 não devem ser confundidos com o elemento de seção

  • As tags de cabeçalho (h1–h6) não estão relacionadas à tag head.

  • Conteúdo permitido: conteúdo de fraseamento

  • Os diferentes estilos CSS para títulos diferem geralmente em font-size e margin. As seguintes configurações de CSS para elementos h1–h6 podem servir como orientação (caracterizada como ‘informativa’ pelo W3C)

  • Search engine spiders (o código que adiciona uma página a um mecanismo de busca) automaticamente presta mais atenção aos cabeçalhos de maior importância (h1 tem mais, h2 tem menos, h3 tem ainda menos, …) para discernir sobre o que é uma página.

Usando títulos #

Os títulos podem ser usados ​​para descrever o tópico que precedem e são definidos com as tags <h1> a <h6>. Os títulos suportam todos os atributos globais.

  • <h1> define o título mais importante.
  • <h6> define o título menos importante.

Definindo um título:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

A estrutura correta importa #

Os mecanismos de pesquisa e outros agentes de usuário geralmente indexam o conteúdo da página com base em elementos de título, por exemplo, para criar um índice. Portanto, é importante usar a estrutura correta para os títulos.

Em geral, um artigo deve ter um elemento h1 para o título principal seguido por legendas h2 – descendo uma camada, se necessário. Se houver elementos h1 em um nível superior, eles não devem ser usados ​​para descrever qualquer conteúdo de nível inferior.

Exemplo de documento (intenção extra para ilustrar a hierarquia):

<h1>Main title</h1>
<p>Introduction</p>

    <h2>Reasons</h2>

        <h3>Reason 1</h3>
        <p>Paragraph</p>

        <h3>Reason 2</h3>
        <p>Paragraph</p>

    <h2>In conclusion</h2>
    <p>Paragraph</p>
