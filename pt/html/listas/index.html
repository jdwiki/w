O HTML oferece três maneiras de especificar listas: listas ordenadas, listas não ordenadas e listas de descrição. As listas ordenadas usam sequências ordinais para indicar a ordem dos elementos da lista, as listas não ordenadas usam um símbolo definido, como um marcador, para listar elementos sem ordem designada e as listas de descrição usam recuos para listar elementos com seus filhos. Este tópico explica a implementação e a combinação dessas listas na marcação HTML.

Sintaxe #

  • <ol> itens de lista ordenada </ol>
  • <ul> itens de lista não ordenados </ul>
  • <li> item de lista (ordenado e não) </li>
  • <dl> itens da lista de descrição </dl>
  • <dt> título da lista de descrição </dt>
  • <dd> descrição da lista de descrição </dd>

Veja também

Você pode adicionar uma propriedade CSS do tipo list-style a uma tag <ul> para alterar o tipo de ícone usado para marcar cada item da lista, por exemplo <ul style="list-style-type:disc ">. Os seguintes tipos de estilo de lista são permitidos:

  • “list-style-type:disc” é o ponto padrão.
  • “list-style-type:circle” é um círculo não preenchido.
  • “list-style-type:square” é um quadrado preenchido.
  • “list-style-type:none” não usa nenhuma marca.

Você também pode adicionar um atributo type a uma tag <ol> para alterar como a numeração é feita, por exemplo <ol type="1">. Os seguintes tipos são permitidos:

  • type=“1” é o formulário padrão.
  • type=“A” usa letras maiúsculas em ordem alfabética
  • type=“a” usa letras minúsculas em ordem alfabética
  • type=“I” usa algarismos romanos com letras maiúsculas
  • type=“i” usa algarismos romanos com letras minúsculas

Lista ordenada #

Uma lista ordenada pode ser criada com a tag <ol> e cada item da lista pode ser criado com a tag <li> como no exemplo abaixo:

<ol>
  <li>Item</li>
  <li>Another Item</li>
  <li>Yet Another Item</li>
</ol>

Isso produzirá uma lista numerada (que é o estilo padrão):

  1. Artigo
  2. Outro item
  3. Mais um item

Alterando manualmente os números #

Existem algumas maneiras de brincar com os números que aparecem nos itens da lista em uma lista ordenada. A primeira maneira é definir um número inicial, usando o atributo start. A lista começará neste número definido e continuará aumentando em um como de costume.

<ol start="3">
  <li>Item</li>
  <li>Some Other Item</li>
  <li>Yet Another Item</li>
</ol>

Isso produzirá uma lista numerada (que é o estilo padrão):

  1. Artigo
  2. Algum Outro Item
  3. Mais um item

Você também pode definir explicitamente um determinado item de lista para um número específico. Outros itens de lista após um com um valor especificado continuarão sendo incrementados em um do valor desse item de lista, ignorando onde a lista pai estava.

<li value="7"></li>

Também vale a pena notar que, usando o atributo value diretamente em um item de lista, você pode substituir o sistema de numeração existente de uma lista ordenada reiniciando a numeração em um valor menor. Portanto, se a lista pai já estiver no valor 7 e encontrar um item da lista no valor 4, esse item da lista ainda será exibido como 4 e continuará contando a partir desse ponto novamente.

<ol start="5">
  <li>Item</li>
  <li>Some Other Item</li>
  <li value="4">A Reset Item</li>
  <li>Another Item</li>
  <li>Yet Another Item</li>
</ol>

Assim, o exemplo acima produzirá uma lista que segue o padrão de numeração de 5, 6, 4, 5, 6 - começando novamente em um número menor que o anterior e duplicando o número 6 na lista.

Observação: Os atributos start e value aceitam apenas um número - mesmo se a lista ordenada estiver configurada para ser exibida como algarismos romanos ou letras.

Você pode reverter a numeração adicionando reversed em seu elemento ol:

<ol reversed>
  <li>Item</li>
  <li>Some Other Item</li>
  <li value="4">A Reset Item</li>
  <li>Another Item</li>
  <li>Yet Another Item</li>
</ol>

A numeração reversa é útil se você estiver adicionando continuamente a uma lista, como em novos episódios ou apresentações de podcast, e quiser que os itens mais recentes apareçam primeiro.

Alterando o tipo de numeral #

Você pode alterar facilmente o tipo de numeral mostrado no marcador de item da lista usando o atributo type

<ol type="1|a|A|i|I">
Tipo Descrição Exemplos
1 Valor padrão - Números decimais 1,2,3,4
a Em ordem alfabética (minúsculas) a,b,c,d
A Ordem alfabética (maiúsculas) A,B,C,D
eu Algarismos Romanos (minúsculos) eu,ii,iii,iv
’eu' Algarismos Romanos (maiúsculas) I,II,III,IV

Você deve usar ol para exibir uma lista de itens, onde os itens foi intencionalmente ordenado e a ordem deve ser enfatizada. Se mudar a ordem dos itens NÃO torna a lista incorreta, você deve use <ul>.

1: https://www.wikiod.com/pt/html/listas#Lista não ordenada

Lista não ordenada #

Uma lista não ordenada pode ser criada com a tag <ul> e cada item da lista pode ser criado com a tag <li> conforme mostrado no exemplo abaixo:

<ul>
  <li>Item</li>
  <li>Another Item</li>
  <li>Yet Another Item</li>
</ul>

Isso produzirá uma lista com marcadores (que é o estilo padrão):

  • Artigo
  • Outro item
  • Mais um item

Você deve usar ul para exibir uma lista de itens, onde a ordem dos itens é não é importante. Se alterar a ordem dos itens torna a lista incorreto, você deve usar <ol>.

1: https://www.wikiod.com/pt/html/listas#Lista ordenada

Listas aninhadas #

Você pode aninhar listas para representar subitens de um item de lista.

<ul>
  <li>item 1</li>
  <li>item 2
    <ul>
      <li>sub-item 2.1</li>
      <li>sub-item 2.2</li>
    </ul>
  </li>
  <li>item 3</li>
</ul>
  • item 1
  • item 2
    • subitem 2.1
    • subitem 2.2
  • item 3

A lista aninhada deve ser filha do elemento li.

Você também pode aninhar diferentes tipos de lista:

<ol>
    <li>Hello, list!</li>
    <li>
        <ul>
            <li>Hello, nested list!</li>
        </ul>
    </li>
</ol>

Lista de descrição #

Uma lista de descrição (ou lista de definição, como era chamada antes do HTML5) pode ser criada com o elemento dl. Consiste em grupos nome-valor, onde o nome é dado no elemento dt e o valor é dado no elemento dd.

<dl>
  <dt>name 1</dt>
  <dd>value for 1</dd>
  <dt>name 2</dt>
  <dd>value for 2</dd>
</dl>

Demonstração ao vivo

Um grupo nome-valor pode ter mais de um nome e/ou mais de um valor (que representam alternativas):

<dl>

  <dt>name 1</dt>
  <dt>name 2</dt>
  <dd>value for 1 and 2</dd>

  <dt>name 3</dt>
  <dd>value for 3</dd>
  <dd>value for 3</dd>

</dl>
