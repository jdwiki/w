Introdução #

HTML (Hypertext Markup Language) usa um sistema de marcação composto por elementos que representam conteúdo específico. Markup significa que com HTML você declara o que é apresentado a um visualizador, não como é apresentado. As representações visuais são definidas por Cascading Style Sheets (CSS) e realizadas por navegadores. Elementos ainda existentes que permitem isso, como, por exemplo, font, “são totalmente obsoletos e não devem ser usados ​​pelos autores”[1] .

HTML às vezes é chamado de linguagem de programação, mas não tem lógica, portanto, é uma linguagem de marcação. As tags HTML fornecem significado semântico e legibilidade por máquina ao conteúdo da página.

Um elemento geralmente consiste em uma tag de abertura (<element_name>), uma tag de fechamento (</element_name>), que contém o nome do elemento entre colchetes angulares e o conteúdo entre: <element_name>.. .conteúdo...</element_name>

Existem alguns elementos HTML que não possuem uma tag de fechamento ou nenhum conteúdo. Estes são chamados de elementos vazios. Elementos vazios incluem <img>, <meta>, <link> e <input>.

Os nomes dos elementos podem ser considerados palavras-chave descritivas para o conteúdo que eles contêm, como vídeo, áudio, tabela, rodapé.

Uma página HTML pode consistir em potencialmente centenas de elementos que são lidos por um navegador da Web, interpretados e transformados em conteúdo legível ou audível na tela.

Para este documento é importante observar a diferença entre elementos e tags:

Elementos: vídeo, áudio, tabela, rodapé

Tags: <video>, <audio>, <table>, <footer>, </html>, </body>


Insight do elemento #

Vamos quebrar uma tag…

A tag <p> representa um parágrafo comum.

Os elementos geralmente têm uma tag de abertura e uma tag de fechamento. A tag de abertura contém o nome do elemento entre colchetes angulares (<p>). A tag de fechamento é idêntica à tag de abertura com a adição de uma barra (/) entre o colchete de abertura e o nome do elemento (</p>).

O conteúdo pode então ir entre essas duas tags: <p>Este é um parágrafo simples.</p>.


Criando uma página simples #

O exemplo de HTML a seguir cria uma página da Web simples “Hello World”.

Arquivos HTML podem ser criados usando qualquer editor de texto. Os arquivos devem ser salvos com um .html ou .htm[2]</ sup> para serem reconhecidos como arquivos HTML.

Uma vez criado, este arquivo pode ser aberto em qualquer navegador da web.


<!DOCTYPE html>
<html lang="en">

    <head>
        <meta charset="UTF-8">
        <title>Hello!</title>
    </head>

    <body>
        <h1>Hello World!</h1>
        <p>This is a simple paragraph.</p>
    </body>

</html>


Quebra de página simples #

Estas são as tags usadas no exemplo:

Etiqueta Significado
<!DOCTYPE> Define a versão HTML usada no documento. Neste caso é HTML5.
Veja o tópico de tipos de documentos para mais informações.
<html> Abre a página. Nenhuma marcação deve vir após a tag de fechamento (</html>). O atributo lang declara o idioma principal da página usando os códigos de idioma ISO (en para inglês).
Consulte o [tópico de idioma do conteúdo](http://stackoverflow.com/ document/html/737/content-languages#t=201607221637059314928) para obter mais informações.
<cabeça> Abre a seção head, que não aparece na janela principal do navegador, mas contém principalmente informações sobre o documento HTML, chamado metadados. Ele também pode conter importações de folhas de estilo e scripts externos. A tag de fechamento é </head>.
<meta> Fornece ao navegador alguns metadados sobre o documento. O atributo charset declara a codificação de caracteres. Documentos HTML modernos devem sempre usar UTF-8, mesmo que não seja um requisito. Em HTML, a tag <meta> não requer uma tag de fechamento.
Consulte o tópico Meta para obter mais informações.
<título> O título da página. O texto escrito entre esta abertura e a tag de fechamento (</title>) será exibido na aba da página ou na barra de título do navegador.
<corpo> Abre a parte do documento exibida aos usuários, ou seja, todo o conteúdo visível ou audível de uma página. Nenhum conteúdo deve ser adicionado após a tag de fechamento </body>.
<h1> Um título de nível 1 para a página.
Veja headings para mais informações.
<p> Representa um parágrafo comum de texto.
