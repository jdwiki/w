Olá Mundo #

Para criar um plug-in Hello World para Eclipse, clique em: Arquivo ➜ Novo ➜ Outro…

Selecione Projeto de Plug-in e clique em Avançar>

O assistente Novo projeto de plug-in o guiará pelas opções para criar um novo plug-in.

Digite um nome de projeto (como HelloWorld) e clique em Next >

Na página Content, você pode definir o ID, Version, Name e Vendor do plug-in.

A Versão será 1.0.0.qualifier por padrão. Você pode deixar isso como está, mas é melhor mudar isso para algo significativo. O wiki do eclipse recomenda uma sintaxe como vYYYYMMDD (ano, mês dia).

Na página Modelos, você pode optar por criar seu plug-in a partir de qualquer modelo selecionando-o e clicando em Avançar >. Como alternativa, você pode combinar esses modelos escolhendo Assistente de plug-in personalizado ou para criar um novo plug-in sem um modelo desmarcando a caixa de seleção na frente de Criar um plug-in usando um dos modelos.

Para o modelo Hello, World Command, há configurações adicionais: o nome do pacote, o nome da classe do manipulador e o texto da caixa de mensagem.

Quando o plug-in é criado, você pode executá-lo clicando com o botão direito do mouse em plugin.xml ➜ Executar como ➜ Aplicativo Eclipse

Isso iniciará uma nova instância do Eclipse (com sua própria área de trabalho) que terá seu plug-in carregado.

Este plug-in Hello World terá feito 3 contribuições para a GUI do Eclipse:

1. Um Menu de Amostra (com Comando de Amostra):

Plugin.xml:

<extension
      point="org.eclipse.ui.menus">
   <menuContribution
         locationURI="menu:org.eclipse.ui.main.menu?after=additions">
      <menu
            label="Sample Menu"
            mnemonic="M"
            id="HelloWorld.menus.sampleMenu">
         <command
               commandId="HelloWorld.commands.sampleCommand"
               mnemonic="S"
               id="HelloWorld.menus.sampleCommand">
         </command>
      </menu>
   </menuContribution>
</extension>

2. Um ícone da barra de ferramentas:

Plugin.xml:

<extension
      point="org.eclipse.ui.menus">
   <menuContribution
         locationURI="toolbar:org.eclipse.ui.main.toolbar?after=additions">
      <toolbar
            id="HelloWorld.toolbars.sampleToolbar">
         <command
               commandId="HelloWorld.commands.sampleCommand"
               icon="icons/sample.gif"
               tooltip="Say hello world"
               id="HelloWorld.toolbars.sampleCommand">
         </command>
      </toolbar>
   </menuContribution>
</extension>

3. Um atalho de tecla (Ctrl+6)

Plugin.xml:

<extension
      point="org.eclipse.ui.bindings">
   <key
         commandId="HelloWorld.commands.sampleCommand"
         contextId="org.eclipse.ui.contexts.window"
         sequence="M1+6"
         schemeId="org.eclipse.ui.defaultAcceleratorConfiguration">
   </key>
</extension>

Ao ativar qualquer um desses 3, a classe Handler será executada:

Plugin.xml:

<extension
      point="org.eclipse.ui.commands">
   <category
         name="Sample Category"
         id="HelloWorld.commands.category">
   </category>
   <command
         name="Sample Command"
         categoryId="HelloWorld.commands.category"
         id="HelloWorld.commands.sampleCommand">
   </command>
</extension>
<extension
      point="org.eclipse.ui.handlers">
   <handler
         commandId="HelloWorld.commands.sampleCommand"
         class="helloworld.handlers.SampleHandler">
   </handler>
</extension>

SampleHandler.java:

package helloworld.handlers;

import org.eclipse.core.commands.AbstractHandler;
import org.eclipse.core.commands.ExecutionEvent;
import org.eclipse.core.commands.ExecutionException;
import org.eclipse.ui.IWorkbenchWindow;
import org.eclipse.ui.handlers.HandlerUtil;
import org.eclipse.jface.dialogs.MessageDialog;

/**
 * Our sample handler extends AbstractHandler, an IHandler base class.
 * @see org.eclipse.core.commands.IHandler
 * @see org.eclipse.core.commands.AbstractHandler
 */
public class SampleHandler extends AbstractHandler {

    @Override
    public Object execute(ExecutionEvent event) throws ExecutionException {
        IWorkbenchWindow window = HandlerUtil.getActiveWorkbenchWindowChecked(event);
        MessageDialog.openInformation(
                window.getShell(),
                "HelloWorld",
                "Hello, Eclipse world");
        return null;
    }
}

Instalação ou Configuração #

Supondo que você tenha o Eclipse IDE for Java Developers instalado, inicie o Eclipse, clique em “Help” -> “Install New Software…”

Selecione “–All Available Sites–” em “Work with:” e navegue até “Eclipse Plugin Development Tools”. Selecione “Eclipse Plug-in Development Environment” marcando a caixa de seleção na frente dele.

Clique em “Next” para permitir que o Eclipse verifique as dependências necessárias. Clique em “Next” novamente para iniciar a instalação.

Quando terminar, reinicie o Eclipse.
