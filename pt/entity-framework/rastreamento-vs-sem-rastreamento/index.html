Rastreamento vs. Sem Rastreamento

  • Consultas sem rastreamento
  • Consultas de rastreamento
  • Acompanhamento e projeções

Nesta página

  • Consultas sem rastreamento
  • Consultas de rastreamento
  • Acompanhamento e projeções

O comportamento de rastreamento controla se o Entity Framework manterá ou não informações sobre uma instância de entidade em seu rastreador de alterações. Se uma entidade for rastreada, quaisquer alterações detectadas na entidade serão mantidas no banco de dados durante SaveChanges().

Consultas sem rastreamento #

  • Nenhuma consulta de rastreamento é útil quando os resultados são usados ​​em um cenário somente leitura
  • Eles são mais rápidos de executar porque não há necessidade de configurar a mudança informações de rastreamento

Exemplo :

using (var context = new BookContext())
{
    var books = context.Books.AsNoTracking().ToList();
}

Com o EF Core 1.0, você também pode alterar o comportamento de rastreamento padrão no contexto nível de instância.

Exemplo :

using (var context = new BookContext())
{
    context.ChangeTracker.QueryTrackingBehavior = QueryTrackingBehavior.NoTracking;

    var books = context.Books.ToList();
}

Consultas de rastreamento #

  • Por padrão, as consultas que retornam tipos de entidade são de rastreamento
  • Isso significa que você pode fazer alterações nessas instâncias de entidade e ter essas mudanças persistiram por SaveChanges()

Exemplo :

  • A mudança na classificação do book será detectada e mantida até o banco de dados durante SaveChanges().

    using (var context = new BookContext())
{
  var book = context.Books.FirstOrDefault(b => b.BookId == 1);
  book.Rating = 5;
  context.SaveChanges();
}

Acompanhamento e projeções #

  • Mesmo que o tipo de resultado da consulta não seja um tipo de entidade, se o resultado contém tipos de entidade eles ainda serão rastreados por padrão

Exemplo :

  • Na consulta a seguir, que retorna um tipo anônimo, o instâncias de Book no conjunto de resultados serão rastreadas

      using (var context = new BookContext())
  {
     var book = context.Books.Select(b => new { Book = b, Authors = b.Authors.Count() });
  }

  • Se o conjunto de resultados não contiver nenhum tipo entidade, então sem rastreamento é desempenhado

Exemplo :

  • Na consulta a seguir, que retorna um tipo anônimo com alguns dos os valores da entidade (mas sem instâncias da entidade real type), não há nenhum rastreamento realizado.

     using (var context = new BookContext())
 {
    var book = context.Books.Select(b => new { Id = b.BookId, PublishedDate = b.Date });
 }
