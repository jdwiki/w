Você pode fornecer ajuda para formulários e controles em aplicativos Windows Forms de diferentes maneiras. Você pode mostrar uma ajuda pop-up, abrir um arquivo CHM ou um URL. Você pode mostrar ajuda sensível ao contexto para formulários, controles e diálogos.

Componente HelpProvider #

Você pode configurar um componente HelpProvider para fornecer ajuda sensível ao contexto para o componente. Desta forma, quando o usuário pressiona a tecla F1 ou o botão Ajuda do formulário, você pode automaticamente:

  • Mostrar um pop-up de ajuda sensível ao contexto para controles
  • Abra um arquivo CHM com base no contexto (Mostrar tabela de conteúdo, Mostrar uma palavra-chave ou índice, mostrar um tópico)
  • Navegue até um URL usando o navegador padrão

Aula de ajuda #

Você pode usar a classe Help no código, para fornecer esses tipos de ajuda:

  • Mostrar um pop-up de ajuda para um controle
  • Abra um arquivo CHM com base no contexto (Mostrar tabela de conteúdo, Mostrar uma palavra-chave ou índice, mostrar um tópico)
  • Navegue até um URL usando o navegador padrão

Evento Solicitado Ajuda #

Você pode manipular o evento HelpRequested de objetos Control ou Formulário para realizar ações personalizadas quando o usuário pressiona F1 ou clica no botão Ajuda do formulário.

Botão de ajuda do formulário #

Você pode configurar o Formulário para mostrar o botão Ajuda na barra de título. Desta forma, se o usuário clicar no botão Help, o cursor mudará para um cursor ? e após clicar em qualquer ponto, qualquer ajuda contextual associada ao controle usando HelpProvider será mostrada.

Botão de ajuda do MessgeBox e CommonDialogs #

Você pode fornecer ajuda para MessageBox, OpenFileDialog, SaveDialog e ColorDialog usando o botão Ajuda dos componentes.

Componente de dica de ferramenta #

Você pode usar o componente ToolTip para exibir algum texto de ajuda quando o usuário apontar para os controles. Uma ToolTip pode ser associada a qualquer controle.

Observação

Usando as classes HelpProvider e Help Você pode mostrar arquivos de Ajuda compilados (.chm) ou arquivos HTML no formato de Ajuda HTML. Os arquivos de Ajuda compilados fornecem um índice, um índice, capacidade de pesquisa e links de palavras-chave nas páginas. Os atalhos funcionam apenas em arquivos de Ajuda compilados. Você pode gerar arquivos HTML Help 1.x usando HTML Help Workshop. Para obter mais informações sobre a Ajuda HTML, consulte “HTML Help Workshop” e outros tópicos da Ajuda HTML em Microsoft HTML Help .

Mostrar arquivo de ajuda #

A Help Class encapsula o mecanismo HTML Help 1.0. Você pode usar o objeto Ajuda para mostrar arquivos de Ajuda compilados (.chm) ou arquivos HTML no formato de Ajuda HTML. Os arquivos de Ajuda compilados fornecem links de índice, índice, pesquisa e palavras-chave nas páginas. Os atalhos funcionam apenas em arquivos de Ajuda compilados. Você pode gerar arquivos HTML Help 1.x com uma ferramenta gratuita da Microsft chamada HTML Help Workshop.

Uma maneira fácil de mostrar um arquivo de ajuda compilado em uma segunda janela:

C#

Help.ShowHelp(this, helpProviderMain.HelpNamespace);

VB.NET

Help.ShowHelp(Me, hlpProviderMain.HelpNamespace)

Manipulando o evento HelpRequested de controles e formulário #

Quando um usuário pressiona F1 em um controle ou clica no botão Ajuda do formulário (?) e então clica em um controle o [HelpRequested](https://msdn .microsoft.com/en-us/library/system.windows.forms.control.helprequested(v=vs.110).aspx) será gerado.

Você pode manipular esse evento para fornecer uma ação personalizada quando o usuário solicitar ajuda para controles ou formulários.

O HelpRequested dá suporte ao mecanismo de bolha. Ele é acionado para seu controle ativo e se você não manipular o evento e não definir [Handled](https://msdn.microsoft.com/en-us/library/system.windows.forms.helpeventargs.handled( v=vs.110).aspx) de seu evento arg para true, então ele borbulha para a hierarquia de controle pai até o formulário.

Por exemplo, se você manipular o evento HelpRequested do formulário como abaixo, então quando você pressionar F1 uma caixa de mensagem aparecerá e mostrará o nome do controle ativo, mas para textBox1 ele mostrará um mensagem:

private void Form1_HelpRequested(object sender, HelpEventArgs hlpevent)
{
    var c = this.ActiveControl;
    if(c!=null)
        MessageBox.Show(c.Name);
}
private void textBox1_HelpRequested(object sender, HelpEventArgs hlpevent)
{
    hlpevent.Handled = true;
    MessageBox.Show("Help request handled and will not bubble up");
}

Você pode executar qualquer outra ação personalizada, como navegar para uma URL ou mostrar um arquivo CHM usando [Help](https://msdn.microsoft.com/en-us/library/system.windows.forms.help(v =vs.110).aspx).

Crie um botão de ajuda personalizado que funciona como um botão de ajuda de formulário padrão #

Se você tem um Form com MinimizeBox e MaximizeBox definido como true, então você não pode mostrar o botão Ajuda na barra de título do Form e perderá o recurso de clicar no botão de ajuda para convertê-lo para cursor de ajuda para poder clicar nos controles para mostrar a ajuda.

Você pode fazer um item de menu em MenuStrip agir como um botão de ajuda padrão. Para fazer isso, adicione um MenuStrip ao formulário e adicione um ToolStripMenuItem a ele, então manipule o evento Click do item:

private const int WM_SYSCOMMAND = 0x0112;
private const int SC_CONTEXTHELP = 0xF180;
[System.Runtime.InteropServices.DllImport("user32.dll")]
static extern IntPtr SendMessage(IntPtr hWnd, int Msg, int wParam, int lParam);
private void helpToolStripMenuItem_Click(object sender, EventArgs e)
{
    SendMessage(this.Handle, WM_SYSCOMMAND, SC_CONTEXTHELP, 0);
}

Nota: Se você quiser fazer isso usando um Button, você também precisa definir button1.Capture = false; antes de enviar a mensagem. Mas não é necessário para um ToolStripMenuItem.

Então, quando você clicar no menu de ajuda, o cursor será alterado para o cursor ? e agirá como quando você clica no botão Ajuda padrão:

Mostrar ajuda para MessageBox #

Você pode fornecer ajuda para a caixa de mensagens de diferentes maneiras. Você pode configurar um MessageBox para mostrar um botão Help ou não. Além disso, você pode configurar o MessageBox de forma que quando o usuário solicitar ajuda clicando no botão Ajuda ou pressionando F1, ele mostre um arquivo CHM ou navegue até uma URL ou execute uma ação personalizada. Aqui estão alguns exemplos neste tópico.

Em todos os exemplos abaixo, o MessageBox ficaria assim:

Mostra um arquivo CHM e navega até uma palavra-chave (índice) #

MessageBox.Show("Some Message", "Title", MessageBoxButtons.YesNoCancel,
    MessageBoxIcon.Question, MessageBoxDefaultButton.Button3, 0,
    "help.chm", HelpNavigator.KeywordIndex, "SomeKeyword");

Mostra um arquivo CHM e navega para um tópico #

MessageBox.Show("Some Message", "Title", MessageBoxButtons.YesNoCancel,
    MessageBoxIcon.Question, MessageBoxDefaultButton.Button3, 0,
    "help.chm", HelpNavigator.Topic, "/SomePath/SomePage.html");

Mostra um arquivo CHM e navega pela primeira página de ajuda no índice #

MessageBox.Show("Some Message", "Title", MessageBoxButtons.YesNoCancel,
    MessageBoxIcon.Question, MessageBoxDefaultButton.Button3, 0,
    "help.chm");

Abra o navegador padrão e navegue até um URL #

MessageBox.Show("Some Message", "Title", MessageBoxButtons.YesNoCancel,
    MessageBoxIcon.Question, MessageBoxDefaultButton.Button3, 0,
    "http://example.com");

Execute a ação personalizada ao pressionar o botão Ajuda ou a tecla F1 #

Nesse caso, você deve manipular o evento HelpRequested do pai de MessageBox e execute a operação personalizada:

private void Form1_HelpRequested(object sender, HelpEventArgs hlpevent)
{
    // Perform custom action, for example show a custom help form
    var f = new Form();  
    f.ShowDialog();
}

Então você pode mostrar o MessageBox com o botão Ajuda:

MessageBox.Show("Some Message", "Title", MessageBoxButtons.YesNoCancel,
    MessageBoxIcon.Question, MessageBoxDefaultButton.Button3, 0, true);

Ou mostre sem o botão Ajuda:

MessageBox.Show("Some Message", "Title", MessageBoxButtons.YesNoCancel,
    MessageBoxIcon.Question, MessageBoxDefaultButton.Button3, 0, false);

Mostrar Ajuda para CommonDialogs #

Você pode fornecer ajuda para OpenFileDialog, SaveFileDialog e ColorDialog. Para fazer isso, defina a propriedade ShowHelp da caixa de diálogo como true e manipule [HelpRequest](https://msdn.microsoft.com/en-us/library/system.windows.forms.commondialog.helprequest(v= vs.110).aspx) para diálogo:

void openFileDialog1_HelpRequest(object sender, EventArgs e)
{
    //Perform custom action
    Help.ShowHelp(this, "Http://example.com");
}

Observação

  • O evento será gerado somente se você definir ShowHelp como true.
  • O evento será gerado apenas clicando no botão ‘Ajuda’ e não será gerado usando a tecla F1.

Mostrar Ajuda usando a classe Help #

Você pode usar a classe Help no código, para fornecer esses tipos de ajuda:

  • Mostrar um pop-up de ajuda para um controle
  • Abra um arquivo CHM com base no contexto (Mostrar tabela de conteúdo, Mostrar uma palavra-chave ou índice, mostrar um tópico)
  • Navegue até um URL usando o navegador padrão

Mostrar janela pop-up de ajuda #

Você pode usar Help.ShowPopup para exibir uma ajuda janela de pop-up:

private void control_MouseClick(object sender, MouseEventArgs e)
{
    var c = (Control)sender;
    var help = "Lorem ipsum dolor sit amet, consectetur adipiscing elit, " +
               "sed do eiusmod tempor incididunt ut labore et dolore magna aliqua."
    if (c != null)
        Help.ShowPopup(c, "Lorem ipsum dolor sit amet.", c.PointToScreen(e.Location));
}

Ele mostrará esse pop-up de ajuda no local do ponteiro do mouse:

Mostrar arquivo de ajuda do CHM #

Você pode usar diferentes sobrecargas do método Help.ShowHelp, para mostrar um arquivo CHM e navegar para uma palavra-chave, um tópico, índice ou índice:

Mostrar índice de ajuda #

Help.ShowHelp(this, "Help.chm");

Mostrar ajuda para palavra-chave específica (índice) #

Help.ShowHelp(this, "Help.chm", HelpNavigator.Index, "SomeKeyword");

Mostrar ajuda para um tópico específico #

Help.ShowHelp(this, "Help.chm", HelpNavigator.Topic, "/SomePath/SomePage.html");

Mostrar URL #

Você pode mostrar qualquer URL no navegador padrão usando o método ShowHelp:

Help.ShowHelp(this, "Http://example.com");

Mostrar botão de ajuda na barra de título do formulário #

Você pode mostrar um botão de ajuda na barra de título de um ‘Formulário’. Para isso, você deve:

  1. Defina a propriedade HelpButton do formulário como true .
  2. Defina MinimizeBox e MaximizeBox para false.

Em seguida, um botão de ajuda aparecerá na barra de título do Formulário:

Além disso, quando você clicar no botão Ajuda, o cursor será alterado para um cursor ?:

Então se você clicar em um Control ou Form, o evento HelpRequested será gerado e também se você configurou um HelpProvider, a ajuda para o controle será mostrada usando HelpProvider.

Manipulando o evento HelpButtonClicked do Form #

Você pode detectar quando um usuário clicou em um HelpButton na barra de título do formulário manipulando [HelpButtonClicked](https://msdn.microsoft.com/en-us/library/system.windows.forms.form. helpbuttonclicked(v=vs.110).aspx). Você pode deixar o evento continuar ou cancelá-lo definindo a propriedade Cancel de seus argumentos de evento para true.

private void Form1_HelpButtonClicked(object sender, CancelEventArgs e)
{
    e.Cancel = true;
    //Perform some custom action
    MessageBox.Show("Some Custom Help");
}
