Botão #

Os botões são um dos controles mais simples e usados ​​principalmente para executar algum código quando o usuário quiser.

Aqui temos um caso bem simples, mostrar uma caixa de mensagem quando um botão é clicado. Adicionamos um botão a um formulário, nomeamos como cmdShowMessage conforme usado no código para identificar o objeto e definimos o texto dos botões para Mostrar mensagem.

Só precisamos clicar duas vezes no botão no designer visual e o Visual Studio gerará o código para o evento de clique. Agora só precisamos adicionar o código para o MessageBox lá:

private void cmdShowMessage_Click(object sender, EventArgs e)
{
  MessageBox.Show("Hello world!");
}

Se executarmos o programa agora e clicarmos no botão veremos a mensagem aparecendo:

Caixa de texto #

TextBoxes permitem que o usuário insira dados no programa.

Vamos modificar o formulário e adicionar uma caixa de texto para que a caixa de mensagem nos mostre a mensagem que o usuário deseja. Agora nosso formulário se parece com:

E, em seguida, modifique o evento de clique do botão para usar o texto da caixa de texto:

private void cmdShowMessage_Click(object sender, EventArgs e)
{
  string UserText = txtUserMessage.Text;
  MessageBox.Show(UserText);
}

Como você pode ver estamos usando a propriedade .Text do Textbox que é o texto contido no texbox.

Se executarmos o programa, poderemos escrever na caixa de texto. Ao clicar no botão MessageBox mostrará o texto que escrevemos:

Caixa combo #

ComboBoxes permitem que o usuário escolha uma das várias opções fornecidas pelo desenvolvedor.

Vamos modificar o formulário e adicionar um combobox para que o messagebox nos mostre a mensagem que o usuário deseja de uma lista que forneceremos.

Depois de adicionar o combo ao formulário, agora adicionamos uma lista de opções ao combo. Para isso, precisamos modificar a propriedade Items:

Agora precisamos modificar o código do evento click:

private void cmdShowMessage_Click(object sender, EventArgs e)
{
  string OptionText = cboSelectOption.SelectedItem.ToString();
  MessageBox.Show(OptionText);
}

Como você pode ver usamos a propriedade SelectedItem, ela contém o objeto da opção selecionada. Como precisamos de uma string para mostrar e o compilador não sabe se o objeto é ou não uma string, precisamos usar o método ToString().

Se executarmos o programa poderemos escolher a opção que preferirmos e quando clicarmos no botão aparecerá a caixa de mensagem:

Para ser notificado quando um usuário selecionar um item da caixa de combinação, use o evento SelectionChangeCommitted. Poderíamos usar o evento SelectedIndexChanged, mas isso também é gerado quando alteramos programaticamente o item selecionado na caixa de combinação.

Caixa de seleção #

Checkbox é um controle que permite ao usuário obter valores booleanos do usuário para uma pergunta específica como “Você está bem?”.

Possui um evento chamado CheckedChanged, que ocorre sempre que a propriedade check é alterada.

Aqui está um CheckBox que tem uma pergunta “Is Checked?”.

Recebemos este MessageBox do evento CheckedChanged,

private void checkBox1_CheckedChanged(object sender, EventArgs e)
        {
            bool IsChecked = checkBox1.Checked; 
            MessageBox.Show(IsChecked.ToString());
        }

Se CheckBox estiver marcado -> a variável IsChecked será true.

Se CheckBox não estiver marcado -> a variável IsChecked será false.

Caixa de Listagem #

Listbox é um controle que pode conter uma coleção de objetos. Listbox é semelhante ao Combobox mas em Combobox; Apenas os itens selecionados são visíveis para o usuário. Em Caixa de listagem; todos os itens são visíveis para o usuário.

Como adicionar itens ao ListBox?

private void Form3_Load(object sender, EventArgs e)
        {
            string test = "Trial";
            string test2 = "45";
            int numberTest = 43;
            decimal decimalTest = 130;
            
            listBox1.Items.Add(test);
            listBox1.Items.Add(test2);
            listBox1.Items.Add(numberTest);
            listBox1.Items.Add(decimalTest);
        }

Resultado;

Ou datasources pode ser fornecido,

private void Form3_Load(object sender, EventArgs e)
        {
            List<string> TestList = new List<string> { "test1", "test2", "test3", "44", "55" };
            listBox1.DataSource = TestList;
        }

NumericUpDown #

NumericUpDown é um controle que se parece com TextBox. Este controle permite que o usuário exiba/selecione o número de um intervalo. As setas para cima e para baixo estão atualizando o valor da caixa de texto.

Controle parece;

No intervalo Form_Load pode ser definido.

private void Form3_Load(object sender, EventArgs e)
    {
        numericUpDown1.Maximum = 10;
        numericUpDown1.Minimum = -10; 
    }

Resultado;

UpDownAlign definirá a posição das setas;

 private void Form3_Load(object sender, EventArgs e)
        {
            numericUpDown1.UpDownAlign = LeftRightAlignment.Left;
            
        }

Resultado;

O método UpButton() aumenta o número do controle. (pode ser chamado de qualquer lugar. Eu usei um ‘botão’ para chamá-lo.)

private void button1_Click(object sender, EventArgs e)
        {
            numericUpDown1.UpButton();
        }

**Resultado

O método DownButton() diminui o número do controle. (pode ser chamado de qualquer lugar. Usei um ‘botão’ para chamá-lo novamente.)

 private void button2_Click(object sender, EventArgs e)
        {
            numericUpDown1.DownButton();
        }

Resultado;

Eventos úteis #

ValorAlterado;

Esse evento funcionará quando a seta para cima ou para baixo for clicada.

private void numericUpDown1_ValueChanged(object sender, EventArgs e)
        {
            decimal result = numericUpDown1.Value; // it will get the current value
            if (result == numericUpDown1.Maximum) // if value equals Maximum value that we set in Form_Load.
            {
                label1.Text = result.ToString() + " MAX!"; // it will add "MAX" at the end of the label 
            }
            else if (result == numericUpDown1.Minimum) //  if value equals Minimum value that we set in Form_Load.
            {
                label1.Text = result.ToString() + " MIN!"; // it will add "MIN" at the end of the label 
            }
            else
            {
                label1.Text = result.ToString(); // If Not Max or Min, it will show only the number.
            }
        }

Resultado;
