Criando um aplicativo WinForms simples usando o Visual Studio #

Este exemplo mostrará como criar um projeto de aplicativo Windows Forms no Visual Studio.

Criar projeto Windows Forms #

  1. Inicie o Visual Studio.

  2. No menu Arquivo, aponte para Novo e selecione Projeto. A caixa de diálogo Novo projeto é exibida.

  3. No painel Modelos Instalados, selecione “Visual C#” ou “Visual Basic”.

  4. Acima do painel do meio, você pode selecionar a estrutura de destino na lista suspensa.

  5. No painel do meio, selecione o modelo Windows Forms Application.

  6. Na caixa de texto Nome, digite um nome para o projeto.

  7. Na caixa de texto Local, escolha uma pasta para salvar o projeto.

  8. Clique em OK.

  9. O Windows Forms Designer abre e exibe Form1 do projeto.

Adicionar controles ao formulário #

  1. Na paleta Toolbox, arraste um controle Button para o formulário.

  2. Clique no botão para selecioná-lo. Na janela Propriedades, defina a propriedade Texto para Say Hello.

Escrever código #

  1. Clique duas vezes no botão para adicionar um manipulador de eventos para o evento Click. O Editor de Código será aberto com o ponto de inserção colocado dentro da função do manipulador de eventos.

  2. Digite o seguinte código:

    C#

    MessageBox.Show("Hello, World!");

    VB.NET

    MessageBox.Show("Hello, World!")

Executar e testar #

  1. Pressione F5 para executar o aplicativo.

  2. Quando seu aplicativo estiver em execução, clique no botão para ver a mensagem “Hello, World!” mensagem.

  3. Feche o formulário para retornar ao Visual Studio.

Criando um aplicativo WinForms C# simples usando um editor de texto #

  1. Abra um editor de texto (como o Bloco de Notas) e digite o código abaixo:
    using System;
    using System.ComponentModel;
    using System.Drawing;
    using System.Windows.Forms;

    namespace SampleApp
    {
        public class MainForm : Form
        {
            private Button btnHello;

            // The form's constructor: this initializes the form and its controls.
            public MainForm()
            {
                // Set the form's caption, which will appear in the title bar.
                this.Text = "MainForm";

                // Create a button control and set its properties.
                btnHello = new Button();
                btnHello.Location = new Point(89, 12);
                btnHello.Name = "btnHello";
                btnHello.Size = new Size(105, 30);
                btnHello.Text = "Say Hello";

                // Wire up an event handler to the button's "Click" event
                // (see the code in the btnHello_Click function below).
                btnHello.Click += new EventHandler(btnHello_Click);

                // Add the button to the form's control collection,
                // so that it will appear on the form.
                this.Controls.Add(btnHello);
            }

            // When the button is clicked, display a message.
            private void btnHello_Click(object sender, EventArgs e)
            {
                MessageBox.Show("Hello, World!");
            }

            // This is the main entry point for the application.
            // All C# applications have one and only one of these methods.
            [STAThread]
            static void Main()
            {
                Application.EnableVisualStyles();
                Application.Run(new MainForm());
            }
        }
    }

  1. Salve o arquivo em um caminho ao qual você tenha acesso de leitura/gravação. É convencional nomear o arquivo de acordo com a classe que ele contém—por exemplo, X:\MainForm.cs.

  2. Execute o compilador C# na linha de comando, passando o caminho para o arquivo de código como um argumento:

    %WINDIR%\Microsoft.NET\Framework64\v4.0.30319\csc.exe /target:winexe "X:\MainForm.cs"

Observação: Para usar uma versão do compilador C# para outras versões do .NET framework, dê uma olhada no caminho, %WINDIR%\Microsoft.NET e modifique o exemplo acima de acordo. Para obter mais informações sobre como compilar aplicativos C#, consulte https://www.wikiod.com/pt/docs/c%23/15/compile-and-run-your-first-c-sharp-program#t=201607240523313550097.

  1. Após a conclusão da compilação, um aplicativo chamado MainForm.exe será criado no mesmo diretório do seu arquivo de código. Você pode executar este aplicativo a partir da linha de comando ou clicando duas vezes nele no Explorer.

Criando um aplicativo WinForms VB.NET simples usando um editor de texto #

  1. Abra um editor de texto (como o Bloco de Notas) e digite o código abaixo:

     Imports System.ComponentModel
 Imports System.Drawing
 Imports System.Windows.Forms

 Namespace SampleApp
     Public Class MainForm : Inherits Form
         Private btnHello As Button

         ' The form's constructor: this initializes the form and its controls.
         Public Sub New()
             ' Set the form's caption, which will appear in the title bar.
             Me.Text = "MainForm"

             ' Create a button control and set its properties.
             btnHello = New Button()
             btnHello.Location = New Point(89, 12)
             btnHello.Name = "btnHello"
             btnHello.Size = New Size(105, 30)
             btnHello.Text = "Say Hello"

             ' Wire up an event handler to the button's "Click" event
             ' (see the code in the btnHello_Click function below).
             AddHandler btnHello.Click, New EventHandler(AddressOf btnHello_Click)

             ' Add the button to the form's control collection,
             ' so that it will appear on the form.
             Me.Controls.Add(btnHello)
         End Sub

         ' When the button is clicked, display a message.
         Private Sub btnHello_Click(sender As Object, e As EventArgs)
             MessageBox.Show("Hello, World!")
         End Sub

         ' This is the main entry point for the application.
         ' All VB.NET applications have one and only one of these methods.
         <STAThread> _
         Public Shared Sub Main()
             Application.EnableVisualStyles()
             Application.Run(New MainForm())
         End Sub
     End Class
 End Namespace

  2. Salve o arquivo em um caminho ao qual você tenha acesso de leitura/gravação. É convencional nomear o arquivo com a classe que ele contém—por exemplo, X:\MainForm.vb.

  3. Execute o compilador VB.NET a partir da linha de comando, passando o caminho para o arquivo de código como argumento:

     %WINDIR%\Microsoft.NET\Framework64\v4.0.30319\vbc.exe /target:winexe "X:\MainForm.vb"

    Note: To use a version of the VB.NET compiler for other .NET framework versions, take a look in the path %WINDIR%\Microsoft.NET and modify the example above accordingly. For more information on compiling VB.NET applications, see https://www.wikiod.com/pt/vb-dotnet/introducao-a-linguagem-visual-basic-net

  4. Após a conclusão da compilação, um aplicativo chamado MainForm.exe será criado no mesmo diretório do seu arquivo de código. Você pode executar este aplicativo a partir da linha de comando ou clicando duas vezes nele no Explorer.
