Arrays são tipos de dados específicos, representando uma coleção ordenada de elementos de outro tipo.

Em Go, os Arrays podem ser simples (às vezes chamados de “listas”) ou multidimensionais (como, por exemplo, um array de 2 dimensões está representando uma coleção ordenada de arrays, que contém elementos)

Sintaxe #

  • var variableName [5]ArrayType // Declarando um array de tamanho 5.
  • var variableName [2][3]ArrayType = { {Value1, Value2, Value3}, {Value4, Value5, Value6} } // Declarando um array multidimensional
  • variableName := […]ArrayType {Value1, Value2, Value3} // Declara um array de tamanho 3 (O compilador contará os elementos do array para definir o tamanho)
  • arrayName[2] // Obtendo o valor por índice.
  • arrayName[5] = 0 // Configurando o valor em index.
  • arrayName[0] // Primeiro valor do Array
  • arrayName[ len(arrayName)-1 ] // Último valor do Array

Criando matrizes #

Um array em go é uma coleção ordenada de elementos do mesmo tipo. A notação básica para representar arrays é usar [] com o nome da variável.

Criar um novo array se parece com var array = [size]Type, substituindo size por um número (por exemplo 42 para especificar que será uma lista de 42 elementos) e substituindo Type pelo tipo de os elementos que o array pode conter (por exemplo int ou string)

Logo abaixo está um exemplo de código mostrando a maneira diferente de criar um array em Go.

// Creating arrays of 6 elements of type int,
// and put elements 1, 2, 3, 4, 5 and 6 inside it, in this exact order:
var array1 [6]int = [6]int {1, 2, 3, 4, 5, 6} // classical way
var array2 = [6]int {1, 2, 3, 4, 5, 6} // a less verbose way
var array3 = [...]int {1, 2, 3, 4, 5, 6} // the compiler will count the array elements by itself

fmt.Println("array1:", array1) // > [1 2 3 4 5 6]
fmt.Println("array2:", array2) // > [1 2 3 4 5 6]
fmt.Println("array3:", array3) // > [1 2 3 4 5 6]


// Creating arrays with default values inside:
zeros := [8]int{}               // Create a list of 8 int filled with 0
ptrs := [8]*int{}               // a list of int pointers, filled with 8 nil references ( <nil> )
emptystr := [8]string{}         // a list of string filled with 8 times ""

fmt.Println("zeroes:", zeros)      // > [0 0 0 0 0 0 0 0]
fmt.Println("ptrs:", ptrs)         // > [<nil> <nil> <nil> <nil> <nil> <nil> <nil> <nil>]
fmt.Println("emptystr:", emptystr) // > [       ]  
// values are empty strings, separated by spaces,
// so we can just see separating spaces


// Arrays are also working with a personalized type
type Data struct {
    Number int
    Text   string
}

// Creating an array with 8 'Data' elements
// All the 8 elements will be like {0, ""} (Number = 0, Text = "")
structs := [8]Data{}  

fmt.Println("structs:", structs) // > [{0 } {0 } {0 } {0 } {0 } {0 } {0 } {0 }] 
// prints {0 } because Number are 0 and Text are empty; separated by a space

Índices de matriz #

Os valores de matrizes devem ser acessados ​​usando um número que especifica a localização do valor desejado na matriz. Esse número é chamado de índice.

Os índices começam em 0 e terminam em comprimento da matriz -1.

Para acessar um valor, você deve fazer algo assim: arrayName[index], substituindo “index” pelo número correspondente ao rank do valor em seu array.

Por exemplo:

var array = [6]int {1, 2, 3, 4, 5, 6}

fmt.Println(array[-42]) // invalid array index -1 (index must be non-negative)
fmt.Println(array[-1]) // invalid array index -1 (index must be non-negative)
fmt.Println(array[0]) // > 1
fmt.Println(array[1]) // > 2
fmt.Println(array[2]) // > 3
fmt.Println(array[3]) // > 4
fmt.Println(array[4]) // > 5
fmt.Println(array[5]) // > 6
fmt.Println(array[6]) // invalid array index 6 (out of bounds for 6-element array)
fmt.Println(array[42]) // invalid array index 42 (out of bounds for 6-element array)

Para definir ou modificar um valor no array, o caminho é o mesmo. Exemplo:

var array = [6]int {1, 2, 3, 4, 5, 6}

fmt.Println(array) // > [1 2 3 4 5 6]

array[0] := 6
fmt.Println(array) // > [6 2 3 4 5 6]

array[1] := 5
fmt.Println(array) // > [6 5 3 4 5 6]

array[2] := 4
fmt.Println(array) // > [6 5 4 4 5 6]

array[3] := 3
fmt.Println(array) // > [6 5 4 3 5 6]

array[4] := 2
fmt.Println(array) // > [6 5 4 3 2 6]

array[5] := 1
fmt.Println(array) // > [6 5 4 3 2 1]

Matriz Multidimensional #

Arrays multidimensionais são basicamente arrays contendo outros arrays como elementos. É representado como [sizeDim1][sizeDim2]..[sizeLastDim]type, substituindo sizeDim por números correspondentes ao comprimento da dimensão, e type pelo tipo de dados no array multidimensional.

Por exemplo, [2][3]int representa uma matriz composta por 2 submatrizes de 3 elementos do tipo int. Basicamente pode ser a representação de uma matriz de 2 linhas e 3 colunas.

Assim, podemos criar um array de números de dimensões enormes como var values ​​:= [2017][12][31][24][60]int por exemplo, se você precisar armazenar um número para cada minuto desde o ano 0.

Para acessar este tipo de array, para o último exemplo, procurando o valor de 2016-01-31 às 19:42, você acessará values[2016][0][30][19][42] (porque os índices de matriz começam em 0 e não em 1 como dias e meses)

Alguns exemplos a seguir:

// Defining a 2d Array to represent a matrix like
// 1 2 3     So with 2 lines and 3 columns;
// 4 5 6     
var multiDimArray := [2/*lines*/][3/*columns*/]int{ [3]int{1, 2, 3}, [3]int{4, 5, 6} }

// That can be simplified like this:
var simplified := [2][3]int{{1, 2, 3}, {4, 5, 6}}

// What does it looks like ?
fmt.Println(multiDimArray)
// > [[1 2 3] [4 5 6]]

fmt.Println(multiDimArray[0]) 
// > [1 2 3]    (first line of the array)

fmt.Println(multiDimArray[0][1])
// > 2          (cell of line 0 (the first one), column 1 (the 2nd one))

// We can also define array with as much dimensions as we need
// here, initialized with all zeros
var multiDimArray := [2][4][3][2]string{} 

fmt.Println(multiDimArray);
// Yeah, many dimensions stores many data
// > [[[["" ""] ["" ""]] [["" ""] ["" ""]] [["" ""] ["" ""]]]
//    [[["" ""] ["" ""]] [["" ""] ["" ""]] [["" ""] ["" ""]]]
//    [[["" ""] ["" ""]] [["" ""] ["" ""]] [["" ""] ["" ""]]]
//    [[["" ""] ["" ""]] [["" ""] ["" ""]] [["" ""] ["" ""]]]]
//   [[[["" ""] ["" ""]] [["" ""] ["" ""]] [["" ""] ["" ""]]]
//    [[["" ""] ["" ""]] [["" ""] ["" ""]] [["" ""] ["" ""]]]
//    [[["" ""] ["" ""]] [["" ""] ["" ""]] [["" ""] ["" ""]]]
//    [[["" ""] ["" ""]] [["" ""] ["" ""]] [["" ""] ["" ""]]]]

// We can set some values in the array's cells
multiDimArray[0][0][0][0] := "All zero indexes"  // Setting the first value
multiDimArray[1][3][2][1] := "All indexes to max"  // Setting the value at extreme location

fmt.Println(multiDimArray);
// If we could see in 4 dimensions, maybe we could see the result as a simple format

//    [[["" ""] ["" ""]] [["" ""] ["" ""]] [["" ""] ["" ""]]]
//    [[["" ""] ["" ""]] [["" ""] ["" ""]] [["" ""] ["" ""]]]
