Sintaxe #

  • func Marshal(v interface{}) ([]byte, erro)
  • erro func Unmarshal(data []byte, v interface{})

O pacote "encoding/json" Package json implementa a codificação e decodificação de objetos JSON em Go.

Os tipos em JSON junto com seus tipos concretos correspondentes em Go são:

Tipo JSON Vá Tipo Concreto
booleano bool
números float64 ou int
seqüência seqüência
nulo nada

Decodificação JSON básica #

json.Unmarshal do pacote "encoding/json" decodifica um valor JSON no valor apontado pela variável fornecida.

Os parâmetros são o valor a ser decodificado em []bytes e uma variável a ser usada como armazenamento para o valor desserializado. O valor retornado é um erro (em caso de falha).

encodedValue := []byte(`{"London":18,"Rome":30}`)

// generic storage for the decoded JSON
var data map[string]interface{}

// decode the value into data
// notice that we must pass the pointer to data using &data
err := json.Unmarshal(encodedValue, &data)

// check if the decoding is successful
if err != nil {
    panic(err)
}

fmt.Println(data)
map[London:18 Rome:30]

Playground

Observe como no exemplo acima sabíamos de antemão tanto o tipo da chave quanto o valor. Mas isso nem sempre é o caso. Na verdade, na maioria dos casos, o JSON contém tipos de valores mistos.

encodedValue := []byte(`{"city":"Rome","temperature":30}`)

// generic storage for the decoded JSON
var data map[string]interface{}

// decode the value into data
if err := json.Unmarshal(encodedValue, &data); err != nil {
    panic(err)
}

// if you want to use a specific value type, we need to cast it
temp := data["temperature"].(float64)
fmt.Println(temp) // 30
city := data["city"].(string)
fmt.Println(city) // "Rome"

Playground

No último exemplo acima, usamos um mapa genérico para armazenar o valor decodificado. Devemos usar uma map[string]interface{} porque sabemos que as chaves são strings, mas não sabemos antecipadamente o tipo de seus valores.

Essa é uma abordagem muito simples, mas também extremamente limitada. No mundo real, você geralmente [decodificaria um JSON em um tipo struct personalizado](https://www.wikiod.com/pt/go/json =201607220810357745507).

Usando estruturas anônimas para decodificação #

O objetivo com o uso de estruturas anônimas é decodificar apenas as informações com as quais nos importamos sem sobrecarregar nosso aplicativo com tipos que são usados ​​apenas em uma única função.

jsonBlob := []byte(`
  {
    "_total": 1,
    "_links": {
      "self": "https://api.twitch.tv/kraken/channels/foo/subscriptions?direction=ASC&limit=25&offset=0",
      "next": "https://api.twitch.tv/kraken/channels/foo/subscriptions?direction=ASC&limit=25&offset=25"
    },
    "subscriptions": [
      {
        "created_at": "2011-11-23T02:53:17Z",
        "_id": "abcdef0000000000000000000000000000000000",
        "_links": {
          "self": "https://api.twitch.tv/kraken/channels/foo/subscriptions/bar"
        },
        "user": {
          "display_name": "bar",
          "_id": 123456,
          "name": "bar",
          "staff": false,
          "created_at": "2011-06-16T18:23:11Z",
          "updated_at": "2014-10-23T02:20:51Z",
          "logo": null,
          "_links": {
            "self": "https://api.twitch.tv/kraken/users/bar"
          }
        }
      }
    ]
  }
`)

var js struct {
    Total int `json:"_total"`
    Links struct {
        Next string `json:"next"`
    } `json:"_links"`
    Subs []struct {
        Created string `json:"created_at"`
        User    struct {
            Name string `json:"name"`
            ID   int    `json:"_id"`
        } `json:"user"`
    } `json:"subscriptions"`
}

err := json.Unmarshal(jsonBlob, &js)
if err != nil {
    fmt.Println("error:", err)
}
fmt.Printf("%+v", js)

Resultado: {Total:1 links:{Próximo:https://api.twitch.tv/kraken/channels/foo/subscriptions?direction=ASC&limit=25&offset=25} Inscrições:[{Created:2011-11-23T02:53: 17Z Usuário:{Nome:ID da barra:123456}}]}

Playground

Para o caso geral, consulte também: https://www.wikiod.com/pt/go/json

Codificação JSON Básica #

json.Marshal do pacote "encoding/json" codifica um valor para JSON.

O parâmetro é o valor a ser codificado. Os valores retornados são uma matriz de bytes representando a entrada codificada em JSON (em caso de sucesso) e um erro (em caso de falha).

decodedValue := []string{"foo", "bar"}

// encode the value
data, err := json.Marshal(decodedValue)

// check if the encoding is successful
if err != nil {
    panic(err)
}

// print out the JSON-encoded string
// remember that data is a []byte
fmt.Println(string(data))
// "["foo","bar"]"

Playground

Veja alguns exemplos básicos de codificação para tipos de dados integrados:

var data []byte

data, _ = json.Marshal(1)
fmt.Println(string(data))
// 1

data, _ = json.Marshal("1")
fmt.Println(string(data))
// "1"

data, _ = json.Marshal(true)
fmt.Println(string(data))
// true

data, _ = json.Marshal(map[string]int{"London": 18, "Rome": 30})
fmt.Println(string(data))
// {"London":18,"Rome":30}

Playground

A codificação de variáveis ​​simples é útil para entender como a codificação JSON funciona em Go. No entanto, no mundo real, você provavelmente codificará dados mais complexos armazenados em structs.

Decodificando dados JSON de um arquivo #

Os dados JSON também podem ser lidos de arquivos.

Vamos supor que temos um arquivo chamado data.json com o seguinte conteúdo:

[
    {
      "Name" : "John Doe",
      "Standard" : 4
    },
    {
      "Name" : "Peter Parker",
      "Standard" : 11
    },
    {
      "Name" : "Bilbo Baggins",
      "Standard" : 150
    }
]

O exemplo a seguir lê o arquivo e decodifica o conteúdo:

package main

import (
    "encoding/json"
    "fmt"
    "log"
    "os"
)

type Student struct {
    Name     string
    Standard int `json:"Standard"`
}

func main() {
    // open the file pointer
    studentFile, err := os.Open("data.json")
    if err != nil {
        log.Fatal(err)
    }
    defer studentFile.Close()

    // create a new decoder
    var studentDecoder *json.Decoder = json.NewDecoder(studentFile)
    if err != nil {
        log.Fatal(err)
    }

    // initialize the storage for the decoded data
    var studentList []Student
    
    // decode the data
    err = studentDecoder.Decode(&studentList)
    if err != nil {
        log.Fatal(err)
    }

    for i, student := range studentList {
        fmt.Println("Student", i+1)
        fmt.Println("Student name:", student.Name)
        fmt.Println("Student standard:", student.Standard)
    }
}

O arquivo data.json deve estar no mesmo diretório do programa executável Go. Leia Documentação de E/S do arquivo Go para obter mais informações sobre como trabalhar com arquivos em Go.

Configurando campos de estrutura JSON #

Considere o seguinte exemplo:

type Company struct {
    Name     string
    Location string
}

Ocultar/Pular Certos Campos #

Para exportar Revenue e Sales, mas ocultá-los da codificação/decodificação, use json:"-" ou renomeie a variável para começar com uma letra minúscula. Observe que isso impede que a variável fique visível fora do pacote.

type Company struct {
    Name     string `json:"name"`
    Location string `json:"location"`
    Revenue  int    `json:"-"`
    sales    int
}

Ignorar campos vazios #

Para evitar que Location seja incluído no JSON quando estiver definido como seu valor zero, adicione ,omitempty à tag json.

type Company struct {
    Name     string `json:"name"`
    Location string `json:"location,omitempty"`
}

Estruturas de marshaling com campos privados #

Como um bom desenvolvedor, você criou a seguinte estrutura com campos exportados e não exportados:

type MyStruct struct {
    uuid string    
    Name string
}

Exemplo no Playground: https://play.golang.org/p/Zk94Il2ANZ

Agora você quer Marshal() este struct em JSON válido para armazenamento em algo como etcd. No entanto, como o uuid não é exportado, o json.Marshal() o ignora. O que fazer? Use uma estrutura anônima e a interface json.MarshalJSON()! Aqui está um exemplo:

type MyStruct struct {
    uuid string    
    Name string
}

func (m MyStruct) MarshalJSON() ([]byte, error {
    j, err := json.Marshal(struct {
        Uuid string
        Name string
    }{
        Uuid: m.uuid,
        Name: m.Name,
    })
    if err != nil {
           return nil, err
    }
    return j, nil
}

Exemplo no Playground: https://play.golang.org/p/Bv2k9GgbzE

Codificação/decodificação usando estruturas Go #

Vamos supor que temos o seguinte struct que define um tipo City:

type City struct {  
    Name string  
    Temperature int  
}

Podemos codificar/decodificar valores de cidade usando o pacote encoding/json.

Antes de tudo, precisamos usar os metadados Go para informar ao codificador a correspondência entre os campos struct e as chaves JSON.

type City struct {  
    Name string `json:"name"`  
    Temperature int `json:"temp"`  
    // IMPORTANT: only exported fields will be encoded/decoded  
    // Any field starting with a lower letter will be ignored  
}

Para manter este exemplo simples, vamos declarar uma correspondência explícita entre os campos e as chaves. No entanto, você pode usar várias variantes dos metadados json: como explicado nos documentos.

IMPORTANTE: Somente [campos exportados](https://www.wikiod.com/pt/go/estruturas#Campos exportados x não exportados (privado x público)) (campos com nome em maiúsculas) serão serializados/desserializados. Por exemplo, se você nomear o campo temperature ele será ignorado mesmo se você definir os metadados json.

Codificação #

Para codificar uma estrutura City, use json.Marshal como no exemplo básico:

// data to encode  
city := City{Name: "Rome", Temperature: 30}  
 
// encode the data  
bytes, err := json.Marshal(city)  
if err != nil {  
    panic(err)  
}  
 
fmt.Println(string(bytes))  
// {"name":"Rome","temp":30}

Playground

Decodificação #

Para decodificar uma estrutura City, use json.Unmarshal como no exemplo básico:

// data to decode  
bytes := []byte(`{"name":"Rome","temp":30}`)  
 
// initialize the container for the decoded data  
var city City  
 
// decode the data  
// notice the use of &city to pass the pointer to city  
if err := json.Unmarshal(bytes, &city); err != nil {  
    panic(err)  
}  
 
fmt.Println(city)  
// {Rome 30}
