Códigos de status #

1xx: Informações #

Mensagem Descrição Descrição longa
100 Continuar O servidor recebeu os cabeçalhos da solicitação, e o cliente deve continuar enviando o corpo da solicitação
101 Comutação de protocolos O solicitante pediu ao servidor para alternar protocolos
103 Checkpoint Usado na proposta de solicitações retomáveis ​​para retomar solicitações PUT ou POST abortadas

2xx: Sucesso #

Mensagem Descrição Descrição longa
200 OK A solicitação está OK (esta é a resposta padrão para solicitações HTTP bem-sucedidas)
201 Criado A solicitação foi atendida e um novo recurso foi criado
202 Aceito A solicitação foi aceita para processamento, mas o processamento não foi concluído
203 Informações não autorizadas A solicitação foi processada com sucesso, mas está retornando informações que podem ser de outra fonte
204 Sem conteúdo A solicitação foi processada com sucesso, mas não está retornando nenhum conteúdo
205 Redefinir conteúdo A solicitação foi processada com sucesso, mas não está retornando nenhum conteúdo e exige que o solicitante redefina a visualização do documento
206 Conteúdo parcial O servidor está entregando apenas parte do recurso devido a um cabeçalho de intervalo enviado pelo cliente

3xx: Redirecionamento #

Mensagem Descrição Descrição longa
300 Múltiplas escolhas Uma lista de links. O usuário pode selecionar um link e ir para esse local. Máximo de cinco endereços
301 Movido permanentemente A página solicitada foi movida para um novo URL
302 Encontrado A página solicitada foi movida temporariamente para um novo URL
303 Veja Outros A página solicitada pode ser encontrada em uma URL diferente
304 Not Modified Indica que a página solicitada não foi modificada desde a última solicitação
306 Switch Proxy Não é mais usado
307 Redirecionamento Temporário A página solicitada foi movida temporariamente para um novo URL
308 Resume Incomplete Usado na proposta de solicitações retomáveis ​​para retomar solicitações PUT ou POST abortadas

4xx: Erro do cliente #

Mensagem Descrição Descrição longa
400 Bad Request A solicitação não pode ser atendida devido à sintaxe incorreta
401 Não autorizado A solicitação foi legal, mas o servidor está se recusando a responder a ela. Para uso quando a autenticação é possível, mas falhou ou ainda não foi fornecida
402 Pagamento obrigatório Reservado para uso futuro
403 Proibido A solicitação foi legal, mas o servidor se recusa a responder a ela
404 Não encontrado A página solicitada não foi encontrada, mas pode estar disponível novamente no futuro
405 Método não permitido Uma solicitação foi feita de uma página usando um método de solicitação não suportado por essa página
406 Não Aceitável O servidor só pode gerar uma resposta que não seja aceita pelo cliente
407 Autenticação Proxy Necessária O cliente deve primeiro autenticar-se com o proxy
408 Request Timeout O servidor expirou aguardando a solicitação
409 Conflito A solicitação não pôde ser concluída devido a um conflito na solicitação
410 Desaparecido A página solicitada não está mais disponível
411 Comprimento Obrigatório O “Comprimento do Conteúdo” não está definido. O servidor não aceitará o pedido sem ele
412 Precondition Failed A pré-condição dada na requisição avaliada como falsa pelo servidor
413 Request Entity Too Large O servidor não aceitará o pedido, porque a entidade do pedido é muito grande
414 Request-URI Too Long O servidor não aceitará o pedido, porque o URL é muito longo. Ocorre quando você converte uma solicitação POST em uma solicitação GET com informações de consulta longas
415 Tipo de mídia não suportado O servidor não aceitará a solicitação, porque o tipo de mídia não é suportado
416 Requested Range Not Satisfiable O cliente solicitou uma parte do arquivo, mas o servidor não pode fornecer essa parte
417 Expectation Failed O servidor não pode atender aos requisitos do campo Expect request-header

5xx: Erro do servidor #

Mensagem Descrição Descrição longa
500 Erro interno do servidor Uma mensagem de erro genérica, dada quando nenhuma mensagem mais específica é adequada
501 Não implementado O servidor não reconhece o método de solicitação ou não tem a capacidade de atender à solicitação
502 Bad Gateway O servidor estava agindo como um gateway ou proxy e recebeu uma resposta inválida do servidor upstream
503 Serviço indisponível O servidor está indisponível no momento (sobrecarregado ou inativo)
504 Tempo limite do gateway O servidor estava agindo como um gateway ou proxy e não recebeu uma resposta oportuna do servidor upstream
505 Versão HTTP não suportada O servidor não suporta a versão do protocolo HTTP usada na solicitação
511 Autenticação de rede necessária O cliente precisa se autenticar para obter acesso à rede
