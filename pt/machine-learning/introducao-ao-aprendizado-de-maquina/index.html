Instalação ou configuração usando Python #

1) scikit aprender

scikit-learn é um módulo Python para aprendizado de máquina construído sobre SciPy e distribuído sob a licença BSD de 3 cláusulas. Ele apresenta vários algoritmos de classificação, regressão e clustering, incluindo máquinas de vetor de suporte, florestas aleatórias, aumento de gradiente, k-means e DBSCAN, e foi projetado para interoperar com as bibliotecas numéricas e científicas do Python NumPy e SciPy.

A versão estável atual do scikit-learn requer:

  • Python (>= 2.6 ou >= 3.3),
  • NumPy (>= 1.6.1),
  • SciPy (>= 0,9).

Para a maioria das instalações, o gerenciador de pacotes python pip pode instalar o python e todas as suas dependências:

pip install scikit-learn

No entanto, para sistemas linux, é recomendado usar o gerenciador de pacotes conda para evitar possíveis processos de compilação

conda install scikit-learn

Para verificar se você tem scikit-learn, execute no shell:

python -c 'import sklearn; print(sklearn.__version__)'

Instalação Windows e Mac OSX:

Canopy e Anaconda ambos trazem uma versão recente do scikit-learn, além de um grande conjunto de bibliotecas científicas python para Windows, Mac OSX (também relevante para Linux).

Repositório oficial do código-fonte: https://github.com/scikit-learn/scikit-learn

2) Plataforma Numenta para Computação Inteligente

A Numenta Platform for Intelligent Computing (NuPIC) é uma plataforma de inteligência de máquina que implementa os algoritmos de aprendizado HTM. HTM é uma teoria computacional detalhada do neocórtex. No núcleo do HTM estão algoritmos de aprendizado contínuo baseados em tempo que armazenam e recuperam padrões espaciais e temporais. NuPIC é adequado para uma variedade de problemas, particularmente detecção de anomalias e previsão de fontes de dados de streaming.

Os binários NuPIC estão disponíveis para:

Linux x86 64 bits OS X 10.9 OS X 10.10 Windows 64 bits

As dependências a seguir são necessárias para instalar o NuPIC em todos os sistemas operacionais.

  • Python 2.7
  • pip>=8.1.2
  • ferramentas de configuração>=25.2.0
  • roda>=0,29,0
  • numpy
  • Compilador C++ 11 como gcc (4.8+) ou clang

Requisitos adicionais do OS X:

  • Ferramentas de linha de comando Xcode

Execute o seguinte para instalar o NuPIC:

pip install nupic

Repo oficial do código-fonte: https://github.com/numenta/nupic

3) niaprender

Nilearn é um módulo Python para aprendizado estatístico rápido e fácil em dados de NeuroImaging. Ele aproveita a caixa de ferramentas Python scikit-learn para estatísticas multivariadas com aplicativos como modelagem preditiva, classificação, decodificação ou análise de conectividade.

As dependências necessárias para usar o software são:

  • Python >= 2.6,
  • ferramentas de configuração
  • Numpy >= 1.6.1
  • SciPy >= 0,9
  • Scikit-learn >= 0,14,1
  • Nibabel >= 1.1.0

Se você estiver usando as funcionalidades de plotagem do nilearn ou executando os exemplos, é necessário matplotlib >= 1.1.1.

Se você deseja executar os testes, precisa de nariz >= 1.2.1 e cobertura >= 3.6.

Primeiro, certifique-se de ter instalado todas as dependências listadas acima. Em seguida, você pode instalar o nilearn executando o seguinte comando em um prompt de comando:

pip install -U --user nilearn

Repo oficial do código-fonte: https://github.com/nilearn/nilearn/

4) Usando o Anaconda

Muitas bibliotecas científicas Python estão prontamente disponíveis no Anaconda. Você pode obter os arquivos de instalação aqui. Por um lado, usando o Anaconda, você não precisa instalar e configurar muitos pacotes, é licenciado em BSD e possui processo de instalação trivial, disponível para Python 3 e Python 2, enquanto, por outro lado, oferece menos flexibilidade. Como exemplo, alguns pacotes python de aprendizado profundo de última geração podem usar uma versão diferente do numpy do que o Anaconda instalado. No entanto, essa desvantagem pode ser tratada usando outra instalação python separadamente (no linux e no MAC, o padrão, por exemplo).

A configuração do Anaconda solicita a seleção do local de instalação e também solicita a opção de adição PATH. Se você adicionar o Anaconda ao seu PATH, espera-se que o seu sistema operacional encontre o Anaconda Python como padrão. Portanto, modificações e instalações futuras estarão disponíveis apenas para esta versão do Python.

Para deixar claro, após a instalação do Anaconda e você adicioná-lo ao PATH, usando o Ubuntu 14.04 via terminal se você digitar

python

Voila, Anaconda Python é seu Python padrão, você pode começar a usar muitas bibliotecas imediatamente. No entanto, se você quiser usar seu antigo Python

/usr/bin/python

Para encurtar a história, o Anaconda é uma das maneiras mais rápidas de iniciar o aprendizado de máquina e a análise de dados com o Python.

Instalação ou configuração usando a linguagem R #

[Packages](https://www.wikiod.com/pt/r/instalando-pacotes#Baixe e instale pacotes de repositórios) são coleções de funções R, dados e compilados código em um formato bem definido. Repositórios públicos (e privados) são usados ​​para hospedar coleções de pacotes R. A maior coleção de pacotes R está disponível no CRAN. Alguns dos pacotes de aprendizado de máquina R mais populares são os seguintes, entre outros:

1) parte

Descrição: Particionamento recursivo para classificação, regressão e árvores de sobrevivência. Uma implementação da maioria das funcionalidades do livro de 1984 de Breiman, Friedman, Olshen e Stone.

Ele pode ser instalado a partir do CRAN usando o seguinte código:

install.packages("rpart")

Carregue o pacote:

library(rpart)

Fonte oficial: https://cran.r-project.org/web/packages/rpart/index.html

2) e1071

Descrição: Funções para análise de classes latentes, transformada de Fourier de curta duração, agrupamento difuso, máquinas de vetor de suporte, cálculo de caminho mais curto, agrupamento ensacado, classificador Bayes ingênuo etc.

Instalação do CRAN:

install.packages("e1071")

Carregando o pacote:

library(e1071)

Fonte oficial: https://cran.r-project.org/web/packages/e1071/index.html

3) Forest aleatória

Descrição: Classificação e regressão baseada em uma floresta de árvores usando entradas aleatórias.

Instalação do CRAN:

install.packages("randomForest")

Carregando o pacote:

library(randomForest)

Fonte oficial: https://cran.r-project.org/web/packages/randomForest/index.html

4) acento circunflexo

Descrição: Funções diversas para treinamento e plotagem de modelos de classificação e regressão.

Instalação do CRAN:

install.packages("caret")

Carregando o pacote:

library(caret)

Fonte oficial: https://cran.r-project.org/web/packages/caret/index.html
