Regressão linear #

Visto que o Aprendizagem Supervisionada consiste em uma variável alvo ou resultado (ou variável dependente) que deve ser prevista a partir de um determinado conjunto de preditores (variáveis ​​independentes). Usando esse conjunto de variáveis, geramos uma função que mapeia as entradas para as saídas desejadas. O processo de treinamento continua até que o modelo atinja um nível desejado de precisão nos dados de treinamento.

Portanto, existem muitos exemplos de algoritmos de Aprendizado Supervisionado, então neste caso eu gostaria de focar em Regressão Linear

Regressão linear É usado para estimar valores reais (custo das casas, número de ligações, vendas totais etc.) com base em variáveis ​​contínuas. Aqui, estabelecemos relação entre variáveis ​​independentes e dependentes ajustando uma melhor linha. Essa linha de melhor ajuste é conhecida como linha de regressão e representada por uma equação linear Y= a *X + b.

A melhor maneira de entender a regressão linear é reviver essa experiência da infância. Digamos que você peça a uma criança na quinta série para organizar as pessoas em sua classe aumentando a ordem de peso, sem perguntar seus pesos! O que você acha que a criança vai fazer? Ele / ela provavelmente olharia (analisaria visualmente) a altura e a constituição das pessoas e as organizaria usando uma combinação desses parâmetros visíveis.

Isso é regressão linear na vida real! A criança realmente descobriu que altura e constituição estariam correlacionadas ao peso por uma relação, que se parece com a equação acima.

Nesta equação:

Y – Dependent Variable
a – Slope
X – Independent variable
b – Intercept

Esses coeficientes aeb são derivados com base na minimização da soma da diferença quadrada da distância entre os pontos de dados e a linha de regressão.

Veja o exemplo abaixo. Aqui nós identificamos a linha de melhor ajuste com equação linear y=0,2811x+13,9. Agora, usando esta equação, podemos encontrar o peso, sabendo a altura de uma pessoa.

A Regressão Linear é basicamente de dois tipos: Regressão Linear Simples e Regressão Linear Múltipla. A Regressão Linear Simples é caracterizada por uma variável independente. E, Regressão Linear Múltipla (como o nome sugere) é caracterizada por múltiplas (mais de 1) variáveis ​​independentes. Ao encontrar a linha de melhor ajuste, você pode ajustar uma regressão polinomial ou curvilínea. E estes são conhecidos como regressão polinomial ou curvilínea.

Apenas uma dica sobre como implementar a regressão linear em Python

#Import Library
#Import other necessary libraries like pandas, numpy...

#Load Train and Test datasets
#Identify feature and response variable(s) and values must be numeric and numpy arrays

Classificação #

Imagine que um sistema queira detectar maçãs e laranjas em uma cesta de frutas. O sistema pode colher uma fruta, extrair alguma propriedade dela (por exemplo, peso dessa fruta).

Suponha que o sistema tenha um professor! que ensina ao sistema quais objetos são maçãs e quais são laranjas. Este é um exemplo de um problema supervisionado de classificação. É supervisionado porque rotulamos exemplos. É classificação porque a saída é uma previsão de qual classe nosso objeto pertence também.

Neste exemplo consideramos 3 características (propriedades/variáveis ​​explicativas):

  1. é o peso da fruta selecionada maior que 0,5 grama
  2. é tamanho maior que 10 cm
  3. a cor é vermelha

(0 significa Não e 1 significa Sim)

Então, para representar uma maçã/laranja, temos uma série (chamada vetor) de 3 propriedades (geralmente chamada de vetor de recursos)

(por exemplo, [0,0,1] significa que este peso da fruta não é maior que 0,5 grama, e seu tamanho não é maior que 10 cm e a cor é vermelha)

Então, escolhemos 10 frutas aleatoriamente e medimos suas propriedades. O professor (humano) então rotula cada fruta manualmente como maçã => 1 ou laranja => [2].

eg) O professor seleciona uma fruta que é maçã. A representação desta maçã para o sistema poderia ser algo assim: [1, 1, 1] => 1, Isso significa que esta fruta tem 1.peso maior que .5grama, 2.tamanho maior que 10cm e 3. a cor desta fruta é vermelha e finalmente é uma maçã(=> 1)

Assim, para todas as 10 frutas, o professor rotula cada fruta como maçã[=>1] ou laranja[=>2] e o sistema encontra suas propriedades. como você adivinha, temos uma série de vetores (que a chamavam de matriz) para representar 10 frutas inteiras.

Introdução à Aprendizagem Supervisionada #

Existem várias situações em que se tem grandes quantidades de dados e usando os quais ele tem que classificar um objeto em uma das várias classes conhecidas. Considere as seguintes situações:

Banking: Quando um banco recebe um pedido de um cliente para um cartão bancário, o banco deve decidir se deve ou não emitir o cartão bancário, com base nas características de seus clientes que já usufruem dos cartões para os quais o histórico de crédito é conhecido .

Medicina: Pode-se estar interessado em desenvolver um sistema médico que diagnostique um paciente se ele está ou não tendo uma determinada doença, com base nos sintomas observados e nos exames médicos realizados nesse paciente.

Finanças: Uma empresa de consultoria financeira gostaria de prever a tendência do preço de uma ação que pode ser classificada em alta, baixa ou nenhuma tendência com base em várias características técnicas que regem o movimento do preço.

Expressão gênica: Um cientista analisando os dados de expressão gênica gostaria de identificar os genes e fatores de risco mais relevantes envolvidos no câncer de mama, a fim de separar pacientes saudáveis ​​de pacientes com câncer de mama.

Em todos os exemplos acima, um objeto é classificado em uma das várias classes conhecidas, com base nas medições feitas em uma série de características, que ele pode pensar discriminar os objetos de diferentes classes. Essas variáveis ​​são chamadas de variáveis ​​previsoras e o rótulo da classe é chamado de variável dependente. Observe que, em todos os exemplos acima, a variável dependente é categórica.

Para desenvolver um modelo para o problema de classificação, são necessários, para cada objeto, dados sobre um conjunto de características prescritas juntamente com os rótulos de classe, às quais os objetos pertencem. O conjunto de dados é dividido em dois conjuntos em uma proporção prescrita. O maior desses conjuntos de dados é chamado de conjunto de dados de treinamento e o outro conjunto de dados de teste. O conjunto de dados de treinamento é usado no desenvolvimento do modelo. Como o modelo é desenvolvido usando observações cujos rótulos de classe são conhecidos, esses modelos são conhecidos como modelos de aprendizagem supervisionada.

Depois de desenvolver o modelo, o modelo deve ser avaliado quanto ao seu desempenho usando o conjunto de dados de teste. O objetivo de um modelo de classificação é ter um mínimo de probabilidade de classificação errada nas observações não vistas. As observações não utilizadas no desenvolvimento do modelo são conhecidas como observações não vistas.

Indução de árvore de decisão é uma das técnicas de construção de modelo de classificação. O modelo de árvore de decisão construído para a variável dependente categórica é chamado de Árvore de Classificação. A variável dependente pode ser numérica em certos problemas. O modelo de árvore de decisão desenvolvido para variáveis ​​dependentes numéricas é chamado de Árvore de Regressão.
