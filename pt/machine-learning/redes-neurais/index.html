Funções de ativação #

As funções de ativação também conhecidas como função de transferência são usadas para mapear nós de entrada para nós de saída de determinada maneira.

Eles são usados ​​para conferir não linearidade à saída de uma camada de rede neural.

Algumas funções comumente usadas e suas curvas são dadas abaixo:

Função sigmóide #

O sigmoid é uma função de esmagamento cuja saída está no intervalo [0, 1].

O código para implementar sigmoid junto com sua derivada com numpy é mostrado abaixo:

def sigmoid(z):
    return 1.0 / (1.0 + np.exp(-z))

def sigmoid_prime(z):
    return sigmoid(z) * (1 - sigmoid(z))

Função Tangente Hiperbólica (tanh) #

A diferença básica entre as funções tanh e sigmoid é que tanh é 0 centrado, esmagando as entradas no intervalo [-1, 1] e é mais eficiente para calcular.

Você pode facilmente usar as funções np.tanh ou math.tanh para calcular a ativação de uma camada oculta.

Função ReLU #

Uma unidade linear retificada faz simplesmente max(0,x). É uma das escolhas mais comuns para funções de ativação de unidades de rede neural.

ReLUs abordam o [problema de gradiente de fuga] 5 de unidades sigmoid / tangente hiperbólica, permitindo assim a propagação eficiente de gradiente em redes profundas.

O nome ReLU vem do artigo de Nair e Hinton, Rectified Linear Units Improve Restricted Boltzmann Machines.

Tem algumas variações, por exemplo, ReLUs com vazamento (LReLUs) e Unidades lineares exponenciais (ELUs).

O código para implementar o vanilla ReLU junto com seu derivado com numpy é mostrado abaixo:

def relU(z):
    return z * (z > 0)

def relU_prime(z):
    return z > 0

Começando: Uma RNA simples com Python #

A listagem de código abaixo tenta classificar dígitos manuscritos do conjunto de dados MNIST. Os dígitos ficam assim:

O código pré-processará esses dígitos, convertendo cada imagem em uma matriz 2D de 0s e 1s e, em seguida, usará esses dados para treinar uma rede neural com precisão de até 97% (50 épocas).

"""
Deep Neural Net 

(Name: Classic Feedforward)

"""

import numpy as np
import pickle, json
import sklearn.datasets
import random
import time
import os


def sigmoid(z):
    return 1.0 / (1.0 + np.exp(-z))

def sigmoid_prime(z):
    return sigmoid(z) * (1 - sigmoid(z))

def relU(z):
    return np.maximum(z, 0, z)
    
def relU_prime(z):
    return z * (z <= 0)
    
def tanh(z):
    return np.tanh(z)

def tanh_prime(z):
    return 1 - (tanh(z) ** 2)

def transform_target(y):
    t = np.zeros((10, 1))
    t[int(y)] = 1.0
    return t

"""--------------------------------------------------------------------------------"""

class NeuralNet:

    def __init__(self, layers, learning_rate=0.05, reg_lambda=0.01):
        self.num_layers = len(layers)
        self.layers = layers
        self.biases = [np.zeros((y, 1)) for y in layers[1:]]
        self.weights = [np.random.normal(loc=0.0, scale=0.1, size=(y, x)) for x, y in zip(layers[:-1], layers[1:])]
        self.learning_rate = learning_rate
        self.reg_lambda = reg_lambda
        self.nonlinearity = relU
        self.nonlinearity_prime = relU_prime

    def __feedforward(self, x):
        """ Returns softmax probabilities for the output layer """
        for w, b in zip(self.weights, self.biases):
            x = self.nonlinearity(np.dot(w, np.reshape(x, (len(x), 1))) + b)

        return np.exp(x) / np.sum(np.exp(x))

    def __backpropagation(self, x, y):
        """
        :param x: input
        :param y: target
        """
        weight_gradients = [np.zeros(w.shape) for w in self.weights]
        bias_gradients = [np.zeros(b.shape) for b in self.biases]

        # forward pass
        activation = x
        hidden_activations = [np.reshape(x, (len(x), 1))]
        z_list = []

        for w, b in zip(self.weights, self.biases):    
            z = np.dot(w, np.reshape(activation, (len(activation), 1))) + b
            z_list.append(z)
            activation = self.nonlinearity(z)
            hidden_activations.append(activation)

        t = hidden_activations[-1] 
        hidden_activations[-1] = np.exp(t) / np.sum(np.exp(t))

        # backward pass
        delta = (hidden_activations[-1] - y) * (z_list[-1] > 0)
        weight_gradients[-1] = np.dot(delta, hidden_activations[-2].T)
        bias_gradients[-1] = delta

        for l in range(2, self.num_layers):
            z = z_list[-l]
            delta = np.dot(self.weights[-l + 1].T, delta) * (z > 0)
            weight_gradients[-l] = np.dot(delta, hidden_activations[-l - 1].T)
            bias_gradients[-l] = delta

        return (weight_gradients, bias_gradients)

    def __update_params(self, weight_gradients, bias_gradients):
        for i in xrange(len(self.weights)):
            self.weights[i] += -self.learning_rate * weight_gradients[i]
            self.biases[i] += -self.learning_rate * bias_gradients[i]

    def train(self, training_data, validation_data=None, epochs=10):
        bias_gradients = None
        for i in xrange(epochs):
            random.shuffle(training_data)
            inputs = [data[0] for data in training_data]
            targets = [data[1] for data in training_data]

            for j in xrange(len(inputs)):
                (weight_gradients, bias_gradients) = self.__backpropagation(inputs[j], targets[j])
                self.__update_params(weight_gradients, bias_gradients)

            if validation_data: 
                random.shuffle(validation_data)
                inputs = [data[0] for data in validation_data]
                targets = [data[1] for data in validation_data]

                for j in xrange(len(inputs)):
                    (weight_gradients, bias_gradients) = self.__backpropagation(inputs[j], targets[j])
                    self.__update_params(weight_gradients, bias_gradients)

            print("{} epoch(s) done".format(i + 1))

        print("Training done.")

    def test(self, test_data):
        test_results = [(np.argmax(self.__feedforward(x[0])), np.argmax(x[1])) for x in test_data]
        return float(sum([int(x == y) for (x, y) in test_results])) / len(test_data) * 100

    def dump(self, file):
        pickle.dump(self, open(file, "wb"))

"""--------------------------------------------------------------------------------"""

if __name__ == "__main__":
    total = 5000
    training = int(total * 0.7)
    val = int(total * 0.15)
    test = int(total * 0.15)

    mnist = sklearn.datasets.fetch_mldata('MNIST original', data_home='./data')

    data = zip(mnist.data, mnist.target)
    random.shuffle(data)
    data = data[:total]
    data = [(x[0].astype(bool).astype(int), transform_target(x[1])) for x in data]

    train_data = data[:training]
    val_data = data[training:training+val]
    test_data = data[training+val:]

    print "Data fetched"

    NN = NeuralNet([784, 32, 10]) # defining an ANN with 1 input layer (size 784 = size of the image flattened), 1 hidden layer (size 32), and 1 output layer (size 10, unit at index i will predict the probability of the image being digit i, where 0 <= i <= 9)  

    NN.train(train_data, val_data, epochs=5)

    print "Network trained"

    print "Accuracy:", str(NN.test(test_data)) + "%"

Este é um exemplo de código independente e pode ser executado sem outras modificações. Certifique-se de ter o numpy e o scikit learn instalado para sua versão do python.

Retropropagação - O coração das redes neurais #

O objetivo da retropropagação é otimizar os pesos para que a rede neural possa aprender a mapear corretamente entradas arbitrárias para saídas.

Cada camada tem seu próprio conjunto de pesos, e esses pesos devem ser ajustados para poder prever com precisão a saída correta dada a entrada.

Uma visão geral de alto nível de retropropagação é a seguinte:

  1. Forward pass - a entrada é transformada em alguma saída. Em cada camada, a ativação é calculada com um produto escalar entre a entrada e os pesos, seguido pela soma da resultante com o viés. Por fim, esse valor é passado por uma função de ativação, para obter a ativação daquela camada que se tornará a entrada para a próxima camada.
  2. Na última camada, a saída é comparada com o rótulo real correspondente a essa entrada e o erro é calculado. Normalmente, é o erro quadrático médio.
  3. Retrocesso - o erro calculado na etapa 2 é propagado de volta para as camadas internas e os pesos de todas as camadas são ajustados para levar em conta esse erro.

1. Inicialização dos pesos #

Um exemplo simplificado de inicialização de pesos é mostrado abaixo:

layers = [784, 64, 10] 
weights = np.array([(np.random.randn(y, x) * np.sqrt(2.0 / (x + y))) for x, y in zip(layers[:-1], layers[1:])])
biases = np.array([np.zeros((y, 1)) for y in layers[1:]])

  • A camada oculta 1 tem peso de dimensão [64, 784] e viés de dimensão 64.

  • A camada de saída tem peso de dimensão [10, 64] e viés de dimensão

Você pode estar se perguntando o que está acontecendo ao inicializar pesos no código acima. Isso é chamado de inicialização Xavier, e é um passo melhor do que inicializar aleatoriamente suas matrizes de peso. Sim, a inicialização importa. Com base em sua inicialização, você poderá encontrar um mínimo local melhor durante a descida do gradiente (a propagação reversa é uma versão glorificada da descida do gradiente).

2. Passe para frente #

activation = x
hidden_activations = [np.reshape(x, (len(x), 1))]
z_list = []

for w, b in zip(self.weights, self.biases):    
    z = np.dot(w, np.reshape(activation, (len(activation), 1))) + b
    z_list.append(z)
    activation = relu(z)
    hidden_activations.append(activation)

t = hidden_activations[-1] 
hidden_activations[-1] = np.exp(t) / np.sum(np.exp(t))

Este código realiza a transformação descrita acima. hidden_activations[-1] contém probabilidades softmax - previsões de todas as classes, cuja soma é 1. Se estivermos prevendo dígitos, a saída será um vetor de probabilidades de dimensão 10, cuja soma é 1.

3. Passe para trás #

weight_gradients = [np.zeros(w.shape) for w in self.weights]
bias_gradients = [np.zeros(b.shape) for b in self.biases]

delta = (hidden_activations[-1] - y) * (z_list[-1] > 0) # relu derivative
weight_gradients[-1] = np.dot(delta, hidden_activations[-2].T)
bias_gradients[-1] = delta

for l in range(2, self.num_layers):
    z = z_list[-l]
    delta = np.dot(self.weights[-l + 1].T, delta) * (z > 0) # relu derivative
    weight_gradients[-l] = np.dot(delta, hidden_activations[-l - 1].T)
    bias_gradients[-l] = delta

As primeiras 2 linhas inicializam os gradientes. Esses gradientes são calculados e serão usados ​​para atualizar os pesos e vieses posteriormente.

As próximas 3 linhas calculam o erro subtraindo a previsão do alvo. O erro é então propagado de volta para as camadas internas.

Agora, trace cuidadosamente o funcionamento do loop. As linhas 2 e 3 transformam o erro de layer[i] para layer[i - 1]. Trace as formas das matrizes que estão sendo multiplicadas para entender.

4. Atualização de pesos/parâmetros #

for i in xrange(len(self.weights)):
    self.weights[i] += -self.learning_rate * weight_gradients[i]
    self.biases[i] += -self.learning_rate * bias_gradients[i]

self.learning_rate especifica a taxa na qual a rede aprende. Você não quer que ele aprenda muito rápido, porque pode não convergir. Uma descida suave é favorecida para encontrar bons mínimos. Normalmente, taxas entre 0,01 e 0,1 são consideradas boas.

Função Softmax #

A regressão Softmax (ou regressão logística multinomial) é uma generalização da regressão logística para o caso em que queremos lidar com várias classes. É particularmente útil para redes neurais onde queremos aplicar classificação não binária. Nesse caso, a regressão logística simples não é suficiente. Precisaríamos de uma distribuição de probabilidade em todos os rótulos, que é o que o softmax nos dá.
