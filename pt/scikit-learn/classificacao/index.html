RandomForestClassifier #

Uma floresta aleatória é um meta estimador que ajusta vários classificadores de árvore de decisão em várias subamostras do conjunto de dados e usa a média para melhorar a precisão preditiva e controlar o ajuste excessivo.

Um exemplo simples de uso:

Importar:

from sklearn.ensemble import RandomForestClassifier

Defina dados de trem e dados de destino:

train = [[1,2,3],[2,5,1],[2,1,7]]
target = [0,1,0]

Os valores em target representam o rótulo que você deseja prever.

Inicie um objeto RandomForest e execute aprender (ajustar):

rf = RandomForestClassifier(n_estimators=100)
rf.fit(train, target)

Prever:

test = [2,2,3]
predicted = rf.predict(test)

Análise de relatórios de classificação #

Crie um relatório de texto mostrando as principais métricas de classificação, incluindo a precisão e recall, f1-score (a média harmônica de precisão e recall) e suporte (o número de observações dessa classe no conjunto de treinamento).

Exemplo de sklearn docs:

from sklearn.metrics import classification_report
y_true = [0, 1, 2, 2, 2]
y_pred = [0, 0, 2, 2, 1]
target_names = ['class 0', 'class 1', 'class 2']
print(classification_report(y_true, y_pred, target_names=target_names))

Resultado -

         precision    recall  f1-score   support

class 0       0.50      1.00      0.67         1
class 1       0.00      0.00      0.00         1
class 2       1.00      0.67      0.80         3

avg / total   0.70      0.60      0.61         5

GradientBoostingClassifier #

Gradient Boosting para classificação. O Gradient Boosting Classifier é um ensemble aditivo de um modelo base cujo erro é corrigido em sucessivas iterações (ou estágios) pela adição de Árvores de Regressão que corrigem os resíduos (o erro do estágio anterior).

Importar:

from sklearn.ensemble import GradientBoostingClassifier

Crie alguns dados de classificação de brinquedos

from sklearn.datasets import load_iris

iris_dataset = load_iris()

X, y = iris_dataset.data, iris_dataset.target

Vamos dividir esses dados em conjunto de treinamento e teste.

from sklearn.model_selection import train_test_split
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.2, random_state=0)

Instanciar um modelo GradientBoostingClassifier usando os parâmetros padrão.

gbc = GradientBoostingClassifier()
gbc.fit(X_train, y_train)

Vamos pontuar no conjunto de teste

# We are using the default classification accuracy score
>>> gbc.score(X_test, y_test)
1

Por padrão, existem 100 estimadores construídos

>>> gbc.n_estimators
100

Isso pode ser controlado definindo n_estimators para um valor diferente durante o tempo de inicialização.

Usando máquinas de vetor de suporte #

Support vector machines é uma família de algoritmos que tentam passar um hiperplano (possivelmente de alta dimensão) entre dois conjuntos de pontos rotulados, de modo que a distância dos pontos do avião é ótimo em algum sentido. SVMs podem ser usados ​​para classificação ou regressão (correspondente a sklearn.svm.SVC e sklearn. svm.SVR, respectivamente.

Exemplo:

Suponha que trabalhamos em um espaço 2D. Primeiro, criamos alguns dados:

import numpy as np

Agora criamos x e y:

x0, x1 = np.random.randn(10, 2), np.random.randn(10, 2) + (1, 1)
x = np.vstack((x0, x1))

y = [0] * 10 + [1] * 10

Observe que x é composto por duas gaussianas: uma centrada em (0, 0) e outra centrada em (1, 1).

Para construir um classificador, podemos usar:

from sklearn import svm

svm.SVC(kernel='linear').fit(x, y)

Vamos verificar a previsão para (0, 0):

>>> svm.SVC(kernel='linear').fit(x, y).predict([[0, 0]])
array([0])

A previsão é que a classe seja 0.

Para regressão, podemos fazer da mesma forma:

svm.SVR(kernel='linear').fit(x, y)

Uma árvore de decisão #

Uma árvore de decisão é um classificador que usa uma sequência de regras detalhadas (como a>7) que podem ser facilmente compreendidas.

O exemplo abaixo treina um classificador de árvore de decisão usando três vetores de recursos de comprimento 3 e, em seguida, prevê o resultado para um quarto vetor de recursos até agora desconhecido, o chamado vetor de teste.

from sklearn.tree import DecisionTreeClassifier

# Define training and target set for the classifier
train = [[1,2,3],[2,5,1],[2,1,7]]
target = [10,20,30]

# Initialize Classifier. 
# Random values are initialized with always the same random seed of value 0 
# (allows reproducible results)
dectree = DecisionTreeClassifier(random_state=0)
dectree.fit(train, target)

# Test classifier with other, unknown feature vector
test = [2,2,3]
predicted = dectree.predict(test)

print predicted

A saída pode ser visualizada usando:

import pydot
import StringIO

dotfile = StringIO.StringIO()
tree.export_graphviz(dectree, out_file=dotfile)
(graph,)=pydot.graph_from_dot_data(dotfile.getvalue())
graph.write_png("dtree.png")
graph.write_pdf("dtree.pdf")

Classificação por Regressão Logística #

No LR Classifier, as probabilidades que descrevem os possíveis resultados de uma única tentativa são modeladas usando uma função logística. Ele é implementado na biblioteca linear_model

from sklearn.linear_model import LogisticRegression

A implementação do sklearn LR pode ajustar-se à regressão logística binária, One-vs-Resto ou multinomial com regularização L2 ou L1 opcional. Por exemplo, vamos considerar uma classificação binária em um conjunto de dados sklearn de amostra

from sklearn.datasets import make_hastie_10_2

X,y = make_hastie_10_2(n_samples=1000)

Onde X é uma matriz n_samples X 10 e y são os rótulos de destino -1 ou +1.

Use a divisão de teste de trem para dividir os dados de entrada em conjuntos de treinamento e teste (70%-30%)

from sklearn.model_selection import train_test_split 
#sklearn.cross_validation in older scikit versions

data_train, data_test, labels_train, labels_test = train_test_split(X,y, test_size=0.3)

O uso do classificador LR é semelhante a outros exemplos

# Initialize Classifier. 
LRC = LogisticRegression()
LRC.fit(data_train, labels_train)

# Test classifier with the test data
predicted = LRC.predict(data_test)

Use a matriz de confusão para visualizar os resultados

from sklearn.metrics import confusion_matrix

confusion_matrix(predicted, labels_test)
