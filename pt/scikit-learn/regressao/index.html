Mínimos Quadrados Ordinários #

[Ordinary Least Squares] (https://en.wikipedia.org/wiki/Ordinary_least_squares) é um método para encontrar a combinação linear de características que melhor se ajusta ao resultado observado no seguinte sentido.

Se o vetor de resultados a serem previstos for y, e as variáveis ​​explicativas formarem a matriz X, então OLS encontrará o vetor β resolvendo

minβ|y^ - y|22,

onde y^ = X β é a previsão linear.

No sklearn, isso é feito usando sklearn.linear_model.LinearRegression.

Contexto do aplicativo

OLS só deve ser aplicado a problemas de regressão, geralmente é inadequado para problemas de classificação: Contraste

  • Um e-mail é spam? (Classificação)
  • Qual é a relação linear entre upvotes depende da duração da resposta? (Regressão)

Exemplo

Vamos gerar um modelo linear com algum ruído, então veja se LinearRegression consegue reconstruir o modelo linear.

Primeiro geramos a matriz X:

import numpy as np

X = np.random.randn(100, 3)

Agora vamos gerar o y como uma combinação linear de X com algum ruído:

beta = np.array([[1, 1, 0]])
y = (np.dot(x, beta.T) + 0.01 * np.random.randn(100, 1))[:, 0]

Observe que a verdadeira combinação linear gerando y é dada por `beta.

Para tentar reconstruir isso apenas a partir de X e y, vamos fazer:

>>> linear_model.LinearRegression().fit(x, y).coef_
array([  9.97768469e-01,   9.98237634e-01,   7.55016533e-04])

Observe que este vetor é muito semelhante ao beta.
