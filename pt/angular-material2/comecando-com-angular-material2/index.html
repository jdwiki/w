Envolvendo todos os módulos juntos #

Você também pode facilmente agrupar todos os módulos angulares, que você vai usar, em um módulo:

import { NgModule } from '@angular/core';
import { MdButtonModule, MdSnackBarModule, MdSidenavModule } from '@angular/material';
 
@NgModule({
    imports: [
        BrowserAnimationsModule,
        MdButtonModule,
        MdSnackBarModule,
        MdSidenavModule
    ],
    exports: [
        MdButtonModule,
        MdSnackBarModule,
        MdSidenavModule
    ]
})
export class MaterialWrapperModule {}

Depois disso, basta importar seu módulo para o módulo principal do aplicativo:

import { NgModule } from '@angular/core';
import { CommonModule } from '@angular/common';
import { RouterModule } from '@angular/router';

import { MaterialWrapperModule } from './material-wrapper.module.ts';
import { AppComponent } from './app.component';


@NgModule({
    imports: [
        BrowserAnimationsModule, 
        MaterialWrapperModule,
        CommonModule,
        // This is optional, use it when you would like routing in your app
        // RouterModule.forRoot([
        //     { path: '', component: HomeView, pathMatch: 'full'}
        // ])
    ],
    declarations: [ AppComponent],
    bootstrap: [ AppComponent ]
})
export class AppModule {}

Instalação e configuração do mestre com Angular CLI #

Este exemplo será como instalar a partir do master e usará @angular/cli também.

  1. Faça um novo projeto com @angular/cli:
   ng new my-master-app

Se você não instalou @angular/cli, use este comando:

   npm install -g @angular/cli
  1. Instale a partir do master:
   npm install --save @angular/animations
   npm install --save angular/material2-builds
   npm install --save angular/cdk-builds
  1. Siga o mesmo guia acima.

Feito!

Instalação ou configuração com CLI Angular #

Neste exemplo, usaremos @angular/cli (mais recente) e a versão mais recente de @angular/material. Você deve pelo menos conhecer o básico do Angular 2/4 antes de continuar as etapas abaixo.

  1. Instale o módulo de material angular de npm:
   npm install @angular/material --save

Isso se aplica apenas às versões 2.0.0-beta.3 e superiores.

Instale o módulo @angular/animations:

npm install @angular/animations --save

Isso se aplica apenas às versões 2.0.0-beta.8 e superiores.

Instale o módulo @angular/cdk:

npm install @angular/cdk --save

  1. Em seu módulo de aplicação importe os componentes que você vai usar:

    import { NgModule } from '@angular/core';
import { CommonModule } from '@angular/common';
import { RouterModule } from '@angular/router';
import { MdButtonModule, MdSnackBarModule, MdSidenavModule } from '@angular/material';

import { AppComponent } from './app.component';

import { BrowserAnimationsModule } from '@angular/platform-browser/animations';

@NgModule({
     imports: [
         BrowserAnimationsModule,
         MdButtonModule,
         MdSnackBarModule,
         MdSidenavModule,
         CommonModule,
         // This is optional unless you want to have routing in your app
         // RouterModule.forRoot([
         //     { path: '', component: HomeView, pathMatch: 'full'}
         // ])
     ],
     declarations: [ AppComponent ],
     boostrap: [ AppComponent ]
})
export class AppModule {}

Agora você está pronto para usar o Angular Material em seus componentes!

Observação: os documentos para componentes específicos estarão disponíveis em breve.

Configure o tema, suporte a gestos e ícones de material #

Tema:

Um tema é necessário para que os componentes do material funcionem corretamente no aplicativo.

Angular Material 2 fornece quatro temas pré-construídos:

  • profundo roxo-âmbar
  • rosa índigo
  • rosa-azul acinzentado
  • verde púrpura

Se você estiver usando Angular CLI, você pode importar um dos temas pré-construídos em style.css.

@import "~@angular/material/prebuilt-themes/indigo-pink.css";

O tema pode ser adicionado usando <link> em index.html também.

<link href="node_modules/@angular/material/prebuilt-themes/indigo-pink.css" rel="stylesheet">

HammerJS

Adicione o HammerJS ao aplicativo via CDN ou npm:

npm install --save hammerjs

Em seu módulo raiz, geralmente app.module.ts, adicione a instrução de importação:

import 'hammerjs';

Ícones de materiais:

A menos que sejam fornecidos ícones personalizados, o Angular Material 2 <md-icon> espera ícones de material.

Em index.html adicione:

<link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
