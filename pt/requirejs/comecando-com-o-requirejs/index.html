Hello World com dependências aninhadas #

O exemplo a seguir expande o Hello World demonstrando várias dependências usando a função define().

Crie um novo arquivo HTML chamado index.html e cole o seguinte conteúdo:

<!DOCTYPE html>
<html lang="en">
<head>
    <title>Hello RequireJS</title>
    <script type="text/javascript" src="http://requirejs.org/docs/release/2.3.2/minified/require.js"></script>
</head>
<body>
    <!-- place content here --->
    <script>
        requirejs(["scripts/say"], function(say) {
            console.log(say.hello("english"));
            console.log(say.hello("spanish"));
            console.log(say.hello("french"));
        });
    </script>
</body>

Crie um novo arquivo JS em scripts/say.js e cole o seguinte conteúdo:

define(["scripts/translations"], function(translations){
    return {
        hello: function(language){
           return translations[language].hello + " " + translations[language].world;
        }
    };
});

Crie um novo arquivo JS em scripts/translations.js e cole o seguinte conteúdo:

define([], function(){
    return {
        "english": {
            hello: "Hello",
            world: "World"
        },
        "spanish": {
            hello: "Hola",
            world: "Mundo"
        },
        "french": {
            hello: "Bonjour",
            world: "Le Monde"
        }
    };
});

A estrutura do seu projeto deve ficar assim:

- project
    - index.html
    - scripts
        - say.js
        - translations.js

Abra o arquivo index.html em um navegador e o console exibirá:

Hello World
Hola Mundo
Bonjour Le Monde

Explicação #

  1. Carregue o arquivo de script require.js.

    <script type="text/javascript" src="http://requirejs.org/docs/release/2.3.2/minified/require.js"></script>

  2. Carregue o módulo say de forma assíncrona da pasta scripts. (Observe que você não precisa da extensão .js ao fazer referência ao módulo.) O módulo retornado é então passado para a função fornecida onde hello() é invocado.

    <script>
     requirejs(["scripts/say"], function(say) {
         console.log(say.hello("english"));
         console.log(say.hello("spanish"));
         console.log(say.hello("french"));
     });
</script>

  3. O módulo say retorna um único objeto com uma função hello definida, mas neste caso definimos uma dependência (scripts/translations) e vamos extrair as traduções de lá.

     define(["scripts/translations"], function(translations){
     return {
         hello: function(language){
            return translations[language].hello + " " + translations[language].world;
         }
     };
 });

  4. O módulo translations simplesmente retorna um objeto com várias traduções de palavras.

     define([], function(){
     return {
         "english": {
             hello: "Hello",
             world: "World"
         },
         "spanish": {
             hello: "Hola",
             world: "Mundo"
         },
         "french": {
             hello: "Bonjour",
             world: "Le Monde"
         }
     };
 });

Olá Mundo #

O exemplo a seguir demonstrará uma instalação e configuração básicas do RequireJS.

Crie um novo arquivo HTML chamado index.html e cole o seguinte conteúdo:

<!DOCTYPE html>
<html lang="en">
<head>
    <title>Hello RequireJS</title>
    <script type="text/javascript" src="http://requirejs.org/docs/release/2.3.2/minified/require.js"></script>
</head>
<body>
    <!-- place content here --->
    <script>
        requirejs(["scripts/say"], function(say) {
            alert(say.hello());
        });
    </script>
</body>

Crie um novo arquivo JS em scripts/say.js e cole o seguinte conteúdo:

define([], function(){
    return {
        hello: function(){
           return "Hello World";
        }
    };
});

A estrutura do seu projeto deve ficar assim:

- project
    - index.html
    - scripts
        - say.js

Abra o arquivo index.html em um navegador e ele irá alertá-lo com ‘Hello World’.

Explicação #

  1. Carregue o arquivo de script require.js.

    <script type="text/javascript" src="http://requirejs.org/docs/release/2.3.2/minified/require.js"></script>

  2. Carregue o módulo say de forma assíncrona da pasta scripts. (Observe que você não precisa da extensão .js ao fazer referência ao módulo.) O módulo retornado é então passado para a função fornecida onde hello() é invocado.

    <script>
   requirejs(["scripts/say"], function(say) {
       alert(say.hello());
   });
</script>

  3. O módulo say retorna um único objeto com uma função hello definida.

     define([], function(){
     return {
         hello: function(){
            return "Hello World";
         }
     };
 });

Usando o ponto de entrada principal de dados #

Um único ponto de entrada para seu aplicativo é possível com o RequireJS usando o data-main atribuído dentro da tag <script>.

<script type="text/javascript" data-main="scripts/main" src="http://requirejs.org/docs/release/2.3.2/minified/require.js"></script>

Ao carregar, o RequireJS procurará o atributo data-main e injetará a tag de script principal no DOM com o atributo async definido. É neste arquivo que você vai querer fazer qualquer configuração antes de iniciar sua aplicação.

Por exemplo:

// contents of scripts/main.js
require.config({
    waitSeconds: 10,
    paths: {
        jquery: 'libs/jquery-1.4.2.min'
    }
});

requirejs(["jquery", "libs/say"], function($, say) {
    var $body = $('body');
    $body.append( $('<p/>').text(say.hello("english")) );
    $body.append( $('<p/>').text(say.hello("spanish")) );
    $body.append( $('<p/>').text(say.hello("french")) );
});

Incorporando bibliotecas não AMD #

Nem todas as bibliotecas são definidas de forma compatível com a função define() da AMD e do RequireJS. Os autores resolveram isso incluindo uma diretiva shim para configurar essas dependências.

Um exemplo é usar o plug-in de layout de interface do usuário do jQuery. Esse plugin depende do jQuery. Você pode configurá-lo assim:

requirejs.config({
    paths: {
        'jquery': '../path/to/jquery.min',
        'jquery.layout': '../path/to/jquery.layout.min'
    },
    shim: {
        'jquery.layout': {
            deps: ['jquery']
        }
    }
});

E então use-o em um módulo de layout como este:

define(['jquery', 'jquery.layout'], function ($, layout) {
    $('body').layout();
});
