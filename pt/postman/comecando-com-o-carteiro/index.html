O que é Carteiro? #

Postman é um ambiente de desenvolvimento de API que ajuda as pessoas a criar, testar, documentar, monitorar e publicar documentação para suas APIs.

As principais características do Postman são

  • Enviando solicitações (com suporte para diferentes esquemas de autenticação, Cookies, certificados, cabeçalhos, parâmetros de consulta, corpo da solicitação e SOAP com /sem WSDL) e depurando e salvando respostas.
  • Organizar suas APIs em grupos chamados Coleções.
  • Compartilhando e colaborando suas coleções com sua equipe.
  • Testes de redação. Os scripts de teste podem executar pré-solicitação, após o recebimento de uma resposta, e podem ter conceitos de loop e ramificação.
  • O conceito de variáveis para definir e obter. Bem como variáveis ​​de ambiente globais.
  • Automação de testes usando Colection Runs. As coleções também podem ser exportadas e executadas na linha de comando usando Newman como parte de seu processo de compilação.
  • Você também pode agendar testes para serem executados de forma automatizada usando Postman Monitors.
  • Uma maneira de gerar e personalizar automaticamente sua documentação da API diretamente de suas coleções. Ele pode ser privado, compartilhado com sua equipe, público e também pode ser configurado em seu domínio personalizado.
  • Simule um backend com Mock Servers.
  • Integrations com serviços como Slack, Github, Bitbucket, Datadog, Keen.io, Microsoft Flow e vários outros.
  • Uma API que permite consumir os dados do Postman como parte de seus processos contínuos de integração e entrega.
  • Uma maneira de importar APIs existentes do Swagger, RAML, cURL e várias outras ferramentas.
  • Geração automática de snippets de código em diferentes idiomas de suas APIs.
  • Solicitações de interceptação.
  • Histórico interativo de todos os seus pedidos.
  • Controle de acesso para sua equipe e Single Sign On (SSO).
  • Componentes principais de código aberto.

Começando

O Postman está disponível como um aplicativo nativo para os sistemas operacionais Mac, Windows e Linux.

Para instalar o Postman, vá para a página de aplicativos e clique em Download para Mac/Windows/Linux dependendo da sua plataforma.

O aplicativo nativo é a abordagem recomendada para usar o Postman conforme a documentação oficial.
