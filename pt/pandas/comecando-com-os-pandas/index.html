Instalação ou configuração #

Instruções detalhadas sobre como configurar ou instalar os pandas podem ser encontradas aqui na documentação oficial.

Instalando pandas com Anaconda

Instalar pandas e o resto da pilha NumPy e SciPy pode ser um pouco difícil para usuários inexperientes.

A maneira mais simples de instalar não apenas pandas, mas Python e os pacotes mais populares que compõem a pilha SciPy (IPython, NumPy, Matplotlib, …) é com Anaconda, uma plataforma cruzada (Linux, Mac OS X, Windows) distribuição Python para análise de dados e computação científica.

Depois de executar um instalador simples, o usuário terá acesso aos pandas e ao restante da pilha SciPy sem precisar instalar mais nada e sem precisar esperar que nenhum software seja compilado.

Instruções de instalação do Anaconda podem ser encontradas aqui.

Uma lista completa dos pacotes disponíveis como parte da distribuição Anaconda pode ser encontrada aqui.

Uma vantagem adicional de instalar com o Anaconda é que você não precisa de direitos de administrador para instalá-lo, ele será instalado no diretório inicial do usuário, e isso também torna trivial excluir o Anaconda posteriormente (apenas exclua essa pasta).

Instalando pandas com Miniconda

A seção anterior descreveu como instalar os pandas como parte da distribuição do Anaconda. No entanto, essa abordagem significa que você instalará mais de cem pacotes e envolve o download do instalador, que tem algumas centenas de megabytes de tamanho.

Se você quiser ter mais controle sobre quais pacotes, ou ter uma largura de banda de internet limitada, instalar pandas com Miniconda pode ser uma solução melhor.

Conda é o gerenciador de pacotes sobre o qual a distribuição do Anaconda é construída. É um gerenciador de pacotes que é multiplataforma e agnóstico de linguagem (pode desempenhar um papel semelhante a uma combinação de pip e virtualenv).

Miniconda permite que você crie uma instalação mínima do Python independente e, em seguida, use o comando Conda para instalar pacotes adicionais.

Primeiro você vai precisar que o Conda esteja instalado e baixar e rodar o Miniconda fará isso para você. O instalador pode ser encontrado aqui.

A próxima etapa é criar um novo ambiente conda (eles são análogos a um virtualenv, mas também permitem especificar com precisão qual versão do Python instalar também). Execute os seguintes comandos em uma janela de terminal:

conda create -n name_of_my_env python

Isso criará um ambiente mínimo com apenas o Python instalado nele. Para se colocar dentro deste ambiente execute:

source activate name_of_my_env

No Windows o comando é:

activate name_of_my_env

A etapa final necessária é instalar o pandas. Isso pode ser feito com o seguinte comando:

conda install pandas

Para instalar uma versão específica do pandas:

conda install pandas=0.13.1

Para instalar outros pacotes, IPython por exemplo:

conda install ipython

Para instalar a distribuição completa do Anaconda:

conda install anaconda

Se você precisar de algum pacote disponível para o pip, mas não para o conda, basta instalar o pip e usar o pip para instalar esses pacotes:

conda install pip
pip install django

Normalmente, você instalaria pandas com um dos gerenciadores de pacotes.

exemplo de pip:

pip install pandas

Isso provavelmente exigirá a instalação de várias dependências, incluindo NumPy, exigirá um compilador para compilar os bits de código necessários e pode levar alguns minutos para ser concluído.

Instalar via anaconda #

Primeiro baixe o anaconda do site do Continuum. Seja através do instalador gráfico (Windows/OSX) ou executando um script de shell (OSX/Linux). Isso inclui pandas!

Se você não quiser os 150 pacotes convenientemente agrupados no anaconda, você pode instalar o miniconda. Seja através do instalador gráfico (Windows) ou shell script (OSX/Linux).

Instale pandas no miniconda usando:

conda install pandas

Para atualizar os pandas para a versão mais recente no anaconda ou miniconda, use:

conda update pandas

Estatísticas descritivas #

Estatísticas descritivas (média, desvio padrão, número de observações, mínimo, máximo e quartis) de colunas numéricas podem ser calculadas usando o método .describe(), que retorna um dataframe pandas de estatísticas descritivas.

In [1]: df = pd.DataFrame({'A': [1, 2, 1, 4, 3, 5, 2, 3, 4, 1], 
                           'B': [12, 14, 11, 16, 18, 18, 22, 13, 21, 17], 
                           'C': ['a', 'a', 'b', 'a', 'b', 'c', 'b', 'a', 'b', 'a']})

In [2]: df
Out[2]: 
   A   B  C
0  1  12  a
1  2  14  a
2  1  11  b
3  4  16  a
4  3  18  b
5  5  18  c
6  2  22  b
7  3  13  a
8  4  21  b
9  1  17  a

In [3]: df.describe()
Out[3]:
               A          B
count  10.000000  10.000000
mean    2.600000  16.200000
std     1.429841   3.705851
min     1.000000  11.000000
25%     1.250000  13.250000
50%     2.500000  16.500000
75%     3.750000  18.000000
max     5.000000  22.000000

Observe que, como C não é uma coluna numérica, ela é excluída da saída.

In [4]: df['C'].describe()
Out[4]:
count     10
unique     3
freq       5
Name: C, dtype: object

Neste caso, o método resume os dados categóricos por número de observações, número de elementos únicos, modo e frequência do modo.

Olá Mundo #

Uma vez instalado o Pandas, você pode verificar se ele está funcionando corretamente criando um conjunto de dados de valores distribuídos aleatoriamente e traçando seu histograma.

import pandas as pd  # This is always assumed but is included here as an introduction.
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(0)

values = np.random.randn(100) # array of normally distributed random numbers
s = pd.Series(values) # generate a pandas series
s.plot(kind='hist', title='Normally distributed random values') # hist computes distribution
plt.show()

Verifique algumas das estatísticas dos dados (média, desvio padrão, etc.)

s.describe()
# Output: count    100.000000
# mean       0.059808
# std        1.012960
# min       -2.552990
# 25%       -0.643857
# 50%        0.094096
# 75%        0.737077
# max        2.269755
# dtype: float64
