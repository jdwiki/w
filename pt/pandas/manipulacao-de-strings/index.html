Expressões regulares #

# Extract strings with a specific regex
df= df['col_name'].str.extract[r'[Aa-Zz]']

# Replace strings within a regex
df['col_name'].str.replace('Replace this', 'With this')

Para obter informações sobre como corresponder strings usando regex, consulte Introdução às expressões regulares.

Capitalização de strings #

In [1]: ser = pd.Series(['lORem ipSuM', 'Dolor sit amet', 'Consectetur Adipiscing Elit'])

Converta tudo para maiúsculas:

In [2]: ser.str.upper()
Out[2]:
0                    LOREM IPSUM
1                 DOLOR SIT AMET
2    CONSECTETUR ADIPISCING ELIT
dtype: object

Todas as letras minúsculas:

In [3]: ser.str.lower()
Out[3]:
0                    lorem ipsum
1                 dolor sit amet
2    consectetur adipiscing elit
dtype: object

Capitalize o primeiro caractere e diminua o restante:

In [4]: ser.str.capitalize()
Out[4]:
0                    Lorem ipsum
1                 Dolor sit amet
2    Consectetur adipiscing elit
dtype: object

Converta cada string em uma caixa de título (coloque em maiúscula o primeiro caractere de cada palavra em cada string, em minúsculas o restante):

In [5]: ser.str.title()
Out[5]:
0                    Lorem Ipsum
1                 Dolor Sit Amet
2    Consectetur Adipiscing Elit
dtype: object

Trocar maiúsculas e minúsculas (converter minúsculas para maiúsculas e vice-versa):

In [6]: ser.str.swapcase()
Out[6]:
0                    LorEM IPsUm
1                 dOLOR SIT AMET
2    cONSECTETUR aDIPISCING eLIT
dtype: object

Além desses métodos que alteram a capitalização, vários métodos podem ser usados ​​para verificar a capitalização de strings.

In [7]: ser = pd.Series(['LOREM IPSUM', 'dolor sit amet', 'Consectetur Adipiscing Elit'])

Verifique se está tudo em minúsculas:

In [8]: ser.str.islower()
Out[8]:
0    False
1     True
2    False
dtype: bool

É tudo maiúsculo:

In [9]: ser.str.isupper()
Out[9]:
0     True
1    False
2    False
dtype: bool

É uma string com título:

In [10]: ser.str.istitle()
Out[10]:
0    False
1    False
2     True
dtype: bool

Verificando o conteúdo de uma string #

O método str.contains() pode ser usado para verificar se um padrão ocorre em cada string de uma Série. Os métodos str.startswith() e str.endswith() também podem ser usados ​​como versões mais especializadas.

In [1]: animals = pd.Series(['cat', 'dog', 'bear', 'cow', 'bird', 'owl', 'rabbit', 'snake'])

Verifique se as strings contêm a letra ‘a’:

In [2]: animals.str.contains('a')
Out[2]:
0      True
1     False
2      True
3     False
4     False
5     False
6      True
7      True
8      True
dtype: bool

Isso pode ser usado como um índice booleano para retornar apenas os animais que contêm a letra ‘a’:

In [3]: animals[animals.str.contains('a')]
Out[3]: 
0       cat
2      bear
6    rabbit
7     snake
dtype: object

Os métodos str.startswith e str.endswith funcionam de forma semelhante, mas também aceitam tuplas como entradas.

In [4]: animals[animals.str.startswith(('b', 'c'))]
# Returns animals starting with 'b' or 'c'
Out[4]: 
0     cat
2    bear
3     cow
4    bird
dtype: object

Cortando cordas #

Strings em uma série podem ser fatiadas usando o método .str.slice(), ou mais convenientemente, usando colchetes (.str[]).

In [1]: ser = pd.Series(['Lorem ipsum', 'dolor sit amet', 'consectetur adipiscing elit'])
In [2]: ser
Out[2]: 
0                    Lorem ipsum
1                 dolor sit amet
2    consectetur adipiscing elit
dtype: object

Obtenha o primeiro caractere de cada string:

In [3]: ser.str[0]
Out[3]: 
0    L
1    d
2    c
dtype: object

Obtenha os três primeiros caracteres de cada string:

In [4]: ser.str[:3]
Out[4]: 
0    Lor
1    dol
2    con
dtype: object

Obtenha o último caractere de cada string:

In [5]: ser.str[-1]
Out[5]:
0    m
1    t
2    t
dtype: object

Obtenha os três últimos caracteres de cada string:

In [6]: ser.str[-3:]
Out[6]: 
0    sum
1    met
2    lit
dtype: object

Obtenha todos os outros caracteres dos primeiros 10 caracteres:

In [7]: ser.str[:10:2]
Out[7]: 
0    Lrmis
1    dlrst
2    cnett
dtype: object

O Pandas se comporta de maneira semelhante ao Python ao manipular fatias e índices. Por exemplo, se um índice estiver fora do intervalo, o Python gera um erro:

In [8]:'Lorem ipsum'[12]
# IndexError: string index out of range

No entanto, se uma fatia estiver fora do intervalo, uma string vazia será retornada:

In [9]: 'Lorem ipsum'[12:15]
Out[9]: ''

Pandas retorna NaN quando um índice está fora do intervalo:

In [10]: ser.str[12]
Out[10]:
0    NaN
1      e
2      a
dtype: object

E retorna uma string vazia se uma fatia estiver fora do intervalo:

In [11]: ser.str[12:15]
Out[11]:
0       
1     et
2    adi
dtype: object
