Obtenha informações do DataFrame e uso de memória #

Para obter informações básicas sobre um DataFrame, incluindo os nomes das colunas e os tipos de dados:

import pandas as pd

df = pd.DataFrame({'integers': [1, 2, 3], 
                   'floats': [1.5, 2.5, 3], 
                   'text': ['a', 'b', 'c'], 
                   'ints with None': [1, None, 3]})

df.info()
<class 'pandas.core.frame.DataFrame'>
Int64Index: 3 entries, 0 to 2
Data columns (total 4 columns):
floats            3 non-null float64
integers          3 non-null int64
ints with None    2 non-null float64
text              3 non-null object
dtypes: float64(2), int64(1), object(1)
memory usage: 120.0+ bytes

Para obter o uso de memória do DataFrame:

>>> df.info(memory_usage='deep')
<class 'pandas.core.frame.DataFrame'>
Int64Index: 3 entries, 0 to 2
Data columns (total 4 columns):
floats            3 non-null float64
integers          3 non-null int64
ints with None    2 non-null float64
text              3 non-null object
dtypes: float64(2), int64(1), object(1)
memory usage: 234.0 bytes

Várias estatísticas de resumo do Dataframe. #

import pandas as pd
df = pd.DataFrame(np.random.randn(5, 5), columns=list('ABCDE'))

Para gerar várias estatísticas de resumo. Para valores numéricos, o número de valores não NA/nulos (count), a média (mean), o desvio padrão std e valores conhecidos como [resumo de cinco números](https://en. wikipedia.org/wiki/Five-number_summary):

  • min: mínimo (menor observação)

  • 25%: quartil inferior ou primeiro quartil (Q1)

  • 50%: mediana (valor médio, Q2)

  • 75%: quartil superior ou terceiro quartil (Q3)

  • max: máximo (maior observação)

Listar nomes de colunas do DataFrame #

df = pd.DataFrame({'a': [1, 2, 3], 'b': [4, 5, 6], 'c': [7, 8, 9]})

Para listar os nomes das colunas em um DataFrame:

>>> list(df)
['a', 'b', 'c']

Este método de compreensão de lista é especialmente útil ao usar o depurador:

>>> [c for c in df]
['a', 'b', 'c']

Este é o longo caminho:

sampledf.columns.tolist()

Você também pode imprimi-los como um índice em vez de uma lista (isso não será muito visível para dataframes com muitas colunas):

df.columns
