Operadores de comparação #

Operador Descrição
== true se os dois valores forem iguais.
!= true se os dois valores não forem iguais.
< true se o valor do operando à esquerda for menor que o valor à direita.
> true se o valor do operando à esquerda for maior que o valor à direita.
>= true se o valor do operando à esquerda for maior que ou igual ao valor à direita.
<= true se o valor do operando à esquerda for menor que ou igual ao valor à direita.
<=> 0 se o valor do operando da esquerda for igual ao valor da direita,
1 se o valor do operando da esquerda for maior que o valor da direita,< BR>-1 se o valor do operando à esquerda for menor que o valor à direita.

Operadores de Atribuição #

Atribuição Simples #

= é uma atribuição simples. Ele cria uma nova variável local se a variável não tiver sido referenciada anteriormente.

x = 3
y = 4 + 5
puts "x is #{x}, y is #{y}"

Isso irá produzir:

x is 3, y is 9

Atribuição Paralela #

As variáveis ​​também podem ser atribuídas em paralelo, por ex. x, y = 3, 9. Isso é especialmente útil para trocar valores:

x, y = 3, 9
x, y = y, x
puts "x is #{x}, y is #{y}"

Isso irá produzir:

x is 9, y is 3

Atribuição Abreviada #

É possível misturar operadores e atribuição. Por exemplo:

x = 1
y = 2
puts "x is #{x}, y is #{y}"

x += y
puts "x is now #{x}"

Mostra a seguinte saída:

x is 1, y is 2
x is now 3

Várias operações podem ser usadas na atribuição abreviada:

Operador Descrição Exemplo Equivalente a
+= Adiciona e reatribui a variável x += y x = x + y
-= Subtrai e reatribui a variável x -= y x = x - y
*= Multiplica e reatribui a variável x *= y x = x * y
/= Divide e reatribui a variável x /= y x = x / y
%= Divide, pega o resto e reatribui a variável x %= y x = x %y
**= Calcula o expoente e reatribui a variável x **= y x = x ** y
