Por que usar R em Weka? #

  1. R é uma ferramenta poderosa para pré-processamento de dados
  2. R tem um grande número de bibliotecas e continua crescendo
  3. R ​​no Weka, pode facilmente obter dados, processá-los e passar para o Weka sem problemas

Como configurar o R ​​no Weka #

Para usuários de Mac

  1. substitua o antigo info.Plist pelo novo fornecido por Mark Hall

  2. baixe e instale o R

  3. instale o rJava dentro do R com

    install.packages(‘rJava’)

  4. instale o Rplugin com o Weka Package Manager

  5. vá para a pasta weka 3-8-0 (se for a versão que você está usando), e abra seu terminal, e

  6. execute as seguintes 2 linhas de códigos (graças a Michael Hall)

    export R_HOME=/Library/Frameworks/R.framework/Resources
    java -Xss10M -Xmx4096M -cp .:weka.jar weka.gui.GUIChooser

  7. para facilitar a vida, dentro de um diretório onde você deseja trabalhar com o weka, salve o código acima em um arquivo chamado weka_r.sh

  8. torná-lo executável, dentro do terminal deste diretório, execute o código abaixo:

    chmod a+x weka_r.sh

  9. cole o weka.jar do weka 3-8-0 no diretório e execute o código abaixo:

    ./weka_r.sh

Agora você está pronto para ir. Da próxima vez, você só precisa ir ao terminal do diretório e executar ./weka_r.sh para iniciar R com Weka.

Como receber dados da Weka? #

abra Weka do terminal: vá para o diretório do Weka 3-8-0, abra seu terminal, execute o seguinte código:

java -jar weka.jar

dados através do Weka Explorer:

  1. painel preprocess, clique em open file, escolha um arquivo de dados da weka data folder;
  2. vá para o painel R console, digite scripts R dentro da caixa R console.

dados através do Weka KnowledgeFlow:

  1. Painel Processos de mineração de dados, clique em DataSources para escolher ArffLoader por exemplo, clique nele na tela;
  2. clique duas vezes em ArffLoader para carregar um arquivo de dados
  3. Painel Scripting, clique em RscriptExecutor na tela
  4. option + clique em ArffLoader, selecione dataset e clique em RScript Executor para vinculá-los
  5. clique duas vezes em RScript Executor para digitar o script R, ou
  6. clique em Settings e selecione R Scripting para usar o console R com os dados do weka

Jogando Códigos R #

  1. carregue iris.arff com Explorer ou KnowledgeFlow;
  2. tente o exemplo Plotting inside R Console acima

Plotando dentro do R Console #

Os seguintes códigos podem ser encontrados em curso Weka

Dado que o iris.arff é carregado no weka, dentro do R console do Weka Explorer ou do R Scripting do Weka KnowledgeFlow, você pode brincar com os seguintes códigos para criar belos gráficos:

library(ggplot2)

ggplot(rdata, aes(x = petallength)) + geom_density()

ggplot(rdata, aes(x = petallength)) + geom_density() + xlim(0,8)

ggplot(rdata, aes(x = petallength)) + geom_density(adjust = 0.5) + xlim(0,8)

ggplot(rdata, aes(x = petallength, color = class)) + geom_density(adjust = 0.5) + xlim(0,8)

ggplot(rdata, aes(x = petallength, color = class, fill = class)) + geom_density(adjust = 0.5) + xlim(0,8)

ggplot(rdata, aes(x = petallength, color = class, fill = class)) + geom_density(adjust = 0.5, alpha = 0.5) + xlim(0,8)





library(reshape2)
ndata = melt(rdata)
ndata

ggplot(ndata, aes(x = value, color = class, fill = class)) + geom_density(adjust = 0.5, alpha = 0.5) + xlim(0,8) + facet_grid(variable ~ .)

ggplot(ndata, aes(x = value, color = class, fill = class)) + geom_density(adjust = 0.5, alpha = 0.5) + xlim(0,8) + facet_grid(. ~ variable)

ggplot(ndata, aes(y = value, x = class, colour = class)) + geom_boxplot() + facet_grid(. ~ variable)
