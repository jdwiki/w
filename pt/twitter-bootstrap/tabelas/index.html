Ordem do conteúdo e tabelas complexas Observe que o estilo de refluxo de tabela altera a ordem visual do conteúdo. Certifique-se de aplicar esse estilo apenas a tabelas de dados bem formadas e simples (e, em particular, não use isso para tabelas de layout) com células de cabeçalho de tabela apropriadas para cada linha e coluna.

Além disso, essa classe não funcionará corretamente para tabelas com células que abrangem várias linhas ou colunas (usando atributos rowspan ou colspan).

Tabela básica #

Bootstrap define um estilo personalizado para tabela usando a classe .table. Basta adicionar a classe .table a qualquer <table> para ver os divisores horizontais e o preenchimento:

<table class="table">
  <thead><tr><th>First Name</th><th>Last name</th></tr></thead>
  <tbody>
    <tr><td>John</td><td>Doe</td></tr>
    <tr><td>Fred</td><td>Bloggs</td></tr>
  </tbody>
</table>

Mesa com estilo avançado #

O Bootstrap fornece algumas classes para estilos avançados de tabelas.

linhas listradas #

Você terá uma tabela com linhas listradas, se adicionar a classe .table-striped:

<table class="table table-striped">
  <thead><tr><th>First Name</th><th>Last name</th></tr></thead>
  <tbody>
    <tr><td>John</td><td>Doe</td></tr>
    <tr><td>Fred</td><td>Bloggs</td></tr>
  </tbody>
</table>

Observe que:

Tabelas distribuídas são estilizadas por meio do seletor CSS :nth-child, que não está disponível no Internet Explorer 8.

#Tabela com borda Você terá uma tabela com bordas em todos os lados da tabela e células, se você adicionar a classe .table-bordered:

<table class="table table-bordered">
  <thead><tr><th>First Name</th><th>Last name</th></tr></thead>
  <tbody>
    <tr><td>John</td><td>Doe</td></tr>
    <tr><td>Fred</td><td>Bloggs</td></tr>
  </tbody>
</table>

Passe o mouse nas linhas #

Se você adicionar a classe .table-hover, você terá uma tabela com linhas destacadas quando o usuário passar o mouse sobre uma linha:

<table class="table table-hover">
  <thead><tr><th>First Name</th><th>Last name</th></tr></thead>
  <tbody>
    <tr><td>John</td><td>Doe</td></tr>
    <tr><td>Fred</td><td>Bloggs</td></tr>
  </tbody>
</table>

Tabela condensada #

Se você adicionar a classe .table-condensed, o preenchimento padrão da célula será cortado pela metade, então você terá uma tabela mais compacta:

<table class="table table-hover">
  <thead><tr><th>First Name</th><th>Last name</th></tr></thead>
  <tbody>
    <tr><td>John</td><td>Doe</td></tr>
    <tr><td>Fred</td><td>Bloggs</td></tr>
  </tbody>
</table>

Classes contextuais #

As tabelas de bootstrap suportam cores contextuais. Para alterar a cor de fundo de uma linha ou célula da tabela, basta adicionar uma das seguintes classes contextuais: .active, .success, .info, .warning, .danger

<table class="table">
  <thead><tr><th>First Name</th><th>Last name</th></tr></thead>
  <tbody>
    <tr class="success"><td>John</td><td>Doe</td></tr>
    <tr><td>Fred</td><td class="info">Bloggs</td></tr>
  </tbody>
</table>

Tabelas responsivas #

Você tem que envolver qualquer .table no container html com a classe .table-responsive para criar tabelas responsivas:

<div class="table-responsive">
  <table class="table">
    <thead><tr><th>First Name</th><th>Last name</th></tr></thead>
    <tbody>
      <tr><td>John</td><td>Doe</td></tr>
      <tr><td>Fred</td><td>Bloggs</td></tr>
    </tbody>
  </table>
</div>

As tabelas responsivas rolarão horizontalmente em dispositivos pequenos (<768px). Não haverá diferenças para telas maiores que 768px de largura.

Refluxo de tabela - cabeçalhos verticais #

Obtendo uma tabela com cabeçalhos verticais.

O bootstrap do Twitter agora suporta cabeçalho vertical em uma tabela normal bem formatada. Para conseguir isso basta usar a classe .table-reflow

Use a classe .table-reflow do bootstrap do twitter em uma tabela bem formada para obter uma tabela com cabeçalhos verticais. Além disso, você pode combinar com o uso de .table-striped e .table-hover para passar o mouse sobre as colunas desta vez.

<table class="table table-striped table-hover table-reflow">
    <thead>
        <tr>
            <th ><strong> First Name: </strong></th>
            <th ><strong> Last Name: </strong></th>
            <th ><strong> Email: </strong></th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td> John </td>
            <td> Doe </td>
            <td> john.doe@email.com </td>
        </tr>
        <tr>
            <td> Joane </td>
            <td> Donald </td>
            <td> jane@email.com </td>
        </tr>
    </tbody>
</table>
