Instalação/Configuração #

Baixando:

  • Faça o download do Bootstrap diretamente ou clone, etc. do repositório GitHub
  • Baixe sua versão personalizada do Bootstrap de docs oficiais
  • Instalar com bower: bower install bootstrap
  • Instale com npm: npm install bootstrap
  • Instale com o compositor: composer requer twbs/bootstrap

A estrutura do arquivo

Instalando:

Dentro da sua página HTML, inclua o CSS do Bootstrap, JS e a dependência do jQuery (pré versão 3, pelo menos a partir da versão mais recente do Bootstrap). Observe que, se você planeja utilizar os recursos JavaScript do Bootstrap, sua referência jQuery deve vir antes de sua referência bootstrap.js em seu HTML.

Você pode utilizar seus arquivos Bootstrap instalados da seção acima ou fazer referência a um CDN fornecido pelos fabricantes do Bootstrap (links retirados de Introdução ao Bootstrap):

<!-- Latest compiled and minified CSS -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css" integrity="sha384-BVYiiSIFeK1dGmJRAkycuHAHRg32OmUcww7on3RYdg4Va+PmSTsz/K68vbdEjh4u" crossorigin="anonymous">

<!-- Optional theme -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap-theme.min.css" integrity="sha384-rHyoN1iRsVXV4nD0JutlnGaslCJuC7uwjduW9SVrLvRYooPp2bWYgmgJQIXwl/Sp" crossorigin="anonymous">

<!-- Latest compiled and minified JavaScript -->
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js" integrity="sha384-Tc5IQib027qvyjSMfHjOMaLkfuWVxZxUPnCJA7l2mCWNIpG9mGCD8wGNIcPD7Txa" crossorigin="anonymous"></script>

Uma página da Web muito básica do Bootstrap:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <!-- The above 3 meta tags *must* come first in the head; any other head content must come *after* these tags -->
    <title>Bootstrap 101 Template</title>

    <!-- Bootstrap -->
    <link href="css/bootstrap.min.css" rel="stylesheet">

    <!-- HTML5 shim and Respond.js for IE8 support of HTML5 elements and media queries -->
    <!-- WARNING: Respond.js doesn't work if you view the page via file:// -->
    <!--[if lt IE 9]>
      <script src="https://oss.maxcdn.com/html5shiv/3.7.2/html5shiv.min.js"></script>
      <script src="https://oss.maxcdn.com/respond/1.4.2/respond.min.js"></script>
    <![endif]-->
  </head>
  <body>

    <!-- jQuery (necessary for Bootstrap's JavaScript plugins) -->

    <!-- Include all compiled plugins (below), or include individual files as needed -->
    <script src="js/bootstrap.min.js"></script>
  </body>
</html>

Quando usar o Bootstrap #

Bootstrap é um framework opinativo para HTML, CSS e Javascript. Ele contém estilo e funcionalidade básicos para o que se tornou elementos aceitos da [interface do usuário], como elementos de formulário, botões, janelas modais e elementos de navegação.

O Bootstrap é um framework web responsivo, o que significa que foi projetado para adaptar o layout e o design para tamanhos de tela grandes e pequenos, como dispositivos móveis, tablets e computadores desktop, tudo em uma única base de código.

Um dos conceitos fundamentais do Bootstrap é o framework de grade. Ao aplicar classes a elementos HTML, é possível criar layouts complexos usando uma grade básica de doze colunas. Por exemplo, um layout de quatro colunas pode se adaptar a duas colunas em tablets e uma coluna em dispositivos móveis. A grade usa consultas de mídia, um método CSS para direcionar tamanhos de tela específicos, para conseguir isso.

O Bootstrap funciona particularmente bem se:

  • O design personalizado não é uma prioridade
  • Você se sente mais confortável editando HTML e adicionando classes do que criando CSS personalizado
  • Você está confortável usando uma estrutura que terá muitas semelhanças visuais com muitos outros sites

Bootstrap pode ser usado por quem é novo em HTML, CSS e Javascript, pois a documentação é excelente. No entanto, há uma curva de aprendizado para aqueles que não estão totalmente familiarizados com as três tecnologias básicas usadas pelo Bootstrap (HTML, CSS e Javascript).

É possível comprar ou baixar temas do Bootstrap para alterar o estilo ou funcionalidade do Bootstrap. Também é possível utilizar o Bootstrap como ponto de partida, com customização de CSS e Javascript.
