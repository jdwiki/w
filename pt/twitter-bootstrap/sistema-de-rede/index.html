O sistema de grade do Bootstrap consiste em 12 unidades conhecidas como Columns (.col-*-* classes CSS) que são usadas para layout de conteúdo da esquerda para a direita na janela de visualização. As colunas estão contidas em Rows (classe CSS .row) para criar grupos horizontais de colunas. As linhas são colocadas em um Contêiner fixo ou de largura total (.container ou .container-fluid, respectivamente) para alinhamento adequado. As colunas têm preenchimento que cria espaçamento (conhecido como “calha”) entre o conteúdo das colunas.

O Bootstrap inclui um sistema de grade de primeiro fluido móvel e responsivo que dimensiona adequadamente até 12 colunas à medida que o tamanho do dispositivo ou da janela de visualização aumenta. Inclui classes predefinidas para criar rapidamente layouts de página por meio de uma série de linhas e colunas que abrigam seu conteúdo.

Linhas e colunas de bootstrap #

O sistema de grade do Bootstrap tem 12 unidades conhecidas como Colunas que podem ser usadas para organizar o conteúdo horizontalmente na janela de visualização.

A razão para uma grade de 12 unidades (em vez de 10, 16, etc.) é que 12 se divide igualmente em 6 (metades), 4 (quartos) e 3 (terços). Isso torna a adaptação a uma variedade de layouts muito mais fácil. As colunas de grade do Bootstrap são identificadas por diferentes classes CSS col-{breakpoint}-{units}. [Saiba mais sobre a largura da janela de visualização e pontos de interrupção (A.K.A. Tiers)](https://www.wikiod.com/pt/twitter-bootstrap/sistema-de-rede#Camadas de grade de bootstrap (pontos de interrupção))

Por exemplo, col-md-3 representa uma coluna que ocupa 3 das 12 unidades (ou 25%) em uma janela de visualização de largura média (md). Para usar uma largura de coluna em seu layout, simplesmente use a classe col-{breakpoint}-{units} apropriada em sua marcação HTML.

<div class="col-{breakpoint}-{units}">

A largura da coluna é fluida (não a largura fixa), portanto, as colunas consomem uma porcentagem de seu contêiner.

Unidades de coluna no Bootstrap 3

  • col-*-1: 1 de 12 (8,33333333% de largura)
  • col-*-2: 2 de 12 (16,66666667% de largura)
  • col-*-3: 3 de 12 (25% de largura)
  • col-*-4: 4 de 12 (33,3333333% de largura)
  • col-*-5: 5 de 12 (41,66666667% de largura)
  • col-*-6: 6 de 12 (50% de largura)
  • col-*-7: 7 de 12 (58,33333333% largura)
  • col-*-8: 8 de 12 (66,66666667% de largura)
  • col-*-9: 9 de 12 (75% de largura)
  • col-*-10: 10 de 12 (83,33333333% de largura)
  • col-*-11: 11 de 12 (91,66666667% de largura)
  • col-*-12: 12 de 12 (100% de largura)

A fila do Bootstrap

A classe Bootstrap .row é usada para conter as Colunas. As colunas devem sempre ser colocadas em Linhas, e as Linhas sempre devem ser colocadas dentro de um Container (container ou container-fluid). A linha usa margens negativas (-15px) para garantir o espaçamento adequado entre o conteúdo da coluna e a borda do navegador. As linhas são usadas para agrupar colunas horizontalmente.

<div class="container">
    <div class="row">
        <!-- one more columns -->
        <div class="col-{breakpoint}-{units}">..</div>
    </div>
</div>

As colunas preencherão o .row horizontalmente da esquerda para a direita e reunirão em uma nova linha a cada 12 unidades de coluna. Portanto, você pode usar .rows para criar quebras horizontais ou adicionar mais de 12 unidades de coluna em um único elemento .row para ter colunas que envolvem ( ou empilhar) verticalmente na viewport.

Ao usar quebra de coluna (mais de 12 unidades em um .row), você precisará usar redefinições responsivas (ou clearfixes) para garantir o envolvimento uniforme do conteúdo de coluna irregular. Isso é essencial quando o conteúdo das colunas varia em altura.

Camadas de grade de inicialização (pontos de interrupção) #

Além do conceito de [unidades de coluna](https://www.wikiod.com/pt/twitter-bootstrap/sistema-de-rede#Linhas e Colunas do Bootstrap), o Bootstrap tem diferentes **pontos de interrupção ** ou tamanhos de grade conhecidos como camadas. A grade do Bootstrap 3 tem quatro (4) camadas para acomodar diferentes larguras de tela (ou viewport). As camadas do Bootstrap 3 são xs, sm, md e lg. As colunas de grade do Bootstrap são identificadas por diferentes classes CSS col-{breakpoint}-{units}.

Cada nível de grade abrange um intervalo projetado para se adequar melhor às larguras de tela típicas de dispositivos, como desktops, laptops, tablets e smartphones.

O Bootstrap usa consultas de mídia CSS para criar pontos de interrupção responsivos que estabelecem um limite para cada tamanho de grade. Esses tamanhos de grade permitem que você altere o layout das colunas para melhor corresponder a diferentes larguras de tela e dispositivos__ a essência do design responsivo.

  • col-xs-* — para as menores larguras de tela, como smartphones < 768 px
  • col-sm-* — para larguras de tela pequenas, como smartphones e tablets >= 768 px
  • col-md-* — para larguras de tela média, como tablets e laptops >= 992 px
  • col-lg-* — para larguras de tela grandes como desktops >= 1200 px

Referência: Sistema de grade

Mesma largura de coluna para cada dispositivo

Para criar uma coluna que seja sempre 50% da largura da janela de visualização (em todos os dispositivos), você pode definir col-*-6 para cada camada.

<div class="col-xs-6 col-sm-6 col-md-6 col-lg-6">..</div>

No entanto, isso é uma marcação extra desnecessária, já que col-xs-6 significa 6 unidades em xs e acima. O menor nível que você definir (xs, sm ou md) também define o tamanho para larguras de tela maiores. Para a coluna mesmo tamanho em todas as camadas, apenas defina a largura da menor janela de visualização.

Código mais curto:

<div class="col-xs-6">..</div>

Largura de coluna diferente para cada dispositivo (design responsivo)

As classes col-*-* podem ser combinadas para controlar as larguras das colunas em diferentes tamanhos de grade.

Por exemplo, crie uma coluna de largura 50% na camada sm e uma coluna de largura 25% na camada md…

<div class="col-md-3 col-sm-6">..</div>

As grades sm, md e lg serão todas “empilhadas” verticalmente em larguras de janela menores que 768 pixels. É aqui que a grade xs se encaixa. As colunas que usam as classes col-xs-* não serão empilhadas verticalmente e continuarão a diminuir nas telas menores.

Manipulação de ordem de coluna usando push e pull #

<div class="container content">
    <div class="row">
        <!--Main Content-->
        <div class="col-lg-9 col-lg-push-3">
            Main Content
        </div>
        
        <!--Sidebar-->
        <div class="col-lg-3 col-lg-pull-9">
            Sidebar
        </div>
    </div>
</div>

Isso altera a ordem das colunas de grade internas.

Sintaxe: .col-md-push-* e .col-md-pull-*.

Recipientes #

O Bootstrap requer um elemento de contenção para envolver o conteúdo do site e abrigar nosso sistema de grade. Você pode escolher um dos dois contêineres para usar em seus projetos.

Use a classe .container para um contêiner de largura fixa responsivo.

<div class="container">
  ...
</div>

Use a classe .container-fluid para um contêiner de largura total, abrangendo toda a largura de sua janela de visualização.

<div class="container-fluid">
  ...
</div>

Nota: Containers não são encaixáveis ​​(você não pode colocar um container dentro de outro container), devido ao padding e mais.

Consultas de mídia #

As consultas de mídia no Bootstrap permitem mover, mostrar e ocultar conteúdo com base no tamanho da janela de visualização. As seguintes consultas de mídia são usadas em arquivos LESS para criar os principais pontos de interrupção no sistema de grade do Bootstrap:

/* Small devices (tablets, 768px and up) */
@media (min-width: @screen-sm-min) { ... }

/* Medium devices (desktops, 992px and up) */
@media (min-width: @screen-md-min) { ... }

/* Large devices (large desktops, 1200px and up) */
@media (min-width: @screen-lg-min) { ... }

Ocasionalmente, eles são expandidos para incluir uma largura máxima para limitar o CSS a um conjunto mais restrito de dispositivos:

@media (max-width: @screen-xs-max) { ... }
@media (min-width: @screen-sm-min) and (max-width: @screen-sm-max) { ... }
@media (min-width: @screen-md-min) and (max-width: @screen-md-max) { ... }
@media (min-width: @screen-lg-min) { ... }

Colunas de compensação #

Essas classes aumentam a margem esquerda de uma coluna em * colunas. Por exemplo, .col-md-offset-4 move .col-md-4 em quatro colunas.

<div class="row">
  <div class="col-lg-4"></div>
  <div class="col-lg-4 col-lg-offset-4"></div>
</div>
<div class="row">
  <div class="col-lg-5 col-lg-offset-1"></div>
  <div class="col-lg-5 col-lg-offset-1"></div>
</div>
