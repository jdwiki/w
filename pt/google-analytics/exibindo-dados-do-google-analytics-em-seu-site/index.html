Exibindo dados do Google Analytics em seu site #

Este documento explica como obter tokens do Google Access e usá-los para que os dados do Google Analytics sejam exibidos em nossos sites.

Exemplo: um exemplo ao vivo está disponível em

https://newtonjoshua.com

nota: Use a mesma conta do Gmail para todas as etapas abaixo.

ETAPA 1: configurar o Google Analytics #

Siga as etapas abaixo para configurar o Google Analytics em seu site

  1. Faça login na sua conta do Analytics.
  2. Selecione a guia Admin.
  3. Selecione uma conta no menu suspenso na coluna CONTA.
  4. Selecione uma propriedade no menu suspenso na coluna PROPERTY.
  5. Em PROPRIEDADE, clique em Informações de Rastreamento > Código de Rastreamento.
  6. Para coletar dados, você deve copiar e colar o código de acompanhamento do Google Analytics into the source code on every web page you wish to track.
  7. Assim que tiver o fragmento do código de acompanhamento Javascript para o seu property, copy the snippet exactly without editing it.
  8. Cole seu snippet de código de acompanhamento (inalterado, na íntegra) antes the closing tag on every web page on your site you wish to track.
  9. Depois de instalar o rastreamento do Google Analytics com sucesso, pode demorar up to 24 hours for data such as traffic referral information, user characteristics, and browsing information to appear in your reports

referir,

  1. https://support.google.com/analytics/answer/1008080?hl=en
  2. https://analytics.google.com

PASSO 2: Obter Tokens #

Projeto Google:

Para criar um projeto do Google Cloud Platform, abra o Google Developers Console ( https://console.developers.google.com ) e clique em Create Project.

Ative o acesso à API OAuth 2.0:

Seu aplicativo precisará acessar os dados do usuário e entrar em contato com outros serviços do Google em seu nome. Use o OAuth 2.0 para conceder acesso à API do seu aplicativo.

Para habilitar isso, você precisa de um ID do cliente:

  1. Abra a página Credenciais do Console de API do Google (https://console.developers.google.com/apis/credentials).

  2. Do projeto menu suspenso, selecione seu projeto.

  3. Selecione Criar credencials e escolha ID do cliente OAuth.

  4. Em Tipo de aplicativo, selecione Aplicativo da Web, insira um Nome e

  5. conjunto the Restrictions by entering JavaScript origins, Redirect URIs to point the website where you are planning to display the data, then click Create.

  6. Anote o OAuth 2.0 client_id and client_secret. You will need them to configure the UI.

Obter Código de Autorização:

Entre no navegador,

https://accounts.google.com/o/oauth2/auth?scope=https://www.googleapis.com/auth/analytics.readonly&response_type=code&client_id={{ client_id}}&redirect_uri={{redirect_uri }} &approval_prompt=force&access_type=offline

Você será redirecionado para

{{redirect_uri }}?code=={{authorization_code}}#

Obter token de atualização:

Envie uma solicitação POST, possivelmente por meio de um console REST para

https://www.googleapis.com/oauth2/v3/token?code={{authorization_code}} &client_id={{client_id}}&client_secret={{client_secret}} &redirect_uri={{redirect_uri }}&grant_type=authorization_code

Você receberá uma resposta JSON com

{“refresh_token”: refresh_token}

Você pode usar o token de atualização para obter o token de acesso às APIs do Google

Obter token de acesso:

Envie uma solicitação POST para,

https://www.googleapis.com/oauth2/v3/token?client_id={{client_id}} &client_secret={{client_id}} &grant_type=refresh_token&refresh_token={{refresh_token}}

Você receberá um JSON com access_token na resposta.

{access_token: {{access_token}}}

Exemplo:

 var access_token = '';
function getAccessToken(){
    $.post('https://www.googleapis.com/oauth2/v3/token', {
            client_id: {{client_id}},
            client_secret: {{client_secret}},
            grant_type: 'refresh_token',
            refresh_token: {{refresh_token}}
        }, function (data, status) {
            if (status === 'success') {
                access_token = data.access_token;
                // Do something eith the access_token
            } else {
                console.error(status);
            }
        });
}

Verifique a validade do token:

Envie uma solicitação POST para,

https://www.googleapis.com/oauth2/v1/tokeninfo?access_token={{access_token}}

Exemplo:

function checkValidity() {
    $.post('https://www.googleapis.com/oauth2/v1/tokeninfo', {
            access_token:{{access_token}}
        }).done(function (data, status) {
            if (status === 'success') {
                console.debug(data.expires_in);
                var check = false;
                check = data.hasOwnProperty('expires_in');
                if (check) {
                    // Token is valid
                }
                if (!check) {
                    getAccessToken();
                }
            } else {
                console.debug(status);
            }

        })
        .fail(function (data) {
            console.error(data);
            getAccessToken();
        });
}

Etapa 3: buscar dados #

API de incorporação:

A GA Embed API é uma biblioteca JavaScript que permite que você crie e incorpore facilmente seu painel GA em seu site em questão de minutos.

consulte https://developers.google.com/analytics/devguides/reporting/embed/v1/getting-started

Explorador de consultas: visite o Embed API Query Explorer e autorize

https://ga-dev-tools.appspot.com/query-explorer/

Selecione a visualização para a qual você deseja buscar os dados.

Selecione as métricas e dimensões necessárias.

Exemplo:

Obter dados do país (quero saber o número de usuários que acessam meu site em cada país)

Para obter esses dados, selecione as métricas como ‘usuários’ e as dimensões como ‘país’

Clique em Executar consulta

Você encontrará os dados analíticos para a consulta exibidos em uma tabela.

Copie o URI de consulta da API. E adicione access_token={{access_token}} ao uri

Exemplo:

https://www.googleapis.com/analytics/v3/data/ga?ids={{ids}}&start-date=2015-07-01&end-date=today&metrics=ga%3Ausers&dimensions=ga%3Acountry &access_token={{access_token}}

Envie uma solicitação POST para os URIs para obter os dados em seu navegador.

Exemplo:

function gaGetCountry() {
    $.get('https://www.googleapis.com/analytics/v3/data/ga?' +
        'ids={{ids}}' +
        'start-date=2015-07-01&' +
        'end-date=today&' +
        'metrics=ga%3Ausers&' +
        'dimensions=ga%3Acountry&' +
        'sort=ga%3Ausers&' +
        'filters=ga%3Ausers%3E10&' +
        'max-results=50' +
        '&access_token=' + {{access_token}},
        function (data, status) {
            if (status === 'success') {

                // Display the Data
                drawRegionsMap(data.rows);

            } else {
                console.debug(status);
            }

        });
}

Etapa 4: exibir dados #

Agora reunimos os dados. Finalmente, temos que exibi-los em nosso site.

“Exibir dados ao vivo em seu site” é o título do Google Charts. E é isso que vamos fazer.

consulte https://developers.google.com/chart/

O exemplo a seguir desenhará um GeoChart no div com id=‘countryChart’

 //Draw country Chart
 function drawRegionsMap(data) {

        var head = data[0];
        head[0] = 'Country';
        head[1] = 'Users';
        for (var i = 1; i < data.length; i++) {
            var d = data[i];
            d[1] = Number(d[1]);
        }

        var chartData = google.visualization.arrayToDataTable(data);
        var options = {
            title: 'My Website is viewed from,',
            domain: '{{Country Code eg: IN for India}}',
            tooltip: {
                textStyle: {
                    color: 'navy'
                },
                showColorCode: true
            },
            legend: {
                textStyle: {
                    color: 'navy',
                    fontSize: 12
                }
            },
            colorAxis: {
                colors: ['#00FFFF', '#0000FF']
            }
        };
    
        var chart = new google.visualization.GeoChart(document.getElementById('countryChart'));
    
        chart.draw(chartData, options); 
}
