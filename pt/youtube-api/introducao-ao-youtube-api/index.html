Introdução às APIs do YouTube #

Atualmente, existem três APIs do YouTube disponíveis ao público:

  1. API de dados do YouTube
  2. API do YouTube Analytics
  3. API de relatórios do YouTube

Cada uma delas oferece funcionalidades diferentes e são tratadas como APIs individuais e separadas.

Como o YouTube é uma subsidiária do Google, as várias APIs do YouTube são fornecidas e mantidas pelo Google. Para acessar qualquer API fornecida pelo Google, são necessárias as seguintes etapas:

  1. Você precisará de uma Conta do Google para acessar o Google Developers Console.
  2. Crie um novo projeto em seu Google Developers Console.
  3. Você pode solicitar uma chave de API e registrar seu aplicativo.
  4. As APIs obrigatórias às quais você precisará acessar precisarão ser ativadas na biblioteca de APIs do Console do Google Cloud.

Quando na tela de credenciais, uma chave de API deve ser gerada. O tipo de chave necessária depende do aplicativo individual.

Existem cinco tipos de chaves de API:

Tipo de chave Descrição Caso de uso / idioma
Nenhum Não tem restrições e é normalmente usado para teste e desenvolvimento aberto a todos os ambientes
Chave do servidor Usado para aplicativos do lado do servidor. A chave é considerada secreta e não pode ser exposta ao público. PHP, Java, Python, Ruby, C, etc.
Chave do navegador Usado para aplicativos do lado do cliente. Como o cliente emitirá solicitações para a API, a chave não pode ser secreta. JavaScript
Chave Android Para uso em um aplicativo Android. Aplicativo Android
chave iOS Para uso em um aplicativo iOS. aplicativo iOS

Depois que uma chave de API é obtida e configurada, você pode usá-la para fazer chamadas para a API.

Para chamadas de API gerais, tudo que você precisa é a chave de API. Mas se você precisar acessar a conta de um usuário para obter mais informações, fazer upload ou geralmente modificar os dados da conta de um usuário, precisará de um “ID do cliente” e um “Segredo do cliente”. Eles são usados ​​para autenticação com a estrutura OAuth2.

API de dados do YouTube #

Essa API, às vezes também chamada de “API v3”, “YouTube Data API v3” ou apenas “YouTube API”, é a API do YouTube mais usada. Ele permite que um aplicativo leia, altere, adicione ou exclua dados relacionados a vídeos, listas de reprodução e canais. Isso inclui, mas não se limita a:

  1. Realizar pesquisas de texto completo para vídeos, canais e/ou listas de reprodução
  2. Recuperando informações sobre um vídeo, canal ou playlist por ID
  3. Carregar vídeos
  4. Editar a descrição, visibilidade e outras informações de um vídeo, canal ou playlist
  5. Gerenciando uma imagem de perfil de canais, banner e outras informações
  6. Lendo, postando e editando comentários

API do YouTube Analytics e API de relatórios do YouTube #

Essas APIs são usadas para consultar análises de vídeo e canal, como visualizações/cliques e votos.
