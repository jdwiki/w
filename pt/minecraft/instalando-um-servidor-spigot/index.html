BuildTools #

O que é isso? #

BuildTools.jar é uma solução para construir Bukkit, CraftBukkit, Spigot e Spigot-API. Tudo isso é feito no seu computador! Alguns programas de pré-requisito são necessários, mas as instruções abaixo irão guiá-lo através de tudo o que você precisa fazer.

Pré-requisitos #

Existem dois aplicativos necessários para usar BuildTools: Git e Java.

janelas #

Git #

Para que o BuildTools seja executado no Windows, você precisará instalar o Git. Para Windows é distribuído via git-scm, que pode ser baixado aqui. Instale-o onde quiser, ele fornecerá o git bash, que será usado para executar o jar do BuildTools. Apenas continue pressionando próximo ao executar o instalador.

Java #

Baixe o JRE 8 de aqui e instale. Apenas continue pressionando próximo ao executar o instalador.

Linux #

Tanto o git quanto o Java, assim como os comandos util, podem ser instalados usando um único comando por meio do gerenciador de pacotes.

Debian/Ubuntu: sudo apt-get install git openjdk-7-jre-headless tar

CentOS/RHEL: sudo dnf install git java-1.7.0-openjdk-devel tar

Arco: pacman -S jdk8-openjdk git

Mac #

Git pode ser baixado de: http://sourceforge.net/projects/git-osx-installer/files/

O Java pode precisar ser atualizado a partir da versão distribuída da Apple e, mesmo se atualizado anteriormente, pode precisar ser vinculado para uso do shell. Siga as etapas encontradas aqui: https://gist.github.com/johan/10590467

Executando BuildTools #

  1. Baixe BuildTools.jar em https://hub.spigotmc.org/jenkins/job/BuildTools/lastSuccessfulBuild/artifact/target/BuildTools.jar.

  2. Abra seu terminal se estiver no Linux ou git bash no Windows.

    1. Git bash can be found on the desktop or in the Start menu under the name “git bash”. It’s also possible to open it by right-clicking on anything, as it is now an item in your context menu.

  3. Navegue até onde você fez download do BuildTools.jar ou use a linha de comando para fazer download do jar para seu diretório atual.

    1. On Windows, you can either use the cd command to change directories, or you can right click the blank space of the folder where BuildTools.jar is (DO NOT click BuildTools.jar itself) and click “git bash”, which will open it in your current directory.

  4. Execute BuildTools.jar a partir do terminal (não clique duas vezes em BuildTools.jar) fazendo o seguinte:

    1. On Linux run git config –global –unset core.autocrlf, then run java -jar BuildTools.jar in bash or another appropriate shell.
    2. On Windows run the below command inside the git bash window that opened: java -jar BuildTools.jar Esteja ciente de que é necessário que você tenha o BuildTools #35 ou posterior, versões mais antigas não funcionarão.
    3. On Mac run the below commands, export MAVEN_OPTS="-Xmx2G" java -Xmx2G -jar BuildTools.jar

  5. Espere enquanto ele constrói seus frascos. Em poucos minutos você deve ter os jars recém-compilados!

  6. Você pode encontrar CraftBukkit e Spigot no mesmo diretório em que executou o BuildTools.jar (craftbukkit-1.10.jar e spigot-1.10.jar). Você pode encontrar o Spigot-API em \Spigot\Spigot-API\target\ (spigot-api-1.10-R0.1-SNAPSHOT.jar).

Instalação da torneira #

janelas #

  1. Obtenha o spigot.jar usando o BuildTools ou aqui.

  2. Cole o texto a seguir em um documento de texto. Salve-o como start.bat no mesmo diretório que spigot.jar: Você precisará renomear seu jar para spigot.jar ou modificar o arquivo no arquivo bat para apontar para o arquivo correto. nb: O Windows (por padrão) ocultará a extensão .jar do arquivo.

     @echo off
 java -Xmx1G -jar spigot.jar
 pause

  3. Clique duas vezes no arquivo de lote.

Linux #

  1. Obtenha o spigot.jar usando o BuildTools ou aqui.

  2. Crie um novo script de inicialização (start.sh) no diretório para ativar o JAR: #!/bin/sh

     java -Xmx1G -jar spigot.jar

  3. Abra seu terminal e execute o seguinte no diretório: chmod +x start.sh

  4. Execute seu script de inicialização:

     ./start.sh

Mac #

  1. Obtenha o spigot.jar usando o BuildTools ou aqui.

  2. Crie um novo script de inicialização (start.command) para iniciar o JAR: #!/bin/sh

     cd "$( dirname "$0" )"
 java -Xmx1G -jar spigot.jar

  3. Abra o Terminal e digite: (Não aperte enter!)

     chmod a+x

  4. Arraste seu arquivo de script de inicialização para a janela Terminal. (Certifique-se de colocar um espaço entre chmod a+x e seu script de inicialização!)

  5. Clique duas vezes no script de inicialização.
