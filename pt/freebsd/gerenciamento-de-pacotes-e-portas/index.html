Pontas:

  • Lembre-se de sempre verificar o arquivo /usr/ports/UPDATING antes de atualizar. Pode haver algumas mudanças significativas nos programas que você usa ou na configuração deles, o que interromperá sua configuração atual.

Construindo e instalando software #

Se você encontrou seu software na árvore de portes, agora é a hora de construí-lo.

Construção e instalação simples com configuração manual #

cd /usr/ports/www/apache24
make
make install

Construção e instalação simples com configuração automática #

cd /usr/ports/www/apache24
make BATCH=yes
make install

Obtendo a árvore de portas #

Encaixe da porta #

portsnap fetch
portsnap extract

atualizando a árvore de ports com o portsnap #

portsnap update

agendar cron job para atualizações diárias #

0 3 * * * root /usr/sbin/portsnap cron

SVN #

cabeça #

cd /usr/ports
svnlite checkout https://svnweb.freebsd.org/ports/head .

trimestralmente #

A equipe do FreeBSD Ports congela a árvore de ports a cada 3 meses. Para obter esta árvore de ports, você pode usar ramificações de ports:

cd /usr/ports
svnlite checkout https://svnweb.freebsd.org/ports/branches/2016Q4 .

Tarball (http ou ftp) #

cd /usr/ports
fetch http://ftp.freebsd.org/pub/FreeBSD/releases/amd64/11.0-RELEASE/ports.txz
tar xJvf ports.txz

Git #

git clone https://github.com/freebsd/freebsd-ports

Pesquisa de software #

pesquisa por palavra-chave #

cd /usr/ports
make search key=apache

pesquisa de nome #

cd /usr/ports
make search name=apache24

Usando fresports #

O site oficial de ports do FreeBSD (http://freshports.org/) oferece uma boa maneira de encontrar ports e todas as informações relacionadas a ele.

Configurando fontes de software #

Se você quiser uma configuração personalizada de portas, você pode configurá-la antes de compilar make config. Todas as configurações de portas são armazenadas em /var/db/ports/${CATEGORY_NAME}/options como makefile.

Configurando www/apache24 #

cd /usr/ports/www/apache24
make config
make
make install

Esta configuração será salva em /var/db/ports/www_apache24/options.

Embalagem #

#Embalagem manual

Você pode fazer seu próprio pacote com base nas portas.

cd /usr/ports/www/apache24
make package BATCH=yes

Este comando irá armazenar seu pacote em /usr/ports/packages/All.

Usando revista de pó #

poudriere é atualmente o construtor de pacotes oficial do FreeBSD.

Instalando o compartimento de pó #

pkg install poudriere
# or
cd /usr/ports/ports-mgmt/poudriere
make
make install

Configurando o compartimento de pó #

A configuração do poudriere é armazenada em /usr/local/etc/poudriere.conf e /usr/local/etc/poudriere.d

Implantando a prisão do barril de pólvora #

poudriere jail -c -j myjail

Atualizando a prisão de poudriere #

poudriere jail -u -j myjail

Implantando a árvore de portas do barril de pólvora #

poudriere ports -c -p myports

Atualizando a árvore de portas do poudriere #

poudriere ports -u -p myports

Construção em massa #

poudriere bulk -j myjail -p myports www/apache24
