Chamando C do Fortran #

O Fortran 2003 introduziu recursos de linguagem que podem garantir a interoperabilidade entre C e Fortran (e para mais linguagens usando C como intermediário). Esses recursos são acessados ​​principalmente através do módulo intrínseco iso_c_binding:

use, intrinsic :: iso_c_binding

A palavra-chave intrínseca aqui garante que o módulo correto seja usado, e não um módulo criado pelo usuário com o mesmo nome.

iso_c_binding dá acesso a parâmetros de tipo interoperable:

integer(c_int) :: foo    ! equivalent of 'int foo' in C
real(c_float) :: bar     ! equivalent of 'float bar' in C

O uso de parâmetros do tipo C garante que os dados possam ser transferidos entre programas C e Fortran.

*A interoperabilidade de caracteres C e Fortran é provavelmente um tópico por si só e, portanto, não discutido aqui *

Para realmente chamar uma função C do Fortran, primeiro a interface deve ser declarada. Isso é essencialmente equivalente ao protótipo da função C e permite que o compilador saiba sobre o número e o tipo dos argumentos, etc. O atributo bind é usado para informar ao compilador o nome da função em C, que pode ser diferente do nome Fortran.

gansos.h

// Count how many geese are in a given flock
int howManyGeese(int flock);

gansos.f90

! Interface to C routine
interface
  integer(c_int) function how_many_geese(flock_num) bind(C, 'howManyGeese')
    ! Interface blocks don't know about their context,
    ! so we need to use iso_c_binding to get c_int definition
    use, intrinsic :: iso_c_binding, only : c_int
    integer(c_int) :: flock_num
  end function how_many_geese
end interface

O programa Fortran precisa ser vinculado à biblioteca C (dependente do compilador, incluir aqui?) que inclui a implementação de howManyGeese(), e então how_many_geese() pode ser chamado a partir do Fortran.

Estruturas C em Fortran #

O atributo bind também pode ser aplicado a tipos derivados:

gansos.h

struct Goose {
   int flock;
   float buoyancy;
}

struct Goose goose_c;

gansos.f90

use, intrinsic :: iso_c_binding, only : c_int, c_float

type, bind(C) :: goose_t
  integer(c_int) :: flock
  real(c_float) :: buoyancy
end type goose_t

type(goose_t) :: goose_f

Os dados agora podem ser transferidos entre goose_c e goose_f. Rotinas C que recebem argumentos do tipo Goose podem ser chamadas do Fortran com type(goose_t).
