Instalação ou Configuração #

Observe que o uso continuado do Sublime Text exige que você compre uma licença e é solicitado que você observe os termos e condições.

O processo de instalação do Sublime Text é diferente para cada plataforma, mas em cada caso você precisa visitar a página de download.

Após instalar o ST3, é comum instalar o gerenciador de pacotes, Package Control.

Mac #

Para Mac, há apenas uma versão do Sublime Text para OS X.

  1. Baixe o arquivo .dmg

  2. Abra o arquivo .dmg

  3. Arraste o pacote Sublime Text 3 para a pasta Aplicativos

  4. Para criar um link simbólico para usar na linha de comando, execute o comando following command at the terminal:

    ln -s “/Applications/Sublime Text.app/Contents/SharedSupport/bin/subl” /usr/local/bin/subl

Janelas #

Para Windows, estão disponíveis versões de 64 bits e 32 bits, portáteis e não portáteis. Você deve conseguir executar a versão de 64 bits se estiver usando uma versão moderna do Windows. Se você estiver tendo problemas para executar a versão de 64 bits, tente a versão de 32 bits.

Portátil ou não portátil? O Sublime Text vem em dois tipos para Windows: normal e portátil. A maioria dos usuários deve ser melhor atendida por uma instalação normal. Use a versão portátil somente se você souber que precisa.

As instalações normais separam os dados entre duas pastas: a pasta de instalação propriamente dita e o diretório de dados (diretório de dados específico do usuário). As instalações normais também integram o Sublime Text com o menu de contexto do Windows Explorer.

As instalações portáteis mantêm todos os arquivos necessários ao Sublime Text em uma única pasta. Esta pasta pode ser movida e o editor ainda funcionará.

Como instalar a versão normal do Sublime Text

  1. Baixe o instalador
  2. Clique duas vezes no instalador

Como instalar a versão portátil do Sublime Text

  1. Baixe os arquivos compactados

  2. Descompacte-os em uma pasta de sua escolha

Você encontrará o executável sublime_text.exe dentro dessa pasta.

#Linux Para Linux, execute este comando em seu terminal para verificar o tipo do seu sistema operacional:

uname -m

Você pode baixar o pacote e descompactá-lo manualmente. Como alternativa, você pode usar a linha de comando.

Ubuntu

  • Para i386

    cd ~
wget http://c758482.r82.cf2.rackcdn.com/sublime-text_build-3083_i386.deb

  • Para x64

    cd ~
wget http://c758482.r82.cf2.rackcdn.com/sublime-text_build-3083_amd64.deb

Outras distribuições Linux

  • Para i386

    cd ~
wget http://c758482.r82.cf2.rackcdn.com/sublime_text_3_build_3083_x32.tar.bz2
tar vxjf sublime_text_3_build_3083_x32.tar.bz2

  • Para x64

    cd ~
wget http://c758482.r82.cf2.rackcdn.com/sublime_text_3_build_3083_x64.tar.bz2
tar vxjf sublime_text_3_build_3083_x64.tar.bz2

Agora devemos mover os arquivos descompactados para um local apropriado.

sudo mv Sublime\ Text\ 3 /opt/
Lastly, we create a symbolic link to use at the command line.

sudo ln -s /opt/Sublime\ Text\ 3/sublime_text /usr/bin/sublime

No Ubuntu, se você também deseja adicionar o Sublime Text ao iniciador do Unity, faça o seguinte:

Primeiro, crie um novo arquivo.

sudo sublime /usr/share/applications/sublime.desktop

Copie o seguinte para ele.

[Desktop Entry]
Version=1.0
Name=Sublime Text 3
# Only KDE 4 seems to use GenericName, so we reuse the KDE strings.
# From Ubuntu's language-pack-kde-XX-base packages, version 9.04-20090413.
GenericName=Text Editor

Exec=sublime
Terminal=false
Icon=/opt/Sublime Text 3/Icon/48x48/sublime_text.png
Type=Application
Categories=TextEditor;IDE;Development
X-Ayatana-Desktop-Shortcuts=NewWindow

[NewWindow Shortcut Group]
Name=New Window
Exec=sublime -n
TargetEnvironment=Unity

Se você registrou sua cópia do Sublime Text, mas toda vez que você a abre, você é solicitado a inserir sua licença, tente executar este comando.

sudo chown -R username:username /home/username/.config /sublime-text-3

Substitua o nome de usuário pelo nome de usuário da sua conta. Isso deve corrigir o erro de permissão no caso de você abrir o Sublime Text como root quando você inseriu a licença pela primeira vez.

Comece com o Tutor Sublime #

Este tutorial é inspirado no clássico vimtutor. Você aprenderá alguns atalhos úteis para trabalhar com o Sublime Text 3. Ao final deste tutorial, você estará familiarizado com os atalhos e recursos mais importantes e usados ​​do ST.

Instalação #

Por meio de Controle de pacote:

  1. Instale o Package Control se ainda não estiver instalado: https://packagecontrol.io/installation#st3
  2. Pressione Cmd+Shift+P para trazer a paleta de comandos em frente
  3. Digite Install Package e pressione enter.
  4. Procure por Sublime Tutor e pressione enter para instalar o plugin.

Instalação manual: #

  1. Verifique se o git já está instalado.
  2. cd no diretório Packages do Sublime Text 3. No Mac, geralmente reside no seguinte caminho: ~/Library/Application Support/Sublime Text 3 /Pacotes/. Alternativamente, você pode abrir através de um item de menu: Preferências > Procurar Pacotes...
  3. Uma vez dentro do diretório Packages, clone este repositório: git clone git@github.org:jai/sublimetutor.git. Alternativamente baixe e extraia a versão mais recente para sua plataforma aqui: https://github.com/jaipandya/SublimeTutor/releases
  4. Reinicie o Sublime Text

Começando #

Se você ainda não o fez, instale o Sublime Tutor usando as etapas de instalação dado anteriormente.

Assim que o Sublime Tutor estiver instalado, pressione Ctrl+Option+K atalho de teclado para abrir este arquivo em Sublime Text. Outra opção é ir para Opção de menu Ajuda > Tutor Sublime para abrir isso.

Via Paleta de Comandos:

  1. Cmd+Shift+P para obter a paleta de comandos em frente.
  2. Digite Sublime Tutor, selecione o primeiro comando que aparece para iniciar o guia interativo.
