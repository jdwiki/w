O Package Control é um gerenciador de pacotes completo que ajuda a descobrir, instalar, atualizar e remover pacotes do Sublime Text.

Package Control é o gerenciador de pacotes do Sublime Text. Ele inclui uma lista de mais de 2.500 pacotes disponíveis para instalação, e os usuários podem adicionar qualquer repositório GitHub ou BitBucket por conta própria. Uma vez instalados, os pacotes são atualizados automaticamente.

O código de exemplo cria a pasta Installed Packages para você (se necessário) e, em seguida, baixa o pacote Package Control.sublime-package para ela. O download será feito por HTTP em vez de HTTPS devido às limitações da biblioteca padrão do Python, porém o arquivo será validado usando SHA-256.

Instalando o Controle de Pacotes #

Se você estiver usando o Sublime Text 3, a maneira mais simples de instalar o Package Control é selecionar Install Package Control no menu Tools ou na Command Palette e o Sublime Text o instalará automaticamente.

Instalando manualmente: #

Para o Sublime Text 2, versões mais antigas do Sublime Text 3, ou se você tiver um problema relacionado ao servidor proxy, o Package Control pode ser instalado usando um script Python que deve ser colado no console do Sublime Text.

  • Abra a Página da Web de Instalação do Controle de Pacotes em seu navegador da Web.
  • Clique na versão do Sublime Text que você precisa.
  • Copie o script Python em sua área de transferência.
  • Abra o console do Sublime Text selecionando Show Console no menu View, ou usando as teclas de atalho ``Ctrl+```, ou selecionando Console: Show na Paleta de Comandos.
  • Cole o script Python no console e pressione a tecla Enter para executar o script.
  • Levará alguns segundos para instalar, mas uma vez instalado um novo buffer Sublime Text será exibido com informações sobre o Package Control.

Você pode ir ao Package Control Web Site e escolher os pacotes que deseja instalar.

Personalizando texto sublime #

Uma vez que você tenha o controle de pacotes instalado, é super fácil instalar qualquer plugin, tema, esquema de cores, sintaxe que você quiser!

  • plugin: execute uma ação (compile seu código menos em css, por exemplo)
  • tema: altere toda a pele do texto sublime (guias, barra lateral, paleta de comandos, etc)
  • esquema de cores: altere a cor do seu código
  • sintaxe: define como o código deve ser destacado.

Como faço para encontrar um pacote #

Você pode pesquisar pacotes neste site: packagecontrol.io, ou simplesmente pesquisar algumas palavras-chave quando decidir que precisa instalar um pacote (veja abaixo).

Depois de escolher qual pacote quero instalar, como faço para instalá-lo? #

A partir do ST, abra a paleta de comandos e digite install package até que package control: install package seja realçado e pressione enter.

Espere um pouco e uma lista com todos os pacotes disponíveis aparecerá. Digite o nome do que você deseja instalar e digite enter. Feito!

Nota: É seguro reiniciar o texto sublime depois de instalar um pacote. Mas torna-se cada vez menos útil (o texto sublime fica melhor, então você quase não precisa agora). Então, apenas reinicie se algo estiver estranho, não apenas porque eu disse isso.

Instalando um pacote não listado #

Por pacote não listado, quero dizer um pacote que não está disponível através do Package Control (ainda). Portanto, você não pode encontrá-lo em packagecontrol.io.

MAS, você ainda pode instalá-lo usando o Package Control, para obter todas as vantagens. Por exemplo, eles serão atualizados automaticamente, assim como uma instalação “regular”.

E é realmente fácil.

  • Encontre seu repositório de pacotes (geralmente estará no GitHub, mas você pode usar um BitBucket) e copie a URL.
  • Procure por Package Control: Add repository na paleta de comandos (ctrl+shift+p)
  • Cole o URL
  • Pressione entrar

E agora, é como se este pacote estivesse no canal padrão (um canal é uma simples lista de pacotes. O padrão é aquele que você recebe… por padrão 😃).

  • Procure por Package Control: Install Package na paleta de comandos (ctrl+shift+p)
  • (talvez seja necessário aguardar alguns segundos) Procure o pacote que deseja instalar
  • Aperte entrar!

É isso! Bem legal, hein? Então, se você é um desenvolvedor de pacotes, não hesite em avisá-los que eles podem fazer desta forma (sempre melhor que o git clone e o git pull 😉).
