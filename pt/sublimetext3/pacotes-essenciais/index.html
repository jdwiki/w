O que está no meu ST (Sublime Text) #

  1. Package Control - Baixe/Instale este plugin para instalar e gerenciar todos os seus outros plugins no sublime.
  2. Git - Mantém o controle de seu sistema de versão git. Também permite que você execute alguns comandos git do próprio ST.
  3. GitGutter - Com o GitGutter, você pode ver quais linhas foram adicionadas, deletadas ou modificadas na sarjeta.
  4. Emmet - Emmet é um plugin útil que economiza tempo fazendo você escrever menos, aumentando assim sua produtividade.
  5. AllAutocomplete - O preenchimento automático padrão do Sublime Text considera apenas palavras presentes no arquivo atual. O plug-in AllAutocomplete, no entanto, pesquisa todos os arquivos abertos para encontrar correspondências enquanto sugere palavras.
  6. Alinhamento - Um plugin muito simples e fácil de usar. Isso ajuda muito quando você revisita o código mais tarde. Destaque as linhas que deseja alinhar e pressione ctrl + alt + a .
  7. BracketHighlighter - Este plugin fornece realce de colchetes para todos os tipos de colchetes na calha.
  8. Material-Theme - Este tema traz a linguagem visual do Material Design para o seu Sublime Text 3.
  9. SublimeLinter - Com isso, seu código pode ser lintado conforme você digita (antes de salvar suas alterações) e quaisquer erros são destacados imediatamente.

Boa codificação :)

Para usuários avançados e avançados #

  • Terminal Sublime. Terminal permite o uso para abrir seu terminal favorito diretamente no arquivo atual ou local do projeto em que você está trabalhando no Sublime Text, com atalhos de teclado úteis. Está disponível através do Package Control.

  • Aprimoramentos do SideBar. Este plug-in adiciona funcionalidades adicionais ao Sublime Text SideBar como opções para Recortar, Copiar e Colar arquivos e pastas, movê-los para Lixeira ou Excluir inteiramente .

  • FileManager Uma alternativa para SideBarEnhancements. Fornece apenas recursos que você usa todos os dias, e esses recursos são otimizados em 100% (preenchimento automático, criação aninhada, navegador de arquivos etc.)
