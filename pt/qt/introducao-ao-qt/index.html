Instalação e configuração no Windows e Linux #

Baixe a versão de código aberto Qt para Linux

Vá para https://www.qt.io/download-open-source/ e clique em Baixar agora, certifique-se de estar baixando o instalador do Qt para Linux.

Um arquivo com o nome qt-unified-linux-x-online.run será baixado e, em seguida, adicione a permissão exec

chmod +x qt-unified-linux-x-online.run

Lembre-se de alterar ‘x’ para a versão real do instalador. Em seguida, execute o instalador

./qt-unified-linux-x-online.run

Baixe a versão de código aberto Qt para Windows

Acesse https://www.qt.io/download-open-source/. A captura de tela a seguir mostra a página de download no Windows:

O que você deve fazer agora depende de qual IDE você vai usar. Se você for usar o Qt Creator, que está incluído no programa de instalação, basta clicar em Download Now e executar o executável.

Se você for usar o Qt no Visual Studio, normalmente o botão Baixar agora também deve funcionar. Certifique-se de que o arquivo baixado seja chamado qt-opensource-windows-x86-msvc2015_64-x.x.x.exe ou qt-opensource-windows-x86-msvc2015_32-x.x.x.exe (onde x.x.x é a versão do Qt, por exemplo 5.7.0). Se não for esse o caso, clique em Exibir todos os downloads e selecione uma das quatro primeiras opções em Windows Host.

Se você for usar Qt in Code::Blocks, clique em View All Downloads e selecione Qt x.x.x for Windows 32-bit (MinGW x.x.x, 1.2 GB) em Windows Host.

Depois de baixar o arquivo de instalação apropriado, execute o executável e siga as instruções abaixo. Observe que você precisa ser um administrador para instalar o Qt. Se você não é um administrador, pode encontrar várias soluções alternativas aqui.

Instale o Qt em qualquer sistema operacional

Depois de baixar o Qt e abrir o programa de instalação, o procedimento de instalação é o mesmo para todos os sistemas operacionais, embora as capturas de tela possam parecer um pouco diferentes. As capturas de tela fornecidas aqui são do Linux.

Faça login com uma conta Qt existente ou crie uma nova:

Selecione um caminho para instalar as bibliotecas e ferramentas do Qt

Selecione a versão da biblioteca e os recursos que você deseja

Após o download e a instalação concluída, vá para o diretório de instalação do Qt e inicie o Qt Creator ou execute-o diretamente na linha de comando.

Olá Mundo #

Neste exemplo, simplesmente criamos e mostramos um botão em uma moldura de janela na área de trabalho. O botão de pressão terá o rótulo Hello world!

Isso representa o programa Qt mais simples possível.

Antes de tudo, precisamos de um arquivo de projeto:

helloworld.pro

QT       += core gui

greaterThan(QT_MAJOR_VERSION, 4): QT += widgets

TARGET = helloworld
TEMPLATE = app

SOURCES += main.cpp
  • QT é usado para indicar quais bibliotecas (módulos Qt) estão sendo usadas neste projeto. Como nosso primeiro aplicativo é uma pequena GUI, precisaremos do QtCore e do QtGui. Como o Qt5 separa o QtWidgets do QtGui, precisamos adicionar a linha greaterThan para compilá-lo com o Qt5.
  • TARGET é o nome do aplicativo ou da biblioteca.
  • TEMPLATE descreve o tipo a ser construído. Pode ser um aplicativo (app), uma biblioteca (lib) ou simplesmente subdiretórios (subdiretórios).
  • SOURCES é uma lista de arquivos de código-fonte a serem usados ​​na construção do projeto.

Também precisamos do main.cpp contendo uma aplicação Qt:

main.cpp

#include <QApplication>
#include <QPushButton>

int main(int argc, char *argv[])
{
    QApplication a(argc, argv);

    QPushButton button ("Hello world!");
    button.show();

    return a.exec(); // .exec starts QApplication and related GUI, this line starts 'event loop'    
}
  • Objeto QApplication. Este objeto gerencia os recursos de toda a aplicação e é necessário para executar qualquer programa Qt que tenha uma GUI. Ele precisa de argv e args porque o Qt aceita alguns argumentos de linha de comando. Ao chamar a.exec() o loop de eventos Qt é iniciado.
  • Objeto QPushButton. O botão de pressão com o rótulo Hello world!. A próxima linha, button.show(), mostra o botão na tela em seu próprio quadro de janela.

Por fim, para executar o aplicativo, abra um prompt de comando e digite o diretório no qual você possui o arquivo .cpp do programa. Digite os seguintes comandos shell para construir o programa.

qmake -project
qmake
make

Aplicação básica com QtCreator e QtDesigner #

QtCreator é, no momento, a melhor ferramenta para criar uma aplicação Qt. Neste exemplo, veremos como criar um aplicativo Qt simples que gerencia um botão e escreve texto.

Para criar uma nova aplicação clique em Arquivo->Novo Arquivo ou Projeto:

Em seguida, escolha Projetos->Aplicativo->Aplicativo Qt Widgets

Então você pode escolher o nome e o caminho do seu projeto:

Em seguida, você pode escolher os kits que você usará. Se você não tiver nenhum kit, o QtCreator criará um kit com sua versão do Qt e o compilador principal do seu computador. Se você não tiver nenhum compilador, poderá instalar um. No Windows: instale o Visual Studio. No Linux/Mac: instale g++ ou clang++.

Então você pode escolher o nome da sua classe de janela principal, a classe herdada, o nome do arquivo correspondente à sua classe de janela principal. Se você está aprendendo ou testando o Qt, você realmente não precisa alterá-los.

O último passo pode ser escolher um subprojeto deste projeto e adicionar um controle de versão como git e svn. Novamente, se for apenas para testes, você não precisa alterá-los.

Em seguida, clique no botão Concluir. Agora você deve estar aqui:

Esta é a base da sua aplicação. se você executá-lo agora clicando em Build->Run ou ctrl+R (por padrão), você verá uma janela vazia.

Você pode clicar em um e arrastar e soltar os objetos. Clique no botão Push e solte-o na janela. Em seguida, pesquise o Label, e faça o mesmo (você tem um filtro no canto superior esquerdo onde você pode escrever o objeto que está procurando).

Aqui você pode ver seu aplicativo com o rótulo e o botão. E se você redimensionar sua janela, o rótulo e o botão também serão redimensionados.

Mas nosso botão ainda não está fazendo nada. Podemos alterá-lo de 2 maneiras diferentes. A primeira é conectar o botão com um método que criamos. Podemos fazer isso com o nome do método connect. Então vamos voltar ao nosso código e ir para mainwindow.cpp agora adicione:

connect(ui->pushButton, SIGNAL(clicked(bool)), this, SLOT(whenButtonIsClicked()));

No construtor da sua MainWindow APÓS o ui->setupUI(this); que inicializa o ui.

Então podemos criar a MainWindow::whenButtonIsClicked() em nossa classe .cpp que poderia alterar o texto do rótulo assim:

void MainWindow::whenButtonIsClicked()
{
    ui->label->setText("the button has been clicked !");
}

E em nosso mainwindow.h, precisamos adicionar:

public slots:
    void whenButtonIsClicked();

Slots públicos significam que esse método pode ser chamado quando um sinal é recebido. connect link o sinal quando clicamos no botão e um método para chamar.

Então, agora, se executarmos nosso aplicativo e clicarmos no botão, obteremos:

O que significa que nossa conexão está funcionando. Mas com o Qt Designer temos uma forma ainda mais simples de fazer isso. Se você quiser fazer de outra forma, remova o connect para desconectar o botão (porque vamos conectá-lo de forma diferente), volte para mainwindow.ui e clique com o botão direito do mouse no botão. Clique em Go to slot… , selecione clicked() e pressione ok.

Então você deve ser movido para esta função:

void MainWindow::on_pushButton_clicked()
{
}

Esta é a função que será chamada quando você clicar no botão. Então você pode adicionar

ui->label->setText("it's even easier !");

Afim disso. Vá para mainwindow.h para salvar a mudança (quando você vai para o slot, ele cria um método vinculado ao sinal que você pediu. Ele define a função no .h mas não a salva. Então você deve ir para o arquivo e salve-o).

E agora quando você executa seu aplicativo e pressiona o botão, você pode ver a nova mensagem (se você ainda vê a antiga, é que você não removeu a conexão).

Nós também podemos adicionar um int, double, etc em nosso rótulo graças ao QVariant que é uma classe incrível que pode converter muitas coisas em muitas outras coisas. Então, à esquerda, adicione um int que aumenta quando pressionamos o botão.

Então o .h:

#ifndef MAINWINDOW_H
#define MAINWINDOW_H

#include <QMainWindow>

namespace Ui {
class MainWindow;
}

class MainWindow : public QMainWindow
{
    Q_OBJECT

public:
    explicit MainWindow(QWidget *parent = 0);
    ~MainWindow();

public slots:
    void whenButtonIsClicked();

private slots:
    void on_pushButton_clicked();

private:
    Ui::MainWindow  *ui;
    double          _smallCounter;
};

#endif // MAINWINDOW_H

O .cpp:

#include "mainwindow.h"
#include "ui_mainwindow.h"

MainWindow::MainWindow(QWidget *parent) :
    QMainWindow(parent),
    ui(new Ui::MainWindow)
{
    ui->setupUi(this);
//    connect(ui->pushButton, SIGNAL(clicked(bool)), this, SLOT(whenButtonIsClicked()));
    _smallCounter = 0.0f;
}

MainWindow::~MainWindow()
{
    delete ui;
}

void MainWindow::whenButtonIsClicked()
{
    ui->label->setText("the button has been clicked !");
}

void MainWindow::on_pushButton_clicked()
{
    _smallCounter += 0.5f;
    ui->label->setText("it's even easier ! " + QVariant(_smallCounter).toString());
}
