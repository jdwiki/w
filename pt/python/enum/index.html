Enums foram adicionados ao Python na versão 3.4 por PEP 435.

Criando um enum (Python 2.4 a 3.3) #

Os enums foram retroportados do Python 3.4 para o Python 2.4 até o Python 3.3. Você pode obter este backport enum34 do PyPI.

pip install enum34

A criação de um enum é idêntica à forma como funciona no Python 3.4+

from enum import Enum

class Color(Enum):
    red = 1
    green = 2
    blue = 3

print(Color.red)  # Color.red    
print(Color(1))  # Color.red    
print(Color['red'])  # Color.red

Iteração #

Enums são iteráveis:

class Color(Enum):
    red = 1
    green = 2
    blue = 3

[c for c in Color]  # [<Color.red: 1>, <Color.green: 2>, <Color.blue: 3>]
