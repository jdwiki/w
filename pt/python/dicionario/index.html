Sintaxe #

  • mydict = {}
  • mydict[k] = valor
  • valor = meudict[k]
  • valor = meudict.get(k)
  • valor = mydict.get(k, “valor_default”)

Parâmetros #

Parâmetro Detalhes
chave A chave desejada para pesquisa
valor O valor a ser definido ou retornado

Itens úteis a serem lembrados ao criar um dicionário:

  • Cada chave deve ser única (caso contrário, será substituída)
  • Cada chave deve ser hashable (pode usar a função hash para fazer o hash; caso contrário, TypeError será lançado)
  • Não existe uma ordem específica para as chaves.

Introdução ao Dicionário #

Um dicionário é um exemplo de key value store também conhecido como Mapping em Python. Ele permite que você armazene e recupere elementos referenciando uma chave. Como os dicionários são referenciados por chave, eles têm pesquisas muito rápidas. Como eles são usados ​​principalmente para referenciar itens por chave, eles não são classificados.

#criando um ditado

Os dicionários podem ser iniciados de várias maneiras:

sintaxe literal #

d = {}                        # empty dict
d = {'key': 'value'}          # dict with initial values

# Also unpacking one or multiple dictionaries with the literal syntax is possible

# makes a shallow copy of otherdict
d = {**otherdict}
# also updates the shallow copy with the contents of the yetanotherdict.
d = {**otherdict, **yetanotherdict}

Também é possível adicionar list e dictionary como valor:

Evitando exceções de KeyError #

Uma armadilha comum ao usar dicionários é acessar uma chave inexistente. Isso normalmente resulta em uma exceção KeyError

mydict = {}
mydict['not there']

Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
KeyError: 'not there'

Uma maneira de evitar erros de chave é usar o método dict.get, que permite especificar um valor padrão a ser retornado no caso de uma chave ausente.

value = mydict.get(key, default_value)

Que retorna mydict[key] se existir, mas de outra forma retorna default_value. Observe que isso não adiciona key ao mydict. Portanto, se você quiser manter esse par de valores de chave, você deve usar mydict.setdefault(key, default_value), que armazena o par de valores de chave.

mydict = {}
print(mydict)
# {}
print(mydict.get("foo", "bar"))
# bar
print(mydict)
# {}
print(mydict.setdefault("foo", "bar"))
# bar
print(mydict)
# {'foo': 'bar'}

Uma maneira alternativa de lidar com o problema é capturar a exceção

try:
    value = mydict[key]
except KeyError:
    value = default_value

Você também pode verificar se a chave está ’no’ dicionário.

if key in mydict:
    value = mydict[key]
else:
    value = default_value

Observe, no entanto, que em ambientes multithread é possível que a chave seja removida do dicionário após a verificação, criando uma condição de corrida em que a exceção ainda pode ser lançada.

Outra opção é usar uma subclasse de dict, collections.defaultdict, que tem um default_factory para criar novas entradas no dict quando recebe uma new_key.

Iterando em um dicionário #

Se você usar um dicionário como um iterador (por exemplo, em uma instrução for), ele percorre as chaves do dicionário. Por exemplo:

d = {'a': 1, 'b': 2, 'c':3}
for key in d:
    print(key, d[key])
# c 3
# b 2
# a 1

O mesmo é verdadeiro quando usado em uma compreensão

print([key for key in d])
# ['c', 'b', 'a']

O método items() pode ser usado para fazer um loop sobre a chave e o valor simultaneamente:

for key, value in d.items():
    print(key, value)
# c 3
# b 2
# a 1

Enquanto o método values() pode ser usado para iterar apenas os valores, como seria de esperar:

for key, value in d.values():
    print(key, value)
    # 3
    # 2
    # 1

Aqui, os métodos keys(), values() e items() retornam listas, e existem três métodos extras iterkeys() itervalues() e iteritems() para retornar iteradores .

Dicionário com valores padrão #

Disponível na biblioteca padrão como defaultdict

from collections import defaultdict

d = defaultdict(int)
d['key']                         # 0
d['key'] = 5
d['key']                         # 5

d = defaultdict(lambda: 'empty')
d['key']                         # 'empty'
d['key'] = 'full'
d['key']                         # 'full'

[*] Alternativamente, se você precisar usar a classe interna dict, using dict.setdefault() permitirá que você crie um padrão sempre que você acessar uma chave que não existia antes:

>>> d = {}
{}
>>> d.setdefault('Another_key', []).append("This worked!")
>>> d
{'Another_key': ['This worked!']}

Tenha em mente que se você tiver muitos valores para adicionar, dict.setdefault() criará uma nova instância do valor inicial (neste exemplo um []) toda vez que for chamado - o que pode criar cargas de trabalho desnecessárias.

[*] Python Cookbook, 3ª edição, por David Beazley e Brian K. Jones (O’Reilly). Copyright 2013 David Beazley e Brian Jones, 978-1-449-34037-7.

Mesclando dicionários #

Considere os seguintes dicionários:

>>> fish = {'name': "Nemo", 'hands': "fins", 'special': "gills"}
>>> dog = {'name': "Clifford", 'hands': "paws", 'color': "red"}

Python 3.5+ #

>>> fishdog = {**fish, **dog}
>>> fishdog
{'hands': 'paws', 'color': 'red', 'name': 'Clifford', 'special': 'gills'}

Como este exemplo demonstra, as chaves duplicadas são mapeadas para seu último valor (por exemplo, “Clifford” substitui “Nemo”).

Python 3.3+ #

>>> from collections import ChainMap
>>> dict(ChainMap(fish, dog))
{'hands': 'fins', 'color': 'red', 'special': 'gills', 'name': 'Nemo'}

Com esta técnica, o valor mais importante tem precedência para uma determinada chave em vez da última (“Clifford” é descartado em favor de “Nemo”).

Python 2.x, 3.x #

>>> from itertools import chain
>>> dict(chain(fish.items(), dog.items()))
{'hands': 'paws', 'color': 'red', 'name': 'Clifford', 'special': 'gills'}

Isso usa o último valor, como acontece com a técnica baseada em ** para mesclagem (“Clifford” substitui “Nemo”).

>>> fish.update(dog)
>>> fish
{'color': 'red', 'hands': 'paws', 'name': 'Clifford', 'special': 'gills'}

dict.update usa o último dict para sobrescrever o anterior.

Acessando valores de um dicionário #

dictionary = {"Hello": 1234, "World": 5678}
print(dictionary["Hello"])

O código acima imprimirá 1234.

A string "Hello" neste exemplo é chamada de key. Ele é usado para pesquisar um valor no dict colocando a chave entre colchetes.

O número 1234 é visto após os respectivos dois pontos na definição dict. Isso é chamado de valor para o qual "Hello" mapeia neste dict.

Procurar um valor como este com uma chave que não existe gerará uma exceção KeyError, interrompendo a execução se não for detectada. Se quisermos acessar um valor sem arriscar um KeyError, podemos usar o método dictionary.get. Por padrão, se a chave não existir, o método retornará None. Podemos passar um segundo valor para retornar em vez de Nenhum no caso de uma pesquisa com falha.

w = dictionary.get("whatever")
x = dictionary.get("whatever", "nuh-uh")

Neste exemplo, w obterá o valor None e x obterá o valor "nuh-uh".

Acessando chaves e valores #

Ao trabalhar com dicionários, muitas vezes é necessário acessar todas as chaves e valores no dicionário, seja em um loop for, uma compreensão de lista ou apenas como uma lista simples.

Dado um dicionário como:

mydict = {
    'a': '1',
    'b': '2'
}

Você pode obter uma lista de chaves usando o método keys():

print(mydict.keys())
# Python2: ['a', 'b']
# Python3: dict_keys(['b', 'a'])

Se, em vez disso, você quiser uma lista de valores, use o método values():

print(mydict.values())
# Python2: ['1', '2']
# Python3: dict_values(['2', '1'])

Se você quiser trabalhar com a chave e seu valor correspondente, você pode usar o método items():

print(mydict.items())
# Python2: [('a', '1'), ('b', '2')]
# Python3: dict_items([('b', '2'), ('a', '1')])

OBSERVAÇÃO: Como um dict não é classificado, keys(), values() e items() não têm ordem de classificação. Use sort(), sorted() ou um OrderedDict se você se importa com a ordem que esses métodos retornam.

Python 2/3 Difference: No Python 3, esses métodos retornam objetos iteráveis ​​especiais, não listas, e são equivalentes ao Python 2 iterkeys(), itervalues() e iteritems() métodos. Esses objetos podem ser usados ​​como listas na maioria das vezes, embora haja algumas diferenças. Consulte PEP 3106 para obter mais detalhes.

Criando um dicionário ordenado #

Você pode criar um dicionário ordenado que seguirá uma ordem determinada ao iterar sobre as chaves no dicionário.

Use OrderedDict do módulo coleções. Isso sempre retornará os elementos do dicionário na ordem de inserção original quando for iterado.

from collections import OrderedDict

d = OrderedDict()
d['first'] = 1
d['second'] = 2
d['third'] = 3
d['last'] = 4

# Outputs "first 1", "second 2", "third 3", "last 4"
for key in d:
    print(key, d[key])

Descompactando dicionários usando o operador ** #

Você pode usar o operador de descompactação do argumento da palavra-chave ** para entregar os pares chave-valor em um dicionário nos argumentos de uma função. Um exemplo simplificado da documentação oficial:

>>>
>>> def parrot(voltage, state, action):
...     print("This parrot wouldn't", action, end=' ')
...     print("if you put", voltage, "volts through it.", end=' ')
...     print("E's", state, "!")
...
>>> d = {"voltage": "four million", "state": "bleedin' demised", "action": "VOOM"}
>>> parrot(**d)

This parrot wouldn't VOOM if you put four million volts through it. E's bleedin' demised !

A partir do Python 3.5, você também pode usar essa sintaxe para mesclar um número arbitrário de objetos dict.

>>> fish = {'name': "Nemo", 'hands': "fins", 'special': "gills"}
>>> dog = {'name': "Clifford", 'hands': "paws", 'color': "red"}
>>> fishdog = {**fish, **dog}
>>> fishdog

{'hands': 'paws', 'color': 'red', 'name': 'Clifford', 'special': 'gills'}

Como este exemplo demonstra, as chaves duplicadas são mapeadas para seu último valor (por exemplo, “Clifford” substitui “Nemo”).

Criando um dicionário #

Regras para criar um dicionário:

  • Cada chave deve ser única (caso contrário, será substituída)
  • Cada chave deve ser hashable (pode usar a função hash para fazer o hash; caso contrário, TypeError será lançado)
  • Não existe uma ordem específica para as chaves.
# Creating and populating it with values
stock = {'eggs': 5, 'milk': 2}

# Or creating an empty dictionary
dictionary = {}

# And populating it after
dictionary['eggs'] = 5
dictionary['milk'] = 2

# Values can also be lists
mydict = {'a': [1, 2, 3], 'b': ['one', 'two', 'three']}

# Use list.append() method to add new elements to the values list
mydict['a'].append(4)   # => {'a': [1, 2, 3, 4], 'b': ['one', 'two', 'three']}
mydict['b'].append('four')  # => {'a': [1, 2, 3, 4], 'b': ['one', 'two', 'three', 'four']}

# We can also create a dictionary using a list of two-items tuples
iterable = [('eggs', 5), ('milk', 2)]
dictionary = dict(iterables)

# Or using keyword argument:
dictionary = dict(eggs=5, milk=2)

# Another way will be to use the dict.fromkeys:
dictionary = dict.fromkeys((milk, eggs))  # => {'milk': None, 'eggs': None}
dictionary = dict.fromkeys((milk, eggs), (2, 5))  # => {'milk': 2, 'eggs': 5}

O construtor dict() #

O construtor dict() pode ser usado para criar dicionários a partir de argumentos de palavras-chave, ou de um único iterável de pares de valores-chave, ou de um único dicionário e argumentos de palavras-chave.

dict(a=1, b=2, c=3)                   # {'a': 1, 'b': 2, 'c': 3}
dict([('d', 4), ('e', 5), ('f', 6)])  # {'d': 4, 'e': 5, 'f': 6}
dict([('a', 1)], b=2, c=3)            # {'a': 1, 'b': 2, 'c': 3}
dict({'a' : 1, 'b' : 2}, c=3)         # {'a': 1, 'b': 2, 'c': 3}

Todas as combinações de valores do dicionário #

options = {
    "x": ["a", "b"],
    "y": [10, 20, 30]
}

Dado um dicionário como o mostrado acima, onde há uma lista representando um conjunto de valores a serem explorados para a chave correspondente. Suponha você quer explorar "x"="a" com "y"=10, então "x"="a" com"y"=10, e assim até que você tenha explorado todas as combinações possíveis.

Você pode criar uma lista que retorne todas essas combinações de valores usando o código a seguir.

import itertools

options = {
    "x": ["a", "b"],
    "y": [10, 20, 30]}

keys = options.keys()
values = (options[key] for key in keys)
combinations = [dict(zip(keys, combination)) for combination in itertools.product(*values)]
print combinations

Isso nos dá a seguinte lista armazenada na variável combinations:

[{'x': 'a', 'y': 10},
 {'x': 'b', 'y': 10},
 {'x': 'a', 'y': 20},
 {'x': 'b', 'y': 20},
 {'x': 'a', 'y': 30},
 {'x': 'b', 'y': 30}]
