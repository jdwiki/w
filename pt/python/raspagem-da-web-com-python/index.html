Web scraping é um processo automatizado e programático por meio do qual os dados podem ser constantemente “raspados” de páginas da web. Também conhecido como screen scraping ou web harvesting, o web scraping pode fornecer dados instantâneos de qualquer página da Web publicamente acessível. Em alguns sites, a raspagem da web pode ser ilegal.

Pacotes Python úteis para web scraping (ordem alfabética) #

Fazendo solicitações e coletando dados #

requests #

Um pacote simples, mas poderoso para fazer solicitações HTTP.

requests-cache #

Cache para requests; cache de dados é muito útil. No desenvolvimento, isso significa que você pode evitar acessar um site desnecessariamente. Ao executar uma coleção real, isso significa que se o seu scraper travar por algum motivo (talvez você não tenha lidado com algum conteúdo incomum no site…? talvez o site tenha caído…?) você pode repetir a coleção muito rapidamente de onde você parou.

scrapy #

Útil para construir rastreadores da web, onde você precisa de algo mais poderoso do que usar requests e iterar pelas páginas.

selenium #

Associações Python para Selenium WebDriver, para automação do navegador. Usar requests para fazer solicitações HTTP diretamente é geralmente mais simples para recuperar páginas da web. No entanto, esta continua sendo uma ferramenta útil quando não é possível replicar o comportamento desejado de um site usando apenas requests, particularmente quando JavaScript é necessário para renderizar elementos em uma página.

Análise HTML #

BeautifulSoup #

Consultar documentos HTML e XML, usando vários analisadores diferentes (Parser HTML integrado do Python, html5lib, lxml ou lxml.html)

lxml #

Processa HTML e XML. Pode ser usado para consultar e selecionar conteúdo de documentos HTML por meio de seletores CSS e XPath.

Raspagem usando o framework Scrapy #

Primeiro você precisa configurar um novo projeto Scrapy. Digite um diretório onde você gostaria de armazenar seu código e execute:

scrapy startproject projectName

Para raspar precisamos de uma aranha. As aranhas definem como um determinado site será raspado. Aqui está o código para um spider que segue os links para as perguntas mais votadas no StackOverflow e extrai alguns dados de cada página (source):

import scrapy

class StackOverflowSpider(scrapy.Spider):
    name = 'stackoverflow'  # each spider has a unique name
    start_urls = ['http://stackoverflow.com/questions?sort=votes']  # the parsing starts from a specific set of urls

    def parse(self, response):  # for each request this generator yields, its response is sent to parse_question
        for href in response.css('.question-summary h3 a::attr(href)'):  # do some scraping stuff using css selectors to find question urls 
            full_url = response.urljoin(href.extract())
            yield scrapy.Request(full_url, callback=self.parse_question)

    def parse_question(self, response): 
        yield {
            'title': response.css('h1 a::text').extract_first(),
            'votes': response.css('.question .vote-count-post::text').extract_first(),
            'body': response.css('.question .post-text').extract_first(),
            'tags': response.css('.question .post-tag::text').extract(),
            'link': response.url,
        }

Salve suas classes de aranha no diretório projectName\spiders. Neste caso - projectName\spiders\stackoverflow_spider.py.

Agora você pode usar sua aranha. Por exemplo, tente executar (no diretório do projeto):

scrapy crawl stackoverflow

Exemplo básico de uso de solicitações e lxml para raspar alguns dados #

# For Python 2 compatibility.
from __future__ import print_function

import lxml.html
import requests


def main():
    r = requests.get("https://httpbin.org")
    html_source = r.text
    root_element = lxml.html.fromstring(html_source)
    # Note root_element.xpath() gives a *list* of results.
    # XPath specifies a path to the element we want.
    page_title = root_element.xpath('/html/head/title/text()')[0]
    print(page_title)

if __name__ == '__main__':
    main()

Mantendo a sessão de web-scraping com solicitações #

É uma boa ideia manter uma sessão de raspagem da web para persistir os cookies e outros parâmetros. Além disso, pode resultar em uma melhoria de desempenho porque requests.Session reutiliza a conexão TCP subjacente a um host:

import requests

with requests.Session() as session:
    # all requests through session now have User-Agent header set
    session.headers = {'User-Agent': 'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_11_4) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/51.0.2704.103 Safari/537.36'}

    # set cookies
    session.get('http://httpbin.org/cookies/set?key=value')

    # get cookies
    response = session.get('http://httpbin.org/cookies')
    print(response.text)

Raspagem usando Selenium WebDriver #

Alguns sites não gostam de ser raspados. Nesses casos, pode ser necessário simular um usuário real trabalhando com um navegador. O Selenium inicia e controla um navegador da web.

from selenium import webdriver

browser = webdriver.Firefox()  # launch firefox browser

browser.get('http://stackoverflow.com/questions?sort=votes')  # load url

title = browser.find_element_by_css_selector('h1').text  # page title (first h1 element)

questions = browser.find_elements_by_css_selector('.question-summary')  # question list

for question in questions:  # iterate over questions
    question_title = question.find_element_by_css_selector('.summary h3 a').text
    question_excerpt = question.find_element_by_css_selector('.summary .excerpt').text
    question_vote = question.find_element_by_css_selector('.stats .vote .votes .vote-count-post').text
    
    print "%s\n%s\n%s votes\n-----------\n" % (question_title, question_excerpt, question_vote)

O selênio pode fazer muito mais. Ele pode modificar os cookies do navegador, preencher formulários, simular cliques do mouse, fazer capturas de tela de páginas da web e executar JavaScript personalizado.

Raspagem usando BeautifulSoup4 #

from bs4 import BeautifulSoup
import requests

# Use the requests module to obtain a page
res = requests.get('https://www.codechef.com/problems/easy')

# Create a BeautifulSoup object
page = BeautifulSoup(res.text, 'lxml')   # the text field contains the source of the page

# Now use a CSS selector in order to get the table containing the list of problems
datatable_tags = page.select('table.dataTable')  # The problems are in the <table> tag,
                                                 # with class "dataTable"
# We extract the first tag from the list, since that's what we desire
datatable = datatable_tags[0]
# Now since we want problem names, they are contained in <b> tags, which are
# directly nested under <a> tags
prob_tags = datatable.select('a > b')
prob_names = [tag.getText().strip() for tag in prob_tags]

print prob_names

Modificar o agente de usuário do Scrapy #

Às vezes, o agente de usuário padrão do Scrapy (“Scrapy/VERSION (+http://scrapy.org)"`) é bloqueado pelo host. Para alterar o agente de usuário padrão, abra settings.py, remova o comentário e edite a seguinte linha para o que você quiser.

#USER_AGENT = 'projectName (+http://www.yourdomain.com)'

Por exemplo

USER_AGENT = 'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_11_4) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/51.0.2704.103 Safari/537.36'

Download simples de conteúdo da web com urllib.request #

O módulo de biblioteca padrão urllib.request pode ser usado para baixar conteúdo da web:

from urllib.request import urlopen

response = urlopen('http://stackoverflow.com/questions?sort=votes')    
data = response.read()

# The received bytes should usually be decoded according the response's character set
encoding = response.info().get_content_charset()
html = data.decode(encoding)

Raspagem com ondulação #

importações:

from subprocess import Popen, PIPE
from lxml import etree
from io import StringIO

Baixando:

user_agent = 'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_11_6) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/55.0.2883.95 Safari/537.36'
url = 'http://stackoverflow.com'
get = Popen(['curl', '-s', '-A', user_agent, url], stdout=PIPE)
result = get.stdout.read().decode('utf8')

-s: download silencioso

-A: sinalizador do agente do usuário

Análise:

tree = etree.parse(StringIO(result), etree.HTMLParser())
divs = tree.xpath('//div')
