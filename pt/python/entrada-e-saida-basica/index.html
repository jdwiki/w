Usando a função de impressão #

No Python 3, a funcionalidade de impressão está na forma de uma função:

print("This string will be displayed in the output")
# This string will be displayed in the output

print("You can print \n escape characters too.")
# You can print escape characters too.

No Python 2, print era originalmente uma instrução, conforme mostrado abaixo.

print "This string will be displayed in the output"
# This string will be displayed in the output

print "You can print \n escape characters too."
# You can print escape characters too.

Nota: usar from __future__ import print_function no Python 2 permitirá que os usuários usem a função print() da mesma forma que o código Python 3. Isso só está disponível no Python 2.6 e superior.

Usando input() e raw_input() #

raw_input esperará que o usuário digite o texto e então retornará o resultado como uma string.

foo = raw_input("Put a message here that asks the user for input")

No exemplo acima, o foo armazenará qualquer entrada que o usuário fornecer.

input esperará que o usuário digite o texto e então retorne o resultado como uma string.

foo = input("Put a message here that asks the user for input")

No exemplo acima, o foo armazenará qualquer entrada que o usuário fornecer.

Função para solicitar ao usuário um número #

def input_number(msg, err_msg=None):
    while True:
        try:
            return float(raw_input(msg))
        except ValueError:
            if err_msg is not None:
                print(err_msg)


def input_number(msg, err_msg=None):
    while True:
        try:
            return float(input(msg))
        except ValueError:
            if err_msg is not None:
                print(err_msg)

E para usar:

user_number = input_number("input a number: ", "that's not a number!")

Ou, se você não quiser uma “mensagem de erro”:

user_number = input_number("input a number: ")

Imprimindo uma string sem uma nova linha no final #

No Python 2.x, para continuar uma linha com print, termine a instrução print com uma vírgula. Ele adicionará automaticamente um espaço.

print "Hello,",
print "World!"
# Hello, World!

No Python 3.x, a função print tem um parâmetro opcional end que é impresso no final da string fornecida. Por padrão, é um caractere de nova linha, então equivalente a isso:

print("Hello, ", end="\n")
print("World!")
# Hello, 
# World!

Mas você poderia passar em outras cordas

print("Hello, ", end="")
print("World!")
# Hello, World!

print("Hello, ", end="<br>")
print("World!")
# Hello, <br>World!

print("Hello, ", end="BREAK")
print("World!")
# Hello, BREAKWorld!

Se você quiser mais controle sobre a saída, você pode usar sys.stdout.write:

import sys

sys.stdout.write("Hello, ")
sys.stdout.write("World!")
# Hello, World!

Ler de stdin #

Os programas Python podem ler de pipelines do unix. Aqui está um exemplo simples de como ler de stdin:

import sys

for line in sys.stdin:
    print(line)

Esteja ciente de que sys.stdin é um fluxo. Isso significa que o loop for só terminará quando o fluxo terminar.

Agora você pode canalizar a saída de outro programa em seu programa python da seguinte maneira:

$ cat myfile | python myprogram.py

Neste exemplo, cat myfile pode ser qualquer comando unix que seja enviado para stdout.

Alternativamente, usar o módulo fileinput pode ser útil:

import fileinput
for line in fileinput.input():
    process(line)

Entrada de um arquivo #

A entrada também pode ser lida de arquivos. Os arquivos podem ser abertos usando a função interna open. Usar uma sintaxe com <comando> como <nome> (chamada de ‘Gerenciador de Contexto’) torna o uso de open e obter um identificador para o arquivo super fácil:

with open('somefile.txt', 'r') as fileobj:
    # write code here using fileobj

Isso garante que quando a execução do código sair do bloco, o arquivo seja fechado automaticamente.

Os arquivos podem ser abertos em diferentes modos. No exemplo acima, o arquivo é aberto como somente leitura. Para abrir um arquivo existente apenas para leitura, use r. Se você quiser ler esse arquivo como bytes, use rb. Para anexar dados a um arquivo existente, use a. Use w para criar um arquivo ou sobrescrever qualquer arquivo existente com o mesmo nome. Você pode usar r+ para abrir um arquivo para leitura e escrita. O primeiro argumento de open() é o nome do arquivo, o segundo é o modo. Se o modo for deixado em branco, o padrão será r.

# let's create an example file:
with open('shoppinglist.txt', 'w') as fileobj:
    fileobj.write('tomato\npasta\ngarlic')

with open('shoppinglist.txt', 'r') as fileobj:
    # this method makes a list where each line 
    # of the file is an element in the list
    lines = fileobj.readlines()

print(lines)
# ['tomato\n', 'pasta\n', 'garlic']

with open('shoppinglist.txt', 'r') as fileobj:
    # here we read the whole content into one string:
    content = fileobj.read()
    # get a list of lines, just like int the previous example:
    lines = content.split('\n')

print(lines)
# ['tomato', 'pasta', 'garlic']

Se o tamanho do arquivo for pequeno, é seguro ler todo o conteúdo do arquivo na memória. Se o arquivo for muito grande, geralmente é melhor ler linha por linha ou por partes e processar a entrada no mesmo loop. Fazer isso:

with open('shoppinglist.txt', 'r') as fileobj:
    # this method reads line by line:
    lines = []
    for line in fileobj:
        lines.append(line.strip())

Ao ler arquivos, esteja ciente dos caracteres de quebra de linha específicos do sistema operacional. Embora for line in fileobj automaticamente os retire, é sempre seguro chamar strip() nas linhas lidas, como mostrado acima.

Arquivos abertos (fileobj nos exemplos acima) sempre apontam para um local específico no arquivo. Quando eles são abertos pela primeira vez, o identificador do arquivo aponta para o início do arquivo, que é a posição ‘0’. O identificador de arquivo pode exibir sua posição atual com tell:

fileobj = open('shoppinglist.txt', 'r')
pos = fileobj.tell()
print('We are at %u.' % pos) # We are at 0.

Ao ler todo o conteúdo, a posição do manipulador de arquivos será apontada no final do arquivo:

content = fileobj.read()
end = fileobj.tell()
print('This file was %u characters long.' % end)
# This file was 22 characters long.
fileobj.close()

A posição do manipulador de arquivos pode ser definida para o que for necessário:

fileobj = open('shoppinglist.txt', 'r')
fileobj.seek(7)
pos = fileobj.tell()
print('We are at character #%u.' % pos)

Você também pode ler qualquer tamanho do conteúdo do arquivo durante uma determinada chamada. Para fazer isso passe um argumento para read(). Quando read() é chamado sem argumento, ele será lido até o final do arquivo. Se você passar um argumento, ele lerá esse número de bytes ou caracteres, dependendo do modo (rb e r respectivamente):

# reads the next 4 characters 
# starting at the current position
next4 = fileobj.read(4)
# what we got?
print(next4) # 'cucu'
# where we are now?
pos = fileobj.tell()
print('We are at %u.' % pos) # We are at 11, as we was at 7, and read 4 chars.

fileobj.close()

Para demonstrar a diferença entre caracteres e bytes:

with open('shoppinglist.txt', 'r') as fileobj:
    print(type(fileobj.read())) # <class 'str'>

with open('shoppinglist.txt', 'rb') as fileobj:
    print(type(fileobj.read())) # <class 'bytes'>
