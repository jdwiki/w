  • Codificação e Decodificação Base85

A codificação Base 64 representa um esquema comum para codificar binário no formato de string ASCII usando a base 64. O módulo base64 é parte da biblioteca padrão, o que significa que ele é instalado junto com o Python. A compreensão de bytes e strings é fundamental para este tópico e pode ser revisada aqui. Este tópico explica como usar os vários recursos e bases numéricas do módulo base64.

Sintaxe #

  • base64.b64encode(s, altchars=Nenhum)
  • base64.b64decode(s, altchars=Nenhum, validar=Falso)
  • base64.standard_b64encode(s)
  • base64.standard_b64decode(s)
  • base64.urlsafe_b64encode(s)
  • base64.urlsafe_b64decode(s)
  • base64.b32encode(s)
  • base64.b32decode(s)
  • base64.b16encode(s)
  • base64.b16decode(s)
  • base64.a85encode(b, foldspaces=False, wrapcol=0, pad=False, adobe=False)
  • base64.a85decode(b, foldpaces=False, adobe=False, ignorechars=b’\t\n\r\v')
  • base64.b85encode(b, pad=Falso)
  • base64.b85decode(b)

Parâmetros #

Parâmetro Descrição
base64.b64encode(s, altchars=Nenhum)  
s Um objeto semelhante a bytes
altchars Um objeto semelhante a bytes de comprimento 2+ de caracteres para substituir os caracteres ‘+’ e ‘=’ ao criar o alfabeto Base64. Caracteres extras são ignorados.
base64.b64decode(s, altchars=Nenhum, validate=False)  
s Um objeto semelhante a bytes
altchars Um objeto semelhante a bytes de comprimento 2+ de caracteres para substituir os caracteres ‘+’ e ‘=’ ao criar o alfabeto Base64. Caracteres extras são ignorados.
validar Se valide for True, os caracteres que não estão no alfabeto normal Base64 ou no alfabeto alternativo não são descartados antes da verificação de preenchimento
base64.standard_b64encode(s)  
s Um objeto semelhante a bytes
base64.standard_b64decode(s)  
s Um objeto semelhante a bytes
base64.urlsafe_b64encode(s)  
s Um objeto semelhante a bytes
base64.urlsafe_b64decode(s)  
s Um objeto semelhante a bytes
b32encode(s)  
s Um objeto semelhante a bytes
b32decode(s)  
s Um objeto semelhante a bytes
base64.b16encode(s)  
s Um objeto semelhante a bytes
base64.b16decode(s)  
s Um objeto semelhante a bytes
base64.a85encode(b, foldspaces=False, wrapcol=0, pad=False, adobe=False)  
b Um objeto semelhante a bytes
espaços de dobra Se foldspaces for True, o caractere ‘y’ será usado ao invés de 4 espaços consecutivos.

Até o lançamento do Python 3.4, as funções de codificação e decodificação base64 só funcionavam com tipos bytes ou bytearray. Agora, essas funções aceitam qualquer objeto semelhante a bytes.

Codificação e Decodificação Base64 #

Para incluir o módulo base64 em seu script, você deve importá-lo primeiro:

import base64

As funções de codificação e decodificação base64 exigem um objeto semelhante a bytes. Para obter nossa string em bytes, devemos codificá-la usando a função codificar incorporada do Python. Mais comumente, a codificação UTF-8 é usada, no entanto, uma lista completa dessas codificações padrão (incluindo idiomas com caracteres diferentes) pode ser encontrada [aqui](https://docs.python.org/3/library/codecs .html#standard-encodings) na documentação oficial do Python. Abaixo está um exemplo de codificação de uma string em bytes:

s = "Hello World!"
b = s.encode("UTF-8")

A saída da última linha seria:

b'Olá Mundo!'

O prefixo b é usado para denotar que o valor é um objeto de bytes.

Para codificar esses bytes em Base64, usamos a função base64.b64encode():

import base64
s = "Hello World!"
b = s.encode("UTF-8")
e = base64.b64encode(b)
print(e)

Esse código produziria o seguinte:

b'SGVsbG8gV29ybGQh'

que ainda está no objeto bytes. Para obter uma string desses bytes, podemos usar o método decode() do Python com a codificação UTF-8:

import base64
s = "Hello World!"
b = s.encode("UTF-8")
e = base64.b64encode(b)
s1 = e.decode("UTF-8")
print(s1)

A saída seria então:

SGVsbG8gV29ybGQh

Se quiséssemos codificar a string e depois decodificá-la, poderíamos usar o método base64.b64decode():

import base64
# Creating a string
s = "Hello World!"
# Encoding the string into bytes
b = s.encode("UTF-8")
# Base64 Encode the bytes
e = base64.b64encode(b)
# Decoding the Base64 bytes to string
s1 = e.decode("UTF-8")
# Printing Base64 encoded string
print("Base64 Encoded:", s1)
# Encoding the Base64 encoded string into bytes
b1 = s1.encode("UTF-8")
# Decoding the Base64 bytes
d = base64.b64decode(b1)
# Decoding the bytes to string
s2 = d.decode("UTF-8")
print(s2)

Como você pode ter esperado, a saída seria a string original:

Base64 Encoded: SGVsbG8gV29ybGQh
Hello World!

Codificação e Decodificação ASCII85 #

A Adobe criou sua própria codificação chamada ASCII85 que é semelhante ao Base85, mas tem suas diferenças. Essa codificação é usada com frequência em arquivos Adobe PDF. Essas funções foram lançadas no Python versão 3.4. Caso contrário, as funções base64.a85encode() e base64.a85encode() são semelhantes às anteriores:

import base64
# Creating a string
s = "Hello World!"
# Encoding the string into bytes
b = s.encode("UTF-8")
# ASCII85 Encode the bytes
e = base64.a85encode(b)
# Decoding the ASCII85 bytes to string
s1 = e.decode("UTF-8")
# Printing ASCII85 encoded string
print("ASCII85 Encoded:", s1)
# Encoding the ASCII85 encoded string into bytes
b1 = s1.encode("UTF-8")
# Decoding the ASCII85 bytes
d = base64.a85decode(b1)
# Decoding the bytes to string
s2 = d.decode("UTF-8")
print(s2)

Isso gera o seguinte:

ASCII85 Encoded: 87cURD]i,"Ebo80
Hello World!
