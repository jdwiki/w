“Arrays” em Python não são os arrays em linguagens de programação convencionais como C e Java, mas mais próximos de listas. Uma lista pode ser uma coleção de elementos homogêneos ou heterogêneos e pode conter ints, strings ou outras listas.

Acesse elementos individuais por meio de índices #

Elementos individuais podem ser acessados ​​por meio de índices. Arrays Python são indexados a zero. Aqui está um exemplo :

my_array = array('i', [1,2,3,4,5])
print(my_array[1])
# 2
print(my_array[2])
# 3
print(my_array[0])
# 1

Introdução básica aos arrays #

Uma matriz é uma estrutura de dados que armazena valores do mesmo tipo de dados. Em Python, esta é a principal diferença entre arrays e listas.

Enquanto as listas python podem conter valores correspondentes a diferentes tipos de dados, as matrizes em python podem conter apenas valores correspondentes ao mesmo tipo de dados. Neste tutorial, vamos entender os arrays Python com alguns exemplos.

Se você é novo no Python, comece com o artigo Introdução ao Python.

Para usar arrays em linguagem python, você precisa importar o módulo array padrão. Isso ocorre porque array não é um tipo de dados fundamental como strings, integer etc. Aqui está como você pode importar o módulo array em python:

from array import *

Depois de importar o módulo array, você pode declarar um array. Aqui está como você faz isso:

arrayIdentifierName = array(typecode, [Initializers])

Na declaração acima, arrayIdentifierName é o nome do array, typecode permite que o python saiba o tipo de array e Initializers são os valores com os quais o array é inicializado.

Os códigos de tipo são os códigos usados ​​para definir o tipo de valores de matriz ou o tipo de matriz. A tabela na seção de parâmetros mostra os valores possíveis que você pode usar ao declarar um array e seu tipo.

Aqui está um exemplo do mundo real de declaração de matriz python:

my_array = array('i',[1,2,3,4])

No exemplo acima, o typecode usado é i. Este typecode representa um inteiro assinado cujo tamanho é de 2 bytes.

Aqui está um exemplo simples de uma matriz contendo 5 inteiros

from array import *
my_array = array('i', [1,2,3,4,5])
for i in my_array:
    print(i)
# 1
# 2
# 3
# 4
# 5

Anexe qualquer valor ao array usando o método append() #

my_array = array('i', [1,2,3,4,5])
my_array.append(6)
# array('i', [1, 2, 3, 4, 5, 6])

Observe que o valor 6 foi anexado aos valores de matriz existentes.

Inserir valor em um array usando o método insert() #

Podemos usar o método insert() para inserir um valor em qualquer índice do array. Aqui está um exemplo :

my_array = array('i', [1,2,3,4,5])
my_array.insert(0,0)
#array('i', [0, 1, 2, 3, 4, 5])

No exemplo acima, o valor 0 foi inserido no índice 0. Observe que o primeiro argumento é o índice enquanto o segundo argumento é o valor.

Estenda a matriz python usando o método extend() #

Um array python pode ser estendido com mais de um valor usando o método extend(). Aqui está um exemplo :

my_array = array('i', [1,2,3,4,5])
my_extnd_array = array('i', [7,8,9,10])
my_array.extend(my_extnd_array)
# array('i', [1, 2, 3, 4, 5, 7, 8, 9, 10])

Vemos que o array my_array foi estendido com valores de my_extnd_array.

Adicione itens da lista no array usando o método fromlist() #

Aqui está um exemplo:

my_array = array('i', [1,2,3,4,5])
c=[11,12,13]
my_array.fromlist(c)
# array('i', [1, 2, 3, 4, 5, 11, 12, 13])

Então vemos que os valores 11,12 e 13 foram adicionados da lista c para my_array.

Remova qualquer elemento do array usando o método remove() #

Aqui está um exemplo :

my_array = array('i', [1,2,3,4,5])
my_array.remove(4)
# array('i', [1, 2, 3, 5])

Vemos que o elemento 4 foi removido do array.

Remova o último elemento do array usando o método pop() #

pop remove o último elemento do array. Aqui está um exemplo :

my_array = array('i', [1,2,3,4,5])
my_array.pop()
# array('i', [1, 2, 3, 4])

Então vemos que o último elemento (5) foi retirado do array.

Busca qualquer elemento através de seu índice usando o método index() #

index() retorna o primeiro índice do valor correspondente. Lembre-se de que os arrays são indexados a zero.

my_array = array('i', [1,2,3,4,5])
print(my_array.index(5))
# 5
my_array = array('i', [1,2,3,3,5])
print(my_array.index(3))
# 3

Observe nesse segundo exemplo que apenas um índice foi retornado, mesmo que o valor exista duas vezes na matriz

Inverta um array python usando o método reverse() #

O método reverse() faz o que o nome diz que fará - inverte o array. Aqui está um exemplo :

my_array = array('i', [1,2,3,4,5])
my_array.reverse()
# array('i', [5, 4, 3, 2, 1])

Obtenha informações de buffer de array através do método buffer_info() #

Este método fornece o endereço inicial do buffer do array na memória e o número de elementos no array. Aqui está um exemplo:

my_array = array('i', [1,2,3,4,5])
my_array.buffer_info()
(33881712, 5)

Verifica o número de ocorrências de um elemento usando o método count() #

count() retornará o número de vezes que o elemento aparece em um array. No exemplo a seguir, vemos que o valor 3 ocorre duas vezes.

my_array = array('i', [1,2,3,3,5])
my_array.count(3)
# 2

Converta array em string usando o método tostring() #

tostring() converte o array em uma string.

my_char_array = array('c', ['g','e','e','k'])
# array('c', 'geek')
print(my_char_array.tostring())
# geek

Converta array em uma lista python com os mesmos elementos usando o método tolist() #

Quando você precisa de um objeto list do Python, você pode utilizar o método tolist() para converter seu array em uma lista.

my_array = array('i', [1,2,3,4,5])
c = my_array.tolist()
# [1, 2, 3, 4, 5]

Anexar uma string ao array char usando o método fromstring() #

Você pode anexar uma string a um array de caracteres usando fromstring()

my_char_array = array('c', ['g','e','e','k'])
my_char_array.fromstring("stuff")
print(my_char_array)
#array('c', 'geekstuff')
