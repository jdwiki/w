Instalação com Nix #

Aviso: esta não é a forma padrão de instalação do Coq.

Para usuários de Linux (e MacOS) que desejam ter acesso a versões atualizadas do Coq ou poder usar várias versões do Coq na mesma máquina, sem o incômodo de usar opam e sem ter que compilar a partir de fonte, esta é uma solução alternativa.

Nix é um gerenciador de pacotes para sistemas operacionais do tipo Unix, como Linux e MacOS. Ele vem com sua própria coleção de pacotes que geralmente é mantida muito mais atualizada do que a do Debian ou do Ubuntu. Não entra em conflito com o gerenciador de pacotes da sua distribuição porque não instala nada em /usr/bin e tal.

Primeiro, você precisa instalar o Nix:

$ curl https://nixos.org/nix/install | sh

Para garantir que as variáveis ​​de ambiente necessárias sejam definidas, faça login novamente ou digite:

. $HOME/.nix-profile/etc/profile.d/nix.sh

Em seguida, o seguinte comando instalará a versão mais recente do Coq:

$ nix-env -iA nixpkgs.coq_8_6

Você também pode executar o CoqIDE sem adicionar nada ao seu PATH:

$ nix-shell -p coq_8_6 --run coqide

Da mesma forma (supondo que você já tenha o Emacs e o Proof-General instalados):

$ nix-shell -p coq_8_6 --run emacs

Isso é muito útil para executar versões diferentes quando você precisar delas. Por exemplo, para executar o Coq 8.5, use o seguinte comando:

$ nix-shell -p coq_8_5 --run coqide

Instale o Coq no MacOS #

Você pode instalar o pacote completo baixando o pacote dmg aqui.

O pacote contém um CoqIDE que pode ser usado para escrever suas provas ou você pode usar o comando coqtop para executar o interpretador em seu terminal

A instalação do Coq no MacOS também é fácil usando o homebrew

brew install coq

ou se você usa MacPorts

sudo port install coq

Não há um bom suporte vi para Coq. Você pode usar o Proof General dentro do emacs que tem uma boa usabilidade.

Para instalar o Proof General, remova as versões antigas do Proof General, clone a nova versão do GitHub

git clone https://github.com/ProofGeneral/PG ~/.emacs.d/lisp/PG
cd ~/.emacs.d/lisp/PG
make

Em seguida, adicione o seguinte ao seu .emacs:

;; Open .v files with Proof General's Coq mode
(load "~/.emacs.d/lisp/PG/generic/proof-site")

Certifique-se de que o emacs que você está executando é o Emacs real. Se você enfrentar problemas de incompatibilidade de versão, talvez seja necessário executar o makefile novamente especificando o caminho do Emacs explicitamente

make clean; make EMACS=/Applications/Emacs.app/Contents/MacOS/Emacs

Para novos usuários que desejam começar a testar o Coq sem instalá-lo em sua máquina, existe um IDE online chamado JsCoq (apresentação [aqui](https://x80.org /rino-coq/)). A subjanela de pacotes permite testar vários pacotes adicionais conhecidos.

Exemplo de uma prova por indução #

Aqui está uma demonstração simples por indução.

Require Import Coq.Setoids.Setoid.
Require Import Coq.Arith.Lt.

(* A number is less than or equal to itself *)
Theorem aLTEa : forall a,
    a <= a.
    auto with arith. (* This follows by simple arithmetic *)
    Qed.

Theorem simplALTE : forall a b,
    S a <= S b <-> a <= b. (* If a <= b, then a + 1 <= b + 1 *)
Proof.
Admitted.

Theorem ltAlwaysLt: forall a b,
    a <= a + b.
Proof.
  intros. (* Introduce relevant variables *)
  induction a, b. (* Induction on every variable *)
  simpl. apply aLTEa. (* 0 <= 0 + S b *)
  rewrite -> plus_O_n. auto with arith. (* 0 <= S b *)
  rewrite <- plus_n_O. apply aLTEa. (* S a <= S a + 0 *)
  rewrite <- simplALTE in IHa. (* IHa: a <= a + S b. Goal: S a <= S a + S b. *)
  apply IHa. (* We rewrote the induction hypothesis to be in the same form as the goal, so it applies immediately now *)
  Qed.
