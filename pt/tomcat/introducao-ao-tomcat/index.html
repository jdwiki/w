Instalação ou Configuração #

Instruções detalhadas sobre como configurar ou instalar o tomcat.

Instalando o Tomcat como um serviço no Ubuntu #

Este exemplo demonstra como instalar o Tomcat como um serviço no Ubuntu usando as versões *.tar.gz tanto do Tomcat quanto do Java.

1. Instale o Java Runtime Environment (JRE) #

  1. Faça o download da versão .tar.gz do jre desejada
  2. Extrair para /opt/
    Isso criará um diretório /opt/jre1.Xxxx/
  3. Crie um link simbólico para o diretório inicial do java:
    cd /opt; sudo ln -s jre1.Xxxxx java
  4. adicione o JRE à variável de ambiente JAVA_HOME:
    sudo vim /etc/environment
    JAVA_HOME="/opt/java"

2. Instale o Tomcat: #

  1. Faça o download do tomcat em uma versão .tar.gz (ou similar).
  2. Crie um usuário do sistema tomcat:
    sudo useradd -r tomcat
  3. Extrair para /opt/
    Isso criará um diretório /opt/apache-tomcat-XXXX
    atribua este diretório ao usuário e grupo do sistema tomcat:
    sudo chown -R tomcat ./*
    sudo chgrp -R tomcat ./*
  4. Crie a variável de ambiente CATALINA_HOME:
    sudo vim /etc/environment
    CATALINA_HOME="/opt/tomcat"
  5. Adicionar usuário administrador em tomcat-users.xml
    sudo vim /opt/tomcat/conf/tomcat-users.xml
    e adicione algo como <user username="admin" password="adminpw" roles="manager-gui">
    entre as tags <tomcat-users> ... </tomcat-users>

3. Fazendo a inicialização do Tomcat na inicialização #

Adicione um script em /etc/init.d chamado tomcat e torne-o executável. O conteúdo do script pode ser algo como:

RETVAL=$?
CATALINA_HOME="/opt/tomcat"

case "$1" in
 start)
    if [ -f $CATALINA_HOME/bin/startup.sh ];
      then
        echo $"Starting Tomcat"
        sudo -u tomcat $CATALINA_HOME/bin/startup.sh
    fi
    ;;
 stop)
    if [ -f $CATALINA_HOME/bin/shutdown.sh ];
      then
        echo $"Stopping Tomcat"
        sudo -u tomcat $CATALINA_HOME/bin/shutdown.sh
    fi
    ;;
 *)
    echo $"Usage: $0 {start|stop}"
    exit 1
    ;;
esac

exit $RETVAL

Para fazê-lo iniciar na inicialização, execute: sudo update-rc.d tomcat defaults

Você também pode adicionar uma linha bash ao /etc/rc.local por exemplo service tomcat start

Alterando classpath ou outras variáveis ​​de ambiente relacionadas ao Tomcat: #

Edite o arquivo $CATALINA_HOME/bin/setenv.sh e adicione as propriedades aqui, por exemplo: CLASSPATH=/additional/class/directories
