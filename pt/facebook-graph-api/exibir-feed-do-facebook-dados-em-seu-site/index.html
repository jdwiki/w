Obtendo o Token de Acesso do FaceBook para ler e escrever no gráfico social do Facebook #

Este documento detalha as etapas para obter os tokens de acesso do facebook e como usar os tokens para buscar feeds do FB.

Exemplo: Um exemplo ao vivo está disponível em

https://newtonjoshua.com

Introdução à Graph API: A Graph API é a principal maneira de obter dados dentro e fora da plataforma do Facebook. É uma API baseada em HTTP de baixo nível que você pode usar para consultar dados, postar novas histórias, gerenciar anúncios, fazer upload de fotos e uma variedade de outras tarefas que um aplicativo pode precisar fazer.

Aplicativos do Facebook:

https://developers.facebook.com

Crie um aplicativo do Facebook. Você receberá um App_Id e um App_Secret

Explorador de API de gráfico:

https://developers.facebook.com/tools/explorer/{{App_Id}}/?method=GET&path=me%2Ffeed&version=v2.8

Você receberá um access_token que dura pouco. Então este será nosso short_lived_access_token.

nota: ao criar o token de acesso, selecione todos os campos fb necessários. Isso dará permissão ao token de acesso para buscar esses campos.

Extensão do token de acesso:

https://graph.facebook.com/oauth/access_token?grant_type=fb_exchange_token&client_id={{App_Id}}&client_secret={{App_Secret}}&fb_exchange_token={{short-lived-access_token }}

Você receberá um access_token com validade de 2 meses.

Depurador de token de acesso:

https://developers.facebook.com/tools/debug/accesstoken?q={{access_token}}&version=v2.8

você pode verificar os detalhes do access_token.

Facebook SDK para JavaScript: Inclua o JavaScript abaixo em seu HTML para carregar de forma assíncrona o SDK em sua página

<script>
        (function (d, s, id) {
            var js, fjs = d.getElementsByTagName(s)[0];
            if (d.getElementById(id)) {
                return;
            }
            js = d.createElement(s);
            js.id = id;
            js.src = "//connect.facebook.net/en_US/sdk.js";
            fjs.parentNode.insertBefore(js, fjs);
        }(document, 'script', 'facebook-jssdk'));
    </script>

API de gráfico: Vamos fazer uma chamada de API para obter nosso ID do FB, foto de perfil, foto de capa e feeds.

<script>
window.fbAsyncInit = function () {
    FB.init({
        appId: '{{App_Id }}',
        xfbml: true,
        version: 'v2.7'
    });
    FB.api(
        '/me',
        'GET', {
            fields: 'id,picture{url},cover,feed',
            access_token: {{access_token}}
        },
        function (response) {
        if (response.error) {
                console.error(response.error.message);
            }
            if (response.picture.data.url) {
                profilePic = response.picture.data.url;
            }
            if (response.cover.source) {
                coverPic = response.cover.source;
            }
            if (response.feed.data) {
        feeds = response.feed.data;
                feeds.forEach(function (feed) {
            // view each feed content
                });
            }
            if (response.feed.paging.next) {
                nextFeedPage = response.feed.paging.next;
        // a request to nextFeedPage will give the next set of feeds
            }
            
        }
    );
};

</script>

Use o Graph API Explorer para criar sua consulta que deve ser inserida nos ‘campos’ (por exemplo: ‘id,picture{url},cover,feed’)

Agora você pode buscar seus dados do Facebook da API de gráfico do Facebook usando seu access_token. consulte https://developers.facebook.com/docs/graph-api/overview/

nota: Seu access_token expirará em 2 meses. Crie um novo access_token depois disso.
