Como instalar o Json.Net em projetos do Visual Studio #

Você pode instalar o Json.Net em seu projeto do Visual Studio de 1 de 2 maneiras.

Instale o Json.Net usando o Console do Gerenciador de Pacotes. #

  1. Abra a janela Package Manager Console no Visual Studio digitando package manager console na caixa Quick Launch e selecionando-a

ou clicando em Exibir -> Outros Windows -> Console do Gerenciador de Pacotes.

  1. Assim que o Package Manager Console estiver visível, selecione o projeto em sua solução, no qual você deseja instalar o Json.Net, selecionando-o no menu suspenso Default Project.

  1. Agora digite o seguinte comando e pressione enter.

Instalar-Pacote Newtonsoft.Json

  1. Isso instalará a versão mais recente do Json.Net. Você verá o progresso da instalação no Console do Gerenciador de Pacotes. Se for bem sucedido, você verá a mensagem de que foi instalado com sucesso no seu projeto selecionado.

  1. Uma vez instalado com sucesso, você poderá ver o Json.Net Assembly nas referências do seu projeto selecionado. O nome do assembly Json.Net é Newtonsoft.Json

Json.Net agora está pronto para uso em seu projeto!

Instale o Json.Net usando o Visual Studio Solution Explorer. #

Você também pode instalar o json.net usando o Solution Explorer no Visual Studio.

  1. Clique com o botão direito do mouse no nó References em seu projeto e clique em Manage Nuget Packages…

  2. Na caixa de diálogo Nuget Package Manager, certifique-se de que Online esteja selecionado no painel esquerdo. Digite Json.Net na caixa de pesquisa no canto superior direito. Isso exibirá o pacote Json.Net Nuget no painel de resultados da pesquisa no meio.

  3. Clique no botão Instalar.

  1. Você verá o progresso da instalação na janela de progresso que aparecerá.

  1. Quando a instalação estiver concluída, você verá uma marca de seleção verde ao lado do pacote Json.Net na caixa de diálogo Nuget Package Manager.

  1. Você também verá o assembly Newtonsoft.Json no nó References em seu gerenciador de soluções para o projeto selecionado.

Isso conclui a instalação do Json.Net. Agora você está pronto para usá-lo em seu projeto para realizar várias operações em dados json.

Como desserializar dados JSON para objeto usando Json.Net em C #

O exemplo a seguir mostra como você pode desserializar uma string JSON que contém em um objeto (ou seja, em uma instância de uma classe).

using System;
using System.Collections.Generic;
using Newtonsoft.Json;
                    
public class Program
{
    public class Employee
    {
        public string FirstName { get; set; }
        public string LastName { get; set; }
        public bool IsManager { get; set; }
        public DateTime JoinedDate { get; set; }
        public IList<string> Titles { get; set; }
    }
    
    public static void Main()
    {
        string json = @"{
                          'FirstName': 'Shiva',
                          'LastName': 'Kumar',
                          'IsManager': true,
                          'JoinedDate': '2014-02-10T00:00:00Z',
                          'Titles': [
                            'Sr. Software Engineer',
                            'Applications Architect'
                          ]
                        }";
        
        Employee employee = JsonConvert.DeserializeObject<Employee>(json);

        Console.WriteLine(employee.FirstName);
        Console.WriteLine(employee.LastName);
        Console.WriteLine(employee.JoinedDate);
        foreach (string title in employee.Titles)
        {
            Console.WriteLine("  {0}", title);    
        }
    }
}

Se você executar este programa de console, a saída das várias instruções Console.WriteLine será a seguinte.

Shiva
Kumar
2/10/2014 12:00:00 AM
  Sr. Software Engineer
  Applications Architect

Algumas coisas a serem observadas

  1. A linha a seguir executa a desserialização real dos dados na string json na instância do objeto employee da classe Employee.

Funcionário funcionário = JsonConvert.DeserializeObject<Employee>(json);

  1. Como employee.Titles é um tipo List<string>, usamos a construção de loop foreach para percorrer cada item dessa List.

Como serializar um objeto para JSON usando Json.Net em C# #

O exemplo a seguir mostra como você pode usar Json.Net para serializar os dados em uma instância de um objeto C# para uma string JSON.

using System;
using System.Collections.Generic;
using Newtonsoft.Json;

public class Employee
{
    public string FirstName { get; set; }
    public string LastName { get; set; }
    public bool IsManager { get; set; }
    public DateTime JoinedDate { get; set; }
    public IList<string> Titles { get; set; }
}
                    
public class Program
{
    public static void Main()
    {
        Employee employee = new Employee
        {
            FirstName = "Shiva",
            LastName = "Kumar",            
            IsManager = true,
            JoinedDate = new DateTime(2013, 1, 20, 0, 0, 0, DateTimeKind.Utc),
            Titles = new List<string>
            {
                "Sr. Software Engineer",
                "Applications Architect"
            }
        };

        string json = JsonConvert.SerializeObject(employee, Formatting.Indented);


Console.WriteLine(json);
        
    }
}

Se você executar este programa de console, a saída do Console.WriteLine(json) será

{
  "FirstName": "Shiva",
  "LastName": "Kumar",
  "IsManager": true,
  "JoinedDate": "2013-01-20T00:00:00Z",
  "Titles": [
    "Sr. Software Engineer",
    "Applications Architect"
  ]
}

Algumas coisas a serem observadas

  1. A linha a seguir executa a serialização real dos dados dentro da instância da classe employee em uma string json

    string json = JsonConvert.SerializeObject(employee, Formatting.Indented);

  2. O parâmetro Formatting.Indented diz ao Json.Net para serializar os dados com recuo e novas linhas. Se você não fizer isso, a string serializada será uma string longa sem recuo ou quebras de linha.
