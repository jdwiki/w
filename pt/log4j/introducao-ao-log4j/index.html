Instalação e configuração #

Instalação #

A instalação do Log4j2 é tão simples quanto colocar o jar do log4j2 no classpath do aplicativo. Embora você possa querer personalizar a saída de logs por meio de arquivo de configuração adicional

Configuração #

especialista #

Para adicionar log4j ao projeto no maven, adicione sua dependência: Em pom.xml adicione a seguinte dependência:

<dependencies>
  <dependency>
    <groupId>org.apache.logging.log4j</groupId>
    <artifactId>log4j-api</artifactId>
    <version>${log4j2.version}</version>
  </dependency>
  <dependency>
    <groupId>org.apache.logging.log4j</groupId>
    <artifactId>log4j-core</artifactId>
    <version>${log4j2.version}</version>
  </dependency>
</dependencies>

springboot com maven #

Spring-boot é um framework comumente usado para aplicações web. Possui suporte à configuração automática para muitos recursos, incluindo fachada de log como log4j2. Para adicionar log4j2 ao seu projeto spring-boot, certifique-se de excluir a fachada de log padrão: commons-logging. Log4j será usado, quando estiver apenas logando a fachada no classpath.

<dependencies>
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-data-jpa</artifactId>
        <!-- exclude spring-boot java commons logging in favour of log4j2 -->
        <exclusions>
            <exclusion>
                <groupId>org.springframework.boot</groupId>
                <artifactId>spring-boot-starter-logging</artifactId>
            </exclusion>
        </exclusions>
    </dependency>
    <!-- add log4j2 to spring-boot: -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-log4j2</artifactId>
    </dependency>

Observe que não há versão no snippet acima. É porque o projeto herda a versão do pai. Certifique-se de também herdar de spring-boot-starter-parent, adicionando:

<parent>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-parent</artifactId>
    <version>1.3.3.RELEASE</version>
</parent>

hera #

Em ivy.xml, adicione a seguinte dependência:

<dependencies>
  <dependency org="org.apache.logging.log4j" name="log4j-api" rev="${log4j2.version}" />
  <dependency org="org.apache.logging.log4j" name="log4j-core" rev="${log4j2.version}" />
</dependencies>

grade #

No seu arquivo .gradle:

dependencies {
  compile group: 'org.apache.logging.log4j', name: 'log4j-api', version: '2.6.2'
  compile group: 'org.apache.logging.log4j', name: 'log4j-core', version: '2.6.2'
}
