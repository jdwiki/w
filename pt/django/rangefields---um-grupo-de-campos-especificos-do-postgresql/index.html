Sintaxe #

  • de django.contrib.postgres.fields import *RangeField

  • FloatRangeField(**opções)
  • DateTimeRangeField(**opções)
  • DateRangeField(**opções)

Incluindo campos de intervalo numérico em seu modelo #

Existem três tipos de RangeFields numéricos em Python. IntegerField, BigIntegerField e FloatField. Eles convertem para psycopg2 NumericRanges, mas aceitam a entrada como tuplas nativas do Python. O limite inferior é incluído e o limite superior é excluído.

class Book(models.Model):
    name = CharField(max_length=200)
    ratings_range = IntegerRange()

Configurando para RangeField #

  1. adicione 'django.contrib.postgres' ao seu INSTALLED_APPS
  2. instale o psycopg2

Criando modelos com campos de intervalo numérico #

É mais simples e fácil inserir valores como uma tupla Python em vez de um NumericRange.

Book.objects.create(name='Pro Git', ratings_range=(5, 5))

Método alternativo com NumericRange:

Book.objects.create(name='Pro Git', ratings_range=NumericRange(5, 5))

Usando contém #

Esta consulta seleciona todos os livros com qualquer classificação inferior a três.

bad_books = Books.objects.filter(ratings_range__contains=(1, 3))

Usando contido_by #

Esta consulta obtém todos os livros com classificações maiores ou iguais a zero e menores que seis.

all_books = Book.objects.filter(ratings_range_contained_by=(0, 6))

Usando sobreposição #

Esta consulta obtém todos os compromissos sobrepostos de seis a dez.

Appointment.objects.filter(time_span__overlap=(6, 10))
