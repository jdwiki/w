Operações com contadores #

Este exemplo mostra como usar operações matemáticas com contadores. Pode ser útil para loops em látex.

Adição: \addtocounter{num}{n}

este comando adiciona n a num, onde num é um contador e n é um inteiro positivo.

Subtração: \addtocounter{num}{-n}

este comando subtrai n de num, onde num é um contador e n é um inteiro positivo.

Multiplicação: \multiplicar\valor{num} por n

este comando multiplica num por n, onde num é um contador e n é um inteiro.

Divisão \divide\value{num} por n

este comando divide num por n e obtém a parte inteira do quociente (num é um contador e n é um inteiro)

\documentclass{article}
    \begin{document}
    \newcounter{num}
    \setcounter{num}{3}
    \addtocounter{num}{10}
    \thenum\\%prints 13
    \addtocounter{num}{-3}
    \thenum\\%prints 10
    \stepcounter{num}
    \thenum\\%prints 11
    \multiply\value{num} by \value{num}
    \thenum\\%prints 121
    \multiply\value{num} by 2
    \thenum\\%prints 242
    \divide\value{num} by 60
    \thenum%prints 4
\end{document}

\newcommand{num} declara contador. \setcounter{num}{3} define o valor de num como 3.

\addtocounter{num}{10} adiciona 10 a num.

\addtocounter{num}{-3} subtrai 3 de num.

\stepcounter{num} adiciona 1 a num

\multiply\value{num} por \value{num} quadrados num.

\multiply\value{num} por 2 doubles num.

\divide\value{num} por 60 divide num por 60 e obtém a parte inteira.

O resultado do código: 13\\10\\11\\121\\242\\4

(\\ simboliza nova linha)

O pacote intcalc adiciona algumas outras operações inteiras, por exemplo mod, pow, sng, abs, inv…

Declaração do contador, inicialização e impressão em pdf #

É possível usar variáveis ​​inteiras com látex. Para criar uma nova variável precisamos do comando \newcounter{name}, onde name é o nome do novo contador. O nome deve conter apenas letras. Este comando cria um novo com o nome \thename. Com este comando podemos imprimir a variável name no papel. O valor inicial de name é 0. Para dar valor a “name” podemos usar \setcounter{name}{n} onde n é um inteiro. \value{name} é uma função que retorna com o valor de name.

\documentclass{article}
\begin{document}
\newcounter{num}                    %new counter, initial value is 0
\thenum                             %print 0
\setcounter{num}{3}                 %set num to 3
\thenum                             %print 3
\newcounter{number}
\setcounter{number}{\value{num}}    %set number to value of num
\thenumber                          %print 3

Latex provides some other formats to print a number.

Other types of printing:

\arabic{num}\\
\Roman{num}\\ %→ I, II, III, IV, . . . (num = 1, 2, 3, . . . )
\roman{num}\\ %→ i, ii, iii, iv, . . . (num = 1, 2, 3, . . . )
\Alph{num}\\  %→ A, B, C, D, . . . (num = 1, 2, 3, . . . , 26)
\alph{num}\\  %→ a, b, c, d, . . . (num = 1, 2, 3, . . . , 26)
\fnsymbol{num}\\ %→ ∗, †, ‡, §, ¶, k, ∗∗, ††, ‡‡ (num = 1, 2, 3, . . . , 9)
\end{document}

Declarações if #

No latex, podemos usar comandos embutidos para executar o código, sejam as condições verdadeiras ou não.

Comparando dois números inteiros: \ifnum\value{num}>n {A} \else {B}\fi

Este código executa A se num>n else B. Podemos substituir > por < e =.

Se um número for ímpar: \ifodd\value{num} {A}\else {B}\fi

Se num for ímpar, ele executa A else B.

Se com condição: \ifthenelse{condição}{A}{B}

Temos que carregar ifthen package para usar este comando. Se a condição for verdadeira, ele executa A senão B.

É possível criar condições complexas com \( \), \AND, \OR, \NOT.

Por exemplo: \ifthenelse{\(\NOT 4<2 \OR 4>11\)\AND\isodd{4}}{A}{B}

Este pedaço de código escreve “B” na página. \NOT 4<2 é verdadeiro e 4>11 é falso. Se conectarmos uma declaração falsa e uma verdadeira com “OR”, o resultado será verdadeiro. Portanto, \(\NOT 4<2 \OR 4>11\) é verdadeiro. \isodd{4} é falso porque 4 é par. Uma declaração falsa e verdadeira conectada com “AND” é falsa, então a saída é B.

Um código de exemplo:

\documentclass{article}
\usepackage{ifthen}
\begin{document}
    \newcounter{num}
    \setcounter{num}{10}

    If num$>$100 then the next sentence will be "Num is large." else "Num is small."

    Num is \ifnum \value{num}>100 {large} \else {small}.

    If num is odd then the next sentence will begin with "Odd" if not then with "Even"

    \ifodd \value{num} {Odd} \else {Even} numbers are cool.

    If (num$>$3 and (1$<$0 or num$=$10)) is true then the next sentence will be "True." else "False."

    \ifthenelse{\value{num}>3\AND\(1<0 \OR \value{num}=10\)}{True.}{False.}


\end{document}

Loops - repetindo coisas #

Podemos criar loops em látex. Eles são semelhantes, mas não tão personalizáveis ​​quanto os loops em outras linguagens de programação. Uma alternativa para usar loops são @loops. Se usarmos um comando que inclui “@” em seu nome, devemos colocá-lo entre \makeatletter e \makeatother. Não é permitido usá-los em uma macro que descreva uma nova definição.

Errado:

\def\is#1#2{\makeatletter\@ifstar{#1}{#2}\makeatother

Certo:

\makeatletter\def\is#1#2{\@ifstar{#1}{#2}}\makeatother

@for loop: \@for\command:={list}\do{commands}

Exemplo:

\makeatletter
\@for\sun:={rising,setting}\do{The sun is \sun.}
\makeatother

Ele cria o seguinte texto: O sol está nascendo. O sol está se pondo.

@whilenum loop: \@whilenum condition\do{commands}

Exemplo:

\makeatletter
\newcounter{int}
\@whilenum\value{int}<10\do
{\stepcounter{int}\ifthenelse{\isodd{\value{int}}}{\theint}{}}
\makeatother

Este código escreve números ímpares de 1 a 9.

“loop repeat” loop: \loop {commands} \ifnum condition \repeat

Executa comandos até que a condição seja verdadeira.

Exemplo

\setcounter{int}{1}
\loop
\theint
\addtocounter{int}{2}
\ifnum \value{int}<10
\repeat

Este código faz o mesmo que o loop @whilenum.

Um código de exemplo:

\documentclass{article}
\usepackage{ifthen}
\usepackage{amsmath} %\text{} command needs this package
\begin{document}
    Demonstration of @for loop:

    \makeatletter
    \@for\sun:={rising,setting}\do{The sun is \sun. }
    \makeatother

    \newcounter{int}

    @whilenum loop:
    
    \setcounter{int}{0}
    \makeatletter
    \@whilenum\value{int}<20\do
    {\stepcounter{int}\ifthenelse{\isodd{\value{int}}}{\theint\text{ }}{}}
    \makeatother
    
    "loop repeat" loop:
    
    \setcounter{int}{1}
    \loop
    \theint
    \text{ }\addtocounter{int}{2}\ifnum\value{int}<20
    \repeat
\end{document}

Usando loops no Tikz #

Loops são úteis no Tikz.

O código a seguir desenha um relógio sem números:

\documentclass{article}
\usepackage{ifthen}
\usepackage{intcalc}
\usepackage{tikz}
\newcounter{num}

\begin{document}       
\begin{tikzpicture}
    \makeatletter
    \setcounter{num}{1}
    \newcounter{angle}
    \draw (0,0) circle (3cm);
    \@whilenum\value{num}<13\do{
    \setcounter{angle}{360}
    \multiply\value{angle} by \value{num}
    \divide\value{angle} by 12
    \ifnum \intcalcMod{\value{num}}{3}=0{
    \draw[line width=4pt] (\theangle:2cm) -- (\theangle:3cm);    }\else
    {
    \draw[line width=1pt] (\theangle:2.3cm) -- (\theangle:3cm);
    }\fi
    \addtocounter{num}{1}
    }
    \makeatother
\end{tikzpicture}
\end{document}
