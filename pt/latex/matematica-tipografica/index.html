Uma das maiores vantagens do LaTeX é sua habilidade em formatar equações. Aqui, são descritos os fundamentos das equações de composição, alguns dos vários pacotes que podem ser usados, bem como os símbolos comuns.

Sintaxe #

  • \begin{equação} … \end{equação}
  • texto $ … $ texto
  • \usepackage{amsmath} … \begin{equation*} … \end{equation*}

Aqui estão algumas ideias básicas para garantir que seu código não seja interrompido e que suas equações tenham uma aparência melhor:

  1. Certifique-se de que todos os colchetes, chaves, cifrões e comandos \begin{} \end{} sejam correspondentes. Isso é algo em que um pequeno erro pode atrapalhar todo o seu código em grande forma.
  2. Se você receber erros, certifique-se de ter o pacote apropriado carregado (por exemplo, não use o comando \begin{equation*} sem o pacote amsmath).
  3. Nunca, nunca, nunca use cifrões duplos ($$uma equação aqui$$) em vez de \begin{equation}.
  4. Nunca use o modo matemático como forma de deixar seu texto em itálico.
  5. Completamente preso? Experimente o TeX.SX, um site para responder perguntas sobre TeX, LaTeX e idiomas relacionados.

Boa sorte!

Equações Básicas #

Equações simples, em linha

Você pode fazer uma equação inline simples usando $anequation here$.

Por exemplo, você pode fazer

$\lim\limits_{n \to \infty} \frac{1}{2^n} i\bar z$

que, se colocarmos um pequeno texto falso em volta, dá

Equações numeradas e centradas

Ao escrever artigos ou outros documentos, às vezes é preferível ter suas equações centralizadas e numeradas, em vez de em linha. Em seguida, use os comandos \begin{equation} e \end{equation}.

Por exemplo, se usarmos o código

\begin{equation}
\lim\limits_{n \to \infty} \frac{1}{2^n} i\bar z
\end{equation}

E adicione um pequeno texto ao redor dele, temos

Você pode remover a numeração da equação usando \begin{equation*} e \end{equation*}.

Por exemplo, se usarmos o código

\begin{equation*}
\lim\limits_{n \to \infty} \frac{1}{2^n} i\bar z
\end{equation*}

e adicionar um pequeno texto em torno dele, temos

(embora deva ser notado que você tem que usar o pacote amsmath para isso).

Pacotes disponíveis para uso #

Embora o LaTeX padrão seja tudo o que é necessário para a maioria das fórmulas e equações matemáticas simples, às vezes são necessários mais símbolos e ferramentas. Existem vários pacotes disponíveis que aprimorarão suas equações e fornecerão mais para você trabalhar. Três dos principais pacotes são descritos abaixo. Lembre-se, para carregar um pacote, digite \usepackage{package} no preâmbulo do seu documento.

amsmath

O pacote amsmath é um pacote incrivelmente útil. Ele é usado para permitir que suas equações sejam centralizadas, mas não numeradas, como em \begin{equation*}, é usado para criar matrizes (como descrito abaixo) e introduz muitos outros comandos úteis, como \overset e \underset, descritos abaixo. A documentação do pacote amsmath pode ser encontrada aqui.

mathtools

O pacote mathtools se baseia no pacote amsmath, adicionando mais símbolos e ferramentas úteis. Ele carrega automaticamente o pacote amsmath, então você não precisa carregar ambos no preâmbulo do documento. A documentação do mathtools pode ser encontrada aqui.

amssymb

O pacote amssymb fornece muitos símbolos extras que podem ser muito úteis para equações mais complexas. A documentação do amssymb pode ser encontrada aqui.

Bons comandos para saber #

Alguns dos comandos mais comuns incluem:

  • Frações e Raízes Quadradas: Para frações, use \frac {numerador}{denominador}. Para raízes quadradas, use \sqrt[root]{number}.
  • Letras gregas: use os comandos fornecidos na tabela abaixo:

  • Operadores: \leq dá o símbolo menor ou igual, \geq dá o símbolo maior ou igual, \neq dá o símbolo diferente, \sum dá a soma símbolo, \partial dá o símbolo da derivada parcial, \nabla dá o operador Laplaciano, \times dá o produto cruzado ou símbolo de multiplicação, \cdot dá o produto escalar ou símbolo de multiplicação, e \int dá o símbolo integral.
  • Setas: \rightarrow e \leftarrow fornecem setas para a direita e para a esquerda, respectivamente.
  • Porcentagens: Ao digitar % em LaTeX, é importante incluir uma barra invertida, \%, pois o símbolo de porcentagem é normalmente usado para comentários.
  • Sobrescritos e subscritos: para fazer um sobrescrito, você pode digitar x^2 ou, para sobrescritos mais longos, x^{2x}. Para fazer um subscrito, você pode digitar x_a, ou, para subscritos mais longos, x_{ab}.
  • Negrito: Use \boldmath{...} para deixar seus símbolos matemáticos em negrito. Outras opções são fornecidas em esta pergunta do TeX.SX. Os símbolos matemáticos ficam automaticamente em itálico; se você não quiser que isso seja verdade, faça o texto da equação conforme descrito abaixo.
  • Infinity: Para escrever infinito, use o comando \infty.
  • Movendo itens para cima ou para baixo de outro: Primeiro, apenas para operadores matemáticos, há um método alternativo. Você pode digitar o operador matemático, digamos \int, e então usar o comando \limits. Um exemplo é \int\limits_{\infty} ou \int\limits^{\infty}. Então, para casos normais, você pode fazer \overset{top}{normal} ou \underset{bottom}{normal}. Isso pode ser muito útil para fazer vetores. Por exemplo, você pode fazer \overset{\rightarrow}{x} O pacote amsmath é necessário para overset e underset.
  • Chaves: Como as chaves são usadas em comandos, é necessário digitar \{ ou \} para obter as chaves.
  • Texto: Para incluir texto nas equações, digite \usepackage{amsmath} no preâmbulo e digite \text{...}.
  • Espaço: para adicionar espaço em suas equações, digite \quad entre os dois itens que deseja separar (por exemplo, você pode ter $2x \quad cos).

Criando novos símbolos #

Digamos que você não consiga encontrar o símbolo de que precisa em nenhum lugar. Você pode criar um símbolo personalizado. Por exemplo, o código

\documentclass{article}
\usepackage{graphicx,amsmath,amssymb}
\DeclareRobustCommand{\diamondtimes}{%
  \mathbin{\text{\rotatebox[origin=c]{45}{$\boxplus$}}}%
}

\begin{document}
$a\diamondtimes b$
\end{document}

cria e chama um símbolo, dando

Este é um exemplo mais simples; ele simplesmente tem que girar um símbolo já existente. No entanto, você pode criar símbolos mais complexos.

Matrizes #

Matrizes

Você deve sempre usar o pacote amsmath se for usar os comandos a seguir. Existem quatro tipos principais de matriz, conforme mostrado no código abaixo:

\begin{matrix} 
    a & b \\
    c & d 
\end{matrix}
\quad
\begin{pmatrix} 
   a & b \\
   c & d 
\end{pmatrix}
\quad
\begin{bmatrix} 
    a & b \\
    c & d 
\end{bmatrix}
\quad
\begin{vmatrix} 
    a & b \\
    c & d 
\end{vmatrix}
\quad
\begin{Vmatrix} 
    a & b \\
    c & d 
\end{Vmatrix}

Este código produz

Há algumas coisas importantes a serem observadas sobre isso:

  1. É importante que você coloque sua matriz dentro do ambiente equation, equation* ou $...$ - o comando bmatrix não é um ambiente matemático por si só.
  2. A construção da matriz é bastante simples. Para cada linha, você cria cada elemento (digamos x_{11}), coloca um & e escreve o próximo elemento. Para várias linhas, no final de cada linha coloque \\ (você não precisa fazer isso para a última linha). É bastante semelhante a uma tabela neste.
