Criando cartazes usando o beamer

Criar um pôster usando o pacote beamerposter é muito semelhante a criar um único quadro. Coloque o conteúdo em colunas. Dentro de cada coluna, separe o conteúdo usando blocos.

Orientação e tamanho #

Ao adicionar o pacote beamerposter, forneça os parâmetros necessários.

\usepackage[orientation=landscape,size=a1]{beamerposter}

Você também pode personalizar o tamanho do pôster.

\usepackage[orientation=portrait,size=custom,height=110,width=80,scale=1.4]{beamerposter}

As dimensões de altura e largura aqui, estão em cms. A scale é usada para o tamanho da fonte.
