Editores LaTeX #

Embora você possa criar documentos LaTeX usando qualquer editor e compilando usando o console, existem vários plugins para editores amplamente usados ​​para simplificar a criação de documentos LaTeX, e existem editores LaTeX especializados. Uma lista exaustiva de editores LaTeX está disponível em TeX.SE (o site StackExchange, dedicado ao TeX, LaTeX & Friends).

Os editores mais utilizados, de acordo com esta lista, são:

  • O editor Emacs com a extensão AUCTeX.
  • O editor Vim com o plugin LaTeX-suite.
  • Texmaker – um IDE especializado em LaTeX.
  • TeXstudio – outro LaTeX IDE.
  • TeXworks – mais um LaTeX IDE.

Enquanto usuários experientes de Emacs ou Vim podem querer ficar com seu editor (cujos plugins fornecem uma série de funcionalidades indisponíveis em outros lugares), um IDE especializado pode ser mais fácil de instalar/usar para iniciantes. Os três últimos da lista têm uma função de pré-visualização onde se pode ver os resultados da compilação do documento.

Além disso, existem ferramentas de LaTeX online que podem ser úteis para iniciantes ou pessoas que precisam colaborar, por exemplo. ShareLaTeX e Overleaf.

Instalação e configuração #

Você pode escolher entre as principais distribuições do LaTeX:

  • TeX Live (Windows, Linux e OS X), a distribuição padrão multiplataforma.
  • MacTeX (Mac) Uma versão empacotada do TeX Live feita para OS X com algumas ferramentas específicas para Mac
  • MiKTeX (Windows) Uma distribuição separada inteiramente que

Todas as distribuições são mais ou menos equivalentes em um mundo ideal. O TeX Live tem a vantagem de estar disponível em todas as plataformas e, portanto, tem um suporte muito melhor da comunidade. O MiKTeX pode aproveitar os recursos específicos do Windows. Por motivos de licenciamento, o MiKTeX também distribuirá alguns pacotes que o TeX Live não distribuirá.

Em todos os casos, a instalação completa é recomendada. Especificamente, usar o recurso de download sob comando do MiKTeX irá travar/travar muitos editores.

Instalação #

Windows (TeXLive) #

  1. Baixe o mais recente do TeXLive install-tl-windows.exe de seu website.
  2. Execute install-tl-windows.exe e siga as instruções.

Windows (MiKTeX) #

  1. Baixe o instalador mais recente do MiKTeX de seu website.
  2. Execute o instalador e siga as instruções.

Mac OS X (TeXLive) #

  1. Baixe o MacTeX mais recente de seu site.
  2. Execute MacTeX.pkg e siga as instruções.

Linux (TeXLive) #

Os usuários do Linux têm duas opções:

  1. Instale através do gerenciador de pacotes da sua distribuição (geralmente vários lançamentos atrás)
  2. Instale a partir do upstream (lançado anualmente, atualizado com frequência)

Usando gerenciadores de pacotes #

  • Arch Linux: pacman -S texlive-most
  • Debian/Ubuntu/Mint: apt-get install texlive-full
  • Fedora: yum install texlive

Observe que usar este método significa que você será dependente do mantenedor desse pacote para a distribuição de atualizações. Esses pacotes geralmente serão vários lançamentos atrás da distribuição mais recente, geralmente significando que atualizações críticas estarão ausentes. É quase sempre melhor instalar a partir do upstream. Observe também que o gerenciador de pacotes da distribuição provavelmente não reconhecerá a instalação direta e poderá tentar instalá-la quando instalar outros pacotes de suporte relacionados.

Instalando a partir do Upstream #

  1. Baixe o TeXLive install-tl-unx.tar.gz mais recente do [site] 1.

  2. Extraia os arquivos do arquivo com tar -zxvf install-tl-unx.tar.gz.

  3. Mude para a pasta baixada com cd install-tl-unx.

  4. Execute ./install-tl e siga as instruções.

    TeXLive should now be installed under /usr/local/texlive/YEAR/, where YEAR is the four digit year (e.g. 2016). In this way, it is possible to have multiple TeXLive versions alongside each other and switch between them by changing your PATH variable.

    Open this folder and check the bin folder. It should contain a subfolder, which (depending on your platform) will be something like i386-linux or x86_64-linux.

  5. Adicione a pasta binária TeX Live ao seu caminho com

     EXPORT PATH=/usr/local/texlive/YEAR/bin/PLATFORM:$PATH

    where YEAR is the four digit year (e.g. 2016), and PLATFORM is your platform (e.g. x86_64-linux).

Instalação de teste #

A instalação do LaTeX está concluída. Para testá-lo, crie um novo arquivo com seu editor de texto favorito, nomeie-o como test.tex e adicione o seguinte conteúdo:

\documentclass{article}
\begin{document}
  Hello World!
\end{document}

Agora, abra o console ou terminal, navegue até a pasta onde você salvou test.tex e execute

pdflatex test

(Observe que seu editor pode ter recursos para executar isso para você.)
