Pontos fixos #

Fix pega um tipo de “modelo” e amarra o nó recursivo, sobrepondo o modelo como uma lasanha.

newtype Fix f = Fix { unFix :: f (Fix f) }

Dentro de um Fix f encontramos uma camada do template f. Para preencher o parâmetro de f, Fix f insere itself. Então quando você olha dentro do template f você encontra uma ocorrência recursiva de Fix f.

Aqui está como um tipo de dados recursivo típico pode ser traduzido em nossa estrutura de modelos e pontos fixos. Removemos as ocorrências recursivas do tipo e marcamos suas posições usando o parâmetro r.

{-# LANGUAGE DeriveFunctor #-}

-- natural numbers
-- data Nat = Zero | Suc Nat
data NatF r = Zero_ | Suc_ r deriving Functor
type Nat = Fix NatF

zero :: Nat
zero = Fix Zero_
suc :: Nat -> Nat
suc n = Fix (Suc_ n)


-- lists: note the additional type parameter a
-- data List a = Nil | Cons a (List a)
data ListF a r = Nil_ | Cons_ a r deriving Functor
type List a = Fix (ListF a)

nil :: List a
nil = Fix Nil_
cons :: a -> List a -> List a
cons x xs = Fix (Cons_ x xs)


-- binary trees: note two recursive occurrences
-- data Tree a = Leaf | Node (Tree a) a (Tree a)
data TreeF a r = Leaf_ | Node_ r a r deriving Functor
type Tree a = Fix (TreeF a)

leaf :: Tree a
leaf = Fix Leaf_
node :: Tree a -> a -> Tree a -> Tree a
node l x r = Fix (Node_ l x r)

Dobrando uma estrutura uma camada de cada vez #

Catamorphisms, ou folds, modelo de recursão primitiva. O cata desmonta um ponto fixo camada por camada, usando uma função álgebra (ou função de dobramento) para processar cada camada. cata requer uma instância Functor para o tipo de template f.

cata :: Functor f => (f a -> a) -> Fix f -> a
cata f = f . fmap (cata f) . unFix

-- list example
foldr :: (a -> b -> b) -> b -> List a -> b
foldr f z = cata alg
    where alg Nil_ = z
          alg (Cons_ x acc) = f x acc

Desdobrar uma estrutura uma camada de cada vez #

Anamorphisms, ou unfolds, modela a correcursão primitiva. ana constrói um ponto fixo camada por camada, usando uma função coalgebra (ou função desdobramento) para produzir cada nova camada. ana requer uma instância Functor para o tipo de modelo f.

ana :: Functor f => (a -> f a) -> a -> Fix f
ana f = Fix . fmap (ana f) . f

-- list example
unfoldr :: (b -> Maybe (a, b)) -> b -> List a
unfoldr f = ana coalg
    where coalg x = case f x of
                         Nothing -> Nil_
                         Just (x, y) -> Cons_ x y

Observe que ana e cata são dual. Os tipos e implementações são imagens espelhadas um do outro.

Desdobrar e depois dobrar, fundido #

É comum estruturar um programa como a construção de uma estrutura de dados e então recolhê-la em um único valor. Isso é chamado de hilomorfismo ou refold. É possível eliminar completamente a estrutura intermediária para melhorar a eficiência.

hylo :: Functor f => (a -> f a) -> (f b -> b) -> a -> b
hylo f g = g . fmap (hylo f g) . f  -- no mention of Fix!

Derivação:

hylo f g = cata g . ana f
         = g . fmap (cata g) . unFix . Fix . fmap (ana f) . f  -- definition of cata and ana
         = g . fmap (cata g) . fmap (ana f) . f  -- unfix . Fix = id
         = g . fmap (cata g . ana f) . f  -- Functor law
         = g . fmap (hylo f g) . f  -- definition of hylo

Recursão primitiva #

Paramorfismos modelo de recursão primitiva. A cada iteração da dobra, a função de dobra recebe a subárvore para processamento adicional.

para :: Functor f => (f (Fix f, a) -> a) -> Fix f -> a
para f = f . fmap (\x -> (x, para f x)) . unFix

As caudas do Prelude podem ser modeladas como um paramorfismo.

tails :: List a -> List (List a)
tails = para alg
    where alg Nil_ = cons nil nil  -- [[]]
          alg (Cons_ x (xs, xss)) = cons (cons x xs) xss  -- (x:xs):xss

Correcursão primitiva #

Apomorfismos modelo de corecursão primitiva. A cada iteração do desdobramento, a função de desdobramento pode retornar uma nova semente ou uma subárvore inteira.

apo :: Functor f => (a -> f (Either (Fix f) a)) -> a -> Fix f
apo f = Fix . fmap (either id (apo f)) . f

Observe que apo e para são dual. As setas no tipo são invertidas; a tupla em para é dual ao Either em apo, e as implementações são imagens espelhadas uma da outra.
