  • Haskell não suporta tuplas com um componente nativamente.

  • Unidades (escritas ()) podem ser entendidas como tuplas com zero componentes.

  • Não há funções predefinidas para extrair componentes de tuplas com mais de dois componentes. Se você achar que precisa de tais funções, considere usar um tipo de dados personalizado com rótulos de registro em vez do tipo de tupla. Então você pode usar as etiquetas de registro como funções para extrair os componentes.

Extrair componentes de tupla #

Use as funções fst e snd (de Prelude ou Data.Tuple) para extrair o primeiro e o segundo componente dos pares.

fst (1, 2) -- evaluates to 1

snd (1, 2) -- evaluates to 2

Ou use a correspondência de padrões.

case (1, 2) of (result, _) => result -- evaluates to 1

case (1, 2) of (_, result) => result -- evaluates to 2

A correspondência de padrões também funciona para tuplas com mais de dois componentes.

case (1, 2, 3) of (result, _, _) => result -- evaluates to 1

case (1, 2, 3) of (_, result, _) => result -- evaluates to 2

case (1, 2, 3) of (_, _, result) => result -- evaluates to 3

Haskell não fornece funções padrão como fst ou snd para tuplas com mais de dois componentes. A biblioteca tuple no Hackage fornece tais funções no [Data.Tuple.Select](https://hackage.haskell.org/package /tuple/docs/Data-Tuple-Select.html).

Rigor na correspondência de uma tupla #

O padrão (p1, p2) é estrito no construtor de tupla mais externo, o que pode levar a comportamento de rigidez inesperado. Por exemplo, a seguinte expressão diverge (usando Data.Function.fix):

fix $ \(x, y) -> (1, 2)

já que a correspondência em (x, y) é estrita no construtor da tupla. No entanto, a expressão a seguir, usando um [padrão irrefutável](https://www.wikiod.com/pt/haskell/rigor#Padrões preguiçosos), é avaliada como (1, 2) como esperado:

fix $ \ ~(x, y) -> (1, 2)

Construir valores de tupla #

Use parênteses e vírgulas para criar tuplas. Use uma vírgula para criar um par.

(1, 2)

Use mais vírgulas para criar tuplas com mais componentes.

(1, 2, 3)

(1, 2, 3, 4)

Observe que também é possível declarar tuplas usando sua forma não açucarada.

(,) 1 2     -- equivalent to (1,2)
(,,) 1 2 3  -- equivalent to (1,2,3)

Tuplas podem conter valores de diferentes tipos.

("answer", 42, '?')

As tuplas podem conter valores complexos, como listas ou mais tuplas.

([1, 2, 3], "hello", ('A', 65))

(1, (2, (3, 4), 5), 6)

Escrever tipos de tupla #

Use parênteses e vírgulas para escrever tipos de tupla. Use uma vírgula para escrever um tipo de par.

(Int, Int)

Use mais vírgulas para escrever tipos de tupla com mais componentes.

(Int, Int, Int)

(Int, Int, Int, Int)

Tuplas podem conter valores de diferentes tipos.

(String, Int, Char)

As tuplas podem conter valores complexos, como listas ou mais tuplas.

([Int], String, (Char, Int))

(Int, (Int, (Int, Int), Int), Int)

Correspondência de padrões em tuplas #

A correspondência de padrões em tuplas usa os construtores de tupla. Para corresponder a um par, por exemplo, usaríamos o construtor (,):

myFunction1 (a, b) = ...

Usamos mais vírgulas para combinar tuplas com mais componentes:

myFunction2 (a, b, c) = ...

myFunction3 (a, b, c, d) = ...

Os padrões de tupla podem conter padrões complexos, como padrões de lista ou mais padrões de tupla.

myFunction4 ([a, b, c], d, e) = ...

myFunction5 (a, (b, (c, d), e), f) = ...

Aplicar uma função binária a uma tupla (sem curry) #

Use a função uncurry (de Prelude ou Data.Tuple) para converter uma função binária em uma função em tuplas.

uncurry (+) (1, 2) -- computes 3

uncurry map (negate, [1, 2, 3]) -- computes [-1, -2, -3]

uncurry uncurry ((+), (1, 2)) -- computes 3

map (uncurry (+)) [(1, 2), (3, 4), (5, 6)] -- computes [3, 7, 11]

uncurry (curry f) -- computes the same as f

Aplicar uma função de tupla a dois argumentos (currying) #

Use a função curry (de Prelude ou Data.Tuple) para converter uma função que recebe tuplas em uma função que recebe dois argumentos.

curry fst 1 2 -- computes 1

curry snd 1 2 -- computes 2

curry (uncurry f) -- computes the same as f

import Data.Tuple (swap)
curry swap 1 2 -- computes (2, 1)

Trocar componentes de pares #

Use swap (de Data.Tuple) para trocar os componentes de um par.

import Data.Tuple (swap)
swap (1, 2) -- evaluates to (2, 1)

Ou use a correspondência de padrões.

case (1, 2) of (x, y) => (y, x) -- evaluates to (2, 1)
