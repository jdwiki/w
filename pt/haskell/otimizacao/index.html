Compilando seu programa para criação de perfil #

O compilador GHC tem suporte maduro para compilar com anotações de criação de perfil.

Usar os sinalizadores -prof e -fprof-auto ao compilar adicionará suporte ao seu binário para sinalizadores de perfil para uso em tempo de execução.

Suponha que temos este programa:

main = print (fib 30)
fib n = if n < 2 then 1 else fib (n-1) + fib (n-2)

Compilei assim:

ghc -prof -fprof-auto -rtsopts Main.hs

Em seguida, executei-o com opções do sistema de tempo de execução para criação de perfil:

./Main +RTS -p

Veremos um arquivo main.prof criado após a execução (depois que o programa for encerrado), e isso nos dará todos os tipos de informações de perfil, como centros de custo, que nos fornecem um detalhamento do custo associado à execução das várias partes do o código:

Centros de Custo #

Centros de custo são anotações em um programa Haskell que podem ser adicionados automaticamente pelo compilador GHC – usando -fprot-auto – ou por um programador usando {-# SCC "name" #-} <expression>, onde “name” é qualquer nome que você desejar e <expression> é qualquer expressão Haskell válida:

-- Main.hs
main :: IO ()
main = do let l = [1..9999999]
          print $ {-# SCC "print_list" #-} (length l)

Compilando com -fprof e executando com +RTS -p e.g. ghc -prof -rtsopts Main.hs && ./Main.hs +RTS -p produziria Main.prof assim que o programa fosse encerrado.
