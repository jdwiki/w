Vinculação de propriedade

A propriedade de um objeto pode receber um valor estático que permanece constante até que seja explicitamente atribuído um novo valor. No entanto, para aproveitar ao máximo o QML e seu suporte interno para comportamentos de objetos dinâmicos, a maioria dos objetos QML usa associações de propriedade.

As associações de propriedade são um recurso central do QML que permite que os desenvolvedores especifiquem relacionamentos entre diferentes propriedades de objetos. Quando as dependências de uma propriedade mudam de valor, a propriedade é atualizada automaticamente de acordo com o relacionamento especificado.

Um exemplo mais complicado #

No exemplo simples, simplesmente definimos a largura do retângulo para a de seu pai. Vamos considerar um exemplo mais complicado:

ApplicationWindow {
    visible: true
    width: 400
    height: 640

    Rectangle{
        id: rect
        anchors.centerIn: parent
        height: 100
        width: parent.width/2 + parent.width/3
        color: "blue"
    }
}

No exemplo, realizamos uma operação aritmética no valor que está sendo associado. Se você redimensionar a janela do aplicativo em execução para a largura máxima, o espaço entre o retângulo e a janela do aplicativo será maior e vice-versa.

Noções básicas sobre vinculações de propriedade #

Considere este exemplo simples:

import QtQuick 2.7
import QtQuick.Controls 2.0

ApplicationWindow {
    visible: true
    width: 400
    height: 640

    Rectangle{
        id: rect
        anchors.centerIn: parent
        height: 100
        width: parent.width
        color: "blue"
    }
}

No exemplo acima, a largura de Rectangle é vinculada à de seu pai. Se você alterar a largura da janela do aplicativo em execução, a largura do retângulo também será alterada.

Crie ligações com arquivos QML criados dinamicamente #

Ao usar instâncias de arquivos QML declarando-os diretamente, cada propriedade cria uma ligação. Isso é explicado nos exemplos acima.

É assim que você cria componentes dinamicamente:

var component = Qt.createComponent("Popup.qml");
var popup = component.createObject(parent, {"width": mainWindow.width, "height": mainWindow.height});

Quando o tamanho da mainWindow muda, o tamanho do PopUp criado não é afetado. Para criar uma ligação, você define o tamanho do popup assim:

var component = Qt.createComponent("Popup.qml");
var options = {
    "width": Qt.binding(function() { return mainWindow.width }),
    "height": Qt.binding(function() { return mainWindow.height }),
};
var popup = component.createObject(parent, options);

Agora o tamanho do PopUp dependerá da mainWindow.
