Instalando o SOLR #

Introdução #

O procedimento a seguir foi testado em uma instância de teste na AWS, com Redhat e Solr 6.1.0. Pode ser necessário ajustar o processo ao seu sistema operacional e ambiente de acordo.

Pré-requisitos #

  1. Certifique-se de usar o RedHat ou um sistema operacional similar (baseado no Fedora). cat /etc/redhat-release exibe a versão do seu sistema operacional.

  2. Verifique se o java 1.6 ou superior está instalado qual java

  3. Instale o Java, se necessário sudo yum lista disponível java* sudo yum install java-1.8.0-openjdk.x86_64

  4. Verifique se está instalado corretamente which java exibe a página inicial do Java java -version exibe a versão do Java

  5. Crie um usuário SOLR. ``sudo adduser solr''

  6. Adicione uma senha para o usuário. sudo passwd solr

  7. Habilite sudo no usuário SOLR, execute visudo sudo visudo

  8. Localize as seguintes linhas: ## Permite que as pessoas na roda do grupo executem todos os comandos # %wheel ALL=(ALL) ALL

  9. Se %wheel for comentado, descomente a segunda linha removendo o caractere #. %wheel ALL=(ALL) ALL

  10. Se você fez uma alteração use :wq caso contrário use :q para sair.

  11. Adicione o usuário solr ao grupo wheel. sudo usermod -aG wheel solr

  12. Mude para o usuário solr e verifique se você tem privilégios de root: su sol - sudo whoami

Baixando SOLR #

  1. Encontre seu espelho local em: http://www.apache.org/dyn/closer.lua/lucene/solr/

  2. Altere o local do seu diretório para optar: cd /opt/

  3. Baixe uma cópia do pacote do espelho: sudo curl -O http://www.trieuvan.com/apache/lucene/solr/6.1.0/solr-6.1.0.tgz

  4. Descompacte o pacote: sudo tar zxvf solr-6.1.0.tgz

  5. Copie o script do instalador para sua pasta: sudo cp /opt/solr-6.1.0/bin/install_solr_service.sh .

  6. Remova os arquivos desnecessários: sudo rm -rf solr-6.1.0

Instalar #

  1. Execute o script de instalação: sudo ./install_solr_service.sh solr-6.1.0.tgz
  2. Faça a inicialização automática do serviço SOLR quando o servidor for reinicializado. sudo chkconfig --add solr chkconfig | grep solr
  3. Alterar proprietário do serviço sudo chown -R solr:solr /var/solr/

Testando sua instalação #

  1. Crie um núcleo a partir da linha de comando: sudo su - solr -c "/opt/solr/bin/solr create -c NewCore1 -n data_driven_schema_configs"

  2. Abra o Admin em um navegador: http://<solr_server>:8983/solr/#/

  3. Abra a lista de núcleos no menu para ver o núcleo NewCore1.

  4. Teste se os núcleos estão pegajosos: sudo service solr restart

  5. Atualize o Admin em um navegador: http://<solr_server>:8983/solr/#/ Certifique-se de que a página Admin seja recarregada e o núcleo reapareça após a reinicialização.

  6. Visualize o status do servidor na linha de comando: status do solr do serviço sudo

  7. Prepare uma nova configuração de núcleo para criação de núcleo no Web Admin, alterando o local do diretório para dados: cd var/solr/data/

  8. É aqui que os novos núcleos são armazenados: ll

  9. A pasta conf do núcleo recém-criado pode ser usada como modelo: ll NewCore1/

  10. Crie uma pasta para outro núcleo que você criará no Web Admin: mkdir CoreFromWebAdmin

  11. Copie o diretório conf para o novo local: sudo cp -R NewCore1/conf/ CoreFromWebAdmin

  12. Mude para a interface Web Admin em seu navegador

  13. Clique em Adicionar núcleo

  14. Adicione CoreFromWebAdmin como o nome e a pasta do novo núcleo.

  15. Abra o novo núcleo.

  16. Clique em documentos para adicionar documentos.

  17. Selecione o formato XML e cole o código abaixo:

    F9V7464-APL-KIT Belkin Mobile Power Cord for iPod w/ Dock Belkin

    belkin electronics connector car power adapter, white 4.0 19.95 1 false

    45.18014,-93.87741 2005-08-01T16:30:25Z

    IW-032 iPod & iPod Mini USB 2.0 Cable Belkin belkin electronics connector car power adapter for iPod, white 2.0 11.50 1 false 37.7752,-122.4232 2006-02-14T23:55:59Z F887464-APL-KIT Belkin Mobile Power Cord for iPod w/ Dock Belkin belkin electronics connector car power adapter, black 4.0 19.95 1 true 45.18014,-93.87741 FAV7464-APL-KIT Belkin Mobile Power Cord for iPod w/ Dock Belkin belkin electronics connector car power adapter, blue 4.0 15.95 2 true 45.18014,-93.87741 2015-09-21T16:30:25Z

Se sua resposta retornar um sucesso, você instalou o SOLR com êxito e verificou sua instalação.

Introdução #

O Solr é um servidor de pesquisa corporativa autônomo com uma API semelhante a REST. Você coloca documentos nele (chamado “indexação”) via JSON, XML, CSV ou binário sobre HTTP. Você o consulta via HTTP GET e recebe resultados JSON, XML, CSV ou binários. O Solr usa a biblioteca de pesquisa Lucene e a estende.

Aqui estão alguns dos principais recursos que o solr fornece:

- Recursos avançados de pesquisa de texto completo

Desenvolvido por Lucene™, o Solr permite recursos avançados de correspondência, incluindo frases, curingas, junções, agrupamento e muito mais em qualquer tipo de dados

  • Otimizado para tráfego de alto volume

Solr é comprovado em escalas extremamente grandes em todo o mundo

  • Interfaces abertas baseadas em padrões - XML, JSON e HTTP

O Solr usa as ferramentas que você usa para tornar a criação de aplicativos em um piscar de olhos

  • Interfaces de administração abrangentes

O Solr é fornecido com uma interface de usuário administrativa responsiva e integrada para facilitar o controle de suas instâncias do Solr

  • Monitoramento fácil

Precisa de mais informações sobre suas instâncias? Solr publica cargas de dados métricos via JMX

  • Altamente escalável e tolerante a falhas

Construído no Apache Zookeeper testado em batalha, o Solr facilita a expansão e a redução. A Solr prepara a replicação, distribuição, reequilíbrio e tolerância a falhas fora da caixa.

  • Flexível e adaptável com fácil configuração

O Solr’s foi projetado para se adaptar às suas necessidades enquanto simplifica a configuração

  • Indexação quase em tempo real

Quer ver suas atualizações agora? O Solr aproveita os recursos de indexação em tempo quase real do Lucene para garantir que você veja seu conteúdo quando quiser

  • Arquitetura de plug-in extensível

O Solr publica muitos pontos de extensão bem definidos que facilitam a inserção de plug-ins de índice e tempo de consulta. Claro, como é um código aberto licenciado pelo Apache, você pode alterar qualquer código que desejar!

Alguns recursos interessantes:

  • Esquema quando quiser, sem esquema quando não quiser

Use o modo sem esquema orientado a dados do Solr ao começar e, em seguida, bloqueie-o quando for a hora da produção.

  • Extensões poderosas

O Solr é fornecido com plug-ins opcionais para indexação de conteúdo rico (por exemplo, PDFs, Word), detecção de idioma, agrupamento de resultados de pesquisa e muito mais

  • Pesquisa e Filtragem Facetada

Fatie e divida seus dados como achar melhor usando uma grande variedade de algoritmos de facetação

  • Pesquisa Geoespacial

A ativação da pesquisa baseada em localização é simples com o suporte integrado do Solr para pesquisa espacial

  • Sugestões de consulta, ortografia e mais

O Solr é fornecido com recursos avançados para preenchimento automático (pesquisa antecipada), verificação ortográfica e muito mais

  • Análise de documentos avançados

O Solr é fornecido com Apache Tika integrado, facilitando a indexação de conteúdo rico, como Adobe PDF, Microsoft Word e muito mais.
