Localizar elemento na lista separada por vírgulas #

SELECT FIND_IN_SET('b','a,b,c');

Valor de retorno:

2

SELECT FIND_IN_SET('d','a,b,c');

Valor de retorno:

0

STR_TO_DATE - Converte string em data #

Com uma coluna de um dos tipos de string, chamada my_date_field com um valor como [the string] 07/25/2016, a seguinte declaração demonstra o uso da função STR_TO_DATE:

SELECT STR_TO_DATE(my_date_field, '%m/%d/%Y') FROM my_table;

Você também pode usar esta função como parte da cláusula WHERE.

LOWER() / LCASE() #

Converter em minúsculas o argumento string

Sintaxe: LOWER(str)

LOWER('fOoBar') -- 'foobar'
LCASE('fOoBar') -- 'foobar'

SUBSTITUIR() #

Converter em minúsculas o argumento string

Sintaxe: REPLACE(str, from_str, to_str)

REPLACE('foobarbaz', 'bar', 'BAR') -- 'fooBARbaz'
REPLACE('foobarbaz', 'zzz', 'ZZZ') -- 'foobarbaz'

SUBSTRING() #

SUBSTRING (ou equivalente: SUBSTR) retorna a substring começando na posição especificada e, opcionalmente, com o comprimento especificado

Sintaxe: SUBSTRING(str, start_position)

SELECT SUBSTRING('foobarbaz', 4); -- 'barbaz'
SELECT SUBSTRING('foobarbaz' FROM 4); -- 'barbaz'

-- using negative indexing
SELECT SUBSTRING('foobarbaz', -6); -- 'barbaz'
SELECT SUBSTRING('foobarbaz' FROM -6); -- 'barbaz'

Sintaxe: SUBSTRING(str, start_position, comprimento)

SELECT SUBSTRING('foobarbaz', 4, 3); -- 'bar'
SELECT SUBSTRING('foobarbaz', FROM 4 FOR 3); -- 'bar'

-- using negative indexing
SELECT SUBSTRING('foobarbaz', -6, 3); -- 'bar'
SELECT SUBSTRING('foobarbaz' FROM -6 FOR 3); -- 'bar'

UPPER() / UCASE() #

Converter em maiúsculas o argumento string

Sintaxe: UPPER(str)

UPPER('fOoBar') -- 'FOOBAR'
UCASE('fOoBar') -- 'FOOBAR'

COMPRIMENTO() #

Retorna o comprimento da string em bytes. Como alguns caracteres podem ser codificados usando mais de um byte, se você quiser o comprimento em caracteres, consulte CHAR_LENGTH()

Sintaxe: LENGTH(str)

LENGTH('foobar') -- 6
LENGTH('fööbar') -- 8 -- contrast with CHAR_LENGTH(...) = 6

CHAR_LENGTH() #

Retorna o número de caracteres na string

Sintaxe: CHAR_LENGTH(str)

CHAR_LENGTH('foobar') -- 6
CHAR_LENGTH('fööbar') -- 6 -- contrast with LENGTH(...) = 8

HEX(str) #

Converta o argumento para hexadecimal. Isso é usado para strings.

HEX('fööbar') -- 66F6F6626172 -- in "CHARACTER SET latin1" because "F6" is hex for ö
HEX('fööbar') -- 66C3B6C3B6626172  -- in "CHARACTER SET utf8 or utf8mb4" because "C3B6" is hex for ö
