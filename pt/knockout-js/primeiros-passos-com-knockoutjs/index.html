Começando: Olá, mundo! #

Criando um documento HTML e ativando knockout.js #

Crie um arquivo HTML e inclua knockout.js por meio de uma tag <script>.

<!DOCTYPE html>
<html>
<head>
    <title>Hello world! (knockout.js)</title>
</head>
<body>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/knockout/3.4.0/knockout-debug.js"></script>
</body>
</html>

Adicione uma segunda tag <script> sob o script knockout. Nesta tag de script, inicializaremos um modelo de exibição e aplicaremos data binds ao nosso Document.

<script>
  var ViewModel = function() {
    this.greeting = ko.observable("Hello");
    this.name = ko.observable("World");

    this.appHeading = ko.pureComputed(function() {
      return this.greeting() + ", " + this.name() + "!";
    }, this);
  };

  var appVM = new ViewModel();

  ko.applyBindings(appVM);
</script>

Agora, continue criando uma view adicionando algum HTML ao corpo:

<section>
  <h1 data-bind="text: appHeading"></h1>
  <p>Greeting: <input data-bind="textInput: greeting" /></p>
  <p>Name: <input data-bind="textInput: name" /></p>
</section>

Quando o documento HTML for aberto e os scripts executados, você verá uma página que diz Hello, World!. Quando você altera as palavras nos elementos <input>, o texto <h1> é atualizado automaticamente.

Como funciona o arquivo criado #

  1. Uma versão de depuração do knockout é carregada de uma fonte externa (cdnjs)

Código:

<script src="https://cdnjs.cloudflare.com/ajax/libs/knockout/3.4.0/knockout-debug.js"></script>
  1. Uma instância de viewmodel é criada com propriedades observable. Isso significa que o knockout é capaz de detectar alterações e atualizar a interface do usuário de acordo.

Código:

var appVM = new ViewModel();
  1. Knockout verifica o DOM em busca de atributos data-bind e atualiza a UI usando o viewmodel fornecido.

Código:

ko.applyBindings(appVM);
  1. Quando encontra uma ligação text, knockout usa o valor da propriedade conforme definido no viewmodel vinculado para injetar um nó de texto:

Código:

<h1 data-bind="text: appHeading"></h1>

Instalação ou configuração #

Knockout está disponível na maioria das plataformas JavaScript ou como um script independente.

Incluir como script #

Você pode baixar o script da página de download e incluí-lo em sua página com uma tag de script padrão

<script type='text/javascript' src='knockout-3.4.0.js'></script>

Usando um CDN #

Você também pode incluir knockout do Microsoft CDN ou CDNJS

<script type='text/javascript' src='//ajax.aspnetcdn.com/ajax/knockout/knockout-3.4.0.js'></script>

OU

<script type='text/javascript' src='//cdnjs.cloudflare.com/ajax/libs/knockout/3.4.0/knockout-min.js'></script>

#Instala a partir do npm

npm install knockout

opcionalmente você pode adicionar o parâmetro --save para mantê-lo em seu arquivo package.json

#Instalar do bower

bower install knockout

opcionalmente, você pode adicionar o parâmetro --save para mantê-lo em seu arquivo bower.json

#Instalação do NuGet

Install-Package knockoutjs

Observáveis ​​calculados #

Observáveis ​​computados são funções que podem “observar” ou “reagir” a outros observáveis ​​no modelo de exibição. O exemplo a seguir mostra como você exibiria o número total de usuários e a idade média.

Observação: o exemplo abaixo também pode usar pureComputed() (introduzido na v3.2.0), pois a função simplesmente calcula algo com base em outras propriedades do modelo de visualização e retorna um valor.

<div>
  Total Users: <span data-bind="text: TotalUsers">2</span><br>
  Average Age: <span data-bind="text: UsersAverageAge">32</span>
</div>

var viewModel = function() {

    var self = this;

    this.Users = ko.observableArray([
        { Name: "John Doe", Age: 30 },        
        { Name: "Jane Doe", Age: 34 }
    ]);

    this.TotalUsers = ko.computed(function() {
        return self.Users().length;
    });

    this.UsersAverageAge = ko.computed(function() {
        var totalAge = 0;
        self.Users().forEach(function(user) {
            totalAge += user.Age;
        });
   
        return totalAge / self.TotalUsers();
    });
};

ko.applyBindings(viewModel);
