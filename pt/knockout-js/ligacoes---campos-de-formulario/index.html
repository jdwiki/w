Clique #

A ligação click pode ser usada com qualquer elemento DOM visível para adicionar um manipulador de eventos, que invocará uma função JavaScript, quando o elemento for clicado.

<button data-bind="click: onClick">Click me</button>

ko.applyBindings({
  onClick: function(data, event) {
    // data: the context of the element that triggered the event
    console.log(data);

    // event: the click event
    console.log(event);
  }
});

Opções #

Use esta ligação para criar opções para um item selecionado

<select data-bind="options: gasGiants"></select>
 
<script type="text/javascript">
    var viewModel = {
        gasGiants: ko.observableArray(['Jupiter', 'Saturn', 'Neptune', 'Uranus'])
    };
</script>

Você também pode usar propriedades dentro do array para exibir na lista e para salvar no viewModel: optionsText habilita um texto de exibição personalizado

optionsValue define a propriedade value da <option> correspondente

value armazena o valor da opção selecionada em um observável do viewModel

<select data-bind="options: gasGiants, optionsText:'name', optionsValue:'id', value:selectedPlanetId"></select>
 
<script type="text/javascript">
    var viewModel = {
        selectedPlanetId: ko.observable(),
        gasGiants: ko.observableArray([{name:'Jupiter', id:'0'},
                                       {name:'Saturn', id:'1'},
                                       {name:'Neptune', id:'2'},
                                       {name:'Uranus', id:'3'}])
    };
</script>

Para armazenar os resultados de uma lista de seleção múltipla, a associação de opções pode ser combinada com a associação selectedOptions.

<select data-bind="options: gasGiants, selectedOptions: chosenGasGiants" multiple="true"></select>

<script type="text/javascript">
    var viewModel = {
        gasGiants: ko.observableArray(['Jupiter', 'Saturn', 'Neptune', 'Uranus'])
        chosenGasGiants: ko.observableArray(['Jupiter','Saturn']) // Initial selection
    }; </script>

desativado ativado #

A ligação desabilitada adiciona um atributo disabled a um elemento html fazendo com que ele não seja mais editável ou clicável. Isso é útil principalmente para elementos <input>, <select>, <textarea>, <a> e <button>

<input data-bind="disabled: disableInput"/> 

<script type="text/javascript">
var viewModel = {
    disableInput: ko.observable(true);
};
</script>

O inverso da ligação disabled é enabled

A visibilidade também pode ser calculada usando funções JavaScript. Quaisquer observáveis ​​usados ​​nestas funções devem ser chamados com parênteses 

<script type="text/javascript">
var viewModel = {
    disableInput: ko.observable(true);
};
</script>

ou

<input data-bind="disabled: allValues().length>4"/> 

<script type="text/javascript">
var viewModel = {
    allValues: ko.observableArray([1,2,3,4,5]);
};
</script>

enviar #

Manipulador de eventos a ser invocado quando um elemento DOM é enviado.

<form data-bind="submit: doSomething">
    <!-- form content here -->
    <button type="submit"></button>
</form>

<script type="text/javascript">    
    var vm = { 
        doSomething: function(data){
            //do something here
        }; 
    }
</script>

Knockout impedirá a ação de envio padrão do navegador para esse formulário. Se você quiser que seu formulário seja enviado como um formulário HTML normal, basta retornar true no manipulador de envio.

Valor #

Use a ligação de valor para obter o valor de um elemento. A associação de valor pode ser aplicada a qualquer controle de formulário, no entanto, existem outras associações que podem ser mais adequadas para caixas de seleção, botões de opção e entradas de texto.

O exemplo a seguir ilustra como aplicar o elemento de ligação a vários campos de entrada de formulário e como preencher os valores padrão:

Definição de ViewModel:

var MyViewModel = function(){
    var self = this;
  //Initialize valueOne
  self.valueOne = ko.observable();
  //Initialize valueTwo with a default value of "Value two"
  self.valueTwo = ko.observable("Value two");
  //Initialize the color dropdown, and by default, select the "blue" option
  self.color = ko.observable("blue");
  
  self.valueOne.subscribe(function(newValue){
      console.log("valueOne: " + newValue);
  });
  
  self.valueTwo.subscribe(function(newValue){
      console.log("valueTwo: " + newValue);
  });
  
  self.color.subscribe(function(newValue){
      console.log("color: " + newValue);
  });
}

Marcação associada:

<input type="text" data-bind="value: valueOne" />
<input type="text" data-bind="value: valueTwo" />

<select data-bind="value: color">
  <option value="red">Red</option>
  <option value="green">Green</option>
  <option value="blue">Blue</option>
</select>

No exemplo acima, quando um valor muda, o novo valor será registrado no console. Os valores iniciais não acionarão um evento de alteração.

Por padrão, a associação de valor define uma alteração como uma alteração no valor dos elementos e o foco sendo transferido para outro elemento. Isso pode ser alterado usando a opção valueUpdate:

<input type="text" data-bind="value: valueOne, valueUpdate: 'keyup'" />

O exemplo acima alterará a atualização do valor para disparar na ativação. As opções disponíveis são input, keyup, keypress e afterkeydown.
