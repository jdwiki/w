Leitura de teclas pressionadas e diferença entre GetKey, GetKeyDown e GetKeyUp #

A entrada deve ser lida da função Update.

Referência para todas as enumerações Keycode disponíveis.

1. Leitura de teclas pressionadas com Input.GetKey:

Input.GetKey retornará repetidamente true enquanto o usuário mantém pressionada a chave especificada. Isso pode ser usado para repetidamente disparar uma arma enquanto mantém pressionada a tecla especificada. Abaixo está um exemplo de disparo automático de bala quando a tecla Espaço é pressionada. O jogador não precisa pressionar e soltar a tecla repetidamente.

public GameObject bulletPrefab;
public float shootForce = 50f;

void Update()
{
    if (Input.GetKey(KeyCode.Space))
    {
        Debug.Log("Shooting a bullet while SpaceBar is held down");

        //Instantiate bullet
        GameObject bullet = Instantiate(bulletPrefab, transform.position, transform.rotation) as GameObject;

        //Get the Rigidbody from the bullet then add a force to the bullet
        bullet.GetComponent<Rigidbody>().AddForce(bullet.transform.forward * shootForce);
    }
}

2. Lendo a tecla pressionada com Input.GetKeyDown:

Input.GetKeyDown será verdadeiro apenas uma vez quando a tecla especificada for pressionada. Esta é a principal diferença entre Input.GetKey e Input.GetKeyDown. Um exemplo de uso de seu uso é ativar/desativar uma interface do usuário ou lanterna ou um item.

public Light flashLight;
bool enableFlashLight = false;

void Update()
{
    if (Input.GetKeyDown(KeyCode.Space))
    {
        //Toggle Light 
        enableFlashLight = !enableFlashLight;
        if (enableFlashLight)
        {
            flashLight.enabled = true;
            Debug.Log("Light Enabled!");
        }
        else
        {
            flashLight.enabled = false;
            Debug.Log("Light Disabled!");
        }
    }
}

3. Lendo a tecla pressionada com Input.GetKeyUp:

Este é exatamente o oposto de Input.GetKeyDown. É usado para detectar quando a tecla pressionada é liberada/levantada. Assim como Input.GetKeyDown, ele retorna true apenas uma vez. Por exemplo, você pode ativar a luz quando a tecla é pressionada com Input.GetKeyDown e desabilitar a luz quando a tecla é solta com Input.GetKeyUp.

public Light flashLight;
void Update()
{
    //Disable Light when Space Key is pressed
    if (Input.GetKeyDown(KeyCode.Space))
    {
        flashLight.enabled = true;
        Debug.Log("Light Enabled!");
    }

    //Disable Light when Space Key is released
    if (Input.GetKeyUp(KeyCode.Space))
    {
        flashLight.enabled = false;
        Debug.Log("Light Disabled!");
    }
}

Leia os cliques do botão do mouse (esquerdo, meio, direito) #

Essas funções são usadas para verificar os cliques do botão do mouse.

  • Input.GetMouseButton(int button);
  • Input.GetMouseButtonDown(int button);
  • Input.GetMouseButtonUp(int button);

Todos eles usam o mesmo parâmetro.

  • 0 = Clique esquerdo do mouse.
  • 1 = Clique com o botão direito do mouse.
  • 2 = Clique do meio do mouse.

GetMouseButton é usado para detectar quando o botão do mouse é continuamente pressionado. Ele retorna true enquanto o botão do mouse especificado está sendo pressionado.

void Update()
{
    if (Input.GetMouseButton(0))
    {
        Debug.Log("Left Mouse Button Down");
    }

    if (Input.GetMouseButton(1))
    {
        Debug.Log("Right Mouse Button Down");
    }

    if (Input.GetMouseButton(2))
    {
        Debug.Log("Middle Mouse Button Down");
    }
}

GetMouseButtonDown é usado para detectar quando há clique do mouse. Ele retorna true se for pressionado uma vez. Ele não retornará true novamente até que o botão do mouse seja liberado e pressionado novamente.

void Update()
{
    if (Input.GetMouseButtonDown(0))
    {
        Debug.Log("Left Mouse Button Clicked");
    }

    if (Input.GetMouseButtonDown(1))
    {
        Debug.Log("Right Mouse Button Clicked");
    }

    if (Input.GetMouseButtonDown(2))
    {
        Debug.Log("Middle Mouse Button Clicked");
    }
}

GetMouseButtonUp é usado para detectar quando o botão do mouse especificado é liberado. Isso só retornará true quando o botão do mouse especificado for liberado. Para retornar true novamente, ele deve ser pressionado e liberado novamente.

void Update()
{
    if (Input.GetMouseButtonUp(0))
    {
        Debug.Log("Left Mouse Button Released");
    }

    if (Input.GetMouseButtonUp(1))
    {
        Debug.Log("Right Mouse Button Released");
    }

    if (Input.GetMouseButtonUp(2))
    {
        Debug.Log("Middle Mouse Button Released");
    }
}

Ler Sensor do Acelerômetro (Básico) #

Input.acceleration é usado para ler o sensor do acelerômetro. Ele retorna Vector3 como resultado que contém valores de eixo x,y e z no espaço 3D.

void Update()
{
    Vector3 acclerometerValue = rawAccelValue();
    Debug.Log("X: " + acclerometerValue.x + "  Y: " + acclerometerValue.y + "  Z: " + acclerometerValue.z);
}

Vector3 rawAccelValue()
{
    return Input.acceleration;
}

Ler Sensor do Acelerômetro (Avançado) #

Usar valores brutos diretamente do sensor do acelerômetro para mover ou girar um GameObject pode causar problemas como movimentos bruscos ou vibrações. Recomenda-se suavizar os valores antes de usá-los. Na verdade, os valores do sensor do acelerômetro devem sempre ser suavizados antes do uso. Isso pode ser feito com um filtro passa-baixa e é aqui que o Vector3.Lerp entra em ação.

//The lower this value, the less smooth the value is and faster Accel is updated. 30 seems fine for this
const float updateSpeed = 30.0f;

float AccelerometerUpdateInterval = 1.0f / updateSpeed;
float LowPassKernelWidthInSeconds = 1.0f;
float LowPassFilterFactor = 0;
Vector3 lowPassValue = Vector3.zero;

void Start()
{
    //Filter Accelerometer
    LowPassFilterFactor = AccelerometerUpdateInterval / LowPassKernelWidthInSeconds;
    lowPassValue = Input.acceleration;
}

void Update()
{

    //Get Raw Accelerometer values (pass in false to get raw Accelerometer values)
    Vector3 rawAccelValue = filterAccelValue(false);
    Debug.Log("RAW X: " + rawAccelValue.x + "  Y: " + rawAccelValue.y + "  Z: " + rawAccelValue.z);

    //Get smoothed Accelerometer values (pass in true to get Filtered Accelerometer values)
    Vector3 filteredAccelValue = filterAccelValue(true);
    Debug.Log("FILTERED X: " + filteredAccelValue.x + "  Y: " + filteredAccelValue.y + "  Z: " + filteredAccelValue.z);
}

//Filter Accelerometer
Vector3 filterAccelValue(bool smooth)
{
    if (smooth)
        lowPassValue = Vector3.Lerp(lowPassValue, Input.acceleration, LowPassFilterFactor);
    else
        lowPassValue = Input.acceleration;

    return lowPassValue;
}

Ler Sensor do Acelerômetro (Precisão) #

Leia o sensor do acelerômetro com precisão.

Este exemplo aloca memória:

void Update()
{
    //Get Precise Accelerometer values 
    Vector3 accelValue = preciseAccelValue();
    Debug.Log("PRECISE X: " + accelValue.x + "  Y: " + accelValue.y + "  Z: " + accelValue.z);

}

Vector3 preciseAccelValue()
{
    Vector3 accelResult = Vector3.zero;
    foreach (AccelerationEvent tempAccelEvent in Input.accelerationEvents)
    {
        accelResult = accelResult + (tempAccelEvent.acceleration * tempAccelEvent.deltaTime);
    }
    return accelResult;
}

Este exemplo não aloca memória:

void Update()
{
    //Get Precise Accelerometer values 
    Vector3 accelValue = preciseAccelValue();
    Debug.Log("PRECISE X: " + accelValue.x + "  Y: " + accelValue.y + "  Z: " + accelValue.z);

}

Vector3 preciseAccelValue()
{
    Vector3 accelResult = Vector3.zero;
    for (int i = 0; i < Input.accelerationEventCount; ++i)
    {
        AccelerationEvent tempAccelEvent = Input.GetAccelerationEvent(i);
        accelResult = accelResult + (tempAccelEvent.acceleration * tempAccelEvent.deltaTime);
    }
    return accelResult;
}

Observe que isso não é filtrado. Por favor, veja aqui como suavizar os valores do acelerômetro para remover o ruído.
