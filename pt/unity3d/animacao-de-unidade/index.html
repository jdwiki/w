Animação Sprite 2D #

A animação Sprite consiste em mostrar uma sequência existente de imagens ou frames.

Primeiro importe uma sequência de imagens para a pasta de ativos. Crie algumas imagens do zero ou baixe algumas da Asset Store. (Este exemplo usa este recurso gratuito.)

Arraste cada imagem individual de uma única animação da pasta de ativos para a visualização da cena. O Unity mostrará uma caixa de diálogo para nomear o novo clipe de animação.

Este é um atalho útil para:

  • criando novos objetos de jogo
  • atribuindo dois componentes (um Sprite Renderer e um Animator)
  • criar controladores de animação (e vincular o novo componente Animator a eles)
  • criando clipes de animação com os quadros selecionados

Visualize a reprodução na guia de animação clicando em Reproduzir:

O mesmo método pode ser usado para criar novas animações para o mesmo objeto de jogo e, em seguida, excluir o novo objeto de jogo e o controlador de animação. Adicione o novo clipe de animação ao controlador de animação desse objeto da mesma maneira que na animação 3D.

Animação básica para corrida #

Este código mostra um exemplo simples de animação no Unity.

Para este exemplo, você deve ter 2 clipes de animação; Correr e ocioso. Essas animações devem ser movimentos Stand-In-Place. Depois que os clipes de animação forem selecionados, crie um Animator Controller. Adicione este controlador ao jogador ou objeto de jogo que você deseja animar.

Abra a janela Animator a partir da opção Windows. Arraste os 2 clipes de animação para a janela Animator e 2 estados serão criados. Uma vez criado, use o esquerda para adicionar 2 parâmetros, ambos como bool. Nomeie um como “PerformRun” e outro como “PerformIdle”. Defina “PerformIdle” como verdadeiro.

Faça as transições do estado Idle para Run e Run para idle (consulte a imagem). Clique em Idle->Run transition e na janela Inspector, desmarque HasExit. Faça o mesmo para o outro transição. Para a transição Idle->Run, adicione uma condição: PerformIdle. Para Run->Idle, adicione uma condição: PerformRun. Adicione o script C# fornecido abaixo ao objeto do jogo. Ele deve ser executado com animação usando o botão Para cima e girar com os botões Esquerdo e Direito.

using UnityEngine;
using System.Collections;

public class RootMotion : MonoBehaviour {

//Public Variables
[Header("Transform Variables")]
public float RunSpeed = 0.1f;
public float TurnSpeed = 6.0f;


Animator animator;

void Start()
{
    /**
    * Initialize the animator that is attached on the current game object i.e. on which you will attach this script.
    */
    animator = GetComponent<Animator>();
}

void Update()
{
    /**
    * The Update() function will get the bool parameters from the animator state machine and set the values provided by the user.
    * Here, I have only added animation for Run and Idle. When the Up key is pressed, Run animation is played. When we let go, Idle is played.
    */
    
    if (Input.GetKey (KeyCode.UpArrow)) {
            animator.SetBool ("PerformRun", true);
            animator.SetBool ("PerformIdle", false);
        } else {
            animator.SetBool ("PerformRun", false);
            animator.SetBool ("PerformIdle", true);
        }
}

void OnAnimatorMove()
    {
        /**
         * OnAnimatorMove() function will shadow the "Apply Root Motion" on the animator. Your game objects psoition will now be determined 
         * using this fucntion.
         */
        if (Input.GetKey (KeyCode.UpArrow)){
            transform.Translate (Vector3.forward * RunSpeed);
            if (Input.GetKey (KeyCode.RightArrow)) {
                transform.Rotate (Vector3.up * Time.deltaTime * TurnSpeed);
            }
            else if (Input.GetKey (KeyCode.LeftArrow)) {
                transform.Rotate (-Vector3.up * Time.deltaTime * TurnSpeed);
            }
        }

    }


   
}

Criando e usando clipes de animação #

Este exemplo mostrará como criar e usar clipes de animação para objetos ou jogadores do jogo.

Observe que os modelos usados ​​neste exemplo são baixados da Unity Asset Store. O player foi baixado no seguinte link: https://www.assetstore.unity3d.com/en/#!/content/21874.

Para criar animações, primeiro abra a Janela de Animação. Você pode abri-lo clicando em Window and Select Animation ou pressionando Ctrl+6. O selecione o objeto do jogo para o qual você deseja aplicar o clipe de animação, a partir da Janela de Hierarquia, e então clique no botão Criar na Janela de Animação.

Nomeie sua animação (como IdlePlayer, SprintPlayer, DyingPlayer etc.) e salve-a. Agora, na janela Animation, clique no botão Add Property. Isso permitirá que você altere a propriedade do objeto do jogo ou jogador em relação ao tempo. Isso pode incluir Transformações como rotação, posição e escala e qualquer outra propriedade anexada ao objeto do jogo, por exemplo. Colisor, renderizador de malha etc.

Para criar uma animação em execução para o objeto do jogo, você precisará de um modelo 3D humanóide. Você pode baixar o modelo no link acima. Siga os passos acima para criar uma nova animação. Adicione uma propriedade Transform e selecione Rotation para uma das pernas do caractere.

Neste momento, o botão Play e os valores de rotação na propriedade do objeto do jogo teriam ficado vermelhos. Clique na seta suspensa para ver os valores de rotação X, Y e Z. O tempo de animação padrão é definido como 1 segundo. Animações usam quadros-chave para interpolar entre valores. Para animar, adicione chaves em diferentes pontos no tempo e altere os valores de rotação na janela Inspetor. Por exemplo o valor de rotação em tempo 0,0s pode ser 0,0. No tempo 0,5s o valor pode ser 20,0 para X. No tempo 1,0s o valor pode ser 0,0. Podemos terminar nossa animação em 1,0s.

A duração da sua animação depende das últimas chaves que você adiciona à animação. Você pode adicionar mais chaves para tornar a interpolação mais suave.

Curvas de Animação #

As curvas de animação permitem alterar um parâmetro de flutuação à medida que a animação é reproduzida. Por exemplo, se houver uma animação de duração de 60 segundos e você deseja que um valor/parâmetro flutuante, chame-o de X, varie pela animação (como no tempo da animação = 0,0s; X = 0,0 , em tempo de animação = 30,0s; X = 1,0, em tempo de animação = 60,0s; X = 0,0).

Depois de ter o valor flutuante, você pode usá-lo para traduzir, girar, dimensionar ou usá-lo de qualquer outra maneira.

Para o meu exemplo, mostrarei um objeto de jogo do jogador em execução. Quando a animação de execução é reproduzida, a velocidade de tradução do jogador deve aumentar à medida que a animação avança. Quando a animação chega ao fim, a velocidade de tradução deve diminuir.

Eu tenho um clipe de animação em execução criado. Selecione o clipe e, na janela do inspetor, clique em Editar.

Uma vez lá, role para baixo até Curvas. Clique no sinal + para adicionar uma curva. Nomeie a curva, por exemplo ForwardRunCurve. Clique na curva em miniatura em o certo. Ele abrirá uma pequena janela com uma curva padrão.

Queremos uma curva em forma parabólica onde ela sobe e depois desce. Por padrão, existem 2 pontos na linha. Você pode adicionar mais pontos clicando duas vezes na curva. Arraste os pontos para criar uma forma semelhante à seguinte.

Na janela Animator, adicione o clipe em execução. Além disso, adicione um parâmetro float com o mesmo nome da curva, ou seja, ForwardRunCurve.

Quando a Animação é reproduzida, o valor do float mudará de acordo com a curva. O código a seguir mostrará como usar o valor float:

using UnityEngine;
using System.Collections;

public class RunAnimation : MonoBehaviour {

Animator animator;
float curveValue;

void Start()
{
    animator = GetComponent<Animator>();
}

void Update()
{
    curveValue = animator.GetFloat("ForwardRunCurve");

    transform.Translate (Vector3.forward * curveValue);
}

}

A variável curveValue mantém o valor da curva (ForwardRunCruve) a qualquer momento. Estamos usando esse valor para alterar a velocidade da tradução. Você pode anexar este script ao objeto do jogo do jogador.
