Referência de sintaxe #

Você pode usar @Value("#{expression}") para injetar valor em tempo de execução, no qual a expression é uma expressão SpEL.

Expressões literais #

Os tipos com suporte incluem strings, datas, valores numéricos (int, real e hex), boolean e null.

"#{'Hello World'}"  //strings
"#{3.1415926}"      //numeric values (double)
"#{true}"           //boolean
"#{null}"           //null

Lista em linha #

"#{1,2,3,4}"              //list of number
"#{{'a','b'},{'x','y'}}"  //list of list

Mapas Inline #

"#{name:'Nikola',dob:'10-July-1856'}" 
"#{name:{first:'Nikola',last:'Tesla'},dob:{day:10,month:'July',year:1856}}" //map of maps

Métodos de Invocação #

"#{'abc'.length()}"      //evaluates to 3
"#{f('hello')}"   //f is a method in the class to which this expression belongs, it has a string parameter
