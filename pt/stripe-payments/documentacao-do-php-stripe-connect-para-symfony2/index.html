Parâmetros #

Parâmetro Detalhes
montante obrigatório - Um número inteiro positivo na menor unidade monetária (por exemplo, 100 centavos para cobrar $ 1,00 ou 100 para cobrar ¥ 100, uma moeda 0 decimal) representando quanto cobrar do cartão. O valor mínimo é de US$ 0,50 ou equivalente na moeda de cobrança.
moeda obrigatório - código ISO de 3 letras para moeda.
descrição opcional, o padrão é nulo - Uma string arbitrária que você pode anexar a um objeto de cobrança. Ele é exibido quando na interface da web ao lado da carga. Observe que, se você usar o Stripe para enviar recibos de e-mail automáticos para seus clientes, seus e-mails de recibo incluirão a descrição da(s) cobrança(s) que eles estão descrevendo.
recibo_email o padrão opcional é null - O endereço de e-mail para o qual enviar o recibo desta cobrança. O recibo não será enviado até que a cobrança seja paga. Se esta cobrança for para um cliente, o endereço de e-mail especificado aqui substituirá o endereço de e-mail do cliente. Não serão enviados recibos para cobranças do modo de teste. Se o recibo_email for especificado para uma cobrança no modo ao vivo, um recibo será enviado independentemente do seu
exp_mês obrigatório - Número de dois dígitos que representa o mês de vencimento do cartão.
exp_ano obrigatório - Número de dois ou quatro dígitos que representa o ano de validade do cartão.
número obrigatório - O número do cartão, como uma string sem separadores.
cvc geralmente necessário -Código de segurança do cartão. Obrigatório, a menos que sua conta esteja registrada na Austrália, Canadá ou Estados Unidos. Altamente recomendado incluir sempre este valor.

Symfony2- Exemplo de Integração com Stripe #

Faça o download da Biblioteca da API Stripe e coloque-a na pasta do fornecedor

fonte: https://github.com/stripe/stripe-php

inclua a biblioteca em seu controlador

use Stripe\BalanceTransaction;
use Stripe\Charge;
use Stripe\Stripe;
require_once('../vendor/stripe/init.php');

definir a chave de tira

 \Stripe\Stripe::setApiKey('stripe_secret_key');

Chame a função de cobrança para transação

$card = array(
    'number' =>'cardccn',
    'cvc' =>'cardcvc',
    'exp_month' => 'expMonth',
    'exp_year' => 'expYear',
    );
    
$charge = Charge::create(
    array(
        'amount' => ('amount') * 100, // Amount will store in cent in Stripe Account
        'currency' => 'usd',
        'card' => $card,
        'description' => '$data['description',
        'receipt_email'=>'receipt_email'
        )
    );

obter os detalhes da carga

 $data = Charge::retrieve('ch_%');
