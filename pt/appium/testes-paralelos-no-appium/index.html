Processo passo a passo #

Testes paralelos com Appium usando GRID: Vou descrever a maneira que funcionou para mim. Criar grade de selênio com Appium

  1. Configure a grade de selênio Faça o download do jar do servidor autônomo do selênio no sistema de arquivos local Abra seu terminal e navegue até o diretório onde você colocou o arquivo jar e execute o seguinte comando:

    java -jar selenium-server-standalone-2.53.3.jar -role hub Open http://localhost:4444/grid/console and you should be able to see GRID console in your browser.

  2. Configure os nós do Appium Aqui você deve criar os arquivos json. Suponha que você queira executar em dois dispositivos e crie dois arquivos json diferentes. Aqui está um arquivo json, eu tenho como: { “capacidades”: [ { “applicationName”: “ONEPLUS A3003”, “browserName”: “ONEPLUS A3003”, “platformName”: “ANDROID”, “maxInstances”: 1 } ], “configuração”: { “cleanUpCycle”: 2000, “tempo limite”: 30000, “proxy”: “org.openqa.grid.selenium.proxy.DefaultRemoteProxy”, “host”: “127.0.0.1”, “porta”: 4723, “maxSession”: 1, “registrar”: verdadeiro, “registrarCiclo”: 5000, “hubPort”: 4444, “hubHost”: “seu endereço IP” } } salve o arquivo acima como jasonFile1.json Aqui applicationName será -> Seu celular-> configurações-> sobre o telefone-> Número do modelo Aqui hubHost será seu endereço IP Aqui, observe que você precisa ir como local do cmd padrão e execute o comando abaixo

appium –nodeconfig C:/richa/jasonfile1.json -p 4723 -bp 4724 -U xxxx

i) Observe que você precisa fornecer o passado absoluto do arquivo json localizado ii) portar como 4723 iii) porta Bootstrap como 4724 iv) -U por exemplo, eu dei como xxxx

você pode encontrar o ID do dispositivo como -> Seu celular->configurações->status->Número de série Você também pode fazer “adb device” e verificar este id de dispositivo.

Em seguida, ele criará o Selenium Grid com um dispositivo.

Agora execute novamente o segundo arquivo json e você iniciará o appium Aqui está o segundo arquivo json:

{ “capacidades”: [ { “applicationName”: “Lenovo K50a40”, “browserName”: “Lenovo K50a40”, “platformName”: “ANDROID”, “maxInstances”: 1 } ], “configuração”: { “cleanUpCycle”: 2000, “tempo limite”: 30000, “proxy”: “org.openqa.grid.selenium.proxy.DefaultRemoteProxy”, “host”: “127.0.0.1”, “porta”: 4730, “maxSession”: 1, “registrar”: verdadeiro, “registrarCiclo”: 5000, “hubPort”: 4444, “hubHost”: “seu endereço IP” } } salve o arquivo acima como jasonFile2.json

Inicie o segundo nó com o celular Lenovo. appium –nodeconfig C:/richa/ jasonFile2.json -p 4730 -bp 4731 -U xxxx

Selenium Grid ficará assim
