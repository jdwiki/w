Neste tópico, você aprenderá como criar, editar, copiar, mover e excluir arquivos em lote.

Criando um arquivo em lote #

Pode haver vários motivos pelos quais você deseja criar um arquivo de texto em lote. Mas seja qual for o motivo, é assim que você faz.

Se você quiser sobrescrever um arquivo de texto existente, use >. Exemplo:

@echo off
echo info to save > text.txt

Mas se você quiser anexar texto a um arquivo de texto já existente, use >>. Exemplo:

@echo off
echo info to save >> text.txt

Se você precisar salvar várias linhas de texto em um arquivo, use ()>text.txt Exemplo:

@echo off
(echo username
 echo password)>text.txt

Como copiar arquivos em lote #

Você pode querer copiar arquivos de um lugar para outro. Neste exemplo vamos te ensinar.

Você pode usar o comando xcopy. A sintaxe é xcopy c:\From C:\To

Exemplo:

@echo off
xcopy C:\Folder\text.txt C:\User\Username\Desktop

Há também interruptores que você pode usar. Se você quiser visualizá-los, abra o prompt de comando em Menu Iniciar -> Acessórios -> Prompt de Comando e digite xcopy /?

Movendo arquivos #

Usando o comando move, você pode mover arquivos:

@echo off
cd C:\Foo\Bat\Baz
move /Y Meow.bat "Meow Folder" >nul

Meow.bat significa qual arquivo mover. A “Pasta Meow” move o Meow.bat para a Pasta Meow. /Y diz para não solicitar confirmação. Substituir isso por /-Y faz com que o arquivo de lote solicite confirmação. O >nul oculta a saída do comando. Se não tivesse >nul, seria gerado:

    1 File Moved

Excluindo arquivos #

Usando o comando DEL(alias para ERASE), pode-se remover arquivos.

@echo off
del foo.ext

Este comando excluirá foo.ext do diretório atual. Também é possível especificar o caminho e o arquivo, como:

del C:\Foo\Bar\Baz.ext

Mas é sempre ideal colocar aspas (") nos caminhos, veja aqui para o motivo.

Existem alguns sinalizadores disponíveis para DEL.

Bandeira Função
/P Avisa o usuário antes de excluir o(s) arquivo(s)
/F Remova com força o(s) arquivo(s) somente leitura
/S Remover arquivo(s) em subdiretórios
/Q Impede o prompt do usuário
/A Filtro: Remova apenas o arquivo atribuído específico,
usar o caractere - significa não atribuído como esse tipo.

Copiar arquivos sem xcopy #

Neste exemplo, o usuário BoeNoe mostrou como usar o comando xcopy para copiar arquivos. Há também um comando extra chamado copy.

Aqui está um exemplo simples:

copy foo.ext bar.ext

Isto copia foo.ext para bar.ext, e cria bar.ext quando não existe. Também podemos especificar caminhos para o arquivo, mas é sempre ideal colocar aspas (") em torno dos caminhos, veja aqui para o motivo.

Há também muitos sinalizadores disponíveis para copy, veja copy /? ou help copy em um prompt de comando para ver mais.

Editando a enésima linha de um arquivo #

O arquivo de lote não vem com um método embutido para substituir nésima linha de um arquivo, exceto replace e append(> e >>). Usando loops for, podemos emular esse tipo de função.

@echo off
set file=new2.txt

call :replaceLine "%file%" 3 "stringResult"

type "%file%"
pause
exit /b

:replaceLine <fileName> <changeLine> <stringResult>
setlocal enableDelayedExpansion

set /a lineCount=%~2-1

for /f %%G in (%~1) do (
    if !lineCount! equ 0 pause & goto :changeLine
    echo %%G>>temp.txt
    set /a lineCount-=1
)

:changeLine
echo %~3>>temp.txt

for /f "skip=%~2" %%G in (%~1) do (
    echo %%G>>temp.txt
)

type temp.txt>%~1
del /f /q temp.txt

endlocal
exit /b

  • O script principal chama a função replaceLine, com o nome do arquivo/ qual linha alterar/ e a string a substituir.

  • Função recebe a entrada

  • Faz um loop por todas as linhas e as echo para um arquivo temporário antes da linha de substituição

  • Ele “ecoa” a linha de substituição para o arquivo

  • Continua a saída para o resto do arquivo

  • Copia o arquivo temporário para o arquivo original

  • E remove o arquivo temporário.

  • O script principal recupera o controle e digita o resultado.
