Sintaxe #

  • for /l %%p in (startNumber, increment, endNumber) do comando
  • for /f %%p em (nome do arquivo) do comando
  • for /f %%p em (“textStrings”) do comando
  • for /f %%p em (‘comando’) do comando
  • for /r drive:\path %%p in (set) do comando
  • para /d %%p no (diretório) do comando

O comando for aceita opções quando o sinalizador /f é usado. Aqui está uma lista de opções que podem ser usadas:

  • delims=x Caracter(es) delimitador(es) para separar tokens

  • skip=n Número de linhas a serem ignoradas no início do arquivo e das strings de texto

  • eol=; Caractere no início de cada linha para indicar um comentário

  • tokens=n Itens numerados para ler de cada linha ou string para processar

  • usebackq Use outro estilo de citação:

    Use double quotes for long file names in “files”

    Use single quotes for ’textStrings’

    Use back quotes for `command`

Visitar diretórios recursivamente em uma árvore de diretórios #

O comando for /r pode ser usado para visitar recursivamente todos os diretórios em uma árvore de diretórios e executar um comando.

@echo off
rem start at the top of the tree to visit and loop though each directory
for /r %%a in (.) do (
  rem enter the directory
  pushd %%a
  echo In directory:
  cd
  rem leave the directory
  popd
  )

Notas:

  • for /r - Percorrer arquivos (subpastas recursivas).
  • pushd - Altere o diretório/pasta atual e armazene a pasta/caminho anterior para uso pelo comando POPD.
  • popd - Altere o diretório de volta para o caminho/pasta armazenado mais recentemente pelo comando PUSHD.

Percorrendo cada linha em um conjunto de arquivos #

O seguinte irá ecoar cada linha no arquivo C:\scripts\testFile.txt. Linhas em branco não serão processadas.

for /F "tokens=*" %%A in (C:\scripts\testFile.txt) do (
  echo %%A
  rem do other stuff here
  )

Um exemplo mais avançado mostra como derivados no loop FOR de um conjunto de arquivos restritos podem ser usados ​​para redirecionar a execução em lote, enquanto salva o conteúdo pesquisado em um arquivo:

@echo off
setlocal enabledelayedexpansion

for /f %%i in ('dir "%temp%\test*.log" /o:-d /t:w /b') do (
    set "last=%temp%\%%i"
    type !last! | find /n /i "Completed" >nul 2>&1 >> %temp%\Completed.log ^
     && (echo Found in log %%i & goto :end) || (echo Not found in log %%i & set "result=1"))

:: add user tasks code here
if defined result echo Performing user tasks...

:end    
echo All tasks completed
exit /b

Observe quanto tempo as cadeias de comando são divididas em várias linhas de código e os grupos de comando são separados por parênteses.

Renomeando todos os arquivos no diretório atual #

O seguinte usa uma variável com um loop for para renomear um grupo de arquivos.

SetLocal EnableDelayedExpansion

for %%j in (*.*) do (
  set filename=%%~nj
  set filename=!filename:old=new!
  set filename=Prefix !filename!
  set filename=!filename! Suffix
  ren "%%j" "!filename!%%~xj"
  )

Definindo o nome da variável %%j e associando-o a todos os arquivos atuais (*.*), podemos usar a variável em um loop for para representar cada arquivo no diretório atual.

Cada iteração (ou passagem) pelo loop processa um arquivo diferente do grupo definido (que pode igualmente ter sido qualquer grupo, por exemplo, *.jpg ou *.txt).

No primeiro exemplo, substituímos text: a string de texto “old” é substituída pela string de texto “new” (se, mas somente se, o texto “old” estiver presente no nome do arquivo).

No segundo exemplo, adicionamos texto: o texto “Prefixo” é adicionado ao início do nome do arquivo. Isso não é uma substituição. Essa alteração será aplicada a todos os arquivos do grupo.

No terceiro exemplo, novamente adicionamos texto: o texto “Suffix” é adicionado ao final do nome do arquivo. Novamente, isso não é uma substituição. Essa alteração será aplicada a todos os arquivos do grupo.

A linha final realmente lida com a renomeação.

Iteração #

for /L %%A in (1,2,40) do echo %%A

Esta linha irá iterar de 1 a 39, aumentando em 2 a cada vez.

O primeiro parâmetro, 1, é o número inicial.

O segundo parâmetro, 2, é o incremento.

O terceiro parâmetro, 40, é o máximo.
