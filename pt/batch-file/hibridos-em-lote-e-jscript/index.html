JScript é na verdade o superconjunto de Javascript (é a versão 1.8.1 - então alguns recursos mais novos não estão disponíveis), e eles podem ser embutidos em um script batch para estender as funções do script batch. Normalmente, as técnicas de incorporação estão usando as diretivas JScript (não fazem parte do padrão Javascript oficial) para separar o lote e o código JScript. JScript permite trabalhar com objetos Com/ActiveX, bem como com objetos WMI além do Javascript padrão.

JScript incorporado em um arquivo em lote #

Este exemplo a seguir foi criado pelo usuário Michael Dillon de esta resposta.

Considere o seguinte roteiro:

@set @junk=1 /*
@echo off
cscript //nologo //E:jscript %0 %*
goto :eof
*/

//JScript aka Javascript here

Este trecho de script faz:

  • Execute o comando cscript que chama a si mesmo com todos os argumentos fornecidos.

  • Como a parte após @set @junk=1 é comentada (/* e */ são comentários JScript válidos),

  • JScript irá ignorá-los.

  • Nota: Precisamos da parte @set @junk=1 porque o arquivo de lote não reconhece /* como um comando, mas uma instrução set será uma solução alternativa. O JScript reconhecerá /* como um comentário para que o outro arquivo batch não seja executado pelo mecanismo JScript.

Você pode adicionar seu JScript após */ e começar a estender seu script de arquivo em lote!

Executar JScript com arquivos temporários #

Como mencionado aqui, o método antigo para executar outro script é usando arquivos temporários. Simples echo em um arquivo e, em seguida, execute-o (e remova-o opcionalmente).

Aqui está o conceito básico:

@echo off
echo //A JS Comment > TempJS.js
echo //Add your code>>TempJS.js

cscript //nologo //e:cscript.exe TempJS.js

del /f /s /q TempJS.js

Mas isso exigiria muitas instruções echo para criar um JScript relativamente grande. Aqui está um método melhor por Aacini.

@echo off
setlocal

rem Get the number of the "<resource>" line
for /F "delims=:" %%a in ('findstr /N "<resource>" "%~F0"') do set "start=%%a"

rem Skip such number of lines and show the rest of this file
(for /F "usebackq skip=%start% delims=" %%a in ("%~F0") do echo %%a) > TempJS.js

cscript //nologo //e:cscript.txt TempJS.js
del /f /s /q TempJS.js

goto :EOF

<resource>
