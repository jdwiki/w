Este tópico ensinará a você uma das muitas coisas úteis a serem conhecidas na linguagem de script, arquivo em lote; Adicionando um atraso/pausa/tempo limite ao seu arquivo de lote.

Tempo esgotado #

Tempo esgotado #

A maneira mais simples de fazer um atraso ou pausa por um determinado período de tempo é com o comando padrão TIMEOUT. Para fazer um tempo limite que dure exatamente um minuto, digitamos:

timeout /t 60

Agora o que está acontecendo aqui?

Primeiro usamos o comando TIMEOUT com o parâmetro /T (que simplesmente significa tempo limite) e então especificamos a quantidade de segundos a esperar. Neste caso… 60 segundos.

Tempo limite com o parâmetro /NOBREAK

Se pegarmos o exemplo de antes e executarmos isso em um arquivo BATCH: timeout /t 60, enquanto espera esses 60 segundos, você pode quebrar o tempo limite pressionando qualquer tecla do teclado. Para evitar isso, simplesmente adicionamos o parâmetro /NOBREAK ao final dele.

timeout /t 60 /nobreak

Ao fazer isso, o tempo limite será de 60 segundos e, se você quiser quebrar o tempo limite, precisará pressionar (CTRL-C) no teclado.

Tempo limite silencioso

Quando está fazendo um tempo limite, ele exibirá:

Waiting for X seconds, press a key to continue ...
or 
Waiting for X seconds, press CTRL+C to quit ... [This is with the /NOBREAK parameter]

Para ocultar a mensagem, use o argumento NUL (Para explicação de NUL: Click Here)

timeout /t 60 > nul
or
timeout /t 60 /nobreak > nul

Pausa #

Para fazer seu script pausar simplesmente use o comando PAUSE.

PAUSE

Isso exibirá o texto Pressione qualquer tecla para continuar. . . e, em seguida, adicione uma nova linha na entrada do usuário.

Digamos que queremos criar um programa “Hello World” e depois de clicarmos em algo em nosso teclado, queremos que ele saia do programa com o comando EXIT.

echo Hello World
pause
exit

Aqui ele usa o comando [ECHO] 1 para dizer “Hello World”. Em seguida, usamos o comando PAUSE que exibe Pressione qualquer tecla para continuar. . . e então usamos o comando EXIT para encerrar o script BATCH atual.

Quando estiver pausando, ele exibirá:

Press any key to continue . . .

**Oculte o prompt “Pressione qualquer tecla para continuar…”

Para ocultar a mensagem, redirecionamos a saída para um dispositivo especial chamado nul. Este não é realmente um dispositivo real, mas tudo o que enviamos para ele é jogado fora.

pause > nul

Ping #

Ping #

Um dos comandos mais usados ​​para atrasar por um certo tempo é o ping.

Uso básico

PING -n 1 -w 1000 1.1.1.1

REM the -n 1 flag means to send 1 ping request.
REM the -w 1000 means when the IP(1.1.1.1) does not respond, go to the next command
REM 1.1.1.1 is an non-existing IP so the -w flag can ping a delay and go to next command

Isso produziria o seguinte em seu arquivo de lote/console:

C:\Foo\Bar\Baz>ping -n -w 1000 1.1.1.1

Pinging 1.1.1.1 (Using 32 bytes of data)
Request timed out

Ping statistics for 1.1.1.1
    Packets: Sent = 2,Received = 0, Lost = 1(100% loss)

Ocultar o texto ecoado

Basta adicionar >nul na parte de trás do comando para redirecioná-lo para null.

ping -n w 1000 1.1.1.1 >nul

Isso não produziria nada.

Dorme #

Dorme #

No sistema Windows mais antigo, o timeout não está disponível. No entanto, podemos usar o comando sleep.

Uso

sleep 1

Muito auto-explicativo; dormir por 1 segundo. No entanto, sleep é um comando desativado e deve ser substituído por timeout.

Disponibilidade

Este comando está disponível no sistema Windows antigo. O SLEEP.exe também está incluído no 2003 Resource Kit.

Para usar sleep.exe, coloque o arquivo executável na pasta %Windir%\System32. Então você pode usá-lo como comando normal.
