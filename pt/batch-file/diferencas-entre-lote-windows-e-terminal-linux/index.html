Diferenças entre Lote (Windows) e Terminal (Linux)

  • Comandos em lote e seus equivalentes Bash
  • Variáveis ​​de lote e seus equivalentes de Bash

Batch e bash são bem diferentes. Os sinalizadores de lote são indicados com um /, enquanto os sinalizadores de bash usam um -. A capitalização é importante no bash, mas (quase) não no batch. Nomes de variáveis ​​de lote podem conter espaços, nomes de variáveis ​​de bash não.

Em última análise, ambas são formas de manipular e interagir com a linha de comando. Não surpreendentemente, há uma quantidade razoável de sobreposição entre os recursos dos dois sistemas.

  • bitsadmin está obsoleto em favor do cmdlet BITS do PowerShell, mas ainda funciona a partir do Windows 10 versão 1607
  • certutil separa pares de dígitos hexadecimais com um espaço, então md5sum retornará um valor de exemplo de d41d8cd98f00b204e9800998ecf8427e, enquanto certutil exibe o mesmo valor que d4 1d 8c d9 8f 00 b2 04 e9 80 09 98 ec f8 42 7e
  • Para cd para outra unidade (por exemplo, de C: para D:) o sinalizador /d deve ser usado
  • del não pode deletar pastas, use rm ao invés
  • grep é muito mais poderoso que find e findstr, é quase injusto compará-los; find não tem recursos regex e findstr tem recursos regex extremamente limitados ([a-z]{2} não é uma sintaxe válida, mas [a-z][a-z] é)
  • Loops for no prompt de comando do Windows só podem usar nomes de variáveis ​​de caractere único; esta é a única vez que os nomes das variáveis ​​de lote diferenciam maiúsculas de minúsculas
  • loops for no prompt de comando também usam a forma variável %A em vez de %A% - forloops em scripts em lote usam a forma variável %%A
  • md cria automaticamente qualquer diretório pai necessário, enquanto mkdir precisa do sinalizador -p para fazer isso
  • rem não pode ser usado como um caractere de comentário embutido, a menos que seja precedido por um &
  • :: não pode ser usado como um comentário embutido e também não deve ser usado dentro de um bloco de código (conjunto de parênteses)
  • Observe que alguns comandos do Windows como ping ainda usam - como sinalizadores

Comandos em lote e seus equivalentes Bash #

Lote Bash Descrição
command /? man command Mostra a ajuda para command
bitsadmin wget ou curl Baixa um arquivo remoto
certutil -hashfile file_name MD5 md5sum file_name Obtém a soma de verificação MD5 de file_name
cd pwd Exibe o diretório atual
diretório cd diretório cd Altera o diretório atual para o especificado
cls clear Limpa a tela
copy cp Copia um arquivo ou arquivos de um caminho de origem para um caminho de destino
date date Exibe a data ou a define com base na entrada do usuário
del rm Exclui um arquivo ou arquivos
dir ls exibe uma lista de arquivos e diretórios no diretório atual
echo echo Exibe texto na tela
exit return Sai de um script ou sub-rotina
exit logout Fecha o prompt de comando ou terminal
fc diff Compara o conteúdo de dois arquivos
find "string" file_name grep "string" file_name Procura file_name por string
findstr "string" file_name grep "string" file_name Procura file_name por string
for /F %A em (fileset*) faça algo for item in fileset*; Faz; algo; done Faça algo para cada arquivo em um conjunto de arquivos
for /F %A in ('command') faça algo `command` Retorna a saída de um comando
para /L %A em (primeiro,incremento,último) faça algo para item em `seq primeiro incremento último`; Faz; algo; done Começa em first e conta por increment até atingir last
forfiles find Procura por arquivos que correspondem a um determinado critério
if "%variable%"=="valor" ( if [ "variable"="valor" ]; then Compara dois valores
ipconfig ifconfig Exibe informações de IP
md mkdir Cria novas pastas
mklink ln -s Cria um link simbólico
more more Exibe uma tela de saída por vez
move mv Move um arquivo ou arquivos de um caminho de origem para um caminho de destino
pause read -p "Pressione qualquer tecla para continuar" Pausa a execução do script até que o usuário pressione um botão
popd popd Remove a entrada superior da pilha de diretórios e vai para o novo diretório superior
pushd pushd Adiciona o diretório atual à pilha de diretórios e vai para o novo diretório superior
ren mv Renomeia arquivos
rem ou :: # Comenta uma linha de código
rd rmdir Remove diretórios vazios
rd /s rm -rf Remove diretórios independentemente de estarem ou não vazios
set variable=value variable=value Define o valor de variable para value
set /a variable=equation variable=$((equation)) Executa matemática (o lote só pode usar inteiros de 32 bits)
set /p variável=promptstring read -p variável "promptstring" Obtém a entrada do usuário e a armazena em variable
shift shift Desloca os argumentos em 1 (ou n se fornecido)
sort sort Classifica a saída alfabeticamente por linha
tasklist ps Mostra uma lista de processos em execução
taskkill /PID processid kill processid Mata o processo com PID processid
time /t date Exibe a hora atual
type cat Exibe o conteúdo de um arquivo
where which Pesquisa o diretório atual e o PATH por um arquivo ou comando
whoami id Exibe o nome e grupo do usuário atual

Variáveis ​​de lote e seus equivalentes de Bash #

Lote Bash Descrição
%variable% $variable Uma variável regular
!variable! $variable Uma variável dentro de um bloco de código quando setlocal enabledelayedexpansion está ativado
%errorlevel% ou ERRORLEVEL $? Retorna o status do comando anterior: 0 se for bem sucedido, 1 (ou algo mais) se não
%1, %2, %3, etc. $1, $2, $3, etc. Os parâmetros dados a um script
%* $* Todos os parâmetros dados a um script
