  2. Lista dos sistemas operacionais suportados pelo IDE.
  3. Processo de instalação.
  4. Etapas de configuração adicionais (se houver).

SW4STM32: Bancada de trabalho do sistema para STM32 #

Introdução #

O System Workbench para STM32 é um IDE gratuito para Windows, Linux e OS X. Descrição de ST Microelectronics:

A cadeia de ferramentas do System Workbench, chamada SW4STM32, é um multi-SO gratuito ambiente de desenvolvimento de software baseado em Eclipse, que suporta a gama completa de microcontroladores STM32 e placas associadas.

A cadeia de ferramentas SW4STM32 pode ser obtida no site www.openstm32.org, que inclui fóruns, blogs e treinamentos para suporte técnico. Uma vez registrado neste site, os usuários receberão instruções de instalação na página Documentação > System Workbench para prosseguir com o download do conjunto de ferramentas gratuito.

A cadeia de ferramentas System Workbench e seu site colaborativo foram construído pela AC6, uma empresa de serviços que oferece treinamento e consultoria em sistemas embarcados.

Este produto é fornecido por terceiros não afiliados à ST. Por as informações mais recentes sobre a especificação, consulte o terceiro site do partido: www.ac6.fr.

Principais recursos

  • Suporte abrangente para microcontroladores STM32, placas Nucleo STM32, kits Discovery e placas de avaliação, bem como STM32 firmware (biblioteca de periféricos padrão ou STM32Cube HAL)
  • Compilador GCC C/C++
  • Depurador baseado em GDB
  • Eclipse IDE com gerenciamento de trabalho em equipe
  • Compatível com plug-ins Eclipse
  • Suporte ST-LINK
  • Sem limite de tamanho de código
  • Suporte a vários sistemas operacionais: Windows®, Linux e OS X®

Instalação #

  1. Acesse: http://www.openstm32.org/HomePage .
  2. Registre-se e faça login no site.
  3. Navegue até: http://www.openstm32.org/Downloading+the+System+Workbench+for+STM32+installer .
  4. Baixe a versão mais recente para seu sistema operacional.
  5. Execute o instalador baixado.

IAR-EWARM #

Introdução #

IAR Ambiente de desenvolvimento integrado e otimizando o compilador C/C++ para ARM Cortex-M. Descrição de ST Microelectronics:

O IAR-EWARM é uma suíte de desenvolvimento de software entregue com arquivos de configuração de dispositivos prontos, carregadores flash e exemplo 4300 projetos incluídos. O IAR Embedded Workbench é compatível com outros Compiladores compatíveis com ARM®EABI e suporta os seguintes ARM®cores para STM32:

Principais recursos

  1. Componentes principais:
  • Ambiente de desenvolvimento integrado com ferramentas de gerenciamento de projetos e editor
  • Compilador C e C++ altamente otimizado para ARM®
  • Verificação automática das regras MISRA C (MISRA C:2004)
  • Conformidade com ARM® EABI e CMSIS
  • Amplo suporte ao sistema de destino HW
  • Sondas de depuração em circuito I-jet e JTAGjet™-Trace opcionais
  • Depuração de energia para visualizar o consumo de energia em correlação com o código-fonte
  • Bibliotecas de tempo de execução, incluindo código-fonte
  • Realocação do montador ARM®
  • Ferramentas de linker e bibliotecário
  • Depurador C-SPY® com simulador ARM®, suporte JTAG e suporte para depuração com reconhecimento de RTOS em hardware
  • Plugins RTOS disponíveis nos fornecedores IAR Systems e RTOS
  • Mais de 3100 projetos de amostra para placas de avaliação de diversos fabricantes
  • Guias do usuário e de referência em formato PDF
  • Ajuda on-line sensível ao contexto
  1. Suporte específico para chip:
  • 4300 projetos de exemplo incluindo para placas de avaliação STMicroelectronics
  • Suporte para aplicativos de 4 Gbyte em modo ARM® e Thumb®
  • Cada função pode ser compilada no modo ARM® ou Thumb®
  • Geração de código do coprocessador VFP Vector Floating Point
  • Suporte intrínseco NEON™
  1. Suporte para depuração de hardware:
  • STMicroelectronics ST-LINK V2 : Suporta dispositivos STM32
  • STMicroelectronics ST-LINK : Suporta dispositivos STM32
  1. Suporte RTOS: consulte o site da IAR http://www.iar.com
  2. Dispositivos suportados: consulte o site da IAR http://www.iar.com

Atollic - TrueSTUDIO #

Introdução #

IDE C/C++ para desenvolvimento ARM.

O Atollic TrueSTUDIO® é testado e verificado nos seguintes sistemas operacionais:

  • Microsoft® Windows®Vista (versão de 32 bits)
  • Microsoft® Windows® Vista (versão de 64 bits)
  • Microsoft® Windows® 7 (versão de 32 bits)
  • Microsoft® Windows® 7 (versão de 64 bits)
  • Microsoft® Windows® 8 (versão de 64 bits)
  • Microsoft® Windows® 10 (versão de 64 bits)
  • Suporte ao Linux esperado para o final de 2016 Q4
  • Suporte para Mac OS X esperado para o segundo trimestre de 2017

O TrueSTUDIO está disponível apenas como um aplicativo de 32 bits.

Instalação #

O produto Atollic TrueSTUDIO é fornecido como um executável instalador. Certifique-se de que a conta de usuário, da qual o o instalador é iniciado, tem privilégios administrativos. Não há necessidade para registro ou conexão com a internet durante a instalação. Quando TrueSTUDIO está instalado, ele será executado no modo Lite se não houver licenças detectado.

  1. Acesse: http://atollic.com/resources/downloads/ .
  2. Baixe a versão estável ou beta mais recente.
  3. Execute o instalador.

Código #

Introdução #

CooCox CoIDE, um ambiente de desenvolvimento de software gratuito e altamente integrado para MCUs ARM Cortex. Descrição de ST Microelectronics:

CoIDE é um ambiente de desenvolvimento de software livre baseado em Eclipse e Cadeia de ferramentas GCC, que foi customizada e simplificada para dar aos usuários um acesso fácil aos microcontroladores ARM® Cortex®-M.

Este produto é fornecido por terceiros não afiliados à ST. Por informações completas e atualizadas sobre as especificações e pacotes de as peças adquiridas, consulte o site do terceiro www.coocox.org.

Principais recursos

  • Suporte completo para microcontroladores STM32, placas STM32 Nucleo e bibliotecas de software STM32Cube.
  • Compilador GCC C/C++.
  • Depurador baseado em GDB.
  • IDE Eclipse simplificado.
  • Suporte ST-Link.
  • Suporte multilíngue: inglês, chinês.
