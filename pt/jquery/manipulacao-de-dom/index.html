Criando elementos DOM #

A função jQuery (geralmente conhecida como $) pode ser usada tanto para selecionar elementos quanto para criar novos elementos.

var myLink = $('<a href="http://stackexchange.com"></a>');

Opcionalmente, você pode passar um segundo argumento com atributos de elemento:

var myLink = $('<a>', { 'href': 'http://stackexchange.com' });

'<a>' –> O primeiro argumento especifica o tipo de elemento DOM que você deseja criar. Neste exemplo é uma âncora, mas pode ser qualquer coisa nesta lista. Veja a especificação para uma referência do elemento a.

{ 'href': 'http://stackexchange.com' } –> o segundo argumento é um Objeto JavaScript contendo pares nome/valor do atributo.

os pares ’name’:‘value’ aparecerão entre o < > do primeiro argumento, por exemplo <a name:value> que para nosso exemplo seria <a href="http:// stackexchange.com"></a>

Manipulando classes de elementos #

Supondo que a página inclua um elemento HTML como:

<p class="small-paragraph">
  This is a small <a href="https://en.wikipedia.org/wiki/Paragraph">paragraph</a>
  with a <a class="trusted" href="http://stackexchange.com">link</a> inside.
</p>

jQuery fornece funções úteis para manipular classes DOM, principalmente hasClass(), addClass(), removeClass() e toggleClass(). Essas funções modificam diretamente o atributo class dos elementos correspondentes.

$('p').hasClass('small-paragraph'); // true
$('p').hasClass('large-paragraph'); // false

// Add a class to all links within paragraphs
$('p a').addClass('untrusted-link-in-paragraph');

// Remove the class from a.trusted
$('a.trusted.untrusted-link-in-paragraph')
.removeClass('untrusted-link-in-paragraph')
.addClass('trusted-link-in-paragraph');

Alternar uma classe

Dada a marcação de exemplo, podemos adicionar uma classe com nosso primeiro .toggleClass():

$(".small-paragraph").toggleClass("pretty");

Agora isso retornaria true: $(".small-paragraph").hasClass("pretty")

toggleClass fornece o mesmo efeito com menos código que:

if($(".small-paragraph").hasClass("pretty")){
   $(".small-paragraph").removeClass("pretty");}
else {
   $(".small-paragraph").addClass("pretty"); }

alternar Duas classes:

$(".small-paragraph").toggleClass("pretty cool");

Booleano para adicionar/remover classes:

$(".small-paragraph").toggleClass("pretty",true); //cannot be truthy/falsey

$(".small-paragraph").toggleClass("pretty",false);

Função para alternância de classe (veja o exemplo mais abaixo para evitar um problema)

$( "div.surface" ).toggleClass(function() {
  if ( $( this ).parent().is( ".water" ) ) {
    return "wet";
  } else {
    return "dry";
  }
});

  return num % 2;
}

function stringContains(myString, mySubString) {
  return myString.indexOf(mySubString) !== -1;
}
var showClass = true; //we want to add the class

Adicione uma classe ímpar/par a elementos com uma classe gridrow

$("div.gridrow").toggleClass(function(index, oldClasses, showThisClass) {
  if (isOdd(index)) {
    return "odd";
  } else {
    return "even";
  }
  return oldClasses;
}, showClass);

Se a linha tiver uma classe gridfooter, remova as classes ímpares/pares, mantenha o resto.

$("div.gridrow").toggleClass(function(index, oldClasses, showThisClass) {
  var isFooter = stringContains(oldClasses, "gridfooter");
  if (isFooter) {
    oldClasses = oldClasses.replace('even', ' ').replace('odd', ' ');
    $(this).toggleClass("even odd", false);
  }
  return oldClasses;
}, showClass);

As classes que são retornadas são as que são efetuadas. Aqui, se um elemento não tiver um gridfooter, adicione uma classe para par/ímpar. Este exemplo ilustra o retorno da lista de classes OLD. Se este else return oldClasses; for removido, apenas as novas classes serão adicionadas, portanto, a linha com uma classe gridfooter teria todas as classes removidas se não tivéssemos retornado aquelas antigas - elas teriam sido alternadas (removidas) caso contrário.

$("div.gridrow").toggleClass(function(index, oldClasses, showThisClass) {
  var isFooter = stringContains(oldClasses, "gridfooter");
  if (!isFooter) {
    if (isOdd(index)) {
      return "oddLight";
    } else {
      return "evenLight";
    }
  } else return oldClasses;
}, showClass);

Outros métodos de API #

jQuery oferece uma variedade de métodos que podem ser usados ​​para manipulação de DOM.

O primeiro é o método .empty().

Imagine a seguinte marcação:

<div id="content">
  <div>Some text</div>
</div>

Ao chamar $('#content').empty();, o div interno seria removido. Isso também pode ser feito usando $('#content').html('');.

Outra função útil é a função .closest():

<tr id="row_1">
  <td><button type="button" class="delete">Delete</button>
</tr>

Se você quiser encontrar a linha mais próxima de um botão que foi clicado em uma das células da linha, faça o seguinte:

$('.delete').click(function() {
  $(this).closest('tr');
});

Como provavelmente haverá várias linhas, cada uma com seus próprios botões delete, usamos $(this) dentro da função .click() para limitar o escopo ao botão em que realmente clicamos.

Se você quiser obter o id da linha que contém o botão Delete que você clicou, você poderia algo assim:

$('.delete').click(function() {
  var $row = $(this).closest('tr');
  var id = $row.attr('id');
});

Geralmente é considerado uma boa prática prefixar variáveis ​​contendo objetos jQuery com um $ (sinal de dólar) para deixar claro qual é a variável.

Uma alternativa para .closest() é o método .parents():

$('.delete').click(function() {
  var $row = $(this).parents('tr');
  var id = $row.attr('id');
});

e também há uma função .parent():

$('.delete').click(function() {
  var $row = $(this).parent().parent();
  var id = $row.attr('id');
});

.parent() só sobe um nível da árvore DOM, então é bastante inflexível, se você mudar o botão delete para estar contido em um span por exemplo, então o seletor jQuery seria quebrado.

Classificando elementos #

Para classificar os elementos de forma eficiente (tudo de uma vez e com o mínimo de interrupção do DOM), precisamos:

  1. Encontre os elementos

  2. Classificar com base em uma condição definida

  3. Insira novamente no DOM

    • Red
    • Green
    • Purple
    • Orange

  4. Encontre-os - .children() ou .find()

Isso nos devolverá um objeto do tipo Array para brincar.

   var $myColorList = $('#my-color-list');

   // Elements one layer deep get .children(), any deeper go with .find()
   var $colors = $myColorList.children('li');
  1. Reorganize-os - Array.prototype.sort()

No momento, isso está configurado para retornar os elementos em ordem crescente com base no conteúdo HTML (também conhecido como cores).

   /**
    * Bind $colors to the sort method so we don't have to travel up
    * all these properties more than once.
    */
   var sortList = Array.prototype.sort.bind($colors);

   sortList(function ( a, b ) {

       // Cache inner content from the first element (a) and the next sibling (b)
       var aText = a.innerHTML;
       var bText = b.innerHTML;
    
       // Returning -1 will place element `a` before element `b`
       if ( aText < bText ) {
           return -1;
       }

       // Returning 1 will do the opposite
       if ( aText > bText ) {
           return 1;
       }

       // Returning 0 leaves them as-is
       return 0;
   });

  1. Insira-os - .append()

    Note that we don’t need to detach the elements first - append() will move elements that already exist in the DOM, so we won’t have extra copies

    // Put it right back where we got it
$myColorList.append($colors);

Envolva tudo em uma função doSort() #

// Put the sortList() and detach/append calls in this portable little thing
var doSort = function ( ascending ) {

    sortList(function ( a, b ) {
        // ...
    });

    $myColorList.append($colors);
};

Adicionar manipulador de cliques para $('#btn-sort') #

$('#btn-sort').on('click', function () {
    // Run the sort and pass in the direction value
    doSort(ascending);

    // Toggle direction and save
    ascending = !ascending;
});

Todos juntos agora #

var ascending = true;

var $myColorList = $('#my-color-list');
var $colors = $myColorList.children('li');
var sortList = Array.prototype.sort.bind($colors);

var doSort = function ( ascending ) {
    
    sortList(function ( a, b ) {

        var aText = a.innerHTML;
        var bText = b.innerHTML;

        if ( aText < bText ) {
            return ascending ? -1 : 1;
        }

        if ( aText > bText ) {
            return ascending ? 1 : -1;
        }

        return 0;
    });
    
    $myColorList.append($colors);

};

$('#btn-sort').on('click', function () {
    doSort(ascending);
    ascending = !ascending;
});

Classificação em vários níveis (agrupando elementos classificados) #

// ...

var doSort = function ( ascending ) {

    sortList(function ( a, b ) {

        // ...initial sorting...

    }).sort(function ( a, b ) {
        
        // We take the returned items and sort them once more
        var aClass = a.className;
        var bClass = b.className;
        
        // Let's group the disabled items together and put them at the end

        /**
         * If the two elements being compared have the same class
         * then there's no need to move anything.
         */
        if ( aClass !== bClass ) {
            return aClass === 'disabled' ? 1 : -1;
        }
        return 0;
    });

    // And finalize with re-insert
    $myColorList.append($colors);
};

// ...

.html() #

Você pode usar esse método para substituir todo o HTML no seletor. Supondo que você tenha um elemento html como este

<div class="row">
    <div class="col-md-12">
        <div id="information">
            <p>Old message</p>
        </div>
    </div>
</div>

Você pode usar .html(). para remover e adicionar um alerta ou texto informativo para alertar os usuários com uma linha.

$("#information").html("<p>This is the new alert!</p>");
