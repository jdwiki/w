O problema da mochila surge principalmente nos mecanismos de alocação de recursos. O nome “mochila” foi introduzido pela primeira vez por Tobias Dantzig.

Espaço Auxiliar: O(nw)
Complexidade de tempo O(nw)

Noções básicas do problema da mochila #

O problema: Dado um conjunto de itens em que cada item contém um peso e um valor, determine o número de cada um para incluir em uma coleção para que o peso total é menor ou igual a um determinado limite e o valor total é o maior possível.

Pseudo código para o problema da mochila

Dado:

  1. Valores (matriz v)

  2. Pesos (matriz w)

  3. Número de itens distintos (n)

  4. Capacidade (W)

    for j from 0 to W do: m[0, j] := 0 for i from 1 to n do: for j from 0 to W do: if w[i] > j then: m[i, j] := m[i-1, j] else: m[i, j] := max(m[i-1, j], m[i-1, j-w[i]] + v[i])

Uma implementação simples do pseudocódigo acima usando Python:

def knapSack(W, wt, val, n):
    K = [[0 for x in range(W+1)] for x in range(n+1)]
    for i in range(n+1):
        for w in range(W+1):
            if i==0 or w==0:
                K[i][w] = 0
            elif wt[i-1] <= w:
                K[i][w] = max(val[i-1] + K[i-1][w-wt[i-1]],  K[i-1][w])
            else:
                K[i][w] = K[i-1][w]
    return K[n][W]
val = [60, 100, 120]
wt = [10, 20, 30]
W = 50
n = len(val)
print(knapSack(W, wt, val, n))

Executando o código: Salve isso em um arquivo chamado knapSack.py

$ python knapSack.py
220

Tempo Complexidade do código acima: O(nW) onde n é o número de itens e W é a capacidade da mochila.

Solução implementada em C# #

public class KnapsackProblem
{
    private static int Knapsack(int w, int[] weight, int[] value, int n)
    {
        int i;
        int[,] k = new int[n + 1, w + 1];
        for (i = 0; i <= n; i++)
        {
            int b;
            for (b = 0; b <= w; b++)
            {
                if (i==0 || b==0)
                {
                    k[i, b] = 0;
                }
                else if (weight[i - 1] <= b)
                {
                    k[i, b] = Math.Max(value[i - 1] + k[i - 1, b - weight[i - 1]], k[i - 1, b]);
                }
                else
                {
                    k[i, b] = k[i - 1, b];
                }
            }
        }
        return k[n, w];
    }

    public static int Main(int nItems, int[] weights, int[] values)
    {
        int n = values.Length;
        return Knapsack(nItems, weights, values, n);
    }
}
