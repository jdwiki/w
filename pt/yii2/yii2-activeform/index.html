Campos de formulário no Yii2 #

Mostrando Exemplo Básico da Página de Visualização no Yii2 para Novos Alunos

Estas são as classes básicas que você deve adicionar para criar o formulário usando o yii2 ActiveForm

<?php

Use yii\helpers\Html;
Use yii\widgets\ActiveForm;

A linha abaixo iniciará a tag de formulário para nosso formulário abaixo, mostrando o exemplo de como especificar id para o formulário e como aplicar quaisquer classes para o formulário.

$form =ActiveForm::begin([   'id'=> 'login-form',   'options'=> ['class' => 'form-horizontal'],]) ?>

Aqui $model Especifique qual campo da tabela de banco de dados queremos vincular com o objeto de modelo armazenado aqui nesta variável que foi passada do controlador relevante.

   <?= $form->field($model, 'username') ?>
   <?= $form->field($model, 'password')->passwordInput() ?>

‘username’ e ‘password’ é o nome do campo da tabela ao qual nosso valor será vinculado.

Aqui no código abaixo, estamos colocando o botão de envio para envio de formulário e aplicando ‘Login’ como Texto de botão e classes css básicas a ele.

 <div class="form-group">
       <div class="col-lg-offset-1 col-lg-11">
           <?= Html::submitButton('Login', ['class' => 'btn btn-primary']) ?>
       </div>
   </div>

Aqui no código abaixo estamos fechando o formulário

<?php ActiveForm::end() ?>

Criar campo de senha:

<?= $form->field($model, 'password')->passwordInput() ?>

Criar campo de texto:

<?= $form->field($model, 'username') ?>

Criar campo de formulário oculto :

echo $form->field($model, 'hidden1')->hiddenInput()->label(false);

Criar lista suspensa:

<?php echo $form->field($model, 'name')
->dropdownList(
Stud::find()->select(['name'])
->indexBy('name')->column(),
['prompt'=>'Select no']); ?>

Lista suspensa com ID e nome

<?= $form->field($model, 'name')->dropDownList(
            ArrayHelper::map(Stud::find()->all(), 'no', 'name'),['prompt' => 'Select Car Name']
 ) ?>

Criar FileUploader:

echo $form->field($model, 'imagepath')->fileInput();

Adicionando um marcador de posição e um rótulo personalizado

 <?= $form->field($model, 'username')->textInput()->hint('Please enter your  name')->label('Name') ?>

Validações do ActiveForm #

Você pode ativar/desativar ajax e validações de cliente no formulário ativo.

$form = ActiveForm::begin([
    'id' => 'signup-form',
    'enableClientValidation' => true,
    'enableAjaxValidation' => true,
    'validationUrl' => Url::to('signup'),
]);
  1. enableClientValidation é ativado por padrão no ActiveForm. Se você não precisar de validação do cliente no formulário, podemos desabilitar atribuindo como false.
  2. enableAjaxValidation está desabilitado por padrão no ActiveForm. Se você quiser habilitá-lo, temos que adicionar manualmente no ActiveForm como acima.
  3. validationUrl - se você quiser manter toda a codificação de validação em ação separada do controlador para este formulário, podemos configurar o formulário ativo usando validationUrl. Se não definirmos isso, ele assumirá o valor de ação do formulário.

Os dois argumentos acima afetarão o formulário inteiro. Se você quiser verificar a validação do ajax apenas para um campo específico no formulário, você pode adicionar enableAjaxValidation para esse campo específico. Ele funcionará apenas para esse campo e não para o formulário inteiro.

Por exemplo, no formulário de registro você deseja verificar se o nome de usuário já existe validação no momento do usuário entrar no formulário. você pode usar este argumento enableAjaxValidation para esse campo.

echo $form->field($model, 'username', ['enableAjaxValidation' => true]);
