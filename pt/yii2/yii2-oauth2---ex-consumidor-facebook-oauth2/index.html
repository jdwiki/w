Crie um aplicativo no desenvolvedor do facebook #

Acesse https://developers.facebook.com/ e crie seu aplicativo.

Clique em ‘Adicionar produto’ e escolha ‘Login do Facebook’

Instale o yii2-authclient #

Antes de instalar esta extensão, você deve instalar o yii2-app. Neste exemplo, eu uso o template yii2-basic. Guia para instalação em aqui.

Corre

composer require --prefer-dist yiisoft/yii2-authclient

ou adicione

"yiisoft/yii2-authclient": "~2.1.0"

para a seção require do seu composer.json.

Adicione a configuração authClientCollection à sua configuração components:

return [
    'components' => [
        'authClientCollection' => [
            'class' => 'yii\authclient\Collection',
            'clients' => [
                'facebook' => [
                    'class' => 'yii\authclient\clients\Facebook',
                    'clientId' => 'facebook_client_id',
                    'clientSecret' => 'facebook_client_secret',
                ],
            ],
        ]
    ],
    // ...
];

facebook_client_id é o ID do aplicativo e facebook_client_secret é o segredo do aplicativo.

Adicione a ação de autenticação e configure o retorno de chamada #

  1. Adicione o botão ‘Login como conta do facebook’ à sua visualização de login:

Edite site/login.php na pasta views, adicione essa linha ao conteúdo da página de login:

<?= yii\authclient\widgets\AuthChoice::widget([
     'baseAuthUrl' => ['site/auth'],
     'popupMode' => false,
]) ?>

Acima, definimos que a ação auth em SiteController manipulará o fluxo OAuth2.

Agora nós criamos.

class SiteController extends Controller
{
    public function actions()
    {
        return [
            'auth' => [
                'class' => 'yii\authclient\AuthAction',
                'successCallback' => [$this, 'onAuthSuccess'],
            ],
        ];
    }

    public function onAuthSuccess($client)
    {
        // do many stuff here, save user info to your app database
    }
}

Usamos yii\authclient\AuthAction para criar url e redirecionar para a página de login do facebook.

Função onAuthSuccess usada para obter informações do usuário, faça login no seu aplicativo.

Adicione redirect_url à configuração do aplicativo do Facebook #

Se você habilitar prettyUrl em seu aplicativo yii2, seu redirect_uri será:

http://<base_url>/web/site/auth

E desative o URL bonito:

http://<base_url>/web/index.php?r=site%2Fauth

Exemplo para a função onAuthSuccess #

/**
 * @param $client ClientInterface
 */
public function onAuthSuccess($client)
{
    //Get user info
    /** @var array $attributes */
    $attributes = $client->getUserAttributes();
    $email = ArrayHelper::getValue($attributes, 'email'); //email info
    $id = ArrayHelper::getValue($attributes, 'id'); // id facebook user
    $name = ArrayHelper::getValue($attributes, 'name'); // name facebook account

    //Login user
    //For demo, I will login with admin/admin default account
    $admin = User::findByUsername('admin');
    Yii::$app->user->login($admin);
}
