Instalação ou configuração #

Yii2 pode ser instalado de duas maneiras. Eles são

  1. Instalando via Composer
  2. Instalando a partir de um arquivo morto

Instalando pelo Composer #

Instalando o Composer #

Se você ainda não tem o Composer instalado, pode fazê-lo seguindo as instruções em getcomposer.org. No Linux e no Mac OS X, você executará os seguintes comandos:

curl -sS https://getcomposer.org/installer | php
mv composer.phar /usr/local/bin/composer

Para windows basta baixar e instalar composer-setup.exe Talvez seja necessário configurar o token de acesso da API do Github para substituir o limite de taxa da API do Github.

Instalando o Yii #

Com o Composer instalado, você pode instalar o Yii executando os seguintes comandos em uma pasta acessível pela Web:

composer global require "fxp/composer-asset-plugin:^1.2.0"
composer create-project --prefer-dist yiisoft/yii2-app-basic basic

então execute o seguinte comando para instalar o Yii2 com template básico.

composer create-project --prefer-dist --stability=dev yiisoft/yii2-app-basic basic

Para instalar o Yii2 com template avançado, execute

composer create-project --prefer-dist --stability=dev yiisoft/yii2-app-advanced advanced
cd advanced
php init

Depois disso, crie um novo banco de dados e ajuste a configuração dos componentes[‘db’] em common/config/main-local.php de acordo. em seguida, execute o seguinte comando para

php yii migrate

Instalando de um arquivo morto #

  1. Baixe o arquivo de Yii-download

  2. Descompacte o arquivo baixado em uma pasta acessível pela Web.

  3. Modifique o arquivo config/web.php inserindo uma chave secreta para o item de configuração cookieValidationKey

    You can add any type of key you want:

     'cookieValidationKey' => '',

 For example : xyctuyvibonp

 'cookieValidationKey' => 'xyctuyvibonp',

    //insert a secret key in the following (if it is empty) - this is required by cookie validation ‘cookieValidationKey’ => ’enter your secret key here’,

Instale o Yii2 advanced no Ubuntu #

Primeiro precisamos instalar o composer. Etapas para instalar o compositor Instale o Compositor.

curl -sS https://getcomposer.org/installer | php

Agora mude o diretório:

sudo mv composer.phar /usr/local/bin/composer

Verifique o trabalho do compositor

composer

Agora o Composer instalado.

Existem duas maneiras de instalar o Yii2 advance.

1.Instalando de um arquivo morto

Obtenha o arquivo zip no link abaixo.

Descompacte-o no diretório de destino, por exemplo /var/www/html.

https://github.com/yiisoft/yii2/releases/download/2.0.8/yii-advanced-app-2.0.8.tgz

Mova para dentro da pasta “avançado”. Mova manualmente ou digite o comando abaixo.

cd advanced

Execute abaixo do comando.

php init

2.Instalando via Composer

A instalação via compositor requer token de autenticação do github. Para token, você precisa se inscrever no GitHub.

Após a inscrição, você pode gerar seu token:

Etapas para gerar um token

  1. No canto superior direito de qualquer página, clique na foto do seu perfil e, em seguida, clique em Configurações.
  2. Na barra lateral de configurações do usuário, clique em Tokens de acesso pessoal.
  3. Clique em Gerar novo token.
  4. Dê um nome descritivo ao seu token.
  5. Selecione os escopos que deseja conceder a este token.
  6. Clique em Gerar token.
  7. Copie o token para a área de transferência. Por motivos de segurança, depois de navegar fora desta página, ninguém poderá ver o token novamente.

Referência: https://help.github.com/articles/creating-an-access-token-for-command-line-use/

Depois de gerar o token copie-o

Alterar diretório

cd /var/www/html/

Execute abaixo do comando

composer config -g github-oauth.github.com <AuthToken>

exemplo:

composer config -g github-oauth.github.com f1eefb8f188c22dd6467f1883cb2615c194d1ce1

Instale o yii2

composer create-project --prefer-dist yiisoft/yii2-app-advanced advanced

Mova para dentro da pasta “avançado”. Mova manualmente ou digite abaixo comando.

cd advanced

Execute abaixo do comando.

php init

Está feito!

Agora você pode verificar.

http://localhost/advanced/frontend/web

e

http://localhost/advanced/backend/web
