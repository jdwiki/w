Instalação ou Configuração #

A instalação no Windows 10 é tão fácil como sempre foi e, apesar dos rumores (verdadeiros) sobre o Windows 10 ser a última versão ’normal’ do Windows, ele sempre permaneceu o mesmo processo.

Instalando uma nova versão

  1. Obtenha a imagem do Windows (vá direto para a etapa 4 se você tiver um disco comprado em loja)
    2. Acesse o site do Windows technet e baixe a versão apropriada do Windows 10. Você precisará inserir a chave do produto adquirida.
  2. Faça o download de um gerador de imagens USB
    3. Pesquise e baixe a ferramenta de imagem Rufus USB. Rode isto.
  3. Carregue a imagem em seu USB
    4. Clique no menu suspenso na parte superior e selecione seu USB. Em seguida, clique no segundo menu suspenso e selecione a opção de imagem ISO. Por fim, clique na imagem ao lado do menu suspenso e encontre seu arquivo ISO. Selecione-o e clique em 'Iniciar'. Quando isso for feito, ejete seu USB.
  4. Instalando
    5. Desligue o laptop e ligue-o novamente. Observe atentamente qualquer texto que apareça dizendo 'Pressione para entrar no menu de inicialização'. Quando vir isso, anote mentalmente o nome do botão e encontre-o no teclado. Em seguida, insira sua mídia de instalação e reinicie novamente, e quando o texto aparecer, pressione o botão. Se tudo correr bem, você deverá ter a opção de executar a mídia de instalação. Quando isso acontecer, basta seguir as instruções na tela para instalar o Windows 10.

Atualizando

  1. Baixando a iso
    2. Siga o passo 1 no guia acima
  2. Monte o arquivo ISO
    3. Clique com o botão direito do mouse neste arquivo e clique na opção 'Montar'
  3. Execute o instalador
    4. Vá para 'Este PC' e abra a imagem do disco (deve aparecer como um disco). Em seguida, execute o arquivo 'setup.exe' lá. Por fim, basta seguir as instruções na tela. Isso precisa de direitos de administrador e demora um pouco, mas depois que isso acontecer, todos os seus arquivos estarão exatamente onde estavam antes, será apenas o Windows 10!

    OneGet e gerenciadores de pacotes #

    O OneGet era originalmente um produto do Open Source Technology Center da Microsoft. Não é apenas inspirado em gerenciadores de pacotes Linux de código aberto, o próprio OneGet também é de código aberto. Agora faz parte do PowerShell

    Ao contrário dos gerenciadores de pacotes baseados em Unix (como apt-get, yum ou dpkg), o Windows permite o uso do Oneget através do PowerShell. Para fazer isso, primeiro você precisa abrir um Powershell como Admin e executar Set-ExecutionPolicy RemoteSigned. Isso é para garantir que os scripts do Powershell disponíveis no Oneget possam ser executados. Em seguida, (usando a mesma instância do PowerShell), execute Import-Module –Name OneGet. Isso importa o módulo PoerShell. Em seguida, reinicie sua instância do powershell e execute Get-Command –Module OneGet para ver todos os comandos disponíveis. Para instalar um pacote, execute o comando Find-Package | <Package_Name. Quando solicitado a instalar o Nuget, clique em sim.

    Adicionando Chocolatey como provedor ao OneGet #

    Infelizmente, o OneGet não possui o provedor de pacotes chocolatey instalado por padrão, mas é possível adicioná-lo.

    Abra uma janela do powershell e execute get-packageprovider -name chocolatey. Você deve então ver a mensagem abaixo:

    O provedor ‘chocolatey v2.8.5.130’ não está instalado. chocolate pode ser baixado manualmente de https://oneget.org/ChocolateyPrototype-2.8.5.130.exe e instalado. Deseja que o PackageManagement baixe automaticamente e instalar ‘chocolatey’?

    [S] Sim [N] Não [S] Suspender [?] Ajuda (o padrão é “S”):

    Digite ‘Y’ e pressione enter.

    Name                     Version          
----                     -------           
Chocolatey               2.8.5.130

    Encontrando pacotes #

    Para os programas mais conhecidos, o nome do aplicativo geralmente funciona (por exemplo, Chrome). mas, assim como o chocolate pode ser difícil de navegar, o OneGet também pode. Felizmente, o OneGet tem uma implementação de choco search. Este comando agora é find-package <Nome do Pacote>. Isso lhe dará uma lista de pacotes como visto abaixo:

    Name                           Version          Source                         Summary
----                           -------          ------                         -------
thunderbird                    45.2.0           chocolatey                     A free email client from Mozilla
Google Chrome                  47.5.8           OneGet                         A high end web browser

    Instalando pacotes #

    Para instalar um pacote, você precisa executar um PowerShell como administrador e executar o comando install-package <PAckage Name>

    O(s) pacote(s) vem de uma fonte de pacote que não está marcada como confiável. Tem certeza de que deseja instalar o software de ‘chocolatey’? [S] Sim [A] Sim para todos [N] Não [L] Não para todos [S] Suspender [?] Ajuda (o padrão é “N”):

    Para isso, basta pressionar y e/ou enter. Você verá uma tela semelhante à abaixo

    Name                           Version          Source           Summary
----                           -------          ------           -------
<Package Name>                 <Most recent>    <Server name>    <Description>

    Desinstalando #

    Para desinstalar um pacote, basta usar o comando abaixo

    uninstall-package vlc
