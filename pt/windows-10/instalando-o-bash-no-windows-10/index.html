Instale o Bash no Windows 10 #

Esta documentação é um pequeno resumo do oficial

Pré-requisitos #

  1. Seu PC deve estar executando uma versão de 64 bits do Windows 10 Anniversary Update build 14393 ou posterior

    To find your PC’s CPU architecture and Windows version/build number, open Settings>System>About. Look for the OS Build and System Type fields.

  2. Ative o modo de desenvolvedor
    • Open Settings -> Update and Security -> For developers
    • Select the Developer Mode radio button
  3. Habilite o recurso Windows Subsystem for Linux (GUI ou linha de comando
  • Em Iniciar, procure por “Ativar ou desativar recursos do Windows” - Select Windows Subsystem for Linux (beta) - Click OK
  • OU Abra um prompt do PowerShell como administrador e execute: Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux
  1. Reinicie o computador

Instale o Bash no Windows #

Abra um prompt de comando e execute bash

Depois de aceitar a Licença, a imagem do modo de usuário do Ubuntu será baixada e um atalho “Bash no Ubuntu no Windows” será adicionado ao seu menu Iniciar.

Após a instalação, sua distribuição Linux estará localizada em: %localappdata%\lxss\

Execute o Bash no Windows #

Para executar o bash como na outra etapa

Abra um prompt de comando e execute bash

Ou use o atalho “Bash no Ubuntu no Windows”

Na primeira vez que você instalar o Bash no Windows, será solicitado que você crie um nome de usuário e senha do UNIX.

Créditos ao link oficial!

Primeiro passo #

Olá pessoal,

Este primeiro artigo ajuda algumas pessoas a instalar o bash no Windows 10.

Na primeira etapa você precisa desinstalar todos os antivírus que possui, por exemplo, eu uso o Avast, precisei desinstalar isso para instalar o bash. Se você usa proteção do Windows, não pode encaminhar.

Instalando #

Abra o prompt no Windows.
Então você vai executar: lxrun /install
Se esta for a primeira vez, você informará um usuário ao console Bash.
Ou você pode simplesmente usar lxrun /install /y
Para fazer esta etapa automaticamente.

Desinstalando #

Você só precisa executar no prompt:
lxrun /uninstall /full
