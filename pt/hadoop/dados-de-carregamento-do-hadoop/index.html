Carregar dados no hadoop hdfs #

PASSO 1: CRIAR UM DIRETÓRIO EM HDFS, CARREGAR UM ARQUIVO E LISTA DE CONTEÚDO

Vamos aprender escrevendo a sintaxe. Você poderá copiar e colar os seguintes comandos de exemplo em seu terminal:

hadoop fs -mkdir: #

Toma os URIs do caminho como argumento e cria um diretório ou vários diretórios.

Uso: #

    # hadoop fs -mkdir <paths>

Exemplo: #

    hadoop fs -mkdir /user/hadoop
    hadoop fs -mkdir /user/hadoop/dir1 /user/hadoop/dir2 /user/hadoop/dir3

hadoop fs -put: #

Copia um único arquivo src ou vários arquivos src do sistema de arquivos local para o Hadoop Distributed File System.

Uso: #

    # hadoop fs -put <local-src> ... <HDFS_dest_path>

Exemplo: #

    hadoop fs -put popularNames.txt /user/hadoop/dir1/popularNames.txt

hadoop fs -copyFromLocal: #

Copia um único arquivo src ou vários arquivos src do sistema de arquivos local para o Hadoop Distributed File System.

Uso: #

    # hadoop fs -copyFromLocal <local-src> ... <HDFS_dest_path>

Exemplo: #

    hadoop fs -copyFromLocal popularNames.txt /user/hadoop/dir1/popularNames.txt

hadoop fs -moveFromLocal: #

Semelhante ao comando put, exceto que o localsrc de origem é excluído após ser copiado.

Uso: #

    # hadoop fs -moveFromLocal <local-src> ... <HDFS_dest_path>

Exemplo: #

    hadoop fs -moveFromLocal popularNames.txt /user/hadoop/dir1/popularNames.txt

FERRAMENTA DE TRANSFERÊNCIA DE DADOS SQOOP:

Também podemos carregar dados no HDFS diretamente de bancos de dados relacionais usando Sqoop (uma ferramenta de linha de comando para transferência de dados de RDBMS para HDFS e vice-versa).

Uso: #

$ sqoop import --connect CONNECTION_STRING --username USER_NAME --table TABLE_NAME

Exemplo: #

$ sqoop import --connect jdbc:mysql://localhost/db --username foo --table TEST
