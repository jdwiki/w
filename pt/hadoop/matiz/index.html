Hue é uma interface de usuário para conectar e trabalhar com a maioria das tecnologias de Bigdata comumente usadas, como HDFS, Hive, Spark, Hbase, Sqoop, Impala, Pig, Oozie etc. Hue também suporta a execução de consultas em bancos de dados relacionais.

Hue, um aplicativo da web django, foi construído principalmente como um ambiente de trabalho para executar consultas do Hive. Mais tarde, a funcionalidade do Hue aumentou para suportar diferentes componentes do Ecossistema Hadoop. Está disponível como software de código aberto sob a licença Apache.

Instalação do Hue no Ubuntu #

Esta instalação assume que o hadoop está pré-instalado no usuário hadoop.

Pré-requisitos:

Hue depende destes seguintes pacotes

  1. gcc
  2. g++
  3. libxml2-dev
  4. libxlst-dev
  5. libsasl2-dev
  6. libsasl2-modules-gssapi-mit
  7. libmysqlclient-dev
  8. python-dev
  9. ferramentas de configuração do python
  10. libsqlite3-dev
  11. formiga
  12. libkrb5-dev
  13. libtidy-0.99-0
  14. libldap2-dev
  15. libssl-dev
  16. libgmp3-dev

Instalando todos os pacotes

sudo apt-get update
sudo apt-get install gcc g++ libxml2-dev libxslt-dev libsasl2-dev libsasl2-modules-gssapi-mit libmysqlclient-dev python-dev python-setuptools libsqlite3-dev ant libkrb5-dev libtidy-0.99-0 libldap2-dev libssl-dev libgmp3-dev

Instalação e Configuração

Executando a instalação como usuário hadoop.

su - hadoop

  1. Faça o download do Hue em gethue.com (este link é um exemplo obtido no site do Hue)

    wget https://dl.dropboxusercontent.com/u/730827/hue/releases/3.9.0/hue-3.9.0.tgz

  2. Extraia o tarball baixado

    tar -xvf hue-3.9.0.tgz

  3. Execute o comando de instalação

     cd hue-3.9.0
 PREFIX=/home/hadoop/ make install

  4. Uma vez concluído o processo acima,

    Update ~/.bashrc file,

     export HUE_HOME=/home/hadoop/hue
 export PATH=$PATH:$HUE_HOME/build/env/bin

    source after adding the entries, source ~/.bashrc

  5. Configure o Hue (3 arquivos para editar)

cd $HUE_HOME/desktop/conf

  • hue.ini

    [desktop]
 server_user=hadoop
 server_group=hadoop
 default_user=hadoop
 default_hdfs_superuser=hadoop

cd $HADOOP_CONF_DIR

  • core-site.xml

      <property>
      <name>hadoop.proxyuser.hadoop.hosts</name>
      <value>*</value> 
  </property> 
  <property>
      <name>hadoop.proxyuser.hadoop.groups</name>
      <value>*</value> 
  </property>

  • hdfs-site.xml

      <property>
      <name>dfs.webhdfs.enabled</name>
      <value>true</value> 
  </property>

  1. Inicie o Hue (inicia os daemons do Hadoop, se ainda não tiver sido iniciado)

    nohup supervisor &

  2. Faça login na interface da Web do Hue: http://localhost:8888

    username: hadoop

    password: user_choice

Dependências de instalação #

Detalhes do processo de instalação do Hue não estão disponíveis para a maioria dos sistemas operacionais, portanto, dependendo do sistema operacional, pode haver variações nas dependências que você precisa instalar antes de executar o script de instalação fornecido no pacote de instalação :

CentOS

sudo yum install ant
sudo yum install python-devel.x86_64
sudo yum install krb5-devel.x86_64
sudo yum install krb5-libs.x86_64
sudo yum install libxml2.x86_64
sudo yum install python-lxml.x86_64
sudo yum install libxslt-devel.x86_64
sudo yum install mysql-devel.x86_64
sudo yum install openssl-devel.x86_64
sudo yum install libgsasl-devel.x86_64
sudo yum install sqlite-devel.x86_64
sudo yum install openldap-devel.x86_64
sudo yum install -y libffi libffi-devel
sudo yum install mysql-devel gcc gcc-devel python-devel
sudo yum install rsync
sudo yum install maven
wget https://bootstrap.pypa.io/ez_setup.py -O - | sudo python
  1. BPF

  • CentOS > 7.x sudo yum install libgmp3-dev

  • CentOS < 6.x sudo yum install gmp gmp-devel gmp-status
