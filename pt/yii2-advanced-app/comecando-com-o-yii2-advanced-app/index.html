Instalação ou Configuração #

Requerimentos de instalação

O requisito mínimo deste modelo de projeto é que seu servidor Web suporte PHP 5.4.0.

O Yii2-advanced-app pode ser instalado de duas maneiras. Eles são

  1. Instalando pelo Composer
  2. Instalando a partir de um arquivo morto

1) Instalando via Composer

Se você ainda não tem o Composer instalado, pode fazê-lo seguindo as instruções em getcomposer.org. No Linux e no Mac OS X, você executará os seguintes comandos:

Primeiro precisamos instalar o composer.

Passo 1 Instale o Compositor. Execute o comando abaixo no seu terminal

curl -sS https://getcomposer.org/installer | php

Passo-2 Agora mude o diretório

sudo mv composer.phar /usr/local/bin/composer

Etapa-3 verifique o trabalho do compositor

composer

Se o Composer foi instalado com sucesso.

Em seguida, instale o aplicativo usando os seguintes comandos:

composer global require "fxp/composer-asset-plugin:~1.1.1"
composer create-project --prefer-dist yiisoft/yii2-app-advanced yii-application

O primeiro comando instala o composer asset plugin que permite gerenciar as dependências dos pacotes bower e npm através do Composer. Você só precisa executar este comando uma vez por todas. O segundo comando instala o aplicativo avançado em um diretório chamado yii-application. Você pode escolher um nome de diretório diferente se quiser.

2) Instalando a partir de um arquivo morto

Extraia o arquivo baixado de yiiframework.com para um diretório chamado advanced que esteja diretamente sob a raiz da Web.

Modifique o arquivo config/web.php inserindo uma chave secreta para o item de configuração cookieValidationKey (isso é feito automaticamente se você estiver instalando o Yii usando o Composer):

// !!! insert a secret key in the following (if it is empty) - this is required by cookie validation
'cookieValidationKey' => 'enter your secret key here',

Em seguida, siga as instruções fornecidas na próxima subseção.

Preparando inscrição

Depois de instalar o aplicativo, você deve realizar as etapas a seguir para inicializar o aplicativo instalado. Você só precisa fazer isso uma vez por todas.

  1. Abra um terminal de console, execute o comando init e selecione dev como ambiente.

    /path/to/php-bin/php /path/to/yii-application/init

Se você automatizá-lo com um script, poderá executar o init no modo não interativo.

/path/to/php-bin/php /path/to/yii-application/init --env=Production --overwrite=All

  1. Crie um novo banco de dados e ajuste a configuração components['db'] em common/config/main-local.php de acordo.

  2. Abra um terminal de console, aplique as migrações com o comando /path/to/php-bin/php /path/to/yii-application/yii migrate.
