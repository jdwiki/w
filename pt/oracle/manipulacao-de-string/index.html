INITCAP #

A função INITCAP converte as maiúsculas e minúsculas de uma string para que cada palavra comece com uma letra maiúscula e todas as letras subseqüentes sejam em minúsculas.

SELECT INITCAP('HELLO mr macdonald!') AS NEW FROM dual;

Resultado

NEW
-------------------
Hello Mr Macdonald!

Concatenação: Operador || ou função concat() #

O operador Oracle SQL e PL/SQL || permite concatenar 2 ou mais strings juntas.

Exemplo:

Assumindo a seguinte tabela clientes:

 id  firstname    lastname
---  -----------  ----------
  1  Thomas       Woody

Consulta:

 SELECT firstname || ' ' || lastname || ' is in my database.' as "My Sentence" 
   FROM customers;

Resultado:

My Sentence
---------------------------------
Thomas Woody is in my database.

O Oracle também suporta a função padrão SQL CONCAT(str1, str2):

Exemplo:

Consulta:

 SELECT CONCAT(firstname, ' is in my database.') from customers;

Resultado:

Expr1
---------------------------------
Thomas is in my database.

SUPERIOR #

A função UPPER permite converter todas as letras minúsculas de uma string em maiúsculas.

SELECT UPPER('My text 123!') AS result FROM dual;

Resultado:

RESULT    
------------
MY TEXT 123!

MAIS BAIXO #

LOWER converte todas as letras maiúsculas em uma string para minúsculas.

SELECT LOWER('HELLO World123!') text FROM dual;

Saídas:

texto
olá mundo123!

Expressão regular #

Digamos que queremos substituir apenas números com 2 dígitos: a expressão regular os encontrará com (\d\d)

SELECT REGEXP_REPLACE ('2, 5, and 10 are numbers in this example', '(\d\d)', '#')
FROM dual;

Resulta em:

'2, 5, and # are numbers in this example'

Se eu quiser trocar partes do texto, eu uso \1, \2, \3 para chamar as strings correspondentes:

 SELECT REGEXP_REPLACE ('swap around 10 in that one ', '(.*)(\d\d )(.*)', '\3\2\1\3')
 FROM dual;

SUBSTR #

SUBSTR recupera parte de uma string indicando a posição inicial e o número de caracteres a serem extraídos

SELECT SUBSTR('abcdefg',2,3) FROM DUAL;

retorna:

bcd

Para contar a partir do final da string, SUBSTR aceita um número negativo como segundo parâmetro, por exemplo.

SELECT SUBSTR('abcdefg',-4,2) FROM DUAL;

retorna:

de

Para obter o último caractere em uma string: SUBSTR(mystring,-1,1)

LTRIM / RTRIM #

LTRIM e RTRIM removem caracteres do início ou do fim (respectivamente) de uma string. Um conjunto de um ou mais caracteres pode ser fornecido (o padrão é um espaço) para remover.

Por exemplo,

select LTRIM('<===>HELLO<===>', '=<>')
      ,RTRIM('<===>HELLO<===>', '=<>')
from dual;

Devoluções:

HELLO<===>
<===>HELLO
