Introdução #

O que é Kibana:

O Kibana é usado para fazer visualizações e criar painéis para os índices apresentados no elasticsearch. Basicamente, é um plug-in de código aberto para elasticsearch.

[

Existem seis abas:

  1. Descubra:

    You can explore your data from Discover tab

  2. Visualização:

    Creating visualization with different charts such as bar chart,line chart,pie chart etc.

  3. Painel:

    It is used for creating dashboard with created visualization. It contains your visualization to represent your process.

  4. Cronograma:

    It is timeseries of kibana

  5. Ferramentas de desenvolvimento

    It is like sense plugin used for elasticsearch. It contains request and response field

  6. Gerenciamento

    This is used for managing your kibana. It contains Settings for the kibana plugin

Instale o kibana usando o apt-get

Para instalar o kibana, você precisa descobrir qual versão do elastic search está instalada em seu sistema. Isso pode ser verificado usando o seguinte comando:

curl "http://localhost:9200"

Na saída, você precisa verificar o número, ou seja, a versão de pesquisa elástica.

Agora encontre a versão adequada para isso. Aqui está o comando para instalar o kibana 5.4.0 para pesquisa elástica 5.4.0.

wget -qO - https://artifacts.elastic.co/GPG-KEY-elasticsearch | sudo apt-key add -
sudo apt-get install apt-transport-https
echo "deb https://artifacts.elastic.co/packages/5.x/apt stable main" | sudo tee -a /etc/apt/sources.list.d/elastic-5.x.list
sudo apt-get update && sudo apt-get install kibana

Execute o Kibana como um serviço

Após a instalação o Kiabana não inicia a execução automaticamente. Como iniciar e parar depende se o seu sistema suporta init ou systemd do SysV. verifique isso usando o seguinte comando.

ps -p 1

Para inicialização do Sysv

execute o seguinte comando:

sudo update-rc.d kibana defaults 95 10

Para iniciar e parar o servidor, use o seguinte comando.

sudo -i service kibana start
sudo -i service kibana stop

Para sistema :

execute os seguintes comandos:

sudo /bin/systemctl daemon-reload
sudo /bin/systemctl enable kibana.service

Para iniciar e parar o servidor, use o seguinte comando.

sudo systemctl start kibana.service
sudo systemctl stop kibana.service

Configurar #

Para configuração do kibana, abra config/kibana.yml e aponte seu endereço elasticsearch.

By default it is http://localhost:9200

Antes de iniciar o elasticsearch, verifique se o elasticsearch está em execução. Porque ele depende do Elasticsearch.

Para janelas:

execute o arquivo de correção bin/kibana.

Para Mac:

execute ./character

Para Linux:

execute bin/kibana.
