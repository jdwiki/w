AtivarDesativar a tecla Shift no DB Open

  • Desativar código de função de deslocamento
  • Ativar código de função de turno

Nesta página

  • Desativar código de função de deslocamento
  • Ativar código de função de turno

Esse código desativará a capacidade de um usuário manter pressionada a tecla Shift ao abrir um banco de dados para ignorar a abertura de formulário padrão e permitir que o usuário acesse o Painel de Navegação e o Editor VB. Em bancos de dados que você não deseja que os usuários tenham acesso a nenhum deles (além de desabilitar o uso de chaves especiais nas opções atuais do banco de dados), esse código ajudará a manter o banco de dados bloqueado.

Geralmente, o código abaixo é colocado em seu próprio módulo que pode ser chamado de módulo basEnableDisableShift, mas isso é apenas uma sugestão e você pode colocá-lo em qualquer módulo que já tenha.

Para desabilitar a tecla Shift, na tela do Editor VB, digite ‘DisableShift’ na janela Immediate e pressione Enter. Você receberá uma mensagem informando que a tecla Shift foi desabilitada.

Para reativar a tecla Shift, você precisará mais uma vez retornar à tela do Editor VB, digite ‘EnableShift’ na sua janela imediata e pressione Enter. Você receberá novamente uma mensagem em sua janela imediata informando que a tecla Shift foi habilitada.

NOTA: Esta não é uma maneira infalível de habilitar e desabilitar a tecla Shift, mas se você estiver implantando um banco de dados para usuários que não são proficientes com MS Access e VBA, deve ser útil para evitar que os usuários acessando o Editor VB e/ou Painel de Navegação em seu banco de dados.

Desativar código de função de deslocamento #

Function DisableShift()
'This function disable the shift at startup. This action causes
'the Autoexec macro and Startup properties to always be executed.

On Error GoTo errDisableShift

    Dim db As DAO.Database
    Dim prop As DAO.Property
    Const conPropNotFound = 3270

    Set db = CurrentDb()

    'This next line disables the shift key on startup.
    db.Properties("AllowByPassKey") = False

    'The function is successful.
    Debug.Print "Disabled Shift Key - Successful"
    Exit Function

errDisableShift:
    'The first part of this error routine creates the "AllowByPassKey
    'property if it does not exist.
    If Err = conPropNotFound Then
        Set prop = db.CreateProperty("AllowByPassKey", _
        dbBoolean, False)
        db.Properties.Append prop
        Resume Next
        Else
            MsgBox "Function 'ap_DisableShift' did not complete successfully."
            GoTo ExitHere
    End If

ExitHere:
    Set prop = Nothing
    Set db = Nothing
    Exit Function

End Function

Ativar código de função de turno #

Function EnableShift()
'This function enables the SHIFT key at startup. This action causes
'the Autoexec macro and the Startup properties to be bypassed
'if the user holds down the SHIFT key when the user opens the database.

On Error GoTo errEnableShift

    Dim db As DAO.Database
    Dim prop As DAO.Property
    Const conPropNotFound = 3270

    Set db = CurrentDb()

    'This next line of code disables the SHIFT key on startup.
    db.Properties("AllowByPassKey") = True

    'function successful
    Debug.Print "Enabled Shift Key - Successful"
    GoTo ExitHere

errEnableShift:
    'The first part of this error routine creates the "AllowByPassKey
    'property if it does not exist.
    If Err = conPropNotFound Then
        Set prop = db.CreateProperty("AllowByPassKey", _
        dbBoolean, True)
        db.Properties.Append prop
        Resume Next
        Else
            MsgBox "Function 'ap_DisableShift' did not complete successfully."
            GoTo ExitHere
    End If

ExitHere:
    Set prop = Nothing
    Set db = Nothing
    Exit Function

End Function
