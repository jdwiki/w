Baixando e instalando os produtos DevExpress .NET #

Para instalar os produtos DevExpress .NET em sua máquina de desenvolvimento, primeiro é necessário obter o Instalador de Produtos DevExpress .NET. Você pode baixá-lo do site http://www.devexpress.com de uma das seguintes maneiras, com base no status de sua conta atual.

  • Baixe a versão de avaliação Se você ainda não comprou os produtos DevExpress .NET, mas gostaria de experimentá-los, você é livre para baixar a versão de avaliação.

  • Baixe a versão registrada Se você já possui uma licença para produtos DevExpress .NET, você pode baixar a versão registrada na Central do Cliente.

O Instalador de Produtos .NET DevExpress instala assemblies .NET, que contêm os controles e componentes visuais do DevExpress e os integra ao IDE do Visual Studio. Além disso, esta ferramenta instala aplicativos de demonstração que ilustram a funcionalidade fornecida pelos produtos DevExpress .NET.

Você pode executar o instalador em um dos dois modos.

  • Modo de instalação GUI Este é o modo de interface gráfica do usuário (GUI), no qual você pode seguir as etapas do assistente de instalação e especificar manualmente todas as configurações de instalação visualmente.

  • Modo de instalação silenciosa Este é o modo de instalação silenciosa, que permite executar o instalador a partir de uma linha de comando como um aplicativo de console e especificar todos os parâmetros necessários.
