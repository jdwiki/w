Shopify desenvolvimento local com o Kit de Tema #

Os temas Shopify também podem ser desenvolvidos localmente. Siga as etapas abaixo para configurar o ambiente de desenvolvimento local:

  • Instale o kit de temas da Shopify
  • Obtenha acesso à API
  • Use um novo tema ou configure um tema existente com o kit de temas

A explicação das etapas acima são as seguintes:


*

Instale o kit de temas da Shopify


Instalação Automática

Se você estiver no Mac ou Linux, poderá usar o seguinte script de instalação para baixar e instalar automaticamente o Kit de Tema para você.

curl -s https://raw.githubusercontent.com/Shopify/themekit/master/scripts/install | sudo python

Homebrew

Se você tiver o homebrew instalado, poderá instalar o Theme Kit executando os seguintes comandos.

brew tap shopify/shopify
brew install themekit

**Instalação do Windows**


Instalação automática do Powershell

Execute os seguintes comandos no Powershell como administrador.

(New-Object System.Net.WebClient).DownloadString("https://raw.githubusercontent.com/Shopify/themekit/master/scripts/install.ps1") | powershell -command -

Manual

  • Crie uma pasta dentro de C:\Program Files\ chamada Theme Kit
  • Baixe o themekit (abaixo) e copie o programa extraído em C:\Program Files\Theme Kit
  • Navegue até Painel de controle > Sistema e segurança > Sistema. Outra maneira de chegar lá é clicar com o botão direito do mouse em Meu computador e escolher o item de propriedades
  • Procure o botão ou link chamado Variáveis ​​de Ambiente
  • No segundo painel procure o item chamado Path e dê um duplo clique nele. Isso deve abrir uma janela com um campo de texto que está transbordando de conteúdo.
  • Mova o cursor até o final e adicione o seguinte: ;C:\Program Files\Theme Kit\
  • Clique em OK até que todas as janelas desapareçam.
  • Para verificar se o Theme Kit foi instalado, abra cmd.exe e digite theme.

Instalação Manual

Baixe o executável da versão mais recente.

**Obter acesso à API**


Você precisará configurar uma chave de API para adicionar à nossa configuração e criar uma conexão entre sua loja e o Theme Kit. A chave de API permite que o Theme Kit converse e acesse sua loja, bem como seus arquivos de tema.

Para fazer isso, faça login na loja da Shopify e crie um aplicativo privado. No Shopify Admin, vá para Apps e clique em View private apps. A partir daí, clique em Gerar credenciais de API para criar seu aplicativo privado. Certifique-se de definir as permissões de modelos de tema e ativos de tema para ter acesso de leitura e gravação para gerar as credenciais de API apropriadas e clique em Salvar.

Preencha as informações na parte superior e defina as permissões de modelos de tema e ativos de tema para acesso de leitura e gravação. Pressione Salvar e você será apresentado à próxima tela. Nele você verá suas credenciais de acesso. Por favor, anote a senha. Você precisará disso mais tarde.


Use um novo tema.

Se você está começando do zero e deseja começar rapidamente, execute o seguinte:

theme bootstrap --password=[your-password] --store=[your-store.myshopify.com]

Isso criará um novo tema para sua loja virtual a partir do modelo Timber. Em seguida, ele fará o download de todos esses ativos da Shopify e criará automaticamente um arquivo config.yml para você.


Configure um tema existente.

Para conectar um tema existente, você precisa do número de identificação do tema. A maneira mais fácil de obter o número de ID do seu tema é ir até o Editor de Temas, clicar em Editar HTML/CSS e copiar o número de ID do tema da URL — serão os últimos dígitos depois de mystore.myshopify.com/admin/themes/.

Depois de anotar o ID do seu tema, execute os seguintes comandos:

# create configuration
theme configure --password=[your-password] --store=[you-store.myshopify.com] --themeid=[your-theme-id]
# download and setup project in the current directory
theme download

Instalação ou configuração #

Instruções detalhadas sobre como configurar ou instalar o shopify.

Configuração de tema local #

Existem algumas maneiras de configurar uma cópia local do seu tema da Shopify:

  • Node.js ( gulp/grunt )
  • Rubi
  • Aplicativo de tema (somente Mac)

Pode haver algumas outras maneiras, mas achei a maneira rubi a mais fácil de trabalhar.

Para trabalhar localmente nos arquivos do tema Shopify, você precisará de algumas coisas:

  • Instalador do Ruby ( http://rubyinstaller.org/downloads/ )
  • Kit de Desenvolvimento ( http://rubyinstaller.org/downloads/ )
  • Certificado SSL - ( https://curl.haxx.se/ca/cacert.pem )
  • Shopify Theme Gem - ( https://rubygems.org/gems/shopify_theme/versions/0.0.25 )

Instalador do Ruby #

A primeira coisa é instalar o instalador do Ruby, use a versão mais recente.

O processo de instalação é simples, mas ele solicitará que você adicione Ruby ao Caminho Global do Ambiente Windows e você deve clicar na caixa de seleção e adicioná-lo lá, para que você não precise fazê-lo manualmente depois.

Kit de desenvolvimento #

Você deve escolher a versão correta do DevKit que acompanha o instalador Ruby que você adicionou.

O processo de instalação também é fácil.

  1. Você descompacta o arquivo em C:\DevKit por exemplo
  2. Você cmd nessa pasta e escreve ruby dk.rb init que deve criar um config.yml no filder
  3. Se você abrir o arquivo config.yml, você deverá ver na parte inferior - C:/Ruby23-x64 (dependendo de onde você instalou seu Ruby e qual versão está) se não estiver lá, você deve adicioná-lo manualmente
  4. Na mesma pasta depois você executa o seguinte ruby dk.rb install
  5. Se tudo estiver OK, você poderá executar o seguinte comando - gem install json --platform=ruby que confirmará que sua instalação do Ruby está funcionando

Instruções mais detalhadas também podem ser vistas aqui: https://github.com/oneclick/rubyinstaller/wiki/Development-Kit

Certificado SSL #

Você precisará adicionar o arquivo cacert.pem às variáveis ​​globais de sua janela.

  1. Você deve baixar o seguinte arquivo: https://curl.haxx.se/ca/cacert.pem ( salve-o como .pem e não como .pem.txt )
  2. Você deve entrar na tela de variáveis ​​​​de ambiente e, em variáveis ​​​​do sistema, clique em Novo
  3. Para o nome da variável você insere SSL_CERT_FILE e para o valor da variável você insere o caminho para o arquivo, por exemplo C:\RailsInstaller\cacert.pem.

Joia do tema Shopify #

Abra seu cmd e digite o seguinte código: gem install shopify_theme

Isso deve instalar a gema do tema Shopify que precisaremos.

Se você passou por todas as etapas acima uma vez, não as fará mais.

Agora vem as etapas que você terá que fazer toda vez que iniciar um novo projeto da Shopify:

  • Crie um aplicativo privado (da Shopify Store)
  • Crie um arquivo de configuração usando as informações do aplicativo privado

Criar um aplicativo privado #

Você deve fazer login no painel de administração da sua loja da Shopify e ir para a tela do aplicativo -> Aplicativo privado:

https://YOUR-STORE-NAME.myshopify.com/admin/apps/private

E clique no botão Criar aplicativo privado.

Ele vai pedir um título. (insira o que quiser)

A parte importante é alterar todas as permissões para Ler e gravar

Depois de alterar as permissões e salvar o aplicativo, ele o redirecionará para uma página da qual você obterá a chave e a senha da API para este aplicativo. (você precisa deles para o arquivo de configuração que criaremos abaixo)

Criar um arquivo de configuração #

Vá para uma pasta local onde você deseja que os arquivos do tema estejam e cmd dentro.

Escreva o seguinte comando:

tema configurar API-KEY PASSWORD DOMAIN ID

Onde:

  • Chave de API - é tirada deles Aplicativo privado que criamos
  • Senha - é tirada deles Aplicativo privado que criamos
  • Domínio - este é o seu domínio YOUR-STORE-NAME.myshopify.com, sem https:// antes dele
  • ID - este é o ID do tema, se você for trabalhar no tema ao vivo, pode pular isso, mas se estiver trabalhando em um clone deles, precisará inserir o ID do tema aqui

Depois de executar o código acima, ele deve criar um arquivo chamado config.yml onde todas as informações fornecidas estarão presentes.

Agora você deve fazer o cmd ( caso tenha fechado o console ) na pasta onde está o config.yml e escrever o seguinte código: theme download que deve iniciar o download dos arquivos um por um.

Depois de concluir o processo de download, você escreve theme watch e toda vez que modificar os arquivos baixados, ele atualizará os arquivos na loja Shopify após 1-2 segundos também. Isso é verdade se você adicionar arquivos também, mas não quando excluir arquivos.

Informações mais detalhadas para a Gem podem ser encontradas aqui: https://github.com/Shopify/shopify_theme
