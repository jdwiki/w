Sintaxe #

  • for ($i = 0; $i < count($array); $i++) { incremental_iteration(); }
  • for ($i = count($array) - 1; $i >= 0; $i–) { reverse_iteration(); }
  • foreach ($dados como $datum) { }
  • foreach ($dados como $chave => $datum) { }
  • foreach ($dados como &$datum) { }

Comparação de métodos para iterar um array #

Método Vantagem
foreach O método mais simples para iterar uma matriz.
foreach por referência Método simples para iterar e alterar elementos de um array.
for com índice incremental Permite iterar a matriz em uma sequência livre, por exemplo. pulando ou invertendo vários elementos
Ponteiros de matriz interna Não é mais necessário usar um loop (para que ele possa iterar uma vez a cada chamada de função, recepção de sinal, etc.)

Iterando vários arrays juntos #

Às vezes, duas matrizes do mesmo tamanho precisam ser iteradas juntas, por exemplo:

$people = ['Tim', 'Tony', 'Turanga'];
$foods = ['chicken', 'beef', 'slurm'];

array_map é a maneira mais simples de fazer isso:

array_map(function($person, $food) {
    return "$person likes $food\n";
}, $people, $foods);

que irá produzir:

Tim likes chicken
Tony likes beef
Turanga likes slurm

Isso pode ser feito através de um índice comum:

assert(count($people) === count($foods));
for ($i = 0; $i < count($people); $i++) {
    echo "$people[$i] likes $foods[$i]\n";
}

Se os dois arrays não tiverem as chaves incrementais, array_values($array)[$i] pode ser usado para substituir $array[$i].

Se ambos os arrays tiverem a mesma ordem de chaves, você também pode usar um loop foreach-with-key em um dos arrays:

foreach ($people as $index => $person) {
    $food = $foods[$index];
    echo "$person likes $food\n";
}

Arrays separados só podem ser repetidos se tiverem o mesmo comprimento e também tiverem o mesmo nome de chave. Isso significa que se você não fornecer uma chave e elas forem numeradas, você ficará bem, ou se você nomear as chaves e colocá-las na mesma ordem em cada array.

Você também pode usar array_combine.

$combinedArray = array_combine($people, $foods);
// $combinedArray = ['Tim' => 'chicken', 'Tony' => 'beef', 'Turanga' => 'slurm'];

Então você pode percorrer isso fazendo o mesmo que antes:

foreach ($combinedArray as $person => $meal) {
    echo "$person likes $meal\n";
}

Usando um índice incremental #

Esse método funciona incrementando um inteiro de 0 ao maior índice na matriz.

$colors = ['red', 'yellow', 'blue', 'green'];
for ($i = 0; $i < count($colors); $i++) {
    echo 'I am the color ' . $colors[$i] . '<br>';
}

Isso também permite iterar um array na ordem inversa sem usar array_reverse, o que pode resultar em sobrecarga se o array for grande.

$colors = ['red', 'yellow', 'blue', 'green'];
for ($i = count($colors) - 1; $i >= 0; $i--) {
    echo 'I am the color ' . $colors[$i] . '<br>';
}

Você pode pular ou retroceder o índice facilmente usando este método.

$array = ["alpha", "beta", "gamma", "delta", "epsilon"];
for ($i = 0; $i < count($array); $i++) {
    echo $array[$i], PHP_EOL;
    if ($array[$i] === "gamma") {
        $array[$i] = "zeta";
        $i -= 2;
    } elseif ($array[$i] === "zeta") {
        $i++;
    }
}

Resultado:

alpha
beta
gamma
beta
zeta
epsilon

Para matrizes que não possuem índices incrementais (incluindo matrizes com índices em ordem inversa, por exemplo, [1 => "foo", 0 => "bar"], ["foo" => "f", "bar" => "b"]), isso não pode ser feito diretamente. array_values ou array_keys podem ser usados ​​em vez disso:

$array = ["a" => "alpha", "b" => "beta", "c" => "gamma", "d" => "delta"];
$keys = array_keys($array);
for ($i = 0; $i < count($array); $i++) {
    $key = $keys[$i];
    $value = $array[$key];
    echo "$value is $key\n";
}

Usando ponteiros de array internos #

Cada instância de array contém um ponteiro interno. Ao manipular esse ponteiro, diferentes elementos de um array podem ser recuperados da mesma chamada em momentos diferentes.

Usando each #

Cada chamada para each() retorna a chave e o valor do elemento atual do array e incrementa o ponteiro interno do array.

$array = ["f" => "foo", "b" => "bar"];
while (list($key, $value) = each($array)) {
    echo "$value begins with $key";
}

Usando next #

$array = ["Alpha", "Beta", "Gamma", "Delta"];
while (($value = next($array)) !== false) {
    echo "$value\n";
}

Observe que este exemplo assume que nenhum elemento no array é idêntico ao booleano false. Para evitar tal suposição, use key para verificar se o ponteiro interno atingiu o final do array:

$array = ["Alpha", "Beta", "Gamma", "Delta"];
while (key($array) !== null) {
    echo current($array) . PHP_EOL;
    next($array);
}

Isso também facilita a iteração de um array sem um loop direto:

class ColorPicker {
    private $colors = ["#FF0064", "#0064FF", "#64FF00", "#FF6400", "#00FF64", "#6400FF"];
    public function nextColor() : string {
        $result = next($colors);
        // if end of array reached
        if (key($colors) === null) {
            reset($colors);
        }
        return $result;
    }
}

Usando foreach #

Circuito direto #

foreach ($colors as $color) {
    echo "I am the color $color<br>";
}

Loop com teclas #

$foods = ['healthy' => 'Apples', 'bad' => 'Ice Cream'];
foreach ($foods as $key => $food) {
    echo "Eating $food is $key";
}

Loop por referência #

Nos loops foreach nos exemplos acima, modificar o valor ($color ou $food) diretamente não altera seu valor no array. O operador & é necessário para que o valor seja um ponteiro de referência para o elemento na matriz.

$years = [2001, 2002, 3, 4];
foreach ($years as &$year) {
    if ($year < 2000) $year += 2000;
}

Isso é semelhante a:

$years = [2001, 2002, 3, 4];
for($i = 0; $i < count($years); $i++) { // these two lines
    $year = &$years[$i];                // are changed to foreach by reference
    if($year < 2000) $year += 2000;
}

Simultaneidade #

Matrizes PHP podem ser modificadas de qualquer maneira durante a iteração sem problemas de simultaneidade (diferente de, por exemplo, Java Lists). Se a matriz for iterada por referência, as iterações posteriores serão afetadas por alterações na matriz. Caso contrário, as alterações na matriz não afetarão as iterações posteriores (como se você estivesse iterando uma cópia da matriz). Compare o loop por valor:

$array = [0 => 1, 2 => 3, 4 => 5, 6 => 7];
foreach ($array as $key => $value) {
    if ($key === 0) {
        $array[6] = 17;
        unset($array[4]);
    }
    echo "$key => $value\n";
}

Resultado:

0 => 1
2 => 3
4 => 5
6 => 7

Mas se a matriz for iterada com referência,

$array = [0 => 1, 2 => 3, 4 => 5, 6 => 7];
foreach ($array as $key => &$value) {
    if ($key === 0) {
        $array[6] = 17;
        unset($array[4]);
    }
    echo "$key => $value\n";
}

Resultado:

0 => 1
2 => 3
6 => 17

O conjunto de valores-chave de 4 => 5 não é mais iterado e 6 => 7 é alterado para 6 => 17.

Usando ArrayObject Iterator #

Php arrayiterator permite que você modifique e desative os valores enquanto itera sobre arrays e objetos.

Exemplo:

$array = ['1' => 'apple', '2' => 'banana', '3' => 'cherry'];

$arrayObject = new ArrayObject($array);

$iterator = $arrayObject->getIterator();

for($iterator; $iterator->valid(); $iterator->next()) {
    echo $iterator->key() . ' => ' . $iterator->current() . "</br>";
}

Resultado:

1 => apple
2 => banana
3 => cherry
