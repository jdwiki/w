A programação funcional do PHP depende de funções. Funções em PHP fornecem código organizado e reutilizável para executar um conjunto de ações. As funções simplificam o processo de codificação, evitam lógica redundante e tornam o código mais fácil de seguir. Este tópico descreve a declaração e utilização de funções, argumentos, parâmetros, declarações de retorno e escopo em PHP.

Fechamentos #

Um encerramento é uma função anônima que não pode acessar fora do escopo.

Ao definir uma função anônima como tal, você está criando um “namespace” para essa função. Atualmente, ele só tem acesso a esse namespace.

$externalVariable = "Hello";
$secondExternalVariable = "Foo"; 
$myFunction = function() { 
  
  var_dump($externalVariable, $secondExternalVariable); // returns two error notice, since the variables aren´t defined 

}

Ele não tem acesso a nenhuma variável externa. Para conceder esta permissão para este namespace acessar variáveis ​​externas, você precisa introduzi-lo através de closures (use()).

$myFunction = function() use($externalVariable, $secondExternalVariable) {
   var_dump($externalVariable, $secondExternalVariable); // Hello Foo
}

Isso é fortemente atribuído ao escopo de variável restrito do PHP - Se uma variável não for definida dentro do escopo, ou não for trazida com global, então ela não existe.

Observe também:

Herdar variáveis ​​do escopo pai não é o mesmo que usar variáveis ​​globais. As variáveis ​​globais existem no escopo global, que é o mesmo, não importa qual função está sendo executada.

O escopo pai de um encerramento é a função na qual o encerramento foi declarado (não necessariamente a função de onde foi chamada).

Extraído da Documentação do PHP para funções anônimas

Em PHP, os closures usam uma abordagem early-binding. Isso significa que as variáveis ​​passadas para o namespace da closure usando a palavra-chave use terão os mesmos valores quando a closure foi definida.

Para alterar este comportamento você deve passar a variável por referência.

$rate = .05;

// Exports variable to closure's scope
$calculateTax = function ($value) use ($rate) {
    return $value * $rate;
};

$rate = .1; 

print $calculateTax(100); // 5

$rate = .05;

// Exports variable to closure's scope
$calculateTax = function ($value) use (&$rate) { // notice the & before $rate
    return $value * $rate;
};

$rate = .1;

print $calculateTax(100); // 10

Os argumentos padrão não são necessários implicitamente ao definir funções anônimas com/sem encerramentos.

$message = 'Im yelling at you';

$yell = function() use($message) {
    echo strtoupper($message);
};

$yell(); // returns: IM YELLING AT YOU

Atribuição a variáveis #

Funções anônimas podem ser atribuídas a variáveis ​​para uso como parâmetros onde um retorno de chamada é esperado:

$uppercase = function($data) {
    return strtoupper($data);
};

$mixedCase = ["Hello", "World"];
$uppercased = array_map($uppercase, $mixedCase);
print_r($uppercased);

Essas variáveis ​​também podem ser usadas como chamadas de função autônomas:

echo $uppercase("Hello world!"); // HELLO WORLD!

Objetos como uma função #

class SomeClass {
    public function __invoke($param1, $param2) {
        // put your code here
    }
}

$instance = new SomeClass();
$instance('First', 'Second'); // call the __invoke() method

Um objeto com um método __invoke pode ser usado exatamente como qualquer outra função.

O método __invoke terá acesso a todas as propriedades do objeto e poderá chamar qualquer método.

Usando variáveis ​​externas #

A construção use é usada para importar variáveis ​​para o escopo da função anônima:

$divisor = 2332;
$myfunction = function($number) use ($divisor) {
    return $number / $divisor;
};

echo $myfunction(81620); //Outputs 35

As variáveis ​​também podem ser importadas por referência:

$collection = [];

$additem = function($item) use (&$collection) {
    $collection[] = $item;
};

$additem(1);
$additem(2);

//$collection is now [1,2]

Usando funções internas como retornos de chamada #

Em funções que usam callable como argumento, você também pode colocar uma string com a função embutida do PHP. É comum usar trim como parâmetro array_map para remover espaços em branco à esquerda e à direita de todas as strings no array.

$arr = ['   one  ', 'two   ', '   three'];
var_dump(array_map('trim', $arr));

// array(3) {
//   [0] =>
//   string(3) "one"
//   [1] =>
//   string(3) "two"
//   [2] =>
//   string(5) "three"
// }

Função anônima #

Uma função anônima é apenas uma função que não tem nome.

// Anonymous function
function() {
    return "Hello World!";
};

Em PHP, uma função anônima é tratada como uma expressão e, por isso, deve ser finalizada com um ponto e vírgula ;.

Uma função anônima deve ser atribuída a uma variável.

// Anonymous function assigned to a variable
$sayHello = function($name) {
    return "Hello $name!";
};

print $sayHello('John'); // Hello John

Ou deve ser passado como parâmetro de outra função.

$users = [
    ['name' => 'Alice', 'age' => 20], 
    ['name' => 'Bobby', 'age' => 22], 
    ['name' => 'Carol', 'age' => 17]
];

// Map function applying anonymous function
$userName = array_map(function($user) {
    return $user['name'];
}, $users);

print_r($usersName); // ['Alice', 'Bobby', 'Carol']

Ou até mesmo foi retornado de outra função.

Funções anônimas auto-executáveis:

// For PHP 7.x
(function () {
    echo "Hello world!";
})();

// For PHP 5.x
call_user_func(function () {
    echo "Hello world!";
});

Passando um argumento para funções anônimas auto-executáveis:

// For PHP 7.x
(function ($name) {
    echo "Hello $name!";
})('John');

// For PHP 5.x
call_user_func(function ($name) {
    echo "Hello $name!";
}, 'John');

Funções puras #

Uma função pura é uma função que, dada a mesma entrada, sempre retornará a mesma saída e é livre de efeitos colaterais.

// This is a pure function
function add($a, $b) {
    return $a + $b;
}

Alguns efeitos colaterais são alterando o sistema de arquivos, interagindo com bancos de dados, imprimindo na tela.

// This is an impure function
function add($a, $b) {
    echo "Adding...";
    return $a + $b;
}

Métodos funcionais comuns em PHP #

Mapeamento #

Aplicando uma função a todos os elementos de um array:

array_map('strtoupper', $array);

Esteja ciente de que este é o único método da lista em que o retorno de chamada vem primeiro.

Reduzir (ou dobrar) #

Reduzindo um array para um único valor:

$sum = array_reduce($numbers, function ($carry, $number) {
   return $carry + $number;
});

Filtragem #

Retorna apenas os itens da matriz para os quais o retorno de chamada retorna true:

$onlyEven = array_filter($numbers, function ($number) {
    return ($number % 2) === 0;
});

Alcance #

Em PHP, uma função anônima tem seu próprio escopo como qualquer outra função PHP.

Em JavaScript, uma função anônima pode acessar uma variável fora do escopo. Mas em PHP, isso não é permitido.

$name = 'John';

// Anonymous function trying access outside scope
$sayHello = function() {
    return "Hello $name!";
}

print $sayHello('John'); // Hello !
// With notices active, there is also an Undefined variable $name notice

Passando uma função de callback como parâmetro #

Existem várias funções PHP que aceitam funções de retorno de chamada definidas pelo usuário como parâmetro, como: call_user_func(), usort() e [array_map()](https://secure.php.net/manual /en/function.array-map.php).

Dependendo de onde a função de retorno de chamada definida pelo usuário foi definida, existem diferentes maneiras de passá-las:

Estilo de procedimento: #

function square($number)
{
    return $number * $number;
}

$initial_array = [1, 2, 3, 4, 5];
$final_array = array_map('square', $initial_array);
var_dump($final_array); // prints the new array with 1, 4, 9, 16, 25

Estilo Orientado a Objetos: #

class SquareHolder
{
    function square($number)
    {
        return $number * $number;
    }
}

$squaredHolder = new SquareHolder();
$initial_array = [1, 2, 3, 4, 5];
$final_array = array_map([$squaredHolder, 'square'], $initial_array);

var_dump($final_array); // prints the new array with 1, 4, 9, 16, 25

Estilo Orientado a Objetos usando um método estático: #

class StaticSquareHolder
{
    public static function square($number)
    {
        return $number * $number;
    }
}

$initial_array = [1, 2, 3, 4, 5];
$final_array = array_map(['StaticSquareHolder', 'square'], $initial_array);
// or:
$final_array = array_map('StaticSquareHolder::square', $initial_array); // for PHP >= 5.2.3

var_dump($final_array); // prints the new array with 1, 4, 9, 16, 25
