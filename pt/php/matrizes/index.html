Um array é uma estrutura de dados que armazena um número arbitrário de valores em um único valor. Um array em PHP é na verdade um mapa ordenado, onde map é um tipo que associa valores a chaves.

Sintaxe #

  • $array = array(‘Valor1’, ‘Valor2’, ‘Valor3’); // Chaves padrão para 0, 1, 2, …,
  • $array = array(‘Valor1’, ‘Valor2’, ); // Vírgula final opcional
  • $array = array(‘chave1’ => ‘Valor1’, ‘chave2’ => ‘Valor2’, ); // Chaves explícitas
  • $array = array(‘chave1’ => ‘Valor1’, ‘Valor2’, ); // Array ( [‘key1’] => Valor1 1 => ‘Valor2’)
  • $array = [‘chave1’ => ‘Valor1’, ‘chave2’ => ‘Valor2’, ]; // PHP 5.4+ abreviação
  • $array[] = ‘ValorX’; // Anexa ‘ValueX’ ao final do array
  • $array[‘keyX’] = ‘ValueX’; // Atribui ‘valueX’ à chave ‘keyX’
  • $array += [‘chaveX’ => ‘valorX’, ‘chaveY’ => ‘valorY’]; // Adicionando/Substituindo elementos em um array existente

Parâmetros #

Parâmetro Detalhe
Chave A chave é o identificador exclusivo e o índice de uma matriz. Pode ser uma string ou um integer. Portanto, chaves válidas seriam 'foo', '5', 10, 'a2b', ...
Valor Para cada key existe um valor correspondente (null caso contrário e um aviso é emitido no acesso). O valor não tem restrições no tipo de entrada.

Inicializando um Array #

Um array pode ser inicializado vazio:

// An empty array
$foo = array();

// Shorthand notation available since PHP 5.4
$foo = [];

Um array pode ser inicializado e predefinido com valores:

// Creates a simple array with three strings
$fruit = array('apples', 'pears', 'oranges');

// Shorthand notation available since PHP 5.4
$fruit = ['apples', 'pears', 'oranges'];

Um array também pode ser inicializado com índices personalizados (também chamado de array associativo):

// A simple associative array
$fruit = array(
   'first'  => 'apples',
   'second' => 'pears', 
   'third'  => 'oranges'
);

// Key and value can also be set as follows
$fruit['first'] = 'apples';

// Shorthand notation available since PHP 5.4
$fruit = [
    'first'  => 'apples', 
    'second' => 'pears', 
    'third'  => 'oranges'
];

Se a variável não foi usada antes, o PHP a criará automaticamente. Embora conveniente, isso pode dificultar a leitura do código:

$foo[] = 1;     // Array( [0] => 1 )
$bar[][] = 2;   // Array( [0] => Array( [0] => 2 ) )

O índice geralmente continuará de onde você parou. O PHP tentará usar strings numéricas como números inteiros:

$foo = [2 => 'apple', 'melon'];  // Array( [2] => apple, [3] => melon )
$foo = ['2' => 'apple', 'melon']; // same as above
$foo = [2 => 'apple', 'this is index 3 temporarily', '3' => 'melon']; // same as above! The last entry will overwrite the second!

Para inicializar um array com tamanho fixo você pode usar SplFixedArray:

$array = new SplFixedArray(3);

$array[0] = 1;
$array[1] = 2;
$array[2] = 3;
$array[3] = 4; // RuntimeException

// Increase the size of the array to 10
$array->setSize(10);

Nota: Um array criado usando SplFixedArray tem um consumo de memória reduzido para grandes conjuntos de dados, mas as chaves devem ser números inteiros.


Para inicializar um array com um tamanho dinâmico, mas com elementos n não vazios (por exemplo, um espaço reservado), você pode usar um loop da seguinte forma:

$myArray = array();
$sizeOfMyArray = 5;
$fill = 'placeholder';

for ($i = 0; $i < $sizeOfMyArray; $i++) {
    $myArray[] = $fill;
}

// print_r($myArray); results in the following:
// Array ( [0] => placeholder [1] => placeholder [2] => placeholder [3] => placeholder [4] => placeholder )

Se todos os seus espaços reservados forem iguais, você também poderá criá-lo usando a função array_fill():

No PHP 5.5 e posterior você pode usar array_column() em conjunto com array_search().

Isso é particularmente útil para verificar se existe um valor em uma matriz associativa:

$userdb = [
    [
        "uid" => '100',
        "name" => 'Sandra Shush',
        "url" => 'urlof100',
    ],
    [
        "uid" => '5465',
        "name" => 'Stefanie Mcmohn',
        "pic_square" => 'urlof100',
    ],
    [
        "uid" => '40489',
        "name" => 'Michael',
        "pic_square" => 'urlof40489',
    ]
];

$key = array_search(40489, array_column($userdb, 'uid'));

Interfaces ArrayAccess e Iterator #

Outro recurso útil é acessar suas coleções de objetos personalizados como arrays em PHP. Existem duas interfaces disponíveis no núcleo do PHP (>=5.0.0) para suportar isso: ArrayAccess e Iterator. O primeiro permite que você acesse seus objetos personalizados como array.

Acesso à matriz

Suponha que temos uma classe de usuário e uma tabela de banco de dados armazenando todos os usuários. Gostaríamos de criar uma classe UserCollection que irá:

  1. permitir-nos abordar determinado usuário por seu identificador exclusivo de nome de usuário
  2. realizar operações básicas (nem todas CRUD, mas pelo menos Criar, Recuperar e Excluir) em nossa coleção de usuários

Considere a seguinte fonte (daqui em diante estamos usando a sintaxe de criação de array curta [] disponível desde a versão 5.4):

class UserCollection implements ArrayAccess {
    protected $_conn;
    
    protected $_requiredParams = ['username','password','email'];
    
    public function __construct() {
        $config = new Configuration();

        $connectionParams = [
            //your connection to the database
        ];
        
        $this->_conn = DriverManager::getConnection($connectionParams, $config);
    }
    
    protected function _getByUsername($username) {
        $ret = $this->_conn->executeQuery('SELECT * FROM `User` WHERE `username` IN (?)',
            [$username]
        )->fetch();
        
        return $ret;
    }
    
    // START of methods required by ArrayAccess interface
    public function offsetExists($offset) {
        return (bool) $this->_getByUsername($offset);
    }

    public function offsetGet($offset) {
        return $this->_getByUsername($offset);
    }

    public function offsetSet($offset, $value) {
        if (!is_array($value)) {
            throw new \Exception('value must be an Array');
        }

        $passed = array_intersect(array_values($this->_requiredParams), array_keys($value));
        if (count($passed) < count($this->_requiredParams)) {
            throw new \Exception('value must contain at least the following params: ' . implode(',', $this->_requiredParams));
        }
        $this->_conn->insert('User', $value);
    }

    public function offsetUnset($offset) {
        if (!is_string($offset)) {
            throw new \Exception('value must be the username to delete');
        }
        if (!$this->offsetGet($offset)) {
            throw new \Exception('user not found');
        }
        $this->_conn->delete('User', ['username' => $offset]);
    }
    // END of methods required by ArrayAccess interface
}

então nós podemos :

$users = new UserCollection();

var_dump(empty($users['testuser']),isset($users['testuser']));
$users['testuser'] = ['username' => 'testuser', 
                      'password' => 'testpassword',
                      'email'    => 'test@test.com'];
var_dump(empty($users['testuser']), isset($users['testuser']), $users['testuser']);
unset($users['testuser']);
var_dump(empty($users['testuser']), isset($users['testuser']));

que produzirá o seguinte, assumindo que não havia testuser antes de lançarmos o código:

bool(true)
bool(false)
bool(false)
bool(true)
array(17) {
  ["username"]=>
  string(8) "testuser"
  ["password"]=>
  string(12) "testpassword"
  ["email"]=>
  string(13) "test@test.com"
}
bool(true)
bool(false)

IMPORTANTE: offsetExists não é chamado quando você verifica a existência de uma chave com a função array_key_exists. Portanto, o código a seguir produzirá false duas vezes:

var_dump(array_key_exists('testuser', $users));
$users['testuser'] = ['username' => 'testuser', 
                      'password' => 'testpassword',
                      'email'    => 'test@test.com'];
var_dump(array_key_exists('testuser', $users));

Iterador

Vamos estender nossa classe acima com algumas funções da interface Iterator para permitir iterar sobre ela com foreach e while.

Primeiro, precisamos adicionar uma propriedade contendo nosso índice atual de iterador, vamos adicioná-la às propriedades da classe como $_position:

// iterator current position, required by Iterator interface methods
protected $_position = 1;

Segundo, vamos adicionar a interface Iterator à lista de interfaces que estão sendo implementadas pela nossa classe:

class UserCollection implements ArrayAccess, Iterator {

em seguida, adicione o necessário pelas próprias funções da interface:

// START of methods required by Iterator interface
public function current () {
    return $this->_getById($this->_position);
}
public function key () {
    return $this->_position;
}
public function next () {
    $this->_position++;
}
public function rewind () {
    $this->_position = 1;
}
public function valid () {
    return null !== $this->_getById($this->_position);
}
// END of methods required by Iterator interface

Portanto, aqui está a fonte completa da classe que implementa ambas as interfaces. Observe que este exemplo não é perfeito, porque os IDs no banco de dados podem não ser sequenciais, mas isso foi escrito apenas para dar a ideia principal: você pode endereçar suas coleções de objetos de qualquer maneira possível implementando ArrayAccess e Iterator interfaces:

class UserCollection implements ArrayAccess, Iterator {
    // iterator current position, required by Iterator interface methods
    protected $_position = 1;
    
    // <add the old methods from the last code snippet here>
    
    // START of methods required by Iterator interface
    public function current () {
        return $this->_getById($this->_position);
    }
    public function key () {
        return $this->_position;
    }
    public function next () {
        $this->_position++;
    }
    public function rewind () {
        $this->_position = 1;
    }
    public function valid () {
        return null !== $this->_getById($this->_position);
    }
    // END of methods required by Iterator interface
}

e um foreach percorrendo todos os objetos de usuário:

foreach ($users as $user) {
    var_dump($user['id']);
}

que irá produzir algo como

string(2) "1"
string(2) "2"
string(2) "3"
string(2) "4"
...
