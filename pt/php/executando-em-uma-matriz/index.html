Aplicando uma função a cada elemento de um array #

Para aplicar uma função a cada item em um array, use array_map(). Isso retornará uma nova matriz.

$array = array(1,2,3,4,5);
//each array item is iterated over and gets stored in the function parameter.
$newArray = array_map(function($item) {
    return $item + 1;
}, $array);

$newArray agora é array(2,3,4,5,6);.

Em vez de usar uma função anônima, você pode usar uma função nomeada. O acima poderia ser escrito como:

function addOne($item) {
    return $item + 1;
}

$array = array(1, 2, 3, 4, 5);
$newArray = array_map('addOne', $array);

Se a função nomeada for um método de classe, a chamada da função deve incluir uma referência a um objeto de classe ao qual o método pertence:

class Example {
    public function addOne($item) {
        return $item + 1;
    }

    public function doCalculation() {
        $array = array(1, 2, 3, 4, 5);
        $newArray = array_map(array($this, 'addOne'), $array);
    }
}

Outra maneira de aplicar uma função a cada item em um array é array_walk() e array_walk_recursive(). O retorno de chamada passado para essas funções recebe a chave/índice e o valor de cada item da matriz. Essas funções não retornarão um novo array, mas sim um booleano para o sucesso. Por exemplo, para imprimir cada elemento em um array simples:

$array = array(1, 2, 3, 4, 5);
array_walk($array, function($value, $key) {
    echo $value . ' ';
});
// prints "1 2 3 4 5"

O parâmetro value do callback pode ser passado por referência, permitindo que você altere o valor diretamente no array original:

$array = array(1, 2, 3, 4, 5);
array_walk($array, function(&$value, $key) {
    $value++;
});

$array agora é array(2,3,4,5,6);

Para arrays aninhados, array_walk_recursive() irá aprofundar cada sub-array:

$array = array(1, array(2, 3, array(4, 5), 6);
array_walk_recursive($array, function($value, $key) {
    echo $value . ' ';
});
// prints "1 2 3 4 5 6"

Nota: array_walk e array_walk_recursive permitem que você altere o valor dos itens do array, mas não as chaves. Passar as chaves por referência para o retorno de chamada é válido, mas não tem efeito.

Dividir array em pedaços #

array_chunk() divide um array em pedaços

Digamos que seguimos uma matriz unidimensional,

$input_array = array('a', 'b', 'c', 'd', 'e');

Agora usando array_chunk() no array PHP acima,

$output_array = array_chunk($input_array, 2);

O código acima fará pedaços de 2 elementos de array e criará um array multidimensional como segue.

Array
(
    [0] => Array
        (
            [0] => a
            [1] => b
        )

    [1] => Array
        (
            [0] => c
            [1] => d
        )

    [2] => Array
        (
            [0] => e
        )

)

Se todos os elementos da matriz não forem divididos uniformemente pelo tamanho do bloco, o último elemento da matriz de saída serão os elementos restantes.

Se passarmos o segundo argumento como menor que 1, E_WARNING será lançado e o array de saída será NULL.

Parâmetro Detalhes
$array (matriz) Matriz de entrada, a matriz para trabalhar
$tamanho (int) Tamanho de cada pedaço (valor inteiro)
$preserve_keys (booleano) (opcional) Se você quiser que a matriz de saída preserve as chaves, defina-a como TRUE, caso contrário, FALSE.

Implodindo um array em string #

implode() combina todos os valores do array, mas perde todas as informações principais:

$arr = ['a' => "AA", 'b' => "BB", 'c' => "CC"];

echo implode(" ", $arr); // AA BB CC

A implosão de chaves pode ser feita usando a chamada array_keys():

$arr = ['a' => "AA", 'b' => "BB", 'c' => "CC"];

echo implode(" ", array_keys($arr)); // a b c

Implodir chaves com valores é mais complexo, mas pode ser feito usando o estilo funcional:

$arr = ['a' => "AA", 'b' => "BB", 'c' => "CC"];

echo implode(" ", array_map(function($key, $val) { 
    return "$key:$val"; // function that glues key to the value
}, array_keys($arr), $arr)); 

// Output: a:AA b:BB c:CC

array_reduce #

array_reduce reduces array into a single value. Basically, The array_reduce will go through every item with the result from last iteration and produce new value to the next iteration.

Usage: array_reduce ($array, function($carry, $item){...}, $defaul_value_of_first_carry)

  • $carry é o resultado da última rodada de iteração.
  • $item é o valor da posição atual no array.

Soma da matriz

$result = array_reduce([1, 2, 3, 4, 5], function($carry, $item){
    return $carry + $item;
});

resultado:15

O maior número na matriz

$result = array_reduce([10, 23, 211, 34, 25], function($carry, $item){
        return $item > $carry ? $item : $carry;
});

resultado:211

Todos os itens têm mais de 100

$result = array_reduce([101, 230, 210, 341, 251], function($carry, $item){
        return $carry && $item > 100;
}, true); //default value must set true

resultado:true

Qualquer item tem menos de 100

$result = array_reduce([101, 230, 21, 341, 251], function($carry, $item){
        return $carry || $item < 100;
}, false);//default value must set false

resultado:true

Como implodir($array, $piece)

$result = array_reduce(["hello", "world", "PHP", "language"], function($carry, $item){
        return !$carry ? $item : $carry . "-" . $item ;
});

resultado:"hello-world-PHP-language"

se fizer um método implode, o código fonte será:

function implode_method($array, $piece){
    return array_reduce($array, function($carry, $item) use ($piece) {
            return !$carry ? $item : ($carry . $piece . $item);
    });
}

$result = implode_method(["hello", "world", "PHP", "language"], "-");

resultado:"hello-world-PHP-language"

“Desestruturando” arrays usando list() #

Use list() para atribuir rapidamente uma lista de valores de variáveis ​​a um array. Veja também [compact()][2]

// Assigns to $a, $b and $c the values of their respective array elements in           $array with keys numbered from zero
list($a, $b, $c) = $array;

Com o PHP 7.1 (atualmente em beta) você poderá usar [short list syntax][3]:

// Assigns to $a, $b and $c the values of their respective array elements in $array with keys numbered from zero
[$a, $b, $c] = $array;

// Assigns to $a, $b and $c the values of the array elements in $array with the keys "a", "b" and "c", respectively
["a" => $a, "b" => $b, "c" => $c] = $array;

[2]: https://www.wikiod.com/pt/php/matrizes#Criando um array de variáveis [3]: https://wiki.php.net/rfc/short_list_syntax

Empurre um valor em uma matriz #

Existem duas maneiras de enviar um elemento para um array: array_push e $array[] =

O array_push é usado assim:

$array = [1,2,3];
$newArraySize = array_push($array, 5, 6); // The method returns the new size of the array
print_r($array); // Array is passed by reference, therefore the original array is modified to contain the new elements

Este código irá imprimir:

Array
(
    [0] => 1
    [1] => 2
    [2] => 3
    [3] => 5
    [4] => 6
)

$array[] = é usado assim:

$array = [1,2,3];
$array[] = 5;
$array[] = 6;
print_r($array);

Este código irá imprimir:

Array
(
    [0] => 1
    [1] => 2
    [2] => 3
    [3] => 5
    [4] => 6
)
