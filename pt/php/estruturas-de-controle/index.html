se mais #

A instrução if no exemplo acima permite executar um fragmento de código, quando a condição for atendida. Quando você deseja executar um fragmento de código, quando a condição não for atendida, você estende o if com um else.

if ($a > $b) {
  echo "a is greater than b";
} else {
  echo "a is NOT greater than b";
}

Manual PHP - Estruturas de Controle - Senão

O operador ternário como sintaxe abreviada para if-else

O operador ternário avalia algo baseado em uma condição ser verdadeira ou não. É um operador de comparação e frequentemente usado para expressar uma condição if-else simples em uma forma mais curta. Ele permite testar rapidamente uma condição e muitas vezes substitui uma instrução if de várias linhas, tornando seu código mais compacto.

Este é o exemplo acima usando uma expressão ternária e valores de variáveis: $a=1; $b=2;

echo ($a > $b) ? "a is greater than b" : "a is NOT greater than b";

Saídas: a NÃO é maior que b.

Sintaxe alternativa para estruturas de controle #

O PHP fornece uma sintaxe alternativa para algumas estruturas de controle: if, while, for, foreach e switch.

Quando comparada com a sintaxe normal, a diferença é que a chave de abertura é substituída por dois pontos (:) e a chave de fechamento é substituída por endif;, endwhile;, endfor;, endforeach; , ou endswitch;, respectivamente. Para exemplos individuais, consulte o tópico em sintaxe alternativa para estruturas de controle.

if ($a == 42):
    echo "The answer to life, the universe and everything is 42.";
endif;

Várias instruções elseif usando sintaxe curta:

if ($a == 5):
    echo "a equals 5";
elseif ($a == 6):
    echo "a equals 6";
else:
    echo "a is neither 5 nor 6";
endif;

Manual PHP - Estruturas de Controle - Sintaxe Alternativa

enquanto #

O loop while itera através de um bloco de código enquanto uma condição especificada for verdadeira.

$i = 1;
while ($i < 10) {
    echo $i;
    $i++;
}

Saída: 123456789

Para obter informações detalhadas, consulte o tópico Loops.

fazer enquanto #

O loop do-while primeiro executa um bloco de código uma vez, em todos os casos, depois itera por esse bloco de código enquanto uma condição especificada for verdadeira.

$i = 0;
do {
    $i++;
    echo $i;
} while ($i < 10);

Output: `12345678910`

Para obter informações detalhadas, consulte [o tópico Loops](https://www.wikiod.com/pt/php/rotacoes#fazer enquanto).

Vá para #

O operador goto permite pular para outra seção do programa. Está disponível desde o PHP 5.3.

A instrução goto é um goto seguido pelo rótulo de destino desejado: goto MyLabel;.

O destino do salto é especificado por um rótulo seguido por dois pontos: MyLabel:.

Este exemplo imprimirá Hello World!:

<?php
goto MyLabel;
echo 'This text will be skipped, because of the jump.';
 
MyLabel:
echo 'Hello World!';
?>

declarar #

declare é usado para definir uma diretiva de execução para um bloco de código.

As seguintes diretivas são reconhecidas:

  • tiques
  • codificação
  • strict_types

Por exemplo, defina ticks para 1:

declare(ticks=1);

Para habilitar o modo de tipo estrito, a instrução declare é usada com a declaração strict_types:

declare(strict_types=1);

incluir e exigir #

exigir #

require é semelhante a include, exceto que produzirá um erro fatal de nível E_COMPILE_ERROR em caso de falha. Quando o require falhar, ele interromperá o script. Quando o include falha, ele não irá parar o script e apenas emitirá E_WARNING.

require 'file.php';

Manual PHP - Estruturas de Controle - Exigir

incluir #

A instrução include inclui e avalia um arquivo.

./variables.php

$a = 'Hello World!';

./main.php`

include 'variables.php';
echo $a;
// Output: `Hello World!`

Tenha cuidado com essa abordagem, pois ela é considerada um code smell, pois o arquivo incluído está alterando a quantidade e o conteúdo das variáveis ​​definidas no escopo fornecido.

Você também pode incluir arquivo, que retorna um valor. Isso é extremamente útil para lidar com arrays de configuração:

configuração.php

<?php 
return [
    'dbname' => 'my db',
    'user' => 'admin',
    'pass' => 'password',
];

main.php

<?php
$config = include 'configuration.php';

Essa abordagem impedirá que o arquivo incluído polua seu escopo atual com variáveis ​​alteradas ou adicionadas.

Manual PHP - Estruturas de Controle - Incluir

include & require também pode ser usado para atribuir valores a uma variável quando retornado algo por arquivo.

Exemplo :

arquivo include1.php:

<?php
    $a = "This is to be returned";

    return $a;
?>

arquivo index.php:

    $value = include 'include1.php';
   // Here, $value = "This is to be returned"

Retorna #

A instrução return retorna o controle do programa para a função de chamada.

Quando return for chamado de dentro de uma função, a execução da função atual terminará.

function returnEndsFunctions()
{
   echo 'This is executed';
   return;
   echo 'This is not executed.';
}

Ao executar returnEndsFunctions(); você obterá a saída Isto é executado;

Quando return for chamado de dentro de uma função com um argumento, a execução da função atual terminará e o valor do argumento será retornado para a função de chamada.

por #

Os loops for são normalmente usados ​​quando você tem um pedaço de código que deseja repetir um determinado número de vezes.

for ($i = 1; $i < 10; $i++) {
    echo $i;
}

Saídas: 123456789

Para obter informações detalhadas, consulte o tópico Loops.

para cada #

foreach é uma construção, que permite iterar sobre arrays e objetos facilmente.

$array = [1, 2, 3];
foreach ($array as $value) {
    echo $value;
}

Saídas: 123.

Para usar o loop foreach com um objeto, ele deve implementar a interface Iterator.

Quando você itera sobre matrizes associativas:

$array = ['color'=>'red']; 

foreach($array as $key => $value){
    echo $key . ': ' . $value; 
}

Saídas: cor: vermelho

Para obter informações detalhadas, consulte [o tópico Loops](https://www.wikiod.com/pt/php/rotacoes#para cada).

if elseif else #

outro

elseif combina if e else. A instrução if é estendida para executar uma instrução diferente caso a expressão if original não seja atendida. Mas, a expressão alternativa só é executada, quando a expressão condicional elseif for atendida.

O código a seguir exibe “a é maior que b”, “a é igual a b” ou “a é menor que b”:

if ($a > $b) {
    echo "a is bigger than b";
} elseif ($a == $b) {
    echo "a is equal to b";
} else {
    echo "a is smaller than b";
}

Várias declarações elseif

Você pode usar várias instruções elseif dentro da mesma instrução if:

if ($a == 1) {
    echo "a is One";
} elseif ($a == 2) {
    echo "a is Two";
} elseif ($a == 3) {
    echo "a is Three";
} else {
    echo "a is not One, not Two nor Three";
}

E se #

A construção if permite a execução condicional de fragmentos de código.

if ($a > $b) {
  echo "a is bigger than b";
}

Manual PHP - Estruturas de Controle - Se

trocar #

A estrutura switch executa a mesma função que uma série de instruções if, mas pode fazer o trabalho em menos linhas de código. O valor a ser testado, conforme definido na instrução switch, é comparado quanto à igualdade com os valores em cada uma das instruções case até que uma correspondência seja encontrada e o código nesse bloco seja executado. Se nenhuma instrução case correspondente for encontrada, o código no bloco default será executado, se existir.

Cada bloco de código em uma instrução case ou default deve terminar com a instrução break. Isso interrompe a execução da estrutura switch e continua a execução do código imediatamente depois. Se a instrução break for omitida, o código da próxima instrução case será executado, mesmo que não haja correspondência. Isso pode causar execução inesperada de código se a instrução break for esquecida, mas também pode ser útil quando várias instruções case precisam compartilhar o mesmo código.

switch ($colour) {
case "red":
    echo "the colour is red";
    break;
case "green":
case "blue":
    echo "the colour is green or blue";
    break;
case "yellow":
    echo "the colour is yellow";
    // note missing break, the next block will also be executed
case "black":
    echo "the colour is black";
    break;
default:
    echo "the colour is something else";
    break;
}

Além de testar valores fixos, a construção também pode ser forçada a testar instruções dinâmicas fornecendo um valor booleano para a instrução switch e qualquer expressão para a instrução case. Lembre-se de que o primeiro valor correspondente é usado, portanto, o código a seguir produzirá “mais de 100”:

$i = 1048;
switch (true) {
case ($i > 0):
    echo "more than 0";
    break;
case ($i > 100):
    echo "more than 100";
    break;
case ($i > 1000):
    echo "more than 1000";
    break;
}

Para possíveis problemas com digitação solta ao usar a construção switch, consulte Switch Surprises
