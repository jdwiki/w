getTimestamp #

getTimeStemp é uma representação unix de um objeto datetime.

$date = new DateTime();
echo $date->getTimestamp();

isso colocará uma indicação de número inteiro os segundos decorridos desde 00:00:00 UTC, quinta-feira, 1º de janeiro de 1970.

setData #

setDate define a data em um objeto DateTime.

$date = new DateTime();
$date->setDate(2016, 7, 25);

este exemplo define a data como 25 de julho de 2015, produzirá o seguinte resultado:

2016-07-25 17:52:15.819442

Adicionar ou subtrair intervalos de data #

Podemos usar a classe DateInterval para adicionar ou subtrair algum intervalo em um objeto DateTime.

Veja o exemplo abaixo, onde estamos adicionando um intervalo de 7 dias e imprimindo uma mensagem na tela:

$now = new DateTime();// empty argument returns the current date
$interval = new DateInterval('P7D');//this objet represents a 7 days interval
$lastDay = $now->add($interval); //this will return a DateTime object
$formatedLastDay = $lastDay->format('Y-m-d');//this method format the DateTime object and returns a String
echo "Samara says: Seven Days. You'll be happy on $formatedLastDay.";

Isso resultará (em execução em 1º de agosto de 2016):

Samara diz: Sete Dias. Você será feliz em 2016-08-08.

Podemos usar o método sub de maneira semelhante para subtrair datas

$now->sub($interval);
echo "Samara says: Seven Days. You were happy last on $formatedLastDay.";

Isso resultará (em execução em 1º de agosto de 2016):

Samara diz: Sete Dias. Você foi feliz pela última vez em 25/07/2016.

Crie DateTime a partir do formato personalizado #

O PHP é capaz de analisar vários formatos de data. Se você deseja analisar um formato não padrão ou se deseja que seu código indique explicitamente o formato a ser usado, você pode usar o estático [DateTime::createFromFormat](https://php.net/manual /en/datetime.createfromformat.php):

Estilo orientado a objetos

$format = "Y,m,d";
$time = "2009,2,26";
$date = DateTime::createFromFormat($format, $time);

Estilo de procedimento

$format = "Y,m,d";
$time = "2009,2,26";
$date = date_create_from_format($format, $time);

Imprimindo DateTimes #

O PHP 4+ fornece um método, formato que converte um objeto DateTime em uma string com o formato desejado. De acordo com o PHP Manual, esta é a função orientada a objetos:

public string DateTime::format ( string $format )

A função date() recebe um parâmetro - um formato, que é uma string

Formato #

O formato é uma string e usa caracteres únicos para definir o formato:

  • Y: representação de quatro dígitos do ano (por exemplo: 2016)
  • y: representação de dois dígitos do ano (por exemplo: 16)
  • m: mês, como um número (01 a 12)
  • M: mês, como três letras (Jan, Fev, Mar, etc)
  • j: dia do mês, sem zeros à esquerda (1 a 31)
  • D: dia da semana, como três letras (Seg, Ter, Quarta, etc)
  • h: hora (formato de 12 horas) (01 a 12)
  • H: hora (formato de 24 horas) (00 a 23)
  • A: AM ou PM
  • i: minuto, com zeros à esquerda (00 a 59)
  • s: segundo, com zeros à esquerda (00 a 59)
  • A lista completa pode ser encontrada aqui

Uso #

Esses caracteres podem ser usados ​​em várias combinações para exibir as horas em praticamente qualquer formato. aqui estão alguns exemplos:

$date = new DateTime('2000-05-26T13:30:20'); /* Friday, May 26, 2000 at 1:30:20 PM */

$date->format("H:i");
/* Returns 13:30 */

$date->format("H i s");
/* Returns 13 30 20 */

$date->format("h:i:s A");
/* Returns 01:30:20 PM */

$date->format("j/m/Y");
/* Returns 26/05/2000 */

$date->format("D, M j 'y - h:i A");
/* Returns Fri, May 26 '00 - 01:30 PM */

Processual #

O formato processual é semelhante:

Orientado a Objeto #

$date->format($format)

Processual Equivalente #

date_format($date, $format)

Crie uma versão imutável de DateTime a partir de Mutable anterior ao PHP 5.6 #

Para criar \DateTimeImmutable no PHP 5.6+ use:

\DateTimeImmutable::createFromMutable($concrete);

Antes do PHP 5.6 você pode usar:

\DateTimeImmutable::createFromFormat(\DateTime::ISO8601, $mutable->format(\DateTime::ISO8601), $mutable->getTimezone());
