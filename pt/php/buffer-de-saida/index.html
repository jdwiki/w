Parâmetros #

Função Detalhes
ob_start() Inicia o buffer de saída, qualquer saída colocada após isso será capturada e não exibida
ob_get_contents() Retorna todo o conteúdo capturado por ob_start()
ob_end_clean() Esvazia o buffer de saída e o desativa para o nível de aninhamento atual
ob_get_clean() Aciona ob_get_contents() e ob_end_clean()
ob_get_level() Retorna o nível de aninhamento atual do buffer de saída
ob_flush() Esvazie o buffer de conteúdo e envie-o para o navegador sem terminar o buffer
ob_implicit_flush() Habilita a liberação implícita após cada chamada de saída.
ob_end_flush() Esvazie o buffer de conteúdo e envie para o navegador também finalizando o buffer

Uso básico obtendo conteúdo entre buffers e limpando #

O buffer de saída permite que você armazene qualquer conteúdo textual (Texto, HTML) em uma variável e envie para o navegador como uma peça no final do seu script. Por padrão, o php envia seu conteúdo conforme o interpreta.

<?php

// Turn on output buffering
ob_start();

// Print some output to the buffer (via php)
print 'Hello ';

// You can also `step out` of PHP
?>
<em>World</em>
<?php
// Return the buffer AND clear it
$content = ob_get_clean();

// Return our buffer and then clear it
# $content = ob_get_contents();
# $did_clear_buffer = ob_end_clean();

print($content);

#> "Hello <em>World</em>"

Qualquer conteúdo gerado entre ob_start() e ob_get_clean() será capturado e colocado na variável $content.

Chamar ob_get_clean() aciona ob_get_contents() e ob_end_clean().

Buffers de saída aninhados #

Você pode aninhar buffers de saída e buscar o nível para que eles forneçam conteúdo diferente usando a função ob_get_level().

<?php

$i = 1;
$output = null;

while( $i <= 5 ) {
    // Each loop, creates a new output buffering `level`
    ob_start();
    print "Current nest level: ". ob_get_level() . "\n";
    $i++;
}

// We're at level 5 now
print 'Ended up at level: ' . ob_get_level() . PHP_EOL;

// Get clean will `pop` the contents of the top most level (5)
$output .= ob_get_clean();
print $output;

print 'Popped level 5, so we now start from 4' . PHP_EOL;

// We're now at level 4 (we pop'ed off 5 above)

// For each level we went up, come back down and get the buffer
while( $i > 2 ) {
    print "Current nest level: " . ob_get_level() . "\n";
    echo ob_get_clean();
    $i--;
}

Saídas:

Current nest level: 1
Current nest level: 2
Current nest level: 3
Current nest level: 4
Current nest level: 5
Ended up at level: 5
Popped level 5, so we now start from 4
Current nest level: 4
Current nest level: 3
Current nest level: 2
Current nest level: 1

Executando buffer de saída antes de qualquer conteúdo #

ob_start();

$user_count = 0;
foreach( $users as $user ) {
    if( $user['access'] != 7 ) { continue; }
    ?>
    <li class="users user-<?php echo $user['id']; ?>">
        <a href="<?php echo $user['link']; ?>">
            <?php echo $user['name'] ?>
        </a>
    </li>
<?php
    $user_count++;
}
$users_html = ob_get_clean();

if( !$user_count ) {
    header('Location: /404.php');
    exit();
}
?>
<html>
<head>
    <title>Level 7 user results (<?php echo $user_count; ?>)</title>
</head>

<body>
<h2>We have a total of <?php echo $user_count; ?> users with access level 7</h2>
<ul class="user-list">
    <?php echo $users_html; ?>
</ul>
</body>
</html>

Neste exemplo, assumimos que $users é um array multidimensional, e fazemos um loop por ele para encontrar todos os usuários com nível de acesso 7.

Se não houver resultados, redirecionamos para uma página de erro.

Estamos usando o buffer de saída aqui porque estamos acionando um redirecionamento header() com base no resultado do loop

Processando o buffer por meio de um retorno de chamada #

Você pode aplicar qualquer tipo de processamento adicional à saída passando um callable para ob_start().

<?php
function clearAllWhiteSpace($buffer) {
    return str_replace(array("\n", "\t", ' '), '', $buffer);
}

ob_start('clearAllWhiteSpace');
?>
<h1>Lorem Ipsum</h1>

<p><strong>Pellentesque habitant morbi tristique</strong> senectus et netus et malesuada fames ac turpis egestas. <a href="#">Donec non enim</a> in turpis pulvinar facilisis.</p>

<h2>Header Level 2</h2>

<ol>
   <li>Lorem ipsum dolor sit amet, consectetuer adipiscing elit.</li>
   <li>Aliquam tincidunt mauris eu risus.</li>
</ol>

<?php
/* Output will be flushed and processed when script ends or call
     ob_end_flush();
*/

Usando buffer de saída para armazenar conteúdo em um arquivo, útil para relatórios, faturas etc #

<?php
ob_start();
?>
    <html>
    <head>
        <title>Example invoice</title>
    </head>
    <body>
    <h1>Invoice #0000</h1>
    <h2>Cost: &pound;15,000</h2>
    ...
    </body>
    </html>
<?php
$html = ob_get_clean();

$handle = fopen('invoices/example-invoice.html', 'w');
fwrite($handle, $html);
fclose($handle);

Este exemplo pega o documento completo e o grava no arquivo, ele não exibe o documento no navegador, mas o faz usando echo $html;

Saída de fluxo para o cliente #

/**
 * Enables output buffer streaming. Calling this function
 * immediately flushes the buffer to the client, and any
 * subsequent output will be sent directly to the client.
 */
function _stream() {
    ob_implicit_flush(true);
    ob_end_flush();
}

Capturando o buffer de saída para reutilizar mais tarde #

Neste exemplo, temos um array contendo alguns dados.

Capturamos o buffer de saída em $items_li_html e o usamos duas vezes na página.

<?php

// Start capturing the output
ob_start();

$items = ['Home', 'Blog', 'FAQ', 'Contact'];

foreach($items as $item):

// Note we're about to step "out of PHP land"
?>
  <li><?php echo $item ?></li>
<?php
// Back in PHP land
endforeach;

// $items_lists contains all the HTML captured by the output buffer
$items_li_html = ob_get_clean();
?>

<!-- Menu 1: We can now re-use that (multiple times if required) in our HTML. -->
<ul class="header-nav">
    <?php echo $items_li_html ?>
</ul>

<!-- Menu 2 -->
<ul class="footer-nav">
    <?php echo $items_li_html ?>
</ul>

Salve o código acima em um arquivo output_buffer.php e execute-o via php output_buffer.php.

Você deve ver os 2 itens de lista que criamos acima com os mesmos itens de lista que geramos em PHP usando o buffer de saída:

<!-- Menu 1: We can now re-use that (multiple times if required) in our HTML. -->
<ul class="header-nav">
  <li>Home</li>
  <li>Blog</li>
  <li>FAQ</li>
  <li>Contact</li>
</ul>

<!-- Menu 2 -->
<ul class="footer-nav">
  <li>Home</li>
  <li>Blog</li>
  <li>FAQ</li>
  <li>Contact</li>
</ul>

Uso típico e razões para usar ob_start #

ob_start é especialmente útil quando você tem redirecionamentos em sua página. Por exemplo, o código a seguir não funcionará:

Hello!
<?php
  header("Location: somepage.php");
?>

O erro que será dado é algo como: cabeçalhos já enviados por <xxx> na linha <xxx>.

Para corrigir esse problema, você escreveria algo assim no início da sua página:

<?php
  ob_start();
?>

E algo assim no final da sua página:

<?php
  ob_end_flush();
?>

Isso armazena todo o conteúdo gerado em um buffer de saída e o exibe de uma só vez. Portanto, se você tiver chamadas de redirecionamento em sua página, elas serão acionadas antes que qualquer dado seja enviado, eliminando a possibilidade de ocorrer um erro de cabeçalhos já enviados.
