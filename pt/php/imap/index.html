Conectando-se a uma caixa de correio #

Para fazer qualquer coisa com uma conta IMAP, você precisa se conectar a ela primeiro. Para fazer isso, você precisa especificar alguns parâmetros necessários:

  • O nome do servidor ou endereço IP do servidor de correio
  • A porta na qual você deseja se conectar
    • IMAP is 143 or 993 (secure)
    • POP is 110 or 995 (secure)
    • SMTP is 25 or 465 (secure)
    • NNTP is 119 or 563 (secure)
  • Sinalizadores de conexão (veja abaixo)
Bandeira Descrição Opções Padrão
/serviço=serviço Qual serviço usar imap, pop3, nntp, smtp imap
/usuário=usuário nome de usuário remoto para login no servidor
/authuser=usuário usuário de autenticação remota; se especificado, este é o nome de usuário cuja senha é usada (por exemplo, administrador)
/anônimo acesso remoto como usuário anônimo
/depurar registrar telemetria de protocolo no log de depuração do aplicativo desativado
/seguro não transmita uma senha de texto simples pela rede
/norte não use rsh ou ssh para estabelecer uma sessão IMAP pré-autenticada
/ssl use o Secure Socket Layer para criptografar a sessão
/validate-cert certificados do servidor TLS/SSL habilitado
/novalidate-cert não valide certificados do servidor TLS/SSL, necessário se o servidor usar certificados autoassinados. USE COM CUIDADO desativado
/tls forçar o uso de start-TLS para criptografar a sessão e rejeitar a conexão com servidores que não o suportam
/notls não faça start-TLS para criptografar a sessão, mesmo com servidores que o suportem
/somente leitura solicitação de caixa de correio somente leitura aberta (somente IMAP; ignorado no NNTP e um erro com SMTP e POP3)

Sua string de conexão ficará mais ou menos assim:

{imap.example.com:993/imap/tls/secure}

Observe que, se algum dos caracteres em sua string de conexão não for ASCII, ele deverá ser codificado com utf7_encode($string).

Para conectar à caixa de correio, usamos o comando imap_open que retorna um valor de recurso apontando para um stream:

<?php
$mailbox = imap_open("{imap.example.com:993/imap/tls/secure}", "username", "password");
if ($mailbox === false) {
    echo "Failed to connect to server";
}

Instale a extensão IMAP #

Para usar as funções IMAP em PHP, você precisará instalar a extensão IMAP:

Debian/Ubuntu com PHP5

sudo apt-get install php5-imap
sudo php5enmod imap

Debian/Ubuntu com PHP7

sudo apt-get install php7.0-imap

Distro baseada em YUM

sudo yum install php-imap

Mac OS X com php5.6

brew reinstall php56 --with-imap

Lista todas as pastas na caixa de correio #

Depois de se conectar à sua caixa de correio, você vai querer dar uma olhada lá dentro. O primeiro comando útil é imap_list. O primeiro parâmetro é o recurso que você adquiriu de imap_open, o segundo é sua string de caixa de correio e o terceiro é uma string de pesquisa difusa (* é usado para corresponder a qualquer padrão).

$folders = imap_list($mailbox, "{imap.example.com:993/imap/tls/secure}", "*");
if ($folders === false) {
    echo "Failed to list folders in mailbox";
} else {
    print_r($folders);
}

A saída deve ser semelhante a esta

Array
(
    [0] => {imap.example.com:993/imap/tls/secure}INBOX
    [1] => {imap.example.com:993/imap/tls/secure}INBOX.Sent
    [2] => {imap.example.com:993/imap/tls/secure}INBOX.Drafts
    [3] => {imap.example.com:993/imap/tls/secure}INBOX.Junk
    [4] => {imap.example.com:993/imap/tls/secure}INBOX.Trash
)

Você pode usar o terceiro parâmetro para filtrar esses resultados assim:

$folders = imap_list($mailbox, "{imap.example.com:993/imap/tls/secure}", "*.Sent");

E agora o resultado contém apenas entradas com .Sent no nome:

Array
(
    [0] => {imap.example.com:993/imap/tls/secure}INBOX.Sent
)

Observação: Usar * como uma pesquisa difusa retornará todas as correspondências recursivamente. Se você usar %, ele retornará apenas correspondências na pasta atual especificada.

Encontrando mensagens na caixa de correio #

Você pode retornar uma lista de todas as mensagens em uma caixa de correio usando imap_headers.

<?php
$headers = imap_headers($mailbox);

O resultado é um array de strings com o seguinte padrão:

[FLAG] [MESSAGE-ID])[DD-MM-YYY] [FROM ADDRESS] [SUBJECT TRUNCATED TO 25 CHAR] ([SIZE] chars)

Aqui está uma amostra de como cada linha pode ser:

A     1)19-Aug-2016 someone@example.com Message Subject (1728 chars)
D     2)19-Aug-2016 someone@example.com RE: Message Subject (22840 chars)
U     3)19-Aug-2016 someone@example.com RE: RE: Message Subject (1876 chars)
N     4)19-Aug-2016 someone@example.com RE: RE: RE: Message Subje (1741 chars)
Símbolo Bandeira Significado
A Respondido A mensagem foi respondida
D Excluído A mensagem é excluída (mas não removida)
F Sinalizado A mensagem é sinalizada/encarnada para atenção
N Novo A mensagem é nova e não foi vista
R Recente A mensagem é nova e foi vista
U Não lido A mensagem não foi lida
X Rascunho A mensagem é um rascunho

Observe que esta chamada pode levar um bom tempo para ser executada e pode retornar uma lista muito grande.

Uma alternativa é carregar mensagens individuais conforme necessário. Cada um de seus e-mails recebe um ID de 1 (o mais antigo) ao valor de imap_num_msg($mailbox).

Existem várias funções para acessar um e-mail diretamente, mas a maneira mais simples é usar imap_header que retorna informações de cabeçalho estruturadas:

$header = imap_headerinfo($mailbox , 1);
