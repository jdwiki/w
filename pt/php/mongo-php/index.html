Tudo entre MongoDB e PHP #

Requisitos

  • Servidor MongoDB rodando na porta normalmente 27017. (digite mongod no prompt de comando para executar o servidor mongodb)

  • Php instalado como cgi ou fpm com a extensão MongoDB instalada (a extensão MongoDB não é fornecida com o php padrão)

  • Composer library(mongodb/mongodb).(Na raiz do projeto execute php composer.phar require "mongodb/mongodb=^1.0.0" para instalar a biblioteca MongoDB)

Se tudo estiver bem, você está pronto para seguir em frente.

Verificar instalação do PHP

se não tiver certeza, verifique a instalação do Php executando php -v no prompt de comando retornará something like this

PHP 7.0.6 (cli) (construído: 28 de abril de 2016 14:12:14) ( ZTS ) Copyright (c) 1997-2016 O Grupo PHP Zend Engine v3.0.0, Copyright (c) 1998-2016 Zend Technologies

Verifique a instalação do MongoDB

Verifique a instalação do MongoDB executando mongo --version retornará MongoDB shell version: 3.2.6

Verifique a instalação do Composer

Verifique a instalação do Composer executando php composer.phar --version retornará Composer version 1.2-dev (3d09c17b489cd29a0c0b3b11e731987e7097797d) 2016-08-30 16:12:39 `

Conectando ao MongoDB do php

<?php

  //This path should point to Composer's autoloader from where your MongoDB library will be loaded
  require 'vendor/autoload.php';


 // when using custom username password
  try {
        $mongo = new MongoDB\Client('mongodb://username:password@localhost:27017');
        print_r($mongo->listDatabases());
  } catch (Exception $e) {
        echo $e->getMessage();
  }


 // when using default settings
  try {
        $mongo = new MongoDB\Client('mongodb://localhost:27017');
        print_r($mongo->listDatabases());
  } catch (Exception $e) {
        echo $e->getMessage();
  }

O código acima se conectará usando a biblioteca do compositor MongoDB (mongodb/mongodb) incluída como vendor/autoload.php para se conectar ao servidor MongoDB rodando em port: 27017. Se tudo estiver ok ele irá conectar e listar um array, se ocorrer exceção ao conectar ao servidor MongoDB a mensagem será impressa.

CREATE(Inserindo) no MongoDB

<?php

 //MongoDB uses collection rather than Tables as in case on SQL.
 //Use $mongo instance to select the database and collection
 //NOTE: if database(here demo) and collection(here beers) are not found in MongoDB both will be created automatically by MongoDB.
  $collection = $mongo->demo->beers;

 //Using $collection we can insert one document into MongoDB
 //document is similar to row in SQL.
  $result = $collection->insertOne( [ 'name' => 'Hinterland', 'brewery' => 'BrewDog' ] );

 //Every inserted document will have a unique id.
  echo "Inserted with Object ID '{$result->getInsertedId()}'";
?>

No exemplo estamos usando a instância $mongo usada anteriormente na parte Connecting to MongoDB from php. O MongoDB usa o formato de dados do tipo JSON, então em php usaremos array para inserir dados no MongoDB, essa conversão de array para Json e vice-versa será feita pela biblioteca mongo. Todo documento no MongoDB tem um id único chamado _id, durante a inserção podemos obtê-lo usando $result->getInsertedId();

LER (Localizar) no MongoDB

<?php
 //use find() method to query for records, where parameter will be array containing key value pair we need to find.
 $result = $collection->find( [ 'name' => 'Hinterland', 'brewery' => 'BrewDog' ] );

 // all the data(result) returned as array
 // use for each  to filter the required keys
 foreach ($result as $entry) {
   echo $entry['_id'], ': ', $entry['name'], "\n";
 }

?>

Solte no MongoDB

<?php

 $result = $collection->drop( [ 'name' => 'Hinterland'] );

 //return 1 if the drop was sucessfull and 0 for failure
 print_r($result->ok);

?>

Existem muitos métodos que podem ser executados em $collection veja Documentação oficial do MongoDB
