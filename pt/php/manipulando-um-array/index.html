Filtrando uma matriz #

Para filtrar valores de um array e obter um novo array contendo todos os valores que satisfazem a condição de filtro, você pode usar a função array_filter.

Filtrando valores não vazios #

O caso mais simples de filtragem é remover todos os valores “vazios”:

$my_array = [1,0,2,null,3,'',4,[],5,6,7,8];
$non_empties = array_filter($my_array); // $non_empties will contain [1,2,3,4,5,6,7,8];

Filtrando por retorno de chamada #

Desta vez, definimos nossa própria regra de filtragem. Suponha que queremos obter apenas números pares:

$my_array = [1,2,3,4,5,6,7,8];

$even_numbers = array_filter($my_array, function($number) {
    return $number % 2 === 0;   
});

A função array_filter recebe o array a ser filtrado como seu primeiro argumento e um retorno de chamada definindo o predicado do filtro como seu segundo.

Filtrando por índice #

Um terceiro parâmetro pode ser fornecido para a função array_filter, que permite ajustar quais valores são passados ​​para o retorno de chamada. Este parâmetro pode ser definido como ARRAY_FILTER_USE_KEY ou ARRAY_FILTER_USE_BOTH, o que resultará no retorno de chamada recebendo a chave em vez do valor para cada elemento na matriz, ou valor e chave como seus argumentos. Por exemplo, se você quiser lidar com índices em vez de valores:

$numbers = [16,3,5,8,1,4,6];

$even_indexed_numbers = array_filter($numbers, function($index) {
    return $index % 2 === 0;
}, ARRAY_FILTER_USE_KEY);

Índices na matriz filtrada #

Observe que array_filter preserva as chaves originais do array. Um erro comum seria tentar usar um loop for sobre o array filtrado:

<?php

$my_array = [1,0,2,null,3,'',4,[],5,6,7,8];
$filtered = array_filter($my_array); 

error_reporting(E_ALL); // show all errors and notices

// innocently looking "for" loop
for ($i = 0; $i < count($filtered); $i++) {
   print $filtered[$i];
}

/*
Output:
1
Notice: Undefined offset: 1
2
Notice: Undefined offset: 3
3
Notice: Undefined offset: 5
4
Notice: Undefined offset: 7
*/

Isso acontece porque os valores que estavam nas posições 1 (havia 0), 3 (null), 5 (string vazia '') e 7 (array vazio []) foram removidos juntamente com seus chaves de índice.

Se você precisar percorrer o resultado de um filtro em um array indexado, você deve primeiro chamar array_values no resultado de array_filter para criar um novo array com os índices corretos:

$my_array = [1,0,2,null,3,'',4,[],5,6,7,8];
$filtered = array_filter($my_array); 
$iterable = array_values($filtered);

error_reporting(E_ALL); // show all errors and notices

for ($i = 0; $i < count($iterable); $i++) {
   print $iterable[$i];
}

// No warnings!

Removendo elementos de um array #

Para remover um elemento dentro de uma matriz, por exemplo. o elemento com o índice 1.

$fruit = array("bananas", "apples", "peaches");
unset($fruit[1]);

Isso removerá as maçãs da lista, mas observe que unset não altera os índices dos elementos restantes. Então $fruit agora contém os índices 0 e 2.

Para array associativo, você pode remover assim:

$fruit = array('banana', 'one'=>'apple', 'peaches');

print_r($fruit);
/*
    Array
    (
        [0] => banana
        [one] => apple
        [1] => peaches
    )
*/

unset($fruit['one']);

Agora $ fruta é

print_r($fruit);

/*
Array
(
    [0] => banana
    [1] => peaches
)
*/

Observe que

unset($fruit);

desativa a variável e, portanto, remove todo o array, significando que nenhum de seus elementos está mais acessível.

Removendo elementos terminais #

array_shift() - Desloca um elemento do início do array.

Exemplo:

  $fruit = array("bananas", "apples", "peaches");
  array_shift($fruit);
  print_r($fruit);

Resultado:

 Array
(
    [0] => apples
    [1] => peaches
)

array_pop() - Retire o elemento do final do array.

Exemplo:

  $fruit = array("bananas", "apples", "peaches");
  array_pop($fruit);
  print_r($fruit);

Resultado:

 Array
(
    [0] => bananas
    [1] => apples
)

Classificando uma matriz #

Existem várias funções de classificação para arrays em php:

ordenar() #

Classifique uma matriz em ordem crescente por valor.

$fruits = ['Zitrone', 'Orange', 'Banane', 'Apfel'];
sort($fruits);
print_r($fruits);

resulta em

Array
(
    [0] => Apfel
    [1] => Banane
    [2] => Orange
    [3] => Zitrone
)

ordenar() #

Classifique uma matriz em ordem decrescente por valor.

$fruits = ['Zitrone', 'Orange', 'Banane', 'Apfel'];
rsort($fruits);
print_r($fruits);

resulta em

Array
(
    [0] => Zitrone
    [1] => Orange
    [2] => Banane
    [3] => Apfel
)

um tipo() #

Classifique uma matriz em ordem crescente por valor e preserve as indecisões.

$fruits = [1 => 'lemon', 2 => 'orange',  3 => 'banana', 4 => 'apple'];
asort($fruits);
print_r($fruits);

resulta em

Array
(
    [4] => apple
    [3] => banana
    [1] => lemon
    [2] => orange
)

arsort() #

Classifique uma matriz em ordem decrescente por valor e preserve as indecisões.

$fruits = [1 => 'lemon', 2 => 'orange',  3 => 'banana', 4 => 'apple'];
arsort($fruits);
print_r($fruits);

resulta em

Array
(
    [2] => orange
    [1] => lemon
    [3] => banana
    [4] => apple
)

ksort() #

Classifique uma matriz em ordem crescente por chave

$fruits = ['d'=>'lemon', 'a'=>'orange', 'b'=>'banana', 'c'=>'apple'];
ksort($fruits);
print_r($fruits);

resulta em

Array
(
    [a] => orange
    [b] => banana
    [c] => apple
    [d] => lemon
)

krssort() #

Classifique uma matriz em ordem decrescente por chave.

$fruits = ['d'=>'lemon', 'a'=>'orange', 'b'=>'banana', 'c'=>'apple'];
krsort($fruits);
print_r($fruits);

resulta em

Array
(
    [d] => lemon
    [c] => apple
    [b] => banana
    [a] => orange
)

natsort() #

Classifique uma matriz de uma maneira que um ser humano faria (ordem natural).

$files = ['File8.stack', 'file77.stack', 'file7.stack', 'file13.stack', 'File2.stack'];
natsort($files);
print_r($files);

resulta em

Array
(
    [4] => File2.stack
    [0] => File8.stack
    [2] => file7.stack
    [3] => file13.stack
    [1] => file77.stack
)

#natcasesort()

Classifique uma matriz de uma maneira que um ser humano faria (ordem natural), mas com uso intensivo de maiúsculas e minúsculas

$files = ['File8.stack', 'file77.stack', 'file7.stack', 'file13.stack', 'File2.stack'];
natcasesort($files);
print_r($files);

resulta em

Array
(
    [4] => File2.stack
    [2] => file7.stack
    [0] => File8.stack
    [3] => file13.stack
    [1] => file77.stack
)

embaralhar() #

Embaralha uma matriz (classificada aleatoriamente).

$array = ['aa', 'bb', 'cc'];
shuffle($array);
print_r($array);

Como está escrito na descrição, é aleatório, então aqui apenas um exemplo do que pode resultar

Array
(
    [0] => cc
    [1] => bb
    [2] => aa
)

usort() #

Classifique uma matriz com uma função de comparação definida pelo usuário.

function compare($a, $b)
{
    if ($a == $b) {
        return 0;
    }
    return ($a < $b) ? -1 : 1;
}

$array = [3, 2, 5, 6, 1];
usort($array, 'compare');
print_r($array);

resulta em

Array
(
    [0] => 1
    [1] => 2
    [2] => 3
    [3] => 5
    [4] => 6
)

#uasort()

Classifique uma matriz com uma função de comparação definida pelo usuário e preserve as chaves.

function compare($a, $b)
{
    if ($a == $b) {
        return 0;
    }
    return ($a < $b) ? -1 : 1;
}

$array = ['a' => 1, 'b' => -3, 'c' => 5, 'd' => 3, 'e' => -5];
uasort($array, 'compare');
print_r($array);

resulta em

Array
(
    [e] => -5
    [b] => -3
    [a] => 1
    [d] => 3
    [c] => 5
)

uksort() #

Classifique uma matriz por chaves com uma função de comparação definida pelo usuário.

function compare($a, $b)
{
    if ($a == $b) {
        return 0;
    }
    return ($a < $b) ? -1 : 1;
}

$array = ['ee' => 1, 'g' => -3, '4' => 5, 'k' => 3, 'oo' => -5];

uksort($array, 'compare');
print_r($array);

resulta em

Array
(
    [ee] => 1
    [g] => -3
    [k] => 3
    [oo] => -5
    [4] => 5
)

Adicionando elemento ao início do array #

Às vezes, você deseja adicionar um elemento ao início de um array sem modificar nenhum dos elementos atuais (ordem) dentro do array. Sempre que este for o caso, você pode usar array_unshift().

array_unshift() precede os elementos passados ​​na frente do array. Observe que a lista de elementos é prefixada como um todo, de modo que o elementos prefixados permanecem na mesma ordem. Todas as chaves de matriz numérica será modificado para começar a contar do zero, enquanto as chaves literais não ser tocado.

Retirado da documentação do PHP para array_unshift().

Se você quiser conseguir isso, tudo o que você precisa fazer é o seguinte:

$myArray = array(1, 2, 3);

array_unshift($myArray, 4);

Isso agora adicionará 4 como o primeiro elemento em seu array. Você pode verificar isso por:

print_r($myArray);

Isso retorna uma matriz na seguinte ordem: 4, 1, 2, 3.

Como array_unshift força o array a redefinir os pares chave-valor como o novo elemento permite que as seguintes entradas tenham as chaves n+1 é mais inteligente criar um novo array e anexar o array existente ao novo matriz criada.

Exemplo:

$myArray = array('apples', 'bananas', 'pears');
$myElement = array('oranges');
$joinedArray = $myElement;

foreach ($myArray as $i) {
  $joinedArray[] = $i;
}

Saída ($joinedArray):

Array ( [0] => oranges [1] => apples [2] => bananas [3] => pears )

Exemplo/Demonstração

Lista de permissões apenas algumas chaves de array #

Quando você deseja permitir apenas certas chaves em seus arrays, especialmente quando o array vem de parâmetros de solicitação, você pode usar array_intersect_key junto com array_flip.

$parameters = ['foo' => 'bar', 'bar' => 'baz', 'boo' => 'bam'];
$allowedKeys = ['foo', 'bar'];
$filteredParameters = array_intersect_key($parameters, array_flip($allowedKeys));

// $filteredParameters contains ['foo' => 'bar', 'bar' => 'baz]

Se a variável parameters não contiver nenhuma chave permitida, então a variável filteredParameters consistirá em um array vazio.

Desde o PHP 5.6 você pode usar array_filter para esta tarefa também, passando o sinalizador ARRAY_FILTER_USE_KEY como o terceiro parâmetro:

$parameters  = ['foo' => 1, 'hello' => 'world'];
$allowedKeys = ['foo', 'bar'];
$filteredParameters = array_filter(
    $parameters,
    function ($key) use ($allowedKeys) {
        return in_array($key, $allowedKeys);
    },
    ARRAY_FILTER_USE_KEY
);

Usar array_filter dá a flexibilidade adicional de realizar um teste arbitrário contra a chave, por exemplo $allowedKeys pode conter padrões regex em vez de strings simples. Ele também declara mais explicitamente a intenção do código do que array_intersect_key() combinado com array_flip().

Trocar valores com chaves #

A função array_flip trocará todas as chaves com seus elementos.

$colors = array(
    'one' => 'red',
    'two' => 'blue',
    'three' => 'yellow',
);

array_flip($colors); //will output

array(
    'red' => 'one',
    'blue' => 'two',
    'yellow' => 'three'
)

Mesclar dois arrays em um array #

$a1 = array("red","green");
$a2 = array("blue","yellow");
print_r(array_merge($a1,$a2));

/*
    Array ( [0] => red [1] => green [2] => blue [3] => yellow )
*/

Matriz associativa:

$a1=array("a"=>"red","b"=>"green");
$a2=array("c"=>"blue","b"=>"yellow");
print_r(array_merge($a1,$a2));
/*
    Array ( [a] => red [b] => yellow [c] => blue )
*/
  1. Mescla os elementos de um ou mais arrays para que os valores de um sejam anexados ao final do anterior. Ele retorna a matriz resultante.
  2. Se as matrizes de entrada tiverem as mesmas chaves de string, o valor posterior dessa chave substituirá o anterior. Se, no entanto, as matrizes contiverem chaves numéricas, o valor posterior não substituirá o valor original, mas será anexado.
  3. Os valores na matriz de entrada com teclas numéricas serão renumerados com chaves de incremento a partir de zero na matriz de resultados.
