O Google Spreadsheets tem um complemento poderoso chamado Google Forms que permite que um desenvolvedor da Web adicione formulários simples facilmente a sites para coletar dados de usuários .

Este artigo discute a maneira de incorporá-los em um aplicativo da Web.

Também criei um vídeo do YouTube com comentários em execução, capturas de tela e assim por diante.

Os exemplos acima são adaptados de um site totalmente funcional e este artigo pressupõe um conhecimento razoável existente de HTML/Javascript/CSS para usar esses trechos de código.

Crie um formulário do Google #

Faça login em uma Conta do Google e clique em Novo > Mais > Formulários Google.

Construa os campos de formulário necessários usando o editor.

Se o formulário foi criado com uma conta que faz parte de uma organização, clique na engrenagem e desmarque a opção de que apenas membros podem preencher o formulário.

Defina o formulário para salvar as respostas em uma planilha clicando na guia Respostas e clique no ícone da planilha. O pop-up oferece a opção de salvar esses dados de formulário em uma planilha nova ou existente. Ao selecionar existente permite vários formulários por planilha. Siga as instruções para concluir esta tarefa. Este é um bom momento para salvar alguns dados de teste para garantir que tudo esteja funcionando.

Opcionalmente, o aplicativo da web pode definir algumas respostas pré-preenchidas nos campos. Se for esse o caso, volte ao formulário e clique no menu suspenso de três pontos, depois clique em ‘Obter link pré-preenchido’. Isso carregará o formulário em um modo especial onde os campos podem ser preenchidos sem enviar os dados. Ao preencher os campos, use o nome do rótulo como valor pré-preenchido. Em seguida, salve a URL que terá parâmetros semelhantes a entry.123=labelname1&entry.456=labelname2. Salve uma cópia desse URL para mais tarde.

Incorporar o formulário do Google #

Isso é feito adicionando um botão, caixa de diálogo e iframe, conforme explicado abaixo.

Os exemplos abaixo usam MDL para aparência porque é usado pelos formulários do Google e, portanto, faz com que os elementos adicionais pareçam bastante uniformes.

As caixas de diálogo podem exigir um polyfill se você planeja oferecer suporte a navegadores mais antigos.

Adicionar botão e caixa de diálogo ao html #

<button id="googleFormButton" class="mdl-button mdl-js-button mdl-button--raised">
  Load Form
</button>

<dialog id="googleFormsDialog" class="mdl-dialog">
  <!-- <h4 class="mdl-dialog__title">Google Form</h4> -->
  <div id="googleformparent" class="mdl-dialog__content">
    <div id="googleFormsDialogIFrameLoading">Loading...</div>
    <!-- IFrame element googleFormsDialogIFrame is added dynamically due to google forms popup issue. -->
  </div>
  <div class="mdl-dialog__actions">
    <button id="dialogclose" type="button" class="mdl-button">Close</button>
  </div>
</dialog>

Adicione um ouvinte de evento ao botão. #

O valor de GOOGLE-FORM-PREFILLED-URL deve ser algo assim: https://docs.google.com/forms/…/?usp=pp_url&entry.1739003583=labelname1

jQuery('#googleFormButton').click(showGoogleForm)
jQuery('#googleFormButton').attr('googleFormsURL', 'GOOGLE-FORM-PREFILLED-URL')

Gerenciar a caixa de diálogo e o iframe do formulário do Google #

Adicione uma nova função chamada showGoogleForm e adapte o seguinte código para se adequar. Observe que, para simplificar, este exemplo não contém nenhuma verificação de erro que deve ser adicionada em um ambiente de produção.

A url deve ficar mais ou menos assim: https://docs.google.com/forms/…/?usp=pp_url&entry.1739103583=labelname1

  var showGoogleForm = function (e) {
    var url = e.currentTarget.googleFormsURL
    url = url.replace('labelname1', 'Some prefilled value')
    url = url.replace('labelname2', 'Another prefilled value')

    // Add the iFrame dynamically to avoid popup issue
    jQuery('<iframe id="#googleform" src="" width="100%" height="100%" frameborder="0" marginheight="0" marginwidth="0">Loading...</iframe>').appendTo('#googleformparent')    

    // Set the prefilled url as the iFrame source
    jQuery('#googleform').attr('src', url)

    // Remove the iframe element when the user closes the dialog to avoid the popup if the user did not submit the form.
    jQuery('#dialogclose').click(function(e) {
        jQuery('#googleform').remove()        
    })

  }
