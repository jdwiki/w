Reexportar #

Typescript permite reexportar declarações.

//Operator.ts
interface Operator {
    eval(a: number, b: number): number;
}
export default Operator;

//Add.ts
import Operator from "./Operator";
export class Add implements Operator {
    eval(a: number, b: number): number {
        return a + b;
    }
}

//Mul.ts
import Operator from "./Operator";
export class Mul implements Operator {
    eval(a: number, b: number): number {
        return a * b;
    }
}

Você pode agrupar todas as operações em uma única biblioteca

//Operators.ts
import {Add} from "./Add";
import {Mul} from "./Mul";

export {Add, Mul};

Declarações nomeadas podem ser reexportadas usando uma sintaxe mais curta

//NamedOperators.ts
export {Add} from "./Add";
export {Mul} from "./Mul";

Exportações padrão também podem ser exportadas, mas nenhuma sintaxe curta está disponível. Lembre-se, apenas uma exportação padrão por módulo é possível.

//Calculator.ts
export {Add} from "./Add";
export {Mul} from "./Mul";
import Operator from "./Operator";

export default Operator;

Possível é a reexportação de importação em pacote

//RepackedCalculator.ts
export * from "./Operators";

Ao reexportar o pacote, as declarações podem ser substituídas quando declaradas explicitamente.

//FixedCalculator.ts
export * from "./Calculator"
import Operator from "./Calculator";
export class Add implements Operator {
    eval(a: number, b: number): number {
        return 42;
    }
}

Exemplo de uso

//run.ts
import {Add, Mul} from "./FixedCalculator";

const add = new Add();
const mul = new Mul();

console.log(add.eval(1, 1)); // 42
console.log(mul.eval(3, 4)); // 12

Módulo Olá mundo #

//hello.ts
export function hello(name: string){
    console.log(`Hello ${name}!`);
} 
function helloES(name: string){
    console.log(`Hola ${name}!`);
}
export {helloES};
export default hello;

Carregar usando o índice de diretório

Se o diretório contém um arquivo chamado index.ts, ele pode ser carregado usando apenas o nome do diretório (para index.ts o nome do arquivo é opcional).

//welcome/index.ts
export function welcome(name: string){
    console.log(`Welcome ${name}!`);
}

Exemplo de uso de módulos definidos

import {hello, helloES} from "./hello";  // load specified elements
import defaultHello from "./hello";      // load default export into name defaultHello
import * as Bundle from "./hello";       // load all exports as Bundle
import {welcome} from "./welcome";       // note index.ts is omitted

hello("World");                          // Hello World!
helloES("Mundo");                        // Hola Mundo!
defaultHello("World");                   // Hello World!

Bundle.hello("World");                   // Hello World!
Bundle.helloES("Mundo");                 // Hola Mundo!

welcome("Human");                        // Welcome Human!

Exportando/Importando declarações #

Qualquer declaração (variável, const, função, classe, etc.) pode ser exportada do módulo para ser importada em outro módulo.

O Typescript oferece dois tipos de exportação: nomeado e padrão.

Exportação nomeada

// adams.ts
export function hello(name: string){
    console.log(`Hello ${name}!`);
}
export const answerToLifeTheUniverseAndEverything = 42;
export const unused = 0;

Ao importar exportações nomeadas, você pode especificar quais elementos deseja importar.

import {hello, answerToLifeTheUniverseAndEverything} from "./adams";
hello(answerToLifeTheUniverseAndEverything);   // Hello 42!

Exportação padrão

Cada módulo pode ter uma exportação padrão

// dent.ts
const defaultValue = 54;
export default defaultValue;

que pode ser importado usando

import dentValue from "./dent";
console.log(dentValue);        // 54

Importação em pacote

Typescript oferece método para importar módulo inteiro para variável

// adams.ts
export function hello(name: string){
    console.log(`Hello ${name}!`);
}
export const answerToLifeTheUniverseAndEverything = 42;

import * as Bundle from "./adams";
Bundle.hello(Bundle.answerToLifeTheUniverseAndEverything);  // Hello 42!
console.log(Bundle.unused);                                 // 0
