Introdução #

O cabeçalho ctype.h faz parte da biblioteca C padrão. Ele fornece funções para classificar e converter caracteres.

Todas essas funções usam um parâmetro, um int que deve ser EOF ou representável como um caractere não assinado.

Os nomes das funções de classificação são prefixados com ‘é’. Cada um retorna um valor inteiro diferente de zero (TRUE) se o caractere passado a ele satisfizer a condição relacionada. Se a condição não for satisfeita, a função retornará um valor zero (FALSE).

Essas funções de classificação operam como mostrado, assumindo a localidade C padrão:

int a;
int c = 'A';
a = isalpha(c); /* Checks if c is alphabetic (A-Z, a-z), returns non-zero here. */
a = isalnum(c); /* Checks if c  is alphanumeric (A-Z, a-z, 0-9), returns non-zero here. */
a = iscntrl(c); /* Checks is c is a control character (0x00-0x1F, 0x7F), returns zero here. */
a = isdigit(c); /* Checks if c is a digit (0-9), returns zero here. */
a = isgraph(c); /* Checks if c has a graphical representation (any printing character except space), returns non-zero here. */
a = islower(c); /* Checks if c is a lower-case letter (a-z), returns zero here. */
a = isprint(c); /* Checks if c is any printable character (including space), returns non-zero here. */
a = isupper(c); /* Checks if c is a upper-case letter (a-z), returns zero here. */
a = ispunct(c); /* Checks if c is a punctuation character, returns zero here. */
a = isspace(c); /* Checks if c is a white-space character, returns zero here. */
a = isupper(c); /* Checks if c is an upper-case letter (A-Z), returns non-zero here. */
a = isxdigit(c); /* Checks if c is a hexadecimal digit (A-F, a-f, 0-9), returns non-zero here. */

a = isblank(c); /* Checks if c is a blank character (space or tab), returns non-zero here. */

Existem duas funções de conversão. Estes são nomeados usando o prefixo ’to’. Essas funções recebem o mesmo argumento das anteriores. No entanto, o valor de retorno não é um simples zero ou diferente de zero, mas o argumento passado mudou de alguma maneira.

Essas funções de conversão operam conforme mostrado, assumindo a localidade C padrão:

int a;
int c = 'A';

/* Converts c to a lower-case letter (a-z). 
 * If conversion is not possible the unchanged value is returned. 
 * Returns 'a' here. 
 */
a = tolower(c);

/* Converts c to an upper-case letter (A-Z). 
 * If conversion is not possible the unchanged value is returned. 
 * Returns 'A' here. 
 */
a = toupper(c);

Classificando caracteres lidos de um stream #

#include <ctype.h>
#include <stdio.h>

typedef struct {
  size_t space;
  size_t alnum;
  size_t punct;
} chartypes;

chartypes classify(FILE *f) {
  chartypes types = { 0, 0, 0 };
  int ch;

  while ((ch = fgetc(f)) != EOF) {
    types.space += !!isspace(ch);
    types.alnum += !!isalnum(ch);
    types.punct += !!ispunct(ch);
  }

  return types;
}

A função classify lê os caracteres de um stream e conta o número de espaços, caracteres alfanuméricos e de pontuação. Evita várias armadilhas.

  • Ao ler um caractere de um fluxo, o resultado é salvo como um int, caso contrário, haveria uma ambiguidade entre a leitura de EOF (o marcador de fim de arquivo) e um caractere que tenha o mesmo padrão de bits.
  • As funções de classificação de caracteres (por exemplo, isspace) esperam que seu argumento seja ou representável como um unsigned char, ou o valor da macro EOF. Como isso é exatamente o que o fgetc retorna, não há necessidade de conversão aqui.
  • O valor de retorno das funções de classificação de caracteres apenas distingue entre zero (significando falso) e diferente de zero (significando verdadeiro). Para contar o número de ocorrências, este valor precisa ser convertido para 1 ou 0, o que é feito pela dupla negação, !!.

Classificando caracteres de uma string #

#include <ctype.h>
#include <stddef.h>

typedef struct {
  size_t space;
  size_t alnum;
  size_t punct;
} chartypes;

chartypes classify(const char *s) {
  chartypes types = { 0, 0, 0 };
  const char *p;
  for (p= s; p != '\0'; p++) {
    types.space += !!isspace((unsigned char)*p);
    types.alnum += !!isalnum((unsigned char)*p);
    types.punct += !!ispunct((unsigned char)*p);
  }

  return types;
}

A função classify examina todos os caracteres de uma string e conta o número de espaços, caracteres alfanuméricos e de pontuação. Evita várias armadilhas.

  • As funções de classificação de caracteres (por exemplo, isspace) esperam que seu argumento seja ou representável como um unsigned char, ou o valor da macro EOF.
  • A expressão *p é do tipo char e, portanto, deve ser convertida para corresponder ao texto acima.
  • O tipo char é definido para ser equivalente a signed char ou unsigned char.
  • Quando char é equivalente a unsigned char, não há problema, pois todo valor possível do tipo char é representável como unsigned char.
  • Quando char é equivalente a signed char, ele deve ser convertido em unsigned char antes de ser passado para as funções de classificação de caracteres. E embora o valor do caractere possa mudar por causa dessa conversão, é exatamente isso que essas funções esperam.
  • O valor de retorno das funções de classificação de caracteres apenas distingue entre zero (significando falso) e diferente de zero (significando verdadeiro). Para contar o número de ocorrências, este valor precisa ser convertido para 1 ou 0, o que é feito pela dupla negação, !!.
