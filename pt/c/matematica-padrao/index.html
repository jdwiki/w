Sintaxe #

  • #include <math.h>
  • double pow(double x, double y);
  • float powf(float x, float y);
  • powl duplo longo (x duplo longo, y duplo longo);
  1. Para vincular com a biblioteca de matemática, use -lm com sinalizadores gcc.
  2. Um programa portátil que precisa verificar um erro de uma função matemática deve definir errno para zero, e fazer a seguinte chamada feclearexcept(FE_ALL_EXCEPT); antes de chamar uma função matemática. Ao retornar da função matemática, se errno for diferente de zero, ou a seguinte chamada retornará diferente de zero fetestexcept(FE_INVALID | FE_DIVBYZERO | FE_OVERFLOW | FE_UNDERFLOW); então ocorreu um erro na função matemática. Leia manpage de math_error para mais informações.

Restante de ponto flutuante de precisão dupla: fmod() #

Esta função retorna o resto de ponto flutuante da divisão de x/y. O valor retornado tem o mesmo sinal de x.

#include <math.h> /* for fmod() */
#include <stdio.h> /* for printf() */

int main(void)
{
    double x = 10.0;
    double y = 5.1;

    double modulus = fmod(x, y);

    printf("%lf\n", modulus); /* f is the same as lf. */

    return 0;
}

Resultado:

4.90000

Importante: Use esta função com cuidado, pois ela pode retornar valores inesperados devido à operação de valores de ponto flutuante.

#include <math.h>
#include <stdio.h>

int main(void)
{
    printf("%f\n", fmod(1, 0.1));
    printf("%19.17f\n", fmod(1, 0.1));
    return 0;
}

Resultado:

0.1
0.09999999999999995

Resto de vírgula flutuante de precisão simples e precisão dupla longa: fmodf(), fmodl() #

Essas funções retornam o restante de ponto flutuante da divisão de x/y. O valor retornado tem o mesmo sinal de x.

Precisão única:

#include <math.h> /* for fmodf() */
#include <stdio.h> /* for printf() */

int main(void)
{
    float x = 10.0;
    float y = 5.1;

    float modulus = fmodf(x, y);

    printf("%f\n", modulus); /* lf would do as well as modulus gets promoted to double. */
}

Resultado:

4.90000

Dupla Precisão Dupla:

#include <math.h> /* for fmodl() */
#include <stdio.h> /* for printf() */

int main(void)
{
    long double x = 10.0;
    long double y = 5.1;

    long double modulus = fmodl(x, y);

    printf("%Lf\n", modulus); /* Lf is for long double. */
}

Resultado:

4.90000

Funções de energia - pow(), powf(), powl() #

O código de exemplo a seguir calcula a soma da série 1+4(3+3^2+3^3+3^4+…+3^N) usando a família pow() da biblioteca matemática padrão.

#include <stdio.h>
#include <math.h>
#include <errno.h>
#include <fenv.h>

int main()
{
        double pwr, sum=0;
        int i, n;

        printf("\n1+4(3+3^2+3^3+3^4+...+3^N)=?\nEnter N:");
        scanf("%d",&n);
        if (n<=0) {
                printf("Invalid power N=%d", n);
                return -1;
        }

        for (i=0; i<n+1; i++) {
                errno = 0;
                feclearexcept(FE_ALL_EXCEPT);
                pwr = powl(3,i);
                if (fetestexcept(FE_INVALID | FE_DIVBYZERO | FE_OVERFLOW |
                        FE_UNDERFLOW)) {
                        perror("Math Error");
                }
                sum += i ? pwr : 0;
                printf("N= %d\tS= %g\n", i, 1+4*sum);
        }

        return 0;
}
