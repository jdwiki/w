Sintaxe #

  • valor de retorno; /* Retorna da função atual. val pode ser um valor de qualquer tipo que seja convertido para o tipo de retorno da função. */
  • Retorna; /* Retorna da função void atual. */
  • parar; /* Pula incondicionalmente além do final (“sair”) de uma instrução de iteração (loop) ou da instrução switch mais interna. */
  • Prosseguir; /* Salta incondicionalmente para o início de uma instrução de iteração (loop). */
  • vá para LBL; /* Salta para o rótulo LBL. */
  • LBL: instrução /* qualquer instrução na mesma função. */

Estes são os saltos que são integrados em C por meio de palavras-chave.

C também tem outra construção de salto, long jump, que é especificado com um tipo de dados, jmp_buf, e chamadas de biblioteca C, setjmp e longjmp.

Usando retorno #

Retornando um valor #

Um caso comumente usado: retornando de main()

#include <stdlib.h> /* for EXIT_xxx macros */

int main(int argc, char ** argv)
{
  if (2 < argc)
  {
    return EXIT_FAILURE; /* The code expects one argument: 
                            leave immediately skipping the rest of the function's code */
  }

  /* Do stuff. */

  return EXIT_SUCCESS;
}

Notas Adicionais:

  1. Para uma função com um tipo de retorno como void (não incluindo void * ou tipos relacionados), a instrução return não deve ter nenhuma expressão associada; ou seja, a única instrução de retorno permitida seria return;.

  2. Para uma função com um tipo de retorno não void, a instrução return não deve aparecer sem uma expressão.

  3. Para main() (e apenas para main()), uma instrução explicit return não é necessária (em C99 ou posterior). Se a execução atingir o final }, um valor implícito de 0 é retornado. Algumas pessoas pensam que omitir este ‘retorno’ é uma má prática; outros sugerem ativamente deixá-lo de fora.

Não retornando nada #

Retornando de uma função void

void log(const char * message_to_log)
{
  if (NULL == message_to_log)
  {
    return; /* Nothing to log, go home NOW, skip the logging. */
  }

  fprintf(stderr, "%s:%d %s\n", __FILE__, _LINE__, message_to_log);

  return; /* Optional, as this function does not return a value. */
}

Usando goto para sair de loops aninhados #

Sair de loops aninhados normalmente exigiria o uso de uma variável booleana com uma verificação para essa variável nos loops. Supondo que estamos iterando sobre i e j, poderia ficar assim

size_t i,j;
for (i = 0; i < myValue && !breakout_condition; ++i) {
    for (j = 0; j < mySecondValue && !breakout_condition; ++j) {
        ... /* Do something, maybe modifying breakout_condition */
            /* When breakout_condition == true the loops end */
     }
}

Mas a linguagem C oferece a cláusula goto, que pode ser útil neste caso. Ao usá-lo com um rótulo declarado após os loops, podemos facilmente sair dos loops.

size_t i,j;
for (i = 0; i < myValue; ++i) {
    for (j = 0; j < mySecondValue; ++j) {
        ...
        if(breakout_condition) 
          goto final;
    }
}
final:

No entanto, muitas vezes, quando essa necessidade surge, um retorno pode ser melhor usado. Essa construção também é considerada “não estruturada” na teoria da programação estrutural.

Outra situação em que goto pode ser útil é para pular para um manipulador de erros:

ptr = malloc(N *  x);
if(!ptr)
  goto out_of_memory;

/* normal processing */
free(ptr);
return SUCCESS;

out_of_memory:
 free(ptr); /* harmless, and necessary if we have further errors */
 return FAILURE;

O uso de goto mantém o fluxo de erro separado do fluxo normal de controle do programa. No entanto, também é considerado “não estruturado” no sentido técnico.

Usando break e continue #

Imediatamente continue lendo na entrada inválida ou break na solicitação do usuário ou no final do arquivo:

#include <stdlib.h> /* for EXIT_xxx macros */
#include <stdio.h>  /* for printf() and getchar() */
#include <ctype.h> /* for isdigit() */

void flush_input_stream(FILE * fp);


int main(void)
{
  int sum = 0;
  printf("Enter digits to be summed up or 0 to exit:\n");

  do
  {
    int c = getchar();
    if (EOF == c)
    {
      printf("Read 'end-of-file', exiting!\n");

      break;
    }

    if ('\n' != c)
    {
      flush_input_stream(stdin);
    }

    if (!isdigit(c))
    {
      printf("%c is not a digit! Start over!\n", c);

      continue;
    }

    if ('0' == c)
    {
      printf("Exit requested.\n");

      break;
    }

    sum += c - '0';

    printf("The current sum is %d.\n", sum);
  } while (1);

  return EXIT_SUCCESS;
}

void flush_input_stream(FILE * fp)
{
  size_t i = 0;
  int c;
  while ((c = fgetc(fp)) != '\n' && c != EOF) /* Pull all until and including the next new-line. */
  {
    ++i;
  }

  if (0 != i)
  {
    fprintf(stderr, "Flushed %zu characters from input.\n", i);
  }
}
