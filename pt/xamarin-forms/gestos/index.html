Torne uma imagem tocável adicionando um TapGestureRecognizer #

Existem alguns reconhecedores padrão disponíveis no Xamarin.Forms, um deles é o TapGestureRecognizer.

Você pode adicioná-los a praticamente qualquer elemento visual. Dê uma olhada em uma implementação simples que se liga a uma Image. Aqui está como fazer isso no código.

var tappedCommand = new Command(() =>
{
    //handle the tap
});

var tapGestureRecognizer = new TapGestureRecognizer { Command = tappedCommand };
image.GestureRecognizers.Add(tapGestureRecognizer);

Ou em XAML:

<Image Source="tapped.jpg">
    <Image.GestureRecognizers>
        <TapGestureRecognizer
                Command="{Binding TappedCommand}"
                NumberOfTapsRequired="2" />
  </Image.GestureRecognizers>
</Image>

Aqui, o comando é definido usando a vinculação de dados. Como você pode ver, você também pode definir o NumberOfTapsRequired para habilitá-lo para mais toques antes de agir. O valor padrão é 1 toque.

Outros gestos são Pinch e Pan.

Amplie uma imagem com o gesto de pinça #

Para tornar uma Image (ou qualquer outro elemento visual) com zoom, temos que adicionar um PinchGestureRecognizer a ela. Veja como fazer isso no código:

var pinchGesture = new PinchGestureRecognizer();
pinchGesture.PinchUpdated += (s, e) => {
// Handle the pinch
};
    
image.GestureRecognizers.Add(pinchGesture);

Mas também pode ser feito a partir de XAML:

<Image Source="waterfront.jpg">
  <Image.GestureRecognizers>
    <PinchGestureRecognizer PinchUpdated="OnPinchUpdated" />
  </Image.GestureRecognizers>
</Image>

No manipulador de eventos acompanhado, você deve fornecer o código para ampliar sua imagem. Claro que outros usos também podem ser implementados.

void OnPinchUpdated (object sender, PinchGestureUpdatedEventArgs e)
{
    // ... code here
}

Outros gestos são Tap e Pan.

Mostre todo o conteúdo da imagem ampliada com o PanGestureRecognizer #

Quando você tem uma Image ampliada (ou outro conteúdo), você pode querer arrastar a Image para mostrar todo o seu conteúdo no estado ampliado.

Isso pode ser alcançado implementando o PanGestureRecognizer. A partir do código, isso se parece com isso:

var panGesture = new PanGestureRecognizer();
panGesture.PanUpdated += (s, e) => {
  // Handle the pan
};

image.GestureRecognizers.Add(panGesture);

Isso também pode ser feito a partir de XAML:

<Image Source="MonoMonkey.jpg">
  <Image.GestureRecognizers>
    <PanGestureRecognizer PanUpdated="OnPanUpdated" />
  </Image.GestureRecognizers>
</Image>

No evento code-behind, agora você pode manipular o panning de acordo. Use esta assinatura de método para lidar com isso:

void OnPanUpdated (object sender, PanUpdatedEventArgs e)
{
  // Handle the pan
}

Coloque um alfinete onde o usuário tocou na tela com MR.Gestures #

Os reconhecedores de gestos integrados do Xamarins fornecem apenas um manuseio de toque muito básico. Por exemplo. não há como obter a posição de um dedo em contato. MR.Gestures é um componente que adiciona 14 eventos de manipulação de toque diferentes. A posição dos dedos que se tocam faz parte do EventArgs passado para todos os eventos MR.Gestures.

Se você quiser colocar um alfinete em qualquer lugar da tela, a maneira mais fácil é usar um MR.Gestures.AbsoluteLayout que trata do evento Tapping.

<mr:AbsoluteLayout x:Name="MainLayout" Tapping="OnTapping">
    ...
</mr:AbsoluteLayout>

Como você pode ver, o Tapping="OnTapping" também se parece mais com a sintaxe .NET do que Xamarins com os GestureRecognizers aninhados. Essa sintaxe foi copiada do iOS e cheira um pouco para desenvolvedores .NET.

No seu código por trás, você pode adicionar o manipulador OnTapping assim:

private void OnTapping(object sender, MR.Gestures.TapEventArgs e)
{
    if (e.Touches?.Length > 0)
    {
        Point touch = e.Touches[0];
        var image = new Image() { Source = "pin" };
        MainLayout.Children.Add(image, touch);
    }
}

Em vez do evento Tapping, você também pode usar o TappingCommand e vincular ao seu ViewModel, mas isso complicaria as coisas neste exemplo simples.

Mais amostras de MR.Gestures podem ser encontradas no aplicativo GestureSample no GitHub e no site do MR.Gestures. Eles também mostram como usar todos os outros eventos de toque com manipuladores de eventos, comandos, MVVM, …
