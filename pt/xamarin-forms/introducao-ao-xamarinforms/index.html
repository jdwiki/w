Instalação (Visual Studio) #

O Xamarin.Forms é uma abstração do kit de ferramentas de interface do usuário com suporte nativo de plataforma cruzada que permite aos desenvolvedores criar facilmente interfaces de usuário que podem ser compartilhadas no Android, iOS, Windows e Windows Phone. As interfaces de usuário são renderizadas usando os controles nativos da plataforma de destino, permitindo que os aplicativos Xamarin.Forms mantenham a aparência e o comportamento apropriados para cada plataforma.

Plugin Xamarin para Visual Studio #

Para começar a usar o Xamarin.Forms for Visual Studio, você precisa ter o próprio plug-in do Xamarin. A maneira mais fácil de instalá-lo é baixar e instalar o Visual Studio mais recente.

Se você já tiver o Visual Studio mais recente instalado, vá para Painel de Controle > Programas e Recursos, clique com o botão direito do mouse em Visual Studio e clique em Alterar. Quando o instalador abrir, clique em Modificar e selecione as ferramentas de desenvolvimento móvel multiplataforma:

Você também pode optar por instalar o Android SDK:

Desmarque-o se você já tiver o SDK instalado. Você poderá configurar o Xamarin para usar o Android SDK existente posteriormente.

Xamarin.Forms #

Xamarin.Forms é um conjunto de bibliotecas para sua biblioteca de classes portáteis e assemblies nativos. A própria biblioteca Xamarin.Forms está disponível como um pacote NuGet. Para adicioná-lo ao seu projeto, basta usar o comando regular Install-Package do Console do Gerenciador de Pacotes:

Install-Package Xamarin.Forms

para todos os seus assemblies iniciais (por exemplo, MyProject, MyProject.Droid e MyProject.iOS).

A maneira mais fácil de começar a usar o Xamarin.Forms é criar um projeto vazio no Visual Studio:

Como você pode ver, existem 2 opções disponíveis para criar o aplicativo em branco - Portátil e Compartilhado. Eu recomendo que você comece com o Portable porque é o mais usado no mundo real (diferenças e mais explicações a serem adicionadas).

Depois de criar o projeto, verifique se você está usando a versão mais recente do Xamarin.Forms, pois seu modelo inicial pode conter o antigo. Use o Console do Gerenciador de Pacotes ou a opção Gerenciar Pacotes NuGet para atualizar para o Xamarin.Forms mais recente (lembre-se de que é apenas um pacote NuGet).

Embora os modelos do Visual Studio Xamarin.Forms criem um projeto de plataforma iOS para você, você precisará conectar o Xamarin a um host de compilação do Mac para poder executar esses projetos no iOS Simulator ou dispositivos físicos.

Hello World Xamarin Forms: Visual Studio #

Depois de instalar com êxito o Xamarin conforme descrito no primeiro exemplo, é hora de iniciar o primeiro aplicativo de exemplo.

Passo 1: Criando um novo projeto. #

No Visual Studio, escolha New -> Project -> Visual C# -> Cross-Platform -> Blank App (Xamarin.Forms Portable)

Nomeie o aplicativo “Hello World” e selecione o local para criar o projeto e clique em OK. Isso criará uma solução para você que contém três projetos:

  1. HelloWorld (é onde sua lógica e visualizações são colocadas, ou seja, o projeto portátil)
  2. HelloWorld.Droid (o projeto Android)
  3. HelloWorld.iOS (o projeto iOS)

Etapa 2: Investigando a amostra #

Depois de criar a solução, um aplicativo de amostra estará pronto para ser implantado. Abra o App.cs localizado na raiz do projeto portátil e investigue o código. Como visto abaixo, o Contents da amostra é um StackLayout que contém um Label:

using Xamarin.Forms;

namespace Hello_World
{
    public class App : Application
    {
        public App()
        {
            // The root page of your application
            MainPage = new ContentPage
            {
                Content = new StackLayout
                {
                    VerticalOptions = LayoutOptions.Center,
                    Children = {
                        new Label {
                            HorizontalTextAlignment = TextAlignment.Center,
                            Text = "Welcome to Xamarin Forms!"
                        }
                    }
                }
            };
        }
        protected override void OnStart()
        {
            // Handle when your app starts
        }
        protected override void OnSleep()
        {
            // Handle when your app sleeps
        }
        protected override void OnResume()
        {
            // Handle when your app resumes
        }
    }
}

Passo 3: Iniciando o aplicativo #

Agora simplesmente clique com o botão direito do mouse no projeto que deseja iniciar (HelloWorld.Droid ou HelloWorld.iOS) e clique em Set as StartUp Project. Em seguida, na barra de ferramentas do Visual Studio, clique no botão ‘Iniciar’ (o botão triangular verde que se assemelha a um botão Reproduzir) para iniciar o aplicativo no simulador/emulador de destino.
