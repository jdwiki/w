Introdução ao Keras: 30 segundos #

A estrutura de dados principal do Keras é um modelo, uma forma de organizar camadas. O principal tipo de modelo é o modelo Sequencial, uma pilha linear de camadas. Para arquiteturas mais complexas, você deve usar a API funcional Keras.

Aqui está o modelo Sequencial:

from keras.models import Sequential

model = Sequential()

Empilhar camadas é tão fácil quanto .add():

from keras.layers import Dense, Activation

model.add(Dense(output_dim=64, input_dim=100))
model.add(Activation("relu"))
model.add(Dense(output_dim=10))
model.add(Activation("softmax"))

Assim que seu modelo estiver bom, configure seu processo de aprendizado com .compile():

model.compile(loss='categorical_crossentropy', optimizer='sgd', metrics=['accuracy'])

Se precisar, você pode configurar ainda mais seu otimizador. Um princípio central do Keras é tornar as coisas razoavelmente simples, ao mesmo tempo em que permite que o usuário esteja totalmente no controle quando necessário (o controle final é a fácil extensibilidade do código-fonte).

from keras.optimizers import SGD
model.compile(loss='categorical_crossentropy', optimizer=SGD(lr=0.01, momentum=0.9, nesterov=True))

Agora você pode iterar seus dados de treinamento em lotes:

model.fit(X_train, Y_train, nb_epoch=5, batch_size=32)

Como alternativa, você pode alimentar os lotes em seu modelo manualmente:

model.train_on_batch(X_batch, Y_batch)

Avalie seu desempenho em uma linha:

loss_and_metrics = model.evaluate(X_test, Y_test, batch_size=32)

Ou gere previsões sobre novos dados:

classes = model.predict_classes(X_test, batch_size=32)
proba = model.predict_proba(X_test, batch_size=32)

Construir um sistema de resposta a perguntas, um modelo de classificação de imagem, uma Neural Turing Machine, um word2vec embedder ou qualquer outro modelo é tão rápido quanto. As ideias por trás do aprendizado profundo são simples, então por que sua implementação deveria ser dolorosa?

Você encontrará modelos mais avançados: resposta a perguntas com redes de memória, geração de texto com LSTMs empilhados, etc. em pasta de exemplo.

Instalação e configuração #

Keras é uma biblioteca de redes neurais de alto nível, escrita em Python e capaz de ser executada no TensorFlow ou no Theano. Ele foi desenvolvido com foco em permitir a experimentação rápida. Ser capaz de ir da ideia ao resultado com o menor atraso possível é fundamental para fazer uma boa pesquisa. Use Keras se precisar de uma biblioteca de aprendizado profundo que:

  • Permite prototipagem fácil e rápida (através de total modularidade, minimalismo e extensibilidade).
  • Suporta redes convolucionais e redes recorrentes, bem como como combinações dos dois.
  • Suporta esquemas de conectividade arbitrários (incluindo multi-entrada e treinamento multi-saída).
  • Funciona perfeitamente em CPU e GPU.

Instalação #

Keras usa as seguintes dependências:

  • numpy, scipy
  • pyyaml
  • HDF5 e h5py (opcional, necessário se você usar salvar/carregar o modelo funções)
  • Opcional, mas recomendado se você usar CNNs: cuDNN
  • scikit-image (opcional, necessário se você usar funções internas do keras para pré-processamento e aumento de dados de imagem)

Keras é uma biblioteca de alto nível que fornece uma API de aprendizado de máquina conveniente em cima de outras bibliotecas de baixo nível para processamento e manipulação de tensores, chamadas Backends. Neste momento, o Keras pode ser usado em qualquer um dos três back-ends disponíveis: TensorFlow, Theano e CNTK.

Theano é instalado automaticamente se você instalar Keras usando pip. Se você deseja instalar o Theano manualmente, consulte as instruções de instalação do Theano.

TensorFlow é uma opção recomendada e, por padrão, o Keras usa o back-end TensorFlow, se disponível. Para instalar o TensorFlow, a maneira mais fácil é fazer

$ pip install tensorflow

Se você quiser instalá-lo manualmente, consulte as instruções de instalação do TensorFlow.

Para instalar o Keras, vá para a pasta Keras e execute o comando de instalação:

$ python setup.py install

Você também pode instalar Keras do PyPI:

$ pip install keras

Configuração #

Se você executou o Keras pelo menos uma vez, encontrará o arquivo de configuração do Keras em:

~/.keras/keras.json

Se não estiver lá, você pode criá-lo. O arquivo de configuração padrão se parece com isso:

{
    "image_dim_ordering": "tf",
    "epsilon": 1e-07,
    "floatx": "float32",
    "backend": "tensorflow"
}

Mudando de TensorFlow para Theano #

Por padrão, o Keras usará o TensorFlow como sua biblioteca de manipulação de tensores. Se você quiser usar outro backend, basta alterar o campo backend para "theano" ou "tensorflow", e o Keras usará a nova configuração na próxima vez que você executar qualquer código Keras.
