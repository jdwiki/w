NppExec #

NppExec [sourceforge] permite executar comandos e scripts de uma janela de console no Notepad++. Ele pode ser encontrado na barra de menu em Plugins -> NppExec ou simplesmente pressionando a tecla F6 (o atalho Ctrl+F6 executará o comando mais recente).

Exemplo: o seguinte será

  • Defina o console para output_var: on, o que significa que podemos usar a saída do console

  • execute uma consulta SQL do arquivo C:\scripts\query.sql

  • pegue a saída do console e coloque-a no arquivo ativo no cursor

  • feche o console

  • limpar o console

    NPE_CONSOLE v+ sqlcmd -S 111.111.1.1 -U UserName -P “password” -i C:\scripts\query.sql sel_settext $(OUTPUT) NPP_CONSOLE 0 cls

NppExec também permite que você salve seus scripts. Depois de salvá-los, você pode ir para Plugins -> NppExec -> Advanced Options e executá-lo sempre que o Notepad++ iniciar, fechar ou até mesmo adicionar esse script aos comandos do Plugin usando Macros. Por exemplo, salvando o exemplo acima como “Executar Consulta”, eu poderia usar os campos inferiores esquerdos nas Opções Avançadas para adicioná-lo ao menu.

O script estará disponível como uma macro após a reinicialização do Notepad++, desde que a caixa “Colocar no submenu Macros” esteja marcada.

Finalmente, um atalho pode ser atribuído à macro/comando usando Configurações -> Mapeador de atalhos -> Comandos de plug-in.

Visualizador JSON #

JSON Viewer SourceForge é um plugin para visualização e formatação JSON. É útil para recuar/formatar documentos JSON e pode ser usado para navegar em arquivos JSON complexos usando uma ferramenta de visualização em árvore.

A imagem a seguir mostra os comandos oferecidos pelo plugin:

A partir de um fragmento JSON não formatado (exemplo de http://www.json.org ):

{“glossary”: {“title”: “example glossary”,“GlossDiv”: {“title”: “S”,“GlossList”: {“GlossEntry”: {“ID”: “SGML”,“SortAs”: “SGML”,“GlossTerm”: “Linguagem de marcação generalizada padrão”,“Acrônimo”: “SGML”,“Abbrev”: “ISO 8879:1986”,“GlossDef”: {“for”: “Uma meta-marcação idioma, usado para criar linguagens de marcação como DocBook.”,“GlossSeeAlso”: [“GML”, “XML”]},“GlossSee”: “markup”}}}}}

Para formatar e recuar o código:

  • selecione todo o fragmento json
  • clique em “Plugins”/“JSON Viewer”/“Format JSON” ou use o atalho Ctrl + Alt + Shift + M

  • selecione todo o fragmento json

SourceCookifier #

SourceCookifier analisa o código-fonte atual para componentes como nomes de classes, funções e variáveis ​​e os exibe em uma visualização em árvore em um painel lateral. A navegação entre esses membros é possível clicando duas vezes no nome do componente. O plug-in suporta vários idiomas e personalizações são possíveis para idiomas indeterminados ou definidos pelo usuário. Isso é útil ao trabalhar com códigos grandes.

Por padrão, o modo de sessão SourceCookifier é definido como Modo de arquivo único. Outros modos disponíveis são modo de sessão N++ (todos os arquivos são abertos no Notepad++) ou modo de sessão Cookie.

Para usar o Modo de sessão de cookie:

  1. Arraste e solte suas pastas com código-fonte na janela Source Cookifier

  2. Selecione o tipo de arquivo que deseja analisar

Notas:

  • Você pode salvar e carregar as sessões de cookies; além disso, Source Cookifier exibe o histórico das últimas sessões
  • Abrir o arquivo de uma sessão salva no Notepad++ mudará automaticamente o SourceCookifier para o modo de sessão de cookie e carregará esta sessão
  • Manter um modificador de teclado (CTRL, SHIFT ou ALT - eles são todos semelhantes) ao arrastar e soltar a pasta irá acelerar a análise adicionando apenas INCLUDES e não tags

Outro recurso útil é a capacidade de saltar para a definição de um símbolo (por exemplo, função)

  1. Selecione o “modo de sessão” correto: use modo de cookie se a definição do símbolo estiver em outro arquivo não aberto no Notepadd++, ou modo de arquivo único se a definição estiver no mesmo arquivo
  2. No Notepad++, coloque o cursor na função/tipo que você deseja obter a definição e pressione CTRL+SHIFT+ENTER para pular à definição. Se houver várias definições, você pode escolher o arquivo que deseja abrir

  1. Você pode pressionar ALT+ESQUERDA para voltar ao local anterior (e ALT+DIREITA para ir de volta à definição)

TextFX #

TextFX [SourceForge] é um plugin para conversões avançadas de caracteres (caracteres de escape, etc.) e formatação de código (código HTML ou C++).
