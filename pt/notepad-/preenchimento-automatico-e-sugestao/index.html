Configurações básicas de preenchimento automático e sugestões #

O Notepad++ fornece 2 tipos de recursos para preenchimento automático e sugestões:

  • Autocompletar que lê o arquivo aberto e fornece sugestões de palavras e/ou funções dentro do arquivo
  • Sugestão com os argumentos das funções (específicas para a linguagem)

Para habilitá-lo, você precisa alterar uma configuração.

  1. Vá para Configurações > Preferências... > Preenchimento automático
  2. Marque “Ativar preenchimento automático em cada entrada”
  3. Selecione o tipo de conclusão que deseja:
    • words only
    • name of functions only
    • both of them
  4. Opcionalmente, marque “Dica de parâmetro de função na entrada” para exibir os parâmetros enquanto digita um nome de função (veja a imagem abaixo)

Como criar arquivos de definição de preenchimento automático de palavras-chave

Desde a versão 5.0, o Notepad++ tem suporte para os chamados Calltips e introduziu uma nova maneira de armazenar dados de preenchimento automático. Agora tudo é armazenado no formato XML, o que permite uma fácil extensão da funcionalidade. Ao fazer isso, os dados de preenchimento automático e de dica de chamada são combinados em um único arquivo. Arquivos de texto simples .api mais antigos não são mais usados ​​pelo Notepad++ e podem ser excluídos com segurança se estiverem presentes.

Você pode escolher qual srt de autocompletar você deseja ter, em Settings -> Preferences -> Auto Completion -> Enable Autocompletion em cada entrada: palavras do documento atual, funções do idioma atual ou ambos .

Os arquivos de AutoCompletar estão localizados na pasta “plugins\APIs”, localizada na pasta de instalação do Notepad++, na maioria das vezes C:\Program Files\Notepad++.

A sintaxe dos arquivos AutoCompletar é simples, mas possui algumas regras, principalmente a sintaxe correta e a classificação adequada. Se a sintaxe estiver incorreta, o arquivo XML não será carregado e o AutoCompletar será desabilitado. Uma descrição mais formal pode ser encontrada em Editando arquivos de preenchimento automático.

A classificação inadequada (veja abaixo) pode fazer com que a função AutoCompletar se comporte de forma errática, fazendo com que ela falhe em certas palavras.

O conjunto básico de caracteres usado para reconhecer palavras-chave é feito de letras (ou seja, a-z, A-Z, dígitos 0-9 e o underscore). […]

<?xml version="1.0" encoding="Windows-1252" ?>
<NotepadPlus>
   <AutoComplete language="C++">
       <Environment ignoreCase="no" startFunc="(" stopFunc=")" paramSeparator="," terminal=";" additionalWordChar = "."/>
       <KeyWord name="abs" func="yes">
           <Overload retVal="int" descr="Returns absolute value of given integer">
               <Param name="int number" />
           </Overload>
       </KeyWord>
   </AutoComplete>
</NotepadPlus>

Um pequeno exemplo de como o arquivo XML é construído é dado acima. NotepadPlus, AutoComplete e Environment são elementos singleton, deve haver apenas um de cada, e todos eles devem estar presentes para correção, embora seja permitido remover o elemento <Environment>. Ao fazer isso, todos os valores serão padronizados para aqueles fornecidos no exemplo acima.

Para palavras-chave que não são funções, a tag Keyword é de fechamento automático e possui apenas o atributo name. Para indicar que uma palavra-chave pode ser exibida em uma dica de chamada, adicione o atributo func com o valor yes. Nesse caso, a tag Keyword é um nó e contém outras tags.

Então, para cada sobrecarga da função, deve ser adicionado um elemento Overload, que especifica o comportamento e os parâmetros da função. Uma função deve ter pelo menos uma sobrecarga ou não será exibida como uma dica de chamada. O atributo retVal deve estar presente e especifica o tipo do valor de retorno, mas o atributo descr é opcional e descreve o comportamento das funções, como um comentário.

Você pode adicionar novas linhas na descrição, se desejar. Para cada parâmetro que a função usa, um elemento Param pode ser adicionado. O atributo name deve estar presente e especifica o tipo dos parâmetros e/ou qualquer nome do parâmetro.

No elemento AutoComplete você pode adicionar o atributo language mas ele não é usado pelo Notepad++, você pode adicioná-lo para completar se desejar e pode pegar qualquer string que desejar.
