Ici, nous allons discuter de l’historique de Node.js, des informations de version et de son état actuel.

Événements clés chaque année #

2009 #

  • 3 mars : Le projet a été nommé “nœud”
  • 1er octobre : Premier aperçu très précoce de npm, le package Node gestionnaire
  • 8 novembre : Ryan Dahl’s (Creator of Node.js) Original Présentation de Node.js à la JSConf 2009

2010 #

  • Express : Un framework de développement web Node.js
  • Version initiale de Socket.io
  • 28 avril : Support Node.js expérimental sur Heroku
  • 28 juillet : Google Tech Talk de Ryan Dahl sur Node.js
  • 20 août : Node.js 0.2.0 publié

2011 #

  • 31 mars : Guide Node.js
  • 1er Mai : npm 1.0 : Sortie
  • 1er Mai : AMA de Ryan Dahl sur Reddit
  • 10 juillet : Le Node Beginner Book, une introduction à Node.js, est terminé.
    • A comprehensive Node.js tutorial for beginners.
  • 16 août : LinkedIn utilise Node.js
    • LinkedIn launched its completely overhauled mobile app with new features and new parts under the hood.
  • 5 octobre : Ryan Dahl parle de l’histoire de Node.js et pourquoi il l’a créé
  • 5 décembre : Node.js en production chez Uber
    • Uber Engineering Manager Curtis Chambers explains why his company completely re-engineered their application using Node.js to increase l’efficacité et d’améliorer l’expérience des partenaires et des clients.

2012 #

  • 30 janvier : le créateur de Node.js, Ryan Dahl, s’éloigne du quotidien de Node
  • 25 juin : Node.js v0.8.0 [stable] est sorti
  • 20 décembre : Hapi, un framework Node.js est sorti

2013 #

  • 30 avril : The MEAN Stack : MongoDB, ExpressJS, AngularJS et Node.js
  • 17 mai : Comment nous avons construit la première application Node.js d’eBay
  • 15 novembre : PayPal lance Kraken, un framework Node.js
  • 22 novembre : Leak de mémoire Node.js chez Walmart
    • Eran Hammer of Wal-Mart labs came to the Node.js core team complaining of a memory leak he had been tracking down for months.
  • 19 décembre : Koa - Framework web pour Node.js

2014 #

  • 15 janvier : TJ Fontaine reprend le projet Node
  • 23 octobre : Conseil consultatif Node.js
    • Joyent and several members of the Node.js community announced a proposal for a Node.js Advisory Board as a next step towards a fully open modèle de gouvernance pour le projet open source Node.js.
  • 19 novembre : Node.js dans Flame Graphs - Netflix
  • 28 novembre : IO.js – E/S événementielles pour Javascript V8

2015 #

Q1 #

  • 14 janvier : IO.js 1.0.0
  • 10 février : Joyent se déplace pour établir la fondation Node.js
    • Joyent, IBM, Microsoft, PayPal, Fidelity, SAP and The Linux Foundation Join Forces to Support Node.js Community With Neutral and Open Governance
  • 27 février : Proposition de réconciliation IO.js et Node.js

Q2 #

  • 14 avril : Modules privés npm
  • 28 mai : [Node lead TJ Fontaine quitte Joyent](http://venturebeat.com/2015/05/08/node-lead-tj-fontaine-is-stepping-down-and-leaving-joyent -aussi/)
  • 13 mai : Node.js et io.js fusionnent sous la Node Foundation

Q3 #

  • 2 août : Trace - Surveillance et débogage des performances de Node.js
    • Trace is a visualized microservice monitoring tool that gets you all the metrics you need when operating microservices.
  • 13 août : 4.0 est le nouveau 1.0

Q4 #

  • 12 octobre : Node v4.2.0, première version de support à long terme
  • 8 décembre : Apigee, RisingStack et Yahoo rejoignent la Fondation Node.js
  • 8 & 9 décembre : Node Interactive
    • The first annual Node.js conference by the Node.js Foundation

2016 #

Q1 #

  • 10 février : Express devient un projet incubé
  • 23 mars : L’incident du leftpad
  • 29 mars : Google Cloud Platform rejoint la Fondation Node.js

Q2 #

  • 26 avril : npm a 210.000 utilisateurs

Q3 #

  • 18 juillet : CJ Silverio devient le CTO de npm
  • 1er août : Trace, la solution de débogage Node.js devient disponible
  • 15 septembre : Le premier Node Interactive en Europe

Q4 #

  • 11 octobre : Le gestionnaire de paquets de fils est sorti
  • 18 octobre : Node.js 6 devient la version LTS
