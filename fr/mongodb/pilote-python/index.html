Syntaxe #

  • mongodb://[username:password@]host1[:port1][,host2[:port2],…[,hostN[:portN]]][/[database][?options]]

Paramètres #

Paramètre Détail
hostX Facultatif. Vous pouvez spécifier autant d’hôtes que nécessaire. Vous spécifieriez plusieurs hôtes, par exemple, pour les connexions aux jeux de répliques.
:portX Facultatif. La valeur par défaut est :27017 si elle n’est pas spécifiée.
base de données Facultatif. Le nom de la base de données à authentifier si la chaîne de connexion inclut les informations d’authentification Si /database n’est pas spécifié et que la chaîne de connexion inclut les informations d’identification, le pilote s’authentifiera auprès de la base de données d’administration.
?options Options spécifiques à la connexion

Connectez-vous à MongoDB en utilisant pymongo #

from pymongo import MongoClient

uri = "mongodb://localhost:27017/"

client = MongoClient(uri)

db = client['test_db']
# or
# db = client.test_db

# collection = db['test_collection']
# or
collection = db.test_collection


collection.save({"hello":"world"})

print collection.find_one()

Requêtes PyMongo #

Une fois que vous avez un objet collection, les requêtes utilisent la même syntaxe que dans le shell mongo. Quelques légères différences sont :

  • chaque clé doit être placée entre parenthèses. Par exemple:

    db.find({frequencies: {$exists: true}})

devient en pymongo (notez le True en majuscule):

  db.find({"frequencies": { "$exists": True }})

  • les objets tels que les identifiants d’objet ou ISODate sont manipulés à l’aide de classes python. PyMongo utilise ses propres ObjectId classe pour gérer les identifiants d’objet, tandis que les dates utilisent le package standard datetime. Par exemple, si vous souhaitez interroger tous les événements entre 2010 et 2011, vous pouvez faire :

      from datetime import datetime

  date_from = datetime(2010, 1, 1)
  date_to = datetime(2011, 1, 1)
  db.find({ "date": { "$gte": date_from, "$lt": date_to } }):

Mettre à jour tous les documents d’une collection à l’aide de PyMongo #

Supposons que vous deviez ajouter un champ à chaque document d’une collection.

import pymongo

client = pymongo.MongoClient('localhost', 27017)
db = client.mydb.mycollection

for doc in db.find():
   db.update(
       {'_id': doc['_id']}, 
       {'$set': {'newField': 10} }, upsert=False, multi=False)

La méthode find renvoie un Cursor, sur lequel vous pouvez facilement parcourir en utilisant la syntaxe for in. Ensuite, nous appelons la méthode update, en spécifiant le _id et que nous ajoutons un champ ($set). Les paramètres upsert et multi viennent de mongodb (voir ici pour plus d’informations).
