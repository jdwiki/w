Démarrage de mongo avec un fichier de configuration spécifique #

Utilisation du drapeau --config.

$ /bin/mongod --config /etc/mongod.conf
$ /bin/mongos --config /etc/mongos.conf

Notez que -f est le synonyme le plus court de --config.
