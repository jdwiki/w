Connecter le code avec l’interface utilisateur #

Comme iOS où vous utilisez @IBOutlet et @IBAction, vous pouvez également les utiliser ici.

Disons que nous avons un bouton qui, lorsqu’il est cliqué, change le texte de l’étiquette en autre chose.

Pour commencer:

  1. Ajoutez un WKInterfaceLabel et un WKInterfaceLabel à InterfaceController.

  2. Ctrl-Faites glisser de WKInterfaceLabel vers InterfaceController.swift et entrez les détails comme indiqué dans l’image suivante pour ajouter une propriété de sortie :

[![entrez la description de l’image ici][1]][1]

  1. Ctrl-Drag de WKInterfaceButton à InterfaceController.swift et entrez les détails comme indiqué dans l’image suivante pour ajouter une méthode d’action :

[![entrez la description de l’image ici][2]][2]

  1. Remplissez la méthode d’action :

Rapide #

outputLabel.setText("Button Tapped!")

Objectif c #

[[self outputLabel] setText:@"Button Tapped!"]
  1. Exécutez le programme et appuyez sur le bouton pour voir le résultat.

[1] : https://i.stack.imgur.com/7HbET.png [2] : https://i.stack.imgur.com/bHtyu.png

Création d’un nouveau projet watchOS #

Pour développer une application pour watchOS, vous devez commencer par Xcode. Xcode ne fonctionne que sur macOS. Au moment de la rédaction, la dernière version est Xcode 8.3.

Si vous souhaitez démarrer un nouveau projet à partir de zéro :

  1. Démarrez votre Mac et installez Xcode depuis l’App Store s’il n’est pas déjà installé.

  2. Choisissez de créer un nouveau projet.

  3. Dans les modèles, choisissez watchOS, puis “Application iOS avec application WatchKit”.

[![Dans les modèles, choisissez watchOS puis “Application iOS avec application WatchKit”.][1]][1]

  1. Remplissez les détails de votre projet et choisissez un emplacement.

Si vous avez déjà un projet iOS et souhaitez ajouter une cible watchOS :

  1. Allez dans Fichier -> Nouveau -> Cible.

[![Aller dans Fichier -> Nouveau -> Cible][2]][2]

  1. Choisissez l’application WatchKit.

[![Choisir l’application WatchKit][3]][3]

  1. Remplissez les détails de votre cible et choisissez un emplacement.

[1] : https://i.stack.imgur.com/8j8FH.png [2] : https://i.stack.imgur.com/BcTlC.png [3] : https://i.stack.imgur.com/5lgdp.png

Faire un simple “Hello, World!” application #

Chaque cible watchOS comprend une application et une extension. L’application contient les éléments de l’interface utilisateur et l’extension contient la logique réelle (similaire aux vues et aux modèles dans l’architecture MVC dans iOS).

Chaque application WatchKit possède un fichier Interface.storyboard dans lequel vous concevez l’application, et un fichier Assets.xcassets dans lequel placer vos ressources.

Chaque extension WatchKit a un fichier InterfaceController.swift (en fait une sous-classe WKInterfaceController) qui est similaire au fichier ViewController dans iOS.

Pour créer une application Hello World :

  1. Ouvrez le Interface.storyboard.

  2. Localisez le principal InterfaceController.

[![Localisez le contrôleur d’interface principal.][1]][1]

  1. À partir de la bibliothèque dans le volet de droite, ajoutez un WKInterfaceLabel.

[![Depuis la bibliothèque dans le volet de droite, ajoutez un WKInterfaceLabel.][2]][2]

  1. Faites glisser l’étiquette et définissez son texte dans le volet de droite sur “Hello, World!”.

[![Faites glisser l’étiquette et définissez son texte dans le volet de droite sur “Hello, World!”.][3]][3]

  1. Sélectionnez le schéma correct (selon l’image suivante), puis exécutez le projet en appuyant sur le bouton Exécuter dans la barre supérieure, en utilisant le menu Produit, en appuyant sur Cmd-R ou en appuyant sur Exécuter dans la barre tactile.
