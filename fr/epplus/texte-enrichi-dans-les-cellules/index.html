Texte enrichi dans les cellules

  • Ajout de RichText à une cellule
  • Propriétés de formatage du texte
  • Insertion de RichText dans une cellule

Sur cette page

  • Ajout de RichText à une cellule
  • Propriétés de formatage du texte
  • Insertion de RichText dans une cellule

La plupart du temps, lorsque nous créons des feuilles de calcul, nous utilisons simplement la propriété Valeur d’une cellule pour mettre le contenu dans la cellule et la propriété Style pour le formater.

Parfois, cependant, nous pouvons souhaiter appliquer plusieurs styles à une cellule - peut-être mettre un titre en gras et souligné avant le reste du contenu, ou mettre en surbrillance une partie particulière du texte en rouge - c’est là que la propriété RichText de la cellule entre en jeu .

Ajout de RichText à une cellule #

Chaque élément de texte sur lequel vous souhaitez utiliser une mise en forme distincte doit être ajouté séparément, en ajoutant à la propriété de collection RichText de la cellule.

var cell = ws.Cells[1,1];
cell.IsRichText = true;     // Cell contains RichText rather than basic values
cell.Style.WrapText = true; // Required to honor new lines

var title = cell.RichText.Add("This is my title");
var text = cell.RichText.Add("\nAnd this is my text");

Notez que chaque fois que vous ajoutez() une nouvelle chaîne, elle héritera de la mise en forme de la section précédente. Ainsi, si vous souhaitez modifier la mise en forme par défaut, vous n’aurez qu’à la modifier sur la première chaîne ajoutée.

Ce comportement peut toutefois entraîner une certaine confusion lors de la mise en forme de votre texte. En utilisant l’exemple ci-dessus, le code suivant mettra tout le texte dans la cellule en gras et en italique - ce n’est pas le comportement souhaité :

// Common Mistake
var title = cell.RichText.Add("This is my title");
title.Bold = true;
title.Italic = true;

var text = cell.RichText.Add("\nAnd this is my text"); // Will be Bold and Italic too

L’approche préférée consiste à ajouter d’abord toutes vos sections de texte, puis à appliquer ensuite une mise en forme spécifique à la section, comme indiqué ici :

var title = cell.RichText.Add("This is my title");
title.FontName = "Verdana";    // This will be applied to all subsequent sections as well

var text = cell.RichText.Add("\nAnd this is my text");

// Format JUST the title
title.Bold = true;
title.Italic = true;

Propriétés de formatage du texte #

Il existe un certain nombre de propriétés qui peuvent être appliquées aux sections de RichText.

var title = cell.RichText.Add("This is my title");

// Data Type:     bool
// Default Value: false
title.Bold = true;

// Data Type:     System.Drawing.Color
// Default Value: Color.Black
title.Color = Color.Red;
title.Color = Color.FromArgb(255, 0, 0);
title.Color = ColorTranslator.FromHtml("#FF0000");

// Data Type:     string
// Default Value: "Calibri"
title.FontName = "Verdana";

// Data Type:     bool
// Default Value: false
title.Italic = true;

// Data Type:     bool
// Default Value: true
// If this property is set to false, any whitespace (including new lines) 
// is trimmed from the start and end of the Text
title.PreserveSpace = true;

// Data Type:     float
// Default Value: 11
// The font size is specified in Points
title.Size = 16;

// Data Type:     bool
// Default Value: false
// Strikethrough
title.Strike = false;

// Data Type:     string
// Default Value: Whatever was set when the text was added to the RichText collection
title.Text += " (updated)";

// Data Type:     bool
// Default Value: false
title.UnderLine = true;

// Data Type:     OfficeOpenXml.Style.ExcelVerticalAlignmentFont
// Default Value: ExcelVerticalAlignmentFont.None
title.VerticalAlign = ExcelVerticalAlignmentFont.None;

Insertion de RichText dans une cellule #

EPPlus prend également en charge la possibilité d’insérer du texte dans une cellule à l’aide de la méthode Insert(). Par exemple:

var file = new FileInfo(filePath);
using (var p = new ExcelPackage(file))
{
    var wb = p.Workbook;
    var ws = wb.Worksheets.FirstOrDefault() ?? wb.Worksheets.Add("Sheet1");

    var cell = ws.Cells[1, 1];
    cell.IsRichText = true;
    cell.RichText.Clear(); // Remove any RichText that may be in the cell already
    var s1 = cell.RichText.Add("Section 1.");
    var s2 = cell.RichText.Add("Section 2.");

    var s3 = cell.RichText.Insert(1, "Section 3.");

    s3.Bold = true;
    p.Save();
}

Notez que la méthode Insert() n’insère PAS à un index de caractère, mais à un index de section. Étant donné que les sections sont indexées à zéro, le code ci-dessus produira le texte suivant dans la cellule :

Section 1.Section 3.Section 2.
