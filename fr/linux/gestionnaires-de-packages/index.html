Comment mettre à jour les packages avec le gestionnaire de packages apt #

Le Advanced Package Tool, nommé à juste titre le gestionnaire de packages “apt”, peut gérer l’installation et la suppression de logiciels sur Debian, Slackware et d’autres distributions Linux. Ci-dessous quelques exemples simples d’utilisation :

mettre à jour Cette option récupère et analyse les fichiers Packages.gz, afin que les informations sur les packages nouveaux et mis à jour soient disponibles. Pour ce faire, saisissez la commande suivante :

sudo apt-obtenir la mise à jour

mise à niveau Cette option est utilisée pour installer les versions les plus récentes de tous les packages actuellement installés sur le système. Les packages actuellement installés avec de nouvelles versions disponibles sont récupérés et mis à niveau ; en aucun cas les packages actuellement installés ne sont supprimés, ou les packages non déjà installés sont récupérés et installés. Pour effectuer la mise à niveau, saisissez la commande suivante :

sudo apt-get upgrade

dist-upgrade En plus d’effectuer la fonction de mise à niveau, dist-upgrade gère également intelligemment les dépendances changeantes avec les nouvelles versions de packages. Il tentera de mettre à niveau les packages les plus importants au détriment des moins importants si nécessaire. Pour ce faire, saisissez la commande suivante :

sudo apt-get dist-upgrade

Comment mettre à jour des packages avec le gestionnaire de packages pacman #

Pour mettre à jour un programme spécifique :

 sudo pacman -S <programName>

Pour mettre à jour tout le système :

sudo pacman -Syu

Comment installer un paquet avec le gestionnaire de paquets pacman #

Pour rechercher des packages dans la base de données, recherchez à la fois dans les noms et les descriptions des packages :

 pacman -Ss string1 string2 ...

Pour installer un seul package ou une liste de packages (y compris les dépendances), exécutez la commande suivante :

sudo pacman -S package_name1 package_name2 ...

[source][1]

[1] : https://wiki.archlinux.org/index.php/Pacman#Querying_package_databases

Comment mettre à jour les packages avec yum #

Yellowdog Updater, Modified, l’un des derniers vestiges de Yellow Dog Linux, est le gestionnaire de paquets utilisé par les systèmes Red Hat, Fedora et CentOS et leurs dérivés. Il peut gérer l’installation et la suppression de logiciels packagés en tant que rpms pour ces distributions Linux. Ci-dessous quelques exemples simples d’utilisation :

chercher Cette commande tentera de localiser les packages logiciels dans les référentiels de logiciels configurés qui correspondent aux critères de recherche donnés et affichera le nom / la version / l’emplacement du référentiel des correspondances trouvées. Pour l’utiliser, saisissez la commande suivante :

yum search <queryString>

installer Cette commande tentera de localiser et d’installer le logiciel nommé à partir des référentiels de logiciels configurés, en localisant et en installant de manière récursive tout logiciel prérequis nécessaire. Pour l’utiliser, saisissez la commande suivante :

sudo yum install <packageName>

mettre à jour Cette option est utilisée pour installer les versions les plus récentes de tous les packages actuellement installés sur le système. Les packages actuellement installés avec de nouvelles versions disponibles sont récupérés et mis à niveau ; de nouveaux prérequis sont également récupérés et installés si nécessaire, et les packages remplacés ou obsolètes sont supprimés. Pour effectuer la mise à niveau, saisissez la commande suivante :

sudo yum update

Contrairement à apt, la plupart des commandes yum vérifient également automatiquement les mises à jour des métadonnées du référentiel si une vérification n’a pas été effectuée récemment (ou si elles sont forcées de le faire) et récupèrent et analysent les métadonnées mises à jour afin que les informations sur les packages nouveaux et mis à jour sont disponibles avant que l’opération demandée ne soit effectuée.
