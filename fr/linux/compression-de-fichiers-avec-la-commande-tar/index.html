Compression de fichiers avec la commande 'tar'

Paramètres #

Options communes -
-c –create Créez une nouvelle archive.
-x –extrait Extraire des fichiers d’une archive.
-t –liste Lister le contenu d’une archive.
-f –file=ARCHIVE Utilisez le fichier d’archive ou le répertoire ARCHIVE.
-v –verbose Liste détaillée des fichiers traités.
Options de compression -
-a –auto-compresser Utilisez le suffixe d’archive pour déterminer le programme de compression.
-j –bzip2 Filtrez l’archive via bzip2.
-J –xz –lzma Filtrez l’archive par xz.
-z –gzip Filtrez l’archive via gzip.

Compresser un dossier #

Cela crée une simple archive d’un dossier :

tar -cf ./my-archive.tar ./my-folder/

La sortie détaillée montre quels fichiers et répertoires sont ajoutés à l’archive, utilisez l’option -v :

tar -cvf ./my-archive.tar ./my-folder/

Pour archiver un dossier compressé ‘gzip’, il faut utiliser l’option -z :

tar -czf ./my-archive.tar.gz ./my-folder/

Vous pouvez à la place compresser l’archive avec ‘bzip2’, en utilisant l’option -j :

tar -cjf  ./my-archive.tar.bz2 ./my-folder/

Ou compressez avec ‘xz’, en utilisant l’option -J :

tar -cJf  ./my-archive.tar.xz ./my-folder/

Extraire un dossier d’une archive #

Il existe un exemple pour extraire un dossier d’une archive à l’emplacement courant :

tar -xf archive-name.tar

Si vous souhaitez extraire un dossier d’une archive vers une destination précise :

tar -xf archive-name.tar -C ./directory/destination

## Répertorier le contenu de l’archive Voici un exemple de contenu de liste :

tar -tvf archive.tar

L’option -t est utilisée pour la liste. Pour lister le contenu d’une archive tar.gz, vous n’avez plus besoin d’utiliser l’option -z :

tar -tzvf archive.tar.gz

Compresser et exclure un ou plusieurs dossiers #

Si vous souhaitez extraire un dossier, mais que vous souhaitez exclure un ou plusieurs dossiers lors de l’extraction, vous pouvez utiliser l’option --exclude.

tar -cf archive.tar ./my-folder/ --exclude="my-folder/sub1" --exclude="my-folder/sub3"

Avec cette arborescence de dossiers :

my-folder/
   sub1/
   sub2/
   sub3/

Le résultat sera :

./archive.tar
   my-folder/
      sub2/

Dénuder les composants principaux #

Pour supprimer n’importe quel nombre de composants principaux, utilisez l’option –strip-components :

 --strip-components=NUMBER
   strip NUMBER leading components from file names on extraction

Par exemple, pour supprimer le dossier principal, utilisez :

tar -xf --strip-components=1 archive-name.tar

Liste le contenu d’une archive #

Lister le contenu d’un fichier archive sans l’extraire :

tar -tf archive.tar.gz
Folder-In-Archive/
Folder-In-Archive/file1
Folder-In-Archive/Another-Folder/
Folder-In-Archive/Another-Folder/file2
