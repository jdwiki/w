Mise à jour d’Android Studio dans Ubuntu #

Si vous parvenez à télécharger une mise à jour d’Android Studio, mais qu’il ne se passe rien après le redémarrage, consultez l’exemple suivant :

  1. Une fois le correctif téléchargé et Android Studio fermé, ouvrez le terminal
  2. Accédez au dossier de votre studio Android, par ex. cd ~/android-studio
  3. Allez dans le sous-dossier bin : cdbin
  4. Assurez-vous que votre fichier studio.sh dispose des autorisations d’exécution : chmod +x studio.sh
  5. Lancez Android Studio à partir d’ici : ./studio.sh

Après cela, Android Studio trouvera le correctif et l’installera. Ensuite, vous pouvez fermer Android Studio et l’exécuter de la manière habituelle (quelle que soit la manière que vous préférez, je l’exécute à partir du Launcher pad).

Canaux de mise à jour d’Android Studio #

Aperçu #

Le mécanisme de mise à jour intégré d’Android Studio peut être configuré pour recevoir des mises à jour via l’un de ces quatre canaux :

  • Canary : Bleeding edge, publié environ chaque semaine. Il s’agit de premières prévisualisations publiées afin d’obtenir un retour d’information réel pendant le développement. La chaîne Canary aura toujours la dernière version, y compris les mises à jour des versions bêta ou stables. Nous vous recommandons d’exécuter des builds Canary côte à côte avec une version bêta ou une installation stable.
  • Dev : Les versions de Canary après une série complète de tests internes sont promues dans le canal de développement.
  • Bêta : Versions candidates basées sur des versions stables de Canary, publiées et mises à jour pour obtenir des commentaires avant la version stable. Le canal bêta sera mis à jour avec de nouvelles versions stables jusqu’à ce qu’une nouvelle version Canary passe en version bêta.
  • Stable : La version stable officielle, disponible sur le site des développeurs Android.

Téléchargez les installations complètes de la version proposées dans chacune de ces chaînes : Canary, [Dev](http://tools.android.com/download /studio/dev), Bêta, Stable.

Vous pouvez également le construire vous-même, en suivant les instructions de Présentation de la construction.

Sélection d’un canal de mise à jour #

Pour sélectionner le canal de mise à jour pour une installation d’Android Studio, passez par :

Fichier > Paramètres > Paramètres système > Mises à jour

et choisissez le canal approprié sur lequel vérifier les mises à jour :

[![entrez la description de l’image ici][1]][1]

Lorsqu’une mise à jour est disponible, vous serez invité par l’IDE : [![entrez la description de l’image ici][2]][2]

La sélection de la mise à jour affichera la boîte de dialogue d’informations sur la mise à jour avec des informations détaillées sur le patch disponible, sa taille et son canal :

[![entrez la description de l’image ici][3]][3]
