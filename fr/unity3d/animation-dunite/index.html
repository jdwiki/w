Animation de sprites 2D #

L’animation sprite consiste à montrer une séquence existante d’images ou de cadres.

Importez d’abord une séquence d’images dans le dossier de ressources. Créez des images à partir de rien ou téléchargez-en à partir de l’Asset Store. (Cet exemple utilise [cet actif gratuit][1].)

[![images-objets d’animation sélectionnées][2]][2]

Faites glisser chaque image individuelle d’une seule animation du dossier des ressources vers la vue de la scène. Unity affichera une boîte de dialogue pour nommer le nouveau clip d’animation.

[![boîte de dialogue du nom du clip d’animation][3]][3]

C’est un raccourci utile pour :

  • créer de nouveaux objets de jeu
  • assigner deux composants (un Sprite Renderer et un Animator)
  • créer des contrôleurs d’animation (et y lier le nouveau composant Animator)
  • créer des clips d’animation avec les images sélectionnées

[![le nouvel inspecteur d’objets de jeu montrant les nouveaux composants][4]][4]

Prévisualisez la lecture dans l’onglet animation en cliquant sur Lecture :

[![Aperçu de l’animation 2d][5]][5]

La même méthode peut être utilisée pour créer de nouvelles animations pour le même objet de jeu, puis supprimer le nouvel objet de jeu et le contrôleur d’animation. Ajoutez le nouveau clip d’animation au contrôleur d’animation de cet objet de la même manière qu’avec une animation 3D.

Animation de base pour la course #

Ce code montre un exemple simple d’animation dans Unity.

Pour cet exemple, vous devriez avoir 2 clips d’animation ; Exécuter et ralenti. Ces animations doivent être des mouvements Stand-In-Place. Une fois les clips d’animation sélectionnés, créez un Animator Controller. Ajoutez ce contrôleur au joueur ou à l’objet de jeu que vous souhaitez animer.

Ouvrez la fenêtre Animator à partir de l’option Windows. Faites glisser les 2 clips d’animation vers la fenêtre Animator et 2 états seront créés. Une fois créé, utilisez le onglet paramètres de gauche pour ajouter 2 paramètres, tous deux en tant que booléen. Nommez-en un comme “PerformRun” et l’autre comme “PerformIdle”. Définissez “PerformIdle” sur vrai.

Effectuez des transitions de l’état inactif à l’exécution et à l’exécution à l’état inactif (reportez-vous à l’image). Cliquez sur Idle->Run transition et dans la fenêtre de l’inspecteur, désélectionnez HasExit. Faites de même pour l’autre transition. Pour la transition Idle->Run, ajoutez une condition : PerformIdle. Pour Run->Idle, ajoutez une condition : PerformRun. Ajoutez le script C# donné ci-dessous à l’objet de jeu. Il devrait fonctionner avec une animation à l’aide du bouton Haut et tourner avec les boutons Gauche et Droite.

using UnityEngine;
using System.Collections;

public class RootMotion : MonoBehaviour {

//Public Variables
[Header("Transform Variables")]
public float RunSpeed = 0.1f;
public float TurnSpeed = 6.0f;


Animator animator;

void Start()
{
    /**
    * Initialize the animator that is attached on the current game object i.e. on which you will attach this script.
    */
    animator = GetComponent<Animator>();
}

void Update()
{
    /**
    * The Update() function will get the bool parameters from the animator state machine and set the values provided by the user.
    * Here, I have only added animation for Run and Idle. When the Up key is pressed, Run animation is played. When we let go, Idle is played.
    */
    
    if (Input.GetKey (KeyCode.UpArrow)) {
            animator.SetBool ("PerformRun", true);
            animator.SetBool ("PerformIdle", false);
        } else {
            animator.SetBool ("PerformRun", false);
            animator.SetBool ("PerformIdle", true);
        }
}

void OnAnimatorMove()
    {
        /**
         * OnAnimatorMove() function will shadow the "Apply Root Motion" on the animator. Your game objects psoition will now be determined 
         * using this fucntion.
         */
        if (Input.GetKey (KeyCode.UpArrow)){
            transform.Translate (Vector3.forward * RunSpeed);
            if (Input.GetKey (KeyCode.RightArrow)) {
                transform.Rotate (Vector3.up * Time.deltaTime * TurnSpeed);
            }
            else if (Input.GetKey (KeyCode.LeftArrow)) {
                transform.Rotate (-Vector3.up * Time.deltaTime * TurnSpeed);
            }
        }

    }


   
}

Création et utilisation de clips d’animation #

Cet exemple montre comment créer et utiliser des clips d’animation pour des objets de jeu ou des joueurs.

Notez que les modèles utilisés dans cet exemple sont téléchargés depuis Unity Asset Store. Le lecteur a été téléchargé à partir du lien suivant : https://www.assetstore.unity3d.com/en/#!/content/21874.

Pour créer des animations, ouvrez d’abord la fenêtre d’animation. Vous pouvez l’ouvrir en cliquant sur Fenêtre et Sélectionner l’animation ou en appuyant sur Ctrl+6. Le sélectionnez l’objet de jeu auquel vous souhaitez appliquer le clip d’animation, à partir de la fenêtre Hiérarchie, puis cliquez sur le bouton Créer dans la fenêtre d’animation.

[![entrez la description de l’image ici][1]][1]

Nommez votre animation (comme IdlePlayer, SprintPlayer, DyingPlayer, etc.) et enregistrez-la. Maintenant, depuis la fenêtre d’animation, cliquez sur le bouton Ajouter une propriété. Cela vous permettra de changer la propriété de l’objet du jeu ou du joueur par rapport au temps. Cela peut inclure des transformations telles que la rotation, la position et l’échelle et toute autre propriété attachée à l’objet du jeu, par exemple. Collisionneur, rendu de maillage, etc.

[![entrez la description de l’image ici][2]][2]

Pour créer une animation en cours d’exécution pour un objet de jeu, vous aurez besoin d’un modèle 3D humanoïde. Vous pouvez télécharger le modèle à partir du lien ci-dessus. Suivez les étapes ci-dessus pour créer une nouvelle animation. Ajoutez une propriété Transform et sélectionnez Rotation pour l’une des jambes du personnage.

[![entrez la description de l’image ici][3]][3]

À ce moment, vos valeurs de bouton de lecture et de rotation dans la propriété de l’objet de jeu seraient devenues rouges. Cliquez sur la flèche déroulante pour voir les valeurs de rotation X, Y et Z. Le temps d’animation par défaut est fixé à 1 seconde. Les animations utilisent des images clés pour interpoler entre les valeurs. Pour animer, ajoutez des clés à différents moments dans le temps et modifiez les valeurs de rotation à partir de la fenêtre Inspecteur. Par ex. la valeur de rotation à le temps 0.0s peut être 0.0. Au temps 0,5 s, la valeur peut être 20,0 pour X. Au temps 1,0 s, la valeur peut être 0,0. Nous pouvons terminer notre animation à 1.0s.

La durée de votre animation dépend des dernières clés que vous ajoutez à l’animation. Vous pouvez ajouter plus de clés pour rendre l’interpolation plus fluide.

Courbes animées #

Les courbes d’animation vous permettent de modifier un paramètre flottant pendant la lecture de l’animation. Par exemple, s’il y a une animation de longueur 60 secondes et vous voulez qu’une valeur flottante/un paramètre, appelez-le X, pour varier à travers l’animation (comme au moment de l’animation = 0,0 s ; X = 0.0 , au temps d’animation = 30.0s ; X = 1.0, au temps d’animation = 60.0s ; X = 0,0).

Une fois que vous avez la valeur flottante, vous pouvez l’utiliser pour traduire, faire pivoter, mettre à l’échelle ou l’utiliser de toute autre manière.

Pour mon exemple, je vais montrer un objet de jeu de joueur en cours d’exécution. Lors de la lecture de l’animation de la course, la vitesse de traduction du joueur devrait augmenter au fur et à mesure de l’animation. Lorsque l’animation arrive à sa fin, la vitesse de traduction devrait diminuer.

J’ai créé un clip d’animation en cours d’exécution. Sélectionnez le clip puis dans la fenêtre de l’inspecteur, cliquez sur Modifier.

[![entrez la description de l’image ici][1]][1]

Une fois là-bas, faites défiler jusqu’à Courbes. Cliquez sur le signe + pour ajouter une courbe. Nommez la courbe, par ex. ForwardRunCurve. Cliquez sur la courbe miniature sur la droite. Cela ouvrira une petite fenêtre avec une courbe par défaut.

[![entrez la description de l’image ici][2]][2]

Nous voulons une courbe de forme parabolique où elle monte puis descend. Par défaut, il y a 2 points sur la ligne. Vous pouvez ajouter d’autres points en double-cliquant sur la courbe. Faites glisser les points pour créer une forme similaire à la suivante.

[![entrez la description de l’image ici][3]][3]

Dans la fenêtre Animator, ajoutez le clip en cours d’exécution. Ajoutez également un paramètre float avec le même nom que la courbe, c’est-à-dire ForwardRunCurve.

Lors de la lecture de l’animation, la valeur flottante changera en fonction de la courbe. Le code suivant montre comment utiliser la valeur flottante :

using UnityEngine;
using System.Collections;

public class RunAnimation : MonoBehaviour {

Animator animator;
float curveValue;

void Start()
{
    animator = GetComponent<Animator>();
}

void Update()
{
    curveValue = animator.GetFloat("ForwardRunCurve");

    transform.Translate (Vector3.forward * curveValue);
}

}

La variable curveValue contient la valeur de la courbe (ForwardRunCruve) à un moment donné. Nous utilisons cette valeur pour modifier la vitesse de traduction. Vous pouvez attacher ce script à l’objet de jeu du joueur.
