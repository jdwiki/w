Calculer la moyenne des valeurs d’une colonne à partir de données tabulaires #

Étant donné un fichier utilisant ; comme délimiteur de colonne. Nous calculons la moyenne des valeurs de la deuxième colonne avec le programme suivant, l’entrée fournie est la liste des notes d’un groupe d’étudiants :

awk -F';' '{ sum += $2 } END { print(sum / NR) }' <<EOF
Alice;2
Victor;1
Barbara;1
Casper;4
Deborah;0
Ernest;1
Fabiola;4
Giuseppe;4
EOF

La sortie de ce programme est 2.125.

Rappelez-vous que NR contient le numéro de la ligne en cours de traitement, dans le bloc END il contient donc le nombre total de lignes dans le fichier.

N’oubliez pas que dans de nombreuses applications (surveillance, statistiques), la médiane est une information beaucoup plus utile. Voir l’exemple correspondant.

Traitement robuste des données tabulaires (CSV et al.) #

Le traitement des données tabulaires avec awk est très facile, à condition que l’entrée soit correctement formatée. La plupart des logiciels produisant des données tabulaires utilisent des fonctionnalités spécifiques de cette famille de formats, et les programmes awk traitant des données tabulaires sont souvent spécifiques à une donnée produite par un logiciel spécifique. Si une solution plus générique ou robuste est requise, la plupart des langages populaires fournissent des bibliothèques prenant en charge de nombreuses fonctionnalités trouvées dans les données tabulaires :

  • noms de colonne facultatifs sur la première ligne
  • mélange de valeurs de colonnes entre guillemets et sans guillemets
  • divers délimiteurs
  • formats localisés pour les nombres flottants

Bien qu’il soit tout à fait possible de gérer toutes ces fonctionnalités de manière propre et générique avec awk, cela n’en vaut probablement pas la peine.

Échange de deux colonnes dans des données tabulaires #

Étant donné un fichier utilisant ; comme délimiteur de colonne. La permutation de la première et de la deuxième colonne est accomplie par

awk -F';' -v 'OFS=;' '{ swap = $2; $2 = $1; $1 = swap; print }'

Sélection de colonnes spécifiques dans les données tabulaires #

Nous supposons un fichier utilisant ; comme délimiteur de colonne. La sélection d’un ensemble spécifique de colonnes nécessite uniquement une instruction print. Par exemple, le programme suivant sélectionne les colonnes 3, 4 et 7 à partir de son entrée :

awk -F';' -v 'OFS=;' '{ print $3, $4, $7 }'

Il est comme d’habitude possible de choisir plus soigneusement les lignes à imprimer. Le programme suivant sélectionne les colonnes 3, 4 et 7 à partir de son entrée lorsque le premier champ est Alice ou Bob :

awk -F';' -v 'OFS=;' '($1 == "Alice") || ($1 == "Bob") { print $3, $4, $7 }'

Calculer la médiane des valeurs d’une colonne à partir de données tabulaires #

Étant donné un fichier utilisant ; comme délimiteur de colonne. Nous calculons la médiane des valeurs de la deuxième colonne avec le programme suivant, écrit pour GNU awk. L’entrée fournie est la liste des notes d’un groupe d’étudiants :

gawk -F';' '{ sample[NR] = $2 }
 END {
   asort(sample);
   if(NR % 2 == 1) {
     print(sample[int(NR/2) + 1])
   } else {
     print(sample[NR/2])
   }
}' <<EOF
Alice;2
Victor;1
Barbara;1
Casper;4
Deborah;0
Ernest;1
Fabiola;4
Giuseppe;4
EOF

La sortie de ce programme est ‘1’.

Rappelez-vous que NR contient le numéro de la ligne en cours de traitement, dans le bloc END il contient donc le nombre total de lignes dans le fichier.

De nombreuses implémentations de awk n’ont pas de fonction pour trier les tableaux, qui doivent donc être définis avant que le code ci-dessus puisse être utilisé.

Sélection d’un ensemble de lignes entre deux motifs #

La correspondance de modèle peut être utilisée efficacement avec awk car elle contrôle les actions qui le suivent, c’est-à-dire { pattern } { action }. Une utilisation intéressante de la correspondance de motifs consiste à sélectionner plusieurs motifs entre deux motifs dans un fichier, par exemple “motifA” et “motifB”.

$ awk '/patternA/,/patternB/' file

Supposons que le contenu de mon fichier soit le suivant et que je souhaite extraire les lignes uniquement entre le modèle ci-dessus : -

$ cat file
This is line - 1
This is line - 2
patternA
This is line - 3
This is line - 4
This is line - 5
patternB
This is line - 6

$ awk '/patternA/,/patternB/' file
patternA
This is line - 3
This is line - 4
This is line - 5
patternB

La commande ci-dessus ne fait aucune { action } spécifique autre que l’impression des lignes correspondantes, mais toute action spécifique dans le sous-ensemble de lignes peut être appliquée avec un bloc d’action ({}).
