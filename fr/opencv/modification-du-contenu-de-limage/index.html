Modification de la couleur de l’image dans OpenCV - kmeans(). Pour scanner tous les pixels d’une image et remplacer les valeurs de pixel par des couleurs génériques. #

#include opencv2/opencv.hpp> #include vecteur> en utilisant l'espace de noms std ; en utilisant l'espace de noms cv ; int main() { Mat3b img = imread("test.jpg"); 
imshow("Original", img);

// Cluster

int K = 8;
int n = img.rows * img.cols;
Mat data = img.reshape(1, n);
data.convertTo(data, CV_32F);

Mat labels;
Mat1f colors;
kmeans(data, K, labels, cv::TermCriteria(), 1, cv::KMEANS_PP_CENTERS, colors);

for (int i = 0; i < n; ++i)
{
    data.at<float>(i, 0) = colors(labels.at<int>(i), 0);
    data.at<float>(i, 1) = colors(labels.at<int>(i), 1);
    data.at<float>(i, 2) = colors(labels.at<int>(i), 2);
}

Mat reduced = data.reshape(3, img.rows);
reduced.convertTo(reduced, CV_8U);



imshow("Reduced", reduced);
waitKey(0);

return 0;

}

Modification pixel par pixel des images #

Dans OpenCV, les images peuvent être RVB/BGR, HSV, en niveaux de gris, noir-blanc, etc. Il est crucial de connaître le type de données avant de traiter les images.

Les types de données d’image sont principalement CV_8UC3 (Matrice d’uchar à 3 canaux) et CV_8U (Matrice d’uchar à 1 canal), cependant, la conversion vers d’autres types tels que CV_32FC3, CV_64F est également possible. (voir [types de données][1])

Considérez que l’image est une image RVB qui est lue par la fonction “imread”.

Mat rgb = imread('path/to/rgb/image', CV_LOAD_IMAGE_COLOR);
//to set RED pixel value of (i,j)th to X,
rgb.at<Vec3b>(i,j)[0] = X;

De même, si l’image est en niveaux de gris,

gray.at<uchar>(i,j) = X;

Notez que, dans OpenCV, les images en noir et blanc sont stockées sous le type CV_8U avec les valeurs 0 et 255. Par conséquent, la modification des images BW est la même que celle des images grises.

[1] : https://www.wikiod.com/fr/opencv/structures-de-base#Type de données

Définir l’image entière sur une couleur unie #

Étant donné un cv::Mat img non vide d’une certaine taille, nous pouvons le remplir d’une couleur unie de plusieurs manières :

img = cv::Scalar(blueVal,greenVal,redVal);

ou, le plus général, masque prenant en charge, cv::Mat::setTo() :

img.setTo(cv::Scalar(blueVal,greenVal,redVal));

Si vous utilisez l’ancienne API OpenCV C avec IplImage* img :

Utilisation:

cvSet(img, CV_RGB(redVal,greenVal,blueVal));
