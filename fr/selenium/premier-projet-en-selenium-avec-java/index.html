Obtenir des éléments dans Selenium #

Chaque élément Html dans Selenium est appelé un WebElement. Par exemple, une balise p serait un WebElement, une balise a serait un WebElement, etc. Considérez la structure html suivante :

<a id="link1" href="https://www.google.com">google</a>
<p class="p1">
This is a paragraph
</p>

Maintenant, si nous voulions obtenir la balise a, nous pourrions faire

WebElement link = driver.findElement(By.id("link1"));

Maintenant, nous pouvons cliquer dessus, en

link.click();

Prenons un autre exemple. Si nous voulions le texte de la balise p, c’est-à-dire, “Ceci est un paragraphe”, nous pouvons faire

WebElement p = driver.findElement(By.className("p1"));
System.out.println(p.getText());

Nous pouvons également obtenir des éléments par des balises, comme

WebElement tag = driver.findElement(By.tagName("a"));

Configuration d’IntelliJ Idea pour Selenium #

Conditions préalables:

  1. Java est installé
  2. Le sélénium est extrait dans un dossier (Contient 2 fichiers et 1 dossier)

Suivez ces étapes pour configurer IntelliJ Idea pour Selenium.

  1. Cliquez sur “Nouveau projet”.
  2. Choisissez l’application Java < “Hello World”
  3. Tapez le nom du projet et créez-le.

Votre écran devrait ressembler à ceci [![entrez la description de l’image ici][1]][1]

Maintenant, allez à

File < Project Structure < Modules < Dependencies

Là, cliquez sur l’icône verte plus (+) et choisissez Bibliothèque. Accédez ensuite au dossier Selenium extrait et ajoutez “selenium-java 2.4.0.jar”. Après l’avoir ajouté, cliquez à nouveau sur l’icône verte plus (+) et choisissez maintenant * “Répertoire”. * Cette fois, localisez le dossier * libs * de Selenium et cliquez sur ok tout en le sélectionnant.

À la fin, votre structure de projet devrait ressembler à ceci[![entrer la description de l’image ici][2]][2]

Maintenant, cliquez sur OK, et vous êtes prêt.

## Configuration de ChromeDriver Prérequis : ChromeDriver est téléchargé

Copiez le code suivant dans votre classe.

public static void main(String[] args) {
    System.setProperty("webdriver.chrome.driver", "path of the exe file\\chromedriver.exe");
}

Si vous utilisez Linux, indiquez le chemin d’accès au binaire ChromeDriver.

Ouvrir un site Web avec Selenium #

Nous utilisons la méthode “get” pour accéder à un site Web. Par exemple, cela ouvrirait google

public static void main(String[] args) throws InterruptedException {
    System.setProperty("webdriver.chrome.driver", "path of the exe file\\chromedriver.exe");
    WebDriver driver = new ChromeDriver();
    driver.get("https:www.google.com");
    Thread.sleep(3000); //wait for 3 seconds
    driver.quit();      //close Chrome
}

driver.quit() ferme le navigateur. Pour créer un délai, nous utilisons Thread.sleep(3000).

Exemple de travail dans Selenium #

Maintenant que nous connaissons les bases de Selenium, nous pouvons créer notre propre projet. Pour cet exemple, nous allons créer un programme qui trouve les questions les plus récentes sur le débordement de pile.

Nous commençons facilement, ouvrons le stack-overflow.

public static void main(String[] args) throws InterruptedException {
    System.setProperty("webdriver.chrome.driver", "path of the exe file\\chromedriver.exe");
    WebDriver driver = new ChromeDriver();
    driver.get("https:stackoverflow.com");
    Thread.sleep(3000);
    driver.quit();
}

Maintenant, si vous regardez la source de la page, vous constatez que toutes les questions sont des balises a, avec un nom de classe de question-hyperlink. Cependant, comme il y a plusieurs questions, nous utilisons une List de WebElement, au lieu de WebElement. Ainsi, nous pouvons faire

public static void main(String[] args) throws InterruptedException {
    System.setProperty("webdriver.chrome.driver", "path to chromedriver\\chromedriver.exe");
    WebDriver driver = new ChromeDriver();
    driver.get("https:stackoverflow.com");
    List<WebElement> list = driver.findElements(By.className("question-hyperlink"));
}

Maintenant, nous devons obtenir l’attribut href de la balise a, qui contient le lien de la question. Pour ce faire, nous pouvons utiliser getAttribute("href") sur chaque WebElement, comme

public static void main(String[] args) throws InterruptedException {
    System.setProperty("webdriver.chrome.driver", "path to chromedriver\\chromedriver.exe");
    WebDriver driver = new ChromeDriver();
    driver.get("https:stackoverflow.com");
    List<WebElement> list = driver.findElements(By.className("question-hyperlink"));
    System.out.println(list.size());
    list.forEach(e->System.out.println(e.getAttribute("href")));
    driver.quit();
}

Cela imprime les liens des principales questions sur Stack-overflow.

Obtenir les attributs de WebElements dans Selenium #

Pour obtenir l’attribut d’un WebElement, nous utilisons getAttribute sur ce WebElement. Par exemple, considérez la balise html suivante

<a id="click" href="https://www.google.com">

Nous pouvons trouver l’attribut href de l’élément par

WebElement e = driver.findElement(By.id("click"));
System.out.println(e.getAttribute("href")); //prints https://www.google.com
