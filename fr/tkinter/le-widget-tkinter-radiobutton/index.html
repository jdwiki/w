Syntaxe #

  • radiobutton = tk.Radiobutton(parent, **kwargs)

Paramètres #

Paramètre Descriptif
père Les widgets tkinter existent dans une hiérarchie. À l’exception de la fenêtre racine, tous les widgets ont un parent. Certains didacticiels en ligne appellent cela “maître”. Lorsque le widget est ajouté à l’écran avec pack, place ou grid, il apparaîtra à l’intérieur de ce widget parent.
commande fonction appelée à chaque fois que l’utilisateur change l’état du radiobutton
indicateur activé 1 ou Vrai pour les boutons radio, 0 ou Faux pour les boîtes à boutons
texte Texte à afficher à côté du bouton radio.
valeur Lorsque le bouton radio est sélectionné, la variable de contrôle associée est définie sur valeur.
variables Variable de contrôle que le radiobutton partage avec l’autre radiobutton du groupe.

Ces exemples supposent que tkinter a été importé avec import tkinter as tk (python 3) ou import Tkinter as tk (python 2).

Référence:

[![entrez la description de l’image ici][1]][1]

Pour transformer l’exemple ci-dessus en une “boîte à boutons” plutôt qu’en un ensemble de boutons radio, définissez l’option indicatoron sur 0. Dans ce cas, il n’y a pas d’indicateur de bouton radio séparé et le bouton sélectionné est dessiné comme SUNKEN au lieu de RAISED :

Voici un exemple de la façon de transformer les boutons radio en boîtes à boutons : #

import tkinter as tk
root = tk.Tk()

rbvar = StringVar()
rbvar.set(" ")

rb1 = tk.Radiobutton(root, text="Option 1", variable=rbvar, value='a', indicatoron=0)
rb1.pack()

rb2 = tk.Radiobutton(root, text="Option 2", variable=rbvar, value='b', indicatoron=0)
rb2.pack()

Créer un groupe de radiobuttons #

Un tel groupe est constitué de boutons radio qui partagent une variable de contrôle de sorte qu’il n’est pas possible d’en sélectionner plus d’un.

# control variable
var = tk.IntVar(parent, 0)

# group of radiobuttons
for i in range(1,4):
    tk.Radiobutton(parent, text='Choice %i' % i, value=i, variable=var).pack()

tk.Button(parent, text='Print choice', command=lambda: print(var.get())).pack()
