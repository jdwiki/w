Dessinez derrière les formes existantes avec “destination-over” #

context.globalCompositeOperation = "destination-over"

La composition “destination-over” place le nouveau dessin sous les dessins existants.

context.drawImage(rainy,0,0);
context.globalCompositeOperation='destination-over';  // sunny UNDER rainy
context.drawImage(sunny,0,0);

Effacer les formes existantes avec “destination-out” #

context.globalCompositeOperation = "destination-out"

la composition “destination-out” utilise de nouvelles formes pour effacer les dessins existants.

La nouvelle forme n’est pas réellement dessinée - elle est simplement utilisée comme “emporte-pièce” pour effacer les pixels existants.

context.drawImage(apple,0,0);
context.globalCompositeOperation = 'destination-out';   // bitemark erases
context.drawImage(bitemark,100,40);

[![entrez la description de l’image ici][1]][1]

[1] : http://i.stack.imgur.com/09Qgz.png

Composition par défaut : les nouvelles formes sont dessinées sur les formes existantes #

context.globalCompositeOperation = "source-over"

composition “source-over” [default], place tous les nouveaux dessins sur tous les dessins existants.

context.globalCompositeOperation='source-over';  // the default
context.drawImage(background,0,0);
context.drawImage(parachuter,0,0);

[![entrez la description de l’image ici][1]][1]

[1] : http://i.stack.imgur.com/C61XF.png

Découpez des images à l’intérieur de formes avec “destination-in” #

context.globalCompositeOperation = "destination-in"

la composition “destination-in” clipse les dessins existants à l’intérieur d’une nouvelle forme.

Remarque : Toute partie du dessin existant qui tombe en dehors du nouveau dessin est effacée.

context.drawImage(picture,0,0);
context.globalCompositeOperation='destination-in';  // picture clipped inside oval
context.drawImage(oval,0,0);

[1] : http://i.stack.imgur.com/I5DS2.png

Coupez les images à l’intérieur des formes avec “source-in” #

context.globalCompositeOperation = "source-in";

La composition “source-in” découpe de nouveaux dessins à l’intérieur d’une forme existante.

Remarque : Toute partie du nouveau dessin qui tombe en dehors du dessin existant est effacée.

context.drawImage(oval,0,0);
context.globalCompositeOperation='source-in';  // picture clipped inside oval
context.drawImage(picture,0,0);

[![entrez la description de l’image ici][1]][1]

[1] : http://i.stack.imgur.com/KTqsN.png

Ombres intérieures avec “source-au-dessus” #

context.globalCompositeOperation = 'source-atop'

La composition source-atop coupe une nouvelle image à l’intérieur d’une forme existante.

// gold filled rect
ctx.fillStyle='gold';
ctx.fillRect(100,100,100,75);
// shadow
ctx.shadowColor='black';
ctx.shadowBlur=10;
// restrict new draw to cover existing pixels
ctx.globalCompositeOperation='source-atop';
// shadowed stroke
// "source-atop" clips off the undesired outer shadow
ctx.strokeRect(100,100,100,75);
ctx.strokeRect(100,100,100,75);

[1] : http://i.stack.imgur.com/QXUzY.png

Inverser ou inverser l’image avec “différence” #

Rendre un rectangle blanc sur une image avec l’opération composite

ctx.globalCompositeOperation = 'difference';

La quantité de l’effet peut être contrôlée avec le réglage alpha

// Render the image
ctx.globalCompositeOperation='source-atop';
ctx.drawImage(image, 0, 0);

// set the composite operation
ctx.globalCompositeOperation='difference';
ctx.fillStyle = "white";
ctx.globalAlpha = alpha;  // alpha 0 = no effect 1 = full effect
ctx.fillRect(0, 0, image.width, image.height);

Noir & Blanc avec “couleur” #

Supprimer la couleur d’une image via

ctx.globalCompositeOperation = 'color';

La quantité de l’effet peut être contrôlée avec le réglage alpha

// Render the image
ctx.globalCompositeOperation='source-atop';
ctx.drawImage(image, 0, 0);

// set the composite operation
ctx.globalCompositeOperation='color';
ctx.fillStyle = "white";
ctx.globalAlpha = alpha;  // alpha 0 = no effect 1 = full effect
ctx.fillRect(0, 0, image.width, image.height);

Augmenter le contraste des couleurs avec “saturation” #

Augmentez le niveau de saturation d’une image avec

ctx.globalCompositeOperation = 'saturation';

La quantité de l’effet peut être contrôlée avec le paramètre alpha ou la quantité de saturation dans la superposition de remplissage

// Render the image
ctx.globalCompositeOperation='source-atop';
ctx.drawImage(image, 0, 0);

// set the composite operation
ctx.globalCompositeOperation ='saturation';
ctx.fillStyle = "red";
ctx.globalAlpha = alpha;  // alpha 0 = no effect 1 = full effect
ctx.fillRect(0, 0, image.width, image.height);

Sépia FX avec “luminosité” #

Créez un effet sépia coloré avec

ctx.globalCompositeOperation = 'luminosity';

Dans ce cas, la couleur sépia est rendue en premier sur l’image.

La quantité de l’effet peut être contrôlée avec le paramètre alpha ou la quantité de saturation dans la superposition de remplissage

// Render the image
ctx.globalCompositeOperation='source-atop';
ctx.fillStyle = "#F80";  // the color of the sepia FX
ctx.fillRect(0, 0, image.width, image.height);

// set the composite operation
ctx.globalCompositeOperation ='luminosity';

ctx.globalAlpha = alpha;  // alpha 0 = no effect 1 = full effect
ctx.drawImage(image, 0, 0);

[1] : http://i.stack.imgur.com/hDrhq.png

Changer l’opacité avec “globalAlpha” #

context.globalAlpha=0.50

Vous pouvez modifier l’opacité des nouveaux dessins en définissant globalAlpha sur une valeur comprise entre 0,00 (entièrement transparent) et 1,00 (entièrement opaque).

Le globalAlpha par défaut est 1.00 (entièrement opaque).

Les dessins existants ne sont pas affectés par globalAlpha.

// draw an opaque rectangle
context.fillRect(10,10,50,50);

// change alpha to 50% -- all new drawings will have 50% opacity
context.globalAlpha=0.50;

// draw a semi-transparent rectangle
context.fillRect(100,10,50,50);
