Introduction à “context.getImageData” #

Html5 Canvas vous donne la possibilité de récupérer et de modifier la couleur de n’importe quel pixel sur le canevas.

Vous pouvez utiliser la manipulation de pixels de Canvas pour :

  • Créez un sélecteur de couleurs pour une image ou sélectionnez une couleur sur une roue chromatique.
  • Créez des filtres d’image complexes comme le flou et la détection des contours.
  • Recolorez n’importe quelle partie d’une image au niveau du pixel (si vous utilisez HSL, vous pouvez même recolorer une image tout en conservant l’éclairage et la saturation importants afin que le résultat ne ressemble pas à quelqu’un qui a giflé de la peinture sur l’image). Remarque : Canvas dispose désormais d’un Blend Compositing qui peut également recolorer une image dans certains cas.
  • “Knockout” l’arrière-plan autour d’une personne/élément dans une image,
  • Créer un pot de peinture pour détecter et remplir une partie d’une image (par exemple, changer la couleur d’un pétale de fleur cliqué par l’utilisateur du vert au jaune).
  • Examiner une image pour le contenu (par exemple, la reconnaissance faciale).

Problèmes courants :

  • Pour des raisons de sécurité, getImageData est désactivé si vous avez dessiné une image provenant d’un domaine différent de celui de la page Web elle-même.
  • getImageData est une méthode relativement coûteuse car elle crée un grand tableau de données de pixels et parce qu’elle n’utilise pas le GPU pour l’aider dans ses efforts. Remarque : Canvas dispose désormais d’une composition de mélange qui peut effectuer une partie de la même manipulation de pixels que getImageData.
  • Pour les images .png, getImageData peut ne pas indiquer exactement les mêmes couleurs que dans le fichier .png d’origine, car le navigateur est autorisé à effectuer une correction gamma et une prémultiplication alpha lors du dessin d’images sur le canevas.

Obtenir des couleurs de pixel

Utilisez getImageData pour récupérer les couleurs des pixels pour tout ou partie du contenu de votre canevas.

La méthode getImageData renvoie un objet imageData

L’objet imageData a une propriété .data qui contient les informations de couleur des pixels.

La propriété data est un Uint8ClampedArray contenant les données de couleur rouge, vert, bleu et alpha (opacité) pour tous les pixels demandés.

// determine which pixels to fetch (this fetches all pixels on the canvas)
var x=0;
var y=0;
var width=canvas.width;
var height=canvas.height;

// Fetch the imageData object 
var imageData = context.getImageData(x,y,width,height);

// Pull the pixel color data array from the imageData object
var pixelDataArray = imageData.data;

Vous pouvez obtenir la position de n’importe quel pixel [x,y] dans le tableau data comme ceci :

// the data[] array position for pixel [x,y]
var n = y * canvas.width + x;

Et ensuite, vous pouvez récupérer les valeurs rouge, verte, bleue et alpha de ce pixel comme ceci :

// the RGBA info for pixel [x,y]
var red=data[n];
var green=data[n+1];
var blue=data[n+2];
var alpha=data[n+3];
