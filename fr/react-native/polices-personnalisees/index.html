Polices personnalisées pour Android et IOS #

  • Créez un dossier dans votre dossier de projet et ajoutez-y vos polices. Exemple:

  • Exemple : Ici, nous avons ajouté un dossier à la racine appelé “mystuff”, puis “fonts”, et à l’intérieur, nous avons placé nos polices :

  • Ajoutez le code ci-dessous dans package.json.

    {
    ...

    "rnpm": {
        "assets": [
          "path/to/fontfolder"
        ]
    },

    ...
}

  • Pour l’exemple ci-dessus, notre package.json aurait maintenant un chemin de “mystuff/fonts”:

      "rnpm": {
    "assets": [
      "mystuff/fonts"
    ]
  }

  • Exécutez la commande react-native link.

  • Utilisation de polices personnalisées sur le projet sous le code

    <Text style={{ fontFamily: 'FONT-NAME' }}>
    My Text
</Text>

Où FONT-NAME est le préfixe spécifique à la plate-forme.

Android #

FONT-NAME est les mots avant l’extension dans le fichier. Exemple : le nom de fichier de votre police est Roboto-Regular.ttf, vous devez donc définir fontFamily : Roboto-Regular.

iOS #

FONT-NAME est “Nom complet” trouvé après un clic droit, sur le fichier de police, puis en cliquant sur “Obtenir des informations”. (Source : https://stackoverflow.com/a/16788493/2529614), dans la capture d’écran ci-dessous, le nom du fichier est “MM Proxima Nova Ultra bold.otf”, mais “Nom complet” est “Proxima Nova Semibold”, donc vous définiriez fontFamily: Proxima Nova Semibold. Capture d’écran -

  • Exécutez à nouveau react-native run-ios ou react-native run-android (ceci recompilera avec les ressources)

Étapes pour utiliser des polices personnalisées dans React Native (Android) #

  1. Collez votre fichier de polices dans android/app/src/main/assets/fonts/font_name.ttf
  2. Recompilez l’application Android en exécutant react-native run-android
  3. Maintenant, vous pouvez utiliser fontFamily: 'font_name' dans vos styles natifs React

Étapes pour utiliser des polices personnalisées dans React Native (iOS) #

1. Incluez la police dans votre projet Xcode.

[![Inclure la police au projet Xcode][1]][1]

2. Assurez-vous qu’ils sont inclus dans la colonne Adhésion cible

Cliquez sur la police dans le navigateur et vérifiez si la police est incluse.

[![Vérifier la police dans l’appartenance cible][2]][2]

**3. Vérifiez si la police est incluse en tant que ressource dans votre bundle **

cliquez sur votre fichier de projet Xcode, sélectionnez “Build Phases, sélectionnez “Copy Bundle Resources”. Vérifiez si votre police est ajoutée.

[![vérifier dans les phases de construction][3]][3]

4. Inclure la police dans Application Plist (Info.plist)

à partir du dossier principal de l’application, ouvrez Info.plist, cliquez sur “Information Property List”, puis cliquez sur le signe plus (+). dans la liste déroulante, choisissez “Polices fournies par l’application”.

[![entrez la description de l’image ici][4]][4]

**5. Ajouter le nom de la police dans les polices fournies par l’application **

développez les polices fournies par l’application et ajoutez le nom de la police exactement à la colonne de valeur

[![entrez la description de l’image ici][5]][5]

  1. Utilisez-le dans l’application

     <Text style={{fontFamily:'IndieFlower'}}>
   Welcome to React Native!
 </Text>
