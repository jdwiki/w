Navigation de base #

Aller à l’éditeur (depuis la fenêtre de l’outil) #

Échap

Basculer le focus vers la fenêtre d’outils correspondante #

Windows : Alt + <numéro de la fenêtre d’outils> #

OS X / macOS : Cmd + <numéro de fenêtre d’outil> #

Par exemple, basculer le focus sur la fenêtre du projet #

Windows : Alt + 1 #

OS X / macOS : Cmd + 1 #

Fenêtre contextuelle des fichiers récents #

Windows : Ctrl + E #

OS X / macOS : Cmd + E #

Rechercher une action #

Windows : Ctrl + Maj + A #

OS X / macOS : Cmd + Maj + A #

Aller vers #

  • Dossier : #

    • Windows: Ctrl + Shift + N #

    • OS X / macOS: Cmd + Shift + N #

  • Classer : #

    • Windows: Ctrl + N #

    • OS X / macOS: Cmd + N #

  • Symbole (classe/méthode/variable/nom de constante) : #

    • Windows: Ctrl + Alt + Shift + N #

    • OS X / macOS: Cmd + Option + Shift + N #

    Note that you can use class name to narrow down the method/variable/constant search, for example to find symbol usersCollection in class UserDAO type:

      UserDAO.usersCollection

  • Partout : #

    • Windows: Shift + Shift #

    • OS X / macOS: Shift + Shift #

Pour rechercher quelque chose qui a plusieurs mots, par exemple, InetAddressCachePolicy vous pouvez simplement taper InAddCacPo ou quelque chose de similaire qui contient des parties de mots dans le nom entier.

Aller au numéro de ligne #

Windows : Ctrl + G #

OS X / macOS : Cmd + L #

Revenir au dernier emplacement de modification #

Windows : Ctrl + Maj + Retour arrière #

OS X / macOS : Cmd + Maj + Retour arrière #

Sélection #

Sélection à portée croissante

Cela est pratique lorsque vous souhaitez sélectionner un bloc pour extraire une variable / méthode, etc., pas besoin de faire une correspondance précise entre parenthèses, placez simplement le caret quelque part dans l’instruction et continuez ainsi

Windows : Ctrl + W #

OS X / macOS : Cmd + W #

Sélection à périmètre décroissant

Windows : Ctrl + Maj + W #

OS X / macOS : Cmd + Maj + W #

Cette fonctionnalité est également très utile lors de l’édition/de la lecture de documents json dans votre IDE.

Sélection verticale

Appuyez et maintenez

Windows : Alt #

OS X / macOS : Opt #

et sélectionnez normalement à l’aide de la souris / du trackpad (la façon dont vous sélectionnez un mot à la suite, etc.)

C’est comme ça que ça devrait ressembler

Carets multiples

Appuyez et maintenez

Windows : Alt + Maj #

OS X / macOS : Opt + Maj #

et cliquez là où vous voulez mettre un signe d’insertion. Vous pouvez choisir de placer plusieurs carets sur une seule ligne ou sur plusieurs lignes à différentes positions.

Vous pouvez maintenant effectuer toutes les opérations que vous auriez pu effectuer sur un seul mot sélectionné (maintenez Ctrl (windows) ou option (mac OS) et utilisez Touches gauche ou droite pour passer d’un mot à l’autre) et toutes celles-ci affecteront toutes les positions du curseur.

Vous pouvez même couper/coller plusieurs sélections d’un endroit à un autre.

Avoir plusieurs carets est très utile lorsque vous souhaitez modifier la structure du texte sur plusieurs lignes / plusieurs positions sur la même ligne.

Sélectionner les occurrences en double

Sélectionnez du texte et appuyez sur

Windows : Alt + J #

OS X / macOS : ctrl + G #

pour sélectionner l’occurrence suivante du même texte.

Vous obtenez un curseur à chaque occurrence sélectionnée qui peut être utilisé pour modifier chaque occurrence simultanément.
