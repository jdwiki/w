Bonjour le monde! #

Cela vous apprendra comment réaliser votre premier projet avec IDEA.

Lancez IDEA et cliquez sur “Créer un nouveau projet” dans l’écran de démarrage : [![Écran de démarrage][1]][1] Cliquez sur “Suivant” sur l’écran suivant. Nous créons un projet Java simple, nous n’avons donc pas besoin d’addons ou d’extras pour ce projet [![Pas d’action requise! Cliquez simplement sur Suivant !][2]][2] Utilisez l’écran suivant pour créer le modèle de projet “Java Hello World” : [![Créer à partir d’un modèle][3]][3] Enfin, nommez votre projet et sélectionnez un emplacement sur le disque, puis cliquez sur “Terminer” : [![Nommez votre projet][4]][4] Vous devriez vous retrouver avec une fenêtre qui ressemble à ceci : [![Fenêtre principale][5]][5] À ce stade, le projet est prêt à démarrer, cliquez simplement sur le bouton “Exécuter” ou allez dans “Exécuter -> Exécuter “Main” [![Exécutez votre projet][6]][6] Et tu as fini! La console apparaîtra automatiquement, donnant ses salutations au monde ! [![Bonjour le monde !][7]][7]

Installation ou configuration #

Il existe deux versions principales d’IntelliJ IDEA : l’édition Community et l’édition Ultimate. L’édition Community est gratuite et ne manque pas de fonctionnalités en termes de développement Java SE.

Windows et Linux #

Téléchargez IntelliJ IDEA à partir du [site Web de JetBrains][1] et suivez les procédures d’installation. Si le kit de développement Java (JDK) n’est pas installé, [téléchargez et installez le JDK][2]. Notez que vous avez besoin du JDK, avoir uniquement l’environnement d’exécution Java (JRE) n’est pas suffisant.

Une fois IntelliJ IDEA téléchargé :

  1. Exécutez le programme d’installation
  2. Appuyez sur suivant
  3. Choisissez un dossier pour installer IntelliJ IDEA (dans la plupart des cas, laissez-le par défaut)
  4. Choisissez un dossier de menu Démarrer pour créer des raccourcis IntelliJ IDEA (dans la plupart des cas, laissez-le par défaut)
  5. Choisissez de créer ou non un raccourci sur le bureau et choisissez d’associer divers fichiers Java à IntelliJ IDEA
  6. Appuyez sur suivant et attendez qu’il s’installe

OS X / macOS #

Téléchargez IntelliJ IDEA depuis [le site Web de JetBrains][1], ouvrez le fichier d’image disque (*.dmg) téléchargé, puis faites glisser et déposez l’application vers l’alias de votre dossier /Applications.

![Installateur Mac][3]

Arch Linux #

IntelliJ IDEA peut être installé sur Arch Linux à l’aide de son gestionnaire de packages, “pacman”. Ouvrez un terminal et entrez la commande suivante.

sudo pacman -S intellij-idea-community-edition

L’utilisation de sudo n’est pas nécessaire si vous exécutez en tant qu’utilisateur root.

[![Installation d’Arch Linux][4]][4]

Ubuntu #

(1) Installez le paquet ubuntu-make.

Pour Ubuntu 16.04 et versions ultérieures,

sudo apt install ubuntu-make

Pour les versions précédentes d’ubuntu,

sudo add-apt-repository ppa:ubuntu-desktop/ubuntu-make  
sudo apt-get update
sudo apt-get install ubuntu-make

(2) Après avoir installé Ubuntu Make, effectuez une

umake ide idea

Chemin d’installation par défaut : /home/current-user/.local/share/umake/ide/idea

Suivez le projet hello_world listé ci-dessus.

Suivez [ubuntu-make page][5] pour modifier l’installation par défaut et pour installer d’autres IDE.

Migrer depuis Eclipse #

Intellij IDEA tente de faire appel à la large base de fans Java qui utilise Eclipse pour leur développement en permettant aux développeurs de migrer leurs projets Eclipse vers une structure IDEA en quelques clics simples !

Tout d’abord, démarrez IDEA et cliquez sur “Importer un projet” dans la fenêtre de démarrage : [![Cliquez sur “Importer un projet”][1]][1] Ensuite, sélectionnez votre projet Eclipse à l’aide de la fenêtre de l’explorateur [![Sélectionnez votre projet Eclipse][2]][2]

Intellij vous demandera le modèle à partir duquel vous importez, assurez-vous que “Eclipse” est sélectionné avant de cliquer sur “Suivant” [![Cliquez simplement sur “Suivant”][3]][3]

L’écran suivant affichera une confirmation du chemin que vous souhaitez importer, cliquez simplement sur “Suivant”: [![Cliquez sur “Suivant”][4]][4]

Ensuite, sélectionnez les modules que vous souhaitez créer. Dans l’exemple de projet particulier, seuls les projets Alice et BuggyRos importaient lorsque vous travailliez dans Eclipse [![Sélectionnez les modules que vous souhaitez créer][5]][5]

Enfin, assurez-vous que la version correcte du JDK est sélectionnée avant de “Terminer” [![Sélectionnez le bon JDK][6]][6]

Et le projet Eclipse a été entièrement migré vers Intellij ! Le projet s’ouvrira toujours dans les deux IDE et sera entièrement fonctionnel dans les deux
