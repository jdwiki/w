Syntaxe #

  • window.history.pushState(domaine, titre, chemin);
  • window.history.replaceState (domaine, titre, chemin);

Paramètres #

Paramètre Détails
domaine Le domaine vers lequel vous souhaitez mettre à jour
titre Le titre à mettre à jour
chemin Le chemin d’accès à la mise à jour

L’API HTML5 History n’est pas implémentée par tous les navigateurs et les implémentations ont tendance à différer entre les fournisseurs de navigateurs. Il est actuellement pris en charge par les navigateurs suivants :

-Firefox 4+

  • Google Chrome
  • Internet Explorer 10+ -Safari 5+ -iOS 4

Si vous souhaitez en savoir plus sur les implémentations et les méthodes de l’API History, veuillez consulter [l’état de l’API History HTML5][1].

[1] : https://github.com/browserstate/history.js/wiki/The-State-of-the-HTML5-History-API#coherence

history.pushState() #

Syntaxe :

history.pushState(state object, title, url)

Cette méthode permet d’AJOUTER des entrées d’historiques. Pour plus de référence, veuillez consulter ce document : [méthode pushState()][1]

Exemple :

window.history.pushState("http://example.ca", "Sample Title", "/example/path.html");

Cet exemple insère un nouvel enregistrement dans l’historique, la barre d’adresse et le titre de la page.

Notez que ceci est différent de history.replaceState(). Ce qui met à jour l’entrée d’historique actuelle, plutôt que d’en ajouter une nouvelle.

[1] : https://developer.mozilla.org/en-US/docs/Web/API/History_API#The_pushState()_method

history.replaceState() #

Syntaxe :

history.replaceState(data, title [, url ])

Cette méthode modifie l’entrée d’historique actuelle au lieu d’en créer une nouvelle. Principalement utilisé lorsque nous voulons mettre à jour l’URL de l’entrée d’historique actuelle.

window.history.replaceState("http://example.ca", "Sample Title", "/example/path.html");

Cet exemple remplace l’historique actuel, la barre d’adresse et le titre de la page.

Notez que ceci est différent de history.pushState(). Ce qui insère une nouvelle entrée d’historique, plutôt que de remplacer l’actuelle.

Charger une URL spécifique à partir de la liste de l’historique #

méthode go()

La méthode go() charge une URL spécifique à partir de la liste d’historique. Le paramètre peut être soit un nombre qui va à l’URL dans la position spécifique (-1 recule d’une page, 1 avance d’une page), soit une chaîne. La chaîne doit être une URL partielle ou complète, et la fonction ira à la première URL qui correspond à la chaîne.

Syntaxe

history.go(number|URL)

Exemple

Cliquez sur le bouton pour revenir en arrière de deux pages :

<html>
    <head>
    <script type="text/javascript">
        function goBack()
        {
            window.history.go(-2)
        }
    </script>
    </head>
    <body>    
        <input type="button" value="Go back 2 pages" onclick="goBack()" />    
    </body>
</html>
