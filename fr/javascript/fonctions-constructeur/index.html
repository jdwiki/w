Fonctions constructeur

  • Déclarer une fonction constructeur

Sur cette page

  • Déclarer une fonction constructeur

Les fonctions de constructeur ne sont en fait que des fonctions normales, elles n’ont rien de spécial. C’est seulement le mot-clé new qui provoque le comportement spécial montré dans les exemples ci-dessus. Les fonctions de constructeur peuvent toujours être appelées comme une fonction normale si vous le souhaitez, auquel cas vous devrez lier explicitement la valeur this.

Déclarer une fonction constructeur #

Les fonctions constructeurs sont des fonctions conçues pour construire un nouvel objet. Dans une fonction constructeur, le mot-clé “this” fait référence à un objet nouvellement créé auquel des valeurs peuvent être attribuées. Les fonctions constructeur “renvoyent” automatiquement ce nouvel objet.

function Cat(name) {
  this.name = name;
  this.sound = "Meow";
}

Les fonctions du constructeur sont appelées à l’aide du mot-clé new :

let cat = new Cat("Tom");
cat.sound; // Returns "Meow"

Les fonctions constructeur ont également une propriété prototype qui pointe vers un objet dont les propriétés sont automatiquement héritées par tous les objets créés avec ce constructeur :

Cat.prototype.speak = function() {
  console.log(this.sound);
}

cat.speak(); // Outputs "Meow" to the console

Les objets créés par les fonctions constructeur ont également une propriété spéciale sur leur prototype appelée constructor, qui pointe vers la fonction utilisée pour les créer :

cat.constructor // Returns the `Cat` function

Les objets créés par les fonctions constructeur sont également considérés comme des “instances” de la fonction constructeur par l’opérateur instanceof :

cat instanceof Cat // Returns "true"
