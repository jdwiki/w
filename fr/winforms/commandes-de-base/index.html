Bouton #

Les boutons sont l’un des contrôles les plus simples et sont principalement utilisés pour exécuter du code lorsque l’utilisateur le souhaite.

Ici, nous avons un cas très simple, afficher une boîte de message lorsqu’un bouton est cliqué. Nous ajoutons un bouton à un formulaire, nommons-le cmdShowMessage tel qu’il est utilisé dans le code pour identifier l’objet et définissons le texte des boutons sur Afficher le message.

[![entrez la description de l’image ici][1]][1]

Nous avons juste besoin de double-cliquer sur le bouton du concepteur visuel et Visual Studio générera le code pour l’événement de clic. Il ne nous reste plus qu’à y ajouter le code de la MessageBox :

private void cmdShowMessage_Click(object sender, EventArgs e)
{
  MessageBox.Show("Hello world!");
}

Si nous exécutons le programme maintenant et cliquons sur le bouton, nous verrons apparaître le message :

[![entrez la description de l’image ici][2]][2]

Zone de texte #

Les zones de texte permettent à l’utilisateur d’entrer des données dans le programme.

Nous allons modifier le formulaire et ajouter une zone de texte afin que la boîte de message nous montre le message que l’utilisateur souhaite. Maintenant, notre formulaire ressemble à :

[![entrez la description de l’image ici][1]][1]

Et puis modifiez l’événement de clic de bouton pour utiliser le texte de la zone de texte :

private void cmdShowMessage_Click(object sender, EventArgs e)
{
  string UserText = txtUserMessage.Text;
  MessageBox.Show(UserText);
}

Comme vous pouvez le voir, nous utilisons la propriété .Text de la Textbox qui est le texte contenu dans la texbox.

Si nous exécutons le programme, nous pourrons écrire dans la zone de texte. Lorsque nous cliquons sur le bouton, la MessageBox affichera le texte que nous avons écrit :

Boîte combo #

Les ComboBoxes permettent à l’utilisateur de choisir l’une des différentes options fournies par le développeur.

Nous allons modifier le formulaire et ajouter une combobox pour que la messagebox nous montre le message que l’utilisateur souhaite parmi une liste que nous fournirons.

Après avoir ajouté le combo au formulaire, nous ajoutons maintenant une liste d’options au combo. Pour ce faire, nous devons modifier la propriété Items :

[![entrez la description de l’image ici][1]][1]

Nous devons maintenant modifier le code de l’événement click :

private void cmdShowMessage_Click(object sender, EventArgs e)
{
  string OptionText = cboSelectOption.SelectedItem.ToString();
  MessageBox.Show(OptionText);
}

Comme vous pouvez le voir, nous utilisons la propriété SelectedItem, elle contient l’objet de l’option sélectionnée. Comme nous avons besoin d’une chaîne à afficher et que le compilateur ne sait pas si l’objet est ou non une chaîne, nous devons utiliser la méthode ToString().

Si nous exécutons le programme, nous pourrons choisir l’option que nous préférons et lorsque nous cliquerons sur le bouton, la boîte de message l’affichera :

[![entrez la description de l’image ici][2]][2]

Pour être averti lorsqu’un utilisateur sélectionne un élément dans la liste déroulante, utilisez l’événement SelectionChangeCommitted. Nous pourrions utiliser l’événement SelectedIndexChanged, mais cela est également déclenché lorsque nous modifions par programme l’élément de sélection dans la liste déroulante.

Case à cocher #

La case à cocher est un contrôle qui permet à l’utilisateur d’obtenir des valeurs “booléennes” de l’utilisateur pour une question spécifique telle que “Ça va ?”.

Possède un événement appelé CheckedChanged, qui se produit chaque fois que la propriété check est modifiée.

Voici une case à cocher qui a une question “Is Checked?”.

[![entrez la description de l’image ici][1]][1]

Nous avons reçu cette MessageBox de l’événement CheckedChanged,

private void checkBox1_CheckedChanged(object sender, EventArgs e)
        {
            bool IsChecked = checkBox1.Checked; 
            MessageBox.Show(IsChecked.ToString());
        }

Si CheckBox est coché -> la variable IsChecked sera true.

Si CheckBox n’est pas coché -> la variable IsChecked sera false.

Zone de liste #

Listbox est un contrôle qui peut contenir une collection d’objets. Listbox est similaire à Combobox mais dans Combobox ; Seuls les éléments sélectionnés sont visibles pour l’utilisateur. Dans Listebox ; tous les éléments sont visibles pour l’utilisateur.

Comment ajouter des éléments à ListBox ?

private void Form3_Load(object sender, EventArgs e)
        {
            string test = "Trial";
            string test2 = "45";
            int numberTest = 43;
            decimal decimalTest = 130;
            
            listBox1.Items.Add(test);
            listBox1.Items.Add(test2);
            listBox1.Items.Add(numberTest);
            listBox1.Items.Add(decimalTest);
        }

NumericUpDown #

NumericUpDown est un contrôle qui ressemble à TextBox. Ce contrôle permet à l’utilisateur d’afficher/sélectionner un nombre dans une plage. Les flèches haut et bas mettent à jour la valeur de la zone de texte.

Le contrôle ressemble à;

[![entrez la description de l’image ici][1]][1]

Dans Form_Load, la plage peut être définie.

private void Form3_Load(object sender, EventArgs e)
    {
        numericUpDown1.Maximum = 10;
        numericUpDown1.Minimum = -10; 
    }

Production;

[![entrez la description de l’image ici][2]][2]

UpDownAlign définira la position des flèches ;

 private void Form3_Load(object sender, EventArgs e)
        {
            numericUpDown1.UpDownAlign = LeftRightAlignment.Left;
            
        }

Production;

[![entrez la description de l’image ici][3]][3]

La méthode UpButton() augmente le numéro du contrôle. (peut être appelé de n’importe où. J’ai utilisé un “bouton” pour l’appeler.)

private void button1_Click(object sender, EventArgs e)
        {
            numericUpDown1.UpButton();
        }

Événements utiles #

Valeur modifiée ;

Cet événement fonctionnera lorsque vous cliquez sur la flèche Haut ou Bas.

private void numericUpDown1_ValueChanged(object sender, EventArgs e)
        {
            decimal result = numericUpDown1.Value; // it will get the current value
            if (result == numericUpDown1.Maximum) // if value equals Maximum value that we set in Form_Load.
            {
                label1.Text = result.ToString() + " MAX!"; // it will add "MAX" at the end of the label 
            }
            else if (result == numericUpDown1.Minimum) //  if value equals Minimum value that we set in Form_Load.
            {
                label1.Text = result.ToString() + " MIN!"; // it will add "MIN" at the end of the label 
            }
            else
            {
                label1.Text = result.ToString(); // If Not Max or Min, it will show only the number.
            }
        }
