APCu

  • Itération sur les entrées

  • Informations sur le magasin

APCu est un magasin clé-valeur de mémoire partagée pour PHP. La mémoire est partagée entre Processus PHP-FPM du même pool. Les données stockées persistent entre les requêtes.

Itération sur les entrées #

Le APCUIterator permet de itérer sur les entrées du cache :

foreach (new APCUIterator() as $entry) {
    print_r($entry);
}

L’itérateur peut être initialisé avec une expression régulière facultative pour sélectionner uniquement les entrées avec des clés correspondantes :

foreach (new APCUIterator($regex) as $entry) {
    print_r($entry);
}

Les informations sur une seule entrée de cache peuvent être obtenues via :

$key = '…';
$regex = '(^' . preg_quote($key) . '$)';
print_r((new APCUIterator($regex))->current());

Stockage et récupération simples #

apcu_store peut être utilisé pour stocker, apcu_fetch pour récupérer les valeurs :

$key = 'Hello';
$value = 'World';
apcu_store($key, $value);
print(apcu_fetch('Hello')); // 'World'

Informations sur le magasin #

apcu_cache_info fournit informations sur le magasin et ses entrées:

print_r(apcu_cache_info());

Notez que l’invocation de apcu_cache_info() sans limite renverra le données actuellement stockées. Pour obtenir uniquement les métadonnées, utilisez apcu_cache_info(true). Pour obtenir des informations sur certaines entrées de cache, mieux vaut utiliser APCUITerator.
