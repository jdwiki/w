getTimestamp #

getTimeStemp est une représentation unix d’un objet datetime.

$date = new DateTime();
echo $date->getTimestamp();

cela affichera une indication entière des secondes qui se sont écoulées depuis 00:00:00 UTC, jeudi 1er janvier 1970.

régler la date #

setDate définit la date dans un objet DateTime.

$date = new DateTime();
$date->setDate(2016, 7, 25);

cet exemple définit la date comme étant le vingt-cinq juillet 2015, il produira le résultat suivant :

2016-07-25 17:52:15.819442

Ajouter ou soustraire des intervalles de dates #

Nous pouvons utiliser la classe [DateInterval][1] pour ajouter ou soustraire un intervalle dans un objet DateTime.

Voir l’exemple ci-dessous, où nous ajoutons un intervalle de 7 jours et imprimons un message à l’écran :

$now = new DateTime();// empty argument returns the current date
$interval = new DateInterval('P7D');//this objet represents a 7 days interval
$lastDay = $now->add($interval); //this will return a DateTime object
$formatedLastDay = $lastDay->format('Y-m-d');//this method format the DateTime object and returns a String
echo "Samara says: Seven Days. You'll be happy on $formatedLastDay.";

Cela affichera (fonctionnant le 1er août 2016):

Samara dit : Sept jours. Vous serez heureux le 2016-08-08.

Nous pouvons utiliser la sous-méthode de la même manière pour soustraire des dates

$now->sub($interval);
echo "Samara says: Seven Days. You were happy last on $formatedLastDay.";

Cela affichera (fonctionnant le 1er août 2016):

Samara dit : Sept jours. Vous étiez heureux pour la dernière fois le 2016-07-25.

[1] : http://php.net/manual/pt_BR/class.dateinterval.php

Créer DateTime à partir d’un format personnalisé #

PHP est capable d’analyser un certain nombre de formats de date. Si vous voulez analyser un format non standard, ou si vous voulez que votre code indique explicitement le format à utiliser, vous pouvez utiliser le [DateTime::createFromFormat](https://php.net/manual /fr/datetime.createfromformat.php) méthode :

Style orienté objet

$format = "Y,m,d";
$time = "2009,2,26";
$date = DateTime::createFromFormat($format, $time);

Style procédural

$format = "Y,m,d";
$time = "2009,2,26";
$date = date_create_from_format($format, $time);

Impression des dates et heures #

PHP 4+ fournit une méthode, format qui convertit un objet DateTime en une chaîne avec un format souhaité. Selon le manuel PHP, il s’agit de la fonction orientée objet :

public string DateTime::format ( string $format )

La fonction date() prend un paramètre - un format, qui est une chaîne

#Format

Le format est une chaîne et utilise des caractères uniques pour définir le format :

  • Y : représentation à quatre chiffres de l’année (ex : 2016)
  • y : représentation à deux chiffres de l’année (ex : 16)
  • m : mois, sous forme de chiffre (01 à 12)
  • M : mois, en trois lettres (Jan, Fév, Mar, etc.)
  • j : jour du mois, sans zéro de tête (1 à 31)
  • D : jour de la semaine, en trois lettres (Lun, Mar, Mer, etc.)
  • h : heure (format 12 heures) (01 à 12)
  • H : heure (format 24 heures) (00 à 23)
  • A : soit AM soit PM
  • i : minute, avec des zéros non significatifs (00 à 59)
  • s : seconde, avec des zéros non significatifs (00 à 59)
  • La liste complète peut être trouvée [ici][1]

Utilisation #

Ces caractères peuvent être utilisés dans diverses combinaisons pour afficher les heures dans pratiquement n’importe quel format. Voici quelques exemples:

$date = new DateTime('2000-05-26T13:30:20'); /* Friday, May 26, 2000 at 1:30:20 PM */

$date->format("H:i");
/* Returns 13:30 */

$date->format("H i s");
/* Returns 13 30 20 */

$date->format("h:i:s A");
/* Returns 01:30:20 PM */

$date->format("j/m/Y");
/* Returns 26/05/2000 */

$date->format("D, M j 'y - h:i A");
/* Returns Fri, May 26 '00 - 01:30 PM */

procédural #

Le format procédural est similaire :

Orienté objet #

$date->format($format)

Équivalent procédural #

date_format($date, $format)

[1] : http://php.net/manual/en/function.date.php

Créer une version immuable de DateTime à partir de Mutable avant PHP 5.6 #

Pour créer \DateTimeImmutable en PHP 5.6+, utilisez :

\DateTimeImmutable::createFromMutable($concrete);

Avant PHP 5.6, vous pouvez utiliser :

\DateTimeImmutable::createFromFormat(\DateTime::ISO8601, $mutable->format(\DateTime::ISO8601), $mutable->getTimezone());
