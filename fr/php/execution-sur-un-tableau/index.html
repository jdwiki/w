Appliquer une fonction à chaque élément d’un tableau #

Pour appliquer une fonction à chaque élément d’un tableau, utilisez array_map(). Cela renverra un nouveau tableau.

$array = array(1,2,3,4,5);
//each array item is iterated over and gets stored in the function parameter.
$newArray = array_map(function($item) {
    return $item + 1;
}, $array);

$newArray est maintenant array(2,3,4,5,6);.

Au lieu d’utiliser une [fonction anonyme][1], vous pouvez utiliser une fonction nommée. Ce qui précède pourrait s’écrire comme suit :

function addOne($item) {
    return $item + 1;
}

$array = array(1, 2, 3, 4, 5);
$newArray = array_map('addOne', $array);

Si la fonction nommée est une méthode de classe, l’appel de la fonction doit inclure une référence à un objet de classe auquel la méthode appartient :

class Example {
    public function addOne($item) {
        return $item + 1;
    }

    public function doCalculation() {
        $array = array(1, 2, 3, 4, 5);
        $newArray = array_map(array($this, 'addOne'), $array);
    }
}

Une autre façon d’appliquer une fonction à chaque élément d’un tableau est array_walk() et array_walk_recursive(). Le rappel transmis à ces fonctions prend à la fois la clé/l’index et la valeur de chaque élément du tableau. Ces fonctions ne renverront pas un nouveau tableau, mais un booléen en cas de succès. Par exemple, pour imprimer chaque élément d’un tableau simple :

$array = array(1, 2, 3, 4, 5);
array_walk($array, function($value, $key) {
    echo $value . ' ';
});
// prints "1 2 3 4 5"

Le paramètre value du callback peut être passé par référence, vous permettant de modifier la valeur directement dans le tableau d’origine :

$array = array(1, 2, 3, 4, 5);
array_walk($array, function(&$value, $key) {
    $value++;
});

$array est maintenant array(2,3,4,5,6);

Pour les tableaux imbriqués, array_walk_recursive() ira plus loin dans chaque sous-tableau :

$array = array(1, array(2, 3, array(4, 5), 6);
array_walk_recursive($array, function($value, $key) {
    echo $value . ' ';
});
// prints "1 2 3 4 5 6"

Remarque : array_walk et array_walk_recursive vous permettent de modifier la valeur des éléments du tableau, mais pas les clés. Le passage des clés par référence dans le rappel est valide mais n’a aucun effet.

[1] : https://www.wikiod.com/fr/php/programmation-fonctionnelle

Diviser le tableau en morceaux #

[array_chunk()][1] divise un tableau en morceaux

Disons que nous suivons un tableau unidimensionnel,

$input_array = array('a', 'b', 'c', 'd', 'e');

Maintenant, en utilisant array_chunk() sur le tableau PHP ci-dessus,

$output_array = array_chunk($input_array, 2);

Le code ci-dessus créera des morceaux de 2 éléments de tableau et créera un tableau multidimensionnel comme suit.

Array
(
    [0] => Array
        (
            [0] => a
            [1] => b
        )

    [1] => Array
        (
            [0] => c
            [1] => d
        )

    [2] => Array
        (
            [0] => e
        )

)

Si tous les éléments du tableau ne sont pas divisés de manière égale par la taille du bloc, le dernier élément du tableau de sortie sera les éléments restants.

Si nous passons le deuxième argument à moins de 1, alors E_WARNING sera lancé et le tableau de sortie sera NULL.

Paramètre Détails
$tableau (tableau) Tableau d’entrée, le tableau sur lequel travailler
$taille (entier) Taille de chaque morceau (valeur entière)
$preserve_keys (booléen) (facultatif) Si vous voulez que le tableau de sortie préserve les clés, définissez-le sur TRUE sinon FALSE.

[1] : http://php.net/manual/en/function.array-chunk.php

Imploser un tableau en chaîne #

implode() combine toutes les valeurs du tableau mais perd toutes les informations clés :

$arr = ['a' => "AA", 'b' => "BB", 'c' => "CC"];

echo implode(" ", $arr); // AA BB CC

L’implosion des clés peut être effectuée à l’aide de l’appel array_keys() :

$arr = ['a' => "AA", 'b' => "BB", 'c' => "CC"];

echo implode(" ", array_keys($arr)); // a b c

Imploser des clés avec des valeurs est plus complexe mais peut être fait en utilisant le style fonctionnel :

$arr = ['a' => "AA", 'b' => "BB", 'c' => "CC"];

echo implode(" ", array_map(function($key, $val) { 
    return "$key:$val"; // function that glues key to the value
}, array_keys($arr), $arr)); 

// Output: a:AA b:BB c:CC

array_reduce #

array_reduce reduces array into a single value. Basically, The array_reduce will go through every item with the result from last iteration and produce new value to the next iteration.

Usage: array_reduce ($array, function($carry, $item){...}, $defaul_value_of_first_carry)

  • $carry est le résultat du dernier tour d’itération.
  • $item est la valeur de la position actuelle dans le tableau.

Somme du tableau

$result = array_reduce([1, 2, 3, 4, 5], function($carry, $item){
    return $carry + $item;
});

résultat :15

Le plus grand nombre dans le tableau

$result = array_reduce([10, 23, 211, 34, 25], function($carry, $item){
        return $item > $carry ? $item : $carry;
});

résultat :211

** Est-ce que tous les articles sont supérieurs à 100 **

$result = array_reduce([101, 230, 210, 341, 251], function($carry, $item){
        return $carry && $item > 100;
}, true); //default value must set true

résultat :vrai

Est-ce qu’un élément est inférieur à 100

$result = array_reduce([101, 230, 21, 341, 251], function($carry, $item){
        return $carry || $item < 100;
}, false);//default value must set false

résultat :vrai

Comme imploser($array, $piece)

$result = array_reduce(["hello", "world", "PHP", "language"], function($carry, $item){
        return !$carry ? $item : $carry . "-" . $item ;
});

résultat :"hello-world-PHP-language"

si faire une méthode implode, le code source sera :

function implode_method($array, $piece){
    return array_reduce($array, function($carry, $item) use ($piece) {
            return !$carry ? $item : ($carry . $piece . $item);
    });
}

$result = implode_method(["hello", "world", "PHP", "language"], "-");

résultat :"hello-world-PHP-language"

“Destructuration” des tableaux en utilisant list() #

Utilisez [list()][1] pour assigner rapidement une liste de valeurs de variables dans un tableau. Voir aussi [compact()][2]

// Assigns to $a, $b and $c the values of their respective array elements in           $array with keys numbered from zero
list($a, $b, $c) = $array;

Avec PHP 7.1 (actuellement en version bêta), vous pourrez utiliser [short list syntax][3] :

// Assigns to $a, $b and $c the values of their respective array elements in $array with keys numbered from zero
[$a, $b, $c] = $array;

// Assigns to $a, $b and $c the values of the array elements in $array with the keys "a", "b" and "c", respectively
["a" => $a, "b" => $b, "c" => $c] = $array;

[1] : http://php.net/manual/en/function.list.php [2] : https://www.wikiod.com/fr/php/tableaux#Création d’un tableau de variables [3] : https://wiki.php.net/rfc/short_list_syntax

Pousser une valeur sur un tableau #

Il existe deux manières de pousser un élément vers un tableau : array_push et $array[] =

Le [array_push][1] est utilisé comme ceci :

$array = [1,2,3];
$newArraySize = array_push($array, 5, 6); // The method returns the new size of the array
print_r($array); // Array is passed by reference, therefore the original array is modified to contain the new elements

Ce code imprimera :

Array
(
    [0] => 1
    [1] => 2
    [2] => 3
    [3] => 5
    [4] => 6
)

$array[] = est utilisé comme ceci :

$array = [1,2,3];
$array[] = 5;
$array[] = 6;
print_r($array);

Ce code imprimera :

Array
(
    [0] => 1
    [1] => 2
    [2] => 3
    [3] => 5
    [4] => 6
)

[1] : http://php.net/manual/fr/function.array-push.php
