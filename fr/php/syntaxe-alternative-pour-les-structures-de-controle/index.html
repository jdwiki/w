Syntaxe #

  • structure : /* code */ endstructure ;

Lorsque vous mélangez la structure alternative pour switch avec HTML, il est important de ne pas avoir d’espace entre le switch($condition): initial et le premier case $value:. Faire cela, c’est essayer de faire écho à quelque chose (espace blanc) avant un cas.

Toutes les structures de contrôle suivent la même idée générale. Au lieu d’utiliser des accolades pour encapsuler le code, vous utilisez deux-points et l’instruction endstructure; : structure: /* code */ endstructure;

Autre instruction if/else #

<?php

if ($condition):
    do_something();
elseif ($another_condition):
    do_something_else();
else:
    do_something_different();
endif;

?>

<?php if ($condition): ?>
    <p>Do something in HTML</p>
<?php elseif ($another_condition): ?>
    <p>Do something else in HTML</p>
<?php else: ?>
    <p>Do something different in HTML</p>
<?php endif; ?>

Alternative pour l’instruction #

<?php

for ($i = 0; $i < 10; $i++):
    do_something($i);
endfor;

?>

<?php for ($i = 0; $i < 10; $i++): ?>
    <p>Do something in HTML with <?php echo $i; ?></p>
<?php endfor; ?>

Autre instruction while #

<?php

while ($condition):
    do_something();
endwhile;

?>

<?php while ($condition): ?>
    <p>Do something in HTML</p>
<?php endwhile; ?>

Instruction foreach alternative #

<?php

foreach ($collection as $item):
    do_something($item);
endforeach;

?>

<?php foreach ($collection as $item): ?>
    <p>Do something in HTML with <?php echo $item; ?></p>
<?php endforeach; ?>

Autre instruction switch #

<?php

switch ($condition):
    case $value:
        do_something();
        break;
    default:
        do_something_else();
        break;
endswitch;

?>

<?php switch ($condition): ?>
<?php case $value: /* having whitespace before your cases will cause an error */ ?>
    <p>Do something in HTML</p>
    <?php break; ?>
<?php default: ?>
    <p>Do something else in HTML</p>
    <?php break; ?>
<?php endswitch; ?>
