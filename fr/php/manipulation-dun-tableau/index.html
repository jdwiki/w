Filtrer un tableau #

Afin de filtrer les valeurs d’un tableau et d’obtenir un nouveau tableau contenant toutes les valeurs qui satisfont la condition de filtre, vous pouvez utiliser la fonction array_filter.

Filtrage des valeurs non vides #

Le cas le plus simple de filtrage consiste à supprimer toutes les valeurs “vides”:

$my_array = [1,0,2,null,3,'',4,[],5,6,7,8];
$non_empties = array_filter($my_array); // $non_empties will contain [1,2,3,4,5,6,7,8];

Filtrage par rappel #

Cette fois, nous définissons notre propre règle de filtrage. Supposons que nous ne voulions obtenir que des nombres pairs :

$my_array = [1,2,3,4,5,6,7,8];

$even_numbers = array_filter($my_array, function($number) {
    return $number % 2 === 0;   
});

La fonction array_filter reçoit le tableau à filtrer comme premier argument, et un callback définissant le prédicat du filtre comme second.

Filtrage par index #

Un troisième paramètre peut être fourni à la fonction array_filter, qui permet de modifier les valeurs transmises au rappel. Ce paramètre peut être défini sur ARRAY_FILTER_USE_KEY ou ARRAY_FILTER_USE_BOTH, ce qui entraînera la réception par le rappel de la clé au lieu de la valeur de chaque élément du tableau, ou à la fois de la valeur et de la clé comme arguments. Par exemple, si vous souhaitez traiter des index plutôt que des valeurs :

$numbers = [16,3,5,8,1,4,6];

$even_indexed_numbers = array_filter($numbers, function($index) {
    return $index % 2 === 0;
}, ARRAY_FILTER_USE_KEY);

Index dans le tableau filtré #

Notez que array_filter préserve les clés du tableau d’origine. Une erreur courante serait d’essayer une boucle use for sur le tableau filtré :

<?php

$my_array = [1,0,2,null,3,'',4,[],5,6,7,8];
$filtered = array_filter($my_array); 

error_reporting(E_ALL); // show all errors and notices

// innocently looking "for" loop
for ($i = 0; $i < count($filtered); $i++) {
   print $filtered[$i];
}

/*
Output:
1
Notice: Undefined offset: 1
2
Notice: Undefined offset: 3
3
Notice: Undefined offset: 5
4
Notice: Undefined offset: 7
*/

Cela se produit parce que les valeurs qui étaient sur les positions 1 (il y avait 0), 3 (null), 5 (chaîne vide '') et 7 (tableau vide []) ont été supprimées avec leur correspondant clés d’index.

Si vous devez parcourir le résultat d’un filtre sur un tableau indexé, vous devez d’abord appeler array_values sur le résultat de array_filter afin de créer un nouveau tableau avec les index corrects :

$my_array = [1,0,2,null,3,'',4,[],5,6,7,8];
$filtered = array_filter($my_array); 
$iterable = array_values($filtered);

error_reporting(E_ALL); // show all errors and notices

for ($i = 0; $i < count($iterable); $i++) {
   print $iterable[$i];
}

// No warnings!

Supprimer des éléments d’un tableau #

Pour supprimer un élément à l’intérieur d’un tableau, par ex. l’élément d’indice 1.

$fruit = array("bananas", "apples", "peaches");
unset($fruit[1]);

Cela supprimera les pommes de la liste, mais notez que unset ne modifie pas les index des éléments restants. Donc $fruit contient maintenant les index 0 et 2.

Pour le tableau associatif, vous pouvez supprimer comme ceci :

$fruit = array('banana', 'one'=>'apple', 'peaches');

print_r($fruit);
/*
    Array
    (
        [0] => banana
        [one] => apple
        [1] => peaches
    )
*/

unset($fruit['one']);

Maintenant $fruit est

print_r($fruit);

/*
Array
(
    [0] => banana
    [1] => peaches
)
*/

Notez que

unset($fruit);

annule la variable et supprime ainsi tout le tableau, ce qui signifie qu’aucun de ses éléments n’est plus accessible.

Suppression des éléments terminaux #

[array_shift()][1] - Décaler un élément du début du tableau.

Exemple:

  $fruit = array("bananas", "apples", "peaches");
  array_shift($fruit);
  print_r($fruit);

Production:

 Array
(
    [0] => apples
    [1] => peaches
)

[array_pop()][2] - Pop l’élément à la fin du tableau.

Exemple:

  $fruit = array("bananas", "apples", "peaches");
  array_pop($fruit);
  print_r($fruit);

Production:

 Array
(
    [0] => bananas
    [1] => apples
)

[1] : http://php.net/manual/en/function.array-shift.php [2] : http://php.net/manual/en/function.array-pop.php

Trier un tableau #

Il existe plusieurs fonctions de tri pour les tableaux en php :

trier() #

Trier un tableau par ordre croissant de valeur.

$fruits = ['Zitrone', 'Orange', 'Banane', 'Apfel'];
sort($fruits);
print_r($fruits);

résulte en

Array
(
    [0] => Apfel
    [1] => Banane
    [2] => Orange
    [3] => Zitrone
)

trier() #

Trier un tableau par ordre décroissant de valeur.

$fruits = ['Zitrone', 'Orange', 'Banane', 'Apfel'];
rsort($fruits);
print_r($fruits);

résulte en

Array
(
    [0] => Zitrone
    [1] => Orange
    [2] => Banane
    [3] => Apfel
)

un tri() #

Trier un tableau par ordre croissant de valeur et conserver les indecies.

$fruits = [1 => 'lemon', 2 => 'orange',  3 => 'banana', 4 => 'apple'];
asort($fruits);
print_r($fruits);

résulte en

Array
(
    [4] => apple
    [3] => banana
    [1] => lemon
    [2] => orange
)

arsort() #

Trier un tableau par ordre décroissant de valeur et conserver les indices.

$fruits = [1 => 'lemon', 2 => 'orange',  3 => 'banana', 4 => 'apple'];
arsort($fruits);
print_r($fruits);

résulte en

Array
(
    [2] => orange
    [1] => lemon
    [3] => banana
    [4] => apple
)

trier() #

Trier un tableau dans l’ordre croissant par clé

$fruits = ['d'=>'lemon', 'a'=>'orange', 'b'=>'banana', 'c'=>'apple'];
ksort($fruits);
print_r($fruits);

résulte en

Array
(
    [a] => orange
    [b] => banana
    [c] => apple
    [d] => lemon
)

krssort() #

Trier un tableau par ordre décroissant de clé.

$fruits = ['d'=>'lemon', 'a'=>'orange', 'b'=>'banana', 'c'=>'apple'];
krsort($fruits);
print_r($fruits);

résulte en

Array
(
    [d] => lemon
    [c] => apple
    [b] => banana
    [a] => orange
)

natsort() #

Trier un tableau comme le ferait un être humain (ordre naturel).

$files = ['File8.stack', 'file77.stack', 'file7.stack', 'file13.stack', 'File2.stack'];
natsort($files);
print_r($files);

résulte en

Array
(
    [4] => File2.stack
    [0] => File8.stack
    [2] => file7.stack
    [3] => file13.stack
    [1] => file77.stack
)

natcasesort() #

Trier un tableau comme le ferait un être humain (ordre naturel), mais en cas de casse

$files = ['File8.stack', 'file77.stack', 'file7.stack', 'file13.stack', 'File2.stack'];
natcasesort($files);
print_r($files);

résulte en

Array
(
    [4] => File2.stack
    [2] => file7.stack
    [0] => File8.stack
    [3] => file13.stack
    [1] => file77.stack
)

mélanger() #

Mélange un tableau (trié aléatoirement).

$array = ['aa', 'bb', 'cc'];
shuffle($array);
print_r($array);

Comme écrit dans la description c’est aléatoire donc ici un seul exemple de ce que cela peut entraîner

Array
(
    [0] => cc
    [1] => bb
    [2] => aa
)

usort() #

Trier un tableau avec une fonction de comparaison définie par l’utilisateur.

function compare($a, $b)
{
    if ($a == $b) {
        return 0;
    }
    return ($a < $b) ? -1 : 1;
}

$array = [3, 2, 5, 6, 1];
usort($array, 'compare');
print_r($array);

résulte en

Array
(
    [0] => 1
    [1] => 2
    [2] => 3
    [3] => 5
    [4] => 6
)

uasort() #

Trier un tableau avec une fonction de comparaison définie par l’utilisateur et conserver les clés.

function compare($a, $b)
{
    if ($a == $b) {
        return 0;
    }
    return ($a < $b) ? -1 : 1;
}

$array = ['a' => 1, 'b' => -3, 'c' => 5, 'd' => 3, 'e' => -5];
uasort($array, 'compare');
print_r($array);

résulte en

Array
(
    [e] => -5
    [b] => -3
    [a] => 1
    [d] => 3
    [c] => 5
)

uksort() #

Trier un tableau par clés avec une fonction de comparaison définie par l’utilisateur.

function compare($a, $b)
{
    if ($a == $b) {
        return 0;
    }
    return ($a < $b) ? -1 : 1;
}

$array = ['ee' => 1, 'g' => -3, '4' => 5, 'k' => 3, 'oo' => -5];

uksort($array, 'compare');
print_r($array);

résulte en

Array
(
    [ee] => 1
    [g] => -3
    [k] => 3
    [oo] => -5
    [4] => 5
)

Ajout d’un élément au début du tableau #

Parfois, vous souhaitez ajouter un élément au début d’un tableau ** sans modifier aucun des éléments actuels (* ordre *) dans le tableau **. Chaque fois que c’est le cas, vous pouvez utiliser [array_unshift()][1].

array_unshift() ajoute les éléments passés au début du tableau. Notez que la liste des éléments est préfixée dans son ensemble, de sorte que les éléments ajoutés restent dans le même ordre. Toutes les clés de tableau numérique sera modifié pour commencer à compter à partir de zéro alors que les clés littérales ne le seront pas être touché.

Tiré de la [documentation PHP][2] pour array_unshift().

Si vous souhaitez y parvenir, il vous suffit de procéder comme suit :

$myArray = array(1, 2, 3);

array_unshift($myArray, 4);

Cela va maintenant ajouter 4 comme premier élément de votre tableau. Vous pouvez le vérifier en :

print_r($myArray);

Ceci renvoie un tableau dans l’ordre suivant : 4, 1, 2, 3.

Puisque array_unshift force le tableau à réinitialiser les paires clé-valeur lorsque le nouvel élément laisse les entrées suivantes avoir les clés n+1, il est plus intelligent de créer un nouveau tableau et d’ajouter le tableau existant au nouveau tableau créé.

Exemple:

$myArray = array('apples', 'bananas', 'pears');
$myElement = array('oranges');
$joinedArray = $myElement;

foreach ($myArray as $i) {
  $joinedArray[] = $i;
}

Sortie ($joinedArray) :

Array ( [0] => oranges [1] => apples [2] => bananas [3] => pears )

[Exemple/Démo][3]

[1] : http://php.net/array_unshift [2] : http://php.net/array_unshift#refsect1-function.array-unshift-description [3] : http://www.tehplayground.com/#egwNCrZgr

Liste blanche uniquement certaines clés de tableau #

Lorsque vous souhaitez autoriser uniquement certaines clés dans vos tableaux, en particulier lorsque le tableau provient de paramètres de requête, vous pouvez utiliser array_intersect_key avec array_flip.

$parameters = ['foo' => 'bar', 'bar' => 'baz', 'boo' => 'bam'];
$allowedKeys = ['foo', 'bar'];
$filteredParameters = array_intersect_key($parameters, array_flip($allowedKeys));

// $filteredParameters contains ['foo' => 'bar', 'bar' => 'baz]

Si la variable parameters ne contient aucune clé autorisée, la variable filteredParameters consistera en un tableau vide.

Depuis PHP 5.6, vous pouvez également utiliser [array_filter][1] pour cette tâche, en passant le drapeau [ARRAY_FILTER_USE_KEY][2] comme troisième paramètre :

$parameters  = ['foo' => 1, 'hello' => 'world'];
$allowedKeys = ['foo', 'bar'];
$filteredParameters = array_filter(
    $parameters,
    function ($key) use ($allowedKeys) {
        return in_array($key, $allowedKeys);
    },
    ARRAY_FILTER_USE_KEY
);

L’utilisation de array_filter donne la flexibilité supplémentaire d’effectuer un test arbitraire sur la clé, par ex. $allowedKeys pourrait contenir des motifs regex au lieu de chaînes simples. Il indique également plus explicitement l’intention du code que array_intersect_key() combiné avec array_flip().

[1] : http://php.net/manual/en/function.array-filter.php#refsect1-function.array-filter-changelog [2] : http://php.net/manual/en/array.constants.php#constant.array-filter-use-key

Échange de valeurs avec des clés #

La fonction array_flip échangera toutes les clés avec ses éléments.

$colors = array(
    'one' => 'red',
    'two' => 'blue',
    'three' => 'yellow',
);

array_flip($colors); //will output

array(
    'red' => 'one',
    'blue' => 'two',
    'yellow' => 'three'

Fusionner deux tableaux en un seul tableau #

$a1 = array("red","green");
$a2 = array("blue","yellow");
print_r(array_merge($a1,$a2));

/*
    Array ( [0] => red [1] => green [2] => blue [3] => yellow )
*/

Tableau associatif :

$a1=array("a"=>"red","b"=>"green");
$a2=array("c"=>"blue","b"=>"yellow");
print_r(array_merge($a1,$a2));
/*
    Array ( [a] => red [b] => yellow [c] => blue )
*/
  1. Fusionne les éléments d’un ou plusieurs tableaux ensemble afin que les valeurs de l’un soient ajoutées à la fin du précédent. Il renvoie le tableau résultant.
  2. Si les tableaux d’entrée ont les mêmes clés de chaîne, la dernière valeur de cette clé écrasera la précédente. Si, toutefois, les tableaux contiennent des touches numériques, la dernière valeur n’écrasera pas la valeur d’origine, mais sera ajoutée.
  3. Les valeurs du tableau d’entrée avec des touches numériques seront renumérotées avec des clés d’incrémentation à partir de zéro dans le tableau de résultat.
