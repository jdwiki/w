Une belle introduction aux matières et aux textures.

Textures diffuses, en relief, spéculaires et transparentes.

Paramètres #

Paramètre Détails
couleur Valeur numérique du composant RVB de la couleur.
intensité Valeur numérique de la force/intensité de la lumière.
fov Champ de vision vertical du tronc de la caméra.
aspect Un morceau de caméra regarde le système.
près de Tronc de caméra près de l’avion.
loin Un morceau de caméra est clairement loin.
rayon rayon de la sphère. La valeur par défaut est 50.
largeurSegments nombre de segments horizontaux. La valeur minimale est 3 et la valeur par défaut est 8.
hauteurSegments nombre de segments verticaux. La valeur minimale est 2 et la valeur par défaut est 6.
phiDébut spécifier l’angle de départ horizontal. La valeur par défaut est 0.
phiLongueur spécifier la taille de l’angle de balayage horizontal. La valeur par défaut est Math.PI * 2.
thetaStart spécifier l’angle de départ vertical. La valeur par défaut est 0.
thetaLength spécifier la taille de l’angle de balayage vertical. La valeur par défaut est Math.PI.

Création d’un modèle de Terre #

Les textures de cet exemple sont disponibles sur : http://planetpixelemporium.com/planets.html

Installation ou configuration

Vous pouvez en installer trois via npm

npm install three

Ou ajoutez-le sous forme de script à votre page HTML

<script type="text/javascript" src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r85/three.min.js" />

HTML :

<html>
<head>
    <meta charset=utf-8>
    <title>Earth Model</title>
    <style>
        body { margin: 0; }
        canvas { width: 100%; height: 100% }
    </style>
</head>
<body>
   <script type="text/javascript" src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r83/three.js" />
    <script>
        // Our Javascript will go here.
    </script>
</body>
</html>

Création de la scène

Pour pouvoir afficher quoi que ce soit avec three.js, nous avons besoin de trois choses : une scène, une caméra et un moteur de rendu. Nous rendrons la scène avec la caméra.

var scene = new THREE.Scene();
var camera = new THREE.PerspectiveCamera( 75, window.innerWidth / window.innerHeight, 0.1, 1000 );

var renderer = new THREE.WebGLRenderer();
renderer.setSize( window.innerWidth, window.innerHeight );
document.body.appendChild( renderer.domElement );

Création de la sphère

  • Créer une géométrie pour la sphère
  • Créer un matériau phong
  • Créer un objet 3D
  • Ajoutez-le à la scène

[![Sphère ordinaire][1]][1]

var geometry = new THREE.SphereGeometry(1, 32, 32);
var material = new THREE.MeshPhongMaterial();
var earthmesh = new THREE.Mesh(geometry, material);

Ajouter une texture diffuse

La texture diffuse définit la couleur principale de la surface. Lorsque nous l’appliquons à une sphère, nous obtenons l’image suivante.

[![Texture diffuse][2]][2]

[![Résultat][3]][3]

material.map    = THREE.ImageUtils.loadTexture('images/earthmap1k.jpg');

Ajout d’une texture de carte en relief

  • Chacun de ses pixels agit comme une hauteur sur la surface.
  • Les montagnes apparaissent plus clairement grâce à leur ombre.
  • Il est possible de modifier l’impact de la carte sur l’éclairage avec Paramètre bumpScale.
  • Aucun sommet supplémentaire n’est créé ou nécessaire pour utiliser une carte en relief (contrairement à une carte de déplacement)

[![Texture bosse][4]][4]

[![Résultat][5]][5]

material.bumpMap   = THREE.ImageUtils.loadTexture('images/earthbump1k.jpg');     
material.bumpScale = 0.05;

Ajout d’une texture spéculaire

  • Modifie la “brillance” d’un objet avec une texture.

  • Chaque pixel détermine l’intensité de la spécularité.

  • Dans ce cas, seule la mer est spéculaire car l’eau réfléchit la lumière plus que la terre.

  • Vous pouvez contrôler la couleur spéculaire avec le paramètre spéculaire.

    [![Specular Texture][6]][6]

[![Résultat][7]][7]

material.specularMap = THREE.ImageUtils.loadTexture('images/earthspec1k.jpg')
material.specular  = new THREE.Color('grey')

Ajout d’une couche cloud

  • Nous créons canvasCloud avec un canevas et l’utilisons comme texture.
  • Nous faisons cela parce que jpg ne gère pas de canal alpha. (Cependant, une image PNG le fait)
  • Nous devons créer le code pour construire la texture basée sur les images suivantes.

[![Carte des nuages][8]][8]

[![Cloud Map Trans][9]][9]

[![entrez la description de l’image ici][10]][10]

var geometry   = new THREE.SphereGeometry(0.51, 32, 32)
var material  = new THREE.MeshPhongMaterial({
  map         : new THREE.Texture(canvasCloud),
  side        : THREE.DoubleSide,
  opacity     : 0.8,
  transparent : true,
  depthWrite  : false,
});
var cloudMesh = new THREE.Mesh(geometry, material)
earthMesh.add(cloudMesh)
  • Nous attachons le cloudMesh à earthMesh pour qu’ils se déplacent ensemble.
  • Nous désactivons depthWrite et définissons transparent: true pour dire à three.js que le cloudmesh est transparent.
  • Nous définissons les côtés sur “THREE.DoubleSide” pour que les deux côtés soient visibles.
  • Cela évite de créer des artefacts au bord de la terre.
  • Enfin, nous définissons opacity: 0.8 pour rendre les nuages ​​plus translucides

Ajout d’un mouvement de rotation

Dans votre boucle de rendu, vous augmentez simplement la rotation

Comme touche finale, nous allons animer la couche de nuages ​​afin de la rendre plus réaliste.

updateFcts.push(function(delta, now) {
    cloudMesh.rotation.y += 1 / 8 * delta;
    earthMesh.rotation.y += 1 / 16 * delta;
  })
