Jquery - Installation et configuration de Select2 #

Vous pouvez inclure/installer Select2 de l’une des deux manières

  1. En utilisant directement les CDN dans votre projet sous la section principale de votre projet.

lien href=“https://cdnjs.cloudflare.com/ajax/libs/select2/4.0.3/css/select2.min.css” rel=“stylesheet”/>

<script src="https://cdnjs.cloudflare.com/ajax/libs/select2/4.0.3/js/select2.min.js"/>
  1. Téléchargez le code sur votre ordinateur local et incluez-le dans votre projet. cela devrait ressembler à ceci [![entrez la description de l’image ici][1]][1]

Ensuite, incluez ces lignes dans la section d’en-tête de votre page html

  <link href="select2.min.css" rel="stylesheet" />
  <script src="select2.min.js"></script>

Remarque : Vous devez avoir Jquery inclus dans votre projet pour Select2 pour fonctionner correctement.

PIÈGE : si la version actuelle de jquery et Select2 sont en conflit ou Les fonctionnalités de Select2 ne fonctionnent pas. Ensuite, déplacez les variables select2 et implémentations dans un bloc séparé du corps du document $(document).ready(function () { //votre code select2 ….. });

Comment l’utiliser: Vous pouvez définir un <select> comme suit :

<select id="select2_example">
    <option>Test</option>
</select>

Approche1 :

      var _mSelect2 = $("#select_example").select2();

Approche2 :

  <script type="text/javascript">
      $('select').select2();
  </script>

Veuillez noter qu’en utilisant cette approche, tous les contrôles ‘select’ définis de la page héritera des fonctionnalités de Select2. Si vous souhaitez uniquement utiliser Select2 pour certains contrôles ignorent alors cette étape et écrivent votre propre code pour utiliser directement la fonction select2 et la fonction jquery/javascript séparément pour les autres contrôles de sélection. Cependant, vous pouvez utiliser la valeur de manipulé via Select2 pour Jquery ou vice-versa.

D’autres exemples de ‘The Basics’ peuvent être trouvés dans la documentation Select2 GIT [ici][2]

Le projet Select2 GIT est [ici][3]. Veuillez consulter le site Select2 Github pour connaître les caractéristiques détaillées de ce produit.

Pour effacer les éléments sélectionnés de la liste déroulante Select2. #

Afin d’effacer la sélection des valeurs sélectionnées à l’aide d’un menu déroulant “Select2”, nous pouvons utiliser la fonction empty ().

<select id="select2_example">
<option>Option1</option>
<option>Option2</option>
<option>Option3</option>
</select>

La liste déroulante peut être réinitialisée à l’aide de Jquery.

$("#select2_example").empty();

Il peut également être réinitialisé comme ci-dessous.

$("#select2_example").select2("val", "");
