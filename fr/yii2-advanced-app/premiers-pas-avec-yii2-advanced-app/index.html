Installation ou configuration #

Exigences d’installation

La configuration minimale requise par ce modèle de projet est que votre serveur Web prenne en charge PHP 5.4.0.

L’application Yii2-advanced peut être installée de deux manières. Elles sont

  1. Installation via Composer
  2. Installation à partir d’un fichier d’archive

1) Installation via Composer

Si vous n’avez pas encore installé Composer, vous pouvez le faire en suivant les instructions sur [getcomposer.org][1]. Sous Linux et Mac OS X, vous exécuterez les commandes suivantes :

Nous devons d’abord installer composer.

Étape 1 Installez Compositeur. Exécutez la commande ci-dessous dans votre terminal

curl -sS https://getcomposer.org/installer | php

Étape 2 Maintenant, changez de répertoire

sudo mv composer.phar /usr/local/bin/composer

Étape-3 vérifier que le compositeur fonctionne

composer

Si Composer a été installé avec succès.

Installez ensuite l’application à l’aide des commandes suivantes :

composer global require "fxp/composer-asset-plugin:~1.1.1"
composer create-project --prefer-dist yiisoft/yii2-app-advanced yii-application

La première commande installe le [composer asset plugin][2] qui permet de gérer les dépendances des packages bower et npm via Composer. Vous n’avez besoin d’exécuter cette commande qu’une fois pour toutes. La deuxième commande installe l’application avancée dans un répertoire nommé “yii-application”. Vous pouvez choisir un nom de répertoire différent si vous le souhaitez.

2) Installation à partir d’un fichier d’archive

Extrayez le fichier d’archive téléchargé depuis [yiiframework.com][3] dans un répertoire nommé advanced qui se trouve directement sous la racine Web.

Modifiez le fichier config/web.php en saisissant une clé secrète pour l’élément de configuration cookieValidationKey (cela se fait automatiquement si vous installez Yii à l’aide de Composer) :

// !!! insert a secret key in the following (if it is empty) - this is required by cookie validation
'cookieValidationKey' => 'enter your secret key here',

Suivez ensuite les instructions données dans la sous-section suivante.

Préparation de la candidature

Après avoir installé l’application, vous devez effectuer les étapes suivantes pour initialiser l’application installée. Vous n’avez qu’à les faire une fois pour toutes.

  1. Ouvrez un terminal de console, exécutez la commande init et sélectionnez dev comme environnement.

    /path/to/php-bin/php /path/to/yii-application/init

Si vous l’automatisez avec un script, vous pouvez exécuter init en mode non interactif.

/path/to/php-bin/php /path/to/yii-application/init --env=Production --overwrite=All

  1. Créez une nouvelle base de données et ajustez la configuration de components['db'] dans common/config/main-local.php en conséquence.

  2. Ouvrez un terminal console, appliquez les migrations avec la commande /path/to/php-bin/php /path/to/yii-application/yii migrate.

[1] : http://getcomposer.org [2] : https://github.com/fxpio/composer-asset-plugin [3] : http://www.yiiframework.com/download/
