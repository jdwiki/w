Installation de Gloss #

Le brillant s’installe facilement à l’aide de l’outil Cabal. Après avoir installé Cabal, on peut exécuter cabal install gloss pour installer Gloss.

Alternativement, le package peut être construit à partir de la source, en téléchargeant la source depuis [Hackage][1] ou [GitHub][2], et en procédant comme suit :

  1. Entrez dans le répertoire gloss/gloss-rendering/ et faites cabal install
  2. Entrez dans le répertoire gloss/gloss/ et faites à nouveau cabal install

[1] : https://hackage.haskell.org/package/gloss [2] : https://github.com/benl23x5/gloss

Afficher quelque chose à l’écran #

Dans Gloss, on peut utiliser la fonction display pour créer des graphiques statiques très simples.

Pour l’utiliser, il faut d’abord “importer Graphics.Gloss”. Ensuite, dans le code, il devrait y avoir ce qui suit :

main :: IO ()
main = display window background drawing

window est de type Display qui peut être construit de deux manières :

-- Defines window as an actual window with a given name and size
window = InWindow name (width, height) (0,0)

-- Defines window as a fullscreen window
window = FullScreen

Ici, le dernier argument (0,0) dans InWindow marque l’emplacement du coin supérieur gauche.

Pour les versions antérieures à 1.11 : Dans les anciennes versions de Gloss, FullScreen prend un autre argument qui est censé être la taille du cadre qui est dessiné sur lequel à son tour est étiré en plein écran, par exemple : FullScreen (1024 768)

background est de type Color. Il définit la couleur de fond, donc c’est aussi simple que :

background = white

Ensuite, nous arrivons au dessin lui-même. Les dessins peuvent être très complexes. Comment les spécifier sera couvert ailleurs ([on peut s’y référer pour le moment][1]), mais cela peut être aussi simple que le cercle suivant avec un rayon de 80 :

drawing = Circle 80

Exemple récapitulatif

Comme plus ou moins indiqué dans la documentation sur Hackage, obtenir quelque chose à l’écran est aussi simple que :

import Graphics.Gloss

main :: IO ()
main = display window background drawing
    where
      window = InWindow "Nice Window" (200, 200) (0, 0) 
      background = white 
      drawing = Circle 80
