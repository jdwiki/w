Préparer l’ensemble de données d’image pour la tâche de classification d’image #

Caffe dispose d’une couche d’entrée intégrée adaptée aux tâches de classification d’images (c’est-à-dire une seule étiquette entière par image d’entrée). Cette couche “Données” d’entrée est construite sur une structure de données [tag:lmdb] ou [tag:leveldb]. Pour utiliser la couche "Data", il faut construire la structure de données avec toutes les données d’apprentissage.

Un guide rapide du `convert_imageset` de Caffe

Construire

La première chose que vous devez faire est de construire caffe et les outils de caffe (`convert_imageset` est l'un de ces outils). Après avoir installé caffe et l'avoir créé, assurez-vous que vous avez également exécuté "make tools". Vérifiez qu'un fichier binaire `convert_imageset` est créé dans `$CAFFE_ROOT/build/tools`.

Préparez vos données

*Images :* mettre toutes les images dans un dossier (je l'appellerai ici `/path/to/jpegs/`). *Labels :* créez un fichier texte (par exemple, `/path/to/labels/train.txt`) avec une ligne par image d'entrée
