Introduction #

Qu’est-ce que Kibana :

Kibana est utilisé pour faire des visualisations et créer des tableaux de bord pour les index présentés dans elasticsearch. Fondamentalement, il s’agit d’un plug-in open source pour elasticsearch.

[![Voir Page][1]

Il y a six onglets :

  1. Découvrez :

    You can explore your data from Discover tab

  2. Visualisation :

    Creating visualization with different charts such as bar chart,line chart,pie chart etc.

  3. Tableau de bord :

    It is used for creating dashboard with created visualization. It contains your visualization to represent your process.

  4. Timelion :

    It is timeseries of kibana

  5. Outils de développement

    It is like sense plugin used for elasticsearch. It contains request and response field

  6. Gestion

    This is used for managing your kibana. It contains Settings for the kibana plugin

  • Pour connaître le passé de kibana, utilisez cette page Web [kibana version antérieure] [3]

Installez Kibana en utilisant apt-get

Pour installer kibana, vous devez savoir quelle version de recherche élastique est installée sur votre système. Cela peut être vérifié à l’aide de la commande suivante :

curl "http://localhost:9200"

Dans la sortie, vous devez vérifier le nombre, c’est-à-dire la version de recherche élastique.

Trouvez maintenant la version appropriée pour cela. Voici la commande pour installer kibana 5.4.0 pour elastic search 5.4.0.

wget -qO - https://artifacts.elastic.co/GPG-KEY-elasticsearch | sudo apt-key add -
sudo apt-get install apt-transport-https
echo "deb https://artifacts.elastic.co/packages/5.x/apt stable main" | sudo tee -a /etc/apt/sources.list.d/elastic-5.x.list
sudo apt-get update && sudo apt-get install kibana

Exécuter Kibana en tant que service

Après l’installation, Kiabana ne démarre pas automatiquement. Comment démarrer et arrêter dépend de si votre système prend en charge SysV init ou systemd. vérifiez cela en utilisant la commande suivante.

ps -p 1

Pour l’initialisation Sysv

exécutez la commande suivante :

sudo update-rc.d kibana defaults 95 10

Pour démarrer et arrêter le serveur, utilisez la commande suivante.

sudo -i service kibana start
sudo -i service kibana stop

Pour systemd :

exécutez les commandes suivantes :

sudo /bin/systemctl daemon-reload
sudo /bin/systemctl enable kibana.service

Pour démarrer et arrêter le serveur, utilisez la commande suivante.

sudo systemctl start kibana.service
sudo systemctl stop kibana.service

Installer #

Pour la configuration de kibana, ouvrez config/kibana.yml et indiquez votre adresse elasticsearch.

By default it is http://localhost:9200

Avant de démarrer elasticsearch, vérifiez si elasticsearch est en cours d’exécution. Parce qu’il s’appuie sur Elasticsearch.

Pour les fenêtres :

exécutez le fichier de correctif bin/kibana.

Pour Mac :

exécuter ./caractère

Pour Linux :

lancez bin/kibana.
