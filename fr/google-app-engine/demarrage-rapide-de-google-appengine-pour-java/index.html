Avant que tu commences #

Avant d’exécuter cet exemple, vous devez :

Téléchargez et installez le kit de développement Java SE (JDK) :

[Télécharger JDK][1]

Téléchargez [Apache Maven][2] version 3.3.9 ou ultérieure :

Installez et configurez Maven pour votre environnement de développement local.

[1] : http://www.oracle.com/technetwork/java/javase/downloads/index.html [2] : https://maven.apache.org/download.cgi

Téléchargez l’application Hello World #

Nous avons créé une application Hello World simple pour Java afin que vous puissiez rapidement vous faire une idée du déploiement d’une application sur Google Cloud Platform. Suivez ces étapes pour télécharger Hello World sur votre ordinateur local.

Clonez le dépôt de l’exemple d’application Hello World sur votre ordinateur local :

git clone https://github.com/GoogleCloudPlatform/java-docs-samples.git

Accédez au répertoire contenant l’exemple de code :

cd java-docs-samples/appengine/helloworld

Dans les fichiers helloworld résultants, vous trouverez le répertoire src pour un package appelé com.example.appengine.helloworld qui implémente un simple HTTPServlet.

Vous pouvez également [télécharger l’exemple][1] sous forme de fichier zip et l’extraire.

[1] : https://github.com/GoogleCloudPlatform/java-docs-samples/archive/master.zip

Testez l’application #

Testez l’application à l’aide du serveur Web de développement, qui est inclus avec le SDK App Engine.

  1. Exécutez la commande Maven suivante depuis votre répertoire “helloworld” pour compiler votre application et démarrer le serveur Web de développement :

    mvn appengine:devserver

Le serveur Web écoute maintenant les requêtes sur le port 8080.

  1. Visitez http://localhost:8080/ dans votre navigateur Web pour voir l’application en action.

Pour plus d’informations sur l’exécution du serveur Web de développement, consultez la [Référence du serveur de développement Java][1].

[1] : https://cloud.google.com/appengine/docs/java/tools/devserver

Faire un changement #

Vous pouvez laisser le serveur Web en cours d’exécution pendant que vous développez votre application. Lorsque vous apportez une modification, utilisez la commande mvn clean package pour créer et mettre à jour votre application.

  1. Essayez-le maintenant : laissez le serveur Web en cours d’exécution, puis modifiez HelloServlet.java to change Hello, world to something else.
  2. Exécutez mvn clean package, puis rechargez http://localhost:8080/ pour voir le results.

Déployez votre application #

Pour déployer votre application sur App Engine, vous devrez enregistrer un projet pour créer votre ID de projet, qui déterminera l’URL de l’application.

  1. Dans la console Cloud Platform, accédez à la page Projets et sélectionnez ou créez un nouveau projet.

  2. Notez l’ID de projet que vous avez créé ci-dessus et saisissez-le dans src/main/webapp/WEB-INF/appengine-web.xml. Vous pouvez également définir la version de l’application dans ce fichier.

  3. Chargez votre application sur Google App Engine en appelant la commande suivante.

    mvn appengine:update

  4. Votre application est maintenant déployée et prête à diffuser du trafic sur http://<YOUR_PROJECT_ID>.appspot.com/.
