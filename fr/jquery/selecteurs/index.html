Un sélecteur jQuery sélectionne ou trouve un élément DOM (document object model) dans un document HTML. Il est utilisé pour sélectionner des éléments HTML en fonction de l’identifiant, du nom, des types, des attributs, de la classe, etc. Il est basé sur les sélecteurs CSS existants.

Syntaxe #

  • Balise : Pas de marqueur, utilisez directement la balise
  • Identifiant : #id
  • Classe : .className
  • Attribut : [attributeName]
  • Attribut avec valeur : [attributeName ='value']
  • L’attribut commence par la valeur ^= : [attributeName ^= 'value']
  • L’attribut se termine par la valeur $= : [attributeName $='value']
  • L’attribut contient la valeur *= : [attributeName *= 'value']
  • Pseudo-sélecteur : :pseudo-selector
  • Tout descendant : espace entre les sélecteurs
  • Enfants directs : > entre les sélecteurs
  • Frère adjacent suivant le premier : +
  • Frère non adjacent suivant le premier : ~
  • Ou : , ​​(virgule) entre le sélecteur

Lors de l’écriture de selectors pour class ou id ou attribute qui contient des caractères spéciaux comme

! " # $ % & ' ( ) * + , . / : ; < = > ? @ [ \ ] ^ { | } ~

les caractères doivent être échappés à l’aide de deux barres obliques inverses \\ .

par exemple.

<span id="temp.foobar"><span>

les sélecteurs seront encadrés comme,

$('#temp\\.foobar')

Types de sélecteurs #

Dans jQuery, vous pouvez sélectionner des éléments dans une page en utilisant de nombreuses propriétés de l’élément, notamment :

  • Taper
  • Classer
  • IDENTIFIANT
  • Possession d’attribut
  • Valeur d’attribut
  • Sélecteur indexé
  • [Pseudo-état][1]

Si vous connaissez les [sélecteurs CSS][2], vous remarquerez que les sélecteurs dans jQuery sont les mêmes (à quelques exceptions près).

Prenez par exemple le code HTML suivant :

<a href="index.html"></a>                  <!-- 1 -->
<a id="second-link"></a>                   <!-- 2 -->
<a class="example"></a>                    <!-- 3 -->
<a class="example" href="about.html"></a>  <!-- 4 -->
<span class="example"></span>              <!-- 5 -->

Sélection par type :

Le sélecteur jQuery suivant sélectionnera tous les éléments <a>, y compris 1, 2, 3 et 4.

$("a")

Sélection par classe

Le sélecteur jQuery suivant sélectionnera tous les éléments de la classe “example” (y compris les éléments non-a), qui sont 3, 4 et 5.

$(".example")

Sélection par ID

Le sélecteur jQuery suivant sélectionnera l’élément avec l’ID donné, qui est 2.

$("#second-link")

Sélection par possession d’attribut

Le sélecteur jQuery suivant sélectionnera tous les éléments avec un attribut href défini, y compris 1 et 4.

$("[href]")

Sélection par valeur d’attribut

Le sélecteur jQuery suivant sélectionnera tous les éléments où l’attribut href existe avec une valeur de index.html, qui est juste 1.

$("[href='index.html']")

Sélection par position indexée (Sélecteur indexé)

Le sélecteur jQuery suivant ne sélectionnera que 1, le second <a> c’est-à-dire. le second-link car l’index fourni est 1 comme eq(1) (Notez que l’index commence à 0 donc le second a été sélectionné ici !).

$("a:eq(1)")

Sélection avec exclusion indexée

Pour exclure un élément en utilisant son index :not(:eq())

Ce qui suit sélectionne les éléments <a>, sauf qu’avec la classe example, qui est 1

$("a").not(":eq(0)")

Sélection avec exclusion

Pour exclure un élément d’une sélection, utilisez :not()

Ce qui suit sélectionne les éléments <a>, sauf ceux avec la classe example, qui sont 1 et 2.

$("a:not(.example)")

Sélection par pseudo-état

Vous pouvez également sélectionner dans jQuery en utilisant des pseudo-états, y compris :first-child, :last-child, :first-of-type, :last-of-type, etc.

Le sélecteur jQuery suivant ne sélectionnera que le premier élément <a> : le numéro 1.

$("a:first-of-type")

Combinaison de sélecteurs jQuery

Vous pouvez également augmenter votre spécificité en combinant plusieurs sélecteurs jQuery ; vous pouvez combiner n’importe quel nombre d’entre eux ou les combiner tous. Vous pouvez également sélectionner plusieurs classes, attributs et états en même temps.

$("a.class1.class2.class3#someID[attr1][attr2='something'][attr3='something']:first-of-type:first-child")

Cela sélectionnerait un élément <a> qui :

  • A les classes suivantes : class1, class2 et class3
  • A l’identifiant suivant : someID
  • Possède l’attribut suivant : attr1
  • Possède les attributs et valeurs suivants : attr2 avec la valeur something, attr3 avec la valeur something
  • A les états suivants : first-child et first-of-type

Vous pouvez également séparer différents sélecteurs par une virgule :

$("a, .class1, #someID")

Cela sélectionnerait :

  • Tous les éléments <a>
  • Tous les éléments qui ont la classe class1
  • Un élément avec l’identifiant #someID

Sélection enfant et frère

Les sélecteurs jQuery sont généralement conformes aux mêmes conventions que CSS, ce qui vous permet de sélectionner les enfants et les frères et sœurs de la même manière.

  • Pour sélectionner un enfant non direct, utilisez un espace
  • Pour sélectionner un enfant direct, utilisez un >
  • Pour sélectionner un frère adjacent à la suite du premier, utilisez un +
  • Pour sélectionner un frère non adjacent à la suite du premier, utilisez un ~

Sélection générique

Il peut y avoir des cas où nous voulons sélectionner tous les éléments mais il n’y a pas de propriété commune à sélectionner (classe, attribut, etc.). Dans ce cas, nous pouvons utiliser le sélecteur * qui sélectionne simplement tous les éléments :

$('#wrapper *')    // Select all elements inside #wrapper element

[1] : https://developer.mozilla.org/en-US/docs/Web/CSS/Pseudo-classes [2] : https://www.wikiod.com/fr/css/selecteurs

Aperçu #

Les éléments peuvent être sélectionnés par jQuery en utilisant [jQuery Selectors][1]. La fonction renvoie soit un élément, soit une liste d’éléments.

Sélecteurs de base #

$("*")                       // All elements
$("div")                     // All <div> elements
$(".blue")                   // All elements with class=blue
$(".blue.red")               // All elements with class=blue AND class=red
$(".blue,.red")              // All elements with class=blue OR class=red
$("#headline")               // The (first) element with id=headline
$("[href]")                  // All elements with an href attribute
$("[href='example.com']")    // All elements with href=example.com

Opérateurs relationnels #

$("div span")          // All <span>s that are descendants of a <div>
$("div > span")        // All <span>s that are a direct child of a <div>
$("a ~ span")          // All <span>s that are siblings following an <a>
$("a + span")          // All <span>s that are immediately after an <a>

[1] : https://api.jquery.com/category/selectors/

Sélecteurs de mise en cache #

Chaque fois que vous utilisez un sélecteur dans jQuery, le DOM recherche les éléments qui correspondent à votre requête. Faire cela trop souvent ou à plusieurs reprises réduira les performances. Si vous faites référence à un sélecteur spécifique plus d’une fois, vous devez l’ajouter au cache en l’affectant à une variable :

var nav = $('#navigation');
nav.show();

Cela remplacerait :

$('#navigation').show();

La mise en cache de ce sélecteur peut s’avérer utile si votre site Web doit souvent afficher/masquer cet élément. S’il y a plusieurs éléments avec le même sélecteur, la variable deviendra un tableau de ces éléments :

<div class="parent">
    <div class="child">Child 1</div>
    <div class="child">Child 2</div>
</div>

<script>
    var children = $('.child');
    var firstChildText = children[0].text();
    console.log(firstChildText); 
    
    // output: "Child 1"
</script>

REMARQUE : L’élément doit exister dans le DOM au moment de son affectation à une variable. S’il n’y a pas d’élément dans le DOM avec une classe appelée “enfant”, vous stockerez un tableau vide dans cette variable.

<div class="parent"></div>

<script>
    var parent   = $('.parent');
    var children = $('.child');
    console.log(children);

    // output: []

    parent.append('<div class="child">Child 1</div>');
    children = $('.child');
    console.log(children[0].text());

    // output: "Child 1"
</script>

N’oubliez pas de réaffecter le sélecteur à la variable après avoir ajouté/supprimé des éléments dans le DOM avec ce sélecteur.

Remarque : lors de la mise en cache des sélecteurs, de nombreux développeurs commenceront le nom de la variable par un $ pour indiquer que la variable est un objet jQuery comme ceci :

var $nav = $('#navigation');
$nav.show();

Combinaison de sélecteurs #

Envisagez de suivre la structure DOM

<ul class="parentUl">
    <li> Level 1
        <ul class="childUl">
            <li>Level 1-1 <span> Item - 1 </span></li>
            <li>Level 1-1 <span> Item - 2 </span></li>
        </ul>
    </li>
    <li> Level 2
        <ul class="childUl">
            <li>Level 2-1 <span> Item - 1 </span></li>
            <li>Level 2-1 <span> Item - 1 </span></li>
        </ul>
    </li>
</ul>

Sélecteurs descendant et enfant #

Étant donné un parent <ul> - parentUl trouve ses descendants (<li>),

  1. Simple $('enfant parent')

>> $('ul.parentUl li')

Cela obtient tous les descendants correspondants de l’ancêtre spécifié tous les niveaux inférieurs.

  1. > - $('parent > enfant')

>> $('ul.parentUl > li')

Cela trouve tous les enfants correspondants (seulement 1er niveau vers le bas).

  1. Sélecteur contextuel - $('child','parent')

>> $('li','ul.parentUl')

Cela fonctionne comme 1. ci-dessus.

  1. trouver() - $('parent').trouver('enfant')

>> $('ul.parentUl').find('li')

Cela fonctionne comme 1. ci-dessus.

  1. enfants() - $('parent').find('enfant')

>> $('ul.parentUl').children('li')

Cela fonctionne comme 2. ci-dessus.

Autres combinateurs #

Sélecteur de groupe : “,” #

Sélectionnez tous les éléments <ul> ET tous les éléments <li> ET tous les éléments <span> :

$('ul, li, span')

Sélecteur multiple : "" (pas de caractère) #

Sélectionnez tous les éléments <ul> avec la classe parentUl :

$('ul.parentUl')

Sélecteur de frères et sœurs adjacents : “+” #

Sélectionnez tous les éléments <li> qui sont placés immédiatement après un autre élément <li> :

$('li + li')

Sélecteur général des frères et sœurs : “~” #

Sélectionnez tous les éléments <li> qui sont frères d’autres éléments <li> :

$('li ~ li')

Éléments DOM comme sélecteurs #

jQuery accepte une grande variété de paramètres, et l’un d’eux est un élément DOM réel. Passer un élément DOM à jQuery entraînera la structure sous-jacente de type tableau de [l’objet jQuery] (https://www.wikiod.com/fr/jquery/premiers-pas-avec-jquery#L’objet jQuery ) pour contenir cet élément.

jQuery détectera que l’argument est un élément DOM en inspectant son nodeType.

L’utilisation la plus courante d’un élément DOM est dans les rappels, où l’élément actuel est passé au constructeur jQuery afin d’accéder à l’API jQuery.

Comme dans le rappel each (note : each est une fonction d’itération).

$(".elements").each(function(){
    //the current element is bound to `this` internally by jQuery when using each
    var currentElement = this;

    //at this point, currentElement (or this) has access to the Native API
    
    //construct a jQuery object with the currentElement(this)
    var $currentElement = $(this);

    //now $currentElement has access to the jQuery API
});

Chaînes HTML comme sélecteurs #

jQuery accepte une grande variété de paramètres en tant que “sélecteurs”, et l’un d’eux est une chaîne HTML. Passer une chaîne HTML à jQuery entraînera la structure sous-jacente de type tableau de [l’objet jQuery] (https://www.wikiod.com/fr/jquery/premiers-pas-avec-jquery#L’objet jQuery ) pour contenir le HTML construit résultant.

jQuery utilise regex pour déterminer si la chaîne transmise au constructeur est une chaîne HTML, et aussi qu’elle doit commencer par <. Cette expression régulière est définie comme rquickExpr = /^(?:\s*(<[\w\W]+>)[^>]*|#([\w-]*))$/ (explication à regex101.com).

L’utilisation la plus courante d’une chaîne HTML en tant que sélecteur est lorsque des ensembles d’éléments DOM doivent être créés uniquement dans le code, ce qui est souvent utilisé par les bibliothèques pour des éléments tels que les popouts modaux.

Par exemple, une fonction qui a renvoyé une balise d’ancrage enveloppée dans un div comme modèle

function template(href,text){
    return $("<div><a href='" + href + "'>" + text + "</a></div>");
}

Renverrait un objet jQuery contenant

<div>
    <a href="google.com">Google</a>
</div>

s’il est appelé en tant que template("google.com","Google").
