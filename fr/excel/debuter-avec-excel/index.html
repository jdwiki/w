Exemples de certaines formules de base disponibles dans Excel #

Excel intègre de nombreuses formules. Voici quelques exemples de formules de base qu’il peut être utile de connaître lors de la mise en route d’Excel :

Remarque importante : Le nom et la syntaxe de ces formules varient selon la langue de votre installation Excel ! Par exemple cette fonction ici :

En anglais : =left("Hello World",5)

Idem en allemand : =links("Bonjour le monde";5)

Non seulement le nom est différent, mais aussi certaines parties mineures de la syntaxe ! Par exemple, dans la version allemande, ; est utilisé à la place de , pour séparer les paramètres.

Toutes les formules peuvent être saisies dans n’importe quelle cellule en tapant d’abord “=” puis le nom de la formule. Vous pouvez saisir les formules directement dans la cellule ou vous pouvez sélectionner la cellule et saisir la formule dans la barre de formule. Pour afficher toutes les formules sur une feuille, vous pouvez sélectionner le bouton “Afficher les formules” dans le groupe de boutons “Audit des formules” de l’onglet Formules :

[![Afficher les formules][1]][1]

De nombreuses formules utilisent une plage de cellules. Pour donner une plage, vous référencez la première cellule dans le coin supérieur gauche tel que A1 et le coin inférieur droit tel que B3 et placez deux points “:” entre eux comme ceci A1: B3. Cela vous donnera une plage de 6 cellules en deux colonnes et trois lignes :

[![Plage de cellules][2]][2]

SOMME()

Dans les cellules D1 - D3 se trouvent les valeurs 2,3 et 6 Dans la cellule D4 se trouve la formule suivante : =SUM(D1:D3), le résultat est 11 qui s’affiche dans la cellule.

Sum Additionne tous les nombres dans une plage de cellules.

Si vous avez sélectionné cette cellule, la formule s’affichera dans la barre de formule au-dessus de la grille de cellules :

[![Formule I AM][3]][3]

COUNT()/COUNTA()/COUNTIF()/COUNTBLANK()

Dans les cellules E1 - E10 se trouvent les valeurs 2,3,5,6, blanc,8,9, blanc,11 et 12

Dans la cellule F2 se trouve la formule : =count(E1:E10), le résultat est 8 qui s’affiche dans la cellule.

Dans la cellule F3 se trouve la formule : =counta(E1:E10), le résultat est 8 qui s’affiche dans la cellule.

Dans la cellule F4 se trouve la formule : =countif(E1:E10,">5"), le résultat est 5 qui s’affiche dans la cellule.

Dans la cellule F5 se trouve la formule : =countblank(E1:E10), le résultat est 2 qui s’affiche dans la cellule.

Count() Compte le nombre de cellules dans une plage contenant des nombres.

Counta() Compte le nombre de cellules dans une plage qui ne sont pas vides.

CountIF() Compte le nombre de cellules dans une plage qui répondent à une condition donnée.

CountBlank() Compte le nombre de cellules vides dans une plage de cellules spécifiée.

[![Compter][4]][4]

Il existe également de nombreuses formules de chaînes.

GAUCHE()/DROITE()/MID()

Dans la cellule D1 se trouve “Hello World!”.

Dans la cellule E1 se trouve la formule : =left(D1,5), le résultat est : “Bonjour”

Dans la cellule F1 se trouve la formule : =right(D1,6), le résultat est : “Monde !”

Dans la cellule G1 se trouve la formule : =mid(D1,7,5), le résultat est : “Monde”

left Renvoie le nombre spécifié de caractères à partir du début d’une chaîne de texte. =GAUCHE(Chaîne,Nombre de caractères)

right Renvoie le nombre spécifié de caractères à partir de la fin d’une chaîne de texte. =DROITE(chaîne, nombre de caractères)

mid Renvoie les caractères à partir du milieu d’une chaîne de texte, en fonction de la position de départ et de la longueur. =MID(chaîne, position de départ, nombre de caractères)

Ces trois formules comptent les caractères de la chaîne en commençant par la position 1 étant le premier caractère.
