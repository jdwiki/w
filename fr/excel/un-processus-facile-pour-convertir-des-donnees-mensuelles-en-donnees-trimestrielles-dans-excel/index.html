Un processus facile pour convertir des données mensuelles en données trimestrielles dans Excel

  • Un moyen simple d’effectuer une tâche de routine parfois fastidieuse :

Sur cette page

  • Un moyen simple d’effectuer une tâche de routine parfois fastidieuse :

Un désir fréquent est de convertir les données mensuelles en format de données trimestrielles. Une méthode simple consiste à 1) additionner les mois appropriés en sommes trimestrielles, puis 2) identifier ces mois comme des trimestres, et enfin 3) filtrer les données trimestrielles de vos données mensuelles. Voici une méthode relativement simple et rapide illustrée à travers l’exemple suivant :

Un moyen simple d’effectuer une tâche de routine parfois fastidieuse : #

Si vous avez vos données avec chaque mois de données disposées en lignes comme suit :

[![Exemple de données][1]][1]

commencez par créer des sommes trimestrielles dans les colonnes adjacentes, D & E dans notre exemple.

Commencez par la troisième ligne de la nouvelle colonne ou le premier trimestre que vous souhaitez créer. Dans cet exemple, nous utiliserons le 31 mars (2000-03-31). Utilisez la fonction de somme sur les deux mois précédents et le quart de mois, pour une utilisation en mars :

=SOMME(B2:B4)

Utilisez ensuite une fonction similaire pour le total dans la colonne suivante E. Elle aura la formule suivante :

=SOMME(C2:C4)

Ne vous inquiétez pas que les sommes soient créées pour des mois qui ne sont pas des trimestres. Ceci est fait pour vous faciliter la copie. Vous n’aurez de toute façon pas besoin de ces valeurs dans les prochaines étapes.

Enfin, nous devons identifier les quartiers sans beaucoup de travail. Si votre colonne Date n’est pas dans un format de date Excel, tel que du texte, vous souhaiterez peut-être la convertir. Dans une nouvelle colonne, F dans cet exemple, commencez cette formule dans la première ligne de données. Nous allons marquer les lignes en utilisant la formule suivante :

=SI(MOD(MOIS(A2),3)=0, “Trimestre”, “Mois”)

Copiez maintenant les trois formules dans les nouvelles colonnes D, E et F dans les cellules restantes de votre tableau. Ça devrait ressembler à ça:

[![Échantillon de données avec des formules ajoutées à trois colonnes][2]][2]

Enfin, sélectionnez ce tableau, choisissez Format As Table, choisissez un format simple et Excel produira un résultat comme celui-ci :

[![entrez la description de l’image ici][3]][3]

La dernière étape consiste à sélectionner le bouton de filtre sur la nouvelle colonne “Période” dans F et à sélectionner uniquement les valeurs “Trimestres” sur lesquelles filtrer :

[![Choix du menu de filtrage sur la colonne Période][4]][4]

Votre tableau devrait maintenant ressembler à ceci :

[![Données trimestrielles filtrées][5]][5]

[1] : https://i.stack.imgur.com/mqYp5.png [2] : https://i.stack.imgur.com/Q0Ub2.png [3] : https://i.stack.imgur.com/OPHlK.png [4] : https://i.stack.imgur.com/UZSrF.png [5] : https://i.stack.imgur.com/QkrO7.png

À ce stade, je suggérerais de sélectionner uniquement les colonnes dont vous avez besoin dans votre tableau final, puis d’utiliser Coller->Spécial->Valeurs pour déplacer vos données.
