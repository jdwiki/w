Création d’une connexion à d’autres fichiers Excel à interroger #

Il existe de nombreuses façons d’y parvenir, que d’autres ont déjà suggérées. En suivant la “récupération de données Excel via SQL”, voici quelques conseils.

Excel dispose de “l’assistant de connexion de données” qui vous permet d’importer ou de créer un lien à partir d’une autre source de données ou même dans le même fichier Excel. Dans le cadre de Microsoft Office (et des systèmes d’exploitation), il existe deux fournisseurs d’intérêt : l’ancien “Microsoft.Jet.OLEDB” et le dernier “Microsoft.ACE.OLEDB”. Recherchez-les lors de la configuration d’une connexion (comme avec l’assistant de connexion de données). Une fois connectée à un classeur Excel, une feuille de calcul ou une plage est l’équivalent d’un tableau ou d’une vue. Le nom de table d’une feuille de calcul est le nom de la feuille de calcul suivi du signe dollar ("$") et entouré de crochets ("[" et “]”); d’une gamme, c’est simplement le nom de la gamme. Pour spécifier une plage de cellules sans nom comme source d’enregistrement, ajoutez la notation de ligne/colonne Excel standard à la fin du nom de la feuille entre crochets.

Nommer la plage de données que vous souhaitez interroger à l’aide du gestionnaire de noms est très utile, car le nommer “Base de données” peut traiter les données d’une feuille de calcul comme une table de base de données, vous permettant d’exécuter des instructions SQL sur vos données pour un accès facile. Il convient également de noter que la plage nommée ne doit pas contenir de données vides ou manquantes, car cela entraînerait la rupture du SQL.

Le SQL natif sera (plus ou moins) le SQL de Microsoft Access. (Dans le passé, il s’appelait JET SQL, mais Access SQL a évolué et je pense que JET est une ancienne technologie obsolète.) Exemple, lecture d’une feuille de calcul : SELECT * FROM [Sheet1$] Exemple, lecture d’une plage : SELECT * FROM MyRange Exemple, lecture d’une plage de cellules sans nom : SELECT * FROM [Sheet1$A1:B10] Il existe de nombreux livres et sites Web disponibles pour vous aider à comprendre les détails.
