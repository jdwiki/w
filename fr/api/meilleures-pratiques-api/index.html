Codes d’état #

1xx : Informations #

Message Descriptif Description longue
100 Continuer Le serveur a reçu les en-têtes de requête et le client doit continuer à envoyer le corps de la requête
101 Changement de protocole Le demandeur a demandé au serveur de changer de protocole
103 Point de contrôle Utilisé dans la proposition de demandes de reprise pour reprendre les demandes PUT ou POST abandonnées

## 2xx : réussi

Message Descriptif Description longue
200 OK La requête est OK (il s’agit de la réponse standard pour les requêtes HTTP réussies)
201 Créé La demande a été satisfaite et une nouvelle ressource est créée
202 Accepté La demande a été acceptée pour traitement, mais le traitement n’est pas terminé
203 Informations ne faisant pas autorité La demande a été traitée avec succès, mais renvoie des informations qui peuvent provenir d’une autre source
204 No Content La demande a été traitée avec succès, mais ne renvoie aucun contenu
205 Réinitialiser le contenu La demande a été traitée avec succès, mais ne renvoie aucun contenu et nécessite que le demandeur réinitialise la vue du document
206 Contenu partiel Le serveur ne fournit qu’une partie de la ressource en raison d’un en-tête de plage envoyé par le client

3xx : redirection #

Message Descriptif Description longue
300 Choix multiples Une liste de liens. L’utilisateur peut sélectionner un lien et accéder à cet emplacement. Maximum cinq adresses
301 Déplacé définitivement La page demandée a été déplacée vers une nouvelle URL
302 Trouvé La page demandée a temporairement été déplacée vers une nouvelle URL
303 Voir autre La page demandée peut être trouvée sous une URL différente
304 Non modifié Indique que la page demandée n’a pas été modifiée depuis la dernière requête
306 Changer de proxy N’est plus utilisé
307 Redirection temporaire La page demandée a été déplacée temporairement vers une nouvelle URL
308 Reprendre incomplet Utilisé dans la proposition de demandes de reprise pour reprendre les demandes PUT ou POST abandonnées

4xx : erreur client #

Message Descriptif Description longue
400 Bad Request La requête ne peut pas être satisfaite en raison d’une mauvaise syntaxe
401 Non autorisé La demande était une demande légale, mais le serveur refuse d’y répondre. À utiliser lorsque l’authentification est possible mais a échoué ou n’a pas encore été fournie
402 Paiement requis Réservé pour une utilisation future
403 Interdit La demande était une demande légale, mais le serveur refuse d’y répondre
404 Introuvable La page demandée est introuvable mais pourrait être à nouveau disponible à l’avenir
405 Méthode non autorisée Une requête a été effectuée sur une page à l’aide d’une méthode de requête non prise en charge par cette page
406 Non acceptable Le serveur ne peut générer qu’une réponse qui n’est pas acceptée par le client
407 Authentification proxy requise Le client doit d’abord s’authentifier auprès du proxy
408 Délai d’expiration de la demande Le serveur a expiré en attendant la demande
409 Conflit La requête n’a pas pu être traitée en raison d’un conflit dans la requête
410 Gone La page demandée n’est plus disponible
411 Longueur requise Le “Content-Length” n’est pas défini. Le serveur n’acceptera pas la demande sans elle
412 Échec de la précondition La précondition donnée dans la requête a été évaluée comme fausse par le serveur
413 Entité de demande trop grande Le serveur n’acceptera pas la demande, car l’entité de demande est trop grande
414 Request-URI Too Long Le serveur n’acceptera pas la requête car l’URL est trop longue. Se produit lorsque vous convertissez une requête POST en une requête GET avec une longue information de requête
415 Type de média non pris en charge Le serveur n’acceptera pas la demande, car le type de média n’est pas pris en charge
416 Plage demandée non satisfaisante Le client a demandé une partie du fichier, mais le serveur ne peut pas fournir cette partie
417 Échec de l’attente Le serveur ne peut pas répondre aux exigences du champ d’en-tête de demande Expect

5xx : erreur de serveur #

Message Descriptif Description longue
500 Erreur de serveur interne Un message d’erreur générique, donné lorsqu’aucun message plus spécifique ne convient
501 Non implémenté Le serveur ne reconnaît pas la méthode de requête ou n’a pas la capacité de répondre à la requête
502 Bad Gateway Le serveur agissait comme une passerelle ou un proxy et a reçu une réponse invalide du serveur en amont
503 Service indisponible Le serveur est actuellement indisponible (surchargé ou en panne)
504 Délai d’expiration de la passerelle Le serveur agissait en tant que passerelle ou proxy et n’a pas reçu de réponse rapide du serveur en amont
505 Version HTTP non prise en charge Le serveur ne prend pas en charge la version du protocole HTTP utilisée dans la requête
511 Authentification réseau requise Le client doit s’authentifier pour accéder au réseau

Référence : [Codes HTTP][1]

[1] : https://www.w3schools.com/tags/ref_httpmessages.asp
