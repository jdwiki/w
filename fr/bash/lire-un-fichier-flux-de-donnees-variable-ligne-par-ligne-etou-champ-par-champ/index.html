Paramètres #

Paramètre Détails
IFS Séparateur de champ interne
fichier Un nom de fichier/chemin
-r Empêche l’interprétation de la barre oblique inverse lorsqu’il est utilisé avec read
-t Supprime une nouvelle ligne de fin de chaque ligne lue par readarray
-d DELIM Continuez jusqu’à ce que le premier caractère de DELIM soit lu (avec read), plutôt que le retour à la ligne

Boucle dans un fichier ligne par ligne #

while IFS= read -r line; do
   echo "$line"
done <file

Si le fichier peut ne pas inclure de retour à la ligne à la fin, alors :

while IFS= read -r line || [ -n "$line" ]; do
   echo "$line"
done <file

Boucle sur la sortie d’une commande champ par champ #

Supposons que le séparateur de champs est :

while IFS= read -d : -r field || [ -n "$field" ];do
    echo "**$field**"
done < <(ping google.com)

Ou avec un tuyau :

ping google.com | while IFS= read -d : -r field || [ -n "$field" ];do
    echo "**$field**"
done

Lit les lignes d’un fichier dans un tableau #

readarray -t arr <file

Ou avec une boucle :

arr=()
while IFS= read -r line; do
   arr+=("$line")
done <file

Lit les lignes d’une chaîne dans un tableau #

var='line 1
line 2
line3'
readarray -t arr <<< "$var"

ou avec une boucle :

arr=()
while IFS= read -r line; do
   arr+=("$line")
done <<< "$var"

Boucler une chaîne ligne par ligne #

var='line 1
line 2
line3'
while IFS= read -r line; do
   echo "-$line-"
done <<< "$var"

ou

readarray -t arr <<< "$var"
for i in "${arr[@]}";do
    echo "-$i-"
done

Boucler la sortie d’une ligne de commande par ligne #

while IFS= read -r line;do
    echo "**$line**"
done < <(ping google.com)

ou avec un tuyau :

ping google.com |
while IFS= read -r line;do
    echo "**$line**"
done

Lire un fichier champ par champ #

Supposons que le séparateur de champ est : (deux-points) dans le fichier file.

while IFS= read -d : -r field || [ -n "$field" ]; do
   echo "$field"
done <file

Pour un contenu :

first : se
con
d:
    Thi rd:
    Fourth

La sortie est :

**first **
** se
con
d**
**
    Thi rd**
**
    Fourth
**

Lire une chaîne champ par champ #

Supposons que le séparateur de champs est :

var='line: 1
line: 2
line3'
while IFS= read -d : -r field || [ -n "$field" ]; do
   echo "-$field-"
done <<< "$var"

Production:

-line-
- 1
line-
- 2
line3
-

Lire les champs d’un fichier dans un tableau #

Supposons que le séparateur de champs est :

arr=()
while IFS= read -d : -r field || [ -n "$field" ]; do
   arr+=("$field")
done <file

Lire les champs d’une chaîne dans un tableau #

Supposons que le séparateur de champs est :

var='1:2:3:4:
newline'
arr=()
while IFS= read -d : -r field || [ -n "$field" ]; do
   arr+=("$field")
done <<< "$var"
echo "${arr[4]}"

Production:

newline
