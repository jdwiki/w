L’un des plus grands avantages de LaTeX est sa capacité à composer des équations. Ici, les principes fondamentaux des équations de composition, certains des différents packages pouvant être utilisés, ainsi que les symboles courants, sont décrits.

Syntaxe #

  • \begin{équation} … \end{équation}
  • texte $ … $ texte
  • \usepackage{amsmath} … \begin{equation*} … \end{equation*}

Voici quelques idées de base pour vous assurer que votre code ne se brise pas et que vos équations sont plus belles :

  1. Assurez-vous que tous les crochets, les accolades, les signes dollar et les commandes \begin{} \end{} correspondent. C’est quelque chose où une petite erreur peut gâcher tout votre morceau de code de manière considérable.
  2. Si vous obtenez des erreurs, assurez-vous que le package approprié est chargé (par exemple, n’utilisez pas la commande \begin{equation*} sans le package amsmath).
  3. N’utilisez jamais, jamais, * jamais * le signe dollar double ("$$une équation ici$$) au lieu de \begin{equation}`.
  4. N’utilisez jamais le mode mathématique pour mettre votre texte en italique.
  5. Complètement coincé ? Essayez [TeX.SX][1], un site pour répondre aux questions sur TeX, LaTeX et les langages associés.

Bonne chance!

[1] : http://tex.stackexchange.com/

Équations de base #

** Équations simples en ligne **

Vous pouvez faire une simple équation en ligne en utilisant $une équation ici$.

Par exemple, vous pourriez faire

$\lim\limits_{n \to \infty} \frac{1}{2^n} i\bar z$

ce qui, si on met un petit faux texte autour, donne

[![entrez la description de l’image ici][1]][1]

** Équations numérotées et centrées **

Lors de la rédaction d’articles ou d’autres documents, il est parfois préférable d’avoir vos équations centrées et numérotées, par opposition à en ligne. Ensuite, utilisez les commandes \begin{equation} et \end{equation}.

Par exemple, si nous utilisons le code

\begin{equation}
\lim\limits_{n \to \infty} \frac{1}{2^n} i\bar z
\end{equation}

Et ajoutez un peu de texte autour, on obtient

[![entrez la description de l’image ici][2]][2]

Vous pouvez supprimer la numérotation de l’équation en utilisant \begin{equation*} et \end{equation*}.

Par exemple, si nous utilisons le code

\begin{equation*}
\lim\limits_{n \to \infty} \frac{1}{2^n} i\bar z
\end{equation*}

et ajouter un peu de texte autour, on obtient

[![entrez la description de l’image ici][3]][3]

(bien qu’il faille noter que vous devez utiliser le package amsmath pour cela).

Forfaits disponibles à l’utilisation #

Alors que LaTeX standard est tout ce qui est nécessaire pour la plupart des formules et équations mathématiques simples, parfois plus de symboles et d’outils sont nécessaires. Il existe plusieurs packages disponibles qui amélioreront vos équations et vous fourniront plus de travail. Trois des principaux packages sont décrits ci-dessous. N’oubliez pas que pour charger un package, tapez \usepackage{package} dans le préambule de votre document.

amsmath

Le paquet amsmath est un paquet incroyablement utile. Il est utilisé pour permettre à vos équations d’être centrées mais non numérotées, comme dans \begin{equation*}, il est utilisé pour créer des matrices (comme décrit ci-dessous) et il introduit de nombreuses autres commandes utiles, telles que \overset et \underset, décrits ci-dessous. La documentation du package amsmath peut être trouvée [ici][1].

mathtools

Le paquet mathtools s’appuie sur le paquet amsmath, ajoutant d’autres symboles et outils utiles. Il charge automatiquement le package amsmath, vous n’avez donc pas besoin de charger les deux dans le préambule de votre document. La documentation mathtools peut être trouvée [ici][2].

amssymb

Le package amssymb fournit de nombreux symboles supplémentaires qui peuvent être très pratiques pour des équations plus complexes. La documentation amssymb peut être trouvée [ici][3].

Packs de polices

Il existe également diverses polices que vous pouvez utiliser pour vos équations, comme décrit sur [cette question] [4] sur l’échange de pile TeX, pour TeX, LaTeX et leurs amis.

[Cet article][5] est une explication concise des différentes fonctionnalités fournies par certains packages ainsi que par LaTeX standard ; c’est très utile.

Bonnes commandes à connaître #

Certaines des commandes les plus courantes incluent :

  • Fractions et racines carrées : Pour les fractions, utilisez \frac {numerator}{denominator}. Pour les racines carrées, utilisez \sqrt[root]{nombre}.
  • lettres grecques : utilisez les commandes données dans le tableau ci-dessous :

[![entrez la description de l’image ici][1]][1]

  • Opérateurs : \leq donne le symbole inférieur ou égal à, \geq donne le symbole supérieur ou égal à, \neq donne le symbole différent, \sum donne la somme , \partial donne le symbole de la dérivée partielle, \nabla donne l’opérateur laplacien, \times donne le produit croisé ou le symbole de multiplication, \cdot donne le produit scalaire ou le symbole de multiplication, et \int donne le symbole intégral.
  • Flèches : \rightarrow et \leftarrow donnent respectivement des flèches droite et gauche.
  • Pourcentages : Si vous tapez % dans LaTeX, il est important d’inclure une barre oblique inverse, \% car le symbole de pourcentage est normalement utilisé pour les commentaires.
  • Exposants et indices : Pour créer un exposant, vous pouvez taper x^2 ou, pour les exposants plus longs, x^{2x}. Pour faire un indice, vous pouvez taper x_a, ou, pour les indices plus longs, x_{ab}.
  • Gras : Utilisez \boldmath{...} pour mettre vos symboles mathématiques en gras. D’autres options sont données à [cette question TeX.SX][2]. Les symboles mathématiques sont automatiquement mis en italique ; si vous ne voulez pas que cela soit vrai, créez votre texte d’équation comme décrit ci-dessous.
  • Infinity : Pour écrire l’infini, utilisez la commande \infty.
  • Déplacement d’éléments au-dessus ou en dessous d’un autre : Tout d’abord, pour les opérateurs mathématiques uniquement, il existe une autre méthode. Vous pouvez taper l’opérateur mathématique, dire \int, puis utiliser la commande \limits. Un exemple est \int\limits_{\infty} ou \int\limits^{\infty}. Ensuite, pour les cas normaux, vous pouvez faire \overset{top}{normal} ou \underset{bottom}{normal}. Cela peut être très utile pour faire des vecteurs. Par exemple, vous pourriez faire \overset{\rightarrow}{x} Le paquet amsmath est nécessaire pour overset et underset.
  • Accolades : Comme les accolades sont utilisées dans les commandes, il est nécessaire de taper \{ ou \} pour obtenir des accolades.
  • Text : Pour inclure du texte dans les équations, tapez \usepackage{amsmath} dans le préambule, puis tapez \text{...}.
  • Espace : Pour ajouter de l’espace dans vos équations, tapez \quad entre les deux éléments que vous souhaitez séparer (par exemple, vous pourriez avoir $2x \quad cos).

[1] : http://i.stack.imgur.com/vKdIS.png [2] : http://tex.stackexchange.com/questions/595/how-can-i-get-bold-math-symbols

Création de nouveaux symboles #

Disons que vous ne trouvez nulle part le symbole dont vous avez besoin. Vous pouvez créer un symbole personnalisé. Par exemple, le code

\documentclass{article}
\usepackage{graphicx,amsmath,amssymb}
\DeclareRobustCommand{\diamondtimes}{%
  \mathbin{\text{\rotatebox[origin=c]{45}{$\boxplus$}}}%
}

\begin{document}
$a\diamondtimes b$
\end{document}

crée et appelle un symbole, donnant

[![entrez la description de l’image ici][1]][1]

Ceci est un exemple plus simple; il suffit de faire pivoter un symbole déjà existant. Cependant, vous pouvez créer des symboles plus complexes.

Cette rubrique est en cours d’extension.

Matrices #

Matrices

Vous devez toujours utiliser le package amsmath si vous comptez utiliser les commandes suivantes. Il existe quatre principaux types de matrice, comme indiqué dans le code ci-dessous :

\begin{matrix} 
    a & b \\
    c & d 
\end{matrix}
\quad
\begin{pmatrix} 
   a & b \\
   c & d 
\end{pmatrix}
\quad
\begin{bmatrix} 
    a & b \\
    c & d 
\end{bmatrix}
\quad
\begin{vmatrix} 
    a & b \\
    c & d 
\end{vmatrix}
\quad
\begin{Vmatrix} 
    a & b \\
    c & d 
\end{Vmatrix}

Ce code produit

[![entrez la description de l’image ici][1]][1]

Il y a quelques points importants à noter à ce sujet :

  1. Il est important que vous placiez votre matrice dans l’environnement equation, equation* ou $...$ - la commande bmatrix n’est pas un environnement mathématique en soi.
  2. La construction de la matrice est en fait assez simple. Pour chaque ligne, vous créez chaque élément (disons x_{11}), puis mettez un &, puis écrivez l’élément suivant. Pour plusieurs lignes, à la fin de chaque ligne, mettez \\ (vous n’avez pas à le faire pour la dernière ligne). C’est assez similaire à un tableau dans ce cas.
