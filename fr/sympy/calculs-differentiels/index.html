Optimisation non linéaire contrainte #

Énoncé du problème :

Trouver le minimum (sur x, y) de la fonction f(x,y), sous réserve de g(x,y)=0, où f(x,y) = 2 * x**2 + 3 * y**2 et g(x,y) = x**2 + y* *2 - 4.

Solution : nous allons résoudre ce problème en procédant comme suit :

  1. Spécifiez la fonction lagrangienne pour le problème
  2. Déterminer les conditions de Karush-Kuhn-Tucker (KKT)
  3. Trouvez les tuples (x,y) qui satisfont les conditions KKT
  4. Déterminez lequel de ces tuples (x,y) correspond au minimum de f(x,y)

Définissez d’abord les variables d’optimisation ainsi que les fonctions objectifs et contraintes :

import sympy as sp
x, y = sp.var('x,y',real=True);
f = 2 * x**2 + 3 * y**2
g = x**2 + y**2 - 4

Ensuite, définissez la fonction lagrangienne qui inclut un multiplicateur de Lagrange lam correspondant à la contrainte

lam = sp.symbols('lambda', real = True)
L = f - lam* g

Maintenant, nous pouvons calculer l’ensemble des équations correspondant aux conditions KKT.

gradL = [sp.diff(L,c) for c in [x,y]] # gradient of Lagrangian w.r.t. (x,y)
KKT_eqs = gradL + [g]
KKT_eqs

[-2*lambda*x + 4*x, -2*lambda*y + 6*y, x**2 + y**2 - 4]

Les minimiseurs potentiels de f (étant donné g=0) sont obtenus en résolvant les équations KKT_eqs sur x, y, lam :

stationary_points = sp.solve(KKT_eqs, [x, y, lam], dict=True) # solve the KKT equations
stationary_points

[{x : -2, y : 0, lambda : 2}, {x : 2, y : 0, lambda : 2}, {x : 0, y : -2, lambda : 3}, {x : 0, y : 2, lambda : 3}]

Enfin, vérifiez la fonction objectif pour chacun des points ci-dessus afin de déterminer le minimum

[f.subs(p) for p in stat_points]

[8, 8, 12, 12]

Il s’ensuit que le minimum contraint de ‘f’ est égal à ‘8’ et est atteint à ‘(x,y)=(-2,0)’ et ‘(x,y)=(2,0)’.
