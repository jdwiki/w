Les caractères peuvent être constitués de plusieurs points de code

Un point de code Unicode, ce que les programmeurs pensent souvent d’un caractère, correspond souvent à ce que l’utilisateur pense être un caractère. Parfois, cependant, un “caractère” est composé de plusieurs points de code, comme le montrent les exemples ci-dessus.

Cela signifie que des opérations telles que le découpage d’une chaîne ou l’obtention d’un caractère à un index donné peuvent ne pas fonctionner comme prévu. Par exemple le 4ème caractère de la chaîne "Café" est 'e' (sans l’accent). De même, couper la chaîne à la longueur 4 supprimera l’accent.

Le terme technique pour un tel groupe de points de code est un grappe de graphèmes. Voir UAX #29 : Segmentation de texte Unicode

Diacritiques #

Une lettre avec un signe diacritique peut être représentée par la lettre et une lettre modificative de combinaison. Vous pensez normalement à é comme un seul caractère, mais il s’agit en réalité de 2 points de code :

  • U+0065 — LETTRE MINUSCULE LATINE E
  • U+0301 — COMBINAISON D’UN ACCENT AIGU

De même ç = c + ¸, et å = a + ˚

formulaires combinés #

Pour compliquer les choses, il existe souvent un point de code pour le formulaire composé également :

"Café" = 'C' + 'a' + 'f' + 'e' + '´'
"Café" = 'C' + 'a' + 'f' + 'é'

Bien que ces chaînes se ressemblent, elles ne sont pas égales et n’ont même pas la même longueur (respectivement 5 et 4).

Emoji et drapeaux #

Beaucoup d’emoji se composent de plus d’un point de code.

  • 🇯🇵 : Un drapeau est défini comme une paire de “lettres indicatrices de symboles régionaux” (🇯 + 🇵)
  • 🙋🏿 : Certains emoji peuvent être suivis d’un modificateur de teint : 🙋 + 🏿
  • 😀︎ ou 😀️ : Windows 10 permet de spécifier si un emoji est coloré ou noir/blanc en ajoutant un sélecteur de variation (U+FE0E ou U+FE0F)
  • 👨‍👩‍👧‍👦 : une famille. Encodé en joignant les emoji pour garçon, fille, femme et homme (👦, 👧, 👩, 👨) avec des menuisiers de largeur nulle (U+200D). Sur les plates-formes qui le prennent en charge, cela est rendu comme un emoji d’une famille avec deux enfants.

Texte Zalgo #

Il y a cette chose appelée [Zalgo Text] (http://stackoverflow.com/questions/6579844/how-does-zalgo-text-work) qui pousse cela à l’extrême. Voici le premier groupe de graphèmes de l’exemple. Il se compose de 15 points de code : la lettre latine « H » et 14 marques de combinaison.

 

H̡̫̤̤̣͉̤ͭ̓̓̇͗̎̀

 

Bien que cela n’apparaisse pas dans le texte normal, cela montre qu’un “caractère” peut vraiment consister en un nombre arbitraire de points de code
