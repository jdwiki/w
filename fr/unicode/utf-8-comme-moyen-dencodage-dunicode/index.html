Qu’est-ce que l’UTF-8 ?

UTF-8 est un encodage, qui est de longueur variable et utilise des unités de code 8 bits - c’est pourquoi UTF-8. Sur Internet, UTF-8 est le codage dominant (avant 2008, ASCII était, qui peut également gérer n’importe quel point de code Unicode.).

** UTF-8 est-il identique à Unicode ? **

“Unicode” n’est pas un encodage - c’est un jeu de caractères codés - c’est-à-dire un jeu de caractères et un mappage entre les caractères et les points de code entiers les représentant. Mais beaucoup de documentation l’utilise pour faire référence aux encodages. Sous Windows, par exemple, le terme Unicode est utilisé pour désigner UTF-16.

UTF-8 n’est qu’un des moyens d’encoder Unicode et en tant qu’encodage, il convertit les séquences d’octets en séquences de caractères et vice versa. UTF-16 et -32 sont d’autres formats de transformation Unicode.

** Nomenclature d’UTF-8 **

Tous les trois peuvent avoir des marques d’ordre d’octet spécifiques, qui étant un nombre magique signalent plusieurs choses importantes à un programme (par exemple, Notepad ++) - par exemple, le fait que le flux de texte importé est Unicode ; cela aide également à détecter l’art d’Unicode utilisé pour ce flux. Cependant, le consortium Unicode recommande de stocker UTF-8 sans aucune signature. Certains logiciels, par exemple le compilateur gcc, se plaignent si un fichier contient la signature UTF-8. De nombreux programmes Windows utilisent par contre la signature. Et essayer de détecter l’encodage d’un flux d’octets ne fonctionne pas toujours.

Comment vérifier si votre projet a un encodage UTF-8 ou non

UTF-8 n’est pas encore universel, et les ingénieurs en logiciel et les scientifiques des données sont souvent confrontés à des problèmes d’encodage des flux de texte. Parfois, UTF-8 est censé être utilisé dans le projet, mais un autre ecndoing est utilisé. Il existe plusieurs outils pour détecter l’encodage du fichier :

  • Certains outils CMD, comme l’outil de ligne de commande Linux ‘[file][1]’ ou powershell ;
  • Paquet Python “chardet”
  • Notepad ++ comme peut-être l’outil le plus populaire pour la vérification manuelle.

[1] : http://gnuwin32.sourceforge.net/packages/file.htm

Comment changer l’encodage par défaut du serveur en UTF-8 #

Parfois, les utilisateurs d’autres régions que les anglophones ont des problèmes d’encodage lors de la programmation d’un projet php, par exemple. Il se peut que le serveur ait un autre encodage que l’UTF-8, et si quelqu’un veut créer un projet php en UTF-8 sur ce serveur, son texte pourrait s’afficher incorrect.

Exemple : il se peut que l’encodage par défaut de votre serveur soit Windows-1251 - vous devez alors supprimer le AddDefaultCharset windows-1251 du fichier de serveur .htaccess et écrire AddDefaultCharset utf-8.

Pour vérifier quel encodage possède votre serveur, ne définissez pas la balise <META charset> et activez la "détection automatique de l'encodage" dans votre navigateur.

Enregistrer un fichier Excel en UTF-8 #

Excel -> Enregistrer sous -> Enregistrer sous type -> “Valeur séparée par des virgules (*.csv)” ET Outils (à gauche pour le bouton Enregistrer) -> Options Web -> Encodage -> Enregistrer ce document sous -> Unicode (UTF-8 )

Comment convertir un tableau d’octets de données UTF-8 en une chaîne Unicode en Python #

def make_unicode(data):
    if type(data) != unicode:
        data =  data.decode('utf-8')
        return data
    else:
        return data
