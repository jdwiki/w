Affichage de la page de manuel d’une commande système #

man <command>

Cela affichera la page de manuel de la commande spécifiée.

Par exemple, man ping affichera :

PING(8) Manuel du gestionnaire du système BSD PING(8)

NOM ping - envoie des paquets ICMP ECHO_REQUEST aux hôtes du réseau

SOMMAIRE ping [-AaCDdfnoQqRrv] [-b limite si] [-c compte] [-G balayagemaxsize] [-g tailleminbalayage] [-h tailleincrbalayage] [-i attente] [-k classetrafic] [-l préchargement] [-M masque | heure] [-m ttl] [-P politique] [-p modèle] [-S src_addr] [-s packetsize] [-t timeout] [-W waittime] [-z tos] hôte ping [-AaDdfLnoQqRrv] [-b limite si] [-c compte] [-I iface] [-i attend] [-k classe de trafic] [-l préchargement] [-M masque | heure] [-m ttl] [-P politique] [-p pattern] [-S src_addr] [-s packetsize] [-T ttl] [-t timeout] [-W temps d’attente] [-z tos] groupe mcast

ECHO_REQUEST les datagrammes (pings'') ont un en-tête IP et ICMP, suivi d'une struct timeval’’ puis un nombre arbitraire d’octets ``pad’’ utilisés pour remplir le paquet. Lors de la visualisation de la page de manuel, elle peut être recherchée. Taper une barre oblique (/) suivie du terme de recherche sautera à la première occurrence du terme. Exemple : /ping

Appuyez ensuite sur N pour passer à l’occurrence suivante. Shift+N sautera à l’occurrence précédente.

Rechercher une page de manuel #

Vous pouvez rechercher des pages “man” contenant une chaîne particulière dans leur description en utilisant :

man -k <string>

Par exemple:

man -k unzip

Peut revenir :

man -k décompresser IO::Uncompress::Bunzip2(3pm) - Lire les fichiers/tampons bzip2 IO::Uncompress::Gunzip(3pm) - Lire les fichiers/tampons RFC 1952 IO::Uncompress::Unzip(3pm) - Lire les fichiers zip/tampons PerlIO::gzip(3pm) - Extension Perl pour fournir une couche PerlIO à gzip/gunzip gzip(1), gunzip(1), zcat(1) - compresser ou décompresser des fichiers IO::Uncompress::Bunzip2(3pm) - Lire les fichiers/tampons bzip2 IO::Uncompress::Gunzip(3pm) - Lire les fichiers/tampons RFC 1952 IO::Uncompress::Unzip(3pm) - Lire les fichiers zip/tampons PerlIO::gzip(3pm) - Extension Perl pour fournir une couche PerlIO à gzip/gunzip bzip2(1), bunzip2(1) - un compresseur de fichiers de tri par blocs, v1.0.6 bzcat - décompresse les fichiers vers stdout bzip2recover - récupère les données des fichiers bzip2 endommagés funzip(1) - filtre pour extraire d’une archive ZIP dans un tube unzip(1) - liste, teste et extrait les fichiers compressés dans une archive ZIP unzipsfx(1) - stub auto-extractible à ajouter aux archives ZIP

Trouver une page de manuel dans une section différente #

Parfois, un terme est défini dans plusieurs sections du manuel. Par défaut, man n’affichera que la première page trouvée, ce qui peut être ennuyeux pour les programmeurs car les fonctions C sont documentées dans une section ultérieure aux commandes et aux appels système. Utilisez ce qui suit pour afficher toutes les pages qui correspondent à un nom :

$ man -wa printf
/usr/share/man/man1/printf.1.gz
/usr/share/man/man1p/printf.1p.gz
/usr/share/man/man3/printf.3.gz
/usr/share/man/man3p/printf.3p.gz

Pour afficher la page d’une section spécifique, placez-la simplement avant le terme :

man 3 printf

Lire un fichier manuel avec man #

Cela revient à lire un manuel pour une commande :

man /path/to/man/file

Obtenir le chemin du fichier pour une page de manuel #

$ man -w find
/usr/share/man/man1/find.1.gz

$ man -w printf
/usr/share/man/man1/printf.1.gz

$ man -w man
/usr/share/man/man1/man.1.gz
