Installation ou configuration (Mac OS X) #

APERÇU #

cocos2d-x est un moteur de jeu multiplateforme open source. Il permet aux développeurs de coder en déploiement C++, Lua et Javascript sur iOS, Android, Windows Phone, Mac OS X, Windows Desktop et Linux.

CONDITIONS PRÉALABLES #

Exigences de construction

  • Mac OS X 10.7+, Xcode 4.6+
  • Windows 7+, CONTRE 2012+ -Ubuntu 12.10+, CMake 2.6+ -Python 2.7.5
  • NDK, JDK, ANT (développement Android)

Exigences d’exécution

  • iOS 5.0+ pour les jeux iPhone/iPad
  • Android 2.3+ pour les jeux Android
  • Windows Phone 8+ pour les jeux Windows Phone
  • OS X v10.6+ pour les jeux Mac
  • Windows 7+ pour les jeux Windows

Installation #

Téléchargez tout ci-dessous :

  • [Cocos2d-x v3.x][1]
  • [JDK/SDK 1.6+][2]
  • [Outils de ligne de commande Android][3]
  • [RDA][4]
  • [Fourmi Apache] [5]
  • [Python 2.7.5][6]

[1] : http://cocos2d-x.org/download [2] : http://www.oracle.com/technetwork/java/javase/downloads/index.html [3] : http://%20https://developer.android.com/studio/index.html [4] : https://developer.android.com/tools/sdk/ndk/index.html [5] : http://ant.apache.org/bindownload.cgi [6] : https://www.python.org/downloads/

Après avoir téléchargé tout ce qui précède,

  • décompressez Cocos2d-x
  • installer le JDK.
  • décompressez le SDK Android et le NDK au même emplacement racine. (peut être: ~/AndroidDev)
  • vérifiez que Python 2.7 est installé et accessible.
  • installez Apache Ant et vérifiez qu’il fonctionne.

Dans votre répertoire Cocos2d-x, exécutez :

configuration python.py

attention : vous ne devez pas utiliser le signe ~. Utilisez le chemin complet vers votre répertoire home. Sinon, les scripts échoueront en raison de la valeur du chemin d’erreur.

  • Maintenant, il demandera le chemin NDK_ROOT. Ici, entrez le répertoire NDK extrait chemin:

~/android-ndk-r10e

  • SDK_ROOT. Ici, entrez le chemin du SDK Android :

~/android-sdk-macosx

  • ANT_ROOT. Ici, entrez le chemin de la corbeille Apache ANT :

~/apache-ant-1.9.6/bin

Après avoir défini les variables d’environnement ci-dessus, exécutez :

source ~/.bash_profile

Vola, le cocos2d-x a été installé avec succès sur votre système.

Installation ou configuration (Windows) #

APERÇU #

cocos2d-x est un moteur de jeu multiplateforme open source. Il permet aux développeurs de coder en déploiement C++, Lua et Javascript sur iOS, Android, Windows Phone, Mac OS X, Windows Desktop et Linux.

CONDITIONS PRÉALABLES #

Exigences de construction

  • Mac OS X 10.7+, Xcode 4.6+
  • Windows 7+, CONTRE 2012+ -Ubuntu 12.10+, CMake 2.6+ -Python 2.7.5
  • NDK, JDK, ANT (développement Android)

Exigences d’exécution

  • iOS 5.0+ pour les jeux iPhone/iPad
  • Android 2.3+ pour les jeux Android
  • Windows Phone 8+ pour les jeux Windows Phone
  • OS X v10.6+ pour les jeux Mac
  • Windows 7+ pour les jeux Windows

Installation #

Téléchargez tout ci-dessous :

  • [Cocos2d-x v3.x][1]
  • [JDK/SDK 1.6+][2]
  • [Outils de ligne de commande Android][3]
  • [RDA][4]
  • [Fourmi Apache] [5]
  • [Python 2.7.5][6]

Après avoir téléchargé tout ce qui précède,

  • décompressez Cocos2d-x
  • installez le JDK et créez en même temps une nouvelle variable appelée : JAVA_HOME et donnez-lui le chemin vers l’endroit où vous avez installé le JDK ci-dessus. [![entrez la description de l’image ici][7]][7]
  • décompressez le SDK Android et le NDK au même emplacement racine. (peut être: ~/AndroidDev)
  • installez Python et notez où vous l’avez placé.
  • extrayez Apache Ant et placez le dossier entier où vous souhaitez le conserver. Prenez note de l’endroit où vous l’avez placé.
  • ajoutez les chemins pour Apache Ant et Python à votre variable PATH. [![entrez la description de l’image ici][8]][8]
  • redémarrer
  • maintenant, testez votre environnement avant de continuer. Lancez une invite de commande et exécutez :

fourmi

`python`

Si tout fonctionne, il est temps d’exécuter setup.py pour configurer votre environnement de développement Android. Cela définira les variables d’environnement nécessaires. Si vous n’avez pas configuré cet environnement auparavant, vous serez invité à saisir les chemins des variables introuvables. Vous exécutez setup.py à partir du répertoire dans lequel se trouve Cocos2d-x.

Dans votre répertoire Cocos2d-x, exécutez :

configuration python.py

attention : vous ne devez pas utiliser le signe ~. Utilisez le chemin complet vers votre répertoire home. Sinon, les scripts échoueront en raison de la valeur du chemin d’erreur.

  • Maintenant, il demandera le chemin NDK_ROOT. Ici, entrez le répertoire NDK extrait chemin:

~/android-ndk-r10e

  • SDK_ROOT. Ici, entrez le chemin du SDK Android :

~/android-sdk-macosx

  • ANT_ROOT. Ici, entrez le chemin de la corbeille Apache ANT :

~/apache-ant-1.9.6/bin

  • Redémarrer

Vola, le cocos2d-x a été installé avec succès sur votre système.
