Vérifiez si Redis fonctionne #

redis-cli ping

Cela devrait retourner PONG

Accéder à Redis Cli #

En supposant que vous exécutez le serveur Redis sur localhost, vous pouvez taper la commande

redis-cli

Après cette commande apparaît l’invite de ligne de commande redis

127.0.0.1:6379>

Types de données Redis #

Voici la liste de toutes les structures de données prises en charge par Redis :

  • Chaînes binaires sûres
  • Listes : collections d’éléments de chaîne triés selon l’ordre d’insertion.
  • Ensembles : collections d’éléments de chaîne uniques et non triés.
  • Ensembles triés : similaire aux ensembles mais où chaque élément de chaîne est associé à une valeur numérique flottante, appelée score.
  • Hashes : sont des cartes composées de champs associés à des valeurs.
  • HyperLogLogs : il s’agit d’une structure de données probabiliste qui est utilisée pour estimer la cardinalité d’un ensemble.

Basé sur la documentation officielle redis.io

Installer Redis #

wget http://download.redis.io/redis-stable.tar.gz
tar xvzf redis-stable.tar.gz
cd redis-stable
make

Démarrage de Redis #

redis-server

Installation et exécution de Redis Server sous Windows #

Remarque : Le projet Redis ne prend pas officiellement en charge Windows.

Cependant, le groupe Microsoft Open Tech développe et maintient ce port Windows ciblant Win64. [Redis.io officiel/download][1]

Vous pouvez choisir de télécharger différentes versions ou la dernière version de Redis [ github.com/MSOpenTech/redis/releases][2]

  1. Téléchargez le fichier .msi ou .zip, ce tutoriel vous permettra de télécharger le dernier fichier zip [Redis-x64-3.2.100.zip][3].
  2. Extraire le fichier zip dans le répertoire préparé. [![entrez la description de l’image ici][4]][4]
  3. Exécutez redis-server.exe, vous pouvez exécuter directement redis-server.exe en cliquant sur ou exécuter via l’invite de commande. [![entrez la description de l’image ici][5]][5]
  4. Exécutez redis-cli.exe, après avoir exécuté avec succès le serveur redis. Vous pouvez y accéder et tester les commandes en exécutant redis-cli.exe La

[![entrez la description de l’image ici][6]][6]

La commande PING est utilisée pour tester si une connexion est toujours active. [![enter image description here][7]][7]

Vous pouvez maintenant commencer à utiliser Redis, veuillez vous référer pour plus de [commandes dans les documentations officielles][8]
