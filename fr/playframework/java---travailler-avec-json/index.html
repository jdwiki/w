Création manuelle de JSON #

import play.libs.Json;


public JsonNode createJson() {
    // {"id": 33, "values": [3, 4, 5]}
    ObjectNode rootNode = Json.newObject();
    ArrayNode listNode = Json.newArray();

    long values[] = {3, 4, 5};
    for (long val: values) {
        listNode.add(val);
    }
    
    rootNode.put("id", 33);
    rootNode.set("values", listNode);
    return rootNode;
}

Chargement de json à partir d’une chaîne/fichier #

import play.libs.Json;
// (...)

Chargement d’un fichier depuis votre dossier public #

// Note: "app" is an play.Application instance
JsonNode node = Json.parse(app.resourceAsStream("public/myjson.json"));

Charger à partir d’une chaîne #

String myStr = "{\"name\": \"John Doe\"}";
JsonNode node = Json.parse(myStr);

Transversaliser un document JSON #

Dans les exemples suivants, json contient un objet JSON avec les données suivantes :

[
  {
    "name": "John Doe",
    "work": {
      "company": {
        "name": "ASDF INC",
        "country": "USA"
      },
      "cargo": "Programmer"
    },
    "tags": ["java", "jvm", "play"]
  },
  {
    "name": "Bob Doe",
    "work": {
      "company": {
        "name": "NOPE INC",
        "country": "AUSTRALIA"
      },
      "cargo": "SysAdmin"
    },
    "tags": ["puppet", "ssh", "networking"],
    "active": true
  }
]

Obtenir le nom d’un utilisateur (non sécurisé) #

JsonNode node = json.get(0).get("name"); // --> "John Doe"
// This will throw a NullPointerException, because there is only two elements
JsonNode node = json.get(2).get("name"); // --> *crash*

Obtenir le nom d’utilisateur (manière sûre) #

JsonNode node1 = json.at("/0/name"); // --> TextNode("John Doe")
JsonNode node2 = json.at("/2/name"); // --> MissingNode instance
if (! node2.isMissingNode()) {
    String name = node2.asText();    
}

Obtenir le pays où travaille le premier utilisateur #

JsonNode node2 = json.at("/0/work/company/country"); // TextNode("USA")

Obtenez tous les pays #

List<JsonNode> d = json.findValues("country"); // List(TextNode("USA"), TextNode("AUSTRALIA"))

Trouver tous les utilisateurs contenant l’attribut “actif” #

List<JsonNode> e = json.findParents("active"); // List(ObjectNode("Bob Doe"))

Conversion entre les objets JSON et Java (de base) #

Par défaut, Jackson (la bibliothèque utilisée par Play JSON) essaiera de mapper chaque champ public sur un champ json portant le même nom. Si l’objet a des getters/setters, il en déduira le nom. Donc, si vous avez une classe Book avec un champ privé pour stocker l’ISBN et que vous avez des méthodes get/set nommées getISBN/setISBN, Jackson va

  • Créez un objet JSON avec le champ “ISBN” lors de la conversion de Java en JSON
  • Utilisez la méthode setISBN pour définir le champ isbn dans l’objet Java (si l’objet JSON a un champ “ISBN”).

Créer un objet Java à partir de JSON #

public class Person {
    String id, name;
}

JsonNode node = Json.parse("{\"id\": \"3S2F\", \"name\", \"Salem\"}");
Person person = Json.fromJson(node, Person.class);
System.out.println("Hi " + person.name); // Hi Salem

Créer un objet JSON à partir d’un objet Java #

// "person" is the object from the previous example
JsonNode personNode = Json.toJson(person)

Création d’une chaîne JSON à partir d’un objet JSON #

// personNode comes from the previous example
String json = personNode.toString();
// or 
String json = Json.stringify(json);

JSON jolie impression #

System.out.println(personNode.toString());
/* Prints:
{"id":"3S2F","name":"Salem"}
*/   

System.out.println(Json.prettyPrint(personNode));
/* Prints:
{
  "id" : "3S2F",
  "name" : "Salem"
}
*/
