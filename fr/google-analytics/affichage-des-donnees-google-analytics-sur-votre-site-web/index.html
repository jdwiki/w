Affichage des données Google Analytics sur votre site Web #

Ce document explique comment obtenir des jetons d’accès Google et les utiliser pour obtenir des données Google Analytics à afficher sur nos sites Web.

Exemple : Un exemple en direct est disponible dans

https://newtonjoshua.com

Remarque : Utilisez le même compte Gmail pour toutes les étapes ci-dessous.

ÉTAPE 1 : Configurer Google Analytics #

Suivez les étapes ci-dessous pour configurer Google Analytics sur votre site Web

  1. Connectez-vous à votre compte Analytics.
  2. Sélectionnez l’onglet Admin.
  3. Sélectionnez un compte dans le menu déroulant de la colonne COMPTE.
  4. Sélectionnez une propriété dans le menu déroulant de la colonne PROPRIÉTÉ.
  5. Sous PROPRIÉTÉ, cliquez sur Informations de suivi > Code de suivi.
  6. Pour collecter des données, vous devez copier et coller le code de suivi Analytics into the source code on every web page you wish to track.
  7. Une fois que vous avez l’extrait de code de suivi Javascript pour votre property, copy the snippet exactly without editing it.
  8. Collez votre extrait de code de suivi (non modifié, dans son intégralité) avant the closing tag on every web page on your site you wish to track.
  9. Une fois que vous avez installé avec succès le suivi Analytics, cela peut prendre up to 24 hours for data such as traffic referral information, user characteristics, and browsing information to appear in your reports

se référer à,

  1. https://support.google.com/analytics/answer/1008080?hl=fr
  2. https://analytics.google.com

ÉTAPE 2 : Obtenez des jetons #

Projet Google :

Pour créer un projet Google Cloud Platform, ouvrez la Google Developers Console ( https://console.developers.google.com ) et cliquez sur * Créer un projet.*

Activer l’accès à l’API OAuth 2.0 :

Votre application devra accéder aux données utilisateur et contacter d’autres services Google en votre nom. Utilisez OAuth 2.0 pour accorder l’accès à l’API de votre application.

Pour activer cela, vous avez besoin d’un ID client :

  1. Ouvrez la page des informations d’identification de la console d’API Google (https://console.developers.google.com/apis/credentials).

  2. Du projet liste déroulante, sélectionnez votre projet.

  3. Sélectionnez Créer des identifiants et choisissez ID client OAuth.

  4. Sous Type d’application, sélectionnez Application Web, entrez un Nom et

  5. ensemble the Restrictions by entering JavaScript origins, Redirect URIs to point the website where you are planning to display the data, then click Create.

  6. Notez le OAuth 2.0 client_id and client_secret. You will need them to configure the UI.

Obtenir le code d’autorisation:

Entrez dans le navigateur,

https://accounts.google.com/o/oauth2/auth?scope=https://www.googleapis.com/auth/analytics.readonly&response_type=code&client_id={{ client_id}}&redirect_uri={{redirect_uri }} &approval_prompt=force&access_type=hors ligne

Vous serez redirigé vers

{{redirect_uri }}?code=={{authorization_code}}#

Obtenir un jeton d’actualisation :

Envoyez une requête POST, éventuellement via une console REST à

https://www.googleapis.com/oauth2/v3/token?code={{authorization_code}} &client_id={{client_id}}&client_secret={{client_secret}} &redirect_uri={{redirect_uri }}&grant_type=authorization_code

Vous obtiendrez une réponse JSON avec

{“refresh_token”: refresh_token}

Vous pouvez utiliser le jeton d’actualisation pour obtenir un jeton d’accès pour accéder aux API Google

Obtenir un jeton d’accès :

Envoyez une requête POST à,

https://www.googleapis.com/oauth2/v3/token?client_id={{client_id}} &client_secret={{client_id}} &grant_type=refresh_token&refresh_token={{refresh_token}}

Vous obtiendrez un JSON avec access_token dans la réponse.

{access_token : {{access_token}}}

Exemple:

 var access_token = '';
function getAccessToken(){
    $.post('https://www.googleapis.com/oauth2/v3/token', {
            client_id: {{client_id}},
            client_secret: {{client_secret}},
            grant_type: 'refresh_token',
            refresh_token: {{refresh_token}}
        }, function (data, status) {
            if (status === 'success') {
                access_token = data.access_token;
                // Do something eith the access_token
            } else {
                console.error(status);
            }
        });
}

Vérifiez la validité du jeton :

Envoyez une requête POST à,

https://www.googleapis.com/oauth2/v1/tokeninfo?access_token={{access_token}}

Exemple:

function checkValidity() {
    $.post('https://www.googleapis.com/oauth2/v1/tokeninfo', {
            access_token:{{access_token}}
        }).done(function (data, status) {
            if (status === 'success') {
                console.debug(data.expires_in);
                var check = false;
                check = data.hasOwnProperty('expires_in');
                if (check) {
                    // Token is valid
                }
                if (!check) {
                    getAccessToken();
                }
            } else {
                console.debug(status);
            }

        })
        .fail(function (data) {
            console.error(data);
            getAccessToken();
        });
}

Étape 3 : Récupérer les données #

API intégrée :

L’API GA Embed est une bibliothèque JavaScript qui vous permet de créer et d’intégrer facilement votre tableau de bord GA sur votre site Web en quelques minutes.

consultez https://developers.google.com/analytics/devguides/reporting/embed/v1/getting-started

Explorateur de requêtes : visitez Embed API Query Explorer et autorisez

https://ga-dev-tools.appspot.com/query-explorer/

Sélectionnez la vue pour laquelle vous souhaitez récupérer les données.

Sélectionnez les statistiques et dimensions requises.

Exemple:

Obtenir les données du pays (je veux connaître le nombre d’utilisateurs accédant à mon site Web depuis chaque pays)

Pour obtenir ces données, sélectionnez les métriques comme “utilisateurs” et les dimensions comme “pays”.

Cliquez sur Exécuter la requête

Vous trouverez les données d’analyse de la requête affichées dans un tableau.

Copiez l’URI de requête d’API. Et ajoutez access_token={{access_token}} à l’uri

Exemple:

https://www.googleapis.com/analytics/v3/data/ga?ids={{ids}}&start-date=2015-07-01&end-date=today&metrics=ga%3Auzers&dimensions=ga%3Apays &access_token={{access_token}}

Envoyez une requête POST aux URI pour obtenir les données dans votre navigateur.

Exemple:

function gaGetCountry() {
    $.get('https://www.googleapis.com/analytics/v3/data/ga?' +
        'ids={{ids}}' +
        'start-date=2015-07-01&' +
        'end-date=today&' +
        'metrics=ga%3Ausers&' +
        'dimensions=ga%3Acountry&' +
        'sort=ga%3Ausers&' +
        'filters=ga%3Ausers%3E10&' +
        'max-results=50' +
        '&access_token=' + {{access_token}},
        function (data, status) {
            if (status === 'success') {

                // Display the Data
                drawRegionsMap(data.rows);

            } else {
                console.debug(status);
            }

        });
}

Étape 4 : Afficher les données #

Maintenant, nous avons rassemblé les données. Enfin, nous devons les afficher sur notre site Web.

“Afficher des données en direct sur votre site” est le titre de Google Charts. Et c’est ce que nous allons faire.

consultez https://developers.google.com/chart/

L’exemple suivant dessinera un [GeoChart][1] dans la div avec id=‘countryChart’

 //Draw country Chart
 function drawRegionsMap(data) {

        var head = data[0];
        head[0] = 'Country';
        head[1] = 'Users';
        for (var i = 1; i < data.length; i++) {
            var d = data[i];
            d[1] = Number(d[1]);
        }

        var chartData = google.visualization.arrayToDataTable(data);
        var options = {
            title: 'My Website is viewed from,',
            domain: '{{Country Code eg: IN for India}}',
            tooltip: {
                textStyle: {
                    color: 'navy'
                },
                showColorCode: true
            },
            legend: {
                textStyle: {
                    color: 'navy',
                    fontSize: 12
                }
            },
            colorAxis: {
                colors: ['#00FFFF', '#0000FF']
            }
        };
    
        var chart = new google.visualization.GeoChart(document.getElementById('countryChart'));
    
        chart.draw(chartData, options); 
}

Reportez-vous à https://newtonjoshua.com pour voir l’exple ci-dessus en action.
