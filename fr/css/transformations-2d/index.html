Syntaxe #

  • Tourner Transformer
  • transformer : faire pivoter (<angle>)
  • Traduire Transformer
  • transformer : translate(<longueur-ou-pourcentage> [, <longueur-ou-pourcentage>] ?)
  • transformer : translateX(<length-or-percentage>)
  • transformer : translateY(<length-or-percentage>)
  • Transformation oblique
  • transformer : obliquer(<angle> [, <angle>] ?)
  • transformation : skewX(<angle>)
  • transformation : skewY(<angle>)
  • Transformation d’échelle
  • transformer : échelle(<facteur d’échelle> [, <facteur d’échelle>] ?)
  • transformer : scaleX(<scale-factor>)
  • transformer : scaleY(<scale-factor>)
  • Transformation matricielle
  • transformation : matrice(<nombre> [, <nombre> ]{5,5})

Paramètres #

Fonction/Paramètre Détails
tourner(x) Définit une transformation qui déplace l’élément autour d’un point fixe sur l’axe Z
translate(x,y) Déplace la position de l’élément sur les axes X et Y
traduireX(x) Déplace la position de l’élément sur l’axe X
translateY(y) Déplace la position de l’élément sur l’axe Y
échelle(x,y) Modifie la taille de l’élément sur les axes X et Y
échelleX(x) Modifie la taille de l’élément sur l’axe X
échelleY(y) Modifie la taille de l’élément sur l’axe Y
obliquer(x,y) Mappage de cisaillement, ou transvection, déformant chaque point d’un élément d’un certain angle dans chaque direction
obliqueX(x) Mappage de cisaillement horizontal déformant chaque point d’un élément d’un certain angle dans la direction horizontale
obliqueY(y) Cartographie de cisaillement vertical déformant chaque point d’un élément d’un certain angle dans la direction verticale
matrice() Définit une transformation 2D sous la forme d’une matrice de transformation.
angle L’angle de rotation ou d’inclinaison de l’élément (selon la fonction avec laquelle il est utilisé). L’angle peut être fourni en degrés (deg), grades (grad), radians (rad) ou tours (turn). Dans la fonction skew(), le deuxième angle est facultatif. S’il n’est pas fourni, il n’y aura pas d’inclinaison (0) sur l’axe Y.
longueur ou pourcentage La distance exprimée en longueur ou en pourcentage par laquelle l’élément doit être translaté. Dans la fonction translate(), la deuxième longueur ou pourcentage est facultative. S’il n’est pas fourni, il n’y aurait pas de translation (0) sur l’axe Y.
facteur d’échelle Un nombre qui définit combien de fois l’élément doit être mis à l’échelle dans l’axe spécifié. Dans la fonction scale(), le deuxième facteur d’échelle est facultatif. S’il n’est pas fourni, le premier facteur d’échelle sera également appliqué pour l’axe Y.

Système de coordonnées 2D #

Les transformations sont faites selon un système de coordonnées 2D X/Y. L’axe X va de droite à gauche et l’axe Y va vers le bas comme indiqué dans l’image suivante :

[![Système de coordonnées CSS 2D][1]][1]

Ainsi, un translateY() positif va vers le bas et un translateX() positif va vers la droite.

Prise en charge du navigateur et préfixes #

  • IE supporte cette propriété depuis IE9 avec le préfixe -ms-. Les anciennes versions et Edge n’ont pas besoin du préfixe
  • Firefox le supporte depuis la version 3.5 et a besoin du préfixe -moz- jusqu’à la version 15
  • Chrome depuis la version 4 et jusqu’à la version 34 a besoin du préfixe -webkit-
  • Safari a besoin du préfixe -webkit- jusqu’à la version 8
  • Opera a besoin du préfixe -o- pour la version 11.5 et du préfixe -webkit- de la version 15 à 22
  • Android a besoin du préfixe -webkit- de la version 2.1 à 4.4.4

Exemple de transformation préfixée : #

-webkit-transform: rotate(45deg);
    -ms-transform: rotate(45deg);
        transform: rotate(45deg);

[1] : http://i.stack.imgur.com/epKMy.jpg

Rotation #

HTML

<div class="rotate"></div>

CSS

.rotate {
    width: 100px;
    height: 100px;
    background: teal;
    transform: rotate(45deg);
}

Cet exemple fera pivoter la div de 45 degrés dans le sens des aiguilles d’une montre. Le centre de rotation est au centre de la div, “50 %” à partir de la gauche et “50 %” à partir du haut. Vous pouvez modifier le centre de rotation en définissant la propriété transform-origin.

transform-origin: 100% 50%;

L’exemple ci-dessus définira le centre de rotation au milieu de l’extrémité droite.

Échelle #

HTML

<div class="scale"></div>

CSS

.scale {
    width: 100px;
    height: 100px;
    background: teal;
    transform: scale(0.5, 1.3);
}

Cet exemple mettra à l’échelle la div à 100px * 0.5 = 50px sur l’axe X et à 100px * 1.3 = 130px sur l’axe Y.

Le centre de la transformation est au centre de la div, “50 %” à partir de la gauche et “50 %” à partir du haut.

Incliner #

HTML

<div class="skew"></div>

CSS

.skew {
    width: 100px;
    height: 100px;
    background: teal;
    transform: skew(20deg, -30deg);
}

Cet exemple inclinera la div de 20 degrés sur l’axe X et de - 30 degrés sur l’axe Y. Le centre de la transformation est au centre de la div, “50 %” à partir de la gauche et “50 %” à partir du haut.

Voir le résultat [ici][1].

[1] : https://jsfiddle.net/MadalinaTn/gtt4osms/1/

Transformations multiples #

Plusieurs transformations peuvent être appliquées à un élément dans une propriété comme ceci :

transform: rotate(15deg) translateX(200px);

Cela fera pivoter l’élément de 15 degrés dans le sens des aiguilles d’une montre, puis le translatera de 200 pixels vers la droite.

Dans les transformations chaînées, le système de coordonnées se déplace avec l’élément. Cela signifie que la translation ne sera pas horizontale mais sur un axe pivotera de 15 degrés dans le sens des aiguilles d’une montre comme indiqué dans l’image suivante :

[![Plusieurs transformations tournent puis traduisent][1]][1]

Changer l’ordre des transformations changera la sortie. Le premier exemple sera différent de

transform: translateX(200px) rotate(15deg);

<div class="transform"></div>

.transform {
  transform: rotate(15deg) translateX(200px);
}

Comme le montre cette image :

[![Plusieurs transformations traduisent puis tournent][2]][2]

[1] : http://i.stack.imgur.com/Ur0RG.jpg [2] : http://i.stack.imgur.com/QozeC.jpg

Traduire #

HTML

<div class="translate"></div>

CSS

.translate {
    width: 100px;
    height: 100px;
    background: teal;
    transform: translate(200px, 50%);
}

Cet exemple déplacera la div de 200px sur l’axe X et de 100px * 50% = 50px sur l’axe Y.

Vous pouvez également spécifier des translations sur un seul axe.

Sur l’axe X :

.translate {
    transform: translateX(200px);
}

Sur l’axe Y :

.translate {
    transform: translateY(50%);
}

Origine de la transformation #

Les transformations sont effectuées par rapport à un point défini par la propriété transform-origin.

La propriété prend 2 valeurs : transform-origin: X Y;

Dans l’exemple suivant, la première div (.tl) est tournée autour du coin supérieur gauche avec transform-origin: 0 0; et la seconde (.tr) est transformée autour de son coin supérieur droit avec transform- origine : 100 % 0. La rotation est appliquée au survol :

HTML :

<div class="transform originl"></div>
<div class="transform origin2"></div>

CSS :

.transform {
    display: inline-block;
    width: 200px;
    height: 100px;
    background: teal;
    transition: transform 1s;
}

.origin1 {
    transform-origin: 0 0;
}

.origin2 {
    transform-origin: 100% 0;
}

.transform:hover {
    transform: rotate(30deg);
}

La valeur par défaut de la propriété transform-origin est 50% 50% qui est le centre de l’élément.
