Utilisation de Flexbox #

HTML :

<div class="container">
  <img src="http://lorempixel.com/400/200" />
</div>

CSS :

html, body, .container {
  height: 100%;
}    
.container {
  display: flex;
  justify-content: center; /* horizontal center */
}
img {
  align-self: center; /* vertical center */
}

[Voir le résultat] (https://jsfiddle.net/aLu05kjy/1/)

HTML :

<img src="http://lorempixel.com/400/200" />

CSS :

html, body {
  height: 100%;
}   
body {
  display: flex;
  justify-content: center; /* horizontal center */
  align-items: center;     /* vertical center */
}

[Voir le résultat] (https://jsfiddle.net/ttp0bzfm/1/)

Voir [Centrage vertical et horizontal dynamique] (https://www.wikiod.com/fr/css/disposition-flexible-de-la-boite-flexbox#Centrage vertical et horizontal dynamique (aligner les éléments, justifier le contenu) #t=201607231920579121055) sous la documentation Flexbox pour plus de détails sur flexbox et la signification des styles.

Prise en charge du navigateur

Flexbox est pris en charge par tous les principaux navigateurs, à l’exception des versions d’IE antérieures à 10.

Certaines versions récentes de navigateur, telles que Safari 8 et IE10, nécessitent des préfixes de fournisseur.

Pour générer rapidement des préfixes, il existe Autoprefixer, un outil tiers.

Pour les navigateurs plus anciens (comme IE 8 et 9), un [Polyfill est disponible][2].

Pour un aperçu plus détaillé de la prise en charge du navigateur flexbox, voir cette réponse.

[1] : http://caniuse.com/#search=flex [2] : https://github.com/jonathantneal/flexibility

Utilisation de la transformation CSS #

[Les transformations CSS][1] sont basées sur la taille des éléments, donc si vous ne savez pas quelle est la hauteur ou la largeur de votre élément, vous pouvez le positionner absolument à 50 % du haut et de la gauche d’un conteneur relatif et le traduire par 50 % vers la gauche et vers le haut pour le centrer verticalement et horizontalement.

Gardez à l’esprit qu’avec cette technique, l’élément pourrait finir par être rendu à une limite de pixel non entière, ce qui le rendrait flou. Voir cette réponse dans SO pour une solution de contournement.

HTML

<div class="container">
  <div class="element"></div>
</div>

CSS

.container {
  position: relative;
}

.element {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

[Voir l’exemple dans JSFiddle][2]

COMPATIBILITÉ CROISÉE AVEC LES NAVIGATEURS #

La propriété transform a besoin de préfixes pour être prise en charge par les anciens navigateurs. Des préfixes sont nécessaires pour Chrome<=35, Safari<=8, Opera<=22, Android Browser<=4.4.4 et IE9. Les transformations CSS ne sont pas prises en charge par IE8 et les versions antérieures.

Voici une déclaration de transformation courante pour l’exemple précédent :

-webkit-transform: translate(-50%, -50%); /* Chrome, Safari, Opera, Android */
    -ms-transform: translate(-50%, -50%); /* IE 9 */
        transform: translate(-50%, -50%);

Pour plus d’informations, voir [canIuse][3].

PLUS D’INFORMATION #

  • L’élément est positionné en fonction du premier parent non statique (‘position : relative’, ‘absolue’ ou ‘fixe’). Explorez plus dans ce [violon][4] et ce [sujet de documentation][5].

  • Pour un centrage horizontal uniquement, utilisez left : 50 % et transform : translateX(-50%). Il en va de même pour le centrage vertical uniquement : centrer avec top : 50 % et transform : translateY(-50%).

  • L’utilisation d’éléments largeur/hauteur non statiques avec cette méthode de centrage peut faire apparaître l’élément centré écrasé. Cela se produit principalement avec des éléments contenant du texte, et peut être corrigé en ajoutant : margin-right: -50%; et margin-bottom: -50%;. Voir ce [violon] [6] pour plus d’informations.

[1] : https://www.wikiod.com/fr/css/transformations-2d [2] : https://jsfiddle.net/webtiki/rz3p3ufs/ [3] : http://caniuse.com/#feat=transforms2d [4] : https://jsfiddle.net/siavasfiroozbakht/ox8kyypa/ [5] : https://www.wikiod.com/fr/css/positionnement#Position absolue [6] : https://jsfiddle.net/4xxmxca0/

Utilisation de la marge : 0 auto ; #

Les objets peuvent être centrés en utilisant margin: 0 auto; s’ils sont des éléments de bloc et ont une largeur définie.

HTML

<div class="containerDiv">
    <div id="centeredDiv"></div>
</div>

<div class="containerDiv">
    <p id="centeredParagraph">This is a centered paragraph.</p>
</div>

<div class="containerDiv">
    <img id="centeredImage" src="https://i.kinja-img.com/gawker-media/image/upload/s--c7Q9b4Eh--/c_scale,fl_progressive,q_80,w_800/qqyvc3bkpyl3mfhr8all.jpg" />
</div>

CSS

.containerDiv {
    width: 100%;
    height: 100px;
    padding-bottom: 40px;
}

#centeredDiv {
    margin: 0 auto;
    width: 200px;
    height: 100px;
    border: 1px solid #000;
}

#centeredParagraph {
    width: 200px;
    margin: 0 auto;
}

#centeredImage {
    display: block;
    width: 200px;
    margin: 0 auto;
}

Exemple JSFiddle : [Centrage d’objets avec marge : 0 auto ;][2]

Utilisation de text-align #

Le type de centrage le plus courant et le plus simple est celui des lignes de texte dans un élément. CSS a la règle text-align: center à cette fin :

HTML

<p>Lorem ipsum</p>

CSS

p {
    text-align: center;
}

Cela ne fonctionne pas pour centrer des éléments de bloc entiers. text-align contrôle uniquement l’alignement du contenu en ligne comme le texte dans son élément de bloc parent.

En savoir plus sur text-align dans la section [Typographie][1].

[1] : https://www.wikiod.com/fr/css/typographie

Position d’utilisation : absolue #

Travailler dans les anciens navigateurs (IE >= 8)

Les marges automatiques, associées à des valeurs de zéro pour les décalages “gauche” et “droite” ou “haut” et “bas”, centreront un élément en position absolue dans son parent.

[Voir le résultat] (https://jsfiddle.net/stuttufu/sj2m0oo2/1/)

HTML

<div class="parent">
  <img class="center" src="http://lorempixel.com/400/200/" />
</div>

CSS

.parent {
  position: relative;
  height: 500px;
}

.center {
  position: absolute;
  margin: auto;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
}

Les éléments qui n’ont pas leur propre largeur et hauteur implicites comme les images, auront besoin de ces valeurs définies.

Autres ressources : [Absolute Centering in CSS][1]

[1] : http://codepen.io/shshaw/details/gEiDt

Utilisation de calc() #

La fonction calc() fait partie d’une nouvelle syntaxe en CSS3 dans laquelle vous pouvez calculer (mathématiquement) la taille/position occupée par votre élément en utilisant une variété de valeurs comme les pixels, les pourcentages, etc. Remarque :- Chaque fois que vous utilisez cette fonction , faites toujours attention à l’espace entre deux valeurs calc(100% - 80px).

CSS

.center {
    position: absolute;
    height: 50px;
    width: 50px;
    background: red;
    top: calc(50% - 50px / 2); /* height divided by 2*/
    left: calc(50% - 50px / 2); /* width divided by 2*/
}

HTML

<div class="center"></div>

Aligner verticalement n’importe quoi avec 3 lignes de code #

[Pris en charge par IE11+][1]

[Voir le résultat][2]

Utilisez ces 3 lignes pour aligner pratiquement tout à la verticale. Assurez-vous simplement que la div/image à laquelle vous appliquez le code a un parent avec une hauteur.

CSS

div.vertical {
  position: relative;
  top: 50%;
  transform: translateY(-50%);
}

HTML

<div class="vertical">Vertical aligned text!</div>

[1] : http://caniuse.com/#search=transform [2] : https://jsfiddle.net/bnqrLgk9/1/

Utilisation de la hauteur de ligne #

Vous pouvez également utiliser line-height pour centrer verticalement une seule ligne de texte à l’intérieur d’un conteneur :

CSS

div {
    height: 200px;
    line-height: 200px;
}

C’est assez moche, mais cela peut être utile à l’intérieur d’un élément <input />. La propriété line-height ne fonctionne que lorsque le texte à centrer s’étend sur une seule ligne. Si le texte s’étend sur plusieurs lignes, la sortie résultante ne sera pas centrée.

Technique de l’élément fantôme (hack de Michał Czernow) #

Cette technique fonctionne même lorsque les dimensions du conteneur sont inconnues.

Configurez un élément “fantôme” à l’intérieur du conteneur à centrer qui a une hauteur de 100 %, puis utilisez vertical-align: middle à la fois sur cet élément et sur l’élément à centrer.

CSS

/* This parent can be any width and height */
.block {
  text-align: center;

  /* May want to do this if there is risk the container may be narrower than the element inside */
  white-space: nowrap;
}
 
/* The ghost element */
.block:before {
  content: '';
  display: inline-block;
  height: 100%;
  vertical-align: middle;

  /* There is a gap between ghost element and .centered,
  caused by space character rendered. Could be eliminated by
  nudging .centered (nudge distance depends on font family),
  or by zeroing font-size in .parent and resetting it back
  (probably to 1rem) in .centered. */
  margin-right: -0.25em;
}

/* The element to be centered, can also be of any width and height */ 
.centered {
  display: inline-block;
  vertical-align: middle;
  width: 300px;
  white-space: normal; /* Resetting inherited nowrap behavior */
}

HTML

<div class="block">
  <div class="centered"></div>
</div>

Centrage par rapport à un autre élément #

Nous verrons comment centrer le contenu en fonction de la hauteur d’un élément proche.

Compatibilité : IE8+, tous les autres navigateurs modernes.

HTML

<div class="content">
  <div class="position-container">
    <div class="thumb">
      <img src="http://lorempixel.com/400/200/">
    </div>
    <div class="details">
      <p class="banner-title">text 1</p>
      <p class="banner-text">content content content content content content content content content content content content content content</p>
      <button class="btn">button</button>
    </div>
  </div>
</div>

CSS

.content * {
  box-sizing: border-box;
}
.content .position-container {
  display: table;
}
.content .details {
  display: table-cell;
  vertical-align: middle;
  width: 33.333333%;
  padding: 30px;
  font-size: 17px;
  text-align: center;
}
.content .thumb {
  width: 100%;
}
.content .thumb img {
  width: 100%;
}

Lien vers [JSFiddle][1]

Les points principaux sont les 3 conteneurs .thumb, .details et .position-container :

  • Le .position-container doit avoir display: table.
  • Le .details doit avoir la largeur réelle définie width: .... et display: table-cell, vertical-align: middle.
  • Le .thumb doit avoir width: 100% si vous voulez qu’il occupe tout l’espace restant et qu’il soit influencé par la largeur de .details.
  • L’image (si vous avez une image) à l’intérieur de .thumb devrait avoir width: 100%, mais ce n’est pas nécessaire si vous avez des proportions correctes.

[1] : https://jsfiddle.net/gasp10/6bv92mko/4/

Aligner verticalement une image à l’intérieur de div #

HTML

<div class="wrap">
    <img src="http://lorempixel.com/400/200/" />
</div>

CSS

.wrap {
    height: 50px;/* max image height */
    width: 100px;
    border: 1px solid blue;
    text-align: center;
}
.wrap:before {
  content:"";
  display: inline-block;
  height: 100%;
  vertical-align: middle;
  width: 1px;
}

img {
    vertical-align: middle;
}

Centrage vertical et horizontal sans se soucier de la hauteur ou de la largeur #

La technique suivante vous permet d’ajouter votre contenu à un élément HTML et de le centrer à la fois horizontalement et verticalement sans vous soucier de sa hauteur ou de sa largeur.

Le conteneur extérieur #

  • devrait avoir display: table;

Le conteneur intérieur #

  • devrait avoir display: table-cell;
  • devrait avoir vertical-align: middle;
  • devrait avoir text-align: center;

La zone de contenu #

  • devrait avoir display: inline-block;
  • devrait réajuster l’alignement horizontal du texte, par exemple. text-align: left; ou text-align: right;, sauf si vous souhaitez que le texte soit centré

Démo #

HTML

<div class="outer-container">
   <div class="inner-container">
     <div class="centered-content">
        You can put anything here!
     </div>
   </div>
</div>

CSS

body {
    margin : 0;
}

.outer-container {
    position : absolute;
    display: table;
    width: 100%; /* This could be ANY width */
    height: 100%; /* This could be ANY height */
    background: #ccc;
}

.inner-container {
    display: table-cell;
    vertical-align: middle;
    text-align: center;
}

.centered-content {
    display: inline-block;
    text-align: left;
    background: #fff;
    padding: 20px;
    border: 1px solid #000;
}

Voir aussi [ce violon][1] !

[1] : http://jsfiddle.net/WXLsY/621/

Centrage avec taille fixe #

Si la taille de votre contenu est fixe, vous pouvez utiliser le positionnement absolu à 50 % avec une “marge” qui réduit de moitié la largeur et la hauteur de votre contenu :

HTML

<div class="center">
    Center vertically and horizontally
</div>

CSS

.center {
    position: absolute;
    background: #ccc;

    left: 50%;
    width: 150px;
    margin-left: -75px;  /* width * -0.5 */

    top: 50%;
    height: 200px;
    margin-top: -100px;  /* height * -0.5 */
}

Centrage horizontal avec uniquement une largeur fixe #

Vous pouvez centrer l’élément horizontalement même si vous ne connaissez pas la hauteur du contenu :

HTML

<div class="center">
    Center only horizontally
</div>

CSS

.center {
    position: absolute;
    background: #ccc;

    left: 50%;
    width: 150px;
    margin-left: -75px;  /* width * -0.5 */
}

Centrage vertical à hauteur fixe #

Vous pouvez centrer l’élément verticalement si vous connaissez sa hauteur :

HTML

<div class="center">
    Center only vertically
</div>

CSS

.center {
    position: absolute;
    background: #ccc;

    top: 50%;
    height: 200px;
    margin-top: -100px;  /* width * -0.5 */
}

Aligner verticalement les éléments de hauteur dynamiques #

L’application intuitive de CSS ne produit pas les résultats souhaités car

  • vertical-align:middle [n’est pas applicable aux éléments de niveau bloc][1]
  • margin-top:auto et margin-bottom:auto [valeurs utilisées calculerait comme zéro][2]
  • margin-top:-50% [les valeurs de marge basées sur le pourcentage sont calculé par rapport à la largeur du bloc contenant ][3]

Pour une prise en charge plus large des navigateurs, une solution de contournement avec des éléments d’assistance :

HTML

<div class="vcenter--container">
  <div class="vcenter--helper">
    <div class="vcenter--content">
      <!--stuff-->
    </div>
  </div>
</div>

CSS

.vcenter--container {
  display: table;
  height: 100%;
  position: absolute;
  overflow: hidden;
  width: 100%;
}
.vcenter--helper {
  display: table-cell;
  vertical-align: middle;
}
.vcenter--content {
  margin: 0 auto;
  width: 200px;
}

Centrage horizontal et vertical à l’aide de la disposition du tableau #

On pourrait facilement centrer un élément enfant en utilisant la propriété d’affichage table.

HTML

<div class="wrapper">
    <div class="parent">
        <div class="child"></div>
    </div>
</div>

CSS

.wrapper { 
    display: table; 
    vertical-align: center; 
    width: 200px; 
    height: 200px; 
    background-color: #9e9e9e; 
}
.parent { 
    display: table-cell; 
    vertical-align: middle; 
    text-align: center; 
}
.child { 
    display: inline-block; 
    vertical-align: middle; 
    text-align: center; 
    width: 100px; 
    height: 100px; 
    background-color: teal; 
}
