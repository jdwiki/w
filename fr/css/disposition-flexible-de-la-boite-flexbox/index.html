Le module Flexible Box, ou simplement “flexbox” en abrégé, est un modèle de boîte conçu pour les interfaces utilisateur, et il permet aux utilisateurs d’aligner et de répartir l’espace entre les éléments d’un conteneur de sorte que les éléments se comportent de manière prévisible lorsque la mise en page doit s’adapter à des éléments différents, inconnus. tailles d’écran. Un conteneur flexible agrandit les éléments pour remplir l’espace disponible et les rétrécit pour éviter tout débordement.

Syntaxe #

  • affichage : flexible ;
  • flex-direction : rangée | ligne-inverse | colonne | colonne-inverse ;
  • flex-wrap : nowrap | envelopper | wrap-inverse;
  • flux flexible : <‘flex-direction’> || <‘flex-wrap’>
  • justifier-contenu : flex-start | extrémité flexible | centre | espace-entre | espace-autour ;
  • aligner les éléments : flex-start | extrémité flexible | centre | ligne de base | extensible;
  • aligner le contenu : flex-start | extrémité flexible | centre | espace-entre | espace-autour | extensible;
  • ordre : <entier> ;
  • flex-croissance : <nombre> ; /* par défaut 0 */
  • flex-shrink : <nombre> ; /* par défaut 1 */
  • flex-basis : <longueur> | auto; /* automatique par défaut */
  • flexible : aucun | [ <‘flex-grow’> <‘flex-shrink’> ? || <‘flex-base’> ]
  • align-self : auto | flex-start | extrémité flexible | centre | ligne de base | extensible;

préfixes de vendeur #

  • affichage : -webkit-box ; /* Chrome <20 */
  • affichage : -webkit-flex ; /* Chrome 20+ */
  • affichage : -moz-box ; /* Firefox */
  • affichage : -ms-flexbox ; /* C’EST À DIRE */
  • affichage : flexible ; /* Navigateurs modernes */

Centrage vertical et horizontal dynamique (align-items, justifier-content) #

Exemple simple (centrage d’un seul élément) #

HTML #

<div class="aligner">
  <div class="aligner-item">…</div>
</div>

##CSS

.aligner {
  display: flex; 
  align-items: center;
  justify-content: center;
}

.aligner-item {
  max-width: 50%; /*for demo. Use actual width instead.*/
}

Voici une démo.

Raisonnement #

Propriété Valeur Descriptif
aligner-items centre Cela centre les éléments le long d’un axe autre que celui spécifié par flex-direction, c’est-à-dire un centrage vertical pour une boîte flexible horizontale et un centrage horizontal pour une boîte flexible verticale.
justifier-contenu centre Cela centre les éléments le long de l’axe spécifié par flex-direction. C’est-à-dire, pour une flexbox horizontale (flex-direction: row), c’est centré horizontalement, et pour une flexbox verticale (flex-direction: column), c’est centré verticalement)

Pied de page collant à hauteur variable #

Ce code crée un pied de page collant. Lorsque le contenu n’atteint pas la fin de la fenêtre, le pied de page reste collé au bas de la fenêtre. Lorsque le contenu s’étend au-delà du bas de la fenêtre, le pied de page est également poussé hors de la fenêtre. Voir le résultat

HTML :

<div class="header">
  <h2>Header</h2>
</div>

<div class="content">
  <h1>Content</h1>
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio. Praesent libero. Sed cursus ante dapibus diam. Sed nisi. Nulla quis sem at nibh elementum imperdiet. Duis sagittis ipsum. Praesent mauris. Fusce nec tellus sed augue semper porta. Mauris massa. Vestibulum lacinia arcu eget nulla. Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos. Curabitur sodales ligula in libero. </p>
</div>

<div class="footer">
  <h4>Footer</h4>
</div>

CSS :

html, body {
  height: 100%;
}

body {
  display: flex;
  flex-direction: column;
}

.content {
  /* Include `0 auto` for best browser compatibility. */
  flex: 1 0 auto;
}

.header, .footer {
  background-color: grey;
  color: white;
  flex: none;
}

Ajustez les éléments de manière optimale à leur conteneur #

L’une des plus belles fonctionnalités de flexbox est de permettre aux conteneurs de s’adapter de manière optimale à leur élément parent.

[Démo en direct][1].

HTML :

<div class="flex-container">
  <div class="flex-item">1</div>
  <div class="flex-item">2</div>
  <div class="flex-item">3</div>
  <div class="flex-item">4</div>
  <div class="flex-item">5</div>
</div>

CSS :

.flex-container {
    background-color: #000;
    height: 100%;
    display:flex;
    flex-direction: row;
    flex-wrap: wrap;
    justify-content: flex-start;
    align-content: stretch;
    align-items: stretch;
}

.flex-item {
    background-color: #ccf;
    margin: 0.1em;
    flex-grow: 1;
    flex-shrink: 0;
    flex-basis: 200px; /* or % could be used to ensure a specific layout */
}

Résultat:

Les colonnes s’adaptent à mesure que l’écran est redimensionné.

Mise en page du Saint Graal avec Flexbox #

[Mise en page Holy Grail][1] est une mise en page avec un en-tête et un pied de page à hauteur fixe, et un centre avec 3 colonnes. Les 3 colonnes incluent un sidenav à largeur fixe, un centre fluide et une colonne pour d’autres contenus comme les publicités (le centre fluide apparaît en premier dans le balisage). CSS Flexbox peut être utilisé pour y parvenir avec un balisage très simple :

Balisage HTML :

<div class="container">
  <header class="header">Header</header>
  <div class="content-body">
    <main class="content">Content</main>
    <nav class="sidenav">Nav</nav>
    <aside class="ads">Ads</aside>
  </div>
  <footer class="footer">Footer</footer>
</div>

CSS :

body {
  margin: 0;
  padding: 0;
}

.container {
  display: flex;
  flex-direction: column;
  height: 100vh;
}

.header {
  flex: 0 0 50px;
}

.content-body {
  flex: 1 1 auto;
  
  display: flex;
  flex-direction: row;
}

.content-body .content {
  flex: 1 1 auto;
  overflow: auto;
}

.content-body .sidenav {
  order: -1;
  flex: 0 0 100px;
  overflow: auto;
}

.content-body .ads {
  flex: 0 0 100px;
  overflow: auto;
}

.footer {
  flex: 0 0 50px;
}

[Démo][2]

[1] : http://alistapart.com/article/holygrail [2] : https://jsfiddle.net/adityarb88/hek6ms0x/

Boutons parfaitement alignés à l’intérieur des cartes avec flexbox #

C’est un modèle régulier dans la conception de nos jours pour aligner verticalement call to actions à l’intérieur de ses cartes contenantes comme ceci :

[![Ce que nous essayons de réaliser][1]][1]

Ceci peut être réalisé en utilisant une astuce spéciale avec flexbox

HTML

<div class="cards">
  <div class="card">
    <p>Lorem ipsum Magna proident ex anim dolor ullamco pariatur reprehenderit culpa esse enim mollit labore dolore voluptate ullamco et ut sed qui minim.</p>
    <p><button>Action</button></p>
  </div>
  <div class="card">
    <p>Lorem ipsum Magna proident ex anim dolor ullamco pariatur reprehenderit culpa esse enim mollit labore dolore voluptate ullamco et ut sed qui minim.</p>
    <p>Lorem ipsum Magna proident ex anim dolor ullamco pariatur reprehenderit culpa esse enim mollit labore dolore voluptate ullamco et ut sed qui minim.</p>
    <p>Lorem ipsum Magna proident ex anim dolor ullamco pariatur reprehenderit culpa esse enim mollit labore dolore voluptate ullamco et ut sed qui minim.</p>
    <p>Lorem ipsum Magna proident ex anim dolor ullamco pariatur reprehenderit culpa esse enim mollit labore dolore voluptate ullamco et ut sed qui minim.</p>
    <p><button>Action</button></p>
  </div>
</div>

Tout d’abord, nous utilisons CSS pour appliquer display: flex; au conteneur. Cela créera 2 colonnes de hauteur égale avec le contenu coulant naturellement à l’intérieur

CSS

.cards {
    display: flex;
}
.card {
    border: 1px solid #ccc;
    margin: 10px 10px;
    padding: 0 20px;
}
button {
    height: 40px;
    background: #fff;
    padding: 0 40px;
    border: 1px solid #000;
}
p:last-child {
    text-align: center;
}

La mise en page changera et deviendra comme ceci :

[![Ce que nous essayons de réaliser][2]][2]

Afin de déplacer les boutons vers le bas du bloc, nous devons appliquer display: flex; à la carte elle-même avec la direction définie sur column. Après cela, nous devons sélectionner le dernier élément à l’intérieur de la carte et définir le margin-top sur auto. Cela poussera le dernier paragraphe vers le bas de la carte et obtiendra le résultat requis.

CSS final :

.cards {
    display: flex;
}
.card {
    border: 1px solid #ccc;
    margin: 10px 10px;
    padding: 0 20px;
    display: flex;
    flex-direction: column;
}
button {
    height: 40px;
    background: #fff;
    padding: 0 40px;
    border: 1px solid #000;
}
p:last-child {
    text-align: center;
    margin-top: auto;
}

Même hauteur sur les conteneurs imbriqués #

Ce code garantit que tous les conteneurs imbriqués ont toujours la même hauteur. Cela se fait en s’assurant que tous les éléments imbriqués ont la même hauteur que la div parente contenante. [Voir exemple de travail][1] : https://jsfiddle.net/3wwh7ewp/

Cet effet est obtenu grâce à la propriété align-items définie sur stretch par défaut.

HTML

<div class="container">
  <div style="background-color: red">
    Some <br />
    data <br />
    to make<br />
    a height <br />
 </div>
  <div style="background-color: blue">
    Fewer <br />
    lines <br />
 </div>
</div>

CSS

.container {
    display: flex;
    align-items: stretch; // Default value
}

Remarque : [Ne fonctionne pas sur les versions d’IE inférieures à 10][2]
