Syntaxe #

  • curseur : automatique | par défaut | aucun | menu contextuel | aide | pointeur | progrès | attendre | cellule | réticule | texte | texte-vertical | alias | copier | déplacer | sans goutte | non-permis | e-redimensionner | n-redimensionner | ne-redimensionner | nw-redimensionner | s-redimensionner | se-redimensionner | sw-redimensionner | w-redimensionner | ew-redimensionner | ns-redimensionner | nouveau-redimensionner | nwse-resize | col-redimensionner | redimensionner la ligne | tout défilement | zoomer | dézoomer | saisir | saisir;

Modification du type de curseur #

cursor: value;

[![visualisation][1]][1]

Exemples:

Valeur Descriptif
aucun Aucun curseur n’est rendu pour l’élément
automatique Défaut. Le navigateur définit un curseur
aide Le curseur indique que l’aide est disponible
attendre Le curseur indique que le programme est occupé
déplacer Le curseur indique que quelque chose doit être déplacé
pointeur Le curseur est un pointeur et indique un lien

[1] : http://i.stack.imgur.com/E76ws.png

événements de pointeur #

La propriété pointer-events permet de contrôler la façon dont les éléments HTML répondent aux événements de souris/toucher.

.disabled {
  pointer-events: none;
}

Dans cet exemple,

‘aucun’ empêche toutes les options de clic, d’état et de curseur sur le code HTML spécifié élément [[1]]

Les autres valeurs valides pour les éléments HTML sont :

  • automatique ;
  • hériter.
  1. https://css-tricks.com/almanac/properties/p/pointer-events/

Autres ressources:

  • https://davidwalsh.name/pointer-events

caret-color #

La propriété CSS caret-color spécifie la couleur du caret, l’indicateur visible du point d’insertion dans un élément où du texte et d’autres contenus sont insérés par la saisie ou l’édition de l’utilisateur.

HTML

<input id="example" />

CSS

#example {
  caret-color: red;
}

Ressources:
