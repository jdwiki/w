Paramètres #

Paramètre Détails
style de police italique ou oblique
variante de police normal ou petites majuscules
font-weight normal, gras ou numérique de 100 à 900.
taille-police La taille de la police indiquée en %, px, em ou toute autre mesure CSS valide
hauteur-ligne La hauteur de ligne donnée en %, px, em ou toute autre mesure CSS valide
font-family C’est pour définir le nom de la famille.
couleur Toute [représentation de couleur CSS] valide (https://www.wikiod.com/fr/css/couleurs), comme red, #00FF00, hsl(240, 100%, 50%) etc.
font-stretch S’il faut ou non utiliser un visage confenced ou développé à partir de la police. Les valeurs valides sont normal, ultra-condensed, extra-condensed, condensed, semi-condensed, semi-expanded, expanded, extra-expanded or ultra-expanded
texte-aligner start, end, left, right, center, justify, match-parent
texte-décoration none, underline, overline, line-through, initial, inherit ;
  • La propriété text-shadow n’est pas prise en charge par les versions d’Internet Explorer inférieures à 10.

Le raccourci de la police #

Avec la syntaxe :

element {
    font: [font-style] [font-variant] [font-weight] [font-size/line-height] [font-family];
}

Vous pouvez avoir tous vos styles liés aux polices dans une seule déclaration avec le raccourci font. Utilisez simplement la propriété font et mettez vos valeurs dans le bon ordre.

Par exemple, pour mettre tous les éléments p en gras avec une taille de police de 20 pixels et en utilisant Arial comme famille de polices, vous le coderiez généralement comme suit :

p {
    font-weight: bold;
    font-size: 20px;
    font-family: Arial, sans-serif;
}

Cependant, avec le raccourci de police, il peut être condensé comme suit :

p {
    font: bold 20px Arial, sans-serif;
}

Remarque : puisque font-style, font-variant, font-weight et line-height sont facultatifs, les trois sont ignorés dans cet exemple. Il est important de noter que l’utilisation du raccourci réinitialise les autres attributs non renseignés. Un autre point important est que les deux attributs nécessaires pour que le raccourci de police fonctionne sont font-size et font-family. S’ils ne sont pas tous les deux inclus, le raccourci est ignoré.

Valeur initiale pour chacune des propriétés :

  • style de police : normal ;
  • variante de police : normale ;
  • poids de la police : normal ;
  • font-stretch: normal;
  • taille de police : moyenne ;
  • hauteur de ligne : normal ;
  • font-family - dépend de l’agent utilisateur

Devis #

La propriété quotes est utilisée pour personnaliser les guillemets ouvrants et fermants de la balise <q>.

q {
    quotes: "«" "»";
}

Taille de police #

HTML :

<div id="element-one">Hello I am some text.</div>
<div id="element-two">Hello I am some smaller text.</div>

CSS :

#element-one {
    font-size: 30px;   
}

#element-two {
    font-size: 10px;
}

Le texte à l’intérieur de #element-one aura une taille de 30px, tandis que le texte dans #element-two aura une taille de 10px.

Sens du texte #

div {
    direction: ltr; /* Default, text read read from left-to-right */
}
.ex {
    direction: rtl; /* text read from right-to-left */
}
.horizontal-tb {
    writing-mode: horizontal-tb; /* Default, text read from left-to-right and top-to-bottom. */
}
.vertical-rtl {
    writing-mode: vertical-rl; /* text read from right-to-left and top-to-bottom */
}
.vertical-ltr {
    writing-mode: vertical-rl; /* text read from left-to-right and top to bottom */
}

La propriété direction est utilisée pour changer la direction horizontale du texte d’un élément.

Syntaxe: direction : litre | RTL | initiale | hériter;

La propriété writing-mode modifie l’alignement du texte afin qu’il puisse être lu de haut en bas ou de gauche à droite, selon la langue.

Syntaxe: direction : horizontal-tb | vertical-rl | vertical-lr;

Piles de polices #

font-family: 'Segoe UI', Tahoma, sans-serif;

Le navigateur tentera d’appliquer la police “Segoe UI” aux caractères des éléments ciblés par la propriété ci-dessus. Si cette police n’est pas disponible ou si la police ne contient pas de glyphe pour le caractère requis, le navigateur se rabattra sur Tahoma et, si nécessaire, sur n’importe quelle police sans empattement sur l’ordinateur de l’utilisateur. Notez que tous les noms de police contenant plus d’un mot, tels que “Segoe UI”, doivent être entourés de guillemets simples ou doubles.

font-family: Consolas, 'Courier New', monospace;

Le navigateur tentera d’appliquer la police “Consolas” aux caractères des éléments ciblés par la propriété ci-dessus. Si cette police n’est pas disponible ou si la police ne contient pas de glyphe pour le caractère requis, le navigateur reviendra à “Courier New” et, si nécessaire, à toute police à espacement fixe sur l’ordinateur de l’utilisateur.

Transformation de texte #

La propriété text-transform vous permet de changer la capitalisation du texte. Les valeurs valides sont : uppercase, capitalize, lowercase, initial, inherit et none

CSS :

.example1 {
    text-transform: uppercase;
}
.example2 {
    text-transform: capitalize;
}
.example3 {
    text-transform: lowercase;
}

HTML

<p class="example1">
    all letters in uppercase  <!-- "ALL LETTERS IN UPPERCASE" -->
</p>
<p class="example2">
    all letters in capitalize <!-- "All Letters In Capitalize (Sentence Case)" -->
</p>
<p class="example3">
    all letters in lowercase  <!-- "all letters in lowercase" -->
</p>

Débordement de texte #

La propriété text-overflow traite de la façon dont le contenu débordé doit être signalé aux utilisateurs. Dans cet exemple, les points de suspension représentent du texte coupé.

.text {
  overflow: hidden;
  text-overflow: ellipsis;
}

Malheureusement, text-overflow: ellipsis ne fonctionne que sur une seule ligne de texte. Il n’y a aucun moyen de prendre en charge les points de suspension sur la dernière ligne dans le CSS standard, mais cela peut être réalisé avec une implémentation non standard de Webkit uniquement de flexboxes.

.giveMeEllipsis {
   overflow: hidden;
   text-overflow: ellipsis;
   display: -webkit-box;
   -webkit-box-orient: vertical;
   -webkit-line-clamp: N; /* number of lines to show */
   line-height: X;        /* fallback */
   max-height: X*N;       /* fallback */
}

Exemple (ouvert dans Chrome ou Safari) :

http://jsfiddle.net/csYjC/1131/

Ressources:

https://www.w3.org/TR/2012/WD-css3-ui-20120117/#text-overflow0

Ombre de texte #

Pour ajouter des ombres au texte, utilisez la propriété text-shadow. La syntaxe est la suivante :

text-shadow: horizontal-offset vertical-offset blur color;

Ombre sans rayon de flou #

h1 {
  text-shadow: 2px 2px #0000FF;
}

Cela crée un effet d’ombre bleue autour d’un titre

Ombre avec rayon de flou #

Pour ajouter un effet de flou, ajoutez un argument d’option “rayon de flou”

h1 {
  text-shadow: 2px 2px 10px #0000FF;
}

Ombres multiples #

Pour donner à un élément plusieurs ombres, séparez-les par des virgules

h1 {
    text-shadow: 0 0 3px #FF0000, 0 0 5px #0000FF;
}

L’espacement des lettres #

h2 {
  /* adds a 1px space horizontally between each letter;
     also known as tracking */
  letter-spacing: 1px;
}

La propriété letter-spacing est utilisée pour spécifier l’espace entre les caractères dans un texte.

! letter-spacing prend également en charge les valeurs négatives :

p {
  letter-spacing: -1px;
}

Ressources : https://developer.mozilla.org/en-US/docs/Web/CSS/letter-spacing

Retrait du texte #

p {
    text-indent: 50px;
}

La propriété text-indent spécifie de combien d’espacement horizontal le texte doit être déplacé avant le début de la première ligne du contenu textuel d’un élément.

Ressources:

  • http://stackoverflow.com/questions/5856952/indenting-only-the-first-line-of-text-in-a-paragraph
  • https://www.w3.org/TR/CSS21/text.html#propdef-text-indent
  • https://developer.mozilla.org/en-US/docs/Web/CSS/text-indent

Décoration de texte #

La propriété text-decoration est utilisée pour définir ou supprimer les décorations du texte.

h1 { text-decoration: none; }
h2 { text-decoration: overline; }
h3 { text-decoration: line-through; }
h4 { text-decoration: underline; }

text-decoration peut être utilisé en combinaison avec text-decoration-style et text-decoration-color comme propriété abrégée :

.title { text-decoration: underline dotted blue; }

Il s’agit d’une version abrégée de

.title {
  text-decoration-style: dotted;
  text-decoration-line: underline;
  text-decoration-color: blue;
}

Il convient de noter que les propriétés suivantes ne sont prises en charge que dans Firefox

  • texte-décoration-couleur
  • texte-décoration-ligne
  • style de décoration de texte
  • texte-décoration-sauter

Espacement des mots #

La propriété word-spacing spécifie le comportement d’espacement entre les balises et les mots.

Valeurs possibles

  • une longueur positive ou négative (en utilisant em px vh cm etc.) ou un pourcentage (en utilisant %)
  • le mot-clé normal utilise l’espacement des mots par défaut de la police
  • le mot clé inherit prend la valeur de l’élément parent

CSS

.normal    { word-spacing: normal; }
.narrow    { word-spacing: -3px; }
.extensive { word-spacing: 10px; }

HTML

<p>
  <span class="normal">This is an example, showing the effect of "word-spacing".</span><br>
  <span class="narrow">This is an example, showing the effect of "word-spacing".</span><br>
  <span class="extensive">This is an example, showing the effect of "word-spacing".</span><br>
</p>

Variante de police #

Les attributs:

Ordinaire

Attribut par défaut des polices.

en minuscule

Définit chaque lettre en majuscule, ** mais ** rend les lettres minuscules (du texte original) plus petites que les lettres majuscules à l’origine.

CSS :

.smallcaps{
   font-variant: small-caps;
}

HTML :

<p class="smallcaps">
    Documentation about CSS Fonts
    <br>
    aNd ExAmpLe
</p>
