Syntaxe #

  • clair : aucun | gauche | droite | les deux | démarrage en ligne | fin de ligne ;
  • flottant : gauche | droite | aucun | démarrage en ligne | fin de ligne ;

Comme float implique l’utilisation du block layout, il modifie le valeur calculée des valeurs d’affichage dans certains cas [1]

[1] : https://developer.mozilla.org/en-US/docs/Web/CSS/float MDN

Faire flotter une image dans le texte #

L’utilisation la plus basique d’un flotteur est d’avoir du texte autour d’une image. Le code ci-dessous produira deux paragraphes et une image, le deuxième paragraphe circulant autour de l’image. Notez qu’il est toujours contenu après l’élément flottant qui entoure l’élément flottant.

HTML :

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio. Praesent libero. Sed cursus ante dapibus diam. Sed nisi. Nulla quis sem at nibh elementum imperdiet. Duis sagittis ipsum. Praesent mauris. Fusce nec tellus sed augue semper porta. Mauris massa. Vestibulum lacinia arcu eget nulla. </p>

<img src="http://lorempixel.com/200/100/" />

<p>Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos. Curabitur sodales ligula in libero. Sed dignissim lacinia nunc. Curabitur tortor. Pellentesque nibh. Aenean quam. In scelerisque sem at dolor. Maecenas mattis. Sed convallis tristique sem. Proin ut ligula vel nunc egestas porttitor. Morbi lectus risus, iaculis vel, suscipit quis, luctus non, massa. Fusce ac turpis quis ligula lacinia aliquet. </p>

CSS :

img {
  float:left;
  margin-right:1rem;
}

propriété claire #

La propriété clear est directement liée aux flottants. Valeurs de propriété :

  • aucun - Par défaut. Permet des éléments flottants des deux côtés

  • gauche - Aucun élément flottant n’est autorisé sur le côté gauche

  • droite - Aucun élément flottant n’est autorisé sur le côté droit

  • les deux - Aucun élément flottant n’est autorisé sur le côté gauche ou droit

  • initial - Définit cette propriété à sa valeur par défaut. En savoir plus sur l’initiale

  • inherit - Hérite cette propriété de son élément parent. En savoir plus sur l’héritage

Clearfix #

Le hack clearfix est un moyen populaire de contenir les flottants (N. Gallagher aka @necolas)

À ne pas confondre avec la propriété [clear](https://www.wikiod.com/fr/css/flotteurs#propriété claire), clearfix est un concept (qui est également lié aux flottants, d’où la confusion possible). Pour contenir des flottants, vous devez ajouter la classe .cf ou .clearfix sur le conteneur (le parent) et styliser cette classe avec quelques règles décrites ci-dessous.

3 versions avec des effets légèrement différents (sources :A new micro clearfix hack par N. Gallagher et [clearfix reloaded](http://yuiblog.com/ blog/2010/09/27/clearfix-reloaded-overflowhidden-demystified/) par T. J. Koblentz) :

Clearfix (avec l’effondrement de la marge supérieure des flottants contenus toujours en cours) #

.cf:after {
    content: "";
    display: table;
}

.cf:after {
    clear: both;
}

Clearfix empêche également l’effondrement de la marge supérieure des flottants contenus #

/**
 * For modern browsers
 * 1. The space content is one way to avoid an Opera bug when the
 *    contenteditable attribute is included anywhere else in the document.
 *    Otherwise it causes space to appear at the top and bottom of elements
 *    that are clearfixed.
 * 2. The use of `table` rather than `block` is only necessary if using
 *    `:before` to contain the top-margins of child elements.
 */
.cf:before,
.cf:after {
    content: " "; /* 1 */
    display: table; /* 2 */
}

.cf:after {
    clear: both;
}

Clearfix avec prise en charge des navigateurs obsolètes IE6 et IE7 #

.cf:before,
.cf:after {
    content: " ";
    display: table;
}

.cf:after {
    clear: both;
}

/**
 * For IE 6/7 only
 * Include this rule to trigger hasLayout and contain floats.
 */
.cf {
    *zoom: 1;
}

DIV en ligne utilisant le flotteur #

Le div est un élément de niveau bloc, c’est-à-dire qu’il occupe toute la largeur de la page et que les frères et sœurs sont placés l’un en dessous de l’autre quelle que soit leur largeur.

<div>
    <p>This is DIV 1</p>
</div>
<div>
    <p>This is DIV 2</p>
</div>

La sortie du code suivant sera [![entrez la description de l’image ici][1]][1]

Nous pouvons les mettre en ligne en ajoutant une propriété CSS float à la div.

HTML :

<div class="outer-div">
    <div class="inner-div1">
        <p>This is DIV 1</p>
    </div>
    <div class="inner-div2">
        <p>This is DIV 2</p>
    </div>
</div>

CSS

.inner-div1 {
    width: 50%;
    margin-right:0px;
    float:left;
    background : #337ab7;
    padding:50px 0px;
}
 
.inner-div2 {
    width: 50%;
    margin-right:0px;
    float:left;
    background : #dd2c00;
    padding:50px 0px;
}
 
p {
    text-align:center;
}

Disposition simple à deux colonnes à largeur fixe #

Une mise en page simple à deux colonnes se compose de deux éléments flottants à largeur fixe. Notez que la barre latérale et la zone de contenu n’ont pas la même hauteur dans cet exemple. C’est l’une des parties délicates avec des dispositions multi-colonnes utilisant des flottants, et nécessite des solutions de contournement pour que plusieurs colonnes semblent avoir la même hauteur.

HTML :

<div class="wrapper">

<div class="sidebar">
  <h2>Sidebar</h2>

  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio.</p>
</div>

<div class="content">
  <h1>Content</h1>

  <p>Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos. Curabitur sodales ligula in libero. Sed dignissim lacinia nunc. Curabitur tortor. Pellentesque nibh. Aenean quam. In scelerisque sem at dolor. Maecenas mattis. Sed convallis tristique sem. Proin ut ligula vel nunc egestas porttitor. Morbi lectus risus, iaculis vel, suscipit quis, luctus non, massa. Fusce ac turpis quis ligula lacinia aliquet. </p>
</div>

</div>

CSS :

.wrapper {
  width:600px;
  padding:20px;
  background-color:pink;

  /* Floated elements don't use any height. Adding "overflow:hidden;" forces the
     parent element to expand to contain its floated children. */
  overflow:hidden;
}

.sidebar {
  width:150px;
  float:left;
  background-color:blue;
}

.content {
  width:450px;
  float:right;
  background-color:yellow;
}

Disposition simple à trois colonnes à largeur fixe #

HTML :

<div class="wrapper">
  <div class="left-sidebar">
    <h1>Left Sidebar</h1>
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. </p>
  </div>
  <div class="content">
    <h1>Content</h1>
    <p>Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos. Curabitur sodales ligula in libero. Sed dignissim lacinia nunc. Curabitur tortor. Pellentesque nibh. Aenean quam. In scelerisque sem at dolor. Maecenas mattis. Sed convallis tristique sem. Proin ut ligula vel nunc egestas porttitor. Morbi lectus risus, iaculis vel, suscipit quis, luctus non, massa. </p>
  </div>
  <div class="right-sidebar">
    <h1>Right Sidebar</h1>
    <p>Fusce ac turpis quis ligula lacinia aliquet.</p>
  </div>
</div>

CSS :

.wrapper {
  width:600px;
  background-color:pink;
  padding:20px;

  /* Floated elements don't use any height. Adding "overflow:hidden;" forces the
     parent element to expand to contain its floated children. */
  overflow:hidden;
}

.left-sidebar {
  width:150px;
  background-color:blue;
  float:left;
}

.content {
  width:300px;
  background-color:yellow;
  float:left;
}

.right-sidebar {
  width:150px;
  background-color:green;
  float:right;
}

Mise en page paresseuse/gourmande à deux colonnes #

Cette disposition utilise une colonne flottante pour créer une disposition à deux colonnes sans largeurs définies. Dans cet exemple, la barre latérale gauche est “paresseuse”, en ce sens qu’elle ne prend que l’espace dont elle a besoin. Une autre façon de dire cela est que la barre latérale gauche est “rétractable”. La colonne de contenu de droite est “gourmande”, en ce sens qu’elle occupe tout l’espace restant.

HTML :

<div class="sidebar">
<h1>Sidebar</h1>
<img src="http://lorempixel.com/150/200/" />
</div>

<div class="content">
<h1>Content</h1>
<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio. Praesent libero. Sed cursus ante dapibus diam. Sed nisi. Nulla quis sem at nibh elementum imperdiet. Duis sagittis ipsum. Praesent mauris. Fusce nec tellus sed augue semper porta. Mauris massa. Vestibulum lacinia arcu eget nulla. </p>
<p>Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos. Curabitur sodales ligula in libero. Sed dignissim lacinia nunc. Curabitur tortor. Pellentesque nibh. Aenean quam. In scelerisque sem at dolor. Maecenas mattis. Sed convallis tristique sem. Proin ut ligula vel nunc egestas porttitor. Morbi lectus risus, iaculis vel, suscipit quis, luctus non, massa. Fusce ac turpis quis ligula lacinia aliquet. Mauris ipsum. Nulla metus metus, ullamcorper vel, tincidunt sed, euismod in, nibh. </p>
</div>

CSS :

.sidebar {
  /* `display:table;` shrink-wraps the column */
  display:table;
  float:left;
  background-color:blue;
}

.content {
  /* `overflow:hidden;` prevents `.content` from flowing under `.sidebar` */
  overflow:hidden;
  background-color:yellow;
}

Violon

Utilisation de la propriété overflow pour effacer les flottants #

Définir la valeur overflow sur hidden, auto ou scroll sur un élément effacera tous les flottants de cet élément.

Remarque : l’utilisation de overflow:scroll affichera toujours la boîte de défilement
