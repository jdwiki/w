Syntaxe #

  • forme extérieure : aucune | [ <forme_de_base> || <boîte de forme> ] | <image>

  • marge de forme : <longueur> | <pourcentage>

  • shape-image-threshold : <nombre>

Paramètres #

Paramètre Détails
aucun Une valeur de none signifie que la zone flottante (la zone utilisée pour envelopper le contenu autour d’un élément flottant) n’est pas affectée. Il s’agit de la valeur par défaut/initiale.
forme de base Fait référence à un élément parmi inset(), circle(), ellipse() ou polygon(). En utilisant l’une de ces fonctions et ses valeurs, la forme est définie.
boîte-forme Fait référence à l’un parmi margin-box, border-box, padding-box, content-box. Lorsque seule <shape-box> est fournie (sans <basic-shape>), cette boîte est la forme. Lorsqu’il est utilisé avec <basic-shape>, il agit comme la zone de référence.
image Lorsqu’une image est fournie comme valeur, la forme est calculée en fonction du canal alpha de l’image spécifiée.

La prise en charge du navigateur pour le module CSS Shapes est très limitée pour le moment.

Il est pris en charge dans Chrome v37+ et Opera 24+ sans préfixes de navigateur/fournisseur. Safari le prend en charge à partir de la v7.1+ mais avec le préfixe -webkit-.

Il n’est pas encore pris en charge dans IE, Edge et Firefox.

Forme à l’extérieur avec la forme de base - cercle () #

Avec la propriété CSS shape-outside, on peut définir des valeurs de forme pour la zone flottante afin que le contenu en ligne s’enroule autour de la forme au lieu de la boîte du flottant.

CSS

img:nth-of-type(1) {
  shape-outside: circle(80px at 50% 50%);
  float: left;
  width: 200px;
}
img:nth-of-type(2) {
  shape-outside: circle(80px at 50% 50%);
  float: right;
  width: 200px;  
}
p {
  text-align: center;
  line-height: 30px; /* purely for demo */
}

HTML

<img src="http://images.clipartpanda.com/circle-clip-art-circlergb.jpg">
<img src="http://images.clipartpanda.com/circle-clip-art-circlergb.jpg">
<p>Some paragraph whose text content is required to be wrapped such that it follows the curve of the circle on either side. And then there is some filler text just to make the text long enough. Lorem Ipsum Dolor Sit Amet....</p>

Dans l’exemple ci-dessus, les deux images sont en fait des images carrées et lorsque le texte est placé sans la propriété shape-outside, il ne circulera pas autour du cercle de chaque côté. Il circulera autour de la boîte contenant l’image uniquement. Avec shape-outside, la zone flottante est redéfinie comme un cercle et le contenu est amené à circuler autour de ce cercle imaginaire créé à l’aide de shape-outside.

Le cercle imaginaire qui est utilisé pour redéfinir la zone flottante est un cercle avec un rayon de 80 pixels dessiné à partir du centre-milieu de la boîte de référence de l’image.

Marge de forme #

La propriété CSS shape-margin ajoute une margin à shape-outside.

CSS

img:nth-of-type(1) {
  shape-outside: circle(80px at 50% 50%);
  shape-margin: 10px;
  float: left;
  width: 200px;
}
img:nth-of-type(2) {
  shape-outside: circle(80px at 50% 50%);
  shape-margin: 10px;
  float: right;
  width: 200px;  
}
p {
  text-align: center;
  line-height: 30px; /* purely for demo */
}

HTML

<img src="http://images.clipartpanda.com/circle-clip-art-circlergb.jpg">
<img src="http://images.clipartpanda.com/circle-clip-art-circlergb.jpg">
<p>Some paragraph whose text content is required to be wrapped such that it follows the curve of the circle on either side. And then there is some filler text just to make the text long enough. Lorem Ipsum Dolor Sit Amet....</p>

Dans cet exemple, une marge de 10px est ajoutée autour de la forme en utilisant shape-margin. Cela crée un peu plus d’espace entre le cercle imaginaire qui définit la zone flottante et le contenu réel qui circule.
