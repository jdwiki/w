Une mesure de distance CSS est un nombre immédiatement suivi d’une unité de longueur (px, em, pc, in, …)

CSS prend en charge un certain nombre d’unités de mesure de longueur. Ils sont absolus ou relatifs.

Syntaxe #

  • valeurunité
  • 1em

Paramètres #

Unité Descriptif
% Définissez les tailles en termes d’objets parents ou d’objet courant en fonction de la propriété
em Relatif à la taille de la police de l’élément (2em signifie 2 fois la taille de la police actuelle)
rem Relatif à la taille de police de l’élément racine
vw Par rapport à 1% de la largeur de la fenêtre d’affichage*
vh Par rapport à 1 % de la hauteur de la fenêtre d’affichage*
vmin Par rapport à 1 % de la plus petite dimension de la fenêtre d’affichage*
vmax Par rapport à 1 % de la plus grande dimension de la fenêtre d’affichage*
cm centimètres
millimètre millimètres
dans pouces (1in = 96px = 2,54cm)
pixels pixels (1px = 1/96ème de 1in)
pt point (1pt = 1/72 de 1po)
pc picas (1pc = 12 pt)
s secondes (utilisées pour les animations et les transitions)
Mme millisecondes (utilisées pour les animations et les transitions)
ex Relatif à la hauteur x de la police actuelle
ch Basé sur la largeur du caractère zéro (0)
en unité fractionnaire (utilisée pour CSS Grid Layout)

  • Un espace ne peut pas apparaître entre le nombre et l’unité. Cependant, si la valeur est 0, l’unité peut être omise.

  • Pour certaines propriétés CSS, les longueurs négatives sont autorisées.

Création d’éléments évolutifs à l’aide de rems et ems #

Vous pouvez utiliser rem défini par la font-size de votre balise html pour styliser les éléments en définissant leur font-size sur une valeur de rem et utiliser em à l’intérieur de l’élément pour créer des éléments qui mettre à l’échelle avec votre font-size global.

HTML :

<input type="button" value="Button">
<input type="range">
<input type="text" value="Text">

CSS pertinent :

html {
  font-size: 16px;
}

input[type="button"] {
  font-size: 1rem;
  padding: 0.5em 2em;
}

input[type="range"] {
  font-size: 1rem;
  width: 10em;
}

input[type=text] {
  font-size: 1rem;
  padding: 0.5em;
}

Taille de la police avec rem #

CSS3 introduit quelques nouvelles unités, dont l’unité rem, qui signifie “root em”. Regardons comment rem fonctionne.

Tout d’abord, regardons les différences entre em et rem.

  • em : Relatif à la taille de la police du parent. Cela provoque le problème de composition
  • rem : Relatif à la taille de la police de l’élément racine ou <html>. Cela signifie qu’il est possible de déclarer une seule taille de police pour l’élément html et de définir toutes les unités rem comme un pourcentage de celle-ci.

Le principal problème avec l’utilisation de rem pour le dimensionnement de la police est que les valeurs sont quelque peu difficiles à utiliser. Voici un exemple de certaines tailles de police courantes exprimées en unités rem, en supposant que la taille de base est de 16 pixels :

  • 10px = 0.625rem
  • 12px = 0.75rem
  • 14px = 0.875rem
  • 16px = 1rem (base)
  • 18px = 1.125rem
  • 20px = 1.25rem
  • 24px = 1.5rem
  • 30px = 1.875rem
  • 32px = 2rem

CODE:

html { 
  font-size: 16px; 
}

h1 { 
  font-size: 2rem;          /* 32px */
}

p { 
  font-size: 1rem;          /* 16px */ 
}

li { 
  font-size: 1.5em;         /* 24px */
}

vmin et vmax #

  • vmin : par rapport à 1 % de la plus petite dimension de la fenêtre d’affichage
  • vmax : par rapport à 1 % de la plus grande dimension de la fenêtre d’affichage

En d’autres termes, 1 vmin est égal au plus petit de 1 vh et 1 vw

1 vmax est égal au plus grand de 1 vh et 1 vw

Remarque : vmax est non pris en charge dans :

  • toute version d’Internet Explorer
  • Safari avant la version 6.1

vh et vw #

CSS3 a introduit deux unités pour représenter la taille.

  • “vh”, qui signifie “hauteur de la fenêtre”, est relatif à 1 % de la hauteur de la fenêtre
  • vw, qui signifie viewport width est relatif à 1% de la largeur de la fenêtre
div { 
  width: 20vw; 
  height: 20vh;
}

Ci-dessus, la taille de la div occupe 20 % de la largeur et de la hauteur de la fenêtre

en utilisant pourcentage % #

Une des unités utiles lors de la création d’une application réactive.

Sa taille dépend de son conteneur parent.

Équation:

( Largeur du conteneur parent ) * ( Pourcentage (%) ) = Sortie

Par exemple:

Parent a une largeur de 100 px tandis que l’enfant a 50 %.

Sur la sortie, la largeur de Child sera la moitié (50%) de celle de Parent, qui est de 50px.

HTML

<div class="parent">
   PARENT
   <div class="child">
     CHILD
   </div>
</div>

CSS

<style>

*{
  color: #CCC;
}

.parent{
  background-color: blue;
  width: 100px;
}

.child{
  background-color: green;
  width: 50%;
}

</style>
