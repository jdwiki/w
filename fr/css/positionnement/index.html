Syntaxe #

  • position : statique|absolue|fixe|relative|collante|initiale|héritée|non définie ;
  • z-index : auto|nombre|initial|hériter ;

Paramètres #

Paramètre Détails
statique Valeur par défaut. Les éléments sont rendus dans l’ordre, tels qu’ils apparaissent dans le flux de documents. Les propriétés top, right, bottom, left et z-index ne s’appliquent pas.
relatif L’élément est positionné par rapport à sa position normale, donc left:20px ajoute 20 pixels à la position GAUCHE de l’élément
fixe L’élément est positionné par rapport à la fenêtre du navigateur
absolu L’élément est positionné par rapport à son premier élément ancêtre positionné (non statique)
initiale Définit cette propriété à sa valeur par défaut.
hériter Hérite cette propriété de son élément parent.
collant Fonction expérimentale. Il se comporte comme position : statique dans son parent jusqu’à ce qu’un seuil de décalage donné soit atteint, puis il agit comme position : fixe.
non défini Combinaison d’initiale et d’héritage. Plus d’infos [ici][1].

[1] : https://developer.mozilla.org/en-US/docs/Web/CSS/unset

Flux normal est le flux d’éléments si la position de l’élément est statique.

  1. définir width est bénéfique car dans certains cas, cela empêche le chevauchement du contenu de l’élément.

Éléments superposés avec z-index #

Pour modifier l’ordre de la pile éléments positionnés par défaut (propriété position définie sur relative, absolute ou fixed), utilisez la propriété z-index.

Plus l’index z est élevé, plus il est placé haut dans le contexte d’empilement (sur l’axe z).

Exemple #

Dans l’exemple ci-dessous, une valeur de z-index de 3 place le vert en haut, un z-index de 2 place le rouge juste en dessous et un z-index de 1 place le bleu en dessous.

HTML #

<div id="div1"></div>
<div id="div2"></div>
<div id="div3"></div>

##CSS

div {
    position: absolute;
    height: 200px;
    width: 200px;
}
div#div1 {
    z-index: 1;
    left: 0px;
    top: 0px;
    background-color: blue;
}
div#div2 {
    z-index: 3;
    left: 100px;
    top: 100px;
    background-color: green;
}
div#div3 {
    z-index: 2;
    left: 50px;
    top: 150px;
    background-color: red;
}

Voir un exemple de travail sur JSFiddle.

Syntaxe #

z-index: [ number ] | auto;
Paramètre Détails
nombre Une valeur entière. Un nombre plus élevé est plus élevé sur la pile z-index. ‘0’ est la valeur par défaut. Les valeurs négatives sont autorisées.
auto Donne à l’élément le même contexte d’empilement que son parent. (Par défaut)

Remarques #

Tous les éléments sont disposés dans un axe 3D en CSS, y compris un axe de profondeur, mesuré par la propriété z-index. z-index ne fonctionne que sur les éléments positionnés : (voir : [Pourquoi z-index a-t-il besoin d’une position définie pour fonctionner ?](https://www.sitepoint.com/community/t/why-does-z-index -besoin-d-un-poste-defini-pour-travailler/46115)). La seule valeur où elle est ignorée est la valeur par défaut, static.

En savoir plus sur la propriété z-index et les contextes d’empilement dans la Spécification CSS sur la présentation en couches et sur le [Mozilla Developer Network](https: //developer.mozilla.org/en-US/docs/Web/CSS/z-index).

Position absolue #

Lorsque le positionnement absolu est utilisé, la case de l’élément souhaité est retirée du flux normal et n’affecte plus la position des autres éléments sur la page. Propriétés de décalage :

  1. haut
  2. à gauche
  3. à droite
  4. bas

spécifiez que l’élément doit apparaître par rapport à son prochain élément conteneur non statique.

.abspos{
    position:absolute;
    top:0px;
    left:500px;
}

Ce code déplacera la boîte contenant l’élément avec l’attribut class="abspos" vers le bas de 0px et vers la droite de 500px par rapport à son élément contenant.

Emploi stable #

En définissant la position comme fixe, nous pouvons supprimer un élément du flux de documents et définir sa position par rapport à la fenêtre du navigateur. Une utilisation évidente est lorsque nous voulons que quelque chose soit visible lorsque nous faisons défiler vers le bas d’une longue page.

#stickyDiv {
    position:fixed;
    top:10px;
    left:10px;
}

Position relative #

Le positionnement relatif déplace l’élément par rapport à l’endroit où il aurait été dans le flux normal. Propriétés de décalage :

  1. haut
  2. à gauche
  3. à droite
  4. bas

sont utilisés pour indiquer jusqu’où déplacer l’élément à partir de l’endroit où il aurait été en flux normal.

.relpos{
    position:relative;
    top:20px;
    left:30px;
}

Ce code déplacera la boîte contenant l’élément avec l’attribut class=“relpos” 20px vers le bas et 30px vers la droite à partir de l’endroit où il aurait été en flux normal.

Positionnement statique #

La position par défaut d’un élément est static. Pour citer MDN :

Ce mot-clé permet à l’élément d’utiliser le comportement normal, c’est-à-dire qu’il est disposé dans sa position actuelle dans le flux. Les propriétés top, right, bottom, left et z-index ne s’appliquent pas.

.element{
  position:static;
}
