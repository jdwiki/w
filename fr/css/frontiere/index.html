border-radius #

La propriété border-radius vous permet de modifier la forme du modèle de boîte de base.

Chaque coin d’un élément peut avoir jusqu’à deux valeurs, pour le rayon vertical et horizontal de ce coin (pour un maximum de 8 valeurs).

[![border-radius][1]][1]

[1] : http://i.stack.imgur.com/FnIqF.png

Le premier ensemble de valeurs définit le rayon horizontal. Le deuxième ensemble de valeurs facultatif, précédé d’un “/” , définit le rayon vertical. Si un seul ensemble de valeurs est fourni, il est utilisé à la fois pour le rayon vertical et horizontal.

border-radius: 10px 5% / 20px 25em 30px 35em;

Le 10px est le rayon horizontal du haut-gauche et du bas-droit. Et le 5% est le rayon horizontal du haut-droit et du bas-gauche. Les quatre autres valeurs après ‘/’ sont les rayons verticaux pour le haut-gauche, le haut-droit, le bas-droit et le bas-gauche.

Comme pour de nombreuses propriétés CSS, les raccourcis peuvent être utilisés pour tout ou partie valeurs possibles. Vous pouvez donc spécifier entre une et huit valeurs. Le raccourci suivant vous permet de définir le rayon horizontal et vertical de chaque coin sur la même valeur :

HTML :

<div class='box'></div>

CSS :

.box {
    width: 250px;
    height: 250px;
    background-color: black;
    border-radius: 10px;
}

Border-radius est le plus souvent utilisé pour convertir des éléments de boîte en cercles. En définissant border-radius sur la moitié de la longueur d’un élément carré, un élément circulaire est créé :

.circle {
    width: 200px;
    height: 200px;
    border-radius: 100px;
}

Étant donné que border-radius accepte les pourcentages, il est courant d’utiliser 50 % pour éviter de calculer manuellement la valeur border-radius :

.circle {
    width: 150px;
    height: 150px;
    border-radius: 50%;
}

Si les propriétés width et height ne sont pas égales, la forme résultante sera un ovale plutôt qu’un cercle.

Exemple de rayon de bordure spécifique au navigateur :

  -webkit-border-top-right-radius: 4px;
  -webkit-border-bottom-right-radius: 4px;
  -webkit-border-bottom-left-radius: 0;
  -webkit-border-top-left-radius: 0;
  -moz-border-radius-topright: 4px;
  -moz-border-radius-bottomright: 4px;
  -moz-border-radius-bottomleft: 0;
  -moz-border-radius-topleft: 0;
  border-top-right-radius: 4px;
  border-bottom-right-radius: 4px;
  border-bottom-left-radius: 0;
  border-top-left-radius: 0;

style de bordure #

La propriété border-style définit le style de la bordure d’un élément. Cette propriété peut avoir de une à quatre valeurs (pour chaque côté de l’élément une valeur.)

Exemples:

border-style: dotted;

border-style: dotted solid double dashed;

[![styles de bordure][1]][1]

border-style peut également avoir les valeurs none et hidden. Ils ont le même effet, sauf que hidden fonctionne pour la résolution des conflits de bordure pour les éléments <table>. Dans un <table> avec plusieurs bordures, none a la priorité la plus basse (ce qui signifie qu’en cas de conflit, la bordure s’afficherait) et hidden a la priorité la plus élevée (ce qui signifie qu’en cas de conflit, la bordure ne s’afficherait pas) .

[1] : http://i.stack.imgur.com/gieW6.png

Frontières multiples #

Utilisation du contour :

.div1{
  border: 3px solid black; 
  outline: 6px solid blue;
  width: 100px;
  height: 100px;
  margin: 20px;
}

Utilisation de box-shadow :

.div2{
  border: 5px solid green; 
  box-shadow: 0px 0px 0px 4px #000;
  width: 100px;
  height: 100px;
  margin: 20px;
}

Utilisation d’un pseudo-élément :

.div3 {
  position: relative;
  border: 5px solid #000;
  width: 100px;
  height: 100px;
  margin: 20px;
}
.div3:before {
  content: " ";
  position: absolute;
  border: 5px solid blue;
  z-index: -1;
  top: 5px;
  left: 5px;
  right: 5px;
  bottom: 5px;
}

bordure (raccourcis) #

Dans la plupart des cas, vous souhaitez définir plusieurs propriétés de bordure (border-width, border-style et border-color) pour tous les côtés d’un élément.

Au lieu d’écrire :

border-width: 1px;
border-style: solid;
border-color: #000;

Vous pouvez simplement écrire :

border: 1px solid #000;

Ces raccourcis sont également disponibles pour chaque côté d’un élément : border-top, border-left, border-right et border-bottom. Ainsi vous pouvez faire :

border-top: 2px double #aaaaaa;

border-image #

Avec la propriété border-image vous avez la possibilité de définir une image à utiliser à la place des styles de bordure normaux.

Une border-image se compose essentiellement d’un

  • border-image-source : le chemin vers l’image à utiliser
  • border-image-slice : spécifie le décalage utilisé pour diviser l’image en neuf régions (quatre coins, quatre bords et un milieu)
  • border-image-repeat : spécifie comment les images pour les côtés et le milieu de l’image de bordure sont mises à l’échelle

Prenons l’exemple suivant où border.png est une image de 90x90 pixels :

border-image: url("border.png") 30 stretch;

L’image sera divisée en neuf régions de 30x30 pixels. Les bords seront utilisés comme coins de la bordure tandis que le côté sera utilisé entre les deux. Si l’élément est plus haut/large que 30 pixels, cette partie de l’image sera étirée. La partie centrale de l’image est par défaut transparente.

border-collapse #

La propriété border-collapse s’applique uniquement aux tables (et aux éléments affichés comme display: table ou inline-table) et définit si les bordures du tableau sont réduites en une seule bordure ou détachées comme dans le HTML standard.

table {
  border-collapse: separate; /* default */
  border-spacing: 2px; /* Only works if border-collapse is separate */
}

Voir également l’entrée de documentation Tables - border-collapse

Création d’une bordure multicolore à l’aide de border-image #

CSS #

.bordered {
  border-image: linear-gradient(to right, red 20%, green 20%, green 40%, blue 40%, blue 60%, maroon 60%, maroon 80%, chocolate 80%); /* gradient with required colors */
  border-image-slice: 1;
}

HTML #

<div class='bordered'>Border on all sides</div>

L’exemple ci-dessus produirait une bordure composée de 5 couleurs différentes. Les couleurs sont définies par un “dégradé linéaire” (vous pouvez trouver plus d’informations sur les dégradés dans les [docs](https://www.wikiod.com/fr/css/arriere-plans#Dégradés de fond )). Vous pouvez trouver plus d’informations sur la propriété border-image-slice dans l’exemple [border-image](https://www.wikiod.com/fr/css/frontiere#image de bordure) sur la même page .

[![entrez la description de l’image ici][1]][1]

(Remarque : des propriétés supplémentaires ont été ajoutées à l’élément à des fins de présentation.)

Vous auriez remarqué que la bordure gauche n’a qu’une seule couleur (la couleur de début du dégradé) tandis que la bordure droite n’a également qu’une seule couleur (la couleur de fin du dégradé). Cela est dû au fonctionnement de la propriété d’image de bordure. C’est comme si le dégradé était appliqué à l’ensemble de la boîte, puis les couleurs étaient masquées dans les zones de remplissage et de contenu, donnant ainsi l’impression que seule la bordure avait le dégradé.

La ou les bordures ayant une seule couleur dépendent de la définition du dégradé. Si le dégradé est un dégradé “vers la droite”, la bordure gauche serait la couleur de début du dégradé et la bordure droite serait la couleur de fin. S’il s’agissait d’un dégradé “vers le bas”, la bordure supérieure serait la couleur de départ du dégradé et la bordure inférieure serait la couleur de fin. Vous trouverez ci-dessous la sortie d’un dégradé de 5 couleurs “vers le bas”.

[![entrez la description de l’image ici][2]][2]

Si la bordure n’est requise que sur des côtés spécifiques de l’élément, la propriété border-width peut être utilisée comme avec n’importe quelle autre bordure normale. Par exemple, l’ajout du code ci-dessous produirait une bordure uniquement en haut de l’élément.

border-width: 5px 0px 0px 0px;

[![entrez la description de l’image ici][3]][3]

Notez que tout élément qui a la propriété border-image ne respectera pas le border-radius (c’est-à-dire que la bordure ne se courbera pas). Ceci est basé sur la déclaration ci-dessous dans la spécification :

Les arrière-plans d’une boîte, mais pas son image de bordure, sont écrêtés sur la courbe appropriée (comme déterminé par ‘background-clip’).

bordure-[gauche|droite|haut|bas] #

La propriété border-[left|right|top|bottom] est utilisée pour ajouter une bordure à un côté spécifique d’un élément.

Par exemple, si vous vouliez ajouter une bordure sur le côté gauche d’un élément, vous pourriez faire :

#element {
    border-left: 1px solid black;
}
