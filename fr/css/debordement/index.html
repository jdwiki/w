Syntaxe #

  • débordement : visible | caché | faire défiler | automatique | initiale | hériter;

Paramètres #

Valeur Débordement Détails
visible Affiche tout le contenu débordant en dehors de l’élément
faire défiler Masque le contenu débordant et ajoute une barre de défilement
caché Masque le contenu débordant, les deux barres de défilement disparaissent et la page devient fixe
auto Identique à scroll si le contenu déborde, mais n’ajoute pas de barre de défilement si le contenu tient
hériter Hériter de la valeur de l’élément parent pour cette propriété

[La propriété overflow spécifie s’il faut couper le contenu, rendre barres de défilement ou étirer un conteneur pour afficher le contenu lorsqu’il déborde de son conteneur de niveau bloc.][1]

Lorsqu’un élément est trop petit pour afficher son contenu, que se passe-t-il ? Par défaut, le contenu débordera et s’affichera en dehors de l’élément. Cela donne une mauvaise image de votre travail. Vous voulez que votre travail soit beau, vous définissez donc la propriété overflow pour gérer le contenu débordant de manière souhaitable.

Les valeurs de la propriété overflow sont identiques à celles des propriétés overflow-x et overflow-y, sauf qu’elles s’appliquent le long de chaque axe

La propriété overflow-wrap est également connue sous le nom de propriété word-wrap.

Note importante : [Utiliser la propriété overflow avec une valeur différente de visible créera un nouveau contexte de formatage de bloc.][1]

[1] : https://developer.mozilla.org/en/docs/Web/CSS/overflow

overflow-wrap #

overflow-wrap indique à un navigateur qu’il peut diviser une ligne de texte à l’intérieur d’un élément ciblé sur plusieurs lignes dans un endroit autrement incassable. Utile pour éviter qu’une longue chaîne de texte ne cause des problèmes de mise en page en raison du débordement de son conteneur.

CSS

div {
    width:100px;
    outline: 1px dashed #bbb;
}

#div1 {
    overflow-wrap:normal;
}

#div2 {
    overflow-wrap:break-word;
}

HTML

<div id="div1">
      <strong>#div1</strong>: Small words are displayed normally, but a long word like <span style="red;">supercalifragilisticexpialidocious</span> is too long so it will overflow past the edge of the line-break
</div>

<div id="div2">
      <strong>#div2</strong>: Small words are displayed normally, but a long word like <span style="red;">supercalifragilisticexpialidocious</span> will be split at the line break and continue on the next line.
</div>

[![entrez la description de l’image ici][1]][1]

overflow-wrap – Valeur Détails
normale Laisse déborder un mot s’il est plus long que la ligne
break-word Divisera un mot en plusieurs lignes, si nécessaire
hériter Hérite de la valeur de l’élément parent pour cette propriété

[1] : https://i.stack.imgur.com/VRNhr.png

débordement : défilement #

HTML

<div>
    This div is too small to display its contents to display the effects of the overflow property.
</div>

CSS

div {
    width:100px;
    height:100px;
    overflow:scroll;
}

Résultat

[![Image montrant une div de 100 px sur 100 px avec des barres de défilement][1]][1]

Le contenu ci-dessus est découpé dans une boîte de 100 pixels sur 100 pixels, avec un défilement disponible pour afficher le contenu débordant.

La plupart des navigateurs de bureau affichent à la fois des barres de défilement horizontales et verticales, qu’un contenu soit tronqué ou non. Cela peut éviter les problèmes d’apparition et de disparition des barres de défilement dans un environnement dynamique. Les imprimantes peuvent imprimer un contenu débordant.

[1] : http://i.stack.imgur.com/krWW2.png

débordement : visible #

HTML

<div>
     Even if this div is too small to display its contents, the content is not clipped.
</div>

CSS

div {
     width:50px;
     height:50px;
     overflow:visible;
}

Résultat

[![entrez la description de l’image ici][1]][1]

Le contenu n’est pas tronqué et sera affiché en dehors de la zone de contenu s’il dépasse la taille de son conteneur.

[1] : https://i.stack.imgur.com/HLZHC.png

Contexte de formatage de bloc créé avec débordement #

L’utilisation de la propriété overflow avec une valeur différente de visible créera un nouveau contexte de formatage de bloc. Ceci est utile pour aligner un élément de bloc à côté d’un élément flottant.

CSS

img {
    float:left;
    margin-right: 10px;
}
 div {
     overflow:hidden; /* creates block formatting context */
}

HTML

<img src="http://placehold.it/100x100">
<div>
  <p>Lorem ipsum dolor sit amet, cum no paulo mollis pertinacia.</p>
  <p>Ad case omnis nam, mutat deseruisse persequeris eos ad, in tollit debitis sea.</p>
</div>

Résultat

[![entrez la description de l’image ici][1]][1]

Cet exemple montre comment les paragraphes d’un div avec le jeu de propriétés overflow interagiront avec une image flottante.

[1] : http://i.stack.imgur.com/s0Pch.png

débordement-x et débordement-y #

Ces deux propriétés fonctionnent de la même manière que la propriété overflow et acceptent les mêmes valeurs. Le paramètre overflow-x ne fonctionne que sur l’axe x ou de gauche à droite. Le overflow-y fonctionne sur l’axe y ou de haut en bas.

HTML

<div id="div-x">
    If this div is too small to display its contents, 
    the content to the left and right will be clipped.
</div>

<div id="div-y">
    If this div is too small to display its contents, 
    the content to the top and bottom will be clipped.
</div>

CSS

div {
    width: 200px;
    height: 200px;
}

#div-x {
    overflow-x: hidden;
}

#div-y {
    overflow-y: hidden;
}
