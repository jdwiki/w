Didacticiel du langage Haskell

[![Logo Haskell][1]][1]

[Haskell][2] est un langage de programmation avancé purement fonctionnel.

Fonctionnalités: #

  • Type statique : Chaque expression dans Haskell a un type qui est déterminé au moment de la compilation. La vérification de type statique est le processus de vérification de la sécurité de type d’un programme basé sur l’analyse du texte d’un programme (code source). Si un programme passe un vérificateur de type statique, alors le programme est garanti pour satisfaire un ensemble de propriétés de sécurité de type pour toutes les entrées possibles.
  • Puretement fonctionnel : Chaque fonction dans Haskell est une fonction au sens mathématique. Il n’y a pas d’instructions ou d’instructions, seulement des expressions qui ne peuvent pas muter les variables (locales ou globales) ni accéder à l’état comme le temps ou les nombres aléatoires.
  • Concurrent : Son compilateur phare, GHC, est livré avec un ramasse-miettes parallèle hautes performances et une bibliothèque de concurrence légère contenant un certain nombre de primitives et d’abstractions de concurrence utiles.
  • Évaluation paresseuse : Les fonctions n’évaluent pas leurs arguments. Retarde l’évaluation d’une expression jusqu’à ce que sa valeur soit nécessaire.
  • Usage général : Haskell est conçu pour être utilisé dans tous les contextes et environnements.
  • Packages : La contribution open source à Haskell est très active avec une large gamme de packages disponibles sur les serveurs de packages publics.

Le dernier standard de Haskell est Haskell 2010. Depuis mai 2016, un groupe travaille sur la prochaine version, Haskell 2020.

La [documentation officielle de Haskell][3] est également une ressource complète et utile. Super endroit pour trouver des livres, des cours, des tutoriels, des manuels, des guides, etc.
