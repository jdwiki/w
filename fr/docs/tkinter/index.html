Tutoriel tkinter

Tkinter ("Tk Interface") est le package multiplateforme standard de Python pour la création d’interfaces utilisateur graphiques (GUI). Il donne accès à un interpréteur Tcl sous-jacent avec la boîte à outils Tk, qui est elle-même une bibliothèque d’interface utilisateur graphique multiplateforme et multilingue.

Tkinter n’est pas la seule bibliothèque graphique pour python, mais c’est celle qui est standard. Les bibliothèques GUI supplémentaires pouvant être utilisées avec python incluent [wxPython][1], [PyQt][2] et [kivy][3].

La plus grande force de Tkinter est son omniprésence et sa simplicité. Il fonctionne prêt à l’emploi sur la plupart des plates-formes (linux, OSX, Windows) et est livré avec une large gamme de widgets nécessaires à la plupart des tâches courantes (boutons, étiquettes, canevas de dessin, texte multiligne, etc.).

En tant qu’outil d’apprentissage, tkinter possède certaines fonctionnalités uniques parmi les boîtes à outils GUI, telles que les polices nommées, les balises de liaison et le traçage des variables.

Différences entre python 2 et 3 #

Tkinter est en grande partie inchangé entre python 2 et python 3, la principale différence étant que le package et les modules tkinter ont été renommés.

Importation en python 2.x #

En python 2.x, le package tkinter est nommé Tkinter et les packages associés ont leurs propres noms. Par exemple, ce qui suit montre un ensemble typique d’instructions d’importation pour Python 2.x :

import Tkinter as tk
import tkFileDialog as filedialog
import ttk

Importation en python 3.x #

Bien que les fonctionnalités n’aient pas beaucoup changé entre python 2 et 3, les noms de tous les modules tkinter ont changé. Voici un ensemble typique d’instructions d’importation pour Python 3.x :

import tkinter as tk
from tkinter import filedialog
from tkinter import ttk
