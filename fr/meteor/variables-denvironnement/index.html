Utilisation des variables d’environnement avec Meteor #

Les variables d’environnement peuvent être définies avant l’appel du météore, comme ceci :

PORT=4000 meteor
NODE_ENV="staging" meteor

Configuration du serveur SMTP Meteor #

Exemple Gmail

MAIL_URL=smtp://username%40gmail.com:password@smtp.gmail.com:465/

Remarque : Cette configuration ne permet d’envoyer que 2 000 e-mails par jour. Veuillez consulter https://support.google.com/a/answer/176600?hl=en pour d’autres configurations.
