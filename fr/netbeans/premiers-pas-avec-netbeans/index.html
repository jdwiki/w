Exécutez votre programme Java avec Netbeans IDE #

Donc, vous avez terminé l’installation, il est maintenant temps d’exécuter votre programme avec. Netbeans a créé un raccourci vers votre bureau, “Double-cliquez” pour ouvrir l’IDE.

Pour créer un nouveau projet

1. Cliquez sur ce bouton [![ce bouton][2]][2] OU

  • Goto : fichiers » Nouveau projet, puis cette fenêtre apparaîtra

[![entrez la description de l’image ici][3]][3]

Sélectionnez “Java” dans les catégories (à gauche) et “Java Application” dans les projets (à droite) comme indiqué dans l’image ci-dessus.

2. Cliquez sur le bouton Suivant > puis cette fenêtre apparaîtra

[![entrez la description de l’image ici][4]][4]

Fournissez un “Nom de projet” approprié Vous pouvez lui donner le nom que vous voulez ** Mais ** il est bon de le nommer pertinent pour votre travail. Vous pouvez modifier l’emplacement de votre dossier de projet en cliquant sur le bouton Parcourir…. Si vous souhaitez que Netbeans crée automatiquement une “classe” pour vous, laissez la case “Créer une classe principale” cochée.

3. Cliquez sur le bouton Terminer.

Après cela, l’IDE ouvrira la fenêtre de l’éditeur de code pour vous. Nécessairement cela va créer 3 choses :

  1. nom du paquet
  2. nom de classe
  3. méthode principale

Ça devrait être comme ça

package your_package_name;

public class your_class_name {
   
    public static void main(String[] args) {
        // TODO code application logic here
    }
    
}

Il se peut que certains commentaires ne tiennent pas compte de cela pour le moment.

4. Remplacez maintenant

     // TODO code application logic here

avec celui-ci

  System.out.println("Welcome New Programmer");

5. Maintenant “clic droit” dans l’éditeur de code et sélectionnez Exécuter le fichier OU appuyez sur maj+f6 il imprimera

"Welcome New Programmer"

Logiciel requis #

Le kit de développement Java SE (JDK) 7 Update 10 (ou version ultérieure) ou JDK 8 est requis pour installer NetBeans IDE. Vous pouvez télécharger la dernière mise à jour de JDK 7 et JDK 8 à l’adresse http://www.oracle.com/technetwork/java/javase/downloads.

Les bundles PHP et C/C++ NetBeans nécessitent uniquement l’installation et l’exécution de Java Runtime Environment (JRE) 7 ou 8. JDK 7 ou 8 est requis si vous envisagez d’utiliser l’une des fonctionnalités Java.

JDK 7 Update 6 (ou plus récent) est requis pour utiliser les fonctionnalités de JavaFX 2.2 (ou plus récent) dans NetBeans IDE 8.1.

JDK 8 est requis pour utiliser les fonctionnalités de JavaFX 8 dans NetBeans IDE 8.1.

Noter:

  • JavaFX 2.2 est fourni avec le dernier JDK 7 pour Windows, Mac et Linux.

  • JavaFX 8 est fourni avec JDK 8 pour Windows, Mac et Linux.

  • Contrairement aux versions précédentes de NetBeans IDE, NetBeans IDE 8.1 ne nécessite pas que vous configuriez une plate-forme Java “FX-enabled” afin d’utiliser le support JavaFX dans l’IDE. Vous pouvez développer des projets JavaFX dans l’IDE si vous installez une plate-forme Java standard qui est JDK 7 Update 6 (ou plus récent) ou JDK 8.

  • Pour utiliser les fonctionnalités JavaFX dans NetBeans IDE, nous vous recommandons fortement de mettre à niveau votre version JDK vers JDK 7 Update 10 (ou plus récent) ou d’utiliser JDK 8.

Personnalisation de votre installation #

Si vous téléchargez l’option Tous les téléchargements, vous pouvez choisir exactement les outils et les runtimes à installer. Cliquez sur Personnaliser sur la page d’accueil du programme d’installation et sélectionnez les fonctionnalités que vous souhaitez utiliser. Pour plus d’informations, voir Installation du logiciel.

Si vous souhaitez ajouter des fonctionnalités à votre configuration IDE NetBeans, utilisez le gestionnaire de plugins NetBeans. Par exemple, supposons que vous commenciez par télécharger et installer le pack PHP. Vous décidez ensuite d’essayer la fonctionnalité C/C++. Pour ce faire, accédez au gestionnaire de plugins NetBeans depuis l’EDI (choisissez Outils > Plugins) et ajoutez le pack C/C++ à votre installation existante.

Il existe plusieurs remarques importantes sur l’installation et la configuration de certaines fonctionnalités de l’IDE NetBeans :

  • Glassfish est fourni avec NetBeans IDE comme implémentation de référence de Java Enterprise Edition (EE). Il n’est pas destiné à être utilisé dans des environnements de production.

  • Les développeurs d’applications doivent s’assurer que toutes les applications développées à l’aide de Glassfish comme version de référence utilisent des protocoles et des algorithmes cryptographiques sécurisés et à jour, et sont testées en conséquence, avant de déployer ces applications en production.

Notez que la version de Glassfish fournie avec NetBeans IDE n’est pas à jour avec les correctifs de sécurité et est fournie à titre de commodité. Par conséquent, les utilisateurs de NetBeans doivent mettre à niveau vers la version la plus récente de Glassfish lors du déploiement de NetBeans IDE et doivent mettre à niveau vers de nouvelles versions de Glassfish chaque fois que les distributions Glassfish incluent de nouveaux correctifs de sécurité. Les instructions de mise à niveau de Glassfish sont fournies dans la note MOS 2073920.1.

  • Apache Tomcat 8.0.27 est inclus dans les options de téléchargement Java EE et Tous, mais il n’est installé par défaut à partir d’aucune de ces options. Pour installer Apache Tomcat, sélectionnez l’option appropriée ci-dessous :

  • depuis le téléchargement de Java EE, lancez le programme d’installation et cochez la case Apache Tomcat 8.0.27 sur la page d’accueil ;

  • à partir du téléchargement Tout, lancez le programme d’installation et sélectionnez Apache Tomcat 8.0.27 dans la boîte de dialogue Personnaliser l’installation.

  • Pour le développement C/C++ (applicable pour les téléchargements C/C++ et Tous), vous devez en plus installer des compilateurs et des outils.

  • Pour le développement HTML5 et JavaScript pour les appareils mobiles (Android et iOS), vous devez également installer des SDK et d’autres outils de développement pour la plate-forme de développement cible. Pour plus d’informations, reportez-vous à la page wiki des navigateurs mobiles.

  • Pour OS X, vous pouvez personnaliser l’installation de l’IDE NetBeans à partir des téléchargements All et Java EE. L’installation de l’IDE à partir des téléchargements Java SE, HTML5/JavaScript, PHP et C/C++ ne peut pas être personnalisée. You can also download a platform-independent zip file and run the NetBeans IDE executable file. Note that the GlassFish Server Open Source Edition and Apache Tomcat are only available with platform-specific installers.

[1] : https://netbeans.org/community/releases/81/install.html#troubleshooting

Téléchargement de l’IDE NetBeans #

Pour télécharger NetBeans IDE, visitez simplement le [site NetBeans] [1] et téléchargez la version appropriée de l’IDE en fonction de votre système d’exploitation, de votre architecture et de vos technologies. Vous pouvez choisir parmi les technologies suivantes :

  • Java SE. Prend en charge toutes les fonctionnalités de développement Java SE standard ainsi que la prise en charge du développement de la plate-forme NetBeans et du SDK JavaFX 2.2 (ou plus récent) ou du SDK JavaFX 8.

  • Java EE. Fournit des outils pour développer des applications Java SE et Java EE ainsi qu’une prise en charge du développement de la plate-forme NetBeans et du SDK JavaFX 2.2 (ou plus récent) ou du SDK JavaFX 8. Cette option de téléchargement inclut également le logiciel GlassFish Server Open Source Edition 4.1.1 et Apache Tomcat 8.0.27.

  • HTML5/JavaScript. Fournit des outils pour le développement HTML5/Javascript. Inclut Java Runtime Environment et ne nécessite pas d’installation Java distincte.

- PHP. Fournit des outils pour le développement PHP 5.x, le support Zend et Symfony Framework. Inclut Java Runtime Environment et ne nécessite pas d’installation Java distincte.

- C/C++. Prend en charge le développement dans les langages C, C++, Qt, Fortran et Assembly. Inclut Java Runtime Environment et ne nécessite pas d’installation Java distincte.

  • Tout. Il s’agit d’une option de téléchargement complète, qui contient tous les runtimes et technologies disponibles pour l’IDE NetBeans.

La taille du programme d’installation varie selon le système d’exploitation et également selon les technologies sélectionnées. Java JDK doit être installé sur votre ordinateur. En option, vous pouvez également télécharger le [JDK fourni avec NetBeans IDE][2].

Après l’installation, vous pouvez également supprimer ou ajouter des technologies dans le centre de mise à jour de l’IDE.

[1] : https://netbeans.org/downloads/ [2] : http://www.oracle.com/technetwork/java/javase/downloads/jdk-netbeans-jsp-142931.html

Installation : Microsoft Windows et Linux #

Pour installer le logiciel :

  1. Une fois le téléchargement terminé, exécutez le programme d’installation.

    • For Windows, the installer executable file has the .exe extension. Double-click the installer file to run it.

    • For Linux platforms, the installer file has the .sh extension. For these platforms, you need to make the installer files executable by using the following command: chmod +x . Type ./ to run the installer.

  2. Effectuez les étapes suivantes sur la page d’accueil de l’assistant d’installation :

    1. Click Customize. In the Customize Installation dialog box, make your selections.
    2. Click OK.
    3. At the Welcome page of the installation wizard, click Next.

  3. Sur la page Contrat de licence, lisez le contrat de licence, cochez la case d’acceptation, puis cliquez sur Suivant.

  4. Sur la page d’installation de l’IDE NetBeans, procédez comme suit :

    1. Accept the default installation directory for the NetBeans IDE or specify another directory.
    **Note**: The installation directory must be empty and the user profile you are using to run the installer must have read/write permissions for this directory.
2. (Applicable only to All or Java EE bundle.) Accept the default JDK installation to use with the NetBeans IDE or select a different installation from the drop-down list. If the installation wizard did not find a compatible JDK installation to use with the NetBeans IDE, your JDK is not installed in the default location. In this case, specify the path to an installed JDK and click Next, or cancel the current installation. After installing the required JDK version you can restart the installation.

**Note**: If the JDK version is older than the recommended JDK 7 Update 10, download and install the latest JDK update from Java SE Downloads page and restart the NetBeans IDE installer.

  5. Si la page d’installation de GlassFish Server Open Source Edition 4.1.1 s’ouvre, acceptez le répertoire d’installation par défaut ou spécifiez un autre emplacement d’installation.

  6. Si vous installez Apache Tomcat, sur sa page d’installation, acceptez le répertoire d’installation par défaut ou spécifiez un autre emplacement d’installation. Cliquez sur Suivant.

    • At the Summary page, do the following: Verify that the list of components to be installed is correct. Select the Check for Updates check box if you want to check the Update Center for possible updates.
    • Verify that you have adequate space on your system for the installation. Click Install to begin the installation.
    • At the Setup Complete page, provide anonymous usage data if desired, and click Finish.

Remarque : Si vous rencontrez des problèmes pour terminer avec succès l’installation du logiciel, consultez [Dépannage][1] pour obtenir des descriptions et des solutions de contournement suggérées pour les problèmes non résolus qui pourraient affecter le processus d’installation.

##Installation : OS X Pour installer le logiciel :

  1. Une fois le téléchargement terminé, exécutez le programme d’installation. Le fichier d’installation a l’extension .dmg.
  2. Dans le panneau qui s’ouvre, double-cliquez sur l’icône du package. Le paquet a l’extension .pkg. L’assistant d’installation démarre.
  3. Cliquez sur Continuer lorsque le message “Ce package va exécuter un programme pour déterminer si le logiciel peut être installé.” boîte de dialogue s’affiche.
  4. Sur la page Introduction de l’assistant d’installation, cliquez sur Continuer.

Remarque : Si la version du JDK est antérieure à la mise à jour 10 recommandée du JDK 7, téléchargez et installez la dernière mise à jour du JDK à partir de la page de téléchargement de Java SE et redémarrez le programme d’installation de l’IDE NetBeans. 5. Lisez le contrat de licence et cliquez sur Continuer. Cliquez sur Accepter dans la fenêtre contextuelle pour accepter la licence. 6. Sur la page Sélectionner une destination, sélectionnez le lecteur et cliquez sur Continuer. 7. Si vous avez téléchargé le bundle All ou Java EE, vous pouvez personnaliser votre installation. Dans le dernier panneau de l’assistant d’installation, appuyez sur le bouton Personnaliser en bas à gauche du panneau. L’arborescence des produits s’affiche. 8. Sélectionnez les produits que vous souhaitez installer. 9. Entrez le nom et le mot de passe de l’administrateur de votre système et cliquez sur OK pour commencer l’installation.

Installation : package indépendant de la plate-forme #

Pour exécuter le logiciel :

  1. Une fois le téléchargement du fichier ZIP indépendant de la plate-forme terminé, extrayez-le dans n’importe quel dossier de votre système.
  2. Exécutez le fichier exécutable situé dans le répertoire netbeans/bin.
  3. Acceptez le contrat de licence.
  4. Cliquez sur Suivant. L’installation de l’EDI NetBeans démarre.

Remarque : Si une installation JDK compatible est introuvable, vous devrez peut-être ajouter manuellement un chemin d’accès au répertoire d’installation JDK en procédant comme suit :

  1. Ouvrez le fichier netbeans.conf situé dans le répertoire netbeans/etc dans un éditeur de texte.
  2. Entrez l’emplacement d’une installation JDK compatible pour l’option netbeans_jdkhome. L’emplacement par défaut dans Windows est C:\Program Files\Java\jdk1.7.0_67 ou similaire.
  3. Enregistrez le fichier netbeans.conf et exécutez le fichier exécutable dans le répertoire netbeans/bin.

Remarque : GlassFish Server Open Source Edition et Apache Tomcat ne sont disponibles qu’avec des programmes d’installation spécifiques à la plate-forme.

Remarque : Pour utiliser GlassFish Server Open Source Edition et/ou Apache Tomcat qui sont déjà installés sur votre machine, dans l’IDE NetBeans, cliquez sur Serveurs dans le menu Outils. Cela affiche l’assistant d’ajout d’instance de serveur qui aide à enregistrer une instance du serveur Web requis avec l’IDE NetBeans.
