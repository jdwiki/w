Hello World - Fonction Lambda #

C’est l’une des fonctions lambda les plus simples. C’est l’équivalent du programme Hello World. Pour créer votre premier programme, suivez les étapes mentionnées ci-dessous.

  1. Connectez-vous à la console AWS
  2. Cliquez sur Lambda sous calcul
  3. Cliquez sur créer une fonction Lambda [![entrer la description de l’image ici][1]][1]
  4. Ignorer la section de plan de sélection
  5. Dans configurer le déclencheur, cliquez sur le rectangle en pointillé [![entrez la description de l’image ici][2]][2]
  6. Sélectionnez la passerelle API
  7. Remplissez les détails requis comme dans l’image. [![enter image description here][3]][3] API Name est le nom de votre API que vous allez créer. Resource Pattern est le chemin d’URL que vous pouvez appeler votre fonction Lambda. Sélectionnez la méthode http requise. Dans notre exemple, nous choisissons GET. Dans AWS, la mise en scène peut être effectuée de différentes manières, comme prod, dev … Cela vous aidera à différencier les fonctions de prod et dev. À des fins de démonstration, choisissons la sécurité comme ouverte (ce n’est pas recommandé en production). Cliquez sur suivant
  8. Configurez la fonction comme ci-dessous [![entrez la description de l’image ici][4]][4] Fournissez le nom de la fonction, la description de votre fonction et l’environnement d’exécution. Nous choisissons Python comme environnement d’exécution.
  9. Modifiez le code. [![entrer la description de l’image ici][5]][5] Ici, nous imprimons l’événement aws lambda dans cloudtrail qui est gratuit. Il renvoie également une chaîne.
  10. Fournissez le gestionnaire et le rôle de la fonction Lambda [![enter image description here][6]][6] Assurez-vous que le nom du gestionnaire doit commencer par lambda_function.. Créez également un nouveau rôle pour exécuter la fonction lambda. Sélectionnez la quantité de mémoire principale requise pour exécuter votre fonction. Sélectionnez le délai d’attente par défaut et cliquez sur suivant
  11. Cliquez sur créer une fonction
  12. Votre fonction est maintenant prête à être exécutée. Cliquez sur le lien fourni par aws [![enter image description here][7]][7][![enter image description here][8]][8] Lorsque vous cliquez sur le lien, votre fonction lambda sera exécutée dans le arrière-plan et vous obtiendrez une sortie dans le navigateur.

Pourquoi AWS Lambda ? #

AWS Lambda prend en charge

  • Évolutivité et disponibilité transparentes
  • Opérations conviviales pour les développeurs et pas besoin de gérer des serveurs
  • Intégration native aux services AWS
  • Pas besoin de payer pour le temps d’inactivité
  • Intégration RESTful
  • Surveillance de l’interface RESTful à l’aide de la passerelle API AWS
