Syntaxe #

  • Les nombres réels doivent commencer par un ou plusieurs chiffres suivis d’un point suivi d’un ou plusieurs chiffres.
  • ~ est l’opérateur pour désigner les nombres négatifs
  • div est l’opérateur de division entière.
  • / est l’opérateur de division réelle.

Entier #

Division entière

- 6 div 3;
val it = 2 : int

- 3 div 6;
val it = 0 : int

Conversion des valeurs réelles en nombres entiers #

Arrondi

Les valeurs à mi-chemin entre deux entiers vont vers la valeur paire la plus proche.

- round(4.5);
val it = 4 : int
- round(3.5);
val it = 4 : int

Troncation

val it = 4 : int
- trunc(4.5);
val it = 4 : int
- trunc(3.5);
val it = 3 : int

Plancher et Plafond

- ceil(4.5);
val it = 5 : int
- floor(4.5);
val it = 4 : int

Erreur d’opérateur arithmétique avec des types numériques mixtes #

Impossible d’ajouter un entier et un réel*

- 5 + 1.0;
stdIn:1.2-10.4 Error: operator and operand don't agree [overload conflict]
  operator domain: [+ ty] * [+ ty]
  operand:         [+ ty] * real
  in expression:
    5 + 1.0

Coersion de la valeur entière au réel #

- real(6);
val it = 6.0 : real
