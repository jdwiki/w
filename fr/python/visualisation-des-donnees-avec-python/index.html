Seaborn #

Seaborn est un wrapper autour de Matplotlib qui facilite la création de graphiques statistiques communs. La liste des tracés pris en charge comprend des tracés de distribution univariés et bivariés, des tracés de régression et un certain nombre de méthodes pour tracer des variables catégorielles. La liste complète des parcelles fournies par Seaborn se trouve dans leur [référence API] (https://stanford.edu/~mwaskom/software/seaborn/api.html).

Créer des graphiques dans Seaborn est aussi simple que d’appeler la fonction graphique appropriée. Voici un exemple de création d’un histogramme, d’une estimation de la densité du noyau et d’un tracé de tapis pour des données générées de manière aléatoire.

import numpy as np  # numpy used to create data from plotting
import seaborn as sns  # common form of importing seaborn

# Generate normally distributed data
data = np.random.randn(1000)

# Plot a histogram with both a rugplot and kde graph superimposed
sns.distplot(data, kde=True, rug=True)

[![Exemple de distribution][1]][1]

Le style du tracé peut également être contrôlé à l’aide d’une syntaxe déclarative.

# Using previously created imports and data.

# Use a dark background with no grid.
sns.set_style('dark')
# Create the plot again
sns.distplot(data, kde=True, rug=True)

[![Exemple de style][2]][2]

En prime, les commandes matplotlib normales peuvent toujours être appliquées aux tracés Seaborn. Voici un exemple d’ajout de titres d’axes à notre histogramme créé précédemment.

# Using previously created data and style

# Access to matplotlib commands
import matplotlib.pyplot as plt

# Previously created plot. 
sns.distplot(data, kde=True, rug=True)
# Set the axis labels.
plt.xlabel('This is my x-axis')
plt.ylabel('This is my y-axis')

[![Exemple matplotlib][3]][3]

[1] : http://i.imgur.com/CyxeSfj.png [2] : http://i.imgur.com/JMjHv6C.png [3] : http://i.imgur.com/ei2fa17.png

Matplotlib #

Matplotlib est une bibliothèque de traçage mathématique pour Python qui fournit une variété de fonctionnalités de traçage différentes.

** La documentation matplotlib peut être trouvée ici, avec les SO Docs disponibles ici. **

Matplotlib fournit deux méthodes distinctes de traçage, bien qu’elles soient pour la plupart interchangeables :

  • Premièrement, matplotlib fournit l’interface pyplot, une interface directe et simple à utiliser qui permet de tracer des graphiques complexes dans un style de type MATLAB.
  • Deuxièmement, matplotlib permet à l’utilisateur de contrôler les différents aspects (axes, lignes, graduations, etc.) directement à l’aide d’un système basé sur des objets. Ceci est plus difficile mais permet un contrôle complet sur l’ensemble de l’intrigue.

Vous trouverez ci-dessous un exemple d’utilisation de l’interface pyplot pour tracer certaines données générées :

import matplotlib.pyplot as plt

# Generate some data for plotting.
x = [0, 1, 2, 3, 4, 5, 6]
y = [i**2 for i in x]

# Plot the data x, y with some keyword arguments that control the plot style.
# Use two different plot commands to plot both points (scatter) and a line (plot).

plt.scatter(x, y, c='blue', marker='x', s=100) # Create blue markers of shape "x" and size 100
plt.plot(x, y, color='red', linewidth=2) # Create a red line with linewidth 2.

# Add some text to the axes and a title.
plt.xlabel('x data')
plt.ylabel('y data')
plt.title('An example plot')

# Generate the plot and show to the user.
plt.show()

[![Exemple de tracé][1]][1]

Notez que plt.show() est connu pour être [problématique][2] dans certains environnements en raison de l’exécution de matplotlib.pyplot en mode interactif, et si c’est le cas, le comportement de blocage peut être remplacé explicitement en passant un facultatif argument, plt.show(block=True), pour résoudre le problème.

Maya VI #

[MayaVI][1] est un outil de visualisation 3D de données scientifiques. Il utilise le kit d’outils de visualisation ou [VTK][2] sous le capot. En utilisant la puissance de [VTK][2], MayaVI est capable de produire une variété de tracés et de figures tridimensionnels. Il est disponible en tant qu’application logicielle distincte et également en tant que bibliothèque. Semblable à [Matplotlib][3], cette bibliothèque fournit une interface de langage de programmation orientée objet pour créer des tracés sans avoir à connaître VTK.

**MayaVI est disponible uniquement dans la série Python 2.7x ! On espère qu’il sera bientôt disponible dans la série Python 3-x ! (Bien qu’un certain succès soit remarqué lors de l’utilisation de ses dépendances dans Python 3) **

La documentation peut être trouvée [ici][1]. Quelques exemples de galeries se trouvent [ici][4]

Voici un exemple de tracé créé à l’aide de MayaVI à partir de la documentation.

# Author: Gael Varoquaux <gael.varoquaux@normalesup.org>
# Copyright (c) 2007, Enthought, Inc.
# License: BSD Style.


from numpy import sin, cos, mgrid, pi, sqrt
from mayavi import mlab

mlab.figure(fgcolor=(0, 0, 0), bgcolor=(1, 1, 1))
u, v = mgrid[- 0.035:pi:0.01, - 0.035:pi:0.01]

X = 2 / 3. * (cos(u) * cos(2 * v)
        + sqrt(2) * sin(u) * cos(v)) * cos(u) / (sqrt(2) -
                                                 sin(2 * u) * sin(3 * v))
Y = 2 / 3. * (cos(u) * sin(2 * v) -
        sqrt(2) * sin(u) * sin(v)) * cos(u) / (sqrt(2)
        - sin(2 * u) * sin(3 * v))
Z = -sqrt(2) * cos(u) * cos(u) / (sqrt(2) - sin(2 * u) * sin(3 * v))
S = sin(u)

mlab.mesh(X, Y, Z, scalars=S, colormap='YlGnBu', )

# Nice view from the front
mlab.view(.0, - 5.0, 4)
mlab.show()

Complot #

[Plotly][1] est une plate-forme moderne de traçage et de visualisation de données. Utile pour produire une variété de graphiques, en particulier pour les sciences des données, Plotly est disponible en tant que bibliothèque pour Python, R, JavaScript, Julia et, * MATLAB*. Il peut également être utilisé comme une application Web avec ces langages.

Les utilisateurs peuvent installer la bibliothèque plotly et l’utiliser hors ligne après l’authentification de l’utilisateur. L’installation de cette bibliothèque et l’authentification hors ligne sont données [ici][2]. De plus, les tracés peuvent également être réalisés dans Jupyter Notebooks.

L’utilisation de cette bibliothèque nécessite un compte avec nom d’utilisateur et mot de passe. Cela donne l’espace de travail pour enregistrer les tracés et les données sur le cloud.

La version gratuite de la bibliothèque a des fonctionnalités légèrement limitées et est conçue pour créer 250 tracés par jour. La version payante possède toutes les fonctionnalités, des téléchargements de tracés illimités et davantage de stockage de données privées. Pour plus de détails, on peut visiter la page principale [ici][1].

Pour de la documentation et des exemples, on peut aller [ici][3]

Un exemple de tracé à partir des exemples de documentation :

import plotly.graph_objs as go
import plotly as ply

# Create random data with numpy
import numpy as np

N = 100
random_x = np.linspace(0, 1, N)
random_y0 = np.random.randn(N)+5
random_y1 = np.random.randn(N)
random_y2 = np.random.randn(N)-5

# Create traces
trace0 = go.Scatter(
    x = random_x,
y = random_y0,
mode = 'lines',
name = 'lines'
)
trace1 = go.Scatter(
    x = random_x,
    y = random_y1,
    mode = 'lines+markers',
    name = 'lines+markers'
)
trace2 = go.Scatter(
    x = random_x,
    y = random_y2,
    mode = 'markers',
    name = 'markers'
)
data = [trace0, trace1, trace2]

ply.offline.plot(data, filename='line-mode')
