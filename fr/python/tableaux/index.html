Les “tableaux” en Python ne sont pas les tableaux des langages de programmation conventionnels comme C et Java, mais plus proches des listes. Une liste peut être une collection d’éléments homogènes ou hétérogènes et peut contenir des entiers, des chaînes ou d’autres listes.

Accéder à des éléments individuels via des index #

Les éléments individuels sont accessibles via des index. Les tableaux Python sont indexés à zéro. Voici un exemple :

my_array = array('i', [1,2,3,4,5])
print(my_array[1])
# 2
print(my_array[2])
# 3
print(my_array[0])
# 1

Introduction de base aux tableaux #

Un tableau est une structure de données qui stocke des valeurs de même type de données. En Python, c’est la principale différence entre les tableaux et les listes.

Alors que les listes python peuvent contenir des valeurs correspondant à différents types de données, les tableaux en python ne peuvent contenir que des valeurs correspondant au même type de données. Dans ce tutoriel, nous allons comprendre les tableaux Python avec quelques exemples.

Si vous débutez avec Python, commencez par l’article Python Introduction.

Pour utiliser des tableaux en langage python, vous devez importer le module standard array. En effet, array n’est pas un type de données fondamental comme les chaînes, les entiers, etc. Voici comment importer le module array en python :

from array import *

Une fois que vous avez importé le module array, vous pouvez déclarer un tableau. Voici comment procéder :

arrayIdentifierName = array(typecode, [Initializers])

Dans la déclaration ci-dessus, arrayIdentifierName est le nom du tableau, typecode permet à python de connaître le type de tableau et Initializers sont les valeurs avec lesquelles le tableau est initialisé.

Les codes de type sont les codes utilisés pour définir le type de valeurs de tableau ou le type de tableau. Le tableau dans la section des paramètres montre les valeurs possibles que vous pouvez utiliser lors de la déclaration d’un tableau et de son type.

Voici un exemple concret de déclaration de tableau python :

my_array = array('i',[1,2,3,4])

Dans l’exemple ci-dessus, le code de type utilisé est i. Ce typecode représente un entier signé dont la taille est de 2 octets.

Voici un exemple simple d’un tableau contenant 5 entiers

from array import *
my_array = array('i', [1,2,3,4,5])
for i in my_array:
    print(i)
# 1
# 2
# 3
# 4
# 5

Ajouter n’importe quelle valeur au tableau en utilisant la méthode append() #

my_array = array('i', [1,2,3,4,5])
my_array.append(6)
# array('i', [1, 2, 3, 4, 5, 6])

Notez que la valeur “6” a été ajoutée aux valeurs de tableau existantes.

Insérer une valeur dans un tableau en utilisant la méthode insert() #

Nous pouvons utiliser la méthode insert() pour insérer une valeur à n’importe quel index du tableau. Voici un exemple :

my_array = array('i', [1,2,3,4,5])
my_array.insert(0,0)
#array('i', [0, 1, 2, 3, 4, 5])

Dans l’exemple ci-dessus, la valeur 0 a été insérée à l’index 0. Notez que le premier argument est l’index tandis que le second argument est la valeur.

Étendre le tableau python à l’aide de la méthode extend () #

Un tableau python peut être étendu avec plus d’une valeur en utilisant la méthode extend(). Voici un exemple :

my_array = array('i', [1,2,3,4,5])
my_extnd_array = array('i', [7,8,9,10])
my_array.extend(my_extnd_array)
# array('i', [1, 2, 3, 4, 5, 7, 8, 9, 10])

Nous voyons que le tableau my_array a été étendu avec des valeurs de my_extnd_array.

Ajouter des éléments de la liste dans le tableau à l’aide de la méthode fromlist() #

Voici un exemple:

my_array = array('i', [1,2,3,4,5])
c=[11,12,13]
my_array.fromlist(c)
# array('i', [1, 2, 3, 4, 5, 11, 12, 13])

Nous voyons donc que les valeurs 11,12 et 13 ont été ajoutées de la liste c à my_array.

Supprimer tout élément de tableau à l’aide de la méthode remove() #

Voici un exemple :

my_array = array('i', [1,2,3,4,5])
my_array.remove(4)
# array('i', [1, 2, 3, 5])

On voit que l’élément 4 a été supprimé du tableau.

Supprimer le dernier élément du tableau en utilisant la méthode pop() #

pop supprime le dernier élément du tableau. Voici un exemple :

my_array = array('i', [1,2,3,4,5])
my_array.pop()
# array('i', [1, 2, 3, 4])

Nous voyons donc que le dernier élément (‘5’) a été retiré du tableau.

Récupère n’importe quel élément via son index en utilisant la méthode index() #

index() renvoie le premier index de la valeur correspondante. N’oubliez pas que les tableaux sont indexés à zéro.

my_array = array('i', [1,2,3,4,5])
print(my_array.index(5))
# 5
my_array = array('i', [1,2,3,3,5])
print(my_array.index(3))
# 3

Notez dans ce deuxième exemple qu’un seul index a été renvoyé, même si la valeur existe deux fois dans le tableau

Inverser un tableau python en utilisant la méthode reverse() #

La méthode reverse() fait ce que son nom l’indique - inverse le tableau. Voici un exemple :

my_array = array('i', [1,2,3,4,5])
my_array.reverse()
# array('i', [5, 4, 3, 2, 1])

Obtenir des informations sur le tampon de tableau via la méthode buffer_info() #

Cette méthode vous fournit l’adresse de début du tampon de tableau en mémoire et le nombre d’éléments dans le tableau. Voici un exemple:

my_array = array('i', [1,2,3,4,5])
my_array.buffer_info()
(33881712, 5)

Vérifier le nombre d’occurrences d’un élément à l’aide de la méthode count() #

count() renverra le nombre de fois où l’élément apparaît dans un tableau. Dans l’exemple suivant, nous voyons que la valeur “3” apparaît deux fois.

my_array = array('i', [1,2,3,3,5])
my_array.count(3)
# 2

Convertir un tableau en chaîne en utilisant la méthode tostring() #

tostring() convertit le tableau en chaîne.

my_char_array = array('c', ['g','e','e','k'])
# array('c', 'geek')
print(my_char_array.tostring())
# geek

Convertir un tableau en une liste python avec les mêmes éléments en utilisant la méthode tolist() #

Lorsque vous avez besoin d’un objet Python list, vous pouvez utiliser la méthode tolist() pour convertir votre tableau en liste.

my_array = array('i', [1,2,3,4,5])
c = my_array.tolist()
# [1, 2, 3, 4, 5]

Ajouter une chaîne au tableau de caractères en utilisant la méthode fromstring() #

Vous pouvez ajouter une chaîne à un tableau de caractères en utilisant fromstring()

my_char_array = array('c', ['g','e','e','k'])
my_char_array.fromstring("stuff")
print(my_char_array)
#array('c', 'geekstuff')
