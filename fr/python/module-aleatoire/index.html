Création d’un mot de passe utilisateur aléatoire #

Afin de créer un mot de passe utilisateur aléatoire, nous pouvons utiliser les symboles fournis dans le module string. Spécifiquement punctuation pour les symboles de ponctuation, ascii_letters pour les lettres et digits pour les chiffres :

from string import punctuation, ascii_letters, digits

Nous pouvons alors combiner tous ces symboles dans un nom nommé symbols :

symbols = ascii_letters + digits + punctuation

Supprimez l’un ou l’autre pour créer un pool de symboles avec moins d’éléments.

Après cela, nous pouvons utiliser random.SystemRandom pour générer un mot de passe. Pour un mot de passe de 10 longueurs :

secure_random = random.SystemRandom()
password = "".join(secure_random.choice(symbols) for i in range(10))
print(password)  # '^@g;J?]M6e'

Notez que d’autres routines rendues immédiatement disponibles par le module random — comme random.choice, random.randint, etc. — sont inadaptés à des fins cryptographiques.

Derrière les rideaux, ces routines utilisent le [Mersenne Twister PRNG][mersenne-prng], qui ne satisfait pas les exigences d’un [CSPRNG][csprng]. Ainsi, en particulier, vous ne devez en utiliser aucun pour générer les mots de passe que vous envisagez d’utiliser. Utilisez toujours une instance de SystemRandom comme indiqué ci-dessus.

[mersenne-prng] : https://en.wikipedia.org/wiki/Mersenne_Twister [csprng] : https://en.wikipedia.org/wiki/Cryptographically_secure_pseudorandom_number_generator

À partir de Python 3.6, le module secrets est disponible, qui expose des fonctionnalités cryptographiquement sûres.

Citant la [documentation officielle][pydoc-secrets-3.6], pour générer “un mot de passe alphanumérique à dix caractères avec au moins une lettre minuscule, au moins une lettre majuscule et au moins trois chiffres,” vous pouvez :

import string
alphabet = string.ascii_letters + string.digits
while True:
    password = ''.join(choice(alphabet) for i in range(10))
    if (any(c.islower() for c in password)
            and any(c.isupper() for c in password)
            and sum(c.isdigit() for c in password) >= 3):
        break

Aléatoire et séquences : mélange, choix et échantillon #

import random

mélanger() #

Vous pouvez utiliser random.shuffle() pour mélanger/randomiser les éléments dans une séquence mutable et indexable. Par exemple une ’liste’ :

laughs = ["Hi", "Ho", "He"]

random.shuffle(laughs)     # Shuffles in-place! Don't do: laughs = random.shuffle(laughs)

print(laughs)
# Out: ["He", "Hi", "Ho"]  # Output may vary!

choix() #

Prend un élément aléatoire d’une séquence arbitraire :

print(random.choice(laughs))
# Out: He                  # Output may vary!

goûter() #

Comme choice, il prend des éléments aléatoires à partir d’une séquence arbitraire, mais vous pouvez spécifier combien :

#                   |--sequence--|--number--|
print(random.sample(    laughs   ,     1    ))  # Take one element
# Out: ['Ho']                    # Output may vary!

il ne prendra pas deux fois le même élément :

print(random.sample(laughs, 3))  # Take 3 random element from the sequence.
# Out: ['Ho', 'He', 'Hi']        # Output may vary!

print(random.sample(laughs, 4))  # Take 4 random element from the 3-item sequence.

ValueError : échantillon supérieur à la population

Création d’entiers et de flottants aléatoires : randint, randrange, random et uniform #

import random

randint() #

Renvoie un entier aléatoire entre x et y (inclus):

random.randint(x, y)

Par exemple, obtenir un nombre aléatoire entre ‘1’ et ‘8’ :

random.randint(1, 8) # Out: 8

randrange() #

random.randrange a la même syntaxe que range et contrairement à random.randint, la dernière valeur est non inclusive :

random.randrange(100)       # Random integer between 0 and 99
random.randrange(20, 50)    # Random integer between 20 and 49
random.rangrange(10, 20, 3) # Random integer between 10 and 19 with step 3 (10, 13, 16 and 19)

Créer des nombres aléatoires cryptographiquement sécurisés #

Par défaut, le module aléatoire Python utilise le Mersenne Twister [PRNG][1] pour générer des nombres aléatoires, qui, bien qu’appropriés dans des domaines comme les simulations, ne répondent pas aux exigences de sécurité dans des environnements plus exigeants.

Afin de créer un nombre pseudo-aléatoire cryptographiquement sécurisé, on peut utiliser [SystemRandom][2] qui, en utilisant os.urandom, est capable d’agir comme un générateur de nombres pseudo-aléatoires cryptographiquement sécurisé, [CPRNG][3].

La façon la plus simple de l’utiliser consiste simplement à initialiser la classe SystemRandom. Les méthodes fournies sont similaires à celles exportées par le module random.

from random import SystemRandom
secure_rand_gen = SystemRandom()

Afin de créer une séquence aléatoire de 10 ints dans la plage [0, 20], on peut simplement appeler randrange() :

print([secure_rand_gen.randrange(10) for i in range(10)])
# [9, 6, 9, 2, 2, 3, 8, 0, 9, 9]

Pour créer un entier aléatoire dans une plage donnée, on peut utiliser randint :

print(secure_rand_gen.randint(0, 20))
# 5

et, par conséquent, pour toutes les autres méthodes. L’interface est exactement la même, le seul changement est le générateur de nombres sous-jacent.

Vous pouvez également utiliser [os.urandom][4] directement pour obtenir des octets aléatoires cryptographiquement sécurisés.

[1] : https://en.wikipedia.org/wiki/Pseudorandom_number_generator [2] : https://docs.python.org/3/library/random.html#random.SystemRandom [3] : https://en.wikipedia.org/wiki/Cryptographically_secure_pseudorandom_number_generator [4] : https://docs.python.org/3/library/os.html#os.urandom

Nombres aléatoires reproductibles : graine et état #

La définition d’une graine spécifique créera une série de nombres aléatoires fixes :

random.seed(5)                 # Create a fixed state         
print(random.randrange(0, 10))  # Get a random integer between 0 and 9
# Out: 9
print(random.randrange(0, 10))
# Out: 4

La réinitialisation de la graine créera à nouveau la même séquence “aléatoire”:

random.seed(5)                 # Reset the random module to the same fixed state.
print(random.randrange(0, 10))
# Out: 9
print(random.randrange(0, 10))
# Out: 4

Comme la graine est fixe, ces résultats sont toujours ‘9’ et ‘4’. S’il n’est pas nécessaire d’avoir des nombres spécifiques, seulement que les valeurs seront les mêmes, on peut aussi simplement utiliser getstate et setstate pour revenir à un état précédent :

save_state = random.getstate()  # Get the current state
print(random.randrange(0, 10))
# Out: 5
print(random.randrange(0, 10))
# Out: 8

random.setstate(save_state)     # Reset to saved state
print(random.randrange(0, 10))
# Out: 5
print(random.randrange(0, 10))
# Out: 8

Pour pseudo-randomiser à nouveau la séquence, vous “seed” avec “None” :

random.seed(None)

Ou appelez la méthode seed sans arguments :

random.seed()
