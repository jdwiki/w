Syntaxe #

  • plot(Y)

  • plot(Y,LineSpec)

  • parcelle(X,Y)

  • plot(X,Y,LineSpec)

  • plot(X1,Y1, X2,Y2, …, Xn,Yn)

  • plot(X1,Y1,LineSpec1, X2,Y2,LineSpec2, …, Xn,Yn,LineSpecn)

  • parcelle(___, nom, valeur)

  • h = parcelle(___)

Paramètres #

Paramètre Détails
X valeurs x
Y valeurs y
SpécificationLigne Style de ligne, symbole ponctuel et couleur, spécifiés sous forme de chaîne
Nom,Valeur Paires facultatives d’arguments nom-valeur pour personnaliser les propriétés de ligne
h poignée à l’objet graphique de ligne

Plusieurs lignes dans un seul tracé #

Dans cet exemple, nous allons tracer plusieurs lignes sur un seul axe. De plus, nous choisissons une apparence différente pour les lignes et créons une légende.

% create sample data
x = linspace(-2,2,100);                 % 100 linearly spaced points from -2 to 2
y1 = x.^2;
y2 = 2*x.^2;
y3 = 4*x.^2;

% create plot
figure;                                 % open new figure
plot(x,y1, x,y2,'--', x,y3,'-.');       % plot lines
grid minor;                             % add minor grid
title('Quadratic functions with different curvatures');
xlabel('x');
ylabel('f(x)');
legend('f(x) = x^2', 'f(x) = 2x^2', 'f(x) = 4x^2', 'Location','North');

Dans l’exemple ci-dessus, nous avons tracé les lignes avec une seule commande plot. Une alternative consiste à utiliser des commandes distinctes pour chaque ligne. Nous devons maintenir le contenu d’une figure avec hold on au plus tard avant d’ajouter la deuxième ligne. Sinon, les lignes précédemment tracées disparaîtront de la figure. Pour créer le même tracé que ci-dessus, nous pouvons utiliser ces commandes suivantes :

figure; hold on;
plot(x,y1);
plot(x,y2,'--');
plot(x,y3,'-.');

La figure résultante ressemble à ceci dans les deux cas :

Commandes personnalisées de couleurs et de styles de ligne #

Dans MATLAB, nous pouvons définir de nouveaux ordres personnalisés default, tels qu’un ordre de couleur et un ordre de style de trait. Cela signifie que les nouvelles commandes seront appliquées à toute figure créée après l’application de ces paramètres. Les nouveaux paramètres restent jusqu’à ce que la session MATLAB soit fermée ou que de nouveaux paramètres aient été définis.

** Ordre des couleurs et des styles de ligne par défaut **

Par défaut, MATLAB utilise deux couleurs différentes et uniquement un style de trait plein. Par conséquent, si plot est appelé pour dessiner plusieurs lignes, MATLAB alterne dans un ordre de couleurs pour dessiner des lignes de différentes couleurs.

[![entrez la description de l’image ici][1]][1]

Nous pouvons obtenir l’ordre des couleurs par défaut en appelant get avec un handle global 0 suivi de cet attribut DefaultAxesColorOrder :

>> get(0, 'DefaultAxesColorOrder')
ans =
         0    0.4470    0.7410
    0.8500    0.3250    0.0980
    0.9290    0.6940    0.1250
    0.4940    0.1840    0.5560
    0.4660    0.6740    0.1880
    0.3010    0.7450    0.9330
    0.6350    0.0780    0.1840

Commande personnalisée de couleur et de style de ligne

Une fois que nous avons décidé de définir un ordre de couleur personnalisé ET un ordre de style de ligne, MATLAB doit alterner les deux. Le premier changement appliqué par MATLAB est une couleur. Lorsque toutes les couleurs sont épuisées, MATLAB applique le style de ligne suivant à partir d’un ordre de style de ligne défini et définit un index de couleur sur 1. Cela signifie que MATLAB recommencera à alterner toutes les couleurs, mais en utilisant le style de ligne suivant dans son ordre. Lorsque tous les styles et couleurs de ligne sont épuisés, MATLAB commence évidemment à recommencer depuis le début en utilisant la première couleur et le premier style de ligne.

Pour cet exemple, j’ai défini un vecteur d’entrée et une fonction anonyme pour faciliter un peu le traçage des chiffres :

F = @(a,x) bsxfun(@plus, -0.2*x(:).^2, a);
x = (-5:5/100:5-5/100)';

Pour définir une nouvelle couleur ou un nouveau style de ligne, nous appelons la fonction set avec un handle global 0 suivi d’un attribut DefaultAxesXXXXXXX ; XXXXXXX peut être ColorOrder ou LineStyleOrder. La commande suivante définit un nouvel ordre de couleurs sur noir, rouge et bleu, respectivement :

set(0, 'DefaultAxesColorOrder', [0 0 0; 1 0 0; 0 0 1]);
plot(x, F([1 2 3 4 5 6],x));

[![entrez la description de l’image ici][2]][2]

Comme vous pouvez le voir, MATLAB alterne uniquement les couleurs car l’ordre des styles de ligne est défini sur une ligne continue par défaut. Lorsqu’un ensemble de couleurs est épuisé, MATLAB démarre à partir de la première couleur dans l’ordre des couleurs.

Les commandes suivantes définissent à la fois les ordres de couleur et de style de ligne :

set(0, 'DefaultAxesColorOrder', [0 0 0; 1 0 0; 0 0 1]);
set(0, 'DefaultAxesLineStyleOrder', {'-' '--'});
plot(x, F([1 2 3 4 5 6],x));

[![entrez la description de l’image ici][3]][3]

Désormais, MATLAB alterne différentes couleurs et différents styles de ligne en utilisant la couleur comme attribut le plus fréquent.
