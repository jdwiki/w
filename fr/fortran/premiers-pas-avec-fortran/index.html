Bonjour le monde #

Tout programme Fortran doit inclure end comme dernière instruction. Par conséquent, le programme Fortran le plus simple ressemble à ceci :

end

Voici quelques exemples de programmes “hello, world”:

print *, "Hello, world" 
end

Avec l’instruction ‘write’ :

write(*,*) "Hello, world"
end

Pour plus de clarté, il est maintenant courant d’utiliser l’instruction program pour démarrer un programme et lui donner un nom. L’instruction end peut alors faire référence à ce nom pour indiquer clairement à quoi il se réfère, et laisser le compilateur vérifier l’exactitude du code. De plus, tous les programmes Fortran doivent inclure une instruction “implicit none”. Ainsi, un programme Fortran minimal devrait ressembler à ceci :

program hello
  implicit none
  write(*,*) 'Hello world!'
end program hello

La prochaine étape logique à partir de ce point est de savoir comment voir le résultat du programme hello world. Cette section montre comment y parvenir dans un environnement de type Linux. Nous supposons que vous avez quelques notions de base sur les [commandes shell] [1], principalement vous savez comment accéder au terminal shell. Nous supposons également que vous avez déjà [configuré votre environnement fortran][2]. À l’aide de votre éditeur de texte préféré (notepad, notepad++, vi, vim, emacs, gedit, kate, etc.), enregistrez le programme hello ci-dessus (copier et coller) dans un fichier nommé “hello.f90” dans votre répertoire personnel. hello.f90 est votre fichier source. Accédez ensuite à la ligne de commande et accédez au répertoire (répertoire personnel ?) où vous avez enregistré votre fichier source, puis tapez la commande suivante :

>gfortran -o hello hello.f90

Vous venez de créer votre programme exécutable hello world. En termes techniques, vous venez de compiler votre programme. Pour l’exécuter, tapez la commande suivante :

>./hello

Vous devriez voir la ligne suivante imprimée sur votre terminal shell.

> Hello world!

Félicitations, vous venez d’écrire, de compiler et d’exécuter le programme “Hello World”.

[1] : https://www.wikiod.com/fr/shell/premiers-pas-avec-shell [2] : https://www.wikiod.com/fr/fortran/premiers-pas-avec-fortran#Installation ou configuration

Équation quadratique #

Aujourd’hui Fortran est principalement utilisé pour le calcul numérique. Cet exemple très simple illustre la structure de base du programme pour résoudre des équations quadratiques :

program quadratic
  !a comment

  !should be present in every separate program unit
  implicit none

  real :: a, b, c
  real :: discriminant
  real :: x1, x2

  print *, "Enter the quadratic equation coefficients a, b and c:"
  read *, a, b, c

  discriminant = b**2 - 4*a*c

  if ( discriminant>0 ) then

    x1 = ( -b + sqrt(discriminant)) / (2 * a)
    x2 = ( -b - sqrt(discriminant)) / (2 * a)
    print *, "Real roots:"
    print *, x1, x2

    ! Comparison of floating point numbers for equality is often not recommended. 
    ! Here, it serves the purpose of illustrating the "else if" construct. 
  else if ( discriminant==0 ) then

    x1 = - b / (2 * a)
    print *, "Real root:"
    print *, x1
  else

    print *, "No real roots."
  end if
end program quadratic

Insensibilité à la casse #

Les lettres majuscules et minuscules de l’alphabet sont équivalentes dans le Jeu de caractères Fortran. En d’autres termes, Fortran est insensible à la casse. Ce comportement contraste avec les langages sensibles à la casse, tels que C++ et bien d’autres.

Par conséquent, les variables ‘a’ et ‘A’ sont la même variable. En principe, on pourrait écrire un programme comme suit

pROgrAm MYproGRaM
..
enD mYPrOgrAM

C’est au bon programmeur d’éviter ces mauvais choix.

Installation ou configuration #

Fortran est un langage qui peut être compilé à l’aide de compilateurs fournis par de nombreux fournisseurs. Différents compilateurs sont disponibles pour différentes plates-formes matérielles et systèmes d’exploitation. Certains compilateurs sont des logiciels libres, certains peuvent être utilisés gratuitement et certains nécessitent l’achat d’une licence.

Le compilateur Fortran gratuit le plus courant est GNU Fortran ou gfortran. Le code source est disponible auprès de GNU dans le cadre de GCC, la collection de compilateurs GNU. Les binaires pour de nombreux systèmes d’exploitation sont disponibles sur https://gcc.gnu.org/wiki/GFortranBinaries. Les distributions Linux contiennent souvent gfortran dans leur gestionnaire de packages.

D’autres compilateurs sont disponibles, par exemple :

  • [EKOPath][1] par PathScale

  • [LLVM (système principal via DragonEgg)][2] * [Oracle Developer Studio][3]

  • [Compilateur Absoft Fortran] [4]

  • [Compilateur Intel Fortran] [5]

  • [Compilateur NAG Fortran] [6]

  • [Compilateurs PGI][7]

Sur les systèmes HPC, il existe souvent des compilateurs spécialisés disponibles par le fournisseur du système, comme par exemple les compilateurs [IBM][8] ou [Cray][9].

Tous ces compilateurs supportent le standard Fortran 95. Un aperçu du [statut Fortran 2003][10] et du [statut Fortran 2008][11] par divers compilateurs est proposé par le forum ACM Fortran et disponible dans le wiki Fortran.
