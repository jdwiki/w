Utilisation de .PHONY pour les cibles non-fichiers #

Utilisez .PHONY pour spécifier les cibles qui ne sont pas des fichiers, par exemple, clean ou mrproper.

Bon exemple

.PHONY: clean
clean:
    rm *.o temp

Mauvais exemple

clean:
    rm *.o temp

Dans le bon exemple make sait que clean n’est pas un fichier, donc il ne cherchera pas s’il est à jour ou non et exécutera la recette.

Dans le mauvais exemple, make recherchera un fichier nommé clean. S’il n’existe pas ou n’est pas à jour, il exécutera la recette, mais s’il existe et est à jour, la recette ne sera pas exécutée.

Utilisation de .PHONY pour les invocations récursives de la commande ‘make’ #

L’utilisation récursive de make signifie utiliser make comme commande dans un makefile. Cette technique est utile lorsqu’un grand projet contient des sous-répertoires, chacun ayant ses makefiles respectifs. L’exemple suivant vous aidera à comprendre l’avantage d’utiliser .PHONY avec un make récursif.

/main
     |_ Makefile
     |_ /foo
            |_ Makefile
            |_ ... // other files
     |_ /bar
            |_ Makefile
            |_ ... // other files
     |_ /koo
            |_ Makefile
            |_ ... // other files

Pour exécuter le makefile du sous-répertoire à partir du makefile de main, le makefile du main aurait une boucle comme indiqué ci-dessous (il existe d’autres façons d’y parvenir, mais cela sort du cadre du sujet actuel)

SUBDIRS = foo bar koo

subdirs:
        for dir in $(SUBDIRS); do \
          $(MAKE) -C $$dir; \
        done

Cependant, il y a des pièges avec cette méthode.

  1. Toute erreur détectée dans une sous-marque est ignorée par cette règle, elle continue to build the rest of the directories even when one fails.
  2. Capacité de Make à effectuer une exécution parallèle de plusieurs builds targets is not utilized since only one rule is used.

En déclarant les sous-répertoires comme cibles .PHONY (vous devez le faire car le sous-répertoire existe évidemment toujours, sinon il ne sera pas construit), ces problèmes peuvent être surmontés.

SUBDIRS = foo bar koo

.PHONY: subdirs $(SUBDIRS)

subdirs: $(SUBDIRS)

$(SUBDIRS):
        $(MAKE) -C $@
