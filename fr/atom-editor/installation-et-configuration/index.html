Installer Atom sous Windows #

Utilisation du programme d’installation officiel #

Téléchargez le programme d’installation depuis le site officiel. Il ajoutera automatiquement atom et apm (Atom Package Manager) à votre variable %PATH%.

Construire à partir de la source #

Conditions:

  • Node.js 4.4.x ou version ultérieure -Python 2.7.x
  • 7zip
  • Visual Studio (Une des versions ci-dessous)
    • Visual C++ Build Tools 2015
    • Visual Studio 2013 Update 5 (Express Edition or better)
    • Visual Studio 2015 (Community Edition or better)
  • Gite

Exécutez les commandes suivantes dans l’invite de commande :

cd C:/
git clone https://github.com/atom/atom.git
cd atom
script/build

Installer Atom sur Mac #

Installation à partir d’un zip #

  1. Téléchargez le fichier zip atom-mac.zip à partir du référentiel Atom GitHub ici
  2. Décompressez le fichier en double-cliquant dessus dans le Finder
  3. Faites glisser l’application Atom dans votre dossier “Applications”
  4. Exécutez l’application Atom.

Construire à partir de la source #

Conditions:

  • macOS 10.8 ou supérieur
  • Node.js 4.4x ou version ultérieure
  • npm 3.10.x ou version ultérieure -Xcode

Installation:

git clone https://github.com/atom/atom.git
cd atom
script/build

Après la construction, installez avec script/build --install

Installer Atom sur Linux #

Installation à partir d’un package #

Debian, Ubuntu, etc. #

$ sudo dpkg -i atom-amd64.deb
$ sudo apt-get -f install

RedHat Enterprise, CentOS, Oracle Linux, Scientific Linux, etc. #

$ sudo yum install -y atom.x86_64.rpm

Fedora (gestionnaire de paquets DNF) #

$ sudo dnf install -y atom.x86_64.rpm

SUSE (gestionnaire de paquets Zypp) #

$ sudo zypper in -y atom.x86_64.rpm

Construire à partir de la source #

Conditions:

  • OS avec architecture 64 ou 32 bits
  • Chaîne d’outils C++ 11
  • Gite
  • Node.js 4.4x ou version ultérieure
  • npm 3.10.x ou version ultérieure
  • En-têtes de développement de porte-clés GNOME

Exécutez les commandes suivantes :

git clone https://github.com/atom/atom.git
cd atom
script/build

Pour obtenir des instructions spécifiques relatives à une seule distribution Linux, lisez ces.
