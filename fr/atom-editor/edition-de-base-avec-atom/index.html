Arborescence de fichiers interactive #

Afin de suivre la structure des fichiers de vos projets, Atom, comme de nombreux éditeurs de texte et IDE, utilise un modèle d’arborescence de fichiers. Ces arborescences indiquent les emplacements et les noms de vos fichiers et répertoires. Pour basculer l’arborescence entre visible et masqué, les touches Ctrl+\ peuvent être utilisées (⌘+\ pour MacOS). Cette arborescence comprend également de nombreuses opérations pour les fichiers et les répertoires, comme indiqué ci-dessous :

[![Menu contextuel des options de l’arborescence des fichiers Atom][1]][1]

Opération Descriptif

Split Down Divise l’éditeur en deux volets avec le fichier sélectionné en bas
Split Left Divise l’éditeur en deux volets avec le fichier sélectionné à gauche
Split Right Divise l’éditeur en deux volets avec le fichier sélectionné sur
Rechercher dans le répertoire Ouvre l’outil de recherche et de remplacement pour rechercher le fichier ou le répertoire sélectionné
Nouveau fichier Crée un nouveau fichier dans la portée du répertoire où le clic s’est produit
Nouveau dossier Crée un nouveau dossier dans la portée du répertoire où le clic s’est produit
Renommer Change le nom du fichier ou du répertoire
Dupliquer Crée une copie exacte du fichier ou du répertoire
Supprimer Supprime le fichier ou le répertoire
Copier Copie le fichier ou le répertoire dans le presse-papiers
Coller Colle un fichier ou un répertoire copié depuis le presse-papiers
Ajouter un dossier de projet Permet de sélectionner un autre répertoire à inclure à la racine de l’arborescence

Les fichiers cachés (sauf indication contraire dans les paramètres d’Atom) s’afficheront avec des noms de fichiers ombrés. Un exemple courant est les données de configuration du référentiel de GitHub dans le répertoire .git.

Ouverture de fichiers et de répertoires #

Avec d’autres éditeurs de texte plus avancés, Atom permet aux développeurs d’ouvrir un seul fichier ou un répertoire.

Ouvrir des fichiers #

Pour ouvrir des fichiers avec Atom, utilisez Fichier > Ouvrir un fichier… dans le menu comme indiqué ci-dessous :

[![Option Ouvrir un fichier dans le menu Atom][1]][1]

ou utilisez le raccourci clavier plus rapide : Ctrl+O (Pour Mac OS : ⌘+O). Cela ouvrira un explorateur de fichiers (Finder pour Mac OS) à partir duquel vous pouvez sélectionner un fichier à ouvrir, ou pour sélectionner plusieurs fichiers, utilisez la touche Ctrl (Pour Mac ⌘) enfoncée tout en cliquant sur d’autres fichiers ou maintenez la touche Maj enfoncée tout en sélectionnant d’autres fichiers pour sélectionner une plage. Lorsque vous avez sélectionné les fichiers que vous souhaitez ouvrir, appuyez sur le bouton “Ouvrir” de l’explorateur de fichiers. Atom, en tant qu’éditeur de texte, ne choisit de gérer que les fichiers de moins de 2 mégaoctets.

Ouverture de répertoires #

Surtout pour les projets, la fonction d’ouverture de répertoire d’Atom peut être très utile. Pour ce faire, vous pouvez soit utiliser l’option du menu fichier d’Atom :

[![Option Ouvrir le dossier dans le menu Atom][2]][2]

ou utilisez le raccourci clavier Ctrl+Shift+O (Pour Mac OS : ⌘+Shift< /kbd>+O). L’ouverture de répertoires vous permettra d’accéder à d’autres répertoires et fichiers sous le répertoire racine :

[![Exemple d’arborescence de fichiers du répertoire ouvert][3]][3]

Trouver et remplacer #

La fonction de recherche et de remplacement d’Atom fonctionne de deux manières, l’une n’opère localement que sur le fichier dans lequel vous vous trouvez et l’autre sur un ensemble de fichiers ou de répertoires.

Pour ouvrir l’outil de recherche et de remplacement pour un seul fichier, appuyez sur Ctrl+F (pour Mac OS, utilisez ⌘+F</ kbd>). Entrez dans le premier espace la chaîne que vous recherchez. Appuyez sur la touche Entrée pour rechercher toutes les instances de la chaîne. À droite du bouton Rechercher se trouvent les boutons d’option regex, sensible à la casse, en sélection et mot entier. Le bouton Utiliser Regex vous permet de rechercher des caractères regex tels que \n, \t, \r et les déclarations regex /^[a-z0-9_-]{3,16} $/. Le bouton Case Sensitive - lorsqu’il est actif - ne trouvera que les chaînes avec la même casse (majuscules). L’option Uniquement dans la sélection ne trouvera que les instances de la chaîne dans les sections en surbrillance du fichier. L’option Whole Word ne trouvera que des instances délimitées, pas lorsque la chaîne fait partie d’une plus grande portion. Cliquer sur le bouton Remplacer prendra la première instance trouvée avec la méthode Rechercher et la remplacera par le contenu du champ de remplacement (même s’il est vide). Cliquer sur le bouton Remplacer tout remplacera toutes les instances trouvées avec la méthode Rechercher et les remplacera toutes en même temps par le contenu du champ de remplacement.
