Syntaxe #

  • contraste(<reference-color>, <output-for-light-refcolor>, <output-for-dark-refcolor>, <threshold>)
  • éclaircir(<reference-color>, <amount>, <method>)
  • assombrir(<reference-color>, <amount>, <method>)

Paramètres #

Paramètre Détails
couleur-de-référence La couleur basée sur laquelle l’opération de couleur doit être effectuée.
sortie-pour-lumière-ref-couleur La valeur de couleur qui doit être sortie lorsque la couleur de référence est une couleur sombre. Ceci est facultatif et la valeur par défaut est blanc.
sortie pour dark-ref-color La valeur de couleur qui doit être sortie lorsque la couleur de référence est une couleur claire. Ceci est facultatif et la valeur par défaut est noire.
seuil Il s’agit d’une valeur en pourcentage qui définit quand la couleur de référence est considérée comme une couleur foncée et quand elle est considérée comme une couleur claire. La comparaison des couleurs est effectuée à l’aide de valeurs de luminance corrigées gamma. Ceci est facultatif et la valeur par défaut est 43 %
montant Il s’agit d’une valeur en pourcentage qui spécifie la quantité d’assombrissement ou d’éclaircissement de la couleur de référence.
méthode Cela peut être “absolu” ou “relatif” et définit si l’ajustement doit être relatif à la valeur actuelle ou non. C’est un champ facultatif et la valeur par défaut est absolue.

Contraste #

Dans la fonction de contraste, la sortie pour la couleur de référence sombre et la sortie pour la couleur de référence claire peuvent être fournies dans n’importe quel ordre. La fonction identifie automatiquement la couleur sombre (à utiliser lorsque la couleur de référence est claire) et la couleur claire (à utiliser lorsque la couleur de référence est sombre) en fonction de leurs propres valeurs de luminance.

Réglage de la couleur du texte en fonction de l’obscurité ou de la luminosité de la couleur d’arrière-plan #

#demo-element {
  background: @theme-color;
  color: contrast(@theme-color, black, white, 50%);
}

@theme-color: red;

L’exemple ci-dessus définira la couleur du texte de l’élément sur white si la background-color est sombre et vice-versa. Ceci est réalisé à l’aide de la fonction d’opération de couleur contrast().

La fonction de contraste accepte quatre paramètres dont le premier est la couleur de référence sur la base de laquelle la sortie doit être fournie. Une couleur sombre qui doit être émise lorsque la couleur de référence est claire et une couleur claire qui doit être émise lorsque la couleur de référence est sombre. Le seuil est un pourcentage qui définit quand une couleur est considérée comme une couleur claire et quand elle est foncée.

Une démo de cet exemple peut être trouvée ici.

Définir une teinte plus foncée ou plus claire de la couleur d’arrière-plan pour une autre propriété #

Assombrir #

#demo {
  @refcolor: #f0b9b8;
  background: @refcolor;
  border: 1px solid darken(@refcolor, 25%);
}

Le code ci-dessus utilise la fonction darken() pour définir la couleur de la bordure comme une nuance 25 % plus sombre que la couleur de référence (qui est également la couleur d’arrière-plan).

Le compilateur Less ne peut pas lire la valeur affectée à une propriété et l’utiliser pour une autre. Par conséquent, une variable distincte doit être définie. Cette variable sera directement affectée à la couleur de fond alors que pour la couleur de bordure, la fonction darken est utilisée pour obtenir une teinte plus foncée.

Alléger #

#demo {
  @refcolor: #f0b9b8;
  background: @refcolor;
  box-shadow: 2px 2px 0px lighten(@refcolor, 5%);
}

Le code ci-dessus utilise la fonction lighten() pour définir la couleur de l’ombre comme une teinte 5 % plus claire que la couleur de référence (qui est également la couleur d’arrière-plan).

Changer l’opacité #

Il est possible de changer l’opacité d’une couleur avec la fonction fade().

fade() prend 2 paramètres :

  • une couleur
  • opacité (en %)

Exemple:

@elegant: #eeffgg;

.light-elegant {
    background-color: fade(@elegant, 20%);
}

<div class="light-elegant">
    I have a 20% elegant background!
</div>
