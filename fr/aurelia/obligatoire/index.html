Liaison aux cases à cocher #

Cases à cocher de base

export class MyViewModel {
    favoriteColors = [];
    colors = ['Red', 'Yellow', 'Pink', 'Green', 'Purple', 'Orange', 'Blue'];
}

<template>
  <label repeat.for="color of colors">
    <input type="checkbox" value.bind="color" checked.bind="favoriteColors" />
    ${color}
  </label>
  
  <p>Favourite colors:</p>
  <ul if.bind="favoriteColors">
    <li repeat.for="color of favoriteColors">${color}</li>
  </ul>
</template>

** Cases à cocher avec des tableaux d’objets **

export class MyViewModel {
    people = [];
    selectedPeople = [];
    
    constructor() {
        this.people = [
            {name: 'John Michaels'},
            {name: 'Gary Stevens'},
            {name: 'Carrie Smitch'},
            {name: 'Jesus Wohau'}
        ];
    }
}

<template>
  <label repeat.for="person of people">
    <input type="checkbox" model.bind="person" checked.bind="selectedPeople" />
    ${person.name}
  </label>
  
  <p>Selected people:</p>
  <ul if.bind="selectedPeople">
    <li repeat.for="person of selectedPeople">${person.name}</li>
  </ul>
</template>

Case à cocher avec un booléen

export class MyViewModel {
    agreeToTerms = false;
}

<template>
  <label><input type="radio" name="terms" model.bind="true" checked.bind="agreeToTerms" />Yes</label>
  <label><input type="radio" name="terms" model.bind="false" checked.bind="agreeToTerms" />No</label>
  <br><br>
  <strong>${agreeToTerms ? 'I agree' : 'I disagree'}</strong>
</template>

Liaison pour sélectionner des éléments #

Tableau de chaînes

Lors de la sélection d’une valeur dans la liste déroulante de sélection et de la fourniture d’un tableau de chaînes, la valeur sélectionnée sera liée à la propriété de valeur de l’élément de sélection en tant que chaîne que nous pouvons afficher à l’aide de l’interpolation de chaîne.

export class MyViewModel {
    animals = [];
    favouriteAnimal = null;

    constructor() {
        this.animals = [
            'Cat',
            'Dog',
            'Fish',
            'Rabbit',
            'Tiger',
            'Bat'
        ];
    }
}

<template>
    ${favouriteAnimal}
    <select name="animals" value.bind="favouriteAnimal">
        <option repeat.for="animal of animals">${animal}</option>
    </select>
</template>

Tableau d’objets

Contrairement à l’exemple ci-dessus, lors de la fourniture d’un tableau d’objets, lorsqu’une valeur est sélectionnée dans une liste déroulante, le modèle lié à cette option particulière est l’objet fourni.

export class MyViewModel {
    animals = [];
    favouriteAnimal = null;

    constructor() {
        this.animals = [
            {label: 'Cat', value: 99},
            {label: 'Dog', value: 493},
            {label: 'Fish', value: 934839200},
            {label: 'Rabbit', value: 8311},
            {label: 'Tiger', value: 22},
            {label: 'Bat', value: 3711}
        ];
    }
}

<template>
    <p>Favourite animal ID: ${favouriteAnimal.value}</p>
    <p>Favourite animal name: ${favouriteAnimal.label}</p>
    <select name="animals" value.bind="favouriteAnimal">
        <option repeat.for="animal of animals" model.bind="animal">${animal.label}</option>
    </select>
</template>

Styles de reliure #

Liaison à l’attribut style natif du navigateur à l’aide d’Aurelia. Si vous utilisez l’interpolation de chaîne, vous devez utiliser l’alias css pour que le style fonctionne dans Internet Explorer.

Chaîne de style

export class MyViewModel {
  constructor() {
    this.styleString = 'color: #F2D3D6; background-color: #333';
  }
}

<template>
  <div style.bind="styleString"></div>
</template>

Objet de style

export class MyViewModel {
  constructor() {
    this.styles = {color: '#F2D3D6', 'background-color': '#333'};
  }
}

<template>
  <div style.bind="styles"></div>
</template>

Interpolation de chaîne

Très similaire à la liaison de chaîne ci-dessus, cela vous permet d’utiliser l’interpolation de chaîne pour lier les styles à la place. Si l’une des valeurs change, elle sera mise à jour dans la vue en conséquence.

Remarque : pour la compatibilité avec Internet Explorer, nous utilisons l’alias “css” pour lier les styles. Cela garantit que l’interpolation de chaîne fonctionne dans Internet Explorer.

export class MyViewModel {
    color = 'red';
    height = 350;
    width = 350;
}

<template>
    <div css="width: ${width}px; height: ${height}px; color:${color}">My Text</div>
</template>

Afficher et masquer conditionnellement un élément HTML #

Lors de l’utilisation de show.bind, l’élément reste dans la page et est soit caché, soit visible grâce à l’utilisation de display:none ou display:block dans les coulisses.

export class MyViewModel {
    isVisible = false;
}

<template>
    <div show.bind="isVisible"><strong>I can be both hidden or visible, but not at the same time</strong></div>
</template>

Ajouter ou supprimer conditionnellement un élément HTML #

Contrairement à show.bind lors de l’utilisation de if.bind, l’élément sera supprimé de la page si la valeur de liaison fournie est false ou ajouté à la page si la valeur est true.

export class MyViewModel {
    isVisible = false;
}

<template>
    <div if.bind="isVisible"><strong>If you can read this, I am still here.</strong></div>
</template>

Éléments groupés

Parfois, il peut arriver que vous souhaitiez ajouter ou supprimer tout un groupe d’éléments à la fois. Pour cela, nous pouvons utiliser un élément <template> pour afficher ou masquer des éléments supplémentaires sans avoir besoin d’un élément d’espace réservé comme un DIV.

export class MyViewModel {
    hasErrors = false;
    errorMessage = '';
}

<template if.bind="hasErrors">
    <i class="icon error"></i>
    ${errorMessage}
</template>
