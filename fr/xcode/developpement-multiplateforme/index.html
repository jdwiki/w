CibleConditions #

L’en-tête système TargetConditionals.h définit plusieurs macros que vous pouvez utiliser à partir de C et Objective-C pour déterminer la plate-forme que vous utilisez.

#import <TargetConditionals.h>  // imported automatically with Foundation

- (void)doSomethingPlatformSpecific {
#if TARGET_OS_IOS
    // code that is compiled for iPhone / iPhone Simulator
#elif TARGET_OS_MAC && !TARGET_OS_IPHONE
    // code that is compiled for OS X only
#else
    // code that is compiled for other platforms
#endif
}

Les valeurs des macros sont :

Lors de l’utilisation des SDK iOS 9.1, tvOS 9.0, watchOS 2.0, OS X 10.11 ou plus récents :
