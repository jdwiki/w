Remplacer le pack de langue i18n

  • Syntaxe

Sur cette page

  • Syntaxe
  • Exemple de syntaxe du package de langue override i18n

Syntaxe #

  • <Vendor Namespace> - Ici, l’espace de noms du thème personnalisé du fournisseur ou de l’espace de noms de thème intégré C.-à-d. Magento/Luma Ici, luma est l'espace de noms du fournisseur

  • <répertoire du package de langue> - Ici le répertoire du package de langue comme en_us ou nl_nl ou en_gb

  • <description du package de langue> - Ajoutez ici une description du package comme “English Us Package”

  • <code du package de langue> - Ici le code du package de langue I.E en_US ou nl_NL ou en_GB

Après avoir créé les fichiers et répertoires ci-dessus, language_package_code.csv ira dans le répertoire Vendor Namespace

Exemple

/app/i18n/luma/en_us/en_US.csv

ou

/app/i18n/luma/en_gb/en_GB.csv

ou

/app/i18n/luma/nl_NL/nl_NL.csv

Exemple de syntaxe du package de langue override i18n #

/app/i18n/<Vendor Namespace>/<répertoire du package de langue>/composer.json

{
    "name": "<vendor namespance>/<language package directory>",
    "description": "<language package description>",
    "version": "100.0.1",
    "license": [
        "OSL-3.0",
        "AFL-3.0"
    ],
    "require": {
        "magento/framework": "100.0.*"
    },
    "type": "magento2-language",
    "autoload": {
        "files": [
            "registration.php"
        ]
    }
}

/app/i18n/<Vendor Namespace>/<package de langue>/language.xml

<?xml version="1.0"?>
<language xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:noNamespaceSchemaLocation="urn:magento:framework:App/Language/package.xsd">
    <code><language package code></code>
    <vendor><vendor namespace></vendor>
    <package><language package directory></package>
</language>

/app/i18n/<Vendor Namespace>/<package de langue>/registration.php

<?php
\Magento\Framework\Component\ComponentRegistrar::register(
    \Magento\Framework\Component\ComponentRegistrar::LANGUAGE,
    '<vendor namespance>_<language package directory>',
    __DIR__
);
