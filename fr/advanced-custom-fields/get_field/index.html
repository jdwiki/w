Syntaxe #

  • get_field($field_name, $post_id, $format_value);

Paramètres #

Paramètre Détails
$nom_champ (chaîne) Le nom du champ que vous demandez. Lors de la configuration de votre groupe de champs, il s’agit du paramètre “Nom du champ”.
$post_id (entier) Il s’agit de l’ID de la publication à partir de laquelle vous demandez le nom du champ. Il peut également s’agir d’‘options’ ou d’une référence de taxonomie. Cela peut être laissé vide à l’intérieur de la boucle WordPress, si vous demandez l’ID de publication/page actuel.
$format_value Déterminez si vous souhaitez formater la valeur chargée à partir de la base de données. La valeur par défaut est true. Non requis.

get_field() contre the_field()

the_field() affiche automatiquement la valeur du champ sur votre page, contrairement à get_field(). Utilisez get_field() lors de l’affectation de valeurs de champ à des variables ou lors de la manipulation du contenu renvoyé dans votre code.

Nettoyer la sortie de get_field() #

C’est une bonne idée de nettoyer la sortie de get_field(), en particulier lors de l’utilisation frontale des champs de champs personnalisés avancés (avec acf_form()). Sinon, votre site est probablement vulnérable aux attaques de script intersite (XSS).

La fonction suivante vous permet d’utiliser

echo get_field_escaped('my_custom_field', $post_id, true);

à la place de

echo get_field('my_custom_field', $post_id, true);

La fonction utilise esc_html par défaut, mais vous permet de le modifier en tant que quatrième paramètre

echo get_field_escaped('url', $post_id, true, 'esc_url');

Ajoutez ce qui suit à functions.php pour activer la fonction :

/**
 * Helper function to get escaped field from ACF
 * and also normalize values.
 *
 * @param $field_key
 * @param bool $post_id
 * @param bool $format_value
 * @param string $escape_method esc_html / esc_attr or NULL for none
 * @return array|bool|string
 */
function get_field_escaped($field_key, $post_id = false, $format_value = true, $escape_method = 'esc_html')
{
    $field = get_field($field_key, $post_id, $format_value);
 
    /* Check for null and falsy values and always return space */
    if($field === NULL || $field === FALSE)
        $field = '';
 
    /* Handle arrays */
    if(is_array($field))
    {
        $field_escaped = array();
        foreach($field as $key => $value)
        {
            $field_escaped[$key] = ($escape_method === NULL) ? $value : $escape_method($value);
        }
        return $field_escaped;
    }
    else
        return ($escape_method === NULL) ? $field : $escape_method($field);
}

Source : https://snippets.khromov.se/sanitizing-and-securing-advanced-custom-fields-output/

En savoir plus sur les différentes options de désinfection dans le WordPress Codex : https://codex.wordpress.org/Data_Validation#Output_Sanitization

Exemple simple #

<?php echo get_field('my_field_name'); ?>

Cela fera écho à la valeur du champ “my_field_name” du message actuel.

Obtenir la valeur du champ à partir d’un autre article #

<?php echo get_field('my_field_name', 123); ?>

Cela fera écho à la valeur de “my_field_name” de la publication avec 123 comme ID.

Obtenir la valeur du champ à partir d’une page d’options #

<?php echo get_field('my_field_name', 'option'); ?>

Cela fera écho à la valeur de “my_field_name” de la page d’options créée via ACF.

Vérifier si le champ existe #

<?php if( get_field('my_field_name') ){ ?>
    <?php echo get_field('my_field_name'); ?>
<?php }; ?>

Cela n’affichera le champ que si le contenu existe en fonction du type de contenu (c’est-à-dire qu’une image est téléchargée dans le champ, du texte est saisi, il est sélectionné, etc.).

Obtenir un champ avec des boutons radio #

Voici un exemple sur la façon d’utiliser ACF pour produire différemment en fonction des options (sélections de couleurs dans ce cas). Bien que vous puissiez utiliser <?php echo get_field('color_options'); ?> pour afficher la valeur directement, vous pouvez également modifier le balisage en fonction de la sélection.

<?php $option = get_field('color_options'); ?>
    
    <?php if( $option == 'red' ){ ?>
    
    <?php } else if( $option == 'blue' ){ ?>
        
    <?php } else if( $option == 'green' ){ ?>
        
    <?php } else if( $option == 'yellow' ){ ?>

<?php } ?>
