La publication a ignoré le contenu de personnalisation déjà appliqué

  • Publier avec nettoyage depuis l’écran de personnalisation
  • Publier avec nettoyage depuis l’intérieur d’un projet de personnalisation

Sur cette page

  • Publier avec nettoyage depuis l’écran de personnalisation
  • Publier avec nettoyage depuis l’intérieur d’un projet de personnalisation

Lors de la publication d’un projet de personnalisation, il se peut que certains éléments soient ignorés parce qu’ils sont déjà appliqués. Ex:

EntityEndpoint EntityEndpoint#6.00.001§DefaultPlus(ignoré, déjà appliqué)

Cela peut se produire pour tous les éléments contenus dans la base de données. Ex: Demandes génériques, rapports, nœuds de plan de site, scripts de base de données, paramètres régionaux du système, scénarios d’importation/exportation, filtres partagés, droits d’accès, wikis, points de terminaison de service Web et rapports analytiques.

Publier avec nettoyage depuis l’écran de personnalisation #

  1. Vous devez évidemment sélectionner le projet que vous souhaitez publier.
  2. Vous devez cliquer sur la petite flèche juste à côté du bouton “Publier”.
  3. Vous devez cliquer sur l’option “Publier à plusieurs sociétés”.

[![entrez la description de l’image ici][1]][1]

  1. Sur le panneau intelligent qui apparaîtra, vous devez sélectionner les entreprises dont vous souhaitez publier le(s) projet(s). Une seule entreprise est également une possibilité.
  2. Cochez la case indiquant “Publier avec nettoyage”, cela garantira de réappliquer tous les éléments présents dans le projet de personnalisation en remplaçant ceux déjà présents par leur version plus récente.

[![entrez la description de l’image ici][2]][2]

[1] : https://i.stack.imgur.com/28g4V.jpg [2] : https://i.stack.imgur.com/UTwmQ.jpg

Publier avec nettoyage depuis l’intérieur d’un projet de personnalisation #

  1. Ouvrez le projet de personnalisation que vous souhaitez publier avec cette méthode.
  2. Ouvrez le menu de publication en haut et sélectionnez l’option “Publier avec nettoyage”.

[![entrez la description de l’image ici][1]][1]

[1] : https://i.stack.imgur.com/O0j3Z.jpg

  • Veuillez noter que tous les projets de personnalisation sélectionnés sur l’écran de personnalisation seront republiés même si vous n’êtes qu’à l’intérieur d’un seul projet.
