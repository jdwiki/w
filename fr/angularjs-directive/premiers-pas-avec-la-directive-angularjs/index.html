Construire un composant réutilisable #

Les directives peuvent être utilisées pour créer des composants réutilisables. Voici un exemple de composant “boîte utilisateur”:

userBox.js

angular.module('simpleDirective', []).directive('userBox', function() {
  return {
    scope: {
        username: '=username',
        reputation: '=reputation'
    },
    templateUrl: '/path/to/app/directives/user-box.html'
  };
});

Contrôleur.js

var myApp = angular.module('myApp', ['simpleDirective']);

myApp.controller('Controller', function($scope) {
    $scope.user = "John Doe";
    $scope.rep = 1250;
});

maPage.js

<html lang="en" ng-app="myApp">
  <head>
    <script src="/path/to/app/angular.min.js"></script>
    <script src="/path/to/app/controllers/Controller.js"></script>
    <script src="/path/to/app/directives/userBox.js"></script>
  </head>

  <body>
  
    <div ng-controller="Controller">
        <user-box username="user" reputation="rep"></user-box>
    </div>
    
  </body>
</html>

user-box.html

<div>{{username}}</div>
<div>{{reputation}} reputation</div>

Votre première directive #

Notre première directive d’élément ne fera pas grand-chose : elle calculera simplement 2+2 et sera appelée en html comme ceci :

<my-calculator></my-calculator>

Notez que le nom de la directive est myCalculator (en CamelCase), mais en html, il est utilisé comme my-calculator (en lisp-case).

Puisque nous voulons que notre directive soit utilisée comme élément html, nous utiliserons restrict: 'E'.

Chaque directive a le modèle qui sera compilé et inséré. Notre directive est très simple, nous allons donc insérer notre html en tant que chaîne dans un paramètre template.

// directives/my-calculator.js

angular.module('exampleApp', [])
.directive('myCalculator', function() {
  return {
    restrict: 'E',
    template: '<span> My directive can calculate 2+2: {{2+2}} </span>'
  };
});

HTML

<!DOCTYPE html>
<html ng-app="exampleApp">

  <head>
    <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.5.6/angular.min.js"></script>
    <script src="my-calculator.js"></script>
  </head>

  <body>
    Here is my first directive: 
    <my-calculator></my-calculator>
  </body>

</html>

Le résultat ressemblera à ceci :

Voici ma première directive : Ma directive peut calculer 2+2 : 4

Si vous voulez jouer avec l’exemple en direct, allez sur [plunkr][1].

[1] : https://plnkr.co/edit/p9yVSCf7wp8014il4AMb?p=preview

Installation ou configuration #

Les directives sont fournies avec la bibliothèque AngularJS elle-même. Un exemple de directive peut être créé comme :

angular.module('simpleDirective', [])
.directive('helloData', function() {
  return {
    template: 'Hello, {{data}}'
  };
});

Et peut être utilisé comme :

JS :

angular.module('app', ['simpleDirective'])
.controller('Controller', ['$scope', function($scope) {
  $scope.data = 'World';
}])

HTML

<div ng-controller="Controller">
  <div hello-data></div>
</div>

Sera compilé comme :

Bonjour, Monde

Message contextuel Succès/Erreur utilisant la fonction de lien simple #

La fonction de lien est le meilleur moyen dans les directives personnalisées de manipuler le DOM. Il prend trois attributs en entrée (portée, élément, attribut) dans l’ordre

portée : son objet de portée locale de la directive.

element : élément html sur lequel la directive est utilisée.

attribut : il donne accès à tous les attributs utilisés dans l’élément référencé.

// on success call or similarly error, warning, info in controller
    $scope.message={
                text: "Saved Successfully",
                type: "SUCCESS"
                };    
                
    <user-info msg="message"> </user-info>   //in html



    var mainApp = angular.module("mainApp", []);
     mainApp.directive('userInfo', function() {
        var directive = {};
        directive.restrict = 'E';
        
        directive.scope = {
           message : "=msg"
        },
        
        directive.link = function(scope, element, attributes) {
            if(scope.message.type==='SUCCESS')
              scope.message.text = 'SUCCESS: '+scope.message.text+' !';
            else if(scope.message.type==='ERROR')  
              scope.message.text = 'ERROR: '+scope.message.text+' !';
            else if(scope.message.type==='WARNING')  
              scope.message.text = 'WARNING: '+scope.message.text+' !'
            else if(scope.message.type==='INFO')  
              scope.message.text = 'INFO: '+scope.message.text+' !'
          
            element.on('click', function(event) {     //on click of div pop-up will smoothly close
                      $(this).fadeOut();   
                      });
           },
           directive.template = '<div ng-class={{message.type}}>'+            // one can create different bg-color as per type of message and width/height
                                '<div class="message-text">{{message.text}}<div>'+   //message text will be printed
                                '<div>';
           
        return directive;
     });
