Cette section fournit une vue d’ensemble des glyphicons Bootstrap et décrit comment utiliser les glyphicons.

Comment utiliser les glyphicons #

Twitter Bootstrap prend en charge les icônes appelées glyphicons et elles peuvent être utilisées avec toutes les balises HTML.

Toutes les icônes nécessitent une classe de base et une classe d’icônes individuelles.

Gardez à l’esprit que les classes d’icônes ne peuvent pas être directement combinées avec d’autres composants, utilisez donc toujours la balise interne <span></span>.

Si votre code HTML contient des éléments enfants internes, vous ne pouvez pas utiliser de classes d’icônes pour cette balise particulière.

[Exemples][1]

Par exemple, vous créez un bouton d’amorçage, la syntaxe de ce bouton devrait ressembler à ceci :

<button type="button" class="btn btn-default btn-lg">
   Star
</button>

Ainsi, dans l’exemple ci-dessus, un simple bouton d’amorçage est créé, mais vous souhaitez maintenant ajouter un glyphicon dans ce bouton. Pour cela, ajoutez simplement un élément <span> à l’intérieur d’une balise <button>. Comme ça:

<button type="button" class="btn btn-default btn-lg">
       <span class="glyphicon glyphicon-star" aria-hidden="true"></span>Star
</button>
