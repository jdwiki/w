Utilisation de SSIS pour extraire des données d’un fichier CSV et les insérer dans une table SQL Server

Lire à partir d’un fichier CSV et insérer des données dans un tableau #

Tout d’abord, vous devez préparer l’environnement en créant la table SQL Server et le fichier CSV.

Exécutez le script ci-dessous dans SQL Server pour créer la table SQL sur une nouvelle base de données ou sur une existante. Pour cet exemple, j’ai utilisé ma base de données “TrainingDB”.

/* Creates table for Students.csv */
CREATE TABLE StudentDetails
(
    Surname varchar(50),
    Firstname varchar(50),
    DateofBirth datetime,
    PostCode varchar(50),
    PhoneNumber varchar(50),
    EmailAddress varchar(50)
)

Créez maintenant un fichier CSV avec les données ci-dessous.

Nom Prénom Date de naissance Code postal N° de téléphone Adresse e-mail
Bonga Fred 24-02-1990 SA1 5XR 08100900647 snow.fd@gmail.com
forgeron Gill 05-08-1992 RMT 12TY 08200900793 tastey0256@yahoo.co.uk
Taylor Jeanne 01-12-1979 PM2E 3NG 09600900061 jane0098@aol.com
Marron Jean 06-10-1986 CQ7 1JK 08200900063 johnbrown@hotmail.com

Lewis marque 30-09-1975 DN28 2UR 08000900200 m.lewis@mail.net

Vous pouvez le copier dans Excel et l’enregistrer en tant que fichier CSV.

Après avoir lancé Microsoft Visual Studio, accédez à Fichier - Nouveau - Projet, comme indiqué ci-dessous. [![entrez la description de l’image ici][1]][1]

Sous le groupe Business Intelligence, sélectionnez Integration Services et Integration Services Project. Entrez un nom pour le projet et un nom pour la solution, par exemple “Load CSV”. Vous pouvez cocher la case “Créer un répertoire pour la solution” si vous souhaitez créer une solution. [![entrez la description de l’image ici][2]][2] Cliquez sur OK

Sur le côté droit de l’écran affiché, dans la fenêtre “Explorateur de solutions”, changez le nom du package par défaut en “Charger le fichier CSV dans le tableau” [![entrez la description de l’image ici][3]][3]

Sur le côté gauche de l’écran, dans la barre d’outils SSIS, faites glisser le “Flux de données” vers la fenêtre “Flux de contrôle” et renommez la tâche en “Charger le fichier CSV”

Ensuite, vous devez configurer les gestionnaires de connexions pour le fichier CSV et la table SQL Server, également appelés respectivement source et destination. En bas de l’écran, sous Gestionnaires de connexion, faites un clic droit et sélectionnez “Nouvelle connexion de fichier plat” et configurez le gestionnaire de connexion de fichier plat comme indiqué ci-dessous. [![entrez la description de l’image ici][4]][4]

Entrez un nom de gestionnaire de connexion approprié et spécifiez le chemin d’accès au fichier Students.csv. Cliquez sur OK.

Pour le gestionnaire de connexion de la table, refaites un clic droit dans la fenêtre Gestionnaires de connexion et cliquez sur « Nouvelle connexion OLE DB ». Cliquez sur Nouveau et spécifiez le nom du serveur et le nom de la base de données contenant la table StudentsDetail.

[![entrez la description de l’image ici][5]][5]

Vous pouvez tester la connexion en cliquant sur “Tester la connexion”, puis cliquez sur OK et à nouveau sur OK. Vous devriez maintenant avoir les 2 gestionnaires de connexion en bas de l’écran.

Faites glisser la “Source de fichier plat” de la boîte à outils SSIS dans la fenêtre “Flux de données” et renommez-la en “Fichier CSV”. [![entrez la description de l’image ici][6]][6]

Double-cliquez sur cette source et sélectionnez le gestionnaire de connexion « Student CSV File ». Cliquez sur Colonnes sur le côté gauche de l’écran pour revoir les colonnes du fichier. Cliquez sur OK.

Faites ensuite glisser la “Destination OLE DB” de la boîte à outils SSIS vers la fenêtre “Flux de données” et renommez-la en “Table SQL”. Faites glisser la flèche bleue de la source vers la destination.

Double-cliquez sur la destination et configurez comme indiqué ci-dessous.

[![entrez la description de l’image ici][7]][7]

Cliquez sur Mappages sur le côté gauche de l’écran et assurez-vous que tous les champs sont correctement mappés de la source à la destination.

[![entrez la description de l’image ici][8]][8]

Cliquez sur OK. Votre écran devrait ressembler à l’image ci-dessous. [![entrez la description de l’image ici][9]][9]

Exécutez le package en cliquant sur Démarrer. Une fois l’exécution du package terminée, vous pouvez consulter le tableau pour afficher les données du fichier CSV.
