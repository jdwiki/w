Naviguer () [Java] #

Pour Naviguer vers n’importe quelle URL :

driver.navigate().to("http://www.example.com");

Pour reculer :

driver.navigate().back();

Pour avancer :

driver.navigate().forward();

Pour rafraichir la page :

driver.navigate().refresh();

Naviguer() [C#] #

Il est possible de naviguer directement dans le navigateur, comme en utilisant les commandes standard de la barre d’outils disponibles sur tous les navigateurs :

[![entrez la description de l’image ici][1]][1]

Vous pouvez créer un objet de navigation en appelant Navigate() sur le pilote :

IWebDriver driver
INavigation navigation = driver.Navigate();

Un objet de navigation vous permet d’effectuer de nombreuses actions qui naviguent dans le navigateur sur le Web :

//like pressing the back button
navigation.Back();
//like pressing the forward button on a browser
navigation.Forward();
//navigate to a new url in the current window
navigation.GoToUrl("www.stackoverflow.com");
//Like pressing the reload button
navigation.Refresh();

[1] : http://i.stack.imgur.com/eZaZE.png

Méthodes du navigateur dans WebDriver #

WebDriver, L’interface principale à utiliser pour les tests, qui représente un navigateur Web idéalisé. Les méthodes de cette classe se répartissent en trois catégories :

  • Contrôle du navigateur lui-même
  • Sélection de WebElements
  • Aides au débogage

Les méthodes clés sont get(String), qui est utilisée pour charger une nouvelle page Web, et les différentes méthodes similaires à findElement(By), qui est utilisée pour trouver des WebElements. Dans cet article, nous allons apprendre les méthodes de contrôle du navigateur. obtenir

void get(java.lang.String url)

Charger une nouvelle page Web dans la fenêtre actuelle du navigateur. Cela se fait à l’aide d’une opération HTTP GET, et la méthode se bloquera jusqu’à ce que le chargement soit terminé. il est préférable d’attendre la fin de ce délai, car si la page sous-jacente change pendant l’exécution de votre test, les résultats des futurs appels sur cette interface seront sur la page fraîchement chargée. Usage

//Initialising driver
 WebDriver driver = new FirefoxDriver();
 
 //setting timeout for page load
 driver.manage().timeouts().pageLoadTimeout(20, TimeUnit.SECONDS);
 
 //Call Url in get method
 driver.get("https://www.google.com");
 //or
 driver.get("https://seleniumhq.org");

getCurrentUrl

java.lang.String getCurrentUrl()

Obtenez une chaîne représentant l’URL actuelle consultée par le navigateur. Il renvoie l’URL de la page actuellement chargée dans le navigateur.

Usage

//Getting current url loaded in browser & comparing with expected url
 String pageURL = driver.getCurrentUrl();
 Assert.assertEquals(pageURL, "https://www.google.com");

** obtenirTitre **

java.lang.String getTitle()

Il renvoie le titre de la page actuelle, avec les espaces blancs de début et de fin supprimés, ou null s’il n’y en a pas déjà un.

Usage

//Getting current page title loaded in browser & comparing with expected title
 String pageTitle = driver.getTitle();
 Assert.assertEquals(pageTitle, "Google");

getPageSource

java.lang.String getPageSource()

Obtenez la source de la dernière page chargée. Si la page a été modifiée après le chargement (par exemple, par Javascript) il n’y a aucune garantie que le texte renvoyé soit celui de la page modifiée.

Usage

//get the current page source
 String pageSource = driver.getPageSource();

proche

void close()

Ferme la fenêtre en cours, quitte le navigateur s’il s’agit de la dernière fenêtre actuellement ouverte. S’il y a plus d’une fenêtre ouverte avec cette instance de pilote, cette méthode fermera la fenêtre qui a actuellement le focus dessus.

Usage

//Close the current window
     driver.close();

quitter

void quit()

Quitte ce pilote en fermant toutes les fenêtres associées. Après avoir appelé cette méthode, nous ne pouvons utiliser aucune autre méthode utilisant la même instance de pilote.

Usage

//Quit the current driver session / close all windows associated with driver
     driver.quit();

Ce sont toutes des méthodes très utiles disponibles dans Selenium 2.0 pour contrôler le navigateur selon les besoins.
