Configuration de TestNG #

TestNG est votre framework de test mis à jour pour junit. Nous allons utiliser testng.xml pour invoquer des suites de tests. Ceci est utile lorsque nous allons utiliser CI à l’avance.

testng.xml #

Dans le dossier racine de votre projet, créez un fichier xml avec le nom testng.xml. Notez que le nom peut également être différent, mais pour plus de commodité, il est utilisé comme “testng” partout.

Vous trouverez ci-dessous le code simple du fichier testng.xml.

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE suite SYSTEM "http://testng.org/testng-1.0.dtd">
<suite name="Smoke"> //name of the suite 
    <test name="Test1"> //name of the test
        <classes>
            <class name="test.SearchTest">
            <methods>
                <include name="searchTest"/>
            </methods>
            </class>
        </classes>
    </test>
</suite>
