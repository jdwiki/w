Paramètres #

paramètre détails
nomOuId Sélectionnez un cadre par son nom d’id.
index Sélectionnez une image par son index de base zéro.
frameElement Sélectionnez un cadre en utilisant son WebElement précédemment localisé

Pour passer à un cadre en utilisant Java #

Pour une instance, si le code source html d’une vue ou d’un élément html est enveloppé par un iframe comme celui-ci :

<iframe src="../images/eightball.gif" name="imgboxName" id="imgboxId">
   <p>iframes example</p>
   <a href="../images/redball.gif" target="imgbox">Red Ball</a>
</iframe><br />

… puis pour effectuer une action sur les éléments Web de l’iframe, vous devez d’abord basculer le focus sur l’iframe, en utilisant l’une des méthodes ci-dessous :

Utilisation de l’ID de cadre (doit être utilisé uniquement si vous connaissez l’ID de l’iframe).

driver.switchTo().frame("imgboxId"); //imgboxId - Id of the frame

Utilisation du nom du cadre (doit être utilisé uniquement si vous connaissez le nom de l’iframe).

driver.switchTo().frame("imgboxName"); //imgboxName - Name of the frame

Utilisation de l’index de cadre (doit être utilisé uniquement si vous n’avez pas l’identifiant ou le nom de l’iframe), où l’index définit la position de l’iframe parmi tous les cadres.

driver.switchTo().frame(0); //0 - Index of the frame

Remarque : Si vous avez trois cadres dans la page, le premier cadre sera à l’index 0, le second à l’index 1 et le troisième à l’index 2.

Utilisation d’un élément Web précédemment localisé (doit être utilisé uniquement si vous avez déjà localisé le cadre et l’avez renvoyé en tant que WebElement).

driver.switchTo().frame(frameElement); //frameElement - webelement that is the frame

Donc, pour cliquer sur l’ancre Red Ball :

driver.switchTo().frame("imgboxId");
driver.findElement(By.linkText("Red Ball")).Click();

Passer à un cadre en utilisant C# #

1. Passer à une image par Index.

Ici, nous passons à l’index 1. L’index fait référence à l’ordre des cadres sur la page. Cela devrait être utilisé en dernier recours, car l’identifiant ou les noms de cadre sont beaucoup plus fiables.

driver.SwitchTo().Frame(1);

2. Basculer vers un cadre par Nom

driver.SwitchTo().Frame("Name_Of_Frame");

**3. Passez à un cadre par titre, identifiant ou autres en passant IWebElement **

Si vous souhaitez passer à un cadre par identifiant ou titre, vous devez passer un élément Web en paramètre :

driver.SwitchTo().Frame(driver.FindElement(By.Id("ID_OF_FRAME")));
driver.SwitchTo().Frame(driver.FindElement(By.CssSelector("iframe[title='Title_of_Frame']")));

Notez également que si votre cadre met quelques secondes à apparaître, vous devrez peut-être utiliser un [wait][1] :

new WebDriverWait(driver, TimeSpan.FromSeconds(10))
    .Until(ExpectedConditions.ElementIsVisible(By.Id("Id_Of_Frame")));

Sortir d’un cadre :

driver.SwitchTo().DefaultContent()

[1] : https://www.wikiod.com/fr/selenium-webdriver/attendre

Pour sortir d’un cadre en utilisant Java #

Pour basculer le focus vers le document principal ou la première image de la page. Vous devez utiliser la syntaxe ci-dessous.

driver.switchTo().defaultContent();

Attendez que vos cadres se chargent #

Dans de nombreux cas, votre cadre peut ne pas s’afficher immédiatement et vous devrez probablement attendre qu’il soit chargé pour basculer. Sinon, vous aurez NoSuchFrameException.

C’est donc toujours un bon choix d’attendre avant de changer. Voici un moyen idéal d’attendre qu’un cadre soit chargé.

try{
        new WebDriverWait(driver, 300).ignoring(StaleElementReferenceException.class).
                        ignoring(WebDriverException.class).
                        until(ExpectedConditions.visibilityOf((driver.findElement(By.id("cpmInteractionDivFrame"))));}
    catch{

// lance une exception uniquement si votre cadre n’est pas visible avec dans votre temps d’attente 300 secondes }

Pour sortir d’un cadre en utilisant C# #

Pour basculer le focus vers le document principal ou la première image de la page. Vous devez utiliser la syntaxe ci-dessous.

webDriver.SwitchTo().DefaultContent();

Basculer entre les cadres enfants d’un cadre parent. #

Considérez que vous avez un cadre parent (Frame-Parent). et 2 cadres enfants (Frame_Son,Frame_Daughter). Voyons différentes conditions et comment gérer.

1.De parent à fils ou fille :

 driver.switchTo().frame("Frame_Son");
 driver.switchTo().frame("Frame_Daughter");

2.Du fils au parent : si le parent est le cadre par défaut, passez au cadre par défaut, sinon, du cadre par défaut, passez au cadre parent. Mais vous ne pouvez pas passer directement de fils à parent.

driver.switchTo().defaultContent(); 
driver.switchTo().frame("Frame_Parent");

  1. De fils à fille : si votre sœur fait une erreur, ne lui criez pas dessus, contactez simplement votre parent. De même, vous donnez le contrôle au cadre parent puis au cadre fille.

    driver.switchTo().defaultContent(); driver.switchTo().frame(“Frame_Parent”); driver.switchTo().frame(“Frame_Daughter”);
