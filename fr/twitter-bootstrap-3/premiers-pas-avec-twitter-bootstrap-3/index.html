Installation #

  • Téléchargement direct - lien de téléchargement
  • CDN - arriver ici
  • Bower - booter install bootstrap [lire]
  • NPM - npm install bootstrap [lire]
  • Composer - composer require twbs/bootstrap [read]
  • Personnaliser - Si vous avez votre propre configuration, vous pouvez la personnaliser ici.
  • Sass - Pour les projets liés à Sass, vous pouvez l’obtenir [ici][1].

Usage

<head>
  <link rel="stylesheet" href="path/to/bootstrap.min.css">
</head>

La référence au fichier bootstrap js est faite à l’aide de la balise script juste au-dessus de la balise body (voir ci-dessous). Notez également que bootstrap utilise jQuery pour la plupart de ses widgets - comme le carrousel d’accordéon, etc., donc référencez le fichier bootstrap js sous le fichier jQuery js.

**Goûter** 
<!DOCTYPE html>
<html lang="en">

<head>

  <title>Form Bootstrap Example</title>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">
</head>

<body>

  <div class="container">
    <h2>Form Email</h2>
    <form role="form">
      <div class="form-group">
        <label for="email">Email:</label>
        <input type="email" class="form-control" id="email" placeholder="Enter email">
      </div>
      <div class="form-group">
        <label for="pwd">Password:</label>
        <input type="password" class="form-control" id="pwd" placeholder="Enter password">
      </div>
      <div class="checkbox">
        <label><input type="checkbox"> Remember me</label>
      </div>
      <button type="submit" class="btn btn-default">Submit</button>
    </form>
  </div>

  <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js"></script>

</body>

</html>

[1] : https://github.com/twbs/bootstrap-sass/archive/v3.3.7.tar.gz

Bonjour le monde #

La page HTML suivante illustre une simple page Hello World utilisant Bootstrap 3.

La page contient une barre de navigation pleine largeur avec des exemples de liens et un contrôle déroulant. La barre de navigation tire parti des premières capacités mobiles de Bootstrap. Il commence réduit dans les vues mobiles et devient horizontal à mesure que la largeur de la fenêtre d’affichage disponible augmente.

De plus, un élément « jumbotron » a été utilisé pour afficher les informations présentées.

<!DOCTYPE html>
<html lang="en">
<head>
    <title>Bootstrap Hello World</title>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">
</head>

<body>

    <nav class="navbar navbar-default">
        <div class="container-fluid">
            <div class="navbar-header">
                <button type="button" class="navbar-toggle collapsed" data-toggle="collapse" data-target="#bs-example-navbar-collapse-1" aria-expanded="false">
                    <span class="sr-only">Toggle navigation</span>
                    <span class="icon-bar"></span>
                    <span class="icon-bar"></span>
                    <span class="icon-bar"></span>
                </button>
                <a class="navbar-brand" href="#">Hello, World!</a>
            </div>


            <div class="collapse navbar-collapse" id="bs-example-navbar-collapse-1">

                <ul class="nav navbar-nav navbar-right">
                    <li><a href="#">Link</a></li>
                    <li class="dropdown">
                        <a href="#" class="dropdown-toggle" data-toggle="dropdown" role="button" aria-haspopup="true" aria-expanded="false">Dropdown <span class="caret"></span></a>
                        <ul class="dropdown-menu">
                            <li><a href="#">Action</a></li>
                            <li><a href="#">Another action</a></li>
                            <li><a href="#">Something else here</a></li>
                            <li role="separator" class="divider"></li>
                            <li><a href="#">Separated link</a></li>
                        </ul>
                    </li>
                </ul>
            </div>
        </div>
    </nav>

    <div class="container">
        <div class="jumbotron">
            <h1>Bootstrap</h1>
            <p>This is a simple hero unit, a simple jumbotron-style component for calling extra attention to featured content or information.</p>
            <p><a class="btn btn-primary btn-lg">Learn more</a></p>
        </div>
    </div>


    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>
    <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js"></script>

</body>
</html>

Installation de Bootstrap et démarrage #

Présentation
Alors vous voulez commencer à utiliser bootstrap pour votre projet ? Super! alors commençons dès maintenant !.

Qu’est-ce que le démarrage ? Bootstrap est une bibliothèque open source que vous pouvez utiliser pour créer d’étonnants projets ** réactifs ** en utilisant un design réactif et un code simple. La conception réactive est une philosophie de conception où, dans la conception du système (la représentation et la disposition), répond/s’adapte en fonction de la disposition de l’appareil. La principale raison de garder votre conception réactive est d’augmenter la portée de votre application auprès d’une base d’utilisateurs plus large à l’aide d’un éventail d’appareils.
Montage
Bootstrap peut être installé de différentes manières et pour de nombreux types de projets. Dans la liste ci-dessous, j’ai placé des liens de téléchargement et de didacticiel sur l’installation de bootstrap

Informations de base
Vous avez donc maintenant installé bootstrap dans votre projet. Et maintenant, il est temps de commencer à utiliser les grands avantages du bootstrap. Tout d’abord, je vais vous expliquer quelques utilisations de base de bootstrap, après cela, je montrerai quelques petits exemples et à la fin, je vous donnerai un exemple de code de démarrage que vous pourrez utiliser comme modèle de départ !

Le système de grille
Bootstrap utilise un système de grille. Ce système de grille se compose normalement de 12 colonnes. Chacune de ces 12 colonnes a la même largeur mais peut avoir des hauteurs différentes.

Nous avons donc un système de grille composé de 12 colonnes. Nous pouvons utiliser ces colonnes pour créer notre site Web de base. disons que nous voulons réaliser la mise en page suivante :

menu - pleine largeur
barre latérale - 1/3 de l’écran
Contenu principal 2/3 de l’écran
Pied de page - pleine largeur

La carte
Nous allons d’abord regarder le menu. Comme nous le savons, le système de grille fonctionne avec 12 colonnes. Puisque nous voulons le menu sur toute la largeur, nous devons mettre le menu dans les 12 colonnes. Cela ressemblera à l’exemple ci-dessous

<div class="col-lg-12 col-md-12 col-sm-12 col-xs-12">
    Menu
</div>

En donnant au menu une classe col-lg-12, nous indiquons ce qui suit :
col - Le col dans le nom de la classe représente les colonnes.
lg - Le lg dans le nom de la classe correspond à la largeur de l’écran, dans ce cas grand.
12 - le 12 dans le nom de la classe représente le nombre de colonnes que nous voulons que notre menu possède. puisque dans ce cas on veut le menu sur toute la largeur on obtient toutes les 12 colonnes (12/12)

Puisque nous avons utilisé 12 des 12 colonnes pour notre menu, tout après le menu sera défini sur une nouvelle ligne.

La barre latérale
Le deuxième élément que nous voulons ajouter à notre modèle est la barre latérale. Maintenant, comme je l’ai dit, nous voulons que la barre latérale soit 1/3 de l’écran. Donc, ce que nous allons faire, c’est diviser les 12 colonnes en 3, soit 4. Nous savons maintenant combien de colonnes nous voulons remplir pour atteindre 1/3 de l’écran. Suivez le code ci-dessous.

<div class="col-lg-12 col-md-12 col-sm-12 col-xs-12">
    The menu
</div>
<div class="col-lg-4 col-md-4 col-sm-4 col-xs-4">
    The sidebar
</div>

La même chose qu’avec le menu seulement maintenant notre nombre de cols diffère du menu.
col - Le col dans le nom de la classe représente les colonnes.
lg - Le lg dans le nom de la classe correspond à la largeur de l’écran, dans ce cas grand.
4 - le 12 dans le nom de la classe représente le nombre de colonnes que nous voulons que notre barre latérale possède. puisque dans ce cas, nous voulons que la barre latérale remplisse 1/3 de l’écran, nous ne saisirons donc que 4 des 12 colonnes (4/12)

Le contenu
Maintenant, sur cette ligne, nous avons encore 8 colonnes à côté de notre barre latérale. Alors maintenant, nous allons les remplir avec notre contenu. Voir l’exemple de code ci-dessous

<div class="col-lg-12 col-md-12 col-sm-12 col-xs-12">
    The menu
</div>
<div class="col-lg-4 col-md-4 col-sm-4 col-xs-4">
    The sidebar
</div>
<div class="col-lg-8 col-md-8 col-sm-8 col-xs-8">
    The main content
</div>

Maintenant que nous avons rempli les 8 colonnes restantes de nos 12 colonnes sur cette ligne, la section suivante commencera à nouveau sur une nouvelle ligne avec 12 colonnes.

Le pied de page
Le pied de page est, encore une fois, tout comme le menu va être un bloc pleine largeur à l’écran, nous allons donc saisir les 12 colonnes de cette ligne pour notre pied de page, voir l’exemple de code ci-dessous.

<div class="col-lg-12 col-md-12 col-sm-12 col-xs-12">
    The menu
</div>
<div class="col-lg-4 col-md-4 col-sm-4 col-xs-4">
    The sidebar
</div>
<div class="col-lg-8 col-md-8 col-sm-8 col-xs-8">
    The main content
</div>
<div class="col-lg-12 col-md-12 col-sm-12 col-xs-12">
    The footer
</div>i are described below.

Alors maintenant, nous avons créé, avec juste un très petit fichier html, notre premier modèle de bootstrap. Mais c’est la base même. Normalement, nous formerions un peu plus ce code pour lui donner l’expérience d’amorçage complète. Certaines de ces expériences sont décrites ci-dessous.

Utilisation de rangées et de conteneurs Comme je l’ai dit dans l’exemple simple ci-dessus, bootstrap utilise des lignes de 12 colonnes. lorsqu’une ligne est remplie de 12 colonnes, le bootstrap démarre sur une nouvelle ligne. la meilleure façon d’afficher ces lignes est d’utiliser des classes de lignes. Nous utiliserons également une classe de conteneur. c’est comme une balise body, dans ce conteneur nous mettrons tout notre code. Vous pouvez choisir entre un conteneur ou une classe de fluide de conteneur. La seule différence est que la classe container-fluid utilise toute la largeur d’un écran et pas la classe container. Un exemple de ces fonctionnalités de base est ci-dessous.

<div class="container-fluid">
    <div class="row">
        <div class="col-lg-12 col-md-12 col-sm-12 col-xs-12">
            The menu consisting of 12 columns
        </div>
    </div>
    <div class="row">
        <div class="col-lg-4 col-md-4 col-sm-4 col-xs-4">
            The sidebar consisting of 4 columns
        </div>
        <div class="col-lg-8 col-md-8 col-sm-8 col-xs-8">
            The main content consisting of 8 columns
        </div>
    </div>
    <div class="row">
        <div class="col-lg-12 col-md-12 col-sm-12 col-xs-12">
            The footer consisting of 12 columns
        </div>
    </div>
</div>

Nous avons donc créé un modèle de page complète avec bootstrap. C’est en effet très simple, mais commencez par les bases et plus tard, vous pourrez utiliser toutes sortes de classes et de fonctionnalités d’amorçage. Dernière chose. Les noms de colonne lg - md - sm et xs dans les classes correspondent, comme je l’ai dit à la largeur de la page. Grand, Moyen, Petit et XSmall. N’oubliez pas que vous pouvez l’utiliser pour appliquer un style différent aux colonnes sur différentes largeurs en modifiant le nombre de colonnes. N’oubliez pas qu’une ligne complète se compose de 12 colonnes !
