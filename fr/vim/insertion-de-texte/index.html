Exécution de commandes normales à partir du mode insertion #

En mode insertion, appuyez sur <C-o> pour quitter temporairement le mode insertion et exécuter une commande normale unique.

Exemple #

<C-o>2w saute au deuxième mot à gauche et revient en mode insertion.

Remarque : Répéter avec . ne répétera que les actions du retour au mode d’insertion

Cela permet des mappages utiles, par ex.

inoremap <C-f> <Right>
inoremap <C-b> <Left>
inoremap <C-a> <C-o>^
inoremap <C-e> <C-o>$

Maintenant, ctrl+a placera le curseur au début de la ligne et ctrl+e - à la fin de la ligne. Ces mappages sont utilisés par défaut dans readline, ils peuvent donc être utiles pour les personnes qui recherchent la cohérence.

Différentes façons d’accéder au mode d’insertion #

Commande La description
a Ajouter du texte après la position actuelle du curseur
A Ajouter du texte à la fin de la ligne courante
je Insérer du texte avant la position actuelle du curseur
je Insérer du texte avant le premier caractère non vide de la ligne courante
gI Insérer du texte dans la première colonne de la ligne du curseur
gi Insérer du texte à la même position où il a été laissé la dernière fois en mode Insertion
‘O’ Ouvrez une nouvelle ligne au-dessus de la ligne actuelle et ajoutez-y du texte (CAPITAL O)
o Ouvrez une nouvelle ligne sous la ligne actuelle et ajoutez-y du texte (minuscule o)
s ou cl Supprimer le caractère sous le curseur et passer en mode insertion
S ou cc Supprimer toute la ligne et passer en mode Insertion
C Supprimer de la position du curseur à la fin de la ligne et démarrer le mode insertion
c{mouvement} Supprimez {motion} et démarrez le mode d’insertion (voir https://www.wikiod.com/fr/vim/mouvement#Mouvement de base)

Raccourcis du mode Insertion #

Commande La description
<C-w> Supprimer le mot avant le curseur
<C-t> Indenter la ligne actuelle d’un shiftwidth
<C-d> Désindenter la ligne actuelle avec un shiftwidth
<C-f> réindenter la ligne, (déplacer le curseur à la position d’indentation automatique)
<C-a> Insérer le texte précédemment inséré
<C-e> Insérez le caractère ci-dessous
<C-h> Supprimer un caractère en arrière
<C-y> Insérez le caractère ci-dessus
<C-r>{registre} Insérez le contenu de {register}
<C-o>{commande du mode normal} exécuter {commande en mode normal} sans quitter le mode insertion
<C-n> Prochaine option de saisie semi-automatique pour le mot actuel
<C-p> Option de saisie semi-automatique précédente pour le mot actuel
<C-v> Insérer un caractère par sa valeur ASCII en décimal
<C-v>x Insérer un caractère par sa valeur ASCII en hexadécimal
<C-v>u Insérer un caractère par sa valeur unicode en hexadécimal
<C-k> Entrer un digramme

Quitter le mode insertion #

Commande La description
<Échap> Quitte le mode insertion, déclenche les autocommandes et les abréviations
<C-[> Synonyme exact de <Esc>
<C-c> Quitte le mode insertion, ne déclenche pas les autocommandes

Certaines personnes aiment utiliser une paire de caractères relativement rare comme jk comme raccourci pour <Esc> ou <C-[> qui peut être difficile à atteindre sur certains claviers :

inoremap jk <Esc>l

Insérer du texte sur plusieurs lignes à la fois #

Appuyez sur Ctrl + v pour entrer en mode bloc visuel.

Utilisez ↑ / ↓ / j / k pour sélectionner plusieurs lignes.

Appuyez sur Shift + i et commencez à taper ce que vous voulez.

Après avoir appuyé sur Esc, le texte sera inséré dans toutes les lignes que vous avez sélectionnées.

N’oubliez pas que Ctrl+c n’est pas équivalent à 100 % à Esc et ne fonctionnera pas dans cette situation !

Il existe de légères variations de Shift + i sur lesquelles vous pouvez appuyer en mode bloc visuel :

Clé La description
c ou s Supprimer le bloc sélectionné et passer en mode insertion
C Supprimer les lignes sélectionnées (du curseur jusqu’à la fin) et passer en mode insertion
R Supprimer les lignes sélectionnées et passer en mode insertion
A Ajouter aux lignes sélectionnées, avec la colonne à la fin de la première ligne

Notez également qu’appuyer sur . après une opération de bloc visuel répétera cette opération là où se trouve le curseur !

Coller le texte à l’aide de la commande “coller” du terminal #

Si vous utilisez la commande paste de votre programme d’émulation de terminal, Vim interprétera le flux de caractères comme s’ils avaient été saisis. Cela provoquera toutes sortes d’effets indésirables, en particulier une mauvaise indentation.

Pour résoudre ce problème, depuis le mode commande :

:set paste

Passez ensuite au mode insertion, avec i, par exemple. Notez que le mode est maintenant -- INSERT (coller) --. Collez maintenant avec la commande de votre émulateur de terminal ou avec la souris. Lorsque vous avez terminé, passez en mode commande, avec Esc et exécutez :

:set nopaste

Il existe un moyen plus simple, quand on veut coller une seule fois. Mettez ceci dans votre .vimrc (ou utilisez le plugin unimpaired.vim) :

function! s:setup_paste() abort
  set paste
  augroup unimpaired_paste
    autocmd!
    autocmd InsertLeave *
      \ set nopaste |
      \ autocmd! unimpaired_paste
  augroup end
endfunction

nnoremap <silent> yo :call <SID>setup_paste()<CR>o
nnoremap <silent> yO :call <SID>setup_paste()<CR>O

Maintenant, on peut simplement appuyer sur yo pour coller le code sous le curseur, puis sur <Esc> pour revenir au mode normal/nopaste.

Coller à partir d’un registre en mode insertion #

En mode insertion, vous pouvez utiliser <C-r> pour coller à partir d’un registre, qui est spécifié par la frappe suivante. <C-r>" par exemple colle depuis le registre sans nom (").

Voir :registres d'aide.

Commandes et raccourcis d’insertion avancés #

Voici une référence rapide pour les commandes/raccourcis avancés d’insertion, de formatage et de filtrage.

Commande/Raccourci Résultat
g + ? + m Effectuer l’encodage rot13, sur mouvement m
n + ctrl + a +n au numéro sous le curseur
n + ctrl + x -n au nombre sous le curseur
g + q+ m Formater les lignes de mouvement m à largeur fixe
:rce w Lignes centrales dans la plage r à largeur w
:rle i Aligner à gauche les lignes dans la plage r avec le retrait i
:rri w Aligner à droite les lignes comprises entre r et la largeur w
!mc Filtrer les lignes de mouvement m via la commande c
n!!c Filtrez n lignes via la commande c
:r!c Filtrer la plage r lignes via la commande c

Désactiver l’indentation automatique pour coller le code #

Lorsque vous collez du texte via un émulateur de terminal, la fonction d’indentation automatique peut détruire l’indentation du texte collé.

Par exemple:

function () {
    echo 'foo'
    echo 'bar'
    echo 'baz'
}

sera collé comme :

function () {
    echo 'foo'
        echo 'bar'
            echo 'baz'
            }

Dans ces cas, utilisez l’option paste/nopaste pour désactiver/activer la fonction d’indentation automatique :

:set paste
:set nopaste

En plus de cela, il existe une approche plus simple du problème : ajoutez la ligne suivante dans votre .vimrc :

set pastetoggle=<F3>

Et si vous voulez coller tel quel depuis le presse-papiers. Appuyez simplement sur F3 en mode insert et collez. Appuyez à nouveau sur F3 pour quitter le mode coller.
