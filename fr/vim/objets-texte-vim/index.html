Sélectionnez un mot sans espace blanc environnant #

Supposons que nous voulions sélectionner un mot sans espaces blancs environnants, utilisez l’objet texte “iw” pour le mot intérieur en utilisant le mode visuel :

  1. Passez en mode normal en appuyant sur “ESC”
  2. Tapez “viw” au début d’un mot
  3. Cela sélectionnera le mot intérieur

[![voir][1]][1]

[1] : http://i.stack.imgur.com/hQf4X.png

Sélectionnez un mot entouré d’un espace blanc #

Supposons que nous voulions sélectionner un mot entouré d’un espace blanc, utilisez l’objet texte “aw” pour entourer un mot en utilisant le mode visuel :

  1. Passez en mode normal en appuyant sur ESC

  2. Tapez “vaw” au début d’un mot

  3. Cela sélectionnera le mot avec un espace blanc

    [![vaw][1]][1]

[1] : http://i.stack.imgur.com/UUq8i.png

## Sélectionner du texte à l’intérieur d’une balise Nous pouvons sélectionner un texte dans une balise html ou xml en utilisant la sélection visuelle v et l’objet texte it .

  1. Passez en mode normal en appuyant sur ESC
  2. Tapez vit de n’importe où dans la section html ou xml
  3. Cela sélectionnera visuellement tout le texte à l’intérieur de la tag

[![entrez la description de l’image ici][1]][1]

Tous les autres objets texte peuvent également être utilisés pour opérer sur le texte à l’intérieur de la balise

  1. cit - supprimer le texte à l’intérieur de la balise et le placer en mode insert
  2. dit - supprime le texte à l’intérieur de la balise et reste en mode normal
  3. cat - supprime autour de la balise et place en mode insert
  4. dat - supprime le texte autour de la balise et reste en mode normal

[1] : https://i.stack.imgur.com/dZtfw.png
