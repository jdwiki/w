Configuration de LWJGL3 dans Eclipse #

Remarque : Ce guide d’installation suppose que Java et Eclipse sont déjà installés sur votre machine. Si ce n’est pas le cas, téléchargez la dernière version du kit de développement Java (ci-après appelé JDK) et la dernière version d’Eclipse (Neon 3 au moment de la rédaction).

Première étape : télécharger LWJGL #

Pour commencer, vous aurez besoin d’une copie de LWJGL, accessible depuis le [site Web de LWJGL][1]. Pour être assuré de la version la plus stable, choisissez Release. Ce guide d’installation utilisera le ZIP Bundle pour ajouter LWJGL à votre projet. À l’extrême droite de la page Web, choisissez les dépendances que vous souhaitez télécharger. Une fois que vous avez sélectionné le contenu de l’archive ZIP, sélectionnez les natifs que vous souhaitez, ainsi que si vous voulez les sources et/ou JavaDocs pour chaque liaison sélectionnée. Une fois terminé, sélectionnez “Télécharger Zip” en bas de la boîte de sélection.

[![Page Web LWJGL][2]][2]

Deuxième étape : Créer un nouveau projet dans Eclipse #

Une fois que vous avez téléchargé les fichiers JAR nécessaires à partir du site Web LWJGL, vous devrez créer un nouveau projet dans Eclipse. Pour ce faire, ouvrez Eclipse dans le dossier de l’espace de travail souhaité. Une fois la fenêtre Eclipse ouverte, accédez à la boîte de dialogue Nouveau projet Java en allant dans Fichier -> Nouveau -> Projet Java. Une fois que vous avez terminé le dialogue, appuyez sur Terminer et vous aurez un nouveau projet dans votre explorateur de packages et votre navigateur.

[![Nouveau dialogue de projet][3]][3]

Ensemble trois : ajout de LWJGL au chemin de génération #

Vous verrez également un nouveau répertoire dans votre explorateur de fichiers. Dans ce nouveau répertoire, créez un dossier appelé lib. Dans ce dossier, créez un dossier pour vos fichiers jar appelé jar. Vous aurez également des bibliothèques natives, alors créez un dossier pour celles-ci également appelé natives. Ouvrez votre jar et placez les fichiers jar de la bibliothèque dans le dossier jar et les natifs dans le dossier natives. Les natifs sont stockés dans un fichier .jar, vous devrez donc Si vous avez choisi d’ajouter des fichiers source et des JavaDocs à votre téléchargement, ajoutez ces fichiers au répertoire lib (vous pouvez également créer un nouveau sous-répertoire pour chacun, nommé respectivement javadocs et sources). Une fois ajouté, vous devrez retourner dans Eclipse et actualiser le projet (raccourci Windows F5). Le dossier lib doit se trouver dans les vues du navigateur et de l’explorateur de packages. Si ce n’est pas le cas, actualisez à nouveau. Une fois que vous voyez le dossier, ouvrez la boîte de dialogue Propriétés du projet. Cela peut être trouvé dans Projet -> Propriétés. Accédez à l’onglet Java Build Path sur la droite. Vous ajouterez les fichiers JAR à partir de l’endroit où vous les avez enregistrés. En suivant ce didacticiel, vous ajouterez les fichiers JAR à partir du bouton Ajouter des fichiers JAR…, car nous les avons enregistrés dans l’espace de travail. Dans la boîte de dialogue de fichier qui s’ouvre, sélectionnez tous les fichiers JAR que vous souhaitez ajouter. Une fois ajouté, vous devrez développer chaque JAR, en sélectionnant l’emplacement du natif pour chaque JAR. Si vous avez ajouté des sources et des JavaDocs à votre dossier lib, vous les ajouterez également ici.
