Paramètres #

Paramètres Descriptif
chaîne=“Nom” Libellé facultatif du champ
calculate="_compute_name_custom" Transformer les champs en champs calculés
magasin=Vrai S’il est calculé, il stockera le résultat
sélectionnez=Vrai Indice de force sur le terrain
lecture seule=Vrai Le champ sera en lecture seule dans les vues
inverse="_write_name" Sur déclencheur de mise à jour
requis=Vrai Champ obligatoire
traduire=Vrai Activation de la traduction
help=‘blablabla’ Texte de l’info-bulle d’aide
comodel_name=“model.name” Nom du modèle associé
inverse_name=“nom_champ” colonne relationnelle du modèle opposé
relation=‘many2many_table_name’ nom de la table relationnelle pour many2many
colonnes1=‘nom_colonne_gauche’ nom de la colonne de gauche de la table relationnelle
column2=‘right_column_name’ nom de la colonne de droite de la table relationnelle

Odoo et ORM : Odoo utilise la technique ORM (Object Relational Mapping) pour interagir avec la base de données. ORM aidera à créer une base de données d’objets virtuels pouvant être utilisée à partir de Python. Dans la technique ORM, chaque modèle est représenté par une classe qui sous-classe Models.model.

Models.model est la super classe principale pour les modèles Odoo persistants dans la base de données régulière. Les modèles Odoo sont créés en héritant de cette classe.

Exemple:

class Employee(Models.model):
    _name = 'module.employee'

    #Rest of the code goes here

Ici, _name est un attribut structurel, qui indique au système le nom de la table de base de données à créer.

Chaque modèle a un certain nombre de variables de classe, dont chacune représente un champ de base de données dans le modèle. Chaque champ est représenté par une instance d’une classe openerp.fields.Field. Les champs dans Odoo sont listés ci-dessous.

1 champ booléen

ex: flag = fields.Boolean()

2 Champ de caractères

ex: flag = fields.Char()

3 Texte

ex: flag = fields.Text()

4 HTML

ex: flag = fields.Html()

5 Entier

ex: flag = fields.Integer()

6 Flotteur

ex: flag = fields.Float()

7 Rendez-vous

ex: flag = fields.Date()

8 Date et heure

ex: flag = fields.Datetime()

9 Sélection

ex: flag = fields.Selection()

10 Many2one

ex: flag = fields.Many2one()

11 Un à plusieurs

ex: flag = fields.One2many()

12 Beaucoup2beaucoup

ex: flag = fields.Many2many()

Exemples de champs d’Odoo 8 #

Odoo utilise la technique ORM (Object Relational Mapping) pour interagir avec la base de données. ORM aidera à créer une base de données d’objets virtuels pouvant être utilisée à partir de Python. Dans la technique ORM, chaque modèle est représenté par une classe qui sous-classe Models.model. Models.model est la super classe principale pour les modèles Odoo persistants dans la base de données régulière. Les modèles Odoo sont créés en héritant de cette classe

name = fields.Char(string='New Value')

flag = fields.Boolean(string='Flag',default=False)

amount = fields.Float(string='Amount',digits=(32, 32))

code = fields.Selection(string='Code',selection=[('a', 'A'),('b','B')])

customer = fields.Many2one(comodel_name='res.users')

sale_order_line = fields.One2many(comodel_name='res.users', inverse_name='rel_id')

tags = fields.Many2many(comodel_name='res.users',
                        relation='table_name',
                        column1='col_name',
                        column2='other_col_name')
