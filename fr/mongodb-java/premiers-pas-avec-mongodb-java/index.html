Installation ou configuration #

Prérequis

Un serveur MongoDB en cours d’exécution, plus de détails à ce sujet ici.

Configuration d’un projet Java

Le pilote Java MongoDB est fourni sous forme de fichier JAR et peut être inclus dans un projet comme n’importe quel autre JAR. Par exemple:

  • Maven : ajoutez une entrée à la section <dependencies> de votre pom.xml.

<dépendance> org.mongodb mongo-java-driver 3.4.2 </dépendance>

  • Gradle : ajoutez la dépendance suivante à votre build.gradle

compiler 'org.mongodb:mongo-java-driver:3.4.2'

  • Inclusion manuelle : téléchargez le fichier JAR depuis ici et ajoutez-le au chemin de classe de votre projet.

Remarque : la version indiquée dans les exemples ci-dessus est la dernière version stable au moment de la rédaction, mais vous pouvez choisir n’importe quelle version disponible [ici](http://search.maven.org/#search%7Cga%7C1%7Corg.mongodb %3A%3Amongo-java-driver).

Utilisation du pilote Java MongoDB

Créons un MongoClient, l’invocation la plus simple est :

MongoClient mongoClient = new MongoClient(new ServerAddress("localhost", 27017));

Cela suppose que le serveur MongoDB s’exécute sur localhost au port par défaut et qu’il n’est pas sécurisé. Il existe de nombreuses variantes pour un accès authentifié, des options de connexion personnalisées, la connexion à des jeux de répliques et des clusters fragmentés, tous couverts en détail dans la [documentation du pilote Java de Mongo DB] (http://mongodb.github.io/mongo-java-driver/ 3.4/driver/tutorials/connect-to-mongodb/).

Maintenant que nous sommes connectés, créons une base de données et une collection. MongoDB crée implicitement une collection lorsque le premier document y est enregistré, alors faisons cela :

MongoDatabase database = mongoClient.getDatabase("starter");
MongoCollection<Document> collection = database.getCollection("superheroes");
Document document = new Document("name", "Superman");
collection.insertOne(document);

Remarque : vous pouvez créer explicitement une collection mais l’utilisation ci-dessus est plus courante et correspond certainement mieux à un sujet “démarrage”.

Voyons maintenant si nous pouvons trouver le document que nous avons créé :

FindIterable<Document> documents = purchases.find(Filters.eq("name", "Superman"));
for (Document found : documents) {
    System.out.println(found.toJson());
}

Le code ci-dessus affichera ce qui suit (le champ _id est la clé de document générée automatiquement par MongoDB) :

Document trouvé : { “_id” : { “$oid” : “5981bd586d47c203904a9cf9” }, “name” : “Superman” }

Maintenant que nous avons vérifié que nous avons créé un document avec succès, supprimons-le :

DeleteResult deleteResult = purchases.deleteOne(Filters.eq("name", "Superman"));
System.out.println(String.format("Deleted %s document(s)", deleteResult.getDeletedCount()));

Le code ci-dessus affichera ce qui suit :

Supprimé 1 document(s)

Exemple complet

En rassemblant les différents éléments, nous obtenons :

import com.google.common.collect.Lists;
import com.mongodb.MongoClient;
import com.mongodb.MongoCredential;
import com.mongodb.ServerAddress;
import com.mongodb.client.FindIterable;
import com.mongodb.client.MongoCollection;
import com.mongodb.client.model.Filters;
import com.mongodb.client.result.DeleteResult;
import org.bson.Document;

import java.util.List;

public class IntroducingTheMongoDBJavaDriver {

    public static void main(String... args) throws Exception {
        MongoClient mongoClient = new MongoClient(new ServerAddress("localhost", 27017));

        MongoCollection<Document> superheroes = mongoClient.getDatabase("starter").getCollection("superheroes");

        Document document = new Document("name", "Superman");
        superheroes.insertOne(document);

        FindIterable<Document> documents = superheroes.find(Filters.eq("name", "Superman"));
        for (Document found : documents) {
            System.out.println(String.format("Found document: %s", found.toJson()));
        }

        DeleteResult deleteResult = superheroes.deleteOne(Filters.eq("name", "Superman"));
        System.out.println(String.format("Deleted %s document(s)", deleteResult.getDeletedCount()));

        long count = superheroes.count();
        System.out.println(String.format("Number of superheroes: %s", count));
    }
}
