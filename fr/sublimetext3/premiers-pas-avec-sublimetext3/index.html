Installation ou configuration #

Veuillez noter que l’utilisation continue de Sublime Text nécessite que vous [achetiez une licence][1] et que vous soyez invité à prendre note des [termes et conditions][2].

Le processus d’installation de Sublime Text est différent pour chaque plate-forme, mais dans chaque cas, vous devez visiter la [page de téléchargement][4].

Après avoir installé ST3, il est courant d’installer le gestionnaire de packages, [Package Control][5].

Mac #

Pour Mac, il n’y a qu’une seule version de Sublime Text pour OS X.

  1. Téléchargez le fichier .dmg

  2. Ouvrez le fichier .dmg

  3. Faites glisser le bundle Sublime Text 3 dans le dossier Applications

  4. Pour créer un lien symbolique à utiliser sur la ligne de commande, lancez le following command at the terminal:

    ln -s “/Applications/Sublime Text.app/Contents/SharedSupport/bin/subl” /usr/local/bin/subl

Les fenêtres #

Pour Windows, les versions 64 bits et 32 ​​bits sont disponibles, portables et non portables. Vous devriez pouvoir exécuter la version 64 bits si vous utilisez une version moderne de Windows. Si vous rencontrez des difficultés pour exécuter la version 64 bits, essayez la version 32 bits.

Portable ou non portable ? Sublime Text est disponible en deux versions pour Windows : normal et portable. La plupart des utilisateurs devraient être mieux servis par une installation normale. Utilisez la version portable uniquement si vous savez que vous en avez besoin.

Les installations normales séparent les données entre deux dossiers : le dossier d’installation proprement dit et le répertoire de données (répertoire spécifique à l’utilisateur pour les données). Les installations normales intègrent également Sublime Text au menu contextuel de l’Explorateur Windows.

Les installations portables conservent tous les fichiers nécessaires à Sublime Text dans un seul dossier. Ce dossier peut être déplacé et l’éditeur fonctionnera toujours.

Comment installer la version normale de Sublime Text

  1. Téléchargez le programme d’installation
  2. Double-cliquez sur le programme d’installation

Comment installer la version portable de Sublime Text

  1. Téléchargez les fichiers compressés

  2. Décompressez-les dans un dossier de votre choix

Vous trouverez l’exécutable sublime_text.exe dans ce dossier.

#Linux Pour Linux, exécutez cette commande dans votre terminal pour vérifier le type de votre système d’exploitation :

uname -m

Vous pouvez télécharger le package et le décompresser manuellement. Alternativement, vous pouvez utiliser la ligne de commande.

Ubuntu

  • Pour i386

    cd ~
wget http://c758482.r82.cf2.rackcdn.com/sublime-text_build-3083_i386.deb

  • Pour x64

    cd ~
wget http://c758482.r82.cf2.rackcdn.com/sublime-text_build-3083_amd64.deb

Autres distributions Linux

  • Pour i386

    cd ~
wget http://c758482.r82.cf2.rackcdn.com/sublime_text_3_build_3083_x32.tar.bz2
tar vxjf sublime_text_3_build_3083_x32.tar.bz2

  • Pour x64

    cd ~
wget http://c758482.r82.cf2.rackcdn.com/sublime_text_3_build_3083_x64.tar.bz2
tar vxjf sublime_text_3_build_3083_x64.tar.bz2

Nous devons maintenant déplacer les fichiers non compressés vers un emplacement approprié.

sudo mv Sublime\ Text\ 3 /opt/
Lastly, we create a symbolic link to use at the command line.

sudo ln -s /opt/Sublime\ Text\ 3/sublime_text /usr/bin/sublime

Dans Ubuntu, si vous souhaitez également ajouter Sublime Text au lanceur Unity, procédez comme suit :

Tout d’abord, créez un nouveau fichier.

sudo sublime /usr/share/applications/sublime.desktop

Copiez-y ce qui suit.

[Desktop Entry]
Version=1.0
Name=Sublime Text 3
# Only KDE 4 seems to use GenericName, so we reuse the KDE strings.
# From Ubuntu's language-pack-kde-XX-base packages, version 9.04-20090413.
GenericName=Text Editor

Exec=sublime
Terminal=false
Icon=/opt/Sublime Text 3/Icon/48x48/sublime_text.png
Type=Application
Categories=TextEditor;IDE;Development
X-Ayatana-Desktop-Shortcuts=NewWindow

[NewWindow Shortcut Group]
Name=New Window
Exec=sublime -n
TargetEnvironment=Unity

Si vous avez enregistré votre copie de Sublime Text, mais que chaque fois que vous l’ouvrez, vous êtes invité à entrer votre licence, vous devriez essayer d’exécuter cette commande.

sudo chown -R username:username /home/username/.config /sublime-text-3

Remplacez username par le nom d’utilisateur de votre compte. Cela devrait corriger l’erreur d’autorisation dans le cas où vous avez ouvert Sublime Text en tant que root lorsque vous avez entré la licence pour la première fois.

Commencez avec le Sublime Tutor #

Ce tutoriel est inspiré du vimtutor classique. Vous apprendrez quelques raccourcis pratiques pour travailler avec Sublime Text 3. À la fin de ce didacticiel, vous serez familiarisé avec les raccourcis et fonctionnalités les plus importants et les plus fréquemment utilisés de ST.

Installation #

Via [Contrôle des paquets] (https://Sublime.wbond.net/) :

  1. Installez Package Control s’il n’est pas déjà installé : https://packagecontrol.io/installation#st3
  2. Appuyez sur Cmd+Shift+P pour amener la palette de commandes devant
  3. Tapez “Installer le package” et appuyez sur Entrée.
  4. Recherchez “Sublime Tutor” et appuyez sur Entrée pour installer le plugin.

### Installation manuelle :

  1. Assurez-vous que [git][1] est déjà installé.
  2. cd dans le répertoire Packages de Sublime Text 3. Sur Mac, il est généralement réside dans le chemin suivant : ~/Library/Application Support/Sublime Text 3 /Forfaits/. Vous pouvez également ouvrir via un élément de menu : Préférences > Parcourir les packages...
  3. Une fois dans le répertoire Packages, clonez ce référentiel : git clone git@github.org:jai/sublimetutor.git. Vous pouvez également télécharger et extraire la dernière version de votre plate-forme ici : https://github.com/jaipandya/SublimeTutor/releases
  4. Redémarrez Sublime Text

Commencer #

Si vous ne l’avez pas déjà fait, installez Sublime Tutor en suivant les étapes d’installation donnée ci-dessus.

Une fois Sublime Tutor installé, appuyez sur Ctrl+Option+K raccourci clavier pour ouvrir ce fichier dans Sublime Text. Une autre option est d’aller à Option de menu Aide> Sublime Tutor pour l’ouvrir.

Via la palette de commandes :

  1. Cmd+Shift+P pour obtenir la palette de commandes de face.
  2. Tapez Sublime Tutor, sélectionnez la première commande qui apparaît pour démarrer le guides interactifs.
