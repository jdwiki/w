Zones de texte et étiquettes #

Les contrôles de zone de texte sont généralement utilisés pour accepter les entrées de l’utilisateur. Un contrôle de zone de texte peut accepter une ou plusieurs lignes de texte selon les paramètres de l’attribut TextMode.

Les contrôles d’étiquette offrent un moyen simple d’afficher du texte qui peut être modifié d’une exécution d’une page à l’autre. Si vous souhaitez afficher un texte qui ne change pas, vous utilisez le texte littéral.

Syntaxe de base du contrôle de texte :

<asp:TextBox ID="txtstate" runat="server" ></asp:TextBox>

Propriétés communes de la zone de texte et des étiquettes :

Propriétés Descriptif
ModeTexte Spécifie le type de zone de texte. SingleLine crée une zone de texte standard, MultiLINE crée une zone de texte qui accepte plus d’une ligne de texte et le mot de passe entraîne le masquage des caractères saisis. La valeur par défaut est SingleLine.
Texte Le contenu textuel de la zone de texte.
LongueurMax Le nombre maximal de caractères pouvant être saisis dans la zone de texte.
Enveloppe Il détermine si le texte s’habille automatiquement ou non pour une zone de texte multiligne ; la valeur par défaut est true.
Lecture seule Détermine si l’utilisateur peut modifier le texte dans la zone ; la valeur par défaut est false, c’est-à-dire que l’utilisateur peut modifier le texte.
Colonnes La largeur de la zone de texte en caractères. La largeur réelle est déterminée en fonction de la police utilisée pour la saisie de texte.
Lignes La hauteur d’une zone de texte multiligne en lignes. La valeur par défaut est 0, signifie une zone de texte sur une seule ligne.
L’attribut le plus utilisé pour un contrôle d’étiquette est ‘Texte’, ce qui implique le texte affiché sur l’étiquette.

Cases à cocher et boutons radio #

Une case à cocher affiche une seule option que l’utilisateur peut cocher ou décocher et les boutons radio présentent un groupe d’options parmi lesquelles l’utilisateur peut sélectionner une seule option.

Pour créer un groupe de boutons radio, vous spécifiez le même nom pour l’attribut GroupName de chaque bouton radio du groupe. Si plusieurs groupes sont requis dans un même formulaire, spécifiez un nom de groupe différent pour chaque groupe.

Si vous souhaitez que la case à cocher ou le bouton radio soit sélectionné lors de l’affichage initial du formulaire, définissez son attribut Checked sur true. Si l’attribut Checked est défini sur true pour plusieurs boutons radio dans un groupe, seul le dernier est considéré comme true.

Syntaxe de base de la case à cocher :

<asp:CheckBox ID= "chkoption" runat= "Server"> </asp:CheckBox>

Syntaxe de base du bouton radio :

<asp:RadioButton ID= "rdboption" runat= "Server"> </asp: RadioButton>

Propriétés communes des cases à cocher et des boutons radio :

Propriétés Descriptif
Texte Le texte affiché à côté de la case à cocher ou du bouton radio.
Vérifié Spécifie s’il est sélectionné ou non, la valeur par défaut est false.
NomGroupe Nom du groupe auquel appartient le contrôle.

Contrôles de liste #

ASP.NET fournit les contrôles suivants

  • La liste déroulante
  • Zone de liste
  • Liste des boutons radio
  • Liste des cases à cocher
  • Liste à puces

Ces contrôles permettent à un utilisateur de choisir parmi un ou plusieurs éléments de la liste. Les zones de liste et les listes déroulantes contiennent un ou plusieurs éléments de liste. Ces listes peuvent être chargées soit par code, soit par l’éditeur ListItemCollection.

Syntaxe de base du contrôle list box :

<asp:ListBox ID="ListBox1" runat="server" AutoPostBack="True"    OnSelectedIndexChanged="ListBox1_SelectedIndexChanged">
</asp:ListBox>

Syntaxe de base du contrôle de liste déroulante :

<asp:DropDownList ID="DropDownList1" runat="server" AutoPostBack="True"   OnSelectedIndexChanged="DropDownList1_SelectedIndexChanged">
</asp:DropDownList>

Propriétés communes des listes déroulantes et des listes déroulantes :

Propriétés Descriptif
Articles Collection d’objets ListItem qui représente les éléments du contrôle. Cette propriété renvoie un objet de type ListItemCollection.
Lignes Spécifie le nombre d’éléments affichés dans la boîte. Si la liste réelle contient plus de lignes que celles affichées, une barre de défilement est ajoutée.
IndexSélectionné L’index de l’élément actuellement sélectionné. Si plus d’un élément est sélectionné, alors l’index du premier élément sélectionné. Si aucun élément n’est sélectionné, la valeur de cette propriété est -1.
ValeurSélectionnée La valeur de l’élément actuellement sélectionné. Si plusieurs éléments sont sélectionnés, la valeur du premier élément sélectionné. Si aucun élément n’est sélectionné, la valeur de cette propriété est une chaîne vide ("").
ModeSélection Indique si une zone de liste autorise les sélections simples ou multiples.
Propriétés communes de chaque objet d’élément de liste :
Propriétés Descriptif
Texte Le texte affiché pour l’élément.
Sélectionné Une valeur de chaîne associée à l’élément.
Valeur Indique si l’élément est sélectionné.

Il est important de noter que :

  • Pour travailler avec les éléments d’une liste déroulante ou d’une zone de liste, vous utilisez le Propriété Items du contrôle. Cette propriété renvoie un Objet ListItemCollection qui contient tous les éléments de la liste.
  • L’événement SelectedIndexChanged est déclenché lorsque l’utilisateur sélectionne un élément différent d’une liste déroulante ou d’une zone de liste.

Liste des boutons radio et liste des cases à cocher #

Une liste de boutons radio présente une liste d’options mutuellement exclusives. Une liste de cases à cocher présente une liste d’options indépendantes. Ces contrôles contiennent une collection d’objets ListItem auxquels il est possible de faire référence via la propriété Items du contrôle.

Syntaxe de base de la liste des boutons radio :

<asp:RadioButtonList ID="RadioButtonList1" runat="server" AutoPostBack="True" 
   OnSelectedIndexChanged="RadioButtonList1_SelectedIndexChanged">
</asp:RadioButtonList>

Syntaxe de base de la liste des cases à cocher :

<asp:CheckBoxList ID="CheckBoxList1" runat="server" AutoPostBack="True" 
   OnSelectedIndexChanged="CheckBoxList1_SelectedIndexChanged">
</asp:CheckBoxList>

Propriétés communes des listes de cases à cocher et de boutons radio :

Propriétés Descriptif
RépéterMise en page Cet attribut spécifie si les balises de table ou le flux html normal à utiliser lors du formatage de la liste lors de son rendu. La valeur par défaut est Table.
RépéterDirection Il précise le sens dans lequel les commandes doivent être répétées. Les valeurs disponibles sont Horizontal et Vertical. La valeur par défaut est Verticale.
RépéterColonnes Il précise le nombre de colonnes à utiliser lors de la répétition des contrôles ; la valeur par défaut est 0.

Listes à puces et listes numérotées #

Le contrôle de liste à puces crée des listes à puces ou des listes numérotées. Ces contrôles contiennent une collection d’objets ListItem auxquels il est possible de faire référence via la propriété Items du contrôle.

Syntaxe de base d’une liste à puces :

<asp:BulletedList ID="BulletedList1" runat="server">
</asp:BulletedList>

Propriétés communes de la liste à puces :

Propriétés Descriptif
Style de puce Cette propriété spécifie le style et l’apparence des puces ou des numéros.
RépéterDirection Il précise le sens dans lequel les commandes doivent être répétées. Les valeurs disponibles sont Horizontal et Vertical. La valeur par défaut est Verticale.
RépéterColonnes Il précise le nombre de colonnes à utiliser lors de la répétition des contrôles ; la valeur par défaut est 0.

Contrôle de lien hypertexte #

Le contrôle HyperLink est comme l’élément HTML .

Syntaxe de base pour un contrôle de lien hypertexte :

<asp:HyperLink ID="HyperLink1" runat="server">
   HyperLink
</asp:HyperLink>

Il a les propriétés importantes suivantes :

Propriétés Descriptif
ImageUrl Chemin de l’image à afficher par le champ.
NavigateUrl URL du lien cible.
Texte Le texte à afficher comme lien.
Cible La fenêtre ou le cadre qui charge la page liée.

Contrôle des images #

Le contrôle d’image est utilisé pour afficher des images sur la page Web, ou un texte alternatif, si l’image n’est pas disponible.

Syntaxe de base pour un champ image :

<asp:Image ID="Image1" runat="server">

Il a les propriétés importantes suivantes :

Propriétés Descriptif
Texte alternatif Texte alternatif à afficher en l’absence de l’image.
ImageAlign Options d’alignement du champ.
ImageUrl Chemin de l’image à afficher par le champ.
