Introduction aux requêtes #

Tableau des sources

|Ligne|A|B|C|D| |—:| :——: | :——: | :——: | —- : | |1|Code|Produit|Couleur|Prix| |2|1|stylo|rouge|500| |3|2|stylo|bleu|-50| |4|3|stylo|rouge|0| |5|4|crayon|bleu|17| |6|5|crayon|vert|-1.5|

pour tout sélectionner :

= QUERY(A1:D5, "select *")

ou

= QUERY(A1:D5, "select A, B, C, D")

ou convertissez la plage de données en tableau et utilisez cette formule :

= QUERY({A1:D5}, "select Col1, Col2, Col3, Col4")

Trier avec QUERY() #

Un B C D
1 stylo rouge 500
2 stylo bleu -50
3 stylo rouge 0
4 crayon bleu 17
5 crayon vert -1.5

Pour trier par colonne D avec “trier par”:

=QUERY("A1:D6","select * order by D desc",1)

Filtrage avec QUERY() #

Un B C D
1 stylo rouge 500
2 stylo bleu -50
3 stylo rouge 0
4 crayon bleu 17
5 crayon vert -1.5

Pour ne renvoyer que les données “crayon”:

=QUERY("A1:D6","select * where B='pencil' ",1)

Pour renvoyer uniquement les lignes contenant “pen” (toutes les lignes) :

=QUERY("A1:D6","select * where B contains 'pen' ",1)

Pour ne renvoyer que les lignes dont le prix est supérieur à 0 :

=QUERY("A1:D6","select * where D>0 ",1)

Notez que les chaînes de texte nécessitent des apostrophes, contrairement aux valeurs numériques.

Filtrer une requête par un résultat d’agrégation #

=QUERY(QUERY(A1:D6,“select C,SUM(D) group by C”,1)(“select Col2>0”,1)
