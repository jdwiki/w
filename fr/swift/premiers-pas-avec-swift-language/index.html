Votre premier programme Swift #

Écrivez votre code dans un fichier nommé hello.swift :

print("Hello, world!")
  • Pour compiler et exécuter un script en une seule étape, utilisez swift depuis le terminal (dans un répertoire où se trouve ce fichier) :

Pour lancer un terminal, appuyez sur CTRL+ALT+T sur Linux, ou trouvez-le dans Launchpad sur macOS. Pour changer de répertoire, entrez cdnom_répertoire (ou cd .. pour revenir en arrière)

$ swift bonjour.swift
Bonjour, tout le monde !

Un compilateur est un programme informatique (ou un ensemble de programmes) qui transforme le code source écrit dans un langage de programmation (le langage source) en un autre langage informatique (le langage cible), ce dernier ayant souvent une forme binaire connu sous le nom de code objet. (Wikipédia)

  • Pour compiler et exécuter séparément, utilisez swiftc :
$ swiftc bonjour.swift

Cela compilera votre code dans le fichier hello. Pour l’exécuter, entrez ./, suivi d’un nom de fichier.

$ ./bonjour
Bonjour, tout le monde !
  • Ou utilisez le Swift REPL (Read-Eval-Print-Loop), en tapant swift depuis la ligne de commande, puis en saisissant votre code dans l’interpréteur :

Code:

<pré> func greet(nom : chaîne, nom de famille : chaîne) { print(“Greetings (name) (surname)”) }

let myName = “Homer” laissez monNom = “Simpson”

saluer(nom : monNom, nom de famille : monNom)

Découpons ce gros code en morceaux :

  • func greet(name: String, surname: String) { // corps de la fonction } - crée une fonction qui prend un name et un surname.

  • print("Greetings \(name) \(surname)") - Cela imprime sur la console “Greetings”, puis name, puis surname. Fondamentalement, \(variable_name) imprime la valeur de cette variable.

  • let myName = "Homer" et let mySurname = "Simpson" - créez des constantes (variables dont vous ne pouvez pas modifier la valeur) en utilisant let avec des noms : myName, mySurname et des valeurs : "Homer", "Simpson" respectivement.

  • greet(name: myName, surname: mySurname) - appelle une fonction que nous avons créée précédemment en fournissant les valeurs de constantes myName, mySurname.

Exécuter en utilisant REPL :

$ rapide
Bienvenue sur Apple Swift. Tapez :aide pour obtenir de l'aide.
1> func greet(nom : chaîne, nom : chaîne) { 
2. print("Salutations \(nom) \(prénom)") 
3. } 
4>
5> let myName = "Homer"
monNom : Chaîne = "Homer"
6> let mySurname = "Simpson"
monNom : Chaîne = "Simpson"
7> saluer(nom : monNom, nom de famille : monNom)
Salutations Homer Simpson
8> ^D

Appuyez sur CTRL+D pour quitter REPL.

Votre premier programme en Swift sur un Mac (en utilisant un Playground) #

Depuis votre Mac, téléchargez et installez Xcode depuis le Mac App Store en suivant [ce lien][1].

Une fois l’installation terminée, ouvrez Xcode et sélectionnez Get started with a Playground :

[![entrez la description de l’image ici][2]][2]

Sur le panneau suivant, vous pouvez donner un nom à votre Playground ou vous pouvez le laisser “MyPlayground” et appuyer sur Suivant :

[![entrez la description de l’image ici][3]][3]

Sélectionnez un emplacement où enregistrer le Playground et appuyez sur Créer :

[![entrez la description de l’image ici][4]][4]

Le Playground s’ouvrira et votre écran devrait ressembler à ceci :

[![entrez la description de l’image ici][5]][5]

Maintenant que Playground est à l’écran, appuyez sur ⇧ + cmd + Y pour afficher la zone de débogage.

Enfin, supprimez le texte à l’intérieur de Playground et tapez :

print("Hello world")

Vous devriez voir “Hello world” dans la zone de débogage et “Hello world\n” dans la barre latérale de droite :

[![entrez la description de l’image ici][6]][6]

Toutes nos félicitations! Vous avez créé votre premier programme dans Swift !

Installer Swift #

Tout d’abord, [téléchargez][1] le compilateur et les composants.

Ensuite, ajoutez Swift à votre chemin. Sur macOS, l’emplacement par défaut de la chaîne d’outils téléchargeable est /Library/Developer/Toolchains. Exécutez la commande suivante dans Terminal :

export PATH=/Library/Developer/Toolchains/swift-latest.xctoolchain/usr/bin:"${PATH}"

Sous Linux, vous devrez installer clang :

$ sudo apt-get install clang

Si vous avez installé la chaîne d’outils Swift dans un répertoire autre que la racine du système, vous devrez exécuter la commande suivante, en utilisant le chemin réel de votre installation Swift :

$ export PATH=/path/to/Swift/usr/bin:"${PATH}"

Vous pouvez vérifier que vous disposez de la version actuelle de Swift en exécutant cette commande :

$ swift --version

[1] : https://swift.org/download/

Votre premier programme dans l’application Swift Playgrounds sur iPad #

L’application Swift Playgrounds est un excellent moyen de commencer à coder Swift lors de vos déplacements. Pour l’utiliser:

1- Téléchargez [Swift Playgrounds][1] pour iPad depuis l’App Store.

[![entrez la description de l’image ici][2]][2]

2- Ouvrez l’application.

3- Dans l’onglet Mes terrains de jeu, appuyez sur *+ dans le coin supérieur gauche, puis sélectionnez Modèle vierge.

4- Entrez votre code.

5- Appuyez sur Exécuter mon code pour exécuter votre code.

6- Au début de chaque ligne, le résultat sera stocké dans un petit carré. Appuyez dessus pour révéler le résultat.

7- Pour parcourir lentement le code pour le tracer, appuyez sur le bouton à côté de Exécuter mon code.

[1] : https://itunes.apple.com/us/app/swift-playgrounds/id908519492?mt=8 [2] : https://i.stack.imgur.com/Ig7wZ.png

Valeur facultative et énumération facultative #

Type Optionals, qui gère l’absence de valeur. Les options indiquent soit “il y a une valeur, et elle est égale à x” soit “il n’y a pas de valeur du tout”.

Un Optional est un type à part entière, en fait l’un des nouveaux enums super puissants de Swift. Il a deux valeurs possibles, “None” et “Some(T)”, où T est une valeur associée du type de données correct disponible dans Swift.

Jetons un œil à ce morceau de code par exemple :

let x: String? = "Hello World"

if let y = x {
   print(y)
}

En fait, si vous ajoutez une instruction print(x.dynamicType) dans le code ci-dessus, vous verrez ceci dans la console :

Optional<String>

Chaîne de caractères? est en fait du sucre syntaxique pour Optional, et Optional est un type à part entière.

Voici une version simplifiée de l’en-tête de Optional, que vous pouvez voir en cliquant sur le mot Optional dans votre code depuis Xcode :

enum Optional<Wrapped> {

 /// The absence of a value.
 case none

 /// The presence of a value, stored as `Wrapped`.
 case some(Wrapped)
}

Optional est en fait une énumération, définie par rapport à un type générique Wrapped. Il a deux cas : .none pour représenter l’absence d’une valeur et .some pour représenter la présence d’une valeur, qui est stockée comme sa valeur associée de type Wrapped.

Permettez-moi de répéter : String ? n’est pas une String mais une Optional<String>. Le fait que Optional soit un type signifie qu’il a ses propres méthodes, par exemple map et flatMap.
