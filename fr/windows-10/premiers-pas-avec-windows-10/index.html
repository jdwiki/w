Installation ou configuration #

L’installation sur Windows 10 est aussi simple que jamais, et malgré les (vraies) rumeurs qui circulent selon lesquelles Windows 10 est la dernière version “normale” de Windows, il a toujours été le même processus.

Installer une nouvelle version

  1. Obtenez l'image Windows (Passez directement à l'étape 4 si vous avez un disque acheté en magasin)
    2. Accédez au site Windows technet et téléchargez la version appropriée de Windows 10. Vous devrez entrer la clé de produit que vous avez achetée.
  2. Télécharger un imageur USB
    3. Recherchez et téléchargez l'outil d'imagerie USB Rufus. Lance ça.
  3. Chargez l'image sur votre clé USB
    4. Cliquez sur le menu déroulant en haut et sélectionnez votre clé USB. Cliquez ensuite sur la deuxième liste déroulante et sélectionnez l'option d'image ISO. Enfin, cliquez sur l'image à côté de la liste déroulante et recherchez votre fichier ISO. Sélectionnez-le puis cliquez sur 'Démarrer'. Lorsque cela est fait, éjectez votre clé USB.
  4. Installation
    5. Éteignez et rallumez votre ordinateur portable. Surveillez de très près tout texte qui apparaît indiquant "Appuyez sur pour accéder au menu de démarrage". Lorsque vous voyez cela, notez mentalement le nom du bouton et trouvez-le sur votre clavier. Insérez ensuite votre support d'installation et redémarrez à nouveau, et lorsque le texte apparaît, appuyez sur le bouton. Si tout se passe bien, vous devriez avoir la possibilité d'exécuter le support d'installation. Une fois que cela s'est produit, suivez simplement les invites à l'écran pour installer Windows 10.

Mise à jour

  1. Télécharger l'iso
    2. Suivez l'étape 1 du guide ci-dessus
  2. Monter le fichier ISO
    3. Faites un clic droit sur ce fichier et cliquez sur l'option 'Monter'
  3. Exécuter le programme d'installation
    4. Allez dans "Ce PC" et ouvrez l'image disque (devrait apparaître comme un disque). Ensuite, exécutez le fichier 'setup.exe' dedans. Enfin, suivez simplement les invites à l'écran. Cela nécessite des droits d'administrateur et prend un certain temps, mais une fois que cela s'est produit, tous vos fichiers seront là où ils étaient avant, ce ne sera que Windows 10 !

    OneGet et les gestionnaires de paquets #

    OneGet était à l’origine un produit du Open Source Technology Center de Microsoft. Non seulement il est inspiré des gestionnaires de packages Linux open source, mais OneGet lui-même est également open source. Il fait maintenant partie de PowerShell

    Contrairement aux gestionnaires de packages basés sur Unix (tels que apt-get, yum ou dpkg), Windows autorise l’utilisation de Oneget via PowerShell. Pour ce faire, vous devez d’abord ouvrir un Powershell en tant qu’administrateur et exécuter Set-ExecutionPolicy RemoteSigned. Cela permet de s’assurer que les scripts Powershell disponibles dans Oneget sont autorisés à s’exécuter. Ensuite (en utilisant la même instance PowerShell), exécutez Import-Module –Name OneGet. Cela importe le module PoerShell. Redémarrez ensuite votre instance powershell et exécutez Get-Command –Module OneGet pour voir toutes les commandes disponibles. Pour installer un package, exécutez la commande Find-Package | <Package_Name. Lorsque vous êtes invité à installer Nuget, cliquez sur Oui.

    Ajout de Chocolatey comme fournisseur à OneGet #

    Malheureusement, OneGet n’a pas le fournisseur de packages chocolatey installé par défaut, mais il est possible de l’ajouter.

    Ouvrez une fenêtre powershell et exécutez get-packageprovider -name chocolatey. Vous devriez alors voir le message ci-dessous :

    Le fournisseur ‘chocolatey v2.8.5.130’ n’est pas installé. chocolaté peut être téléchargé manuellement depuis https://oneget.org/ChocolateyPrototype-2.8.5.130.exe et installé. Souhaitez-vous que PackageManagement télécharge automatiquement et installer ‘chocolatey’ ?

    [Y] Oui [N] Non [S] Interrompre [?] Aide (la valeur par défaut est “Y”) :

    Tapez ‘Y’ puis appuyez sur Entrée.

    Name                     Version          
----                     -------           
Chocolatey               2.8.5.130

    Trouver des forfaits #

    Pour la plupart des programmes connus, le nom de l’application fonctionne généralement (par exemple, “Chrome”). mais, tout comme le chocolat peut être difficile à naviguer, il en va de même pour OneGet. Heureusement, OneGet a une implémentation de choco search. Cette commande est maintenant find-package <Package Name>. Cela vous donnera une liste de packages comme indiqué ci-dessous:

    Name                           Version          Source                         Summary
----                           -------          ------                         -------
thunderbird                    45.2.0           chocolatey                     A free email client from Mozilla
Google Chrome                  47.5.8           OneGet                         A high end web browser

    Installer des packages #

    Pour installer un package, vous devez exécuter un PowerShell en tant qu’administrateur et exécuter la commande install-package <PAckage Name>

    Le ou les packages proviennent d’une source de package qui n’est pas marquée comme de confiance. Êtes-vous sûr de vouloir installer le logiciel de ‘chocolatey’ ? [Y] Oui [A] Oui à tous [N] Non [L] Non à tous [S] Suspendre [?] Aide (la valeur par défaut est “N”) :

    Pour cela, vous pouvez simplement appuyer sur y et/ou entrer. Vous verrez alors un affichage similaire à celui ci-dessous

    Name                           Version          Source           Summary
----                           -------          ------           -------
<Package Name>                 <Most recent>    <Server name>    <Description>

    Désinstallation #

    Pour désinstaller un package, il vous suffit d’utiliser la commande ci-dessous

    uninstall-package vlc
