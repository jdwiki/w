Variables globales dans les classes

  • Informations utilisateur

Sur cette page

  • Informations utilisateur

Dans cette rubrique, je voudrais mentionner toutes les variables globales possibles qui peuvent être utilisées dans le code Apex. Comme [Classe UserInfo][1].

Je suggère que nous listions simplement des classes/variables globales et des liens. Si vous connaissez une classe/variable globale mais que vous ne trouvez pas de documentation, veuillez fournir autant d’informations que possible.

[1] : https://developer.salesforce.com/docs/atlas.en-us.apexcode.meta/apexcode/apex_methods_system_userinfo.htm

Informations utilisateur #

[getFirstName()][1] - renvoie le prénom de l’utilisateur du contexte. [getLastName()][2] - renvoie le nom de famille de l’utilisateur du contexte.

[1] : https://developer.salesforce.com/docs/atlas.en-us.apexcode.meta/apexcode/apex_methods_system_userinfo.htm#apex_System_UserInfo_getFirstName [2] : https://developer.salesforce.com/docs/atlas.en-us.apexcode.meta/apexcode/apex_methods_system_userinfo.htm#apex_System_UserInfo_getLastName
