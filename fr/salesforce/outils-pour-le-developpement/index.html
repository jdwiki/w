IDE #

Une liste des IDE disponibles pour créer des classes, des déclencheurs, des pages/composants Visualforce/Lightning.

  • [Force.com IDE][1] – plug-in pour Eclipse
  • [JedIDE][12] – plugin pour IntelliJ IDEA & [standalone Force.com IDE][13]
  • [MavensMate][2] - plugin pour [Sublime Text][14] et [Atom][15] et [VS Code][11]
  • [FuseIT SFDC Explorer][3] - Il s’agit d’un outil autonome
  • [Welkin Suite] [4] - Il s’agit d’un outil autonome ;
  • [Illuminated Cloud] [6] - plugin pour IntelliJ IDE
  • [Aside.io][5] - EDI basé sur le Web
  • [Cloud 9][7] - IDE basé sur le Web
  • [VimAwesome][8] - Plugin VIM pour Force.com
  • [HaoIDE][9] - Plug-in Sublime Text pour Force.com
  • [Metaforce][10] - Une application Chrome légère pour le développement de Salesforce
