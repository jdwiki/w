Fonction boîte à moustaches #

[Matplotlib][1] a sa propre implémentation de [boxplot][2]. Les aspects pertinents de cette fonction sont que, par défaut, la boîte à moustaches affiche la médiane (percentile 50 %) avec une ligne rouge. La boîte représente Q1 et Q3 (percentiles 25 et 75), et les moustaches donnent une idée de la plage des données (éventuellement à Q1 - 1,5IQR ; Q3 + 1,5IQR ; étant IQR la plage interquartile, mais cela manque de confirmation ). Notez également que les échantillons au-delà de cette plage sont affichés sous forme de marqueurs (ceux-ci sont nommés dépliants).

REMARQUE : Toutes les implémentations de boxplot ne suivent pas les mêmes règles. Peut-être que le diagramme de boîte à moustaches le plus courant utilise les moustaches pour représenter le minimum et le maximum (ce qui rend les flyers inexistants). Remarquez également que ce tracé est parfois appelé boîte à moustaches et boîte à moustaches schéma.

La recette suivante montre certaines des choses que vous pouvez faire avec l’implémentation matplotlib actuelle de boxplot :

import matplotlib.pyplot as plt
import numpy as np

X1 = np.random.normal(0, 1, 500)
X2 = np.random.normal(0.3, 1, 500)

# The most simple boxplot
plt.boxplot(X1)
plt.show()

# Changing some of its features
plt.boxplot(X1, notch=True, sym="o") # Use sym="" to shown no fliers; also showfliers=False
plt.show()

# Showing multiple boxplots on the same window
plt.boxplot((X1, X2), notch=True, sym="o", labels=["Set 1", "Set 2"])
plt.show()

# Hidding features of the boxplot
plt.boxplot(X2, notch=False, showfliers=False, showbox=False, showcaps=False, positions=[4], labels=["Set 2"])
plt.show()

# Advanced customization of the boxplot
line_props = dict(color="r", alpha=0.3)
bbox_props = dict(color="g", alpha=0.9, linestyle="dashdot")
flier_props = dict(marker="o", markersize=17)
plt.boxplot(X1, notch=True, whiskerprops=line_props, boxprops=bbox_props, flierprops=flier_props)
plt.show()

Si vous avez l’intention de faire une personnalisation avancée de votre boîte à moustaches, vous devez savoir que les dictionnaires [props][8] que vous construisez (par exemple) :

line_props = dict(color="r", alpha=0.3)
bbox_props = dict(color="g", alpha=0.9, linestyle="dashdot")
flier_props = dict(marker="o", markersize=17)
plt.boxplot(X1, notch=True, whiskerprops=line_props, boxprops=bbox_props, flierprops=flier_props)
plt.show()

… font référence principalement (sinon à tous) aux objets [Line2D][9]. Cela signifie que seuls les arguments disponibles dans cette classe sont modifiables. Vous remarquerez l’existence de mots clés tels que whiskerprops, boxprops, flierprops et capprops. Ce sont les éléments dont vous avez besoin pour fournir un dictionnaire d’accessoires afin de le personnaliser davantage.

REMARQUE : personnalisation supplémentaire de la boîte à moustaches à l’aide de cette implémentation pourrait s’avérer difficile. Dans certains cas, l’utilisation d’autres matplotlib des éléments tels que [patches][10] pour construire son propre boxplot peuvent être avantageux (évolutions importantes de l’élément caisson par exemple).
