En ce qui concerne ionic2 le constructeur : en termes simples, nous l’utilisons pour créer une instance de nos plugins, services, etc. par exemple : vous avez une page (vue) où vous souhaitez afficher la liste de tous les étudiants, et vous avez un json qui contient tous les étudiants (ce fichier est votre fichier de données) ce que vous devez faire est de créer un service dans ce service, vous allez créer une méthode et lancer une requête http.get pour obtenir les données json, donc ici vous avez besoin quelle? http faites simplement de cette façon:

Exemple de méthode de service étudiant pour l’utilisation de Http dans le constructeur #

import {Http} from '@angular/http';
@Injectable()
export class StudentService{
    constructor(public http: Http){}
    getAllStudents(): Observable<Students[]>{
        return this.http.get('assets/students.json')
        .map(res => res.json().data)     
        }
    }

remarquez à nouveau le constructeur si nous voulons utiliser cette méthode de service, nous irons à notre vue/page et :

import {StudentService} from './student.service';
import { SocialSharing } from '@ionic-native/social-sharing';
export class HomePage implements OnInit {

  constructor(public _studentService: StudentService, public socialSharing: SocialSharing) {
   }

remarquez encore une fois le constructeur ici, nous créons une instance de StudentService dans le constructeur et encore une chose, nous utilisons le plugin socialSharing afin d’utiliser que nous créons également une instance de cela dans le constructeur.

Méthode ngOnInit pour obtenir la liste des étudiants en charge #

OnInit : c’est une chose vraiment incroyable dans ionic2 ou nous pouvons dire dans AngularJs2. Avec le même exemple ci-dessus, nous pouvons voir ce qu’est ngOnInit. Vous êtes donc prêt avec la méthode de service, maintenant dans votre vue/page, vous voulez que les données de la liste des étudiants soient disponibles dès que votre vue va apparaître, cela devrait être la première opération qui se produit automatiquement au chargement, car au fur et à mesure que la vue charge l’étudiant la liste doit être visible. Ainsi, la classe implémente OnInit et vous définissez ngOnInit. Exemple:
