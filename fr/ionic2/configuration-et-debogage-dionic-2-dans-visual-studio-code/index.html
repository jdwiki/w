Configuration et débogage d'Ionic 2 dans Visual Studio Code

Visual Studio est un IDE open source qui fournit intellisense et une fonction d’édition pour le code. Cet IDE prend en charge de nombreux langages tels que (Ionic, C, C #, AngularJs, TypeScript, Android, etc.). Ces langages sont capables d’exécuter du code en ajoutant ses extensions dans VSCode. En utilisant VSCode, nous avons pu exécuter et déboguer le code de différentes langues.

Créez et ajoutez votre projet Ionic dans VSCode #

VsCode n’est pas en mesure de créer le projet ionique car il s’agit d’un éditeur de code. Vous pouvez donc créer votre projet ionique par CLI ou cmd. créez votre projet par la commande ci-dessous

$ ionic start appName blank

La commande ci-dessus est utilisée pour créer une application ionique de modèle vierge. Ionic2 fournit trois types de modèles ** vierges, onglets et menu latéral **. Alors. vous pouvez remplacer le modèle vierge par deux autres modèles selon vos besoins.

Maintenant, votre projet Ionic est créé. Ainsi, vous pouvez ajouter votre projet dans VSCode pour le modifier. Pour ajouter votre projet, suivez les points ci-dessous.

  1. Allez dans le menu Fichier dans VScode.
  2. Cliquez sur Ouvrir le dossier dans le menu Fichier.
  3. Recherchez et ouvrez votre dossier de projet.

Vous pouvez ouvrir directement le dossier en utilisant la touche de raccourci ctrl+O ou ctrl+k

`

Installation de VSCode #

Tout d’abord, vous devez télécharger et installer le VSCode. Cette dernière version de VSCode est disponible en téléchargement sur son [Site officiel][1] . Après avoir téléchargé le VSCode, vous devez l’installer et l’ouvrir.

 Introduction of Extensions in VSCode

VSCode est un éditeur ouvert, il fournit donc un éditeur pour toutes les langues, mais pour exécuter un code, vous devez ajouter l’extension pour cette langue particulière. Pour exécuter et modifier votre code ionique, vous devez ajouter l’extension ionic2-vscode dans votre VSCode. Dans le côté gauche de l’éditeur VSCode, il y a 5 icônes dans lesquelles l’icône la plus basse est utilisée pour l’extension. Les extensions que vous pouvez obtenir en utilisant touche de raccourci (ctrl+maj+X).

Add Extension for Ionic2 in VsCode

En appuyant sur ctrl+shift+X vous avez montré la partie de l’extension où en haut trois points sont affichés … ces points sont connus sous le nom d’icône plus. En cliquant dessus, une boîte de dialogue s’affiche ouvert et affiche le nombre d’options à choisir. Vous pouvez choisir l’option en fonction de vos besoins, mais pour obtenir toutes les extensions, vous devez sélectionner Extension recommandée affichée. Dans la liste de toutes les extensions, vous pouvez installer votre extension (ionic2- vscode),npm

[1] : https://code.visualstudio.com/docs/setup/setup-overview

Exécutez et déboguez votre projet Ionic #

> Exécuter et déboguer dans Chrome

Pour exécuter le projet ionique, utilisez la commande ci-dessous dans terminal ou cmd ou CLI

 $ ionic serve

Pour déboguer le projet ionique, vous devez d’abord ajouter l’extension (Debugger for chrome) puis configurer le fichier launch.json comme ceci.

 {
        "version": "0.2.0",
        "configurations": [
           
            {
          "name": "Launch in Chrome",
          "type": "chrome",
          "request": "launch",
          "url": "http://localhost:8100",
          "sourceMaps": true,
          "webRoot": "${workspaceRoot}/src"
        }
]
}

** *** Exécuter et déboguer sous Android ****

Pour le projet Run ionic dans Android, vous devez ajouter la plate-forme Android par la commande ci-dessous dans le terminal ou cmd ou CLI :

$ ionic cordova platform add android

Construisez Android avec cette commande

$ ionic cordova build android

Exécuter la commande pour la plate-forme Android

$ ionic cordova run android

Maintenant, votre application s’exécute sur un véritable appareil Android.
