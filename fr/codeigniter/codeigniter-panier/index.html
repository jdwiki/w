Nous pouvons utiliser la bibliothèque de paniers d’achat de CI lorsque nous construisons un site de commerce électronique. nous pouvons configurer l’ajout au panier, mettre à jour les éléments du panier, supprimer des éléments du panier et même effacer les fonctionnalités du panier à l’aide de cette bibliothèque.

De CodeIgniter Doc : La classe Cart permet d’ajouter des éléments à une session qui reste active pendant qu’un utilisateur navigue sur votre site. Ces articles peuvent être récupérés et affichés dans un format standard de “panier d’achat”, permettant à l’utilisateur de mettre à jour la quantité ou de supprimer des articles du panier.

Ajout d’articles dans le panier #

Vous devez créer des fonctions dans un contrôleur telles que l’insertion, la mise à jour, la suppression et l’effacement du panier, etc. Par exemple : pour insérer un nouvel article dans le panier, écrivez ci-dessous le code qui accepte la valeur.

$cartItem = array(
'id'      => 'MOTOG5',
'qty'     => 5,
'price'   => 100.99,
'name'    => 'Motorola Moto G5 - 16 GB',
'options' => array(
                'ram' => '3GB',
                'Color' => 'Fine Gold'
             )
);

Et créez des fonctions dans le modèle pour les tâches de panier telles que l’insertion, la mise à jour, la suppression, l’effacement, etc.

ex : pour insérer des articles dans le panier

$this->cart->insert($cartItem);

La méthode insert() renverra le $rowid si vous insérez avec succès un seul élément. afin que vous puissiez vérifier que l’élément a été inséré ou non et afficher le message connexe à l’utilisateur.

et vous pouvez insérer plusieurs éléments comme celui-ci #

$data = array(
    array(
            'id'      => 'sku_123ABC',
            'qty'     => 1,
            'price'   => 39.95,
            'name'    => 'T-Shirt',
            'options' => array('Size' => 'L', 'Color' => 'Red')
    ),
    array(
            'id'      => 'sku_567ZYX',
            'qty'     => 1,
            'price'   => 9.95,
            'name'    => 'Coffee Mug'
    ),
    array(
            'id'      => 'sku_965QRS',
            'qty'     => 1,
            'price'   => 29.95,
            'name'    => 'Shot Glass'
    )
);

$this->cart->insert($data);

Éléments de base du panier CI #

Comme nous pouvons ajouter plusieurs éléments dans le tableau Cart, puis l’ajouter à la session de panier, mais il existe 4 éléments de base dont la classe Cart a besoin pour ajouter des données avec succès dans la session de panier.

  1. identifiant (chaîne)
  2. quantité (nombre)
  3. prix (nombre, décimal)
  4. nom (chaîne)

Et si vous souhaitez ajouter plus d’options concernant le produit, vous pouvez utiliser le 5ème élément qui est “options”. vous pouvez définir un tableau d’options dans cet élément.

Il ressemblera à ceci :

$cartItem = array(
    'id'      => 'MOTOG5',
    'qty'     => 5,
    'price'   => 100.99,
    'name'    => 'Motorola Moto G5 - 16 GB',
    'options' => array(
                    'ram' => '3GB',
                    'Color' => 'Fine Gold'
                 )
);

Afficher les articles du panier #

Vous pouvez afficher les articles du panier en boucle dans le panier ou vous pouvez afficher un seul article du panier.

$cartContents = $this->cart->contents();

Cela renverra un tableau d’éléments du panier afin que vous puissiez parcourir ce tableau en utilisant la boucle foreach.

 foreach ($cartContents as $items){
     echo "ID : ". $items["id"] . "<br>";
     echo "Name : ". $items["name"] . "<br>";
     echo "Quantity : ". $items["qty"] . "<br>";
     echo "Price : ". $items["price"] . "<br>";
  }

Vous pouvez formater ces données sous forme de cellule de tableau ou de div, puis les afficher dans la vue.

Mettre à jour l’article du panier #

Rowid : L’ID de ligne est un identifiant unique qui est généré par le code du panier lorsqu’un article est ajouté au panier. La raison pour laquelle un identifiant unique est créé est que des produits identiques avec différentes options peuvent être gérés par le panier.

Chaque article du panier a un élément rowid et par rowid, vous pouvez mettre à jour l’élément du panier.

$updateItem = array(
    'rowid' => 'b99ccdf16028f015540f341130b6d8ec',
    'qty'   => 3
);

puis sous le code

$this->cart->update($data);

Supprimer les articles du panier #

En utilisant l’élément rowid, vous pouvez supprimer un article du panier. il vous suffit de définir la quantité de l’article sur 0

$deleteItem = array(
    'rowid' => 'b99ccdf16028f015540f341130b6d8ec',
    'qty'   => 0
);

$this->cart->update($data);

cela supprimera l’élément avec ce rowid.
