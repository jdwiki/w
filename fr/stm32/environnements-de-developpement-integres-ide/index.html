  3. Processus d’installation.
  4. Étapes de configuration supplémentaires (le cas échéant).

SW4STM32 : Atelier système pour STM32 #

Introduction #

System Workbench pour STM32 est un IDE gratuit sur Windows, Linux et OS X. Description de [ST Microelectronics][1] :

La chaîne d’outils System Workbench, appelée SW4STM32, est un multi-OS gratuit environnement de développement logiciel basé sur Eclipse, qui prend en charge gamme complète de microcontrôleurs STM32 et cartes associées.

La chaîne d’outils SW4STM32 peut être obtenue sur le site www.openstm32.org, qui comprend des forums, des blogs et des formations pour assistance technique. Une fois inscrits sur ce site, les utilisateurs obtiendront instructions d’installation sur la page Documentation > System Workbench pour procéder au téléchargement de la chaîne d’outils gratuite.

La chaîne d’outils System Workbench et son site Web collaboratif ont été construit par AC6, société de services de formation et de conseil sur systèmes embarqués.

Ce produit est fourni par un tiers non affilié à ST. Pour les dernières informations sur le cahier des charges, reportez-vous au troisième site du parti : www.ac6.fr.

Caractéristiques principales

  • Prise en charge complète des microcontrôleurs STM32, des cartes Nucleo STM32, des kits de découverte et des cartes d’évaluation, ainsi que de STM32 firmware (bibliothèque de périphériques standard ou STM32Cube HAL)
  • Compilateur GCC C/C++
  • Débogueur basé sur GDB
  • Eclipse IDE avec gestion du travail en équipe
  • Compatible avec les plugins Eclipse
  • Support ST-LINK
  • Aucune limite de taille de code
  • Prise en charge de plusieurs systèmes d’exploitation : Windows®, Linux et OS X®

Installation #

  1. Allez sur : http://www.openstm32.org/HomePage .
  2. Enregistrez-vous et connectez-vous au site.
  3. Accédez à : http://www.openstm32.org/Downloading+the+System+Workbench+for+STM32+installer .
  4. Téléchargez la dernière version de votre système d’exploitation.
  5. Exécutez le programme d’installation téléchargé.

[1] : http://www.st.com/content/st_com/en/products/development-tools/software-development-tools/stm32-software-development-tools/stm32-ides/sw4stm32.html

IAR-WARM #

Introduction #

IAR Environnement de développement intégré et optimisation du compilateur C/C++ pour ARM Cortex-M. Description de [ST Microelectronics][1] :

L’IAR-EWARM est une suite logicielle de développement livrée avec fichiers de configuration de périphérique prêts à l’emploi, chargeurs flash et exemple 4300 projets inclus. IAR Embedded Workbench est compatible avec d’autres Compilateurs compatibles ARM®EABI et prenant en charge les cœurs ARM® suivants pour STM32 :

Caractéristiques principales

  1. Composants clés :
  • Environnement de développement intégré avec outils de gestion de projet et éditeur
  • Compilateur C et C++ hautement optimisé pour ARM®
  • Vérification automatique des règles MISRA C (MISRA C:2004)
  • Conformité ARM® EABI et CMSIS
  • Prise en charge étendue du système cible HW
  • Sondes de débogage en circuit I-jet et JTAGjet™-Trace en option
  • Débogage de l’alimentation pour visualiser la consommation d’énergie en corrélation avec le code source
  • Bibliothèques d’exécution, y compris le code source
  • Déplacement de l’assembleur ARM®
  • Outils de liaison et de bibliothèque
  • Débogueur C-SPY® avec simulateur ARM®, prise en charge JTAG et prise en charge du débogage compatible RTOS sur le matériel
  • Plugins RTOS disponibles auprès des fournisseurs IAR Systems et RTOS
  • Plus de 3100 exemples de projets pour cartes d’évaluation de nombreux fabricants différents
  • Guides d’utilisation et de référence au format PDF
  • Aide en ligne contextuelle
  1. Support spécifique aux puces :
  • 4300 exemples de projets y compris pour les cartes d’évaluation STMicroelectronics
  • Prise en charge des applications 4 Go en mode ARM® et Thumb®
  • Chaque fonction peut être compilée en mode ARM® ou Thumb®
  • Génération de code co-processeur VFP Vector Floating Point
  • Prise en charge intrinsèque de NEON™
  1. Prise en charge du débogage matériel :
  • STMicroelectronics ST-LINK V2 : Prend en charge les appareils STM32
  • STMicroelectronics ST-LINK : Prend en charge les appareils STM32
  1. Support RTOS : consulter le site internet de l’IAR http://www.iar.com
  2. Appareils supportés : consulter le site web de l’IAR http://www.iar.com

Atollic - TrueSTUDIO #

Introduction #

IDE C/C++ pour le développement ARM.

Atollic TrueSTUDIO® est testé et vérifié sur les systèmes d’exploitation suivants :

  • Microsoft® Windows® Vista (version 32 bits)
  • Microsoft® Windows® Vista (version 64 bits)
  • Microsoft® Windows® 7 (version 32 bits)
  • Microsoft® Windows® 7 (version 64 bits)
  • Microsoft® Windows® 8 (version 64 bits)
  • Microsoft® Windows® 10 (version 64 bits)
  • Support Linux prévu fin 2016 Q4
  • Prise en charge de Mac OS X prévue 2017 Q2

TrueSTUDIO est uniquement disponible en tant qu’application 32 bits.

Installation #

Le produit Atollic TrueSTUDIO est livré sous forme d’exécutable installateur. Veuillez vous assurer que le compte utilisateur, à partir duquel le le programme d’installation est lancé, a des privilèges d’administration. Il n’y a pas besoin pour l’enregistrement ou la connexion internet lors de l’installation. Lorsque TrueSTUDIO est installé, il fonctionnera en mode Lite si aucune licence n’est détecté.

  1. Allez sur : http://atollic.com/resources/downloads/ .
  2. Téléchargez la dernière version stable ou la dernière version bêta.
  3. Exécutez le programme d’installation.

CoIDE #

Introduction #

CooCox CoIDE, un environnement de développement logiciel gratuit et hautement intégré pour les microcontrôleurs ARM Cortex. Description de [ST Microelectronics][1] :

CoIDE est un environnement de développement logiciel libre basé sur Eclipse et Chaîne d’outils GCC, qui a été personnalisée et simplifiée pour donner aux utilisateurs un accès facile aux microcontrôleurs ARM® Cortex®-M.

Ce produit est fourni par un tiers non affilié à ST. Pour des informations complètes et à jour sur les spécifications et les packages de les pièces achetées, reportez-vous au site Web du tiers www.coocox.org.

Caractéristiques principales

  • Prise en charge complète des microcontrôleurs STM32, des cartes STM32 Nucleo ainsi que des bibliothèques logicielles STM32Cube.
  • Compilateur GCC C/C++.
  • Débogueur basé sur GDB.
  • IDE Eclipse simplifié.
  • Prise en charge ST-Link.
  • Prise en charge multilingue : anglais, chinois.
