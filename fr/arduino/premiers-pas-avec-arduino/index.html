Juste le minimum #

Voici le croquis Arduino “minimum”. Cela peut être chargé dans l’IDE Arduino en choisissant Fichier> Exemples> 01. Bases> Bare Minimum.

void setup() {
  // put your setup code here, to run once
}

void loop() {
  // put your main code here, to run repeatedly
}

Le code de la fonction setup() sera exécuté une fois au démarrage du programme. Ceci est utile pour configurer les broches d’E/S, initialiser les variables, etc. Le code de la fonction loop() sera exécuté à plusieurs reprises jusqu’à ce que l’Arduino soit éteint ou qu’un nouveau programme soit téléchargé. Effectivement, le code ci-dessus ressemble à ceci dans la bibliothèque d’exécution Arduino :

setup();
while(1) {
  loop();
}

Contrairement aux programmes exécutés sur votre ordinateur, le code Arduino ne peut jamais quitter. En effet, le microcontrôleur ne contient qu’un seul programme chargé. Si ce programme se terminait, rien ne dirait au microcontrôleur quoi faire.

Première configuration #

Logiciel nécessaire : [Arduino IDE][1]

[![EDI Arduino][2]][2]

Installer #

La plupart des cartes compatibles Arduino ont un port USB et sont livrées avec un câble USB. Branchez l’Arduino à l’aide du câble USB et démarrez l’IDE Arduino.

L’IDE Arduino démarrera avec une nouvelle esquisse, généralement avec les fonctions vides setup() et loop(). C’est suffisant pour télécharger sur une carte Arduino, mais cela ne fera rien du tout. L’exemple de croquis “Blink” fonctionne comme un test simple lors de la première utilisation d’une carte Arduino. Allez dans Fichier → Exemples → 01.Les bases → Cligner. Cela ouvrira une nouvelle fenêtre avec l’esquisse Blink.

Sélectionnez votre planche. Allez dans Outils → Conseil → [nom de votre carte Arduino].

[![Outils, tableau][3]][3]

Sélectionnez le port COM de votre carte. La plupart des cartes compatibles Aurduino créeront un faux port COM, qui est utilisé pour la communication série (débogage) et pour programmer la carte. COM 1 est généralement déjà présent, et votre carte en créera un nouveau, par ex. COM 4. Sélectionnez ceci dans Outils → Port→ COM 4 (ou autre numéro COM).

Certaines cartes ont des paramètres supplémentaires dans le menu Outils, tels que la vitesse d’horloge. Celles-ci varient d’une carte à l’autre, mais généralement un ensemble acceptable de valeurs par défaut est déjà sélectionné.

Télécharger #

Vous êtes maintenant prêt à télécharger Blink. Cliquez sur le bouton Télécharger ou sélectionnez Esquisse → Télécharger. L’esquisse sera compilée, puis téléchargée sur votre carte Arduino. Si tout a fonctionné, la LED intégrée commencera à clignoter toutes les secondes.

[![Télécharger][4]][4]

Moniteur série #

Dans l’IDE Arduino, vous avez un moniteur série. Pour l’ouvrir, utilisez le bouton moniteur série sur le côté droit de la fenêtre.

[![Moniteur série][5]][5]

Assurez-vous que le code est téléchargé avant d’ouvrir le moniteur. Le téléchargement et le moniteur ne fonctionneront pas en même temps !

Cligner #

Voici un court exemple qui illustre les fonctions setup() et loop(). Cela peut être chargé dans l’IDE Arduino en choisissant Fichier> Exemples> 01. Bases> Blink. (Remarque : La plupart des cartes Arduino ont une LED déjà connectée à la broche 13, mais vous devrez peut-être ajouter une LED externe pour voir les effets de ce croquis.)

// the setup function runs once when you press reset or power the board
void setup() {
  // initialize digital pin 13 as an output.
  pinMode(13, OUTPUT);
}

// the loop function runs over and over again forever
void loop() {
  digitalWrite(13, HIGH);   // turn the LED on (HIGH is the voltage level)
  delay(1000);              // wait for a second
  digitalWrite(13, LOW);    // turn the LED off by making the voltage LOW
  delay(1000);              // wait for a second
}

L’extrait ci-dessus :

  1. Définit la fonction setup(). La fonction setup() est appelée en premier lors de l’exécution dans chaque programme Arduino.

    1. Sets pin 13 as an output.

      Without this, it might be set to an input, which would make the LED not work; however once it is set as an output it will stay that way so this only needs to be done once when the program starts.

  2. Définit la fonction loop(). La fonction loop() est appelée à plusieurs reprises tant que le programme est en cours d’exécution.

    1. digitalWrite(13, HIGH); turns the LED on.
    2. delay(1000); waits one second (1000 milliseconds).
    3. digitalWrite(13, LOW); turns the LED off.
    4. delay(1000); waits one second (1000 milliseconds).

Étant donné que loop() est exécuté à plusieurs reprises tant que le programme est en cours d’exécution, la LED clignotera pendant 2 secondes (1 seconde allumée, 1 seconde éteinte). Cet exemple est basé sur l’Arduino Uno et sur toute autre carte qui a déjà une LED connectée à la broche 13. Si la carte utilisée n’a pas de LED intégrée connectée à cette broche, une peut être attachée à l’extérieur.

Plus d’informations sur le timing (par exemple les retards et le temps de mesure) : https://www.wikiod.com/fr/arduino/gestion-du-temps

LED - Avec bouton de contrôle #

Vous pouvez également utiliser ce code pour configurer une LED avec un interrupteur à bouton avec une résistance de rappel, cela pourrait de préférence être à l’étape suivante après la configuration du contrôleur de LED initial

int buttonState = 0; // variable for reading the pushbutton status

void setup()
{
    // initialize the LED pin as an output:
    pinMode(13, OUTPUT); // You can set it just using its number
    // initialize the pushbutton pin as an input:
    pinMode(2, INPUT);
}

void loop()
{
    // read the state of the pushbutton value:
    buttonState = DigitalRead(2);

    // check if the pushbutton is pressed.
    // If it's not, the buttonState is HIGH : if (buttonState == HIGH)
    {
        // turn LED off:
        digitalWrite(13, LOW);
    }
    else
    {
        // turn LED off:
        digitalWrite(13, HIGH);
    }
}
