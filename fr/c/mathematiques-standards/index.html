Syntaxe #

  • #include <math.h>
  • double pow(double x, double y);
  • float powf(float x, float y);
  • long powl double (long double x, long double y);
  1. Pour établir un lien avec la bibliothèque mathématique, utilisez -lm avec les drapeaux gcc.
  2. Un programme portable qui a besoin de vérifier une erreur d’une fonction mathématique doit mettre errno à zéro et effectuer l’appel suivant feclearexcept(FE_ALL_EXCEPT); avant d’appeler une fonction mathématique. Au retour de la fonction mathématique, si errno est non nul, ou si l’appel suivant renvoie non nul fetestexcept(FE_INVALID | FE_DIVBYZERO | FE_OVERFLOW | FE_UNDERFLOW); alors une erreur s’est produite dans la fonction mathématique. Lisez la page de manuel de math_error pour plus d’informations.

Reste à virgule flottante double précision : fmod() #

Cette fonction renvoie le reste en virgule flottante de la division de x/y. La valeur renvoyée a le même signe que x.

#include <math.h> /* for fmod() */
#include <stdio.h> /* for printf() */

int main(void)
{
    double x = 10.0;
    double y = 5.1;

    double modulus = fmod(x, y);

    printf("%lf\n", modulus); /* f is the same as lf. */

    return 0;
}

Production:

4.90000

Important : Utilisez cette fonction avec précaution, car elle peut renvoyer des valeurs inattendues en raison de l’utilisation de valeurs à virgule flottante.

#include <math.h>
#include <stdio.h>

int main(void)
{
    printf("%f\n", fmod(1, 0.1));
    printf("%19.17f\n", fmod(1, 0.1));
    return 0;
}

Production:

0.1
0.09999999999999995

Simple précision et long reste à virgule flottante double précision : fmodf(), fmodl() #

Ces fonctions renvoient le reste en virgule flottante de la division de x/y. La valeur renvoyée a le même signe que x.

Précision unique :

#include <math.h> /* for fmodf() */
#include <stdio.h> /* for printf() */

int main(void)
{
    float x = 10.0;
    float y = 5.1;

    float modulus = fmodf(x, y);

    printf("%f\n", modulus); /* lf would do as well as modulus gets promoted to double. */
}

Production:

4.90000

Double Double Précision :

#include <math.h> /* for fmodl() */
#include <stdio.h> /* for printf() */

int main(void)
{
    long double x = 10.0;
    long double y = 5.1;

    long double modulus = fmodl(x, y);

    printf("%Lf\n", modulus); /* Lf is for long double. */
}

Production:

4.90000

Fonctions de puissance - pow(), powf(), powl() #

L’exemple de code suivant calcule la somme des séries 1+4(3+3^2+3^3+3^4+…+3^N) en utilisant la famille pow() de la bibliothèque mathématique standard.

#include <stdio.h>
#include <math.h>
#include <errno.h>
#include <fenv.h>

int main()
{
        double pwr, sum=0;
        int i, n;

        printf("\n1+4(3+3^2+3^3+3^4+...+3^N)=?\nEnter N:");
        scanf("%d",&n);
        if (n<=0) {
                printf("Invalid power N=%d", n);
                return -1;
        }

        for (i=0; i<n+1; i++) {
                errno = 0;
                feclearexcept(FE_ALL_EXCEPT);
                pwr = powl(3,i);
                if (fetestexcept(FE_INVALID | FE_DIVBYZERO | FE_OVERFLOW |
                        FE_UNDERFLOW)) {
                        perror("Math Error");
                }
                sum += i ? pwr : 0;
                printf("N= %d\tS= %g\n", i, 1+4*sum);
        }

        return 0;
}
