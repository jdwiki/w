<ctype.h> — classification et conversion des caractères

Introduction #

L’en-tête ctype.h fait partie de la bibliothèque C standard. Il fournit des fonctions de classification et de conversion de caractères.

Toutes ces fonctions prennent un paramètre, un int qui doit être soit EOF soit représentable comme un caractère non signé.

Les noms des fonctions de classification sont préfixés par ’est’. Chacun renvoie une valeur entière différente de zéro (TRUE) si le caractère qui lui est transmis satisfait la condition associée. Si la condition n’est pas satisfaite, la fonction renvoie une valeur nulle (FALSE).

Ces fonctions de classification fonctionnent comme indiqué, en supposant que les paramètres régionaux C par défaut :

int a;
int c = 'A';
a = isalpha(c); /* Checks if c is alphabetic (A-Z, a-z), returns non-zero here. */
a = isalnum(c); /* Checks if c  is alphanumeric (A-Z, a-z, 0-9), returns non-zero here. */
a = iscntrl(c); /* Checks is c is a control character (0x00-0x1F, 0x7F), returns zero here. */
a = isdigit(c); /* Checks if c is a digit (0-9), returns zero here. */
a = isgraph(c); /* Checks if c has a graphical representation (any printing character except space), returns non-zero here. */
a = islower(c); /* Checks if c is a lower-case letter (a-z), returns zero here. */
a = isprint(c); /* Checks if c is any printable character (including space), returns non-zero here. */
a = isupper(c); /* Checks if c is a upper-case letter (a-z), returns zero here. */
a = ispunct(c); /* Checks if c is a punctuation character, returns zero here. */
a = isspace(c); /* Checks if c is a white-space character, returns zero here. */
a = isupper(c); /* Checks if c is an upper-case letter (A-Z), returns non-zero here. */
a = isxdigit(c); /* Checks if c is a hexadecimal digit (A-F, a-f, 0-9), returns non-zero here. */

a = isblank(c); /* Checks if c is a blank character (space or tab), returns non-zero here. */

Il existe deux fonctions de conversion. Ceux-ci sont nommés en utilisant le préfixe ’to’. Ces fonctions prennent le même argument que celles ci-dessus. Cependant, la valeur de retour n’est pas un simple zéro ou non nul, mais l’argument passé a changé d’une manière ou d’une autre.

Ces fonctions de conversion fonctionnent comme indiqué, en supposant que les paramètres régionaux C par défaut :

int a;
int c = 'A';

/* Converts c to a lower-case letter (a-z). 
 * If conversion is not possible the unchanged value is returned. 
 * Returns 'a' here. 
 */
a = tolower(c);

/* Converts c to an upper-case letter (A-Z). 
 * If conversion is not possible the unchanged value is returned. 
 * Returns 'A' here. 
 */
a = toupper(c);

Classer les caractères lus à partir d’un flux #

#include <ctype.h>
#include <stdio.h>

typedef struct {
  size_t space;
  size_t alnum;
  size_t punct;
} chartypes;

chartypes classify(FILE *f) {
  chartypes types = { 0, 0, 0 };
  int ch;

  while ((ch = fgetc(f)) != EOF) {
    types.space += !!isspace(ch);
    types.alnum += !!isalnum(ch);
    types.punct += !!ispunct(ch);
  }

  return types;
}

La fonction classify lit les caractères d’un flux et compte le nombre d’espaces, de caractères alphanumériques et de ponctuation. Il évite plusieurs écueils.

  • Lors de la lecture d’un caractère à partir d’un flux, le résultat est enregistré sous la forme d’un int, car sinon il y aurait une ambiguïté entre la lecture de EOF (le marqueur de fin de fichier) et un caractère qui a le même modèle de bits.
  • Les fonctions de classification de caractères (par exemple, isspace) s’attendent à ce que leur argument soit _soit représentable comme un unsigned char, soit la valeur de la macro EOF _. Puisque c’est exactement ce que renvoie fgetc, il n’y a pas besoin de conversion ici.
  • La valeur de retour des fonctions de classification de caractères ne fait la distinction qu’entre zéro (signifiant « faux ») et différent de zéro (signifiant « vrai »). Pour compter le nombre d’occurrences, cette valeur doit être convertie en 1 ou 0, ce qui est fait par la double négation, ‘!!’.

Classer les caractères d’une chaîne #

#include <ctype.h>
#include <stddef.h>

typedef struct {
  size_t space;
  size_t alnum;
  size_t punct;
} chartypes;

chartypes classify(const char *s) {
  chartypes types = { 0, 0, 0 };
  const char *p;
  for (p= s; p != '\0'; p++) {
    types.space += !!isspace((unsigned char)*p);
    types.alnum += !!isalnum((unsigned char)*p);
    types.punct += !!ispunct((unsigned char)*p);
  }

  return types;
}

La fonction classifier examine tous les caractères d’une chaîne et compte le nombre d’espaces, de caractères alphanumériques et de ponctuation. Il évite plusieurs écueils.

  • Les fonctions de classification de caractères (par exemple, isspace) s’attendent à ce que leur argument soit _soit représentable comme un unsigned char, soit la valeur de la macro EOF _.
  • L’expression *p est de type char et doit donc être convertie pour correspondre au libellé ci-dessus.
  • Le type char est défini comme étant équivalent à signed char ou unsigned char.
  • Lorsque char est équivalent à unsigned char, il n’y a pas de problème, puisque chaque valeur possible du type char est représentable comme unsigned char.
  • Lorsque char est équivalent à signed char, il doit être converti en unsigned char avant d’être transmis aux fonctions de classification des caractères. Et bien que la valeur du caractère puisse changer à cause de cette conversion, c’est exactement ce qu’attendent ces fonctions.
  • La valeur de retour des fonctions de classification de caractères ne fait la distinction qu’entre zéro (signifiant « faux ») et différent de zéro (signifiant « vrai »). Pour compter le nombre d’occurrences, cette valeur doit être convertie en 1 ou 0, ce qui est fait par la double négation, ‘!!’.
