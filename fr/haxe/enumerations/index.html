Syntaxe #

  • enum identifiant { constructeurs }

Capture des valeurs d’énumération #

Les valeurs transmises en tant qu’arguments du constructeur enum peuvent être capturées dans des variables à l’aide de [pattern matching][1].

Supposons l’énumération suivante :

enum Color {
    RGB(r : Int, g : Int, b : Int);
    HSV(h : Int, s : Float, v : Float);
}

La valeur du canal rouge peut être capturée comme suit :

var color = Color.RGB(255, 127, 0);
var red = switch (color) {
    // Match the Color.RGB constructor and capture value into `r`
    case Color.RGB(r, _, _):
        // Return the captured red value
        r;
    // Catch-all for matching remaining constructors
    case _:
        // Return -1
        -1;
}

Essayez l’exemple sur [try.haxe.org][2].

  • [“Capture variable”, manuel Haxe][3]

Constructeurs enum correspondants #

Les constructeurs Enum peuvent être mis en correspondance à l’aide de [pattern matching][1].

Supposons l’énumération suivante :

enum Color {
    Red;
    Green;
    Blue;
    RGB(r : Int, g : Int, b : Int);
}

Les couleurs avec uniquement une valeur de canal vert peuvent être mises en correspondance comme suit :

var color = Color.RGB(0, 127, 0);
var isGreenOnly = switch (color) {
    // Match Green or RGB with red and blue values at 0
    case Color.RGB(0, _, 0) | Color.Green: true;
    case _: false;
}

Essayez l’exemple sur [try.haxe.org][2].

Références #

  • [“Enum matching”, manuel Haxe][3]
  • [“Or patterns”, manuel Haxe][4]

[1] : https://haxe.org/manual/lf-pattern-matching.html [2] : http://try.haxe.org/#ce180 [3] : https://haxe.org/manual/lf-pattern-matching-enums.html [4] : https://haxe.org/manual/lf-pattern-matching-or.html

Aperçu #

Les types d’énumération de Haxe sont les [types de données algébriques][1] (ADT). Leur utilisation principale est de décrire des structures de données. [Enums] [2] sont désignés par le mot-clé enum et contiennent un ou plusieurs constructeurs enum.

enum Color {
    Red;
    Green;
    Blue;
    RGB(r : Int, g : Int, b : Int);
}

L’énumération ci-dessus peut être instanciée comme suit :

var c1 = Color.Red;
var c2 = Color.RGB(255, 0, 0);

Essayez l’exemple sur [try.haxe.org][3].
