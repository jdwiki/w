Récupère le nom de l’interface pour la vue actuelle du magasin #

Mage::app()->getStore()->getFrontendName();

Obtenir l’ID de magasin actuel #

Mage::app()->getStore()->getStoreId();

Obtenir le code de magasin actuel #

Mage::app()->getStore()->getCode();

Cela renvoie le code du magasin, par ex. ’en’ pour une vitrine configurée pour l’anglais et appelée ’en’, et non l’identifiant numérique.

Déterminer si la vue du magasin est activée #

Mage::app()->getStore()->getIsActive();

Obtenir l’identifiant du site Web pour le magasin actuel #

Mage::app()->getStore()->getWebsiteId();

Obtenir le modèle de magasin actuel #

Mage::app()->getStore();

Renvoie une instance de Mage_Core_Model_Store

Obtenir le nom du groupe pour le magasin #

Mage::app()->getStore()->getGroup()->getName()

Obtenez tous les magasins Magento #

Mage::app()->getStores();

Renvoie un tableau de modèles Mage_Core_Model_Store.
