Sur cette page

  • Syntaxe
  • Paramètres

Syntaxe #

  • <img usemap="#[nom-carte]">
  • <map name="[map-name]"></map>
  • <zone>

Paramètres #

Balise/Attribut Valeur
<img> Vous trouverez ci-dessous les attributs spécifiques à l’image cliquable à utiliser avec <img>. Les attributs <img> normaux s’appliquent.
usemap Le nom de la carte avec un symbole dièse en préfixe. Par exemple, pour une carte avec name="map", l’image doit avoir usemap="#map".
   
<carte>  
nom Le nom de la carte pour l’identifier. À utiliser avec l’attribut usemap de l’image.
   
<zone> Vous trouverez ci-dessous des attributs spécifiques à <area>. Lorsque href est spécifié, faisant de <area> un lien, <area> prend également en charge tous les attributs de la balise d’ancrage (<a>) sauf ping. Consultez-les dans la documentation MDN.
alt Texte alternatif à afficher si les images ne sont pas prises en charge. Ceci n’est nécessaire que si href est également défini sur <area>.
coordonnées Les coordonnées décrivant la zone sélectionnable. Lorsque shape="polygon", cela doit être défini sur une liste de paires “x, y” séparées par des virgules (c’est-à-dire, shape="polygon" coords="x1, y1, x2, y2, x3, y3, ..."). Lorsque shape="rectangle", cela doit être défini sur gauche, haut, droite, bas. Lorsque shape="circle", cela doit être défini sur centerX, centerY, radius.
href L’URL du lien hypertexte, si spécifié. S’il est omis, la <zone> ne représentera pas un lien hypertexte.
forme La forme de la <zone>. Peut être défini sur default pour sélectionner l’image entière (aucun attribut coords nécessaire), circle ou circle pour un cercle, rectangle ou rect pour un rectangle, et polygon ou poly pour une zone polygonale spécifiée par des points d’angle.
  • La liste des paramètres ci-dessus est modifiée à partir de la documentation MDN : [<map>][1] et [<area>][2].
  • Il est possible de créer les coordonnées d’une carte d’image avec une image aux formes plus simples (comme dans l’exemple d’introduction ci-dessus) avec un éditeur d’image qui affiche les coordonnées (comme GIMP). Cependant, il peut être plus facile en général d’utiliser un générateur d’images cliquables, tel que [celui-ci][4].

[1] : https://developer.mozilla.org/en-US/docs/Web/HTML/Element/map [2] : https://developer.mozilla.org/en-US/docs/Web/HTML/Element/area [3] : http://www.w3schools.com/tags/tryit.asp?filename=tryhtml_areamap [4] : http://imagemap-generator.dariodomi.de/

## Introduction aux images cliquables

La description #

Une carte d’image est une image avec des zones cliquables qui agissent généralement comme des hyperliens.

L’image est définie par la balise [<img>][3] et la carte est définie par une balise [<map>][1] avec des balises [<area>][2] pour indiquer chaque zone cliquable. Utilisez les attributs usemap et name pour lier l’image et la carte.

Exemple de base #

Pour créer une image cliquable afin que chacune des formes de l’image ci-dessous soit cliquable :

![triangle, carré, cercle image][4]

Le code serait le suivant :

<img src="http://jaced.com/blogpix/2007/trisquarecircle/002.gif" usemap="#shapes">
<map name="shapes">
    <area shape="polygon" coords="79,6,5,134,153,134">
    <area shape="rectangle" coords="177,6,306,134">
    <area shape="circle" coords="397,71,65">
</map>

Vous devriez voir que le navigateur reconnaît les zones lorsque le curseur devient un pointeur. Voir une [démo en direct][5] sur JSFiddle, ou voir une démonstration ci-dessous :

![démonstration de l’image cliquable et du curseur][6]
