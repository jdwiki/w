Hue est une interface utilisateur permettant de se connecter et de travailler avec la plupart des technologies Bigdata couramment utilisées telles que HDFS, Hive, Spark, Hbase, Sqoop, Impala, Pig, Oozie, etc. Hue prend également en charge l’exécution de requêtes sur des bases de données relationnelles.

Hue, une application Web Django, a été principalement conçue comme un atelier pour exécuter des requêtes Hive. Plus tard, la fonctionnalité de Hue a augmenté pour prendre en charge différents composants de l’écosystème Hadoop. Il est disponible en tant que logiciel open source sous licence Apache.

Installation de Hue dans Ubuntu #

Cette installation suppose que hadoop est préinstallé sous l’utilisateur hadoop.

Conditions préalables:

La teinte dépend de ces packages suivants

  1. gcc
  2. g++
  3. libxml2-dev
  4. libxlst-dev
  5. libsasl2-dev
  6. libsasl2-modules-gssapi-mit
  7. libmysqlclient-dev
  8. python-dev
  9. python-setuptools
  10. libsqlite3-dev
  11. fourmi
  12. libkrb5-dev
  13. libtidy-0.99-0
  14. libldap2-dev
  15. libssl-dev
  16. libgmp3-dev

Installation de tous les packages

sudo apt-get update
sudo apt-get install gcc g++ libxml2-dev libxslt-dev libsasl2-dev libsasl2-modules-gssapi-mit libmysqlclient-dev python-dev python-setuptools libsqlite3-dev ant libkrb5-dev libtidy-0.99-0 libldap2-dev libssl-dev libgmp3-dev

Installation et configuration

Exécution de l’installation en tant qu’utilisateur “hadoop”.

su - hadoop

  1. Téléchargez Hue sur gethue.com (ce lien est un exemple obtenu sur le site Web de Hue)

    wget https://dl.dropboxusercontent.com/u/730827/hue/releases/3.9.0/hue-3.9.0.tgz

  2. Extrayez l’archive téléchargée

    tar -xvf hue-3.9.0.tgz

  3. Exécutez la commande d’installation

     cd hue-3.9.0
 PREFIX=/home/hadoop/ make install

  4. Une fois le processus ci-dessus terminé,

    Update ~/.bashrc file,

     export HUE_HOME=/home/hadoop/hue
 export PATH=$PATH:$HUE_HOME/build/env/bin

    source after adding the entries, source ~/.bashrc

  5. Configurez Hue (3 fichiers à éditer)

cd $HUE_HOME/desktop/conf

  • teinte.ini

    [desktop]
 server_user=hadoop
 server_group=hadoop
 default_user=hadoop
 default_hdfs_superuser=hadoop

cd $HADOOP_CONF_DIR

  • core-site.xml

      <property>
      <name>hadoop.proxyuser.hadoop.hosts</name>
      <value>*</value> 
  </property> 
  <property>
      <name>hadoop.proxyuser.hadoop.groups</name>
      <value>*</value> 
  </property>

  • hdfs-site.xml

      <property>
      <name>dfs.webhdfs.enabled</name>
      <value>true</value> 
  </property>

  1. Démarrer Hue (Démarrer les démons Hadoop s’ils ne sont pas déjà démarrés)

    nohup supervisor &

  2. Connectez-vous à l’interface Web Hue : http://localhost:8888

    username: hadoop

    password: user_choice

Dépendances d’installation #

[Les détails du processus d’installation de Hue][1] ne sont pas disponibles pour la plupart des systèmes d’exploitation, donc selon le système d’exploitation, il peut y avoir des variations sur les dépendances que vous devez installer avant d’exécuter le script d’installation fourni dans le [package d’installation][2] :

CentOS

sudo yum install ant
sudo yum install python-devel.x86_64
sudo yum install krb5-devel.x86_64
sudo yum install krb5-libs.x86_64
sudo yum install libxml2.x86_64
sudo yum install python-lxml.x86_64
sudo yum install libxslt-devel.x86_64
sudo yum install mysql-devel.x86_64
sudo yum install openssl-devel.x86_64
sudo yum install libgsasl-devel.x86_64
sudo yum install sqlite-devel.x86_64
sudo yum install openldap-devel.x86_64
sudo yum install -y libffi libffi-devel
sudo yum install mysql-devel gcc gcc-devel python-devel
sudo yum install rsync
sudo yum install maven
wget https://bootstrap.pypa.io/ez_setup.py -O - | sudo python
  1. BPF -> CentOS > 7.x sudo yum install libgmp3-dev -> CentOS < 6.x sudo yum install gmp gmp-devel gmp-status
