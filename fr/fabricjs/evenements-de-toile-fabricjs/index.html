Syntaxe #

  1. on(eventName, handler) - Attache un écouteur d’événement avec un rappel à l’objet.

  2. off(eventName, handler) - Supprime l’écouteur d’événement de l’objet. L’appel de cette fonction sans aucun argument supprimera tous les écouteurs d’événements sur l’objet.

  3. trigger(eventName, optionsopt) - Déclenche l’événement et l’objet d’options facultatives.

Paramètres #

Paramètre Descriptif
eventName Le nom de l’événement auquel vous souhaitez vous abonner, tel que ‘object:moving’
eventHandler La fonction que vous souhaitez exécuter lorsque cet événement particulier est déclenché
optionopt Objet Options

Fabric prend en charge un certain nombre d’événements pour permettre l’interactivité et l’extensibilité. Pour vous abonner aux événements d’un canevas, utilisez la méthode “on” de la même manière qu’elle est utilisée dans jQuery. Et vous souhaitez déclencher manuellement n’importe quel événement en utilisant la méthode trigger. Tous les événements sont dans le cadre d’une instance de canevas particulière. Visitez [Link][1] pour plus d’informations sur les événements

[1] : https://github.com/kangax/fabric.js/wiki/Working-with-events

Démo des événements de canevas Fabric js #

<canvas id="c" width="400" height="400"></canvas>

var canvas = new fabric.Canvas("c");

canvas.on('mouse:up', function () {
  console.log('Event mouse:up Triggered');
});

canvas.on('mouse:down', function () {
  console.log('Event mouse:down Triggered');
});

canvas.on('after:render', function () {
  console.log('Event after:render Triggered');
});

canvas.on('object:moving', function () {
  console.log('Event object:moving Triggered');
});

canvas.on('object:modified', function () {
  console.log('Event object:modified Triggered');
});

var text = new fabric.Textbox('Hello world', {
  width:250,
  cursorColor :"blue"
});
canvas.add(text);

Le code ci-dessus montre comment fonctionne l’API d’événement dans Fabric.js. En appelant on sur l’instance canvas, ou même sur les autres objets Fabric.js, comme l’instance Rect, vous pouvez écouter leurs événements et quand le les écouteurs sont déclenchés, le rappel que vous leur avez passé sera déclenché également.
