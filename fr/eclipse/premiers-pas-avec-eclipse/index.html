Installation et configuration #

Pour installer Eclipse, accédez à la page Web Téléchargements Eclipse où se trouve généralement un lien direct pour télécharger la dernière version d’Eclipse. À partir d’Eclipse Mars (version 4.5), un programme d’installation peut être téléchargé qui vous guide tout au long de la procédure d’installation, au lieu de télécharger l’ensemble de l’installation sous forme de fichier compressé (cette option est cependant toujours disponible). Il existe également des liens pour télécharger les anciens packages Eclipse.

Eclipse est disponible en plusieurs packages différents qui ciblent différents utilisateurs, comme indiqué dans la capture d’écran ci-dessous du programme d’installation. Par exemple, l’Eclipse IDE for Java Developers contient des outils de base qui prennent en charge le développement, le débogage et la création d’applications Java, ainsi qu’une prise en charge de base du contrôle de version, comme un plug-in qui permet de versionner des projets à l’aide de Git, tandis que Eclipse for Android Developers fournit un environnement pour créer des applications Android.

[![entrez la description de l’image ici][1]][1]

Une fois qu’un package est sélectionné, la page suivante permet à l’utilisateur de sélectionner le répertoire d’installation, ainsi que d’autres options. La capture d’écran suivante illustre la procédure sur une machine Windows.

[![entrez la description de l’image ici][2]][2]

Cliquez sur le bouton Installer pour lancer l’installation du package Eclipse dans ce répertoire.

Si Java est déjà correctement installé sur la machine, Eclipse devrait se lancer correctement et aucune configuration ou configuration n’est généralement requise. Cependant, il est recommandé de modifier certaines options de configuration pour Eclipse, par exemple pour spécifier dans quelle JVM Eclipse doit s’exécuter et pour configurer la mémoire minimale et maximale pour cette JVM. Pour ce faire, un fichier appelé eclipse.ini existe dans le répertoire d’installation, où se trouve cette configuration de démarrage. Cette page contient des détails sur la configuration d’Eclipse à l’aide de ce fichier.

Importation de projets existants #

Dans le menu Fichier, choisissez l’option ‘Importer…’.

[![entrez la description de l’image ici][1]][1]

Cela ouvre la boîte de dialogue Importer, qui demande le type de projet/fichier que vous souhaitez importer. Pour un projet Java de base, choisissez “Projets existants dans l’espace de travail” dans le dossier “Général”.

[![entrez la description de l’image ici][2]][2]

Ensuite, sélectionnez le répertoire où se trouve le(s) projet(s) à l’aide du bouton ‘Parcourir’. Tous les projets pouvant être importés dans Eclipse s’afficheront dans la section “Projets :”. Si le projet a déjà été importé, il sera toujours affiché mais la case à cocher sera désactivée.

[![entrez la description de l’image ici][3]][3]

Vous pouvez également importer des projets directement à partir d’un fichier compressé en choisissant « Sélectionner le fichier d’archive », puis en cliquant sur le bouton « Parcourir ».

Une fois que vous avez cliqué sur “Terminer”, le projet est maintenant visible dans votre explorateur de projets et prêt à être utilisé.

[![entrez la description de l’image ici][4]][4]

Installer Marketplace dans Eclipse #

Peu de versions classiques d’eclipse ne sont pas préinstallées avec le marché, cela peut être installé en suivant les étapes suivantes :

  1. Accédez à l’aide → Installer un nouveau logiciel
  2. Ajouter un nouveau référentiel (site spécifié ci-dessous)
  3. Outils à usage général → Client Marketplace
  4. Cliquez sur Terminer et vous avez terminé.

Sites de mise à jour de la place de marché :

Oxygen - http://download.eclipse.org/releases/oxygen/
Neon - http://download.eclipse.org/releases/neon/
Mars - http://download.eclipse.org/releases/mars/
Luna - http://download.eclipse.org/mpc/luna
Helios - http://download.eclipse.org/releases/helios
Juno - http://download.eclipse.org/releases/juno/

Raccourcis clavier utiles #

Gérer les fichiers et les projets #

  • Ctrl+Maj+R : Ouvrir une ressource (fichier, dossier ou projet)
  • Ctrl+Maj+S : Enregistrer tous les fichiers
  • Ctrl+W : Fermer le fichier courant
  • Ctrl+Maj+W : Ferme tous les fichiers

Fenêtre de l’éditeur #

  • F12 : Passer à la fenêtre de l’éditeur
  • Ctrl+E : Afficher la liste des éditeurs ouverts. Utilisez les touches fléchées et entrez pour passer
  • Ctrl+Page Down/Up : Passer à l’éditeur suivant / Passer à l’éditeur précédent
  • Ctrl+M : Agrandit ou minimise la fenêtre de l’éditeur en cours

Naviguer dans l’éditeur #

  • Ctrl+L : Aller à la ligne
  • Ctrl+Q : Aller au dernier emplacement modifié
  • Ctrl+Maj+P : Avec une parenthèse sélectionnée : passer à la parenthèse fermante ou ouvrante correspondante
  • Ctrl+Flèche Bas/Haut : Faire défiler l’éditeur sans changer la position du curseur

Éditer le texte #

  • Ctrl+D : Supprimer la ligne
  • Alt+Maj+Y : Enveloppe le texte sélectionné (adapte la largeur du texte à l’écran)
  • Alt+Maj+S : Ouvrir les options du menu Source
  • Alt+Maj+R : Refactoriser le mot en surbrillance dans tous les fichiers
  • Ctrl+Alt+Haut/Ctrl+Alt+Bas : Copie les lignes sélectionnées en haut/bas.

Rechercher et remplacer #

  • Ctrl+J : Tapez un terme à rechercher puis utilisez Ctrl+J / Ctrl+shift+J pour remonter/reculer
  • Ctrl+K/Ctrl+Maj+K : Ctrl+H puis ferme la fenêtre de recherche. Ensuite, recherchez l’occurrence précédente / suivante du terme de recherche.

Déplacer un bloc de code #

  • Shift+tab : Déplacer vers la gauche
  • Alt+Haut/Alt+Bas : Déplacer vers le haut/bas.

Création et exécution d’un programme Java HelloWorld #

Dans la barre d’outils, ouvrez la perspective Java.

Créer un nouveau projet Java #

Faites un clic droit dans l’explorateur de packages, et dans le menu, sélectionnez Nouveau -> Projet Java

Dans la boîte de dialogue suivante, entrez un nom de projet, puis cliquez sur “Terminer”.

Vous avez maintenant le nouveau projet dans votre espace de travail.

Créer une nouvelle classe Java #

Cliquez avec le bouton droit sur votre projet et dans le menu, sélectionnez “Nouveau -> Classe”.

Dans la boîte de dialogue, tapez le nom de la classe (il doit commencer par une majuscule), cochez également la case public static void main(String[] args), puis cliquez sur “Terminer”.

Vous avez maintenant le premier fichier Java de votre projet. L’éditeur ouvrira automatiquement ce nouveau fichier.

Dans la méthode main, tapez du code pour afficher Hello world!.

Exécutez votre classe Java #

Cliquez avec le bouton droit sur votre classe Java et, dans le menu, sélectionnez “Exécuter en tant que -> application Java”.

Voilà, vous voyez la sortie de votre programme Java dans la console.
