Les attributs de sous-type 'Image et 'Value prendront respectivement une valeur scalaire et une chaîne et ils retourneront respectivement une chaîne et une valeur scalaire. Le résultat de 'Image peut être entré dans 'Value pour obtenir la valeur d’origine. L’inverse est également vrai.

L’attribut __Scalar_Object__'Image peut être utilisé directement sur les objets (depuis Ada 2012-TC-1).

Syntaxe #

  • fonction Scalar’Image (Argument : Scalar’Base) return String;
  • fonction Discrete’Image (Argument : Discrete’Base) return String;
  • fonction Entier’Image (Argument : Entier’Base) return String;
  • fonction Enumeration’Image (Argument : Enumeration’Base) return String;
  • fonction Real’Image (Argument : Real’Base) return String;
  • fonction Numeric’Image (Argument : Numeric’Base) return String;
  • function Scalar’Value (Argument : String) return Scalar’Base;
  • fonction Discrete’Value (Argument : String) return Discrete’Base;
  • function Integer’Value (Argument : String) return Integer’Base;
  • function Enumeration’Value (Argument : String) return Enumeration’Base;
  • fonction Real’Value (Argument : String) return Real’Base;
  • fonction Scalar_Object‘Image return String ;

Notez que 'Image peut entraîner des résultats définis par l’implémentation (RM 3.5), notamment lorsque certains caractères graphiques nécessaires au résultat String ne sont pas définis dans Character. Considérez les répertoires plus larges de 'Wide_Image et 'Wide_Wide_Image.

L’autorisation d’utiliser l’attribut __Scalar_Object__'Image directement sur un objet a été ajoutée dans Ada 2012-TC-1 (avril 2016).

Imprimer le flotteur en utilisant l’attribut Image #

with Ada.Text_IO;

procedure Main is
   type Some_Float digits 8 range 0.0 .. 10.0;
   X : Some_Float := 2.71;
begin
   Ada.Text_IO.Put_Line (X'Image);
end Main;

Résultat #

2.71000E+00

Imprimer un entier en utilisant l’attribut Image #

with Ada.Text_IO;

procedure Main is
   type Some_Integer is range -42 .. 42;
   X : Some_Integer := 17;
begin
   Ada.Text_IO.Put_Line (X'Image);
end Main;

Résultat #

17

Imprime l’énumération à l’aide de l’attribut Image #

with Ada.Text_IO;

procedure Main is
   type Fruit is (Banana, Orange, Pear);
   X : Fruit := Orange;
begin
   Ada.Text_IO.Put_Line (X'Image);
   Ada.Text_IO.Put_Line (Pear'Image);
end Main;

Résultat #

ORANGE
PEAR

Imprimer l’énumération en utilisant l’attribut Image #

with Ada.Text_IO;

procedure Main is
   type Fruit is (Banana, Orange, Pear);
   X : Fruit := Orange;
begin
   Ada.Text_IO.Put_Line (Fruit'Image (X));
end Main;

Résultat #

ORANGE

Imprimer un entier en utilisant l’attribut Image #

with Ada.Text_IO;

procedure Main is
   X : Integer := 17;
begin
   Ada.Text_IO.Put_Line (Integer'Image (X));
end Main;

Résultat #

17

Imprimer Float en utilisant l’attribut Image #

with Ada.Text_IO;

procedure Main is
   X : Float := 2.71;
begin
   Ada.Text_IO.Put_Line (Float'Image (X));
end Main;

Résultat #

As Inverses #

with Ada.Text_IO;

procedure Image_And_Value is
   type Fruit is (Banana, Orange, Pear);
   X  : Fruit := Orange;
begin
   Ada.Text_IO.Put_Line (Boolean'Image
      (Fruit'Value (Fruit'Image (X)) = X
          and
       Fruit'Image (Fruit'Value ("ORANGE")) = "ORANGE"));
end Image_And_Value;

Résultat #

TRUE
