Créer un site Web sur Azure #

Créer un site Web sur Azure à l'aide de la page du portail

Démarrer

La première chose à faire est de vous connecter à votre compte Azure et d'accéder à votre [page de portail](https://portal.azure.com/). Si vous n'avez pas encore de compte, vous pouvez démarrer un essai gratuit [ici](https://azure.microsoft.com/en-us/free/).

Après vous être connecté à votre compte, vous pouvez commencer à suivre ce guide.



Le vrai guide

Dans le portail, appuyez sur le bouton Nouveau dans le menu de gauche.

[![Page du portail][1]][1]

Il apparaîtra un nouveau panneau contenant quelques services. Dans ce guide, nous nous concentrerons uniquement sur la section Web + Mobile.

[![Processus de création][2]][2]

Nous pouvons maintenant cliquer sur Web App. Dans le nouveau panneau, nous devons entrer quelques détails :

  • App name: This will be the name of your website
  • Subscription: You could have some different options based on your subscription. Pick the one you want to use
  • Resoruce group: It's like a container that will hold every kind of service. In this example I created a new one, you are free to use one you had already created
  • App service plan/Location: When you click on this option it will prompt out a new panel that will show all the App service plans you have created before. If it's empty, probably you need to create a new one. To do this just press the Create New button in the App Service panel
  • Application Insights: If you press the On button, it will create a new Application Insights resource that will allow you to collect some cool statistic about your app. We will skip it for the moment as we can add it later.
À ce stade, vérifiez toutes les données que vous avez insérées jusqu'à présent. Lorsque vous êtes sûr que vos données sont correctes, mettez une coche à côté de l'option Épingler au tableau de bord (vous verrez ainsi votre application directement sur votre tableau de bord) et appuyez sur Créer bouton.
Terminé ! Votre site Web est maintenant créé !



Exploration de votre site Web

Maintenant que votre site web est créé, cliquez sur la vignette créée sur votre tableau de bord. Il apparaîtra une fenêtre comme celle-ci :

[![Exploration de votre application Web][3]][3]

Sur la gauche, vous pouvez voir un menu qui vous permettra de personnaliser votre site Web. Maintenant, nous allons nous concentrer uniquement sur la première option, la Vue d’ensemble. En plus de cette vue, vous pouvez voir quelques boutons utiles (ceux soulignés en vert). Ils vous permettront respectivement de Parcourir, Arrêter, Redémarrer et Supprimer votre application web. Dans le même panneau, vous pouvez voir (sur le côté gauche) également le statut de votre site Web et son emplacement.

Sur la droite, il y a de très belles informations telles que l’URL de notre application Web. Nous trouvons également les identifiants FTP et son URL FTP.

Maintenant, si vous appuyez sur le bouton Explorer ou si vous saisissez l’URL de votre application dans la barre d’URL, vous verrez votre site Web. Au début, ce sera comme ça :

[![Votre site pas si cool, pour le moment][4]][4]

Comme vous pouvez le constater, ce n’est pas le site Web le plus cool sur Internet, mais vous apprendrez à le transformer en l’une des meilleures applications Web de tous les temps. Ou du moins, l’idée est celle-là.
