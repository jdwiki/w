Syntaxe #

  • $(fonction() { $( “#selecter” ).accordion(); });
  • $(fonction() { $( “#selecter” ).accordion({ active: 2 }); });
  • $(fonction() { $( “#selecter” ).accordion({ animate: 200 }); });
  • $(fonction() { $( “#selecter” ).accordion({ collapsible: true }); });

Paramètres #

Paramètre Détail
actif Tapez Boolean ou Integer, Boolean requiert repliable pour être vrai
animer Tapez Booléen, Nombre, Chaîne ou Objet
pliable Tapez Booléen
événement Tapez Chaîne
en-tête Élément de sélecteur de type
hauteurStyle Tapez Chaîne
icônes Tapez l’objet icône de l’interface utilisateur jQuery

Plus d’informations peuvent être trouvées ici: http://api.jqueryui.com/accordion/

Utilisation de base de l’accordéon #

Pour utiliser un accordéon, il faut avoir des en-têtes et du contenu à l’intérieur des en-têtes dans leur code HTML. Ensuite, il faut instancier la méthode accordion() de jQuery UI.

<script>
$(function() {
    $( "#accordion" ).accordion();
});
</script>

Dans le HTML :

<div id="accordion">
    <h3>First header</h3>
        <div>First content panel</div>
    <h3>Second header</h3>
        <div>Second content panel</div>
</div>

Utilisation de destruction d’accordéon #

$( "#accordion" ).accordion( "destroy" );

Cela supprimera complètement la fonctionnalité d’accordéon et affichera le code HTML par défaut en supprimant tous les éléments jQuery-UI.

Cette méthode ne prend aucun argument.

Accordéon désactivé #

$( "#accordion" ).accordion( "disable" );

Cette méthode désactivera l’accordéon, c’est-à-dire que les en-têtes ne sont pas sélectionnables, ce qui rend le contenu en lecture seule et statique.

Cette méthode ne prend aucun argument.

Accordéon activé #

$( "#accordion" ).accordion( "enable" );

Cette méthode permettra d’activer un accordéon. Cela activera un accordéon désactivé ou ne fera simplement rien sur un accordéon déjà activé.

Cette méthode ne prend aucun argument.

Option accordéon #

var options = $( "#accordion" ).accordion( "option" );

Cela renverra un PlainObject donnant toutes les options représentant l’accordéon sélectionné. Celui-ci contiendra toutes les valeurs des clés qui sont expliquées dans la section Paramètres.

Cette méthode prend des paramètres qui sont les noms d’options de base expliqués dans le paramètre. On peut définir les options comme ceci :

$( "#accordion" ).accordion( "option", "disabled", true );

Actualisation de l’accordéon #

$( "#accordion" ).accordion( "refresh" );

Cette méthode recalcule la hauteur des panneaux en accordéon si des en-têtes ou du contenu ont été ajoutés ou supprimés dans le DOM.

Utilisation du widget Accordéon #

var widget = $( "#accordion" ).accordion( "widget" );

Cette méthode renvoie un objet jQuery contenant l’accordéon.
