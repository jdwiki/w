Syntaxe #

  • $("#sortable").sortable({ /*Options Here*/ }); //Initialise Sortable

  • $("#sortable").sortable(“option”, “nom_option”, valeur_option); //Définir l’option en dehors de l’initialiseur

  • var valeur = $("#sortable").sortable(“option”, “nom_option”); // Obtient la valeur d’une option

Paramètres #

Paramètre Descriptif
Options  
ajouter à (jQuery, Element, Selector, String) [Par défaut : “parent”] L’élément auquel l’assistant est ajouté
axe (Chaîne) [Par défaut : false] Les directions dans lesquelles l’élément peut être déplacé (x ou y)
annuler (Sélecteur) [Par défaut : “input,textarea,button,select,option”] Ne démarre pas le tri si vous démarrez sur un élément correspondant au sélecteur
cours (Objet) [Par défaut : {}] Spécifiez des classes supplémentaires à ajouter aux éléments triables lorsque les classes structurelles sont ajoutées. ({ui-sortable-helper : classe_personnalisée})
connecterAvec (Sélecteur) [Défaut : false] Permet aux éléments d’un triable d’être glissés vers un autre
confinement (Élément, Sélecteur, Chaîne) [Par défaut : false] L’élément auquel les éléments sont contraints
curseur (Chaîne) [Défaut : “auto”] Définit le type de curseur à afficher lors du tri
curseurÀ (Objet) [Par défaut : false] Définit la position à partir de laquelle l’assistant semble être déplacé
désactivé (Booléen) [Défaut : false] Désactive le tri si true
dropOnEmpty (Booléen) [Valeur par défaut : true] Si les éléments faux de cet élément de tri ne peuvent pas être placés dans des éléments de tri vides
forceHelperSize (Booléen) [Par défaut : false] Force l’assistant à avoir une taille
forcePlaceholderSize (Booléen) [Par défaut : false] Force l’espace réservé à avoir une taille
grille (Array) [Default: false] Définit une grille pour aligner l’assistant sur ([ x,y ])
poignée (Sélecteur, Elément) [Par défaut : false] Définit les éléments sur lesquels le tri peut commencer. En face d’annuler
aide (Chaîne, Fonction) [Par défaut : “original”] Chaîne “original” ou “clone”, ou fonction qui renvoie l’élément à utiliser comme assistant.
articles (Sélecteur) [Par défaut : “> *”] Définit les éléments qui doivent être triables
opacité (Nombre de 0,01 à 1) [Par défaut : false] Définit l’opacité de l’assistant
espace réservé (Chaîne) [Par défaut : false] Définit une ou plusieurs classes à appliquer à l’espace réservé
revenir (Boolean, Number) [Défaut : false] Le temps qu’il faut à l’assistant pour glisser dans sa nouvelle position
faire défiler (Booléen) [Défaut : true] Défilement s’il faut faire défiler les bords de la page
scrollSensitivity (Nombre) [Par défaut : 20] Définit à quelle distance du bord de la page le curseur doit se trouver pour commencer le défilement
scrollSpeed ​​ (Nombre) [Par défaut : 20] La vitesse de défilement
tolérance (Chaîne) [Par défaut : “intersection”] Définit le mode à utiliser lors du calcul lorsqu’un élément est au-dessus d’un autre (“intersection” ou “pointeur”)
zIndex (Entier) [Par défaut : 1000] Définit le z-index de l’assistant lors du tri
Méthodes  
annuler() Annule le tri en cours et ramène les éléments à leur position avant le début du tri
détruire() Supprime la fonctionnalité triable et remet l’élément dans son état avant l’initialisation
désactiver() Désactive le
activer() Active le triable
instance() Renvoie l’objet d’instance sortables
option() Obtient les paires clé-valeur de toutes les options pour le triable
option(chaîne) Obtient la valeur d’une option
option(chaîne, quelconque) Définit la valeur de l’option spécifiée par la chaîne
option(Objet) Définit une ou plusieurs options avec l’objet étant des paires clé-valeur d’options
rafraîchir() Actualise les options triables en rechargeant tous les éléments triables. Cela provoque la reconnaissance de nouveaux éléments
refreshPositions() Rrefreses les positions mises en cache des éléments triables
sérialiser(Objet) Sérialise les identifiants des éléments (par défaut) dans une chaîne qui peut être soumise ou ajoutée à une URL Options de l’objet : {clé : définit la clé dans la chaîne sérialisée, attribut : [Default “id”] définit l’attribut à examiner, expression :[Par défaut : “/(.+)-=_/”] regex pour diviser l’attribut en paires clé-valeur}
versTableau(Objet) Sérialise l’ID des éléments triables dans un tableau. L’objet peut contenir un attribut de paramètre qui a l’attribut à mettre dans le tableau par défaut est id
widget() Renvoie un objet jQuery de l’élément triable
Événements  
activer (événement, ui) Déclenché lors d’une liste connectée, chaque liste connectée au démarrage du glisser la reçoit
avantArrêt(événement, ui) Les déclencheurs avant que le tri ne s’arrête lorsque l’assistant a été déplacé vers la même position que l’espace réservé
changement(événement, ui) Déclenché lorsque les éléments changent de position, c’est-à-dire. lorsque l’espace réservé se déplace
créer(événement, ui) Déclenché lors de la création de triable
désactiver (événement, interface utilisateur) Déclenché lorsque le tri s’arrête. Cela vaut également pour toutes les listes connectées
sortie(événement, ui) Déclenché lorsque l’élément est déplacé hors de la liste triable
sur(événement, ui) Déclenché lorsque l’élément est déplacé dans une liste triable
recevoir(événement, ui) Déclenché lorsqu’un élément d’une liste connectée a été déposé dans une autre. La cible est la liste de réception
supprimer (événement, interface utilisateur) Déclenché lorsqu’un élément d’une liste connectée a été déposé dans une autre. La cible est la liste des donateurs
sort(event, ui) Déclenché lors du tri
début(événement, ui) Déclenché au démarrage du tri
stop(événement, ui) Déclenché à l’arrêt du tri
update(event, ui) Déclenché lorsque le tri s’arrête et que la position DOM a été mise à jour

Documentation officielle [ici][1]

[1] : http://api.jqueryui.com/sortable/#option-appendTo

Grille triable avec mise en page flexible #

Cela a utilisé la disposition flexible avec le sortable pour créer une grille de boîtes réactives qui peuvent être déplacées par glisser-déposer.

HTML

<div id="sortable">
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
  <div>5</div>
</div>

JS

$(function(){
    $('#sortable').sortable({
        //pass all options in here
    });
});

CSS

#sortable{
    width: 500px;
    display: flex;
    flex-wrap: wrap;
}
#sortable div {
    margin: 10px;
    background-color: #f00;
    flex-basis: 100px;
    height: 100px;
}

Éléments fixes lors du déplacement #

Cet exemple utilise une classe sur l’espace réservé pour le transformer en une ligne et lui faire prendre aucune place.

HTML

<div id="sortable">
    <div>1</div>
    <div>2</div>
    <div>3</div>
    <div>4</div>
</div>

JS

$("#sortable").sortable({
    placeholder: 'placeholder',
    helper: 'clone',
    start: function(event, ui){
        ui.item.show();
    }
});

CSS

#sortable div{
  background-color: #f00;
  width: 50px;
  height: 50px;
  margin: 10px;
  padding: 0px;
}
#sortable div.placeholder{
  height: 4px;
  margin: -7px 10px;
}

Sortable - Animer le retour d’un élément non accepté #

Exemple de travail : https://jsfiddle.net/Twisty/4f5yh3pa/7/

L’annulation et la restauration d’un triable ne sont pas fortement documentées. Les aides montrent comment le déplacement d’un élément d’une liste vers une autre liste connectée peut être conditionnellement annulé. par défaut, ce n’est pas animé par sortable, cet exemple inclut une animation.

Résultat : la liste 2 n’accepte que les éléments dont la classe est “acceptable”. Sinon, les deux listes peuvent être triées naturellement.

HTML

<div class="ui-widget">
  <ul id="sortable1" class="connectedSortable">
    <li class="ui-state-default acceptable">Item 1</li>
    <li class="ui-state-default">Item 2</li>
    <li class="ui-state-default">Item 3</li>
    <li class="ui-state-default">Item 4</li>
    <li class="ui-state-default">Item 5</li>
  </ul>
  <ul id="sortable2" class="connectedSortable">
    <li class="ui-state-default">Item 6</li>
    <li class="ui-state-default acceptable">Item 7</li>
  </ul>
</div>

CSS

.ui-widget {
  position: relative;
}

.connectedSortable {
  border: 1px solid #eee;
  width: 142px;
  min-height: 20px;
  list-style-type: none;
  margin: 0;
  padding: 5px 0 0 0;
  float: left;
  margin-right: 10px;
}

#sortable1 {
  background: #fff;
}

#sortable2 {
  background: #999;
}

.connectedSortable li {
  margin: 0 5px 5px 5px;
  padding: 5px;
  font-size: 1.2em;
  width: 120px;
}

JavaScript

$(function() {
  $(".connectedSortable").sortable({
    connectWith: ".connectedSortable",
    receive: function(e, ui) {
      var $self = $(this);
      var $item = ui.item;
      var $sender = ui.sender;
      // Restrict condition to only one specific list if desired
      if ($(e.target).attr("id") == "sortable2") {
        if ($item.hasClass("acceptable")) {
          // Item Accepted
          console.log($self.attr("id") + " accepted item from: #" + $sender.attr("id") + " > " + $item.text());
        } else {
          // Item Rejected
          console.log($self.attr("id") + " rejected item from: #" + $sender.attr("id") + " > " + $item.text());
          // Animate the return of the items position
          $item.css("position", "absolute").animate(ui.originalPosition, "slow", function() {
            // Return the items position control to it's parent
            $item.css("position", "inherit");
            // Cancel the sortable action to return it to it's origin
            $sender.sortable("cancel");
          });
        }
      }
    }
  }).disableSelection();
});

Exemple simple #

Prenez n’importe quelle liste et ajoutez un identifiant au wrapper externe (ul, div)

<ul id="sortable">
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
<li>Item 4</li>
</ul>

Dans votre jquery :

$(function(){
    $('#sortable').sortable({
        //pass all options in here
    });
});

Cela permettra à tous les li dans le wrapper #sortable d’être glissés et déposés dans la liste
