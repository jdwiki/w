Interfaces fonctionnelles

  • Liste des interfaces fonctionnelles standard de Java Runtime Library par signature

Sur cette page

  • Liste des interfaces fonctionnelles standard de Java Runtime Library par signature

Dans Java 8+, une interface fonctionnelle est une interface qui n’a qu’une seule méthode abstraite (à part les méthodes d’Object). Voir JLS §9.8. Interfaces fonctionnelles.
