Application Hello World #

Étapes pour créer “Hello, World!” application dans Oracle Application Express :

  1. Connectez-vous à votre espace de travail.
  2. Cliquez sur Générateur d’applications. La page Générateur d’applications s’ouvre.
  3. Cliquez sur Créer. Une page avec 4 types d’application apparaît. La page contient un court texte d’aide sur chaque type d’application.
  4. Cliquez sur Bureau. La page avec certaines propriétés de l’application apparaît. Choisir:
    • Schema: default database schema for the application.
    • Name: the name of your application. By default it will be displayed in the page header on all pages.
    • Application: ID number, which have to be unique through all workspaces. By default APEX suggests next free number.
    • Theme and Theme Style: define appearance of application pages.
  5. Cliquez sur Créer une application. L’application est prête à être lancée.
  6. Cliquez sur Exécuter l’application. La page de connexion de votre application apparaît.

Les applications de bureau créées ont deux pages par défaut : la page de connexion et la page d’accueil. La page d’accueil a un accès restreint, pour la voir, entrez le login et le mot de passe de votre espace de travail APEX.

Essayer gratuitement #

Vous pouvez demander un espace de travail Oracle Application Express gratuit sur le site officiel : [apex.oracle.com][1].

[1] : https://apex.oracle.com/en/

Installation ou configuration #

Lorsque vous installez Oracle 11g ou 12cR1, une ancienne version d’Apex est préinstallée. Il est fortement recommandé de le mettre à niveau vers la dernière version.

  1. Accédez à [apex.oracle.com][1] et téléchargez la dernière version d’Apex.

  2. Suivez la documentation de la version spécifique pour l’installer.

  3. Lien de documentation important pour les trois dernières versions

  4. [Oracle Application Express Version 4.1][2]

  5. [Oracle Application Express Version 4.2][3]

  6. [Oracle Application Express version 5] [4]

  7. [Oracle Application Express version 5.1][6]

  8. Si vous êtes développeur Oracle Apex, visitez [Denes Kubicek Demo Application] [5] une fois, ce qui résout vos nombreux problèmes.
