Opérations de base sur les ensembles

Un ensemble est une collection d’éléments qui peuvent être n’importe quoi. Quel que soit l’opérateur dont nous avons besoin pour travailler sur ces ensembles, il s’agit en bref des opérateurs d’ensemble et l’opération est également connue sous le nom d’opération d’ensemble. L’opération d’ensemble de base comprend l’union, l’intersection ainsi que l’addition, la soustraction, etc.

Syntaxe #

  • Groupe-Objet

  • Group-Object -Property <propertyName>

  • Group-Object -Property <propertyName>, <propertyName2>

  • Group-Object -Property <propertyName> -CaseSensitive

  • Group-Object -Property <propertyName> -Culture <culture>

  • Group-Object -Property <ScriptBlock>

  • Sort-Objet

  • Sort-Object -Property <propertyName>

  • Sort-Object -Property <ScriptBlock>

  • Sort-Object -Property <propertyName>, <propertyName2>

  • Sort-Object -Property <propertyObject> -CaseSensitive

  • Sort-Object -Property <propertyObject> -Descending

  • Sort-Object -Property <propertyObject> -Unique

  • Sort-Object -Property <propertyObject> -Culture <culture>

Filtrage : Where-Object / where / ? #

Filtrer une énumération à l’aide d’une expression conditionnelle

Synonymes :

Where-Object
where
?

Exemple:

$names = @( "Aaron", "Albert", "Alphonse","Bernie", "Charlie", "Danny", "Ernie", "Frank")

$names | Where-Object { $_ -like "A*" }
$names | where { $_ -like "A*" }
$names | ? { $_ -like "A*" }

Retour:

Aaron Albert Alphonse

Ordre : Sort-Object / sort #

Trier une énumération par ordre croissant ou décroissant

Synonymes :

Sort-Object
sort

En supposant:

$names = @( "Aaron", "Aaron", "Bernie", "Charlie", "Danny" )

Le tri croissant est le tri par défaut :

$names | Sort-Object
$names | sort

Aaron Aaron Bernie Charly Dany

Pour demander l’ordre décroissant :

$names | Sort-Object -Descending
$names | sort -Descending

Dany Charly Bernie Aaron Aaron

Vous pouvez trier à l’aide d’une expression.

$names | Sort-Object { $_.length }

Aaron Aaron Danny Bernie Charlie

Regroupement : Group-Object / group #

Vous pouvez regrouper une énumération en fonction d’une expression.

Synonymes :

Group-Object
group

Exemples:

$names = @( "Aaron", "Albert", "Alphonse","Bernie", "Charlie", "Danny", "Ernie", "Frank")

$names | Group-Object -Property Length
$names | group -Property Length

Réponse:

Compte Nom Groupe
4 5 {Aaron, Danny, Ernie, Frank}
2 6 {Albert, Bernie}
1 8 {Alphonse}
1 7 {Charlie}

Projection : Select-Object / select #

La projection d’une énumération vous permet d’extraire des membres spécifiques de chaque objet, d’extraire tous les détails ou de calculer des valeurs pour chaque objet

Synonymes :

Select-Object
select

Sélection d’un sous-ensemble de propriétés :

$dir = dir "C:\MyFolder"

$dir | Select-Object Name, FullName, Attributes
$dir | select Name, FullName, Attributes
Nom Nomcomplet Attributs
Images C:\MonDossier\Images Annuaire
données.txt C:\MonDossier\data.txt Archives
source.c C:\MonDossier\source.c Archives
