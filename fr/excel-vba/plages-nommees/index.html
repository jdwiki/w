La rubrique doit inclure des informations spécifiquement liées aux plages nommées dans Excel, y compris les méthodes de création, de modification, de suppression et d’accès aux plages nommées définies.

Définir une plage nommée #

L’utilisation de plages nommées vous permet de décrire la signification du contenu d’une ou plusieurs cellules et d’utiliser ce nom défini à la place d’une adresse de cellule réelle.

Par exemple, la formule =A5*B5 peut être remplacée par =Largeur*Hauteur pour faciliter la lecture et la compréhension de la formule.

Pour définir une nouvelle plage nommée, sélectionnez la ou les cellules à nommer, puis tapez un nouveau nom dans la zone Nom à côté de la barre de formule.

[![entrez la description de l’image ici][1]][1]

Remarque : les plages nommées ont par défaut une portée globale, ce qui signifie qu’elles peuvent être accessible depuis n’importe où dans le classeur. Les anciennes versions d’Excel autorisent les noms en double, il faut donc veiller à empêcher les noms en double de portée globale sinon les résultats seront imprévisible. Utilisez le gestionnaire de noms à partir de l’onglet Formules pour modifier la portée.

[1] : https://i.stack.imgur.com/KqLbf.gif

Utilisation de plages nommées dans VBA #

Créer une nouvelle plage nommée appelée « MyRange » attribuée à la cellule « A1 »

ThisWorkbook.Names.Add Name:="MyRange", _
    RefersTo:=Worksheets("Sheet1").Range("A1")

Supprimer la plage nommée définie par nom

ThisWorkbook.Names("MyRange").Delete

Accéder à la plage nommée par nom

Dim rng As Range
Set rng = ThisWorkbook.Worksheets("Sheet1").Range("MyRange")
Call MsgBox("Width = " & rng.Value)

Accéder à une plage nommée avec un raccourci

Comme toute autre plage, les plages nommées sont accessibles directement via une notation de raccourci qui ne nécessite pas de “Plage”. objet à créer. Les trois lignes de l’extrait de code ci-dessus peuvent être remplacées par une seule ligne :

Call MsgBox("Width = " & [MyRange])

Remarque : La propriété par défaut d’une plage est sa valeur, donc [MyRange] est identique à [MyRange].Value

Vous pouvez également appeler des méthodes sur la plage. Ce qui suit sélectionne MyRange :

[MyRange].Select

Remarque : Une mise en garde est que la notation de raccourci ne fonctionne pas avec les mots qui sont utilisés ailleurs dans la bibliothèque VBA. Par exemple, une plage nommée Width ne serait pas accessible en tant que [Width] mais fonctionnerait comme prévu si accessible via ThisWorkbook.Worksheets("Sheet1").Range("Width")

Gérer les plages nommées à l’aide du gestionnaire de noms #

Onglet Formules > groupe Noms définis > bouton Gestionnaire de noms

Named Manager vous permet de :

  1. Créer ou changer de nom
  2. Créer ou modifier la référence de cellule
  3. Créer ou modifier la portée
  4. Supprimer la plage nommée existante

Tableaux de plages nommées #

Fiche d’exemple

[![entrez la description de l’image ici][1]][1]

Code

Sub Example()
    Dim wks As Worksheet
    Set wks = ThisWorkbook.Worksheets("Sheet1")
    
    Dim units As Range
    Set units = ThisWorkbook.Names("Units").RefersToRange
    
    Worksheets("Sheet1").Range("Year_Max").Value = WorksheetFunction.Max(units)
    Worksheets("Sheet1").Range("Year_Min").Value = WorksheetFunction.Min(units)
End Sub
