##Installation et configuration Ces instructions concernent une installation de base de Community Edition à des fins de développement local.

Mac #

(La documentation officielle est conservée [ici][1])

1. Installer #

Dépendances : [Homebrew][2]

brew install arangodb

Si vous ne voulez pas d’Homebrew ou si vous préférez un binaire, vous pouvez le télécharger à la place [ici][3].

2. Courir #

/usr/local/sbin/arangod &

3. Essayez-le #

Ouvrez votre navigateur et pointez sur [localhost:8529][4].

Exécutez arangosh dans votre shell. (Si ce n’est pas dans votre PATH, vous pouvez le trouver dans /usr/local/bin/arangosh).

4. Ajustez les paramètres (facultatif) #

Le fichier de configuration se trouve dans /usr/local/etc/arangodb3/arangod.conf .

#Ubuntu

(La documentation officielle est conservée [ici][5])

Si vous voulez juste le fichier deb et pas de gestion des paquets, téléchargez-le [ici][6].

Si vous souhaitez installer pour 14.04 au lieu de 16.04, changez la version en 14.04 où vous voyez 16.04 ci-dessous.

1. Ajouter le dépôt #

wget https://www.arangodb.com/repositories/arangodb31/xUbuntu_16.04/Release.key
apt-key add - < Release.key

2. Installer #

echo 'deb https://www.arangodb.com/repositories/arangodb31/xUbuntu_16.04/ /' | sudo tee /etc/apt/sources.list.d/arangodb.list
sudo apt install apt-transport-https
sudo apt update
sudo apt install arangodb3

3. Courir #

sudo service arangodb3 start

4. Essayez-le #

Ouvrez votre navigateur et pointez sur [localhost:8529][4].

Exécutez arangosh dans votre shell.

5. Ajustez les paramètres #

Vous pouvez modifier ArangoDB selon vos désirs dans /etc/arangodb3/arangod.conf.
