Créer un bot avec le BotFather #

Ouvrez une conversation avec [@BotFather][1] dans Telegram et cliquez sur la commande / start.

Après avoir obtenu une liste de commandes, sélectionnez la commande /newbot pour obtenir ce message :

D’accord, un nouveau bot. Comment allons-nous l’appeler ? Veuillez choisir un nom pour votre robot.

Entrez un nom pour votre bot, qui peut être n’importe quoi, et envoyez-le. Après cela, BotFather vous demandera un nom d’utilisateur pour votre bot :

Bien. Choisissons maintenant un nom d’utilisateur pour votre bot. Il doit se terminer par bot. Comme ceci, par exemple : TetrisBot ou tetris_bot.

Cela doit se terminer par “bot”. Si votre nom d’utilisateur souhaité est pris ou n’est pas disponible, réessayez avec un autre nom.

Enfin, votre bot est créé et le message suivant s’affiche. Vous devez utiliser le jeton du bot pour communiquer avec le télégramme, alors assurez-vous de l’écrire.

 C’est fait ! Félicitations pour votre nouveau robot. Vous le trouverez à telegram.me/???bot. Vous pouvez maintenant ajouter une description, à propos de la section et photo de profil de votre bot, voir /help pour une liste de commandes. Par le En fait, lorsque vous avez fini de créer votre super bot, envoyez un ping à notre Bot Support si vous voulez un meilleur nom d’utilisateur pour cela. Assurez-vous simplement que le bot est entièrement opérationnel avant de faire cela.

Utilisez ce token pour accéder à l’API HTTP : xxx:xxx

Avec le jeton gagné, vous pouvez maintenant envoyer un message de test en appelant le site Web https://api.telegram.org/botBOTTOKEN/sendmessage?chat_id=YOURCHATID&text=YOURTEXT

Il existe deux méthodes générales pour les bots de télégrammes pour l’interaction : une méthode push et une méthode pull. En utilisant la méthode pull, vous devez appeler /getupdates de temps en temps pour vérifier si de nouveaux messages sont envoyés à votre bot. La méthode push utilise un webhook (méthode /setwebhook) pour votre script qui est appelé chaque fois qu’un utilisateur envoie un message à votre bot.

Les informations fournies sont un [fichier au format JSON][2] avec toutes les informations (de l’expéditeur et du message) dont vous avez besoin.

Vous pouvez écrire votre bot dans de nombreux langages (PHP, Python, Lua …) tant qu’ils peuvent gérer le webhook en entrée et peuvent appeler des sites Web.

Pour plus d’informations, vous pouvez toujours utiliser la documentation de l’API BOT que vous pouvez trouver [ici.][3]

[1] : https://telegram.me/BotFather [2] : https://core.telegram.org/bots/api#update [3] : https://core.telegram.org/bots/api
