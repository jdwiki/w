Syntaxe #

  • ext Un objet qui spécifie la source de sortie et les extensions de fichier minifiées.
  • source La chaîne de suffixes des noms de fichiers par lesquels se terminent les fichiers source de sortie.
  • min When string : La chaîne de suffixe des noms de fichiers par lesquels les fichiers de sortie minifiés se terminent.
  • When Array : les expressions régulières à remplacer par les noms de fichiers d’entrée. Par exemple : [/.(.*)-source.js$/, ‘$1.js’]
  • exclude Ne minifiera pas les fichiers dans les répertoires.
  • noSource Ne sortira pas le code source dans les répertoires dest.
  • ignoreFiles Ne minimisera pas les fichiers qui correspondent au modèle.
  • mangle Passez false pour ignorer la modification des noms.
  • output Passez un objet si vous souhaitez spécifier des options de sortie supplémentaires. Les valeurs par défaut sont optimisées pour une meilleure compression.
  • compress Passez un objet pour spécifier des options de compression personnalisées. Passez false pour ignorer complètement la compression.
  • preserveComments Une option pratique pour options.output.comments. Par défaut, aucun commentaire n’est conservé.
  • all Préserve tous les commentaires dans les blocs de code
  • someConserver les commentaires commençant par un bang (!) ou inclure une directive Closure Compiler (@preserve, @license, @cc_on)
  • function Spécifiez votre propre fonction de conservation des commentaires. Le nœud actuel et le commentaire actuel vous seront transmis et vous devrez renvoyer soit “true” soit “false”.

Minifier JS en utilisant gulp-minify #

Tout d’abord, installez gulp et gulp-minify dans le répertoire du projet localement

npm install --save-dev gulp gulp-minify

Ajoutez ensuite la tâche min-js suivante à votre gulpfile.js

var gulp = require('gulp');
var minify = require('gulp-minify');

gulp.task('min-js', function() {
    return gulp.src('lib/*.js')
        .pipe(minify({
            ext: {
                min: '.min.js'
            },
            ignoreFiles: ['-min.js']
        }))
        .pipe(gulp.dest('lib'))
});

gulp.task('watch', function(){
  gulp.watch('lib/*.js', ['min-js']); 
  // Other watchers
});

gulp.task('default', ['min-js', 'watch']);

Cette tâche trouve tous les fichiers js dans le répertoire lib, les minfy et les enregistre dans le répertoire lib avec le suffixe .min.js. Par exemple, après avoir minifié le fichier lib/app.js sera créé un fichier lib/app.min.js

En plus de s’exécuter en tant que dépendance de la tâche gulp “par défaut”, cette tâche peut être exécutée manuellement en tapant la commande suivante :

gulp min-js
